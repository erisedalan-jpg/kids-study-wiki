# paper_aliases 规则与维护

`00-元/scripts/exam_pipeline_config.yaml` 中 `paper_aliases` 决定 PDF 文件名 → **卷别短简称**（如 `新课标Ⅰ` / `全国乙` / `北京`），短简称写入真题词条 frontmatter `卷别` 字段、文件名 `{year}-{gender}-{NN}.md` 与索引「试卷地图」一列。

## 工作原理

`_exam_utils.normalize_paper(filename, cfg)` 按 yaml 数组**顺序**逐条 `pattern in filename` 子串匹配，**第一个命中即返回**。无命中 → `"未知"`。

```python
for pattern, simple in cfg["paper_aliases"]:
    if pattern in pdf_filename:
        return simple
```

关键：**不是正则、不是 longest-first 自动排序，纯顺序敏感**。短前缀必须排长后缀之后，否则裸 prefix 提前命中。

## 同卷归一原则

不同省份用同卷时，`paper_aliases` 把不同省命名收敛到同一短简称：

```
全国Ⅱ卷(2008–2009) → 新课标(2010–2012) → 新课标Ⅱ(2013–2020)
→ 全国乙(2021–2022) → 新课标Ⅱ(2023–2024)
```

吉林 + 黑龙江 2008-2024 全段同卷（PDF 内容比 ratio=1.0000，见 `索引/真题/黑龙江-同卷说明.md`），仅入吉林一份。**扩省份前必先跑 `python 00-元/scripts/pdf_content_diff.py --dir-a ... --dir-b ...` 判同卷**，不要看文件名。

## 扩省份 checklist

1. **判同卷**：跑 `pdf_content_diff.py`。ratio ≥ .995 → 不建镜像目录，仅写同卷说明；ratio < .97 → 独立卷，进 5 步管线
2. **看文件名命名**：在 `素材/真题/{省}/.../{年}*.pdf` 列样本，识别卷别在文件名中的字面形式（如「新课标Ⅰ卷」/「新高考Ⅰ卷」/「（湖南）」）
3. **加 alias**：在 `paper_aliases` 加新条目。**严守顺序**——长 pattern 在前，裸词在后。如「（甘肃）」自主卷裸词放底；「全国丙卷」要在「全国丙」前
4. **跑单元测试**：`pytest 00-元/scripts/tests/test_exam_utils.py`（normalize_paper 有 fixture）
5. **干跑首学科**：`python 00-元/scripts/exam_screenshot.py --province <新省> --subject 数学` 看 `卷别=未知` 是否出现；出现即 alias 漏

## 易错点

- **顺序倒置**：「新课标」（无 Ⅱ）排在「新课标Ⅱ卷」前 → 所有新课标Ⅱ 卷被错归到「新课标」。修：长 pattern 必须先列
- **裸词省名兜底过强**：如「北京」放在所有北京特定卷 pattern 之前 → 卷别永远是裸「北京」，失去细分。修：自主卷裸省名放底；省内若有多卷型（如机考无听力 vs 正常卷）应加 pattern 区分
- **Ⅱ 位置差异**：2008-2009 数理化生用「全国卷Ⅱ」、语文英语用「全国Ⅱ卷」（Ⅱ 与「卷」位置反），两条 pattern 必须都加
- **机考特殊后缀**：北京英语「（机考 无听力）」与正常听力卷需要不同短简称区分时，须在裸「北京」前加更具体的 pattern

## 自主命题省补丁

部分自主命题省（如北京）`exam_extract_meta` 公式碎片严重时走 `split_by_answer_tag` 降级（按【答案】序号直接映射题号）。这是 alias 之外的另一层归一，详见 `exam_extract_meta.py` docstring。
