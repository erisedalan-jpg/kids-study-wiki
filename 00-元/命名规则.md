# 命名规则

## 文件名

- 文件名 = 中文概念名，如 `光.md`、`凑十法.md`、`红绿灯.md`
- 不要带学段前缀（"学前-数数.md" ❌；"数数.md" ✅）
- 不要带空格、标点；纯中文 + 必要的阿拉伯数字

## 序号前缀（按学年顺序）

学科目录内词条按知识点**首次出现学期**排序，文件名加序号前缀（数量 ≤99 用 2 位 `XX-`；>99 用 3 位 `XXX-`），方便在文件树中按学习路径顺序浏览。

格式示例：
- 数学 81 词条用 2 位：`16-加法.md`、`73-圆周率.md`
- 语文 240 词条用 3 位：`001-拼音.md`、`240-两小儿辩日.md`

每学期内顺序：**概念 → 古诗 → 课文**（语文）；**核心 → 拓展**（数学）；按教材单元次序（其它学科）。

学段-序号映射示例：

| 学段 | 数学 (01-81) | 语文 (001-240) |
|---|---|---|
| 学前 | 01-14 | — |
| 一上 | 15-24 | 001-038 |
| 一下 | 25-26 | 039-056 |
| 二上 | 27-31 | 057-077 |
| 二下 | 32-37 | 078-100 |
| 三上 | 38-42 | 101-123 |
| 三下 | 43-45 | 124-142 |
| 四上 | 46-51 | 143-161 |
| 四下 | 52-56 | 162-179 |
| 五上 | 57-63 | 180-195 |
| 五下 | 64-69 | 196-210 |
| 六上 | 70-77 | 211-225 |
| 六下 | 78-81 | 226-240 |

**同一概念跨学期出现时，序号取首次出现学期**。

### Bare-name alias 必填

序号前缀使 `[[加法]]` 类链接无法直接匹配文件名 `16-加法.md`。须在 frontmatter `aliases` 数组**首位**加 bare-name：

```yaml
aliases: [加法, addition, plus]
```

Obsidian 通过 alias 解析 `[[加法]]` → `16-加法.md`，**无需更新引用**。

## 别名（aliases）

frontmatter `aliases` 字段挂同义中文 + 英文，便于 `[[light]]` 也能跳到《光》：

```yaml
aliases: [光, light, 光线]
```

首位为 bare-name（解析 [[]] 引用），其余按需添加同义词与英文。

## 跨学科归属

- 同时涉及多学科的概念（"光"既是物理也是诗词意象），**主学科归位 + 在其它学科建短链词条**
- 短链词条只含 frontmatter + 一句话 + `[[主词条]]` 跳转，不重复正文

## 跨学段归属

- 跨学段的概念（"光"涵盖学前→初中）**只放一份**在主学科文件夹内
- frontmatter `学段: [学前, 小学, 初中]` 让多个学段索引都能查到

## 小学"科学"特例

不单建"科学/"文件夹；按主题拆：
- 光、声、力、电 → `物理/`
- 植物、动物、人体 → `生物/`
- 岩石、土壤、天气 → `地理/`
- 水、空气、燃烧 → `化学/`
