## 推荐学习路径

### 阶段 1: 化学入门 (绪言 + 第一单元, 2 周)
- 了解化学发展史：道尔顿原子论 → 门捷列夫元素周期表 → 现代绿色化学
- 物理变化 vs 化学变化：本质是有无新物质生成 → [[001-化学变化|化学变化]] [[004-物理变化|物理变化]]
- 实验基本操作：酒精灯、试管、烧杯使用规范；药品取用原则（固体/液体）
- 重点：玻璃仪器的连接与洗涤；实验安全红线

### 阶段 2: 空气与氧气 (第二单元, 2.5 周)
- 空气组成 (体积分数)：N₂ 78% / O₂ 21% / 稀有气体 0.94% / CO₂ 0.03%
- 氧气的物理性质（无色无味、密度比空气大、可液化）
- 氧气的化学性质：与 C、S、Fe、Mg 等反应 → [[043-氧化反应|氧化反应]]
- 氧气的实验室制备（KMnO₄ 加热 / H₂O₂ + MnO₂）→ [[040-分解反应|分解反应]]
- 催化剂概念：改变反应速率但自身质量和化学性质不变 → [[039-催化剂|催化剂]]

### 阶段 3: 微粒模型 (第三单元, 3 周)
- 分子、原子、离子：概念与区别 → [[027-分子|分子]] [[030-原子|原子]] [[036-离子|离子]]
- 元素与原子关系：同种元素原子的总称 → [[025-元素|元素]]
- 元素符号记忆：前 20 号元素（H Li Be B C N O F Ne Na Mg Al Si P S Cl Ar K Ca）
- 化合价规则：正负化合价 → 化学式的书写（根下标 = 化合价交叉法）
- 相对原子质量与相对分子质量计算

### 阶段 4: 水与净化 (第四单元, 2 周)
- 水的电解实验：2H₂O → 2H₂↑ + O₂↑（正极 O₂，负极 H₂，体积比 1:2）
- 水的净化步骤：沉降 → 过滤 → 吸附 → 消毒 → [[080-过滤|过滤]] [[079-蒸馏|蒸馏]]
- 硬水与软水：含可溶性 Ca²⁺、Mg²⁺ 盐的水为硬水；软化方法（煮沸/蒸馏）
- 溶液初步：溶质、溶剂、溶液概念

### 阶段 5: 化学方程式 (第五单元, 2.5 周)
- 质量守恒定律：反应前后各元素原子种类和数目不变 → [[058-质量守恒定律|质量守恒定律]]
- 化学方程式书写步骤：写→配→注→等
- 配平方法：最小公倍数法、观察法
- 利用方程式计算：已知质量求另一质量（比例法）

### 阶段 6: 碳的化合物 (第六单元, 2 周)
- 碳的同素异形体：金刚石 / 石墨 / C₆₀ → [[066-同素异形体|同素异形体]]
- CO₂ 的实验室制备：CaCO₃ + HCl → 收集（向上排空气法）→ 检验（澄清石灰水变浑浊）
- CO₂ 性质：与水反应、与碱反应（NaOH/Ca(OH)₂）→ [[064-二氧化碳|二氧化碳]]
- CO 的性质：可燃性、毒性、还原性（CO + CuO → Cu + CO₂）

### 阶段 7: 燃料与能源 (第七单元, 1.5 周)
- 燃烧的三个条件：可燃物 + 氧气 + 达到着火点 → 灭火原理（破坏任一条件）
- 化石燃料：煤（固体）/ 石油（液体）/ 天然气（气体，主要成分 CH₄）
- 天然气燃烧：CH₄ + 2O₂ → CO₂ + 2H₂O（蓝色火焰）
- 新能源：太阳能、风能、氢能简介

总课时约 17 周 ≈ 1 学期.

| # | 反应名称 | 化学方程式 |
|---|---|---|
| 1 | 氧气实验室制备（KMnO₄法） | 2KMnO₄ $\overset{\Delta}{=}$ K₂MnO₄ + MnO₂ + O₂↑ |
| 2 | 氧气实验室制备（H₂O₂法） | 2H₂O₂ $\overset{MnO_2}{=}$ 2H₂O + O₂↑ |
| 3 | 碳在氧气中充分燃烧 | C + O₂ $\overset{点燃}{=}$ CO₂ |
| 4 | 硫在氧气中燃烧 | S + O₂ $\overset{点燃}{=}$ SO₂ |
| 5 | 铁在氧气中燃烧 | 3Fe + 2O₂ $\overset{点燃}{=}$ Fe₃O₄ |
| 6 | 镁在氧气中燃烧 | 2Mg + O₂ $\overset{点燃}{=}$ 2MgO |
| 7 | 水的电解 | 2H₂O $\overset{通电}{=}$ 2H₂↑ + O₂↑ |
| 8 | CO₂ 实验室制备 | CaCO₃ + 2HCl = CaCl₂ + H₂O + CO₂↑ |

| 10 | CO 还原 CuO | CO + CuO $\overset{\Delta}{=}$ Cu + CO₂ |

### 典型实验

| 实验名称 | 目的 | 关键操作 | 现象 |
|---|---|---|---|
| 空气中 O₂ 含量测定（红磷法） | 测定氧气体积分数 | 红磷点燃后密封；冷却后进水 | 水升入约 1/5 体积 |
| H₂O₂ 制 O₂ | 实验室制备氧气 | 锥形瓶 + MnO₂ 催化；排水收集 | 带火星木条复燃 |
| 水的电解 | 验证水的组成 | 电解水；正负极收集气体体积比 | 正极 O₂ : 负极 H₂ = 1:2 |
| CO₂ 制备与检验 | 实验室制 CO₂ | 大理石 + 稀盐酸；导入澄清石灰水 | 石灰水变浑浊 |
| CO₂ 与 NaOH 反应（验证） | 验证 CO₂ 与碱反应 | 充满 CO₂ 的矿泉水瓶注入 NaOH 溶液 | 瓶子明显变瘪（待家长核对实验细节） |
| 铁丝在 O₂ 中燃烧 | 验证氧气助燃性 | 螺旋铁丝 + 火星引燃；瓶底铺沙/水 | 剧烈燃烧，火星四射，生成黑色固体 |

## 易错点 / 难点

- **化学变化 vs 物理变化**：爆炸不一定是化学变化（如气球爆炸）；判断依据是有无新物质生成
- **催化剂误区**：催化剂加快反应速率，但不影响生成物的量；反应后质量和化学性质不变
- **方程式配平**：Fe 与 O₂ 反应生成 Fe₃O₄ 而非 FeO 或 Fe₂O₃（初中常考易错）
- **化合价符号位置**：化合价写在元素符号正上方（+2），不是右上角（²⁺，那是离子符号）
- **CO 与 CO₂ 检验顺序**：先通 NaOH 除去 CO₂，再检验 CO（与 CuO 反应变红）
- **电解水正负极**：正极产生 O₂（体积少），负极产生 H₂（体积多）——"正氧负氢"
