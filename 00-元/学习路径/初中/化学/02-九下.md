## 推荐学习路径

### 阶段 1: 金属与金属材料 (第八单元, 2.5 周)
- 金属共同物理性质：导电、导热、延展性、金属光泽（汞常温液态）
- 铁的化学性质：与 O₂、HCl、CuSO₄ 的反应 → [[104-铁|铁]] [[095-置换反应|置换反应]]
- 金属活动性顺序（口诀记忆）：
  K Ca Na Mg Al Zn Fe Ni Sn Pb (H) Cu Hg Ag Pt Au
  钾钙钠镁铝，锌铁镍锡铅，(氢) 铜汞银铂金
- 活动性顺序的应用：判断金属能否置换稀酸中的氢；判断金属能否置换盐溶液中的金属
- 合金：铁合金（钢/生铁）、铜合金（黄铜/青铜）；合金性能优于纯金属（待家长核对合金具体牌号）

### 阶段 2: 溶液与溶解度 (第九单元, 2.5 周)
- 溶液三要素：均一性、稳定性、混合物 → [[083-溶液|溶液]] [[082-溶剂|溶剂]] [[087-溶质|溶质]]
- 饱和溶液与不饱和溶液的相互转化（降温/蒸发结晶）
- 溶解度曲线：横轴温度，纵轴溶解度（g/100g 水）；硝酸钾 vs 氯化钠曲线特征
- 溶质质量分数计算：w = 溶质质量 / 溶液质量 × 100%
- 配制一定质量分数的溶液：计算 → 称量（固体/量取液体）→ 溶解 → 转移 → [[待家长核对：量筒规格选择]]

### 阶段 3: 酸和碱 (第十单元, 3 周)
- 盐酸（HCl）性质：与指示剂/活泼金属/金属氧化物/碱/某些盐反应
- 硫酸（H₂SO₄）：浓 H₂SO₄ 有腐蚀性、吸水性；稀 H₂SO₄ 参与化学反应
- NaOH（烧碱）：白色固体，强碱，易潮解，与 CO₂/酸反应 → [[139-氢氧化钠|氢氧化钠]]
- Ca(OH)₂（熟石灰）：微溶于水，溶液称石灰水 → [[138-氢氧化钙|氢氧化钙]]
- 酸碱中和反应：酸 + 碱 → 盐 + 水 → [[137-中和反应|中和反应]]
- pH：0-14，< 7 酸性，= 7 中性，> 7 碱性；常见物质 pH 估记
- 酸碱指示剂：石蕊（酸红碱蓝）、酚酞（酸无色碱变红）

### 阶段 4: 盐与化肥 (第十一单元, 2 周)
- 盐的定义：由金属离子（或铵根离子）和酸根离子构成的化合物 → [[114-盐|盐]]
- 重要的盐：NaCl（食盐）/ Na₂CO₃（纯碱，碱性）/ NaHCO₃（小苏打）/ CaCO₃（石灰石）
- 复分解反应条件：生成沉淀、气体或水（三者之一），才能发生 → [[109-复分解反应|复分解反应]]
- 化肥种类：氮肥（NH₄HCO₃/CO(NH₂)₂）/ 磷肥（Ca₃(PO₄)₂）/ 钾肥（K₂SO₄/KCl）/ 复合肥
- 铵盐与碱共热放 NH₃（氨气检验法：湿润红色石蕊试纸变蓝）

### 阶段 5: 化学与生活 (第十二单元, 1.5 周)
- 有机物：含碳化合物（CO、CO₂、碳酸盐除外）→ [[128-有机物|有机物]] [[127-无机物|无机物]]
- 六大营养素：糖类 / 蛋白质 / 脂肪 / 维生素 / 水 / 无机盐（待家长核对分类细节）
- 合成材料三大类：合成纤维 / 合成橡胶 / 合成塑料（三大有机合成材料）
- 白色污染与塑料降解；废旧金属回收利用
- 化学与环境：酸雨（SO₂/NOₓ 排放）/ 温室效应（CO₂）/ 臭氧层破坏（氟利昂）

总课时约 17 周 ≈ 1 学期.

### 典型实验

| 实验名称 | 目的 | 关键操作 | 现象 |
|---|---|---|---|
| 铁与 CuSO₄ 溶液反应 | 验证铁活动性强于铜 | 铁片浸入蓝色 CuSO₄ 溶液 | 铁片表面析出红色固体，溶液蓝色变浅 |
| 铁钉锈蚀实验对比 | 探究铁生锈条件 | 对比干燥空气 / 蒸馏水 / 空气+水三组 | 空气+水组铁钉最先生锈（待家长核对实验时长） |
| 酸碱中和滴定（定性） | 验证中和反应 | NaOH 溶液逐滴加入盐酸（含酚酞指示剂） | 溶液由红色恰好变为无色时为终点 |
| pH 测定 | 测定溶液酸碱性 | pH 试纸蘸取溶液 → 与标准色卡比对 | 颜色对应 pH 值；不可将试纸伸入溶液 |
| 检验 NH₄⁺（铵根离子） | 鉴别铵盐 | 铵盐与 NaOH 溶液共热，用湿润红色石蕊试纸检验 | 试纸变蓝，说明产生 NH₃ |
| 粗盐提纯 | 去除可溶性杂质 | 溶解 → 过滤 → 蒸发结晶 → 称量 | 得到较纯 NaCl 晶体 |

## 易错点 / 难点

- **金属活动性顺序应用误区**：排在 H 前的金属才能置换稀酸中的 H₂；Fe 与 CuSO₄ 反应生成 FeSO₄（亚铁盐），而非 Fe₂(SO₄)₃
- **饱和溶液 ≠ 浓溶液**：Ca(OH)₂ 饱和溶液很稀；NaCl 饱和溶液较浓——饱和与浓稀是两回事
- **溶解度曲线读图**：固体溶解度随温度升高而增大（KNO₃）；NaCl 受温度影响小；Ca(OH)₂ 溶解度随温度升高而减小（特殊）
- **复分解反应条件**：必须有沉淀、气体或水生成，反应物也需是可溶性盐/酸/碱（不可溶物无法在溶液中接触反应）
- **Na₂CO₃ 与 NaOH 的区别**：Na₂CO₃ 是盐（碳酸钠），俗称"纯碱/苏打"，呈碱性但不是碱；NaOH 才是碱
- **浓 H₂SO₄ 的稀释**：必须将浓 H₂SO₄ 缓缓注入水中并搅拌，不能将水注入浓 H₂SO₄（会剧烈放热飞溅）
