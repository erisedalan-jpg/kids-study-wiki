## 推荐学习路径

### 阶段 1: 史前时期 (2 周)
- 元谋人 → 北京人 → 山顶洞人：体质特征与生产工具演进
- 半坡遗址 / 河姆渡遗址 → [[仰韶文化]] [[河姆渡文化]]
- 炎黄二帝传说 → [[禅让制]] → 尧舜禹

### 阶段 2: 夏商周 (3 周)
- 夏朝建立（约前2070）→ [[世袭制]] 取代禅让制
- 商朝：[[甲骨文]] [[青铜器]] → 商纣失政 → 武王伐纣
- 西周：[[分封制]] [[宗法制]] → 礼乐制度
- 春秋五霸：齐桓公 → 晋文公 → [[尊王攘夷]]

### 阶段 3: 战国与秦统一 (3 周)
- 战国七雄格局 → [[百家争鸣]]（儒、道、法、墨）
- [[商鞅变法]]（前356）→ 秦国强大
- 秦灭六国（前221）→ [[郡县制]] [[统一度量衡]] [[书同文]]
- 陈胜吴广起义 → 楚汉之争

### 阶段 4: 两汉 (3 周)
- 汉高祖休养生息 → [[文景之治]]
- 汉武帝：[[推恩令]] [[罢黜百家独尊儒术]] → 西域都护府
- [[丝绸之路]]（张骞出使）→ 东西方文明交流
- 东汉：光武中兴 → 外戚宦官交替专权 → 黄巾起义

### 阶段 5: 三国两晋南北朝 (3 周)
- [[赤壁之战]]（208）→ 三国鼎立格局
- 西晋短暂统一 → [[八王之乱]] → 五胡十六国
- 东晋南迁 → [[江南经济开发]]
- [[北魏孝文帝改革]]（471-499）→ 民族交融

总课时约 16 周 ≈ 1 学期.

## 重点知识点

### 关键时间节点（时间轴）
- 约170万年前：元谋人（中国境内已知最早人类）
- 约70万-20万年前：北京人
- 约前2070：夏朝建立（中国第一个王朝）
- 约前1600：商朝建立
- 约前1046：武王伐纣，西周建立
- 前770：平王东迁，东周（春秋）开始
- 前475：战国时期开始
- 前356：商鞅变法
- 前221：秦统一六国
- 前209：陈胜吴广起义
- 前202：西汉建立
- 公元8：王莽篡汉
- 25：东汉建立（光武帝）
- 184：黄巾起义
- 208：赤壁之战
- 220：曹魏建立，东汉结束
- 280：西晋统一

- 471-499：北魏孝文帝改革
- 589：隋统一（七上终点）

### 核心人物
- **黄帝**：传说中华夏族祖先，教材定位为部落联盟领袖
- **商鞅**：推行变法，奠定秦统一基础
- **秦始皇**：统一六国，建立中央集权制度
- **汉武帝**：推恩令、独尊儒术、开疆拓土
- **张骞**：两次出使西域，开拓丝绸之路
- **司马迁**：《史记》作者，中国第一部纪传体通史
- **曹操**：挟天子以令诸侯，赤壁之战失利
- **诸葛亮**：辅佐刘备建立蜀汉，《出师表》
- **北魏孝文帝**：推行汉化改革，促进民族交融
- **祖冲之**：圆周率计算（南朝，七上拓展）

### 重大事件
- **分封制推行**：西周通过分封诸侯建立统治秩序
- **商鞅变法**：以法治国，推动秦国从诸侯国走向统一
- **秦统一六国**：结束春秋战国长期分裂，建立统一多民族国家
- **罢黜百家独尊儒术**：汉武帝确立儒学官方地位
- **丝绸之路开通**：连接中国与中亚、西亚乃至欧洲的贸易与文化通道
- **赤壁之战**：奠定三国鼎立格局
- **北魏孝文帝改革**：推行汉语、汉服、汉俗，加速北方民族融合

## 易错点 / 难点

- **夏朝是否有考古实证**：教材说明夏朝存在尚无文字记载，考古仍在研究中（二里头遗址相关）；避免将传说与史实混淆
- **东周 = 春秋 + 战国**：平王东迁（前770）→春秋（前770-前476）→战国（前475-前221）
- **郡县制 vs 分封制**：郡县官员由皇帝任命，非世袭；分封制为世袭，易造成地方割据
- **推恩令的作用**：通过允许诸侯王将封地分给子弟，逐步削弱诸侯势力，而非直接废除
- **丝绸之路两次出使**：张骞第一次（前138）被匈奴扣押13年；第二次（前119）顺利抵达
- **三国鼎立建立年份**：220年曹丕称帝建魏，221年刘备称帝建汉，222年孙权称王建吴
