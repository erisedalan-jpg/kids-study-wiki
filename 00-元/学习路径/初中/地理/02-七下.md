## 推荐学习路径

### 阶段 1: 亚洲总论 (第6章, 2 周)
- 亚洲位置 → 世界最大洲；东临太平洋，南临印度洋，北临北冰洋
- 地形 → 中部高四周低；青藏高原、西西伯利亚平原、德干高原 → [[亚洲地形]]
- 气候 → 气候类型最复杂大洲；季风气候显著 → [[季风气候]]
- 主要河流 → 鄂毕河/叶尼塞河/长江/黄河/恒河/湄公河 → [[亚洲河流]]
- 人口 → 世界人口最多大洲；东亚/东南亚/南亚三大稠密区

### 阶段 2: 东南亚 (第7章, 1.5 周)
- 位置 → 十字路口；马六甲海峡战略地位 → [[马六甲海峡]]
- 地形 → 中南半岛山河相间纵列分布；马来群岛多火山地震
- 气候 → 热带季风/热带雨林气候；水稻、橡胶、椰子、油棕主产区
- 华人华侨 → 世界最大华人聚居地区 → [[东南亚]]

### 阶段 3: 印度 (第7章, 1 周)
- 地形三大区 → 喜马拉雅山脉/恒河平原/德干高原 → [[印度地形]]
- 气候 → 热带季风气候；旱涝灾害频繁；西南季风强弱影响农业
- 人口 → 世界第一人口大国 (已超中国)；种姓制度历史影响
- 农业与工业 → 水稻/小麦分布；软件外包产业 → [[印度]]

### 阶段 4: 日本 (第7章, 1 周)
- 位置 → 太平洋西北岸岛国；地震火山多发 (环太平洋火山地震带)
- 地形 → 山地丘陵为主；关东平原最大; 富士山 → [[日本地形]]
- 气候 → 温带/亚热带季风气候；海洋性强
- 经济 → 加工贸易型经济；"进口原料→出口产品"模式 → [[日本经济]]
- 文化 → 东西方兼容；樱花/和服/茶道 vs 西方科技

### 阶段 5: 俄罗斯 (第7章, 1 周)
- 位置 → 世界面积最大国；横跨亚欧两洲；首都莫斯科 → [[俄罗斯]]
- 地形 → 西部平原/中部西伯利亚高原/东部山地；乌拉尔山分亚欧
- 气候 → 温带大陆性气候为主；冬季漫长严寒 → [[温带大陆性气候]]
- 资源 → 矿产丰富；石油/天然气出口大国；贝加尔湖世界最深湖泊
- 交通 → 西伯利亚大铁路 (亚欧大陆桥)

### 阶段 6: 中东 (第8章, 1 周)
- 位置 → "三洲五海之地"；苏伊士运河/霍尔木兹海峡 → [[中东]]
- 资源 → 石油储量世界最大 (波斯湾沿岸)；水资源极度匮乏
- 气候 → 热带沙漠气候为主；干旱少雨 → [[热带沙漠气候]]
- 文化宗教 → 伊斯兰教发源地；阿拉伯民族为主；地区冲突根源

### 阶段 7: 欧洲西部 (第8章, 1 周)
- 位置 → 亚欧大陆西端；大西洋沿岸 → [[欧洲西部]]
- 地形 → 平原为主；波德平原/西欧平原广阔；阿尔卑斯山最高
- 气候 → 温带海洋性气候为主 (全年温和湿润) → [[温带海洋性气候]]
- 经济 → 高度发达；乳畜业典型；旅游业（法国/意大利/西班牙）
- 欧盟 → 区域合作典范；统一货币欧元 → [[欧盟]]

### 阶段 8: 撒哈拉以南非洲 (第8章, 1 周)
- 位置 → 撒哈拉沙漠以南；黑色人种的故乡 → [[撒哈拉以南非洲]]
- 地形 → 高原大陆 (平均海拔最高大洲除南极洲)；刚果盆地/东非高原
- 气候 → 热带草原气候为主；围绕热带雨林对称分布 → [[热带草原气候]]
- 资源 → 矿产丰富 ("富饶的大陆")；钻石/黄金/铀矿等
- 发展问题 → 人口增长快/粮食不足/疾病/殖民地历史影响

### 阶段 9: 澳大利亚 (第8章, 0.5 周)
- 位置 → 南半球；大洋洲最大国；四面环海 → [[119-Australia|澳大利亚]]
- 地形 → 西部高原/中部平原/东部山地 (大分水岭)
- 气候 → 半环状分布；中部热带沙漠；东南温带
- 特色 → 特有动植物（袋鼠/考拉/鸭嘴兽）；"骑在羊背上的国家"→ 畜牧业

### 阶段 10: 美国 (第9章, 1 周)
- 位置 → 北美洲；本土48州+阿拉斯加+夏威夷 → [[美国]]
- 地形三区 → 西部山地/中部平原/东部高原 (阿巴拉契亚山)
- 农业 → 农业地区专门化 (小麦带/玉米带/乳畜带/棉花带) → [[农业地区专门化]]
- 工业 → 世界最大经济体；高新技术产业(硅谷)；传统工业（五大湖区）
- 环境 → 资源消耗大；环境污染问题突出

### 阶段 11: 巴西 (第9章, 0.5 周)
- 位置 → 南美洲最大国；大部在热带 → [[巴西]]
- 地形 → 北部亚马孙平原/南部巴西高原
- 气候 → 热带雨林/热带草原气候；亚马孙雨林"地球之肺"
- 人口 → 种族多元（欧洲/非洲/印第安混合）；首都巴西利亚（内陆迁都）
- 工业 → 南美洲最发达；圣保罗工业中心

### 阶段 12: 极地地区 (第10章, 0.5 周)
- 南极 → 纬度最高/最冷/风最大/淡水资源最多（冰川） → [[南极洲]]
- 北极 → 北冰洋为主；北极熊/因纽特人
- 科考 → 中国长城站（南极）/黄河站（北极）→ 待家长核对站名与位置
- 保护 → 《南极条约》禁止军事活动与资源开采

总课时约 16 周 ≈ 1 学期.

## 重点知识点

### 地形类
- 亚洲地形特征：中部高、四周低；高原山地面积广大
- 日本、东南亚：地处板块碰撞带，地震火山活跃
- 非洲地形：高原大陆；东非裂谷带（板块张裂）

### 气候类
- 季风气候三种：热带/亚热带/温带季风（亚洲东部分布）
- 热带气候：雨林→草原→沙漠 随纬度分布；东非热带草原最典型
- 欧洲气候：西侧温带海洋性→向东过渡温带大陆性

### 区位特征类
- 马六甲海峡：沟通印度洋与太平洋；日本"海上生命线"
- 苏伊士运河：大西洋与印度洋捷径；中东战略核心
- 中东石油：波斯湾→霍尔木兹海峡→石油输出路线
- 巴西亚马孙：热带雨林破坏与全球气候变化关联

## 易错点 / 难点

- **亚洲河流流向**：中部高→四周低，河流呈放射状向四周流（鄂毕/叶尼塞/长江/恒河）
- **日本气候**：不是热带气候，而是温带/亚热带季风气候，海洋性明显
- **欧洲西部平原**：地形平坦有利于农业和交通，非地中海气候区
- **非洲气候对称**：热带雨林气候（赤道附近）向南北两侧依次是热带草原→热带沙漠
- **澳大利亚气候半环状**：内陆干旱，北部热带，东南温带，西南地中海
- **美国农业专门化**：各农业带记忆时注意纬度位置对应
