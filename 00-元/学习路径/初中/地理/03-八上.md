## 推荐学习路径

### 阶段 1: 从世界看中国 (第1章, 2 周)
- 中国位置 → 亚洲东部、太平洋西岸；半球位置（北半球/东半球）
- 领土范围 → 最北/最南/最东/最西四个极点；南北纬度差约50°; 东西经度差约60°
- 濒临海洋 → 渤海(内海)/黄海/东海/南海；台湾海峡；岛屿（台湾/海南）→ [[中国海洋]]
- 行政区划 → 34个省级单位 (23省/4直辖市/5自治区/2特区)；记忆省区轮廓 → [[中国行政区划]]
- 人口 → 世界第一人口大国（现已被印度超过，待家长核对）；人口政策演变
- 民族 → 56个民族；汉族占91%；少数民族分布特点（西部/南部） → [[中国民族]]

### 阶段 2: 地形地势 (第2章第一节, 2 周)
- 地势特征 → 西高东低，呈三级阶梯状 → [[中国地势三级阶梯]]
  - 第一级：青藏高原（平均海拔4000m以上）
  - 第二级：黄土高原/云贵高原/内蒙古高原/塔里木盆地/准噶尔盆地/四川盆地
  - 第三级：东部平原（东北平原/华北平原/长江中下游平原）
- 四大高原 → 青藏/黄土/云贵/内蒙古 → [[四大高原]]
- 四大盆地 → 塔里木/准噶尔/柴达木/四川 → [[四大盆地]]
- 三大平原 → 东北/华北/长江中下游 → [[三大平原]]
- 主要山脉 → 走向规律（东西/南北/东北-西南/西北-东南向）→ [[中国山脉]]

### 阶段 3: 气候 (第2章第二节, 2 周)
- 气候特征 → 气候复杂多样；季风气候显著；大陆性强 → [[中国气候]]
- 季风区与非季风区 → 大兴安岭-阴山-贺兰山-巴颜喀拉山-冈底斯山一线
- 温度带 → 寒温带/中温带/暖温带/亚热带/热带/高原气候区 → [[中国温度带]]
- 干湿地区 → 湿润/半湿润/半干旱/干旱四类 → [[中国干湿地区]]
- 季风气候特点 → 雨热同期 (利农业)；降水年际变化大 (易旱涝)
- 气候影响农业 → 不同区域适宜作物不同（水稻/小麦/玉米分布规律）

### 阶段 4: 河流湖泊 (第2章第三节, 2 周)
- 外流区与内流区 → 分界线（北段大兴安岭-阴山-贺兰山-祁连山-冈底斯山）→ [[外流河与内流河]]
- 主要外流河 → 长江/黄河/珠江/松花江/淮河/辽河/海河
- 长江 → 发源唐古拉山；最长河流(6300km)；水能宝库；"黄金水道" → [[长江]]
- 黄河 → 发源巴颜喀拉山；含沙量最大；下游地上河；治黄重点 → [[黄河]]
- 主要湖泊 → 青藏高原湖区（咸水湖为主）/东部平原湖区（淡水湖）→ [[中国湖泊]]
- 自然灾害 → 洪涝/干旱/台风/寒潮/地震/滑坡泥石流 → [[中国自然灾害]]

### 阶段 5: 自然资源 (第3章, 2 周)
- 土地资源 → 总量大、人均少；类型多样；耕地/草地/林地分布 → [[中国土地资源]]
- 水资源 → 总量丰富、时空分布不均；南多北少/夏多冬少 → [[中国水资源]]
- 矿产资源 → 煤/铁/石油主要分布；能源基地
- 资源问题 → 人均不足 + 浪费 + 污染；解决措施 → [[538-conservation|资源保护]]

### 阶段 6: 经济发展 (第4章, 3 周)
- 农业 → 东耕西牧/南稻北麦；农业布局与自然条件关系 → [[中国农业]]
- 工业 → 四大工业基地（辽中南/京津唐/沪宁杭/珠三角）→ [[中国工业基地]]
- 高新技术 → 北京中关村/深圳/上海浦东；科技产业园 → [[高新技术产业]]
- 交通运输 → 铁路网格局；"五纵三横"主干线；高铁发展 → [[中国交通]]
- 地区差异 → 东中西三大经济地带；西部大开发 → [[中国经济地带]]

总课时约 16 周 ≈ 1 学期.

## 重点知识点

### 地形类
- 三级阶梯分界线：第一/二级以昆仑山-祁连山-横断山为界；第二/三级以大兴安岭-太行山-巫山-雪峰山为界
- 地形对气候影响：青藏高原阻挡南亚季风；秦岭南北气候差异显著
- 地形对河流：地势西高东低，大河自西向东流；阶梯交界处水能丰富

### 气候类
- 秦岭-淮河线：南北气候/植被/农业/水文的重要分界线
- 季风气候优势：雨热同期，有利于农作物生长
- 西北内陆：温带大陆性气候；降水少，气温日较差和年较差大

### 区位特征类
- 黄河中游：黄土高原水土流失严重→下游泥沙淤积→地上河
- 长江流域：上游水能/中下游航运/洞庭湖调蓄洪水
- 中国水资源南丰北缺：南水北调工程（东线/中线/西线）

## 易错点 / 难点

- **行政区划记忆**：34个省级单位数量与分类；特别注意直辖市（4个）与自治区（5个）
- **地势阶梯界线山脉**：第二/三级界线 "大太巫雪"（大兴安岭/太行山/巫山/雪峰山）易记混
- **气候类型分布**：中国无热带雨林气候；热带季风气候仅在云南西南部、海南、台湾南部
- **黄河地上河成因**：中游水土流失（黄土高原疏松土质+暴雨）→下游流速减缓泥沙堆积
- **内外流河判断**：最终流入海洋=外流河；最终流入内陆湖泊/消失=内流河
