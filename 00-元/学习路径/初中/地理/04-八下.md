## 推荐学习路径

### 阶段 1: 四大地理区域划分 (第5章, 1 周)
- 划分依据 → 综合地理位置/自然地理/人文地理差异
- 四大区域 → 北方地区/南方地区/西北地区/青藏地区 → [[中国四大地理区域]]
- 分界线：
  - 北方 vs 南方：**秦岭-淮河**（1月0℃等温线/800mm等降水量线）
  - 北方/南方 vs 西北：**400mm等降水量线**（大兴安岭-张家口-兰州-拉萨附近）
  - 青藏 vs 其他：**地势第一阶梯边缘**（昆仑山-祁连山-横断山）
- 四大区域自然特征对比表 → [[区域比较]]

### 阶段 2: 北方地区 (第6章, 3 周)

#### 2a 东北三省
- 位置 → 黑龙江/吉林/辽宁；温带季风气候；冬季严寒漫长 → [[东北三省]]
- 地形 → 山环水绕中间平原（东北平原/三江平原/松嫩平原/辽河平原）→ [[东北平原]]
- 黑土地 → 世界三大黑土区之一；土壤肥沃；大豆/玉米/春小麦主产区
- 工业 → 重工业基地（钢铁/机械/石化）；老工业基地振兴 → [[东北老工业基地]]
- 湿地 → 三江平原湿地保护；黑土地保护问题 → 待家长核对最新政策

#### 2b 黄土高原
- 位置 → 太行山以西/乌鞘岭以东/秦岭以北/长城以南 → [[黄土高原]]
- 黄土地貌 → 塬/梁/峁/川/沟谷；千沟万壑地形特征
- 水土流失 → 成因（黄土疏松/暴雨集中/植被破坏/人为活动）→ 治理措施
- 水土保持 → 退耕还林/还草；淤地坝；梯田 → [[水土保持]]
- 黄河泥沙来源 → 黄河中游穿越黄土高原是泥沙主要来源

#### 2c 北京
- 首都功能 → 政治中心/文化中心/国际交往中心/科技创新中心 → [[北京]]
- 位置 → 华北平原西北角；北部燕山；西部太行山
- 历史文化 → 故宫/长城/天坛/颐和园等世界遗产 → 待家长核对世界遗产数量
- 城市发展 → 城市化问题（交通/环境/住房）；京津冀协同发展

### 阶段 3: 南方地区 (第7章, 3 周)

#### 3a 长江三角洲
- 位置 → 长江下游；上海为核心；苏南/浙北 → [[长江三角洲]]
- 地形 → 低平水网密集；"鱼米之乡""丝绸之府"
- 经济 → 全国最大经济核心区；金融/外贸/制造业集聚
- 城市群 → 上海/南京/杭州/苏州；长三角一体化 → [[城市群]]

#### 3b 珠江三角洲
- 位置 → 广东省南部；毗邻港澳；紧靠东南亚 → [[珠江三角洲]]
- 发展条件 → 对外开放政策（经济特区）；港澳引资；劳动力丰富
- 工业 → 出口加工制造业→现代高新技术产业转型
- 城市 → 广州/深圳/珠海；粤港澳大湾区 → [[粤港澳大湾区]]

#### 3c 云贵高原
- 地形 → 地表崎岖；喀斯特地貌（石灰岩地貌）为主 → [[喀斯特地貌]]
- 气候 → "天无三日晴"（降水多）；"四季如春"（贵阳气候）
- 民族 → 少数民族种类最多地区；傣族/苗族/彝族/白族等 → [[西南少数民族]]
- 农业限制 → 地形崎岖/耕地少；梯田农业

### 阶段 4: 西北地区 (第8章, 2 周)
- 位置 → 大兴安岭以西/长城-祁连山以北/国境线 → [[西北地区]]
- 气候 → 温带大陆性气候；降水从东向西递减；典型干旱区
- 地形 → 内蒙古高原/新疆地形（山脉-盆地相间"三山夹两盆"）→ [[新疆地形]]
  - 阿尔泰山/天山/昆仑山；准噶尔盆地/塔里木盆地
- 植被 → 草原→荒漠草原→荒漠 (自东向西) → [[荒漠化]]
- 农业 → 灌溉农业（河套平原/河西走廊）；新疆绿洲农业；坎儿井 → [[坎儿井]]
- 新疆 → 棉花主产区；石油/天然气资源丰富；瓜果糖分高（光照强/昼夜温差大）

### 阶段 5: 青藏地区 (第9章, 2 周)
- 位置 → 青海/西藏；世界屋脊；平均海拔4000m以上 → [[青藏高原]]
- 气候 → 高原山地气候；气温低/太阳辐射强/昼夜温差大 → [[高原气候]]
- 地形 → 雪山/冰川/湖泊；世界最高峰珠穆朗玛峰（8848m）→ 待家长核对最新测量值
- 农业 → 河谷农业（雅鲁藏布江谷地/湟水谷地）；高原牧场（牦牛/藏绵羊）→ [[河谷农业]]
- 三江源 → 长江/黄河/澜沧江发源地；"中华水塔" → [[三江源]]
- 生态保护 → 高原生态脆弱；三江源国家公园；藏羚羊保护

### 阶段 6: 综合复习 (2 周)
- 四大区域横向对比 → 自然环境/农业类型/工业特色/文化差异一览表
- 区域联系 → 西气东输/西电东送/南水北调 → [[区域资源调配]]
- 可持续发展 → 发展与保护协调；"绿水青山就是金山银山"理念

总课时约 16 周 ≈ 1 学期.

## 重点知识点

### 地形类
- 四大区域界线：秦岭-淮河（南北方）；400mm等降水量线（西北）；第一阶梯边缘（青藏）
- 黄土高原地貌：塬（大片平坦高原面）/梁（长条形岭）/峁（孤立圆丘）
- 新疆地形：三山夹两盆——阿尔泰山/准噶尔盆地/天山/塔里木盆地/昆仑山
- 喀斯特地貌：石灰岩被水溶蚀；云贵高原典型

### 气候类
- 秦岭-淮河气候意义：北方旱地/南方水田；亚热带/温带分界；常绿林/落叶林分界
- 西北干旱成因：距海远/地形阻挡→降水少；内陆型气候
- 青藏高寒：海拔高→气温低→高原气候；同纬度比较最冷

### 区位特征类
- 珠三角发展优势：政策（经济特区）+市场+劳动力+侨乡（港澳资金）+区位（临港）
- 东北老工业基地振兴：资源枯竭/技术落后/体制问题→政策支持
- 新疆棉花高产原因：光照充足/昼夜温差大（有利糖分/纤维积累）+灌溉水源（雪山融水）
- 西部大开发战略：西气东输/西电东送/青藏铁路；促进区域协调发展

## 易错点 / 难点

- **四大区域界线记忆**：西北与北方分界≠秦岭淮河；要用400mm等降水量线
- **黄土高原 vs 内蒙古高原**：同在北方地区，但地形/地貌/植被完全不同
- **珠三角发展阶段**：初期出口加工业→现代转型高新技术；不能只说"廉价劳动力"
- **喀斯特地貌成因**：必须有石灰岩+水；不是所有石灰岩区都一样发育
- **坎儿井原理**：利用地下暗渠引山地雪水；减少蒸发；适合极端干旱区
- **三江源"三江"**：长江/黄河/澜沧江（不是"三条支流"）
