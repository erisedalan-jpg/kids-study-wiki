## 推荐学习路径

### 阶段 1: 有理数 (第1章, 4 周)
- 正负数概念 → 数轴 → [[绝对值]]
- 有理数加法 → 减法（加法逆运算）→ 乘法（符号法则）→ 除法 → 乘方
- 混合运算顺序（乘方→乘除→加减）→ [[有理数运算]]
- 难点：负数乘除符号判断，混合运算不出错

### 阶段 2: 整式加减 (第2章, 3 周)
- 字母表示数 → 单项式（系数+次数）→ [[多项式]]（项数/次数）
- 同类项识别 → 合并同类项 → [[整式]]加减
- 去括号/添括号（注意符号变换）

### 阶段 3: 一元一次方程 (第3章, 4 周)
- 方程与方程的解 → 等式的两条性质
- 移项法则 → 系数化为1 → 标准解题步骤
- [[一元一次方程]]列方程解实际问题（行程/工程/分配类）

### 阶段 4: 几何图形初步 (第4章, 3 周)
- 立体图形 vs 平面图形分类 → 展开图
- 点/线/面 → 线段/射线/直线三概念辨析
- 角的度量（量角器）→ 余角/补角 → [[角的平分线]]

总课时约 14 周 ≈ 1 学期.

## 重点知识点

### 核心公式与定理

1. **绝对值定义**：|a| = a（a≥0）；|a| = -a（a<0）
2. **有理数乘法符号法则**：同号得正，异号得负；`(-a)×(-b) = ab`
3. **乘方**：aⁿ = a×a×…×a（n个a相乘）；(-1)²ⁿ = 1；(-1)²ⁿ⁺¹ = -1
4. **合并同类项**：系数相加，字母及指数不变
5. **等式性质**：两边同加/同减/同乘/同除同一数（不为0），等式成立
6. **余角与补角**：互余之和 = 90°；互补之和 = 180°
7. **线段中点**：M为AB中点 ⟹ AM = MB = ½AB

### 典型例题类型

1. **有理数混合运算**：含乘方、括号的多步运算（注意运算顺序）
2. **合并同类项化简**：多变量多项式的整理
3. **解一元一次方程**：含分母、含括号的方程
4. **列方程解应用题**：设未知数→建立方程→求解→检验
5. **几何角度计算**：余角/补角关系求未知角

## 易错点 / 难点

- **负数运算**：减负变加正，如 `-3 - (-5) = -3 + 5 = 2`；负数乘方（奇次仍负，偶次为正）
- **移项变号**：只有加减项移项才变号，乘除不移项
- **同类项识别**：字母种类和指数须完全相同；`2xy²` 与 `3x²y` 不是同类项
- **去括号**：括号前有负号，括号内所有项变号
- **角的类型**：锐角<90°，直角=90°，钝角90°~180°，平角=180°，周角=360°
