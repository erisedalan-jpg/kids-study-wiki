## 推荐学习路径

### 阶段 1: 相交线与平行线 (第5章, 4 周)
- 对顶角（相等）→ 邻补角（互补）→ [[125-垂线|垂线]]
- 平行线的判定（同位角相等/内错角相等/同旁内角互补）
- 平行线的性质（判定逆用）→ 命题、真命题、假命题
- 难点：区分判定条件（已知角→推平行）与性质（已知平行→推角）

### 阶段 2: 实数 (第6章, 3 周)
- 平方根概念 → 算术平方根 → [[140-立方根|立方根]]
- 无理数 → [[127-实数|实数]]（有理数 + 无理数）
- 实数大小比较 → 实数运算（含根号的化简）

### 阶段 3: 平面直角坐标系 (第7章, 2 周)
- x 轴/y 轴/原点/象限 → 点的坐标 (x, y) 读写
- [[132-平面直角坐标系|平面直角坐标系]]中坐标符号与象限关系
- 已知坐标描点；已知点读坐标

### 阶段 4: 二元一次方程组 (第8章, 4 周)
- 二元一次方程概念 → 方程组的解（满足两个方程的公共解）
- 代入消元法 → 加减消元法 → [[084-二元一次方程组|二元一次方程组]]
- 列方程组解应用题（两个未知量→设两元）

### 阶段 5: 不等式与不等式组 (第9章, 3 周)
- 不等号方向 → [[117-不等式性质|不等式性质]]（两边乘负数不等号变向！）
- 解一元一次不等式 → 数轴表示解集
- [[115-一元一次不等式组|不等式组]]：取各不等式解集的交集

### 阶段 6: 数据收集整理 (第10章, 2 周)
- 普查 vs 抽样调查 → 频率/频数
- 三种统计图特点与适用场景 → [[数据收集整理]]

总课时约 18 周 ≈ 1 学期.

## 重点知识点

### 核心公式与定理

1. **对顶角定理**：对顶角相等
2. **平行线判定（三条）**：同位角相等 ⟹ 两直线平行；内错角相等 ⟹ 平行；同旁内角互补 ⟹ 平行
3. **平行线性质（三条）**：两直线平行 ⟹ 同位角相等/内错角相等/同旁内角互补
4. **算术平方根**：√a 是非负数（a≥0）；(√a)² = a；√(a²) = |a|
5. **不等式性质3**：两边乘以（或除以）同一负数，不等号方向改变
6. **坐标象限**：第一象限(+,+)；第二(−,+)；第三(−,−)；第四(+,−)
7. **方程组解法**：代入法（将一个方程中的一元用另一元表示再代入）；加减法（两方程加减消去一元）

### 典型例题类型

1. **平行线几何角度计算**：利用平行线性质求多个角
2. **化简含根号的实数表达式**：如 `√12 - √3 + √48`
3. **列二元一次方程组解实际问题**：和差/价格/速度类
4. **解一元一次不等式组并在数轴上表示解集**
5. **读图分析**：条形图/折线图/扇形图的数据读取与比较

## 易错点 / 难点

- **平行线判定 vs 性质混用**：判定是"由角推线平行"，性质是"由线平行推角"，方向相反
- **不等式乘负数**：最易忘记翻转不等号，如 `-2x > 6` ⟹ `x < -3`
- **根号化简**：√a² = |a|，不是直接等于 a（a 可能为负）
- **坐标轴上的点**：x 轴上的点纵坐标为 0；y 轴上的点横坐标为 0
- **方程组增根**：代入验算是否满足原两个方程
