## 推荐学习路径

### 阶段 1: 二次根式 (第16章, 3 周)
- 二次根式定义（√a，a≥0）→ 最简二次根式条件（被开方数无完全平方因数；分母无根号）
- 根式化简 → [[分母有理化]]（乘共轭根式）
- 二次根式加减（合并同类根式）→ 乘除运算

### 阶段 2: 勾股定理 (第17章, 3 周)
- [[勾股定理]]：直角三角形两直角边的平方和等于斜边的平方（a²+b²=c²）
- 勾股定理逆定理：三边满足 a²+b²=c² ⟹ 是直角三角形
- 常见勾股数：3-4-5，5-12-13，8-15-17（待家长核对其余常用组）
- 应用：求对角线、梯子问题、坐标系中两点距离

### 阶段 3: 平行四边形 (第18章, 5 周)
- 平行四边形性质（对边相等/对角相等/对角线互相平分）→ 判定（五条）
- [[矩形]]：平行四边形+一个直角 → 对角线相等
- [[菱形]]：平行四边形+一组邻边相等 → 对角线互相垂直平分
- [[正方形]]：矩形+菱形 → 综合所有性质
- 四边形包含关系：正方形⊂矩形⊂平行四边形；正方形⊂菱形⊂平行四边形

### 阶段 4: 一次函数 (第19章, 5 周)
- 函数概念（自变量/因变量/定义域/值域）→ [[函数]]三种表示法（解析式/图像/表格）
- 正比例函数 y=kx → [[一次函数]] y=kx+b
- 图像：斜率 k 决定方向（k>0 递增，k<0 递减）；b 决定与 y 轴交点
- 一次函数与方程/不等式的联系（图像交点即方程组的解）

### 阶段 5: 数据分析 (第20章, 2 周)
- 平均数（加权平均数）→ 中位数 → 众数
- [[方差]]公式 → 标准差（方差开方）→ 比较数据波动性

总课时约 18 周 ≈ 1 学期.

## 重点知识点

### 核心公式与定理

1. **勾股定理**：在直角三角形中，a² + b² = c²（c 为斜边）
2. **勾股定理逆定理**：若三角形三边满足 a² + b² = c²，则该三角形是直角三角形
3. **分母有理化**：`a/√b = a√b/b`；`a/(√m+√n) = a(√m-√n)/(m-n)`
4. **平行四边形对角线**：对角线互相平分（互相平分是判定的充要条件）
5. **矩形对角线**：对角线相等且互相平分
6. **菱形对角线**：对角线互相垂直平分
7. **一次函数**：y = kx + b（k≠0）；k 为斜率；b 为纵截距
8. **方差公式**：S² = (1/n)×[(x₁-x̄)²+(x₂-x̄)²+…+(xₙ-x̄)²]

### 典型例题类型

1. **二次根式化简**：含多重根号或有理化的综合化简
2. **勾股定理应用题**：实际场景（梯子/城市路线/坐标距离）
3. **四边形性质与判定**：已知部分条件证明是某种特殊四边形
4. **一次函数图像问题**：求解析式、判断增减性、求交点坐标
5. **统计数据分析**：计算平均数/方差，比较两组数据波动

## 易错点 / 难点

- **二次根式条件**：√a 要求 a≥0；`√(a²) = |a|` 不是 a（a 可为负）
- **勾股数记忆**：3-4-5 类（6-8-10等倍数）最常用，注意斜边是最大边
- **平行四边形判定**：五条判定条件（一条对角线平分另一条 ≠ 对角线互相平分，不够条件！）
- **一次函数 k 与增减性**：k>0 从左到右递增；k<0 递减；纵截距 b 不影响增减
- **方差计算**：每项与均值的差要平方，再求平均，不能直接对差求平均
