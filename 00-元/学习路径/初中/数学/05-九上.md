## 推荐学习路径

### 阶段 1: 一元二次方程 (第21章, 5 周)
- [[205-一元二次方程|一元二次方程]]概念（ax²+bx+c=0，a≠0）→ 直接开平方法（适合无一次项）
- 配方法 → [[215-公式法-一元二次方程|求根公式]] `x = (-b ± √(b²-4ac)) / (2a)`
- 因式分解法（逆用乘法公式）
- [[234-韦达定理-初中|韦达定理]]（根与系数关系）：x₁+x₂ = -b/a；x₁×x₂ = c/a
- 判别式 Δ = b²-4ac：Δ>0 两个不相等实根；Δ=0 两个相等实根；Δ<0 无实根
- 列方程解应用题（面积/整数/速度/增长率类）

### 阶段 2: 二次函数 (第22章, 5 周)
- [[212-二次函数定义|二次函数]] y = ax²+bx+c（a≠0）的一般式 → 顶点式 y = a(x-h)²+k
- 顶点坐标：h = -b/(2a)；k = c - b²/(4a)
- 抛物线开口方向（a>0 开口向上，a<0 向下）→ 对称轴 x = -b/(2a)
- 与 x 轴交点（令 y=0 解一元二次方程）→ 与 y 轴交点（令 x=0）
- 二次函数最值：顶点处取最大/最小值
- 二次函数在闭区间上的最值（需分 a>0/a<0 讨论顶点位置）

### 阶段 3: 旋转 (第23章, 2 周)
- 旋转三要素：旋转中心、旋转角、旋转方向
- [[054-平移与旋转|旋转]]性质：对应点到旋转中心距离相等；对应点与旋转中心连线所成角等于旋转角
- 中心对称（旋转 180°的特例）→ 中心对称图形

### 阶段 4: 圆 (第24章, 6 周)
- 圆的基本元素（圆心/半径/弦/弧/圆心角/圆周角）
- 垂径定理：垂直于弦的直径平分该弦及其对应的弧
- 圆心角与圆周角：同弧所对圆周角等于圆心角的一半
- [[217-切线|切线]]性质：切线垂直于过切点的半径
- 切线长定理：从圆外一点到两切点的距离相等
- 点与圆的位置（d<r/d=r/d>r）→ 直线与圆（相交/相切/相离）→ 两圆位置关系（待家长核对具体条件）
- 弧长公式：`l = nπR/180`（n 为圆心角度数）；扇形面积：`S = nπR²/360`

### 阶段 5: 概率初步 (第25章, 2 周)
- 必然事件/不可能事件/随机事件 → [[946-概率|概率]]的定义（P 满足 0≤P≤1）
- 古典概型：等可能事件 → P = 有利结果数/总可能结果数
- 列举法（表格/树形图）→ 频率稳定性

总课时约 20 周 ≈ 1 学期.

## 重点知识点

### 核心公式与定理

1. **一元二次方程求根公式**：x = (-b ± √(b²-4ac)) / (2a)
2. **判别式**：Δ = b² - 4ac；Δ>0 两不等实根；Δ=0 两相等实根；Δ<0 无实根
3. **韦达定理**：x₁+x₂ = -b/a；x₁·x₂ = c/a
4. **二次函数顶点坐标**：(-b/(2a), (4ac-b²)/(4a))；对称轴 x = -b/(2a)
5. **圆周角定理**：同弧所对的圆周角相等，等于对应圆心角的一半
6. **切线性质**：过切点的半径垂直于切线
7. **弧长公式**：l = nπR/180；**扇形面积**：S = nπR²/360 = lR/2
8. **古典概型**：P(A) = A 包含的基本事件数 / 基本事件总数

### 典型例题类型

1. **用三种方法解一元二次方程**：开平方/配方/公式/因式分解灵活选用
2. **已知方程根的条件，用韦达定理求参数**
3. **二次函数图像分析**：已知两点或顶点求解析式；求最值
4. **圆中角度与线段计算**：综合垂径定理、圆周角定理、切线性质
5. **概率列举计算**：用树形图列出所有等可能结果

## 易错点 / 难点

- **求根公式符号**：b²-4ac 中 b 和 c 代入时要注意负数的平方处理
- **韦达定理方向**：仅适用于已是标准形式（a≠0）的方程，且根是实数根
- **二次函数最值**：在开区间与闭区间上最值不同，需分情况讨论
- **圆周角与圆心角**：圆心角是圆周角的 2 倍（同弧），不是任意两角的关系
- **切线判定与性质混用**：性质是"已知切线求垂直"，判定是"已知垂直求切线"
