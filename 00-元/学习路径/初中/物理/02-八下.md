## 推荐学习路径

### 阶段 1: 力的基本概念 (第7章, 3 周)
- 力的三要素: 大小 / 方向 / 作用点; 力的示意图画法
- 弹力: 弹簧测力计使用 (调零、量程、分度值); 胡克定律定性
- 重力: G = mg (g = 10 N/kg 或 9.8 N/kg); 重力方向竖直向下
- 摩擦力: 影响因素（压力 + 接触面粗糙程度）; 增大 / 减小摩擦的方法

### 阶段 2: 牛顿定律与平衡 (第8章, 2 周)
- 牛顿第一定律: 不受力 → 静止保持静止 / 运动保持匀速直线运动
- 惯性: 一切物体固有属性, 只与质量有关
- 二力平衡条件: 同体 / 等大 / 反向 / 共线
- 受力分析: 先画重力, 再找接触力 (支持力 / 摩擦力 / 拉力)

### 阶段 3: 压强专题 (第9章, 3 周)
- 固体压强: p = F/S; 增大 / 减小压强的方法 (改变 F 或 S)
- 液体压强: p = ρgh; 连通器原理; 液压机 (帕斯卡定律)
- 大气压: 托里拆利实验 (760 mmHg ≈ 1.013×10⁵ Pa); 流体速度与压强关系 (定性)

### 阶段 4: 浮力专题 (第10章, 3 周)
- 浮力产生: 上下液体压力差
- **阿基米德原理**: F浮 = G排液 = ρ液 g V排
- 称重法: F浮 = G - F示
- 浮沉条件: F浮 > G 上浮; F浮 = G 悬浮; F浮 < G 下沉
- 密度比较应用: 轮船 / 潜水艇 / 气球

### 阶段 5: 功与能 + 简单机械 (第11-12章, 4 周)
- W = Fs; 做功两要素: 有力 + 沿力方向有位移
- P = W/t = Fv; 单位瓦 (W)、千瓦 (kW)
- 动能 / 重力势能 / 弹性势能; 机械能守恒 (无摩擦)
- 杠杆平衡: F₁l₁ = F₂l₂; 省力杠杆 / 费力杠杆 / 等臂杠杆
- 滑轮: 定滑轮改变力方向、动滑轮省一半力; 滑轮组 F = G/n (理想)
- 机械效率 η = W有 / W总 × 100%

总课时约 15 周 ≈ 1 学期 (每周 3 课时).

## 重点知识点

### 核心公式 (含 SI 单位)

| 公式                  | 含义        | 单位                         |
| ------------------- | --------- | -------------------------- |
| G = mg              | 重力        | N (m: kg, g = 10 N/kg)     |
| p = F / S           | 固体压强      | Pa = N/m²                  |
| p = ρgh             | 液体压强      | Pa (ρ: kg/m³, h: m)        |
| F浮 = ρ液 g V排        | 阿基米德原理    | N                          |
| W = F·s             | 功         | J = N·m                    |
| P = W / t           | 功率        | W = J/s                    |
| F₁l₁ = F₂l₂         | 杠杆平衡      | N·m (力矩)                   |
| η = W有 / W总 × 100%  | 机械效率      | %（无单位）                     |

### 典型实验

- **探究影响摩擦力大小的因素**: 弹簧测力计匀速拉动木块; 控制变量法
- **探究液体压强规律**: U形管压强计; 改变深度 / 密度 / 方向
- **验证阿基米德原理**: 弹簧测力计 + 量筒; 称重法 + 排水法比较
- **探究杠杆平衡条件**: 铁架台 + 刻度杠杆; F₁l₁ = F₂l₂ 的验证
- **测滑轮组机械效率**: 弹簧测力计测拉力; 刻度尺测距离; η = G×h / F×s

## 易错点 / 难点

- **压强 vs 压力**: p = F/S; 同一力作用面积越小压强越大; 压力不等于重力
- **液体压强只与深度和密度有关**: 与容器形状、液体体积无关
- **浮力方向**: 始终竖直向上; F浮 = G排液, 不等于液体总重
- **功的两要素**: 推墙不动 W=0; 水平拉车前进 → 竖直方向重力不做功
- **杠杆力臂**: 力臂是支点到力作用线的垂直距离, 不是支点到作用点的距离
- **机械效率永远 < 1**: 有额外功 (克服摩擦 + 提升动滑轮自重)
