## 推荐学习路径

### 阶段 1: 内能与热机 (第13-14章, 3 周)
- 分子动理论三要点 → 内能 ↔ 热量 ↔ 温度 概念辨析
- Q = cmΔt 计算: 比热容 c 单位 J/(kg·℃), 水的 c 最大 (4.2×10³)
- 热机四冲程: 吸气→压缩→做功(爆炸)→排气; 只有做功冲程对外做功
- 热机效率 = W有 / Q燃料 × 100%

### 阶段 2: 电路基础 (第15-16章, 3 周)
- 电荷: 同种相斥 / 异种相吸; 电量单位库 (C)
- I = Q/t; 电流方向: 正电荷定向移动方向
- 串联: I相等, U分配; 并联: U相等, I分配
- 电流表 (内阻小, 串联) vs 电压表 (内阻大, 并联) 的使用规范
- 电阻影响因素: 材料 / 长度 / 横截面积 / 温度; 滑动变阻器读数

### 阶段 3: 欧姆定律专题 (第17章, 3 周)
- I = U/R (同一导体, 同一温度)
- 串联等效电阻: R总 = R₁ + R₂; 并联等效电阻: 1/R总 = 1/R₁ + 1/R₂
- 伏安法测电阻: 电流表外接法 / 内接法选择（定性了解）
- 欧姆定律应用: 已知两量求第三量; 电路计算步骤

### 阶段 4: 电功率与安全用电 (第18-19章, 3 周)
- W = UIt = Pt; P = UI = I²R = U²/R
- 焦耳定律: Q = I²Rt (纯电阻电路 Q = W)
- 电能表读数: 度 (kW·h); 1 kW·h = 3.6×10⁶ J
- 家庭电路: 火线 (L) / 零线 (N) / 地线 (PE); 插座 "左零右火上地"
- 短路危害; 保险丝 / 空气开关的作用

### 阶段 5: 电磁学 (第20章, 3 周)
- 磁场方向: 小磁针 N 极指向 = 磁场方向
- 奥斯特实验: 电流→磁场; 右手定则判断螺线管磁极
- 电动机: 通电导体在磁场中受力 (左手定则 F, B, I 三指方向)
- 电磁感应: 导体切割磁感线 → 感应电流; 发电机原理 → 机械能转化为电能

### 阶段 6: 信息与能源 (第21-22章, 2 周)
- 电磁波: c = λf = 3×10⁸ m/s; 光、无线电波、X射线同属电磁波
- 能量守恒定律: 能量不灭只转化; 热值 q (J/kg)
- 核能: 裂变 (核电站) vs 聚变 (太阳、氢弹)

总课时约 17 周 ≈ 1 学期 (每周 3 课时).

## 重点知识点

### 核心公式 (含 SI 单位)

| 公式                      | 含义       | 单位                              |
| ----------------------- | -------- | ------------------------------- |
| Q = cmΔt                | 热量       | J (c: J/(kg·℃), m: kg, Δt: ℃)  |
| I = U / R               | 欧姆定律     | A (U: V, R: Ω)                  |
| R串 = R₁ + R₂            | 串联电阻     | Ω                               |
| 1/R并 = 1/R₁ + 1/R₂      | 并联电阻     | Ω                               |
| W = UIt                 | 电功       | J (U: V, I: A, t: s)            |
| P = UI = I²R = U²/R     | 电功率      | W                               |
| Q = I²Rt                | 焦耳热 (电热) | J                               |
| c = λf                  | 电磁波      | m/s (λ: m, f: Hz)               |

### 典型实验

- **探究比热容**: 相同质量水和沙子吸收相同热量; 温度计测温升; 比较 c 大小
- **伏安法测电阻**: 滑动变阻器调节 U; 画 U-I 图线; 斜率倒数 = R; 注意安培表接法
- **探究串并联电路电流电压规律**: 多用电流表 / 电压表; 串联 I 处处相等; 并联 U 相等
- **探究电功率与电阻关系 (焦耳定律)**: 相同电流下 R 越大发热越多; 控制变量
- **探究电磁感应条件**: 导体切割磁感线 → 电流计偏转; 速度方向 / 磁场方向影响电流方向

## 易错点 / 难点

- **内能 vs 热量 vs 温度**: 温度高不一定内能大 (需考虑质量/物质); 热量是过程量
- **欧姆定律适用条件**: 纯电阻 / 温度不变; 灯丝电阻随温度变化 (非线性)
- **串并联判断**: 逐段分析电流路径; 并联支路电压必须相同
- **电功率额定 vs 实际**: P实 = U实²/R; 只有 U = U额定 时 P实 = P额定
- **左手定则 vs 右手定则**: 左手 → 受力 (电动机); 右手 → 感应电流方向 (发电机)
- **焦耳定律 Q = I²Rt**: 并联电路中并联支路 U 相同, 应用 Q = U²t/R 更简便
- **电磁波频率与波长成反比**: f 越高, λ 越短 (c 恒定)
