## 推荐学习路径

### 阶段 1: 认识生命与生物圈 (第一单元, 2 周)
- 生物的基本特征 → 区别生物与非生物
- 生态因素（非生物因素 + 生物因素）→ [[生态因素]]
- 生态系统组成 → 食物链与食物网 → [[食物链]] [[食物网]]
- 生物圈概念 → 人类与生物圈的关系

### 阶段 2: 细胞——生命的基本单位 (第二单元 第一章, 3 周)
- 显微镜的结构与使用规范 → 典型实验：制作临时装片
- 植物细胞结构（细胞壁/细胞膜/细胞质/细胞核/液泡/叶绿体）→ [[植物细胞]]
- 动物细胞结构（无细胞壁/液泡/叶绿体）→ [[动物细胞]]
- 细胞膜的功能 → 物质进出细胞
- 细胞核与遗传信息 → DNA 初步
- 细胞分裂 → 染色体/DNA/蛋白质

### 阶段 3: 从细胞到生物体 (第二单元 第二章, 1 周)
- 细胞分化 → 组织（上皮/肌肉/神经/结缔）
- 植物组织 vs 动物组织 → [[组织]]
- 器官 → 系统（人体八大系统预览）→ 个体
- 结构层次：细胞→组织→器官→系统→个体 → [[生命的结构层次]]

### 阶段 4: 绿色植物的多样性与特征 (第三单元 第一章, 1 周)
- 藻类（无根茎叶，水生）→ 苔藓（无维管束）→ 蕨类（有维管束，无种子）
- 裸子植物 vs 被子植物 → [[种子植物]]
- 分类依据：形态结构特征

### 阶段 5: 被子植物的一生 (第三单元 第二章, 3 周)
- 种子的结构与萌发条件（温度/水分/空气）→ [[种子萌发]]
- 根的形态与吸水（根毛细胞，渗透吸水）→ [[根]]
- 茎的结构与运输（导管/筛管）→ [[茎]]
- 花的结构 → 传粉 → 受精 → 果实与种子形成 → [[花的结构]]
- 典型实验：茎的运输（红色染液）

### 阶段 6: 光合作用与呼吸作用 (第三单元 第三—五章, 2 周)
- 蒸腾作用 → 气孔的开闭
- 光合作用：原料（CO₂+H₂O）、产物（有机物+O₂）、场所（叶绿体）、条件（光）
- 典型实验：碘液检测叶片淀粉（光合作用证明）
- 呼吸作用：有机物分解，释放能量（CO₂+H₂O）
- 光合 vs 呼吸对比 → 碳-氧平衡 → [[光合作用]] [[呼吸作用]]

总课时约 18 周 ≈ 1 学期。

## 重点知识点

### 系统 / 器官分类
- 生命的结构层次（初中生物核心框架）：细胞 → 组织 → 器官 → 系统 → 个体
- 植物四大组织：保护组织、营养组织、输导组织、分生组织
- 动物四大组织：上皮组织、肌肉组织、神经组织、结缔组织
- 植物器官：根、茎、叶（营养）+ 花、果实、种子（生殖）

### 典型实验
| 实验 | 材料 | 目的 |
|------|------|------|
| 显微镜观察洋葱表皮细胞 | 洋葱、碘液 | 认识植物细胞结构 |
| 观察人体口腔上皮细胞 | 棉签、亚甲基蓝 | 认识动物细胞结构、对比植物细胞 |
| 种子萌发实验 | 大豆/玉米种子 | 探究萌发条件（对照实验） |
| 茎的运输 | 芹菜茎、红色染液 | 观察导管位置 |
| 光合作用实验（碘液法） | 叶片、酒精脱色、碘液 | 证明光合产物为淀粉 |
| 蒸腾作用实验 | 植物枝条、塑料袋 | 测量蒸腾失水量 |

## 易错点 / 难点

- **细胞结构对比**：植物细胞有细胞壁、液泡、叶绿体；动物细胞均无
- **光合 vs 呼吸**：二者同时进行；光补偿点概念（待家长核对具体考纲要求）
- **导管 vs 筛管**：导管输水和无机盐（向上）；筛管输有机物（向下）
- **种子萌发**：内因（活种子）+ 外因（温度/水分/空气），光照不是必要条件
- **食物链书写**：从生产者写起，箭头指向消费者，不写分解者
