## 推荐学习路径

### 阶段 1: 人类的起源 (第一章, 1 周)
- 现代类人猿 vs 人类 → 化石证据
- 直立行走 → 手的解放 → 劳动 → 语言 → 脑的发达
- 人类种族与平等 → [[056-人类起源|人类起源]]

### 阶段 2: 消化系统与营养 (第二章, 3 周)
- 六大营养物质：糖类/蛋白质/脂肪/维生素/无机盐/水 → [[065-营养物质|营养物质]]
- 消化系统组成：口腔→食道→胃→小肠→大肠→肛门 + 肝脏/胰腺
- 消化酶的作用：唾液淀粉酶（淀粉→麦芽糖）、胃蛋白酶（蛋白质）、肠液/胰液
- 小肠适于吸收的结构特点：长、皱褶、绒毛、毛细血管 → [[小肠]] [[060-消化|消化]]
- 典型实验：唾液淀粉酶对淀粉的消化（碘液对照）
- 合理膳食 → 食物中各营养素的功能 → [[131-维生素|维生素]]

### 阶段 3: 呼吸系统 (第三章, 2 周)
- 呼吸系统组成：鼻→咽→喉→气管→支气管→肺
- 肺泡结构特点：多、壁薄（一层细胞）、外有毛细血管 → [[肺泡]]
- 肺通气：吸气（膈肌收缩下降，肋骨上升，胸腔扩大）
- 气体交换：肺泡—毛细血管（O₂进/CO₂出）；组织—毛细血管（O₂出/CO₂进）
- 呼吸全过程 → [[051-呼吸系统|呼吸系统]] [[气体交换]]

### 阶段 4: 循环系统 (第四章, 3 周)
- 血液成分：血浆 + 血细胞（红细胞/白细胞/血小板）
- 红细胞：含血红蛋白，携带O₂；白细胞：吞噬细菌（免疫）；血小板：凝血
- 心脏四腔：左心室（壁最厚）/右心室/左心房/右心房
- 瓣膜：防止血液倒流（房室瓣/动脉瓣）
- 体循环：左心室→主动脉→全身→上下腔静脉→右心房
- 肺循环：右心室→肺动脉→肺泡毛细血管→肺静脉→左心房
- 动脉/静脉/毛细血管特点对比 → [[096-心脏|心脏]] [[102-血液循环|血液循环]] [[101-血液|血液]]
- 血压与脉搏；ABO 血型（待家长核对具体考纲范围）

### 阶段 5: 泌尿系统与排泄 (第五章, 1 周)
- 泌尿系统：肾脏（肾单位）→ 输尿管 → 膀胱 → 尿道
- 肾单位：肾小球 + 肾小囊 + 肾小管
- 尿的形成：过滤（肾小球→肾小囊）+ 重吸收（肾小管）
- 排泄三途径：泌尿（尿）/ 呼吸（CO₂+水蒸气）/ 皮肤（汗）→ [[082-肾脏|肾脏]] [[080-泌尿系统|泌尿系统]]

### 阶段 6: 神经调节与激素调节 (第六章, 3 周)
- 神经系统：中枢（脑+脊髓）+ 周围（脑神经+脊神经）
- 神经元（神经细胞）结构：细胞体 + 树突 + 轴突
- 反射与反射弧：感受器→传入神经→神经中枢→传出神经→效应器
- 非条件反射（简单/遗传）vs 条件反射（复杂/后天）→ [[085-反射弧（初中）|反射弧]] [[条件反射]]
- 脑的功能：大脑（高级）/ 小脑（协调）/ 脑干（基本生命活动）
- 激素调节：胰岛素（血糖）/ 甲状腺激素（代谢）/ 生长激素
- 神经—激素协调调节 → [[087-激素（初中）|激素]] [[血糖调节]]（待家长核对考纲深度）

### 阶段 7: 人与生物圈 (第七章, 1 周)
- 环境污染类型：大气/水/土壤污染
- 外来入侵物种危害
- 保护生物圈：减少污染、保护生物多样性

总课时约 18 周 ≈ 1 学期。

## 重点知识点

### 系统 / 器官分类
人体八大系统（七下重点涉及四个）：
- 消化系统：口腔、食道、胃、小肠、大肠、肝脏（消化腺）
- 呼吸系统：鼻、气管、肺（肺泡为气体交换场所）
- 循环系统：心脏、血管（动脉/静脉/毛细血管）、血液
- 泌尿系统：肾脏（肾单位）、输尿管、膀胱、尿道
- 神经系统：脑（大脑/小脑/脑干）、脊髓、神经

### 典型实验
| 实验 | 材料 | 目的 |
|------|------|------|
| 唾液淀粉酶消化实验 | 唾液、淀粉溶液、碘液、37°C水浴 | 探究消化酶的作用条件（温度/pH）|
| 观察血液涂片 | 血液涂片标本、显微镜 | 识别红细胞/白细胞/血小板 |
| 模拟膈肌运动 | 气球、注射器、橡皮膜 | 理解呼吸运动原理 |
| 测量脉搏/心率 | 秒表 | 了解心跳与脉搏关系 |
| 膝跳反射 | 叩诊锤（或手指弹膝）| 体验非条件反射与反射弧 |

## 易错点 / 难点

- **动脉 vs 静脉**：动脉血（含O₂多，鲜红）≠ 动脉（血管类型）；肺静脉流动脉血
- **体/肺循环方向**：左心室出，右心房回（体）；右心室出，左心房回（肺）
- **肾小球滤过**：大分子蛋白质和血细胞不能滤过；葡萄糖滤过后全部重吸收
- **反射弧缺一不可**：任一环节受损，反射弧不完整，反射不能完成
- **消化 vs 吸收**：消化在消化道；吸收主要在小肠（毛细血管/毛细淋巴管）
