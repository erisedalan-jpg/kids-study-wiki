## 推荐学习路径

### 阶段 1: 负数 (ch1, 1 周)
- 负数的引入(温度/海拔/收支) → (缺口: 负数)
- 数轴; 正负数大小比较(-3 < -1 < 0 < 1)
- 正负数的绝对值直觉

### 阶段 2: 百分数应用 ⭐ (ch2, 2 周)
- 折扣/成数; 税率应用题 → (缺口: 百分数应用题)
- 利率/本金/利息计算: 利息=本金×利率×时间
- 增长率/折扣率综合题

### 阶段 3: 圆柱与圆锥 ⭐难关 (ch3, 2.5 周)
- 圆柱侧面展开=长方形; 侧面积=底面周长×高 → [[圆柱]]
- 圆柱表面积=侧面积+2个底面积
- 圆柱体积=底面积×高=πr²h → (缺口: 圆柱体积)
- 圆锥体积=1/3×底面积×高=1/3πr²h → (缺口: 圆锥)
- 等底等高圆柱体积=3×圆锥体积

### 阶段 4: 比例 ⭐ (ch4, 3 周)
- 比例的意义; 比例的基本性质(外项积=内项积) → (缺口: 比例)
- 正比例: y/x=k(常数); 反比例: xy=k
- 比例尺; 图上距离与实际距离换算 → (缺口: 比例尺)
- 用比例解方程(正反比例判断与列式)

### 阶段 5: 鸽巢问题 (ch5, 0.5 周)
- 鸽巢原理: n+1个物品放n个抽屉必有一屉≥2 → (缺口: 鸽巢原理)

### 阶段 6: 整理复习 (ch6, 4 周)
- 数与代数/图形与几何/统计与概率/综合实践 全面回顾

总课时约 14 周 ≈ 1 学期 (含大量复习).

## 重点知识点全单 (六下必掌握 / 小升初关键)

1. **负数**: 数轴; 正负数比较; 绝对值概念
2. **百分数应用**: 利息/税率/折扣综合实题
3. **圆柱**: 侧面积=2πrh; 表面积; 体积=πr²h
4. **圆锥**: 体积=1/3πr²h; 与圆柱关系
5. **比例**: 正比例/反比例判断; 比例尺换算; 用比例解问题
6. **全面复习**: 整数/小数/分数/百分数运算; 几何全面梳理

## 易错点 / 难点

- **圆锥体积**: 乘以1/3; 最常丢; 等底等高圆柱体积=3倍圆锥
- **正反比例判断**: 正比例 y/x=常数; 反比例 xy=常数; 题目要看"哪两量"
- **比例尺**: 图上÷比例尺=实际; 注意单位统一
- **负数大小**: 数轴越右越大; -10 < -1; 不是绝对值大就大
