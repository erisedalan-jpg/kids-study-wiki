## 推荐学习路径

### 阶段 1: 天气观测 (单元1, 3周)
- 气温计读数: 背阴处1.5m高 → [[035-温度|温度]] [[015-天气|天气]]
- 风向: 用风向标测; 风力0-12级 → [[风]] (新)
- 云量估算; 降雨量用雨量器测 → [[雨]] (新)
- 天气 (每日变化) vs 气候 (多年平均) → [[015-天气|天气]] [[气候]] (新)

### 阶段 2: 蚕的一生 (单元2, 持续全期)
- 蚕卵→蚕幼虫→蛹→蚕蛾 (完全变态) → [[蚕]] (新)
- 蚕的食物 (桑叶); 吐丝结茧; 丝绸来源 → [[蚕丝]] (新)
- 长期观察记录 (贯穿本学期)

### 阶段 3: 生物与环境 (单元3, 2周)
- 生物特征 (生长/呼吸/繁殖/对刺激反应) → [[021-生物|生物]] (新)
- 仙人掌 (旱) / 睡莲 (水) / 松树 (寒) 适应性对比
- 北极熊 (白色保护色) / 骆驼 (储水) 适应性 → [[适应]] (新)

### 阶段 4: 天文基础 (单元4, 2周)
- 地球球形证据 → [[103-只有一个地球|地球]]
- 太阳直径约为月球的400倍 ⚠️待家长核对
- 月相: 新月→上弦月→满月→下弦月 → [[月相]]

### 阶段 5: 影子与时间 (单元5+6, 2周)
- 影子在光源对面; 光源高影子短 → [[005-影子|影子]]
- 日晷原理: 影子长短/方向判断时间 → [[日晷]] (新)
- 制作日晷 (工程实践)

总课时约 12 周 ≈ 1 学期.

## 重点知识点

- **物质科学**: 本册无专门单元
- **生命科学**: 蚕的完全变态 (卵→幼→蛹→成虫); 生物/非生物区分; 生物适应环境
- **地球与宇宙**: 天气观测四要素 (气温/风/云/降水); 月相变化; 影子规律; 日晷计时
- **技术与工程**: 日晷设计制作; 雨量器制作

## 易错点 / 难点

- **天气 vs 气候**: "今天下雨"是天气; "武汉夏季多雨"是气候; 四年级重点区分
- **月相方向**: 上弦月右侧亮, 下弦月左侧亮 (北半球视角); 常混淆
- **完全变态 vs 不完全变态**: 蚕/蝴蝶为完全变态 (有蛹期); 蚱蜢为不完全变态

## 学习难度说明

- 整体定位: **中等**, 适合9-10岁
- 养蚕是长期观察项目 (需贯穿整学期); 需在课前备蚕卵
- 天文单元与四上月相内容衔接, 注意不重复而是深化
