## 推荐学习路径

### 阶段 1: 造型表现 — 色彩与构图进阶 (课 1/3/4/6/7/8, 约 4 周)
- 魔幻的颜色 (课 1) → 三原色 + 间色调配实验；吴冠中/米罗作品
- 大人国与小人国 (课 3) → 大小比例夸张; 透视初步
- 前前后后 (课 4) → 前后遮挡构图规律; 米勒《拾穗者》欣赏
- 黑与白 (课 6) → 对比色极值; 埃舍尔版画欣赏
- 快乐的回忆 (课 7) → 叙事人物场景; 情感表达
- 星空的联想 (课 8) → 刮蜡技法 (蜡笔底色+墨+刮)

### 阶段 2: 设计应用 (课 2/11/12/13/15/16, 约 3 周)
- 万花筒 (课 2) → 对称图案; 旋转对称概念
- 各式各样的鞋 (课 11) → 功能性产品装饰设计
- 团包纸工 (课 12) → 纸工立体造型; 包裹/捏塑
- 化平凡为神奇 (课 13) → 废旧材料创意再造
- 我设计的自行车/新颖的电脑 (课 15-16) → 工业产品创意改造

### 阶段 3: 欣赏评述 (课 20, 约 1 周)
- 迷人的动画片 (课 20) → 动画原理 (帧/关键帧); 动画欣赏

### 阶段 4: 综合探索 (课 5/17/19, 约 2 周)
- 登山游戏 (课 5) → 地图设计 + 棋子制作 + 玩游戏
- 会响的玩具 (课 17) → 声音 + 美术综合探索
- 美术小记者 (课 19) → 社会调查 + 图文报道

## 重点知识点

- **造型表现**: 三原色/间色系统; 大小比例对比; 前后遮挡构图; 刮蜡肌理技法; 黑白对比
- **设计应用**: 旋转对称纹样; 产品功能性与装饰性统一; 废旧材料再利用
- **欣赏评述**: 埃舍尔 (黑与白对比); 米勒《拾穗者》; 动画制作原理
- **综合探索**: 游戏设计 + 制作 + 规则; 声音与造型跨界; 新闻采访图文

## 易错点 / 难点

- **三原色混色**: 三原色等量混合变黑/灰，需控制比例
- **前后遮挡构图**: 低年级习惯将物体并排，需引导遮挡练习
- **刮蜡技法**: 墨层需完全干透才能刮，急于刮会糊
- **万花筒对称**: 六方对称较难，需先画好基本单元再旋转复制
