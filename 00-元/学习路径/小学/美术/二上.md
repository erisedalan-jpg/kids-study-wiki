## 推荐学习路径

### 阶段 1: 造型表现 — 色彩实验 (课 1-2/7-8, 约 3 周)
- 流动的颜色 (课 1) → 水彩滴吹体验；三原色相融变化
- 染色游戏 (课 2) → 浸染法；折叠产生对称纹样
- 对折剪纸 (课 7) → 对称形概念 + 剪纸步骤
- 对印的图形 (课 8) → 对称/添画联想

### 阶段 2: 设计应用 (课 3/5/6/12/17/18, 约 3 周)
- 装饰名字 (课 3) → 文字变形/美术字初步
- 奇妙的电话机 (课 5) → 功能性创意设计；欣赏达利装置艺术
- 花花衣 (课 6) → 物品装饰；马勺脸谱欣赏
- 巧用瓶盖 (课 12) → 废旧材料再利用拓印
- 会转的玩具 (课 17) → 旋转结构理解

### 阶段 3: 欣赏评述 (课 20, 穿插)
- 丰富多彩的玩具 (课 20) → 玩具美感欣赏; 比较与描述

### 阶段 4: 综合探索 (课 11/13/14/19, 约 2 周)
- 儿歌变画 (课 11) → 语言转图像
- 回家的路 (课 13) → 地图思维/空间认知
- 做一道拿手"菜" (课 14) → 食物造型与生活美学
- 指偶剧场 (课 19) → 综合制作 + 表演

## 重点知识点

- **造型表现**: 三原色 (红黄蓝) 相融实验；对称概念 (对折/对印)；线条表现动感 (风来了)
- **设计应用**: 美术字装饰；生活物品创意改造；废旧材料利用；旋转玩具结构
- **欣赏评述**: 玩具艺术欣赏；达利装置艺术 🚧 (龙虾电话机简介)；马勺脸谱民间工艺
- **综合探索**: 诗歌图像转换；地图空间认知；食物造型设计；指偶表演综合

## 易错点 / 难点

- **染色浸染**: 纸折叠不均会导致纹样不对称；纸质需选吸水性强的
- **对折剪纸**: 剪前需在折叠面画图，不能直接剪；图形需接边留连
- **三原色混色**: 儿童常将颜料调成灰黑色，需控制水量与混色比例
- **旋转玩具**: 重心不均导致不能旋转，需反复调整
