## 推荐学习路径

### 阶段 1: 造型表现 — 抽象与构图 (课 2/5/7, 约 3 周)
- 抽象的画 (课 2) → 抽象艺术概念; 色彩与形式构成 (康定斯基等 🚧)
- 中国画的构图 (课 5) → 中国画S形/三角形/疏密构图原则; 留白经营
- 色彩的联想 (课 7) → 色彩与情感/季节/音乐的联觉关系

### 阶段 2: 造型表现 — 人物综合 (课 4, 约 1 周)
- 画人物 (课 4) → 速写动态; 六年级人物比例综合应用

### 阶段 3: 欣赏评述 — 中西名作毕业总结 (课 11/12/13, 约 3 周)
- 设计生活 (课 11) → 工业设计欣赏; 功能美与形式美统一
- 优秀的中国美术作品 (课 12) → 六年小学中国画名作系统梳理
- 外国的优秀美术作品 (课 13) → 西方美术史代表作品综合欣赏

### 阶段 4: 设计应用 (课 6, 约 1 周)
- 电脑美术 (课 6) → 数字图像合成; 图层概念初步

### 阶段 5: 综合探索 (课 1/3/8/9/10/14/15, 约 3 周)
- 面具 (课 3) → 世界各地面具文化; 综合材料制作
- 我们的社区/鸟的天地 (课 8-9) → 社会与自然调查综合
- 我的成长故事 (课 10) → 绘本叙事小学阶段总结
- 毕业纪念/我们的梦想 (课 14-15) → 综合创作 + 小学阶段收尾

## 重点知识点

- **造型表现**: 抽象艺术基本语言 (色彩/形状/线条的纯粹表达); 中国画构图原则; 速写动态线
- **设计应用**: 工业设计功能与美观平衡; 数字图像合成基础
- **欣赏评述**: 小学阶段中西美术史名作总梳理; 工业设计经典案例
- **综合探索**: 世界面具文化比较; 绘本创作叙事结构; 毕业综合创作

## 易错点 / 难点

- **抽象画**: 学生常误以为"乱画就是抽象"，需引导理解形式与情感的关系
- **中国画构图留白**: 六年级要求留白有意图性 (呼应/平衡)，不是简单留空
- **速写动态**: 速写需抓主要动势线，细节可省略，与写实画区分
