## 推荐学习路径

按 6 阶段 (一学期约 6 单元 + 2 复习):

### 阶段 1: 教室与方位 (Unit 1, 2 周)
- classroom / window / door / picture / computer / fish bowl → [[教室词汇]]缺口
- Where is...? / It's near/in front of/behind the... — 方位表达扩展

### 阶段 2: 文具 + 颜色复习 (Unit 2, 2 周)
- English book / maths book / storybook / notebook / ruler / eraser → [[教室词汇]]缺口
- What colour is it? / It's blue and white. — 颜色复习 + 双色描述

### 阶段 3: 外貌描述 + 朋友 (Unit 3, 2 周)
- tall / strong / quiet / cute / glasses — 外貌形容词 → [[大小词]] 扩展
- What's his/her name? / He/She is... + 形容词; He has...

### 阶段 4: Recycle 1 + 房间 (Unit 4, 2 周)
- living room / bedroom / bathroom / kitchen / study → [[家居词汇]]缺口
- Is she in the...? / Yes, she is. / No, she isn't. — be 动词一般疑问句

### 阶段 5: 食物 + 餐桌用语 (Unit 5, 2 周)
- tomato / tofu / green beans / soup / rice / chicken → [[食物-主食]] 扩展
- Have some...! / I'm hungry/thirsty. / Yummy!

### 阶段 6: 家庭 + 数量 (Unit 6, 2 周)
- grandfather / grandmother / aunt / uncle / cousin → [[家庭成员]] 扩展
- How many people? / There are four. / Who's...?
- Recycle 2 综合复习

## 重点知识点

- **词汇**: 教室物品 (8件), 书包文具, 房间名称 (5间), 外貌形容词 (高矮胖瘦), 家居食物
- **句型**: Where is/are...? / It's in/on/near... / What's in...? / Is he/she in...? / How many...?
- **语法点**: be 动词一般疑问句 Is...? → Yes, he is. / No, she isn't.; 所有格代词 his/her; There are... 初现
- **语音/拼读**: 四上引入 Let's spell — 元音字母 Aa/Ee/Ii/Oo/Uu 字母音 (单词中间短元音)

## 易错点 / 难点

- **his/her 区分**: 与三下 he/she 对应, 但 his/her 作定语修饰名词, 易与 he/she 混淆
- **Is she in...? — Yes, she is.**: 肯定回答不能省略主语, "Yes, is." 是错误的
- **There is/are**: 单复数一致性; There is a book. / There are two books. — 本册初现, 五上深化
- **Let's spell 元音**: a/e/i/o/u 字母音与字母名称不同 (Aa 名称 /eɪ/ vs 音 /æ/)
