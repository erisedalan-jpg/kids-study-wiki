## 推荐学习路径

按 6 阶段 (一学期约 6 单元 + 2 复习):

### 阶段 1: 学校场所 + 楼层 (Unit 1, 2 周)
- library / art room / computer room / teachers' office / playground → [[学校场所]]缺口
- Where's the...? / It's on the first/second floor. / next to / near

### 阶段 2: 时间表达 (Unit 2, 2 周)
- o'clock / half past / a.m. / p.m. — 时间读法 → [[时间词汇]]缺口
- What time is it? / It's 7 o'clock. / It's time to get up/eat/go to school.

### 阶段 3: 天气 (Unit 3, 2 周)
- sunny / cloudy / windy / rainy / snowy / cold / hot / warm / cool → [[天气]]
- What's the weather like? / Can I go outside? / Yes, you can. / No, you can't.

### 阶段 4: Recycle 1 + 农场动物 (Unit 4, 2 周)
- cow / horse / sheep / pig / hen / goat — 农场动物 → [[动物-家养]] 扩展
- Are these/those...? / Yes, they are. / No, they aren't. — 复数 be 动词疑问句

### 阶段 5: 衣物 (Unit 5, 2 周)
- T-shirt / jacket / dress / skirt / jeans / shoes / hat / coat → [[衣物词汇]]缺口
- What are you wearing? / I'm wearing... / It's too big. — 现在进行时初现

### 阶段 6: 购物 + 价格 (Unit 6, 2 周)
- How much is it/are they? / It's 20 yuan. / I'll take it. → [[数字]] 扩展 (大数)
- Recycle 2 综合复习

## 重点知识点

- **词汇**: 学校场所 (5处), 时间词 (钟点/半点), 天气词 (8种), 衣物 (8件), 农场动物 (牛马羊猪鸡山羊)
- **句型**: Where's the...? / What time is it? / What's the weather like? / Are these...? / How much is it?
- **语法点**: 复数 be 动词 Are...? / They are/aren't; 情态动词 can (能力与许可); 现在进行时 I'm wearing... 初步引入
- **语音/拼读**: Let's spell — 辅音字母组合 bl/cl/fl/gl/pl; 字母组合音入门

## 易错点 / 难点

- **时间表达**: "half past 6" vs "six thirty" 两种说法; It's time to + 动词原形 (不加 to do)
- **Are these...? — Yes, they are.**: 复数 they 回答, 不能用 it
- **can 的双重用法**: can 表能力 (I can swim.) 与 can 表许可 (Can I go out?) 在本册同时出现
- **How much**: 不可数名词和可数名词均可用 How much, 与 How many 区别须后续巩固

## 学习建议

四下引入现在进行时 (I'm wearing...) 和 can 的许可用法, 是语法快速扩展期. 天气主题是本册亮点, 建议结合实际天气做日记式口头练习 "Today is sunny. I can go outside.". 时间表达 (Unit 2) 与五上课表衔接紧密, 务必熟练.
