## 推荐学习路径

按 6 阶段 (一学期约 6 单元 + 2 复习):

### 阶段 1: 性格描述 (Unit 1, 2 周)
- young / old / kind / clever / hard-working / polite / quiet / funny → [[性格形容词]]缺口
- What's he/she like? / He's/She's + 形容词 — 第三人称单数 be 动词

### 阶段 2: 星期 + 课表 (Unit 2, 2 周)
- Monday–Sunday → [[星期词汇]]缺口
- 学科: maths / English / PE / science / music / art → [[学科名称]]缺口
- What do you have on Fridays? / I have... — 一般现在时第三人称复数

### 阶段 3: 食物 + 喜好深化 (Unit 3, 2 周)
- sandwich / salad / noodles / soup / hot dog → [[食物-主食]] 扩展
- What's your favourite food? / My favourite food is... / They're delicious.

### 阶段 4: Recycle 1 + 能力 (Unit 4, 2 周)
- sing / dance / draw / play football/chess — 动作词 → [[动作词-基本]] 扩展
- What can you do? / I can... / Can you...? — can 表能力系统梳理

### 阶段 5: There is/are 结构 (Unit 5, 2 周)
- clock / mirror / end table / air-conditioner / curtain — 家居 → [[家居词汇]]缺口
- There is a... / There are some... / Is there...? / Are there...?

### 阶段 6: 自然景观 (Unit 6, 2 周)
- forest / river / lake / mountain / path / bridge → [[自然地貌]]缺口
- Is there a river in the forest? / Yes, there is. / No, there isn't.
- Recycle 2 综合复习

## 重点知识点

- **词汇**: 性格形容词 (8个), 星期 (7天), 学科名 (6门), 自然地貌 (6词), 家居物品 (镜钟窗帘等)
- **句型**: What's he/she like? / What do you have on...? / What would you like? / What can you do? / There is/are...
- **语法点**: 一般现在时第三人称单数 (he/she has/does); There is/are 存在句; can 能力疑问; 情态动词 would
- **语音/拼读**: Let's spell — 元音字母组合 ea/ee (meat/feet); 谚语 (Proverbs) 首次出现

## 易错点 / 难点

- **第三人称单数**: He has (not have); She does (not do); 动词三单变化规律须重点练习
- **There is/are 选择**: There is + 单数/不可数; There are + 复数 — 就近原则在六年级深化
- **What's he like? vs What does he like?**: 前者问性格外貌, 后者问爱好 — 极易混淆 ⭐
- **would like vs like**: I'd like... (此刻想要) vs I like... (一般喜好) — 语境判断

## 学习建议

五上是语法密集期: 一节课可能同时遇到第三人称单数、There is/are、情态动词三项. 建议优先攻克 "What's he like? vs What does he like?" 这对高频混淆对, 并用课表 (Unit 2) 做每周真实场景输出练习.
