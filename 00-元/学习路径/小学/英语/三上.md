## 推荐学习路径

按 6 阶段 (一学期约 6 单元 + 2 复习):

### 阶段 1: 问候与自我介绍 (Unit 1, 2 周)
- Hello / Hi / Goodbye → [[问候语]]
- What's your name? / My name's... → [[问候语]]
- 字母 Aa–Ff, 数字 1–5

### 阶段 2: 颜色 (Unit 2, 2 周)
- red / blue / yellow / green / orange / purple / white / black → [[颜色-基本]]
- What colour is it? / It's...

### 阶段 3: 身体部位 + 问候语扩展 (Unit 3, 2 周)
- face / nose / mouth / ear / eye / hand → [[身体部位]]
- Good morning / Good afternoon → [[问候语]]
- Let's + 动词 结构入门

### 阶段 4: Recycle 1 + 动物 (Unit 4, 2 周)
- dog / cat / duck / panda / rabbit / fish → [[动物-家养]]
- I have a... / I like...

### 阶段 5: 食物 (Unit 5, 2 周)
- bread / milk / egg / rice / fish / juice → [[食物-主食]] [[食物-水果]]
- I'd like... / Do you like...?

### 阶段 6: 数字 + 生日 (Unit 6, 2 周)
- one–ten → [[数字 1-10]]
- How old are you? / I'm six. / Happy birthday!
- Recycle 2 综合复习

## 重点知识点

- **词汇**: 颜色 (8色), 动物 (狗猫鸭熊猫兔鱼), 食物 (面包牛奶鸡蛋米饭), 数字 1–10
- **句型**: Hello, I'm... / What's your name? / How are you? / I'm... / I'd like... / How old are you?
- **语法点**: 三上为入门级, 无明确语法; be 动词 I'm / It's 初步感知; 祈使句 Let's / Look at...
- **语音/拼读**: 26 个字母认读 (Aa–Zz); 字母歌; 本册不含 Let's spell (三下起引入)

## 易错点 / 难点

- **My name's vs. I'm**: 同义但结构不同, 孩子常混用; 需反复操练两种均可
- **How are you? / Fine, thank you.**: 套语须整体记忆, 不能拆分理解
- **数字拼写**: one/two/three 容易写错, 结合数量感知多练
- **颜色形容词**: 英语颜色词直接修饰名词 (a red bag), 中文习惯"红色的包", 注意语序
