## 推荐学习路径

### 阶段 1: 感受与欣赏
- 感受民族风情 → 《新疆舞曲第二号》(丁善德曲，钢琴独奏；主导节奏型铿锵)
- 感受少数民族管弦乐 → 《北京喜讯到边寨》(郑路/马洪业曲；引子+ABCDE五段)
- 感受欧洲古典 → 《小步舞曲》(莫扎特)、《蓝色多瑙河圆舞曲》(约翰·施特劳斯)
- 感受戏剧性音乐 → 《彼得与狼》(普罗科菲耶夫；各乐器角色分工)
- 感受序曲 → 《威廉·退尔》序曲片段 (罗西尼)

### 阶段 2: 表现 (歌唱/演奏)
- 演唱民族风格歌曲 → 《依马呀吉松》(四川巴塘藏族堆谐；衬词处理)
- 演唱 → 《三峡的孩子爱三峡》(魏岚词/王原平曲；领唱/齐唱设计)
- 欣赏演唱《上去高山望平川》(青海"花儿"，四度跳进音准)
- 演唱《卢沟谣》，感受历史题材
- 口风琴演奏 → G大调旋律巩固

### 阶段 3: 创造 (节奏创编)
- 用双响筒、沙槌、三角铁、铃鼓为《四季调》设计三段不同节拍伴奏
- 《大家一起来》歌表演动作编创
- 新年音乐会情景创编 (单元6 活动)

### 阶段 4: 音乐文化 (民歌/作曲家)
- 藏族堆谐 → 《依马呀吉松》(四川巴塘)；衬词文化
- 青海花儿 → 《上去高山望平川》(世界非物质文化遗产)；民间歌手朱仲禄 (1922-2007)
- 莫扎特 → 《小步舞曲》(小步舞曲体裁：三拍子，典雅风格)
- 丁善德 (1911-1995) → 《新疆舞曲第二号》(钢琴曲；作曲家/钢琴家/教育家)
- 普罗科菲耶夫 → 《彼得与狼》(管弦乐故事：长笛=鸟、双簧管=鸭、单簧管=猫、大管=爷爷、圆号=狼、弦乐=彼得)
- 约翰·施特劳斯 → 《蓝色多瑙河圆舞曲》《雷鸣电闪波尔卡》
- 罗西尼 → 《威廉·退尔》序曲

## 重点知识点

- **听辨**: 乐器音色与角色对应 (《彼得与狼》)；圆舞曲三拍子律动；波尔卡二拍子跳跃感；民族乐器与西洋乐器对比
- **歌唱**: 四度/大跳音程音准 (花儿)；衬词处理；视唱练习 (视唱小节)
- **乐理**: G大调 (升 F 音)；衬词概念；圆舞曲/波尔卡体裁；序曲概念
- **音乐文化**: 藏族堆谐舞；青海花儿 (世遗)；小步舞曲 (Menuet) 体裁；维也纳新年音乐会传统

## 易错点 / 难点

- **G大调升 F**: 每次遇 F 须升半音，易遗忘
- **花儿大跳音程**: 四度跳进音准需反复练习
- **彼得与狼乐器对应**: 7个角色对应7种乐器，记忆量大
- **圆舞曲强拍**: 三拍子中第一拍重拍需稳定感受
- **衬词演唱**: "依马呀吉松"等衬词发音需跟录音模仿
