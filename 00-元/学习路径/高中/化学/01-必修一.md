## 推荐学习路径

### 阶段 1: 物质分类与离子反应 (第一章前半, 2 周)
- 纯净物 / 混合物 → 单质 / 化合物 → 酸 / 碱 / 盐 / 氧化物 → [[纯净物]] [[混合物]] [[电解质]]
- 分散系：溶液 / 胶体 / 浊液；丁达尔效应鉴别胶体 → [[胶体]]
- 离子反应：电解质在溶液中的电离 → 写离子方程式三步（写→拆→删→查）→ [[离子方程式]]
- 强酸：HCl / H₂SO₄ / HNO₃；强碱：NaOH / KOH / Ba(OH)₂；难溶盐不拆

### 阶段 2: 氧化还原反应 (第一章后半, 2 周)
- 氧化还原反应本质：电子转移（化合价升高→被氧化→失电子；化合价降低→被还原→得电子）
- 氧化剂 / 还原剂 / 氧化产物 / 还原产物 → [[氧化还原反应]] [[氧化剂]] [[还原剂]]
- 双线桥标注法：标出得失电子数与化合价变化
- 常见氧化剂：KMnO₄、K₂Cr₂O₇、Cl₂、O₂、HNO₃（浓）、H₂SO₄（浓）

### 阶段 3: 钠及其化合物 (第二章前半, 2 周)
- Na 的物理性质（银白色、密度小、熔点低）和化学性质
- Na 与 O₂ 反应（常温 / 点燃）：Na₂O vs Na₂O₂ 的区别
- Na 与 H₂O 反应：浮、熔、游、响、红 → [[钠]]
- Na₂O₂ 作供氧剂：与 H₂O、CO₂ 反应 → [[过氧化钠]]
- NaOH、Na₂CO₃、NaHCO₃ 性质对比；Na₂CO₃ vs NaHCO₃ 鉴别

### 阶段 4: 氯及其化合物 (第二章后半, 2 周)
- Cl₂ 的实验室制备（MnO₂ + 浓 HCl 加热）与工业来源（氯碱工业）
- Cl₂ 的物理性质（黄绿色、有毒、密度大于空气）
- Cl₂ 与水反应：Cl₂ + H₂O ⇌ HCl + HClO；HClO 的漂白性
- Cl₂ 与碱反应：氯水、漂白粉、漂白液 → [[氯气]] [[次氯酸]]
- HCl 性质与盐酸的应用；Cl⁻ 的鉴定（AgNO₃ 法）

### 阶段 5: 铁铝铜 (第三章, 3 周)
- Na、Mg、Al、Fe 活动性比较；铝的钝化（浓硫酸/浓硝酸）→ [[铝]] [[铁]]
- Fe 的氧化物：FeO（黑）/ Fe₂O₃（红）/ Fe₃O₄（黑）
- Fe 的化合物转化：Fe²⁺ ↔ Fe³⁺；Fe(OH)₂（白）→ Fe(OH)₃（红褐）
- Al₂O₃ 的两性：既与酸反应又与碱反应 → [[两性氧化物]]
- Al(OH)₃ 的两性制备与应用；铝热反应
- Cu 的性质：与 O₂、稀硝酸、浓硝酸反应；铜与 Fe³⁺ 溶液反应

### 阶段 6: 非金属及其化合物 (第四章, 3 周)
- Si：自然界存在形式（SiO₂）；SiO₂ 的性质（与 HF、NaOH 反应）→ [[硅]]
- S：硫磺 → SO₂ → SO₃ → H₂SO₄（接触法工业制硫酸）→ [[硫酸]] [[二氧化硫]]
- 浓 H₂SO₄ 三大特性：吸水性、脱水性、强氧化性
- N₂ 的稳定性；NO / NO₂ 的性质与转化；HNO₃ 的强氧化性
- NH₃ 的制备（工业合成氨：N₂ + 3H₂ ⇌ 2NH₃）；NH₃ 的喷泉实验 → [[氨气]]
- 铵盐的热不稳定性；NH₄⁺ 的检验（NaOH 加热，湿润红色石蕊试纸变蓝）

总课时约 15-18 周 ≈ 1 学期.

| 3 | Na 与 H₂O 反应 | 2Na + 2H₂O = 2NaOH + H₂↑ |
| 4 | Na₂O₂ 与 H₂O 反应 | 2Na₂O₂ + 2H₂O = 4NaOH + O₂↑ |
| 5 | Na₂O₂ 与 CO₂ 反应 | 2Na₂O₂ + 2CO₂ = 2Na₂CO₃ + O₂ |
| 6 | Cl₂ 实验室制备 | MnO₂ + 4HCl(浓) $\overset{\Delta}{=}$ MnCl₂ + Cl₂↑ + 2H₂O |
| 7 | Cl₂ 与 H₂O 反应 | Cl₂ + H₂O ⇌ HCl + HClO |
| 8 | Cl₂ 与 NaOH 反应 | Cl₂ + 2NaOH = NaCl + NaClO + H₂O |

| 13 | Al(OH)₃ 与 NaOH 反应 | Al(OH)₃ + NaOH = NaAlO₂ + 2H₂O |
| 14 | 铝热反应 | 2Al + Fe₂O₃ $\overset{高温}{=}$ Al₂O₃ + 2Fe |
| 15 | Cu 与浓 HNO₃ 反应 | Cu + 4HNO₃(浓) = Cu(NO₃)₂ + 2NO₂↑ + 2H₂O |

### 典型实验（核心，共 8 个）

| 实验名称 | 目的 | 关键操作 | 现象 | 安全注意 |
|---|---|---|---|---|
| Na 与 H₂O 反应 | 验证 Na 的化学活泼性 | 切一小块 Na 放入盛水的烧杯；滴入酚酞 | 浮、熔、游、响；溶液变红 | Na 有强腐蚀性，切勿用手触碰 |
| Na₂O₂ 与 H₂O/CO₂ 反应 | 验证 Na₂O₂ 的供氧功能 | 向 Na₂O₂ 中加水后检验生成气体 | 产生气体（带火星木条复燃） | Na₂O₂ 强氧化性，接触皮肤有灼伤危险 |
| Cl₂ 的实验室制备 | 制备并收集 Cl₂ 气体 | 圆底烧瓶中 MnO₂ + 浓 HCl 加热；向上排空气收集；多余 Cl₂ 通入 NaOH 吸收 | 黄绿色气体产生 | Cl₂ 剧毒，必须在通风橱中操作；尾气必须碱液吸收 |
| Fe(OH)₂ 制备与转化 | 观察 Fe²⁺→Fe³⁺ 转化 | NaOH 溶液用长导管插入 FeSO₄ 溶液底部注入 | 先白色 Fe(OH)₂，迅速变红褐色 Fe(OH)₃ | 注意制备时隔绝 O₂ |
| Al 与 NaOH 溶液反应 | 验证 Al 的两性 | Al 片加入 NaOH 溶液 | 产生大量气泡（H₂）；Al 溶解 | NaOH 具有强腐蚀性 |
| 铝热反应 | 演示铝的还原性 | 将 Al 粉与 Fe₂O₃ 粉末混合点燃（需用 Mg 引燃） | 剧烈燃烧，产生铁珠；高温（~2500°C） | 高温剧烈反应，严禁家庭操作 |
| NH₃ 喷泉实验 | 验证 NH₃ 的溶解性 | 充满 NH₃ 的圆底烧瓶倒置于盛水（含酚酞）的烧杯上 | 水迅速喷入烧瓶，溶液变红 | NH₃ 有强烈刺激性气味，注意通风 |
| 浓 H₂SO₄ 脱水性 | 验证浓 H₂SO₄ 脱水性 | 向蔗糖中加入浓 H₂SO₄ | 蔗糖变黑，体积膨胀，产生 SO₂ 刺激性气味 | 浓 H₂SO₄ 强腐蚀性，严禁皮肤接触；SO₂ 有毒 |

## 易错点 / 难点

1. **Na₂O vs Na₂O₂**：Na 在常温下与 O₂ 反应生成 Na₂O；点燃或加热才生成 Na₂O₂；Na₂O₂ 与 H₂O、CO₂ 反应均放出 O₂（作供氧剂原理）
2. **离子方程式拆写规则**：强酸强碱可溶盐拆开写；难溶物（BaSO₄、AgCl 等）、弱酸弱碱、气体不拆；HF、H₂CO₃ 不拆（弱酸）
3. **Fe²⁺ 与 Fe³⁺ 混淆**：Fe 与盐酸/稀硫酸反应生成 Fe²⁺；Fe 与 Cl₂ 反应生成 Fe³⁺；FeCl₃ 溶液遇 Fe 片会再生 FeCl₂
4. **Al 钝化误区**：铝遇浓硫酸/浓硝酸钝化（形成致密氧化膜），不是不反应，常温下钝化保护
5. **NH₃ 的催化氧化与直接燃烧**：NH₃ 在 O₂ 中燃烧生成 N₂；催化氧化（工业）生成 NO，是制硝酸的关键步骤
6. **NO、NO₂ 与水的反应**：3NO₂ + H₂O = 2HNO₃ + NO（不是全部变成酸）；NO 不与水反应
7. **氧化还原双线桥标注**：升价失电子被氧化（还原剂），降价得电子被还原（氧化剂）；口诀"升失氧，降得还"

## 典型例题类型

1. **物质分类题**：给出一组物质，判断纯净物/混合物，单质/化合物，酸/碱/盐，电解质/非电解质
2. **离子方程式书写**：Na₂CO₃ + HCl、NaOH + FeCl₃、Ba(OH)₂ + H₂SO₄ 等的离子方程式
3. **氧化还原分析题**：标双线桥，判断氧化剂/还原剂，求转移电子数
4. **Na₂O₂ 计算题**：Na₂O₂ 与水或 CO₂ 反应，已知生成 O₂ 体积求 Na₂O₂ 质量（或与 Na₂O₂ 物质的量相关计算）
5. **铁的化合物转化链**：由题目条件判断溶液中 Fe²⁺/Fe³⁺，写出转化离子方程式
6. **混合气体与 NaOH 溶液反应**：NO₂ 与 NO 的混合气体通入 NaOH 溶液，剩余气体体积计算
7. **铵盐鉴别与 NH₄⁺ 的计算**：铵盐溶液与 NaOH 反应，根据 NH₃ 体积求铵盐含量
