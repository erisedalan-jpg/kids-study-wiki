## 推荐学习路径

### 阶段 1: 原子结构与元素周期表 (第五章前半, 2.5 周)
- 原子结构：质子数 / 中子数 / 核外电子数；核电荷数 = 质子数 = 原子序数
- 核外电子排布：2-8-8 规律；前 20 号元素各层电子数 → [[426-原子结构|原子结构]] [[电子层]]
- 元素周期表的结构：7 个周期（短周期 1-3，长周期 4-6，超长周期 7）；18 个族（IA-VIIA、0、IB-VIIB、VIII）
- 主族元素的判断与元素位置（族序数 = 最外层电子数）

### 阶段 2: 元素周期律 (第五章后半, 2 周)
- 同周期从左到右：原子半径减小、失电子能力减弱、得电子能力增强、金属性减弱、非金属性增强
- 同族从上到下：原子半径增大、失电子能力增强、得电子能力减弱、金属性增强、非金属性减弱
- 元素金属性/非金属性强弱的判断依据（与 H₂O/O₂ 反应，最高价含氧酸的酸性）→ [[252-元素周期律|元素周期律]]
- 氢化物的稳定性：同周期右强左弱；同族上强下弱

### 阶段 3: 化学反应与热效应 (第六章前半, 1.5 周)
- 放热反应 vs 吸热反应：燃烧/中和为放热；大多数分解反应为吸热
- 热化学方程式写法：注明物质聚集状态；ΔH 的含义（kJ/mol）→ [[309-热化学方程式|热化学方程式]] [[288-反应热|反应热]]
- 盖斯定律（初步）：多步反应的 ΔH 可以叠加

### 阶段 4: 原电池与电解（入门）(第六章后半, 2 周)
- 原电池原理：负极氧化（失电子）/ 正极还原（得电子）；电流方向（外电路正极流向负极）→ [[287-原电池|原电池]]
- 铜锌原电池：Zn 为负极（失 e⁻，溶解）；Cu 为正极（H⁺ 得 e⁻，产生 H₂）
- 化学能转化为电能（电池）；电能转化为化学能（电解）
- 金属腐蚀：吸氧腐蚀 vs 析氢腐蚀；防腐措施（牺牲阳极法、外加电流法）→ [[金属腐蚀与防护]]

### 阶段 5: 有机化合物基础 (第七章, 3.5 周)
- 甲烷：正四面体结构；取代反应（CH₄ + Cl₂ 光照）→ [[010-天然气|甲烷]] [[652-取代反应|取代反应]]
- 乙烯：平面结构；不饱和键（C=C）；加成反应；乙烯与 Br₂ 水褪色 → [[232-乙烯|乙烯]] [[233-乙烯加成反应|加成反应]]
- 苯：平面正六边形；苯环结构；取代反应（硝化/卤代）；苯的加氢 → [[247-苯|苯]] [[芳香化合物]]
- 乙醇：官能团 —OH（羟基）；与 Na 反应放 H₂；催化氧化 → [[238-乙醇|乙醇]] [[羟基]]
- 乙酸：官能团 —COOH（羧基）；弱酸性；酯化反应（与乙醇生成乙酸乙酯）→ [[236-乙酸|乙酸]] [[250-酯化反应|酯化反应]]
- 有机物命名简介：系统命名（烷烃的命名）；官能团决定化学性质

### 阶段 6: 化学与可持续发展 (第八章, 1 周)
- 金属冶炼：热还原法（Fe、Cu）/ 电解法（Na、Al、Mg）/ 热分解法（Hg）
- 海水资源：海水提溴、海水提镁、海水淡化原理
- 化工污染与绿色化学：废气（SO₂、NOₓ）处理；废水处理；原子利用率
- 新材料：高分子材料、纳米材料、复合材料简介 → [[279-绿色化学|绿色化学]] [[992-原子利用率|原子利用率]]

总课时约 16-18 周 ≈ 1 学期.

## 重点知识点

### 元素周期律规律对比表

| 性质 | 同周期（左→右） | 同族（上→下） |
|---|---|---|
| 原子半径 | 减小 | 增大 |
| 金属性 | 减弱 | 增强 |
| 非金属性 | 增强 | 减弱 |
| 最高价氧化物对应水化物酸性 | 增强 | 减弱（酸） |
| 氢化物稳定性 | 增强 | 减弱 |
| 元素最高化合价 | +1 → +7（规律） | 相同（同族） |

### 关键化学方程式（必修二核心，共 15 条）

| # | 反应名称 | 化学方程式 |
|---|---|---|
| 1 | Na 与 O₂ 在不同条件 | 4Na + O₂ = 2Na₂O (常温)；2Na + O₂ $\overset{点燃}{=}$ Na₂O₂ |
| 2 | 铜锌原电池负极 | Zn - 2e⁻ = Zn²⁺（负极氧化） |
| 3 | 铜锌原电池正极 | 2H⁺ + 2e⁻ = H₂↑（正极还原） |
| 4 | CH₄ 与 Cl₂ 取代（第一步） | CH₄ + Cl₂ $\overset{光照}{=}$ CH₃Cl + HCl |
| 5 | C₂H₄ 与 Br₂ 加成 | CH₂=CH₂ + Br₂ → CH₂BrCH₂Br |
| 6 | C₂H₄ 与 H₂O 加成 | CH₂=CH₂ + H₂O $\overset{催化剂}{=}$ CH₃CH₂OH |
| 7 | 乙醇催化氧化为乙醛 | 2CH₃CH₂OH + O₂ $\overset{Cu/\Delta}{=}$ 2CH₃CHO + 2H₂O |
| 8 | 乙酸与乙醇酯化 | CH₃COOH + C₂H₅OH $\underset{浓H_2SO_4}{\overset{\Delta}{\rightleftharpoons}}$ CH₃COOC₂H₅ + H₂O |
| 9 | 苯的硝化 | C₆H₆ + HNO₃ $\overset{浓H_2SO_4}{\underset{\Delta}{\rightleftharpoons}}$ C₆H₅NO₂ + H₂O |
| 10 | 工业冶炼铁（高炉） | Fe₂O₃ + 3CO $\overset{高温}{=}$ 2Fe + 3CO₂ |
| 11 | 电解氯化铝（冶铝） | 2Al₂O₃(熔融) $\overset{通电}{=}$ 4Al + 3O₂↑ |
| 12 | 乙烯的聚合 | n CH₂=CH₂ $\overset{催化剂}{=}$ [-CH₂-CH₂-]ₙ |
| 13 | 乙醇与 Na 反应 | 2C₂H₅OH + 2Na = 2C₂H₅ONa + H₂↑ |
| 14 | 吸氧腐蚀总反应（简化） | 2Fe + O₂ + 2H₂O = 2Fe(OH)₂ → 进一步氧化为 Fe(OH)₃ |
| 15 | 乙酸电离 | CH₃COOH ⇌ CH₃COO⁻ + H⁺（弱酸部分电离） |

### 典型实验（核心，共 6 个）

| 实验名称 | 目的 | 关键操作 | 现象 | 安全注意 |
|---|---|---|---|---|
| 铜锌原电池 | 验证原电池原理 | Zn 和 Cu 片分别接导线插入稀硫酸；接灵敏电流计 | 电流计偏转；Cu 片产生气泡；Zn 片溶解 | 稀硫酸有腐蚀性 |
| 乙烯使溴水褪色 | 验证 C=C 加成反应 | 乙醇与浓硫酸共热至 170°C 制乙烯；通入溴水 | 溴水橙红色褪去 | 浓硫酸高腐蚀性、高温操作；乙烯可燃 |
| 酯化反应（乙酸乙酯制备） | 验证酯化反应 | 乙酸 + 乙醇 + 少量浓 H₂SO₄ 加热；导管通入饱和 Na₂CO₃ 溶液 | 有果香气味的液体分层 | 浓硫酸；防止暴沸（不可用橡皮管连接以防倒吸） |
| 乙醇催化氧化 | 观察 Cu 催化乙醇氧化 | 铜丝在酒精灯上反复烧红后插入乙醇蒸气 | Cu 丝在乙醇中由黑变红；有刺激气味（乙醛） | 乙醇可燃；乙醛有毒 |
| Na 与乙醇反应 | 比较 Na 与水/醇的活泼性 | Na 小块放入无水乙醇 | 产生气泡（H₂）；比与水反应缓慢 | Na 活泼金属，腐蚀性 |
| 苯的取代反应（溴代） | 验证苯环的取代反应 | 苯 + 液溴 + 铁粉催化；导出气体通入 AgNO₃ 溶液 | 产生 HBr 气体；AgNO₃ 溶液中产生淡黄色沉淀 | 苯和液溴均有毒，通风橱操作 |

## 易错点 / 难点

1. **元素周期律中的例外**：第 IA 族 H 是非金属；第 VIII 族（Fe、Co、Ni）按规律最高化合价 ≠ 族序数；O 和 F 无最高正化合价
2. **原电池正负极混淆**：负极是活泼金属（失电子，氧化）；正极是不活泼金属或电极（得电子，还原）；外电路电流从正极流出（与电子流向相反）
3. **吸热 vs 放热反应**：不是反应条件加热就是吸热——大多数燃烧/中和反应需要点燃/加热但本质是放热；C + CO₂ → 2CO 是吸热
4. **乙烯与乙烷结构区别**：乙烷（C₂H₆）饱和无双键，只能取代；乙烯（C₂H₄）含 C=C，以加成为主
5. **酯化反应可逆性**：酯化反应是可逆反应，浓 H₂SO₄ 起催化和吸水双重作用；产物要用饱和 Na₂CO₃ 分液（除乙醇和乙酸）
6. **苯不使溴水褪色**：苯与溴水不反应，萃取而非反应（有机层分层）；乙烯才通过加成使溴水褪色

## 典型例题类型

1. **元素推断题**：根据原子序数、族周期位置、性质推断元素是哪种
2. **原电池正负极判断**：给出原电池装置图，判断正负极、写出电极反应式
3. **有机化合物结构分析**：根据分子式判断可能的结构简式（同分异构体）
4. **官能团反应类型判断**：加成/取代/氧化/酯化——根据反应条件和有机物结构判断类型
5. **有机计算题**：已知有机物的燃烧产物（CO₂、H₂O 的量），反推分子式
6. **周期律规律应用题**：比较同周期/同族元素的原子半径、金属性、非金属性大小
7. **金属冶炼方法选择题**：根据金属活动性判断应用热还原法/电解法/热分解法
