## 推荐学习路径

### 阶段 1: 反应热与焓变 (第一章, 2.5 周)
- 热力学概念：系统/环境；放热 ΔH < 0，吸热 ΔH > 0 → [[焓变]] [[反应热]]
- 热化学方程式规范：物质聚集状态必须注明；ΔH 单位 kJ/mol；方程式系数变化 ΔH 等比例变化
- 中和热：强酸强碱的中和热约为 -57.3 kJ/mol（稀溶液）；弱酸弱碱中和热 > 57.3 kJ/mol
- 盖斯定律：同一反应无论经过多少步，总 ΔH 不变 → 利用盖斯定律计算目标反应 ΔH → [[盖斯定律]]
- 标准摩尔生成焓（ΔfH°）与标准摩尔燃烧焓（ΔcH°）定义与计算

### 阶段 2: 化学反应速率 (第二章前半, 2 周)
- 反应速率定义：v = Δc/Δt（单位 mol/(L·s) 或 mol/(L·min)）→ [[化学反应速率]]
- 同一反应不同物质速率之比 = 化学计量数之比
- 影响速率的因素：浓度（增大→加快）/ 温度（升高 10°C，速率约增大 2-4 倍）/ 催化剂 / 压强（对气体）/ 接触面积
- 活化能概念：活化能越低，反应速率越快；催化剂降低活化能 → [[活化能]] [[催化剂]]

### 阶段 3: 化学平衡 (第二章后半, 3 周)
- 平衡状态特征：正反应速率 = 逆反应速率；各组分浓度不变；动态平衡
- 平衡状态判断：同一物质的消耗速率 = 生成速率（必须是正逆方向）→ [[化学平衡]]
- 平衡常数 K：只与温度有关（温度变化 K 才变）；K 越大，正反应程度越大；写法注意纯固液不写入
- 等效平衡方法：判断等效条件（等温等容时，按比例等效即等价）
- 勒夏特列原理：平衡向减小影响的方向移动 → [[勒夏特列原理]]
  - 浓度增大→正向移动；浓度减小→逆向移动
  - 温度升高→向吸热方向移动
  - 加压→向气体分子数少的方向移动
- 平衡转化率计算：三段式（初始/变化/平衡）→ 求 K 或求平衡组成

### 阶段 4: 电解质溶液平衡 (第三章, 3 周)
- 水的电离平衡：Kw = c(H⁺) × c(OH⁻) = 10⁻¹⁴（25°C）；升温 Kw 增大 → [[水的电离]] [[Kw]]
- pH 的定义：pH = -lg c(H⁺)；酸性 pH < 7，碱性 pH > 7，中性 pH = 7（25°C 下）
- 弱酸 / 弱碱的电离平衡：Ka（酸电离常数）/ Kb；Ka 越大酸性越强 → [[电离平衡]] [[Ka]]
- 盐的水解：强酸弱碱盐（酸性）/ 弱酸强碱盐（碱性）/ 双弱盐；水解促进水的电离 → [[盐的水解]]
- 缓冲溶液概念：弱酸及其盐体系（如 CH₃COOH / CH₃COONa）抵抗 pH 变化
- 沉淀溶解平衡：Ksp = c(Mⁿ⁺) × c(Aᵐ⁻)；Ksp 只与温度有关 → [[溶度积]] [[Ksp]]
- Qc 与 Ksp 比较：Qc > Ksp 有沉淀析出；Qc < Ksp 沉淀溶解

### 阶段 5: 电化学（进阶）(第四章, 2.5 周)
- 原电池进阶：电极反应式的书写（氧化反应为负极，还原反应为正极）→ [[原电池]] [[电极反应式]]
- 电解池进阶：阳极接正极（氧化），阴极接负极（还原）→ [[电解池]]
- 惰性电极 vs 活性电极：惰性电极（C、Pt）阳极氧化溶液中阴离子；活性电极（金属阳极）本身溶解
- 电解顺序：阳离子在阴极的放电顺序（Ag⁺ > Fe³⁺ > Cu²⁺ > H⁺ > Fe²⁺ > Zn²⁺）
- 法拉第定律：n(e⁻转移) = I × t / F（F = 96500 C/mol）→ 由转移电子数计算沉积/溶解质量 → [[法拉第定律]]
- 实用电池：铅蓄电池、氢氧燃料电池（电极反应）→ [[燃料电池]] [[铅蓄电池]]
- 电镀与电解精炼原理

总课时约 18-20 周 ≈ 1 学期.

## 重点知识点

### 化学平衡三段式示例

以 N₂(g) + 3H₂(g) ⇌ 2NH₃(g) 为例，设初始 n(N₂) = 1 mol，n(H₂) = 3 mol，容器体积 1 L，NH₃ 转化率为 50%（待验证）：

| | N₂ | H₂ | NH₃ |
|---|---|---|---|
| 初始 (mol) | 1 | 3 | 0 |
| 变化 (mol) | -x | -3x | +2x |
| 平衡 (mol) | 1-x | 3-3x | 2x |

### 典型实验（核心，共 6 个）

| 实验名称 | 目的 | 关键操作 | 现象 | 安全注意 |
|---|---|---|---|---|
| 中和热测定 | 测定强酸强碱中和热 | 保温杯中混合等体积等浓度 HCl 和 NaOH；精确测温 | 温度升高约 3°C（0.5 mol/L，50 mL） | 温度计不可搅拌，防止破损 |

| 电解精炼铜 | 理解电解精炼原理 | 粗铜（阳极）+ 纯铜（阴极）+ CuSO₄ 溶液电解 | 阳极质量减少，阴极质量增加，溶液浓度基本不变 | 注意防止短路 |

## 易错点 / 难点

1. **ΔH 符号与大小比较**：放热 ΔH < 0，吸热 ΔH > 0；比较 ΔH 大小时，-285 kJ/mol < -57.3 kJ/mol（数值越小放热越多，但 "-285 < -57.3" 数学上更小，放热更多），需注意"大"的含义
2. **K 的书写规则**：纯固体和纯液体不写入 K 表达式；H₂O(l) 不写入（溶液体系）；H₂O(g) 要写入（气相体系）
3. **平衡判断误区**：v正 = v逆 不代表浓度不变（浓度不变才是平衡）；同一物质消耗速率等于生成速率才是平衡（不能同方向比较）
4. **pH 计算陷阱**：强酸强碱混合后先算剩余 H⁺ 或 OH⁻ 浓度；弱酸 0.1 mol/L 的 pH 不等于 1（弱酸部分电离）
5. **盐水解与电离的共存**：含弱酸根的盐溶液，水解趋势与酸的电离趋势同时存在；醋酸钠溶液中 c(Na⁺) > c(CH₃COO⁻) > c(OH⁻) > c(H⁺)
6. **法拉第定律计算**：n(e⁻) = It/F；注意 F = 96500 C/mol ≈ 9.65×10⁴ C/mol；析出金属 n = n(e⁻)/化合价
7. **电解池阳极判断**：活性金属阳极（如 Cu）优先溶解，不是溶液中阴离子放电；用惰性电极才是溶液中阴离子放电顺序

## 典型例题类型

1. **盖斯定律计算**：给出两个或三个反应的 ΔH，求目标反应的 ΔH（线性组合）
2. **平衡常数 K 的计算**：给出平衡浓度，写出 K 表达式并计算；判断 K 与条件的关系
3. **勒夏特列原理分析**：给出平衡体系后改变条件（加压/升温/加某物质），判断移动方向及各量变化
4. **电解质溶液 pH 计算**：强酸/强碱溶液 pH；混合溶液的 pH；稀释后的 pH 变化
5. **盐水解与溶液 pH 判断**：给出盐溶液，判断酸碱性；比较混合溶液中各离子浓度大小
6. **Ksp 计算与沉淀判断**：给出 Ksp 值，判断是否产生沉淀（Qc 与 Ksp 比较）；计算沉淀完全的条件
7. **法拉第电解计算**：给出电流 I、时间 t，计算阴极析出金属质量或阳极溶解质量
