## 推荐学习路径

### 阶段 1: 有机化合物的结构基础 (第一章, 2 周)
- 有机物的特征：C 是四价；H、O、N、卤素等常见元素；共价键为主
- 有机物性质特征：难溶于水（多数）/ 熔点低 / 易燃 / 反应慢（与无机相比）
- 结构表示方式：电子式 → 结构式 → 结构简式 → 键线式 → [[结构简式]] [[键线式]]
- 同分异构体概念：分子式相同，结构不同 → 碳链异构 / 官能团位置异构 / 官能团种类异构 → [[同分异构体]]
- 有机物分析方法简介：质谱法（相对分子质量）/ 红外光谱（官能团）/ 核磁共振（H 环境）→ [[核磁共振]]

### 阶段 2: 烷烃与系统命名 (第二章前段, 1.5 周)
- 烷烃通式 CₙH₂ₙ₊₂；官能团：无（仅 C-C 和 C-H 单键）；sp³ 杂化
- IUPAC 系统命名法步骤：选最长碳链 → 给取代基最小编号 → 命名取代基 → 组合 → [[烷烃命名]]
- 烷烃的性质：化学性质稳定；卤代（取代反应，需光照）；燃烧生成 CO₂ + H₂O → [[取代反应]]
- 甲烷同系物：乙烷、丙烷、丁烷；正丁烷与异丁烷（结构异构）

### 阶段 3: 烯烃与炔烃 (第二章中段, 2 周)
- 烯烃通式 CₙH₂ₙ；含 C=C；sp² 杂化；官能团：碳碳双键 → [[烯烃]]
- 烯烃的命名：含 C=C 的最长链为主链；C=C 编号最小
- 烯烃的化学性质：加成反应（与 Br₂、H₂、HX、H₂O）/ 聚合反应 / 氧化（与 KMnO₄ 褪色）→ [[加成反应]]
- 炔烃通式 CₙH₂ₙ₋₂；含 C≡C；sp 杂化；官能团：碳碳三键 → [[炔烃]]
- 乙炔（CH≡CH）的制备（CaC₂ + H₂O → CH≡CH + Ca(OH)₂）与性质
- 顺反异构（cis/trans）：双键两侧基团不同，旋转受限 → [[顺反异构]]

### 阶段 4: 芳香烃 (第二章后段, 1.5 周)
- 苯的结构：所有 C 为 sp² 杂化；苯环中 6 个 C-C 键完全等价（介于单双键之间）→ [[苯]] [[芳香烃]]
- 苯的化学性质：取代反应（卤代、硝化、磺化）；加氢（还原）；不与 KMnO₄ 褪色（区别于烯烃）
- 苯的同系物：甲苯（苯甲基）、二甲苯（邻/间/对）；侧链可被 KMnO₄ 氧化（苯环不被氧化）
- 多环芳烃简介：萘（C₁₀H₈）；芳香性的判断（休克尔规则简介）

### 阶段 5: 卤代烃、醇、酚、醚 (第三章, 2.5 周)
- 卤代烃官能团 —X（卤素）；水解（取代，生成醇）；消去（生成烯烃）→ [[卤代烃]] [[消去反应]]
- 消去与取代的条件区别：消去用醇的 NaOH 溶液（加热）；取代水解用 NaOH 水溶液
- 醇（—OH 羟基）分类：伯醇 / 仲醇 / 叔醇 → [[醇]] [[羟基]]
  - 与 Na 反应（放 H₂）；催化氧化（伯醇→醛，仲醇→酮，叔醇不被氧化）
  - 消去反应（生成烯烃，条件：浓 H₂SO₄，170°C）；酯化反应
- 酚（苯环 —OH）：苯酚（C₆H₅OH）酸性比 H₂CO₃ 弱但比 HCO₃⁻ 强；苯酚与 FeCl₃ 显紫色（鉴别）→ [[苯酚]]
- 醚（R—O—R'）：化学性质稳定；乙醚（CH₃CH₂OCH₂CH₃）作麻醉剂（可燃！）

### 阶段 6: 醛、酮、羧酸及衍生物 (第四章, 2.5 周)
- 醛（—CHO 醛基）：官能团在碳链末端 → [[醛]] [[醛基]]
  - 氧化（银镜反应：醛 + [Ag(NH₃)₂]⁺ → 银镜；新制 Cu(OH)₂：砖红色沉淀）→ [[银镜反应]]
  - 加成（与 H₂ 还原为醇）
- 酮（—CO— 羰基在链中间）：不与银镜反应；加成还原为醇 → [[酮]]
- 羧酸（—COOH 羧基）：弱酸；酯化反应（与醇生成酯，浓 H₂SO₄ 催化，可逆）→ [[羧酸]] [[羧基]]
- 酯（—COO—）：水解分酸式和碱式（皂化反应）→ [[酯]] [[皂化反应]]
- 酰胺（—CONH—）：肽键是氨基酸连接的官能团 → [[酰胺]] [[肽键]]
- 油脂：高级脂肪酸的甘油酯；皂化反应生成脂肪酸钠 + 甘油 → [[油脂]] [[皂化反应]]

### 阶段 7: 生命中的基础化学物质 (第五章, 2 周)
- 糖类（碳水化合物）通式 (CH₂O)ₙ → [[糖类]]
  - 单糖：葡萄糖（C₆H₁₂O₆）/ 果糖（同分异构体）；含多个 —OH 和 —CHO（或酮基）
  - 二糖：蔗糖（非还原性）/ 麦芽糖（还原性，含醛基）
  - 多糖：淀粉（遇碘变蓝）/ 纤维素（氢键网络，结晶）→ [[淀粉]] [[纤维素]]
- 氨基酸：含 —NH₂ 和 —COOH；两性（既显酸性又显碱性）→ [[氨基酸]]
- 蛋白质：氨基酸通过肽键缩合；一级/二级/三级/四级结构；酶是蛋白质（具有催化活性）→ [[蛋白质]] [[肽键]]
- 核酸：DNA（双螺旋，脱氧核糖）/ RNA（单链，核糖）；碱基互补配对 → [[核酸]]

### 阶段 8: 高分子化合物 (第六章, 1.5 周)
- 加聚反应：单体含 C=C；乙烯 → 聚乙烯（PE）；氯乙烯 → PVC；苯乙烯 → 聚苯乙烯 → [[加聚反应]]
- 缩聚反应：单体含两种可反应官能团（—OH 和 —COOH 等）；生成小分子副产物（H₂O）→ [[缩聚反应]]
  - 己二酸 + 己二胺 → 尼龙 66；对苯二甲酸 + 乙二醇 → 聚酯（PET）
- 常见高分子：橡胶（天然 / 合成）/ 纤维（天然：棉麻丝毛；合成：涤纶/尼龙/腈纶）/ 塑料
- 高分子特点：相对分子质量大（n 重复单元）/ 链长分布不均一 / 可能交联
- 功能高分子：离子交换树脂 / 导电高分子 / 高吸水树脂

总课时约 18-20 周 ≈ 1 学期.

### 银镜反应与新制 Cu(OH)₂ 对比

| 试剂 | 反应条件 | 现象 | 鉴别 |
|---|---|---|---|
| 银镜反应（托伦斯试剂） | 水浴加热 | 银白色镜面（Ag 沉积） | 检验醛基（—CHO） |
| 新制 Cu(OH)₂ | 加热 | 砖红色沉淀（Cu₂O） | 检验醛基（—CHO） |

注意：葡萄糖含醛基，蔗糖不含（非还原性），麦芽糖含（还原性）。

| 实验名称 | 目的 | 关键操作 | 现象 | 安全注意 |
|---|---|---|---|---|
| 乙烯制备与加成 | 制备乙烯并验证加成 | 乙醇 + 浓 H₂SO₄ 加热至 170°C；通入溴水 | 溴水红棕色褪去 | 浓 H₂SO₄ 高温高腐蚀；乙烯可燃，通风橱操作 |

| 乙醇消去 | 制备乙烯 | 乙醇 + 浓 H₂SO₄，170°C；用 KMnO₄ 检验纯度（先除 SO₂） | 带火星木条不复燃；乙烯可燃 | 浓 H₂SO₄；高温操作 |

## 易错点 / 难点

1. **消去与取代的条件区分**：卤代烃消去用醇的 NaOH 溶液（加热）；取代水解用水的 NaOH 溶液；醇消去用浓 H₂SO₄ 加热（不同温度产物不同：140°C 生成醚，170°C 生成烯烃）
2. **酚与醇的羟基不同**：苯酚的 —OH 直接连苯环，酸性比醇强（能与 Na₂CO₃ 溶液反应，不能使石蕊变红）；醇的 —OH 连烷基，不与 Na₂CO₃ 反应
3. **银镜反应误区**：银镜反应必须在碱性条件下；检验葡萄糖时不能混入醇（醇不反应但会影响）；Cu₂O 是砖红色（不是铜红，铜红是纯铜）
4. **同分异构体计数**：写 C₄H₁₀ 的同分异构体时按碳链骨架枚举，再移动取代基位置；防止遗漏或重复，建议系统画法
5. **聚合物的判断方向**：加聚产物写重复单元时，注意括号内的结构和下标 n；从结构简式逆推单体时，切断 C-C 键（打开双键）
6. **蔗糖与麦芽糖**：蔗糖（葡萄糖 + 果糖连接）是非还原性糖，无游离醛基，不能银镜反应；麦芽糖（2 个葡萄糖）有游离醛基，能银镜反应
7. **皂化反应中碱用量**：油脂皂化时 1 mol 甘油三酯 = 3 mol 脂肪酸 + 1 mol 甘油；计算需要 NaOH 物质的量时按 3 mol 计算

## 典型例题类型

1. **有机结构推断（推断题）**：根据分子式、官能团反应特征（银镜/褪色/与 Na 反应等）推断有机物结构，多步推断
2. **同分异构体数目计算**：给定分子式和官能团条件，系统写出所有可能结构
3. **有机合成路线设计**：从简单原料（乙烯、苯等）出发，设计多步合成目标有机物
4. **官能团反应类型判断**：加成/取代/消去/氧化/聚合——根据反应物、条件、产物判断
5. **聚合物与单体互推**：已知聚合物结构，求单体；已知单体，写聚合物结构简式
6. **有机物与生物大分子**：判断糖类/蛋白质/核酸的结构特征；计算肽键数目（n 个氨基酸缩合失去 (n-1) 个 H₂O，形成 (n-1) 个肽键）
7. **有机物含氮化合物计算**：氨基酸分子量计算；蛋白质水解产生氨基酸数量的计算
