## 推荐学习路径

### 阶段 1: 集合与逻辑 (第1章, 3 周)
- 集合概念（元素与集合、空集、有限集/无限集）→ [[集合的运算]]（交、并、补）
- [[子集]]与[[真子集]] → 幂集元素个数（$2^n$）
- [[充分条件]]与[[必要条件]]的判断 → [[充要条件]]
- [[全称量词]]（$\forall$）与[[存在量词]]（$\exists$）→ 否命题与命题的否定的区别
- 难点：充分/必要条件方向不混淆；集合运算符号（∩∪∁）书写规范

### 阶段 2: 一元二次三合一 (第2章, 4 周)
- 二次函数 $y = ax^2 + bx + c$ 图象（开口、顶点、对称轴）
- [[判别式]] $\Delta = b^2 - 4ac$：三种情况对应根的个数
- 一元二次方程与不等式的解集（数形结合）
- $ax^2 + bx + c > 0$ 解集依据 $a$ 正负、$\Delta$ 正负六情况分类
- [[韦达定理]]：$x_1 + x_2 = -b/a$，$x_1 x_2 = c/a$

### 阶段 3: 函数概念与性质 (第3章, 4 周)
- 函数三要素（定义域、对应法则、值域）→ [[映射]]概念
- [[单调函数]]定义（用定义证单调性三步法）
- [[奇函数]]与[[偶函数]]定义 → 图象对称性
- [[复合函数]]基本判断 → 定义域求法

### 阶段 4: 指数函数与对数函数 (第4章, 4 周)
- 整数指数幂 → 有理数指数幂 → [[指数函数]] $y = a^x$（$a>0,a\neq1$）
- [[对数]]定义：$\log_a N = b \Leftrightarrow a^b = N$
- 对数运算法则（积/商/幂）→ [[换底公式]]：$\log_a b = \dfrac{\ln b}{\ln a}$
- [[对数函数]] $y = \log_a x$ 图象与性质（单调性依 $a$ 大小）
- 指数函数与对数函数互为反函数

### 阶段 5: 三角函数 (第5章, 6 周)
- [[任意角]]与[[弧度制]]：$1\text{ rad} = \dfrac{180°}{\pi}$
- 单位圆定义六个三角函数 → [[三角函数线]]
- [[诱导公式]]（口诀"奇变偶不变，符号看象限"）
- $\sin/\cos/\tan$ 图象与性质（周期、振幅、相位）
- $y = A\sin(\omega x + \varphi)$ 图象变换（$A/\omega/\varphi$ 物理意义）

总课时约 21 周 ≈ 高一上+下（视学校进度）。

## 重点公式与定理

### 集合与逻辑

$$A \cup B,\quad A \cap B,\quad \complement_U A$$

- $n$ 个元素集合的子集个数：$2^n$；真子集：$2^n - 1$

### 一元二次

$$\Delta = b^2 - 4ac$$

$$x = \frac{-b \pm \sqrt{\Delta}}{2a}$$

**韦达定理**：若 $x_1, x_2$ 是 $ax^2+bx+c=0$ 的两根，则

$$x_1 + x_2 = -\frac{b}{a}, \qquad x_1 x_2 = \frac{c}{a}$$

### 指数与对数

$$a^m \cdot a^n = a^{m+n}, \quad (a^m)^n = a^{mn}, \quad (ab)^n = a^n b^n$$

$$\log_a(MN) = \log_a M + \log_a N$$

$$\log_a \frac{M}{N} = \log_a M - \log_a N$$

$$\log_a M^n = n\log_a M$$

**换底公式**：$\log_a b = \dfrac{\log_c b}{\log_c a}$

### 三角函数

$$\sin^2\alpha + \cos^2\alpha = 1$$

$$\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$$

**诱导公式（常用）**：

$$\sin(\pi - \alpha) = \sin\alpha, \quad \cos(\pi - \alpha) = -\cos\alpha$$

$$\sin\left(\frac{\pi}{2} - \alpha\right) = \cos\alpha, \quad \cos\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha$$

**$y = A\sin(\omega x + \varphi)$**：振幅 $A$，周期 $T = \dfrac{2\pi}{\omega}$，初相 $\varphi$

## 典型例题类型

### 计算题
1. **集合运算**：给定两集合，求交集、并集、补集（含含参数情形）
2. **一元二次不等式求解**：$ax^2 + bx + c \leq 0$，分讨论 $a$ 正负
3. **指对混合运算**：$\log_2 8 + \log_4 16 - \log_2 \sqrt{2}$ 类化简
4. **三角函数值计算**：利用诱导公式化简 $\sin 150°$，$\tan 225°$ 等
5. **$y=A\sin(\omega x+\varphi)$ 参数求解**：已知图象特征求 $A,\omega,\varphi$

### 证明题
6. **用定义证函数单调性**：取 $x_1 < x_2 \in D$，作差/商比较符号
7. **奇偶性证明**：验证 $f(-x) = \pm f(x)$，先验证定义域关于原点对称

### 综合题
8. **含参集合问题**：集合相等/包含时分类讨论参数取值范围
9. **函数最值问题**：换元法求含根式/对数的函数最值
10. **三角函数图象变换**：从 $y=\sin x$ 经平移/伸缩变为目标函数

## 易错点

1. **集合元素互异性**：含参数集合需检验元素是否重复，如 $\{a, a^2, 3\}$ 需 $a \neq 3, a^2 \neq 3, a \neq a^2$
2. **补集误用**：$\complement_U A$ 必须在全集 $U$ 的范围内，不明确全集时不能求补集
3. **一元二次不等式**：$a=0$ 时退化为一次（或矛盾/恒成立），需单独讨论
4. **对数定义域**：$\log_a x$ 要求 $x>0$ 且 $a>0,a\neq1$，真数必须严格大于 0
5. **弧度与角度混用**：$\sin 30$ (弧度) $\neq \sin 30°$，代入计算前统一单位
6. **$\varphi$ 范围**：$y=A\sin(\omega x+\varphi)$ 中 $|\varphi| \leq \dfrac{\pi}{2}$ 是常见附加限制，不能随意取多个答案
7. **诱导公式符号**："奇变偶不变"指函数名，"符号看象限"指原角所在象限
