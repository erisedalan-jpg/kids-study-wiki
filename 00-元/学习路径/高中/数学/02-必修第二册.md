## 推荐学习路径

### 阶段 1: 平面向量 (第6章, 4 周)
- [[向量]]定义（大小+方向）、零向量、单位向量、相等向量、共线向量
- 向量加减法（三角形/平行四边形法则）→ [[数乘向量]]（方向+模）
- 平面向量基本定理：任意向量可由两不共线向量线性表示
- 建立坐标系 → [[向量的坐标运算]]（加减/数乘/模长公式）
- [[数量积]]：$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta$，坐标形式 $x_1x_2+y_1y_2$
- 应用：向量解三角形（正弦/余弦定理证明）、向量平行垂直条件

### 阶段 2: 复数 (第7章, 2 周)
- 虚数单位 $i$（$i^2 = -1$）→ [[复数]]定义 $a+bi$（$a,b\in\mathbb{R}$）
- 复数加减、乘除（共轭复数）→ 模 $|z| = \sqrt{a^2+b^2}$
- [[复平面]]：实轴/虚轴，复数对应点（向量）
- 共轭复数 $\bar{z} = a-bi$ 的性质

### 阶段 3: 立体几何初步 (第8章, 5 周)
- 空间基本概念（点线面）→ 公理与推论
- 线面关系：平行/相交/异面直线；线面平行/垂直；面面平行/垂直
- 判定定理与性质定理（用两步法证明线面关系）
- 常见几何体（棱柱/棱锥/棱台/球）的表面积与体积公式
- 截面与展开图问题

### 阶段 4: 统计 (第9章, 3 周)
- [[总体]]与[[样本]]、[[抽样方法]]（系统/分层/简单随机）
- [[频率分布直方图]]的读取与绘制
- 统计量：[[均值]]、[[中位数]]、[[众数]]；[[方差]]与[[标准差]]
- 茎叶图、箱线图

### 阶段 5: 概率 (第10章, 3 周)
- [[随机事件]]、[[必然事件]]、[[不可能事件]]
- [[古典概型]]：$P(A) = \dfrac{k}{n}$（等可能）
- [[几何概型]]：$P(A) = \dfrac{A\text{的度量}}{S\text{的度量}}$
- [[加法公式]]：$P(A\cup B) = P(A)+P(B)-P(A\cap B)$
- [[互斥事件]]：$P(A\cup B) = P(A)+P(B)$
- [[条件概率]]与[[独立事件]]：$P(B|A) = \dfrac{P(AB)}{P(A)}$；独立时 $P(AB)=P(A)P(B)$

总课时约 17 周 ≈ 高一下+高二上（视学校进度）。

## 重点公式与定理

### 平面向量

$$|\vec{a}| = \sqrt{x^2 + y^2}$$

**数量积**：
$$\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta = x_1x_2 + y_1y_2$$

**向量垂直条件**：$\vec{a}\perp\vec{b} \Leftrightarrow \vec{a}\cdot\vec{b} = 0$

**向量平行条件**：$\vec{a}\parallel\vec{b} \Leftrightarrow x_1y_2 - x_2y_1 = 0$

**平面向量基本定理**：若 $\vec{e_1},\vec{e_2}$ 不共线，则对任意 $\vec{p}$ 有唯一实数对 $(\lambda_1,\lambda_2)$，使 $\vec{p}=\lambda_1\vec{e_1}+\lambda_2\vec{e_2}$

### 复数

$$i^2 = -1, \quad i^3 = -i, \quad i^4 = 1$$

$$(a+bi)(c+di) = (ac-bd)+(ad+bc)i$$

$$\frac{a+bi}{c+di} = \frac{(a+bi)(c-di)}{c^2+d^2}$$

$$|z| = \sqrt{a^2+b^2}, \quad z\bar{z} = |z|^2$$

### 立体几何（常用体积）

$$V_{\text{棱柱}} = S_{\text{底}} \cdot h, \quad V_{\text{棱锥}} = \frac{1}{3}S_{\text{底}} \cdot h$$

$$V_{\text{球}} = \frac{4}{3}\pi r^3, \quad S_{\text{球}} = 4\pi r^2$$

$$V_{\text{台}} = \frac{h}{3}(S_1 + S_2 + \sqrt{S_1 S_2})$$

### 统计

$$\bar{x} = \frac{1}{n}\sum_{i=1}^n x_i$$

$$s^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2, \quad s = \sqrt{s^2}$$

### 概率

$$P(A\cup B) = P(A)+P(B)-P(A\cap B)$$

$$P(B|A) = \frac{P(AB)}{P(A)}, \quad P(AB) = P(A)\cdot P(B)\text{（独立）}$$

## 典型例题类型

### 计算题
1. **向量坐标运算**：已知向量坐标，求和/差/数乘/模/夹角
2. **向量应用证明**：用向量法证正弦/余弦定理
3. **复数四则运算**：化简含 $i$ 的复数表达式，求模
4. **几何体体积**：已知棱长/高，求棱锥/台/球的体积
5. **古典概型计算**：从 $n$ 个元素中选 $k$ 个，求满足条件的概率

### 证明题
6. **线面平行证明**：找三角形中位线或平行四边形，用定理两步推出线面平行
7. **线面垂直证明**：找面内两相交直线都垂直于该直线，用定理得线面垂直

### 综合题
8. **向量与三角综合**：已知数量积/模，求角度/边长再解三角形
9. **条件概率与独立性**：Urn 模型或列联表中求条件概率、验证独立性
10. **频率分布直方图计算**：读取频率/频数、估计均值方差

## 易错点

1. **向量方向忽略**：$\overrightarrow{AB} = -\overrightarrow{BA}$，箭头方向决定符号，不能任意交换
2. **数量积非向量**：$\vec{a}\cdot\vec{b}$ 是实数（标量），不能再做向量运算
3. **复数除法忘乘共轭**：分母含虚部必须乘以共轭分母，才能变为实数分母
4. **棱锥高与斜高混淆**：棱锥的高是顶点到底面的垂线段，斜面上的高需另算
5. **线面平行判定条件**：线不在面内 + 线与面内某直线平行，两个条件缺一不可
6. **古典概型等可能**：只有每个基本事件等可能才能用 $P=k/n$，否则不适用
7. **独立与互斥混淆**：两事件互斥（不能同时发生）≠ 独立（一个不影响另一个的概率）
