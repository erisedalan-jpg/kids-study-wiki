## 推荐学习路径

### 阶段 1: 计数原理 (第11章, 4 周)

#### 基本原理
- **加法原理**（分类）：完成任务的方法数 = 各类方法数之和（各类互斥）
- **乘法原理**（分步）：完成任务的方法数 = 各步方法数之积（各步独立）
- 原理应用：数字问题（含0/不含0/有重复）、路径问题

#### 排列
- [[487-排列概念|排列]]：从 $n$ 个不同元素中取 $m$ 个，有顺序地排成一列
- $A_n^m = P_n^m = \dfrac{n!}{(n-m)!} = n(n-1)\cdots(n-m+1)$
- 全排列：$n! = A_n^n$
- 特殊排列：相邻问题（捆绑法）、不相邻问题（插空法）、圆排列（固定一个）

#### 组合
- [[505-组合概念|组合]]：从 $n$ 个不同元素中取 $m$ 个，不计顺序
- $C_n^m = \binom{n}{m} = \dfrac{n!}{m!(n-m)!}$
- 组合与排列关系：$A_n^m = C_n^m \cdot m!$
- 组合性质：$C_n^m = C_n^{n-m}$；$C_n^m + C_n^{m+1} = C_{n+1}^{m+1}$（帕斯卡恒等式）

#### 二项式定理
- $(a+b)^n = \sum_{k=0}^{n} C_n^k a^{n-k}b^k$
- 第 $k+1$ 项（通项）：$T_{k+1} = C_n^k a^{n-k}b^k$
- 二项式系数（帕斯卡三角）；各项系数之和 $= 2^n$（令 $a=b=1$）

### 阶段 2: 随机变量及其分布 (第12章, 5 周)

#### 离散随机变量
- [[499-离散型随机变量|离散随机变量]]：取有限或可列无限个值的随机变量
- [[476-分布列|分布列]]（概率分布表）：列出所有可能取值及对应概率，要求 $\sum p_i = 1$

#### 数学期望与方差
- **期望**（均值）：$E(X) = \sum x_i p_i$
- **方差**：$D(X) = E[(X-E(X))^2] = \sum (x_i - E(X))^2 p_i = E(X^2) - [E(X)]^2$
- **标准差**：$\sigma = \sqrt{D(X)}$
- 线性变换：$E(aX+b) = aE(X)+b$；$D(aX+b) = a^2D(X)$

#### 二项分布
- $X\sim B(n,p)$：$n$ 次独立重复试验，每次成功概率 $p$
- $P(X=k) = C_n^k p^k(1-p)^{n-k}$ （$k=0,1,\ldots,n$）
- $E(X) = np$，$D(X) = np(1-p)$

#### 正态分布
- $X\sim N(\mu,\sigma^2)$：均值 $\mu$，方差 $\sigma^2$
- 正态分布曲线（钟形）：关于 $x=\mu$ 对称，拐点在 $x=\mu\pm\sigma$
- 标准正态分布 $N(0,1)$：标准化 $Z = \dfrac{X-\mu}{\sigma}$
- **$3\sigma$ 原则**：

  | 区间 | 概率（近似） |
  |---|---|
  | $(\mu-\sigma, \mu+\sigma)$ | 68.27% |
  | $(\mu-2\sigma, \mu+2\sigma)$ | 95.45% |
  | $(\mu-3\sigma, \mu+3\sigma)$ | 99.73% |

### 阶段 3: 成对数据的统计分析 (第13章, 3 周)
- [[488-散点图|散点图]]：直观判断两变量相关性（正/负/无相关）
- **线性回归方程**（最小二乘法）：$\hat{y} = \hat{b}x + \hat{a}$

  $$\hat{b} = \frac{\sum_{i=1}^n x_i y_i - n\bar{x}\bar{y}}{\sum_{i=1}^n x_i^2 - n\bar{x}^2}, \quad \hat{a} = \bar{y} - \hat{b}\bar{x}$$

- 回归直线过样本中心点 $(\bar{x},\bar{y})$
- **相关系数** $r$：$|r|$ 越接近 1，线性相关越强；$|r| < 0.75$ 一般认为线性相关不显著（教材阈值待核对）
- 回归方程的预测与外推的局限性

总课时约 12 周 ≈ 高二下或高三（选考理科）。

## 重点公式与定理

### 计数原理

$$A_n^m = \frac{n!}{(n-m)!}, \quad C_n^m = \frac{n!}{m!(n-m)!}$$

$$C_n^m = C_n^{n-m}, \quad C_n^m + C_n^{m+1} = C_{n+1}^{m+1}$$

**二项式定理**：

$$(a+b)^n = \sum_{k=0}^{n} C_n^k a^{n-k}b^k$$

通项 $T_{k+1} = C_n^k a^{n-k}b^k$

**各项系数之和**：令 $a=b=1$，$\sum_{k=0}^n C_n^k = 2^n$

**各项系数之和**（奇偶项之差）：令 $a=1,b=-1$，$\sum_{k=0}^n (-1)^k C_n^k = 0$

### 随机变量

**离散随机变量期望**：

$$E(X) = \sum_{i} x_i p_i$$

**方差**：

$$D(X) = \sum_{i}(x_i - E(X))^2 p_i = E(X^2) - [E(X)]^2$$

**线性变换**：

$$E(aX+b) = aE(X)+b, \quad D(aX+b) = a^2D(X)$$

**二项分布 $B(n,p)$**：

$$P(X=k) = C_n^k p^k(1-p)^{n-k}$$

$$E(X) = np, \quad D(X) = np(1-p)$$

### 线性回归

$$\hat{b} = \frac{\displaystyle\sum_{i=1}^n x_i y_i - n\bar{x}\bar{y}}{\displaystyle\sum_{i=1}^n x_i^2 - n\bar{x}^2}, \quad \hat{a} = \bar{y} - \hat{b}\bar{x}$$

**相关系数**：

$$r = \frac{\displaystyle\sum_{i=1}^n (x_i-\bar{x})(y_i-\bar{y})}{\sqrt{\displaystyle\sum_{i=1}^n(x_i-\bar{x})^2 \cdot \sum_{i=1}^n(y_i-\bar{y})^2}}$$

## 典型例题类型

### 计算题
1. **排列计算**：8 人中选 3 人站成一排，有多少种站法（含特殊位置限制）
2. **组合计算**：从 5 男 4 女中选 3 人，含至少 1 女的选法数
3. **二项式通项**：$(2x - \sqrt{y})^6$ 展开中 $x^2y$ 项的系数
4. **分布列与期望**：袋中4红2白，摸3个，求红球数 $X$ 的分布列与期望
5. **二项分布**：某产品合格率 0.9，抽检 10 件，求恰好 8 件合格的概率

### 证明题
6. **组合恒等式证明**：用计数意义或数学归纳法证 $\sum_{k=0}^n C_n^k = 2^n$
7. **二项分布期望推导**：证明 $E[B(n,p)] = np$

### 综合题
8. **正态分布应用**：已知 $\mu, \sigma$，求落在某区间的概率（查表型）
9. **线性回归建模**：给出散点数据，求 $\hat{b}, \hat{a}$，解读回归方程物理意义
10. **概率与计数综合**：古典概型 + 组合计数，求满足多条件事件的概率

## 易错点

1. **加法与乘法原理混用**：加法原理用于"分类"（不同类别的方案），乘法原理用于"分步"（先做A再做B）；若同时用到，需先分清结构
2. **含 0 的数字排列**：首位不能为 0，需先排首位（$n-1$ 种）再排其余，不能直接用全排列
3. **全排列与组合的关系**：$A_n^m = C_n^m \cdot m!$，有顺序就要乘以 $m!$
4. **二项式定理中 $k$ 从 0 开始**：通项是第 $k+1$ 项，求第几项时注意 $k$ 的偏移
5. **分布列概率之和验证**：写完分布列必须验证 $\sum p_i = 1$，否则计算有误
6. **期望与方差公式**：$D(X) = E(X^2) - [E(X)]^2$ 中第二项是期望的平方，不是期望的平方的期望
7. **正态分布对称性**：$P(X < \mu) = P(X > \mu) = 0.5$，利用对称性可简化计算
8. **线性回归的外推风险**：回归方程只在样本数据范围内可靠，大幅外推无效
