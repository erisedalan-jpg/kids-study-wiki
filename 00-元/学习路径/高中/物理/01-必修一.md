## 推荐学习路径

### 阶段 1: 运动学基础 (第1章, 2 周)
- 建立坐标系概念 → [[235-质点-高中|质点]] → [[011-参照物|参考系]]
- 区分路程与[[203-位移-矢量|位移]]（标量 vs 矢量）
- [[045-速度|速度]]（平均速度 / 瞬时速度）→ [[214-加速度|加速度]]定义
- x-t 图象与 v-t 图象的斜率与面积含义
- 关键：矢量方向不能忽略，正负号代表方向

### 阶段 2: 匀变速直线运动 (第2章, 3 周)
- 三大[[734-运动学公式|运动学公式]]推导与记忆
- [[741-逐差法|逐差法]]：纸带实验数据处理核心方法
- 自由落体：g = 9.8 m/s²（近似 10 m/s²）的适用条件
- v-t 图象面积 = 位移；斜率 = 加速度
- 练习：追及问题、相遇问题建方程

### 阶段 3: 力的分析 (第3章, 3 周)
- [[423-gravity-高中|重力]]来源 → 重心位置判断
- [[078-弹力|弹力]]：形变产生，胡克定律 F = kx（弹簧弹力）
- [[081-摩擦力|摩擦力]]：静摩擦 / 滑动摩擦 f = μN；方向判断是难点
- [[460-受力分析|受力分析]]步骤：隔离物体 → 逐一检查（重力、支持力、摩擦力、绳力）
- [[702-矢量合成|矢量合成]]：平行四边形定则；力的分解用正交分解法

### 阶段 4: 牛顿定律 (第4章, 3 周)
- [[100-牛顿第一定律|牛顿第一定律]]：惯性的量度 = 质量
- [[229-牛顿第二定律|牛顿第二定律]]：F = ma（矢量方程，合力决定加速度）
- [[228-牛顿第三定律|牛顿第三定律]]：作用力与反作用力等大反向
- [[237-超重与失重|超重与失重]]：a 向上 → 超重；a 向下 → 失重
- 单位制（SI）：N = kg·m/s²

总课时约 11 周 ≈ 1 学期前半段.

## 重点公式与定理

| 公式 / 定理 | 适用场景 | 注意事项 |
|---|---|---|
| v = v₀ + at | 匀变速直线运动速度 | a 可为负（减速）；方向一致才能直接代入 |
| x = v₀t + ½at² | 匀变速位移 | t 从运动开始计；x 是矢量 |
| v² = v₀² + 2ax | 不含时间的速度-位移关系 | 常用于求末速度或位移 |
| F = kx | 胡克定律 | 弹簧劲度系数 k；弹性限度内有效 |
| f = μN | 滑动摩擦力 | μ 是动摩擦因数；N 是法向力 |
| F合 = ma | 牛顿第二定律 | F 是合外力；方向与 a 相同 |

## 典型例题类型

1. **纸带分析题**：用逐差法求加速度；判断是否匀变速
2. **受力分析 + 运动学联立**：已知力求运动参数（如滑块沿斜面运动）
3. **追及相遇问题**：甲匀速、乙匀加速，何时相遇 / 最近距离
4. **超重失重判断**：电梯加速 / 减速时，体重计示数变化
5. **摩擦力方向判断**：物体受到的摩擦力是静摩擦还是滑动摩擦，方向如何

## 易错点 / 难点

- **位移与路程混淆**：位移是矢量（起点到终点的有向线段），路程是标量（实际轨迹长度），折返运动时两者差异最大
- **加速度方向与速度方向的关系**：加速度向与速度方向相同 → 加速；相反 → 减速；加速度不为零不代表速度大
- **摩擦力方向**：静摩擦力方向与相对运动趋势相反（不一定与运动方向相反）；常见错误：认为摩擦力必然阻碍运动
- **牛顿第三定律与平衡力的区别**：作用力反作用力 = 等大反向同种力作用在两个物体上；平衡力 = 等大反向不同种力作用在同一物体上
- **矢量合成方向性**：两力夹角越大，合力越小；合力可小于任一分力（当夹角为钝角时）
- **自由落体的条件**：初速度为零且只受重力；空气阻力不可忽略时不适用
