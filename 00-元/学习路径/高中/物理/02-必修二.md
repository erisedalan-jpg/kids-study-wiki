## 推荐学习路径

### 阶段 1: 抛体运动 (第5章, 2 周)
- 运动合成：水平匀速 + 竖直自由落体 → [[261-平抛运动|平抛运动]]
- 分解求解：x = v₀t；y = ½gt²；合速度 v = √(vₓ² + vᵧ²)
- 落地时间由高度决定，水平射程由初速度决定
- 关键实验：平抛运动轨迹验证（控制变量法）

### 阶段 2: 圆周运动 (第6章, 3 周)
- [[286-角速度|角速度]] ω = 2π/T = 2πf；[[283-线速度|线速度]] v = rω
- [[257-向心加速度|向心加速度]] a = v²/r = ω²r；指向圆心（矢量方向时刻改变）
- [[256-向心力|向心力]] F = mv²/r = mω²r（不是新的力，是合力提供）
- 典型模型：竖直圆周（最高点最低点受力）、弯道行驶、匀速圆周的"拉力 - 重力 - 摩擦力"分析

### 阶段 3: 万有引力 (第7章, 3 周)
- [[246-万有引力定律|万有引力定律]] F = GMm/r²；G = 6.67×10⁻¹¹ N·m²/kg²
- 卫星圆周运动：GMm/r² = mv²/r → v = √(GM/r)
- [[281-第一宇宙速度|第一宇宙速度]] v₁ = 7.9 km/s（最小发射速度）
- [[264-开普勒三定律|开普勒三定律]]：椭圆轨道、面积相等、T² ∝ r³
- 地球表面 g = GM/R²；g 与 G 的换算

### 阶段 4: 机械能 (第8章, 3 周)
- [[062-功-物理|功]]：W = Fs cosθ（力与位移夹角）；功是标量
- [[251-动能定理|动能定理]]：W合 = ΔEk = ½mv² - ½mv₀²
- [[116-重力势能|重力势能]] Ep = mgh（参考面可自选）；重力做功与路径无关
- [[085-机械能守恒-初中|机械能守恒定律]]：E机 = Ek + Ep = 常数（只有重力做功）
- [[063-功率|功率]] P = W/t = Fv；额定功率与最大速度

### 阶段 5: 动量 (第9章, 3 周)
- [[346-动量|动量]] p = mv（矢量）；[[345-冲量|冲量]] I = Ft（矢量）
- [[349-动量定理|动量定理]]：F合·Δt = Δp = mv - mv₀（矢量方程）
- [[347-动量守恒定律|动量守恒定律]]：系统不受外力或外力之和为零 → p总 = 常数
- 碰撞类型：弹性碰撞（动能守恒）/ 非弹性碰撞（动能减少）/ 完全非弹性碰撞（最大动能损失）

总课时约 14 周 ≈ 1 学期.

## 重点公式与定理

| 公式 / 定理 | 适用场景 | 注意事项 |
|---|---|---|
| x = v₀t，y = ½gt² | 平抛运动分量 | 水平初速 v₀，竖直初速为零 |
| v = rω，a = v²/r | 匀速圆周运动 | a 指向圆心；v 方向时刻改变 |
| F = GMm/r² | 万有引力 | r 为质心距离；点质量或球体均适用 |
| W合 = ΔEk | 动能定理 | W 含所有力的功（含摩擦力）|
| Ek + Ep = 常数 | 机械能守恒 | 只有重力做功时成立 |
| F合·Δt = Δp | 动量定理 | 矢量方程；可分方向列分量式 |

## 典型例题类型

1. **平抛运动求落点**：给定初速度和高度，求水平距离与落地速度方向
2. **竖直圆周运动**：最高点 / 最低点绳（轨道）的拉力，临界条件（最高点 N=0）
3. **卫星轨道计算**：已知轨道半径求速度、周期、加速度
4. **机械能守恒综合**：滑块从斜面滑下，有无摩擦的速度对比
5. **碰撞问题**：子弹射入木块，用动量守恒求末速度，再用能量判断碰撞类型

## 易错点 / 难点

- **向心力不是新力**：向心力是合力（或合力的一个分量），不应在受力分析时单独添加"向心力"
- **平抛运动的时间**：落地时间只由初始高度决定，与水平初速度无关
- **万有引力与重力**：地球表面附近万有引力 ≈ 重力（忽略地球自转）；严格意义上重力是万有引力的分力
- **机械能守恒条件**：有摩擦力做功或有弹性力做功时机械能不守恒；弹性势能也属于机械能
- **动量守恒方向性**：动量是矢量，需统一正方向；碰后速度出现负值表示方向反转
- **功率的两个公式**：P = W/t（平均功率）vs P = Fv（瞬时功率，v 为瞬时速度）
