## 推荐学习路径

### 阶段 1: 静电场 (第10章, 4 周)
- [[279-电荷|电荷]]守恒；摩擦起电、感应起电、接触起电
- [[263-库仑定律|库仑定律]] F = kq₁q₂/r²（k = 9×10⁹ N·m²/C²）
- [[277-电场强度|电场强度]] E = F/q（定义式）；E = kq/r²（点电荷场）
- [[308-电势|电势]]与[[163-电压|电势差]]：U_AB = φ_A - φ_B = W_AB/q；顺电场线电势降低
- [[282-等势面|等势面]]：与电场线垂直；沿等势面移动电场力不做功
- [[313-电容器|电容器]] C = Q/U；平行板电容器 C = εS/4πkd

### 阶段 2: 电路分析 (第11章, 3 周)
- [[141-欧姆定律-初中|欧姆定律]] I = U/R（注意适用条件：纯电阻电路）
- 电阻率 ρ：R = ρL/S；温度对电阻率的影响
- [[123-串联电路|串联电路]]：R总 = R₁ + R₂；电流相同；电压按比例分配
- [[139-并联电路|并联电路]]：1/R总 = 1/R₁ + 1/R₂；电压相同；电流按比例分配
- [[307-电动势|电动势]] ε = E内 + U外；内阻 r；路端电压随电流增大而减小
- [[159-电功率|电功率]] P = UI = I²R = U²/R；[[154-焦耳定律|焦耳定律]] Q = I²Rt

### 阶段 3: 电磁场基础 (第12章, 3 周)
- [[187-磁场|磁场]]：磁感线；[[323-磁感应强度|磁感应强度]] B（T，特斯拉）；右手螺旋定则
- [[188-磁场对通电导体的作用|安培力]] F = BIL（导线在磁场中受力）；方向：左手定则
- [[304-洛伦兹力|洛伦兹力]] F = qvB（带电粒子在磁场中受力）；方向：左手定则；洛伦兹力不做功
- 带电粒子在匀强磁场中做匀速圆周运动：r = mv/qB
- 电磁感应初步：变化的磁场产生电场（定性，定量在选必二）

总课时约 10 周 ≈ 1 学期后半段.

## 重点公式与定理

| 公式 / 定理 | 适用场景 | 注意事项 |
|---|---|---|
| F = kq₁q₂/r² | 库仑定律 | 真空中点电荷；r 为两电荷间距 |
| E = F/q = kq/r² | 电场强度 | 定义式适用任何场；点电荷场公式仅用于点电荷 |
| U = W/q，W = qEd | 电势差与功 | d 为沿电场线方向的位移分量 |
| I = U/R | 欧姆定律 | 纯电阻；非线性元件（灯泡等）不严格适用 |
| ε = U外 + Ir | 全路欧姆定律 | r 为电源内阻；短路时 U外 = 0 |
| F = BIL（或 qvB）| 安培力 / 洛伦兹力 | 方向用左手定则；洛伦兹力不做功 |

## 典型例题类型

1. **库仑定律 + 受力平衡**：两等量异种电荷的受力合力；三电荷平衡位置
2. **电场线与等势面综合**：判断电场力做功正负；电势高低比较；带电粒子在电场中的加速 / 偏转
3. **复杂电路分析**：多电阻混联求等效电阻、各支路电流电压
4. **电源内外电路**：已知 ε、r，变化外阻时路端电压和电流的变化
5. **带电粒子在磁场中的圆周运动**：求轨道半径、偏转角、出射位置

## 易错点 / 难点

- **电场强度与电场力的区别**：E = F/q 中 E 与试探电荷 q 无关；E 是场的属性，不随 q 变化
- **电势与电势差**：高电势不等于强电场；等势面上移动不做功，但仍存在电场力（只是合位移沿等势面）
- **欧姆定律适用范围**：仅限纯电阻（灯泡、热敏电阻等非线性元件 I-U 图不是直线）
- **安培力方向 vs 洛伦兹力**：两者均用左手定则，但安培力对象是通电导线，洛伦兹力对象是运动电荷
- **洛伦兹力不做功**：始终垂直于速度方向，只改变速度方向，不改变速度大小（动能不变）
- **内阻影响**：路端电压 U = ε - Ir，电流越大（外阻越小），路端电压越低，此为最常见混淆点
