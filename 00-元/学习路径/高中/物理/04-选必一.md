## 推荐学习路径

### 阶段 1: 动量深化 (第1章, 2 周)
- 回顾必修二动量定理 → 深化：变力的冲量（Ft 取平均力或用积分思想）
- [[碰撞类型]]的判断方法：计算碰后动能，比较碰前；Δp=0 时用守恒
- [[爆炸与反冲]]：内力做功 → 动能增加（火箭、枪击）
- [[质心]]概念；质心运动定理：系统合外力 = 总质量 × 质心加速度

### 阶段 2: 机械振动 (第2章, 3 周)
- [[373-简谐运动|简谐运动]]的特征：回复力 F = -kx（弹簧振子）；加速度与位移反向
- 振幅 A、周期 T、频率 f 的关系：T = 1/f
- [[803-弹簧振子|弹簧振子]]周期：T = 2π√(m/k)；与振幅无关（等时性）
- [[350-单摆|单摆]]周期：T = 2π√(L/g)；与质量、振幅（小角度下）无关
- [[354-受迫振动|受迫振动]]：驱动频率 = 固有频率 → [[344-共振|共振]]（应用与危害）
- 能量：势能 ↔ 动能（平衡位置动能最大；端点势能最大）

### 阶段 3: 机械波 (第3章, 3 周)
- [[363-横波与纵波|横波]]（质点振动方向 ⊥ 波传播方向）vs [[363-横波与纵波|纵波]]（质点振动方向 ∥ 波传播方向）
- 波速 v = λf = λ/T；波速由介质决定，频率由振源决定
- [[366-波的干涉|波的干涉]]：相干波（同频率、固定相位差）叠加 → 加强 / 减弱条纹
- [[衍射]]：障碍物或缝隙尺寸与波长相当时明显
- [[357-多普勒效应|多普勒效应]]：声源靠近 → 频率偏高；远离 → 频率偏低（应用：测速雷达、医用超声）

### 阶段 4: 光的本性 (第4章, 3 周)
- [[360-折射率|折射率]] n = sin θ₁ / sin θ₂ = c/v；n 与介质有关
- [[343-全反射|全反射]]条件：光从密介质到疏介质，入射角 ≥ 临界角 C（sin C = 1/n）
- 光导纤维原理 = 全反射
- [[340-光的干涉|光的干涉]]（杨氏双缝）：条纹间距 Δy = Lλ/d；用于精密测量波长
- [[342-光的衍射-高中|光的衍射]]：单缝衍射；衍射使光的直线传播有限制（刀刃 / 小孔）
- [[338-光的偏振|光的偏振]]：横波才有偏振；偏振片可验证光的横波性
- 激光：单色性、相干性、方向性强

总课时约 11 周 ≈ 选择性必修阶段.

## 重点公式与定理

| 公式 / 定理 | 适用场景 | 注意事项 |
|---|---|---|
| T = 2π√(m/k) | 弹簧振子周期 | m 为振子质量；k 为弹簧劲度系数 |
| T = 2π√(L/g) | 单摆周期 | L 为摆长；仅限小角度（<5°）|
| v = λf | 波速-波长-频率 | 波速由介质决定；频率由振源决定 |
| n = c/v = sin θ₁/sin θ₂ | 折射率 | c 为光在真空中速度；v 为光在介质中速度 |
| sin C = 1/n | 全反射临界角 | n 为光密介质折射率；θ₁ > C 则全反射 |
| Δy = Lλ/d | 杨氏双缝条纹间距 | L 为双缝到屏距离；d 为缝间距 |

## 典型例题类型

1. **弹簧振子 + 动量守恒综合**：弹簧相连两物体碰撞后的振动分析
2. **单摆周期设计**：利用 T = 2π√(L/g) 估算 g；控制变量法实验设计
3. **波的图象分析**：从波形图读取波长 / 振幅；判断振动方向（用"同侧法"）
4. **双缝干涉**：已知条纹间距和装置参数反求波长
5. **全反射临界角计算**：光导纤维、棱镜内全反射的角度判断

## 易错点 / 难点

- **简谐运动的加速度方向**：始终指向平衡位置（与位移反向），非运动方向；在端点时速度为零但加速度最大
- **波速与振动速度混淆**：波速是波形传播的速度（由介质决定）；质点振动速度是质点本身的运动速度，两者截然不同
- **干涉与衍射的条件**：干涉需要相干波（同频率固定相位差）；衍射是单个波的现象；两者可同时存在
- **折射率恒大于 1（对光密介质）**：光从真空进入任何介质速度都减慢；n ≥ 1
- **全反射只发生在光密→光疏界面**：不能倒置（光疏→光密时不会全反射）
- **多普勒效应的频率变化**：是频率（音调 / 颜色）的变化，不是振幅（响度）的变化
