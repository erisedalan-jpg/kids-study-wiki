## 推荐学习路径

### 阶段 1: 电磁感应深化 (第1章 + 第2章, 4 周)
- 回顾安培力 → [[325-磁通量|磁通量]] Φ = BS cosθ（B、S 均为矢量方向要匹配）
- [[395-法拉第电磁感应定律|法拉第电磁感应定律]]：ε = ΔΦ/Δt（平均）；ε = BLv（导体切割）
- [[393-楞次定律|楞次定律]]：感应电流产生的磁场阻碍磁通量的变化（"来拒去留"口诀）
- 右手定则（切割法则）vs 楞次定律（通量变化法则）的统一
- 典型模型：导轨上的滑动导体棒；螺线管接近 / 远离磁铁

### 阶段 2: 交变电流 (第3章, 4 周)
- 交流发电机原理：矩形线圈在匀强磁场中匀速转动 → u = U_m sin(ωt)
- [[624-有效值|有效值]]定义：热效应等效；U = U_m/√2 ≈ 0.707 U_m；I = I_m/√2
- [[电感]]：自感系数 L（H 亨利）；电感阻碍电流变化；感抗 X_L = ωL
- [[312-电容|电容]]在交流中：容抗 X_C = 1/(ωC)；高频时容抗小（通高频阻低频）
- [[381-变压器|变压器]]：n₁/n₂ = U₁/U₂；n₁I₁ = n₂I₂（理想变压器，功率守恒）
- [[414-远距离输电|远距离输电]]：升压减小电流 → 减少线路热损耗 P = I²R

### 阶段 3: 电磁振荡与电磁波 (第4章, 2 周)
- [[376-LC振荡电路|LC振荡电路]]：电容放电 → 电感充电；电场能 ↔ 磁场能周期转换
- 固有频率 f = 1/(2π√(LC))；改变 L 或 C 可调频
- [[176-电磁波|电磁波]]的产生：振荡电路 → 开放回路（天线）→ 向空间辐射
- 电磁波在真空中速度 c = 3×10⁸ m/s；波速、频率、波长关系 c = λf
- [[408-电磁波谱|电磁波谱]]（频率从低到高）：无线电波 → 微波 → 红外线 → 可见光 → 紫外线 → X射线 → γ射线

总课时约 10 周 ≈ 选择性必修阶段.

## 重点公式与定理

| 公式 / 定理 | 适用场景 | 注意事项 |
|---|---|---|
| Φ = BS cosθ | 磁通量 | θ 为 B 与面法线夹角；均匀磁场 |
| ε = ΔΦ/Δt | 法拉第定律（平均） | Δt 为时间间隔；ΔΦ 含正负号 |
| ε = BLv | 导体切割磁场线 | L 为有效长度；v ⊥ B 且 v ⊥ L |
| U = U_m/√2 | 交流有效值 | 纯正弦波；非正弦波需重新计算 |
| n₁/n₂ = U₁/U₂ | 变压器电压比 | 理想变压器（无损耗）|
| f₀ = 1/(2π√(LC)) | LC振荡固有频率 | 调节 L 或 C 改变频率（收音机调谐原理）|

## 典型例题类型

1. **导体棒在轨道上运动**：已知 B、v、L，求感应电动势、感应电流、安培力（判断运动是否匀速）
2. **楞次定律判断电流方向**：磁铁接近 / 远离螺线管时线圈中电流方向
3. **交流电有效值计算**：给定峰值求有效值；利用有效值计算功率和热量
4. **变压器输电问题**：给定发电机电压、输电线电阻，求升压比使损耗最小
5. **LC 电路能量分析**：某时刻电容带电量或电感电流，判断电场能与磁场能的大小关系

## 易错点 / 难点

- **楞次定律的"阻碍"含义**：阻碍的是磁通量的变化，不是磁通量本身；磁通量增加 → 感应磁场与原磁场反向（减弱增加）
- **有效值的定义**：必须是热效应等效（均方根值）；不能简单取平均值（正弦函数平均值为零）
- **电感"通直流阻交流"**：直流稳定时电感不阻碍；交流时感抗 X_L = ωL 随频率增大而增大
- **变压器的功率条件**：理想变压器功率守恒（P₁ = P₂）；升压 → 电流减小（非升压 → 电流也升大）
- **电磁波的传播**：电磁波可在真空中传播（与机械波不同）；速度 c 在真空中恒定，在介质中减慢
- **LC电路的能量转换**：与弹簧振子类比（电场能 ↔ 弹性势能；磁场能 ↔ 动能）；电容电压最大时磁场能为零
