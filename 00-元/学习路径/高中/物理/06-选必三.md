## 推荐学习路径

### 阶段 1: 分子动理论 (第1章, 2 周)
- [[分子大小]]估算：油膜法 d = V/S（球状分子直径约 10⁻¹⁰ m）
- 分子间的相互作用力：引力与斥力同时存在；平衡距离 r₀
- [[130-分子热运动|布朗运动]]：悬浮粒子不规则运动 → 体现液体分子的热运动（非分子本身运动）
- 温度是分子[[热运动]]平均动能的量度；内能 = 分子动能 + 分子势能

### 阶段 2: 气体规律 (第2章, 3 周)
- [[444-玻意耳定律|玻意耳定律]] pV = 常数（等温）；[[430-查理定律|查理定律]] p/T = 常数（等容）；[[447-盖吕萨克定律|盖-吕萨克定律]] V/T = 常数（等压）
- [[446-理想气体状态方程|理想气体状态方程]] pV/T = nR（R = 8.31 J/(mol·K)）；T 必须用开尔文（T = t + 273 K）
- 气体压强的微观解释：大量分子碰撞器壁的平均效果
- 实验方法：控制变量法验证各气体定律（注意密闭气体、缓慢变化条件）

### 阶段 3: 热力学定律 (第3章, 2 周)
- [[440-热力学第一定律|热力学第一定律]]：ΔU = W + Q（做功和热传递均可改变内能）；能量守恒的普遍形式
- [[441-热力学第二定律|热力学第二定律]]：热量不自动从低温流向高温；自然过程具有方向性（熵增原理）
- 热机原理：从高温热源吸热 Q₁，对外做功 W，向低温热源放热 Q₂；效率 η = W/Q₁ < 1
- 卡诺热机效率（了解）：η_最大 = 1 - T₂/T₁

### 阶段 4: 原子结构 (第4章, 3 周)
- 氢原子光谱：线状谱 → 能级不连续
- [[443-玻尔原子模型|玻尔模型]]：电子只在特定轨道运行；轨道能量量子化；E₁ = -13.6 eV（基态）
- 能级跃迁：吸收光子（跃迁到高能级）/ 辐射光子（跃迁到低能级）；hν = E_高 - E_低
- [[417-光电效应|光电效应]]：光照射金属 → 逸出电子；存在截止频率；爱因斯坦方程 E_k = hν - W
- 光的波粒二象性：干涉衍射 → 波动性；光电效应 → 粒子性

### 阶段 5: 原子核与核物理 (第5章, 3 周)
- 核的组成：质子数 Z + 中子数 N；质量数 A = Z + N
- [[放射性衰变]]：α 衰变（²⁴He 射出，Z-2 A-4）；β 衰变（电子射出，Z+1 A 不变）；γ 射线（伴随 α/β，不改变核素）
- [[424-半衰期|半衰期]] T₁/₂：放射性核素数量减半所需时间；N = N₀(½)^(t/T₁/₂)
- [[449-质能方程|质能方程]] E = mc²（c = 3×10⁸ m/s）；结合能与质量亏损 Δm
- [[140-核能|核裂变]]：重核分裂（铀-235 + 中子）→ 中等质量核 + 中子 + 能量；链式反应
- [[140-核能|核聚变]]：轻核合并（氘 + 氚）→ 氦 + 中子 + 能量；需极高温度（热核反应）
- 核物理应用：核电站（受控裂变）、射线医疗（伽马刀、PET）、放射性同位素示踪

总课时约 13 周 ≈ 选择性必修阶段.

## 重点公式与定理

| 公式 / 定理 | 适用场景 | 注意事项 |
|---|---|---|
| pV = 常数（等温） | 玻意耳定律 | T 不变；密闭气体 |
| pV/T = C（恒量） | 理想气体状态方程 | T 用开尔文（K）；非绝对零度无意义 |
| ΔU = W + Q | 热力学第一定律 | W 为外界对系统做的功；Q 为吸收热量 |
| E_k = hν - W | 爱因斯坦光电方程 | h = 6.626×10⁻³⁴ J·s；W 为逸出功 |
| hν = E_高 - E_低 | 能级跃迁辐射频率 | 光子能量等于两能级能量差 |
| E = mc² | 质能方程 | Δm 为质量亏损；E 为释放能量 |
| N = N₀(½)^(t/T) | 放射性衰变 | T 为半衰期；t 为经历时间 |

## 典型例题类型

1. **气体状态变化综合**：气柱密封问题（竖直管 / 倾斜管）；利用状态方程求末态参量
2. **玻尔模型能级计算**：已知能级差求辐射光子频率；判断光子能否被吸收
3. **光电效应参数计算**：已知频率求最大初动能；已知截止频率求逸出功
4. **核反应方程配平**：根据质量数守恒（A 守恒）和电荷数守恒（Z 守恒）填写未知粒子
5. **半衰期计算**：已知初始质量和经历时间（为半衰期整数倍），求剩余放射性物质质量

## 易错点 / 难点

- **布朗运动的本质**：是液体（气体）分子热运动的间接体现，不是分子本身在做布朗运动；颗粒越小，布朗运动越明显
- **温度必须用开尔文**：气体定律中 T 必须是热力学温度（K）；用摄氏度代入会导致错误结果
- **热力学第一定律符号规定**：各版教材符号可能不同（ΔU = W + Q 或 ΔU = Q - W），考试前需确认教材规定
- **光电效应的截止频率**：截止频率由金属的逸出功决定，与光强无关；增大光强只增加光电子数目，不增加最大初动能
- **核反应中的守恒律**：质量数守恒（非质量守恒！质量有亏损）；电荷数守恒；这两个守恒律是配平核方程的唯一依据
- **裂变与衰变的区别**：衰变是自发过程（放射性）；裂变是受中子轰击的人为诱发过程（可受控）
