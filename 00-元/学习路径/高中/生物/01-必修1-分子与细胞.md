## 推荐学习路径

### 阶段 1: 走近细胞 (第1章, 1 周)
- 生命活动离不开细胞 → 单细胞与多细胞生物
- 细胞学说的建立（施莱登、施旺）→ [[细胞学说]]
- 原核细胞（无核膜，如细菌、蓝藻）vs 真核细胞（有核膜）→ [[原核细胞]] [[真核细胞]]
- 典型图示：原核细胞与真核细胞结构对比图

### 阶段 2: 细胞的化学组成 (第2章, 2 周)
- 组成细胞的元素：C、H、O、N 为主，微量元素
- 水的存在形式：自由水 vs 结合水；自由水/结合水比值与代谢
- 糖类：单糖（葡萄糖、果糖、核糖、脱氧核糖）/ 二糖 / 多糖（淀粉、糖原、纤维素）→ [[糖类]]
- 脂质：脂肪（储能）、磷脂（膜结构）、固醇（胆固醇、性激素、维生素D）→ [[脂质]]
- 蛋白质：氨基酸结构通式 → 脱水缩合 → 肽键 → 蛋白质结构层次 → 蛋白质功能多样性 → [[蛋白质]]
- 核酸：DNA（脱氧核苷酸，含T不含U，双链）/ RNA（核糖核苷酸，含U不含T，单链）→ [[核酸]] [[DNA]] [[RNA]]
- 典型实验：蛋白质检测（双缩脲）、还原糖检测（斐林）、脂肪检测（苏丹III/IV）、DNA/RNA检测（甲基绿/吡罗红）

### 阶段 3: 细胞的基本结构 (第3章, 2 周)
- 细胞膜的组成（磷脂双分子层 + 蛋白质）与流动镶嵌模型 → [[细胞膜]] [[流动镶嵌模型]]
- 各细胞器功能：
  - 线粒体：有氧呼吸的主要场所（双层膜）→ [[线粒体]]
  - 叶绿体：光合作用场所（双层膜，含基粒）→ [[叶绿体]]
  - 内质网：蛋白质合成与运输（粗面）/ 脂质合成（光面）
  - 高尔基体：蛋白质加工、分类与发送（"邮局"）
  - 核糖体：蛋白质合成场所（无膜）
  - 液泡：调节细胞渗透压（植物细胞为主）
  - 中心体：动物及低等植物细胞分裂时纺锤体形成
  - 溶酶体：细胞内消化
- 细胞核：核膜（双层，有核孔）/ 核仁 / 染色质 → [[细胞核]]
- 典型图示：动植物细胞亚显微结构模式图；各细胞器分工协作（分泌蛋白合成与分泌流程图）

### 阶段 4: 物质输入和输出 (第4章, 2 周)
- 渗透作用原理：半透膜 + 浓度差 → 水分子定向移动 → [[渗透作用]]
- 植物细胞的质壁分离与质壁分离复原实验
- 三种跨膜运输方式对比：

| 方式 | 浓度方向 | 载体蛋白 | 能量 | 举例 |
|------|--------|---------|------|------|
| 自由扩散 | 高→低 | 无 | 不耗能 | O₂、CO₂、H₂O、酒精 |
| 协助扩散 | 高→低 | 有 | 不耗能 | 葡萄糖进入红细胞 |
| 主动运输 | 低→高 | 有 | 耗能 | 离子吸收、葡萄糖进入小肠上皮细胞 |

- 胞吞与胞吐（大分子物质）→ 体现细胞膜流动性

### 阶段 5: 酶与ATP (第5章前半, 1 周)
- 酶的本质：大多数是蛋白质（少数为RNA）→ [[酶]]
- 酶的特性：高效性、专一性、作用条件温和
- 影响酶活性的因素：温度（最适温度）/ pH / 底物浓度
- ATP 结构：腺嘌呤 + 核糖 + 3 个磷酸基团；高能磷酸键 → [[ATP]]
- ATP-ADP 循环：能量的储存与释放
- 典型实验：探究温度/pH 对酶活性的影响（对照实验设计）

### 阶段 6: 细胞呼吸 (第5章中段, 2 周)
- 有氧呼吸三阶段：
  1. 糖酵解（细胞质基质）：葡萄糖 → 丙酮酸 + 少量ATP + NADH
  2. 丙酮酸氧化（线粒体基质）：丙酮酸 + H₂O → CO₂ + NADH
  3. 氧化磷酸化（线粒体内膜）：NADH + O₂ → H₂O + 大量ATP
- 无氧呼吸（细胞质基质）：葡萄糖 → 酒精 + CO₂（植物/酵母菌）或乳酸（动物/乳酸菌）
- 有氧 vs 无氧呼吸能量利用率对比 → [[细胞呼吸]] [[有氧呼吸]] [[无氧呼吸]]
- 典型实验：酵母菌呼吸方式探究（澄清石灰水 + 溴麝香草酚蓝）

### 阶段 7: 光合作用 (第5章后段, 3 周)
- 光合作用场所：叶绿体（基粒类囊体薄膜 + 叶绿体基质）
- 光反应（类囊体薄膜）：
  - 光能 → ATP + NADPH（活跃化学能）
  - 水的光解：2H₂O → 4H⁺ + 4e⁻ + O₂
- 暗反应/卡尔文循环（叶绿体基质）：
  - CO₂固定：CO₂ + C₅（RuBP）→ 2C₃
  - C₃还原：C₃ + ATP + NADPH → C₅ + (CH₂O)
- 影响光合速率的因素：光照强度、CO₂浓度、温度
- 光合速率 vs 呼吸速率：光补偿点、光饱和点、净光合量与总光合量
- C₃植物 vs C₄植物（选学）→ [[光合作用]] [[光反应]] [[暗反应]]
- 典型图示：光合作用与呼吸作用物质变化流程图；光合速率与呼吸速率的光照-坐标图

### 阶段 8: 细胞的生命历程 (第6章, 3 周)
- 细胞周期：间期（DNA复制、蛋白质合成）+ 分裂期（前中后末）
- 有丝分裂各时期特征 → 染色体数 / DNA数变化曲线 → [[有丝分裂]]
- 减数分裂（生殖细胞形成）：减数第一次分裂（同源染色体分离）/ 减数第二次分裂（染色单体分开）→ [[减数分裂]]
- 有丝分裂 vs 减数分裂：图像识别（染色体数目/四分体/同源染色体）
- 细胞分化：基因的选择性表达 → 细胞全能性（越低分化越高全能性越低）→ [[细胞分化]] [[细胞全能性]]
- 细胞衰老：端粒缩短、酶活性降低
- 细胞凋亡：程序性死亡（正常生命现象）
- 细胞癌变：原癌基因/抑癌基因突变 → 无限增殖 → [[细胞癌变]]
- 典型图示：有丝分裂各时期细胞图；减数分裂图像系列；DNA与染色体数量变化坐标图

总课时约 36 周（高一必修1全年或一学期集中学习）。

## 典型例题类型

1. **实验检测题**：双缩脲/斐林/碘液/苏丹III试剂的使用条件与颜色变化（注意：斐林需水浴加热，双缩脲不需加热）
2. **细胞器功能与分布题**："哪些细胞器含有DNA"、"分泌蛋白经过哪些结构"（注：植物细胞无中心体、无溶酶体说法需核对）
3. **物质运输判断题**：根据浓度差/是否耗能/是否需载体蛋白判断运输类型
4. **光合呼吸计算题**：给定光照强度下CO₂释放量/吸收量，计算净光合量、总光合量、积累有机物量
5. **分裂图像识别题**：根据细胞图判断分裂类型（有丝/减I/减II）、所处时期、染色体数目
6. **酶特性实验设计题**：设置对照组，控制单一变量，分析温度/pH对酶活性的影响

## 易错点

1. **自由水/结合水**：代谢旺盛的细胞自由水比例高；种子萌发时自由水/结合水比值升高
2. **ATP高能磷酸键数目**：ATP有2个高能磷酸键，ADP有1个，AMP有0个
3. **无氧呼吸产物**：植物/酵母菌产酒精+CO₂；动物/乳酸菌只产乳酸（无CO₂）
4. **光合与呼吸场所**：光反应在类囊体薄膜（非叶绿体基质）；糖酵解在细胞质基质（非线粒体）
5. **减数分裂四分体**：四分体 = 1对同源染色体 = 2条染色体 = 4条染色单体（1四分体≠4条染色体）
6. **细胞分化与DNA**：分化后各细胞DNA相同（基因相同）；基因的选择性表达导致细胞形态功能不同
7. **癌变**：癌变是基因突变（体细胞），不可遗传给后代
