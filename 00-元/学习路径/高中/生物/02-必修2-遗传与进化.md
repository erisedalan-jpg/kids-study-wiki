## 推荐学习路径

### 阶段 1: 孟德尔遗传定律 (第1章, 3 周)
- 豌豆作为实验材料的优点（严格自花传粉、相对性状明显）
- 一对相对性状的杂交实验：P → F₁ → F₂；显性性状 vs 隐性性状
- 分离定律：等位基因在形成配子时分离（减数分裂时同源染色体分离）→ [[基因分离定律]]
- 两对相对性状的杂交实验：9:3:3:1 的来源
- 自由组合定律：非同源染色体上的非等位基因自由组合 → [[基因自由组合定律]]
- 遗传图解规范书写（P、F₁、F₂、配子比例）
- 概率计算：加法原理（互斥事件）/ 乘法原理（独立事件）

### 阶段 2: 基因与染色体 (第2章, 2 周)
- 减数分裂与基因传递：同源染色体携带等位基因 → 减I分离
- 伴性遗传：X染色体连锁遗传（色觉、血友病）→ 遗传特点（交叉遗传）
- 性别决定：XY型（人类）/ ZW型（鸟类）
- 伴X显性 vs 伴X隐性：遗传图解分析 → [[276-伴性遗传|伴性遗传]]
- 常染色体遗传 vs 伴性遗传判断依据
- 典型图示：减数分裂中染色体与基因的对应关系图

### 阶段 3: DNA的结构与复制 (第3章, 2 周)
- 格里菲斯肺炎双球菌体内实验 → 艾弗里体外转化实验（DNA是遗传物质）
- 赫尔希-蔡斯噬菌体侵染实验（同位素标记法）→ [[遗传物质的证明]]
- DNA双螺旋结构（沃森-克里克，1953年）：
  - 两条链反向平行；碱基配对：A-T（2个氢键）、G-C（3个氢键）
  - 磷酸-脱氧核糖骨架在外，碱基在内
- DNA半保留复制：亲代链作模板；拓扑异构酶/解旋酶/DNA聚合酶参与 → [[285-DNA复制|DNA复制]]
- 半保留复制计算：n次复制后含亲代链的DNA个数（始终为2）
- 典型图示：DNA双螺旋模型图；半保留复制示意图

### 阶段 4: 基因的表达 (第4章, 2 周)
- 中心法则：DNA → (转录) → mRNA → (翻译) → 蛋白质 → [[295-中心法则|中心法则]]
- 转录（细胞核 + 线粒体/叶绿体）：以DNA一条链为模板；原料为四种核糖核苷酸；产物为mRNA
- 翻译（核糖体）：以mRNA为模板；密码子（mRNA上）/ 反密码子（tRNA上）/ 氨基酸对应关系
- 密码子特点：64种密码子（61种编码氨基酸 + 3种终止密码子）；简并性（多个密码子对应同一氨基酸）
- 基因表达的计算：
  - 碱基数目：mRNA碱基数 = 蛋白质氨基酸数 × 3（忽略终止密码子）
  - 氢键数目：AT间2个，GC间3个
- 基因与性状的关系：一基因多效、多基因一性状；环境影响基因表达 → [[基因与性状]]
- 典型图示：转录与翻译同步进行的多聚核糖体图

### 阶段 5: 变异与育种 (第5、6章, 3 周)
- 基因突变：DNA碱基对的增添/缺失/替换 → 等位基因的来源（不定向性）→ [[301-基因突变|基因突变]]
- 基因重组：减数分裂时交叉互换（四分体时期）+ 非同源染色体自由组合 → [[302-基因重组|基因重组]]
- 染色体变异：
  - 染色体结构变异（缺失/重复/倒位/易位）
  - 染色体数目变异：整倍体（多倍体）/ 非整倍体（三体、单体）→ [[304-染色体变异|染色体变异]]
- 变异类型判断：可遗传变异（基因突变/重组/染色体变异）vs 不可遗传变异（环境引起的表现型改变）
- 育种方法比较：

| 方法 | 原理 | 年限 | 优缺点 |
|------|------|------|--------|
| 杂交育种 | 基因重组 | 较长（6-7年） | 可综合优良性状；年限长 |
| 诱变育种 | 基因突变 | 较短 | 不定向；有益突变少 |
| 单倍体育种 | 染色体数目变异（花粉离体培养+秋水仙素处理） | 最短（2年） | 快速纯合；操作复杂 |
| 多倍体育种 | 染色体数目变异（秋水仙素处理） | — | 果实大产量高；高度不育 |
| 基因工程育种 | 基因重组（跨物种） | — | 定向改造；伦理争议 |

- 单倍体育种原理：花粉（单倍体植株，n）→ 秋水仙素（抑制纺锤体，染色体数加倍）→ 纯合二倍体（2n）
- 多倍体：西瓜（3n无子）、八倍体小黑麦；秋水仙素作用机理：抑制纺锤体形成，染色体不能移向两极

### 阶段 6: 现代生物进化理论 (第7章, 2 周)
- 拉马克进化学说：用进废退（获得性遗传）— 历史意义，现已修正
- 达尔文自然选择学说：过度繁殖、遗传变异、生存斗争、适者生存 → [[311-自然选择学说|自然选择学说]]
- 现代综合进化论核心要点：
  - 进化的实质：种群基因频率的定向改变
  - 进化的原材料：突变和重组
  - 进化的方向：自然选择（决定基因频率变化方向）
  - 物种形成的必要条件：隔离（地理隔离 → 生殖隔离）
- 基因频率计算（Hardy-Weinberg定律应用，待家长核对是否高考要求）→ [[基因频率]]
- 共同祖先说：所有生物共同起源于原始祖先（中性表述）
- 人类起源：古猿进化（直立行走、脑容量增大、语言与工具）— 采用主流科学表述
- 物种的形成：隔离是关键；不同物种间存在生殖隔离
- 典型图示：自然选择（抗药性细菌）频率变化图；地理隔离与生殖隔离示意图

总课时约 36 周（高一或高二学习）。

## 典型例题类型

1. **遗传图解书写题**：根据亲本基因型写出配子、F₁/F₂基因型比例与表现型比例（注意显隐性判断）
2. **概率计算题**：多对基因自由组合时，某表现型概率 = 各单对性状概率之积（乘法原理）
3. **伴性遗传判断题**：根据遗传特征判断是常染色体遗传还是伴X染色体遗传（女患父女必患/男患母女必患等口诀验证）
4. **DNA复制计算题**：n次复制后：DNA总数 = 2ⁿ；含亲代链DNA = 2；某种碱基总量变化
5. **转录翻译计算题**：已知mRNA碱基数/氨基酸数，计算对应DNA碱基数、氢键数、ATP消耗等
6. **育种方法选择题**：根据育种目标（快速纯合/跨物种改造/多产）选择最佳育种方法并说明原理

## 易错点

1. **自由组合定律适用条件**：非同源染色体上的非等位基因；连锁在同一染色体上的基因不符合自由组合（需考虑连锁与互换）
2. **伴X遗传书写**：基因型写法 X^A X^a（上标为等位基因），Y染色体上不标基因（或标 X^A Y）
3. **半保留复制亲代链**：无论复制多少代，含亲代链的DNA始终为2个（各含1条亲代链）
4. **密码子在mRNA上**：密码子是mRNA上的三联体；反密码子在tRNA上；与DNA上的碱基序列互补（但不是密码子）
5. **基因突变不定向**：自然条件下基因突变频率很低（10⁻⁸~10⁻⁵）；突变方向不固定；多为有害突变
6. **基因重组vs染色体变异**：四分体时期交叉互换是基因重组（非染色体变异）；易位（非同源染色体间片段交换）才是染色体结构变异
7. **进化的实质**：进化 = 基因频率改变（非个体进化）；物种形成必须有生殖隔离
