## 推荐学习路径

### 阶段 1: 内环境与稳态 (第1章, 1 周)
- 内环境的组成：血浆 / 组织液 / 淋巴液（细胞外液的三种形式）→ [[313-内环境|内环境]]
- 细胞内液 vs 细胞外液：各占体液比例
- 内环境各成分的来源与转化：血浆 ⇌ 组织液 ⇌ 淋巴液
- 内环境理化性质：pH（7.35-7.45）/ 温度（37°C左右）/ 渗透压
- 稳态：内环境理化性质保持相对稳定的状态 → [[314-内环境稳态|稳态]]
- 稳态调节机制：神经-体液-免疫协调配合
- 稳态失调：血糖/体温/pH 异常举例
- 典型图示：内环境各成分关系图（血浆-组织液-淋巴循环图）

### 阶段 2: 神经调节 (第2章, 3 周)
- 神经元结构：细胞体 / 树突（接收信号）/ 轴突（传出信号）/ 髓鞘 / 突触小体 → [[089-神经元|神经元]]
- 静息电位：外正内负（K⁺外流）
- 动作电位：外负内正（Na⁺内流）→ 兴奋产生 → [[318-动作电位|动作电位]]
- 兴奋在神经纤维上的传导：双向传导（电信号）
- 突触结构：突触小体（突触前膜）/ 突触间隙 / 突触后膜 → [[326-突触|突触]]
- 突触处信号传递：电信号 → 化学信号（神经递质）→ 电信号
- 突触传递方向：单向（突触前 → 突触后），原因：神经递质只能从前膜释放
- 反射弧五要素：感受器 → 传入神经 → 神经中枢 → 传出神经 → 效应器 → [[085-反射弧（初中）|反射弧]]
- 反射：非条件反射（先天）vs 条件反射（后天），需完整反射弧
- 神经系统分级调控：脊髓（低级）→ 脑干 → 小脑 → 大脑皮层（高级）
- 大脑皮层功能区：语言中枢（运动/感觉/书写/视觉）
- 典型图示：神经元结构图；突触结构放大图；反射弧示意图

### 阶段 3: 体液调节 (第3章, 3 周)
- 体液调节概念：激素弥散到体液中，随血液运输到靶细胞/靶器官
- 激素特点：微量高效、通过体液运输、作用于靶细胞、不直接参与反应（作为信号分子）
- 主要内分泌腺与激素：

| 腺体 | 激素 | 主要功能 |
|------|------|----------|
| 下丘脑 | 促激素释放激素 / 抗利尿激素 | 调控垂体；调节水盐平衡 |
| 垂体 | 促甲状腺激素 / 生长激素 / 促性腺激素 | 调控靶腺 |
| 甲状腺 | 甲状腺激素（含碘） | 促进物质氧化分解；促进生长发育（幼年缺乏→呆小症） |
| 肾上腺髓质 | 肾上腺素 | 应激反应 |
| 胰岛A细胞 | 胰高血糖素 | 升血糖 |
| 胰岛B细胞 | 胰岛素 | 降血糖（唯一降糖激素） |
| 性腺 | 性激素（雌/雄性激素，固醇类） | 促进性腺发育与生殖 |

- 甲状腺激素分级调节与负反馈：下丘脑→垂体→甲状腺；甲状腺激素浓度升高→负反馈抑制下丘脑/垂体 → [[分级调节]] [[负反馈调节]]
- 血糖调节：
  - 血糖升高 → 胰岛素分泌增加（促进组织细胞摄取/利用/储存葡萄糖）
  - 血糖降低 → 胰高血糖素分泌增加（促进肝糖原分解、非糖物质转化为葡萄糖）
  - 胰岛素 vs 胰高血糖素：拮抗关系
  - 糖尿病主要类型（待家长核对最新课标要求）→ [[血糖调节]]
- 水盐调节：抗利尿激素（下丘脑合成/垂体释放）→ 促进肾小管/集合管重吸收水分
- 神经调节 vs 体液调节对比：速度/持续时间/作用范围
- 典型图示：甲状腺激素分级调节与负反馈图；血糖调节双向调节图

### 阶段 4: 免疫调节 (第4章, 3 周)
- 免疫的三道防线：
  1. 皮肤与黏膜（非特异性）
  2. 体液中的杀菌物质与吞噬细胞（非特异性）
  3. 特异性免疫（体液免疫 + 细胞免疫）
- 免疫系统组成：免疫器官（骨髓/胸腺/淋巴结/脾）/ 免疫细胞 / 免疫活性物质
- 抗原：引起机体产生特异性免疫反应的物质（多为蛋白质或糖蛋白）→ [[163-抗原|抗原]]
- 抗体：由浆细胞（效应B细胞）分泌的免疫球蛋白，与抗原特异性结合 → [[162-抗体|抗体]]
- 体液免疫流程：
  抗原 → 吞噬细胞（呈递）→ T细胞（活化）→ B细胞（增殖分化）→ 浆细胞（分泌抗体）+ 记忆B细胞
- 细胞免疫流程：
  抗原（胞内）→ T细胞活化 → 效应T细胞 → 与靶细胞密切接触 → 裂解靶细胞 + 记忆T细胞
- 记忆细胞：二次免疫应答快速增殖分化 → 免疫应答更强更快（疫苗原理）→ [[361-记忆细胞|记忆细胞]]
- 免疫失调：
  - 过敏反应（变态反应）：已免疫机体再次接触相同抗原；IgE参与；不会遗传
  - 自身免疫病：自身免疫系统攻击自身组织（如系统性红斑狼疮、类风湿性关节炎）
  - 免疫缺陷病：AIDS（HIV侵染T细胞）
- 典型图示：体液免疫与细胞免疫流程图；二次免疫应答抗体浓度图

### 阶段 5: 三大调节综合 (第5章, 1 周)
- 神经-体液-免疫三大调节网络的协调配合
- 应激反应（下丘脑-肾上腺轴）
- 稳态维持：各调节机制共同作用，维持内环境相对稳定
- 综合情境分析（高考常考多调节机制综合题）

总课时约 18 周（高二选修上学期）。

## 典型例题类型

1. **内环境判断题**：判断某物质是否属于内环境（血浆/组织液/淋巴液中的成分算内环境；细胞内/消化道内/膀胱内不算）
2. **神经传导方向判断题**：根据突触结构判断兴奋传导方向；分析某处受损后反射弧功能变化
3. **激素分级调节分析题**：给定激素浓度异常，推断调节轴哪个环节出问题；负反馈分析
4. **血糖调节计算/分析题**：给定情景（饭后/运动后/糖尿病患者），分析相关激素变化及靶细胞反应
5. **免疫流程分析题**：区分体液免疫与细胞免疫的适用情景；分析二次免疫应答机制（疫苗原理）
6. **综合调节情境题**：涉及神经+体液双重调节（如体温调节、血糖-神经-激素联合）

## 易错点

1. **内环境成分边界**：血液 ≠ 内环境（血液中血细胞内的成分不是内环境）；消化液、尿液不属于内环境
2. **兴奋传导方向**：神经纤维上双向传导；突触处单向传递（前→后）；注意区分
3. **抗利尿激素来源**：下丘脑合成，由垂体释放（非垂体合成）
4. **胰岛素分泌刺激**：血糖升高是主要刺激，神经调节也参与（不能只写血糖升高）
5. **浆细胞不能识别抗原**：浆细胞只分泌抗体，不能识别抗原，也不能增殖（记忆细胞才能）
6. **过敏反应 vs 自身免疫**：过敏反应是对外来抗原的过度反应；自身免疫病是攻击自身正常细胞/组织
7. **效应T细胞直接接触靶细胞**：细胞免疫中效应T细胞通过密切接触发挥作用，非分泌抗体
