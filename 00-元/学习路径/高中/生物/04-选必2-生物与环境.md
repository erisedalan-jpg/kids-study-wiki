## 推荐学习路径

### 阶段 1: 种群及其动态 (第1章, 2 周)
- 种群定义：同一地域、同一物种的全部个体集合 → [[368-种群|种群]]
- 种群的数量特征：出生率/死亡率（决定种群密度）/ 年龄组成（增长/稳定/衰退型）/ 性别比例
- 种群密度调查方法：
  - 样方法（植物/活动能力弱的动物）：随机取样
  - 标志重捕法（活动能力强的动物）：N = M × n / m → [[365-标志重捕法|标志重捕法]]
- 种群增长模型：
  - "J"型增长：食物/空间无限，无天敌；λ（增长倍数）= 常数；指数增长 → [[J型增长]]
  - "S"型增长：自然界有限条件下；K值（最大环境容纳量）；增长速率在K/2时最大 → [[926-S型增长|S型增长]] [[367-环境容纳量|K值]]
- K值应用：捕鱼/害虫防治（捕至K/2时种群恢复最快；灭害虫降低K值）
- 影响种群数量变化的因素：非密度制约因素（温度/光照）/ 密度制约因素（食物竞争/天敌）
- 典型图示：J型与S型增长曲线对比；年龄组成三种类型图；种群密度与增长速率坐标图

### 阶段 2: 群落及其演替 (第2章, 3 周)
- 群落定义：同一地域内各种生物种群的集合 → [[374-群落|群落]]
- 群落的空间结构：
  - 垂直结构：植物的分层现象（光照决定）；动物分层（食物/栖息空间）
  - 水平结构：种群呈镶嵌分布
- 种间关系类型：

| 关系 | 特征 | 举例 |
|------|------|------|
| 捕食 | 一方得利，一方受害；数量波动相互依存 | 狼与兔 |
| 竞争 | 争夺相同资源；一般弱者被排除（竞争排除原理） | 农作物与杂草 |
| 互利共生 | 相互依存，互惠互利 | 豆科植物与根瘤菌；地衣 |
| 寄生 | 一方得利，另一方受害（宿主不立即死亡） | 蛔虫与人；噬菌体与细菌 |

- 种群增长与群落的关系（待家长核对课标最新表述）
- 群落演替：
  - 初生演替：裸岩/沙漠等从未有生物生长的地方；历时长（裸岩→地衣→苔藓→草本→灌木→乔木）
  - 次生演替：原有群落被破坏后的恢复；历时较短（弃耕农田→草原→灌木→乔木）→ [[407-群落演替|群落演替]]
- 演替方向：一般朝向更复杂、更稳定的群落方向（人类活动可改变方向）
- 生物多样性的形成与维持：物种多样性高 → 群落稳定性高（营养结构复杂）
- 典型图示：垂直结构示意图；捕食关系数量波动图；初生/次生演替对比图

### 阶段 3: 生态系统及其稳定性 (第3章, 3 周)
- 生态系统组成：非生物物质和能量（无机环境）/ 生产者 / 消费者 / 分解者 → [[020-生态系统|生态系统]]
- 食物链与食物网：只从生产者（植物）写起；只写捕食关系；不写非生物成分和分解者 → [[028-食物链|食物链]] [[027-食物网|食物网]]
- 能量流动：
  - 起点：生产者固定的太阳能（总初级生产量）
  - 特点：单向流动、逐营养级递减（传递效率约10%-20%）
  - 计算：消耗最多能量（最低传递效率）/ 消耗最少能量（最高传递效率）
  - 能量去向：自身呼吸消耗；流向下一营养级；流向分解者（含残落物）
  - 生态效率应用：最短食物链消耗能量最少 → [[385-生态系统能量流动|能量流动]]
- 物质循环（生物地化循环）：
  - 碳循环：CO₂→（光合/化能合成）→有机碳→（呼吸/分解）→CO₂；化石燃料燃烧加速碳释放
  - 氮循环：固氮菌/根瘤菌（N₂→NH₃）/ 硝化细菌（氨氧化）/ 反硝化细菌（NO₃⁻→N₂）→ [[387-碳循环|碳循环]] [[850-氮循环|氮循环]]
  - 物质循环与能量流动的区别：物质循环（循环往复）vs 能量流动（单向递减）
- 信息传递：物理信息（光/声/温度）/ 化学信息（信息素/挥发物）/ 行为信息 → 调节种群繁殖、种间关系
- 生态系统稳定性：
  - 抵抗力稳定性：抵抗外界干扰并维持原状的能力（物种多样性高 → 抵抗力强）
  - 恢复力稳定性：受破坏后恢复到原状的能力（一般与抵抗力负相关）→ [[396-抵抗力稳定性|抵抗力稳定性]] [[恢复力稳定性]]
  - 负反馈调节是维持生态系统稳定的基础
- 典型图示：能量流动桑基鱼塘模式图；碳循环示意图；抵抗力/恢复力负相关坐标图

### 阶段 4: 人与环境 (第4章, 2 周)
- 人类活动对生态环境的影响：过度捕捞、栖息地破坏、外来物种入侵、环境污染
- 生物多样性三个层次：基因（遗传）多样性 / 物种多样性 / 生态系统多样性 → [[144-生物多样性|生物多样性]]
- 生物多样性的价值：直接价值（食用/药用/工业原料）/ 间接价值（生态功能）/ 潜在价值
- 生物多样性保护措施：
  - 就地保护（建立自然保护区）— 最根本有效的措施
  - 迁地保护（动物园/植物园/基因库）
  - 法律保护
- 外来物种入侵：无天敌制约 → 种群J型增长 → 破坏原生态系统
- 可持续发展：满足当代需求，不损害后代满足其需求的能力
- 典型图示：生物多样性三层次关系图

## 典型例题类型

1. **标志重捕法计算题**：给定M、m、n，计算种群密度N；分析误差来源（标记影响死亡率、被捕概率）
2. **K值应用题**：捕鱼在K/2时捕获量最大；害虫防治应在K/2之前防治并尽量降低K值
3. **种间关系判断题**：根据两种生物种群数量变化曲线，判断捕食/竞争/互利共生关系
4. **能量流动计算题**：给定某营养级同化量，按10%-20%传递效率计算上/下级能量；注意"最多/最少"对应效率选择
5. **物质循环分析题**：在碳循环/氮循环图中补全箭头；分析温室效应与碳循环的关系
6. **稳定性分析题**：比较不同生态系统的抵抗力与恢复力；分析物种减少对稳定性的影响

## 易错点

1. **种群密度 vs 种群数量**：密度 = 数量/面积，二者不等同；调查方法针对不同生物类型选择
2. **K/2的增长速率最大**：S型增长中增长速率（dN/dt）在K/2最大，增长率（dN/dt/N）一直下降
3. **食物链书写**：只包含生产者和消费者；不包含分解者；箭头方向从猎物→捕食者（能量流动方向）
4. **能量计算最多/最少**：消耗最多→选最低传递效率10%、最长食物链；消耗最少→选最高传递效率20%、最短食物链
5. **初生 vs 次生演替**：判断依据是原来有无土壤（或有无植被存在条件），非是否有生物
6. **抵抗力与恢复力的负相关**：通常情况下成立（热带雨林抵抗力强恢复力弱），但北极苔原是例外（抵抗力弱恢复力也弱）
7. **物种多样性 ≠ 数量多**：多样性是物种种类的丰富程度，非某一物种数量
