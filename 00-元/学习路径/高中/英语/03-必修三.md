## 推荐学习路径

### 阶段 1: 节日文化与定语从句综合 (U1, 2 周)
- 必修一/二定语从句综合复习：限制性 vs 非限制性定语从句
- 非限制性定语从句：which 指代整个句子（逗号隔开）
- 节日起源文化词汇：origin, mythology, custom, ritual, lantern, carnival
- 写作：对比两种节日文化的说明文

### 阶段 2: 品德与状语从句 (U2, 3 周)
- [[617-状语从句|状语从句]]系统梳理：
  - 原因：because/since/as
  - 目的：so that/in order that + 情态动词
  - 结果：so...that/such...that
  - 让步：although/though/even though/even if/however/no matter what
- 名人品质词汇：integrity, perseverance, courage, compassion, dedication
- 写作：人物评价议论文（名人的品质与贡献）

### 阶段 3: 多元文化与虚拟语气 (U3, 3 周)
- [[445-虚拟语气-入门|虚拟语气]]重点：
  - wish + 宾语从句：wish I were/had done/would do
  - if only（要是...就好了）
  - as if/as though（好像）
  - 条件句虚拟：if I were you...; if I had known...
- 多元文化词汇：diversity, integration, stereotype, prejudice, tolerance, multicultural
- 写作：文化差异反思文（尊重多元文化）

### 阶段 4: 太空探索与倒装 (U4, 3 周)
- [[328-文言倒装句|倒装句]]：
  - 否定副词前置：Never/Seldom/Rarely/Hardly/Not until + 倒装
  - Only 前置：Only in this way can we...
  - 部分倒装 vs 完全倒装
- 太空词汇：spacecraft, orbit, astronaut, satellite, gravity, cosmos, launch
- 写作：科技说明文（太空探索的意义）

### 阶段 5: 强调句与综合 (U5, 2 周)
- [[强调句型]]：It is/was + 被强调部分 + that/who + 其余部分
- 与分裂句区别；强调时间/地点/原因/方式状语
- 期末专项：虚拟语气 + 倒装 + 强调句综合练习
- 写作：演讲稿（关于金钱与价值观的演讲）

总课时约 13 周 ≈ 1 学期。

| 语法点 | 本册核心内容 | 高考考查方式 |
|---|---|---|
| **从句** | 状语从句（原因/目的/结果/让步）；非限制性定语从句 | 完形上下文逻辑；长句分析 |
| **虚拟语气** | wish/if only/as if + 虚拟；条件句（与现在/过去事实相反）| 语法填空；单句改写 |

| **强调** | It is/was...that/who... 强调句 | 单句改写；判断强调还是形式主语 |

## 易错点 / 难点

- **非限制性定语从句 which 指代句子**: The sun rises in the east, which is known to all.（which 指代整个主句内容；不可用 that 替换非限制性定语从句）
- **虚拟语气时态对应**:
  - 与现在相反：if + 过去时（were/did）→ 主句 would/could/might + do
  - 与过去相反：if + 过去完成时（had done）→ 主句 would/could/might + have done
  - wish + 从句：对现在遗憾 → were/did；对过去遗憾 → had done
- **倒装 only 的使用条件**: Only + 状语（副词/介词短语/状语从句）前置才倒装；only + 主语前置不倒装（Only Tom knows.）
- **强调句 vs 形式主语**: It is important that... 是形式主语；It is...that/who...（去掉 It is...that 句子不完整）才是强调句
- **让步状语从句 although vs however**: although/though 是连词（接从句）；however 是副词（连接句子，需加分号或句号）
