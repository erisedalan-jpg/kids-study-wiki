## 推荐学习路径

### 阶段 1: 人物品质与动名词 (U1, 2 周)
- [[动名词]]系统：-ing 作主语（Reading is fun）/ 宾语 / 表语 / 介词宾语
- 动名词 vs 不定式作宾语：动名词强调习惯/既成事实；不定式强调将来/具体
- 人物特质词汇：ambitious, determined, creative, innovative, influential, pioneering
- 写作：成功人物分析议论文（结合具体事例论证）

### 阶段 2: 未来与发明 (U2, 2 周)
- 将来时态辨析：
  - will + do（临时决定/预测）
  - be going to + do（计划/有迹象）
  - be about to + do（即将）
  - 现在进行时表将来（确定安排）
  - 一般现在时表将来（时刻表/程序）
- 发明词汇：patent, prototype, innovation, breakthrough, engineer, devise
- 写作：未来科技预测说明文

### 阶段 3: 长城与情态推测 (U3, 3 周)
- 情态动词推测用法（现在推测）：
  - must be（一定是）/ can't be（不可能是）/ may/might be（可能是）
- 情态动词推测（过去推测）：
  - must have done / can't have done / may/might have done
- 沟通与长城文化词汇：communicate, barrier, civilisation, fortification, architectural
- 写作：解释说明文（长城的历史意义）

### 阶段 4: 过去分词作状语 (U4, 3 weeks)
- [[过去分词作状语]]：
  - 表原因：Exhausted by the journey, he fell asleep.
  - 表方式：Written in simple English, the book is easy to understand.
  - 表条件：Given more time, we could finish the project.
  - 独立主格结构：The task completed, we went home.
- 智慧与经典词汇：wisdom, philosophy, inspiration, quote, enlightenment
- 写作：读书笔记/读后感（结合引用与分析）

### 阶段 5: 主谓一致与农业 (U5, 2 周)
- [[主谓一致]]特殊情况：
  - 集合名词（team/class/family 单复数视语境）
  - either...or/neither...nor（就近原则）
  - as well as / together with（主语决定）
  - 不定代词（everyone/nobody 单数）
- 农业合作词汇：harvest, sustainable, agriculture, irrigation, cooperation, yield
- 期末复习：选必一语法体系综合（动名词/情态推测/分词状语）

总课时约 12 周 ≈ 1 学期。

## 典型题型

1. **完形填空**: 人物传记（成功人士奋斗历程）；动词形式选择（动名词/不定式）
2. **阅读理解**: 科技报道（发明创造类）；文化介绍（长城/中国历史）
3. **语法填空**: 情态动词推测（must have done / can't have done）；分词状语填写
4. **书面表达·议论文**: 成功需要哪些品质？结合名人事例（150 词）
5. **书面表达·说明文**: 介绍一项重要发明对生活的影响（150 词）

## 易错点 / 难点

- **动名词 vs 不定式作宾语辨析**:
  - 只接动名词：enjoy/finish/avoid/suggest/mind/keep/consider/practice
  - 只接不定式：decide/want/hope/wish/plan/agree/refuse/promise
  - 两者均可但意义不同：remember/forget/stop/regret/try/mean
- **情态动词过去推测**: must have done（对过去肯定推测）/ can't have done（否定推测）/ may/might have done（不确定推测）；易混淆：should have done 表"本应该做但没做"（含批评/遗憾，非推测）
- **过去分词作状语的主语**: 分词的隐含主语必须与句子主语一致（悬垂分词是常见错误）
- **主谓一致就近原则**: either A or B / neither A nor B / not only A but also B → 动词随 B 变化；not A but B → 动词随 B 变化
- **将来时态选择**: be about to（不与具体时间状语连用）；be going to（有计划/有迹象）；will（纯预测/当场决定）
