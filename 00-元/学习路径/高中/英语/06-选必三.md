## 推荐学习路径

### 阶段 1: 艺术与虚拟语气综合 (U1, 3 周)
- [[445-虚拟语气-入门|虚拟语气]]综合深化：
  - 条件句三种类型（真实/与现在相反/与过去相反）
  - wish + 宾语从句（三种时态虚拟）
  - as if / as though + 虚拟
  - would rather + 宾语从句（would rather sb. did/had done）
  - if it were not for / if it hadn't been for（虚拟混合）
- 艺术鉴赏词汇：aesthetic, visual, rhythm, tone, symbolism, metaphor, irony
- 写作：艺术评论文（描述一件艺术品并分析其象征意义）

### 阶段 2: 道德伦理与名词性从句 (U2, 2 周)
- 名词性从句深化：复杂语境中的嵌套与变形
  - 主语从句后置（形式主语 It is...that）
  - whoever/whatever/wherever 等复合关系词引导名词性从句
- 道德伦理词汇：integrity, conscience, dilemma, obligation, accountability, empathy, virtue
- 写作：道德两难议论文（呈现困境 → 分析 → 立场）

### 阶段 3: 战争与和平——文学视角 (U3, 3 周)
- [[倒装句]]深化：
  - 否定意义词前置完全梳理：not/never/seldom/rarely/hardly/scarcely/no sooner...than/not until
  - so/nor/neither 倒装（So do I. / Neither can she.）
  - only 倒装的各种形式
  - 完全倒装（表示强调位置/存在的 here/there/out/in）
- 文学手法分析词汇：narrative, imagery, metaphor, allegory, symbolism, irony, foreshadowing
- 写作：文学评论文（分析战争题材文学作品的叙事手法与文化反思）
  - 聚焦：作者如何通过文学手法传达对战争的反思，避免涉及当代政治判断

### 阶段 4: 神话童话与 as 定语从句 (U4, 2 周)
- as 引导定语从句：as is known to all / as is often the case / as was reported
  - as 与 which 引导非限制性定语从句的区别（as 可置于主句前后/which 只置于后）
- 神话童话词汇：mythology, legend, deity, mortal, fate, prophecy, enchant, quest
- 写作：神话故事改写 / 道德寓意分析

### 阶段 5: 文学探索与省略结构 (U5, 3 周)
- [[省略结构]]：
  - 不定式省略（to 的保留）：I'd like to, but I don't have time. / Will you go? I hope to.
  - 状语从句省略：When (she was) a child, she loved reading.
  - 比较状语从句省略
- 文学探索词汇：venture, expedition, chronicle, account, documentation, witness
- 期末综合：选必三语法体系（虚拟/倒装/省略）综合写作提分
- 写作：探险叙事文（叙事视角分析）

总课时约 13 周 ≈ 1 学期。

## 跨文化背景

- **U1**: 西方经典艺术作品（Monet/Picasso/Dali）的文化背景；东西方艺术美学对比（写意 vs 写实）
- **U2**: 西方哲学伦理传统（Aristotle 的美德伦理/Kant 的义务伦理）在日常表达中的体现
- **U3**: 英语战争文学经典（Hemingway 《A Farewell to Arms》/ Sassoon 战争诗歌）的文学手法；重点关注文学对战争的人道主义反思
- **U4**: 希腊神话体系（Zeus/Apollo/Athena）及其在英语词汇中的遗存；中国神话与西方神话的叙事结构对比
- **U5**: 英语探险文学传统（Darwin 的航行记录/Shackleton 的南极探险）

## 典型题型

1. **完形填空**: 文学改编叙事文；道德困境故事（逻辑连贯选词）
2. **阅读理解**: 艺术评论文（推断作者观点）；神话文化介绍（细节题/词义题）
3. **语法填空**: 虚拟语气混合条件句；否定词倒装；省略结构识别
4. **书面表达·评论文**: 分析一部文学作品中对战争的描写手法（150 词，聚焦文学技巧）
5. **书面表达·议论文**: 艺术鉴赏的价值——艺术教育重要吗？（150 词）

## 易错点 / 难点

- **虚拟语气混合条件句**: If I had studied harder (过去虚拟), I would be successful now (现在虚拟)；注意时间参照点切换
- **would rather 虚拟**: would rather + 动词原形（主语自身）vs would rather + 宾语从句 + 过去式/过去完成时（他人）
- **否定词前置倒装精确规则**: no sooner...than / hardly/scarcely...when（not yet 时态：过去完成时 + 一般过去时）；Hardly had he left when it began to rain.
- **so/neither 倒装**: 前句肯定 → So + 助/be/情 + 主语；前句否定 → Neither/Nor + 助/be/情 + 主语；注意时态一致性
- **as 引导非限制性定语从句 vs which**: as 可置于主句前（As is known, ...）；which 只能置于主句后；as 常用固定搭配（as is known/as is often the case/as was mentioned）
