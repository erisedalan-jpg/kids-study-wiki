## 推荐学习路径

### 阶段 1: 现代诗歌鉴赏（第一单元，3 周）
- [[沁园春·长沙]] → 词的格律、意象群、豪放风格
- 郭沫若《立在地球边上放号》→ 自由诗节奏与意象
- 闻一多《红烛》→ 象征手法；诗人精神
- 昌耀《峨日朵雪峰之侧》→ 现代诗的跳跃性与张力
- 雪莱《致云雀》→ 外国诗意象比较；浪漫主义风格
- 写作：仿写一首现代短诗（12 行以上）

### 阶段 2: 新闻通讯（第二单元，2 周）
- 通讯特征：真实性 + 文学性 + 时效性
- [[焦裕禄]] → 典型人物通讯；细节选取
- 袁隆平（《喜看稻菽千重浪》）→ 人物通讯结构
- 《芣苢》《插秧歌》→ 古诗中的劳动意象；诵读节奏
- 写作：为身边劳动者写一篇 400 字人物速写

### 阶段 3: 小说精读（第三单元，3 周）
- [[百合花]]：情节结构、百合花被的象征意义、战争与人性
- [[哦，香雪]]：细节描写、铁凝笔法、少女心理刻画
- 对比：两篇中国小说的叙事视角与情感基调
- 肖洛霍夫《一个人的遭遇》→ 战争小说；外国叙事风格
- 写作：分析小说中的一个细节，写文学短评草稿

### 阶段 4: 散文经典（第四单元，3 周）
- [[故都的秋]]：郁达夫；以悲凉写秋；情景交融
- [[荷塘月色]]：朱自清；通感修辞；意境营造
- [[我与地坛]]（节选）：史铁生；生死哲思；散文哲理化
- [[赤壁赋]]：苏轼；骈散结合；主客问答结构；重点实词虚词
- 《登泰山记》：姚鼐；游记结构；方位词运用
- 五篇连读：归纳"写景抒情散文"三种结构模式

### 阶段 5: 整本书阅读——《乡土中国》（第五单元，3 周）
- 精读第 1-4 章（差序格局 / 礼治秩序 / 无为政治 / 长老统治）
- 摘录核心概念 → 制作概念卡 → [[差序格局]] [[礼治秩序]]
- 学术文章阅读方法：勾画中心句；标注论证结构
- 写作：用"差序格局"分析一个身边的社会现象（600 字议论文）

### 阶段 6: 文言文精读（第六单元，3 周）
- [[劝学]]：荀子；论证结构（比喻论证）；文言实词虚词积累
- [[师说]]：韩愈；批驳论证；"从师"主题；常考句式
- 古诗词诵读：背诵全部规定篇目，逐句落实字词
- 文言复习：虚词 "之/乎/者/也/而/以/于/其" 系统归纳

### 阶段 7: 文学短评写作（第七单元，2 周）
- 文学短评要素：中心论点 + 文本证据 + 鉴赏语言
- 结构示范：总-分-总；引文 + 分析 + 评价
- 练习：为本册任意一篇选文写 600 字文学短评

总课时约 19 周 ≈ 1 学期。

|---|---|---|
| 之 | 《劝学》《师说》《赤壁赋》 | 助词（的）/ 动词（往）/ 代词 / 取独 |
| 而 | 《劝学》《师说》 | 顺承 / 转折 / 修饰 / 递进 |
| 以 | 《师说》《赤壁赋》 | 用 / 因为 / 凭借 / 来 |

| 于 | 《劝学》《师说》 | 在 / 比 / 向 / 被 |
| 乎 | 《师说》《赤壁赋》 | 疑问语气 / 叹息 / 于（介词用法） |

### 本册重点实词（常考）
- **劝学**：木直中绳（中：符合）/ 声非加疾（疾：强）/ 用心一也（一：专一）
- **师说**：师道之不传（道：风尚）/ 圣人无常师（常：固定的）/ 不耻相师（耻：以…为耻）
- **赤壁赋**：冯虚御风（冯：通"凭"）/ 白露横江（横：漫布）/ 正襟危坐（危：端正）

## 整本书阅读：《乡土中国》

- **作者**：费孝通（社会学家，1910-2005）
- **性质**：学术著作；描述中国农村社会结构
- **核心概念**：
  - 差序格局（vs. 西方团体格局）
  - 礼治秩序（传统道德规范）
  - 无为政治（长老权威）
  - 名实分离（文字下乡）
- **阅读策略**：
  1. 先读目录，理解全书框架（14 章）
  2. 逐章标注概念定义句
  3. 制作"概念 → 对应农村现象"二列笔记
  4. 重点精读：差序格局、礼治秩序、无为政治 3 章
- **高考考查方向**：非连续性文本阅读；学术文章信息筛选；概念辨析

## 典型例题类型

1. **现代诗歌鉴赏**：分析意象作用；赏析修辞手法（通感/比喻/象征）；比较中外诗歌情感差异
2. **文言文翻译**：实词虚词推断；特殊句式翻译（《劝学》《师说》《赤壁赋》常考）
3. **散文阅读**：分析"情景交融"手法；概括散文主旨；赏析语言风格
4. **小说阅读**：分析人物形象；探究细节的象征意义；分析叙事视角效果
5. **文学短评写作**：选一篇选文，围绕某一角度（结构/手法/主题/语言）写 400-600 字短评
6. **通讯文本阅读**：筛选人物事迹信息；分析通讯语言特色
7. **整本书阅读**：解释费孝通某概念含义；用概念分析给定材料

## 易错点 / 难点

- **"之"字用法**：取独结构（"吾之学也"）与主谓之间"之"易与代词用法混淆；需逐句语境判断
- **赤壁赋主客关系**：主客实为苏轼一人的两种情绪，非真实对话——考生常误答为两人讨论
- **散文"形散神聚"**：须能指出"形散"的具体表现（跳跃的联想）和"神"的落脚点（中心情感）
- **通讯 vs. 新闻消息**：通讯有文学描写，可有作者议论；新闻消息以客观事实为主
- **词牌格律**：《沁园春》上下阕字数/韵脚不对称，不可套用律诗规则
- **《乡土中国》概念偷换**：差序格局强调"私人道德"而非"自私自利"，考生易混淆
- **文学短评审题**：题目若限定"某一手法"，不能泛泛而谈；需紧扣题目角度举证
