# 教材索引

本地教材库: `素材/教材/ChinaTextbook/` (45 学科版本, 287 PDF, 5.2 GB)

新增/升级词条时, 在「📑 出处与参考资料」「教材」行按学段+学科查表, 写具体 PDF 路径 + 章节. 不在表内的学科 (如初中科学) 写 `(无对应教材, 待家长补充)`.

## 引用示例

```markdown
- **教材**: [[素材/教材/ChinaTextbook/小学/数学/人教版/义务教育教科书 · 数学一年级上册.pdf]] 第三单元
```

学段-学科未明确时, 写最贴近主用学段的; 跨学段词条 (如《光》) 列多个 PDF.

## 词条 → 学段-学科映射

- `数学/` 词条: 按 frontmatter `学段` 取最早学段对应路径
- `英语/` 词条: 同上 (学前借用小学英语)
- `生活与社会/` 词条: 多映射到「道德与法治」(规则/品德类) 或「科学」(自然现象类) 或 「人文地理」(场所/职业类) 或多个
- 跨学段概念 (如《光》《数数》): 列多个 PDF
