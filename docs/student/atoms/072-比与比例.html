比与比例

一句话：比是两个量相除的关系，比例是两个比相等的等式——它们是连接乘除、方程和函数的桥梁。 English: A ratio compares two quantities by division; a proportion is an equation stating that two ratios are equal.

🧒 给 3-6 岁（共读版）

你和爸爸一起分苹果：你拿 2 个，爸爸拿 3 个。

我们说：你和爸爸的苹果数量是 2 比 3，写成 2:3。

这就是比（ratio）——它告诉我们两堆东西的关系是"2 份对 3 份"。

共读小活动： 拿出一些积木，试着让爸爸拿 2 个你拿 1 个，再试着两人各拿 4 个和 2 个——比一比，哪次的"比"是一样的关系？（2:1 和 4:2 是同一个比！）

📚 给 6-12 岁（自读版）

六上：比的意义与化简

比（ratio） 表示两个数相除的关系：

$$a:b = a \div b \quad (b \neq 0)$$

  • 前项（antecedent）：冒号左边的数（a）
  • 后项（consequent）：冒号右边的数（b）
  • 比值（ratio value）：前项 ÷ 后项的结果

例：小明走 6 步，小红走 9 步，比是 6:9，比值 = 6÷9 = 2/3。

化简比：分子分母同除公因数，化到最简。

6:9 → 前后项都除以 3 → 2:3（化简比）

比与分数、除法的关系：

$$6:9 = \frac{6}{9} = 6 \div 9$$

三者本质相同，只是写法不同！

按比分配： 把一个整体按照给定的比分成各份。

例：橙汁和水按 1:3 混合，共 400 mL，橙汁多少 mL？

总份数 = 1+3 = 4，每份 = 400÷4 = 100 mL，橙汁 = 1×100 = 100 mL

六下：比例与正反比例

比例（proportion） 是两个比相等构成的等式：

$$a:b = c:d \quad \text{即} \quad \frac{a}{b} = \frac{c}{d}$$

比例的基本性质（外项积 = 内项积）：

$$a:b = c:d \implies a \times d = b \times c$$

  • a、d 是外项（extremes）
  • b、c 是内项（means）

例：3:4 = 6:8，验证：3×8 = 4×6 = 24 ✓

用比例解方程（解比例）：

例：x:5 = 6:15，外项积=内项积：15x = 30，x = 2

正比例（direct proportion）：

$$y = kx \quad \text{即} \quad \frac{y}{x} = k \text{（常数）}$$

两个量的比值不变，就是正比例关系。判断方法：列出几组数据，看 y÷x 是否一样。

例：速度一定，路程 s 和时间 t 正比例：s = vt，s/t = v（常数）。

反比例（inverse proportion）：

$$xy = k \quad \text{（常数）}$$

两个量的乘积不变，就是反比例关系。

例：路程一定，速度 v 和时间 t 反比例：vt = s（常数）。

判断口诀： 两量相除是常数 → 正比例；两量相乘是常数 → 反比例。

正比例 反比例
关系式 y/x = k xy = k
图像 过原点的直线 双曲线
一个量增大 另一个也增大 另一个减小

🎓 给 12+（进阶版）

黄金比（Golden Ratio）φ = 0.618…

最著名的比是黄金比（golden ratio） φ（phi）：

$$\varphi = \frac{1 + \sqrt{5}}{2} \approx 1.618, \quad \frac{1}{\varphi} \approx 0.618$$

性质：$\varphi^2 = \varphi + 1$，即 φ 满足方程 $x^2 - x - 1 = 0$。

帕特农神庙、蒙娜丽莎、向日葵的螺旋排列都接近黄金比——它出现在斐波那契数列的相邻项之比：$1, 1, 2, 3, 5, 8, 13, \ldots$，相邻两项之比趋向 φ。

相似变换与比例不变性

几何相似（geometric similarity）：两个图形相似，当且仅当对应边之比相等，对应角相等。

$$\triangle ABC \sim \triangle DEF \implies \frac{AB}{DE} = \frac{BC}{EF} = \frac{CA}{FD} = k$$

k 称为相似比（similarity ratio）。面积之比 = 相似比的平方；体积之比 = 相似比的三次方。

这是正比例在几何中的深刻体现：相似变换保持所有角度和所有比例。

函数视角：正比例与反比例

  • 正比例 $y = kx$ 是一次函数的特殊形式（无常数项），图像是过原点的直线，k 为斜率。
  • 反比例 $y = k/x$ 是双曲线，在坐标系中关于原点对称，渐近线为两坐标轴。

反比例函数是代数曲线中最简单的，研究其性质是初学解析几何的入口。

连比与比例链

多个量的连比（compound ratio）：$a:b:c = 2:3:5$ 表示 a、b、c 的相互关系。

用处：调配饮料比例、化学配方、乐队音量调节……连比可以通分（找公倍数）组合：

若 $a:b = 2:3$ 且 $b:c = 4:5$，则 $a:b:c = 8:12:15$（b 统一为 12）。

🗣️ 家长讲解话术

  • 孩子可能问 "比和分数有什么区别？" → 可以说："数学上几乎是一回事——a:b 就等于分数 a/b。区别在于用法：比通常用来比较两个量的关系（如男女比例），分数通常表示部分与整体的关系（如吃掉了 3/4）。"

  • 孩子可能问 "正比例和反比例怎么区分？" → 可以说："一起变大就是正比例（买的越多花钱越多）；一个变大另一个变小、乘积不变就是反比例（速度快了时间短了，但路程一样）。"

  • 孩子可能问 "为什么外项积等于内项积？" → 可以说："a/b = c/d，两边同乘 bd，左边变 ad，右边变 bc，所以 ad = bc。这只是等式两边乘同一个数的结果！"

  • 共读小活动： 调橙汁：橙汁原液和水按 1:4 混合，量出 50 mL 原液，问需要多少水（200 mL），让孩子亲手量，体验按比分配的实际应用。
