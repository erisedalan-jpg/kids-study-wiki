负数

负数

一句话：比零还小的数叫负数，用负号"−"表示，数轴上在零的左边。 English: A negative number is any number less than zero, written with a minus sign (−), and sits to the left of zero on the number line.

🧒 给 3-6 岁（共读版）

冬天特别冷的时候，温度计里的红线会跑到"0"的下面去——那里写着 −1、−2、−3……

"零下五度"就是 −5°C，意思是比零还要冷 5 度！

还有一个好玩的例子：小猪存了 10 元零花钱，但是它买饼干花了 12 元，这时候它就"欠了" 2 元——可以记作 −2 元。

负数就是"比零还少"的数，数数的时候我们通常不用负数，但生活里到处都有它的身影。

共读小活动： 找一个有温度计图案的绘本或玩具温度计，一起找找 0 在哪里，然后手指往下数 −1、−2、−3，感受"零下"的概念。

📚 给 6-12 岁（自读版）

负数在哪里出现？

生活中有很多"对立"的情况，正好用正负数来表示：

情境 正数 负数
温度 零上 5°C → +5 零下 3°C → −3
海拔 珠峰顶 8848m → +8848 死海水面 −430m → −430
收支 收入 500 元 → +500 支出 200 元 → −200
方向 向东走 3km → +3 向西走 2km → −2

规律：选定一个参考点（零），"一个方向"为正，"相反方向"为负。

数轴：所有数的"家"

把所有数排在一条直线上，就叫数轴：

← 负数              正数 →
  ···  −3  −2  −1  0  +1  +2  +3  ···
                    ↑
                  原点 (零)

数轴的三个要素： 1. 原点：0 所在的点 2. 正方向：向右（箭头方向） 3. 单位长度：每格代表多少

越靠右，数越大——这是数轴最重要的规律！

正负数大小比较

用数轴比较很直观：

  • 正数 > 0 > 负数：所有正数都比零大，所有负数都比零小
  • −1 > −10：−1 在数轴上比 −10 靠右，所以 −1 更大
  • −0.5 > −3：道理相同

易错陷阱：负数的绝对值越大，这个负数反而越小！ - ∣−10∣ = 10 比 ∣−1∣ = 1 大，但 −10 < −1

相反数

两个数在数轴上距离原点相同，但方向相反，互为相反数：

  • +3 和 −3 互为相反数
  • +0.5 和 −0.5 互为相反数
  • 0 的相反数就是 0 本身

记法：一个数 a 的相反数就是 −a。

绝对值预告

一个数在数轴上离原点有多远，就叫这个数的绝对值，用 | | 表示：

  • ∣+3∣ = 3（正数的绝对值等于它本身）
  • ∣−3∣ = 3（负数的绝对值等于它的相反数）
  • ∣0∣ = 0（零的绝对值是零）

绝对值表示"距离"，距离永远 ≥ 0。初中会系统学习。

典型例题

例 1：用正负数表示下面的量： - 气温零上 7 度 → +7°C - 海拔低于海平面 150 米 → −150 m - 亏损 800 元 → −800 元

例 2：把 −5、+3、0、−1、+7 按从小到大排列： → −5 < −1 < 0 < +3 < +7（在数轴上从左往右）

🎓 给 12+（进阶版）

整数环：负数让加法完美

自然数（0, 1, 2, 3, …）引入负数后变成整数（…, −2, −1, 0, 1, 2, …）。

整数在加法和乘法下构成整数环（ℤ，ring）：

性质 说明
加法交换律 a + b = b + a
加法结合律 (a + b) + c = a + (b + c)
加法单位元 a + 0 = a
加法逆元（负数！） a + (−a) = 0
乘法交换律 a × b = b × a
分配律 a(b + c) = ab + ac

加法逆元的存在，正是负数最本质的代数意义：每个整数都有一个"对抗者"让它归零。

有序域与实数轴

实数集 ℝ 是一个有序域（ordered field）：不仅能做加减乘除，还能比较大小。数轴（实数轴）是这个有序结构的几何表示。

负数在数轴上的位置由全序关系确定：

对任意实数 a, b，有且仅有 a < b、a = b、a > b 三种情况之一成立。

复数：超越实数轴

虽然实数轴可以延伸到无穷负数，但 x² = −1 在实数中无解。引入虚数单位 i（i² = −1），构成复数 ℂ = a + bi，数轴变为复数平面（高斯平面）。

负数是实数轴上的点；复数是平面上的点，虚轴垂直于实轴。

物理中的正负方向

物理中，负号不仅表示"小于零"，还表示方向：

  • 电荷：质子带 +e，电子带 −e
  • 速度：向右为正，向左速度记为负值
  • 力：压力为正，张力为负（具体视坐标系定义）
  • 温度：热力学温度（开尔文）无负数，但摄氏度和华氏度有负数

负号在矢量中是"方向翻转"的编码，这是负数最深层的物理直觉。

数论中的负数与模运算

在模运算（钟表算术）中，−1 ≡ n − 1 (mod n)： - 在 mod 12 中，−1 ≡ 11（钟表上"少走1小时"等同于"多走11小时"）

这揭示了负数是循环结构中的逆元，广泛用于密码学（RSA 算法）和计算机科学（二进制补码表示负数）。

🗣️ 家长讲解话术

  • 孩子可能问 "负数是真实存在的吗？" → 可以说："当然！你摸一摸冬天零下的温度，欠别人的钱，海底的深度……这些都是负数在描述的真实情况。数学发明负数，就是为了表达'比零还少'的状态。"

  • 孩子可能问 "为什么 −10 比 −1 小？" → 可以说："画一条数轴，0 在中间，往左走 1 步到 −1，往左走 10 步才到 −10。在数轴上，越靠左的数越小——所以 −10 在 −1 的左边，−10 < −1。"

  • 孩子可能问 "负数加负数等于什么？" → 可以说："两个负数相加，结果更负。比如我欠了你 3 元，又欠了他 2 元，我一共欠了 5 元，就是 (−3) + (−2) = −5。"

  • 共读小活动： 做一条纸质数轴，从 −10 到 +10，用小磁铁或棋子当"棋子"，提问"零下3度再降低5度是多少度"，让孩子在数轴上移动棋子找答案（即 −3 − 5 = −8）。
