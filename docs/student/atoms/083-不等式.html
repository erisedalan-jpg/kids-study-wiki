不等式

权重20吉林命中7路径1学段[初中, 高中]主题[代数, 七上, 方程]

aliases: 不等式, inequality, 不等式预备, 三元均值不等式

课本一句话定义

用不等号（$>$、$<$、$\geq$、$\leq$、$\neq$）连接两个式子所组成的式子叫做不等式；使不等式成立的未知数的值叫做不等式的解；所有解组成解集。

📚 给 6-12 岁（小学高年级回溯版）

你已经认识了等式（$=$），现在认识不等式（$>$、$<$）。

  • $3 > 2$：3 大于 2，是不等式（确定成立）
  • $x + 1 > 5$：未知数 $x$ 满足什么条件时成立？（$x > 4$）

不等号的方向： - $>$：大于 - $<$：小于 - $\geq$：大于等于（"≥"，含等号） - $\leq$：小于等于（"≤"，含等号）

不等式和方程的区别： - 方程（$x + 1 = 5$）：解是一个点（$x = 4$） - 不等式（$x + 1 > 5$）：解是一段范围（$x > 4$，无数个解）

🎓 给 12+（进阶版）

定义

不等式（inequality）：含有不等号的数学表达式。

解集（solution set）：使不等式成立的所有值组成的集合；在数轴上用区间表示。

不等式的基本性质

性质 内容 注意
性质 1（加法） $a > b \Rightarrow a + c > b + c$ 两边加减同数，方向不变
性质 2（乘正数） $a > b, c > 0 \Rightarrow ac > bc$ 乘正数，方向不变
性质 3（乘负数） $a > b, c < 0 \Rightarrow ac < bc$ 乘负数，方向改变！
传递性 $a > b, b > c \Rightarrow a > c$

核心区别：不等式乘（除）负数时方向改变，这与等式不同！

一元一次不等式（预备）

标准形式：$ax + b > 0$（或 $< 0$，$\geq 0$，$\leq 0$）

解法与方程类似，但乘除负数时反向：

解 $-2x + 4 > 0$： $-2x > -4$（移项） $x < 2$（除以 $-2$，不等号反向）

解集：$x < 2$，数轴表示：在 2 的左边（空心圆，不含 2）。

典型例题

例 1：解不等式 $3x - 1 \geq 2x + 3$。

解：移项：$3x - 2x \geq 3 + 1$，$x \geq 4$。

解集：$x \geq 4$（数轴上 4 右边，含 4 实心圆）。

例 2：解不等式 $-\dfrac{x}{2} < 3$。

解：两边乘 $(-2)$，不等号反向：$x > -6$。

例 3：若 $a > b$，比较 $2a - 3$ 与 $2b - 3$ 的大小。

解：$a > b \Rightarrow 2a > 2b$（乘正数 2 方向不变）$\Rightarrow 2a - 3 > 2b - 3$（减同数方向不变）。

不等式 vs 方程

方程 $ax = b$ 不等式 $ax > b$
解 通常一个值 一段区间
乘负数 等号方向不变 不等号方向改变
数轴表示 一个点 射线或线段

知识联系

  • 前置：方程、等式性质、有理数大小比较
  • 后续：一元一次不等式（七下正式学习）→ 不等式组 → 线性规划（高中）
  • 高中拓展：绝对值不等式、二次不等式、分式不等式
