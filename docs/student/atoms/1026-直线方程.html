直线方程

一句话：在平面直角坐标系中，用一个关于 $x, y$ 的一次方程 $Ax + By + C = 0$（$A, B$ 不同时为 $0$）来表示一条直线的全体点集。 English: An equation of the form $Ax + By + C = 0$ ($A$ and $B$ not both zero) that represents all points on a straight line in a Cartesian plane.

🎓 考点精讲

定义与形式

直线的本质是平面内与一个方向向量或法向量相关的点集。其方程均为一元一次方程，高考核心考查五种形式及其互化：

形式 方程 适用条件
点斜式 $y - y_0 = k(x - x_0)$ 已知斜率 $k$ 及一点 $(x_0, y_0)$
斜截式 $y = kx + b$ 已知斜率 $k$ 及纵截距 $b$
两点式 $\frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1}$ 已知两点 $(x_1, y_1), (x_2, y_2)$，且 $x_1 \neq x_2, y_1 \neq y_2$
截距式 $\frac{x}{a} + \frac{y}{b} = 1$ 已知横截距 $a$、纵截距 $b$，且 $a \neq 0, b \neq 0$
一般式 $Ax + By + C = 0$ $A, B$ 不同时为 $0$，万能形式

其中，斜率 $k = \tan\alpha$（$\alpha$ 为倾斜角，$\alpha \in [0°, 180°)$），且当 $B \neq 0$ 时，一般式的斜率为 $k = -\frac{A}{B}$，纵截距为 $-\frac{C}{B}$。

关键方法：位置关系的判定

两直线 $l_1: A_1x + B_1y + C_1 = 0$，$l_2: A_2x + B_2y + C_2 = 0$ 的位置关系判定流程：

  • 平行：$A_1B_2 - A_2B_1 = 0$ 且 $A_1C_2 - A_2C_1 \neq 0$（斜率相等但截距不等）
  • 重合：$A_1B_2 - A_2B_1 = 0$ 且 $A_1C_2 - A_2C_1 = 0$（方向向量共线且公共点存在）
  • 相交：$A_1B_2 - A_2B_1 \neq 0$
  • 垂直特例：$A_1A_2 + B_1B_2 = 0$（或斜率之积 $k_1k_2 = -1$，前提是斜率均存在）

距离公式是高考计算类题目的必用工具： - 点 $(x_0, y_0)$ 到直线 $Ax + By + C = 0$ 的距离：$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$ - 两平行直线 $Ax + By + C_1 = 0$ 与 $Ax + By + C_2 = 0$ 的距离：$d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$

典型应用

高考常以以下角度设问： 1. 求直线方程：给定几何条件（如过定点、与已知直线垂直/平行、截距相等），选择合适形式设方程并解参数。选择题偏爱考查截距式与一般式的互化。 2. 对称问题：点关于直线对称、直线关于点对称、直线关于直线对称，核心是转化为点对称坐标公式 $\left( x' = 2a - x,\ y' = 2b - y \right)$ 及中点落在直线上。 3. 与圆、圆锥曲线的综合：直线与曲线的位置关系（联立消元，$\Delta$ 判别式），弦长公式 $|AB| = \sqrt{1+k^2} \cdot |x_1 - x_2|$ 是高频计算点。

⚠️ 高频易错点

  • 忽略斜率不存在的情况：使用点斜式、斜截式设直线方程时，必须单独检验 $x = x_0$ 型的竖直直线是否符合题意，否则会漏解。
  • 截距式漏条件：截距式 $\frac{x}{a} + \frac{y}{b} = 1$ 不能表示过原点或与坐标轴平行的直线。遇到“截距相等”条件时，务必考虑截距为 $0$（过原点）的情形。
  • 距离公式与分母混淆：点 $(x_0, y_0)$ 到直线 $x = a$ 的距离是 $|x_0 - a|$，若盲目套公式 $d = \frac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$ 而不先判断 $B=0$，容易导致分母为零的运算错误。
  • 含参直线过定点问题：不会将直线方程整理成关于参数的多项式——如将 $(m+2)x + (m-1)y - 3m = 0$ 整理为 $m(x+y-3) + (2x - y) = 0$，令括号内为零求定点。
