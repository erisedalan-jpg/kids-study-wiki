线性规划

一句话：在平面直角坐标系中，求二元一次不等式组（约束条件）所表示的可行域内，线性目标函数的最大值或最小值。 English: A method for finding the maximum or minimum value of a linear objective function subject to a set of linear constraints in a planar region.

🎓 考点精讲

定义/原理
若目标函数为 $z = ax + by$（$a,b$ 为常数，且 $a,b$ 不同时为零），约束条件是由若干个关于 $x,y$ 的二元一次不等式组成的线性约束组，则问题可表示为： $$ \begin{cases} a_1x + b_1y \le c_1, \ \vdots \ a_mx + b_my \le c_m, \ x \ge 0, \, y \ge 0 \quad (\text{常含非负约束}) \end{cases} $$ 求 $z$ 的最值。根据线性规划基本定理，若可行域有界，最优解必在可行域的某个顶点处取得；若无界，则可能无最优解。

关键方法
1. 画可行域：将每个不等式化为直线 $a_ix + b_iy = c_i$，取定界（实线或虚线），并用特殊点（如 $(0,0)$）判断区域。所有不等式区域的交集即为可行域（通常为凸多边形）。
2. 作目标函数线：令 $z = 0$ 得到直线 $ax + by = 0$，并按目标函数的系数方向平移该直线。将直线在可行域内平移至与可行域首次/最后接触的顶点，即为使 $z$ 取最小/最大的点。
3. 计算顶点坐标：解接触点所在的两条边界直线的方程组，得到精确坐标，代入 $z$ 求最值。若目标函数直线斜率与边界平行，整条边界上的点均为最优解（无穷多解）。

典型应用
高考常以选择题或填空题出现，设问：“若 $x,y$ 满足约束条件，求 $z = 3x + 2y$ 的最大值（或最小值）”；或求 $\frac{y}{x}$、$x^2 + y^2$ 等几何意义的非线性目标，但此时仍用可行域顶点或几何意义求解。例如，约束条件为 $\begin{cases} x+y \le 4 \ x-y \le 2 \ x\ge0, y\ge0 \end{cases}$，求 $z=2x+y$ 的最大值，通过平移可得最优解在 $(3,1)$ 取得 $z=7$。

⚠️ 高频易错点

  • 不等式方向画反，导致可行域取错；尤其当不等式为 $ax+by \ge c$ 时，不注意 $a$ 的符号导致区域判断失误。
  • 忽略隐含的非负约束（如实际问题中 $x \ge 0, y \ge 0$），造成可行域扩大或顶点遗漏。
  • 目标函数直线平移方向与最值对应关系混淆：将直线沿系数矢量 $(a,b)$ 方向平移时 $z$ 增大，沿相反方向平移 $z$ 减小。学生常记反。
  • 最优解在顶点取得，但计算顶点坐标时解方程组错误或取错交点，特别是可行域边界含多条直线时未精确判断哪两条线交于最优顶点。
