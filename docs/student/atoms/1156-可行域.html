可行域

一句话：满足所有约束条件（线性不等式组）的点的集合，在平面上对应一个凸多边形区域。 English: The set of all points that satisfy all constraints of a linear programming problem, typically a convex polygonal region in the plane.

🎓 考点精讲

定义/原理
在线性规划问题中，约束条件通常由一组二元一次不等式表示，如
$$ \begin{cases} x + y \leq 4 \ 2x - y \geq 1 \ x \geq 0, \; y \geq 0 \end{cases} $$
所有满足这些不等式的有序数对 $(x, y)$ 构成的集合称为可行域。在直角坐标系中，每一个不等式划定一个半平面，这些半平面的交集即为可行域，它是一个凸多边形（可能无界或为空）。

关键方法
求解线性规划问题的标准步骤：
1. 画域：将约束不等式化为直线，用虚线（不包含边界，对应 $<, >$）或实线（包含边界，对应 $\leq, \geq$）画出边界；取测试点（如原点）判定半平面，正确标出公共区域。
2. 找顶点：确定可行域的所有顶点坐标（联立边界直线方程）。
3. 代值：将顶点坐标代入目标函数 $z = Ax + By$，求其最大值或最小值。若目标函数对应斜率和可行域某边界斜率相同，最优解可能在线段上任意点取得，此时有无穷多组最优解。对于整数解问题（网格点），需结合打网格或调整边界单独求解。

典型应用
高考以选择题、填空题为主，常见设问：
- “若 $z = 2x + 3y$ 在约束条件下取得最大值，则最大值是___”
- “已知目标函数 $z = ax + by$ 在给定可行域上的最大值为 12，求参数 $a, b$ 的值”
还会出现距离型（$z = \sqrt{x^2 + y^2}$）、斜率型（$z = \frac{y}{x}$）的线性规划变形，本质上仍是将目标函数几何化为与可行域有交点的直线系、圆系或斜率线。

⚠️ 高频易错点

  • 边界虚实线混淆：不等式带等号画实线（可行域包含边界），不带等号画虚线（不含边界），画错直接导致可行域范围错误。
  • 最优解一定在顶点取得：当目标函数直线与可行域某一边界平行时，最优解可能落在一整条线段上，此时有无穷多最优解，而非唯一顶点。
  • 约束条件遗漏非负：实际问题中常隐含 $x \ge 0, y \ge 0$，学生易忽略。无界可行域也可能有唯一最优解，勿草率判定无最大值。
  • 参数问题中的斜率讨论：当目标函数含参数时，需将目标直线斜率与边界斜率比较，数形结合讨论临界位置，切忌直接代点。
