平面直角坐标系

权重12吉林命中2路径2学段[初中, 高中]主题[代数, 七下, 平面直角坐标系]

aliases: 平面直角坐标系, Cartesian coordinate system, 直角坐标系, 坐标系

课本定义

在平面内画两条互相垂直、原点重合的数轴，组成平面直角坐标系。水平的数轴叫 x 轴（横轴），竖直的数轴叫 y 轴（纵轴），两轴的交点 O 叫原点。平面上的每个点都有唯一的坐标 (x, y) 与之对应。

严格定义

平面直角坐标系（Cartesian coordinate system）：在平面内，取两条互相垂直且相交于原点 O 的数轴，分别称 x 轴（水平，正方向向右）和 y 轴（竖直，正方向向上），合称坐标轴。

平面上的点与有序数对 (x, y) 一一对应，x 称为横坐标（abscissa），y 称为纵坐标（ordinate）。

坐标系的构成要素

要素 定义 说明
原点 O (0, 0) x轴与y轴交点
x 轴 水平数轴 正方向向右
y 轴 竖直数轴 正方向向上
单位长度 两轴刻度单位 通常相等

点与坐标的对应关系

已知点求坐标： 过点 P 作 x 轴的垂线，垂足在 x 轴上的读数为 x 坐标； 过点 P 作 y 轴的垂线，垂足在 y 轴上的读数为 y 坐标。

已知坐标找点： 在 x 轴上找 x 坐标的位置，在 y 轴上找 y 坐标的位置，过两点分别作坐标轴的平行线，交点即为所求点。

各区域坐标特征

位置 x 坐标 y 坐标
第一象限 x > 0 y > 0
第二象限 x < 0 y > 0
第三象限 x < 0 y < 0
第四象限 x > 0 y < 0
x 轴正半轴 x > 0 y = 0
y 轴负半轴 x = 0 y < 0
原点 x = 0 y = 0

典型例题

例题1： 在坐标系中描出 A(3, 2)，B(-1, 4)，C(-2, -3)，D(4, -1) 各点，判断各点所在象限或坐标轴。

解： - A(3, 2)：x>0, y>0，第一象限 - B(-1, 4)：x<0, y>0，第二象限 - C(-2, -3)：x<0, y<0，第三象限 - D(4, -1)：x>0, y<0，第四象限

例题2： 点 P(a-1, 2a+3) 在 x 轴上，求 a 的值。

解：在 x 轴上的点，y 坐标为 0 ∴ 2a + 3 = 0 a = -3/2

历史背景

平面直角坐标系由法国数学家笛卡尔（René Descartes, 1596-1650）创立，故又称笛卡尔坐标系。传说他生病卧床，看着天花板上的苍蝇移动，想到用两个数描述位置，由此发明坐标系，将代数和几何统一（解析几何的诞生）。

知识联系

平面直角坐标系
├── 基础 → 有序数对 (x, y)
├── 与数轴 → 数轴是一维，坐标系是二维
├── 应用 → 函数图像（y = f(x) 在坐标系中的图形）
└── 扩展 → 三维坐标系 (x, y, z)（高中/大学）
