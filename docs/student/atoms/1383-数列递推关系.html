数列递推关系

一句话：数列递推关系是用数列中前若干项表示后续项的等式，结合初始条件唯一确定数列。 English: A recurrence relation for a sequence is an equation that expresses each term as a function of its preceding terms, together with initial conditions.

🎓 考点精讲

定义/原理
给定数列 ${a_n}$，若存在函数 $f$ 使得对任意 $n \ge k$（$k$ 为某正整数）有 $$a_n = f(a_{n-1}, a_{n-2}, \dots, a_{n-k})$$ 则称该式为数列的 $k$ 阶递推关系，并需给出前 $k$ 项初始值才能唯一确定数列。高考主要考查一阶和二阶递推，核心是 由递推求通项公式。

关键方法
1. 基本等差/等比递推：$a_n - a_{n-1} = d$ 或 $\dfrac{a_n}{a_{n-1}} = q$，直接代公式。
2. 累加法：当 $a_n - a_{n-1} = f(n)$ 时，$a_n = a_1 + \sum_{k=2}^{n} f(k)$。
3. 累乘法：当 $\dfrac{a_n}{a_{n-1}} = f(n)$ 时，$a_n = a_1 \prod_{k=2}^{n} f(k)$。
4. 一阶线性递推 $a_n = p a_{n-1} + q$（$p \neq 1$）：待定系数法构造 $a_n + \lambda = p(a_{n-1} + \lambda)$，化为等比。
5. 二阶线性常系数递推 $a_n = p a_{n-1} + q a_{n-2}$：特征方程 $x^2 - px - q = 0$，两不等实根时 $a_n = A x_1^n + B x_2^n$；重根时 $a_n = (A + Bn)x_0^n$。
6. 分式递推 $a_{n+1} = \dfrac{p a_n + q}{r a_n + s}$：常用不动点法或倒数变换。

典型应用
高考中常以解答题形式出现：给出递推式与首项，求 ${a_n}$ 通项公式，再结合求和、不等式证明或恒成立问题。例如 “已知 $a_1=1,\ a_{n+1}=2a_n+3$，求 $a_n$”，或者 “$a_{n+1}=\dfrac{a_n}{2a_n+1}$，求通项并证明 $\sum_{k=1}^n a_k < 1$”。

⚠️ 高频易错点

  • 忽略初始条件的限制：求出的通项公式必须在 $n=1$ 时成立，否则需分段表示。
  • 累加/累乘时未验证 $n=1$ 情形，直接将 $a_n$ 表达式用于求和导致错误。
  • 待定系数构造新数列时系数配错：如 $a_n = p a_{n-1} + q$ 应设 $a_n + \dfrac{q}{p-1} = p\left(a_{n-1} + \dfrac{q}{p-1}\right)$。
  • 特征方程重根情形误写成两个不同根的通解形式，未加 $n$ 倍项。
