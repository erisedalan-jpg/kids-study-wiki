位移（矢量）

权重17吉林命中2路径2学段[高中]主题[必修一, 运动的描述]

aliases: 位移-矢量, 位移, displacement, vector displacement

🎓 位移（矢量）（高中物理）

定义

位移（displacement）是描述物体位置变化的物理量，是从初位置指向末位置的有向线段。

$$\vec{s} = \vec{r}{\text{末}} - \vec{r}$$}

位移是矢量：既有大小（两点间直线距离），又有方向（初位置→末位置）。

矢量的基本性质

矢量（vector）：有大小有方向的量。 - 位移、速度、加速度、力——均为矢量 - 路程、时间、质量、温度——均为标量（只有大小）

矢量的加减遵循平行四边形定则（不是简单代数加减）：

$$|\vec{A} + \vec{B}| \neq |\vec{A}| + |\vec{B}| \quad \text{（一般情况）}$$

$$|\vec{A} + \vec{B}| = |\vec{A}| + |\vec{B}| \quad \text{（同向时取等）}$$

$$|\vec{A} + \vec{B}| \geq \big||\vec{A}| - |\vec{B}|\big| \quad \text{（三角不等式）}$$

一维情形：符号表示方向

在直线运动中，规定正方向后，矢量用正负号表示方向：

$$\Delta x = x_2 - x_1$$

  • $\Delta x = +5\text{ m}$：沿正方向位移5m
  • $\Delta x = -3\text{ m}$：沿负方向位移3m

注意：坐标差 $\Delta x$ 本身就含方向信息（正负号）。

二维情形：分量表示

$$\vec{s} = (s_x, s_y) = (\Delta x, \Delta y)$$

大小：$|\vec{s}| = \sqrt{s_x^2 + s_y^2}$

方向：$\theta = \arctan\frac{s_y}{s_x}$（与 $x$ 轴的夹角）

位移合成（矢量加法）

多段位移的总位移 = 各段位移的矢量和：

$$\vec{s}_{\text{总}} = \vec{s}_1 + \vec{s}_2 + \cdots + \vec{s}_n$$

一维（同直线）：代数相加（带符号）。 二维：用坐标分量分别相加，再合成。

典型例题

例1：一物体先向东位移3m，再向北位移4m，求总位移。

$$s_x = 3\text{ m（东）},\quad s_y = 4\text{ m（北）}$$ $$|\vec{s}| = \sqrt{3^2+4^2} = 5\text{ m}$$

方向：北偏东 $\arctan\frac{3}{4} \approx 37°$（即东偏北 $53°$）

例2：物体做圆周运动，从A运动半圈到B（直径两端），半径 $R$。求位移大小。

A、B是直径两端，位移 = 直径 = $2R$，方向从A指向B。

例3：$t=0$ 时物体在 $x_0 = -2\text{ m}$ 处，$t = 5\text{ s}$ 时在 $x = +8\text{ m}$ 处，求位移。

$$\Delta x = x - x_0 = 8 - (-2) = +10\text{ m（正方向）}$$

易错点

  1. 位移不等于路程（见位移与路程）：路程是路径长度，位移是有向线段。
  2. 矢量不能直接代数相加：两个大小都为5m的位移相加，结果可以是0到10m之间任意值，取决于方向。
  3. "位移为负"不代表"向后退"：负号只代表沿负方向，与参考系正方向选取有关。

与初中物理衔接

初中学路程（标量），高中引入位移（矢量）。矢量概念贯穿整个高中物理（力、速度、加速度、动量、场强），是最重要的数学工具之一。
