aliases: 共点力平衡, concurrent force equilibrium

🎓 共点力平衡（高中物理）

定义

共点力（concurrent forces）：所有力的作用线交于同一点的力系。

对于质点，所有力均视为共点力。

共点力平衡条件：

$$\sum \vec{F} = \vec{0} \iff \sum F_x = 0 \text{ 且 } \sum F_y = 0$$

合力为零，即各力矢量首尾相接构成封闭多边形。

解题方法

方法一：正交分解法（最常用）

  1. 选坐标系（通常一轴沿主要运动方向或最多力的方向）
  2. 将各力分解到 $x$、$y$ 轴
  3. 分别列方程 $\sum F_x = 0$，$\sum F_y = 0$
  4. 联立求解

方法二：图解法（三力平衡）

三力平衡时，三力首尾相接构成封闭三角形，用正弦/余弦定理求解。

方法三：等效合力法

将部分力合成，再与其余力平衡。

常见共点力平衡模型

模型1：悬挂物（两绳夹角问题）

物体重 $G$，两绳夹角 $\alpha$（绳子与竖直方向的夹角分别为 $\theta_1$，$\theta_2$）：

$$T_1\cos\theta_1 + T_2\cos\theta_2 = G$$ $$T_1\sin\theta_1 = T_2\sin\theta_2$$

特殊：两绳对称（$\theta_1=\theta_2=\theta$）：$T_1=T_2=T$，$2T\cos\theta = G$，$T=\frac{G}{2\cos\theta}$

两绳夹角越大，绳中张力越大（$\theta\to90°$ 时，$T\to\infty$）。

模型2：斜面静止

重力 $G$，斜面法线方向支持力 $N$，摩擦力 $f$（沿斜面）：

沿斜面：$f = G\sin\theta$（或反向，视物体趋势）

垂直斜面：$N = G\cos\theta$

模型3：结点法（多绳连接）

在结点处分析受力，各绳拉力与结点所受其他力平衡。

典型例题

例1：一物体用两根绳子悬挂，左绳与水平方向成60°，右绳水平，物体重100N。求两绳张力。

设左绳张力 $T_1$（与水平成60°），右绳张力 $T_2$（水平向左）：

$y$ 方向：$T_1\sin60° = G = 100 \Rightarrow T_1 = \frac{100}{\sin60°} = \frac{100}{\sqrt{3}/2} = \frac{200}{\sqrt{3}} \approx 115.5\text{ N}$

$x$ 方向：$T_1\cos60° = T_2 \Rightarrow T_2 = 115.5\times0.5 \approx 57.7\text{ N}$

例2：电灯挂在两绳中间，两绳与竖直方向各成30°，灯重20N，求绳中张力。

$$2T\cos30° = 20 \Rightarrow T = \frac{20}{2\cos30°} = \frac{20}{\sqrt{3}} \approx 11.5\text{ N}$$

例3：物体在斜面上静止，倾角37°，$\mu_s=0.5$，$m=5\text{ kg}$，$g=10\text{ m/s}^2$。求摩擦力。

$$G\sin37° = mg\times0.6 = 30\text{ N}$$

最大静摩擦力：$f_{\max} = \mu_s N = 0.5\times G\cos37° = 0.5\times40 = 20\text{ N}$

$G\sin37° = 30\text{ N} > f_{\max} = 20\text{ N}$，物体不能静止（会下滑）。

（题目说静止，需重新检验，或题目条件有调整）

例4：结点法——三绳结点，绳1水平（左），绳2向右下45°，绳3向右上60°，绳1拉力10N，求绳2、绳3张力。

$x$：$-10 + T_2\cos45° + T_3\cos60° = 0 \Rightarrow \frac{\sqrt{2}}{2}T_2 + \frac{1}{2}T_3 = 10$

$y$：$-T_2\sin45° + T_3\sin60° = 0 \Rightarrow T_3 = \frac{T_2\sin45°}{\sin60°} = \frac{\frac{\sqrt{2}}{2}T_2}{\frac{\sqrt{3}}{2}} = \frac{\sqrt{2}}{\sqrt{3}}T_2$

代入 $x$ 方程求解 $T_2$，再求 $T_3$（计算过程从略）。

易错点

  1. 正交分解时注意每个力的分量方向：有些力分量可正可负，必须统一坐标系正方向。
  2. 三力平衡不一定成直角三角形：只有特殊情形下三力垂直。
  3. 绳子只能拉，不能推：绳中张力只能沿绳向内（拉力），大小 > 0。

与初中物理衔接

初中只处理简单二力平衡。高中引入多力平衡（三力、四力），用正交分解法定量求解，是高中静力学的核心内容。
