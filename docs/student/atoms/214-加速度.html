🎓 加速度（高中物理）

定义

加速度（acceleration）是描述速度变化快慢的物理量，等于速度的变化量与发生这段变化所用时间的比值：

$$a = \frac{\Delta v}{\Delta t} = \frac{v_t - v_0}{\Delta t}$$

  • 加速度是矢量，方向与速度变化量 $\Delta\vec{v}$ 的方向相同。
  • 单位：m/s²（米每二次方秒）
  • 加速度描述"速度变化的快慢"，不是"速度的大小"。

加速度的方向

加速度方向 = $\Delta\vec{v}$ 方向，不一定与速度方向相同：

情形 $v$ 方向 $a$ 方向 运动状态
加速（speed↑） 向东 向东 加速运动
减速（speed↓） 向东 向西 减速运动
静止 → 向东 0→向东 向东 开始加速
匀速圆周 切向 向心（与v垂直） 速度方向变，大小不变

核心： - $a$ 与 $v$ 同向 → 速度大小增大（加速） - $a$ 与 $v$ 反向 → 速度大小减小（减速） - $a \neq 0$ 不代表速度大；$a = 0$ 代表速度不变（可以是非零的匀速）

单位推导

$$[a] = \frac{[v]}{[t]} = \frac{\text{m/s}}{\text{s}} = \text{m/s}^2$$

重力加速度 $g \approx 9.8\text{ m/s}^2$：表示每秒速度增加9.8m/s。

典型例题

例1：汽车刹车，初速度 $v_0 = 20\text{ m/s}$，经过4s停止（$v_t = 0$）。求加速度。

$$a = \frac{v_t - v_0}{\Delta t} = \frac{0 - 20}{4} = -5\text{ m/s}^2$$

负号表示加速度方向与初速度方向相反（向后），即减速。

例2：飞机起飞，从静止加速到 $v = 72\text{ km/h} = 20\text{ m/s}$，加速时间10s。求加速度。

$$a = \frac{20 - 0}{10} = 2\text{ m/s}^2$$

方向与速度方向相同（向前）。

例3：比较以下两种运动，哪个加速度更大？

  • 甲：速度从0增加到10m/s，用时5s → $a_{\text{甲}} = 2\text{ m/s}^2$
  • 乙：速度从0增加到6m/s，用时2s → $a_{\text{乙}} = 3\text{ m/s}^2$

乙的加速度更大（速度变化更快），虽然速度最终更小。

例4：一物体速度从 $-6\text{ m/s}$（向西）变为 $+4\text{ m/s}$（向东），用时2s，求加速度。

$$a = \frac{4 - (-6)}{2} = \frac{10}{2} = 5\text{ m/s}^2\text{（向东）}$$

易错点

  1. 加速度大≠速度大：速度描述快慢，加速度描述速度变化的快慢，完全不同。
  2. 减速时加速度不为零：刹车时加速度很大（绝对值），但与速度反向。
  3. a=0 时速度不一定为0：匀速运动 $a = 0$，$v \neq 0$。
  4. 加速度符号与速度符号：负加速度不一定表示减速，要看与速度方向的关系。

与初中物理衔接

初中不引入加速度概念，只用语言描述"越来越快/慢"。高中用加速度定量描述速度变化率，是运动学的核心概念，也是牛顿第二定律 $F=ma$ 的关键量。
