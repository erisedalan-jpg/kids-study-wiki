🎓 胡克定律（高中物理）

定律表述

在弹性限度内，弹簧（弹性体）的弹力大小与形变量成正比：

$$\boxed{F = kx}$$

其中： - $F$：弹力（N） - $k$：劲度系数（spring constant / stiffness），单位 N/m，由弹簧材料和结构决定 - $x$：形变量（伸长量或压缩量，m），取正值

适用范围：弹性限度内（不超过弹性限度，否则发生永久形变，胡克定律失效）。

历史背景

英国物理学家罗伯特·胡克（Robert Hooke，1635-1703）于1678年提出，用拉丁文写成"Ut tensio, sic vis"（力如伸长，伸长如力）。

劲度系数 $k$ 的意义

  • $k$ 越大：弹簧越"硬"（刚度大），相同形变需要更大力。
  • $k$ 越小：弹簧越"软"，相同力产生更大形变。

常见 $k$ 值：细软弹簧 $k\sim10\text{ N/m}$；硬弹簧 $k\sim10^3\text{ N/m}$；钢筋 $k\sim10^8\text{ N/m}$。

弹弓、橡皮筋近似满足胡克定律（在弹性范围内）。

$F$-$x$ 图象

胡克定律的图象为过原点的直线，斜率 = $k$。

$$\text{斜率} = k = \frac{\Delta F}{\Delta x}$$

超过弹性限度后，图线偏离直线（非线性区域）。

弹簧的串并联

串联（两弹簧首尾连接）：

$$\frac{1}{k_{\text{串}}} = \frac{1}{k_1} + \frac{1}{k_2} \Rightarrow k_{\text{串}} = \frac{k_1 k_2}{k_1+k_2}$$（等效劲度系数减小，更软）

并联（两弹簧并排，共同支撑）：

$$k_{\text{并}} = k_1 + k_2 \quad \text{（等效劲度系数增大，更硬）}$$

典型例题

例1：弹簧自然长度 $L_0=10\text{ cm}$，$k=200\text{ N/m}$，挂上2kg物体后（$g=10\text{ m/s}^2$）弹簧长度是多少？

重力 $G = 2\times10 = 20\text{ N}$，弹力 $F = G = 20\text{ N}$（平衡）

$$x = \frac{F}{k} = \frac{20}{200} = 0.1\text{ m} = 10\text{ cm}$$

弹簧总长 $= L_0 + x = 10 + 10 = 20\text{ cm}$

例2：从 $F$-$x$ 图求 $k$

图象过 $(0, 0)$ 和 $(0.2\text{ m}, 40\text{ N})$：

$$k = \frac{40}{0.2} = 200\text{ N/m}$$

例3：两弹簧串联，$k_1=100\text{ N/m}$，$k_2=200\text{ N/m}$，总弹力5N，各弹簧形变量？

串联时两弹簧弹力相同（均为5N）：

$$x_1 = \frac{5}{100} = 0.05\text{ m},\quad x_2 = \frac{5}{200} = 0.025\text{ m}$$

等效劲度系数：$k = \frac{100\times200}{100+200} \approx 66.7\text{ N/m}$，总形变 $x = \frac{5}{66.7} = 0.075\text{ m} = x_1+x_2$ ✓

易错点

  1. $x$ 是形变量，不是弹簧总长：$x = L - L_0$（当前长度减自然长度）。
  2. 弹性限度不能超过：超限后 $F=kx$ 不成立，且弹簧会永久变形。
  3. $k$ 由弹簧决定，与 $F$、$x$ 无关：加不同的力 $k$ 不变（弹性范围内）。

与初中物理衔接

初中知道弹簧受力越大形变越大，高中给出定量关系 $F=kx$，引入劲度系数，是力与形变关系的定量化。
