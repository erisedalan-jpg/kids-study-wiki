🎓 一元二次不等式

定义

一元二次不等式：形如 $ax^2 + bx + c > 0$（或 $\geq 0, < 0, \leq 0$）的不等式，其中 $a \neq 0$。

其解集与对应二次方程 $ax^2 + bx + c = 0$ 的根、以及二次函数 $y = ax^2 + bx + c$ 的图象密切相关。

关键公式与解法

判别式 $\Delta = b^2 - 4ac$

设 $a > 0$（开口向上），解 $ax^2 + bx + c > 0$ 和 $ax^2 + bx + c < 0$：

判别式 方程根 $ax^2+bx+c > 0$ 的解集 $ax^2+bx+c < 0$ 的解集
$\Delta > 0$ 两不等实根 $x_1 < x_2$ ${x \mid x < x_1 \text{ 或 } x > x_2}$ ${x \mid x_1 < x < x_2}$
$\Delta = 0$ 两相等实根 $x_1 = x_2$ ${x \mid x \neq x_1}$（即 $\mathbb{R} \setminus {x_1}$） $\emptyset$
$\Delta < 0$ 无实根 $\mathbb{R}$（恒成立） $\emptyset$

当 $a < 0$ 时，将不等号方向对调，或先化为 $a > 0$ 的形式（两边乘 $-1$，不等号翻转）。

推导思路

"数形结合"三步法： 1. 确定开口方向（$a$ 的正负） 2. 求判别式 $\Delta$，若 $\Delta \geq 0$ 则求方程的根 3. 根据抛物线图象，读取对应区间：$> 0$ 取抛物线在 $x$ 轴上方的区间，$< 0$ 取下方的区间

典型例题

例1：解不等式 $x^2 - 3x + 2 > 0$。

解：$\Delta = 9 - 8 = 1 > 0$，方程 $x^2 - 3x + 2 = 0$ 的根为 $x_1 = 1, x_2 = 2$。 $a = 1 > 0$，开口向上，$> 0$ 取两根之外，故解集为 ${x \mid x < 1 \text{ 或 } x > 2}$，即 $(-\infty, 1) \cup (2, +\infty)$。

例2：解不等式 $-x^2 + 4x - 4 \leq 0$。

解：先变形：$x^2 - 4x + 4 \geq 0$，即 $(x-2)^2 \geq 0$。 $\Delta = 16 - 16 = 0$，等根 $x_1 = x_2 = 2$。$(x-2)^2 \geq 0$ 恒成立，解集为 $\mathbb{R}$。

例3：含参数不等式 $x^2 - (a+1)x + a \leq 0$。

解：方程 $x^2 - (a+1)x + a = 0$ 因式分解为 $(x-1)(x-a) = 0$，根 $x_1 = 1, x_2 = a$。

  • 若 $a > 1$：解集为 $[1, a]$
  • 若 $a = 1$：解集为 ${1}$
  • 若 $a < 1$：解集为 $[a, 1]$

易错点

  1. $a < 0$ 时忘记翻转：$a < 0$ 的二次不等式，要将图象"倒扣"，$> 0$ 反而取两根之间
  2. $\Delta = 0$ 的边界：$\geq 0$ 时解集含单点 ${x_1}$，$> 0$ 时不含，要区分严格/非严格不等号
  3. $\Delta < 0$ 恒成立情形：$a > 0, \Delta < 0$ 时 $ax^2+bx+c > 0$ 恒成立，$<0$ 无解，不要遗漏
  4. 含参分类讨论：含参不等式需对参数的不同范围分类讨论，每种情况单独列出
  5. 解集写法：注意区间端点的开闭，严格不等号用开区间 $()$，非严格用闭区间 $[]$

与初中知识的衔接

初中学习了一元一次不等式和一元一次不等式组的解法。高中一元二次不等式将"数轴解题"推广到"抛物线图象+判别式"，是二次函数和不等式的综合应用。
