🎓 三角函数应用

应用场景分类

三角函数在高中数学中的应用主要分为以下几类：

一、三角函数解题中的实际问题：测量高度、距离、仰角、俯角（与初中解直角三角形类似，但扩展到任意三角形）

二、周期现象建模：用 $y = A\sin(\omega t + \varphi) + b$ 建立周期性变化的数学模型

三、三角函数的最值应用：在约束条件下利用三角函数的有界性（$|\sin\alpha| \leq 1$）求最值

四、向量与三角函数结合（了解）：向量夹角的余弦

周期现象建模步骤

  1. 确定参数：
  2. 振幅 $A = \dfrac{\text{最大值} - \text{最小值}}{2}$
  3. 竖向位移 $b = \dfrac{\text{最大值} + \text{最小值}}{2}$（中间值）
  4. 周期 $T$（从题目中读取），$\omega = \dfrac{2\pi}{T}$

  5. 初相 $\varphi$：代入已知点确定

  6. 写出函数解析式

  7. 利用函数回答问题

三角函数最值

利用 $-1 \leq \sin\theta \leq 1$ 和 $-1 \leq \cos\theta \leq 1$，当中间变量 $\theta = f(x)$ 时，在 $\sin$ 或 $\cos$ 的值域内约束整体最值。

典型例题

例1（潮汐建模）：某海湾一天内水位高度（米）随时间 $t$（小时，$0 \leq t \leq 24$）周期变化。已知高潮时水位 $6$ 米（$t = 3$ 时），低潮时水位 $2$ 米（$t = 15$ 时）。建立水位 $y$ 关于时间 $t$ 的函数模型。

解： - $A = \dfrac{6-2}{2} = 2$（振幅） - $b = \dfrac{6+2}{2} = 4$（中间水位） - 高潮到低潮为半个周期：$\dfrac{T}{2} = 15 - 3 = 12$，$T = 24$ 小时，$\omega = \dfrac{2\pi}{24} = \dfrac{\pi}{12}$ - 模型：$y = 2\sin!\left(\dfrac{\pi}{12}t + \varphi\right) + 4$

代入高潮点 $(3, 6)$：$2\sin!\left(\dfrac{\pi}{12} \times 3 + \varphi\right) + 4 = 6$

$\sin!\left(\dfrac{\pi}{4} + \varphi\right) = 1$，$\dfrac{\pi}{4} + \varphi = \dfrac{\pi}{2}$，$\varphi = \dfrac{\pi}{4}$

$$y = 2\sin!\left(\frac{\pi}{12}t + \frac{\pi}{4}\right) + 4$$

例2（三角最值）：求 $y = 3\sin x + 4\cos x$ 的最大值和最小值。

解：辅助角公式（后续学习）：$y = 5\sin(x + \varphi)$，其中 $\tan\varphi = 4/3$。

$-1 \leq \sin(x+\varphi) \leq 1$，故最大值为 $5$，最小值为 $-5$。

（辅助角：$y = R\sin(x + \varphi)$，$R = \sqrt{3^2 + 4^2} = 5$）

例3（仰角问题）：从某点 $A$ 测得楼顶 $B$ 的仰角为 $30°$，从 $A$ 沿水平方向走近 $20$ 米到 $C$ 点，测得仰角为 $60°$，求楼高 $BD$。

解：设楼高 $BD = h$，$CD = d$。

$\tan 60° = h/d \Rightarrow d = h/\sqrt{3}$

$\tan 30° = h/(d+20) \Rightarrow d+20 = h\sqrt{3} \Rightarrow h/\sqrt{3} + 20 = h\sqrt{3}$

$20 = h\sqrt{3} - h/\sqrt{3} = h(3-1)/\sqrt{3} = 2h/\sqrt{3}$

$h = 10\sqrt{3}$ 米

易错点

  1. 建模时振幅与位移的计算：振幅是峰值与中心线的距离（半峰差），不是最大值本身
  2. 周期确定：高潮到低潮是半个周期，不是一个周期
  3. 仰角是向上看的角：从水平线向上测量，俯角是向下测量，注意区分
  4. 三角最值时需验证等号可达：确认 $\sin(x+\varphi) = 1$ 或 $-1$ 在问题允许的范围内可以取到
  5. 初相的确定：代入已知点时，注意 $\sin\theta = 1$ 等方程的多解，用题目中的附加条件（如 $|\varphi| \leq \pi/2$）筛选

与初中知识的衔接

初中用正弦、余弦、正切解直角三角形（测量距离、高度）。高中将其推广到：（1）利用正弦、余弦定理解一般三角形；（2）建立周期变化的函数模型；（3）结合向量分析力的合成与分解。
