自交

权重12吉林命中2路径0学段[高中]主题[必修二, 第1章遗传因子的发现]

aliases: 自交, self-fertilization, selfing

自交

一句话：自交是指同一植株（或动物个体）的雌雄配子相互结合，或同一基因型个体之间的交配；连续自交可使后代纯合比例逐代增加，是育种中纯化基因型的重要手段。 English: Self-fertilization (selfing) is the union of male and female gametes from the same individual; repeated selfing increases homozygosity in offspring.

🎓 高中进阶

定义

自交（self-fertilization / selfing）：同一植株花朵的花粉落到同一朵花（或同株另一花）的柱头上完成受精，即雌雄配子来自同一遗传个体。

豌豆是天然自花传粉植物，孟德尔实验中豌豆的自交是最基础的实验操作。 对于动物或雌雄异株植物，自交指相同基因型个体间的交配（近交）。

原理 / 规律

杂合子（Aa）连续自交各代纯合比例

设第n代自交：

$$\text{纯合子比例} = 1 - \left(\frac{1}{2}\right)^{n-1}$$

$$\text{杂合子（Aa）比例} = \left(\frac{1}{2}\right)^{n-1}$$

代次 AA Aa aa 纯合比例
F₁（Aa） 0 1 0 0%
F₂ 1/4 1/2 1/4 50%
F₃ 3/8 1/4 3/8 75%
F₄ 7/16 1/8 7/16 87.5%
F∞ 1/2 0 1/2 100%

连续自交使杂合子比例每代减半，纯合子比例趋向100%，但理论上永远不能达到100%（除非无限代）。

纯合子自交结果

  • AA × AA → 全AA（后代基因型不变）
  • aa × aa → 全aa（后代基因型不变）

结论：纯合子自交后代基因型永远不变；只有杂合子自交才会发生基因型分离。

自交在育种中的应用

用途 说明
纯化基因型 连续自交使杂合植株趋于纯合，获得纯合优良品种
鉴别纯合与杂合 自交后代发生性状分离→亲本为杂合子
维持品种纯度 自交品种自然纯合，无需人工授粉隔离

典型例题

例题：基因型为Aa的豌豆连续自交3代（F₁→F₃→F₄），求F₄中各基因型的比例以及显性性状个体的比例。

解题： F₁ = Aa，经3次自交到F₄（n=3，即第3代自交产生的F₄）：

$$\text{Aa 比例} = \left(\frac{1}{2}\right)^3 = \frac{1}{8}$$

$$\text{AA} = \text{aa} = \frac{1 - 1/8}{2} = \frac{7}{16}$$

  • F₄基因型：AA = 7/16, Aa = 1/8 = 2/16, aa = 7/16
  • 验证：7/16 + 2/16 + 7/16 = 16/16 ✓
  • 显性个体（AA + Aa）= 7/16 + 2/16 = 9/16

易错点

  1. 自交≠近亲交配：自交特指同一个体（或相同基因型个体）间的受精；近亲交配是血缘关系近的不同个体之间的交配，效果相似但有区别。
  2. 每次自交杂合子减半，不是后代总数减半：自交后代数量不减少，只是杂合子比例减少。
  3. 植物"自花传粉"不一定是自交：自花传粉的植物通常是自交，但人工异花传粉时不是自交。
  4. 连续自交3代不是n=3：要明确是"第3次自交后的后代"还是"自交3代后"，注意起点计数。
