三角恒等变换

权重37吉林命中8路径0学段[高中]主题[必修一, 三角函数, 第五章]

aliases: 三角恒等变换, trigonometric identities, 三角恒等式变换

🎓 三角恒等变换

定义

三角恒等变换：利用三角恒等式，将一个三角表达式化为另一种等价形式的变换过程。

三角恒等变换是三角函数学习的核心技能，涉及的公式包括：同角关系、诱导公式、两角和差公式、二倍角公式等。

基础恒等式体系

同角关系（见 同角三角函数关系）：

$$\sin^2\alpha + \cos^2\alpha = 1, \quad \tan\alpha = \frac{\sin\alpha}{\cos\alpha}$$

两角和差公式（见 两角和差正余弦）：

$$\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$$

$$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$$

$$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$$

二倍角公式（令 $\beta = \alpha$）：

$$\sin 2\alpha = 2\sin\alpha\cos\alpha$$

$$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 1 - 2\sin^2\alpha = 2\cos^2\alpha - 1$$

$$\tan 2\alpha = \frac{2\tan\alpha}{1 - \tan^2\alpha}$$

半角公式（由二倍角变形）：

$$\sin^2\frac{\alpha}{2} = \frac{1 - \cos\alpha}{2}$$

$$\cos^2\frac{\alpha}{2} = \frac{1 + \cos\alpha}{2}$$

$$\tan\frac{\alpha}{2} = \frac{1 - \cos\alpha}{\sin\alpha} = \frac{\sin\alpha}{1 + \cos\alpha}$$

积化和差（由和差公式导出）：

$$\sin\alpha\cos\beta = \frac{1}{2}[\sin(\alpha+\beta) + \sin(\alpha-\beta)]$$

$$\cos\alpha\cos\beta = \frac{1}{2}[\cos(\alpha-\beta) + \cos(\alpha+\beta)]$$

$$\sin\alpha\sin\beta = \frac{1}{2}[\cos(\alpha-\beta) - \cos(\alpha+\beta)]$$

辅助角公式：

$$a\sin x + b\cos x = \sqrt{a^2+b^2}\sin(x+\varphi)$$

其中 $\tan\varphi = b/a$，$R = \sqrt{a^2+b^2}$ 称为合振幅。

典型例题

例1：化简 $\dfrac{1 - \cos 2\alpha}{\sin 2\alpha}$。

解：$\dfrac{1 - \cos 2\alpha}{\sin 2\alpha} = \dfrac{2\sin^2\alpha}{2\sin\alpha\cos\alpha} = \dfrac{\sin\alpha}{\cos\alpha} = \tan\alpha$

例2：已知 $\sin\alpha = \dfrac{3}{5}$，$\alpha \in \left(0, \dfrac{\pi}{2}\right)$，求 $\sin 2\alpha, \cos 2\alpha$。

解：$\cos\alpha = \dfrac{4}{5}$（第一象限，正值）。

$\sin 2\alpha = 2\sin\alpha\cos\alpha = 2 \times \dfrac{3}{5} \times \dfrac{4}{5} = \dfrac{24}{25}$

$\cos 2\alpha = 1 - 2\sin^2\alpha = 1 - 2 \times \dfrac{9}{25} = 1 - \dfrac{18}{25} = \dfrac{7}{25}$

例3：化简 $(\sin\alpha + \cos\alpha)^2$。

解：$(\sin\alpha + \cos\alpha)^2 = \sin^2\alpha + 2\sin\alpha\cos\alpha + \cos^2\alpha = 1 + \sin 2\alpha$

例4：证明 $\cos^4\alpha - \sin^4\alpha = \cos 2\alpha$。

证：$\cos^4\alpha - \sin^4\alpha = (\cos^2\alpha + \sin^2\alpha)(\cos^2\alpha - \sin^2\alpha) = 1 \cdot \cos 2\alpha = \cos 2\alpha$ $\square$

易错点

  1. $\cos 2\alpha$ 有三种形式：$\cos^2\alpha - \sin^2\alpha = 1-2\sin^2\alpha = 2\cos^2\alpha - 1$，根据已知条件选择最简形式
  2. $\sin 2\alpha$ 不是 $2\sin\alpha$：$\sin 2\alpha = 2\sin\alpha\cos\alpha$，多了 $\cos\alpha$ 因子
  3. 辅助角公式的相位：$a\sin x + b\cos x = R\sin(x+\varphi)$，$\tan\varphi = b/a$（注意：$b$ 对应 $\cos x$ 的系数）
  4. $\sin(2\alpha)$ vs $(\sin\alpha)^2$：$\sin 2\alpha$ 是二倍角，$\sin^2\alpha$ 是正弦的平方，完全不同
  5. 变换方向：恒等变换没有固定方向，要根据目标形式选择变换路径

与初中知识的衔接

初中接触了完全平方公式、差的平方等代数恒等式。高中的三角恒等变换是类似的思想在三角函数领域的应用，本质上都是"等价变形"。学好代数恒等变换有助于掌握三角恒等变换的技巧。
