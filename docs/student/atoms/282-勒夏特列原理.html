勒夏特列原理

权重21吉林命中3路径2学段[高中]主题[必修二, 第六章]

aliases: 勒夏特列原理, Le Chatelier's principle, 平衡移动原理

勒夏特列原理

一句话：勒夏特列原理（平衡移动原理）指出，如果改变影响平衡的一个因素，平衡会向能减弱该改变的方向移动。 English: Le Chatelier's Principle states that if a change is made to a system at equilibrium, the system will shift in the direction that partially counteracts that change.

🎓 高中进阶

定义

勒夏特列原理（亨利·路易·勒夏特列，1884年）：

如果改变影响化学平衡的一个条件（浓度、温度或压强），平衡就会向能减弱这种改变的方向移动。

关键："减弱"不是"消除"——平衡移动后，那个条件的变化会被部分抵消，但不会完全回到原来的状态。

原理的应用

改变条件 平衡移动方向（减弱改变的方向） 说明
增大反应物浓度 正向移动（消耗反应物） 减弱浓度增大
减小生成物浓度 正向移动（生成更多产物） 减弱浓度减小
升高温度 向吸热方向（吸收热量） 减弱温度升高
降低温度 向放热方向（释放热量） 减弱温度降低
增大压强 向气体mol数少的方向（缩小体积效果） 减弱压强增大
减小压强 向气体mol数多的方向（扩大体积效果） 减弱压强减小

勒夏特列原理的适用范围

适用： - 化学平衡（温度、浓度、压强的影响） - 溶解平衡、电离平衡（延伸）

不适用（注意）： - 催化剂：不移动平衡，原理不适用 - 固体/纯液体：加入不影响平衡（浓度视为常数）

典型实例分析

实例1：N₂+3H₂⇌2NH₃（合成氨）

操作 勒夏特列解释
升温（负面：平衡逆移） 升温使平衡向吸热方向（逆向）移动，NH₃产率下降
高压（正面：平衡正移） 增压平衡向mol数少方向（正向，2mol<4mol），NH₃增多
及时移走NH₃ 减小生成物浓度→正向移动→继续生成NH₃

实例2：2NO₂⇌N₂O₄（颜色实验）

将充有NO₂/N₂O₄平衡混合气的注射器压缩： - 增大压强→平衡向mol数少的方向（N₂O₄，右侧）移动 - c(NO₂)降低，c(N₂O₄)升高 - 但颜色瞬间加深（因体积减小，浓度瞬间增大），之后再变浅（平衡移动减小c(NO₂)） - 最终颜色比原来深（因体积缩小，总体浓度更大）

实例3：工业制盐酸（HCl）溶于水

HCl(g)溶于水：HCl → H⁺ + Cl⁻（不可逆） 但若将HCl溶液蒸发，H⁺和Cl⁻会重新结合为HCl气体散出（体现可逆性）。

勒夏特列原理的局限性

  1. 只预测方向，不给出移动的程度
  2. 催化剂不改变平衡位置（原理不适用）
  3. 对不可逆反应不适用（不存在平衡）

典型例题

例1：向FeCl₃溶液中加入KSCN，溶液变血红色（Fe³⁺+3SCN⁻⇌Fe(SCN)₃）。若加入稀盐酸，颜色变化如何？

分析：加入HCl不提供Fe³⁺和SCN⁻，但Cl⁻不影响该平衡。 实际上酸可能影响Fe³⁺的水解平衡，使c(Fe³⁺)增大→平衡正向→颜色加深。

例2：在CO₂+C⇌2CO（△H>0，吸热），升高温度时，平衡向哪个方向移动？K如何变化？

分析： - 升温→勒夏特列：向吸热方向（正向）移动 - CO增多，CO₂和C减少 - K = c(CO)²/c(CO₂)增大（升温使吸热反应的K增大）

易错点

  • 勒夏特列原理是"减弱"改变，不是"消除"（平衡移动后浓度仍有所变化）
  • 催化剂不适用勒夏特列原理（不移动平衡）
  • 增大压强充入惰性气体（恒容）→各组分分压不变→平衡不移动
