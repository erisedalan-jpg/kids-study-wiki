原电池

权重36吉林命中5路径4学段[高中]主题[必修二, 第六章]

aliases: 原电池, galvanic cell, 伽伐尼电池, 化学电池

原电池

一句话：原电池是将化学能（氧化还原反应的能量）直接转化为电能的装置，由两个电极、电解质溶液和导线构成。 English: A galvanic cell (voltaic cell) is a device that directly converts chemical energy from a redox reaction into electrical energy, consisting of two electrodes, an electrolyte solution, and conducting wires.

🎓 高中进阶

定义

原电池：利用自发进行的氧化还原反应产生电流的装置。

工作原理： - 在原电池中，氧化反应和还原反应分开在两个电极上进行 - 电子通过外电路（导线）从负极流向正极 - 离子通过内电路（电解质溶液）传导电流

原电池的构成条件

条件 说明
自发的氧化还原反应 提供能量来源
两个活性不同的电极 活泼金属为负极，不活泼金属/石墨为正极
电解质溶液 允许离子移动形成内电路
闭合回路 外电路（导线）和内电路（溶液）组成

锌铜原电池（铜锌硫酸电池）

装置：锌片（Zn）和铜片（Cu）插入稀H₂SO₄，通过导线相连。

电极反应：

极 材料 反应 类型
负极 Zn（活泼金属） Zn - 2e⁻ → Zn²⁺（氧化反应） 失去电子
正极 Cu（不活泼） 2H⁺ + 2e⁻ → H₂↑（还原反应） 得到电子

总反应：Zn + H₂SO₄ → ZnSO₄ + H₂↑

电子流向：从负极（Zn）→ 导线 → 正极（Cu）→ 外电路

电流方向（与电子反向）：从正极 → 外电路 → 负极

离子移动：阳离子（H⁺、Zn²⁺）向正极移动；阴离子（SO₄²⁻）向负极移动

原电池图示分析

        负极(Zn) ←←←←←← 外电路（电子流）→→→→→ 正极(Cu)
             |                                        |
             Zn→Zn²⁺+2e⁻                    2H⁺+2e⁻→H₂
             |                                        |
             └──────────── H₂SO₄ 溶液 ──────────────┘
                     H⁺ 向右（正极）移动
                     SO₄²⁻ 向左（负极）移动

各种原电池的正负极规律

规律 举例
活泼金属为负极 Zn-Cu电池：Zn为负极
非金属（石墨）通常为正极 Zn-C（石墨）电池
与电解质溶液反应的电极 通常是负极
正极发生还原反应，负极发生氧化反应 金属溶解：负极；气体析出（如H₂）：正极

原电池 vs 电解池

比较项 原电池 电解池
能量转化 化学能→电能 电能→化学能
反应类型 自发氧化还原 非自发反应
电极名称 负极（氧化），正极（还原） 阴极（还原），阳极（氧化）
驱动力 反应自发 外加电源

典型例题

例1：判断Zn-Fe在稀HCl中组成的原电池，各极发生什么反应？

分析：Zn比Fe活泼，Zn为负极，Fe为正极。 - 负极（Zn）：Zn - 2e⁻ → Zn²⁺（Zn溶解） - 正极（Fe）：2H⁺ + 2e⁻ → H₂↑（Fe表面产生气泡） - 注意：铁不溶解（正极不溶），气泡在铁表面产生

例2：为什么原电池比直接燃烧能量利用率更高？

分析： - 燃烧：化学能→热能→机械能→电能（多步转化，每步有损失，总效率约30%） - 原电池：化学能→电能（直接转化，效率约60-80%） - 原电池减少了中间转化步骤，能量损失少

易错点

  • 原电池负极是活泼金属，发生氧化反应（失去电子），不是还原
  • 电流方向与电子流向相反（电流：正→负；电子：负→正，均在外电路）
  • 盐桥中阳离子向正极移动，阴离子向负极移动（与电子流方向相关）
