函数零点

权重28吉林命中7路径0学段[高中]主题[必修一, 指数函数与对数函数, 第四章]

aliases: 函数零点, zero of function, 零点, 函数的零点

🎓 函数零点

定义

设函数 $f(x)$ 的定义域为 $D$。若存在 $x_0 \in D$ 使得 $f(x_0) = 0$，则称 $x_0$ 为函数 $f(x)$ 的零点（zero）。

几何意义：函数零点就是函数图象与 $x$ 轴的交点的横坐标。

等价关系： $$f(x_0) = 0 \Leftrightarrow x_0 \text{ 是方程 } f(x) = 0 \text{ 的根}$$

函数零点问题与方程求根问题等价。

零点存在定理（区间定理）

零点存在定理：若函数 $f(x)$ 在闭区间 $[a, b]$ 上连续，且 $f(a) \cdot f(b) < 0$（即 $f(a)$ 与 $f(b)$ 异号），则在开区间 $(a, b)$ 内至少存在一个零点 $x_0$，使得 $f(x_0) = 0$。

注意： - 这只是存在性定理，不能确定零点的个数（可能有多个） - 条件是 $f(a) \cdot f(b) < 0$，即两端函数值异号 - 函数必须在 $[a, b]$ 上连续

二分法（逐步逼近零点）

利用零点定理，反复将包含零点的区间二等分，逐步缩小零点所在范围：

  1. 确认 $[a, b]$ 内有零点：$f(a) \cdot f(b) < 0$
  2. 取中点 $c = \dfrac{a+b}{2}$，计算 $f(c)$
  3. 若 $f(c) = 0$，$c$ 即为零点；若 $f(a) \cdot f(c) < 0$，零点在 $[a, c]$ 内；否则在 $[c, b]$ 内
  4. 重复步骤 2-3，直到区间长度满足精度要求

典型例题

例1：证明方程 $x^3 - 2x - 5 = 0$ 在区间 $(2, 3)$ 内有实根。

设 $f(x) = x^3 - 2x - 5$（在 $\mathbb{R}$ 上连续）。

$f(2) = 8 - 4 - 5 = -1 < 0$

$f(3) = 27 - 6 - 5 = 16 > 0$

$f(2) \cdot f(3) = (-1)(16) < 0$，由零点存在定理，$(2, 3)$ 内至少有一个零点，即方程至少有一个根在 $(2, 3)$ 内。$\square$

例2：用二分法求方程 $x^2 - 2 = 0$ 在 $(1, 2)$ 内的正根的近似值（精确到 $0.1$）。

解：$f(x) = x^2 - 2$，$f(1) = -1 < 0$，$f(2) = 2 > 0$，零点在 $(1, 2)$ 内。

  • $f(1.5) = 2.25 - 2 = 0.25 > 0$，零点在 $(1, 1.5)$ 内
  • $f(1.25) = 1.5625 - 2 = -0.4375 < 0$，零点在 $(1.25, 1.5)$ 内
  • $f(1.375) = 1.890625 - 2 = -0.109 < 0$，零点在 $(1.375, 1.5)$ 内
  • $f(1.4) = 1.96 - 2 = -0.04 < 0$，零点在 $(1.4, 1.5)$ 内

取 $x \approx 1.4$（精确到 $0.1$）。（实际上 $\sqrt{2} \approx 1.414$）

例3：函数 $f(x) = \log_2 x + x - 3$ 的零点个数。

解：$f(1) = 0 + 1 - 3 = -2 < 0$，$f(4) = 2 + 4 - 3 = 3 > 0$，在 $(1, 4)$ 内有零点。

$f(x)$ 是单调递增函数（$\log_2 x$ 递增，$x$ 递增，两者之和递增），故零点唯一。

精确零点：$\log_2 x = 3 - x$，数值解约在 $(2, 3)$ 内（$f(2) = 1+2-3=0$！直接验证 $x = 2$ 是零点）。

易错点

  1. 异号才能保证零点存在：$f(a) \cdot f(b) < 0$ 是条件，若同号（$> 0$）则无法用此定理（可能有零点，也可能没有）
  2. 连续性要求：若 $f(x)$ 在 $(a,b)$ 有间断点，定理不适用
  3. 零点个数：定理只保证"至少一个"，实际可能有多个（若函数振荡）
  4. 零点 ≠ 极小值点：图象切 $x$ 轴的点是极值点（切点），穿过 $x$ 轴的点是零点（交点）
  5. 二分法的精度：每次二分后区间缩短为原来的 $1/2$，$n$ 次后区间长度为 $\dfrac{b-a}{2^n}$

与初中知识的衔接

初中学习了方程的根（解方程），如一元一次方程、一元二次方程的求根。高中从函数角度重新解读"根"——方程的根就是函数的零点，函数图象与 $x$ 轴的交点——建立了"方程与函数"之间的桥梁。
