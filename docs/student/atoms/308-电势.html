电势

权重21吉林命中3路径2学段[高中]主题[必修三, 静电场深化]

aliases: 电势, electric potential, 电位

🎓 电势（Electric Potential）

定义：电场中某点的电势 $\varphi$ 等于将单位正电荷从该点移到零势点（通常取无穷远处或大地）时，电场力所做的功。

$$\varphi = \frac{W}{q}$$

其中 $W$ 为电场力做的功，$q$ 为试探正电荷量。单位：伏特（V），$1\,\text{V} = 1\,\text{J/C}$。

物理意义

  • 电势是描述电场能量性质的标量，是场的属性，与试探电荷无关。
  • 电荷 $q$ 在电场中某点的电势能：$E_p = q\varphi$。
  • 零势点选取不同，各点电势值不同，但任意两点电势差不变。

电势分布规律

情形 规律
正点电荷 周围 $\varphi > 0$，由近到远电势降低
负点电荷 周围 $\varphi < 0$，由近到远电势升高（绝对值减小）
匀强电场 沿电场线方向电势均匀降低
等势面 同一等势面上各点电势相等，与电场线垂直

电势与电势差

$$U_{AB} = \varphi_A - \varphi_B$$

$$W_{AB} = q(\varphi_A - \varphi_B) = qU_{AB}$$

电场线总是从高电势指向低电势。沿等势面移动电荷，电场力做功为零。

典型例题

例：匀强电场中 A 点电势 $\varphi_A = 10\,\text{V}$，B 点电势 $\varphi_B = 4\,\text{V}$，将电荷量 $q = 2\times10^{-3}\,\text{C}$ 的正电荷从 A 移到 B，电场力做功为多少？

$$W_{AB} = q(\varphi_A - \varphi_B) = 2\times10^{-3}\times(10-4) = 1.2\times10^{-2}\,\text{J}$$

电场力做正功，正电荷从高电势移向低电势，电势能减小。

易错点

  • 电势高不代表电场强（等势面越密集，场越强）。
  • 负电荷从高电势到低电势，电场力做负功（与正电荷相反）。
  • 导体内部和导体表面是等势体和等势面，导体内部 $E = 0$。

与必修二衔接

必修二学习了功与能的关系；本节将功的概念推广到电场，引入电势能 $E_p = q\varphi$，体现电场储存和转化能量的性质。
