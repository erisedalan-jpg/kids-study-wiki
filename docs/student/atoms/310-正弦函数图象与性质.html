🎓 正弦函数图象与性质

定义

正弦函数：$y = \sin x$，$x \in \mathbb{R}$

基本性质

性质 内容
定义域 $\mathbb{R}$
值域 $[-1, 1]$
最大值 $1$，在 $x = \dfrac{\pi}{2} + 2k\pi$（$k \in \mathbb{Z}$）时取得
最小值 $-1$，在 $x = -\dfrac{\pi}{2} + 2k\pi$（$k \in \mathbb{Z}$）时取得
周期 $T = 2\pi$
零点 $x = k\pi$（$k \in \mathbb{Z}$），与 $x$ 轴的交点
奇偶性 奇函数（$\sin(-x) = -\sin x$），关于原点对称
对称轴 $x = \dfrac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$）（过极值点的竖线）

单调区间

$$\text{单调递增区间：} \left[-\frac{\pi}{2} + 2k\pi, \frac{\pi}{2} + 2k\pi\right], \quad k \in \mathbb{Z}$$

$$\text{单调递减区间：} \left[\frac{\pi}{2} + 2k\pi, \frac{3\pi}{2} + 2k\pi\right], \quad k \in \mathbb{Z}$$

图象特征（正弦曲线 / sinusoidal wave）

  • 图象是一条在 $[-1, 1]$ 之间上下波动的曲线
  • 从 $(0, 0)$ 开始，向右先升到最高点 $(\pi/2, 1)$，再降到 $(\pi, 0)$，继续降到最低点 $(3\pi/2, -1)$，最后回到 $(2\pi, 0)$ 完成一个完整周期
  • 关键点（5个点描绘一个周期）：$(0, 0), (\pi/2, 1), (\pi, 0), (3\pi/2, -1), (2\pi, 0)$

典型例题

例1：求 $y = \sin x$ 在 $\left[-\dfrac{\pi}{2}, \pi\right]$ 上的最大值和最小值。

解：在此区间内，$x = \dfrac{\pi}{2}$ 时 $\sin x = 1$（最大值）；$x = -\dfrac{\pi}{2}$ 时 $\sin x = -1$（最小值）。

例2：解方程 $\sin x = \dfrac{1}{2}$（$x \in [0, 2\pi]$）。

解：在 $[0, 2\pi]$ 内，$\sin x = \dfrac{1}{2}$ 有两个解：$x_1 = \dfrac{\pi}{6}$ 和 $x_2 = \pi - \dfrac{\pi}{6} = \dfrac{5\pi}{6}$（图象上两处经过 $y = 1/2$ 的点）。

例3：判断 $f(x) = \sin x$ 在 $\left(\dfrac{\pi}{2}, \pi\right)$ 上的单调性。

解：$\left(\dfrac{\pi}{2}, \pi\right) \subset \left[\dfrac{\pi}{2}, \dfrac{3\pi}{2}\right]$（单调递减区间），故 $f(x) = \sin x$ 在 $\left(\dfrac{\pi}{2}, \pi\right)$ 上单调递减。

易错点

  1. 最大值点在 $\pi/2$，不是 $\pi$：$\sin(\pi/2) = 1$，$\sin\pi = 0$，不要混淆
  2. 零点在 $k\pi$，包括 $x = \pi, 2\pi$ 等：不只是 $x = 0$
  3. 周期 $2\pi$，不是 $\pi$：正弦函数完成一个完整循环需 $2\pi$
  4. 奇函数：$y = \sin x$ 是奇函数（关于原点对称），不是偶函数（关于 $y$ 轴对称）
  5. 单调区间不能合并：递增区间是一段一段的（$[-\pi/2, \pi/2], [3\pi/2, 5\pi/2], \ldots$），不能写成"在所有递增区间上递增"

与初中知识的衔接

初中了解了正弦作为三角比（对边/斜边），高中将正弦扩展为定义在全体实数上的函数，研究其图象（正弦曲线）和性质。正弦函数是最基本的周期函数，是学习 $y = A\sin(\omega x + \varphi)$ 型函数的基础。
