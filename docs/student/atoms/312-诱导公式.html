🎓 诱导公式

定义

诱导公式（reduction formulas）：将任意角的三角函数值化为 $0$ 到 $\dfrac{\pi}{2}$ 之间的锐角三角函数值的一组公式。

核心口诀

"奇变偶不变，符号看象限"

  • 奇变偶不变：$k$ 为奇数时（$k \cdot \dfrac{\pi}{2} \pm \alpha$），函数名改变（$\sin \leftrightarrow \cos$，$\tan \leftrightarrow \cot$）；$k$ 为偶数时，函数名不变
  • 符号看象限：将 $\alpha$ 视为锐角时，$k \cdot \dfrac{\pi}{2} \pm \alpha$ 所在象限的函数符号

常用诱导公式（16个分4组）

第一组（$2\pi + \alpha$ 或 $-\alpha$）：

$$\sin(2k\pi + \alpha) = \sin\alpha \quad (k \in \mathbb{Z})$$

$$\cos(2k\pi + \alpha) = \cos\alpha \quad (k \in \mathbb{Z})$$

$$\sin(-\alpha) = -\sin\alpha \quad \text{（正弦为奇函数）}$$

$$\cos(-\alpha) = \cos\alpha \quad \text{（余弦为偶函数）}$$

第二组（$\pi + \alpha$）：

$$\sin(\pi + \alpha) = -\sin\alpha$$

$$\cos(\pi + \alpha) = -\cos\alpha$$

$$\tan(\pi + \alpha) = \tan\alpha$$

第三组（$\pi - \alpha$）：

$$\sin(\pi - \alpha) = \sin\alpha$$

$$\cos(\pi - \alpha) = -\cos\alpha$$

$$\tan(\pi - \alpha) = -\tan\alpha$$

第四组（$\dfrac{\pi}{2} \pm \alpha$）（奇数倍 $\pi/2$，函数名改变）：

$$\sin!\left(\frac{\pi}{2} + \alpha\right) = \cos\alpha$$

$$\cos!\left(\frac{\pi}{2} + \alpha\right) = -\sin\alpha$$

$$\sin!\left(\frac{\pi}{2} - \alpha\right) = \cos\alpha$$

$$\cos!\left(\frac{\pi}{2} - \alpha\right) = \sin\alpha$$

$$\tan!\left(\frac{\pi}{2} - \alpha\right) = \cot\alpha$$

推导思路（以 $\pi - \alpha$ 为例）

$\pi - \alpha$ 与 $\alpha$ 的终边关于 $y$ 轴对称。设 $\alpha$ 对应单位圆上点 $P(x, y)$，则 $\pi - \alpha$ 对应点 $P'(-x, y)$。故 $\sin(\pi-\alpha) = y = \sin\alpha$，$\cos(\pi-\alpha) = -x = -\cos\alpha$。

典型例题

例1：计算 $\sin 150°$，$\cos 240°$，$\tan 315°$。

解： - $\sin 150° = \sin(180° - 30°) = \sin 30° = \dfrac{1}{2}$ - $\cos 240° = \cos(180° + 60°) = -\cos 60° = -\dfrac{1}{2}$ - $\tan 315° = \tan(360° - 45°) = \tan(-45°) = -\tan 45° = -1$

例2：化简 $\sin(\pi + \alpha)\cos(\pi - \alpha)\tan(2\pi - \alpha)$。

解： - $\sin(\pi + \alpha) = -\sin\alpha$ - $\cos(\pi - \alpha) = -\cos\alpha$ - $\tan(2\pi - \alpha) = \tan(-\alpha) = -\tan\alpha = -\dfrac{\sin\alpha}{\cos\alpha}$

原式 $= (-\sin\alpha)(-\cos\alpha)\left(-\dfrac{\sin\alpha}{\cos\alpha}\right) = \sin\alpha \cdot \cos\alpha \cdot \left(-\dfrac{\sin\alpha}{\cos\alpha}\right) = -\sin^2\alpha$

例3：已知 $\sin\dfrac{11\pi}{6}$，利用诱导公式化简。

解：$\dfrac{11\pi}{6} = 2\pi - \dfrac{\pi}{6}$，故 $\sin\dfrac{11\pi}{6} = \sin!\left(2\pi - \dfrac{\pi}{6}\right) = \sin!\left(-\dfrac{\pi}{6}\right) = -\sin\dfrac{\pi}{6} = -\dfrac{1}{2}$。

易错点

  1. "奇变偶不变"看的是 $k$，不是 $\alpha$：$k \cdot \dfrac{\pi}{2} + \alpha$ 中，看 $k$ 的奇偶性来决定是否改变函数名
  2. 符号由"假设 $\alpha$ 为锐角时所在象限"决定：不是 $\alpha$ 实际所在象限
  3. $\pi + \alpha$ 与 $\pi - \alpha$ 区分：$\pi + \alpha$ 在第三象限（$\sin, \cos$ 都变负），$\pi - \alpha$ 在第二象限（只有 $\cos$ 变负）
  4. $\tan(\pi + \alpha) = \tan\alpha$ 不变号：第三象限 $\tan > 0$（与第一象限同）
  5. 化到锐角：诱导公式的目的是化为 $0 \sim \pi/2$ 的角，可能需要多次应用

与初中知识的衔接

初中只计算锐角的三角函数值，如 $\sin 30° = 0.5$。高中用诱导公式将所有角的三角函数化归到第一象限的锐角，实现"万角归一"，是学习三角函数计算的核心技巧。
