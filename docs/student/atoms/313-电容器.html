aliases: 电容器, capacitor

🎓 电容器（Capacitor）

定义

电容器是两个相互靠近、彼此绝缘的导体（极板）组成的储能元件。两极板之间夹有绝缘介质（空气、云母、陶瓷等）。

充电与放电

过程 操作 结果
充电 接上电源 两极板分别积累等量异号电荷，两板间建立电场，储存电场能量
放电 两端短接 电荷中和，电场消失，能量以电流形式释放

充电完成后，若与电源断开：极板电荷量 $Q$ 不变，电压 $U$ 不变，电场 $E$ 不变。

电容的定义式

$$C = \frac{Q}{U}$$

  • $C$：电容量（法拉，F）
  • $Q$：极板上的电荷量（C）
  • $U$：两板间电压（V）

电容是电容器储存电荷能力的量度，由电容器本身结构（面积、间距、介质）决定，与 $Q$、$U$ 无关。

常用单位换算

$$1\,\text{F} = 10^6\,\mu\text{F} = 10^{12}\,\text{pF}$$

实际电路常用 $\mu\text{F}$（微法）和 $\text{pF}$（皮法）。

储存的电场能量

$$E_C = \frac{1}{2}CU^2 = \frac{Q^2}{2C} = \frac{QU}{2}$$

在电路中的作用

  • 直流稳定电路：电容器相当于断路（充满电后无稳定电流）。
  • 交流电路：允许交流通过（频率越高，容抗越小）。
  • 应用：滤波、隔直、储能、定时电路等。

典型例题

例：某电容器 $C = 200\,\mu\text{F}$，充电后两极板间电压 $U = 50\,\text{V}$，求极板电荷量和储存的电场能量。

$$Q = CU = 200\times10^{-6}\times50 = 0.01\,\text{C}$$

$$E_C = \frac{1}{2}CU^2 = \frac{1}{2}\times200\times10^{-6}\times50^2 = 0.25\,\text{J}$$

易错点

  • 电容 $C$ 是固有属性，改变 $Q$ 或 $U$ 不改变 $C$（除非改变结构）。
  • 充电完毕断开电源后，$Q$ 不变，若改变极板间距，则 $U$ 改变，$C$ 改变。
  • 充电完毕保持与电源相连时，$U$ 不变，若改变极板间距，则 $Q$ 改变，$C$ 改变。

与必修二衔接

必修二学习了电能与功；电容器将电能以电场形式储存，体现了电场也能储存能量这一重要物理思想。
