事件的关系运算

🎓 事件的关系与运算

事件的关系

包含关系

若事件 $A$ 发生必然导致事件 $B$ 发生，则称 $A$ 包含于 $B$，记作 $A \subset B$（或 $B \supset A$）。

推论：$P(A) \leq P(B)$（若 $A \subset B$）

相等关系

$A = B$：$A \subset B$ 且 $B \subset A$，即两事件完全相同。

互斥关系（不相容）

$A \cap B = \emptyset$：$A$ 和 $B$ 不能同时发生，称 $A$ 与 $B$ 互斥。

加法公式（互斥）：$P(A \cup B) = P(A) + P(B)$

推广：$A_1, A_2, \ldots, A_n$ 两两互斥时：$P(A_1 \cup A_2 \cup \cdots \cup A_n) = \sum_{i=1}^n P(A_i)$

对立关系（互余）

$A$ 与 $\bar{A}$ 互为对立事件（complementary events）：

$$A \cup \bar{A} = \Omega, \quad A \cap \bar{A} = \emptyset$$

$$P(A) + P(\bar{A}) = 1 \implies P(\bar{A}) = 1 - P(A)$$

对立是互斥的特殊情形（还需要两者之和为全集）。

事件的运算

运算 符号 含义
并（和） $A \cup B$ $A$ 或 $B$ 至少一个发生
交（积） $A \cap B$ 或 $AB$ $A$ 与 $B$ 同时发生
差 $A - B$ 或 $A\bar{B}$ $A$ 发生但 $B$ 不发生
补 $\bar{A}$ $A$ 不发生

概率的加法公式（一般）

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

三个事件： $$P(A \cup B \cup C) = P(A) + P(B) + P(C) - P(AB) - P(AC) - P(BC) + P(ABC)$$

典型例题

例1：$P(A) = 0.3$，$P(B) = 0.4$，$P(AB) = 0.1$，求 $P(A \cup B)$。

解：$P(A \cup B) = 0.3 + 0.4 - 0.1 = 0.6$

例2：$A$ 与 $B$ 互斥，$P(A) = 0.4$，$P(B) = 0.3$，求 $P(A \cup B)$。

解：$P(A \cup B) = 0.4 + 0.3 = 0.7$（互斥时无需减去 $P(AB)$，因 $P(AB) = 0$）

例3："至少一个"类问题——求至少一人击中靶的概率。

策略：用对立事件，$P(\text{至少一人击中}) = 1 - P(\text{全未击中})$

甲乙击中概率各为 0.8，0.7，则 $P(\text{至少一人}) = 1 - 0.2 \times 0.3 = 0.94$

关系辨析

关系 互斥 对立
能同时发生？ 否 否
两者之和为全集？ 不一定 是
概率之和 $\leq 1$ $= 1$

易错点

  • 对立事件一定互斥，但互斥事件不一定对立（可能有第三种结果）
  • $P(A\bar{B}) = P(A) - P(AB)$（差事件公式）
  • "至少"类问题优先考虑对立事件法
