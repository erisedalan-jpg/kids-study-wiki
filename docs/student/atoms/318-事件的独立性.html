事件的独立性

权重10吉林命中1路径1学段[高中]主题[必修二, 概率, 第十章]

aliases: 事件的独立性, independence of events, 独立事件, 独立重复试验

🎓 事件的独立性

定义

若事件 $A$ 的发生不影响事件 $B$ 发生的概率，则称 $A$ 与 $B$ 相互独立。

数学定义：

$$A \text{ 与 } B \text{ 独立} \iff P(AB) = P(A) \cdot P(B)$$

独立的性质

若 $A$ 与 $B$ 独立，则以下事件对也独立： - $A$ 与 $\bar{B}$ - $\bar{A}$ 与 $B$ - $\bar{A}$ 与 $\bar{B}$

推论：$P(A\bar{B}) = P(A) \cdot P(\bar{B}) = P(A)[1 - P(B)]$

独立 vs 互斥

独立 互斥
定义 $P(AB) = P(A)P(B)$ $P(AB) = 0$（$A \cap B = \emptyset$）
含义 发生与否互不影响 不能同时发生
关系 互斥的非零概率事件不独立 独立事件不互斥（除非某概率为0）

若 $A, B$ 互斥且 $P(A) > 0, P(B) > 0$，则 $P(AB) = 0 \neq P(A)P(B)$，故互斥 $\neq$ 独立。

独立重复试验与二项分布

独立重复试验：在相同条件下，重复进行 $n$ 次独立试验，每次成功概率为 $p$。

$n$ 次试验中恰好成功 $k$ 次的概率（二项分布）：

$$P(X = k) = C_n^k p^k (1-p)^{n-k}, \quad k = 0, 1, 2, \ldots, n$$

记 $X \sim B(n, p)$，期望 $E(X) = np$，方差 $D(X) = np(1-p)$。

伯努利试验：只有两种结果（成功/失败）的独立重复试验。

典型例题

例1：甲乙射击，命中率分别为 0.8 和 0.7，两人各射一次，求：

（1）两人都命中；（2）至少一人命中；（3）恰好一人命中

解：（设甲命中为 $A$，乙命中为 $B$，$A, B$ 独立）

（1）$P(AB) = 0.8 \times 0.7 = 0.56$

（2）$P(A \cup B) = 1 - P(\bar{A}\bar{B}) = 1 - 0.2 \times 0.3 = 0.94$

（3）$P(A\bar{B}) + P(\bar{A}B) = 0.8 \times 0.3 + 0.2 \times 0.7 = 0.24 + 0.14 = 0.38$

例2（二项分布）：某产品次品率 10%，随机抽 5 件，恰好 2 件次品的概率。

解：$P(X=2) = C_5^2 \times 0.1^2 \times 0.9^3 = 10 \times 0.01 \times 0.729 = 0.0729$

例3：射手命中率 0.6，独立射击直到命中为止，求恰好第 3 次命中的概率。

解：$P = (1-0.6)^2 \times 0.6 = 0.16 \times 0.6 = 0.096$

易错点

  • 独立不是互斥，两者是不同的关系
  • 独立事件的积概率 $P(AB) = P(A)P(B)$（乘法），互斥事件 $P(AB) = 0$
  • 二项分布要求每次试验独立，概率恒为 $p$
