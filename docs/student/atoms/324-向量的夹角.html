向量的夹角

权重12吉林命中2路径0学段[高中]主题[必修二, 平面向量及其应用, 第六章]

aliases: 向量的夹角, angle between vectors, 向量夹角

🎓 向量的夹角

定义

两个非零向量 $\boldsymbol{a}$ 和 $\boldsymbol{b}$ 的夹角 $\theta$ 定义为将二者移至同一起点后所成的角：

$$\theta \in [0°, 180°]$$

夹角范围是 $[0°, 180°]$，不包括大于 $180°$ 的角。

夹角公式

$$\cos\theta = \frac{\boldsymbol{a} \cdot \boldsymbol{b}}{|\boldsymbol{a}||\boldsymbol{b}|}$$

坐标形式（$\boldsymbol{a} = (a_1, a_2)$，$\boldsymbol{b} = (b_1, b_2)$）：

$$\cos\theta = \frac{a_1 b_1 + a_2 b_2}{\sqrt{a_1^2 + a_2^2} \cdot \sqrt{b_1^2 + b_2^2}}$$

特殊角对应关系

夹角 $\cos\theta$ 关系
$0°$ $1$ 同向
$90°$ $0$ 垂直
$180°$ $-1$ 反向（共线）
$60°$ $\frac{1}{2}$ 常见题型
$120°$ $-\frac{1}{2}$ 常见题型

典型例题

例1：$\boldsymbol{a} = (1, \sqrt{3})$，$\boldsymbol{b} = (2, 0)$，求夹角 $\theta$。

解：
$\boldsymbol{a} \cdot \boldsymbol{b} = 2$，$|\boldsymbol{a}| = 2$，$|\boldsymbol{b}| = 2$
$\cos\theta = \dfrac{2}{4} = \dfrac{1}{2}$，故 $\theta = 60°$

例2：$|\boldsymbol{a}| = 2$，$|\boldsymbol{b}| = \sqrt{2}$，$\boldsymbol{a} \cdot \boldsymbol{b} = -2$，求夹角。

解：$\cos\theta = \dfrac{-2}{2\sqrt{2}} = -\dfrac{\sqrt{2}}{2}$，故 $\theta = 135°$

例3：已知三角形 $ABC$，$\overrightarrow{AB} = (1, 2)$，$\overrightarrow{AC} = (3, 1)$，求 $\angle BAC$。

解：
$\cos\angle BAC = \dfrac{1 \times 3 + 2 \times 1}{\sqrt{5} \cdot \sqrt{10}} = \dfrac{5}{5\sqrt{2}} = \dfrac{\sqrt{2}}{2}$，故 $\angle BAC = 45°$

易错点

  • 夹角是 $[0°, 180°]$ 范围内的角，不会是负角或大于 $180°$
  • 求夹角时先算 $\cos\theta$，再用反余弦得角度；不能反过来先判断角度
  • $\boldsymbol{a} \cdot \boldsymbol{b} > 0 \Rightarrow \theta$ 为锐角；$< 0 \Rightarrow$ 钝角；$= 0 \Rightarrow$ 直角
