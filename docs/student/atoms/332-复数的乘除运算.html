复数的乘除运算

🎓 复数的乘除运算

乘法法则

设 $z_1 = a + b\text{i}$，$z_2 = c + d\text{i}$：

$$z_1 \cdot z_2 = (a + b\text{i})(c + d\text{i}) = (ac - bd) + (ad + bc)\text{i}$$

推导（多项式展开，利用 $\text{i}^2 = -1$）：

$$= ac + ad\text{i} + bc\text{i} + bd\text{i}^2 = (ac - bd) + (ad + bc)\text{i}$$

除法法则

$$\frac{z_1}{z_2} = \frac{a + b\text{i}}{c + d\text{i}} = \frac{(a + b\text{i})(c - d\text{i})}{(c + d\text{i})(c - d\text{i})} = \frac{(ac + bd) + (bc - ad)\text{i}}{c^2 + d^2}$$

方法：分子分母同乘分母的共轭复数 $\bar{z}_2 = c - d\text{i}$，使分母化为实数。

分母：$(c + d\text{i})(c - d\text{i}) = c^2 + d^2$（恒为正实数）

运算律

  • 交换律：$z_1 z_2 = z_2 z_1$
  • 结合律：$(z_1 z_2)z_3 = z_1(z_2 z_3)$
  • 分配律：$z_1(z_2 + z_3) = z_1 z_2 + z_1 z_3$

模的乘除

$$|z_1 z_2| = |z_1||z_2|, \quad \left|\frac{z_1}{z_2}\right| = \frac{|z_1|}{|z_2|}$$

共轭复数乘积

$$z \cdot \bar{z} = (a + b\text{i})(a - b\text{i}) = a^2 + b^2 = |z|^2$$

这是"复数分母实数化"的核心公式。

典型例题

例1：$(1 + 2\text{i})(3 - \text{i})$

解：$= 3 - \text{i} + 6\text{i} - 2\text{i}^2 = 3 + 5\text{i} + 2 = 5 + 5\text{i}$

例2：$\dfrac{3 + \text{i}}{1 - 2\text{i}}$

解：
$$= \frac{(3 + \text{i})(1 + 2\text{i})}{(1 - 2\text{i})(1 + 2\text{i})} = \frac{3 + 6\text{i} + \text{i} + 2\text{i}^2}{1 + 4} = \frac{1 + 7\text{i}}{5} = \frac{1}{5} + \frac{7}{5}\text{i}$$

例3：$(1 + \text{i})^4$

解：$(1 + \text{i})^2 = 1 + 2\text{i} + \text{i}^2 = 2\text{i}$，故 $(1+\text{i})^4 = (2\text{i})^2 = -4$

易错点

  • 除法必须分子分母同乘共轭（不是乘分母本身）
  • 乘法展开后 $\text{i}^2 = -1$，不能漏换
  • $|z_1 z_2| = |z_1||z_2|$，但 $z_1 z_2$ 的辐角是两辐角之和（三角表示更清晰）
