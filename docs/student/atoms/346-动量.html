动量

权重22吉林命中3路径2学段[高中]主题[选必一, 动量守恒定律]

aliases: 动量, momentum, 线动量, 角动量

🎓 动量

定义

动量（momentum）是描述物体运动状态的物理量，等于质量与速度的乘积。

$$\vec{p} = m\vec{v}$$

  • $\vec{p}$：动量，单位 kg·m/s
  • $m$：质量，单位 kg（标量）
  • $\vec{v}$：速度，单位 m/s（矢量）

动量是矢量，方向与速度方向相同。

物理意义

动量衡量物体"运动的量"，综合考虑了质量和速度两个因素。

情形 质量 速度 动量大小
子弹 小 极大 较大
卡车 很大 较小 很大
静止物体 任意 0 0

同一物体，速度越大动量越大；同一速度，质量越大动量越大。

动量的计算

例1：质量 $m = 0.1\text{ kg}$ 的球以 $v = 20\text{ m/s}$ 向右运动，求动量。

$$p = mv = 0.1 \times 20 = 2\text{ kg·m/s，方向向右}$$

例2：质量 $m = 2\text{ kg}$ 的物体，速度由 $v_1 = 3\text{ m/s}$ 向右变为 $v_2 = 3\text{ m/s}$ 向左，求动量变化量。

取向右为正方向：

$$\Delta\vec{p} = m\vec{v_2} - m\vec{v_1} = 2\times(-3) - 2\times3 = -12\text{ kg·m/s}$$

即动量变化量大小为 12 kg·m/s，方向向左。

动量与动能的区别

物理量 公式 性质 单位
动量 $\vec{p} = m\vec{v}$ 矢量 kg·m/s
动能 $E_k = \frac{1}{2}mv^2$ 标量 J
  • 动量相同，动能不一定相同（如质量不同）
  • 动能相同，动量不一定相同（如质量不同）
  • 动量为零 → 动能必为零；动能为零 → 动量必为零

易错点

  1. 动量是矢量：计算动量变化时必须考虑方向，不能只用大小相减。
  2. 单位区分：动量单位 kg·m/s，不是焦耳（J）。
  3. 方向与速度一致：动量方向就是速度方向，不受力的方向影响。
