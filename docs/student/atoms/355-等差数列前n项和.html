🎓 等差数列前 $n$ 项和

公式推导（高斯求和法）

设等差数列首项 $a_1$，公差 $d$，前 $n$ 项和 $S_n$：

$$S_n = a_1 + a_2 + \cdots + a_n$$

$$S_n = a_n + a_{n-1} + \cdots + a_1$$

两式相加：$2S_n = n(a_1 + a_n)$，故

$$\boxed{S_n = \frac{n(a_1 + a_n)}{2}}$$

代入 $a_n = a_1 + (n-1)d$：

$$\boxed{S_n = na_1 + \frac{n(n-1)}{2}d}$$

两个公式选用：知道 $a_n$ 用第一个；只知 $a_1, d$ 用第二个。

$S_n$ 的函数性质

$S_n$ 是关于 $n$ 的二次函数（$d \neq 0$ 时）：

$$S_n = \frac{d}{2}n^2 + \left(a_1 - \frac{d}{2}\right)n$$

  • 常数项为 $0$（$n=0$ 时 $S_0 = 0$）
  • 对称轴：$n = -\dfrac{a_1}{d} + \dfrac{1}{2}$（与最值有关）

重要性质

性质1（等差中项型）：

$$S_n, S_{2n} - S_n, S_{3n} - S_{2n}$$ 成等差数列

性质2：若 $S_n$ 为 $n$ 的二次函数且常数项为 0，则 ${a_n}$ 是等差数列。

性质3（最值）：当公差 $d < 0$，$a_1 > 0$ 时，$S_n$ 有最大值，当 $a_k > 0, a_{k+1} \leq 0$ 时 $S_k$ 最大。

典型例题

例1：等差数列 $a_1 = 2$，$d = 3$，求 $S_{20}$。

解：$S_{20} = 20 \times 2 + \dfrac{20 \times 19}{2} \times 3 = 40 + 570 = 610$

例2：等差数列 $a_1 = 10$，$d = -1$，求 $S_n$ 的最大值。

解：$a_n = 10 - (n-1) = 11 - n$，令 $a_n > 0$：$n < 11$，令 $a_{11} = 0$。

$S_{10} = S_{11} = \dfrac{11(10+0)}{2} = 55$（$S_{10} = S_{11}$，两者均为最大值）

例3：已知 $S_n = 3n^2 - 2n$，求 $a_n$。

解：
$n \geq 2$：$a_n = S_n - S_{n-1} = 3n^2 - 2n - [3(n-1)^2 - 2(n-1)] = 6n - 5$
$n = 1$：$a_1 = S_1 = 1 = 6(1) - 5 = 1$ ✓
故 $a_n = 6n - 5$（公差 $d = 6$，首项 $a_1 = 1$）

例4：$1 + 2 + 3 + \cdots + 100 = ?$

解：$S_{100} = \dfrac{100 \times 101}{2} = 5050$（高斯求和）

易错点

  • 两个公式都要记牢，根据题目条件选用
  • $S_n$ 是关于 $n$ 的二次函数，常数项为 0（不是一般二次函数）
  • 最大值问题：找 $a_k \geq 0$ 且 $a_{k+1} < 0$ 的分界
