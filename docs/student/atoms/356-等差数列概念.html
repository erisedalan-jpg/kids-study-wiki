等差数列概念

权重40吉林命中14路径1学段[高中]主题[必修二, 数列, 第十一章]

aliases: 等差数列概念, arithmetic sequence, 等差数列, 公差

🎓 等差数列概念

定义

如果一个数列 ${a_n}$ 从第二项起，每一项与前一项的差都等于同一常数 $d$，则称这个数列为等差数列（arithmetic sequence），常数 $d$ 称为公差（common difference）。

$$a_n - a_{n-1} = d \quad (n \geq 2, d \text{ 为常数})$$

通项公式

$$a_n = a_1 + (n-1)d$$

其中 $a_1$ 为首项，$d$ 为公差，$n$ 为项数。

变形：$a_n = a_m + (n-m)d$（以任意项 $a_m$ 为基准）

等差数列的性质

性质1（等差中项）：若 $A = \dfrac{a+b}{2}$，则 $a, A, b$ 成等差，$A$ 称为 $a$ 与 $b$ 的等差中项。

性质2（线性性）：若 $m + n = p + q$，则 $a_m + a_n = a_p + a_q$。

性质3：等差数列的每一项是其相邻项的算术平均数：

$$a_n = \frac{a_{n-1} + a_{n+1}}{2}$$

性质4（图形特征）：以 $n$ 为横轴，$a_n$ 为纵轴，等差数列各点在一条直线上（斜率 $= d$，截距 $= a_1 - d$）。

判断等差数列

验证 $a_{n+1} - a_n =$ 常数，或通项公式 $a_n = An + B$（$A, B$ 为常数）的数列是等差数列。

典型例题

例1：等差数列 ${a_n}$，$a_1 = 3$，$d = 2$，求 $a_{10}$。

解：$a_{10} = 3 + (10-1) \times 2 = 3 + 18 = 21$

例2：等差数列中，$a_4 = 12$，$a_8 = 20$，求 $a_1$ 和 $d$。

解：$a_8 - a_4 = 4d = 8$，$d = 2$；$a_1 = a_4 - 3d = 12 - 6 = 6$

例3：等差数列 ${a_n}$，$a_3 + a_5 = 14$，求 $a_4$。

解：由等差中项，$a_3 + a_5 = 2a_4$，故 $a_4 = 7$

例4：$3, 7, 11, \ldots$ 求第 100 项。

解：$a_1 = 3$，$d = 4$，$a_{100} = 3 + 99 \times 4 = 399$

与初中衔接

初中学习了数列的基本概念，高中系统介绍等差、等比数列，是高考重要考点。

易错点

  • 公差 $d$ 可以为负数（递减等差数列）、零（常数列）
  • $a_n = a_1 + (n-1)d$ 中是 $(n-1)$，不是 $n$
  • 等差中项：$A = \frac{a+b}{2}$，不是 $A = a + b$
