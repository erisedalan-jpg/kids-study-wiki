弹性碰撞

权重13吉林命中2路径1学段[高中]主题[选必一, 动量守恒定律]

aliases: 弹性碰撞, elastic collision

🎓 弹性碰撞

定义

弹性碰撞（elastic collision）是碰撞前后动能完全守恒的碰撞，同时动量也守恒。

条件： - 动量守恒：$m_1v_1 + m_2v_2 = m_1v_1' + m_2v_2'$ - 动能守恒：$\frac{1}{2}m_1v_1^2 + \frac{1}{2}m_2v_2^2 = \frac{1}{2}m_1v_1'^2 + \frac{1}{2}m_2v_2'^2$

弹性碰撞的特点

  • 无能量损失（无形变、无热量、无声音）
  • 碰撞过程中弹性势能 → 动能完全恢复
  • 理想模型（实际中只有硬球接近弹性碰撞）

一维弹性碰撞的解析公式

联立动量守恒和动能守恒方程，解得：

$$v_1' = \frac{m_1 - m_2}{m_1 + m_2}v_1 + \frac{2m_2}{m_1 + m_2}v_2$$

$$v_2' = \frac{2m_1}{m_1 + m_2}v_1 + \frac{m_2 - m_1}{m_1 + m_2}v_2$$

（适用于 $m_2$ 初始静止时，令 $v_2 = 0$）

$$v_1' = \frac{m_1 - m_2}{m_1 + m_2}v_1 \quad v_2' = \frac{2m_1}{m_1 + m_2}v_1$$

三种特殊情形（$m_2$ 静止）

条件 $v_1'$ $v_2'$ 特点
$m_1 = m_2$ 0 $v_1$ 速度完全交换（台球）
$m_1 \gg m_2$ $\approx v_1$ $\approx 2v_1$ 重球几乎不减速，轻球获得 2 倍速度
$m_1 \ll m_2$ $\approx -v_1$ $\approx 0$ 轻球被弹回，重球几乎不动（弹球碰墙）

典型例题

例：台球碰撞，质量均为 $m = 0.1\text{ kg}$，$v_1 = 2\text{ m/s}$，$v_2 = 0$，弹性碰撞，求碰后速度。

由 $m_1 = m_2$：

$$v_1' = 0\text{ m/s}, \quad v_2' = 2\text{ m/s}$$

碰后 $m_1$ 停止，$m_2$ 以 2 m/s 向右运动——速度完全转移。

验证动能：$E_k = \frac{1}{2}\times0.1\times4 = 0.2\text{ J}$；$E_k' = \frac{1}{2}\times0.1\times4 = 0.2\text{ J}$ ✓

易错点

  1. 弹性碰撞要同时满足两个方程：仅动量守恒不够，还需动能守恒。
  2. "弹性"不是"弹力大"：弹性碰撞是指动能守恒的碰撞，与弹力大小无关。
  3. 完全非弹性碰撞动能损失最大：弹性碰撞动能损失为零，两者是极端情形。
