🎓 随机事件概率

基本概念

随机试验：在相同条件下可重复，结果不确定但所有可能结果已知。

样本空间 $\Omega$：随机试验所有可能结果的集合。

随机事件：样本空间的子集（$A \subset \Omega$），是试验中可能发生也可能不发生的事件。

特殊事件： - 必然事件：每次试验必然发生（$\Omega$ 本身） - 不可能事件：每次试验必然不发生（空集 $\emptyset$）

概率的定义

频率稳定性：随机事件 $A$ 在大量重复试验中，发生的频率趋向于一个固定的常数，这个常数称为事件 $A$ 的概率，记作 $P(A)$。

$$P(A) = \lim_{n \to \infty} \frac{f_n(A)}{n}$$

其中 $f_n(A)$ 为 $n$ 次试验中 $A$ 发生的频数。

概率的基本性质

$$0 \leq P(A) \leq 1$$

$$P(\Omega) = 1, \quad P(\emptyset) = 0$$

$$P(A) + P(\bar{A}) = 1 \quad \text{（对立事件）}$$

事件的运算

  • 和事件（并）：$A \cup B$，$A$ 或 $B$ 发生
  • 积事件（交）：$A \cap B$（或 $AB$），$A$ 且 $B$ 同时发生
  • 差事件：$A - B$（或 $A\bar{B}$），$A$ 发生但 $B$ 不发生
  • 对立事件：$\bar{A}$，$A$ 不发生（$A \cup \bar{A} = \Omega$，$A \cap \bar{A} = \emptyset$）

概率的加法公式

互斥事件（不相容事件）：$A \cap B = \emptyset$（不能同时发生）

$$P(A \cup B) = P(A) + P(B) \quad \text{（互斥时）}$$

一般情形：

$$P(A \cup B) = P(A) + P(B) - P(A \cap B)$$

典型例题

例1：抛一枚骰子，$A =$ "出现偶数"，$B =$ "出现 3 以上"，求 $P(A \cup B)$。

解：$A = {2,4,6}$，$B = {4,5,6}$，$A \cap B = {4,6}$

$P(A) = \frac{1}{2}$，$P(B) = \frac{1}{2}$，$P(A \cap B) = \frac{1}{3}$

$P(A \cup B) = \frac{1}{2} + \frac{1}{2} - \frac{1}{3} = \frac{2}{3}$

例2：$P(A) = 0.6$，$P(\bar{A}) = ?$

解：$P(\bar{A}) = 1 - 0.6 = 0.4$

易错点

  • 互斥 $\neq$ 对立：对立是互斥且互余（两者加起来是全集）
  • 两事件互斥时加法公式无 $P(AB)$ 项，一般情形要减去
  • 概率值必须在 $[0, 1]$ 之间，超出则计算有误
