🎓 双曲线几何性质

以标准双曲线 $\dfrac{x^2}{a^2} - \dfrac{y^2}{b^2} = 1$（$a>0, b>0$，焦点在 $x$ 轴）为例。

对称性

  • 关于 $x$ 轴对称
  • 关于 $y$ 轴对称
  • 关于原点（中心）对称

顶点与轴

实轴：两顶点 $A_1(-a,0), A_2(a,0)$ 的连线，实轴长 $= 2a$

虚轴：虚顶点 $B_1(0,-b), B_2(0,b)$ 的连线，虚轴长 $= 2b$

（虚顶点不在双曲线上，但用于作渐近线和辅助矩形）

范围

$$|x| \ge a \quad (\text{即} x \le -a \text{ 或 } x \ge a)$$

$y$ 无界（$y$ 可以取任意值，当 $|x|$ 足够大时）

焦半径公式

双曲线上点 $P(x_0, y_0)$，$|x_0| \ge a$：

$$\text{右支（}x_0 > 0\text{）}:\quad |PF_1| = a + ex_0,\quad |PF_2| = ex_0 - a$$

$$\text{左支（}x_0 < 0\text{）}:\quad |PF_1| = -(a + ex_0) = -a-ex_0,\quad |PF_2| = -(ex_0-a) = a-ex_0$$

（利用 $|PF_1|-|PF_2| = \pm 2a$ 及 $e=c/a$）

统一写法（注意正负号由 $P$ 所在支决定）。

焦点三角形

椭圆上点 $P$，$\triangle PF_1F_2$：

$$||PF_1| - |PF_2|| = 2a, \quad |F_1F_2| = 2c$$

  • 余弦定理求 $\angle F_1PF_2$：$|F_1F_2|^2 = |PF_1|^2 + |PF_2|^2 - 2|PF_1||PF_2|\cos\angle F_1PF_2$
  • 面积：$S = \dfrac{1}{2}|PF_1||PF_2|\sin\angle F_1PF_2 = b^2 \tan\dfrac{\angle F_1PF_2}{2}$（待家长核对）

通径

过焦点垂直于实轴的弦称为通径，长 $= \dfrac{2b^2}{a}$（与椭圆通径公式形式相同）

典型例题

例1：双曲线 $\dfrac{x^2}{9}-\dfrac{y^2}{7}=1$，点 $P$ 在右支，$|PF_1|=6$，求 $|PF_2|$。

解：$a=3$，右支满足 $|PF_1|-|PF_2|=2a=6$，故 $|PF_2|=6-6=0$？不对，再查：$|PF_1|-|PF_2|=2a=6$，$|PF_1|=6$，$|PF_2|=0$，这不可能（$P$ 在双曲线上不能到焦点距离为0）。

需重新设定：设 $|PF_2|=x$，则 $|PF_1|=x+6$（右支满足 $|PF_1|>|PF_2|$），$|PF_1|=6$ 则$|PF_2|=6-6=0$，矛盾。说明例题数据应设 $|PF_1|=9$，则 $|PF_2|=9-6=3$。

例2：双曲线 $\dfrac{x^2}{16}-\dfrac{y^2}{9}=1$，求通径长。

解：$a^2=16, b^2=9, a=4$，通径 $=\dfrac{2\times9}{4}=\dfrac{9}{2}$

例3：双曲线上 $P$ 点，$|PF_1|=3, |PF_2|=5$，且 $F_1, F_2$ 是两焦点，求双曲线实轴长。

解：$||PF_1|-|PF_2||=|3-5|=2=2a$，故 $a=1$，实轴长 $2a=2$。

易错点

  • 双曲线有两支，焦半径公式对左支和右支符号不同
  • $a$ 是实半轴（顶点到中心），不必比 $b$ 大
  • 通径公式 $\dfrac{2b^2}{a}$ 与椭圆相同，但 $b^2$ 来源不同（$b^2 = c^2 - a^2$）
