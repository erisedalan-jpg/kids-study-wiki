🎓 双曲线定义与方程

定义

双曲线：平面内到两定点 $F_1$、$F_2$ 的距离之差的绝对值等于常数 $2a$（$2a < |F_1F_2|$）的点的轨迹。

$$\bigl||PF_1| - |PF_2|\bigr| = 2a \quad (0 < 2a < |F_1F_2|)$$

两定点称为焦点，$|F_1F_2| = 2c$，条件要求 $a < c$。

与椭圆对比：

椭圆 双曲线
定义 距离之和 $= 2a$ 距离之差绝对值 $= 2a$
条件 $2a > 2c$ $2a < 2c$（即 $a < c$）
形状 封闭曲线 两支开曲线

标准方程

焦点在 $x$ 轴上（$a>0, b>0$，$c^2=a^2+b^2$）：

$$\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$$

焦点：$F_1(-c,0)$，$F_2(c,0)$；顶点：$(\pm a, 0)$

焦点在 $y$ 轴上（$a>0, b>0$，$c^2=a^2+b^2$）：

$$\frac{y^2}{a^2} - \frac{x^2}{b^2} = 1$$

焦点：$F_1(0,-c)$，$F_2(0,c)$；顶点：$(0, \pm a)$

判断焦点位置：看哪一项为正（$x^2$ 正 → 焦点在 $x$ 轴；$y^2$ 正 → 焦点在 $y$ 轴）

基本量关系

$$c^2 = a^2 + b^2 \quad (a>0, b>0, c>0)$$

注意：双曲线中 $c^2 = a^2 + \mathbf{b^2}$（比椭圆多一个加号！）

  • $a$：实半轴（顶点到中心距离）
  • $b$：虚半轴（辅助矩形的另一半边）
  • $c$：半焦距

双曲线中 $b$ 不必小于 $a$，可以 $a=b$（等轴双曲线）。

等轴双曲线

当 $a=b$ 时，$\dfrac{x^2}{a^2} - \dfrac{y^2}{a^2} = 1$，即 $x^2-y^2=a^2$，两条渐近线互相垂直。

典型例题

例1：双曲线方程 $\dfrac{x^2}{9} - \dfrac{y^2}{16} = 1$，求 $a, b, c$ 及焦点。

解：$a^2=9, b^2=16, a=3, b=4, c=\sqrt{9+16}=5$，焦点 $(\pm5, 0)$

例2：双曲线焦点 $F_1(-5,0), F_2(5,0)$，$a=4$，求方程。

解：$c=5, b^2=c^2-a^2=25-16=9$，方程：$\dfrac{x^2}{16}-\dfrac{y^2}{9}=1$

例3：双曲线上点 $P$，$|PF_1|=8, |PF_2|=2$，判断 $P$ 在哪一支。

解：$|PF_1|-|PF_2|=6=2a$，故 $a=3$；$P$ 满足 $|PF_1|>|PF_2|$，说明 $P$ 靠近 $F_2$，在右支（$F_2$ 侧的支）。

易错点

  • 双曲线 $c^2 = a^2 + b^2$（不是 $a^2 = b^2 + c^2$）
  • $b$ 不需要小于 $a$，$b > a$ 也合法
  • 双曲线有两支，定义中是"距离差的绝对值"
