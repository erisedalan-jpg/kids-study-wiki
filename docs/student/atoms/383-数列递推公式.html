🎓 数列递推公式

定义

递推公式（递推关系）：用前若干项表示后一项的公式。给定初始值，可以逐步求出所有项。

$$a_{n+1} = f(a_n) \quad \text{或} \quad a_{n+1} = f(a_n, a_{n-1}, \ldots)$$

与通项公式的区别： - 通项公式：$a_n = f(n)$，直接由 $n$ 求 $a_n$ - 递推公式：$a_{n+1} = g(a_n)$，由已知项推下一项，需给定初始值

常见递推类型

类型一：等差型

$$a_{n+1} = a_n + d \quad (d \text{ 为常数})$$

通项：$a_n = a_1 + (n-1)d$

类型二：等比型

$$a_{n+1} = q \cdot a_n \quad (q \text{ 为常数}, q \ne 0)$$

通项：$a_n = a_1 q^{n-1}$

类型三：一阶线性递推

$$a_{n+1} = p \cdot a_n + q \quad (p \ne 1)$$

解法（特征根法/不动点法）：找不动点 $r = pr + q$，$r = \dfrac{q}{1-p}$，令 $b_n = a_n - r$：

$$b_{n+1} = a_{n+1} - r = p(a_n - r) = p \cdot b_n$$

故 ${b_n}$ 是公比为 $p$ 的等比数列：$b_n = b_1 p^{n-1}$，$a_n = r + (a_1 - r)p^{n-1}$

例：$a_{n+1} = 2a_n + 1$，$a_1 = 1$。

不动点：$r = 2r+1$，$r=-1$，$b_n = a_n+1$，$b_{n+1}=2b_n$，$b_n=2b_1\cdot2^{n-1}=2^n$，$a_n=2^n-1$

类型四：$a_{n+1} = a_n + f(n)$（累加法）

$$a_n = a_1 + \sum_{k=1}^{n-1}f(k)$$

类型五：$a_{n+1} = a_n \cdot f(n)$（累乘法）

$$a_n = a_1 \cdot \prod_{k=1}^{n-1}f(k)$$

典型例题

例1：$a_1=2, a_{n+1}=3a_n-4$，求通项公式。

解：不动点 $r=3r-4$，$r=2$，令 $b_n=a_n-2$，$b_{n+1}=3b_n$，$b_1=0$。

故 $b_n=0$，$a_n=2$（常数列）。（因为初始值恰好等于不动点）

例2：$a_1=1, a_{n+1}=a_n+n$，求 $a_n$。

解：$a_n=1+\displaystyle\sum_{k=1}^{n-1}k=1+\dfrac{(n-1)n}{2}=\dfrac{n^2-n+2}{2}$

例3：$a_1=1, a_2=2, a_{n+2}=a_{n+1}+a_n$（斐波那契数列），写出前 8 项。

解：$1, 2, 3, 5, 8, 13, 21, 34$（此类递推一般无简单闭合公式，高中了解即可）

递推与数学归纳法的关系

递推公式常用于数学归纳法的证明： - 已知 $n=k$ 时命题成立 - 利用递推关系，推出 $n=k+1$ 时也成立

易错点

  • 递推公式必须给出初始值（$a_1$ 或 $a_1, a_2$ 等）才能确定数列
  • 一阶线性递推不动点法：注意 $p=1$ 时退化为等差，无需不动点法
  • 累加法的求和上限是 $n-1$（不是 $n$），最终加上 $a_1$
