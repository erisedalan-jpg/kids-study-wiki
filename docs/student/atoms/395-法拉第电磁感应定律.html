🎓 法拉第电磁感应定律（高中物理）

定律表述

法拉第电磁感应定律：闭合回路中感应电动势的大小，与穿过回路的磁通量的变化率成正比。

$$\varepsilon = \frac{\Delta\Phi}{\Delta t}$$

对于 $N$ 匝线圈：

$$\varepsilon = N\frac{\Delta\Phi}{\Delta t}$$

其中： - $\varepsilon$：感应电动势（V） - $\Delta\Phi$：磁通量变化量（Wb） - $\Delta t$：时间间隔（s） - $N$：线圈匝数

完整形式（含方向）

包含楞次定律方向信息的完整表达式：

$$\varepsilon = -N\frac{\Delta\Phi}{\Delta t}$$

负号表示感应电动势的方向与磁通量变化方向相反（阻碍原因）。

磁通量的计算

$$\Phi = BS\cos\theta$$

其中 $\theta$ 为磁感线方向与线圈面法线方向的夹角。

特殊情形： - 磁场垂直线圈平面：$\Phi = BS$ - 磁场平行线圈平面：$\Phi = 0$

导体切割磁感线的等效形式

当导体以速度 $v$ 切割磁感线时，单位时间内"扫过"的面积为 $\Delta S = Lv\Delta t$，

$$\Delta\Phi = B\Delta S = BLv\Delta t$$

$$\varepsilon = \frac{\Delta\Phi}{\Delta t} = BLv$$

这与安培力/洛伦兹力的微观推导结果一致。

历史背景

法拉第于 1831 年通过系列实验总结出这一规律（没有数学公式，靠实验归纳），麦克斯韦后来将其纳入电磁场方程组，赋予精确的数学形式。

典型例题

例1：一线圈匝数 $N = 200$，在 $t = 0.2$ s 内磁通量从 $0.05$ Wb 变为 $0.01$ Wb，求感应电动势。

$$\Delta\Phi = 0.01 - 0.05 = -0.04\text{ Wb}$$ $$|\varepsilon| = N\frac{|\Delta\Phi|}{\Delta t} = 200 \times \frac{0.04}{0.2} = 40\text{ V}$$

例2：线圈在匀强磁场中匀速转动（$B = 0.5$ T，$S = 0.1$ m²，$N = 50$，$\omega = 100$ rad/s），求最大电动势。

$$\varepsilon_{\max} = NBA\omega = 50 \times 0.5 \times 0.1 \times 100 = 250\text{ V}$$

（线圈平面平行于磁场时，$\Delta\Phi/\Delta t$ 最大）

与安培力公式的联系

法拉第定律（宏观） ↔ 安培力/洛伦兹力（微观）：两者描述同一电磁感应过程的不同层面，结论一致。

易错点

  1. $\varepsilon = \Delta\Phi / \Delta t$ 求的是平均感应电动势（$\Delta t$ 不能太大，若磁通量非线性变化则为平均值）。
  2. 感应电动势大小与磁通量本身无关，只与变化率 $\Delta\Phi/\Delta t$ 有关。
  3. 不要把公式写成 $\varepsilon = \Phi / t$（误），正确是 $\varepsilon = \Delta\Phi / \Delta t$。
