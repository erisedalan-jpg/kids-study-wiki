🎓 空间向量基本概念

定义

空间向量：在空间中，具有大小和方向的量称为空间向量（也称三维向量）。与平面向量类似，用有向线段表示，记作 $\overrightarrow{AB}$ 或 $\boldsymbol{a}$。

自由向量：空间向量是自由向量，只要大小相同、方向相同，则无论起点在何处，都视为同一向量（$\overrightarrow{AB} = \overrightarrow{CD}$ 当且仅当 $\overrightarrow{AB} \parallel \overrightarrow{CD}$，方向相同且 $|\overrightarrow{AB}|=|\overrightarrow{CD}|$）。

基本概念

概念 说明
零向量 $\boldsymbol{0}$ 长度为 0，方向任意
单位向量 长度为 1 的向量
相等向量 大小相等且方向相同
共线向量（平行向量） 方向相同或相反（包括零向量）
共面向量 若干向量平移后能放入同一平面

空间向量与平面向量的对比

平面向量 空间向量
维度 2维（$x,y$） 3维（$x,y,z$）
坐标表示 $(a_1, a_2)$ $(a_1, a_2, a_3)$
运算规则 加减/数乘/数量积 同，但无"向量积"（高中范围）
基底 两个不共线向量 三个不共面向量

空间中两向量共线的条件

$\boldsymbol{a}$ 与 $\boldsymbol{b}$ 共线（$\boldsymbol{b} \neq \boldsymbol{0}$）$\Leftrightarrow$ 存在实数 $\lambda$，使得 $\boldsymbol{a} = \lambda \boldsymbol{b}$。

典型例题

例1：判断以下说法是否正确：
"空间中所有长度为 1 的向量都相等。"

解：错误。相等向量不仅要求大小相同，还要求方向相同。空间中长度为 1 的向量方向可以任意，不一定相等。

例2：设 $\overrightarrow{OA} = \boldsymbol{a}$，$\overrightarrow{OB} = \boldsymbol{b}$，求 $\overrightarrow{AB}$ 用 $\boldsymbol{a},\boldsymbol{b}$ 表示。

解：$\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA} = \boldsymbol{b} - \boldsymbol{a}$

例3：已知 $\boldsymbol{a} = (1, 2, -3)$，$\boldsymbol{b} = (k, 4, -6)$，若 $\boldsymbol{a} \parallel \boldsymbol{b}$，求 $k$。

解：$\boldsymbol{a} \parallel \boldsymbol{b}$ 则 $\boldsymbol{b} = \lambda \boldsymbol{a}$，由 $4 = 2\lambda$ 得 $\lambda = 2$，故 $k = 1 \times 2 = 2$。

易错点

  • 空间向量是自由向量，起点不固定；与几何中的"有向线段"不同
  • 共线向量包含零向量，但不能用"方向平行"来定义零向量
  • 空间中两向量共线 $\Rightarrow$ 可平移到同一直线，但原始向量未必在同一直线上
