基因工程

权重31吉林命中5路径2学段[高中]主题[选必三, 基因工程]

aliases: 基因工程, genetic engineering, recombinant DNA technology

基因工程

一句话：在分子水平上，按照人类意愿将不同来源的DNA重新组合，并导入受体生物中表达的技术。 English: A set of molecular techniques that allow DNA from different sources to be combined and introduced into a host organism for expression.

🎓 高中进阶

定义

基因工程（Genetic Engineering）又称重组DNA技术，是指在体外将目的基因与载体DNA重新组合，然后导入受体细胞，使目的基因在受体中稳定遗传并表达，从而改造生物遗传性状或获得目的产物的技术。

基因工程是在分子（DNA）水平进行的操作，因此也称为分子育种。

原理 / 步骤 / 应用

四大工具（"剪刀、针线、运输工具、受体"）

工具 名称 作用
分子剪刀 限制性核酸内切酶（限制酶） 在特定位点切割DNA
分子针线 DNA连接酶 连接DNA片段，形成重组DNA
运输工具 载体（质粒、噬菌体等） 将目的基因导入受体细胞
受体 受体细胞（大肠杆菌、酵母、动植物细胞等） 接受并表达目的基因

基本操作步骤

1. 获取目的基因
   ├─ 从生物体基因组中直接分离（限制酶切割）
   └─ 人工合成（PCR扩增 / 化学合成）

2. 构建重组DNA分子
   目的基因 + 限制酶切割
   载体DNA + 同种限制酶切割
   ↓ DNA连接酶连接 → 重组DNA分子

3. 将重组DNA导入受体细胞
   ├─ 原核细胞（大肠杆菌）：Ca²⁺处理后热激转化
   ├─ 植物细胞：农杆菌介导 / 基因枪
   └─ 动物细胞：显微注射 / 病毒载体

4. 检测与筛选
   分子水平：PCR、Southern杂交
   个体水平：观察表现型

目的基因的表达

导入的目的基因在受体细胞中需经转录（mRNA）→ 翻译（蛋白质）才能表达。因此载体上需含有启动子、终止子等调控元件，确保目的基因正常转录。

应用举例

领域 示例
医药 大肠杆菌生产胰岛素、干扰素、乙肝疫苗
农业 转基因抗虫棉、转基因大豆（抗除草剂）
工业 转基因微生物降解石油污染物
环境 转基因植物修复重金属污染土壤

典型例题

例：在基因工程中，为什么常用大肠杆菌作为受体细胞？用大肠杆菌表达的人胰岛素蛋白与人体细胞产生的有何不同？

解析：大肠杆菌繁殖速度快、遗传背景清楚、操作简便、成本低，适合大量生产目的蛋白。但大肠杆菌为原核生物，缺乏真核细胞的糖基化等加工修饰功能，因此某些需要翻译后修饰的蛋白（如部分抗体）在大肠杆菌中表达的产物功能可能略有差异。胰岛素结构较简单，大肠杆菌表达的胰岛素经适当加工后与天然产品基本等效。

易错点

  • 基因工程在分子（DNA）水平操作，不涉及染色体数目变化，不是诱变育种或多倍体育种
  • 限制酶和DNA连接酶切割/连接的都是磷酸二酯键（不是氢键）
  • 目的基因必须连接启动子等调控序列才能正常表达；仅插入目的基因编码区不够
