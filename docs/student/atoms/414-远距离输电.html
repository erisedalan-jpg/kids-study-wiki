🎓 远距离输电（高中物理）

核心问题

电力从发电站传输到用户，输电线有电阻 $R_{\text{线}}$，会产生功率损耗：

$$P_{\text{损}} = I_{\text{线}}^2 R_{\text{线}}$$

为减小损耗，需减小输电电流 $I_{\text{线}}$。在输送功率 $P$ 一定的情况下：

$$I_{\text{线}} = \frac{P}{U_{\text{线}}}$$

提高输电电压 → 减小输电电流 → 减小线路损耗

输电流程

发电机（低压）→ 升压变压器 → 高压输电线 → 降压变压器 → 用户（低压）

具体电压等级（中国，待家长核对）： - 发电机：约 10 kV（大型机组） - 升压后：110 kV、220 kV、500 kV、1000 kV（特高压） - 用户：10 kV（工业）→ 380 V（三相工业）→ 220 V（单相民用）

定量分析

设输送功率 $P$，输电线电阻为 $R_{\text{线}}$，输电电压（升压后）为 $U$：

$$I = \frac{P}{U}$$

$$P_{\text{损}} = I^2 R_{\text{线}} = \frac{P^2 R_{\text{线}}}{U^2}$$

$$\Delta U_{\text{线}} = IR_{\text{线}} = \frac{PR_{\text{线}}}{U} \quad \text{（线路电压降）}$$

$$U_{\text{末端}} = U - \Delta U_{\text{线}} \quad \text{（到达降压变压器的电压）}$$

输电电压提高 $n$ 倍 → 电流减小 $n$ 倍 → 损耗减小 $n^2$ 倍

典型例题

例：某发电站发电功率 $P = 5\times10^5$ W，输电线总电阻 $R_{\text{线}} = 40$ Ω。

（1）若直接以 1000 V 输电，求线路损耗功率和损耗百分比。

$$I = \frac{P}{U} = \frac{5\times10^5}{1000} = 500\text{ A}$$ $$P_{\text{损}} = I^2R = 500^2\times40 = 10^7\text{ W} = 10000\text{ kW}$$

损耗远超发电量，不可行！

（2）改用升压变压器升至 10000 V 输电：

$$I = \frac{5\times10^5}{10000} = 50\text{ A}$$ $$P_{\text{损}} = 50^2\times40 = 100000\text{ W} = 100\text{ kW}$$

损耗 $= 100/500 = 20\%$，尚可接受

（3）升至 100000 V（100 kV）输电：

$$I = 5\text{ A}, \quad P_{\text{损}} = 25\times40 = 1000\text{ W} = 1\text{ kW}$$

损耗 $= 1/500 = 0.2\%$，非常低

高压输电的安全措施

  1. 输电线架设在高空（绝缘）
  2. 采用绝缘子（串型、盘型）
  3. 高压直流输电（HVDC）：减少交流损耗，适合超远距离
  4. 特高压直流（中国自主技术，世界领先）

易错点

  1. $P_{\text{损}} = I^2R_{\text{线}}$，不是 $U^2/R_{\text{线}}$（此处 $U$ 是线路两端电压降，不是输电电压）。
  2. "升压后电压"是升压变压器次级电压，不是发电机电压。
  3. 用户端（降压变压器初级）接收的电压 = 输电电压 - 线路电压降，不等于升压变压器次级电压。
