玻意耳定律

权重46吉林命中8路径2学段[高中]主题[选必三, 气体]

aliases: 玻意耳定律, Boyle's law, 等温变化

🎓 玻意耳定律

表述

在温度不变（等温过程）的条件下，一定质量的气体，压强与体积成反比：

$$\boxed{pV = C \quad (\text{T 不变，C 为常数})}$$

即：

$$p_1 V_1 = p_2 V_2$$

玻意耳定律（Boyle's Law）由英国科学家罗伯特·玻意耳（Robert Boyle）于 1662 年发现，是最早定量描述气体行为的实验定律。

适用条件

  • 气体质量一定（密闭系统，无漏气）
  • 温度不变（等温过程）
  • 气体可近似为理想气体（高温低压）

p-V 图像（等温线）

等温线是双曲线：

$$p = \frac{C}{V} \quad \Rightarrow \quad p\text{-}V \text{ 图为双曲线}$$

  • 同一等温线上，各点 $pV$ 值相等
  • 温度越高，等温线越靠右上方（$pV$ 值越大）

微观解释

等温条件下，分子平均动能不变。若压缩体积： - 单位体积内分子数增多 - 单位时间内撞击器壁的分子数增多 - 压强增大

典型例题

例题：某密闭气缸内气体压强为 $1.5 \times 10^5\ \text{Pa}$，体积为 $2\ \text{L}$。保持温度不变，将体积压缩至 $1\ \text{L}$，求末态压强。

解：

$$p_1 V_1 = p_2 V_2$$

$$p_2 = \frac{p_1 V_1}{V_2} = \frac{1.5 \times 10^5 \times 2}{1} = 3 \times 10^5\ \text{Pa}$$

例题 2（开口管封闭气柱）：长 $60\ \text{cm}$ 的竖直细管，上端封闭，下端开口，大气压 $p_0 = 75\ \text{cmHg}$。初始封闭气柱长 $20\ \text{cm}$，管竖直放置。若将管翻转（开口朝上），求封闭气柱长度。

解（待家长核对）： - 初态：气体在上端，初始压强 $p_1 = p_0 + \rho g l_{液} = 75 + 40 = 115\ \text{cmHg}$ （液柱长 40 cm，向下压气体） - 末态：翻转后开口朝上，气体在下端，压强 $p_2 = p_0 - \rho g l_{液}$，需联立求解

注：具体数值需核对题目中液体高度，以上为思路框架，待家长核对。

易错点

  1. 等温过程中，理想气体内能不变（$\Delta U = 0$），但做功和吸热可能不为零
  2. $p_1V_1 = p_2V_2$ 仅适用于温度不变的情况，温度改变要用完整状态方程
  3. 压强单位要统一：Pa 或 cmHg 选其一，不能混用
