理想气体状态方程

权重51吉林命中9路径2学段[高中]主题[选必三, 气体]

aliases: 理想气体状态方程, ideal gas law, 气体状态方程

🎓 理想气体状态方程

公式

$$\boxed{pV = nRT}$$

其中： - $p$：气体压强（Pa） - $V$：气体体积（m³） - $n$：气体物质的量（mol） - $R = 8.31\ \text{J/(mol·K)}$：摩尔气体常数 - $T$：热力学温度（K）

高中常用形式（同一质量气体两态比较）

$$\frac{p_1 V_1}{T_1} = \frac{p_2 V_2}{T_2}$$

（$n$ 相同消去，$R$ 相同消去）

这是三大气体定律的统一表达式。令其中某参量不变，即退化为玻意耳、查理或盖-吕萨克定律。

各特殊过程对应

过程 不变量 退化为
等温 $T_1=T_2$ $T$ $p_1V_1 = p_2V_2$（玻意耳定律）
等容 $V_1=V_2$ $V$ $p_1/T_1 = p_2/T_2$（查理定律）
等压 $p_1=p_2$ $p$ $V_1/T_1 = V_2/T_2$（盖-吕萨克定律）

与克拉伯龙方程的关系

$pV = nRT$ 是克拉伯龙方程的高中简化版（见克拉伯龙方程）。

若用分子数 $N = nN_A$ 表示：

$$pV = N k_B T \quad (k_B = R/N_A = 1.38 \times 10^{-23}\ \text{J/K})$$

典型例题

例题 1：一密闭气缸，初态：$p_1 = 2 \times 10^5\ \text{Pa}$，$V_1 = 3\ \text{L}$，$T_1 = 300\ \text{K}$。末态：$p_2 = 3 \times 10^5\ \text{Pa}$，$T_2 = 450\ \text{K}$，求末态体积 $V_2$。

解：

$$V_2 = \frac{p_1 V_1 T_2}{T_1 p_2} = \frac{2 \times 10^5 \times 3 \times 450}{300 \times 3 \times 10^5} = \frac{2 \times 3 \times 450}{300 \times 3} = 3\ \text{L}$$

（恰好回到原体积，因为 $p$ 和 $T$ 以相同比例增加。）

例题 2：用状态方程求物质的量：某气体 $p = 10^5\ \text{Pa}$，$V = 22.4\ \text{L}$，$T = 273\ \text{K}$，求 $n$。

$$n = \frac{pV}{RT} = \frac{10^5 \times 22.4 \times 10^{-3}}{8.31 \times 273} \approx 1\ \text{mol}$$

（标准状态下 $1\ \text{mol}$ 气体体积约 $22.4\ \text{L}$，与化学知识吻合。）

解题步骤

  1. 确认系统为密闭气体（质量不变）
  2. 列出初末态参量（$p_1, V_1, T_1$ 和 $p_2, V_2, T_2$）
  3. 温度换算为 K：$T = t + 273$
  4. 代入 $\dfrac{p_1 V_1}{T_1} = \dfrac{p_2 V_2}{T_2}$ 求解未知量

易错点

  1. 温度必须用 K，不得用摄氏度
  2. 单位要统一：压强用 Pa，体积用 m³（或都用 L，注意前后一致）
  3. 若气体质量变化（漏气或混入气体），不能用 $\frac{p_1V_1}{T_1} = \frac{p_2V_2}{T_2}$，需用 $pV = nRT$ 分别计算
