正态分布概念

aliases: 正态分布概念, normal distribution, 正态分布, 高斯分布, N(μ, σ²)

🎓 正态分布概念

定义

若连续型随机变量 $X$ 的概率密度函数为：

$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} e^{-\frac{(x-\mu)^2}{2\sigma^2}} \quad (-\infty < x < +\infty)$$

则称 $X$ 服从正态分布，记作 $X \sim N(\mu, \sigma^2)$，其中 $\mu$ 为均值，$\sigma^2$ 为方差（$\sigma > 0$）。

期望和方差：$E(X) = \mu$，$D(X) = \sigma^2$

标准正态分布

当 $\mu = 0$，$\sigma = 1$ 时，称为标准正态分布 $N(0,1)$，密度函数为：

$$\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2/2}$$

标准化：若 $X \sim N(\mu, \sigma^2)$，令 $Z = \dfrac{X - \mu}{\sigma}$，则 $Z \sim N(0,1)$。

正态分布的特征

  1. 对称性：密度曲线关于 $x = \mu$ 对称
  2. 单峰：最高点在 $x = \mu$（均值、中位数、众数三合一）
  3. 钟形曲线（bell curve）：以 $\mu$ 为中心向两侧逐渐降低
  4. 拐点：曲线在 $x = \mu \pm \sigma$ 处有拐点
  5. 总面积 = 1：$\int_{-\infty}^{+\infty} f(x)dx = 1$

参数的影响

参数变化 对图形的影响
$\mu$ 增大 曲线整体右移（峰值位置右移）
$\sigma$ 增大 曲线变矮变宽（更分散）
$\sigma$ 减小 曲线变高变窄（更集中）

口诀：$\mu$ 定位置，$\sigma$ 定形状。

现实中的正态分布

许多自然现象近似服从正态分布： - 人群身高、体重 - 测量误差 - 产品尺寸（工业生产） - 考试成绩（大规模样本） - 股票日收益率（粗略近似）

中心极限定理（简述）：大量独立同分布随机变量之和（如 $n$ 重伯努利试验），当 $n$ 很大时，其分布近似正态分布。这解释了正态分布的普遍性。

正态分布概率的计算

由于密度函数无法用初等函数积分，通常通过查标准正态分布表（$\Phi$ 函数表）计算：

$$\Phi(x) = P(Z \leq x) = \int_{-\infty}^x \varphi(t)dt$$

对称性：$\Phi(-x) = 1 - \Phi(x)$

典型例题

例1：$X \sim N(10, 4)$（即 $\mu = 10, \sigma = 2$），求 $P(X \leq 14)$。

$$P(X \leq 14) = P\left(\frac{X-10}{2} \leq \frac{14-10}{2}\right) = P(Z \leq 2) = \Phi(2) \approx 0.9772$$

例2：$X \sim N(\mu, \sigma^2)$，求 $P(X \leq \mu)$。

由对称性：$P(X \leq \mu) = 0.5$

例3（参数估计）：某校学生身高 $X \sim N(170, 25)$（$\mu = 170$，$\sigma = 5$），求身高在 165 到 175 之间的概率（查表 $\Phi(1) = 0.8413$）。

$$P(165 \leq X \leq 175) = P\left(-1 \leq \frac{X-170}{5} \leq 1\right) = \Phi(1) - \Phi(-1)$$ $$= 0.8413 - (1-0.8413) = 0.8413 - 0.1587 = 0.6826$$

易错点

  1. $\sigma^2$ 是方差，不是 $\sigma$：$N(\mu, \sigma^2)$ 中第二个参数是方差，不是标准差
  2. 标准化方向：$Z = (X-\mu)/\sigma$，减均值除标准差，不是除方差
  3. 对称性使用：$P(Z \leq -a) = 1 - P(Z \leq a) = 1 - \Phi(a)$
  4. 正态分布不是离散的：$P(X = c) = 0$（连续型随机变量单点概率为零）
