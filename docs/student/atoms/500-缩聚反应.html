缩聚反应

权重10吉林命中1路径2学段[高中]主题[选必三, 第五章]

aliases: 缩聚反应, condensation polymerization

缩聚反应

一句话：缩聚反应是含有两个或多个官能团的单体通过官能团间的缩合（失去小分子）反复连接形成高分子的反应，典型例子有涤纶（乙二醇+对苯二甲酸）和尼龙（氨基酸缩聚）。 English: Condensation polymerization links monomers bearing two or more functional groups by repeated condensation reactions that expel small molecules (like H₂O or HCl); examples include polyester (PET) and nylon.

🎓 高中进阶

定义 / 结构特征

缩聚反应（condensation polymerization）： - 单体：含 ≥2 个官能团（—OH、—COOH、—NH₂ 等）的化合物 - 机理：官能团间发生缩合反应，脱去小分子（H₂O、HCl、CH₃OH 等） - 产物：高分子 + 副产物小分子 - 与加聚反应区别：有小分子副产物

缩聚反应类型： 1. 酯化缩聚（二元醇 + 二元酸）→ 聚酯（如涤纶 PET） 2. 酰胺化缩聚（二元胺 + 二元酸）→ 聚酰胺（如尼龙 Nylon） 3. 氨基酸自身缩聚→ 聚酰胺（α-氨基酸）

性质 / 反应

涤纶（PET）的缩聚： nHO—CH₂CH₂—OH + nHOOC—C₆H₄—COOH → (—OCH₂CH₂OOC—C₆H₄—CO—)ₙ + 2nH₂O （乙二醇 + 对苯二甲酸 → 聚对苯二甲酸乙二醇酯（PET）+ 水）

尼龙-6,6 的缩聚： nH₂N(CH₂)₆NH₂ + nHOOC(CH₂)₄COOH → (—NH(CH₂)₆NHCO(CH₂)₄CO—)ₙ + 2nH₂O （己二胺 + 己二酸 → 尼龙-6,6 + 水）

氨基酸缩聚（蛋白质类比）： nH₂N—CHR—COOH → (—NH—CHR—CO—)ₙ + nH₂O

对比表：

性质 加聚反应 缩聚反应
单体要求 含不饱和键 含 ≥2 个官能团
副产物 无 有（H₂O 等）
产物 全部原子保留 部分原子以副产物失去
举例 聚乙烯、PVC 涤纶、尼龙

典型例题

例：涤纶（PET）的制备，写出方程式，并指出副产物。

解： nHOCH₂CH₂OH + nHOOC-C₆H₄-COOH →(催化剂，△) [—OCH₂CH₂OOC-C₆H₄-CO—]ₙ + 2nH₂O 副产物：H₂O（2n mol）

例：尼龙-6 由 ε-己内酰胺（环状酰胺，H₂N(CH₂)₅COOH 的环形）开环聚合，属于缩聚还是加聚？

解：ε-己内酰胺开环聚合（ring-opening polymerization），无小分子副产物脱出，但机理类似加成（环打开），高中阶段通常归为缩聚（因产品含酰胺键），但严格说是开环聚合（介于两者之间）。

例：如何由结构单元判断缩聚物的单体？

解： 1. 看结构单元中的特征键（酯键—COO—、酰胺键—CO—NH—） 2. 在键处断开，各补 H 或 OH（或 NH₂） 3. 还原得单体

如涤纶重复单元—OCH₂CH₂OOC-C₆H₄-CO—：断酯键→ HO-CH₂CH₂-OH（乙二醇）+ HOOC-C₆H₄-COOH（对苯二甲酸）。

易错点

  • 缩聚反应的高聚物中，每个重复单元少于单体原子数（因脱去 H₂O 等）
  • 加聚（无副产物）vs 缩聚（有副产物）：区分方法是看单体是否含不饱和键（加聚）还是两种以上官能团（缩聚）
  • 蛋白质类比：氨基酸缩聚类似于缩聚反应（每形成一个肽键脱 1 个 H₂O）
