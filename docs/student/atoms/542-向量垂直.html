向量垂直

一句话：两个非零向量垂直当且仅当它们的数量积为零，坐标形式为 $x_1x_2 + y_1y_2 = 0$。 English: Two nonzero vectors are perpendicular (orthogonal) if and only if their dot product equals zero.

🎓 考点精讲

  • 定义/原理：设 $\vec{a}$ 与 $\vec{b}$ 为非零向量，若它们的夹角 $\theta = \frac{\pi}{2}$，则称 $\vec{a}$ 与 $\vec{b}$ 垂直，记作 $\vec{a} \perp \vec{b}$。由数量积定义 $\vec{a}\cdot\vec{b} = |\vec{a}||\vec{b}|\cos\theta$，得 垂直充要条件：
    $$\vec{a} \perp \vec{b} \iff \vec{a} \cdot \vec{b} = 0$$
    在平面直角坐标系中，若 $\vec{a} = (x_1, y_1)$，$\vec{b} = (x_2, y_2)$，则
    $$\vec{a} \perp \vec{b} \iff x_1x_2 + y_1y_2 = 0$$
    该条件与斜率表达式等价：斜率存在时 $k_1 k_2 = -1$，但向量垂直的坐标条件更一般，不受斜率不存在情形的限制。

  • 关键方法：

  • 垂直判定：直接计算数量积是否为零。
  • 参数求解：已知 $\vec{a} \perp \vec{b}$，代入坐标方程 $x_1x_2 + y_1y_2 = 0$，解出未知参数。

  • 几何转化：在平面几何或立体几何中，将垂直关系（如线线垂直、线面垂直）转化为向量数量积为零来列方程。

  • 典型应用：

  • 选择题/填空题：给出两个向量的含参坐标，利用垂直求参数值。例如“已知 $\vec{a}=(1, t)$，$\vec{b}=(-2, 3)$，且 $\vec{a}\perp\vec{b}$，求 $t$”。
  • 解答题：在解析几何或三角函数背景下，将图形中的垂直条件（如菱形对角线互相垂直、圆的直径所对圆周角为直角）用向量垂直表示，简化计算。

⚠️ 高频易错点

  • 遗漏“非零”前提：零向量与任何向量的数量积为 $0$，但零向量方向不确定，不能由此推出垂直几何关系。题目若未说明向量非零，不能直接由 $a\cdot b=0$ 断言垂直；反之，给定垂直通常已默认向量非零。
  • 混淆平行与垂直条件：容易将垂直的坐标条件 $x_1x_2+y_1y_2=0$ 与平行的比例条件 $x_1y_2 - x_2y_1=0$ 记反，导致判断错误。
  • 忽略斜率不存在的情况：若直接用 $k_1k_2=-1$ 判断，当遇水平或竖直向量（斜率为 $0$ 或不存在）时公式失效，而向量数量积形式永远适用。
  • 向量与几何垂直的不等价转化：在立体几何中，直线垂直未必直接对应方向向量数量积为零（还需考虑异面情况），需先确认共面或建立正确坐标系。
