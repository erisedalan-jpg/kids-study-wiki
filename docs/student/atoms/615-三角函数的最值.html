三角函数的最值

权重17吉林命中4路径0学段[高中]主题[]

aliases: "三角函数的最值", "Maximum and Minimum Values of Trigonometric Functions"

三角函数的最值

一句话：在给定区间或定义域内，求三角函数表达式所能达到的最大值与最小值的问题，核心是通过恒等变换、换元或利用函数性质转化为熟悉的基本函数最值模型。 English: The problem of finding the maximum and minimum values that a trigonometric expression can attain within a given interval or domain, typically solved by transforming the expression into a standard form using identities, substitution, or properties of trigonometric functions.

🎓 考点精讲

定义与原理
三角函数的最值本质上是根据正弦、余弦函数的有界性（$|\sin x| \le 1$，$|\cos x| \le 1$）以及变换后的复合函数性质，确定表达式 $y = A\sin(\omega x + \varphi) + k$ 或类似形式在指定范围内的最大值与最小值。其中 $A$ 决定振幅，$k$ 使图像上下平移，$\omega, \varphi$ 影响周期与相位，但最值主要取决于 $A$ 和 $k$：当 $\sin(\omega x + \varphi) = 1$ 时取得最大值 $|A| + k$（需注意 $A$ 符号，严格最大值是 $|A|+k$ 或 $ -|A|+k$，具体视 $A$ 的正负而定） ；当 $\sin(\omega x + \varphi) = -1$ 时取得最小值 $-|A| + k$。若有限制区间，则需结合单调性和单位圆确定最值。

关键方法
1. 辅助角公式（化一法）：形如 $y = a\sin x + b\cos x$ 可化为 $\sqrt{a^2+b^2}\sin(x+\varphi)$（或余弦形式），其中 $\tan\varphi = \frac{b}{a}$，再利用有界性求最值。特别注意 $\varphi$ 的象限由 $(a, b)$ 符号确定。
2. 换元法：对于 $y = a\sin^2 x + b\sin x + c$ 或 $y = a\cos^2 x + b\cos x + c$，令 $t = \sin x$（或 $\cos x$），则 $t \in [-1,1]$，转化为二次函数在闭区间上的最值问题，须验证对称轴是否在 $[-1,1]$ 内。
3. 利用函数单调性：当 $x$ 被限制在某区间（如 $[0, \frac{\pi}{2}]$）时，可直接分析 $\sin x, \cos x$ 的单调性求最值，或通过整体代换 $\omega x + \varphi$ 的对应区间，结合正弦/余弦曲线的增减性求解。
4. 导数法：对较复杂的三角函数组合，可求导并解 $y'=0$，再比较驻点与端点值。
5. 三角形中的最值：结合正弦定理、余弦定理将边的关系转化为角的三角函数，然后按上述方法求角或边的范围。

典型应用
高考中常以解答题的一个环节，或选择填空题出现。例如：
- “求函数 $f(x)=\sin 2x + \sqrt{3}\cos 2x$ 在 $[0, \frac{\pi}{2}]$ 上的最大值与最小值”，先用辅助角公式化为 $2\sin(2x+\frac{\pi}{3})$，再由 $x$ 范围确定 $2x+\frac{\pi}{3} \in [\frac{\pi}{3}, \frac{4\pi}{3}]$，利用正弦的图像求得最大值为 $2$，最小值为 $-\sqrt{3}$。
- “在 $\triangle ABC$ 中，已知 $a=2, \sin B = \sqrt{3}\sin C$，求 $\triangle ABC$ 面积的最大值”，化边用余弦定理，最终转换为角 $A$ 的三角函数求最值。

⚠️ 高频易错点

  • 忽略三角函数的有界性：换元时忘记 $t=\sin x \in [-1,1]$，错误地把二次函数的定义域当成全体实数，导致最值计算错误。
  • 辅助角公式的 $\varphi$ 确定错误：仅根据 $\tan\varphi = \frac{b}{a}$ 取锐角，导致表达式与原始不等价，最值所对应的 $x$ 出现偏差。
  • 不讨论 $A$ 的符号：直接将 $y = A\sin(\omega x+\varphi)+k$ 的最大值写成 $A+k$，最小值写成 $-A+k$，而忽略 $A$ 可能为负的情形。
  • 区间端点处理不当：当限定区间时，未比较区间内驻点与端点处的函数值，或错误地认为最大值一定在端点取到。
  • 复合函数定义域遗漏：如含 $\tan x$ 或分母含 $\sin x$ 的表达式求最值，忘记剔除无定义的点。
