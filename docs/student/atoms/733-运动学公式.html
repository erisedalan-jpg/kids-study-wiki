运动学公式

一句话：运动学公式是描述物体位置、速度、加速度与时间之间定量关系的一组物理方程，核心适用于匀变速直线运动。 English: Kinematic equations are a set of formulas that quantitatively relate displacement, velocity, acceleration, and time for an object in uniformly accelerated rectilinear motion.

🎓 考点精讲

定义/原理
运动学公式的核心是匀变速直线运动的五个基本物理量：初速度 $v_0$、末速度 $v$、加速度 $a$、位移 $x$、时间 $t$。
基本公式：
1. 速度-时间关系：$v = v_0 + at$
2. 位移-时间关系：$x = v_0 t + \frac{1}{2}at^2$
3. 速度-位移关系：$v^2 - v_0^2 = 2ax$
4. 平均速度求位移：$x = \frac{v_0+v}{2}t$ （仅适用于匀变速直线运动）

所有矢量均需选定正方向，与正方向相同的物理量取正值，相反取负值。公式只对匀变速直线运动严格成立，非匀变速场景需用微元思想或图像辅助。

关键方法
解决高考运动学问题的标准步骤：
1. 画运动过程示意图，标出已知量、待求量。
2. 规定统一的正方向，将各矢量化成代数量。
3. 判断运动性质是否为匀变速直线运动，若涉及多段过程，分段选用恰当公式，优先选择不含时间的公式（$v^2 - v_0^2 = 2ax$）以简化运算。
4. 联立求解，结果需还原为带有方向的物理量（如速度的正负表示方向）。

典型应用
高考常考两类情境：
- 刹车类问题：给定初速度、加速度（减速时a为负），求刹车位移或停车时间。必须先通过 $0 = v_0 + at$ 计算实际运动时间，判断所求时刻物体是否已停止，不能盲目将时间直接代入位移公式。
- 自由落体与竖直上抛：本质是 $v_0=0$ 或a=-g的匀变速直线运动，可分段使用运动学公式求解，也可全程用位移公式处理矢量性。

⚠️ 高频易错点

  • 不规定正方向就直接列式：速速、位移、加速度的正负号混乱导致全式错误，必须养成先设正方向的习惯。
  • 刹车问题直接代入给定时间计算位移：未先判定车辆在所求时刻前已停止，落入“刹车陷阱”，得出汽车反向倒退的荒谬结论。
  • 混淆运动学公式的适用条件：将平均速度公式 $x=\frac{v_0+v}{2}t$ 误用于非匀变速运动；或在不知道是否为匀变速时随意使用速度-位移公式。
