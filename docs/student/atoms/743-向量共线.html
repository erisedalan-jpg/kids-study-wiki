向量共线

一句话：如果两个向量方向相同或相反，则称这两个向量共线（也称平行），记作 a∥b；对于非零向量 b，a 与 b 共线当且仅当存在实数 λ 使得 a = λb。 English: Two vectors are collinear (or parallel) if they point in the same or opposite direction, meaning one is a scalar multiple of the other.

🎓 考点精讲

定义与原理 向量共线（平行）是高考平面向量部分的基础概念。设有非零向量 a 与 b，则 a∥b ⇔ 存在唯一的非零实数 λ，使 a = λb。零向量与任意向量共线。在坐标表示下，若 a = (x₁, y₁)，b = (x₂, y₂)，且 b ≠ 0，则 a∥b ⇔ x₁y₂ - x₂y₁ = 0。这一充要条件可拆分为比例形式：当 x₂、y₂ 均不为 0 时，共线等价于 x₁/x₂ = y₁/y₂；当出现分母为零时，必须用交叉相乘式 x₁y₂ - x₂y₁ = 0 统一处理。

关键方法 1. 共线判定：直接验证是否存在 λ 或坐标是否满足 x₁y₂ - x₂y₁ = 0。注意必须先确认向量非零（或分情况讨论零向量）。 2. 三点共线：若 A、B、C 三点满足任意有公共起点的两个向量共线（如 AB⃗ = λ AC⃗），则三点共线；反之，三点共线可推出对应向量系数关系。解题时常设 AB⃗ = λBC⃗ 或利用定比分点公式。 3. 参数求解：若已知含参向量共线，直接联立坐标等式，消去 λ 或用比例关系求解参数，避免遗漏分母为零的特殊情况。

典型应用 在高考中，该考点常以选择题、填空题或解答题的中间步骤出现。常见设问：①给定两向量坐标，问它们是否共线或求参数值；②在三角函数或解析几何背景下，利用向量共线转化出点的坐标关系，进而求斜率、方程或最值；③三点共线证明——如证明平面内三点共线，只需证向量 AC⃗ 与 BC⃗ 共线。例如，已知点 A(1,2), B(3,4), C(k,6) 共线，由 AB⃗=(2,2), AC⃗=(k-1,4)，由共线条件得 2×4 - 2(k-1)=0，解得 k=5。

⚠️ 高频易错点

  • 忽略零向量的特殊性：直接使用比例形式 x₁/x₂ = y₁/y₂ 而不检查分母为零的情况，导致漏解。必须使用 x₁y₂ - x₂y₁ = 0 进行判定。
  • 误用共线条件求定理反向：由 a = λb 推出共线时，必须确保 b ≠ 0，否则无法唯一确定 λ。
  • 三点共线中起点不统一：仅凭 AB⃗ 与 CD⃗ 共线不能推出 A、B、C、D 共线，必须使用有公共起点的向量（如 AB⃗ 与 AC⃗）。
  • 混淆共线与重合：向量共线不要求直线重合，只要求方向相同或相反，对应直线可以平行。
