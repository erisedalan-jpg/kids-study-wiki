离子方程式书写

一句话：用实际参加反应的离子符号表示离子反应的方程式，遵循“写、拆、删、查”四步法。 English: A chemical equation that represents only the ions and molecules directly involved in a reaction in aqueous solution, obtained through a standard four-step procedure.

🎓 考点精讲

定义与原理

离子方程式是用实际参加反应的离子符号表示离子反应的式子。其核心在于：在溶液中，电解质以离子形式参与反应，而实际发生变化的仅是部分离子。书写时只保留真正参与反应的微粒，删去旁观离子。判断依据是通过“拆”与“不拆”来区分微粒的真实存在形态——强酸、强碱、可溶性盐在水中完全电离，必须拆成离子形式；而弱电解质(弱酸、弱碱、水)、难溶物、气体、氧化物、单质等则保留化学式。最终必须满足原子守恒、电荷守恒、得失电子守恒(若为氧化还原反应)。

关键方法——“写拆删查”四步法

  • 写：写出正确的化学方程式，并配平。
  • 拆：将易溶且易电离的物质拆成离子形式。必须拆：强酸(HCl、HNO₃、H₂SO₄、HBr、HI、HClO₄)、强碱(NaOH、KOH、Ba(OH)₂、Ca(OH)₂澄清石灰水时)、可溶性盐(钾盐、钠盐、铵盐、硝酸盐、多数氯化物/硫酸盐等)；保留化学式：弱酸/弱碱(H₂CO₃、CH₃COOH、NH₃·H₂O)、水(H₂O)、难溶物(沉淀)、气体(↑)、氧化物(Fe₂O₃、Al₂O₃)、单质(Fe、Cl₂)、非电解质。
  • 删：删除方程式两边未参加反应的相同离子(旁观离子)，使方程式最简。
  • 查：检查质量(原子种类和个数)守恒、电荷守恒(方程式两边总电荷相等)、氧化还原反应得失电子守恒。

高考典型设问与场景应用

高考中该考点常见题型包括：选择题中判断离子方程式正误、非选择题中书写陌生反应的离子方程式。前者集中考查“拆”的正确性、守恒和对反应原理(是恰好沉淀、过量、还是氧化还原)的理解；后者经常出现在化工流程题和实验探究题中，例如：

  • 场景一：向FeCl₂溶液中通入Cl₂。化学方程式为2FeCl₂ + Cl₂ → 2FeCl₃。书写离子方程式时先把可溶性强电解质拆开：Fe²⁺+Cl⁻和Cl₂，生成Fe³⁺+Cl⁻，删去Cl⁻得 2Fe²⁺ + Cl₂ → 2Fe³⁺ + 2Cl⁻，电荷+4→+4，原子守恒。
  • 场景二：NaHCO₃与稀盐酸反应。HCO₃⁻作为弱酸酸式根应完整保留，不拆为H⁺和CO₃²⁻，正确离子方程式为 HCO₃⁻ + H⁺ → H₂O + CO₂↑，当NaHCO₃过量或少量均为此式。

⚠️ 高频易错点

  • “拆”的尺度混淆：将沉淀、气体、弱电解质或浓硫酸拆成离子形式；误拆微溶物(如Ca(OH)₂悬浊液不拆，澄清石灰水拆)；酸式盐的酸根只有强酸的酸式根继续拆(如HSO₄⁻可拆为H⁺+SO₄²⁻，HCO₃⁻、HSO₃⁻则不拆)。
  • 电荷不守恒：仅注意原子种类和个数，忽略左右两边总电荷的相等。氧化还原反应中得失电子与电荷变化不一致时必错。
  • 用量反映不当：过量/少量问题中产物发生变化时离子方程式有异，如NaOH溶液滴入Ca(HCO₃)₂溶液：少量NaOH时OH⁻ + Ca²⁺ + HCO₃⁻ → CaCO₃↓ + H₂O，过量NaOH则2OH⁻ + Ca²⁺ + 2HCO₃⁻ → CaCO₃↓ + 2H₂O + CO₃²⁻，必须根据相对量确定实际反应比例。
  • 忽略反应真实环境：如NO₃⁻在中性或碱性溶液中无强氧化性，但在酸性条件下与还原性离子(Fe²⁺、I⁻等)发生氧化还原，书写时将H⁺参与反应遗漏是高频错误。
