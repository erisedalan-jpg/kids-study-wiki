离子方程式的书写

一句话：用实际参加反应的离子符号来表示化学反应的式子，删去不参与反应的旁观离子，揭示水溶液中反应的本质微粒变化。 English: An ionic equation is a chemical equation in which soluble strong electrolytes are written as dissociated ions, showing only the species that actually participate in the reaction.

🎓 考点精讲

定义与核心原理

离子方程式是用实际参加反应的离子符号表示离子反应的式子。其书写基于“强电解质完全电离、弱电解质与难溶物等主要以分子形式存在”这一化学事实。离子反应的本质是溶液中某些离子浓度的降低，书写时必须遵循原子守恒、电荷守恒、得失电子守恒（若为氧化还原反应）。

书写四步法（高考标准步骤）

  1. 写：正确写出反应的化学方程式。
  2. 例：$BaCl_2 + Na_2SO_4 = BaSO_4 \downarrow + 2NaCl$
  3. 拆：把易溶且易电离的强电解质拆成离子形式；单质、氧化物、弱电解质（弱酸、弱碱、水）、难溶物、气体等一律保留化学式。
  4. 上例中拆为：$Ba^{2+} + 2Cl^- + 2Na^+ + SO_4^{2-} = BaSO_4 \downarrow + 2Na^+ + 2Cl^-$
  5. 删：删去方程式两边未参与反应的相同离子（即旁观离子）。
  6. 删去 $2Na^+$ 和 $2Cl^-$，得：$Ba^{2+} + SO_4^{2-} = BaSO_4 \downarrow$
  7. 查：检查离子方程式两边各元素的原子个数和电荷总数是否相等，沉淀/气体符号是否恰当，反应条件是否合理。

高考典型设问与关键场景

  • 正误判断型：给出若干离子方程式，要求判断正误并纠错。此为最常见题型，约占选择题 6 分。
  • 书写型：结合化工流程、实验现象，根据描述直接书写离子方程式。
  • 定量书写：涉及酸式盐与碱的反应，如 $NaHSO_4$ 与 $Ba(OH)_2$ 溶液混合至中性或至 $SO_4^{2-}$ 恰好完全沉淀，两种情况下离子方程式不同。这是高考中区分度极高的设问，必须抓住“谁少量，谁反应完”的配比原则，以少量物质的离子化学计量数为基准，按需取用过量物质中的离子。

⚠️ 高频易错点

  • “拆”与“不拆”规则混淆：误拆弱电解质（如 $CH_3COOH, NH_3 \cdot H_2O$）、难溶物（如 $CaCO_3, BaSO_4, Fe(OH)_3$）、非溶液中进行的反应（如固体氯化铵与消石灰共热制氨气）；或忘记拆可溶性强电解质（如将澄清石灰水写成 $Ca(OH)_2$ 而未拆为 $Ca^{2+}$ 和 $OH^-$，但石灰乳则保留化学式）。
  • 遗漏隐含反应：书写氧化还原离子方程式时，只关注主要变价元素，忽略水溶液环境中 $H^+$、$OH^-$ 或 $H_2O$ 的参与（如 $MnO_4^-$ 在不同酸碱性下还原产物不同），导致电荷无法配平或产物错误。
  • 忽略“量”的关系：酸式盐与碱的反应、多元弱酸盐与酸的反应中，反应物物质的量之比不同则离子方程式不同（如 $Na_2CO_3$ 溶液中逐滴滴加盐酸与一次性加入过量盐酸，方程式分别为 $CO_3^{2-} + H^+ = HCO_3^-$ 与 $CO_3^{2-} + 2H^+ = CO_2 \uparrow + H_2O$）。
  • 电荷不守恒与物料不守恒：配平后仅检查原子个数，忽略离子方程式独特的电荷守恒检查。氧化还原离子方程式中先目测配平价态而漏掉非变价离子的配平。
