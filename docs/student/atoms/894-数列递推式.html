数列递推式

一句话：数列递推式是给出初始项，并利用前若干项通过固定法则确定后续项的等式，它是定义数列的基本方式之一。 English: A sequence recurrence relation is an equation that defines each term of a sequence using one or more previous terms, together with given initial values.

🎓 考点精讲

定义/原理
数列递推式形如 $a_{n+1} = f(a_n, a_{n-1}, \dots, n)$，必须搭配初始项（如 $a_1$）才能唯一确定数列。高考重点考查由递推式求通项公式。

关键方法
1. 待定系数化等比（一阶线性）：对 $a_{n+1} = pa_n + q$，若 $p \neq 1$，设 $a_{n+1} - \lambda = p(a_n - \lambda)$，解 $\lambda = \frac{q}{1-p}$，${a_n - \lambda}$ 为等比数列。
2. 累加法：若 $a_{n+1} - a_n = f(n)$，则 $a_n = a_1 + \sum_{k=1}^{n-1} f(k)$。
3. 累乘法：若 $\frac{a_{n+1}}{a_n} = g(n)$，则 $a_n = a_1 \prod_{k=1}^{n-1} g(k)$。
4. 二阶线性递推（拓展）：$a_{n+2} = p a_{n+1} + q a_n$，用特征方程 $x^2 - p x - q = 0$ 的根构造通项。
5. 分式线性递推：$a_{n+1} = \frac{A a_n + B}{C a_n + D}$，常结合不动点 $x = \frac{A x + B}{C x + D}$ 转化为等比或等差。

典型应用
高考中常以填空题或解答题第一问出现：给出 $a_{n+1}$ 与 $a_n$ 的关系及 $a_1$，求 ${a_n}$ 通项公式；或证明 ${b_n}$（$b_n = a_n + c$ 或 $b_n = \frac{1}{a_n}$ 等代换）为等差/等比数列，进而求 $a_n$。例如，已知 $a_1 = 1$，$a_{n+1} = 2a_n + 3$，证明 ${a_n+3}$ 为等比并求通项；又如 $a_1 = 1$，$a_{n+1} = \frac{2a_n}{a_n+2}$，取倒数化等差数列。

⚠️ 高频易错点

  • 忽视 $n$ 的起始范围：累加/累乘时，项数计算错误或忽略 $n \ge 2$ 的分段讨论，导致通项对 $n=1$ 不成立。
  • 待定系数计算错误：解 $\lambda = \frac{q}{1-p}$ 时符号或分母弄错，忘记检验 $p \neq 1$ 的情况。
  • 递推变形不等价：两边除以含 $a_n$ 的式子时未讨论其是否可能为零，致遗漏或增解。
  • 特征方程使用场景混淆：将线性递推的特征方程误用于分式递推或对方程无实数根时不会处理（高考若出现，多引导用周期或构造）。
