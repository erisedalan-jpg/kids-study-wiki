极坐标方程

一句话：在极坐标系下，用极径 $\rho$ 与极角 $\theta$ 满足的关系式 $F(\rho,\theta)=0$ 或 $\rho=f(\theta)$ 表示的平面曲线方程。 English: An equation that defines a curve in the plane using polar coordinates $(\rho, \theta)$, typically written as $\rho = f(\theta)$ or $F(\rho, \theta) = 0$.

🎓 考点精讲

定义与原理
在平面内取定点 $O$ 为极点，以 $Ox$ 为极轴，选定长度单位与角的正方向（通常逆时针）。对任意点 $M$，设 $|OM|=\rho$（极径，$\rho \geq 0$），$\angle xOM=\theta$（极角），则有序数对 $(\rho,\theta)$ 为点 $M$ 的极坐标。一条曲线上每点的极坐标满足的共同关系式即为该曲线的极坐标方程。

最常见直角坐标互化：
$$x = \rho\cos\theta,\quad y = \rho\sin\theta, \quad \rho^2 = x^2 + y^2,\quad \tan\theta = \frac{y}{x}\ (x \neq 0).$$

一些标准极坐标方程
- 过极点且倾斜角为 $\alpha$ 的直线：$\theta = \alpha$ $(\rho \in \mathbb{R})$。
- 圆心在极点、半径为 $r$ 的圆：$\rho = r$。
- 圆心在 $(a,0)$、半径为 $a$ 的圆：$\rho = 2a\cos\theta$。
- 圆心在 $(a,\frac{\pi}{2})$、半径为 $a$ 的圆：$\rho = 2a\sin\theta$。
- 以焦点为极点的圆锥曲线统一方程（$e$：离心率，$p$：焦点到相应准线的距离）：
$$\rho = \frac{ep}{1 - e\cos\theta}.$$

关键方法
1. 互化：将极坐标方程化为直角坐标方程是解题基础。先用 $\rho^2=x^2+y^2$、$\rho\cos\theta=x$、$\rho\sin\theta=y$ 代入，整理得直角坐标方程，进而判断曲线类型。由直角坐标化极坐标时需根据点所在象限确定 $\theta$。
2. 求交点与距离：既可直接在极坐标系下联立方程求极径和极角，也可化为直角坐标方程求解。极坐标系下两点 $P(\rho_1,\theta_1)$、$Q(\rho_2,\theta_2)$ 的距离公式：
$$|PQ| = \sqrt{\rho_1^2 + \rho_2^2 - 2\rho_1\rho_2\cos(\theta_1-\theta_2)}.$$
处理弦长、面积等问题时，常用极径的几何意义简化运算。

典型高考设问
- 将极坐标方程 $\rho = 4\cos\theta$ 化为直角坐标方程，并说明曲线形状。
- 已知曲线 $C$ 的极坐标方程为 $\rho^2\cos2\theta = 1$，写出其直角坐标方程——由 $\cos2\theta = \cos^2\theta - \sin^2\theta$ 得 $(\rho\cos\theta)^2 - (\rho\sin\theta)^2 = 1$，即 $x^2 - y^2 = 1$，为等轴双曲线。
- 求直线 $\theta = \alpha$ 与圆 $\rho = 2r\cos\theta$ 的交点坐标，或求两曲线围成图形的面积。

⚠️ 高频易错点

  • 互化时漏解或多解：由 $\tan\theta = y/x$ 求 $\theta$ 时，必须结合点 $(x,y)$ 的象限确定极角；若方程允许 $\rho$ 取负值（如直线 $\theta=\alpha$），需注意对应反向延长线。
  • 极坐标圆方程混淆：误将 $\rho = 2a\cos\theta$（圆心在极轴上）写成 $\rho = 2a\sin\theta$（圆心在过极点垂直于极轴的直线上），或把半径 $a$ 记成 $2a$。
  • 距离公式用错：在极坐标系下直接使用直角坐标的两点间距离公式 $\sqrt{(x_1-x_2)^2+(y_1-y_2)^2}$ 而忘记用余弦定理的极坐标形式。
  • 圆锥曲线统一方程参数不清：$\rho = \frac{ep}{1 - e\cos\theta}$ 中，分母的正负号、$p$ 的含义与直角坐标系下焦点准线不对应时，容易出错。

中文 English 说明
极坐标系 polar coordinate system 用极径和极角定位平面点的坐标系
极径 radial coordinate 点到极点的距离，通常记作 $\rho$
