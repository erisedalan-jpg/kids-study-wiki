# 初中 Phase 1 — 42 学习路径文档 实施计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 一次性生成初中 9 学科共 42 个《学习路径》文档（七上→九下），作为词条体系骨架。

**Architecture:** 9 个 sonnet 子代理并行执行，每代理负责 1 学科的全部学期。每文档沿用小学 7 段模板（教材链接 / 单元构成 / 推荐学习路径 / 重点知识点 / 易错点 / 词条覆盖 / 教材引用）。

**Tech Stack:** Claude Code Agent tool (subagent_type=general-purpose, model=sonnet, run_in_background=true) + Bash + git。

**Spec:** `docs/superpowers/specs/2026-05-08-初中9学科规划-design.md`
