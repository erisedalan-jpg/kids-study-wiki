## 3. Phase 1：42 个学习路径文档

### 3.1 目录结构

```
00-元/学习路径/初中/
  ├── 政治/01-七上.md ... 06-九下.md
  ├── 地理/01-七上.md ... 04-八下.md
  ├── 化学/01-九上.md ... 02-九下.md
  ├── 历史/01-七上.md ... 06-九下.md
  ├── 生物/01-七上.md ... 04-八下.md
  ├── 数学/01-七上.md ... 06-九下.md
  ├── 物理/01-八上.md ... 03-九年级全一.md
  ├── 英语/01-七上.md ... 05-九年级全一.md
  └── 语文/01-七上.md ... 06-九下.md
```

**命名规则**：
- 学科子目录下文档按学期出现顺序加 2 位序号前缀（同小学：`01-七上.md` `02-七下.md` …）
- 学期数 ≤ 6，2 位前缀足够

### 3.2 文档模板

参考 `00-元/学习路径/小学/语文/01-一上.md`，每个学期文档含：

```markdown
# 学习路径: 初中<学科><年级><学期> (<版本>)

教材: [[<本地 PDF 链接>]]

## 单元构成
| Unit | 主题 | 核心概念/句型/事件 | 教材页 |

## 推荐学习路径
### 阶段 1: <主题> (<单元号>, X 周)
- 知识点 → [[词条候选]]

## 重点知识点
- 分类列举

## 易错点 / 难点
- 学习中常见误解、混淆

## 与本 wiki 词条覆盖
✅ 已覆盖: [[xx]]（Phase 1 一律为空）
⚠️ 缺口: 待建词条 xx, xx

## 教材引用
每个词条的 `📑 出处` 教材行链此 PDF 对应单元页码.
```

**与小学模板差异**：
- "与 wiki 词条覆盖" 一律标 ⚠️ 缺口（词条体系全空，待 Phase 2 补）
- 理科（数学/物理/化学）多列「公式与定理」「典型例题类型」
- 历史多列「关键时间节点」「核心人物」
- 文科（政治/历史/地理）注意中性陈述、不下价值观结论

| D | 历史 | 6 | 统编版 | 七上下中国古代 / 八上下中国近现代 / 九上下世界史 |
| E | 生物 | 4 | 人教版 | 教材链接引 `生物学/...pdf` |

**统一 prompt 骨架**（每代理）：
1. 项目目录、目标、模板路径

5. 禁用：TaskCreate / 子代理嵌套；不读 PDF 内容

**预期产出**：42 个文档，每个 60-100 行，总 ~3000-4000 行。

## 4. Phase 2：词条路线图（仅约定方向，不在本规划实施）

### 4.1 预估词条数

| 学科 | 预估 | 重点类型 |
|---|---|---|
| 政治 | ~120 | 法律概念、宪法条文、价值观（中性陈述） |
| 地理 | ~200 | 地形 / 气候 / 国家 / 省份；附地图位置 |
| 化学 | ~150 | 元素 / 反应 / 方程式；附化学式与离子方程 |
| 历史 | ~250 | 朝代 / 事件 / 人物；附时间轴位置 |
| 生物 | ~200 | 系统 / 物种 / 实验；解剖图描述 |
| 数学 | ~180 | 代数 / 几何 / 统计；公式与典型解法 |
| 物理 | ~200 | 概念 / 公式 / 实验；SI 单位与图示 |
| 英语 | ~250 | 词汇 / 语法 / 课文；两层正文 |
| 语文 | ~180 | 古文 / 现代文 / 作家 / 文学常识 |
| **合计** | **~1730** | |

### 4.2 词条模板调整（vs 小学）

- 三层正文 → **两层**：📚 6-12 岁版（小学高年级回溯）+ 🎓 12+ 进阶（主体）。删除 🧒 3-6 岁共读层
- 序号前缀：单学科 ≤ 99 用 2 位；> 99 用 3 位（如 政治 001-120）
- frontmatter `学段: [初中]` 或 `[初中, 高中]`（部分概念可拓展高中）
- 红线沿用小学：必引本地 PDF / 不编 URL / 价值观题不下结论

### 4.3 触发方式

家长按优先级下令"补 <学科> <学期> 词条"，按学期分批生成。9 学科可同步并行 3-5 个子代理。

## 5. 实施顺序

3. **更新 CLAUDE.md**：标记初中路径完成
4. **Phase 2**（待用户触发）：按学科优先级分批补词条

## 7. 风险与缓解

| 风险 | 概率 | 缓解 |
|---|---|---|
| 子代理 API rate limit | 中 | 失败重发同 prompt；分批降并发 |
| 内容过滤拦截（历史 / 政治） | 中 | prompt 明确"中性陈述、不下结论"；触发后改写后重发 |
| 教材 PDF 路径写错 | 低 | prompt 中预先列出全部路径 |
| 文档结构差异（理科 vs 文科） | 低 | 模板留弹性，每代理按学科调整段落 |
