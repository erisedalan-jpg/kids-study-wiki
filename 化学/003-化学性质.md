# 化学性质

> **一句话**：化学性质是物质在化学变化中表现出来的性质，如可燃性、氧化性、还原性、酸碱性等。
> **English**: A chemical property describes a substance's ability to change into a different substance through a chemical reaction.

---

## 📚 给 6-12 岁（自读版）

铁会生锈，木头能燃烧，盐酸能腐蚀金属——这些都是物质的"脾气"，只有在发生化学变化时才能看到。这种"脾气"就叫**化学性质**。

**化学性质要通过化学变化才能看出来**

比如：
- 你把铁放在桌上，不看到它生锈，你不知道铁会生锈——但一旦它生锈了（发生化学变化），就说明铁有"易被氧化"的化学性质。
- 木头能不能烧？你不点火，不知道！点了火（化学变化），才知道木头有可燃性。

**三个例子**

1. **氢气的可燃性**：氢气遇火能燃烧，生成水。这是氢气的化学性质。
2. **铁的可锈性**：铁在水和空气中会生成铁锈，这是铁的化学性质。
3. **石灰石与盐酸反应**：石灰石遇到盐酸会冒泡，生成二氧化碳。这是石灰石（碳酸钙）的化学性质。

**口诀**：化学性质看变化，不反应就不知道！

## 🎓 给 12+（进阶版）

### 定义

化学性质是指物质在发生化学变化时所表现出的性质。它必须通过化学变化才能体现，描述的是物质参与化学反应的能力或倾向。常见化学性质包括：可燃性、氧化性、还原性、酸性、碱性、稳定性、活泼性等。

### 性质 / 反应 / 公式

| 化学性质 | 物质示例 | 典型反应 |
|----------|----------|---------|
| 可燃性 | 氢气、甲烷 | $2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$ |
| 氧化性 | 氧气 | $C + O_2 \overset{\text{点燃}}{=} CO_2$ |
| 还原性 | 一氧化碳 | $CO + CuO \overset{\Delta}{=} Cu + CO_2$ |
| 酸性 | 盐酸 | $HCl + NaOH = NaCl + H_2O$ |
| 稳定性 | 氮气 | 常温下不与大多数物质反应 |

**化学性质与物理性质的比较**

| 比较项目 | 化学性质 | 物理性质 |
|----------|----------|----------|
| 表现方式 | 通过化学变化体现 | 不通过化学变化即可观察 |
| 举例 | 可燃性、酸碱性 | 颜色、硬度、熔点 |
| 是否产生新物质 | 是 | 否 |

### 典型实验 / 例题

**例题**：下列属于化学性质的是（ ）
A. 铁的硬度  B. 铜的颜色  C. 酒精的可燃性  D. 食盐的溶解性

**解析**：A 硬度、B 颜色、D 溶解性均不需要通过化学变化表现，属物理性质；C 可燃性必须通过燃烧（化学变化）才能表现，选 **C**。

### 易错点

1. 溶解性通常是物理性质（溶于水不产生新物质）；但活泼金属（Na）溶于水时发生反应，则同时包含化学性质。
2. 稳定性（如氮气很稳定）也是化学性质，虽然表现为"不发生反应"。
3. "某物质有可燃性"说的是化学性质，"某物质正在燃烧"描述的是化学变化，两者有区别。
