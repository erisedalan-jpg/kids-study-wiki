# 燃烧条件

> **一句话**：燃烧必须同时满足三个条件：①有可燃物；②可燃物与氧气（或空气）接触；③温度达到可燃物的着火点——三者缺一不可。
> **English**: Combustion requires three simultaneous conditions: ① a combustible substance, ② contact with oxygen (or air), and ③ temperature reaching the ignition point of the combustible material — all three are necessary.

---

## 📚 给 6-12 岁（自读版）

为什么湿木头很难点燃？为什么把火焰旁的空气吹走，火就会灭？这都是因为燃烧需要三个"缺一不可"的条件！

**燃烧三要素（火三角）**

想象一个三角形，三条边分别是：

```
        氧气（O₂）
       /          \
可燃物 ————————— 达到着火点
```

**缺少任何一条边，火三角就垮掉，燃烧就停止！**

**三个条件详解**

**① 可燃物**
- 能够燃烧的物质：木头、酒精、天然气、氢气、纸张……
- 不是可燃物：铁（初中认为不助燃，但在纯氧中可以燃烧）、石头、玻璃、沙子……
- 注意：CO 是可燃物（有毒气体也能燃！）

**② 与氧气接触**
- 可燃物必须和 O₂（或空气中的氧气）接触
- 用沙子覆盖火苗 → 隔绝 O₂ → 灭火！
- 罩上玻璃罩 → O₂ 耗尽 → 蜡烛熄灭！

**③ 达到着火点**
- 着火点是可燃物**开始燃烧的最低温度**
- 每种可燃物的着火点不同：

| 物质 | 着火点 |
|------|--------|
| 白磷 | 约 40°C（极易自燃！）|
| 红磷 | 约 240°C |
| 木材 | 约 250-270°C |
| 酒精 | 约 363°C |
| 煤 | 约 320-370°C |

- 不能通过降低着火点来灭火（着火点是物质固有性质，不能改变）！

**实验：验证燃烧条件**

铜片实验（课本经典实验）：

1. 白磷（铜片上，有 O₂，加热 → 达到 40°C）→ **燃烧**（三条件都满足）
2. 红磷（铜片上，有 O₂，加热 → 未达 240°C）→ **不燃烧**（温度不够）
3. 白磷（水中，没有 O₂，即使加热）→ **不燃烧**（缺少 O₂）

**口诀**：燃烧三要素牢记心，可燃接氧着火点，三者缺一就灭火，控制条件保安全！

## 🎓 给 12+（进阶版）

### 定义

燃烧条件是启动并维持燃烧反应必须同时具备的物理和化学条件：可燃物（提供还原剂）、氧化剂（通常为 O₂）、温度达到可燃物的着火点（ignition temperature / flash point）。从热力学角度，着火点是反应放热速率与散热速率相等时的最低温度；超过此温度，放热速率 > 散热速率，反应自持续。

### 性质 / 反应 / 公式

**三个燃烧条件的本质**

| 条件 | 化学意义 | 控制方式 |
|------|---------|---------|
| 可燃物 | 提供还原性物质（被氧化）| 移走可燃物 |
| 氧气 | 提供氧化剂（提供 O）| 隔绝 O₂ |
| 着火点 | 活化能阈值，启动链式反应 | 降低温度至着火点以下 |

**着火点对比表**

| 物质 | 着火点（℃）| 注意事项 |
|------|-----------|---------|
| 白磷（P₄）| ~40 | 储存于水中，极易自燃 |
| 红磷 | ~240 | 火柴头摩擦面 |
| 木材 | ~250-270 | 视含水量和密度 |
| 煤 | ~320-370 | 需引燃 |
| 酒精（乙醇）| ~363 | 引闪点约 13°C（挥发蒸气点燃）|
| 汽油 | ~246-280 | 引闪点约 -43°C（极易挥发点燃）|

**验证燃烧条件的经典实验**

实验设计（铜片对比实验）：

| 实验组 | 白磷/铜片（加热）| 红磷/铜片（加热）| 白磷/水中（通 O₂ 前）|
|--------|-----------|-----------|------------|
| 有无 O₂ | 有 | 有 | 无 |
| 温度 | >40°C | <240°C | >40°C |
| 可燃物 | 有 | 有 | 有 |
| 结果 | **燃烧** | **不燃烧** | **不燃烧** |
| 证明 | 三条件满足 | 缺温度条件 | 缺 O₂ 条件 |

**着火点的固定性**

着火点是物质的固有化学性质，**不能通过降温方式改变**（灭火是通过降温至着火点以下，不是降低着火点本身）。这是初中常见易错点。

### 典型实验 / 例题

**例题**：下列关于燃烧条件的说法，正确的是（ ）

A. 物质燃烧时必须通入纯氧，空气中的 O₂ 浓度不够  
B. 可以通过降低可燃物的着火点来灭火  
C. 白磷常温下容易自燃，是因为其着火点很低（约 40°C）  
D. 同一物质在不同情况下的着火点不同

**解析**：  
A 错：大多数燃烧在空气中即可进行（O₂ 约 21% 足够）；  
B 错：着火点是物质固有性质，无法降低；  
C 对：白磷着火点约 40°C，常温（夏天 >40°C）可自燃；  
D 错：着火点是物质的固定性质，不随条件改变。  
选 **C**。

### 易错点

1. **着火点不能改变**——灭火是使温度降到着火点以下，而不是降低着火点。
2. **三个条件必须同时满足**——可燃物加热到着火点，但隔绝 O₂，也不能燃烧。
3. 白磷着火点极低（约 40°C），夏天裸露在空气中即可自燃，必须储存在水中。
4. 着火点（ignition temperature）与燃点（闪点/flash point）是不同概念——闪点针对液体挥发蒸气。
