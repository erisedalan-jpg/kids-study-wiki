# 甲烷燃烧

> **一句话**：甲烷（CH₄）完全燃烧的方程式为CH₄+2O₂→CO₂+2H₂O，产生淡蓝色火焰，是天然气和沼气的主要燃烧反应，也是判断有机物含氢的经典实验（产物有水蒸气）。
> **English**: The complete combustion of methane (CH₄) is CH₄+2O₂→CO₂+2H₂O, producing a pale blue flame; this is the main reaction for burning natural gas and biogas, and the water produced is a classic test for hydrogen in organic compounds.

---

## 📚 给 6-12 岁（自读版）

家里的燃气灶点燃时，看到的蓝色火焰是什么？那是**甲烷（CH₄）在燃烧**！天然气的主要成分就是甲烷，理解甲烷的燃烧反应，就能理解家里燃气灶的化学原理。

**甲烷的基本介绍**

甲烷（CH₄）是：
- 最简单的有机物（一个碳 + 四个氢）
- 无色无味气体，无毒（但能使人窒息）
- 比空气轻（M=16，空气≈29）

**甲烷完全燃烧**

$$CH_4 + 2O_2 \overset{\text{点燃}}{=} CO_2 + 2H_2O$$

**读方程式意思**：
- 1 分子甲烷 + 2 分子氧气 → 1 分子 CO₂ + 2 分子水
- 现象：淡蓝色火焰，产生 CO₂ 和 H₂O

**甲烷不完全燃烧（氧气不足时）**

$$2CH_4 + 3O_2 \overset{\text{点燃（不足量）}}{=} 2CO + 4H_2O$$

产生有毒 CO！所以煤气灶一定要保持通风。

**怎么验证甲烷燃烧产生了水？**

用**干燥的玻璃片**放在火焰上方：
- 玻璃片出现水雾（小水珠）→ 证明产生了 H₂O

**怎么验证甲烷燃烧产生了 CO₂？**

将导管通入澄清石灰水：
- 石灰水变浑浊 → 证明产生了 CO₂

**甲烷在哪里可以找到？**

- 天然气：地下开采（70-90% 是 CH₄）
- 沼气：有机物腐烂产生（CH₄ 约 60%）
- 可燃冰：深海中甲烷水合物（未来能源）
- 瓦斯：煤矿中渗出的 CH₄（会爆炸，危险！）

**口诀**：甲烷点燃蓝色焰，一碳两氧出 CO₂，别忘产物还有水，检验石灰和玻璃！

## 🎓 给 12+（进阶版）

### 定义

甲烷（methane，CH₄）燃烧是甲烷与氧气在点燃条件下发生的氧化反应，是天然气、沼气、煤层气等的主要能量来源反应。甲烷是最简单的烷烃（饱和碳氢化合物），C-H 键能约 413 kJ/mol，完全燃烧标准生成热 ΔHcombustion = -890 kJ/mol（25°C，标准状态）。

### 性质 / 反应 / 公式

**甲烷的化学性质**

| 性质 | 方程式 | 特点 |
|------|--------|------|
| 完全燃烧 | $CH_4 + 2O_2 \overset{\text{点燃}}{=} CO_2 + 2H_2O$ | 淡蓝色火焰，高热值 |
| 不完全燃烧 | $2CH_4 + 3O_2 \overset{\text{点燃（不足）}}{=} 2CO + 4H_2O$ | 产生有毒 CO |
| 水蒸气重整 | $CH_4 + H_2O \overset{\text{高温催化}}{=} CO + 3H_2$ | 工业制 H₂ |
| 热裂解 | $CH_4 \overset{\text{高温}}{=} C + 2H_2$ | 工业制炭黑和 H₂ |

**甲烷燃烧的配平过程**

$$CH_4 + O_2 \rightarrow CO_2 + H_2O \quad \text{（未配平）}$$

1. C 原子：左 1，右 1，已平衡
2. H 原子：左 4，右 2（水中），系数×2：$CH_4 + O_2 \rightarrow CO_2 + 2H_2O$
3. O 原子：右 2+2=4，左 O₂ 系数 = 4/2 = 2
4. 配平：$CH_4 + 2O_2 \overset{\text{点燃}}{=} CO_2 + 2H_2O$

**甲烷与其他有机物燃烧规律**

含碳氢元素的有机物完全燃烧，产物**必定**含 CO₂ 和 H₂O：
- 若含 C、H、O：产物 CO₂ + H₂O
- 若含 C、H（无 O）：产物 CO₂ + H₂O（同上）

判断有机物是否含 H：**在玻璃片上观察是否出现水雾**（试管内壁出现水珠）。

**甲烷在沼气中的利用**

沼气成分（农村厌氧发酵）：

| 成分 | 含量（%）|
|------|---------|
| CH₄ | 50-70 |
| CO₂ | 30-45 |
| N₂、H₂S | 少量 |

沼气发酵（总反应简示）：

$$\text{有机物（葡萄糖等）} \overset{\text{厌氧细菌}}{=} CH_4 + CO_2 + H_2O$$

沼气是生物质能的代表，属于可再生能源（与化石燃料的 CH₄ 不同）。

**甲烷的温室效应**

CH₄ 的温室效应效力是 CO₂ 的约 20-25 倍（100 年全球变暖潜能 GWP=28-36）。畜牧业（牛反刍）和稻田是全球 CH₄ 排放的重要来源。

### 典型实验 / 例题

**实验：验证甲烷燃烧的产物**

1. 在燃气灶（天然气）火焰上方放一片干冷玻璃片：出现水雾（证明有 H₂O 生成）
2. 将燃烧产物导入澄清石灰水：变浑浊（证明有 CO₂ 生成）
3. 结论：CH₄ 含 H 和 C，完全燃烧生成 H₂O 和 CO₂

**例题**：将天然气（主要为 CH₄）点燃，在火焰上方放一片干燥的冷玻璃片，玻璃片上出现水珠，将玻璃片迅速移开，用澄清石灰水检验收集的气体，石灰水变浑浊。上述实验说明（ ）

A. 甲烷中含有水分子  
B. 甲烷燃烧产生了 CO₂ 和 H₂O，说明甲烷中含 C 和 H 元素  
C. 甲烷燃烧产生了 CO₂ 和 H₂O，说明甲烷中含 C、H 和 O 元素  
D. 甲烷是有机物，所以一定含有氧元素

**解析**：  
A 错：玻璃片上出现的水是**燃烧产物**（H₂O），不是甲烷中含水；  
B 对：有 H₂O 产生→含 H 元素；有 CO₂ 产生→含 C 元素；  
C 错：产物有 H₂O 和 CO₂ 不能证明含 O——O 可能来自反应的 O₂；  
D 错：有机物不一定含 O（甲烷 CH₄ 就不含 O）。  
选 **B**。

### 易错点

1. 有机物燃烧有水产生，**不能说明**该有机物含氧——产物中的 O 可能全部来自 O₂，不是有机物中的 O。
2. 甲烷完全燃烧产物是 CO₂ **和** H₂O——比较常见的错误是只写 CO₂，忘了 H₂O。
3. 沼气中的 CH₄（生物来源）≠ 化石燃料中的 CH₄（地质来源）——前者是可再生能源（循环碳），后者是不可再生的（新增碳排放）。
4. 甲烷本身**无毒**（窒息性气体），但天然气和沼气中可能含 CO（不完全燃烧）和 H₂S，才是真正的毒素。
