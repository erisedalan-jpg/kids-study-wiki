# 着火点

> **一句话**：着火点是可燃物开始持续燃烧所需的最低温度，是物质的固有属性，不能被人为改变，白磷着火点约40°C（极低），木材约250°C。
> **English**: The ignition temperature (also called ignition point) is the minimum temperature at which a combustible substance begins to burn continuously; it is an intrinsic property of the material that cannot be changed artificially — white phosphorus ~40°C, wood ~250°C.

---

## 📚 给 6-12 岁（自读版）

为什么白磷（一种化学品）在夏天会"自己着火"？为什么我们不用担心铁锅会在炉火上燃烧？这一切都和**着火点**有关！

**什么是着火点？**

着火点（燃点）是可燃物**开始持续燃烧**的**最低温度**。

每种可燃物都有自己固定的着火点，到了这个温度，就会燃烧！没到这个温度，就算有 O₂ 也不会燃烧。

**不同物质的着火点**

| 物质 | 着火点（约）| 特别说明 |
|------|-----------|---------|
| 白磷 | 约 40°C | 夏天气温超 40°C 即自燃！储存于水中 |
| 红磷 | 约 240°C | 火柴头用到，比白磷安全很多 |
| 木材 | 约 250-270°C | 普通木头需要引燃 |
| 煤炭 | 约 320-370°C | 需要助燃引火 |
| 酒精 | 约 363°C | 用明火点燃 |
| 铁 | 在空气中不燃；在纯氧中约 1000°C | 普通情况下不燃烧 |

**着火点的重要特点：不能改变！**

着火点是物质的"天生"特性，就像人的指纹一样固定。我们**不能**通过加化学品、用特殊方式让木头的着火点变成 5°C（那样就太危险了）。

所以：
- **灭火**不是"降低着火点"，而是**把温度降到着火点以下**（降温冷却）
- 白磷之所以危险，是因为着火点太低，不需要明火，夏天就可能自燃

**白磷为什么储存在水中？**

白磷着火点只有约 40°C，空气中暴露后容易自燃。水能隔绝 O₂ 且导热降温，让白磷保持低于 40°C ——所以实验室里白磷必须放在水里！

**口诀**：着火点是最低温，每种物质不一样，白磷 40 易自燃，着火点不能降！

## 🎓 给 12+（进阶版）

### 定义

着火点（ignition temperature / ignition point）是指可燃物在有充足氧气条件下，开始并能维持持续燃烧的最低温度。在该温度下，可燃物的燃烧放热速率等于或超过散热速率，形成热量积累，使燃烧能够自持。着火点是物质的固有化学性质，不因外界条件（湿度、压力除外）而改变。

注意与**闪点（flash point）**的区别：闪点是液态可燃物蒸气与空气混合能被点燃的最低温度（不能持续燃烧）；着火点是能持续燃烧的最低温度（着火点 > 闪点）。

### 性质 / 反应 / 公式

**常见物质着火点对比**

| 物质 | 着火点（℃）| 注意 |
|------|-----------|------|
| 白磷（P₄）| ~40 | 夏天可自燃，存于水中 |
| 红磷 | ~240 | 火柴摩擦头 |
| 硫磺（S）| ~232 | 淡黄色固体 |
| 木材 | ~250-270 | 视种类和含水量 |
| 煤 | ~320-370 | 视煤化程度 |
| 酒精（乙醇）| ~363 | 闪点约 13°C |
| 汽油 | ~246-280 | 闪点约 -43°C（极危险）|
| 氢气（H₂）| ~500 | 与空气混合可爆炸 |
| 天然气（甲烷）| ~537 | 低于常见可燃气体 |

**着火点与灭火的关系**

$$\text{灭火} \neq \text{降低着火点}$$

$$\text{灭火} = \text{使可燃物温度} < \text{其（固有）着火点}$$

降温措施（喷水、干冰等）降低的是**可燃物当前温度**，而不是着火点本身。

**影响着火点的因素（初中知识范围外，了解即可）**

| 因素 | 影响 |
|------|------|
| 粉碎程度 | 粉末比块状着火点略低（表面积大，散热快也接触 O₂ 多）|
| 压力 | 高压略升高着火点（O₂ 分压增加会使一些物质着火点降低）|
| 含水量 | 含水多的物质（湿木材）着火点略高（需先蒸发水）|

**白磷 vs 红磷（着火点对比）**

| 对比项 | 白磷（P₄）| 红磷 |
|--------|----------|------|
| 着火点 | ~40°C | ~240°C |
| 外观 | 黄白色蜡状固体 | 红棕色粉末 |
| 毒性 | 剧毒 | 无毒 |
| 存储 | 水中（隔绝 O₂ + 降温）| 密封即可 |
| 燃烧产物 | $P_4 + 5O_2 = 2P_2O_5$（白烟）| 相同 |

### 典型实验 / 例题

**实验：铜片白磷红磷着火点对比**

同一铜片：一端放白磷，另一端放红磷，加热铜片中部：
- 当铜片温度约 40°C：白磷开始燃烧（产生白烟），红磷不燃
- 继续加热到约 240°C：红磷开始燃烧
- 水中白磷（通入 O₂）：温度不变，通 O₂ 后也不燃（O₂ 足但温度不够）—— 证明需要氧气和温度双重条件

**例题**：下列关于着火点的说法，正确的是（ ）

A. 用水浇灭是通过降低可燃物的着火点实现灭火  
B. 白磷比红磷危险，是因为白磷的着火点更低  
C. 可以通过加入阻燃剂来降低物质的着火点  
D. 同种物质的着火点因温度条件不同而改变

**解析**：  
A 错：水是降低温度，不是降低着火点；  
B 对：白磷着火点约 40°C，远低于红磷 240°C，更容易自燃；  
C 错：阻燃剂不能降低着火点，而是通过其他方式（如产生隔绝层）阻止燃烧；  
D 错：着火点是固有性质，不随温度条件改变。  
选 **B**。

### 易错点

1. **着火点不能改变**——灭火使温度降低至着火点以下，不是降低着火点本身。
2. **着火点不等于闪点**——液体的闪点（挥发蒸气点燃的最低温度）< 着火点（持续燃烧的最低温度）。
3. 白磷和红磷是磷的**同素异形体**，化学式和燃烧产物相同（P₄/P→P₂O₅），但着火点差异巨大。
4. 着火点越低≠越危险的固定规律——还要考虑挥发性（汽油闪点极低，所以很危险）。
