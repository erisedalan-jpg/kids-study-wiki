# 元素符号

> **一句话**：元素符号是用一个或两个字母表示元素的国际通用符号，第一个字母大写，第二个字母小写，如 O（氧）、Fe（铁）、Ca（钙）。
> **English**: An element symbol is an internationally recognized one- or two-letter abbreviation for a chemical element, with the first letter capitalized, e.g., O (oxygen), Fe (iron), Ca (calcium).

---

## 📚 给 6-12 岁（自读版）

化学家用字母来给元素取"代号"，这就是元素符号。全世界化学家都用同样的符号，不管你说中文还是英文，Fe 都代表铁，O 都代表氧气！

**元素符号的写法规则**

- 一个字母的：大写，如 O（氧）、C（碳）、H（氢）、N（氮）、S（硫）
- 两个字母的：第一个大写，第二个小写，如 Fe（铁）、Cu（铜）、Ca（钙）、Mg（镁）

**为什么是字母，不是汉字？**

元素符号来源于元素的拉丁文名称（或英文名称），是国际通用的，全世界通用一套符号，方便交流。

**最常考的元素符号**（一定要背！）

| 中文 | 符号 | 记忆方法 |
|------|------|---------|
| 氢 | H | Hydrogen，最轻的元素 |
| 氦 | He | Helium，气球里的气 |
| 碳 | C | Carbon，钻石和铅笔芯都是碳 |
| 氮 | N | Nitrogen，空气中最多 |
| 氧 | O | Oxygen，呼吸用的 |
| 钠 | Na | Natrium（拉丁文），食盐里有它 |
| 镁 | Mg | Magnesium，轻金属 |
| 铝 | Al | Aluminium，易拉罐 |
| 铁 | Fe | Ferrum（拉丁文），最常用的金属 |
| 铜 | Cu | Cuprum（拉丁文），导线用铜 |
| 锌 | Zn | Zinc，电池里有 |
| 钙 | Ca | Calcium，骨头里有它 |

**口诀**：元素符号第一大，第二个字母小，拉丁文来源，全球通用记心间！

## 🎓 给 12+（进阶版）

### 定义

元素符号是国际化学联合会（IUPAC）规定的用于表示化学元素的缩写符号，由 1~2 个拉丁字母组成，首字母大写，第二个字母小写。符号来源于元素的拉丁文名（如 Fe ← Ferrum）、英文名（如 O ← Oxygen）或发现地/人名（如 Cm ← Curium 居里）。

### 典型实验 / 例题

**例题**：下列元素符号书写正确的是（ ）
A. fe（铁）  B. CL（氯）  C. MG（镁）  D. Cu（铜）

**解析**：A 错（应 Fe，首字母大写）；B 错（应 Cl，第二字母小写）；C 错（应 Mg）；D 正确，Cu 的 C 大写 u 小写，选 **D**。

### 易错点

1. **大小写严格区分**：Co（钴）和 CO（一氧化碳）完全不同——符号大小写错误可能代表完全不同的物质。
2. 来自拉丁文的符号与中文名无直接关联，需要死记：Na（钠）、K（钾）、Fe（铁）、Cu（铜）、Ag（银）、Au（金）、Hg（汞）。
3. 元素符号既表示元素，也表示该元素的一个原子；而分子或化合物须用化学式表示，不能用单个元素符号。
