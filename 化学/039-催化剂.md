# 催化剂

> **一句话**：催化剂是能改变化学反应速率（通常是加快），但自身在反应前后质量和化学性质不变的物质。
> **English**: A catalyst is a substance that changes the rate of a chemical reaction (usually speeds it up) without being consumed or permanently altered in the process.

---

## 📚 给 6-12 岁（自读版）

你有没有见过把双氧水（消毒水）倒在伤口上冒泡泡的样子？那些泡泡是氧气！双氧水会自己分解，但速度很慢。如果加入一点点二氧化锰，泡泡就会猛地冒出来——这是因为二氧化锰是**催化剂**，能让反应变快！

**催化剂的"神奇"之处**

1. **加快反应速率**：本来要几个小时的反应，有催化剂可能只需几秒。
2. **自己不变化**：反应结束后，催化剂还在，质量没变，化学性质没变，还可以继续用！

**生活中的催化剂**

- **二氧化锰（MnO₂）**：实验室里，用它催化双氧水分解产生氧气。
- **酶**：我们身体里有很多酶（淀粉酶、蛋白酶），帮助消化食物，本质就是生物催化剂。
- **汽车尾气催化转化器**：里面有铂、钯等金属催化剂，把有毒的 CO、NO 转化成无害的 CO₂ 和 N₂。
- **工业合成氨**：铁作催化剂，使氮气和氢气在相对温和条件下反应生成氨。

**口诀**：催化剂能改变速率，自身质量和性质不变，反应前后它在场，双氧水 + 二氧化锰！

## 🎓 给 12+（进阶版）

### 定义

催化剂是一种能改变化学反应速率（通常为加快），但自身在反应前后**质量不变、化学性质不变**的物质。催化剂参与反应过程（提供新的反应路径，降低活化能），但最终恢复原来的化学状态。能减慢反应速率的催化剂称为**负催化剂（抑制剂）**。

### 性质 / 反应 / 公式

**催化剂典型反应**

| 催化剂 | 反应 | 方程式 |
|--------|------|--------|
| $MnO_2$ | 双氧水分解 | $2H_2O_2 \overset{MnO_2}{=} 2H_2O + O_2\uparrow$ |
| $MnO_2$ | 氯酸钾分解 | $2KClO_3 \overset{MnO_2,\Delta}{=} 2KCl + 3O_2\uparrow$ |
| $Fe$（铁） | 合成氨 | $N_2 + 3H_2 \underset{\text{催化剂}}{\overset{\text{高温高压}}{⇌}} 2NH_3$ |
| 酶 | 消化淀粉 | 淀粉 $\overset{\text{淀粉酶}}{\rightarrow}$ 麦芽糖 |

**催化剂的特点总结**

| 特征 | 说明 |
|------|------|
| 改变速率 | 通常加快反应，少数减慢 |
| 质量不变 | 反应前后催化剂质量相等 |
| 化学性质不变 | 催化剂可以循环使用 |
| 参与中间过程 | 提供替代路径，降低活化能 |
| 具有选择性 | 不同催化剂对不同反应有效 |

**催化剂与活化能**：催化剂降低反应所需的活化能（Ea），使更多分子具备足够能量碰撞反应，从而加快反应速率。（图示：有无催化剂的能量曲线对比）

### 典型实验 / 例题

**实验：二氧化锰催化 H₂O₂ 分解**
- 向两支试管分别加入等量 H₂O₂，一支加入少量 MnO₂，一支不加。
- 现象：加 MnO₂ 的试管迅速产生大量气泡，带火星木条复燃；另一支只有极少气泡。
- 结论：MnO₂ 是双氧水分解的催化剂，加快反应速率。

**例题**：关于催化剂，说法正确的是（ ）
A. 催化剂能使不发生的反应也能发生  
B. 使用催化剂后，生成物的质量增加  
C. 催化剂反应后质量和化学性质不变  
D. 催化剂一定加快反应速率

**解析**：A 错（催化剂改变速率，不能让热力学不允许的反应发生）；B 错（生成物质量由质量守恒决定）；D 错（负催化剂减慢速率）；C 正确，选 **C**。

### 易错点

1. 催化剂质量和化学性质不变，但可能**物理状态改变**（如粉末状MnO₂可能松散）——这不影响其催化性质。
2. 催化剂不改变平衡位置（即最终生成物的量），只改变达到平衡的速度（两个方向都加快）。
3. "MnO₂ 催化 H₂O₂ 分解"——MnO₂ 是催化剂，不是反应物，方程式上方或下方标注，不写在箭头左边。
