# 分解反应

> **一句话**：分解反应是一种物质在一定条件下分解成两种或两种以上新物质的化学反应，特征是"一变多"（AB → A + B）。
> **English**: A decomposition reaction is a chemical reaction in which a single compound breaks down into two or more simpler substances (AB → A + B).

---

## 📚 给 6-12 岁（自读版）

分解反应和化合反应正好相反——化合是"多变一"，分解是**"一变多"**！

**一种物质 → 多种物质**

就像把一块大蛋糕切成几块小蛋糕，分解反应就是把一种物质拆分成多种物质。

**三个例子**

1. **电解水**：通电后，水（H₂O）分解成氢气和氧气。一种物质变成两种——典型的分解反应！
2. **加热高锰酸钾**：加热后高锰酸钾（KMnO₄）分解，生成锰酸钾、二氧化锰和氧气——一变三！
3. **双氧水分解**：双氧水（过氧化氢，H₂O₂）在二氧化锰催化下分解，生成水和氧气。

**分解反应常需要条件**

- 加热（Δ）
- 通电
- 催化剂
- 光照

**口诀**：分解反应一变多，AB变成A加B，和化合正相反，一种物质来分解！

## 🎓 给 12+（进阶版）

### 定义

分解反应是四大基本反应类型之一，指一种化合物在一定条件（加热、通电、光照或催化剂）下，分解生成两种或两种以上新物质的化学反应。通式为：

$$AB \rightarrow A + B$$

分解反应的反应物有且只有**一种**，生成物有两种及以上。

### 性质 / 反应 / 公式

| 反应 | 化学方程式 | 条件 |
|------|-----------|------|
| 水电解 | $2H_2O \overset{\text{通电}}{=} 2H_2\uparrow + O_2\uparrow$ | 通电 |
| 高锰酸钾加热 | $2KMnO_4 \overset{\Delta}{=} K_2MnO_4 + MnO_2 + O_2\uparrow$ | 加热 |
| 双氧水分解 | $2H_2O_2 \overset{MnO_2}{=} 2H_2O + O_2\uparrow$ | 催化剂 |
| 碳酸钙分解 | $CaCO_3 \overset{\text{高温}}{=} CaO + CO_2\uparrow$ | 高温 |
| 氯酸钾分解 | $2KClO_3 \overset{MnO_2,\Delta}{=} 2KCl + 3O_2\uparrow$ | 催化剂+加热 |

**与化合反应对比**

| 项目 | 分解反应 | 化合反应 |
|------|----------|----------|
| 通式 | AB → A + B | A + B → AB |
| 反应物数量 | 一种 | 多种 |
| 生成物数量 | 多种 | 一种 |
| 举例 | 水电解 | 碳燃烧 |

### 典型实验 / 例题

**实验：加热高锰酸钾制备氧气**
- 现象：紫黑色固体变色（锰酸钾绿色 + 棕黑色MnO₂），带火星的木条伸入收集的气体中复燃。
- 注意：试管口要略向下倾斜，防止冷凝水倒流入热试管（炸裂）；管口放棉花防止高锰酸钾粉末进入导管。

**例题**：下列属于分解反应的是（ ）
A. $C + O_2 \overset{\text{点燃}}{=} CO_2$
B. $2H_2O \overset{\text{通电}}{=} 2H_2\uparrow + O_2\uparrow$
C. $Fe + CuSO_4 = FeSO_4 + Cu$
D. $CaO + H_2O = Ca(OH)_2$

**解析**：A 化合反应；C 置换反应；D 化合反应；B 中 H₂O（一种物质）→ H₂ + O₂（两种物质），一变多，选 **B**。

### 易错点

1. 分解反应的反应物**必须只有一种**，若有多种反应物则不是分解反应。
2. 不是所有分解反应都需要加热——水电解需要通电，双氧水分解需要催化剂。
3. 高锰酸钾分解生成三种物质（K₂MnO₄、MnO₂、O₂），是一变三，但仍是分解反应（只要一种反应物即可）。
