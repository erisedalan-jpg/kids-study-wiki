# 化合反应

> **一句话**：化合反应是指由两种或两种以上物质生成一种新物质的化学反应，特征是"多变一"（A + B → AB）。
> **English**: A combination reaction is a chemical reaction in which two or more substances combine to form a single new product (A + B → AB).

---

## 📚 给 6-12 岁（自读版）

化合反应就像"合并同类项"——多种物质合在一起变成一种新物质！

**记公式：多变一**

化合反应的特点：**两种（或多种）东西 → 一种新东西**

用字母表示就是：A + B → AB

**三个例子**

1. **碳 + 氧气 → 二氧化碳**：碳（炭）在空气中燃烧，和氧气结合，生成二氧化碳。这是两种物质变成一种物质——化合反应！
2. **铁 + 氧气 → 四氧化三铁**：铁丝在纯氧中燃烧，铁和氧气合并，生成一种黑色物质。
3. **氢气 + 氧气 → 水**：氢气在氧气中燃烧，生成水。

**化合反应的特点：反应前物质种类多，反应后只有一种产物。**

**口诀**：化合反应多变一，A加B变AB，两种变一种，记住这公式！

## 🎓 给 12+（进阶版）

### 定义

化合反应是四大基本反应类型之一，指由两种或两种以上的物质（单质或化合物）反应生成**一种**新物质的化学反应。通式为：

$$A + B \rightarrow AB$$

化合反应包括：单质 + 单质、单质 + 化合物、化合物 + 化合物等组合方式。

### 性质 / 反应 / 公式

| 反应物组合 | 举例 | 化学方程式 |
|-----------|------|-----------|
| 单质 + 单质 | 氢气和氧气点燃 | $2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$ |
| 单质 + 单质 | 铁在氧气中燃烧 | $3Fe + 2O_2 \overset{\text{点燃}}{=} Fe_3O_4$ |
| 单质 + 单质 | 碳充分燃烧 | $C + O_2 \overset{\text{点燃}}{=} CO_2$ |
| 单质 + 化合物 | CO₂ 与水反应 | $CO_2 + H_2O = H_2CO_3$ |
| 化合物 + 化合物 | CaO 与水 | $CaO + H_2O = Ca(OH)_2$ |

**四大基本反应类型对比**

| 类型 | 特征 | 举例 |
|------|------|------|
| 化合反应 | 多变一（A+B→AB） | C + O₂ → CO₂ |
| 分解反应 | 一变多（AB→A+B） | 2H₂O → 2H₂ + O₂ |
| 置换反应 | 单质 + 化合物 ↔ 单质 + 化合物 | Fe + CuSO₄ → FeSO₄ + Cu |
| 复分解反应 | 两化合物交换成分 | HCl + NaOH → NaCl + H₂O |

### 典型实验 / 例题

**例题**：下列反应属于化合反应的是（ ）
A. $2KMnO_4 \overset{\Delta}{=} K_2MnO_4 + MnO_2 + O_2\uparrow$
B. $CaO + H_2O = Ca(OH)_2$
C. $Fe + CuSO_4 = FeSO_4 + Cu$
D. $HCl + NaOH = NaCl + H_2O$

**解析**：A 是分解反应（一变多）；C 是置换反应；D 是复分解反应；B 中 CaO（一种物质）+ H₂O（一种物质）= Ca(OH)₂（一种物质），两种变一种，选 **B**。

### 易错点

1. 化合反应不要求反应物是单质，化合物也可以参与化合反应（如 CaO + H₂O）。
2. 化合反应的产物只能有一种，若有两种及以上产物，则不是化合反应。
3. 生成氧化物的反应（如 C + O₂ → CO₂）是化合反应，但不是所有化合反应都是燃烧。
