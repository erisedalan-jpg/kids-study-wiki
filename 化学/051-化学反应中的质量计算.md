# 化学反应中的质量计算

> **一句话**：利用化学方程式中各物质的相对分子质量之比（等于质量比）和质量守恒定律，可以计算参与反应或生成的物质质量。
> **English**: Using the ratios of relative molecular masses in a balanced chemical equation (which equal mass ratios) and the law of conservation of mass, we can calculate the masses of reactants consumed or products formed.

---

## 📚 给 6-12 岁（自读版）

学了化学方程式，就能做一个很实用的事情——计算化学反应时要用多少原料，能产生多少产品！这就是**化学计量计算**。

**方法：比例法**

化学方程式中每种物质的质量比 = 相对分子质量 × 系数

以氢气燃烧为例：$2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$

各物质质量比：
- H₂ : O₂ : H₂O = (2×2) : (1×32) : (2×18) = 4 : 32 : 36 = 1 : 8 : 9

**例题**：4 g 氢气完全燃烧，需要多少氧气？生成多少水？

用比例：
- H₂ 质量 4 g，O₂ 需要：$4 \times \frac{32}{4} = 32$ g
- 生成水：$4 \times \frac{36}{4} = 36$ g
- 验证：4 + 32 = 36 ✓（质量守恒）

**步骤**

1. 写出配平的化学方程式
2. 在相关物质下面写出相对分子质量 × 系数
3. 设未知量为 x，列比例
4. 解方程求 x
5. 验证（反应物质量和 = 生成物质量和）

**口诀**：方程式下写质量，已知未知列比例，相对质量乘系数，交叉相乘解方程！

## 🎓 给 12+（进阶版）

### 定义

化学计量计算（stoichiometry）是依据配平的化学方程式，利用各物质的相对分子质量与化学计量数之积的比值（即质量比），通过比例关系计算参与反应或生成的物质质量。

**核心原理**：化学方程式中，各物质的质量比 = $(M_r \times \text{系数})$ 之比，由质量守恒定律保证。

### 性质 / 反应 / 公式

**计算模型（标准步骤）**

以 $CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$ 为例，计算 100 g $CaCO_3$ 反应生成多少 $CO_2$。

**步骤一**：写配平方程式，标注各物质相对分子质量 × 系数

$$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$$

在需要的物质下方写质量比：

$$\underbrace{CaCO_3}_{100} \quad ... \quad \underbrace{CO_2}_{44}$$

**步骤二**：设未知量，列比例

设生成 CO₂ 为 $x$：

$$\frac{100}{100} = \frac{44}{x}$$

$$x = 44 \text{ g}$$

**常见反应的质量关系**

| 反应 | 反应物质量 → 生成物质量 |
|------|----------------------|
| $2H_2 + O_2 \rightarrow 2H_2O$ | 4g H₂ + 32g O₂ = 36g H₂O |
| $CaCO_3 \rightarrow CaO + CO_2$ | 100g CaCO₃ → 56g CaO + 44g CO₂ |
| $C + O_2 \rightarrow CO_2$ | 12g C + 32g O₂ = 44g CO₂ |
| $2H_2O \rightarrow 2H_2 + O_2$ | 36g H₂O → 4g H₂ + 32g O₂ |

**注意事项**

| 容易出错点 | 正确做法 |
|---------|---------|
| 使用未配平方程式 | 必须先配平 |
| 质量比使用错误 | $M_r \times$ 系数，不只是 $M_r$ |
| 单位不统一 | 统一用 g 或 kg |
| 忘记验证 | 检查：反应物质量和 = 生成物质量和 |

### 典型实验 / 例题

**例题**：实验室制备 $CO_2$：$CaCO_3 + 2HCl = CaCl_2 + H_2O + CO_2\uparrow$

已知：25 g $CaCO_3$ 与足量盐酸反应，求生成 $CO_2$ 的质量。

**解析**：

$$\underbrace{CaCO_3}_{100} + 2HCl = CaCl_2 + H_2O + \underbrace{CO_2\uparrow}_{44}$$

设生成 $CO_2$ 为 $x$：

$$\frac{100}{25} = \frac{44}{x}$$

$$x = \frac{25 \times 44}{100} = 11 \text{ g}$$

答：生成 $CO_2$ 11 g。

**例题 2**：配制 NH₃ 的哈伯法：$N_2 + 3H_2 \underset{\text{催化剂}}{\overset{\text{高温高压}}{⇌}} 2NH_3$

问：14 g N₂ 完全反应需要多少 H₂？

$$\underbrace{N_2}_{28} + 3\underbrace{H_2}_{6} = 2NH_3$$

$$\frac{28}{14} = \frac{6}{x} \Rightarrow x = 3 \text{ g}$$

### 易错点

1. 质量比中的"质量"是**相对分子质量 × 系数**，不是只用相对分子质量，不能忽略系数！
2. 计算时要保持**单位一致**（全用 g 或全用 kg）。
3. 题目中"足量"的试剂不需要计算，只计算有限量（已知量）的那种物质。
