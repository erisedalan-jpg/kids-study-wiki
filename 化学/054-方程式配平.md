# 方程式配平

> **一句话**：方程式配平是通过在化学式前添加适当的化学计量数，使方程式两边各元素原子数目相等，满足质量守恒定律，常用最小公倍数法。
> **English**: Balancing chemical equations involves adding appropriate coefficients before formulas so that the number of each type of atom is equal on both sides, satisfying the law of conservation of mass; the least common multiple method is commonly used.

---

## 📚 给 6-12 岁（自读版）

写完化学方程式，还要检查两边的原子数是否相等——如果不相等，就要在化学式前面加数字（系数）来"平衡"，这就叫**配平**！

**为什么要配平？**

化学反应遵守质量守恒定律——反应前后原子种类和数量不变。所以方程式两边，每种原子的数目必须相等。

**配平的步骤（以水的生成为例）**

未配平的方程式：$H_2 + O_2 \rightarrow H_2O$

检查两边原子数：
- 左边：2个H，2个O
- 右边：2个H，1个O ← O不相等！

修改：给 H₂O 前加系数 2：
$$H_2 + O_2 \rightarrow 2H_2O$$

再检查：
- 左：2个H，2个O；右：4个H，2个O ← H不相等！

再给 H₂ 前加系数 2：
$$2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$$

检查：左：4个H，2个O；右：4个H，2个O ✓ 配平完成！

**配平口诀**

1. 先写出反应物和生成物（化学式必须正确）
2. 用最小公倍数找出需要的系数
3. 验证两边每种原子数相等
4. 如果系数有公约数，化简到最简整数比

**记住：只能改系数，不能改化学式里的数字！**

## 🎓 给 12+（进阶版）

### 定义

化学方程式配平是指通过调整各物质前的化学计量数（整数，且为最简比），使方程式两边每种元素的原子总数相等，从而满足质量守恒定律。不能通过改变化学式中的角标（原子数）来配平。

### 性质 / 反应 / 公式

**配平方法一：最小公倍数法（LCM 法）**

步骤：找出各种原子在两边出现最多的情况，利用最小公倍数确定系数。

**例：配平高锰酸钾分解方程式**

已知反应：$KMnO_4 \rightarrow K_2MnO_4 + MnO_2 + O_2\uparrow$

- K 在右边：K₂MnO₄ 含 2 个 K，故左边需 2 个 KMnO₄
- 加系数：$2KMnO_4 \rightarrow K_2MnO_4 + MnO_2 + O_2\uparrow$
- 检查 Mn：左边 2，右边 1+1=2 ✓
- 检查 O：左边 8，右边 4+2+2=8 ✓

$$2KMnO_4 \overset{\Delta}{=} K_2MnO_4 + MnO_2 + O_2\uparrow$$

**配平方法二：观察法（简单反应）**

对简单的反应，通过观察直接调整系数。

**例：配平 $Fe + O_2 \rightarrow Fe_3O_4$**

- Fe₃O₄ 含 3Fe 和 4O
- 需 3Fe（左边系数 3）和 2O₂（提供 4O）
- $3Fe + 2O_2 \overset{\text{点燃}}{=} Fe_3O_4$ ✓

**常见配平实例**

| 反应 | 配平方程式 |
|------|-----------|
| 碳燃烧（完全）| $C + O_2 \overset{\text{点燃}}{=} CO_2$ |
| 碳不完全燃烧 | $2C + O_2 \overset{\text{点燃（不足氧）}}{=} 2CO$ |
| 铁燃烧 | $3Fe + 2O_2 \overset{\text{点燃}}{=} Fe_3O_4$ |
| 氢气燃烧 | $2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$ |
| 水电解 | $2H_2O \overset{\text{通电}}{=} 2H_2\uparrow + O_2\uparrow$ |
| 碳酸钙分解 | $CaCO_3 \overset{\text{高温}}{=} CaO + CO_2\uparrow$ |

**配平验证**

配平完成后，逐一核对每种元素原子数：

| 元素 | 左边 | 右边 |
|------|------|------|
| （以方程式中出现的元素逐一列表）| ... | ... |

### 典型实验 / 例题

**例题**：配平方程式 $P + O_2 \rightarrow P_2O_5$

**解析**：
- P₂O₅ 含 2 个 P → 左边需 2P（P 前系数 2）
- P₂O₅ 含 5 个 O → 需要 5/2 个 O₂，但系数必须是整数
- 将所有系数乘 2：$4P + 5O_2 \rightarrow 2P_2O_5$
- 检查：P 左 4 = 右 4 ✓；O 左 10 = 右 10 ✓

$$4P + 5O_2 \overset{\text{点燃}}{=} 2P_2O_5$$

### 易错点

1. **只能加系数，不能改化学式**：如把 H₂O 写成 H₄O₂ 是错误的（改变了物质本身）。
2. 配平后，各系数要是**最简整数比**：如果 $4H_2 + 2O_2 = 4H_2O$，应约分为 $2H_2 + O_2 = 2H_2O$。
3. 方程式配平要在**化学式确定正确后**再配平——如果化学式（产物）写错了，无论怎么配都不对。
