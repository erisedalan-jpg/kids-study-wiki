# 最小公倍数法

> **一句话**：最小公倍数法是配平化学方程式的常用方法，通过找出方程式两边某种原子出现数量的最小公倍数来确定各化学式的系数。
> **English**: The least common multiple (LCM) method is a common technique for balancing chemical equations by finding the LCM of the number of atoms of a key element on both sides to determine the coefficients.

---

## 📚 给 6-12 岁（自读版）

配平方程式时，有个聪明的方法叫**最小公倍数法**——先找出某种原子在两边个数的最小公倍数，然后用它来确定系数！

**什么是最小公倍数？**

2 和 3 的最小公倍数是 6（既是 2 的倍数，又是 3 的倍数）。

在配平时，如果某种原子在左边有 2 个，右边有 3 个，它们的最小公倍数是 6。让两边的这种原子都变成 6 个，就通过加系数来实现。

**步骤（以 Al + O₂ → Al₂O₃ 为例）**

1. 选择最复杂的化合物（Al₂O₃），包含 Al 和 O 各自的数量
2. Al₂O₃ 里 O 是 3，O₂ 里 O 是 2，LCM(3,2)=6
   - O₂ 前面加系数 3（3×2=6个O）
   - Al₂O₃ 前面加系数 2（2×3=6个O）
3. $Al + 3O_2 \rightarrow 2Al_2O_3$，左边 Al 共需 4 个
4. $4Al + 3O_2 \overset{\text{高温}}{=} 2Al_2O_3$

验证：Al 左4=右4 ✓；O 左6=右6 ✓ 配平完成！

**口诀**：找出关键原子，左右数量求最小公倍，除以各自数量得系数，验证两边相等就搞定！

## 🎓 给 12+（进阶版）

### 定义

最小公倍数法（LCM method）是化学方程式配平的常用策略：选择方程式中某种原子（通常选出现次数较多、较难配平的原子），计算其在方程两边各化学式中出现数量的最小公倍数（LCM），以此确定各化学式的系数。

### 性质 / 反应 / 公式

**操作步骤**

1. 正确写出反应物和生成物的化学式（化学式必须正确）
2. 选择"关键元素"（通常是化合物中出现最多次的元素，或连接两边的共同元素）
3. 计算该元素在两边各化学式中原子数的 LCM
4. 用 LCM 除以各化学式中该元素的原子数，得到各化学式的系数
5. 根据已定系数推出其他化学式的系数
6. 验证所有元素两边原子数相等，并化简到最简整数比

**例：配平 $Al + O_2 \rightarrow Al_2O_3$**

| 步骤 | 操作 |
|------|------|
| 1 | 写出：$Al + O_2 \rightarrow Al_2O_3$ |
| 2 | 选 O 作关键元素（连接 O₂ 和 Al₂O₃）|
| 3 | O₂ 中 O=2，Al₂O₃ 中 O=3，LCM(2,3)=6 |
| 4 | O₂ 系数 = 6÷2 = 3；Al₂O₃ 系数 = 6÷3 = 2 |
| 5 | $Al + 3O_2 \rightarrow 2Al_2O_3$；Al 需 4 个，Al 系数 = 4 |
| 6 | $4Al + 3O_2 = 2Al_2O_3$，验证：Al 左4=右4，O 左6=右6 ✓ |

**例：配平 $C + CO_2 \overset{\text{高温}}{=} CO$（不完全燃烧）**

- 右边 CO 中 C=1，O=1；左边 CO₂ 中 C=1，O=2
- 设 CO 系数为 a，$C + CO_2 = aCO$
- O：2 = a → a=2；C：1+1=2 ✓（验证）
- $C + CO_2 \overset{\text{高温}}{=} 2CO$ ✓

**常见配平结果记忆表**

| 反应 | 配平方程式 |
|------|-----------|
| 氢气 + 氧气 | $2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$ |
| 磷 + 氧气 | $4P + 5O_2 \overset{\text{点燃}}{=} 2P_2O_5$ |
| 铁 + 氧气 | $3Fe + 2O_2 \overset{\text{点燃}}{=} Fe_3O_4$ |
| 铝 + 氧气 | $4Al + 3O_2 \overset{\text{高温}}{=} 2Al_2O_3$ |
| 水电解 | $2H_2O \overset{\text{通电}}{=} 2H_2\uparrow + O_2\uparrow$ |

### 典型实验 / 例题

**例题**：用最小公倍数法配平 $Na + H_2O \rightarrow NaOH + H_2\uparrow$

**解析**：
- 选 H 作关键元素：H₂O 含 2H，NaOH 含 1H，H₂ 含 2H
- 设 Na 系数为 a，H₂O 系数为 b，NaOH 系数为 c，H₂ 系数为 d
- Na：a = c；O：b = c；H：2b = c + 2d
- 设 c=2：a=2，b=2，2×2=2+2d→d=1
- $2Na + 2H_2O = 2NaOH + H_2\uparrow$
- 验证：Na 2=2 ✓，O 2=2 ✓，H 4=2+2=4 ✓ 配平 ✓

### 易错点

1. LCM 法选的"关键元素"要连接方程式两边的化合物——孤立的单质（如 C、Fe）通常最后确定系数。
2. 所有系数确定后，必须检查**每种元素**两边原子数相等，不只是配平用的那种。
3. 如果系数有公约数，必须约分到最简整数比——如 $4H_2 + 2O_2 = 4H_2O$ 要化简为 $2H_2 + O_2 = 2H_2O$。
