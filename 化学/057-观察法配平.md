# 观察法配平

> **一句话**：观察法配平（试凑法）是通过直接观察化学方程式各元素原子数目，逐步调整系数，直到两边原子数相等的配平方法，适用于较简单的反应。
> **English**: The inspection method (trial-and-error) balances chemical equations by directly observing and adjusting coefficients until the number of each type of atom is equal on both sides; suitable for simpler reactions.

---

## 📚 给 6-12 岁（自读版）

**观察法**又叫"试凑法"，就是靠眼睛看和直觉猜，一步步把方程式两边的原子数调平。适合简单的方程式，特别是只有两三种物质的反应。

**观察法的步骤**

1. 先观察最复杂的物质（化合物），确定它的系数
2. 根据这个系数推出其他物质的系数
3. 反复检查，直到两边每种原子数相等

**例子：配平 $CaCO_3 \rightarrow CaO + CO_2$**

左边：1 Ca，1 C，3 O
右边：1 Ca，1 C，3 O（CaO 中 1O + CO₂ 中 2O = 3O）✓

所有原子数相等！方程式已配平，系数都是 1（省略不写）：

$$CaCO_3 \overset{\text{高温}}{=} CaO + CO_2\uparrow$$

**另一个例子：配平 $H_2O_2 \rightarrow H_2O + O_2$**

左边：2H，2O
右边：2H，3O ← 不平衡！

加系数：给 H₂O₂ 加系数 2，给 H₂O 加系数 2：

$2H_2O_2 = 2H_2O + O_2$

检查：H 左4=右4 ✓，O 左4=右(2+2)=4 ✓ 配平完成！

**口诀**：观察法先看复杂式，逐步推出其他系数，反复检查每种原子，两边相等就完成！

## 🎓 给 12+（进阶版）

### 定义

观察法（inspection method）又称试凑法，是配平化学方程式最基础的方法：通过肉眼观察，从最复杂的化学式（通常是含元素种类最多的化合物）入手，逐步确定各化学式的化学计量数，直到方程式两边每种元素原子数相等为止。

### 性质 / 反应 / 公式

**观察法操作策略**

| 策略 | 说明 |
|------|------|
| 先找复杂物 | 从原子数最多的化合物开始，其系数通常先确定 |
| 单质系数最后定 | 单质只含一种元素，最灵活，留到最后调整 |
| 氧原子最后配 | 含氧物质通常多，先配其他元素，最后验证氧 |
| 奇数变偶数技巧 | 若某元素一边为奇数，另一边为偶数，乘 2 |

**奇偶技巧实例**

配平 $CH_4 + O_2 \rightarrow CO_2 + H_2O$

- C: 左1，右1 ✓（先不动 C）
- H: 左4，右2 → H₂O 前加 2：$CH_4 + O_2 = CO_2 + 2H_2O$
- O: 左2，右(2+2)=4 → O₂ 前加 2：$CH_4 + 2O_2 = CO_2 + 2H_2O$
- 检查：C 左1=右1 ✓，H 左4=右4 ✓，O 左4=右4 ✓

$$CH_4 + 2O_2 \overset{\text{点燃}}{=} CO_2 + 2H_2O$$

**常见用观察法配平的方程式**

| 反应 | 配平结果 |
|------|---------|
| 水电解 | $2H_2O \rightarrow 2H_2 + O_2$ |
| 碳酸钙分解 | $CaCO_3 \rightarrow CaO + CO_2$ |
| 甲烷燃烧 | $CH_4 + 2O_2 \rightarrow CO_2 + 2H_2O$ |
| 氧化铜还原 | $CuO + H_2 \rightarrow Cu + H_2O$ |
| 铁与盐酸 | $Fe + 2HCl = FeCl_2 + H_2\uparrow$ |

### 典型实验 / 例题

**例题**：用观察法配平 $CO + Fe_2O_3 \rightarrow Fe + CO_2$

**解析**：
- Fe₂O₃ 中有 2 个 Fe，Fe 前加系数 2
- $CO + Fe_2O_3 = 2Fe + CO_2$
- O：左(1+3)=4，右 2 → CO 前加系数 3
- $3CO + Fe_2O_3 = 2Fe + 3CO_2$
- 检查：C 左3=右3 ✓，O 左(3+3)=6=右6 ✓，Fe 左2=右2 ✓

$$3CO + Fe_2O_3 \overset{\text{高温}}{=} 2Fe + 3CO_2$$

### 易错点

1. 观察法适合简单方程式；复杂的反应（多种元素）建议用代数法（设未知系数解方程）。
2. 配平时不能改化学式中的角标（下标），只能改化学式前面的系数。
3. 配平完成后必须全部检查，不能只验证一两种元素就认为完成。
