# 二氧化碳

> **一句话**：二氧化碳（CO₂）是碳完全燃烧的产物，无色无味，密度比空气大，不支持燃烧，能与水反应生成碳酸，是温室气体之一，也是光合作用的原料。
> **English**: Carbon dioxide (CO₂) is the product of complete carbon combustion; colorless and odorless, denser than air, does not support combustion, reacts with water to form carbonic acid, is a greenhouse gas, and is the raw material for photosynthesis.

---

## 📚 给 6-12 岁（自读版）

我们呼出的气体里有 CO₂，植物吸收 CO₂ 进行光合作用，碳酸饮料里的泡泡也是 CO₂——这种气体在生活中无处不在！

**二氧化碳的物理性质**

- 无色无味的气体
- 密度比空气大（沉底！）
- 能溶于水（所以碳酸饮料可以充入 CO₂）
- 常压下 -78.5°C 会变成"干冰"（固态 CO₂，直接升华，不经过液态）

**二氧化碳的化学性质**

1. **不能燃烧，也不支持燃烧**：把蜡烛放在低处，向杯子里缓缓倒入 CO₂，蜡烛会被"熄灭"（因为 CO₂ 沉底，把氧气挤走了）。
2. **能与水反应**：$CO_2 + H_2O = H_2CO_3$（生成碳酸，有酸性）——这就是碳酸饮料喝起来有点酸的原因！
3. **能与石灰水反应**：$CO_2 + Ca(OH)_2 = CaCO_3\downarrow + H_2O$——石灰水变浑浊，用来检验 CO₂。
4. **能与 NaOH 反应**：$CO_2 + 2NaOH = Na_2CO_3 + H_2O$

**CO₂ 的用途**

- **灭火**：不支持燃烧，可灭油类火灾（不能用水）
- **碳酸饮料**：充 CO₂ 加压溶入
- **干冰（固态 CO₂）**：食品保鲜、舞台烟雾效果
- **植物光合作用**：CO₂ + H₂O + 光能 → 有机物 + O₂

**口诀**：CO₂ 无色无味比空气重，不燃不助燃，变浑石灰水，温室效应会增强！

## 🎓 给 12+（进阶版）

### 定义

二氧化碳（CO₂）是碳（+4 价）的氧化物，是碳完全燃烧或碳酸盐分解的主要产物。CO₂ 是非极性线性分子（O=C=O），键角 180°，无永久偶极矩，但有大的四极矩。是自然界碳循环的关键环节，也是地球大气层的温室气体（占温室效应约 26%）。

### 性质 / 反应 / 公式

**物理性质**

| 性质 | 数值 |
|------|------|
| 相对分子质量 | 44 |
| 密度（标准状况）| 1.977 g/L（比空气 1.293 g/L 重）|
| 沸点 | -78.5°C（直接升华为干冰）|
| 溶解度（0°C）| 1.713 L/L water |
| 颜色/气味 | 无色无味 |

**化学反应**

| 反应 | 化学方程式 |
|------|-----------|
| 与水反应 | $CO_2 + H_2O \rightleftharpoons H_2CO_3$ |
| 与石灰水 | $CO_2 + Ca(OH)_2 = CaCO_3\downarrow + H_2O$ |
| 与 NaOH（少量 CO₂）| $CO_2 + 2NaOH = Na_2CO_3 + H_2O$ |
| 与 NaOH（过量 CO₂）| $CO_2 + NaOH = NaHCO_3$ |
| 与 C 反应 | $CO_2 + C \overset{\text{高温}}{=} 2CO$ |
| 光合作用 | $6CO_2 + 6H_2O \overset{\text{光能}}{=} C_6H_{12}O_6 + 6O_2$ |
| 被 CO 还原 | $CO_2 + C \rightarrow 2CO$（逆水气变换）|

**CO₂ 的检验与鉴别**

- 方法 1：通入澄清石灰水，变浑浊（CaCO₃ 沉淀生成）
- 方法 2：用湿润的石蕊试纸，变红（碳酸显酸性）
- 区别：方法 1 更灵敏，方法 2 也会对 SO₂ 等酸性气体反应，不专一

**温室效应**

CO₂ 能吸收并再辐射红外线，使地球表面温度升高。工业革命以来，大气 CO₂ 浓度从 280 ppm 增至 420+ ppm（2023 年），造成全球变暖。

### 典型实验 / 例题

**例题**：鉴别 CO₂ 和 CO 的方法是（ ）
A. 点燃，观察火焰颜色  
B. 分别通入澄清石灰水，观察是否变浑浊  
C. 观察颜色  
D. 将燃着的木条放入两种气体中

**解析**：A 中 CO 可燃，CO₂ 不可燃，点燃后有区别，但操作危险；B 中 CO₂ 使石灰水变浑浊，CO 不能，最简单有效；C 两者均无色；D 均会熄灭木条（CO₂ 灭火，CO 无助燃性）。选 **B**。

### 易错点

1. CO₂ 不支持燃烧，不等于 CO₂ 完全"没有反应性"——CO₂ 与碱（NaOH、Ca(OH)₂）等可以反应。
2. 向澄清石灰水通入**过量** CO₂，变浑浊后会**再变清**——CaCO₃ + CO₂ + H₂O = Ca(HCO₃)₂（可溶）。
3. CO₂ 的密度比空气大，所以扑灭油类火灾从上方倒入 CO₂ 有效（沉到底部，隔绝氧气）。
