# 二氧化碳与水反应

> **一句话**：CO₂溶于水后与水反应生成碳酸（H₂CO₃），碳酸能使石蕊变红，但碳酸不稳定，加热会分解还原为CO₂和H₂O。
> **English**: CO₂ dissolves in water and reacts to form carbonic acid (H₂CO₃), which turns litmus red; however, carbonic acid is unstable and decomposes back to CO₂ and H₂O upon heating.

---

## 📚 给 6-12 岁（自读版）

把 CO₂ 通进水里，会发生什么？它不只是溶解——它还会和水"结合"，生成一种酸！这就是**碳酸（H₂CO₃）**。

**CO₂ 和水反应**

$$CO_2 + H_2O \rightleftharpoons H_2CO_3$$

这个反应是**可逆的**（用 ⇌ 表示），意思是碳酸也可以分解还原成 CO₂ 和 H₂O。

**碳酸是酸性的！**

用**紫色石蕊试液**来检验：
- 把 CO₂ 通入含紫色石蕊的水中
- 紫色石蕊**变红**（酸性溶液会使石蕊变红）
- 把溶液加热，CO₂ 跑走，溶液**变回紫色**（碳酸分解，酸性消失）

**生活中的例子**

- **碳酸饮料**：向水中加压溶入 CO₂，生成碳酸，所以喝起来酸酸的、有气泡！开盖后 CO₂ 逸出，酸味减弱。
- **雨水偏酸**：空气中 CO₂ 溶入雨水，使正常雨水 pH 约 5.6（弱酸性），不是中性！
- **海洋酸化**：大气 CO₂ 增加 → 溶入海洋 → 海水变酸 → 影响珊瑚礁（贝壳溶解）。

**碳酸很不稳定！**

碳酸（H₂CO₃）在常温下就会慢慢分解，加热分解更快：

$$H_2CO_3 \overset{\Delta}{=} H_2O + CO_2\uparrow$$

这就是为什么：碳酸饮料放置后会越来越不酸，石灰石洞穴里的钟乳石（碳酸氢钙分解）慢慢形成。

**口诀**：CO₂ 入水生碳酸，酸性让石蕊变红，碳酸不稳加热分，CO₂ 和水又分开！

## 🎓 给 12+（进阶版）

### 定义

CO₂ 与水的反应是 CO₂ 溶于水后与 H₂O 发生的可逆反应，生成弱酸——碳酸（H₂CO₃）。碳酸是二元弱酸，在水溶液中部分电离，Ka₁=4.3×10⁻⁷，Ka₂=4.7×10⁻¹¹。碳酸极不稳定，常温即可分解，是自然界碳循环的重要环节。

### 性质 / 反应 / 公式

**主反应（可逆）**

$$CO_2 + H_2O \rightleftharpoons H_2CO_3$$

注意：方程式用 ⇌（可逆号），因为反应不能进行到底，平衡时水中 CO₂ 大部分以 CO₂(aq) 形式存在，仅约 0.3% 转化为 H₂CO₃。

**碳酸的分解（加热）**

$$H_2CO_3 \overset{\Delta}{=} H_2O + CO_2\uparrow$$

**碳酸与石蕊的反应（指示剂测酸性）**

H₂CO₃ 为弱酸，溶液 pH < 7，使紫色石蕊变红（酸碱指示剂检测）；加热驱走 CO₂ 后溶液碳酸消失，pH 升高，石蕊变回紫色。

**碳酸与碱反应（延伸）**

| 碱 | 反应方程式 | 产物 |
|----|-----------|------|
| NaOH（少量 CO₂）| $CO_2 + 2NaOH = Na_2CO_3 + H_2O$ | 碳酸钠 |
| NaOH（过量 CO₂）| $CO_2 + NaOH = NaHCO_3$ | 碳酸氢钠 |
| Ca(OH)₂（适量）| $CO_2 + Ca(OH)_2 = CaCO_3\downarrow + H_2O$ | 碳酸钙（沉淀）|
| Ca(OH)₂（过量 CO₂）| $CaCO_3 + CO_2 + H_2O = Ca(HCO_3)_2$ | 碳酸氢钙（溶解）|

**CO₂ 与水反应的意义**

| 场景 | 说明 |
|------|------|
| 碳酸饮料 | 加压溶入 CO₂ → H₂CO₃，赋予酸味和气泡 |
| 正常雨水 pH≈5.6 | 大气 CO₂ 溶入雨水形成 H₂CO₃ |
| 海洋酸化 | 大气 CO₂ 升高 → 海水 pH 下降（自工业革命降低约 0.1）|
| 溶洞形成 | CaCO₃ + CO₂ + H₂O = Ca(HCO₃)₂（可溶）→ 石灰岩溶蚀 |
| 钟乳石形成 | Ca(HCO₃)₂ 加热分解 → CaCO₃ 沉积 |
| 人体血液缓冲 | H₂CO₃/HCO₃⁻ 是血液最重要的 pH 缓冲对 |

### 典型实验 / 例题

**实验：CO₂ 与水反应（石蕊指示实验）**

1. 准备：将紫色石蕊喷在 4 朵纸花上
2. 对照组：将干燥的 CO₂ 通入干燥纸花 → 不变色（CO₂ 本身不是酸）
3. 另一朵纸花用水喷湿 → 放入干燥 CO₂ 中 → 变红（CO₂+H₂O→H₂CO₃）
4. 加热变红纸花 → 颜色消退（H₂CO₃ 分解，CO₂ 逸出）
5. 结论：CO₂ + H₂O → H₂CO₃（酸性），H₂CO₃ 不稳定

**例题**：CO₂ 通入紫色石蕊溶液后，溶液变红，原因是（ ）

A. CO₂ 是酸性气体，直接与石蕊反应  
B. CO₂ 溶于水生成 H₂CO₃，H₂CO₃ 使石蕊变红  
C. CO₂ 本身使石蕊变红  
D. CO₂ 与石蕊发生化学反应生成红色物质

**解析**：干燥 CO₂ 通入干燥石蕊不变色，说明 CO₂ 本身不是酸，不能直接使石蕊变红；只有 CO₂ 溶于水生成 H₂CO₃ 后，H₂CO₃ 电离出 H⁺，才使石蕊变红。选 **B**。

### 易错点

1. **CO₂ 是酸性氧化物，但 CO₂ 本身不是酸**——CO₂ 不能直接使石蕊变色，必须先与水反应生成 H₂CO₃。
2. **碳酸极不稳定**——在水溶液中只有少量 CO₂ 转化为 H₂CO₃，碳酸无法以纯物质形式分离（稳定存在是不可能的）。
3. 石蕊变红的本质是**H⁺（氢离子）**——任何酸溶液都能使石蕊变红，不是 CO₂ 专有。
4. 加热含 CO₂ 的碳酸溶液后石蕊**变回紫色**，不是蓝色（加热本身不能使石蕊变蓝）。
