# 水

> **一句话**：水（H₂O）是由氢元素和氧元素组成的化合物，是最常见的液体，既是生命的基础，也是重要的化学溶剂和反应参与者。
> **English**: Water (H₂O) is a compound of hydrogen and oxygen; it is the most common liquid, essential for life, and an important solvent and reactant in chemistry.

---

## 📚 给 6-12 岁（自读版）

水是我们最熟悉的物质！喝水、洗手、下雨……水无处不在。在化学里，水也非常重要，它有很多特别的性质。

**水的组成**

水的化学式是 H₂O，表示：
- 每个水分子由 2 个**氢原子**和 1 个**氧原子**构成
- 水是由氢元素和氧元素组成的化合物（不是混合物！）

**水的物理性质**

- 常温下是无色无味的液体
- 密度 1 g/mL（4°C 时最大）
- 沸点 100°C，熔点 0°C（在标准大气压下）
- 是极性溶剂，能溶解很多物质

**水的化学性质**

1. **能发生电解**：通电后，水分解成氢气和氧气。
2. **能与活泼金属反应**：如钠放入水中，剧烈反应生成 NaOH 和 H₂。
3. **能与某些氧化物反应**：如 CaO + H₂O → Ca(OH)₂（生石灰加水放热）。

**水在自然界的循环**

水通过蒸发（液→气）、降水（气→液/固）不断在地球表面和大气层之间循环，称为"水循环"。

**口诀**：H₂O 水分子，两氢一氧来构成，无色无味常温液，溶剂之王是水！

## 🎓 给 12+（进阶版）

### 定义

水（H₂O）是由氢（H）和氧（O）以 2:1 的原子比结合形成的极性共价化合物。水分子呈 V 形结构，键角约 104.5°，因 O 的强电负性使分子具有较大偶极矩，极性强，能与其他极性物质形成氢键，是生命活动不可缺少的溶剂。

### 性质 / 反应 / 公式

**物理性质**

| 性质 | 数值 |
|------|------|
| 密度（4°C）| 1.000 g/cm³ |
| 沸点（1 atm）| 100°C |
| 熔点 | 0°C |
| 比热容 | 4.18 J/(g·°C)（极高，温度缓冲能力强）|
| 电离常数（25°C）| $K_w = 10^{-14}$ |

**化学反应**

| 反应 | 化学方程式 |
|------|-----------|
| 电解水 | $2H_2O \overset{\text{通电}}{=} 2H_2\uparrow + O_2\uparrow$ |
| 与 Na 反应 | $2Na + 2H_2O = 2NaOH + H_2\uparrow$ |
| 与 CaO 反应 | $CaO + H_2O = Ca(OH)_2$ |
| 与 CO₂ 反应 | $CO_2 + H_2O = H_2CO_3$ |
| 与 SO₃ 反应 | $SO_3 + H_2O = H_2SO_4$ |
| 光合作用（生物）| $6CO_2 + 6H_2O \overset{\text{光}}{=} C_6H_{12}O_6 + 6O_2$ |

### 典型实验 / 例题

**例题**：下列关于水的说法正确的是（ ）
A. 水是由氢气和氧气组成的  
B. 纯水是纯净物，蒸馏水含杂质  
C. 水（H₂O）中氢和氧的质量比为 1:8  
D. 液态水变成水蒸气是化学变化

**解析**：A 错（水是化合物，由氢元素和氧元素组成，不是混合物）；B 错（蒸馏水是纯水）；D 错（是物理变化，只是状态变化）；C 中 H₂O 的质量比：H : O = 2×1 : 1×16 = 2 : 16 = 1 : 8，选 **C**。

### 易错点

1. **水由氢元素和氧元素"组成"**（宏观），而不是由"氢气和氧气组成"——氢气和氧气是物质，不是元素。
2. 水是纯净物（固定化学式 H₂O），自然界中的水（江河湖海）是混合物（含有其他溶质）。
3. 水蒸发是物理变化；水电解是化学变化。两者区别记清楚。
