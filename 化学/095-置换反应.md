# 置换反应

> **一句话**：置换反应是指一种单质与一种化合物反应，生成另一种单质和另一种化合物的反应，反应通式为 A + BC → AC + B。
> **English**: A displacement reaction is one where a single element reacts with a compound to produce another element and another compound: A + BC → AC + B.

---

## 📚 给 6-12 岁（自读版）

想象一下这个场景：你正坐在一把椅子上，突然一个更"强壮"的人来了，把你从椅子上"赶走"，坐到了你的位置上。这就是**置换反应**的道理！

在化学里：
- **活泼的金属**（单质 A）
- **进入**化合物（BC）
- **把不活泼的金属**（B）**赶出来**，自己占了位置

**经典例子：**

**铁把铜"赶走"：**
铁片放入蓝色的硫酸铜溶液里，铁片表面慢慢出现**红色**（铜），溶液变成**浅绿色**（FeSO₄）。
$$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$$

**锌和盐酸产生氢气：**
锌放入盐酸，锌把氢"赶走"，产生氢气泡。
$$\text{Zn} + 2\text{HCl} \rightarrow \text{ZnCl}_2 + \text{H}_2\uparrow$$

**铜把银"赶走"：**
铜丝放入硝酸银溶液，铜把银"赶走"，铜丝上出现银白色的银晶体。
$$\text{Cu} + 2\text{AgNO}_3 \rightarrow \text{Cu(NO}_3)_2 + 2\text{Ag}$$

**规律：只有活动性强的才能"赶走"活动性弱的！**

## 🎓 给 12+（进阶版）

### 定义
置换反应（displacement reaction）：一种单质与一种化合物反应，生成另一种单质和另一种化合物。

**通式：** $A + BC \rightarrow AC + B$

其中 A 和 B 都是单质，BC 和 AC 都是化合物。

置换反应属于四大基本反应类型之一（化合、分解、置换、复分解）。

### 常见置换反应分类

**金属置换金属（活泼置换不活泼）：**

$$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$$

$$\text{Cu} + 2\text{AgNO}_3 \rightarrow \text{Cu(NO}_3)_2 + 2\text{Ag}$$

$$\text{Zn} + \text{CuSO}_4 \rightarrow \text{ZnSO}_4 + \text{Cu}$$

**金属置换氢（活泼金属+酸）：**

$$\text{Zn} + 2\text{HCl} \rightarrow \text{ZnCl}_2 + \text{H}_2\uparrow$$

$$\text{Fe} + \text{H}_2\text{SO}_4 \rightarrow \text{FeSO}_4 + \text{H}_2\uparrow$$

$$2\text{Al} + 6\text{HCl} \rightarrow 2\text{AlCl}_3 + 3\text{H}_2\uparrow$$

**非金属置换非金属（活泼非金属置换不活泼）：**

$$\text{Cl}_2 + 2\text{NaBr} \rightarrow 2\text{NaCl} + \text{Br}_2$$

**碳还原金属氧化物：**

$$\text{C} + 2\text{CuO} \xrightarrow{\text{高温}} 2\text{Cu} + \text{CO}_2\uparrow$$

$$2\text{C} + \text{SiO}_2 \xrightarrow{\text{高温}} \text{Si} + 2\text{CO}\uparrow$$

### 判断置换反应的三步法

1. 反应物：一种**单质** + 一种**化合物**
2. 生成物：一种**单质** + 一种**化合物**
3. 符合**活动性规律**（活泼赶不活泼）

### 典型实验 / 例题
**例题**：下列反应属于置换反应的是（）
A. $\text{CaO} + \text{H}_2\text{O} \rightarrow \text{Ca(OH)}_2$（化合）
B. $\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$（置换）
C. $\text{NaOH} + \text{HCl} \rightarrow \text{NaCl} + \text{H}_2\text{O}$（复分解）
D. $\text{2H}_2\text{O}_2 \rightarrow 2\text{H}_2\text{O} + \text{O}_2\uparrow$（分解）

答案：B

### 易错点
- 置换反应必须有**单质参与**，生成物中也必须有**单质**
- Na 加入 CuSO₄ 溶液，不是置换反应（Na 先与水反应）
- 铁与盐酸反应生成的是 **Fe²⁺（FeCl₂）**，不是 Fe³⁺
- 置换反应和氧化还原反应的关系：所有置换反应都是氧化还原反应
