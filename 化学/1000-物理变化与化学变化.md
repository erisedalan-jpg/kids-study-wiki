# 物理变化与化学变化

> **一句话**：物理变化是指没有新物质生成的变化，化学变化是指有新物质生成的变化。
> **English**: Physical changes are changes in which no new substances are formed, while chemical changes involve the formation of new substances.

## 🎓 考点精讲

**定义与本质**  
物理变化和化学变化的区分依据是 **“有无新物质生成”**。没有新物质生成的变化是物理变化，例如水蒸发（$H_2O(l) \to H_2O(g)$）、石油分馏、焰色反应。化学变化则伴随新物质的产生，微观上表现为分子破裂、原子重新组合成新分子，本质是旧化学键断裂和新化学键形成，如铁生锈（$4Fe + 3O_2 + xH_2O \to 2Fe_2O_3·xH_2O$）、煤的干馏、食物的腐败。

**关键判断方法**  
- **根本判据**：反应前后是否产生了组成、结构或性质与原来不同的新物质。  
- 发光、发热、变色、放出气体、生成沉淀等现象 **不能** 作为绝对依据，必须最终归结到新物质的生成。  
- 微观视角：物理变化前后分子（或原子、离子）本身不变，只是聚集状态或排列方式改变；化学变化中分子种类一定改变。

**典型高考设问**  
1. **直接判断**：下列变化中，属于物理变化/化学变化的是？  
2. **结合情境**：将变化分类与物质性质、实验操作（如蒸馏、升华、电解）联系，要求分析变化类型。  
3. **微观示意图**：给出分子模型变化图，判断属于哪种变化并说明理由。  
4. **过程辨析**：分析有化学键断裂的变化是否一定是化学变化（如 $NaCl$ 溶于水仅离子键断裂，无新物质，属物理变化）。

## ⚠️ 高频易错点

- **错点1**：将有发光、放热的现象一律视为化学变化。例如灯泡通电发光、电热丝发热均为物理变化。  
- **错点2**：认为只要有化学键断裂就是化学变化。$NaCl$ 晶体溶于水时离子键断裂，但只发生电离，没有新分子或新晶体生成，属于物理变化；必须同时有新化学键形成才是化学变化。  
- **错点3**：笼统地将“爆炸”归为化学变化。车胎爆炸、蒸汽锅炉爆炸为物理变化（气体压强过大），而火药爆炸、瓦斯爆炸则为化学变化（生成大量气体并放热）。  
- **错点4**：混淆化学变化与新状态产生的区别。如干冰升华（$CO_2(s) \to CO_2(g)$）是物理变化，仅状态改变，无新物质。
