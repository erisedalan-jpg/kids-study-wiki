# 铜

> **一句话**：铜（Cu）是人类最早使用的金属之一，紫红色，导电性和导热性在常见金属中仅次于银，是电气工业的重要材料。
> **English**: Copper (Cu) is one of the first metals used by humans, reddish-brown in color, with excellent electrical and thermal conductivity second only to silver.

---

## 📚 给 6-12 岁（自读版）

铜是人类使用最早的金属之一。古代没有铁之前，人们就用铜做工具和武器，那个时期叫"青铜时代"。铜的化学符号是**Cu**，来自拉丁文"cuprum"。

**铜的特点：**
- **紫红色（铜红色）**，这是铜的独特颜色
- 非常好的导体：家里的电线里面就是铜
- 导热也很好
- 延展性好，可以拉成很细的铜丝

**铜变绿了？**
老铜器（铜像、古钱币）表面常常有绿色的东西，叫做**铜绿（碱式碳酸铜）**。这是铜和空气中的氧气、二氧化碳、水共同作用生成的。铜绿能保护内部的铜不继续被腐蚀。

**铜的用途：**
- **电线**：铜导电好，大量用于制作导线
- **电路板**：电脑、手机里的电路板布线是铜
- **铜管**：空调、暖气管道
- **青铜器**：铜锡合金，历史悠久
- **黄铜**：铜锌合金，做门把手、水龙头

**铜在金属活动性中的位置：**
铜的活动性比铁弱，比银强。铁可以把铜从硫酸铜溶液中"赶出来"。

## 🎓 给 12+（进阶版）

### 定义
铜（Copper，Cu），原子序数 29，相对原子质量 64，是第四周期第IB族过渡金属。紫红色，密度 8.9 g/cm³，熔点 1083°C，是人类最早大规模使用的金属。

### 性质 / 反应

**物理性质：**
- 密度：8.9 g/cm³
- 熔点：1083°C
- 紫红色，有光泽
- 导电性：排金属第二（仅次于银）
- 延展性好

**化学性质（重要反应）：**

与氧气反应（加热）：

$$2\text{Cu} + \text{O}_2 \xrightarrow{\Delta} 2\text{CuO}$$

（铜加热变黑，CuO 为黑色）

生成铜绿：

$$2\text{Cu} + \text{O}_2 + \text{H}_2\text{O} + \text{CO}_2 \rightarrow \text{Cu}_2(\text{OH})_2\text{CO}_3$$

铜与铁在硫酸铜溶液中的置换：

$$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$$

（铁片上析出红色铜，溶液由蓝色变浅绿色）

铜与银的活动性比较：

$$\text{Cu} + 2\text{AgNO}_3 \rightarrow \text{Cu(NO}_3)_2 + 2\text{Ag}$$

**铜的化合价：** +1（亚铜 Cu⁺）和 +2（铜 Cu²⁺）

| 化合物 | 颜色 |
|--------|------|
| CuO | 黑色 |
| Cu₂O | 砖红色 |
| CuSO₄（无水） | 白色 |
| CuSO₄·5H₂O（胆矾） | 蓝色 |
| Cu₂(OH)₂CO₃（铜绿） | 绿色 |

### 典型实验 / 例题
**胆矾（CuSO₄·5H₂O）加热失水：**
蓝色胆矾加热后变白色（无水 CuSO₄），加水又变蓝，可用于检验水的存在。

**例题**：将铁片放入硫酸铜溶液，观察到铁片表面有红色固体析出，溶液颜色由蓝色变为浅绿色，写出反应方程式：
$$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$$

### 易错点
- 铜是**紫红色**，不是银白色
- 铜加热变**黑色**（CuO），不是红色
- 胆矾（CuSO₄·5H₂O）是蓝色，**无水** CuSO₄ 是白色
- 铜的导电性在金属中排**第二**，不是第一（银第一）
