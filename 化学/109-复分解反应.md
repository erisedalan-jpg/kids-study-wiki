# 复分解反应

> **一句话**：复分解反应是两种化合物相互交换离子，生成两种新化合物的反应，通式为 AB + CD → AD + CB。
> **English**: A double displacement reaction (metathesis) is a reaction between two compounds where ions are exchanged to form two new compounds: AB + CD → AD + CB.

---

## 📚 给 6-12 岁（自读版）

复分解反应就像"交换舞伴"。两对夫妻跳舞，突然两人交换伙伴——A 和 D 配对，B 和 C 配对，就生成了两种新化合物！

**复分解反应的通式：**
$$AB + CD \rightarrow AD + CB$$

**举例：**
- 盐酸（HCl）+ 氢氧化钠（NaOH）→ 氯化钠（NaCl）+ 水（H₂O）
  （H 和 Cl 分开，Na 和 OH 分开，然后 Na 和 Cl 配对，H 和 OH 配对成水）

- 硝酸银（AgNO₃）+ 氯化钠（NaCl）→ 硝酸钠（NaNO₃）+ 氯化银（AgCl）↓
  （银和氯配对，生成白色沉淀 AgCl）

**复分解反应发生的条件（满足其一即可）：**
1. 有**沉淀**生成（不溶的固体析出）
2. 有**气体**生成（CO₂、SO₂ 等逸出）
3. 有**水**生成（中和反应必生成水）

## 🎓 给 12+（进阶版）

### 定义
复分解反应（double displacement reaction，又称 metathesis）：两种化合物相互交换离子，生成两种新化合物的反应。

**通式：** $AB + CD \rightarrow AD + CB$

**实质：** 离子交换（non-redox reaction，不是氧化还原反应）

**特点：** 反应物和生成物都是化合物（无单质参与或生成）

### 发生条件

复分解反应（在离子溶液中）发生，需要满足至少一个条件：

| 条件 | 举例 |
|------|------|
| 生成沉淀（↓）| AgNO₃ + NaCl → AgCl↓ + NaNO₃ |
| 生成气体（↑）| Na₂CO₃ + H₂SO₄ → Na₂SO₄ + H₂O + CO₂↑ |
| 生成水（弱电解质）| HCl + NaOH → NaCl + H₂O |

**条件的本质：** 反应使离子浓度降低（促使反应正向进行）

### 典型复分解反应

**酸碱中和（生成水）：**

$$\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$

**酸与盐（生成气体）：**

$$\text{Na}_2\text{CO}_3 + 2\text{HCl} \rightarrow 2\text{NaCl} + \text{H}_2\text{O} + \text{CO}_2\uparrow$$

**盐与盐（生成沉淀）：**

$$\text{AgNO}_3 + \text{NaCl} \rightarrow \text{AgCl}\downarrow + \text{NaNO}_3$$

$$\text{BaCl}_2 + \text{Na}_2\text{SO}_4 \rightarrow \text{BaSO}_4\downarrow + 2\text{NaCl}$$

**碱与盐（生成沉淀）：**

$$2\text{NaOH} + \text{CuSO}_4 \rightarrow \text{Cu(OH)}_2\downarrow + \text{Na}_2\text{SO}_4$$

### 与四种基本反应类型的关系

| 反应类型 | 通式 | 是否氧化还原 |
|----------|------|------------|
| 化合 | A+B→AB | 不一定 |
| 分解 | AB→A+B | 不一定 |
| 置换 | A+BC→AC+B | 一定是 |
| 复分解 | AB+CD→AD+CB | 一定不是 |

### 典型实验 / 例题
**例题**：下列反应属于复分解反应的是（）
A. $\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$（置换）
B. $2\text{H}_2\text{O} \rightarrow 2\text{H}_2 + \text{O}_2$（分解）
C. $\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$（复分解）
D. $2\text{H}_2 + \text{O}_2 \rightarrow 2\text{H}_2\text{O}$（化合）

答案：C

### 易错点
- 复分解反应**不是**氧化还原反应（没有电子转移，化合价不变）
- 置换反应**一定是**氧化还原反应（有单质参与）
- 复分解反应必须有沉淀、气体或水生成之一，否则反应不发生（离子不能自发交换）
- 中和反应是复分解反应的一种特例
