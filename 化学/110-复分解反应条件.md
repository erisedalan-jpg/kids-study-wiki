# 复分解反应条件

> **一句话**：复分解反应发生的条件是生成物中至少有一种沉淀、气体或水（弱电解质），三个条件满足其一即可使反应正向进行。
> **English**: A double displacement reaction proceeds if at least one of the products is a precipitate, a gas, or water (a weak electrolyte).

---

## 📚 给 6-12 岁（自读版）

不是所有化合物混在一起都会发生复分解反应。就像人约会要有理由，复分解反应发生也需要"理由"——生成以下三种之一：

**复分解反应发生的三个条件（满足一个即可）：**

1. **生成沉淀（↓）**——有固体不溶物从溶液里"跑出来"
   - 举例：AgNO₃ + NaCl → AgCl↓（白色沉淀）+ NaNO₃
   - 银和氯结合生成不溶于水的氯化银，反应发生！

2. **生成气体（↑）**——有气泡冒出
   - 举例：Na₂CO₃ + HCl → NaCl + H₂O + CO₂↑
   - 有气体（CO₂）逸出，反应发生！

3. **生成水（H₂O）**——中和反应
   - 举例：NaOH + HCl → NaCl + H₂O
   - 生成水，反应发生！

**如果什么都不生成？**
比如 NaCl + KNO₃（没有沉淀、没有气体、没有水），混合后什么都没发生——离子照样漂在溶液里，没有复分解反应！

## 🎓 给 12+（进阶版）

### 定义
复分解反应条件（初中理解）：在水溶液中，两种化合物（通常是盐、酸、碱）发生离子交换，若产物使某种离子浓度大幅降低，反应即可正向进行。降低离子浓度的方式：
1. 形成难溶盐（沉淀↓）
2. 形成挥发性气体（↑）
3. 形成弱电解质水分子（H₂O）

**微观解释：**
$$\text{H}^+ + \text{OH}^- \rightarrow \text{H}_2\text{O}$$（中和）
$$\text{Ag}^+ + \text{Cl}^- \rightarrow \text{AgCl}\downarrow$$（沉淀）
$$2\text{H}^+ + \text{CO}_3^{2-} \rightarrow \text{H}_2\text{O} + \text{CO}_2\uparrow$$（气体）

### 常见沉淀、气体、水总结

**常见沉淀（不溶物）：**

| 沉淀 | 颜色 | 形成条件 |
|------|------|----------|
| AgCl | 白色 | Ag⁺ + Cl⁻ |
| BaSO₄ | 白色 | Ba²⁺ + SO₄²⁻ |
| CaCO₃ | 白色 | Ca²⁺ + CO₃²⁻ |
| Cu(OH)₂ | 蓝色 | Cu²⁺ + 2OH⁻ |
| Fe(OH)₃ | 红褐色 | Fe³⁺ + 3OH⁻ |
| Fe(OH)₂ | 白色 | Fe²⁺ + 2OH⁻ |

**常见气体（挥发性）：**

| 气体 | 来源 |
|------|------|
| CO₂↑ | 碳酸盐 + 酸 |
| SO₂↑ | 亚硫酸盐 + 酸 |
| H₂S↑ | 硫化盐 + 酸 |
| NH₃↑ | 铵盐 + 强碱（加热）|

### 典型实验 / 例题
**例题**：下列在水溶液中能发生复分解反应的是（）
A. NaCl + KNO₃　B. BaCl₂ + H₂SO₄　C. NaOH + KCl　D. K₂SO₄ + MgCl₂

分析：
- A：无沉淀、气体、水 → 不反应
- B：BaSO₄↓（沉淀）→ **反应**：$\text{BaCl}_2 + \text{H}_2\text{SO}_4 \rightarrow \text{BaSO}_4\downarrow + 2\text{HCl}$
- C：KCl 和 NaOH 无沉淀/气体/水 → 不反应
- D：无难溶盐生成 → 不反应

答案：B

### 易错点
- 没有沉淀、气体、水生成的两种化合物混合：**不反应**（离子不交换）
- Na₂SO₄ + BaCl₂ 能反应（生成 BaSO₄↓），但 K₂SO₄ + MgCl₂ 的产物 MgSO₄ 可溶，KCl 可溶，无沉淀→不反应
- 铵盐 + 碱必须**加热**才产生 NH₃↑；不加热可能不发生
