# 复合肥

> **一句话**：复合肥是含有两种或两种以上植物营养元素（氮、磷、钾）的化肥，能同时提供多种营养，使用更方便高效。
> **English**: Compound fertilizers contain two or more plant nutrient elements (N, P, K), providing multiple nutrients simultaneously for more efficient use.

---

## 📚 给 6-12 岁（自读版）

如果植物既需要氮又需要磷，难道要分两次施不同的肥料？这样太麻烦了！**复合肥**就是把多种营养元素"打包"在一起的肥料，一次施就能补充多种元素！

**复合肥的优点：**
- 一包含多种营养，使用方便
- 养分均衡，不容易出现缺某种元素的问题
- 通常是颗粒状，便于储运

**常见复合肥：**

| 名称 | 化学式 | 含有元素 | 用途 |
|------|--------|---------|------|
| 磷酸二铵 | (NH₄)₂HPO₄ | N + P | 大田作物基肥 |
| 磷酸一铵 | NH₄H₂PO₄ | N + P | 通用型 |
| 硝酸钾 | KNO₃ | N + K | 蔬菜、果树 |
| 氮磷钾复合肥 | 混合配制 | N + P + K | 全能肥料 |

**"氮磷钾"比例的标注：**
复合肥包装上会标注 **N-P-K**（如 15-15-15 表示含氮15%、含磷15%、含钾15%）。

## 🎓 给 12+（进阶版）

### 定义
复合肥（compound fertilizer）：含有两种或两种以上大量营养元素（N、P、K）的化肥。按来源分为化成复合肥（单一化合物）和混配复合肥（多种肥料混合）。

### 主要复合肥品种

| 名称 | 化学式 | N-P-K 含量 | 性质 |
|------|--------|-----------|------|
| 磷酸二铵（DAP）| (NH₄)₂HPO₄ | 18-46-0 | 弱碱性 |
| 磷酸一铵（MAP）| NH₄H₂PO₄ | 11-44-0 | 弱酸性 |
| 硝酸钾 | KNO₃ | 13-0-46 | 中性 |
| 硝酸磷 | NH₄NO₃·Ca(H₂PO₄)₂ | 含N、P | — |

**N-P-K 三元复合肥：** 如 15-15-15，均衡型，适合大多数作物。

### 复合肥的鉴别

**判断复合肥依据：**
1. 含 N：铵盐 + NaOH → NH₃↑（刺激性气味）；含尿素则不产生 NH₃
2. 含 P：加硝酸 + 钼酸铵 → 黄色沉淀（磷的特征反应）
3. 含 K：焰色反应→紫色（K 的特征色）；加入 AgNO₃→白色沉淀（KCl）

### 典型实验 / 例题
**例题**：下列属于复合肥的是（）
A. CO(NH₂)₂（尿素）只含 N　B. KCl 只含 K　C. (NH₄)₂HPO₄ 含 N 和 P　D. Ca₃(PO₄)₂ 只含 P

答案：C

**例题**：某复合肥标注 N-P-K = 16-8-16，则：
含氮 16%，含 P₂O₅ 8%，含 K₂O 16%。100 g 该复合肥中含氮 16 g，含 P₂O₅ 8 g，含 K₂O 16 g。

### 易错点
- 复合肥必须含**两种或两种以上**营养元素（N、P、K）
- KNO₃（硝酸钾）是复合肥（含 N 和 K）
- 复合肥标注中 P 用 **P₂O₅** 表示，K 用 **K₂O** 表示（不是 P 和 K）
