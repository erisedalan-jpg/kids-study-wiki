# 有机物

> **一句话**：有机物是含有碳元素的化合物（CO₂、CO、碳酸盐等少数含碳化合物除外），种类繁多，是生命体的主要组成物质。
> **English**: Organic compounds are carbon-containing compounds (excluding CO₂, CO, carbonates, etc.), extremely diverse, and the main components of living organisms.

---

## 📚 给 6-12 岁（自读版）

"有机"这个词让人觉得很神秘，其实**有机物**就是**含碳的化合物**！植物、动物、食物……生命中的大多数物质都是有机物。

**有机物和无机物的区别：**
- **有机物**：含有碳元素（C）的化合物（但 CO₂、CO、碳酸盐除外）
- **无机物**：不含碳的化合物，或少数含碳但不按有机物规律反应的物质

**生活中的有机物：**
- **糖**（葡萄糖 C₆H₁₂O₆、蔗糖 C₁₂H₂₂O₁₁）：甜味的来源
- **油脂**（植物油、动物脂肪）：储存能量
- **蛋白质**（肉类、鸡蛋）：生命的基础
- **淀粉**（米饭、面条）：主食
- **酒精**（乙醇 C₂H₅OH）：酒的成分
- **醋酸**（CH₃COOH）：醋的酸味

**有机物的特点：**
- 种类极多（已知超过 3000 万种！）
- 大部分可以燃烧（含 C 和 H）
- 结构比无机物复杂

## 🎓 给 12+（进阶版）

### 定义
有机化合物（organic compound）：含有碳元素的化合物。

**例外（含碳但归入无机物）：**
CO₂、CO、H₂CO₃、碳酸盐（Na₂CO₃、CaCO₃）、碳化物（SiC）

**原因（历史）：** 早期认为有机物只能由生物体产生（"生命力论"），但 1828 年维勒人工合成尿素（有机物），打破了这一理论。

### 有机物的特点

| 特点 | 说明 | 举例 |
|------|------|------|
| 数量多 | 已知超过3000万种 | C 可形成多样骨架 |
| 多含 H、O、N | 复杂组成 | 葡萄糖 C₆H₁₂O₆ |
| 多可燃 | 含 C 和 H | 酒精燃烧 |
| 熔点低 | 分子晶体为主 | 多固体熔点 < 300°C |
| 难溶于水 | 多数非极性 | 油脂不溶于水 |
| 反应慢 | 共价键断裂活化能高 | 酯化反应需加热 |

### 有机物的分类

| 类别 | 特征 | 举例 |
|------|------|------|
| 烃 | 只含 C 和 H | 甲烷 CH₄、乙烯 C₂H₄ |
| 醇 | 含 -OH | 乙醇 C₂H₅OH |
| 醛 | 含 -CHO | 乙醛 CH₃CHO |
| 酸 | 含 -COOH | 醋酸 CH₃COOH |
| 酯 | 含 -COO- | 乙酸乙酯 |
| 糖 | 多羟基醛/酮 | 葡萄糖 C₆H₁₂O₆ |
| 蛋白质 | 氨基酸聚合 | — |

### 典型实验 / 例题
**例题**：下列属于有机物的是（）
A. CO₂　B. NaHCO₃　C. CH₄（甲烷）　D. CaCO₃

答案：C（CH₄ 是有机物；CO₂、NaHCO₃、CaCO₃ 虽含碳但归入无机物）

### 易错点
- CO₂、CO、碳酸盐是含碳化合物，但不是有机物（归入无机物）
- 有机物**不一定**可以燃烧（某些含 F、Cl 的有机物难燃烧）
- 有机物可以是固体（如尿素）、液体（如酒精）、气体（如甲烷）
