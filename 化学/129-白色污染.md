# 白色污染

> **一句话**：白色污染是指废旧塑料（尤其是白色塑料袋、泡沫）对环境造成的污染，因难以降解而长期危害生态系统。
> **English**: White pollution refers to the environmental pollution caused by waste plastics (especially white plastic bags and foam), which are difficult to degrade and pose long-term ecological hazards.

---

## 📚 给 6-12 岁（自读版）

马路上、河流里、山坡上……到处都能见到**白色的塑料袋**飘来飘去，这就是"**白色污染**"！

**白色污染的危害：**
1. **影响景观**：废旧塑料破坏城市、自然风景
2. **危害动物**：动物误食塑料袋，无法消化，导致死亡（鸟类、海龟、鲸鱼都是受害者）
3. **污染土壤**：塑料在土壤中阻碍水分和空气流通，影响植物生长
4. **微塑料问题**：塑料分解成极小的微塑料颗粒，进入食物链，最终可能进入人体！
5. **难以降解**：普通塑料需要数百年才能自然分解

**减少白色污染的方法：**
- **少用**：尽量减少使用一次性塑料
- **重复使用**：用布袋代替塑料袋
- **分类回收**：将塑料放入对应的回收桶
- **生物降解塑料**：用玉米淀粉等制成的可降解塑料
- **立法限制**：很多国家和地区已禁止一次性塑料袋

## 🎓 给 12+（进阶版）

### 定义
白色污染（white pollution）：废弃的塑料制品（以白色为主的塑料袋、泡沫塑料）对环境造成的视觉和生态污染，本质是难降解高分子材料的环境残留问题。

### 白色污染的主要来源

| 来源 | 塑料种类 |
|------|----------|
| 购物袋 | PE、PP |
| 快递包装 | PE、PS（泡沫）|
| 餐盒 | PS、PP |
| 矿泉水瓶 | PET |
| 农业地膜 | PE |

### 微塑料问题

**微塑料（microplastics）：** 直径小于 5 mm 的塑料颗粒

**来源：**
1. 塑料制品老化破碎（次生微塑料）
2. 合成纤维洗涤脱落（如涤纶衣物洗涤产生微纤维）
3. 轮胎磨损（次生微塑料）

**危害链：**
塑料 → 微塑料 → 浮游生物摄入 → 鱼类 → 人类餐桌

研究发现人体血液、肺组织中均检出微塑料。

### 解决方案

| 方案 | 说明 |
|------|------|
| 减量使用 | 限制一次性塑料的生产和使用 |
| 可降解塑料 | PLA（聚乳酸）、PBAT 等 |
| 回收利用 | 热塑性塑料熔融再造 |
| 化学回收 | 解聚得到单体，重新聚合 |
| 立法政策 | 禁限塑令 |

**PLA（聚乳酸）：**
以玉米淀粉为原料，发酵得到乳酸，聚合得到 PLA——可生物降解的"绿色塑料"。

### 典型实验 / 例题
**例题**：下列减少白色污染的措施中，最根本的是（）
A. 增加废旧塑料的焚烧量　B. 研发和使用可降解塑料　C. 将废旧塑料深埋地下　D. 把废旧塑料倒入河流

答案：B（A 焚烧产生有毒气体；B 从源头解决，可降解；C 地下仍污染；D 加剧污染）

### 易错点
- "白色污染"不是指"白色物质的污染"，而是指废弃**塑料**（多为白色）的污染
- 焚烧塑料**不是**好的解决方案（产生有毒气体 HCl、二噁英等）
- 微塑料是当前**全球性环境问题**，危害不可忽视
