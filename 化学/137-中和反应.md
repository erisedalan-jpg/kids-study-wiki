# 中和反应

> **一句话**：中和反应是酸与碱发生的反应，生成盐和水，实质是 H⁺ 与 OH⁻ 结合生成水，属于复分解反应。
> **English**: A neutralization reaction is the reaction between an acid and a base, producing salt and water; the essence is H⁺ combining with OH⁻ to form water.

---

## 📚 给 6-12 岁（自读版）

中和反应就像"互相抵消"——酸和碱"打架"，最后两个都消失了，变成了盐和水。

**中和反应的通式：**
酸 + 碱 → 盐 + 水

**日常生活中的中和：**
- 被蜜蜂蛰了（蜂毒是酸性）→ 涂小苏打水（碱性）中和，减轻疼痛
- 被黄蜂蛰了（蜂毒是碱性）→ 涂醋（酸性）中和
- 胃酸过多（胃里盐酸太多）→ 吃胃药（含碱性成分，中和胃酸）
- 工厂酸性废水→ 加石灰（Ca(OH)₂）中和处理后才能排放

**中和反应举例：**
$$\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$
（盐酸 + 烧碱 → 食盐 + 水）

中和反应会**放热**！（酸和碱混合后溶液温度升高）

## 🎓 给 12+（进阶版）

### 定义
中和反应（neutralization reaction）：酸与碱反应，生成盐和水的化学反应。

**通式：** 酸 + 碱 → 盐 + 水

**离子方程式（实质）：**

$$\text{H}^+ + \text{OH}^- \rightarrow \text{H}_2\text{O}$$

中和反应属于**复分解反应**（离子交换），同时是**放热反应**。

### 典型中和反应

| 酸 | 碱 | 产物 | 方程式 |
|----|----|----|--------|
| HCl | NaOH | NaCl + H₂O | $\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$ |
| H₂SO₄ | NaOH | Na₂SO₄ + H₂O | $\text{H}_2\text{SO}_4 + 2\text{NaOH} \rightarrow \text{Na}_2\text{SO}_4 + 2\text{H}_2\text{O}$ |
| HCl | Ca(OH)₂ | CaCl₂ + H₂O | $2\text{HCl} + \text{Ca(OH)}_2 \rightarrow \text{CaCl}_2 + 2\text{H}_2\text{O}$ |
| H₂SO₄ | Ba(OH)₂ | BaSO₄↓ + H₂O | $\text{H}_2\text{SO}_4 + \text{Ba(OH)}_2 \rightarrow \text{BaSO}_4\downarrow + 2\text{H}_2\text{O}$ |

### 中和反应的实验（酸碱滴定）

用 NaOH 溶液滴定盐酸，以酚酞为指示剂：
- 刚好中和时，溶液 pH = 7，酚酞为无色
- 加入一滴过量 NaOH，溶液变红（碱性）

**放热验证：**
将酸碱混合，用温度计测量，溶液温度升高（中和热约 57.3 kJ/mol）。

### 典型实验 / 例题
**例题**：下列反应属于中和反应的是（）
A. $\text{CO}_2 + 2\text{NaOH} \rightarrow \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$（酸性氧化物与碱，不是酸）
B. $\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$（酸与碱，是中和）
C. $\text{CaO} + \text{H}_2\text{O} \rightarrow \text{Ca(OH)}_2$（化合反应）
D. $\text{Na}_2\text{CO}_3 + 2\text{HCl} \rightarrow 2\text{NaCl} + \text{H}_2\text{O} + \text{CO}_2\uparrow$（酸与盐，不是碱）

答案：B

### 易错点
- 中和反应是**酸与碱**的反应，酸与盐的反应**不是**中和
- CO₂ + NaOH 的反应，虽然生成盐和水，但 CO₂ 不是酸，**不是**中和反应
- 中和反应一定**放热**（中和热）
- 中和反应不一定是 pH = 7（强酸弱碱中和，溶液显酸性）
