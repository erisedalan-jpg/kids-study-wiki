# 盐酸

> **一句话**：盐酸是氯化氢（HCl）气体溶于水形成的酸性溶液，是重要的强酸，也存在于人体胃液中帮助消化。
> **English**: Hydrochloric acid is an aqueous solution of hydrogen chloride (HCl) gas, an important strong acid also found in gastric juice.

---

## 📚 给 6-12 岁（自读版）

你的胃里就有盐酸！胃液含有盐酸，帮助消化食物、杀死细菌。但工业用的盐酸比胃酸浓得多，非常危险，会腐蚀皮肤。

**盐酸是什么？**
盐酸 = 氯化氢（HCl）气体溶解在水里。HCl 是气体，我们平时用的盐酸是无色有刺激性气味的液体，有时候浓盐酸会"冒烟"（其实是挥发出来的 HCl 气体在空气中形成小液滴）。

**盐酸的用途：**
- **清洗铁锈**：铁锈（Fe₂O₃）溶于盐酸，露出光亮的铁
- **制造药物**：很多药物的盐酸盐（如盐酸地西泮）
- **胃病**：胃酸过多→用抗酸药中和；胃酸不足→需要补充盐酸
- **工业用途**：制作 PVC、金属表面处理

**盐酸的危险：**
- 浓盐酸具有挥发性，挥发出的 HCl 气体有毒
- 盐酸会腐蚀皮肤，使用时需要戴防护手套和眼镜
- 万一溅到皮肤，用大量水冲洗

## 🎓 给 12+（进阶版）

### 定义
盐酸（hydrochloric acid）是 HCl 气体溶于水的水溶液，完全电离（强酸）：

$$\text{HCl} \xrightarrow{\text{H}_2\text{O}} \text{H}^+ + \text{Cl}^-$$

浓盐酸 HCl 质量分数约 36%~38%，密度约 1.19 g/mL，有挥发性（浓盐酸在空气中"冒烟"）。

### 性质 / 反应

**盐酸与金属的反应：**

$$\text{Fe} + 2\text{HCl} \rightarrow \text{FeCl}_2 + \text{H}_2\uparrow$$

$$\text{Zn} + 2\text{HCl} \rightarrow \text{ZnCl}_2 + \text{H}_2\uparrow$$

$$2\text{Al} + 6\text{HCl} \rightarrow 2\text{AlCl}_3 + 3\text{H}_2\uparrow$$

**盐酸与碱的反应（中和反应）：**

$$\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$

$$2\text{HCl} + \text{Ca(OH)}_2 \rightarrow \text{CaCl}_2 + 2\text{H}_2\text{O}$$

**盐酸与碳酸盐的反应：**

$$2\text{HCl} + \text{Na}_2\text{CO}_3 \rightarrow 2\text{NaCl} + \text{H}_2\text{O} + \text{CO}_2\uparrow$$

$$2\text{HCl} + \text{CaCO}_3 \rightarrow \text{CaCl}_2 + \text{H}_2\text{O} + \text{CO}_2\uparrow$$

**盐酸与氧化物的反应：**

$$6\text{HCl} + \text{Fe}_2\text{O}_3 \rightarrow 2\text{FeCl}_3 + 3\text{H}_2\text{O}$$

### 典型实验 / 例题
**实验：大理石（CaCO₃）与盐酸反应，检验 CO₂：**

现象：大理石溶解，冒气泡，气泡通入澄清石灰水，石灰水变浑浊。

**例题**：盐酸有多少种化学性质？各举一例方程式。
答：与金属反应、与碱反应（中和）、与碱性氧化物反应、与碳酸盐反应（4大酸的通性）

### 易错点
- 盐酸的溶质是 **HCl（气体）**，不是 HCl 的水溶液
- 浓盐酸"冒烟"是 HCl **气体** 挥发，不是燃烧
- 盐酸与铁反应生成 **FeCl₂**（+2 价），不是 FeCl₃
- 盐酸是**无色**的（不是绿色），CuCl₂ 溶液才是蓝绿色
