# 稀硫酸性质

> **一句话**：稀硫酸是硫酸的稀溶液，具有酸的通性，能与活泼金属、碱性氧化物、碱和某些盐发生反应，但与铜不反应（不同于浓硫酸）。
> **English**: Dilute sulfuric acid is a dilute solution of H₂SO₄ with typical acid properties: it reacts with active metals, basic oxides, bases, and certain salts, but not with copper.

---

## 📚 给 6-12 岁（自读版）

稀硫酸就是把浓硫酸加水稀释后得到的，没有了浓硫酸那么危险的特殊性质（脱水、强腐蚀），但仍然是酸，有酸的共同性质。

**稀硫酸的特征：**
- 无色透明液体（稀的）
- 酸味（不能尝！）
- 能使石蕊变红
- 能和活泼金属反应，放出氢气

**稀硫酸与铁的反应：**
$$\text{Fe} + \text{H}_2\text{SO}_4(\text{稀}) \rightarrow \text{FeSO}_4 + \text{H}_2\uparrow$$
铁片溶解，溶液变成浅绿色（FeSO₄），同时有气泡（H₂）

**稀硫酸的用途：**
- 铅酸蓄电池（汽车电池内的电解液）
- 工业生产各种盐（如 ZnSO₄、FeSO₄）
- 实验室制备 H₂（锌粒+稀硫酸）

## 🎓 给 12+（进阶版）

### 定义
稀硫酸指 H₂SO₄ 质量分数较低的溶液，不具备浓硫酸的强氧化性、脱水性和吸水性，只表现出酸的通性。

### 稀硫酸的四大通性反应

**1. 与活泼金属反应（放出 H₂）：**

$$\text{Zn} + \text{H}_2\text{SO}_4 \rightarrow \text{ZnSO}_4 + \text{H}_2\uparrow$$

$$\text{Fe} + \text{H}_2\text{SO}_4 \rightarrow \text{FeSO}_4 + \text{H}_2\uparrow$$

$$2\text{Al} + 3\text{H}_2\text{SO}_4 \rightarrow \text{Al}_2(\text{SO}_4)_3 + 3\text{H}_2\uparrow$$

（注：Cu 不与稀 H₂SO₄ 反应，但可与热浓 H₂SO₄ 反应）

**2. 与碱性氧化物反应：**

$$\text{H}_2\text{SO}_4 + \text{CuO} \rightarrow \text{CuSO}_4 + \text{H}_2\text{O}$$

$$3\text{H}_2\text{SO}_4 + \text{Fe}_2\text{O}_3 \rightarrow \text{Fe}_2(\text{SO}_4)_3 + 3\text{H}_2\text{O}$$

**3. 与碱反应（中和反应）：**

$$\text{H}_2\text{SO}_4 + 2\text{NaOH} \rightarrow \text{Na}_2\text{SO}_4 + 2\text{H}_2\text{O}$$

$$\text{H}_2\text{SO}_4 + \text{Ba(OH)}_2 \rightarrow \text{BaSO}_4\downarrow + 2\text{H}_2\text{O}$$

**4. 与某些盐反应：**

$$\text{H}_2\text{SO}_4 + \text{Na}_2\text{CO}_3 \rightarrow \text{Na}_2\text{SO}_4 + \text{H}_2\text{O} + \text{CO}_2\uparrow$$

$$\text{H}_2\text{SO}_4 + \text{BaCl}_2 \rightarrow \text{BaSO}_4\downarrow + 2\text{HCl}$$

### 典型实验 / 例题
**例题**：下列不属于稀硫酸化学性质的是（）
A. 与锌反应放出 H₂　B. 使石蕊溶液变红　C. 与铜反应放出 H₂　D. 与氢氧化钠中和

答案：C（稀 H₂SO₄ 不与 Cu 反应，只有热浓 H₂SO₄ 才与 Cu 反应）

### 易错点
- 稀硫酸与铁反应生成 **FeSO₄**（Fe²⁺，浅绿色），不是 Fe₂(SO₄)₃
- 稀硫酸与铜**不反应**（铜在活动性顺序中排在氢后面）
- Ba(OH)₂ 与 H₂SO₄ 反应生成 BaSO₄ 沉淀和水（特殊现象：溶液变浑浊）
