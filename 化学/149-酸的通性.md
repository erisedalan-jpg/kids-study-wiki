# 酸的通性

> **一句话**：酸的通性是所有酸共有的化学性质，包括使指示剂变色、与活泼金属反应、与碱性氧化物反应、与碱发生中和反应、与某些盐反应五大类。
> **English**: The common properties of acids include changing indicator colors, reacting with active metals, basic oxides, bases (neutralization), and certain salts.

---

## 📚 给 6-12 岁（自读版）

不管是盐酸还是硫酸，所有的酸都有几个共同的"脾气"，这就是**酸的通性**。

**酸的五大通性：**

1. **使指示剂变色**
   - 蓝色石蕊→变红（石蕊遇酸变红）
   - 酚酞不变色（酸不使酚酞变色）

2. **和活泼金属反应，放出氢气（H₂）**
   - 铁 + 盐酸 → 氯化亚铁 + 氢气（气泡）
   - 锌 + 稀硫酸 → 硫酸锌 + 氢气

3. **和碱性氧化物反应，生成盐和水**
   - 盐酸 + 氧化铁（铁锈）→ 氯化铁 + 水
   - 这就是用盐酸除铁锈的原理！

4. **和碱发生中和反应，生成盐和水**
   - 盐酸 + 氢氧化钠 → 氯化钠（食盐）+ 水
   - 中和是酸和碱的"消消乐"

5. **和某些盐反应**
   - 盐酸 + 碳酸钠 → 氯化钠 + 水 + 二氧化碳（气泡）

## 🎓 给 12+（进阶版）

### 定义
酸的通性是酸溶液中 H⁺ 的化学性质，是所有酸共有的。

### 五大通性及方程式

**1. 使酸碱指示剂变色**
- 蓝色石蕊→红色
- 酚酞不变色（酸中酚酞无色）

**2. 与金属（活动性在 H 之前）反应，产生 H₂：**

$$\text{Mg} + \text{H}_2\text{SO}_4 \rightarrow \text{MgSO}_4 + \text{H}_2\uparrow$$

$$2\text{Al} + 6\text{HCl} \rightarrow 2\text{AlCl}_3 + 3\text{H}_2\uparrow$$

**3. 与碱性氧化物（金属氧化物）反应，生成盐和水：**

$$\text{H}_2\text{SO}_4 + \text{CuO} \rightarrow \text{CuSO}_4 + \text{H}_2\text{O}$$

$$6\text{HCl} + \text{Fe}_2\text{O}_3 \rightarrow 2\text{FeCl}_3 + 3\text{H}_2\text{O}$$

**4. 与碱反应（中和），生成盐和水：**

$$\text{HCl} + \text{NaOH} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$

$$\text{H}_2\text{SO}_4 + 2\text{KOH} \rightarrow \text{K}_2\text{SO}_4 + 2\text{H}_2\text{O}$$

**5. 与某些盐反应（强酸制弱酸）：**

$$2\text{HCl} + \text{Na}_2\text{CO}_3 \rightarrow 2\text{NaCl} + \text{H}_2\text{O} + \text{CO}_2\uparrow$$

$$\text{H}_2\text{SO}_4 + \text{BaCl}_2 \rightarrow \text{BaSO}_4\downarrow + 2\text{HCl}$$

### 典型实验 / 例题
**例题**：盐酸与下列物质不能发生反应的是（）
A. Zn（锌）　B. Cu(OH)₂（氢氧化铜）　C. NaCl（氯化钠）　D. Fe₂O₃（氧化铁）

答案：C（NaCl 是强酸强碱盐，不与盐酸反应；其余均符合酸的通性）

### 易错点
- 铜（Cu）在活动性顺序 H 后面，**不与稀酸反应**（通性不适用）
- 酸与碱性氧化物反应，是酸的**第三条**通性，不要和金属反应混淆
- 酸的通性本质上是 **H⁺ 的性质**
