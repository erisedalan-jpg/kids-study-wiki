# 化合价变化

> **一句话**：氧化还原反应的特征标志，元素化合价的升高或降低反映电子的转移方向。
> **English**: The hallmark of a redox reaction; a rise or fall in oxidation state reflects the direction of electron transfer.

---

## 🎓 高中进阶

### 定义

化合价（oxidation state）是表示元素在化合物中结合能力的数值。在氧化还原反应中，**化合价的升降**是判断氧化还原的核心依据：

- 化合价**升高** → 该物质**失电子** → **被氧化** → 该物质是**还原剂**
- 化合价**降低** → 该物质**得电子** → **被还原** → 该物质是**氧化剂**

### 化合价规则（高中）

| 规则 | 内容 |
|------|------|
| 单质 | 化合价为**0** |
| 氢 | 通常为+1（金属氢化物中为-1，如NaH）|
| 氧 | 通常为-2（过氧化物中为-1，如H₂O₂；OF₂中为+2）|
| 代数和 | 化合物中各元素化合价代数和为**0** |
| 离子 | 单原子离子化合价等于离子电荷数 |
| 变价元素 | Fe（+2/+3）、Mn（+2/+4/+7）、S（-2/0/+4/+6）、N（-3/0/+2/+3/+4/+5）|

### 常见变价元素化合价

| 元素 | 常见化合价 | 示例化合物 |
|------|-----------|-----------|
| Fe | +2, +3 | FeO, Fe₂O₃ |
| Mn | +2, +4, +7 | MnO, MnO₂, KMnO₄ |
| S | -2, 0, +4, +6 | H₂S, S, SO₂, H₂SO₄ |
| N | -3, 0, +2, +4, +5 | NH₃, N₂, NO, NO₂, HNO₃ |
| Cl | -1, 0, +1, +5, +7 | HCl, Cl₂, HClO, HClO₃, HClO₄ |
| C | -4, 0, +2, +4 | CH₄, C, CO, CO₂ |

### 化合价计算示例

$\text{Na}_2\text{SO}_4$中S的化合价：
$$2(+1) + x + 4(-2) = 0 \Rightarrow x = +6$$

$\text{KMnO}_4$中Mn的化合价：
$$(+1) + x + 4(-2) = 0 \Rightarrow x = +7$$

$\text{Fe}_3\text{O}_4$中Fe的平均化合价：
$$3x + 4(-2) = 0 \Rightarrow x = +\frac{8}{3}$$（实际含Fe²⁺和Fe³⁺）

### 典型例题

**例1**：在 $\text{K}_2\text{Cr}_2\text{O}_7 + 14\text{HCl} \rightarrow 2\text{KCl} + 2\text{CrCl}_3 + 3\text{Cl}_2\uparrow + 7\text{H}_2\text{O}$ 中，标出各元素化合价变化并判断氧化剂和还原剂。

- Cr：K₂Cr₂O₇中+6 → CrCl₃中+3（降低）→ K₂Cr₂O₇是**氧化剂**
- Cl（HCl中）：-1 → Cl₂中0（升高，部分）→ HCl是**还原剂**

### 易错点 / 拓展

- 化合价用"+""-"表示，离子电荷用"²⁺"等表示，书写方式不同
- $\text{Fe}_3\text{O}_4$中铁的平均化合价是+8/3，但实际是+2和+3共存
- NH₄⁺中N化合价：$x + 4(+1) = +1 \Rightarrow x = -3$（NH₄⁺整体带+1）
- H₂O₂中O为-1，不是-2，常考易错
