# 化合物

> **一句话**：由两种或两种以上元素组成的纯净物，各元素以化学键结合，有固定的化学式。
> **English**: A pure substance formed from two or more elements chemically bonded together in a fixed ratio.

---

## 🎓 高中进阶

### 定义

化合物是由**不同种元素的原子**以化学键（离子键、共价键或金属键）结合而成的纯净物。化合物有固定的化学式，各元素质量比恒定，不能通过物理方法分离。

### 化合物分类

| 分类角度 | 类别 | 示例 |
|---------|------|------|
| 含氧与否 | 氧化物 / 非氧化物 | H₂O / HCl |
| 酸碱性 | 酸 / 碱 / 盐 | HCl / NaOH / NaCl |
| 键型 | 离子化合物 / 共价化合物 | NaCl / HCl |
| 有机无机 | 有机物 / 无机物 | C₆H₁₂O₆ / NaCl |

### 离子化合物 vs 共价化合物

| 特征 | 离子化合物 | 共价化合物 |
|------|-----------|-----------|
| 键型 | 离子键（+金属键或共价键）| 共价键 |
| 典型示例 | NaCl、NaOH、Na₂O₂ | HCl、H₂O、CO₂ |
| 熔沸点 | 较高 | 一般较低 |
| 溶于水 | 多数电离 | 部分电离或不电离 |

### 典型例题

**例1**：下列属于化合物的是（ ）
A. 液氧　B. 干冰　C. 铁　D. 铜合金

**分析**：液氧是O₂（单质）；干冰是CO₂（化合物，含C和O）；铁是Fe单质；铜合金是混合物。**答案：B**

**例2**：NaOH属于哪类化合物？

**分析**：NaOH含Na⁺和OH⁻，以离子键结合，属于**离子化合物**，同时也是**碱**。

### 易错点 / 拓展

- H₂O₂（过氧化氢）是化合物，不是氧化物（氧化物中氧元素化合价为-2，H₂O₂中O为-1）
- 化合物分解可得单质或其他化合物
- 有机化合物（含C-H键）是化合物的重要分支
- 离子化合物一定含离子键；共价化合物一定不含离子键
