# 双线桥法

> **一句话**：用两条弧线分别标注氧化剂→还原产物（得电子）和还原剂→氧化产物（失电子）的分析方法。
> **English**: A method using two arcs to show electron transfer: one from oxidizing agent to reduction product, another from reducing agent to oxidation product.

---

## 🎓 高中进阶

### 定义

双线桥法是分析氧化还原反应的标准方法：
- **第一条线**：从**氧化剂**（反应物）→ **还原产物**（生成物），标"得X个电子"
- **第二条线**：从**还原剂**（反应物）→ **氧化产物**（生成物），标"失X个电子"

两线均弯向反应箭头方向，形状如"桥"。

### 与单线桥的区别

| 方法 | 箭头方向 | 标注内容 | 特点 |
|------|---------|---------|------|
| 单线桥 | 从还原剂→氧化剂 | 转移电子数 | 简洁，表示方向 |
| 双线桥 | 两条，各自指向产物 | 得/失电子数 | 详细，强调产物 |

高中主要考查**双线桥法**。

### 书写步骤

1. 写出配平的化学方程式
2. 标出化合价变化的元素的化合价（反应物和产物）
3. 画出第一条线：氧化剂（反应物）→ 还原产物，标"得电子数"
4. 画出第二条线：还原剂（反应物）→ 氧化产物，标"失电子数"
5. 验证：得电子总数 = 失电子总数（电子守恒）

### 示例

**反应：$\text{Fe} + 2\text{HCl} \rightarrow \text{FeCl}_2 + \text{H}_2\uparrow$**

```
       失2e⁻（被氧化）
Fe ─────────────────→ FeCl₂（中Fe²⁺）
  ⁰                      ⁺²

2HCl ──────────────→ H₂↑
  ⁻¹                   ⁰
       得2e⁻（被还原）
```

- Fe（0→+2）：失2个电子/每个Fe原子，还原剂，生成FeCl₂（氧化产物）
- 2HCl（H：+1→0，2个H各得1e⁻）：共得2e⁻，HCl是氧化剂，生成H₂（还原产物）

**反应：$\text{MnO}_2 + 4\text{HCl(浓)} \rightarrow \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$**

```
         得2e⁻（被还原）
MnO₂ ───────────────→ MnCl₂
 Mn⁺⁴                   Mn⁺²

2HCl（还原）────────→ Cl₂↑
  Cl⁻¹                  Cl⁰
         失2e⁻（被氧化）
```

注意：只有两个HCl（其中Cl被氧化），另两个HCl提供Cl⁻与Mn结合，不参与氧化还原。

### 典型例题

**例1**：用双线桥法分析 $\text{Cl}_2 + 2\text{NaOH} \rightarrow \text{NaCl} + \text{NaClO} + \text{H}_2\text{O}$

Cl₂（0）→ NaCl中Cl⁻（-1，降低1）：得电子（氧化剂 → 还原产物）
Cl₂（0）→ NaClO中Cl⁺¹（+1，升高1）：失电子（还原剂 → 氧化产物）

这是**歧化反应**，Cl₂既是氧化剂又是还原剂。

### 易错点 / 拓展

- 双线桥中**两条线的得失电子数必须相等**
- 注意原子系数：若方程式中系数>1，要乘以系数计算总电子数
- 不参与氧化还原的离子（如旁观离子）不画线桥
- 双线桥是标准考试书写格式，单线桥较简洁但表达信息少
