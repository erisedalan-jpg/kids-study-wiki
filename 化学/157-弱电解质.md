# 弱电解质

> **一句话**：在水溶液中只能部分电离的电解质，溶液中离子和未电离分子共存。
> **English**: An electrolyte that only partially ionizes in aqueous solution, with both ions and undissociated molecules coexisting.

---

## 🎓 高中进阶

### 定义

弱电解质是指溶于水时**部分电离**（电离度 < 100%）的电解质。电离是可逆过程，存在电离平衡，溶液中同时存在未电离的分子和水合离子。电离方程式用"⇌"（可逆箭头）表示。

### 常见弱电解质分类

| 类别 | 举例 |
|------|------|
| 弱酸 | CH₃COOH（醋酸）、HF、H₂CO₃、H₂S、HClO、HNO₂ |
| 弱碱 | NH₃·H₂O（氨水）、Mg(OH)₂、Al(OH)₃、Fe(OH)₃ |
| 水 | H₂O（极弱电解质）|

### 弱酸电离方程式

$$\text{CH}_3\text{COOH} \rightleftharpoons \text{CH}_3\text{COO}^- + \text{H}^+$$
$$\text{H}_2\text{CO}_3 \rightleftharpoons \text{H}^+ + \text{HCO}_3^-\text{（第一步电离）}$$
$$\text{HCO}_3^- \rightleftharpoons \text{H}^+ + \text{CO}_3^{2-}\text{（第二步电离，更弱）}$$

多元弱酸分步电离，且各步电离程度逐步减小。

### 弱电解质的电离平衡

以醋酸为例：
$$K_a = \frac{[\text{CH}_3\text{COO}^-][\text{H}^+]}{[\text{CH}_3\text{COOH}]} = 1.75 \times 10^{-5}$$

$K_a$ 越小，酸性越弱（高中通常不要求计算，了解概念）。

### 强弱电解质对比

| 特征 | 强电解质 | 弱电解质 |
|------|---------|---------|
| 电离程度 | 100% | < 100% |
| 溶液中粒子 | 只有离子 | 离子 + 分子 |
| 电离可逆性 | 不可逆 | 可逆 |
| 电离方程式 | → | ⇌ |
| 离子方程式中 | 必须拆开 | 保留分子式 |

### 典型例题

**例1**：下列属于弱电解质的是（ ）
A. HClO₄　B. HNO₃　C. CH₃COOH　D. HI

**分析**：HClO₄、HNO₃、HI是强酸（强电解质）；CH₃COOH（醋酸）只部分电离，是**弱电解质**。**答案：C**

**例2**：在0.1 mol/L醋酸溶液中，离子浓度大小关系？

**分析**：CH₃COOH部分电离，c(H⁺) = c(CH₃COO⁻) < c(CH₃COOH)
加上水的自电离：c(H⁺) > c(OH⁻)
完整顺序：$$c(\text{CH}_3\text{COOH}) > c(\text{H}^+) = c(\text{CH}_3\text{COO}^-) > c(\text{OH}^-)$$

### 易错点 / 拓展

- **HF**是弱酸（弱电解质），不要因为腐蚀性强而误判为强酸
- H₂O是弱电解质：H₂O ⇌ H⁺ + OH⁻，$K_w = 10^{-14}$（25°C）
- 弱酸盐（如NaHCO₃）是**强电解质**（盐是强电解质），不要与弱酸混淆
- 稀释弱电解质溶液，电离程度**增大**（平衡右移）
