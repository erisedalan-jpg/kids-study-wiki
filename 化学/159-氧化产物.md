# 氧化产物

> **一句话**：氧化还原反应中，还原剂被氧化后生成的产物（化合价升高的一侧产物）。
> **English**: The product formed from the reducing agent after it is oxidized in a redox reaction.

---

## 🎓 高中进阶

### 定义

氧化产物是指氧化还原反应中，**还原剂**（失去电子、化合价升高）转变成的产物。

对应关系：
- **还原剂** → 被氧化 → 生成**氧化产物**
- **氧化剂** → 被还原 → 生成**还原产物**

### 对应关系表

| 反应物（角色）| 变化 | 产物（类型）|
|------------|------|------------|
| 还原剂 | 失电子，化合价升高 | 氧化产物 |
| 氧化剂 | 得电子，化合价降低 | 还原产物 |

### 典型例题分析

**例1**：$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$

| 物质 | 化合价变化 | 角色 | 产物 |
|------|-----------|------|------|
| Fe | 0 → +2（升高）| 还原剂 | FeSO₄（**氧化产物**）|
| CuSO₄ | +2 → 0（降低）| 氧化剂 | Cu（还原产物）|

**例2**：$2\text{Na} + \text{Cl}_2 \rightarrow 2\text{NaCl}$

Na（0→+1，升高）：还原剂 → 产物NaCl中的Na⁺ → **氧化产物**（即NaCl）
Cl₂（0→-1，降低）：氧化剂 → 产物NaCl中的Cl⁻ → 还原产物（即NaCl）

注意：此处同一产物NaCl既含氧化产物成分又含还原产物成分，通常指"还原剂被氧化后产物的来源"。

**例3**：$\text{MnO}_2 + 4\text{HCl(浓)} \rightarrow \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$

Cl（HCl中 -1 → Cl₂中 0，升高）：HCl是还原剂 → **Cl₂是氧化产物**

### 特殊情形：歧化反应

$\text{Cl}_2 + 2\text{NaOH} \rightarrow \text{NaCl} + \text{NaClO} + \text{H}_2\text{O}$

Cl₂（0）→ Cl⁻（-1，降低，生成NaCl = 还原产物）
Cl₂（0）→ ClO⁻（+1，升高，生成NaClO = **氧化产物**）

Cl₂既是氧化剂又是还原剂，同时生成两种产物。

### 易错点

- 氧化产物来自**还原剂**（不是氧化剂）
- 歧化反应中，氧化产物和还原产物可能都来自同一物质
- 化合价升高→被氧化→生成氧化产物（口诀：升失氧）
