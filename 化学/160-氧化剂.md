# 氧化剂

> **一句话**：在氧化还原反应中，得到电子（化合价降低）、使其他物质被氧化的物质。
> **English**: A substance that gains electrons (decreases in oxidation state) in a redox reaction, causing other substances to be oxidized.

---

## 🎓 高中进阶

### 定义

氧化剂（oxidizing agent）是氧化还原反应中**得到电子**的物质：
- 化合价**降低**
- 自身被**还原**（发生还原反应）
- 生成**还原产物**

口诀：**氧化剂 → 得电子 → 降化合价 → 被还原 → 生成还原产物**

### 常见氧化剂

| 氧化剂 | 化学式 | 常见反应 |
|-------|-------|---------|
| 氯气 | Cl₂ | 与H₂、Na、Fe等反应 |
| 高锰酸钾 | KMnO₄ | 实验室制Cl₂等 |
| 二氧化锰 | MnO₂ | 与HCl(浓)制Cl₂ |
| 浓硫酸 | H₂SO₄（浓）| 与Cu加热反应 |
| 浓硝酸/稀硝酸 | HNO₃ | 与Cu、Fe反应 |
| 氧气 | O₂ | 燃烧等大多数氧化反应 |
| 过氧化氢 | H₂O₂ | 温和氧化剂 |
| 铁离子 | Fe³⁺ | 与Cu、Fe等反应 |

### 氧化剂强弱比较

强氧化剂（高中常见）：KMnO₄ > Cl₂ > Br₂ > Fe³⁺ > I₂

判断依据：强氧化剂能将弱还原剂氧化；强氧化剂在反应中优先被还原。

### 氧化剂的变化分析

以 $2\text{Fe} + 3\text{Cl}_2 \rightarrow 2\text{FeCl}_3$ 为例：

$$\text{Cl}_2 \xrightarrow{+2e^-/\text{Cl原子}} 2\text{Cl}^-\quad\text{（Cl：0→-1，氧化剂被还原）}$$
$$\text{Fe} \xrightarrow{-3e^-} \text{Fe}^{3+}\quad\text{（Fe：0→+3，还原剂被氧化）}$$

### 典型例题

**例1**：$\text{MnO}_2 + 4\text{HCl(浓)} \xrightarrow{\Delta} \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$ 中，MnO₂是什么？

**分析**：Mn化合价：+4→+2（降低），MnO₂得到电子，是**氧化剂**。

**例2**：工业制硫酸中，$\text{SO}_2 + \frac{1}{2}\text{O}_2 \rightarrow \text{SO}_3$，氧化剂是什么？

**分析**：O₂中O：0→-2（降低，得电子）→ **O₂是氧化剂**；SO₂中S：+4→+6（升高）→ SO₂是还原剂。

### 易错点 / 拓展

- 氧化剂本身被**还原**（不是被氧化），生成还原产物
- 一种物质可以在不同反应中分别充当氧化剂或还原剂（如H₂O₂）
- **歧化反应**：同一物质中同种元素同时升价和降价，既是氧化剂又是还原剂（如Cl₂+NaOH→NaCl+NaClO）
- 氧化性强弱 ≠ 物质的腐蚀性（HF腐蚀性强但氧化性不强）
