# 氧化还原反应

> **一句话**：有电子转移（化合价升降）的化学反应。
> **English**: A chemical reaction involving the transfer of electrons, characterized by changes in oxidation states.

---

## 🎓 高中进阶

### 定义

氧化还原反应（redox reaction）是指**有电子转移**的反应，其特征是反应物中元素化合价**升高**（被氧化）和**降低**（被还原）同时发生。

两种理解角度：
- **化合价角度**：有元素化合价升降的反应
- **电子转移角度**：有电子得失（或共用电子对偏移）的反应

### 核心概念关系

| 概念 | 描述 | 口诀 |
|------|------|------|
| 氧化 | 化合价**升高**，**失**去电子 | 升失氧 |
| 还原 | 化合价**降低**，**得**到电子 | 降得还 |
| 氧化剂 | 得到电子，化合价降低，被还原 | 氧化剂被还原 |
| 还原剂 | 失去电子，化合价升高，被氧化 | 还原剂被氧化 |

**记忆口诀**：**升失氧、降得还；若说剂，正相反**
- 化合价升高 → 失电子 → 发生氧化反应 → 该物质是**还原剂**
- 化合价降低 → 得电子 → 发生还原反应 → 该物质是**氧化剂**

### 典型氧化还原反应

**铁与硫酸铜**（置换反应，均为氧化还原）：
$$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$$
Fe：0 → +2（升高，失电子，被氧化，是**还原剂**）
Cu：+2 → 0（降低，得电子，被还原，是**氧化剂**）

**氯气与氢气**：
$$\text{Cl}_2 + \text{H}_2 \rightarrow 2\text{HCl}$$

### 四种基本反应类型与氧化还原的关系

| 类型 | 是否一定是氧化还原 |
|------|----------------|
| 化合反应 | 不一定（如CO₂+H₂O→H₂CO₃不是；2H₂+O₂→2H₂O是）|
| 分解反应 | 不一定 |
| 置换反应 | **一定是**氧化还原反应 |
| 复分解反应 | **一定不是**氧化还原反应 |

### 典型例题

**例1**：判断 $2\text{Na} + \text{Cl}_2 \rightarrow 2\text{NaCl}$ 中的氧化剂和还原剂。

**分析**：Na：0 → +1（升高，失电子）→ 被氧化 → Na是**还原剂**
Cl₂：0 → -1（降低，得电子）→ 被还原 → Cl₂是**氧化剂**

**例2**：$\text{MnO}_2 + 4\text{HCl(浓)} \xrightarrow{\Delta} \text{MnCl}_2 + \text{Cl}_2\uparrow + 2\text{H}_2\text{O}$

Mn：+4 → +2（降低）→ MnO₂是**氧化剂**
Cl（HCl中）：-1 → 0（升高，部分）→ HCl是**还原剂**（只有两个HCl被氧化）

### 易错点 / 拓展

- 氧化还原反应中：氧化剂的化合价**降低**；还原剂的化合价**升高**（与直觉相反）
- 一个反应中，氧化剂与还原剂**可以是同一种物质**（歧化反应，如Cl₂+NaOH→NaCl+NaClO）
- 失去电子数 = 得到电子数（电子守恒）
- 不能仅凭"有氧参与"判断是否为氧化还原（CO+CuO→Cu+CO₂ 是；Na₂CO₃+H₂O+CO₂→2NaHCO₃ 不是）

|------|---------|
