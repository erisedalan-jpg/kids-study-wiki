# 离子反应

> **一句话**：在溶液中以离子为主要反应形式发生的化学反应，用离子方程式表示。
> **English**: A chemical reaction that occurs in solution primarily between ions, represented by ionic equations.

---

## 🎓 高中进阶

### 定义

离子反应是指在电解质溶液（或熔融体）中，**离子之间**发生的化学反应。其本质是反应前后离子种类或浓度发生了改变。用**离子方程式**表示。

### 离子反应发生的条件

离子反应本质上是为了减小离子浓度，驱动力来自：

| 条件 | 示例 |
|------|------|
| 生成难溶物（沉淀）| Ba²⁺ + SO₄²⁻ → BaSO₄↓ |
| 生成气体 | CO₃²⁻ + 2H⁺ → H₂O + CO₂↑ |
| 生成弱电解质（水）| H⁺ + OH⁻ → H₂O |
| 发生氧化还原反应 | Fe + 2H⁺ → Fe²⁺ + H₂↑ |

### 离子反应类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 复分解型 | 无电子转移 | 中和、沉淀反应 |
| 氧化还原型 | 有电子转移 | 铁与酸反应 |

### 典型离子反应方程式汇总

**中和（酸碱）**：
$$\text{H}^+ + \text{OH}^- = \text{H}_2\text{O}$$

**铁与稀盐酸**：
$$\text{Fe} + 2\text{H}^+ = \text{Fe}^{2+} + \text{H}_2\uparrow$$

**铁与氯化铁溶液**：
$$\text{Fe} + 2\text{Fe}^{3+} = 3\text{Fe}^{2+}$$

**AgNO₃与HCl**：
$$\text{Ag}^+ + \text{Cl}^- = \text{AgCl}\downarrow$$

**Na₂SO₄与BaCl₂**：
$$\text{Ba}^{2+} + \text{SO}_4^{2-} = \text{BaSO}_4\downarrow$$

**Na₂CO₃（足量）与CO₂**：
$$\text{CO}_2 + 2\text{OH}^- = \text{CO}_3^{2-} + \text{H}_2\text{O}$$

### 典型例题

**例1**：向Na₂CO₃溶液中逐滴加入盐酸，先加少量HCl，再加过量HCl，各阶段离子方程式？

**少量HCl（酸不足）**：
$$\text{CO}_3^{2-} + \text{H}^+ = \text{HCO}_3^-$$

**足量HCl（酸过量）**：
$$\text{HCO}_3^- + \text{H}^+ = \text{H}_2\text{O} + \text{CO}_2\uparrow$$

**例2**：区分Na₂SO₄和Na₂CO₃溶液，加BaCl₂和稀盐酸的现象？

**分析**：加BaCl₂均有白色沉淀；再加稀HCl，若沉淀溶解且产生气泡→原为Na₂CO₃（BaCO₃溶于HCl）；若沉淀不溶→原为Na₂SO₄（BaSO₄不溶于HCl）。

### 易错点 / 拓展

- 离子反应不一定在溶液中（熔融状态也可以，但高中主要讨论溶液中）
- 不是所有电解质溶液混合都发生离子反应（若无沉淀、气体、弱电解质生成，则不反应）
- 离子方程式两边**原子数**和**电荷数**都必须守恒
- 离子反应的本质：某些离子在溶液中浓度减小，平衡被破坏
