# 还原剂

> **一句话**：在氧化还原反应中，失去电子（化合价升高）、使其他物质被还原的物质。
> **English**: A substance that loses electrons (increases in oxidation state) in a redox reaction, causing other substances to be reduced.

---

## 🎓 高中进阶

### 定义

还原剂（reducing agent）是氧化还原反应中**失去电子**的物质：
- 化合价**升高**
- 自身被**氧化**（发生氧化反应）
- 生成**氧化产物**

口诀：**还原剂 → 失电子 → 升化合价 → 被氧化 → 生成氧化产物**

### 常见还原剂

| 还原剂 | 化学式 | 特点 |
|-------|-------|------|
| 氢气 | H₂ | 清洁还原剂 |
| 碳/一氧化碳 | C / CO | 工业冶金还原剂 |
| 铁 | Fe | 与酸、盐反应 |
| 铝 | Al | 铝热反应还原剂 |
| 亚铁离子 | Fe²⁺ | 溶液中还原剂 |
| 硫化氢 | H₂S | 较强还原性 |
| 碘化钾/碘离子 | KI / I⁻ | 检验氧化剂 |
| 浓盐酸（Cl⁻）| HCl(浓) | 与MnO₂等反应 |

### 还原性强弱比较

常见还原性顺序（高中参考）：
$$\text{Al > Zn > Fe > Ni > Sn > Pb > H}_2 > \text{Cu}$$
（由活动性顺序推断）

溶液中：I⁻ > Fe²⁺ > Br⁻ > Cl⁻

### 典型反应分析

$2\text{Na} + \text{Cl}_2 \rightarrow 2\text{NaCl}$：
- Na：0→+1（升高），Na是**还原剂**，被氧化，生成Na⁺（氧化产物）
- Cl₂：0→-1（降低），Cl₂是氧化剂

$\text{C} + 2\text{CuO} \xrightarrow{\text{高温}} 2\text{Cu} + \text{CO}_2\uparrow$：
- C：0→+4（升高），C是**还原剂**
- Cu：+2→0（降低），CuO是氧化剂

### 典型例题

**例1**：$\text{Fe} + \text{CuSO}_4 \rightarrow \text{FeSO}_4 + \text{Cu}$ 中，还原剂是什么？生成什么氧化产物？

**分析**：Fe：0→+2（升高，失电子），Fe是**还原剂**；生成FeSO₄，Fe²⁺是**氧化产物**。

**例2**：判断Fe和Fe²⁺、Fe³⁺的氧化还原性。

**分析**：Fe（单质，0价）→ 还原剂（失电子→Fe²⁺或Fe³⁺）
Fe²⁺（+2价）→ 既可作还原剂（→Fe³⁺）也可作氧化剂（→Fe）
Fe³⁺（+3价）→ 氧化剂（→Fe²⁺）

### 易错点 / 拓展

- 还原剂本身被**氧化**，不是被还原
- 还原性强的物质容易失电子，但不一定活泼（如CO₂中C的还原性弱）
- 判断还原剂：看化合价升高的那种物质
- 高中考查重点：还原剂→氧化产物配对，电子转移计算
