# 酸（高中）

> **一句话**：电离时产生的阳离子全部为氢离子（H⁺）的化合物。
> **English**: A compound that ionizes in solution to produce hydrogen ions (H⁺) as the only cation.

---

## 🎓 高中进阶

### 定义（阿伦尼乌斯定义）

酸是溶于水后**电离出的阳离子全部为H⁺**的化合物。高中阶段主要使用阿伦尼乌斯酸碱理论。

$$\text{HCl} \rightarrow \text{H}^+ + \text{Cl}^-$$
$$\text{H}_2\text{SO}_4 \rightarrow 2\text{H}^+ + \text{SO}_4^{2-}$$

### 酸的分类

| 分类依据 | 类别 | 示例 |
|---------|------|------|
| 含氧与否 | 含氧酸 / 无氧酸 | H₂SO₄、HNO₃ / HCl、HF |
| 强弱 | 强酸 / 弱酸 | HCl、H₂SO₄、HNO₃ / CH₃COOH、HF、H₂CO₃ |
| 元数 | 一元酸 / 二元酸 / 多元酸 | HCl / H₂SO₄ / H₃PO₄ |
| 挥发性 | 挥发酸 / 不挥发酸 | HCl / H₂SO₄（浓）|

### 常见强酸（高中必记）

| 酸 | 化学式 | 特点 |
|----|-------|------|
| 盐酸 | HCl | 挥发性，强酸 |
| 硫酸 | H₂SO₄ | 浓时强氧化性、吸水性 |
| 硝酸 | HNO₃ | 强氧化性，挥发性 |
| 氢溴酸 | HBr | 强酸 |
| 氢碘酸 | HI | 强酸，有还原性 |
| 高氯酸 | HClO₄ | 最强的含氧酸 |

### 酸的通性

1. 与酸碱指示剂反应：使紫色石蕊变红
2. 与活泼金属反应：Zn + H₂SO₄ → ZnSO₄ + H₂↑
3. 与碱中和：HCl + NaOH → NaCl + H₂O
4. 与碱性氧化物反应：3H₂SO₄ + Fe₂O₃ → Fe₂(SO₄)₃ + 3H₂O
5. 与某些盐反应：HCl + Na₂CO₃ → NaCl + NaHCO₃（少量酸）

### 典型例题

**例1**：下列属于强酸的是（ ）
A. CH₃COOH　B. HF　C. H₂SO₄　D. H₂CO₃

**分析**：CH₃COOH是弱酸；HF是弱酸（尽管腐蚀性强）；H₂SO₄是强酸（稀）；H₂CO₃是弱酸。**答案：C**

**例2**：写出盐酸与碳酸钙反应的方程式。

$$2\text{HCl} + \text{CaCO}_3 \rightarrow \text{CaCl}_2 + \text{H}_2\text{O} + \text{CO}_2\uparrow$$

### 易错点 / 拓展

- HF是**弱酸**，尽管有强腐蚀性（能腐蚀玻璃）
- H₂SO₄稀时为强酸，浓时具额外的强氧化性
- 酸性强弱≠氢离子数：H₃PO₄是三元酸但属**中强酸**
- 高中不要求布朗斯特-劳里酸碱理论
