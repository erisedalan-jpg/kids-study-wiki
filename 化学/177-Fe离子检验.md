# Fe离子检验

> **一句话**：用KSCN检验Fe³⁺（血红色），用NaOH或先加KSCN再加氧化剂检验Fe²⁺，两种方法各有特点。
> **English**: Fe³⁺ is detected with KSCN (blood-red color); Fe²⁺ is detected with NaOH or by first adding KSCN then an oxidizer.

---

## 🎓 高中进阶

### Fe³⁺的检验

**方法：加KSCN（硫氰酸钾）溶液**

| 步骤 | 操作 |
|------|------|
| 1 | 取少量待测液于试管 |
| 2 | 加几滴KSCN溶液 |
| 3 | 观察颜色 |
| 结论 | 变血红色 → 含Fe³⁺ |

**原理**：
$$\text{Fe}^{3+} + n\text{SCN}^- \rightarrow [\text{Fe(SCN)}_n]^{3-n}\quad(\text{血红色配合物})$$

注意：KSCN极灵敏，极少量Fe³⁺即可显色。

### Fe²⁺的检验

**方法一：NaOH沉淀法**

1. 加NaOH溶液：白色沉淀 → Fe(OH)₂（含Fe²⁺）
2. 在空气中放置，观察颜色变化：白色→灰绿→红褐（确认是Fe²⁺而非Cu²⁺）

$$\text{Fe}^{2+} + 2\text{OH}^- \rightarrow \text{Fe(OH)}_2\downarrow\quad(\text{白色})$$
$$4\text{Fe(OH)}_2 + \text{O}_2 + 2\text{H}_2\text{O} \rightarrow 4\text{Fe(OH)}_3\quad(\text{红褐色})$$

**方法二：KSCN + 氧化剂法（推荐）**

1. 加KSCN：无红色（说明无Fe³⁺）
2. 再加少量Cl₂水（或H₂O₂）：变红色 → 证明有Fe²⁺（Fe²⁺被氧化为Fe³⁺后与KSCN显色）

### 系统鉴别流程

```
待测溶液
↓ 加KSCN
变红色 → 含Fe³⁺（且可能含Fe²⁺）
不变色 → 不含Fe³⁺
   ↓ 加NaOH
   白色沉淀→变红褐 → 含Fe²⁺
   或加Cl₂水→KSCN变红 → 含Fe²⁺
```

### 同时含Fe²⁺和Fe³⁺的鉴别

溶液中可能同时有Fe²⁺和Fe³⁺：
- 加KSCN：变红色 → 有Fe³⁺
- 加NaOH：先白色沉淀后红褐 → 有Fe²⁺

### 典型例题

**例1**：取一瓶溶液，加入KSCN后变血红色，加入足量NaOH后，沉淀是什么颜色？

**分析**：溶液含Fe³⁺（KSCN变红），加足量NaOH → Fe³⁺ + 3OH⁻ → Fe(OH)₃↓（**红褐色**沉淀）。若同时含Fe²⁺，则也有白色Fe(OH)₂（迅速变红褐），总体呈红褐色。

**例2**：鉴别FeCl₂和FeCl₃溶液最快方法？

**分析**：各取少量，加入KSCN溶液。变血红色的是**FeCl₃**（含Fe³⁺）；无颜色变化的是FeCl₂（含Fe²⁺）。

### 易错点

- KSCN检验的是**Fe³⁺**，不能直接检验Fe²⁺
- Fe²⁺检验需先排除Fe³⁺干扰（KSCN不变红），再加氧化剂
- Cu²⁺与NaOH也生成沉淀（蓝色，不变色）→区别于Fe²⁺（白色→红褐）
- 检验Fe³⁺用KSCN（不用淀粉）；检验Cl₂用淀粉碘化钾试纸
