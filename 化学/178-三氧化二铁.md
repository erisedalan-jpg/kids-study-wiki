# 三氧化二铁

> **一句话**：红棕色固体，Fe³⁺的氧化物，俗称铁红，是铁锈的主要成分，与酸反应生成铁盐（Fe³⁺）。
> **English**: A reddish-brown solid, the oxide of Fe³⁺, commonly called iron red; the main component of rust; reacts with acids to form ferric salts.

### 基本信息

| 属性 | 数据 |

| 类型 | 碱性氧化物 |

### 化学性质

**与稀盐酸反应**：
$$\text{Fe}_2\text{O}_3 + 6\text{HCl} \rightarrow 2\text{FeCl}_3 + 3\text{H}_2\text{O}$$
$$\text{（离子方程式）：Fe}_2\text{O}_3 + 6\text{H}^+ \rightarrow 2\text{Fe}^{3+} + 3\text{H}_2\text{O}$$

（溶液变黄色，因Fe³⁺水溶液为黄色）

**与Al（铝热反应）**：
$$\text{Fe}_2\text{O}_3 + 2\text{Al} \xrightarrow{\text{高温}} \text{Al}_2\text{O}_3 + 2\text{Fe}$$

（铝热反应，高温，生成熔融铁，可用于焊接铁轨）

**被CO还原**（高炉冶铁）：
$$\text{Fe}_2\text{O}_3 + 3\text{CO} \xrightarrow{\text{高温}} 2\text{Fe} + 3\text{CO}_2$$

**被H₂还原**：
$$\text{Fe}_2\text{O}_3 + 3\text{H}_2 \xrightarrow{\text{高温}} 2\text{Fe} + 3\text{H}_2\text{O}$$

### 铁锈的成分

铁锈的主要成分是Fe₂O₃·nH₂O（含水氧化铁），俗称铁锈。红棕色。
纯Fe₂O₃不含结晶水，铁锈是含水的非化学计量组成的混合物。

### 典型例题

**例1**：铁红（Fe₂O₃）与稀H₂SO₄反应，溶液颜色变化？

**分析**：Fe₂O₃ + 3H₂SO₄ → Fe₂(SO₄)₃ + 3H₂O；Fe₂(SO₄)₃（铁盐，Fe³⁺）溶液呈**黄色**。

**例2**：铝热反应有什么工业意义？

**分析**：铝热反应产生极高温度（≈2000°C）和熔融铁，可用于**焊接铁轨**（野外施工便捷）。

### 易错点

- Fe₂O₃是红棕色（不要与黑色FeO和Fe₃O₄混淆）
- Fe₂O₃是碱性氧化物，与酸反应，但**不与NaOH溶液反应**（不是两性）
- Fe₂O₃与Al的铝热反应是氧化还原：Al（0→+3，还原剂）；Fe（+3→0，氧化剂）
- 铁锈（Fe₂O₃·nH₂O）是混合物，不同于纯Fe₂O₃
