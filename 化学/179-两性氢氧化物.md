# 两性氢氧化物

> **一句话**：既能与强酸又能与强碱反应的氢氧化物，高中代表是Al(OH)₃。
> **English**: A hydroxide that reacts with both strong acids and strong bases; Al(OH)₃ is the representative in high school.

---

## 🎓 高中进阶

### 定义

两性氢氧化物是既能与酸反应（作为碱）又能与强碱反应（作为酸）的氢氧化物。

$$\text{Al(OH)}_3 + 3\text{HCl} \rightarrow \text{AlCl}_3 + 3\text{H}_2\text{O}\quad\text{（作为碱）}$$
$$\text{Al(OH)}_3 + \text{NaOH} \rightarrow \text{NaAlO}_2 + 2\text{H}_2\text{O}\quad\text{（作为酸）}$$

### 高中两性氢氧化物

高中阶段主要学习**Al(OH)₃**（氢氧化铝），另有ZnO(OH)（氢氧化锌，了解）。

### 两性比较

| 类型 | 与酸反应 | 与碱反应 | 示例 |
|------|---------|---------|------|
| 强碱 | 中和 | ✗ | NaOH、Ca(OH)₂ |
| 弱碱 | 中和 | ✗ | NH₃·H₂O、Mg(OH)₂、Fe(OH)₃ |
| 两性氢氧化物 | ✓（作碱）| ✓（作酸）| **Al(OH)₃**、Zn(OH)₂ |

### Al(OH)₃的两性反应（离子方程式）

**与HCl（作碱）**：
$$\text{Al(OH)}_3 + 3\text{H}^+ \rightarrow \text{Al}^{3+} + 3\text{H}_2\text{O}$$

**与NaOH（作酸）**：
$$\text{Al(OH)}_3 + \text{OH}^- \rightarrow \text{AlO}_2^- + 2\text{H}_2\text{O}$$

产物NaAlO₂（偏铝酸钠），溶液呈碱性。

### NaOH用量与Al(OH)₃沉淀的关系

向AlCl₃溶液中逐滴加NaOH：

| NaOH用量 | 反应 | 沉淀量 |
|---------|------|-------|
| 少量（< 3n(AlCl₃)）| Al³⁺ + 3OH⁻ → Al(OH)₃↓ | 增多 |
| 恰好3倍 | 全部沉淀 | 最多 |
| 过量 | Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O | 减少 |
| 大量过量 | 沉淀全部溶解 | 0 |

### 典型例题

**例1**：向NaAlO₂溶液中通入过量CO₂，发生什么？

**分析**：CO₂与H₂O反应形成H₂CO₃，H₂CO₃酸性强于Al(OH)₃（两性中的碱性）：
NaAlO₂ + CO₂ + 2H₂O → Al(OH)₃↓ + NaHCO₃
（CO₂足量时→NaHCO₃而非Na₂CO₃，因H₂CO₃在Al(OH)₃析出后仍有余）
现象：白色沉淀生成。

**例2**：Al(OH)₃能溶于氨水吗？

**分析**：氨水（NH₃·H₂O）是弱碱，碱性不足以溶解Al(OH)₃，Al(OH)₃**不溶于氨水**（只溶于强碱）。

### 易错点

- Al(OH)₃溶于**NaOH等强碱**，不溶于弱碱（NH₃·H₂O）
- Al(OH)₃加热分解：2Al(OH)₃ → Al₂O₃ + 3H₂O
- Mg(OH)₂是弱碱，不是两性氢氧化物（不溶于NaOH）
- Fe(OH)₃是弱碱，不是两性氢氧化物
