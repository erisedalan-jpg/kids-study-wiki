# 亚铁离子

> **一句话**：Fe²⁺，浅绿色，具有还原性，可被氧化为Fe³⁺；与OH⁻生成白色Fe(OH)₂沉淀，与SCN⁻无红色。
> **English**: Fe²⁺, pale green in solution; has reducing ability and can be oxidized to Fe³⁺; forms white Fe(OH)₂ with OH⁻; no red color with SCN⁻.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 符号 | Fe²⁺ |
| Fe化合价 | +2 |
| 溶液颜色 | **浅绿色** |
| 氧化还原性 | 具有还原性（可被氧化为Fe³⁺）|

### Fe²⁺的来源

| 来源 | 方程式 |
|------|-------|
| Fe与稀盐酸/稀H₂SO₄ | Fe + 2H⁺ → Fe²⁺ + H₂↑ |
| FeO与稀酸 | FeO + 2H⁺ → Fe²⁺ + H₂O |
| Fe与FeCl₃（还原Fe³⁺）| Fe + 2Fe³⁺ → 3Fe²⁺ |
| Fe₃O₄与稀酸 | Fe₃O₄ + 8H⁺ → Fe²⁺ + 2Fe³⁺ + 4H₂O |

### Fe²⁺的化学性质

**还原性**（被氧化为Fe³⁺）：
$$2\text{Fe}^{2+} + \text{Cl}_2 \rightarrow 2\text{Fe}^{3+} + 2\text{Cl}^-$$
$$2\text{Fe}^{2+} + \text{H}_2\text{O}_2 + 2\text{H}^+ \rightarrow 2\text{Fe}^{3+} + 2\text{H}_2\text{O}$$

**与OH⁻生成沉淀**：
$$\text{Fe}^{2+} + 2\text{OH}^- \rightarrow \text{Fe(OH)}_2\downarrow\quad(\text{白色})$$

### 检验Fe²⁺

**方法一（KSCN法）**：
1. 加KSCN（SCN⁻）：无红色（无Fe³⁺）
2. 再加几滴氯水（或H₂O₂）：溶液变红（Fe²⁺→Fe³⁺→Fe(SCN)₃）

**方法二（NaOH法）**：
加NaOH：白色沉淀（Fe(OH)₂），在空气中迅速变红褐色（Fe(OH)₃）→证明含Fe²⁺

### Fe²⁺ vs Fe³⁺对比

| 特征 | Fe²⁺ | Fe³⁺ |
|------|------|------|
| 溶液颜色 | 浅绿色 | 黄色 |
| KSCN检验 | 无红色 | **红色**（显著）|
| NaOH沉淀 | 白色Fe(OH)₂ | 红褐色Fe(OH)₃ |
| 氧化还原性 | 还原性（→Fe³⁺）| 氧化性（→Fe²⁺）|

### 典型例题

**例1**：向FeCl₂溶液中加少量氯水，发生什么？

$$2\text{Fe}^{2+} + \text{Cl}_2 \rightarrow 2\text{Fe}^{3+} + 2\text{Cl}^-$$
溶液颜色从**浅绿色**变为**黄色**（Fe²⁺→Fe³⁺）；Cl₂将Fe²⁺氧化（Cl₂是氧化剂）。

**例2**：如何除去FeCl₃溶液中混有的FeCl₂？

**分析**：加入适量Cl₂（或H₂O₂）：2FeCl₂ + Cl₂ → 2FeCl₃；即将Fe²⁺全部氧化为Fe³⁺。

### 易错点

- Fe²⁺是还原剂（失电子→Fe³⁺），Fe³⁺是氧化剂（得电子→Fe²⁺）
- KSCN不能直接检验Fe²⁺（需先氧化为Fe³⁺再检验）
- 亚铁盐溶液呈浅绿色，不是无色
