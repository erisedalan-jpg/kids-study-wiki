# 氢氧化亚铁

> **一句话**：白色固体，Fe²⁺对应的碱，极不稳定，在空气中迅速被氧化为红褐色Fe(OH)₃。
> **English**: A white solid, the base corresponding to Fe²⁺; highly unstable in air, quickly oxidized to reddish-brown Fe(OH)₃.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | Fe(OH)₂ |
| Fe化合价 | +2 |
| 颜色 | **白色**沉淀 |
| 稳定性 | 极不稳定（空气中迅速变色）|

### 制备

向FeCl₂（或FeSO₄）溶液中加入NaOH溶液：
$$\text{FeCl}_2 + 2\text{NaOH} \rightarrow \text{Fe(OH)}_2\downarrow + 2\text{NaCl}$$
$$\text{（离子方程式）：Fe}^{2+} + 2\text{OH}^- \rightarrow \text{Fe(OH)}_2\downarrow$$

### 颜色变化（经典实验）

Fe(OH)₂在空气中的颜色变化：
$$\text{Fe(OH)}_2 \xrightarrow{\text{O}_2，H}_\text{白色} \rightarrow \text{灰绿色} \rightarrow \underset{\text{红褐色}}{\text{Fe(OH)}_3}$$

氧化反应：
$$4\text{Fe(OH)}_2 + \text{O}_2 + 2\text{H}_2\text{O} \rightarrow 4\text{Fe(OH)}_3$$

### 化学性质

**与稀盐酸**（溶解，弱碱）：
$$\text{Fe(OH)}_2 + 2\text{HCl} \rightarrow \text{FeCl}_2 + 2\text{H}_2\text{O}$$

**加热分解**：
$$\text{Fe(OH)}_2 \xrightarrow{\Delta} \text{FeO} + \text{H}_2\text{O}$$（在无空气条件下）

### 实验：保持白色沉淀

实验中制备Fe(OH)₂时，为防止被氧化变绿变红：
1. 用**煮沸的蒸馏水**配FeCl₂溶液（减少溶解O₂）
2. NaOH溶液也用煮沸蒸馏水配制
3. 滴管伸入液面下加入NaOH（减少与空气接触）
4. 实验在N₂保护下进行

### 典型例题

**例1**：向FeSO₄溶液中加NaOH，观察到白色沉淀，静置后变红褐色，原因？

**分析**：首先生成白色Fe(OH)₂沉淀；在空气中Fe(OH)₂被O₂氧化：
4Fe(OH)₂ + O₂ + 2H₂O → 4Fe(OH)₃（红褐色）

**例2**：如何检验溶液中存在Fe²⁺（而非Fe³⁺）？

**分析**：向溶液中加NaOH：白色沉淀（Fe(OH)₂，迅速变红褐）→含Fe²⁺；或加KSCN无红色，再加氯水后变红（Fe²⁺被氧化为Fe³⁺）→含Fe²⁺。

### 易错点

- Fe(OH)₂是**白色**沉淀（Fe(OH)₃是红褐色）
- Fe(OH)₂在空气中**迅速**变色，实验现象为：白色→灰绿→红褐
- Fe(OH)₂是弱碱（不溶于水，溶于稀盐酸和稀硫酸）
