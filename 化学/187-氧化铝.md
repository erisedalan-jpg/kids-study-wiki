# 氧化铝

> **一句话**：Al的两性氧化物，既能与酸又能与强碱反应，是刚玉和宝石的主要成分，熔点极高。
> **English**: The amphoteric oxide of aluminum; reacts with both acids and strong bases; the main component of corundum and gemstones; has a very high melting point.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | Al₂O₃ |
| Al化合价 | +3 |
| 颜色 | 白色固体 |
| 熔点 | 2054°C（极高）|
| 硬度 | 极高（9级，仅次于金刚石）|
| 类型 | **两性氧化物** |
| 天然矿物 | 刚玉（刚玉宝石：红宝石/蓝宝石是含杂质Al₂O₃）|

### 两性氧化物的性质

**与酸反应**：
$$\text{Al}_2\text{O}_3 + 6\text{HCl} \rightarrow 2\text{AlCl}_3 + 3\text{H}_2\text{O}$$
$$\text{（离子方程式）：Al}_2\text{O}_3 + 6\text{H}^+ \rightarrow 2\text{Al}^{3+} + 3\text{H}_2\text{O}$$

**与NaOH溶液反应**（典型两性性质）：
$$\text{Al}_2\text{O}_3 + 2\text{NaOH} \rightarrow 2\text{NaAlO}_2 + \text{H}_2\text{O}$$
$$\text{（离子方程式）：Al}_2\text{O}_3 + 2\text{OH}^- \rightarrow 2\text{AlO}_2^- + \text{H}_2\text{O}$$

### 与碱性氧化物的区别

| 特征 | 碱性氧化物（如CaO）| 两性氧化物（Al₂O₃）|
|------|-----------------|------------------|
| 与酸反应 | ✓ | ✓ |
| 与碱反应 | ✗ | ✓ |
| 示例 | Na₂O、CaO、Fe₂O₃ | Al₂O₃（高中唯一两性氧化物）|

### 工业应用

| 应用 | 原理 |
|------|------|
| 刚玉磨料 | 极高硬度和熔点 |
| 耐火材料（坩埚）| 熔点高 |
| 电解制铝原料 | 电解Al₂O₃（加冰晶石）制Al |
| 宝石（红宝石/蓝宝石）| 含Cr²⁺（红）/ Ti⁴⁺（蓝）的Al₂O₃ |

### 典型例题

**例1**：Al₂O₃溶于NaOH的离子方程式？

$$\text{Al}_2\text{O}_3 + 2\text{OH}^- \rightarrow 2\text{AlO}_2^- + \text{H}_2\text{O}$$

产物NaAlO₂（偏铝酸钠），Al在其中化合价为+3。

**例2**：向含Al³⁺的溶液中逐滴加入NaOH，发生什么？

**分析**：
少量NaOH：Al³⁺ + 3OH⁻ → Al(OH)₃↓（白色沉淀出现）
足量NaOH：Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O（沉淀溶解）

### 易错点

- Al₂O₃是**两性氧化物**，高中阶段的代表，Fe₂O₃是碱性氧化物（不是两性）
- Al₂O₃与水不反应（不溶于水，不像Na₂O那样溶于水生成NaOH）
- 冰晶石（Na₃AlF₆）是电解Al₂O₃时加入降低熔点的助熔剂，本身不是原料
