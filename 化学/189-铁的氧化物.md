# 铁的氧化物

> **一句话**：铁有三种重要氧化物：FeO（氧化亚铁）、Fe₂O₃（氧化铁）和Fe₃O₄（四氧化三铁），性质各异。
> **English**: Iron has three important oxides: FeO, Fe₂O₃, and Fe₃O₄, each with distinct properties.

---

## 🎓 高中进阶

### 三种铁的氧化物对比

| 特征 | FeO | Fe₂O₃ | Fe₃O₄ |
|------|-----|-------|-------|
| 名称 | 氧化亚铁 | 氧化铁 | 四氧化三铁 |
| Fe化合价 | +2 | +3 | +2和+3混合（平均+8/3）|
| 颜色 | 黑色 | 红棕色 | 黑色 |
| 俗称 | — | 铁红 | 磁性氧化铁 |
| 磁性 | 无 | 无 | 有磁性 |
| 与酸反应 | 生成Fe²⁺ | 生成Fe³⁺ | 生成Fe²⁺和Fe³⁺ |

### 与酸的反应方程式

**FeO + 稀H₂SO₄**：
$$\text{FeO} + \text{H}_2\text{SO}_4 \rightarrow \text{FeSO}_4 + \text{H}_2\text{O}$$

**Fe₂O₃ + 稀H₂SO₄**：
$$\text{Fe}_2\text{O}_3 + 3\text{H}_2\text{SO}_4 \rightarrow \text{Fe}_2(\text{SO}_4)_3 + 3\text{H}_2\text{O}$$

**Fe₃O₄ + 稀H₂SO₄**：
$$\text{Fe}_3\text{O}_4 + 4\text{H}_2\text{SO}_4 \rightarrow \text{FeSO}_4 + \text{Fe}_2(\text{SO}_4)_3 + 4\text{H}_2\text{O}$$

离子方程式：
$$\text{Fe}_3\text{O}_4 + 8\text{H}^+ \rightarrow \text{Fe}^{2+} + 2\text{Fe}^{3+} + 4\text{H}_2\text{O}$$

### 铁被氧化程度与产物

| 条件 | 产物 |
|------|------|
| Fe在O₂中充分燃烧 | Fe₃O₄ |
| Fe在潮湿空气中慢速氧化（生锈）| Fe₂O₃·nH₂O（铁锈）|
| Fe与水蒸气在高温 | Fe₃O₄ + H₂ |

$$3\text{Fe} + 4\text{H}_2\text{O} \xrightarrow{\text{高温}} \text{Fe}_3\text{O}_4 + 4\text{H}_2\uparrow$$

### 典型例题

**例1**：铁在足量Cl₂中燃烧，生成什么氧化物？

**分析**：铁在Cl₂中生成FeCl₃（不是氧化物）：2Fe + 3Cl₂ → 2FeCl₃。题目应改为"铁在O₂中燃烧"→ Fe₃O₄。

**例2**：1 mol Fe₃O₄与足量稀盐酸反应，溶液中Fe²⁺和Fe³⁺的物质的量之比？

**分析**：Fe₃O₄中含1个Fe²⁺和2个Fe³⁺（结构上）；反应后得Fe²⁺:Fe³⁺ = **1:2**。

### 易错点

- Fe₃O₄不是Fe²⁺和Fe³⁺氧化物的简单混合，是特殊的磁性复合氧化物
- 铁锈不是纯Fe₂O₃，而是Fe₂O₃·nH₂O（含水氧化铁）
- Fe₂O₃俗称"铁红"，是红色颜料
- FeO不稳定，在空气中会被氧化为Fe₂O₃
