# 铁离子

> **一句话**：Fe³⁺，黄色溶液，具有氧化性，与KSCN反应显血红色（用于鉴定），可被还原为Fe²⁺。
> **English**: Fe³⁺, yellow in solution; has oxidizing ability; reacts with KSCN to give blood-red color (used for detection); can be reduced to Fe²⁺.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 符号 | Fe³⁺ |
| Fe化合价 | +3 |
| 溶液颜色 | **黄色** |
| 氧化还原性 | 具有氧化性（可被还原为Fe²⁺）|

### Fe³⁺的来源

| 来源 | 方程式 |
|------|-------|
| Fe与足量Cl₂ | 2Fe + 3Cl₂ → 2FeCl₃ |
| Fe₂O₃与稀酸 | Fe₂O₃ + 6H⁺ → 2Fe³⁺ + 3H₂O |
| FeCl₃溶液中 | FeCl₃ → Fe³⁺ + 3Cl⁻（强电解质）|
| Fe²⁺被氧化 | 2Fe²⁺ + Cl₂ → 2Fe³⁺ + 2Cl⁻ |

### Fe³⁺的化学性质

**氧化性**（被还原为Fe²⁺）：
$$2\text{Fe}^{3+} + \text{Fe} \rightarrow 3\text{Fe}^{2+}\quad(\text{Fe将Fe}^{3+}\text{还原})$$
$$2\text{Fe}^{3+} + \text{Cu} \rightarrow 2\text{Fe}^{2+} + \text{Cu}^{2+}\quad(\text{腐蚀铜电路板})$$
$$2\text{Fe}^{3+} + 2\text{I}^- \rightarrow 2\text{Fe}^{2+} + \text{I}_2$$

**与OH⁻**：
$$\text{Fe}^{3+} + 3\text{OH}^- \rightarrow \text{Fe(OH)}_3\downarrow\quad(\text{红褐色})$$

**与KSCN（检验Fe³⁺）**：
$$\text{Fe}^{3+} + n\text{SCN}^- \rightarrow [\text{Fe(SCN)}_n]^{3-n}\quad(\text{血红色})$$

### 检验Fe³⁺的方法

**最常用：KSCN法**：
向待测溶液中加入KSCN（硫氰酸钾）溶液，若变为**血红色**→证明含Fe³⁺。

（KSCN检验Fe³⁺是高中定性分析最重要的检验方法之一）

### FeCl₃的工业应用

**腐蚀铜电路板**（印刷电路板制造）：
$$2\text{FeCl}_3 + \text{Cu} \rightarrow 2\text{FeCl}_2 + \text{CuCl}_2$$
$$\text{（离子方程式）：}2\text{Fe}^{3+} + \text{Cu} \rightarrow 2\text{Fe}^{2+} + \text{Cu}^{2+}$$

（Fe³⁺将Cu氧化溶解，Cu²⁺溶于溶液，铜被腐蚀去掉，留下电路图案）

### 典型例题

**例1**：向FeCl₃溶液中加入适量铁粉，完全反应后，溶液中含哪些离子？

**分析**：两个反应发生：
- Fe + 2Fe³⁺ → 3Fe²⁺
若Fe过量：所有Fe³⁺被还原，最终溶液含Fe²⁺和Cl⁻（浅绿色）
若Fe不足：溶液含Fe²⁺、Fe³⁺和Cl⁻

**例2**：区别FeCl₂和FeCl₃溶液最简便的方法？

**分析**：加入KSCN溶液：变血红色→FeCl₃（含Fe³⁺）；无变化→FeCl₂（含Fe²⁺）。

### 易错点

- KSCN检验的是**Fe³⁺**（血红色），不是Fe²⁺（无颜色变化）
- Fe³⁺氧化性：能氧化Cu（不能氧化Ag）；能氧化I⁻（I₂从碘化钾溶液中置换出来）
- FeCl₃腐蚀铜电路板：Fe³⁺氧化Cu，不是Cl₂腐蚀
