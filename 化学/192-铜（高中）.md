# 铜（高中）

> **一句话**：第29号元素，紫红色金属，不与稀盐酸/稀H₂SO₄反应，但与浓HNO₃/稀HNO₃反应，还能被Fe³⁺溶液腐蚀。
> **English**: Element No.29, a reddish-purple metal; does not react with dilute HCl or H₂SO₄, but reacts with concentrated/dilute HNO₃; also corroded by Fe³⁺ solutions.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 元素符号 | Cu |
| 原子序数 | 29 |
| 化合价 | +1（氧化亚铜Cu₂O）或+2（氧化铜CuO）|
| 密度 | 8.96 g/cm³ |
| 熔点 | 1085°C |
| 颜色 | **紫红色**（铜红色）|
| 导电性 | 优良（仅次于银）|

### 铜的化学性质

**1. 与O₂反应**（加热）：
$$2\text{Cu} + \text{O}_2 \xrightarrow{\Delta} 2\text{CuO}\quad(\text{黑色})$$

（铜在空气中加热变黑，生成CuO）

**2. 与浓HNO₃反应**（常温）：
$$\text{Cu} + 4\text{HNO}_3(\text{浓}) \rightarrow \text{Cu(NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$
$$\text{（离子）：Cu} + 4\text{H}^+ + 2\text{NO}_3^- \rightarrow \text{Cu}^{2+} + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$

（生成红棕色NO₂气体）

**3. 与稀HNO₃反应**：
$$3\text{Cu} + 8\text{HNO}_3(\text{稀}) \rightarrow 3\text{Cu(NO}_3)_2 + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$$
$$\text{（离子）：}3\text{Cu} + 8\text{H}^+ + 2\text{NO}_3^- \rightarrow 3\text{Cu}^{2+} + 2\text{NO}\uparrow + 4\text{H}_2\text{O}$$

（生成无色NO气体，遇空气变红棕色）

**4. 与FeCl₃溶液反应**（重要）：
$$\text{Cu} + 2\text{FeCl}_3 \rightarrow \text{CuCl}_2 + 2\text{FeCl}_2$$
$$\text{（离子）：Cu} + 2\text{Fe}^{3+} \rightarrow \text{Cu}^{2+} + 2\text{Fe}^{2+}$$

（Cu被Fe³⁺腐蚀，用于蚀刻电路板）

**5. 与浓H₂SO₄（加热）**：
$$\text{Cu} + 2\text{H}_2\text{SO}_4(\text{浓}) \xrightarrow{\Delta} \text{CuSO}_4 + \text{SO}_2\uparrow + 2\text{H}_2\text{O}$$

**铜不与稀盐酸、稀H₂SO₄反应**（活动性顺序中Cu在H之后）。

### 铜的活动性位置

$$\text{Al > Zn > Fe > Ni > Sn > Pb > H}_2 > \text{Cu > Ag > Au}$$

Cu在H₂后：不与稀盐酸、稀H₂SO₄反应；但与浓HNO₃/稀HNO₃反应（HNO₃有强氧化性）。

### 典型例题

**例1**：Cu与足量稀HNO₃反应，现象？

**分析**：Cu溶解，溶液变蓝色（Cu²⁺），生成无色气体（NO），NO接触空气变红棕色（NO₂）。

**例2**：铜放入FeCl₃溶液中，铜溶解还是析出铁？

**分析**：Cu + 2Fe³⁺ → Cu²⁺ + 2Fe²⁺（Cu被氧化溶解，无铁析出，因Fe²⁺不能被Cu还原为Fe）。

### 易错点

- 铜不与**稀盐酸**、**稀H₂SO₄**反应（区别于活泼金属Zn、Fe）
- 铜与**浓硫酸**（加热）才反应，稀硫酸不与铜反应
- Cu²⁺溶液呈蓝色（不是绿色）
- 铜在潮湿空气中生成铜绿（Cu₂(OH)₂CO₃，碱式碳酸铜，绿色）
