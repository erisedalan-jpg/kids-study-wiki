# 铝热反应

> **一句话**：铝粉与金属氧化物在高温下发生的剧烈氧化还原反应，产生极高温度和熔融金属，用于焊接和冶金。
> **English**: A vigorous redox reaction between aluminum powder and a metal oxide at high temperature, producing extreme heat and molten metal; used in welding and metallurgy.

---

## 🎓 高中进阶

### 定义

铝热反应是指**铝粉**（Al）与**金属氧化物**（通常是Fe₂O₃）在高温（镁条点燃）下发生的剧烈放热的氧化还原反应，生成熔融金属（铁）和Al₂O₃。

### 反应方程式

**铝热反应（Fe₂O₃）**：
$$2\text{Al} + \text{Fe}_2\text{O}_3 \xrightarrow{\text{高温}} \text{Al}_2\text{O}_3 + 2\text{Fe}$$

$$\text{（用氯酸钾或镁条点燃引发）}$$

**铝热反应（其他金属氧化物）**：
$$8\text{Al} + 3\text{Fe}_3\text{O}_4 \xrightarrow{\text{高温}} 4\text{Al}_2\text{O}_3 + 9\text{Fe}$$
$$2\text{Al} + \text{Cr}_2\text{O}_3 \xrightarrow{\text{高温}} \text{Al}_2\text{O}_3 + 2\text{Cr}$$

### 氧化还原分析

以2Al + Fe₂O₃ → Al₂O₃ + 2Fe为例：

- Al：0→+3（升高，失电子）→ Al是**还原剂**，被氧化，生成Al₂O₃（氧化产物）
- Fe：+3→0（降低，得电子）→ Fe₂O₃是**氧化剂**，被还原，生成Fe（还原产物）

### 实验操作

1. 将Al粉和Fe₂O₃按2:1（摩尔比）混合，置于坩埚中
2. 上方放一小堆Na₂O₂（或KClO₃/MgO混合物），再放镁条
3. 点燃镁条引发高温（镁条燃烧温度高达3000°C以上）
4. 反应剧烈，产生耀眼白光；熔融铁（约2000°C）滴落

注意：实验中产生的Al₂O₃和Fe分层，铁沉底，Al₂O₃在上。

### 工业应用

| 应用 | 原理 |
|------|------|
| 焊接铁轨（野外）| 产生熔融铁，填充接合处 |
| 冶炼难熔金属（Cr、Mn）| 铝热法制备金属 |
| 铝热剂（军事）| 极高温度 |

### 典型例题

**例1**：铝热反应中，哪种物质是氧化剂？哪种是还原剂？

**分析**：Fe₂O₃中Fe：+3→0（降低），Fe₂O₃是**氧化剂**；Al：0→+3（升高），Al是**还原剂**。

**例2**：铝热反应属于哪种基本反应类型？

**分析**：2Al + Fe₂O₃ → Al₂O₃ + 2Fe，是单质置换化合物中的金属，属于**置换反应**，也是**氧化还原反应**。

### 易错点

- 铝热反应是Al（单质）将Fe₂O₃中的Fe还原为铁，不是Fe置换Al
- 铝热反应需要极高温度引发（镁条点燃），不能用酒精灯直接点燃
- 反应后产物中铁是**熔融状态**（不是固体粉末）
- Al的还原性比Fe强（活动性顺序：Al > Fe），才能发生铝热反应
