# 铝（高中）

> **一句话**：第13号元素，银白色轻金属，既能与酸又能与强碱反应（两性），表面氧化膜Al₂O₃使其具有良好耐腐蚀性。
> **English**: Element No.13, a silvery-white light metal; amphoteric — reacts with both acids and strong bases; the surface oxide film Al₂O₃ gives it good corrosion resistance.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 元素符号 | Al |
| 原子序数 | 13 |
| 化合价 | +3 |
| 密度 | 2.7 g/cm³（轻金属，密度约为铁的1/3）|
| 熔点 | 660°C |
| 颜色光泽 | 银白色 |

### Al的化学性质

**1. 与稀盐酸反应**：
$$2\text{Al} + 6\text{HCl} \rightarrow 2\text{AlCl}_3 + 3\text{H}_2\uparrow$$
$$\text{（离子方程式）：}2\text{Al} + 6\text{H}^+ \rightarrow 2\text{Al}^{3+} + 3\text{H}_2\uparrow$$

**2. 与NaOH溶液反应（特殊性质）**：
$$2\text{Al} + 2\text{NaOH} + 2\text{H}_2\text{O} \rightarrow 2\text{NaAlO}_2 + 3\text{H}_2\uparrow$$
$$\text{（离子方程式）：}2\text{Al} + 2\text{OH}^- + 2\text{H}_2\text{O} \rightarrow 2\text{AlO}_2^- + 3\text{H}_2\uparrow$$

（Al既溶于酸又溶于强碱，是两性金属）

**3. 与O₂反应（耐腐蚀原理）**：
$$4\text{Al} + 3\text{O}_2 \rightarrow 2\text{Al}_2\text{O}_3$$

常温下迅速生成致密Al₂O₃膜，阻止进一步氧化（钝化）。

**4. 铝热反应（与Fe₂O₃）**：
$$2\text{Al} + \text{Fe}_2\text{O}_3 \xrightarrow{\text{高温}} \text{Al}_2\text{O}_3 + 2\text{Fe}$$

（Al还原性强于Fe，铝热反应温度极高，生成熔融铁）

### 铝的冶炼

工业上通过**电解熔融Al₂O₃**制铝：
$$2\text{Al}_2\text{O}_3 \xrightarrow{\text{电解}} 4\text{Al} + 3\text{O}_2\uparrow$$

（需加冰晶石Na₃AlF₆降低熔点）

### 特殊：铝与浓HNO₃、浓H₂SO₄

铝在**冷**浓HNO₃或浓H₂SO₄中**钝化**（形成致密氧化膜），不发生明显化学反应。

这是为什么浓硫酸、浓硝酸可以用铝罐运输的原因。

### 典型例题

**例1**：向铝箔中加入NaOH溶液，观察到什么现象？

**分析**：铝与NaOH溶液反应：
2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑
现象：铝箔溶解，产生大量气泡（H₂）。

**例2**：等质量的铝分别与足量盐酸和NaOH溶液反应，产生H₂的物质的量相同吗？

**分析**：
与HCl：2Al + 6HCl → 2AlCl₃ + 3H₂↑（每2mol Al → 3mol H₂）
与NaOH：2Al + 2NaOH + 2H₂O → 2NaAlO₂ + 3H₂↑（每2mol Al → 3mol H₂）
两个反应产生H₂量**相同**（相同质量Al → 相同H₂）。

### 易错点

- Al既溶于酸又溶于强碱（两性金属），但不溶于弱碱
- 铝在冷浓HNO₃或浓H₂SO₄中钝化，不反应（但加热会反应）
- 铝表面氧化膜是Al₂O₃，使铝具有耐腐蚀性
