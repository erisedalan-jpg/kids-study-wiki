# 次氯酸

> **一句话**：弱酸，HClO，具有强氧化性和漂白性，是氯水中起漂白作用的主要成分。
> **English**: A weak acid (HClO) with strong oxidizing and bleaching properties; the main bleaching component in chlorine water.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | HClO |
| Cl化合价 | +1 |
| 酸碱性 | 弱酸（弱于H₂CO₃）|
| 颜色 | 无色 |
| 稳定性 | 极不稳定，易分解 |

### 生成方式

**1. Cl₂与水反应**（可逆）：
$$\text{Cl}_2 + \text{H}_2\text{O} \rightleftharpoons \text{HCl} + \text{HClO}$$

**2. Cl₂与碱反应生成次氯酸盐**（NaClO）：
$$\text{Cl}_2 + 2\text{NaOH} \rightarrow \text{NaCl} + \text{NaClO} + \text{H}_2\text{O}$$

**3. 次氯酸盐+酸→HClO**：
$$\text{NaClO} + \text{HCl} \rightarrow \text{NaCl} + \text{HClO}$$（注意：HCl强，HClO弱，强酸制弱酸）

### 化学性质

**1. 弱酸性**（弱于H₂CO₃）：
$$\text{HClO} \rightleftharpoons \text{H}^+ + \text{ClO}^-\quad(\text{部分电离})$$

**酸性强弱**：$\text{HCl} > \text{H}_2\text{CO}_3 > \text{HClO} > \text{HCO}_3^-$

**2. 强氧化性**（漂白性）：
HClO能氧化有机色素，破坏发色基团，使有色物质褪色。
$$\text{HClO} + \text{有色物质} \rightarrow \text{无色（氧化破坏）}$$

注意：这是**化学漂白**（不可逆），与活性炭的物理吸附漂白不同。

**3. 不稳定，见光分解**：
$$2\text{HClO} \xrightarrow{\text{光}} 2\text{HCl} + \text{O}_2\uparrow$$

### HClO的漂白原理

HClO（+1价Cl）具有强氧化性，能将有机色素氧化（破坏其π键/苯环等发色结构），使之失去颜色。漂白后的产物为无色，加热或加酸后颜色不能恢复（区别于SO₂的可逆漂白）。

### 酸性比较

| 酸 | 强弱 |
|----|------|
| HCl | 强酸 |
| H₂CO₃（CO₂+H₂O）| 弱酸（第一步）|
| HClO | 比H₂CO₃更弱 |
| HCO₃⁻（第二步电离）| 更弱 |

**应用**：向NaClO溶液中通入CO₂可置换出HClO：
$$\text{NaClO} + \text{CO}_2 + \text{H}_2\text{O} \rightarrow \text{NaHCO}_3 + \text{HClO}$$（强酸制弱酸，CO₂+H₂O=H₂CO₃，H₂CO₃比HClO强）

### 典型例题

**例1**：判断酸性：H₂SO₄、H₂CO₃、HClO的强弱顺序。

**分析**：H₂SO₄是强酸；H₂CO₃是中强酸（弱酸）；HClO是弱酸，且弱于H₂CO₃。
顺序：**H₂SO₄ > H₂CO₃ > HClO**

**例2**：为什么干燥的Cl₂不能漂白，而氯水能漂白？

**分析**：漂白性来自HClO。干燥Cl₂没有水，不能生成HClO，故无漂白性；氯水中含HClO，能漂白。

### 易错点 / 拓展

- HClO漂白是**氧化漂白**（不可逆）；SO₂是**化合漂白**（与有色物质结合，可逆）
- NaClO溶液（漂白液/84消毒液）在酸性条件下释放HClO，漂白效果更强
- 氯水久置漂白性减弱：HClO光照分解
