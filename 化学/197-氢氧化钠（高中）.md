# 氢氧化钠（高中）

> **一句话**：强碱，白色固体，极易溶于水放热，腐蚀性强，是重要的工业原料，俗称烧碱。
> **English**: A strong base, white solid, highly soluble in water with heat release; highly corrosive and an important industrial chemical, commonly called caustic soda.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | NaOH |
| 俗称 | 烧碱、苛性钠、火碱 |
| 颜色 | 白色固体 |
| 溶解性 | 极易溶于水（放大量热）|
| 密度 | 2.13 g/cm³ |
| 熔点 | 318°C |
| 属性 | 强电解质、强碱、离子化合物 |

### 物理性质

- 白色固体，有强烈吸水性（潮解性）→ 在空气中会**潮解**
- 溶于水放大量热
- 腐蚀性极强，能腐蚀皮肤、玻璃（与SiO₂反应）

### 化学性质

**1. 与酸中和**：
$$\text{NaOH} + \text{HCl} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$
$$\text{离子方程式：H}^+ + \text{OH}^- \rightarrow \text{H}_2\text{O}$$

**2. 与酸性氧化物**：
$$2\text{NaOH} + \text{CO}_2 \rightarrow \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$$
$$\text{NaOH} + \text{CO}_2(\text{少量}) \rightarrow \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$$（足量NaOH+少量CO₂→Na₂CO₃）
$$2\text{NaOH} + \text{CO}_2(\text{足量}) \rightarrow \text{NaHCO}_3$$（少量NaOH+足量CO₂→NaHCO₃）

注意：NaOH少量+CO₂足量→NaHCO₃

**3. 与某些盐反应（沉淀）**：
$$2\text{NaOH} + \text{CuSO}_4 \rightarrow \text{Cu(OH)}_2\downarrow + \text{Na}_2\text{SO}_4$$
$$3\text{NaOH} + \text{FeCl}_3 \rightarrow \text{Fe(OH)}_3\downarrow + 3\text{NaCl}$$

**4. 与铝反应（特殊）**：
$$2\text{NaOH} + 2\text{Al} + 2\text{H}_2\text{O} \rightarrow 2\text{NaAlO}_2 + 3\text{H}_2\uparrow$$

**5. 与玻璃反应（腐蚀玻璃）**：
$$\text{SiO}_2 + 2\text{NaOH} \rightarrow \text{Na}_2\text{SiO}_3 + \text{H}_2\text{O}$$（NaOH不能放在玻璃磨口瓶中）

### NaOH变质

NaOH暴露空气中与CO₂反应变质：
$$2\text{NaOH} + \text{CO}_2 \rightarrow \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$$

检验NaOH是否变质：取样溶于水，加入足量稀盐酸，若有气泡→含Na₂CO₃→已变质。

### 典型例题

**例1**：NaOH固体保存时应注意什么？

**分析**：密封保存，防止潮解和与CO₂反应变质；不能放在磨口玻璃瓶中（与SiO₂反应）；应保存在聚乙烯（塑料）瓶中。

**例2**：向NaOH溶液中通入少量CO₂，写离子方程式。

$$\text{OH}^-(\text{足量}) + \text{CO}_2(\text{少量}) \rightarrow \text{CO}_3^{2-} + \text{H}_2\text{O}$$（生成Na₂CO₃，离子式为CO₃²⁻）

### 易错点 / 拓展

- NaOH与CO₂：NaOH足量→Na₂CO₃；NaOH少量→NaHCO₃
- NaOH不能放在**磨口玻璃瓶**中（与SiO₂反应，磨口被粘住）
- 工业上电解饱和食盐水制NaOH（氯碱工业）：2NaCl + 2H₂O →(电解) 2NaOH + Cl₂↑ + H₂↑
- NaOH属于**离子化合物**（含Na⁺和OH⁻）
