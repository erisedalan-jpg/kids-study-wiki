# 氧化钠

> **一句话**：钠在常温下与O₂反应生成的白色碱性氧化物，不稳定，继续氧化为Na₂O₂。
> **English**: The white basic oxide formed when sodium reacts with O₂ at room temperature; unstable and further oxidized to Na₂O₂.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | Na₂O |
| 颜色 | 白色固体 |
| O化合价 | -2 |
| 类型 | 碱性氧化物 |

### 制备

$$4\text{Na} + \text{O}_2 \xrightarrow{\text{常温}} 2\text{Na}_2\text{O}$$

Na暴露在空气（常温）中即被氧化，生成白色的Na₂O。

### 化学性质

**与水反应**（碱性氧化物特征）：
$$\text{Na}_2\text{O} + \text{H}_2\text{O} \rightarrow 2\text{NaOH}$$

**与酸反应**（碱性氧化物特征）：
$$\text{Na}_2\text{O} + 2\text{HCl} \rightarrow 2\text{NaCl} + \text{H}_2\text{O}$$

**与CO₂反应**：
$$\text{Na}_2\text{O} + \text{CO}_2 \rightarrow \text{Na}_2\text{CO}_3$$

**继续被氧化**（不稳定）：
$$2\text{Na}_2\text{O} + \text{O}_2 \rightarrow 2\text{Na}_2\text{O}_2$$

或：$$4\text{Na} + \text{O}_2 \xrightarrow{\text{点燃}} 2\text{Na}_2\text{O}_2$$（直接生成过氧化钠）

### Na₂O与Na₂O₂对比

| 特征 | Na₂O | Na₂O₂ |
|------|------|-------|
| 化学式 | Na₂O | Na₂O₂ |
| O化合价 | -2 | -1 |
| 颜色 | 白色 | 淡黄色 |
| 类型 | 碱性氧化物 | 过氧化物 |
| 稳定性 | 不稳定 | 较稳定 |
| 与H₂O产物 | NaOH | NaOH + O₂ |

### 典型例题

**例1**：Na₂O是碱性氧化物，而Na₂O₂不是，为什么？

**分析**：碱性氧化物定义：与酸反应只生成盐和水，且该氧化物中O为-2价。Na₂O₂中O为-1价，且与酸反应：2Na₂O₂ + 2HCl → 2NaCl + H₂O + O₂↑（还有O₂生成），不符合碱性氧化物定义。

**例2**：Na₂O溶于水后，溶液呈什么性？

**分析**：Na₂O + H₂O → 2NaOH，NaOH完全电离，溶液显**碱性**，pH > 7。

### 易错点

- Na₂O不稳定，在空气中会继续被氧化为Na₂O₂
- Na在空气中放置：先生成Na₂O → NaOH → Na₂CO₃（最终产物）
- Na₂O₂的O化合价是-1，不是-2
