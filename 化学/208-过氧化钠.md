# 过氧化钠

> **一句话**：钠点燃后与O₂反应生成的淡黄色过氧化物，能与H₂O和CO₂反应放出O₂，用作供氧剂。
> **English**: A pale yellow peroxide formed when sodium ignites in O₂; reacts with H₂O and CO₂ to release O₂, used as an oxygen source.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | Na₂O₂ |
| 颜色 | 淡黄色固体 |
| O化合价 | -1（含O₂²⁻，过氧根）|
| 类型 | 过氧化物（不是碱性氧化物）|

### 制备

$$2\text{Na} + \text{O}_2 \xrightarrow{\text{点燃}} \text{Na}_2\text{O}_2$$

### 化学性质（重点）

**1. 与H₂O反应**（放O₂，氧化还原）：
$$2\text{Na}_2\text{O}_2 + 2\text{H}_2\text{O} \rightarrow 4\text{NaOH} + \text{O}_2\uparrow$$

**2. 与CO₂反应**（放O₂，氧化还原）：
$$2\text{Na}_2\text{O}_2 + 2\text{CO}_2 \rightarrow 2\text{Na}_2\text{CO}_3 + \text{O}_2\uparrow$$

**3. 与HCl反应**（放O₂）：
$$2\text{Na}_2\text{O}_2 + 4\text{HCl} \rightarrow 4\text{NaCl} + 2\text{H}_2\text{O} + \text{O}_2\uparrow$$

**氧化还原分析**：Na₂O₂中O为-1价，发生歧化：
- 一部分O（-1→-2）：O²⁻（还原产物，在NaOH或Na₂CO₃中）
- 一部分O（-1→0）：O₂（氧化产物）
- Na₂O₂既是氧化剂又是还原剂

### Na₂O₂的应用

| 应用场景 | 原理 |
|---------|------|
| 潜水艇供氧 | 与CO₂反应放O₂，同时消耗CO₂ |
| 矿山救援 | 与H₂O、CO₂反应放O₂ |
| 漂白剂 | 强氧化性 |

### 典型例题

**例1**：1 mol Na₂O₂与足量H₂O反应，转移几mol电子？

**分析**：2 mol Na₂O₂反应，2 mol O（-1）→ O₂（0价），共失1 mol e⁻（氧化）；2 mol O（-1）→ NaOH中O²⁻（-2价），共得1 mol e⁻（还原）。
每2 mol Na₂O₂转移1 mol e⁻，故1 mol Na₂O₂转移**0.5 mol e⁻**

**例2**：Na₂O₂加入石蕊溶液中，溶液最终变色情况？

**分析**：Na₂O₂与水：4NaOH + O₂，先碱性（蓝色）；O₂和Na₂O₂本身有强氧化性，使石蕊**褪色**（漂白）。最终溶液无色。

### 易错点

- Na₂O₂**不是碱性氧化物**（含O₂²⁻，O化合价-1）
- Na₂O₂与H₂O反应放出的O₂：歧化反应，O₂是**氧化产物**
- 区分Na₂O（白色）和Na₂O₂（淡黄色）
- Na₂O₂与足量CO₂反应，最终产物是Na₂CO₃（不是NaHCO₃）
