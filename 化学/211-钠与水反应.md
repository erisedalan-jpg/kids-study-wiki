# 钠与水反应

> **一句话**：钠与水常温下剧烈反应，生成NaOH和H₂，伴随浮、熔、游、响、红等现象。
> **English**: Sodium reacts vigorously with water at room temperature, producing NaOH and H₂, with characteristic floating, melting, and movement.

---

## 🎓 高中进阶

### 化学方程式

$$2\text{Na} + 2\text{H}_2\text{O} \rightarrow 2\text{NaOH} + \text{H}_2\uparrow$$

**离子方程式**：
$$2\text{Na} + 2\text{H}_2\text{O} \rightarrow 2\text{Na}^+ + 2\text{OH}^- + \text{H}_2\uparrow$$

**氧化还原分析**：Na（0→+1，失电子，被氧化，是还原剂）；H（+1→0，得电子，被还原，是氧化剂）

### 实验现象（浮、熔、游、响、红）

| 现象 | 解释 |
|------|------|
| **浮**：Na浮在水面 | Na密度(0.97 g/cm³) < 水(1 g/cm³) |
| **熔**：Na熔化为小球 | 反应放热，熔点低(97.8°C) |
| **游**：小球在水面游动 | H₂气体推动 |
| **响**：嘶嘶声（大量时爆炸）| 反应剧烈，产生H₂ |
| **红**：加酚酞后变红 | 生成NaOH，溶液呈碱性 |

### 与酸、碱溶液中水的反应对比

| 情形 | 描述 |
|------|------|
| Na加入纯水 | 生成NaOH + H₂↑ |
| Na加入盐酸 | 先与H⁺反应（更剧烈），再与水反应 |
| Na加入NaOH溶液 | 与水反应（NaOH不与Na反应）|
| Na加入盐溶液 | 先与水反应，再NaOH与盐反应 |

### 典型例题

**例1**：0.23 g Na加入100 mL水中（忽略体积变化），求溶液pH。

**分析**：n(Na) = 0.23/23 = 0.01 mol；生成NaOH = 0.01 mol；c(NaOH) = 0.01/0.1 = 0.1 mol/L；
NaOH强碱完全电离：c(OH⁻) = 0.1 mol/L；pOH = 1；**pH = 13**

**例2**：Na加入CuSO₄溶液，为什么不能置换Cu？

**分析**：Na的化学性质决定它遇水先反应：
$$2\text{Na} + 2\text{H}_2\text{O} \rightarrow 2\text{NaOH} + \text{H}_2\uparrow$$
$$2\text{NaOH} + \text{CuSO}_4 \rightarrow \text{Cu(OH)}_2\downarrow + \text{Na}_2\text{SO}_4$$
现象：气泡 + 蓝色沉淀，无铜析出。

### 易错点 / 拓展

- Na加入水中**不会**直接置换其他金属（先与水反应）
- 反应产物NaOH使溶液显**碱性**
- Na与足量水反应：Na完全反应；Na与少量水：水可能先耗尽
- 实验安全：用镊子取小块Na，切勿用手接触；Na着火不能用水扑灭（应用砂土）
