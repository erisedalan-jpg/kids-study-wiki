# 二氧化氮

> **一句话**：红棕色有毒气体，N化合价+4，能与水反应生成HNO₃和NO，是工业制硝酸的关键中间体，也是光化学烟雾的成因之一。
> **English**: A reddish-brown toxic gas with N in +4 oxidation state; reacts with water to produce HNO₃ and NO; a key intermediate in industrial nitric acid production and a contributor to photochemical smog.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | NO₂ |
| N化合价 | +4 |
| 颜色气味 | **红棕色**，有刺激性气味，有毒 |
| 密度 | 比空气大 |
| 溶解性 | 易溶于水 |

### 化学性质

**1. 与水反应**（重要，产生酸和NO）：
$$3\text{NO}_2 + \text{H}_2\text{O} \rightarrow 2\text{HNO}_3 + \text{NO}$$

（3体积NO₂溶于水后，产生1体积NO）

**与热水**（产物相同，速率更快）：同上方程式

**2. 与NaOH反应**：
$$\text{NO}_2 + \text{NaOH} \rightarrow \text{NaNO}_3 + \text{NaNO}_2 + \text{H}_2\text{O}$$
更准确（歧化）：$2\text{NO}_2 + 2\text{NaOH} \rightarrow \text{NaNO}_3 + \text{NaNO}_2 + \text{H}_2\text{O}$

尾气处理：
$$\text{NO}_2 + 2\text{NaOH} \rightarrow \text{NaNO}_3 + \text{H}_2\text{O}$$ （足量NaOH）

**3. NO与O₂来生成NO₂**：
$$2\text{NO} + \text{O}_2 \rightarrow 2\text{NO₂}$$

### NO₂与水的特殊计算

向密闭容器通入V(NO₂) mL + V(O₂) mL，加水后气体恢复原体积还是剩余？

**分析**：
- NO₂+H₂O → HNO₃ + NO（3:1反应）
- 剩余NO继续：2NO + O₂ → 2NO₂（O₂用量）
- O₂是否足够将NO再次氧化？

**口诀**：
- NO₂和O₂混合通水：当V(NO₂):V(O₂) = 4:1时，恰好完全吸收（无气体剩余）

**推导**：
4NO₂ + O₂ + 2H₂O → 4HNO₃（总反应）

### 工业合成硝酸全流程中的NO₂

1. 4NH₃ + 5O₂ →(Pt, 800°C) 4NO + 6H₂O
2. **2NO + O₂ → 2NO₂**
3. 3NO₂ + H₂O → 2HNO₃ + NO（NO循环）

### 典型例题

**例1**：向烧瓶中收集满NO₂，倒置于水中，水上升后最终如何？

**分析**：3NO₂ + H₂O → 2HNO₃ + NO
- 3L NO₂ → 1L NO（剩余）
- 水上升占了约2/3体积（3L中2L被消耗），剩余1L NO气体
- 烧瓶内气体约为原体积的1/3

**例2**：NO₂和O₂以4:1体积通入NaOH溶液，有什么气体剩余？

**分析**：4NO₂ + O₂ + 2H₂O → 4HNO₃；恰好完全吸收，**无气体剩余**。

### 易错点

- NO₂是**红棕色**，NO是**无色**（常混淆）
- 3NO₂ + H₂O → 2HNO₃ + NO（3体积NO₂+1体积H₂O→2mol HNO₃+1体积NO）
- NO₂不是酸性氧化物（酸性氧化物与水只生成一种酸，NO₂与水生成HNO₃和NO，有氧化还原）
- NO₂是**有毒**气体，不能随意排放
