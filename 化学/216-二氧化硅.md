# 二氧化硅

> **一句话**：酸性氧化物，是玻璃、水晶、石英的主要成分，能与HF和NaOH反应，不溶于其他酸，不与水反应。
> **English**: An acidic oxide; the main component of glass, crystal, and quartz; reacts with HF and NaOH, but not with water or most acids.

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | SiO₂ |
| Si化合价 | +4 |
| 颜色 | 无色（纯净）或白色固体 |
| 熔点 | 1713°C（极高）|
| 溶解性 | 不溶于水，不与水反应 |
| 类型 | 酸性氧化物（特殊：不与水反应）|

### 化学性质

**1. 与HF反应（特殊，能腐蚀玻璃）**：
$$\text{SiO}_2 + 4\text{HF} \rightarrow \text{SiF}_4\uparrow + 2\text{H}_2\text{O}$$

HF**腐蚀玻璃**的原因：玻璃主要成分是SiO₂，HF与SiO₂反应。
→ 因此HF不能储存在玻璃瓶中，应存放在**塑料容器**中。

**2. 与NaOH溶液反应**（酸性氧化物特征）：
$$\text{SiO}_2 + 2\text{NaOH} \rightarrow \text{Na}_2\text{SiO}_3 + \text{H}_2\text{O}$$

（NaOH能腐蚀玻璃，所以NaOH不能放在磨口玻璃瓶中）

**3. 与Na₂CO₃在高温下反应**（制玻璃）：
$$\text{SiO}_2 + \text{Na}_2\text{CO}_3 \xrightarrow{\text{高温}} \text{Na}_2\text{SiO}_3 + \text{CO}_2\uparrow$$

（强酸制弱酸：H₂CO₃比H₂SiO₃强，但高温下热力学驱动）

**4. 与CaO（石灰石）高温**：
$$\text{SiO}_2 + \text{CaO} \xrightarrow{\text{高温}} \text{CaSiO}_3$$

**SiO₂不与水反应**（区别于SO₂、CO₂等酸性氧化物）

### SiO₂的特殊性

- SiO₂是**酸性氧化物**，但**不与水反应**（无对应的硅酸稳定水合物直接生成）
- 对应的酸是H₂SiO₃（硅酸），但H₂SiO₃不能由SiO₂直接与水反应得到

### SiO₂的应用

| 应用 | 形式 |
|------|------|
| 光导纤维 | 高纯SiO₂ |
| 玻璃 | Na₂SiO₃·CaSiO₃·SiO₂（普通玻璃）|
| 水晶工艺品 | 纯SiO₂晶体 |
| 砂纸 | 石英砂（SiO₂，高硬度）|
| 芯片生产 | SiO₂作绝缘层 |

### 典型例题

**例1**：HF不能储存在玻璃瓶中，应存放在什么容器中？

**分析**：HF与SiO₂（玻璃）反应，腐蚀玻璃。应存放在**聚乙烯（PE）塑料瓶**中，或石蜡容器中。

**例2**：向Na₂SiO₃溶液中通入过量CO₂？

**分析**：CO₂+H₂O=H₂CO₃；H₂CO₃酸性强于H₂SiO₃，强酸制弱酸：
Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃（足量CO₂）→ Na₂CO₃+CO₂→NaHCO₃，最终：
Na₂SiO₃ + 2CO₂ + 2H₂O → H₂SiO₃↓ + 2NaHCO₃

现象：白色胶状沉淀（H₂SiO₃）生成。

### 易错点

- SiO₂是酸性氧化物，但**不与水反应**（CO₂、SO₂可与水反应，SiO₂不能）
- SiO₂能与HF反应（腐蚀玻璃），不与HCl、H₂SO₄反应
- NaOH不能存放在玻璃磨口瓶（NaOH+SiO₂→Na₂SiO₃，粘住磨口）
- 光导纤维≠金属导线，SiO₂传导光信号，不传导电子
