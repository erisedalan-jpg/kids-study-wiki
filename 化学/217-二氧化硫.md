# 二氧化硫

> **一句话**：无色有刺激性气味的有毒气体，酸性氧化物，有漂白性（化合漂白，可逆），是酸雨和大气污染的主要原因。
> **English**: A colorless toxic gas with a pungent odor; an acidic oxide with reversible bleaching properties (combination bleaching); a major cause of acid rain and air pollution.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | SO₂ |
| S化合价 | +4 |
| 颜色气味 | 无色，有强烈刺激性气味 |
| 毒性 | 有毒，损害呼吸系统 |
| 密度 | 比空气大 |
| 溶解性 | 易溶于水（1:40体积比）|
| 类型 | 酸性氧化物 |

### 化学性质

**1. 与水反应**（酸性氧化物）：
$$\text{SO}_2 + \text{H}_2\text{O} \rightleftharpoons \text{H}_2\text{SO}_3\quad(\text{亚硫酸，弱酸})$$

**2. 与碱反应**（酸性氧化物）：
$$\text{SO}_2 + 2\text{NaOH} \rightarrow \text{Na}_2\text{SO}_3 + \text{H}_2\text{O}\quad(\text{NaOH足量})$$
$$\text{SO}_2 + \text{NaOH} \rightarrow \text{NaHSO}_3\quad(\text{SO}_2足量})$$

**3. 被氧化（还原性）**：
$$2\text{SO}_2 + \text{O}_2 \rightleftharpoons 2\text{SO}_3\quad(\text{接触法制硫酸，催化剂})$$
$$5\text{SO}_2 + 2\text{KMnO}_4 + 2\text{H}_2\text{O} \rightarrow 2\text{MnSO}_4 + \text{K}_2\text{SO}_4 + 2\text{H}_2\text{SO}_4$$

**4. 漂白性**（特殊，可逆）：
SO₂与有色物质（如品红）**结合**形成无色化合物（不是氧化有色物质），加热可恢复颜色：

$$\text{SO}_2 + \text{有色物质} \rightleftharpoons \text{无色化合物（可逆）}$$

**5. 氧化性（较弱）**：
$$\text{SO}_2 + 2\text{H}_2\text{S} \rightarrow 3\text{S} + 2\text{H}_2\text{O}\quad(\text{SO}_2\text{氧化H}_2\text{S})$$

### SO₂漂白性 vs Cl₂（HClO）漂白性

| 特征 | SO₂漂白 | Cl₂（HClO）漂白 |
|------|---------|----------------|
| 本质 | 化合漂白（与色素结合）| 氧化漂白（破坏色素结构）|
| 可逆性 | **可逆**（加热恢复颜色）| **不可逆** |
| 对象 | 有机色素（品红等）| 广谱 |
| 对于有色食品 | 用于食品漂白（限量）| 不能用于食品 |

### 酸雨

SO₂是形成酸雨的主要原因：
$$\text{SO}_2 + \text{H}_2\text{O} \rightarrow \text{H}_2\text{SO}_3$$
$$2\text{H}_2\text{SO}_3 + \text{O}_2 \rightarrow 2\text{H}_2\text{SO}_4\quad(\text{氧化进一步形成硫酸，pH}<5.6)$$

### 典型例题

**例1**：用品红溶液检验SO₂，现象？加热后？

**分析**：SO₂通入品红：**品红褪色**（SO₂化合漂白）；加热：**品红复色**（可逆）。

**例2**：SO₂能使澄清石灰水变浑浊，CO₂也能，如何区分？

**分析**：向溶液中加KMnO₄（高锰酸钾，紫色）：
- 含SO₂：KMnO₄被SO₂还原，**紫色褪去**（SO₂具有还原性）
- 含CO₂：KMnO₄不被还原，**紫色不褪**
故用KMnO₄溶液区分SO₂和CO₂。

### 尾气处理

SO₂有毒，实验后用**NaOH溶液**吸收：
$$\text{SO}_2 + 2\text{NaOH} \rightarrow \text{Na}_2\text{SO}_3 + \text{H}_2\text{O}$$

### 易错点

- SO₂的漂白是**可逆的化合漂白**，不是氧化漂白（Cl₂才是氧化漂白）
- SO₂通入品红褪色后加热复色，可鉴别SO₂（vs Cl₂：Cl₂漂白后加热不复色）
- SO₂本身是弱酸（非电解质），H₂SO₃是弱电解质
- SO₂不能漂白石蕊（石蕊遇酸变红，不褪色）
