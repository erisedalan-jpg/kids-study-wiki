# 氨气

> **一句话**：无色、有强烈刺激性气味的气体，极易溶于水生成弱碱性NH₃·H₂O，是工业合成含氮化合物的关键原料，催化氧化可得NO进而制硝酸。
> **English**: A colorless gas with a pungent smell; extremely soluble in water to form weakly basic NH₃·H₂O; a key feedstock for nitrogen-containing compounds; catalytic oxidation yields NO for nitric acid production.

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | NH₃ |
| N化合价 | −3（最低价） |
| 颜色气味 | 无色，**强烈刺激性气味**，有毒 |
| 密度 | 比空气小（M=17 g/mol < 29）|
| 溶解性 | **极易溶于水**（1体积水溶700体积NH₃，20°C）|
| 沸点 | −33.35°C |

### 化学性质

**1. 与水反应**（弱碱性）：
$$\text{NH}_3 + \text{H}_2\text{O} \rightleftharpoons \text{NH}_3\cdot\text{H}_2\text{O} \rightleftharpoons \text{NH}_4^+ + \text{OH}^-$$

（NH₃·H₂O是弱碱，部分电离；氨水显碱性，能使酚酞变红）

**2. 与酸反应**（产生白烟）：
$$\text{NH}_3 + \text{HCl} \rightarrow \text{NH}_4\text{Cl}\quad\text{（白烟）}$$
$$\text{NH}_3 + \text{HNO}_3 \rightarrow \text{NH}_4\text{NO}_3$$
$$2\text{NH}_3 + \text{H}_2\text{SO}_4 \rightarrow (\text{NH}_4)_2\text{SO}_4$$

与挥发性酸（HCl、HNO₃等）相遇产生**白烟**（铵盐固体），用于检验NH₃。

**3. 催化氧化**（工业制硝酸关键步骤）：
$$4\text{NH}_3 + 5\text{O}_2 \underset{800°\text{C}}{\overset{\text{Pt}}{\rightarrow}} 4\text{NO} + 6\text{H}_2\text{O}$$

（选择性氧化，Pt催化剂，NH₃从−3升至+2价，被氧化）

**4. 与O₂直接燃烧**（无催化剂）：
$$4\text{NH}_3 + 3\text{O}_2 \xrightarrow{\text{点燃}} 2\text{N}_2 + 6\text{H}_2\text{O}$$

（无催化剂时，NH₃燃烧产生N₂，而非NO）

**5. 还原性**（NH₃作还原剂，N为最低价−3）：
$$8\text{NH}_3 + 3\text{Cl}_2 \rightarrow \text{N}_2 + 6\text{NH}_4\text{Cl}$$
（NH₃与Cl₂反应，用于制氮化合物）

### 工业合成氨（哈伯法）

$$\text{N}_2 + 3\text{H}_2 \underset{400-500°\text{C}}{\overset{\text{Fe催化,高压}}{\rightleftharpoons}} 2\text{NH}_3\quad\Delta H<0$$

| 条件 | 选择原因 |
|------|----------|
| 高温（400-500°C）| 反应放热，高温降转化率，但高温提高速率；取折中温度 |
| 高压（10-30 MPa）| 正反应气体mol数减小，高压提高转化率 |
| Fe催化剂 | 降低活化能，加快速率 |

转化率约 10-20%，未反应的N₂和H₂循环利用。

### 氨水的性质

| 成分 | 分子/离子 |
|------|----------|
| 分子 | NH₃·H₂O、NH₃、H₂O |
| 离子 | NH₄⁺、OH⁻（少量H⁺）|

氨水是混合物（非纯NH₃·H₂O），浓度越大→NH₃逸出多→密度反而减小。

### 喷泉实验原理

NH₃极易溶于水（溶解700倍体积）→密封烧瓶内压强迅速降低→外部水被大气压压入→形成喷泉（溶液为碱性，使酚酞变红）。

### 实验室制备NH₃

$$2\text{NH}_4\text{Cl} + \text{Ca(OH)}_2 \xrightarrow{\Delta} \text{CaCl}_2 + 2\text{NH}_3\uparrow + 2\text{H}_2\text{O}$$

| 步骤 | 操作 |
|------|------|
| 原料 | NH₄Cl固体 + Ca(OH)₂固体，加热 |
| 干燥 | 碱石灰（NaOH + CaO）—— 不能用浓H₂SO₄或无水CaCl₂ |
| 收集 | 向下排气法（NH₃密度比空气小）|
| 验满 | 湿润红色石蕊试纸变蓝；或用浓盐酸靠近瓶口出现白烟 |
| 尾气 | 稀硫酸吸收（防止NH₃污染空气）|

### 典型例题

**例1**：足量NH₃通入过量AlCl₃溶液，有什么沉淀生成？能否继续溶解？

**分析**：
$$\text{AlCl}_3 + 3\text{NH}_3\cdot\text{H}_2\text{O} \rightarrow \text{Al(OH)}_3\downarrow + 3\text{NH}_4\text{Cl}$$
生成白色Al(OH)₃沉淀。NH₃是弱碱，**不能**溶解Al(OH)₃（需强碱NaOH才能溶解两性Al(OH)₃）。故沉淀不溶解。这也是实验室用NH₃·H₂O制Al(OH)₃（而非NaOH）的原因。

**例2**：工业合成氨中，增大压强为什么能提高NH₃产率？若再升温至600°C，产率如何变化？

**分析**：
- N₂+3H₂→2NH₃：反应物气体4mol→产物2mol，高压使平衡向mol数减小方向（正方向）移动，NH₃产率增大。
- 该反应为放热反应（ΔH<0）；升温使平衡**向逆方向**移动，NH₃产率**下降**。
- 实际选400-500°C：在催化剂活性范围内兼顾速率和产率。

### 易错点

- NH₃与盐酸相遇产生**白烟**（NH₄Cl固体），不是白雾（白雾是水蒸气液化，白烟是固体小颗粒）
- 氨水是**弱碱**（NH₃·H₂O部分电离），不是强碱
- 干燥NH₃不能用浓H₂SO₄（酸碱中和）、CaCl₂（CaCl₂+8NH₃→CaCl₂·8NH₃，形成配合物）；只能用**碱石灰**
- NH₃催化氧化（Pt，800°C）→NO；NH₃直接燃烧→N₂（两者条件和产物不同）
- 氨水密度**比水小**（溶有NH₃，浓氨水密度＜稀氨水密度），与HCl溶液相反
