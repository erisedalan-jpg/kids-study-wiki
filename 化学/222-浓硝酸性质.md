# 浓硝酸性质

> **一句话**：浓HNO₃具有强挥发性、强氧化性（产生NO₂），能使Fe/Al钝化，见光分解，比稀HNO₃氧化性更强。
> **English**: Concentrated HNO₃ has strong volatility and oxidizing power (producing NO₂); passivates Fe/Al; decomposes in light; more oxidizing than dilute HNO₃.

---

## 🎓 高中进阶

### 浓HNO₃的特殊性质

| 性质 | 描述 |
|------|------|
| 挥发性 | 浓HNO₃易挥发（沸点83°C，开瓶有白雾）|
| 见光分解 | 4HNO₃ →(光) 4NO₂↑ + O₂↑ + 2H₂O |
| 强氧化性 | 与Cu→NO₂（红棕色）；与Fe→钝化；与非金属→SO₂/CO₂ |
| 钝化作用 | Fe、Al在冷浓HNO₃中钝化 |
| 颜色 | 淡黄色（含溶解的NO₂）|

### 浓HNO₃与金属的反应

**与Cu**（常温）：
$$\text{Cu} + 4\text{HNO}_3(\text{浓}) \rightarrow \text{Cu(NO}_3)_2 + 2\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$
（产生**红棕色**NO₂气体，区别于稀HNO₃产生无色NO）

**与Ag**（类似）：
$$3\text{Ag} + 4\text{HNO}_3(\text{浓}) \rightarrow 3\text{AgNO}_3 + \text{NO}\uparrow + 2\text{H}_2\text{O}$$（稀HNO₃）

**Fe/Al在冷浓HNO₃中钝化**（不反应，表面形成致密氧化膜）：
- 可以用铁罐（或铝罐）运输浓HNO₃
- 加热可与Fe反应（高温破坏氧化膜）

### 浓HNO₃ vs 稀HNO₃

| 对比 | 浓HNO₃（≥65%）| 稀HNO₃（<65%）|
|------|-------------|-------------|
| 与Cu产物 | NO₂（红棕色）| NO（无色）|
| 氧化性 | **更强** | 较弱 |
| 与Fe（冷）| 钝化 | 3Fe + 8HNO₃→Fe(NO₃)₂+... |
| 气体颜色 | 红棕色 | 无色（需接触空气变色）|

### 浓HNO₃与非金属

**与碳**（加热）：
$$\text{C} + 4\text{HNO}_3(\text{浓}) \xrightarrow{\Delta} \text{CO}_2\uparrow + 4\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$

**与硫**：
$$\text{S} + 6\text{HNO}_3(\text{浓}) \xrightarrow{\Delta} \text{H}_2\text{SO}_4 + 6\text{NO}_2\uparrow + 2\text{H}_2\text{O}$$

### 典型例题

**例1**：实验室用Cu和浓HNO₃制备NO₂，尾气如何处理？

**分析**：NO₂有毒，需用NaOH溶液吸收：
2NO₂ + 2NaOH → NaNO₃ + NaNO₂ + H₂O
（或：NO₂ + NaOH → NaNO₃ + H₂O，需足量NaOH）

**例2**：一支铁钉放入冷浓HNO₃中，后放入稀盐酸，有什么现象？

**分析**：铁钉放入冷浓HNO₃→钝化（表面形成致密Fe₂O₃膜）→无明显气泡；取出放入稀盐酸→HCl能溶解氧化膜（Fe₂O₃+6HCl→2FeCl₃+3H₂O），然后Fe与HCl反应→气泡（H₂）。
实际上，稀盐酸先溶解表面氧化膜，然后Fe才与HCl反应产生H₂。

### 易错点

- 浓HNO₃产生**NO₂**（红棕色）；稀HNO₃产生**NO**（无色）
- Fe在**冷浓HNO₃**中钝化，在**稀HNO₃**中反应（生成Fe³⁺）
- 浓HNO₃见光分解，所以储存于**棕色瓶中**，避光阴凉处
- 浓HNO₃有强挥发性，实验应在通风橱中进行
