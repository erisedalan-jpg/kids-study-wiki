# 硅酸盐

> **一句话**：含SiO₄⁴⁻（硅酸根）为基本结构单元的盐类，是地壳中最丰富的矿物，也是玻璃、陶瓷、水泥等工业材料的基础。
> **English**: Salts containing SiO₄⁴⁻ (silicate) as the basic structural unit; the most abundant minerals in the Earth's crust; the basis of glass, ceramics, and cement industries.

---

## 🎓 高中进阶

### 定义

硅酸盐是含有**硅酸根**（由Si和O组成的含氧酸根，如SiO₃²⁻、SiO₄⁴⁻等）的盐类总称。地壳的主要成分是硅酸盐矿物（约占地壳质量的75%以上）。

### 常见硅酸盐

| 名称 | 化学式 | 用途 |
|------|-------|------|
| 硅酸钠（水玻璃）| Na₂SiO₃ | 防火剂、粘合剂 |
| 硅酸钙 | CaSiO₃ | 水泥成分 |
| 长石 | KAlSi₃O₈ | 陶瓷原料 |
| 高岭土（瓷土）| Al₂(Si₂O₅)(OH)₄ | 瓷器原料 |
| 云母 | KAl₂(AlSi₃O₁₀)(OH)₂ | 绝缘材料 |
| 石棉 | Mg₃Si₂O₅(OH)₄ | 隔热（有毒，已限用）|

### 硅酸盐工业（三大传统硅酸盐工业）

| 工业 | 主要原料 | 产品 |
|------|---------|------|
| 玻璃工业 | SiO₂、Na₂CO₃、CaCO₃ | 玻璃（Na₂SiO₃·CaSiO₃·SiO₂）|
| 水泥工业 | 石灰石（CaCO₃）、黏土 | 硅酸三钙等（水泥）|
| 陶瓷工业 | 高岭土、石英、长石 | 陶瓷 |

### 玻璃的制备（化学反应）

$$\text{Na}_2\text{CO}_3 + \text{SiO}_2 \xrightarrow{\text{高温}} \text{Na}_2\text{SiO}_3 + \text{CO}_2\uparrow$$
$$\text{CaCO}_3 + \text{SiO}_2 \xrightarrow{\text{高温}} \text{CaSiO}_3 + \text{CO}_2\uparrow$$

（在高温下CO₂逸出，SiO₂与碳酸盐反应生成硅酸盐）

### 硅酸盐的书写（氧化物形式）

无机化学中，硅酸盐常用氧化物形式表示：

| 化学式 | 氧化物形式 |
|-------|----------|
| Na₂SiO₃ | Na₂O·SiO₂ |
| CaSiO₃ | CaO·SiO₂ |
| Al₂Si₂O₇ | Al₂O₃·2SiO₂ |
| 普通玻璃 | Na₂O·CaO·6SiO₂ |

### Na₂SiO₃（水玻璃）的性质

- Na₂SiO₃是可溶性硅酸盐，溶液呈碱性（SiO₃²⁻水解）
- 与酸反应：Na₂SiO₃ + 2HCl → H₂SiO₃↓ + 2NaCl（H₂SiO₃是白色胶状沉淀）
- 与CO₂反应：Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃

**应用**：Na₂SiO₃溶液（水玻璃）用作防火剂（遇火生成SiO₂绝热层），也是蛋的保鲜剂（碱性抑菌）。

### 典型例题

**例1**：说出三大传统硅酸盐工业的原料和产品。

**分析**：
- 玻璃：SiO₂+Na₂CO₃+CaCO₃高温→玻璃
- 水泥：石灰石+黏土高温→水泥
- 陶瓷：高岭土+石英+长石高温→陶瓷

**例2**：Na₂SiO₃溶液中通入CO₂气体，发生什么现象？

**分析**：CO₂+H₂O=H₂CO₃；H₂CO₃酸性>H₂SiO₃；Na₂SiO₃ + CO₂ + H₂O → H₂SiO₃↓ + Na₂CO₃
现象：产生**白色胶状沉淀**（硅酸）。

### 易错点

- 大多数硅酸盐**不溶于水**（Na₂SiO₃、K₂SiO₃可溶）
- 硅酸盐结构复杂，组成不唯一（化学式形式多样）
- 玻璃是混合物，不是纯净物
