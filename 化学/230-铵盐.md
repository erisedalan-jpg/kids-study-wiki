# 铵盐

> **一句话**：NH₄⁺与酸根形成的盐类，均溶于水，加热易分解，与碱共热放出NH₃（检验NH₄⁺的方法），是重要的氮肥。
> **English**: Salts formed by NH₄⁺ and acid radicals; all soluble in water; decompose easily on heating; release NH₃ when heated with alkali (the test for NH₄⁺); important nitrogen fertilizers.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 描述 |
|------|------|
| 组成 | NH₄⁺（铵离子）+ 酸根离子 |
| 溶解性 | **全部溶于水**（铵盐无难溶例外）|
| 外观 | 一般为白色晶体 |
| 挥发性 | 多数受热分解，挥发性强 |

### 常见铵盐

| 铵盐 | 化学式 | 用途 | 受热分解产物 |
|------|--------|------|------------|
| 氯化铵 | NH₄Cl | 氮肥、焊接助焊剂 | NH₃↑ + HCl↑（两者接触又重新结合）|
| 碳酸铵 | (NH₄)₂CO₃ | 氮肥、发酵粉 | NH₃↑ + CO₂↑ + H₂O |
| 碳酸氢铵 | NH₄HCO₃ | 氮肥（不稳定，易分解）| NH₃↑ + CO₂↑ + H₂O |
| 硫酸铵 | (NH₄)₂SO₄ | 氮肥（稳定，优质）| 高温才分解 |
| 硝酸铵 | NH₄NO₃ | 氮肥、炸药原料 | 高温→N₂O + H₂O；可爆炸 |
| 磷酸铵 | (NH₄)₃PO₄ | 复合肥 | —— |

### 化学性质

**1. 受热分解**（热不稳定性）：

氯化铵加热：
$$\text{NH}_4\text{Cl} \xrightarrow{\Delta} \text{NH}_3\uparrow + \text{HCl}\uparrow$$
（升华-再结合现象：管口白色固体NH₄Cl再析出，管壁中部无固体，管口有固体——"上坡运动"）

碳酸氢铵加热（最不稳定，常温也缓慢分解）：
$$\text{NH}_4\text{HCO}_3 \xrightarrow{\Delta} \text{NH}_3\uparrow + \text{CO}_2\uparrow + \text{H}_2\text{O}$$

硝酸铵加热（可爆炸，慎用）：
$$\text{NH}_4\text{NO}_3 \xrightarrow{\Delta} \text{N}_2\text{O}\uparrow + 2\text{H}_2\text{O}$$

**2. 与碱反应**（NH₄⁺的特征反应，重要）：

加热条件下：
$$\text{NH}_4\text{Cl} + \text{NaOH} \xrightarrow{\Delta} \text{NaCl} + \text{NH}_3\uparrow + \text{H}_2\text{O}$$

通用离子方程式（加热）：
$$\text{NH}_4^+ + \text{OH}^- \xrightarrow{\Delta} \text{NH}_3\uparrow + \text{H}_2\text{O}$$

常温下也可反应，但NH₃不逸出（需加热才有气体产生）。

### NH₄⁺的检验方法

**标准操作**：
1. 取少量待测样品于试管中
2. 加入足量NaOH溶液
3. **加热**
4. 将湿润的红色石蕊试纸置于试管口
5. 若试纸变蓝→有NH₄⁺（产生NH₃气体）

| 步骤 | 注意事项 |
|------|----------|
| NaOH需足量 | 过量碱确保NH₄⁺全部转化 |
| 必须加热 | 不加热NH₃逸出量少，现象不明显 |
| 湿润石蕊试纸 | 干燥试纸不灵敏；用湿润红色（NH₃显碱性→蓝色）|
| 试纸在试管口 | 不要伸入试管内，防止液体沾污试纸 |

### 铵盐作氮肥的注意事项

| 不当操作 | 原因 |
|----------|------|
| 铵盐与草木灰（K₂CO₃/KOH）混用 | K₂CO₃+2NH₄⁺→2K⁺+NH₃↑，氮素损失 |
| 铵盐与碱性土壤同用 | 碱性条件促进NH₄⁺→NH₃逸散 |
| NH₄HCO₃在高温天气施用 | 常温分解加速，损失氮素 |
| 铵盐深施覆土 | 正确做法，减少NH₃挥发 |

### 硫酸铵vs碳酸氢铵对比

| 对比 | (NH₄)₂SO₄ | NH₄HCO₃ |
|------|-----------|---------|
| 氮含量 | 21.2%（较低）| 17.7% |
| 稳定性 | **稳定**（高温才分解）| **不稳定**（常温缓慢分解）|
| 保存 | 普通低温存放 | 必须密封低温保存 |
| 优点 | 稳定，可机械施肥 | 价格低廉 |
| 缺点 | 长期施用使土壤酸化 | 易分解，运输困难 |

### 典型例题

**例1**：某白色固体可能含有Na₂SO₄、NH₄Cl、BaCl₂、Na₂CO₃，设计实验方案确定成分。

**分析**：
- 先取样溶于水：若全溶→无BaCl₂（BaSO₄、BaCO₃不溶）；若有沉淀→有BaCl₂
- 实际：BaCl₂+Na₂SO₄→BaSO₄↓（白色，不溶于盐酸）；BaCl₂+Na₂CO₃→BaCO₃↓（白色，溶于盐酸）
- 取样+NaOH加热→湿润红色石蕊试纸变蓝→有NH₄Cl
- 取滤液+AgNO₃溶液+稀HNO₃→白色沉淀不溶→有Cl⁻（来自NH₄Cl或BaCl₂）

**例2**：NH₄HCO₃施入农田后，最终转化为什么物质？

**分析**：
$$\text{NH}_4\text{HCO}_3 \rightarrow \text{NH}_3 + \text{CO}_2 + \text{H}_2\text{O}$$
NH₃被植物吸收（氮源）；CO₂逸入空气（参与碳循环）；同时NH₄⁺可被土壤微生物硝化为NO₃⁻，植物可直接吸收。最终氮以NH₃/NH₄⁺/NO₃⁻形式被植物利用。

### 易错点

- 铵盐加热分解不一定只产生NH₃，如NH₄NO₃加热产生N₂O（氧化还原），硝酸铵过量加热可爆炸
- NH₄⁺检验必须**加热**，常温加NaOH可产生NH₃但量少，现象不明显
- "铵盐与碱不能混合施肥"：本质是NH₄⁺+OH⁻→NH₃↑，氮素损失
- NH₄Cl加热分解→管口重新出现白色NH₄Cl，看似"升华"但实质是分解再化合
- 铵盐均溶于水，无例外（这与大多数盐不同，大多数盐有难溶的）
