# 丙烷

> **一句话**：丙烷（C₃H₈）是含三个碳原子的烷烃，液化石油气（LPG）的主要成分，可燃烧，与卤素发生取代反应，有2种一氯代产物。
> **English**: Propane (C₃H₈) is a three-carbon alkane and a main component of liquefied petroleum gas (LPG), combustible, undergoing substitution reactions with halogens, with 2 possible monochlorination products.

### 基本信息

| 项目 | 数据 |
|------|------|
| 化学式 | C₃H₈ |
| 分子量 | 44 g/mol |
| 结构式 | CH₃-CH₂-CH₃ |
| 状态 | 无色气体（常温常压），可加压液化 |
| 沸点 | -42.1°C |
| 熔点 | -187.6°C |
| 溶解性 | 极难溶于水 |

### 丙烷的结构

$$\text{CH}_3 - \text{CH}_2 - \text{CH}_3$$

- 碳链为3个碳（中间为亚甲基CH₂，两端为甲基CH₃）
- 所有碳均为sp³杂化
- 8个H原子：两侧CH₃各3个（共6个），中间CH₂共2个

**H原子的种类**：
- **甲基氢（-CH₃的H）**：6个，等价（位于1位和3位）
- **亚甲基氢（-CH₂-的H）**：2个，等价（位于2位）
共2种H → 取代反应时有**2种**一氯代产物

### 丙烷的化学性质

**1. 燃烧**：

$$C_3H_8 + 5O_2 \xrightarrow{\text{点燃}} 3CO_2 + 4H_2O$$

（液化石油气燃烧，释放大量热）

**2. 取代反应**（与Cl₂，光照）：

$$C_3H_8 + Cl_2 \xrightarrow{\text{光照}} C_3H_7Cl + HCl$$

有**2种**一氯代丙烷（同分异构体）：
- **1-氯丙烷**（正丙基氯）：CH₃CH₂CH₂Cl（取代端基CH₃的H）
- **2-氯丙烷**（异丙基氯）：CH₃CHClCH₃（取代中间CH₂的H）

**3. 裂解**（高温）：

$$C_3H_8 \rightarrow C_3H_6 + H_2 \text{（丙烯）}$$
$$C_3H_8 \rightarrow C_2H_4 + CH_4 \text{（乙烯+甲烷）}$$

### 丙烷的同分异构体

**C₃H₈**：只有1种结构（丙烷，无支链异构）
（但丁烷 C₄H₁₀有2种：正丁烷和异丁烷）

### 丙烷与液化石油气（LPG）

| 成分 | 比例 |
|------|------|
| 丙烷（C₃H₈）| 主要成分 |
| 丁烷（C₄H₁₀）| 主要成分 |
| 少量丙烯、乙烷 | 微量 |

LPG特点：常温常压下为气体，加压可液化（方便储存和运输），热值约46 MJ/kg，比天然气热值（55 MJ/kg）低，比煤热值（29 MJ/kg）高。

### 典型例题

**例1**：丙烷与Cl₂在光照下发生一氯代反应，有几种产物（不计HCl）？写出各产物的结构式。

**分析**：
- 丙烷有2种H：甲基H（6个）和亚甲基H（2个）
- 取代甲基H → CH₃CH₂CH₂Cl（1-氯丙烷）
- 取代亚甲基H → CH₃CHClCH₃（2-氯丙烷）
- 共**2种**一氯代产物

**例2**：写出丙烷完全燃烧的化学方程式，计算1 mol丙烷燃烧需要多少mol O₂？

**分析**：
$$C_3H_8 + 5O_2 \rightarrow 3CO_2 + 4H_2O$$
1 mol丙烷需要**5 mol O₂**（配平方程式验证：碳平衡3=3，氢平衡8=8，氧平衡10=10 ✓）

### 易错点
- 丙烷一氯代有2种产物（不是3种）：1位和3位等价，只有2种H
- LPG不是纯甲烷（天然气主要是甲烷），LPG以丙烷丁烷为主
- 丙烷无双键，不与溴水/KMnO₄反应
