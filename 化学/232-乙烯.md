# 乙烯

> **一句话**：乙烯（CH₂=CH₂）是最简单的烯烃，含一个碳碳双键，是重要的化工原料，可发生加成反应和聚合反应，也是植物天然催熟激素。
> **English**: Ethylene (CH₂=CH₂) is the simplest alkene with one C=C double bond, an important chemical raw material, capable of addition and polymerization reactions, and also a natural plant ripening hormone.

| 项目 | 数据 |

| 化学式 | C₂H₄ |
| 分子量 | 28 g/mol |
| 结构式 | CH₂=CH₂ |
| 状态 | 无色气体，略带甜香气味 |
| 沸点 | -104°C |
| 溶解性 | 难溶于水 |
| 密度 | 1.26 g/L（比空气略重）|

### 乙烯的结构

**结构特点**：
- C=C**双键**（1σ键 + 1π键）
- 两个C原子均为**sp²杂化**
- 分子**平面型**（所有原子共面）
- 键角约**120°**（H-C=C）
- 双键**不能自由旋转**（π键约束）

**空间构型**：平面形分子，H-C=C-H键角=120°

### 乙烯的化学性质

**1. 加成反应**（最重要）：

| 试剂 | 反应 | 产物 |
|------|------|------|
| Br₂（溴水）| CH₂=CH₂+Br₂→CH₂BrCH₂Br | 1,2-二溴乙烷 |
| H₂（催化剂）| CH₂=CH₂+H₂→CH₃CH₃ | 乙烷 |
| HCl | CH₂=CH₂+HCl→CH₃CH₂Cl | 氯乙烷 |
| H₂O（催化）| CH₂=CH₂+H₂O→CH₃CH₂OH | 乙醇 |
| HBr | CH₂=CH₂+HBr→CH₃CH₂Br | 溴乙烷 |

**2. 聚合反应（加聚）**：

$$n\text{CH}_2=\text{CH}_2 \xrightarrow{\text{催化剂/高压}} (-\text{CH}_2-\text{CH}_2-)_n$$

产物：**聚乙烯**（polyethylene, PE），是最重要的塑料之一。

**3. 燃烧**：

$$C_2H_4 + 3O_2 \xrightarrow{\text{点燃}} 2CO_2 + 2H_2O$$

火焰较亮（含不完全燃烧的碳微粒）。

**4. 氧化反应**（被KMnO₄）：

乙烯被酸性KMnO₄溶液氧化，紫色褪去（生成CO₂和COOH）。

### 乙烯的检验

| 试验 | 现象 | 结论 |
|------|------|------|
| 通入溴水 | 溴水橙红色褪色 | 含不饱和键（加成） |
| 通入KMnO₄溶液 | 紫色褪色 | 含C=C或还原性基团 |

**注意**：CO₂也能使石灰水变浑浊，但不使溴水褪色（需排除干扰）。

### 乙烯的工业制法

**石油裂解**（工业主要来源）：
- 长链烃在高温下裂解为短链烃（包括乙烯）
- 温度约700-900°C，蒸汽裂解

**乙醇脱水**（实验室方法）：

$$C_2H_5OH \xrightarrow{\text{浓H₂SO₄},170°C} CH_2=CH_2\uparrow + H_2O$$

（用于实验室制备乙烯，收集后检验性质）

### 乙烯的用途

| 用途 | 说明 |
|------|------|
| 制聚乙烯 | 最大用途，塑料袋、薄膜、管道 |
| 制乙醇 | 水化反应（工业乙醇） |
| 制环氧乙烷 | 进一步制乙二醇（防冻液、涤纶）|
| 催熟剂 | 植物天然激素，促进果实成熟 |
| 制乙苯→苯乙烯 | 进一步制聚苯乙烯（PS）塑料 |

### 典型例题

**例1**：乙烯与溴水反应，写出方程式，说明现象及反应类型。

**分析**：
$$CH_2=CH_2 + Br_2 \rightarrow BrCH_2-CH_2Br$$（1,2-二溴乙烷）
现象：溴水（橙红色）→无色
反应类型：**加成反应**（不是取代，无HBr生成）

**例2**：实验室制备乙烯时，为什么温度必须控制在170°C？低于170°C会发生什么？

**分析**：
- 170°C：乙醇发生分子内脱水（消去反应）→乙烯
- 140°C：乙醇发生分子间脱水（醚化反应）→乙醚（C₂H₅OC₂H₅）
- 温度控制关键：高于140°C且尽快升至170°C

### 易错点
- 乙烯与溴水反应是**加成**，不是取代（产物无HBr，溴全部加入有机物）
- 乙烯是**气体**，密度略大于空气（不能用向下排空气法收集，用排水法）
- 聚合后的聚乙烯中无碳碳双键（π键已断裂，形成长链中的单键）
