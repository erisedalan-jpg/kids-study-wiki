# 乙烯聚合

> **一句话**：乙烯聚合是乙烯单体在催化剂、高温高压作用下，C=C双键依次打开并相互连接，形成聚乙烯长链高分子的加聚反应。
> **English**: Ethylene polymerization is an addition polymerization reaction where ethylene monomers, under catalysts and high temperature/pressure, have their C=C bonds broken and joined successively to form long-chain polyethylene macromolecules.

---

## 🎓 高中进阶

### 定义
**乙烯聚合**：乙烯（CH₂=CH₂，单体）分子中的C=C双键在一定条件下反复断裂，形成碳链，生成**聚乙烯**的反应。

$$n \text{CH}_2=\text{CH}_2 \xrightarrow{\text{催化剂，高温高压}} (-\text{CH}_2-\text{CH}_2-)_n$$

- **单体**（monomer）：CH₂=CH₂（乙烯）
- **重复单元**（repeat unit）：-CH₂-CH₂-（亚乙基）
- **聚合物**（polymer）：(-CH₂-CH₂-)ₙ，即聚乙烯（PE）
- **聚合度** n：通常数百到数万

### 加聚反应的特征

| 特征 | 说明 |
|------|------|
| 反应类型 | 加聚反应（addition polymerization）|
| 底物 | 含不饱和键（C=C）的单体 |
| 产物 | 只有高分子聚合物（无副产物）|
| 条件 | 催化剂、高温高压或引发剂 |
| 可逆性 | 通常不可逆（热固性或热塑性）|

### 聚乙烯（PE）的结构

$$\underbrace{(-\text{CH}_2-\text{CH}_2-)_n}_{\text{聚乙烯}}$$

- 长链碳骨架，全为C-C单键
- **无碳碳双键**（双键全部断裂参与成键）
- 链间靠分子间力（色散力）结合

### 聚乙烯的性质与应用

| 性质 | 说明 |
|------|------|
| 密度 | 0.91-0.97 g/cm³（低于水） |
| 熔点 | 110-135°C（热塑性，可回收）|
| 化学稳定性 | 耐酸碱，耐有机溶剂 |
| 绝缘性 | 良好的电绝缘体 |
| 透明度 | LDPE（低密度）柔软透明，HDPE（高密度）不透明 |

| 用途 | 产品 |
|------|------|
| 包装材料 | 塑料袋、食品包装膜 |
| 管道 | 给水管、燃气管 |
| 电线绝缘 | 电缆外皮 |
| 日用品 | 容器、瓶子、玩具 |

### 其他常见加聚反应

| 单体 | 聚合物 | 用途 |
|------|--------|------|
| CH₂=CH-Cl（氯乙烯）| 聚氯乙烯（PVC）| 管道、雨衣、地板 |
| CH₂=CH-C₆H₅（苯乙烯）| 聚苯乙烯（PS）| 泡沫、包装 |
| CF₂=CF₂（四氟乙烯）| 聚四氟乙烯（PTFE，特氟龙）| 不粘锅、密封件 |
| CH₂=C(CH₃)-COOCH₃（甲基丙烯酸甲酯）| 有机玻璃（PMMA）| 透明板材 |

### 加聚反应的书写规则

单体→聚合物：
1. 把单体中C=C双键改为C-C单键（打开π键）
2. 两端各用键与其他重复单元相连
3. 加方括号和下标n

$$n \text{CH}_2=\text{CH}_2 \rightarrow [-\text{CH}_2-\text{CH}_2-]_n$$

**逆推**（由聚合物推单体）：将重复单元中相邻两个碳之间的C-C键改为C=C。

### 典型例题

**例1**：聚乙烯是高分子化合物，其结构简式为(-CH₂-CH₂-)ₙ，判断其是否能使溴水褪色。

**分析**：聚乙烯中**没有碳碳双键**（乙烯聚合时双键全部断裂），化学性质与烷烃类似，**不能使溴水褪色**，不与KMnO₄反应，化学稳定性好。

**例2**：由CH₂=CHCl（氯乙烯）写出加聚产物的结构简式。

**分析**：
$$n \text{CH}_2=\text{CH-Cl} \rightarrow [-\text{CH}_2-\text{CH}(\text{Cl})-]_n$$
产物为聚氯乙烯（PVC），重复单元为 -CH₂-CHCl-。

**例3**：聚乙烯废料如何处理才更环保？

**分析**：
- 回收再利用（热塑性，可再熔融加工）
- 焚烧：会产生CO₂（温室效应）和有毒物（若含杂质）
- 降解：普通聚乙烯难以降解（几百年）；光降解PE或生物降解PE是研究方向

### 易错点
- 聚乙烯中无碳碳双键，不是乙烯简单堆叠
- 加聚反应只有一种产物（聚合物），无小分子副产物
- 热塑性聚合物可以回收重新熔融，热固性（如酚醛树脂）不能
