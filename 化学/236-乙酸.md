# 乙酸

> **一句话**：乙酸（CH₃COOH）俗称醋酸，含有羧基（-COOH），是弱酸，具有酸的通性，能与醇发生酯化反应，沸点较高（118°C），与水混溶。
> **English**: Acetic acid (CH₃COOH), commonly called vinegar acid, contains a carboxyl group (-COOH), is a weak acid with general acid properties, can undergo esterification with alcohols, has a relatively high boiling point (118°C), and is miscible with water.

| 溶解性 | 与水、乙醇任意比互溶 |

### 乙酸的结构

**结构式**：$CH_3-C(=O)-OH$ 或 $CH_3COOH$

**官能团**：羧基 -COOH（含C=O和O-H两个基团，相互影响使O-H酸性增强）

**酸性**：乙酸是弱酸（部分电离）：

$$CH_3COOH \rightleftharpoons CH_3COO^- + H^+$$（Ka = 1.8×10⁻⁵）

### 乙酸的化学性质

**1. 酸的通性**（因含-COOH，是弱酸）：

| 反应类型 | 方程式 |
|---------|--------|
| 与活泼金属 | 2CH₃COOH + 2Na → 2CH₃COONa + H₂↑ |
| 与碱（中和）| CH₃COOH + NaOH → CH₃COONa + H₂O |
| 与碱性氧化物 | 2CH₃COOH + CaO → (CH₃COO)₂Ca + H₂O |
| 与碳酸盐 | 2CH₃COOH + Na₂CO₃ → 2CH₃COONa + H₂O + CO₂↑ |
| 与碳酸氢盐 | CH₃COOH + NaHCO₃ → CH₃COONa + H₂O + CO₂↑ |

**判断酸性强弱**：乙酸 > 碳酸 > 苯酚 > 碳酸氢根（CO₃²⁻对应的酸）
- 乙酸能与Na₂CO₃和NaHCO₃反应放CO₂，说明酸性强于H₂CO₃

**2. 酯化反应（与醇）**：

$$CH_3COOH + C_2H_5OH \underset{\Delta}{\overset{\text{浓H}_2\text{SO}_4}{\rightleftharpoons}} CH_3COOC_2H_5 + H_2O$$

（乙酸乙酯，详见"[[250-酯化反应|酯化反应]]"）

**3. 不与溴水/KMnO₄反应**（无不饱和键，化学稳定）

### 食醋与冰醋酸

| 类型 | 乙酸含量 | 用途 |
|------|---------|------|
| 食醋 | 3-9%（质量分数）| 调味品 |
| 工业乙酸 | 高浓度 | 化工原料 |
| 冰醋酸 | ~100%（纯乙酸）| 低温固化（冬天呈冰状）|

### 乙酸的工业制备

| 方法 | 反应 |
|------|------|
| 甲醇羰基化 | CH₃OH + CO → CH₃COOH（主流工业方法）|
| 乙醛氧化 | CH₃CHO + O₂ → CH₃COOH |
| 发酵法 | 乙醇经醋酸菌氧化（传统）|

### 乙酸的用途

| 用途 | 说明 |
|------|------|
| 食品工业 | 食醋、食品防腐剂（醋酸钠）|
| 化工 | 制乙酸乙酯、醋酸纤维素、PVA |
| 溶剂 | 工业溶剂 |
| 医药 | 水杨酸合成（阿司匹林前体）|

### 典型例题

**例1**：如何用实验证明乙酸的酸性强于碳酸？

**分析**：向Na₂CO₃溶液中加入乙酸：
$$2CH_3COOH + Na_2CO_3 \rightarrow 2CH₃COONa + H_2O + CO_2\uparrow$$
若产生气泡（CO₂），说明乙酸酸性强于H₂CO₃（强酸制弱酸）。

**例2**：乙酸与乙醇的酯化反应为可逆反应，如何提高乙酸乙酯的产率？

**分析**（勒夏特列原理）：
- 增大反应物浓度：加入过量乙醇或乙酸
- 减小生成物浓度：将乙酸乙酯（沸点77°C，低于乙酸118°C和乙醇78°C）蒸馏移走
- 加入浓H₂SO₄：吸水（减少H₂O浓度），加快速率，使平衡正向移动

### 易错点
- 乙酸是弱酸（不完全电离），不是强酸
- 乙酸能与NaHCO₃反应放CO₂（说明酸性 > H₂CO₃）；苯酚不能与NaHCO₃反应（酸性 < H₂CO₃）
- 冰醋酸是纯乙酸，低温固化，不是冰和醋酸的混合物
