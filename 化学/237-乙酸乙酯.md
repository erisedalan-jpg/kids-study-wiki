# 乙酸乙酯

> **一句话**：乙酸乙酯（CH₃COOC₂H₅）是乙酸与乙醇发生酯化反应的产物，是最简单的羧酸酯，有果香气味，可用于香料和溶剂，在碱性条件下可水解。
> **English**: Ethyl acetate (CH₃COOC₂H₅) is the product of esterification between acetic acid and ethanol, the simplest carboxylate ester, with a fruity aroma, used as a flavoring agent and solvent, and hydrolyzable under alkaline conditions.

## 🎓 高中进阶

### 基本信息

| 项目 | 数据 |
|------|------|
| 化学式 | CH₃COOC₂H₅ 或 CH₃COOCH₂CH₃ |
| 分子量 | 88 g/mol |
| 官能团 | 酯基 -COO- |
| 状态 | 无色液体，有果香气味（水果糖香）|
| 沸点 | 77.1°C |
| 密度 | 0.900 g/mL（比水轻）|
| 溶解性 | 难溶于水，溶于有机溶剂 |

### 乙酸乙酯的结构

$$CH_3-C(=O)-O-CH_2-CH_3$$

**酯基结构**：-C(=O)-O-（C=O双键+C-O-C）

### 乙酸乙酯的制备（酯化反应）

$$CH_3COOH + C_2H_5OH \underset{\Delta}{\overset{\text{浓H}_2\text{SO}_4}{\rightleftharpoons}} CH_3COOC_2H_5 + H_2O$$

**实验条件**：
- 催化剂：浓硫酸（浓H₂SO₄）
- 温度：加热（约70-80°C）
- 可逆反应（产率一般约65%）

**收集方法**（实验室）：
- 蒸馏收集沸点77°C的馏分
- 馏出液用饱和Na₂CO₃溶液洗涤（除乙酸和乙醇）
- 分液，有机层干燥，即为乙酸乙酯

**洗涤用饱和Na₂CO₃溶液的原因**：
- Na₂CO₃与乙酸反应：CH₃COOH+Na₂CO₃→CH₃COONa（溶于水，除乙酸）
- 乙醇溶于水（乙醇-Na₂CO₃盐析）或直接与Na₂CO₃水溶液分离（除乙醇）
- 乙酸乙酯不与Na₂CO₃反应，浮在上层，分液取有机层

### 乙酸乙酯的水解

**酸性水解（可逆）**：

$$CH_3COOC_2H_5 + H_2O \underset{\Delta}{\overset{\text{稀H}^+}{\rightleftharpoons}} CH_3COOH + C_2H_5OH$$

**碱性水解（不可逆，皂化反应）**：

$$CH_3COOC_2H_5 + NaOH \xrightarrow{\Delta} CH_3COONa + C_2H_5OH$$

碱性水解不可逆（乙酸盐在碱性中不与乙醇反应）

### 标记实验（同位素追踪）

酯化反应的反应机理——**酸失去-OH，醇失去H**（酸失醇得HO）：

$$CH_3COO\mathbf{H} + HOC_2H_5 \rightarrow CH_3COOC_2H_5 + H_2O$$

**说明**：水中的O来自醇（-OH中的O），可用¹⁸O标记证明。

### 乙酸乙酯的用途

| 用途 | 说明 |
|------|------|
| 香料 | 水果糖、薄荷糖的香料成分 |
| 溶剂 | 油漆、指甲油的溶剂 |
| 有机合成 | 医药中间体 |
| 萃取剂 | 萃取水中的有机物 |

### 典型例题

**例1**：实验室制备乙酸乙酯，混合液中加入浓H₂SO₄的目的是什么？

**分析**：浓H₂SO₄的作用有两个：
1. **催化剂**：加快酯化反应速率
2. **吸水剂**：吸收生成的水，使平衡正向移动（提高产率）

**例2**：分离乙酸乙酯与乙醇和乙酸混合物，应采用哪些步骤？

**分析**：
1. 加入饱和Na₂CO₃溶液振荡：
   - 乙酸与Na₂CO₃反应→Na₂CO₃水层（乙酸被除去）
   - 乙醇溶于水层（乙醇水溶性好）
   - 乙酸乙酯浮于上层（有机层）
2. 分液：分离有机层（乙酸乙酯）
3. 干燥（无水Na₂SO₄）
4. 蒸馏（77°C收集馏分）→纯净乙酸乙酯

**例3**：乙酸乙酯加NaOH水溶液加热，写出方程式。

**分析**（碱性水解，皂化）：
$$CH_3COOC_2H_5 + NaOH \xrightarrow{\Delta} CH_3COONa + C_2H_5OH$$

### 易错点
- 酯化反应可逆，产率<100%（需浓H₂SO₄催化并吸水提高产率）
- 碱性水解（皂化）不可逆，产率接近100%
- 乙酸乙酯比水轻（密度0.9），在分液漏斗上层（有机层）
