# 乙醇氧化

> **一句话**：乙醇可被催化氧化（Cu或Ag催化，空气中O₂）生成乙醛，也可被强氧化剂（如KMnO₄、K₂Cr₂O₇）进一步氧化为乙酸，这是醇的重要化学性质。
> **English**: Ethanol can be catalytically oxidized (by Cu or Ag catalyst with O₂ in air) to form acetaldehyde, and further oxidized by strong oxidants (like KMnO₄ or K₂Cr₂O₇) to acetic acid—an important chemical property of alcohols.

## 🎓 高中进阶

### 定义
**乙醇的氧化**：乙醇中羟基（-OH）的α-碳（连OH的碳）失去氢原子，氧化程度增加，生成醛或酸的过程。

### 催化氧化（温和氧化）

**条件**：铜（Cu）或银（Ag）催化，加热

**方程式**：

$$2CH_3CH_2OH + O_2 \xrightarrow{\text{Cu/Ag}, \Delta} 2CH_3CHO + 2H_2O$$

**产物**：乙醛（CH₃CHO，又称醋醛）

**催化剂机制（铜做催化剂）**：
1. $2Cu + O_2 \xrightarrow{\Delta} 2CuO$（铜被氧化为黑色CuO）
2. $CuO + CH_3CH_2OH \rightarrow Cu + CH_3CHO + H_2O$（CuO氧化乙醇，Cu再生）
3. 总反应：$2CH_3CH_2OH + O_2 \xrightarrow{Cu} 2CH_3CHO + 2H_2O$（Cu量不变，是催化剂）

**实验现象**：
- 铜丝放入乙醇，加热至红热后插入乙醇中，反复进行
- 铜丝表面颜色变化：变黑（CuO）→变红（Cu再生）
- 产物：乙醛（有刺激性气味）

### 乙醇的深度氧化

**被酸性KMnO₄氧化（强氧化）**：
$$C_2H_5OH + 2KMnO_4 \rightarrow (\text{多步}) \rightarrow CH_3COOH + MnO_2 + K_2CO_3 + ...$$

（产物复杂，高中了解即可）现象：紫色KMnO₄溶液褪色。

**被K₂Cr₂O₇（重铬酸钾）氧化**：

$$3C_2H_5OH + 2K_2Cr_2O_7 + 8H_2SO_4 \rightarrow 3CH_3COOH + 2Cr_2(SO_4)_3 + 2K_2SO_4 + 11H_2O$$

现象：橙色（Cr₂O₇²⁻）→绿色（Cr³⁺）
**应用**：酒驾检测（呼气测试仪）：

驾驶员呼气中的乙醇使重铬酸钾（橙色）变为绿色，颜色变化的程度反映乙醇浓度。

### 燃烧（完全氧化）

$$C_2H_5OH + 3O_2 \xrightarrow{\text{点燃}} 2CO_2 + 3H_2O$$

（蓝色火焰，无烟）

### 氧化产物的比较

| 氧化程度 | 产物 | 状态 |
|---------|------|------|
| 轻度（催化）| 乙醛（CH₃CHO）| 无色液体，有刺激气味 |
| 较深（KMnO₄/K₂Cr₂O₇）| 乙酸（CH₃COOH）| 无色液体，酸味 |
| 完全（燃烧）| CO₂ + H₂O | 气体+液体 |

### 乙醛的进一步氧化（银镜反应，了解）

乙醛可被银氨溶液氧化（银镜反应）：

$$CH_3CHO + 2[Ag(NH_3)_2]^+ + 2OH^- \rightarrow CH_3COO^- + 2Ag\downarrow + 4NH_3 + H_2O$$

现象：试管内壁出现光亮银镜（工业镀银原理，也用于检测醛基）。

### 典型例题

**例1**：铜丝在空气中加热变黑后，插入乙醇中变回红色，写出两步反应方程式并解释。

**分析**：
- 第1步（铜被氧化）：$2Cu + O_2 \xrightarrow{\Delta} 2CuO$（Cu→黑色CuO）
- 第2步（CuO被乙醇还原）：$CuO + CH_3CH_2OH \rightarrow Cu + CH_3CHO + H_2O$（CuO→红色Cu）
- 净效果：Cu是催化剂，乙醇被氧化为乙醛

**例2**：酒驾检测为何使用K₂Cr₂O₇（重铬酸钾）而不是KMnO₄？

**分析**：
- K₂Cr₂O₇颜色变化明显：橙色→绿色（Cr³⁺），颜色对比强烈，便于定量判断
- KMnO₄虽也会褪色（紫→无色），但可能受其他还原性物质干扰
- 实际酒驾仪常用燃料电池（电化学法），更精确

### 易错点
- 铜催化氧化乙醇：Cu是催化剂，净反应中Cu不消耗（但中间有CuO生成）
- 乙醇催化氧化产物是乙醛（CH₃CHO），不是乙酸（乙醛再氧化才得乙酸）
- 酒驾检测变色是乙醇→乙酸，Cr⁶⁺→Cr³⁺（橙→绿），不是Cr→黑色
