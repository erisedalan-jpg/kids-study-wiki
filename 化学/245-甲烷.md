# 甲烷

> **一句话**：甲烷（CH₄）是最简单的有机物，天然气的主要成分，正四面体结构，化学性质稳定，可燃烧和发生取代反应。
> **English**: Methane (CH₄) is the simplest organic compound and the main component of natural gas, with a regular tetrahedral structure, chemically stable, capable of combustion and substitution reactions.

### 基本信息

| 项目 | 数据 |
|------|------|
| 化学式 | CH₄ |
| 分子量 | 16 g/mol |
| 结构 | 正四面体，C在中心，4个H在顶点 |
| 键角 | H-C-H = 109°28'（正四面体角）|
| 状态 | 无色无味气体 |
| 密度 | 0.717 g/L（比空气轻）|
| 溶解性 | 难溶于水 |
| 沸点 | -161.5°C |

### 甲烷的结构

**结构式**：H-C(-H)(-H)-H

**结构特点**：
- C为**sp³杂化**（正四面体构型）
- 4个C-H键等价，键角相等（109.5°）
- 无极性（正负电荷中心重合）→ **非极性分子**
- 所有键为σ键（单键，旋转自由）

### 甲烷的化学性质

**1. 燃烧（放热反应）**：

$$CH_4 + 2O_2 \xrightarrow{\text{点燃}} CO_2 + 2H_2O \quad \Delta H = -890 \text{ kJ/mol}$$

现象：燃烧呈蓝色火焰，生成CO₂和H₂O。

**不完全燃烧**（氧气不足）：
$$2CH_4 + 3O_2 \xrightarrow{\text{点燃}} 2CO + 4H_2O$$（生成CO）

**2. 取代反应（与卤素，光照条件）**：

详见"[[246-甲烷取代反应|甲烷取代反应]]"词条。

$$CH_4 + Cl_2 \xrightarrow{\text{光照}} CH_3Cl + HCl$$（第一步）

**3. 热分解（高温）**：

$$CH_4 \xrightarrow{1500°C} C + 2H_2$$（制炭黑/氢气）

**4. 与水蒸气重整**：

$$CH_4 + H_2O \xrightarrow{\text{催化剂}} CO + 3H_2$$（制合成气）

**5. 不反应的情况**：
- 不与酸、碱反应
- 不与强氧化剂（KMnO₄）反应
- 不使溴水褪色

### 甲烷的来源与用途

| 来源/存在 | 说明 |
|---------|------|
| 天然气 | 主要成分（约80-90%），化石能源 |
| 沼气 | 有机物发酵产生（CH₄约60-70%）|
| 可燃冰（甲烷水合物）| 深海/冻土层，巨大能源储量 |
| 煤矿（瓦斯）| 矿井中的甲烷，危险气体 |
| 生物代谢 | 反刍动物（牛）和沼泽产生 |

| 用途 | 说明 |
|------|------|
| 燃料 | 天然气发电、家用燃气 |
| 化工原料 | 制合成气、乙炔、炭黑 |
| 温室气体 | CH₄的温室效应是CO₂的25倍 |

### 典型例题

**例1**：收集一瓶甲烷气体，如何检验其纯度（防止爆炸）？

**分析**：
- 收集时用排水法（CH₄难溶于水）或向下排空气法（CH₄比空气轻）
- 纯度检验：用点燃法，点燃后声音"噗"一声，火焰持续稳定=纯净；若有爆鸣声=不纯（含空气）

**例2**：CH₄和CH₂Cl₂是否有同分异构体？

**分析**：
- CH₂Cl₂（二氯甲烷）：C为sp³杂化，结构为CCl₂H₂，正四面体形，只有1种结构（由于4个方向等价，H-C-Cl键角相同）
- CH₂Cl₂没有同分异构体（2Cl和2H可以在四面体上的任意位置，但都等价）

### 易错点
- 甲烷是无味气体，天然气中的"气味"来自加入的臭味剂（乙硫醇）
- 煤矿瓦斯爆炸是CH₄与O₂的混合气爆炸，不是单纯CH₄燃烧
- CH₄与Cl₂的取代反应需要**光照**，不是加热（与某些反应不同）
