# 苯的取代反应

> **一句话**：苯的取代反应是苯环上的H原子被其他原子或基团替换，苯环骨架保持不变，主要包括卤代反应、硝化反应和磺化反应，需要特定催化剂。
> **English**: Benzene substitution reactions involve replacing H atoms on the benzene ring with other atoms or groups while keeping the ring intact, including halogenation, nitration, and sulfonation, each requiring specific catalysts.

---

## 🎓 高中进阶

### 定义
**苯的取代反应**：苯环上的H原子被其他原子或基团（-Cl、-NO₂、-SO₃H等）替换，且**苯环结构保持不变**的反应。

与烷烃取代反应的区别：苯的取代需要催化剂（路易斯酸），不需要光照，且苯环不破坏。

### 三类重要取代反应

**一、卤代反应（卤化）**

$$C_6H_6 + Cl_2 \xrightarrow{\text{FeCl}_3(\text{无水})} C_6H_5Cl + HCl$$

| 参数 | 说明 |
|------|------|
| 条件 | 无水FeCl₃（路易斯酸催化）|
| 产物 | 氯苯（C₆H₅Cl）+ HCl气体 |
| 现象 | 放热，产生HCl气体（白雾）|
| 用途 | 制农药、有机合成 |

**FeCl₃的作用**：催化Cl₂极化，增强Cl⁺的亲电性。

$$C_6H_6 + Br_2 \xrightarrow{\text{FeBr}_3} C_6H_5Br + HBr$$（溴苯）

**二、硝化反应**

$$C_6H_6 + HNO_3 \xrightarrow{\text{浓H}_2\text{SO}_4, 50\text{-}60°C} C_6H_5NO_2 + H_2O$$

| 参数 | 说明 |
|------|------|
| 条件 | 浓H₂SO₄催化，50-60°C水浴 |
| 试剂 | 混酸（浓HNO₃+浓H₂SO₄）|
| 产物 | 硝基苯（C₆H₅NO₂，淡黄色液体，有苦杏仁气味，有毒）+ H₂O |
| 注意 | 温度不能超过60°C，否则多硝化 |

**浓H₂SO₄的作用**：质子化HNO₃，产生NO₂⁺（硝酰基），增强亲电性。

**三、磺化反应**

$$C_6H_6 + H_2SO_4 \xrightarrow{\text{浓H}_2\text{SO}_4, \Delta} C_6H_5SO_3H + H_2O$$

| 参数 | 说明 |
|------|------|
| 条件 | 浓H₂SO₄，加热 |
| 产物 | 苯磺酸（可溶于水）+ H₂O |
| 可逆性 | 可逆（加水、稀酸可水解脱磺）|
| 用途 | 制合成洗涤剂原料 |

### 苯的取代与甲烷取代的比较

| 比较项 | 苯与Cl₂ | 甲烷与Cl₂ |
|--------|---------|----------|
| 条件 | FeCl₃催化 | 光照 |
| 产物类型 | 芳香取代（保留苯环）| 自由基取代 |
| 副产物 | HCl | HCl |
| 可多取代吗？| 可（二氯苯等）| 可（多氯甲烷）|

### 苯的衍生物的命名

**一取代**（一个H被取代）：命名直接加官能团名：
- C₆H₅-Cl = 氯苯
- C₆H₅-NO₂ = 硝基苯
- C₆H₅-CH₃ = 甲苯

**二取代**（两个H被取代，需标位置）：
- 邻（o-）：1,2位
- 间（m-）：1,3位
- 对（p-）：1,4位

### 典型例题

**例1**：苯与液溴（Br₂）在FeBr₃催化下反应，写出方程式并说明现象。

**分析**：
$$C_6H_6 + Br_2 \xrightarrow{\text{FeBr}_3} C_6H_5Br + HBr\uparrow$$

现象：
- 反应放热
- 生成HBr气体（白雾）
- 液层分层（溴苯与水不混溶，沉于下层，比水重）
- 与苯混入溴水不同（苯混入溴水只有物理萃取，无HBr产生）

**例2**：苯的硝化反应温度为何控制在50-60°C？

**分析**：
- <50°C：反应速率太慢，效率低
- 50-60°C：适宜温度，主要生成一硝基苯（C₆H₅NO₂）
- >60°C：产生过多二硝基苯（C₆H₄(NO₂)₂），产品不纯且危险

**例3**：苯与溴水、苯与液溴（有FeBr₃）的区别是什么？

**分析**：
- 苯+溴水（无催化剂）：**萃取**（物理过程），Br₂溶于苯，溴水褪色（非化学反应）
- 苯+Br₂（有FeBr₃）：**取代反应**（化学过程），生成溴苯+HBr

### 易错点
- 苯与溴水不反应，是苯萃取Br₂（物理溶解）
- 苯的硝化需用混酸（浓HNO₃+浓H₂SO₄），不是单用浓HNO₃
- 苯的卤代需FeCl₃（或FeBr₃），不是光照条件
