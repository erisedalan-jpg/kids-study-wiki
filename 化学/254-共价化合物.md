# 共价化合物

> **一句话**：共价化合物是只含共价键（无离子键）的化合物，通常由非金属元素组成，以分子形式存在，熔沸点一般低于离子化合物。
> **English**: A covalent compound contains only covalent bonds (no ionic bonds), is typically composed of nonmetals, exists as discrete molecules, and generally has lower melting/boiling points than ionic compounds.

---

## 🎓 高中进阶

### 定义
**共价化合物**：微粒间只有共价键（无离子键）的化合物。其基本微粒为**分子**（分子晶体）或共价网络结构（原子晶体）。

与**离子化合物**的根本区别：无离子键，不含离子（NH₄⁺等特殊除外）。

### 共价化合物的特征

| 特征 | 说明 |
|------|------|
| 组成元素 | 多为非金属-非金属，或非金属-氢 |
| 存在形式 | 分子（分子晶体）或原子晶体 |
| 熔沸点 | 分子晶体：较低；原子晶体（金刚石、SiO₂）：极高 |
| 导电性 | 固态和液态通常不导电 |
| 溶解性 | "相似相溶"原则 |

### 常见共价化合物

| 化合物 | 类型 | 晶体类型 |
|--------|------|---------|
| H₂O | 极性共价化合物 | 分子晶体 |
| HCl | 极性共价化合物 | 分子晶体（气体） |
| NH₃ | 极性共价化合物 | 分子晶体 |
| CO₂ | 非极性共价化合物 | 分子晶体 |
| CH₄ | 非极性共价化合物 | 分子晶体 |
| SiO₂ | 共价化合物 | 原子晶体（高熔点） |
| 金刚石(C) | 共价单质 | 原子晶体 |

### 共价化合物 vs 离子化合物

| 比较项 | 共价化合物 | 离子化合物 |
|--------|-----------|----------|
| 成键类型 | 只含共价键 | 含离子键（可也含共价键） |
| 基本微粒 | 分子 | 离子 |
| 熔沸点 | 较低（分子晶体） | 较高 |
| 导电性 | 通常不导电 | 熔融或溶液导电 |
| 典型例子 | H₂O, HCl, CO₂ | NaCl, KOH, CaCl₂ |

### 如何判断是否为共价化合物

**方法**：
1. 看元素组成：全部为非金属 → 可能是共价化合物
2. 看结构：含离子（如NH₄⁺）→ 离子化合物
3. 看是否导电：熔融后不导电 → 共价化合物

**注意例外**：
- HCl虽不导电（纯液态/气态），但溶于水电离 → 溶液导电
- NH₄Cl含NH₄⁺（离子），属于离子化合物，尽管N和H均为非金属

### 典型例题

**例1**：下列物质中，属于共价化合物的是：①NaCl ②HCl ③NH₄Cl ④H₂O ⑤Na₂O

**分析**：
- NaCl：含离子键，离子化合物 ✗
- HCl：H-Cl共价键，无离子，共价化合物 ✓
- NH₄Cl：含NH₄⁺和Cl⁻，离子化合物 ✗
- H₂O：O-H共价键，无离子，共价化合物 ✓
- Na₂O：含离子键，离子化合物 ✗

**例2**：为什么HCl（气态）不导电，而盐酸（HCl溶液）能导电？

**分析**：
- 纯HCl是共价化合物，无自由离子，不导电
- HCl溶于水时发生电离：HCl → H⁺ + Cl⁻（完全电离）
- 溶液中有自由移动的H⁺和Cl⁻，能导电

### 易错点
- 共价化合物熔融时不导电，但其水溶液可能导电（酸电离）
- NH₄Cl不是共价化合物（含NH₄⁺离子键）
- 原子晶体（SiO₂、金刚石）含共价键，熔点极高（和普通共价化合物不同）
