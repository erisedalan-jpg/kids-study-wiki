# 副族元素

> **一句话**：副族元素（B族）是元素周期表中ⅢB～ⅦB族及第Ⅷ族的元素，均为金属，也称过渡金属，价电子包括d轨道电子。
> **English**: Transition elements (B groups) are metals in groups IIIB through VIII of the periodic table, also called transition metals, with valence electrons including d-orbital electrons.

---

## 🎓 高中进阶

### 定义
副族元素位于元素周期表第4～7周期的中部，共包含：ⅢB、ⅣB、ⅤB、ⅥB、ⅦB（各1列）和第Ⅷ族（3列），以及ⅠB、ⅡB（共10列）。广义上ⅠB和ⅡB也归入过渡金属。

### 副族特征

| 族 | 代表元素 | 特征 |
|----|----------|------|
| ⅢB | Sc, Y, La | 较活泼过渡金属 |
| ⅣB | Ti, Zr, Hf | Ti密度小，强度高 |
| ⅤB | V, Nb, Ta | 用于合金 |
| ⅥB | Cr, Mo, W | W熔点最高（3422℃） |
| ⅦB | Mn, Tc, Re | Mn用于钢铁冶炼 |
| 第Ⅷ族 | Fe, Co, Ni; Ru, Rh, Pd; Os, Ir, Pt | 铁族、铂族，催化剂 |
| ⅠB | Cu, Ag, Au | 货币金属，导电性好 |
| ⅡB | Zn, Cd, Hg | Zn活泼，Hg常温液态 |

### 副族元素的共同特征

1. **均为金属**：具有金属光泽，导电导热
2. **多种化合价**：d轨道电子也可参与成键（Fe有+2和+3价）
3. **易形成配合物**：如 [Fe(SCN)₃]、[Cu(NH₃)₄]²⁺
4. **催化活性**：Fe（合成氨）、Pt（制硫酸）、Ni（加氢）
5. **颜色多样**：离子和化合物常有颜色（Fe²⁺浅绿，Fe³⁺棕黄）

### 重要副族元素（高中范围）

**铁（Fe，第Ⅷ族）**：
- Fe²⁺：浅绿色，还原性较强
- Fe³⁺：棕黄色，氧化性较强
- Fe + 2HCl → FeCl₂ + H₂↑
- 2Fe + 3Cl₂ → 2FeCl₃（Cl₂过量时生成+3价）

**铜（Cu，ⅠB族）**：
- 红色金属，导电性仅次于Ag
- Cu + 2H₂SO₄(浓) → CuSO₄ + SO₂↑ + 2H₂O
- 蓝色CuSO₄溶液

### 典型例题

**例1**：Fe与稀盐酸反应和Fe与Cl₂反应的产物有何不同？为什么？

**分析**：
- Fe + 2HCl → FeCl₂ + H₂↑（稀HCl氧化性弱，Fe被氧化为+2价）
- 2Fe + 3Cl₂ → 2FeCl₃（Cl₂氧化性强，Fe被氧化为+3价）
- 氧化剂氧化能力越强，Fe被氧化到越高价态。

**例2**：检验溶液中是否含有Fe³⁺，用什么试剂？

**分析**：用硫氰酸钾（KSCN）溶液。若溶液变血红色，则含有Fe³⁺：
Fe³⁺ + 3SCN⁻ → Fe(SCN)₃（血红色）

### 易错点
- 副族元素均为金属，不存在非金属副族元素
- 第Ⅷ族是3列（8、9、10列），不是1列
- Fe与足量稀盐酸反应生成FeCl₂（+2价），不是FeCl₃
- Cu不与稀盐酸/稀硫酸反应（Cu排在H后面）
