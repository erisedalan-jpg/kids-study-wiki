# 原子序数

> **一句话**：原子序数等于元素原子核中的质子数（核电荷数），是元素在周期表中的排列依据，也是区分不同元素的根本标志。
> **English**: The atomic number equals the number of protons in an atom's nucleus, serving as the basis for arranging elements in the periodic table and the fundamental identifier of each element.

---

## 🎓 高中进阶

### 定义
原子序数（Z）= 核电荷数 = 质子数 = （中性原子）核外电子数

对于中性原子：**质子数 = 电子数**
对于离子：质子数 ≠ 电子数（阳离子质子数>电子数；阴离子质子数<电子数）

### 原子序数与结构关系

| 量 | 符号 | 关系 |
|----|------|------|
| 原子序数 | Z | Z = 质子数 |
| 质子数 | Z | 决定元素种类 |
| 中子数 | N | N = 质量数(A) - Z |
| 质量数 | A | A = Z + N |
| 核外电子数 | | 中性原子 = Z |

**符号表示**：$_{Z}^{A}X$（如 $_{6}^{12}C$，Z=6，A=12，N=6）

### 原子序数与周期表位置

通过原子序数可推断元素位置：
1. 逐步减去各周期元素数（2, 8, 8, 18, 18, 32…）
2. 余数对应该元素在该周期的位置（族）

**示例**：原子序数35（Br）
- 35 - 2(第1) - 8(第2) - 8(第3) - 18(第4) = -1（超过第4周期末Kr(36)之前1位）
- Br位于第4周期第ⅦA族

### 常见元素原子序数

| 元素 | 符号 | Z |
|------|------|---|
| 氢 | H | 1 |
| 碳 | C | 6 |
| 氮 | N | 7 |
| 氧 | O | 8 |
| 钠 | Na | 11 |
| 铝 | Al | 13 |
| 硅 | Si | 14 |
| 氯 | Cl | 17 |
| 钙 | Ca | 20 |
| 铁 | Fe | 26 |

### 典型例题

**例1**：某元素原子序数为16，写出其原子结构示意图，判断其位置和主要化学性质。

**分析**：
- Z=16，为S（硫）
- 电子排布：2, 8, 6（第1层2，第2层8，第3层6）
- 位置：第3周期第ⅥA族
- 非金属性较强，最高化合价+6，最低-2；可形成H₂S、SO₂、SO₃、H₂SO₄

**例2**：比较原子序数11（Na）和12（Mg）的金属性强弱。

**分析**：同为第3周期，Na(Z=11)在Mg(Z=12)左侧，金属性Na > Mg。
- Na与冷水剧烈反应：2Na + 2H₂O → 2NaOH + H₂↑
- Mg只与热水缓慢反应

### 易错点
- 原子序数 = 质子数，不是质量数
- 同一元素不同同位素：原子序数相同，质量数不同
- 离子的电子数 ≠ 质子数（Z）
