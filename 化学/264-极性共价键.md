# 极性共价键

> **一句话**：极性共价键是不同原子间形成的共价键，共用电子对偏向电负性较强的原子，使该原子略显负电性，另一原子略显正电性。
> **English**: A polar covalent bond forms between different atoms, where the shared electron pair is displaced toward the more electronegative atom, creating partial charges on each atom.

---

## 🎓 高中进阶

### 定义
**极性共价键**：由电负性不同的原子共用电子对形成的键。电子对偏向电负性大（吸电子能力强）的原子，该原子带部分负电（δ⁻），另一原子带部分正电（δ⁺）。

### 形成条件

| 条件 | 说明 |
|------|------|
| 不同非金属原子成键 | 电负性不同导致电子对偏移 |
| 电负性差 | 0 < 差值 < 1.7（大于1.7通常形成离子键） |

### 常见极性共价键

| 键 | 电负性较大的原子 | 电子对偏向 |
|----|----------------|-----------|
| H-F | F（电负性最强） | F |
| H-O（水、乙醇） | O | O |
| H-N（氨、胺） | N | N |
| H-Cl | Cl | Cl |
| C-H | C（略大于H） | C |
| C-O | O | O |
| C=O | O | O |

### 极性键与分子极性的关系

含极性键的分子不一定是极性分子：

| 分子 | 极性键 | 分子极性 | 原因 |
|------|--------|---------|------|
| HCl | H-Cl（极性） | 极性分子 | 极性键，结构不对称 |
| H₂O | O-H（极性）×2 | 极性分子 | V形结构，极性不对称 |
| CO₂ | C=O（极性）×2 | 非极性分子 | 直线型，极性相互抵消 |
| CCl₄ | C-Cl（极性）×4 | 非极性分子 | 正四面体，极性抵消 |
| NH₃ | N-H（极性）×3 | 极性分子 | 三角锥形，不对称 |

### 极性键的表示

用部分电荷符号（δ⁺、δ⁻）或箭头标示电子对偏移方向：

$$\delta^+ \text{H} - \text{Cl}^{\delta^-}$$（H-Cl键，电子对偏向Cl）

### 典型例题

**例1**：下列分子中，含极性共价键但分子为非极性的是：①HF ②CO₂ ③H₂O ④CCl₄

**分析**：
- HF：含极性键，且为极性分子 ✗
- CO₂：含极性键（C=O），但直线型结构，极性抵消 → **非极性分子** ✓
- H₂O：含极性键，V形，极性不抵消，为极性分子 ✗
- CCl₄：含极性键（C-Cl），正四面体，极性抵消 → **非极性分子** ✓

答：CO₂和CCl₄

**例2**：判断O-H键和N-H键的极性大小（电子对偏移程度）。

**分析**：电负性 O(3.44) > N(3.04) > H(2.20)
- O-H键极性 > N-H键极性（O对电子的吸引力更强）
- 因此H₂O比NH₃中O-H键极性更强，水中氢键比NH₃中氢键强，水的沸点高于NH₃

### 易错点
- 含极性键的分子不一定是极性分子（CO₂、CCl₄是反例）
- 非极性分子不一定只含非极性键（CO₂只含C=O极性键，却是非极性分子）
- C-H键是极性键（C和H电负性不同，但差别较小）
