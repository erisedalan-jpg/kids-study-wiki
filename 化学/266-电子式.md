# 电子式

> **一句话**：电子式（Lewis结构式）是用"·"或"×"表示原子最外层（价层）电子，同时表示化学键（共用电子对）和孤对电子的结构式。
> **English**: A Lewis structure (electron dot structure) uses dots to represent valence electrons of atoms, showing both bonding pairs (chemical bonds) and lone pairs.

---

## 🎓 高中进阶

### 定义
**电子式**：在元素符号周围用"·"或"×"标出价电子（最外层电子），并用两个相邻电子点表示共用电子对（化学键）。

### 书写规则
1. 确定各原子价电子数
2. 确定中心原子和端原子
3. 先写骨架（单键连接）
4. 按"八隅体规则"（H为2）分配电子
5. 用"—"表示共用电子对（一条线=两个共用电子）

### 常见分子电子式

| 分子 | 电子式描述 | 键型 |
|------|-----------|------|
| H₂ | H:H（1对共用电子） | 单键 |
| Cl₂ | :Cl:Cl:（1对共用+各3对孤对） | 单键 |
| HCl | H:Cl:（1对共用） | 单键 |
| H₂O | H:O:H（2对共用，O有2对孤对） | 2单键 |
| NH₃ | H:N:H（3对共用，N有1对孤对）H | 3单键 |
| N₂ | :N:::N:（3对共用=三键） | 三键 |
| CO₂ | O::C::O（C与各O各2对共用=双键） | 2双键 |
| CH₄ | H-C(-H)₃（4对共用，无孤对） | 4单键 |

### 离子化合物的电子式
离子化合物中，阴离子用方括号括起，标注电荷：

**NaCl**：[Na]⁺ [:Cl:]⁻

**NaOH**：[Na]⁺ [:O:H]⁻（OH⁻内部O-H共价键）

**Na₂O**：[Na]⁺ [:O:]²⁻ [Na]⁺

### 电子式与结构式的关系
- **电子式**：显示所有价电子（孤对电子+共用电子对）
- **结构式**：只显示共价键（用"—"代替共用电子对，省略孤对电子）

| 分子 | 电子式 | 结构式 |
|------|--------|--------|
| H₂O | H:O:H + 两对孤对 | H-O-H |
| CO₂ | 各含双键 | O=C=O |
| N₂ | 三对共用 | N≡N |

### 典型例题

**例1**：写出H₂O的电子式。

**分析**：
- O有6个价电子，H各有1个价电子
- O与各H共用1对电子（形成O-H键），O还剩2对孤对电子
- 结果：H周围2个电子（满足），O周围8个电子（满足八隅体）

**例2**：写出CO₂的电子式并判断是否满足八隅体规则。

**分析**：
- C有4个价电子，O各有6个价电子
- C与各O形成双键（各2对共用电子），各O还有2对孤对电子
- C：4（双键）×2=8电子 ✓；O各：2（双键）+4（孤对）=8电子 ✓

### 易错点
- NH₃的N上有1对孤对电子（不要忘记），不是只有3对共用电子
- H₂O的O上有2对孤对电子
- 离子化合物电子式中，阴离子整体用方括号
- CO₂是直线型（C与两O各形成双键），不是折线型
