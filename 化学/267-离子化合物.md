# 离子化合物

> **一句话**：离子化合物是由阴阳离子通过离子键结合形成的化合物，熔融或溶于水后能导电，熔沸点通常较高。
> **English**: An ionic compound is formed by the combination of cations and anions through ionic bonds; it conducts electricity when molten or dissolved in water, with relatively high melting and boiling points.

---

## 🎓 高中进阶

### 定义
**离子化合物**：由**阳离子**和**阴离子**通过**离子键**结合而成的化合物。基本微粒是离子，通常以晶体（离子晶体）形式存在，无单独"分子"。

### 形成离子化合物的常见组合

| 组合类型 | 举例 |
|---------|------|
| 活泼金属 + 活泼非金属 | NaCl（Na⁺ + Cl⁻）|
| 活泼金属 + 酸根离子 | Na₂SO₄, CaCO₃ |
| 铵盐（NH₄⁺ + 阴离子） | NH₄Cl, (NH₄)₂SO₄ |
| 碱（金属离子 + OH⁻） | NaOH, Ca(OH)₂ |
| 过氧化物 | Na₂O₂（Na⁺ + O₂²⁻）|

### 离子化合物的特征

| 特征 | 说明 |
|------|------|
| 基本微粒 | 阴阳离子（无分子） |
| 晶体类型 | 离子晶体 |
| 熔沸点 | 较高（离子键强） |
| 导电性 | 固态不导电；熔融或溶液导电 |
| 硬度 | 较硬，但脆 |
| 溶解性 | 多数溶于水（少数不溶，如BaSO₄）|

### 常见离子化合物

| 化合物 | 阳离子 | 阴离子 | 备注 |
|--------|--------|--------|------|
| NaCl | Na⁺ | Cl⁻ | 食盐 |
| KNO₃ | K⁺ | NO₃⁻ | 硝酸钾（化肥） |
| CaCO₃ | Ca²⁺ | CO₃²⁻ | 石灰石 |
| Na₂SO₄ | Na⁺ | SO₄²⁻ | 硫酸钠 |
| NaOH | Na⁺ | OH⁻ | 强碱（含O-H共价键+Na-O离子键） |
| NH₄Cl | NH₄⁺ | Cl⁻ | 铵盐（NH₄⁺内部是共价键） |
| Na₂O₂ | Na⁺ | O₂²⁻ | 含O-O非极性键 |

### 判断是否为离子化合物的方法

1. 看是否含活泼金属离子（Na⁺, K⁺, Ca²⁺, Mg²⁺等）
2. 看是否含铵根NH₄⁺
3. 实验：熔融后导电 → 离子化合物

### 离子化合物中也可含共价键

| 化合物 | 离子键 | 共价键（离子内部） |
|--------|--------|----------------|
| NaOH | Na⁺-OH⁻ | O-H（OH⁻内部） |
| Na₂SO₄ | Na⁺-SO₄²⁻ | S-O（SO₄²⁻内部） |
| NH₄Cl | NH₄⁺-Cl⁻ | N-H（NH₄⁺内部） |
| Na₂O₂ | Na⁺-O₂²⁻ | O-O（O₂²⁻内部）|

### 典型例题

**例1**：NaOH中含有哪些化学键？

**分析**：
- Na⁺与OH⁻之间：**离子键**（NaOH整体的成键）
- OH⁻内部O-H：**极性共价键**
- 结论：NaOH含离子键和极性共价键

**例2**：固态NaCl能导电吗？NaCl溶液能导电吗？为什么？

**分析**：
- 固态NaCl：Na⁺和Cl⁻被固定在晶格中，不能自由移动 → **不导电**
- NaCl溶液：Na⁺和Cl⁻在水中自由移动 → **能导电**
- 熔融NaCl：离子可流动 → **能导电**

### 易错点
- 离子化合物固态不导电（离子不能移动），液态或溶液才导电
- NaOH既含离子键又含共价键，仍是离子化合物（只要有离子键就是离子化合物）
- 铵盐（如NH₄Cl）是离子化合物，尽管组成元素全是非金属
