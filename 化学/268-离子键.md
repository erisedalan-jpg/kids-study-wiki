# 离子键

> **一句话**：离子键是阴阳离子之间通过静电引力形成的化学键，存在于离子化合物中，无方向性和饱和性。
> **English**: An ionic bond is a chemical bond formed by the electrostatic attraction between cations and anions, found in ionic compounds, with no directionality or saturation.

---

## 🎓 高中进阶

### 定义
**离子键**：阴离子和阳离子之间通过**静电引力**（库仑力）结合而成的化学键。

形成过程：活泼金属将电子转移给活泼非金属 → 形成阳离子和阴离子 → 阴阳离子通过静电引力结合。

### 形成条件

| 条件 | 说明 |
|------|------|
| 常见组合 | 活泼金属（ⅠA、ⅡA族）+ 活泼非金属（ⅦA、ⅥA族） |
| 电负性差 | 两原子电负性差较大（通常>1.7） |
| 举例 | Na与Cl, K与Br, Ca与O |

### 离子键的特点

| 特点 | 说明 |
|------|------|
| 无方向性 | 离子向各方向的吸引力相同 |
| 无饱和性 | 一个离子可吸引多个异号离子 |
| 强度 | 键能较大，熔点沸点高 |
| 导电性 | 熔融或溶于水后可导电 |

### NaCl的形成（典型例子）

$$\text{Na} - e^- \rightarrow \text{Na}^+$$
$$\text{Cl} + e^- \rightarrow \text{Cl}^-$$
$$\text{Na}^+ + \text{Cl}^- \rightarrow \text{NaCl（离子键）}$$

NaCl晶体中，每个Na⁺被6个Cl⁻包围，每个Cl⁻被6个Na⁺包围（八面体结构），无单独NaCl分子。

### 含离子键的常见物质

| 物质 | 阳离子 | 阴离子 |
|------|--------|--------|
| NaCl | Na⁺ | Cl⁻ |
| KOH | K⁺ | OH⁻ |
| Na₂SO₄ | Na⁺ | SO₄²⁻ |
| CaCl₂ | Ca²⁺ | Cl⁻ |
| Na₂O₂ | Na⁺ | O₂²⁻ |
| NH₄Cl | NH₄⁺ | Cl⁻ |

注：KOH、Na₂SO₄中，OH⁻、SO₄²⁻内部是共价键，但整体作为阴离子与阳离子之间是离子键。

### 电子式写法（离子键）
离子化合物的电子式中，每个离子用方括号括起并标注电荷：

NaCl：$[\text{Na}]^+ [\ddot{\text{C}}\text{l}:]^-$（方括号法）

### 典型例题

**例1**：下列物质中含有离子键的是：①NaCl ②HCl ③H₂O ④KOH ⑤NH₃

**分析**：
- NaCl：Na⁺和Cl⁻，含离子键 ✓
- HCl：H-Cl共价键，无离子键 ✗
- H₂O：O-H共价键，无离子键 ✗
- KOH：K⁺和OH⁻，含离子键 ✓（OH⁻内部O-H为共价键）
- NH₃：N-H共价键，无离子键 ✗

**例2**：Na₂O₂（过氧化钠）中含有哪些化学键？

**分析**：Na₂O₂由2个Na⁺和O₂²⁻（过氧根）构成。
- Na⁺与O₂²⁻之间：**离子键**
- O₂²⁻内部O-O：**非极性共价键**

### 易错点
- 离子键无方向性，不能说"NaCl分子"（NaCl是离子晶体，无分子）
- NH₄⁺中NH₄⁺与阴离子之间是离子键，NH₄⁺内部是共价键
- Na₂O₂含离子键和非极性共价键（O₂²⁻内部）
