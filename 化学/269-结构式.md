# 结构式

> **一句话**：结构式是用短线（—）表示共价键，显示分子中各原子的连接方式和键型（单键、双键、三键）的化学式，但省略孤对电子。
> **English**: A structural formula uses lines to represent covalent bonds, showing how atoms are connected and the bond types (single, double, triple), but omits lone pairs.

---

## 🎓 高中进阶

### 定义
**结构式**：在分子式基础上，用"—"（单键）、"="（双键）、"≡"（三键）表示各原子间化学键的式子。比分子式更详细，比电子式更简洁。

### 常见分子结构式

| 分子 | 分子式 | 结构式 |
|------|--------|--------|
| 氢气 | H₂ | H-H |
| 水 | H₂O | H-O-H |
| 氨 | NH₃ | H-N(-H)-H（三角锥型） |
| 甲烷 | CH₄ | H-C(-H)(-H)-H（正四面体） |
| 乙烷 | C₂H₆ | CH₃-CH₃ |
| 乙烯 | C₂H₄ | CH₂=CH₂ |
| 乙炔 | C₂H₂ | CH≡CH |
| 二氧化碳 | CO₂ | O=C=O |
| 氮气 | N₂ | N≡N |
| 盐酸 | HCl | H-Cl |
| 乙醇 | C₂H₅OH | CH₃-CH₂-OH |
| 乙酸 | CH₃COOH | CH₃-C(=O)-OH |

### 结构式与其他化学式的对比

| 类型 | 特征 | 信息量 | 例（乙醇）|
|------|------|--------|----------|
| 分子式 | 只显示各原子数 | 最少 | C₂H₆O |
| 结构式 | 显示键的连接 | 中等 | CH₃-CH₂-OH |
| 电子式 | 显示所有价电子 | 最多 | 含孤对电子 |
| 结构简式 | 省略C-H键 | 有机化学常用 | CH₃CH₂OH |

### 有机化合物结构式特点

有机化学中常用**结构简式**（省略部分C-H键）：

| 结构式 | 结构简式 |
|--------|----------|
| H-C(-H)(-H)-OH | CH₃OH（甲醇） |
| CH₃-CH₂-OH | C₂H₅OH 或 CH₃CH₂OH |
| CH₃-C(=O)-OH | CH₃COOH |

**键线式**（碳骨架式）：只画碳链折线，省略C和H，用折点代表C，端点连H：

乙烷（C₂H₆）：简单短横线段两端

### 典型例题

**例1**：写出乙烯（CH₂=CH₂）的结构式，并说明其化学键类型。

**分析**：
```
H   H
|   |
C = C
|   |
H   H
```
- C=C：非极性双键（π键+σ键）
- C-H：极性单键（σ键）
- 共4个C-H极性键，1个C=C非极性双键

**例2**：乙炔（C₂H₂）和乙烯（C₂H₄）结构式的区别是什么？

**分析**：
- 乙炔：H-C≡C-H（碳碳三键，直线型）
- 乙烯：CH₂=CH₂（碳碳双键，平面型）
- 乙炔C≡C含1σ+2π键，乙烯C=C含1σ+1π键
- 三键比双键更短、键能更大，但π键更活泼（可加成）

### 易错点
- 结构式省略孤对电子，但不省略所有化学键
- 结构简式省略C-H键和部分结构，但不省略官能团（如-OH、-COOH）
- O=C=O是直线型，不是折线
