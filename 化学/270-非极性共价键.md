# 非极性共价键

> **一句话**：非极性共价键是相同原子间形成的共价键，共用电子对不偏向任何一方，电荷分布对称，键的极性为零。
> **English**: A nonpolar covalent bond forms between identical atoms, where the shared electron pair is not displaced toward either atom, resulting in symmetric charge distribution and zero bond polarity.

---

## 🎓 高中进阶

### 定义
**非极性共价键**：相同原子（或电负性相同的原子）之间形成的共价键，共用电子对处于两原子正中间，不偏移，因此无极性（偶极矩为零）。

### 形成条件

| 条件 | 举例 |
|------|------|
| 同种非金属单质 | H₂（H-H）、Cl₂（Cl-Cl）、O₂（O=O）、N₂（N≡N） |
| 相同原子成键 | 金刚石（C-C）、Na₂O₂（O-O） |
| 电负性相同原子 | 极少见（化合物中同种原子成键） |

### 常见非极性共价键

| 物质 | 非极性键 | 键型 |
|------|---------|------|
| H₂ | H-H | 单键 |
| Cl₂ | Cl-Cl | 单键 |
| O₂ | O=O | 双键 |
| N₂ | N≡N | 三键 |
| 金刚石 | C-C | 单键（无数个） |
| Na₂O₂ | O-O | 单键（O₂²⁻内部） |
| H₂O₂ | O-O | 单键（H-O-O-H结构） |

### 非极性键与极性键的对比

| 比较项 | 非极性共价键 | 极性共价键 |
|--------|------------|-----------|
| 成键原子 | 相同原子 | 不同原子 |
| 电子对偏移 | 不偏移（居中） | 偏向电负性强的原子 |
| 极性 | 无（偶极矩=0） | 有 |
| 举例 | H-H, N≡N | H-Cl, O-H |

### 键能与稳定性

| 键 | 键能（kJ/mol） | 说明 |
|----|--------------|------|
| H-H | 436 | 较强 |
| Cl-Cl | 243 | 中等 |
| O=O | 498 | 双键较强 |
| N≡N | 945 | 极强，N₂极稳定 |

### 典型例题

**例1**：判断Na₂O₂中的化学键类型。

**分析**：Na₂O₂含有：
- Na⁺与O₂²⁻之间：**离子键**
- O₂²⁻内部O-O：**非极性共价键**（两个O原子相同，电子对居中）

**例2**：H₂O₂（双氧水）结构式为H-O-O-H，其中哪些键是极性键，哪些是非极性键？

**分析**：
- O-H键：O和H不同，电子对偏向O → **极性共价键**（2个）
- O-O键：相同原子，电子对居中 → **非极性共价键**（1个）

### 易错点
- 非极性共价键不只存在于单质中，Na₂O₂、H₂O₂等化合物中也有
- N₂中是三键（N≡N），不是三条单键
- 非极性键的分子不一定是非极性分子（如O₂是非极性分子，但CCl₄虽只含极性键也是非极性分子）
