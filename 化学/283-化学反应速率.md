# 化学反应速率

> **一句话**：化学反应速率是单位时间内反应物或生成物浓度的变化量，衡量反应进行的快慢，单位为mol/(L·s)或mol/(L·min)。
> **English**: Chemical reaction rate measures how fast a reaction proceeds, defined as the change in concentration of reactants or products per unit time, in units of mol/(L·s) or mol/(L·min).

---

## 🎓 高中进阶

### 定义

$$v = \frac{\Delta c}{\Delta t}$$

- **v**：反应速率 [mol/(L·min) 或 mol/(L·s)]
- **Δc**：浓度变化量 (mol/L)
- **Δt**：时间间隔 (min 或 s)

**注意**：速率始终取**正值**，无论反应物（浓度减小）还是生成物（浓度增大）。

### 对于不同物质，速率之比 = 化学计量数之比

对于反应：$a\text{A} + b\text{B} \rightarrow c\text{C} + d\text{D}$

$$\frac{v(\text{A})}{a} = \frac{v(\text{B})}{b} = \frac{v(\text{C})}{c} = \frac{v(\text{D})}{d}$$

**示例**：N₂ + 3H₂ → 2NH₃

若 v(N₂) = 0.1 mol/(L·min)，则：
- v(H₂) = 3×0.1 = 0.3 mol/(L·min)
- v(NH₃) = 2×0.1 = 0.2 mol/(L·min)

### 影响反应速率的因素

| 因素 | 增大速率 | 减小速率 |
|------|---------|---------|
| 浓度 | 增大反应物浓度 | 减小浓度 |
| 温度 | 升高温度 | 降低温度 |
| 催化剂 | 加入正催化剂 | 加入负催化剂 |
| 压强 | 增大压强（气体反应） | 降低压强 |
| 固体表面积 | 研磨成粉末 | 大块固体 |

**温度规律**：温度每升高10°C，反应速率约增大2-4倍（范特霍夫规则）。

### 速率与温度的定量关系（阿伦尼乌斯方程，了解）

$$k = A e^{-E_a/(RT)}$$

- Ea：活化能（越小，反应越快）
- 催化剂降低活化能，从而加快速率

### 典型例题

**例1**：在2L容器中，反应2SO₂ + O₂ → 2SO₃，前2 min内SO₃浓度从0增加到0.4 mol/L，求各物质的反应速率。

**分析**：
- v(SO₃) = Δc/Δt = 0.4/2 = 0.2 mol/(L·min)
- 化学计量数：SO₂:O₂:SO₃ = 2:1:2
- v(SO₂) = v(SO₃) = 0.2 mol/(L·min)
- v(O₂) = v(SO₃)/2 = 0.1 mol/(L·min)

**例2**：温度从25°C升至55°C（升高30°C），如果每升高10°C速率增大2倍，原速率增大几倍？

**分析**：升高30°C = 升高3个10°C
速率增大 = 2³ = 8倍

### 易错点
- 速率不能用纯固体或纯液体（水）表示（浓度不变）
- 不同物质速率值不同，需要比较时必须除以化学计量数
- 增大压强相当于增大浓度（适用于气体），纯固体液体体积不变
