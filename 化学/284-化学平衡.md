# 化学平衡

> **一句话**：化学平衡是可逆反应正反应速率等于逆反应速率时的动态平衡状态，各组分的浓度不再随时间变化，但反应并未停止。
> **English**: Chemical equilibrium is a dynamic state in a reversible reaction where the forward and reverse reaction rates are equal, so the concentrations of all components remain constant over time, though the reaction continues.

---

## 🎓 高中进阶

### 定义
**化学平衡**：在密闭体系中，可逆反应正反应速率（v正）= 逆反应速率（v逆），各组分浓度不再随时间变化的状态。

**本质**：动态平衡（不是静止，而是正逆反应以相同速率持续进行）。

### 化学平衡的特征（"等、动、定、变"）

| 特征 | 说明 |
|------|------|
| **等**（速率相等） | v正 = v逆 ≠ 0 |
| **动**（动态平衡） | 正逆反应持续进行，不是静止 |
| **定**（浓度不变） | 各组分浓度保持不变（不一定等于）|
| **变**（条件改变） | 改变温度/浓度/压强，平衡可能移动 |

### 化学平衡常数（K）

对于反应：$a\text{A}(g) + b\text{B}(g) \rightleftharpoons c\text{C}(g) + d\text{D}(g)$

$$K_c = \frac{c(\text{C})^c \cdot c(\text{D})^d}{c(\text{A})^a \cdot c(\text{B})^b}$$（浓度平衡常数）

**特点**：
- K只与温度有关（温度改变→K改变）
- K越大，正反应趋势越大，平衡时产物越多
- 纯固体和纯液体不写入K表达式

### 判断是否达到平衡的标准

| 判断依据 | 是否平衡 |
|---------|---------|
| v正 = v逆（同一物质方向相同） | ✓ 平衡 |
| 各组分浓度不随时间变化 | ✓ 平衡 |
| 混合物组成不变（颜色、压强不变） | ✓ 平衡（适用时）|
| v正(A) = v逆(B) | ✗ 不能直接判断（需化学计量数关系）|
| 混合气体平均摩尔质量不变 | 看情况（正逆两侧气体mol数相等则不能判断）|

### 影响平衡的因素

| 改变条件 | 平衡移动方向 |
|---------|------------|
| 增大某反应物浓度 | 向正反应方向 |
| 减小某生成物浓度 | 向正反应方向 |
| 升温 | 向吸热反应方向 |
| 增大压强 | 向气体分子数减小的方向 |
| 加入催化剂 | 不移动（同等加快正逆速率）|

### 典型例题

**例1**：N₂(g) + 3H₂(g) ⇌ 2NH₃(g)在500°C达平衡。下列哪种操作能使v正增大但平衡不移动？

**分析**：
- 加催化剂→v正和v逆均增大，v正=v逆，**平衡不移动**（符合）✓
- 升温→v正和v逆均增大，但v逆增大更多（吸热方向），平衡向逆方向移动 ✗
- 增大N₂浓度→v正增大，v逆不变，平衡正向移动 ✗

**例2**：在2SO₂(g) + O₂(g) ⇌ 2SO₃(g)中，若SO₂的转化率为80%，初始n(SO₂)=0.2 mol，n(O₂)=0.1 mol（总体积1 L），计算平衡时各组分浓度及K。

**分析**：
| | SO₂ | O₂ | SO₃ |
|----|-----|-----|-----|
| 初始（mol/L） | 0.2 | 0.1 | 0 |
| 变化 | -0.16 | -0.08 | +0.16 |
| 平衡 | 0.04 | 0.02 | 0.16 |

$$K = \frac{(0.16)^2}{(0.04)^2 \times (0.02)} = \frac{0.0256}{3.2 \times 10^{-5}} = 800 \text{ L/mol}$$

### 易错点
- 平衡时v正 = v逆，但不为零（动态平衡）
- K只与温度有关，浓度和压强改变不改变K
- 催化剂不移动平衡（不改变K）
- 固体（如C、Fe）浓度不写入K表达式
