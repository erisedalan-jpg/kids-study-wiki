# 化学电源

> **一句话**：化学电源是利用氧化还原反应将化学能转化为电能的装置，包括一次电池（不可充电）和二次电池（可充电蓄电池）等。
> **English**: A chemical power source is a device that converts chemical energy to electrical energy via redox reactions, including primary batteries (non-rechargeable) and secondary batteries (rechargeable accumulators).

---

## 🎓 高中进阶

### 定义
**化学电源**（电池）：基于原电池原理，将化学能转化为电能的实用装置。

**分类**：

| 类型 | 特点 | 举例 |
|------|------|------|
| 一次电池（原电池）| 不可充电，用完即弃 | 干电池、锌汞电池 |
| 二次电池（蓄电池）| 可充电重复使用 | 铅蓄电池、锂离子电池 |
| 燃料电池 | 持续供给燃料，连续发电 | 氢氧燃料电池 |

### 主要化学电源

**一、干电池（锌锰电池）**

| 项目 | 内容 |
|------|------|
| 负极 | 锌筒（Zn） |
| 正极 | 石墨棒（C） |
| 电解质 | NH₄Cl和ZnCl₂（糊状）|
| 负极反应 | Zn - 2e⁻ → Zn²⁺ |
| 正极反应 | 2MnO₂ + 2NH₄⁺ + 2e⁻ → Mn₂O₃ + 2NH₃ + H₂O |
| 电压 | 约1.5 V |

**二、铅蓄电池（铅酸电池）**

| 项目 | 放电（原电池）| 充电（电解池）|
|------|------------|-------------|
| 负极/阴极 | Pb-2e⁻+SO₄²⁻→PbSO₄ | PbSO₄+2e⁻→Pb+SO₄²⁻ |
| 正极/阳极 | PbO₂+4H⁺+SO₄²⁻+2e⁻→PbSO₄+2H₂O | PbSO₄+2H₂O-2e⁻→PbO₂+4H⁺+SO₄²⁻ |
| 电解质 | H₂SO₄溶液 | H₂SO₄溶液 |
| 电压 | 约2 V/单体 | — |
| 用途 | 汽车启动电池 | — |

**三、锂离子电池**

| 项目 | 内容 |
|------|------|
| 负极 | 石墨（碳材料，存储Li⁺）|
| 正极 | LiCoO₂等含锂金属氧化物 |
| 电解质 | 有机电解液 |
| 特点 | 轻质高能（能量密度大）、无记忆效应 |
| 应用 | 手机、笔记本、电动汽车 |

**四、氢氧燃料电池**

| 项目 | 内容 |
|------|------|
| 负极 | H₂（氢气）侧 |
| 正极 | O₂（氧气）侧 |
| 负极反应（酸性）| H₂-2e⁻→2H⁺ |
| 正极反应（酸性）| O₂+4H⁺+4e⁻→2H₂O |
| 总反应 | 2H₂+O₂→2H₂O |
| 优点 | 无污染，效率高（约60-80%）|

### 电源选择的考虑因素

| 因素 | 说明 |
|------|------|
| 能量密度 | 锂离子 > 镍氢 > 铅酸 |
| 循环寿命 | 锂离子 > 镍氢 > 铅酸 |
| 安全性 | 铅酸最安全，锂离子需要保护电路 |
| 成本 | 铅酸最便宜 |
| 环保 | 燃料电池最环保（产物为水）|

### 典型例题

**例1**：铅蓄电池放电后，H₂SO₄浓度如何变化？说明原因。

**分析**：
- 放电时：
  - 负极：Pb + SO₄²⁻ → PbSO₄ + 2e⁻（消耗SO₄²⁻）
  - 正极：PbO₂ + 4H⁺ + SO₄²⁻ + 2e⁻ → PbSO₄ + 2H₂O（消耗H⁺和SO₄²⁻）
- 综合：放电过程消耗H₂SO₄，溶液密度减小，H₂SO₄浓度降低

**例2**：氢氧燃料电池的优点是什么？为什么尚未普及？

**分析**：
- 优点：效率高（直接化学能→电能），产物为水（无污染），理论上可连续供电
- 制约：H₂储存困难（高压储氢有安全隐患）、H₂生产成本高、催化剂（Pt）昂贵

### 易错点
- 一次电池放完电后不能充电恢复（化学反应不可逆）
- 蓄电池充电时，原来的负极接外电源负极（成为电解池的阴极），原正极接正极（成为阳极）
- 锂电池正极是含锂氧化物，不是锂金属（锂金属与电解液反应危险）
