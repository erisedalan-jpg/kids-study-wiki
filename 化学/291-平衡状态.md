# 平衡状态

> **一句话**：平衡状态是可逆反应中正反应速率等于逆反应速率（v正=v逆），各组分浓度不再随时间变化的稳定状态。
> **English**: The equilibrium state is the stable condition in a reversible reaction where the forward rate equals the reverse rate (v_forward = v_reverse), and the concentration of each component no longer changes with time.

---

## 🎓 高中进阶

### 定义
**平衡状态（化学平衡）**：在密闭体系中，可逆反应的正反应速率等于逆反应速率，各组分的浓度保持不变的状态。

**注意**：这是**动态平衡**，反应持续进行，只是宏观上看不到变化。

### 平衡状态的判断标准

**必要条件**：v正 = v逆（正逆反应速率相等）

**充分条件（宏观不变）**：
- 各组分浓度不变
- 混合物颜色不变（若有颜色差异）
- 气体压强不变（反应前后气体分子数不等的情况）
- 平均摩尔质量不变（总质量不变，气体总物质的量不变）

### 判断是否达到平衡的常见题型

对于：$a\text{A}(g) + b\text{B}(g) \rightleftharpoons c\text{C}(g) + d\text{D}(g)$

| 判断依据 | 是否为平衡标志 | 原因 |
|---------|-------------|------|
| v正(A) = v逆(A)（消耗A=生成A）| ✓ | 正逆速率相等 |
| v正(A) : v逆(B) = a : b（消耗A与消耗B之比）| ✗ | 都是正反应方向 |
| 混合气密度不变 | 恒容时通常✗（总质量恒定，密度必不变）| 不能判断 |
| 各组分浓度不随时间变化 | ✓ | 直接表示宏观不变 |
| 气体总压强不变 | 只有a+b ≠ c+d时✓ | 否则总压可能始终不变 |
| 混合气总物质的量不变 | 只有a+b ≠ c+d时✓ | |

### 平衡状态的建立过程

以N₂ + 3H₂ ⇌ 2NH₃为例：

| 时间 | v正 | v逆 | 平衡状态 |
|------|-----|-----|---------|
| t=0 | 最大 | 0 | 未平衡 |
| 0<t<t₁ | 减小 | 增大 | 未平衡（正>逆）|
| t=t₁ | v₁ | v₁ | **平衡建立** |
| t>t₁ | v₁ | v₁ | 平衡维持 |

### 不同起始条件下的平衡

**等效平衡**：不同的起始量，但按比例换算后等同，最终达到相同的平衡状态：

**恒容等效**：如果将所有组分按化学计量比全部换算为同侧反应物，若总量相同，则达到等效平衡。

### 典型例题

**例1**：在2NO₂(g)⇌N₂O₄(g)中，下列哪些能表明达到平衡？

**分析**：
- ① 气体密度不变：总质量不变+恒容体积不变→密度恒为常数→**不能判断** ✗
- ② 混合气颜色不变：NO₂为红棕色，N₂O₄无色；颜色不变→c(NO₂)不变→**可以判断** ✓
- ③ 消耗NO₂的速率等于生成N₂O₄的速率（比=2:1）：这两个都是正反应→**不能判断** ✗
- ④ 容器内压强不变（恒温恒容）：反应前后气体mol数不同（2→1）→压强不变代表总mol数不变→**可以判断** ✓

**例2**：在恒压密闭容器中，H₂(g)+I₂(g)⇌2HI(g)，下列能表明达到平衡的是？

**分析**：此反应两侧气体mol数相等（1+1=2），恒压下体积不变，密度不变，压强不变，这些都不能判断。
- v正(H₂) = v逆(H₂)：相同物质正逆消耗相等→✓
- c(HI)/c(H₂)不变：代表浓度不变→✓

### 易错点
- 平衡时各组分浓度不一定相等（只是不随时间变化）
- "速率相等"判断中，必须是同一物质的正逆速率，或符合化学计量比的不同物质
- 恒容条件下密度必然不变（总质量/体积均不变），不能用密度判断平衡
