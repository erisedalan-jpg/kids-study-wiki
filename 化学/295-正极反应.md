# 正极反应

> **一句话**：原电池正极发生还原反应，外电路传来的电子被溶液中的氧化性物质（如H⁺或O₂）接受，从而在正极表面发生还原反应。
> **English**: At the positive electrode (cathode) of a galvanic cell, reduction occurs: electrons arriving from the external circuit are accepted by oxidizing species (such as H⁺ or O₂) at the electrode surface.

---

## 🎓 高中进阶

### 定义
**原电池正极**：
- 不活泼金属、石墨或其他较不活泼电极材料
- 发生**还原反应**（得到电子）
- 外电路电子流入正极，被溶液中的氧化性物质接受

**规则**：原电池中，活泼性**较弱**的金属（或非金属导体）为正极。

### 常见正极反应方程式

| 原电池类型 | 正极材料 | 正极反应 |
|----------|---------|---------|
| 锌铜稀H₂SO₄ | Cu | 2H⁺ + 2e⁻ → H₂↑ |
| 铁铜稀H₂SO₄ | Cu | 2H⁺ + 2e⁻ → H₂↑ |
| 锌炭盐水（中性） | C（石墨）| O₂ + 2H₂O + 4e⁻ → 4OH⁻ |
| 铅蓄电池（放电）| PbO₂ | PbO₂ + 2e⁻ + SO₄²⁻ + 4H⁺ → PbSO₄ + 2H₂O |
| 燃料电池（酸性）| O₂侧 | O₂ + 4H⁺ + 4e⁻ → 2H₂O |
| 燃料电池（碱性）| O₂侧 | O₂ + 2H₂O + 4e⁻ → 4OH⁻ |

### 正极反应的书写规律

| 溶液类型 | 正极氧化剂 | 典型正极反应 |
|---------|---------|------------|
| 酸性溶液 | H⁺ | 2H⁺ + 2e⁻ → H₂↑ |
| 中性/碱性溶液（有O₂）| O₂ | O₂+2H₂O+4e⁻→4OH⁻ |
| 含Cu²⁺溶液 | Cu²⁺ | Cu²⁺ + 2e⁻ → Cu（铜析出）|

### 各种电源的正极反应

**干电池（锌锰电池）**：
- 负极：Zn - 2e⁻ → Zn²⁺
- 正极（石墨）：2MnO₂ + 2NH₄⁺ + 2e⁻ → Mn₂O₃ + 2NH₃ + H₂O

**铅蓄电池（放电）**：
- 负极：Pb - 2e⁻ + SO₄²⁻ → PbSO₄
- **正极**：PbO₂ + 4H⁺ + SO₄²⁻ + 2e⁻ → PbSO₄ + 2H₂O

**氢氧燃料电池（酸性膜）**：
- 负极：H₂ - 2e⁻ → 2H⁺
- **正极**：O₂ + 4H⁺ + 4e⁻ → 2H₂O

### 典型例题

**例1**：在Cu-Ag-AgNO₃溶液原电池中，写出正极反应方程式，并说明正极现象。

**分析**：
- Cu比Ag活泼，Cu为负极（Cu - 2e⁻ → Cu²⁺）
- Ag为正极，溶液中Ag⁺得到电子：
- **正极**：Ag⁺ + e⁻ → Ag（Ag析出）
- 现象：Ag片质量增大（Ag析出），溶液中Ag⁺浓度减小（溶液颜色可能变浅）

**例2**：铅蓄电池放电时，正极（PbO₂）的电极反应方程式中为何含SO₄²⁻？

**分析**：
- 铅蓄电池电解质为H₂SO₄溶液（含SO₄²⁻和H⁺）
- PbO₂还原时，需要H⁺，同时生成的Pb²⁺与SO₄²⁻结合沉淀（PbSO₄不溶）
- 正极：PbO₂ + 4H⁺ + SO₄²⁻ + 2e⁻ → PbSO₄ + 2H₂O
- 这样写电荷守恒和原子守恒均满足

### 易错点
- 正极发生**还原**反应（得电子），不是氧化
- 正极不一定是金属，也可以是石墨（如干电池中的石墨棒）
- 铅蓄电池正极是PbO₂，它本身被还原为PbSO₄（不要认为正极溶解）
