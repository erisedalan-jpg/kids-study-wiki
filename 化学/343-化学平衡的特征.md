# 化学平衡的特征

> **一句话**：化学平衡是动态平衡，具有"逆、动、等、定、变"五大特征，在特定条件下可逆反应达到的最终稳定状态。
> **English**: Chemical equilibrium is a dynamic equilibrium characterized by "reversible, dynamic, equal rates, constant composition, and changeable" — the final stable state of a reversible reaction under specific conditions.

---

## 🎓 高中进阶

### 定义

**化学平衡**（chemical equilibrium）：在一定条件下，正逆反应速率相等，各组分浓度不再随时间变化的状态。

化学平衡的五大特征（"逆动等定变"）：

| 特征 | 含义 |
|------|------|
| **逆**（可逆性）| 只适用于可逆反应（⇌），不可逆反应无平衡 |
| **动**（动态性）| 正逆反应仍在进行，并非静止；$v_{\text{正}} = v_{\text{逆}} \neq 0$ |
| **等**（速率相等）| 正逆反应速率相等（同一物质的生成速率 = 消耗速率）|
| **定**（浓度不变）| 各组分浓度（摩尔分数）保持恒定，不随时间变化 |
| **变**（条件改变则移动）| 改变温度/浓度/压强，平衡可移动到新状态 |

### 公式 / 方程式

化学平衡的判断依据：

| 标志 | 是否平衡 |
|------|---------|
| 正逆反应速率相等 | 是（关键标志）|
| 各组分浓度不再变化 | 是 |
| 混合气体密度不变（恒容，各组分摩尔质量不同）| 是 |
| 混合气体平均摩尔质量不变（$\Delta n_g \neq 0$ 时）| 是 |
| 某组分的生成速率 = 其消耗速率 | 是 |
| 正反应速率 = 逆反应速率（用不同物质表示）| 需按计量比换算后才能判断 |

**非平衡标志**：
- 某物质的浓度保持不变（但这是平衡的结果，不是充分条件）
- 颜色不变（若体系有颜色物质）
- 压强不变（恒温恒容，$\Delta n_g \neq 0$ 时）

### 典型例题

**例**：对于 $\text{H}_2(g) + \text{I}_2(g) \rightleftharpoons 2\text{HI}(g)$，下列哪些是平衡标志？

(A) $v(\text{H}_2) = v(\text{HI})$ — 不能直接判断（需按计量数换算：$v(\text{H}_2) = \frac{1}{2}v(\text{HI})$ 时平衡）  
(B) $v_{\text{正}}(\text{H}_2) = v_{\text{逆}}(\text{H}_2)$ — **是**（同物质正逆速率相等）  
(C) 混合气体密度不变（恒容，总质量不变，体积不变，密度始终不变）— **不是**（始终成立，非平衡专属标志）  
(D) c(HI) 不再变化 — **是**（浓度恒定是平衡标志）

### 易错点

- 平衡是**动态**的，正逆反应速率不为零
- "某物质浓度不变"是平衡的结果，不能用某物质浓度原本就为常数的情况作为平衡依据
- "颜色不变"需确认体系有色物质，且颜色不变一定对应浓度不变
- $\Delta n_g = 0$ 的反应，恒容下总压强始终不变，不能用压强不变判断平衡

### 词汇

| 中文 | 英文 |

| 动态平衡 | dynamic equilibrium |

| 可逆反应 | reversible reaction |

### 例句

- At chemical equilibrium, the forward and reverse reaction rates are equal but not zero.（化学平衡时，正逆反应速率相等但不为零。）
- Chemical equilibrium is a dynamic state, not a static one.（化学平衡是动态状态，而非静态。）
