# 平衡移动

> **一句话**：改变影响平衡的条件（浓度、温度、压强），平衡向能减弱该改变的方向移动（勒夏特列原理）。
> **English**: When conditions affecting equilibrium (concentration, temperature, pressure) are changed, the equilibrium shifts to counteract the change (Le Chatelier's principle).

---

## 🎓 高中进阶

### 定义

**化学平衡移动**：可逆反应处于平衡状态时，若某一条件改变导致 $Q \neq K$，则正逆反应速率暂时不等，反应向使 $Q \rightarrow K$ 的方向进行，直至建立新平衡的过程。

**勒夏特列原理**（Le Chatelier's principle，1884年）：若对平衡体系施加外部影响，体系会自发向**减弱**该影响的方向移动。

### 公式 / 方程式

以 $\text{N}_2(g) + 3\text{H}_2(g) \rightleftharpoons 2\text{NH}_3(g)$，$\Delta H = -92\ \text{kJ/mol}$ 为例：

| 改变条件 | Q 与 K 的关系 | 移动方向 | 新平衡特点 |
|----------|-------------|---------|-----------|
| 增大 [N₂] | Q < K | 正向 | NH₃ 增多，[N₂] 仍高于原平衡 |
| 减小 [NH₃] | Q < K | 正向 | NH₃ 补充恢复 |
| 升温 | K 减小，Q > K | 逆向 | NH₃ 减少，K 减小 |
| 增大压强 | Q < K（气体减少方向）| 正向 | NH₃ 增多，压强仍高于原平衡 |
| 加催化剂 | Q = K（不变）| 不移动 | 更快达到相同平衡 |

### 典型例题

**例**：对于 $\text{CO}(g) + \text{H}_2\text{O}(g) \rightleftharpoons \text{CO}_2(g) + \text{H}_2(g)$，$\Delta H < 0$：

(1) 升温：平衡逆向移动，CO₂ 和 H₂ 减少，$K$ 减小  
(2) 增压（缩小体积）：$\Delta n_g = 0$，平衡不移动，各物质浓度等比增大，$K$ 不变  
(3) 加入更多 CO：Q < K，平衡正向移动，H₂O 转化率升高，CO 转化率降低

### 易错点

- 勒夏特列原理是"减弱"影响，不是"消除"影响——新平衡与原平衡不同
- 催化剂不影响平衡位置（$K$ 不变），只加快达到平衡
- 对 $\Delta n_g = 0$ 的反应，改变压强不移动平衡
- 升温必然使 $K$ 改变（放热反应 $K$ 减小，吸热反应 $K$ 增大）

## 🌐 中英对照

### 词汇

| 中文 | 英文 |
|------|------|
| 平衡移动 | equilibrium shift |
| 勒夏特列原理 | Le Chatelier's principle |
| 正向移动 | shift to the right / forward shift |
| 逆向移动 | shift to the left / reverse shift |

### 例句

- According to Le Chatelier's principle, adding more reactant shifts the equilibrium to the right.（根据勒夏特列原理，增加更多反应物使平衡向右移动。）
- A catalyst does not shift the equilibrium; it only speeds up the rate at which equilibrium is reached.（催化剂不使平衡移动，只加快达到平衡的速率。）
