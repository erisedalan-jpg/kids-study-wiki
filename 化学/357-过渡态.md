# 过渡态

> **一句话**：反应过程中能量最高的中间构型，处于反应物转化为生成物的能量"山顶"，寿命极短（~10⁻¹³ s）。
> **English**: The highest-energy intermediate structure along the reaction pathway, at the "top of the energy hill" between reactants and products, with an extremely short lifetime (~10⁻¹³ s).

---

## 🎓 高中进阶

### 定义

**过渡态**（transition state）又称**活化络合物**（activated complex）：反应物分子在有效碰撞过程中，旧键部分断裂、新键部分形成时形成的高能中间态。

特点：
- 能量最高（位于势能面的鞍点）
- 不稳定，寿命极短，无法分离
- 既可向前形成产物，也可逆向恢复反应物

### 公式 / 方程式

反应坐标（势能图）：

```
势能
  |
  |        [过渡态 TS]
  |       *
  |      / \
  |     /   \
E_a→  /     \  ← E_a(逆)
  |  /       \
  | /         \
  |/  反应物   \ 生成物
  |_________________________ 反应坐标
```

$$
E_a(\text{正}) = E(\text{TS}) - E(\text{反应物})
$$

$$
E_a(\text{逆}) = E(\text{TS}) - E(\text{生成物})
$$

$$
\Delta H = E_a(\text{正}) - E_a(\text{逆})
$$

催化剂提供新路径，形成新的过渡态（能量更低）：

$$
E_a(\text{催化}) < E_a(\text{无催化})
$$

### 典型例题

**例**：某反应 A → B 的势能图数据：反应物势能 200 kJ/mol，过渡态势能 280 kJ/mol，生成物势能 170 kJ/mol。求：

1. 正反应活化能：$E_a(\text{正}) = 280 - 200 = 80\ \text{kJ·mol}^{-1}$
2. 逆反应活化能：$E_a(\text{逆}) = 280 - 170 = 110\ \text{kJ·mol}^{-1}$
3. 反应焓变：$\Delta H = 170 - 200 = -30\ \text{kJ·mol}^{-1}$（放热）
4. 验证：$\Delta H = E_a(\text{正}) - E_a(\text{逆}) = 80 - 110 = -30\ \text{kJ·mol}^{-1}$ ✓

### 易错点

- 过渡态不是中间体（intermediate）；中间体位于能量低谷，可相对稳定存在
- 催化剂不改变反应物和生成物的能量，只改变过渡态能量
- $\Delta H$ 与 $E_a$ 无直接大小关系：可能 $\Delta H < 0$ 但 $E_a$ 很大
