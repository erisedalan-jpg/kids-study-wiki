# 主量子数

> **一句话**：用正整数 n（n=1,2,3…）表示的量子数，决定电子所在能层和轨道的能量大小。
> **English**: A positive integer n (n=1,2,3…) that determines the electron's principal energy level and orbital size.

---

## 🎓 高中进阶

### 定义
**主量子数**（n）是四个量子数中最基本的一个，取正整数值 n = 1, 2, 3, 4, …，对应电子层 K, L, M, N, …

- n 决定轨道的**能量**（主要）和**大小**
- n 越大，轨道能量越高，电子平均距核越远
- 对氢原子：轨道能量 $E_n = -\frac{13.6 \text{ eV}}{n^2}$

### 取值与含义

| n | 能层符号 | 含义 |
|---|---------|------|
| 1 | K | 最低能层，最靠近核 |
| 2 | L | 第二能层 |
| 3 | M | 第三能层 |
| 4 | N | 第四能层（含 4f，镧系） |
| … | … | … |

### 限制条件
- n ≥ 1（正整数）
- 角量子数 l 的范围由 n 决定：0 ≤ l ≤ n−1
  - n=1：l 只能为 0（仅 s）
  - n=2：l 可为 0,1（s 和 p）
  - n=3：l 可为 0,1,2（s、p、d）
  - n=4：l 可为 0,1,2,3（s、p、d、f）

### 对各性质的影响

| 性质 | 与 n 的关系 |
|------|-----------|
| 轨道大小 | 随 n 增大而增大 |
| 轨道能量 | 随 n 增大而升高 |
| 每层轨道数 | n² 个 |
| 每层最多电子 | 2n² 个 |

### 典型例题
**题**：主量子数 n=3 时，允许的角量子数 l 有哪些值？共有多少个原子轨道？
**解**：l = 0, 1, 2（即 3s、3p、3d）
轨道数：1 + 3 + 5 = 9 个（= n² = 3² = 9 ✓）

### 易错点
- n 只取正整数，从 1 开始（不是 0）
- 周期表的周期数 = 最外层主量子数（s 区和 p 区元素）
- d 区和 f 区元素的最高 n 层不一定是电子最多的能层
