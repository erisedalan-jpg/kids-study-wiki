# 能级

> **一句话**：同一能层内按轨道类型（s、p、d、f）细分的能量亚层，同一能级的轨道能量相同。
> **English**: Energy sublevels within an energy level, classified by orbital type (s, p, d, f); orbitals within the same sublevel have equal energy.

---

## 🎓 高中进阶

### 定义
**能级**（亚层/能量子层）是对电子能量的更精细划分。在同一**能层**（n 相同）内，不同类型的轨道（s、p、d、f）具有不同的能量，这些能量差异形成不同的能级。

- 能层（主量子数 n）：粗分，n=1,2,3,4…
- 能级（s,p,d,f）：细分，在同一能层内 E(s) < E(p) < E(d) < E(f)

### 能级能量顺序

单电子原子：能量仅由 n 决定，同一能层内各能级简并。

多电子原子（考虑电子间排斥后）：
$$E_{ns} < E_{np} < E_{nd} < E_{nf}$$

**能量近似顺序（填充顺序）：**
1s < 2s < 2p < 3s < 3p < **4s** < **3d** < 4p < 5s < 4d < 5p < 6s < 4f < 5d …

注意：4s 能量低于 3d（能量交错）

### 各能层能级汇总

| 能层 n | 能级 | 轨道数 | 总轨道数 |
|--------|------|--------|---------|
| 1 | 1s | 1 | 1 |
| 2 | 2s, 2p | 1, 3 | 4 |
| 3 | 3s, 3p, 3d | 1, 3, 5 | 9 |
| 4 | 4s, 4p, 4d, 4f | 1, 3, 5, 7 | 16 |

### 典型例题
**题**：按能量由低到高排列以下能级：3d、4s、3p、2p。
**解**：2p < 3p < 4s < 3d（注意 4s 低于 3d 的能量交错）

### 易错点
- 能层 ≠ 能级：能层是 n=1,2,3…；能级是 1s、2p、3d 等
- 多电子原子中 4s < 3d（填充时先填 4s），但失电子时先失 4s 电子（过渡金属）
- 同一能级内的多个轨道能量相等（简并）
