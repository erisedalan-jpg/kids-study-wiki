# 角量子数

> **一句话**：决定原子轨道形状的量子数 l，取值 0,1,2,3…对应 s,p,d,f 轨道。
> **English**: The quantum number l that determines the shape of an atomic orbital; l=0,1,2,3 correspond to s,p,d,f orbitals.

---

## 🎓 高中进阶

### 定义
**角量子数**（l，也称副量子数或方位量子数）是描述电子角动量大小的量子数，决定轨道的**形状**（及对多电子原子的能量贡献）。

取值范围：0 ≤ l ≤ n−1（整数），即对给定 n，l = 0, 1, 2, …, n−1

### l 值与轨道对应

| l 值 | 轨道类型 | 形状 | 轨道数（2l+1）|
|------|---------|------|--------------|
| 0 | s | 球形 | 1 |
| 1 | p | 哑铃形 | 3 |
| 2 | d | 花瓣形 | 5 |
| 3 | f | 复杂 | 7 |

### 与主量子数的关系

| n | 允许的 l 值 | 对应轨道类型 |
|---|-----------|------------|
| 1 | 0 | 1s |
| 2 | 0, 1 | 2s, 2p |
| 3 | 0, 1, 2 | 3s, 3p, 3d |
| 4 | 0, 1, 2, 3 | 4s, 4p, 4d, 4f |

### 对能量的影响
- 单电子原子（氢）：能量仅由 n 决定，l 不影响能量（轨道简并）
- 多电子原子：相同 n、不同 l 的轨道能量不同：E(s) < E(p) < E(d) < E(f)

### 典型例题
**题**：n=3 时，角量子数 l 可取哪些值？各对应几个轨道？
**解**：l = 0（3s，1个）、l = 1（3p，3个）、l = 2（3d，5个）
共 9 个轨道。

### 易错点
- l 不能等于 n，必须小于 n（l ≤ n−1）
- l 决定**形状**，不直接决定能量（多电子原子中通过"穿透效应"间接影响能量）
- 每种 l 值对应的轨道数为 2l+1
