# 量子力学模型

> **一句话**：用波函数和概率分布描述电子运动状态的原子模型，取代了玻尔的固定轨道模型。
> **English**: A model that describes electron behavior using wave functions and probability distributions, replacing Bohr's fixed-orbit model.

---

## 🎓 高中进阶

### 定义
量子力学模型（也称**现代原子模型**）是基于量子力学原理建立的原子结构模型。其核心思想：
- 电子的运动状态无法用经典力学描述
- 电子的位置只能以**概率分布**（即电子云）来表示
- 每个电子的状态由四个量子数完全确定

### 与玻尔模型的对比

| 特征 | 玻尔模型 | 量子力学模型 |
|------|---------|------------|
| 电子轨道 | 固定圆形轨道 | 概率分布（轨道） |
| 适用范围 | 仅氢原子 | 所有原子 |
| 电子能量 | 量子化（正确） | 量子化（正确） |
| 轨道形状 | 圆形 | s/p/d/f 各异 |
| 理论基础 | 半经典 | 完整量子力学 |

### 核心概念
- **波函数 ψ**：描述电子运动状态的数学函数；|ψ|² 表示电子在某点出现的概率密度
- **原子轨道**：波函数 ψ 所对应的电子运动状态，每个轨道由三个量子数 (n, l, m) 确定
- **电子云**：|ψ|² 的空间图像，点密集处表示电子出现概率大

### 典型例题
**题**：为什么说量子力学模型比玻尔模型更能正确描述多电子原子？
**解**：玻尔模型仅成功解释氢原子光谱，无法处理电子间相互作用；量子力学通过薛定谔方程考虑了电子间排斥，能解释多电子原子的光谱和化学键。

### 易错点
- "原子轨道"不是电子运动的真实轨迹，而是波函数对应的空间状态
- |ψ|² 是概率密度，不是电子本身
