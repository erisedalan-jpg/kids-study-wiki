# 晶体的物理性质

> **一句话**：晶体的熔沸点、硬度、导电性、溶解性等物理性质由晶体类型（决定力的类型和强度）决定，是判断晶体类型的重要依据。
> **English**: Physical properties of crystals—melting/boiling point, hardness, conductivity, solubility—are determined by crystal type (which determines the type and strength of bonding forces) and are key criteria for identifying crystal types.

## 🎓 高中进阶

### 四种晶体物理性质综合比较

| 物理性质 | 离子晶体 | 分子晶体 | 原子晶体 | 金属晶体 |
|---------|---------|---------|---------|---------|
| 结合力 | 离子键 | 分子间力 | 共价键 | 金属键 |
| 熔点 | 较高 | 低 | 极高 | 差异大 |
| 沸点 | 较高 | 低 | 极高 | 差异大 |
| 硬度 | 硬而脆 | 软 | 极硬 | 中等（可变）|
| 固态导电 | 否 | 否 | 否（除石墨）| 是 |
| 熔融导电 | 是 | 否 | 否 | 是 |
| 水溶液导电 | 多数是 | 多数否 | 否 | — |
| 延展性 | 无（脆）| 无 | 无（脆）| 有 |
| 光泽 | 无（多数）| 无 | 无（多数）| 有 |

### 熔点规律

**整体趋势（多数情况）：**
原子晶体 > 离子晶体 > 金属晶体（多变）> 分子晶体

**同类型晶体内的熔点影响因素：**

- **离子晶体**：离子电荷越大、半径越小 → 熔点越高
  - MgO(2852°C) > CaO > NaCl(801°C) > KCl

- **分子晶体**：分子量越大，分子间力越强 → 熔点越高（但氢键使沸点异常偏高）
  - F₂ < Cl₂ < Br₂ < I₂（卤素）

- **原子晶体**：键能越大、键长越短 → 熔点越高
  - 金刚石（3550°C）> SiC > SiO₂(1710°C) > Si(1414°C)

- **金属晶体**：自由电子浓度越高、离子半径越小 → 熔点越高
  - W(3407°C) >> Na(98°C) >> Hg(−39°C，液态）

### 导电性的原因

| 晶体 | 固态 | 熔融 | 水溶液 |
|------|------|------|--------|
| 离子晶体 | 不导（离子固定）| 导（离子移动）| 导 |
| 分子晶体 | 不导 | 不导 | 视分子（酸电解质导，有机物多不导）|
| 原子晶体 | 不导（除石墨）| 不导 | 不导 |
| 金属晶体 | 导（自由电子）| 导（e⁻+离子）| — |

### 硬度解释

| 晶体 | 硬度特征 | 原因 |
|------|---------|------|
| 原子晶体 | 最硬（金刚石莫氏10）| 共价键最强，高度方向性 |
| 离子晶体 | 硬而脆 | 离子键强，错位后同号离子排斥→碎裂 |
| 金属晶体 | 中等，可变形 | 金属键无方向性，晶层可滑动 |
| 分子晶体 | 软 | 分子间力弱 |

### 典型例题
**题**：下列物质中，熔点最高的是？A. NaCl B. SiC C. CH₄ D. Fe
**解**：SiC 是原子晶体（熔点 2730°C），最高
NaCl 离子晶体（801°C）；Fe 金属（1538°C）；CH₄ 分子晶体（−182°C）
答案：**B. SiC**

### 易错点
- "导电"要区分固态/熔融/水溶液三种状态
- 金属晶体熔点差异极大（Hg −39°C 到 W 3407°C），不能一概而论
- SiO₂ 不溶于任何酸（HF 除外），不要混淆
