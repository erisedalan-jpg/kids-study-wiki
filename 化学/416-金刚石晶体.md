# 金刚石晶体

> **一句话**：碳原子以sp³杂化形成正四面体网络的原子晶体，是自然界中最硬的物质，熔点3550°C。
> **English**: An atomic crystal in which carbon atoms form a three-dimensional sp³ tetrahedral network; the hardest natural substance with a melting point of 3550°C.

---

## 🎓 高中进阶

### 定义
**金刚石**（Diamond）：碳的同素异形体之一，每个 C 原子以 **sp³ 杂化**与 4 个相邻 C 原子形成 4 个等价 C−C 共价键（σ 键），组成无限三维正四面体网络的**原子晶体**。

### 晶体结构

| 特征 | 参数 |
|------|------|
| 晶体类型 | 原子晶体（面心立方结构）|
| 碳的杂化 | sp³ |
| 配位数 | 4（每个 C 与 4 个 C 相连）|
| 键角 | 109.5° |
| C−C 键长 | 154 pm |
| 晶胞 | fcc + 4 个额外 C（体对角线 1/4 处）|
| 每晶胞 C 数 | 8 个（fcc: 4 + 体内 4）|

### 金刚石的性质

| 性质 | 数值/特征 | 原因 |
|------|----------|------|
| 硬度 | 10（莫氏，最高）| 大量强 C−C 共价键 |
| 熔点 | 3550°C | 断键需极大能量 |
| 导电性 | 不导电（绝缘体）| 无自由电子 |
| 密度 | 3.51 g/cm³ | |
| 折射率 | 2.42（高）| 光学应用 |
| 热导率 | 极高（最高单质）| 声子导热 |

### 金刚石 vs 石墨

| | 金刚石 | 石墨 |
|-|--------|------|
| 碳的杂化 | sp³ | sp² |
| 结构 | 三维网络 | 层状（平面+层间）|
| 晶体类型 | 原子晶体 | 混合型（层内共价，层间范德华）|
| 导电性 | 不导电 | 导电（层内大π键自由电子）|
| 硬度 | 最硬（10）| 很软（1~2）|
| 颜色 | 透明 | 黑色不透明 |
| 用途 | 切割、宝石 | 电极、润滑、铅笔 |

### 金刚石的结构计算
**每晶胞 C 原子数：**
- fcc 贡献：顶点 8×(1/8) + 面心 6×(1/2) = 4
- 体内：4 个（位于体对角线 1/4 处）
- 总计：**8 个 C**

**空间利用率：** 约 34%（四面体堆积效率较低）

### 典型例题
**题**：金刚石晶胞（面心立方型）的边长为 356.7 pm，计算 C 的密度。
**解**：每晶胞 8 个 C，M(C) = 12 g/mol
$$\rho = \frac{8 \times 12}{6.022 \times 10^{23} \times (356.7 \times 10^{-10})^3} \approx 3.51 \text{ g/cm}^3$$

### 易错点
- 金刚石每个 C 的配位数是 4（不是 8 或 6）
- 金刚石导热性极好，但导电性极差（绝缘体）
- 石墨和金刚石是 C 的同素异形体，物理性质迥异
