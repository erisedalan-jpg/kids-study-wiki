# sp2杂化

> **一句话**：1个s轨道与2个p轨道杂化，形成3个sp²杂化轨道，在平面内呈120°排列，剩余1个p轨道可形成1个π键。
> **English**: One s orbital mixes with two p orbitals to form three sp² hybrid orbitals in a trigonal planar arrangement at 120°, with one remaining p orbital for π bonding.

---

## 🎓 高中进阶

### 定义
**sp² 杂化**：中心原子的 1 个 s 轨道与 2 个 p 轨道杂化，形成 **3 个 sp² 杂化轨道**：
- 三个 sp² 杂化轨道在同一平面内，互成 **120°**
- 剩余 **1 个 p 轨道**（垂直于平面）可形成 1 个 π 键
- 分子（或原子）呈**平面三角形**排列（无孤对时）

### sp² 杂化的特征

| 特征 | 说明 |
|------|------|
| 参与轨道 | 1s + 2p |
| 杂化轨道数 | 3 个 |
| 键角 | 120° |
| 轨道排列 | 平面三角形 |
| 剩余 p 轨道 | 1 个（可形成 1 个 π 键）|

### 典型 sp² 杂化分子

| 分子/离子 | 中心原子 | σ+π | 形状 |
|---------|---------|-----|------|
| BF₃ | B（sp²）| 3σ（无π）| 平面三角形 |
| CH₂=CH₂（乙烯）| C（sp²）| 5σ+1π | 平面分子 |
| SO₂ | S（sp²）| 2σ+1π+1孤对 | V形 |
| NO₃⁻ | N（sp²）| 3σ+离域π | 平面三角形 |
| 苯 C₆H₆ | C（sp²）| 大π键 | 平面六边形 |

### 乙烯的 sp² 杂化分析
- 每个 C：sp² 杂化 → 3 个杂化轨道（2 个与 H 的 s 轨道形成 C−H σ 键，1 个与另一 C 形成 C−C σ 键）
- 每个 C 剩余 1 个 p 轨道（垂直分子平面），两个 p 轨道侧面重叠 → 1 个 C=C π 键
- 乙烯所有 6 个原子**共面**（sp² 平面+π键的要求）

### 苯的特殊性
苯环中 6 个 C 均为 sp²：
- sp² 杂化的 3 个轨道：2 个与相邻 C 形成 C−C σ 键，1 个与 H 形成 C−H σ 键
- 每个 C 剩余 1 个 p 轨道，6 个 p 轨道形成**离域大 π 键**（π₆⁶）
- 苯环所有原子共面，键角 120°

### 典型例题
**题**：SO₂ 中 S 是什么杂化？分子形状如何？
**解**：S 与 2 个 O 形成 2 个 σ 键，加上 1 个孤对电子，σ 键数 + 孤对数 = 3 → **sp² 杂化**
分子几何：有 1 个孤对电子对，不呈现在分子形状中 → **V 形**，键角约 119°

### 易错点
- sp² 杂化不一定是平面三角形（含孤对时为 V 形，如 SO₂）
- 苯中没有固定单键/双键，6 个 C−C 键等长（键长介于单双键之间）
- 碳的 sp² 轨道之间夹角是 120°，不是 90°
