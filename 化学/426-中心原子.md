# 中心原子

> **一句话**：配合物中接受配体孤对电子的原子（通常是过渡金属离子），通过配位键与配体结合，具有空的接受轨道。
> **English**: The atom in a coordination compound that accepts lone pairs from ligands (usually a transition metal ion); has empty acceptor orbitals and is bonded to ligands via coordinate bonds.

---

## 🎓 高中进阶

### 定义
**中心原子**（Central Atom，M）：配合物内界中居中的原子，接受配体提供的孤对电子，形成配位键。

在 VSEPR/杂化轨道理论中，"中心原子"也指分子中直接与多个原子键合的原子（如 CO₂ 的 C）。但在配位化学中，特指配合物中的受体原子。

### 中心原子的特征

| 特征 | 说明 |
|------|------|
| 通常是过渡金属离子 | 如 Cu²⁺、Fe³⁺、Co³⁺、Pt²⁺ |
| 有空轨道（接受配体）| d 轨道、s 轨道或 p 轨道 |
| 路易斯酸 | 接受电子对的物种 |
| 常见氧化态 | +1 到 +4（视金属而定）|

### 常见中心原子及特性

| 中心原子 | 常见配合物 | 特点 |
|---------|---------|------|
| Cu²⁺（d⁹）| [Cu(NH₃)₄]²⁺ | 深蓝色，配位数 4 |
| Fe³⁺（d⁵）| [Fe(CN)₆]³⁻ | 5 个未成对 d 电子 |
| Ag⁺（d¹⁰）| [Ag(NH₃)₂]⁺ | d¹⁰，配位数 2 |
| Co³⁺（d⁶）| [Co(NH₃)₆]³⁺ | 配位数 6，八面体 |
| Pt²⁺（d⁸）| [PtCl₄]²⁻ | 平面四方，抗癌药前体 |

### 中心原子的路易斯酸性
中心原子是**路易斯酸**（电子对受体），配体是**路易斯碱**（电子对供体）。
过渡金属离子的 d 轨道空位（或空 s/p 轨道）使其成为优秀的路易斯酸。

### 中心原子在分子构型中的角色
在 VSEPR 分析中，每个分子都有"中心原子"：
- H₂O 中 O 是中心原子（sp³ 杂化）
- NH₃ 中 N 是中心原子（sp³ 杂化）
- CO₂ 中 C 是中心原子（sp 杂化）

### 典型例题
**题**：[Fe(SCN)]²⁺ 中，中心原子是什么？写出其配位数和氧化态。
**解**：
- 中心原子：**Fe**（铁）
- 配位数：**1**（只有 1 个 SCN⁻ 配体）
- 氧化态：**+3**（[Fe(SCN)]²⁺ = Fe³⁺ + SCN⁻，故 Fe 氧化态 = +3）
- 用途：检验 Fe³⁺，溶液呈血红色

### 易错点
- 中心原子不一定是金属（分子中如 CO₂ 的 C、PCl₅ 的 P 均为"中心原子"）
- 配合物中的中心原子通常有具体的氧化态，不要写成"原子态"
- Ag⁺ 的 d¹⁰ 满壳层，配位数通常为 2（直线形）
