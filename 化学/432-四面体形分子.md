# 四面体形分子

> **一句话**：中心原子与4个相同原子键合，键角109.5°，呈正四面体空间构型，sp³杂化且无孤对，为非极性分子。
> **English**: A molecule with a central atom bonded to four identical atoms at 109.5° angles in a regular tetrahedral arrangement; sp³ hybridized with no lone pairs, making it nonpolar.

---

## 🎓 高中进阶

### 定义
**正四面体形分子**（Tetrahedral）：中心原子 sp³ 杂化，与 4 个原子成键（0 孤对），键角为 **109.5°**（正四面体角），4 个键等价。

注意：sp³ 杂化含孤对时，形状变为三角锥（1孤）或 V 形（2孤）。

### 典型四面体形分子

| 分子/离子 | 中心原子 | 孤对 | 键角 | 极性 |
|---------|---------|------|------|------|
| CH₄ | C（sp³）| 0 | 109.5° | 非极性 |
| CCl₄ | C（sp³）| 0 | 109.5° | 非极性 |
| SiH₄ | Si（sp³）| 0 | 109.5° | 非极性 |
| NH₄⁺ | N（sp³）| 0 | 109.5° | 非极性 |
| SO₄²⁻ | S（sp³）| 0 | 109.5° | 非极性 |
| BF₄⁻ | B（sp³）| 0 | 109.5° | 非极性 |

含孤对的 sp³（非四面体形，列出对比）：
- NH₃：3键+1孤 → 三角锥形，107°
- H₂O：2键+2孤 → V 形，104.5°

### CH₄ 的正四面体分析
- C 的 sp³ 杂化：4 个杂化轨道指向正四面体 4 顶点
- 各轨道与 H 的 1s 轨道形成 C−H σ 键
- 4 个等价 C−H 键，键角 109.5°，分子对称
- 4 个极性 C−H 键向量对称叠加 → 抵消 → **非极性分子**

### 四面体与三角锥的对比

| | 四面体（CH₄）| 三角锥（NH₃）|
|-|------------|------------|
| 中心原子 | C | N |
| 成键对 | 4 | 3 |
| 孤对 | 0 | 1 |
| 键角 | 109.5° | 107° |
| 极性 | 非极性 | 极性 |

### 典型例题
**题**：CCl₄ 是正四面体，是否为极性分子？若将一个 Cl 换成 F（CHCl₃），是否极性分子？
**解**：CCl₄：4 个等价 C−Cl 键对称叠加，偶极矩为零 → **非极性分子**
CHCl₃：C−H 键和 C−Cl 键极性不同，不完全抵消 → **极性分子**

### 易错点
- 四面体形不一定全是 sp³（如某些超价化合物），但高中阶段 sp³ = 四面体
- 正四面体 ≠ 平面四边形（注意三维空间排列）
- 等价配体四面体为非极性，但配体不同则为极性
