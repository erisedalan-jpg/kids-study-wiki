# 有机化合物结构

> **一句话**：有机化合物的性质由其原子种类、连接方式（键型）和空间排布共同决定，结构决定性质是有机化学的核心思想。
> **English**: The properties of an organic compound are determined by the types of atoms, their bonding connectivity, and spatial arrangement.

---

## 🎓 高中进阶

### 定义 / 结构特征

有机化合物结构包含三个层次：
1. **组成**：碳骨架 + 氢、氧、氮、卤素等杂原子
2. **构造**（connectivity）：原子间的连接顺序与键型（单键、双键、三键、环）
3. **构型/构象**：原子在空间中的排布方式（顺反、手性等）

有机结构理论基石：
- **碳四价理论**：碳原子有 4 个共价键
- **价键理论**：共用电子对形成σ键（可旋转）和π键（限制旋转）
- **共轭体系**：多个π键交替出现，电子离域，稳定性增大

结构式表达方式：

| 表达方式 | 示例（乙醇）| 特点 |
|---------|------------|------|
| 电子式 | H:C:C:O:H | 显示共用电子对 |
| 结构式 | CH₃—CH₂—OH | 显示所有化学键 |
| 简式（结构简式）| CH₃CH₂OH | 省略碳氢键 |
| 键线式 | /\OH | 省略碳和氢 |
| 球棍模型 | 三维模型 | 直观空间关系 |

### 性质 / 反应

有机结构与性质的关系：

| 结构特征 | 对应性质举例 |
|---------|------------|
| C=C 双键 | 可发生加成反应、氧化反应 |
| —OH（醇羟基）| 可发生取代、氧化、酯化 |
| —COOH（羧基）| 显酸性，可酯化 |
| 苯环（共轭）| 稳定，以取代反应为主 |
| 手性中心 | 旋光性，对映体 |

### 典型例题

**例**：写出分子式为 C₂H₆O 的有机物的两种结构简式，并说明其性质差异。

解：
- 乙醇 CH₃CH₂OH：含醇羟基，能与钠反应放 H₂，能氧化为乙醛
- 甲醚 CH₃OCH₃：含醚键，化学性质较稳定，不与钠反应

→ 二者互为同分异构体，结构不同，性质差异显著，体现"结构决定性质"。

### 易错点

- 结构简式≠分子式：CH₃CH₂OH 是结构简式，C₂H₆O 是分子式
- 键线式中每个折点和末端均代表一个碳原子（碳氢省略）
- π键不能自由旋转→C=C 双键导致顺反异构；σ键可旋转→不产生新的立体异构体
