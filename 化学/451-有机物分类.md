# 有机物分类

> **一句话**：有机化合物按碳骨架类型（链状/环状）和官能团种类进行分类，形成系统的有机物家族体系。
> **English**: Organic compounds are classified by carbon skeleton type (acyclic/cyclic) and functional group, forming a systematic family of organic substances.

---

## 🎓 高中进阶

### 定义 / 结构特征

**按碳骨架分类**：

```
有机化合物
├── 开链化合物（脂肪族化合物）
│   ├── 饱和链烃（烷烃）
│   └── 不饱和链烃（烯烃、炔烃、二烯烃）
└── 环状化合物
    ├── 脂环化合物（环烷烃、环烯烃等）
    └── 芳香族化合物（含苯环）
```

**按官能团分类**：

| 类别 | 官能团 | 结构特征 | 代表物 |
|------|--------|---------|--------|
| 卤代烃 | —X（卤素）| C—X 键 | 氯乙烷 |
| 醇 | —OH（醇羟基）| C—O—H，O 连烃基 | 乙醇 |
| 酚 | —OH（酚羟基）| 苯环—OH | 苯酚 |
| 醚 | —O—（醚键）| R—O—R | 乙醚 |
| 醛 | —CHO（醛基）| R—C(=O)H | 乙醛 |
| 酮 | —CO—（羰基）| R—C(=O)—R | 丙酮 |
| 羧酸 | —COOH（羧基）| R—COOH | 乙酸 |
| 酯 | —COO—（酯基）| R—COO—R | 乙酸乙酯 |
| 胺 | —NH₂（氨基）| R—NH₂ | 甲胺 |
| 氨基酸 | —NH₂ + —COOH | 同时含两种官能团 | 甘氨酸 |

### 性质 / 反应

官能团决定有机物的主要化学性质：

| 官能团 | 代表反应类型 |
|--------|------------|
| C=C | 加成、加聚、氧化 |
| —OH（醇）| 取代、氧化、酯化、消去 |
| —OH（酚）| 弱酸性、与 FeCl₃ 显色 |
| —CHO | 银镜反应、被新制 Cu(OH)₂ 氧化 |
| —COOH | 酸性、酯化 |
| —COO— | 水解 |

### 典型例题

**例**：甘氨酸（H₂NCH₂COOH）属于哪类有机物？含哪些官能团？

解：甘氨酸属于**氨基酸**，同时含有：
- **氨基**（—NH₂）：碱性基团
- **羧基**（—COOH）：酸性基团
→ 氨基酸既有酸性又有碱性，是两性化合物。

**例**：将乙酸乙酯归类。

解：乙酸乙酯（CH₃COOC₂H₅）含**酯基**（—COO—），属于**酯类**；碳骨架为开链，属**脂肪族化合物**。

### 易错点

- 醇羟基（—OH 连接饱和碳）≠ 酚羟基（—OH 连接苯环）：性质差异显著
- 醛（—CHO 连接 H）≠ 酮（—CO— 连接两个烃基）：醛具有还原性，酮无
- 羧酸的酸性由—COOH 提供，不是由—OH 直接给出 H⁺（与醇不同）
