# 顺反异构

> **一句话**：由于 C=C 双键（或环）不能自由旋转，当双键两端各碳上连有不同基团时，产生顺式和反式两种空间异构体。
> **English**: Cis-trans isomerism arises from restricted rotation around a C=C double bond (or ring), when each carbon bears two different substituents.

---

## 🎓 高中进阶

### 定义 / 结构特征

**顺反异构**（几何异构）产生的条件：
1. 存在 C=C 双键（或环）
2. 双键每端碳原子连接的两个基团**不相同**

命名规则：
- **顺式（cis-）**：相同（或较大）基团在双键同侧
- **反式（trans-）**：相同（或较大）基团在双键异侧
- IUPAC 系统命名用 **Z/E** 标记（按原子序数优先规则判断）

以 2-丁烯为例：

```
顺-2-丁烯（cis）：       反-2-丁烯（trans）：
  CH₃   CH₃               CH₃   H
   \   /                    \   /
    C=C                      C=C
   /   \                    /   \
  H     H                  H   CH₃
```

### 性质 / 反应

| 性质 | 顺-2-丁烯 | 反-2-丁烯 |
|------|----------|----------|
| 沸点 | 3.7°C | 0.9°C |
| 熔点 | -139°C | -106°C |
| 偶极矩 | 较大（偶极不抵消）| 较小（偶极部分抵消）|
| 稳定性 | 稍低（空间排斥）| 稍高 |
| 化学性质 | 均可加成、氧化 | 均可加成、氧化 |

### 典型例题

**例**：判断下列化合物是否存在顺反异构。

1. CH₂=CH₂（乙烯）→ 每端碳均有 2 个 H，不满足条件，**无**顺反异构
2. CH₃CH=CHCH₃（2-丁烯）→ 每端碳连有不同基团（H 和 CH₃），**有**顺反异构
3. CH₂=CHCH₃（丙烯）→ 一端碳有 2 个 H，**无**顺反异构
4. CHCl=CHCl（1,2-二氯乙烯）→ 每端均有 H 和 Cl，**有**顺反异构

### 易错点

- 顺反异构属于立体异构，不是构造异构（分子式、原子连接顺序均相同）
- 环状化合物（如环己烷二取代）也可产生顺反异构
- Z/E 命名与顺/反不完全对应：Z 不一定是"顺"，需按优先顺序判断
