# 一元醇

> **一句话**：一元醇是分子中只含一个醇羟基（—OH）的醇，包括甲醇、乙醇等，性质以醇羟基的反应为核心。
> **English**: A monohydric alcohol contains exactly one hydroxyl group (—OH); examples include methanol and ethanol; its chemistry centers on the hydroxyl group reactions.

---

## 🎓 高中进阶

### 定义 / 结构特征

**一元醇**：分子中仅含 **1 个—OH** 的醇，通式 R—OH（R 为烃基）。

**常见一元醇**：

| 名称 | 分子式 | 沸点 | 特点 |
|------|--------|------|------|
| 甲醇 | CH₃OH | 64.7°C | 有毒，饮用致盲 |
| 乙醇 | C₂H₅OH | 78.4°C | 俗称酒精 |
| 正丙醇 | C₃H₇OH | 97.2°C | 1-丙醇，伯醇 |
| 2-丙醇（异丙醇）| C₃H₇OH | 82.3°C | 仲醇 |
| 正丁醇 | C₄H₉OH | 117.7°C | 伯醇 |
| 叔丁醇 | (CH₃)₃COH | 82.2°C | 叔醇，不被 Cu 催化氧化 |

**沸点规律**：随碳数增大而升高（氢键）；同碳数：伯 > 仲 > 叔（支链减小接触面积）

### 性质 / 反应

一元醇的化学性质由**醇羟基**决定：

| 反应 | 举例（乙醇）| 产物 |
|------|------------|------|
| 与 Na | 2ROH + 2Na → 2RONa + H₂↑ | 醇钠 + H₂ |
| 催化氧化（伯醇）| RCH₂OH + O₂ →(Cu) RCHO + H₂O | 醛 |
| 催化氧化（仲醇）| R₂CHOH + O₂ →(Cu) R₂CO + H₂O | 酮 |
| 叔醇 | 不被 Cu 催化氧化（α-C 无 H）| — |
| 酯化 | ROH + RCOOH ⇌ RCOOR + H₂O | 酯 |
| 消去 | ROH →(浓H₂SO₄,△) 烯 + H₂O | 烯烃 |
| 与 HX | ROH + HX → RX + H₂O | 卤代烃 |

### 典型例题

**例**：如何区分伯醇、仲醇和叔醇（用 Lucas 试剂，即 ZnCl₂/浓 HCl）？

解（大学知识，了解即可）：
- 叔醇：室温立即浑浊（生成叔氯代烃，不溶于试剂）
- 仲醇：约 5 分钟后浑浊
- 伯醇：不浑浊（反应太慢）

**例（高中重点）**：1-丙醇和 2-丙醇的催化氧化产物分别是什么？

解：
- 1-丙醇（伯醇，CH₃CH₂CH₂OH）→ 丙醛（CH₃CH₂CHO）
- 2-丙醇（仲醇，CH₃CHOHCH₃）→ 丙酮（CH₃COCH₃）

**例**：叔丁醇（(CH₃)₃COH）能否被 Cu 催化氧化？

解：叔丁醇的 α-碳（连—OH 的碳）上无 H（连 3 个—CH₃），不能被 Cu 催化氧化（无法脱 H）。

### 易错点

- 伯醇（伯碳—OH）→ 醛；仲醇（仲碳—OH）→ 酮；叔醇无法被 Cu 催化氧化
- 一元醇与 Na₂CO₃ 不反应（酸性太弱），但乙酸可以（区分方法）

### 例句

- **Primary alcohols** are oxidized to **aldehydes** by copper-catalyzed air oxidation.
- **Tertiary alcohols** cannot be oxidized by this method because the α-carbon bears no hydrogen.
