# 卤代烃水解

> **一句话**：卤代烃在 NaOH 水溶液中加热，C—X 键断裂，—X 被—OH 取代，生成相应的醇，该反应是卤代烃转化为醇的重要方法。
> **English**: Haloalkanes react with NaOH in aqueous solution under heating; the C—X bond breaks and —X is replaced by —OH, giving an alcohol; this is a key method for converting haloalkanes to alcohols.

---

## 🎓 高中进阶

### 定义 / 结构特征

**卤代烃水解**（hydrolysis）：卤代烃与水（或 NaOH 水溶液）反应，卤素原子被羟基（—OH）取代的反应。

**反应本质**：取代反应（—X 被 OH⁻ 取代）

**反应条件**：
- 试剂：**NaOH 水溶液**（碱性水解，不可逆，效率更高）
- 温度：加热（提高反应速率）
- 注意：若只用纯水（中性水解），反应很慢且可逆

**通式**：R—X + NaOH(水) →(△) R—OH + NaX

### 性质 / 反应

| 卤代烃 | 水解方程式 | 产物 |
|--------|-----------|------|
| 氯乙烷 CH₃CH₂Cl | CH₃CH₂Cl + NaOH(水) → CH₃CH₂OH + NaCl | 乙醇 |
| 溴乙烷 CH₃CH₂Br | CH₃CH₂Br + NaOH(水) → CH₃CH₂OH + NaBr | 乙醇 |
| 溴丙烷 CH₃CH₂CH₂Br | CH₃CH₂CH₂Br + NaOH(水) → CH₃CH₂CH₂OH + NaBr | 1-丙醇 |
| 三氯甲烷 CHCl₃ | 逐步水解（较复杂）| 甲酸 |

**活性比较**：R—I > R—Br > R—Cl > R—F（碘代烃最易水解）

**与消去反应区别**：
| 反应 | 条件 | 产物 |
|------|------|------|
| 水解（取代）| NaOH **水**溶液，加热 | 醇 |
| 消去 | NaOH **醇**溶液，加热 | 烯烃 |

### 典型例题

**例**：设计实验检验溴乙烷发生了水解反应。

解（步骤）：
1. 取适量溴乙烷于试管中，加入 NaOH 水溶液，振荡后加热
2. 冷却后，向试管中加入足量**稀 HNO₃**（酸化，中和多余 NaOH）
3. 再加入**AgNO₃ 溶液**，若产生淡黄色沉淀（AgBr↓），则证明水解发生（产生了 Br⁻）

**例**：1-溴丙烷（CH₃CH₂CH₂Br）碱性水解的方程式。

解：CH₃CH₂CH₂Br + NaOH(水) →(△) CH₃CH₂CH₂OH + NaBr

**例**：为什么碘代烃比氯代烃水解更容易？

解：C—I 键能（222 kJ/mol）< C—Br（285）< C—Cl（339）< C—F（485），键能越小越容易断裂；同时 I⁻ 作为离去基团，其稳定性（极化性）强，离去能力更强。

### 易错点

- 必须用 NaOH **水溶液**（不是醇溶液）才是水解；醇溶液→消去
- 检验卤素 X⁻ 必须先酸化（消除 OH⁻），再加 AgNO₃
- 纯水的水解（中性）可逆且缓慢，NaOH 将产物 HX 中和使反应正向进行（不可逆）
