# 卤代烃消去

> **一句话**：卤代烃在 NaOH 醇溶液中加热，脱去卤化氢（—X 和 β-碳上的 H），生成烯烃，是由卤代烃制备烯烃的重要方法。
> **English**: Haloalkanes react with NaOH in alcoholic solution under heating; HX is eliminated (—X and a β-H are removed), giving an alkene; this is a key method for preparing alkenes from haloalkanes.

---

## 🎓 高中进阶

### 定义 / 结构特征

**消去反应**（elimination，脱卤化氢）：卤代烃分子中脱去卤化氢（HX），形成 C=C 双键，生成烯烃。

**条件**：NaOH 的**醇溶液**（乙醇等），加热

**消去反应要求**：
- 卤代烃的 β-碳（与含 X 的碳相邻的碳）上必须有 H
- 若无 β-氢，不能消去（如叔丁基溴 (CH₃)₃CBr 无 β-H→不能消去，实际上叔醇才能消去，卤代烃只需β-H）

**通式**：R—CHX—CH₂—R' + NaOH(醇) →(△) R—CH=CH—R' + NaX + H₂O

**扎依采夫规则**（Zaitsev's rule）：消去时，优先脱去含 H 较少的 β-碳上的 H，得到双键碳上取代基最多（最稳定）的烯烃为主产物。

### 性质 / 反应

| 卤代烃 | 消去方程式 | 产物 |
|--------|-----------|------|
| CH₃CH₂Br | CH₃CH₂Br + NaOH(醇) → CH₂=CH₂ + NaBr + H₂O | 乙烯 |
| CH₃CHBrCH₃（2-溴丙烷）| CH₃CHBrCH₃ + NaOH(醇) → CH₃CH=CH₂ + NaBr + H₂O | 丙烯 |
| CH₃CH₂CHBrCH₃（2-溴丁烷）| 可得 1-丁烯或 2-丁烯（主：2-丁烯，Zaitsev）| |

**水解 vs 消去**：

| 对比项 | 水解（取代）| 消去 |
|--------|-----------|------|
| 试剂溶剂 | NaOH **水**溶液 | NaOH **醇**溶液 |
| 产物 | 醇 | 烯烃 |
| 断裂键 | C—X 键 | C—X + β C—H 键 |

### 典型例题

**例**：2-溴丁烷消去的主要产物是什么？

解：2-溴丁烷 CH₃CHBrCH₂CH₃，β-碳有两个：C₁（CH₃，3个H）和 C₃（CH₂，2个H）。
- 脱 C₁—H：得 1-丁烯（CH₂=CHCH₂CH₃）
- 脱 C₃—H：得 2-丁烯（CH₃CH=CHCH₃）
按 Zaitsev 规则，**2-丁烯**为主产物（双键碳取代基更多，更稳定）。

**例**：CH₃Br（甲基溴）能发生消去反应吗？

解：甲基溴无 β-碳（无与含 Br 碳相邻的其他碳），**不能**发生消去反应，只能水解生成甲醇。

**例**：由溴乙烷制备乙醇和乙烯，各用什么条件？

解：
- 制乙醇：NaOH **水**溶液 + 加热（水解）
- 制乙烯：NaOH **醇**溶液 + 加热（消去）

### 易错点

- 消去反应需要 β-H，无 β-H 的卤代烃（如 CH₃X）不能消去
- 消去产物可能有多种烯烃（β-碳不止一个时），注意 Zaitsev 规则选主产物
- 消去反应中醇（乙醇）是溶剂，不是底物（不被消耗）
