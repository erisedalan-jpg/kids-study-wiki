# 甲酸

> **一句话**：甲酸（HCOOH）是最简单的羧酸，也是最强的脂肪酸，结构上同时含有羧基和醛基性质，能发生银镜反应，兼具酸和醛的双重性质。
> **English**: Formic acid (HCOOH) is the simplest and strongest simple carboxylic acid; it has both carboxyl and aldehyde character, can undergo the silver mirror reaction, and possesses dual acid-aldehyde properties.

---

## 🎓 高中进阶

### 定义 / 结构特征

**甲酸**（formic acid，蚁酸）：
- 分子式 CH₂O₂，结构简式 HCOOH
- 含**羧基**（—COOH）
- 结构特殊：H—**C**(=O)—OH，即 H 直接连在羰基碳上，与醛基（—CHO）结构相似
- 无色液体，刺激性气味（蚂蚁螫伤分泌），沸点 100.8°C，与水互溶
- pKa = 3.75（比乙酸强，最强的简单羧酸）

**甲酸的双重性质**：
- 作为**羧酸**：与 NaOH 中和，与醇酯化，与 Na₂CO₃/NaHCO₃ 放 CO₂
- 作为**含醛性质的化合物**：H—CO—OH 中的 H（连在 C=O 碳上）具还原性 → 能与银氨溶液银镜反应，与新制 Cu(OH)₂ 加热反应

### 性质 / 反应

| 反应类型 | 方程式 | 条件 |
|---------|--------|------|
| 与 NaOH 中和 | HCOOH + NaOH → HCOONa + H₂O | 常温 |
| 与 NaHCO₃ | HCOOH + NaHCO₃ → HCOONa + H₂O + CO₂↑ | 常温 |
| 银镜反应（特殊）| HCOOH + 2Ag(NH₃)₂OH → (NH₄)₂CO₃ + 2Ag↓ + 2NH₃ + H₂O | 水浴加热 |
| 与 Cu(OH)₂ | HCOOH + 2Cu(OH)₂ → CO₂ + Cu₂O↓ + 3H₂O | 加热 |
| 脱水（热分解）| HCOOH →(浓H₂SO₄,△) CO↑ + H₂O（制 CO）| 浓H₂SO₄ |
| 酯化 | HCOOH + C₂H₅OH ⇌ HCOOC₂H₅ + H₂O | 浓H₂SO₄，△ |

**甲酸制备 CO**（实验室）：
HCOOH →(浓H₂SO₄,70-80°C) CO↑ + H₂O
（浓 H₂SO₄ 脱水，甲酸分解）

### 典型例题

**例**：甲酸（HCOOH）与乙酸（CH₃COOH）各 1 mol，与银氨溶液反应，各生成几 mol Ag？

解：
- 乙酸：无—CHO 性质，**不发生**银镜反应，生成 0 mol Ag
- 甲酸：H—COO—H（含—CHO 性质），**能发生**银镜反应，生成 **2 mol Ag**

**例**：用甲酸制备 CO 的步骤（实验室）。

解：向甲酸中加入浓 H₂SO₄，加热（70-80°C），收集尾气（CO，有毒，注意通风）。
HCOOH →(浓H₂SO₄) CO↑ + H₂O

**例**：甲酸比乙酸酸性强的原因。

解：甲酸中 HCOO⁻（甲酸根）共轭碱的稳定性略低于 CH₃COO⁻（乙酸根），但甲酸 H 连在 C=O 上（电负性强），O—H 键更弱，H⁺ 更易解离，酸性更强（pKa甲酸=3.75 < pKa乙酸=4.76）。

### 易错点

- 甲酸是**羧酸**，但同时具有**醛的性质**（还原性，能银镜反应），是有机化学重要易错点
- 甲酸脱水（浓 H₂SO₄）生成 CO，不是 CO₂（不是完全氧化，而是脱水）
- 甲酸酯（甲酸乙酯 HCOOC₂H₅）含甲酸骨架，仍有—CHO 性质，能发生银镜反应（常考）

### 例句

- **Formic acid** (HCOOH) is both a **carboxylic acid** and has **aldehyde-like** reducing properties.
- Unlike other carboxylic acids, **formic acid** gives a positive **silver mirror reaction** due to the H—C=O unit.
