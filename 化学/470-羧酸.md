# 羧酸

> **一句话**：羧酸含有羧基（—COOH），具有酸性（弱酸），能与醇发生酯化反应，能与活泼金属、碱、碳酸盐反应，乙酸是最典型代表。
> **English**: Carboxylic acids contain the carboxyl group (—COOH) and are weakly acidic; they react with alcohols (esterification), active metals, bases, and carbonates; acetic acid is the most representative example.

---

## 🎓 高中进阶

### 定义 / 结构特征

**羧酸**（carboxylic acid）：含**羧基**（—COOH，羰基 C=O + 羟基 O—H）的有机酸。

**羧基结构**：
- —C(=O)—OH：C 为 sp² 杂化
- 由于 C=O 的吸电子效应，—OH 的 H 比醇更容易解离（酸性更强）
- 羧基可提供 H⁺（弱酸）

**常见羧酸**：

| 名称 | 结构简式 | pKa | 特点 |
|------|---------|-----|------|
| 甲酸 | HCOOH | 3.75 | 最强的简单羧酸，含—CHO 性质 |
| 乙酸 | CH₃COOH | 4.76 | 醋酸，食品中使用 |
| 草酸 | HOOCCOOH | 1.25/4.27 | 有毒，二元酸 |
| 苯甲酸 | C₆H₅COOH | 4.20 | 防腐剂 |
| 乳酸 | CH₃CHOHCOOH | 3.86 | 含手性碳 |

**酸性强弱**：甲酸 > 苯甲酸 > 乙酸（取代基推电子越强，酸性越弱）

### 性质 / 反应

| 反应类型 | 方程式示例 | 条件 |
|---------|-----------|------|
| 与活泼金属 | 2CH₃COOH + 2Na → 2CH₃COONa + H₂↑ | 常温 |
| 与碱中和 | CH₃COOH + NaOH → CH₃COONa + H₂O | 常温 |
| 与碳酸盐 | 2CH₃COOH + Na₂CO₃ → 2CH₃COONa + H₂O + CO₂↑ | 常温 |
| 与 NaHCO₃ | CH₃COOH + NaHCO₃ → CH₃COONa + H₂O + CO₂↑ | 常温 |
| 酯化（与醇）| CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O | 浓H₂SO₄，△ |
| 与 PCl₃（制酰氯）| 大学内容 | — |

**与碳酸氢钠反应（鉴别用）**：
羧酸（酸性 > 碳酸）+ NaHCO₃ → 放 CO₂↑（酚不放 CO₂，醇不反应）
→ 可用于区分羧酸、酚、醇

### 典型例题

**例**：如何用化学方法区分乙酸、苯酚、乙醇三种液体？

解：
1. 加 NaHCO₃ 溶液：
   - 乙酸 → 产生 CO₂ 气泡（酸性强于 HCO₃⁻）
   - 苯酚 → 无气泡（酸性弱于 H₂CO₃，但强于 HCO₃⁻... 注：苯酚酸性约等于 HCO₃⁻，高中简化处理：苯酚不与 NaHCO₃ 反应放 CO₂）
   - 乙醇 → 无气泡
2. 加 FeCl₃ 溶液（剩余两者）：
   - 苯酚 → 紫色
   - 乙醇 → 无变化

**例**：乙酸与乙醇酯化反应是可逆反应，如何提高乙酸乙酯的产率？

解：使用**浓 H₂SO₄**（催化剂 + 吸水，使平衡正移）；**加热**提高速率；及时蒸出乙酸乙酯（分馏，降低产物浓度，平衡正移）；乙醇过量（减小乙酸浓度使更多转化）。

### 易错点

- 甲酸（HCOOH）结构上含—CHO 性质（H—COO—H），能与银氨溶液发生银镜反应（区别于其他羧酸）
- 羧酸与碳酸盐/碳酸氢盐反应放 CO₂，证明羧酸酸性强于碳酸（H₂CO₃）
- 酯化是可逆反应（用⇌），不是单向反应
