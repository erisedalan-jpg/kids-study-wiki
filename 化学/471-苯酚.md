# 苯酚

> **一句话**：苯酚（C₆H₅OH）是最简单的酚类，弱酸性（pKa≈10），与溴水生成白色三溴苯酚沉淀，与 FeCl₃ 显紫色，是重要的化工原料（树脂、染料、药物）。
> **English**: Phenol (C₆H₅OH) is the simplest phenol; it is weakly acidic (pKa ≈ 10), gives a white precipitate (2,4,6-tribromophenol) with bromine water, shows purple with FeCl₃, and is an important chemical feedstock.

## 🎓 高中进阶

### 定义 / 结构特征

**苯酚**（phenol）：
- 分子式 C₆H₆O，结构简式 C₆H₅OH（苯环 + 酚羟基）
- 白色固体（晶体），熔点 43°C，沸点 182°C
- 微溶于水（常温，65°C 以上完全互溶），易溶于乙醇等有机溶剂
- 有特殊气味，有毒，皮肤接触需立即用酒精擦洗
- 贮存中被氧化变粉红色（酚类易被空气中 O₂ 氧化）

**结构特征**：
- —OH 直接连苯环 sp² 碳
- 苯环 π 电子与 O 的孤对电子共轭，使 O—H 键更弱（H 更易解离→酸性强于醇）
- 苯环电子密度增大（特别是邻/对位），亲电取代活性增大（比苯活泼）

### 性质 / 反应

| 反应 | 方程式 | 条件 |
|------|--------|------|
| 与 NaOH | C₆H₅OH + NaOH → C₆H₅ONa + H₂O | 常温 |
| 与 Na₂CO₃ | C₆H₅OH + Na₂CO₃ → C₆H₅ONa + NaHCO₃ | 常温 |
| CO₂ 还原苯酚钠 | C₆H₅ONa + CO₂ + H₂O → C₆H₅OH↓ + NaHCO₃ | 通 CO₂ |
| FeCl₃ 显色 | C₆H₅OH + FeCl₃ → 紫色（络合）| 常温 |
| 与溴水（三溴化）| C₆H₅OH + 3Br₂ → 2,4,6-C₆H₂Br₃OH↓ + 3HBr | 常温，溴水褪色 |
| 硝化 | C₆H₅OH + HNO₃ → 邻/对硝基苯酚 | 稀HNO₃，室温 |
| 氧化 | 暴露空气后变粉红色 | 空气 O₂ |

**酸性比较**：
- H₂CO₃（pKa=6.35）> **苯酚（pKa≈10）** > HCO₃⁻（pKa=10.33 近似相当，可与 Na₂CO₃ 反应）> H₂O > 乙醇

（注：高中阶段：苯酚酸性强于 HCO₃⁻，故与 Na₂CO₃ 反应生成 NaHCO₃；苯酚酸性弱于 H₂CO₃，故通 CO₂ 可从苯酚钠中重新析出苯酚）

### 典型例题

**例**：苯酚钠溶液中通入 CO₂，有何现象？写出方程式。

解：溶液变浑浊（苯酚析出，微溶于水），方程式：
C₆H₅ONa + CO₂ + H₂O → C₆H₅OH↓ + NaHCO₃
（强酸制弱酸，CO₂/H₂O 的酸性强于苯酚）

**例**：苯酚变粉红色的原因，以及如何防止？

解：苯酚被空气中 O₂ 氧化，生成有色的醌类物质，导致变色。防止方法：密封保存，或在氮气保护下存储。

**例**：苯酚与溴水反应的实验现象（鉴别用途）。

解：向苯酚溶液中加入溴水，立即产生**白色沉淀**（2,4,6-三溴苯酚），溴水**褪色**。此反应极灵敏，可用于定性/定量分析苯酚。

### 易错点

- 苯酚与 Na₂CO₃ 反应只到 NaHCO₃（不产生 CO₂）；苯酚与 NaHCO₃ 不反应
- 苯酚微溶于冷水（约 8.3 g/100 mL），加热后完全互溶
- 苯酚腐蚀性强，皮肤接触不能用水洗（会扩散），应用乙醇洗去
