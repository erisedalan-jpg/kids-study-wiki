# 苯酚显色反应

> **一句话**：苯酚（及含酚羟基的化合物）与 FeCl₃ 溶液反应显**紫色**，是鉴别酚羟基的特征反应，用于检验有机物中是否含有酚基团。
> **English**: Phenol (and compounds containing phenolic OH) reacts with FeCl₃ solution to give a **purple color**; this is the characteristic test for phenolic groups, used to detect phenol in organic compounds.

---

## 🎓 高中进阶

### 定义 / 结构特征

**苯酚显色反应**：苯酚（C₆H₅OH）与三氯化铁（FeCl₃）溶液反应，生成紫色络合物。

**反应本质**：配位络合（不是氧化还原）
C₆H₅OH + Fe³⁺ → [Fe(OC₆H₅)₆]³⁻ 等络合物（紫色）

**反应现象**：
- 少量苯酚加入 FeCl₃ 溶液 → 溶液显**紫色**（特征色）
- 浓度不同，颜色深浅不同，但均为紫色调
- 不同酚类化合物显色可能略有不同（如邻苯二酚→绿色，间苯三酚→紫色等）

### 性质 / 反应

**显色反应适用范围**：

| 化合物 | 与 FeCl₃ | 颜色 |
|--------|---------|------|
| 苯酚 | 反应 | 紫色 |
| 对苯二酚 | 反应 | 深绿→褐色 |
| 邻苯二酚 | 反应 | 绿色 |
| 乙醇（醇）| 不反应 | 无变化 |
| 乙酸（羧酸）| 不变色（乙酸铁无特殊色）| 黄色（Fe³⁺ 本色）|

**注意**：醛、醇、醚等无酚羟基 → 不与 FeCl₃ 显紫色。

**与碘-淀粉显色的对比**：
- 苯酚 + FeCl₃ → 紫色（检验酚）
- 淀粉 + I₂ → 蓝黑色（检验淀粉或碘）

### 典型例题

**例**：如何用 FeCl₃ 溶液区分苯酚溶液和乙醇溶液？

解：分别滴加 FeCl₃ 溶液：
- 苯酚溶液：出现**紫色**（酚羟基与 Fe³⁺ 络合）
- 乙醇溶液：**无明显颜色变化**（乙醇无酚羟基）

**例**：某有机物 A 与 FeCl₃ 显紫色，A 一定含有什么基团？

解：A 一定含有**酚羟基**（Ar—OH），即—OH 直接连在苯环上。

**例**：淀粉水解是否完全，如何检验？

解：取水解液，分为两份：
1. 加碘液：若**变蓝色**，说明还有淀粉未水解（葡萄糖不与碘反应）
2. 加银氨溶液水浴：若有**银镜**，说明有葡萄糖（说明有水解）
（注：FeCl₃ 显色是检验酚，不直接用于检验淀粉水解）

### 易错点

- FeCl₃ 显紫色是**配位反应**（络合），不是氧化还原（Fe³⁺ 不被还原）
- 不是所有含—OH 的化合物都与 FeCl₃ 显紫色，必须是**酚羟基**（连苯环）
- 醋酸（乙酸）加 FeCl₃ 会使 Fe³⁺ 水解减少，溶液颜色变化为黄色→变浅（不是紫色）
