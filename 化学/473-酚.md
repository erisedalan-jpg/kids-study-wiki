# 酚

> **一句话**：酚是含有酚羟基（—OH 直接连在苯环上）的有机化合物，酸性比醇强（弱酸，pKa≈10），与 FeCl₃ 显紫色，苯酚是最重要的代表物。
> **English**: Phenols contain a hydroxyl group (—OH) directly bonded to a benzene ring; they are weakly acidic (pKa ≈ 10), give a purple color with FeCl₃, and phenol itself is the most important representative.

---

## 🎓 高中进阶

### 定义 / 结构特征

**酚**（phenols）：苯环（或芳环）上的 H 被**羟基（—OH）**直接取代所得的化合物。

**酚羟基 vs 醇羟基**：

| 性质 | 酚羟基（Ar—OH）| 醇羟基（R—OH）|
|------|--------------|-------------|
| 连接碳的杂化 | sp²（苯环）| sp³（饱和碳）|
| 酸性 | 弱酸（pKa≈10），强于碳酸氢盐 | 极弱酸（pKa≈16），弱于碳酸 |
| 与 NaOH | 反应（中和）| 不反应 |
| 与 Na₂CO₃ | 反应放 CO₂？注意：酚酸性弱于碳酸，不放 CO₂，但可与 Na₂CO₃ 反应（苯酚酸性 > H₂CO₃ 二级酸，强于 HCO₃⁻）| 不反应 |
| FeCl₃ 显色 | 紫色（鉴别酚的特征）| 无颜色变化 |
| 被 KMnO₄ 氧化 | 易被氧化（苯环活化）| 较难 |

**酚的酸性**：C₆H₅OH ⇌ C₆H₅O⁻ + H⁺（苯酚的 pKa ≈ 10，比 H₂CO₃ 弱，比 HCO₃⁻ 强）

### 性质 / 反应

| 反应 | 方程式 | 条件 |
|------|--------|------|
| 与 NaOH | C₆H₅OH + NaOH → C₆H₅ONa + H₂O | 常温 |
| 与 Na | 2C₆H₅OH + 2Na → 2C₆H₅ONa + H₂↑ | 常温 |
| 与 Na₂CO₃ | 2C₆H₅OH + Na₂CO₃ → 2C₆H₅ONa + H₂O + CO₂（不对）| 实际：C₆H₅OH + Na₂CO₃ → C₆H₅ONa + NaHCO₃（只能到 NaHCO₃，不能 CO₂↑）|
| FeCl₃ 显色 | C₆H₅OH + FeCl₃ → 紫色络合物 | 常温 |
| 苯环溴代 | C₆H₅OH + 3Br₂ → 2,4,6-三溴苯酚↓ + 3HBr | 常温，溴水 |
| 苯环硝化 | C₆H₅OH + HNO₃ → 硝基苯酚（邻、对位）| 稀HNO₃，室温 |

### 典型例题

**例**：苯酚（C₆H₅OH）、乙醇（C₂H₅OH）、碳酸（H₂CO₃）的酸性强弱排列。

解：**碳酸 > 苯酚 > 碳酸氢根（HCO₃⁻）> 乙醇**
（苯酚能与 Na₂CO₃ 反应生成 NaHCO₃，说明苯酚酸性强于 HCO₃⁻；但苯酚不能与 NaHCO₃ 反应，说明苯酚酸性弱于 H₂CO₃）

**例**：如何用化学方法鉴别苯酚和乙醇？

解：加入 FeCl₃ 溶液：
- 苯酚：溶液变**紫色**
- 乙醇：溶液颜色**无明显变化**

**例**：苯酚与足量溴水反应，现象是什么？

解：苯酚与溴水迅速反应，生成白色沉淀**2,4,6-三溴苯酚**（不溶于水），溴水褪色。
C₆H₅OH + 3Br₂ → C₆H₂Br₃OH↓(白) + 3HBr

### 易错点

- 苯酚酸性：弱于 H₂CO₃，强于 HCO₃⁻（= 碳酸一级酸度 < 苯酚，二级酸度 > 苯酚）
- 苯酚与 Na₂CO₃ 反应只生成 NaHCO₃（不放 CO₂！），与苯酚酸性强于 HCO₃⁻ 一致
- 苯酚溶于 NaOH 溶液（生成苯酚钠），向其中通 CO₂ 可重新析出苯酚（CO₂ + H₂O 酸性强于苯酚）
