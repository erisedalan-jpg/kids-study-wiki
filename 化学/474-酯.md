# 酯

> **一句话**：酯是羧酸与醇发生酯化反应（酸脱羟基，醇脱氢）生成的化合物，含酯基（—COO—），可在酸或碱催化下水解，乙酸乙酯是典型代表。
> **English**: Esters are formed from carboxylic acids and alcohols by esterification (acid loses —OH, alcohol loses H); they contain the ester group (—COO—) and can be hydrolyzed under acid or base catalysis; ethyl acetate is the classic example.

---

## 🎓 高中进阶

### 定义 / 结构特征

**酯**（ester）：含**酯基**（—COO—，—C(=O)—O—）的有机物，通式 RCOOR'。

**命名**：酸名 + 醇名 + 酯（如乙酸乙酯 = 乙酸 + 乙醇 + 酯）

**常见酯**：

| 名称 | 结构简式 | 来源 | 气味 |
|------|---------|------|------|
| 乙酸乙酯 | CH₃COOC₂H₅ | 乙酸 + 乙醇 | 果香 |
| 乙酸甲酯 | CH₃COOCH₃ | 乙酸 + 甲醇 | |
| 甲酸乙酯 | HCOOC₂H₅ | 甲酸 + 乙醇 | |
| 乙酸异戊酯 | CH₃COOCH₂CH₂CH(CH₃)₂ | 乙酸 + 异戊醇 | 香蕉香 |
| 油脂（甘油三酯）| 甘油 + 脂肪酸 | 天然 | |

**酯的物理性质**：
- 低级酯：无色液体，有芳香气味，密度小于水，难溶于水
- 与有机溶剂互溶（良好有机溶剂）

### 性质 / 反应

**酯化反应**（可逆）：
RCOOH + R'OH ⇌ RCOOR' + H₂O（浓H₂SO₄，△）

**酯的水解**（可逆程度不同）：

| 水解类型 | 条件 | 方程式 | 特点 |
|---------|------|--------|------|
| 酸催化水解 | 稀酸，加热 | RCOOR' + H₂O ⇌ RCOOH + R'OH | 可逆，平衡限制 |
| 碱催化水解（皂化）| NaOH，加热 | RCOOR' + NaOH → RCOONa + R'OH | 不可逆，产物为羧酸盐 |

**碱催化水解更彻底**（NaOH 与羧酸反应，使平衡完全正向移动）。

### 典型例题

**例**：乙酸乙酯（CH₃COOC₂H₅）在 NaOH 溶液中水解的方程式和产物。

解：CH₃COOC₂H₅ + NaOH →(△) CH₃COONa + C₂H₅OH
产物：乙酸钠（CH₃COONa）和乙醇（C₂H₅OH）。

**例**：酯化反应和水解反应互逆关系。

解：
- 酯化：RCOOH + R'OH ⇌ RCOOR' + H₂O（浓H₂SO₄，正方向）
- 水解：RCOOR' + H₂O ⇌ RCOOH + R'OH（稀酸，逆方向）
两者是互逆反应，化学平衡。

**例**：从分子式 C₄H₈O₂ 出发，写出可能的酯类同分异构体。

解：C₄H₈O₂ 不饱和度 = (2×4+2-8)/2 = 1，含一个 C=O（为酯基）：
1. 甲酸丙酯：HCOOCH₂CH₂CH₃
2. 甲酸异丙酯：HCOOCH(CH₃)₂
3. 乙酸乙酯：CH₃COOC₂H₅
4. 丙酸甲酯：CH₃CH₂COOCH₃
共 4 种酯类异构体。

### 易错点

- 酯化反应断键：酸的—C(=O)—OH 中 **O—H 键断**（酸脱羟基），醇的 **H—O—R 中 O—H 键断**（醇脱 H），水中 O 来自酸
- 酯水解在碱性条件下不可逆（生成羧酸盐），在酸性条件下可逆
- 酯与醚（R—O—R）结构不同：酯含 C=O，醚不含 C=O；它们不互为官能团异构

### 例句

- An **ester** is formed by the reaction of a **carboxylic acid** and an **alcohol**, eliminating water.
- **Saponification** (alkaline hydrolysis) of esters is **irreversible**, giving a carboxylate salt and an alcohol.
