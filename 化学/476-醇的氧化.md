# 醇的氧化

> **一句话**：醇的氧化分催化氧化（Cu/O₂）和强氧化剂氧化（KMnO₄）两类：伯醇→醛→羧酸，仲醇→酮，叔醇不被 Cu 催化氧化。
> **English**: Alcohol oxidation includes catalytic oxidation (Cu/O₂) and strong oxidant oxidation (KMnO₄): primary alcohols give aldehydes then carboxylic acids, secondary give ketones, tertiary cannot be oxidized by catalytic method.

---

## 🎓 高中进阶

### 定义 / 结构特征

**醇的氧化**：醇被氧化剂脱去 α-H（连—OH 碳上的 H），官能团由—OH 转变为—CHO（醛）或 >C=O（酮）。

**氧化产物规律**：
- **伯醇**（α-碳有 2 个 H）：→ **醛** → 再氧化 → **羧酸**
- **仲醇**（α-碳有 1 个 H）：→ **酮**（酮不易被进一步氧化）
- **叔醇**（α-碳无 H）：**不能**被 Cu 催化氧化

### 性质 / 反应

| 类型 | 方程式示例 | 条件 |
|------|-----------|------|
| 伯醇→醛（Cu 催化）| 2CH₃CH₂OH + O₂ → 2CH₃CHO + 2H₂O | Cu，△ |
| 仲醇→酮（Cu 催化）| 2CH₃CHOHCH₃ + O₂ → 2CH₃COCH₃ + 2H₂O | Cu，△ |
| 醛→酸（KMnO₄）| CH₃CHO + [O] → CH₃COOH | 酸性 KMnO₄ |
| 醇被酸性 KMnO₄ | 伯/仲醇被 KMnO₄ 氧化，溶液褪色 | 酸性 KMnO₄ |

**铜催化氧化机理**：
- 步骤 1：Cu + O₂ →(△) CuO（铜丝烧黑）
- 步骤 2：CuO + RCH₂OH → Cu + RCHO + H₂O（铜丝重新变红）
- 总：2RCH₂OH + O₂ →(Cu,△) 2RCHO + 2H₂O

**实验现象**：铜丝在酒精灯上交替变黑（CuO）变红（Cu），可闻到刺激性气味（醛）

### 典型例题

**例**：1-丙醇和 2-丙醇的催化氧化产物各是什么？

解：
- 1-丙醇（伯醇）CH₃CH₂CH₂OH → **丙醛 CH₃CH₂CHO**（—CHO 在 C₁）
- 2-丙醇（仲醇）CH₃CH(OH)CH₃ → **丙酮 CH₃COCH₃**（酮）

**例**：如何鉴别乙醛（产物）和丙酮？

解：
- 银镜反应：乙醛有银镜，丙酮无（醛有还原性，酮无）
- Cu(OH)₂ 加热：乙醛出现砖红色沉淀 Cu₂O，丙酮无

**例**：甲醇（CH₃OH，伯醇）催化氧化产物是什么？

解：甲醇氧化 → **甲醛（HCHO）**；甲醛进一步氧化 → **甲酸（HCOOH）**；甲酸再氧化 → CO₂ + H₂O

### 易错点

- 叔醇（如叔丁醇）α-C 无 H，不能被 Cu 催化氧化（但能被强 KMnO₄ 氧化，碳链断裂）
- 醛可以被 KMnO₄ 继续氧化为羧酸（比酮更易被氧化）
- 铜催化氧化是"变红→变黑→变红"循环，Cu 作催化剂被循环利用

### 例句

- **Primary alcohols** are oxidized to **aldehydes** by Cu/O₂ catalytic oxidation.
- **Secondary alcohols** give **ketones**, which are harder to oxidize further.
