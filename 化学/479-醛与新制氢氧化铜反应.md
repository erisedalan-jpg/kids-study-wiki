# 醛与新制氢氧化铜反应

> **一句话**：醛与新制 Cu(OH)₂ 在加热条件下反应，醛被氧化为羧酸，Cu(OH)₂ 被还原为砖红色的 Cu₂O 沉淀，此反应（类 Fehling 检验）可鉴别含醛基的化合物。
> **English**: Aldehydes react with freshly prepared Cu(OH)₂ on heating; the aldehyde is oxidized to a carboxylic acid and Cu(OH)₂ is reduced to brick-red Cu₂O precipitate; this Fehling-type test identifies compounds containing aldehyde groups.

---

## 🎓 高中进阶

### 定义 / 结构特征

**醛与新制 Cu(OH)₂ 反应**（Fehling-type test）：
- 试剂：**新制 Cu(OH)₂**（在 NaOH 溶液中加 CuSO₄，现制现用，蓝色絮状沉淀）
- 底物：含—CHO 的化合物（醛、葡萄糖等还原糖）
- 条件：**加热**（直接加热或水浴均可）
- 现象：蓝色沉淀 → **砖红色沉淀**（Cu₂O），溶液变色

**新制 Cu(OH)₂ 制备**：CuSO₄(稀) + 2NaOH(稀) → Cu(OH)₂↓(蓝色) + Na₂SO₄（现用现制，不能久置变为 CuO）

### 性质 / 反应

**反应方程式（以乙醛为例）**：
CH₃CHO + 2Cu(OH)₂ →(△) CH₃COOH + Cu₂O↓(砖红) + 2H₂O

**与银镜反应对比**：

| 对比项 | 银镜反应（Tollens）| Cu(OH)₂ 反应（Fehling）|
|--------|---------------------|----------------------|
| 试剂 | 银氨溶液（Ag⁺）| 新制 Cu(OH)₂ |
| 产物（氧化）| 银（Ag，光亮镜面）| Cu₂O（砖红色沉淀）|
| 产物（底物）| 羧酸铵盐 | 羧酸 |
| 加热 | 水浴（温和）| 直接加热可 |
| 用途 | 检验—CHO | 检验—CHO（含还原糖）|

**能发生此反应的化合物**：
- 所有醛（RCHO）
- 甲酸（HCOOH，含—CHO 性质）
- 葡萄糖、麦芽糖（还原糖，含—CHO）
- 不能：酮（R₂CO）、蔗糖（非还原糖）

### 典型例题

**例**：实验步骤——检验葡萄糖溶液含有—CHO。

解：
1. 向试管中加入 2 mL NaOH 溶液（2 mol/L）
2. 加入 4 滴 CuSO₄ 溶液（0.5 mol/L），生成蓝色 Cu(OH)₂
3. 加入 1 mL 葡萄糖溶液
4. 直接加热（酒精灯）或水浴加热
5. 现象：蓝色变消失，出现**砖红色沉淀**（Cu₂O）→ 证明含—CHO

**例**：为什么必须用"新制"Cu(OH)₂？

解：Cu(OH)₂ 放置后会脱水转变为 CuO（黑色）：Cu(OH)₂ →(△) CuO + H₂O
CuO 无相同的检验效果。故必须现制现用，保证 Cu(OH)₂ 为蓝色絮状形态。

**例**：丙酮（CH₃COCH₃）能与新制 Cu(OH)₂ 反应吗？

解：丙酮是酮，含>C=O 但无—CHO 中的 H，无还原性，**不能**与新制 Cu(OH)₂ 在加热下生成砖红色沉淀（不发生 Fehling 型反应）。

### 易错点

- 砖红色沉淀是 **Cu₂O**（一氧化二铜），不是 Cu（铜）或 CuO（黑色）
- 新制 Cu(OH)₂ 必须在**碱性**环境下制备和使用（NaOH 过量），否则 Cu²⁺ 会沉淀不完全
- 蔗糖（非还原糖）不反应，但水解产物（葡萄糖+果糖）均能反应

### 例句

- **Aldehydes** react with freshly prepared **Cu(OH)₂** on heating to give **brick-red Cu₂O** precipitate.
- This **Fehling-type test** is positive for **reducing sugars** (like glucose) but negative for **sucrose** (a non-reducing sugar).
