# 不饱和烃

> **一句话**：不饱和烃含有碳碳双键或三键，化学性质活泼，能发生加成反应、氧化反应和加聚反应，包括烯烃、炔烃和二烯烃。
> **English**: Unsaturated hydrocarbons contain C=C double bonds or C≡C triple bonds; they are chemically reactive and undergo addition, oxidation, and addition polymerization; they include alkenes, alkynes, and dienes.

---

## 🎓 高中进阶

### 定义 / 结构特征

**不饱和烃**：分子中含有 C=C 双键或 C≡C 三键的烃，碳的价键未被氢"充满"（不饱和）。

**主要类型**：

| 类型 | 通式 | 不饱和度 | 代表物 |
|------|------|---------|--------|
| 单烯烃 | CₙH₂ₙ（n≥2）| 1 | 乙烯、丙烯 |
| 二烯烃 | CₙH₂ₙ₋₂（n≥3）| 2 | 1,3-丁二烯 |
| 单炔烃 | CₙH₂ₙ₋₂（n≥2）| 2 | 乙炔、丙炔 |
| 芳香烃（苯系）| 特殊大π键 | 4（苯）| 苯、甲苯 |

**不饱和度（Ω）计算**：
Ω = (2C + 2 - H + N - X) / 2（其中 O 不计入）
- 每个 C=C 或 C=O 贡献 1；C≡C 贡献 2；苯环贡献 4；每个环贡献 1

### 性质 / 反应

| 反应类型 | 条件 | 示例 |
|---------|------|------|
| 加成（与 H₂）| Ni 催化，加热 | CH₂=CH₂ + H₂ → CH₃CH₃ |
| 加成（与 Br₂）| 常温 | CH₂=CH₂ + Br₂ → CH₂BrCH₂Br |
| 加成（与 HX）| 常温 | CH₂=CH₂ + HBr → CH₃CH₂Br |
| 加成（与 H₂O）| 酸催化 | CH₂=CH₂ + H₂O → CH₃CH₂OH |
| 氧化（KMnO₄）| 酸性，常温 | 双键或三键被氧化，溶液褪色 |
| 加聚（聚合）| 催化剂 | nCH₂=CH₂ → (—CH₂—CH₂—)ₙ |

**马氏规则（Markovnikov 规则）**：不对称烯烃与 HX 加成时，H 加到含 H 较多的碳上，X 加到含 H 较少的碳上：
CH₃CH=CH₂ + HBr → CH₃CHBrCH₃（主产物，2-溴丙烷）

### 典型例题

**例**：比较乙烷、乙烯、乙炔与 Br₂/CCl₄ 的反应。

解：
- 乙烷（饱和烃）：不反应（需光照才能取代，与 Br₂ 的 CCl₄ 溶液不反应）
- 乙烯（含 C=C）：**溴水/Br₂ 迅速褪色**，发生加成：CH₂=CH₂ + Br₂ → CH₂BrCH₂Br
- 乙炔（含 C≡C）：同样使溴水褪色（先加成一次，再加成一次）

**例**：计算苯（C₆H₆）的不饱和度。

解：Ω = (2×6 + 2 - 6) / 2 = (12+2-6)/2 = 8/2 = **4**（3 个双键等效 + 1 个环，或大π键贡献）

### 易错点

- 不饱和烃使溴水褪色是**加成**（不是取代），溴水中溴分子被消耗
- 苯不能使溴水褪色（苯环不是独立 C=C，不能加成），但在铁催化下发生**取代**（溴代苯）
- 炔烃与过量 HX 反应可加成两次，生成偕二卤代烃
