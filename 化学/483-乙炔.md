# 乙炔

> **一句话**：乙炔（HC≡CH）是最简单的炔烃，工业上由电石（CaC₂）与水反应制备，是重要的有机合成原料，可制备氯乙烯、乙醛等。
> **English**: Acetylene (HC≡CH) is the simplest alkyne; industrially produced by the reaction of calcium carbide with water, it is an important organic synthesis feedstock for vinyl chloride, acetaldehyde, and more.

---

## 🎓 高中进阶

### 定义 / 结构特征

**乙炔**（acetylene，ethyne）：
- 分子式 C₂H₂，结构简式 HC≡CH
- 两个碳均为 **sp 杂化**，分子为**直线形**，键角 180°
- C≡C 键长 120 pm，键能 838 kJ/mol
- 不饱和度 Ω = 2

**制备方法**：

| 方法 | 方程式 | 规模 |
|------|--------|------|
| 电石水解（实验室/工业）| CaC₂ + 2H₂O → Ca(OH)₂ + HC≡CH↑ | 中小规模 |
| 甲烷裂解（工业）| 2CH₄ →(高温) HC≡CH + 3H₂ | 大规模 |

**物理性质**：无色气体，微溶于水，可溶于丙酮；纯乙炔在一定压力下不稳定（易爆），工业上用丙酮吸收存储。

### 性质 / 反应

| 反应类型 | 方程式 | 条件 |
|---------|--------|------|
| 加氢（→乙烯）| HC≡CH + H₂ → CH₂=CH₂ | Pd/BaSO₄，林德拉催化 |
| 加氢（→乙烷）| HC≡CH + 2H₂ → CH₃CH₃ | Ni，加热 |
| 加 HCl（→氯乙烯）| HC≡CH + HCl → CH₂=CHCl | HgCl₂/活性炭，加热 |
| 加 H₂O（→乙醛）| HC≡CH + H₂O → CH₃CHO | HgSO₄/H₂SO₄，加热（Markovnikov）|
| 加 Br₂（一次）| HC≡CH + Br₂ → CHBr=CHBr | CCl₄ |
| 三聚（→苯）| 3HC≡CH →(高温，活性炭) C₆H₆ | 高温 |
| 端炔银盐 | HC≡CH + 2Ag(NH₃)₂⁺ → AgC≡CAg↓ + 2NH₄⁺ + 2NH₃ | 室温 |
| 燃烧 | 2HC≡CH + 5O₂ → 4CO₂ + 2H₂O | 点燃，火焰温度高（氧炔焰）|

**氧炔焰**：乙炔与氧气充分燃烧，温度可达 3000°C 以上，用于金属焊接和切割。

### 典型例题

**例**：由电石制乙炔，向锥形瓶中加水，如何控制反应速率？

解：用饱和食盐水代替纯水（降低反应速率，因 CaC₂ 与水反应剧烈），或用分液漏斗控制加水速度，收集乙炔前需用浓硫酸除去 H₂S、PH₃ 等杂质气体。

**例**：乙炔与银氨溶液反应的方程式。

解：HC≡CH + 2Ag(NH₃)₂OH → AgC≡CAg↓(白) + 4NH₃ + 2H₂O

**例**：如何由乙炔制备氯乙烯（聚氯乙烯单体）？

解：HC≡CH + HCl →(HgCl₂/活性炭，加热) CH₂=CHCl（氯乙烯）
再加聚：nCH₂=CHCl → (—CH₂CHCl—)ₙ（聚氯乙烯 PVC）

### 易错点

- 乙炔银盐（AgC≡CAg，灰白色）干燥后极不稳定，有爆炸危险，实验完毕需用稀硝酸销毁
- 乙炔加水（水化）主产物是**乙醛**（不是乙醇）——遵循马氏规则
- 乙炔三聚成苯需高温，实验室不易实现
