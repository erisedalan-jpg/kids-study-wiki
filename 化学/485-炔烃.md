# 炔烃

> **一句话**：炔烃含有碳碳三键（C≡C），通式 CₙH₂ₙ₋₂，化学性质比烯烃更活泼，端炔的端位 C—H 具有微弱酸性，乙炔是最重要的代表物。
> **English**: Alkynes contain a C≡C triple bond, have the formula CₙH₂ₙ₋₂, are more reactive than alkenes, and terminal alkynes have weakly acidic C—H bonds; acetylene (ethyne) is the most important representative.

---

## 🎓 高中进阶

### 定义 / 结构特征

**炔烃**（alkynes）：含有 C≡C 三键的开链不饱和烃，通式 **CₙH₂ₙ₋₂**（n≥2）。

**三键结构**：
- C≡C = 1 个 σ 键 + 2 个 π 键
- 两端碳均为 **sp 杂化**，键角 180°，C≡C—C 呈直线形
- 键长 120 pm（短于 C=C 的 134 pm）
- 键能 838 kJ/mol（最强的 C—C 键）

**端炔（terminal alkyne）**的 C—H 酸性：
- 端炔：C≡C 在链端，结构如 R—C≡C—H
- sp 碳电负性强，使 ≡C—H 的 H 具有微弱酸性（pKa ≈ 25）
- 能与 AgNO₃ 氨溶液反应：R—C≡C—H + Ag(NH₃)₂⁺ → R—C≡C—Ag↓（白色沉淀）

### 性质 / 反应

| 反应类型 | 方程式 | 条件 |
|---------|--------|------|
| 加氢（一次）| HC≡CH + H₂ → CH₂=CH₂ | Pd/BaSO₄，降低活性 |
| 加氢（二次）| HC≡CH + 2H₂ → CH₃CH₃ | Ni，加热 |
| 加溴（一次）| HC≡CH + Br₂ → CHBr=CHBr | CCl₄ |
| 加溴（二次）| HC≡CH + 2Br₂ → CHBr₂CHBr₂ | 过量 Br₂ |
| 加 HCl | HC≡CH + HCl → CH₂=CHCl（氯乙烯）| HgCl₂，加热 |
| 端炔银盐 | RC≡CH + Ag(NH₃)₂⁺ → RC≡CAg↓ | 室温 |
| 氧化 | C≡C 被 KMnO₄ 氧化，溶液褪色 | 酸性 KMnO₄ |

**炔烃与烯烃的区别**：
- 炔烃不饱和度为 2，烯烃为 1
- 炔烃与 Br₂ 可加成两次（需过量）
- 端炔有特殊酸性（检验方法：银氨溶液沉淀）

### 典型例题

**例**：如何用化学方法区分乙烯（CH₂=CH₂）和乙炔（HC≡CH）？

解：将气体通入**银氨溶液**（AgNO₃ 氨水）：
- 乙炔：产生白色沉淀（HC≡CAg，炔化银）→ **乙炔**
- 乙烯：无沉淀（无端炔 C—H）→ **乙烯**

**例**：计算 1 mol 乙炔完全加氢需要多少 mol H₂？

解：HC≡CH + 2H₂ → CH₃CH₃，需 **2 mol H₂**。

**例**：乙炔与过量 Br₂/CCl₄ 反应，最终产物是什么？

解：两步加成：HC≡CH + Br₂ → CHBr=CHBr（1,2-二溴乙烯）；再 + Br₂ → CHBr₂CHBr₂（1,1,2,2-四溴乙烷）

### 易错点

- 炔烃中 C≡C 含 2 个 π 键，加成可发生 1 次或 2 次（取决于 Br₂/H₂ 的用量）
- 端炔银盐（炔化银）不稳定，干燥后加热或撞击会爆炸，实验后需用稀硝酸分解
- 乙炔（HC≡CH）线形分子，无顺反异构
