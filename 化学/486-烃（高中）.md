# 烃（高中）

> **一句话**：烃是仅由碳和氢两种元素组成的有机化合物，是有机化学的基础，按结构分为饱和烃（烷烃）和不饱和烃（烯烃、炔烃、芳香烃）。
> **English**: Hydrocarbons are organic compounds containing only carbon and hydrogen; they are classified into saturated hydrocarbons (alkanes) and unsaturated hydrocarbons (alkenes, alkynes, arenes).

---

## 🎓 高中进阶

### 定义 / 结构特征

**烃**（hydrocarbon）：仅含 **C** 和 **H** 的有机化合物。

**分类体系**：

```
烃
├── 开链烃（脂肪烃）
│   ├── 饱和烃：烷烃 CₙH₂ₙ₊₂
│   └── 不饱和烃
│       ├── 烯烃 CₙH₂ₙ（单烯）
│       ├── 二烯烃 CₙH₂ₙ₋₂（如 1,3-丁二烯）
│       └── 炔烃 CₙH₂ₙ₋₂（单炔）
└── 环状烃
    ├── 脂环烃（环烷烃等）
    └── 芳香烃（含苯环）
        ├── 苯
        └── 苯的同系物（甲苯、二甲苯…）
```

**通式对比**：

| 类别 | 通式 | 不饱和度 |
|------|------|---------|
| 烷烃 | CₙH₂ₙ₊₂ | 0 |
| 烯烃（单烯）| CₙH₂ₙ | 1 |
| 炔烃（单炔）| CₙH₂ₙ₋₂ | 2 |
| 苯 | C₆H₆ | 4 |

### 性质 / 反应

| 类别 | 代表反应 | 特征 |
|------|---------|------|
| 烷烃 | 取代（卤代）、燃烧 | 无官能团，化学性质稳定 |
| 烯烃 | 加成、加聚、氧化 | C=C 双键活泼 |
| 炔烃 | 加成、取代（端炔 H 有酸性）| C≡C 三键 |
| 芳香烃 | 取代（卤代/硝化/磺化）| 苯环稳定，以取代为主 |

### 典型例题

**例**：乙烷、乙烯、乙炔的分子式分别是什么？各属于哪类烃？

解：
- 乙烷 C₂H₆（n=2, 2×2+2=6）→ 烷烃（饱和烃）
- 乙烯 C₂H₄（2×2=4）→ 烯烃（不饱和烃）
- 乙炔 C₂H₂（2×2-2=2）→ 炔烃（不饱和烃）

**例**：苯（C₆H₆）不能使溴水褪色，但甲苯可以使高锰酸钾溶液褪色，如何解释？

解：苯环高度共轭稳定，不含独立 C=C，不能与 Br₂ 发生加成；甲苯苯环侧链—CH₃ 受苯环活化，被 KMnO₄ 氧化为—COOH，使溶液褪色（不是苯环被氧化）。

### 易错点

- 苯既不是饱和烃也不能简单说是不饱和烃——苯环具有特殊大π键，不等同于三个 C=C
- 不饱和度（Ω）= (2C + 2 + N - H - X) / 2，用于判断环和不饱和键数目
