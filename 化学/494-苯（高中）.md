# 苯（高中）

> **一句话**：苯（C₆H₆）是最简单的芳香烃，6个碳 sp² 杂化形成平面六边形，大π键赋予其独特稳定性，主要发生取代反应而非加成反应。
> **English**: Benzene (C₆H₆) is the simplest aromatic hydrocarbon; 6 sp²-hybridized carbons form a planar hexagon; its delocalized π system confers unique stability, making substitution—not addition—its dominant reaction.

## 🎓 高中进阶

### 定义 / 结构特征

**苯**（benzene）：
- 分子式 C₆H₆，不饱和度 Ω = 4
- 结构：平面六边形，6 个 C 等价，6 个 H 等价
- 所有 C—C 键等长（140 pm），介于单双键之间
- 6 个碳各贡献 1 个 p 轨道，形成覆盖整个分子的**大π键**（6π 电子离域）

**物理性质**：
- 无色液体，有特殊香味，沸点 80.1°C，熔点 5.5°C
- 密度 0.879 g/mL（小于水）
- 难溶于水，与乙醇、乙醚等有机溶剂互溶
- **有毒**（致癌物），使用时注意通风

**苯的结构表示**：
- 凯库勒结构式（交替单双键）——历史模型，不准确
- 正确认识：六边形中间画圆圈表示离域π电子

### 性质 / 反应

| 反应 | 方程式 | 条件 |
|------|--------|------|
| 溴代（取代）| C₆H₆ + Br₂ → C₆H₅Br + HBr | Fe（铁催化）|
| 硝化（取代）| C₆H₆ + HNO₃ → C₆H₅NO₂ + H₂O | 浓H₂SO₄，50-60°C |
| 磺化（取代）| C₆H₆ + H₂SO₄(浓) → C₆H₅SO₃H + H₂O | 加热 |
| 加氢（加成）| C₆H₆ + 3H₂ → C₆H₁₂ | Ni，高温高压（难）|
| 燃烧 | 2C₆H₆ + 15O₂ → 12CO₂ + 6H₂O | 点燃，浓烟 |

**苯不能发生的反应**：
- ✗ 不能使溴水褪色（不加成）
- ✗ 不能使酸性 KMnO₄ 褪色（苯环稳定）
- ✗ 不能使石蕊变色（苯无酸碱性）

### 典型例题

**例**：苯与液溴（Fe 催化）和溴水各有何现象？

解：
- 与液溴（Fe 催化）：**发生取代反应**，生成溴苯（C₆H₅Br）和 HBr，有刺激性气味气体产生
- 与溴水：苯**萃取**溴水中的溴（有机层变橙红，水层褪色），**不是化学反应**（苯不与 Br₂ 发生化学反应）

**例**：硝化苯（C₆H₅NO₂）的制备。

解：苯 + 浓 HNO₃ →(浓H₂SO₄,50-60°C) 硝基苯（黄色油状液体，有苦杏仁味，密度 > 水，不溶于水）

**例**：1 mol 苯完全燃烧需多少 mol O₂？

解：C₆H₆ + 15/2 O₂ → 6CO₂ + 3H₂O，需 **7.5 mol O₂**。

### 易错点

- 苯与溴水"褪色"是**萃取**（物理过程），不是苯与 Br₂ 发生化学反应
- 硝化反应中，浓 H₂SO₄ 是催化剂和脱水剂，同时提高 NO₂⁺（硝酰基）浓度
- 苯的燃烧产物是 CO₂ 和 H₂O（完全燃烧），但实际燃烧会有黑烟（碳粒）
