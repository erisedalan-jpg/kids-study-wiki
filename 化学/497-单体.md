# 单体

> **一句话**：单体是合成高分子化合物的小分子原料，加聚单体含不饱和键（C=C），缩聚单体含两个或多个官能团（—OH、—COOH、—NH₂等）。
> **English**: A monomer is the small-molecule building block for synthesizing polymers; addition polymerization monomers contain unsaturated bonds (C=C), while condensation polymerization monomers contain two or more functional groups (—OH, —COOH, —NH₂, etc.).

---

## 🎓 高中进阶

### 定义 / 结构特征

**单体**（monomer）：合成高分子的**小分子原料**，通过聚合反应形成高分子链的重复单元。

**加聚单体特征**：
- 含 C=C 双键（烯烃、二烯烃等）
- 双键打开，各加成连接

**缩聚单体特征**：
- 含 ≥2 个可以缩合的官能团（如—OH + —COOH、—NH₂ + —COOH）
- 缩合脱去小分子

**常见单体与聚合物对照**：

| 单体 | 聚合物 | 反应类型 |
|------|--------|---------|
| 乙烯 CH₂=CH₂ | 聚乙烯（PE）| 加聚 |
| 丙烯 CH₂=CHCH₃ | 聚丙烯（PP）| 加聚 |
| 氯乙烯 CH₂=CHCl | 聚氯乙烯（PVC）| 加聚 |
| 苯乙烯 CH₂=CHC₆H₅ | 聚苯乙烯（PS）| 加聚 |
| 1,3-丁二烯 CH₂=CHCH=CH₂ | 顺丁橡胶 | 加聚 |
| 乙二醇 + 对苯二甲酸 | 涤纶（PET）| 缩聚 |
| 己二胺 + 己二酸 | 尼龙-6,6 | 缩聚 |

### 性质 / 反应

**由聚合物推断单体的方法**：

1. **加聚物**：链节恢复 C=C（将相邻两个碳之间恢复双键）
   - (—CH₂—CHCl—)ₙ → 单体：CH₂=CHCl（氯乙烯）
   
2. **缩聚物**：在特征键（酯键、酰胺键）处断裂，各端补 H 或 OH（或 NH₂）
   - 涤纶 (—OCH₂CH₂OOCC₆H₄CO—)ₙ → 乙二醇 + 对苯二甲酸

**加聚单体的计算**：
n mol 单体（M₀）→ 1 mol 聚合物（分子量 = n × M₀，无副产物）

### 典型例题

**例**：聚丙烯的结构单元为 (—CH₂—CHCH₃—)，其单体是什么？

解：恢复相邻两碳间的 C=C：**丙烯（CH₂=CHCH₃）**

**例**：以下哪些物质可作为加聚单体？

A. 乙烷 CH₃CH₃（✗，无 C=C）
B. 乙烯 CH₂=CH₂（✓，含 C=C）
C. 乙酸 CH₃COOH（✗，无 C=C，但含—COOH 可作缩聚单体之一）
D. 氯乙烯 CH₂=CHCl（✓，含 C=C）

**例**：一种高分子的结构单元为 —NH(CH₂)₆NHCO(CH₂)₄CO—，推断其单体。

解：含酰胺键（—CO—NH—），为缩聚产物，在酰胺键处断裂：
- H₂N(CH₂)₆NH₂（己二胺）
- HOOC(CH₂)₄COOH（己二酸）
→ **尼龙-6,6** 的两种单体

### 易错点

- 加聚单体必须含**不饱和键**（C=C 或 C≡C），饱和烃不能直接加聚
- 缩聚单体必须含**两个**可反应的官能团（单官能团只能给端基，不能成链）
- 由聚合物推单体：加聚→恢复 C=C；缩聚→断特征键，各端补 H 或 OH
