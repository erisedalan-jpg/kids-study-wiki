# 氢气

> **一句话**：氢气（H₂）是**最轻、最简单**的气体，**无色无味**，化学性质活泼，**燃烧只生成水**——是清洁能源的重要候选，也是化学中最早研究的气体之一。
> **English**: *Hydrogen gas* (H₂) is the lightest and simplest gas — colorless, odorless; chemically active; combusts to produce only water; a key candidate for clean energy and one of the earliest gases studied in chemistry.

---

## 📚 给 6-12 岁

氢气：**最轻的气体**——比空气轻得多。

身边的氢气：
- 氢气球（漂在空中）
- 火箭燃料（液氢）
- 一些燃料电池（如某些公交车）

特性：
- **无色无味**：看不见也闻不到
- **极易燃烧**：遇火会"砰"地燃起来
- **燃烧只生成水**：H₂ + ½O₂ → H₂O，没有任何污染！
- **比空气轻**：所以氢气球会飘起来

**重要安全提示**：氢气和空气混合后遇火会爆炸——千万不要在密闭空间里玩氢气球。

## 🎓 给 12+

**物理性质**：
- 状态：常温常压下气体
- 颜色/气味：无色无味
- 密度：0.0899 g/L（最轻气体，比空气小约 14 倍）
- 沸点：−253 °C（液氢）
- 溶解度：难溶于水

**化学性质**：

### 1. 可燃性
- 在空气中点燃：H₂ + ½O₂ → H₂O（蓝色火焰，仅生成水）
- 在氯气中点燃：H₂ + Cl₂ → 2HCl（苍白色火焰，生成氯化氢）
- **危险**：氢气与空气混合达到爆炸极限（4%—74.2%）遇火爆炸，必须**先验纯**

### 2. 还原性（高中重点）
- 氢气能从金属氧化物中夺取氧——是常见还原剂
- H₂ + CuO → Cu + H₂O（黑色 CuO 变红色 Cu）
- H₂ + Fe₂O₃ → Fe + H₂O（高炉炼铁也用 H₂ 作还原剂）

### 3. 与某些非金属反应
- 与 N₂：N₂ + 3H₂ → 2NH₃（合成氨工业）
- 与 S：H₂ + S → H₂S（高温）

**实验室制法**：
- **原理**：活泼金属（Zn、Fe）+ 稀酸（HCl、稀 H₂SO₄）→ 盐 + H₂↑
- 反应：Zn + 2HCl → ZnCl₂ + H₂↑
- **装置**：固液不加热（启普发生器）
- **收集**：向下排空气法（H₂ 比空气轻）或排水法（H₂ 难溶于水）

**工业制法**：
- **天然气重整**：CH₄ + H₂O → CO + 3H₂（最大产能源）
- **电解水**：2H₂O → 2H₂ + O₂（绿氢，依赖电力来源）
- **煤气化**：C + H₂O → CO + H₂

**应用**：
1. **能源**：燃料电池、火箭推进剂（液氢）
2. **化工**：合成氨、合成甲醇、有机加氢
3. **金属冶炼**：还原剂
4. **食品工业**：油脂氢化（人造黄油）
5. **半导体工业**：保护气体

**绿色能源中的"氢经济"**：氢气燃烧只产水，被认为是清洁能源——但目前 95% 的氢来自化石燃料（"灰氢"），仅少量来自电解水（"绿氢"）。
