# 一定物质的量浓度溶液配制

> **一句话**：以物质的量浓度（c = n / V）为依据，通过计算、称量/量取、溶解/稀释、转移、洗涤、定容、摇匀等一系列操作，配制具有指定体积和确定物质的量浓度的溶液。
> **English**: The experimental procedure to prepare a solution with a specific volume and a known amount-of-substance concentration, based on the definition c = n / V.

## 🎓 考点精讲

**定义与原理**
物质的量浓度（c）定义为单位体积溶液中所含溶质的物质的量，数学表达式为：
\[
c = \frac{n}{V}
\]
其中 n 为溶质的物质的量（mol），V 为溶液的体积（L）。配制溶液的实质是准确量取溶质的物质的量，并将其全部转移至一定体积的溶剂中，最终得到均匀溶液。固体溶质的配制常用容量瓶（一定温度下准确容积的细颈梨形玻璃瓶），液体浓溶液的稀释则遵守稀释定律：
\[
c_1V_1 = c_2V_2
\]
主要仪器：托盘天平（或分析天平）、量筒（或移液管）、烧杯、玻璃棒、容量瓶、胶头滴管。

**关键方法——标准配制步骤与误差分析**
1. **计算**：按目标 c 和 V 算所需溶质质量 m 或浓溶液体积 V₁。
2. **称量/量取**：固体用托盘天平（精确至 0.1 g）或分析天平，液体用量筒或移液管。
3. **溶解/稀释**：在烧杯中用适量水溶解或稀释，冷却至室温（容量瓶有温度限制，避免热胀冷缩引入误差）。
4. **转移**：用玻璃棒引流将烧杯中溶液转入容量瓶，玻璃棒下端靠在刻度线以下内壁。
5. **洗涤**：用蒸馏水洗涤烧杯内壁和玻璃棒 2~3 次，洗涤液全部转入容量瓶，确保溶质全部转移。
6. **定容**：向容量瓶中加水至液面离刻度线 1~2 cm 时，改用胶头滴管逐滴加水，视线与凹液面最低处相切，滴至刻度线。
7. **摇匀**：盖紧瓶塞，反复上下颠倒摇匀。
8. **装瓶贴签**：将配好的溶液转移至试剂瓶，贴上标签。

**误差分析核心**：所有操作引起的溶质物质的量 n 或溶液体积 V 的偏差，最终体现在 c = n / V 中。凡使 n 偏小或 V 偏大的操作，导致 c 偏低；反之则 c 偏高。

**典型高考设问**
- **实验选择/排序题**：判断给定操作步骤的正确顺序或找出错误操作用（如未冷却即转移、定容俯视等）。
- **误差分析选择题或填空题**：给出具体不规范操作（如砝码生锈、量浓盐酸时仰视量筒、未洗涤烧杯等），判断所配溶液浓度如何变化。
- **综合实验题**：结合物质分离、滴定等，考查容量瓶的检漏、选择合适规格的容量瓶（如 100 mL、250 mL）以及结合化学方程式计算所需溶质的质量。

**示例情境**
配制 250 mL 0.1 mol·L⁻¹ 的 Na₂CO₃ 溶液：计算需称取 Na₂CO₃·10H₂O 的质量为 m = 0.1 mol·L⁻¹ × 0.250 L × 286 g·mol⁻¹ = 7.15 g（托盘天平称 7.2 g）。若称量时砝码与物品放反且使用游码，实际称得质量偏低，导致所配浓度偏低。

## ⚠️ 高频易错点

- **容量瓶的使用细节**：容量瓶不能用待配溶液润洗，但用蒸馏水洗净后无需干燥；必须检漏；只能配制体积与容量瓶容积一致的溶液（不能配制任意体积）。
- **定容时的视线**：仰视使液面超过刻度线，V 偏大，c 偏低；俯视使液面低于刻度线，V 偏小，c 偏高。
- **溶质损失**：未洗涤烧杯和玻璃棒（n 偏小，c 偏低）；转移时有溶液溅出（n 偏小，c 偏低）；溶解时搅拌造成液体飞溅亦属于溶质损失。
- **温度与热效应**：溶解或稀释过程中有明显热效应（如 NaOH 溶解、浓 H₂SO₄ 稀释）时，必须冷却至室温后再转移和定容，否则冷却后体积收缩导致 V 偏小（c 偏高），或容量瓶因热胀变形引入系统误差。
