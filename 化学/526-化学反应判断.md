# 化学反应判断

> **一句话**：基于化学键断裂与生成、元素价态变化或能量转换，对物质变化过程是否属于化学反应及所属反应类型进行判定的逻辑方法。
> **English**: The logical method for determining whether a process constitutes a chemical reaction and its specific type, based on bond breaking/formation, oxidation state changes, or energy conversion.

## 🎓 考点精讲

**定义/原理**：化学反应判断的核心依据是新物质的生成，即旧化学键的断裂与新化学键的形成，常伴随能量变化（吸/放热）、颜色改变、气体逸出、沉淀生成等现象。高考主要围绕两大维度考查：
1. **化学变化 vs 物理变化**：是否有新化学键生成。如石油分馏（物理变化，利用沸点差异）与煤的干馏（化学变化，热分解生成焦炭、煤焦油等）。
2. **四种基本反应类型及氧化还原反应的归属**：化合反应（`A + B → AB`）、分解反应（`AB → A + B`）、置换反应（`A + BC → AC + B`，一定属于氧化还原反应）、复分解反应（`AB + CD → AD + CB`，条件为生成沉淀/气体/弱电解质，一定不属于氧化还原反应）。

离子反应是否发生的判断依据是离子浓度的减少，实质是生成难溶物（如 `Ba²⁺ + SO₄²⁻ = BaSO₄↓`）、弱电解质（如 `H⁺ + CH₃COO⁻ = CH₃COOH`）或挥发性物质（如 `2H⁺ + CO₃²⁻ = H₂O + CO₂↑`）。

**关键方法**：
1. **价态分析法**：标出各元素化合价，判断电子转移。凡有单质参与或生成的化合/分解反应，以及所有置换反应，一定是氧化还原反应。
2. **强制弱规律**：用于判断复分解反应或氧化还原反应的方向性。例如，酸性较强的 `HCl` 可与 `CH₃COONa` 反应制得弱酸 `CH₃COOH`。
3. **有条件的不反应陷阱**：注意常见惰性金属（Au、Pt）不与普通酸碱反应；浓硫酸/硝酸与金属反应不生成氢气；氧化物与水的反应需可溶或特定条件（如 `SiO₂` 不与水反应生成 `H₂SiO₃`）。

**典型应用**：
高考常以化学与社会生活（如“古法酿酒”、“蜡炬成灰”涉及化学变化；“铁杵磨针”、“滴水成冰”属于物理变化）、微观粒子示意图判断、离子大量共存及陌生方程式正误判断等题型出现。例如，用单线桥/双线桥分析反应 `2Na₂O₂ + 2H₂O = 4NaOH + O₂↑` 中的电子转移与氧化还原关系，便是经典的综合运用设问。

## ⚠️ 高频易错点

- **混淆现象与本质**：误将发光（如电灯发光）、放热（如浓硫酸稀释放热）、颜色变化（如活性炭吸附色素）、产生气体（如打开汽水瓶）等物理变化判为化学变化。
- **忽视“有条件才反应”**：错误认为“盐+酸”必定反应，如 `NaCl(固) + H₂SO₄(浓)` 强热可制 `HCl` 是利用高沸点酸制低沸点酸，但 `NaCl` 溶液与稀硫酸在常温下不满足离子反应发生条件（无沉淀/气体/弱电解质生成）。
- **金属活动性顺序带入非水环境**：金属与盐溶液发生置换反应须遵循“前金换后金”（不含K、Ca、Na），但在熔融态或高温固体置换（如 `Na + KCl = NaCl + K↑`）中，钾的沸点低于钠从而以气体逸出使平衡右移，不能机械套用溶液中的金属活动性顺序。
- **氧化还原反应类型误判**：误认为有氧元素参与的反应必为氧化还原反应（如 `CaO + H₂O = Ca(OH)₂` 无价态变化，属于非氧化还原反应）；误认为某些特殊氧化还原反应（如歧化反应 `Cl₂ + 2NaOH = NaCl + NaClO + H₂O`）不是氧化还原反应。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |

| 化学反应 | Chemical Reaction | 生成新物质、涉及化学键断裂与形成的过程 |
