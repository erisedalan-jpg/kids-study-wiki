# 氨气性质

> **一句话**：氨气（NH₃）是一种三角锥形极性分子，具有还原性、易液化、极易溶于水且水溶液呈弱碱性的特征。
> **English**: Ammonia is a trigonal pyramidal polar molecule that acts as a reducing agent, liquefies easily, is highly soluble in water, and forms weakly basic aqueous solutions.

## 🎓 考点精讲

**定义/原理**
氨气是氮的最基本氢化物，分子构型为三角锥形，氮原子采取不等性 $sp^3$ 杂化，有一对孤电子对，因此是良好的 Lewis 碱。核心性质围绕以下反应展开：

1. **与水反应**（显碱性的本质）：
   $$NH_3 + H_2O \rightleftharpoons NH_3\cdot H_2O \rightleftharpoons NH_4^+ + OH^-$$
   该反应可逆，$NH_3\cdot H_2O$ 为弱碱，能使湿润的红色石蕊试纸变蓝（核心检验方法）。

2. **与酸反应**（生成铵盐）：
   $$NH_3 + HCl = NH_4Cl$$
   浓氨水与浓盐酸相遇会产生白烟，用于检验 $NH_3$ 或 $HCl$ 气体。与硫酸反应因酸用量不同可生成正盐或酸式盐：
   $$2NH_3 + H_2SO_4 = (NH_4)_2SO_4$$
   $$NH_3 + H_2SO_4(\text{过量}) = NH_4HSO_4$$

3. **还原性**（催化氧化，工业制硝酸的基础）：
   $$4NH_3 + 5O_2 \xrightarrow[\Delta]{Pt} 4NO + 6H_2O$$
   氨气还能在纯氧中燃烧：
   $$4NH_3 + 3O_2(\text{纯氧}) \xrightarrow{\text{点燃}} 2N_2 + 6H_2O$$
   氨气也可还原 $CuO$ 等金属氧化物。

4. **配合性**：能与 $Ag^+$、$Cu^{2+}$ 等形成配合物，如银氨溶液配制时先生成沉淀后溶解：
   $$Ag^+ + NH_3\cdot H_2O = AgOH\downarrow + NH_4^+$$
   $$AgOH + 2NH_3\cdot H_2O = [Ag(NH_3)_2]^+ + OH^- + 2H_2O$$

**关键方法**
解决含氨气流程问题的核心在于：
- **先辨主次**：看到物质转化图时，先确定 $N$ 元素化合价变化，判断是发生氧化还原（如催化氧化，价态升）还是非氧化还原（如与酸反应，价态不变）。
- **再析环境**：水溶液中关注 $NH_3\cdot H_2O$ 的电离平衡和配位反应；干燥气体时关注其碱性，**不可用浓硫酸或无水氯化钙**（会形成 $CaCl_2\cdot 8NH_3$），通常选用碱石灰。

**典型应用**
高考中常以工业合成氨或硝酸制备流程为载体，考查氨气的转化。典型设问如：
1. 从含 $NH_4^+$、$Al^{3+}$、$Fe^{3+}$ 等离子的废液中回收金属，先加碱加热除去 $NH_4^+$（$NH_4^+ + OH^- \xrightarrow{\Delta} NH_3\uparrow + H_2O$）。
2. 设计实验检验固体中含有 $NH_4^+$：取样，加浓碱加热，用湿润的红色石蕊试纸检验逸出的气体，若变蓝则含 $NH_4^+$。

## ⚠️ 高频易错点

- **干燥剂选错**：极易认为氨气是碱性气体、浓硫酸是酸性干燥剂所以冲突（正确），但常忽略 **无水氯化钙也与氨气反应**（生成配合物），不能用其干燥。
- **酸碱反应产物不清**：氨气与多元酸（如硫酸、磷酸）反应时，酸过量生成酸式盐，氨气过量生成正盐，产物取决于两者比例，不可机械记成“只生成正盐”。
- **喷泉实验原理混淆**：误认为喷泉只与溶解度有关。氨气喷泉实验成功的核心是气体在挤入液体中的溶剂里 **迅速溶解，造成足够大的内外压差**，即不仅要求溶解度大，溶解速率也必须快。
- **铵盐分解理解片面**：并非所有铵盐受热分解都产生氨气，如 $NH_4NO_3$ 热分解产物复杂，$NH_4NO_2$ 分解产生 $N_2$，不能用“铵盐加热必出氨”来概括。
