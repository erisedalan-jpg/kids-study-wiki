# 离子共存

> **一句话**：离子共存是指多种离子在同一溶液中能够大量存在，彼此间不发生复分解反应（生成沉淀、气体、弱电解质）、氧化还原反应或络合反应的条件判断。
> **English**: Ion coexistence refers to the condition under which multiple ions can exist in large amounts in the same solution without undergoing precipitation, gas evolution, weak electrolyte formation, redox reactions, or complexation.

## 🎓 考点精讲

### 定义与核心原理
离子能否大量共存，取决于离子间是否发生**离子反应**。若离子间满足下列任一条件，则不能大量共存：

**1. 生成难溶物或微溶物**  
如 $Ba^{2+}$ 与 $SO_4^{2-}$、$Ag^+$ 与 $Cl^-$、$Ca^{2+}$ 与 $CO_3^{2-}$ 等。

**2. 生成挥发性气体**  
如 $H^+$ 与 $CO_3^{2-}$（$CO_3^{2-} + 2H^+ = H_2O + CO_2 \uparrow$）、$NH_4^+$ 与 $OH^-$（加热时$NH_3$逸出）等。

**3. 生成弱电解质**  
如 $H^+$ 与 $OH^-$ 生成 $H_2O$、$H^+$ 与 $CH_3COO^-$ 生成乙酸、$H^+$ 与 $F^-$ 生成 $HF$ 等。

**4. 发生氧化还原反应**  
具强氧化性的离子（如 $Fe^{3+}$、$MnO_4^-$、$Cr_2O_7^{2-}$、$ClO^-$ 等）与具强还原性的离子（如 $I^-$、$S^{2-}$、$SO_3^{2-}$、$Fe^{2+}$ 等）不能共存。特别注意：$NO_3^-$ 在中性或碱性条件下氧化性极弱，但在 **酸性**（$H^+$）介质中会表现出强氧化性，与 $Fe^{2+}$、$I^-$ 等还原性离子不能共存。

**5. 发生络合反应**  
如 $Fe^{3+}$ 与 $SCN^-$ 生成血红色 $[Fe(SCN)]^{2+}$，$Ag^+$ 与 $NH_3 \cdot H_2O$ 先生成沉淀后溶解，均不得大量共存。

**6. 发生强烈双水解**  
如 $Al^{3+}$ 与 $HCO_3^-、CO_3^{2-}、S^{2-}、AlO_2^-$ 等，$Fe^{3+}$ 与 $CO_3^{2-}、HCO_3^-$ 等在溶液中相遇时彻底水解生成沉淀和气体，不能共存。

### 解题关键三步法
1. **看颜色**：题干要求“无色溶液”时，排除有色离子（$Cu^{2+}$ 蓝色、$Fe^{3+}$ 棕黄、$Fe^{2+}$ 浅绿、$MnO_4^-$ 紫红等）。
2. **看酸碱环境**：明确溶液的酸碱性（如 pH=1、使石蕊变红等为酸性；pH=13、使酚酞变红为碱性；与 Al 反应放出 $H_2$ 可能为强酸或强碱），先排除与 $H^+$ 或 $OH^-$ 反应的离子。
3. **看反应**：在剩余离子中，利用上述六类反应逐对排查，尤其注意 **隐含氧化性**（如酸性 $NO_3^-$）、**双水解** 和 **络合** 等易漏点。

### 高考典型设问
-   **选择判断式**：“在给定条件的溶液中，下列各组离子一定/可能大量共存的是？”
-   **陷阱条件**：① 由水电离出的 $c(H^+)=1 \times 10^{-13} mol/L$（溶液可能为强酸或强碱）；② 能与 Al 反应放出 $H_2$（溶液可能为非氧化性强酸或强碱）；③ $\frac{c(H^+)}{c(OH^-)}=10^{12}$ 间接给出酸性条件；④ 加入某物质能反应，问原溶液中离子能否共存。

## ⚠️ 高频易错点

-   **忽视酸性条件下的 $NO_3^-$ 氧化性**：在“酸性溶液”或“pH=1”的给定环境中，若选项同时含有 $NO_3^-$ 与 $Fe^{2+}$、$I^-$、$S^{2-}$ 等，会发生氧化还原反应，不能共存。学生常仅将其视作普通酸根而误判。
-   **混淆“一定”与“可能”共存**：问“一定能大量共存”时，需满足题给所有可能情境；与 Al 放 $H_2$ 的溶液可能酸可能碱，若某组离子在酸和碱中均不反应，才选“一定”，否则仅算“可能”。
-   **漏判双水解反应**：$Al^{3+}$ 与 $AlO_2^-$ 在水中直接反应生成 $Al(OH)_3$ 沉淀（$Al^{3+} + 3AlO_2^- + 6H_2O = 4Al(OH)_3 \downarrow$），学生易忽略此类彻底双水解。
-   **忽略络合反应**：常见 $Fe^{3+}$ 遇 $SCN^-$ 不能大量共存，易在多个离子组合中疏忽。
