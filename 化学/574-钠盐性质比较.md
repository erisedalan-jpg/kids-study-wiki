# 钠盐性质比较

> **一句话**：比较常见钠盐（主要是 Na₂CO₃ 与 NaHCO₃）在溶解度、碱性、热稳定性、与酸反应等方面的差异，用于鉴别、除杂与定量分析。
> **English**: Comparison of properties of sodium salts, mainly focusing on differences between Na₂CO₃ and NaHCO₃ in solubility, basicity, thermal stability, and reaction with acids.

## 🎓 考点精讲

**定义与原理**  
本考点以 Na₂CO₃（碳酸钠）和 NaHCO₃（碳酸氢钠）的性质差异为核心，兼顾 Na₂SO₄、Na₂SO₃ 等其他钠盐。关键差异源于阴离子的结构（CO₃²⁻ 与 HCO₃⁻）及其在溶液中的水解、热分解行为。  
主要方程式：  
- 热稳定性：2NaHCO₃ △→ Na₂CO₃ + H₂O + CO₂↑，Na₂CO₃ 受热不分解。  
- 与盐酸反应：NaHCO₃ + HCl = NaCl + H₂O + CO₂↑（一步放出气体）；Na₂CO₃ + HCl = NaCl + NaHCO₃（先无气体），NaHCO₃ + HCl → NaCl + H₂O + CO₂↑（后有气体），因此等浓度时 NaHCO₃ 产生气体更快。  
- 与 CaCl₂/BaCl₂ 溶液：Na₂CO₃ + CaCl₂ = CaCO₃↓ + 2NaCl，NaHCO₃ 与 CaCl₂ 不反应（因 Ca(HCO₃)₂ 可溶）。  

**关键方法**  
- 鉴别：固体用加热法（产生使澄清石灰水变浑浊的气体者为 NaHCO₃）；溶液可滴加 BaCl₂ 或 CaCl₂ （有沉淀者为 Na₂CO₃），或用 pH 计（同浓度时 Na₂CO₃ 碱性更强）。  
- 除杂：除去 Na₂CO₃ 中 NaHCO₃ 用加热；除去 NaHCO₃ 中 Na₂CO₃ 可通入过量 CO₂（Na₂CO₃ + CO₂ + H₂O = 2NaHCO₃）。  
- 图像分析：向 NaOH 与 Na₂CO₃ 混合液中滴加盐酸，曲线分阶段：OH⁻ 中和 → CO₃²⁻→HCO₃⁻ → HCO₃⁻+H⁺→CO₂↑。  

**典型应用**  
高考常以实验流程题、物质鉴别题、图像计算题出现。例如：用两种不同顺序向等量 Na₂CO₃ 与 NaHCO₃ 中滴加盐酸，比较 CO₂ 体积与消耗 HCl 量；或结合侯氏制碱法考察 Na₂CO₃ 与 NaHCO₃ 的转化。

## ⚠️ 高频易错点

- 误以为 Na₂CO₃ 受热也分解（实际很稳定，只在高温下才微弱分解为 Na₂O 和 CO₂，但中学阶段认为不分解）。  
- 混淆 Na₂CO₃ 与 NaHCO₃ 与盐酸反应的计量关系：等质量时 NaHCO₃ 产生的 CO₂ 多；等物质的量时产生的 CO₂ 相等，但 NaHCO₃ 耗酸更少。  
- 忽略滴加顺序的影响：向 Na₂CO₃ 溶液中逐滴加盐酸，开始无气泡，不能据此认为二者不反应；反过来向盐酸中滴加 Na₂CO₃ 溶液则立即产生气泡。  
- NaHCO₃ 水溶液与 CaCl₂ 混合无明显现象，错误认为能生成 CaCO₃ 沉淀，实际上 Ca(HCO₃)₂ 可溶于水。
