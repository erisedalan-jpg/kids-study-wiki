# 二氧化碳与盐反应

> **一句话**：CO₂ 作为一种酸性氧化物，溶于水形成碳酸后可与某些盐溶液发生复分解反应，遵循“强酸制弱酸”原理，常见产物为碳酸盐、酸式盐或更难溶的弱酸盐。
> **English**: Reactions between carbon dioxide and salts, in which CO₂ acts as an acidic oxide to form carbonic acid, then undergoes double decomposition with certain salts, following the principle of stronger acid displacing weaker acid.

## 🎓 考点精讲

**定义/原理**  
CO₂ 本身是非金属氧化物，与水结合生成碳酸（H₂CO₃）。当通入某些盐溶液时，碳酸与盐中的阴离子对应的弱酸发生“强酸制弱酸”的复分解反应，生成新的弱酸和碳酸盐（或酸式盐）。核心比较对象是 H₂CO₃ 的电离平衡常数，及其与 HClO、H₂SiO₃、Al(OH)₃、苯酚等弱酸性的强弱。  

常见反应通式可归纳为：  
- 生成正盐：CO₂ + H₂O + Na₂SiO₃ → Na₂CO₃ + H₂SiO₃↓  
- 生成酸式盐：CO₂（过量）+ H₂O + NaAlO₂ → Al(OH)₃↓ + NaHCO₃  
- 生成溶解度较小的酸式盐：CO₂ + H₂O + Na₂CO₃（饱和）→ 2NaHCO₃↓  

**关键方法**  
解答本考点问题，需三步走：  
1. **判酸性**：比较 H₂CO₃ 与目标盐阴离子对应酸的强弱。若盐对应的酸比碳酸强（如 CaCl₂ 对应 HCl），则反应不能发生。  
2. **定产物**：依据通入 CO₂ 的量（少量/过量）及酸式盐溶解度、酸式根离子的存在稳定性，判断产物是正盐还是酸式盐。原则是：CO₂ 少量通常生成正盐；过量生成酸式盐；但当产物酸式盐溶解度极低时，即使少量也可析出（如 NaHCO₃ 从饱和 Na₂CO₃ 溶液中析出）。  
3. **写方程**：注意离子方程式中，H₂CO₃ 写成 CO₂ 和 H₂O，不可拆；弱酸、难溶物保留化学式。  

**典型应用**  
高考常以离子方程式正误判断、物质转化实验分析等形式出现。例如：  
- 判断“CO₂ 通入 CaCl₂ 溶液是否产生 CaCO₃ 沉淀”——不反应，因 HCl 酸性强于 H₂CO₃，无法发生。  
- “向 NaAlO₂ 溶液中通入过量 CO₂”的离子方程式应为：AlO₂⁻ + CO₂ + 2H₂O → Al(OH)₃↓ + HCO₃⁻，易错写成生成 CO₃²⁻。  
- “将 CO₂ 通入饱和 Na₂CO₃ 溶液”，现象为析出白色晶体（NaHCO₃），因为 NaHCO₃ 溶解度小于 Na₂CO₃ 且反应消耗水。  

## ⚠️ 高频易错点

- **误判“强酸制弱酸”边界**：学生常认为 CO₂ 通入 CaCl₂ 或 BaCl₂ 溶液可生成碳酸盐沉淀，忽略 Cl⁻ 对应的 HCl 酸性强于 H₂CO₃，该反应无法发生。  
- **过量与少量的产物混淆**：NaAlO₂ 溶液通 CO₂，少量生成 CO₃²⁻，过量生成 HCO₃⁻；而苯酚钠溶液通 CO₂，无论少量还是过量，都只生成 HCO₃⁻（因为苯酚酸性弱于碳酸、但强于 HCO₃⁻）。  
- **离子方程式书写遗漏水或错拆**：H₂CO₃ 容易拆成 CO₂ 和 H₂O 的混合形式，但在复分解中常直接写成 CO₂ + H₂O；忘记水是反应物，导致方程式不配平。  
- **不理解 NaHCO₃ 析出的溶解度原理**：饱和 Na₂CO₃ 溶液中通入 CO₂，析出 NaHCO₃ 晶体并非只由反应发生产生，还因为 NaHCO₃ 溶解度远小于 Na₂CO₃，且反应消耗溶剂水。
