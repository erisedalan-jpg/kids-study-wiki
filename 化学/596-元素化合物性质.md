# 元素化合物性质

> **一句话**：基于“结构决定性质，性质反映结构”的核心思想，系统研究典型元素及其常见化合物的物理性质、化学性质、制备与用途的知识模块。
> **English**: A knowledge module that systematically studies the physical properties, chemical properties, preparation, and uses of typical elements and their common compounds, based on the core concept that "structure determines properties".

## 🎓 考点精讲

**定义/原理**：
该考点要求从**原子结构、分子结构、晶体类型**等微观视角出发，理解和预测单质及其化合物的宏观性质。其核心逻辑链为：`结构（原子半径、价电子、化学键） → 性质（酸碱性、氧化性/还原性、热稳定性、溶解性） → 用途与制备`。性质的核心是**化学反应**，高中阶段要求掌握以下几类典型反应方程式：
- **酸性氧化物**与碱反应：`SO₂ + 2NaOH = Na₂SO₃ + H₂O`（少量SO₂）
- **两性化合物**与强酸、强碱反应：`Al(OH)₃ + 3H⁺ = Al³⁺ + 3H₂O`；`Al(OH)₃ + OH⁻ = AlO₂⁻ + 2H₂O`（或写作 `Al(OH)₃ + OH⁻ = [Al(OH)₄]⁻`）（需核对版本）
- **特征变价金属**的氧化还原反应：`2Fe³⁺ + Cu = 2Fe²⁺ + Cu²⁺`（用于检验或电路板刻蚀）

**关键方法**：
1. **“单质→氧化物→酸碱盐”转化主线法**：将元素化合价作为纵轴，物质类别作为横轴，构建价类二维图。通过标注各节点物质的化学式及相互转化方程式，形成知识网络。例如，氮元素从 `NH₃`（-3价）至 `HNO₃`（+5价）的连续氧化过程。
2. **类比与递变规律法**：利用元素周期表进行横向（同周期）和纵向（同主族）比较，判断物质性质的相似性与递变性。
3. **特性突破法**：精准识记某些物质的唯一特性（如 `Al(OH)₃` 是唯一不溶于弱碱和弱酸的氢氧化物；`SiO₂` 是唯一与氢氟酸反应的酸性氧化物等），用以应对物质的推断与检验。

**典型应用**：
高考中本部分常以两种题型呈现：
1. **物质转化推断题**：给出若干物质的颜色、状态、特征反应（如焰色反应黄色为Na元素，紫色为K元素；白色沉淀迅速变灰绿最终变红褐色涉及 `Fe²⁺` 到 `Fe(OH)₃` 的转化），要求推断未知物质并书写相关反应方程式。
2. **工业流程题**：以真实化工生产（如侯氏制碱法、海水提溴）为背景，考查特定条件下元素化合物之间的转化与除杂，涉及反应条件的控制（如pH调节沉淀特定金属离子 `Fe³⁺` 完全沉淀时 `pH≥3.2`，`Al³⁺` 完全沉淀时 `pH≥5.2` 等（待家长核对）。

## ⚠️ 高频易错点

- **反应产物的价态与条件混淆**：铁在常温下遇浓硫酸/浓硝酸**钝化**（生成致密氧化膜，视为化学变化但未持续溶解），而在加热时剧烈反应且 `Fe` 被氧化为 `Fe³⁺`，而非 `Fe²⁺`。变价金属（如Fe）与弱氧化剂（如S、I₂）反应通常生成低价态产物（如 `FeS`、`FeI₂`）。
- **两性概念的边界模糊**：`Al₂O₃` 和 `Al(OH)₃` 均属于两性化合物，既能溶于强酸也能溶于**强碱**。学生常误认为它们能溶于任何碱，实际上它们不溶于弱碱（如 `NH₃·H₂O`）。`Al` 单质虽能溶于强酸和强碱，但其本身属于活泼金属，不是两性单质。
- **二氧化硫漂白性的机理混淆**：`SO₂` 的漂白是与有机色素**化合**生成不稳定的无色化合物（随时间推移可恢复颜色），这完全不同于 `Na₂O₂`、`HClO` 的强氧化性漂白（不可逆），也不同于活性炭的物理吸附漂白。
- **“氧化物对应水化物”的酸碱性判定**：不可机械记忆。如常温下 `SiO₂` 不溶于水，其对应水化物为 `H₂SiO₃`，是通过可溶性硅酸盐与酸反应制得。
