# 化学用语

> **一句话**：化学用语是国际通用的，用符号、图式、字母和数字精确表示物质的组成、结构及变化规律的化学语言工具。
> **English**: A chemical language tool that uses internationally recognized symbols, diagrams, letters, and numbers to precisely represent the composition, structure, and transformation of matter.

## 🎓 考点精讲

**定义/原理**
化学用语是化学学科特有的符号系统，主要包括四大核心板块：
1. **元素符号与原子（离子）结构示意**：如 $\ce{Cl}$、$\ce{Na+}$、原子结构示意图（核电荷数+电子层排布）。
2. **物质组成与结构**：化学式（如 $\ce{H2O}$）、电子式（如 $\ce{H:\ddot{O}:H}$）、结构式（如 $\ce{H-O-H}$）。
3. **物质变化**：化学方程式（遵循质量守恒定律，如 $\ce{2H2 + O2 ->[\text{点燃}] 2H2O}$）、离子方程式（以实际参加反应的离子符号表示，如 $\ce{H+ + OH- = H2O}$）、热化学方程式（须标注物质聚集状态与 $\Delta H$，如 $\ce{H2(g) + 1/2 O2(g) = H2O(l)}$ $\Delta H = -285.8\,\text{kJ}\cdot\text{mol}^{-1}$）、电极反应式。
4. **电子转移**：单线桥/双线桥法标明氧化还原反应中的电子得失。

**关键方法**
解答化学用语类考题的标准步骤：
1. **审清“四式”要求**：明确要求书写的是分子式/化学式、电子式、结构式还是结构简式。混淆必失分。
2. **遵循“守恒”三重奏**：书写方程式必须先后通过原子守恒（质量）、电荷守恒（离子反应）和得失电子守恒（氧化还原）的检验。
3. **电子式书写重“八隅体”**：准确判断原子最外层电子是否满足稳定结构，离子化合物和共价化合物的区别在“括号与电荷标注”。

**典型应用**
高考中常以“正误判断”选择题或“规范书写”填空题形式考查。例如：判断 $\ce{CO2}$ 的电子式是否错误写成 $\ce{:\ddot{O}:C:\ddot{O}:}$；书写酸性高锰酸钾与草酸反应的离子方程式时，漏写 $\ce{H+}$ 或未配平电荷；或给出合成路线，让考生推导并书写特定有机物的结构简式。

## ⚠️ 高频易错点

- **混淆电子式书写规则**：共价化合物（如 $\ce{HCl}$）不标电荷不用括号，离子化合物（如 $\ce{NaCl}$）阴离子必须加括号标电荷，且相同离子不能合并。
- **离子方程式的“拆”与“不拆”**：将难溶物（如 $\ce{BaSO4}$）、弱电解质（如 $\ce{CH3COOH, H2O}$）、气体单质及氧化物（如 $\ce{Fe2O3}$）错误拆开写成离子符号。
- **忽略有机物结构简式的官能团重构**：书写时习惯性漏掉双键/三键，或误将硝基（$\ce{-NO2}$）写成 $\ce{-O2N}$（当连接方向错误时）。
- **电极反应式未考虑介质环境**：在酸性或熔融盐环境下，生成物中错误出现 $\ce{OH-}$ 或 $\ce{O^{2-}}$ 且不进行配平转化。
