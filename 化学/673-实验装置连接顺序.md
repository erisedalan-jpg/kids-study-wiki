# 实验装置连接顺序

> **一句话**：实验装置连接顺序指在气体制备与性质验证实验中，按“发生→除杂→干燥→主体反应（收集/性质实验）→尾气处理”的逻辑对洗气瓶、干燥管、反应管、收集装置等玻璃仪器进行科学组装的规则。
> **English**: The rules for scientifically assembling glass instruments like wash bottles, drying tubes, reaction tubes, and collection devices in gas preparation and property verification experiments, following the logic of “generation → purification → drying → main reaction (collection/property test) → tail gas treatment”.

## 🎓 考点精讲

**定义/原理**：化学实验中涉及气体反应时，装置连接的核心原则是**先净化后干燥，先干燥后主体反应，最后进行尾气处理**。气体发生装置产生的气体常含有酸雾、水蒸气或其他杂质气体，这些杂质会干扰后续反应或腐蚀仪器，必须按顺序去除。干燥操作必须置于除杂之后、主体反应之前，否则已干燥的气体在经过水溶液除杂时又会混入水蒸气。装置连接的另一原理是**接口匹配**：玻璃管口对橡胶管，玻璃接口对橡胶塞，洗气瓶遵循“长进短出”，干燥管遵循“大口进小口出”，储气/集气瓶排水法遵循“短进长出”。

**关键方法**：解决装置连接顺序问题的标准步骤是：①明确实验目的与涉及的气体；②罗列需去除的杂质种类（酸性气体、水蒸气、多余反应物等）；③选择合适试剂与装置（如饱和 NaHCO₃ 溶液除 CO₂ 中的 HCl，浓硫酸干燥 Cl₂，碱石灰吸收尾气 CO₂ 或 SO₂）；④按“发生 → 除杂 → 干燥 → 主体 → 尾气”排列。判定接口方向时，凡通过液体试剂的装置一律“长管进、短管出”以保证气体与试剂充分接触；干燥管则必须“大口进、小口出”以防止固体干燥剂堵塞气流或冲散棉花团。

**典型应用**：高考中该考点常以实验题装置排序或装置选择形式出现。例如：“用 MnO₂ 与浓盐酸制取纯净、干燥的 Cl₂，并验证其与 Fe 的反应。”装置连接顺序为：发生装置（圆底烧瓶+分液漏斗，加热）→ 饱和食盐水除 HCl → 浓硫酸干燥 → 硬质玻璃管中铁粉加热反应 → NaOH 溶液吸收尾气。另一典型情境是“验证 SO₂ 的漂白性、酸性和还原性”：SO₂ 发生装置 → 品红溶液（验证漂白性）→ 酸性 KMnO₄ 溶液（验证还原性，除 SO₂ 防止干扰）→ 澄清石灰水（验证酸性氧化物性质）→ 尾气 NaOH 吸收。考查设问包括排出正确接口顺序、判断某装置是否多余、分析倒置连接的错误后果等。

## ⚠️ 高频易错点

- **除杂与干燥顺序颠倒**：误将干燥装置（如浓硫酸洗气瓶）放在除杂装置（如饱和食盐水）之前，导致干燥后的气体重新混入水蒸气，后续无水反应（如 Fe 与 Cl₂ 反应）因水蒸气存在而失败。
- **洗气瓶通入方向错误**：所有洗气瓶中气体必须“长管进、短管出”，若接反对形成液封或气体无法通过引发爆炸风险；排水集气时则恰好相反，应“短管进、长管排出水”，此处极易混淆。
- **忽视尾气处理或选错吸收剂**：制取 CO 或进行含 SO₂、Cl₂、NOₓ 实验时遗漏尾气处理装置，或用水吸收 Cl₂（Cl₂ 在水中溶解度小，应改用 NaOH 溶液）、用饱和石灰水吸收大量 CO（不反应），均为高频失分点。
- **干燥剂与气体性质不匹配**：用碱石灰干燥 CO₂ 或 SO₂、用浓硫酸干燥 H₂S 或 NH₃（浓硫酸与 H₂S 发生氧化还原反应，与 NH₃ 发生酸碱反应），误认为所有干燥剂可通用。
