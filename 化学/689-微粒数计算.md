# 微粒数计算

> **一句话**：微粒数计算是基于物质的量(n)将宏观可测量量（质量、气体体积、溶液浓度）转化为微观粒子数目（分子、原子、离子、电子、质子、中子等）的化学计量方法，核心桥梁公式为 $N = n \cdot N_A$。
> **English**: Particle number calculation is a stoichiometric method in chemistry where the amount of substance (n), derived from measurable macroscopic quantities (mass, gas volume, solution concentration), is used to determine the number of microscopic entities (molecules, atoms, ions, electrons, etc.) via the core bridge formula $N = n \cdot N_A$.

## 🎓 考点精讲

**定义/原理**：
微粒数（$N$）的计算建立在“物质的量（$n$）”这一核心物理量的基础上。其核心关系式为：

$$N = n \times N_A$$

其中，$N_A$ 为阿伏加德罗常数（约 $6.02 \times 10^{23} \text{ mol}^{-1}$）。高考中，关键在于根据题目条件准确求出相应微粒的物质的量（$n$），其求算路径通常为：

$$n = \frac{m}{M} = \frac{V}{V_m} = c \times V(\text{aq})$$

- 对于**物质**整体：质量 $m$ 除以摩尔质量 $M$。
- 对于**气体**：在标准状况下（STP，$0^\circ\text{C}$，$101\text{ kPa}$），体积 $V$ 除以气体摩尔体积 $V_m$（$22.4 \text{ L}\cdot\text{mol}^{-1}$）。
- 对于**溶液中的溶质**：物质的量浓度 $c$ 乘以溶液体积 $V$。

在求得 $n$ 后，必须根据物质的**化学式组成**或**反应计量关系**，将其乘以一个关联因子（如下标数、化学键数、电子转移数等）才能得到目标微粒的数目。

**关键方法**：
1. **抓核心桥梁**：一切计算先求 $n$，养成“见量先化摩”的思维。
2. **审题破陷阱**：精确识别题目所求微粒的类型。原子、离子、电子、质子、中子与分子的数量关系遵循 $N(\text{目标微粒}) = N(\text{物质分子}) \times \text{下标/计量数}$。
3. **避坑验公式**：列式后用 $N_A = \frac{N}{n}$ 的变体反推公母，检验是否出现分数粒子（如半个水分子）等计算异常。

**典型应用**：
1. **$N_A$ 正误判断（微粒数正算）**：设问“$18\text{ g } \ce{D2O}$ 含有的质子数为 $10N_A$”（错误，$M(\ce{D2O}) = 20 \text{ g}\cdot\text{mol}^{-1}$，且重水中每个分子含 $10$ 个质子，$18\text{ g}$ 为 $0.9\text{ mol}$，应含 $9N_A$ 个质子）。
2. **混合物中的原子数计算**：如“$46\text{ g } \ce{NO2}$ 与 $\ce{N2O4}$ 的混合气体中所含原子总数”，可利用二者最简式均为 $\ce{NO2}$，直接计算 $46\text{ g}$ 的 $\ce{NO2}$ 原子构块中所含原子为 $3 \times \frac{46}{46}N_A = 3N_A$。
3. **氧化还原中的电子转移数**：“$1\text{ mol } \ce{Fe}$ 与足量稀 $\ce{HNO3}$ 反应转移电子数”，铁被氧化为 $\ce{Fe^{3+}}$，转移电子为 $3N_A$。

## ⚠️ 高频易错点

- **忽略气体摩尔体积适用条件**：看到“气体体积”直接除以 $22.4\text{ L}\cdot\text{mol}^{-1}$，但题目条件并非标准状况（STP），或物质在STP下并非气态（如水、$\ce{SO3}$、$\ce{HF}$、$\ce{CCl4}$、碳数大于$4$的烃等）。
- **混淆“物质”与“微粒”的量**：直接求得 $n(\text{溶质})$ 就代入 $N_A$，忽略了溶液中溶剂 $\ce{H2O}$ 也含大量的 $\ce{H}$、$\ce{O}$ 原子，或忽略弱电解质的部分电离与盐类水解导致的离子数目变化。
- **同位素与核素组成差异**：计算中子数、质子数时，未注意普通氢（$\ce{^1H}$）与重氢（$\ce{^2H}$或$\ce{D}$）、普通氧（$\ce{^16O}$）与重氧（$\ce{^18O}$）等核素在质量数上的区别，导致中子数计算失真。
- **可逆反应与水解陷阱**：将可逆反应（如 $\ce{N2 + 3H2 <=> 2NH3}$、$\ce{2SO2 + O2 <=> 2SO3}$）当作不可逆完全反应计算分子总数；或忽略盐溶液中弱酸根/弱碱阳离子的水解对离子数目的影响，直接按化学式下标记数。
