# 无机物的推断

> **一句话**：无机物的推断是基于元素化合物特征性质（颜色、状态、特征反应、特征转化关系）和实验现象，通过逻辑推理确定未知无机物组成的化学分析方法。
> **English**: The inference of inorganic substances is a chemical analytical method that identifies unknown inorganic compounds through logical reasoning based on their characteristic properties, reactions, and transformation relationships.

## 🎓 考点精讲

**定义/原理**  
无机物的推断以物质结构决定性质、性质反映存在为核心原理。推断的基石是元素化合物的“特征性”：特征颜色（如 Fe³⁺ 棕黄、Cu²⁺ 蓝、Fe²⁺ 浅绿、MnO₄⁻ 紫红）、特征沉淀（如 AgCl 白不溶于酸、BaSO₄ 白不溶于酸、Fe(OH)₃ 红褐）、特征气体（如 SO₂ 品红褪色、NH₃ 使湿润红色石蕊试纸变蓝）以及特征转化关系（连续氧化、三角转化、相互滴加产生不同现象等）。

**关键方法**  
1. **审题抓突破口**：优先锁定题干中的特殊性质、特殊条件（高温、电解、催化剂）、特殊数据及特殊现象。  
2. **顺推或逆推**：从突破口出发，结合物质转化关系框图，正向或逆向推导，注意反应规律（氧化还原规律、强制弱规律）。例如：A → B → C 连续氧化，常见的有 S(H₂S) → SO₂ → SO₃、N₂(NH₃) → NO → NO₂、Na → Na₂O → Na₂O₂。  
3. **验证与排除**：将推断结果带入所有框线验证，确保与全部信息吻合。

**典型应用**  
高考中常以文字陈述或框图推断题呈现，设问角度包括：书写化学式/电子式、书写化学方程式/离子方程式、描述实验现象、设计物质检验方案。例如：已知某盐溶液与 NaOH 逐滴加入先沉淀后溶解，可推断含 Al³⁺；某气体使湿润淀粉-KI 试纸变蓝，可能是 Cl₂ 或 O₃ 等氧化性气体。解题必须熟练掌握钠、铝、铁、氯、硫、氮及其重要化合物的转化网络。

## ⚠️ 高频易错点

- **特征颜色混淆**：将 Fe²⁺ 的浅绿色与 Cu²⁺ 的蓝色混淆；将 Fe(OH)₂ 白色沉淀被氧化为 Fe(OH)₃ 的红褐色过程忽略。  
- **反应产物与量有关**：如 AlCl₃ 与 NaOH 反应，NaOH 少量生成 Al(OH)₃ 沉淀，过量则生成 NaAlO₂；CO₂ 与澄清石灰水反应，过量生成 Ca(HCO₃)₂ 使沉淀消失。  
- **特殊物质性质误判**：Al₂O₃ 与 Al(OH)₃ 为两性物质，既能溶于强酸也能溶于强碱；SiO₂ 只与 HF 反应，不与其他酸反应。  
- **连续氧化链断裂**：片面推断 N₂ → NO → NO₂ 而忽略 N₂ 与 O₂ 反应条件为放电；将 Fe 的氧化链误推为 Fe → Fe₂O₃ → Fe₃O₄ 等不合理转化。
