# 有机高分子化合物

> **一句话**：由小分子（单体）通过共价键重复连接而成、相对分子质量巨大（通常 >10⁴）的化合物称为有机高分子化合物。
> **English**: Organic polymer compounds are large molecules composed of repeating structural units (monomers) linked by covalent bonds, with very high molecular masses.

## 🎓 考点精讲

- **定义/原理**：有机高分子化合物以**聚合反应**为核心制备原理。聚合反应主要分为两类：
  - **加聚反应**：含有不饱和键（如 `C=C`）的单体，在引发剂或一定条件下，π 键打开并相互加成，形成高分子。通式可简示为：
    ```
    n CH₂=CHX → ─[ CH₂–CHX ]─ₙ
    ```
  - **缩聚反应**：单体间通过官能团反应（如 `–COOH` 与 `–OH`、`–COOH` 与 `–NH₂`）脱去小分子（如水、醇等）而逐步连接。典型实例：
    ```
    n HOOC–C₆H₄–COOH + n HO–CH₂CH₂–OH → 
        HO─[ OC–C₆H₄–COO–CH₂CH₂–O ]─ₙH + (2n-1)H₂O
    ```
  - **链节与聚合度**：高分子中重复的结构单元称为**链节**，链节的数目 **n** 为**聚合度**。高分子实为聚合度不同的同系物混合物（相对分子质量具有统计平均值）。
- **关键方法**：高考中常要求判断单体与高分子的关系，解题关键步骤为：
  1. **识别链节结构**：在碳骨架图中标出重复单元。
  2. **逆向推导单体**：加聚物需在链节中加入双键恢复单体；缩聚物需在连接处（酯基、肽键等）水解并补充脱去的小分子原子。
  3. **正写聚合方程式**：注意缩聚反应须配平小分子数目，反应物与产物末端用短线或原子标明端基。
- **典型应用**：本考点在高考中主要通过选择题和有机推断题考查，常见设问角度：①给定单体，写出其加聚或缩聚产物结构简式；②给出高分子片段，逆推单体结构简式；③判断某高分子是否属于线型或体型结构及其热塑/热固性。真实情境如：以聚乙烯、聚酯纤维（涤纶）、聚酰胺（尼龙）为载体，分析合成路线。

## ⚠️ 高频易错点

- **混淆加聚与缩聚的单体判断**：错误地在缩聚链节中强行插入双键，或忘记缩聚反应会脱去小分子导致链节与单体原子组成不符。
- **误以为聚乙烯分子中含有碳碳双键**：聚合后双键全部打开，聚乙烯链节均为饱和单键，其化学性质稳定、不能使溴水褪色。
- **书写缩聚反应方程式时漏写或错写端基原子/原子团**：链端常为 `-H`、`-OH` 等，且小分子系数（如 `2n-1`、`n-1`）易算错。
- **混淆“聚合度 n”与“参与反应的单体物质的量”**：聚合度是统计平均值，方程式系数 n 代表物质的量比例，书写时不应误用为质量比。
