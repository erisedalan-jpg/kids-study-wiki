# 极端假设法

> **一句话**：将复杂混合体系推向单一组成或反应进程的极限状态（如全为A或全为B），快速确定参数取值范围或反应终点的一种定性/半定量分析思维。
> **English**: A chemical problem-solving strategy that pushes a complex mixture or reaction to an extreme limit (e.g., pure component A, complete conversion) to rapidly determine parameter ranges or endpoints.

## 🎓 考点精讲

**定义/原理**：极端假设法是一种将多元变量体系简化为单一变量的推理方法。在化学中，常将混合物假设为纯物质，或将可逆反应推向完全进行，或让反应物按某一极端比例投料。核心逻辑是：真实体系的物理量（质量分数、气体体积、反应热、pH等）必然介于两个极端假设所得值之间。例如，对于 `CO` 与 `CO₂` 的混合气体，可分别假设全为 `CO` 和全为 `CO₂`，计算其燃烧耗氧量或碳元素质量分数，从而锁定混合气体的真实取值范围。

**关键方法**：
1. **界定极值**：明确混合体系或反应序列的两个极限状态（如只含轻组分/只含重组分，反应仅生成A/仅生成B）。
2. **孤立计算**：分别在两个极值下，利用化学方程式（如 `n = m/M`、理想气体状态方程 `V = n·Vm`）或守恒定律（原子守恒、电荷守恒）计算出待求量的边界值。
3. **判定范围**：真实值必定介于两边界之间的开区间或闭区间，据此排除选项中超出范围的数据。
4. **反应进程极值**：在连续反应（如 `Na₂CO₃` 逐滴加酸：`CO₃²⁻ → HCO₃⁻ → CO₂`）或竞争反应中，可假设反应全停于第一步或全进行到终点，结合原子守恒判断产物区间。

**典型应用**：
- **混合物组成分析**：高考选择题常给出一组金属混合物与酸反应生成的氢气体积，要求判断可能组成。极端假设使总质量全为某种金属，通过各自反应的化学计量关系（如 `Mg + 2H⁺ → Mg²⁺ + H₂↑`，`Al + 3H⁺ → Al³⁺ + 1.5H₂↑`）计算产氢量的极值上下限。
- **化学平衡移动**：对于 `2NO₂(g) ⇌ N₂O₄(g)` （正反应放热），求某温度下混合气体平均摩尔质量的取值范围。可极端假设全为 `NO₂`（M=46 g·mol⁻¹）或全为 `N₂O₄`（M=92 g·mol⁻¹），平均摩尔质量必介于46~92之间，且随温度或压强改变在此区间移动。

## ⚠️ 高频易错点

- **忽略化学意义的逻辑上限**：在气体混合物问题中，若极端假设某一组分占比100%，必须判断该组分在该条件下能否稳定存在或反应是否真能进行到底（如全为生成物时是否已达平衡极限），盲目套用边界会造成取值区间错误。
- **极端值与平均值关系的混淆**：求平均摩尔质量、平均相对原子质量时，易直接将两纯物质的物理量取算术平均，忽略了混合比例。真实平均值由各自物质的量分数加权决定，是落在极值区间内的某一点而非正中点。
- **反应顺序错位**：在多重反应（如 `OH⁻`、`CO₃²⁻` 混合液中滴加 `H⁺`）使用极值法时，误将反应产物直接叠加，忽略了反应有严格的先后顺序（酸碱中和优先于酸式盐生成）导致的产物分段极值。
- **区间开闭不当**：题干中的“某两种金属组成的合金”意味着不含单一金属，真实值不能等于极端值，取值范围应为开区间；若题干包含“可能只有一种”，则包括端点。
