# 湿法冶铜

> **一句话**：湿法冶铜是利用较活泼金属（如铁）从铜盐溶液中置换出单质铜的化学方法。
> **English**: A chemical method to extract copper by displacing it from its salt solution with a more reactive metal such as iron.

## 🎓 考点精讲

**定义/原理**：湿法冶铜基于金属活动性顺序，核心反应为铁与铜离子之间的置换反应：
\[ \text{Fe} + \text{Cu}^{2+} \rightarrow \text{Fe}^{2+} + \text{Cu} \]
典型实例为铁屑与硫酸铜溶液反应：
\[ \text{Fe} + \text{CuSO}_4 = \text{FeSO}_4 + \text{Cu} \]
反应中铁被氧化（化合价从 0 → +2），铜离子被还原（+2 → 0）。工业上常将低品位铜矿用稀硫酸（或微生物浸出）转化为 CuSO₄ 溶液，再投入废铁屑置换出海绵铜。

**关键方法**：解决“湿法冶铜”相关题目的关键步骤包括：
1. **理清流程**：酸浸 → 置换 → 过滤 → 洗涤、干燥。
2. **判断试剂用量**：为保证铜完全析出，常加入**过量铁粉**；反应后滤液中主要含 FeSO₄，滤渣为铜与多余铁粉的混合物。
3. **除杂与提纯**：向滤渣中加入适量稀盐酸或稀硫酸除去过量的铁粉，再次过滤可得较纯的单质铜。
4. **定量计算**：利用化学方程式，结合铁与酸可能存在的副反应（Fe + 2H⁺ → Fe²⁺ + H₂↑）进行质量或物质的量计算。

**典型应用**：高考中该考点常出现在**化学工艺流程题**或**无机综合题**中，设问角度如：
- 书写“置换”工序的离子方程式。
- 分析滤液、滤渣的成分（尤其铁过量或不足时的差异）。
- 评价加入“过量铁粉”的目的，或用锌、铝代替铁的优缺点。
- 结合 pH 调控、结晶、氧化还原滴定等进行综合考查。

## ⚠️ 高频易错点

- **混淆湿法冶铜与火法冶铜**：湿法属置换反应（溶液中进行）；火法冶铜是高温下用碳或氢气还原氧化铜，反应类型完全不同。
- **忘记铁与酸的反应**：若酸浸液中含有过量的 H₂SO₄，加入铁粉不仅置换铜，还会与酸反应产生氢气，影响铁的实际用量及滤液成分。
- **滤渣成分判定错误**：若铁粉不足，滤渣只含铜，滤液中残留 Cu²⁺；若铁粉过量，滤渣含铜和剩余铁，此时需用酸洗除去铁。考题常通过 pH 变化或后续操作暗示铁是否完全反应。
- **方程式书写不严谨**：将离子方程式误写成化学方程式，或漏写“Fe²⁺”与“Cu”，需注意电荷守恒与化合价变化。
