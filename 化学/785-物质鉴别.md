# 物质鉴别

> **一句话**：通过物质特有的物理性质或特征化学反应对未知成分进行判断与区分的实验方法。
> **English**: An experimental method that identifies or distinguishes unknown substances based on their characteristic physical properties or chemical reactions.

## 🎓 考点精讲

**定义/原理**  
物质鉴别是根据物质在颜色、状态、气味、溶解性、焰色等物理性质，以及沉淀、气体、显色等特征化学反应上的差异，将两种或多种物质区分开的操作。其核心在于抓住“某一性质或反应只存在于目标物质而不存在于其他待检物中”，从而实现明确区分。

常见原理：
- **物理法**：焰色反应（如 Na⁺ 黄、K⁺ 紫）、溶解度差异、气味等。  
- **化学法**：利用特征试剂与离子或官能团发生特有反应，典型如：  
  - 沉淀反应：Cl⁻ + Ag⁺ → AgCl↓（白色，不溶于稀硝酸）；SO₄²⁻ + Ba²⁺ → BaSO₄↓（白色，不溶于酸）；  
  - 气体生成：CO₃²⁻ + 2H⁺ → CO₂↑ + H₂O，NH₄⁺ + OH⁻ ─△→ NH₃↑；  
  - 显色反应：Fe³⁺ + SCN⁻ → 血红色配合物；醛基（—CHO）与银氨溶液水浴得银镜，与新制 Cu(OH)₂ 共热生成砖红色 Cu₂O↓。

**关键方法**  
1. **取样**：取少量固体或溶液于试管中（若为固体先研磨或加水配成溶液）。  
2. **操作**：逐滴加入所选试剂，必要时控制顺序与用量，如先加酸排干扰再加沉淀剂。  
3. **观察**：记录沉淀颜色、气体气味及能使湿润的红色石蕊试纸变蓝等特征现象。  
4. **结论**：对比待鉴别物质，唯一性地确定种类，注意排除干扰离子的影响。

**典型应用**  
高考常以实验设计与评价的形式出现：  
- **离子鉴别情境**：给出四种白色粉末（NaCl、Na₂CO₃、Na₂SO₄、NH₄Cl），要求只用一种试剂或任选试剂鉴别。解题时可用稀盐酸：有气泡的为 Na₂CO₃，另取样品与浓 NaOH 加热，使湿润红色石蕊试纸变蓝的为 NH₄Cl；再用 BaCl₂ 区分 NaCl 和 Na₂SO₄（硫酸根重现不溶于酸的白色沉淀）。  
- **有机官能团鉴别**：鉴别乙醇、乙醛、乙酸，可用新制 Cu(OH)₂ 悬浊液，羧基能使悬浊液变澄清，醛基加热出现砖红色沉淀，无变化者为乙醇。  

设问方式常为“设计简单实验方案鉴别……”“下列方法能否鉴别……请说明理由”，需严格遵循“操作→现象→结论”的答题结构。

## ⚠️ 高频易错点

- **混淆鉴别与鉴定**：鉴别只需区分给定物质，不一定推断全部成分；鉴定则需确定某一物质是什么，通常须完成一系列检验并排除干扰。  
- **忽视干扰离子**：如用 AgNO₃ 鉴别 Cl⁻ 和 SO₄²⁻ 时，未酸化可能因 CO₃²⁻ 生成 Ag₂CO₃ 白色沉淀而误判。应先加稀硝酸酸化，无气泡后再检验。  
- **物理法与化学法割裂**：焰色反应鉴别 Na⁺、K⁺ 时，若直接观察不透过蓝色钴玻璃，会因钠离子干扰而将 K⁺ 的紫色误判为无，从而遗漏线索。  
- **操作顺序失误**：鉴别 Na₂CO₃ 和 Na₂SO₄ 时，若先加 BaCl₂ 再加盐酸，沉淀部分溶解只能判断两者混合，无法一一区分，正确顺序应单独取样分别检验。
