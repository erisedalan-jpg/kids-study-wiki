# 用盖斯定律进行有关反应热的计算

> **一句话**：利用盖斯定律，将已知热化学方程式通过线性组合（加减、乘以系数、逆向）构造出目标方程式，从而间接计算目标反应的反应热（ΔH），适用于难以直接测定的反应。
> **English**: A method of determining the enthalpy change (ΔH) of a target reaction by algebraically combining known thermochemical equations based on Hess's law, which states that the total enthalpy change depends only on the initial and final states, not the pathway.

## 🎓 考点精讲

**定义/原理**：
盖斯定律指出：在恒压或恒容条件下，一个化学反应的反应热（焓变 ΔH）只取决于反应的始态和终态，而与反应是一步完成还是分多步完成无关。数学上表现为 ΔH 是状态函数的变化量，具有加和性。

若目标热化学方程式可表示为已知方程式（1）、（2）、（3）… 的线性组合：
$$
\text{目标} = a \times \text{式(1)} + b \times \text{式(2)} + c \times \text{式(3)} + \cdots
$$
则目标反应的焓变 ΔH 遵循相同的线性关系：
$$
\Delta H_{\text{目标}} = a \times \Delta H_1 + b \times \Delta H_2 + c \times \Delta H_3 + \cdots
$$
系数 a、b、c 为有理数。若某已知方程式需要“倒过来”使用（生成物变反应物，或反应物变生成物），则其 ΔH 需变号（如 ΔH → -ΔH）。

**关键方法（“加加减减”配凑法）**：
解决该类计算题的标准步骤如下：
1.  **确定唯一物质**：在目标反应中找出只出现在某一个已知反应中的物质（通常优先选择在目标反应中系数较特殊的物质）。
2.  **调整方向**：通过对比该物质在该已知反应与目标反应中是位于箭头同一侧还是相反侧，决定是否将该已知反应逆向。（逆向则 ΔH 变号）。
3.  **调整系数**：通过乘以适当系数，使该物质在已知反应中的化学计量数与目标反应中完全一致。（ΔH 需乘以相同系数）。
4.  **消元验证**：将调整后的各已知反应相加，消去中间产物（即目标反应中不出现的物质），最终得到目标反应。
5.  **ΔH 求和**：将各步调整后的 ΔH 代数和相加，即为目标反应的 ΔH。

**典型应用**：
高考常以非选择题中的计算填空形式出现，设问角度包括：计算陌生反应的反应热、计算燃烧热或生成热、结合“能量-反应进程图”计算活化能或焓变。

- **场景一（合成氨催化）**：已知 ① N₂(g) + 3H₂(g) ⇌ 2NH₃(g) ΔH₁ = -92.4 kJ/mol，② 2H₂(g) + O₂(g) = 2H₂O(g) ΔH₂ = -483.6 kJ/mol，③ N₂(g) + O₂(g) = 2NO(g) ΔH₃ = +180.5 kJ/mol。求“4NH₃(g) + 5O₂(g) = 4NO(g) + 6H₂O(g) 的 ΔH”。
    - **配凑**：目标反应中 NH₃ 在式①是生成物且系数为 2，需将式① ×2 并逆向，得 ΔH' = +184.8 kJ/mol。NO 在式③是生成物系数为 2，需将式③ ×2，得 ΔH'' = +361.0 kJ/mol。H₂O 在式②是生成物系数为 2，需将式② ×3，得 ΔH''' = -1450.8 kJ/mol。三式叠加，H₂、N₂、O₂ 均消去，恰好得到目标反应，ΔH = +184.8 + 361.0 + (-1450.8) = -905.0 kJ/mol。

- **场景二（碳燃烧不完全）**：已知 C(s) + O₂(g) = CO₂(g) ΔH₁ = -393.5 kJ/mol，CO(g) + 1/2 O₂(g) = CO₂(g) ΔH₂ = -283.0 kJ/mol。求 C(s) + 1/2 O₂(g) = CO(g) 的 ΔH。
    - **配凑**：把式②逆向（ΔH₂ 变号为 +283.0），再与式①相加，正好消去 CO₂，得目标反应，ΔH = -393.5 + 283.0 = -110.5 kJ/mol。

## ⚠️ 高频易错点

- **遗漏系数或系数配错**：对已知反应进行等比例缩放时，常忘记将 ΔH 乘以相同的系数。例如将方程式系数减半时，ΔH 也必须严格减半。
- **焓变符号遗漏或反向**：当为了消去中间产物需要将某个已知反应逆向书写时，学生只改写了方程式，但忘记将 ΔH 的“+”与“-”变号，导致计算结果正负号完全相反。
- **物质状态被忽略**：目标反应与已知反应中，某些物质虽化学式相同但聚集状态不同（如 H₂O(l) 与 H₂O(g)），不能直接用盖斯定律消去，必须先通过相变热或额外方程式将状态统一（常见于水蒸气与液态水的焓值差异，约 44 kJ/mol）。
- **加减运算时混淆**：多个 ΔH 代数相加时，因既有加又有减，导致纯计算错误。建议先理清各步 ΔH 的正负与大小，再分步求和。
