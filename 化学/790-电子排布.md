# 电子排布

> **一句话**：电子排布指原子核外电子在各原子轨道（能级）上的分布方式，遵循构造原理、泡利不相容原理与洪特规则。
> **English**: Electron configuration describes how electrons are distributed among atomic orbitals (energy levels) in an atom, governed by the Aufbau principle, Pauli exclusion principle, and Hund's rule.

## 🎓 考点精讲

### 定义与核心原理

电子排布决定了元素的化学性质，其书写必须同时满足三条基本规则：

**1. 构造原理（Aufbau Principle）**

核外电子总是优先占据能量最低的轨道。多电子原子中，能级顺序由 $(n+0.7l)$ 规则近似判断（$n$ 为主量子数，$l$ 为角量子数），或直接记忆填充顺序图（1s → 2s → 2p → 3s → 3p → 4s → 3d → 4p → 5s → 4d → 5p → 6s → 4f → 5d → 6p → 7s → 5f → 6d → 7p）。注意**能级交错**：$E_{4s}<E_{3d}$、$E_{5s}<E_{4d}$ 等，这是书写 3d 过渡金属排布时的关键陷阱。

**2. 泡利不相容原理**

每个原子轨道最多容纳 2 个电子，且自旋方向相反（用 $\uparrow \downarrow$ 表示）。同一原子内不可能有四个量子数完全相同的两个电子。

**3. 洪特规则**

基态原子中，电子在简并轨道（如 $2p_x, 2p_y, 2p_z$）上分布时，优先以自旋平行的方式单独占据不同轨道，使体系总自旋最大（能量最低）。**洪特规则特例**：等价轨道处于全充满（$p^6, d^{10}, f^{14}$）、半充满（$p^3, d^5, f^7$）或全空（$p^0, d^0, f^0$）时体系较稳定。实例：$\text{Cr}$ 的基态价电子排布为 $3d^5 4s^1$ 而非 $3d^4 4s^2$；$\text{Cu}$ 为 $3d^{10} 4s^1$ 而非 $3d^9 4s^2$。

### 书写电子排布式的标准步骤

1. **确定原子序数**，即核外电子总数 $Z$。
2. **按能级顺序**逐个填充电子，每个轨道最多 2 个电子。
3. **写成电子排布式**，将同一主层（$n$ 相同）的能级合并书写，如 $\text{Fe},\ Z=26:\ 1s^2 2s^2 2p^6 3s^2 3p^6 3d^6 4s^2$（按 $n$ 值升序写，而非填充顺序）。
4. **简化写法**：用前一周期的稀有气体符号表示内层电子（原子实），如 $\text{Fe}:\ [\text{Ar}] 3d^6 4s^2$。
5. **价层电子排布式**与**轨道表示式（电子排布图）** 必须体现洪特规则的方向：用小方框表示轨道，箭号表示自旋，先单独后配对。

### 高考典型考查方式

- **书写与正误判断**：给出原子或离子（如 $\text{Fe}^{2+}$、$\text{Mn}^{2+}$、$\text{Cu}^+$），要求写出基态电子排布式或价层电子排布（注意失电子顺序：先失去最外层 $4s$ 电子，再失去内层 $3d$ 电子）。
- **洪特规则特例判断**：识别 $\text{Cr}$ 族与 $\text{Cu}$ 族的特殊排布，及能级交错引起的常见错误排布。
- **元素推断**：结合电离能、电负性等数据反向推断原子序数或元素种类。
- **微粒磁性判断**：根据是否存在未成对电子判断顺磁性（有）或抗磁性（无）。

## ⚠️ 高频易错点

- **能级顺序混淆**：错误地将 $3d$ 填在 $4s$ 之前（应按 $4s$ 后 $3d$ 填充，但书写时按 $n$ 值写回先 $3d$ 后 $4s$）。
- **失电子顺序与填充顺序混淆**：基态原子书写先 $4s$ 后 $3d$，但形成阳离子时**先失去最外层 $4s$ 电子**。如 $\text{Fe}^{2+}$ 的价层排布是 $3d^6$，而非 $3d^4 4s^2$。
- **洪特规则画图错误**：轨道式书写中，$p^3$ 应画为 $\uparrow\ \uparrow\ \uparrow$（三个轨道各一个自旋相同），而非先配对。
- **忽略洪特规则特例**：$\text{Cr}$、$\text{Cu}$、$\text{Mo}$、$\text{Ag}$、$\text{Au}$ 等存在 $d^4s^2 \to d^5s^1$ 或 $d^9s^2 \to d^{10}s^1$ 的稳定化，22Ti~30Zn 周期常作为区分题命题。
