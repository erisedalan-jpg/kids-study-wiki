# 硅和二氧化硅

> **一句话**：硅是位于元素周期表第三周期第ⅣA族的亲氧性原子晶体非金属，二氧化硅是硅最稳定的氧化物，由硅氧四面体构成的空间网状原子晶体。
> **English**: Silicon is an atomic crystalline nonmetal in the third period, group ⅣA of the periodic table with strong oxophilicity, and silicon dioxide is its most stable oxide, existing as an atomic crystal with a three-dimensional network of SiO₄ tetrahedra.

## 🎓 考点精讲

**硅的晶体结构与性质**：硅单质是原子晶体，每个Si原子以sp³杂化轨道与4个Si原子形成Si—Si共价键，正四面体空间网状结构。硅的化学性质稳定，常温下仅与F₂、HF、强碱反应。核心反应方程式：`Si + 2NaOH + H₂O = Na₂SiO₃ + 2H₂↑`，考点是碱解法中氢气的来源分析——1 mol Si对应2 mol H₂，其中H既来自NaOH也来自H₂O。硅在加热条件下可与O₂、Cl₂等非金属反应，高温下可与碳反应生成碳化硅（金刚砂）。

**二氧化硅的晶体结构与性质**：SiO₂是原子晶体，每个Si原子被4个O原子包围呈正四面体，每个O原子连接2个Si原子，空间网状结构无限延伸。化学式SiO₂仅表示晶体中最简原子个数比，不代表单个分子。SiO₂是酸性氧化物，但与水不反应，只能与碱或碱性氧化物在高温下反应，如`SiO₂ + 2NaOH = Na₂SiO₃ + H₂O`、`SiO₂ + CaO \xrightarrow{\text{高温}} CaSiO₃`。特征反应是SiO₂与氢氟酸反应：`SiO₂ + 4HF = SiF₄↑ + 2H₂O`，这是高考中“唯一能与氢氟酸反应的酸性氧化物”高频考点。

**高考考查方式**：常以非选择题形式考查硅的提纯方法（粗硅→纯硅三步法：`SiO₂ + 2C \xrightarrow{\text{高温}} Si(粗) + 2CO↑`、`Si(粗) + 2Cl₂ \xrightarrow{\Delta} SiCl₄`、`SiCl₄ + 2H₂ \xrightarrow{\text{高温}} Si(纯) + 4HCl`），以及SiO₂与碳酸盐高温反应的方程式：`SiO₂ + Na₂CO₃ \xrightarrow{\text{高温}} Na₂SiO₃ + CO₂↑`、`SiO₂ + CaCO₃ \xrightarrow{\text{高温}} CaSiO₃ + CO₂↑`——此反应体现“高沸点酸酐制低沸点酸酐”原理。物质鉴别题中，SiO₂与其他白色固体的区分要用氢氟酸或浓NaOH溶液。

## ⚠️ 高频易错点

- **混淆晶体类型**：误以为SiO₂是分子晶体而写成“二氧化硅分子”。SiO₂是原子晶体，由原子直接构成，不存在单个分子。
- **SiO₂与水的反应**：错误类比“酸性氧化物+水→含氧酸”的规律（如SO₂、CO₂），认为SiO₂也能与水反应生成硅酸。实际上SiO₂不与水反应，硅酸只能用可溶性硅酸盐与酸反应制取。
- **硅与NaOH反应的计算陷阱**：产物H₂中的H原子来源分配错误。`Si + 2NaOH + H₂O = Na₂SiO₃ + 2H₂↑`中，4个H原子有2个来自NaOH、2个来自H₂O。
- **粗硅提纯中的副反应**：粗硅用Cl₂氯化得到SiCl₄时，需蒸馏提纯以除去杂质氯化物（如FeCl₃、AlCl₃），再被H₂还原——忽视蒸馏步骤会导致产物不纯的分析错误。
