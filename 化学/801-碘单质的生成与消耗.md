# 碘单质的生成与消耗

> **一句话**：碘单质(I₂)的生成与消耗本质是碘元素在氧化剂或还原剂作用下发生氧化还原反应的过程，核心围绕 I⁻ ⇌ I₂ 的转化。
> **English**: The formation and consumption of iodine (I₂) refers to the redox processes in which iodine element is converted to or from its elemental form via the I⁻ ⇌ I₂ interconversion under the action of oxidizing or reducing agents.

## 🎓 考点精讲

**定义/原理**  
碘单质的生成反应主要是 I⁻ 被氧化成 I₂：
- 常见氧化剂：Cl₂、Br₂、酸性 KMnO₄、H₂O₂、浓 H₂SO₄、Fe³⁺ 等。  
  反应举例：2I⁻ + Cl₂ → I₂ + 2Cl⁻；2Fe³⁺ + 2I⁻ → 2Fe²⁺ + I₂。
- 从化合价看，I 从 -1 价升到 0 价，表现还原性。

碘单质的消耗反应主要是 I₂ 被还原成 I⁻：
- 常见还原剂：S²⁻、SO₃²⁻、S₂O₃²⁻、Fe²⁺、活泼金属等。  
  反应举例：I₂ + 2S₂O₃²⁻ → 2I⁻ + S₄O₆²⁻（定量反应，用于碘量法）；I₂ + H₂S → 2HI + S↓。
- 另外，I₂ 还可发生歧化反应：I₂ + 2OH⁻ → I⁻ + IO⁻ + H₂O（在碱性条件下消耗自身）。

**关键方法**  
解决碘单质生成与消耗问题的标准步骤：
1. **确定价态变化**：找出 I⁻ 或 I₂ 在反应中的角色，判断是被氧化还是被还原。
2. **选择合适反应**：根据实验目的（检验 I⁻、除去 I₂、定量测定等）匹配经典反应。
3. **注意介质酸性**：许多氧化剂氧化 I⁻ 需在酸性条件下进行，如酸性 KMnO₄ + I⁻ → Mn²⁺ + I₂。
4. **关注现象**：I₂ 水溶液呈棕黄色，遇淀粉变蓝是特征检验；I₂ 在有机溶剂中呈紫色或粉红色，可用于萃取判断。

**典型应用**  
- **高考常见设问**：给出几种氧化剂/还原剂，要求书写离子方程式并分析电子转移；或设计实验证明 I⁻ 还原性比 Br⁻ 强（如通入氯水，先氧化 I⁻ 后氧化 Br⁻）。
- **真实情境**：海水提碘流程（海带灰浸取液酸化后加 H₂O₂ 或通 Cl₂ 氧化 I⁻ 得到 I₂）；加碘食盐中碘含量的测定（利用 I⁻ + IO₃⁻ + H⁺ → I₂ + H₂O 归中反应，再用 Na₂S₂O₃ 滴定生成的 I₂）。

## ⚠️ 高频易错点

- **混淆卤素氧化性与卤离子还原性顺序**：氧化性 Cl₂ > Br₂ > I₂；还原性 I⁻ > Br⁻ > Cl⁻。在与同一氧化剂反应时，I⁻ 优先被氧化，这是卤素置换与分步氧化的核心。
- **忽略碘单质在不同溶剂中的颜色**：水中呈棕黄色（溶解度小），CCl₄ 中呈紫红色，淀粉溶液中显蓝色。答题时需根据溶剂判断颜色，不可一概写“紫色”。
- **歧化反应条件混淆**：I₂ 与碱反应生成 I⁻ 和 IO⁻（或 IO₃⁻ 加热），而 Cl₂ 与碱低温生成 Cl⁻ 和 ClO⁻，加热生成 ClO₃⁻。两者条件与产物不同，易搞混。
- **碘量法滴定时忽略淀粉指示剂加入时机**：在用 Na₂S₂O₃ 滴定 I₂ 时，近终点（溶液呈浅黄色）方可加入淀粉，过早加入会因大量 I₂ 吸附淀粉导致终点滞后且变色不明显。
