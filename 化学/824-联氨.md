# 联氨

> **一句话**：联氨（肼）是化学式为 N₂H₄ 的氮氢化合物，氮原子采取 sp³ 杂化，具有弱碱性和强还原性。
> **English**: Hydrazine is a nitrogen-hydrogen compound with the formula N₂H₄, in which nitrogen atoms are sp³ hybridized, exhibiting weak basicity and strong reducibility.

## 🎓 考点精讲

**定义/原理**  
联氨的结构可视为两个氨基（-NH₂）直接相连，每个 N 原子上有一对孤电子对，因此能结合质子，呈现弱碱性：  
`N₂H₄ + H₂O ⇌ N₂H₅⁺ + OH⁻   K_b ≈ 8.5×10⁻⁷（弱于 NH₃）`  
联氨中 N 的化合价为 **–2**，处于中间价态，既有氧化性又有还原性，但以还原性为主，可被氧化为 N₂：  
`N₂H₄ + O₂ → N₂ + 2H₂O`（可作火箭燃料）  
与强氧化剂（如 H₂O₂、浓 HNO₃）剧烈反应，放热明显，常用于燃料和推进剂。

**关键方法**  
1. **化合价与方程式配平**：联氨中 N 为 –2 价，氧化产物一般为 N₂（N 为 0 价）。配平氧化还原反应时，准确标注化合价，利用电子守恒。  
2. **电化学书写**：在碱性燃料电池中，联氨在负极失去电子，结合 OH⁻ 生成 N₂ 和水：  
`N₂H₄ + 4OH⁻ – 4e⁻ = N₂↑ + 4H₂O`  
正极通入氧气：`O₂ + 2H₂O + 4e⁻ = 4OH⁻`。  
3. **热化学计算**：熟练运用盖斯定律，通过已知热化学方程式求联氨燃烧或分解的反应热。

**典型应用**  
- 考查联氨-空气碱性燃料电池的电极反应式书写及总反应。  
- 以联氨与过氧化氢反应 `N₂H₄(l) + 2H₂O₂(l) = N₂(g) + 4H₂O(g)` 为载体，计算反应热或考查化学键与能量的关系。  
- 结合实验设计，检验联氨还原性及产物 N₂ 的验证。

## ⚠️ 高频易错点

- **误判 N 的化合价**：联氨（N₂H₄）中 N 为 –2，常被错当为 –3（类比 NH₃）。注意 H 为 +1，N—N 非极性键不改变化合价。  
- **混淆碱性相对强弱**：联氨的 K_b 约为 8.5×10⁻⁷，碱性比氨（K_b≈1.8×10⁻⁵）弱，并非强碱，且二级电离极弱。  
- **电极反应忽略介质条件**：在碱性介质中，负极反应必须配有 OH⁻ 配平电荷，若误写成 `N₂H₄ – 4e⁻ = N₂ + 4H⁺` 则错误。  
- **产物不明确**：联氨被氧化通常生成 N₂，但在某些复杂条件下（如浓硝酸过量）可能生成氮氧化物，高考题若无特殊说明，一般按生成 N₂ 处理。
