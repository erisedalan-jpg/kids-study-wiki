# 酚羟基

> **一句话**：酚羟基是直接连接在苯环上的羟基（–OH），因其氧原子的孤对电子与苯环大π键形成p-π共轭，导致O–H键极性增大、更易断裂，表现出弱酸性。
> **English**: A phenolic hydroxyl group is an –OH group directly bonded to an aromatic ring, which exhibits weak acidity due to p-π conjugation between the oxygen lone pair and the aromatic π system.

## 🎓 考点精讲

**定义与结构本质**
酚羟基的化学特性源于其电子结构：氧原子的孤对电子轨道与苯环π轨道平行重叠，形成p-π共轭体系。这一共轭导致两个关键后果：① 氧上电子云向苯环转移，使O–H键电子对更偏向氧，氢更易以质子形式离去，故苯酚具有酸性（$K_a \approx 1.0 \times 10^{-10}$，酸性强于醇但弱于碳酸）；② 苯环上电子云密度升高，尤其邻、对位，使其在亲电取代（如溴代、硝化）中活性远强于苯。

**核心反应与逻辑主线**
高考围绕酚羟基主要考查三大反应类型：
1. **弱酸性**：苯酚与NaOH反应生成苯酚钠（`C₆H₅OH + NaOH → C₆H₅ONa + H₂O`），但**不能**与NaHCO₃反应（酸性弱于碳酸）。向苯酚钠溶液中通入CO₂会析出苯酚（`C₆H₅ONa + CO₂ + H₂O → C₆H₅OH + NaHCO₃`），此为分离提纯的经典方法。
2. **苯环上的亲电取代**：酚羟基的活化效应使苯酚与浓溴水常温下立即生成2,4,6-三溴苯酚白色沉淀，无需催化剂——这是苯酚的定性检验与定量分析的依据。
3. **显色反应**：含酚羟基的化合物遇FeCl₃溶液显紫色，用于鉴别酚类。

**高考典型设问**
- **有机推断/鉴别**：给定某芳香族化合物，根据其遇FeCl₃显色、能与NaOH反应但不与NaHCO₃反应，推断含酚羟基。
- **实验操作**：用NaOH溶液分离苯酚与苯的混合物，再通CO₂回收苯酚。
- **排序与比较**：比较乙醇、水、苯酚、碳酸、乙酸的酸性强弱顺序（乙醇 < 水 < 苯酚 < 碳酸 < 乙酸）。

## ⚠️ 高频易错点

- **混淆酸性比较**：误认为苯酚能电离出H⁺就能与NaHCO₃反应生成CO₂。实际上，苯酚的酸性弱于碳酸（$K_{a1}$），通CO₂于苯酚钠只生成NaHCO₃而非Na₂CO₃，苯酚不能使酸碱指示剂变色。
- **误判酚羟基与醇羟基的性质**：酚羟基因直接连苯环而有酸性，醇羟基连脂肪烃基无酸性。结构上必须严格区分：`C₆H₅–OH`为酚，`C₆H₅–CH₂–OH`为芳香醇（苯甲醇），后者没有酚的特性（不与NaOH反应、不显三氯化铁紫色）。
- **遗忘酚羟基对苯环的活化**：在多官能团有机题中，忽视酚羟基使苯环邻对位电子云密度升高，导致定位效应与反应活性判断错误（如溴水取代时是苯环发生三取代而非侧链反应）。
- **苯酚的氧化干扰**：酚羟基也容易被氧化，苯酚久置空气中呈粉红色即为被氧化产物。在推断题中，若看到颜色变化，需区分是酚羟基被氧化的颜色还是与FeCl₃的紫色反应。
