# 铜的冶炼

> **一句话**：铜的冶炼主要包括火法炼铜（造锍熔炼）与湿法炼铜，本质是利用氧化还原反应将化合态铜（Cu⁺/Cu²⁺）还原为金属单质（Cu），核心反应是粗铜的获得与电解精炼。
> **English**: Copper smelting involves extracting metallic copper from its ores through pyrometallurgical (matte smelting) or hydrometallurgical processes, fundamentally employing redox reactions to reduce combined copper (Cu⁺/Cu²⁺) to elemental copper (Cu), with the core steps being crude copper production and electrolytic refining.

## 🎓 考点精讲

**核心原理**：从化合态铜（如黄铜矿 $\rm CuFeS_2$、辉铜矿 $\rm Cu_2S$、赤铜矿 $\rm Cu_2O$、孔雀石 $\rm Cu_2(OH)_2CO_3$ 等）获得金属铜，基于氧化还原反应，根据矿石类型选择熔炼或浸出路径。关键化学方程式：

- **火法炼铜—造渣反应**（造锍熔炼）：
  $$\rm 2CuFeS_2 + O_2 \xrightarrow{\Delta} Cu_2S + 2FeS + SO_2$$（部分氧化，生成冰铜）
  $$\rm 2FeS + 3O_2 \xrightarrow{\Delta} 2FeO + 2SO_2$$（$\rm FeO$ 与 $\rm SiO_2$ 造渣：$\rm FeO + SiO_2 = FeSiO_3$）
- **火法炼铜—粗铜的获得**（氧化除杂）：
  $$\rm 2Cu_2S + 3O_2 \xrightarrow{\Delta} 2Cu_2O + 2SO_2$$
  $$\rm 2Cu_2O + Cu_2S \xrightarrow{\Delta} 6Cu + SO_2 \uparrow$$（“冰铜吹炼”核心自还原反应）
- **湿法炼铜—浸出与置换/电积**：
  $$\rm CuO + H_2SO_4 = CuSO_4 + H_2O$$（氧化铜矿酸浸）
  $$\rm CuSO_4 + Fe = FeSO_4 + Cu \downarrow$$（铁屑置换，适用于低品位矿/尾矿）
- **电解精炼**（获取高纯铜）：
  阳极（粗铜）：$\rm Cu - 2e^- = Cu^{2+}$；阴极（纯铜）：$\rm Cu^{2+} + 2e^- = Cu$；电解液：$\rm CuSO_4 / H_2SO_4$。比铜活泼的金属（Zn、Fe、Ni 等）溶解进入溶液，不如铜活泼的金属（Ag、Au、Pt 等）形成阳极泥。

**关键方法**：处理此类问题需抓住氧化还原主线，区分不同阶段的目的：①造渣（去除 Fe、S 等杂质）；②吹炼（获得粗铜的自还原）；③精炼（电化学提纯）。解题时，依据所给矿石成分或流程图，先判断炼铜方法（火法还是湿法），再写出对应的分步反应或电极反应式，并注意杂质去向（气体、炉渣、阳极泥）。

**高考典型设问**：
- 物质转化型推断题：给出铜冶炼流程或简单框图，要求书写陌生氧化还原方程式，并判断某步操作的化学原理（如造渣剂 $\rm SiO_2$ 的作用）。
- 电解原理应用题：给出电解精炼铜装置，要求书写电极反应式，判断电解液 $\rm Cu^{2+}$ 浓度变化（因阳极活泼金属共溶而阴极析出纯铜，通常浓度略减），或分析阳极泥成分。

## ⚠️ 高频易错点

- **混淆火法炼铜的两个核心反应**：误以为 $\rm Cu_2S$ 还原 $\rm Cu_2O$ 是外加还原剂（如 $\rm C$、$\rm CO$），实际是 $\rm Cu_2S$ 自身的负二价硫将 $\rm Cu^+$ 还原为 $\rm Cu$，产物有 $\rm SO_2$。
- **电解精炼时电极判断与产物混淆**：误将粗铜接电源负极（应为阳极）；误认为阳极只有 $\rm Cu$ 溶解（Zn、Fe 等比铜活泼的金属也溶出）；误认为电解液 $\rm Cu^{2+}$ 浓度始终不变（实际上因阳极杂质金属共溶而阴极析出纯铜，$\rm Cu^{2+}$ 浓度通常略降）。
- **杂质去向不清**：比铜活泼的金属形成阳离子进入溶液，不如铜活泼的金属形成阳极泥，造渣过程产生的 $\rm SO_2$ 不能直接排放（需回收制 $\rm H_2SO_4$）。若将阳极泥成分误写为 $\rm Fe$、$\rm Zn$ 等，属于典型错误。
- **湿法炼铜仅记置换而忽略浸出前提**：需明确湿法炼铜通常先转化为可溶性铜盐（$\rm CuSO_4$），若有氧化铜矿可直接酸浸；若为硫化矿则需先氧化焙烧（$\rm CuS \rightarrow CuO$），再酸浸。
