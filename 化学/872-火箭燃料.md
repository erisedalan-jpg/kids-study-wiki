# 火箭燃料

> **一句话**：火箭燃料是由燃料与氧化剂构成的高能化学推进剂体系，通过剧烈氧化还原反应瞬间产生大量高温高压气体，以反冲原理产生推力。
> **English**: Rocket fuel is a high-energy chemical propellant system consisting of a fuel and an oxidizer that undergoes violent redox reactions to rapidly generate large volumes of high-temperature, high-pressure gas for thrust via recoil.

## 🎓 考点精讲

**定义与原理**：火箭燃料在高考化学中主要关联「化学反应与能量变化」及「氧化还原反应」两大核心板块。化学推进的本质是**燃料（还原剂）与氧化剂之间的剧烈氧化还原反应**，反应物化学键断裂吸收能量、产物化学键形成释放能量，净放出的热量使产物气体急剧膨胀，高速喷出产生推力。高考考查的核心是**热化学方程式书写**与**反应热（ΔH）计算**，常用盖斯定律、键能法或燃烧热数据进行求解。

常见火箭燃料体系及代表反应：

- **液氢 – 液氧**：`2H₂(l) + O₂(l) → 2H₂O(g)`，ΔH = −571.6 kJ·mol⁻¹（按 2 mol H₂ 计），比冲极高，产物无污染。
- **偏二甲肼 – 四氧化二氮**：`(CH₃)₂NNH₂(l) + 2N₂O₄(l) → 3N₂(g) + 2CO₂(g) + 4H₂O(g)`。该反应中 N₂O₄ 为氧化剂（N 从 +4 降至 0），偏二甲肼为还原剂（C 从 −2 升至 +4，N 从 −2 升至 0），电子转移需整体分析（待家长核对转移电子总数）。
- **煤油 – 液氧**：煤油近似以 C₁₂H₂₆ 代表，反应通式 `2C₁₂H₂₆ + 37O₂ → 24CO₂ + 26H₂O`，高考通常给出具体热化学数据，用盖斯定律计算 ΔH。

**关键方法**：
1. **热化学方程式书写**：必须标注物质聚集状态（l/g/s）及对应 ΔH，注意 ΔH 与化学计量数的比例关系。
2. **盖斯定律**：将目标反应拆分为已知 ΔH 的若干步反应，进行代数加减，消去中间物质。
3. **键能估算 ΔH**：`ΔH ≈ ΣE(反应物键断裂) − ΣE(产物键形成)`，需准确判断分子结构（如 N₂O₄ 含 N—N 键与 N—O 键，N₂ 含 N≡N 三键）。

**典型应用**：高考常以火箭发射为真实情境，设问方式包括：①根据给定热化学方程式计算一定质量燃料燃烧的放热量；②利用盖斯定律推导新型推进剂反应的热效应；③分析推进剂反应中的氧化剂、还原剂及电子转移数目；④结合 N₂O₄ ⇌ 2NO₂ 平衡，考查条件改变对平衡移动的影响。

## ⚠️ 高频易错点

- **偏二甲肼与 N₂O₄ 反应方程式配平错误**：产物中 N 全部以 N₂ 形式存在（来自偏二甲肼的氨基与 N₂O₄ 的还原），C 以 CO₂ 形式、H 以 H₂O 形式存在，配平时需抓住 N 原子守恒整体配平。
- **混淆燃烧热与反应热**：火箭燃料反应通常不是标准燃烧热（如偏二甲肼与 N₂O₄ 的反应并非与 O₂ 燃烧），答题时不可套用「燃烧热」定义，应直接依据盖斯定律或键能进行计算。
- **键能计算时遗漏 N₂O₄ 中的 N—N 键**：N₂O₄ 结构为 O₂N—NO₂，含 1 个 N—N 单键、4 个 N—O 键（每个 NO₂ 中 N—O 键级约为 1.5，高考通常简化为 N—O 单键与 N=O 双键各两个，或直接给出键能数据），产物 N₂ 含 N≡N 三键，键能差异显著，计算时需逐键核对。
- **忽视反应物状态对 ΔH 的影响**：液氢、液氧的汽化热会抵消部分反应放热，若题目给定液态反应物，不可直接套用气态标准生成焓数据，需结合汽化焓进行换算或直接使用题目所给数据。
