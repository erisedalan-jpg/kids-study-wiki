# 氧化还原反应的基本规律

> **一句话**：氧化还原反应中，物质得失电子、化合价变化必然遵循的守恒规律、强弱规律、优先规律以及价态归中与歧化规律。
> **English**: The fundamental rules governing electron transfer and oxidation state changes in redox reactions, including conservation, strength priority, and valence state convergence/disproportionation.

## 🎓 考点精讲

- **定义/原理**：氧化还原反应的基本规律是判断反应方向、产物及进行定量计算的核心依据，主要包括以下五条：
    - **电子守恒（化合价升降守恒）**：还原剂失去的电子总数等于氧化剂得到的电子总数，表现为化合价升高总数等于化合价降低总数。这是氧化还原方程式配平和计算的根本依据。
    - **强弱规律**：自发进行的氧化还原反应中，氧化剂的氧化性大于氧化产物，还原剂的还原性大于还原产物，即“强强制弱弱”。
    - **优先规律**：当一种氧化剂与多种还原剂接触时，优先与还原性最强的还原剂反应（同理，一种还原剂优先与氧化性最强的氧化剂反应）。
    - **价态归中规律**：同种元素不同价态间若发生氧化还原反应，该元素的高价态降低、低价态升高，且“只靠拢，不交叉”，即高价态产物不低于低价态产物，产物价态通常居于反应物价态之间。
    - **歧化规律**：处于中间价态的同种元素，在适宜条件下可同时发生氧化和还原，即一部分原子化合价升高、一部分原子化合价降低，生成该元素的高价态与低价态产物。

- **关键方法**：遇到氧化还原问题，首先标出变价元素的化合价，根据电子守恒建立变化量关系。使用“强弱规律”判断反应能否自发进行；使用“优先规律”确定多个反应发生的先后次序，并画出相应的数量关系图；应用“归中规律”检验同种元素间反应产物的合理性；遇到中间价态物质的分解或反应，用“歧化规律”判断产物。配平复杂反应时，用电子守恒结合质量守恒与电荷守恒。

- **典型应用**：高考中常以选择和填空题考查。例如，向 FeBr₂ 溶液中通入 Cl₂，因还原性 Fe²⁺ > Br⁻，Cl₂ 优先氧化 Fe²⁺，待 Fe²⁺反应完后才氧化 Br⁻，据此可分段书写离子方程式和计算。又如，利用归中规律直接判定 H₂S 与浓 H₂SO₄ 反应只能生成 S 和 SO₂，而不能生成 H₂SO₄ 或 H₂S，避免产物推断错误。电子守恒常在陌生反应的计算中快速求算转移电子数或未知元素化合价。

## ⚠️ 高频易错点

- **误以为得电子越多氧化性越强**：氧化性强弱取决于得电子的难易程度，而非数目，如酸性 KMnO₄ 氧化性强于 I₂，但 I₂ 得电子数可更多。
- **归中规律应用忽略“不交叉”**：同种元素高价与低价反应时，认为可直接交换生成对方原始价态（例如 H₂S + H₂SO₄(浓) → H₂SO₄ + H₂S 为错误），产物必须向中间价态靠拢且不交叉。
- **电子守恒计算未分清“作酸”与“作氧化剂”**：例如 Fe 与 HNO₃ 反应，部分 NO₃⁻ 被还原，部分起酸的作用，计算转移电子时只能按被还原的部分 HNO₃ 来算。
- **忽略溶液中离子反应的竞争优先次序**：当多种还原剂共存时，仅凭浓度直觉判断反应先后，忽视以还原性强弱为先后依据，常将 Fe²⁺ 与 Br⁻ 的氧化次序颠倒。
