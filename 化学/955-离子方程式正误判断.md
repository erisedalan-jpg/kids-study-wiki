# 离子方程式正误判断

> **一句话**：离子方程式正误判断是依据客观事实、物质拆分规则、守恒定律及反应物用量关系，检验离子方程式书写是否规范的综合性考点。
> **English**: The judgment of ionic equation correctness is a comprehensive examination point that verifies whether an ionic equation is written correctly based on chemical facts, substance‑splitting rules, conservation laws, and the relative amounts of reactants.

## 🎓 考点精讲

- **定义/原理**：离子方程式是用实际参加反应的离子符号来表示化学反应的式子。其正误判断的核心原理是：反应必须真实发生（符合物质转化规律），方程式中强电解质需完全拆分为离子，弱电解质、难溶物、气体、单质、氧化物等保留化学式，同时必须满足**质量守恒**（原子种类与数目不变）和**电荷守恒**（离子方程式两边总电荷数相等），涉及氧化还原的还需满足**得失电子守恒**。

- **关键方法**：标准步骤为“**四看一查**”：
  1. **看客观事实**：反应能否发生，生成物是否正确（如铁与稀盐酸生成 $Fe^{2+}$ 而非 $Fe^{3+}$）。
  2. **看拆分规则**：强酸、强碱、可溶性盐拆成离子；弱酸、弱碱、水、难溶盐、气体等用化学式表示。特别注意微溶物的处理（如澄清石灰水拆，石灰乳不拆）。
  3. **看守恒**：检查质量、电荷和得失电子是否守恒。电荷不守恒是最常见的错误之一。
  4. **看量比与滴加顺序**：反应物相对量不同时离子方程式可能不同，如 $NaHCO_3$ 与 $Ca(OH)_2$ 的反应。
  5. **查符号与条件**：沉淀“↓”、气体“↑”、反应条件（如加热“$\triangle$”）是否正确。

- **典型应用**：高考中常以单项或组合选择形式出现，要求判断四个离子方程式的正误。典型设错场景：①拆错物质，如 $CaCO_3 + 2H^+ = Ca^{2+} + CO_2↑ + H_2O$ 为正确，误写成 $CO_3^{2-}$ 则错；②电荷不守恒，如 $Fe^{3+} + Cu = Fe^{2+} + Cu^{2+}$（电荷左边+3，右边+4，错误，应为 $2Fe^{3+} + Cu = 2Fe^{2+} + Cu^{2+}$）；③用量陷阱，如 $NaHSO_4$ 溶液与 $Ba(OH)_2$ 溶液反应至中性：$2H^+ + SO_4^{2-} + Ba^{2+} + 2OH^- = BaSO_4↓ + 2H_2O$。

## ⚠️ 高频易错点

- 误拆强弱电解质：如醋酸分子 $CH_3COOH$ 不能拆，一水合氨 $NH_3·H_2O$ 不能拆，但澄清石灰水中的 $Ca(OH)_2$ 应拆写为 $Ca^{2+}$ 和 $OH^-$。
- 电荷不守恒：书写时只顾原子配平而忽略离子电荷总数，尤其在氧化还原反应中电子转移失衡。
- 忽略“量”的影响：将少量 $NaOH$ 滴入 $Ca(HCO_3)_2$ 的离子方程式错写成 $Ca^{2+} + 2HCO_3^- + 2OH^- = CaCO_3↓ + CO_3^{2-} + 2H_2O$，而正确的少量方程式应优先中和 $HCO_3^-$ 且不残留 $CO_3^{2-}$（因 $Ca^{2+}$ 足量），错误多因照搬过量模式。
- 化学事实错误：臆造不能发生的反应，如将金属钠投入 $CuSO_4$ 溶液中误写为置换出铜，实际应为 $2Na + 2H_2O + Cu^{2+} = Cu(OH)_2↓ + 2Na^+ + H_2↑$。
