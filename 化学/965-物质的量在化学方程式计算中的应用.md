# 物质的量在化学方程式计算中的应用

> **一句话**：利用化学方程式中各物质的化学计量数之比等于物质的量之比，实现已知物与未知物之间定量换算的计算方法。
> **English**: The application of the principle that the ratio of stoichiometric coefficients in a balanced chemical equation equals the ratio of the amounts of substances (in moles) to perform quantitative calculations.

## 🎓 考点精讲

**定义/原理**：对于任一配平的化学方程式 $a\text{A} + b\text{B} \rightarrow c\text{C} + d\text{D}$，各物质的物质的量之比严格等于其化学计量数之比，即 $n(\text{A}) : n(\text{B}) : n(\text{C}) : n(\text{D}) = a : b : c : d$。这是整个计算的数学基础，核心在于将宏观质量、气体体积等物理量先转换为物质的量，再利用比例关系求解。

**关键方法**：**“列三行式”法**是高考解题的标准格式。
1. 写出配平的化学方程式。
2. 在方程式下方设未知数，依次列出各物质的**起始物质的量**、**转化物质的量**、**终态物质的量**（三行）。
3. 转化行严格按化学计量数之比填写（如 A 转化 $ax$ mol，则 B 转化 $bx$ mol）。
4. 利用终态条件（如溶液质量、气体体积、pH等）建立等式，解出未知数。
5. 回代求目标量。

**典型应用**：高考常以工业流程题、原理综合题中的**产率计算**、**多步反应关系式法**设问；选择题则常见**混合物反应**的计算（如 Na₂O₂ 与 Na₂O 的混合物与水反应）、**极值法/守恒法联用**。例：将 $1.0\text{ mol}$ 甲烷与氯气光照取代，得到四种氯代烃物质的量相等，则消耗 $\text{Cl}_2$ 物质的量为 $1.0 \times \frac{0+1+2+3+4}{4} = 2.5\text{ mol}$（待家长核对），此处直接用 C 原子守恒与 Cl 原子守恒联立，绕开逐一代入。

## ⚠️ 高频易错点

- **直接用质量或体积列比例**：未将质量、气体体积（非标况）统一换算为物质的量就代入化学计量数之比，导致比例式错误。
- **忽略过量判断**：在两种反应物均已知量的计算中，未先进行过量判断，错将较少者完全转化后仍用较多者的初始量计算。
- **固液反应中气体体积换算条件**：使用 $V = n \times 22.4\text{ L·mol}^{-1}$ 时未确认是否处于标准状况（0°C、101 kPa），常出现在电解、金属与酸反应等涉及气体产物的场景。
- **多步反应中损失率位置颠倒**：计算产率或原料利用率时，将损失率乘在产物一侧而非原料一侧，或连续多步反应中误将各步产率相加而非连乘。
