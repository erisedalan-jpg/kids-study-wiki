# 电极反应式书写

> **一句话**：电极反应式是描述原电池或电解池中单一电极上发生的氧化反应或还原反应的离子方程式，必须同时满足原子守恒、电荷守恒，且在质子溶剂中满足H/O配平规则。
> **English**: An electrode half-reaction is an ionic equation describing the oxidation or reduction process occurring at a single electrode in a galvanic or electrolytic cell, which must satisfy atom balance, charge balance, and, in protic media, the H/O balance rule.

## 🎓 考点精讲

### 定义与原理

电极反应式书写基于氧化还原反应的基本原理：氧化反应发生于**阳极**（原电池负极，电解池阳极），失电子，化合价升高；还原反应发生于**阴极**（原电池正极，电解池阴极），得电子，化合价降低。书写时必须严格遵循“三步守恒”原则：

1. **原子守恒**：反应前后各元素原子数目相等。
2. **电荷守恒**：反应前后总电荷数相等，通过补充电子（`e⁻`）实现。
3. **H/O配平规则**（适用于酸性或碱性水溶液体系）：
   *   **酸性介质**：用 `H⁺` 和 `H₂O` 配平H、O原子。多O侧加 `H⁺`，少O侧加 `H₂O`。
   *   **碱性介质**：用 `OH⁻` 和 `H₂O` 配平H、O原子。多O侧加 `H₂O`，少O侧加 `OH⁻`。
   *   **熔融盐介质**（如熔融碳酸盐、金属氧化物）：直接用介质阴离子（如 `CO₃²⁻`、`O²⁻`）配平O原子，不引入 `H⁺` 或 `OH⁻`，这是高考的高频设误点。

### 关键方法：标准书写步骤

1. **判断反应类型**：根据装置类型（原电池/电解池）和电极名称（正/负/阴/阳极）判断目标电极上发生的是氧化反应还是失电子反应。
2. **写出核心变化**：列出反应物和主要产物，按化合价变化标出得失电子数。例如在氢氧燃料电池（酸性）中，负极核心变化为 `H₂ → 2H⁺`。
3. **依据介质配平**：根据电解质环境的酸碱性，使用 `H⁺/OH⁻/H₂O` 或特定熔盐离子配平电荷和原子。在碱性氢氧燃料电池中，负极 `H₂` 失电子后生成的 `H⁺` 不能稳定存在，会立即与 `OH⁻` 结合，因此电极反应式写作 `H₂ - 2e⁻ + 2OH⁻ = 2H₂O`。

### 典型高考应用

*   **题型一：给定总反应与介质，书写电极反应式**。如给出铅蓄电池放电总反应 `Pb + PbO₂ + 2H₂SO₄ = 2PbSO₄ + 2H₂O`，书写负极反应式。解题关键是先写出负极失电子的简单式 `Pb - 2e⁻ → PbSO₄`，再根据总反应配平原子，利用电解质中的 `SO₄²⁻` 配平Pb原子，得 `Pb - 2e⁻ + SO₄²⁻ = PbSO₄`。
*   **题型二：给装置图（燃料电池/二次电池），直接书写某极反应式**。如**碱性锌-空气电池**，负极反应式为 `Zn - 2e⁻ + 2OH⁻ = Zn(OH)₂`（或 `ZnO + H₂O`，视产物而定），正极反应式往往是氧气在碱性介质中的还原 `O₂ + 4e⁻ + 2H₂O = 4OH⁻`。易错点在于氧气还原产物的判断——酸性介质生 `H₂O`，碱性介质生 `OH⁻`。

## ⚠️ 高频易错点

*   **混淆介质类型导致H/O配平错误**：在熔融碳酸盐介质（如 `Li₂CO₃-K₂CO₃`）中，用 `CO₂` 和 `CO₃²⁻` 配平C、O原子，而非用 `H⁺` 或 `OH⁻`。例如，熔融碳酸盐燃料电池正极：`O₂ + 4e⁻ + 2CO₂ = 2CO₃²⁻`，学生常误写成酸性或碱性环境下的反应。
*   **原电池与电解池电极极性反转**：原电池负极发生氧化反应（失电子），电解池阳极发生氧化反应（失电子）。学生常将电解池的阴极（得电子）误作为负极书写电极反应式。
*   **可充电电池“充电/放电”状态混淆**：书写充电方程式时，电极的阴阳极角色反转。若题目要求写“充电时阳极反应式”，实际上对应原电池放电时正极反应的逆反应，务必核对箭头方向和电子得失的正负。
*   **反应产物判断不结合电解质环境**：如燃料电池中，烃类或醇类在酸/碱介质中最终产物均为 `CO₂`（酸性）或 `CO₃²⁻`（碱性），不能直接只写生成 `CO₂`，需注意在碱性环境下 `CO₂` 会转化为 `CO₃²⁻`。
