# 化学变化与物理变化

> **一句话**：化学变化是有新物质生成的变化，本质是旧化学键断裂和新化学键形成；物理变化是没有新物质生成、仅改变形态或状态的变化。
> **English**: Chemical changes produce new substances through breaking and forming chemical bonds, while physical changes only alter the form or state without creating new substances.

## 🎓 考点精讲

- **定义/原理**：化学变化的微观本质是反应物分子中化学键断裂，原子重新组合形成生成物分子，宏观表现为有新物质生成。物理变化（如熔化、蒸发、粉碎）不涉及化学键的重组。判断依据核心：**宏观—是否产生新物质；微观—是否有旧化学键断裂和新化学键形成**。  
  例如：水通电分解 `2H₂O ⟶(通电) 2H₂↑ + O₂↑`，H–O键断裂，H–H、O＝O键形成，属于化学变化。液态水蒸发 `H₂O(l) → H₂O(g)` 只改变分子间距离，无新物质生成，属于物理变化。注意：仅有旧键断裂而无新键形成不一定属于化学变化（如NaCl溶于水离子键断裂，但未形成新化学键，是物理变化）。

- **关键方法**：三步判据法：① 看反应前后物质的种类是否改变；② 微观上分析是否存在原子重组；③ 排除干扰现象——发光、放热、变色、产生气体或沉淀等**不能作为绝对判据**。  
  在高考选择题或填空题中，面对生活、工业过程，应联想该过程最终是否产生了新分子或新物质，并结合经典实例对比（如石油分馏为物理变化，煤的干馏为化学变化）。

- **典型应用**：高考常见设问角度：  
  - 辨析一组变化：“①轮胎爆炸 ②粮食酿酒 ③玻璃刻蚀”等。①为气体膨胀的物理变化；②淀粉→酒精，生成新物质，化学变化；③HF与SiO₂反应，化学变化。  
  - 传统文化情境：“丹砂（HgS）烧之成水银，积变又还成丹砂”——`HgS ⟶ Hg + S`，`Hg + S → HgS`，均为化学变化。  
  - 微观示意图题：给出反应前后分子模型，要求判断变化类型并书写方程式。解题关键是识别是否产生了新分子。  
  - 易混淆点：核裂变、核聚变改变了原子核，不属于化学变化范畴，高考常以此作为干扰项。

## ⚠️ 高频易错点

- 错把伴随现象当本质：认为发光、放热、产生气体或沉淀就一定是化学变化。实际上灯泡发光、水沸腾产生蒸汽泡都是物理变化，必须紧扣是否生成新物质。
- 认为“化学键断裂”就等于化学变化：如NaOH溶于水电离出Na⁺、OH⁻，离子键断裂但无新键形成，属于物理变化；而Na与水反应既有旧键断裂又有新键形成，才是化学变化。
- 同素异形体转化误判为物理变化：石墨变为金刚石、红磷变为白磷、O₂变为O₃，均生成了不同单质，属于化学变化，因为生成了新物质。
