# 化学方程式的书写

> **一句话**：化学方程式是用化学式来表示化学反应的式子，其书写必须以客观事实为基础，并严格遵守质量守恒定律。
> **English**: A chemical equation is a symbolic representation of a chemical reaction using chemical formulas, where the number of atoms for each element must be conserved.

## 🎓 考点精讲

化学方程式的书写是高考化学的基础与核心考点，要求能够准确无误地书写教材内常见反应及陌生情境下的新反应方程式。

- **定义与原理**  
  化学方程式由反应物与生成物的化学式、反应条件、以及必要的状态符号（如沉淀“↓”、气体“↑”）组成。其核心原理是**质量守恒定律**——反应前后原子的种类和数目保持不变，因此方程式必须配平。书写必须遵循两条原则：①以客观事实为依据，不能随意臆造不存在的反应或产物；②遵守质量守恒定律。

- **书写方法与关键步骤**  
  标准步骤可概括为“写、配、注、等”：  
  1. **写**：根据反应事实，在左右两侧写出反应物和生成物的正确化学式，中间用“—”连接。  
  2. **配**：选取适当的方法（如观察法、最小公倍数法、氧化还原守恒法）使等号两边各原子的种类与数目相等，将“—”改为“===”。  
  3. **注**：在等号上方或下方标明必要的反应条件（如点燃、加热“△”、催化剂、通电等）；若生成物中有沉淀或气体，在相应化学式右侧标注“↓”或“↑”，但需注意若反应物中已有难溶物或气体，生成物中同类物质不标注。  
  4. **等**：最终将短线改为等号。

- **典型应用与高考考查方式**  
  高考常通过以下形式考查：  
  - **正误判断型**：给出几个方程式，让考生判断是否符合事实、是否配平、条件或箭头是否正确。例如判断“Fe + 2HCl = FeCl₃ + H₂↑”是否合理（铁与盐酸反应应生成FeCl₂）。  
  - **陌生物质转化书写**：结合工艺流程、实验探究或氧化还原信息，书写未学过的反应方程式。如在酸性条件下用H₂O₂处理含Fe²⁺废水：  $$2\text{Fe}^{2+} + \text{H}_2\text{O}_2 + 2\text{H}^+ = 2\text{Fe}^{3+} + 2\text{H}_2\text{O}$$
  - **综合填空**：在非选择题中要求补充完整的化学方程式，常涉及热化学方程式、电极反应式、离子方程式等进阶书写。

## ⚠️ 高频易错点

- **化学式书写错误**：混淆价态导致化学式不正确，如将氯化铁写成“FeCl₂”或硫酸铝写成“AlSO₄”。  
- **配平错误**：只关注金属原子而忽略氢、氧原子数目，特别是在复杂氧化还原反应中未用电子守恒法导致配平错误。  
- **反应条件与箭头漏标或错标**：忘记标注加热“△”、催化剂等条件；在不同类别反应中错标“↑”或“↓”，例如反应物中有气体时生成物气体仍标“↑”。  
- **违背反应事实**：主观臆造不存在的反应，如写出“Cu + 2HCl = CuCl₂ + H₂↑”（铜不与盐酸反应），或铁与盐酸生成+3价铁离子。
