# 氮氧化物

> **一句话**：氮氧化物是氮元素与氧元素形成的多种化合物总称，高考中重点考查 NO 和 NO₂ 的性质、转化、对环境的污染及定量计算。
> **English**: Nitrogen oxides are compounds of nitrogen and oxygen, with NO and NO₂ being key focuses in Gaokao for their properties, interconversion, environmental impact, and quantitative calculations.

## 🎓 考点精讲

**定义/原理**  
常见氮氧化物包括 NO、NO₂、N₂O₃、N₂O₄、N₂O₅ 等。高考核心涉及 NO 和 NO₂：
- **NO**：无色有毒气体，难溶于水，不与水反应，极易与氧气反应：`2NO + O₂ = 2NO₂`
- **NO₂**：红棕色有毒气体，易溶于水并发生歧化反应：`3NO₂ + H₂O = 2HNO₃ + NO`，同时存在可逆平衡：`2NO₂ ⇌ N₂O₄`
- **混合体系计算原理**：当 NO₂ 与 O₂ 混合通入水中时，总反应为 `4NO₂ + O₂ + 2H₂O = 4HNO₃`；当 NO 与 O₂ 混合通入水中时，总反应为 `4NO + 3O₂ + 2H₂O = 4HNO₃`。解题时通常需利用方程式叠加消除中间产物 NO。

**关键方法**  
1. **“循环法”处理氮氧化物溶于水的计算**：将多次反应叠加为总反应，根据气体前后体积差或剩余气体体积比判断最终产物和过量气体。  
2. **尾气处理原理**：工业上常用 NaOH 溶液吸收：`2NO₂ + 2NaOH = NaNO₃ + NaNO₂ + H₂O`，`NO₂ + NO + 2NaOH = 2NaNO₂ + H₂O`。当 n(NO₂) ≥ n(NO) 时可被完全吸收。  
3. **鉴别与检验**：NO 遇空气迅速变红棕色，可用于鉴别 NO 和 NO₂；NO₂ 使湿润的淀粉-KI 试纸变蓝。

**典型应用**  
- **试管倒扣于水中的体积变化**：如盛满 NO₂ 的试管倒扣水中，剩余体积为原来的 1/3（因生成 NO）。  
- **氮氧化物造成的光化学烟雾与酸雨**：NO₂ 在紫外线下分解产生氧原子，引发光化学烟雾；NOₓ 经氧化和吸收形成硝酸型酸雨。  
- **废气转化**：汽车尾气系统通过催化转化器将 NOₓ 还原为 N₂：`2NO + 2CO → N₂ + 2CO₂`（催化剂、加热）。

## ⚠️ 高频易错点

- 误认为 NO₂ 与水反应完全转化为硝酸，忽略会生成 NO，导致试管内气体不会完全消失。  
- 混淆 NO 与 NO₂ 的收集方法：NO 只能用排水法收集，不可用排空气法（易被氧化）；NO₂ 只能用向上排空气法收集，不可用排水法。  
- 在氮氧化物混合气体与 NaOH 溶液反应时，忽略 n(NO₂)/n(NO) 比例对吸收完全程度的影响，误以为 NO 可被 NaOH 直接吸收。  
- 将 NO₂ 与 N₂O₄ 的平衡问题混淆于纯净物计算，忽视一定条件下 NO₂ 中存在部分 N₂O₄ 对气体体积、相对分子质量的影响。
