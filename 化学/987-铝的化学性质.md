# 铝的化学性质

> **一句话**：铝是典型的两性金属，既能与强酸反应放出氢气，也能与强碱溶液反应生成偏铝酸盐和氢气，并在常温下因表面致密氧化膜而呈现钝化特性。
> **English**: Aluminum is a typical amphoteric metal that reacts with both strong acids and strong bases to release hydrogen gas, and it exhibits passivation at room temperature due to a dense surface oxide layer.

## 🎓 考点精讲

**定义/原理**：铝位于元素周期表第三周期第ⅢA族，原子最外层有3个电子，在化学反应中易失去最外层电子而显+3价，表现为较强的还原性。其化学性质的核心可概括为“亲氧性”与“两性”：

1. **与非金属单质反应**（亲氧性）：铝粉在氧气中剧烈燃烧生成氧化铝，化学方程式为 $4\mathrm{Al} + 3\mathrm{O}_2 \xrightarrow{\text{点燃}} 2\mathrm{Al}_2\mathrm{O}_3$。铝也能与氯气、硫等直接化合。

2. **与酸反应**（金属通性）：与非氧化性酸（如盐酸、稀硫酸）反应置换出氢气，离子方程式为 $2\mathrm{Al} + 6\mathrm{H}^+ = 2\mathrm{Al}^{3+} + 3\mathrm{H}_2\uparrow$。**注意**：常温下铝在浓硫酸、浓硝酸中发生**钝化**——表面迅速生成致密氧化膜阻止反应继续，此属于化学变化，常作为物质运输容器的命题切入点。

3. **与强碱反应**（两性特性）：铝是中学阶段唯一能与强碱溶液反应放出氢气的常见金属。其反应分两步理解：先与水反应生成 $\mathrm{Al(OH)}_3$ 和 $\mathrm{H}_2$，随后 $\mathrm{Al(OH)}_3$ 溶于过量强碱。总反应的离子方程式为 $2\mathrm{Al} + 2\mathrm{OH}^- + 6\mathrm{H}_2\mathrm{O} = 2[\mathrm{Al(OH)}_4]^- + 3\mathrm{H}_2\uparrow$。写化学方程式时，偏铝酸盐也可写作 $\mathrm{NaAlO}_2$ 形式：$2\mathrm{Al} + 2\mathrm{NaOH} + 2\mathrm{H}_2\mathrm{O} = 2\mathrm{NaAlO}_2 + 3\mathrm{H}_2\uparrow$（待家长核对课标对四羟基合铝酸盐与偏铝酸盐的书写要求）。

4. **铝热反应**：铝与某些金属氧化物（如 $\mathrm{Fe}_2\mathrm{O}_3$）在高温下发生置换反应，方程式为 $2\mathrm{Al} + \mathrm{Fe}_2\mathrm{O}_3 \xrightarrow{\text{高温}} 2\mathrm{Fe} + \mathrm{Al}_2\mathrm{O}_3$，用于焊接钢轨、冶炼高熔点金属。该反应体现了铝的强还原性及 $\mathrm{Al}_2\mathrm{O}_3$ 的高生成焓。

**关键方法**：解决铝的化学性质类问题的核心思路是“先辨环境，后判产物”。第一步判断反应环境是酸性、碱性还是氧化性酸环境；第二步根据环境确定铝的氧化产物形态（$\mathrm{Al}^{3+}$ 或 $[\mathrm{Al(OH)}_4]^- / \mathrm{AlO}_2^-$）；第三步关注反应是否涉及钝化条件（常温、浓硫酸/浓硝酸）。

**典型应用**：高考中常见设问有：(1) 离子方程式正误判断，尤其针对铝与强碱反应时水是否参与配平；(2) 以铝为媒介的物质转化推断题，利用 $ \mathrm{Al} \rightarrow \mathrm{Al}^{3+} \rightarrow \mathrm{Al(OH)}_3 \rightarrow \mathrm{AlO}_2^- $ 的三角转化关系；(3) 实验题中铝与酸、碱反应产生氢气量的定量计算与图像分析。

## ⚠️ 高频易错点

- **钝化误判为物理变化**：铝遇冷的浓硫酸、浓硝酸后钝化是化学变化，生成了新的氧化膜（$\mathrm{Al}_2\mathrm{O}_3$），而非简单隔离。钝化后的铝不再与稀酸反应，除非打磨去除氧化膜。
- **铝与碱反应的电子转移搞错还原剂**：反应 $2\mathrm{Al} + 2\mathrm{NaOH} + 2\mathrm{H}_2\mathrm{O} = 2\mathrm{NaAlO}_2 + 3\mathrm{H}_2\uparrow$ 中，$\mathrm{H}_2$ 并非来自 $\mathrm{NaOH}$ 中的氢，而是水的电离 $\mathrm{H}^+$ 得电子被还原，铝是唯一还原剂。
- **盲目套用金属活动性顺序**：铝在金属活动性顺序中位于氢之前，能与酸反应是通性，但铝与碱的反应不是金属通性，不能照搬铁、锌等金属的性质。
- **铝热反应条件误写为“点燃”**：铝热反应需要高温引发，常用镁条引燃，不能写“点燃”或“加热”，应写“高温”。
