# 化学平衡移动原理

> **一句话**：如果改变影响化学平衡的一个条件（浓度、压强或温度），平衡就向能够**减弱**这种改变的方向移动。
> **English**: If a dynamic equilibrium is disturbed by changing the conditions, the position of equilibrium moves to counteract the change.

## 🎓 考点精讲

### 定义/原理
**勒夏特列原理（Le Chatelier's principle）**，又称平衡移动原理，适用于一切动态平衡体系。其核心表述为：改变影响平衡的一个条件，平衡将向减弱该改变的方向移动。

**定量基础：浓度商 Q 与平衡常数 K 的关系**
- 若 $Q > K$，反应逆向进行（平衡向左移动）；
- 若 $Q < K$，反应正向进行（平衡向右移动）；
- 若 $Q = K$，反应处于平衡状态。

**影响因素与移动规律（对于有气体参与的反应 $aA(g) + bB(g) \rightleftharpoons cC(g) + dD(g)$，$\Delta H$ 给定）：**

1. **浓度**：增大反应物浓度（或减小生成物浓度），$Q$ 减小，平衡正向移动；反之则逆向移动。 *注意：固体或纯液体的量改变不影响平衡，因其浓度视为常数。*

2. **压强（针对气体反应）**：增大压强，平衡向气体分子数**减小**（即 $c+d < a+b$）的方向移动；减小压强，平衡向气体分子数**增大**的方向移动。若反应前后气体分子数不变（$a+b = c+d$），改变压强平衡不移动。

3. **温度**：升高温度，平衡向**吸热**反应方向（$\Delta H > 0$）移动；降低温度，平衡向**放热**反应方向（$\Delta H < 0$）移动。*温度是唯一同时改变 K 值的因素：升温使吸热反应的 K 增大，放热反应的 K 减小。*

4. **催化剂**：同等程度改变正、逆反应速率，只能缩短达到平衡的时间，**不能**使平衡发生移动。

### 关键方法
**三步法解决平衡移动图像题（"先拐先平，定一议二"）：**
- **"先拐先平"**：在速率—时间图或转化率—时间图中，先出现拐点的曲线对应更高的温度或更大的压强（反应速率更快，先达到平衡）。
- **"定一议二"**：在转化率—温度—压强三维关系图中，固定一个变量（如固定压强作一条垂线），再分析另一个变量对转化率的影响，从而推断反应特征（放热/吸热，分子数变化）。

**等效平衡的简化处理（适用于恒温恒容/恒温恒压条件）**：将投料方式按化学计量数比完全转化为同一侧（通常转为反应物侧）进行比较，若各物质的量之比与原平衡起始量之比相同，则为等效平衡，可简化状态判断。

### 典型应用
- **高考常见题型**：选择题中判断条件改变后平衡移动方向、转化率变化、浓度变化；填空题中书写化学平衡常数的表达式，并结合温度变化讨论 K 值的变化；图像综合题中依据图像信息逆推反应的热效应（吸/放热）和气体分子数变化。
- **工业合成氨条件选择**：反应 $N_2 + 3H_2 \rightleftharpoons 2NH_3，\Delta H < 0$。正向放热、气体分子数减小。理论上低温高压利于氨的产率，但实际生产中需兼顾速率和催化剂活性，选用适当高温（约 500℃）、高压和铁触媒，并及时分离 $NH_3$ 使平衡持续正向移动。

## ⚠️ 高频易错点

- **混淆"减弱"与"抵消"**：勒夏特列原理的核心是"减弱"，而非"完全抵消"。例如，升高温度后平衡向吸热方向移动仅能吸收部分热量，体系最终温度依然比原来高。
- **错误认为压强或惰性气体总能使平衡移动**：恒容条件下充入惰性气体，反应物和生成物分压均不变，平衡不移动；恒压条件下充入惰性气体导致体积膨胀，各组分分压减小，等效于减压，平衡向气体分子数增大的方向移动。
- **将浓度改变与压强改变割裂**：二者统一于浓度商 Q 的变化分析。例如，压缩容器体积本质是同时按比例增大所有气体组分的浓度，最终移动方向由化学计量数决定。
- **忽视温度对平衡常数的唯一影响**：K 随温度变化；加入催化剂或改变浓度、压强不会改变 K 值，只能改变 Q 与 K 的相对大小来影响平衡。
