# SO2性质

> **一句话**：SO₂是硫的+4价氧化物，分子呈V形，属于酸性氧化物，同时具有氧化性、还原性和漂白性，核心化学性质由中间价态硫决定。
> **English**: SO₂ is the +4 oxide of sulfur with a V-shaped molecular geometry; it is an acidic oxide that exhibits oxidizing, reducing, and bleaching properties, all governed by the intermediate oxidation state of sulfur.

## 🎓 考点精讲

**定义/原理**
SO₂（二氧化硫）中硫显+4价，处于中间价态（硫常见价态：-2、0、+4、+6），因此既可被氧化到+6价显还原性，又可被还原到0价显氧化性，但**以还原性为主**。其水溶液为亚硫酸（H₂SO₃，二元中强酸），能使紫色石蕊试液变红但不褪色。分子为V形结构，极性分子，易液化（沸点-10℃）。

**关键反应与判别依据**

1. **酸性氧化物通性**
   - 与水：`SO₂ + H₂O ⇌ H₂SO₃`（可逆）
   - 与碱：`SO₂(少量) + 2NaOH = Na₂SO₃ + H₂O`；`SO₂(过量) + NaOH = NaHSO₃`
   - 与碱性氧化物：`SO₂ + CaO = CaSO₃`

2. **还原性（核心）**
   - 催化氧化：`2SO₂ + O₂ ⇌ 2SO₃`（催化剂V₂O₅，加热，接触法制硫酸关键步骤）
   - 使卤素水褪色：`SO₂ + X₂ + 2H₂O = H₂SO₄ + 2HX`（X=Cl、Br、I）
   - 使酸性KMnO₄溶液褪色：`5SO₂ + 2MnO₄⁻ + 2H₂O = 5SO₄²⁻ + 2Mn²⁺ + 4H⁺`
   - 使Fe³⁺转化为Fe²⁺：`SO₂ + 2Fe³⁺ + 2H₂O = 2Fe²⁺ + SO₄²⁻ + 4H⁺`

3. **氧化性**
   - 与H₂S反应：`SO₂ + 2H₂S = 3S↓ + 2H₂O`（产生黄色硫沉淀，S归中反应）

4. **漂白性**
   - 与品红溶液：化合生成不稳定无色物质，加热后恢复红色——此为SO₂的特征性质，用于**鉴别SO₂与CO₂**。

**典型应用（高考设问）**
- **综合实验题**：以SO₂的制备（Na₂SO₃ + H₂SO₄ →）、除杂、性质检验（酸性、氧化性、还原性、漂白性）、尾气处理（NaOH溶液）为主线，串联多组实验装置，要求分析装置作用、书写方程式、描述现象。
- **离子方程式正误判断**：SO₂与多种氧化剂（Cl₂、Fe³⁺、MnO₄⁻、HNO₃）反应的离子方程式书写是高频考查点，注意电荷守恒、介质（酸性）参与。
- **化学与生活**：SO₂作为食品添加剂（葡萄酒中抗氧化剂）、酸雨成因等。

**解题关键步骤**
1. 紧抓价态：+4价硫遇到强氧化剂只表现还原性，遇到强还原剂（如H₂S）才表现氧化性。
2. 漂白性判定：SO₂→品红褪色（加热恢复）= 化合漂白；SO₂→KMnO₄褪色 = 还原性，非漂白性。
3. 连续反应产物判断：碱不足→酸式盐，碱过量→正盐。

## ⚠️ 高频易错点

- **混淆SO₂与CO₂性质**：SO₂也能使澄清石灰水变浑浊（生成CaSO₃↓），但SO₂有刺激性气味、能使品红褪色、能使酸性KMnO₄褪色，CO₂则不能——鉴别二者不能用石灰水，应用品红或KMnO₄。
- **误判漂白性**：SO₂使溴水、酸性KMnO₄溶液褪色体现的是**还原性**，不是漂白性；只有使品红等有机色质褪色且加热恢复才是漂白性（化合漂白）。
- **误认为SO₂能用浓硫酸干燥**：SO₂中+4价硫可被浓硫酸中+6价硫氧化为+6价（`SO₂ + H₂SO₄(浓) = 不反应？`），实际上**室温下SO₂与浓硫酸不反应，可用浓硫酸干燥**（待家长核对：浓硫酸+4价不被氧化为+6价的反应条件需高温催化剂）。此处易混淆：SO₂用浓硫酸干燥是可行的，但原因不是不反应，而是常温下反应极慢。考试中不可类比为“能用浓硫酸干燥”。
- **忽略了SO₂与H₂S反应的定量关系**：`SO₂ + 2H₂S = 3S↓ + 2H₂O`中，氧化产物与还原产物均为S，物质的量之比为2:1。
