# sp杂化

> **一句话**：1个s轨道与1个p轨道杂化，形成2个sp杂化轨道，呈180°直线排列，剩余2个p轨道可形成2个π键。
> **English**: One s orbital mixes with one p orbital to form two sp hybrid orbitals arranged linearly at 180°, with two remaining p orbitals available for π bonding.

---

## 🎓 高中进阶

### 定义
**sp 杂化**：中心原子的 1 个 s 轨道与 1 个 p 轨道杂化，形成 **2 个 sp 杂化轨道**：
- 两个 sp 杂化轨道之间夹角 **180°**，呈直线排列
- 剩余 **2 个 p 轨道**（未参与杂化）可与相邻原子形成 π 键

### sp 杂化的特征

| 特征 | 说明 |
|------|------|
| 参与轨道 | 1s + 1p |
| 杂化轨道数 | 2 个 |
| 键角 | 180° |
| 分子形状 | 直线形 |
| 剩余 p 轨道 | 2 个（形成 2 个 π 键）|

### 典型 sp 杂化分子

| 分子 | 中心原子 | σ/π | 形状 |
|------|---------|-----|------|
| BeCl₂ | Be（sp）| 2σ | 直线形 |
| CO₂ | C（sp）| 2σ+2π | 直线形 |
| C₂H₂（乙炔）| C（sp）| 各3σ+2π | 直线形 |
| HCN | C（sp）| 2σ+2π | 直线形 |
| CS₂ | C（sp）| 2σ+2π | 直线形 |

### CO₂ 的 sp 杂化分析
- C 的 sp 杂化：形成 2 个 sp 杂化轨道（各指向一个 O）
- 2 个 sp 轨道与 O 的 p 轨道形成 2 个 C=O 的 σ 键
- C 剩余 2 个 p 轨道（px、py）各与对应 O 的 p 轨道侧面重叠，形成 2 个 π 键
- 结果：O=C=O，直线形，键角 180°

### 乙炔 C₂H₂ 的 sp 杂化分析
- 每个 C：sp 杂化 → 2 个杂化轨道（1 个与 H 成 σ 键，1 个与另一 C 成 σ 键）
- 每个 C 剩余 2 个 p 轨道 → 两个 C 间 2 对 p 轨道侧面重叠 → 2 个 π 键
- 结果：H−C≡C−H，直线形，键角 180°

### 典型例题
**题**：BeCl₂ 是 sp 杂化，为什么 BeCl₂ 键角是 180°？
**解**：Be 的 sp 杂化轨道有 2 个，这 2 个轨道的夹角为 180°（直线型排列能量最低），因此 2 个 Be−Cl 键角为 180°，分子呈直线形。

### 易错点
- sp 杂化有 2 个 π 键（2 对剩余 p 轨道各形成 1 个 π 键）
- 直线形分子可能是 sp 杂化（如 CO₂），也可能是无孤对的 AX₂（VSEPR 视角）
- CO₂ 是直线形，但 SO₂（有孤对）是 V 形
