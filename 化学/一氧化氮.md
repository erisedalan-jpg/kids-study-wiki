# 一氧化氮

> **一句话**：无色有毒气体，N化合价+2，遇O₂迅速变为红棕色NO₂，是合成硝酸的中间体，也有重要生理功能。
> **English**: A colorless toxic gas with N in +2 oxidation state; rapidly turns reddish-brown NO₂ upon exposure to O₂; an intermediate in nitric acid synthesis with important physiological functions.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | NO |
| N化合价 | +2 |
| 颜色气味 | **无色**，有毒 |
| 溶解性 | 微溶于水（不与水直接反应）|
| 化学性质 | 不稳定，与O₂迅速反应 |

### 化学性质

**1. 与O₂反应**（重要）：
$$2\text{NO} + \text{O}_2 \rightarrow 2\text{NO}_2\quad\text{（无色→红棕色）}$$

NO遇空气（O₂）**立即变为红棕色**NO₂，这是NO的特征反应，也是判断NO存在的依据。

**2. NO₂与水反应**（合成硝酸，后续）：
$$3\text{NO}_2 + \text{H}_2\text{O} \rightarrow 2\text{HNO}_3 + \text{NO}$$

（NO₂被水吸收生成HNO₃和NO，NO可循环使用）

**3. NO的来源**：
- 自然界雷电：N₂ + O₂ → 2NO
- 汽车尾气：高温引擎中N₂+O₂→NO（大气污染）
- 稀HNO₃与金属反应：3Cu + 8HNO₃（稀）→ 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

### 工业合成硝酸的中间体

**氨的催化氧化**（工业制硝酸第一步）：
$$4\text{NH}_3 + 5\text{O}_2 \underset{800-900°\text{C}}{\overset{\text{Pt}}{\rightarrow}} 4\text{NO} + 6\text{H}_2\text{O}$$

**NO继续氧化**（第二步）：
$$2\text{NO} + \text{O}_2 \rightarrow 2\text{NO}_2$$

**NO₂被水吸收**（第三步）：
$$3\text{NO}_2 + \text{H}_2\text{O} \rightarrow 2\text{HNO}_3 + \text{NO}$$

（NO循环回去，继续氧化为NO₂）

### NO的生理作用

- NO是重要的**信号分子**，控制血管舒张（1998年诺贝尔医学奖）
- 在免疫系统中有杀菌功能
- 硝酸甘油治疗心绞痛：在体内转化为NO，扩张血管

### 典型例题

**例1**：铜和稀硝酸反应，收集到无色气体，将气体暴露空气中变红棕色，证明气体是什么？

**分析**：无色气体遇空气变红棕色，是NO（2NO + O₂ → 2NO₂）。铜与稀HNO₃反应产生**NO**（不是H₂）：
3Cu + 8HNO₃（稀）→ 3Cu(NO₃)₂ + 2NO↑ + 4H₂O

**例2**：工业合成硝酸中，NO不能用排水法收集，而要用向上排气法，为什么？

**分析**：NO微溶于水，排水法可以，但NO遇空气的O₂迅速变为NO₂（红棕色），所以收集NO时要排除空气干扰，通常用**向下排气法**（NO密度与空气接近，实际用排水法较合适）。高中一般说NO用**排水法**收集（防止与空气中O₂反应）。

### 易错点

- NO是**无色**气体（NO₂才是**红棕色**）
- NO不溶于水（不与水反应），NO₂能与水反应
- NO是不成盐氧化物（不与水、酸、碱直接反应生成盐和水）
- NO在生理上有重要作用（不要认为所有有毒物质都无益）
