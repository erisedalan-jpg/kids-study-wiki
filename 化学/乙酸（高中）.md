# 乙酸（高中）

> **一句话**：乙酸（CH₃COOH）是食醋的主要酸性成分，弱酸性，能与醇发生酯化、与碱中和、与 NaHCO₃ 放 CO₂，是有机酸的典型代表。
> **English**: Acetic acid (CH₃COOH) is the main acidic component of vinegar; it is a weak acid that undergoes esterification with alcohols, neutralization with bases, and releases CO₂ with NaHCO₃; it is the archetypal organic acid.

---

## 🎓 高中进阶

### 定义 / 结构特征

**乙酸**（acetic acid）：
- 分子式 C₂H₄O₂，结构简式 CH₃COOH（或 CH₃CO₂H）
- 含**羧基**（—COOH）
- 无色液体，有强烈醋味，沸点 117.9°C，熔点 16.6°C（纯乙酸在冬天为冰醋酸）
- 与水完全互溶（氢键），密度 1.049 g/mL
- pKa = 4.76（弱酸，部分电离：CH₃COOH ⇌ CH₃COO⁻ + H⁺）

**制备**：
- 工业：乙醛氧化：2CH₃CHO + O₂ →(Mn²⁺) 2CH₃COOH
- 工业：甲醇羰基化（Monsanto/Cativa 法）：CH₃OH + CO →(Rh 催化) CH₃COOH
- 发酵：乙醇 + O₂ →(醋酸菌) 乙酸

### 性质 / 反应

| 反应 | 方程式 | 条件 |
|------|--------|------|
| 与 Na | 2CH₃COOH + 2Na → 2CH₃COONa + H₂↑ | 常温 |
| 与 NaOH | CH₃COOH + NaOH → CH₃COONa + H₂O | 常温 |
| 与 Na₂CO₃ | 2CH₃COOH + Na₂CO₃ → 2CH₃COONa + H₂O + CO₂↑ | 常温 |
| 与 NaHCO₃ | CH₃COOH + NaHCO₃ → CH₃COONa + H₂O + CO₂↑ | 常温 |
| 与乙醇酯化 | CH₃COOH + C₂H₅OH ⇌ CH₃COOC₂H₅ + H₂O | 浓H₂SO₄，△ |
| 燃烧 | CH₃COOH + 2O₂ → 2CO₂ + 2H₂O | 点燃 |

**乙酸酯化的断键**（同位素实验）：
CH₃CO—**OH** + **H**—OC₂H₅ → CH₃COOC₂H₅ + **H₂O**（乙酸脱—OH，乙醇脱 H）
→ 产物 H₂O 中 O 来自乙酸，乙醇的 C—O 键保留

### 典型例题

**例**：乙酸与碳酸、苯酚的酸性比较。

解：酸性：CH₃COOH（pKa=4.76）> H₂CO₃（pKa₁=6.35）> C₆H₅OH（pKa≈10）> HCO₃⁻（pKa₂=10.33）
→ 乙酸能与 Na₂CO₃ 和 NaHCO₃ 反应放 CO₂（酸性强于碳酸及碳酸氢根）

**例**：实验室制备乙酸乙酯（酯化）。

解：
- 向圆底烧瓶中加入乙醇（过量）、乙酸、几滴浓 H₂SO₄
- 缓慢加热（防爆沸），导管收集到饱和 Na₂CO₃ 溶液中（除去乙酸和乙醇）
- 分液，有机层即乙酸乙酯（果香液体）

**例**：乙酸与乙醇酯化时，如何用同位素 ¹⁸O 追踪断键位置？

解：用 ¹⁸O 标记乙醇羟基：CH₃COOH + C₂H₅¹⁸OH → CH₃COO¹⁸C₂H₅ + H₂O
¹⁸O 出现在**酯**中（不在水中），说明乙酸断 O—H 键，乙醇断 C—¹⁸O 键保留在酯里（酸脱羟基，醇脱 H）。

### 易错点

- 乙酸沸点（118°C）远高于乙醇（78°C）和乙酸乙酯（77°C），酯化时馏出的是酯+醇（低沸点），不是乙酸
- 乙酸与 NaHCO₃ 反应放 CO₂（与苯酚不同），是乙酸酸性强于碳酸的有力证据
- "冰醋酸"是纯乙酸在低温时凝固（熔点 16.6°C），不含冰（不是水溶液）
