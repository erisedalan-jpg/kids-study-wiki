# 乙醇（高中）

> **一句话**：乙醇（CH₃CH₂OH）是最重要的一元醇，含醇羟基，能与 Na 反应、被氧化为乙醛、与酸酯化，工业上由乙烯水化或粮食发酵制备。
> **English**: Ethanol (CH₃CH₂OH) is the most important monohydric alcohol; it reacts with Na, is oxidized to acetaldehyde, and undergoes esterification; industrially it is produced by ethylene hydration or fermentation.

---

## 🎓 高中进阶

### 定义 / 结构特征

**乙醇**（ethanol）：
- 分子式 C₂H₆O，结构简式 CH₃CH₂OH
- 含**醇羟基**（—OH 连 sp³ 碳），伯醇
- 无色液体，沸点 78.4°C，与水完全互溶（分子间氢键）
- 密度 0.789 g/mL（小于水）

**结构要点**：
- O—H 键（可断：与 Na 反应，酯化时酸提供 H → 醇提供 —OH）
- C—O 键（可断：与 HBr 反应，消去脱水）
- α-碳（连 OH 的碳，即 C₂）有 H → 可被氧化

**制备**：
- 工业：CH₂=CH₂ + H₂O →(H₃PO₄,△) CH₃CH₂OH（乙烯水化）
- 发酵：C₆H₁₂O₆ →(酶) 2CH₃CH₂OH + 2CO₂

### 性质 / 反应

| 反应 | 方程式 | 条件 |
|------|--------|------|
| 与 Na | 2C₂H₅OH + 2Na → 2C₂H₅ONa + H₂↑ | 常温 |
| 催化氧化（→乙醛）| 2C₂H₅OH + O₂ →(Cu,△) 2CH₃CHO + 2H₂O | Cu 催化，加热 |
| 燃烧 | C₂H₅OH + 3O₂ → 2CO₂ + 3H₂O | 点燃 |
| 酯化 | C₂H₅OH + CH₃COOH ⇌ CH₃COOC₂H₅ + H₂O | 浓 H₂SO₄，△ |
| 分子内脱水 | C₂H₅OH →(170°C,浓H₂SO₄) CH₂=CH₂ + H₂O | 170°C |
| 分子间脱水 | 2C₂H₅OH →(140°C,浓H₂SO₄) C₂H₅OC₂H₅ + H₂O | 140°C |
| 与 HBr | C₂H₅OH + HBr →(浓H₂SO₄,△) C₂H₅Br + H₂O | H₂SO₄ 催化 |

**同位素追踪（酯化断键位置）**：
乙酸 + ¹⁸O 标记乙醇：CH₃CO—OH + HO—¹⁸O—C₂H₅ → CH₃COO¹⁸C₂H₅ + H₂O
→ 乙酸贡献 —OH，乙醇 C—O 键保留（酸脱羟基，醇脱氢）

### 典型例题

**例**：乙醇与钠反应和水与钠反应速率比较？

解：水与 Na 反应比乙醇剧烈（水中 O—H 键更易电离，酸性：H₂O > C₂H₅OH）。乙醇中 O—H 键由于烃基的推电子效应，酸性更弱。

**例**：Cu 催化氧化乙醇的实验现象。

解：将铜丝在酒精灯上烧至变黑（Cu + O₂ → CuO），迅速插入乙醇中，铜丝重新变红（CuO + C₂H₅OH → Cu + CH₃CHO + H₂O），并有刺激性气味（乙醛）。

**例**：如何鉴别乙醇、乙酸、乙酸乙酯？

解：
1. 闻气味（均有特殊气味，不够准确）
2. 加 Na₂CO₃：乙酸产生 CO₂ 气泡 → 确认乙酸；余两者分层（乙酸乙酯与 Na₂CO₃ 不互溶，乙醇互溶）→ 确认乙酸乙酯和乙醇

### 易错点

- 乙醇酯化时，断裂的是乙醇的 O—H 键（不是 C—O 键）——通过同位素标记验证
- 乙醇脱水：140°C（分子间）vs 170°C（分子内），温度低→醚，温度高→烯
- 乙醇虽含—OH，但酸性极弱（不能使石蕊变色，不能与 Na₂CO₃ 反应）
