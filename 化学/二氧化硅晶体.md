# 二氧化硅晶体

> **一句话**：SiO₂ 是 Si 原子以sp³杂化与4个O原子形成无限三维网络的原子晶体，熔点高达1710°C，是光导纤维和半导体的重要材料。
> **English**: SiO₂ is a covalent network solid in which sp³-hybridized Si is bonded to four O atoms in an infinite 3D network; melting point 1710°C; key material for optical fibers and semiconductors.

---

## 🎓 高中进阶

### 定义
**二氧化硅晶体**（Silicon Dioxide / Quartz）：SiO₂ 是典型的**原子晶体**，结构中每个 Si 以 **sp³ 杂化**与 4 个 O 原子形成 Si−O 共价键，每个 O 与 2 个 Si 相连，形成无限延伸的 [SiO₄] 四面体网络。

### 结构特征

| 特征 | 说明 |
|------|------|
| 晶体类型 | 原子晶体 |
| Si 的杂化 | sp³ |
| 每个 Si 配位数 | 4（与 4 个 O 键合）|
| 每个 O 配位数 | 2（与 2 个 Si 键合）|
| Si:O 比例 | 1:2（化学式 SiO₂）|
| 基本结构单元 | [SiO₄]⁴⁻ 四面体（共角连接）|
| 键 | Si−O 共价键（键能强）|

### 物理性质

| 性质 | 数值/特征 |
|------|----------|
| 熔点 | 1710°C（石英）|
| 硬度 | 莫氏 7（较硬）|
| 密度 | 2.65 g/cm³（α-石英）|
| 导电性 | 不导电（绝缘体）|
| 溶解性 | 不溶于水，溶于 HF（特殊）|

### SiO₂ 的化学性质
- **与 HF 反应**（唯一能溶解 SiO₂ 的酸）：SiO₂ + 4HF → SiF₄ + 2H₂O
  - 原因：Si−F 键比 Si−O 键强，HF 可将 O 替换为 F
  - 应用：刻蚀玻璃（氢氟酸）
- **与强碱反应**：SiO₂ + 2NaOH → Na₂SiO₃ + H₂O（酸性氧化物）
- **与 Na₂CO₃ 高温反应**：SiO₂ + Na₂CO₃ → Na₂SiO₃ + CO₂（玻璃制造）
- **不与盐酸/稀硫酸反应**（强酸也不行，因 SiO₂ 是酸性氧化物）

### 重要用途
- **光导纤维**：高纯 SiO₂ 用于光通信（全反射传光）
- **半导体工业**：Si 的原料，用于芯片制造
- **建筑材料**：石英砂（SiO₂）→ 水泥、玻璃原料
- **光学仪器**：石英棱镜、透镜

### 与 CO₂ 的对比（都是 EO₂ 式，结构截然不同）

| | CO₂ | SiO₂ |
|-|-----|------|
| 晶体类型 | 分子晶体 | 原子晶体 |
| 结构微粒 | CO₂ 分子 | Si 和 O 原子 |
| 熔点 | −78°C（升华）| 1710°C |
| 与水 | CO₂+H₂O⇌H₂CO₃ | SiO₂ 不溶于水 |

### 典型例题
**题**：为什么盛放 NaOH 溶液不能用玻璃瓶（需用塑料瓶），而盛放 HCl 可以？
**解**：玻璃含 SiO₂，强碱（NaOH）与 SiO₂ 反应：SiO₂ + 2NaOH → Na₂SiO₃ + H₂O，生成有粘性的 Na₂SiO₃ 会粘住玻璃瓶塞；HCl 不与 SiO₂ 反应（强酸不溶 SiO₂），可用玻璃瓶。

### 易错点
- SiO₂ 没有单独的"SiO₂ 分子"，化学式只是比例式
- SiO₂ 是酸性氧化物，能与碱反应，但不溶于强酸（HCl、H₂SO₄）
- HF 能腐蚀玻璃（SiO₂ + 4HF → SiF₄↑ + 2H₂O），因此 HF 须用塑料瓶储存
