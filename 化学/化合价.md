# 化合价

> **一句话**：化合价是元素在化合物中表现的化学结合能力，用正负数表示，化合物中各元素化合价之和为零。
> **English**: Valence (or valency) indicates the combining capacity of an element in a compound; in any compound, the sum of all valences equals zero.

---

## 📚 给 6-12 岁（自读版）

想象原子们在"握手"结合——氢原子只有一只手（化合价1），氧原子有两只手（化合价2）。一个氧原子可以和两个氢原子握手，组成水分子（H₂O）！

**什么是化合价？**

化合价就是原子"愿意握几只手"的数字，表示一种元素与其他元素结合的能力。

**化合价的正负**

- **正化合价**：通常是金属元素（如 Na 是 +1，Ca 是 +2，Al 是 +3，Fe 可以是 +2 或 +3）
- **负化合价**：通常是非金属元素（如 O 是 -2，Cl 是 -1，S 是 -2）

**化合价规律**

化合物中，所有元素化合价的代数和 = 0！

以 H₂O 为例：H 是 +1，O 是 -2
- 2个 H：2 × (+1) = +2
- 1个 O：1 × (-2) = -2
- 合计：+2 + (-2) = 0 ✓

**常见元素化合价记忆表**

| 元素 | 常见化合价 |
|------|---------|
| H、F、Cl、Br | 通常 +1（H负时-1）|
| O | -2 |
| Na、K | +1 |
| Ca、Mg、Zn | +2 |
| Al | +3 |
| Fe | +2 或 +3 |

**口诀**：氢一氧二来记牢，钠钾铝三是主调，铁有二三要分清，化合价和零！

## 🎓 给 12+（进阶版）

### 定义

化合价是元素在化合物中表现出的化学结合力，反映一个原子在形成化学键时得失或共用的电子数。正化合价表示失去电子（金属或电负性小的元素），负化合价表示得到电子（非金属或电负性大的元素）。

**化合价守恒定律**：化合物中各元素化合价与原子个数之积的代数和等于零。

$$\sum n_i \times v_i = 0$$

### 性质 / 反应 / 公式

**常见元素化合价汇总表**

| 元素 | 常见化合价 | 举例 |
|------|---------|------|
| H | +1（金属氢化物中-1）| HCl（+1）、NaH（-1）|
| O | -2（过氧化物中-1）| H₂O（-2）、H₂O₂（-1）|
| F | -1 | HF |
| Cl | -1（氧化物中+1,+3,+5,+7）| NaCl（-1）|
| Na | +1 | NaCl |
| K | +1 | KCl |
| Ca | +2 | CaCl₂ |
| Mg | +2 | MgO |
| Al | +3 | Al₂O₃ |
| Fe | +2 或 +3 | FeO（+2）、Fe₂O₃（+3）|
| Cu | +1 或 +2 | Cu₂O（+1）、CuO（+2）|
| N | -3,+1,+2,+3,+4,+5 | NH₃（-3）、NO₂（+4）|
| C | -4,+2,+4 | CH₄（-4）、CO₂（+4）|

**常见原子团的化合价**

| 原子团 | 化合价 |
|--------|-------|
| $OH^-$ 氢氧根 | -1 |
| $NO_3^-$ 硝酸根 | -1 |
| $SO_4^{2-}$ 硫酸根 | -2 |
| $CO_3^{2-}$ 碳酸根 | -2 |
| $NH_4^+$ 铵根 | +1 |
| $PO_4^{3-}$ 磷酸根 | -3 |

**由化合价写化学式（正交叉法）**

以 Al（+3）和 O（-2）为例：
- 化合价交叉得到原子个数比：Al₂O₃

$$Al^{+3}O^{-2} \xrightarrow{\text{交叉}} Al_2O_3$$

验证：$2 \times (+3) + 3 \times (-2) = +6 - 6 = 0$ ✓

### 典型实验 / 例题

**例题**：在 $Fe_2O_3$ 中，铁元素的化合价为（ ）
A. +2  B. +3  C. -3  D. +6

**解析**：设 Fe 化合价为 x，O 为 -2：
$2x + 3 \times (-2) = 0$
$2x = +6$
$x = +3$，选 **B**。

**例题 2**：用 Al（+3）和 SO₄²⁻（-2）写出硫酸铝的化学式。

**解析**：正交叉法：Al₂(SO₄)₃，验证：$2\times(+3) + 3\times(-2) = 0$ ✓

### 易错点

1. **单质化合价为 0**：O₂ 中氧的化合价是 0，不是 -2（化合价 -2 只存在于化合物中）。
2. 原子团（如 SO₄²⁻）参与计算时整体作为 -2 价，不要单独拆开计算。
3. 铁的化合价有 +2（亚铁 Fe²⁺）和 +3（铁离子 Fe³⁺）之分，对应不同化合物（FeO vs Fe₂O₃），考试中要根据化合物判断。
