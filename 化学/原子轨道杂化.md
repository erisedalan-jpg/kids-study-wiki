# 原子轨道杂化

> **一句话**：原子轨道杂化是同一原子中能量相近的原子轨道重新组合，形成一组能量、形状完全相同的新的等价轨道的过程，用来解释分子的空间构型和键角。
> **English**: Hybridization of atomic orbitals is the mixing of atomic orbitals to form a new set of equivalent hybrid orbitals that are used in the valence bond theory to explain molecular geometry and bond angles.

## 🎓 考点精讲

**定义/原理**  
在形成分子时，中心原子的能量相近的不同原子轨道（如 ns 与 np）可以先行混合，重新组合成一组新轨道，这一过程称为原子轨道的杂化，形成的新轨道称为杂化轨道。杂化轨道理论由鲍林提出，是价键理论的延伸。杂化轨道的数目等于参与杂化的原子轨道数目，杂化轨道的电子云密集于一方，在空间取向对称，从而更利于成键与形成特定的空间构型。

常见的杂化类型与构型：
- **sp 杂化**：1 个 ns 与 1 个 np 杂化，生成 2 条 sp 杂化轨道，夹角 180°，直线形（如 BeCl₂、乙炔中的 C）。
- **sp² 杂化**：1 个 ns 与 2 个 np 杂化，生成 3 条 sp² 杂化轨道，夹角 120°，平面三角形（如 BF₃、乙烯中的 C）。
- **sp³ 杂化**：1 个 ns 与 3 个 np 杂化，生成 4 条 sp³ 杂化轨道，夹角 109.5°，正四面体形（如 CH₄、金刚石中的 C）。
- 等性杂化与不等性杂化：若杂化轨道中孤电子对和成键电子对共同占据，孤电子对排斥作用大，会导致杂化轨道的成分不均，键角偏离理想角度（如 NH₃ 为 sp³ 不等性杂化，键角 107°；H₂O 为 sp³ 不等性杂化，键角 104.5°）。

**关键方法**  
高考中确定中心原子杂化方式的标准步骤：
1. 计算中心原子的 **价层电子对数**。公式：  
   价层电子对数 = σ 键电子对数 + 孤电子对数  
   σ 键电子对数 = 直接结合的配位原子数  
   孤电子对数 = ½(a - xb)  
   其中 a 为中心原子的价电子数；x 为配位原子数；b 为配位原子达到 8 电子稳定结构所需电子数（H 为 1，其他原子为 8－该原子的价电子数）。对于离子，a 应加上或减去电荷数。
2. 杂化轨道数 = 价层电子对数。  
   2 对 → sp 杂化；3 对 → sp² 杂化；4 对 → sp³ 杂化；5 对 → sp³d 杂化；6 对 → sp³d² 杂化（后两种在中学阶段要求了解）。
3. 结合杂化方式与孤电子对数，判断分子的实际空间构型（VSEPR 模型）。

**典型应用**  
- **给出分子/离子式，书写中心原子的杂化方式与空间构型**：如 SO₄²⁻ 中 S 的 σ 键数为 4，孤电子对数 = (6+2-4×2)/2 = 0，价层电子对 4，sp³ 杂化，正四面体形。  
- **分析有机分子中碳原子的杂化**：根据碳原子形成的 σ 键数和未成键情况，快速判断饱和碳为 sp³，双键碳为 sp²，三键碳为 sp。  
- **解释分子极性、键角大小与等电子体结构**：通过杂化与孤电子对数目比较键角，例如键角 CH₄ > NH₃ > H₂O。

## ⚠️ 高频易错点

- **误将配位原子数当作杂化轨道数**：忽略了孤电子对也占据杂化轨道，如水分子中 O 有 2 对孤电子对，杂化轨道总数为 4（sp³），而非 2。  
- **离子的电荷处理错误**：计算价层电子数时，中心原子为阳离子要减去电荷数，阴离子要加上电荷数，丢负号会导致杂化方式判断出错。  
- **混淆 VSEPR 理想构型与分子实际构型**：如有孤电子对时，分子的实际立体构型要去掉孤电子对占位，如 NH₃ 是三角锥形，不是四面体形。  
- **将多重键纳入杂化轨道计数**：双键、三键中的 π 键不参与杂化，未杂化的 p 轨道肩并肩形成，杂化轨道的数目只取决于 σ 键和孤电子对，与多重键无关。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 杂化轨道 | hybrid orbital | 由不同原子轨道混合而成的新轨道，用于形成 σ 键或容纳孤电子对 |
| σ 键 | sigma bond | 原子轨道沿键轴“头碰头”重叠形成的共价键 |
