# 坐标表达

> **一句话**：在化学中用元素符号和原子个数比（下标）表示物质组成的式子，**按一定顺序**书写元素符号，**用右下角数字**标明原子个数。
> **English**: Using element symbols and subscript numbers to represent the composition of a substance, written in a specific order with the atom count indicated by a lower-right numeral.

## 🎓 考点精讲

**定义与原理**：坐标表达是化学式书写的基本规则，包含两重含义：一是**元素符号的书写顺序**，二是**原子个数比的坐标式标注（下标）**。对于离子化合物，**阳离子在前，阴离子在后**，如 NaCl、CaCO₃；对于共价化合物或单质，通常按**电负性由小到大**顺序书写（与元素周期表中位置相关），如 NH₃ 而非 H₃N（特例：CH₄、NH₃、H₂O、HF 等习惯写法须牢记）。原子个数比用元素符号**右下角的阿拉伯数字**表示，“1”省略不写。

**关键解题方法**：
1. **书写顺序判定**：金属与非金属形成的二元化合物，**金属元素写在前**（如 Fe₂O₃、AlCl₃）；非金属与非金属形成的二元化合物，**正化合价（电负性小）的元素写在前**（如 CO₂、SO₂、P₂O₅）。
2. **下标判定**：根据化合价规则（化合物中各元素化合价代数和为零），交叉确定最小整数原子个数比。高考试题常以“已知元素化合价写出化学式”“根据名称写化学式”形式考查。
3. **与电子式/结构式的区别**：电子式表示最外层电子配对情况（用·或×），结构式用短线表示共用电子对，**坐标表达只关注元素种类和原子个数比**，三者不能混淆。

**高考典型设问**：
- “下列物质的化学式正确的是”——考查 NH₃ 与 PH₃ 书写规则差异（P 电负性小于 N → PH₃ 中 P 在前，但 NH₃ 为习惯写法 N 在前）。
- “用化学式表示某物质的组成”——如给定某晶胞结构或分子模型图，要求写出对应的化学式。

## ⚠️ 高频易错点

- **书写顺序混淆**：NH₃ 写为 H₃N（习惯写法）、OF₂ 按电负性 O 在前（F 电负性最大但 OF₂ 中 O 为正价），常与一般规则冲突，高考设错率高。
- **下标与角标混淆**：离子符号右上角标电荷数（如 SO₄²⁻），化学式右下角标原子个数，不可混用，如将硫酸根写为 SO₄²⁻ 而非 (SO₄)²⁻ 是典型错误。
- **有机物与无机物规则差异**：无机物按正价前负价后书写，有机物中 CH₄、C₂H₅OH 等含碳化合物碳写在前、氢次之、其他元素随后，与电负性规则不完全一致。
- **原子团不拆改写**：含原子团的物质，原子团内部元素顺序固定（如 OH⁻、NO₃⁻、NH₄⁺），不可因坐标表达规则而随意调换。
