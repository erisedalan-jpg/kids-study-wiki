# 天然高分子

> **一句话**：由生物体合成、以共价键重复连接而成的高分子量有机化合物，其单体单元通过糖苷键或肽键等聚合而成。
> **English**: High-molecular-weight organic compounds synthesized by living organisms and composed of repeating monomeric units linked by covalent bonds such as glycosidic or peptide bonds.

## 🎓 考点精讲

**定义与原理** 天然高分子指自然界中由生物合成的高分子化合物，其分子链由大量结构单元通过共价键重复连接而成，相对分子质量通常在 $10^4 \sim 10^7$ 量级。高考化学聚焦三大类天然高分子：多糖（如淀粉、纤维素）、蛋白质、核酸（高考阶段重点在前两者）。其化学本质均为缩聚产物——多糖由葡萄糖单元通过 **1,4-糖苷键**（纤维素、直链淀粉）或 **1,6-糖苷键**（支链淀粉）脱水缩合而成；蛋白质由 **α-氨基酸** 通过 **肽键**（酰胺键，$-\text{CO}-\text{NH}-$）缩合而成。

**重要天然高分子与化学反应方程式**：

- **淀粉**：$(C_6H_{10}O_5)_n$，遇碘变蓝（直链淀粉的螺旋空穴包合 $I_2$ 分子）。水解：（$(C_6H_{10}O_5)_n + nH_2O \xrightarrow{\text{H}^+/\text{酶}} nC_6H_{12}O_6$（葡萄糖））。最终水解产物葡萄糖含醛基，能被托伦试剂（银氨溶液）或新制 $Cu(OH)_2$ 悬浊液氧化。
- **纤维素**：$(C_6H_{10}O_5)_n$，结构单体为葡萄糖，但连接方式与淀粉不同（β-1,4-糖苷键），人体无纤维素酶故不能消化。能发生酯化反应生成硝酸纤维素酯或醋酸纤维素酯：$[C_6H_7O_2(OH)_3]_n + 3nHNO_3 \xrightarrow{\text{浓 }H_2SO_4} [C_6H_7O_2(ONO_2)_3]_n + 3nH_2O$（制火棉、珂罗酊）。
- **蛋白质**：由α-氨基酸缩合而成，含肽键。核心反应：
  - **水解**：酸、碱或酶催化下水解，先生成多肽，最终生成α-氨基酸。
  - **盐析**：加浓无机盐溶液（如 $(NH_4)_2SO_4$、$Na_2SO_4$），降低溶解度而析出，**可逆**，用于分离提纯蛋白质。
  - **变性**：加热、强酸强碱、重金属盐（$Pb^{2+}, Cu^{2+}, Hg^{2+}$）、紫外线、某些有机物（乙醇、甲醛）作用下，空间结构破坏，**不可逆**失活。
  - **颜色反应**：含苯环的蛋白质遇浓硝酸变黄（**黄蛋白反应**），用于鉴别。

**典型高考考查方式**：
1. **选择题**：多结合STSE情境（如“淀粉遇碘变蓝”、“误服重金属盐喝牛奶/豆浆”），判断物质成分与变化类型（物理/化学变化、盐析与变性辨析）。
2. **有机推断与信息题**：以二糖或多糖、肽链的水解产物逆向推断单体结构；或通过高聚物链节判断是加聚还是缩聚产物。
3. **计算题**：氨基酸脱水缩合中肽键数、脱水数与氨基酸数的计算关系—— n 个氨基酸形成 m 条肽链，肽键数 = 脱水分子数 = n - m。

## ⚠️ 高频易错点

- **淀粉与纤维素混淆**：两者通式虽均为 $(C_6H_{10}O_5)_n$，但聚合度 n 不同，结构不同（α-1,4 与 β-1,4 糖苷键连接），**不是同分异构体**。碘单质检验只对淀粉显蓝色，纤维素不反应。
- **聚合物表达式书写错误**：缩聚产物必须保留结构的端基或表示出“连接”形式（如写 $[C_6H_{10}O_5]_n$ 而非 $(C_6H_{10}O_5)_n$ 的情况须按教材书写习惯）。部分学生错误地在聚合度 n 前缺中括号。
- **缩聚物链节判断**：蛋白质由α-氨基酸缩合，重复链节不是单个氨基酸，而是“—$NH\!-\!CHR\!-\!CO$—”酰胺结构片段。
- **水解产物手性**：所有α-氨基酸（除甘氨酸外）均含有手性碳原子，水解得到的是光学活性的 L-氨基酸（生物语境）。需明确葡萄糖分子存在链状与环状互变异构，含多个手性碳。
- **盐析与变性不可混淆**：盐析是溶解度变化，物理过程，可逆；变性是空间构象破坏，化学过程（部分涉及非共价键断裂、二硫键断裂等），一般不可逆。重金属盐使蛋白质变性而非盐析。
