# 定量分析

> **一句话**：定量分析是通过实验手段测定物质中各组分确切含量或化学反应中各物质间精确数量关系的分析方法。
> **English**: Quantitative analysis is a chemical analysis method used to determine the exact amounts or concentrations of substances in a sample.

## 🎓 考点精讲

定量分析的核心是建立“待测物”与“已知标准物质”之间的**精确化学计量关系**，高考主要考查**滴定分析法**（酸碱中和滴定、氧化还原滴定、沉淀滴定、配位滴定）和**重量分析法**。

**（一）滴定分析的基本原理与计算核心**
无论何种滴定，其计算依据均为反应方程式中的计量数之比。例如，用已知浓度 $c_1$ 的标准液滴定 $V_2$ mL 待测液，反应为 $a\text{A} + b\text{B} \rightarrow$ 产物（A 为标准液溶质，B 为待测物），则到达化学计量点时满足：
$$\frac{c_1 V_1}{a} = \frac{c_2 V_2}{b}$$
由此可求待测液浓度 $c_2$。这是所有滴定计算题的**唯一根源公式**。

**（二）关键方法与指示剂选择逻辑**
1. **强酸强碱互滴**：计量点 pH=7，可选用甲基橙（变色范围 3.1–4.4）或酚酞（8.2–10.0）。
2. **强碱滴弱酸**（如 NaOH 滴醋酸）：计量点溶液呈碱性（pH>7），必须选**酚酞**。
3. **强酸滴弱碱**（如 HCl 滴氨水）：计量点溶液呈酸性（pH<7），必须选**甲基橙**。
4. **氧化还原滴定**：常利用标准液自身颜色变化指示终点（如 $\text{KMnO}_4$ 滴定，紫色褪去且半分钟不恢复即为终点），或外加淀粉指示剂（碘量法）。

**（三）典型应用与高考设问**
- **酸碱中和滴定**：常用于测定酸（碱）浓度、食品中总酸度等。设问方向：滴定管润洗操作、读数误差分析、指示剂选择、浓度计算与结果评价。
- **氧化还原滴定**：如用酸性 $\text{KMnO}_4$ 滴定 $\text{H}_2\text{O}_2$ 含量（反应式：$2\text{MnO}_4^- + 5\text{H}_2\text{O}_2 + 6\text{H}^+ = 2\text{Mn}^{2+} + 5\text{O}_2\uparrow + 8\text{H}_2\text{O}$），或测定补铁剂中铁含量。

## ⚠️ 高频易错点

- **指示剂选择错误**：忘记考虑计量点时溶液的酸碱性，一律用酚酞或任意选指示剂。必须根据“谁强显谁性”原则判断计量点 pH 环境。
- **滴定管用量与误差混淆**：滴定后滴定管尖嘴内有气泡，导致标准液体积读数偏大，计算出的待测液浓度**偏高**；若气泡在滴定前即存在且滴定后消失，则结果同样偏高。需分清“气泡何时产生”与“读数如何被影响”。
- **氧化还原方程式配平失分**：电荷守恒与质量守恒不同步检查，导致化学计量数错误，后续计算全盘皆输。尤其注意酸性条件下 $\text{MnO}_4^-$ 的还原产物是 $\text{Mn}^{2+}$，而非 $\text{MnO}_2$。
- **直接配制法与间接配制法混淆**：只有基准物质（如 $\text{K}_2\text{Cr}_2\text{O}_7$、$\text{Na}_2\text{CO}_3$）才能直接配成标准液，$\text{NaOH}$（易吸潮、吸收 $\text{CO}_2$）、$\text{HCl}$（易挥发）必须先用近似浓度配制，再用基准物质标定。
