# 工业流程分析

> **一句话**：以真实化工生产为背景，综合考查物质转化、反应原理、条件选择与分离提纯技术的题型。
> **English**: A comprehensive question type set in real chemical production contexts, testing material transformation, reaction principles, condition selection, and separation/purification techniques.

## 🎓 考点精讲

工业流程分析是高考化学综合实验与工艺题的核心，通常给出一张**原料→预处理→核心反应→分离提纯→目标产物**的流程图。

- **核心原理**：流程设计基于**化学反应原理**，需从热力学（自发方向、平衡产率）与动力学（反应速率、活化能）两个维度理解每一步操作的目的。例如，工业合成氨选择高温（～500℃）、高压（～30MPa）、铁触媒，是平衡移动（低温有利）与反应速率（高温有利）的折中优化；而煅烧硫铁矿（$4FeS_2+11O_2 \xrightarrow{\text{高温}} 2Fe_2O_3+8SO_2$）则是利用高温使反应自发、快速进行。

- **关键方法**：读流程题按“**抓主线、析条件、判分离、算经济**”四步走。
  1. **抓主线**：标出每一步进入和流出的物质，明确杂质在何处被除去。
  2. **析条件**：关注温度、压强、催化剂、pH 等条件，判断其目的（如控温防分解、防副反应、调pH使金属离子沉淀）。
  3. **判分离**：识别过滤、萃取、蒸馏、结晶、洗气等操作所对应的物质状态与性质差异。
  4. **算经济**：评价循环利用（提高原子利用率、降低成本）、尾气处理（环保）环节的合理性。

- **典型应用**：高考常见设问有——解释某一温度/溶液pH选择的原因；书写某一步骤的离子方程式；判断可循环使用的物质；由已知 $K_{sp}$ 计算离子沉淀完全时的pH（如 $K_{sp}[Fe(OH)_3]=2.8\times10^{-39}$，$c(Fe^{3+})\leq1\times10^{-5}mol/L$ 时沉淀完全，求pH（待家长核对常数））；判断杂质的最终存在形式等。

## ⚠️ 高频易错点

- **氧化还原顺序颠倒**：多种杂质共存时，加入氧化剂/还原剂后，不分析电极电势或氧化性强弱，错误判断优先氧化还原的物质。
- **分离操作混淆**：将重结晶（利用溶解度差异，适用于溶解度随温度变化大的物质）与蒸馏（利用沸点差异，适用于互溶液体混合物）的使用场景记反。
- **条件控制原理表述模糊**：回答“控制温度在X℃”的作用时，只说“加快反应”，未具体指出防止某物质分解、挥发、分解副反应或保证催化剂活性等化学原因。
- **产率计算忽略副反应或过量投料**：循环利用的物料在计算总投料时被漏算，或未明确指出实际产率低于理论值是因为存在副反应、反应不完全或分离损失。
