# 强电解质

> **一句话**：在水溶液中能完全电离的电解质，溶液中不存在未电离的分子。
> **English**: An electrolyte that completely ionizes in aqueous solution, leaving no undissociated molecules.

---

## 🎓 高中进阶

### 定义

强电解质是指溶于水时**完全（100%）电离**的电解质。溶液中只存在水合离子，没有未电离的溶质分子。电离方程式用"→"（单向箭头）表示。

### 常见强电解质记忆口诀

**强酸**（完全电离）：
$$\text{HCl, H}_2\text{SO}_4, \text{HNO}_3, \text{HBr, HI, HClO}_4$$

**强碱**（完全电离）：
$$\text{NaOH, KOH, Ba(OH)}_2, \text{Ca(OH)}_2\text{（微溶但强碱）}$$

**大多数盐**（离子晶体，溶解即全部电离）：
$$\text{NaCl, CuSO}_4, \text{Na}_2\text{CO}_3, \text{Na}_2\text{SO}_4...$$

### 电离方程式书写

强电解质：用"→"（完全电离，不可逆）

$$\text{H}_2\text{SO}_4 \rightarrow 2\text{H}^+ + \text{SO}_4^{2-}$$
$$\text{NaCl} \rightarrow \text{Na}^+ + \text{Cl}^-$$
$$\text{NaOH} \rightarrow \text{Na}^+ + \text{OH}^-$$

弱电解质：用"⇌"（部分电离，可逆）

$$\text{CH}_3\text{COOH} \rightleftharpoons \text{CH}_3\text{COO}^- + \text{H}^+$$

### 强电解质的特征

| 特征 | 说明 |
|------|------|
| 电离度 | 100% |
| 溶液中粒子 | 只有水合离子 |
| 电离方程式 | 单向箭头"→" |
| 离子方程式中 | 必须拆开写成离子形式 |

### 典型例题

**例1**：下列属于强电解质的是（ ）
A. NH₃·H₂O　B. H₂CO₃　C. CH₃COOH　D. FeCl₃

**分析**：NH₃·H₂O是弱碱（弱电解质）；H₂CO₃是弱酸；CH₃COOH是弱酸；FeCl₃是盐（离子化合物），溶解完全电离，是**强电解质**。**答案：D**

**例2**：写出CuSO₄溶液中的电离方程式。

$$\text{CuSO}_4 \rightarrow \text{Cu}^{2+} + \text{SO}_4^{2-}$$

### 易错点 / 拓展

- 强电解质不一定溶解度大（BaSO₄难溶，但溶解部分完全电离，是强电解质）
- Ca(OH)₂微溶但是**强碱**（强电解质）
- 离子方程式中，强电解质必须**拆开写**，弱电解质保留分子式
- HF是**弱酸**（弱电解质），尽管腐蚀性极强
