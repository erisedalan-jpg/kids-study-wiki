# 有机反应类型

> **一句话**：有机反应类型是根据反应过程中共价键断裂方式（均裂/异裂）和官能团转化规律对有机化学反应进行的分类。
> **English**: Organic reaction types are classifications of organic reactions based on the modes of covalent bond cleavage (homolytic/heterolytic) and the transformation patterns of functional groups.

## 🎓 考点精讲

**定义/原理**  
有机反应的核心是旧键断裂与新键形成，根据键断裂方式分为均裂（产生自由基）和异裂（产生离子），由此衍生出自由基型反应和离子型反应。高中阶段主要按**官能团转化的形式**分为以下类型：

- **取代反应**：有机物分子中某些原子或原子团被其他原子或原子团替代。如甲烷的氯代：$\ce{CH4 + Cl2 ->[光] CH3Cl + HCl}$；苯的溴代：$\ce{C6H6 + Br2 ->[FeBr3] C6H5Br + HBr}$；卤代烃水解：$\ce{C2H5Br + NaOH ->[H2O][Δ] C2H5OH + NaBr}$。
- **加成反应**：不饱和碳原子与其他原子或原子团直接结合生成新化合物。常见于含不饱和键的物质。如乙烯与溴水：$\ce{CH2=CH2 + Br2 -> CH2Br-CH2Br}$；乙炔与$\ce{HCl}$：$\ce{CH≡CH + HCl ->[HgCl2] CH2=CHCl}$；苯与$\ce{H2}$：$\ce{C6H6 + 3H2 ->[Ni][Δ] C6H12}$。
- **消去反应**：有机物分子中脱去小分子（如$\ce{H2O, HX}$）生成不饱和键。消去反应要求至少有两个相邻碳原子，且碳上连有可脱去的原子或原子团。如乙醇分子内脱水：$\ce{CH3CH2OH ->[浓H2SO4][170℃] CH2=CH2↑ + H2O}$；溴乙烷脱$\ce{HBr}$：$\ce{CH3CH2Br + NaOH ->[醇][Δ] CH2=CH2↑ + NaBr + H2O}$。
- **氧化反应**：有机物加氧或去氢的过程。如醇的催化氧化：$\ce{2CH3CH2OH + O2 ->[Cu/Ag][Δ] 2CH3CHO + 2H2O}$；醛基氧化：$\ce{CH3CHO + 2Ag(NH3)2OH ->[Δ] CH3COONH4 + 2Ag↓ + 3NH3 + H2O}$。
- **还原反应**：有机物加氢或去氧。常见是不饱和烃及含氧衍生物的催化加氢。例：$\ce{CH3CHO + H2 ->[Ni][Δ] CH3CH2OH}$。
- **聚合反应**：由小分子（单体）生成高分子化合物的反应，分为加聚反应（如乙烯聚合：$\ce{nCH2=CH2 ->[催化剂] [CH2-CH2]_n}$）和缩聚反应（如二元酸与二元醇缩聚）。

**关键方法**  
判断有机反应类型的标准步骤：① 分析反应物与产物的分子式或不饱和度变化；② 识别断裂与生成的官能团；③ 根据典型反应条件（如“浓$\ce{H2SO4}$、加热”可能消去或酯化，“$\ce{Ni}$、加热”多为加成/还原，“光照”可能为取代）辅助判断；④ 对于给定合成路线，可利用“原子经济性”与“结构变化”推断反应类型。

**典型应用**  
高考设问常以**有机合成路线推断**或**选择题**形式出现。例：给出流程“$\ce{CH3CH2Br ->[NaOH/醇, Δ] A ->[Br2] B}$”，要求写出反应类型——先是消去反应（生成$\ce{CH2=CH2}$），后是加成反应（生成$\ce{CH2Br-CH2Br}$）。或给出一陌生反应，如“$\ce{RCH=CH2 + CO + H2 ->[催化剂] RCH2CH2CHO}$”，要求归类为加成反应（或羰基化反应，但高考视作加成）。解题关键在于牢牢抓住官能团与碳骨架的变化。

## ⚠️ 高频易错点

- **混淆取代反应与加成反应**：取代反应产物一般有两种分子（如$\ce{CH3Cl + HCl}$），加成反应产物只有一种分子。当苯环与卤素反应时，若条件为$\ce{FeX3}$（或铁粉）则为取代，若为光照则是侧链取代或加成，极易混淆。
- **错误判定消去反应的结构条件**：消去要求被脱去的原子或原子团所连碳的邻位碳上必须有氢（β-H）。如$\ce{(CH3)3CCH2OH}$不能发生消去反应，因为与$\ce{-OH}$相连的碳的邻位碳上没有氢。
- **将有机氧化还原与无机氧化还原概念等同**：无机中常用化合价升降定义，有机中则习惯用“加氧去氢为氧化，加氢去氧为还原”。部分反应如醇加$\ce{O2}$氧化，碳的化合价确实升高，但醛加氢还原中碳化合价降低，注意不要与无机反应机理混淆。
- **忽视反应条件的决定作用**：同一反应物在不同条件下可能发生不同反应类型。如乙醇在浓硫酸催化下，170℃发生消去，140℃发生分子间脱水生成乙醚（属于取代反应）。不关注条件就无法正确判断类型。
