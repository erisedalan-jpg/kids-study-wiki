# 气体制备装置

> **一句话**：一套完整的实验室气体发生、净化（干燥）、收集、检验及尾气处理的仪器组合。
> **English**: A complete set of laboratory apparatus for gas generation, purification (drying), collection, testing, and tail gas treatment.

## 🎓 考点精讲

**核心原理**：根据反应物状态（固、液）、反应条件（加热、不加热）及目标气体的性质（密度、溶解性、酸碱性、氧化还原性）选择并组装装置。

**装置构成与选择依据**：
1.  **发生装置**：
    -   **固固加热型**（如制 $O_2$、$NH_3$）：大试管、铁架台、酒精灯。管口略向下倾斜，防止冷凝水回流炸裂试管。反应示例：
        $$2KClO_3 \xrightarrow[\Delta]{MnO_2} 2KCl + 3O_2 \uparrow$$
    -   **固液（液液）不加热型**（如制 $H_2$、$CO_2$、$H_2S$、$SO_2$、$NO$、$NO_2$、$C_2H_2$）：启普发生器（块状固体）或分液漏斗+圆底烧瓶。分液漏斗可控制反应速率，启普发生器可“随开随用，随关随停”。反应示例：
        $$CaCO_3 + 2HCl \rightarrow CaCl_2 + H_2O + CO_2 \uparrow$$
    -   **固液（液液）加热型**（如制 $Cl_2$、$HCl$、$C_2H_4$）：分液漏斗+圆底烧瓶+石棉网+酒精灯。反应示例：
        $$MnO_2 + 4HCl(\text{浓}) \xrightarrow{\Delta} MnCl_2 + Cl_2 \uparrow + 2H_2O$$
2.  **净化（干燥）装置**：洗气瓶（长进短出）、球形干燥管（粗进细出）、U形管。原则是不引入新杂质、不减少目标气体。
    -   **酸性干燥剂**（浓 $H_2SO_4$、$P_2O_5$）：不可干燥碱性气体（$NH_3$）。
    -   **碱性干燥剂**（碱石灰、$CaO$）：不可干燥酸性气体（$CO_2$、$Cl_2$、$SO_2$）。
    -   **中性干燥剂**（$CaCl_2$）：不可干燥 $NH_3$（形成络合物）。
3.  **收集装置**：
    -   **排水法**：收集不溶于水、不与水反应的气体（$O_2$、$H_2$、$NO$、$C_2H_2$、$CH_4$）。
    -   **向上排空气法**：气体密度大于空气（$CO_2$、$Cl_2$、$SO_2$，导气管“长进短出”）。
    -   **向下排空气法**：气体密度小于空气（$H_2$、$NH_3$，导气管“短进长出”）。
4.  **尾气处理**：溶液吸收、点燃、收集袋收集。常见尾气：$Cl_2$、$SO_2$（$NaOH$ 溶液吸收）；$CO$（点燃或收集袋）。

**高考考查方式**：以气体制备流程图为载体，设问角度包括装置选择、仪器名称、除杂顺序、化学方程式书写、实验操作目的、装置连接顺序及误差分析。解题关键是根据气体所含杂质（如制 $Cl_2$ 含 $HCl$，先过饱和食盐水再过浓 $H_2SO_4$）确定洗气瓶顺序：**除杂在前，干燥在后**。

## ⚠️ 高频易错点

- **混淆发生装置适用条件**：将启普发生器用于粉末状固体（如 $MnO_2$）或剧烈放热反应；将固固加热型试管“管口略向下倾斜”遗忘或用于制 $NH_3$ 时忽略碱石灰吸水。
- **干燥剂选择错误**：浓 $H_2SO_4$ 干燥 $NH_3$；碱石灰干燥 $CO_2$ 或 $Cl_2$；$CaCl_2$ 干燥 $NH_3$。
- **导管连接错误**：洗气瓶“短进长出”；排空气法收集气体时进气/出气方向与气体密度不匹配；排水法收集气体时导管未伸至集气瓶口或集满后先撤导管后撤酒精灯导致倒吸。
- **忽略气体摩尔质量影响排空气法**：例如 $NO_2$ 虽然摩尔质量大于空气，但高温易二聚，且与空气密度差值远小于 $CO_2$，排空气法效率低，但高考常考其作为“密度大于空气”的代表，需结合题目暗示。
