# 氢氧化铁

> **一句话**：红褐色固体，Fe³⁺对应的碱，难溶于水，加热分解为Fe₂O₃，也是Fe(OH)₂在空气中被氧化的产物。
> **English**: A reddish-brown solid, the base corresponding to Fe³⁺; sparingly soluble in water; decomposes on heating to Fe₂O₃.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | Fe(OH)₃ |
| Fe化合价 | +3 |
| 颜色 | **红褐色**沉淀 |
| 溶解性 | 难溶于水 |

### 制备

**1. FeCl₃ + NaOH**（直接制备）：
$$\text{FeCl}_3 + 3\text{NaOH} \rightarrow \text{Fe(OH)}_3\downarrow + 3\text{NaCl}$$
$$\text{（离子方程式）：Fe}^{3+} + 3\text{OH}^- \rightarrow \text{Fe(OH)}_3\downarrow$$

**2. Fe(OH)₂ + O₂ + H₂O**（空气氧化）：
$$4\text{Fe(OH)}_2 + \text{O}_2 + 2\text{H}_2\text{O} \rightarrow 4\text{Fe(OH)}_3$$

**3. FeCl₃水解**（制备胶体）：
$$\text{FeCl}_3 + 3\text{H}_2\text{O} \xrightarrow{\Delta} \text{Fe(OH)}_3\text{（胶体）} + 3\text{HCl}$$

（将饱和FeCl₃滴入沸腾蒸馏水中，加热至红褐色）

### 化学性质

**与稀盐酸**：
$$\text{Fe(OH)}_3 + 3\text{HCl} \rightarrow \text{FeCl}_3 + 3\text{H}_2\text{O}$$
$$\text{（离子方程式）：Fe(OH)}_3 + 3\text{H}^+ \rightarrow \text{Fe}^{3+} + 3\text{H}_2\text{O}$$

**加热分解**：
$$2\text{Fe(OH)}_3 \xrightarrow{\Delta} \text{Fe}_2\text{O}_3 + 3\text{H}_2\text{O}$$

### Fe(OH)₂ vs Fe(OH)₃

| 特征 | Fe(OH)₂ | Fe(OH)₃ |
|------|---------|---------|
| 颜色 | 白色 | 红褐色 |
| Fe化合价 | +2 | +3 |
| 稳定性 | 不稳定（被O₂氧化）| 较稳定 |
| 对应离子 | Fe²⁺ | Fe³⁺ |
| 溶液颜色 | 浅绿色（Fe²⁺）| 黄色（Fe³⁺）|

### 胶体制备（实验操作）

向沸腾蒸馏水中滴加几滴饱和FeCl₃溶液，继续加热至液体变为**红褐色**，得Fe(OH)₃胶体。（不能搅拌，不能加热过度，不能直接滴加NaOH）

### 典型例题

**例1**：向FeCl₃溶液中加入足量NaOH，写离子方程式和现象。

$$\text{Fe}^{3+} + 3\text{OH}^- \rightarrow \text{Fe(OH)}_3\downarrow$$
现象：产生**红褐色沉淀**。

**例2**：Fe(OH)₃胶体与FeCl₃溶液的区分？

**分析**：用激光笔照射，胶体出现丁达尔效应（光路可见），溶液不出现。

### 易错点

- Fe(OH)₃是红褐色（不是红色，也不是棕黄色——要记住颜色）
- Fe(OH)₃是弱碱（难溶），不能与NaOH反应（不是两性）
- Fe(OH)₃胶体是混合物，不是纯净物
- 明矾净水：Al(OH)₃胶体，不是Fe(OH)₃
