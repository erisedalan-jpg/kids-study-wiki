# 氢氧化铝

> **一句话**：白色胶状沉淀，两性氢氧化物，既能与酸又能与强碱反应，加热分解为Al₂O₃，有医药用途。
> **English**: A white gelatinous precipitate, amphoteric hydroxide that reacts with both acids and strong bases; decomposes to Al₂O₃ on heating; has medical uses.

---

## 🎓 高中进阶

### 基本信息

| 属性 | 数据 |
|------|------|
| 化学式 | Al(OH)₃ |
| Al化合价 | +3 |
| 颜色 | 白色胶状沉淀 |
| 溶解性 | 难溶于水 |
| 类型 | **两性氢氧化物** |

### 制备

**向AlCl₃溶液中加NH₃·H₂O**（弱碱，不会使Al(OH)₃溶解）：
$$\text{AlCl}_3 + 3\text{NH}_3\cdot\text{H}_2\text{O} \rightarrow \text{Al(OH)}_3\downarrow + 3\text{NH}_4\text{Cl}$$
$$\text{（离子方程式）：Al}^{3+} + 3\text{NH}_3\cdot\text{H}_2\text{O} \rightarrow \text{Al(OH)}_3\downarrow + 3\text{NH}_4^+$$

**不能用NaOH制备纯Al(OH)₃**（NaOH过量会溶解Al(OH)₃）

### 两性氢氧化物的性质

**与酸反应**：
$$\text{Al(OH)}_3 + 3\text{HCl} \rightarrow \text{AlCl}_3 + 3\text{H}_2\text{O}$$
$$\text{（离子方程式）：Al(OH)}_3 + 3\text{H}^+ \rightarrow \text{Al}^{3+} + 3\text{H}_2\text{O}$$

**与NaOH溶液反应**：
$$\text{Al(OH)}_3 + \text{NaOH} \rightarrow \text{NaAlO}_2 + 2\text{H}_2\text{O}$$
$$\text{（离子方程式）：Al(OH)}_3 + \text{OH}^- \rightarrow \text{AlO}_2^- + 2\text{H}_2\text{O}$$

**加热分解**：
$$2\text{Al(OH)}_3 \xrightarrow{\Delta} \text{Al}_2\text{O}_3 + 3\text{H}_2\text{O}$$

### Al(OH)₃与NaOH的反应规律

**向AlCl₃溶液中逐滴加入NaOH**：
- 少量NaOH：Al³⁺ + 3OH⁻ → Al(OH)₃↓（沉淀量增多）
- 过量NaOH：Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O（沉淀量减少，最终溶解）

**向NaOH中逐滴加入AlCl₃**：
- 加入AlCl₃（NaOH足量）：先生成NaAlO₂（无沉淀，NaOH过量）
  Al³⁺ + 4OH⁻（过量）→ AlO₂⁻ + 2H₂O
- 继续加AlCl₃（NaOH耗尽）：才出现Al(OH)₃沉淀

### 医药应用

| 应用 | 原理 |
|------|------|
| 胃药（抗酸）| Al(OH)₃ + 3HCl → AlCl₃ + 3H₂O，中和胃酸 |
| 净水（明矾）| Al(OH)₃胶体吸附悬浮颗粒 |

### 典型例题

**例1**：向AlCl₃溶液中加入足量NaOH，沉淀会溶解吗？

**分析**：会溶解。先生成Al(OH)₃↓（白色），NaOH过量后Al(OH)₃ + OH⁻ → AlO₂⁻ + 2H₂O，沉淀溶解，溶液变为含NaAlO₂的无色溶液。

**例2**：为什么不能用NaOH制备Al(OH)₃的纯沉淀？

**分析**：NaOH是强碱，加入过量会溶解Al(OH)₃；应用氨水（弱碱，NH₃·H₂O），氨水不会使Al(OH)₃溶解。

### 易错点

- Al(OH)₃能溶于NaOH（强碱），不能溶于氨水（弱碱）
- Al(OH)₃是**两性氢氧化物**，高中代表物
- 制备Al(OH)₃纯沉淀用**NH₃·H₂O**，不用NaOH
- Al(OH)₃与HCl反应的记忆：Al(OH)₃ + 3H⁺ → Al³⁺ + 3H₂O（不含配位H₂O）
