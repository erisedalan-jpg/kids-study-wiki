# 氧化还原反应基本概念

> **一句话**：氧化还原反应是有电子转移（得失或偏移）的化学反应，宏观特征为元素化合价升降。
> **English**: A redox reaction is a chemical reaction involving the transfer of electrons between species, characterized by changes in oxidation states.

## 🎓 考点精讲

**定义/原理**  
氧化还原反应的本质是**电子转移**，包括电子得失和共用电子对的偏移；宏观判据是**元素化合价的升高和降低**。  
- 氧化剂：得电子（电子对偏向），化合价降低，被还原，发生还原反应，体现氧化性。  
- 还原剂：失电子（电子对偏离），化合价升高，被氧化，发生氧化反应，体现还原性。  
口诀：“升失氧，降得还”——化合价升高，失电子，被氧化（还原剂）；化合价降低，得电子，被还原（氧化剂）。  
电子守恒规律：氧化剂得电子总数 = 还原剂失电子总数。

**关键方法**  
1. **双线桥法**：标出同一元素反应前后的化合价，用“×”表示失去（升高）或得到（降低）的电子，注明“失”和“得”，并标注电子转移总数。  
2. **单线桥法**：从还原剂中失电子的元素指向氧化剂中得电子的元素，只标转移电子总数，箭头不写“得”“失”。  
3. **氧化还原方程式的配平**——化合价升降法：标变价元素化合价 → 计算化合价升高总数与降低总数 → 求最小公倍数使升降相等 → 以“原子守恒”配平变价物质系数 → 用观察法配平其他物质（注意酸碱性环境补充 H⁺/OH⁻/H₂O）。

**典型应用**  
高考高频设问：  
- 判断给定反应是否为氧化还原反应，并识别氧化剂、还原剂、氧化产物、还原产物。  
- 利用氧化还原原理配平陌生方程式（如工艺流程、电化学中），并进行电子转移数计算。  
- 结合强制弱规律比较物质氧化性或还原性的相对强弱，或以电极反应判断。  
例：实验室制氯气  
`2KMnO₄ + 16HCl(浓) → 2KCl + 2MnCl₂ + 5Cl₂↑ + 8H₂O`  
Mn 从 +7 → +2，得 5×2e⁻=10e⁻；Cl 从 -1 → 0，失 2×5e⁻=10e⁻，电子转移总数为 10e⁻。  
例：工业湿法炼铜 `Fe + CuSO₄ → FeSO₄ + Cu`，Fe 作还原剂，Cu²⁺ 作氧化剂。

## ⚠️ 高频易错点

- **单质参与的反应未必是氧化还原**：同素异形体转化如 `3O₂ (放电) → 2O₃`，元素化合价未变，无电子转移，不属于氧化还原反应。  
- **氧化性/还原性强弱与变价多少无关**：氧化性取决于得电子能力，如稀硝酸氧化性弱于浓硝酸，但自身还原产物中N 化合价可能降得更低，不能由产物价态直接判断氧化性。  
- **反应物用量与反应顺序**：多种还原剂与一种氧化剂时，还原性强的优先反应。如 `Fe` 与稀 `HNO₃`，Fe 少量生成 `Fe(NO₃)₃`，Fe 过量生成 `Fe(NO₃)₂`。  
- **电子转移计算忽略部分变价**：如 `2Na₂O₂ + 2H₂O → 4NaOH + O₂↑`，`Na₂O₂` 中 `O` 为 `-1` 价，既是氧化剂又是还原剂，每生成 1 mol O₂ 转移 2 mol e⁻，而非 4 mol e⁻。
