# 氧化还原反应方程式的配平

> **一句话**：氧化还原反应方程式的配平是在遵循电子守恒、电荷守恒和质量守恒的基础上，通过确定氧化数与化合价升降，使氧化剂与还原剂得失电子总数相等的定量操作。
> **English**: Balancing redox equations is the process of adjusting coefficients so that the total electrons lost by the reducing agent equal the total electrons gained by the oxidizing agent, while satisfying atom and charge conservation.

## 🎓 考点精讲

### 定义与原理
氧化还原反应方程式的配平，核心依据是**化合价升降总数相等**（电子守恒）。配平时需同时满足：
- 电子守恒：氧化剂得电子数 = 还原剂失电子数；
- 电荷守恒：离子方程式中，反应前后的净电荷数相等；
- 质量守恒：反应前后各原子种类和数目不变。

### 关键方法——化合价升降法（氧化数法）
**标准步骤**（以 `MnO₄⁻ + H₂O₂ + H⁺ → Mn²⁺ + O₂↑ + H₂O` 为例）：
1. **标化合价**：找到有化合价变化的元素。Mn：+7 → +2（↓5），O（部分）：-1（H₂O₂）→ 0（O₂）（↑1×2，因每个O₂含两个氧原子）。
2. **列升降**：1个Mn降5价，每生成1个O₂升2价。
3. **求最小公倍数，定系数**：最小公倍数为10，故Mn的系数为2，O₂的系数为5，即 `2MnO₄⁻ + 5H₂O₂ → 2Mn²⁺ + 5O₂↑`。
4. **电荷守恒配介质**：反应物侧电荷：2×(-1) + 0 = -2；生成物侧电荷：2×(+2) = +4。酸性介质中，加H⁺平衡电荷：需在反应物侧加6H⁺，使两侧电荷均为+4。
5. **质量守恒配水**：检查H、O原子数，最后补H₂O：右端补8H₂O。
完整方程式：
`2MnO₄⁻ + 5H₂O₂ + 6H⁺ = 2Mn²⁺ + 5O₂↑ + 8H₂O`

**缺项配平（重要考查形式）**：题干常隐去H⁺/OH⁻/H₂O，需依据介质环境和电荷守恒推断。酸性介质补H⁺和H₂O，碱性介质补OH⁻和H₂O。也可使用**离子-电子法**（半反应法）直接书写氧化与还原半反应后叠加，对复杂电极反应尤为有效。

### 典型应用与高考设问
- **直接配平型**：给出不完整的反应物和生成物，要求填入系数及缺项（H⁺、OH⁻、H₂O），并指出氧化剂与还原剂的物质的量之比。
- **定量计算型**：依托配平结果进行电子转移数目计算，如“生成1 mol O₂转移多少电子？”（上例中生成5 mol O₂转移10 mol e⁻，故每mol O₂对应2 mol e⁻）。
- **陌生情景配平**：在工业流程题或元素化学推导题中，利用氧化还原规则配平陌生的化合物转化，如 `Cr₂O₇²⁻ + Fe²⁺ + H⁺ → Cr³⁺ + Fe³⁺ + H₂O`。

## ⚠️ 高频易错点

- **化合价误判**：在多原子离子（如C₂O₄²⁻、S₂O₃²⁻）或有机物中，勿按离子电荷直接认定某元素化合价，需根据结构或已知元素反推（如C₂O₄²⁻中C为+3价，而非+2）。
- **遗漏介质离子或水量**：酸性条件下忘记添加H⁺与H₂O，碱性条件下漏掉OH⁻与H₂O，导致电荷或原子不守恒。
- **歧化/归中反应系数混乱**：如 `Cl₂ + OH⁻ → Cl⁻ + ClO₃⁻ + H₂O`，应先将Cl₂拆分为两部分考虑升降，再按最小公倍数配系数，不能直接在Cl₂前随意试数。
- **电子转移数计算错误**：易将“化合价变化数”与“系数”割裂，正确算法为：电子转移数 = 化合价升降数 × 发生该变化的原子个数（体现于系数）。
