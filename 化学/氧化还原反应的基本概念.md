# 氧化还原反应的基本概念

> **一句话**：氧化还原反应的本质是电子转移（或偏移），宏观表现为某些元素的化合价在反应前后发生改变。
> **English**: A redox reaction is a chemical reaction involving the transfer of electrons between two species, resulting in changes in their oxidation states.

## 🎓 考点精讲

**定义与本质**
氧化还原反应的核心是**电子守恒**：一个物质失去电子（氧化）的同时，必有另一个物质得到电子（还原），且得失电子总数相等。从化合价视角，反应前后有元素化合价升降的化学反应即为氧化还原反应。

**核心规律——“升失氧、降得还”**
- **化合价升高** → **失去电子** → 发生**氧化反应**（本身作**还原剂**，具有**还原性**）
- **化合价降低** → **得到电子** → 发生**还原反应**（本身作**氧化剂**，具有**氧化性**）

例如反应 $2\ce{Na} + \ce{Cl2} \rightarrow 2\ce{NaCl}$，$\ce{Na}$ 化合价从 0 → +1，失电子被氧化；$\ce{Cl}$ 化合价从 0 → –1，得电子被还原。

**典型高考设问**
高考常考查：①判断一个陌生反应是否属于氧化还原反应（依据：是否存在化合价变化）；②在给定反应中识别氧化剂、还原剂及对应产物；③结合离子反应、电化学、工艺流程进行综合考查。解题时务必将**氧化数变化**作为切入点，画出双线桥或单线桥清晰标明电子转移方向与数目，再逐项分析。

## ⚠️ 高频易错点

- **混淆“氧化”与“氧化剂”**：氧化剂自身被**还原**，发生还原反应。记忆诀窍——“剂”与“性”二字要反过来想。
- **误以为所有涉及氧元素的反应都是氧化还原反应**：如 $\ce{CaO + H2O = Ca(OH)2}$ 中无化合价变化，不属于氧化还原反应。
- **在计算电子转移数目时忽略参与反应的原子个数**：例如 $3\ce{Cu} + 8\ce{HNO3(稀)}...$ 易只算一个氮原子的降价而不乘以系数，正确做法必须按实际参与氧化还原的原子总数计算转移电子。
- **不能正确判断有机物中碳的化合价**：有机物常按 C–H 键中碳为 –1、C–O/C–X 键中碳为 +1 的方法递推平均氧化数，若机械套用无机物规则极易出错。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 氧化还原反应 | Redox reaction | 有电子转移或化合价变化的反应 |
| 氧化反应 | Oxidation | 物质失电子、化合价升高的过程 |
| 还原反应 | Reduction | 物质得电子、化合价降低的过程 |
| 氧化剂 | Oxidizing agent | 得电子的物质，本身被还原 |
| 还原剂 | Reducing agent | 失电子的物质，本身被氧化 |
