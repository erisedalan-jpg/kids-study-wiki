# 氨基酸

> **一句话**：氨基酸是同时含有氨基（—NH₂）和羧基（—COOH）的有机化合物，是蛋白质的基本单体，α-氨基酸通过肽键缩合形成多肽链。
> **English**: Amino acids contain both an amino group (—NH₂) and a carboxyl group (—COOH); they are the basic monomers of proteins, and α-amino acids condense via peptide bonds to form polypeptide chains.

---

## 🎓 高中进阶

### 定义 / 结构特征

**氨基酸**（amino acid）：含**—NH₂**（氨基）和**—COOH**（羧基）的有机物。

**α-氨基酸**（蛋白质的基本单元）：氨基和羧基均连在 α-碳（同一个碳）上
通式：H₂N—C\*(H)(R)—COOH（R 为侧链，决定氨基酸种类）

**常见氨基酸**：

| 名称 | 简写 | 侧链 R | 特点 |
|------|------|--------|------|
| 甘氨酸 | Gly | —H | 最简单，无手性碳 |
| 丙氨酸 | Ala | —CH₃ | 非极性 |
| 缬氨酸 | Val | —CH(CH₃)₂ | 必需氨基酸 |
| 赖氨酸 | Lys | —(CH₂)₄NH₂ | 含 2 个—NH₂ |
| 谷氨酸 | Glu | —CH₂CH₂COOH | 含 2 个—COOH |

**两性**：氨基酸同时含酸性（—COOH）和碱性（—NH₂），是两性化合物：
- 与酸反应：—NH₂ + HCl → —NH₃⁺Cl⁻
- 与碱反应：—COOH + NaOH → —COONa + H₂O

**手性**：除甘氨酸（—H 两个 H 相同）外，α-氨基酸均含手性碳，天然存在 L-构型

### 性质 / 反应

| 反应 | 方程式 | 说明 |
|------|--------|------|
| 两性（与酸）| H₂N—CHR—COOH + HCl → ⁺H₃N—CHR—COO⁻Cl⁻ | — |
| 两性（与碱）| H₂N—CHR—COOH + NaOH → H₂N—CHR—COONa + H₂O | — |
| 成肽（缩合）| 2 H₂N—CHR—COOH → H₂N—CHR—CO—NH—CHR'—COOH + H₂O | 脱水缩合 |
| 水解 | 多肽 + H₂O → 氨基酸（逆反应）| 酸/碱/蛋白酶 |

**肽键的形成**（缩合）：
一个氨基酸的—COOH 与另一个的—NH₂ 脱水缩合：
—COOH + H—NH— → —CO—NH— + H₂O
产生**肽键**（—CO—NH—，酰胺键）

**二肽 → 三肽……→ 多肽 → 蛋白质**

### 典型例题

**例**：甘氨酸（H₂NCH₂COOH）与丙氨酸（H₂NCHCH₃COOH）缩合，可形成几种二肽？

解：两种氨基酸缩合有两个连接方向：
1. 甘氨酸—CO—NH—丙氨酸（甘氨酸 C 端连丙氨酸 N 端）
2. 丙氨酸—CO—NH—甘氨酸（丙氨酸 C 端连甘氨酸 N 端）
共 **2 种**二肽（Gly-Ala 和 Ala-Gly，不同序列）。

**例**：某二肽含有 2 个氨基酸残基，二肽中肽键数目和脱去水分子数。

解：n 个氨基酸形成的多肽含 (n-1) 个肽键，脱去 (n-1) 个水分子。
二肽（n=2）：**1 个肽键**，脱去 **1 个 H₂O**。

### 易错点

- 甘氨酸（H₂NCH₂COOH）的 α-碳连 2 个 H，**不是手性碳**（其余 α-氨基酸均有手性碳）
- 氨基酸既有酸性又有碱性（两性），不要认为含—COOH 只有酸性
- n 个氨基酸成肽，失去 (n-1) 个 H₂O，形成 (n-1) 个肽键
