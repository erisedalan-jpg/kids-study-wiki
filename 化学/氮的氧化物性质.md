# 氮的氧化物性质

> **一句话**：氮的氧化物是由氮、氧两种元素组成的化合物，常见的有 NO、NO₂、N₂O₄、N₂O 等，其性质（尤其是氧化还原性和与水/碱的反应）是高考元素化学的核心考点。
> **English**: Nitrogen oxides are compounds composed of nitrogen and oxygen, and their redox behavior and reactions with water/alkali are central topics in Gaokao element chemistry.

## 🎓 考点精讲

**定义与核心转化链**
氮的氧化物涉及多种价态，高考主干锁定在 NO、NO₂、N₂O₄ 的相互转化及与 HNO₃ 的关联：

1. **NO ⇌ NO₂**：`2NO + O₂ = 2NO₂`（无色 NO 遇空气迅速变红棕色，可用于检验 NO）
2. **NO₂ ⇌ N₂O₄**：`2NO₂(g) ⇌ N₂O₄(g)  ΔH<0`（温度越低，N₂O₄ 越多，颜色越浅）
3. **工业制硝酸关键步骤**：`3NO₂ + H₂O = 2HNO₃ + NO`（NO 循环氧化吸收）

**关键方法——氮氧化物溶于水的计算**
将 NO、NO₂、O₂ 混合气体溶于水，最终剩余气体的判断与体积计算是经典定量题型。通用解法为叠加方程组：
- NO₂ + NO +  H₂O 体系：`NO₂ + NO + H₂O = 2HNO₂`？（此式不存在，实际 3NO₂+ H₂O → 2HNO₃+NO，NO 与水不反应但充当惰性成分，**易错**）。正确思路：先让 NO₂ 与水反应生成 NO，再合并总 NO，看是否通入 O₂ 转化。
- **通式技巧**：任意 NO 与 NO₂ 混合物，在 O₂ 存在下完全转化为 HNO₃ 的化学计量关系：
  `4NO + 3O₂ + 2H₂O = 4HNO₃`
  `4NO₂ + O₂ + 2H₂O = 4HNO₃`
  利用电子守恒（N 升价总数 = O₂ 降价总数）可一步求解，无需分步。

**典型应用**
- **喷泉实验**：NO₂ 溶于水发生 `3NO₂ + H₂O = 2HNO₃ + NO`，气体体积减至 1/3，形成喷泉（因气体减少导致的压强差）。
- **尾气吸收**：工业上用 NaOH 吸收 NO 和 NO₂ 混合气（等物质的量）：`NO + NO₂ + 2NaOH = 2NaNO₂ + H₂O`。若 NO₂ 过量则生成 NaNO₃ 和 NaNO₂；NO 单独不与碱反应，须先氧化为 NO₂。

## ⚠️ 高频易错点

- **NO₂ 溶于水的反应本质不是氧化还原的孤例**：学生常误以为 NO₂ 与水只发生复分解。实际上 `3NO₂ + H₂O = 2HNO₃ + NO` 中 NO₂ 发生歧化（N 从+4 升至+5、降至+2），设计方程式的电子守恒计算时必须识别。
- **收集方法混淆**：NO 只能用排水法（与空气反应）；NO₂ 只能用向上排空气法（与水反应且有较强氧化性/毒性，不宜排水）。**魔鬼细节**——NO₂ 在标况下为液态（沸点 21°C），涉及 NA 类选择题常考“标况下 22.4 L NO₂”为陷阱。
- **隐含条件 N₂O₄**：NO₂ 体系中**永远存在** N₂O₄ 二聚平衡。涉及“通入一定量 NO₂ 气体”计算反应热或平均摩尔质量时，不可忽略平衡移动导致的实测值偏差。
