# 灭火

> **一句话**：灭火的实质是破坏燃烧的三个必要条件之一，方法包括：移走可燃物、隔绝氧气、使温度降低到着火点以下——三种方法可单独或联合使用。
> **English**: Extinguishing a fire means eliminating at least one of the three necessary combustion conditions: removing the fuel, cutting off oxygen supply, or cooling the temperature below the ignition point — these methods can be used alone or in combination.

## 📚 给 6-12 岁（自读版）

火是有用的工具，但失控的火很危险！了解如何**灭火**和**防火**，是每个人都必须掌握的知识。

**灭火的三种方法**

燃烧需要三个条件（可燃物、O₂、着火点），灭火就是破坏其中一个！

**方法一：移走可燃物**

- 炒菜锅着火 → 快速移开燃气灶，关闭燃气阀门（移走燃料）
- 森林火灾 → 在火势前方开辟"防火带"（砍掉树木，移走可燃物）
- 吹灭蜡烛 → 吹走烛焰附近的蜡蒸气（移走气态可燃物）

**方法二：隔绝氧气**

- 用锅盖盖住着火的油锅 → 隔绝 O₂
- 泡沫灭火器喷出泡沫覆盖燃烧物 → 隔绝 O₂
- 沙子覆盖小火 → 隔绝 O₂
- CO₂ 灭火器喷出 CO₂ → 既降温又隔 O₂

**方法三：降温到着火点以下**

- 水是最常用的灭火剂：水蒸发吸热（降温）+ 水蒸气隔绝 O₂（双重效果）
- 酒精着火 → 用水扑灭（降温冷却）

**特殊火灾不能用水！**

| 火灾类型 | 不能用水原因 | 正确方法 |
|---------|------------|---------|
| 油锅/汽油 | 油浮在水上，水让油飞溅扩大火势 | 锅盖/干粉/沙土 |
| 电器/电线 | 水导电，灭火者触电 | 先切电，再用干粉或 CO₂ |
| 金属钠/钾 | 与水剧烈反应，爆炸 | 沙土覆盖 |
| 图书档案 | 水会损坏文件 | CO₂ 灭火器 |

**口诀**：灭火三招记心中，移物隔氧降温度，油锅电器别用水，正确灭火保安全！

## 🎓 给 12+（进阶版）

### 定义

灭火（fire extinguishing）是通过物理或化学手段破坏燃烧三要素（可燃物、氧化剂、温度≥着火点）中的至少一个，使燃烧反应无法继续进行的过程。灭火方法的选择取决于可燃物的种类、火灾规模及现场条件。

### 性质 / 反应 / 公式

**三种灭火原理及实例**

| 灭火原理 | 方法示例 | 适用场景 |
|---------|---------|---------|
| 移走可燃物 | 关闭燃气阀、开防火带、转移可燃材料 | 气体火灾、森林火灾 |
| 隔绝氧气 | 锅盖、沙土、CO₂ 灭火器、泡沫灭火器 | 油类火灾、小面积火灾 |
| 降温至着火点以下 | 水（吸热冷却）、湿布 | 固体可燃物火灾 |

**常用灭火器对比**

| 灭火器类型 | 主要成分 | 灭火原理 | 适用火灾 | 不适用 |
|-----------|---------|---------|---------|--------|
| 水基型 | 水 | 冷却（降温）+ 隔绝 | 固体火灾（A 类）| 油类、电器 |
| 干粉型 | NaHCO₃ 等 | 隔绝 O₂ + 抑制链式反应 | B 类（液体）、C 类（气体）| 精密仪器 |
| CO₂ 型 | 液态 CO₂ | 隔绝 O₂ + 冷却 | 电器（E 类）、图书 | 金属钠钾 |
| 泡沫型 | NaHCO₃ + Al₂(SO₄)₃ | 隔绝 O₂ | 油类火灾（B 类）| 电器（导电）|

**泡沫灭火器化学反应**

$$Al_2(SO_4)_3 + 6NaHCO_3 = 2Al(OH)_3\downarrow + 3Na_2SO_4 + 6CO_2\uparrow$$

（两瓶混合，Al³⁺ 与 HCO₃⁻ 反应，生成 CO₂ 和 Al(OH)₃ 泡沫，覆盖燃烧物）

**火灾分类与灭火方法**

| 火灾类型 | 定义 | 推荐灭火方法 |
|---------|------|------------|
| A 类 | 固体可燃物（木材、纸张）| 水、泡沫 |
| B 类 | 液体或可熔固体（汽油、油漆）| 干粉、CO₂、泡沫 |
| C 类 | 气体（天然气、液化气）| 干粉，先关闭气源 |
| D 类 | 可燃金属（钠、钾、镁）| 专用干沙，绝对不用水 |
| E 类（电气）| 带电设备 | CO₂、干粉（先断电）|
| F 类 | 烹饪油脂 | 专用 F 类灭火器，锅盖 |

### 典型实验 / 例题

**例题**：下列灭火方法正确的是（ ）

A. 电器着火，立即用水浇灭  
B. 图书档案室着火，用 CO₂ 灭火器扑灭  
C. 炒菜时油锅着火，立即向锅中倒水  
D. 酒精灯洒出酒精着火，用嘴吹灭

**解析**：  
A 错：水导电，电器着火不能用水；  
B 对：CO₂ 灭火器不留残余物，适合图书档案；  
C 错：油密度比水小，水会使油四溅扩大火势，正确做法是盖锅盖；  
D 错：吹气可能使燃烧酒精溅散，应用湿布覆盖或用灭火器。  
选 **B**。

### 易错点

1. 灭火的本质是**破坏燃烧条件**，不是"消灭"火——温度低于着火点，火自然熄灭。
2. **不能降低着火点**——着火点是物质固有性质，灭火只能使温度低于已有的着火点。
3. 油类火灾**绝对不能用水**——油的密度比水小，会浮在水上继续燃烧，且可能飞溅造成"爆燃"。
4. 电器着火必须**先断电再灭火**——带电设备用水极危险；首选 CO₂ 或干粉灭火器。
