# 焓变

> **一句话**：焓变（ΔH）是恒压条件下化学反应的热量变化，等于生成物焓（H）减去反应物焓，ΔH<0为放热，ΔH>0为吸热。
> **English**: Enthalpy change (ΔH) is the heat change of a chemical reaction at constant pressure, equal to the enthalpy of products minus that of reactants; ΔH < 0 means exothermic, ΔH > 0 means endothermic.

---

## 🎓 高中进阶

### 定义

**焓（H）**：热力学状态函数，代表物质在恒压条件下的热含量（内能+压力体积乘积）。

**焓变（ΔH）**：
$$\Delta H = H_\text{生成物} - H_\text{反应物}$$

- **ΔH < 0**：生成物焓 < 反应物焓，反应**放热**
- **ΔH > 0**：生成物焓 > 反应物焓，反应**吸热**
- 单位：**kJ/mol**（千焦每摩尔）

### 热化学方程式书写规范

热化学方程式 = 化学方程式 + ΔH

**规范要求**：
1. 各物质注明状态：(g)气态、(l)液态、(s)固态、(aq)溶液
2. 化学计量数可以是分数
3. ΔH的值对应方程式中的计量数
4. ΔH写在方程式末尾，带正负号和单位

**示例**：

$$\text{H}_2(g) + \frac{1}{2}\text{O}_2(g) \rightarrow \text{H}_2\text{O}(l) \quad \Delta H = -285.8 \text{ kJ/mol}$$

$$\text{CaCO}_3(s) \rightarrow \text{CaO}(s) + \text{CO}_2(g) \quad \Delta H = +178.2 \text{ kJ/mol}$$

### 焓变的计算方法

**方法1：键能计算**
$$\Delta H = \sum(\text{反应物键能}) - \sum(\text{生成物键能})$$

**方法2：盖斯定律（间接计算）**

$$\Delta H_\text{总} = \Delta H_1 + \Delta H_2 + \cdots$$

**方法3：生成焓法**
$$\Delta H = \sum \Delta H_f(\text{生成物}) - \sum \Delta H_f(\text{反应物})$$

### 标准生成焓（部分数据）

| 物质 | 标准生成焓 ΔHf° (kJ/mol) |
|------|------------------------|
| H₂O(l) | -285.8 |
| CO₂(g) | -393.5 |
| CH₄(g) | -74.8 |
| NH₃(g) | -46.1 |
| HCl(g) | -92.3 |

### ΔH与温度、压力

在高中阶段，焓变的温度和压力依赖性较小，通常视为不变（标准条件下讨论）。

### 典型例题

**例1**：已知：
- ① H₂(g) + ½O₂(g) → H₂O(g), ΔH₁ = -241.8 kJ/mol
- ② H₂O(g) → H₂O(l), ΔH₂ = -44.0 kJ/mol

求：H₂(g) + ½O₂(g) → H₂O(l)，ΔH = ?

**分析**（盖斯定律）：目标 = ① + ②：
ΔH = ΔH₁ + ΔH₂ = -241.8 + (-44.0) = **-285.8 kJ/mol**（正确值）

**例2**：2H₂(g) + O₂(g) → 2H₂O(l), ΔH = -571.6 kJ/mol
与H₂(g) + ½O₂(g) → H₂O(l), ΔH = -285.8 kJ/mol
两者有什么关系？

**分析**：前者是后者计量数的2倍，所以ΔH也是2倍关系。
-571.6 = 2×(-285.8)，完全一致。

### 易错点
- 焓变ΔH与化学计量数有关，同一反应方程式计量数×2，ΔH也×2
- 热化学方程式中**必须**标注物质状态，H₂O(g)与H₂O(l)的ΔH不同
- ΔH < 0是放热（不是负焓），不要因为负号而搞混
