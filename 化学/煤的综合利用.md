# 煤的综合利用

> **一句话**：煤的综合利用包括干馏（隔绝空气加热）、气化和液化三种方式，可从煤中获得焦炭、煤焦油、煤气等多种有价值的化工产品，而不只是直接燃烧。
> **English**: The comprehensive utilization of coal includes dry distillation (heating without air), gasification, and liquefaction, extracting valuable products like coke, coal tar, and coal gas—rather than simply burning coal directly.

---

## 🎓 高中进阶

### 定义
**煤的综合利用**：将煤作为化工原料，通过干馏、气化、液化等工艺，将煤中的有机物转化为多种有用化学品，提高煤的利用效率和附加值。

**背景**：煤是重要化石能源，直接燃烧效率低且污染大；综合利用可获得更多有价值产品，减少浪费。

### 煤的组成

煤是复杂的有机混合物，主要含：
- **C**（主体，约60-90%）
- **H**（约4-5%）
- **O、N、S**（少量杂原子）
- 无机矿物质（灰分）

### 煤的三种主要加工方法

**一、煤的干馏（Dry Distillation）**

**定义**：将煤在**隔绝空气**的条件下加热，使煤中的有机物分解，得到各种产物的过程。

**产物**：

| 产物 | 说明 |
|------|------|
| 焦炭（Coke）| 固态，主要是C，用于高炉炼铁、制电极 |
| 煤焦油（Coal Tar）| 液态，复杂有机混合物，含苯、甲苯、萘等芳香烃 |
| 粗苯 | 液态，含苯、甲苯等 |
| 氨水（NH₃·H₂O）| 含氨（NH₃），可制化肥 |
| 煤气 | 气态，含H₂、CO、CH₄等，可燃气体 |

**主要用途**：
- 焦炭：还原剂（炼铁）、电极材料
- 煤焦油：提取苯、甲苯、萘→化工原料、染料
- 煤气：城市燃气（历史上）

**二、煤的气化**

**定义**：将煤（固态）转化为气态可燃气体的过程。

**主要反应**：

$$C + H_2O \xrightarrow{\text{高温}} CO + H_2\text{（水煤气，合成气）}$$

$$C + O_2 \xrightarrow{\text{不完全燃烧}} 2CO\text{（发生炉煤气）}$$

**水煤气用途**：合成氨（N₂+3H₂→2NH₃）、合成甲醇等

**三、煤的液化**

**定义**：将煤转化为液态燃料（类似石油）的过程。

**方法**：
- **直接液化**：高温高压下，煤与H₂反应，生成类似石油的液态碳氢化合物
- **间接液化**（费托合成）：先气化为CO+H₂，再合成液态烃：
$$nCO + (2n+1)H_2 \xrightarrow{\text{催化剂}} C_nH_{2n+2} + nH_2O$$

**意义**：将煤转化为柴油、汽油等液态燃料，可替代石油。

### 煤综合利用的意义

| 方面 | 意义 |
|------|------|
| 能源效率 | 比直接燃烧效率高（提取高附加值产品）|
| 环保 | 减少SO₂、粉尘（干馏/气化处理更洁净）|
| 化工原料 | 苯、甲苯、萘等重要化工原料来自煤焦油 |
| 能源安全 | 减少对石油依赖 |

### 典型例题

**例1**：煤的干馏和煤的燃烧有何本质区别？

**分析**：
- **燃烧**：煤与O₂反应，化学能转化为热能，产物主要为CO₂、H₂O（能量利用）
- **干馏**：隔绝空气加热，有机物热分解，得到焦炭、煤焦油、煤气等（物质转化）
- 干馏是化工过程（物理化学），燃烧是能源利用（氧化还原）

**例2**：焦炭在高炉炼铁中起什么作用？写出相关方程式。

**分析**：焦炭有两个作用：
1. **燃料**（提供热量）：C + O₂ → CO₂
2. **还原剂**（间接还原铁矿石）：CO₂ + C → 2CO；3CO + Fe₂O₃ → 2Fe + 3CO₂

### 易错点
- 干馏≠蒸馏：干馏是化学变化（有机物裂解），蒸馏是物理变化（沸点不同分离）
- 煤干馏的条件必须是隔绝空气（否则煤直接燃烧）
- 水煤气（CO+H₂）与天然气（CH₄）不同，水煤气可燃但有毒（CO）
