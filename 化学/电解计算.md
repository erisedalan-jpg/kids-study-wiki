# 电解计算

> **一句话**：电解计算是基于**电子守恒**，利用电极反应式中转移电子数与产物物质的量之间的定量关系进行化学计量的方法。
> **English**: Electrolysis calculation is a stoichiometric method based on the conservation of electrons, using the quantitative relationship between the number of electrons transferred in electrode reactions and the amount of products.

## 🎓 考点精讲

### 定义与原理

电解计算的核心是**电子守恒**：电解池中，阳极失去的电子总数 = 阴极得到的电子总数 = 外电路通过的电量。

定量桥梁为：

$$n(e^-) = \frac{Q}{F} = \frac{It}{96500}$$

其中 $n(e^-)$ 为转移电子的物质的量 (mol)，$Q$ 为电量 (C)，$I$ 为电流强度 (A)，$t$ 为时间 (s)，$F \approx 96500\ \text{C·mol}^{-1}$ 为法拉第常数。

对任意电极反应，产物物质的量与电子转移数成正比。例如：

- 阴极：$\text{Cu}^{2+} + 2e^- \rightarrow \text{Cu}$，生成 1 mol Cu 需 2 mol $e^-$
- 阳极：$2\text{Cl}^- - 2e^- \rightarrow \text{Cl}_2\uparrow$，生成 1 mol $\text{Cl}_2$ 需 2 mol $e^-$

若阴、阳两极串联，通过各电极的电子物质的量相同，可直接列比例式。

### 关键方法

电解计算的标准步骤：

1. **写电极反应式**：先判断阴、阳极的放电顺序，写出两极半反应。
2. **建立电子守恒关系**：设转移电子 $n(e^-)$，根据半反应写出各物质与 $n(e^-)$ 的关系。
3. **利用已知量列比例**：
   - 已知电流和时间 → $n(e^-) = \frac{It}{96500}$
   - 已知某产物质量/体积/浓度变化 → 反推 $n(e^-)$
4. **串联电解池**（多池串联）：各池通过电子数相同，可直接用 $n(e^-)$ 统一计算各池产物。

### 典型应用（高考设问方式）

**题型一：已知电流、时间，求产物的质量或体积**  
例题（格式）：用惰性电极电解 $\text{CuSO}_4$ 溶液，电流 2 A，通电 1930 s。求阴极析出铜的质量。  
解：$n(e^-) = \frac{2 \times 1930}{96500} = 0.04\ \text{mol}$。阴极：$\text{Cu}^{2+} + 2e^- \rightarrow \text{Cu}$，$n(\text{Cu}) = \frac{0.04}{2} = 0.02\ \text{mol}$，$m(\text{Cu}) = 0.02 \times 64 = 1.28\ \text{g}$。

**题型二：两极产物量的互推**  
电解 $\text{NaCl}$ 溶液（惰性电极），若阴极生成 0.1 mol $\text{H}_2$，则阳极生成 $\text{Cl}_2$ 的物质的量为？  
解：阴极：$2\text{H}_2\text{O} + 2e^- \rightarrow \text{H}_2\uparrow + 2\text{OH}^-$，阳极：$2\text{Cl}^- - 2e^- \rightarrow \text{Cl}_2\uparrow$。两极电子守恒，$n(\text{H}_2):n(\text{Cl}_2) = 1:1$，故 $n(\text{Cl}_2) = 0.1\ \text{mol}$。

**题型三：串联电解池计算**  
三个电解池串联，分别盛 $\text{CuSO}_4$、$\text{AgNO}_3$、$\text{NaCl}$ 溶液，通电一段时间后，第一个池阴极析出 3.2 g Cu。求另两池阴极或阳极产物质量。  
解：$n(\text{Cu}) = 0.05\ \text{mol}$ → $n(e^-) = 0.10\ \text{mol}$。串联各池 $n(e^-)$ 相同。第二池：$\text{Ag}^+ + e^- \rightarrow \text{Ag}$，$n(\text{Ag}) = 0.10\ \text{mol}$，$m(\text{Ag}) = 10.8\ \text{g}$。第三池阴极：$2\text{H}_2\text{O} + 2e^- \rightarrow \text{H}_2\uparrow + 2\text{OH}^-$，$n(\text{H}_2) = 0.05\ \text{mol}$。

## ⚠️ 高频易错点

- **混淆串联与并联**：电解池串联时，通过各池的**电子物质的量相等**；并联时电流分流，各池 $n(e^-)$ 按电阻分配，除非题目说明否则不可直接等同。
- **忽视气体摩尔体积的使用条件**：计算气体体积时，若题给条件非标准状况（STP），不可盲目使用 22.4 L/mol。
- **水电离产生的 $\text{H}^+$、$\text{OH}^-$ 参与电极反应被遗漏**：电解含氧酸/强碱/活泼金属盐溶液时，实际放电的可能是水中的 $\text{H}^+$ 或 $\text{OH}^-$，需按放电顺序正确书写半反应。
- **阴、阳极产物量的混淆**：未将阴阳极反应通过电子守恒关联，误将两极产物单独计算而不考虑电子转移数的匹配。
