# 碱

> **一句话**：碱是一类电离时产生的阴离子全部为氢氧根离子（OH⁻）的化合物，具有涩味、能使石蕊变蓝、能与酸中和等共同性质。
> **English**: A base is a compound that produces hydroxide ions (OH⁻) as the only anion upon ionization, characterized by bitter taste and the ability to turn litmus blue.

---

## 📚 给 6-12 岁（自读版）

碱和酸是"对立面"。酸是酸的，而碱的口感是**苦涩的**（不能随便尝！）。碱性的东西比酸更危险，强碱能腐蚀皮肤。

**常见的碱：**
- **氢氧化钠（NaOH）**：也叫火碱、烧碱，工业中常用，能溶解油脂（疏通下水道的产品就含 NaOH）
- **氢氧化钙（Ca(OH)₂）**：也叫熟石灰、消石灰，用于建筑抹墙、改良土壤
- **氨水（NH₃·H₂O）**：弱碱，气味很强，有时用来清洁

**碱的共同特点：**
1. 有**涩味**（不能随便尝！）
2. 能让**石蕊变蓝**（红色石蕊变蓝）
3. 酚酞遇碱变**红**
4. 能和**酸**发生中和反应
5. 强碱（NaOH 等）能腐蚀皮肤

**碱有什么用？**
- 做肥皂：用油脂和 NaOH 反应
- 改良酸性土壤：撒石灰（Ca(OH)₂）
- 制造水泥：用石灰石煅烧

## 🎓 给 12+（进阶版）

### 定义
**阿伦尼乌斯碱（初中定义）**：在水溶液中电离，产生的阴离子全部为 OH⁻ 的化合物。

$$\text{NaOH} \rightarrow \text{Na}^+ + \text{OH}^-$$
$$\text{Ca(OH)}_2 \rightarrow \text{Ca}^{2+} + 2\text{OH}^-$$

**碱的分类：**

| 分类 | 说明 | 举例 |
|------|------|------|
| 强碱 | 完全电离 | NaOH、KOH、Ba(OH)₂ |
| 弱碱 | 部分电离 | Ca(OH)₂（微溶）、NH₃·H₂O |
| 可溶性碱 | 溶于水 | NaOH、KOH |
| 微溶碱 | 微溶于水 | Ca(OH)₂ |
| 难溶碱 | 难溶于水 | Fe(OH)₃、Cu(OH)₂ |

### 碱的通性（化学反应）

与酸反应（中和反应）：

$$\text{NaOH} + \text{HCl} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$

与酸性氧化物反应：

$$2\text{NaOH} + \text{CO}_2 \rightarrow \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$$

$$2\text{NaOH} + \text{SO}_2 \rightarrow \text{Na}_2\text{SO}_3 + \text{H}_2\text{O}$$

与某些盐反应：

$$2\text{NaOH} + \text{CuSO}_4 \rightarrow \text{Cu(OH)}_2\downarrow + \text{Na}_2\text{SO}_4$$

$$3\text{NaOH} + \text{FeCl}_3 \rightarrow \text{Fe(OH)}_3\downarrow + 3\text{NaCl}$$

### 典型实验 / 例题
**例题**：下列物质属于碱的是（）
A. CO₂　B. Ca(OH)₂　C. Na₂CO₃　D. HCl

答案：B（Ca(OH)₂ 电离产生 OH⁻；CO₂ 是酸性氧化物；Na₂CO₃ 是盐；HCl 是酸）

### 易错点
- 碱必须含有 **OH⁻**，氨水（NH₃·H₂O）也是碱（产生 OH⁻）
- Na₂CO₃（纯碱）是**盐**，不是碱（虽然溶液显碱性）
- NaOH 遇空气会**潮解**并吸收 CO₂（变质），需密封保存
- Ca(OH)₂ 的溶解度随温度升高而**减小**（特殊）
