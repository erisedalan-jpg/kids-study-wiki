# 碱的通性

> **一句话**：碱的通性是所有可溶性碱共有的化学性质，包括使指示剂变色、与酸性氧化物反应、与酸发生中和反应、与某些盐反应四大类。
> **English**: The common properties of bases include changing indicator colors, reacting with acidic oxides, neutralizing acids, and reacting with certain salts.

---

## 📚 给 6-12 岁（自读版）

就像酸有共同的"脾气"，碱也有自己的共同特点，叫**碱的通性**。

**碱的四大通性：**

1. **使指示剂变色**
   - 红色石蕊→变蓝（碱使石蕊变蓝）
   - 无色酚酞→变红（碱使酚酞变红）

2. **和酸性氧化物反应**
   - 氢氧化钠（NaOH）+ 二氧化碳 → 碳酸钠 + 水
   - 这就是 NaOH 溶液能吸收 CO₂、SO₂ 的原因

3. **和酸发生中和反应**
   - 碱 + 酸 → 盐 + 水
   - 这是碱最重要的性质

4. **和某些盐反应（生成沉淀）**
   - 氢氧化钠 + 硫酸铜 → 氢氧化铜蓝色沉淀 + 硫酸钠
   - 化验血液检验血糖，就用到了类似反应

**注意！** 碱的通性只适用于**可溶性碱**（如 NaOH、Ca(OH)₂、KOH）。
不溶性碱（如 Cu(OH)₂、Fe(OH)₃）只有部分通性。

## 🎓 给 12+（进阶版）

### 定义
碱的通性是碱溶液中 OH⁻ 的化学性质，适用于所有可溶性碱。

### 四大通性及方程式

**1. 使酸碱指示剂变色**
- 红色石蕊→蓝色
- 无色酚酞→红色

**2. 与酸性氧化物反应，生成盐和水：**

$$2\text{NaOH} + \text{CO}_2 \rightarrow \text{Na}_2\text{CO}_3 + \text{H}_2\text{O}$$

$$2\text{NaOH} + \text{SO}_2 \rightarrow \text{Na}_2\text{SO}_3 + \text{H}_2\text{O}$$

$$\text{Ca(OH)}_2 + \text{CO}_2 \rightarrow \text{CaCO}_3\downarrow + \text{H}_2\text{O}$$

**3. 与酸中和，生成盐和水：**

$$\text{NaOH} + \text{HCl} \rightarrow \text{NaCl} + \text{H}_2\text{O}$$

$$2\text{NaOH} + \text{H}_2\text{SO}_4 \rightarrow \text{Na}_2\text{SO}_4 + 2\text{H}_2\text{O}$$

$$\text{Ca(OH)}_2 + 2\text{HCl} \rightarrow \text{CaCl}_2 + 2\text{H}_2\text{O}$$

**4. 与某些盐反应，生成新碱和新盐：**

$$2\text{NaOH} + \text{CuSO}_4 \rightarrow \text{Cu(OH)}_2\downarrow + \text{Na}_2\text{SO}_4$$

（蓝色沉淀——鉴别 Cu²⁺）

$$3\text{NaOH} + \text{FeCl}_3 \rightarrow \text{Fe(OH)}_3\downarrow + 3\text{NaCl}$$

（红褐色沉淀——鉴别 Fe³⁺）

$$2\text{NaOH} + \text{MgCl}_2 \rightarrow \text{Mg(OH)}_2\downarrow + 2\text{NaCl}$$

（白色沉淀——鉴别 Mg²⁺）

### 典型实验 / 例题
**例题**：下列不属于碱的通性的是（）
A. 使红色石蕊变蓝　B. 与酸发生中和反应　C. 与活泼金属反应放出氢气　D. 与硫酸铜溶液反应生成蓝色沉淀

答案：C（碱的通性中没有与金属反应，只有 NaOH 等与 Al 特殊反应，不是通性）

### 易错点
- 碱的通性本质是 **OH⁻ 的性质**
- 不溶性碱（如 Cu(OH)₂）没有第 2、4 两条通性
- 碱与酸性氧化物反应是碱的第二条通性，不是中和（中和是碱与酸）
