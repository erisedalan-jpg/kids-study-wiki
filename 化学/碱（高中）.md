# 碱（高中）

> **一句话**：电离时产生的阴离子全部为氢氧根离子（OH⁻）的化合物。
> **English**: A compound that ionizes in solution to produce hydroxide ions (OH⁻) as the only anion.

---

## 🎓 高中进阶

### 定义

碱是溶于水后**电离出的阴离子全部为OH⁻**的化合物。碱的水溶液呈碱性（pH > 7）。

$$\text{NaOH} \rightarrow \text{Na}^+ + \text{OH}^-$$
$$\text{Ca(OH)}_2 \rightarrow \text{Ca}^{2+} + 2\text{OH}^-$$

### 碱的分类

| 分类依据 | 类别 | 示例 |
|---------|------|------|
| 强弱 | 强碱 / 弱碱 | NaOH、KOH、Ca(OH)₂ / NH₃·H₂O、Mg(OH)₂ |
| 溶解性 | 可溶 / 微溶 / 难溶 | NaOH / Ca(OH)₂ / Mg(OH)₂ |
| 价态 | 一元碱 / 二元碱 | NaOH / Ca(OH)₂ |

### 常见强碱（高中必记）

| 碱 | 化学式 | 特点 |
|----|-------|------|
| 氢氧化钠（烧碱）| NaOH | 强碱，强腐蚀性，潮解 |
| 氢氧化钾 | KOH | 强碱 |
| 氢氧化钙（熟石灰）| Ca(OH)₂ | 微溶强碱 |
| 氢氧化钡 | Ba(OH)₂ | 可溶强碱 |

**弱碱**：NH₃·H₂O（氨水）、Mg(OH)₂、Al(OH)₂、Fe(OH)₃

### 碱的通性

1. 与指示剂：使紫色石蕊变蓝，使无色酚酞变红
2. 与酸中和：NaOH + HCl → NaCl + H₂O
3. 与酸性氧化物：2NaOH + CO₂ → Na₂CO₃ + H₂O
4. 与某些盐：2NaOH + CuSO₄ → Cu(OH)₂↓ + Na₂SO₄
5. 与铝反应（特殊）：2NaOH + 2Al + 2H₂O → 2NaAlO₂ + 3H₂↑

### 典型例题

**例1**：下列属于强碱的是（ ）
A. NH₃·H₂O　B. Mg(OH)₂　C. Al(OH)₃　D. Ba(OH)₂

**分析**：NH₃·H₂O是弱碱；Mg(OH)₂、Al(OH)₃是弱碱（难溶）；Ba(OH)₂是强碱（完全电离）。**答案：D**

**例2**：NaOH与Al(OH)₃反应的离子方程式：

$$\text{Al(OH)}_3 + \text{OH}^- \rightarrow \text{AlO}_2^- + 2\text{H}_2\text{O}$$

### 易错点 / 拓展

- Ca(OH)₂微溶但属**强碱**（溶解部分完全电离）
- NH₃·H₂O（氨水）属弱碱，不写成NH₄OH
- Al(OH)₃是**两性氢氧化物**，既能与酸又能与强碱反应
- NaOH固体暴露在空气中会吸收CO₂变质：2NaOH + CO₂ → Na₂CO₃ + H₂O
