# 碳的化学性质

> **一句话**：碳在常温下化学性质稳定，但在高温下活泼，能与氧气发生完全或不完全燃烧，还能将金属氧化物（如CuO、Fe₂O₃）还原为金属，是重要的工业还原剂。
> **English**: Carbon is chemically stable at room temperature but reactive at high temperatures; it undergoes complete or incomplete combustion with oxygen and reduces metal oxides (e.g., CuO, Fe₂O₃) to metals, making it an important industrial reducing agent.

---

## 📚 给 6-12 岁（自读版）

碳（C）是一种神奇的元素——在常温下几乎不和任何东西反应，可是一旦温度升高，它就变得非常"活跃"！

**常温下：化学性质稳定**

碳在常温下不易氧化，也不易与其他物质反应。这就是为什么古代用墨（碳）写字、作画，保存几百年甚至上千年都不褪色——因为碳的性质稳定，颜料不会变质！

**高温下：化学性质活泼**

1. **完全燃烧**（氧气充足）：碳和足量氧气反应，生成 CO₂

$$C + O_2 \overset{\text{点燃}}{=} CO_2$$

现象：发出明亮白光，生成无色气体（CO₂），石灰水变浑浊。

2. **不完全燃烧**（氧气不足）：碳和少量氧气反应，生成有毒 CO

$$2C + O_2 \overset{\text{点燃（不足量）}}{=} 2CO$$

这就是煤炉通风不好时产生 CO（煤气）的原因！

3. **还原 CO₂**（超高温）：碳和 CO₂ 反应，生成 CO

$$C + CO_2 \overset{\text{高温}}{=} 2CO$$

4. **还原金属氧化物**（高温）：碳把金属从氧化物中"拉"出来

$$2CuO + C \overset{\text{高温}}{=} 2Cu + CO_2\uparrow$$

$$2Fe_2O_3 + 3C \overset{\text{高温}}{=} 4Fe + 3CO_2\uparrow$$

**碳的两大化学性质总结**

| 性质 | 条件 | 应用 |
|------|------|------|
| 稳定性 | 常温 | 碳素墨水、碳黑颜料 |
| 可燃性 | 高温 + O₂ | 燃烧产热（发电、取暖）|
| 还原性 | 高温 | 冶金（炼铁、炼铜）|

**口诀**：碳常温稳定难反应，高温活泼燃烧还原，完全燃烧变 CO₂，氧不足时 CO 出！

## 🎓 给 12+（进阶版）

### 定义

碳（C）的化学性质包括：常温下高度稳定（由于 C-C 键能强，反应活化能高）；高温下具有较强的可燃性和还原性，能与氧气、金属氧化物及部分非金属氧化物发生反应。碳的化学性质是其所有同素异形体（金刚石、石墨、C60）共有的——尽管物理性质差异巨大，化学性质基本相同。

### 性质 / 反应 / 公式

**1. 稳定性（常温）**

碳在常温下不与 O₂、H₂O、酸（稀盐酸/稀硫酸）发生明显反应。这是由于 C 的原子半径小、电负性适中，常温下反应活化能较高。历史上木炭、骨炭在墓葬中保存数千年不变。

**2. 可燃性（高温 + O₂）**

| 反应 | 化学方程式 | 条件 | 现象 |
|------|-----------|------|------|
| 完全燃烧 | $C + O_2 \overset{\text{点燃}}{=} CO_2$ | O₂ 充足 | 白光，CO₂ 生成 |
| 不完全燃烧 | $2C + O_2 \overset{\text{点燃（不足）}}{=} 2CO$ | O₂ 不足 | 产生有毒 CO |

**3. 还原性（高温）**

碳在高温下是较强的还原剂，可将金属氧化物还原为金属：

| 被还原物 | 化学方程式 | 条件 | 工业应用 |
|---------|-----------|------|---------|
| CuO（氧化铜）| $2CuO + C \overset{\text{高温}}{=} 2Cu + CO_2\uparrow$ | 高温 | 早期冶铜 |
| Fe₂O₃（氧化铁）| $2Fe_2O_3 + 3C \overset{\text{高温}}{=} 4Fe + 3CO_2\uparrow$ | 高温 | 早期炼铁 |
| CO₂（二氧化碳）| $C + CO_2 \overset{\text{高温}}{=} 2CO$ | 高温 | 高炉中产 CO |
| ZnO（氧化锌）| $2ZnO + C \overset{\text{高温}}{=} 2Zn + CO_2\uparrow$ | 高温 | 冶锌 |

**碳与 CO 还原性的比较**

| 比较项目 | C（碳）| CO |
|---------|--------|-----|
| 状态 | 固态 | 气态 |
| 还原产物 | CO₂ 或 CO（视条件）| CO₂ |
| 操作 | 固-固接触，不均匀 | 气-固接触，均匀 |
| 毒性 | 无毒 | 有毒 |
| 工业现用 | 作为中间体（先生成 CO）| 高炉炼铁核心 |

注：工业上实际炼铁时，碳（焦炭）先转化为 CO，再由 CO 还原铁矿石——即碳的还原性最终通过 CO 体现。

**4. 高炉炼铁中碳的作用**

$$C + O_2 \overset{\text{点燃}}{=} CO_2 \quad \text{（底部鼓风燃烧）}$$

$$CO_2 + C \overset{\text{高温}}{=} 2CO \quad \text{（上移与 C 反应）}$$

$$Fe_2O_3 + 3CO \overset{\text{高温}}{=} 2Fe + 3CO_2 \quad \text{（CO 还原铁矿石）}$$

### 典型实验 / 例题

**例题**：下列关于碳的化学性质的说法，正确的是（ ）

A. 碳在常温下能与 O₂ 缓慢反应  
B. 碳的可燃性与还原性本质是相同的（碳失去电子被氧化）  
C. 碳只能将金属氧化物还原为金属，不能还原非金属氧化物  
D. 金刚石和石墨化学性质不同，所以燃烧产物不同

**解析**：
- A 错：碳常温下稳定，不易与 O₂ 反应。
- B 对：无论燃烧还是还原，碳均被氧化（C → CO₂ 或 CO），本质都是碳失去电子（升价）。
- C 错：碳能还原 CO₂（非金属氧化物）：$C + CO_2 \rightarrow 2CO$。
- D 错：金刚石和石墨均为纯碳，完全燃烧均生成 CO₂，化学性质相同。
答案选 **B**。

### 易错点

1. 碳的**稳定性是常温特征**，不代表碳"永远不反应"——高温下活泼。
2. 碳**不完全燃烧生成 CO**（有毒），不是 CO₂——通风不足时危险！
3. 碳的还原产物视条件而定：还原 CO₂ 时产物是 CO；还原金属氧化物时产物通常是 CO₂（碳过量时可能为 CO）。
4. 碳的还原性在工业上通常通过 CO 间接体现，而非碳直接与铁矿石接触。
