# 离子共存问题

> **一句话**：离子共存问题是指判断多种离子在同一溶液中是否能大量共存，其实质是离子间不发生生成沉淀、气体、弱电解质、络合物或发生氧化还原反应等反应。
> **English**: The ion coexistence problem concerns whether multiple ions can coexist in a single solution without undergoing reactions such as precipitation, gas evolution, formation of weak electrolytes, complexation, or redox reactions.

## 🎓 考点精讲

离子共存的判断依据是离子反应发生的条件。只要离子间能满足下列之一，便不能大量共存：

1. **生成沉淀**：如 $Ba^{2+}$ 与 $\text{SO}_4^{2-}$ 生成 $BaSO_4 \downarrow$，$Ag^+$ 与 $Cl^-$ 生成 $AgCl \downarrow$ （需记忆常见沉淀溶解性表）。
2. **生成气体**：如 $H^+$ 与 $\text{CO}_3^{2-}$、$\text{HCO}_3^-$、$\text{S}^{2-}$、$\text{SO}_3^{2-}$ 等易挥发弱酸的酸根离子反应。
3. **生成弱电解质**：$H^+$ 与 $OH^-$ 生成 $H_2O$；$H^+$ 与 $CH_3COO^-$、$F^-$ 等生成弱酸；$NH_4^+$ 与 $OH^-$ 生成 $NH_3\cdot H_2O$。
4. **氧化还原反应**：典型如 $Fe^{3+}$ 与 $I^-$、$S^{2-}$；$MnO_4^-$（酸性）与 $Cl^-$、$Fe^{2+}$ 等；$NO_3^-$（酸性条件下有强氧化性）与 $Fe^{2+}$、$I^-$ 等不能共存。
5. **双水解完全**：$Al^{3+}$ 与 $\text{CO}_3^{2-}$、$\text{HCO}_3^-$、$\text{S}^{2-}$、$\text{AlO}_2^-$；$Fe^{3+}$ 与 $\text{CO}_3^{2-}$、$\text{HCO}_3^-$ 等因互相促进水解至完全而不能共存。
6. **络合反应**：如 $Fe^{3+}$ 与 $SCN^-$ 生成 $Fe(SCN)_3$，$Ag^+$ 与 $NH_3\cdot H_2O$ 等。

**关键方法**：解题时遵循“一色、二性、三特殊、四反应”：
- **一色**：辨别溶液有色离子（$Cu^{2+}$ 蓝、$Fe^{3+}$ 棕黄、$Fe^{2+}$ 浅绿、$MnO_4^-$ 紫等），若题干要求“无色”则以上离子均不能存在。
- **二性**：判断溶液酸碱性（隐含条件如 pH、指示剂变色、水电离出的 $c(H^+)$、与铝反应生成氢气等）。
- **三特殊**：注意附加限制，如“澄清透明”并不代表无色；“与 Al 反应放出 $H_2$”可能为强酸或强碱溶液。
- **四反应**：逐对检验上述反应类型。

**高考典型设问**：常在选择题中以“下列各组离子在指定条件下一定能/不能大量共存”的形式出现，选项附限制条件，如“滴入 KSCN 显红色的溶液中”“由水电离出的 $c(H^+)=1\times 10^{-13}\ \text{mol/L}$ 的溶液中”等，考查学生对隐含酸碱性与氧化还原反应的综合判断。

## ⚠️ 高频易错点

- 忽略**隐含酸碱性**：如“水电离出的 $c(H^+)=10^{-13}\ \text{mol/L}$”抑制了水的电离，溶液可能为 $pH=1$ 的强酸或 $pH=13$ 的强碱，两种情况均需检验，易漏答其一。
- 混淆“澄清透明”与“无色”：“澄清”仅指无浑浊，可含有色离子；题干若要求“无色透明”则必须排除有色离子。
- 误判酸性条件下 $NO_3^-$ 的氧化性：在 $pH<7$ 时 $NO_3^-$ 有强氧化性，与 $Fe^{2+}$、$I^-$ 等不能共存；但在中性或碱性条件下不表现此性质，常因条件忽略而出错。
- 忽略络合反应：如 $Fe^{3+}$ 与 $SCN^-$、$C_6H_5O^-$ 等的络合，这类反应不沉淀也不氧化还原，容易被遗漏。
