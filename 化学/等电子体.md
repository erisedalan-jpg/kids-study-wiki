# 等电子体

> **一句话**：原子总数相同、价电子总数相同的分子或离子互称为等电子体，其电子结构和空间构型往往相似。
> **English**: Molecules or ions that have the same total number of atoms and the same total number of valence electrons are called isoelectronic species, often exhibiting similar electronic structures and spatial geometries.

## 🎓 考点精讲

**定义**：等电子体是指**原子总数**相同且**价电子总数**相同的分子或离子。注意，这里的“价电子总数”计算方式为：分子中 = 各原子价电子数之和；阴离子中 = 各原子价电子数之和 + 离子电荷数绝对值；阳离子中 = 各原子价电子数之和 - 离子电荷数。例如 $CO$ 与 $N_2$，原子数均为 $2$，价电子总数均为 $4+6=10$ 和 $5+5=10$，互为等电子体。

**关键方法与判定依据**：
高考中常要求利用**等电子原理**判断或推导未知分子的空间构型与杂化方式。标准步骤如下：
1. **数原子数**：确保二者原子总数相等。
2. **算价电子总数**：通式为原子价电子数加和 ± 电荷。
3. 若原子数、价电子总数均相等，则二者互为等电子体，通常具备相同的**空间构型**与**中心原子杂化方式**。例如 $SO_2$ 与 $O_3$ 价电子总数均为 $3 × 6 = 18$，原子数均为 $3$，构型相似（均为 $V$ 形、$sp^2$ 杂化）。

**典型应用**：
- **构型预测题**：给出 $CO_3^{2-}$，要求写出一种等电子体并预测构型。答案可为 $NO_3^-$，二者价电子总数均为 $4 + 6×3 + 2 = 24$（$NO_3^-$ 为 $5+6×3+1=24$），均为平面三角形、$sp^2$ 杂化。
- **性质推断题**：已知 $N_2O$ 与 $CO_2$ 为等电子体（原子数 $3$，价电子总数 $16$），推断 $N_2O$ 结构为直线形，中心 $N$ 原子 $sp$ 杂化，存在离域大 $\pi$ 键 $\Pi_3^4$。

## ⚠️ 高频易错点

- **只数原子数，忽略价电子总数必须相同**：例如 $CH_4$ 与 $SiH_4$ 原子数相同（均为 $5$），但 $C$ 低价电子数 $4$、$Si$ 也是 $4$，价电子总数相等，故互为等电子体；而 $CCl_4$ 与 $CF_4$ 虽然原子数相同，但卤族原子的价电子数均为 $7$，故价电子总数相等，互为等电子体。错误常出现在将 $NH_3$ 与 $H_3O^+$（价电子总数均为 $5+1×3=8$ 与 $6+1×3−1=8$）视为等电子体，而忽略**原子种类不同但总数相等**是正确的——原子总数相同即可，不要求原子种类一致。
- **价电子总数计算错误**：离子价电子总数务必加减电荷。$NH_4^+$ 价电子总数 = $5 + 1×4 − 1 = 8$；若漏减正电荷得 $9$，会把 $CH_4$ 错判为非等电子体。
- **构型相似 ≠ 化学性质相似**：等电子体仅保证电子结构和空间构型相似，化学性质可完全不同（如 $CO$ 有强还原性、$N_2$ 极稳定）。
- **混淆“质子数”与“价电子数”**：等电子体不要求质子数相等，只要求价电子总数等同。
