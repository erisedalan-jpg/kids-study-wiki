# 质子数

> **一句话**：质子数是指原子核内所含质子的数目，数值上等于核电荷数，决定元素的种类。
> **English**: The number of protons in the nucleus of an atom, which determines the element’s identity and equals the nuclear charge.

## 🎓 考点精讲

**定义/原理**  
质子数（$Z$）是原子核中质子的个数。其核心关系式：  
- $\text{质子数} = \text{核电荷数} = \text{原子序数} = \text{原子的核外电子数}$（原子电中性时）  
- 对于阳离子 $R^{n+}$：质子数为 $Z$，电子数为 $Z-n$；对于阴离子 $R^{n-}$：质子数为 $Z$，电子数为 $Z+n$。  
- 质量数 $A \approx Z + N$（$N$ 为中子数）。质子数相同、中子数不同的原子互称同位素（如 $^1_1\text{H}$、$^2_1\text{H}$、$^3_1\text{H}$）。  

**关键方法**  
高考中常以“元素推断”题型考查。解题步骤：① 根据原子（或离子）的电子数、质量数等条件，结合电中性或离子电荷计算出质子数 $Z$；② 由 $Z$ 确定元素名称及其在周期表中的位置；③ 利用同主族、同周期规律比较性质（如原子半径、电离能、电负性等）。常用判定：原子（或离子）的化学性质主要由质子数（核电荷数）决定，而非中子数或质量数。  

**典型应用**  
1. **粒子符号推断**：已知某微粒 $A^{2+}$ 核外有 $x$ 个电子，则质子数 $Z = x + 2$，对应元素即原子序数为 $Z$ 的元素。  
2. **同位素辨析**：题干给出几种核素 $^{35}_{17}\text{Cl}$ 和 $^{37}_{17}\text{Cl}$，判断两者质子数相同、中子数不同、化学性质几乎相同，物理性质略有差异。  

## ⚠️ 高频易错点

- 混淆质子数与电子数：对离子一定要先根据电荷算出质子数，切勿直接将电子数当作质子数。  
- 误以为质子数与中子数总是相等：质子数 $Z$ 与中子数 $N$ 可相差较大（如 $^{1}\text{H}$ 无中子），不能以此推断元素。  
- 将“质子数决定元素种类”与“最外层电子数决定化学性质”混为一谈：前者用于辨别元素，后者用于解释同类元素化学行为的细化差异，反应中元素种类自身不变。  
- 在核反应中误写守恒关系：核反应中质子和中子总数守恒的是质量数，而非单纯的质子数守恒（可能发生质子与中子的相互转化）。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 质子数 | proton number / atomic number | 原子核中质子的数目，记为 $Z$ |
| 核电荷数 | nuclear charge number | 原子核所带正电荷数，数值等于质子数 |
| 原子序数 | atomic number | 元素在周期表中的序号，等于质子数 |
| 质量数 | mass number | 质子数与中子数之和，符号 $A$ |
