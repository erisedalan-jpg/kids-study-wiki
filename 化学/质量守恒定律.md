# 质量守恒定律

> **一句话**：质量守恒定律指出，参加化学反应的各物质质量之和等于反应后生成的各物质质量之和，即化学反应前后总质量不变。
> **English**: The law of conservation of mass states that the total mass of reactants equals the total mass of products in any chemical reaction—mass is neither created nor destroyed.

---

## 📚 给 6-12 岁（自读版）

你把木头烧了，灰烬比木头轻——那多余的质量去哪了？其实，生成的烟雾、水蒸气、CO₂ 都跑到空气里了！如果把所有产物都加起来，总质量和原来的木头一模一样！这就是**质量守恒定律**。

**质量守恒定律告诉我们什么？**

化学反应前后，所有参与反应的物质和产生的物质，**总质量不变**！

质量不会凭空出现，也不会凭空消失——只是重新排列组合了。

**为什么质量守恒？**

化学反应的本质是原子的重新排列（旧键断裂，新键形成），原子种类不变、原子数量不变、原子质量不变，所以总质量必然不变！

**生活中的例子**

1. 蜡烛燃烧：蜡（石蜡）+ 氧气 → 二氧化碳 + 水
   - 蜡烛质量变小了（对！因为 CO₂ 和水蒸气跑走了）
   - 但如果把生成的 CO₂ 和水蒸气都收集起来，总质量 = 蜡烛 + 参与的氧气

2. 铁生锈：铁 + 氧气 + 水 → 铁锈
   - 铁锈比铁重！（因为铁和空气中的氧气结合，质量增加了）

**口诀**：化学反应质量守，反应前后总量等，原子种类数量变不了，质量守恒是规律！

## 🎓 给 12+（进阶版）

### 定义

质量守恒定律（Law of Conservation of Mass，又称拉瓦锡定律）：在任何化学反应中，参与反应的各物质的质量总和等于反应后生成的各物质的质量总和。

$$\sum m(\text{反应物}) = \sum m(\text{生成物})$$

**微观解释**：化学反应的本质是原子之间的化学键重新组合，原子种类、数量、质量均不变，因此总质量不变。

### 性质 / 反应 / 公式

**质量守恒定律的内涵**

化学反应前后：

| 不变的量 | 变化的量 |
|---------|---------|
| 原子种类 | 分子种类 |
| 原子数目 | 分子数目 |
| 各元素质量 | 物质种类 |
| 各元素种类 | 物质状态 |
| 总质量 | |

**应用：检验化学方程式是否配平**

以 $2H_2 + O_2 \overset{\text{点燃}}{=} 2H_2O$ 为例：

| | 反应前（2H₂+O₂）| 反应后（2H₂O）|
|--|--------|--------|
| H 原子数 | 4 | 4 ✓ |
| O 原子数 | 2 | 2 ✓ |
| 总质量 | 2×2 + 32 = 36 | 2×18 = 36 ✓ |

**注意"质量不变"需要密封系统**

若有气体生成且未密封，实验中秤显示质量减少（气体逸散）。质量守恒定律仍正确，只是称量不完整。

**质量守恒的适用范围**

- 适用于：一切化学变化
- 不适用于：物理变化（质量本来就守恒，但物理变化无化学反应）
- 核反应（质能方程 $E=mc^2$ 涉及质量-能量转化）

### 典型实验 / 例题

**例题**：在密闭容器中，铁在 O₂ 中点燃后，密闭系统总质量（ ）
A. 增大  B. 减小  C. 不变  D. 无法确定

**解析**：在密封系统中，$3Fe + 2O_2 \overset{\text{点燃}}{=} Fe_3O_4$，依据质量守恒定律，密封系统总质量不变（参与反应的铁和O₂的质量之和等于生成Fe₃O₄的质量），选 **C**。

**例题 2**：某反应为 $A + 2B \overset{\Delta}{=} 2C + D$，若 A 和 B 的相对分子质量分别为 46 和 32，生成物 C 的相对分子质量为 46。求 D 的相对分子质量。

**解析**：$M_r(A) + 2M_r(B) = 2M_r(C) + M_r(D)$

$46 + 2 \times 32 = 2 \times 46 + M_r(D)$

$110 = 92 + M_r(D)$

$M_r(D) = 18$（即 H₂O）

### 易错点

1. **开放系统中气体逸散，天平"示数减小"不违反质量守恒**——只是气体离开了称量范围，但整个系统（包括逸出气体）总质量仍守恒。
2. 质量守恒是**化学反应**的规律，物理变化（如水蒸发）中质量当然也不变，但那不是"质量守恒定律"的讨论范围。
3. 不能说"核反应不遵守质量守恒"——核反应中存在质量-能量转化（$E=mc^2$），这超出初中范围；初中讨论的化学反应完全遵守质量守恒。
