# 轨道表示式

> **一句话**：用方框代表轨道、箭头代表电子自旋方向的图示排布，直观呈现电子在各轨道中的具体分配。
> **English**: A diagram using boxes (orbitals) and arrows (electrons with spin direction) to show the exact distribution of electrons in atomic orbitals.

---

## 🎓 高中进阶

### 定义
**轨道表示式**（Orbital Diagram）是电子排布的图示形式：
- 每个**方框（□）**代表一个原子轨道
- **↑** 代表 ms = +1/2 的电子（自旋向上）
- **↓** 代表 ms = −1/2 的电子（自旋向下）
- 方框内最多放 ↑↓ 两个箭头（泡利原理）
- 简并轨道（同能级）须先各放一个↑（洪特规则）

### 写法示例

**C（Z=6）：1s²2s²2p²**

```
1s      2s      2px  2py  2pz
[↑↓]   [↑↓]   [↑ ] [↑ ] [  ]
```

**N（Z=7）：1s²2s²2p³**

```
1s      2s      2px  2py  2pz
[↑↓]   [↑↓]   [↑ ] [↑ ] [↑ ]
```

**O（Z=8）：1s²2s²2p⁴**

```
1s      2s      2px  2py  2pz
[↑↓]   [↑↓]   [↑↓] [↑ ] [↑ ]
```

### 违反规则的错误示例

| 错误排布 | 违反原则 |
|---------|---------|
| 同一框中 ↑↑ | 泡利原理（自旋相同）|
| 2p² 写成 [↑↓][□][□] | 洪特规则（应分占）|
| 跳过低能级先填高能级 | 能量最低原理 |

### 用途
- 判断**未成对电子数**（顺磁性计算）
- 分析**杂化方式**（哪些轨道参与杂化）
- 确定元素**化学价**和成键方式

### 典型例题
**题**：画出 Fe²⁺（Z=26，失去 2 个 4s 电子）的 3d 轨道表示式，判断未成对电子数。
**解**：Fe²⁺：[Ar]3d⁶
3d：[↑↓][↑ ][↑ ][↑ ][↑ ]
未成对电子数 = **4**（顺磁性）

### 易错点
- 简并轨道（如 2p 三个轨道）一定先各放一个↑，再配对
- 不要把 ↑↓ 全写在同一轨道（除非简并轨道全部先单占）
- 轨道表示式与电子排布式对应，不能矛盾
