# 形状-椭圆

> **一句话**：像鸡蛋一样长长的圆。
> **English**: An oval is a stretched circle — like an egg or a running track.

---

## 🧒 给 3-6 岁（共读版）

你见过鸡蛋吗？它不是圆圆的，而是**长长的圆**——这就是椭圆！

椭圆和圆很像，但：
- 圆：每个方向都一样宽
- 椭圆：一个方向长一点，另一个方向短一点

**共读小活动：** 找找家里有没有椭圆形的东西？鸡蛋、操场跑道（图片里）、有些盘子也是椭圆的……找到就说"这是椭圆！"

---

## 📚 给 6-12 岁（自读版）

### 椭圆 vs 圆：有什么不同？

圆有一个"中心点"（圆心），到边缘每个方向都一样远。

椭圆则不一样，它有**两个特殊的点**，叫做**焦点**（focus，复数 foci）。椭圆上任意一点到两个焦点的距离**加起来**都是一样的。

**用绳子画椭圆（试试看！）：**
1. 把一根绳子的两端分别固定在纸上（这两个点就是焦点）
2. 用笔撑住绳子往外拉
3. 保持绳子绷紧，绕一圈——就画出了椭圆！

**椭圆的两个"轴"：**
- **长轴（major axis）**：椭圆最长的那条直径
- **短轴（minor axis）**：椭圆最短的那条直径

当长轴和短轴一样长时，椭圆就变成了圆——所以**圆是椭圆的特殊情况**。

**生活中的椭圆：** 操场跑道、鸡蛋、很多体育馆的平面图、地球绕太阳的轨道……

想了解更多，可以看 [[009-形状-圆|形状-圆]]、[[014-对称|对称]]。

## 🎓 给 12+（进阶版）

### 椭圆的标准方程

以中心为原点，长半轴为 a，短半轴为 b（a > b > 0），椭圆的标准方程为：

```
x²/a² + y²/b² = 1
```

两个焦点在 x 轴上，焦距 c 满足 **c² = a² - b²**。

当 a = b = r 时，方程退化为 x² + y² = r²，即圆。这确认了圆是椭圆的特例。

### 开普勒定律与行星轨道（待家长核对）

17 世纪，德国天文学家约翰内斯·开普勒（Johannes Kepler）发现行星运动的三大定律，其中**第一定律**指出：

> 所有行星绕太阳运行的轨道都是椭圆，太阳位于椭圆的一个焦点上。

这打破了"行星轨道是完美圆形"的传统观念，是科学史上的重大发现。地球的轨道接近圆形（离心率约 0.017），但木星、水星的轨道则更加明显地呈椭圆形。（**具体数值请家长核对**）

### 椭圆的反射性质

椭圆有一个神奇的物理性质：从一个焦点发出的光（或声音），经过椭圆边缘反射后，必然汇聚到另一个焦点。

这种"焦点反射"性质被用于：
- **椭圆房间（耳语廊，Whispering Gallery）**：站在一个焦点说话，另一个焦点的人可以清晰听到，旁人却听不见
- **医学碎石术（体外冲击波碎石）**：利用椭圆焦点聚焦冲击波，精确击碎肾结石

### 离心率

椭圆的"胖瘦"用**离心率（eccentricity）e = c/a** 描述：
- e = 0：圆（最胖）
- 0 < e < 1：椭圆（越大越扁）
- e = 1：抛物线（极限情况）

## 🗣️ 家长讲解话术

- 孩子可能问 **"鸡蛋为什么是椭圆的不是圆的？"** → 可以说："椭圆形的鸡蛋不容易滚远——它在平面上滚动时会绕弯，不像圆球一样一直滚走。而且鸡蛋从鸡妈妈那里出来的时候，椭圆形更容易通过产道。大自然很聪明！"

- 孩子可能问 **"地球绕太阳的路是圆吗？"** → 可以说："不是完全的圆，是椭圆。太阳不在正中间，而是在椭圆的一个焦点上。所以地球有时候离太阳近一点，有时候远一点。"

- 孩子可能问 **"椭圆和圆哪个先出现？"** → 可以说："从数学上说，圆是椭圆的特殊情况——当椭圆的两个焦点合并成一个点时，就变成圆了。所以它们其实是同一家族的。"

- **共读小活动：** 用一段绳子、两颗图钉和一支铅笔，在纸上画出一个椭圆（两个图钉当焦点）。然后把两个图钉合在一起，画出一个圆，让孩子看到椭圆"变成"圆的过程。
