# 位置

> **一句话**：用上下、前后、左右说东西在哪里。
> **English**: Position words tell where things are: up/down, front/back, left/right.

---

## 🧒 给 3-6 岁（共读版）

你会描述东西在哪里吗？小熊在桌子**上面**，皮球在椅子**下面**，爸爸站在妈妈**前面**，弟弟跟在你**后面**——这些都是在说"位置"！

我们有**6 个基本方位**：
- **上** / **下**：天花板在上，地板在下
- **前** / **后**：脸对着的那一侧叫前，背后是后
- **左** / **右**：右手是你吃饭、写字的那只手；左手是另一只

**小技巧：** 不知道哪只手是右手？做出拿筷子、拿铅笔的动作——那只手就是右手！

**共读小活动：** 妈妈说位置，宝宝指物体！妈妈说"毛绒熊在沙发的上面"，宝宝用手指出来。再换宝宝来出题，妈妈来指——谁说得准谁赢！

## 📚 给 6-12 岁（自读版）

### 6 个方位和参照物

说位置，必须有一个**参照物**（reference point）——"X 在 Y 的左边"，这个 Y 就是参照物。如果不说参照物，位置就说不清楚。

**日常生活中的方位：**
- **上下**：描述高低。"书放在柜子上面" — 柜子是参照物。
- **前后**：描述队列位置。"小明站在队伍的第三位"，就是说他前面有两个人。
- **左右**：描述左右方向。这里有一个重要的事：**我的左边是你的右边！** 面对面站的时候，方向正好相反。

### 做数学题的读法

一上数学题里经常出现："小猫在小狗的**左边**"。读这类题目的步骤：
1. 先找参照物（小狗）
2. 再找方向（左边）
3. 画图或用手指帮助理解

**数学方位词：** 数学上说的"上方/下方/左方/右方"和日常用语一样，但更强调精确的参照。

想了解更多，可以看 [[001-数数|数数]]、[[006-大小|大小]]。

## 🎓 给 12+（进阶版）

### 笛卡尔坐标系：把方位变成数字

17 世纪，法国数学家笛卡尔（René Descartes）发明了**直角坐标系**（Cartesian coordinate system），把"左右"变成 x 轴，把"上下"变成 y 轴——于是任何一个位置都能用一对数字 (x, y) 精确表示。

- x > 0 在右，x < 0 在左
- y > 0 在上，y < 0 在下
- 三维空间再加 z 轴（前后 / 深浅）：(x, y, z)

### 极坐标：用距离+角度表示位置

另一种表示方式：不用"左右上下"，而用"离原点有多远"（r）和"方向角"（θ）来定位。这叫**极坐标（polar coordinates）**，在描述旋转、波动等问题时往往更简洁。

### 中国古代方位 vs 现代数学

中国古代用"东南西北"加"上下"来描述方位，共 6 个基本方向，与现代数学的 x/y/z 三轴六方向本质上一样，只是命名系统不同。值得注意的是，古代中国地图惯例"南上北下"，与现代地图（北上南下）相反。（**具体地图传统请家长核对**）

### 计算机图形学的坐标系

屏幕坐标系的原点在**左上角**，y 轴向下增大——这与数学坐标系 y 轴向上相反。所以在编程画图时要特别注意方向。许多游戏引擎和 UI 框架都使用这个"屏幕坐标系"，而 3D 引擎则各自有不同约定（左手系 vs 右手系，**待家长核对**）。

### 地理方位与数学方位的差异

地理上的"北偏东 30°"从正北顺时针量；数学上的角通常从正东（x 轴正方向）逆时针量。这两个系统之间的转换需要小心，是导航和 GIS 计算中常见的细节陷阱。

## 🗣️ 家长讲解话术

- 孩子可能问 **"我的左边是哪里？"** → 可以说："你平时用哪只手写字、拿筷子？那只手就是你的右手！另一只就是左手。左手这边就是左边，右手那边就是右边。"

- 孩子可能问 **"为什么我说左边，你走的是另一边？"** → 可以说："因为我们面对面站，你的右手对着我的左手。所以你说左边，对我来说是右边。这就是'相对方向'——换一个参照角度，方向就变了。"

- 孩子可能问 **"地图上哪边是上面？"** → 可以说："现代地图通常上面是北边、下面是南边。你看地图的时候，先找到'北'的标记，就能判断方向了。"

- **共读小活动：** 在家里玩"方位接龙"。家长说一个物体，孩子说它在另一个物体的哪个方向。比如"沙发在电视的前面"，孩子继续说"茶几在沙发的右边"……看谁说得又快又准。
