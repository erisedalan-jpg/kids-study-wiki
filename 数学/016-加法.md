# 加法

> **一句话**：把两堆东西合在一起，数一数一共有多少——这就是加法。
> **English**: Addition means putting two groups together and counting how many there are in total.

---

## 🧒 给 3-6 岁（共读版）

你有 3 颗糖，妈妈又给了你 2 颗——现在你一共有几颗？

我们来数一数：先有 3 颗，再加 2 颗……4、5——一共 **5 颗**！

这个"合在一起"的过程，就叫**加法**。我们用 **+** 这个符号来表示"加"，用 **=** 表示"等于"：

> **3 + 2 = 5**

读作：**三加二等于五**。

**共读小活动：** 拿出 3 颗葡萄干和 2 颗葡萄干，合在一起数一数，验证答案。换几个数再试试——4+1=？2+2=？

## 📚 给 6-12 岁（自读版）

### 加法的规律

**交换律（Commutative Property）：** 两个数相加，换个顺序答案不变。
> 3 + 5 = 5 + 3 = 8

无论先放苹果还是先放橙子，合起来的总数一样。

**结合律（Associative Property）：** 三个数相加，先加哪两个都行。
> (2 + 3) + 4 = 2 + (3 + 4) = 9

**连加：** 多个数一起加，从左往右逐个加：
> 1 + 2 + 3 + 4 + 5 = 15

德国数学家高斯小时候（传说中）快速算出 1 + 2 + … + 100 = 5050，方法是发现首尾配对：1+100, 2+99, … 每对都是 101，共 50 对，50 × 101 = 5050。（**待家长核对此传说的出处和准确性**）

### 竖式加法

当数字变大，用竖式：

```
  2 8
+ 1 5
-----
  4 3
```

各位对齐，从个位开始加：8+5=13，写 3 进 1；2+1+1=4。答案 43。

**进位** 是十进制的核心操作——满 10 就向前一位进 1，这和 [[024-凑十|凑十]] 直接相关。

### 加法与数数的联系

加法其实是"快速数数"——3+4 就是从 3 开始再数 4 个：4、5、6、7。理解这一点，加法不再神秘。

## 🎓 给 12+（进阶版）

### 阿贝尔群与加法的代数结构

自然数加法满足的性质，在代数上称为**交换群（Abel 群）**的结构：

1. **封闭性**：两个整数相加还是整数
2. **结合律**：(a + b) + c = a + (b + c)
3. **单位元**：a + 0 = a（0 是加法的"什么都不做"）
4. **逆元**：a + (-a) = 0（每个数都有相反数）
5. **交换律**：a + b = b + a

自然数在加法下只是半群（没有逆元），扩充到整数才形成阿贝尔群。整数 → 有理数 → 实数 → 复数，每次扩充都保持这个结构。

### 进位机制与二进制

十进制加法的"进位"原则（满 10 进 1），在二进制里变成"满 2 进 1"：

```
  1 0 1  (= 5)
+ 0 1 1  (= 3)
-------
  1 0 0 0 (= 8)
```

计算机做加法，用的就是二进制进位，并用"加法器"电路实现。CPU 里最基本的运算就是加法——减法、乘法、除法都可以由加法派生。

### 超限运算

在无限集合的世界里，加法有奇特性质：ℵ₀ + ℵ₀ = ℵ₀（无穷个自然数加上无穷个自然数，还是同样的无穷大）。这是康托尔集合论的结果，直觉在这里完全失效。

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么 3+2 和 2+3 一样？"** → 可以说："因为把 3 个苹果和 2 个橙子放在一起，不管先数哪种，最后总数都是 5 个。顺序不影响合并的结果。"

- 孩子可能问 **"加号是什么意思？"** → 可以说："加号 + 是个记号，告诉你'把两边合起来'。就像一个小指示牌：左边一堆 + 右边一堆 = 合起来一共多少。"

- 孩子可能问 **"100 以内的加法怎么算那么快？"** → 可以说："可以用[[024-凑十|凑十]]法——先把一个数凑成 10，再加剩下的，算起来容易。比如 8+5，先拆 5 为 2+3，8+2=10，10+3=13。"

- **共读小活动：** 用积木玩加法——先搭 4 块，再加 3 块，数数一共几块，然后换个顺序先搭 3 块再加 4 块，让孩子发现答案一样（验证交换律）。
