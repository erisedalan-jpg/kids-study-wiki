# 找规律

> **一句话**：找规律就是发现"什么在重复"——颜色、形状、数字背后藏着的秘密顺序。
> **English**: Finding a pattern means discovering what repeats — the hidden rule behind colors, shapes, or numbers.

---

## 🧒 给 3-6 岁（共读版）

妈妈穿了一件有花纹的衣服：红色、蓝色、红色、蓝色……你发现了什么？

对！是红色和蓝色在**轮流出现**。这就叫**规律**（pattern）——有个秘密在一遍遍重复。

生活里规律到处都是：
- 斑马的花纹：黑白黑白黑白……
- 拍手游戏：拍拍停，拍拍停，拍拍停……
- 红绿灯：红灯亮→绿灯亮→红灯亮……
- 楼梯台阶：上一级、上一级、上一级……

我们来玩个游戏——**接龙规律**：

> 🔴🔵🔴🔵🔴 ___？

下一个是什么颜色？（蓝色！因为红蓝红蓝在轮流。）

**共读小活动：** 用积木摆一排规律，比如"红蓝蓝红蓝蓝"，让孩子猜下一块是什么颜色，然后让孩子自己出题考你。

## 📚 给 6-12 岁（自读版）

### 三类规律

小学里主要学三种规律：

#### 1. 颜色/形状规律（图形规律）

图案按一定顺序重复，找出"重复单元"就找到规律了。

**例：** ○△□○△□○△□ → 重复单元是"○△□"，每 3 个一循环。

**方法：** 把序列分组，找到最短的重复段，验证后面的图案是否符合。

#### 2. 数字规律（数列）

数字按一定方式变化：
- **等差：** 2, 4, 6, 8, 10 → 每次加 2（公差 = 2）
- **等倍：** 1, 2, 4, 8, 16 → 每次乘 2
- **加法累积：** 1, 1, 2, 3, 5, 8, 13 → 前两项之和等于后一项（斐波那契数列）
- **奇偶交替：** 1, 2, 1, 2, 1 → 奇数偶数轮流

**例题：** 3, 6, 9, 12, ___→ 每次加 3，答案是 15。

**例题：** 1, 4, 9, 16, 25, ___ → 1²=1, 2²=4, 3²=9, 4²=16, 5²=25，答案是 6²=36。

#### 3. 颜色×形状组合规律

二上及以后会出现二维规律，例如行列表格里颜色和形状同时变化，需要分开找横向规律和纵向规律。

### 找规律的步骤

1. **看**：仔细观察整个序列
2. **找重复单元**：什么在重复？间隔是多少？
3. **验证**：用找到的规律检验后面的项
4. **延伸**：预测下一个（或第 n 个）

### 常见误区

- **只看前两项就下结论**：前两项相同不代表是"全相同"规律，要多看几项。例：1, 2, 3, 5, 8… 如果只看前两项"1, 2"会猜错。
- **把"变化量"和"项本身"搞混**：2, 4, 6 的公差是 2，不是"每项乘以 2"。
- **漏看图形的方向变化**：有时形状方向也是规律的一部分（如箭头朝向）。

一年级下册第 7 单元（p83-89）：图形规律与简单数字规律，是小学阶段对找规律的正式入门。

## 🎓 给 12+（进阶版）

### 数列与通项公式

找规律在中学升级为**数列**研究，核心是找**通项公式**——用一个含 n 的表达式表示第 n 项。

- **等差数列：** a_n = a_1 + (n-1)d，其中 d 为公差
- **等比数列：** a_n = a_1 × q^(n-1)，其中 q 为公比
- **斐波那契数列：** F(n) = F(n-1) + F(n-2)，F(1)=F(2)=1

等差数列前 n 项之和：S_n = n(a_1 + a_n)/2（高斯求和公式的一般化）

### 编程中的规律：循环与序列

在计算机科学里，规律直接对应**循环结构（loop）**：

```python
# 输出红蓝交替（颜色规律）
colors = ["红", "蓝"]
for i in range(10):
    print(colors[i % 2])

# 输出等差数列 3, 6, 9, ...
for n in range(1, 11):
    print(3 * n)

# 斐波那契数列
a, b = 1, 1
for _ in range(10):
    print(a)
    a, b = b, a + b
```

`i % 2` 是取余运算，结果在 0 和 1 之间循环——这正是颜色规律的数学本质。

### 模式识别与人工智能

机器学习（Machine Learning）的核心任务就是"找规律"——从大量数据中发现隐藏的 pattern：

- 图像识别：从像素规律中识别猫狗
- 语言模型：从文字序列规律预测下一个词
- 股票预测：从历史价格规律中寻找趋势（**注：规律并不总能预测未来，待家长核对**）

小学学的"找规律"是人类智能的基础能力——发现结构、压缩信息、做出预测。

### 数学美：自然界中的规律

斐波那契数列（1,1,2,3,5,8,13,21…）出现在：
- 向日葵的螺旋数量（通常是 34 和 55）
- 松果的鳞片排列
- 鹦鹉螺的螺旋比例

这些数字之间的比值趋近于**黄金比例 φ ≈ 1.618**，被认为是自然界中最美的规律之一。（**待家长核对具体植物数据**）

## 🗣️ 家长讲解话术

- 孩子可能问 **"规律是什么意思？"** → 可以说："规律就是'什么在一遍遍重复'的秘密。你看斑马身上的黑白条纹——黑白黑白，一直这样重复，这就是规律。你穿的衣服上有没有规律呢？"

- 孩子可能问 **"怎么找规律？"** → 可以说："先多看几个，找到'哪一段在重复'。比如 1, 2, 3, 1, 2, 3，你看到'1, 2, 3'重复了两次——这就是重复单元，后面还会接着出现 1, 2, 3。"

- 孩子可能问 **"如果找不到规律怎么办？"** → 可以说："先看相邻两个数之间的变化：加了多少？乘了多少？还是什么别的变化？从'变化'里找规律，比直接看数字更容易。"

- **共读小活动：** 用葡萄干、花生米等小食物摆规律，请孩子接着摆下去；然后角色互换，让孩子出题，大人接。吃完就是奖励！
