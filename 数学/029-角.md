# 角

> **一句话**：两条线从同一个点出发，中间张开的"口"就是角——张开越大，角越大。
> **English**: An angle is formed by two rays sharing the same starting point — the wider they open, the bigger the angle.

---

## 🧒 给 3-6 岁（共读版）

你把两根手指并在一起，然后慢慢张开——中间那个"口"就是一个**角**（angle）！

张开一点点，是**小角**。
张开更多，是**大角**。
张开成一个正方形的角，就是**直角**（right angle）——正好像书本的四个角一样方方正正的。

生活里的角：
- 书的角 → 直角（像个小 L 形）
- 剪刀剪东西时的口 → 小小的锐角
- 时钟 3 点的时候，时针和分针之间 → 直角
- 时钟 6 点的时候，时针和分针 → 平角（一条直线）

**共读小活动：** 找一把三角板，用它的直角对准书本的角，看看是不是一样方——这叫"用三角板验证直角"！

## 📚 给 6-12 岁（自读版）

### 角的组成

一个角由三部分组成：

```
        射线（边）
       /
      / ← 角
顶点 ·
      \
       \
        射线（边）
```

- **顶点（vertex）**：两条射线的公共起点
- **边（side）**：两条射线
- **角的大小**：两条边张开的程度，用**度（°）**来表示

角的记法：∠AOB（读作"角 AOB"，O 是顶点）或直接写 ∠1。

### 五类角的划分

| 名称 | 英文 | 度数范围 | 例子 |
|------|------|----------|------|
| 锐角 | acute angle | 0° < α < 90° | 剪刀的口；三角形的尖角 |
| 直角 | right angle | α = 90° | 书的角；地砖的角 |
| 钝角 | obtuse angle | 90° < α < 180° | 钝角三角形中的大角 |
| 平角 | straight angle | α = 180° | 一条直线（两射线成一直线） |
| 周角 | full angle | α = 360° | 转一整圈 |

**记忆口诀：** "锐直钝平周，0-90-180-360"

### 用量角器量角

1. 量角器的**中心**对准角的**顶点**
2. 量角器的**0° 刻度线**对准角的**一条边**
3. 另一条边指向的刻度，就是角的度数
4. 注意：量角器有内外两圈刻度，要看对应的那圈（边从 0 开始就用同侧的圈）

**常见错误：** 中心没有对准顶点，或读错了内外圈。

### 用三角板画直角

三角板有一个直角（90°），还有两个角分别是 45° 和 45°（等腰直角三角板）或 30° 和 60°（30-60-90 三角板）。

### 教材对应

## 🎓 给 12+（进阶版）

### 弧度制（radian）

度（°）是历史沿用的单位，数学和物理中更常用**弧度（radian, rad）**：

```
1 弧度 = 圆弧长度等于半径时所对应的圆心角
π 弧度 = 180°

常用换算：
  90°  = π/2 rad ≈ 1.571 rad
  180° = π   rad ≈ 3.142 rad
  360° = 2π  rad ≈ 6.283 rad
```

为什么用弧度？因为在弧度制下，三角函数的微积分公式最简洁：
- sin'(x) = cos(x)（x 用弧度）
- 如果用度，会多出一个 π/180 的系数，很麻烦

### 三角函数与角

角进入高中后和**三角函数**结合：

- **sin θ**（正弦）= 对边 / 斜边
- **cos θ**（余弦）= 邻边 / 斜边
- **tan θ**（正切）= 对边 / 邻边

直角三角形的三边关系（勾股定理）：a² + b² = c²

单位圆定义让三角函数扩展到任意角（不只是三角形里的角），甚至可以处理负角和大于 360° 的角，是高中数学的重要桥梁。

### 欧拉公式与角

数学中最美的公式之一：

```
e^(iθ) = cos θ + i·sin θ
```

当 θ = π 时，得到**欧拉恒等式**：

```
e^(iπ) + 1 = 0
```

这个式子把数学中最重要的五个常数 e、i、π、1、0 联系在一起，被称为"数学中最美的公式"。（**待家长核对此说法的出处**）

### 角在物理和工程中的应用

- **斜面力学**：物体在 θ 角斜面上，重力沿斜面分量 = mg·sin θ
- **光的折射**：Snell 定律 n₁·sin θ₁ = n₂·sin θ₂（θ 是入射角/折射角）
- **建筑设计**：屋顶坡角决定排水速度和雪载荷
- **测量与导航**：GPS、陀螺仪均依赖角度的精确测量

## 🗣️ 家长讲解话术

- 孩子可能问 **"直角为什么叫'直'角？"** → 可以说："'直'在这里不是'直线'的意思，而是'端正、正确'的意思——直角是最'标准'的角，像正方形的角那样。三角板上有一个专门的直角，就是用来帮你画出这种标准角的。"

- 孩子可能问 **"角越大越好还是越小越好？"** → 可以说："这得看用途！剪刀尖要锐角，才能剪进去；书本的角是直角，方便对齐；门开到最大是平角（一直线）——不同场合需要不同大小的角。"

- 孩子可能问 **"为什么一圈是 360 度？"** → 可以说："这是古巴比伦人（大概 4000 年前）定的规矩，他们用 60 进制数数，而 360 = 6 × 60，刚好一年大约 360 天，所以定下来了。这个规矩一直用到现在。（**待家长核对此历史说法**）"

- **共读小活动：** 用两根筷子摆成各种角度，说"这是锐角""这是直角"——让孩子来判断，或者让孩子摆，家长来判断。
