# 0

> **一句话**：一个都没有时候用的数。
> **English**: Zero is the number that means "none" — nothing is left.

---

## 🧒 给 3-6 岁（共读版）

盘子里有三块饼干。你吃了一块，还有两块。再吃一块，还有一块。再吃最后一块……盘子空了！

这时候盘子里有几块饼干？

**零块！** 零就是"一个都没有"时候用的数字，写成这样：**0**。

**共读小活动：** 把几颗葡萄放在孩子手心，让他/她一颗一颗吃掉，每吃一颗问"还剩几颗？"直到全吃完——引出"零个了"！

## 📚 给 6-12 岁（自读版）

### 0 不是"没有数"，而是一个特别的数

很多人觉得 0 就是"什么都没有"，所以不算数。其实 0 是一个真实存在的数——它表示"这种东西的数量是零个"。

**0 在加法里是特别的：**
- 任何数加 0，结果不变：5 + 0 = 5，100 + 0 = 100
- 这个性质叫"加法单位元"，0 是加法的"不变按钮"

**0 在位值制中是占位符：**
- 10 = 一个十，零个一
- 100 = 一个百，零个十，零个一
- 如果没有 0 来占位，我们就没办法区分 1、10、100——它们看起来都是"1"

这就是为什么 0 的发明在数学史上非常重要。

想了解更多，可以看 [[数数]]、[[十进制]]、[[1-10]]。

## 🎓 给 12+（进阶版）

### 0 的历史发明（待家长核对）

0 并不是自古就有的。古埃及、古希腊、古罗马都没有"零"这个数字的概念（罗马数字里没有 0）。

0 作为独立的数（而不只是占位符）被系统使用，最早记录在古印度数学中，大约在公元 5 至 7 世纪，通过学者如婆罗摩笈多（Brahmagupta）明确给出了涉及 0 的运算规则。之后阿拉伯数学家将其传入欧洲，形成我们今天使用的阿拉伯数字系统。（**以上历史细节请家长核对，具体人名和年份 LLM 可能有误**）

### 0 在抽象代数中的地位

在抽象代数里，一个集合配上一种运算，如果满足特定条件，就叫"群"（group）。整数在加法下构成群，其中 **0 是加法单位元**（identity element）：

> 对任意整数 a，有 a + 0 = 0 + a = a

每个数 a 还有它的逆元 -a（即 a + (-a) = 0）。0 是这个结构的核心。

### 0 为什么不能做除数？

- **x / 0**：如果 0 × ? = x，对 x ≠ 0，没有任何数满足这个等式，所以无定义。
- **0 / 0**：0 × ? = 0，任何数都满足，所以无法唯一确定结果，也无定义（称为"不定形"indeterminate form）。

这两种情况在微积分极限理论中有更精细的处理（如洛必达法则），但除数为 0 在基础算术中始终是"禁区"。

## 🗣️ 家长讲解话术

- 孩子可能问 **"0 不是数吗？"** → 可以说："是！0 是一个很特别的数。它告诉大家'这里什么都没有'。如果没有 0，我们就没法写 10、100、1000 这些数了。"

- 孩子可能问 **"为什么 1 后面写 0 就变成 10？"** → 可以说："因为我们的数字系统里，位置很重要。'10'里面，左边那个'1'表示'一个十'，右边那个'0'表示'个位上什么都没有'。0 在这里是个占位的角色，非常重要。"（引出 [[十进制]]）

- 孩子可能问 **"我有 0 块糖，可以给弟弟吗？"** → 可以说："一块都没有，就没法给了。0 就是什么都没有的意思，不能再少了。"

- **共读小活动：** 用 5 个小玩具，每次拿走 1 个，让孩子数"还剩几个"，直到变成 0——体验从有到无的全过程。
