# 周长

> **一句话**：绕着一个图形的边走一圈，走过的总长度就是周长。
> **English**: The perimeter is the total length of the boundary around a shape — the distance you'd walk if you traced the edge once.

---

## 🧒 给 3-6 岁（共读版）

你有没有用手指沿着桌子边缘描过一圈？从出发点回到出发点，手指走过的那一圈路的**总长度**，就是桌子的**周长**（perimeter）！

想象一只蚂蚁从正方形的一个角出发，沿着边边爬，最后回到原来那个角——蚂蚁爬过的总距离，就是正方形的周长。

**共读小活动：** 用绳子绕着一本书的封面围一圈，剪下那段绳子，再把绳子拉直用尺子量一量——这就是书封面的周长！

---

## 📚 给 6-12 岁（自读版）

### 周长的定义

**周长 = 围成图形的所有边的长度之和**

不管是什么形状，周长的计算思路都一样：把所有边的长度加起来。

### 长方形的周长

长方形有两条**长**（length）和两条**宽**（width），对边相等：

```
+--------长--------+
|                  |
宽                 宽
|                  |
+--------长--------+
```

**公式（两种写法，都对）：**

> **方法一**：周长 = 长 + 宽 + 长 + 宽 = 长 × 2 + 宽 × 2
>
> **方法二**：周长 = （长 + 宽）× 2  ← 教材推荐，括号先算更简便

**例：** 长方形长 8 厘米，宽 5 厘米
> 周长 = (8 + 5) × 2 = 13 × 2 = **26 厘米**

### 正方形的周长

正方形四条边都相等：

> **周长 = 边长 × 4**

**例：** 正方形边长 6 厘米
> 周长 = 6 × 4 = **24 厘米**

### 不规则图形的周长

有些图形没有简单公式，就把所有边一条一条加起来：

```
不规则六边形，各边：3, 5, 4, 2, 6, 4 厘米
周长 = 3 + 5 + 4 + 2 + 6 + 4 = 24 厘米
```

对于曲线形状（比如树叶），可以用绳子沿边缘围一圈再拉直来量，这叫**测量法**。

### ⚠️ 三年级最大混淆点：周长 vs 面积

| | 周长 | 面积 |
|---|---|---|
| 是什么 | 边的长度（线） | 内部覆盖的大小（面） |
| 单位 | 厘米、米 | 平方厘米、平方米 |
| 比喻 | 围墙的长 | 草坪的大小 |
| 公式（长方形） | (长+宽)×2 | 长×宽 |

**记忆口诀：** 周长是"走一圈"，面积是"铺满地"。

**反例陷阱：** 两个面积相同的图形，周长可以不同；两个周长相同的图形，面积也可以不同！

```
例：面积都是 12 平方厘米
图形 A：长4厘米，宽3厘米 → 周长 = (4+3)×2 = 14 厘米
图形 B：长6厘米，宽2厘米 → 周长 = (6+2)×2 = 16 厘米
面积相同，周长不同！
```

### 测量周长的方法

1. **直接量**：用直尺量每条边，加起来
2. **绳子法**：适合曲线，绳子围一圈后拉直再量
3. **公式法**：用长方形、正方形等规则图形的公式
4. **数格子法**：在方格纸上数边界格子数（格子边长已知）

## 🎓 给 12+（进阶版）

### 等周问题（Isoperimetric Problem）

**经典问题：** 用固定长度的绳子围出最大面积的图形，应该围成什么形状？

答案：**圆形**。

在所有周长相同的封闭图形中，圆的面积最大。这是古典等周不等式：

> 对任意封闭曲线，若其周长为 L，则围成面积 A 满足：
> **A ≤ L² / (4π)**，当且仅当图形为圆时取等。

这个结论在古希腊时期就有直觉性认识，但严格证明直到 19 世纪末才完成（利用变分法）。

### 圆的周长（周长预告）

圆的周长叫**圆周长**，公式：

> **C = 2πr = πd**

其中 r 是半径，d 是直径，π ≈ 3.14159……

π 是无理数（不能写成分数），小数位无限不循环。证明 π 的无理性是数学分析的经典习题（由朗伯于 1761 年首次证明）。

更进一步，π 还是**超越数**（不是任何有理系数多项式的根），由林德曼于 1882 年证明，这同时意味着用圆规直尺"化圆为方"（构造与圆等积的正方形）是不可能的。

### 分形的周长：无穷大？

科赫雪花（Koch snowflake）是一种由等边三角形不断添加锯齿构成的分形图形。每次迭代周长增加 4/3 倍，经过无穷次迭代后，周长趋向**无穷大**，但面积有限！这说明"面积有限，周长无限"在数学上是可能的，打破了直觉。

## 🗣️ 家长讲解话术

- 孩子可能问 **"周长和面积有什么区别？"** → 可以说："周长是'边边'，面积是'里面'。想象你家院子围了一圈篱笆，篱笆总长就是周长；院子里的草坪有多大，就是面积。两个是完全不同的东西！"

- 孩子可能问 **"为什么长方形周长公式要先加再乘以 2？"** → 可以说："长方形有两条一样的长，两条一样的宽。我们先把一条长和一条宽加在一起，算出'半圈'，再乘以 2 就是一整圈。这样比四条边分别加更快。"

- 孩子可能问 **"不规则图形有没有公式？"** → 可以说："不规则图形没有统一公式，只能把每条边量出来再加。但你可以用绳子绕一圈再拉直来量曲线形状，这是实际生活中常用的方法。"

- **共读小活动：** 拿一张方格纸，画出长 5 格、宽 3 格的长方形，让孩子沿边数格子数——4条边共 16 格，对应周长 = (5+3)×2 = 16 格长。再画面积相同（15格）但形状不同的长方形，比较周长是否一样。
