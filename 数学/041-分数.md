# 分数

> **一句话**：把一个东西平均分成几份，取其中的几份——这就是分数，是整数之间的数。
> **English**: A fraction shows how many equal parts of a whole you have — the numerator tells how many parts, the denominator tells how many parts the whole is divided into.

---

## 🧒 给 3-6 岁（共读版）

把一个苹果**平均切成两半**，你拿一半，我拿一半——你手里的那一半，就是这个苹果的**二分之一**，写作 **1/2**。

```
一个苹果 → 平均切成 2 份 → 你拿 1 份 = 1/2
```

为什么要说"**平均**分"？因为如果切得一大一小，那就不公平，大的不是"一半"！

> **分数 = 平均分之后的某几份**

读法：**分母在下先读，分子在上后读**
- 1/2 读作：二分之一
- 3/4 读作：四分之三

**共读小活动：** 拿一张正方形纸，对折一次，问孩子"每份是几分之几？"（答：1/2）；再对折一次，问"现在每份是几分之几？"（答：1/4）。展开让孩子数折痕把纸分成了几份。

## 📚 给 6-12 岁（自读版）

### 三上初识分数（第8单元）

#### 分数的组成

```
  3   ← 分子（numerator）：取了几份
 ─── 
  4   ← 分母（denominator）：平均分成几份
```

读作：**四分之三**（先读分母，再读分子）

**记忆口诀：** 分母在下管"总份数"，分子在上管"取几份"。

#### 几分之一的大小规律

分母越大，每份越小：

```
把同一块饼干平均分成：
2 份 → 每份是 1/2（大）
4 份 → 每份是 1/4（中）
8 份 → 每份是 1/8（小）
```

> **分母越大，分数越小**（分子都是 1 时）

#### 比较分数大小

**同分母比较（分母相同）：** 分子大的大

> 3/5 > 2/5（都是五份里取，取 3 份比取 2 份多）

**同分子比较（分子相同）：** 分母小的大

> 1/3 > 1/5（同样取 1 份，3 等份时每份比 5 等份大）

#### 分数和整数 1 的关系

> 当分子 = 分母时，分数 = 1

> 2/2 = 1，5/5 = 1，100/100 = 1

> 当分子 < 分母时，分数 < 1（还没凑成一整个）

### 五下深入分数（第4单元）

#### 真分数、假分数、带分数

| 类型 | 定义 | 例子 | 大小 |
|------|------|------|------|
| 真分数 | 分子 < 分母 | 3/4, 1/2, 5/7 | < 1 |
| 假分数 | 分子 ≥ 分母 | 5/4, 7/3, 8/8 | ≥ 1 |
| 带分数 | 整数 + 真分数 | 1又3/4, 2又1/2 | > 1 |

**假分数 ↔ 带分数 互化：**

```
7/4 = 4/4 + 3/4 = 1 + 3/4 = 1又3/4
（7÷4=1余3，商是整数部分，余数是新分子）
```

```
2又1/3 = 6/3 + 1/3 = 7/3
（整数×分母+分子 = 新分子，分母不变）
```

#### 约分（用最大公因数）

把分数化简，分子分母同除以它们的**最大公因数（GCF）**：

```
12/18 → 最大公因数是 6
12 ÷ 6 = 2，18 ÷ 6 = 3
化简结果：2/3 ← 最简分数
```

**最简分数**：分子和分母的最大公因数是 1（互质）。

#### 通分（用最小公倍数）

要比较或加减异分母分数，先通分——把分母化成**最小公倍数（LCM）**：

```
比较 2/3 和 3/4：
LCM(3,4) = 12
2/3 = 8/12，3/4 = 9/12
因为 8/12 < 9/12，所以 2/3 < 3/4
```

#### 分数与除法的关系

> **a ÷ b = a/b**（b ≠ 0）

这是分数最深刻的含义：分数就是"除不尽时的精确表达"。

```
3 ÷ 4 = 3/4（三个苹果平均分给 4 个人，每人 3/4 个）
7 ÷ 2 = 7/2 = 3又1/2（7 块饼干分给 2 人，每人 3.5 块）
```

## 🎓 给 12+（进阶版）

### 有理数：分数的代数定义

**有理数（rational number）** 定义为所有能写成 a/b（a, b 是整数，b≠0）的数。符号：ℚ

```
整数 ⊂ 有理数：5 = 5/1, -3 = -3/1
有限小数 ⊂ 有理数：0.75 = 3/4
循环小数 ⊂ 有理数：0.333… = 1/3
```

有理数在数轴上"稠密"（任意两个有理数之间有无穷多个有理数），但并不覆盖所有实数——还有无理数（irrational numbers）：√2, π, e……

### 无理数的发现

古希腊毕达哥拉斯学派相信"万物皆整数比"（即所有数都是分数）。约公元前 5 世纪，学派成员希帕索斯（Hippasus）发现 **√2 不能写成分数**（无理数）——传说他因此被投入海中。

**证明 √2 是无理数（反证法）：**

假设 √2 = p/q（最简分数），则 2 = p²/q²，即 p² = 2q²。
→ p² 是偶数，→ p 是偶数，设 p = 2k
→ 4k² = 2q²，即 q² = 2k²，→ q 是偶数
→ p, q 都是偶数，矛盾（假设最简分数）
→ √2 不是有理数 ∎

### 分数在密码学中的应用

RSA 加密中，"模逆元"计算用到扩展欧几里得算法，本质上是求模意义下的"分数"。有限域中的分数（模 p 意义下 a/b = a×b⁻¹ mod p）是现代密码学的基础。

### 连分数

π 有一种无限持续的分数表示（连分数），比有理数近似更精确：

```
π = 3 + 1/(7 + 1/(15 + 1/(1 + 1/(292 + …))))
```

连分数在数论、密码学（格密码）中有重要应用。

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么 1/4 比 1/2 小？"** → 可以说："把同一块饼干切成 2 份，每份很大；切成 4 份，每份就小了。分母是'切成几份'，份数越多，每份就越小。"

- 孩子可能问 **"假分数有什么用？"** → 可以说："假分数和带分数表示同一个数，只是写法不同。假分数算起来方便（直接乘除），带分数看起来直观（知道有几个整的）。根据情况选哪种用。"

- 孩子可能问 **"3÷4 怎么等于 3/4？"** → 可以说："3 块饼干分给 4 个人，每人一份。4 份里每人 1 份=1/4，3 块就是 3 个 1/4，即 3/4。分数就是'除法的另一种写法'。"

- **共读小活动：** 用 12 块积木，让孩子先分成 4 份（每份 3 块），取其中 3 份，问"取了几分之几？"（3/4）。再分成 6 份取 4 份（4/6=2/3），让孩子发现约分后是一样的量。
