# 运算律

> **一句话**：加法和乘法有几条"不管怎么换顺序、怎么分组，答案都不变"的规律，叫做运算律。
> **English**: Arithmetic laws are rules that say you can change the order or grouping of addition and multiplication without changing the answer.

---

## 🧒 给 3-6 岁（共读版）

你有 3 个苹果，我有 5 个苹果，放在一起是几个？

3 + 5 = **8** 个。

那我的 5 个和你的 3 个放在一起呢？

5 + 3 = **8** 个。

答案一样！**顺序换了，答案不变。** 这就是加法的一个大规律，叫做**交换律**——就像"你先进电梯还是我先进电梯，电梯里一共都是 2 个人"。

**共读小活动：** 拿出 4 颗葡萄干放左边，2 颗放右边，数总数；再把右边的挪到左边，数总数——是不是一样？

## 📚 给 6-12 岁（自读版）

### 五大运算律总览

| 名称 | 适用运算 | 公式 | 记忆方法 |
|------|---------|------|---------|
| 加法交换律 | 加法 | a + b = b + a | 换顺序，和不变 |
| 加法结合律 | 加法 | (a+b)+c = a+(b+c) | 换括号，和不变 |
| 乘法交换律 | 乘法 | a × b = b × a | 换顺序，积不变 |
| 乘法结合律 | 乘法 | (a×b)×c = a×(b×c) | 换括号，积不变 |
| 乘法分配律 | 乘法+加法 | a×(b+c) = a×b + a×c | **最难！** 乘法"分发"进括号 |

> **注意：** 减法和除法**不满足**交换律！5 - 3 ≠ 3 - 5；6 ÷ 2 ≠ 2 ÷ 6。

### 加法交换律

> **a + b = b + a**

**例：** 37 + 48 = 48 + 37 = 85

用处：先想好哪个加哪个比较顺手，再算。

### 加法结合律

> **(a + b) + c = a + (b + c)**

**例：** 用结合律计算 25 + 37 + 75

```
25 + 37 + 75
= 25 + 75 + 37   ← 先用交换律换顺序
= (25 + 75) + 37 ← 再用结合律换括号
= 100 + 37
= 137
```

技巧：凑成整十、整百再算，快得多！

### 乘法交换律 & 结合律

> **a × b = b × a**
> **(a × b) × c = a × (b × c)**

**例：** 计算 25 × 37 × 4

```
25 × 37 × 4
= 25 × 4 × 37    ← 交换律
= (25 × 4) × 37  ← 结合律
= 100 × 37
= 3700
```

凑整百：25 × 4 = 100，乘 37 一下就出来了！

### 乘法分配律（核心难点）

> **a × (b + c) = a × b + a × c**

**正向（展开）：** 4 × (20 + 3) = 4×20 + 4×3 = 80 + 12 = 92

**逆向（提公因数）：** 37×5 + 37×5 = 37×(5+5) = 37×10 = 370

**易错陷阱：**

```
错误：4 × (20 + 3) = 4×20 + 3      ← 漏乘了 3 前面的 4！
正确：4 × (20 + 3) = 4×20 + 4×3
```

乘号外面的数（这里是 4）必须分别和括号里**每一个数**相乘。

**简便计算例题：**

```
99 × 8
= (100 - 1) × 8
= 100×8 - 1×8       ← 分配律（含减法版本）
= 800 - 8
= 792
```

这里用了"减法分配律"：a × (b - c) = a×b - a×c，逻辑一样。

## 🎓 给 12+（进阶版）

### 代数结构：群、环、域

运算律不是数字专属的，而是**代数结构**的性质：

| 结构 | 要求 |
|------|------|
| 半群 | 满足结合律 |
| 群（Group） | 半群 + 单位元 + 逆元 |
| 交换群（阿贝尔群） | 群 + 交换律 |
| 环（Ring） | 两种运算：加法构成交换群，乘法满足结合律，且乘法对加法满足分配律 |
| 域（Field） | 环 + 乘法也构成交换群（即有除法） |

整数 ℤ 是**环**（除法不一定有整数结果），有理数 ℚ 是**域**。

**分配律是"环"的核心要求：** 没有分配律，加法和乘法就是两套互不相干的运算；有了分配律，它们才能互动，才有多项式展开、因式分解等操作。

### 矩阵中的运算律

矩阵（Matrix）满足加法交换律和乘法结合律，**但不满足乘法交换律**：

$$AB \neq BA \quad (\text{一般情况下})$$

这说明"交换律"不是所有乘法都有的，它是实数乘法的"额外特性"，不是天然规律。线性代数里大量技巧依赖于此区别。

### 分配律与多项式展开

分配律是多项式展开和因式分解的基础：

$$(a + b)^2 = a^2 + 2ab + b^2$$

展开过程：

```
(a+b)(a+b)
= a(a+b) + b(a+b)    ← 分配律（第一层）
= a² + ab + ba + b²  ← 分配律（第二层）
= a² + 2ab + b²      ← 乘法交换律 ab=ba
```

高中数学的因式分解、配方法、韦达定理，都是运算律的深层应用。

## 🗣️ 家长讲解话术

- 孩子可能问 **"减法也有交换律吗？"** → 可以说："没有！5 - 3 = 2，但 3 - 5 = -2（负数），不一样。运算律只有加法和乘法才有，减法和除法都不能乱换顺序。"

- 孩子可能问 **"为什么乘法分配律那么难记？"** → 可以说："因为乘号外面的数要'分给'括号里的**每一个数**，一个都不能漏。可以想象成：老师发糖，4 颗糖要分给班上每个同学，不能只分给一部分人。"

- 孩子可能问 **"学这些有什么用？"** → 可以说："用来'简便计算'！比如 99×12，如果硬算很麻烦；但用分配律：99×12 = (100-1)×12 = 1200 - 12 = 1188，心算就出来了。这就是'聪明算法'。"

- **共读小活动：** 出几道看起来很难的计算：125×32、99×25、101×36，让孩子试着用运算律找"凑整"的方法，看谁能不用竖式最快算出来。
