# 平移与旋转

> **一句话**：平移是"往一个方向滑动"，旋转是"绕着一个点转圈"——图形的形状和大小都不变，只是位置或朝向变了。
> **English**: Translation means sliding a shape in a direction without turning it; rotation means spinning it around a fixed point — the shape stays the same, only its position or orientation changes.

---

## 🧒 给 3-6 岁（共读版）

你玩过滑滑梯吗？从上面滑到下面，你的身体形状没有变，只是换了个位置——这就是**平移**（translation）！

你玩过转椅子吗？坐在上面转转转，你的身体还是你，但面朝的方向变了——这就是**旋转**（rotation）！

**平移 = 滑动**：方向对了，距离到了，就到了新地方。

**旋转 = 转动**：绕着一个点，转一个角度，就朝向了新方向。

**共读小活动：** 找一把小椅子，先推着它向右移动 3 步（平移），再把它在原地转个圈（旋转）——感受两种运动的不同！

## 📚 给 6-12 岁（自读版）

### 平移（Translation）

**描述平移需要两个要素：**
1. **方向**：向左、向右、向上、向下，或斜向
2. **距离**：移动了几格（几厘米）

**在格子图上画平移：**

```
原图形（三角形）：
  ▲
 ███

向右平移 4 格：
          ▲
         ███
```

步骤：图形上每一个点，都向同一个方向移动**相同的距离**。

**平移的不变量：**
- 形状不变 ✓
- 大小不变 ✓
- 方向/朝向不变 ✓
- 位置变了（移动了）

### 旋转（Rotation）

**描述旋转需要三个要素：**
1. **旋转中心**：绕哪个点转（图形上的点，或图形外的点）
2. **旋转方向**：顺时针（clockwise）或逆时针（counterclockwise）
3. **旋转角度**：转了多少度（90°、180°、270°等）

**在格子图上画旋转（90°为例）：**

以图形右下角为旋转中心，顺时针旋转 90°：
- 原本向上伸出的部分，转到向右边
- 原本向右伸出的部分，转到向下

技巧：先确定旋转中心，再找图形的几个关键顶点，分别计算每个顶点转动后的新位置，最后连线。

**旋转的不变量：**
- 形状不变 ✓
- 大小不变 ✓
- 旋转中心到图形上每个点的距离不变 ✓
- 方向/朝向变了（转过了角度）

### 平移 vs 旋转 vs 翻折对比

| 变换 | 改变什么 | 不变什么 |
|------|---------|---------|
| 平移 | 位置 | 形状、大小、朝向 |
| 旋转 | 位置、朝向 | 形状、大小 |
| 轴对称（翻折） | 位置、朝向（镜像） | 形状、大小 |

这三种变换合称**刚体运动（isometry）**——"刚体"意味着形状不变形。

### 生活中的例子

**平移：**
- 推拉抽屉（水平平移）
- 电梯上下（竖直平移）
- 平移图案（地砖花纹重复）

**旋转：**
- 时钟指针（以表盘中心为旋转中心，顺时针）
- 风扇叶片（旋转中心在轴心）
- 地球自转（旋转中心是地轴）
- 旋转门（旋转中心在门轴）

### 图案中的对称与旋转

很多美丽的图案都是通过平移和旋转叠加生成的：
- 地砖：平移重复基本图案
- 雪花：六次旋转对称（每 60° 一次）
- 风车图案：旋转对称

## 🎓 给 12+（进阶版）

### 向量描述平移

在坐标系中，平移用**向量**（vector）精确表示：

点 P(x, y) 经过向量 **v** = (a, b) 的平移，得到 P'(x+a, y+b)。

**例：** 三角形三个顶点 A(1,2), B(3,2), C(2,4) 向右 3、向下 1 平移后：
- A' = (1+3, 2-1) = (4, 1)
- B' = (3+3, 2-1) = (6, 1)
- C' = (2+3, 4-1) = (5, 3)

### 旋转矩阵

在平面坐标系中，绕原点旋转角度 θ 的变换用**旋转矩阵**表示：

$$\begin{pmatrix} x' \\ y' \end{pmatrix} = \begin{pmatrix} \cos\theta & -\sin\theta \\ \sin\theta & \cos\theta \end{pmatrix} \begin{pmatrix} x \\ y \end{pmatrix}$$

**例：点 (1, 0) 顺时针旋转 90°（θ = -90°）：**

$$\begin{pmatrix} x' \\ y' \end{pmatrix} = \begin{pmatrix} 0 & 1 \\ -1 & 0 \end{pmatrix} \begin{pmatrix} 1 \\ 0 \end{pmatrix} = \begin{pmatrix} 0 \\ -1 \end{pmatrix}$$

结果 (0, -1)，从正 x 轴转到负 y 轴，符合顺时针 90° 的预期。

旋转矩阵是**正交矩阵**（满足 $R^T R = I$），行列式为 1，它是"保长度、保角度"的线性变换。

### 刚体运动群（等距变换群）

平移和旋转（以及反射）合在一起，构成平面上的**等距变换群（Isometry Group）**：

- **平移群**：所有平移的集合，构成阿贝尔群
- **旋转群 SO(2)**：绕某点的所有旋转，构成李群
- **欧氏群 E(2)**：所有平移 + 旋转 + 反射，是完整的平面等距变换群

这个结构在物理学中非常重要：**经典力学的对称性**（空间平移对称 → 动量守恒；旋转对称 → 角动量守恒），这就是**诺特定理**（Noether's theorem）的核心思想。

### Python 可视化变换

```python
import numpy as np
import matplotlib.pyplot as plt

def translate(points, dx, dy):
    """平移"""
    return points + np.array([dx, dy])

def rotate(points, angle_deg, center=(0, 0)):
    """绕 center 旋转 angle_deg 度"""
    theta = np.radians(angle_deg)
    R = np.array([[np.cos(theta), -np.sin(theta)],
                  [np.sin(theta),  np.cos(theta)]])
    p = points - np.array(center)
    return (R @ p.T).T + np.array(center)

# 定义一个三角形
triangle = np.array([[0, 0], [2, 0], [1, 2], [0, 0]])

# 平移和旋转
tri_translated = translate(triangle, 3, 1)
tri_rotated    = rotate(triangle, 90, center=(1, 1))

fig, axes = plt.subplots(1, 3, figsize=(12, 4))
for ax, pts, title in zip(axes,
    [triangle, tri_translated, tri_rotated],
    ['原图', '平移(+3,+1)', '旋转90°(中心(1,1))']):
    ax.plot(pts[:,0], pts[:,1], 'b-o')
    ax.set_xlim(-1, 6); ax.set_ylim(-1, 5)
    ax.set_aspect('equal'); ax.grid(True)
    ax.set_title(title)
plt.tight_layout(); plt.show()
```

## 🗣️ 家长讲解话术

- 孩子可能问 **"平移和旋转有什么不一样？"** → 可以说："平移就像推箱子——方向和距离固定了，整体挪过去，朝向不变；旋转就像转门把手——绕着一个固定点转，朝向会变，但每个部分到旋转中心的距离不变。"

- 孩子可能问 **"旋转要怎么确定角度？"** → 可以说："想象你站在旋转中心，看着图形上某个点——它转了多少度就是旋转角度。90° 是四分之一圈，180° 是半圈（刚好翻到对面），360° 是转一整圈回到原位。"

- 孩子可能问 **"旋转和翻折有什么区别？"** → 可以说："旋转是在平面上转，不会'翻面'，就像在桌上转一个纸片；翻折是像翻书页一样翻到'镜像'——翻折后，原本写的字会变成镜像反转，而旋转不会。"

- **共读小活动：** 在方格纸上画一个简单的 L 形图案，然后：1) 向右平移 4 格画出新图；2) 以某个顶点为中心顺时针旋转 90° 画出新图——对比三个图形的位置和朝向。
