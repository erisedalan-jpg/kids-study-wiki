# 方程

> **一句话**：方程是含有未知数的等式，解方程就是找出让等式成立的那个数。
> **English**: An equation is a statement that two expressions are equal; solving it means finding the value that makes the equation true.

---

## 🧒 给 3-6 岁（共读版）

想象一架**天平**：左边放了 3 个苹果，右边也放了 3 个苹果——天平平衡了！

现在左边放 x 个苹果和 2 个苹果，右边放 5 个苹果——天平还平衡。那 x 是几个？

x + 2 = 5，把 2 个苹果从左边拿走，右边也拿走 2 个，剩下 x = 3！

这个用字母代表未知数量，再想办法找出它的数学游戏，就叫**方程**（equation）。

**共读小活动：** 找一个玩具天平，左边放 4 个积木，右边放 2 个积木，问孩子"再放几个右边才能平？"——这就是 x + 2 = 4 的直觉版本。

## 📚 给 6-12 岁（自读版）

### 用字母表示数

在数学里，我们用字母（通常是 x、y、a、b 等）来表示**未知的数**或**变化的数**。

**例子：**
- 小明有 x 元，买了一支 2 元的铅笔，还剩 5 元 → x - 2 = 5
- 长方形的周长 = 2 × (长 + 宽)，写成：C = 2(l + w)

字母可以代表任何数——这是代数最强大的思想！

---

### 什么是方程？

**方程**：含有**未知数**的**等式**。

| 式子 | 是不是方程？ | 原因 |
|------|------------|------|
| 3 + 5 = 8 | 不是 | 没有未知数 |
| x + 3 | 不是 | 不是等式 |
| x + 3 = 8 | ✅ 是 | 有未知数，是等式 |
| 2x - 1 = 9 | ✅ 是 | 有未知数，是等式 |

**解（solution）**：让方程成立的未知数的值。如 x + 3 = 8 的解是 x = 5。

---

### 等式的性质（解方程的依据）

**天平原理：** 等式两边同时做同样的事，等式仍然成立。

| 性质 | 数学表示 | 含义 |
|------|---------|------|
| 两边同加相同的数 | a = b → a+c = b+c | 天平两边同加同样重物 |
| 两边同减相同的数 | a = b → a-c = b-c | 天平两边同取走同样重物 |
| 两边同乘相同的数 | a = b → a×c = b×c | 两边同时变成若干倍 |
| 两边同除相同的数（≠0） | a = b → a÷c = b÷c | 两边同时缩小若干倍 |

---

### 解一元一次方程

**步骤：** 用等式性质，把 x 单独"分离"到等号一边。

**例 1：** x + 5 = 12

```
x + 5 = 12
x + 5 - 5 = 12 - 5    ← 两边同减 5
x = 7
```

验证：7 + 5 = 12 ✓

**例 2：** 3x = 21

```
3x = 21
3x ÷ 3 = 21 ÷ 3       ← 两边同除以 3
x = 7
```

验证：3 × 7 = 21 ✓

**例 3：** 2x + 4 = 18（两步方程）

```
2x + 4 = 18
2x + 4 - 4 = 18 - 4   ← 先两边减 4
2x = 14
2x ÷ 2 = 14 ÷ 2       ← 再两边除以 2
x = 7
```

验证：2 × 7 + 4 = 18 ✓

---

### 列方程解应用题

用方程解应用题比算术更强大，尤其是复杂题型。

**步骤：**
1. **设** 未知数（"设 x 为……"）
2. **列** 方程（根据题意写出等式）
3. **解** 方程（用等式性质求 x）
4. **验** 答案（代入原题检查，写结论）

**例题：** 一本书，已读了 45 页，还剩 x 页未读，全书共 120 页，求 x。

```
设剩余页数为 x 页
列方程：45 + x = 120
解：x = 120 - 45 = 75
验：45 + 75 = 120 ✓
答：还剩 75 页未读。
```

---

### 方程 vs 算术

| 比较 | 算术方法 | 方程方法 |
|------|---------|---------|
| 思路 | 逆推（从答案推条件） | 顺推（直接列出关系） |
| 适合 | 简单题 | 复杂题 |
| 优势 | 直觉 | 系统化，可扩展 |

## 🎓 给 12+（进阶版）

### 一元二次方程

当 x 出现平方，就是二次方程：ax² + bx + c = 0

**求根公式（配方法推导）：**

```
x = (-b ± √(b²-4ac)) / (2a)
```

判别式 Δ = b² - 4ac：
- Δ > 0：两个不同实数根
- Δ = 0：两个相等实数根（重根）
- Δ < 0：无实数根（有复数根）

**例：** x² - 5x + 6 = 0

```
Δ = 25 - 24 = 1 > 0
x = (5 ± 1) / 2
x₁ = 3，x₂ = 2
```

验证：(x-3)(x-2) = 0，因式分解更简便。

---

### 丢番图方程

**丢番图方程（Diophantine equation）** 要求解必须是整数。

最简单的丢番图方程：ax + by = c（其中 a, b, c 为整数）

由**贝祖定理（Bézout's identity）**：当且仅当 gcd(a, b) | c 时，方程有整数解。

**著名例子：**
- **费马大定理（Fermat's Last Theorem）**：xⁿ + yⁿ = zⁿ（n > 2）没有正整数解。这个命题由费马于 1637 年提出，直到 1995 年才被安德鲁·怀尔斯证明，历时 358 年！

---

### 伽罗瓦理论与方程的可解性

一元五次及以上方程（n ≥ 5）**没有通用求根公式**——这不是数学家"还没找到"，而是被**伽罗瓦（Galois）** 在 19 世纪用群论彻底证明了不存在。

伽罗瓦在 20 岁决斗前夜写下了关键证明，开创了群论，深刻影响了整个现代代数。这是数学史上最浪漫的故事之一。

**群论的核心问题：** 方程的求根公式能否用加减乘除和开方表达，取决于方程的"伽罗瓦群"是否可解。

---

### 方程在物理中：建模思想

物理中几乎所有定律都是方程：
- 牛顿第二定律：F = ma
- 爱因斯坦质能方程：E = mc²
- 薛定谔方程（量子力学核心）

从算术到方程，是从"计算具体数"到"描述关系"的思维飞跃，这正是科学的语言。

## 🗣️ 家长讲解话术

- 孩子可能问 **"x 是什么意思？"** → 可以说："x 就是一个我们还不知道的数，用字母占个位置。就像谜语里说'我是谁？'——方程就是用算式来描述这个谜，解方程就是猜出谜底。"

- 孩子可能问 **"为什么两边同加同减，等式还成立？"** → 可以说："想想天平：如果两边一样重，同时在两边各加 1 克，还是一样重。等式就是数的天平，两边做同样的事，平衡不会破坏。"

- 孩子可能问 **"列方程和直接算哪个更好？"** → 可以说："简单题用直接算更快；复杂题（比如未知数出现在中间）用方程更清晰、不容易出错。方程是一个更强大的工具，掌握了就不会退回去的。"

- **共读小活动：** 家长出题："我想到了一个数，乘以 3 再加 4 等于 19，这个数是多少？"让孩子先猜，再用方程来解：3x + 4 = 19。
