# 折线统计图

> **一句话**：折线统计图用连起来的折线显示数据随时间的变化趋势——它不只告诉你"多少"，还告诉你"变快变慢、上涨下跌"。
> **English**: A line chart uses connected line segments to show how data changes over time — it reveals trends, rises, and falls at a glance.

---

## 🧒 给 3-6 岁（共读版）

你有没有量过身高？妈妈在墙上给你做了一个个记号，每年量一次。

把这些记号的高度用点画出来，再用线把点连起来——你就画出了一条**折线**！线往上走，说明你长高了；如果线走平了，说明这段时间没怎么长。

这就是**折线统计图**（line chart）！它最厉害的地方是：**一眼看出变化**。

**共读小活动：** 翻出你的成长手册或体检记录，把每次身高画成一个点，用铅笔把点连起来，看看哪段长得最快？

## 📚 给 6-12 岁（自读版）

### 折线统计图的组成

一张完整的折线统计图包括：

| 部分 | 说明 |
|------|------|
| **标题** | 说明图表描述的内容 |
| **横轴（x 轴）** | 通常是时间（年、月、日、星期…） |
| **纵轴（y 轴）** | 数据的数值，附带单位 |
| **数据点（折点）** | 每个实际数据值的位置 |
| **折线** | 连接相邻数据点的线段 |
| **数据标注** | 折点旁边标出的具体数值（可选） |

### 读懂折线图的三步法

**第一步：看整体**

线从左到右整体向上 → **上升趋势**（increasing trend）
线整体向下 → **下降趋势**（decreasing trend）
线基本平稳 → **稳定**（stable）

**第二步：看细节**

哪段线最陡 → 变化最快（增减最多）
哪段线最平 → 变化最慢（接近稳定）
折点的高低 → 对应时间点的具体数据值

**第三步：分析原因**

为什么某段时间上升/下降？结合实际情况思考。

### 折线图 vs 条形统计图

| 比较项目 | 条形统计图 | 折线统计图 |
|---------|-----------|-----------|
| 展示重点 | **各类别数量的大小** | **数据随时间的变化趋势** |
| 适合数据 | 类别数据（不同城市、不同班级…） | 时间序列数据（每月气温、每年人口…） |
| 读图感受 | 哪个最多/最少 | 怎么变化的 |
| 典型例子 | 各班同学人数 | 一周内每天温度变化 |

**选图技巧**：
- 问"哪个更多？" → 用条形图 → 见[[050-条形统计图|条形统计图]]
- 问"怎么变化的？" → 用折线图

### 画折线统计图的步骤

1. **确定数据**：整理好时间和对应数值的表格
2. **画坐标轴**：横轴（时间）、纵轴（数值）
3. **确定刻度**：纵轴从 0 或合适的数开始，间隔均匀
4. **描点**：根据数据在坐标系中找到对应位置，画点
5. **连线**：用直线段依次连接相邻的点
6. **标注**：在折点旁标出数值（可选），写标题和单位

### 注意事项

- 纵轴不从 0 开始时，要画"折断线"（~~）表示截断，避免误导读者
- 相邻点之间的连线是直线段，不是曲线（真正的曲线图叫平滑曲线图）
- 折线图适合连续时间序列，不适合离散的不相关类别

## 🎓 给 12+（进阶版）

### 时间序列分析

折线图是**时间序列分析**（time series analysis）的起点。时间序列数据的几个基本特征：

- **趋势（Trend）**：长期上升或下降的方向
- **季节性（Seasonality）**：周期性规律（每年夏天气温高）
- **周期性（Cycle）**：较长的波动周期（经济周期）
- **不规则波动（Irregular）**：随机噪声

统计学家用移动平均（moving average）、线性回归等方法来提取趋势、过滤噪声。

### 股票 K 线图

K 线图（Candlestick chart，也叫蜡烛图）是一种复杂的折线图变体，每个"蜡烛"表示一段时间（1天/1小时）内的价格信息：

| 部分 | 含义 |
|------|------|
| 上影线 | 最高价 |
| 实体顶 | 开盘/收盘价（较高者） |
| 实体底 | 开盘/收盘价（较低者） |
| 下影线 | 最低价 |

红色/绿色实体：收盘价高于开盘价（上涨）或低于开盘价（下跌）。

### 用 Python 画折线图

```python
import matplotlib.pyplot as plt

months = ['1月', '2月', '3月', '4月', '5月', '6月']
temperature = [3, 5, 12, 18, 23, 28]  # 单位：摄氏度

plt.figure(figsize=(8, 4))
plt.plot(months, temperature, marker='o', color='tomato', linewidth=2)
plt.title('某城市上半年平均气温变化')
plt.xlabel('月份')
plt.ylabel('温度 (°C)')
plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()
```

`marker='o'` 让每个数据点显示为圆点；`linewidth=2` 控制线的粗细。

### 双折线图与比较

一张图上画两条折线，可以直观比较两组数据的变化趋势——比如男女身高增长的对比，或两地气温的对比。用不同颜色或线型（实线/虚线）区分，并加图例（legend）。

## 🗣️ 家长讲解话术

- 孩子可能问 **"折线图和条形图有什么不一样？"** → 可以说："条形图的柱子一个个站在那里，用来比较'谁更高'——适合比较不同类别的数量。折线图的线连起来，用来看'怎么变化'——适合看一件事随时间的变化。比如，比较各班人数用条形图；看某班这一学期每次考试成绩的变化用折线图。"

- 孩子可能问 **"为什么连线要用直线段而不是曲线？"** → 可以说："因为我们只知道折点处的真实数据，两点之间的数据未知，用直线段是最诚实的做法——表示我们'不知道中间是怎么变化的，就先假设是匀速变化'。如果用曲线，反而会让人以为我们知道中间的精确变化，那就'说了我们不知道的事情'了。"

- 孩子可能问 **"纵轴为什么有时候不从 0 开始？"** → 可以说："如果数据值都在 90-100 之间，从 0 开始画，所有折点挤在图的最顶端，看不清变化。从 85 开始，变化就显得很清晰。但这样要小心，折点之间高低差距看起来被'放大'了，要注意纵轴的刻度，避免被图表误导。"

- **共读小活动：** 记录一周内每天喝水的杯数，画成折线图，看看哪天喝得多、哪天喝得少，分析原因。
