# 圆锥

> **一句话**：底面是圆、顶上尖尖的立体图形，体积是同底同高圆柱的三分之一。
> **English**: A cone has a circular base and a pointed top (apex); its volume is one-third that of a cylinder with the same base and height.

---

## 🧒 给 3-6 岁（共读版）

你吃过冰激凌吗？那个脆脆的甜筒——底部圆圆的，往上越来越细，最后是一个尖尖的角——那就是**圆锥**的形状！

圆锥摸起来有什么特点？
- 底部是一个**圆形**（底面）
- 上面是一个**尖点**（顶点）
- 中间的面是**弯曲的**（侧面）

**共读小活动：** 一起找找家里或外面的圆锥形物体：冰激凌甜筒、生日帽、陀螺、路边的橙色路障锥……数一数你能找到几个！

## 📚 给 6-12 岁（自读版）

### 圆锥的三要素

| 要素 | 符号 | 说明 |
|------|------|------|
| 底面半径 | r | 底面圆的半径 |
| 高 | h | 从顶点垂直到底面的距离 |
| 母线 | l | 从顶点到底面圆边缘的线段 |

三者关系（勾股定理）：**l² = r² + h²**，即 l = √(r² + h²)

### 圆锥的面

| 面 | 数量 | 说明 |
|----|------|------|
| 底面（圆形） | 1 | 圆，面积 = πr² |
| 侧面（弯曲） | 1 | 展开后是**扇形** |
| 顶点 | 1 | 一个尖角 |

**侧面展开图**：把圆锥的侧面剪开展平，得到一个扇形。
- 扇形的半径 = 圆锥的母线 l
- 扇形的弧长 = 底面圆的周长 = 2πr

### 体积公式

$$V_{\text{圆锥}} = \frac{1}{3} \pi r^2 h$$

也可以写成：**圆锥体积 = 底面积 × 高 ÷ 3**

**⚠️ 最常见易错点**：做题时忘记除以 3！记忆口诀：**"锥体体积要除三"**

### 圆锥与圆柱的关系

当圆锥和圆柱**底面相同、高相同**（等底等高）时：

```
圆柱体积 = πr²h
圆锥体积 = (1/3)πr²h

圆柱体积 = 3 × 圆锥体积
```

**实验验证**：用同底同高的圆锥形杯和圆柱形杯，把沙子或水倒 3 次从圆锥杯到圆柱杯，恰好装满——这就是 3 倍关系的直观证明！

### 典型例题

**例 1**：已知圆锥底面半径 r = 3 cm，高 h = 4 cm，求体积。
> V = (1/3) × π × 3² × 4 = (1/3) × π × 9 × 4 = 12π ≈ 37.7 cm³

**例 2**：一个等底等高的圆柱体积为 60 cm³，圆锥体积是多少？
> 圆锥体积 = 60 ÷ 3 = **20 cm³**

**例 3**：求上题圆锥的母线长（r = 3, h = 4）：
> l = √(3² + 4²) = √(9 + 16) = √25 = **5 cm**

### 生活中的圆锥

| 实物 | 说明 |
|------|------|
| 冰激凌甜筒 | 经典圆锥容器 |
| 生日尖帽 | 侧面是扇形 |
| 陀螺 | 旋转时稳定，利用圆锥形状 |
| 路障锥（雪糕桶） | 橙色圆锥路障 |
| 圣诞树 | 近似圆锥形 |
| 火山 | 火山锥是自然界的圆锥 |

## 🎓 给 12+（进阶版）

### 旋转体：直角三角形绕直角边旋转

圆锥是**旋转体（solid of revolution）**：将一个直角三角形绕其一条直角边旋转 360°，另一条直角边扫出的圆就是底面，斜边扫出的面就是侧面。

圆锥体积公式的积分推导：
```
V = ∫₀ʰ π [r(x)]² dx

其中 r(x) = (r/h)·x（半径从顶点到底面线性增加）

V = π(r/h)² ∫₀ʰ x² dx
  = π(r/h)² · [x³/3]₀ʰ
  = π(r/h)² · h³/3
  = πr²h/3
```

这正是小学学的公式的微积分证明。

### 圆锥曲线（Conic Sections）

用平面截圆锥，根据截面与轴的角度不同，得到四种截面：

| 截面方式 | 截面形状 | 英文 |
|---------|---------|------|
| 平行底面 | 圆 (circle) | circle |
| 斜切但不平行母线 | 椭圆 (ellipse) | ellipse |
| 平行一条母线 | 抛物线 (parabola) | parabola |
| 与轴平行（或更陡） | 双曲线 (hyperbola) | hyperbola |

这四种曲线统称**圆锥曲线（conic sections）**，是高中数学和大学解析几何的核心内容。

| 曲线 | 方程 | 应用 |
|------|------|------|
| 圆 | x² + y² = r² | 车轮、轨道 |
| 椭圆 | x²/a² + y²/b² = 1 | 行星轨道（开普勒第一定律） |
| 抛物线 | y = ax² | 卫星天线、汽车头灯反射 |
| 双曲线 | x²/a² − y²/b² = 1 | GPS 定位、声呐 |

### 表面积与优化

圆锥全面积（含底面）：
```
S = πrl + πr²
  = πr(l + r)
```

**优化问题**：给定圆锥体积 V，如何选择 r 和 h 使表面积最小？这是微积分优化的经典题目，答案是 h/r = √2（半开角 = arctan(1/√2) ≈ 35.26°）。

### 工程与自然中的圆锥

- **声音聚焦**：抛物面（圆锥曲线之一）能把声音或光汇聚到一点，用于卫星天线、望远镜镜面
- **纸杯设计**：圆锥形纸杯（饮水机旁边的尖底杯）比圆柱形纸杯省材料，且不能放置，迫使人一次用完
- **火箭鼻锥**：火箭顶部圆锥形设计，减少空气阻力（圆锥的曲面分流气体最优）

## 🗣️ 家长讲解话术

- 孩子可能问 **"圆锥和圆柱有什么不同？"** → 可以说："圆柱上下一样大，像易拉罐；圆锥下面是圆、上面是尖，像冰激凌甜筒。最大的区别是：同样大小的底面和高，圆柱装的东西是圆锥的 3 倍——所以吃冰激凌要选圆柱桶不选甜筒！"

- 孩子可能问 **"为什么圆锥体积是圆柱的三分之一？"** → 可以说："这个需要实验来感受——把同底同高的圆锥形杯装满沙子，往圆柱形杯里倒，倒 3 次恰好装满。这是古希腊人发现的，阿基米德证明了它。（高中学积分后你会看到数学证明）"

- 孩子可能问 **"母线是什么？"** → 可以说："母线是从顶点沿着圆锥表面连到底面圆边缘的线——就像你用手指从冰激凌尖尖沿边划到底部圆圈的那条线。母线、高、底面半径形成一个直角三角形，可以用勾股定理算。"

- **共读小活动：** 用纸做一个圆锥：剪一个扇形，卷成圆锥形并粘好，再剪一个圆形做底面。量一量扇形的半径（就是圆锥的母线）和扇形的弧长（就是底面的周长），验证 弧长 = 2πr。
