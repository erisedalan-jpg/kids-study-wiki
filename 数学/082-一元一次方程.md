## 课本一句话定义

**一元一次方程**是只含有**一个未知数**（元），且未知数的次数为 **1**（一次），并且最高次项系数不为 0 的方程；标准形式为 $ax = b$（$a \neq 0$）。

---

## 📚 给 6-12 岁（小学高年级回溯版）

"一元一次"的含义：
- **一元**：只有 1 个未知数（如只有 $x$，没有 $y$）
- **一次**：未知数的最高次方是 1（$x$，不是 $x^2$）

常见的一元一次方程：
- $x + 5 = 12$（解：$x = 7$）
- $2x - 3 = 7$（解：$x = 5$）
- $3(x - 1) = 9$（需要先去括号）

**判断是不是一元一次方程**：
1. 只有一个字母吗？
2. 这个字母的最高次是 1 吗？
3. 是等式吗（有 = 号）？

三个"是"才是一元一次方程。

## 🎓 给 12+（进阶版）

### 定义

**一元一次方程**：只含一个未知数，且未知数的最高次数为 1，整理后可写成：

$$ax + b = 0 \quad (a \neq 0, a, b \in \mathbb{Q})$$

或等价形式 $ax = c$，解为 $x = \dfrac{c}{a}$。

### 标准解题步骤

1. **去分母**：方程两边同乘各分数分母的最小公倍数（LCM）
2. **去括号**：用分配律展开括号
3. **移项**：含未知数的项移到左边，常数项移到右边
4. **合并同类项**：左边合并 $x$ 项，右边合并常数
5. **系数化为 1**：两边除以未知数系数

### 典型例题

**例 1**：解方程 $2x - 3 = x + 5$。

**解**：
移项：$2x - x = 5 + 3$
合并：$x = 8$
验证：左边 $= 2 \times 8 - 3 = 13$；右边 $= 8 + 5 = 13$ ✅

**例 2**：解方程 $\dfrac{x-1}{2} - \dfrac{x+2}{3} = 1$。

**解**：
去分母（×6）：$3(x-1) - 2(x+2) = 6$
去括号：$3x - 3 - 2x - 4 = 6$
合并：$x - 7 = 6$
移项：$x = 13$

**例 3**：若 $3(x - 1) - 2(x + a) = 0$ 的解为 $x = 5$，求 $a$。

**解**：将 $x = 5$ 代入：$3(5-1) - 2(5+a) = 0$，$12 - 10 - 2a = 0$，$2 - 2a = 0$，$a = 1$。

### 易错点

1. **系数不为 0**：$0 \cdot x = 0$ 不是方程，$0x = 3$ 无解。
2. **去分母后分子要加括号**：$\dfrac{x-1}{2}$ 两边乘 2 得 $x - 1$，整体在括号里。
3. **移项必须变号**：从左移到右，正变负，负变正。
4. **验证**：解完一定代入原方程验证。

### 知识联系

- 前置：方程（含义）、等式性质、有理数四则运算
- 后续：一元一次方程应用题 → 二元一次方程组（八年级）→ 一元二次方程（九年级）
- 高中拓展：函数零点，直线 $y = kx + b$ 中令 $y = 0$
