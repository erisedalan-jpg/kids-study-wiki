## 课本一句话定义

**去括号**时，若括号前是"+"号，去括号后各项符号**不变**；若括号前是"-"号，去括号后括号内各项符号**全部改变**（正变负，负变正）。

---

## 📚 给 6-12 岁（小学高年级回溯版）

去括号就像"打开礼品盒"：

- **括号前是 +**：打开后，里面的东西原样不变。
  $+(a - b) = a - b$

- **括号前是 -**：打开后，里面的每样东西都要"翻转"（正变负，负变正）。
  $-(a - b) = -a + b$

可以用**分配律**来理解：
- $+(a - b) = (+1) \times (a - b) = a - b$
- $-(a - b) = (-1) \times (a - b) = -a + b$

练习：去括号后是什么？

$3x + (2x - 1) = 3x + 2x - 1 = 5x - 1$

$3x - (2x - 1) = 3x - 2x + 1 = x + 1$（括号前是 -，括号内每项变号）

## 🎓 给 12+（进阶版）

### 法则

**法则 1（括号前为 +）**：

$$+(A + B - C) = A + B - C$$

括号内各项符号不变。

**法则 2（括号前为 -）**：

$$-(A + B - C) = -A - B + C$$

括号内各项符号全部改变。

**本质**：利用分配律 $k(a + b) = ka + kb$，其中 $k = \pm 1$。

### 扩展——系数展开

若括号前有系数（不仅仅是 ±1）：

$$2(3x - 4) = 6x - 8$$
$$-3(x - 2y) = -3x + 6y$$

### 典型例题

**例 1**：化简 $4x - (3x - 2y) + (x + y)$。

**解**：
$= 4x - 3x + 2y + x + y$（去括号，减号括号内变号，加号括号内不变）
$= (4x - 3x + x) + (2y + y)$
$= 2x + 3y$

**例 2**：化简 $3a - 2(a - b) - (2a + b - 1)$。

**解**：
$= 3a - 2a + 2b - 2a - b + 1$
$= (3a - 2a - 2a) + (2b - b) + 1$
$= -a + b + 1$

**例 3**：若 $3x - (2x - m) = x + 5$，求 $m$。

**解**：左边去括号：$3x - 2x + m = x + m$；右边：$x + 5$；故 $m = 5$。

### 易错点

1. **括号前是 - 必须每项变号**：$-(a - b + c) = -a + b - c$，不能只改第一项。
2. **系数的分配**：$-2(x - 3) = -2x + 6$，不是 $-2x - 6$。
3. **嵌套括号从内到外**：先去最内层括号，再逐层向外。
4. **去括号后再合并同类项**：两步骤顺序不能颠倒。

### 知识联系

- 前置：分配律（小学）、合并同类项
- 后续：整式加减（=去括号+合并同类项）→ 方程中移项
- 高中拓展：多项式乘法展开、行列式按行展开
