## 课本一句话定义

**同类项**是含有相同字母且对应字母指数也相同的项（常数项都是同类项）；**合并同类项**就是把同类项的系数相加，字母及其指数保持不变。

---

## 📚 给 6-12 岁（小学高年级回溯版）

想象整理文具：把同类的放在一起。

$3$ 支铅笔 $+ 2$ 支铅笔 $= 5$ 支铅笔

用字母：$3x + 2x = 5x$（$x$ 代表"铅笔"）

**同类项** = 字母一样、字母的次数也一样的项，可以合并：
- $3x$ 和 $5x$ 是同类项（都含 $x^1$）→ $3x + 5x = 8x$
- $3x^2$ 和 $5x$ **不是**同类项（一个 $x^2$，一个 $x$）→ 不能合并
- $3x$ 和 $3y$ **不是**同类项（字母不同）→ 不能合并

合并规则：**系数相加，字母不变**。

$4a^2 - 2a^2 = (4-2)a^2 = 2a^2$

## 🎓 给 12+（进阶版）

### 同类项的定义

两个（或更多）单项式满足以下条件时，互为**同类项**：
1. 含有**相同的字母**
2. 相同字母的**指数相同**

注意：系数不影响同类项的判断；常数项（无字母）互为同类项。

### 合并法则

$$a \cdot x^m y^n + b \cdot x^m y^n = (a + b) \cdot x^m y^n$$

法则：把系数相加（用有理数加法），字母部分保持不变。

### 典型例题

**例 1**：合并同类项：$3x^2 - 2x + 5x^2 + x - 7$。

**解**：
- $x^2$ 类：$3x^2 + 5x^2 = 8x^2$
- $x$ 类：$-2x + x = -x$
- 常数类：$-7$

结果：$8x^2 - x - 7$。

**例 2**：化简：$2ab - 3a^2b + 5ab - a^2b$。

**解**：
- $a^2b$ 类：$-3a^2b - a^2b = -4a^2b$
- $ab$ 类：$2ab + 5ab = 7ab$

结果：$-4a^2b + 7ab$。

**例 3**：若 $3x^m y^2$ 与 $-2x^3 y^n$ 是同类项，求 $m$、$n$。

**解**：同类项要求对应字母指数相同，故 $m = 3$，$n = 2$。

### 易错点

1. **字母顺序无关**：$ab$ 和 $ba$ 是同类项（乘法交换律）。
2. **指数必须完全相同**：$3x^2y$ 和 $2xy^2$ 不是同类项（指数分布不同）。
3. **系数相加含符号**：$5x + (-3x) = 2x$，不是 $8x$。
4. **不要改变字母部分**：合并后字母部分不变，只改系数。

### 知识联系

- 前置：单项式（系数、次数）、有理数加减
- 后续：去括号 → 整式加减 → 方程（移项后合并同类项）
- 高中拓展：多项式化简、待定系数法
