## 课本一句话定义

**射线**是从一个端点出发，向一个方向无限延伸的直线的一部分；射线有一个**端点**（起点），没有终点，长度无限，**不能度量**。

---

## 📚 给 6-12 岁（小学高年级回溯版）

射线就像光从手电筒射出去：
- 手电筒 = **端点**（有起点）
- 光向外无限延伸 = **无终点**

日常例子：
- 太阳光线（从太阳出发，向地球方向延伸）
- 激光笔射出的光束

**三种线的对比**：

| | 线段 | 射线 | 直线 |
|--|------|------|------|
| 端点 | 两个 | 一个 | 没有 |
| 长度 | 有限（可测量）| 无限 | 无限 |
| 画法 | 两端封闭 | 一端封闭，一端箭头 | 两端箭头 |

射线记作 $OA$（$O$ 是端点，$A$ 是射线上另一点，方向：从 $O$ 向 $A$ 延伸）。

**注意**：$OA$ 和 $AO$ 是不同的射线（端点不同，方向相反）！

## 🎓 给 12+（进阶版）

### 定义

**射线**（ray）$OA$：从点 $O$（端点）出发，经过点 $A$，向 $A$ 方向无限延伸的图形。

数学符号：通常记作 $\overrightarrow{OA}$（有时与向量混淆，注意上下文）。

**射线的有向性**：射线有方向，$\overrightarrow{OA}$ 和 $\overrightarrow{AO}$ 是完全不同的两条射线（方向相反，端点不同）。

### 关键性质

| 性质 | 内容 |
|------|------|
| 一个端点 | 只有起点，没有终点 |
| 无限延伸 | 长度无限，无法度量 |
| 方向确定 | 有唯一的方向 |
| 包含线段 | 射线上任意两点之间是线段 |
| 射线数量 | 过一点可作无数条射线（方向可任意）|

### 与角的关系

**角**是由两条有**公共端点**的射线构成的图形——这是角的定义基础。

$$\angle AOB: \text{两条射线 } OA \text{ 和 } OB \text{，共享端点 } O$$

角的顶点 = 射线的公共端点；角的两边 = 两条射线。

### 典型例题

**例 1**：直线 $l$ 上有一点 $O$，把直线 $l$ 分成几条射线？各是什么？

**解**：$O$ 将直线 $l$ 分成 **2 条**射线：$\overrightarrow{OA}$（$O$ 向左）和 $\overrightarrow{OB}$（$O$ 向右）。

**例 2**：平面内有点 $O$，从 $O$ 出发能画多少条射线？

**解**：可以画**无数条**射线（每个方向对应一条）。

**例 3**：射线 $OA$ 上有一点 $B$（$B$ 在 $O$ 和 $A$ 之间），$OB = 3$，$OA = 7$，求 $BA$。

**解**：$BA = OA - OB = 7 - 3 = 4$（$B, A$ 在同一射线上，故 $OA = OB + BA$）。

### 易错点

1. **射线的名称有顺序**：$\overrightarrow{OA}$ 以 $O$ 为端点，$\overrightarrow{AO}$ 以 $A$ 为端点——是两条不同的射线。
2. **射线无法度量长度**：只有线段才有长度（射线是"半无限"的）。
3. **射线 ≠ 箭头**：射线图示中的箭头只表示"无限延伸"，本身不是终点。

### 知识联系

- 前置：线段（有两端点）、点线面体
- 后续：直线（两端无限）→ 角（两射线夹角）→ 角的度量
- 高中拓展：向量（有方向和大小，从射线概念延伸）→ 极坐标（角度 + 射线）
