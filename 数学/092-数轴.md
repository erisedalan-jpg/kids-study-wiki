## 课本一句话定义

规定了**原点、正方向和单位长度**的直线叫做**数轴**；数轴上每一个点都对应唯一一个实数，每一个实数都对应数轴上唯一一个点。

---

## 📚 给 6-12 岁（小学高年级回溯版）

想象一把"无限长的尺子"：
- 中间的 0 叫**原点**，往右越来越大（正数），往左越来越小（负数）。
- 每两个刻度之间的距离叫**单位长度**，必须相等。

在小学，你用过温度计，温度计其实就是一条**竖着的数轴**！零度以上是正，零度以下是负。

**数轴三要素**（缺一不可）：
1. **原点**（标 0 的那个点）
2. **正方向**（用箭头表示）
3. **单位长度**（刻度间距）

---

## 🎓 给 12+（进阶版）

### 严格定义

数轴是满足以下三个条件的直线：
1. 确定一个点为**原点** $O$，对应数 $0$
2. 确定一个**正方向**（通常向右），对应正数
3. 确定**单位长度**（正方向上每格的长度）

建立数轴后，直线上每一个点与实数之间形成**一一对应**关系。

### 关键性质

| 性质 | 说明 |
|------|------|
| 右大左小 | 数轴上右边的数总大于左边的数 |
| 相反数关于原点对称 | $a$ 与 $-a$ 到原点距离相等，方向相反 |
| 两点间距离 | $\|a - b\|$（绝对值）|
| 正数在原点右侧 | 负数在原点左侧，0 在原点 |

### 数轴示意

```
←——|——|——|——|——|——|——|——|——→
  -4  -3  -2  -1   0   1   2   3   4
```

原点 $O$ 在 0 处，箭头方向（向右）为正方向。

### 典型例题

**例 1**：在数轴上表示 $-2.5$、$\dfrac{3}{2}$、$-\dfrac{1}{3}$。

**解**：
- $-2.5$ 在 $-3$ 和 $-2$ 之间（偏 $-3$ 侧）
- $\dfrac{3}{2} = 1.5$ 在 $1$ 和 $2$ 正中间
- $-\dfrac{1}{3} \approx -0.33$ 在 $0$ 和 $-1$ 之间靠近 0

**例 2**：数轴上 $A$ 点表示 $-3$，$B$ 点表示 $2$，求 $A$、$B$ 两点之间的距离。

**解**：$AB = |(-3) - 2| = |-5| = 5$（单位）。

### 易错点

1. **单位长度必须统一**：不能一格代表 1，另一格代表 2。
2. **方向不能遗漏**：没有箭头的直线不是数轴。
3. **原点不必须在中间**：原点可以画在任何位置，只要满足三要素即可。
4. **数轴上的点 ≠ 有理数点**：无理数（如 $\sqrt{2}$）也在数轴上，只是小学阶段不讨论。

### 知识联系

- 前置：正数与负数、整数、长度/刻度
- 后续：相反数（关于原点对称）→ 绝对值（到原点距离）→ 数的大小比较
- 高中拓展：实数轴、复平面（二维数轴推广）

### 词汇表

| 中文 | 英文 | 说明 |

| 数轴 | number line | 表示数的直线 |
| 原点 | origin | 对应 0 的点 |

### 例句

| 中文 | 英文 |
|------|------|
| 数轴上原点右边的数是正数。 | Numbers to the right of the origin on the number line are positive. |
