## 课本一句话定义

**整式加减**的实质是**去括号**后**合并同类项**；把几个整式（单项式或多项式）相加减，去掉括号后，将所有同类项合并，得到最简结果。

---

## 📚 给 6-12 岁（小学高年级回溯版）

整式加减就是两步操作的组合：

**第一步**：**去括号**（注意负号前的括号要变号）

**第二步**：**合并同类项**（把"同一类"的项加起来）

例子：$(3x + 2) + (x - 5)$

去括号：$3x + 2 + x - 5$（加号括号内不变号）

合并同类项：$(3x + x) + (2 - 5) = 4x - 3$

再来一个：$(3x + 2) - (x - 5)$

去括号：$3x + 2 - x + 5$（减号括号内**变号**！）

合并：$(3x - x) + (2 + 5) = 2x + 7$

## 🎓 给 12+（进阶版）

### 解题步骤

| 步骤 | 操作 |
|------|------|
| ① 去括号 | 加括号不变号，减括号每项变号 |
| ② 合并同类项 | 系数相加，字母部分不变 |
| ③ 降幂排列 | 按次数从高到低排列结果（标准形式）|

### 关键公式（特殊多项式加减）

常见的整式恒等式（为后续因式分解做准备）：

$$a^2 + 2ab + b^2 = (a+b)^2$$
$$a^2 - 2ab + b^2 = (a-b)^2$$
$$a^2 - b^2 = (a+b)(a-b)$$

（七上暂不要求，八年级正式学习，此处作为拓展认识）

### 典型例题

**例 1**：化简 $(2a^2 - 3ab + b^2) - (a^2 - 2ab - 3b^2)$。

**解**：
去括号：$2a^2 - 3ab + b^2 - a^2 + 2ab + 3b^2$

合并 $a^2$ 项：$2a^2 - a^2 = a^2$

合并 $ab$ 项：$-3ab + 2ab = -ab$

合并 $b^2$ 项：$b^2 + 3b^2 = 4b^2$

结果：$a^2 - ab + 4b^2$

**例 2**：若 $A = 3x^2 - 2x + 1$，$B = x^2 + x - 3$，求 $2A - B$。

**解**：
$2A - B = 2(3x^2 - 2x + 1) - (x^2 + x - 3)$

$= 6x^2 - 4x + 2 - x^2 - x + 3$

$= (6x^2 - x^2) + (-4x - x) + (2 + 3)$

$= 5x^2 - 5x + 5$

**例 3**：已知 $A + B = 5x^2 - 3x + 2$，$A = 2x^2 + x - 1$，求 $B$。

**解**：$B = (5x^2 - 3x + 2) - (2x^2 + x - 1)$

$= 5x^2 - 3x + 2 - 2x^2 - x + 1 = 3x^2 - 4x + 3$

### 易错点

1. **减法括号内必须全部变号**：$-(a - b + c) = -a + b - c$，不能只变第一项。
2. **合并时不漏项**：每个同类项都要参与合并，不要遗漏某项。
3. **结果降幂排列**：$x - 3 + 2x^2$ 写成标准形式为 $2x^2 + x - 3$。
4. **系数为 0 时省略**：$3x^2 - 3x^2 = 0$，这一项消失，不要写 $0x^2$。

### 知识联系

- 前置：去括号、合并同类项、单项式、多项式
- 后续：整式乘法（八年级）→ 因式分解 → 分式加减
- 高中拓展：多项式的加减与函数运算（$f(x) \pm g(x)$）
