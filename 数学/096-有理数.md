## 课本一句话定义

整数和分数统称为**有理数**，即所有可以写成两个整数之比（分母不为零）的数，包括正整数、负整数、0、正分数、负分数。

---

## 📚 给 6-12 岁（小学高年级回溯版）

你已经学过：
- **自然数**：0、1、2、3……（数东西用的）
- **整数**：……-3、-2、-1、0、1、2、3……（包括负整数）
- **分数**：$\dfrac{1}{2}$、$\dfrac{3}{4}$……

把这三类全部合起来，就叫**有理数**！

判断一个数是不是有理数，就问自己：**它能写成"一个整数除以另一个非零整数"的形式吗？**

- $0.5 = \dfrac{1}{2}$ ✅ 是有理数
- $-3 = \dfrac{-3}{1}$ ✅ 是有理数
- $\sqrt{2} = 1.41421\ldots$（无限不循环）❌ 不是有理数，是无理数

## 🎓 给 12+（进阶版）

### 严格定义

**有理数**（rational number）是能够表示为 $\dfrac{p}{q}$ 的数，其中 $p, q$ 均为整数且 $q \neq 0$。

有理数集合记作 $\mathbb{Q}$：

$$\mathbb{Q} = \left\{ \frac{p}{q} \mid p, q \in \mathbb{Z},\ q \neq 0 \right\}$$

### 有理数的分类

```
有理数
├── 整数
│   ├── 正整数（1, 2, 3, …）
│   ├── 0
│   └── 负整数（-1, -2, -3, …）
└── 分数
    ├── 正分数（1/2, 3/4, …）
    └── 负分数（-1/2, -3/4, …）
```

### 关键性质

| 性质 | 说明 |
|------|------|
| 有理数的十进制展开是有限小数或无限循环小数 | $\frac{1}{3} = 0.\overline{3}$ |
| 有理数对加减乘封闭 | 两有理数之和/差/积仍是有理数 |
| 有理数对除法（除数非零）封闭 | 两有理数之商（除数≠0）仍是有理数 |
| 有理数在数轴上稠密 | 任意两个有理数之间还有有理数 |

### 典型例题

**例 1**：下列各数哪些是有理数？$\dfrac{22}{7}$、$\pi$、$\sqrt{4}$、$-0.\overline{6}$、$\sqrt{3}$

**解**：
- $\dfrac{22}{7}$：整数比，**是有理数**（但不等于 $\pi$）
- $\pi = 3.14159\ldots$：无限不循环，**不是有理数**（无理数）
- $\sqrt{4} = 2$：整数，**是有理数**
- $-0.\overline{6} = -\dfrac{2}{3}$：循环小数可化分数，**是有理数**
- $\sqrt{3} = 1.732\ldots$：无限不循环，**不是有理数**

**例 2**：将 $0.\overline{36}$ 化为分数。

**解**：设 $x = 0.\overline{36}$，则 $100x = 36.\overline{36}$，两式相减：$99x = 36$，$x = \dfrac{36}{99} = \dfrac{4}{11}$。

### 易错点

1. **$\dfrac{22}{7} \neq \pi$**：$\dfrac{22}{7}$ 是有理数，$\pi$ 是无理数。
2. **循环小数都是有理数**：$0.\overline{9} = 1$（整数！）。
3. **有理数的稠密性不代表覆盖整个数轴**：有理数之间还有无理数。

### 知识联系

- 前置：整数、分数、正负数
- 后续：数轴、绝对值、有理数四则运算
- 高中拓展：实数 = 有理数 ∪ 无理数；复数 = 实数 ∪ 虚数
