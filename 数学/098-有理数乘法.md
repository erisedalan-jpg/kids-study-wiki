## 课本一句话定义

两数相乘，**同号得正，异号得负**，绝对值相乘；任何数与 0 相乘得 0；多个数相乘，负因数个数为偶数时积为正，为奇数时积为负。

---

## 📚 给 6-12 岁（小学高年级回溯版）

乘法的符号规律，用"方向"来理解：

想象"转向游戏"：
- **正数** = 不转向
- **负数** = 转向（掉头）

- 正 × 正 = 不转 × 不转 = **正** ✅
- 负 × 负 = 转 × 转 = 转了两次，又转回来 = **正** ✅（负负得正！）
- 正 × 负 = 不转 × 转 = **负** ✅
- 负 × 正 = 转 × 不转 = **负** ✅

口诀：**同号得正，异号得负，有零得零。**

例子：$(-3) \times (-4) = 12$（两负得正），$(-3) \times 4 = -12$（异号得负）。

## 🎓 给 12+（进阶版）

### 法则

**法则 1**：两个有理数相乘，同号得正，异号得负，绝对值相乘。

$$a \times b = \text{sign}(a) \cdot \text{sign}(b) \cdot |a| \cdot |b|$$

**法则 2**：任何数与 0 相乘，积为 0。

$$a \times 0 = 0$$

**法则 3（多因数）**：几个不为 0 的有理数相乘，积的符号由负因数的个数决定——负因数个数为**偶数**时积为正，为**奇数**时积为负。

### 关键性质

| 性质 | 公式 |
|------|------|
| 交换律 | $a \times b = b \times a$ |
| 结合律 | $(a \times b) \times c = a \times (b \times c)$ |
| 分配律 | $a \times (b + c) = a \times b + a \times c$ |
| 乘法单位元 | $a \times 1 = a$ |
| 乘法零元 | $a \times 0 = 0$ |
| 负负得正 | $(-a) \times (-b) = ab$ |

### 典型例题

**例 1**：计算 $(-3) \times (-\dfrac{2}{3}) \times 5 \times (-1)$。

**解**：负因数有 3 个（奇数），积为负。绝对值：$3 \times \dfrac{2}{3} \times 5 \times 1 = 10$，故结果为 $-10$。

**例 2**：利用分配律计算 $(-25) \times 4.3 + (-25) \times 5.7$。

**解**：$(-25) \times (4.3 + 5.7) = (-25) \times 10 = -250$。

**例 3**：若 $ab > 0$，$bc < 0$，判断 $a$、$b$、$c$ 的符号关系。

**解**：$ab > 0 \Rightarrow a, b$ 同号；$bc < 0 \Rightarrow b, c$ 异号。若 $a, b$ 同正，则 $c$ 为负；若 $a, b$ 同负，则 $c$ 为正。

### 易错点

1. **负负得正是核心**：$(-2) \times (-3) = 6$，初学者常写成 $-6$。
2. **奇偶负因数判断**：$(-1)^{2} = 1$，$(-1)^{3} = -1$，乘方时尤其注意。
3. **0 乘任何数为 0**：哪怕乘以负无穷大（极限概念），0 乘有限数永远是 0。

### 知识联系

- 前置：有理数加减、绝对值、正负数
- 后续：有理数除法（乘法逆运算）→ 有理数乘方 → 整式乘法
- 高中拓展：实数乘法；向量点积；矩阵乘法
