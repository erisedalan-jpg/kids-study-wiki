## 课本一句话定义

减去一个数等于加上这个数的**相反数**，即 $a - b = a + (-b)$；有理数减法可以转化为加法来计算。

---

## 📚 给 6-12 岁（小学高年级回溯版）

减法有一个魔法公式：**减一个数 = 加它的相反数**。

$5 - 3 = 5 + (-3) = 2$ ✅（结果和以前一样！）

但有了负数后，这个魔法特别有用：

- $3 - (-5)$：减去 $-5$ = 加上 $5$，所以 $3 - (-5) = 3 + 5 = 8$

想象在数轴上：
- 减正数 → 向左走（越来越小）
- 减负数 → 向右走（越来越大！）

"减去负数让你变大"——这个反直觉的结论，就是减法法则的核心。

## 🎓 给 12+（进阶版）

### 法则

$$a - b = a + (-b)$$

即：**减去一个数，等于加上这个数的相反数**。

这将减法完全归结为加法，统一了有理数加减运算。

### 推导逻辑

设 $a - b = c$，则 $c + b = a$，即 $c = a + (-b)$（因为 $(-b) + b = 0$）。

### 关键性质

| 性质 | 公式 |
|------|------|
| 减法转加法 | $a - b = a + (-b)$ |
| 减法不满足交换律 | $a - b \neq b - a$（一般情况）|
| 减法不满足结合律 | $(a - b) - c \neq a - (b - c)$（一般情况）|
| 与加法的关系 | $a - b + b = a$ |

### 典型例题

**例 1**：计算 $(-3) - (-8)$。

**解**：$(-3) - (-8) = (-3) + 8 = 5$。

**例 2**：计算 $5 - (+12) - (-8) + (-4)$。

**解**：全部化为加法：$5 + (-12) + 8 + (-4) = [5 + 8] + [(-12) + (-4)] = 13 + (-16) = -3$。

**例 3**：若 $a - b = 3$，$b = -2$，求 $a$。

**解**：$a = b + 3 = (-2) + 3 = 1$。

### 易错点

1. **减去负数变正**：$a - (-b) = a + b$，符号要"双重翻转"。
2. **不要乱用交换律**：$3 - 5 \neq 5 - 3$，减法无交换律。
3. **混合加减转化**：把所有减法化为加法后，再用加法法则计算，避免出错。

### 知识联系

- 前置：相反数、有理数加法
- 后续：有理数混合运算 → 移项（方程中"$-b$ 移到另边变 $+b$"）→ 整式加减
- 高中拓展：向量减法；行列式展开中的符号
