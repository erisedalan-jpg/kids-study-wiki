## 课本一句话定义

有理数加法遵循**同号相加取相同符号、绝对值相加；异号相加取绝对值较大的符号、绝对值相减**的法则；一个数与 0 相加仍得这个数。

---

## 📚 给 6-12 岁（小学高年级回溯版）

加法你已经很熟了！但现在加进了负数，规则稍有不同。

想象在数轴上走路：
- **正数** = 向右走
- **负数** = 向左走

$3 + (-5)$：先向右走 3 步，再向左走 5 步，最终在 $-2$。

两个规则：
1. **同号相加**（两个都正或都负）：绝对值加，符号不变
   - $3 + 4 = 7$（都向右，走得更远）
   - $(-3) + (-4) = -7$（都向左，走得更远）

2. **异号相加**（一正一负）：绝对值大的"赢"，绝对值相减
   - $(-3) + 5 = 2$（右走 5 步 > 左走 3 步，最终向右 2 步）
   - $3 + (-5) = -2$（左走 5 步 > 右走 3 步，最终向左 2 步）

## 🎓 给 12+（进阶版）

### 法则（人教版表述）

**规则 1（同号）**：同号两数相加，取相同的符号，并把绝对值相加。

$$a > 0, b > 0 \Rightarrow a + b = +(|a| + |b|)$$
$$a < 0, b < 0 \Rightarrow a + b = -(|a| + |b|)$$

**规则 2（异号）**：异号两数相加，取绝对值较大的数的符号，并用较大绝对值减去较小绝对值。

$$|a| > |b|,\ a > 0, b < 0 \Rightarrow a + b = +(|a| - |b|)$$

**规则 3（含 0）**：一个数与 0 相加，仍得这个数。

$$a + 0 = a$$

### 关键性质

| 性质 | 公式 |
|------|------|
| 交换律 | $a + b = b + a$ |
| 结合律 | $(a + b) + c = a + (b + c)$ |
| 加法单位元 | $a + 0 = a$ |
| 加法逆元 | $a + (-a) = 0$ |

### 典型例题

**例 1**：计算 $(-23) + (+48)$。

**解**：异号，$|{+48}| = 48 > |{-23}| = 23$，符号取"+"，$48 - 23 = 25$，故结果为 $+25$。

**例 2**：计算 $(-15) + (-27) + 15 + (-3)$。

**解**：利用交换律和结合律，先凑整：$(-15) + 15 = 0$，$(-27) + (-3) = -30$，故结果为 $0 + (-30) = -30$。

**例 3**：计算 $(-\dfrac{1}{2}) + \dfrac{3}{4} + (-\dfrac{5}{4}) + \dfrac{1}{2}$。

**解**：$(-\dfrac{1}{2}) + \dfrac{1}{2} = 0$；$\dfrac{3}{4} + (-\dfrac{5}{4}) = -\dfrac{2}{4} = -\dfrac{1}{2}$；故结果为 $-\dfrac{1}{2}$。

### 易错点

1. **符号判断**：异号相加，符号跟绝对值**大**的那个，不是跟第一个数。
2. **结果可能为 0**：互为相反数相加得 0，不要写成"无结果"。
3. **混合运算顺序**：同一优先级下从左到右，但用结合律先凑整更高效。

### 知识联系

- 前置：相反数、绝对值、正负数
- 后续：有理数减法（化为加法）→ 混合运算 → 整式合并同类项
- 高中拓展：向量加法；复数加法
