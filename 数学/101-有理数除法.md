## 课本一句话定义

除以一个不为零的数，等于乘以这个数的**倒数**；两数相除，同号得正，异号得负，绝对值相除；0 除以任何不为零的数都得 0，但 0 不能作除数。

---

## 📚 给 6-12 岁（小学高年级回溯版）

除法有一个魔法：**÷ 一个数 = × 它的倒数**。

$6 \div 2 = 6 \times \dfrac{1}{2} = 3$ ✅（和以前一样！）

引入负数后：
- **同号相除得正**：$(-6) \div (-2) = 3$
- **异号相除得负**：$(-6) \div 2 = -3$，$6 \div (-2) = -3$

口诀和乘法一样：**同号得正，异号得负**，只是把"乘"换成"除"。

**永远记住**：$0$ 不能当除数（除数为 0 没有意义）！

## 🎓 给 12+（进阶版）

### 法则

**法则 1（转化为乘法）**：

$$a \div b = a \times \frac{1}{b} \quad (b \neq 0)$$

**法则 2（符号规则）**：同号相除得正，异号相除得负，绝对值相除。

**法则 3（含 0）**：$0 \div b = 0 \quad (b \neq 0)$；$b \div 0$ 无意义。

### 倒数定义

若 $a \times b = 1$，则 $a$ 与 $b$ 互为**倒数**（reciprocal）：
- $\dfrac{p}{q}$ 的倒数为 $\dfrac{q}{p}$（$p, q$ 均不为 0）
- $0$ 没有倒数（因为不存在 $x$ 使 $0 \times x = 1$）

### 关键性质

| 性质 | 公式 |
|------|------|
| 除法转乘法 | $a \div b = a \times \frac{1}{b}$ |
| 符号规则 | 同号正，异号负 |
| 除法不满足交换律 | $a \div b \neq b \div a$（一般情况）|
| 除以 1 | $a \div 1 = a$ |
| 除以 -1 | $a \div (-1) = -a$ |

### 典型例题

**例 1**：计算 $(-36) \div (-\dfrac{3}{4})$。

**解**：$(-36) \div (-\dfrac{3}{4}) = (-36) \times (-\dfrac{4}{3}) = 36 \times \dfrac{4}{3} = 48$（同号得正）。

**例 2**：计算 $(-\dfrac{5}{8}) \div \dfrac{15}{4} \times (-6)$。

**解**：从左到右，先将除法转乘法：$(-\dfrac{5}{8}) \times \dfrac{4}{15} \times (-6) = \dfrac{5 \times 4 \times 6}{8 \times 15} = \dfrac{120}{120} = 1$（两负相乘，正）。

**例 3**：若 $\dfrac{a}{b} < 0$ 且 $a + b > 0$，判断 $a$、$b$ 的符号。

**解**：$\dfrac{a}{b} < 0 \Rightarrow a, b$ 异号；$a + b > 0 \Rightarrow$ 正数绝对值更大；故 $a > 0, b < 0$（正大负小）。

### 易错点

1. **除以负数 ≠ 结果变负**：$(-6) \div (-3) = 2$（正）。
2. **0 不能做除数**：$5 \div 0$ 无意义，不要和 $0 \div 5 = 0$ 混淆。
3. **连除时逐一转化**：$a \div b \div c = a \times \dfrac{1}{b} \times \dfrac{1}{c}$，不要错误结合。

### 知识联系

- 前置：有理数乘法、倒数（小学）
- 后续：有理数混合运算 → 整式运算 → 分式化简
- 高中拓展：实数除法；矩阵的逆（广义倒数）
