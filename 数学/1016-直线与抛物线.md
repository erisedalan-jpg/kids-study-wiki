# 直线与抛物线

> **一句话**：研究直线与抛物线公共点个数、位置关系判定及弦长、中点弦、切线等度量计算的解析几何考点。
> **English**: The study of the positional relationships between a line and a parabola—including intersection, tangency, disjointness—and the computation of associated geometric quantities.

## 🎓 考点精讲

- **定义/原理**  
  给定抛物线 $C: y^2 = 2px$（或 $x^2 = 2py$，$p \neq 0$）与直线 $l: y = kx + m$（或 $x = ty + n$）。联立方程消元后得关于 $x$（或 $y$）的一元二次方程：
  $$A x^2 + B x + C = 0 \quad (\text{或 } A' y^2 + B' y + C' = 0)$$
  判别式 $\Delta = B^2 - 4AC$（或对应形式）决定位置关系：
  - $\Delta > 0 \implies$ 相交（两个不同公共点）；
  - $\Delta = 0 \implies$ 相切（一个公共点，切线情形）；
  - $\Delta < 0 \implies$ 相离（无公共点）。  
  特别地，当二次项系数为零时，直线与抛物线对称轴平行，此时恰有一个公共点（相交非相切）。

- **关键方法**  
  1. **弦长公式**：若相交于 $A(x_1,y_1), B(x_2,y_2)$，弦长
     $$|AB| = \sqrt{1+k^2}\,|x_1 - x_2| = \sqrt{1+\frac{1}{k^2}}\,|y_1-y_2|$$
     其中 $|x_1-x_2| = \frac{\sqrt{\Delta}}{|A|}$（二次项系数 $A \neq 0$）。
  2. **中点弦（点差法）**：设弦端点坐标代入抛物线方程相减，利用斜率与中点坐标的关系直接求出弦所在直线斜率，避免联立。
  3. **切线问题**：已知切点用导数或替换法；过抛物线外一点求切线，可设切线方程联立抛物线令 $\Delta=0$ 解参数。

- **典型应用**  
  高考常以抛物线与直线为载体考查：  
  - 求弦长或由弦长求直线参数；  
  - 中点弦所在直线方程；  
  - 与焦点相关的弦（焦点弦）性质，如 $|AB| = x_1+x_2+p$（开口向右时）；  
  - 抛物线上点到直线的最短距离（切线法）；  
  - 存在性问题：是否存在直线满足某几何条件，需结合判别式与韦达定理。

## ⚠️ 高频易错点

- **忽略斜率不存在的情形**：设直线为 $y=kx+m$ 会漏掉竖直直线 $x = t$，需单独讨论。
- **二次项系数为零导致误判**：联立后二次项系数可能为 $0$，此时不是二次方程，不能用判别式判断相切，实际为相交（一个公共点）。
- **弦长公式记混坐标系**：注意直角坐标系下弦长公式用 $\sqrt{1+k^2}$ 或 $\sqrt{1+1/k^2}$，使用不当会出错。
- **点差法未检验判别式**：求出中点弦直线方程后，必须代回原方程检验 $\Delta>0$，确保直线与抛物线确实相交，否则可能产生增解。
