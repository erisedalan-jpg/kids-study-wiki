## 课本一句话定义

像 1、2、3 这样大于 0 的数叫做**正数**；像 -1、-2、-3 这样在正数前面加上负号"-"的数叫做**负数**；0 既不是正数也不是负数。

---

## 📚 给 6-12 岁（小学高年级回溯版）

你已经认识了 1、2、3……这些数字，它们都比 0 大，叫**正数**。

但是生活中有些情况需要用"比 0 还小"的数来表示：

- 冬天气温零下 5 摄氏度 → 写作 **-5°C**
- 电梯地下一层 → 写作 **-1 层**
- 欠别人 10 元 → 写作 **-10 元**

这些带"-"号的数叫**负数**。正数和负数是一对"方向相反"的数，0 是中间的分界线，它两边都不属于。

**小口诀**：大于零是正，小于零是负，0 站中间不偏心。

## 🎓 给 12+（进阶版）

### 严格定义

- **正数**：所有大于 0 的实数，记作 $a > 0$，包括正整数、正分数、正无理数。
- **负数**：所有小于 0 的实数，记作 $a < 0$，由正数前加"-"号得到。
- **0**：中性元素，$0$ 既非正数也非负数。

### 关键性质

| 性质 | 说明 |
|------|------|
| 正数 > 0 > 负数 | 在数轴上正数在右，负数在左 |
| $-(-a) = a$ | 负号的双重否定 |
| 正数 × 正数 = 正数 | 同号积为正 |
| 正数 × 负数 = 负数 | 异号积为负 |

### 典型例题

**例 1**：用正负数表示：海拔 3000 米、海拔 -200 米、平均气温零下 8 度。

**解**：海拔 3000 米 → $+3000$ 米（或直接写 $3000$ 米）；海拔 -200 米 → $-200$ 米；零下 8 度 → $-8°C$。

**例 2**：判断 $-\dfrac{3}{4}$、$0$、$+0.5$、$-100$ 各是正数还是负数？

**解**：$-\dfrac{3}{4}$ 是负数；$0$ 既不是正数也不是负数；$+0.5$ 是正数；$-100$ 是负数。

### 易错点

1. **0 的归属**：0 不是正数，也不是负数，是正负数的分界。
2. **正数前的"+"可省略**：$+3$ 和 $3$ 相同，但 $-3$ 的负号不可省。
3. **负数不一定"小"**：$-1 > -100$，绝对值大的负数反而更小。

### 知识联系

- 前置：自然数、整数概念（小学）
- 后续：有理数 → 数轴 → 绝对值 → 四则运算
- 高中拓展：实数系（加入无理数）、复数（引入虚数）
