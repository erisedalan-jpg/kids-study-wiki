## 课本一句话定义

只有符号不同的两个数互为**相反数**；$a$ 的相反数是 $-a$；0 的相反数是 0 本身；互为相反数的两个数之和为 0。

---

## 📚 给 6-12 岁（小学高年级回溯版）

想象一条绳子，你往右走 3 步，再往左走 3 步，正好回到出发点。

3 和 -3 就是一对**相反数**——大小一样（都是 3），方向相反（一个向右，一个向左）。

在数轴上，相反数关于**原点对称**：
- 3 在原点右边 3 格
- -3 在原点左边 3 格

**互为相反数的两个数相加等于 0**：$3 + (-3) = 0$。

特别记住：**0 的相反数是 0 本身**，因为 $0 + 0 = 0$。

---

## 🎓 给 12+（进阶版）

### 严格定义

若两个数 $a$ 和 $b$ 满足 $a + b = 0$，则称 $a$ 与 $b$ **互为相反数**，记作 $b = -a$（即 $a$ 的相反数）。

在实数（有理数）体系中，相反数即**加法逆元**（additive inverse）：

$$a + (-a) = 0 \quad \forall a \in \mathbb{R}$$

### 关键性质

| 性质 | 公式 |
|------|------|
| 双重否定 | $-(-a) = a$ |
| 相反数之和 | $a + (-a) = 0$ |
| 原点对称 | $a$ 与 $-a$ 在数轴上关于原点对称 |
| 绝对值相等 | $\|a\| = \|-a\|$ |
| 0 的相反数 | $-0 = 0$ |

### 典型例题

**例 1**：写出 $-7$、$\dfrac{3}{5}$、$-0.8$、$0$ 的相反数。

**解**：
- $-7$ 的相反数：$-(-7) = 7$
- $\dfrac{3}{5}$ 的相反数：$-\dfrac{3}{5}$
- $-0.8$ 的相反数：$0.8$
- $0$ 的相反数：$0$

**例 2**：若 $a$ 与 $b$ 互为相反数，且 $a = 2x - 1$，$b = x + 4$，求 $x$。

**解**：由互为相反数知 $a + b = 0$，即 $(2x - 1) + (x + 4) = 0$，$3x + 3 = 0$，$x = -1$。

### 易错点

1. **$-a$ 不一定是负数**：若 $a < 0$，则 $-a > 0$（负数的相反数是正数）。
2. **"互为"是对称关系**：若 $a$ 是 $b$ 的相反数，则 $b$ 也是 $a$ 的相反数。
3. **相反数 ≠ 倒数**：$3$ 的相反数是 $-3$，倒数是 $\dfrac{1}{3}$，完全不同。

### 知识联系

- 前置：正数与负数、数轴
- 后续：有理数加法（$a + (-a) = 0$ 的逆运算）→ 绝对值 → 方程解法中移项
- 高中拓展：向量的相反向量；加法群中逆元概念
