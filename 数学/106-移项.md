## 课本一句话定义

把方程一边的某项改变符号后移到另一边，叫做**移项**；移项的依据是等式性质 1（两边加减相同的量等式成立），移项时该项的符号**必须改变**。

---

## 📚 给 6-12 岁（小学高年级回溯版）

移项就像"搬家"——但搬过去要换身衣服（变号）：

$x + 3 = 7$

把 $+3$ 从左边搬到右边，变成 $-3$：

$x = 7 - 3 = 4$

口诀：**移项变号，不移不变**。

直觉理解：
$x + 3 = 7$
两边同减 3：$x + 3 - 3 = 7 - 3$
即：$x = 4$

"两边减 3" 的效果就等于"把 $+3$ 移到右边变成 $-3$"，这就是移项的本质。

## 🎓 给 12+（进阶版）

### 定义与依据

**移项**（transposition）：把方程 $A = B$ 中某一项从一边移到另一边，同时改变该项的符号。

**依据**：等式性质 1——$a = b \Rightarrow a + c = b + c$。

设方程为 $f(x) + p = g(x) + q$，对两边同时减去 $p$：
$$f(x) = g(x) + q - p$$

这等价于"将 $+p$ 从左边移到右边，变为 $-p$"。

### 移项规则

| 原式 | 移项后 | 说明 |
|------|------|------|
| $x + 5 = 10$ | $x = 10 - 5$ | $+5$ 移右变 $-5$ |
| $x - 3 = 7$ | $x = 7 + 3$ | $-3$ 移右变 $+3$ |
| $2x = x + 4$ | $2x - x = 4$ | $+x$ 移左变 $-x$ |
| $3 - x = 5$ | $-x = 5 - 3$ | $+3$ 移右变 $-3$ |

### 典型例题

**例 1**：解 $4x - 5 = 2x + 7$。

**解**：
移项（$-5$ 移右变 $+5$，$2x$ 移左变 $-2x$）：
$4x - 2x = 7 + 5$
合并：$2x = 12$
$x = 6$

**例 2**：解 $-3x + 2 = 5 - x$。

**解**：
移项：$-3x + x = 5 - 2$
合并：$-2x = 3$
$x = -\dfrac{3}{2}$

**例 3**：设 $m - 2n = 3$，写出 $m$ 的表达式。

**解**：移项：$m = 3 + 2n$（$-2n$ 移右变 $+2n$）。

### 常见移项对象

- **含未知数的项**：移到左边
- **常数项**：移到右边
- 这是将方程化为 $ax = b$ 标准形式的核心步骤。

### 易错点

1. **移项必须变号**：$x + 3 = 7$ 移 $3$ 后是 $x = 7 - 3$，不是 $x = 7 + 3$。
2. **不能直接移整个方程**：只能移一项，不能移一个括号（先去括号）。
3. **同时移多项**：可以同时移多项，但每项都要变号。

### 知识联系

- 前置：等式性质 1、有理数加减
- 后续：一元一次方程完整解法 → 不等式移项（同样法则）→ 消元法中移项
- 高中拓展：行列式变换、方程组变形
