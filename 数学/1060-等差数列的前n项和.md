# 等差数列的前n项和

> **一句话**：等差数列 $\{a_n\}$ 的前 $n$ 项和 $S_n = a_1 + a_2 + \dots + a_n$，可通过公式 $S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$ 直接计算，其中 $d$ 为公差。
> **English**: The sum of the first $n$ terms of an arithmetic sequence can be computed using $S_n = \frac{n(a_1 + a_n)}{2} = na_1 + \frac{n(n-1)}{2}d$, where $d$ is the common difference.

## 🎓 考点精讲

**定义/原理**  
等差数列 $\{a_n\}$ 满足 $a_{n} - a_{n-1} = d$（常数）。其前 $n$ 项和  
$$S_n = a_1 + a_2 + \cdots + a_n = \frac{n(a_1 + a_n)}{2}$$  
将通项 $a_n = a_1 + (n-1)d$ 代入，得  
$$S_n = na_1 + \frac{n(n-1)}{2}d$$  
上述公式推导的核心方法是**倒序相加**：由 $S_n = a_1 + a_2 + \cdots + a_n$，倒序写作 $S_n = a_n + a_{n-1} + \cdots + a_1$，两式相加后利用 $a_1+a_n=a_2+a_{n-1}=\cdots$ 得 $2S_n = n(a_1+a_n)$。

**关键方法**  
对于涉及 $a_1, d, n, a_n, S_n$ 的问题，采用**知三求二**：已知其中任意三个量，通过通项公式与求和公式联立可求出其余两个。常用步骤：  
1. 明确已知量和待求量；  
2. 根据条件选择 $S_n = \frac{n(a_1 + a_n)}{2}$ 或 $S_n = na_1 + \frac{n(n-1)}{2}d$；  
3. 解方程（组）得到参数，再转化为所需结果。  
若已知 $S_n$ 的结构式 $S_n = An^2 + Bn$，可直接判定该数列为等差数列，且公差 $d = 2A$，首项 $a_1 = S_1 = A+B$。

**典型应用**  
高考常以以下形式考查：  
- 直接求值：给出 $a_1, d, n$，求 $S_n$；或给出前几项和，反求项数 $n$ 或公差 $d$。  
- 最值问题：求等差数列前 $n$ 项和的最大值或最小值。通常转化为二次函数 $S_n = An^2 + Bn$ 在 $n\in\mathbb{N}^*$ 上的最值，借助对称轴 $n_0 = -\frac{B}{2A}$ 讨论 $n_0$ 附近的整数取值。  
- 综合判定：利用 $S_n$ 与 $a_n$ 的关系 $a_n = \begin{cases} S_1, & n=1 \\ S_n - S_{n-1}, & n\ge 2 \end{cases}$ 证明数列为等差数列，或由 $S_n$ 的表达式求通项。

## ⚠️ 高频易错点

- **混淆公式中 $n$ 的起始项**：求和公式中 $n$ 表示项数，当题目涉及片段和（如从第 $m$ 项到第 $n$ 项的和）时，需用 $S_n - S_{m-1}$，而不是直接代 $n-m+1$ 进行计算。
- **忽略 $n=1$ 的检验**：由 $S_n$ 求 $a_n$ 时，利用 $a_n = S_n - S_{n-1}$ 得到的式子对 $n\ge 2$ 成立，必须单独检验 $n=1$ 是否满足该式，否则可能出错。
- **公差符号导致的最值判断**：在求 $S_n$ 的最值时，容易不考虑 $d$ 的正负。若 $d>0$，$S_n$ 有最小值无最大值；若 $d<0$，$S_n$ 有最大值无最小值；若 $d=0$，$S_n = na_1$，无传统最值。
- **项数确定错误**：在应用题或含文字描述的题目中，容易数错项数。例如“从第3项到第10项”共 $10-3+1=8$ 项，误算为7项。
