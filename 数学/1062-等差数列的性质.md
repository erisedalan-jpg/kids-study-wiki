# 等差数列的性质

> **一句话**：等差数列是一类从第二项起每一项与前一项的差为同一个常数的数列，其核心性质体现为项与项的线性关系、等和性及项数对称下的等和规律。
> **English**: In an arithmetic sequence, the difference between any term and its preceding term is a constant, leading to linear expressions for terms and symmetric sum properties.

## 🎓 考点精讲

**定义与核心公式**
若数列 $\{a_n\}$ 满足 $a_{n+1} - a_n = d$（$d$ 为常数），则称 $\{a_n\}$ 为等差数列，$d$ 为公差。
- 通项公式：$a_n = a_1 + (n-1)d = a_m + (n-m)d$。
- 等差中项：若 $a, A, b$ 成等差数列，则 $A = \dfrac{a+b}{2}$。
- 前 $n$ 项和公式：$S_n = \dfrac{n(a_1 + a_n)}{2} = na_1 + \dfrac{n(n-1)}{2}d$。

**重要性质**
1. **等和性**：若 $m+n = p+q$（$m,n,p,q \in \mathbb{N}^*$），则 $a_m + a_n = a_p + a_q$，特别地，当 $m+n$ 为偶数时 $2a_{\frac{m+n}{2}} = a_m + a_n$。
2. **从函数视角看**：$a_n = dn + (a_1-d)$ 是 $n$ 的一次函数（$d \neq 0$）或常函数（$d=0$）；$S_n = \dfrac{d}{2}n^2 + (a_1-\dfrac{d}{2})n$ 是 $n$ 的二次函数缺常数项，$d>0$ 时 $S_n$ 有最小值，$d<0$ 时有最大值。
3. **子数列性质**：下标成等差的子数列仍为等差数列（如 $a_1,a_3,a_5,\dots$），公差为原公差的 $k$ 倍。

**关键方法与高考考查**
- 基本量法：用 $a_1$ 和 $d$ 表示其他量，列方程（组）求解。
- 性质巧用：涉及多下标和的问题优先考虑等和性，如 $a_3+a_7=a_4+a_6=a_2+a_8=2a_5$。
- 典型设问：求通项公式、求前 $n$ 项和的最值（利用 $S_n$ 的二次函数特征或转折项 $a_n$ 变号）、判断某数列是否为等差数列（常考查 $S_n$ 与 $a_n$ 的关系或新定义数列）、与方程或不等式结合（如 $a_1, a_2, a_5$ 成等比数列求公差）。

## ⚠️ 高频易错点

- **忽略公差 $d=0$ 的情况**：讨论等差数列单调性时必须单独考虑 $d=0$，此时为常数列，既是递增也是递减（非严格），且性质仍成立，但很多涉及二次函数最值、等比中项的问题会因此漏解。
- **等和性下标乱用**：等和性要求下标之和相等，不可误记为 $a_2+a_4=a_3+a_5$ 这样的错位相等。
- **混淆等差中项与等比中项**：等差数列的中项是算术平均，等比数列的中项是几何平均，解题时需看清数列类型，尤其在“成等比”“成等差”条件转换时不要混用。
- **前 $n$ 项和的最值问题机械套用二次函数顶点**：当公差不为整数或顶点对应 $n$ 不是正整数时，最值在顶点两侧最近的整数点取得，需结合 $a_n$ 的符号变化精确判断。
