# 等差数列的通项公式

> **一句话**：若一个数列从第二项起，每一项与前一项的差都等于同一个常数，则该数列为等差数列，其第 n 项可由首项和公差表示为 aₙ = a₁ + (n-1)d。
> **English**: The formula aₙ = a₁ + (n-1)d gives the n-th term of an arithmetic sequence, where a₁ is the first term and d is the common difference.

## 🎓 考点精讲

- **定义/原理**：数列 {aₙ} 满足 aₙ₊₁ - aₙ = d（常数），则称 {aₙ} 为等差数列，d 叫做公差。通项公式为 **aₙ = a₁ + (n-1)d**，它反映了项数与项值的线性关系。推广形式：aₙ = aₘ + (n-m)d，适用于任意已知项推求其它项。

- **关键方法**：（1）基本量法——通过已知条件列出关于 a₁ 和 d 的方程（组），解出 a₁ 与 d 后代入通项公式。（2）性质法——利用等差中项 2aₙ = aₙ₋₁ + aₙ₊₁ 或下标和性质 m+n = p+q ⇒ aₘ + aₙ = aₚ + aₔ 来整体代换，减少计算量。

- **典型应用**：高考常以两种方式考查——① 给出数列的某几项（或某些项的关系等式），直接求通项公式或指定项；② 将通项作为中间工具，结合前 n 项和、不等式或与函数性质交汇命题。例如：已知 a₃=7，a₇=15，求 a₂₀；或给出 Sₙ 的表达式，先利用 aₙ = Sₙ - Sₙ₋₁ (n≥2) 过渡到通项，再套用等差通项公式验证是否为等差数列。

## ⚠️ 高频易错点

- 混淆通项公式中 n 的起点：默认 n∈N* 时从 1 开始，若题目定义数列下标从 0 开始或给出的是 a₀，公式需调整为 aₙ = a₀ + nd。
- 公差符号判定失误：当数列递减时 d<0，计算或写出通项时常忽略负号，导致单调性判断和后续求最值出错。
- 误用推广形式：aₙ = aₘ + (n-m)d 中的项数差为 (n-m)，若误记为 (n+m) 或 (m-n) 符号反了，会导致等式错误。
- 由 Sₙ 求 aₙ 时忽略 n=1 的检验：在利用 aₙ = Sₙ - Sₙ₋₁ 得到通项后，必须验证 a₁=S₁ 是否满足该表达式，否则容易遗漏分段情况。
