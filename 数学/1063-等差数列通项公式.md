# 等差数列通项公式

> **一句话**：若数列 $\{a_n\}$ 为等差数列，首项为 $a_1$，公差为 $d$，则其第 $n$ 项 $a_n = a_1 + (n-1)d$。
> **English**: For an arithmetic sequence with first term $a_1$ and common difference $d$, the general term is given by $a_n = a_1 + (n-1)d$.

## 🎓 考点精讲

**定义/原理**：等差数列是指从第二项起，每一项与它的前一项的差等于同一个常数 $d$ 的数列。这一常数 $d$ 称为公差。通项公式 $a_n = a_1 + (n-1)d$ 揭示了等差 数列中任意一项与首项、公差及项数之间的线性关系。若已知任意一项 $a_k$ 和公差 $d$，通项也可写成 $a_n = a_k + (n-k)d$，这一推广形式在解题中更灵活。等差中项的概念是：若 $a,A,b$ 成等差数列，则 $A = \frac{a+b}{2}$。

**关键方法**：求解等差数列问题的核心是**首项 $a_1$ 和公差 $d$**。基本方法为“基本量法”：将条件转化为关于 $a_1$ 和 $d$ 的方程（组），解出两者后即可得到通项公式。另一种常见思路是直接利用通项公式的推广形式 $a_n = a_m + (n-m)d$ 求公差 $d$ 或某项，避免每次都回到首项。

**典型应用**：高考中该考点通常以小题或解答题第一问出现，考查形式包括：
1. **求通项**：给出数列中某几项的关系（如 $a_3+a_5=14$，$S_4=16$），求 $a_n$，即建立关于 $a_1,d$ 的方程组。
2. **求指定项**：已知通项中的若干项，求公差 $d$ 或某一项的值，如“已知 $a_2=3$，$a_6=11$，求 $a_{20}$”。
3. **判定与证明**：给定一个递推关系或含 $n$ 的表达式，要求证明某数列为等差数列，此时常用定义 $a_{n+1} - a_n = \text{常数}$ 进行判定。

## ⚠️ 高频易错点

- **混淆项数与下标**：在公式 $a_n = a_1 + (n-1)d$ 中误将 $(n-1)$ 写成 $n$，导致第一项 $a_1$ 被错误地代入计算。
- **忽略基本量法中的整体代换**：有时无需分别求出 $a_1$ 和 $d$，可直接利用 $a_n = a_m + (n-m)d$ 整体求出所需项，但很多学生仍习惯每次都先求 $a_1$，增加计算量且易出错。
- **判定等差数列时只验证有限项**：必须证明对任意 $n\in\mathbb{N}^*$，$a_{n+1} - a_n$ 为常数，仅验证前几项相等不能作为充分条件。
- **公差符号与单调性混淆**：误以为公差 $d>0$ 时数列一定递增，但当数列为各项递减时 $d<0$，需要注意等差数列的单调性仅由公差 $d$ 的符号决定。
