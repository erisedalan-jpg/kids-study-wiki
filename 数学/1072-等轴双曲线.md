# 等轴双曲线

> **一句话**：实轴和虚轴长度相等的双曲线称为等轴双曲线，其标准方程可写作 `x² - y² = a²` 或 `y² - x² = a²`。
> **English**: A rectangular hyperbola is a hyperbola whose transverse and conjugate axes are equal in length, with its standard equation given by `x² - y² = a²` or `y² - x² = a²`.

## 🎓 考点精讲

- **定义/原理**  
  在双曲线中，若半实轴长 `a` 等于半虚轴长 `b`，即 `a = b`，则该双曲线为**等轴双曲线**。  
  由 `b = a` 可得其离心率固定为  
  `e = c/a = √(a² + a²)/a = √2`。  
  渐近线方程为 `y = ±x`（当标准方程为 `x² - y² = a²` 时），两条渐近线互相垂直，因此等轴双曲线也是直角双曲线。  
  标准方程的两种形式：  
  - 焦点在 `x` 轴上：`x² - y² = a²`，焦点坐标 `(±√2 a, 0)`；  
  - 焦点在 `y` 轴上：`y² - x² = a²`，焦点坐标 `(0, ±√2 a)`。

- **关键方法**  
  - **判定方法**：双曲线为标准方程 `x²/m - y²/n = 1`（`m>0,n>0`）时，若 `m = n` 则为等轴双曲线；若为非标准位置的等轴双曲线，可通过 `a = b` 或渐近线互相垂直来判断。  
  - **方程求法**：已知双曲线为等轴双曲线时，可设方程为 `x² - y² = λ`（`λ ≠ 0`），再根据其他条件（如过定点）求 `λ`。  
  - **求离心率与渐近线**：直接应用 `e = √2`，渐近线为 `y = ±x`（当对称轴与坐标轴重合时），若曲线旋转需谨慎处理。

- **典型应用**  
  高考常以两种形式考查：  
  1. **直接判断与计算**：给出双曲线方程，判断其是否为等轴双曲线，并直接写出离心率、渐近线方程。设问如：“已知双曲线 `x²/4 - y²/4 = 1`，求其离心率和渐近线夹角。”（答案：`e=√2`，夹角 `90°`）。  
  2. **求等轴双曲线方程**：已知双曲线是等轴双曲线且经过某点 `(2,1)`，求标准方程。可设 `x² - y² = λ`，代入点得 `λ=3`，方程为 `x²/3 - y²/3 = 1`。

## ⚠️ 高频易错点

- 误认为等轴双曲线的离心率可变，忘记 `a=b` 必然导出 `e=√2`，在任何标准方程形式下都不变。  
- 将等轴双曲线的渐近线与一般双曲线的渐近线混淆：只要对称轴是坐标轴，渐近线必为 `y=±x`；若题目给出双曲线经过旋转（如反比例函数 `xy=k` 也是等轴双曲线），则渐近线为坐标轴，学生容易忽略。  
- 方程 `x² - y² = a²` 与 `y² - x² = a²` 的焦点位置混淆，焦点总是在实轴所在坐标轴上，需要先确定哪个变量的系数为正。  
- 与椭圆中的“等轴”混淆：椭圆没有等轴概念，不可类比。
