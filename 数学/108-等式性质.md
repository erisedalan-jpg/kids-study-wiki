## 课本一句话定义

**等式性质 1**：等式两边都加（或减）同一个数或式子，结果仍相等；**等式性质 2**：等式两边都乘（或除以）同一个不为零的数，结果仍相等。

---

## 📚 给 6-12 岁（小学高年级回溯版）

想象一杆天平：两边现在是平衡的（等式成立）。

**性质 1（加减）**：两边同时加或减同一个量，天平仍然平衡。

如果 $a = b$，那么 $a + 3 = b + 3$（两边都加 3，还是平衡）。

**性质 2（乘除）**：两边同时乘或除以同一个**不为零**的数，天平仍然平衡。

如果 $a = b$，那么 $2a = 2b$（两边都乘 2，还是平衡）。

**为什么不能除以 0？** 因为除以 0 没有意义（0 当除数无意义）。

这两条性质是解方程"合法操作"的依据——每一步都要对两边做同样的事。

## 🎓 给 12+（进阶版）

### 两条等式性质

**性质 1（加法性质）**：

$$a = b \Rightarrow a + c = b + c \quad (c \text{ 为任意数或代数式})$$

推论：$a = b \Rightarrow a - c = b - c$（减相同量）。

**性质 2（乘法性质）**：

$$a = b \Rightarrow ac = bc \quad (c \neq 0)$$

推论：$a = b \Rightarrow \dfrac{a}{c} = \dfrac{b}{c}$（除以同一非零数）。

### 应用：变形等式的标准步骤

| 操作 | 依据 | 目的 |
|------|------|------|
| 两边同加/减某项 | 等式性质 1 | 移项 |
| 两边同乘某系数 | 等式性质 2 | 去分母 |
| 两边同除以系数 | 等式性质 2 | 系数化 1 |

### 典型例题

**例 1**：用等式性质解 $x - 7 = 3$。

**解**：两边同加 7（等式性质 1）：$x - 7 + 7 = 3 + 7$，$x = 10$。

**例 2**：用等式性质解 $\dfrac{x}{3} = 4$。

**解**：两边同乘 3（等式性质 2）：$\dfrac{x}{3} \times 3 = 4 \times 3$，$x = 12$。

**例 3**：用等式性质解 $-2x = 8$。

**解**：两边同除以 $-2$（等式性质 2，$-2 \neq 0$）：$x = \dfrac{8}{-2} = -4$。

### 等式 vs 不等式

等式的两条性质在**不等式**中有对应版本，但有重要区别：
- 不等式两边乘**负数**时，**不等号方向改变**（如 $3 > 2$，$\times (-1)$ 得 $-3 < -2$）。

### 易错点

1. **性质 2 中 $c \neq 0$**：除以 0 无意义，乘以 0 会使等式退化。
2. **两边都要操作**：只对一边加/减/乘/除，等式被破坏。
3. **负数除法方向**：等式两边除以负数，等号方向不变（等号无方向），但以后学不等式时方向要改变。

### 知识联系

- 前置：方程的概念、有理数除法
- 后续：解一元一次方程（移项本质是等式性质 1）→ 分式方程（去分母用性质 2）
- 高中拓展：等价变形（同解变换）、不等式性质（乘负数变号）
