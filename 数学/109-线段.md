## 课本一句话定义

**线段**是直线上两点之间的部分，有两个**端点**，具有确定的**长度**；连接两点的线段的长度叫做这两点之间的距离，两点之间线段最短。

---

## 📚 给 6-12 岁（小学高年级回溯版）

想象用铅笔连接两个点 $A$ 和 $B$，画出来的那段叫**线段**。

线段的特点：
- 有两个**端点**（$A$ 和 $B$）
- 有**有限长度**（可以测量）
- 是连接两点的最短路径（"两点之间，线段最短"）

线段记作 $AB$ 或 $BA$（两端点的字母组合）。

**对比三种"线"**：

| 名称 | 端点 | 长度 | 示意 |
|------|------|------|------|
| 线段 | 两个端点 | 有限 | ——A————B—— |
| 射线 | 一个端点 | 无限（单向）| —O————→ |
| 直线 | 没有端点 | 无限（双向）| ←——————→ |

## 🎓 给 12+（进阶版）

### 定义

**线段**（line segment）$AB$：直线上点 $A$ 和点 $B$ 之间的部分，包含端点 $A$、$B$ 及它们之间所有的点。

**长度**：线段 $AB$ 的长度记作 $|AB|$ 或 $AB$（上下文明确时）。

**两点间距离**：连接两点的线段的长度，即点 $A$ 与点 $B$ 之间的最短距离。

### 线段的中点

若 $M$ 是线段 $AB$ 的**中点**（midpoint），则：
$$AM = MB = \frac{AB}{2}$$

中点将线段等分为两段。

### 关键性质

| 性质 | 内容 |
|------|------|
| 最短性 | 两点之间，线段最短 |
| 唯一性 | 两点确定唯一一条线段 |
| 加法 | $M$ 在 $AB$ 上：$AM + MB = AB$ |
| 中点 | $M$ 是中点：$AM = MB = \frac{AB}{2}$ |

### 数轴上两点距离

数轴上两点 $A(a)$ 和 $B(b)$ 之间的距离：

$$AB = |a - b| = |b - a|$$

（等于坐标差的绝对值）

### 典型例题

**例 1**：已知线段 $AB = 10$ cm，$M$ 是 $AB$ 的中点，$N$ 是 $MB$ 的中点，求 $MN$。

**解**：$AM = MB = 5$ cm；$MN = \dfrac{MB}{2} = 2.5$ cm。

**例 2**：数轴上 $A$ 在 $-3$ 处，$B$ 在 $5$ 处，求 $AB$ 的长度和中点坐标。

**解**：$AB = |(-3) - 5| = 8$；中点坐标 $= \dfrac{-3 + 5}{2} = 1$。

**例 3**：若 $C$ 在线段 $AB$ 上，$AC = 4$，$CB = 6$，求 $AB$。

**解**：$AB = AC + CB = 4 + 6 = 10$。

### 易错点

1. **线段有两个端点，射线只有一个**：写射线时要指明从哪个端点出发。
2. **$AB$ 和 $BA$ 是同一条线段**：线段没有方向，$|AB| = |BA|$。
3. **中点条件**：$M$ 是中点需要 $AM = MB$（两段相等），只有 $M$ 在 $AB$ 上还不够。

### 知识联系

- 前置：点线面体、有理数（长度计算）
- 后续：射线 → 直线 → 角的定义 → 平行与垂直
- 高中拓展：向量（线段的方向扩展）→ 解析几何中两点距离公式 $d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$
