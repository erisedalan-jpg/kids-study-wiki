# 绝对值不等式

> **一句话**：绝对值不等式是形如 |f(x)| < g(x) 或 |f(x)| > g(x) 的不等式，根据绝对值的定义将其转化为等价的不含绝对值的不等式组来求解。
> **English**: An absolute value inequality is an inequality containing an absolute value expression, solved by transforming it into equivalent compound inequalities based on the definition of absolute value.

## 🎓 考点精讲

- **定义/原理**：  
  绝对值的基本定义：|a| = a (a ≥ 0)，|a| = -a (a < 0)。由此，不等式 |x| < a (a > 0) 等价于 -a < x < a；|x| > a (a > 0) 等价于 x > a 或 x < -a。  
  一般情形 |f(x)| < g(x) 等价于 -g(x) < f(x) < g(x)，且隐含 g(x) > 0；|f(x)| > g(x) 等价于 f(x) > g(x) 或 f(x) < -g(x)。当 g(x) ≤ 0 时，|f(x)| > g(x) 恒成立（因为绝对值非负）。  
  绝对值三角不等式：||a| - |b|| ≤ |a ± b| ≤ |a| + |b|，常用于求最值或处理含参恒成立问题。

- **关键方法**：  
  - **零点分段法**：求每个绝对值内部表达式的零点，用零点将数轴分段，逐段去绝对值化为普通不等式求解，最后取并集。  
  - **平方法**：当两边均非负时，可将不等式平方去绝对值，如 |f(x)| < |g(x)| 等价于 f(x)² < g(x)²，适用于次数不高的情况。  
  - **几何法**：利用绝对值的几何意义（数轴上的距离），如 |x - a| + |x - b| 表示数轴上点到 a、b 两点的距离之和，可求最小值、解不等式。

- **典型应用**：  
  高考中常见设问有：① 解绝对值不等式，如 |2x - 1| < 3；② 含参绝对值不等式恒成立/有解问题，如 |x - a| + |x + 1| ≥ 3 恒成立，求 a 的范围；③ 结合分段函数或函数零点，求不等式的解集或参数范围。例如，解不等式 |x - 2| + |x + 3| ≥ 7，需按零点 -3, 2 分段转化为三个不等式组，分别求解后合并。

## ⚠️ 高频易错点

- 解 |f(x)| < g(x) 时忘记隐含条件 g(x) > 0，直接写 -g(x) < f(x) < g(x)，当 g(x) ≤ 0 时解集实则为空，失掉对 g(x) 的讨论导致增解。
- 解 |f(x)| > g(x) 时未先判断 g(x) 的符号：若 g(x) < 0，不等式恒成立（因绝对值非负），需单独说明，直接按“f(x) > g(x) 或 f(x) < -g(x)”会漏掉这一情况。
- 零点分段法多个绝对值交叉时，分段点处等号归属不清，或区间端点的开闭处理不当，导致最终解集边界错误。
- 用三角不等式 ||a| - |b|| ≤ |a ± b| ≤ |a| + |b| 求参数范围时，忽略等号成立的条件（a 与 b 同号或至少一个为零等），造成参数区间漏端点或扩大。
