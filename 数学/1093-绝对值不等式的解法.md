# 绝对值不等式的解法

> **一句话**：解形如 |ax+b|≤c 或 |ax+b|≥c 的不等式，利用绝对值的定义或几何意义，将其转化为不含绝对值的不等式组（或不等式）求解。
> **English**: Solving absolute value inequalities means transforming expressions like |f(x)| ≤ c or |f(x)| ≥ c into equivalent compound inequalities using the definition of absolute value.

## 🎓 考点精讲

**定义与原理**
绝对值不等式的基本等价关系（设 c>0）：
- |f(x)| ≤ c 等价于 -c ≤ f(x) ≤ c ；
- |f(x)| ≥ c 等价于 f(x) ≤ -c 或 f(x) ≥ c 。
更一般地，对于 |f(x)| ≤ g(x)，等价于 -g(x) ≤ f(x) ≤ g(x) ，且隐含条件 g(x)≥0 （若 g(x)<0，则不等式无解）；对于 |f(x)| ≥ g(x)，等价于 f(x) ≤ -g(x) 或 f(x) ≥ g(x) ，此处 g(x) 可负。若不等式两边均为绝对值，如 |f(x)|≤|g(x)|，因两边非负，可通过平方去掉绝对值，等价于 [f(x)]² ≤ [g(x)]²。

**关键方法**
1. **基本型** |ax+b|≤c 直接转化为 -c ≤ ax+b ≤c ；|ax+b|≥c 转化为 ax+b ≤ -c 或 ax+b ≥c。
2. **变量在右型** |f(x)|≤g(x) 转化为不等式组 -g(x)≤f(x)≤g(x) 并保证 g(x)≥0；|f(x)|≥g(x) 转化为 f(x)≤-g(x) 或 f(x)≥g(x) 。
3. **双边绝对值型** |f(x)|≤|g(x)| 利用平方 (f(x))² ≤ (g(x))² 转化为整式不等式。
4. **含多个绝对值的不等式** 采用零点分段法：令每个绝对值内的式子为零，标出零点，分区间去掉绝对值符号，转化为一般不等式求解，最后取并集。

**典型应用**
高考中常在选填题出现解不等式，或结合集合运算求交集、补集；解答题中（尤其旧高考选做题“不等式选讲”）常考查含参绝对值不等式的解集与参数范围。例如：
- 解不等式 |2x-1| < 3 ，得 -3 < 2x-1 < 3，解得 -1 < x < 2。
- 若不等式 |x-a|+|x+2| ≥ 5 的解集为 R，求 a 的取值范围，需利用绝对值三角不等式或分段讨论。

## ⚠️ 高频易错点

- 解 |f(x)| ≥ c 时，易漏掉“或”的逻辑，将解集错误地取交集或用“且”连接。
- 处理 |f(x)| ≤ g(x) 时，忘记要求 g(x) ≥ 0，直接转化为 -g(x)≤f(x)≤g(x) 而不检验，导致解集中包含使 g(x)<0 的 x。
- 两边平方去绝对值时，未考虑等价性：|f(x)|≤g(x) 不能直接平方，因为 g(x) 可能为负；只有 |f(x)|≤|g(x)| 等两侧都非负才可随意平方。
- 零点分段时，区间端点处的等号归属混淆，导致解集边界缺失或重复。
