# 统计预测

> **一句话**：统计预测是基于样本数据建立的回归模型，对未知的响应变量取值进行点估计或区间估计。
> **English**: Statistical prediction is the use of a regression model built from sample data to make point or interval estimates for unknown values of the response variable.

## 🎓 考点精讲

**定义/原理**
统计预测的核心是**回归分析**。若已知解释变量 $x$ 与响应变量 $y$ 满足近似线性关系，则可通过样本数据 $(x_i,y_i)(i=1,2,...,n)$ 求出经验回归方程：
$$\hat{y} = \hat{b}x + \hat{a}$$
其中回归系数由最小二乘法确定：
$$\hat{b} = \frac{\sum_{i=1}^{n}(x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^{n}(x_i - \bar{x})^2}, \quad \hat{a} = \bar{y} - \hat{b}\bar{x}$$

若非线性相关（如指数增长），常通过变换转化为线性回归。例如 $y = ce^{dx}$ 可两边取对数得 $\ln y = \ln c + dx$，对 $(\ln y, x)$ 做线性回归。

预测分为两类：**点预测**（将 $x_0$ 代入回归方程得 $\hat{y}_0$）和**区间预测**（在给定置信水平下给出 $\hat{y}_0$ 的取值范围）。高考侧重考查点预测及利用残差分析评价模型。

**关键方法**
1. **画散点图**：观察点的分布趋势，判断线性或非线性关系。
2. **求回归方程**（线性情形）：
   - 计算 $\bar{x}, \bar{y}$
   - 计算 $\hat{b}$（分子分母可借助公式 $\sum x_i y_i - n\bar{x}\bar{y}$ 和 $\sum x_i^2 - n\bar{x}^2$ 简化运算）
   - 计算 $\hat{a}$，写出方程
3. **预测与决策**：代入 $x_0$ 得 $\hat{y}_0$；比较预测值与实际值的差异（残差分析）；依据预测结果做合理推断（如产能预估、费用控制）。

**典型应用**
- **根据广告投入预测销售额**：给出若干组（广告费，销售额）数据，求线性回归方程，并预测当广告费为某值时销售额。
- **非线性回归转换**：如细菌繁殖数量 $y$ 与时间 $t$ 呈指数关系，给出数据表，要求通过取对数建立线性回归模型并预测某时刻数量。

## ⚠️ 高频易错点

- **混淆回归系数公式中的分子分母**：$\hat{b}$ 分子为 $\sum(x_i-\bar{x})(y_i-\bar{y})$，分母为 $\sum(x_i-\bar{x})^2$，不可颠倒。尤其当数据以表格形式给出时，切忌误将 $y$ 的平方和放入分母。
- **将非线性关系直接做线性回归**：必须先观察散点图或计算相关系数 $r$，若 $|r|$ 接近 0 说明线性相关性弱，需考虑转换模型（指数型或幂函数型），否则预测结果失真。
- **外推范围过大**：回归方程的预测仅适用于样本数据所在的自变量范围或附近，不可随意向远处延伸。若 $x_0$ 远超样本极值，预测值可靠性极低。
- **混淆相关关系与因果关系**：回归分析揭示统计关联，不代表 $x$ 的变化一定“导致” $y$ 的变化。高考应用题有时设问“能否认为……的因果关系”，需要明确区分。
