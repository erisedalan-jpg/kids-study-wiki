# 补集

> **一句话**：给定全集 $U$，集合 $A \subseteq U$，由 $U$ 中所有不属于 $A$ 的元素构成的集合叫做 $A$ 在 $U$ 中的补集，记作 $\complement_U A$。
> **English**: The complement of a set $A$ relative to a universal set $U$ is the set of all elements in $U$ that are not in $A$, denoted $\complement_U A$.

## 🎓 考点精讲

**定义与符号**  
设 $U$ 为全集，$A \subseteq U$，则 $A$ 的补集 $\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$。在文氏图中常用矩形表示全集，圆圈表示 $A$，圆圈外部、矩形内部即为补集。基本性质：  
- $A \cup (\complement_U A) = U$  
- $A \cap (\complement_U A) = \varnothing$  
- $\complement_U (\complement_U A) = A$  
- $\complement_U U = \varnothing,\ \complement_U \varnothing = U$  
- 德摩根律：$\complement_U (A \cap B) = (\complement_U A) \cup (\complement_U B)$，$\complement_U (A \cup B) = (\complement_U A) \cap (\complement_U B)$

**关键方法**  
1. **明确全集**：一切补集运算的前提是确定全集 $U$。常见全集有实数集 $\mathbf{R}$、整数集 $\mathbf{Z}$，或题目限定的特定范围。  
2. **集合运算转化**：遇到“非”“补”等描述，利用补集性质与德摩根律将复杂表达式化简为标准交集/并集形式。  
3. **数轴与区间表示**：对连续数集，用数轴标出全集与子集，补集即为全集挖去子集后的区间。

**高考典型考查方式**  
- **求补集**：已知全集和子集，写出补集（如 $U=\{1,2,3,4,5\},\ A=\{1,3\}$，则 $\complement_U A=\{2,4,5\}$）。  
- **补集参与的交并运算**：求 $\complement_U (A \cap B)$ 或 $\complement_U A \cup B$ 等，常结合不等式解集。  
- **含参数的补集问题**：已知补集满足某些条件，反推参数取值范围。  
- **应用题中说明“不满足某条件的对象”**，本质是补集思想。

## ⚠️ 高频易错点

- 忽视全集而直接答题：如未指明全集 $U$ 就写 $\complement A$，或默认为实数集而出错。
- 混淆补集与差集：补集是差集 $U \setminus A$，但差集不要求 $A\subseteq U$；若全集不明确，补集计算无意义。
- 端点取舍错误：在不等式确定的连续数集中，补集区间开闭常错，需严格按“不属于 $A$”判断端点。
- 德摩根律应用符号混乱：误将 $\complement_U (A \cap B)$ 写成 $(\complement_U A) \cap (\complement_U B)$，应改为并集。
