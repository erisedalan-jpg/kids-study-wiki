# 角度范围分析

> **一句话**：在三角、解三角形、向量等问题中，由已知条件确定角的边界或取值集合，以保证解的准确性与合理性。
> **English**: Determining the exact interval or set of values for an angle based on given constraints, to ensure valid and complete solutions.

## 🎓 考点精讲

- **定义/原理**：角度范围分析是指根据三角函数的定义域、值域、单调性以及几何条件（如三角形内角和为 $\pi$，边长关系、向量夹角范围 $[0,\pi]$，倾斜角范围 $[0,\pi)$ 等）严格推出角的允许范围。核心原理包括：正弦、余弦的有界性（$|\sin\theta|\le 1,|\cos\theta|\le 1$）；同角三角函数的平方关系（$\sin^2\theta+\cos^2\theta=1$）；三角形中 $A+B+C=\pi$，且每个内角满足 $0<A,B,C<\pi$，大边对大角等。
- **关键方法**：
  1. **挖掘隐含条件**：列出题目中所有对角的直接或间接约束（如等式、不等式、几何位置）。
  2. **转化为角的方程或不等式**：利用三角恒等变换、正弦定理 $\dfrac{a}{\sin A}=\dfrac{b}{\sin B}=2R$、余弦定理 $\cos A=\dfrac{b^2+c^2-a^2}{2bc}$ 等，将条件化为角的关系。
  3. **利用函数性质锁定范围**：结合三角函数在特定区间上的单调性、符号、以及图像特征（如 $y=\sin x$ 在 $[0,\pi]$ 上非负），解不等式或方程的角集。
  4. **检验取舍**：将初步所得角代入原条件验证，排除增根或不满足隐含范围的角。
- **典型应用**：
  - 解三角形中已知两边及其中一边的对角（SSA），需依据边角关系判定角的可能取值（一解、两解或无解）。如已知 $a,b,\angle A$，由 $\sin B=\dfrac{b\sin A}{a}$，需结合 $a$ 与 $b\sin A$ 的大小以及 $\angle A$ 的锐钝分析 $\angle B$ 的范围。
  - 求含参三角函数的定义域或值域时，需先确定角的范围。例如 $y=\sin(2x-\frac{\pi}{3})$ 在 $x\in[0,\frac{\pi}{2}]$ 上的最值，必须先确定 $2x-\frac{\pi}{3}\in[-\frac{\pi}{3},\frac{2\pi}{3}]$ 再利用正弦图像求解。
  - 向量夹角问题：已知 $\vec{a},\vec{b}$ 的坐标，求夹角 $\theta$ 的范围常需利用 $\cos\theta=\dfrac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}$，并由 $\theta\in[0,\pi]$ 及余弦值范围反推角。

## ⚠️ 高频易错点

- **忽视三角形内角的隐含范围**：仅由 $\sin A=\frac{1}{2}$ 得 $A=30^\circ$ 或 $150^\circ$，但未验证若 $A=150^\circ$ 是否会导致三角形内角和超过 $180^\circ$ 或边长关系矛盾。
- **SSA 判别不清**：由 $\sin B$ 求 $B$ 时，漏掉 $B=180^\circ-\arcsin(\dots)$ 的可能性，或未通过 $a$ 与 $b\sin A$ 的大小关系判断解的个数。
- **混淆角的不同度量范围**：例如直线倾斜角范围是 $[0,\pi)$，向量夹角是 $[0,\pi]$，异面直线夹角是 $(0,\frac{\pi}{2}]$，未加区分直接套用导致范围错误。
- **忽视函数的单调区间影响**：用反三角函数表示角时，没有根据角的真实区间选择正确分支，例如在 $[\frac{\pi}{2},\frac{3\pi}{2}]$ 上 $\sin$ 递减，直接用 $\arcsin$ 取值会导致符号错误。
