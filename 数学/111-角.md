## 课本一句话定义

**角**是由两条有**公共端点**的射线所组成的图形；公共端点叫做角的**顶点**，两条射线叫做角的**边**；角也可以看作是一条射线绕端点旋转到另一位置所形成的图形。

---

## 📚 给 6-12 岁（小学高年级回溯版）

你在小学已经认识了角：三角形有 3 个角，正方形有 4 个直角。

**角的组成**：
- 两条射线（边）
- 一个公共端点（顶点）

就像张开的剪刀：两个刀片是"边"，铆钉是"顶点"。

**角的符号**：$\angle AOB$ 或 $\angle O$（$O$ 是顶点），也可以用数字 $\angle 1$、$\angle 2$ 命名。

**特殊角复习**：
- **直角**：90°（像正方形的角）
- **平角**：180°（像一条直线）
- **锐角**：0° ~ 90° 之间
- **钝角**：90° ~ 180° 之间

## 🎓 给 12+（进阶版）

### 定义

**角**（angle）有两种等价定义：
1. **静态定义**：由两条有公共端点的射线组成的图形（$\overrightarrow{OA}$ 和 $\overrightarrow{OB}$ 共顶点 $O$）
2. **动态定义**：一条射线绕其端点旋转到另一位置形成的图形（旋转角）

记号：$\angle AOB$（顶点在中间）；或 $\angle O$（仅一个角时）；或 $\angle 1$（编号）。

### 角的分类

| 角的类型 | 度数范围 | 特征 |
|--------|---------|------|
| 零角 | $0°$ | 两边重合（初始位置）|
| 锐角（acute） | $0° < \alpha < 90°$ | 小于直角 |
| 直角（right） | $\alpha = 90°$ | 垂直的两边 |
| 钝角（obtuse） | $90° < \alpha < 180°$ | 介于直角和平角之间 |
| 平角（straight） | $\alpha = 180°$ | 两边成一条直线 |
| 优角/反角（reflex） | $180° < \alpha < 360°$ | 大于平角（高中涉及）|
| 周角（full） | $\alpha = 360°$ | 旋转一圈 |

### 关键关系

**互补**（supplementary）：$\angle A + \angle B = 180°$（和为平角）

**互余**（complementary）：$\angle A + \angle B = 90°$（和为直角）

**对顶角**（vertical angles）：两直线相交形成的对面角，对顶角相等。

**相邻补角**：两角共顶点共一边，另两边成一直线，则互补。

### 典型例题

**例 1**：一个角的补角是它的 3 倍，求这个角。

**解**：设角为 $\alpha$，补角为 $180° - \alpha = 3\alpha$，$4\alpha = 180°$，$\alpha = 45°$。

**例 2**：两直线相交，$\angle 1 = 70°$，求其他三个角。

**解**：$\angle 1 = 70°$；$\angle 2 = 180° - 70° = 110°$（邻补角）；$\angle 3 = 70°$（对顶角 = $\angle 1$）；$\angle 4 = 110°$（对顶角 = $\angle 2$）。

**例 3**：一个角的余角比这个角的 $\dfrac{1}{2}$ 少 $15°$，求这个角。

**解**：设角为 $\alpha$，余角 $90° - \alpha = \dfrac{\alpha}{2} - 15°$，$90 + 15 = \dfrac{\alpha}{2} + \alpha = \dfrac{3\alpha}{2}$，$\alpha = \dfrac{105 \times 2}{3} = 70°$。

### 易错点

1. **$\angle AOB$ 中顶点写中间**：$\angle AOB$ 正确，$\angle OAB$（A 是顶点时才这样写）。
2. **互余互补的和**：余角和为 90°，补角和为 180°，别记混。
3. **对顶角 ≠ 邻角**：对顶角不共边，相等；邻补角共边，互补。

### 知识联系

- 前置：射线（角的边）、点线面体
- 后续：角的度量 → 平行线与角（同位角、内错角、同旁内角）→ 三角形内角和
- 高中拓展：三角函数（角的量化）→ 弧度制（新的角度单位）
