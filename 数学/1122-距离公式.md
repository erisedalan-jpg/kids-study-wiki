# 距离公式

> **一句话**：用于计算空间中两点、点与直线、或平行直线之间几何距离的代数表达式。
> **English**: An algebraic expression for computing the geometric distance between two points, a point and a line, or two parallel lines in space.

## 🎓 考点精讲

**定义与原理**

1.  **两点间距离公式**（二维）：点 $P_1(x_1, y_1)$ 与 $P_2(x_2, y_2)$ 间的距离
    $$d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2}$$
    三维空间中点 $P_1(x_1, y_1, z_1)$ 与 $P_2(x_2, y_2, z_2)$ 间距离则为 $d = \sqrt{(x_2 - x_1)^2 + (y_2 - y_1)^2 + (z_2 - z_1)^2}$。其本质是勾股定理在坐标系中的直接推广。

2.  **点到直线的距离公式**（二维）：点 $P(x_0, y_0)$ 到直线 $l: Ax + By + C = 0$ 的距离
    $$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$
    此公式要求直线方程必须化为一般式。

3.  **两平行直线间的距离公式**（二维）：平行线 $l_1: Ax + By + C_1 = 0$ 与 $l_2: Ax + By + C_2 = 0$ 的距离
    $$d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$
    应用此公式的前置条件是 $x$ 与 $y$ 的对应系数必须先化成相等。

**关键方法**

-   **坐标化**：将几何条件转化为坐标及方程，是解决解析几何最值、轨迹等问题的核心手段。
-   **向量法**：在立体几何或判断点相对于直线的位置时，可化为投影向量与叉乘模长求解，点到直线的距离 $d = \frac{|\vec{PA} \times \vec{n}|}{|\vec{n}|}$（$\vec{n}$ 为直线的方向向量）。
-   **降维思想**：将空间点到平面、异面直线的距离，通过构造直角三角形或转换顶点的方法化为点线距或两点距处理。

**高考典型应用**

1.  **最值与范围问题**：在给定曲线上寻找一点，使其到某定点或某直线的距离最大或最小。常通过设点坐标、代入距离公式构造关于坐标的单变量函数，并利用二次函数或三角代换求最值。
2.  **存在性与距离和差**：探索是否存在某点满足到两定点的距离之和/差为定值（对应椭圆/双曲线定义），或到某点与到某直线的距离相等（抛物线定义）。
3.  **圆与直线的位置关系判定**：比较圆心到直线的距离 $d$ 与半径 $r$，直接得到相离 ($d>r$)、相切 ($d=r$) 或相交 ($d<r$)。

## ⚠️ 高频易错点

-   **直线方程非一般式**：直接套用点线距公式前，务必确认直线方程为 $Ax + By + C = 0$ 的标准形式（$A$、$B$ 必须与 $x$、$y$ 对齐）。
-   **遗忘绝对值符号**：点线距公式的分子必须带绝对值，否则在求解含参数方程或涉及多个点位于直线不同侧时极易漏解或出错。
-   **平行线距公式的系数未统一**：求两平行线距离时，误将形如 $2x + 3y + 4 = 0$ 和 $4x + 6y - 1 = 0$ 的系数 $C_1$、$C_2$ 直接代入；必须先将两方程对应系数化为相同（如后者化为 $2x + 3y - 0.5 = 0$）再代入公式。
-   **混淆二维与三维公式**：求三维空间中向量或两点距离时，误用二维两点距公式，漏掉 $z$ 分量。
