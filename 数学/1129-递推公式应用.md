# 递推公式应用

> **一句话**：递推公式是给出数列相邻项之间数量关系（如 $a_{n}$ 与 $a_{n-1}$、$a_{n-2}$）的等式，结合首项可唯一确定整个数列。
> **English**: A recurrence formula is an equation that defines a sequence by expressing each term in relation to its preceding terms, and together with initial terms it uniquely determines the whole sequence.

## 🎓 考点精讲

**定义/原理**
若数列 $\{a_n\}$ 的任意项 $a_n$（$n\ge k$）能用其前若干项 $a_{n-1}, a_{n-2}, \dots, a_{n-k}$ 显式表示，即
$$a_n = f(a_{n-1}, a_{n-2}, \dots, a_{n-k})$$
则称该关系为 **$k$ 阶递推公式**。高考主要考查一阶线性递推、二阶线性递推以及可通过变形转化为等差数列或等比数列的递推。例如：
- 一阶线性：$a_{n+1} = pa_n + q$（$p,q$ 为常数）
- 邻项递推：$a_{n+1} = a_n + f(n)$（累加法）
- 邻项作商：$a_{n+1} = a_n \cdot g(n)$（累乘法）

**关键方法**
1. **累加法**：形如 $a_{n} - a_{n-1} = f(n)$，将 $n$ 从 $2$ 到 $n$ 的等式累加，得 $a_n = a_1 + \sum_{k=2}^n f(k)$。
2. **累乘法**：形如 $\frac{a_n}{a_{n-1}} = g(n)$，累乘得 $a_n = a_1 \cdot \prod_{k=2}^n g(k)$。
3. **待定系数法**（构造等比）：对 $a_{n+1} = pa_n + q$（$p\ne 1$），可设 $a_{n+1} + \lambda = p(a_n + \lambda)$，解得 $\lambda = \frac{q}{p-1}$，则 $\{a_n + \lambda\}$ 为等比数列。
4. **取倒数法**：对 $a_{n+1} = \frac{pa_n}{qa_n + r}$ 等分式型，两边取倒数转化为线性递推。
5. **特征根法**（二阶常系数线性递推）：对 $a_{n+2} = pa_{n+1} + qa_n$，解特征方程 $x^2 - px - q = 0$，根据根的情况写出通项结构。

**典型应用**
高考常以“给出递推公式求通项公式”或“证明一个由递推生成的数列为等差/等比数列”设问。如：
- 给出 $a_1=1$，$a_{n+1}=2a_n+3$，求 $a_n$。  
  构造：$a_{n+1}+3 = 2(a_n+3)$，故 $\{a_n+3\}$ 是首项 $4$、公比 $2$ 的等比数列，得 $a_n = 2^{n+1} - 3$。
- 给出 $a_1=1$，$a_{n+1}=a_n+2n$，用累加法得 $a_n = n^2 - n + 1$。

## ⚠️ 高频易错点

- 累加/累乘时忽略项数：从 $n=2$ 到 $n$ 共有 $n-1$ 个等式，求和后误漏 $a_1$ 或错定下标。
- 构造等比数列时待定系数计算错误：$a_{n+1}=pa_n+q$ 应配凑 $a_{n+1}+\frac{q}{p-1}=p(a_n+\frac{q}{p-1})$，学生常将分母记混为 $1-p$。
- 对分式型递推直接取倒数后遗忘分母为零的讨论，或未验证新序列的首项非零。
- 对二阶递推使用特征根法时，根为共轭复数或重根时不会正确写出通项（需用辅助角形式或含 $n$ 因式的形式）。
