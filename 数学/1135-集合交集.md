# 集合交集

> **一句话**：由所有同时属于集合 A 与集合 B 的元素构成的集合，称为 A 与 B 的交集，记作 A ∩ B。
> **English**: The intersection of two sets A and B, denoted by A ∩ B, is the set of all elements that belong to both A and B.

## 🎓 考点精讲

**定义/原理**  
对于两个集合 A 与 B，交集定义为  
$$A \cap B = \{ x \mid x \in A \text{ 且 } x \in B \}.$$  
交集的韦恩图表现为两个集合区域的重叠部分。由定义直接推导的性质有：  
- $A \cap B = B \cap A$（交换律）；  
- $A \cap A = A$，$A \cap \varnothing = \varnothing$；  
- 若 $A \subseteq B$，则 $A \cap B = A$。  

**关键方法**  
求集合交集的核心步骤：  
1. **确定元素**：若集合以列举法给出，直接找出公共元素；若以描述法给出（如不等式解集），先化为最简形式再求公共部分。  
2. **数轴或韦恩图辅助**：对于连续数集（区间），画数轴取重合部分，注意端点的开闭。对于有限集，可用韦恩图直观展示。  
3. **转化条件**：涉及方程、不等式的交集，等价于联立条件求解。  

**典型应用**  
高考中通常以选择题或填空题直接考查交集的运算，常见设问：  
- 直接给出两个集合求 $A \cap B$。  
  例：设 $A=\{x\mid x^2-3x+2 \le 0\}=\{x\mid 1 \le x \le 2\}$，$B=\{x\mid 1<x<3\}$，则 $A \cap B = (1,2]$。  
- 结合函数定义域、值域、不等式解集、方程根等构造集合，求交集后解决存在性、参数范围问题。  
  解题关键：先准确化简每个集合，再依交集定义取公共部分，特别注意边界点是否符合“且”的条件。

## ⚠️ 高频易错点

- **忽略空集**：当 $A \cap B = \varnothing$ 时，学生会漏掉这一情况，尤其在含参数的交集问题中忘记讨论空集导致失根或错解。  
- **端点取舍错误**：不等式解集的交集，学生常搞混开区间与闭区间的取舍，例如将 $(1,2]$ 与 $[1,2)$ 混淆，未严格检验端点是否同时满足两个条件。  
- **描述法化简不彻底**：如 $A=\{x\mid x^2-4=0\}$ 未化为 $\{-2,2\}$，$B=\{x\mid x(x-2)=0\}$ 未化为 $\{0,2\}$ 时，直接凭感觉找公共元素容易漏掉 $2$ 或误取 $-2$。  
- **概念混淆**：将交集记成并集符号 $\cup$，或做运算时误用并集规则。
