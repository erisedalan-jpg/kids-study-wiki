# 集合基本概念

> **一句话**：集合是由确定的研究对象构成的整体，元素与集合之间具有确定的“属于”或“不属于”关系。
> **English**: A set is a well-defined collection of distinct objects, where each object either belongs or does not belong to the set.

## 🎓 考点精讲

**定义/原理**：集合是数学中最基本的不定义概念。给定一个集合 $A$ 和一个对象 $x$，关系 $x \in A$（属于）与 $x \notin A$（不属于）必居其一。集合的表示法有三种：列举法（如 $\{1, 2, 3\}$）、描述法（如 $\{x \mid x > 0\}$）、Venn 图。常用数集符号为 $\mathbb{N}$（自然数集）、$\mathbb{Z}$（整数集）、$\mathbb{Q}$（有理数集）、$\mathbb{R}$（实数集）。集合间的基本关系包括子集 $A \subseteq B$、真子集 $A \subsetneqq B$、相等 $A = B$。空集 $\varnothing$ 是任何集合的子集，即 $\varnothing \subseteq A$ 恒成立。含有 $n$ 个元素的集合，其子集个数为 $2^n$，真子集个数为 $2^n - 1$，非空真子集个数为 $2^n - 2$。

**关键方法**：处理集合问题首选三种翻译策略——将文字翻译为符号语言、将描述法翻译为列举法、将抽象关系翻译为Venn图或数轴。解决含参集合问题时，必须抓住三点：先确定空集是否成立（空集是任何集合的子集）；再处理端点等号问题（借助数轴判断边界能否取等）；最后检验互异性（集合内元素不得重复）。

**典型应用**：高考对集合的考查通常置于第1题，以选择题形式出现。典型设问有两类：一是给定两个集合的表示，求交、并、补运算结果（如设 $A=\{x \mid -1<x<3\}$，$B=\{x \mid x \geq 1\}$，求 $A \cap B$）；二是以集合为载体综合考查不等式、函数定义域与值域（如 $A=\{x \mid y=\sqrt{1-x}\}$，$B=\{y \mid y=2^x+1\}$，求 $A \cup B$）。解题核心是准确将集合“翻译”成具体元素或范围。

## ⚠️ 高频易错点

- **空集遗忘**：讨论 $A \subseteq B$ 时忘记考虑 $A=\varnothing$ 的情形，导致含参分类讨论漏解。例如“$A \subseteq B$，其中 $A=\{x \mid ax-1=0\}$”，必须单独讨论 $a=0$ 时 $A=\varnothing$。
- **端点等号判断失误**：用数轴做交、并、补运算时，边界点能否取“=”判断错误。应严格逐点验算端点的归属。
- **元素互异性遗忘**：列举法给出的集合中出现重复元素，或解方程后检验时忽略根是否相等。例如 $\{x \mid x^2 - 2x + 1=0\}=\{1\}$，而非 $\{1, 1\}$。
- **集合元素含义混淆**：未区分描述法中竖线前后的变量含义，如 $\{x \mid y=\sqrt{x}\}$ 表示定义域 $[0, +\infty)$，而 $\{y \mid y=\sqrt{x}\}$ 表示值域 $[0, +\infty)$，$\{(x,y) \mid y=\sqrt{x}\}$ 表示曲线上的点集，三者完全不同。
