# 集合的交集

> **一句话**：由所有既属于集合A又属于集合B的元素组成的集合，记作A∩B。
> **English**: The intersection of two sets A and B is the set containing all elements that belong to both A and B, denoted A∩B.

## 🎓 考点精讲

**定义/原理**  
对于任意两个集合A和B，它们的交集定义为：  
$$
A \cap B = \{ x \mid x \in A \ \text{且} \ x \in B \}
$$  
交集满足交换律 $A \cap B = B \cap A$、结合律 $(A \cap B) \cap C = A \cap (B \cap C)$，且 $A \cap A = A$，$A \cap \varnothing = \varnothing$。  
当 $A \cap B = \varnothing$ 时，称A与B不交。

**关键方法**  
求解集合交集的标准方法：  
1. 明确集合的表示方式（列举法、描述法、区间或不等式）。  
2. 将集合元素或条件具体化，例如解出描述法中的不等式或方程。  
3. 求公共部分：对于列举法直接取共同元素；对于数集（连续区间）取数轴的公共覆盖区间；对于抽象集合借助Venn图或性质推理。  
4. 最后用列举法或描述法写出交集，注意元素的互异性与端点取舍。

**典型应用**  
高考中常以选择题、填空题出现，主要设问方式有：  
- 给定两个具体集合（如 $A=\{1,2,3\}, B=\{2,3,4\}$），直接求 $A \cap B$。  
- 给出不等式描述的集合（如 $A=\{x \mid -1 < x \le 3\}, B=\{x \mid 0 \le x < 4\}$），求交集并用区间表示。  
- 结合函数定义域、值域、方程或不等式的解集（如 $A=\{y \mid y=\sqrt{x-1}\}, B=\{y \mid y=x^2+1\}$），通过求公共范围得出交集。  
解题关键：正确转化条件，画数轴帮助确定公共区间，注意空心、实心端点。

## ⚠️ 高频易错点

- **忽视空集情形**：当两个集合无公共元素时，交集为空集 $\varnothing$，不能错误地写成 $\{ \}$ 以外的其他形式，也不可遗漏空集作为结果。  
- **描述法中元素混淆**：未能准确区分集合的代表元素，如 $\{x \mid y=\sqrt{x}\}$ 与 $\{y \mid y=\sqrt{x}\}$ 是不同的集合，错误理解导致交集求错。  
- **区间端点开闭错误**：利用数轴求交集时，对端点处是否包含（空心/实心）判断失误，尤其当边界值等于端点时漏掉或多余包含。  
- **忽略元素的互异性**：列举法表示交集时，若同一元素重复出现只写一次，不能写成多重元素。
