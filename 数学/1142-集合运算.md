# 集合运算

> **一句话**：集合运算是由已知集合构造新集合的数学操作，主要包括并、交、补（差）运算，是描述与处理分类关系的核心工具。
> **English**: Set operations are mathematical processes, such as union, intersection, and complement, used to construct new sets from given ones, serving as a fundamental tool for describing and handling classification relationships.

## 🎓 考点精讲

**定义与原理**
对于两个集合 $A$ 和 $B$（通常在某全集 $U$ 下讨论）：
- **并集** $A \cup B = \{ x \mid x \in A \text{ 或 } x \in B \}$；
- **交集** $A \cap B = \{ x \mid x \in A \text{ 且 } x \in B \}$；
- **补集** $\complement_U A = \{ x \mid x \in U \text{ 且 } x \notin A \}$（常记作 $\overline{A}$）；
- **差集** $A \setminus B = \{ x \mid x \in A \text{ 且 } x \notin B \}$。
核心性质：交换律、结合律、分配律（$A\cap(B\cup C)=(A\cap B)\cup(A\cap C)$，其对偶亦成立）以及德摩根律：
$\complement_U (A\cup B)=\complement_U A \cap \complement_U B$，
$\complement_U (A\cap B)=\complement_U A \cup \complement_U B$。

**关键方法**
1. **Venn 图与数轴**：对于离散有限集，利用 Venn 图直观处理容斥关系；对于连续数集（不等式解集），借助数轴进行交、并、补运算，尤其注意端点虚实。
2. **分类讨论**：参数含集合关系时，常需讨论子集是否为空集；补集运算常转化为“全集减去集合本身”进行元素列举。
3. **转化与化归**：将抽象集合条件转化为具体不等式或方程，如 $A\subseteq B$ 转化为对应区间端点的比较关系。

**典型应用**
高考常以选择题或填空题出现，主要设问角度：
- 给定两个不等式解集，求 $A\cap B$、$A\cup (\complement_U B)$ 等，考查数集运算与区间端点取舍。
- 已知集合关系（如 $A\cap B = B$），求参数取值范围，需注意 $B=\varnothing$ 的情形。
- 新定义运算：定义一种新的集合运算（如 $A\otimes B = \complement_U(A\cap B)$），要求计算具体结果或判断运算律是否成立，考查抽象符号理解与性质验证。

## ⚠️ 高频易错点

- **忽略空集**：在 $A\subseteq B$ 或 $A\cap B=A$ 等条件中，忽视 $A=\varnothing$ 的可能，导致参数漏解。
- **端点取舍混淆**：用数轴求解不等式解集的交并补时，等号是否可取判断错误，常见误把开区间当成闭区间或反之。
- **补集范围误认**：计算补集时未明确全集 $U$，例如题目隐含 $U=\mathbf{R}$，却误当成整数集或给定有限集。
- **运算律记混**：德摩根律记忆错误，写反并集与交集符号，或在分配律应用中变形出错。
