# 零点存在定理

> **一句话**：若函数 f(x) 在闭区间 [a,b] 上连续，且 f(a)·f(b) < 0，则在开区间 (a,b) 内至少存在一个零点。
> **English**: If a function f is continuous on [a,b] and f(a) and f(b) have opposite signs, then there exists at least one point c in (a,b) such that f(c)=0.

## 🎓 考点精讲

**定义/原理**：零点存在定理是连续函数介值定理的直接推论。严格叙述为：若函数 y=f(x) 满足：
1. 在闭区间 [a,b] 上连续；
2. f(a)·f(b) < 0（即端点函数值异号）；

则在开区间 (a,b) 内至少存在一点 c，使得 f(c)=0。注意定理仅保证“至少有一个”零点，不告知具体个数，也不提供精确求解方法。

**关键方法**：高考中解决“判断零点所在区间”问题的标准步骤：
1. 查验函数是否在给定区间连续（基本初等函数在定义域内均连续）；
2. 计算区间端点函数值，判断是否异号；
3. 若异号，应用定理直接判定存在零点；若同号，则定理失效，需结合单调性另法判断。

**典型应用**：
- **零点区间锁定**：如判断方程 ln x + x - 2 = 0 的根所在区间，可构造函数 f(x)=ln x + x - 2，分别计算 f(1) = -1 < 0，f(2) = ln 2 > 0，因 f(x) 在 (0,+∞) 连续，故在 (1,2) 内必有根。
- **结合二分法近似求解**：利用定理反复缩小区间，逐步逼近零点近似值，该类流程题常见于选择题或填空压轴的位置。

## ⚠️ 高频易错点

- **忽视连续性前提**：若函数在区间内存在间断点（如 f(x)=1/x 在含0区间），即使端点值异号也不能用定理。必须先行确认连续性。
- **f(a)·f(b) < 0 仅为充分条件**：函数有零点不一定端点函数值异号（如 f(x)=x² 在 [-1,1] 有零点但端点同号），切勿将定理逆用。
- **混淆“存在唯一”条件**：零点存在定理只保证存在性。若要判定唯一，必须附加函数在该区间上严格单调的条件，通常需结合导数符号判断。
- **计算端点值时忽略定义域边界**：对数、根式等函数需先确认区间端点是否在定义域内，取不在定义域的端点值代入属无效操作。
