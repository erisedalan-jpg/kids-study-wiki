# 频数分布表

> **一句话**：频数分布表是将数据按一定组距分组，列出各组内数据个数（频数）的统计表格。
> **English**: A frequency distribution table is a table that groups data into intervals of equal width and lists the number of observations (frequency) in each interval.

## 🎓 考点精讲

**定义与原理**  
对于一组样本数据，先求极差 $R = \max - \min$，再选定组距 $d$ 与组数 $k$（常用 $k = \lceil R/d \rceil$ 或由数据量 $n$ 按经验公式确定），将数据从小到大分为 $k$ 个左闭右开（或左开右闭）区间。统计落入每个区间的数据个数，称为**频数** $f_i$；计算**频率** $p_i = f_i/n$，其中 $n$ 为样本容量。频数分布表的核心是记录各组区间、频数、频率及**累积频数**（从小到大累加频数）。

**关键方法**  
绘制频数分布表的标准步骤：  
1. **求极差** $R$；  
2. **决定组距与组数**：组距宜取整数或 $0.5$ 的倍数，组数通常在 $5\sim15$ 之间，避免过少使信息损失、过多使频数过于零散；  
3. **确定分点**：第一组下限略小于最小值，保证所有数据被覆盖，且分点一般比原数据多取一位小数；  
4. **列频数分布表**：按组别划记、计数频数，进一步计算频率与累积频数。  
高考中常将频数分布表与**频率分布直方图**互推：直方图的小长方形面积 $=$ 频率 $=$ 频数$/n$，其高 $=$ 频率/组距。因此，由频数分布表可直接得直方图各矩形高度，反之亦可补全表格缺失数据。

**典型应用**  
- **根据原始数据列出频数分布表**：如给出 $30$ 名学生的考试成绩，要求将成绩分成 $[0,60),[60,70),\dots,[90,100]$ 并列出各组频数；  
- **补全不完整的频数分布表/直方图**：已知部分组频数或频率及样本容量，利用“频数和为 $n$”“频率和为 $1$”列方程求缺失值；  
- **由频数分布表估计数字特征**：用组中值 $x_i$ 计算平均数 $\bar{x} = \frac{\sum x_i f_i}{n}$，用累加频率估计中位数（中位数对应累积频率 $50\%$ 的分界点），结合直方图由面积比例求中位数。

## ⚠️ 高频易错点

- **组距与组数不合理**：组距过大导致分布信息掩盖，组距过小使得频数分布过于零乱，影响后续直方图与数字特征的合理性；未考虑“上限不在内”原则导致分点边界数据重复计数或遗漏。  
- **频数与频率混淆**：误将频数填为频率，或在频率分布直方图中误将纵轴“频率/组距”当作频率，导致计算面积时忘记乘以组距。  
- **中位数估计公式用错**：用累加频率求中位数时，未正确使用“中位数所在组的下限 $L$、该组频数 $f_m$、组距 $d$ 及前一组累积频率 $F$”，公式为 $L + \frac{0.5 - F}{f_m/n} \cdot d$，其中分子分母带不带 $n$ 必须根据累积是频数还是频率保持一致。  
- **频率分布直方图与频数分布表数据不一致**：绘制直方图时，忘记将频数转换为频率，导致纵坐标错误，使图形无法真实反映分布密度。
