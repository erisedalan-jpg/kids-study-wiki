# 绝对值三角不等式

> **一句话**：对于任意实数 $a,b$，有 $||a|-|b|| \le |a \pm b| \le |a|+|b|$，刻画了绝对值的“和”与“差的绝对值”之间的不等关系。
> **English**: For any real numbers $a,b$, the inequality $||a|-|b|| \le |a \pm b| \le |a|+|b|$ holds.

## 🎓 考点精讲

- **定义/原理**  
  绝对值三角不等式由两个核心不等式构成：  
  ① **和的不等式**：$|a+b| \le |a|+|b|$，当且仅当 $ab \ge 0$ 时取等。  
  ② **差的不等式**：$||a|-|b|| \le |a-b|$，当且仅当 $ab \ge 0$ 时取等。  
  统一形式：$||a|-|b|| \le |a \pm b| \le |a|+|b|$。  
  其本质利用了绝对值的非负性与向量模长的三角不等式，在实数域中可由绝对值的分段定义推导。右侧等号成立对应同号，左侧等号成立对应异号或至少一个为零。

- **关键方法**  
  1. **直接放缩**：已知某些量的绝对值范围，通过 $|x+y| \le |x|+|y|$ 或 $|x-y| \ge ||x|-|y||$ 构造目标绝对值的不等式。  
  2. **“取等条件”逆向分析**：求最值时，先写出不等式，再根据同号/异号条件确定参数范围或验证能否取等。  
  3. **绝对值恒等式的三角形式**：处理形如 $|f(x)|+|g(x)|$ 或 $|f(x)-g(x)|$ 的问题时，常借助三角不等式转化为定值，实质是距离和或差的最值问题。  
  解题核心是**观察绝对值内代数式的符号关联**，选择正确的放缩方向（放大用右半部，缩小用左半部）。

- **典型应用**  
  高考常以两种面貌出现：  
  **① 含绝对值的不等式证明或最值**：如已知 $|x-2| < 1, |y-3| < 2$，求证 $|2x+y-7| < 4$。直接用 $|2x+y-7| = |2(x-2)+(y-3)| \le 2|x-2|+|y-3| < 2\times1+2=4$。  
  **② 函数零点或绝对值和函数的最值**：如求 $f(x)=|x-1|+|x-3|$ 的最小值，可直接由 $|x-1|+|x-3| \ge |(x-1)-(x-3)|=2$，当 $1 \le x \le 3$ 时取等；或求 $g(x)=|x-1|-|x-3|$ 的最值，利用 $||x-1|-|x-3|| \le |(x-1)-(x-3)|=2$ 得到值域 $[-2,2]$。  

## ⚠️ 高频易错点

- **放缩方向混淆**：不等式 $|a|-|b| \le |a-b|$ 与 $|a-b| \le |a|+|b|$ 往往用反，尤其由条件 $|x|<1$ 导出 $|x-2|$ 的范围时，错误地写 $|x-2| < 1+2=3$ 而忽略等号条件或符号。
- **取等条件分析不全**：$|a+b|=|a|+|b|$ 要求 $ab \ge 0$，而 $||a|-|b||=|a-b|$ 同样要求 $ab \ge 0$。常见错误是仅处理 $ab>0$ 漏掉 $a=0$ 或 $b=0$ 时也成立。
- **忽略绝对值本身的非负性**：在证明恒成立问题中，直接使用三角不等式时忘记补充说明“当且仅当…”，导致最值问题缺失取等验证，或证明过程不严密而扣分。
- **符号误用**：将 $|a \pm b| \le |a|+|b|$ 错误记忆为 $|a \pm b| \le |a|-|b|$，或对相减形式的三角不等式方向记反（正确的是 $||a|-|b|| \le |a-b|$，而非 $|a-b| \le ||a|-|b||$）。
