## 课本定义

把两个含相同未知数的二元一次方程合在一起，就组成了一个**二元一次方程组**。使方程组中每个方程都成立的一组未知数的值，叫做这个方程组的**解**。解方程组用**代入消元法**或**加减消元法**，将"二元"化为"一元"。

---

## 📚 6-12 岁回溯版

### "两个条件，确定两个未知数"

**为什么需要方程组？**

一个二元一次方程（如 x + y = 10）有无数个解。
但如果再加一个条件（如 x - y = 2），就能确定唯一答案！

**生活例子：鸡兔同笼变体**
"小华和小明共有20张贴纸，小华比小明多4张，各有多少张？"

设小华有 x 张，小明有 y 张：
- 条件1：x + y = 20
- 条件2：x - y = 4

这就是一个二元一次方程组！

**求解（直觉法）：**
两式相加：2x = 24，x = 12
代回：12 + y = 20，y = 8

答：小华12张，小明8张。

**方程组的写法：**
$$\begin{cases} x + y = 20 \\ x - y = 4 \end{cases}$$

用大括号"{"连在一起，表示两个方程**同时**成立。

## 🎓 12+ 进阶版

### 严格定义

**二元一次方程组**：由两个含相同两个未知数（x, y）的二元一次方程组成的方程组。

$$\begin{cases} a_1x + b_1y = c_1 \\ a_2x + b_2y = c_2 \end{cases}$$

**解**：使两个方程**同时**成立的有序数对 (x₀, y₀)。

### 解的情况

| 情况 | 条件（系数关系） | 解的个数 | 几何意义 |
|------|----------------|---------|---------|
| 有唯一解 | a₁/a₂ ≠ b₁/b₂ | 1个 | 两直线相交 |
| 无穷多解 | a₁/a₂ = b₁/b₂ = c₁/c₂ | 无穷多 | 两直线重合 |
| 无解 | a₁/a₂ = b₁/b₂ ≠ c₁/c₂ | 0个 | 两直线平行 |

（七年级一般只研究有唯一解的情形）

### 解方程组的方法概述

**代入消元法**：从一个方程中解出一个未知数，代入另一个方程，化二元为一元。

**加减消元法**：将两方程适当乘以系数后相加或相减，消去一个未知数。

（详见 [[120-代入消元法|代入消元法]] 和 [[123-加减消元法|加减消元法]] 词条）

### 典型例题

**例题1：** 解方程组

$$\begin{cases} 2x + y = 7 \quad (1)\\ x - y = 2 \quad (2) \end{cases}$$

解（加减法）：(1) + (2)：3x = 9，x = 3
代入 (2)：3 - y = 2，y = 1
答：x = 3，y = 1（即解为 (3, 1)）

验证：(1) 2(3)+1=7✓，(2) 3-1=2✓

**例题2：** 解方程组

$$\begin{cases} x + 2y = 8 \quad (1)\\ 3x - y = 3 \quad (2) \end{cases}$$

解（代入法）：由 (1) 得 x = 8 - 2y，代入 (2)：
3(8-2y) - y = 3
24 - 6y - y = 3
-7y = -21
y = 3
x = 8 - 2(3) = 2
答：x = 2，y = 3

**例题3（应用题）：** 商店有苹果和橙子共50斤，苹果每斤3元，橙子每斤5元，共收款190元。各有多少斤？

设苹果 x 斤，橙子 y 斤：
$$\begin{cases} x + y = 50 \\ 3x + 5y = 190 \end{cases}$$
由第一式 x = 50 - y，代入第二式：
3(50-y) + 5y = 190
150 - 3y + 5y = 190
2y = 40，y = 20，x = 30

答：苹果30斤，橙子20斤。

### 易错点

1. **验证解时，必须代入方程组中每个方程**，只代一个不算完全验证
2. **解是有序数对 (x, y)**，不能只写 x 或只写 y
3. **消元时注意符号**：加减时不要漏乘

### 知识联系

```
二元一次方程组
├── 基础 → 二元一次方程
├── 方法 → 代入消元法（替换）
├── 方法 → 加减消元法（合并）
├── 几何 → 两直线的交点（坐标）

```
