## 课本定义

能使不等式成立的未知数的值，叫做**不等式的解**；所有解的集合，叫做不等式的**解集**。求不等式解集的过程，叫做**解不等式**。解集通常在**数轴**上表示。

---

## 📚 6-12 岁回溯版

### "满足不等式的所有答案"

**不等式的"解"和方程的"解"有什么不同？**

- 方程 x + 1 = 5 的解：x = 4（只有一个值）
- 不等式 x + 1 > 5 的解：x > 4（所有大于4的数）

不等式的解**不是一个值，而是一片数值**，这片数值叫**解集**。

**在数轴上表示解集：**

x > 4 的数轴表示：
```
———————————o——————————→
          4    5    6
（4右边，空心圆，表示不含4）
```

x ≤ 2 的数轴表示：
```
←——————————•————————
   0   1   2
（2左边，实心圆，表示含2）
```

**空心圆 vs 实心圆：**
- **空心圆**：端点不在解集里（严格不等式 >, <）
- **实心圆**：端点在解集里（≥, ≤）

**解集的常见形式：**
| 解集 | 数轴 | 意思 |
|------|------|------|
| x > a | a 右，空心圆 | a 右边所有数 |
| x ≥ a | a 右，实心圆 | a 及 a 右边所有数 |
| x < a | a 左，空心圆 | a 左边所有数 |
| x ≤ a | a 左，实心圆 | a 及 a 左边所有数 |

## 🎓 12+ 进阶版

### 严格定义

**不等式的解**：使不等式成立的未知数的值。

**不等式的解集**：不等式所有解的集合。

**解不等式**：求不等式解集的过程（用变换等价不等式的方法）。

### 解集的表示方式

**数轴表示：**
在数轴上，满足条件的 x 的范围用粗线段（或射线）表示：
- 端点为 > 或 < 时：空心圆（开区间端点）
- 端点为 ≥ 或 ≤ 时：实心圆（闭区间端点）

**区间表示（初中了解，高中用）：**
| 解集 | 区间记号 | 含义 |
|------|---------|------|
| x > a | (a, +∞) | 开区间 |
| x ≥ a | [a, +∞) | 闭区间 |
| x < a | (-∞, a) | 开区间 |
| x ≤ a | (-∞, a] | 闭区间 |
| a < x < b | (a, b) | 开区间 |
| a ≤ x ≤ b | [a, b] | 闭区间 |

### 解集的性质

**等价变换**：对不等式进行合法操作（两边同加减、同乘除正数、同乘除负数且翻转），得到等价不等式，解集不变。

**解集可以是：**
- 射线（x > a 或 x < a）：无穷解
- 有限区间（a < x < b）：有限范围的无穷解
- 全体实数（如 x² ≥ 0）
- 空集（如 x > x + 1，无解）

### 典型例题

**例题1：** 解不等式 4x - 3 > 2x + 1，并在数轴上表示解集。

解：4x - 2x > 1 + 3
2x > 4
x > 2

数轴：在 2 右边画空心圆，向右延伸的射线。

**例题2：** 解不等式 2(x+1) ≤ 3(x-2)，并用区间记号表示。

解：2x + 2 ≤ 3x - 6
2x - 3x ≤ -6 - 2
-x ≤ -8
x ≥ 8（除以-1，翻转）

解集：x ≥ 8，区间记号 [8, +∞)。

**例题3：** 对解集 x < -1，举出3个属于解集的数和2个不属于解集的数。

属于解集：-2, -100, -1.5（均小于-1）
不属于解集：-1（等于，不满足严格小于），0（大于-1）

### 判断解是否在解集中

将值代入不等式，若不等式成立，则该值在解集中；否则不在。

### 易错点

1. **严格不等式（<, >）端点不在解集，用空心圆**
2. **空集也是解集**（当不等式无解时，解集为空集 ∅）
3. **解集是一个集合，用花括号或数轴表示**，不是单个值
