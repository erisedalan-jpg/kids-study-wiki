# 数列的概念及简单表示法

> **一句话**：数列是按照一定次序排列的一列数，每一项 $a_n$ 与其序号 $n$ 满足函数关系 $a_n = f(n)$。
> **English**: A sequence is an ordered list of numbers, where each term is related to its index by a function $a_n = f(n)$.

## 🎓 考点精讲

**定义与原理**  
数列是按一定次序排成的一列数：$a_1, a_2, a_3, \dots, a_n, \dots$，其中 $a_n$ 是数列的第 $n$ 项。  
- **通项公式**：若数列的第 $n$ 项 $a_n$ 与 $n$ 之间的关系可用一个公式 $a_n = f(n)$ 表示，则此公式为通项公式。例如 $a_n = 2n-1$ 给出奇数数列 $1,3,5,7,\dots$。  
- **递推公式**：给出数列的若干初始项及相邻项之间的关系。如 $a_1 = 1$，$a_{n+1} = a_n + 2$ 即为递推数列。  
- **前 $n$ 项和 $S_n$**：$S_n = a_1 + a_2 + \cdots + a_n$。当 $n \ge 2$ 时，$a_n = S_n - S_{n-1}$；当 $n=1$ 时，$a_1 = S_1$。这是联系通项与和的核心关系。

**关键方法**  
1. **由前几项写通项**：观察序号与项的数值特征（如符号、分子分母、常数增减），构造分式、指数、多项式等形式。  
2. **由递推求通项**：  
   - $a_{n+1} - a_n = f(n)$ 型：累加法，$a_n = a_1 + \sum_{k=1}^{n-1} f(k)$。  
   - $\frac{a_{n+1}}{a_n} = f(n)$ 型：累乘法，$a_n = a_1 \prod_{k=1}^{n-1} f(k)$。  
   - $a_{n+1} = pa_n + q$ ($p \neq 1$) 型：构造等比数列，如设 $a_{n+1} + \lambda = p(a_n + \lambda)$，解出 $\lambda$。  
3. **利用 $a_n$ 与 $S_n$ 关系**：已知 $S_n$ 求 $a_n$ 时，分段求解后务必检验 $n=1$ 是否满足通项，否则需分段表示。

**高考典型应用**  
- 给数列前几项，选择或填空写出一个可能的通项公式。例如 $-\frac{1}{2}, \frac{2}{3}, -\frac{3}{4}, \frac{4}{5}, \dots$，可归纳 $a_n = (-1)^n \frac{n}{n+1}$。  
- 给出递推关系求特定项或通项：如 $a_1 = 1$，$a_{n+1} = 2a_n + 1$，构造为 $a_{n+1}+1 = 2(a_n+1)$，得 $a_n = 2^n - 1$。  
- 给出 $S_n$ 表达式求 $a_n$ 或证明与数列相关的不等式，需熟练使用 $a_n = S_n - S_{n-1}$（$n \ge 2$）分段讨论。

## ⚠️ 高频易错点

- 在由 $S_n$ 求 $a_n$ 时忘记单独验证 $n=1$ 的情形，导致通项在 $n=1$ 时不成立而未分段表示。  
- 累加或累乘时，项数下标极易从 $1$ 到 $n-1$ 误写为 $1$ 到 $n$，导致 $a_n$ 多出一项。  
- 构造新数列时配凑常数错误，例如 $a_{n+1} = pa_n + q$ 中未正确解出待定系数 $\lambda = \frac{q}{p-1}$（$p \neq 1$）。  
- 混淆数列的前 $n$ 项和 $S_n$ 与通项 $a_n$ 的概念，例如在已知 $S_n = n^2$ 时直接认为 $a_n = n^2$。
