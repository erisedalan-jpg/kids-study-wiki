# 等差数列前n项和公式

> **一句话**：等差数列前n项和等于首项与末项之和乘以项数的一半，也可用首项、公差、项数表达为 $S_n = \frac{n}{2}[2a_1 + (n-1)d]$。
> **English**: The sum of the first n terms of an arithmetic sequence is given by $S_n = \frac{n}{2}(a_1 + a_n)$ or $S_n = \frac{n}{2}[2a_1 + (n-1)d]$.

## 🎓 考点精讲

**定义/原理**：  
若数列 $\{a_n\}$ 是等差数列，首项为 $a_1$，公差为 $d$，则其前 $n$ 项和 $S_n = a_1 + a_2 + \cdots + a_n$ 可由倒序相加法推导得到：

$$
S_n = \frac{n(a_1 + a_n)}{2} = \frac{n}{2}[2a_1 + (n-1)d].
$$

其中 $a_n = a_1 + (n-1)d$ 是第 $n$ 项。两个公式等价，前者已知首项、末项与项数时使用，后者已知首项、公差与项数时使用。

**关键方法**：  
1. **公式选择**：若题目给出 $a_1$、$a_n$ 和 $n$，直接用 $S_n = \frac{n(a_1+a_n)}{2}$；若给出 $a_1$、$d$ 和 $n$，用 $S_n = \frac{n}{2}[2a_1 + (n-1)d]$；若涉及 $S_n$ 与 $n$ 的关系，可构造关于 $n$ 的二次函数 $S_n = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$，利用二次函数性质处理最值或图像问题。  
2. **$S_n$ 的性质**：$S_n, S_{2n}-S_n, S_{3n}-S_{2n}$ 仍成等差数列，可用于快速求解片段和。  
3. **方程思想**：在已知 $S_n$ 与其他量（如某项、公差）的等式中，通常将 $S_n$ 用含 $a_1$ 和 $d$ 的表达式代入，转化为关于基本量的方程组求解。

**典型应用**：  
高考常考题型包括：直接求和计算；根据 $S_n$ 与项的关系求通项或参数（尤其注意 $n=1$ 时 $a_1=S_1$，$n\geq 2$ 时 $a_n = S_n - S_{n-1}$）；$S_n$ 的最值问题（如 $a_1>0,d<0$ 时，$S_n$ 有最大值，通过解 $a_n \geq 0$ 得 $n$ 的范围确定）；与函数、不等式结合，如证明 $\frac{S_n}{n}$ 为等差数列等。

## ⚠️ 高频易错点

- **错用项数**：从中间项开始求和时，误把项数当 $n$，实际应为下标差加一。例如求 $a_3$ 到 $a_{10}$ 的和，项数为 $10-3+1=8$，而非 $10$。
- **混淆 $S_n$ 与 $a_n$ 的关系**：由 $S_n$ 求 $a_n$ 时，忘记验证 $n=1$ 的情况，或直接认为 $a_n = S_n - S_{n-1}$ 对所有 $n$ 成立而不单独计算 $a_1$。
- **对 $S_n$ 的二次式理解不足**：$S_n = An^2 + Bn$（$A,B$ 为常数）是 $\{a_n\}$ 为等差数列的充要条件，但部分学生误以为任何 $S_n$ 的二次式都对应等差数列，忘记公差 $d=2A$ 必须为常数，且缺少常数项。
- **片段和性质误用**：片段和 $S_m, S_{2m}-S_m, S_{3m}-S_{2m}$ 成等差数列的前提是原数列为等差数列，且片段长度相同，有时误用于一般数列或长度不同的片段。
