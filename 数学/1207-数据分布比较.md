# 数据分布比较

> **一句话**：通过比较两组或多组数据的集中趋势（平均数、中位数、众数）与离散程度（极差、方差、标准差）等统计特征量，判断其分布的中心位置、波动大小及形状差异。
> **English**: Comparing data distributions means using measures of central tendency and dispersion to assess the differences in center, spread, and shape between data sets.

## 🎓 考点精讲

**定义/原理**  
数据分布比较的核心在于定量刻画两组数据在“中心”与“散布”上的异同。  
- **集中趋势**：描述数据向中心聚集的程度，常用特征量为  
  - 平均数 \(\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i\)  
  - 中位数（将数据排序后处于中间位置的数）  
  - 众数（出现次数最多的数）  
- **离散程度**：描述数据分散的程度，常用特征量为  
  - 极差 = 最大值 - 最小值  
  - 方差 \(s^2 = \frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2\)（样本方差）  
  - 标准差 \(s = \sqrt{s^2}\)  

比较时通常需结合统计图表（频率分布直方图、茎叶图、折线图等）提取数据，再计算并分析特征量。

**关键方法**  
1. **读取与整理**：从图表中提取原始数据或频数分布，明确每组数据的样本量。  
2. **计算特征量**：分别求出两组数据的平均数、中位数、方差（标准差），必要时计算极差或四分位距。  
3. **对比分析**：  
   - 比较平均数/中位数 → 判断整体水平的高低；  
   - 比较方差/标准差 → 判断成绩（或指标）的稳定性与波动大小，方差越小越稳定；  
   - 结合极差或直方图形状 → 分析极端值影响或分布偏态。  
4. **写出结论**：用规范的统计语言表述，如“甲的平均成绩与乙相当，但甲的成绩更稳定”。

**典型应用**  
高考常以解答题或选择题形式出现，如：  
- 给出甲乙两名运动员十次射击成绩的茎叶图，问谁的平均成绩更好、谁发挥更稳定；  
- 给出两个社区人群某项指标的频率分布直方图，比较众数、中位数的大小，并说明分布特点；  
- 结合方差公式进行决策：若需选派一人参加比赛，在平均成绩相近时，选择方差小的选手。

## ⚠️ 高频易错点

- **混淆平均数与中位数的适用场景**：当数据中含有极端值时，平均数会被拉高或拉低，失去代表性，此时中位数更能反映数据的一般水平。许多学生直接比较平均数而忽略极端值的影响。  
- **误把方差比较当作水平比较**：方差小仅说明数据波动小、更稳定，不等于数据本身的数值更好（如成绩方差小不代表成绩高）。必须同时结合集中趋势（平均数、中位数）进行综合判断。  
- **方差计算时忽略“除以样本容量”**：容易忘记除以 \(n\)，或混淆总体方差（除以 \(n\)）与样本方差（除以 \(n-1\)）的公式。在高考中，若未特别说明，通常按给定公式直接计算。  
- **直方图中中位数的错误估计**：中位数是面积平分点，而非最高矩形的中点。学生常直接从高矩形位置判断中位数，导致错误。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 平均数 | Mean | 所有数据之和除以个数 |
| 中位数 | Median | 按顺序排列后中间位置的数值 |
| 众数 | Mode | 出现次数最多的数值 |
| 方差 | Variance | 衡量数据离散程度的平方均值 |
| 标准差 | Standard Deviation | 方差的算术平方根，与原始数据单位一致 |
