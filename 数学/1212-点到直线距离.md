# 点到直线距离

> **一句话**：在平面直角坐标系中，点到直线的距离是指该点到已知直线的垂线段长度，可由点坐标与直线方程的一般式直接代入公式计算。
> **English**: The distance from a point to a line in a coordinate plane is the length of the perpendicular segment from the point to the given line, calculated using a standard formula involving the point's coordinates and the line's general equation.

## 🎓 考点精讲

- **定义/原理**：已知点 $P(x_0, y_0)$ 与直线 $l: Ax + By + C = 0$（其中 $A, B$ 不全为零），则点 $P$ 到直线 $l$ 的距离为：

  $$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$$

  该公式由向量投影或面积法推导而来。核心前提是**直线方程必须化为一般式**，否则代入会导致系统性错误。

- **关键方法**：问题求解遵循“一代两化三算”步骤。① 将直线方程统一化为 $Ax + By + C = 0$ 形式并确认 $A, B$；② 提取点坐标 $(x_0, y_0)$；③ 代入公式计算即可。高考中常遇两种延伸：其一，求两平行线的距离，先取其中一条直线上任一点，再用点到直线距离公式；其二，与圆结合判断位置关系，如比较圆心到直线的距离与半径的大小。

- **典型应用**：选择题中常直接计算点线距离或平行线间距。解答题中频繁作为中间步骤出现，例如求切线方程（令圆心到直线距离等于半径求解参数）、求面积最值（点到直线距离作为三角形的高）、判断直线与圆的位置关系等。典型设问如：“求点 $(2, -1)$ 到直线 $3x - 4y + 5 = 0$ 的距离”——直接代入得 $d = \frac{|6 + 4 + 5|}{5} = 3$。

## ⚠️ 高频易错点

- **忘记加绝对值**：分子漏写 $|\cdot|$ 导致结果出现负数。距离是非负量，分子带绝对值是几何要求的数学表现。
- **直线方程形式未统一**：直接将斜截式、点斜式代入，未先转化为一般式。例如误将 $y = 2x + 3$ 的 $A, B, C$ 直接取为 $2, -1, 3$ 通常会出错，应整理为 $2x - y + 3 = 0$ 后再代入。
- **分母平方和漏开根号**：只计算 $A^2 + B^2$ 而忘记取算术平方根，导致距离单位错误。
- **与平行线间距公式混淆**：求两平行线距离时，必须确认两直线 $A, B$ 值相同（需等价变形为 $Ax + By + C_1 = 0$ 和 $Ax + By + C_2 = 0$ 的形式），再使用 $d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$，直接在原式上套用极易出错。
