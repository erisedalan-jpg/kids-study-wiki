## 课本定义

**几何证明**：从已知条件出发，根据数学定义、公理和定理，经过一系列逻辑推理，证明某个几何命题正确的过程。证明的格式通常包括：**已知**、**求证**、**证明**三个部分，每步推理必须写出依据。

---

## 📚 6-12 岁回溯版

### "一步一步讲清楚'为什么'"

**证明**就是：用已经知道的数学规律，一步一步地解释清楚为什么某件事是对的。

不能只说"我觉得对"或"看起来是对的"——必须用**有根据的逻辑**说清楚！

**证明和验算的区别：**
- **验算**：代入具体数字看对不对（比如验证3,4,5满足勾股定理）
- **证明**：对所有情况都说清楚为什么对（勾股定理对所有直角三角形都成立）

**几何证明的格式：**

```
已知：（题目给的条件）
求证：（要证明的结论）
证明：
  第1步：__________（理由：__________）
  第2步：__________（理由：__________）
  ……
  所以，__________。∎（证毕）
```

**常用的理由（依据）：**
- 已知条件
- 对顶角相等（定理）
- 平行线的性质（定理）
- 邻补角互补（定义）
- 等量代换

**一个简单的证明例子：**

已知：直线AB与直线CD交于点O，∠AOC = 40°
求证：∠BOD = 40°

证明：
∵ ∠AOC 与 ∠BOD 是对顶角
∴ ∠BOD = ∠AOC（对顶角相等）
∵ ∠AOC = 40°
∴ ∠BOD = 40° ∎

## 🎓 12+ 进阶版

### 几何证明的要素

**证明的三部分：**
1. **已知**：题目给出的所有条件（写清楚已知什么）
2. **求证**：要证明的结论（写清楚要证什么）
3. **证明**：推理过程（每步写出做了什么，以及依据是什么）

**证明的书写规范：**
- 使用"∵"（因为）和"∴"（所以）
- 每个"所以"后面的结论，必须有对应的"因为"给出依据
- 推理链要连贯，不能跳步
- 最后用"∎"或"得证"结束

### 常用证明依据

| 依据 | 使用场景 |
|------|---------|
| 对顶角相等 | 两直线相交 |
| 邻补角互补（∠A + ∠B = 180°）| 角在一条直线上 |
| 平行线同位角相等 | 已知平行线 |
| 平行线内错角相等 | 已知平行线 |
| 平行线同旁内角互补 | 已知平行线 |
| 同位角相等 → 两线平行 | 已知角相等，证平行 |
| 等量代换 | a=b, b=c, 则 a=c |

### 典型例题

**例题1（基础型）：**

已知：如图，直线 MN 与直线 PQ 交于 O，∠MOP = 3∠NOP
求证：∠MOQ = 135°

证明：
设 ∠NOP = x，则 ∠MOP = 3x
∵ ∠MOP + ∠NOP = 180°（邻补角，MON 为直线）
∴ 3x + x = 180°
4x = 180°，x = 45°
∴ ∠NOP = 45°，∠MOP = 135°

∵ ∠MOQ 与 ∠MOP 是邻补角（QPN为直线……应为QON→修正：MON直线上）
实际上：∠MOQ = 180° - ∠MON？

修正解：
∵ ∠NOP 与 ∠MOQ 是对顶角
∴ ∠MOQ = ∠NOP × 3 ……

（注：实际题图标注决定角的关系，此题以教材版本为准，上述为示意解法）

**例题2（平行线型）：**

已知：AB ∥ CD，直线 EF 截两线于 G, H 点，∠1 = 65°（∠1 为 AB 上方同侧角）
求证：∠2 = 115°（∠2 为 CD 下方同侧角，与∠1 为同旁内角）

证明：
∵ AB ∥ CD（已知）
∴ ∠1 + ∠2 = 180°（两平行线间同旁内角互补）
∵ ∠1 = 65°（已知）
∴ ∠2 = 180° - 65° = 115°

**例题3（综合型）：**

已知：∠1 + ∠2 = 180°，∠2 = ∠3
求证：∠1 + ∠3 = 180°

证明：
∵ ∠2 = ∠3（已知）
∴ ∠1 + ∠3 = ∠1 + ∠2（等量代换，用∠3代替∠2）
∵ ∠1 + ∠2 = 180°（已知）
∴ ∠1 + ∠3 = 180° ∎

### 证明的思路

解题思路（倒推法）：
1. 先看**要证什么**（求证）
2. 想**需要什么条件**才能得出该结论
3. 再看已知中**有没有那些条件**
4. 如果没有，继续找中间步骤

### 易错点

1. **每步推理必须有依据**，不能"想当然"
2. **不能循环论证**（用结论来证明结论）
3. **证明不是计算**，每步不是算出数值，而是推出关系
4. **图形辅助看，但结论由逻辑保证**（图可能不准确）
