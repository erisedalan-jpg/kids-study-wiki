# 交集

> **一句话**：由所有同时属于集合A与集合B的元素构成的集合，称为A与B的交集，记作$A \cap B$。
> **English**: The intersection of sets A and B, denoted $A \cap B$, is the set of all elements that belong to both A and B.

## 🎓 考点精讲

**定义**：对于给定的两个集合$A$、$B$，由既属于$A$又属于$B$的所有元素组成的集合，叫做$A$与$B$的交集。用符号表示为：

$$ A \cap B = \{ x \mid x \in A \text{ 且 } x \in B \} $$

交集的本质是逻辑联结词“且”在集合运算中的体现。若$A \cap B = \varnothing$，则称$A$与$B$不相交。

**关键方法**：
1. **数轴法**（适用于连续数集）：将集合在数轴上标出，取公共覆盖区域即为交集。这是解不等式交集的通用方法。
2. **韦恩图法**（适用于抽象集合关系）：用封闭曲线表示集合，交集为重叠部分。
3. **列举法**（适用于有限集）：直接列出所有元素，取公共元素组成新集合。

**典型应用**：高考以选择题、填空题为主，常见于第1–3题。
- **集合运算混合**：已知$A=\{x\mid x^2-3x+2\leq 0\}$，$B=\{x\mid 1<x<3\}$，求$A \cap B$。解：$A=[1,2]$，$B=(1,3)$，在数轴上取公共部分得$(1,2]$。
- **含参交集问题**：已知$A=\{x\mid x>a\}$，$B=\{x\mid -1\leq x\leq 2\}$，若$A \cap B \neq \varnothing$，求$a$的范围。解：需满足$a<2$，即$a\in(-\infty,2)$。

## ⚠️ 高频易错点

- **端点取舍混淆**：求不等式解集的交集时，若一端在某个集合中为开区间，在另一个集合中为闭区间，交集中该端点取开。例如$(1,3]\cap[3,5)= \{3\}$，而$(1,3)\cap[3,5)=\varnothing$。务必逐个端点单独判断。
- **混淆“交集”与“并集”的符号和逻辑**：$\cap$（交）对应“且”，要求同时满足；$\cup$（并）对应“或”，只要满足一个即可。审题时将题目中的“且”“或”与运算符号严格对应。
- **忽视空集情形**：当两个集合没有公共元素时，交集为$\varnothing$，而非“不存在”或无解。讨论含参问题时，必须单独考虑交集为空的情况。
- **代表元素不清导致求错交集**：如$A=\{y\mid y=x^2+1\}$（值域），$B=\{(x,y)\mid y=x^2+1\}$（抛物线上的点集），$A \cap B = \varnothing$，因为一个是数集，一个是点集，元素类型不同，无法求交集。
