# 三角函数的最值问题

> **一句话**：求含三角函数表达式在给定条件下最大值或最小值的问题，核心方法包括利用有界性、辅助角公式、换元化为二次函数或借助导数求解。
> **English**: The problem of determining the maximum or minimum values of expressions containing trigonometric functions under given conditions, typically approached via boundedness, auxiliary angle formulas, substitution to quadratic forms, or differentiation.

## 🎓 考点精讲

**定义与原理**  
三角函数的最值问题，指在一定自变量范围（通常为实数集或其子区间）内，求形如 `y = A sin(ωx + φ) + k`、`y = a sin x + b cos x`、`y = a sin²x + b sin x cos x + c cos²x` 等表达式的最值。其理论根据是正（余）弦函数的有界性：`|sin x| ≤ 1`，`|cos x| ≤ 1`。因此，`y = A sin(ωx + φ) + k` 的最大值为 `|A| + k`，最小值为 `-|A| + k`（当 `A > 0` 时，最大值 `A + k`，最小值 `-A + k`）；若 `A < 0`，函数单调性相反，但仍可以通过转化为标准形式处理。

**关键方法**  
1. **有界性直接法**：适用于 `f(x) = A sin(ωx + φ) + k` 型，但需要结合给定区间与 `ωx + φ` 的范围判断最值是否恰在边界取得。  
2. **辅助角公式**：`a sin x + b cos x = √(a² + b²) sin(x + φ)`（其中 `tan φ = b/a`，并根据点 `(a, b)` 所在象限确定 φ 的象限），最值即为 `±√(a² + b²)`。  
3. **换元化为二次函数**：含有 `sin x ± cos x` 与 `sin x cos x` 的表达式，令 `t = sin x + cos x`，则 `t ∈ [-√2, √2]`，且 `sin x cos x = (t² - 1)/2`；若为 `sin x - cos x`，则 `sin x cos x = (1 - t²)/2`。将原式转化为关于 `t` 的二次函数，在闭区间上求最值。  
4. **导数法**：对可导三角函数在闭区间上求导，通过极值点与端点比较得到最值。  
5. **不等式法**：部分分式型或乘积型可利用基本不等式求最值，但必须验证等号成立条件。

**高考典型应用**  
高考中常在解答题设问“求 `f(x)` 的最大（小）值，并求此时 `x` 的集合”或“求 `f(x)` 的值域”。例如：
- `f(x) = 2 sin(2x − π/6)` 在 `[0, π/2]` 上的最值：由 `0 ≤ x ≤ π/2` 得 `-π/6 ≤ 2x − π/6 ≤ 5π/6`，结合正弦图像得到 `f(x) ∈ [-1, 2]`。
- `f(x) = sin x + cos x + sin x cos x`：令 `t = sin x + cos x ∈ [-√2, √2]`，得 `f(x) = t + (t²−1)/2 = (1/2)(t+1)² − 1`，当 `t=√2` 时得最大值 `(1/2)(√2+1)² − 1`，当 `t=−1` 时得最小值 `−1`。

## ⚠️ 高频易错点

- **忽视自变量范围直接套用有界性**：只记住 `sin x ∈ [-1, 1]`，却未结合定义区间具体判断，导致把局部最值当作全局最值。如求 `y = sin x` 在 `[π/4, 3π/4]` 上的值域时，误写为 `[-1, 1]`，正确应为 `[√2/2, 1]`。
- **辅助角公式中辅助角 φ 确定错误**：仅用 `tan φ = b/a` 求 φ，忽略点 `(a, b)` 所在象限，导致表达式变形出错。
- **换元遗忘范围限制**：令 `t = sin x + cos x` 后忘记 `t ∈ [-√2, √2]`，直接将二次函数对称轴带入求最值，导致超出实际范围。
- **二次函数型最值求错截断点**：化为二次函数后未判断对称轴与区间的位置关系，错误地把区间端点值或顶点值作为最值。
