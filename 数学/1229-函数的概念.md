# 函数的概念

> **一句话**：函数是数集间的一种特殊映射，要求定义域中每一个元素在值域中有唯一确定的元素与之对应。
> **English**: A function is a relation from a set of inputs to a set of possible outputs where each input is related to exactly one output.

## 🎓 考点精讲

**定义/原理**  
设 $A, B$ 是非空的数集，如果按照某种确定的对应关系 $f$，使对于集合 $A$ 中的任意一个数 $x$，在集合 $B$ 中都有唯一确定的数 $f(x)$ 和它对应，那么就称 $f: A \to B$ 为从集合 $A$ 到集合 $B$ 的一个函数，记作 $y = f(x), x \in A$。其中 $x$ 叫自变量，$x$ 的取值范围 $A$ 叫函数的定义域；与 $x$ 的值相对应的 $y$ 值叫函数值，函数值的集合 $\{f(x) \mid x \in A\}$ 叫函数的值域。函数的三要素：**定义域、对应法则、值域**。值域由定义域和对应法则唯一确定，因此两个函数相等当且仅当它们的定义域和对应法则完全相同。

**关键方法**  
1. **判断对应是否为函数**：检验定义域内每一个元素是否都有唯一确定的像，即“一对一”或“多对一”可以，“一对多”不行。  
2. **判断两个函数是否相等**：先看定义域是否相同，再看对应法则（解析式化简后）是否一致，与字母符号无关。  
3. **求复合函数的定义域**：已知 $f(x)$ 定义域为 $[a,b]$，求 $f(g(x))$ 定义域时，由 $a \le g(x) \le b$ 解出 $x$ 的范围，注意整体代换思想。

**典型应用**  
高考中常以选择题或填空题形式出现：  
- 给出若干对应关系图形或解析式，判断哪些是函数图像或函数（如通过作竖直线检验图像是否为函数：与 $y$ 轴平行的直线与图像最多一个交点）。  
- 判断选项中的函数是否表示同一函数，例如 $f(x)=x$ 与 $g(x)=\sqrt{x^2}$ 定义域不同，不是同一函数。  
- 与定义域结合的小综合题，如求抽象函数 $y=f(x+1)$ 的定义域，已知 $f(x)$ 的定义域是 $(0,1)$，则 $x+1 \in (0,1)$，解得 $x \in (-1,0)$。

## ⚠️ 高频易错点

- **忽视定义域优先性**：化简解析式后容易忽略定义域的变化，例如 $y=\dfrac{x^2}{x}$ 化简为 $y=x$，但前者的定义域为 $x \neq 0$，与 $y=x$（定义域 $\mathbb{R}$）不是同一函数。  
- **混淆 $f(x)$ 与 $f(x+1)$ 定义域**：已知 $f(x)$ 定义域是 $(0,1)$ ，求 $f(x+1)$ 定义域时，误认为直接平移区间。应理解为括号内整体范围不变，即 $x+1 \in (0,1)$，解得 $x \in (-1,0)$。  
- **函数与映射关系混淆**：函数是特殊的映射，须是非空数集；但判断对应关系时，常忽略值域是 $B$ 的子集，允许 $B$ 中有多余元素，但定义域中不能有元素无对应像。  
- **相等函数判断忽略定义域**：只看解析式是否变形相同，不验证自然定义域，如 $f(x)=x$ 与 $g(x)=(\sqrt{x})^2$ 不相等，因为后者定义域为 $[0,+\infty)$。
