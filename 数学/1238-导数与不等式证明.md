# 导数与不等式证明

> **一句话**：通过构造辅助函数并利用导数研究其单调性、极值与最值，从而推导并证明不等式恒成立的逻辑推理方法。
> **English**: A logical reasoning method that proves inequalities by constructing auxiliary functions and using derivatives to analyze their monotonicity, extreme values, and bounds.

## 🎓 考点精讲

**定义/原理**：该考点的核心是建立函数与不等式的映射关系。通常将待证不等式 $f(x) \ge g(x)$ 转化为研究差函数 $h(x) = f(x) - g(x)$ 的最值问题。利用求导判定 $h'(x)$ 的符号，若 $h(x)_{\text{min}} \ge 0$，则原不等式成立。其理论支柱是拉格朗日中值定理（高考中常以极限保号性与单调性的形式考查）。

**关键方法**：
1.  **直接作差法**：直接构造函数 $F(x) = left - right$，常规求导。
2.  **分离参数法**：当不等式含参数 $a$ 时，变形为 $a \le \phi(x)$，转化为求 $\phi(x)$ 的最值。
3.  **经典放缩法**：利用切线放缩（如 $e^x \ge x+1$，$\ln x \le x-1$，当且仅当 $x=0$ 或 $x=1$ 取等）进行“函数代换”，简化导数运算。
4.  **隐零点法**：对于存在不可求出的极值点 $x_0$，设而不求，代入函数表达整体代换。

**典型应用**：高考导数压轴题中，常要求证明如“当 $x>0$ 时，$e^x - \ln(x+2) > \frac{1}{2}$”或“对任意实数 $x$，求证 $x\ln x \ge x-1$”等超越不等式。

## ⚠️ 高频易错点

-   **忽略定义域**：在构造 $F(x) = \ln x + \dots$ 时，忘记标明 $x>0$，导致单调区间分析错误。
-   **求导法则混乱**：在处理 $e^{2x}$ 或 $\ln\left(\frac{1}{x}\right)$ 等复合函数求导时漏乘系数或内层导数。
-   **隐零点代换断裂**：找到一阶导的零点 $x_0$ 满足 $f'(x_0)=0$，但在代入原函数 $f(x_0)$ 化简时，未将指数与对数关系进行等价转化，导致无法判定最值符号。
-   **极值点偏移混淆**：在证明 $x_1 + x_2 > 2$ 类不等式时，错误地直接使用极值点对称，而未通过构造对称差函数进行严格论证。
