# 集合间的基本关系

> **一句话**：集合间的基本关系包括子集、真子集与相等，是描述集合之间包含与等价结构的数学语言。
> **English**: The basic relationships between sets include subset, proper subset, and equality, describing the inclusion and equivalence structure among sets.

## 🎓 考点精讲

**定义/原理**  
设 $A,B$ 为两个集合。  
- **子集**：若对任意 $x \in A$，都有 $x \in B$，则称 $A$ 是 $B$ 的子集，记作 $A \subseteq B$（或 $B \supseteq A$）。  
- **真子集**：若 $A \subseteq B$ 且存在 $x \in B$ 但 $x \notin A$，则称 $A$ 是 $B$ 的真子集，记作 $A \subsetneqq B$（或 $A \subsetneq B$）。  
- **相等**：$A = B \iff A \subseteq B$ 且 $B \subseteq A$。  
- **空集**：$\varnothing$ 是任何集合的子集，且是任何非空集合的真子集。  
集合关系的传递性：若 $A \subseteq B$，$B \subseteq C$，则 $A \subseteq C$。

**关键方法**  
1. **判断关系**：将集合用列举法、描述法（不等式、方程解集）或区间表示，利用数轴或韦恩图直观比较。  
2. **已知关系求参**：由 $A \subseteq B$ 或 $A \subsetneqq B$ 列不等式（组），特别注意：  
   - 必须讨论 $A = \varnothing$ 的情形；  
   - 端点能否取等需单独检验（代入原集合验证是否符合子集定义）。

**典型应用**  
高考中常以选择题、填空题出现，或作为函数定义域、值域、不等式解集的归宿。典型设问：  
- 已知 $A = \{x \mid x^2 - 3x + 2 = 0\}$，$B = \{x \mid 0 < x < 5, x \in \mathbb{N}\}$，判断 $A$ 与 $B$ 的关系。  
- 已知 $A = [1, 2]$，$B = \{x \mid x^2 - (a+1)x + a \leq 0\}$，若 $A \subseteq B$，求实数 $a$ 的取值范围。

## ⚠️ 高频易错点

- **忽视空集**：当子集含参数且定义在方程或不等式上时，未考虑空集满足子集条件，导致漏解。  
- **端点取值混乱**：在用区间或不等式表示集合时，端点能否取等取决于是否让子集定义成立，常因直接套用“大于等于”而错选。  
- **混淆子集与真子集**：将 $A \subseteq B$ 误写为 $A \subsetneqq B$ 或反之，忽略两集合可能相等的情况。  
- **数形结合失准**：在数轴上标出集合范围时，忽视空心点与实心点的区别，或未将不等式等价转化为标准形式即开始画图。
