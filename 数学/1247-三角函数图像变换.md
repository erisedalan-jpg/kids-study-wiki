# 三角函数图像变换

> **一句话**：指通过平移、伸缩等变换，由基本三角函数 $y = \sin x$（或 $y = \cos x$）得到形如 $y = A\sin(\omega x + \varphi) + k$ 的函数图像的过程与规律。
> **English**: The process and rules of obtaining the graph of a general sinusoidal function $y = A\sin(\omega x + \varphi) + k$ from $y = \sin x$ through amplitude scaling, period scaling, phase shifting, and vertical translation.

## 🎓 考点精讲

- **定义/原理**  
  三角函数图像变换的核心是研究参数 $A$（振幅）、$\omega$（角频率）、$\varphi$（初相）、$k$（纵向位移）对正弦或余弦曲线的影响。从 $y = \sin x$ 出发，变换顺序通常有两种：
  1. 先平移后伸缩：$y = \sin x \;\xrightarrow{\text{向左}|\varphi|\text{平移}}\; y = \sin(x + \varphi) \;\xrightarrow{\text{横坐标变为}\frac{1}{\omega}}\; y = \sin(\omega x + \varphi) \;\xrightarrow{\text{纵坐标变为}A\text{倍}}\; y = A\sin(\omega x + \varphi) \;\xrightarrow{\text{向上平移}k}\; y = A\sin(\omega x + \varphi) + k$。  
  2. 先伸缩后平移：$y = \sin x \xrightarrow{\text{横坐标变为}\frac{1}{\omega}} y = \sin(\omega x) \xrightarrow{\text{向左平移}\left|\frac{\varphi}{\omega}\right|} y = \sin(\omega x + \varphi)$，再依次进行纵向伸缩与平移。  
  注意：横向平移的符号遵循“左加右减”，且必须针对 $x$ 本身，即需要先提取 $\omega$ 因子，实际平移量为 $\left| \frac{\varphi}{\omega} \right|$。

- **关键方法**  
  1. **由图像求解析式**：根据最高点与最低点确定 $A = \frac{y_{\max} - y_{\min}}{2}$，$k = \frac{y_{\max} + y_{\min}}{2}$；利用相邻最值点或零点间的距离求周期 $T = \frac{2\pi}{|\omega|}$，得 $\omega$；再代入一个已知点（常选最值点或零点）解出 $\varphi$。  
  2. **描述变换过程**：严格区分变换顺序，写出每一步对应的函数解析式，避免直接将平移量加到 $\omega x$ 上。  
  3. **图像变换的“五点法”**：令 $\omega x + \varphi$ 分别取 $0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi$，求出对应的 $x$ 与 $y$，描点作图。

- **典型应用**  
  高考中常以选择题、填空题形式出现，设问角度包括：  
  - 给定 $f(x) = 2\sin\left(2x + \frac{\pi}{3}\right) + 1$，问其可由 $y = \sin 2x$ 经过怎样的平移得到（答案：向左平移 $\frac{\pi}{6}$ 个单位）。  
  - 由部分图像（如给出一个最高点和相邻零点）反求解析式，进而求单调区间或对称轴。  
  - 判断不同函数图像之间的变换关系，例如“要得到 $y = \cos\left(2x - \frac{\pi}{4}\right)$ 的图像，只需将 $y = \sin 2x$ 的图像如何变换？”

## ⚠️ 高频易错点

- **平移变换未提取 $\omega$**：误认为 $y = \sin(2x + \frac{\pi}{3})$ 是由 $y = \sin 2x$ 向左平移 $\frac{\pi}{3}$ 得到，正确平移量应为 $\frac{\pi}{6}$（需化为 $2\big(x + \frac{\pi}{6}\big)$）。
- **伸缩方向与倍数混淆**：$\omega > 1$ 时周期变小，图像横向压缩；$0 < \omega < 1$ 时横向拉伸，学生常将“压缩”与“拉伸”弄反。$A > 1$ 纵向伸长，$0 < A < 1$ 纵向缩短，同理易错。
- **变换顺序不同导致平移量不同**：先伸缩再平移时，平移量为 $\left|\frac{\varphi}{\omega}\right|$；先平移再伸缩时，平移量为 $|\varphi|$。考试中若混淆顺序，所求平移量会相差 $\omega$ 倍。
- **求 $\varphi$ 时不注意点的性质**：由图像求 $\varphi$ 时，代入选点不加判断，把零点代入未区分是上升零点还是下降零点，导致 $\varphi$ 值多解或错解。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 振幅 | Amplitude | 函数图像最高点与平衡位置的距离，决定纵向伸缩量 |
| 周期 | Period | 函数值重复出现的最小横轴长度，由 $\omega$ 决定 |
| 相位 | Phase | 表达式 $\omega x + \varphi$，控制波形的水平位置 |
| 平移变换 | Translation | 图像沿坐标轴整体移动，不改变形状 |
| 伸缩变换 | Scaling | 图像沿坐标轴方向拉伸或压缩，改变周期或振幅 |
