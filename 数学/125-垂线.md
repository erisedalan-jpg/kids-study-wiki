## 课本定义

两条直线相交成直角（90°），就说这两条直线**互相垂直**，其中一条叫做另一条的**垂线**，交点叫做**垂足**。从直线外一点到直线的垂线段的长度，叫做点到直线的**距离**。

---

## 📚 6-12 岁回溯版

### "完美的直角"

**垂线**就是两条线交叉时，角度恰好是90°（直角）的情况。

**怎么认识垂线？**
- 普通相交：两线交叉，角度可以是任意的（30°、60°、75°……）
- **垂直相交**：角度恰好是90°，这就是垂直！
- 用符号"⊥"表示：AB⊥CD 读作"AB垂直于CD"
- 交叉的点叫**垂足**

**生活中的垂线：**
| 场景 | 哪里垂直 |
|------|---------|
| 墙角 | 墙面与地面垂直 |
| 字母"T" | 竖线与横线垂直 |
| 十字架 | 竖横两条杆垂直 |
| 正方形 | 每条边与相邻边垂直 |
| 直立的旗杆 | 旗杆与地面垂直 |

**点到直线的距离：**
从一个点向一条直线画线，可以画无数条，但**最短的那条**就是垂线段，它的长度就是"点到直线的距离"。

**口诀：** "直角相交叫垂直，最短连线是垂足"

**动手试试：** 用三角板的直角在纸上画一条垂线，测量一下，两条线夹的角是不是90°？

## 🎓 12+ 进阶版

### 严格定义

**垂线**：若两直线 l₁ ⊥ l₂ 相交，且交角为 90°，则称 l₁ 与 l₂ **互相垂直**，记作 l₁ ⊥ l₂，交点称为**垂足**（foot of perpendicular）。

### 核心定理

**定理1（垂线的存在唯一性）：** 过直线外一点，有且只有一条直线与已知直线垂直。

**定理2（最短距离定理）：** 直线外一点到直线的所有连线中，垂线段最短。

**点到直线的距离**：直线外一点 P 到直线 l 的垂线段的长度，记为 d(P, l)。

### 性质汇总

| 性质 | 内容 |
|------|------|
| 垂直的角 | 四个角均为90° |
| 唯一性 | 过一点，对一直线只有唯一垂线 |
| 最短路径 | 垂线段是点到直线最短连线 |
| 对称性 | 直线是其垂线上各点的对称轴 |

### 典型例题

**例题1：** 已知点P在直线l外，PA⊥l，垂足为A，PA=3cm；另有一点B在l上，PB=5cm，求AB。

解：在直角三角形PAB中，∠PAB=90°（PA⊥l）
AB² = PB² - PA²（勾股定理）
AB² = 25 - 9 = 16
AB = 4cm

**例题2：** 如图，已知∠AOB=90°，OC平分∠AOB，OD是OA的延长线，求∠COD。

解：OC平分∠AOB，故∠COB = 45°
∠COD = 180° - ∠COB = 180° - 45° = 135°（邻补角）

### 易错点

1. **垂直 vs 垂线**："垂直"是两线关系，"垂线"是一条线（另一条线的垂线）
2. **垂足不一定在线段内**：垂足在直线上，可能在端点外（此时作垂线段需延长）
3. **距离是垂线段的长**，不是随意连线的长

### 知识联系

```
垂线
├── 特殊相交线（交角90°）
├── 点到直线距离 → 几何面积计算
├── 垂直平分线 → 线段中点性质
└── 勾股定理 → 直角三角形
```
