# 三角函数的诱导公式

> **一句话**：诱导公式是利用三角函数的周期性与终边对称性，将任意角的三角函数转化为锐角三角函数的一组恒等式，核心口诀为“奇变偶不变，符号看象限”。
> **English**: A set of identities that transform trigonometric functions of arbitrary angles into those of acute angles using periodicity and symmetry of terminal sides, summarized as “odd multiples of \(\frac{\pi}{2}\) change the function name, even leave it unchanged; the sign is determined by the original function in the quadrant where the angle would lie if treated as acute.”

## 🎓 考点精讲

**定义与本质**  
诱导公式基于正弦、余弦、正切函数的周期性（\(2k\pi+\alpha\)）、奇偶性（\(-\alpha\)）以及坐标系中不同终边对称关系（\(\pi\pm\alpha,\ \frac{\pi}{2}\pm\alpha\) 等）。常用公式包括：
\[
\begin{aligned}
\sin(2k\pi+\alpha)&=\sin\alpha,\quad \cos(2k\pi+\alpha)=\cos\alpha,\quad \tan(2k\pi+\alpha)=\tan\alpha \\[4pt]
\sin(-\alpha)&=-\sin\alpha,\quad \cos(-\alpha)=\cos\alpha,\quad \tan(-\alpha)=-\tan\alpha \\[4pt]
\sin(\pi-\alpha)&=\sin\alpha,\quad \cos(\pi-\alpha)=-\cos\alpha,\quad \tan(\pi-\alpha)=-\tan\alpha \\[4pt]
\sin(\pi+\alpha)&=-\sin\alpha,\quad \cos(\pi+\alpha)=-\cos\alpha,\quad \tan(\pi+\alpha)=\tan\alpha \\[4pt]
\sin\!\left(\frac{\pi}{2}-\alpha\right)&=\cos\alpha,\quad \cos\!\left(\frac{\pi}{2}-\alpha\right)=\sin\alpha \\[4pt]
\sin\!\left(\frac{\pi}{2}+\alpha\right)&=\cos\alpha,\quad \cos\!\left(\frac{\pi}{2}+\alpha\right)=-\sin\alpha
\end{aligned}
\]
（\(k\in\mathbb{Z}\)，公式可经周期性推广到任意整数 \(k\)。）

**关键方法：“奇变偶不变，符号看象限”**
1. **奇变偶不变**：将角写成 \(k\cdot\frac{\pi}{2}\pm\alpha\)（\(k\in\mathbb{Z}\)），若 \(k\) 为**奇数**，则正弦变余弦、余弦变正弦、正切变余切（函数名改变）；若 \(k\) 为**偶数**，函数名不变。  
2. **符号看象限**：假设 \(\alpha\) 为**锐角**，判断原角 \(k\cdot\frac{\pi}{2}\pm\alpha\) 所在的象限，取原三角函数在该象限的符​​号，即为化简后的符号。  
   例：\(\cos\!\left(\frac{3\pi}{2}+\alpha\right)\)，\(k=3\)（奇）→ 函数名变为 \(\sin\)，\(\frac{3\pi}{2}+\alpha\) 在第四象限，余弦为正 → 结果为 \(+\sin\alpha\)。

**高考典型考查**
- **求值与化简**：给非特殊角（如 \(\sin(-930^\circ)\)）化为特殊角，或含参化简如 \(\frac{\cos(\pi+\theta)\sin(-\theta)}{\tan(-\theta-\pi)}\)。  
- **条件求值**：已知 \(\sin(\pi-\theta)=a\)，求 \(\cos(\frac{3\pi}{2}+\theta)\) 等，常结合同角关系。  
- **证明恒等式**：利用诱导公式统一角或函数名，再变形证明。  

解题一般步骤：负角化正角 → 大角化小角（\(\pi\) 整数倍消去） → 再用 \(\frac{\pi}{2}\) 变名 → 辐角化为锐角求值。

## ⚠️ 高频易错点

- **符号判断错误**：使用“符号看象限”时，忘记将 \(\alpha\) 视为锐角，或错误判断原函数在所在象限的正负（尤其涉及 \(\cos,\tan\) 时）。例如，从 \(\sin(\pi+\alpha)\) 误得 \(+\sin\alpha\)。  
- **奇偶变名混淆**：对 \(k\) 的奇偶判断不准，或对非 \(\frac{\pi}{2}\) 倍数的角滥用变名。如误把 \(\sin(\pi-\alpha)\) 中的 \(\pi\) 视作 \(90^\circ\) 的 2 倍，实际 \(k=2\) 偶，函数名不变。  
- **忽略角的范围**：已知一个三角函数值，利用诱导公式求另一函数值时，未注意所给角所在的象限导致符号多解或多值。例如由 \(\sin(\pi+\theta)=\frac12\) 求 \(\cos\theta\)，需结合 \(\theta\) 范围定号。  
- **公式记忆混淆**：错记如 \(\cos(-\alpha)=-\cos\alpha\)（正确为正），或混淆 \(\frac{\pi}{2}\pm\alpha\) 与 \(\pi\pm\alpha\) 的正弦余弦互换规则。
