## 课本定义

**定理**：经过**推理证明**的真命题，可以作为继续推理的依据。与定理相关的概念：**公理**（不需要证明就承认为正确的命题，作为推理出发点）；**推论**（直接从定理得出的命题）。

---

## 📚 6-12 岁回溯版

### "数学里的"已证明的真理""

**数学知识的层次：**

```
公理（不证自明的基础）
   ↓ 证明
定理（经过证明的真命题）
   ↓ 推导
推论（从定理直接得出的结论）
```

**公理**：大家都同意、不需要证明的基础规则。
例："两点之间线段最短"，"过两点只能画一条直线"

**定理**：经过严格证明的结论，可以当作工具用。
例："对顶角相等"，"平行线的内错角相等"，"三角形内角和=180°"

**推论**：从定理里直接得出的小结论，相对容易。
例：从"三角形内角和=180°"推出"直角三角形两个锐角互余"

**生活类比：**
- 公理 ≈ 游戏规则（大家都同意的）
- 定理 ≈ 经过裁判判定的规则（经过验证的）
- 推论 ≈ 从规则延伸出的规定（直接推出来的）

**数学里熟悉的定理：**
| 定理 | 内容 |
|------|------|
| 对顶角定理 | 对顶角相等 |
| 平行线内错角定理 | 两平行线内错角相等 |
| 三角形内角和定理 | 三角形三内角和=180° |
| 勾股定理 | 直角三角形 a²+b²=c² |

## 🎓 12+ 进阶版

### 严格定义

**公理**（axiom）：数学体系中无需证明、作为推理出发点的基本命题。公理是整个数学大厦的"地基"。

**定理**（theorem）：从公理出发，经过严格逻辑推理（证明）得到的真命题。定理证明后可以反复使用。

**推论**（corollary）：从某定理直接得出的命题（证明简单，通常只需一两步）。

**引理**（lemma）：为证明重要定理而事先证明的辅助命题。

### 定理的结构

定理 = 条件 + 结论（"若……则……"形式）

**例：** 定理"平行线的同位角相等"
- 条件：两直线平行
- 结论：同位角相等

**逆定理**：将条件和结论互换，不一定成立（逆命题不保证真）。

**例：** "同位角相等"不能直接推出"两直线平行"（平行线判定是另一个定理）

### 公理体系（欧几里得几何）

欧几里得在《几何原本》中建立了以5条公设（公理）为基础的平面几何体系：
1. 过两点可作且只作一条直线
2. 线段可以无限延长
3. 以任意点为圆心、任意长为半径可作圆
4. 所有直角相等
5. 平行公设（过一点有且只有一条线与已知线平行）

从这5条公理出发，推导出整个欧氏几何体系（数百个定理）。

### 七年级常用定理

| 定理 | 内容 | 类别 |
|------|------|------|
| 对顶角定理 | 对顶角相等 | 相交线 |
| 平行线性质定理1 | 平行线同位角相等 | 平行线 |
| 平行线性质定理2 | 平行线内错角相等 | 平行线 |
| 平行线性质定理3 | 平行线同旁内角互补 | 平行线 |
| 平行线判定定理1 | 同位角相等→平行 | 平行线 |
| 垂线定理 | 过一点有且只有一条垂线 | 垂直 |

### 典型例题

**例题1：** 指出下列哪些是公理，哪些是定理：

(1) 两点确定一条直线
(2) 对顶角相等
(3) 过一点有无数条直线
(4) 三角形内角和等于180°

解：
(1) 公理（直接承认，不证明）
(2) 定理（可以从对顶角定义+邻补角推导证明）
(3) 公理（线的性质）
(4) 定理（可以通过平行线性质证明）

**例题2：** 定理"平行线的内错角相等"的推论是什么？

解：推论：若两直线平行，则同旁内角互补。
（因为同旁内角 = 180° - 内错角 = 180° - 相等角，所以互补）

### 定理与命题的关系

- 所有定理都是命题（且是真命题）
- 并非所有真命题都是定理（公理也是真命题，但不是定理）
- 假命题不可能成为定理

```
真命题
├── 公理（无需证明）
└── 定理（经过证明）
    └── 推论（从定理直接得出）
```
