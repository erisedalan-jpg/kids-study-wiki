# 五点法作图

> **一句话**：五点法作图是通过在一个周期内取五个关键点（通常使相位等于0、π/2、π、3π/2、2π）来精确绘制正弦型或余弦型函数图像的方法。
> **English**: The five-point plotting method is a technique for sketching the graphs of sine or cosine functions by determining five key points whose phase angles take the values 0, π/2, π, 3π/2, and 2π within one period.

## 🎓 考点精讲

**定义与原理**  
对于形如 $y = A\sin(\omega x + \varphi) + k$ 或 $y = A\cos(\omega x + \varphi) + k$ 的函数（$A \neq 0, \omega > 0$），五点法作图的原理是：令相位 $\theta = \omega x + \varphi$ 依次取 $0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi$，这五个值刚好将一个周期 $T = \frac{2\pi}{\omega}$ 分为四等份，对应图像在一个周期内的起点、最高点（或最低点）、中点、最低点（或最高点）和终点。通过这五个点的坐标即可勾勒出函数的简图，进而分析其性质。

**关键方法**  
标准步骤：
1. 确定周期 $T = \frac{2\pi}{|\omega|}$，相邻两点横坐标间隔为 $\frac{T}{4}$。  
2. 求第一个点的横坐标：令 $\omega x + \varphi = 0$，得 $x_1 = -\frac{\varphi}{\omega}$。  
3. 依次加 $\frac{T}{4}$ 得到其余四点横坐标：$x_2 = x_1 + \frac{T}{4}, x_3 = x_1 + \frac{T}{2}, x_4 = x_1 + \frac{3T}{4}, x_5 = x_1 + T$。  
4. 计算纵坐标：对于正弦型，五个纵坐标依次为 $k, A+k, k, -A+k, k$；对于余弦型，纵坐标依次为 $A+k, k, -A+k, k, A+k$（均基于 $A > 0$ 的情形，若 $A < 0$ 则最值点相反）。  
5. 在坐标系中描出五点，用光滑曲线连接，并向左右延伸以体现周期性。

**典型应用（高考视角）**  
高考中常以解答题或选择题形式出现，主要设问类型：
- 给定解析式，用五点法画出函数在一个周期内的简图，并求单调区间、对称轴方程、对称中心等。  
- 由函数图像的部分信息（如给出一个周期内的五个特殊点）反求解析式。  
例如：已知函数 $f(x) = 2\sin(2x + \frac{\pi}{3})$，用五点法画出 $x \in [0, \pi]$ 上的图像，并写出单调递增区间。解题时需先求出 $T=\pi$，五点横坐标由 $2x+\frac{\pi}{3} = 0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi$ 解得 $x = -\frac{\pi}{6}, \frac{\pi}{12}, \frac{\pi}{3}, \frac{7\pi}{12}, \frac{5\pi}{6}$，再结合振幅 $2$ 和纵向平移 $0$ 得到纵坐标 $0,2,0,-2,0$，描点连线即可。

## ⚠️ 高频易错点

- 计算五点横坐标时未将 $\omega x + \varphi$ 整体等于目标值求解 $x$，而直接令 $x$ 等于 $0, \frac{\pi}{2}$ 等，导致相位错误，尤其当 $\omega \neq 1$ 或 $\varphi \neq 0$ 时。  
- 混淆正弦与余弦函数的五点对应纵坐标：当函数为 $y = A\cos(\omega x + \varphi) + k$ 时，五点对应的纵坐标不再是 $k, A+k, k, -A+k, k$，须根据余弦函数值规律重新确定，否则图像左右颠倒或上下错位。  
- 忽略纵向平移 $k$，或误将振幅 $A$ 与最大值混淆（最大值应为 $k + |A|$），导致图像整体上下偏移或最值点坐标错误。  
- 在由图像反求解析式时，五点选取不规范：常见错误是将任意五个特殊点直接套用标准五点顺序，但实际图像给出的可能是零点、最值点混合，需结合函数类型和位移方向准确判断对应相位。
