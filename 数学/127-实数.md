## 课本定义

有理数和无理数统称为**实数**。实数与数轴上的点一一对应：每个实数对应数轴上唯一一个点，数轴上每个点对应唯一一个实数。

---

## 📚 6-12 岁回溯版

### "数家族的全体成员"

我们学过很多种数，它们合在一起，就叫**实数**：

```
实数
├── 有理数
│   ├── 整数：...,-2,-1,0,1,2,3,...
│   └── 分数：1/2, 3/4, -1/3, 0.5,...
└── 无理数：√2, √3, π, ∛5,...
```

**实数的"家谱"：**
- 自然数 ⊂ 整数 ⊂ 有理数 ⊂ 实数
- 无理数 ⊂ 实数
- 有理数 和 无理数 加在一起 = 实数（没有重叠！）

**数轴和实数：**
数轴上**每一个点**对应一个实数（包括无理数的位置！）
- 整数对应数轴上的整数点
- 分数对应整数点之间
- √2 ≈ 1.414，也在数轴上（在1和2之间）

**这意味着：** 数轴上的点是连续的，没有空隙——实数"填满"了数轴！

**实数的运算法则和有理数一样：**
加减乘除、绝对值、大小比较，对实数都适用。

## 🎓 12+ 进阶版

### 严格定义

**实数**（real number）：有理数与无理数的并集，构成**实数集** ℝ。

**数系的包含关系：**
$$\mathbb{N} \subset \mathbb{Z} \subset \mathbb{Q} \subset \mathbb{R}$$
- ℕ（自然数）⊂ ℤ（整数）⊂ ℚ（有理数）⊂ ℝ（实数）

### 实数的分类

**按有无理性：**
| 类型 | 定义 | 例子 |
|------|------|------|
| 有理数 | 可表示为 p/q（q≠0） | 1, -2, 1/3, 0.5 |
| 无理数 | 无限不循环小数 | √2, π, ∛3 |

**按正负：**
| 类型 | 范围 |
|------|------|
| 正实数 | > 0 |
| 零 | = 0 |
| 负实数 | < 0 |

### 实数的运算性质

实数满足以下运算律（a, b, c ∈ ℝ）：

| 运算律 | 加法 | 乘法 |
|--------|------|------|
| 交换律 | a+b = b+a | ab = ba |
| 结合律 | (a+b)+c = a+(b+c) | (ab)c = a(bc) |
| 分配律 | a(b+c) = ab+ac | |
| 零元素 | a+0 = a | a·1 = a |
| 逆元素 | a+(-a) = 0 | a·(1/a)=1（a≠0） |

### 实数与数轴的对应

**完备性**：实数与数轴上的点**一一对应**（实数的完备性），即：
- 每个实数对应数轴上唯一的点
- 数轴上每个点对应唯一的实数
- 数轴是"连续"的，没有空隙（区别于有理数的"稠密但不完备"）

### 实数的大小比较

- a - b > 0 ⟺ a > b
- a - b = 0 ⟺ a = b
- a - b < 0 ⟺ a < b
- 数轴上：右边的数大于左边的数

### 典型例题

**例题1：** 将下列各数分类：-3, √9, 0.5, √2, π, -1/3, 0

解：
- 有理数：-3（整数），√9=3（整数），0.5（有限小数），-1/3（分数），0（整数）
- 无理数：√2（无限不循环），π（无限不循环）
- 整数（有理数子集）：-3, 3, 0

**例题2：** 比较 √3 + 1 与 √5 的大小。

解：(√3+1)² = 3 + 2√3 + 1 = 4 + 2√3
(√5)² = 5

需比较 4 + 2√3 与 5：即比较 2√3 与 1，即 √3 与 0.5。
显然 √3 > 1 > 0.5，故 2√3 > 1，故 4+2√3 > 5。
因为两数均正，(√3+1)² > (√5)²，故 √3+1 > √5。

### 易错点

1. **√9 = 3 是有理数**（整数），不是无理数
2. **实数 = 有理数 + 无理数**，两者不重叠（一个数不能既是有理数又是无理数）
3. **虚数不是实数**：√(-1) = i 是虚数，不属于实数范围

### 知识联系

```
实数
├── 包含 → 有理数（含整数、分数）
├── 包含 → 无理数（√2, π, ∛3...）
├── 对应 → 数轴（完备性）
├── 扩展 → 复数（加上虚数单位 i）
└── 运算 → 与有理数运算律相同
```
