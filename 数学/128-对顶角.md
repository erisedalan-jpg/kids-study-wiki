## 课本定义

两条直线相交所形成的四个角中，有公共顶点且两边互为反向延长线的两个角叫做**对顶角**。对顶角相等。

---

## 📚 6-12 岁回溯版

### "对面看，角相等"

两条直线交叉，像画了一个"×"，形成4个角。其中**对面**的两个角（不相邻的）叫**对顶角**。

**最重要的一件事：对顶角永远相等！**

**怎么找对顶角？**
- 两条线交叉，在交叉点形成4个角，给它们编号1、2、3、4
- 1和3是对顶角（对面），2和4是对顶角（对面）
- ∠1 = ∠3，∠2 = ∠4

```
        ∠2
   ∠1  ×  ∠3
        ∠4
```

**生活例子：**
- 剪刀交叉：两个刀刃形成的角，对面的角总相等
- 铁路道岔：两条轨道交叉，对面角相等
- 烤箱架的X形支撑：对角相等，结构稳定

**为什么相等呢？**
因为∠1和∠2加起来是180°（它们在一条直线上），∠2和∠3加起来也是180°，所以∠1=∠3！

## 🎓 12+ 进阶版

### 严格定义

两直线相交，形成的4个角中，顶点相同、且各自两边互为对方两边的反向延长线的一对角，称为**对顶角**（vertical angles）。

### 定理：对顶角相等

**证明：**

设两直线 AB 与 CD 交于点 O，形成 ∠AOC、∠COB、∠BOD、∠DOA。

∠AOC 与 ∠BOD 是对顶角，证 ∠AOC = ∠BOD：

- ∠AOC + ∠COB = 180°（AOB 为直线，邻补角）
- ∠BOD + ∠COB = 180°（COD 为直线，邻补角）
- ∴ ∠AOC = ∠BOD（等量代换）∎

### 典型例题

**例题1：** 如图，直线 l₁ 与 l₂ 相交于点 O，∠1 = 3x + 10°，∠3 = 5x - 20°（∠1 与 ∠3 为对顶角），求 x 及各角度数。

解：由对顶角相等，∠1 = ∠3
3x + 10° = 5x - 20°
30° = 2x
x = 15°
∠1 = ∠3 = 55°，∠2 = ∠4 = 180° - 55° = 125°

**例题2：** 三条直线交于一点，共形成几对对顶角？

解：3条线交于一点形成6个角，每条线产生一对对顶角，共3对对顶角（∠1=∠4，∠2=∠5，∠3=∠6）。

### 易错点

1. **对顶角必须是两直线相交形成的**，不能是三角形内角等其他情形
2. **对顶角相等是定理，不是定义**——定义说的是"位置关系"，定理说的是"数量关系"
3. 两条射线的"对顶"没有意义，必须是完整直线

### 知识联系

```
对顶角
├── 来自 → 相交线
├── 性质 → 对顶角相等（定理）
├── 搭档 → 邻补角（和为180°）

```
