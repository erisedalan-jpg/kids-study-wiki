# 圆与圆的位置关系

> **一句话**：圆与圆的位置关系由两圆圆心距与两圆半径之和、半径之差的绝对值之间的数值大小关系决定，分为外离、外切、相交、内切、内含五种情况。
> **English**: The positional relationship between two circles is determined by the comparison of the distance between their centers with the sum and the absolute difference of their radii, resulting in five cases: separate, externally tangent, intersecting, internally tangent, and one circle inside the other.

## 🎓 考点精讲

**定义与判定式**  
设两圆 $C_1$ 圆心 $O_1$，半径 $r_1$；$C_2$ 圆心 $O_2$，半径 $r_2$，且 $r_1 \ge r_2 > 0$，圆心距 $d = |O_1O_2|$。位置关系的充要条件为：

- **外离** $\Leftrightarrow d > r_1 + r_2$  
- **外切** $\Leftrightarrow d = r_1 + r_2$  
- **相交** $\Leftrightarrow |r_1 - r_2| < d < r_1 + r_2$  
- **内切** $\Leftrightarrow d = |r_1 - r_2|$（$r_1 \neq r_2$）  
- **内含** $\Leftrightarrow d < |r_1 - r_2|$  

特别地，$d = 0$ 且 $r_1 = r_2$ 时为同心圆，属于内含特例；$d = 0$ 且 $r_1 \neq r_2$ 为同心内含。

**关键方法**  
1. **代数法**：联立两圆方程，消去二次项得关于 $x,y$ 的一次方程（公共弦方程），代入判别式 $\Delta$。$\Delta > 0 \Leftrightarrow$ 相交；$\Delta = 0 \Leftrightarrow$ 相切（外切或内切）；$\Delta < 0 \Leftrightarrow$ 外离或内含。需结合半径和与差进一步区分相切类型。  
2. **几何法**（优先使用）：直接计算圆心距 $d$，与 $r_1 + r_2$、$|r_1 - r_2|$ 比较大小，一步到位。

**典型应用**  
高考主要考查：已知两圆方程判断位置关系、由位置关系求参数范围（如半径、圆心坐标中的参数）、与公共弦相关的问题（弦长、弦所在直线方程）、以及圆与圆位置关系在解析几何综合题中作为条件出现（如动圆与两定圆相切时轨迹方程的求解）。  
例1：已知圆 $C_1: x^2 + y^2 = 1$ 与 $C_2: (x-2)^2 + (y-2)^2 = r^2$ 外切，求 $r$。解：圆心距 $d = 2\sqrt{2}$，外切条件 $d = r_1 + r_2$，即 $2\sqrt{2} = 1 + r$，得 $r = 2\sqrt{2} - 1$。  
例2：两圆 $x^2 + y^2 = 4$ 与 $(x-3)^2 + (y-4)^2 = 9$ 相交，求公共弦长。先作差得公共弦直线方程 $3x + 4y - 2 = 0$，再用其中一个圆的圆心到此直线的距离及半径用垂径定理求弦长。

## ⚠️ 高频易错点

- **混淆内切与内含的判断式**：内切是 $d = |r_1 - r_2|$（且半径不等），内含是 $d < |r_1 - r_2|$；常误把内切记为 $d = r_1 - r_2$（未加绝对值），当 $r_1 < r_2$ 时出错。
- **用代数法直接判别外离与内含时出错**：联立方程后 $\Delta < 0$ 只说明无交点，但究竟是外离还是内含，必须再比较 $d$ 与 $|r_1 - r_2|$ 或 $r_1+r_2$。仅凭 $\Delta$ 无法区分。
- **忽视两圆半径大小的前提**：当两圆半径不作 $r_1 \ge r_2$ 约定时，$|r_1 - r_2|$ 必须带绝对值，直接使用 $r_1 - r_2$ 可能得到负值导致误判。
- **公共弦方程误用**：两圆相交时，作差一次消去二次项直接得到公共弦所在直线方程，该直线必为两圆公共弦所在直线，但当两圆相切时，该方程为过切点的公切线方程；不能见到一次方程就认为一定是公共弦。
