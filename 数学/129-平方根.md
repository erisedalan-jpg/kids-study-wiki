## 课本定义

如果 x² = a（a ≥ 0），那么 x 叫做 a 的**平方根**（也称**二次方根**）。正数 a 的平方根有两个，互为相反数：√a 和 -√a；零的平方根是零；负数没有平方根。其中 √a（a ≥ 0）叫做 a 的**算术平方根**。

---

## 📚 6-12 岁回溯版

### "找那个'被平方'之前的数"

**平方**是把一个数乘以自身：3² = 9，4² = 16。

**平方根**就是反过来问：**什么数平方后等于9？** 答案是3（或-3）。

**规律：**
- 9 的平方根是 ±3（因为3²=9，(-3)²=9）
- 16 的平方根是 ±4
- 25 的平方根是 ±5
- 0 的平方根是 0
- **负数没有平方根**（任何实数的平方都 ≥ 0）

**常用平方根表：**
| a | a 的平方根 | 算术平方根 √a |
|---|---------|------------|
| 0 | 0 | 0 |
| 1 | ±1 | 1 |
| 4 | ±2 | 2 |
| 9 | ±3 | 3 |
| 16 | ±4 | 4 |
| 25 | ±5 | 5 |
| 100 | ±10 | 10 |

**符号说明：**
- √9 = 3（算术平方根，只取正值）
- ±√9 = ±3（平方根，两个值）

**开平方**：求平方根的运算叫**开平方**，与平方互为逆运算。

## 🎓 12+ 进阶版

### 严格定义

设 a ≥ 0，若 x² = a，则 x 称为 a 的**平方根**（square root）。

- **算术平方根**：a 的非负平方根，记作 √a（根号下 a），√a ≥ 0
- **负平方根**：-√a
- **完整平方根**：±√a

**定义域**：
- a > 0：两个平方根 ±√a，均为无理数（若 a 不是完全平方数）或有理数
- a = 0：唯一平方根为 0
- a < 0：在实数范围内**无平方根**（复数范围内有虚数解）

### 完全平方数

整数 n 满足 n = k²（k 为整数），称 n 为**完全平方数**。
例：0, 1, 4, 9, 16, 25, 36, 49, 64, 81, 100……

对完全平方数，√n 为整数（有理数）；否则 √n 为无理数。

### 性质公式

| 性质 | 公式 | 条件 |
|------|------|------|
| 算术平方根非负 | √a ≥ 0 | a ≥ 0 |
| 平方的平方根 | (√a)² = a | a ≥ 0 |
| 平方根的平方 | √(a²) = \|a\| | 任意 a |
| 乘法法则 | √a · √b = √(ab) | a,b ≥ 0 |
| 除法法则 | √a / √b = √(a/b) | a ≥ 0, b > 0 |

注意：√(a²) = |a|，不是 a！若 a < 0，√(a²) = -a > 0。

### 典型例题

**例题1：** 求 49 的平方根和算术平方根。

解：因为 7² = 49，(-7)² = 49
49 的平方根为 ±7
49 的算术平方根为 7（正值）

**例题2：** 化简 √(x²-2x+1)（x < 1）

解：x²-2x+1 = (x-1)²
√(x-1)² = |x-1|
因为 x < 1，故 x-1 < 0，|x-1| = -(x-1) = 1-x

答：√(x²-2x+1) = 1-x（当 x < 1 时）

### 易错点

1. **√a 只取正值**：√9 = 3，不是 ±3（±√9 才是 ±3）
2. **√(a²) = |a|，不是 a**：√((-3)²) = √9 = 3 = |-3|
3. **负数无实数平方根**：√(-4) 在实数内无意义

### 知识联系

```
平方根
├── 逆运算 → 平方（a²）
├── 特殊情况 → 完全平方数（整数根）
├── 一般情况 → 无理数
├── 类比 → 立方根（三次方根）
└── 扩展 → 复数（负数的平方根 = 虚数）
```
