# 导数的四则运算

> **一句话**：导数的四则运算是求两个可导函数经过加、减、乘、除组合后所得函数导数的运算法则，核心公式为 (u±v)′=u′±v′，(uv)′=u′v+uv′，(u/v)′=(u′v−uv′)/v² (v≠0)。
> **English**: The four arithmetic operations of derivatives are the rules for differentiating the sum, difference, product, and quotient of two differentiable functions, with core formulas: (u±v)′=u′±v′, (uv)′=u′v+uv′, and (u/v)′=(u′v−uv′)/v² (v≠0).

## 🎓 考点精讲

**定义与原理**  
若函数 u(x), v(x) 在点 x 处可导，则它们的和、差、积、商（分母不为零）也在该点可导，且导数满足以下法则：

- 加法法则：`[u(x) + v(x)]′ = u′(x) + v′(x)`  
- 减法法则：`[u(x) − v(x)]′ = u′(x) − v′(x)`  
- 乘法法则：`[u(x)v(x)]′ = u′(x)v(x) + u(x)v′(x)`  
- 除法法则：`[u(x)/v(x)]′ = [u′(x)v(x) − u(x)v′(x)] / [v(x)]²`，（v(x) ≠ 0）

这些法则可以推广到有限个函数的和差积，以及常数因子情形：`[cu(x)]′ = c·u′(x)`（常数 c 可视为乘法法则特例）。

**关键方法**  
解决高考导数计算题的标准步骤：  
1. 识别函数结构，确定由哪些基本初等函数（幂、指、对、三角等）通过四则运算复合而成。  
2. 分别求出各部分的导数，严格套用四则运算法则，不跳步。  
3. 整理结果，通常需要通分、因式分解、合并同类项以达到最简形式。  
4. 若含有复合函数，先使用链式法则求内层导数，再结合四则运算。  

**典型应用**  
高考中，该考点常在以下设问中出现：  
- **显式函数求导**：如 `f(x) = x²·ln x`，需用乘法法则得 `f′(x) = 2x·ln x + x²·(1/x) = 2x ln x + x`。  
- **切线问题**：求曲线 `y = (sin x)/x` 在 `x=π` 处的切线斜率，先用除法法则求导，再代入横坐标。  
- **含参数导数**：已知 `f(x) = (ax² + b)/e^x`，求 `f′(x)` 并讨论单调性，要求准确运用除法和乘法法则。  
- **导数的工具性**：在求极值、最值、证明不等式时，第一步往往是利用四则运算法则求导函数。

## ⚠️ 高频易错点

- **乘法法则漏项**：误记作 `(uv)′ = u′v′`，或只写 `u′v` 而遗漏 `uv′`。此错误在乘积项较多时尤为常见。  
- **除法法则分子记反**：将分子错记为 `u′v + uv′` 或 `u′v′ − uv`，正确形式是“上导下不导减上不导下导”，顺序必须严格 `u′v − uv′`，且分母恒平方。  
- **忘记分母不为零条件**：使用除法法则时忽略 `v(x) ≠ 0` 的定义域限制，导致在使分母为零的点错误求导。  
- **复合函数与四则运算混淆**：例如 `sin(x²+1)` 不能用乘法法则拆成 `sin x²` 和 `sin 1`，必须先用链式法则；而 `x²·sin x` 才是乘法法则。分辨函数“最后一步运算”是关键。
