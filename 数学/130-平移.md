## 课本定义

在平面内，将一个图形沿**某个方向**移动**一定的距离**，这种图形变换叫做**平移**。平移后，图形的**形状和大小不变**，只是位置发生了变化。平移前后对应点的连线段互相平行且相等。

---

## 📚 6-12 岁回溯版

### "整体搬家，形状不变"

**平移**就是把一个图形"整体搬过去"：
- 方向固定（比如向右、向上斜……）
- 距离固定（搬多远）
- 形状和大小**完全不变**（只是换了个位置）

**生活中的平移：**
| 例子 | 说明 |
|------|------|
| 推拉窗 | 向一侧移动，窗的形状不变 |
| 火车沿直线行驶 | 车厢整体平移 |
| 打印机走纸 | 纸张整体平移 |
| 棋子移动 | 棋子在棋盘上平移 |

**平移的三要素：**
1. **方向**：朝哪个方向移？（上/下/左/右/斜方向）
2. **距离**：移多远？
3. **图形本身**：什么图形在移动

**平移的性质：**
- 平移后，图形的**形状**和**大小**不变
- 平移后，图形上**每个点**都移动了同样的方向和距离
- 对应点的连线**平行**且**相等**

**和旋转、翻转的区别：**
| 变换 | 改变 | 不变 |
|------|------|------|
| 平移 | 位置 | 形状、大小、方向 |
| 旋转 | 位置、方向 | 形状、大小 |
| 翻转（轴对称）| 位置、方向 | 形状、大小 |

## 🎓 12+ 进阶版

### 严格定义

**平移**（translation）：将平面内的图形沿某一方向移动一定距离的变换。

设平移向量为 $\vec{v} = (a, b)$，则图形上每个点 P(x, y) 平移后变为 P'(x+a, y+b)。

**注：** 七年级不要求向量，但概念上可以理解为"向右 a，向上 b"。

### 平移的基本性质

1. **全等性**：平移后的图形与原图形全等（形状、大小完全相同）
2. **方向性**：平移方向和距离对图形所有点相同
3. **对应线平行**：对应点连线互相平行且相等（或共线）
4. **平行线保持**：平移不改变直线的方向（平行线平移后仍平行）

### 坐标中的平移

在坐标系中，将点 P(x, y) 向右平移 a 个单位，向上平移 b 个单位，得到 P'(x+a, y+b)。

| 平移方向 | 变化 | 坐标变化 |
|---------|------|---------|
| 向右 a | x 增大 a | (x, y) → (x+a, y) |
| 向左 a | x 减小 a | (x, y) → (x-a, y) |
| 向上 b | y 增大 b | (x, y) → (x, y+b) |
| 向下 b | y 减小 b | (x, y) → (x, y-b) |

### 典型例题

**例题1：** 将△ABC（A(1,2), B(3,2), C(2,4)）向右平移3个单位，求平移后各顶点坐标。

解：向右平移3，x 坐标各加3，y 坐标不变。
A' = (1+3, 2) = (4, 2)
B' = (3+3, 2) = (6, 2)
C' = (2+3, 4) = (5, 4)

**例题2：** 将图形向右平移2、向下平移3，点 P(-1, 4) 平移到哪里？

解：x 增加 2，y 减少 3。
P' = (-1+2, 4-3) = (1, 1)

**例题3：** △A'B'C' 是 △ABC 平移后的图形，已知 A(0, 0)，A'(3, -2)，B(2, 5)，求 B'。

解：平移向量 = A' - A = (3-0, -2-0) = (3, -2)
B' = B + 平移向量 = (2+3, 5-2) = (5, 3)

### 平移与对称、旋转的比较

| 变换 | 定义 | 保持不变 | 改变 |
|------|------|---------|------|
| 平移 | 沿方向移动距离 | 形状、大小、方向 | 位置 |
| 轴对称 | 关于直线翻折 | 形状、大小 | 位置、方向 |
| 旋转 | 绕点旋转角度 | 形状、大小 | 位置、方向 |

### 易错点

1. **平移方向和距离要对所有点一致**，不能有的点移动多，有的移动少
2. **平移不改变图形的朝向**（区别于旋转和轴对称）
3. **坐标轴上的平移**：向右是加，向左是减；向上是加，向下是减（对应 x, y 轴正方向）
