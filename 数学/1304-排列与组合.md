# 排列与组合

> **一句话**：排列与组合研究从给定元素集合中选取部分元素形成序列或子集的计数问题，其核心分界在于是否考虑元素的顺序。
> **English**: Permutations and combinations are counting methods for forming sequences or subsets from a given set, distinguished by whether the order of elements matters.

## 🎓 考点精讲

**定义与核心原理**
- **排列**：从 $n$ 个不同元素中取出 $m$（$m \leq n$）个元素，**按照一定顺序**排成一列。其排列数公式为 $A_n^m = n(n-1)(n-2)\cdots(n-m+1) = \frac{n!}{(n-m)!}$。
- **组合**：从 $n$ 个不同元素中取出 $m$（$m \leq n$）个元素，**不论顺序**并成一组。其组合数公式为 $C_n^m = \frac{A_n^m}{m!} = \frac{n!}{m!(n-m)!}$。
- **核心原理**：计数依据分步乘法计数原理与分类加法计数原理。

**关键方法与判定依据**
1. **有序无序判定**：将选出的元素交换位置，若产生不同结果则为排列，若结果不变则为组合。这是区分两类问题的唯一标准。
2. **优先策略**：高考解题通常采用“特殊元素(位置)优先法”——先安排受限元素，再处理其他；相邻问题用“捆绑法”，不相邻问题用“插空法”。
3. **间接法**：当正面讨论情况复杂时，用总方案数减去对立情况的方案数（“正难则反”）。

**高考典型设问**
- **直接计数**：如“7 人站一排，甲不站排头”，考查直接分步或间接排除。
- **分组分配**：如“6 本书均分给 3 人”或“分成三堆”，需注意平均分组中的重复计数问题，必须除以组数的阶乘。
- **路径与多元约束**：如网格图中含障碍物的最短路径数、数字排列中奇数位必须为偶数等复合限制条件。

## ⚠️ 高频易错点

- **混淆排列与组合的本质**：遇题不先判断“顺序”是否影响结果，盲目套用 $A_n^m$ 或 $C_n^m$ 导致全错。例如，选 3 人开班会（组合）与选 3 人分别任班长、学委、体委（排列）。
- **平均分组忘记去重**：将 $km$ 个元素均分成 $k$ 组（组间无差异），单纯连乘组合数会导致重复，必须除以 $k!$。例如，6 本书平均分成 3 堆，正确方案数为 $\frac{C_6^2 C_4^2 C_2^2}{3!}$。
- **漏用“特殊优先”导致分类混乱**：处理“甲不在两端”时，不先排甲而直接从左到右排位，极易造成分类重复或遗漏。应先安排受限元素甲（去中间位置），再排其余元素。
- **对“0”的特殊性忽视**：在数字排列问题中，0 若出现在首位则不是有效数字（最高位不能是 0），需单独排除 0 在首位的排列数。
