# 数列的递推关系

> **一句话**：递推关系是通过数列的若干前项表达后续项的等式，是定义数列和求解通项公式的核心工具。
> **English**: A recurrence relation is an equation that defines subsequent terms of a sequence based on preceding terms.

## 🎓 考点精讲

**定义/原理**：若数列 $\{a_n\}$ 满足形如 $a_{n+k} = f(a_{n+k-1}, a_{n+k-2}, \dots, a_n, n)$ 的等式，称为 $k$ 阶递推关系。高考中最常见的一阶、二阶线性递推标准形式为：

- 一阶线性递推：$a_{n+1} = p a_n + q(n)$（$p$ 为常数）
- 二阶线性齐次递推：$a_{n+2} = p a_{n+1} + q a_n$（需已知 $a_1, a_2$）

**关键方法**：

1. **累加法**（适用于 $a_{n+1} - a_n = f(n)$ 型）：$a_n = a_1 + \sum_{k=1}^{n-1} f(k)$
2. **累乘法**（适用于 $a_{n+1} / a_n = f(n)$ 型）：$a_n = a_1 \prod_{k=1}^{n-1} f(k)$
3. **待定系数法**（适用于 $a_{n+1} = p a_n + q$ 型，$p \ne 1$）：构造 $a_{n+1} + \lambda = p(a_n + \lambda)$，解得 $\lambda = \frac{q}{p-1}$，转化为等比数列。
4. **取倒数法**（适用于 $a_{n+1} = \frac{a_n}{k a_n + b}$ 型）：两边取倒数化为 $\frac{1}{a_{n+1}} = \frac{k a_n + b}{a_n} = \frac{b}{a_n} + k$，再设 $b_n = 1/a_n$ 转化为线性递推。

**典型应用**：高考选择题常考由递推关系求特定项（如 $a_{2023}$ 模周期）；填空题多考构造法求通项或判断数列的性质；解答题为压轴常客，综合给出递推关系、要求推导通项并证明不等式。例如：“已知 $a_1=1$，$a_{n+1}=2a_n+3$，求 $a_n$”。解：设 $a_{n+1}+\lambda=2(a_n+\lambda)$，比较得 $\lambda=3$，于是 $\{a_n+3\}$ 是首项 $4$、公比 $2$ 的等比数列，故 $a_n=4\cdot 2^{n-1}-3=2^{n+1}-3$。

## ⚠️ 高频易错点

- **符号错误**：累加时忽视 $n$ 的取值范围导致项数偏移，如将 $\sum_{k=1}^{n-1} f(k)$ 写为 $\sum_{k=1}^n$。
- **构造方向混淆**：待定系数法解 $\lambda$ 时代错方程，应始终以能提取公因式 $p$ 为目标检验。
- **首项遗漏**：构造新数列后只写通项形式而忘记加回构造所减的常数，如误答 $a_n = 4\cdot 2^{n-1}$。
- **累乘遗漏对数化**：对 $a_{n+1} = p \cdot a_n^{k}$ 型非线性递推，忘取对数化为线性再处理。
