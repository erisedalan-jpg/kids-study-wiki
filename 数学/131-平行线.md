## 课本定义

在同一平面内，不相交的两条直线叫做**平行线**，记作 l₁ ∥ l₂（读作"l₁ 平行于 l₂"）。平行线有**判定**（怎么判断两线平行）和**性质**（两线平行能推出什么）。

---

## 📚 6-12 岁回溯版

### "永不相遇的两条路"

**平行线**：在同一个平面上，两条直线无论怎么延伸，永远**不相交**、保持**等距**。

**生活中的平行线：**
| 场景 | 说明 |
|------|------|
| 铁路轨道 | 两条铁轨永不相交 |
| 五线谱 | 五条线互相平行 |
| 跑道 | 跑道的两边线平行 |
| 课本的横线 | 每行横线互相平行 |
| 楼梯扶手 | 上下两根扶手平行 |

**用截线"检验"平行**

当第三条线（截线）切过两条平行线，会产生一些特别的角：

```
         /
    ∠1  / ∠2
———————/————————  l₁
  ∠3  / ∠4
     /
∠5  / ∠6
———/————————————  l₂
∠7/ ∠8
 /
```

- **同位角**（F形）：∠1和∠5，∠2和∠6 —— 平行时，同位角相等
- **内错角**（Z形）：∠3和∠6，∠4和∠5 —— 平行时，内错角相等
- **同旁内角**（U形）：∠3和∠5，∠4和∠6 —— 平行时，同旁内角互补

**口诀：** "同位角F形等，内错角Z形等，同旁内角U形补"

## 🎓 12+ 进阶版

### 严格定义

**平行线**：在同一平面内，两条直线没有公共点（不相交），称两直线互相平行，记 l₁ ∥ l₂。

**注**：必须"在同一平面内"——空间中两条不相交的直线可能是异面直线，不一定平行。

### 平行线的判定（由角推平行）

设截线与两直线 l₁、l₂ 相交：

| 判定方法 | 条件 | 结论 |
|---------|------|------|
| 同位角相等 | ∠F = ∠F' | l₁ ∥ l₂ |
| 内错角相等 | ∠Z = ∠Z' | l₁ ∥ l₂ |
| 同旁内角互补 | ∠U + ∠U' = 180° | l₁ ∥ l₂ |
| 同一直线的垂线 | l₁⊥t，l₂⊥t | l₁ ∥ l₂ |

### 平行线的性质（由平行推角）

设 l₁ ∥ l₂，截线与两线相交：

| 性质 | 结论 |
|------|------|
| 同位角相等 | ∠F = ∠F' |
| 内错角相等 | ∠Z = ∠Z' |
| 同旁内角互补 | ∠U + ∠U' = 180° |

**判定与性质的区别**：
- **判定**：已知角关系 → 推出平行（条件→结论）
- **性质**：已知平行 → 推出角关系（条件→结论）

### 典型例题

**例题1：** 已知 l₁ ∥ l₂，截线与 l₁ 的同位角为 70°，求截线与 l₂ 所成的同位角。

解：由平行线性质，同位角相等，故截线与 l₂ 的同位角 = 70°。

**例题2：** 已知两直线被截线所截，内错角分别为 (2x+10)° 和 (3x-20)°，判断两线是否平行。

解：若平行，则内错角相等：2x+10 = 3x-20，x = 30。
内错角均为 70°。代入验证：2(30)+10=70，3(30)-20=70，相等，故两线平行。

### 易错点

1. **平行必须在同一平面内**，空间异面直线≠平行
2. **判定和性质方向相反**，不要混用
3. **同旁内角补**（不是相等），与同位角/内错角区别

### 知识联系

```
平行线
├── 判定 ← 同位角、内错角、同旁内角
├── 性质 → 同位角、内错角、同旁内角
├── 特殊：平行+垂直 → 两平行线间距离
└── 三角形内角和 → 借助平行辅助线证明
```
