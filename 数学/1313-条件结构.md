# 条件结构

> **一句话**：条件结构是算法中根据给定条件是否成立而选择执行不同分支的一种基本逻辑结构。
> **English**: A conditional structure is a basic logical construct in algorithms that selects a branch of execution based on whether a specified condition is true.

## 🎓 考点精讲

条件结构是算法三种基本结构之一（另两种为顺序结构与循环结构），用于实现选择分支。在程序框图中，条件结构通过**判断框**（菱形框）体现：判断框有一个入口和两个出口，分别对应条件“是”和“否”时执行的步骤。其一般形式可用伪代码表示为：

```
IF 条件 P THEN
    处理 A
ELSE
    处理 B
END IF
```

也可省略 `ELSE` 部分，形成单分支结构。高考中，条件结构通常融入程序框图、算法语句或与函数分段、数列递推等结合考查。

**关键方法**  
1. **读懂框图**：识别菱形框及 `Y`/`N` 指向的流程线。  
2. **确定条件表达式**：常为比较运算或逻辑组合，注意表达式中变量的当前值。  
3. **跟踪执行**：逐步计算直到程序结束，注意是否多次循环中重复进入条件判断。  
4. **补全框图**：根据题目要求的算法功能，反推出判断框中应填的条件或分支中应填的步骤。

**典型应用**  
- **分段函数求值**：程序框图根据自变量取值范围选择不同的表达式，如计算  
  $$y=\begin{cases} x^2, & x\leq 0\\ 2x+1, & x>0\end{cases}$$  
  判断框条件通常为“$x>0$”或“$x\leq 0$”。  
- **求最大/最小值**：通过多次比较并用条件结构更新当前最值，如“若 $a_i > max$，则 $max \leftarrow a_i$”。

## ⚠️ 高频易错点

- **条件表达方向写反**：补全判断框时，把“$x \ge 0$”写成“$x > 0$”，导致算法功能错误。
- **混用单分支与双分支**：需要处理两路操作时只画了一条路径，导致逻辑缺失。
- **忽略等号边界**：分段函数在分界点处定义可能属于不同分支，条件中等号位置易出错。
- **嵌套条件结构时执行顺序混乱**：多个判断框套用时，对“是”与“否”的出口连接错误，尤其是 `ELSE IF` 对应的框图表达。
