# 直线的方程

> **一句话**：直线的方程是描述平面直角坐标系中直线上所有点的坐标满足的二元一次方程，通过确定直线上一点和方向（斜率）唯一确定。
> **English**: An equation of a line is a linear equation in two variables that represents all points on the line in the Cartesian plane, uniquely determined by a point on the line and its direction (slope).

## 🎓 考点精讲

直线方程的核心是**斜率**与**直线上的定点**。斜率的定义：若直线上两点 $P_1(x_1,y_1)$、$P_2(x_2,y_2)$，且 $x_1 \neq x_2$，则斜率 $k = \frac{y_2 - y_1}{x_2 - x_1}$。斜率刻画了直线的倾斜程度，与倾斜角 $\alpha \in [0,\pi)$ 满足 $k = \tan\alpha$（$\alpha \neq \frac{\pi}{2}$）。

**基本形式与适用条件**：
- **点斜式**：过点 $P_0(x_0,y_0)$ 且斜率为 $k$：$y - y_0 = k(x - x_0)$。适用于斜率存在的情形；斜率不存在时直线为 $x = x_0$。
- **斜截式**：斜率为 $k$，在 $y$ 轴上的截距为 $b$：$y = kx + b$。是点斜式的特例，常用于快速作图与判断平行（$k_1 = k_2$）和垂直（$k_1 k_2 = -1$）。
- **两点式**：过两点 $(x_1,y_1)$、$(x_2,y_2)$（$x_1 \neq x_2, y_1 \neq y_2$）：$\frac{y - y_1}{y_2 - y_1} = \frac{x - x_1}{x_2 - x_1}$。需保证分母不为零，否则直接用 $x = x_1$ 或 $y = y_1$。
- **截距式**：在 $x$、$y$ 轴上的截距分别为 $a, b$（$a \neq 0, b \neq 0$）：$\frac{x}{a} + \frac{y}{b} = 1$。适合求与坐标轴围成的三角形面积 $S = \frac{1}{2}|a b|$。
- **一般式**：$Ax + By + C = 0$（$A, B$ 不同时为零）。可表示所有直线，斜率为 $-\frac{A}{B}$（$B \neq 0$）。用于距离公式与位置关系的判定。

**高考考查方式**：常以选择/填空题直接求直线方程，或在解析几何综合题（与圆、圆锥曲线）中作为基础环节。常见设问：根据条件（如平行、垂直、过定点、截距相等）求直线方程；利用直线方程求距离、最值；判断直线与曲线位置关系。

**解题关键步骤**：① 明确需要哪些要素（一点一斜率、两点、截距等）；② 根据条件选择最简形式设方程；③ 考虑斜率不存在的情形（分类讨论）；④ 将几何条件转化为代数方程求解参数；⑤ 最后化成一般式或指定形式。

## ⚠️ 高频易错点

- **忽略斜率不存在的情形**：许多同学设斜截式 $y = kx + b$ 后直接代入条件求解，忘记验证 $k$ 不存在时（即直线与 $x$ 轴垂直）是否满足题意，导致漏解。
- **斜率公式分母为零的问题**：使用两点式或斜率公式时，未先判断 $x_1$ 与 $x_2$ 是否相等；若相等则直线方程为 $x = x_1$，斜率不存在。
- **截距与距离的混淆**：截距是实数，可正可负可为零；距离是非负数。例如“在坐标轴上的截距相等”包含截距均为零（过原点）的情况，易误作 $a = b \neq 0$ 而漏掉 $y = kx$ 形式。
- **一般式系数符号的处理**：使用点到直线距离公式 $d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$ 时，未将直线方程化为一般式，或 $A, B$ 未提取公因子导致计算错误。
