## 课本定义

从**总体**中抽取**部分个体**进行调查，用样本的结果来估计总体，这种调查方式叫做**抽样调查**。被抽到的部分个体叫做**样本**，样本中个体的数目叫做**样本容量**。

---

## 📚 6-12 岁回溯版

### "问一部分人，猜全体的情况"

**全面调查**：问所有人（费时费力）
**抽样调查**：只问其中一部分人，用这部分人的情况来推测所有人

**生活例子：**
- 医生验血：只抽一管血（不是抽光你所有血），就能了解全身血液的情况
- 民意调查：不可能问遍全国14亿人，随机问几千人，用结果代表全体意见
- 食品安全检验：不能把所有饼干都吃完测试，只抽几袋检验

**关键词：**
- **总体**：全部研究对象（如全国消费者）
- **个体**：总体中的每一个（如每个消费者）
- **样本**：被抽取的部分个体（如抽到的1000个消费者）
- **样本容量**：样本中个体的数量（如 n = 1000）

**一个好的样本要满足什么？**

**随机性**——每个个体被抽到的机会要相等，不能专门挑或者只选方便的。

好的例子：摇号、随机抽签
不好的例子：只问班里坐在前排的同学（前排的同学可能坐得近，被老师提问多，和全班不一样）

## 🎓 12+ 进阶版

### 严格定义

**抽样调查**：从总体中按照一定规则抽取一部分个体（样本），通过对样本的调查分析来推断总体特征的方法。

**核心概念：**
| 概念 | 定义 |
|------|------|
| 总体（population）| 研究对象的全体 |
| 个体（individual）| 总体中的每一个元素 |
| 样本（sample）| 从总体中抽取的部分个体 |
| 样本容量（sample size n）| 样本中个体的数目 |
| 抽样误差 | 样本结果与总体真值的差异 |

### 常见抽样方法

| 方法 | 说明 | 适用场景 |
|------|------|---------|
| 简单随机抽样 | 每个个体被选中概率相等 | 总体规模不太大 |
| 系统抽样（等距抽样）| 按固定间隔抽取 | 总体按顺序排列 |
| 分层抽样 | 按比例从各子群体中抽 | 总体有明显分层 |

### 抽样调查的优缺点

| | 优点 | 缺点 |
|--|------|------|
| 抽样调查 | 节省时间和成本；必要时唯一可行方法 | 有抽样误差，结果是估计值 |

**减少误差的方法：**
- 增大样本容量（n 越大，误差越小）
- 保证样本的随机性（避免选择性偏差）
- 分层抽样（保证各子群体都有代表）

### 典型例题

**例题1：** 某市有初中生 50000 人，从中随机抽取 1000 人调查视力，其中近视 650 人，估计该市初中生近视人数。

解：样本中近视比例 = 650/1000 = 65%
估计总体近视人数 = 50000 × 65% = **32500 人**

（注：这是估计值，实际可能有误差）

**例题2：** 抽样调查时，以下哪种做法更科学？
(A) 只调查放学时在门口碰到的同学
(B) 用班级花名册随机抽取编号

解：(B) 更科学。(A) 只调查特定时间特定地点的同学，有选择性偏差，不能代表全体。(B) 随机抽取保证了随机性。

**例题3：** 工厂生产了10000个零件，用系统抽样（等距）抽取100个检验，间隔是多少？

解：间隔 = 10000 / 100 = **100 个**
每隔100个取1个，从前100个中随机选一个起始位置。

### 样本估计总体

抽样调查的核心应用：**用样本数据估计总体**

若样本中某特征的频率为 f，则估计总体中该特征的比例约为 f。

适用条件：样本必须具有代表性（随机、足量）。

### 易错点

1. **样本容量不是样本的个数，是样本中个体的数目**
2. **抽样结果是估计，不是精确值**
3. **样本要有随机性**，非随机样本不能代表总体
