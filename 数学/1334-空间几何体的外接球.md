# 空间几何体的外接球

> **一句话**：若一个多面体的所有顶点都在同一个球面上，则这个球叫做该多面体的**外接球**，其半径记为 \(R\)，是高考立体几何中求空间几何体表面积、体积的核心桥梁。
> **English**: A sphere that passes through all vertices of a polyhedron is called the circumscribed sphere of that polyhedron.

## 🎓 考点精讲

- **定义/原理**：对于给定的空间几何体（如棱柱、棱锥、棱台等），存在一个球，使得该几何体每一个顶点都在球面上。其充要条件是球心到各顶点距离相等。  
  几个基础结论：  
  - 长方体（正方体）的外接球球心在体对角线中点，直径 \(2R = \sqrt{a^2 + b^2 + c^2}\)。  
  - 直棱柱的外接球可通过上下底面外接圆圆心连线的中点得到球心，满足 \(R^2 = r^2 + \left(\dfrac{h}{2}\right)^2\)，其中 \(r\) 为底面外接圆半径，\(h\) 为高。  
  - 正棱锥的外接球球心在高线上，利用勾股关系 \(R^2 = (h-R)^2 + r^2\) 建立方程（\(h\) 为高，\(r\) 为底面外接圆半径）。  
  核心原理是**空间两点距离相等**，常转化为平面截面问题。

- **关键方法**：
  1. **补形法**：将三棱锥补成长方体或正方体（当棱满足共点垂直条件或对棱相等时），直接使用长方体对角线公式。
  2. **坐标法**：建立空间直角坐标系，设球心坐标 \((x_0,y_0,z_0)\)，利用到各顶点距离相等列方程，尤其适用于棱长已知但无特殊垂直关系的情形。
  3. **轴截面法**：画出过球心和某条棱或高线的轴截面，在平面图形中用勾股定理求 \(R\)，常用于直棱柱、正棱锥。
  4. **侧棱垂直于底面的棱锥**：先求底面外接圆半径 \(r\)，高为 \(h\)，则 \(R = \sqrt{r^2 + \left(\dfrac{h}{2}\right)^2}\)。

- **典型应用**：高考以选择题、填空为主，设问多为“某三棱锥/四棱锥的外接球的表面积为____”或“体积为____”。例如已知一个三棱锥 \(P-ABC\) 中，\(PA, PB, PC\) 两两垂直且 \(PA=1, PB=2, PC=3\)，求其外接球的表面积。解法：补形为长宽高分别为 \(1,2,3\) 的长方体，球直径 \(2R = \sqrt{1^2+2^2+3^2} = \sqrt{14}\)，\(S=4\pi R^2 = 14\pi\)。

## ⚠️ 高频易错点

- **混淆内切球与外接球**：外接球球心到各顶点距离相等，内切球球心到各面距离相等，审题时务必区分，尤其正四面体两者半径容易用错。
- **盲目套用模型**：补形法仅适用于对棱相等或共点垂直的三棱锥，若将不满足条件的棱锥强行补成长方体会导致错误，需先验证几何特征。
- **球心位置误判**：如非正棱锥，球心不一定在高线上；直棱柱若上下底面外形不对称，球心不在体对角线交点；常需通过方程确定。
- **计算细节错误**：在用 \(R^2 = r^2 + (h/2)^2\) 时，容易忘记 \(r\) 为底面**外接圆**半径，而非内切圆半径或边长的一半；计算表面积/体积时易看错半径与直径（如 \(2R=\sqrt{a^2+b^2+c^2}\) 写成 \(R=\sqrt{a^2+b^2+c^2}\)）。
