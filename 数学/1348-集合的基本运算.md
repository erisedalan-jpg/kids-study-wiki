# 集合的基本运算

> **一句话**：在给定全集下，通过并集、交集、补集等运算由一个或多个已知集合构造新集合的过程。
> **English**: The fundamental processes of constructing new sets from given ones within a universal set, primarily via union, intersection, and complement.

## 🎓 考点精讲

- **定义/原理**：设 $A,B$ 为集合，全集为 $U$。
  - **交集** $A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$；
  - **并集** $A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$；
  - **补集** $\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$。
  常用运算律：交换律 $A \cap B = B \cap A$，$A \cup B = B \cup A$；结合律 $(A \cap B) \cap C = A \cap (B \cap C)$；分配律 $A \cap (B \cup C) = (A \cap B) \cup (A \cap C)$；德摩根律 $\complement_U(A \cap B) = (\complement_U A) \cup (\complement_U B)$，$\complement_U(A \cup B) = (\complement_U A) \cap (\complement_U B)$。有限集的元素个数满足容斥原理：$|A \cup B| = |A| + |B| - |A \cap B|$，可推广至三个集合。

- **关键方法**：高考解答集合运算题的标准步骤：（1）明确全集与各集合的含义，必要时用列举法、描述法或数轴、Venn图直观表示；（2）根据交、并、补定义逐级计算，先括号内后括号外；（3）遇到参数问题时，利用集合相等或包含关系列出方程或不等式，并检验集合元素的互异性及空集情形。对于无限数集，常借助数轴标根法处理不等式解集间的运算。

- **典型应用**：高考常以选择题、填空题出现，设问方式包括：求 $A \cap B$、$A \cup (\complement_U B)$、已知运算结果反求参数、判断集合关系等。情境多结合不等式解集、函数定义域值域、点集、方程的解集。例如：$U = \mathbb{R}$，$A = \{x \mid x^2 - 3x + 2 \le 0\}$，$B = \{x \mid \log_2 x < 1\}$，求 $\complement_U A \cap B$，需先解出 $A=[1,2]$，$B=(0,2)$，再用数轴得到结果。

## ⚠️ 高频易错点

- **忽略空集**：子集或交集运算中含参数时，忘记讨论空集导致漏解。例如 $A \subseteq B$ 且 $A$ 含参数，必须考虑 $A = \varnothing$ 的情形。
- **混淆“或”“且”含义**：并集对应逻辑“或”（至少一个成立），交集对应“且”（同时成立），将不等式解集的“并”与“交”写反是常见错误。
- **补集运算未先确定全集**：$\complement_U A$ 依赖于全集 $U$，如题目中隐含全集变化（如从实数集变为整数集），直接套用原全集结论会出错。
- **无限集运算时端点处理不当**：在用数轴求并、交、补时，端点是否能取等号（开闭区间）常被忽略，尤其在多次运算后易出错。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 并集 | union | $A \cup B$，元素属于 $A$ 或 $B$ |
| 交集 | intersection | $A \cap B$，元素同时属于 $A$ 和 $B$ |
| 补集 | complement | $\complement_U A$，全集 $U$ 中不属于 $A$ 的元素构成的集合 |

| 空集 | empty set | 不含任何元素的集合，记作 $\varnothing$ |
