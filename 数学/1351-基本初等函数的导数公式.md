# 基本初等函数的导数公式

> **一句话**：基本初等函数的导数公式是给出常数函数、幂函数、指数函数、对数函数、三角函数等最基本函数求导结果的公式集，是复合函数求导、切线斜率计算和函数性质研究的基础。
> **English**: The derivative formulas for basic elementary functions are a set of rules that directly give the derivatives of constant, power, exponential, logarithmic, and trigonometric functions, serving as the foundation for the chain rule, tangent slope calculations, and function analysis.

## 🎓 考点精讲

**定义/原理**：基本初等函数的导数公式是指下列函数不需经过极限定义推导、可直接使用的求导结论：

| 函数类型 | $f(x)$ | 导数 $f'(x)$ |
|----------|--------|--------------|
| 常数函数 | $c$（$c$为常数） | $0$ |
| 幂函数 | $x^\alpha$（$\alpha \in \mathbb{R}$） | $\alpha x^{\alpha-1}$ |
| 指数函数 | $a^x$（$a>0$，$a \neq 1$） | $a^x \ln a$ |
| 指数函数（特例） | $e^x$ | $e^x$ |
| 对数函数 | $\log_a x$（$a>0$，$a \neq 1$，$x>0$） | $\frac{1}{x \ln a}$ |
| 对数函数（特例） | $\ln x$（$x>0$） | $\frac{1}{x}$ |
| 正弦函数 | $\sin x$ | $\cos x$ |
| 余弦函数 | $\cos x$ | $-\sin x$ |

在此基础上，还需要掌握和、差、积、商的求导法则以及复合函数求导的链式法则，以求解由基本初等函数组合而成的函数的导数。

**关键方法**：高考中直接考查公式识记的题目较少，更多是将公式嵌入问题解决流程，标准步骤为：
1. 识别函数结构，明确由哪些基本初等函数通过加、乘、除及复合构成。
2. 选用相应求导法则（积、商、链式法则）拆解。
3. 代入基本初等函数导数公式算出具体表达式。
4. 利用导数研究切线方程（代入$x_0$得$f'(x_0)$）、单调性（解$f'(x)>0$或$<0$）、极值与最值（求导数的零点并判断符号变化）。

**典型应用**：高考中常见设问有：
- **求切线方程**：例如“曲线$y=\ln x + x^2$在点$(1,1)$处的切线方程”。需先求导$f'(x)=\frac{1}{x}+2x$，再代入$x=1$得斜率$k=3$，得到切线$y-1=3(x-1)$。
- **函数单调性与极值**：比如“讨论$f(x)=e^x - 2x - a$的零点个数”。求导$f'(x)=e^x - 2$，由$f'(x)=0$得$x=\ln 2$，结合单调性画出草图解决零点问题。

## ⚠️ 高频易错点

- **幂函数与指数函数求导混淆**：幂函数$x^a$的导数是$a x^{a-1}$（指数字不变、指数减一降到底），而指数函数$a^x$的导数是$a^x \ln a$（底数不变、乘以$\ln a$），两者极易用错，尤其是见到$x^e$或$\pi^x$时需先判定函数类型。
- **对数函数导数漏掉绝对值或定义域**：$(\ln |x|)' = \frac{1}{x}$ 常用于积分，但在高考导数题中通常给定$x>0$可直接用$(\ln x)'=\frac{1}{x}$，忘记检验定义域会导致讨论缺失。
- **复合函数求导“链”不到位**：对$\sin(2x)$或$e^{-x}$求导时，学生常只记外部函数导数，漏乘内部函数导数（$\sin(2x)$的导数应为$2\cos(2x)$，漏乘$2$则错）。
- **公式记忆混乱导致符号错误**：$(\cos x)' = -\sin x$的负号易忘；$(\ln x)' = 1/x$，不少学生误记成导数仍带$\ln$的形式。
