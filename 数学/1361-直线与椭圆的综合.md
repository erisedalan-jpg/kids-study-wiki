# 直线与椭圆的综合

> **一句话**：研究直线与椭圆相交时满足的代数条件、几何关系及相关量的计算，核心是联立方程后运用韦达定理。
> **English**: Comprehensive problems of lines and ellipses.

## 🎓 考点精讲

**核心原理：联立方程与判别式**

给定直线 $l: y = kx + m$ （或 $x = ty + n$） 与椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 \ (a > b > 0)$，将其联立消去 $y$（或 $x$），得到关于 $x$ 的一元二次方程：
$$(b^2 + a^2k^2)x^2 + 2a^2kmx + a^2(m^2 - b^2) = 0$$
其判别式 $\Delta$ 决定了直线与椭圆的位置关系：
- $\Delta > 0 \iff$ 相交（两个公共点）
- $\Delta = 0 \iff$ 相切（一个公共点）
- $\Delta < 0 \iff$ 相离（无公共点）

**关键方法：设而不求，整体代换**

高考解答题中，处理直线与椭圆相交的核心步骤为：
1. **设点设线**：设交点 $A(x_1, y_1), B(x_2, y_2)$，设直线方程（注意讨论斜率是否存在）。
2. **联立消元**：将直线方程代入椭圆方程，化为关于 $x$ 或 $y$ 的一元二次方程。
3. **韦达定理**：直接写出 $x_1 + x_2$ 和 $x_1x_2$（或 $y_1+y_2, y_1y_2$）的表达式，**不具体求出坐标**。
4. **翻译条件**：将题目中的几何条件（长度、角度、垂直、共线等）转化为关于 $x_1+x_2$ 与 $x_1x_2$ 的代数等式。常用工具包括弦长公式 $|AB| = \sqrt{1+k^2}\sqrt{(x_1+x_2)^2 - 4x_1x_2}$、向量内积、中点坐标等。
5. **求解参数**：解关于 $k$ 或 $m$ 的方程，并验证 $\Delta > 0$。

**典型应用**

- **弦中点问题**：求以某定点为中点的弦所在直线方程，常用“点差法”。将 $A, B$ 坐标代入椭圆方程并作差，由 $k_{AB} = \frac{y_1-y_2}{x_1-x_2} = -\frac{b^2 x_0}{a^2 y_0}$（其中 $(x_0, y_0)$ 为中点坐标）快速求出斜率。
- **定值与定点问题**：证明直线过定点，可设含参直线方程，利用已知的几何关系消去参数，求出定点坐标。证明某量为定值，则将该量用韦达定理表示为参数的函数，化简消去参数。
- **面积最值**：涉及 $\triangle OAB$ 面积，常用 $S = \frac{1}{2}|m|\cdot|x_1 - x_2|$ 或分割为两个同底三角形，结合基本不等式或函数求最值。

## ⚠️ 高频易错点

- **忽视斜率不存在的情况**：设直线方程 $y = kx + m$ 前，未单独讨论斜率不存在的竖直直线（如 $x = t$），导致漏解。
- **忽略判别式检验**：利用韦达定理得出参数值后，忘记代回判别式验证 $\Delta > 0$，可能导致将相切或相离情况的解误作符合题意的结果。
- **韦达定理代换错误**：混淆 $x_1+x_2$ 与 $x_1x_2$ 的表达式符号，或在化简复杂等式时出现计算失误。使用弦长公式 $\sqrt{1+k^2}\sqrt{(x_1+x_2)^2 - 4x_1x_2}$ 时，遗漏根号或系数。
- **点差法适用条件误判**：点差法只能用于“已知弦中点”，且必须保证弦存在（中点在椭圆内部），否则求出的直线方程可能不与椭圆相交。
