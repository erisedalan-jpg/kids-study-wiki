# 二元一次不等式组与平面区域

> **一句话**：二元一次不等式组是由两个或多个二元一次不等式联立而成，其解集在平面直角坐标系中对应各不等式所表示的半平面区域的交集。
> **English**: A system of binary linear inequalities consists of two or more binary linear inequalities; its solution set is the intersection of the half-planes determined by each inequality.

## 🎓 考点精讲

- **定义/原理**：二元一次不等式的一般形式为 $Ax+By+C>0$（或 $\ge,<,\le 0$），其几何意义是直线 $l:Ax+By+C=0$ 某一侧的所有点构成的半平面（含边界时不等号带等号，边界画实线；不含边界时画虚线）。二元一次不等式组则是若干个这样的不等式联立，如 $\begin{cases} A_1x+B_1y+C_1>0 \\ A_2x+B_2y+C_2\le 0 \end{cases}$，解集为各不等式对应半平面的交集，在坐标平面上呈现为一个平面区域（可能是多边形或无界区域）。

- **关键方法**：  
  1. 将每个不等式化为 $Ax+By+C$ 与 $0$ 比较的形式。  
  2. 画出对应的直线 $Ax+By+C=0$，含等号画实线，不含等号画虚线。  
  3. 选定一个不在直线上的特殊点（通常若直线不过原点，可取原点 $(0,0)$）代入不等式检验，若成立则该点所在的一侧即为所求半平面；若不成立则取另一侧。  
  4. 将所有不等式的区域叠加，取公共部分（交集）即得不等式组表示的区域。  
  5. 若区域为封闭多边形，可通过解方程组求交点坐标，再利用坐标公式或割补法计算面积。

- **典型应用**：高考常以“求不等式组表示的平面区域的面积”直接设问。例如，不等式组 $\begin{cases} x+y-2\le 0 \\ x-y+1\ge 0 \\ y\ge -1 \end{cases}$ 围成一个三角形区域，需先画出三条直线 $x+y-2=0$（实线）、$x-y+1=0$（实线）、$y=-1$（实线），取公共交集，求出三个顶点 $A(3,-1), B(-2,-1), C(0.5,1.5)$，再用面积公式 $S=\frac{1}{2}|(x_1y_2-x_2y_1)+\cdots|$ 或割补法求出面积。此外，该考点也是线性规划中“可行域”的基础，常与目标函数最值结合命题。

## ⚠️ 高频易错点

- 画边界直线时，忽略不等号是否含有等号，导致虚实线画错，进而误判区域是否包含边界。
- 判定半平面方向时，选用的特殊点恰在直线上（如原点在直线 $x+y=0$ 上），导致误判。此时应另选点如 $(1,0)$ 或 $(0,1)$ 进行检验。
- 取公共区域时，混淆不等号方向，误将“同向”区域当作交集，或忘记对每个不等式分别判断区域，导致最终区域选错。
- 求多边形区域面积时，顶点坐标计算错误，或顶点顺序排列不按顺时针（逆时针），导致行列式面积公式出现符号错误或计算失误。
