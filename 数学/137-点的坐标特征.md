## 课本定义

在平面直角坐标系中，不同位置的点，其坐标 (x, y) 的符号有固定规律：各象限内点的坐标符号、坐标轴上点的坐标特征各有不同。利用坐标特征，可以由位置判断坐标符号，也可由坐标符号判断位置。

---

## 📚 6-12 岁回溯版

### "坐标里藏着位置信息"

知道了坐标系，现在来总结：不同位置的点，坐标有什么规律？

**四个象限的坐标特征：**

```
      y轴
      ↑
第二象限  |  第一象限
(-,+)  |  (+,+)
       |
———————O———————→ x轴
       |
(-,-)  |  (+,-)
第三象限  |  第四象限
      ↓
```

| 位置 | x（横坐标） | y（纵坐标） | 记忆法 |
|------|-----------|-----------|-------|
| 第一象限 | + | + | 右上，都正 |
| 第二象限 | - | + | 左上，x负y正 |
| 第三象限 | - | - | 左下，都负 |
| 第四象限 | + | - | 右下，x正y负 |

**坐标轴上的特征：**
- **x 轴上的点**：y = 0（不偏上也不偏下）
- **y 轴上的点**：x = 0（不偏左也不偏右）
- **原点 O**：x = 0，y = 0

**快速记忆：**
- 横坐标 x：在 y 轴右边是正，左边是负
- 纵坐标 y：在 x 轴上方是正，下方是负

**用坐标判断位置（反过来用）：**
- 已知点 P(3, -2)：x=3>0（右），y=-2<0（下）→ 第四象限
- 已知点 Q(-1, 0)：x=-1<0，y=0 → 在 x 轴上（y轴左侧）

## 🎓 12+ 进阶版

### 各位置点的坐标完整总结

**象限内的点：**

| 象限 | 坐标条件 | 说明 |
|------|---------|------|
| 第一象限 | x > 0 且 y > 0 | |
| 第二象限 | x < 0 且 y > 0 | |
| 第三象限 | x < 0 且 y < 0 | |
| 第四象限 | x > 0 且 y < 0 | |

**坐标轴上的点：**

| 位置 | 坐标条件 | 说明 |
|------|---------|------|
| x 轴正半轴 | x > 0，y = 0 | |
| x 轴负半轴 | x < 0，y = 0 | |
| y 轴正半轴 | x = 0，y > 0 | |
| y 轴负半轴 | x = 0，y < 0 | |
| 原点 O | x = 0，y = 0 | |

**注意**：坐标轴上的点**不属于任何象限**。

### 关于对称点的坐标特征

| 对称关系 | 点 P(a, b) 的对称点 |
|---------|-----------------|
| 关于 x 轴对称 | P'(a, -b) |
| 关于 y 轴对称 | P'(-a, b) |
| 关于原点对称 | P'(-a, -b) |
| 关于直线 y=x 对称 | P'(b, a) |

### 典型例题

**例题1：** 判断下列各点所在位置：
A(3, -5)，B(-2, -4)，C(0, 3)，D(-1, 0)，E(0, 0)，F(2, 7)

解：
- A(3, -5)：x>0, y<0 → 第四象限
- B(-2, -4)：x<0, y<0 → 第三象限
- C(0, 3)：x=0, y>0 → y轴正半轴
- D(-1, 0)：y=0, x<0 → x轴负半轴
- E(0, 0) → 原点
- F(2, 7)：x>0, y>0 → 第一象限

**例题2：** 点 P(a, b) 在第二象限，求点 Q(-a, -b) 在哪个象限？

解：P 在第二象限 → a < 0，b > 0
Q 的坐标为 (-a, -b)
-a = -（负数）= 正数 > 0
-b = -（正数）= 负数 < 0
Q(-a, -b) 的 x > 0，y < 0 → 第四象限

**例题3：** 点 P(2a-1, a+2) 在 y 轴上，求 a 及点的坐标。

解：在 y 轴上 → x 坐标 = 0
2a - 1 = 0
a = 1/2
y 坐标 = a + 2 = 1/2 + 2 = 5/2
点的坐标为 (0, 5/2)，在 y 轴正半轴上。

### 易错点

1. **坐标轴上的点不属于任何象限**
2. **x 轴上 y = 0，y 轴上 x = 0**（容易搞混哪个是0）
3. **原点 O(0, 0) 既不在象限，也不在轴的正/负半轴**
