# 复合三角函数的单调性

> **一句话**：形如 $y = A\sin(\omega x + \varphi) + B$ 或 $y = A\cos(\omega x + \varphi) + B$ 的复合函数，通过整体代换 $\omega x + \varphi$ 并利用基本三角函数的单调区间来求解 $x$ 的范围。
> **English**: The monotonicity of composite trigonometric functions, referring to solving the increasing or decreasing intervals of functions like $y = A\sin(\omega x + \varphi) + B$ by substituting the inner linear expression into the monotonic intervals of the basic sine or cosine.

## 🎓 考点精讲

**定义/原理**  
对于 $y = f[g(x)]$ 型的复合三角函数，最常见的是 $y = A\sin(\omega x + \varphi) + k$ 与 $y = A\cos(\omega x + \varphi) + k$（$A \neq 0,\ \omega \neq 0$）。其单调性取决于内层函数 $u = \omega x + \varphi$ 在外层函数 $y = A\sin u$ 或 $y = A\cos u$ 的单调区间的映射。  
当 $\omega > 0$ 时，内层 $u = \omega x + \varphi$ 是严格递增的一次函数。此时可由 $u$ 在基本三角函数的增/减区间解出 $x$ 的范围，从而得到复合函数的增/减区间。  
当 $\omega < 0$ 时，内层为严格递减函数。此时必须先利用诱导公式（如 $\sin(-\alpha) = -\sin\alpha$）将 $\omega$ 化为正数，或直接用复合函数“同增异减”法则判断。

**关键方法**  
以 $y = A\sin(\omega x + \varphi)\ (\omega > 0)$ 为例：  
- **增区间**：由 $-\frac{\pi}{2} + 2k\pi \le \omega x + \varphi \le \frac{\pi}{2} + 2k\pi,\ k \in \mathbb{Z}$ 解出 $x$ 的区间。  
- **减区间**：由 $\frac{\pi}{2} + 2k\pi \le \omega x + \varphi \le \frac{3\pi}{2} + 2k\pi,\ k \in \mathbb{Z}$ 解出 $x$ 的区间。  
对于 $y = A\cos(\omega x + \varphi)$ 则分别用 $[2k\pi, \pi + 2k\pi]$（减区间）和 $[-\pi + 2k\pi, 2k\pi]$（增区间）。  
若 $\omega < 0$，务必先用诱导公式变形为 $\omega' > 0$ 的形式再求解。

**典型应用**  
高考常考查：  
1. 直接求 $f(x) = 2\sin\left(2x - \frac{\pi}{3}\right)$ 的单调递增区间。  
   解：令 $-\frac{\pi}{2} + 2k\pi \le 2x - \frac{\pi}{3} \le \frac{\pi}{2} + 2k\pi$，解得 $-\frac{\pi}{12} + k\pi \le x \le \frac{5\pi}{12} + k\pi,\ k \in \mathbb{Z}$，即增区间为 $\left[-\frac{\pi}{12} + k\pi, \frac{5\pi}{12} + k\pi\right]$。  
2. 已知 $\omega > 0$，$f(x) = \sin(\omega x + \frac{\pi}{4})$ 在 $(\frac{\pi}{2}, \pi)$ 上单调递减，求 $\omega$ 的范围。  
   需将 $(\frac{\pi}{2}, \pi)$ 包含于某个减区间，结合区间长度与周期限制建立不等式。

## ⚠️ 高频易错点

- 直接套用 $\omega > 0$ 时的公式却忽略 $\omega < 0$ 的情况，导致单调区间反向。
- 解不等式 $\omega x + \varphi \le \cdots$ 时，处理 $k\pi$ 与 $2k\pi$ 混淆，或忘记加上 $k \in \mathbb{Z}$。
- 将单调区间误写为开区间而题目要求闭区间（或反之），尤其当 $A$ 的正负影响端点可达性时。
- 将 $y = A\sin(\omega x + \varphi)$ 的单调性误当作 $y = A\cos(\omega x + \varphi)$ 的单调性，记混两者的单调区间起点。
