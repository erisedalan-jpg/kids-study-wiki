# 应用题

> **一句话**：将现实情境转化为数学语言，通过建立数学模型（方程、不等式、函数等）求解实际问题的题目类型。
> **English**: A type of problem that requires translating a real‑world situation into a mathematical model and solving it.

## 🎓 考点精讲

- **定义/原理**：应用题的核心是 **数学建模**。即从题干中提取数量关系、几何特征或变化规律，构造一个或多个数学结构（如方程 $f(x)=0$、不等式组、函数关系 $y=f(x)$、数列递推 $a_{n+1}=qa_n + b$、概率分布列等），再运用对应数学工具求解，并将结果还原回原情境解释与检验。  
- **关键方法**：高考应用题的通用步骤为 **审题—建模—求解—检验**。  
  1. 审题：圈画关键量，明确已知与未知，注意单位与变化过程；  
  2. 建模：选择合适模型，如涉及相等关系列方程，变化趋势列函数，周期与递推用数列，最优化用导数或线性规划，随机现象用统计或概率；  
  3. 求解：严格按所选模型的公式、定理、运算法则计算；  
  4. 检验：将数学解代回实际背景，淘汰不合常理（如负长度、非整数人数）的解，并明确作答。  
- **典型应用**：高考常见设问如“建立日利润与售价的函数关系，并求最大利润”考查函数与导数；“根据分期付款方案写出欠款数列”考查数列；“依据样本数据预测销量”考查线性回归；“设计方案使材料最省”考查不等式或几何极值。  
  示例：某商品成本为每件 $c$ 元，售价为 $p$ 元时销量 $Q = a - bp$，则利润函数 $L(p) = (p-c)(a-bp)$，定义域 $c < p < \frac{a}{b}$，由 $L'(p)=0$ 得最优定价 $p^* = \frac{a+bc}{2b}$（待家长核对）。此过程完整体现了应用题从实际到模型再回实际的全貌。

## ⚠️ 高频易错点

- **忽略定义域的实际意义**：函数建模后忘记自变量的取值范围（如价格非负、人数为整数、时间非零），导致最值出现在区间外时误答。  
- **单位不统一或换算错误**：题干中出现“万元/年”“元/m²”，建模时未统一单位便代入公式，造成数量级错误。  
- **误选模型**：将本应使用等比数列的增长率问题当作等差数列处理，或将非线性回归强行线性化。  
- **结果未检验舍去增根**：解分式方程、无理方程或利用不等式放缩时产生增根或等号条件不成立，未代入原题检验即草率作答。
