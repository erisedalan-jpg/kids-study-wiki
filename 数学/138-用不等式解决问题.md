## 课本定义

用不等式解决实际问题的步骤：设未知数 → 依题意列不等式（或不等式组）→ 解不等式 → 检验解的合理性 → 得出结论。核心是将实际问题中的"范围、至少、至多、不超过"等转化为不等式。

---

## 📚 6-12 岁回溯版

### "把生活里的'够不够'变成数学"

**什么时候用不等式？**

生活中很多问题不是"等于多少"，而是"至少""至多""不超过""不低于"这样的范围问题，就需要用不等式！

**关键词翻译：**
| 生活用语 | 数学不等式 |
|---------|-----------|
| 至少 a | ≥ a |
| 至多 a | ≤ a |
| 不超过 a | ≤ a |
| 不低于 a | ≥ a |
| 大于 a | > a |
| 小于 a | < a |
| 在 a 到 b 之间 | a ≤ x ≤ b |

**解题步骤（和方程类似）：**
1. 设未知数（设 x 为……）
2. 找出不等关系，列不等式
3. 解不等式
4. 看答案是否符合实际（如人数要是整数、不能为负数等）
5. 写结论

**例子：** 小明去买铅笔，每支 1.5 元，他手里有 20 元，最多能买几支？

设买 x 支，1.5x ≤ 20，x ≤ 20/1.5 ≈ 13.3

因为 x 是整数，所以最多买 **13 支**。

## 🎓 12+ 进阶版

### 用不等式解实际问题的模型

**常见问题类型：**

| 问题类型 | 建立不等式的思路 |
|---------|---------------|
| 费用/价格 | 总价 ≤ 预算，或 总价 ≥ 最低要求 |
| 路程/时间 | 路程/速度的约束 |
| 生产/利润 | 利润/收益的范围 |
| 数量/计数 | 人数/个数的整数约束 |
| 温度/参数 | 工业/生活中的参数范围 |

### 典型例题

**例题1（费用问题）：** 某同学有 50 元，要买单价 3 元的钢笔和单价 2 元的铅笔，计划至少买 20 支，且钢笔不超过 10 支，最多能买钢笔几支？

设买钢笔 x 支，则铅笔 (20-x) 支（至少20支取边界）。

条件：钢笔不超过10支，x ≤ 10
费用：3x + 2(20-x) ≤ 50
3x + 40 - 2x ≤ 50
x ≤ 10

两个条件均给出 x ≤ 10，结合铅笔数 ≥ 0：20 - x ≥ 0，x ≤ 20。

综合：x ≤ 10，且 x 为非负整数。最多买 **10 支**钢笔。

**例题2（路程问题）：** 甲步行速度 5 km/h，乙骑车速度 15 km/h，同时从同地出发同方向走，乙至少需要多少时间才能领先甲 20 km？

设经过 t 小时：
乙领先甲 ≥ 20 km：15t - 5t ≥ 20
10t ≥ 20
t ≥ 2

答：至少需要 **2 小时**。

**例题3（利润问题）：** 某商店进货价 60 元/件，为保证利润率不低于 25%，售价应定在多少元以上？

设售价 x 元，利润率 = (x-60)/60 ≥ 25%
x - 60 ≥ 15
x ≥ 75

答：售价至少定 **75 元**。

**例题4（整数范围问题）：** n 是整数，且满足 -3 < 2n-1 < 5，求所有满足条件的 n。

解：-3 < 2n-1 < 5
加1各处：-2 < 2n < 6
除2：-1 < n < 3

n 为整数，则 n ∈ {0, 1, 2}。

### 解题注意事项

1. **检验实际约束**：个数必须为整数、人数不能为负、长度不能为0等
2. **不等式端点的取舍**：≥ 包含等号，如最小是整数时可取到端点
3. **多条件联立**：多个约束用不等式组，取公共解集
4. **单位一致**：时间、路程、价格的单位要统一
