# 新定义

> **一句话**：新定义问题是在题目中临时给出一个教材中未出现的新概念、新运算或新性质，要求考生在“现场学习”后立即运用其解决问题。
> **English**: A "new definition" problem introduces a novel concept, operation, or property not found in the textbook, requiring students to learn it on the spot and apply it immediately.

## 🎓 考点精讲

**定义/原理**：  
“新定义”本身不是某个固定的数学对象，而是一种命题范式。题中常见形式：  
- **新运算**：如对于实数 $a,b$ 定义 $a \oplus b = a^2 - ab + b$。  
- **新概念**：如定义“等和点”：若点 $P$ 满足 $PA + PB = PC + PD$，则称 $P$ 为某四边形的等和点。  
- **新性质**：如定义数列 $\{a_n\}$ 的“平滑度”为 $\lim\limits_{n \to \infty} \frac{a_{n+1}}{a_n}$。  

这类题目的本质是**阅读理解 + 数学建模**，特点是“现学现用”。所给定义必须逐字解读，并在后续小题中反复使用。

**关键方法**：  
1. **逐字翻译**：将文字定义翻译成严格的数学表达式。对运算律（交换律、结合律）要逐一验证。  
2. **套用定义**：用定义直接计算或证明，不要添加主观经验的额外假设。  
3. **逆用定义**：常见设问“若某对象满足新定义，求参数范围”——此时需将满足定义的条件列成方程或不等式，然后解之。  
4. **迁移类比**：若新定义与已知函数、数列、几何性质相似（如新距离类似于曼哈顿距离），可借鉴已知模型的解法，但必须严格在新定义框架下操作。

**典型应用**：  
高考常以第19–20题（中档或压轴）出现。  
- **场景1—新运算与抽象函数**：定义 $f(x)*g(x) = f(x) + g(x) + f(x)g(x)$，讨论其是否满足结合律，并求特定函数值。  
- **场景2—新定义数列**：定义“和谐数列”：对任意 $n\in\mathbb{N}^*$ 有 $a_{n+2} \leq \frac{a_{n+1}+a_n}{2}$，判断某具体数列是否和谐，并证明与单调性的关系。

## ⚠️ 高频易错点

- **擅自添加运算律**：看到新运算 $\oplus$ 就默认满足交换律 $a \oplus b = b \oplus a$ 或结合律，必须由定义严格验证，否则后继推理全盘错误。  
- **忽略定义域/前提条件**：新定义往往带有限制（如 $a \neq 0$、角度在 $(0,\pi)$ 内），跳过此步可能导致结果错误或讨论遗漏。  
- **将新概念等同于已有概念**：例如定义“近距数对”，不可直接套用“最小二乘”或“欧氏距离”的结论，除非新定义与之一致。  
- **题干要求“是否存在”却只给出“存在”的特例**：必须给出存在性的严格证明或反证，单举一例不构成完整解答。
