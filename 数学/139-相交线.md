## 课本定义

两条直线有且只有一个公共点，这样的两条直线叫做**相交线**，公共点称为**交点**。

---

## 📚 6-12 岁回溯版

### 生活里的"十字路口"

想象你走在街上，看到两条路交叉在一起——这就是相交线！

**相交线是什么？**
- 两条直线（无限延伸的线）碰到一起，只在**一个点**相遇
- 这个相遇的点叫**交点**
- 就像剪刀的两个刀刃，合拢时只在一个点交叉

**生活中的例子：**
| 场景 | 说明 |
|------|------|
| 十字路口 | 两条马路交叉 |
| 剪刀刀刃 | 两个刃在铰接点相交 |
| 汉字"十" | 一横一竖相交于中心 |
| 星星射线 | 多条线交于一点 |

**两直线相交会产生什么？**
两条直线相交，会形成**4个角**：
- 邻补角（相邻的角，加起来是180°）
- 对顶角（对面的角，大小相等）

**动手试试：**
拿两根筷子，在桌上交叉摆放。数一数形成了几个角？移动一根筷子，角的大小变了，但对面的角是不是始终相等？

## 🎓 12+ 进阶版

### 严格定义

**相交线**：在同一平面内，有且仅有一个公共点的两条直线。该公共点称为**交点**（point of intersection）。

### 核心性质

两条直线相交产生**4个角**，设相交形成角 ∠1、∠2、∠3、∠4（按顺序排列）：

| 角对关系 | 定义 | 性质 |
|---------|------|------|
| **对顶角** | 顶点相同，两边互为反向延长线 | 对顶角相等 |
| **邻补角** | 有公共顶点和公共边，另一边互为反向延长线 | 互补，和为 180° |

**公式：**
- 若 ∠1 = α，则 ∠3 = α（对顶角）
- ∠1 + ∠2 = 180°，∠2 + ∠3 = 180°（邻补角）
- 四角之和 = 360°

### 典型例题

**例题1：** 两直线相交，其中一个角为 65°，求其余三个角。

解：设 ∠1 = 65°
- ∠3 = 65°（对顶角）
- ∠2 = 180° - 65° = 115°（邻补角）
- ∠4 = 115°（∠2 的对顶角）

答：另外三个角分别为 65°、115°、115°。

**例题2：** 两直线相交，已知一对对顶角之和为 130°，求每个对顶角的度数及另一对对顶角的度数。

解：一对对顶角相等，设每个为 x，则 2x = 130°，x = 65°
另一对对顶角：(180° - 65°) = 115°，两个均为 115°

### 易错点

1. **相交 ≠ 垂直**：两直线相交角度不限，垂直是特殊相交（成90°）
2. **必须在同一平面内**：空间中两直线可能是异面直线，不相交也不平行
3. **恰好一个交点**：平行线没有交点，重合直线有无数交点

### 知识联系

```
相交线
├── 特殊情况 → 垂直（交角=90°）
├── 产生 → 对顶角（相等）
├── 产生 → 邻补角（互补）
└── 对立 → 平行线（无交点）
```
