## 课本定义

如果 x³ = a，那么 x 叫做 a 的**立方根**（也称**三次方根**）。每个实数（正数、负数、零）都有唯一一个实数立方根，记作 ∛a（三次根号 a）。

---

## 📚 6-12 岁回溯版

### "找那个'被立方'之前的数"

**立方**是把一个数乘三次：2³ = 2×2×2 = 8。

**立方根**就是反过来问：**什么数立方后等于8？** 答案是2。

**立方根的特别之处：**
- 负数也有立方根！（这和平方根不同）
- (-2)³ = -8，所以 -8 的立方根是 -2
- 正数的立方根是正数，负数的立方根是负数，0的立方根是0

**常用立方根：**
| a | ∛a（立方根） |
|---|------------|
| -27 | -3 |
| -8 | -2 |
| -1 | -1 |
| 0 | 0 |
| 1 | 1 |
| 8 | 2 |
| 27 | 3 |
| 64 | 4 |
| 125 | 5 |

**符号：** ∛a 就是 a 的立方根

**和平方根对比：**
| 比较 | 平方根 | 立方根 |
|------|--------|--------|
| 负数有根吗？ | 没有 | 有 |
| 正数有几个根？ | 两个（±） | 一个 |
| 零的根 | 0 | 0 |

**开立方**：求立方根的运算叫**开立方**，与立方互为逆运算。

## 🎓 12+ 进阶版

### 严格定义

设 a 为任意实数，若 x³ = a，则 x 称为 a 的**立方根**（cube root），记作 x = ∛a。

对每个实数 a，有且仅有一个实数立方根（区别于平方根可能有两个或零个）。

### 完全立方数

整数 n 满足 n = k³（k 为整数），称 n 为**完全立方数**。
例：..., -27, -8, -1, 0, 1, 8, 27, 64, 125, ...

### 性质

| 性质 | 公式 | 说明 |
|------|------|------|
| 唯一性 | 每个实数有唯一立方根 | 含负数 |
| 奇函数性 | ∛(-a) = -∛a | 负数可开立方 |
| 还原性 | (∛a)³ = a | 开方再立方 |
| 立方再开方 | ∛(a³) = a | 立方再开方 |
| 乘法法则 | ∛a · ∛b = ∛(ab) | |
| 除法法则 | ∛a / ∛b = ∛(a/b)（b≠0） | |

注意：∛(a³) = a（不是|a|），因为立方根不需要取绝对值。

### 典型例题

**例题1：** 求 -125 的立方根。

解：因为 (-5)³ = -125
∴ -125 的立方根为 -5，即 ∛(-125) = -5

**例题2：** 化简 ∛(8x³)（x 为任意实数）

解：∛(8x³) = ∛8 · ∛(x³) = 2 · x = 2x

注意：这里 x 可以是负数，立方根不需要绝对值。

**例题3：** 比较 ∛2 与 ∛3 的大小。

解：因为 2 < 3，立方根函数单调递增（负数的立方根也保持顺序），所以 ∛2 < ∛3。

### 与平方根的对比

| 项目 | 平方根 √a | 立方根 ∛a |
|------|---------|---------|
| 要求 | a ≥ 0 | a 任意实数 |
| 个数 | 正数有2个，0有1个 | 恒为1个 |
| 正负性 | 算术平方根 ≥ 0 | 与 a 同号 |
| 简化 | √(a²) = |a| | ∛(a³) = a |

### 易错点

1. **立方根只有一个**，不是 ±（区别于平方根的 ±）
2. **负数有立方根**：∛(-8) = -2（不是"无意义"）
3. **∛(a³) = a，不是|a|**：∛((-2)³) = ∛(-8) = -2

### 知识联系

```
立方根
├── 逆运算 → 立方（a³）
├── 类比 → 平方根（但负数有根，只有一个根）
├── 完全立方数 → 整数根
├── 一般情况 → 无理数
└── n次方根 → 高次根（高中）
```
