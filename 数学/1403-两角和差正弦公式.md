# 两角和差正弦公式

> **一句话**：两角和差的正弦公式为 $\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$，是三角恒等变换中由已知角三角函数值求复合角三角函数值的核心工具。
> **English**: The sine of a sum or difference is given by $\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$, a fundamental tool for finding trigonometric values of compound angles.

## 🎓 考点精讲

**定义/原理**  
对于任意角 $\alpha,\beta$，有  
$$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta,$$  
$$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta.$$  
该公式可通过单位圆中的几何关系或向量数量积推导。其本质是将复合角的正弦拆解为单角正、余弦的线性组合，公式结构与两角差的余弦公式对称但符号不同：正弦同名异号，余弦同名同号。

**关键方法**  
1. **给值求值**：已知 $\alpha,\beta$ 的某个三角函数值（通常需结合 $\sin\alpha,\cos\alpha,\sin\beta,\cos\beta$ 中至少三个值），利用同角关系求出所需值，代入公式求和差正弦。  
2. **角的配凑**：将非特殊角拆成已知三角函数值的特殊角的和或差（如 $15^\circ=45^\circ-30^\circ$ 或 $60^\circ-45^\circ$），直接计算。  
3. **逆向使用**：将 $a\sin x + b\cos x$ 形式通过引入辅助角合成 $R\sin(x+\varphi)$，其逆用即为公式 $R\sin(x+\varphi)=R(\sin x\cos\varphi+\cos x\sin\varphi)$。  
4. **化简与证明**：利用公式将复杂三角函数表达式合并或拆分，实现消元、约简。

**典型应用**  
- **高考真题常见设问**：已知 $\sin\alpha=\frac{3}{5},\cos\beta=-\frac{5}{13}$，且 $\alpha\in(\frac{\pi}{2},\pi),\beta\in(\pi,\frac{3\pi}{2})$，求 $\sin(\alpha+\beta)$。  
- **解题步骤（示例）**：由 $\sin\alpha=\frac{3}{5}$ 及 $\alpha$ 范围得 $\cos\alpha=-\frac{4}{5}$；由 $\cos\beta=-\frac{5}{13}$ 及 $\beta$ 范围得 $\sin\beta=-\frac{12}{13}$；代入 $\sin(\alpha+\beta)=\sin\alpha\cos\beta+\cos\alpha\sin\beta=\frac{3}{5}\times(-\frac{5}{13})+(-\frac{4}{5})\times(-\frac{12}{13})=-\frac{15}{65}+\frac{48}{65}=\frac{33}{65}$。  
- 另一常见角度：化简 $\sin 50^\circ\cos 10^\circ + \cos 50^\circ\sin 10^\circ$，逆用公式直接得 $\sin 60^\circ = \frac{\sqrt{3}}{2}$。

## ⚠️ 高频易错点

- **符号混淆**：把差公式记成 $\sin(\alpha-\beta)=\sin\alpha\cos\beta+\cos\alpha\sin\beta$（正确为减号）。和公式为正号，差公式为负号，与两角和余弦公式相反。
- **三角函数值正负号遗漏**：未根据已知角的象限确定 $\sin$ 或 $\cos$ 的符号，导致整个结果符号错误。
- **同名异名混淆**：与 $\cos(\alpha\pm\beta)=\cos\alpha\cos\beta\mp\sin\alpha\sin\beta$ 相互记错，尤其正弦公式是 $\sin\alpha\cos\beta$ 与 $\cos\alpha\sin\beta$ 的线性组合，易漏项或将系数前后颠倒。
- **角的范围不检验**：求值后忘记检验复合角所在象限，可能导致多解或遗漏限制条件（如三角函数值唯一性问题）。
