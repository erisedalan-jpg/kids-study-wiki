# 解三角不等式

> **一句话**：解三角不等式是求解含有三角函数（如 $ \sin x, \cos x, \tan x $）且带有不等号的不等式，并给出角 $x$ 的取值范围（通常用集合或区间表示）的过程。
> **English**: Solving trigonometric inequalities means finding the sets of angles that satisfy inequalities involving trigonometric functions, such as $\sin x > a$ or $\cos(2x+\varphi) \leq b$.

## 🎓 考点精讲

**定义/原理**  
三角不等式通常形如 $f(\omega x+\varphi) \leq c$ 或 $f(\omega x+\varphi) \geq c$，其中 $f$ 为 $\sin, \cos, \tan$ 等。求解时需依据三角函数的图像与周期性。核心是：先在一个周期内（通常取 $[0,2\pi)$ 或 $(-\pi/2,\pi/2)$ 等基本区间）利用单位圆或函数图像确定基本解集，然后根据函数周期 $T$ 加上 $kT$ 得到通解。对于 $\sin$ 与 $\cos$，利用其最大值 $1$、最小值 $-1$ 及单调区间划分边界；对于 $\tan$，注意其定义域和无限区间。

**关键方法**  
1. **标准化**：将不等式化为 $\sin X < a$ 或 $\cos X \geq b$ 等形式，必要时左右移动项。  
2. **画单位圆或图像**：在 $X$ 的基本区间上标出三角函数值等于 $a$ 的临界角，然后用三角函数线（正弦线、余弦线、正切线）确定满足不等式的角范围。  
3. **求通解**：若 $X=\omega x+\varphi$，先写出 $X$ 的解区间的通式，再解出 $x$ 的范围。例如解 $\sin X > \frac{1}{2}$，由图像得在一个周期内 $\frac{\pi}{6}+2k\pi < X < \frac{5\pi}{6}+2k\pi$，再回代。  
4. **注意定义域**：题目可能给定 $x \in [0,2\pi]$ 等区间，需要在通解中截取相应部分。

**典型应用**  
在高考中，常以选择题或填空题出现，如“在 $[0,2\pi]$ 内，满足 $\sin x \geq \frac{\sqrt{3}}{2}$ 的 $x$ 的集合是______”，需写出 $\left[\frac{\pi}{3}, \frac{2\pi}{3}\right]$。或出现复合型：“求函数 $y=\sqrt{2\cos x -1}$ 的定义域”，转化为解 $2\cos x -1 \geq 0$ 即 $\cos x \geq \frac{1}{2}$，利用图像得 $-\frac{\pi}{3}+2k\pi \leq x \leq \frac{\pi}{3}+2k\pi$。还有与二次不等式结合的情形，如解 $2\sin^2 x - \sin x -1 <0$，先换元为二次不等式求出 $-\frac{1}{2}<\sin x <1$，再解三角不等式。

## ⚠️ 高频易错点

- **忽略周期导致解集不完整**：只写一个周期内的解，没有加 $+2k\pi$ 或 $+k\pi$（对 $\tan$），丢掉了无数个解。  
- **正切函数忽视定义域**：解 $\tan x > a$ 时，需排除 $x = \frac{\pi}{2}+k\pi$ 这些无定义点，易误把无定义点包含进解集。  
- **不等号方向在复合函数中混乱**：当 $\omega<0$ 时，解 $X=\omega x+\varphi$ 的不等式需注意不等号方向改变。例如解 $\sin(-2x+\frac{\pi}{3})>\frac12$，若直接令 $X=-2x+\frac{\pi}{3}$ 求解，回代时乘法除以负数会反号。  
- **边界值处理不当**：题目中是“>”还是“≥”，区间开闭易错。特别当函数值等于极值（如 $\sin x \ge 1$）时解集为单点集而非区间。
