## 课本定义

两个角有公共顶点和一条公共边，且另一边互为反向延长线，这样的两个角叫做**邻补角**。邻补角互补（和为 180°）。

---

## 📚 6-12 岁回溯版

### "挨着的，加起来是180°"

**邻补角 = 邻居角 + 补充到180°**

当两条直线相交，会产生两种角对：
- **对顶角**：对面的两个角（相等）
- **邻补角**：紧挨着的两个角（加起来等于180°）

**怎么认识邻补角？**

```
   A
   |  ∠2
C——O——————D
   |  ∠1
   B
```
- ∠1（在B方向）和∠2（在A方向）是**邻补角**
- 它们共用顶点O，共用射线OC
- ∠1 + ∠2 = 180°（因为AOB是一条直线）

**生活例子：**
- 时针和分针：当时针指向3点，分针指向6点，两根针形成的两个角就是邻补角（一个90°，一个270°……不对，要在一条线上才行）
- 门开了一半：门边和门框形成两个角，加起来是180°（因为门框是直线）
- 折纸：沿一条线折，折线两侧的角是邻补角

**口诀：** "有公共边，在一条线，两角加，等180°"

## 🎓 12+ 进阶版

### 严格定义

**邻补角**：两个角满足以下条件：
1. 有**公共顶点**
2. 有**一条公共边**
3. 另外两条边互为**反向延长线**（构成一条直线）

因第3条，邻补角之和恒为 **180°**（互补）。

### 性质

- ∠α + ∠β = 180°（邻补角互补）
- 若已知 ∠α，则 ∠β = 180° - ∠α
- 邻补角必定相邻（有公共边），但互补角不一定相邻

**区别：互补 vs 邻补**

| 概念 | 条件 | 角的关系 |
|------|------|---------|
| 互补角 | 两角之和=180° | 不要求相邻 |
| 邻补角 | 互补且相邻（公共顶点、公共边、另边共线） | 特殊的互补角 |

### 典型例题

**例题1：** ∠α 与 ∠β 是邻补角，∠α = 2∠β，求两角度数。

解：∠α + ∠β = 180°
2∠β + ∠β = 180°
3∠β = 180°
∠β = 60°，∠α = 120°

**例题2：** 如图，直线AB与直线CD交于O，∠AOC = 40°，求∠AOD、∠BOC、∠BOD。

解：
- ∠AOD = 180° - 40° = 140°（邻补角，COD为直线）
- ∠BOC = ∠AOD = 140°（对顶角）
- ∠BOD = 40°（∠AOC 的对顶角）

### 易错点

1. **互补 ≠ 邻补**：互补只要求和为180°，邻补还要求有公共顶点和公共边
2. **邻补角一定在同一条直线的两侧**，不是随便两个相邻角
3. 邻补角中，"邻"说位置，"补"说数量（互补），两个条件缺一不可

### 知识联系

```
邻补角
├── 特殊的互补角
├── 与对顶角配合 → 角度计算
├── 是平行线证明的基础
└── 由两直线相交产生
```
