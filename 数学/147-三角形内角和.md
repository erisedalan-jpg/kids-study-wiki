# 三角形内角和

> **课本定义**：三角形三个内角的和等于 180°。

---

## 📚 6-12 岁（回溯版）

### 动手验证

拿一张三角形纸片，把三个角剪下来，拼在一起——你会发现它们恰好拼成一条直线，也就是 180°！

这个规律叫做**三角形内角和定理**：

> 任意三角形的三个内角之和 = **180°**

### 应用举例

已知三角形两个角分别是 60° 和 80°，求第三个角。

解：第三个角 = 180° − 60° − 80° = **40°**

### 推广记忆

- 直角三角形：一个角是 90°，其余两个角之和 = 90°（互余）
- 等边三角形：三角相等，每个角 = 180° ÷ 3 = **60°**
- 等腰三角形：两底角相等，顶角 = 180° − 2 × 底角

## 🎓 12+ 进阶版

### 严格证明（平行线辅助）

设 $\triangle ABC$，过顶点 $A$ 作直线 $DE \parallel BC$。

$$\angle DAB = \angle ABC \quad (\text{内错角，}DE \parallel BC)$$
$$\angle EAC = \angle BCA \quad (\text{内错角，}DE \parallel BC)$$

又 $\angle DAB + \angle BAC + \angle EAC = 180°$（直线 $DE$ 上的角）

∴ $\angle ABC + \angle BAC + \angle BCA = 180°$

即 $\angle A + \angle B + \angle C = 180°$ ✓

### 典型例题

**例题 1**：在 $\triangle ABC$ 中，$\angle A = 2\angle B$，$\angle C = 3\angle B$，求三个角。

解：$\angle B + 2\angle B + 3\angle B = 180°$，$6\angle B = 180°$，$\angle B = 30°$

∴ $\angle A = 60°$，$\angle B = 30°$，$\angle C = 90°$（这是一个直角三角形）

**例题 2**：如图，已知 $\triangle ABC$ 中，$\angle A = 50°$，$BD$ 是 $\angle ABC$ 的角平分线，求 $\angle BDC$。

解：设 $\angle ABC = 2\alpha$，则 $\angle DBC = \alpha$

在 $\triangle BDC$ 中：$\angle BDC = 180° - \angle DBC - \angle BCD$

$\angle BCD = \angle BCA = 180° - 50° - 2\alpha = 130° - 2\alpha$

$\angle BDC = 180° - \alpha - (130° - 2\alpha) = 50° + \alpha$

即 $\angle BDC = \angle A + \angle DBC$（外角等于不相邻两内角之和，与[[三角形外角]]定理一致）

### 易错点

1. **内角和定理只适用于三角形**：四边形内角和是 360°，不要混淆。
2. 注意区分"内角"与"外角"：内角在三角形内部，外角在三角形外部。
3. 等腰三角形求角时要考虑**两种情况**（已知角是顶角还是底角）。

### 知识联系

- → [[三角形外角]]：外角 = 180° − 内角，或等于不相邻两内角之和
- → [[多边形内角和外角和]]：$n$ 边形内角和 = $(n-2) \times 180°$
- → [[等腰三角形性质]]：利用内角和求等腰三角形各角
