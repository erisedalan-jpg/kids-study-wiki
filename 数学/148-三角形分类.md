# 三角形分类

> **课本定义**：三角形按角的大小可分为锐角三角形、直角三角形、钝角三角形；按边的长短可分为不等边三角形和等腰三角形（等腰三角形中各边均相等时为等边三角形）。

---

## 📚 6-12 岁（回溯版）

三角形是我们最熟悉的图形之一。你知道吗？根据角和边的不同，三角形可以有很多种！

### 按角分类

| 类型 | 特征 | 例子 |
|------|------|------|
| **锐角三角形** | 三个角都是锐角（< 90°） | 等边三角形 |
| **直角三角形** | 有一个角恰好是 90° | 三角尺 |
| **钝角三角形** | 有一个角是钝角（> 90°） | 斜切的楔形 |

记忆口诀：**三锐、一直、一钝**——三角形只能属于其中一类！

### 按边分类

| 类型 | 特征 |
|------|------|
| **不等边三角形** | 三边长度各不相同 |
| **等腰三角形** | 至少两边相等（相等的两边叫"腰"） |
| **等边三角形** | 三边全相等（特殊的等腰三角形） |

**两种分类可以组合**：一个三角形既可以按角分类，也可以按边分类。例如：等边三角形既是等腰三角形，也是锐角三角形。

### 小思考

一个三角形可以同时是直角三角形和等腰三角形吗？
> 可以！两腰相等、顶角为 90° 的三角形，叫做**等腰直角三角形**，三内角是 45°、45°、90°。

## 🎓 12+ 进阶版

### 严格定义

设三角形三内角为 $\angle A, \angle B, \angle C$，三边为 $a, b, c$（$a$ 对 $\angle A$，以此类推）。

**按角分类**：
- **锐角三角形**：$\angle A < 90°,\ \angle B < 90°,\ \angle C < 90°$
- **直角三角形**：不妨设 $\angle C = 90°$，则 $c^2 = a^2 + b^2$（勾股定理）
- **钝角三角形**：不妨设 $\angle C > 90°$，则 $c^2 > a^2 + b^2$

**按边分类**：
- **不等边三角形**：$a \neq b,\ b \neq c,\ a \neq c$
- **等腰三角形**：至少两边相等，不妨设 $a = b$，则底边为 $c$，腰长为 $a$
- **等边三角形**：$a = b = c$，即正三角形，每个内角均为 60°

### 典型例题

**例题 1**：已知三角形三边长分别为 3、4、5，判断其类型。

解：$3^2 + 4^2 = 9 + 16 = 25 = 5^2$，满足勾股定理，故为**直角三角形**（且三边不等，为不等边三角形）。

**例题 2**：一个等腰三角形的顶角为 120°，判断其类型。

解：顶角为 120° > 90°，故为**钝角三角形**（同时也是等腰三角形）。

### 易错点

1. **三角形只有一种角分类**：一个三角形不可能同时是锐角和直角三角形，三类互斥且完备。
2. **等边三角形是等腰三角形的特例**，不能说"等腰三角形不包括等边三角形"。
3. 判断类型时要用**最大角**：只要有一个角 ≥ 90°，就不是锐角三角形。
