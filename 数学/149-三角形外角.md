# 三角形外角

> **课本定义**：三角形的外角等于与它不相邻的两个内角的和；三角形的外角大于任何一个与它不相邻的内角。

---

## 📚 6-12 岁（回溯版）

### 什么是外角？

在三角形的一条边上，延长某条边后与另一条边形成的角，就叫**外角**。

如图：在 $\triangle ABC$ 中，延长 $BC$ 到点 $D$，则 $\angle ACD$ 就是三角形在顶点 $C$ 处的外角。

每个顶点处有**两个外角**（互为对顶角，相等），每条边两端共 6 个外角，但每顶点外角通常取一个。

### 外角的两个重要性质

**性质 1（外角定理）**：三角形外角 = 不相邻两内角之和

$$\angle ACD = \angle A + \angle B$$

**性质 2**：三角形外角 > 任意一个不相邻内角

$$\angle ACD > \angle A \quad \text{且} \quad \angle ACD > \angle B$$

### 应用举例

已知 $\triangle ABC$ 中 $\angle A = 40°$，$\angle B = 70°$，求 $C$ 点处的外角。

解：$\angle ACD = \angle A + \angle B = 40° + 70° = \mathbf{110°}$

验证：$\angle ACB = 180° - 40° - 70° = 70°$，$\angle ACD = 180° - 70° = 110°$ ✓

## 🎓 12+ 进阶版

### 严格证明

在 $\triangle ABC$ 中，延长 $BC$ 到 $D$，则：

$$\angle ACD + \angle ACB = 180° \quad \text{（直线上的补角）}$$

又由内角和定理：

$$\angle A + \angle B + \angle ACB = 180°$$

两式相减得：

$$\angle ACD = \angle A + \angle B \quad \blacksquare$$

### 典型例题

**例题 1**：在 $\triangle ABC$ 中，$\angle C$ 的外角为 $\angle ACD = 130°$，且 $\angle A = 2\angle B$，求 $\angle A$、$\angle B$。

解：由外角定理，$\angle A + \angle B = 130°$

又 $\angle A = 2\angle B$，代入得 $2\angle B + \angle B = 130°$，$\angle B = \frac{130°}{3} \approx 43.3°$

∴ $\angle A \approx 86.7°$（若题目期待整数解，则条件需调整，此题说明外角定理的代数应用）

**例题 2**：已知 $\triangle ABC$ 中，$\angle BAC = 80°$，$D$ 是 $BC$ 上的点，求证 $\angle ADB > 80°$。

证明：在 $\triangle ABD$ 中，$\angle ADB$ 是 $\angle ADC$ 的邻补角，也是 $\triangle ACD$ 的外角。

由外角定理：$\angle ADB = \angle DAC + \angle ACD > \angle DAC$

且 $\angle BAD + \angle DAC = \angle BAC = 80°$，$\angle ADB = \angle ABD + \angle BAD$

由于 $\angle ABD > 0°$，故 $\angle ADB > \angle BAD$

又 $\angle BAD < 80°$，需要更精细论证……（实际解法：外角 $= $ 不相邻两内角之和 $> $ 单个内角）

### 易错点

1. **外角是延长一条边后形成的角**，不是三角形内部的角。
2. "不相邻"很关键：外角只等于**不相邻**（远端）两内角之和，不包括相邻内角。
3. 每个顶点有两个外角互为**对顶角**，大小相等，通常只算一个。
