# 从分数到分式

> **课本定义**：分数是分子分母均为整数的有理数；分式是分子分母为整式（分母含字母）的代数式。分式是分数在代数域的推广，两者的性质和运算规则高度类比。

---

## 📚 6-12 岁（回溯版）

### 平行对比

分数和分式就像"兄弟"——规则高度相似！

| 概念 | 分数 | 分式 |
|------|------|------|
| 定义 | $\dfrac{a}{b}$（$b$ 为非零整数）| $\dfrac{A}{B}$（$B$ 为含字母整式，$B \neq 0$）|
| 有意义条件 | $b \neq 0$ | $B \neq 0$ |
| 基本性质 | 同乘同除不为零的数，值不变 | 同乘同除不为零的整式，值不变 |
| 约分 | 约去公因数 | 约去公因式 |
| 通分 | 找最小公倍数 | 找最简公分母 |
| 加减 | 通分后分子加减 | 通分后分子加减 |
| 乘法 | 分子乘分子，分母乘分母 | 同左 |
| 除法 | 乘以倒数 | 同左 |

### 从分数推广的思路

**分数**：$\dfrac{3}{4} + \dfrac{1}{4} = \dfrac{4}{4} = 1$

**类比分式**：$\dfrac{x}{x+1} + \dfrac{1}{x+1} = \dfrac{x+1}{x+1} = 1$（$x \neq -1$）

**分数**：$\dfrac{2}{3} \times \dfrac{3}{4} = \dfrac{1}{2}$（约分）

**类比分式**：$\dfrac{x+1}{x-1} \times \dfrac{x-1}{x+2} = \dfrac{x+1}{x+2}$（约分）

### 核心区别

1. 分式有**字母**在分母中，必须考虑使分母为零的情况
2. 分式运算后要**检查是否还有可约分的因式**
3. 分式方程必须**验根**，分数方程不必

## 🎓 12+ 进阶版

### 数学结构的推广

从整数 → 有理数（分数）是数域的扩展：

$$\mathbb{Z} \subset \mathbb{Q} = \left\{\frac{a}{b} \mid a, b \in \mathbb{Z},\ b \neq 0\right\}$$

类似地，从多项式（整式）→ 有理式（分式）：

$$\text{多项式环 } \mathbb{Z}[x] \subset \text{有理函数域 } \mathbb{Q}(x) = \left\{\frac{A(x)}{B(x)} \mid A, B \in \mathbb{Z}[x],\ B \neq 0\right\}$$

这种"推广"是代数学的核心思想之一。

### 类比迁移的边界

并非所有分数的性质都能完全迁移到分式：

| 性质 | 分数 | 分式 |
|------|------|------|
| 大小比较 | 可比较（有序域）| 依赖变量值，不能直接比较 |
| 分母恒为整数 | 是 | 否（可为变量表达式）|
| 约去公因式后限制 | 不需要 | 需要标注 $x \neq \ldots$ |

### 典型例题

**例题 1**：类比 $\dfrac{1}{2} + \dfrac{1}{3} = \dfrac{5}{6}$，写出类似的分式计算。

答：$\dfrac{1}{x} + \dfrac{1}{x+1} = \dfrac{x+1+x}{x(x+1)} = \dfrac{2x+1}{x(x+1)}$（$x \neq 0,-1$）

**例题 2**：判断以下哪些是分式，哪些是分数。

$\dfrac{3x}{2}$（整式，因为分母不含字母）

$\dfrac{2}{x+1}$（分式）

$\dfrac{x^2-1}{3}$（整式，分母为常数）

$\dfrac{x}{x^2+1}$（分式）

**例题 3**：将"分数性质"用于分式：若 $\dfrac{a}{b} = \dfrac{c}{d}$（$bd \neq 0$），则 $ad = bc$。

用于分式：若 $\dfrac{A}{B} = \dfrac{C}{D}$（$BD \neq 0$），则 $AD = BC$（交叉相乘，常用于解分式方程）。

### 易错点

1. $\dfrac{3x}{2}$ **不是分式**：分母 $2$ 不含字母。
2. 分式运算后，别忘了分式定义域的限制。
3. "类比"是工具，不是证明——要真正理解为什么规则相同（都基于等式的等价变换）。
