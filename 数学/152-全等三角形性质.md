# 全等三角形性质

> **课本定义**：全等三角形的对应边相等，对应角相等。

---

## 📚 6-12 岁（回溯版）

### 核心性质

两个三角形全等（$\triangle ABC \cong \triangle DEF$），那么：

**对应边相等**：
$$AB = DE, \quad BC = EF, \quad CA = FD$$

**对应角相等**：
$$\angle A = \angle D, \quad \angle B = \angle E, \quad \angle C = \angle F$$

### 如何找对应关系？

关键：**全等式中对应位置的字母就是对应顶点**。

$\triangle ABC \cong \triangle DEF$ → 第一个字母 A 对 D，第二个 B 对 E，第三个 C 对 F。

### 全等三角形的用途

全等三角形性质让我们能从"已知一个三角形的某些信息"推出"另一个三角形的对应信息"。这在几何证明中极为常用：

**套路**：先证明两个三角形全等（用判定方法），再利用性质得到对应边相等或对应角相等。

### 例子

已知 $\triangle ABC \cong \triangle PQR$，$BC = 6$，$\angle Q = 45°$，求 $EF$ 和 $\angle B$。

（注：此处 $EF$ 应为 $QR$）$QR$ 对应 $BC$，∴ $QR = BC = 6$；$\angle B$ 对应 $\angle Q$，∴ $\angle B = 45°$

## 🎓 12+ 进阶版

### 性质的完整表述

若 $\triangle ABC \cong \triangle DEF$，则：

**三对对应边**：$AB = DE$，$BC = EF$，$CA = FD$

**三对对应角**：$\angle BAC = \angle EDF$，$\angle ABC = \angle DEF$，$\angle BCA = \angle EFD$

**面积相等**：$S_{\triangle ABC} = S_{\triangle DEF}$

**周长相等**：$AB + BC + CA = DE + EF + FD$

### 全等三角形在证明中的应用框架

```
目标：证明某两条线段相等（或某两个角相等）
步骤：
1. 找到包含这两条线段（或角）的两个三角形
2. 证明这两个三角形全等（SSS / SAS / ASA / AAS / HL）
3. 由全等得对应元素相等
```

### 典型例题

**例题 1**：如图，$AB = CD$，$AD = CB$，证明 $\angle ABD = \angle CDB$。

证明：在 $\triangle ABD$ 和 $\triangle CDB$ 中：

$AB = CD$（已知），$AD = CB$（已知），$BD = DB$（公共边）

由 SSS，$\triangle ABD \cong \triangle CDB$

∴ $\angle ABD = \angle CDB$（对应角相等）✓

**例题 2**：已知 $O$ 是线段 $AC$、$BD$ 的交点，$OA = OC$，$OB = OD$，证明 $AB = CD$。

证明：在 $\triangle AOB$ 和 $\triangle COD$ 中：

$OA = OC$（已知），$OB = OD$（已知），$\angle AOB = \angle COD$（对顶角）

由 SAS，$\triangle AOB \cong \triangle COD$

∴ $AB = CD$（对应边相等）✓

### 易错点

1. **对应顶点顺序决定对应关系**：写全等式时不能随意排列字母顺序。
2. 全等三角形性质是"从全等推出边角相等"，**不能反过来**（从边角相等直接得到全等，还需要满足判定条件）。
3. 证明全等要先找到**三个**满足判定条件的对应元素。
