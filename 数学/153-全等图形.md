# 全等图形

> **课本定义**：能够完全重合的两个图形叫做**全等图形**。全等图形的形状和大小都相同，对应边相等，对应角相等。

---

## 📚 6-12 岁（回溯版）

### 什么是全等？

如果把两个图形中的一个搬过去（可以翻转、旋转、平移），能够和另一个**完全重合**，那么这两个图形就是**全等图形**，用符号 **≅** 表示。

**全等 = 形状相同 + 大小相同**

| 操作 | 举例 |
|------|------|
| 平移 | 棋盘上的两个同颜色棋子 |
| 旋转 | 时钟的分针和时针（长度不同，不全等！） |
| 翻转（镜像） | 左手套和右手套（镜像全等） |

### 对应元素

把两个全等图形"重合"时，重合在一起的顶点叫**对应顶点**，重合在一起的边叫**对应边**，重合在一起的角叫**对应角**。

全等图形的对应边相等，对应角相等。

### 符号规范

$\triangle ABC \cong \triangle DEF$

表示 $A$ 对应 $D$，$B$ 对应 $E$，$C$ 对应 $F$。

∴ $AB = DE$，$BC = EF$，$CA = FD$；$\angle A = \angle D$，$\angle B = \angle E$，$\angle C = \angle F$

**注意**：书写全等式时，对应顶点的字母顺序必须一致！

## 🎓 12+ 进阶版

### 严格定义

两个图形 $F_1$ 和 $F_2$ 全等，当且仅当存在一个**等距变换**（距离保持不变的变换：平移、旋转、反射或其组合），使得 $F_1$ 经该变换后与 $F_2$ 完全重合。

等距变换保持：
- 线段长度
- 角度大小
- 面积

### 全等与相似的区别

| | 全等 | 相似 |
|-|------|------|
| 形状 | 相同 | 相同 |
| 大小 | 相同 | 可以不同 |
| 对应边 | 相等 | 成比例 |
| 对应角 | 相等 | 相等 |
| 比例系数 | 1 | 任意正数 |

全等是相似的特例（比例系数为 1）。

### 典型例题

**例题 1**：已知 $\triangle ABC \cong \triangle DEF$，$AB = 5$，$\angle B = 60°$，$EF = 8$，求 $BC$ 和 $\angle E$。

解：由全等，对应顶点 $A↔D$，$B↔E$，$C↔F$

$BC$ 对应 $EF$，∴ $BC = EF = 8$

$\angle B$ 对应 $\angle E$，∴ $\angle E = \angle B = 60°$

**例题 2**：如图，$\triangle ABC \cong \triangle DBC$（共用边 $BC$），$\angle ABD$ 的平分线交 $AD$ 于点 $M$，求证 $AM = DM$。

解：由全等 $\triangle ABC \cong \triangle DBC$，得 $AB = DB$

∴ $\triangle ABM \cong \triangle DBM$（$BM$ 平分 $\angle ABD$，$AB = DB$，$BM = BM$，SAS）

∴ $AM = DM$ ✓

### 易错点

1. **书写全等式必须保持对应顶点顺序**：$\triangle ABC \cong \triangle DEF$ 与 $\triangle ABC \cong \triangle EDF$ 含义不同。
2. **全等 ≠ 形状相同**：形状相同但大小不同是相似，不是全等。
3. 镜像（反射）后也可以全等，不要因为"翻转了"就认为不全等。
