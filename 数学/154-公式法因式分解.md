# 公式法因式分解

> **课本定义**：利用乘法公式（平方差公式、完全平方公式等）逆向进行因式分解，叫做**公式法**。$a^2 - b^2 = (a+b)(a-b)$；$a^2 + 2ab + b^2 = (a+b)^2$；$a^2 - 2ab + b^2 = (a-b)^2$。

---

## 📚 6-12 岁（回溯版）

### 公式法的思路

乘法公式是"展开"，因式分解是"收缩"——把多项式变回乘积形式。

**平方差公式逆用**：

$$a^2 - b^2 = (a+b)(a-b)$$

识别特征：**两个平方的差**（中间是减号，两边都是完全平方）

**完全平方公式逆用**：

$$a^2 + 2ab + b^2 = (a+b)^2$$
$$a^2 - 2ab + b^2 = (a-b)^2$$

识别特征：三项式，首末是完全平方，中间项是首末项平方根乘积的 2 倍。

### 识别口诀

**平方差**：两平方，中减号 → $(a+b)(a-b)$

**完全平方式**：三项式，首末方，中间两倍积 → $(a \pm b)^2$

### 例题练习

| 多项式 | 类型 | 分解结果 |
|--------|------|----------|
| $x^2 - 9$ | 平方差（$x^2 - 3^2$） | $(x+3)(x-3)$ |
| $4a^2 - b^2$ | 平方差（$(2a)^2 - b^2$） | $(2a+b)(2a-b)$ |
| $x^2 + 4x + 4$ | 完全平方（$(x+2)^2$） | $(x+2)^2$ |
| $9x^2 - 6x + 1$ | 完全平方（$(3x-1)^2$） | $(3x-1)^2$ |

## 🎓 12+ 进阶版

### 判断是否为完全平方式

对三项式 $Ax^2 + Bx + C$（$A, C > 0$）：

完全平方式 ↔ $B^2 = 4AC$（即判别式 $\Delta = 0$）

**例**：$x^2 + 6x + 9$：$A=1,B=6,C=9$，$B^2 = 36 = 4 \times 9 = 4AC$ ✓

### 综合运用：先提公因式，再用公式

**规则**：先提出公因式，再对括号内应用公式法。

**例**：分解 $2x^3 - 8x$

$= 2x(x^2 - 4)$（提公因式）

$= 2x(x+2)(x-2)$（平方差公式）

### 典型例题

**例题 1**：分解 $(a+b)^2 - (a-b)^2$。

解：令 $M = a+b$，$N = a-b$，则原式 = $M^2 - N^2 = (M+N)(M-N)$

$M + N = (a+b)+(a-b) = 2a$

$M - N = (a+b)-(a-b) = 2b$

∴ 原式 $= 2a \times 2b = 4ab$

**例题 2**：分解 $x^4 - 1$。

解：$x^4 - 1 = (x^2)^2 - 1^2 = (x^2+1)(x^2-1) = (x^2+1)(x+1)(x-1)$

（注：$x^2+1$ 在实数范围内不可再分解）

**例题 3**：已知 $a^2 + b^2 = 10$，$ab = 3$，求 $(a+b)^2$ 和 $(a-b)^2$。

解：$(a+b)^2 = a^2 + 2ab + b^2 = 10 + 6 = 16$

$(a-b)^2 = a^2 - 2ab + b^2 = 10 - 6 = 4$

### 易错点

1. **$a^2 + b^2$ 不能分解**：平方和在实数范围内不可因式分解（不是平方差！）
2. 识别完全平方式时，注意系数：$4x^2 + 4x + 1 = (2x+1)^2$，$a$ 是 $2x$。
3. 综合分解时，**先提公因式**，再用公式，不要遗漏步骤。
