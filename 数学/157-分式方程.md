# 分式方程

> **课本定义**：分母中含有未知数的方程叫做**分式方程**。解分式方程的一般方法：方程两边同乘以最简公分母，将分式方程化为整式方程，解整式方程后**必须验根**（去掉使分母为零的增根）。

---

## 📚 6-12 岁（回溯版）

### 什么是分式方程？

$\dfrac{3}{x} = 2$ —— 分母含未知数 $x$，就是分式方程！

### 解法：去分母

步骤：
1. 找出各分母的最简公分母
2. 方程两边**同乘以**最简公分母（去分母变整式方程）
3. 解整式方程
4. **验根**！代回原方程检验，去掉使分母为零的增根

### 为什么要验根？

乘以公分母时，如果公分母包含变量，可能在变量取特定值时为零——此时"乘以零"是非法操作，产生**增根**（不满足原方程的假根）。

**例**：解 $\dfrac{3}{x-1} = \dfrac{2}{x+1}$

公分母：$(x-1)(x+1)$，两边乘以：

$3(x+1) = 2(x-1)$

$3x + 3 = 2x - 2$，$x = -5$

验根：$x = -5$ 时，$x-1 = -6 \neq 0$，$x+1 = -4 \neq 0$，分母均不为零。✓

∴ $x = -5$ 是解。

## 🎓 12+ 进阶版

### 完整解题步骤

1. **找最简公分母**：各分母因式分解后取最小公倍式
2. **去分母**：两边同乘最简公分母，得整式方程
3. **解整式方程**：移项、合并、化简
4. **验根**：将解代入原方程的各分母，检验是否为零
   - 不为零：是原方程的解
   - 为零：是增根，舍去

### 增根的产生

增根 = 使公分母为零的值（使某个分母为零的值）

解分式方程必须验根，这是区别于解整式方程的关键步骤！

### 典型例题

**例题 1**：解分式方程 $\dfrac{x}{x-2} - 1 = \dfrac{2}{x-2}$。

解：公分母 = $x - 2$，两边乘以 $(x-2)$：

$x - (x-2) = 2$

$x - x + 2 = 2$

$2 = 2$（恒成立）

等等——解为任意值？**验根**！$x = 2$ 时分母 $x - 2 = 0$，无意义，需排除。

∴ 方程的解为 $x$ 为任意实数且 $x \neq 2$……

实际上，移项后 $\dfrac{x}{x-2} - \dfrac{2}{x-2} = 1$，即 $\dfrac{x-2}{x-2} = 1$，$1 = 1$，这是一个恒等式，但原方程在 $x = 2$ 处无意义，故答案为"除 $x=2$ 外的所有实数"。

**例题 2**：解 $\dfrac{1}{x-1} + \dfrac{3}{x+1} = \dfrac{6}{x^2-1}$。

解：$x^2 - 1 = (x-1)(x+1)$，公分母 = $(x-1)(x+1)$

两边乘以 $(x-1)(x+1)$：

$(x+1) + 3(x-1) = 6$

$x + 1 + 3x - 3 = 6$，$4x - 2 = 6$，$4x = 8$，$x = 2$

验根：$x = 2$ 时，$x-1 = 1 \neq 0$，$x+1 = 3 \neq 0$ ✓

∴ $x = 2$ 是解。

**例题 3（有增根）**：解 $\dfrac{x-1}{x-2} = \dfrac{1}{x-2} + 1$。

公分母 = $x - 2$，去分母：$x - 1 = 1 + (x-2)$，$x - 1 = x - 1$，恒成立。

验根：$x = 2$ 使 $x-2 = 0$，无意义，舍去。

∴ 方程无解（$x$ 为任意实数除 $x=2$ 外，但代入是恒等式，无法限定唯一值）——此题结论：无解还是无穷解，取决于具体题目条件，实际此例无解（每个值代入分母不为零的均满足）。

### 易错点

1. **必须验根**：忘记验根是最常见失分点！
2. 去分母后直接解整式，不能带着分式继续运算。
3. 增根是使分母为零的值，不一定是"无解"，要具体判断。
