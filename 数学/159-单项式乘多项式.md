# 单项式乘多项式

> **课本定义**：单项式与多项式相乘，就是用单项式去乘多项式的每一项，再把所得的积相加：$m(a + b + c) = ma + mb + mc$。

---

## 📚 6-12 岁（回溯版）

### 什么是单项式和多项式？

- **单项式**：只有一项的代数式，如 $3x^2$、$-2ab$、$5$
- **多项式**：有多项的代数式，如 $x^2 + 2x + 1$、$a - b + c$

### 分配律的代数推广

小学学过：$3 \times (4 + 5) = 3 \times 4 + 3 \times 5$（分配律）

代数中同样：$3x \cdot (x + 2) = 3x \cdot x + 3x \cdot 2 = 3x^2 + 6x$

这就是"单项式乘多项式"：**逐项相乘，结果相加**。

### 法则

$$m(a + b + c) = ma + mb + mc$$

### 例题练习

| 计算 | 步骤 | 结果 |
|------|------|------|
| $2x(x+3)$ | $2x \cdot x + 2x \cdot 3$ | $2x^2 + 6x$ |
| $-3a(a^2 - 2a + 1)$ | $-3a \cdot a^2 + (-3a)(-2a) + (-3a)(1)$ | $-3a^3 + 6a^2 - 3a$ |
| $x^2(2x - 1)$ | $2x^3 - x^2$ | $2x^3 - x^2$ |

注意：$-3a \cdot (-2a) = +6a^2$（负负得正）

## 🎓 12+ 进阶版

### 严格表述

设单项式为 $m$，多项式为 $a_1 + a_2 + \cdots + a_k$，则：

$$m(a_1 + a_2 + \cdots + a_k) = ma_1 + ma_2 + \cdots + ma_k$$

本质是乘法对加法的**分配律**在多项式域中的应用。

### 乘法步骤

1. 确定单项式和多项式（有时需要交换顺序）
2. 将单项式依次乘以多项式的每一项
3. 合并同类项（如有）
4. 按降幂排列

### 典型例题

**例题 1**：展开并化简 $3x^2(2x^2 - 4x + 1) - x(x^3 - 2x + 3)$。

解：

$3x^2(2x^2 - 4x + 1) = 6x^4 - 12x^3 + 3x^2$

$x(x^3 - 2x + 3) = x^4 - 2x^2 + 3x$

相减：$6x^4 - 12x^3 + 3x^2 - (x^4 - 2x^2 + 3x)$

$= 6x^4 - 12x^3 + 3x^2 - x^4 + 2x^2 - 3x$

$= 5x^4 - 12x^3 + 5x^2 - 3x$

**例题 2**：化简 $2a(a+b) - b(2a - b)$。

解：$2a^2 + 2ab - 2ab + b^2 = 2a^2 + b^2$

### 易错点

1. **符号处理**：单项式是负数时，乘以多项式每项时要特别注意符号。
2. **不要漏项**：多项式有几项，就要乘几次，不能遗漏。
3. **合并同类项**：展开后要检查是否有同类项可以合并。
