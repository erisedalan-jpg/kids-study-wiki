# 多项式乘多项式

> **课本定义**：多项式与多项式相乘，先用一个多项式的每一项乘以另一个多项式的每一项，再把所得的积相加：$(a+b)(c+d) = ac + ad + bc + bd$。

---

## 📚 6-12 岁（回溯版）

### 面积模型理解

把 $(a+b)(c+d)$ 想象成一个矩形的面积：

```
      c        d
  ┌─────────┬───────┐
a │   ac    │  ad   │
  ├─────────┼───────┤
b │   bc    │  bd   │
  └─────────┴───────┘
```

整个矩形面积 = $ac + ad + bc + bd$

### 法则

$$(a+b)(c+d) = ac + ad + bc + bd$$

**FOIL 记忆法**（英语记忆口诀）：
- **F**irst（首×首）：$ac$
- **O**uter（外×外）：$ad$
- **I**nner（内×内）：$bc$
- **L**ast（末×末）：$bd$

### 例题练习

| 计算 | 展开 | 化简 |
|------|------|------|
| $(x+2)(x+3)$ | $x^2+3x+2x+6$ | $x^2+5x+6$ |
| $(x-1)(x+4)$ | $x^2+4x-x-4$ | $x^2+3x-4$ |
| $(2x+1)(x-3)$ | $2x^2-6x+x-3$ | $2x^2-5x-3$ |
| $(x+y)(x-y)$ | $x^2-xy+xy-y^2$ | $x^2-y^2$ ← 平方差！|

## 🎓 12+ 进阶版

### 严格表述

设两多项式 $P = a_1 + a_2 + \cdots + a_m$，$Q = b_1 + b_2 + \cdots + b_n$，则：

$$P \cdot Q = \sum_{i=1}^{m} \sum_{j=1}^{n} a_i b_j$$

即 $m \times n$ 项之积相加，再合并同类项。

### 结果的项数

$(m\text{项}) \times (n\text{项})$ 展开后有 $mn$ 项，合并同类项后可能减少。

常见：$(2\text{项}) \times (2\text{项})$ 展开 4 项，合并后通常 3 项（二次三项式）。

### 典型例题

**例题 1**：展开 $(2x-3)(3x+4)$。

解：
$= 2x \cdot 3x + 2x \cdot 4 + (-3) \cdot 3x + (-3) \cdot 4$

$= 6x^2 + 8x - 9x - 12$

$= 6x^2 - x - 12$

**例题 2**：展开 $(x^2 + x - 1)(x - 2)$。

解：
$= x^2 \cdot x + x \cdot x + (-1) \cdot x + x^2 \cdot (-2) + x \cdot (-2) + (-1) \cdot (-2)$

$= x^3 + x^2 - x - 2x^2 - 2x + 2$

$= x^3 - x^2 - 3x + 2$

### 特殊情形

$(a+b)^2 = a^2 + 2ab + b^2$（完全平方公式，展开自己乘自己）

$(a+b)(a-b) = a^2 - b^2$（平方差公式，相加相减）

这两个特殊公式由多项式乘多项式推导，详见[[166-平方差公式|平方差公式]]和[[163-完全平方公式|完全平方公式]]。

### 易错点

1. **不能漏项**：第一个多项式的每一项都要乘第二个多项式的每一项。
2. **符号要随项走**：$(x-3)$ 的第二项是 $-3$，不是 $3$。
3. **展开后合并同类项**：展开只是第一步，还要整理！
