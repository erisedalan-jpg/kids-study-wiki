# 完全平方公式

> **课本定义**：两数之和（差）的平方，等于这两数的平方和加上（减去）这两数乘积的 2 倍：$(a+b)^2 = a^2 + 2ab + b^2$；$(a-b)^2 = a^2 - 2ab + b^2$。

---

## 📚 6-12 岁（回溯版）

### 图形理解

把 $(a+b)^2$ 想象成边长为 $(a+b)$ 的正方形面积：

```
        a         b
  ┌──────────┬────────┐
a │   a²     │   ab   │
  ├──────────┼────────┤
b │   ab     │   b²   │
  └──────────┴────────┘
总面积 = a² + ab + ab + b² = a² + 2ab + b²
```

### 公式

$$\boxed{(a+b)^2 = a^2 + 2ab + b^2}$$
$$\boxed{(a-b)^2 = a^2 - 2ab + b^2}$$

记忆口诀：**首平方，尾平方，首尾两倍放中央。**

- $(a+b)^2$：中间项是 $+2ab$
- $(a-b)^2$：中间项是 $-2ab$

### 正向使用（展开）

| 计算 | 结果 |
|------|------|
| $(x+3)^2$ | $x^2 + 6x + 9$ |
| $(x-4)^2$ | $x^2 - 8x + 16$ |
| $(2a+b)^2$ | $4a^2 + 4ab + b^2$ |
| $(3x - 2y)^2$ | $9x^2 - 12xy + 4y^2$ |

### 逆向使用（因式分解）

| 分解 | 结果 |
|------|------|
| $x^2 + 6x + 9$ | $(x+3)^2$ |
| $4a^2 - 4ab + b^2$ | $(2a-b)^2$ |

## 🎓 12+ 进阶版

### 严格推导

$$(a+b)^2 = (a+b)(a+b) = a(a+b) + b(a+b) = a^2 + ab + ab + b^2 = a^2 + 2ab + b^2 \quad \blacksquare$$

$$(a-b)^2 = (a-b)(a-b) = a^2 - ab - ab + b^2 = a^2 - 2ab + b^2 \quad \blacksquare$$

注意：$(a-b)^2 = [a+(-b)]^2 = a^2 + 2a(-b) + (-b)^2 = a^2 - 2ab + b^2$，与 $(a+b)^2$ 公式的关系。

### 推广：三项完全平方

$$(a+b+c)^2 = a^2 + b^2 + c^2 + 2ab + 2bc + 2ca$$

推广规律：每两个不同项之积乘以 2，加上每项的平方。

### 配方法（完全平方公式的逆用）

将二次式 $x^2 + px + q$ 变形为完全平方式：

$$x^2 + px + q = \left(x + \frac{p}{2}\right)^2 - \frac{p^2}{4} + q$$

配方法是解一元二次方程和讨论二次函数的重要工具（九年级使用）。

### 典型例题

**例题 1**：已知 $a + b = 5$，$ab = 6$，求 $a^2 + b^2$。

解：$(a+b)^2 = a^2 + 2ab + b^2$

$25 = a^2 + b^2 + 12$，∴ $a^2 + b^2 = 13$

**例题 2**：分解因式 $x^2 - 10x + 25$。

解：$x^2 - 10x + 25 = x^2 - 2 \cdot x \cdot 5 + 5^2 = (x-5)^2$

**例题 3**：不计算，直接写出 $101^2$ 的值。

解：$101^2 = (100+1)^2 = 10000 + 200 + 1 = 10201$

### 易错点

1. **中间项系数是 2**：$(a+b)^2 \neq a^2 + b^2$（漏掉 $2ab$ 是最常见错误！）
2. **$(a-b)^2 \geq 0$**：任何实数的平方非负，等号当 $a = b$ 时成立。
3. 识别完全平方式：$a^2 \pm 2ab + b^2$ 的特征是"首末项是完全平方数，中间项是首末平方根乘积的 2 倍"。
