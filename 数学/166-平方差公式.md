# 平方差公式

> **课本定义**：两数之和乘以这两数之差，等于这两数的平方差：$(a+b)(a-b) = a^2 - b^2$。

---

## 📚 6-12 岁（回溯版）

### 发现规律

$(x+3)(x-3) = x^2 - 3x + 3x - 9 = x^2 - 9$

中间两项 $-3x$ 和 $+3x$ 恰好相消！

### 公式

$$\boxed{(a+b)(a-b) = a^2 - b^2}$$

读作：**和差乘积等于平方差**。

图形理解：

```
大正方形 a² 减去小正方形 b²
= 一个 L 形区域
= (a+b)(a-b)（将 L 形重组为长方形）
```

### 正向使用（乘法展开）

| 计算 | 结果 |
|------|------|
| $(x+5)(x-5)$ | $x^2 - 25$ |
| $(2a+3b)(2a-3b)$ | $4a^2 - 9b^2$ |
| $(1+\sqrt{2})(1-\sqrt{2})$ | $1 - 2 = -1$ |

### 逆向使用（因式分解）

| 分解 | 结果 |
|------|------|
| $x^2 - 16$ | $(x+4)(x-4)$ |
| $9a^2 - 4b^2$ | $(3a+2b)(3a-2b)$ |

### 速算应用

$99 \times 101 = (100-1)(100+1) = 100^2 - 1^2 = 10000 - 1 = 9999$（口算！）

## 🎓 12+ 进阶版

### 严格推导

$$(a+b)(a-b) = a(a-b) + b(a-b) = a^2 - ab + ab - b^2 = a^2 - b^2 \quad \blacksquare$$

### 推广形式

**多项形式**：$a$ 和 $b$ 可以是任意代数式。

- $[(x+y)+z][(x+y)-z] = (x+y)^2 - z^2$
- $[a^2+b^2][a^2-b^2] = a^4 - b^4$（两次应用）

**连续平方差**：

$$a^4 - b^4 = (a^2+b^2)(a^2-b^2) = (a^2+b^2)(a+b)(a-b)$$

### 有理化分母（分式运算应用）

$$\frac{1}{\sqrt{2}+1} = \frac{\sqrt{2}-1}{(\sqrt{2}+1)(\sqrt{2}-1)} = \frac{\sqrt{2}-1}{2-1} = \sqrt{2}-1$$

### 典型例题

**例题 1**：计算 $\dfrac{1}{1 \times 3} + \dfrac{1}{3 \times 5} + \dfrac{1}{5 \times 7} + \cdots + \dfrac{1}{99 \times 101}$。

解：利用 $\dfrac{1}{(2k-1)(2k+1)} = \dfrac{1}{2}\left(\dfrac{1}{2k-1} - \dfrac{1}{2k+1}\right)$（裂项）

这是高中内容，初中主要关注基础应用。

**例题 2**：分解因式 $16x^4 - 81y^4$。

解：$16x^4 - 81y^4 = (4x^2)^2 - (9y^2)^2 = (4x^2 + 9y^2)(4x^2 - 9y^2)$

$= (4x^2 + 9y^2)(2x+3y)(2x-3y)$

### 易错点

1. **公式要求同一个数的和与差**：$(a+b)(a-b)$，不是 $(a+b)(c-d)$。
2. **平方差只有减号**：$a^2 - b^2$，不是 $a^2 + b^2$（两平方之和不能分解为实数乘积）。
3. 识别时注意 $9a^2 = (3a)^2$，$25 = 5^2$，要把系数也平方。
