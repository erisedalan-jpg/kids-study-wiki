# 提公因式法

> **课本定义**：如果一个多项式的各项都含有公因式，就可以把这个公因式提取出来，从而将多项式化成两个因式乘积的形式——这种因式分解方法叫做**提公因式法**。

---

## 📚 6-12 岁（回溯版）

### 公因式是什么？

就像找整数的公因数：$12$ 和 $18$ 的最大公因数是 $6$。

代数式中，各项共有的**最大**因式叫**公因式（最大公因子）**。

### 提公因式的步骤

1. 找出各项的公因式（系数取最大公因数，字母取最小次数）
2. 把公因式"提"到括号外
3. 括号内各项 = 原项 ÷ 公因式

### 例题练习

**例 1**：$6x^2 + 9x$

公因式：系数 $\gcd(6,9)=3$，字母 $x$（最低次），公因式 = $3x$

$6x^2 + 9x = 3x \cdot 2x + 3x \cdot 3 = 3x(2x+3)$

**例 2**：$12a^2b - 8ab^2 + 4ab$

公因式：$\gcd(12,8,4)=4$，字母 $ab$，公因式 = $4ab$

$= 4ab(3a - 2b + 1)$

**例 3**：$-x^2 + x$

公因式 $x$，但通常让括号内最高次项系数为正：

$= -x(x - 1)$ 或 $x(-x + 1)$（一般取前者）

### 验证方法

展开回去看是否等于原式：$3x(2x+3) = 6x^2 + 9x$ ✓

## 🎓 12+ 进阶版

### 严格定义

设多项式 $f = a_1 M + a_2 M + \cdots + a_k M$，其中 $M$ 是各项的最大公因式，则：

$$f = M(a_1 + a_2 + \cdots + a_k)$$

最大公因式 $M$ 的确定：
- **系数部分**：各项系数的最大公因数（GCF）
- **字母部分**：各项共有字母，取**最低次**

### 含括号的公因式

有时公因式是一个多项式：

**例**：$a(x+y) + b(x+y) = (x+y)(a+b)$

这里 $(x+y)$ 是公因式，整体提取。

**例**：$m(m-1) - 2(1-m)$

注意 $(1-m) = -(m-1)$，故：

$= m(m-1) + 2(m-1) = (m-1)(m+2)$

### 典型例题

**例题 1**：分解 $x^3y - x^2y^2 + xy^3$。

解：公因式 = $xy$（系数均含，字母最低次各为 $x^1y^1$）

$= xy(x^2 - xy + y^2)$

**例题 2**：分解 $(a+b)^2 - c(a+b)$。

解：公因式 = $(a+b)$

$= (a+b)[(a+b) - c] = (a+b)(a+b-c)$

**例题 3**：分解 $x(x-2) + (2-x)^2$。

注意 $(2-x) = -(x-2)$，故 $(2-x)^2 = (x-2)^2$

$= x(x-2) + (x-2)^2 = (x-2)[x + (x-2)] = (x-2)(2x-2) = 2(x-2)(x-1)$

### 易错点

1. **符号陷阱**：提出负号后，括号内各项符号要反转。
2. **公因式要取最大的**：如果提出的公因式不是最大公因式，还可以继续分解。
3. **括号内常数项**：提完公因式后，若某项等于公因式本身，则括号内对应项为 $1$，不能漏掉。
