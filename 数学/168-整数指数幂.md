# 整数指数幂

> **课本定义**：规定 $a^0 = 1$（$a \neq 0$）；$a^{-n} = \dfrac{1}{a^n}$（$a \neq 0$，$n$ 为正整数）。整数指数幂将幂运算从正整数指数推广到所有整数指数，运算法则与正整数情形一致。

---

## 📚 6-12 岁（回溯版）

### 规律发现：幂的变化

$2^4 = 16$，$2^3 = 8$，$2^2 = 4$，$2^1 = 2$……每次指数减 1，结果÷2

继续：$2^0 = ?$，按规律应该是 $2 \div 2 = 1$

再继续：$2^{-1} = 1 \div 2 = \dfrac{1}{2}$，$2^{-2} = \dfrac{1}{4} = \dfrac{1}{2^2}$

### 零次幂

$$\boxed{a^0 = 1 \quad (a \neq 0)}$$

**任何不为零的数（或式）的 0 次幂都等于 1！**

注意：$0^0$ 无意义（数学上有争议，中学不讨论）

### 负整数次幂

$$\boxed{a^{-n} = \frac{1}{a^n} \quad (a \neq 0)}$$

| 幂 | 值 |
|----|-----|
| $10^{-1}$ | $0.1 = \dfrac{1}{10}$ |
| $10^{-2}$ | $0.01 = \dfrac{1}{100}$ |
| $2^{-3}$ | $\dfrac{1}{8}$ |
| $x^{-2}$ | $\dfrac{1}{x^2}$ |

### 科学记数法（负指数版）

$0.001 = 10^{-3}$，$0.000001 = 10^{-6}$

$3.6 \times 10^{-4} = 0.00036$（小数点向左移 4 位）

## 🎓 12+ 进阶版

### 严格定义与动机

对于正整数 $m > n$，$a^m \div a^n = a^{m-n}$（同底数幂的除法）。

若令 $m = n$：$a^n \div a^n = 1$，但 $a^{m-n} = a^0$，故**定义** $a^0 = 1$。

若令 $m < n$，设 $n = m + k$（$k > 0$）：

$a^m \div a^{m+k} = \dfrac{1}{a^k}$，但 $a^{m-m-k} = a^{-k}$，故**定义** $a^{-k} = \dfrac{1}{a^k}$。

这种定义保证了**所有整数指数**下，幂的运算法则（同底数幂相乘/除）均成立。

### 整数指数幂的运算法则（全版）

设 $a \neq 0$，$b \neq 0$，$m, n$ 为任意整数：

1. $a^m \cdot a^n = a^{m+n}$
2. $(a^m)^n = a^{mn}$
3. $(ab)^n = a^n b^n$
4. $a^m \div a^n = a^{m-n}$
5. $\left(\dfrac{a}{b}\right)^n = \dfrac{a^n}{b^n}$

### 典型例题

**例题 1**：计算 $(2^{-2} \times 3^0 + 2^{-1}) \div 2^{-3}$。

解：

$2^{-2} = \dfrac{1}{4}$，$3^0 = 1$，$2^{-1} = \dfrac{1}{2}$，$2^{-3} = \dfrac{1}{8}$

$= \left(\dfrac{1}{4} \times 1 + \dfrac{1}{2}\right) \div \dfrac{1}{8}$

$= \dfrac{3}{4} \times 8 = 6$

**例题 2**：化简 $\dfrac{x^{-2} y^3}{x y^{-1}}$。

解：$= x^{-2-1} y^{3-(-1)} = x^{-3} y^4 = \dfrac{y^4}{x^3}$

### 易错点

1. **$0^0$ 没有意义**，不要认为 $0^0 = 1$。
2. $a^{-n} = \dfrac{1}{a^n}$，负指数是倒数，不是"负数"。
3. 负数的负次幂：$(-2)^{-2} = \dfrac{1}{(-2)^2} = \dfrac{1}{4}$（先算底数，再取倒数）。
4. $a^0 = 1$ 要求 $a \neq 0$：$0^0$ 无意义，不要直接说"任何数的 0 次方等于 1"。
