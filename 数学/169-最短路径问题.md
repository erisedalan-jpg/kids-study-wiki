# 最短路径问题

> **课本定义**：利用轴对称原理，将某点关于直线（障碍线/河岸等）作对称点，连接对称点与目标点的线段与直线的交点，即为最优中转点，使路径最短。

---

## 📚 6-12 岁（回溯版）

### 核心思想

**两点之间线段最短。**

当路径需要经过某条直线（比如河岸），把出发点或终点**关于这条线做对称**，然后连成一条线——直线路径就是最短的！

### 经典模型：从 A 出发，经过河岸 l，到达 B

**方法**：
1. 作 $A$ 关于 $l$ 的对称点 $A'$
2. 连接 $A'B$，与 $l$ 交于点 $P$
3. $A'B$ 就是从 $A$ 经 $P$ 到 $B$ 的最短路径长度
4. 最短路径 = $A'B$

**原理**：$AP + PB = A'P + PB = A'B$（直线最短）

任何其他点 $Q$ 在 $l$ 上：$AQ + QB = A'Q + QB > A'B$（折线 > 直线）

### 生活例子

- 蚂蚁从家（$A$）到食物（$B$），中途必须经过一堵墙（$l$）爬过去，怎样走最短？
- 在河边找一个取水点，使从村庄 $A$ 取水后送到农田 $B$ 的路程最短。

## 🎓 12+ 进阶版

### 严格证明

设 $A'$ 是 $A$ 关于直线 $l$ 的对称点，$P$ 是 $A'B$ 与 $l$ 的交点，$Q$ 是 $l$ 上任意一点（$Q \neq P$）。

$$AP = A'P \quad \text{（对称性）}$$

$$AP + PB = A'P + PB = A'B$$

$$AQ + QB = A'Q + QB \geq A'B \quad \text{（三角不等式：两边之和大于第三边）}$$

等号当且仅当 $A'$、$Q$、$B$ 三点共线，即 $Q = P$ 时成立。

∴ 最短路径经过 $P$，最短距离为 $A'B$。$\blacksquare$

### 变体模型

**模型 2：从 A 出发，经过两条平行线 $l_1$、$l_2$，到达 B**

将 $A$ 关于 $l_1$ 作对称点 $A'$，再将 $B$ 关于 $l_2$ 作对称点 $B'$，连接 $A'B'$，分别与 $l_1$ 交于 $P$，与 $l_2$ 交于 $Q$，则 $A \to P \to Q \to B$ 为最短路径。

**模型 3：等腰三角形中的最短路径（费马点）**

在等腰三角形中，利用对称性找出各边上的最优中转点，是数学竞赛常见题型。

### 典型例题

**例题 1**：$A(1, 3)$，$B(4, 1)$，直线 $l: y = 0$（$x$ 轴），在 $l$ 上找点 $P$ 使 $AP + PB$ 最小。

解：$A$ 关于 $x$ 轴的对称点 $A'(1, -3)$

连 $A'B$：斜率 $= \dfrac{1-(-3)}{4-1} = \dfrac{4}{3}$

直线 $A'B$：$y + 3 = \dfrac{4}{3}(x - 1)$，令 $y = 0$：$3 = \dfrac{4}{3}(x-1)$，$x - 1 = \dfrac{9}{4}$，$x = \dfrac{13}{4}$

∴ $P = \left(\dfrac{13}{4}, 0\right)$

最短路径 $= A'B = \sqrt{(4-1)^2 + (1+3)^2} = \sqrt{9+16} = 5$

**例题 2**：如图，村庄 $A$、$B$ 在河流 $l$ 同侧，$A$ 到河流距离为 3，$B$ 到河流距离为 4，$AB$ 水平距离为 5，求最短取水路径。

解：作 $A$ 关于 $l$ 的对称点 $A'$，$A'$ 到 $l$ 距离为 3。

$A'B = \sqrt{5^2 + (3+4)^2} = \sqrt{25 + 49} = \sqrt{74}$（待家长核对题目数据）

### 易错点

1. **$A$、$B$ 在同侧**才需要作对称点；若在异侧，直接连 $AB$ 即可。
2. **作的是一个点的对称点**，然后连另一个点，不能两个点都作对称。
3. 对称点的位置要准确：$A'$ 在 $l$ 的另一侧，且 $AA' \perp l$，中点在 $l$ 上。
