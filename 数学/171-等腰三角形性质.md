# 等腰三角形性质

> **课本定义**：等腰三角形两腰相等；等腰三角形两底角相等（"等边对等角"）；等腰三角形顶角的角平分线、底边上的中线、底边上的高互相重合（"三线合一"）。

---

## 📚 6-12 岁（回溯版）

### 等腰三角形的组成

- **腰**：相等的两条边（有两条）
- **底边**：不等于腰的那条边
- **顶角**：两腰之间的角
- **底角**：腰与底边之间的角（有两个）

### 两大核心性质

**性质 1：等边对等角**
$$AB = AC \implies \angle B = \angle C \quad \text{（两底角相等）}$$

**逆命题（等角对等边）**：
$$\angle B = \angle C \implies AB = AC \quad \text{（可用来判定等腰三角形）}$$

**性质 2：三线合一**

等腰三角形顶角 $A$ 的角平分线、底边 $BC$ 的中线、底边 $BC$ 上的高——三条线**完全重合**！

### 应用

已知等腰三角形底角为 50°，求顶角。

解：顶角 = 180° − 50° − 50° = **80°**

已知等腰三角形顶角为 100°，求底角。

解：底角 = (180° − 100°) ÷ 2 = **40°**

## 🎓 12+ 进阶版

### 性质证明：等边对等角

**定理**：在 $\triangle ABC$ 中，$AB = AC$，则 $\angle B = \angle C$。

**证明**（三线合一法）：作顶角 $A$ 的角平分线 $AD$（$D$ 在 $BC$ 上）。

在 $\triangle ABD$ 和 $\triangle ACD$ 中：$AB = AC$，$\angle BAD = \angle CAD$（角平分线），$AD = AD$（公共边）

由 SAS，$\triangle ABD \cong \triangle ACD$

∴ $\angle B = \angle C$。$\blacksquare$

**同时得到**：$BD = CD$（$AD$ 是中线），$\angle ADB = \angle ADC = 90°$（$AD$ 是高）

即证"三线合一"。

### 逆定理证明：等角对等边

**定理**：在 $\triangle ABC$ 中，$\angle B = \angle C$，则 $AB = AC$。

**证明**：作顶角 $A$ 的角平分线 $AD$。

在 $\triangle ABD$ 和 $\triangle ACD$ 中：$\angle ABD = \angle ACD$（已知），$\angle BAD = \angle CAD$，$AD = AD$

由 AAS，$\triangle ABD \cong \triangle ACD$

∴ $AB = AC$。$\blacksquare$

### 等腰三角形的轴对称性

等腰三角形是**轴对称图形**，对称轴为顶角平分线（即底边的垂直平分线）。

### 典型例题

**例题 1**：等腰三角形一个角为 36°，求其余两角的可能值。

分析：36° 可能是底角或顶角。

- 若 36° 是顶角：底角 = $(180° - 36°) / 2 = 72°$，三角形存在 ✓
- 若 36° 是底角：顶角 = $180° - 72° = 108°$，三角形存在 ✓

答：其余两角为 **72°、72°** 或 **36°、108°** 两种情况。

**例题 2**：等腰 $\triangle ABC$，$AB = AC$，$D$ 是底边 $BC$ 的中点，$\angle ADB = 90°$。求证 $AD$ 是 $BC$ 的垂直平分线。

证明：$D$ 是 $BC$ 中点（已知），$AD \perp BC$（$\angle ADB = 90°$），故 $AD$ 是 $BC$ 的垂直平分线。✓

（实际上这就是"三线合一"的直接体现。）

### 易错点

1. **等腰三角形讨论角时要分类**：不知道某角是顶角还是底角时，要**两种情况**讨论。
2. 三线合一只对**顶角**的角平分线成立，底角的角平分线不具备此性质。
3. "等角对等边"是逆定理，同样成立，常用于判定等腰三角形。
