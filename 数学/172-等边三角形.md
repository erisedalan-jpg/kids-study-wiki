# 等边三角形

> **课本定义**：三条边都相等的三角形叫做**等边三角形**（又叫正三角形）。等边三角形三个内角都等于 60°；等边三角形是等腰三角形的特殊情况。

---

## 📚 6-12 岁（回溯版）

### 等边三角形的特征

| 特征 | 描述 |
|------|------|
| 三边 | 全部相等：$a = b = c$ |
| 三角 | 全部相等：$\angle A = \angle B = \angle C = 60°$ |
| 对称轴 | 有 3 条（每条边的垂直平分线也是对角的角平分线）|
| 重心、外心、内心、垂心 | 四点合一，在中心 |

### 等边三角形的判定方法

| 判定条件 | 说明 |
|----------|------|
| 三边相等 | 定义直接判定 |
| 三角均为 60° | 由角判定 |
| 等腰三角形 + 一角为 60° | 等腰 + 60° 顶角或底角均可推出等边 |

### 与等腰三角形的关系

等边三角形 ⊂ 等腰三角形（任意两边均可作为"腰"）

等腰三角形中，顶角 = 60° → 等边三角形；底角 = 60° → 顶角 = 60° → 等边三角形

### 尺规作图

用圆规和直尺可以作出等边三角形：以一条线段的两端为圆心，以该线段为半径画圆，两圆交点与线段两端点构成等边三角形。

## 🎓 12+ 进阶版

### 严格性质推导

**由等边推等角**：$AB = BC = CA$

取 $AB = AC$，由等腰三角形性质，$\angle B = \angle C$

取 $BA = BC$，由等腰三角形性质，$\angle A = \angle C$

∴ $\angle A = \angle B = \angle C$，由内角和 $\angle A + \angle B + \angle C = 180°$，得 $\angle A = \angle B = \angle C = 60°$。

**由三角均为 60° 推等边**：

$\angle A = \angle B$ → 由等角对等边，$BC = AC$

$\angle B = \angle C$ → $AC = AB$

∴ $AB = BC = CA$，即等边三角形。

### 等边三角形面积公式

设等边三角形边长为 $a$，高为 $h$：

$$h = \frac{\sqrt{3}}{2} a \quad \text{（由勾股定理：}h^2 + \left(\frac{a}{2}\right)^2 = a^2\text{）}$$

$$S = \frac{1}{2} \cdot a \cdot h = \frac{\sqrt{3}}{4} a^2$$

### 典型例题

**例题 1**：等边三角形边长为 6，求面积和高。

解：$h = \dfrac{\sqrt{3}}{2} \times 6 = 3\sqrt{3}$

$S = \dfrac{\sqrt{3}}{4} \times 36 = 9\sqrt{3}$

**例题 2**：等腰三角形 $\triangle ABC$，$AB = AC$，$\angle A = 60°$，证明 $\triangle ABC$ 是等边三角形。

证明：$AB = AC$（等腰），$\angle A = 60°$（已知）

$\angle B = \angle C = \dfrac{180° - 60°}{2} = 60°$

三角均为 60°，∴ $\triangle ABC$ 是等边三角形。✓

### 等边三角形的密铺性质

等边三角形是可以密铺平面的三种正多边形之一（另两种是正方形和正六边形）。

蜂巢由正六边形构成，每个正六边形可分成 6 个等边三角形。

### 易错点

1. **等腰三角形不一定是等边三角形**：需要额外满足"一角为 60°"的条件。
2. 等边三角形的三条对称轴既是**角平分线**也是**中线**也是**高**——与等腰三角形的"三线合一"类似，但有 3 条这样的线。
3. 用角判定时，只需确认两个角为 60°（第三个角自动也是 60°）。
