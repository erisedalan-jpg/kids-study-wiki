# 正方形

> **一句话**：四条边相等且四个角都是直角的平行四边形叫正方形，它是矩形和菱形的"交集"。
> **English**: A square is a rectangle with all four sides equal (equivalently, a rhombus with all right angles); it has the properties of both.

---

## 📚 6-12 岁回溯版

### 定义

**正方形**：有一个直角且有一组邻边相等的平行四边形。等价表述：既是矩形又是菱形。

### 正方形的全部性质

正方形继承了**矩形**的全部性质 + **菱形**的全部性质：

| 性质来源 | 内容 |
|----------|------|
| 平行四边形 | 对边平行相等、对角相等、对角线互平分 |
| 矩形 | 四角均为 90°，对角线相等 |
| 菱形 | 四边相等，对角线互相垂直，对角线平分对角 |

**对角线综合**：$AC=BD$（矩形），$AC \perp BD$（菱形），$OA=OB=OC=OD$（互平分 + 等长）。

### 关键公式

- 面积：$S = a^2$（边长的平方）；也可用 $S = \dfrac{1}{2}d^2$（$d$ 为对角线）
- 对角线：$d = a\sqrt{2}$（$a$ 为边长，由勾股定理）
- 对角线与边长关系：$a = \dfrac{d\sqrt{2}}{2} = \dfrac{\sqrt{2}}{2}d$

### 判定方法

| 判定 | 条件 |
|------|------|
| 判定1 | 四边相等且四角为直角 |
| 判定2 | 对角线相等且互相垂直平分 |
| 判定3 | 有一直角的菱形 |
| 判定4 | 邻边相等的矩形 |

## 🎓 12+ 进阶版

### 正方形的极简判定

正方形 = 矩形 ∩ 菱形（满足其中一种，再加一个额外条件即变成正方形）。

- 矩形 + 邻边相等 → 正方形
- 菱形 + 一个直角 → 正方形
- 对角线相等 + 互相垂直平分 → 正方形

### 典型例题

**例 1**：正方形边长为 6，求对角线长和面积。

$$d = 6\sqrt{2}, \quad S = 36 \text{（或 } \frac{1}{2}\times6\sqrt{2}\times6\sqrt{2}=36 \text{）}$$

**例 2**：正方形 $ABCD$，$E$ 是 $BC$ 的中点，$AE$ 与 $BD$ 交于点 $F$，求 $\angle AFD$。

（需用坐标法或三角形相似，具体步骤待家长配合教材核实。）

### 易错点

- ❌ "对角线垂直等长"不够，还需互平分，才是正方形
- ❌ 混淆菱形（对角线不等长）与正方形（对角线等长）
- ❌ 正方形的对角线平分每个内角（各 45°），忽略此细节

### 知识联系

正方形是所有特殊四边形（平行四边形→矩形/菱形→正方形）的"最特殊"成员；正方形的对角线 $a\sqrt{2}$ 来自[[189-勾股定理|勾股定理]]；正方形的轴对称性→[[174-轴对称图形|轴对称图形]]（有4条对称轴）；正方形面积公式 $S=a^2$ 是[[043-面积|面积]]概念的基础定义。
