# 中心对称

> **一句话**：将一个图形绕某一点旋转 180° 后能与另一图形完全重合，则称这两个图形关于这个点**中心对称**，这个点叫做**对称中心**。
> **English**: Two figures are point symmetric (centrally symmetric) about a point O if one maps to the other by a 180° rotation about O.

---

## 📚 给 6-12 岁（自读版）

### 什么是中心对称？

轴对称是关于一条线翻折（镜像），中心对称是关于一个**点**旋转 180°！

**方法**：从图形上任意一点出发，过对称中心画一条线，在对称中心另一侧相同距离处，就是对应点。

### 直观识别

下面哪些是中心对称？

- 平行四边形 ✅（关于对角线交点对称）
- 字母 S ✅
- 等边三角形 ❌（旋转 120° 重合，但 180° 不能重合）
- 正方形 ✅（关于中心旋转 90° 也重合，180° 当然也行）
- 字母 B ❌（轴对称，不是中心对称）

### 对应点的关系

若点 $A$ 与点 $A'$ 关于点 $O$ 中心对称，则：
- $O$ 是 $AA'$ 的**中点**
- $|OA| = |OA'|$
- $A, O, A'$ 三点共线

## 🎓 给 12+（进阶版）

### 定义

**中心对称**：将图形 $F_1$ 绕点 $O$ 旋转 180°，若所得图形与图形 $F_2$ 完全重合，则称 $F_1$ 与 $F_2$ **关于点 $O$ 中心对称**，点 $O$ 叫做**对称中心**。

本质：180° 旋转是旋转变换的特殊情形。

### 中心对称的性质

**性质1**：对应点连线过对称中心，且对称中心是连线的中点。

若 $A, A'$ 是关于 $O$ 的对称点，则 $O$ 是 $AA'$ 的中点，即 $\overrightarrow{OA} + \overrightarrow{OA'} = \vec{0}$。

**性质2**：中心对称图形全等（180° 旋转是等距变换）。

**性质3**：对应线段平行且相等（或共线）。

若 $AB$ 与 $A'B'$ 关于 $O$ 对称，则 $AB \parallel A'B'$（或共线），且 $AB = A'B'$。

### 中心对称与轴对称的比较

| 特征 | 轴对称 | 中心对称 |
|------|--------|---------|
| 变换方式 | 关于直线翻折 | 绕点旋转 180° |
| 对称元素 | 对称轴（直线） | 对称中心（点） |
| 对应点连线 | 垂直平分对称轴 | 过对称中心（中点） |
| 代表图形 | 等腰三角形、轴 | 平行四边形、圆 |

### 坐标系中的中心对称

**特殊情形**：关于原点 $O(0,0)$ 对称

若 $A(a, b)$ 关于原点对称，则对称点 $A'(-a, -b)$。

详见[[关于原点对称的点的坐标]]。

**一般情形**：关于点 $M(m, n)$ 对称

若 $A(x_0, y_0)$ 关于 $M(m,n)$ 对称，则对称点：

$$A' = (2m - x_0,\ 2n - y_0)$$

（因为 $M$ 是 $AA'$ 的中点：$m = \dfrac{x_0 + x_{A'}}{2}$，同理 $y$ 坐标）

### 典型例题

**例 1**：点 $A(3, -2)$ 关于点 $M(1, 0)$ 的对称点 $A'$ 的坐标。

$A' = (2 \times 1 - 3,\ 2 \times 0 - (-2)) = (-1, 2)$

验证：$M$ 是 $AA'$ 中点：$\left(\frac{3+(-1)}{2}, \frac{-2+2}{2}\right) = (1, 0)$ ✓

**例 2**：判断平行四边形 $ABCD$ 是否中心对称。

对角线 $AC$ 和 $BD$ 互相平分，即交点 $O$ 是两对角线的中点。

- $O$ 是 $AC$ 的中点 → $A, C$ 关于 $O$ 对称
- $O$ 是 $BD$ 的中点 → $B, D$ 关于 $O$ 对称

因此平行四边形关于对角线交点中心对称。✓

**例 3**：线段 $AB$ 的中点为 $O$，$P$ 是不在 $AB$ 上的点，$P'$ 是 $P$ 关于 $O$ 的对称点，证明 $PP' \parallel AB$……（待家长核对完整题目）

### 常见中心对称图形

- 平行四边形（关于对角线交点）
- 矩形（关于对角线交点）
- 正方形（关于中心）
- 菱形（关于对角线交点）
- 圆（关于圆心）
- 正偶数边形（关于中心）

### 易错点

- ❌ 把"中心对称"与"中心对称图形"混淆（前者是两个图形的关系，后者是自身与自身重合）
- ❌ 认为等腰三角形是中心对称（它是轴对称，不是中心对称）
- ❌ 坐标求对称点时用错公式（关于点 $M$ 对称，不是关于原点！）
