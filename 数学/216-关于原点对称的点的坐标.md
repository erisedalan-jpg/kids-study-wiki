# 关于原点对称的点的坐标

> **一句话**：点 $P(a, b)$ 关于原点 $O(0,0)$ 的对称点是 $P'(-a, -b)$——横纵坐标都取反。
> **English**: The point symmetric to $P(a, b)$ about the origin is $P'(-a, -b)$.

---

## 📚 给 6-12 岁（自读版）

### 规律很简单！

关于原点（坐标系中心）对称：**横纵坐标都变为相反数**。

$$P(a, b) \xrightarrow{\text{关于原点对称}} P'(-a, -b)$$

### 例子

| 原来的点 | 关于原点的对称点 |
|---------|---------------|
| $(3, 4)$ | $(-3, -4)$ |
| $(-2, 5)$ | $(2, -5)$ |
| $(0, 6)$ | $(0, -6)$ |
| $(-1, -7)$ | $(1, 7)$ |
| $(a, b)$ | $(-a, -b)$ |

### 为什么？

关于原点对称 = 绕原点旋转 180°。

原点 $O$ 是 $PP'$ 的中点，所以：
$$O = \left(\frac{a + x_{P'}}{2}, \frac{b + y_{P'}}{2}\right) = (0, 0)$$

解得 $x_{P'} = -a, y_{P'} = -b$。

### 图象上的意义

若函数图象关于原点对称（奇函数），则 $f(-x) = -f(x)$，例如 $y = x^3$、$y = x$、$y = \sin x$（高中）。

## 🎓 给 12+（进阶版）

### 公式推导

设 $P(a, b)$ 关于原点 $O(0,0)$ 对称，对称点为 $P'(x, y)$。

由中心对称定义：$O$ 是 $PP'$ 的中点，故：

$$\frac{a+x}{2} = 0 \Rightarrow x = -a$$
$$\frac{b+y}{2} = 0 \Rightarrow y = -b$$

$$\boxed{P(a, b) \text{ 的原点对称点为 } P'(-a, -b)}$$

### 推广：关于任意点 $(m, n)$ 的对称

若 $P(a, b)$ 关于 $M(m, n)$ 对称，设对称点为 $P'(x, y)$，$M$ 是 $PP'$ 中点：

$$\frac{a+x}{2} = m \Rightarrow x = 2m-a$$
$$\frac{b+y}{2} = n \Rightarrow y = 2n-b$$

关于原点是此公式取 $m=n=0$ 的特殊情形。

### 线段和图形的原点对称

**线段的对称**：若 $AB$ 关于原点对称得 $A'B'$，则 $A'(-x_A, -y_A)$，$B'(-x_B, -y_B)$。

且 $AB \parallel A'B'$，$|AB| = |A'B'|$。

**函数图象的原点对称**：

$y = f(x)$ 关于原点对称的图象的方程：将 $(x, y)$ 替换为 $(-x, -y)$（原点对称点），得 $-y = f(-x)$，即 $y = -f(-x)$。

**例**：$y = x^2 + x$ 关于原点对称的图象方程：

$-y = (-x)^2 + (-x) = x^2 - x \Rightarrow y = -x^2 + x$

### 典型例题

**例 1**：若点 $(a+1, 2b-1)$ 与点 $(3, -5)$ 关于原点对称，求 $a, b$。

由对称点坐标互为相反数：

$$a + 1 = -3 \Rightarrow a = -4$$
$$2b - 1 = 5 \Rightarrow b = 3$$

**例 2**：线段 $AB$ 关于原点的对称线段为 $A'B'$，其中 $A(2,3)$，$B(-1,4)$。求 $A'B'$ 的长度。

$A'(-2,-3)$，$B'(1,-4)$

$|A'B'| = \sqrt{(1-(-2))^2 + (-4-(-3))^2} = \sqrt{9+1} = \sqrt{10}$

验证：$|AB| = \sqrt{(-1-2)^2+(4-3)^2} = \sqrt{9+1} = \sqrt{10}$ ✓（旋转不改变长度）

**例 3（坐标系中的对称综合）**：已知 $\triangle ABC$ 的顶点 $A(0,2)$，$B(3,0)$，$C(1,-1)$，求关于原点对称的 $\triangle A'B'C'$ 的顶点坐标。

$A'(0,-2)$，$B'(-3,0)$，$C'(-1,1)$

**例 4（判别）**：$y = x^3$ 的图象关于原点对称吗？

$f(-x) = (-x)^3 = -x^3 = -f(x)$，满足奇函数条件，图象关于原点对称。✓

### 关于坐标轴对称的拓展

| 对称轴/中心 | $P(a,b)$ 的对称点 |
|------------|-----------------|
| $x$ 轴 | $(a, -b)$ |
| $y$ 轴 | $(-a, b)$ |
| 原点 | $(-a, -b)$ |
| 直线 $y=x$ | $(b, a)$ |
| 直线 $y=-x$ | $(-b, -a)$ |

### 易错点

- ❌ 只变一个坐标（关于原点是两个都取反！）
- ❌ 关于原点与关于 $x$ 轴、$y$ 轴混淆
- ❌ 对称点坐标与中点公式搞反（是"$O$ 是 $PP'$ 中点"，而非"$O$ 在 $PP'$ 上"）
- ❌ 图形的对称与点的对称步骤混淆（图形对称 = 所有点都取对称点）
