# 切线长定理

> **一句话**：从圆外一点向圆作两条切线，两切线的长（从该点到切点的线段长）相等，且该点与圆心连线平分两切线所成的角。
> **English**: From a point outside a circle, two tangent lines can be drawn; the two tangent segments are equal in length, and the line joining the point to the center bisects the angle between the tangents.

---

## 📚 给 6-12 岁（自读版）

### 切线长定理

从圆外一点 $P$ 向圆 $O$ 作两条切线，切点分别为 $T_1$ 和 $T_2$，则：

$$|PT_1| = |PT_2|$$

**两条切线一样长！**

### 为什么？

连 $OT_1$、$OT_2$、$OP$：

- $OT_1 \perp PT_1$，$OT_2 \perp PT_2$（切线 $\perp$ 半径）
- $OT_1 = OT_2 = r$（同圆半径相等）
- $OP = OP$（公共边）

在直角三角形 $OT_1P$ 和 $OT_2P$ 中，由勾股定理：

$$|PT_1|^2 = |OP|^2 - r^2 = |PT_2|^2$$

故 $|PT_1| = |PT_2|$ ✓

### 切线长公式

$$\boxed{|PT|^2 = |OP|^2 - r^2, \quad |PT| = \sqrt{|OP|^2 - r^2}}$$

### 例题

$\odot O$ 半径 3，$P$ 在圆外，$|OP| = 5$，求从 $P$ 到圆的切线长。

$$|PT| = \sqrt{25 - 9} = \sqrt{16} = 4$$

## 🎓 给 12+（进阶版）

### 切线长定理（完整）

**定理**：从圆外一点 $P$ 到 $\odot O$ 的两条切线切点为 $T_1, T_2$，则：

$$|PT_1| = |PT_2|$$

且 $OP$ 是 $\angle T_1PT_2$ 的角平分线，也是 $T_1T_2$ 的垂直平分线。

**证明**：在 $\triangle OT_1P$ 与 $\triangle OT_2P$ 中：

- $\angle OT_1P = \angle OT_2P = 90°$
- $OT_1 = OT_2 = r$
- $OP = OP$

由勾股定理（或 HL）：$\triangle OT_1P \cong \triangle OT_2P$

故 $PT_1 = PT_2$，$\angle T_1PO = \angle T_2PO$（$OP$ 平分 $\angle T_1PT_2$）。

### 切线长公式

$$|PT| = \sqrt{|OP|^2 - r^2}$$

**推广**：若 $P(x_0, y_0)$ 在圆 $(x-a)^2+(y-b)^2=r^2$ 外，则切线长：

$$|PT| = \sqrt{(x_0-a)^2+(y_0-b)^2-r^2}$$

### 切线长在三角形内切圆中的应用

$\triangle ABC$ 的内切圆 $\odot I$ 与 $AB, BC, CA$ 分别切于 $D, E, F$，则：

$$AD = AF, \quad BD = BE, \quad CE = CF$$

设 $AD = x, BD = y, CE = z$，则：

$$x + y = AB, \quad y + z = BC, \quad z + x = CA$$

解得：

$$x = \frac{AB + CA - BC}{2}, \quad y = \frac{AB + BC - CA}{2}, \quad z = \frac{BC + CA - AB}{2}$$

内切圆半径：$r = \frac{S_{\triangle}}{s}$，其中 $S$ 为面积，$s = \frac{AB+BC+CA}{2}$ 为半周长。

### 典型例题

**例 1**（基础）：$|OP| = 10$，$\odot O$ 半径 $r = 6$，求切线长。

$$|PT| = \sqrt{100 - 36} = \sqrt{64} = 8$$

**例 2**（三角形内切圆）：$\triangle ABC$ 三边 $AB = 5, BC = 6, CA = 7$，内切圆半径 $r$，求三切线段长。

$s = \frac{5+6+7}{2} = 9$

$x = 9-6=3, y = 9-7=2, z = 9-5=4$

验证：$x+y=5=AB$ ✓，$y+z=6=BC$ ✓，$x+z=7=CA$ ✓

$S = \sqrt{9 \times 4 \times 3 \times 2} = \sqrt{216} = 6\sqrt{6}$，$r = \frac{6\sqrt{6}}{9} = \frac{2\sqrt{6}}{3}$

**例 3**（坐标中切线长）：$P(1, 2)$ 到圆 $(x-3)^2+(y-1)^2=4$ 的切线长。

$$|PT| = \sqrt{(1-3)^2+(2-1)^2-4} = \sqrt{4+1-4} = 1$$

**例 4**（综合）：从圆外点 $P$ 作两切线 $PT_1, PT_2$，连 $T_1T_2$，若 $|PT_1| = 6$，$\angle T_1PT_2 = 60°$，求 $|T_1T_2|$。

$\angle T_1PO = 30°$，$|PT_1| = 6$，$|T_1T_2| = 2|PT_1|\sin 30° = 2 \times 6 \times 0.5 = 6$

（$\triangle PT_1T_2$ 是等边三角形！）

### 相关公式

| 量 | 公式 |
|----|------|
| 切线长 | $|PT| = \sqrt{|OP|^2 - r^2}$ |
| 切线角 | $\sin(\angle OPT) = \frac{r}{|OP|}$ |
| 弦长 $|T_1T_2|$ | $2|PT|\sin\frac{\angle T_1PT_2}{2}$ |

### 易错点

- ❌ 切线长不是从圆外点到圆心，而是到切点
- ❌ 计算 $|PT|^2 = |OP|^2 - r^2$ 中用直径代替半径 $r$
- ❌ 内切圆切线段中 $AD = AF$（同一顶点的两切线相等，不是任意两切线相等）
