# 因式分解法-方程

> **一句话**：将方程左边因式分解为两个一次因式之积，利用"积为零则某因式为零"的原理来解方程，叫做因式分解法。
> **English**: The factoring method solves a quadratic equation by factoring the left side as a product of linear factors, using the zero-product property.

---

## 📚 给 6-12 岁（自读版）

### 核心原理：零积定理

如果两个数相乘等于 0，那么**其中至少有一个数等于 0**！

$$A \times B = 0 \Rightarrow A = 0 \text{ 或 } B = 0$$

### 解方程步骤

以 $x^2 - 5x + 6 = 0$ 为例：

**第一步**：右边为 0（已满足）

**第二步**：左边因式分解：
$$x^2 - 5x + 6 = (x-2)(x-3)$$

**第三步**：用零积定理：
$$(x-2)(x-3) = 0$$
$$\Rightarrow x - 2 = 0 \text{ 或 } x - 3 = 0$$
$$\Rightarrow x_1 = 2,\quad x_2 = 3$$

### 常用因式分解方法

| 方法 | 形式 | 例子 |
|------|------|------|
| 提公因式 | $ax^2 + bx = x(ax+b)$ | $2x^2 - 4x = 2x(x-2)$ |
| 平方差公式 | $x^2 - a^2 = (x+a)(x-a)$ | $x^2 - 9 = (x+3)(x-3)$ |
| 完全平方公式 | $x^2 \pm 2ax + a^2 = (x \pm a)^2$ | $x^2 + 4x + 4 = (x+2)^2$ |
| 十字相乘法 | $x^2 + (a+b)x + ab = (x+a)(x+b)$ | $x^2 + 5x + 6 = (x+2)(x+3)$ |

### 什么时候用这个方法？

- 方程右边已经是 0 ✅
- 左边能快速因式分解 ✅

不能因式分解时，改用[[公式法-一元二次方程]]。

## 🎓 给 12+（进阶版）

### 零积定理（零因子定理）

**定理**：若实数 $A \cdot B = 0$，则 $A = 0$ 或 $B = 0$（或两者都为 0）。

这是因式分解法的理论基础。注意：若 $AB = k \neq 0$，**不能**分别令 $A = k$ 或 $B = 1$。

### 详细步骤

1. 将方程整理为 $ax^2 + bx + c = 0$（右边为 0！）
2. 对左边进行因式分解：$a(x - x_1)(x - x_2) = 0$
3. 利用零积定理分别令每个因式为 0
4. 解出各个根

### 各类因式分解

**类型1**：有公因式型
$$x^2 - 3x = 0 \Rightarrow x(x-3) = 0 \Rightarrow x = 0 \text{ 或 } x = 3$$

**类型2**：平方差型
$$x^2 - 4 = 0 \Rightarrow (x+2)(x-2) = 0 \Rightarrow x = \pm 2$$

**类型3**：完全平方型
$$x^2 - 6x + 9 = 0 \Rightarrow (x-3)^2 = 0 \Rightarrow x = 3$$（二重根）

**类型4**：十字相乘型
$$x^2 + 3x - 10 = 0 \Rightarrow (x+5)(x-2) = 0 \Rightarrow x = -5 \text{ 或 } x = 2$$

**类型5**：提系数后因式分解
$$6x^2 + x - 2 = 0 \Rightarrow (2x-1)(3x+2) = 0 \Rightarrow x = \frac{1}{2} \text{ 或 } x = -\frac{2}{3}$$

### 典型例题（综合）

**例**：解方程 $(x-1)^2 = 2(x-1)$

⚠️ 不能直接两边除以 $(x-1)$（会丢根！）

正确做法：

$$(x-1)^2 - 2(x-1) = 0$$
$$(x-1)[(x-1) - 2] = 0$$
$$(x-1)(x-3) = 0$$
$$x_1 = 1,\quad x_2 = 3$$

### 韦达定理验证

若 $x_1, x_2$ 是 $ax^2 + bx + c = 0$ 的两根，则：
- $x_1 + x_2 = -\dfrac{b}{a}$
- $x_1 \cdot x_2 = \dfrac{c}{a}$

用于验算根的正确性（详见[[韦达定理-初中]]）。

### 易错点

- ❌ 方程右边不为 0 就开始因式分解（如 $x^2 - 3x = 4$，需先移项！）
- ❌ 两边同除以含 $x$ 的因式（丢失 $x=0$ 的根）
- ❌ 十字相乘时符号出错
- ❌ 完全平方型只写一个根（实为二重根，但仍是唯一解）
