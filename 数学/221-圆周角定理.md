# 圆周角定理

> **一句话**：同弧（或等弧）所对的圆周角相等，且等于该弧所对圆心角的一半。直径所对的圆周角是直角。
> **English**: The inscribed angle is half the central angle that subtends the same arc; all inscribed angles subtending the same arc are equal.

---

## 📚 给 6-12 岁（自读版）

### 什么是圆周角？

- **圆心角**：顶点在圆**心**
- **圆周角**：顶点在圆**上**（周上），两边与圆交于另外两点

**圆周角定理**：同一段弧所对的所有圆周角都相等！

### 最重要的推论

**直径所对的圆周角是 90°！**

如果 $AB$ 是圆的直径，$P$ 是圆上不同于 $A, B$ 的点，则：

$$\angle APB = 90°$$

（直径对应的圆心角 = 180°，圆周角 = 180° ÷ 2 = 90°）

**实用记忆**：直径端点之间的圆周角 = 直角。

### 圆周角与圆心角的关系

$$\text{圆周角} = \frac{1}{2} \times \text{同弧所对的圆心角}$$

**例**：$\angle AOB = 80°$（圆心角），则同弧 $\overset{\frown}{AB}$ 上任意圆周角 $\angle APB = 40°$。

### 同弧等角

在同一个圆中，所有对着相同弧的圆周角都相等：

$$\angle APB = \angle AQB = \angle ARB \text{（只要 } P, Q, R \text{ 在弧 } AB \text{ 的同侧）}$$

## 🎓 给 12+（进阶版）

### 定义

**圆周角**（inscribed angle）：顶点在圆上，两条边分别与圆有另外两个交点的角。

设 $\angle APB$ 是 $\odot O$ 的圆周角，所对的弧为 $\overset{\frown}{AB}$（不含 $P$ 的一侧）。

### 圆周角定理

**定理1**（圆周角是圆心角一半）：

$$\angle APB = \frac{1}{2} \angle AOB$$

（$P$ 在弧 $AB$ 不含 $P$ 的另一侧）

**证明思路**（以 $O$ 在 $\angle APB$ 内部的情形为例）：

连 $PO$ 并延长交圆于 $C$，则 $OA = OP = OB = r$，故：

- $\triangle OAP$ 等腰：$\angle OAP = \angle OPA$，外角 $\angle AOC = 2\angle OPA$
- $\triangle OBP$ 等腰：$\angle OBP = \angle OPB$，外角 $\angle BOC = 2\angle OPB$

$$\angle AOB = \angle AOC + \angle BOC = 2\angle OPA + 2\angle OPB = 2\angle APB$$

（其他情形——$O$ 在角外或角上——类似证明）

**定理2**（同弧等圆周角）：

同弧或等弧所对的圆周角相等。

$$\overset{\frown}{AB} \text{ 相同} \Rightarrow \angle APB = \angle AQB \text{（} P, Q \text{ 在同侧）}$$

**定理3**（直径所对圆周角）：

$$AB \text{ 是直径} \Rightarrow \angle APB = 90° \text{（} P \text{ 在圆上，} P \neq A, B\text{）}$$

**逆定理**：若 $\angle APB = 90°$，$P$ 在以 $AB$ 为直径的圆上，则 $AB$ 是圆的直径。

### 推论

**推论1**：圆内接四边形对角互补。

若 $ABCD$ 内接于圆，则 $\angle A + \angle C = 180°$，$\angle B + \angle D = 180°$。

**证明**：$\angle A$ 和 $\angle C$ 分别是弧 $BCD$ 和弧 $BAD$ 所对的圆周角，而 $\overset{\frown}{BCD} + \overset{\frown}{BAD} = 360°$，故圆心角之和为 $360°$，圆周角之和为 $180°$。

### 典型例题

**例 1**（基础）：$\odot O$ 中，圆周角 $\angle APB = 35°$，求弧 $AB$ 所对的圆心角。

$$\angle AOB = 2 \times 35° = 70°$$

**例 2**（直角判断）：$\triangle ABC$ 内接于 $\odot O$，$AB$ 是直径，$C$ 在圆上，求 $\angle ACB$。

由直径所对圆周角：$\angle ACB = 90°$。

**例 3**（综合）：$\odot O$ 中，$AB$ 为直径，$C, D$ 在圆上，$\angle CAB = 30°$，求 $\angle D$（$D$ 与 $C$ 在 $AB$ 同侧）。

$\angle ACB = 90°$（直径对圆周角），$\angle CAB = 30°$，$\angle ABC = 60°$。

$\angle D = \angle ABC = 60°$（同弧 $\overset{\frown}{BC}$ 所对圆周角相等，$C, D$ 在同侧）

**例 4**（内接四边形）：圆内接四边形 $ABCD$，$\angle A = 75°$，求 $\angle C$。

$$\angle C = 180° - 75° = 105°$$

### 易错点

- ❌ 圆周角是弧对应的**圆心角一半**（不是相等！）
- ❌ 顶点不在圆上的角不是圆周角（内角、切线角另论）
- ❌ $P$ 在弧 $AB$ 两侧时，圆周角互补，而非相等（同侧相等！）
- ❌ 直径所对圆周角 = 90°，反之若角为 90° 则弦为直径（逆定理）
