# 圆和圆的位置关系

> **一句话**：两圆的位置关系由圆心距 $d$ 与两半径 $r_1, r_2$ 的关系决定，分为外离、外切、相交、内切、内含五种。
> **English**: Two circles can be externally separate, externally tangent, intersecting, internally tangent, or one inside the other, determined by comparing the center distance d with the radii.

---

## 📚 给 6-12 岁（自读版）

### 五种位置关系

两个圆，大圆半径 $R$，小圆半径 $r$（$R \geq r$），圆心距 $d$（两圆心之间的距离）：

| 名称 | 公共点 | 记忆图形 | 条件 |
|------|--------|---------|------|
| **外离** | 0 | ○ ○（分开） | $d > R + r$ |
| **外切** | 1 | ○ ⊙（外相切） | $d = R + r$ |
| **相交** | 2 | ⊙（重叠） | $R - r < d < R + r$ |
| **内切** | 1 | ⊙（内相切） | $d = R - r$（$R > r$） |
| **内含** | 0 | ⊙（一在另内） | $d < R - r$ |

**特殊情形**：同心圆（$d = 0, R \neq r$）属于内含，无公共点。

### 口诀记忆

外离 $d > R+r$，外切 $d = R+r$，

相交在中间，内切 $d = R-r$，

内含 $d < R-r$，同心是特例。

### 生活类比

想象两个圆形泳圈：
- 完全分开（外离）
- 刚好碰到一个点，在外面（外切）
- 部分重叠（相交）
- 小的刚好嵌在大的边缘内（内切）
- 小的完全在大的里面（内含）

## 🎓 给 12+（进阶版）

### 五种位置关系的判断定理

设 $\odot O_1$ 半径 $r_1$，$\odot O_2$ 半径 $r_2$（$r_1 \geq r_2$），圆心距 $d = |O_1 O_2|$：

| 位置关系 | 条件 | 公共点数 |
|---------|------|---------|
| 外离 | $d > r_1 + r_2$ | 0 |
| 外切 | $d = r_1 + r_2$ | 1 |
| 相交 | $r_1 - r_2 < d < r_1 + r_2$ | 2 |
| 内切 | $d = r_1 - r_2$（$r_1 \neq r_2$） | 1 |
| 内含 | $d < r_1 - r_2$ | 0 |
| 同心 | $d = 0, r_1 \neq r_2$ | 0 |

### 公切线的条数

| 位置关系 | 公切线条数 |
|---------|----------|
| 外离 | 4（两条外切线，两条内切线） |
| 外切 | 3（两条外切线，一条公切线过切点） |
| 相交 | 2（两条外切线） |
| 内切 | 1（过切点的公切线） |
| 内含 | 0 |
| 同心 | 0 |

### 切点的性质

**外切**（$d = r_1 + r_2$）：切点 $P$ 在两圆心连线 $O_1O_2$ 上，且在 $O_1, O_2$ 之间：
$$O_1P = r_1, \quad O_2P = r_2, \quad O_1P + O_2P = d$$

**内切**（$d = r_1 - r_2$）：切点 $P$ 在两圆心连线的延长线上：
$$O_1P = r_1, \quad O_2P = r_2, \quad O_1P - O_2P = d$$

### 典型例题

**例 1**（位置判断）：两圆半径分别为 3 和 5，圆心距 $d = 7$，判断位置关系。

$r_1 + r_2 = 8 > 7$，$r_1 - r_2 = 2 < 7$，故 $r_1 - r_2 < d < r_1 + r_2$，两圆**相交**。

**例 2**：两圆半径 4 和 3，圆心距为 7，判断关系。

$d = 7 = 4 + 3 = r_1 + r_2$，两圆**外切**，有 3 条公切线。

**例 3**（坐标判断）：$\odot O_1: x^2 + y^2 = 9$（圆心原点，$r_1=3$），$\odot O_2: (x-5)^2 + y^2 = 4$（圆心 $(5,0)$，$r_2=2$）。

$d = |O_1O_2| = 5$，$r_1 + r_2 = 5$，故 $d = r_1 + r_2$，**外切**。

**例 4**（综合）：已知两圆外切，公切点为 $P(2, 0)$，两圆心在 $x$ 轴上，且 $r_1 = 2, r_2 = 3$，求两圆心坐标。

外切，切点在两圆心连线上：$O_1$ 在 $P$ 左方距 $r_1 = 2$ 处，$O_1 = (0, 0)$；$O_2$ 在 $P$ 右方距 $r_2 = 3$ 处，$O_2 = (5, 0)$。

### 相交两圆的公共弦

两圆相交时，公共弦（连接两个公共点的线段）是两圆心连线的垂直平分线。

即：公共弦 $\perp O_1O_2$，且 $O_1O_2$ 是公共弦的垂直平分线。

### 易错点

- ❌ 忘记 $r_1 \geq r_2$ 的前提，在"内含"与"相交"判断中用错
- ❌ 外切点在 $O_1O_2$ 上（两圆心之间），内切点在连线延长线上（不在两点之间）
- ❌ 两圆相交有 2 条公切线（不是 4 条）
