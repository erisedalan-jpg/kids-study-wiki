# 垂径定理

> **一句话**：圆的直径垂直于弦，则该直径平分此弦及其所对的两段弧；反之，平分弦的直径垂直于该弦。
> **English**: A diameter perpendicular to a chord bisects the chord and its arcs; conversely, the diameter bisecting a chord is perpendicular to it.

---

## 📚 给 6-12 岁（自读版）

### 垂径定理说什么？

"垂径"= 垂直于弦的直径。

**定理**：如果一条直径垂直于一条弦，那么它——
1. 平分这条弦（把弦切成两等份）
2. 平分弦所对的两段弧

**记忆口诀**：**垂直于弦的直径，三平分（弦、两弧）**

### 图形描述

在 $\odot O$ 中，$AB$ 是弦，$CD$ 是直径，$CD \perp AB$，垂足为 $M$：

- $AM = BM$（直径平分弦）
- $\overset{\frown}{AC} = \overset{\frown}{BC}$（平分上弧）
- $\overset{\frown}{AD} = \overset{\frown}{BD}$（平分下弧）

### 逆定理

- 弦的垂直平分线一定过圆心
- 平分弦的直径垂直于这条弦

### 实际应用

**如何找圆心？**

- 在圆上任取三点 $A, B, C$
- 作 $AB$ 的垂直平分线，作 $BC$ 的垂直平分线
- 两垂直平分线的交点就是圆心 $O$！

## 🎓 给 12+（进阶版）

### 定理陈述

**垂径定理**：在 $\odot O$ 中，若 $AB$ 是弦，直径 $CD$ 满足 $CD \perp AB$，垂足为 $M$，则：

$$AM = MB, \quad \overset{\frown}{AC} = \overset{\frown}{BC}, \quad \overset{\frown}{AD} = \overset{\frown}{BD}$$

**逆定理**：
- 若直径 $CD$ 平分弦 $AB$（$AM = MB$），则 $CD \perp AB$
- 弦 $AB$ 的垂直平分线过圆心 $O$ 并平分弧 $AB$

### 证明（垂径定理）

连 $OA$，$OB$，设 $OA = OB = r$，$OM \perp AB$。

在 $\triangle OAM$ 和 $\triangle OBM$ 中：
- $OA = OB = r$
- $OM = OM$（公共边）
- $\angle OMA = \angle OMB = 90°$

由勾股定理：$AM = \sqrt{OA^2 - OM^2} = \sqrt{OB^2 - OM^2} = BM$

故 $AM = BM$（弦被平分）。

弧相等：因 $OA = OB$，$AM = BM$，可得 $\angle AOC = \angle BOC$（对应的圆心角相等），故 $\overset{\frown}{AC} = \overset{\frown}{BC}$。

### 弦心距公式

设弦 $AB$ 的弦心距（圆心 $O$ 到弦 $AB$ 的垂直距离）为 $d$，半径为 $r$，半弦长为 $\frac{|AB|}{2}$：

$$d^2 + \left(\frac{|AB|}{2}\right)^2 = r^2$$

$$\Rightarrow d = \sqrt{r^2 - \frac{|AB|^2}{4}}, \quad |AB| = 2\sqrt{r^2 - d^2}$$

### 典型例题

**例 1**：$\odot O$ 半径 $r = 10$，弦 $AB = 16$，求弦心距。

$$d = \sqrt{100 - 64} = \sqrt{36} = 6$$

**例 2**：$\odot O$ 中，$AB \perp CD$，$AB$ 是直径，$M$ 是 $CD$ 中点，$OM = 3$，$CD = 8$，求半径。

由垂径定理（$AB \perp CD$，$AB$ 是直径），$M$ 是 $CD$ 中点，故 $CM = MD = 4$。

$OC^2 = OM^2 + CM^2 = 9 + 16 = 25$

$r = OC = 5$

**例 3**（作圆心）：已知圆弧上三点 $A, B, C$，用垂径定理确定圆心。

作 $AB$ 中点 $M$，过 $M$ 作 $AB$ 的垂线；作 $BC$ 中点 $N$，过 $N$ 作 $BC$ 的垂线；两垂线交点即为圆心 $O$。

**例 4**（综合）：$\odot O$ 中，两弦 $AB, CD$ 分别与直径 $MN$ 垂直，垂足分别为 $E, F$，且 $ME = MF$，求 $AB, CD$ 的关系。

$ME = MF \Rightarrow OE = OF$（因为 $OM$ 一定，$OE = |OM - ME|$ 或 $|OM + ME|$ 需具体分析）

若在直径同侧，$OE = OF \Rightarrow |AB| = |CD|$（弦心距相等 → 弦相等）

### 推论总结

同圆中：
- 等弦 ↔ 等弦心距
- 弦越长 ↔ 弦心距越小
- 最长弦（直径）的弦心距 = 0（过圆心）

### 易错点

- ❌ 弦的垂线（不是"直径"）不一定平分弦和弧
- ❌ 逆定理忘记加"直径"条件（平分弦的线段不一定是直径，除非过圆心）
- ❌ 弦心距公式中用直径而非半径
