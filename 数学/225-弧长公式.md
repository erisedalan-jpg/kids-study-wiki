# 弧长公式

> **一句话**：圆心角为 $n°$、半径为 $r$ 的弧，其弧长为 $l = \dfrac{n\pi r}{180}$。
> **English**: The arc length for a central angle of $n°$ in a circle of radius $r$ is $l = \frac{n\pi r}{180}$.

---

## 📚 给 6-12 岁（自读版）

### 弧长是什么？

弧长就是圆弧那段曲线的长度——不是直线距离，而是沿着弧弯过去的长度。

### 推导思路

整个圆的周长（360° 圆心角）：$C = 2\pi r$

圆心角 $n°$ 对应的弧，占整个圆的 $\dfrac{n}{360}$，所以：

$$l = \frac{n}{360} \times 2\pi r = \frac{n\pi r}{180}$$

### 公式

$$\boxed{l = \frac{n\pi r}{180}}$$

（$n$ 是圆心角的度数，$r$ 是半径）

### 例题

**例 1**：$r = 6$，圆心角 $60°$，求弧长。

$$l = \frac{60 \times \pi \times 6}{180} = \frac{360\pi}{180} = 2\pi \approx 6.28$$

**例 2**：$r = 3$，圆心角 $120°$，求弧长。

$$l = \frac{120\pi \times 3}{180} = \frac{360\pi}{180} = 2\pi$$

**特殊弧**：半圆弧长（$n = 180°$）：$l = \pi r$（即半个圆周）

## 🎓 给 12+（进阶版）

### 公式推导

设圆的半径为 $r$，圆心角为 $n°$：

$$l = \frac{n°}{360°} \times 2\pi r = \frac{n\pi r}{180}$$

**弧度制下的弧长公式**（高中预备）：

设圆心角的弧度为 $\alpha$（$\alpha = \frac{n\pi}{180}$），则：

$$l = \alpha r$$

（弧度制下的公式更简洁，是高中的重要内容）

### 弧长与圆心角、半径的关系

$$l = \frac{n\pi r}{180}$$

| 已知 | 求 |
|------|-----|
| $n, r$ | $l = \frac{n\pi r}{180}$ |
| $l, r$ | $n = \frac{180l}{\pi r}$ |
| $l, n$ | $r = \frac{180l}{n\pi}$ |

### 特殊圆心角的弧长

| 圆心角 | 弧长 |
|--------|------|
| $30°$ | $\frac{\pi r}{6}$ |
| $45°$ | $\frac{\pi r}{4}$ |
| $60°$ | $\frac{\pi r}{3}$ |
| $90°$ | $\frac{\pi r}{2}$ |
| $120°$ | $\frac{2\pi r}{3}$ |
| $180°$ | $\pi r$ |
| $360°$ | $2\pi r$ |

### 弧长与扇形面积的联系

$$S_{\text{扇形}} = \frac{n}{360}\pi r^2 = \frac{1}{2}lr$$

（$l$ 为弧长，$r$ 为半径——扇形面积 = $\frac{1}{2}$ × 弧长 × 半径，类似三角形面积 = $\frac{1}{2}$ × 底 × 高）

详见[[226-扇形面积公式|扇形面积公式]]。

### 典型例题

**例 1**（基础）：$\odot O$ 半径 10，圆心角 $\angle AOB = 90°$，求弧 $\overset{\frown}{AB}$ 的长。

$$l = \frac{90 \times \pi \times 10}{180} = 5\pi$$

**例 2**（逆用公式）：弧长 $l = 4\pi$，半径 $r = 6$，求圆心角。

$$n = \frac{180 \times 4\pi}{\pi \times 6} = \frac{720}{6} = 120°$$

**例 3**（弧长与扇形）：圆心角 $120°$，半径 5，求弧长和扇形面积。

$$l = \frac{120\pi \times 5}{180} = \frac{600\pi}{180} = \frac{10\pi}{3}$$

$$S = \frac{1}{2}lr = \frac{1}{2} \times \frac{10\pi}{3} \times 5 = \frac{25\pi}{3}$$

**例 4**（综合）：两个扇形弧长相等，半径分别为 3 和 4，大扇形圆心角 $60°$，求小扇形圆心角。

大扇形弧长：$l = \frac{60\pi \times 4}{180} = \frac{4\pi}{3}$

小扇形（$r=3$）：$\frac{4\pi}{3} = \frac{n\pi \times 3}{180}$，$n = \frac{4 \times 180}{3 \times 3} = 80°$

**例 5**（比较大小，不计算）：两个扇形，圆心角相同，$r_1 < r_2$，比较弧长。

$l = \frac{n\pi r}{180}$，$r$ 越大 $l$ 越大，故 $l_1 < l_2$。

### 易错点

- ❌ 将弧长与弦长混淆（弧长是曲线长度，弦长是直线距离）
- ❌ 公式中用弧度 $\alpha$ 时写成 $l = \frac{\alpha \pi r}{180}$（弧度制直接是 $l = \alpha r$，不乘 $\pi$）
- ❌ $n = 360°$ 时代公式算弧长 $= 2\pi r$（应直接用周长公式）
