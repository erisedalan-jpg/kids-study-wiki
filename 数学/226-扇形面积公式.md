# 扇形面积公式

> **一句话**：圆心角为 $n°$、半径为 $r$ 的扇形面积为 $S = \dfrac{n\pi r^2}{360}$；也可写成 $S = \dfrac{1}{2}lr$（$l$ 为弧长）。
> **English**: The area of a sector with central angle $n°$ and radius $r$ is $S = \frac{n\pi r^2}{360}$, or equivalently $S = \frac{1}{2}lr$.

---

## 📚 给 6-12 岁（自读版）

### 扇形是什么？

扇形 = 两条半径 + 一段弧围成的"饼片"形状（就像一片披萨！）

### 面积公式

整个圆面积 = $\pi r^2$（圆心角 360°）

圆心角 $n°$ 的扇形，占整个圆的 $\dfrac{n}{360}$，所以：

$$\boxed{S = \frac{n}{360}\pi r^2}$$

### 另一种公式（弧长版）

$$S = \frac{1}{2}lr$$

其中 $l$ 是弧长，$r$ 是半径。（类比三角形面积 $= \frac{1}{2} \times$ 底 $\times$ 高）

### 例题

**例 1**：$r = 6$，圆心角 $60°$，求扇形面积。

$$S = \frac{60}{360}\pi \times 36 = \frac{1}{6} \times 36\pi = 6\pi$$

**例 2**：弧长 $l = 4\pi$，$r = 3$，求扇形面积。

$$S = \frac{1}{2} \times 4\pi \times 3 = 6\pi$$

**例 3**：半圆面积（$n = 180°$，$r = 5$）。

$$S = \frac{180}{360}\pi \times 25 = \frac{25\pi}{2}$$

---

## 🎓 给 12+（进阶版）

### 两种公式推导

**公式1（圆心角度数版）**：

$$S_{\text{扇形}} = \frac{n°}{360°} \times \pi r^2 = \frac{n\pi r^2}{360}$$

**公式2（弧长版）推导**：

弧长 $l = \frac{n\pi r}{180}$，故 $n = \frac{180l}{\pi r}$，代入公式1：

$$S = \frac{180l}{\pi r} \cdot \frac{\pi r^2}{360} = \frac{lr}{2} = \frac{1}{2}lr$$

$$\boxed{S = \frac{1}{2}lr}$$

**弧度制下（高中预备）**：圆心角 $\alpha$（弧度），$l = \alpha r$：

$$S = \frac{1}{2}\alpha r^2$$

### 扇形面积与弧长对比

| 量 | 公式 |
|----|------|
| 弧长 | $l = \frac{n\pi r}{180}$ |
| 扇形面积 | $S = \frac{n\pi r^2}{360} = \frac{1}{2}lr$ |
| 圆周长 | $C = 2\pi r$（$n=360°$） |
| 圆面积 | $S = \pi r^2$（$n=360°$） |

### 由公式逆推

| 已知 | 求 |
|------|-----|
| $n, r$ | $S = \frac{n\pi r^2}{360}$ |
| $l, r$ | $S = \frac{1}{2}lr$ |
| $S, l$ | $r = \frac{2S}{l}$ |
| $S, r$ | $n = \frac{360S}{\pi r^2}$ |

### 典型例题

**例 1**（基础）：$r = 10$，$n = 90°$，求扇形面积和弧长。

$$l = \frac{90\pi \times 10}{180} = 5\pi, \quad S = \frac{1}{2} \times 5\pi \times 10 = 25\pi$$

验证：$S = \frac{90}{360}\pi \times 100 = 25\pi$ ✓

**例 2**（逆推半径）：扇形面积 $12\pi$，弧长 $6\pi$，求半径和圆心角。

$$r = \frac{2S}{l} = \frac{24\pi}{6\pi} = 4$$

$$n = \frac{180l}{\pi r} = \frac{180 \times 6\pi}{\pi \times 4} = 270°$$

**例 3**（组合图形）：半径 10 的圆，去掉圆心角 $120°$ 的扇形，剩余面积是多少？

扇形面积：$S_{\text{扇形}} = \frac{120}{360}\pi \times 100 = \frac{100\pi}{3}$

剩余：$S = 100\pi - \frac{100\pi}{3} = \frac{200\pi}{3}$

**例 4**（图形题）：如图，$\odot O$ 半径 6，$\angle AOB = 120°$，$OC$ 平分 $\angle AOB$，$OC \perp AB$。阴影部分面积 = 扇形 $AOB$ 面积 - $\triangle AOB$ 面积。

扇形面积：$S_{\text{扇形}} = \frac{120}{360}\pi \times 36 = 12\pi$

$\triangle AOB$ 面积：$OA = OB = 6$，$\angle AOB = 120°$，$OC = 6\cos 60° = 3$，$AB = 2 \times 6\sin 60° = 6\sqrt{3}$

$S_{\triangle} = \frac{1}{2} \times 6\sqrt{3} \times 3 = 9\sqrt{3}$

阴影面积 $= 12\pi - 9\sqrt{3}$

**例 5**（周长和面积混合）：扇形的弧长等于其内切圆的周长（已知圆心角 $60°$，$r = 6$）。

弧长：$l = \frac{60\pi \times 6}{180} = 2\pi$

内切圆周长 $2\pi = 2\pi r'$，$r' = 1$（内切圆半径为 1）

### 与圆的综合计算

**圆环面积**：大圆半径 $R$，小圆半径 $r$（同心），面积差 $= \pi(R^2 - r^2)$

**弓形面积**：弓形 = 扇形 - 三角形

$$S_{\text{弓形}} = S_{\text{扇形}} - S_{\triangle} = \frac{n\pi r^2}{360} - \frac{1}{2}r^2\sin n°$$

### 易错点

- ❌ 分母写 180 而不是 360（弧长公式分母是 180，面积公式分母是 360！）
- ❌ $S = \frac{1}{2}lr$ 中把 $l$ 和 $r$ 的位置搞混
- ❌ 计算弓形面积时忘记减去三角形部分
- ❌ 圆心角和扇形面积都和 $r^2$ 成正比，弧长和 $r$ 成正比（不要混淆次数）
