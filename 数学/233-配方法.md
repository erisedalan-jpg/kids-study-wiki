# 配方法

> **一句话**：通过添加或减去适当的数，将二次三项式变为完全平方式，从而把一元二次方程化为 $(x+m)^2 = n$ 的形式来求解，这种方法叫做配方法。
> **English**: Completing the square transforms a quadratic equation into the form $(x+m)^2 = n$ by adding a constant to both sides.

---

## 📚 给 6-12 岁（自读版）

### 什么叫"配方"？

"配方"就像拼拼图——缺一块就补一块，使它变成"完整的正方形"。

数学中，$(x+3)^2 = x^2 + 6x + 9$，这叫**完全平方式**。

如果方程是 $x^2 + 6x + 5 = 0$，我们怎么配出完全平方呢？

$$x^2 + 6x + \underbrace{9}_{\text{补！}} = -5 + 9$$
$$(x+3)^2 = 4$$
$$x + 3 = \pm 2 \Rightarrow x_1 = -1,\ x_2 = -5$$

### 配方的口诀

**一次项系数的一半，平方后加两边**

- 一次项系数 $6$ → 一半是 $3$ → 平方是 $9$
- 左边加 $9$，右边也加 $9$，保持等号

### 步骤演示

解 $x^2 - 4x - 3 = 0$：

1. 移常数项：$x^2 - 4x = 3$
2. 配方（加 $\left(\frac{-4}{2}\right)^2 = 4$）：$x^2 - 4x + 4 = 7$
3. 写成完全平方：$(x-2)^2 = 7$
4. 开平方：$x - 2 = \pm\sqrt{7}$
5. 解：$x_1 = 2 + \sqrt{7},\ x_2 = 2 - \sqrt{7}$

## 🎓 给 12+（进阶版）

### 一般步骤

对方程 $ax^2 + bx + c = 0$（$a \neq 0$）：

**步骤 1**：若 $a \neq 1$，两边除以 $a$：
$$x^2 + \frac{b}{a}x + \frac{c}{a} = 0$$

**步骤 2**：常数项移右边：
$$x^2 + \frac{b}{a}x = -\frac{c}{a}$$

**步骤 3**：两边加 $\left(\dfrac{b}{2a}\right)^2$：
$$x^2 + \frac{b}{a}x + \frac{b^2}{4a^2} = \frac{b^2}{4a^2} - \frac{c}{a}$$

**步骤 4**：左边变完全平方：
$$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$$

**步骤 5**：开平方（若右边 $\geq 0$）：
$$x + \frac{b}{2a} = \pm\frac{\sqrt{b^2-4ac}}{2a}$$

**步骤 6**：得到求根公式：
$$x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}$$

配方法是推导求根公式的基础！

### 典型例题

**例 1**：解 $2x^2 - 6x - 1 = 0$

$$x^2 - 3x - \frac{1}{2} = 0$$
$$x^2 - 3x = \frac{1}{2}$$
$$x^2 - 3x + \frac{9}{4} = \frac{1}{2} + \frac{9}{4} = \frac{11}{4}$$
$$\left(x - \frac{3}{2}\right)^2 = \frac{11}{4}$$
$$x - \frac{3}{2} = \pm\frac{\sqrt{11}}{2}$$
$$x_1 = \frac{3+\sqrt{11}}{2},\quad x_2 = \frac{3-\sqrt{11}}{2}$$

**例 2**（配方法求顶点）：将 $y = x^2 - 4x + 1$ 配方为顶点式。

$$y = (x-2)^2 - 3$$

顶点为 $(2, -3)$。（此应用见[[214-二次函数顶点式|二次函数顶点式]]）

### 配方法的价值

1. **推导求根公式**：配方法 → [[215-公式法-一元二次方程|公式法-一元二次方程]]
2. **求二次函数顶点**：$y = a(x-h)^2 + k$ → [[214-二次函数顶点式|二次函数顶点式]]
3. **判断函数最值**：顶点式直接读出最大/最小值

### 公式汇总

$$\left(x + \frac{b}{2a}\right)^2 = \frac{b^2 - 4ac}{4a^2}$$

完全平方公式（配方基础）：
$$(x \pm m)^2 = x^2 \pm 2mx + m^2$$

### 易错点

- ❌ 配方时两边只加一边（破坏等式）
- ❌ $a \neq 1$ 时忘记先除以 $a$ 再配方
- ❌ 开平方后忘记 $\pm$
- ❌ $\left(\frac{b}{2}\right)^2$ 计算错误（如 $b=-4$ 时应为 $4$，而非 $-4$）
