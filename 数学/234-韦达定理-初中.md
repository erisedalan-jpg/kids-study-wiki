# 韦达定理-初中

> **一句话**：一元二次方程 $ax^2+bx+c=0$ 的两根 $x_1, x_2$ 满足：$x_1 + x_2 = -\dfrac{b}{a}$，$x_1 x_2 = \dfrac{c}{a}$，这叫韦达定理（根与系数关系）。
> **English**: Vieta's formulas: if $x_1, x_2$ are roots of $ax^2+bx+c=0$, then $x_1+x_2=-b/a$ and $x_1 x_2=c/a$.

---

## 📚 给 6-12 岁（自读版）

### 神奇的根与系数关系

不用真的解出方程，仅凭 $a, b, c$ 就能知道两个根的**和**与**积**！

对于 $ax^2 + bx + c = 0$（$a \neq 0$，有两个实数根 $x_1, x_2$）：

$$\boxed{x_1 + x_2 = -\frac{b}{a}, \qquad x_1 \cdot x_2 = \frac{c}{a}}$$

### 例题感受神奇

方程 $x^2 - 5x + 6 = 0$，$a=1, b=-5, c=6$：

- 两根之和：$x_1 + x_2 = -\dfrac{-5}{1} = 5$
- 两根之积：$x_1 \cdot x_2 = \dfrac{6}{1} = 6$

直接解方程得 $x_1 = 2, x_2 = 3$，验证：$2+3=5$ ✓，$2 \times 3=6$ ✓

### 记忆方法

$$x^2 - \underbrace{(x_1+x_2)}_{\text{和}}x + \underbrace{x_1 x_2}_{\text{积}} = 0$$

即：知道两根的**和**与**积**，就能反向写出方程！

## 🎓 给 12+（进阶版）

### 定理陈述与证明

**定理**：若 $x_1, x_2$ 是方程 $ax^2+bx+c=0$（$a \neq 0$，$\Delta \geq 0$）的两个根，则：

$$x_1 + x_2 = -\frac{b}{a}, \qquad x_1 x_2 = \frac{c}{a}$$

**证明**（由求根公式）：

$$x_1 = \frac{-b+\sqrt{\Delta}}{2a}, \quad x_2 = \frac{-b-\sqrt{\Delta}}{2a}$$

$$x_1 + x_2 = \frac{(-b+\sqrt{\Delta})+(-b-\sqrt{\Delta})}{2a} = \frac{-2b}{2a} = -\frac{b}{a}$$

$$x_1 x_2 = \frac{(-b+\sqrt{\Delta})(-b-\sqrt{\Delta})}{4a^2} = \frac{b^2-\Delta}{4a^2} = \frac{b^2-(b^2-4ac)}{4a^2} = \frac{4ac}{4a^2} = \frac{c}{a}$$

### 韦达定理的逆

若实数 $x_1, x_2$ 满足 $x_1+x_2 = S$，$x_1 x_2 = P$，则 $x_1, x_2$ 是方程：

$$x^2 - Sx + P = 0$$

的两个根（构造方程逆用韦达定理）。

### 五类应用

**应用1：验算根的正确性**

解方程后，用韦达定理验证根之和与积是否符合。

**应用2：不解方程求对称式**

**例**：$x_1, x_2$ 是 $x^2 - 3x + 1 = 0$ 的两根，求 $x_1^2 + x_2^2$。

$x_1 + x_2 = 3,\ x_1 x_2 = 1$

$$x_1^2 + x_2^2 = (x_1+x_2)^2 - 2x_1 x_2 = 9 - 2 = 7$$

**应用3：求含根的表达式**

**例**：求 $\dfrac{1}{x_1} + \dfrac{1}{x_2}$（其中 $x_1 x_2 \neq 0$）

$$\frac{1}{x_1} + \frac{1}{x_2} = \frac{x_1+x_2}{x_1 x_2} = \frac{-b/a}{c/a} = -\frac{b}{c}$$

对上例：$= \dfrac{3}{1} = 3$

**应用4：构造方程**

**例**：以 $2, -5$ 为两根构造方程。

和 $= 2 + (-5) = -3$，积 $= 2 \times (-5) = -10$

方程：$x^2 - (-3)x + (-10) = 0$，即 $x^2 + 3x - 10 = 0$

**应用5：含参数求值**

**例**：$x^2 - (k-2)x + 3k = 0$ 的两根互为相反数，求 $k$。

互为相反数 $\Rightarrow x_1 + x_2 = 0 \Rightarrow -(k-2) = 0 \Rightarrow k = 2$

验证：$\Delta = 4 - 12 < 0$（无实根！），故 $k=2$ 不满足条件 → 题目无解（待家长核对）

### 常用对称式恒等式

| 表达式 | 化简 |
|--------|------|
| $x_1^2 + x_2^2$ | $(x_1+x_2)^2 - 2x_1 x_2$ |
| $x_1^3 + x_2^3$ | $(x_1+x_2)^3 - 3x_1 x_2(x_1+x_2)$ |
| $(x_1 - x_2)^2$ | $(x_1+x_2)^2 - 4x_1 x_2$ |
| $\frac{1}{x_1}+\frac{1}{x_2}$ | $\frac{x_1+x_2}{x_1 x_2}$ |
| $x_1^2 x_2 + x_1 x_2^2$ | $x_1 x_2(x_1+x_2)$ |

### 易错点

- ❌ 用韦达定理时忘记先验证 $\Delta \geq 0$（必须有实数根才能使用）
- ❌ 符号错误：$x_1+x_2 = -b/a$（不是 $b/a$！）
- ❌ 对非标准形式的方程（$a \neq 1$）仍用 $x_1+x_2 = -b$
- ❌ 两根互为相反数 $\Rightarrow$ 和为 0；两根互为倒数 $\Rightarrow$ 积为 1（不要混淆）
