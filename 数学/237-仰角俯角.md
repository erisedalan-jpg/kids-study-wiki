# 仰角俯角

> **一句话**：从水平线向上看目标的角叫仰角，从水平线向下看目标的角叫俯角，两者在三角函数应用题中常见。

---

## 📚 给 6-12 岁（自读版）

### 仰角和俯角是什么？

**仰角**：站在地面，**抬头**看高处目标，视线与水平方向的夹角。
- 例：抬头看楼顶、山峰、飞机

**俯角**：站在高处，**低头**看低处目标，视线与水平方向的夹角。
- 例：从山顶看山脚、从楼上看地面

### 记忆方法

- 仰 = 仰起头向上看 → 仰角
- 俯 = 俯下头向下看 → 俯角

### 图示关系

```
目标（高处）
   ↗ 仰角α
观察者 ————— 水平线
```

```
观察者（高处）
      ↘ 俯角β
水平线 ————— 目标（低处）
```

## 🎓 给 12+（进阶版）

### 标准定义

**仰角**：在同一竖直平面内，从水平视线**仰视**（向上看）目标，视线与水平线所成的角（$0° < \alpha < 90°$）。

**俯角**：在同一竖直平面内，从水平视线**俯视**（向下看）目标，视线与水平线所成的角（$0° < \beta < 90°$）。

### 关键几何关系

设观察者位于 $A$，目标位于 $B$，$AH$ 为水平线：

**仰角情形**（$B$ 在 $A$ 上方）：

$$\tan \alpha = \frac{h}{d}$$

其中 $h$ 为高差，$d$ 为水平距离。

**俯角情形**（$B$ 在 $A$ 下方）：

$$\tan \beta = \frac{h}{d}$$

形式相同，但 $A$ 为高处观察点，$B$ 为低处目标。

### 仰角与俯角的互余关系

若 $A$ 站在地面看 $B$ 的仰角为 $\alpha$，则 $B$ 看 $A$ 的**俯角也是 $\alpha$**（内错角/等腰原理：同一竖直面内，仰角和俯角关于水平线对称，故相等）。

### 典型例题

**例 1（仰角）**：站在离一幢楼 20 m 处，测得楼顶的仰角为 60°，求楼高（忽略人的身高）。

解：$h = d \times \tan \alpha = 20 \times \tan 60° = 20\sqrt{3} \approx 34.6$ m。

**例 2（俯角）**：从一座 30 m 高的塔顶，测得地面上一点的俯角为 45°，求该点到塔底的距离。

解：$d = \dfrac{h}{\tan \beta} = \dfrac{30}{\tan 45°} = 30$ m。

**例 3（综合）**：在河的两岸，$A$ 在南岸，$B$ 在北岸，$AB$ 为河宽。在 $A$ 处测得某山顶 $C$ 的仰角为 30°，$AC=60$ m，求山高（$BC \perp AB$）。

解：$BC = AC \times \tan 30° = 60 \times \dfrac{\sqrt{3}}{3} = 20\sqrt{3}$ m。

### 易错点

- ❌ 混淆仰角和俯角（是**从**哪里看决定的，不是目标在上还是在下）
- ❌ 忘记"水平距离"（$d$）和"斜线距离"的区别
