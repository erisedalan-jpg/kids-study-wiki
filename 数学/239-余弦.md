# 余弦

> **一句话**：在直角三角形中，某锐角的余弦等于该角的邻边与斜边之比。
> **English**: In a right triangle, the cosine of an acute angle equals the ratio of the adjacent side to the hypotenuse.

---

## 📚 给 6-12 岁（自读版）

### 记忆口诀

$$\cos A = \frac{\text{邻边}}{\text{斜边}} = \frac{\text{A旁边的直角边}}{\text{最长边}}$$

**口诀**：**余弦 = 邻/斜**

### 图示

在直角三角形 $ABC$（$\angle C = 90°$）中，对于 $\angle A$：
- 邻边：$AC$（$A$ 旁边的直角边）
- 斜边：$AB$（直角对面最长的边）

$$\cos A = \frac{AC}{AB}$$

### 特殊角值

| 角度 $A$ | $\cos A$ | 近似值 |
|---------|----------|--------|
| 30° | $\dfrac{\sqrt{3}}{2}$ | ≈ 0.866 |
| 45° | $\dfrac{\sqrt{2}}{2}$ | ≈ 0.707 |
| 60° | $\dfrac{1}{2}$ | 0.5 |

注意：$\cos 30° = \sin 60°$，$\cos 60° = \sin 30°$（互余关系）。

## 🎓 给 12+（进阶版）

### 定义

在直角三角形 $ABC$ 中，$\angle C = 90°$，对于锐角 $A$：

$$\boxed{\cos A = \frac{b}{c}}$$

其中 $b = AC$（$\angle A$ 的邻边），$c = AB$（斜边）。

### 推导特殊角值

**$\cos 30°$**：30-60-90 三角形，$30°$ 角的邻边（较长直角边）与斜边之比 $= \dfrac{\sqrt{3}}{2}$。

**$\cos 45°$**：等腰直角三角形，邻边 1，斜边 $\sqrt{2}$，$\cos 45° = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}$。

**$\cos 60°$**：30-60-90 三角形，$60°$ 角的邻边（较短直角边）与斜边之比 $= \dfrac{1}{2}$。

### 余弦值的范围

对于锐角（$0° < A < 90°$）：$0 < \cos A < 1$

$\cos A$ 随 $A$ 增大而**减小**（单调递减）。

### 互余角关系

$$\cos A = \sin(90° - A)$$

### 正弦与余弦的关系

$$\sin^2 A + \cos^2 A = 1$$

证明：$\sin^2 A + \cos^2 A = \dfrac{a^2}{c^2} + \dfrac{b^2}{c^2} = \dfrac{a^2+b^2}{c^2} = \dfrac{c^2}{c^2} = 1$（由勾股定理）。

### 典型例题

**例 1**：$\angle C=90°$，$AC=4$，$AB=5$，求 $\cos A$ 和 $\cos B$。

解：$BC = \sqrt{5^2-4^2} = 3$。
$\cos A = \dfrac{AC}{AB} = \dfrac{4}{5}$，$\cos B = \dfrac{BC}{AB} = \dfrac{3}{5}$。

**例 2**：已知 $\cos A = \dfrac{3}{5}$，斜边 $c=10$，求邻边 $b$。

解：$b = c \times \cos A = 10 \times \dfrac{3}{5} = 6$。

**例 3**：已知 $\sin A = \dfrac{3}{5}$，求 $\cos A$（锐角）。

解：$\cos A = \sqrt{1 - \sin^2 A} = \sqrt{1 - \dfrac{9}{25}} = \sqrt{\dfrac{16}{25}} = \dfrac{4}{5}$。

### 易错点

- ❌ 混淆邻边和对边（余弦用**邻**边，正弦用**对**边）
- ❌ $\cos A$ 随角度增大而减小，与正弦相反
