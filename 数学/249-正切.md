# 正切

> **一句话**：在直角三角形中，某锐角的正切等于该角的对边与邻边之比。
> **English**: In a right triangle, the tangent of an acute angle equals the ratio of the opposite side to the adjacent side.

---

## 📚 给 6-12 岁（自读版）

### 记忆口诀

$$\tan A = \frac{\text{对边}}{\text{邻边}} = \frac{\text{A对面的边}}{\text{A旁边的直角边}}$$

**口诀**：**正切 = 对/邻**（注意：不含斜边！）

### 图示

在直角三角形 $ABC$（$\angle C = 90°$）中，对于 $\angle A$：
- 对边：$BC$
- 邻边：$AC$

$$\tan A = \frac{BC}{AC}$$

### 特殊角值

| 角度 $A$ | $\tan A$ | 近似值 |
|---------|----------|--------|
| 30° | $\dfrac{\sqrt{3}}{3}$（即 $\dfrac{1}{\sqrt{3}}$）| ≈ 0.577 |
| 45° | $1$ | 1.000 |
| 60° | $\sqrt{3}$ | ≈ 1.732 |

$\tan 45° = 1$，对边等于邻边（等腰直角三角形）。

## 🎓 给 12+（进阶版）

### 定义

在直角三角形 $ABC$ 中，$\angle C = 90°$，对于锐角 $A$：

$$\boxed{\tan A = \frac{a}{b} = \frac{\sin A}{\cos A}}$$

其中 $a = BC$（对边），$b = AC$（邻边）。

### 三角函数之间的关系

$$\tan A = \frac{\sin A}{\cos A}$$

证明：$\dfrac{\sin A}{\cos A} = \dfrac{a/c}{b/c} = \dfrac{a}{b} = \tan A$。

### 正切值的范围

对于锐角（$0° < A < 90°$）：$\tan A > 0$

$\tan A$ 随 $A$ 增大而**增大**（单调递增），且趋向无穷大：当 $A \to 90°$ 时，$\tan A \to +\infty$。

### 特殊值推导

**$\tan 30°$**：30-60-90 三角形，对边 $1$，邻边 $\sqrt{3}$，$\tan 30° = \dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}$。

**$\tan 45°$**：等腰直角三角形，对边 = 邻边 = 1，$\tan 45° = 1$。

**$\tan 60°$**：30-60-90，对边 $\sqrt{3}$，邻边 $1$，$\tan 60° = \sqrt{3}$。

### 正切与坡度

坡度（坡比）就是正切的应用：坡高与坡长之比 $= \tan(\text{坡角})$。

### 典型例题

**例 1**：$\angle C=90°$，$BC=5$，$AC=12$，求 $\tan A$、$\tan B$。

解：$\tan A = \dfrac{BC}{AC} = \dfrac{5}{12}$，$\tan B = \dfrac{AC}{BC} = \dfrac{12}{5}$。

注意：$\tan A \times \tan B = 1$（两者互为倒数，因为 $A+B=90°$）。

**例 2**：已知 $\tan A = \dfrac{3}{4}$，$BC=6$（对边），求 $AC$ 和 $AB$。

解：$\dfrac{BC}{AC} = \dfrac{3}{4}$，$AC = BC \times \dfrac{4}{3} = 8$。
$AB = \sqrt{BC^2+AC^2} = \sqrt{36+64} = 10$。

### 三角函数总表

| 函数 | 定义 | 口诀 |
|------|------|------|
| $\sin A$ | 对/斜 | 正弦对斜 |
| $\cos A$ | 邻/斜 | 余弦邻斜 |
| $\tan A$ | 对/邻 | 正切对邻 |

### 易错点

- ❌ 正切公式写成对/斜（那是正弦）
- ❌ 互余角的正切关系：$\tan A \cdot \tan(90°-A) = 1$，不是相等
