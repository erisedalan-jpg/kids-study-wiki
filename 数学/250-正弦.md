# 正弦

> **一句话**：在直角三角形中，某锐角的正弦等于该角的对边与斜边之比。
> **English**: In a right triangle, the sine of an acute angle equals the ratio of the opposite side to the hypotenuse.

---

## 📚 给 6-12 岁（自读版）

### 记忆口诀

$$\sin A = \frac{\text{对边}}{\text{斜边}} = \frac{\text{A对面的边}}{\text{最长边}}$$

**口诀**：**正弦 = 对/斜**

### 图示

在直角三角形 $ABC$（$\angle C = 90°$）中，对于 $\angle A$：
- 对边：$BC$（$A$ 对面的边）
- 斜边：$AB$（直角对面最长的边）

$$\sin A = \frac{BC}{AB}$$

### 特殊角值

| 角度 $A$ | $\sin A$ | 近似值 |
|---------|----------|--------|
| 30° | $\dfrac{1}{2}$ | 0.5 |
| 45° | $\dfrac{\sqrt{2}}{2}$ | ≈ 0.707 |
| 60° | $\dfrac{\sqrt{3}}{2}$ | ≈ 0.866 |

---

## 🎓 给 12+（进阶版）

### 定义

在直角三角形 $ABC$ 中，$\angle C = 90°$，对于锐角 $A$：

$$\boxed{\sin A = \frac{a}{c}}$$

其中 $a = BC$（$\angle A$ 的对边），$c = AB$（斜边）。

### 推导特殊角值

**$\sin 30°$**：等边三角形作高，得 30-60-90 三角形，边长比 $1:2$，故 $\sin 30° = \dfrac{1}{2}$。

**$\sin 45°$**：等腰直角三角形，直角边 1，斜边 $\sqrt{2}$，故 $\sin 45° = \dfrac{1}{\sqrt{2}} = \dfrac{\sqrt{2}}{2}$。

**$\sin 60°$**：同上 30-60-90 三角形，故 $\sin 60° = \dfrac{\sqrt{3}}{2}$。

### 正弦值的范围

对于锐角（$0° < A < 90°$）：$0 < \sin A < 1$

$\sin A$ 随 $A$ 增大而增大（单调递增）。

### 互余角关系

$$\sin A = \cos(90° - A)$$

即：两个互余角，一个角的正弦等于另一个角的余弦。

### 典型例题

**例 1**：直角三角形中，$\angle C=90°$，$BC=3$，$AB=5$，求 $\sin A$。

解：$\sin A = \dfrac{BC}{AB} = \dfrac{3}{5}$。

**例 2**：已知 $\sin A = \dfrac{5}{13}$，$AB=26$，求 $BC$。

解：$BC = AB \times \sin A = 26 \times \dfrac{5}{13} = 10$。

**例 3**：已知 $\sin A = \dfrac{\sqrt{3}}{2}$，判断 $\angle A$ 的度数。

解：$\angle A = 60°$（查特殊角表）。

### 易错点

- ❌ 混淆对边和邻边（正弦用**对**边，余弦用**邻**边）
- ❌ 在非直角三角形中套用这个定义（初中阶段仅针对直角三角形中的锐角）
