# 比例的基本性质

> **一句话**：若 $\dfrac{a}{b} = \dfrac{c}{d}$，则 $ad = bc$（比例基本性质：内项之积等于外项之积）。

---

## 📚 给 6-12 岁（自读版）

### 什么是比例？

把两个相等的比写在一起，叫做**比例**：

$$\frac{2}{4} = \frac{3}{6}$$

读作：2 比 4 等于 3 比 6。

### 基本性质（交叉相乘）

$$\frac{a}{b} = \frac{c}{d} \implies a \times d = b \times c$$

就是**对角线上的数相乘相等**：外项 $a$、$d$ 之积 = 内项 $b$、$c$ 之积。

**验证**：$\dfrac{2}{4} = \dfrac{3}{6}$，则 $2\times6=12$，$4\times3=12$ ✅

### 用来解比例方程

$$\frac{x}{6} = \frac{4}{3}$$

$$3x = 24 \implies x = 8$$

## 🎓 给 12+（进阶版）

### 比例的定义

若 $\dfrac{a}{b} = \dfrac{c}{d}$（$b,d \neq 0$），则称 $a:b:c:d$ 构成比例，其中 $a$、$d$ 称为**外项**，$b$、$c$ 称为**内项**。

### 基本性质

$$\frac{a}{b} = \frac{c}{d} \iff ad = bc$$

### 比例的变换性质

由 $\dfrac{a}{b} = \dfrac{c}{d}$ 可推出：

| 变换名称 | 结论 |
|---------|------|
| 更比 | $\dfrac{a}{c} = \dfrac{b}{d}$ |
| 反比 | $\dfrac{b}{a} = \dfrac{d}{c}$ |
| 合比 | $\dfrac{a+b}{b} = \dfrac{c+d}{d}$ |
| 分比 | $\dfrac{a-b}{b} = \dfrac{c-d}{d}$ |
| 合分比 | $\dfrac{a+b}{a-b} = \dfrac{c+d}{c-d}$ |

### 等比性质（连比）

若 $\dfrac{a_1}{b_1} = \dfrac{a_2}{b_2} = \cdots = \dfrac{a_n}{b_n} = k$，则：

$$\frac{a_1+a_2+\cdots+a_n}{b_1+b_2+\cdots+b_n} = k$$

即所有分子之和与所有分母之和的比等于公比。

### 典型例题

**例 1**：已知 $\dfrac{x}{3} = \dfrac{8}{12}$，求 $x$。

解：$12x = 24$，$x = 2$。

**例 2**：已知 $\dfrac{a}{b} = \dfrac{3}{5}$，求 $\dfrac{a+b}{b}$。

解：由合比性质：$\dfrac{a+b}{b} = \dfrac{3+5}{5} = \dfrac{8}{5}$。

**例 3（等比）**：已知 $\dfrac{a}{2} = \dfrac{b}{3} = \dfrac{c}{5}$，求 $\dfrac{a+b+c}{10}$。

解：设公比为 $k$，则 $a=2k,b=3k,c=5k$，故 $a+b+c=10k$，
$\dfrac{a+b+c}{10} = k$，即等于公比 $k$。

### 易错点

- ❌ 交叉相乘时乘错方向（$ad=bc$，不是 $ac=bd$）
- ❌ 合比公式记错，分子分母都要加
