# 相似三角形判定-AA

> **一句话**：两角对应相等（AA），则两三角形相似。

---

## 📚 给 6-12 岁（自读版）

### AA 判定

如果一个三角形的**两个角**分别等于另一个三角形的**两个角**，那么这两个三角形相似。

**为什么只需两个角？**

三角形三个角之和等于 180°，知道两个角，第三个角自动确定：$\angle C = 180° - \angle A - \angle B$。所以两个角相等，第三个角也自动相等！

### 常见场景

**场景 1**：两个三角形都有一个公共角，再有一对对应角相等。

**场景 2**：两个三角形都是直角三角形，再有一对锐角相等（直角 + 一个锐角相等，AA 成立）。

---

## 🎓 给 12+（进阶版）

### 定理

**AA 判定定理**：若 $\triangle ABC$ 和 $\triangle DEF$ 中，$\angle A = \angle D$，$\angle B = \angle E$，则 $\triangle ABC \sim \triangle DEF$。

证明思路：由 $\angle A = \angle D, \angle B = \angle E$ 可得 $\angle C = 180° - \angle A - \angle B = 180° - \angle D - \angle E = \angle F$，再利用正弦定理可证对应边成比例（或直接用比例截线证明）。

### 常用推论

**推论 1**：直角三角形中，若两个直角三角形有一对锐角相等，则两三角形相似（RA 推出 AA）。

**推论 2**：两个等边三角形相似（三角均为 60°）。

### 典型例题

**例 1**：$\triangle ABC$ 中，$D$ 是 $BC$ 上一点，$\angle ADB = \angle BAC$，证明 $\triangle ABD \sim \triangle CAB$。

证明：
- $\angle ADB = \angle BAC$（已知）
- $\angle B = \angle B$（公共角）
- 由 AA 判定：$\triangle ABD \sim \triangle CAB$。

**例 2**：如图，$AB \parallel CD$，$AB$ 和 $CD$ 交于点 $O$，证明 $\triangle AOB \sim \triangle DOC$。

证明：
- $\angle OAB = \angle ODC$（$AB \parallel CD$，内错角相等）
- $\angle AOB = \angle DOC$（对顶角相等）
- AA 判定：$\triangle AOB \sim \triangle DOC$。

### AA 的实用性

AA 是三个判定定理中**最常用**的，因为：
- 角度关系往往比边长关系更容易从图形中读出
- 平行线、对顶角、公共角等经常产生相等的角

### 易错点

- ❌ 两角相等但顶点对应顺序写错（结论中的 $\sim$ 号两侧三角形顶点必须按对应顺序）
- ❌ 只写"两角相等"不写具体哪两对角
