# 相似三角形定义

> **一句话**：对应角相等、对应边成比例的两个三角形叫做相似三角形，相似比是对应边的比值。

---

## 📚 给 6-12 岁（自读版）

### 相似三角形是什么？

两个三角形，**形状完全一样，大小可以不同**，就是相似三角形。

就像用复印机把三角形放大或缩小——放大后和原来的三角形就是相似三角形。

### 两个条件

$\triangle ABC \sim \triangle DEF$（读作：三角形 ABC 相似于三角形 DEF），需要：
1. 三对对应角相等：$\angle A = \angle D$，$\angle B = \angle E$，$\angle C = \angle F$
2. 三对对应边成比例：$\dfrac{AB}{DE} = \dfrac{BC}{EF} = \dfrac{AC}{DF}$

### 相似比

$$\frac{AB}{DE} = \frac{BC}{EF} = \frac{AC}{DF} = k \quad \text{（相似比）}$$

---

## 🎓 给 12+（进阶版）

### 定义

若 $\triangle ABC$ 与 $\triangle DEF$ 满足：

$$\angle A = \angle D, \quad \angle B = \angle E, \quad \angle C = \angle F$$

$$\frac{AB}{DE} = \frac{BC}{EF} = \frac{CA}{FD} = \lambda$$

则 $\triangle ABC \sim \triangle DEF$，$\lambda$ 为相似比（$\triangle ABC$ 与 $\triangle DEF$ 的相似比）。

### 相似与全等的关系

| | 全等 $(\cong)$ | 相似 $(\sim)$ |
|--|------------|------------|
| 对应角 | 相等 | 相等 |
| 对应边 | 相等 | 成比例 |
| 相似比 | $1:1$ | 任意正比 |

全等是相似比为 $1$ 的特例：$\triangle ABC \cong \triangle DEF \Rightarrow \triangle ABC \sim \triangle DEF$（比反之不成立）。

### 顶点对应顺序很重要！

$\triangle ABC \sim \triangle DEF$ 中，顶点字母顺序表示对应关系：
- $A \leftrightarrow D$，$B \leftrightarrow E$，$C \leftrightarrow F$

书写错误会导致对应边判断错误！

### 相似三角形中的比例关系

由 $\triangle ABC \sim \triangle DEF$（相似比 $\lambda = m/n$）可得：
- 对应边之比 $= m:n$
- 对应高之比 $= m:n$
- 对应中线之比 $= m:n$
- 对应角平分线之比 $= m:n$
- 周长之比 $= m:n$
- 面积之比 $= m^2:n^2$

### 典型例题

**例**：$\triangle ABC \sim \triangle DEF$，$AB=4,BC=6,AC=8,DE=6$，求 $EF$ 和 $DF$。

解：相似比 $\lambda = \dfrac{DE}{AB} = \dfrac{6}{4} = \dfrac{3}{2}$。

$$EF = BC \times \frac{3}{2} = 9, \quad DF = AC \times \frac{3}{2} = 12$$

### 易错点

- ❌ 顶点对应顺序写错，导致对应边搞混
- ❌ 认为三角形只要三角相等就足够（三角形此时确实相似，但其他多边形不行）
