# 锐角三角函数互余关系

> **一句话**：两个互余角（和为 90°）的三角函数满足：$\sin\alpha = \cos(90°-\alpha)$，$\tan\alpha \cdot \tan(90°-\alpha) = 1$。

---

## 📚 给 6-12 岁（自读版）

### 什么是互余角？

两个角相加等于 90°，叫做**互余角**（互补是 180°，不要混）。

例：30° 和 60° 互余，45° 和 45° 互余。

### 互余关系

在直角三角形 $ABC$（$\angle C=90°$）中，$\angle A + \angle B = 90°$，即 $B = 90° - A$。

观察：对于 $\angle A$，BC 是对边，AC 是邻边；
对于 $\angle B$，AC 是对边，BC 是邻边——两个角的对边和邻边**互换**了！

因此：

$$\sin A = \cos B = \cos(90°-A)$$
$$\cos A = \sin B = \sin(90°-A)$$
$$\tan A = \frac{1}{\tan B} = \frac{1}{\tan(90°-A)}$$

### 口诀

**正弦和余弦是一家**：$\sin$（正弦）和 $\cos$（余弦），$\cos$ 里的"余"正是"互余"的余！

## 🎓 给 12+（进阶版）

### 互余关系公式

$$\boxed{\sin\alpha = \cos(90°-\alpha)}$$
$$\boxed{\cos\alpha = \sin(90°-\alpha)}$$
$$\boxed{\tan\alpha \cdot \tan(90°-\alpha) = 1}$$

### 证明（以正弦/余弦为例）

在直角三角形 $ABC$ 中，$\angle C=90°$，$a=BC,b=AC,c=AB$：

$$\sin A = \frac{a}{c}, \quad \cos A = \frac{b}{c}$$

$$\sin B = \frac{b}{c} = \cos A, \quad \text{而 } B = 90°-A$$

故 $\sin(90°-A) = \cos A$，反之同理。

### 应用：化简三角函数表达式

$$\sin 70° = \cos 20°, \quad \cos 75° = \sin 15°$$
$$\tan 40° \cdot \tan 50° = 1$$

### 典型例题

**例 1**：化简 $\sin^2 30° + \sin^2 60° + \cos^2 30° + \cos^2 60°$。

解：利用 $\sin 30° = \cos 60°$，$\sin 60° = \cos 30°$：
$= (\sin^2 30° + \cos^2 30°) + (\sin^2 60° + \cos^2 60°)$
$= 1 + 1 = 2$（利用 $\sin^2\theta + \cos^2\theta = 1$）。

**例 2**：已知 $\sin(2\alpha - 10°) = \cos(50° - \alpha)$，求 $\alpha$。

解：$\sin\theta = \cos(90°-\theta)$，故 $\cos(50°-\alpha) = \sin(90°-(50°-\alpha)) = \sin(40°+\alpha)$。

所以 $\sin(2\alpha-10°) = \sin(40°+\alpha)$，$2\alpha-10° = 40°+\alpha$，$\alpha = 50°$。

**例 3**：已知 $\alpha + \beta = 90°$，$\tan\alpha = 2$，求 $\tan\beta$。

解：$\tan\beta = \tan(90°-\alpha) = \dfrac{1}{\tan\alpha} = \dfrac{1}{2}$。

### 记忆表

| 函数 | 互余转换 |
|------|---------|
| $\sin\alpha$ | $= \cos(90°-\alpha)$ |
| $\cos\alpha$ | $= \sin(90°-\alpha)$ |
| $\tan\alpha$ | $= \dfrac{1}{\tan(90°-\alpha)}$ |

### 易错点

- ❌ 误用互补关系（$\sin\alpha = \sin(180°-\alpha)$）代替互余关系
- ❌ 正切的互余是倒数，不是相等
