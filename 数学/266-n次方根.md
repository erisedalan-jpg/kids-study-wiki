## 🎓 n次方根

### 定义

对于实数 $a$ 和正整数 $n$（$n \geq 2$），若 $x^n = a$，则称 $x$ 是 $a$ 的 **$n$ 次方根**（nth root）。

**偶次方根**（$n$ 为偶数）：

- 若 $a > 0$，$a$ 有两个实数 $n$ 次方根：$\sqrt[n]{a}$（正数）和 $-\sqrt[n]{a}$（负数），合记为 $\pm\sqrt[n]{a}$
- 若 $a = 0$，$n$ 次方根为 $0$
- 若 $a < 0$，无实数 $n$ 次方根（在实数范围内）

**奇次方根**（$n$ 为奇数）：任何实数 $a$ 都有唯一的实数 $n$ 次方根 $\sqrt[n]{a}$（与 $a$ 同号）。

**算术 $n$ 次方根**：$\sqrt[n]{a}$（要求 $a \geq 0$，取非负值），是 $a$ 的 $n$ 次方根中的那个非负值。

### 符号与运算

$\sqrt[n]{a}$：$n$ 称为根指数（$n=2$ 时通常省略，写 $\sqrt{a}$），$a$ 称为被开方数。

$$(\sqrt[n]{a})^n = a \quad (a \geq 0)$$

$$\sqrt[n]{a^n} = a \quad (n \text{ 为奇数})$$

$$\sqrt[n]{a^n} = |a| \quad (n \text{ 为偶数})$$

### 运算性质

$$\sqrt[n]{ab} = \sqrt[n]{a} \cdot \sqrt[n]{b} \quad (a, b \geq 0)$$

$$\sqrt[n]{\frac{a}{b}} = \frac{\sqrt[n]{a}}{\sqrt[n]{b}} \quad (a \geq 0, b > 0)$$

$$\sqrt[m]{\sqrt[n]{a}} = \sqrt[mn]{a} \quad (a \geq 0)$$

$$\sqrt[n]{a^m} = (\sqrt[n]{a})^m = a^{m/n} \quad (a \geq 0)$$

### 典型例题

**例1**：求 $\sqrt[3]{-8}$，$\sqrt[4]{81}$，$-\sqrt[4]{81}$。

解：
- $\sqrt[3]{-8} = -2$（$(-2)^3 = -8$，奇次方根，与被开方数同号）
- $\sqrt[4]{81} = 3$（$3^4 = 81$，取正值）
- $-\sqrt[4]{81} = -3$

**例2**：化简 $\sqrt[4]{a^2}$（$a > 0$）。

解：$\sqrt[4]{a^2} = a^{2/4} = a^{1/2} = \sqrt{a}$

**例3**：比较 $\sqrt[3]{5}$ 与 $\sqrt[4]{7}$ 的大小。

解：通分指数：$\sqrt[3]{5} = 5^{1/3} = 5^{4/12} = \sqrt[12]{5^4} = \sqrt[12]{625}$；$\sqrt[4]{7} = 7^{1/4} = 7^{3/12} = \sqrt[12]{7^3} = \sqrt[12]{343}$。

因 $625 > 343$，故 $\sqrt[3]{5} > \sqrt[4]{7}$。

### 易错点

1. **偶次方根要求非负**：$\sqrt[4]{-16}$ 无实数意义（不能开偶次方根为负数）
2. **$\sqrt[n]{a^n}$ 含绝对值**：偶数 $n$ 时，$\sqrt[n]{a^n} = |a|$ 而非 $a$（例：$\sqrt{(-3)^2} = \sqrt{9} = 3 = |-3|$）
3. **根指数化简**：$\sqrt[6]{a^4} = a^{4/6} = a^{2/3} = \sqrt[3]{a^2}$，需约分
4. **奇次根的符号**：奇次方根与被开方数同号，$\sqrt[3]{-8} = -2$ 而非 $2$
5. **算术根与完整根**：题目问"$x$ 的 $n$ 次方根"与"$x$ 的算术 $n$ 次方根"含义不同，前者包含负值

### 与初中知识的衔接

初中学习了平方根（二次方根）和立方根（三次方根），了解 $\sqrt{4} = 2$（算术平方根）和 $\sqrt[3]{-8} = -2$。高中将概念推广到一般 $n$ 次方根，并引入分数指数幂 $a^{m/n} = \sqrt[n]{a^m}$ 将其统一到指数运算框架。
