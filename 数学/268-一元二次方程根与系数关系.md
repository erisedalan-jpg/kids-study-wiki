## 🎓 一元二次方程根与系数关系

### 定义（韦达定理）

设一元二次方程 $ax^2 + bx + c = 0$（$a \neq 0$）的两根为 $x_1, x_2$，则：

$$x_1 + x_2 = -\frac{b}{a} \quad \text{（根之和）}$$

$$x_1 \cdot x_2 = \frac{c}{a} \quad \text{（根之积）}$$

这两个公式合称**韦达定理**（Vieta's formulas），由法国数学家弗朗索瓦·韦达（François Viète）发现。

### 推导

由求根公式：

$$x_{1,2} = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$

则：

$$x_1 + x_2 = \frac{-b + \sqrt{\Delta}}{2a} + \frac{-b - \sqrt{\Delta}}{2a} = \frac{-2b}{2a} = -\frac{b}{a}$$

$$x_1 \cdot x_2 = \frac{(-b + \sqrt{\Delta})(-b - \sqrt{\Delta})}{4a^2} = \frac{b^2 - \Delta}{4a^2} = \frac{b^2 - (b^2 - 4ac)}{4a^2} = \frac{4ac}{4a^2} = \frac{c}{a}$$

### 常用衍生公式

$$x_1^2 + x_2^2 = (x_1 + x_2)^2 - 2x_1 x_2 = \left(-\frac{b}{a}\right)^2 - \frac{2c}{a}$$

$$|x_1 - x_2| = \sqrt{(x_1+x_2)^2 - 4x_1 x_2} = \frac{\sqrt{\Delta}}{|a|}$$

### 逆定理（构造方程）

若两数之和为 $S$，之积为 $P$，则以这两数为根的一元二次方程为：

$$x^2 - Sx + P = 0$$

### 典型例题

**例1**：方程 $2x^2 - 5x + 3 = 0$ 的两根 $x_1, x_2$，不解方程，求 $x_1^2 + x_2^2$ 和 $\dfrac{1}{x_1} + \dfrac{1}{x_2}$。

解：$a=2, b=-5, c=3$。
- $x_1 + x_2 = -\dfrac{-5}{2} = \dfrac{5}{2}$，$x_1 x_2 = \dfrac{3}{2}$
- $x_1^2 + x_2^2 = \left(\dfrac{5}{2}\right)^2 - 2 \cdot \dfrac{3}{2} = \dfrac{25}{4} - 3 = \dfrac{13}{4}$
- $\dfrac{1}{x_1} + \dfrac{1}{x_2} = \dfrac{x_1 + x_2}{x_1 x_2} = \dfrac{5/2}{3/2} = \dfrac{5}{3}$

**例2**：已知方程 $x^2 - kx + (k+3) = 0$ 有两个实根，且两根均大于 0，求 $k$ 的范围。

解：需满足三个条件：
1. $\Delta \geq 0$：$k^2 - 4(k+3) \geq 0 \Rightarrow k^2 - 4k - 12 \geq 0 \Rightarrow (k-6)(k+2) \geq 0 \Rightarrow k \leq -2$ 或 $k \geq 6$
2. $x_1 + x_2 = k > 0 \Rightarrow k > 0$
3. $x_1 x_2 = k+3 > 0 \Rightarrow k > -3$

综合：$k \geq 6$。

**例3**：构造以 $3+\sqrt{2}$ 和 $3-\sqrt{2}$ 为根的二次方程。

解：$S = (3+\sqrt{2}) + (3-\sqrt{2}) = 6$，$P = (3+\sqrt{2})(3-\sqrt{2}) = 9-2 = 7$。

方程为 $x^2 - 6x + 7 = 0$。

### 易错点

1. **系数对应**：$x_1+x_2 = -b/a$ 中负号不能漏，$x_1 x_2 = c/a$ 无负号
2. **必须有实根**：韦达定理对复数根也成立，但高中范围内，若要求根为实数，需先确认 $\Delta \geq 0$
3. **含参数时分类**：参数问题往往需要同时考虑 $\Delta, x_1+x_2, x_1 x_2$ 三个条件
4. **衍生公式推导**：$x_1^2 + x_2^2$ 等复杂表达式，先用 $(x_1+x_2)^2 = x_1^2 + 2x_1x_2 + x_2^2$ 展开，再代入
5. **逆定理符号**：$x^2 - Sx + P = 0$ 中，$S$ 前有负号（$-S$），不要写成 $+S$

### 与初中知识的衔接

初中学习了一元二次方程的求根公式和判别式，高中引入韦达定理，使得在不解方程的情况下可以获得根的信息，大大拓展了处理与根有关的代数问题的能力。
