## 🎓 三角函数图象变换

### 核心：从 $y = \sin x$ 到 $y = A\sin(\omega x + \varphi)$

变换步骤（以 $y = A\sin(\omega x + \varphi)$ 为例，$A > 0, \omega > 0$）：

**步骤一（水平移位）**：$y = \sin x \to y = \sin(x + \varphi)$

将 $y = \sin x$ 图象向左移 $\varphi$（若 $\varphi > 0$）或向右移 $|\varphi|$（若 $\varphi < 0$）。

**步骤二（水平伸缩）**：$y = \sin(x + \varphi) \to y = \sin(\omega x + \varphi)$

横坐标变为原来的 $\dfrac{1}{\omega}$（纵坐标不变）。周期 $T = \dfrac{2\pi}{\omega}$。

**步骤三（垂直伸缩）**：$y = \sin(\omega x + \varphi) \to y = A\sin(\omega x + \varphi)$

纵坐标变为原来的 $A$ 倍（横坐标不变）。振幅为 $A$。

### 参数的几何含义

| 参数 | 名称 | 几何含义 |
|:----:|:---:|---------|
| $A$ | 振幅 | 函数值的最大绝对值，图象在 $y = A$ 和 $y = -A$ 之间振荡 |
| $\omega$ | 角频率 | 影响周期：$T = \dfrac{2\pi}{\omega}$，$\omega$ 越大周期越短 |
| $\varphi$ | 初相 | 水平移位量；图象相对于 $y = A\sin(\omega x)$ 的左移量为 $\dfrac{\varphi}{\omega}$ |

### 相位的分析

$y = A\sin(\omega x + \varphi)$ 的相位为 $\omega x + \varphi$，当相位 $= 0$ 时（即 $x = -\dfrac{\varphi}{\omega}$），函数值为 $0$ 且递增，这是图象的"出发点"。

### 典型例题

**例1**：从 $y = \sin x$ 到 $y = \sin(2x - \pi/3)$，描述变换过程。

解：
1. $y = \sin x \to y = \sin(x - \pi/3)$：向右移 $\pi/3$（$\varphi = -\pi/3 < 0$）
2. $y = \sin(x - \pi/3) \to y = \sin(2x - \pi/3) = \sin\left[2\left(x - \dfrac{\pi}{6}\right)\right]$：横坐标变为原来 $\dfrac{1}{2}$（$\omega = 2$）

也可先缩再移：
1. $y = \sin x \to y = \sin 2x$：横坐标缩为 $\dfrac{1}{2}$
2. $y = \sin 2x \to y = \sin(2x - \pi/3)$：向右移 $\dfrac{\pi/3}{2} = \dfrac{\pi}{6}$

**例2**：已知 $y = 2\sin(3x + \pi/6)$，求振幅、周期、初相。

解：$A = 2$（振幅），$\omega = 3$，$T = \dfrac{2\pi}{3}$（周期），初相 $\varphi = \dfrac{\pi}{6}$（相当于向左移 $\dfrac{\pi/6}{3} = \dfrac{\pi}{18}$）。

**例3**：已知函数图象的最高点为 $\left(\dfrac{\pi}{3}, 3\right)$，相邻最低点为 $\left(\dfrac{4\pi}{3}, -3\right)$，求 $A, \omega, \varphi$（$|\varphi| \leq \pi/2$）。

解：$A = 3$（最大值为 $3$）。

相邻最高点与最低点的距离为半个周期：$\dfrac{T}{2} = \dfrac{4\pi/3 - \pi/3}{1} = \pi$，故 $T = 2\pi$，$\omega = \dfrac{2\pi}{T} = 1$。

$f(x) = 3\sin(x + \varphi)$，代入最高点 $\left(\dfrac{\pi}{3}, 3\right)$：$3\sin\left(\dfrac{\pi}{3}+\varphi\right) = 3$，$\sin\left(\dfrac{\pi}{3}+\varphi\right) = 1$。

$\dfrac{\pi}{3} + \varphi = \dfrac{\pi}{2} + 2k\pi$，$\varphi = \dfrac{\pi}{6} + 2k\pi$。取 $|\varphi| \leq \pi/2$ 的值，$\varphi = \dfrac{\pi}{6}$。

### 易错点

1. **先移后缩 vs 先缩后移**：顺序不同，移位量不同。直接从 $\sin(\omega x + \varphi)$ 提出 $\omega$ 确定移位量 $\varphi/\omega$
2. **水平压缩因子**：$y = \sin(2x)$ 的横坐标是 $y=\sin x$ 的 $\dfrac{1}{2}$（压缩），不是 $2$ 倍
3. **$\varphi$ 的范围**：高考题通常要求 $|\varphi| \leq \dfrac{\pi}{2}$ 或 $0 \leq \varphi < 2\pi$，有多解时选满足条件的
4. **垂直平移**：若有 $y = A\sin(\omega x + \varphi) + b$，纵坐标整体上移 $b$
5. **振幅只看 $|A|$**：若 $A < 0$，先处理为 $|A|\sin(\omega x + \varphi + \pi)$（用 $\sin(\cdot + \pi) = -\sin(\cdot)$）

### 与初中知识的衔接

初中接触过函数图象的平移（一次函数、二次函数），高中将平移和伸缩变换系统化应用于三角函数。$y = A\sin(\omega x + \varphi)$ 是描述周期性物理现象（声波、光波、交流电）的核心数学模型。
