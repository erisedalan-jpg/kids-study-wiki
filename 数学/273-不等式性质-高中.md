## 🎓 不等式性质（高中）

### 定义

不等式性质是对不等式两边进行运算时仍保持不等关系成立的规律，是解不等式和推导不等式的基础。

### 基本性质

**对称性**：$a > b \Leftrightarrow b < a$

**传递性**：$a > b$ 且 $b > c \Rightarrow a > c$

**加法保序性**：$a > b \Rightarrow a + c > b + c$（两边加同一实数，不等号方向不变）

**乘法性质**（关键！）：

$$a > b, c > 0 \Rightarrow ac > bc \quad \text{（乘以正数，方向不变）}$$

$$a > b, c < 0 \Rightarrow ac < bc \quad \text{（乘以负数，方向改变！）}$$

**复合运算**：

若 $a > b > 0$ 且 $c > d > 0$，则 $ac > bd$（同号正量相乘保序）

若 $a > b > 0$，则 $a^n > b^n$（$n \in \mathbb{N}^*$，幂运算保序，对正数）

若 $a > b > 0$，则 $\sqrt[n]{a} > \sqrt[n]{b}$（$n \in \mathbb{N}^*$，根号保序，对正数）

**倒数性质**：

$$a > b > 0 \Rightarrow \frac{1}{a} < \frac{1}{b} \quad \text{（同正取倒，方向改变）}$$

$$a < b < 0 \Rightarrow \frac{1}{a} > \frac{1}{b} \quad \text{（同负取倒，方向改变）}$$

若 $a, b$ 异号，取倒后方向不变（但需单独验证）。

### 绝对值不等式

$$|a| \geq 0, \quad |a| = 0 \Leftrightarrow a = 0$$

$$|a + b| \leq |a| + |b| \quad \text{（三角不等式）}$$

$$|a - b| \geq \big||a| - |b|\big|$$

### 推导思路

不等式的核心技巧是"作差法"：比较 $A$ 与 $B$ 的大小，计算 $A - B$，通过分析 $A - B$ 的正负来判断大小。

$$A > B \Leftrightarrow A - B > 0$$

### 典型例题

**例1**：已知 $a > b$，$c < 0$，比较 $ac$ 与 $bc$ 的大小。

解：$a > b$，$c < 0$，由乘法性质（乘负数翻转），$ac < bc$。

**例2**：已知 $a > b > 0$，$c > d > 0$，比较 $a + c$ 与 $b + d$、$ac$ 与 $bd$。

解：$a > b \Rightarrow a + c > b + c$；$c > d \Rightarrow b + c > b + d$；由传递性，$a + c > b + d$。
类似地，$a > b > 0, c > d > 0 \Rightarrow ac > bc > bd$，故 $ac > bd$。

**例3**：解不等式 $\dfrac{x-1}{x+2} > 0$。

解：分子分母同号，分两情况讨论：
- $x - 1 > 0$ 且 $x + 2 > 0$：$x > 1$ 且 $x > -2$，取交集 $x > 1$
- $x - 1 < 0$ 且 $x + 2 < 0$：$x < 1$ 且 $x < -2$，取交集 $x < -2$

解集：$(-\infty, -2) \cup (1, +\infty)$。

### 易错点

1. **乘以负数必须翻转不等号**：这是最常见错误，$a > b$ 两边同乘 $-1$ 得 $-a < -b$
2. **乘以 $x$ 时未分类**：$x$ 符号不确定时，不能直接乘，必须分 $x > 0$ 和 $x < 0$ 两种情况
3. **分式不等式**：$\dfrac{f}{g} > 0$ 不能直接乘 $g$（因为 $g$ 正负未知），应分情况或用"同号异号"法
4. **绝对值去绝对值**：$|x| > a$（$a > 0$）解集为 $x > a$ 或 $x < -a$；$|x| < a$ 解集为 $-a < x < a$
5. **幂运算的前提**：$a > b \Rightarrow a^2 > b^2$ 要求 $a, b > 0$；若含负数，需先判断绝对值大小

### 与初中知识的衔接

初中系统学习了不等式的基本性质（加减保序，乘正不变乘负翻转），高中在此基础上扩展到分式不等式、绝对值不等式、以及与函数结合的综合问题。
