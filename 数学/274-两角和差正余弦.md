## 🎓 两角和差正余弦

### 核心公式

**正弦和差公式**：

$$\sin(\alpha + \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$

$$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$

**余弦和差公式**：

$$\cos(\alpha + \beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$$

$$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$

**正切和差公式**：

$$\tan(\alpha + \beta) = \frac{\tan\alpha + \tan\beta}{1 - \tan\alpha\tan\beta}$$

$$\tan(\alpha - \beta) = \frac{\tan\alpha - \tan\beta}{1 + \tan\alpha\tan\beta}$$

（正切公式在 $\alpha, \beta, \alpha\pm\beta \neq \pi/2 + k\pi$ 时有效）

### 推导思路（余弦加法公式）

以向量/距离法推导（简述）：设单位圆上 $P_1 = (\cos\alpha, \sin\alpha)$，$P_2 = (\cos\beta, \sin\beta)$，则：

$|P_1P_2|^2 = (\cos\alpha - \cos\beta)^2 + (\sin\alpha - \sin\beta)^2 = 2 - 2(\cos\alpha\cos\beta + \sin\alpha\sin\beta)$

另一方面，$P_1$ 与 $P_2$ 对应角度差为 $\alpha - \beta$，所以 $P_0 = (\cos(\alpha-\beta), \sin(\alpha-\beta))$ 与 $(1, 0)$ 的距离相同：

$|P_0O_1|^2 = (\cos(\alpha-\beta) - 1)^2 + \sin^2(\alpha-\beta) = 2 - 2\cos(\alpha-\beta)$

两式相等，得 $\cos(\alpha-\beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$。其他公式由此推导。

### 重要推论（二倍角公式）

令 $\beta = \alpha$：

$$\sin 2\alpha = 2\sin\alpha\cos\alpha$$

$$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 1 - 2\sin^2\alpha = 2\cos^2\alpha - 1$$

$$\tan 2\alpha = \frac{2\tan\alpha}{1-\tan^2\alpha}$$

### 诱导公式的特殊情形

和差公式是诱导公式的一般化：

$$\sin(\pi - \alpha) = \sin\pi\cos\alpha - \cos\pi\sin\alpha = 0 \cdot \cos\alpha - (-1)\sin\alpha = \sin\alpha$$ ✓

$$\cos\!\left(\frac{\pi}{2} - \alpha\right) = \cos\dfrac{\pi}{2}\cos\alpha + \sin\dfrac{\pi}{2}\sin\alpha = 0 + \sin\alpha = \sin\alpha$$ ✓

### 典型例题

**例1**：计算 $\sin 75°$。

解：$\sin 75° = \sin(45° + 30°) = \sin 45°\cos 30° + \cos 45°\sin 30°$

$= \dfrac{\sqrt{2}}{2} \cdot \dfrac{\sqrt{3}}{2} + \dfrac{\sqrt{2}}{2} \cdot \dfrac{1}{2} = \dfrac{\sqrt{6}}{4} + \dfrac{\sqrt{2}}{4} = \dfrac{\sqrt{6}+\sqrt{2}}{4}$

**例2**：已知 $\sin\alpha = \dfrac{1}{3}$，$\alpha \in (0, \pi/2)$，$\cos\beta = -\dfrac{1}{4}$，$\beta \in (\pi/2, \pi)$，求 $\cos(\alpha - \beta)$。

解：$\cos\alpha = \dfrac{2\sqrt{2}}{3}$（第一象限正值，$\cos\alpha = \sqrt{1-1/9} = \sqrt{8/9}$）

$\sin\beta = \dfrac{\sqrt{15}}{4}$（第二象限正值，$\sin\beta = \sqrt{1-1/16} = \sqrt{15/16}$）

$\cos(\alpha-\beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$

$= \dfrac{2\sqrt{2}}{3} \cdot \left(-\dfrac{1}{4}\right) + \dfrac{1}{3} \cdot \dfrac{\sqrt{15}}{4}$

$= -\dfrac{2\sqrt{2}}{12} + \dfrac{\sqrt{15}}{12} = \dfrac{\sqrt{15} - 2\sqrt{2}}{12}$

**例3**：化简 $\sin\!\left(x+\dfrac{\pi}{6}\right) + \cos\!\left(x+\dfrac{\pi}{3}\right)$。

解：$\cos\!\left(x+\dfrac{\pi}{3}\right) = \cos\!\left(x+\dfrac{\pi}{6}+\dfrac{\pi}{6}\right) = \sin\!\left(\dfrac{\pi}{2}-x-\dfrac{\pi}{3}\right)$

更直接地：

$\cos\!\left(x+\dfrac{\pi}{3}\right) = \cos\!\left(x+\dfrac{\pi}{3}\right)$，展开：$= \cos x \cos\dfrac{\pi}{3} - \sin x \sin\dfrac{\pi}{3} = \dfrac{1}{2}\cos x - \dfrac{\sqrt{3}}{2}\sin x$

$\sin\!\left(x+\dfrac{\pi}{6}\right) = \sin x\cos\dfrac{\pi}{6} + \cos x\sin\dfrac{\pi}{6} = \dfrac{\sqrt{3}}{2}\sin x + \dfrac{1}{2}\cos x$

两者相加：$\dfrac{\sqrt{3}}{2}\sin x + \dfrac{1}{2}\cos x + \dfrac{1}{2}\cos x - \dfrac{\sqrt{3}}{2}\sin x = \cos x$

### 易错点

1. **正弦加法中交叉项**：$\sin(\alpha+\beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$（注意"交叉"，不是 $\sin\alpha\sin\beta$）
2. **余弦加法中有减号**：$\cos(\alpha+\beta) = \cos\alpha\cos\beta - \sin\alpha\sin\beta$（同项相积取差）
3. **正切公式的分母**：$\tan(\alpha+\beta)$ 分母是 $1 - \tan\alpha\tan\beta$（减），$\tan(\alpha-\beta)$ 分母是 $1 + \tan\alpha\tan\beta$（加）
4. **代入前确认象限**：使用公式前，需先确定各角所在象限以确定函数值的符号
5. **化简路径多样**：三角恒等变换通常有多种路径，找最短路径需要经验积累

### 与初中知识的衔接

初中只计算特殊角（$30°, 45°, 60°, 90°$）的三角函数值，无法计算 $75°, 15°$ 等角。高中的两角和差公式提供了将任意角的三角函数化为特殊角的工具，是三角计算能力质的飞跃。
