## 🎓 基本不等式

### 定义与内容

**基本不等式**（AM-GM 不等式）：对任意实数 $a, b$，有

$$\frac{a + b}{2} \geq \sqrt{ab} \quad (a, b \geq 0)$$

即两个非负实数的**算术平均数**不小于它们的**几何平均数**，等号当且仅当 $a = b$ 时成立。

等价形式：
$$a + b \geq 2\sqrt{ab} \quad (a, b \geq 0)$$

$$ab \leq \left(\frac{a+b}{2}\right)^2 \quad (a, b \geq 0)$$

对于任意实数（不要求非负）：
$$a^2 + b^2 \geq 2ab$$（展开 $(a-b)^2 \geq 0$ 得）

### 关键推导

$$a + b - 2\sqrt{ab} = (\sqrt{a} - \sqrt{b})^2 \geq 0$$

因为平方项 $(\sqrt{a} - \sqrt{b})^2 \geq 0$，等号当 $\sqrt{a} = \sqrt{b}$，即 $a = b$ 时成立。

### 三个重要推论

**积定和最小**：若 $ab = S$（乘积固定为 $S$），则 $a + b \geq 2\sqrt{S}$，等号 $a = b$ 时成立。

**和定积最大**：若 $a + b = T$（和固定为 $T$），则 $ab \leq \dfrac{T^2}{4}$，等号 $a = b$ 时成立。

**调和-算术-几何-平方平均链**（了解）：
$$H \leq G \leq A \leq Q$$

其中 $H = \dfrac{2ab}{a+b}$（调和均值），$G = \sqrt{ab}$，$A = \dfrac{a+b}{2}$，$Q = \sqrt{\dfrac{a^2+b^2}{2}}$。

### 典型例题

**例1**：已知正实数 $a, b$ 满足 $a + b = 4$，求 $ab$ 的最大值。

解：由基本不等式，$ab \leq \left(\dfrac{a+b}{2}\right)^2 = \left(\dfrac{4}{2}\right)^2 = 4$，等号当 $a = b = 2$ 时成立。故 $ab$ 的最大值为 $4$。

**例2**：已知正实数 $x$，求 $y = x + \dfrac{4}{x}$ 的最小值。

解：$y = x + \dfrac{4}{x} \geq 2\sqrt{x \cdot \dfrac{4}{x}} = 2\sqrt{4} = 4$，

等号当 $x = \dfrac{4}{x}$，即 $x^2 = 4, x = 2$（$x > 0$）时成立。最小值为 $4$。

**例3**：设 $a > 0, b > 0$ 且 $\dfrac{1}{a} + \dfrac{2}{b} = 1$，求 $a + b$ 的最小值。

解：$a + b = (a + b) \cdot 1 = (a + b)\left(\dfrac{1}{a} + \dfrac{2}{b}\right) = 1 + \dfrac{2a}{b} + \dfrac{b}{a} + 2$

$= 3 + \dfrac{2a}{b} + \dfrac{b}{a} \geq 3 + 2\sqrt{\dfrac{2a}{b} \cdot \dfrac{b}{a}} = 3 + 2\sqrt{2}$

等号当 $\dfrac{2a}{b} = \dfrac{b}{a}$，即 $b = \sqrt{2}a$ 时，代入约束求得 $a, b$ 具体值，验证可达。故最小值为 $3 + 2\sqrt{2}$。

### 易错点

1. **等号条件忘记验证**：用基本不等式求最值时，必须确认等号条件（$a = b$）在约束条件下可以同时满足
2. **符号要求**：基本不等式 $a+b \geq 2\sqrt{ab}$ 要求 $a, b \geq 0$，若 $a, b$ 可能为负，需用 $a^2+b^2 \geq 2ab$
3. **"凑"的技巧**：用基本不等式时，常需将表达式凑成 $u + v$ 或 $uv$ 的形式，需要合理拆分/补充
4. **不能盲目套用**：$f(x) = x + \dfrac{4}{x}$ 在 $x < 0$ 时，$x$ 和 $\dfrac{4}{x}$ 都是负数，最大值而非最小值
5. **多步不等式方向**：若用了多次不等式放缩，需保持方向一致，不能有的向上放有的向下放

### 与初中知识的衔接

初中接触过"完全平方公式" $(a-b)^2 \geq 0$，这正是基本不等式的推导基础。高中将其提炼为 AM-GM 不等式，成为解决最值问题的核心工具。
