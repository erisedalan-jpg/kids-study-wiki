## 🎓 对数函数的性质

### 基本性质汇总

设 $y = \log_a x$（$a > 0$，$a \neq 1$，$x > 0$）：

| 性质 | $a > 1$ | $0 < a < 1$ |
|------|---------|------------|
| 定义域 | $(0, +\infty)$ | $(0, +\infty)$ |
| 值域 | $\mathbb{R}$ | $\mathbb{R}$ |
| 单调性 | 单调递增 | 单调递减 |
| 过定点 | $(1, 0)$ | $(1, 0)$ |
| $(a, 1)$ 处 | $y = 1$（$a > 1$，$x = a > 1$）| $y = 1$（$0<a<1$，$x=a<1$）|
| 奇偶性 | 非奇非偶 | 非奇非偶 |
| 渐近线 | $y$ 轴（$x \to 0^+$，$y \to -\infty$）| $y$ 轴（$x \to 0^+$，$y \to +\infty$）|

**与 $x$ 轴的位置关系**：

$$a > 1 \Rightarrow \begin{cases} \log_a x > 0, & x > 1 \\ \log_a x = 0, & x = 1 \\ \log_a x < 0, & 0 < x < 1 \end{cases}$$

$$0 < a < 1 \Rightarrow \begin{cases} \log_a x < 0, & x > 1 \\ \log_a x = 0, & x = 1 \\ \log_a x > 0, & 0 < x < 1 \end{cases}$$

### 对数不等式

$$a > 1: \quad \log_a f(x) > \log_a g(x) \Leftrightarrow f(x) > g(x) > 0$$

$$0 < a < 1: \quad \log_a f(x) > \log_a g(x) \Leftrightarrow 0 < f(x) < g(x)$$

### 典型例题

**例1**：解不等式 $\log_{0.5}(2x-1) > \log_{0.5}(x+2)$。

解：定义域：$2x-1 > 0$ 且 $x+2 > 0$，即 $x > \dfrac{1}{2}$。

底数 $0.5 < 1$，单调递减，不等号反向：$2x-1 < x+2$，即 $x < 3$。

综合：$\dfrac{1}{2} < x < 3$，解集为 $\left(\dfrac{1}{2}, 3\right)$。

**例2**：比较 $\log_2 3$，$\log_2 \pi$，$1$ 的大小。

解：$y = \log_2 x$，$a = 2 > 1$，单调递增。
$3 < \pi < 4$（$\pi \approx 3.14$），故 $\log_2 3 < \log_2 \pi < \log_2 4 = 2$。
又 $\log_2 3 > \log_2 2 = 1$（因 $3 > 2$）。
故 $1 < \log_2 3 < \log_2 \pi < 2$。

**例3**：解对数方程 $\log_2 x + \log_2(x-2) = 3$。

解：定义域：$x > 0$ 且 $x-2 > 0$，即 $x > 2$。

$\log_2[x(x-2)] = 3 \Rightarrow x(x-2) = 8 \Rightarrow x^2-2x-8 = 0 \Rightarrow (x-4)(x+2) = 0$

$x = 4$ 或 $x = -2$。检验：$x = -2 < 2$，舍去。$x = 4 > 2$，满足。故 $x = 4$。

### 易错点

1. **先确定定义域**：解对数方程/不等式，第一步必须写定义域（真数 $> 0$）
2. **不等号方向依底数**：底数 $> 1$ 同向，底数 $< 1$ 反向，这是最常见错误点
3. **对数方程要验根**：变换后解出的 $x$ 值，必须代回验证满足定义域
4. **$x$ 轴两侧的正负**：$a > 1$ 时，$x > 1$ 对数为正，$0 < x < 1$ 对数为负（与直觉相符）；$0 < a < 1$ 时相反
5. **单调性用于比较大小**：不能直接比较不同底的对数值，需换底或用中间量

### 与初中知识的衔接

初中没有对数函数。高中通过反函数关系从指数函数引入对数函数，其性质（定义域、值域、单调性）与指数函数形成对称关系，有助于统一理解这两类重要的超越函数。
