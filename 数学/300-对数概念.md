## 🎓 对数概念

### 定义

若 $a^b = N$（$a > 0$，$a \neq 1$，$N > 0$），则 $b$ 称为以 $a$ 为底 $N$ 的**对数**（logarithm），记作：

$$b = \log_a N$$

读作"以 $a$ 为底 $N$ 的对数"。其中：
- $a$：**底数**（base），满足 $a > 0$ 且 $a \neq 1$
- $N$：**真数**（antilogarithm），满足 $N > 0$
- $b$：**对数值**

**核心等价关系**：

$$\log_a N = b \Leftrightarrow a^b = N \quad (a > 0, a \neq 1, N > 0)$$

### 常用特殊对数

**常用对数**（common logarithm）：以 $10$ 为底，记 $\log_{10} N = \lg N$

**自然对数**（natural logarithm）：以 $e$ 为底（$e \approx 2.71828$），记 $\log_e N = \ln N$

### 基本性质

$$\log_a 1 = 0 \quad (a^0 = 1)$$

$$\log_a a = 1 \quad (a^1 = a)$$

$$\log_a a^n = n \quad (a^n = a^n)$$

$$a^{\log_a N} = N \quad \text{（对数运算与指数运算互为逆运算）}$$

### 对数与指数互化

对数是指数的逆运算，两种形式互化是解对数/指数方程的基本方法：

$$a^b = N \Leftrightarrow \log_a N = b$$

例：$2^3 = 8 \Leftrightarrow \log_2 8 = 3$；$10^2 = 100 \Leftrightarrow \log_{10} 100 = \lg 100 = 2$

### 典型例题

**例1**：计算下列对数值：$\log_2 32$，$\log_3 \dfrac{1}{9}$，$\lg 0.001$，$\ln e^5$。

解：
- $\log_2 32 = \log_2 2^5 = 5$
- $\log_3 \dfrac{1}{9} = \log_3 3^{-2} = -2$
- $\lg 0.001 = \log_{10} 10^{-3} = -3$
- $\ln e^5 = \log_e e^5 = 5$

**例2**：将指数式化为对数式，将对数式化为指数式。

- $5^3 = 125 \Leftrightarrow \log_5 125 = 3$
- $\log_4 2 = \dfrac{1}{2} \Leftrightarrow 4^{1/2} = 2$（即 $\sqrt{4} = 2$）

**例3**：解方程 $\log_x 81 = 4$（$x > 0, x \neq 1$）。

解：由对数定义，$x^4 = 81 = 3^4$，故 $x = 3$（需验证 $x = 3 > 0, x \neq 1$，满足）。

### 易错点

1. **真数必须为正数**：$\log_a N$ 中 $N > 0$ 是严格要求，$\log_2(-4)$ 无意义
2. **底数不能为 1**：若 $a = 1$，$1^b = 1 \neq N$（$N \neq 1$ 时无解），故底数 $a \neq 1$
3. **对数值可以为负**：$\log_2 \dfrac{1}{4} = -2 < 0$，负对数值完全合法
4. **$\log_a$ 与 $\lg$、$\ln$ 区分**：$\lg$ 固定底数 $10$，$\ln$ 固定底数 $e$，$\log_a$ 底数为 $a$
5. **互化方向**：熟悉"指数 $\leftrightarrow$ 对数"的互化，这是计算对数值的基本方法

### 与初中知识的衔接

初中学习了整数指数幂和科学记数法（使用 $10$ 的幂次），感性接触了对数思想（$1000 = 10^3$，指数 $3$ 就是对数值）。高中正式引入对数定义，将"已知底数和指数求幂"的逆问题"已知底数和幂求指数"形式化，是反函数概念的具体体现。
