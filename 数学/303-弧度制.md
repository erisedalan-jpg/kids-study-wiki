## 🎓 弧度制

### 定义

**弧度**（radian，rad）：一种度量角大小的单位。

定义：以角的顶点为圆心，任意长度 $r$ 为半径作圆，若所对弧的长度为 $l$，则该角的**弧度数**为：

$$\theta = \frac{l}{r}$$

**1 弧度**的定义：弧长等于半径时（$l = r$），所对圆心角的大小为 1 弧度（1 rad）。

### 弧度与角度的换算

由圆的周长 $2\pi r$ 对应圆心角 $360°$，得：

$$360° = 2\pi \text{ rad} \quad \Rightarrow \quad 180° = \pi \text{ rad}$$

$$1° = \frac{\pi}{180} \text{ rad} \approx 0.01745 \text{ rad}$$

$$1 \text{ rad} = \frac{180°}{\pi} \approx 57.30°$$

**换算公式**：

$$\text{弧度} = \text{角度} \times \frac{\pi}{180°}$$

$$\text{角度} = \text{弧度} \times \frac{180°}{\pi}$$

### 常用特殊角的弧度值

| 角度 | 弧度 |
|:----:|:----:|
| $0°$ | $0$ |
| $30°$ | $\dfrac{\pi}{6}$ |
| $45°$ | $\dfrac{\pi}{4}$ |
| $60°$ | $\dfrac{\pi}{3}$ |
| $90°$ | $\dfrac{\pi}{2}$ |
| $120°$ | $\dfrac{2\pi}{3}$ |
| $180°$ | $\pi$ |
| $270°$ | $\dfrac{3\pi}{2}$ |
| $360°$ | $2\pi$ |

### 弧长公式与扇形面积

设圆半径为 $r$，圆心角为 $\theta$（弧度），则：

**弧长**：$l = r\theta$

**扇形面积**：$S = \dfrac{1}{2}r^2\theta = \dfrac{1}{2}rl$

### 典型例题

**例1**：将 $150°$ 化为弧度，将 $\dfrac{5\pi}{4}$ rad 化为度数。

解：
- $150° = 150 \times \dfrac{\pi}{180} = \dfrac{5\pi}{6}$ rad
- $\dfrac{5\pi}{4} = \dfrac{5\pi}{4} \times \dfrac{180°}{\pi} = \dfrac{5 \times 180°}{4} = 225°$

**例2**：半径为 $6$ cm，圆心角为 $\dfrac{2\pi}{3}$ rad 的扇形，求弧长和面积。

解：
- 弧长：$l = r\theta = 6 \times \dfrac{2\pi}{3} = 4\pi$ cm
- 面积：$S = \dfrac{1}{2}r^2\theta = \dfrac{1}{2} \times 36 \times \dfrac{2\pi}{3} = 12\pi$ cm²

或：$S = \dfrac{1}{2}rl = \dfrac{1}{2} \times 6 \times 4\pi = 12\pi$ cm²

**例3**：某车轮半径为 $30$ cm，车速为 $72$ km/h，求轮子每秒转过的弧度数。

解：$72$ km/h $= 72 \times \dfrac{1000}{3600}$ m/s $= 20$ m/s $= 2000$ cm/s。

每秒轮缘走过的弧长 $l = 2000$ cm。$\theta = \dfrac{l}{r} = \dfrac{2000}{30} = \dfrac{200}{3}$ rad/s。

### 易错点

1. **弧度没有单位**：弧度是纯数（比值 $l/r$），有时写 rad 表强调，但计算中不必写
2. **$\sin 30$ ≠ $\sin 30°$**：$\sin 30$ 表示弦长中 $x = 30$ rad，$\sin 30° = 0.5$，数值完全不同
3. **特殊角弧度记忆**：$30° = \pi/6$，$45° = \pi/4$，$60° = \pi/3$，$90° = \pi/2$ 必须牢记
4. **扇形面积公式**：$S = \dfrac{1}{2}r^2\theta$ 中 $\theta$ 必须是弧度，不能代入角度数值
5. **负角弧度**：负角的弧度也可以是负数，$-90° = -\dfrac{\pi}{2}$ rad

### 与初中知识的衔接

初中用"角度制"（°）度量角，如 $30°, 90°, 180°$。高中引入弧度制，这是更自然的角度单位（来源于弧长与半径的比），在微积分（$\sin x$ 的导数为 $\cos x$ 当且仅当 $x$ 用弧度）中不可替代。两种单位制需要熟练互换。
