## 🎓 指数函数与对数函数关系

### 核心关系：互为反函数

$y = a^x$（$a > 0, a \neq 1$）与 $y = \log_a x$（$a > 0, a \neq 1$）互为反函数：

$$f(x) = a^x \Leftrightarrow f^{-1}(x) = \log_a x$$

**恒等式**（反函数性质）：

$$a^{\log_a x} = x \quad (x > 0)$$

$$\log_a (a^x) = x \quad (x \in \mathbb{R})$$

### 图象关系

两函数图象关于直线 $y = x$ 对称，对应点互换横纵坐标：

- $y = a^x$ 的图象上点 $(t, a^t)$ 对应 $y = \log_a x$ 图象上点 $(a^t, t)$
- 两图象的公共点在 $y = x$ 上，即满足 $a^x = x$ 的点（对于某些 $a$ 有解）

### 定义域与值域的互换

| 函数 | 定义域 | 值域 |
|------|--------|------|
| $y = a^x$ | $\mathbb{R} = (-\infty, +\infty)$ | $(0, +\infty)$ |
| $y = \log_a x$ | $(0, +\infty)$ | $\mathbb{R} = (-\infty, +\infty)$ |

定义域和值域完全互换，体现反函数关系。

### 单调性对应关系

| 底数 $a$ | $y = a^x$ 单调性 | $y = \log_a x$ 单调性 |
|:--------:|:---------------:|:--------------------:|
| $a > 1$ | 递增 | 递增 |
| $0 < a < 1$ | 递减 | 递减 |

互为反函数的两函数单调性相同（反函数的反函数恢复单调方向）。

### 互化应用

**指数转对数**：$a^b = c \Leftrightarrow \log_a c = b$

**对数转指数**：$\log_a c = b \Leftrightarrow a^b = c$

在解方程时，可根据需要在两种形式间灵活转换。

### 典型例题

**例1**：利用恒等式化简：$2^{\log_2 5}$，$\log_3(3^7)$，$10^{\lg 3}$。

解：
- $2^{\log_2 5} = 5$（$a^{\log_a x} = x$）
- $\log_3(3^7) = 7$（$\log_a(a^x) = x$）
- $10^{\lg 3} = 10^{\log_{10} 3} = 3$（$a^{\log_a x} = x$）

**例2**：已知 $y = \log_2 x$ 的图象上有点 $(4, 2)$，求 $y = 2^x$ 的图象上的对应点。

解：$(4, 2)$ 在 $y = \log_2 x$ 上，关于 $y = x$ 的对称点为 $(2, 4)$，即点 $(2, 4)$ 在 $y = 2^x$ 的图象上（验证：$2^2 = 4$ ✓）。

**例3**：解方程 $2^{x+1} = 3^x$（利用对数）。

解：两边取对数：$\lg(2^{x+1}) = \lg(3^x)$，$(x+1)\lg 2 = x\lg 3$，$x\lg 2 + \lg 2 = x\lg 3$，$\lg 2 = x(\lg 3 - \lg 2) = x\lg\dfrac{3}{2}$，

$x = \dfrac{\lg 2}{\lg 3 - \lg 2} = \dfrac{\lg 2}{\lg(3/2)} = \log_{3/2} 2$

### 易错点

1. **恒等式的条件**：$a^{\log_a x} = x$ 要求 $x > 0$；$\log_a(a^x) = x$ 要求 $a^x$ 在对数的定义域内（$a^x > 0$ 恒成立，无额外限制）
2. **反函数图象对称轴是 $y = x$**：不是 $x$ 轴或 $y$ 轴
3. **定义域互换**：指数函数定义域 $\mathbb{R}$，值域 $(0, +\infty)$；对数函数恰好反过来
4. **解指数方程用对数**：$a^x = b$ 的解为 $x = \log_a b$，这是换底的直接应用
5. **不同底的互化**：$y = 2^x$ 与 $y = \log_3 x$ 不是反函数关系，底数必须相同

### 与初中知识的衔接

初中了解"加法与减法互逆""乘法与除法互逆"，高中将"互逆运算"推广到指数与对数。$a^{\log_a x} = x$ 正是"先取对数再做指数"恢复原值的体现，是反函数概念在具体函数上的应用。
