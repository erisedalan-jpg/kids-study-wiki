## 🎓 指数函数的性质

### 基本性质汇总

设 $y = a^x$（$a > 0$，$a \neq 1$，$x \in \mathbb{R}$）：

| 性质 | $a > 1$ | $0 < a < 1$ |
|------|---------|------------|
| 定义域 | $\mathbb{R}$ | $\mathbb{R}$ |
| 值域 | $(0, +\infty)$ | $(0, +\infty)$ |
| 单调性 | 在 $\mathbb{R}$ 上单调递增 | 在 $\mathbb{R}$ 上单调递减 |
| 过定点 | $(0, 1)$，即 $a^0 = 1$ | $(0, 1)$ |
| 上界/下界 | 无上界；下界趋近 $0^+$ | 无下界趋近 $0^+$；无上界 |
| 奇偶性 | 非奇非偶 | 非奇非偶 |
| $x=1$ 处 | $y = a > 1$ | $y = a < 1$ |
| $x=-1$ 处 | $y = 1/a < 1$ | $y = 1/a > 1$ |

**对称性**：$y = a^x$ 与 $y = \left(\dfrac{1}{a}\right)^x = a^{-x}$ 的图象关于 $y$ 轴对称。

### 指数函数的大小比较

**同底比较**：底数相同时，$a > 1$ 则指数越大函数值越大；$0 < a < 1$ 则指数越大函数值越小。

**异底比较**（间接法）：将两个值化为以公共值为中间量进行比较，如都与 $1$ 比较。

$$a > 1 \Rightarrow \begin{cases} a^x > 1, & x > 0 \\ a^x = 1, & x = 0 \\ a^x < 1, & x < 0 \end{cases}$$

$$0 < a < 1 \Rightarrow \begin{cases} a^x < 1, & x > 0 \\ a^x = 1, & x = 0 \\ a^x > 1, & x < 0 \end{cases}$$

### 指数方程与不等式

**指数方程**（同底）：$a^{f(x)} = a^{g(x)} \Rightarrow f(x) = g(x)$

**指数不等式**（同底）：
- $a > 1$：$a^{f(x)} > a^{g(x)} \Rightarrow f(x) > g(x)$（同向）
- $0 < a < 1$：$a^{f(x)} > a^{g(x)} \Rightarrow f(x) < g(x)$（反向）

### 典型例题

**例1**：比较 $0.3^{1.5}$，$0.3^{2}$，$0.3^{0.5}$ 的大小。

解：$y = 0.3^x$，底数 $0.3 \in (0,1)$，单调递减。$0.5 < 1.5 < 2$，故 $0.3^{0.5} > 0.3^{1.5} > 0.3^{2}$。

**例2**：解方程 $4^x - 2^{x+1} - 8 = 0$。

解：令 $t = 2^x > 0$，则 $4^x = (2^x)^2 = t^2$，$2^{x+1} = 2t$。
方程变为 $t^2 - 2t - 8 = 0$，$(t-4)(t+2) = 0$，$t = 4$ 或 $t = -2$。
因 $t = 2^x > 0$，舍去 $t = -2$，取 $t = 4 = 2^2$，故 $2^x = 2^2$，$x = 2$。

**例3**：证明 $y = a^x$ 在 $\mathbb{R}$ 上严格单调递增（$a > 1$）。

证：设 $x_1, x_2 \in \mathbb{R}$，$x_1 < x_2$。

$a^{x_1} - a^{x_2} = a^{x_1}(1 - a^{x_2 - x_1})$。

因 $a > 1$，$x_2 - x_1 > 0$，故 $a^{x_2-x_1} > 1$，即 $1 - a^{x_2-x_1} < 0$；又 $a^{x_1} > 0$，故 $a^{x_1}(1-a^{x_2-x_1}) < 0$，即 $a^{x_1} < a^{x_2}$。

故 $y = a^x$ 在 $\mathbb{R}$ 上单调递增。$\square$

### 易错点

1. **底数条件**：写指数函数性质时，必须分 $a > 1$ 和 $0 < a < 1$ 两种情况
2. **指数不等式变向**：$0 < a < 1$ 时，指数越大函数值越小，不等号方向与指数大小相反
3. **换元后的约束**：令 $t = a^x$ 后，$t > 0$ 是约束，不满足的解需舍去
4. **图象不过 $x$ 轴**：$y = a^x > 0$ 恒成立，图象不与 $x$ 轴相交
5. **异底比较**：不同底的指数函数比较，不能直接套用单调性，需先统一或借助中间量
