## 🎓 集合的运算

### 定义

**交集**（intersection）：$A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$，两集合的公共元素。

**并集**（union）：$A \cup B = \{x \mid x \in A \text{ 或 } x \in B\}$，两集合所有元素（不重复）。

**补集**（complement）：设 $U$ 为全集，$\complement_U A = \{x \mid x \in U \text{ 且 } x \notin A\}$，$U$ 中不属于 $A$ 的元素。

### 关键公式与性质

$$A \cap B \subseteq A \subseteq A \cup B$$

$$A \cap B \subseteq B \subseteq A \cup B$$

$$A \cap A = A, \quad A \cup A = A \quad \text{（幂等律）}$$

$$A \cap \emptyset = \emptyset, \quad A \cup \emptyset = A$$

$$A \cap U = A, \quad A \cup U = U$$

$$\complement_U(\complement_U A) = A \quad \text{（双重补集）}$$

**德摩根定律**（De Morgan's laws）：
$$\complement_U(A \cup B) = (\complement_U A) \cap (\complement_U B)$$
$$\complement_U(A \cap B) = (\complement_U A) \cup (\complement_U B)$$

### 元素个数公式（容斥原理）

$$|A \cup B| = |A| + |B| - |A \cap B|$$

其中 $|A|$ 表示集合 $A$ 的元素个数。

### 推导思路

韦恩图（Venn diagram）是理解集合运算最直观的工具。画两个相交的圆形区域，交叉部分为 $A \cap B$，两圆合并区域为 $A \cup B$，矩形（全集 $U$）中 $A$ 圆外的区域为 $\complement_U A$。

### 典型例题

**例1**：设 $A = \{1, 2, 3, 4, 5\}$，$B = \{3, 4, 5, 6, 7\}$，求 $A \cap B$，$A \cup B$。

解：
- $A \cap B = \{3, 4, 5\}$（公共元素）
- $A \cup B = \{1, 2, 3, 4, 5, 6, 7\}$（合并，不重复）

**例2**：全集 $U = \mathbb{R}$，$A = \{x \mid x < -1 \text{ 或 } x > 3\}$，求 $\complement_U A$。

解：$\complement_U A = \{x \mid -1 \leq x \leq 3\}$（即区间 $[-1, 3]$）

**例3**：全集 $U = \{1,2,3,4,5,6\}$，$A = \{1,3,5\}$，$B = \{2,3,4\}$，求 $(\complement_U A) \cap B$。

解：$\complement_U A = \{2,4,6\}$，$(\complement_U A) \cap B = \{2,4,6\} \cap \{2,3,4\} = \{2,4\}$

### 易错点

1. **补集依赖全集**：$\complement_U A$ 必须在全集 $U$ 范围内定义，不同的全集得到不同的补集
2. **"或"不是"异或"**：$A \cup B$ 中 $x \in A$ 或 $x \in B$ 是"包含或"，两者都满足也算
3. **区间运算**：用数轴辅助理解，$(-\infty, 2) \cap (1, +\infty) = (1, 2)$，注意端点开闭
4. **$A \cap B = \emptyset$ 的含义**：表示 $A, B$ 无公共元素（互不相交），不代表 $A$ 或 $B$ 为空集
5. **德摩根定律方向**：补集把 $\cup$ 变 $\cap$，把 $\cap$ 变 $\cup$，不能弄反

### 与初中知识的衔接

初中学习韦恩图（如四种命题的关系图），初步接触"交"与"并"的直观含义，但未用符号系统化表达。高中引入 $\cap, \cup, \complement$ 符号，将集合运算形式化。
