## 🎓 古典概型

### 定义

满足以下两个条件的概率模型称为**古典概型**：

1. **有限性**：样本空间只含**有限个**基本事件
2. **等可能性**：每个基本事件发生的可能性**相同**

### 古典概型概率公式

$$P(A) = \frac{\text{事件} A \text{ 包含的基本事件数}}{\text{样本空间的基本事件总数}} = \frac{|A|}{|\Omega|}$$

其中 $|A|$ 表示 $A$ 中基本事件的个数（有利结果数），$|\Omega|$ 为总数（所有可能结果数）。

### 计算方法

古典概型的计算核心是**排列组合**：

- **排列**（顺序有关）：$P_n^r = A_n^r = n(n-1)\cdots(n-r+1) = \dfrac{n!}{(n-r)!}$
- **组合**（顺序无关）：$C_n^r = \dbinom{n}{r} = \dfrac{n!}{r!(n-r)!}$

### 典型例题

**例1**：掷两枚骰子，求两点数之和为 7 的概率。

**解**：总结果 $|\Omega| = 36$。点数和为 7 的结果：$(1,6),(2,5),(3,4),(4,3),(5,2),(6,1)$，共 6 种。

$P = \dfrac{6}{36} = \dfrac{1}{6}$

**例2**：从 5 男 3 女中随机选 3 人，恰好有 2 女的概率。

**解**：$|\Omega| = C_8^3 = 56$，$|A| = C_3^2 \times C_5^1 = 3 \times 5 = 15$

$P = \dfrac{15}{56}$

**例3**：袋中红球 3 个、白球 2 个，不放回取 2 个，全是红球的概率。

**解**：$|\Omega| = C_5^2 = 10$，$|A| = C_3^2 = 3$，$P = \dfrac{3}{10}$

**例4**：4 张牌（A,2,3,4）随机排列，A 在首位的概率。

**解**：总排列 $4! = 24$，A 在首位的排列 $3! = 6$，$P = \dfrac{6}{24} = \dfrac{1}{4}$

### 应用范围

古典概型适用：抽签、摸球、掷骰子、扑克牌问题、抽样等**有限等可能**情形。

### 易错点

- 确认等可能性：抛硬币（正反面各 $\frac{1}{2}$）vs 不匀质骰子（各面不等可能）
- 用组合还是排列：无放回取球且"同时取"用组合（顺序无关）；先后取且顺序有关用排列
- 分子（有利结果）和分母（总结果）的计数方法要统一（都排列或都组合）
