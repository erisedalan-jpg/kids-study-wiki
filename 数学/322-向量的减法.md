## 🎓 向量的减法

### 相反向量

与向量 $\boldsymbol{a}$ 大小相等、方向相反的向量称为 $\boldsymbol{a}$ 的**相反向量**，记作 $-\boldsymbol{a}$。

$$\boldsymbol{a} + (-\boldsymbol{a}) = \boldsymbol{0}, \quad -(-\boldsymbol{a}) = \boldsymbol{a}$$

### 向量减法定义

$$\boldsymbol{a} - \boldsymbol{b} = \boldsymbol{a} + (-\boldsymbol{b})$$

**几何意义**：若 $\boldsymbol{a} = \overrightarrow{OA}$，$\boldsymbol{b} = \overrightarrow{OB}$，则

$$\boldsymbol{a} - \boldsymbol{b} = \overrightarrow{OA} - \overrightarrow{OB} = \overrightarrow{BA}$$

即：**从减数的终点指向被减数的终点**（同起点时，对角线反向）。

### 三角形法则（减法）

将 $\boldsymbol{a}$ 和 $\boldsymbol{b}$ 移到同一起点 $O$，$\boldsymbol{a} - \boldsymbol{b}$ 是从 $\boldsymbol{b}$ 的终点指向 $\boldsymbol{a}$ 的终点的向量。

### 典型例题

**例1**：已知平行四边形 $ABCD$ 中，$\overrightarrow{AB} = \boldsymbol{a}$，$\overrightarrow{AD} = \boldsymbol{b}$，求 $\overrightarrow{BD}$。

**解**：  
$$\overrightarrow{BD} = \overrightarrow{AD} - \overrightarrow{AB} = \boldsymbol{b} - \boldsymbol{a}$$

**例2**：已知 $\overrightarrow{OA} = (3, 4)$，$\overrightarrow{OB} = (1, 2)$，求 $\overrightarrow{BA}$。

**解**：$\overrightarrow{BA} = \overrightarrow{OA} - \overrightarrow{OB} = (3-1, 4-2) = (2, 2)$

### 重要结论

对任意三点 $A, B, C$：
$$\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA}$$

这是坐标系中求两点间向量的基本公式。

### 易错点

- $\boldsymbol{a} - \boldsymbol{b}$ 的方向：从 $\boldsymbol{b}$ 的终点指向 $\boldsymbol{a}$ 的终点，方向容易搞反
- $\overrightarrow{AB} = B - A$（终点减起点），记住"终减始"
- 不能直接用模做减法：$|\boldsymbol{a} - \boldsymbol{b}| \neq |\boldsymbol{a}| - |\boldsymbol{b}|$
