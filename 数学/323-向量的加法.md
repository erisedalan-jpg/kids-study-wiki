## 🎓 向量的加法

### 定义

**向量加法**遵循**三角形法则**或**平行四边形法则**。

#### 三角形法则

将向量 $\boldsymbol{b}$ 的起点移到 $\boldsymbol{a}$ 的终点，从 $\boldsymbol{a}$ 的起点到 $\boldsymbol{b}$ 的终点所得的向量称为 $\boldsymbol{a}$ 与 $\boldsymbol{b}$ 的**和**，记作 $\boldsymbol{a} + \boldsymbol{b}$。

即：若 $\overrightarrow{AB} = \boldsymbol{a}$，$\overrightarrow{BC} = \boldsymbol{b}$，则 $\boldsymbol{a} + \boldsymbol{b} = \overrightarrow{AC}$。

#### 平行四边形法则

以 $\boldsymbol{a}$ 和 $\boldsymbol{b}$ 为邻边作平行四边形，从公共起点出发的**对角线向量**即为 $\boldsymbol{a} + \boldsymbol{b}$。

### 运算律

设 $\boldsymbol{a}, \boldsymbol{b}, \boldsymbol{c}$ 为任意向量：

$$\boldsymbol{a} + \boldsymbol{b} = \boldsymbol{b} + \boldsymbol{a} \quad \text{（交换律）}$$

$$(\boldsymbol{a} + \boldsymbol{b}) + \boldsymbol{c} = \boldsymbol{a} + (\boldsymbol{b} + \boldsymbol{c}) \quad \text{（结合律）}$$

$$\boldsymbol{a} + \boldsymbol{0} = \boldsymbol{a}$$

### 多向量求和（多边形法则）

$$\overrightarrow{A_1A_2} + \overrightarrow{A_2A_3} + \cdots + \overrightarrow{A_{n-1}A_n} = \overrightarrow{A_1A_n}$$

首尾相接，取首到尾的向量。若首尾重合（封闭多边形），则和为零向量。

### 典型例题

**例1**：已知 $\overrightarrow{OA} = \boldsymbol{a}$，$\overrightarrow{OB} = \boldsymbol{b}$，$M$ 为 $AB$ 中点，求 $\overrightarrow{OM}$。

**解**：  
$$\overrightarrow{OM} = \overrightarrow{OA} + \overrightarrow{AM} = \boldsymbol{a} + \frac{1}{2}\overrightarrow{AB} = \boldsymbol{a} + \frac{1}{2}(\boldsymbol{b} - \boldsymbol{a}) = \frac{\boldsymbol{a} + \boldsymbol{b}}{2}$$

**例2**：化简 $\overrightarrow{AB} + \overrightarrow{BC} + \overrightarrow{CA}$。

**解**：$\overrightarrow{AB} + \overrightarrow{BC} = \overrightarrow{AC}$，再加 $\overrightarrow{CA}$ 得 $\overrightarrow{AA} = \boldsymbol{0}$。

### 易错点

- 向量加法用**三角形法则**（首尾相接），不是数量相加
- 平行四边形法则中，合向量是**对角线**，不是邻边
- 零向量加任意向量等于原向量，类比 $0 + a = a$

### 与初中衔接

初中学习了数轴上正负数的"加法"，方向相同加大、方向相反减小，是一维向量加法的特例。高中推广到平面任意方向。
