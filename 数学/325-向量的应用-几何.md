## 🎓 向量的应用——几何

### 核心思路

将几何问题转化为向量运算，利用向量代数证明几何定理或求解几何量。

**基本策略**：选取合适基底（如 $\overrightarrow{OA}, \overrightarrow{OB}$），将图中各点、线段用向量表示，再用代数方法处理。

### 重要公式

#### 中点公式

$M$ 是 $AB$ 中点：$\overrightarrow{OM} = \dfrac{\overrightarrow{OA} + \overrightarrow{OB}}{2}$

#### 内分点公式

$P$ 分 $AB$ 为 $AP:PB = m:n$：

$$\overrightarrow{OP} = \frac{n\overrightarrow{OA} + m\overrightarrow{OB}}{m+n}$$

#### 三角形重心

三角形 $ABC$ 的重心 $G$：

$$\overrightarrow{OG} = \frac{\overrightarrow{OA} + \overrightarrow{OB} + \overrightarrow{OC}}{3}$$

### 典型例题

**例1（证明平行四边形对角线互相平分）**：

设平行四边形 $ABCD$ 中，对角线 $AC$ 和 $BD$ 的中点分别为 $M, N$，证明 $M = N$。

**证**：$\overrightarrow{OM} = \dfrac{\overrightarrow{OA} + \overrightarrow{OC}}{2}$，$\overrightarrow{ON} = \dfrac{\overrightarrow{OB} + \overrightarrow{OD}}{2}$。

由 $ABCD$ 为平行四边形：$\overrightarrow{OA} + \overrightarrow{OC} = \overrightarrow{OB} + \overrightarrow{OD}$（对角线中点重合），故 $M = N$。

**例2（利用向量求三角形面积）**：

已知 $A(0,0)$，$B(3,0)$，$C(1,2)$，用向量叉乘思路求面积。

**解**：$\overrightarrow{AB} = (3,0)$，$\overrightarrow{AC} = (1,2)$，  
面积 $= \dfrac{1}{2}|3 \times 2 - 0 \times 1| = \dfrac{1}{2} \times 6 = 3$

**例3（证明共线）**：

$O$ 为原点，$\overrightarrow{OA} = \boldsymbol{a}$，$\overrightarrow{OB} = \boldsymbol{b}$，$\overrightarrow{OC} = 2\boldsymbol{a} - \boldsymbol{b}$。证明 $A, B, C$ 不共线，而 $A, B$ 与 $\boldsymbol{a} + \boldsymbol{b}$ 端点的关系。

### 常见几何定理向量证法

| 定理 | 向量方法 |
|------|---------|
| 三角形中位线定理 | $\overrightarrow{MN} = \frac{1}{2}\overrightarrow{BC}$ |
| 平行四边形对角线互平分 | 中点坐标相等 |
| 直角判定 | 两向量数量积为 0 |
| 点在线段上 | $\overrightarrow{AP} = \lambda\overrightarrow{AB}$，$0 < \lambda < 1$ |

### 易错点

- 向量证明几何要先固定参考点（原点 $O$），再表示其他向量
- 中位线 $\overrightarrow{MN} = \frac{1}{2}\overrightarrow{BC}$，注意方向（$M$ 是 $AB$ 中点，$N$ 是 $AC$ 中点）
