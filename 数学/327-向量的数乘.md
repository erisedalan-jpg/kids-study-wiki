## 🎓 向量的数乘

### 定义

实数 $\lambda$ 与向量 $\boldsymbol{a}$ 的乘积 $\lambda\boldsymbol{a}$ 是一个向量：

- **模**：$|\lambda\boldsymbol{a}| = |\lambda||\boldsymbol{a}|$
- **方向**：
  - $\lambda > 0$ 时，$\lambda\boldsymbol{a}$ 与 $\boldsymbol{a}$ **同向**
  - $\lambda < 0$ 时，$\lambda\boldsymbol{a}$ 与 $\boldsymbol{a}$ **反向**
  - $\lambda = 0$ 时，$\lambda\boldsymbol{a} = \boldsymbol{0}$

### 运算律

设 $\lambda, \mu$ 为实数，$\boldsymbol{a}, \boldsymbol{b}$ 为向量：

$$\lambda(\mu\boldsymbol{a}) = (\lambda\mu)\boldsymbol{a}$$

$$(\lambda + \mu)\boldsymbol{a} = \lambda\boldsymbol{a} + \mu\boldsymbol{a} \quad \text{（数的分配律）}$$

$$\lambda(\boldsymbol{a} + \boldsymbol{b}) = \lambda\boldsymbol{a} + \lambda\boldsymbol{b} \quad \text{（向量的分配律）}$$

$$1 \cdot \boldsymbol{a} = \boldsymbol{a}, \quad (-1)\boldsymbol{a} = -\boldsymbol{a}, \quad 0 \cdot \boldsymbol{a} = \boldsymbol{0}$$

### 共线向量定理

向量 $\boldsymbol{b}$ 与非零向量 $\boldsymbol{a}$ **共线**，当且仅当存在实数 $\lambda$，使得

$$\boldsymbol{b} = \lambda\boldsymbol{a}$$

这是判断两向量共线的代数方法。

### 典型例题

**例1**：$M$ 是线段 $AB$ 的三等分点（靠近 $A$），$\overrightarrow{OA} = \boldsymbol{a}$，$\overrightarrow{OB} = \boldsymbol{b}$，求 $\overrightarrow{OM}$。

**解**：$\overrightarrow{OM} = \overrightarrow{OA} + \overrightarrow{AM} = \boldsymbol{a} + \frac{1}{3}\overrightarrow{AB} = \boldsymbol{a} + \frac{1}{3}(\boldsymbol{b} - \boldsymbol{a}) = \frac{2}{3}\boldsymbol{a} + \frac{1}{3}\boldsymbol{b}$

**例2**：已知 $\boldsymbol{a} = (2, -1)$，求 $3\boldsymbol{a}$。

**解**：$3\boldsymbol{a} = (6, -3)$，模 $|3\boldsymbol{a}| = 3|\boldsymbol{a}| = 3\sqrt{5}$。

### 单位向量

若 $\boldsymbol{a} \neq \boldsymbol{0}$，则 $\boldsymbol{a}$ 方向上的单位向量为：

$$\hat{\boldsymbol{a}} = \frac{\boldsymbol{a}}{|\boldsymbol{a}|}$$

### 易错点

- $\lambda\boldsymbol{a}$ 是向量，$|\lambda||\boldsymbol{a}|$ 是数量（模）
- $\lambda < 0$ 时方向反向，容易忘记
- 分配律两种形式都要熟练
