## 🎓 复数概念

### 为什么需要复数

实数范围内，方程 $x^2 + 1 = 0$ 无解（任何实数的平方 $\geq 0$）。  
为扩充数系，引入**虚数单位** $\text{i}$，规定：

$$\text{i}^2 = -1, \quad \text{i} = \sqrt{-1}$$

### 复数的定义

形如 $z = a + b\text{i}$（$a, b \in \mathbb{R}$）的数称为**复数**：

- $a$ 称为复数的**实部**（real part），记作 $\text{Re}(z) = a$
- $b$ 称为复数的**虚部**（imaginary part），记作 $\text{Im}(z) = b$
- $b\text{i}$ 称为**虚部**项

### 复数的分类

| 条件 | 名称 |
|------|------|
| $b = 0$ | 实数（$z = a$） |
| $b \neq 0$ | 虚数 |
| $a = 0, b \neq 0$ | 纯虚数（如 $3\text{i}$） |
| $b \neq 0$ | 虚数（含纯虚数） |

实数是复数的特例，全体复数集记作 $\mathbb{C}$（$\mathbb{R} \subset \mathbb{C}$）。

### 复数相等

$$a + b\text{i} = c + d\text{i} \iff a = c \text{ 且 } b = d$$

**推论**：$a + b\text{i} = 0 \iff a = 0 \text{ 且 } b = 0$

### 虚数单位的幂次

$$\text{i}^1 = \text{i},\quad \text{i}^2 = -1,\quad \text{i}^3 = -\text{i},\quad \text{i}^4 = 1$$

规律：以 $4$ 为周期循环。$\text{i}^n$ 的值由 $n \mod 4$ 决定。

### 典型例题

**例1**：若 $(2x-1) + (y+3)\text{i} = 3 - \text{i}$，求实数 $x, y$。

**解**：$2x - 1 = 3 \Rightarrow x = 2$；$y + 3 = -1 \Rightarrow y = -4$

**例2**：$\text{i}^{2026}$ 的值。

**解**：$2026 = 4 \times 506 + 2$，故 $\text{i}^{2026} = \text{i}^2 = -1$

**例3**：$\dfrac{1}{\text{i}} = ?$

**解**：$\dfrac{1}{\text{i}} = \dfrac{\text{i}}{\text{i}^2} = \dfrac{\text{i}}{-1} = -\text{i}$

### 易错点

- 虚部是 $b$（实数），不是 $b\text{i}$
- "虚数"包含纯虚数，但实数不是虚数
- 复数没有大小关系（不能比较 $z_1 > z_2$）
