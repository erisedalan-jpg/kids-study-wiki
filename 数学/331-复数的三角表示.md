## 🎓 复数的三角表示

### 三角形式

任意非零复数 $z = a + b\text{i}$ 可以表示为：

$$z = r(\cos\theta + \text{i}\sin\theta)$$

其中：
- $r = |z| = \sqrt{a^2 + b^2}$：复数的**模**
- $\theta$：复数的**辐角**（主辐角 $\theta \in (-\pi, \pi]$）
- $\cos\theta = \dfrac{a}{r}$，$\sin\theta = \dfrac{b}{r}$

### 乘法的三角形式

$$z_1 z_2 = r_1(\cos\theta_1 + \text{i}\sin\theta_1) \cdot r_2(\cos\theta_2 + \text{i}\sin\theta_2)$$

$$= r_1 r_2 [\cos(\theta_1 + \theta_2) + \text{i}\sin(\theta_1 + \theta_2)]$$

**结论**：乘法中，**模相乘，辐角相加**。

### 除法的三角形式

$$\frac{z_1}{z_2} = \frac{r_1}{r_2}[\cos(\theta_1 - \theta_2) + \text{i}\sin(\theta_1 - \theta_2)]$$

**结论**：除法中，**模相除，辐角相减**。

### 棣莫弗定理（De Moivre's Theorem）

$$[r(\cos\theta + \text{i}\sin\theta)]^n = r^n(\cos n\theta + \text{i}\sin n\theta)$$

特别地，$n$ 为正整数时：$(\cos\theta + \text{i}\sin\theta)^n = \cos n\theta + \text{i}\sin n\theta$

### 欧拉公式（选学）

$$e^{\text{i}\theta} = \cos\theta + \text{i}\sin\theta$$

故 $z = re^{\text{i}\theta}$，称为**指数形式**。

### 典型例题

**例1**：将 $z = 1 + \text{i}$ 化为三角形式。

**解**：$r = \sqrt{1^2 + 1^2} = \sqrt{2}$，$\theta = 45°$（第一象限，$\tan\theta = 1$）  
$z = \sqrt{2}(\cos 45° + \text{i}\sin 45°)$

**例2**：$z_1 = \sqrt{2}(\cos 30° + \text{i}\sin 30°)$，$z_2 = \sqrt{3}(\cos 60° + \text{i}\sin 60°)$，求 $z_1 z_2$。

**解**：$z_1 z_2 = \sqrt{6}[\cos 90° + \text{i}\sin 90°] = \sqrt{6}\text{i}$

**例3**：$(\cos 18° + \text{i}\sin 18°)^{10}$

**解**：由棣莫弗定理：$= \cos 180° + \text{i}\sin 180° = -1$

### 易错点

- 辐角 $\theta$ 要在正确的象限（$\tan\theta$ 只给参考角，还需看 $a, b$ 的符号）
- 乘法辐角相加后可能超过 $\pi$，需化为主辐角范围
- 棣莫弗定理只对三角形式直接有效，代数形式要先转换
