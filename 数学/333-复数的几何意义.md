## 🎓 复数的几何意义

### 复平面

建立平面直角坐标系，将复数 $z = a + b\text{i}$ 对应平面上的点 $Z(a, b)$：

- $x$ 轴称为**实轴**（real axis）：表示实部
- $y$ 轴称为**虚轴**（imaginary axis）：表示虚部
- 此坐标平面称为**复平面**（complex plane），又称**高斯平面**

**对应关系**：
$$z = a + b\text{i} \longleftrightarrow Z(a, b) \longleftrightarrow \overrightarrow{OZ}$$

每个复数对应复平面上唯一一个点，也对应从原点出发的唯一一个向量。

### 复数的模

复数 $z = a + b\text{i}$ 的**模**（绝对值）：

$$|z| = |a + b\text{i}| = \sqrt{a^2 + b^2}$$

几何意义：点 $Z(a, b)$ 到原点的距离。

### 共轭复数

$z = a + b\text{i}$ 的**共轭复数**：

$$\bar{z} = a - b\text{i}$$

几何意义：$Z$ 关于实轴的对称点。

性质：
$$z + \bar{z} = 2a \in \mathbb{R}, \quad z \cdot \bar{z} = a^2 + b^2 = |z|^2$$

### 复数的辐角

向量 $\overrightarrow{OZ}$ 与正实轴正方向的夹角 $\theta$ 称为复数 $z$ 的**辐角**（argument）：

$$\cos\theta = \frac{a}{|z|}, \quad \sin\theta = \frac{b}{|z|}$$

辐角不唯一（相差 $2k\pi$），主辐角范围 $(-\pi, \pi]$。

### 两复数对应点的距离

$z_1 = a_1 + b_1\text{i}$，$z_2 = a_2 + b_2\text{i}$ 对应点 $Z_1, Z_2$ 的距离：

$$|z_1 - z_2| = \sqrt{(a_1-a_2)^2 + (b_1-b_2)^2}$$

### 典型例题

**例1**：在复平面内画出 $z_1 = 2 + 3\text{i}$，$z_2 = -1 - \text{i}$ 对应的点，并求两点距离。

**解**：$Z_1(2, 3)$，$Z_2(-1, -1)$，$|z_1 - z_2| = \sqrt{9 + 16} = 5$

**例2**：复数 $z = 1 - \sqrt{3}\text{i}$ 的模和辐角。

**解**：$|z| = \sqrt{1 + 3} = 2$，$\cos\theta = \frac{1}{2}$，$\sin\theta = -\frac{\sqrt{3}}{2}$，$\theta = -60°$（即 $300°$）。

**例3**：满足 $|z - 1| = 2$ 的复数 $z$ 对应的点的轨迹。

**解**：以 $Z_0(1, 0)$ 为圆心，$2$ 为半径的圆。

### 易错点

- 虚轴上的点（$a=0, b \neq 0$）表示**纯虚数**，实轴上的点表示**实数**
- 模是非负实数，$|z| \geq 0$
- 共轭复数 $\bar{z}$ 与 $z$ 关于**实轴**对称（不是虚轴）
