## 🎓 复数的加减运算

### 运算法则

设 $z_1 = a + b\text{i}$，$z_2 = c + d\text{i}$（$a, b, c, d \in \mathbb{R}$）：

$$z_1 + z_2 = (a + c) + (b + d)\text{i}$$

$$z_1 - z_2 = (a - c) + (b - d)\text{i}$$

**规则**：实部与实部相加减，虚部与虚部相加减。

### 运算律

$$z_1 + z_2 = z_2 + z_1 \quad \text{（交换律）}$$

$$(z_1 + z_2) + z_3 = z_1 + (z_2 + z_3) \quad \text{（结合律）}$$

$$z + 0 = z, \quad z + (-z) = 0$$

### 几何意义

复数加法对应**向量加法**（平行四边形法则）：

$z_1 + z_2$ 对应 $\overrightarrow{OZ_1} + \overrightarrow{OZ_2}$，几何上是平行四边形的对角线。

$z_1 - z_2$ 对应 $\overrightarrow{Z_2 Z_1}$（从 $Z_2$ 指向 $Z_1$ 的向量）。

### 模的不等式

$$|z_1 + z_2| \leq |z_1| + |z_2| \quad \text{（三角不等式）}$$

$$\left||z_1| - |z_2|\right| \leq |z_1 - z_2|$$

### 典型例题

**例1**：$(3 + 2\text{i}) + (-1 + 4\text{i}) - (2 - \text{i})$

**解**：  
实部：$3 + (-1) - 2 = 0$  
虚部：$2 + 4 - (-1) = 7$  
结果：$7\text{i}$

**例2**：已知 $z_1 + z_2 = 4 + 3\text{i}$，$z_1 - z_2 = 2 + \text{i}$，求 $z_1, z_2$。

**解**：  
$2z_1 = 6 + 4\text{i} \Rightarrow z_1 = 3 + 2\text{i}$  
$2z_2 = 2 + 2\text{i} \Rightarrow z_2 = 1 + \text{i}$

**例3**：若 $z = a + b\text{i}$，$\bar{z} = a - b\text{i}$，求 $z + \bar{z}$ 和 $z - \bar{z}$。

**解**：$z + \bar{z} = 2a$（实数），$z - \bar{z} = 2b\text{i}$（纯虚数或零）

### 与实数运算的联系

实数是 $b = 0$ 时的特殊复数，此时 $(a_1 + 0\text{i}) + (a_2 + 0\text{i}) = (a_1 + a_2) + 0\text{i}$，退化为实数加法。复数加减运算是实数运算的推广。

### 易错点

- 虚部只加减虚部（带 $\text{i}$），不能与实部混合
- 减法记得变号：$-( c + d\text{i}) = -c - d\text{i}$
- 三角不等式在向量中也成立，是同一道理
