## 🎓 平面向量坐标运算

### 向量坐标表示

设 $\boldsymbol{a} = (a_1, a_2)$，$\boldsymbol{b} = (b_1, b_2)$，$\lambda$ 为实数。

### 基本运算公式

| 运算 | 公式 |
|------|------|
| 加法 | $\boldsymbol{a} + \boldsymbol{b} = (a_1 + b_1,\ a_2 + b_2)$ |
| 减法 | $\boldsymbol{a} - \boldsymbol{b} = (a_1 - b_1,\ a_2 - b_2)$ |
| 数乘 | $\lambda\boldsymbol{a} = (\lambda a_1,\ \lambda a_2)$ |
| 模 | $|\boldsymbol{a}| = \sqrt{a_1^2 + a_2^2}$ |
| 数量积 | $\boldsymbol{a} \cdot \boldsymbol{b} = a_1 b_1 + a_2 b_2$ |

### 两点间向量

若 $A(x_1, y_1)$，$B(x_2, y_2)$，则

$$\overrightarrow{AB} = (x_2 - x_1,\ y_2 - y_1)$$

$$|\overrightarrow{AB}| = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}$$

（即两点间距离公式）

### 中点公式

线段 $AB$ 的中点 $M$ 坐标为：

$$M = \left(\frac{x_1+x_2}{2},\ \frac{y_1+y_2}{2}\right)$$

### 共线条件

$\boldsymbol{a} = (a_1, a_2)$ 与 $\boldsymbol{b} = (b_1, b_2)$ 共线，当且仅当：

$$a_1 b_2 - a_2 b_1 = 0$$

### 典型例题

**例1**：$A(1, 2)$，$B(4, 6)$，求 $\overrightarrow{AB}$、$|\overrightarrow{AB}|$ 和中点 $M$。

**解**：  
$\overrightarrow{AB} = (3, 4)$，$|\overrightarrow{AB}| = 5$，$M = (2.5, 4)$。

**例2**：已知 $\boldsymbol{a} = (2, 3)$，$\boldsymbol{b} = (1, -1)$，求 $2\boldsymbol{a} - 3\boldsymbol{b}$。

**解**：$2\boldsymbol{a} - 3\boldsymbol{b} = (4-3, 6+3) = (1, 9)$。

**例3**：判断 $\boldsymbol{a} = (3, -2)$ 与 $\boldsymbol{b} = (-6, 4)$ 是否共线。

**解**：$3 \times 4 - (-2) \times (-6) = 12 - 12 = 0$，故共线（且反向）。

### 易错点

- $\overrightarrow{AB}$ 是"终点坐标 $-$ 起点坐标"，方向很重要
- 共线条件 $a_1 b_2 = a_2 b_1$（交叉相乘相等），别漏负号
- 模的计算是平方和开根号，不是坐标之和
