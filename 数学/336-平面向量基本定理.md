## 🎓 平面向量基本定理

### 定理内容

如果 $\boldsymbol{e}_1, \boldsymbol{e}_2$ 是同一平面内的两个**不共线向量**，则对该平面内的**任意向量** $\boldsymbol{a}$，存在**唯一**一对实数 $\lambda_1, \lambda_2$，使得

$$\boldsymbol{a} = \lambda_1 \boldsymbol{e}_1 + \lambda_2 \boldsymbol{e}_2$$

$\{\boldsymbol{e}_1, \boldsymbol{e}_2\}$ 称为该平面的一组**基底**（basis），$\lambda_1, \lambda_2$ 称为 $\boldsymbol{a}$ 在该基底下的**坐标**（分量）。

### 直角坐标基底

最常用的基底是**单位正交基底** $\{\boldsymbol{i}, \boldsymbol{j}\}$：

- $\boldsymbol{i} = (1, 0)$：$x$ 轴正方向单位向量
- $\boldsymbol{j} = (0, 1)$：$y$ 轴正方向单位向量

任意平面向量 $\boldsymbol{a} = (x, y) = x\boldsymbol{i} + y\boldsymbol{j}$

### 为什么需要"不共线"

若 $\boldsymbol{e}_1 \parallel \boldsymbol{e}_2$，则 $\lambda_1\boldsymbol{e}_1 + \lambda_2\boldsymbol{e}_2$ 只能表示沿某一方向的向量，无法覆盖整个平面。

### 向量共线的充要条件（坐标形式）

设 $\boldsymbol{a} = (x_1, y_1)$，$\boldsymbol{b} = (x_2, y_2)$，则

$$\boldsymbol{a} \parallel \boldsymbol{b} \iff x_1 y_2 - x_2 y_1 = 0$$

### 典型例题

**例1**：已知 $\boldsymbol{e}_1, \boldsymbol{e}_2$ 不共线，$\overrightarrow{OA} = 3\boldsymbol{e}_1 + \boldsymbol{e}_2$，$\overrightarrow{OB} = \boldsymbol{e}_1 - \boldsymbol{e}_2$，$\overrightarrow{OC} = 2\boldsymbol{e}_1 + t\boldsymbol{e}_2$，若 $A, B, C$ 共线，求 $t$。

**解**：$\overrightarrow{AB} = \overrightarrow{OB} - \overrightarrow{OA} = -2\boldsymbol{e}_1 - 2\boldsymbol{e}_2$，$\overrightarrow{AC} = \overrightarrow{OC} - \overrightarrow{OA} = -\boldsymbol{e}_1 + (t-1)\boldsymbol{e}_2$。

$A, B, C$ 共线 $\Rightarrow \overrightarrow{AC} = \lambda\overrightarrow{AB}$：
$$-\boldsymbol{e}_1 + (t-1)\boldsymbol{e}_2 = \lambda(-2\boldsymbol{e}_1 - 2\boldsymbol{e}_2)$$

因 $\boldsymbol{e}_1, \boldsymbol{e}_2$ 不共线，比较系数：$-1 = -2\lambda \Rightarrow \lambda = \frac{1}{2}$，$t - 1 = -2\lambda = -1 \Rightarrow t = 0$。

### 易错点

- 基底必须是**不共线**的两向量（不必是单位向量或正交向量）
- 表示方式**唯一**：若 $\lambda_1\boldsymbol{e}_1 + \lambda_2\boldsymbol{e}_2 = \mu_1\boldsymbol{e}_1 + \mu_2\boldsymbol{e}_2$，则 $\lambda_1 = \mu_1, \lambda_2 = \mu_2$
