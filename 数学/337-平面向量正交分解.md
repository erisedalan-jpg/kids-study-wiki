## 🎓 平面向量正交分解

### 定义

将向量 $\boldsymbol{a}$ 分解为两个**互相垂直**方向上的分量之和，称为**正交分解**。

以直角坐标系为基准：

$$\boldsymbol{a} = a_x \boldsymbol{i} + a_y \boldsymbol{j}$$

其中 $a_x = |\boldsymbol{a}|\cos\theta$，$a_y = |\boldsymbol{a}|\sin\theta$，$\theta$ 为 $\boldsymbol{a}$ 与 $x$ 轴正方向的夹角。

### 向量的坐标表示

$$\boldsymbol{a} = (a_x, a_y)$$

- **模**：$|\boldsymbol{a}| = \sqrt{a_x^2 + a_y^2}$
- **方向角**：$\tan\theta = \dfrac{a_y}{a_x}$（$a_x \neq 0$）

### 任意方向的分解

将向量 $\boldsymbol{a}$ 分解到 $\boldsymbol{e}$ 方向和其垂直方向：

- 沿 $\boldsymbol{e}$ 方向的分量（射影）：$|\boldsymbol{a}|\cos\theta$（其中 $\theta$ 为 $\boldsymbol{a}$ 与 $\boldsymbol{e}$ 的夹角）
- 垂直 $\boldsymbol{e}$ 方向的分量：$|\boldsymbol{a}|\sin\theta$

### 典型例题

**例1**：向量 $\boldsymbol{a}$ 模为 $10$，方向角为 $60°$，将其正交分解。

**解**：  
$$a_x = 10\cos 60° = 5, \quad a_y = 10\sin 60° = 5\sqrt{3}$$
$$\boldsymbol{a} = (5, 5\sqrt{3})$$

**例2**（物理应用）：质量 $m$ 的物体在斜面（倾角 $\theta$）上，重力 $G = mg$ 向量正交分解：
- 沿斜面方向（向下）：$mg\sin\theta$
- 垂直斜面方向（压力）：$mg\cos\theta$

**例3**：已知 $\boldsymbol{a} = (3, 4)$，求其模和方向角。

**解**：$|\boldsymbol{a}| = \sqrt{9 + 16} = 5$，$\tan\theta = \frac{4}{3}$，$\theta = \arctan\frac{4}{3} \approx 53.1°$。

### 易错点

- 分解方向必须**垂直**，才叫正交分解；否则只是普通的向量分解
- 方向角 $\theta$ 从正 $x$ 轴逆时针计量，范围 $[0°, 360°)$
- $\tan\theta$ 只给出参考角，还需判断所在象限确定实际角度
