## 🎓 数列综合应用

### 常见题型分类

#### 1. 求通项公式

**已知递推关系求通项**：

- **叠加法**（累加）：$a_{n+1} - a_n = f(n)$ 型
  $$a_n = a_1 + \sum_{k=1}^{n-1}(a_{k+1} - a_k) = a_1 + \sum_{k=1}^{n-1}f(k)$$

- **累乘法**：$a_{n+1}/a_n = f(n)$ 型
  $$a_n = a_1 \times \prod_{k=1}^{n-1} \frac{a_{k+1}}{a_k} = a_1 \prod_{k=1}^{n-1}f(k)$$

- **换元法**：令 $b_n = a_n + c$（凑成等差）或 $b_n = \frac{1}{a_n}$（凑成等差）

#### 2. 等差等比混合

**例（错位相减法）**：求 $S = 1 \cdot 2 + 2 \cdot 2^2 + 3 \cdot 2^3 + \cdots + n \cdot 2^n$

$$S = \sum_{k=1}^n k \cdot 2^k, \quad 2S = \sum_{k=1}^n k \cdot 2^{k+1} = \sum_{k=2}^{n+1} (k-1) \cdot 2^k$$

两式相减，化简得 $S = (n-1) \cdot 2^{n+1} + 2$。

#### 3. 数列求和（分组）

将数列分为等差部分和等比部分分别求和：

**例**：$1 + \frac{1}{2} + 2 + \frac{1}{4} + 3 + \frac{1}{8} + \cdots + n + \frac{1}{2^n}$

$= (1 + 2 + \cdots + n) + (\frac{1}{2} + \frac{1}{4} + \cdots + \frac{1}{2^n})$

$= \frac{n(n+1)}{2} + 1 - \frac{1}{2^n}$

### 实际应用场景

#### 复利计算（等比数列）

初始本金 $P$，年利率 $r$，$n$ 年后本利和：

$$A_n = P(1+r)^n$$

这是公比 $q = 1+r$ 的等比数列。

**分期付款**：每期付款 $x$ 元，共 $n$ 期，利率 $r$，总现值：

$$P = x \cdot \frac{1-(1+r)^{-n}}{r}$$

#### 人口增长模型

设初始人口 $P_0$，年增长率 $r$，第 $n$ 年末人口 $P_n = P_0(1+r)^n$（等比模型）。

#### 分期储蓄

每年末存入 $a$ 元，年利率 $r$，存 $n$ 年后总额：

$$S = a \cdot \frac{(1+r)^n - 1}{r}$$

### 典型综合题

**例1**：数列 $\{a_n\}$，$a_1 = 1$，$a_{n+1} = 2a_n + 1$，求通项公式。

**解**（换元法）：令 $b_n = a_n + 1$，则 $b_{n+1} = a_{n+1} + 1 = 2a_n + 2 = 2b_n$，$\{b_n\}$ 是公比 $2$ 的等比数列，$b_1 = a_1 + 1 = 2$，故 $b_n = 2^n$，$a_n = 2^n - 1$。

**例2**：利用等差等比判断数列 $1, 3, 9, 27, \ldots$ 的类型及第 $n$ 项。

**解**：相邻比 $= 3$（公比），等比数列，$a_n = 3^{n-1}$。

**例3**（金融）：某人贷款 100 万元，年利率 6%，按年还款，30 年还清，每年还多少？

**解**：设每年还 $x$ 万，现值方程：$100 = x \cdot \dfrac{1-(1.06)^{-30}}{0.06}$，$x \approx 7.27$ 万。

### 数列与函数的联系

- 等差数列：$a_n = dn + b$（线性函数的离散版）
- 等比数列：$a_n = a_1 q^{n-1}$（指数函数的离散版）

### 易错点

- 换元法凑等差等比时，要验证第一项满足新数列的形式
- 错位相减法：$S$ 和 $qS$（或 $2S$）的相减，注意对齐项
- 实际问题中注意复利的"利率"是以期为单位（月利率 vs 年利率）
