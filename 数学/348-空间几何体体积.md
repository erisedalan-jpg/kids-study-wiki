## 🎓 空间几何体体积

### 基本公式汇总

| 几何体 | 体积公式 |
|--------|---------|
| 长方体 | $V = abc$（$a,b,c$ 为三边） |
| 正方体 | $V = a^3$ |
| 棱柱 | $V = S_{\text{底}} \cdot h$ |
| 圆柱 | $V = \pi r^2 h$ |
| 棱锥 | $V = \dfrac{1}{3} S_{\text{底}} \cdot h$ |
| 圆锥 | $V = \dfrac{1}{3}\pi r^2 h$ |
| 球 | $V = \dfrac{4}{3}\pi r^3$ |

**核心关系**：柱体 $= 3 \times$ 同底等高锥体

### 棱台、圆台体积

$$V_{\text{台}} = \frac{h}{3}(S_1 + S_2 + \sqrt{S_1 S_2})$$

其中 $S_1, S_2$ 为两底面面积，$h$ 为高。

### 祖暅原理（等积原理）

**祖暅原理**：两个几何体被平行于底面的截面截得的截面面积恒相等，则两几何体体积相等。

应用：推导圆锥体积 $= \dfrac{1}{3}$ 圆柱体积（通过与等底等高棱锥/棱柱对比）。

### 组合体体积

复杂几何体 = 简单几何体的**叠加**或**挖减**：

$$V_{\text{组合}} = V_1 + V_2 - V_{\text{重叠部分}}$$

### 典型例题

**例1**：底面半径 $r = 3$，高 $h = 4$ 的圆柱，体积？

**解**：$V = \pi \times 9 \times 4 = 36\pi$

**例2**：底面边长 $a = 6$，高 $h = 5$ 的正四棱锥，体积？

**解**：$V = \dfrac{1}{3} \times 36 \times 5 = 60$

**例3**：一个圆台，上底半径 $1$，下底半径 $3$，高 $2$，求体积。

**解**：$S_1 = \pi$，$S_2 = 9\pi$，$V = \dfrac{2}{3}(\pi + 9\pi + 3\pi) = \dfrac{2}{3} \times 13\pi = \dfrac{26\pi}{3}$

**例4**（组合体）：圆柱上方放一个圆锥，底面半径均为 $2$，圆柱高 $3$，圆锥高 $2$，求总体积。

**解**：$V = \pi \times 4 \times 3 + \dfrac{1}{3} \times \pi \times 4 \times 2 = 12\pi + \dfrac{8\pi}{3} = \dfrac{44\pi}{3}$

### 与初中衔接

初中学习了长方体、圆柱、圆锥体积（小学已有初步认识），高中新增：棱锥与圆锥的 $\dfrac{1}{3}$ 关系推导、圆台体积、祖暅原理等。

### 易错点

- 棱锥体积是 $\boldsymbol{\dfrac{1}{3}}$ 倍同底等高棱柱（不是 $\dfrac{1}{3}$ 底面积乘母线）
- 圆台公式记不住时可用微积分思路推导
- 组合体注意重叠部分不要重复计算
