## 🎓 等比数列前 $n$ 项和

### 公式推导

设等比数列首项 $a_1$，公比 $q$，前 $n$ 项和 $S_n$：

**当 $q \neq 1$ 时**（错位相减法）：

$$S_n = a_1 + a_1 q + a_1 q^2 + \cdots + a_1 q^{n-1}$$

$$qS_n = a_1 q + a_1 q^2 + \cdots + a_1 q^{n-1} + a_1 q^n$$

两式相减：$(1-q)S_n = a_1(1 - q^n)$

$$\boxed{S_n = \frac{a_1(1-q^n)}{1-q} = \frac{a_1(q^n-1)}{q-1}} \quad (q \neq 1)$$

**当 $q = 1$ 时**（等差，公差为 0）：

$$S_n = na_1$$

### 公式的等价形式

$$S_n = \frac{a_1 - a_n q}{1-q} = \frac{a_n q - a_1}{q-1} \quad (q \neq 1)$$

（利用 $a_n = a_1 q^{n-1}$，即 $a_1 q^n = a_n q$）

### 重要性质

**性质（等比分段）**：$S_n, S_{2n} - S_n, S_{3n} - S_{2n}$ 成等比数列（$q \neq 1$ 时）

$$\frac{S_{2n} - S_n}{S_n} = \frac{S_{3n} - S_{2n}}{S_{2n} - S_n} = q^n$$

### 无穷等比级数（$|q| < 1$）

当 $|q| < 1$ 时，$q^n \to 0$（$n \to \infty$），无穷项之和：

$$S = \frac{a_1}{1-q} \quad (|q| < 1)$$

**应用**：循环小数化分数——$0.\overline{3} = 0.3 + 0.03 + \cdots = \dfrac{0.3}{1-0.1} = \dfrac{1}{3}$

### 典型例题

**例1**：$a_1 = 1$，$q = 2$，求 $S_{10}$。

**解**：$S_{10} = \dfrac{1 \times (2^{10}-1)}{2-1} = 2^{10} - 1 = 1023$

**例2**：$a_1 = 8$，$q = \dfrac{1}{2}$，求 $S_n$ 趋近的极限。

**解**：$|q| = \dfrac{1}{2} < 1$，$S = \dfrac{8}{1-\frac{1}{2}} = 16$

**例3**：等比数列 $a_1 = 3$，$S_3 = 21$，求公比 $q$。

**解**：$S_3 = 3(1 + q + q^2) = 21$，$1 + q + q^2 = 7$，$q^2 + q - 6 = 0$，$(q+3)(q-2) = 0$，$q = 2$ 或 $q = -3$

**例4**：已知 $S_n = 3^n - 1$，求 $a_n$。

**解**：$n \geq 2$：$a_n = S_n - S_{n-1} = (3^n-1) - (3^{n-1}-1) = 3^n - 3^{n-1} = 2 \times 3^{n-1}$

$n = 1$：$a_1 = S_1 = 2 = 2 \times 3^0$ ✓，故 $a_n = 2 \times 3^{n-1}$（等比，$q = 3$）

### 易错点

- $q = 1$ 时必须单独讨论（$q \neq 1$ 的公式在 $q=1$ 时分母为 0）
- $S_n - S_{n-1} = a_n$ 仅对 $n \geq 2$ 成立，$n=1$ 时要单独验证
- 错位相减后要检验 $q \neq 1$ 的前提
