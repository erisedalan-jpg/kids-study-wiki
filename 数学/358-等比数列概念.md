## 🎓 等比数列概念

### 定义

如果一个数列 $\{a_n\}$ 从第二项起，每一项与前一项的**比**都等于同一常数 $q$（$q \neq 0$），则称这个数列为**等比数列**（geometric sequence），常数 $q$ 称为**公比**（common ratio）。

$$\frac{a_n}{a_{n-1}} = q \quad (n \geq 2,\ q \neq 0,\ a_n \neq 0)$$

### 通项公式

$$a_n = a_1 \cdot q^{n-1}$$

**变形**：$a_n = a_m \cdot q^{n-m}$（以任意项 $a_m$ 为基准）

### 等比数列的性质

**性质1（等比中项）**：若 $G^2 = ab$（$ab > 0$），则 $a, G, b$ 成等比，$G$ 称为 $a$ 与 $b$ 的**等比中项**。

**性质2**：若 $m + n = p + q$，则 $a_m \cdot a_n = a_p \cdot a_q$。

**性质3**：等比数列每一项（不为零）是其相邻项的**几何平均数**：

$$a_n^2 = a_{n-1} \cdot a_{n+1}$$

**性质4（指数性质）**：以 $n$ 为横轴，$\lg a_n$ 为纵轴，各点在一条**直线**上（等比数列取对数变等差）。

### 等比数列的单调性

- $a_1 > 0$，$q > 1$：递增
- $a_1 > 0$，$0 < q < 1$：递减
- $a_1 < 0$，$q > 1$：递减（各项均为负）
- $q < 0$：各项正负交替（非单调）

### 典型例题

**例1**：等比数列 $a_1 = 2$，$q = 3$，求 $a_5$。

**解**：$a_5 = 2 \times 3^4 = 2 \times 81 = 162$

**例2**：等比数列中，$a_2 = 6$，$a_5 = 48$，求公比 $q$。

**解**：$\dfrac{a_5}{a_2} = q^3 = 8$，故 $q = 2$

**例3**：$a, b, c$ 成等比数列，$b^2 = ac = 9$，$a + c = 6$，求 $a, c$。

**解**：$a + c = 6$，$ac = 9$，由韦达定理，$a, c$ 是 $x^2 - 6x + 9 = 0$ 的根，$(x-3)^2 = 0$，故 $a = c = 3$。

**例4**：细菌每 20 分钟分裂一次，初始 100 个，2 小时后多少个？

**解**：$q = 2$，$n = 6$（分裂 6 次），数量 $= 100 \times 2^6 = 6400$ 个

### 与等差数列的对比

| | 等差数列 | 等比数列 |
|--|---------|---------|
| 递推关系 | $a_n - a_{n-1} = d$ | $a_n / a_{n-1} = q$ |
| 通项公式 | $a_1 + (n-1)d$ | $a_1 \cdot q^{n-1}$ |
| 中间项 | 算术平均数 | 几何平均数 |
| 图像 | 直线 | 指数曲线 |
| 求和 | $\frac{n(a_1+a_n)}{2}$ | 见[[等比数列前n项和]] |

### 易错点

- 公比 $q \neq 0$，且各项 $a_n \neq 0$
- $q < 0$ 时数列正负交替，不是单调的
- 等比中项：$G^2 = ab$，注意 $G$ 可以是正值也可以是负值（$G = \pm\sqrt{ab}$，$ab > 0$）
