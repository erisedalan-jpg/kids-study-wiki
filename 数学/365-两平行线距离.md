## 🎓 两平行线距离

### 公式

若两平行线方程为：
$$l_1: Ax + By + C_1 = 0 \quad \text{和} \quad l_2: Ax + By + C_2 = 0$$

（$A, B$ 系数**完全相同**，$C_1 \neq C_2$），则距离为：

$$d = \dfrac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$$

### 推导

在 $l_1$ 上取任意一点（例如令 $y=0$，得 $x=-\dfrac{C_1}{A}$，点为 $\left(-\dfrac{C_1}{A},0\right)$），求其到 $l_2$ 的距离：

$$d = \dfrac{\left|A\left(-\dfrac{C_1}{A}\right)+B\cdot0+C_2\right|}{\sqrt{A^2+B^2}} = \dfrac{|-C_1+C_2|}{\sqrt{A^2+B^2}} = \dfrac{|C_1-C_2|}{\sqrt{A^2+B^2}}$$

### 注意：系数需统一

使用公式前，需确保两直线方程中 $x$ 和 $y$ 的系数**完全相同**（可能需要先化简）。

**例**：$l_1: 2x-y+3=0$，$l_2: 4x-2y+1=0$

先将 $l_2$ 化为系数与 $l_1$ 相同的形式：$l_2 \div 2$：$2x-y+\dfrac{1}{2}=0$

$$d = \dfrac{\left|3-\dfrac{1}{2}\right|}{\sqrt{4+1}} = \dfrac{\dfrac{5}{2}}{\sqrt{5}} = \dfrac{5}{2\sqrt{5}} = \dfrac{\sqrt{5}}{2}$$

### 典型例题

**例1**：两平行线 $l_1: x+2y-1=0$，$l_2: x+2y+4=0$，求距离。

**解**：$d=\dfrac{|-1-4|}{\sqrt{1+4}}=\dfrac{5}{\sqrt{5}}=\sqrt{5}$

**例2**：直线 $l: 3x-4y+6=0$，求与 $l$ 平行且距离为 $2$ 的直线方程。

**解**：设平行线为 $3x-4y+C=0$，距离 $d=\dfrac{|6-C|}{\sqrt{9+16}}=\dfrac{|6-C|}{5}=2$

$|6-C|=10$，故 $C=16$ 或 $C=-4$

方程为：$3x-4y+16=0$ 或 $3x-4y-4=0$

**例3**：等腰梯形两底所在直线 $y=2$ 和 $y=-1$，求两底距离。

**解**：化为一般式：$y-2=0$ 和 $y+1=0$，即 $0\cdot x+y-2=0$ 和 $0\cdot x+y+1=0$

$d=\dfrac{|-2-1|}{\sqrt{0+1}}=3$

### 联系：点到直线距离的特殊情形

两平行线距离 = $l_1$ 上任一点到 $l_2$ 的距离（因为所有点到平行线的距离相同）

### 易错点

- 使用公式前必须统一系数（$A,B$ 完全相同）
- $|C_1-C_2|$ 不是 $C_1-C_2$，要取绝对值
- 求"与已知直线平行且距离为 $d$ 的直线"有两条，不要漏解
