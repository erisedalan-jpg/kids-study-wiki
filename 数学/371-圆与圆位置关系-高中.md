## 🎓 圆与圆的位置关系（高中）

### 判断依据

设两圆：$C_1$（圆心 $(a_1,b_1)$，半径 $r_1$），$C_2$（圆心 $(a_2,b_2)$，半径 $r_2$）。

圆心距：$d = |C_1C_2| = \sqrt{(a_1-a_2)^2+(b_1-b_2)^2}$

| 位置关系 | 距离条件 | 公切线数 |
|---------|---------|---------|
| 外离 | $d > r_1 + r_2$ | 4 条 |
| 外切 | $d = r_1 + r_2$ | 3 条 |
| 相交 | $|r_1-r_2| < d < r_1+r_2$ | 2 条 |
| 内切 | $d = |r_1-r_2|$（且 $r_1\ne r_2$）| 1 条 |
| 内含 | $d < |r_1-r_2|$（且 $r_1\ne r_2$）| 0 条 |
| 同心圆 | $d=0, r_1\ne r_2$ | 0 条 |

（$r_1 \ge r_2$ 无妨）

### 公切线

**外公切线**：与两圆都相切，且两圆在切线同侧。

**内公切线**：与两圆都相切，且两圆分居切线两侧（只有外离和外切时存在）。

| 位置 | 外公切线 | 内公切线 |
|------|---------|---------|
| 外离 | 2 | 2 |
| 外切 | 2 | 1（过切点）|
| 相交 | 2 | 0 |
| 内切 | 1（过切点）| 0 |
| 内含/同心 | 0 | 0 |

### 两圆相交的弦（公共弦）

两圆相交时，连线（公共弦）的直线方程：

设 $C_1: x^2+y^2+D_1x+E_1y+F_1=0$，$C_2: x^2+y^2+D_2x+E_2y+F_2=0$

两方程相减（消去 $x^2+y^2$）：

$$(D_1-D_2)x + (E_1-E_2)y + (F_1-F_2) = 0$$

此即**公共弦所在直线**（两圆的"根轴"）。

### 典型例题

**例1**：圆 $C_1: x^2+y^2=9$，圆 $C_2: (x-4)^2+y^2=4$，判断位置关系。

**解**：$r_1=3, r_2=2$，$d=|C_1C_2|=4$，$r_1+r_2=5, |r_1-r_2|=1$

$1 < 4 < 5$，**相交**。

**例2**：圆 $C_1: x^2+y^2=1$，$C_2: x^2+y^2-6x-8y+20=0$，求 $d$。

**解**：$C_2$ 圆心 $(3,4)$，$r_2=\sqrt{9+16-20}=\sqrt{5}$（$r_2^2=9+16-20=5$，待核对）

$d=\sqrt{9+16}=5$，$r_1=1, r_2=\sqrt{5}\approx2.24$，$r_1+r_2\approx3.24<5$，**外离**。

**例3**：求两圆 $x^2+y^2=4$ 与 $x^2+y^2-2x-2y-6=0$ 的公共弦方程。

**解**：两方程相减：$(0-(-2))x+(0-(-2))y+(0-(-6))=0$，即 $2x+2y+6=0$，化简 $x+y+3=0$

（需验证两圆确实相交，待家长核对圆心距）

**例4**：已知两圆外切，$C_1$ 圆心 $(0,0)$，$C_2$ 圆心 $(5,0)$，$r_1=2$，求 $r_2$。

**解**：外切 $\Rightarrow d=r_1+r_2$，$5=2+r_2$，$r_2=3$

### 与初中的联系

初中学过两圆位置关系的定性判断；高中扩展到：
- 利用方程精确求圆心距
- 求公共弦方程（根轴）
- 联系解析几何综合题

### 易错点

- 内切条件 $d=|r_1-r_2|$（绝对值，不论哪个圆大）
- 公共弦直线方程：两圆方程相减（不是相加）
- 将圆方程化标准形式后才能读出圆心和半径
