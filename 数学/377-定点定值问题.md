## 🎓 定点定值问题

### 什么是定点/定值问题

**定点问题**：在某一条件下，证明直线（或点）恒过某一固定点，与参数无关。

**定值问题**：在某一条件下，证明某一表达式（长度、面积、比值等）为常数，与参数无关。

这类问题是高考解析几何的核心难题类型。

### 定点问题的解题思路

#### 策略一：参数消去法

**步骤**：
1. 设直线含参数（如斜率 $k$）
2. 写出与曲线的交点方程
3. 整理得关于 $x$（或 $y$）含 $k$ 的等式
4. 令等式对**所有 $k$** 成立，消去 $k$，得定点坐标

**模板**：直线 $y = kx + m$（$m$ 固定），代入曲线方程，整理为：

$$A(k)x + B(k) = 0 \quad \text{对所有}\ k\ \text{成立}$$

分离 $k$：$x[\text{含}k\text{项系数}] + [\text{常数项}] = 0$，令各系数为零，解出定点。

#### 策略二：特殊值法（验证）

取 $k$ 的两个特殊值，求出对应直线与曲线的交点，两次结果的公共点即为定点。（只能验证，严格证明需参数消去法）

#### 策略三：构造法

将条件改写为某点始终在某直线/曲线上的形式。

### 定值问题的解题思路

**常见定值目标**：面积、比值、乘积、和、差。

**策略**：用韦达定理表示对称函数（$x_1+x_2$，$x_1 x_2$），再计算目标量，化简后得常数。

### 典型例题

**例1（定点）**：椭圆 $\dfrac{x^2}{4}+y^2=1$，过点 $A(0,1)$ 和椭圆上动点 $B$ 的直线交 $x$ 轴于点 $M$，求 $M$ 的范围/轨迹。

（若要问定点：过椭圆上两动点 $B,C$ 且满足某条件的弦，其延长线恒过固定点，才是定点类型）

**例2（定点）**：椭圆 $\dfrac{x^2}{4}+\dfrac{y^2}{3}=1$，弦 $AB$ 的中点在 $y$ 轴上（$x=0$），过 $A,B$ 的直线斜率为 $k$，证明直线恒过定点。

**解**：设 $A(x_1,y_1), B(x_2,y_2)$ 在椭圆上，代入相减（点差法）：

$\dfrac{x_1^2-x_2^2}{4}+\dfrac{y_1^2-y_2^2}{3}=0$，设中点 $x_0=0$，$x_1+x_2=0$（关于 $y$ 轴对称）。

$A(-t,y_A), B(t,y_B)$（$t>0$），代入椭圆：$y_A^2=3(1-t^2/4)=y_B^2$，故 $y_A=y_B$（或 $y_A=-y_B$）。若 $y_A=y_B$，$A=B$（退化）；故 $y_A=-y_B$，$AB$ 为 $y$ 轴平行的弦，不过定点（斜率不存在）。

（本例说明需具体题目才能展示定点，请家长补充完整题目核对）

**例3（定值）**：椭圆 $\dfrac{x^2}{4}+y^2=1$，过 $F(1,0)$ 的弦 $AB$，证明 $\dfrac{1}{|AF|}+\dfrac{1}{|BF|}$ 为定值。

**解**：设 $A(x_1,y_1), B(x_2,y_2)$，$|AF|=a-ex_1=2-\dfrac{1}{2}x_1$（$a=2,e=\dfrac{1}{2}$），$|BF|=2-\dfrac{1}{2}x_2$。

$\dfrac{1}{|AF|}+\dfrac{1}{|BF|}=\dfrac{1}{2-x_1/2}+\dfrac{1}{2-x_2/2}=\dfrac{2}{4-x_1}+\dfrac{2}{4-x_2}$

$=2\cdot\dfrac{(4-x_1)+(4-x_2)}{(4-x_1)(4-x_2)}=2\cdot\dfrac{8-(x_1+x_2)}{16-4(x_1+x_2)+x_1x_2}$

利用韦达定理（直线方程代入椭圆），$x_1+x_2, x_1 x_2$ 用斜率 $k$ 表达后，化简发现结果与 $k$ 无关（待家长核对细节），为定值 $\dfrac{2a}{b^2}=\dfrac{4}{3}$（椭圆焦点弦调和关系）。

### 常见结论（定值速查）

| 情形 | 定值 |
|------|------|
| 椭圆焦点弦 $\dfrac{1}{|AF|}+\dfrac{1}{|BF|}$ | $\dfrac{2a}{b^2}$ |
| 抛物线焦点弦 $\dfrac{1}{|AF|}+\dfrac{1}{|BF|}$ | $\dfrac{2}{p}$ |
| 椭圆上切线截面积（特定类型）| 与 $e$ 有关，具体题目具体分析 |

### 易错点

- 定点问题中，要说明对**所有合法参数**成立
- 特殊值法只能"猜测"定点，需补完整证明
- 韦达定理仅在交点存在（$\Delta>0$）时有效，需验证
