## 🎓 数列错位相减法

### 适用场景

**错位相减法**：用于求形如 $\sum_{k=1}^n a_k \cdot b_k$ 的数列前 $n$ 项和，其中 $\{a_k\}$ 是等差数列，$\{b_k\}$ 是等比数列（或反过来也可适用）。

典型例子：$S_n = 1\cdot2 + 2\cdot2^2 + 3\cdot2^3 + \cdots + n\cdot2^n = \sum_{k=1}^n k\cdot2^k$

### 推导过程（以 $S_n = \sum_{k=1}^n k q^k$ 为例）

**第一步**：写出 $S_n$：

$$S_n = 1\cdot q + 2\cdot q^2 + 3\cdot q^3 + \cdots + n\cdot q^n$$

**第二步**：错位（乘以公比 $q$）：

$$q\cdot S_n = 1\cdot q^2 + 2\cdot q^3 + 3\cdot q^4 + \cdots + n\cdot q^{n+1}$$

**第三步**：相减（$S_n - q\cdot S_n$）：

$$S_n(1-q) = q + q^2 + q^3 + \cdots + q^n - n q^{n+1}$$

$$= \frac{q(1-q^n)}{1-q} - nq^{n+1} \quad (q \ne 1)$$

**第四步**：解出 $S_n$（除以 $1-q$）：

$$S_n = \frac{q(1-q^n)}{(1-q)^2} - \frac{nq^{n+1}}{1-q} \quad (q \ne 1)$$

### 标准模板

| 数列类型 | 求和方法 |
|---------|---------|
| 等差 $\sum a_k$ | 首尾相加法 |
| 等比 $\sum b_k$ | 等比求和公式 |
| 等差 × 等比 $\sum a_k b_k$ | **错位相减法** |
| 分母为等差（裂项型）| 裂项相消法 |

### 典型例题

**例1**：求 $S_n = 1\cdot2 + 2\cdot2^2 + 3\cdot2^3 + \cdots + n\cdot2^n$

**解**：

$$S_n = \sum_{k=1}^n k\cdot2^k$$

$$2S_n = \sum_{k=1}^n k\cdot2^{k+1} = 1\cdot2^2+2\cdot2^3+\cdots+(n-1)\cdot2^n + n\cdot2^{n+1}$$

$$S_n - 2S_n = (1\cdot2+1\cdot2^2+1\cdot2^3+\cdots+1\cdot2^n) - n\cdot2^{n+1}$$

$$-S_n = \frac{2(1-2^n)}{1-2} - n\cdot2^{n+1} = 2^{n+1}-2 - n\cdot2^{n+1} = (1-n)2^{n+1}-2$$

$$S_n = (n-1)2^{n+1}+2$$

**验证**：$n=1$：$S_1=1\times2=2$，代入 $(0)\times4+2=2$ ✓

$n=2$：$S_2=2+8=10$，代入 $(1)\times8+2=10$ ✓

**例2**：求 $T_n = \sum_{k=1}^n (2k-1)\cdot3^k$

**解**：令 $S_n = T_n$，错位：$3S_n$，相减：

$$T_n - 3T_n = \sum_{k=1}^n(2k-1)3^k - \sum_{k=1}^n(2k-1)3^{k+1}$$

$$-2T_n = 1\cdot3 + \sum_{k=2}^n[(2k-1)-(2k-3)]\cdot3^k - (2n-1)\cdot3^{n+1}$$

$$= 3 + 2\sum_{k=2}^n 3^k - (2n-1)3^{n+1} = 3 + 2\cdot\frac{3^2(1-3^{n-1})}{1-3} - (2n-1)3^{n+1}$$

$$= 3 + 9\cdot\frac{1-3^{n-1}}{-2}\times2... $$

（计算较长，建议家长核对完整推导）

### 易错点

- 错位相减后要检查**首尾项**是否配对，避免漏项或多算
- 特殊情形 $q=1$：数列为 $\sum_{k=1}^n k = \dfrac{n(n+1)}{2}$（等差，不用错位法）
- 最后别忘除以 $1-q$（注意 $q \ne 1$ 的条件）
