## 🎓 椭圆光学性质

### 光学性质（反射定律）

**椭圆光学性质**：从椭圆一个焦点 $F_1$ 发出的光线，经椭圆内壁反射后，必过另一个焦点 $F_2$。

**数学表述**：椭圆上任意点 $P$ 处的切线，是 $\angle F_1PF_2$ 的角平分线；法线平分 $\angle F_1PF_2$ 的外角（待家长核对描述方向）。

### 反射原理推导（直觉）

光的反射定律：入射角 = 反射角（相对于法线）。

椭圆上点 $P$ 处，$F_1P$ 与 $F_2P$ 关于该点切线的夹角相等（即 $PF_1$ 与 $PF_2$ 关于切线对称），这由椭圆的极值性质（$|PF_1|+|PF_2|=$ 极小值问题）可以推出。

**费马原理视角**：从 $F_1$ 出发经椭圆上点 $P$ 到 $F_2$ 的路程 $|PF_1|+|PF_2|=2a$（常数），光沿使路程稳定的路径传播，即椭圆上各点反射均能汇聚到 $F_2$。

### 应用场景

| 场景 | 说明 |
|------|------|
| 椭圆形台球桌 | 从一焦点击出的球必经另一焦点 |
| 碎石术（医学） | 体外冲击波聚焦到肾结石处 |
| 椭圆形音乐厅 | 低语者说的话在另一焦点处被清晰听到 |
| 光学仪器 | 椭圆反射镜汇聚光线 |

### 典型例题

**例1**：椭圆形镜面，两焦点 $F_1, F_2$，从 $F_1$ 发出一束光线打到椭圆上点 $P(3,0)$，已知椭圆方程为 $\dfrac{x^2}{25}+\dfrac{y^2}{16}=1$，问反射光线过哪里？

**解**：由椭圆光学性质，反射光线必过 $F_2$。$c=\sqrt{25-16}=3$，$F_2=(3,0)$。

注意：$P(3,0)$ 恰好是焦点 $F_2$，这说明题目数据需调整（点不能在焦点上）。若题目正常给出椭圆上非焦点处的 $P$，答案直接为"反射光线过另一焦点 $F_2$"。

**例2**（应用型）：椭圆形手术室，$F_1$ 处放震源，$F_2$ 处正好是患者结石位置。已知椭圆 $\dfrac{x^2}{100}+\dfrac{y^2}{75}=1$（单位：cm），求焦距。

**解**：$c=\sqrt{100-75}=5$ cm，焦距 $2c=10$ cm。

### 与双曲线光学性质的对比

| 曲线 | 光学性质 |
|------|---------|
| 椭圆 | 从 $F_1$ 射出 → 反射后过 $F_2$ |
| 双曲线 | 朝 $F_1$ 射去 → 反射后背向 $F_2$（延长线过 $F_2$）|
| 抛物线 | 从焦点射出 → 反射后平行于轴 |

### 易错点

- 光学性质是**椭圆内壁**反射，不是外壁
- 反射后经过的是另一个焦点，不是同一个焦点
- 该性质对椭圆上**所有点**成立
