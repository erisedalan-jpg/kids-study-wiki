## 🎓 椭圆几何性质

以标准椭圆 $\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1$（$a>b>0$，焦点在 $x$ 轴）为例。

### 对称性

- 关于 $x$ 轴对称
- 关于 $y$ 轴对称
- 关于原点（中心）对称

### 顶点

| 顶点 | 坐标 | 说明 |
|------|------|------|
| 长轴右顶点 $A_2$ | $(a,0)$ | |
| 长轴左顶点 $A_1$ | $(-a,0)$ | |
| 短轴上顶点 $B_2$ | $(0,b)$ | |
| 短轴下顶点 $B_1$ | $(0,-b)$ | |

### 范围

$$-a \le x \le a, \quad -b \le y \le b$$

### 焦半径公式

椭圆上点 $P(x_0,y_0)$，两焦点 $F_1(-c,0)$，$F_2(c,0)$：

$$|PF_1| = a + ex_0, \quad |PF_2| = a - ex_0$$

其中 $e=\dfrac{c}{a}$ 为离心率。

**验证**：$|PF_1|+|PF_2| = 2a$ ✓（椭圆定义）；$|PF_1|-|PF_2|=2ex_0$。

**范围**：$a-c \le |PF_i| \le a+c$（最近为 $a-c$，最远为 $a+c$）

### 焦点三角形

椭圆上点 $P$，两焦点 $F_1,F_2$ 构成 $\triangle PF_1F_2$：

- $|PF_1|+|PF_2|=2a$（周长 $= 2a+2c$）
- $\angle F_1PF_2$ 用余弦定理：$|F_1F_2|^2 = |PF_1|^2+|PF_2|^2-2|PF_1||PF_2|\cos\angle F_1PF_2$
- **面积**：$S = \dfrac{1}{2}|PF_1||PF_2|\sin\angle F_1PF_2 = b^2\tan\dfrac{\angle F_1PF_2}{2}$（待家长核对）

### 典型例题

**例1**：椭圆 $\dfrac{x^2}{16}+\dfrac{y^2}{9}=1$，点 $P$ 在椭圆上，$|PF_1|=3$，求 $|PF_2|$。

**解**：$|PF_1|+|PF_2|=2a=8$，故 $|PF_2|=5$。

**例2**：椭圆 $\dfrac{x^2}{25}+\dfrac{y^2}{16}=1$，求短轴端点到焦点的距离。

**解**：$a=5, b=4, c=3$，短轴端点 $(0,4)$ 到焦点 $(\pm3,0)$：

$d=\sqrt{9+16}=\sqrt{25}=5=a$ ✓（短轴端点到焦点距离等于长半轴 $a$）

**例3**：椭圆焦点三角形中，$|PF_1|=3, |PF_2|=5, |F_1F_2|=4$，求椭圆方程（焦点在 $x$ 轴）。

**解**：$2a=3+5=8$，$a=4$；$2c=4$，$c=2$；$b^2=a^2-c^2=16-4=12$

方程：$\dfrac{x^2}{16}+\dfrac{y^2}{12}=1$

### 易错点

- 焦半径公式 $|PF_2|=a-ex_0$ 中，$x_0$ 是点 $P$ 的横坐标（可以为负），且 $|PF_i|\ge a-c>0$
- 短轴端点到焦点的距离等于 $a$（长半轴），这是重要结论
- 焦点三角形面积公式有多种，高考中常用余弦定理+面积公式结合
