## 🎓 空间向量数量积

### 定义

设两非零空间向量 $\boldsymbol{a}$，$\boldsymbol{b}$，它们夹角为 $\theta$（$0° \le \theta \le 180°$），则：

$$\boldsymbol{a} \cdot \boldsymbol{b} = |\boldsymbol{a}||\boldsymbol{b}|\cos\theta$$

若 $\boldsymbol{a}$ 或 $\boldsymbol{b}$ 为零向量，则 $\boldsymbol{a} \cdot \boldsymbol{b} = 0$。

数量积（内积、点积）的结果是**实数**，不是向量。

### 坐标计算公式

设 $\boldsymbol{a}=(a_1,a_2,a_3)$，$\boldsymbol{b}=(b_1,b_2,b_3)$：

$$\boldsymbol{a} \cdot \boldsymbol{b} = a_1b_1 + a_2b_2 + a_3b_3$$

### 由数量积求夹角

$$\cos\theta = \dfrac{\boldsymbol{a}\cdot\boldsymbol{b}}{|\boldsymbol{a}||\boldsymbol{b}|} = \dfrac{a_1b_1+a_2b_2+a_3b_3}{\sqrt{a_1^2+a_2^2+a_3^2}\cdot\sqrt{b_1^2+b_2^2+b_3^2}}$$

### 垂直条件

$$\boldsymbol{a} \perp \boldsymbol{b} \Leftrightarrow \boldsymbol{a}\cdot\boldsymbol{b} = 0 \Leftrightarrow a_1b_1+a_2b_2+a_3b_3 = 0$$

### 运算律

| 律名 | 表达式 |
|------|--------|
| 交换律 | $\boldsymbol{a}\cdot\boldsymbol{b}=\boldsymbol{b}\cdot\boldsymbol{a}$ |
| 分配律 | $(\boldsymbol{a}+\boldsymbol{b})\cdot\boldsymbol{c}=\boldsymbol{a}\cdot\boldsymbol{c}+\boldsymbol{b}\cdot\boldsymbol{c}$ |
| 数乘结合 | $(\lambda\boldsymbol{a})\cdot\boldsymbol{b}=\lambda(\boldsymbol{a}\cdot\boldsymbol{b})$ |
| 自乘 | $\boldsymbol{a}\cdot\boldsymbol{a}=|\boldsymbol{a}|^2 \ge 0$ |

**注意**：$(\boldsymbol{a}\cdot\boldsymbol{b})\cdot\boldsymbol{c}$ 无意义（数乘向量，不是数量积）；数量积**不满足结合律**。

### 典型例题

**例1**：$\boldsymbol{a}=(1,2,-1)$，$\boldsymbol{b}=(3,-1,2)$，求 $\boldsymbol{a}\cdot\boldsymbol{b}$ 及两向量夹角。

**解**：
$$\boldsymbol{a}\cdot\boldsymbol{b} = 1\times3+2\times(-1)+(-1)\times2 = 3-2-2=-1$$
$$|\boldsymbol{a}|=\sqrt{1+4+1}=\sqrt{6}, \quad |\boldsymbol{b}|=\sqrt{9+1+4}=\sqrt{14}$$
$$\cos\theta = \dfrac{-1}{\sqrt{6}\cdot\sqrt{14}} = \dfrac{-1}{\sqrt{84}} = -\dfrac{1}{2\sqrt{21}}$$

**例2**：验证 $\boldsymbol{a}=(2,1,-1)$，$\boldsymbol{b}=(1,-3,-1)$ 是否垂直。

**解**：$\boldsymbol{a}\cdot\boldsymbol{b}=2-3+1=0$，故 $\boldsymbol{a}\perp\boldsymbol{b}$。

**例3**：已知 $|\boldsymbol{a}|=2$，$|\boldsymbol{b}|=3$，$\boldsymbol{a}\cdot\boldsymbol{b}=3$，求 $|\boldsymbol{a}+\boldsymbol{b}|$。

**解**：
$$|\boldsymbol{a}+\boldsymbol{b}|^2 = |\boldsymbol{a}|^2 + 2\boldsymbol{a}\cdot\boldsymbol{b} + |\boldsymbol{b}|^2 = 4+6+9=19$$
$$|\boldsymbol{a}+\boldsymbol{b}| = \sqrt{19}$$

### 易错点

- 数量积结果是**数**，不是向量；不能对数量积再做数量积
- $\boldsymbol{a}\cdot\boldsymbol{b}=0$ 只说明 $\boldsymbol{a}\perp\boldsymbol{b}$，若 $\boldsymbol{a}$ 或 $\boldsymbol{b}$ 为零向量，方向无定义，也满足 $\boldsymbol{a}\cdot\boldsymbol{b}=0$
- 夹角范围 $[0°, 180°]$，$\cos\theta$ 可以为负（夹角为钝角）
