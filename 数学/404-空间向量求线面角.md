## 🎓 空间向量求线面角

### 定义回顾

**线面角**：直线 $l$ 与平面 $\alpha$ 所成角 $\varphi$（$0° \le \varphi \le 90°$）。

- $l \subset \alpha$ 或 $l \parallel \alpha$：$\varphi = 0°$
- $l \perp \alpha$：$\varphi = 90°$
- 其他情况：$\varphi$ 为 $l$ 与其在 $\alpha$ 上射影所成锐角

### 向量法公式

设直线 $l$ 的方向向量为 $\boldsymbol{v}$，平面 $\alpha$ 的法向量为 $\boldsymbol{n}$，则：

$$\sin\varphi = \dfrac{|\boldsymbol{v}\cdot\boldsymbol{n}|}{|\boldsymbol{v}||\boldsymbol{n}|}$$

**理解**：$\boldsymbol{v}$ 与 $\boldsymbol{n}$ 的夹角为 $\theta$，则线面角 $\varphi = |90° - \theta|$（互余关系），故 $\sin\varphi = |\cos\theta|$。

### 步骤

1. 建立空间直角坐标系
2. 求直线的方向向量 $\boldsymbol{v}$（取直线上两点坐标差）
3. 求平面的法向量 $\boldsymbol{n}$（取面内两不共线向量，解 $\boldsymbol{n}\cdot\boldsymbol{a}=\boldsymbol{n}\cdot\boldsymbol{b}=0$）
4. 代入公式，得 $\sin\varphi$，再求 $\varphi$

### 典型例题

**例1**：正方体 $ABCD$-$A_1B_1C_1D_1$，棱长为 1，以 $A$ 为原点（$AB$ 为 $x$ 轴，$AD$ 为 $y$ 轴，$AA_1$ 为 $z$ 轴），求对角线 $A_1C$ 与底面 $ABCD$ 所成角。

**解**：
坐标：$A_1(0,0,1)$，$C(1,1,0)$，故 $\overrightarrow{A_1C}=(1,1,-1)$，$\boldsymbol{v}=(1,1,-1)$

底面 $ABCD$ 的法向量 $\boldsymbol{n}=(0,0,1)$

$$\sin\varphi = \dfrac{|(1,1,-1)\cdot(0,0,1)|}{|(1,1,-1)|\cdot|(0,0,1)|} = \dfrac{|-1|}{\sqrt{3}\cdot1} = \dfrac{1}{\sqrt{3}} = \dfrac{\sqrt{3}}{3}$$

$$\varphi = \arcsin\dfrac{\sqrt{3}}{3} \approx 35.3°$$

**例2**：四棱锥 $P$-$ABCD$，底面为正方形，边长为 2，$P$ 在底面上的投影为底面中心 $O$，$PO=2$，求 $PA$ 与底面所成角。

**解**（直接法）：$PA$ 在底面射影为 $OA$，$|OA|=\sqrt{2}$，$|PO|=2$，故：
$$\tan\varphi = \dfrac{PO}{OA} = \dfrac{2}{\sqrt{2}} = \sqrt{2}, \quad \varphi = \arctan\sqrt{2} \approx 54.7°$$

向量法验证（建坐标后计算结果相同）。

**例3**：三棱锥中，已知直线方向向量 $\boldsymbol{v}=(1,2,2)$，平面法向量 $\boldsymbol{n}=(2,1,-2)$，求线面角。

**解**：
$$\sin\varphi = \dfrac{|2+2-4|}{3\cdot3} = \dfrac{0}{9} = 0, \quad \varphi = 0°$$

（方向向量与法向量垂直，说明直线与平面平行，线面角为 $0°$）

### 易错点

- 线面角用 $\sin$ 公式（而非 $\cos$），因为方向向量和法向量夹角与线面角**互余**
- 绝对值不可省略（保证 $\sin\varphi \ge 0$）
- 若 $\boldsymbol{v}\cdot\boldsymbol{n}=0$，线面角为 $0°$（平行）；若 $\boldsymbol{v}\parallel\boldsymbol{n}$，线面角为 $90°$（垂直）
