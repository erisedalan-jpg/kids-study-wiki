## 🎓 空间向量综合应用

### 空间向量核心工具回顾

| 工具 | 公式 |
|------|------|
| 向量夹角 | $\cos\theta = \dfrac{\vec{a}\cdot\vec{b}}{|\vec{a}||\vec{b}|}$ |
| 法向量 | 满足 $\vec{n}\perp \vec{v}$ 对平面内所有向量 $\vec{v}$，即 $\vec{n}\cdot\vec{v}=0$ |
| 点到平面距离 | $d = \dfrac{|\vec{n}\cdot\overrightarrow{AP}|}{|\vec{n}|}$（$A$ 在平面上，$\vec{n}$ 为法向量）|
| 线面角 | $\sin\theta = \dfrac{|\vec{m}\cdot\vec{n}|}{|\vec{m}||\vec{n}|}$（$\vec{m}$ 方向向量，$\vec{n}$ 法向量）|
| 二面角 | $\cos\theta = \dfrac{\vec{n_1}\cdot\vec{n_2}}{|\vec{n_1}||\vec{n_2}|}$（或其补角，取决于法向量指向）|

### 综合解题框架

**建立空间直角坐标系的原则**：
1. 选取已知角度为直角的顶点作原点
2. 三条棱方向作 $x, y, z$ 轴
3. 求出所有顶点坐标

**常见立体图形的坐标建立**：

| 图形 | 建系建议 |
|------|---------|
| 正方体（棱长 $a$）| 以一顶点为原点，三棱为坐标轴 |
| 直三棱柱 | 以底面顶点为原点，两底边为 $x,y$，高为 $z$ |
| 正四棱锥 | 以底面中心为原点，两对角线为 $x,y$，高为 $z$ |
| 正三棱锥 | 以底面某顶点或中心为原点（视情况选择）|

### 典型综合例题

**例1（二面角）**：正方体 $ABCD-A'B'C'D'$（棱长 1），求二面角 $A'-AB-B$ 的大小。

**解**：建系：$A$ 为原点，$AB$ 方向为 $x$，$AD$ 方向为 $y$，$AA'$ 方向为 $z$。

各点坐标：$A(0,0,0), B(1,0,0), A'(0,0,1), D(0,1,0)$

面 $A'AB$（$A', A, B$ 三点确定）：法向量 $\vec{n_1} = \overrightarrow{AB}\times\overrightarrow{AA'} = (1,0,0)\times(0,0,1)$

叉积（待补）= $(0\cdot1-0\cdot0, 0\cdot0-1\cdot1, 1\cdot0-0\cdot0) = (0,-1,0)$，$\vec{n_1}=(0,-1,0)$

面 $ABB'$（$A,B,B'$）：类似求法向量（待家长核对）

**例2（点到平面距离）**：正三棱锥 $P-ABC$，底面边长 2，高为 $\sqrt{2}$，求顶点 $P$ 到侧面 $PAB$ 的距离（即体积法验证）。

**体积法**：$V = \dfrac{1}{3}S_{ABC}\cdot h = \dfrac{1}{3}\times\sqrt{3}\times\sqrt{2}=\dfrac{\sqrt{6}}{3}$

设 $P$ 到侧面 $PAB$ 距离 $d$（此例不合理，$P$ 在面 $PAB$ 上），改为求 $C$ 到面 $PAB$ 的距离：

$V = \dfrac{1}{3}S_{PAB}\cdot d_{C,PAB}$，故 $d = \dfrac{3V}{S_{PAB}}$（需具体计算 $S_{PAB}$）

**例3（线面角）**：正四棱锥 $P-ABCD$，底面边长 2，侧棱长 2，求斜面对角线与底面所成角。

**解**：建系：$O$（底面中心）为原点，$OA$ 方向为 $x$，$OB$ 方向为 $y$，$OP$ 方向为 $z$。

$A(1,0,0), B(0,1,0), P(0,0, h)$（求 $h$：$|PA|^2=1+0+h^2=4$，$h=\sqrt{3}$）

斜面对角线，如 $AC$ 的方向向量 $\overrightarrow{AC}=(-2,0,0)$，在底面，所成角为 $0$（与底面重合），题意应为 $PC$ 与底面所成角。

$\overrightarrow{PC}=(1,-1,-\sqrt{3})$，与底面法向量 $(0,0,1)$ 夹角 $\theta$：

$\sin\theta=\dfrac{|\overrightarrow{PC}\cdot\hat{z}|}{|\overrightarrow{PC}|}=\dfrac{\sqrt{3}}{\sqrt{1+1+3}}=\dfrac{\sqrt{3}}{\sqrt{5}}$，$\theta=\arcsin\dfrac{\sqrt{3}}{\sqrt{5}}$

### 常见错误与纠正

| 错误 | 正确做法 |
|------|---------|
| 法向量取为任意方向 | 必须满足与平面内**两个不共线向量**都垂直 |
| 二面角直接取两法向量夹角 | 注意法向量方向，可能需取补角 |
| 线面角用法向量夹角 | 线面角用 $\sin$ 不用 $\cos$；用方向向量与法向量夹角的余角 |
| 建系时轴不互相垂直 | 必须是直角坐标系 |

### 解题步骤规范

1. 画立体图，标注已知量
2. 建立空间直角坐标系，求所有关键点坐标
3. 求所需向量（方向向量/法向量）
4. 代入公式，计算点积/模
5. 求反三角函数，得角度
