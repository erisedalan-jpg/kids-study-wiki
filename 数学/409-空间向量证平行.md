## 🎓 空间向量证平行

### 核心条件

用空间向量法证明平行关系，关键在于以下等价条件：

| 关系 | 向量条件 |
|------|----------|
| 两直线平行 $l_1 \parallel l_2$ | 方向向量共线：$\boldsymbol{v}_1 = \lambda\boldsymbol{v}_2$ |
| 直线平行于平面 $l \parallel \alpha$ | 直线方向向量 $\boldsymbol{v}$ 与平面法向量 $\boldsymbol{n}$ 垂直：$\boldsymbol{v}\cdot\boldsymbol{n}=0$，且直线不在平面内 |
| 两平面平行 $\alpha \parallel \beta$ | 两平面法向量共线：$\boldsymbol{n}_1 = \lambda\boldsymbol{n}_2$ |

### 方法步骤

**证线面平行**（$l \parallel \alpha$）：
1. 建立空间直角坐标系
2. 求直线 $l$ 的方向向量 $\boldsymbol{v}$
3. 求平面 $\alpha$ 的法向量 $\boldsymbol{n}$（取面内两不共线向量，解方程组）
4. 计算 $\boldsymbol{v}\cdot\boldsymbol{n}$，若为 0 则方向向量 $\perp$ 法向量
5. 验证直线不在平面内（取直线上一点，证不满足平面方程）

**证面面平行**（$\alpha \parallel \beta$）：
1. 分别求两面的法向量 $\boldsymbol{n}_1,\boldsymbol{n}_2$
2. 验证 $\boldsymbol{n}_1 = \lambda\boldsymbol{n}_2$（即两法向量共线）

### 典型例题

**例1**：正方体 $ABCD$-$A_1B_1C_1D_1$，棱长为 1，以 $A$ 为原点建坐标系，证明 $BD_1 \parallel$ 平面 $A_1BC$。（待家长核对具体坐标计算步骤）

**解**：
各坐标：$A(0,0,0)$，$B(1,0,0)$，$D(0,1,0)$，$A_1(0,0,1)$，$C(1,1,0)$，$D_1(0,1,1)$。

方向向量：$\overrightarrow{BD_1} = (-1,1,1)$

平面 $A_1BC$ 内向量：$\overrightarrow{A_1B}=(1,0,-1)$，$\overrightarrow{A_1C}=(1,1,-1)$

求法向量 $\boldsymbol{n}=(x,y,z)$：
$$x - z = 0 \Rightarrow x = z$$
$$x + y - z = 0 \Rightarrow y = 0$$

取 $x=1$：$\boldsymbol{n}=(1,0,1)$

验证：$\overrightarrow{BD_1}\cdot\boldsymbol{n} = -1+0+1=0$ ✓（方向向量 $\perp$ 法向量）

再验证 $B$ 不在平面 $A_1BC$ 的"外面"——$B$ 本身就在平面上，所以取 $D_1$：$D_1(0,1,1)$，$\overrightarrow{A_1D_1}=(0,1,0)$，$\boldsymbol{n}\cdot\overrightarrow{A_1D_1}=0$，说明 $D_1$ 也在平面内？（需结合具体几何关系判断，此处为示例框架）

**例2**（面面平行）：在正四棱柱中，底面 $ABCD$ 与顶面 $A_1B_1C_1D_1$ 平行，用向量验证。

**解**：底面法向量 $\boldsymbol{n}_1=(0,0,1)$，顶面法向量同为 $\boldsymbol{n}_2=(0,0,1)$，$\boldsymbol{n}_1=\boldsymbol{n}_2$，故两面平行。

### 与传统几何法的对比

| | 传统综合法 | 向量法 |
|--|---------|------|
| 优势 | 直观，不依赖坐标 | 计算化，步骤固定 |
| 难点 | 需要辅助线思路 | 需要准确建系求向量 |
| 高考应用 | 均可，向量法更程序化 | |

### 易错点

- 证线面平行时，需同时满足：①方向向量 $\perp$ 法向量；②直线**不在**平面内
- 法向量 $\boldsymbol{n}_1 \parallel \boldsymbol{n}_2$ 指共线（$\boldsymbol{n}_1=\lambda\boldsymbol{n}_2$），注意是方向相同或相反都算
