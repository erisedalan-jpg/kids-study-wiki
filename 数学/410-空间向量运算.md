## 🎓 空间向量运算

### 加法与减法

设 $\boldsymbol{a} = (a_1, a_2, a_3)$，$\boldsymbol{b} = (b_1, b_2, b_3)$：

$$\boldsymbol{a} + \boldsymbol{b} = (a_1+b_1,\; a_2+b_2,\; a_3+b_3)$$

$$\boldsymbol{a} - \boldsymbol{b} = (a_1-b_1,\; a_2-b_2,\; a_3-b_3)$$

**几何意义**：
- 加法：**平行四边形法则**（以 $\boldsymbol{a},\boldsymbol{b}$ 为邻边，对角线为 $\boldsymbol{a}+\boldsymbol{b}$）
- 减法：同一起点，$\boldsymbol{b}$ 指向 $\boldsymbol{a}$ 为 $\boldsymbol{a}-\boldsymbol{b}$

### 数乘运算

$$\lambda\boldsymbol{a} = (\lambda a_1,\; \lambda a_2,\; \lambda a_3)$$

**性质**：
- $|\lambda\boldsymbol{a}| = |\lambda| \cdot |\boldsymbol{a}|$
- $\lambda > 0$：方向与 $\boldsymbol{a}$ 相同；$\lambda < 0$：方向相反；$\lambda = 0$：结果为零向量

### 运算律

| 律名 | 表达式 |
|------|--------|
| 交换律 | $\boldsymbol{a}+\boldsymbol{b}=\boldsymbol{b}+\boldsymbol{a}$ |
| 结合律 | $(\boldsymbol{a}+\boldsymbol{b})+\boldsymbol{c}=\boldsymbol{a}+(\boldsymbol{b}+\boldsymbol{c})$ |
| 数乘分配律 | $\lambda(\boldsymbol{a}+\boldsymbol{b})=\lambda\boldsymbol{a}+\lambda\boldsymbol{b}$ |
| 数乘结合律 | $(\lambda\mu)\boldsymbol{a}=\lambda(\mu\boldsymbol{a})$ |

### 模（长度）

$$|\boldsymbol{a}| = \sqrt{a_1^2 + a_2^2 + a_3^2}$$

**单位向量**：$\hat{\boldsymbol{a}} = \dfrac{\boldsymbol{a}}{|\boldsymbol{a}|}$（方向与 $\boldsymbol{a}$ 相同，长度为 1）

### 典型例题

**例1**：设 $\boldsymbol{a}=(2,-1,3)$，$\boldsymbol{b}=(-1,2,1)$，求 $2\boldsymbol{a}-3\boldsymbol{b}$。

**解**：
$$2\boldsymbol{a}-3\boldsymbol{b} = (4,-2,6)-(-3,6,3) = (7,-8,3)$$

**例2**：求向量 $\boldsymbol{a}=(1,2,-2)$ 的长度和单位向量。

**解**：
$$|\boldsymbol{a}| = \sqrt{1^2+2^2+(-2)^2} = \sqrt{9} = 3$$

$$\hat{\boldsymbol{a}} = \dfrac{1}{3}(1,2,-2) = \left(\dfrac{1}{3},\dfrac{2}{3},-\dfrac{2}{3}\right)$$

**例3**（几何应用）：设 $\overrightarrow{OA}=\boldsymbol{a}$，$\overrightarrow{OB}=\boldsymbol{b}$，$\overrightarrow{OC}=\boldsymbol{c}$，$M$ 为 $\triangle ABC$ 重心，求 $\overrightarrow{OM}$。

**解**：重心 $M$ 坐标为三顶点平均，故 $\overrightarrow{OM} = \dfrac{\boldsymbol{a}+\boldsymbol{b}+\boldsymbol{c}}{3}$

### 易错点

- 空间向量加减法与平面向量完全类似，只是多了第三分量
- 计算模时不要漏算第三分量
- $-\boldsymbol{a}=(-1)\boldsymbol{a}$，方向相反，大小不变
