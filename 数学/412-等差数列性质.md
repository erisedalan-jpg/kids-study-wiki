## 🎓 等差数列性质

### 基本定义回顾

**等差数列**：相邻两项之差为常数 $d$（公差）的数列。

$$a_{n+1} - a_n = d \quad (d \in \mathbb{R})$$

通项公式：$a_n = a_1 + (n-1)d$

前 $n$ 项和：$S_n = na_1 + \dfrac{n(n-1)}{2}d = \dfrac{n(a_1+a_n)}{2}$

### 等差中项

若 $a, A, b$ 成等差数列，则 $A = \dfrac{a+b}{2}$，称 $A$ 为 $a$ 与 $b$ 的**等差中项**。

等价条件：$A - a = b - A$，即 $2A = a + b$。

### 等差数列的重要性质

**性质1（等间距项的关系）**：

$$a_m + a_n = a_p + a_q \quad \Leftrightarrow \quad m + n = p + q$$

特别地：$a_m + a_n = 2a_{\frac{m+n}{2}}$（$m+n$ 为偶数时）

**性质2（等差数列的子列）**：

等差数列 $\{a_n\}$ 中，每隔 $k$ 项取一项构成新数列 $a_1, a_{k+1}, a_{2k+1}, \ldots$，新数列也是等差数列，公差为 $kd$。

**性质3（前 $n$ 项和）**：

$S_n$ 是关于 $n$ 的**二次函数**（$d \ne 0$ 时）或线性函数（$d=0$ 时）：

$$S_n = \frac{d}{2}n^2 + \left(a_1 - \frac{d}{2}\right)n$$

常数项为 0！这是 $S_n$ 判断等差的依据。

**性质4**：$S_n, S_{2n}-S_n, S_{3n}-S_{2n}$ 也成等差数列（公差为 $n^2d$，待家长核对）

### 等差数列判断条件

三个等价条件：
1. $a_{n+1} - a_n = d$（常数）
2. $a_n = pn + q$（$p,q$ 为常数，$p \ne 0$，即线性通项）
3. $2a_{n+1} = a_n + a_{n+2}$（中间项等于两端平均）

### 典型例题

**例1**：等差数列 $\{a_n\}$，$a_3=7, a_8=17$，求公差和通项公式。

**解**：$a_8-a_3=5d=10$，$d=2$，$a_1=a_3-2d=3$，$a_n=3+(n-1)\times2=2n+1$

**例2**：等差数列前 10 项和为 100，前 20 项和为 300，求前 30 项和。

**解**：$S_{10}=100$，$S_{20}-S_{10}=200$，$S_{30}-S_{20}=300$（等差数列性质）

$S_{30}=100+200+300=600$

**例3**：$S_n = n^2 + 3n$，判断是否等差并求公差。

**解**：$a_n = S_n - S_{n-1} = n^2+3n - (n-1)^2-3(n-1) = 2n+2$（$n\ge2$）

$a_1=S_1=4=2\times1+2$ ✓，$d = a_{n+1}-a_n = 2$（等差，公差 2）

**例4**：等差数列 $\{a_n\}$ 中，$a_1 > 0, d < 0$，$S_n$ 在哪一项达最大？

**解**：$S_n = \dfrac{n(a_1+a_n)}{2}$，当 $a_n > 0, a_{n+1} < 0$ 时 $S_n$ 最大。

令 $a_n = a_1+(n-1)d \ge 0$，$n \le 1-\dfrac{a_1}{d} = 1+\dfrac{a_1}{|d|}$，最大整数 $n$ 即为所求。

### 易错点

- $S_n = An^2 + Bn$ 型（无常数项）才是等差；$S_n=An^2+Bn+C$（$C\ne0$）对应等差数列的前 $n$ 项和**加了常数**，不是等差
- 等差中项公式 $A=\dfrac{a+b}{2}$，只适用于三项成等差
- 性质 $a_m+a_n=a_p+a_q$ 要求 $m+n=p+q$，不能随意使用
