## 🎓 复合函数导数（链式法则）

### 链式法则

设 $y=f(u)$，$u=g(x)$，则复合函数 $y=f[g(x)]$ 的导数为：

$$\boxed{[f(g(x))]' = f'(g(x)) \cdot g'(x)}$$

用"中间变量"的语言：$\dfrac{\mathrm{d}y}{\mathrm{d}x} = \dfrac{\mathrm{d}y}{\mathrm{d}u} \cdot \dfrac{\mathrm{d}u}{\mathrm{d}x}$

**口诀**：外函数对中间变量求导，乘以中间变量对 $x$ 的导数（**不要忘乘内层导数**）。

### 分解步骤

1. 识别"外层函数"$f$ 和"内层函数"$g$
2. 对外层求导，内层当成整体（$f'(g(x))$，不对 $g(x)$ 内部求导）
3. 再乘以内层对 $x$ 的导数 $g'(x)$

### 典型例题

**例 1** $f(x) = \sin(2x)$

外层 $\sin$，内层 $2x$：$f'(x) = \cos(2x)\cdot 2 = 2\cos(2x)$

**例 2** $f(x) = (x^2+1)^5$

外层幂 $(\cdot)^5$，内层 $x^2+1$：$f'(x) = 5(x^2+1)^4 \cdot 2x = 10x(x^2+1)^4$

**例 3** $f(x) = e^{x^2+1}$

外层 $e^{(\cdot)}$，内层 $x^2+1$：$f'(x) = e^{x^2+1}\cdot 2x = 2xe^{x^2+1}$

**例 4** $f(x) = \ln(\sin x)$（$\sin x > 0$ 时）

外层 $\ln$，内层 $\sin x$：$f'(x) = \dfrac{1}{\sin x}\cdot\cos x = \dfrac{\cos x}{\sin x} = \cot x$

**例 5**（三重复合）$f(x) = \sin^2(3x)$

$f(x) = [\sin(3x)]^2$。逐层剥：
$$f'(x) = 2\sin(3x)\cdot[\sin(3x)]' = 2\sin(3x)\cdot\cos(3x)\cdot 3 = 6\sin(3x)\cos(3x) = 3\sin(6x)$$

### 常用复合形式

| 复合函数 | 导数 |
|---|---|
| $[g(x)]^n$ | $n[g(x)]^{n-1}\cdot g'(x)$ |
| $e^{g(x)}$ | $e^{g(x)}\cdot g'(x)$ |
| $\ln g(x)$ | $\dfrac{g'(x)}{g(x)}$ |
| $\sin g(x)$ | $\cos g(x)\cdot g'(x)$ |
| $\cos g(x)$ | $-\sin g(x)\cdot g'(x)$ |

### 与必修衔接

复合函数在必修一已学（$f\circ g$），链式法则是其求导版本。链式法则是所有复杂函数求导的核心工具——几乎所有复合结构都用它处理。

### 易错点

- **最常见错误**：只对外层求导，忘乘内层导数
  例：$(\sin(x^2))' = \cos(x^2)$（错）→ 应为 $\cos(x^2)\cdot 2x$（对）
- 多重复合要逐层剥，每层都要乘以对应的内层导数
- $\ln|g(x)|$ 的导数仍为 $\dfrac{g'(x)}{g(x)}$（高中阶段通常不涉及绝对值里的求导，知道即可）
