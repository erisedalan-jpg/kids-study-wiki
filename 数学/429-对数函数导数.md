## 🎓 对数函数导数

### 基本公式

$$\boxed{(\ln x)' = \frac{1}{x} \quad (x>0)} \qquad \boxed{(\log_a x)' = \frac{1}{x\ln a} \quad (a>0,a\neq 1,x>0)}$$

### 公式推导

**推导 $(\ln x)' = \dfrac{1}{x}$**（利用 $e^x$ 的反函数关系）：

设 $y=\ln x$，则 $x=e^y$。对 $x$ 求导：

$$\frac{dy}{dx} = \frac{1}{\frac{dx}{dy}} = \frac{1}{e^y} = \frac{1}{x}$$

**$\log_a x$ 换底**：

$$\log_a x = \frac{\ln x}{\ln a}$$，故 $(\log_a x)' = \frac{1}{\ln a}\cdot(\ln x)' = \frac{1}{x\ln a}$

### 典型例题

**例 1** $f(x) = \ln x + \log_2 x$

$$f'(x) = \frac{1}{x} + \frac{1}{x\ln 2}$$

**例 2** $f(x) = x\ln x$（乘法法则）

$$f'(x) = \ln x + x\cdot\frac{1}{x} = \ln x + 1$$

**例 3** $f(x) = \ln(x^2+1)$（链式法则）

$$f'(x) = \frac{2x}{x^2+1}$$

**例 4** $f(x) = \ln\dfrac{x+1}{x-1}$（化简再求导）

$$f(x) = \ln(x+1)-\ln(x-1), \quad f'(x) = \frac{1}{x+1}-\frac{1}{x-1} = \frac{-2}{x^2-1}$$

**例 5**（对数求导法）求 $f(x) = x^x$ 的导数（$x>0$）。

$$f(x) = e^{x\ln x}, \quad f'(x) = e^{x\ln x}\cdot(\ln x+1) = x^x(\ln x+1)$$

### $\ln x$ 的特殊意义

$\ln x$ 是面积函数：$\ln t = \int_1^t \dfrac{1}{x}dx$（积分的初步理解，高中可选）。

$\ln x$ 和 $e^x$ 互为反函数，导数互成倒数关系（反函数求导）：若 $f=e^x$，$f^{-1}=\ln x$，则 $(f^{-1})'(x) = \dfrac{1}{f'(f^{-1}(x))} = \dfrac{1}{e^{\ln x}} = \dfrac{1}{x}$。

### 易错点

- $(\ln x)'=\dfrac{1}{x}$，不是 $\dfrac{1}{x}\cdot\ln e=\dfrac{1}{x}$（这倒是对的，但不能写成 $\dfrac{\ln e}{x}$）
- 链式法则：$[\ln g(x)]' = \dfrac{g'(x)}{g(x)}$，分子是 $g'(x)$，不是 $1$
- 对数函数要求**真数大于 $0$**，定义域问题不可忽视
