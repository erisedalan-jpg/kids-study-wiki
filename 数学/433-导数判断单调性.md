## 🎓 导数判断单调性

### 判断准则

$$f'(x)>0 \text{ 在 } (a,b) \text{ 上} \Longrightarrow f(x) \text{ 在 } (a,b) \text{ 上单调递增}$$

$$f'(x)<0 \text{ 在 } (a,b) \text{ 上} \Longrightarrow f(x) \text{ 在 } (a,b) \text{ 上单调递减}$$

### 求单调区间的完整步骤

**步骤**：
1. 确定定义域（分母 $\neq 0$，根号下 $\geq 0$，对数真数 $> 0$）
2. 求 $f'(x)$
3. 解 $f'(x)=0$ 找驻点，结合定义域和 $f'(x)$ 不存在的点，将定义域分段
4. 在每段上判断 $f'(x)$ 的符号（用数轴符号分析或取样点）
5. 写出单调区间

### 符号分析法（数轴法）

在数轴上标出驻点和不连续点，在各区间内取一个点代入 $f'(x)$ 判断符号（比完整分解更高效）。

### 典型例题

**例 1** $f(x) = 2x^3 - 9x^2 + 12x - 3$，求单调区间。

$f'(x) = 6x^2 - 18x + 12 = 6(x-1)(x-2)$

驻点 $x=1$，$x=2$。取样：$f'(0)=12>0$，$f'(1.5)=6(0.5)(-0.5)<0$，$f'(3)=6>0$。

单调递增：$(-\infty,1)$ 和 $(2,+\infty)$；单调递减：$(1,2)$。

**例 2** $f(x) = \dfrac{x}{1+x^2}$，求单调区间。

$f'(x) = \dfrac{(1+x^2)-x\cdot 2x}{(1+x^2)^2} = \dfrac{1-x^2}{(1+x^2)^2}$

$f'(x)=0$：$x=\pm 1$；$f'(x)$ 分母恒正，所以 $f'(x)$ 的符号由 $1-x^2$ 决定。

- $1-x^2>0$ 即 $-1<x<1$：递增
- $1-x^2<0$ 即 $x<-1$ 或 $x>1$：递减

**例 3** $f(x) = x - \ln x$（$x>0$），求单调区间。

$f'(x) = 1 - \dfrac{1}{x} = \dfrac{x-1}{x}$

$x>0$ 时分母正，分子符号：$x>1$ 时 $f'>0$（递增），$0<x<1$ 时 $f'<0$（递减）。

极小值 $f(1)=1$，函数在 $x=1$ 处有最小值 $1$。

**例 4** 含参数的单调性：$f(x)=\ln x + \dfrac{a}{x}$ 在 $(0,+\infty)$ 上单调递增，求 $a$ 范围。

$f'(x)=\dfrac{1}{x}-\dfrac{a}{x^2}=\dfrac{x-a}{x^2}\geq 0$ 对所有 $x>0$ 成立。

因分母 $x^2>0$，需 $x-a\geq 0$ 对所有 $x>0$ 成立，即 $a\leq\inf_{x>0} x=0$。

故 $a\leq 0$。

### 注意事项

- 单调区间用**区间**表示，不用不等式
- 定义域的限制可能使驻点不在考察范围内
- 复杂函数可先化简（有理化、换底、展开）再求导，降低计算复杂度

### 易错点

- 忘记分析**定义域**就直接令 $f'=0$
- 分段函数在分界点处需检验左右导数
- 写单调区间时不能将两个区间合并（如不能把递增区间 $(-\infty,1)$ 和 $(2,+\infty)$ 写成一个）
