## 🎓 导数四则运算

### 四则求导法则

设 $u=u(x)$，$v=v(x)$ 均可导，则：

| 法则 | 公式 |
|---|---|
| 加减法则 | $(u\pm v)' = u' \pm v'$ |
| 数乘法则 | $(cu)' = cu'$（$c$ 为常数） |
| 乘法法则 | $(uv)' = u'v + uv'$ |
| 除法法则 | $\left(\dfrac{u}{v}\right)' = \dfrac{u'v - uv'}{v^2}$（$v\neq 0$） |

### 乘法法则推导

$$\frac{d}{dx}[u(x)v(x)] = \lim_{h\to 0}\frac{u(x+h)v(x+h)-u(x)v(x)}{h}$$

加减同一项 $u(x)v(x+h)$：

$$= \lim_{h\to 0}\left[\frac{u(x+h)-u(x)}{h}v(x+h) + u(x)\frac{v(x+h)-v(x)}{h}\right]$$

$$= u'(x)\cdot v(x) + u(x)\cdot v'(x)$$

### 典型例题

**例 1** 求 $f(x)=(x^2+1)(x^3-2x)$ 的导数。

法一（展开）：$f(x)=x^5-2x^3+x^3-2x=x^5-x^3-2x$，$f'(x)=5x^4-3x^2-2$

法二（乘法法则）：$f'(x)=2x(x^3-2x)+(x^2+1)(3x^2-2)$
$=2x^4-4x^2+3x^4-2x^2+3x^2-2=5x^4-3x^2-2$ ✓

**例 2** 求 $f(x)=\dfrac{x^2+1}{x-1}$ 的导数。

$$f'(x) = \frac{2x(x-1)-(x^2+1)\cdot 1}{(x-1)^2} = \frac{2x^2-2x-x^2-1}{(x-1)^2} = \frac{x^2-2x-1}{(x-1)^2}$$

**例 3** 求 $f(x)=x^2\sin x$ 的导数。

$$f'(x) = 2x\sin x + x^2\cos x$$

**例 4** 求 $f(x)=\dfrac{e^x}{\sin x}$ 的导数（$\sin x\neq 0$）。

$$f'(x) = \frac{e^x\sin x - e^x\cos x}{\sin^2 x} = \frac{e^x(\sin x - \cos x)}{\sin^2 x}$$

### 多项相乘的推广

三个函数乘积的求导（由乘法法则两次应用）：

$$(uvw)' = u'vw + uv'w + uvw'$$

一般地，$n$ 个函数乘积求导，每次对一个函数求导，其余不动，求和。

### 易错点

- 乘法法则：$(uv)' = u'v + uv'$，**不是** $u'v'$
- 除法法则分子：**分子减分母**（$u'v - uv'$），顺序不能颠倒
- 除法法则分母是 $v^2$（原分母的平方），不是 $v$
- 数乘法则：$(cu)' = cu'$，常数提到导数外，不要对常数求导再相减
