## 🎓 导数运算法则

### 法则总览

求导的核心工具是以下三类法则，组合使用可以求任意初等函数的导数：

#### 1. 基本公式

基础砖块——直接套用（详见[[447-常见函数导数|常见函数导数]]）。

#### 2. 四则法则

设 $u(x)$、$v(x)$ 可导：

$$\begin{aligned}
(u\pm v)' &= u' \pm v' \\
(cu)' &= cu' \quad (c\text{ 为常数}) \\
(uv)' &= u'v + uv' \\
\left(\frac{u}{v}\right)' &= \frac{u'v-uv'}{v^2} \quad (v\neq 0)
\end{aligned}$$

#### 3. 链式法则（复合函数求导）

$$[f(g(x))]' = f'(g(x))\cdot g'(x)$$

### 综合运用步骤

1. **分析结构**：函数是加减？乘除？复合？
2. **选择法则**：加减用线性法则；乘除用乘除法则；复合用链式法则
3. **逐层求导**：从外到内，每层施加对应法则
4. **化简结果**

### 典型例题（综合）

**例 1** $f(x) = x^2 e^x \sin x$

三函数乘积，两次乘法法则：
$$f'(x) = 2xe^x\sin x + x^2 e^x\sin x + x^2 e^x\cos x = x e^x(2\sin x + x\sin x + x\cos x)$$

**例 2** $f(x) = \dfrac{\ln x}{x^2}$

除法法则：$f'(x) = \dfrac{\frac{1}{x}\cdot x^2 - \ln x\cdot 2x}{x^4} = \dfrac{x - 2x\ln x}{x^4} = \dfrac{1-2\ln x}{x^3}$

**例 3** $f(x) = \sin^2 x + \cos^2 x$ 的导数（验证）

法一（恒等式）：$f(x)=1$，故 $f'(x)=0$

法二（链式法则）：$f'(x) = 2\sin x\cos x + 2\cos x(-\sin x) = 0$ ✓

**例 4** $f(x) = \ln\sqrt{1+x^2}$

化简：$f(x) = \dfrac{1}{2}\ln(1+x^2)$

$$f'(x) = \frac{1}{2}\cdot\frac{2x}{1+x^2} = \frac{x}{1+x^2}$$

### 求导流程图

```
给定函数
   ↓
是否是基本函数？→ 是 → 直接套基本公式
   ↓ 否
是否是加减组合？→ 是 → 各项分别求导再组合
   ↓ 否
是否是乘除组合？→ 是 → 用乘法/除法法则
   ↓ 否
是否是复合？→ 是 → 链式法则（外层导×内层导）
```

### 易错点

- 遇到 $f(x)^{g(x)}$（幂指函数，如 $x^x$）：高中阶段一般先取对数，再隐函数求导，或转化为 $e^{g(x)\ln f(x)}$ 再用链式法则
- 多层复合不能跳步，每层都要明确"对谁求导"
- 加减法则只对有限个项成立，无穷级数不能直接逐项求导（高中不涉及）
