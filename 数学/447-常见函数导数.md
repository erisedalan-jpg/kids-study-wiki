## 🎓 常见函数导数

### 基本导数公式表（必须熟记）

| 原函数 $f(x)$ | 导函数 $f'(x)$ | 备注 |
|---|---|---|
| $C$ | $0$ | $C$ 为常数 |
| $x^n$ | $nx^{n-1}$ | $n\in\mathbb{R}$ |
| $\sin x$ | $\cos x$ | $x$ 单位：弧度 |
| $\cos x$ | $-\sin x$ | $x$ 单位：弧度 |
| $\tan x$ | $\dfrac{1}{\cos^2 x}=\sec^2 x$ | $x\neq\dfrac{\pi}{2}+k\pi$ |
| $e^x$ | $e^x$ | 自然指数，自身导数仍为自身 |
| $a^x$ | $a^x\ln a$ | $a>0,a\neq 1$ |
| $\ln x$ | $\dfrac{1}{x}$ | $x>0$ |
| $\log_a x$ | $\dfrac{1}{x\ln a}$ | $a>0,a\neq 1,x>0$ |

### 典型公式推导

**$(\sin x)' = \cos x$ 的推导**（利用和差化积）：

$$(\sin x)' = \lim_{h\to 0}\frac{\sin(x+h)-\sin x}{h} = \lim_{h\to 0}\frac{2\cos\left(x+\frac{h}{2}\right)\sin\frac{h}{2}}{h}$$

$$= \lim_{h\to 0}\cos\left(x+\frac{h}{2}\right)\cdot\frac{\sin\frac{h}{2}}{\frac{h}{2}} = \cos x \cdot 1 = \cos x$$

其中用到重要极限 $\lim_{\theta\to 0}\dfrac{\sin\theta}{\theta}=1$。

**$(e^x)' = e^x$ 的推导**：

$$\lim_{h\to 0}\frac{e^{x+h}-e^x}{h} = e^x\lim_{h\to 0}\frac{e^h-1}{h} = e^x \cdot 1 = e^x$$

其中用到 $\lim_{h\to 0}\dfrac{e^h-1}{h}=1$（$e^x$ 自然指数的特征极限）。

### 典型例题

**例 1** 求下列函数的导数：

(1) $f(x) = x^5$ → $f'(x) = 5x^4$

(2) $f(x) = \dfrac{1}{x^2} = x^{-2}$ → $f'(x) = -2x^{-3} = -\dfrac{2}{x^3}$

(3) $f(x) = \sqrt{x} = x^{1/2}$ → $f'(x) = \dfrac{1}{2}x^{-1/2} = \dfrac{1}{2\sqrt{x}}$

(4) $f(x) = e^x + \ln x$ → $f'(x) = e^x + \dfrac{1}{x}$

**例 2** 验证 $f(x)=e^x$ 满足 $f'(x)=f(x)$。

$f'(x) = e^x = f(x)$，满足。这是 $e^x$ 的独特性质，也是 $e$ 这一底数的来源。

**例 3** 已知 $f(x) = \log_2 x$，求 $f'(4)$。

$$f'(x) = \frac{1}{x\ln 2}, \quad f'(4) = \frac{1}{4\ln 2}$$

### 记忆技巧

- **幂函数**：指数下移做系数，指数减 $1$
- **三角函数**：$\sin\leftrightarrow\cos$，求导 $\cos$ 不变号，$\sin$ 变号（$\cos$ 求导加负号）
- **指数**：$e^x$ 求导不变，$a^x$ 求导乘 $\ln a$
- **对数**：$\ln x$ 求导得 $\dfrac{1}{x}$，换底则多除一个 $\ln a$

### 易错点

- $\sqrt{x}$ 求导：$(x^{1/2})' = \dfrac{1}{2}x^{-1/2} = \dfrac{1}{2\sqrt{x}}$，不是 $\dfrac{1}{\sqrt{x}}$
- $\ln x$ 求导是 $\dfrac{1}{x}$，不是 $\dfrac{1}{x\ln e}$（因为 $\ln e=1$）
- $a^x$ 求导后多了因子 $\ln a$，忘记会导致错误
