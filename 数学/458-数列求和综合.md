## 🎓 数列求和综合

### 求和方法总览

| 方法 | 适用场景 | 关键操作 |
|---|---|---|
| 等差公式 | $a_n=An+B$ | $S_n=\dfrac{n(a_1+a_n)}{2}$ |
| 等比公式 | $a_n=A\cdot q^n$ | $S_n=\dfrac{a_1(1-q^n)}{1-q}$（$q\neq 1$） |
| 倒序相加 | 等差数列 | 首尾配对 |
| 错位相减 | $a_n=f(n)\cdot q^n$（等差×等比） | $S_n-qS_n$，消去内部项 |
| 裂项相消 | $a_n=\dfrac{1}{g(n)h(n)}$（可分解分母） | 化为 $\dfrac{c}{g(n)}-\dfrac{c}{h(n)}$ |
| 分组求和 | $a_n$ 可拆为两类 | 分别求和再相加 |

### 错位相减法详解

求 $S_n=\sum_{k=1}^n k\cdot q^k$（等差 $\times$ 等比型）：

$$S_n = 1\cdot q + 2\cdot q^2 + 3\cdot q^3 + \cdots + n\cdot q^n$$

$$qS_n = 1\cdot q^2 + 2\cdot q^3 + \cdots + (n-1)\cdot q^n + n\cdot q^{n+1}$$

两式相减（$S_n - qS_n$）：

$$(1-q)S_n = q+q^2+q^3+\cdots+q^n-nq^{n+1} = \frac{q(1-q^n)}{1-q} - nq^{n+1}$$

$$S_n = \frac{q(1-q^n)}{(1-q)^2} - \frac{nq^{n+1}}{1-q} \quad (q\neq 1)$$

### 裂项相消法详解

对 $a_n=\dfrac{1}{n(n+1)}$，注意 $\dfrac{1}{n(n+1)}=\dfrac{1}{n}-\dfrac{1}{n+1}$：

$$S_n = \left(1-\frac{1}{2}\right)+\left(\frac{1}{2}-\frac{1}{3}\right)+\cdots+\left(\frac{1}{n}-\frac{1}{n+1}\right) = 1-\frac{1}{n+1} = \frac{n}{n+1}$$

常用裂项：$\dfrac{1}{n(n+k)}=\dfrac{1}{k}\left(\dfrac{1}{n}-\dfrac{1}{n+k}\right)$

### 典型综合例题

**例 1** $a_n=n\cdot\left(\dfrac{1}{2}\right)^n$，求 $S_n$。

$S_n = 1\cdot\dfrac{1}{2}+2\cdot\dfrac{1}{4}+\cdots+n\cdot\dfrac{1}{2^n}$

$\dfrac{1}{2}S_n = 1\cdot\dfrac{1}{4}+2\cdot\dfrac{1}{8}+\cdots+(n-1)\cdot\dfrac{1}{2^n}+n\cdot\dfrac{1}{2^{n+1}}$

$\dfrac{1}{2}S_n = \dfrac{1}{2}+\dfrac{1}{4}+\cdots+\dfrac{1}{2^n}-n\cdot\dfrac{1}{2^{n+1}}$

$= \dfrac{1/2(1-1/2^n)}{1-1/2}-\dfrac{n}{2^{n+1}} = 1-\dfrac{1}{2^n}-\dfrac{n}{2^{n+1}}$

$S_n = 2-\dfrac{2}{2^n}-\dfrac{n}{2^n} = 2-\dfrac{n+2}{2^n}$

**例 2** $S_n=1+\dfrac{1}{1\times 2}+\dfrac{1}{2\times 3}+\cdots+\dfrac{1}{n(n+1)}$

裂项：$S_n = 1+\sum_{k=1}^n\left(\dfrac{1}{k}-\dfrac{1}{k+1}\right) = 1 + 1-\dfrac{1}{n+1} = 2-\dfrac{1}{n+1}$

**例 3** $a_n=(2n-1)\cdot 3^n$，求 $S_n$。

错位相减（等差 $2n-1$ × 等比 $3^n$）：

$S_n = 1\cdot 3 + 3\cdot 9 + 5\cdot 27 + \cdots + (2n-1)\cdot 3^n$

$3S_n = 1\cdot 9 + 3\cdot 27 + \cdots + (2n-3)\cdot 3^n + (2n-1)\cdot 3^{n+1}$

$S_n-3S_n = 3+2\cdot 9+2\cdot 27+\cdots+2\cdot 3^n-(2n-1)\cdot 3^{n+1}$

$-2S_n = 3+2(9+27+\cdots+3^n)-(2n-1)3^{n+1}$

$= 3+2\cdot\dfrac{9(3^{n-1}-1)}{2} - (2n-1)3^{n+1} = 3+3^{n+1}-9-(2n-1)3^{n+1}$

$= -6+(2-(2n-1))\cdot 3^{n+1} = -6+(3-2n)3^{n+1}$

$S_n = 3+(n-\frac{3}{2})3^{n+1}$（需仔细整理，以计算为准）

### 选择方法的流程

```
a_n 的形式？
├─ 等差 → 公式
├─ 等比 → 公式
├─ 等差+等比 → 分组
├─ 等差×等比 → 错位相减
├─ 分子为常数分母可分解 → 裂项
└─ 混合多类型 → 分组+各自方法
```
