## 🎓 两点分布

### 定义

若随机变量 $X$ 只取 0 和 1 两个值，且：

$$P(X=1) = p, \quad P(X=0) = 1-p = q \quad (0 < p < 1)$$

则称 $X$ 服从**两点分布**（也称伯努利分布或 0-1 分布），记作 $X \sim B(1,p)$。

### 分布列

| $X$ | 0 | 1 |
|-----|---|---|
| $P$ | $1-p$ | $p$ |

### 期望与方差

$$E(X) = 0 \times (1-p) + 1 \times p = p$$

$$E(X^2) = 0^2(1-p) + 1^2 \cdot p = p$$

$$D(X) = E(X^2) - [E(X)]^2 = p - p^2 = p(1-p) = pq$$

**记忆**：$E(X) = p$，$D(X) = pq$（$q = 1-p$）

### 实际背景

凡是"只有两种结果"的随机试验，均可用两点分布描述：
- 投硬币：正面 $p = 0.5$
- 产品质量：合格 $p = 0.95$，不合格 $1-p = 0.05$
- 射击：命中 $p$，未命中 $1-p$

**关键**：将一类结果定为"成功"（$X=1$），另一类定为"失败"（$X=0$）。

### 与二项分布的关系

$n$ 次独立重复的两点试验（伯努利试验），$X$ = 总成功次数，则 $X \sim B(n,p)$。

**两点分布是二项分布在 $n=1$ 时的特例**：$B(1,p) = $ 两点分布。

**分解**：若 $X \sim B(n,p)$，则 $X = X_1 + X_2 + \cdots + X_n$，其中 $X_i$ 独立同 $B(1,p)$ 分布。

$$E(X) = nE(X_i) = np, \quad D(X) = nD(X_i) = np(1-p)$$

### 典型例题

**例1**：某产品合格率为 0.9，随机取一件，$X$ = 该件是否合格（合格为 1），写出 $X$ 的分布列并求期望方差。

| $X$ | 0 | 1 |
|-----|---|---|
| $P$ | 0.1 | 0.9 |

$$E(X) = 0.9, \quad D(X) = 0.9 \times 0.1 = 0.09$$

**例2**：若 $X \sim B(1,p)$ 且 $D(X) = 0.16$，求 $p$。

$$p(1-p) = 0.16 \Rightarrow p^2 - p + 0.16 = 0$$
$$p = \frac{1 \pm \sqrt{1 - 0.64}}{2} = \frac{1 \pm 0.6}{2}$$

$p = 0.8$ 或 $p = 0.2$（两个解均合法，$0 < p < 1$）。

### 易错点

1. **$D(X) = p(1-p)$，非 $p^2$**：方差不是期望的平方
2. **两点分布要求 $p \neq 0,1$**：极端情形退化为常数，无随机性
3. **$X=1$ 的定义**：需明确哪个结果对应 $X=1$，哪个对应 $X=0$
4. **期望 = 成功概率**：$E(X) = p$，两点分布期望就是成功的概率
