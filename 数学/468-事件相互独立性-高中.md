## 🎓 事件相互独立性-高中

### 定义

若事件 $A$ 的发生不影响事件 $B$ 发生的概率，即：

$$\boxed{P(AB) = P(A) \cdot P(B)}$$

则称 $A$ 与 $B$ **相互独立**（简称独立）。

**等价条件**：$A, B$ 独立 $\Leftrightarrow$ $P(A|B) = P(A)$（当 $P(B) > 0$）。

### 注意：独立 $\neq$ 互斥

| | 独立 | 互斥 |
|--|------|------|
| 含义 | $A$ 发生不影响 $B$ | $A,B$ 不能同时发生 |
| 数学条件 | $P(AB) = P(A)P(B)$ | $P(AB) = 0$ |
| 关系 | 若 $P(A)>0, P(B)>0$，独立不能互斥 | 互斥事件一般不独立 |

### 独立性的推论

若 $A, B$ 独立，则以下各对事件也独立：
- $A$ 与 $\bar{B}$
- $\bar{A}$ 与 $B$
- $\bar{A}$ 与 $\bar{B}$

**多事件独立**：$A_1, A_2, \ldots, A_n$ 相互独立 $\Leftrightarrow$ 其中任意 $k$ 个（$2 \leq k \leq n$）的积的概率等于各自概率的乘积。

**实际判断**：若两个试验的结果互不影响（如两次掷骰子、甲乙各自射击），则对应事件通常独立。

### 独立时的乘法公式

$$P(A_1 A_2 \cdots A_n) = P(A_1) \cdot P(A_2) \cdots P(A_n)$$

### 典型例题

**例1（两事件）**：甲的命中率 0.8，乙的命中率 0.7，各射击一次且独立，求两人都命中的概率。

$$P(\text{两人都中}) = 0.8 \times 0.7 = 0.56$$

**例2（至少一人命中）**：接例1，求至少一人命中的概率。

**方法一（直接）**：

$$P = P(\text{甲中乙不中}) + P(\text{甲不中乙中}) + P(\text{都中})$$
$$= 0.8 \times 0.3 + 0.2 \times 0.7 + 0.56 = 0.24 + 0.14 + 0.56 = 0.94$$

**方法二（对立事件）**：

$$P = 1 - P(\text{都不中}) = 1 - 0.2 \times 0.3 = 1 - 0.06 = 0.94 \; \checkmark$$

**例3（验证独立性）**：掷两枚骰子，$A$ = "第一枚为奇数"，$B$ = "两枚之和为偶数"，判断是否独立。

$$P(A) = \frac{1}{2}, \quad P(B) = \frac{18}{36} = \frac{1}{2}$$

$$P(AB) = P(\text{第一枚奇，和偶}) = P(\text{两枚都奇}) = \frac{3 \times 3}{36} = \frac{1}{4}$$

$$P(A) \cdot P(B) = \frac{1}{2} \times \frac{1}{2} = \frac{1}{4} = P(AB) \; \Rightarrow \; A, B \text{ 独立}$$

**例4（多事件）**：三架飞机各自独立攻击目标，命中率分别为 0.4, 0.5, 0.7，求目标被摧毁（至少一架命中）的概率。

$$P = 1 - (1-0.4)(1-0.5)(1-0.7) = 1 - 0.6 \times 0.5 \times 0.3 = 1 - 0.09 = 0.91$$

### 独立性与条件概率的关系

- 若 $A, B$ 独立：$P(A|B) = P(A)$，$P(B|A) = P(B)$
- 若 $A, B$ 不独立：需用 $P(AB) = P(B)P(A|B)$ 计算

### 易错点

1. **独立 ≠ 互斥**：两个非零概率事件若互斥，则必定不独立（因为 $P(AB) = 0 \neq P(A)P(B)$）
2. **$P(AB) = P(A)P(B)$ 才是独立的定义**，不是 $P(A|B) = P(A)$（后者需 $P(B)>0$）
3. **"与""或"的计算**：独立时 $P(A \text{ 且 } B) = P(A)P(B)$；$P(A \text{ 或 } B) = 1 - P(\bar{A})P(\bar{B})$
4. **多事件独立的验证**：需要所有子集都满足乘法公式，不只是两两独立
