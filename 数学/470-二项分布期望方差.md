## 🎓 二项分布期望方差

### 核心结论

若 $X \sim B(n, p)$，则：

$$\boxed{E(X) = np}$$

$$\boxed{D(X) = np(1-p) = npq \quad (q = 1-p)}$$

$$\sigma(X) = \sqrt{np(1-p)}$$

### 推导一：利用分解

$X = X_1 + X_2 + \cdots + X_n$，其中 $X_i$ 为第 $i$ 次试验是否成功（独立，均服从 $B(1,p)$）：

$$E(X_i) = p, \quad D(X_i) = p(1-p)$$

由期望的线性性：

$$E(X) = \sum_{i=1}^n E(X_i) = np$$

由独立时方差可加性：

$$D(X) = \sum_{i=1}^n D(X_i) = np(1-p)$$

### 推导二：直接计算（期望）

$$E(X) = \sum_{k=0}^n k \cdot C_n^k p^k(1-p)^{n-k}$$

利用 $k \cdot C_n^k = n \cdot C_{n-1}^{k-1}$（当 $k \geq 1$）：

$$E(X) = n \sum_{k=1}^n C_{n-1}^{k-1} p^k(1-p)^{n-k}$$

令 $j = k-1$：

$$= np \sum_{j=0}^{n-1} C_{n-1}^j p^j (1-p)^{n-1-j} = np \cdot 1 = np \; \checkmark$$

### 典型例题

**例1（求期望方差）**：$X \sim B(12, 0.4)$，求 $E(X)$，$D(X)$。

$$E(X) = 12 \times 0.4 = 4.8$$

$$D(X) = 12 \times 0.4 \times 0.6 = 2.88$$

$$\sigma(X) = \sqrt{2.88} \approx 1.697$$

**例2（反推参数）**：$X \sim B(n,p)$，$E(X) = 6$，$D(X) = 4.2$，求 $n, p$。

$$np = 6, \quad npq = 4.2$$

$$q = \frac{4.2}{6} = 0.7, \quad p = 0.3, \quad n = \frac{6}{0.3} = 20$$

**例3（与正态分布的联系）**：$n$ 很大时，$B(n,p)$ 近似 $N(np, np(1-p))$（中心极限定理）。

这也解释了为什么正态分布的均值和方差参数与二项分布期望方差对应。

**例4（应用）**：某工厂生产的灯泡中，次品率为 5%，箱装 100 个，$X \sim B(100, 0.05)$：

$$E(X) = 100 \times 0.05 = 5 \text{ 个（期望次品数）}$$

$$D(X) = 100 \times 0.05 \times 0.95 = 4.75$$

### 期望方差的数值规律

- $np$ 随 $n$ 或 $p$ 增大而增大
- $npq$ 在 $p = 0.5$ 时最大（$= n/4$），$p$ 极端时趋向 0
- $q = 1-p$：成功概率越高，方差越小（因为结果更可预测）

### 易错点

1. **$D(X) = np(1-p)$ 不是 $(np)^2$**：方差不等于期望的平方
2. **$D(X) \leq E(X)$**：当 $0 < p < 1$ 时，$np(1-p) < np$（方差小于期望）
3. **只知期望不能唯一确定参数**：需同时知道方差才能解出 $n$ 和 $p$
4. **标准差与方差混淆**：$\sigma = \sqrt{D(X)}$，不是 $D(X)$ 本身
