## 🎓 二项式系数性质

### 二项式系数的定义

$(a+b)^n$ 展开式中第 $k+1$ 项的系数 $C_n^k$ 称为**二项式系数**（$k = 0,1,\ldots,n$）。

注意区分：
- **二项式系数**：$C_n^k$（只含组合数）
- **展开式各项系数**：包含 $a,b$ 本身的系数，如 $(2x+3)^n$ 中含 $2^{n-k}\cdot3^k$

### 核心性质

**性质 1（对称性）**：

$$C_n^k = C_n^{n-k}$$

第 $k+1$ 项与第 $n-k+1$ 项的二项式系数相等（关于中间对称）。

**性质 2（各系数之和）**：

$$\sum_{k=0}^n C_n^k = C_n^0 + C_n^1 + \cdots + C_n^n = 2^n$$

（令 $a=b=1$ 代入二项式定理）

**性质 3（奇偶项系数之和相等）**：

$$C_n^0 + C_n^2 + C_n^4 + \cdots = C_n^1 + C_n^3 + \cdots = 2^{n-1}$$

（令 $a=1,b=-1$ 得奇偶项系数和之差为 0，结合性质 2）

**性质 4（帕斯卡恒等式/递推）**：

$$C_n^k + C_n^{k+1} = C_{n+1}^{k+1}$$

**性质 5（最大二项式系数）**：
- $n$ 为偶数时，最大系数为 $C_n^{n/2}$（中间一项）
- $n$ 为奇数时，最大系数为 $C_n^{(n-1)/2} = C_n^{(n+1)/2}$（中间两项相等）

### 典型例题

**例1**：$(1+x)^{10}$ 的二项式系数之和。

$$\sum_{k=0}^{10} C_{10}^k = 2^{10} = 1024$$

**例2**：$(1+x)^{10}$ 奇数项系数之和（第1,3,5,…项的二项式系数之和）。

$$C_{10}^0 + C_{10}^2 + C_{10}^4 + C_{10}^6 + C_{10}^8 + C_{10}^{10} = 2^9 = 512$$

**例3**：$(1+x)^n$ 中最大二项式系数为 $C_n^4 = 70$，求 $n$。

$C_n^4 = 70$ 且 $4 = n/2$，故 $n = 8$，验证：$C_8^4 = 70$。

**例4（利用对称性）**：若 $(1+x)^n$ 展开式第 4 项系数 = 第 8 项系数，求 $n$。

第 4 项：$C_n^3$，第 8 项：$C_n^7$，由 $C_n^3 = C_n^7$ 和对称性 $C_n^k = C_n^{n-k}$ 得 $3 + 7 = n$，$n = 10$。

**例5（综合）**：$(2x - \frac{1}{x})^6$ 展开式中各项二项式系数之和为 $2^6 = 64$；各项系数之和（令 $x=1$）为 $(2-1)^6 = 1$。

### 帕斯卡三角的规律

- 每行对称（性质 1）
- 每行之和为 $2^n$（性质 2）
- 行间递推：$C_n^k = C_{n-1}^{k-1} + C_{n-1}^k$（性质 4 的等价形式）

### 易错点

1. **区分二项式系数与各项系数**：二项式系数是 $C_n^k$，各项系数可能含其他因子
2. **对称性的应用**：$3+7=10 \Rightarrow n=10$，注意下标之和等于 $n$
3. **最大系数项**：需判断 $n$ 的奇偶性，奇数时有两个最大值
4. **令 $x=1$ 求各项系数和**：只有当 $a=b=1$ 时，二项式系数才等于展开式系数
