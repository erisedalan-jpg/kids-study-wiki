## 🎓 全概率公式

### 背景：完备事件组

若事件 $B_1, B_2, \ldots, B_n$ 满足：
- 两两互斥：$B_i \cap B_j = \emptyset$（$i \neq j$）
- 完备：$B_1 \cup B_2 \cup \cdots \cup B_n = \Omega$（全集）
- $P(B_i) > 0$（各事件概率均正）

则称 $\{B_1, B_2, \ldots, B_n\}$ 为样本空间的一个**划分**（完备事件组）。

### 全概率公式

$$\boxed{P(A) = \sum_{i=1}^n P(B_i) P(A|B_i) = P(B_1)P(A|B_1) + P(B_2)P(A|B_2) + \cdots + P(B_n)P(A|B_n)}$$

**直观含义**："全面考虑所有可能的原因（$B_i$），对每种原因下结果 $A$ 的概率加权求和"。

### 常用简化版（$n = 2$）

若 $B$ 和 $\bar{B}$ 构成完备事件组（$P(B) + P(\bar{B}) = 1$）：

$$P(A) = P(B)P(A|B) + P(\bar{B})P(A|\bar{B})$$

### 推导

$$P(A) = P(A \cap \Omega) = P(A \cap (B_1 \cup \cdots \cup B_n))$$
$$= P(AB_1 \cup AB_2 \cup \cdots \cup AB_n)$$
$$= \sum_{i=1}^n P(AB_i) = \sum_{i=1}^n P(B_i)P(A|B_i) \; \checkmark$$

（$AB_1, \ldots, AB_n$ 两两互斥，利用加法公式）

### 典型例题

**例1（两工厂）**：甲厂生产 60% 的产品，乙厂生产 40%，甲厂次品率 2%，乙厂次品率 5%，随机取一件，求是次品的概率。

设 $B_1$ = 来自甲厂，$B_2$ = 来自乙厂；$A$ = 是次品。

$$P(A) = P(B_1)P(A|B_1) + P(B_2)P(A|B_2)$$
$$= 0.6 \times 0.02 + 0.4 \times 0.05 = 0.012 + 0.020 = 0.032$$

**例2（三种情形）**：一批产品由一、二、三级机器各生产 50%、30%、20%，次品率分别为 1%、2%、3%，求随机取一件是次品的概率。

$$P(A) = 0.5 \times 0.01 + 0.3 \times 0.02 + 0.2 \times 0.03$$
$$= 0.005 + 0.006 + 0.006 = 0.017$$

**例3（与贝叶斯联合）**：在例1基础上，已知取出的是次品，求来自甲厂的概率。

$$P(B_1|A) = \frac{P(B_1)P(A|B_1)}{P(A)} = \frac{0.012}{0.032} = \frac{3}{8}$$

（这正是贝叶斯公式的应用）

### 全概率公式的意义

- **分解复杂事件**：将 $P(A)$ 分解为各"原因"$B_i$ 的情形分别计算
- **"由因求果"**：已知各 $B_i$ 的概率和条件概率，求 $A$ 的总概率
- 与贝叶斯公式互补："由果求因" = 贝叶斯公式

### 易错点

1. **$\{B_i\}$ 必须构成完备事件组**：不完备（漏了某情形）会导致结果偏小
2. **权重 $P(B_i)$ 之和 = 1**：$\sum P(B_i) = 1$，可用于验证
3. **条件概率方向**：$P(A|B_i)$，条件在 $B_i$，结果是 $A$，不要颠倒
4. **各项正负**：$P(B_i)$ 和 $P(A|B_i)$ 均为正，全概率公式各项相加
