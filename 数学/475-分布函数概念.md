## 🎓 分布函数概念

### 定义

随机变量 $X$ 的**分布函数**（累积分布函数，CDF）定义为：

$$\boxed{F(x) = P(X \leq x)} \quad (-\infty < x < +\infty)$$

**直观含义**：$F(x)$ 表示随机变量 $X$ 取值不超过 $x$ 的概率。

### 基本性质

1. **单调不减**：$x_1 < x_2 \Rightarrow F(x_1) \leq F(x_2)$
2. **取值范围**：$0 \leq F(x) \leq 1$
3. **极限值**：$F(-\infty) = 0$，$F(+\infty) = 1$
4. **右连续**：$F(x^+) = F(x)$（高中阶段了解即可）

### 用分布函数计算区间概率

$$P(a < X \leq b) = F(b) - F(a)$$

$$P(X > a) = 1 - F(a)$$

$$P(X < a) = F(a^-) \leq F(a)$$

### 离散型随机变量的分布函数

若 $X$ 的分布列为 $P(X = x_k) = p_k$，则：

$$F(x) = P(X \leq x) = \sum_{x_k \leq x} p_k$$

分布函数为**阶梯函数**（在每个 $x_k$ 处跳跃，跳跃量 = $p_k$）。

**例**：$X$ 的分布列：

| $X$ | 1 | 2 | 3 |
|-----|---|---|---|
| $P$ | 0.2 | 0.5 | 0.3 |

$$F(x) = \begin{cases}
0 & x < 1 \\
0.2 & 1 \leq x < 2 \\
0.7 & 2 \leq x < 3 \\
1 & x \geq 3
\end{cases}$$

### 连续型随机变量的分布函数

若 $X$ 有密度函数 $f(x)$，则：

$$F(x) = \int_{-\infty}^x f(t) dt$$

分布函数为**连续函数**，$F'(x) = f(x)$（在 $f$ 连续处）。

**正态分布**：$X \sim N(\mu, \sigma^2)$ 的分布函数为：

$$F(x) = \Phi\left(\frac{x - \mu}{\sigma}\right)$$

其中 $\Phi(z) = \int_{-\infty}^z \frac{1}{\sqrt{2\pi}}e^{-t^2/2}dt$ 为标准正态分布函数（需查表）。

### 标准正态分布函数 $\Phi(z)$

常用值（待家长核对查表数值）：

| $z$ | $\Phi(z)$ |
|-----|-----------|
| 0 | 0.5 |
| 1 | 0.8413 |
| 1.645 | 0.95 |
| 1.96 | 0.975 |
| 2 | 0.9772 |
| 3 | 0.9987 |

**对称性**：$\Phi(-z) = 1 - \Phi(z)$

### 典型例题

**例1（离散型）**：从上面离散型例子，求 $P(1.5 < X \leq 2.5)$。

$$P(1.5 < X \leq 2.5) = F(2.5) - F(1.5) = 0.7 - 0.2 = 0.5$$

（只有 $X=2$ 在区间 $(1.5, 2.5]$ 内，$P(X=2)=0.5$，验证正确。）

**例2（正态分布查表）**：$X \sim N(3, 4)$（$\mu=3, \sigma=2$），求 $P(X \leq 5)$。

$$P(X \leq 5) = \Phi\left(\frac{5-3}{2}\right) = \Phi(1) = 0.8413$$

**例3（区间概率）**：$X \sim N(0,1)$，求 $P(-1.96 \leq X \leq 1.96)$。

$$P(-1.96 \leq X \leq 1.96) = \Phi(1.96) - \Phi(-1.96) = 0.975 - 0.025 = 0.95$$

（统计学中常用 95% 置信区间对应 $\pm 1.96\sigma$）

### 易错点

1. **$F(x) = P(X \leq x)$ 含等号**：分布函数包含等号（$X \leq x$）
2. **离散型 $F(x)$ 是阶梯函数**：在 $x_k$ 处跳跃，不连续
3. **$P(a < X \leq b) = F(b) - F(a)$**：不是 $F(b) - F(a+1)$
4. **$\Phi(-z) = 1 - \Phi(z)$**：对称性，不需要查负值对应的表
