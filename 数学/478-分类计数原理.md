## 🎓 分类计数原理

### 基本定义

**分类计数原理**（又称加法原理）：完成一件事情，有 $n$ 类方法，第 $i$ 类方法中有 $m_i$ 种，则完成这件事共有

$$N = m_1 + m_2 + \cdots + m_n$$

种方法。

**关键条件**：各类方法**互斥**（相互独立不重叠），且每种方法都能单独完成任务。

### 核心理解

- 分类原理对应"或"关系：选 A **或** 选 B
- 各类方法中任选一种，就能完成任务
- 与分步计数原理（乘法原理）的区别：分类用加法，分步用乘法

### 典型例题

**例1（基础）**：从北京到上海，可乘飞机（5 班次）、高铁（8 班次）或轮船（2 班次），共有多少种出行方式？

**解**：三类方式互斥，$N = 5 + 8 + 2 = 15$ 种。

**例2（数字问题）**：用 0,1,2,3,4,5 组成没有重复数字的三位数，有多少个？

**解**：按首位数字分类。
- 首位为 1,2,3,4,5（非零）共 5 种选法，后两位从剩余 5 个数中取 2 个：$5 \times A_5^2 = 5 \times 20 = 100$

（此题实际需结合乘法原理，体现两原理的综合运用）

**例3（路径问题）**：如图，从 A 到 B 有 3 条路，从 B 到 C 有 2 条路，从 A 直接到 C 有 4 条路，则从 A 到 C 共有多少条路？

**解**：经过 B 的路：$3 \times 2 = 6$ 条（乘法原理）；直达：4 条。两类互斥，$N = 6 + 4 = 10$ 条。

### 易错点

1. **分类不完全**：漏了某类方法，导致结果偏小
2. **分类有重叠**：同一方法被计入多类，导致结果偏大（需保证互斥）
3. **与乘法原理混淆**：分类用加法，分步用乘法；"完成任务只需一步"用加法
4. **含特殊元素未单独分类**：如含 0 的数字排列，需将首位含 0 的情况单独处理
