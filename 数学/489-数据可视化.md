## 🎓 数据可视化

### 定义

**数据可视化**：将数据转化为图形、图表等可视形式，帮助人们更快速、直观地理解数据中的规律、趋势和异常。

### 为什么需要数据可视化

- 人脑对图形的处理远快于对数字的处理
- 图形能展示数值表格无法直观呈现的模式
- 有助于发现异常值、趋势和关联
- 便于向他人传递信息

**安斯科姆四重奏（Anscombe's Quartet）**：四组几乎相同均值、方差和相关系数的数据集，但散点图完全不同。说明数值统计不能替代图形展示（待家长核对此例教材是否涉及）。

### 高中阶段的数据可视化工具

| 工具 | 用途 | 关键特征 |
|------|------|---------|
| 频率分布直方图 | 连续数据的分布 | 纵轴：频率/组距；面积=频率 |
| 茎叶图 | 小样本数据分布 | 保留原始数据，便于两组比较 |
| 散点图 | 两变量关系 | 判断相关方向和强弱 |
| 折线图 | 时间序列趋势 | 强调变化过程 |
| 条形图 | 类别频数比较 | 各类别高度=频数/频率 |
| 饼图 | 比例构成 | 扇形面积=比例 |
| 箱线图 | 分布和离散程度 | 五数概括（最小值、Q1、中位数、Q3、最大值） |

### 良好数据可视化的原则

1. **清晰**：图表能独立理解，有标题、轴标签、单位
2. **准确**：不扭曲数据（如纵轴不从非零开始误导读者）
3. **简洁**：去除不必要的装饰（"墨水"最少化原则）
4. **完整**：标注数据来源和样本量

### 典型例题

**例1（读图分析）**：某频率分布直方图（$x$ 轴为成绩，$y$ 轴为频率/组距，组距 10），其中 $[60,70)$ 对应的矩形高为 0.03，则该区间的频率和频数（共 100 人）各为？

频率：$0.03 \times 10 = 0.3$（30%）

频数：$100 \times 0.3 = 30$ 人

**例2（设计图表）**：某班 40 名学生三次模拟考试的平均成绩随时间变化，应选用哪种图表？描述什么？

**应选**：折线图（三次考试是时间序列，展示趋势）

折线图能展示：平均成绩是否整体上升/下降，有无剧烈波动。

**例3（散点图与回归）**：已建立回归方程 $\hat{y} = 2x + 5$，如何在散点图上画出回归直线？

取两点（如 $x=0 \Rightarrow y=5$，$x=10 \Rightarrow y=25$），连线即为回归直线，并标注方程。

**例4（可视化选择对比）**：

| 研究问题 | 数据类型 | 最适合图表 |
|---------|---------|----------|
| 各学科平均分比较 | 分类×数值 | 条形图 |
| 学生身高的分布 | 连续数值 | 频率分布直方图 |
| 降雨量与产量关系 | 两变量 | 散点图 |
| 全国人口年龄构成 | 分类比例 | 饼图或条形图 |
| 近5年学生人数变化 | 时间序列 | 折线图 |

### 数据可视化与统计分析的关系

- **探索性分析（EDA）**：用图形发现数据规律（先可视化，再建模）
- **验证性分析**：用图形展示回归拟合效果、残差分析
- **沟通报告**：用图形向他人呈现分析结论

### 数字时代的数据可视化

现代数据可视化工具（Excel、Python matplotlib、Tableau 等）能制作交互式图表，但核心原则不变：**选对图表类型、确保准确传达信息**。

### 易错点

1. **频率分布直方图纵轴是"频率/组距"**：不是频率本身，面积才是频率
2. **散点图不连线**：不要把散点图的点连成折线
3. **纵轴起点**：若纵轴不从 0 开始，会夸大变化幅度（需标注）
4. **图表与数据的一致性**：图形呈现的关系应与数值计算一致
