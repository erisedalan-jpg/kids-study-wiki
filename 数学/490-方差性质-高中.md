## 🎓 方差性质-高中

### 核心性质

设 $X, Y$ 为随机变量，$a, b$ 为常数：

**性质 1（常数方差为零）**：

$$D(c) = 0$$

**性质 2（线性变换）**：

$$D(aX + b) = a^2 D(X)$$

注意：加减常数 $b$ 不影响方差（平移不改变分散程度），乘以 $a$ 使方差变为 $a^2$ 倍。

**性质 3（独立变量之和）**：

$$D(X + Y) = D(X) + D(Y) \quad \text{（$X, Y$ 相互独立时）}$$

**性质 4（独立变量之差）**：

$$D(X - Y) = D(X) + D(Y) \quad \text{（$X, Y$ 相互独立时）}$$

（减法不等于差，因为方差衡量分散度，符号不影响）

**推论**：

$$D\left(\sum_{i=1}^n X_i\right) = \sum_{i=1}^n D(X_i) \quad \text{（各 $X_i$ 相互独立时）}$$

### 等价公式（复习）

$$D(X) = E(X^2) - [E(X)]^2$$

### 性质 2 的推导

$$D(aX+b) = E[(aX+b - E(aX+b))^2] = E[(aX+b - aE(X)-b)^2]$$
$$= E[a^2(X-E(X))^2] = a^2 E[(X-E(X))^2] = a^2 D(X)$$

### 典型例题

**例1**：$D(X) = 4$，求 $D(3X - 2)$。

$$D(3X-2) = 3^2 D(X) = 9 \times 4 = 36$$

**例2**：$D(X) = 2$，$D(Y) = 3$，$X, Y$ 独立，求 $D(2X - Y + 1)$。

$$D(2X - Y + 1) = D(2X) + D(-Y) + D(1)$$
$$= 4D(X) + 1 \cdot D(Y) + 0 = 4 \times 2 + 3 = 11$$

（注意：$D(2X - Y + 1) = D(2X - Y)$，常数不影响方差）

**例3（标准化变量）**：设 $E(X) = \mu$，$D(X) = \sigma^2$，令 $Z = \frac{X-\mu}{\sigma}$，求 $E(Z)$ 和 $D(Z)$。

$$E(Z) = E\left(\frac{X-\mu}{\sigma}\right) = \frac{E(X)-\mu}{\sigma} = 0$$

$$D(Z) = D\left(\frac{X-\mu}{\sigma}\right) = \frac{1}{\sigma^2}D(X-\mu) = \frac{1}{\sigma^2} \cdot \sigma^2 = 1$$

结论：标准化变量 $Z = \frac{X-\mu}{\sigma}$ 满足 $E(Z)=0$，$D(Z)=1$（标准正态分布的基础）。

**例4**：$X \sim B(n,p)$，$n$ 个独立 $B(1,p)$ 分布随机变量之和，$D(X) = np(1-p)$（见[[470-二项分布期望方差|二项分布期望方差]]）。

### 方差性质汇总表

| 运算 | 对期望的影响 | 对方差的影响 |
|------|------------|------------|
| $aX$ | $E(aX) = aE(X)$ | $D(aX) = a^2D(X)$ |
| $X + b$ | $E(X+b) = E(X)+b$ | $D(X+b) = D(X)$ |
| $X + Y$（独立） | $E(X+Y) = E(X)+E(Y)$ | $D(X+Y) = D(X)+D(Y)$ |
| $X - Y$（独立） | $E(X-Y) = E(X)-E(Y)$ | $D(X-Y) = D(X)+D(Y)$ |

### 易错点

1. **$D(X+b) = D(X)$**：加常数不改变方差，忘记这点会误算
2. **$D(X-Y) = D(X)+D(Y)$**：减法使方差相加，不是相减
3. **$D(aX) = a^2D(X)$**：系数要平方，不是 $aD(X)$
4. **非独立时不可加**：$D(X+Y) = D(X)+D(Y)+2\text{Cov}(X,Y)$（高中阶段不考协方差，但需知道非独立时不能直接加）
