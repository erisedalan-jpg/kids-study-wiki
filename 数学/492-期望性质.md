## 🎓 期望性质

### 核心性质

设 $X, Y$ 为随机变量，$a, b, c$ 为常数：

**性质 1（常数的期望）**：

$$E(c) = c$$

（常数的期望就是常数本身）

**性质 2（线性变换）**：

$$E(aX + b) = aE(X) + b$$

**性质 3（期望的线性性）**：

$$E(X + Y) = E(X) + E(Y)$$

（无论 $X, Y$ 是否独立，此式恒成立）

**性质 4（独立乘积）**：

$$E(XY) = E(X) \cdot E(Y) \quad \text{（$X, Y$ 相互独立时）}$$

### 推论

$$E(aX + bY + c) = aE(X) + bE(Y) + c$$

$$E\left(\sum_{i=1}^n X_i\right) = \sum_{i=1}^n E(X_i)$$

### 性质的直观理解

**线性变换**：若 $X$ 表示气温（℃），$Y = \frac{9X}{5} + 32$ 为华氏温度，则

$$E(Y) = \frac{9}{5}E(X) + 32$$

均值也做同样的线性变换，很自然。

**期望的线性性**：若今天销售额 $X$ 和明天销售额 $Y$ 各有分布，则两天总销售额 $X+Y$ 的期望就是两天期望之和——即使两天的销售额有关联（如天气影响），此式仍成立。

### 典型例题

**例1**：$E(X) = 2$，求 $E(3X - 4)$。

$$E(3X - 4) = 3E(X) - 4 = 3 \times 2 - 4 = 2$$

**例2**：$X$ 的分布列如下，$Y = X^2 - 1$，求 $E(Y)$。

| $X$ | -1 | 0 | 1 | 2 |
|-----|----|---|---|---|
| $P$ | 0.2 | 0.3 | 0.3 | 0.2 |

**解法一**（先求 $E(X^2)$ 再用线性性）：

$$E(X^2) = (-1)^2 \times 0.2 + 0^2 \times 0.3 + 1^2 \times 0.3 + 2^2 \times 0.2$$
$$= 0.2 + 0 + 0.3 + 0.8 = 1.3$$

$$E(Y) = E(X^2 - 1) = E(X^2) - 1 = 1.3 - 1 = 0.3$$

注意：$E(X^2 - 1) = E(X^2) - 1$（$-1$ 是常数），但 $E(X^2) \neq [E(X)]^2$！

验证 $E(X) = (-1)(0.2) + 0(0.3) + 1(0.3) + 2(0.2) = -0.2 + 0 + 0.3 + 0.4 = 0.5$，
$[E(X)]^2 = 0.25 \neq 1.3 = E(X^2)$。

**例3（独立变量之和）**：$X \sim B(5, 0.3)$，$Y \sim B(3, 0.4)$，$X, Y$ 独立，求 $E(2X + Y - 1)$。

$$E(X) = 5 \times 0.3 = 1.5, \quad E(Y) = 3 \times 0.4 = 1.2$$

$$E(2X + Y - 1) = 2 \times 1.5 + 1.2 - 1 = 3 + 1.2 - 1 = 3.2$$

### 性质 3 的证明（简要）

设 $X, Y$ 取有限个值，

$$E(X+Y) = \sum_{i,j}(x_i+y_j)P(X=x_i, Y=y_j)$$
$$= \sum_{i,j}x_i P(X=x_i, Y=y_j) + \sum_{i,j}y_j P(X=x_i, Y=y_j)$$
$$= \sum_i x_i P(X=x_i) + \sum_j y_j P(Y=y_j) = E(X) + E(Y) \; \checkmark$$

### 易错点

1. **$E(X^2) \neq [E(X)]^2$**：二者一般不等，差值就是方差 $D(X) = E(X^2) - [E(X)]^2 \geq 0$
2. **非独立时 $E(XY) \neq E(X)E(Y)$**：独立是乘积期望分解的前提
3. **常数的处理**：$E(aX+b) = aE(X)+b$，$b$ 直接加，不乘以概率
4. **符号不要丢**：若 $a$ 为负，$E(aX+b) = aE(X)+b$ 中 $a$ 要保留负号
