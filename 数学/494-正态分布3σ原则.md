## 🎓 正态分布3σ原则

### 核心内容

若 $X \sim N(\mu, \sigma^2)$，则：

$$\boxed{P(\mu - \sigma < X < \mu + \sigma) \approx 0.6827}$$

$$\boxed{P(\mu - 2\sigma < X < \mu + 2\sigma) \approx 0.9545}$$

$$\boxed{P(\mu - 3\sigma < X < \mu + 3\sigma) \approx 0.9973}$$

**口诀**：约 68%（$1\sigma$）、95%（$2\sigma$）、99.7%（$3\sigma$）。

也称为 **68-95-99.7 法则**。

### 对称区间概率

由对称性，各半侧概率：

$$P(\mu < X < \mu + \sigma) \approx 0.3413, \quad P(\mu - \sigma < X < \mu) \approx 0.3413$$

$$P(X > \mu + \sigma) \approx \frac{1-0.6827}{2} = 0.1587$$

$$P(X > \mu + 2\sigma) \approx \frac{1-0.9545}{2} = 0.0228$$

$$P(X > \mu + 3\sigma) \approx \frac{1-0.9973}{2} = 0.0013$$

### 标准正态分布的对应值

查 $\Phi(x)$ 表（标准正态分布函数）：

| $z$ | $\Phi(z)$ | $P(-z < Z < z)$ |
|-----|-----------|-----------------|
| 1 | 0.8413 | 0.6827 |
| 2 | 0.9772 | 0.9545 |
| 3 | 0.9987 | 0.9973 |

$$P(-z < Z < z) = 2\Phi(z) - 1$$

### 实际意义

$3\sigma$ 原则表明：
- 正常数据几乎都落在 $(\mu - 3\sigma, \mu + 3\sigma)$ 内（99.73%）
- 落在 $3\sigma$ 之外的概率仅约 0.27%（约 1/370）

**质量控制**：若产品尺寸服从正态分布，规格限 $\mu \pm 3\sigma$（"六西格玛"工厂进一步用 $\pm 6\sigma$ 标准）。

**异常检测**：若某数据超出 $\mu \pm 3\sigma$，可认为是"异常值"，需要进一步检查。

### 典型例题

**例1（基础）**：某工厂生产的螺丝直径 $X \sim N(10, 0.04)$（$\mu = 10$ mm，$\sigma = 0.2$ mm），求：

(1) $P(9.8 < X < 10.2)$

注意 $9.8 = \mu - \sigma$，$10.2 = \mu + \sigma$，故：
$$P(9.8 < X < 10.2) \approx 0.6827$$

(2) $P(X > 10.4)$

$10.4 = \mu + 2\sigma$，故 $P(X > 10.4) \approx 0.0228$

**例2（异常判断）**：某地区成年男性身高 $X \sim N(170, 36)$（$\sigma = 6$ cm），一个人身高 188 cm（$= \mu + 3\sigma$），是异常值吗？

$P(X > 188) \approx 0.0013$，约 1000 人中只有 1.3 人，确实很罕见，可认为偏高（但不能断言"异常"，自然界确实存在这样的人）。

**例3（逆向）**：已知 $X \sim N(\mu, \sigma^2)$，$P(\mu - k < X < \mu + k) = 0.9545$，求 $k$。

$$0.9545 = P(\mu - 2\sigma < X < \mu + 2\sigma) \Rightarrow k = 2\sigma$$

**例4（高考型）**：$X \sim N(80, 100)$（$\mu = 80, \sigma = 10$），已知 $P(70 < X < 90) = 0.6827$，求 $P(X > 90)$。

$$P(X > 90) = \frac{1 - 0.6827}{2} = \frac{0.3173}{2} = 0.1587$$

### 与统计分析的联系

$3\sigma$ 原则是统计过程控制（SPC）的基础：
- **控制图**：以 $\mu \pm 3\sigma$ 为控制限，落在界外的点报警
- **假设检验**：在原假设成立时，落在 $\pm 3\sigma$ 外的事件概率极小，可作为拒绝原假设的依据（待高中后深化学习）

### 易错点

1. **$3\sigma$ 对应 99.7%，不是 100%**：极端值仍有可能发生（约 0.3%）
2. **概率查的是对称区间**：$P(\mu - k\sigma < X < \mu + k\sigma)$，两边都要
3. **单侧概率 = (1 - 中间概率)/2**：由对称性计算
4. **$\sigma^2$ vs $\sigma$**：$N(\mu, \sigma^2)$ 中第二参数是**方差**，$3\sigma$ 原则用的是标准差 $\sigma$
