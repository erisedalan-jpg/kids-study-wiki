## 🎓 正态分布密度曲线

### 密度函数

$X \sim N(\mu, \sigma^2)$ 的概率密度函数（概率密度曲线）：

$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

### 曲线的几何特征

| 特征 | 描述 |
|------|------|
| 形状 | 钟形，关于 $x = \mu$ 对称 |
| 最高点 | $x = \mu$ 处，$f(\mu) = \dfrac{1}{\sqrt{2\pi}\sigma}$ |
| 拐点 | $x = \mu \pm \sigma$ 处（凸凹转折） |
| 渐近线 | 以 $x$ 轴为渐近线（$f(x) \to 0$ 当 $|x| \to \infty$） |
| 面积 | 曲线下总面积 = 1 |

### $\mu$ 和 $\sigma$ 对曲线的影响

**改变 $\mu$（位移）**：
- $\mu$ 增大 → 曲线整体右移
- $\mu$ 减小 → 曲线整体左移
- 形状不变，只是水平平移

**改变 $\sigma$（缩放）**：
- $\sigma$ 增大 → 曲线变宽变矮（数据更分散）
- $\sigma$ 减小 → 曲线变窄变高（数据更集中）
- 对称轴不变

**最高点高度 = $\dfrac{1}{\sqrt{2\pi}\sigma}$**：$\sigma$ 越大，峰值越低。

### 面积与概率的关系

密度曲线下 $[a, b]$ 上的面积 = $P(a \leq X \leq b)$。

**关键区间的面积**（$3\sigma$ 原则）：

$$P(\mu - \sigma < X < \mu + \sigma) \approx 0.6827$$
$$P(\mu - 2\sigma < X < \mu + 2\sigma) \approx 0.9545$$
$$P(\mu - 3\sigma < X < \mu + 3\sigma) \approx 0.9973$$

### 标准正态分布曲线

$\varphi(x) = \dfrac{1}{\sqrt{2\pi}}e^{-x^2/2}$（$\mu = 0, \sigma = 1$）

- 关于 $y$ 轴对称
- 最高点在 $(0, \frac{1}{\sqrt{2\pi}}) \approx (0, 0.399)$
- 拐点在 $x = \pm 1$

**标准化**：$P(a < X < b) = P\left(\frac{a-\mu}{\sigma} < Z < \frac{b-\mu}{\sigma}\right) = \Phi\left(\frac{b-\mu}{\sigma}\right) - \Phi\left(\frac{a-\mu}{\sigma}\right)$

### 典型例题

**例1（读图）**：已知 $X \sim N(\mu_1, \sigma_1^2)$ 和 $Y \sim N(\mu_2, \sigma_2^2)$，两曲线中 A 曲线较高较窄，B 曲线较矮较宽，则：

- 若 A、B 轴对称位置相同：$\mu_1 = \mu_2$，$\sigma_1 < \sigma_2$（A 更集中）
- 若 A 的对称轴在 B 左边：$\mu_1 < \mu_2$

**例2（概率计算）**：$X \sim N(5, 1)$（$\sigma = 1$），已知 $P(4 < X < 6) = 0.6827$，求 $P(X > 6)$。

$$P(X > 6) = P(X > \mu + \sigma) = \frac{1 - 0.6827}{2} = \frac{0.3173}{2} \approx 0.1587$$

（利用对称性，两侧各占一半）

**例3（参数估计）**：某种灯泡寿命 $X \sim N(1000, 100^2)$，概率密度曲线在什么点最高？拐点在哪里？

- 最高点：$x = \mu = 1000$（小时）
- 拐点：$x = \mu \pm \sigma = 1000 \pm 100$，即 $x = 900$ 和 $x = 1100$

### 与频率分布直方图的关系

- 频率分布直方图是样本数据的"离散"表示
- 当样本量 $n$ 增大，组距缩小，频率分布直方图趋向**概率密度曲线**
- 正态曲线是大样本数据的理论极限

### 易错点

1. **密度 $f(x) \neq$ 概率**：$f(x)$ 是密度，不是点概率；概率 = 曲线下面积
2. **$f(x) > 1$ 是可能的**：密度函数值可以大于 1（只要积分 = 1 即合法）
3. **拐点 $\neq$ 最大值点**：拐点在 $\mu \pm \sigma$，最大值点在 $\mu$
4. **改变 $\sigma$ 同时改变高度和宽度**：$\sigma$ 增大，峰值降低，宽度增加（面积守恒）
