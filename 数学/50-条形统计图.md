# 条形统计图

> **一句话**：用长短不同的长方形条子来表示数量多少的图——条越长，数越大。
> **English**: A bar chart uses rectangles of different lengths to represent quantities — the longer the bar, the bigger the number.

---

## 🧒 给 3-6 岁（共读版）

你喜欢吃什么水果？苹果、香蕉还是草莓？

如果我们班有 5 个小朋友喜欢苹果，3 个喜欢香蕉，7 个喜欢草莓，怎么让别人一眼就看出来哪个最受欢迎呢？

用**条形统计图**！

我们给每种水果画一根"柱子"——喜欢的人越多，柱子就越长（越高）。草莓的柱子最高，就能马上看出草莓最受欢迎！

**共读小活动：** 问问家里每个人最喜欢哪种水果，然后在纸上给每种水果画一个"柱子"，有几个人喜欢就画几格——做出你家的第一张统计图！

## 📚 给 6-12 岁（自读版）

### 条形统计图的结构

一张完整的条形统计图包括这几个部分：

| 部分 | 说明 |
|------|------|
| **横轴**（x 轴） | 表示分类，比如"月份""班级""水果名" |
| **纵轴**（y 轴） | 表示数量，有刻度（0、5、10、15……） |
| **条形** | 每个分类一根，高度 = 该类的数量 |
| **标题** | 说明这张图统计的是什么 |
| **单位** | 标在纵轴上，如"人数/人"、"温度/℃" |

**读图步骤：**
1. 先看标题，知道统计的是什么
2. 看纵轴的单位和刻度
3. 找到你要查的条形，看它的顶端对应纵轴的哪个数

### 单式条形图 vs 复式条形图

**单式条形图：** 只统计**一类**数据。

例：四年级各班男生人数

```
人数
25 |      █
20 |  █   █   █
15 |  █   █   █   █
10 |  █   █   █   █
 5 |  █   █   █   █
 0 +--+---+---+---+---
     一班 二班 三班 四班
```

**复式条形图：** 同时比较**两类**数据，用**不同颜色**区分。

例：四年级各班男女生人数对比（每班两根条形，颜色不同）

复式条形图的好处：可以同时看到两组数据，方便**比较差异**。

### 如何用条形图回答问题？

根据图表数据，常见题型：

1. **哪个最多/最少？** → 找最高/最低的条
2. **相差多少？** → 用两根条的高度作减法
3. **合计多少？** → 把几根条的高度加起来
4. **平均是多少？** → 总和 ÷ 条的根数

**例题：** 某班同学最喜欢的运动：篮球 12 人、足球 8 人、乒乓球 15 人、游泳 5 人。
- 最受欢迎：乒乓球（15 人）
- 篮球比游泳多：12 - 5 = **7 人**
- 四项合计：12 + 8 + 15 + 5 = **40 人**

### 为什么用图而不用表？

表格数据需要逐行比较，条形图一眼就能看出大小关系。数据越多，图的优势越明显。这就是**数据可视化**的核心思想：把数字变成形状，让眼睛帮助大脑理解。

## 🎓 给 12+（进阶版）

### 用 Python 绘制条形图

```python
import matplotlib.pyplot as plt
import matplotlib

# 支持中文显示
matplotlib.rcParams['font.sans-serif'] = ['SimHei']
matplotlib.rcParams['axes.unicode_minus'] = False

categories = ['篮球', '足球', '乒乓球', '游泳']
values = [12, 8, 15, 5]

plt.figure(figsize=(8, 5))
bars = plt.bar(categories, values, color=['steelblue', 'coral', 'mediumseagreen', 'orange'])

# 在每根条形顶部显示数值
for bar, val in zip(bars, values):
    plt.text(bar.get_x() + bar.get_width()/2, bar.get_height() + 0.2,
             str(val), ha='center', va='bottom', fontsize=12)

plt.title('某班同学最喜欢的运动', fontsize=14)
plt.ylabel('人数 / 人', fontsize=12)
plt.ylim(0, 20)
plt.tight_layout()
plt.savefig('sport_bar_chart.png', dpi=150)
plt.show()
```

### 复式条形图（分组柱状图）

```python
import numpy as np

classes = ['一班', '二班', '三班', '四班']
boys  = [22, 25, 20, 23]
girls = [18, 15, 22, 17]

x = np.arange(len(classes))
width = 0.35

fig, ax = plt.subplots(figsize=(9, 5))
ax.bar(x - width/2, boys,  width, label='男生', color='steelblue')
ax.bar(x + width/2, girls, width, label='女生', color='lightcoral')

ax.set_xticks(x)
ax.set_xticklabels(classes)
ax.set_ylabel('人数 / 人')
ax.set_title('四年级各班男女生人数')
ax.legend()
plt.tight_layout()
plt.show()
```

### 统计学视角：图表选择原则

- **条形图**：适合分类数据（类别有限、无序或有序皆可）
- **折线图**：适合连续时间序列（体现趋势）
- **饼图**：适合展示部分与整体的比例（类别数 ≤ 6）
- **散点图**：适合探索两个变量间的关系

条形图的纵轴**必须从 0 开始**，否则会视觉上夸大差异，产生误导（这是一种常见的统计欺骗手段，称为"截断坐标轴"）。

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么纵轴要从 0 开始？"** → 可以说："如果不从 0 开始，比如从 10 开始，那两根高度差 1 格的柱子看起来就像差了很多，会让人产生误解。从 0 开始才能真实反映数量的比例关系。"

- 孩子可能问 **"复式条形图和单式有什么不同？"** → 可以说："单式只看一件事，比如'每班的男生人数'。复式同时看两件事，比如'同一班的男女生人数'，放在一起就能直接比。"

- 孩子可能问 **"条形图和折线图有什么区别？"** → 可以说："条形图适合比较不同分类的大小，折线图适合看某一件事随着时间怎么变化。比如比较四个班的成绩用条形图，看一个班一整年的成绩变化用折线图。"

- **共读小活动：** 一起调查家里或班里同学最喜欢的颜色/食物/动物，收集数据后在纸上画出条形图，再讨论"哪个最多""差多少"——完成从数据收集到图表阅读的完整统计过程。
