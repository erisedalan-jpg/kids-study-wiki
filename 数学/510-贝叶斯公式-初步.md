## 🎓 贝叶斯公式-初步

### 定义

设 $\{B_1, B_2, \ldots, B_n\}$ 为样本空间的完备事件组（$P(B_i) > 0$），$A$ 为任意事件（$P(A) > 0$），则：

$$\boxed{P(B_j|A) = \frac{P(B_j)P(A|B_j)}{\sum_{i=1}^n P(B_i)P(A|B_i)} = \frac{P(B_j)P(A|B_j)}{P(A)}}$$

### 含义："由果求因"

- **全概率公式**："知道原因 $B_i$ 的概率，求结果 $A$ 的概率"（由因求果）
- **贝叶斯公式**："已知结果 $A$ 发生，反推原因 $B_j$ 的概率"（由果求因）

**贝叶斯思想**：根据新的观测（$A$ 发生），更新对各"假设"（$B_j$）概率的判断。
- **先验概率**：$P(B_j)$（未观测 $A$ 前的判断）
- **后验概率**：$P(B_j|A)$（观测到 $A$ 后的更新判断）

### 推导

由条件概率定义和乘法公式：

$$P(B_j|A) = \frac{P(AB_j)}{P(A)} = \frac{P(B_j)P(A|B_j)}{P(A)}$$

将 $P(A)$ 用全概率公式展开即得贝叶斯公式。

### 典型例题

**例1（质检问题）**：甲厂产 60%，乙厂产 40%；甲厂次品率 2%，乙厂次品率 5%。随机取一件发现是次品，问来自甲厂的概率。

**解**：已知 $P(\text{甲}) = 0.6$，$P(\text{乙}) = 0.4$；$P(A|\text{甲}) = 0.02$，$P(A|\text{乙}) = 0.05$。

$$P(A) = 0.6 \times 0.02 + 0.4 \times 0.05 = 0.032 \text{（全概率公式）}$$

$$P(\text{甲}|A) = \frac{0.6 \times 0.02}{0.032} = \frac{0.012}{0.032} = \frac{3}{8} = 0.375$$

**解读**：虽然甲厂次品率低，但它产量大，实际贡献的次品比例（3/8 = 37.5%）小于其产量比例（60%）。

**例2（医疗检测）**：某疾病发病率 0.1%，检测阳性的灵敏度（患病检阳性）= 99%，特异度（健康检阴性）= 95%（即健康检出假阳性率 = 5%）。已知检测结果为阳性，求真的患病概率。

设 $B_1$ = 患病，$B_2$ = 健康；$A$ = 检测阳性。

$$P(A) = 0.001 \times 0.99 + 0.999 \times 0.05 \approx 0.00099 + 0.04995 = 0.05094$$

$$P(B_1|A) = \frac{0.001 \times 0.99}{0.05094} \approx \frac{0.00099}{0.05094} \approx 0.0194 \approx 1.94\%$$

**解读**：即使检测阳性，实际患病概率仍只有约 2%！这反直觉的结果源于疾病发病率极低。此例说明"假阳性"问题在罕见病检测中非常重要。

**例3（两事件简化）**：$P(B) = 0.3$，$P(A|B) = 0.6$，$P(A|\bar{B}) = 0.1$，已知 $A$ 发生，求 $P(B|A)$。

$$P(A) = 0.3 \times 0.6 + 0.7 \times 0.1 = 0.18 + 0.07 = 0.25$$

$$P(B|A) = \frac{0.3 \times 0.6}{0.25} = \frac{0.18}{0.25} = 0.72$$

### 高中考试要点

高中阶段贝叶斯公式主要考查 $n = 2$ 或 $n = 3$ 的简单情形，步骤：
1. 列出各 $B_i$ 的先验概率和条件概率
2. 用全概率公式求 $P(A)$
3. 代入贝叶斯公式求 $P(B_j|A)$

### 易错点

1. **混淆 $P(A|B)$ 和 $P(B|A)$**：两者通常不相等，不能互换
2. **先验概率与后验概率的关系**：后验 = 先验 × 似然 / 全概率
3. **忘记用全概率公式计算 $P(A)$**：分母必须用全概率公式
4. **条件概率的方向**：$P(A|B_j)$ 是"已知 $B_j$ 发生时 $A$ 的概率"，不是反过来
