## 🎓 超几何分布

### 背景

**不放回抽样**：总体 $N$ 个，其中 $K$ 个为"特殊类"（如次品），从中取出 $n$ 个（不放回），$X$ = 取到的特殊类数量。

### 概率公式

$$\boxed{P(X=k) = \frac{C_K^k C_{N-K}^{n-k}}{C_N^n}}$$

其中 $k$ 的范围：$\max(0, n-(N-K)) \leq k \leq \min(n, K)$

称 $X$ 服从**超几何分布**，记作 $X \sim H(N, K, n)$（各参数含义：总数 $N$、特殊类数 $K$、抽取数 $n$）。

### 期望

$$E(X) = n \cdot \frac{K}{N}$$

**直观理解**：从总体中取 $n$ 个，特殊类占总体的比例是 $K/N$，期望取到的特殊类数就是 $n \cdot \frac{K}{N}$。

### 与二项分布的比较

| | 超几何分布 | 二项分布 |
|--|----------|---------|
| 抽样方式 | 不放回 | 有放回（或独立重复） |
| 每次概率 | 变化（依赖已取结果） | 固定（$p$ 不变） |
| 期望 | $n \cdot \frac{K}{N}$ | $np$ |
| 应用 | 质量抽检、摸球不放回 | 独立重复试验 |

**当 $N$ 很大时**，超几何分布近似于二项分布（$p \approx K/N$）。

### 典型例题

**例1（基础）**：袋中 4 红 3 白，不放回摸 3 个，$X$ = 红球数，求 $P(X=2)$ 和 $E(X)$。

$$P(X=2) = \frac{C_4^2 C_3^1}{C_7^3} = \frac{6 \times 3}{35} = \frac{18}{35}$$

$$E(X) = 3 \times \frac{4}{7} = \frac{12}{7} \approx 1.71$$

**例2（质量抽检）**：一批 10 件产品中有 3 件次品，随机检验 4 件，$X$ = 次品数。

$X \sim H(10, 3, 4)$，$P(X=k) = \dfrac{C_3^k C_7^{4-k}}{C_{10}^4}$

$$P(X=0) = \frac{C_3^0 C_7^4}{C_{10}^4} = \frac{35}{210} = \frac{1}{6}$$

$$P(X=1) = \frac{C_3^1 C_7^3}{C_{10}^4} = \frac{3 \times 35}{210} = \frac{1}{2}$$

$$E(X) = 4 \times \frac{3}{10} = \frac{6}{5} = 1.2 \text{ 件}$$

**例3（完整分布列）**：例2 中，写出 $X$ 的完整分布列。

$k$ 的范围：$0 \leq k \leq 3$（次品最多 3 件）

$$P(X=2) = \frac{C_3^2 C_7^2}{C_{10}^4} = \frac{3 \times 21}{210} = \frac{3}{10}$$

$$P(X=3) = \frac{C_3^3 C_7^1}{C_{10}^4} = \frac{7}{210} = \frac{1}{30}$$

验证：$\frac{1}{6} + \frac{1}{2} + \frac{3}{10} + \frac{1}{30} = \frac{5+15+9+1}{30} = 1 \checkmark$

### 易错点

1. **取值范围**：$k$ 不是从 0 到 $\min(n,K)$ 就完全正确，还需 $k \geq n-(N-K)$（非特殊类不够时）
2. **不放回 vs 放回**：题目说"不放回"用超几何分布，"有放回"用二项分布
3. **期望公式中的比例**：$K/N$ 是特殊类占总体的比例，$n$ 是抽取数
4. **组合数计算**：分子 $C_K^k C_{N-K}^{n-k}$ 两部分分别从特殊类和非特殊类中选
