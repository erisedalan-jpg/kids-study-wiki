## 🎓 随机变量函数

### 定义

若 $X$ 是随机变量，$g$ 是确定的函数，则 $Y = g(X)$ 也是随机变量，称为 **$X$ 的函数**。

$Y$ 的取值由 $X$ 的取值通过 $g$ 变换得到；$Y$ 的概率由 $X$ 的概率导出。

### 离散型随机变量的函数

若 $X$ 离散，取值 $x_1, x_2, \ldots$，则 $Y = g(X)$ 取值 $g(x_1), g(x_2), \ldots$

**注意**：若不同的 $x_i$ 对应相同的 $g(x_i)$，则需合并（将概率相加）。

**例**：$X$ 的分布列如下，求 $Y = X^2$ 的分布列。

| $X$ | -2 | -1 | 0 | 1 | 2 |
|-----|----|----|---|---|---|
| $P$ | 0.1 | 0.2 | 0.4 | 0.2 | 0.1 |

$Y = X^2$ 的取值：$4, 1, 0, 1, 4$，合并得 $Y \in \{0, 1, 4\}$

$$P(Y=0) = P(X=0) = 0.4$$
$$P(Y=1) = P(X=-1) + P(X=1) = 0.2 + 0.2 = 0.4$$
$$P(Y=4) = P(X=-2) + P(X=2) = 0.1 + 0.1 = 0.2$$

| $Y$ | 0 | 1 | 4 |
|-----|---|---|---|
| $P$ | 0.4 | 0.4 | 0.2 |

验证：$0.4 + 0.4 + 0.2 = 1 \checkmark$

### 期望的变换

**离散型**：

$$E(Y) = E(g(X)) = \sum_i g(x_i) p_i \quad \text{（直接用 $X$ 的分布列）}$$

无需先求 $Y$ 的分布列，可直接按 $X$ 的取值计算。

**例**：上例中，$E(Y) = E(X^2) = (-2)^2 \times 0.1 + (-1)^2 \times 0.2 + 0 \times 0.4 + 1^2 \times 0.2 + 2^2 \times 0.1$
$= 0.4 + 0.2 + 0 + 0.2 + 0.4 = 1.2$

### 线性变换（最重要的函数）

若 $Y = aX + b$（线性函数），则：

$$E(Y) = aE(X) + b$$

$$D(Y) = a^2 D(X)$$

**标准化变换**：$Z = \dfrac{X - \mu}{\sigma}$（其中 $\mu = E(X)$，$\sigma = \sqrt{D(X)}$），则：

$$E(Z) = 0, \quad D(Z) = 1$$

**正态分布的标准化**：若 $X \sim N(\mu, \sigma^2)$，$Z = \dfrac{X-\mu}{\sigma}$，则 $Z \sim N(0,1)$。

### 连续型随机变量的函数

若 $X$ 连续，$Y = g(X)$，则 $Y$ 的分布通过变量替换导出（高中阶段主要考查线性变换）：

$$\text{若 } X \sim N(\mu, \sigma^2), \; Y = aX + b, \; \text{则 } Y \sim N(a\mu + b, a^2\sigma^2)$$

**例**：$X \sim N(10, 4)$，$Y = 2X + 3$，则 $Y \sim N(2 \times 10 + 3, 2^2 \times 4) = N(23, 16)$。

### 典型例题

**例1（离散型函数）**：$X \sim B(3, 0.5)$，$Y = 2X - 1$，求 $E(Y)$ 和 $D(Y)$。

$E(X) = 3 \times 0.5 = 1.5$，$D(X) = 3 \times 0.5 \times 0.5 = 0.75$

$$E(Y) = 2 \times 1.5 - 1 = 2, \quad D(Y) = 4 \times 0.75 = 3$$

**例2（标准化）**：$X \sim N(5, 9)$（$\sigma = 3$），求 $P(X > 8)$。

$$P(X > 8) = P\left(\frac{X-5}{3} > 1\right) = P(Z > 1) = 1 - \Phi(1) \approx 1 - 0.8413 = 0.1587$$

**例3（绝对值函数）**：$X \sim N(0,1)$，$Y = |X|$，求 $P(Y \leq 2)$。

$$P(Y \leq 2) = P(|X| \leq 2) = P(-2 \leq X \leq 2) = \Phi(2) - \Phi(-2) \approx 0.9544$$

### 易错点

1. **合并相同取值**：$Y=g(X)$ 时，不同 $x_i$ 映射到同一 $y$ 时，概率相加
2. **$E(g(X)) \neq g(E(X))$**（非线性时）：$E(X^2) \neq [E(X)]^2$
3. **正态分布的线性变换仍正态**：但非线性变换（如 $X^2$）不再服从正态分布
4. **标准化中除以 $\sigma$ 不是 $\sigma^2$**：$Z = (X-\mu)/\sigma$，分母是标准差
