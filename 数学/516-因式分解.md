# 因式分解

> **一句话**：因式分解是把一个**多项式**变形为**几个整式相乘**形式的过程，是初中代数最重要的基本技能之一，是分式化简、解方程、二次根式运算的基础。
> **English**: *Factorization* is the process of expressing a polynomial as a product of integer expressions; one of the most fundamental algebraic skills in middle school, prerequisite for simplifying rational expressions, solving equations, and radical operations.

---

## 📚 给 6-12 岁

因式分解是"展开"的反过程：
- **展开**（乘法分配律）：3(x+2) = 3x + 6
- **因式分解**（反向）：3x + 6 = 3(x+2)

把"加法表达式"变成"乘法表达式"，能让计算和方程求解变简单。

## 🎓 给 12+

**核心方法**：
1. **提公因式法**（最基本）
   - 找出各项的公共因子，提到括号外
   - 例：6x² + 9x = 3x(2x + 3)
2. **公式法**
   - 平方差：a² − b² = (a+b)(a−b)
   - 完全平方：a² ± 2ab + b² = (a ± b)²
   - 立方和差（高中）：a³ ± b³ = (a ± b)(a² ∓ ab + b²)
3. **十字相乘法**
   - 用于二次三项式 ax² + bx + c
   - 例：x² + 5x + 6 = (x + 2)(x + 3)（拆 6 = 2×3，且 2+3 = 5）
4. **分组分解法**
   - 多项式无公共因子时，分组找公因子
   - 例：ax + ay + bx + by = a(x+y) + b(x+y) = (a+b)(x+y)

**注意事项**：
- 因式分解必须**分解到不能再分**为止
- 结果要写成**几个整式相乘**形式
- 不要漏掉公因式（先观察是否有公因子）
- 系数一般要求为整数（与"因式分解"区别于"因式定理"）

**应用场景**：
- 分式化简：$\frac{x^2-4}{x^2+4x+4} = \frac{(x+2)(x-2)}{(x+2)^2} = \frac{x-2}{x+2}$
- 解一元二次方程：x² − 5x + 6 = 0 → (x−2)(x−3) = 0 → x = 2 或 3
- 二次根式化简：根式内提完全平方因式
