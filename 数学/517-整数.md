# 整数

> **一句话**：整数是**正整数、零、负整数**的总称（…, −3, −2, −1, 0, 1, 2, 3, …），用集合符号 **ℤ** 表示，是有理数的子集，是数轴上间隔 1 的离散点。
> **English**: An *integer* is any number in the set {…, −3, −2, −1, 0, 1, 2, 3, …}, denoted by **ℤ** — comprising positive integers, zero, and negative integers; a discrete subset of rational numbers spaced one unit apart on the number line.

---

## 📚 给 6-12 岁

整数包括三类：
- **正整数**：1, 2, 3, …（也叫**自然数**中除 0 之外的部分）
- **0**
- **负整数**：−1, −2, −3, …

特点：
- 整数没有小数部分（10 是整数，10.5 不是）
- 整数没有最大也没有最小（无限延伸）
- 任意两个相邻整数相差 1

身边的整数：年龄、人数、楼层、温度（−5℃ 到 30℃）。

## 🎓 给 12+

**集合关系**：
- ℕ（自然数 = 非负整数）⊂ ℤ（整数）⊂ ℚ（有理数）⊂ ℝ（实数）⊂ ℂ（复数）
- ℕ = {0, 1, 2, 3, …}
- ℤ = ℕ ∪ {负整数} = {…, −2, −1, 0, 1, 2, …}
- ℤ⁺ = 正整数；ℤ⁻ = 负整数

**整数的运算**：
- **加减乘**：结果仍为整数（整数对加减乘**封闭**）
- **除法**：结果不一定是整数（5 ÷ 2 = 2.5 不是整数）
- **整除**：a÷b 商为整数，余 0，记 b | a
- **同号相乘得正，异号相乘得负**

**重要子类**：
- **奇数**：不能被 2 整除的整数（…, −3, −1, 1, 3, 5, …）
- **偶数**：能被 2 整除的整数（…, −2, 0, 2, 4, …）
- **质数（素数）**：大于 1，只能被 1 和自身整除（2, 3, 5, 7, 11, 13, …）
- **合数**：大于 1，除 1 和自身外还有其他正因数（4, 6, 8, 9, 10, …）
- **完全平方数**：可以表示为某个整数平方（1, 4, 9, 16, 25, …）

**整数性质**：
1. 整数有大小：可以比较（−5 < 0 < 3）
2. 任意两个整数都可以加减乘
3. 数轴上整数是离散的点（不是连续的线）
4. 任意整数都可以表示为 a = bq + r（带余除法），其中 0 ≤ r < |b|

**与有理数关系**：每个整数都是有理数（如 5 = 5/1），但有理数不一定是整数（如 1/2）。
