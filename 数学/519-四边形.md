# 四边形

> **一句话**：四边形是**由四条线段首尾相接组成的封闭图形**，**内角和为 360°**——按特殊性分为：一般四边形、平行四边形、矩形、菱形、正方形、梯形、筝形等。
> **English**: A *quadrilateral* is a closed plane figure formed by four line segments joined end to end; the sum of its interior angles is 360°; classified into general quadrilaterals, parallelograms, rectangles, rhombi, squares, trapezoids, kites, etc.

---

## 📚 给 6-12 岁

四边形：**四条边、四个角**。

最常见的几种：
- **正方形**：四条边一样长，四个角都是直角
- **长方形**：对边一样长，四个角都是直角
- **平行四边形**：两组对边都平行
- **梯形**：只有一组对边平行
- **菱形**：四条边一样长，但角不一定是直角

身边的四边形：课本（长方形）、骰子的面（正方形）、风筝（菱形/筝形）、屋顶截面（梯形）。

## 🎓 给 12+

**分类层次（从一般到特殊）**：
```
四边形
├── 任意四边形（无特殊性质）
├── 梯形（只一组对边平行）
│   ├── 一般梯形
│   ├── 等腰梯形（两腰相等）
│   └── 直角梯形（一腰垂直于底）
└── 平行四边形（两组对边平行）
    ├── 一般平行四边形
    ├── 矩形（一个角为直角）
    ├── 菱形（一组邻边相等）
    └── 正方形（既是矩形又是菱形）
```

**核心定理**：
- **内角和**：360°（任意四边形）
- **外角和**：360°（任意凸多边形）
- **平行四边形**：对边平行且相等；对角相等；对角线互相平分
- **矩形**：平行四边形 + 一个直角；对角线相等
- **菱形**：平行四边形 + 一组邻边相等；对角线互相垂直平分
- **正方形**：兼具矩形和菱形所有性质
- **梯形（等腰）**：两底角相等；对角线相等

**判定方法**：
| 图形 | 判定方法 |
|---|---|
| 平行四边形 | 两组对边平行；两组对边相等；一组对边平行且相等；对角线互相平分 |
| 矩形 | 平行四边形 + 一个直角 / 平行四边形 + 对角线相等 / 三个角是直角 |
| 菱形 | 平行四边形 + 一组邻边相等 / 平行四边形 + 对角线互相垂直 / 四条边相等 |
| 正方形 | 矩形 + 一组邻边相等 / 菱形 + 一个角是直角 |
| 等腰梯形 | 在同一底上的两个角相等 / 对角线相等 |

**面积公式**：
- 平行四边形：S = 底 × 高
- 矩形：S = 长 × 宽
- 菱形：S = ½ × 对角线乘积 = a² × sinα（边长 + 一角）
- 正方形：S = 边²
- 梯形：S = ½ × (上底 + 下底) × 高

**对角线性质**：
- 矩形的对角线相等（不一定垂直）
- 菱形的对角线垂直平分（不一定相等）
- 正方形的对角线既相等又垂直平分
