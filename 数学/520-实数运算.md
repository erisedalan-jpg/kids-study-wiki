# 实数运算

> **一句话**：实数运算是在**实数集合 ℝ**（含有理数和无理数）上进行的加、减、乘、除、乘方、开方运算——遵循**有理数运算法则**，但常涉及无理数（如 √2、π）的化简与近似计算。
> **English**: *Real number operations* are the four arithmetic operations plus exponentiation and root extraction performed on real numbers (rationals + irrationals); they follow the rules of rational arithmetic but often involve simplification and approximation of irrationals like √2 and π.

---

## 🎓 给 12+

**实数的范畴**：
- ℝ = ℚ（有理数）∪ I（无理数）
- 有理数：可表示为 p/q（p, q 整数，q ≠ 0）的数
- 无理数：不能写成 p/q 的数（如 π、√2、e）

**运算法则**（与有理数相同）：
1. **加法交换律**：a + b = b + a
2. **加法结合律**：(a + b) + c = a + (b + c)
3. **乘法交换律**：a × b = b × a
4. **乘法结合律**：(a × b) × c = a × (b × c)
5. **乘法分配律**：a × (b + c) = a × b + a × c
6. **加法逆元**：每个数 a 有唯一相反数 −a
7. **乘法逆元**：每个非零数 a 有唯一倒数 1/a

**运算优先级**（与有理数相同）：
1. 括号 → 乘方/开方 → 乘除 → 加减
2. 同级运算从左向右

**实数运算的特殊性**：

### 1. 含无理数的化简
- √8 = √(4×2) = 2√2
- √12 + √27 = 2√3 + 3√3 = 5√3
- (√2 + √3)² = 2 + 2√6 + 3 = 5 + 2√6

### 2. 分母有理化
- 1/√2 = √2/2
- 1/(√3 − 1) = (√3 + 1)/2

### 3. 近似计算
- 用近似值代替（π ≈ 3.14；√2 ≈ 1.414）
- 注意保留位数与有效数字
- 中间结果保留更多位，最后结果按要求取舍

**常见错误**：
- ❌ √(a + b) = √a + √b（错！）
- ✅ √(ab) = √a × √b（仅 a, b ≥ 0 时）
- ❌ √4 = ±2（√x 默认正根）
- ✅ √4 = 2；x² = 4 → x = ±2

**实数性质**：
- **稠密性**：任意两个实数之间有无穷多个实数
- **完备性**：每个有界递增数列都有极限（高中/大学数学）
- **有序性**：任意两个实数 a, b 都可比较 a < b、a = b、a > b
- **数轴**：实数与数轴上的点一一对应（无理数填补了"有理数留下的洞"）

**与"有理数运算"对比**：有理数运算结果一定是有理数；实数运算可能涉及无理数，结果不一定能写成 p/q 形式。
