# 自然数

> **一句话**：自然数是**用来表示物体个数、表示顺序的数**——0, 1, 2, 3, …（无限延伸），用集合符号 **ℕ** 表示，是最早出现、最基本的数。
> **English**: A *natural number* is a non-negative integer used for counting and ordering — 0, 1, 2, 3, … (infinite); denoted by **ℕ**, the most fundamental and earliest-appearing number type.

---

## 📚 给 6-12 岁

自然数就是**数数用的数**：
- **0**（什么都没有）
- **1**（一个）
- **2**（两个）
- **3**（三个）
- ……（一直数下去，没有最大）

特点：
- **从 0 开始**
- **永远是非负数**（没有负数）
- **永远是整数**（没有小数和分数）
- **可以无限大**（数到几亿几兆都还是自然数）

身边的自然数：年龄、人数、楼层数、页码、座位号。

## 🎓 给 12+

**集合关系**：
- ℕ ⊂ ℤ ⊂ ℚ ⊂ ℝ ⊂ ℂ
- 自然数是整数的非负部分

**两种约定**（不同教材、地区）：
- **中国大陆现行教材（2002 年起）**：ℕ = {0, 1, 2, 3, …}（含 0）
- **部分外文教材**：ℕ = {1, 2, 3, …}（不含 0），用 ℕ₀ 表示含 0 的版本

**运算封闭性**：
| 运算 | 自然数对其封闭？ |
|---|---|
| 加法 | ✅ 自然数 + 自然数 = 自然数 |
| 乘法 | ✅ 自然数 × 自然数 = 自然数 |
| 减法 | ❌ 5 − 3 = 2 是自然数，但 3 − 5 = −2 不是 |
| 除法 | ❌ 6 ÷ 2 = 3 是自然数，但 5 ÷ 2 = 2.5 不是 |

**自然数的重要子集**：
- **奇数**：1, 3, 5, 7, …
- **偶数**：0, 2, 4, 6, …
- **质数**：2, 3, 5, 7, 11, …（只能被 1 和自身整除，**至少有 2 个不同因数**）
- **合数**：4, 6, 8, 9, 10, …（除 1 和自身外还有其它因数）
- **完全平方数**：0, 1, 4, 9, 16, 25, …
- **完全立方数**：0, 1, 8, 27, 64, …

**特殊数**：
- **0**：既不是质数也不是合数；任何数加 0 不变；任何数乘 0 等于 0
- **1**：既不是质数也不是合数；是所有非零自然数的因数

**应用**：
- **基数**（个数）：3 个苹果
- **序数**（顺序）：第 3 个
- **编号**：身份证、座位号、页码
- **数学归纳法的基础**：归纳假设建立在自然数上

**数轴上的自然数**：
- 数轴正方向（含原点 0）的整数点
- 间隔 1 单位
- 离散（不连续）
