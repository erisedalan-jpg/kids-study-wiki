# 二次函数应用

> **一句话**：**二次函数 y = ax² + bx + c (a ≠ 0)** 在生活中广泛存在——**抛物运动、最大利润、面积最值、桥梁拱形**等都可以用它建模；解题核心是**列函数 → 找顶点 → 解最值**。
> **English**: *Applications of quadratic functions* are pervasive: projectile motion, profit maximization, area optimization, bridge arches, etc., can all be modeled by *y = ax² + bx + c*; the solution core is **set up the function → find the vertex → determine the extremum**.

---

## 🎓 给 12+

**二次函数核心**：
- 一般形式：y = ax² + bx + c (a ≠ 0)
- 顶点形式：y = a(x − h)² + k（顶点 (h, k)）
- 交点形式（已知零点）：y = a(x − x₁)(x − x₂)

**顶点坐标**：(−b/(2a), (4ac − b²)/(4a))
- a > 0 时为最低点（最小值）
- a < 0 时为最高点（最大值）

---

## 经典应用题型

### 1. 抛物运动（物理结合）
**例**：以初速度 v 抛出物体，h(t) = v·t − ½gt²，何时落地？最高点多高？
- 落地：h(t) = 0 → t = 0 或 t = 2v/g
- 最高点：t = v/g 时 h = v²/(2g)

### 2. 利润最大化（经济结合）
**例**：每件 60 元成本的商品，售价 x 元时月销量 (300 − x) 件，求最大利润售价。
- 利润 P(x) = (x − 60)(300 − x) = −x² + 360x − 18000
- 顶点 x = 180，最大利润 = 14400 元

### 3. 面积最值
**例**：用 100 米栅栏围矩形（一面靠墙），求面积最大的形状。
- 设宽 x，长 (100 − 2x)
- 面积 S(x) = x(100 − 2x) = −2x² + 100x
- 顶点 x = 25，最大面积 1250 m²，长 50 m，宽 25 m

### 4. 桥拱设计（几何结合）
**例**：抛物线形拱桥，跨度 20 m，最高点离水面 4 m，求水面 5 m 处拱高。
- 建坐标系，顶点 (0, 4)，过 (10, 0)
- y = −0.04x² + 4
- 5 m 处即 x = 5：y = −0.04×25 + 4 = 3 m

### 5. 运动轨迹
**例**：篮球以仰角 45° 投出，从 2 m 高度抛出，初速 8 m/s（取 g=10）。
- y = 2 + x − x²/16（x 为水平距离）
- 落地：x = 9.55 m
- 最高点：x = 8 m 时 y = 6 m

## 解题步骤

1. **审题**：识别变量，明确目标（最值/特定值）
2. **建模**：列出函数关系 y = ax² + bx + c
3. **简化**：合并同类项，找标准形
4. **求顶点**：用顶点公式或配方法
5. **判范围**：x 的实际取值范围（如 x > 0）
6. **核对单位**：长度、时间、货币单位
7. **检验合理性**：答案是否符合实际

---

## 关键技巧

- **配方**：y = a(x − h)² + k 直接读出顶点
- **对称性**：抛物线关于 x = h 对称，可利用对称简化计算
- **含参数**：用 a 的正负判断开口方向
- **二次函数 vs 二次方程**：方程 ax² + bx + c = 0 的根是函数图像与 x 轴交点的横坐标
