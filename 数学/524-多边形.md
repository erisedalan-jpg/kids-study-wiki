# 多边形

> **一句话**：**多边形**是由**三条或以上**线段首尾相接组成的封闭平面图形——按边数分为三角形、四边形、五边形、六边形…；按角分凸多边形、凹多边形；按对称性分正多边形、不规则多边形。
> **English**: A *polygon* is a closed plane figure formed by three or more line segments joined end to end; classified by number of sides (triangle, quadrilateral, pentagon, etc.), by angle (convex, concave), and by symmetry (regular, irregular).

---

## 📚 给 6-12 岁

多边形：**用直线围成的封闭图形**。
- 三条边 → 三角形
- 四条边 → 四边形（含正方形、长方形、平行四边形、梯形、菱形）
- 五条边 → 五边形
- 六条边 → 六边形（蜂巢的形状）
- 八条边 → 八边形
- ……

身边的多边形：道路标志（八边形停车标志）、建筑（六边形教室）、自然（雪花六边形）。

## 🎓 给 12+

**核心性质**：

### 1. 内角和定理
- n 边形内角和 = **(n − 2) × 180°**
- 三角形：(3−2)×180° = 180°
- 四边形：(4−2)×180° = 360°
- 五边形：540°；六边形：720°
- 推导：从一个顶点出发可画 n−3 条对角线，把 n 边形分成 n−2 个三角形

### 2. 外角和定理
- **任意凸多边形的外角和 = 360°**
- 与边数无关
- 直观理解：沿外周走一圈，方向变化总和 = 360°

### 3. 对角线条数
- n 边形对角线条数 = **n(n−3)/2**
- 三角形：0；四边形：2；五边形：5；六边形：9

### 4. 正多边形
- 所有边相等且所有内角相等
- **每个内角 = (n−2)×180° / n**
- 正三角形：60°；正方形：90°；正五边形：108°；正六边形：120°
- **每个外角 = 360° / n**

**分类**：
| 维度 | 类型 |
|---|---|
| 凸/凹 | 凸多边形（所有内角 < 180°）/ 凹多边形（有内角 > 180°） |
| 边数 | 三角形 / 四边形 / 五边形 / 六边形 / … |
| 规则性 | 正多边形（边角都相等）/ 非正多边形 |
| 对称性 | 轴对称 / 中心对称 / 既轴又中心对称 |

**周长与面积**：
- **周长** = 各边长度之和
- **正多边形周长** = n × 边长
- **三角形面积** = ½ × 底 × 高
- **四边形面积**：见 [[四边形]] 各类
- **正多边形面积**（外接圆半径 R）= ½ × n × R² × sin(2π/n)

**镶嵌（铺满平面）**：
- 仅 **正三角形、正方形、正六边形**能单独镶嵌平面（内角整除 360°）
- 正五边形不能（因为 108° 不整除 360°）
- 蜂巢用六边形——周长最小、容积最大

**与"圆"关系**：边数无穷多时多边形趋近于圆（割圆术求 π，刘徽、祖冲之都用此法）。
