# 奇数与偶数

> **一句话**：**整数**按能否被 2 整除分为**偶数**（能整除，2k）和**奇数**（不能整除，2k+1）；偶数加偶数 / 奇数加奇数都是偶数，奇数加偶数是奇数。
> **English**: Integers are classified by divisibility by 2: *even numbers* (divisible by 2, of form 2k) and *odd numbers* (not divisible, of form 2k+1); even+even and odd+odd are even, even+odd is odd.

---

## 📚 给 6-12 岁

**偶数**（双数）：能被 2 整除——0, 2, 4, 6, 8, 10, …
**奇数**（单数）：不能被 2 整除——1, 3, 5, 7, 9, 11, …

判断方法：
- 看个位：个位是 **0/2/4/6/8** → 偶数
- 看个位：个位是 **1/3/5/7/9** → 奇数

加减规律：
- 偶 + 偶 = 偶（4 + 6 = 10）
- 奇 + 奇 = 偶（3 + 5 = 8）
- 偶 + 奇 = 奇（4 + 5 = 9）

身边的：座位号、楼层、日期。

## 🎓 给 12+

**正式定义**：
- 整数 n 是**偶数**当且仅当存在整数 k 使 n = 2k
- 整数 n 是**奇数**当且仅当存在整数 k 使 n = 2k + 1
- 0 是偶数（最常见的盲点：0 = 2×0）

**集合表示**：
- 偶数集：{…, −4, −2, 0, 2, 4, …}
- 奇数集：{…, −3, −1, 1, 3, …}
- 整数 = 偶数 ∪ 奇数

**运算性质**：
| 运算 | 结果 |
|---|---|
| 偶 ± 偶 | 偶 |
| 奇 ± 奇 | 偶 |
| 偶 ± 奇 | 奇 |
| 偶 × 偶 | 偶 |
| 奇 × 奇 | 奇 |
| 偶 × 奇 | 偶 |

**奇偶性应用**：
1. **判断结果是否合理**：3+5+7=15（奇数个奇数相加，结果为奇）
2. **数学证明**：反证法、奇偶性矛盾
3. **解方程**：判断解是否存在（如 x²+1=2y 中考虑奇偶）
4. **趣题**：6 个奇数加起来能否等于 100？答：不能（6 个奇数之和必为偶）

**奇偶性扩展**：
- **奇函数**：f(−x) = −f(x)，图像关于原点对称
- **偶函数**：f(−x) = f(x)，图像关于 y 轴对称
- 注意区分：奇/偶**数**和奇/偶**函数**是不同概念

**在编程**：
- 判断奇偶：`n % 2 == 0` 偶 / `n % 2 == 1` 奇
- 位运算：`n & 1 == 0` 偶 / `n & 1 == 1` 奇
