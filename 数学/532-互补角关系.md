# 互补角关系

> **一句话**：若两个角的度数之和为180°（或π rad），则这两个角互为**补角**，其相互关联称为互补角关系。
> **English**: Two angles whose measures sum to 180° (or π rad) are *supplementary angles*, defining the supplementary relationship.

## 🎓 考点精讲

**定义/原理**  
设角 $α$ 与 $β$ 满足 $α + β = 180^\circ$（弧度制下 $α + β = π$），则称 $α$ 与 $β$ 互补。  
核心性质：**同角或等角的补角相等**。即若 $$\angle A + \angle B = 180^\circ,\quad \angle A + \angle C = 180^\circ\ \Longrightarrow\ \angle B = \angle C.$$

**关键方法**  
高考中利用互补角关系主要遵循两条路径：  
1. **几何路径**：在平行线、三角形、圆等图形中识别互补关系，建立等式求解未知角。常见依据：两直线平行，同旁内角互补；圆内接四边形对角互补；一个三角形的外角等于不相邻两内角之和，该外角与相邻内角互补；  
2. **三角路径**：使用诱导公式将钝角或任意角的三角函数化为锐角函数，核心恒等式为  
   $$\begin{aligned} \sin(π - α) = \sin α,\quad \cos(π - α) = -\cos α,\quad \tan(π - α) = -\tan α. \end{aligned}$$  
   解题时需先将角写成 $π - α$ 的形式，再套用公式化简求值或证明。

**典型应用**  
- 已知 $\triangle ABC$ 中 $A = 120^\circ$，求 $\sin B \sin C$ 的取值范围。利用 $B + C = 60^\circ$ 及补角关系 $\sin B = \sin(180^\circ - B?) $ (注意：不是补角，此处用三角形内角和，但可联系 $B = 180^\circ - (A+C)$ 等转化)。更直接的是，求 $\sin B \sin C$ 时，可先将 $C = 60^\circ - B$ 代入，再利差角公式；  
- 解析几何中，若直线倾斜角为 $α$，则其补角 $π - α$ 对应另一方向，有助于讨论两直线的夹角与斜率。但易错点是垂直条件为互余，不可混淆。

## ⚠️ 高频易错点

- **互补与互余混淆**：互补和是180°，互余是90°。三角公式中 $\sin(\frac{π}{2} - α) = \cos α$ 属于互余，而 $\sin(π - α) = \sin α$ 属于互补，高考常混用导致符号错误。
- **正弦相等、余弦相反易忘**：互补的两个角正弦值相等，余弦值互为相反数。学生常误以为 $\cos(π - α) = \cos α$，在求值、解方程中丢解或增解。
- **忽略平行前提**：同旁内角互补仅在两条直线平行时成立。题目未给出平行条件就直接使用“同旁内角互补”是经典错误。
- **单位不一致**：题目可能混用角度与弧度，如 $π$ 与 $180^\circ$ 等价，但若将 $π$ 当作 $180$ 而未加单位，计算 $π + α = 180^\circ$ 之类会出错。
