# 函数恒成立问题

> **一句话**：对含参数函数或不等式，在给定区间上使其恒成立的参数满足的充要条件问题。
> **English**: The problem of finding parameter values that ensure a given functional inequality or equation remains true for all values in a specified domain.

## 🎓 考点精讲

**定义/原理**  
设函数 $f(x;a)$ 中含有参数 $a$，$D$ 是给定区间。若 $\forall x \in D$，不等式 $f(x;a) > 0$（或 $\ge 0$ 等）恒成立，则转化为参数 $a$ 必须满足的条件。核心原理是命题逻辑与函数最值的等价转化：

- $f(x) > 0$ 在 $D$ 上恒成立 $\Leftrightarrow$ $\min_{x\in D} f(x) > 0$  
- $f(x) < 0$ 在 $D$ 上恒成立 $\Leftrightarrow$ $\max_{x\in D} f(x) < 0$  

将含参恒成立问题转化为求函数在区间上的最值，从而解出参数范围。

**关键方法**  
1. **分离参数法**：将不等式变形为 $a > g(x)$（或 $a < g(x)$）的形式，则 $a > g(x)$ 恒成立 $\Leftrightarrow$ $a > g(x)_{\max}$；$a < g(x)$ 恒成立 $\Leftrightarrow$ $a < g(x)_{\min}$。  
2. **直接构造函数法**：保留参数，构造函数 $h(x) = f(x;a)$，利用导数求其在 $D$ 上的最小值（或最大值），令最值满足符号要求解参数。  
3. **数形结合法**：将不等式两边看作两个函数，利用图像位置关系（如直线与二次曲线）列出关于参数的约束。  
4. **区间端点分析**：当区间一端趋于无穷或函数单调性明确时，可由极限与端点值得到参数的必要条件，再验证充分性。

**典型应用**  
高考常以选择、填空或解答题出现，设问如：“已知 $f(x) = x^2 + ax + 1$，若 $f(x) > 0$ 对任意 $x \in \mathbb{R}$ 恒成立，求 $a$ 的取值范围”。标准解法：二次函数开口向上，只需判别式 $\Delta = a^2 - 4 < 0 \Rightarrow -2 < a < 2$。  
另一常见情境为含指数、对数的超越函数，如 “已知 $e^x - ax > 1$ 对 $x>0$ 恒成立，求 $a$ 的取值范围”。分离参数得 $a < \frac{e^x - 1}{x}$，构造函数 $g(x) = \frac{e^x - 1}{x}$，用导数求出 $x>0$ 时 $g(x)_{\min}$（在 $x=1$ 处取得），从而得 $a < e-1$。

## ⚠️ 高频易错点

- **定义域被忽略**：分离参数后求最值时，未注意 $x$ 的取值范围是否开区间，导致最值不可达时遗漏等号。例如 $a < g(x)$ 在 $[1,2)$ 上恒成立，若 $\min g(x)$ 在端点取得但端点不取到，则 $a$ 应 $\le$ 下确界而非直接取 $\min$。  
- **分离参数时不讨论系数为零**：变形时将参数作为除数，未讨论该系数为 $0$ 的特殊情形，导致解集漏解或多解。  
- **混淆恒成立与存在性**：将 “恒成立” 的条件误用为 “存在一个 $x$ 使不等式成立”，错误地使用 $\max$ 或 $\min$。  
- **直接构造函数后求导未讨论端点和不可导点**：仅用导数为零的点判断最值，忽视区间端点处函数值或无穷趋势，造成参数范围错误。
