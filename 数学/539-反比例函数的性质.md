# 反比例函数的性质

> **一句话**：反比例函数是形如 \(y = \dfrac{k}{x}\,(k \neq 0)\) 的函数，图像为双曲线，具有中心对称性、渐近线以及在各象限内的单调性。
> **English**: An inverse proportional function is a function of the form \(y = \frac{k}{x}\,(k \neq 0)\), whose graph is a hyperbola featuring central symmetry, asymptotes, and monotonicity restricted to each quadrant.

## 🎓 考点精讲

**定义与基本性质**  
形如 \(y = \dfrac{k}{x}\;(k \neq 0)\) 的函数称为反比例函数，定义域为 \(\{x \mid x \neq 0\}\)，值域为 \(\{y \mid y \neq 0\}\)。其图像是双曲线，以坐标轴为渐近线：  
- 当 \(k > 0\) 时，图像位于第一、三象限，在每一象限内 \(y\) 随 \(x\) 的增大而减小；  
- 当 \(k < 0\) 时，图像位于第二、四象限，在每一象限内 \(y\) 随 \(x\) 的增大而增大。  

对称性：图像关于原点中心对称，同时也是轴对称图形（对称轴为直线 \(y = x\) 和 \(y = -x\)）。  
\(|k|\) 的几何意义：过图像上任一点向 \(x\) 轴、\(y\) 轴作垂线，所得矩形面积为 \(|k|\)；以此点与原点为顶点的直角三角形面积为 \(\dfrac{|k|}{2}\)。

**关键方法**  
- **待定系数法求解析式**：设 \(y = \frac{k}{x}\)，只需一组非零对应值或一个点即可确定 \(k\)。  
- **函数值比较**：若两点在同一分支可直接用增减性；若跨象限需结合图像判断。  
- **面积问题**：将所求图形转化为与坐标轴相关的矩形或直角三角形，运用 \(|k|\) 的几何意义求解。  
- **与一次函数综合**：联立求交点坐标；利用图像比较函数值大小，写出不等式解集（注意 \(x \neq 0\)）；构造方程求参数。

**典型应用**  
高考中常见选择题考查图像所在象限、增减性判断及利用 \(|k|\) 求面积；解答题常与一次函数综合，如：已知反比例函数 \(y_1 = \frac{m}{x}\) 与一次函数 \(y_2 = kx + b\) 图像交于 \(A(-1, n)\) 和 \(B(2, -2)\)，要求解析式、写出 \(y_1 \ge y_2\) 的解集、计算三角形面积等。解题需先代入点求参数，再结合图像与分象限性质求解。

## ⚠️ 高频易错点

- **忽视“在每个象限内”**：误认为 \(k>0\) 时 \(y\) 随 \(x\) 增大而减小在全体定义域成立，实际上跨象限时不具有单调性。  
- **混淆 \(|k|\) 的几何意义**：矩形面积等于 \(|k|\)，而与坐标轴围成的直角三角形面积为 \(\dfrac{|k|}{2}\)，不可直接用 \(|k|\) 计算三角形面积。  
- **比较函数值忽略象限**：利用增减性比较两个 \(y\) 值时，未判断对应的 \(x\) 是否在同一象限内，导致符号错误。  
- **漏掉分母不为零**：写不等式解集或讨论参数范围时，忘记剔除 \(x = 0\)，导致解集不严谨。
