# 并集

> **一句话**：由所有属于集合A或属于集合B的元素构成的集合，记作 A ∪ B。
> **English**: The union of two sets A and B is the set of elements that belong to A, to B, or to both.

## 🎓 考点精讲

- **定义/原理**  
  对于任意两个集合 A、B，由所有属于 A **或** 属于 B 的元素所组成的集合称为 A 与 B 的并集，记作  
  $$
  A \cup B = \{\,x \mid x \in A \text{ 或 } x \in B \,\}.
  $$  
  并集满足以下算律：交换律 $A\cup B = B\cup A$，结合律 $(A\cup B)\cup C = A\cup(B\cup C)$，幂等律 $A\cup A = A$，零元律 $A\cup\emptyset = A$，以及与全集的并 $A\cup U = U$。利用文氏图可直观表示为两个圆圈的合并区域。

- **关键方法**  
  1. **元素列举法**：对于有限集，列出所有元素并通过去重（保持互异性）得到并集。  
  2. **数轴分析法**：对于用不等式或区间表示的连续数集，把几个集合在数轴上画出，取所有被覆盖的区间，端点处注意开闭符号。  
  3. **性质转化法**：在综合问题中，常把求参数范围归结为集合包含关系，例如 $A \cup B = B \iff A \subseteq B$，进而转为子集问题求解。

- **典型应用**  
  高考常以选择题或填空题出现，直接考查集合运算，或结合不等式、函数定义域、方程解集等。  
  **场景 1**：已知 $A=\{x \mid x^2 -5x +6 >0\}$，$B=\{x \mid 2<x<5\}$，求 $A\cup B$。先解二次不等式得 $A = (-\infty,2)\cup(3,+\infty)$，再与 $B=(2,5)$ 取并集，结果为 $(-\infty,2)\cup(2,+\infty)$。  
  **场景 2**：已知集合 $A=\{1,2,a\}$，$B=\{2,3\}$，若 $A\cup B=\{1,2,3,4\}$，求 $a$ 的值。由并集元素倒推，可知 $a$ 必须为4，同时要排除与已有元素重复引起的矛盾。

## ⚠️ 高频易错点

- 忘记集合元素的**互异性**：并集中同一元素只出现一次，如 $\{1,2\}\cup\{2,3\}=\{1,2,3\}$，不能写成 $\{1,2,2,3\}$。  
- 混淆并集与交集的符号与逻辑：“或”对应并集，“且”对应交集，部分题目将“和”等同“且”，导致错选交集。  
- 数轴求并集时忽略**端点开闭**：某些区间端点不在集合内，并入后需保持原开闭，如 $(-\infty,2]$ 与 $(2,5)$ 的并集是 $(-\infty,5)$，而不是 $(-\infty,5]$，除非包含该点。  
- 空集遗漏：当讨论 $A\cup B=A$ 等条件时，容易忽略 $B=\emptyset$ 的情形，导致参数范围不全。
