# 数据推理

> **一句话**：数据推理是根据样本数据，运用统计原理对总体特征或变量关系进行推断的思维过程。
> **English**: Data reasoning is the process of making inferences about population characteristics or relationships between variables based on sample data using statistical principles.

## 🎓 考点精讲

**定义/原理**：数据推理核心是**由部分推断整体**，其理论基础为概率论与统计学。在高考中，常见的数据推理模型包括：用样本数字特征（均值 $\bar{x} = \frac{1}{n}\sum_{i=1}^n x_i$，方差 $s^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2$）估计总体特征；通过线性回归方程 $\hat{y} = \hat{b}x + \hat{a}$（$\hat{b} = \frac{\sum_{i=1}^n (x_i - \bar{x})(y_i - \bar{y})}{\sum_{i=1}^n (x_i - \bar{x})^2}$）预测变量关系；以及利用独立性检验的 $K^2$ 统计量 $K^2 = \frac{n(ad-bc)^2}{(a+b)(c+d)(a+c)(b+d)}$ 判断两个分类变量是否有关。

**关键方法**：数据推理的一般步骤：① 明确研究问题；② 收集或整理样本数据（频率分布表、列联表等）；③ 选择恰当的统计模型；④ 计算统计量并对照临界值或进行拟合优度检验；⑤ 得出有概率保证的结论。例如，在进行独立性检验时，需先提出假设 $H_0$：两变量无关，计算 $K^2$ 后与临界值比较，若 $K^2 \geq k_0$，则有 $1-\alpha$ 的把握拒绝 $H_0$。

**典型应用**：高考常以解答题形式考查，设问方式有：“根据数据判断两个分类变量是否有关系”、“求线性回归方程并预测”、“用样本频率估计总体概率并决策”。例如，给出某市空气质量与是否下雨的列联表，要求判断空气质量与下雨是否有关；或者给出居民收入与消费数据，求回归方程并预测收入增加时消费的变化。

## ⚠️ 高频易错点

- 将样本的统计量（如平均数、回归系数）直接当作总体的确定值，忽略抽样误差和置信水平。
- 独立性检验中错误理解“有把握”与临界值的对应关系，如 $K^2$ 略大于 $2.706$ 时，结论是“有 $90\%$ 把握认为有关”而非“两者一定有关”。
- 回归分析中混淆相关关系与因果关系，误认为相关性即因果，且忽略残差分析，直接用回归方程远距离预测。
- 用频率代替概率时，未注意大量重复试验的前提，将小样本频率当作概率进行决策推理。
