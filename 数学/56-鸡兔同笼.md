# 鸡兔同笼

> **一句话**：笼子里有鸡和兔，知道头的总数和脚的总数，求各有几只——用"假设全是一种"的方法来破解。
> **English**: A classic puzzle: given the total number of heads and legs of chickens (2 legs) and rabbits (4 legs), find how many of each there are.

---

## 🧒 给 3-6 岁（共读版）

笼子里有一些小鸡和小兔子住在一起。

小鸡有 **2 条腿**，小兔有 **4 条腿**，它们的头数都是 **1 个**。

老师说："笼子里有 **5 个头**，**14 条腿**。"

你能猜猜有几只鸡、几只兔吗？

（答案：3 只鸡 + 2 只兔 = 5 个头；3×2 + 2×4 = 6 + 8 = 14 条腿 ✓）

**共读小活动：** 画一画——先画 5 个圆圈（代表头），再试试给每个圆圈画 2 条腿（全是鸡），数数腿够不够；然后把一些换成 4 条腿的兔，直到腿的数目刚好是 14。

## 📚 给 6-12 岁（自读版）

### 经典原题

> **今有雉（pheasant，野鸡）兔同笼，上有三十五头，下有九十四足，问雉兔各几何？**
> ——《孙子算经》（约3-5世纪）

翻译：笼子里有鸡和兔，共有 **35 个头**，**94 条腿**，问鸡和兔各有几只？

### 方法一：列表法（逐一枚举）

从两种极端情况出发，逐步缩小范围：

| 假设鸡的只数 | 兔的只数 | 总腿数 | 与 94 的差 |
|-------------|---------|--------|-----------|
| 35（全是鸡） | 0 | 35×2=70 | 少 24 |
| 30 | 5 | 30×2+5×4=80 | 少 14 |
| 25 | 10 | 25×2+10×4=90 | 少 4 |
| 23 | 12 | 23×2+12×4=46+48=94 | **刚好！** |
| 20 | 15 | 20×2+15×4=100 | 多 6 |

答案：鸡 **23** 只，兔 **12** 只。

列表法的优点是直观，不需要方程；缺点是数字大时要试很多次。

### 方法二：假设法（核心方法）

**假设全是鸡：**

如果 35 只全是鸡，共有腿：35 × 2 = **70 条**

实际有 94 条，比假设多：94 - 70 = **24 条**

每把一只鸡换成一只兔，腿数增加：4 - 2 = **2 条**

需要换 24 ÷ 2 = **12 次** → 有 **12 只兔**

鸡的只数：35 - 12 = **23 只**

**验证：** 23 × 2 + 12 × 4 = 46 + 48 = 94 ✓

**换方向（假设全是兔）：**

如果 35 只全是兔，共有腿：35 × 4 = **140 条**

实际只有 94 条，比假设少：140 - 94 = **46 条**

每把一只兔换成一只鸡，腿数减少：4 - 2 = **2 条**

需要换 46 ÷ 2 = **23 次** → 有 **23 只鸡**

兔的只数：35 - 23 = **12 只**

### 假设法的逻辑

假设法的核心是"**替换产生差额**"：

```
全是鸡时：腿数 = 35 × 2 = 70（比实际少 24）
每换一只鸡为兔：腿数 +2（兔比鸡多 2 条腿）
换 x 次后：70 + 2x = 94
→ 2x = 24
→ x = 12（兔的数量）
```

这正是初中方程的思维雏形！

### 变形题练习

**题 1（增加难度）：** 停车场有轿车和摩托车，共 30 辆，共 96 个轮子。（轿车 4 轮，摩托车 2 轮）

假设全是轿车：30 × 4 = 120 个轮子，比实际多 24。
每换一辆轿车为摩托车，轮子减少 4 - 2 = 2 个。
需换 24 ÷ 2 = 12 辆 → **12 辆摩托车，18 辆轿车**

**题 2（蜘蛛和蜻蜓）：** 共 8 只，共 54 条腿。（蜘蛛 8 腿，蜻蜓 6 腿）

假设全是蜘蛛：8 × 8 = 64 条腿，比实际多 10。
每换一只蜘蛛为蜻蜓，腿数减少 8 - 6 = 2 条。
需换 10 ÷ 2 = 5 只 → **5 只蜻蜓，3 只蜘蛛**

## 🎓 给 12+（进阶版）

### 二元一次方程组（初中正式解法）

设鸡有 x 只，兔有 y 只，建立方程组：

$$\begin{cases} x + y = 35 & \text{（头数）} \\ 2x + 4y = 94 & \text{（腿数）} \end{cases}$$

**消元法：** 第一个方程 × 2：

$$\begin{cases} 2x + 2y = 70 \\ 2x + 4y = 94 \end{cases}$$

两式相减：$2y = 24$，得 $y = 12$；代入第一式：$x = 23$。

这正是"假设法"的代数版本——假设全是鸡（×2）相当于两边乘以 2。

### 图解：两方程的几何意义

方程组的解是两条直线的交点：
- $x + y = 35$：一条斜率为 -1 的直线
- $2x + 4y = 94$（化简为 $x + 2y = 47$）：另一条斜率为 -1/2 的直线

两线相交于点 (23, 12)，即答案。

```python
import numpy as np
import matplotlib.pyplot as plt

# 解方程组
A = np.array([[1, 1], [2, 4]])
b = np.array([35, 94])
solution = np.linalg.solve(A, b)
print(f"鸡: {solution[0]:.0f} 只, 兔: {solution[1]:.0f} 只")

# 画两条直线
x = np.linspace(0, 40, 100)
y1 = 35 - x                  # x + y = 35
y2 = (94 - 2*x) / 4          # 2x + 4y = 94

plt.figure(figsize=(7, 5))
plt.plot(x, y1, label='x + y = 35 (头数)', color='blue')
plt.plot(x, y2, label='2x + 4y = 94 (腿数)', color='red')
plt.scatter(*solution, color='green', zorder=5, s=100, label=f'解: ({solution[0]:.0f}, {solution[1]:.0f})')
plt.xlabel('鸡的只数 x'); plt.ylabel('兔的只数 y')
plt.xlim(0, 40); plt.ylim(0, 40)
plt.legend(); plt.grid(True)
plt.title('鸡兔同笼——方程组图解')
plt.tight_layout(); plt.show()
```

### 推广：线性规划视角

当条件不止两个（如加入笼子数量限制），问题变为**整数线性规划（ILP）**：

```python
from scipy.optimize import linprog

# 最小化目标：可以是总费用或任意目标
# 约束：x + y = 35, 2x + 4y = 94, x >= 0, y >= 0
# (等式约束在 linprog 中用 A_eq, b_eq 表达)
A_eq = [[1, 1], [2, 4]]
b_eq = [35, 94]
bounds = [(0, None), (0, None)]
res = linprog([0, 0], A_eq=A_eq, b_eq=b_eq, bounds=bounds)
print(res.x)  # [23. 12.]
```

### 历史背景

《孙子算经》是中国南北朝时期（约公元3-5世纪）的数学典籍，其中的鸡兔同笼问题是**现存最早的二元线性方程组问题之一**。

日本江户时代数学家将此类问题称为"龟鹤算"（龟 4 腿，鹤 2 腿），至今仍是日本小学数学的经典内容。（**待家长核对日本教材细节**）

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么要假设全是鸡？"** → 可以说："因为假设一个极端，就能计算出'应该有多少腿'，和实际一对比，就知道差了多少。差的部分一定是因为有兔——每只兔比鸡多 2 条腿，差多少腿就有多少÷2 只兔。"

- 孩子可能问 **"列表法和假设法哪个好？"** → 可以说："列表法更直观，适合数字小时；假设法像公式，一步算出，更高效。等到初中，你会学方程组，那是更正式的工具，和假设法背后的想法完全一样。"

- 孩子可能问 **"这道题是哪里来的？"** → 可以说："这道题在大约 1500 年前的中国数学书《孙子算经》里就有了！古人当时没有方程，就是靠'假设'和'推理'解题。日本人把同样的问题叫'龟鹤算'，一直到今天还在教。"
