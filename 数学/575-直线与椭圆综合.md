# 直线与椭圆综合

> **一句话**：研究直线与椭圆的位置关系，通过联立方程、判别式、韦达定理解决弦长、中点弦、定点定值与范围最值等问题。
> **English**: The study of the positional relationship between a line and an ellipse, using simultaneous equations, discriminant, and Vieta’s formulas to solve problems involving chord length, midpoint chords, fixed points/values, and range/extremum.

## 🎓 考点精讲

**定义/原理**：直线与椭圆的位置关系分为相交、相切、相离。设直线 $l: y = kx + m$（斜率存在时），椭圆 $C: \frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$，联立消去 $y$ 得关于 $x$ 的一元二次方程 $Ax^2 + Bx + C = 0$。其判别式 $\Delta = B^2 - 4AC$ 决定位置关系：$\Delta > 0$ 相交，$\Delta = 0$ 相切，$\Delta < 0$ 相离。涉及弦长时，设交点 $P(x_1, y_1), Q(x_2, y_2)$，弦长公式为 $|PQ| = \sqrt{1 + k^2}|x_1 - x_2| = \sqrt{1 + k^2}\frac{\sqrt{\Delta}}{|A|}$，或对纵坐标也用类似形式。

**关键方法**：
1. **联立与韦达定理**：将直线方程代入椭圆方程，整理成关于 $x$（或 $y$）的二次方程，直接写出 $x_1 + x_2$ 与 $x_1x_2$，这是解决弦长、中点、向量垂直（$x_1x_2 + y_1y_2 = 0$）等问题的基础。
2. **点差法（中点弦问题）**：已知弦中点 $M(x_0, y_0)$，设弦端点 $A(x_1, y_1), B(x_2, y_2)$，代入椭圆方程作差，利用中点坐标可得 $\frac{y_1 - y_2}{x_1 - x_2} = -\frac{b^2x_0}{a^2y_0}$（当 $y_0 \neq 0$），从而直接求出弦所在直线斜率，避免联立。但需验证所求直线是否与椭圆有两个交点（即在椭圆内部）。
3. **参数方程与几何意义**：椭圆参数方程 $x = a\cos\theta, y = b\sin\theta$ 常用来处理与距离、面积有关的最值问题。

**典型应用**：高考中常见设问：①求相交弦长（已知斜率或点）；②已知弦中点求直线方程（用点差法或联立韦达）；③以弦为直径的圆过定点，转化为垂直关系（向量数量积为0）利用韦达定理求解；④求三角形面积的最值或定点定值问题，结合弦长公式和点到直线距离公式，通过函数或不等式求最值。

## ⚠️ 高频易错点

- 忽略直线斜率不存在的情况（如竖直直线 $x = x_0$），未单独讨论，导致漏解。
- 使用点差法求中点弦方程后，未验证该直线与椭圆实际相交（需确保中点确实在椭圆内部），可能出现“增解”。
- 联立方程后整理二次方程时，不注意二次项系数是否为零（如直线与椭圆对称轴平行时的特殊情况），直接将判别式应用于一次方程导致错误。
- 在弦长公式中混淆 $\sqrt{1+k^2}$ 与 $\sqrt{1+\frac{1}{k^2}}$ 的使用场景，或忘记弦长公式中 $\frac{\sqrt{\Delta}}{|A|}$ 的绝对值。
