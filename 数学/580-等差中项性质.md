# 等差中项性质

> **一句话**：若三个数 $a, A, b$ 成等差数列，则 $A$ 叫做 $a$ 与 $b$ 的等差中项，且满足 $2A = a + b$。  
> **English**: If three numbers $a, A, b$ form an arithmetic sequence, then $A$ is the arithmetic mean of $a$ and $b$, and $2A = a + b$.

## 🎓 考点精讲

**定义与原理**  
在等差数列 $\{a_n\}$ 中，从第二项起，每一项都是它的前后两项的等差中项：  
$$2a_n = a_{n-1} + a_{n+1} \quad (n \geq 2).$$  
更一般的推广：若 $m + n = 2p$（$m,n,p \in \mathbb{N}^*$），则 $a_m + a_n = 2a_p$。这一性质直接源于等差数列任意两项的对称性，是等差中项的核心推广。

**关键方法**  
解决等差中项相关问题通常遵循以下步骤：  
1. 明确数列（或三个数）是否为等差数列，或由条件确认三项成等差；  
2. 根据下标和的关系建立等式，例如已知 $a_3 + a_9 = 16$，由 $3+9 = 2 \times 6$ 得 $2a_6 = 16$，即 $a_6 = 8$；  
3. 列方程并求解未知项或参数。该方法常与通项公式、前 $n$ 项和结合，用以简化运算。

**典型应用**  
高考中常见设问形式：  
- **求特定项**：如“在等差数列 $\{a_n\}$ 中，$a_2 + a_8 = 10$，求 $a_5$”直接使用 $2a_5 = a_2 + a_8$ 得 $a_5 = 5$。  
- **证明成等差**：已知 $2a_{n+1} = a_n + a_{n+2}$ 对任意 $n$ 成立，则可判定 $\{a_n\}$ 为等差数列。  
- **三项等差判断**：判断 $x, y, z$ 是否成等差，只需验证 $2y = x + z$ 是否成立。

## ⚠️ 高频易错点

- **误用于非等差数列**：见到三个数 $a, b, c$ 就列式 $2b = a + c$，却未事先确认它们是否成等差数列。
- **下标关系不对称误用**：未注意 $m+n$ 必须为偶数才能推出 $a_{\frac{m+n}{2}}$，错误认为 $a_1 + a_4 = 2a_{2.5}$，下标必须为正整数。
- **与等比中项混淆**：将等差中项公式 $2A = a + b$ 误写为 $A^2 = ab$（等比中项），导致概念错误。
- **局部成立误判全数列**：仅凭 $a_1 + a_3 = 2a_2$ 就断言整个数列为等差数列，而等差数列的判定需要定义对任意相邻三项成立或通项满足常数差。
