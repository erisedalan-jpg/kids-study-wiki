# 绝对值函数

> **一句话**：绝对值函数是一种用 $|x|$ 定义的分段函数，本质是数轴上点到原点的距离，其图像以 $y$ 轴为对称轴，在 $x=0$ 处形成尖点。
> **English**: An absolute value function is a piecewise function defined by $|x|$, representing the distance from a point to the origin on the number line, with a graph that is symmetric about the $y$-axis and has a sharp corner at $x=0$.

## 🎓 考点精讲

绝对值函数的核心定义为分段函数：

$$|x| = \begin{cases} x, & x \ge 0 \\ -x, & x < 0 \end{cases}$$

一般形式 $y = a|x - h| + k$ 的图像是顶点在 $(h, k)$ 的“V”形折线。当 $a > 0$ 时开口向上，$a < 0$ 时开口向下。$|a|$ 的大小决定“V”字开口的宽窄：$|a|$ 越大，开口越窄。

解含绝对值的不等式或方程时，关键步骤是“去绝对值符号”。常用方法：
1. **零点分段法**：令每个绝对值内部的表达式等于零，解出零点，将数轴分为若干区间，分别去掉绝对值符号求解，最后取并集。
2. **几何意义法**：$|x - a|$ 表示数轴上 $x$ 到 $a$ 的距离。如 $|x - 2| < 3$ 表示到 2 的距离小于 3 的点集，即 $-1 < x < 5$。
3. **平方法**：对于 $|f(x)| = |g(x)|$ 型方程，可两边平方化为 $f(x)^2 = g(x)^2$ 求解，但需注意验根。

**高考典型设问**：
- 用分段函数形式给出含绝对值的函数，要求画出图像或用图像判断方程根的个数；
- 解双重绝对值不等式如 $||x - 1| - 2| \le 3$，需由外向内逐层转化；
- 含参数的绝对值不等式恒成立问题，常转化为函数最值比较。

## ⚠️ 高频易错点

- **忘记讨论或分段点错误**：去掉绝对值时未按零点正确划分区间，导致不等式解集遗漏或重复。务必先解零点，在数轴上标注后逐段求解再取并集。
- **误以为绝对值只“去掉负号”**：对含代数式的情形，如 $|x-2|$，当 $x<2$ 时去掉绝对值应为 $2-x$，而非 $x-2$。部分学生在 $x$ 取值不确定时盲目去绝对值。
- **解 $|f(x)| < g(x)$ 型不等式时边界失误**：正确转化为 $-g(x) < f(x) < g(x)$，切忌直接写 $f(x) < g(x)$ 或 $f(x) > -g(x)$ 而未联立。若 $g(x)$ 含参数，还需讨论 $g(x)$ 的符号。
- **混淆 $|f(x)| = |g(x)|$ 与 $f(x)=\pm g(x)$**：后者需注意 $f(x)=g(x)$ 与 $f(x)=-g(x)$ 的解都应保留，但用平方法时容易引入增根，务必回代检验。
