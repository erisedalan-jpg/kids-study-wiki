# 虚数单位

> **一句话**：虚数单位是满足 $i^2 = -1$ 的数 $i$，与实数组合构成复数，是数系扩充的核心基础。
> **English**: The imaginary unit is the number $i$ defined by $i^2 = -1$, which combines with real numbers to form complex numbers, serving as the core foundation for extending the number system.

## 🎓 考点精讲

**定义**：虚数单位 $i$ 是一个满足 $i^2 = -1$ 的数学对象。由 $i$ 和实数 $a, b$ 组成的形如 $a+bi$ 的数称为复数，其中 $a$ 为实部，$b$ 为虚部。当 $b \neq 0$ 时，$a+bi$ 为虚数；当 $a=0, b \neq 0$ 时，$bi$ 为纯虚数。

**$i$ 的幂的周期性**（关键方法）：
$$i^1 = i,\quad i^2 = -1,\quad i^3 = -i,\quad i^4 = 1$$
此后每 4 次幂循环一次，即 $i^{n} = i^{n \bmod 4}$。计算高次幂时先将指数对 4 取余数，再根据结果写出对应值。例如：$i^{2025} = i^{2025 \bmod 4} = i^{1} = i$。

**复数的运算规则**：加减法对应实部与虚部分别相加减；乘法按多项式展开并代入 $i^2 = -1$ 化简；除法利用共轭复数进行分母实数化（如 $\frac{1+i}{1-i} = \frac{(1+i)^2}{(1-i)(1+i)} = \frac{2i}{2} = i$）。

**典型高考应用**：直接计算给定复数的实部、虚部或模；化简含 $i$ 的高次幂表达式；根据复数相等条件 $a+bi = c+di \Leftrightarrow a=c, b=d$ 求参数；在复平面内判断复数对应点的象限。通常以选择题或填空题形式出现，侧重考查运算准确性与对 $i$ 循环规律的理解。

## ⚠️ 高频易错点

- **混淆 $i^2 = -1$ 与 $i^2 = 1$**：误将 $i$ 当成实数符号处理，导致后续乘除全错，尤其出现在分母实数化时忘记符号变化。
- **复数相等的条件用错**：部分学生只比较实部，遗漏虚部必须同时相等，或在带参数的复数方程中错误地只列一个方程。
- **虚数比较大小的陷阱**：实数可以比大小，但虚数与复数一般不能比较大小。若题目中出现 “$a+bi > 0$” 此类表述，实际上隐含 $b=0$ 且 $a>0$，学生常忽略这一限制。
- **$i$ 的幂余数取错**：指数除以 4 时，余数为 0 时对应于 $i^4=1$，但有些学生误以为余数 0 对应 $i^0=1$ 而写错，本质是一致的但容易在负指数或不熟悉时犯错。
