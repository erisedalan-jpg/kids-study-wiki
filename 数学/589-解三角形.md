# 解三角形

> **一句话**：已知三角形的若干边角元素，利用正弦定理、余弦定理等工具求出其余未知边角的过程。
> **English**: The process of determining the unknown sides and angles of a triangle given a sufficient set of known elements, using the Law of Sines, Law of Cosines, and related identities.

## 🎓 考点精讲

**定义与核心定理**

解三角形，即求解三角形的六个基本元素（三边 $a,b,c$ 与三角 $A,B,C$）中未知量的问题。其核心依赖两个基本定理：

- **正弦定理**：$\dfrac{a}{\sin A} = \dfrac{b}{\sin B} = \dfrac{c}{\sin C} = 2R$（$R$ 为外接圆半径）。该定理将边与对角的正弦值建立比例关系，适用于已知两角一边或两边一对角的情形。
- **余弦定理**：$a^2 = b^2 + c^2 - 2bc\cos A$（及其关于 $b^2, c^2$ 的对称形式）。该定理建立了三边与一角余弦的关系，是已知三边求角或已知两边夹角求第三边的主要工具。

**关键方法**

高考解题的关键步骤在于**边角互化**：
1.  **确定已知条件类型**：常见的四种可解三角形条件是 ASA/AAS（两角一对边/两角一夹边）、SAS（两边一夹角）、SSS（三边）、SSA（两边一对角，需讨论解的情况）。
2.  **条件齐次化处理**：当题目给出边角混合等式（如 $a\cos B = b\cos A$）时，优先利用正弦定理将边统一化为角的正弦式，或利用余弦定理将角的余弦统一化为边，化简为纯边或纯角关系。
3.  **三角形形状判定**：通过边角互化得到的 $a=b$ 或 $\sin(A-B)=0$ 等关系，严格判定三角形为等腰、等边或直角三角形。注意，纯边化通常更严谨，避免角度多值的潜在错误。
4.  **面积公式运用**：面积 $S = \frac{1}{2}ab\sin C = \frac{1}{2}ac\sin B = \frac{1}{2}bc\sin A$，是边与角的重要关联桥梁，常用于已知面积反求边或角的问题中。

**典型高考设问与应用**

高考对本考点的考查常见于解答题的第一道或选填难题，设问方式高度集中：
- **求解三角形的基本量**：直接给出三个已知条件，求指定边长、角度或三角形面积。这是最基础的考查形式。
- **条件等式化简与边角互化**：如“已知 $b\sin A = a\cos B$，求角 $B$”。这需要将边用正弦定理转化为角，得到 $\sin B\sin A = \sin A\cos B$，进而求出 $\tan B = 1$，$B = \frac{\pi}{4}$。
- **最值与范围问题**：在已知三角形一角及其对边（或面积）恒定的条件下，求解周长、面积的最大值。常用方法为余弦定理结合基本不等式（如 $a^2 = b^2+c^2 - bc \ge 2bc - bc = bc$），或利用正弦定理将边转化为三角函数求值域。

## ⚠️ 高频易错点

- **SSA 情形解的情况讨论**：已知两边及其中一边的对角（如 $a,b,A$），解三角形时解的个数可能为 0、1 或 2。忘记讨论 $a$ 与 $b\sin A$ 以及 $a$ 与 $b$ 的大小关系，导致漏解或多解，是失分重灾区。
- **忽略三角形内角的隐含约束**：边角互化后直接约去等式两边的 $\sin A$，却未考虑 $\sin A = 0$（即 $A = 0$ 或 $\pi$）在三角形中不可能成立，虽结果可能正确，但逻辑上存在不严谨性。更致命的是，在求值域时忘记角度范围 $(0, \pi)$ 以及内角和为 $\pi$ 这一根本约束，导致范围扩大。
- **余弦定理公式符号混淆**：在角为钝角时，余弦值为负，公式 $a^2 = b^2 + c^2 - 2bc\cos A$ 中“$- 2bc\cos A$”这一项实际上变为加法，部分学生在代数运算中常因符号错误导致边长计算错误。
- **面积公式误用**：使用 $S = \frac{1}{2}ab\sin C$ 时，角 $C$ 必须是边 $a$ 与边 $b$ 的夹角，错误地代入非夹角会导致整个计算无效。
