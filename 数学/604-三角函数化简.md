# 三角函数化简

> **一句话**：运用三角恒等变换公式，将复杂的三角表达式转化为单个三角函数、常数或易于求值的形式。
> **English**: The process of reducing a complex trigonometric expression to a single trigonometric function, constant, or a form easy to evaluate using trigonometric identities.

## 🎓 考点精讲

**定义/原理**  
三角函数化简以三角恒等变换为工具，利用平方关系 $\sin^2\alpha + \cos^2\alpha = 1$、商数关系 $\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$、诱导公式、和差角公式、二倍角公式及辅助角公式等，消去同类项、合并角、统一函数名，达到最简形式。其本质是通过代数与三角结构的转化，揭示函数间的内在联系。

**关键方法**  
化简遵循“三看”原则：**一看角**，通过诱导公式、和差倍角公式把不同角化为相同角或特殊角；**二看名**，常用切化弦、正余互化将异名三角函数化为同名；**三看结构**，利用平方关系升降幂、逆用公式、因式分解等调整表达式结构。核心步骤常为：切化弦 → 通分/约分 → 利用平方关系 → 辅助角公式 $a\sin x + b\cos x = \sqrt{a^2+b^2}\sin(x+\varphi)$ 或化为单一函数。对于给角求值问题，常需将非特殊角拆分为特殊角的和或差。

**典型应用**  
高考中极常见于：（1）**给值求值**：如已知 $\sin \alpha = \frac{3}{5}$，化简求 $\cos 2\alpha$ 的值；（2）**化简函数求性质**：将 $f(x)=\sin^2 x + \sqrt{3}\sin x \cos x$ 降幂并辅助角化为 $f(x)=\sin(2x-\frac{\pi}{6})+\frac{1}{2}$，进而求最小正周期、最值或单调区间；（3）**三角恒等式证明**：从复杂一边化简到简单一边。考查核心是灵活选用公式，快速将式子化为可判结构。

## ⚠️ 高频易错点

- **忽略角的范围导致符号出错**：开方 $\sqrt{1-\sin^2\alpha}=|\cos\alpha|$ 时未根据象限确定正负，直接去掉绝对值。
- **公式记忆混淆**：$\sin(\alpha\pm\beta)$ 与 $\cos(\alpha\pm\beta)$ 的符号记反，或二倍角公式变形 $\cos 2\alpha = 1-2\sin^2\alpha = 2\cos^2\alpha -1$ 选用不当。
- **切化弦后遗忘分母处理**：在通分、约分时漏乘分母导致恒等变形中断。
- **辅助角公式的 $\varphi$ 求错**：直接用 $\tan\varphi = \frac{b}{a}$ 却不验证点 $(a,b)$ 所在象限，导致辅助角取值错误。
