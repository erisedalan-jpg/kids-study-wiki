# 三角函数周期

> **一句话**：三角函数周期是指函数值重复出现时自变量的最小增量，对应最小正周期，是分析周期现象和求解三角方程的核心参数。
> **English**: The period of a trigonometric function is the smallest positive increment of the independent variable for which the function values repeat, defining the fundamental cycle of the function.

## 🎓 考点精讲

三角函数的周期是研究其图像与性质的基础。  
- **定义与基本公式**：对于形如 $y = A\sin(\omega x + \varphi)$ 的函数，若 $\omega > 0$，其最小正周期 $T = \frac{2\pi}{\omega}$；$y = A\cos(\omega x + \varphi)$ 同理。对于 $y = A\tan(\omega x + \varphi)$，最小正周期为 $T = \frac{\pi}{\omega}$。这里的 $\omega$ 是角频率，$\varphi$ 是初相。  
- **抽象函数的周期判定**：若函数 $f(x)$ 满足 $f(x + a) = -f(x)$ 或 $f(x + a) = \frac{1}{f(x)}$（$f(x) \neq 0$），则 $f(x)$ 是周期函数，周期 $T = 2a$。这类关系常结合函数方程在选填题中考查。  
- **利用图像识别周期**：相邻两个最高点（或最低点）之间的水平距离为一个周期；相邻两个零点之间的水平距离是半个周期（对正弦、余弦而言）。  
- **高考典型设问**：（1）给定三角型函数求最小正周期，常与单调性、对称性结合；（2）给出部分图像求解析式并确定周期；（3）利用周期性求函数值（如 $f(2026\pi)$）。解题关键：准确识别 $\omega$，注意复合形式中绝对值和变换对周期的影响。

## ⚠️ 高频易错点

- **混淆正弦与正切周期的公式**：误用 $T = \frac{2\pi}{\omega}$ 求正切周期，正切应为 $T = \frac{\pi}{\omega}$。  
- **忽视 $\omega$ 的符号**：周期公式中的 $\omega$ 取绝对值，若 $\omega < 0$ 需先利用诱导公式化为正值再求周期，否则周期计算结果可能为负或错误。  
- **错误判断最小正周期**：如 $y = |\sin x|$ 的周期是 $\pi$，并非 $2\pi$；同样 $y = \sin |x|$ 不是周期函数，易误判。含有绝对值的三角函数需画图或分段分析。  
- **混淆周期与对称轴间距**：正弦函数相邻两个对称轴的距离是半个周期，但学生常直接将其等同于周期，导致解析式求参错误。
