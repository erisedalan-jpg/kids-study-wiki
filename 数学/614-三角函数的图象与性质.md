# 三角函数的图象与性质

> **一句话**：正弦、余弦、正切函数的图象特征与周期性、奇偶性、单调性、值域等基本性质的总和，是描述周期现象的核心数学工具。
> **English**: The overall graph features and fundamental properties such as periodicity, parity, monotonicity, and range of the sine, cosine, and tangent functions, which serve as the core mathematical tool for describing periodic phenomena.

## 🎓 考点精讲

- **定义/原理**  
  基本三角函数包括 $y = \sin x$，$y = \cos x$，$y = \tan x$。  
  - **正弦函数** $y = \sin x$：定义域 $\mathbb{R}$，值域 $[-1, 1]$，最小正周期 $T = 2\pi$，奇函数，在 $[-\frac{\pi}{2} + 2k\pi, \frac{\pi}{2} + 2k\pi]$ 上单调递增，在 $[\frac{\pi}{2} + 2k\pi, \frac{3\pi}{2} + 2k\pi]$ 上单调递减，对称中心 $(k\pi, 0)$，对称轴 $x = \frac{\pi}{2} + k\pi$ ($k \in \mathbb{Z}$)。  
  - **余弦函数** $y = \cos x$：定义域 $\mathbb{R}$，值域 $[-1, 1]$，$T = 2\pi$，偶函数，在 $[-\pi + 2k\pi, 2k\pi]$ 上递增，在 $[2k\pi, \pi + 2k\pi]$ 上递减，对称中心 $(\frac{\pi}{2} + k\pi, 0)$，对称轴 $x = k\pi$。  
  - **正切函数** $y = \tan x$：定义域 $\{x \mid x \neq \frac{\pi}{2} + k\pi\}$，值域 $\mathbb{R}$，$T = \pi$，奇函数，在每个区间 $(-\frac{\pi}{2} + k\pi, \frac{\pi}{2} + k\pi)$ 内单调递增，对称中心 $(\frac{k\pi}{2}, 0)$，无对称轴。  
  一般正弦型函数 $y = A\sin(\omega x + \varphi) + b$ 的图象可由 $y = \sin x$ 通过平移、伸缩变换得到：周期 $T = \frac{2\pi}{|\omega|}$，频率 $f = \frac{1}{T}$，相位 $\omega x + \varphi$，初相 $\varphi$。

- **关键方法**  
  1. **五点法作图**：对 $y = A\sin(\omega x + \varphi)$ 的一个周期，取使相位等于 $0, \frac{\pi}{2}, \pi, \frac{3\pi}{2}, 2\pi$ 的五个 $x$ 值，计算对应 $y$ 值，描点绘制光滑曲线。  
  2. **求单调区间**：将 $\omega x + \varphi$ 看作整体，代入 $y = \sin z$ 或 $y = \cos z$ 的单调区间解出 $x$。当 $\omega < 0$ 时，需先利用诱导公式化为正系数。  
  3. **求值域与最值**：结合定义域、图象及三角函数的有界性；对于 $y = a\sin x + b\cos x$，可化为 $\sqrt{a^2 + b^2}\sin(x + \varphi)$ 处理。  
  4. **判断周期与对称性**：直接由解析式读出系数，利用公式 $T = \frac{2\pi}{|\omega|}$；对称轴或对称中心通过令 $\omega x + \varphi = k\pi + \frac{\pi}{2}$（对称轴）或 $\omega x + \varphi = k\pi$（对称中心）求解。

- **典型应用**  
  高考中常考查：  
  - 根据部分图象求解析式 $y = A\sin(\omega x + \varphi) + b$，需从图象中提取最值（得 $A, b$）、周期（得 $\omega$）和特征点代入求 $\varphi$。  
  - 结合图象比较三角函数值大小、解三角不等式（如 $\sin x > \frac{1}{2}$），一般画出单位圆或图象，在指定区间内确定解集。  
  - 与导数、解三角形、参数问题结合，如讨论 $f(x) = \sin x + \cos x + a$ 在 $[0, \pi]$ 上的零点个数，需画出图象或利用辅助角公式化为正弦型函数分析。

## ⚠️ 高频易错点

- **忽视正切函数定义域**：解 $y = \tan(\omega x + \varphi)$ 相关问题时，丢失 $ \omega x + \varphi \neq \frac{\pi}{2} + k\pi$ 的限制，导致答案区间包含无定义点。  
- **单调区间写法错误**：对 $y = A\sin(\omega x + \varphi)$（$\omega < 0$）直接套用公式而不先恒等变形，得到的单调性与实际相反；或将多个单调增区间用“并集”连接（应写“每个区间上单调增”，不可用 $\cup$ 连接）。  
- **周期判断混淆**：忘记加绝对值导致周期错误，如 $y = \sin\left(-2x + \frac{\pi}{3}\right)$ 的周期误写为 $-\pi$，正确应为 $T = \frac{2\pi}{|-2|} = \pi$。  
- **复合函数性质讨论不当**：讨论 $y = \sin(\cos x)$ 的值域或单调性时，未考虑内层函数值域对整体性质的制约，直接套用正弦性质出错。
