# 三角形的面积公式

> **一句话**：三角形面积是由底和高、两边及夹角、三边长或顶点坐标决定的平面图形度量，核心为 $S = \frac{1}{2}ah$ 及其等价变形。
> **English**: The area of a triangle is a measure of the plane figure determined by base and height, two sides and the included angle, three side lengths, or vertex coordinates, with the core formula $S = \frac{1}{2}ah$ and its equivalent variations.

## 🎓 考点精讲

**定义与核心公式**  
三角形面积的最基本公式是 $S = \frac{1}{2}ah$，其中 $a$ 为任意一边长，$h$ 为该边上的高。在高考中，仅掌握这一公式难以应对向量、解三角形、解析几何等综合题，必须熟悉以下等价形式：

- **两边夹角公式**：$S = \frac{1}{2}ab\sin C = \frac{1}{2}bc\sin A = \frac{1}{2}ca\sin B$。这是解三角形大题的主要计算工具，尤其适合已知两边及其夹角求面积，或通过已知面积反求角或边。
- **海伦公式**：$S = \sqrt{p(p-a)(p-b)(p-c)}$，其中 $p = \frac{a+b+c}{2}$。适用于已知三边长求面积，常见于涉及三角形周长与面积关系的填空题。
- **坐标/向量形式**：若已知三角形三个顶点坐标 $A(x_1,y_1)$、$B(x_2,y_2)$、$C(x_3,y_3)$，则 $S = \frac{1}{2}\left| (x_2-x_1)(y_3-y_1) - (x_3-x_1)(y_2-y_1) \right|$，也可用向量叉积模长的一半表示：$S = \frac{1}{2}|\overrightarrow{AB}\times\overrightarrow{AC}|$。该形式在解析几何与空间向量题中直接用于求面积或判断共线。

**关键方法与步骤**  
高考解题时，选择面积公式的判断依据是已知条件：
1. 已知底和高 → 直接用 $S=\frac12 ah$。  
2. 已知两边及其夹角 → 用 $S=\frac12 ab\sin C$，尤其注意先用正、余弦定理求出所需边角。  
3. 已知三边 → 先用余弦定理求一角，再用两边夹角公式；或直接代海伦公式（计算量较大时优先求角）。  
4. 已知顶点坐标 → 用行列式或向量叉积，避免求边长和高。  

**典型高考设问**  
- 解三角形大题：给出部分边角，求三角形面积或面积的最值。常需先用正弦定理、余弦定理建立边角关系，再代入 $S=\frac12 ab\sin C$，结合基本不等式或三角函数有界性求最大值。  
- 解析几何综合：在椭圆、抛物线上取点构成三角形，求面积表达式并求最值。利用坐标公式或弦长公式与点到直线距离，将面积转化为函数求最值。  
- 向量题：以向量模长和夹角给出条件，用 $\frac12|\vec{a}||\vec{b}|\sin\langle\vec{a},\vec{b}\rangle$（即叉积模长的一半）计算面积。

## ⚠️ 高频易错点

- **混淆面积公式中的夹角**：$S = \frac12 ab\sin C$ 中，角 $C$ 必须是边 $a$ 与 $b$ 的夹角，切忌错配边与对角。若用 $S = \frac12 bc\sin A$，需确保 $A$ 是 $b$、$c$ 的夹角。  
- **忽略高与底的对应关系**：在三角形中，每条底边都对应唯一的高，不能随意用不同底和高的乘积求面积。  
- **坐标面积公式忘记绝对值**：行列式或叉积得到的值可能为负，面积必须取绝对值或加绝对值符号，否则导致面积错误或出现负值。  
- **海伦公式中 $p$ 是半周长**：直接代入三边和作为 $p$ 是常见计算错误，必须记住 $p = \frac{a+b+c}{2}$。
