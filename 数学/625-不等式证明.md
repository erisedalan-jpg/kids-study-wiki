# 不等式证明

> **一句话**：不等式证明是依据不等式的性质、基本不等式、函数单调性、放缩技巧或数学归纳法等，严格推导出给定不等式成立的过程。
> **English**: Inequality proof is the process of rigorously deducing that a given inequality holds by using properties of inequalities, basic inequalities, function monotonicity, scaling techniques, or mathematical induction.

## 🎓 考点精讲

- **定义/原理**：设 $a,b$ 为实数，需证明 $a\geq b$ 或 $a>b$。常用基础原理包括：
  - **比较法**：$a-b\geq 0 \iff a\geq b$（作差），或当 $a,b>0$ 时 $\dfrac{a}{b}\geq 1 \iff a\geq b$（作商）。
  - **基本不等式链**：$\dfrac{2}{\frac{1}{a}+\frac{1}{b}} \le \sqrt{ab} \le \dfrac{a+b}{2} \le \sqrt{\dfrac{a^2+b^2}{2}}$（$a,b>0$），等号成立条件均为 $a=b$。
  - **重要不等式**：$a^2+b^2\geq 2ab$，三元形式 $a^2+b^2+c^2\geq ab+bc+ca$；柯西不等式 $(a^2+b^2)(c^2+d^2)\geq (ac+bd)^2$。
  - **函数工具**：利用导数判断函数单调性、最值，把不等式转化为函数值比较。

- **关键方法**：高考解题常用六大方法：
  **①作差（商）比较法**：配方、因式分解等变形后判号；**②综合法**：从已知条件与已知不等式出发逐步推导；**③分析法**：从结论倒推寻找充分条件，常写为“要证…，即证…”；**④放缩法**：通过放大或缩小中间项（如裂项放缩、利用 $n-1)n < n^2$ 等）过渡到欲证不等式；**⑤数学归纳法**：用于与正整数 $n$ 有关的不等式，先证 $n=n_0$ 成立，再证 $n=k$ 推出 $n=k+1$；**⑥构造函数法**：将不等式两边移项设作函数，求导分析最值。

- **典型应用**：高考中常以**数列不等式**（如证明 $\sum_{k=1}^n \frac{1}{k^2} <2$）、**极值点偏移**（利用对称化构造函数证明 $x_1+x_2$ 与某常数的关系）、**选考不等式选讲**（含绝对值不等式或三元基本不等式证明）等形式出现。题目设问常为“证明：当 $x>0$ 时，$e^x > x+1$”或“求证：$\frac{a}{b+c}+\frac{b}{c+a}+\frac{c}{a+b}\geq \frac{3}{2}$（$a,b,c>0$）”。

## ⚠️ 高频易错点

- **忽视等号成立条件**：使用基本不等式或柯西不等式时，未验证等号能否取到导致思路不严谨，尤其在“证明不等式”中若等号永远不成立则证明有误。
- **放缩过度**：数列不等式中放缩尺度过大或过小，导致无法推出目标不等式，如错误使用 $\frac{1}{n^2}<\frac{1}{n(n-1)}$ 而忽略对 $n=1$ 的处理。
- **用求和符号后错位抵消错误**：裂项放缩时未正确写出通项或忽略了符号变化，导致不等号方向反向。
- **构造函数后未调整定义域或忽略端点**：用导数证不等式时未验证函数在区间端点的取值或极限，导致最值判断错误，例如对 $f(x)=\ln x$ 在 $(0,+\infty)$ 上的凸性应用不当（待家长核对）。
