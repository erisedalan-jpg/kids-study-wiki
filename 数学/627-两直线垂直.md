# 两直线垂直

> **一句话**：在平面直角坐标系中，若两直线的斜率均存在，它们垂直的充要条件是斜率之积为 -1；若直线方程为一般式，则垂直的充要条件是 `A₁A₂ + B₁B₂ = 0`。
> **English**: Two lines in a Cartesian plane are perpendicular if and only if the product of their slopes is -1, or equivalently, if the sum of the products of their corresponding coefficients in general form is zero.

## 🎓 考点精讲

- **定义/原理**  
  设两直线 `l₁` 与 `l₂`，方向向量分别为 `v₁` 和 `v₂`。两直线垂直的代数判据是 `v₁·v₂ = 0`。  
  当直线用斜截式 `y = kx + b` 表示时：  
  - 若 `k₁`、`k₂` 均存在，则 `l₁ ⊥ l₂ ⇔ k₁·k₂ = -1`。  
  - 若一条直线斜率不存在（垂直于 x 轴），另一条直线斜率必为 0（垂直于 y 轴），二者也垂直。  
  当直线用一般式 `A₁x + B₁y + C₁ = 0` 和 `A₂x + B₂y + C₂ = 0` 表示时，  
  垂直的充要条件为 `A₁A₂ + B₁B₂ = 0`，无需讨论斜率是否存在。

- **关键方法**  
  1. **条件转化**：看到“垂直”，立即转化为斜率之积为 -1 或系数点积为零。  
  2. **分类讨论**：若题设未明确斜率存在，必须按斜率存在与否分情况求解，防止漏解。  
  3. **待定系数法**：已知一条直线方程，求过某点且与之垂直的直线方程时，可直接设一般式并利用 `A₁A₂ + B₁B₂ = 0` 和点坐标联立求解。

- **典型应用**  
  高考中常以填空题或解答题的第一问出现。例如：  
  - 已知直线 `l₁: 2x - y + 1 = 0`，求过点 `(3, 2)` 且与 `l₁` 垂直的直线方程。  
    解：设所求方程为 `x + 2y + C = 0`，代入点得 `3 + 4 + C = 0 ⇒ C = -7`，故方程为 `x + 2y - 7 = 0`。  
  - 在三角形中，通过计算两边所在直线的斜率乘积判断是否为直角三角形。

## ⚠️ 高频易错点

- 直接使用 `k₁k₂ = -1` 而未考虑斜率不存在的情形，导致解答不完整。  
- 将垂直条件 `A₁A₂ + B₁B₂ = 0` 与平行条件 `A₁B₂ - A₂B₁ = 0` 混淆。  
- 在运用方向向量点积时，误以为方向向量可直接用 `(1, k)` 表示后点积为 `1 + k₁k₂ = 0`，推导出错时应回到点积定义验证。  
- 求解参数时忘记检验所得直线是否与已知直线重合（虽然重合时亦满足点积为零的退化情况，但重合不属于垂直）。
