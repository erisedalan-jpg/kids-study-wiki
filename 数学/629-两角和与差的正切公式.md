# 两角和与差的正切公式

> **一句话**：$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha \tan\beta}$，用于求两角和差的正切值。
> **English**: The formulas for the tangent of the sum and difference of two angles: $\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha \tan\beta}$.

## 🎓 考点精讲

**定义/原理**  
两角和与差的正切公式由正余弦的和差公式推导而得，具体为：  
$\tan(\alpha+\beta) = \dfrac{\tan\alpha + \tan\beta}{1 - \tan\alpha \tan\beta}$，  
$\tan(\alpha-\beta) = \dfrac{\tan\alpha - \tan\beta}{1 + \tan\alpha \tan\beta}$。  
成立条件：$\alpha,\beta,\alpha\pm\beta$ 均不等于 $k\pi+\frac{\pi}{2}\ (k\in\mathbb{Z})$，且分母不为零（即 $\tan\alpha\tan\beta\neq\pm1$）。该公式在高考中可直接用于求值、化简与证明。

**关键方法**  
1. **给值求值**：已知 $\tan\alpha$、$\tan\beta$ 或可求出的正切值，通过和差公式直接计算 $\tan(\alpha\pm\beta)$。当已知条件为正弦、余弦值时，往往先转换为正切。  
2. **逆用与变形**：公式逆用可由 $\frac{\tan\alpha\pm\tan\beta}{1\mp\tan\alpha\tan\beta}$ 提取出 $\tan(\alpha\pm\beta)$，在化简分式或证明恒等式时常用。变形如 $\tan\alpha+\tan\beta = \tan(\alpha+\beta)(1-\tan\alpha\tan\beta)$ 可以整体处理两角和的问题。  
3. **角的分拆与配凑**：将非特殊角拆为已知特殊角的和差，例如 $15^\circ = 45^\circ-30^\circ$ 或 $75^\circ=45^\circ+30^\circ$，再用公式计算精确值。

**典型应用**  
高考常见题型包括：  
- 选择题/填空题直接利用和差公式求 $\tan15^\circ$ 或 $\tan75^\circ$ 的值，并结合分母有理化得到最后结果。  
- 解答题中结合三角形内角和关系，如已知 $\tan A$、$\tan B$ 求 $\tan C$，需用诱导公式 $\tan C = -\tan(A+B)$，再套用和角公式。  
- 条件求值与证明题中，将结构 $\frac{a+b}{1-ab}$ 识别为 $\tan(\alpha+\beta)$ 的形式，实现巧解。

## ⚠️ 高频易错点

- 忽略分母为零或正切函数无定义的情形，导致增解或漏解。必须同时考虑 $\tan\alpha$、$\tan\beta$ 存在且 $\tan\alpha\tan\beta\neq\pm1$，以及 $\alpha\pm\beta\neq k\pi+\frac{\pi}{2}$。
- 符号记忆混淆：和角分母为 $1-\tan\alpha\tan\beta$，差角分母为 $1+\tan\alpha\tan\beta$，与正弦、余弦公式的符号规律不同，学生常把差角分母写成减号。
- 在三角形问题中，直接用 $\tan C = \tan(A+B)$ 而忘记添加负号，正确应为 $\tan C = \tan[\pi-(A+B)] = -\tan(A+B)$。
- 逆用时强行拆分角，例如将 $\tan 2\theta$ 错误地用和角公式分成 $\tan(\theta+\theta)$ 后，忘记分子变为 $2\tan\theta$ 分母变为 $1-\tan^2\theta$ 的推导细节，导致与二倍角公式混淆。
