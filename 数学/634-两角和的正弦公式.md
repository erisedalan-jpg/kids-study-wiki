# 两角和的正弦公式

> **一句话**：对于任意角 α、β，有 $\sin(\alpha + \beta) = \sin\alpha \cos\beta + \cos\alpha \sin\beta$，称为两角和的正弦公式。
> **English**: The sine of a sum identity: $\sin(\alpha + \beta) = \sin\alpha \cos\beta + \cos\alpha \sin\beta$.

## 🎓 考点精讲

- **定义/原理**  
  两角和的正弦公式是三角恒等变换的核心公式之一，适用于任意角 α、β：  
  $$
\sin(\alpha + \beta) = \sin\alpha \cos\beta + \cos\alpha \sin\beta
$$  
  其推导通常借助两角和的余弦公式与诱导公式，或利用单位圆与向量数量积。该公式将和角的正弦转化为单角正、余弦的乘积之和，是化简三角函数表达式、求值及证明恒等式的基础。

- **关键方法**  
  1. **给角求值**：将非特殊角拆分为两个特殊角的和（如 $75^\circ = 45^\circ + 30^\circ$），代入公式求值。  
  2. **给值求值**：已知 $\sin\alpha$、$\cos\beta$ 等，先利用同角关系求出所需的正、余弦值（注意符号由角所在象限决定），再代入公式计算 $\sin(\alpha+\beta)$。  
  3. **公式逆用与变形**：如 $a\sin x + b\cos x$ 可化为 $\sqrt{a^2+b^2}\sin(x+\varphi)$，其中 $\tan\varphi = \frac{b}{a}$，这是高考中三角函数图像与性质题的关键步骤。  
  4. **综合应用**：常与诱导公式、二倍角公式、正弦定理等结合，解决三角形内角关系或最值问题。

- **典型应用**  
  高考常见设问：  
  - 直接求值：如已知 $\sin\alpha = \frac{3}{5}, \alpha \in (\frac{\pi}{2}, \pi), \cos\beta = -\frac{5}{13}, \beta \in (\pi, \frac{3\pi}{2})$，求 $\sin(\alpha+\beta)$。  
  - 化简与证明：利用公式将复杂表达式统一为 $\sin(\alpha+\beta)$ 的形式进行化简，或证明三角恒等式。  
  - 函数化简：将 $f(x)=\sin x + \sqrt{3}\cos x$ 化为 $2\sin(x+\frac{\pi}{3})$，进而求周期、最值及单调区间。

## ⚠️ 高频易错点

- **符号记忆混淆**：两角和的正弦公式中间为“+”号，即 $\sin\alpha\cos\beta + \cos\alpha\sin\beta$，切勿错记为减号或与余弦公式 $\cos(\alpha+\beta)=\cos\alpha\cos\beta - \sin\alpha\sin\beta$ 的符号规则混淆。
- **忽视角范围导致符号错误**：由已知三角函数值求同角的其他函数值时，必须根据角的象限确定正、余弦的符号。角范围判断失误，会直接导致最终结果符号相反。
- **公式逆用时系数不匹配**：将 $a\sin x + b\cos x$ 化为 $R\sin(x+\varphi)$ 时，需提取 $\sqrt{a^2+b^2}$，保证两系数分别为 $\cos\varphi$ 和 $\sin\varphi$，不少学生只变形式而忘记调整系数。
- **特殊角拆分不当**：如将 $105^\circ$ 拆为 $60^\circ + 45^\circ$ 是可行的，但不能随意拆成两个非特殊角导致无法求值，或拆分时遗漏角的单位统一。
