# 交集及其运算

> **一句话**：对于两个集合 A 与 B，由所有既属于 A 又属于 B 的元素构成的集合称为 A 与 B 的交集，记作 $A \cap B$。
> **English**: The intersection of two sets A and B, denoted $A \cap B$, is the set of elements that belong to both A and B.

## 🎓 考点精讲

**定义**  
设 $A$、$B$ 为集合，集合 $A \cap B = \{x \mid x \in A \text{ 且 } x \in B\}$ 称为 $A$ 与 $B$ 的交集。若 $A \cap B = \varnothing$，则称 $A$ 与 $B$ 不相交。

**运算性质**  
1. 幂等律：$A \cap A = A$  
2. 交换律：$A \cap B = B \cap A$  
3. 结合律：$(A \cap B) \cap C = A \cap (B \cap C)$  
4. 与全集 $U$ 和空集的关系：$A \cap U = A$，$A \cap \varnothing = \varnothing$  
5. 与子集关系：$A \cap B = A \iff A \subseteq B$

**关键方法**  
- **数轴分析法**：对于用不等式描述的数集，画出各集合对应的区间，交集的公共部分即为各区间在数轴上的重叠区域。操作步骤：① 将每个集合转化为区间或不等式；② 在数轴上标出端点并画出范围；③ 找出所有范围同时覆盖的部分；④ 写出交集，特别注意端点可否取等。  
- **韦恩图辅助**：对于抽象集合关系选择题，用韦恩图标记区域，交集为两圆重叠部分。  
- **分类讨论**：当集合含有参数时，需讨论参数对集合边界的影响，并结合 $A \cap B = \varnothing$ 或 $A \cap B = A$ 等条件列式求解。

**典型应用**  
高考中常以选择题、填空题出现，或融入函数定义域、不等式解集等综合题。  
- 直接求交：如 $A=\{x\mid -1<x\le 3\}$，$B=\{x\mid x>0\}$，则 $A\cap B = (0,3]$。  
- 含参求交：已知 $A=\{x\mid x>a\}$，$B=\{x\mid -2\le x\le 5\}$，且 $A\cap B \neq \varnothing$，求 $a$ 的范围。解：由 $A\cap B\neq \varnothing$ 得 $a<5$（注意端点，当 $a=5$ 时，$A\cap B=\{x\mid x>5\}\cap[-2,5]=\varnothing$）。

## ⚠️ 高频易错点

- **忽略空集情形**：当条件给出 $A\cap B = A$ 或 $A\subseteq B$ 时，易漏掉 $A=\varnothing$ 的讨论；含参集合若为不等式解集，参数取值可能导致集合为空，必须单独检验。
- **端点取舍错误**：用数轴求交集时，端点是否包含取决于同时满足各集合的边界条件。易出现将开端点误作闭端点，或在含参问题中漏掉端点临界值（如 $a=5$ 的情形需代入检验）。
- **未化简描述法集合**：对于用描述法给出的集合（如 $\{y\mid y=x^2+1\}$ 与 $\{x\mid x\ge 2\}$ 的交集），必须先认清集合代表元素的含义并化简为区间或具体数集，再求交集，否则易被形式迷惑。
- **混淆交集与并集符号**：将 $A\cap B$ 与 $A\cup B$ 的运算规则记混，例如误认为 $A\cap (B\cup C)=(A\cap B)\cup C$（正确应为分配律：$A\cap (B\cup C)=(A\cap B)\cup (A\cap C)$）。
