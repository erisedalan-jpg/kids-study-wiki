# 交集运算

> **一句话**：交集是由同时属于集合 A 与集合 B 的所有元素构成的集合，记作 $A \cap B$。
> **English**: The intersection of two sets $A$ and $B$, written $A \cap B$, is the set of elements that belong to both $A$ and $B$.

## 🎓 考点精讲

- **定义/原理**：对于任意两个集合 $A$ 与 $B$，它们的交集定义为  
  $$A \cap B = \{ x \mid x \in A \ \text{且} \ x \in B \}.$$  
  交换律 $A \cap B = B \cap A$，结合律 $(A \cap B) \cap C = A \cap (B \cap C)$ 恒成立。交集与并集、补集满足德摩根律：  
  $$\complement_U (A \cap B) = (\complement_U A) \cup (\complement_U B).$$  
  特别地，任何集合与空集的交集为空集：$A \cap \varnothing = \varnothing$。

- **关键方法**：求交集的核心是准确找到公共元素。对有限集，可直接比较元素或利用文氏图。对无限集（通常以不等式解集、定义域、值域等连续实数集形式出现），须在数轴上同时画出各集合的区间，取公共部分；注意端点处能否取等号由原不等式的等号决定，并在数轴上用实心点与空心点明确区分。含参数时，先对参数分类讨论，分别求出每种情况下的交集，再根据题目要求合并或进一步限制参数。

- **典型应用**：高考中主要考查以下角度：  
  (1) **直接求交**：给出 $A=\{x \mid 2x-1>0\}$，$B=\{x \mid x^2-x-2 \le 0\}$，求 $A \cap B$。解得 $A=\{x \mid x>\frac12\}$，$B=\{x \mid -1 \le x \le 2\}$，在数轴上取公共部分得 $A \cap B = \{x \mid \frac12 < x \le 2\}$。  
  (2) **已知交集反求参数**：设 $A=[a, a+2]$，$B=(1,3)$，若 $A \cap B = A$，求 $a$ 的范围。需解 $A \subseteq B$，即 $a>1$ 且 $a+2<3$，得 $1<a<1$，无解（待家长核对），需进一步讨论空集等情形。  
  (3) **与逻辑结合**：“$x \in A \cap B$”是“$x \in A$”的充分不必要条件（因为 $A \cap B \subseteq A$，但反之不成立），此类问题常出现在充要条件判断中。

## ⚠️ 高频易错点

- **忽略空集导致漏解**：当集合 $A$ 或 $B$ 可能为空集时（尤其在含参不等式中），必须先单独讨论空集情况，否则会丢失参数范围。  
- **端点开闭判断失误**：在数轴上取区间交集时，未根据原不等式是否带等号准确标记端点，导致结果多取或少取端点值。  
- **混淆交集与并集条件**：读题时将“且”错当成“或”，运算结果写成并集，或在分类讨论时错误地合并了不同情况的解。  
- **未将抽象集合具体化**：面对描述法给出的抽象集合（如 $\{y \mid y=f(x)\}$），未先求出具体的取值范围，直接进行逻辑推理而致错。
