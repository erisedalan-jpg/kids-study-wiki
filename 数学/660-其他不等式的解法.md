# 其他不等式的解法

> **一句话**：在高考范围内，“其他不等式”主要指高次不等式、绝对值不等式、无理不等式以及指数对数不等式等，其求解核心是通过等价变形将其转化为标准的不等式组，或借助相应函数的单调性与图像进行求解。
> **English**: Methods for solving “other inequalities” in the college entrance exam include solving higher-degree inequalities, absolute value inequalities, irrational inequalities, and exponential/logarithmic inequalities by transforming them into equivalent systems or using function properties.

## 🎓 考点精讲

**定义/原理**  
除一元二次不等式和简单分式不等式外，高考常将以下类型统归为“其他不等式”：
- 高次不等式：形如 $f(x) = (x-a_1)^{n_1}(x-a_2)^{n_2}\cdots (x-a_k)^{n_k} > 0$（或<0），通常使用**数轴标根法**，依据各因式的奇偶次幂确定符号变化。
- 绝对值不等式：如 $|f(x)| < a$ 等价于 $-a < f(x) < a$，$|f(x)| > a$ 等价于 $f(x) < -a$ 或 $f(x) > a$（$a>0$）；对于含参数或复杂结构，常采用**零点分段法**去绝对值。
- 无理不等式：形如 $\sqrt{f(x)} < g(x)$ 等价于 $f(x) \ge 0,\ g(x) > 0, \ f(x) < [g(x)]^2$；$\sqrt{f(x)} > g(x)$ 需分 $g(x) \ge 0$ 和 $g(x) < 0$ 讨论。
- 指数、对数不等式：利用指数函数 $a^x$ 和对数函数 $\log_a x$ 的单调性：当 $a>1$ 时单调增，$0<a<1$ 时单调减。转化为同底后比较真数或指数，注意定义域限制（对数的真数>0，底数>0且≠1）。

**关键方法**  
1. **标准化**：将所有项移向一边化为与0比较的形式。  
2. **化归**：将新类型不等式通过平方、取对数、换元（如指对换元设 $t = a^x$）等转化为熟悉的一次、二次、分式不等式。  
3. **等价条件**：必须充分注意变形前后的等价性，尤其是平方带来的增解、去分母时符号、去掉对数时定义域变化等。  
4. **数形结合**：对于含有参数或难以直接解的不等式，可利用对应函数的图像分析解集。

**典型应用**  
高考常见设问：  
- 解具体给定系数的绝对值、无理、指对不等式，并求交集或并集。  
- 已知不等式解集求参数值或取值范围。  
- 结合集合的运算（如 $A \cap B$、$A \subseteq B$）考查不等式的解。  
示例：  
解不等式 $\sqrt{x+1} < x-1$。等价于 $x+1 \ge 0,\ x-1>0,\ x+1 < (x-1)^2$，解得 $x>3$。  
解不等式 $2^{x} - 3\cdot 2^{\frac{x}{2}} + 2 \le 0$。令 $t = 2^{\frac{x}{2}} >0$，化为 $t^2 - 3t + 2 \le 0$，得 $1\le t \le 2$，回代得 $0 \le x \le 2$。

## ⚠️ 高频易错点

- 无理不等式盲目两边平方而不加非负条件，导致解集扩大。正确做法必须写出等价不等式组。
- 解对数不等式时忽略真数大于0和底数的取值范围，造成定义域错误；换元后忘记换元未知数的范围（如 $t = a^x >0$）。
- 使用数轴标根法时，对偶次重根处“穿而不穿”判断错误，导致区间符号写反。
- 含参数的绝对值或指对不等式，未对参数分类讨论或讨论不完整，漏掉临界情况。
