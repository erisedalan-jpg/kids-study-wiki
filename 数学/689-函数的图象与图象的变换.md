# 函数的图象与图象的变换

> **一句话**：函数的图象变换指通过平移、对称、伸缩、翻折等规则，由基本初等函数的图象得到一般函数图象的方法，是研究函数性质的重要工具。
> **English**: Function graph transformations refer to methods of obtaining the graph of a general function from basic elementary function graphs through translation, reflection, stretching, and absolute value folding.

## 🎓 考点精讲

- **定义/原理**：设已知 $y=f(x)$ 的图象，则常见变换规则为：
  - **平移**：$y=f(x+a)$ 将图象向左 ($a>0$) 或向右 ($a<0$) 平移 $|a|$ 个单位；$y=f(x)+b$ 将图象向上 ($b>0$) 或向下 ($b<0$) 平移 $|b|$ 个单位。
  - **对称**：$y=-f(x)$ 与 $y=f(x)$ 关于 $x$ 轴对称；$y=f(-x)$ 关于 $y$ 轴对称；$y=-f(-x)$ 关于原点对称；$y=f^{-1}(x)$ 与 $y=f(x)$ 关于直线 $y=x$ 对称。
  - **伸缩**：$y=Af(x)$ ($A>0$)：当 $A>1$ 时，纵坐标伸长为原来的 $A$ 倍，当 $0<A<1$ 时，纵坐标缩短为原来的 $A$ 倍；$y=f(\omega x)$ ($\omega>0$)：当 $\omega>1$ 时，横坐标缩短为原来的 $\frac{1}{\omega}$，当 $0<\omega<1$ 时，横坐标伸长为原来的 $\frac{1}{\omega}$。
  - **翻折**：$y=|f(x)|$ 保留 $x$ 轴上方部分，并将下方部分沿 $x$ 轴翻折上去；$y=f(|x|)$ 保留右侧部分，并关于 $y$ 轴对称到左侧（去掉原左侧部分）。

- **关键方法**：解决图象变换问题遵循“分步变换，先平移后伸缩，注意顺序”的原则。对于复合变换 $y=Af(\omega x+\varphi)+b$，通常先处理 $x$ 的系数：将 $y=f(x)$ 左移 $|\frac{\varphi}{\omega}|$ ($\varphi>0$ 左移)，再横坐标伸缩，然后纵坐标伸缩，最后上/下平移。严格比较不同变换顺序对结果的影响，避免机械记忆出错。

- **典型应用**：高考中常见题型有：（1）给出函数解析式如 $y=2\sin(2x+\frac{\pi}{3})-1$，要求画出或选择其图象，需从 $y=\sin x$ 出发依次平移、伸缩得到；（2）已知部分图象推断初等函数间的变换关系，如判断指数函数 $y=a^x$ 经过怎样的平移、伸缩可得到 $y=b\cdot a^{x+c}+d$ 的图象；（3）利用图象变换直观解决方程根的个数问题，如 $|x^2-2x|=k$ 根的个数可由 $y=|x^2-2x|$ 与 $y=k$ 图象交点个数得出。

## ⚠️ 高频易错点

- 左右平移混淆：将 $f(x+2)$ 误认为向右平移，正确是左移 2 个单位；“左加右减”对应自变量 $x$ 的改变。
- 伸缩变换弄反系数：$y=f(2x)$ 是横坐标缩短到原来的 $\frac12$，而非伸长；$y=\frac12 f(x)$ 是纵坐标缩短，而非伸长。
- 翻折顺序不当：绘制 $y=|\log_2(x+1)|$，应先画出 $y=\log_2(x+1)$，再保留上方部分并将下方翻折；若先翻折再平移会导致解析式错误。
- 对称与函数性质混淆：函数满足 $f(1+x)=f(1-x)$ 表示图象关于直线 $x=1$ 对称，而非对称变换；需区分图象自身的对称性与图象间的对称变换。
- 复合变换忽略系数影响：由 $y=f(x)$ 到 $y=f(2x-1)$ 时，若是“先平移再伸缩”，平移量应为 $\frac12$ 而非 1，即 $f[2(x-\frac12)]$。
