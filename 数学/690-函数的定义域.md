# 函数的定义域

> **一句话**：函数的定义域是指使得函数解析式有意义（或满足实际问题约束）的所有自变量取值的集合。
> **English**: The domain of a function is the set of all possible input values (x-values) for which the function is defined and produces a real output.

## 🎓 考点精讲

**定义/原理**

在函数 $y = f(x), x \in A$ 中，$x$ 叫作自变量，$x$ 的取值范围 $A$ 叫作函数的定义域。在高考的纯数学背景下，求定义域的核心依据是：使得解析式中的每一部分都有意义的自变量的取值集合的交集。
常见的有意义条件包括：
1. 分式：分母不为零，即 $\frac{1}{g(x)}$ 要求 $g(x) \neq 0$
2. 偶次根式：被开方数非负，即 $\sqrt[2n]{g(x)}$ 要求 $g(x) \ge 0$
3. 对数式：真数大于零，底数大于零且不等于 $1$，即 $\log_{a}{g(x)} \ (a>0, a\neq1)$ 要求 $g(x) > 0$
4. 零指数幂：底数不为零，即 $[g(x)]^0$ 要求 $g(x) \neq 0$

若函数由实际问题给出，定义域还需满足实际意义的约束，如长度、人数需为非负整数等。

**关键方法**

求函数定义域的标准步骤：
1. **列约束条件**：将解析式中涉及的所有限制条件（分式、根式、对数等）全部列出为不等式或不等式组。
2. **解不等式组**：求解所有不等式的交集。
3. **规范表示**：最终结果须写成集合或区间的形式，单一区间使用 $(a,b), [a,b]$ 等；多个区间的并集用 $\cup$ 连接。

**典型应用**

高考对定义域的考查通常不单独成题，而是渗透于选填小题、解答题的求解基础步骤或参数讨论中。
- **具体函数求值**：设问“求函数 $y = \frac{\sqrt{4-x^2}}{\ln(x+1)}$ 的定义域”。解题需联立不等式组 $\begin{cases} 4-x^2 \ge 0 \\ x+1 > 0 \\ \ln(x+1) \neq 0 \end{cases}$，解出 $x \in (-1, 0) \cup (0, 2]$。
- **复合函数定义域**：设问“已知 $f(x)$ 的定义域为 $[0,4]$，求 $f(2x-1)$ 的定义域”。关键理解“对应法则 $f$ 的作用范围不变”，即令 $0 \le 2x-1 \le 4$，解出 $x \in [\frac{1}{2}, \frac{5}{2}]$。

## ⚠️ 高频易错点

- **忘记对数真数必须严格大于零**：遇到 $\log_a{x}$ 时，误将 $x \ge 0$ 当作有意义的条件，导致定义域端点错误。
- **忽略分母不为零与根式内非负的联立**：对于 $y = \frac{\sqrt{x}}{x-1}$ 这类同时含根式和分式的函数，解 $x \ge 0$ 后忘记剔除 $x = 1$，错写成 $[0, +\infty)$。
- **复合函数定义域混淆**：混淆“$f(x)$ 的定义域”与“$f(g(x))$ 的定义域”。前者指 $x$ 的范围，后者在求解时令 $g(x)$ 落在前者范围内，解出的仍是新自变量 $x$ 的范围，而非原 $x$ 或 $g(x)$ 的范围。
- **结果表示不规范**：运算正确但最终未写为集合或区间，或并集符号误用为逗号、交集符号，或端点开闭取错。
