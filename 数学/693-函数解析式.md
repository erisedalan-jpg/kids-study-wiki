# 函数解析式

> **一句话**：函数解析式是用数学运算符号和常数将自变量与因变量联系起来的等式，如 $y=f(x)$。
> **English**: A function expression is an equation that defines the dependent variable in terms of the independent variable using mathematical operations and constants.

## 🎓 考点精讲

- **定义/原理**：函数的解析式（也称解析表达式）是函数的表示法之一，它通过加、减、乘、除、乘方、开方、指数、对数、三角等运算，把自变量 $x$ 与因变量 $y$ 之间的对应关系明确写成一个公式。例如一次函数 $f(x)=2x+3$，二次函数 $g(x)=x^2-4x+1$，指数函数 $h(x)=2^{x}$ 等。解析式必须指明自变量的允许范围——定义域，通常要求解析式自身有意义（分母不为零、偶次根号下非负、对数真数大于零、底数大 $0$ 且不等于 $1$ 等）。

- **关键方法**：求函数解析式是高考高频考点，常用方法有：
  1. **待定系数法**：已知函数类型（如一次、二次、指数型、对数型、幂函数等），设出含待定系数的标准形式，再利用已知条件列出方程（组）求解系数。例：已知 $f(x)$ 为一次函数，且 $f(f(x))=4x+3$，设 $f(x)=ax+b$，代入得 $a(ax+b)+b=a^2x+ab+b=4x+3$，对比系数求出 $a,b$。
  2. **换元法（凑配法）**：已知复合函数 $f(g(x))$ 的解析式，令 $t=g(x)$ 反解出 $x$，代入得到 $f(t)$，最后将 $t$ 换回 $x$，并注明新定义域。注意换元后新元的取值范围必须与原复合部分的值域一致。
  3. **方程组法**：当条件中出现 $f(x)$ 与 $f(-x)$、$f(\frac{1}{x})$ 等的关系时，用 $\frac{1}{x}$ 或 $-x$ 替换 $x$，构造方程组消去不需要的部分，解出 $f(x)$。
  4. **赋值法**：对于抽象函数条件，可取特殊值（如 $x=0,1,-1$ 等）逐步推出解析式。

- **典型应用**：高考常以选择题、填空题或解答题的第一问出现：
  - 给出一组对应值或图象特征，求函数解析式；
  - 给出 $f(x)$ 与 $f(g(x))$ 的关系，求 $f(x)$；
  - 结合函数的奇偶性、对称性、周期性求某区间上的解析式；
  - 已知函数模型（如指数增长、对数关系）还原实际问题的解析式。

## ⚠️ 高频易错点

- **忽视定义域**：求出解析式后未考虑原题隐含的自变量限制（分式分母不为零、偶次根号下非负、对数的真数条件等），导致答案不完整。
- **换元不注意新元范围**：令 $t=g(x)$ 后，没有由 $x$ 的范围求出 $t$ 的取值范围，造成解析式对应的定义域错误。
- **待定系数法设错函数形式**：例如二次函数错误设为 $y=ax^2+bx+c$ 而忘记条件中“顶点在原点”时应设为 $y=ax^2$；指数函数设成 $y=ka^x$ 却忽略了指数的系数。
- **方程组法代换错误**：用 $-x$ 代替 $x$ 时，只替换了部分 $x$，或忘记了函数方程的整体结构，导致方程组不对称。
