# 圆的周长与面积

> **一句话**：圆的周长 C = 2πr，面积 S = πr²——两个公式都靠"π"把圆的弯弯曲曲的边界转化成可以计算的数字。
> **English**: A circle's circumference is C = 2πr and its area is S = πr² — both formulas use π to turn a curved boundary into something we can calculate.

---

## 🧒 给 3-6 岁（共读版）

画一个圆——它只有一条弯弯的边，中间是满满的一片。

圆有几个重要的部分：
- **圆心（center）**：圆正中间那个点
- **半径（radius）**：从圆心到边上的距离，简写 **r**
- **直径（diameter）**：穿过圆心从这边到那边的距离，简写 **d**，d = 2r（直径是半径的 2 倍）

**共读小活动：** 找一枚一元硬币，用直尺量量它的直径大约是多少毫米。再折一折纸，想想"如果半径是 1 厘米，直径是多少"？

## 📚 给 6-12 岁（自读版）

### 圆的基本元素

| 名称 | 符号 | 说明 |
|------|------|------|
| 圆心 | O | 圆的中心点 |
| 半径 | r | 圆心到圆上任意一点的距离（每条都等长） |
| 直径 | d | 过圆心的最长弦，d = 2r |
| 弧 | — | 圆上两点之间的曲线部分 |
| 弦 | — | 圆上两点之间的线段（不一定过圆心） |

### 圆的周长公式

$$C = \pi d = 2\pi r$$

**推导思路：** 由圆周率的定义 π = C/d，可得 C = πd。因为 d = 2r，所以 C = 2πr。

> **例 1**：一个圆的半径 r = 5 cm，求周长。
> C = 2 × 3.14 × 5 = **31.4 cm**

> **例 2**：一个圆的直径 d = 8 m，求周长。
> C = 3.14 × 8 = **25.12 m**

**逆用公式：**
- 已知周长求半径：$r = \dfrac{C}{2\pi}$
- 已知周长求直径：$d = \dfrac{C}{\pi}$

### 圆的面积公式

$$S = \pi r^2$$

**推导方法——化圆为长方形：**

把圆剪成许多小扇形（比如 16 个），把这些扇形交错拼接起来，会得到一个近似的**长方形**：

- **长** ≈ 圆周长的一半 = πr
- **宽** = r（半径）
- 面积 = 长 × 宽 = πr × r = **πr²**

扇形越多，拼出来的形状越接近长方形，面积公式也越精确。

> **例 3**：一个圆的半径 r = 3 cm，求面积。
> S = 3.14 × 3² = 3.14 × 9 = **28.26 cm²**

> **例 4**：已知直径 d = 10 cm，求面积。
> r = d/2 = 5 cm，S = 3.14 × 5² = **78.5 cm²**
> ⚠️ 注意：公式中的 r 是**半径**，给出直径要先除以 2！

### 常见组合图形

**半圆：** 周长 = 直径 + 半个圆周 = $2r + \pi r$；面积 = $\dfrac{\pi r^2}{2}$

**圆环：** 面积 = 大圆面积 − 小圆面积 = $\pi R^2 - \pi r^2 = \pi(R^2 - r^2)$

## 🎓 给 12+（进阶版）

### 积分推导圆面积

用积分，可以精确证明 $S = \pi r^2$：

将半径为 r 的圆分成无数个同心环，每个环的半径为 x（0 ≤ x ≤ r），宽度为 dx，面积约为 $2\pi x \cdot dx$：

$$S = \int_0^r 2\pi x \, dx = 2\pi \cdot \frac{x^2}{2} \bigg|_0^r = \pi r^2$$

这正是初中化圆为长方形方法的极限形式——把无穷多个极细的环展开，铺成一个直角三角形，底为 $2\pi r$，高为 r，面积 $= \dfrac{1}{2} \cdot 2\pi r \cdot r = \pi r^2$。

### 球、圆柱、圆锥的关系

设半径均为 r，高为 2r（圆柱高 = 直径）：

| 图形 | 体积 |
|------|------|
| 圆柱 | $\pi r^2 \cdot 2r = 2\pi r^3$ |
| 球 | $\dfrac{4}{3}\pi r^3$ |
| 圆锥（高=2r） | $\dfrac{1}{3}\pi r^2 \cdot 2r = \dfrac{2}{3}\pi r^3$ |

**阿基米德发现**：等高等底时，圆柱 : 球 : 圆锥 = 3 : 2 : 1！这是他最引以为豪的发现，甚至刻在了他的墓碑上。

### 等周不等积（等周问题）

**等周定理（Isoperimetric Theorem）**：在所有周长相同的平面图形中，**圆的面积最大**。

数学上：对任意平面封闭曲线，若周长为 L，面积为 A，则：

$$4\pi A \leq L^2$$

等号当且仅当图形为圆时成立。这解释了为什么肥皂泡是球形（在相同的体积下，球的表面积最小——三维版的等周定理）。

### 欧拉恒等式与圆

复数 $e^{i\theta} = \cos\theta + i\sin\theta$（欧拉公式）描述了单位圆上的点：

当 θ 从 0 变化到 2π，$e^{i\theta}$ 绕单位圆转一圈，"绕行"的弧长正是 $2\pi r = 2\pi$（r=1）。

圆不只是几何图形，更是复数指数函数的几何化身。

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么圆面积公式是 πr²？"** → 可以说："我们可以把圆切成很多细细的扇形，像花瓣一样交错拼起来，就变成了一个长方形：长是半个周长（πr），宽是半径（r），面积就是 πr × r = πr²！"

- 孩子可能问 **"算面积时为什么要用半径不是直径？"** → 可以说："公式 S=πr² 里用的是 r（半径）。如果题目给的是直径，要先把直径除以 2 变成半径，这是最常见的错误！"

- 孩子可能问 **"圆有无穷多条半径吗？"** → 可以说："是的！从圆心出发，朝任何方向画线到圆上都是一条半径，可以画无穷多条，而且每条长度都相同——这正是圆最特别的地方。"

- **共读小活动：** 用圆规画一个半径 3 cm 的圆，让孩子量量周长（用线绕一圈再量直线长度），算算 3.14×6，和实测比较误差。
