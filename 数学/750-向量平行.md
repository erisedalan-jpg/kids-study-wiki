# 向量平行

> **一句话**：若存在实数 $\lambda$ 使 $\vec{a} = \lambda \vec{b}$ ($\vec{b} \neq \vec{0}$)，则 $\vec{a}$ 与 $\vec{b}$ 平行（共线）；零向量平行于任意向量。
> **English**: Two vectors are parallel (or collinear) if one is a scalar multiple of the other; the zero vector is parallel to every vector.

## 🎓 考点精讲

**定义与充要条件**  
对于非零向量 $\vec{a}, \vec{b}$，$\vec{a} \parallel \vec{b} \iff \exists \lambda \in \mathbb{R}, \lambda \neq 0$，使得 $\vec{a} = \lambda \vec{b}$。本质是两向量方向相同或相反。  
坐标形式：若 $\vec{a} = (x_1, y_1)$，$\vec{b} = (x_2, y_2)$，则 $\vec{a} \parallel \vec{b} \iff x_1 y_2 - x_2 y_1 = 0$（交叉积为零），即对应坐标成比例（需讨论分母为零的情形）。对于三维向量 $(x_1,y_1,z_1)$ 与 $(x_2,y_2,z_2)$，条件为 $x_1:y_1:z_1 = x_2:y_2:z_2$。

**关键方法**  
1. **向量法判定**：直接寻找或证明存在实数 $\lambda$ 使得等式成立，常用于含参数或抽象向量关系题。  
2. **坐标法判定**：计算 $x_1 y_2 - x_2 y_1$ 是否为零，是向量平行最直接的代数工具。注意在几何题中先建系，将几何条件转化为坐标运算。  
3. **三点共线转化**：$A,B,C$ 三点共线 $\iff \overrightarrow{AB} \parallel \overrightarrow{AC} \iff \overrightarrow{AB} = \lambda \overrightarrow{AC}$，这是高考解析几何与平面几何证明的高频转化技巧。

**典型应用**  
高考常将向量平行与以下场景结合：  
- **求参数值**：已知两向量平行，利用坐标关系式列方程求解未知参数（如 $k$）。  
- **证明三点共线**：在三角形、四边形或圆锥曲线背景下，通过向量平行证明点共线，进而得出几何性质（如梯形判定、中线性质）。  
- **向量分解与基底**：若 $\vec{a} \parallel \vec{b}$，则它们不能作为平面的一组基底，高考中用于判断基底有效性或向量唯一分解的条件。

## ⚠️ 高频易错点

- **忽略零向量讨论**：解题时直接两边除以向量，未分情况讨论 $\vec{b} = \vec{0}$ 的情形，导致丢解或论证不严谨。  
- **坐标成比例与交叉积混用**：使用 $\frac{x_1}{x_2} = \frac{y_1}{y_2}$ 时未考虑分母为零，正确应使用 $x_1 y_2 - x_2 y_1 = 0$ 并单独处理零坐标。  
- **方向混淆**：平行包括同向与反向。当 $\lambda > 0$ 时同向，$\lambda < 0$ 时反向。涉及“方向相同”“方向相反”的条件限制时，仅得出平行结论会漏掉方向判定。  
- **三维向量条件误用**：误将二维的“交叉积为零”直接套用到三维，三维向量平行需对应坐标成比例，而不能仅用一个行列式判断。
