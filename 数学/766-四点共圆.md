# 四点共圆

> **一句话**：如果平面上四个点位于同一个圆上，则称这四点共圆。
> **English**: Four points are concyclic if they all lie on the same circle.

## 🎓 考点精讲

**定义/原理**  
四点共圆是指平面上四个点 `A, B, C, D` 到某一定点 `O` 的距离相等，即 `OA = OB = OC = OD = r`，等价于存在一个圆能同时经过这四个点。  

**关键判定方法**  
1. **对角互补**：若四边形 `ABCD` 中，`∠A + ∠C = 180°` 或 `∠B + ∠D = 180°`，则 `A, B, C, D` 四点共圆。  
2. **外角等于内对角**：若 `∠CBE = ∠D`（`E` 在 `AB` 延长线上），则 `A, B, C, D` 共圆。  
3. **同弧所对圆周角相等**：若 `∠ADB = ∠ACB`，且 `A, B` 同侧，则 `A, B, C, D` 共圆。  
4. **托勒密定理**：对于凸四边形 `ABCD`，若 `AB·CD + AD·BC = AC·BD`，则 `A, B, C, D` 共圆（充要条件）。  
5. **两点连线的同侧张角相等**：若线段 `AB` 同侧的两点 `C, D`，满足 `∠ACB = ∠ADB`，则 `A, B, C, D` 共圆。  

**典型应用**  
高考中常在以下两类问题中出现：  
- **几何证明**：给出几个角的条件，要求证明四个点共圆，进而导出新的角相等或线段比例关系。例如，已知 `△ABC` 的高 `AD, BE` 交于 `H`，证明 `D, C, E, H` 四点共圆，并求某个角。  
- **轨迹与最值**：在动点问题中，利用四点共圆将角的关系转化为圆的性质，简化轨迹方程或求最值。解题关键是将角度相等或互补关系快速转化为共圆判定，再结合圆周角定理、相似等继续推理。

## ⚠️ 高频易错点

- **混淆判定方向**：把“对角互补”当成唯一判定，忽略“外角等于内对角”或“同弧所对圆周角相等”的等价形式，导致无法识别隐蔽的共圆条件。  
- **忽视点的顺序与同侧**：使用“同弧所对圆周角相等”时，未确认两点在弦的同侧，从而错误判定共圆。  
- **误用托勒密定理**：托勒密定理为充要条件，但只有凸四边形时才直接等价于共圆；对于四点不按正确顺序连接，公式形式会不同，易出错。  
- **遗漏退化情况**：当四点中有三点共线时，仍强行套用共圆判定，实际上共线情况不是常规意义上的四点共圆，需说明特殊性。
