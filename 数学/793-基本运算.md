# 基本运算

> **一句话**：基本运算涵盖实数与复数系中的加、减、乘、除、乘方、开方及指数、对数等运算，是一切代数计算的基础。
> **English**: Basic operations include addition, subtraction, multiplication, division, exponentiation, root extraction, and exponential and logarithmic operations, governed by operational laws and order.

## 🎓 考点精讲

**定义/原理**  
基本运算以实数系与复数系为载体，核心运算律包括：  
- 交换律 $a + b = b + a$，$ab = ba$；  
- 结合律 $(a + b) + c = a + (b + c)$，$(ab)c = a(bc)$；  
- 分配律 $a(b + c) = ab + ac$。  
运算顺序：先乘方、开方，再乘除，最后加减，有括号先算括号内。  
指数运算满足 $a^m \cdot a^n = a^{m+n}$，$(a^m)^n = a^{mn}$，$(ab)^n = a^n b^n$（$a>0$ 或指数为整数时可推广至全体实数）。对数运算满足 $\log_a(MN) = \log_a M + \log_a N$，$\log_a\frac{M}{N} = \log_a M - \log_a N$，$\log_a M^n = n\log_a M$（其中 $M>0, N>0, a>0 \text{且} a \neq 1$）。复数四则运算中 $i^2 = -1$，除法需分母实数化 $\frac{a+bi}{c+di} = \frac{(a+bi)(c-di)}{c^2+d^2}$。

**关键方法**  
- 恒等变形：应用运算律与公式化简代数式，优先观察整体结构，如利用平方差、完全平方公式分解。  
- 定义域优先：涉及偶次根式需满足被开方数非负；对数运算真数必须大于零；零次幂底数不为零。  
- 复数运算规范化：遇 $i^n$ 先利用 $i^4=1$ 降次；除法必须写为 $a+bi$ 形式。  

**典型应用**  
高考中该考点通常单独出现在选择、填空的前几题，或融入解答题计算过程。  
- **集合运算**：求交集时解不等式或方程，最终结果用区间或列举表示，如 $A=\{x\mid x^2<4\}, B=\{x\mid 2^x>1\}$，则 $A\cap B=(0,2)$。  
- **复数运算**：计算 $(1+i)^2 = 2i$，或者 $\frac{2+i}{1-i} = \frac{(2+i)(1+i)}{2} = \frac{1+3i}{2}$。  
- **指对数求值**：$\log_2 8 + 2^3 = 3+8=11$；若 $3^a=5$，则 $a=\log_3 5$，进一步求 $9^a=(3^2)^a=3^{2a}=25$。  
此类设问集中考查学生代数符号的准确操作能力，是后续函数、方程、不等式等大题的基础计算环节。

## ⚠️ 高频易错点

- **对数与指数忽视定义域**：解对数方程 $\log_2 x^2 = 2\log_2 x$ 时，左边定义域为 $x\neq 0$，右边定义域为 $x>0$，直接消去 $\log_2$ 会漏掉 $x<0$ 的情况。  
- **去括号与符号错误**：如 $-(x-1) = -x+1$ 而不是 $-x-1$；乘法分配律 $a(b+c)$ 中，系数为负时易漏乘符号。  
- **分数指数幂与根式混淆**：$\sqrt[3]{x^2} = x^{2/3}$ 定义域为 $\mathbb{R}$，但写成 $(x^2)^{1/3}$ 时若误当作 $(x^{1/3})^2$ 会在 $x<0$ 时引发错误（尽管最终结果等价，但运算律使用需注意 $a^{mn}=(a^m)^n$ 在 $a<0, m, n$ 非整数时受限）。  
- **复数除法运算**：没有乘以分母的共轭复数就进行加减，或计算出 $\frac{1}{i}$ 时不能正确化为 $-i$。
