# 复数的乘除

> **一句话**：复数乘法遵循多项式乘法法则，并将 `i²` 替换为 `-1`；复数除法需将分子分母同乘分母的共轭复数，使分母实数化。
> **English**: Multiplication of complex numbers follows polynomial expansion with `i² = -1`; division is performed by multiplying numerator and denominator by the complex conjugate of the denominator.

## 🎓 考点精讲

### 定义与法则

设 `z₁ = a + bi`, `z₂ = c + di` (a, b, c, d ∈ **R**)，则：

- **乘法**： `z₁·z₂ = (a+bi)(c+di) = (ac - bd) + (ad + bc)i`
- **除法**： `z₁ / z₂ = (a+bi)/(c+di) = (ac+bd)/(c²+d²) + (bc-ad)/(c²+d²)i`  (`z₂ ≠ 0`)

核心依据是 `i² = -1`，乘除运算最终结果均化为 `x + yi` 的标准代数形式。

### 关键方法

1. **乘法直接展开**：按多项式乘法展开，合并实部与虚部。特别可利用公式：`(1±i)² = ±2i`；`(a+bi)(a-bi) = a²+b²` 快速化简。
2. **分母实数化**：除法运算必须将分子、分母同乘分母的 **共轭复数** `c - di`，利用 `(c+di)(c-di) = c²+d²` 消去分母的虚部。
3. **三角形式与乘法意义**（部分省份涉及）：若用模与辐角表示 `z = r(cosθ + i sinθ)`，乘法为模相乘、辐角相加；除法为模相除、辐角相减。适用于复数的乘方（棣莫弗定理）与复数开方。
4. **方程中的虚根**：实系数一元二次方程当 Δ<0 时，求根公式仍适用，两根为共轭复数。此时根的运算常涉及复数乘除与韦达定理。

### 高考典型应用

- **直接计算题**：如“复数 `(1+2i)/(1-i)` 的虚部为（ ）” 或 “`(1+i)(2-i) = ?`”，重在化简能力。
- **共轭与模长结合**：求 `z·\bar z`、`|z₁/z₂|` 等，多利用模的性质 `|z₁z₂| = |z₁||z₂|`、`|z₁/z₂| = |z₁|/|z₂|`，可简化计算。
- **方程与复数综合**：已知一个复数根，求其实系数方程的另一根及参数；或解含复数乘除的等式求参数取值范围。

## ⚠️ 高频易错点

- **i² 符号错误**：展开时忘记 `i² = -1`，导致实部符号弄反，如 `(a+bi)(c+di)` 忘记 `bd·i² = -bd`。
- **分母实数化错误**：只乘分母的共轭而忘记分子同乘，或写成乘原本分母；除法结果未化成 `a+bi` 形式就写上虚部。
- **混淆共轭与相反数**：求 `(1+2i)/(1-2i)` 时，错误地乘以 `1+2i` 而非 `1+2i` 的共轭（实际分母共轭就是 `1+2i`，此处非错但需明确概念）。容易在符号上把 `z` 的共轭记成 `-a-bi`。
- **实数化分母后忘记约分**：分子分母同乘共轭后，实部和虚部分别除以分母实数时出现计算或约分错误。
