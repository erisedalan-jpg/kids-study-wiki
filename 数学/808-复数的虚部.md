# 复数的虚部

> **一句话**：对于复数 \(z = a + bi\ (a,b\in\mathbb{R})\)，其中实数 \(b\) 称为复数 \(z\) 的虚部。
> **English**: For a complex number \(z = a + bi\ (a,b\in\mathbb{R})\), the real number \(b\) is called the imaginary part of \(z\).

## 🎓 考点精讲

- **定义/原理**  
  复数 \(z\) 的标准形式为 \(z = a + bi\)，其中 \(a,b\in\mathbb{R}\)，\(i\) 为虚数单位（\(i^2 = -1\)）。  
  此时 **虚部** 是指实数 \(b\)，记作 \(\operatorname{Im}(z) = b\)。  
  ⚠️ 虚部是实数 \(b\)，**不是** \(bi\) 或 “虚数部分”。

- **关键方法**  
  1. **化标准形式**：将所给复数通过代数运算、分母实数化或利用共轭复数，写为 \(a+bi\) 的确定形式；  
  2. **提取虚部**：直接读出 \(i\) 的系数 \(b\)，其符号必须保留；  
  3. **注意条件**：若题干涉及“虚部满足某范围”或“虚部为0”，需将条件翻译为关于实数 \(b\) 的方程或不等式。

- **典型应用**  
  高考中常见设问：  
  - 给出复数 \(z = (1+2i)(3-i)\)，求 \(\operatorname{Im}(z)\)；  
  - 已知 \(z = \frac{2-i}{1+i}\)，求其虚部；  
  - 设 \(z = m^2 - 1 + (m-1)i\) 的虚部小于 0，求实数 \(m\) 的取值范围。  
  解题核心均为先化简为标准形式，再指明实数系数 \(b\)。

## ⚠️ 高频易错点

- **把虚部写成 \(bi\)**：错将 \(b\) 连同虚数单位 \(i\) 一起作为答案，高考填空必须写实数 \(b\)。
- **虚部符号遗漏或写错**：化简过程中进行复数乘除、分母有理化时，常因虚数单位的幂次运算失误导致虚部符号错误。
- **混淆纯虚数条件**：一个复数为纯虚数，要求 \(\operatorname{Re}(z)=0\) 且 \(\operatorname{Im}(z)\neq0\)，学生常漏掉虚部不为 0 的限制。
- **忽略系数是实数**：当复数包含参数时，未注意到虚部表达式必须是实数，例如 \(\sqrt{k}\) 出现时未考虑 \(k\ge 0\) 的隐含条件。
