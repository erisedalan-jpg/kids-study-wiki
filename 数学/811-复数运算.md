# 复数运算

> **一句话**：复数运算是在虚数单位 `i`（`i² = -1`）下对形如 `a+bi (a,b∈R)` 的数进行加、减、乘、除、乘方等代数操作及几何变换，是高考基础且必考的运算工具。  
> **English**: Complex number operations refer to algebraic manipulations (addition, subtraction, multiplication, division, exponentiation) and geometric transformations of numbers of the form a+bi under the imaginary unit i where i² = -1, a fundamental and mandatory tool in the Gaokao.

## 🎓 考点精讲

**定义与原理**  
复数 `z` 的标准代数形式为 `z = a + bi`（`a, b ∈ R`），其中 `a` 为实部，`b` 为虚部（不含 `i`）。虚数单位满足 `i² = -1`，且 `i` 的幂具有周期性：`i⁴ⁿ = 1`，`i⁴ⁿ⁺¹ = i`，`i⁴ⁿ⁺² = -1`，`i⁴ⁿ⁺³ = -i`。  
核心运算公式：  
- 加减：`(a+bi) ± (c+di) = (a±c) + (b±d)i`  
- 乘法：`(a+bi)(c+di) = (ac - bd) + (ad + bc)i`  
- 除法：`(a+bi)/(c+di) = (a+bi)(c-di)/(c²+d²) = (ac+bd)/(c²+d²) + (bc-ad)/(c²+d²)i` （分母实数化）  
共轭复数 `\bar{z} = a - bi`，模 `|z| = √(a²+b²)`。常用性质：`z·\bar{z} = |z|²`，`|z₁z₂| = |z₁||z₂|`，`|z₁/z₂| = |z₁|/|z₂|(z₂≠0)`。

**关键方法**  
1. **分母实数化**：除法运算时分子分母同乘分母的共轭复数。  
2. **复数相等**：`a+bi = c+di ⇔ a=c 且 b=d`，用于求解含参数的复数方程。  
3. **利用模与共轭整体处理**：若已知条件为 `|z|=1` 或 `z·\bar{z}=1`，可设 `z = x+yi` 或利用性质简化。  
4. **i 的周期性**：将高次幂转化为低次幂计算。

**典型应用（高考设问）**  
- **直接运算求值**：如 “若 `z = (1+i)/(1-i)`，则 `z²⁰ = ?`”，需先化简再结合周期性。  
- **复数相等求参数**：由两个复数相等建立方程组，解出实部、虚部中的未知数。  
- **复平面与几何意义**：判断复数对应点在第几象限，或由条件（如 `|z-(1+i)|=1`）确定动点轨迹（圆）。

## ⚠️ 高频易错点

- **虚部表述错误**：复数 `z = 3+4i` 的虚部是 `4`，而非 `4i`。高考填空题常在此设陷阱。  
- **除法漏乘共轭**：计算 `(1+2i)/(2-i)` 时，只对分母或分子单独乘共轭，导致结果错误。必须分子分母同乘 `(2+i)`。  
- **i 的幂化简遗忘**：`i²⁰²³` 应化为 `i⁴ⁿ⁺³ = -i`，误算为 `-1` 或 `i`。  
- **模与平方混淆**：`|z|²` 等于 `z·\bar{z}`，但 `z²` 是复数乘法，两者一般不相等。例如 `z = 1+i`，`|z|² = 2` 但 `z² = 2i`。  
- **复数相等对应错误**：若 `a+bi = c+di`，必须同时满足实部与虚部分别相等，尤其含参数时易漏条件。
