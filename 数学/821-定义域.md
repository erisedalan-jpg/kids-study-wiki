# 定义域

> **一句话**：使函数解析式有意义的自变量 $x$ 的所有取值组成的集合。
> **English**: The set of all possible input values ($x$) for which a function is defined and yields a real number output.

## 🎓 考点精讲

**定义/原理**：给定函数 $y=f(x)$，定义域是自变量 $x$ 的取值范围，要求对应函数值 $y$ 为实数。常见限制条件源自基本初等函数的自然约束：

- 分式：分母 $\neq 0$
- 偶次根式：被开方数 $\geq 0$
- 对数：真数 $> 0$，底数 $>0$ 且 $\neq 1$
- 零指数幂：底数 $\neq 0$
- 正切函数：$x \neq \frac{\pi}{2}+k\pi,k\in\mathbb{Z}$

若函数由多个部分经四则运算或复合而成，定义域为各部分定义域的**交集**。

**关键方法**：求定义域的标准步骤为①识别解析式中每一类限制条件；②逐条转化为不等式 (组)；③解所有不等式；④取交集并在数轴上确认，最终用集合或区间表示。复合函数 $y=f(g(x))$ 的定义域不仅要使 $g(x)$ 本身有意义，还要使 $g(x)$ 的值落在 $f$ 的定义域内。

**典型应用**：高考常以选择题或填空题直接求具体函数定义域，或隐含于值域、单调性、奇偶性讨论中。例如：求 $f(x)=\sqrt{\ln(x-1)}+\frac{1}{x-3}$ 的定义域，需同时满足 $x-1>0$，$\ln(x-1)\ge 0$ 且 $x-3\neq 0$，解出 $x\ge 2$ 且 $x\neq 3$，写成 $[2,3)\cup(3,+\infty)$。

## ⚠️ 高频易错点

- 忽略对数底数的条件（默认只关注真数 $>0$），如 $y=\log_{x}(x+2)$ 需 $x>0$ 且 $x\neq 1$。
- 复合函数中误将 $f(g(x))$ 的定义域当成求 $g(x)$ 的值域，如已知 $f(x)$ 定义域为 $[-1,1]$，求 $f(2x+1)$ 定义域，应解 $-1 \le 2x+1 \le 1$，而非直接对 $2x+1$ 整体做限制以外的运算。
- 用“或”连接不等式解集，未取交集，导致定义域扩大。
- 结果未写成集合或区间形式，或在表示“排除某点”时区间开闭混淆。
