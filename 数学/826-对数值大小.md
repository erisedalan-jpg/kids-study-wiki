# 对数值大小

> **一句话**：比较对数值大小时，若底数相同则利用单调性比较真数，若真数相同则用换底公式或图象法，若底数和真数都不同则常找中间量（如0、1、$\frac{1}{2}$等）或作差/商构造函数。
> **English**: Comparing the magnitude of logarithmic values involves using monotonicity when bases are identical, the change-of-base formula or graphical methods when arguments are identical, and introducing an intermediate value or constructing a function when both base and argument differ.

## 🎓 考点精讲

**定义/原理**：
比较对数值大小的核心依据是对数函数的单调性与换底公式。设 $y = \log_a x$ ($a>0, a\neq 1$)：
- 当 $a>1$ 时，函数在 $(0,+\infty)$ 上单调递增，即 $x_1 > x_2 \iff \log_a x_1 > \log_a x_2$。
- 当 $0<a<1$ 时，函数在 $(0,+\infty)$ 上单调递减，即 $x_1 > x_2 \iff \log_a x_1 < \log_a x_2$。
换底公式 $\log_a b = \frac{\log_c b}{\log_c a}$（$c$ 为正常数且 $c\neq 1$）是统一底数或真数的桥梁，常用于将不同底对数化为同底比较，或化为同真数后利用倒数关系（$\log_a b = \frac{1}{\log_b a}$，$a,b>0$ 且均不为1）判断。

**关键方法**：
1. **底同真数不同**：直接应用单调性。
2. **真数同底数不同**：用换底公式化为同底，或利用图象法（画 $x$ 取定值的竖线，交点越高对数值越大），或根据 $a>b>1$ 时 $0<\log_a x < \log_b x$（$x>1$）等口诀判断。
3. **底和真数均不同**：首选“中间值法”。常以 $0$、$\pm 1$、$\pm\frac{1}{2}$ 为界，利用 $\log_a 1=0$ 和 $\log_a a=1$ 为桥梁。若此法失效，可采用作差法：$\log_a b - \log_c d$，通过换底、通分或与0比较；或作商法：$\frac{\log_a b}{\log_c d}$，与1比较大小。对于复杂情况，可构造函数 $f(x)=\log_x M$ 或 $f(x)=\log_a x - \log_b x$，利用导数分析单调性。

**典型应用**：
高考中常见题型为选择题或填空题，给出 $3\sim4$ 个对数式，要求排序。例如：已知 $a=\log_3 2$，$b=\ln 2$，$c=5^{-\frac{1}{2}}$，比较 $a,b,c$ 大小。先利用中间量：$a=\frac{1}{\log_2 3}$，$b=\frac{1}{\log_2 e}$，由 $e<3$ 知 $\log_2 e < \log_2 3$，则 $b > a$，且 $0<a,b<1$；而 $c=5^{-\frac{1}{2}}=\frac{1}{\sqrt{5}}$，介于 $0\sim1$ 之间，具体需估算或平方比较。最终常得出 $c<a<b$ 或类似结论。另一高频考法是“与已知数比大小”，如判断 $\log_3 2$、$\log_{0.5} 0.6$、$\log_{\sqrt{2}} \sqrt{3}$ 与 $0$、$1$、$\frac{1}{2}$ 的关系。

## ⚠️ 高频易错点

- **忘记讨论底数**：直接套用“增函数”结论比较真数，忽略底数 $0<a<1$ 时单调性反转。必须先分 $a>1$ 和 $0<a<1$ 两类。
- **误用中间量“1”**：认为所有对数与“1”的比较都得 $\log_a a$，但若底数与真数的大小关系不统一，如比较 $\log_2 3$ 与 $\log_3 2$，前者大于1，后者小于1，不能简单都用1衡量。
- **符号判断出错**：忽视“真数>1且底数>1”或“真数<1且底数<1”时对数值为正，其余组合为负的规律。例如 $\log_{\frac{1}{2}} 3<0$，$\log_3 \frac{1}{2}<0$。
- **换底后运算失误**：换底后取倒数时，忽略分子分母颠倒带来的不等号变化。如 $\log_2 3 > \log_3 2$ 换为 $\frac{1}{\log_3 2} > \log_3 2$ 后，需构造函数或配方法处理。
