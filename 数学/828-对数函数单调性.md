# 对数函数单调性

> **一句话**：对数函数 $y=\log_a x$ 的单调性由底数 $a$ 决定：当 $a>1$ 时在 $(0,+\infty)$ 上严格单调递增；当 $0<a<1$ 时严格单调递减。
> **English**: The monotonicity of the logarithmic function $y=\log_a x$ is determined by the base $a$: it is strictly increasing on $(0,+\infty)$ when $a>1$, and strictly decreasing when $0<a<1$.

## 🎓 考点精讲

- **定义/原理**：设对数函数 $f(x)=\log_a x$（$a>0$ 且 $a\neq 1$），定义域为 $(0,+\infty)$。  
  若 $a>1$，则对任意 $x_1>x_2>0$，有 $\log_a x_1 > \log_a x_2$；若 $0<a<1$，则 $\log_a x_1 < \log_a x_2$。  
  更一般地，对于复合函数 $y=\log_a g(x)$，先保证 $g(x)>0$ 得定义域，再结合 $a$ 与 $g(x)$ 的单调性利用“同增异减”法则判断整体单调性。  
  高考中常与指数函数的反函数性质联动，利用对数值比较大小、解不等式、求参数范围。

- **关键方法**：
  1. **比大小**：先将不同底的对数化成同底（常用换底公式 $\log_a b = \frac{\ln b}{\ln a}$），或利用中间量如 $0,1$ 进行估值；若底数和真数均不同，常借助函数单调性或图像。
  2. **解对数不等式**：$\log_a f(x) > \log_a g(x)$ 型，需分类讨论 $a>1$ 与 $0<a<1$，并注意真数大于 $0$ 的约束，得到等价不等式组。
  3. **求复合函数单调区间**：先求定义域，再拆分内外函数，牢记“同增异减”——内外单调性相同时整体递增，相异时整体递减。

- **典型应用**：  
  - **比较大小**：如比较 $\log_2 3$ 与 $\log_{0.5} 0.3$，可借助单调性与符号判断。  
  - **求参数范围**：已知 $\log_a \frac{2}{3} < 1$，求 $a$ 的取值范围。需对 $a>1$ 和 $0<a<1$ 分别讨论，结合单调性去掉对数符号求解。

## ⚠️ 高频易错点

- 忽略对数真数必须大于 $0$ 的前提，解不等式时遗漏定义域限制，导致范围扩大。
- 在复合函数中错误使用单调性判断：$y=\log_{0.5}(-x^2+4x-3)$，先求 $ -x^2+4x-3>0$ 得定义域 $(1,3)$，再注意到外函数以 $0.5$ 为底为减函数，内层二次函数在 $(1,2]$ 递增，在 $[2,3)$ 递减，因此整体在 $(1,2]$ 上递减，在 $[2,3)$ 上递增。学生常把增减区间写反。
- 底数含参数时忘记分类讨论，或讨论后不验证与条件的交集，导致取值范围错误。
- 换底公式应用不当，如错误地认为 $\log_a b = \frac{\log_c a}{\log_c b}$ 而导致符号混乱。
