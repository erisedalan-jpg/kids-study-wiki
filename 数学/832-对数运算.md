# 对数运算

> **一句话**：对数运算是基于对数的定义与运算性质，对含对数的式子进行化简、求值与证明的过程，核心依赖积、商、幂的对数法则及换底公式。
> **English**: Logarithmic operation refers to the manipulation and simplification of expressions involving logarithms, using properties such as product, quotient, and power rules, as well as the change-of-base formula.

## 🎓 考点精讲

对数运算根植于指数与对数的互逆关系：若 $a>0$ 且 $a \neq 1$，$N>0$，则 $a^x = N \iff x = \log_a N$。高考要求的运算性质如下：

设 $a>0, a \neq 1$，$M>0, N>0$：
1. 积的对数：$\log_a (MN) = \log_a M + \log_a N$
2. 商的对数：$\log_a \frac{M}{N} = \log_a M - \log_a N$
3. 幂的对数：$\log_a M^n = n \log_a M \ (n \in \mathbb{R})$
4. 换底公式：$\log_a b = \frac{\log_c b}{\log_c a}\ (c>0, c \neq 1)$，常用特例为 $\log_a b = \frac{\lg b}{\lg a}$ 或 $\frac{\ln b}{\ln a}$，由此还可推导出 $\log_a b = \frac{1}{\log_b a}$。

**关键方法**：
- **化简与求值**：将和、差、倍数形式的对数式合并为单个对数，或反向拆解。化简时须始终保证真数为正。
- **指数与对数的互化**：当题设出现指数连等式（如 $2^x = 3^y = 5^z$）时，令其等于 $k$，再取对数把指数问题转化为对数运算。
- **统一底数**：遇到不同底数的对数加减乘除，先利用换底公式化成同底（通常为 $10$ 或 $\mathrm{e}$），再进行计算或比较大小。
- **解对数方程/不等式**：利用对数性质化成代数方程，并必须检验真数大于 $0$ 和底数合法，排除增根。

**典型应用**：高考中常见已知 $\log_a m$ 和 $\log_a n$，要求用它们表示 $\log_a (m^p n^q)$，直接套用幂与积的法则；比较不同底数的对数大小时，通过换底统一为常用对数或利用中间量比较；求解含有 $\log$ 的方程或不等式，变形后落实定义域限制。

## ⚠️ 高频易错点

- **忽视定义域限制**：多次变形中忘记 $\log_a N$ 要求 $N>0$，解方程或不等式时未检验真数是否为正，从而引入增根或丢失解。
- **运算法则混淆**：误以为 $\log_a (M+N) = \log_a M \cdot \log_a N$，或把 $\log_a M \cdot \log_a N$ 等同于 $\log_a (MN)$，实际上和差与乘积之间没有直接的对数公式。
- **换底公式颠倒分子分母**：错写 $\log_a b = \frac{\log_c a}{\log_c b}$，导致求值结果为倒数。
- **幂的对数漏掉符号讨论**：将 $\log_a x^2$ 直接写成 $2\log_a x$ 而不考虑 $x$ 可能为负，丢失定义域；或处理 $\log_a \sqrt{x}$ 时忽略 $x>0$ 的前提。
