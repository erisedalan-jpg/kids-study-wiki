# 导数与单调性

> **一句话**：对于可导函数 f(x) 在区间 I 上，若 f'(x) > 0 则 f(x) 单调递增；若 f'(x) < 0 则 f(x) 单调递减；f'(x) = 0 的点可能为极值点或拐点。
> **English**: For a differentiable function f(x) on interval I, if f'(x) > 0, then f(x) is strictly increasing; if f'(x) < 0, then f(x) is strictly decreasing; points where f'(x) = 0 are candidates for local extrema or inflection points.

## 🎓 考点精讲

**定义/原理**：设函数 y = f(x) 在某个区间 I 内可导。

- 若在 I 上恒有 f'(x) > 0，则 f(x) 在 I 上严格单调递增。
- 若在 I 上恒有 f'(x) < 0，则 f(x) 在 I 上严格单调递减。
- 若在 I 上恒有 f'(x) = 0，则 f(x) 在 I 上为常值函数。

注意：上述结论在 f'(x) 只在有限个点处等于 0，其余点均大于零（或小于零）时，单调性仍成立。

**关键方法**：**利用导数判断单调性的标准三步法**
1. **求定义域**：明确函数 f(x) 的天然定义域。
2. **求导并解方程**：计算 f'(x)，求解方程 f'(x) = 0 的根，并以这些根和 f'(x) 无定义的点为界点，将定义域划分为若干子区间。
3. **列表判号**：在每个子区间内任取一个检测点 x₀，计算 f'(x₀) 的符号。若 f'(x₀) > 0，该区间递增；若 f'(x₀) < 0，该区间递减。

**典型应用**：高考常以以下形式考查：
- **求单调区间**：给定含参或不含参的函数表达式，直接套用上述三步法，以表格形式写出单调递增区间和单调递减区间。
- **已知单调性求参数范围**：题目条件如“f(x) 在 [a, b] 上单调递增”，则转化为在该区间上 f'(x) ≥ 0 恒成立，进而用参数分离或分类讨论求参数。例如，若 f(x) = x + a/x 在 [1, +∞) 上递增，由 f'(x) = 1 - a/x² ≥ 0 恒成立，可得 a ≤ x² 对 x ∈ [1, +∞) 恒成立，故 a ≤ 1。这是导数与不等式恒成立问题的经典结合。

## ⚠️ 高频易错点

- **混淆“f'(x) > 0” 与 “单调递增”**：f'(x) > 0 是严格单调递增的充分不必要条件。例如 f(x) = x³ 在 R 上单调递增，但 f'(0) = 0。高考填空或选择题常在此设陷阱，要求学生准确使用 ≥0 或 ≤0 来描述单调性与导数非负的关系。
- **忽略定义域直接求导判断**：在求导数 f'(x) 前未先写出函数的自然定义域（如对数真数大于零、分母不为零），导致将不在定义域内的区间误列为单调区间，造成整题失分。
- **以极值点直接划分单调区间时，忽略不可导点**：单调区间的分界点不仅包括 f'(x) = 0 的根，也包括 f'(x) 不存在的点（如果有的话，如含绝对值或分式指数的函数）。漏掉这些点会导致单调区间划分不全。
- **含参讨论时未对导函数的最高次项系数进行分类**：当 f'(x) 为含参二次式时，必须先讨论二次项系数是否为零、判别式的正负等情形，避免遗漏常值函数或一次函数的特殊情况。
