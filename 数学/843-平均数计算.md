# 平均数计算

> **一句话**：平均数是描述一组数据中心位置或集中趋势的统计量，通常指所有数据之和除以数据个数所得的算术平均数。
> **English**: The average (specifically the arithmetic mean) is a measure of central tendency, typically calculated as the sum of all values divided by the number of values.

## 🎓 考点精讲

**定义/原理**：  
对于 $n$ 个实数 $x_1, x_2, \dots, x_n$，其**算术平均数** $\bar{x}$ 定义为：  
$$\bar{x} = \frac{x_1 + x_2 + \cdots + x_n}{n}$$  
在处理分组数据时，可用**加权平均数**：若数据 $x_i$ 出现的频数为 $f_i$（或权数为 $w_i$），则 $\bar{x} = \frac{\sum x_i f_i}{\sum f_i}$。从几何角度，**几何平均数** $G = \sqrt[n]{x_1 x_2 \cdots x_n}$（$x_i > 0$）常用于平均增长率问题。三者满足均值不等式链：调和平均数 ≤ 几何平均数 ≤ 算术平均数 ≤ 平方平均数（等号当且仅当所有数据相等时成立）。

**关键方法**：  
1. **直接公式法**：已知所有原始数据，直接累加求平均。  
2. **加权修正法**：当部分数据的平均数已知时，利用“全体总和 = 各部分平均数 × 个数之和”反求或修正。  
3. **频数分布表法**：取组中值为代表值，乘以频数求和再除以总频数，得到近似平均数。  
4. **均值不等式应用**：在“定和求积最大”或“定积求和最小”型最值问题中，通过“正、定、等”三步判断并求解。

**典型应用**：  
- **样本特征估计**：选择题或填空题直接给出一组数据求平均数，或结合方差、标准差考查数据特征的综合计算。  
- **混合样本平均数**：如“高三（1）班男生平均分135，女生平均分125，已知男女人数比为3:2，求全班平均分”，需使用加权平均：$\frac{135 \times 3 + 125 \times 2}{5} = 131$。  
- **均值不等式求最值**：解答题中常出现“已知 $a > 0, b > 0, a + b = 1$，求 $\frac{1}{a} + \frac{1}{b}$ 的最小值”，利用算术-调和平均数关系或拆项后使用算术-几何平均数不等式求解。

## ⚠️ 高频易错点

- **忽略权重导致加权平均错用算术平均**：误将不同占比的群体平均数直接相加除以2，必须明确每次平均对应的“分母”是否相等。  
- **均值不等式忽略“正数”前提**：使用 $\frac{a+b}{2} \ge \sqrt{ab}$ 时未先验证 $a > 0, b > 0$，在参数含变量时尤其容易漏判。  
- **混淆几何平均与算术平均的适用场景**：平均增长率、平均利率应使用几何平均数 $\sqrt[n]{(1+r_1)(1+r_2)\cdots(1+r_n)}-1$，错误使用算术平均值会导致结果与实际复利增长不符。  
- **等号成立条件验证不全**：用均值不等式求最值时，只得出不等式方向，未检验“等于”是否能在定义域内取得，导致误判最值。
