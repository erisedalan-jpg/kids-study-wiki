# 平面向量数量积的坐标表示

> **一句话**：若 $\vec a=(x_1,y_1)$，$\vec b=(x_2,y_2)$，则 $\vec a\cdot\vec b=x_1x_2+y_1y_2$，即对应坐标乘积之和。
> **English**: If $\vec a=(x_1,y_1)$ and $\vec b=(x_2,y_2)$，then $\vec a\cdot\vec b=x_1x_2+y_1y_2$，the sum of the products of corresponding coordinates.

## 🎓 考点精讲

**定义/原理**  
在平面直角坐标系中，设向量 $\vec a=(x_1,y_1)$，$\vec b=(x_2,y_2)$，则它们的**数量积（点乘）的坐标表达式**为：
$$\vec a\cdot\vec b = x_1x_2 + y_1y_2$$
该公式由数量积的定义 $\vec a\cdot\vec b = |\vec a||\vec b|\cos\theta$ 结合单位正交基的性质推导得出。  
由坐标表示的模长公式 $|\vec a| = \sqrt{x_1^2+y_1^2}$，可进一步得到向量夹角的余弦：
$$\cos\theta = \frac{x_1x_2 + y_1y_2}{\sqrt{x_1^2+y_1^2}\sqrt{x_2^2+y_2^2}}$$
以及两向量垂直（正交）的充要条件：$x_1x_2 + y_1y_2 = 0$。

**关键方法**  
高考中处理平面向量数量积的坐标问题，基本步骤为：  
1. **将向量坐标化**：根据已知条件建立适当坐标系，把问题中的点写成坐标，进而得到向量的坐标。  
2. **代入坐标公式**：直接计算 $x_1x_2+y_1y_2$，求模长、夹角或判断垂直。  
3. **注意符号与运算次序**：坐标运算中常需结合向量的线性运算（加减、数乘），先化简向量再代入公式可减少计算量。

**典型应用**  
- **求数量积**：直接给出两端点坐标，先写向量坐标差，再代入 $x_1x_2+y_1y_2$。  
  例：已知 $A(1,2)$，$B(4,6)$，$\vec{AB}=(3,4)$；$C(2,1)$，$D(5,5)$，$\vec{CD}=(3,4)$，则 $\vec{AB}\cdot\vec{CD}=3\times3+4\times4=25$。  
- **判断垂直与求夹角**：若要证 $AB\perp CD$，只需验证 $(x_B-x_A)(x_D-x_C)+(y_B-y_A)(y_D-y_C)=0$；求 $\cos\angle(\vec a,\vec b)$ 则分母为模长乘积，分子为坐标积的和。  
- **求模长与距离**：由 $|\vec a|^2=\vec a\cdot\vec a=x_1^2+y_1^2$ 直接得到模长；两点 $P,Q$ 间距离即为 $|\overrightarrow{PQ}|$。

## ⚠️ 高频易错点

- **混淆数量积与向量加法坐标公式**：数量积结果是实数，对应坐标相乘再相加；向量加法是 $(x_1+x_2, y_1+y_2)$，切不可将数量积误写为 $(x_1x_2, y_1y_2)$。  
- **忘记向量坐标需用终点减起点**：直接用点的坐标做数量积，而未将点的坐标转化为向量的坐标差，导致计算错误。  
- **垂直判定漏用充要条件**：认为 $x_1x_2 + y_1y_2 = 0$ 仅是充分条件，忽略它也是必要条件，不敢直接使用。  
- **符号处理不当**：负坐标相乘时遗漏负号，如 $(-2)\times3$ 误作 $6$。
