# 弦化切

> **一句话**：利用同角三角函数关系，将含有正弦、余弦的表达式转化为只含正切的代数形式，常用于已知正切值求三角式的值。
> **English**: A technique that converts expressions involving sine and cosine into expressions solely in terms of tangent, typically used when the value of tanθ is given, by leveraging the identity sinθ/cosθ = tanθ.

## 🎓 考点精讲

**定义/原理**  
弦化切基于同角三角函数的基本关系：$\tan\alpha = \dfrac{\sin\alpha}{\cos\alpha}$ 以及 $\sin^2\alpha + \cos^2\alpha = 1$。对于关于 $\sin\alpha$ 与 $\cos\alpha$ 的齐次式（各项中 $\sin\alpha$ 与 $\cos\alpha$ 的次数之和相等），可通过分子分母同除以 $\cos\alpha$ 的适当次幂，将所有项统一成 $\tan\alpha$ 的表达式。

**关键方法**  
1. **判断齐次性**：现有分式或整式是否是关于 $\sin\alpha,\cos\alpha$ 的齐次式。一次齐次分式如 $\dfrac{a\sin\alpha + b\cos\alpha}{c\sin\alpha + d\cos\alpha}$，可直接分子分母同除以 $\cos\alpha$；二次齐次式如 $\sin^2\alpha,\ \sin\alpha\cos\alpha,\ \cos^2\alpha$，需用“1”的代换：$1 = \sin^2\alpha + \cos^2\alpha$，将目标式改写为分母为 $1$ 的分式，再同除以 $\cos^2\alpha$。  
2. **执行化切**：设 $\tan\alpha = t$，将分式化为 $\dfrac{A t + B}{C t + D}$ 或 $\dfrac{A t^2 + B t + C}{t^2 + 1}$ 等形式，代入已知值求值。  
3. **象限与符号**：使用弦化切时需注意原式中隐含的 $\cos\alpha \neq 0$，且最终结果的符号可能依赖于 $\alpha$ 所在象限，若题目未说明象限，需通过已知条件（如 $\tan\alpha$ 的正负、给定区间）判定。

**典型应用**  
- **给值求值**：已知 $\tan\alpha = 2$，求 $\dfrac{\sin\alpha + \cos\alpha}{\sin\alpha - \cos\alpha}$ 的值。分子分母同除以 $\cos\alpha$ 得 $\dfrac{\tan\alpha + 1}{\tan\alpha - 1} = \dfrac{2+1}{2-1}=3$。  
- **二次齐次式求值**：已知 $\tan\alpha = -\dfrac{1}{2}$，求 $\sin^2\alpha + \sin\alpha\cos\alpha - 2\cos^2\alpha$。原式改写为 $\dfrac{\sin^2\alpha + \sin\alpha\cos\alpha - 2\cos^2\alpha}{\sin^2\alpha + \cos^2\alpha}$，分子分母同除以 $\cos^2\alpha$ 得 $\dfrac{\tan^2\alpha + \tan\alpha - 2}{\tan^2\alpha + 1}$，代入即得值。  
高考中常与诱导公式、和差公式结合，要求先化简至齐次形式再弦化切。

## ⚠️ 高频易错点

- **未检查齐次性直接化切**：面对非齐次式（如 $\sin\alpha + \cos^2\alpha$）强行除以 $\cos\alpha$ 或 $\cos^2\alpha$，导致表达式变形错误。必须先通过“1”的代换或提取公因式构造齐次。  
- **忽略分母为零的情形**：弦化切时默认 $\cos\alpha \neq 0$，若题目未保证 $\cos\alpha \neq 0$，需单独讨论 $\alpha = \dfrac{\pi}{2} + k\pi$ 的情况，否则会漏解或产生无意义表达式。  
- **象限符号判断失误**：在开方或确定 $\sin\alpha,\cos\alpha$ 的符号时，仅依赖 $\tan\alpha$ 的值，忽视角的范围，导致结果符号错误。例如由 $\tan\alpha = \dfrac{1}{2}$ 无法唯一确定 $\sin\alpha$ 的符号，需辅以象限信息。  
- **整式化切时代换遗漏**：如求 $2\sin^2\alpha + \sin\alpha\cos\alpha$，忘记分母配 $1 = \sin^2\alpha + \cos^2\alpha$，直接除以 $\cos^2\alpha$ 得到 $2\tan^2\alpha + \tan\alpha$，丢失了分母的 $\tan^2\alpha+1$。
