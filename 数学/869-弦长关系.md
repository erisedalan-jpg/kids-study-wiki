# 弦长关系

> **一句话**：弦长关系是圆锥曲线中直线与曲线相交所得弦的长度与直线参数、曲线参数间的定量关系，核心工具是弦长公式 $|AB| = \sqrt{1 + k^2}\,|x_1 - x_2|$。
> **English**: Chord length relation refers to the quantitative relationship between the length of a chord formed by the intersection of a line and a conic section, typically expressed via the chord length formula $|AB| = \sqrt{1 + k^2}\,|x_1 - x_2|$.

## 🎓 考点精讲

**定义/原理**：给定圆锥曲线（椭圆、双曲线、抛物线）与一条不垂直坐标轴的直线 $y = kx + m$，联立消元后得一元二次方程 $ax^2 + bx + c = 0$（或 $ay^2 + by + c = 0$），设两交点横坐标为 $x_1, x_2$，则弦长
$$
|AB| = \sqrt{1 + k^2}\,|x_1 - x_2|
    = \sqrt{1 + k^2}\,\frac{\sqrt{\Delta}}{|a|}
    = \sqrt{1 + k^2}\,\sqrt{(x_1 + x_2)^2 - 4x_1x_2}.
$$
若直线为 $x = ty + n$（斜率表示为 $\frac{1}{t}$），对应弦长为 $\sqrt{1 + t^2}\,|y_1 - y_2|$。

圆锥曲线的**焦点弦**有特殊关系：椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2}=1$ 的焦点弦长若过焦点且倾斜角为 $\theta$，可用焦半径公式表示为 $\frac{2ab^2}{a^2 - c^2\cos^2\theta}$（待家长核对）；抛物线 $y^2 = 2px$ 的焦点弦长 $|AB| = x_1 + x_2 + p = \frac{2p}{\sin^2\alpha}$（$\alpha$ 为倾斜角）。

**关键方法**：
1. **设而不求 + 韦达定理**：直线与曲线联立，写出判别式 $\Delta > 0$ 确保相交，利用韦达定理求 $x_1+x_2$ 和 $x_1x_2$，直接代入弦长公式，无需解出交点坐标。
2. **点差法（中点弦）**：若问题涉及弦的中点，不宜用弦长公式硬算，而用点差法推出中点坐标与弦斜率的关系，再利用几何特征求解。
3. **焦点弦专用公式**：当明确直线过焦点时，优先使用焦点弦长公式可简化运算。

**典型应用**：高考解析几何大题中，常常直接要求“求弦长”或“求面积”（面积需弦长作为底），设问如“求直线 $l$ 被椭圆 $C$ 所截弦长的取值范围”或“当弦长为 $\frac{4\sqrt{2}}{3}$ 时，求直线方程”。另一常见考法是结合中点弦：证明某弦被定点平分时满足某种长度关系或斜率积为定值。

## ⚠️ 高频易错点

- **忽略斜率不存在的情况**：设直线为 $y = kx + m$ 时，必须单独讨论 $k$ 不存在（垂直于 $x$ 轴）的情形，此时弦长公式改为 $|y_1-y_2|$。
- **遗忘判别式 $\Delta > 0$**：联立方程后未验证 $\Delta > 0$ 就使用韦达定理，可能得到不存在的弦长。求参数的取值范围时尤其容易忽略。
- **焦点弦长公式记混参数**：椭圆、双曲线的焦点弦长公式形式相似但分母符号不同（双曲线用 $a^2 - c^2\cos^2\theta$ 还是 $c^2\cos^2\theta - a^2$），误用会导致符号错误；抛物线的焦点弦长公式中分母应为 $\sin^2\alpha$，易错记为 $\cos^2\alpha$。
- **点差法适用范围不清**：点差法只适用于弦的中点存在且斜率存在的情形，当弦与渐近线平行或直线垂直于坐标轴时需单独处理。
