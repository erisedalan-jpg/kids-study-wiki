# 抛物线方程

> **一句话**：平面内与一个定点 $F$（焦点）和一条定直线 $l$（准线，$F\notin l$）距离相等的点的轨迹叫做抛物线。
> **English**: A parabola is the set of all points in a plane that are equidistant from a fixed point (the focus) and a fixed line (the directrix).

## 🎓 考点精讲

**定义与标准方程**
抛物线的统一定义：动点 $P$ 到焦点 $F$ 的距离 $|PF|$ 等于 $P$ 到准线 $l$ 的距离 $d$，即 $|PF| = d$，离心率 $e = 1$。

高考考查四种标准方程形式，设焦点到准线的距离（焦准距）为 $p$（$p>0$）：

1.  **开口向右**：$y^2 = 2px$，焦点 $F(\frac{p}{2},0)$，准线 $x = -\frac{p}{2}$
2.  **开口向左**：$y^2 = -2px$，焦点 $F(-\frac{p}{2},0)$，准线 $x = \frac{p}{2}$
3.  **开口向上**：$x^2 = 2py$，焦点 $F(0,\frac{p}{2})$，准线 $y = -\frac{p}{2}$
4.  **开口向下**：$x^2 = -2py$，焦点 $F(0,-\frac{p}{2})$，准线 $y = \frac{p}{2}$

**关键方法**
1.  **定义优先**：抛物线上点到焦点的距离常转化为该点到准线的距离（“以曲化直”），用于解决焦半径、焦点弦与最值问题。
2.  **方程求解**：根据焦点位置或开口方向设标准方程，由已知距离（如焦点坐标、准线方程，或 $p$ 的几何意义）确定参数 $p$。
3.  **联立与韦达定理**：处理直线与抛物线交点、弦长问题时，联立方程，利用韦达定理建立坐标关系，注意斜率不存在的讨论。
4.  **焦点弦性质**：记住常用结论加速解题，例如过焦点 $F(\frac{p}{2},0)$ 的弦 $AB$（$y^2=2px$），有 $x_1x_2 = \frac{p^2}{4}$，$y_1y_2 = -p^2$，$|AB| = x_1+x_2+p$。

**典型应用**
- **最值与范围**：“抛物线上动点到焦点距离与到某定点距离之和”的最小值，常利用定义将到焦点距离转换为到准线距离，再利用三点共线求解。
- **轨迹方程**：求满足“到定点距离等于到定直线距离”的动点轨迹，直接代入距离公式化简得标准方程。
- **定点定值**：给定抛物线上的动点满足某几何条件（如垂直、共线），证明其关联的直线过定点或某量为定值。

## ⚠️ 高频易错点

- **忘记考虑开口方向与 $p$ 的符号**：看到 $x^2 = ay$ 就认为焦点在 $y$ 轴正半轴，却忽略 $a$ 为负值时对应的焦点坐标、准线方程中符号的变化。
- **混淆焦点坐标与准线方程**：将抛物线 $y^2 = 2px$ 的焦点误写为 $F(p,0)$ 或 $F(0,\frac{p}{2})$；将准线方程写成 $x=\frac{p}{2}$。必须记牢一次项系数与焦点、准线的对应关系。
- **忽视斜率不存在的情况**：设直线方程为 $y=kx+b$ 与抛物线联立时，必须单独讨论斜率不存在的直线（如垂直于 $x$ 轴的直线），这类直线常与抛物线仅有一个交点，但不一定相切。
- **混淆定义中的距离匹配关系**：对于非标准位置的抛物线（如由坐标平移得到的），刻板套用标准方程的焦点坐标公式，而忽略了回归到“点到焦点距离 = 点到准线距离”这一定义本身进行计算。
