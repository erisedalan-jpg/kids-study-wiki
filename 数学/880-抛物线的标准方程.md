# 抛物线的标准方程

> **一句话**：平面上到定点（焦点）和定直线（准线）距离相等的点的轨迹，其代数表达式为 `y² = 2px` 或 `x² = 2py`（`p ≠ 0`）。
> **English**: The set of points equidistant from a fixed point (focus) and a fixed line (directrix), expressed algebraically as `y² = 2px` or `x² = 2py` (`p ≠ 0`).

## 🎓 考点精讲

**定义与原理**  
设定点 `F` 为焦点，定直线 `l` 为准线（`F` 不在 `l` 上），抛物线上任意一点 `M` 满足 `|MF| = d(M, l)`。取 `F` 到 `l` 的垂线段中点作为顶点，该垂线段方向作为对称轴，可得四种标准形式（焦点到准线的距离记为 `|p|`）：

- 开口向右：`y² = 2px`，焦点 `(p/2, 0)`，准线 `x = -p/2`（`p > 0`）
- 开口向左：`y² = -2px`，焦点 `(-p/2, 0)`，准线 `x = p/2`（`p > 0` 时形式如此，书写时 `p` 取正，方程带负号）
- 开口向上：`x² = 2py`，焦点 `(0, p/2)`，准线 `y = -p/2`（`p > 0`）
- 开口向下：`x² = -2py`，焦点 `(0, -p/2)`，准线 `y = p/2`（`p > 0`）

其中 `p` 的几何意义：焦点到准线的距离，也是抛物线张口大小的决定性参数。

**关键方法**  
求抛物线标准方程的核心是确定 **焦参数 `p`** 和 **开口方向**。  
步骤：① 根据已知条件（焦点坐标、准线方程、经过某点、定义中的距离等）建立关于 `p` 的方程；② 由条件判断对称轴是 `x` 轴还是 `y` 轴，进而确定方程属于 `y² = ±2px` 还是 `x² = ±2py`；③ 利用抛物线定义转化距离为坐标关系求解。特别注意：若顶点在原点但开口方向不标准，通常先设标准形式再通过旋转或平移思想解决，但高考阶段以标准位置为主。

**典型应用（高考设问）**  
- **直接求方程**：已知抛物线焦点是 `(2, 0)`，求标准方程 → 由焦点坐标得 `p/2 = 2` 即 `p = 4`，开口向右，方程为 `y² = 8x`。  
- **定义转化距离**：抛物线 `y² = 4x` 上一点 `M` 到焦点距离为 `5`，求 `M` 的横坐标 → 利用 `|MF| = x_M + p/2 = x_M + 1 = 5`，得 `x_M = 4`。  
- **参数范围或最值**：结合焦半径公式 `r = x + p/2`（或 `r = y + p/2`）求弦长或距离最值，常与函数、不等式交汇。

## ⚠️ 高频易错点

- **混淆开口方向与方程符号**：`y² = -2px` 中 `p` 习惯取正值，但方程右侧带负号，学生经常认为 `p` 为负而推导错焦点和准线。  
- **一次项变量定对称轴记反**：`y² = 2px` 的对称轴是 `x` 轴（因为 `y` 为二次项），易误判为 `y` 轴。  
- **焦点与准线坐标符号写错**：如 `x² = 2py` 的准线是 `y = -p/2`，常误写为 `y = p/2`，忘记准线与焦点分居顶点两侧。  
- **定义应用不熟**：已知抛物线上点到焦点距离却直接用两点间距离公式求解，而没有用焦半径公式或定义转化，导致运算繁琐甚至出错。
