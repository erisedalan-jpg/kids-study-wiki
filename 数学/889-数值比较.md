# 数值比较

> **一句话**：数值比较指通过作差、作商、函数单调性或中间量等方法，严格判定两个实数（或代数式）的大小关系。
> **English**: The process of determining the order relationship between real numbers or algebraic expressions via difference, quotient, monotonicity, or intermediate values.

## 🎓 考点精讲

**定义与原理**  
两个实数 $a$ 与 $b$ 的大小关系通过以下等价命题判定：
$$
a > b \iff a - b > 0; \quad a = b \iff a - b = 0; \quad a < b \iff a - b < 0
$$
若 $a, b > 0$，还可使用作商法：$a > b \iff \frac{a}{b} > 1$。更广义的比较依赖**函数单调性**：若函数 $f(x)$ 在区间 $I$ 上单调递增，则 $x_1 > x_2 \in I$ 时，$f(x_1) > f(x_2)$。这些原理是对数、指数、幂函数及超越式比较的理论基石。

**关键方法**  
1. **作差比较**：计算 $A - B$，因式分解或配方判断符号。
2. **作商比较**：适用于各项均为正数的指数型或分式型表达式，与 $1$ 比较大小。
3. **利用单调性**：比较对数值 $\log_a x$ 的大小时，若底数相同，直接由真数大小及底数 $a>1$（单调递增）或 $0<a<1$（单调递减）判定；比较指数值 $a^x$ 时同理。
4. **中间量法**：引入 $0$ 或 $1$ 作为桥梁，例如比较 $\log_3 2$ 与 $\log_2 3$，可借助 $1$ 将二者隔开。
5. **构造函数法**：将两个离散数值视为某函数在不同点的取值，利用导数分析该函数的单调性进行比较（常见于压轴小题）。

**典型应用（高考)**  
- **比较数的大小**：题目给出若干对数、指数、幂值，要求按从大到小排列，如比较 $a=2^{0.3}, b=0.3^2, c=\log_2 0.3$，需综合使用中间量法（$0$ 或 $1$）及函数性质。
- **结合函数性质比较**：已知函数 $f(x)$ 的奇偶性、单调性，比较 $f(a), f(b), f(c)$ 的大小，关键在于将自变量转化到同一单调区间后再利用单调性排序。此类问题常出现在选择题第 7–12 题位置，侧重考查转化与化归思想。

## ⚠️ 高频易错点

- **忽略底数对不等式方向的影响**：应用对数函数或指数函数单调性时，未区分底数 $a>1$ 与 $0<a<1$，导致大小关系反向。
- **误用作商法条件**：对符号不确定的代数式直接作商而不讨论正负，导致结论错误。作商法必须保证分母恒正或先行分类讨论。
- **使用均值不等式放缩时等号条件不成立**：在比较大小的放缩过程中，忽略了“一正二定三相等”的验证，得到错误的大小关系。
- **忽视定义域的隐性约束**：比较含根式、对数、分式的式子大小时，未先求定义域就直接构函数或平方操作，可能引入增根或错误的符号判断。
