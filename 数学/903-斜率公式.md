# 斜率公式

> **一句话**：对于不垂直于 x 轴的直线，其上任意两点 \(P_1(x_1,y_1)\)、\(P_2(x_2,y_2)\)（\(x_1 \neq x_2\)）所确定直线的斜率 \(k = \dfrac{y_2 - y_1}{x_2 - x_1} = \tan\alpha\)，其中 \(\alpha\) 为直线的倾斜角。  
> **English**: The slope of a non‑vertical line passing through two points \((x_1,y_1)\) and \((x_2,y_2)\) is \(k = \dfrac{y_2 - y_1}{x_2 - x_1} = \tan\alpha\), where \(\alpha\) is the angle of inclination.

## 🎓 考点精讲

**定义与公式**
在平面直角坐标系中，一条直线相对于 x 轴的倾斜程度用**斜率**描述。对于已知两点 \(A(x_1,y_1)\)、\(B(x_2,y_2)\)，当 \(x_1 \ne x_2\) 时，斜率公式为：
\[
k = \frac{y_2 - y_1}{x_2 - x_1}
\]
该公式与两点顺序无关，且 \(k = \tan\alpha\)，其中倾斜角 \(\alpha \in [0,\pi)\)，当 \(\alpha = \frac{\pi}{2}\) 时斜率不存在（直线垂直于 x 轴）。斜率也可由直线方程求解：一般式 \(Ax + By + C = 0\)（\(B \ne 0\)）的斜率为 \(-\dfrac{A}{B}\)；斜截式 \(y = kx + b\) 中 \(k\) 即为斜率。

**关键方法**
1. **求斜率**：给出两点坐标直接代入公式；给出倾斜角用 \(k = \tan\alpha\)；给出直线方程化成斜截式或利用一般式系数比。
2. **平行与垂直的斜率条件**：两条不重合的直线 \(l_1 \parallel l_2 \iff k_1 = k_2\)（斜率均存在）；\(l_1 \perp l_2 \iff k_1 \cdot k_2 = -1\)。若其中一条斜率不存在，则另一条斜率为 0 时垂直，斜率不存在时平行。
3. **导数与切线斜率**：函数 \(y = f(x)\) 在 \(x = x_0\) 处的导数 \(f'(x_0)\) 即为曲线在该点处切线的斜率。

**高考典型应用**
- 直接求直线上两点间斜率，判断三点共线（\(k_{AB} = k_{AC}\)）。
- 给出直线方程探讨倾斜角范围，如 \(k = \sqrt{3}\) 得 \(\alpha = 60^\circ\)；\(k < 0\) 得 \(\alpha\) 为钝角。
- 解析几何中，结合斜率关系证明垂直或平行，或利用斜率求直线方程的点斜式 \(y - y_0 = k(x - x_0)\)。
- 导数大题中，已知切点或切线斜率反求参数，利用 \(f'(x_0) = k_{\text{切线}}\) 建立方程。

## ⚠️ 高频易错点

- **忽视斜率不存在的情形**：两点横坐标相等（\(x_1 = x_2\)）时，分母为零，斜率不存在，此时直线垂直于 x 轴，倾斜角为 \(90^\circ\)。许多学生直接代入公式忘记分类讨论。
- **混淆倾斜角与斜率的正负关系**：倾斜角为锐角时 \(k>0\)；钝角时 \(k<0\)。当 \(k\) 由正变负时易判断错倾斜角范围，尤其是在涉及参变量的直线方程中。
- **垂直条件误用**：只记住 \(k_1 \cdot k_2 = -1\) 而忽略斜率不存在的情形，误判一条直线斜率不存在、另一条斜率为 0 的情况（此时也垂直）。
- **导数切线斜率与割线斜率混淆**：用平均变化率 \(\frac{f(x_2)-f(x_1)}{x_2-x_1}\) 代替瞬时变化率（切线的斜率），或求切线方程时误将已知点当作切点。
