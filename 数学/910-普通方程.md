# 普通方程

> **一句话**：普通方程是用变量 `x` 与 `y` 的关系式 `F(x, y) = 0` 直接表示曲线的方程，不含参数。
> **English**: A general equation expresses the relationship between variables x and y directly in the form F(x, y) = 0, without using a parameter.

## 🎓 考点精讲

- **定义/原理**：在平面直角坐标系中，曲线 **C** 上任意一点的坐标 `(x, y)` 满足的二元方程 `F(x, y) = 0` 称为该曲线的普通方程。它与参数方程 `{x = f(t), y = g(t)}` 相对，参数方程通过中间变量 `t`（参数）间接表示点的坐标，而普通方程消去了参数，直接揭示 `x` 与 `y` 的结构关系。高考常以“坐标系与参数方程”选做题的形式考查两类方程的互化，并利用普通方程研究曲线的几何性质。

- **关键方法**：将参数方程化为普通方程的**核心是消参**。常用技巧有：
  1. **代入法**：从一个式子解出参数，代入另一式。
  2. **代数恒等式法**：利用 `x` 与 `y` 的表达式构造恒等式，如平方相加、相减、立方和等。
  3. **三角恒等式法**：若参数为角，运用 `sin²θ + cos²θ = 1`、`1 + tan²θ = sec²θ` 等消去参数。
  消参后**必须**根据原参数方程中 `x`、`y` 的取值范围，补足普通方程的自变量或定义域限制，确保方程的等价性。

- **典型应用**：高考中常见设问为：
  1. 把给定的参数方程化为普通方程，判断曲线类型（直线、圆、椭圆、双曲线、抛物线等）。
  2. 在普通方程的基础上，求焦点坐标、离心率、渐近线方程等几何性质。
  3. 求两曲线的交点，常将曲线化为普通方程后联立求解。
  4. 求动点轨迹的普通方程。
  **例1**：参数方程 `x = 2cosθ, y = 3sinθ`（`θ` 为参数），消参得普通方程 `x²/4 + y²/9 = 1`，为椭圆，且 `x∈[-2,2], y∈[-3,3]`。  
  **例2**：参数方程 `x = t + 1/t, y = t - 1/t`（`t ≠ 0`），消参得 `x² - y² = 4`，为双曲线，但结合 `x = t + 1/t` 的值域知 `|x| ≥ 2`，故普通方程为 `x² - y² = 4 (|x| ≥ 2)`，仅表示双曲线的两支。

## ⚠️ 高频易错点

- 忽视参数对定义域的限制：消参后忘记标注 `x` 或 `y` 的取值范围，导致曲线多出一部分。例如由 `x = √(t), y = t - 1 (t ≥ 0)` 得 `y = x² - 1`，必须附加 `x ≥ 0`，否则包含整条抛物线。
- 三角消参时角度区间处理不当：如 `x = cosθ, y = 1 + cos2θ`（`θ ∈ [0, π]`），消参得 `y = 2x²`，但 `θ ∈ [0, π]` 导致 `x ∈ [-1, 1]`，若直接写 `y = 2x² (x ∈ R)` 则出错。
- 消参方法选择僵化：遇到较复杂形式时，强行直接解参数会导致推导繁琐，应优先观察能否整体构造恒等式，如 `x = (1 - t²)/(1 + t²), y = 2t/(1 + t²)` 可直接利用 `x² + y² = 1` 消参。
- 混淆普通方程与参数方程、极坐标方程的转化依据：将极坐标方程化为普通方程时，应使用变换 `ρ² = x² + y², x = ρcosθ, y = ρsinθ`，不能误用消参思维。
