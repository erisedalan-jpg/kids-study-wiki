# 极坐标与直角坐标互化

> **一句话**：给定极坐标系与直角坐标系原点重合、极轴与 x 轴正半轴重合时，利用 `x=ρcosθ, y=ρsinθ` 与 `ρ²=x²+y², tanθ=y/x (x≠0)` 实现两种坐标的相互转化。
> **English**: The conversion between polar coordinates `(ρ, θ)` and Cartesian coordinates `(x, y)` is achieved via the formulas `x = ρ cosθ`, `y = ρ sinθ` and `ρ² = x² + y²`, `tanθ = y/x` (x ≠ 0), with the same origin and polar axis along the positive x‑axis.

## 🎓 考点精讲

**定义/原理**：  
在平面内取定极点 `O` 和极轴 `Ox`，点 `P` 的极坐标为 `(ρ, θ)`，其中 `ρ` 为极径（`ρ ≥ 0` 通常约定），`θ` 为极角。若以 `O` 为原点、`Ox` 为 x 轴正半轴建立直角坐标系，则点 `P` 的直角坐标 `(x, y)` 与极坐标的互化公式为：

- **极 → 直**：`x = ρ cosθ`，`y = ρ sinθ`；  
- **直 → 极**：`ρ = √(x² + y²)` (常取非负)，`tanθ = y/x`（`x ≠ 0`），`θ` 的终边所在象限由点 `(x, y)` 所在象限确定。当 `x = 0` 时，`θ = π/2` 或 `3π/2` 由 `y` 的正负决定。

**关键方法**：
1. 点坐标互化直接代入公式，注意极角的多值性，通常取 `θ ∈ [0, 2π)` 或 `(−π, π]` 的主值。
2. 方程互化时优先将 `ρcosθ` 替换为 `x`，`ρsinθ` 替换为 `y`，`ρ²` 替换为 `x² + y²`，并消去 `ρ, θ` 得到 `F(x, y) = 0`；反之，由直角坐标方程化为极坐标方程时，常用 `x = ρcosθ, y = ρsinθ` 代入并化简。注意两边同乘 `ρ` 等代数变形，需保证等价性，防止增根或漏解。

**典型应用**：
- **题型**：给出极坐标方程（如直线 `θ = α`、圆 `ρ = 2acosθ`、圆锥曲线等），要求化为直角坐标方程并判断曲线类型；或给定直角坐标方程化为极坐标方程，以便计算弦长、面积等（常见于坐标系与参数方程选做题）。
- **具体场景**：将 `ρ = 2sinθ` 两边同乘 `ρ` 得 `ρ² = 2ρ sinθ`，化为 `x² + y² = 2y`，即 `x² + (y−1)² = 1`，是以 `(0,1)` 为圆心、半径为 1 的圆。若方程是 `ρ = 1/(1−cosθ)`，化为 `ρ − ρcosθ = 1`，即 `ρ = 1 + x`，再平方得 `x² + y² = (1 + x)²`，整理为 `y² = 2x + 1`，表示抛物线。

## ⚠️ 高频易错点

- **忽略极角所在象限**：由 `tanθ = y/x` 反求 `θ` 时，仅用反正切函数可能得到错误象限的角，必须结合点 `(x, y)` 的符号确定 `θ` 的终边位置。
- **盲目两边同乘 `ρ` 产生增根**：方程变形如对 `ρ = 2cosθ` 两边乘 `ρ` 得 `ρ² = 2ρcosθ`，此时若原方程有 `ρ = 0` 的解（极点），该操作仍然等价；但若方程不含极点，乘 `ρ` 可能引入 `ρ = 0` 的增解，需检验极点是否满足原方程。
- **混淆 `ρ` 的符号习惯**：教材通常约定 `ρ ≥ 0`，但有些题目允许 `ρ < 0`（即点 `(ρ, θ)` 与 `(−ρ, θ+π)` 表示同一点），方程互化时必须明确所选约定，否则位置判断与曲线形状可能出错。
- **直角坐标方程化极坐标时未整体代换或忽略范围**：只替换部分项而未将 `x, y` 全部转化为 `ρ, θ` 的表达式，或忘记极坐标方程中 `ρ` 可能受定义域限制（如 `ρ ≥ 0`），导致极坐标方程表示的图形与原直角坐标图形不一致。
