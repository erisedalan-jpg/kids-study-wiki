# 构造函数

> **一句话**：构造函数是根据题目中式子的结构特征，创造性地构造一个辅助函数，利用该函数的单调性、奇偶性、导数等性质来解决原问题的高考常用技巧。
> **English**: Constructing a function means creating an auxiliary function based on the structural features of a given expression, then exploiting its properties (e.g., monotonicity, parity, derivatives) to solve the original problem.

## 🎓 考点精讲

**定义/原理**  
构造函数并非指某一个特定的函数，而是一种解题策略。当题目中的代数式无法直接判断大小、证明不等式或分析根的情况时，通过观察式子间的内在联系，将其视为某个新函数的函数值、差值、积商或导数运算的结果，从而把问题转化为研究该新函数的性质。核心原理是：若构造函数 $F(x)$ 满足 $F(a)=F(b)$ 或 $F'(x)$ 具备判别符号的条件，则可利用函数单调性、极值、最值或零点存在性定理完成解答。

**关键方法**  
1. **结构迁移法**：观察不等号两边或要比较的两个数的结构，抽出公共模型。  
   - 比较 $a$ 与 $b$：若 $a=\dfrac{\ln m}{m},\ b=\dfrac{\ln n}{n}$，可构造函数 $f(x)=\dfrac{\ln x}{x}$，通过导数得 $f(x)$ 的单调区间，再比较 $f(m)$ 与 $f(n)$。  
2. **差函数法**：证明 $f(x) \ge g(x)$ 常令 $h(x)=f(x)-g(x)$，转化为证 $h(x)_{\min}\ge 0$。  
3. **积商构造法（导数逆用）**：面对含导数的抽象不等式，如 $xf'(x)+f(x)>0$，识别其为 $[xf(x)]'$，则构造 $F(x)=x f(x)$；若 $f'(x)+f(x)>0$，可联想到 $[e^x f(x)]'$，构造 $F(x)=e^x f(x)$。  

**典型应用**  
- **比较大小**：(2023·全国甲卷理) 设 $a=\dfrac{\ln 1.1}{1.1}$，$b=\dfrac{1}{1.1}$，$c=\ln 1.1$，可构造 $f(x)=\ln x - \dfrac{1}{x}$ 或利用 $\ln x \le x-1$ 放缩，实质是构造辅助函数比较。  
- **抽象函数不等式**：已知定义在 $\mathbb{R}$ 上的函数 $f(x)$ 满足 $f'(x)+f(x)>0$，且 $f(0)=1$，求不等式 $e^x f(x)>1$ 的解集。令 $g(x)=e^x f(x)$，则 $g'(x)=e^x(f'(x)+f(x))>0$，$g(x)$ 递增，$g(x)>g(0)$ 得解集。

## ⚠️ 高频易错点

- **忘记定义域**：构造 $f(x)=\dfrac{\ln x}{x}$ 后，必须明确 $x>0$，忽视定义域会导致单调区间错误。  
- **导数构造模型混淆**：$f'(x)+f(x)>0$ 对应乘 $e^x$，而 $f'(x)-f(x)>0$ 对应乘 $e^{-x}$ 或除 $e^x$，混淆指数符号将导致构造失败。  
- **构造后不验证等号条件**：证明 $f(x)\ge g(x)$ 时构造 $h(x)=f(x)-g(x)$，若 $h(x)_{\min}=0$，必须指明取等时的 $x$ 值，否则证明不严谨。  
- **盲目构造函数比较**：比较 $1.01^{100}$ 与 $2$ 时，若机械地构造 $f(x)=x^{100}$ 不易求解，应联想到 $(1+\frac{1}{100})^{100}$ 与 $e$ 的近似，或构造函数 $g(x)=\ln(1+x)-x$ 等，选错模型耗时且易错。
