# 根式不等式

> **一句话**：根式不等式是指根号内含有未知数的不等式，求解时需在定义域约束下通过平方消去根号，并依据两边符号分类讨论。  
> **English**: A radical inequality is an inequality that contains a variable inside a radical; solving it requires squaring to eliminate the radical while respecting the domain and sign conditions.

## 🎓 考点精讲

**定义/原理**  
一般形式如 $\sqrt{f(x)} > g(x)$ 或 $\sqrt{f(x)} < g(x)$，其中 $f(x) \ge 0$ 是定义域。  
消去根号的核心方法是平方，但需注意平方后不等号方向是否保持，取决于两边符号。以 $\sqrt{f(x)} > g(x)$ 为例，等价于两个不等式组的并集：  
1. $\begin{cases} g(x) \ge 0 \\ f(x) > [g(x)]^2 \end{cases}$  
2. $\begin{cases} g(x) < 0 \\ f(x) \ge 0 \end{cases}$  

对于 $\sqrt{f(x)} < g(x)$，等价于 $\begin{cases} g(x) > 0 \\ f(x) \ge 0 \\ f(x) < [g(x)]^2 \end{cases}$。  
若含有两个根号，如 $\sqrt{f(x)} + \sqrt{h(x)} > k$，通常通过移项、平方向单根号转化。

**关键方法**  
1. 写出使所有根式有意义的定义域（被开方式≥0）。  
2. 根据不等号方向及右端符号，合理分类讨论，将根式不等式转化为有/无理不等式组。  
3. 平方去根号，解所得不等式（常转化为二次不等式），并与其前提条件取交集。  
4. 若有多个分支，最后取并集，并用区间表示解集。

**典型应用**  
高考中常以选择题、填空题中的“解不等式”或解答题中求定义域、参数范围的形式出现。例如：  
- 解 $\sqrt{2x+3} > x$，需分 $x \ge 0$ 和 $x<0$ 讨论，检验端点。  
- 已知函数 $f(x)=\sqrt{x-a} - x$ 的定义域为某范围，求参数 $a$，往往涉及根式不等式恒成立问题。

## ⚠️ 高频易错点

- 忘记考虑被开方式 $f(x) \ge 0$ 的定义域，导致解集扩大或漏掉限制条件。  
- 平方前未判断 $g(x)$ 的符号，直接将 $\sqrt{f(x)} > g(x)$ 平方成 $f(x) > [g(x)]^2$，当 $g(x)<0$ 时该不等式恒成立（只要根式有意义），误为仍需 $f(x) > [g(x)]^2$ 而丢失解。  
- 多个不等式取交集或并集时，区间端点漏掉或端点处未验证定义域，出现开闭区间错误。  
- 忽视平方可能引入增根，未与前提条件取交集直接给最终解集。
