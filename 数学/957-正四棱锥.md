# 正四棱锥

> **一句话**：底面为正方形，且顶点在底面上的投影恰好为底面中心（正方形对角线交点）的四棱锥。
> **English**: A regular quadrangular pyramid is a pyramid whose base is a square and whose vertex projects orthogonally onto the center of the base.

## 🎓 考点精讲

**定义与性质**  
正四棱锥满足：底面 $ABCD$ 是边长为 $a$ 的正方形；顶点 $P$ 在底面的射影 $O$ 为正方形的中心。由此可直接推出所有侧棱相等，所有侧面都是全等的等腰三角形；侧棱与底面所成角相等，侧面与底面所成二面角相等。

设高 $PO = h$，斜高（侧面等腰三角形底边上的高）为 $h'$，则有关系：
$$h' = \sqrt{h^2 + \left(\frac{a}{2}\right)^2}, \quad \text{侧棱长 } l = \sqrt{h^2 + \frac{a^2}{2}}.$$

**常用公式**  
- 体积：$V = \frac{1}{3} S_{\text{底}} h = \frac{1}{3}a^2 h$  
- 侧面积：$S_{\text{侧}} = 4 \times \frac{1}{2}a h' = 2a h'$  
- 全面积：$S_{\text{全}} = a^2 + 2a h'$

**关键方法**  
正四棱锥的几乎所有计算最终都归结为高 $h$、底面边长 $a$、斜高 $h'$、侧棱 $l$ 四个量之间的直角三角形关系。对线面角、二面角问题，标准步骤是：在图形中标出 $P$、$O$、斜高在底边的垂足 $M$（$M$ 为 $BC$ 中点），在 $\triangle POM$ 中处理高与斜高、二面角的平面角；在 $\triangle POB$ 中处理侧棱与底面对角线的关系。

**高考典型考查**  
1. **求体积、表面积**：已知高和底面边长，直接代公式；或给出侧棱/斜高及底面边长，先用勾股定理反求高。  
2. **外接球**：正四棱锥的外接球球心在高 $PO$ 所在的直线上，通常设球心距顶点 $P$ 为 $r$，利用 $OB=r$ 或 $OP=r$ 建立方程，常用 $R^2 = \left(h - R\right)^2 + \frac{a^2}{2}$ 求解（当球心在锥体内部）或 $R^2 = \left(R - h\right)^2 + \frac{a^2}{2}$（当球心在外部）。  
3. **内切球**：利用体积法 $V = \frac{1}{3} S_{\text{全}} r_{\text{内}}$ 求内切球半径，前提是内切球存在（正四棱锥高度大于某个界限）。  
4. **角度与截面**：如求侧棱与底面所成角 $\angle PBO$、侧面与底面所成二面角 $\angle PMO$，以及过顶点或平行于底的截面问题。

## ⚠️ 高频易错点

- **混淆侧棱与斜高**：侧棱 $l$ 是指顶点到底面顶点的线段，斜高 $h'$ 是侧面等腰三角形底边上的高，二者所在的直角三角形不同。  
- **外接球心盲目放在高上**：正四棱锥的外接球心一定在高所在直线上，但具体位置在高线上距离顶点为 $R$ 处，需区分球心在几何体内部还是外部，切忌直接使用 $R^2 = \left(\frac{h}{2}\right)^2 + \frac{a^2}{2}$。  
- **体积计算漏乘 $\frac{1}{3}$**：锥体体积公式 $V = \frac{1}{3} Sh$，与柱体 $Sh$ 混淆是常见错误。  
- **内切球不讨论存在性**：并不是所有正四棱锥都存在内切球，当 $h < \frac{a}{2}$ 时侧面过“尖”无法容球，却直接套用体积法会导致错误半径。
