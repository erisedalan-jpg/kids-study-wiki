# 比较大小

> **一句话**：比较大小是指运用不等式性质、函数单调性或数值计算等方法，判断两个实数或代数式的大小关系。
> **English**: Comparison of magnitudes is the process of determining the order relation between two mathematical objects using inequality properties, function monotonicity, or numerical computation.

## 🎓 考点精讲

比较大小的核心依据是实数公理：对于任意两个实数 $a$ 和 $b$，$a>b \iff a-b>0$。由此衍生出**作差法**和**作商法**。若 $a-b>0$，则 $a>b$；若 $a-b<0$，则 $a<b$。当两数同号时，也可用作商法：若 $\frac{a}{b}>1$ 且 $b>0$，则 $a>b$；若商大于1但分母为负，不等号需反转。

高考中，比较大小进一步结合函数性质：
- 利用**函数的单调性**：若 $f(x)$ 在区间 $I$ 上单调递增，则 $\forall x_1,x_2\in I,\ x_1<x_2 \iff f(x_1)<f(x_2)$。此性质常用于比较幂、指、对函数值。
- **中间量法**：当直接比较困难时，常引入 $0$ 或 $1$ 作为桥梁。例如比较 $a=\log_{0.5}3$ 和 $b=0.2^{0.3}$，可注意到 $a<0$，$0<b<1$，从而 $a<b$。
- **构造函数法**：比较结构相似的两式，可构造辅助函数并借助导数判断单调性。如比较 $e^\pi$ 与 $\pi^e$，可两边取对数后构造函数 $f(x)=\frac{\ln x}{x}$，利用其在 $(e,+\infty)$ 上单调递减得出结论。

典型高考设问：“设 $a=\log_2 3$，$b=\log_3 4$，$c=4^{-0.2}$，比较 $a,b,c$ 的大小。” 解题需综合运用对数单调性、换底公式，并与中间量 $1$ 或 $\frac{3}{2}$ 比对。又如比较 $\sin 1$ 与 $\cos 1$，需借助单位圆或正弦、余弦在 $(0,\pi/2)$ 上的单调区间。

## ⚠️ 高频易错点

- 作差后因式符号未讨论直接定号；作商时忽略分母为零或分母为负时不等号方向反转。
- 滥用指数、对数函数的单调性，混淆底数 $0<a<1$ 与 $a>1$ 时增减性的区别，导致大小关系完全反向。
- 使用中间量 $0$ 或 $1$ 时未严谨确认各数与中间量的位置，如误认为 $\log_a x$（$0<a<1$）恒大于 $0$。
- 构造函数求导后，不分析定义域或忽视导数的零点，凭感觉判断原函数单调性而致错。
