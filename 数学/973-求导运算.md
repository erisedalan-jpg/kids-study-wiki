# 求导运算

> **一句话**：求导运算是基于极限定义，对可导函数求出其导函数的运算过程，导函数刻画了原函数在各点处的瞬时变化率。
> **English**: Derivative computation is the process of finding the derivative of a differentiable function based on the limit definition, where the derivative describes the instantaneous rate of change of the original function at each point.

## 🎓 考点精讲

**定义**：设函数 $y = f(x)$ 在点 $x_0$ 的某一邻域内有定义，若极限
$$\lim_{\Delta x \to 0} \frac{f(x_0 + \Delta x) - f(x_0)}{\Delta x}$$
存在，则称 $f(x)$ 在 $x_0$ 处可导，该极限值称为 $f(x)$ 在 $x_0$ 处的导数，记作 $f'(x_0)$ 或 $\left.\frac{dy}{dx}\right|_{x=x_0}$。若函数在区间内每一点都可导，便得到导函数 $f'(x)$。

**基本公式与运算法则**（高考核心）：

| 类型 | 公式/法则 |
|------|----------|
| 常数函数 | $(C)' = 0$ |
| 幂函数 | $(x^n)' = nx^{n-1}$（$n \in \mathbf{R}$） |
| 指数函数 | $(a^x)' = a^x \ln a$；$(e^x)' = e^x$ |
| 对数函数 | $(\log_a x)' = \frac{1}{x \ln a}$；$(\ln x)' = \frac{1}{x}$ |
| 正弦/余弦 | $(\sin x)' = \cos x$；$(\cos x)' = -\sin x$ |
| 四则运算 | $(u \pm v)' = u' \pm v'$；$(uv)' = u'v + uv'$；$\left(\frac{u}{v}\right)' = \frac{u'v - uv'}{v^2}$（$v \neq 0$） |
| 复合函数（链式法则） | $y = f(g(x))$，则 $y' = f'(g(x)) \cdot g'(x)$ |

**解题关键步骤**：
1. **识别函数结构**：判断是基本初等函数、四则运算组合、还是复合函数。
2. **逐层求导**：复合函数由外向内逐层求导并相乘；四则运算严格套用法则，特别注意乘法的乘积法则和除法的商法则，避免漏项。
3. **化简**：提取公因式、合并同类项，将导数表达式化为最简形式以便后续分析。

**高考典型考查**：
- **显函数求导**：直接给出多项式、分式、根式、三角、指数、对数及其组合，求导函数或某点导数值，常见于解答题第（1）问。
- **复合函数求导**：如 $f(x) = \ln(1 + 3x^2)$ 或 $g(x) = e^{\cos x}$，考查链式法则的准确应用，是导数应用题的必经步骤。

## ⚠️ 高频易错点

- **混淆导数符号与函数写法**：误以为 $f'(2x)$ 是 $f(2x)$ 对 $x$ 的直接导数——应先用链式法则：$[f(2x)]' = f'(2x) \cdot 2$。
- **漏掉复合函数内层导数**：如 $(\sin 2x)'$ 错写成 $\cos 2x$，遗漏内层 $(2x)' = 2$ 的乘积，正确为 $2\cos 2x$。
- **商法则分子顺序写反**：$\left(\frac{u}{v}\right)'$ 正确分子是 $u'v - uv'$，易错记为 $uv' - u'v$。
- **在不可导点错误套用公式**：如 $y = |x|$ 在 $x=0$ 处不可导，直接用公式 $(|x|)' = \frac{x}{|x|}$ 时忽略了定义域限制，需分段讨论。
