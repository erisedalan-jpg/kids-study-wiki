# 点到直线的距离

> **一句话**：点到直线的距离是指从直线外一点向该直线作垂线，该点与垂足之间的线段长度（最小距离）。
> **English**: The distance from a point to a line is the length of the perpendicular segment from the point to the line, representing the shortest distance.

## 🎓 考点精讲

**定义/原理**：  
在平面直角坐标系中，给定点 $P(x_0, y_0)$ 和直线 $Ax + By + C = 0$（$A, B$ 不同时为零），点 $P$ 到直线的距离 $d$ 由公式给出：
$$ d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}} $$
该公式本质是利用直线的法向量求投影长度，几何意义为点 $P$ 到直线上所有点距离的最小值。

**关键方法**：
1. **公式直接代入**：将直线方程化为一般式 $Ax + By + C = 0$，分子代入点坐标取绝对值，分母为系数平方和的算术平方根。
2. **向量法**（备用思路）：在直线上任取一点 $Q$，利用直线的方向向量与法向量关系求点 $P$ 到直线的距离 $d = \frac{|\overrightarrow{PQ} \cdot \vec{n}|}{|\vec{n}|}$，其中 $\vec{n} = (A, B)$ 为法向量。
3. **适用条件**：直线方程必须为一般式；若给定斜截式、点斜式或截距式，需先化为一般式再计算。

**典型应用**：
在高考中，点到直线距离公式常见于：
- **直接求距离**（如点到直线的距离、两平行线间距离转化为点到直线距离）。
- **直线与圆的位置关系**：圆心到直线的距离与半径比较，判断相交、相切、相离；已知相切条件求参数。
- **最值问题**：如动点到定直线距离的最小值，或结合图形特征求面积、范围等。
- **圆锥曲线中**：作为弦长、焦点到准线距离等推导的基础工具。

## ⚠️ 高频易错点

- **未将直线方程化为一般式**：如直线为 $y = kx + b$ 时，需先化为 $kx - y + b = 0$，再确定 $A, B, C$ 代入公式，否则符号易出错。
- **漏掉绝对值**：分子是 $|Ax_0 + By_0 + C|$，丢掉绝对值会得出负距离或错误结果，尤其当点在直线另一侧时。
- **分母混淆系数**：分母为 $\sqrt{A^2 + B^2}$，不是 $\sqrt{A^2 + B^2 + C^2}$，也不是简单的 $A^2 + B^2$。
- **平行线间距离公式混淆**：两平行线 $Ax + By + C_1 = 0$ 与 $Ax + By + C_2 = 0$ 的距离为 $d = \frac{|C_1 - C_2|}{\sqrt{A^2 + B^2}}$，注意分子是常数项差的绝对值，且必须保证 $A, B$ 相同。
