# 由图象求解析式

> **一句话**：根据给定函数的图象，结合函数类型和关键点坐标，逆向确定函数解析式中的参数，从而写出完整的函数表达式。
> **English**: Determining the expression of a function by identifying key features and parameters from its given graph.

## 🎓 考点精讲

**定义/原理**

由图象求解析式，是指已知某个函数类型的图象（通常为正弦型函数 $y = A\sin(\omega x + \varphi) + k$ 或余弦型函数 $y = A\cos(\omega x + \varphi) + k$），通过观察图象的几何特征，反推出表达式中各参数的值。参数与图象的对应关系如下：

- $A$（振幅）：$A = \frac{y_{\text{max}} - y_{\text{min}}}{2}$，反映曲线纵向波动的幅度。
- $k$（纵向位移）：$k = \frac{y_{\text{max}} + y_{\text{min}}}{2}$，是平衡位置的纵坐标。
- $\omega$（角频率）：由周期 $T$ 决定，$\omega = \frac{2\pi}{T}$。$T$ 可通过相邻最高点、相邻最低点或相邻零点间的水平距离求得。
- $\varphi$（初相）：通过代入一个已知点的坐标解出，通常选取最高点、最低点或与其他参数配合求解。

**关键方法**

求解步骤：
1. **观 A 与 k**：从图象的最高点与最低点直接读数，利用公式求出 $A$ 和 $k$。
2. **求 $\omega$**：在图象上找到两个相邻的、处于相同振动状态的点（如相邻最高点），其横坐标之差的绝对值即为一个周期 $T$，再由 $\omega = \frac{2\pi}{T}$ 求得。
3. **定 $\varphi$**：将 $A$、$k$、$\omega$ 代入表达式，选择一个明确且易算的已知点坐标（优先选最高点、最低点或与 $x$ 轴的交点），解三角方程求 $\varphi$。需注意三角函数的多解性，结合点的单调趋势（上升或下降）锁定唯一值。
4. **写解析式**：将求得的参数代入标准形式，必要时注明定义域。

**典型应用**

高考中常见设问：
- **给正弦型函数图象求解析式**：直接要求写出 $y = A\sin(\omega x + \varphi) + k$ 的表达式，并为后续求单调区间、对称中心等做铺垫。
- **与函数变换结合**：先求出 $f(x)$，再求 $f(x+m)$ 平移后的表达式或相关性质。

*示例*：已知函数 $f(x) = A\sin(\omega x + \varphi) + k$（$A > 0, \omega > 0, |\varphi| < \frac{\pi}{2}$）的部分图象，最高点 $(\frac{\pi}{3}, 3)$，相邻最低点 $(\frac{5\pi}{6}, -1)$。由 $A = \frac{3 - (-1)}{2} = 2$，$k = \frac{3 + (-1)}{2} = 1$；周期 $T = 2 \times (\frac{5\pi}{6} - \frac{\pi}{3}) = \pi$，故 $\omega = 2$；代入最高点得 $2\sin(2 \cdot \frac{\pi}{3} + \varphi) + 1 = 3 \Rightarrow \sin(\frac{2\pi}{3} + \varphi) = 1$，结合 $|\varphi| < \frac{\pi}{2}$ 得 $\varphi = -\frac{\pi}{6}$。解析式为 $f(x) = 2\sin(2x - \frac{\pi}{6}) + 1$。

## ⚠️ 高频易错点

- **混淆振幅与纵向位移的计算**：$A$ 是最高点与最低点之差的一半，$k$ 是和的一半，学生常将两者颠倒或直接取最高点为 $A$ 而忘记考虑 $k \neq 0$ 的情况。
- **求 $\omega$ 时周期判断错误**：只看两个零点间距就当作半个周期，忽视零点有上升零点和下降零点之分，导致 $T$ 算错，应优先使用两个相邻最高点或最低点的水平距离。
- **$\varphi$ 求解不严谨**：直接代入一个点解出 $\varphi$ 后未检验该点所处的单调区间，导致求得的值不满足图象趋势（例如 $\sin$ 在上升区间解出的方程有两个可能值，需根据趋势舍去一个）。
- **忽视参数范围限制**：题目通常给定了 $A > 0$、$\omega > 0$、$|\varphi| \leq \pi$ 等约束，学生求值时忘记结合范围取舍，导致答案增根。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 振幅 | amplitude | 函数值偏离平衡位置的最大距离，对应参数 $A$。 |
| 初相 | initial phase | 自变量为 0 时的相位，即参数 $\varphi$。 |
| 周期 | period | 函数图形重复出现的最小水平间隔，记为 $T$。 |
| 正弦波 | sine wave | 指正弦型函数 $y = A\sin(\omega x + \varphi)$ 的图像形态。 |
| 最高点 | maximum point | 图象上纵坐标最大的点，是确定 $A$、$k$ 和代点求 $\varphi$ 的关键。 |
