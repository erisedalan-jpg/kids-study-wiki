# 目标函数

> **一句话**：目标函数是在线性规划或优化问题中，表示要达到最大化或最小化目标的线性函数，通常写作 $z = ax + by$。
> **English**: An objective function is a linear function in linear programming or optimization problems that represents the goal to be maximized or minimized, typically written as $z = ax + by$.

## 🎓 考点精讲

**定义/原理**：在线性规划问题中，目标函数是关于决策变量的一次函数，记为 $z = ax + by$（或多元形式 $z = c_1x_1 + c_2x_2 + \cdots$）。在给定的约束条件（一组线性不等式）下，求 $z$ 的最大值或最小值。从几何角度看，令 $z = k$（$k$ 为参数），$ax + by = k$ 表示一组平行直线，$k$ 的几何意义是该直线在纵轴上的截距（乘以系数）。平移这组平行线，首次或末次接触可行域（约束条件围成的区域）时，$k$ 取得最值。

**关键方法**：
1. **列式**：设出决策变量 $x, y$，依据题意写出目标函数 $z = ax + by$ 和约束条件组。
2. **画域**：在直角坐标系中画出每个不等式表示的平面区域，取交集得可行域。
3. **求解**：将目标函数化为斜截式 $y = -\frac{a}{b}x + \frac{z}{b}$（若 $b \neq 0$），观察纵截距 $\frac{z}{b}$ 在可行域内的最优位置。最优解必在可行域的顶点处取得（线性规划基本定理）。若可行域为开区域或无界，需另行判断解的存在性。
4. **下结论**：代入顶点坐标计算 $z$ 值，比较得最大（小）值，或指出最优解不存在。

**典型应用**：高考中常以实际应用题形式考查，如“某工厂生产甲、乙两种产品，每件利润分别为 $p_1, p_2$，受原材料、工时等限制，如何安排生产使总利润最大？”典型设问为列出约束条件、画出可行域、求目标函数最大值及最优解。

## ⚠️ 高频易错点

- **截距与最值的关系混淆**：将目标函数化为 $y = kx + b$ 形式后，$b$ 与 $z$ 的关系易搞反，特别是 $b = z/b$ 中的分母系数未合理转化时，会得出相反结论。
- **可行域画错**：不等式符号方向取反，或实线/虚线（含不含等号）混淆，导致可行域边界遗漏或错含，进而顶点坐标计算错误。
- **忽视整数解要求**：应用题中决策变量常要求整数（如产品件数），学生直接取顶点坐标而忽略该点未必是整数解，未进行平移调整或网格点检验。
