## 课本定义

由含同一未知数的几个一元一次不等式组成的不等式组，叫做**一元一次不等式组**。不等式组的解集是**各个不等式解集的公共部分**（交集），在数轴上取各解集的重叠部分。

---

## 📚 6-12 岁回溯版

### "两个条件都要同时满足"

**不等式组**就是几个不等式**同时**成立。

**生活例子：** 小华参加跳绳比赛，规定年龄在8到12岁之间（含8岁和12岁）。

设年龄为 x，则：
$$\begin{cases} x \geq 8 \\ x \leq 12 \end{cases}$$

两个条件同时要满足：x 既要 ≥ 8，又要 ≤ 12。

**解法——用数轴找"重叠部分"：**

x ≥ 8 的数轴：
```
————————•—————————→
       8
```

x ≤ 12 的数轴：
```
←—————————•————
           12
```

两者**重叠**的部分：
```
————————•—————•————
       8      12
```

解集就是 8 ≤ x ≤ 12（即 x 在8到12之间，含两端）。

**口诀（找公共部分）：**
- "同大取大"：两个都是"x > 某数"，取较大的那个数
- "同小取小"：两个都是"x < 某数"，取较小的那个数
- "大小取中"：一个 x > a，一个 x < b（且 a < b），取中间的 a < x < b
- "小大无解"：一个 x > b，一个 x < a（且 a < b），无公共部分，无解

## 🎓 12+ 进阶版

### 严格定义

**一元一次不等式组**：由含同一未知数 x 的多个一元一次不等式联立组成的不等式组，记作：

$$\begin{cases} \text{不等式1} \\ \text{不等式2} \\ \vdots \end{cases}$$

**不等式组的解集** = 各不等式解集的**交集**（同时满足所有不等式的 x 的集合）。

### 解不等式组的步骤

1. 分别求各不等式的解集
2. 在数轴上画出各解集
3. 找重叠（公共）部分
4. 用不等号表示公共部分（即不等式组的解集）

### 四种情形总结

设 a < b，则：

| 不等式组 | 数轴重叠情况 | 解集 |
|---------|------------|------|
| x > a 且 x > b | 取右边（大的） | x > b |
| x < a 且 x < b | 取左边（小的） | x < a |
| x > a 且 x < b | 取中间 | a < x < b |
| x > b 且 x < a | 无重叠 | 无解（空集）|

### 典型例题

**例题1：** 解不等式组

$$\begin{cases} 2x - 1 > 3 \quad (1)\\ x + 2 < 6 \quad (2) \end{cases}$$

解：由 (1)：2x > 4，x > 2
由 (2)：x < 4

数轴上：x > 2 且 x < 4，取中间，解集为 **2 < x < 4**。

**例题2：** 解不等式组

$$\begin{cases} 3x + 1 \geq 7 \quad (1)\\ 2x - 3 \geq 5 \quad (2) \end{cases}$$

解：由 (1)：3x ≥ 6，x ≥ 2
由 (2)：2x ≥ 8，x ≥ 4

两个都是"x ≥"，取较大数，解集为 **x ≥ 4**。

**例题3：** 解不等式组，说明是否有解

$$\begin{cases} x < 1 \\ x > 3 \end{cases}$$

解：x < 1 且 x > 3，这两个条件互相矛盾（不可能同时成立），**无解（解集为空集）**。

**例题4（应用）：** 某活动要求人数不少于15人且不超过30人，已报名 x 人。

$$\begin{cases} x \geq 15 \\ x \leq 30 \end{cases}$$

解集：15 ≤ x ≤ 30（x 取整数，即 15, 16, ..., 30）。

### 易错点

1. **公共部分取"交集"**，不是"并集"（两者都满足，不是满足任意一个）
2. **"同大取大，同小取小"要分清哪个更严格**
3. **无解时写"空集"或"无解"**，不要写成错误的不等式
