## 🎓 三角函数导数

### 基本公式

| 原函数 | 导函数 | 备注 |
|---|---|---|
| $\sin x$ | $\cos x$ | |
| $\cos x$ | $-\sin x$ | 注意负号 |
| $\tan x$ | $\dfrac{1}{\cos^2 x} = \sec^2 x$ | $x\neq\dfrac{\pi}{2}+k\pi$ |
| $\cot x$ | $-\dfrac{1}{\sin^2 x} = -\csc^2 x$ | $x\neq k\pi$（高中选考了解） |

### 推导：$(\sin x)' = \cos x$

利用和差化积公式和重要极限 $\lim_{\theta\to 0}\dfrac{\sin\theta}{\theta}=1$：

$$(\sin x)' = \lim_{h\to 0}\frac{\sin(x+h)-\sin x}{h}$$

$$= \lim_{h\to 0}\frac{2\cos\left(x+\frac{h}{2}\right)\sin\frac{h}{2}}{h} = \lim_{h\to 0}\cos\left(x+\frac{h}{2}\right)\cdot\underbrace{\frac{\sin\frac{h}{2}}{\frac{h}{2}}}_{\to 1} = \cos x$$

类似地，$(\cos x)' = -\sin x$（推导时出现负号是因为 $\cos$ 展开后的符号）。

### 典型例题

**例 1** $f(x) = 3\sin x - 2\cos x$

$$f'(x) = 3\cos x + 2\sin x$$

**例 2** $f(x) = x\sin x$（乘法法则）

$$f'(x) = \sin x + x\cos x$$

**例 3** $f(x) = \sin(3x+\pi/4)$（链式法则）

$$f'(x) = \cos(3x+\pi/4)\cdot 3 = 3\cos(3x+\pi/4)$$

**例 4** $f(x) = \sin^2 x$（链式法则）

$$f'(x) = 2\sin x\cdot\cos x = \sin(2x)$$

**例 5** $f(x) = \tan x$（用 $\dfrac{\sin x}{\cos x}$ 和除法法则推导）

$$(\tan x)' = \left(\frac{\sin x}{\cos x}\right)' = \frac{\cos x\cdot\cos x - \sin x\cdot(-\sin x)}{\cos^2 x} = \frac{\cos^2 x+\sin^2 x}{\cos^2 x} = \frac{1}{\cos^2 x}$$

### 三角导数的规律

$$\sin\xrightarrow{\text{求导}}\cos\xrightarrow{\text{求导}}-\sin\xrightarrow{\text{求导}}-\cos\xrightarrow{\text{求导}}\sin\cdots$$

对 $\sin x$ 求导 $4$ 次回到原函数，周期 $4$。

### 易错点

- $(\cos x)' = -\sin x$，负号容易忘
- 链式法则：$[\sin(2x)]' = \cos(2x)\cdot 2 = 2\cos(2x)$，不是 $\cos(2x)$
- $\sin^2 x$ 是 $(\sin x)^2$，用链式法则，不是 $2\sin x$
- 角度单位必须是**弧度**；若用度数，公式不成立
