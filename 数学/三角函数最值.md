# 三角函数最值

> **一句话**：在一定定义域内，求三角函数（如 y=Asin(ωx+φ)+k）所能取得的最大值与最小值的数学问题，核心是三角函数的有界性、单调性和代数变形。
> **English**: The problem of determining the maximum and minimum values attainable by trigonometric functions over a given domain, relying on boundedness, monotonicity, and algebraic transformations.

## 🎓 考点精讲

**定义与原理**  
三角函数最值问题以正弦、余弦的有界性（|sin x|⩽1，|cos x|⩽1）为理论基石。任何正弦型函数 y=A sin(ωx+φ)+B 的最值直接由振幅 |A| 和上下平移量 B 决定：当 sin(ωx+φ)=1 时，y_max=|A|+B；当 sin(ωx+φ)=−1 时，y_min=−|A|+B（余弦类似）。更复杂的表达式需先通过三角恒等变形化为上述标准型或二次型等其他可解形式。

**关键方法**  
1. **化为标准型**：对 a sinθ+b cosθ 型，用辅助角公式  
   a sinθ+b cosθ = √(a²+b²) sin(θ+φ)（其中 tanφ = b/a），再利用有界性求最值。  
2. **二次函数换元**：形如 y = a sin²x + b sin x + c，令 t = sin x ∈ [−1,1]（或根据给定区间截取 t 的范围），转化为闭区间上二次函数求最值。  
3. **其他典型结构**：sin x±cos x 与 sin x cos x 的关系（换元 t = sin x±cos x）；分式型利用分离常数或几何意义；含绝对值或分段函数的题型则结合图像与单调区间求解。  
4. **导数法**：若变形困难，可直接求导、判断极值点，并结合端点值比较求得最值。

**典型应用**  
高考常见题型包括：①求 f(x)=sin 2x+cos x 在规定区间上的值域；②已知 f(x)=2sin(2x+π/3) 在 [0,π/2] 上的最大、最小值；③与三角形结合，如锐角三角形中求 sin A+sin B 的最大值；④含参函数 y=a sin x+b 在给定最值下反求参数。设问多直接要求“最大值”“最小值”或“值域”，或隐藏于恒成立、存在性问题中。

## ⚠️ 高频易错点

- **忽略区间限制**：直接认为 y=sin(x) 的值域就是 [−1,1]，未考虑 x 的给定范围，导致 ωx+φ 的取值区间未完整覆盖最高点或最低点。
- **换元后新变量范围错误**：设 t=sin x，若 x 有区间约束，t 的范围可能不再是完整的 [−1,1]，需重新由 x 区间算出 t 的准确闭区间。
- **辅助角 φ 的象限误差**：由 tanφ=b/a 求 φ 时，未结合 a、b 的符号确定 φ 所在象限，导致最终标准型表达式错误，进而影响最值对应的自变量的取值。
- **忽略三角函数的有界性条件**：如用基本不等式处理 sin x+1/sin x 等形式时，没有注意 sin x 的符号和取值范围，误用“一正二定三相等”导致错误。
