# 三角函数模型

> **一句话**：三角函数模型是借助正弦、余弦等三角函数，对具有周期性变化规律的实际情境进行定量描述的数学模型。  
> **English**: A trigonometric function model is a mathematical model that uses sine, cosine, or other trigonometric functions to quantitatively describe phenomena with periodic behavior.

## 🎓 考点精讲

**定义与形式**  
三角函数模型的标准形式为  
\[
y = A\sin(\omega x + \varphi) + b \quad \text{或} \quad y = A\cos(\omega x + \varphi) + b
\]
其中 $A$ 为振幅（反映波动高度），$\omega$ 为角频率，周期 $T = \frac{2\pi}{|\omega|}$，$\varphi$ 为初相（决定 $x=0$ 时的相位），$b$ 为纵向位移（平衡位置）。物理中常用的简谐振动方程 $x = A\cos(\omega t + \varphi)$ 即为此类模型。

**关键方法**  
1. **由数据或图像求解析式**（高考核心）  
   ① 求 $A$ 与 $b$：$A = \frac{f_{\max} - f_{\min}}{2}$，$b = \frac{f_{\max} + f_{\min}}{2}$。  
   ② 求 $\omega$：先由图像或文字确定周期 $T$，则 $\omega = \frac{2\pi}{T}$。  
   ③ 求 $\varphi$：代入已知点坐标（通常选最值点或平衡位置点），利用五点法对应关系解初相，注意 $\varphi$ 的常用范围（如 $|\varphi| \le \pi$）。  
2. **解决实际应用问题**  
   建立坐标系，将几何量（高度、距离、电流等）表为时间或角度的三角函数，再通过解方程或不等式求特定时刻或区间。

**典型应用（高考设问）**  
- 给定一段正弦型曲线图像，求函数表达式，并讨论单调区间、对称性等；  
- 实际情境类，如“摩天轮上座椅离地高度 $h$（米）与时间 $t$（秒）满足 $h = 40\sin\left(\frac{\pi}{30}t - \frac{\pi}{2}\right) + 50$，求最高点时间和高度”“当 $h \ge 70$ 时，求持续时间”；  
- 结合导数考查最值（如交流电瞬时功率 $p = UI\sin\omega t \cos\omega t$ 的最值问题，简化为正弦型函数求最值）。

## ⚠️ 高频易错点

- **$A$ 的正负与振幅**：振幅一般取正数 $|A|$，若明确给出 $A$ 为负，可化为 $|A|\sin(\omega x + \varphi + \pi)$，切忌直接视 $A$ 为振幅而导致最值错误。  
- **初相 $\varphi$ 的求解**：直接代入点坐标解三角方程时，忽略该点处于“五点”中的具体位置（如上升零点、峰点），导致 $\varphi$ 多解或错选范围（待家长核对）。  
- **周期与角频率**：常混淆 $f = \frac{1}{T}$（频率）与 $\omega = \frac{2\pi}{T}$（角频率），计算时用错倍数。  
- **实际定义域限制**：应用题中 $x$（时间、角度等）有取 $x\ge 0$ 等限制，解三角不等式求区间时，未排除负值或超出周期的重复解，导致答案多出或遗漏。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 振幅 | amplitude | 函数最大偏移平衡位置的距离，即 $|A|$ |
| 角频率 | angular frequency | 参数 $\omega$，与周期关系为 $\omega = 2\pi/T$ |
| 初相 | initial phase | 参数 $\varphi$，决定 $x=0$ 时的状态 |
| 周期 | period | 函数值重复出现的最小正间隔，$T = 2\pi/|\omega|$ |
| 相位 | phase | 括号内的整体 $\omega x + \varphi$，描述波动的位置 |
