# 三角函数求值

> **一句话**：利用三角恒等变换公式，通过已知角的三角函数值或角的关系，求出目标角（式）的三角函数值。
> **English**: Evaluating trigonometric values refers to the process of computing specific trigonometric function values for an angle or expression using identities and known values.

## 🎓 考点精讲

三角函数求值的核心是**角的变换**与**公式的灵活选用**，本质是通过恒等变形将非特殊角化为特殊角、将未知函数值向已知函数值转化。

最常用的公式体系包括：
- **同角基本关系式**：$\sin^2\alpha + \cos^2\alpha = 1$，$\tan\alpha = \frac{\sin\alpha}{\cos\alpha}$
- **诱导公式**：口诀"奇变偶不变，符号看象限"，用于将任意角化为锐角
- **两角和差公式**：
  $$\sin(\alpha \pm \beta) = \sin\alpha\cos\beta \pm \cos\alpha\sin\beta$$
  $$\cos(\alpha \pm \beta) = \cos\alpha\cos\beta \mp \sin\alpha\sin\beta$$
  $$\tan(\alpha \pm \beta) = \frac{\tan\alpha \pm \tan\beta}{1 \mp \tan\alpha\tan\beta}$$
- **二倍角公式**：$\sin 2\alpha = 2\sin\alpha\cos\alpha$，$\cos 2\alpha = \cos^2\alpha - \sin^2\alpha = 2\cos^2\alpha - 1 = 1 - 2\sin^2\alpha$
- **辅助角公式**：$a\sin x + b\cos x = \sqrt{a^2+b^2}\sin(x + \varphi)$（$\tan\varphi = \frac{b}{a}$）

解题关键步骤为"三看"——**看角、看名、看结构**，选择最优变换路径：
1. 观察角之间的关系（和、差、倍、半、互余、互补），确定变换方向；
2. 统一函数名（切化弦、弦化切），减少不同名函数的干扰；
3. 分析式子结构（乘积、和差、平方），匹配对应公式。

高考常见设问角度有：
- **给角求值**：如求 $\sin 75^\circ$ 的值，拆角为 $75^\circ = 45^\circ + 30^\circ$，套用两角和公式；
- **给值求值**：已知 $\cos\alpha = \frac{1}{3}$ 且 $\alpha \in (0,\pi)$，求 $\sin\frac{\alpha}{2}$，需用升降幂公式并关注象限对符号的约束；
- **给值求角**：已知 $\tan\alpha = 2$，$\tan\beta = 3$ 且 $\alpha,\beta$ 为锐角，求 $\alpha+\beta$，先求 $\tan(\alpha+\beta)$ 再结合范围定角。

## ⚠️ 高频易错点

- **忽略角的范围导致符号错误**：利用 $\sin^2\alpha + \cos^2\alpha = 1$ 开平方求 $\sin\alpha$ 或 $\cos\alpha$ 时，必须根据角所在象限确定正负号，未判定范围直接取正是最常见错误。
- **诱导公式符号误判**：对 $\sin(\pi + \alpha)$、$\cos(\frac{\pi}{2} - \alpha)$ 等变形时，不习惯将 $\alpha$ 视为锐角进行符号推演，导致正负号出错。
- **升幂降幂时忽视符号**：如由 $\cos 2\alpha = 2\cos^2\alpha - 1$ 推 $\cos\alpha = \pm\sqrt{\frac{1+\cos 2\alpha}{2}}$，忘记添加 $\pm$ 号，尤其在小题求值时直接漏掉负值可能。
- **拆角配凑中扩大或缩小定义域**：给值求角时，仅根据函数值相等就下结论，未同时考虑角的取值区间是否唯一，可能漏解或多解。
