# 两角差的正弦公式

> **一句话**：两角差的正弦公式是 $$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$，描述两个角之差的正弦值与各自正余弦值的关系。
> **English**: $$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$

## 🎓 考点精讲

**定义**：对于任意角 $$\alpha$$、$$\beta$$，两角差的正弦公式为：
$$\sin(\alpha - \beta) = \sin\alpha\cos\beta - \cos\alpha\sin\beta$$

该公式是三角函数和差公式的核心组成部分，通过单位圆上的几何关系或向量数量积可以证明。其显著特征是**正弦差等于“正弦余弦交替相减”**，即 $$\sin\alpha\cos\beta$$ 减去 $$\cos\alpha\sin\beta$$，符号与差角一致。

**关键方法**：
1. **正向展开**：已知 $$\alpha$$、$$\beta$$ 的三角函数值时，直接代入公式计算 $$\sin(\alpha - \beta)$$。
2. **逆用化简**：将形如 $$\sin A\cos B - \cos A\sin B$$ 的表达式合并为 $$\sin(A - B)$$，用于三角恒等变换、三角方程求解。
3. **配凑角策略**：高考中常给出非特殊角，需将其拆分为两个已知角的差，如 $$\frac{\pi}{12} = \frac{\pi}{3} - \frac{\pi}{4}$$。

**典型应用**：
- **给值求值**：已知 $$\sin\alpha$$、$$\cos\beta$$ 等部分值，求 $$\sin(\alpha - \beta)$$。需先利用同角三角函数基本关系补齐所需的正余弦值，特别注意象限对符号的影响。
- **化简证明**：将复杂的三角函数式通过逆用公式合并，如化简 $$\sin(60^\circ + x)\cos x - \cos(60^\circ + x)\sin x = \sin 60^\circ = \frac{\sqrt{3}}{2}$$。

## ⚠️ 高频易错点

- **错记符号**：易将公式记为 $$\sin(\alpha - \beta) = \sin\alpha\cos\beta + \cos\alpha\sin\beta$$（混淆为两角和的符号），必须牢记**差角公式中间为减号**。
- **顺序写反**：展开时错写为 $$\cos\alpha\sin\beta - \sin\alpha\cos\beta$$，与 $$\sin(\beta - \alpha)$$ 混淆。$$\sin(\alpha - \beta) = -\sin(\beta - \alpha)$$，展开顺序不可颠倒。
- **与余弦差公式混淆**：$$\cos(\alpha - \beta) = \cos\alpha\cos\beta + \sin\alpha\sin\beta$$ 是“余弦同号、正弦异号”，学生常将正余弦公式的符号规律记混。可借助特值（如 $$\alpha=\beta$$）快速检验：$$\sin0=0$$、$$\cos0=1$$。
- **忽略隐含条件**：给值求值时，未根据角的范围判断平方根的正负号，导致 $$\cos\alpha$$ 或 $$\sin\beta$$ 的符号出错。
