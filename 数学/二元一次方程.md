## 课本定义

含有**两个未知数**（如 x 和 y），且未知数的次数都是 **1次**，这样的整式方程叫做**二元一次方程**。其一般形式为 ax + by = c（a, b 不全为零）。

---

## 📚 6-12 岁回溯版

### "两个未知数，一个方程"

我们学过**一元一次方程**：只有一个未知数（x）。
现在认识**二元一次方程**：有**两个**未知数（x 和 y），但每个都是一次的。

**例子：**
- x + y = 10（一瓶水和一个苹果共10元）
- 2x + 3y = 12（买2个橙子和3个香蕉共12元）
- x - y = 5（哥哥比弟弟大5岁）

**特点：**
- 两个未知数 x 和 y
- x 和 y 的指数都是 1（不能是 x²、y³ 等）
- 是方程（有等号）

**二元一次方程的解：**
一元一次方程的解只有一个值（如 x = 3）。
二元一次方程的解是**一对数** (x, y)：

例：x + y = 5 的解有很多：
- x = 1, y = 4
- x = 2, y = 3
- x = 3, y = 2
- x = 0, y = 5
- x = -1, y = 6
……有无数个解！

所以，**单独一个二元一次方程**有无数个解，需要**再加一个方程**（组成方程组）才能确定唯一解。

---

## 🎓 12+ 进阶版

### 严格定义

**二元一次方程**：含两个未知数 x, y，且每个未知数的最高次数均为 1 的整式方程。

标准形式：**ax + by = c**（a, b 不全为 0，a, b, c 为实数）

### 解的结构

二元一次方程 ax + by = c（b ≠ 0）的解集：
- 从中解出 y = (c - ax) / b
- x 取任意实数，对应一个 y 值
- 解的集合是无穷多个有序数对 (x, y)
- 在坐标系中，这些解对应一条**直线**

**图形意义**：ax + by = c 的解集是平面上的一条直线（这是直线方程的雏形）。

### 判断是否是二元一次方程

判断标准：
1. 是**整式**方程（无分母含未知数，无根号下含未知数）
2. 恰好含**两个**未知数
3. 每个未知数的次数均为 **1**
4. 含有**等号**

| 方程 | 是否二元一次方程 | 原因 |
|------|---------------|------|
| 2x + y = 5 | 是 | 满足全部条件 |
| x² + y = 3 | 否 | x 是二次 |
| x + 1/y = 2 | 否 | 1/y 不是整式 |
| x + y + z = 1 | 否 | 三个未知数 |
| 3x - 4y = 0 | 是 | c = 0 也可 |

### 典型例题

**例题1：** 判断 x = 2, y = 1 是否是 2x - 3y = 1 的解。

解：代入 2(2) - 3(1) = 4 - 3 = 1 = 右边
∴ x=2, y=1 是方程 2x - 3y = 1 的解。

**例题2：** 已知 x = a, y = 2 是方程 3x + 2y = 13 的解，求 a。

解：代入 3a + 2(2) = 13
3a + 4 = 13
3a = 9
a = 3

**例题3：** 写出方程 x + 2y = 6 的三组正整数解。

解：令 y = 1: x = 6 - 2(1) = 4，得 (4, 1)
令 y = 2: x = 6 - 2(2) = 2，得 (2, 2)
令 y = 0: x = 6 - 2(0) = 6，得 (6, 0)

### 易错点

1. **二元一次方程有无数个解**，不是唯一解（需要方程组才能唯一确定）
2. **判断次数时看每一项**：xy（x 和 y 各一次，总次数=2，但每个变量次数是1，仍是二元一次方程——注意：xy 项次数为2，严格来说 xy 含 xy 项时不是二元一次！）
3. **ax + by = c 中 a, b 不全为零**（若都为零，则 0 = c 是矛盾或恒等，不是方程）

### 知识联系

```
二元一次方程
├── 单独 → 无穷多个解（一条直线）
├── 组合两个 → 二元一次方程组（唯一解/无解/无穷多解）
├── 解法 → 代入消元法、加减消元法
└── 图形 → 直线方程（初步认识）
```
