# 二次函数顶点式

> **一句话**：$y = a(x-h)^2 + k$（$a \neq 0$）称为二次函数的顶点式，顶点坐标为 $(h, k)$，对称轴为 $x = h$。
> **English**: $y = a(x-h)^2 + k$ is the vertex form of a quadratic function; the vertex is $(h, k)$ and the axis of symmetry is $x = h$.

---

## 📚 给 6-12 岁（自读版）

### 为什么叫"顶点式"？

因为从这种写法里，可以**直接读出顶点的坐标**！

$$y = a\underbrace{(x-h)^2}_{\geq 0} + k$$

- 当 $x = h$ 时，$(x-h)^2 = 0$，函数值最小（$a>0$）或最大（$a<0$）
- 所以顶点就是 $(h, k)$！

### 如何读出顶点

| 函数 | 顶点 $(h, k)$ | 对称轴 |
|------|-------------|--------|
| $y = (x-2)^2 + 3$ | $(2, 3)$ | $x = 2$ |
| $y = (x+1)^2 - 5$ | $(-1, -5)$ | $x = -1$ |
| $y = -2(x-3)^2 + 7$ | $(3, 7)$ | $x = 3$ |
| $y = x^2 + 4$ | $(0, 4)$ | $x = 0$ |

**注意**：$(x - h)$ 中减法对应正数，加法对应负数！

$y = (x\mathbf{+}1)^2 - 5 = (x - \mathbf{(-1)})^2 - 5$，所以 $h = -1$（不是 $1$！）

### 从一般式转化为顶点式

方法：**配方**

将 $y = x^2 - 4x + 1$ 转化：

$$y = (x^2 - 4x + 4) - 4 + 1 = (x-2)^2 - 3$$

顶点：$(2, -3)$，对称轴：$x = 2$

## 🎓 给 12+（进阶版）

### 定义

二次函数的**顶点式**：

$$\boxed{y = a(x-h)^2 + k \quad (a \neq 0)}$$

- 顶点坐标：$(h, k)$
- 对称轴：$x = h$
- 函数最值：
  - $a > 0$：最小值为 $k$（在 $x = h$ 处取得）
  - $a < 0$：最大值为 $k$（在 $x = h$ 处取得）

### 从一般式 $y = ax^2+bx+c$ 转化为顶点式

配方过程：

$$y = a\left(x^2 + \frac{b}{a}x\right) + c = a\left[\left(x + \frac{b}{2a}\right)^2 - \frac{b^2}{4a^2}\right] + c$$
$$= a\left(x + \frac{b}{2a}\right)^2 - \frac{b^2}{4a} + c$$
$$= a\left(x + \frac{b}{2a}\right)^2 + \frac{4ac - b^2}{4a}$$

对比：$h = -\dfrac{b}{2a}$，$k = \dfrac{4ac-b^2}{4a} = c - \dfrac{b^2}{4a}$

### 三种形式互化

$$y = ax^2 + bx + c \xrightarrow{\text{配方}} y = a(x-h)^2 + k \xrightarrow{\text{展开}} y = ax^2 + bx + c$$

$$y = a(x-x_1)(x-x_2) \xrightarrow{\text{展开}} y = ax^2 + \ldots$$

### 顶点式的应用

**1. 直接求最值（不等式中常用）**

$y = 2(x-1)^2 + 3$ 的最小值为 $3$（当 $x=1$ 时）。

**2. 求函数在区间上的最值**

**例**：求 $y = (x-2)^2 - 1$ 在 $1 \leq x \leq 4$ 上的最小值和最大值。

- 顶点 $x=2$ 在区间内，最小值 $y(2) = -1$
- 端点：$y(1) = 1-1=0$，$y(4) = 4-1=3$
- 最大值为 $y(4) = 3$

**3. 已知顶点构造函数**

顶点 $(1, -2)$，过点 $(0, 1)$，求解析式。

设 $y = a(x-1)^2 - 2$，代入 $(0,1)$：$1 = a(-1)^2 - 2 = a - 2$，得 $a = 3$。

解析式：$y = 3(x-1)^2 - 2$

**4. 判断平移关系**

$y = (x-3)^2 + 2$ 是 $y = x^2$ 向右移 3 单位、再向上移 2 单位的结果。

### 典型例题

**例（综合）**：已知二次函数对称轴为 $x=2$，且经过 $(1, 0)$ 和 $(0, -3)$。

对称轴 $x=2$ → 顶点式 $y = a(x-2)^2 + k$

代入 $(0, -3)$：$-3 = 4a + k$ ①

代入 $(1, 0)$：$0 = a + k$ ② 

由②得 $k = -a$，代入①：$-3 = 4a - a = 3a$，$a = -1, k = 1$

解析式：$y = -(x-2)^2 + 1$

验证：过 $(1,0)$：$-(1-2)^2+1=0$ ✓；过 $(0,-3)$：$-(0-2)^2+1=-3$ ✓

### 易错点

- ❌ 顶点坐标读错符号：$y=(x+3)^2-1$ 的顶点是 $(-3,-1)$，不是 $(3,-1)$
- ❌ 配方时系数处理不当（$a\neq1$ 时先提取 $a$）
- ❌ 区间最值只看顶点，忘记验证端点
- ❌ 认为顶点就是最大值（$a<0$ 时顶点是最大值，$a>0$ 时是最小值）
