## 🎓 余弦函数图象与性质

### 定义

**余弦函数**：$y = \cos x$，$x \in \mathbb{R}$

### 基本性质

| 性质 | 内容 |
|------|------|
| 定义域 | $\mathbb{R}$ |
| 值域 | $[-1, 1]$ |
| 最大值 | $1$，在 $x = 2k\pi$（$k \in \mathbb{Z}$）时取得 |
| 最小值 | $-1$，在 $x = \pi + 2k\pi$（$k \in \mathbb{Z}$）时取得 |
| 周期 | $T = 2\pi$ |
| 零点 | $x = \dfrac{\pi}{2} + k\pi$（$k \in \mathbb{Z}$） |
| 奇偶性 | 偶函数（$\cos(-x) = \cos x$），关于 $y$ 轴对称 |
| 对称轴 | $x = k\pi$（$k \in \mathbb{Z}$）（过极值点的竖线） |

### 单调区间

$$\text{单调递减区间：} [2k\pi, (2k+1)\pi], \quad k \in \mathbb{Z}$$

即从 $0, 2\pi, 4\pi, \ldots$ 到 $\pi, 3\pi, 5\pi, \ldots$ 各段递减。

$$\text{单调递增区间：} [(2k-1)\pi, 2k\pi], \quad k \in \mathbb{Z}$$

即从 $-\pi, \pi, 3\pi, \ldots$ 到 $0, 2\pi, 4\pi, \ldots$ 各段递增。

### 与正弦函数的关系

$$\cos x = \sin\!\left(x + \frac{\pi}{2}\right) = \sin\!\left(\frac{\pi}{2} - x\right)$$

余弦函数是正弦函数向左平移 $\dfrac{\pi}{2}$ 个单位所得，两者图象形状完全相同（均为正弦曲线形状），只是相位不同。

### 关键点（描绘一个周期）

$(0, 1), (\pi/2, 0), (\pi, -1), (3\pi/2, 0), (2\pi, 1)$

从 $(0, 1)$（最高点）开始，先降到 $(\pi, -1)$（最低点），再回到 $(2\pi, 1)$。

### 典型例题

**例1**：求 $y = \cos x$ 在 $\left[0, \dfrac{3\pi}{2}\right]$ 上的单调递减区间。

解：余弦递减区间为 $[2k\pi, \pi + 2k\pi]$（$k \in \mathbb{Z}$）。与 $[0, 3\pi/2]$ 的交集为 $[0, \pi]$。

故在 $[0, 3\pi/2]$ 上的单调递减区间为 $[0, \pi]$。

**例2**：比较 $\cos(-\pi/5)$ 与 $\cos(\pi/4)$ 的大小。

解：$\cos(-\pi/5) = \cos(\pi/5)$（偶函数）。

$\pi/5 < \pi/4$（均在 $[0, \pi/2]$ 内，递减区间），$\pi/5$ 更小，故 $\cos(\pi/5) > \cos(\pi/4)$，即 $\cos(-\pi/5) > \cos(\pi/4)$。

**例3**：解方程 $\cos x = -\dfrac{\sqrt{3}}{2}$（$x \in [0, 2\pi]$）。

解：$\cos x < 0$ 说明 $x$ 在第二或第三象限。

$|\cos x| = \sqrt{3}/2 = \cos(\pi/6)$（参考角 $\pi/6$）。

解：$x_1 = \pi - \dfrac{\pi}{6} = \dfrac{5\pi}{6}$（第二象限），$x_2 = \pi + \dfrac{\pi}{6} = \dfrac{7\pi}{6}$（第三象限）。

### 易错点

1. **余弦是偶函数**：$\cos(-x) = \cos x$，关于 $y$ 轴对称，而非关于原点
2. **最大值在 $x = 0$，不是 $x = \pi/2$**：$\cos 0 = 1$，$\cos(\pi/2) = 0$，不要与正弦混淆
3. **递减区间从 $0$ 到 $\pi$**：$\cos$ 在 $[0, \pi]$ 递减（从 $1$ 到 $-1$），在 $[\pi, 2\pi]$ 递增
4. **零点在 $\pi/2 + k\pi$**：$\cos(\pi/2) = 0$，$\cos(3\pi/2) = 0$ 等，不是 $k\pi$
5. **余弦与正弦的偏移**：$\cos x = \sin(x + \pi/2)$，向左移 $\pi/2$ 就变正弦；$\sin x = \cos(x - \pi/2)$

### 与初中知识的衔接

初中了解了余弦作为三角比（邻边/斜边），高中将余弦扩展为全实数上的函数，并与正弦函数建立明确的平移关系（$\cos x = \sin(x+\pi/2)$）。理解余弦曲线是学习 $y = A\cos(\omega x + \varphi)$ 型函数的基础。
