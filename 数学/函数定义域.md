# 函数定义域

> **一句话**：使函数解析式有意义的自变量 $x$ 的全体取值集合（或实际问题限定的自变量范围）。
> **English**: A set of all input values for which a function is defined.

## 🎓 考点精讲

- **定义/原理**：函数 $y=f(x)$ 的**自然定义域**，是指在没有指定实际背景时，使解析式运算有效的所有实数 $x$。数学上即 $D=\{x\mid f(x) \text{ 有意义},\ x\in\mathbb{R}\}$。若为复合函数 $y=f(g(x))$，则要求 $x$ 在 $g$ 的定义域内，且 $g(x)$ 的值落在 $f$ 的定义域内。

- **关键方法**：求定义域的标准步骤为**逐条列出约束条件 → 解不等式（组）→ 求交集**。常见约束包括：
  - 分式中分母 $\neq 0$；
  - 偶次根式（如 $\sqrt{}$， $\sqrt[4]{}$）中被开方数 $\geq 0$；
  - 对数式 $\log_a u$ 中真数 $u&gt;0$ 且底数 $a&gt;0,\ a\neq 1$；
  - 零指数、负指数幂的底数 $\neq 0$；
  - 正切 $\tan x$ 中 $x\neq k\pi+\frac{\pi}{2},\ k\in\mathbb{Z}$。
  复合函数 $f[g(x)]$ 需由外层 $f$ 的定义域反解内层 $g(x)$ 的范围，再与 $g(x)$ 自身定义域取交集。

- **典型应用**：高考常以选择题、填空题直接考查“求下列函数的定义域”，或渗透在函数单调性、奇偶性判断、解不等式等综合题中。例如：
  1. 求 $f(x)=\frac{\sqrt{x^2-4}}{\ln(x-1)}$ 的定义域：需满足 $x^2-4\geq 0$， $x-1&gt;0$ 且 $\ln(x-1)\neq 0$，解得 $[2,+\infty)$。
  2. 已知 $f(x)$ 定义域为 $[-1,3]$，求 $f(2x+1)$ 的定义域：由 $-1\leq 2x+1\leq 3$ 得 $x\in[-1,1]$。

## ⚠️ 高频易错点

- **忽略定义域的“集合”要求**：结果必须写成**集合或区间**形式（如 $\{x\mid x>1\}$ 或 $(1,+\infty)$），不能只写不等式。
- **复合函数定义域混淆**：误把中间变量范围当自变量范围。例如已知 $f(x+1)$ 的定义域为 $[0,2]$，指的是 $x\in[0,2]$，而非 $x+1\in[0,2]$（此时应推得 $x+1\in[1,3]$，即 $f$ 的定义域为 $[1,3]$，待家长核对）。
- **“或”与“且”误用**：多条约束必须同时满足（求交集），不能取并集。
- **实际情境限制忽略**：应用题中自变量常受几何度量（如线段长度>0）、物理量（如时间非负）等隐含条件约束。
