# 单位向量

> **一句话**：单位向量是模（长度）为 1 的向量，用于刻画方向而不关心长度大小。
> **English**: A unit vector is a vector with magnitude (length) equal to 1, used to represent direction independently of length.

## 🎓 考点精讲

- **定义与原理**  
  设 $\vec{a}$ 为非零向量，则其**单位向量** $\vec{e}$ 定义为$\displaystyle \vec{e} = \frac{\vec{a}}{|\vec{a}|}$。  
  显然 $|\vec{e}| = 1$，且 $\vec{e} \parallel \vec{a}$。零向量没有对应的单位向量。

- **关键方法**  
  1. **求任意非零向量的单位向量**：用向量的坐标除以它的模。如 $\vec{a} = (x, y, z)$，单位向量为 $\left(\frac{x}{|\vec{a}|}, \frac{y}{|\vec{a}|}, \frac{z}{|\vec{a}|}\right)$。  
  2. **求与给定向量平行的单位向量**：记 $\vec{e} = \pm \frac{\vec{a}}{|\vec{a}|}$，有两个方向。  
  3. **利用单位向量简化表达**：若已知单位方向向量 $\vec{e}$，则与该方向平行且长度为 $r$ 的向量可表示为 $\pm r\vec{e}$。

- **典型应用（高考常见设问）**  
  • **在平面/空间直角坐标系中**，求与 $\vec{AB}$ 同向（或反向）的单位向量坐标。  
    *示例*：已知点 $A(1,2), B(4,6)$，求与 $\overrightarrow{AB}$ 同向的单位向量。  
    求解：$\overrightarrow{AB} = (3,4)$，$|\overrightarrow{AB}| = 5$，单位向量为 $\left(\frac{3}{5},\frac{4}{5}\right)$。  
  • **作为方向向量或法向量的基底**：直线的方向向量常取为单位向量以简化点到直线的距离公式；平面的单位法向量可直接参与点到面的距离计算。  
  • **结合三角函数**：单位圆上的点对应从原点出发的单位向量 $(\cos\theta, \sin\theta)$。

## ⚠️ 高频易错点

- **忽略零向量**：零向量没有对应的单位向量，因为分母 $|\vec{0}| = 0$ 无意义。解题时需提前判断向量是否为零。
- **漏掉相反方向**：求与某向量平行的单位向量时，只写出同向的一个而忘记反向的 $-\frac{\vec{a}}{|\vec{a}|}$，导致丢解。
- **混淆单位向量与基向量**：基向量（如 $\vec{i}, \vec{j}$）不一定是单位向量，只有在标准正交基下基向量才是单位长度为 1 且两两垂直。
- **运算中忽略分母的有理化或化简**：单位向量的表达式中的分量常带根号，需要化成最简形式，部分同学在后续运算中因未有理化而算错。
