# 反比例函数与一次函数

> **一句话**：反比例函数与一次函数的交点、图象位置关系是综合题常见考点，关键是联立方程求交点。

---

## 📚 给 6-12 岁（自读版）

### 两种函数的基本区别

| | 一次函数 | 反比例函数 |
|--|---------|-----------|
| 表达式 | $y = kx + b$ | $y = \dfrac{k}{x}$ |
| 图象 | 直线 | 双曲线（两支） |
| 过原点 | $b=0$ 时过 | 不过原点 |

### 图象可能相交

一条直线（一次函数）和双曲线（反比例函数）可能有 0 个、1 个或 2 个交点。

### 直观例子

$y = x$（直线）与 $y = \dfrac{4}{x}$（双曲线）：

交点满足 $x = \dfrac{4}{x}$，即 $x^2=4$，$x=\pm 2$，交点为 $(2,2)$ 和 $(-2,-2)$。

---

## 🎓 给 12+（进阶版）

### 联立求交点

设直线 $y = ax + b$ 与双曲线 $y = \dfrac{k}{x}$ 的交点，联立：

$$ax + b = \frac{k}{x} \implies ax^2 + bx - k = 0$$

用判别式 $\Delta = b^2 + 4ak$ 判断交点个数：
- $\Delta > 0$：两个交点
- $\Delta = 0$：一个交点（切线）
- $\Delta < 0$：无交点

### 由图象信息判断 $k$、$b$ 符号

已知一次函数 $y = kx + b$ 的图象，利用其穿过的象限推断 $k$、$b$ 的符号；再利用与反比例函数的位置关系推断反比例函数 $k$ 的符号——这是常见综合判断题型。

### 典型例题

**例 1**：直线 $y = x + 2$ 与反比例函数 $y = \dfrac{k}{x}$ 有交点 $(1, m)$，求 $k$ 和 $m$。

解：将 $x=1$ 代入直线：$m = 1+2 = 3$。
代入反比例函数：$3 = \dfrac{k}{1}$，故 $k=3$。

**例 2**：已知反比例函数 $y=\dfrac{3}{x}$ 和一次函数 $y=x+b$ 的图象都过点 $A(a,1)$，求 $a$ 和 $b$。

解：点 $A$ 在反比例函数上：$1 = \dfrac{3}{a}$，解得 $a=3$。
点 $A(3,1)$ 在一次函数上：$1 = 3 + b$，解得 $b=-2$。

### 综合：比较函数值大小

在交点之间或之外，利用图象位置可以比较 $y_{\text{直线}}$ 与 $y_{\text{双曲线}}$ 的大小，而不必代入计算。

### 易错点

- ❌ 联立时忘记限制 $x \neq 0$
- ❌ 误以为直线与双曲线必有两个交点
