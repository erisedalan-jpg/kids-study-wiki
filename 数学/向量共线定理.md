# 向量共线定理

> **一句话**：向量共线定理表述为：若非零向量 $\vec{a}$ 与 $\vec{b}$ 共线，则存在唯一实数 $\lambda$ 使得 $\vec{b} = \lambda \vec{a}$；反之，若存在实数 $\lambda$ 使该式成立，则两向量共线。
> **English**: The collinear vector theorem states that two nonzero vectors are collinear if and only if one is a scalar multiple of the other.

## 🎓 考点精讲

**定义/原理**  
在平面向量中，向量共线定理（亦称共线向量基本定理）是判断两向量方向关系以及三点共线的核心工具。设 $\vec{a} \neq \vec{0}$，则 $\vec{b}$ 与 $\vec{a}$ 共线（即方向相同或相反）的充要条件是：存在唯一实数 $\lambda$，使得 $\vec{b} = \lambda \vec{a}$。  
该定理的向量形式常与坐标结合：若 $\vec{a}=(x_1,y_1)$，$\vec{b}=(x_2,y_2)$，且 $\vec{a} \neq \vec{0}$，则 $\vec{a} \mathbin{\parallel} \vec{b} \iff x_1y_2 - x_2y_1 = 0$。对于空间向量，坐标比值对应成比例且分母不为零即可判断共线。

**关键方法**  
1. **直接比例法**：找到一组基向量，判断另一向量能否写成其实数倍。  
2. **坐标法**：利用“对应坐标差乘积相等”消去参数，即利用 $x_1y_2 = x_2y_1$ 建立方程。  
3. **三点共线转化**：$A,B,C$ 三点共线 $\iff \overrightarrow{AB} = \lambda \overrightarrow{AC}$ 或 $\overrightarrow{OA} = (1-t)\overrightarrow{OB} + t\overrightarrow{OC}$（其中 $O$ 为平面内任意一点）。此常用于解决几何位置关系、参数求值问题。

**典型应用**  
高考常结合平面向量基本定理、坐标运算、三点共线证明与参数范围等设问。  
- **场景一**：已知 $\vec{a}=(1,2)$，$\vec{b}=(3,m)$，若 $\vec{a} \mathbin{\parallel} \vec{b}$，求 $m$。方法：由 $1\times m = 2\times 3$ 得 $m=6$。  
- **场景二**：在 $\triangle ABC$ 中，$D$ 在 $BC$ 上，且 $\overrightarrow{BD} = 2\overrightarrow{DC}$，用 $\overrightarrow{AB},\overrightarrow{AC}$ 表示 $\overrightarrow{AD}$。利用共线定理将 $\overrightarrow{BD}$ 用 $\overrightarrow{BC}$ 表示，再结合向量加法得到 $\overrightarrow{AD} = \frac{1}{3}\overrightarrow{AB} + \frac{2}{3}\overrightarrow{AC}$。

## ⚠️ 高频易错点

- **忽略“非零向量”条件**：应用 $\vec{b} = \lambda \vec{a}$ 时，必须先确认 $\vec{a} \neq \vec{0}$。若 $\vec{a} = \vec{0}$ 而 $\vec{b} \neq \vec{0}$，则不存在 $\lambda$ 满足等式；若 $\vec{a} = \vec{b} = \vec{0}$，则 $\lambda$ 不唯一。  
- **坐标共线判断漏分母为零情况**：使用 $x_1/x_2 = y_1/y_2$ 时，需讨论分母是否为零，正确通用等价式为 $x_1y_2 - x_2y_1 = 0$。  
- **三点共线系数和是否为1的混淆**：公式 $\overrightarrow{OA} = \alpha \overrightarrow{OB} + \beta \overrightarrow{OC}$，$A,B,C$ 三点共线的充要条件是 $\alpha + \beta = 1$，部分学生错误使用或遗忘前提 $O$ 为平面内任意一点且 $B,C$ 不重合。  
- **向量共线与直线平行的区别**：向量共线包括同向和反向，且零向量与任意向量共线，但几何中直线平行不包括重合，需注意在解析几何中条件转化时是否允许包含重合情形。
