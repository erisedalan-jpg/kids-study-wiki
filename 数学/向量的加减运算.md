# 向量的加减运算

> **一句话**：向量的加减运算是基于平行四边形法则（或三角形法则）对向量进行的线性运算，满足交换律与结合律，几何上体现为力的合成与位移的累积，代数上对应坐标的对应分量相加减。
> **English**: Vector addition and subtraction are linear operations on vectors based on the parallelogram (or triangle) law, satisfying commutativity and associativity, geometrically representing the composition of forces or consecutive displacements, and algebraically adding/subtracting corresponding components.

## 🎓 考点精讲

**定义/原理**：
设有两个非零向量 $\vec{a}$ 与 $\vec{b}$。
- **向量加法（三角形法则）**：将 $\vec{a}$ 的终点作为 $\vec{b}$ 的起点，则从 $\vec{a}$ 的起点指向 $\vec{b}$ 的终点的向量即为和向量 $\vec{a} + \vec{b}$。
- **向量加法（平行四边形法则）**：若 $\vec{a}$ 与 $\vec{b}$ 起点相同，以它们为邻边作平行四边形，则起点所在的对角线向量即为 $\vec{a} + \vec{b}$。
- **向量减法**：$\vec{a} - \vec{b} = \vec{a} + (-\vec{b})$，即减去一个向量等于加上它的相反向量（模相等，方向相反）。减法几何意义：起点相同的两向量差 $\vec{a} - \vec{b}$ 连接 $\vec{b}$ 的终点与 $\vec{a}$ 的终点，方向指向被减向量 $\vec{a}$。
- **坐标运算**：在平面直角坐标系中，若 $\vec{a} = (x_1, y_1)$，$\vec{b} = (x_2, y_2)$，则：
  $\vec{a} + \vec{b} = (x_1 + x_2, y_1 + y_2)$
  $\vec{a} - \vec{b} = (x_1 - x_2, y_1 - y_2)$
- **运算律**：加法满足交换律 $\vec{a} + \vec{b} = \vec{b} + \vec{a}$ 和结合律 $(\vec{a} + \vec{b}) + \vec{c} = \vec{a} + (\vec{b} + \vec{c})$。对于任意向量 $\vec{a}$，有 $\vec{a} + \vec{0} = \vec{a}$。

**关键方法**：
向量加减的核心解题思路为“化归为三角形或平行四边形”或“转化为坐标运算”。
1.  **几何法**：在三角形、四边形中，灵活运用“首尾相接，始终相连”求加法；“起点相同，指向被减”求减法。
2.  **代数坐标法**：建系将几何图形向量关系转化为纯代数坐标运算，直接分量相加减，能规避复杂几何构造。
3.  **模的运算**：求 $|\vec{a} \pm \vec{b}|$ 时，常通过平方转化为 $|\vec{a}|^2 \pm 2\vec{a} \cdot \vec{b} + |\vec{b}|^2$，并利用数量积 $\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}|\cos\theta$ 求解。

**典型应用**：
- **三角形与多边形中的线性表示**：高考常见题型如“在 $\triangle ABC$ 中，$D$ 为 $BC$ 中点，试用 $\overrightarrow{AB}$ 与 $\overrightarrow{AC}$ 表示 $\overrightarrow{AD}$”。解题关键为利用加法：$\overrightarrow{AD} = \overrightarrow{AB} + \overrightarrow{BD}$，通过加减运算转换为目标基底。
- **解析几何中的向量混合条件**：新高考常将向量条件融入解析几何大题。例如给出“$\overrightarrow{OA} + \overrightarrow{OB} = \overrightarrow{OC}$，已知 $A$、$B$ 两点求 $C$ 坐标”，直接套用坐标加法公式即可求出 $C$ 点坐标。
- **物理背景投射**：如“两力合力最大模为两力模之和（方向相同），最小模为两力模之差的绝对值（方向相反）”，即 $|\vec{a}| - |\vec{b}| \leq |\vec{a} + \vec{b}| \leq |\vec{a}| + |\vec{b}|$。

## ⚠️ 高频易错点

- **减法方向颠倒**：将“起点相同”的两向量差 $\vec{a} - \vec{b}$ 的方向误判为指向 $\vec{b}$ 的终点。必须牢记“连接两终点，方向指向被减向量”。
- **混淆向量加减与模的加减**：误以为 $|\vec{a} \pm \vec{b}| = |\vec{a}| \pm |\vec{b}|$，这是仅在向量共线且满足特定方向时才成立的等式，解题时缺乏加绝对值或平方展开意识直接错位使用。
- **忽略零向量与相反向量的特况**：在多边形向量化简中，面对首尾相接闭合的图形，忘记“所有向量和为 $\vec{0}$”这一简洁结论，导致步骤繁琐或遗漏。
- **坐标运算掉向**：从图形关系写坐标时，忘记“终减起”原则。如点 $A(1,2)$ 到 $B(3,5)$ 的向量 $\overrightarrow{AB} = (2, 3)$，而非 $(4, 7)$。
