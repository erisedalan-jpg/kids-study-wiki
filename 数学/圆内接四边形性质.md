# 圆内接四边形性质

> **一句话**：圆内接四边形是指四个顶点均在同一圆上的四边形，其核心性质为对角互补、外角等于内对角，且满足托勒密定理。
> **English**: A cyclic quadrilateral is a quadrilateral with all four vertices on a single circle; its key properties include supplementary opposite angles, exterior angle equal to the interior opposite angle, and Ptolemy's theorem.

## 🎓 考点精讲

**定义与核心性质**  
若四边形的四个顶点都在同一个圆上，则称其为圆内接四边形，该圆为四边形的外接圆。  
常用性质：  
1. **对角互补**：$\angle A + \angle C = 180^\circ$，$\angle B + \angle D = 180^\circ$。  
2. **外角定理**：任一外角等于其内对角。如图，延长$CD$至$E$，则$\angle DCE = \angle A$。  
3. **托勒密定理**：$AC \cdot BD = AB \cdot CD + AD \cdot BC$。该性质也是四点共圆的必要条件。

**判定四点共圆的方法**  
在高考题中识别或证明四点共圆是第一步：  
- 对角互补 → 四点共圆。  
- 外角等于内对角 → 四点共圆。  
- 若点$C$、$D$在直线$AB$同侧，且$\angle ACB = \angle ADB$，则$A,B,C,D$共圆。

**关键解题步骤**  
1. 寻找或证出四点共圆；  
2. 将对角关系转化为互补等式，建立方程；  
3. 结合圆周角定理（同弧所对圆周角相等）进行角度转化；  
4. 涉及线段长度时，优先运用托勒密定理或通过构造相似三角形求比例。  

**典型示例**  
已知圆内接四边形$ABCD$中，$\angle A : \angle B : \angle C = 2:3:4$，求$\angle D$。由对角互补，设$\angle A = 2x$，$\angle C = 4x$，则$2x+4x=180^\circ$，得$x=30^\circ$。$\angle B=3x=90^\circ$，故$\angle D=180^\circ - \angle B = 90^\circ$。

**高考中的典型设问**  
- 几何证明：证明四点共圆，或利用对角互补证明角相等。  
- 角度计算：已知若干角，求其他角或弧的度数。  
- 线段长度：结合托勒密定理或相似求对角线长、边长。  
- 解析几何：给出坐标，证明四点共圆并利用性质求参数。

## ⚠️ 高频易错点

- **对角误认为相等**：圆内接四边形的对角是**互补**（和为180°）而非相等，易与平行四边形等特殊图形混淆。
- **判定时忽略点的共侧条件**：利用“同弧上的圆周角相等”判定四点共圆时，必须确保两个角在公共弦的同侧，否则不成立。
- **托勒密定理盲目套用**：托勒密定理 **仅适用于圆内接四边形**，不能用于任意凸四边形，需先确认四点共圆再使用。
- **顶角对应错误**：在多边形复杂图中，未正确区分哪个角是内对角、哪个是外角，导致互补关系找错。
