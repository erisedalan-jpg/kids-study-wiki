# 复数相等

> **一句话**：两个复数相等的充要条件是它们的实部与实部相等，且虚部与虚部分别相等。
> **English**: Two complex numbers are equal if and only if their real parts are equal and their imaginary parts are equal.

## 🎓 考点精讲

- **定义/原理**：设 $z_1 = a + bi$，$z_2 = c + di$（$a,b,c,d \in \mathbb{R}$），则  
  $$ z_1 = z_2 \iff a = c \;\text{且}\; b = d. $$  
  特别地，$a+bi = 0 \iff a = 0 \;\text{且}\; b = 0$。复数相等的本质是将一个复数等式分解为两个实数等式，是化虚为实的核心桥梁。

- **关键方法**：  
  1. **化为标准形式**：将已知复数统一写成 $x + yi$ 的形式（$x,\,y$ 为实数表达式）。  
  2. **列方程组**：利用实部与虚部分别相等，建立关于未知数的方程组。  
  3. **求解参数**：解所得方程组，得到参数的值，并检验是否满足实数条件。

- **典型应用**：  
  高考中常在以下设问中应用复数相等：  
  - 已知含参数的复数等式（如 $(1+i)x + (1-i)y = 2$），求实数 $x,\,y$；  
  - 求满足 $z + \overline{z} = 4$ 且 $z - \overline{z} = 6i$ 的复数 $z$；  
  - 解复数方程，如 $|z| - z = 1 + 2i$，需设 $z = x + yi$，代入后利用复数相等和模为实数建立方程组。  
  解题核心始终是“实实相等，虚虚相等”，即列两个实方程。

## ⚠️ 高频易错点

- **忽略化为标准形式**：遇到 $(a + bi) + (c + di)i$ 等未整理成实部加虚部的式子时，直接套用实虚相等导致错误。应先合并同类项，写成标准复数形式再列等式。  
- **错把模相等当复数相等**：$|z_1| = |z_2|$ 并不意味着 $z_1 = z_2$，可能出现多解或漏解，尤其当用两边平方消去根号时，要始终记住复数的相等和模的相等是两回事。  
- **忘记虚部系数本身是实数**：设 $z = x + yi$ 后列方程时，误将 $y$ 或含 $i$ 的项当作虚部，未剔除 $i$。正确的虚部是实数 $y$，不是 $yi$。  
- **忽视实数前提**：方程组解出的值如果不是实数，则不能构成复数的实部或虚部，此时原方程无解，但学生常忽略检验步骤，导致用非实数解充当结果。
