# 复数相等的充要条件

> **一句话**：两个复数相等的充要条件是它们的实部与实部相等，虚部与虚部相等。
> **English**: Two complex numbers are equal if and only if their real parts are equal and their imaginary parts are equal.

## 🎓 考点精讲

设 $z_1 = a + b\mathrm{i}$，$z_2 = c + d\mathrm{i}$，其中 $a,b,c,d \in \mathbb{R}$，则
$$
z_1 = z_2 \iff a = c \;\text{且}\; b = d.
$$
特别地，$a + b\mathrm{i} = 0 \iff a = 0 \;\text{且}\; b = 0$。 这个充要条件将 **复数相等** 转化为两个 **实数方程**，是复数运算与复数方程求解的根基。

**关键方法**：处理含参数复数等式时，必须先将等式两边整理成 $A + B\mathrm{i}$ 的标准代数形式，再让 **实部与实部相等、虚部与虚部相等**，列出实数方程组求解。注意虚部是 $b$ 而不是 $b\mathrm{i}$。

**典型应用**：高考中常在复数方程、待定系数法求复数、复数与向量综合等题型中出现。  
- 已知 $(1+2\mathrm{i})x + (3-4\mathrm{i})y = 5 + 6\mathrm{i}$，且 $x,y \in \mathbb{R}$，求 $x,y$。  
  将左边展开为 $(x+3y) + (2x-4y)\mathrm{i}$，根据复数相等得方程组 $\begin{cases} x+3y=5 \\ 2x-4y=6 \end{cases}$，解得 $x=3.8, y=0.4$（结果需验算）。  
- 已知复数 $z$ 满足 $z + \bar{z} = 4$，$(z - \bar{z})\mathrm{i} = -2$，求 $z$。设 $z=x+y\mathrm{i}\ (x,y\in\mathbb{R})$，代入条件利用复数相等的充要条件同样可列出关于 $x,y$ 的方程组。

## ⚠️ 高频易错点

- 混淆虚部：将 $a+b\mathrm{i}$ 的虚部错误地写成 $b\mathrm{i}$，导致列出的方程多带 $\mathrm{i}$，无法得到实数方程组。  
- 未将复数化为 $A+B\mathrm{i}$ 标准形式就强行“实部相等、虚部相等”，忽略分母实数化等变形步骤，造成等式两边实虚部错位。  
- 复数相等与向量相等混淆：复数相等的充要条件是实虚部分别相等，但在极坐标或几何问题中易误认为模相等且辐角相等即可（那是三角形式相等的条件，而非代数形式相等的充要条件）。  
- 复数等于 $0$ 的条件使用不当：认为 $a+b\mathrm{i}=0$ 只要 $a=0$ 或 $b=0$，忘记必须实部与虚部同时为零。
