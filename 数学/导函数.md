## 🎓 导函数

### 定义

若函数 $f(x)$ 在区间 $(a,b)$ 内每一点都可导，则对每个 $x\in(a,b)$，都对应一个导数值 $f'(x)$。这样定义的以 $x$ 为自变量的新函数称为 $f(x)$ 的**导函数**，记作：

$$f'(x),\quad y',\quad \frac{\mathrm{d}y}{\mathrm{d}x},\quad \frac{\mathrm{d}f}{\mathrm{d}x}$$

导函数是由逐点求导得到的函数，即：

$$f'(x) = \lim_{\Delta x\to 0}\frac{f(x+\Delta x)-f(x)}{\Delta x}$$

### 导函数与导数的区别

| | 导数 $f'(x_0)$ | 导函数 $f'(x)$ |
|---|---|---|
| 本质 | 一个**数**（极限值） | 一个**函数** |
| 含义 | $f(x)$ 在 $x_0$ 处的变化率 | $f(x)$ 在各点变化率的总描述 |
| 关系 | $f'(x_0)$ 是 $f'(x)$ 在 $x=x_0$ 处的函数值 |

### 常见函数的导函数

| $f(x)$ | $f'(x)$ |
|---|---|
| $C$（常数） | $0$ |
| $x^n$（$n\in\mathbb{R}$） | $nx^{n-1}$ |
| $\sin x$ | $\cos x$ |
| $\cos x$ | $-\sin x$ |
| $e^x$ | $e^x$ |
| $a^x$（$a>0,a\neq 1$） | $a^x\ln a$ |
| $\ln x$ | $\dfrac{1}{x}$ |
| $\log_a x$ | $\dfrac{1}{x\ln a}$ |

### 典型例题

**例 1** 已知 $f(x)=3x^2-2x+1$，求 $f'(x)$ 及 $f'(2)$。

$$f'(x) = 6x - 2, \quad f'(2) = 6\times 2 - 2 = 10$$

**例 2** 已知 $f'(x) = 2x-3$，问 $f(x)$ 的单调递增区间。

令 $f'(x)>0$：$2x-3>0$，$x>\dfrac{3}{2}$。单调递增区间为 $\left(\dfrac{3}{2}, +\infty\right)$。

**例 3** 已知 $f'(x)=x^2-4$，找 $f(x)$ 的驻点。

令 $f'(x)=0$：$x^2-4=0$，$x=\pm 2$。

### 高阶导数

导函数 $f'(x)$ 仍是 $x$ 的函数，可继续对它求导，得**二阶导数**：

$$f''(x) = [f'(x)]'$$

物理意义：若 $s(t)$ 为位移，$s'(t)=v(t)$ 为速度，$s''(t)=a(t)$ 为加速度。高中阶段主要使用一阶导数，二阶导数在凹凸性分析中简介。

### 易错点

- $f'(x_0)$ 是数，$f'(x)$ 是函数，混用会导致逻辑错误
- 求导函数时，$x$ 是变量，不能当成常数处理
- 导函数可能在某些点不存在（如绝对值函数在 $x=0$ 处）
