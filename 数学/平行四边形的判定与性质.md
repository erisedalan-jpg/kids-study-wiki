# 平行四边形的判定与性质

> **一句话**：平行四边形是两组对边分别平行的四边形，其主要性质与判定围绕边、角、对角线的数量与位置关系展开。
> **English**: A parallelogram is a quadrilateral with both pairs of opposite sides parallel; its properties and criteria for judgment revolve around quantitative and positional relationships among sides, angles, and diagonals.

## 🎓 考点精讲

**定义**：两组对边分别平行的四边形叫平行四边形，记作 $\square ABCD$。

**核心性质定理**（已知 $\square ABCD$ 可推出的结论）：
1. **边**：对边平行且相等，即 $AB \parallel CD$ 且 $AB=CD$，$AD \parallel BC$ 且 $AD=BC$。
2. **角**：对角相等，邻角互补，即 $\angle A = \angle C$，$\angle B = \angle D$；$\angle A + \angle B = 180^\circ$。
3. **对角线**：互相平分，即 $OA=OC$，$OB=OD$（$O$ 为对角线交点）。
4. **对称性**：是中心对称图形，对角线的交点为对称中心，不具备轴对称性（矩形、菱形等特殊情况除外）。

**判定定理**（由条件推出四边形是平行四边形）：
1. **两组对边分别平行**（定义判定）：
   $$AB \parallel CD \text{ 且 } AD \parallel BC \Rightarrow \square ABCD$$
2. **两组对边分别相等**：
   $$AB=CD \text{ 且 } AD=BC \Rightarrow \square ABCD$$
3. **一组对边平行且相等**：
   $$AB \parallel CD \text{ 且 } AB=CD \Rightarrow \square ABCD$$
   注意：必须指向同一组对边，组合不当（如 $AB \parallel CD$ 且 $AD=BC$）只能得到等腰梯形或一般四边形。
4. **对角线互相平分**：
   $$OA=OC \text{ 且 } OB=OD \Rightarrow \square ABCD$$
5. **两组对角分别相等**：
   $$\angle A = \angle C \text{ 且 } \angle B = \angle D \Rightarrow \square ABCD$$

**高考典型设问与解法**：
- **计算求值**：在平行四边形中求边长、角度、对角线长或面积。核心手段是**将条件转化到三角形中**：利用对角线互相平分构造全等三角形或解三角形；已知两边及夹角时，面积公式为 $S = AB \cdot AD \cdot \sin\angle A$。
- **证明判定**：证明某四边形为平行四边形。高考立体几何大题中常在底面证明中使用，或在平面几何选讲中直接以判定定理为推理链条。**关键一步**是依据题设条件在五条判定定理中择一作为目标，补齐所需中间结论（通常需借助全等三角形或中位线定理）。

## ⚠️ 高频易错点

- **混淆性质与判定**：不能把性质反用当判定。例如“对角线互相平分”既是性质也是判定；但“对角互补”不是判定，只有“对角相等”才能判定平行四边形。
- **“一组对边平行另一组对边相等”不是判定**：此条件可构成平行四边形，也可构成等腰梯形，必须严格按定理组合。
- **平行四边形的面积计算选错角**：面积公式 $S = ab\sin\theta$ 中的 $\theta$ 必须是**两边所夹的角**，误用对边夹角会导致结果错误。
- **忽视中心对称带来的中点结论**：过对角线交点作任意直线与对边相交，所得线段被交点平分，这一性质在选择题中常考，学生容易忽略。
