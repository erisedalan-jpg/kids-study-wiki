# 平面区域

> **一句话**：平面区域是指平面直角坐标系中满足一组不等式（或方程）约束的点构成的集合，是线性规划中可行域的几何基础。
> **English**: A planar region is the set of points in the coordinate plane that satisfy a given set of inequalities (or equations), forming the geometric foundation of the feasible region in linear programming.

## 🎓 考点精讲

**定义与原理**  
在平面直角坐标系中，二元一次不等式 $Ax+By+C>0$（或 $<0$）表示直线 $Ax+By+C=0$ 某一侧的所有点组成的**半平面**（不含边界时为开半平面，含等号时包含边界）。多个不等式构成的**不等式组**表示的区域是各半平面的交集，称为**平面区域**。线性规划问题中，约束条件均用二元一次不等式（或等式）给出，这些条件确定的公共区域即为**可行域**，它通常是凸多边形（或凸集）。

**关键方法**  
1. **画线定界**：将不等式中的不等号改为等号，画出对应直线（虚线表示不含边界，实线表示包含边界）。  
2. **特殊点定区**：选择直线一侧的一个特殊点（如原点，若原点在直线上则另选点），代入不等式检验；若成立则点所在侧即为所求区域，否则为另一侧。重复上述步骤处理所有不等式，最终取各区域的重叠部分。  
3. **最值求解**：对于目标函数 $z=ax+by$，将其变形为 $y=-\frac{a}{b}x+\frac{z}{b}$（$b\neq 0$），则 $z$ 的几何意义是直线在 $y$ 轴上的截距乘以 $b$。通过平移该直线，观察其与可行域有公共点时截距的极值，即可得到 $z$ 的最值。当可行域为封闭多边形时，最优解常在其顶点处取得（即**角点法**），只需计算各顶点坐标代入目标函数比较即可。

**典型应用**  
高考中常以线性规划题出现，典型设问有：
- 求目标函数 $z=ax+by$ 的最大值或最小值；
- 求目标函数 $z=\frac{y-b}{x-a}$（斜率型）的取值范围，等价于可行域内的点 $(x,y)$ 与定点 $(a,b)$ 连线的斜率范围；
- 求目标函数 $z=(x-a)^2+(y-b)^2$（距离平方型）的最值，等价于可行域内的点到定点距离的平方的最值；
- 已知目标函数的最优解存在或唯一，反求约束条件中的参数范围。

## ⚠️ 高频易错点

- **选错检验点导致区域画反**：默认用原点 $(0,0)$ 检验，但当原点恰好在边界直线上时，必须另选一点（如 $(1,0)$ 或 $(0,1)$）进行检验，否则无法判定区域侧。
- **忽视直线斜率与不等号方向的统一转化**：遇到 $Ax+By+C>0$，若想直接判断上、下区域，应先将不等式化为 $y>kx+b$（或 $y<$ 形式），否则极易将 $B$ 为负数时的方向弄反。
- **最值点判断不全面**：若目标函数对应的直线斜率与被平移的可行域某一边界斜率相等，且该边界包含在可行域内，则最优解可能有无穷多个（整个边界线段上的点均取同一最值），答题时需说明“无穷多点取得最值”而非只写一个点。
- **非线性目标函数转化时忽略定义域限制**：将 $z=\frac{y-b}{x-a}$ 转化为斜率时，需注意 $x\neq a$ 的点才有效，可行域内若有点横坐标为 $a$ 则这些点不在目标函数定义域内，应从可行域中排除。
