# 序数

> **一句话**：第一、第二、第三……告诉我们排在哪个位置——这就是序数。
> **English**: Ordinal numbers — first, second, third — tell us the position or order of things.

---

## 🧒 给 3-6 岁（共读版）

比赛跑步，谁先到终点，谁就是**第一名**！

- 第一个到的：**第一**（First）
- 第二个到的：**第二**（Second）
- 第三个到的：**第三**（Third）

这和"有几个"不一样——"第几个"说的是**位置**，不是数量。

**共读小活动：** 家里人排成一队，让孩子说"爸爸是第几个？妈妈是第几个？我是第几个？"换个方向再数一次——从另一边数，答案变了！

## 📚 给 6-12 岁（自读版）

### 序数 vs 基数：两种不同的"数"

| 概念 | 问题 | 例子 |
|------|------|------|
| **基数（cardinal）** | 一共有**几个**？ | 队里有 5 个小朋友 |
| **序数（ordinal）** | 排在**第几**位？ | 你排在第 3 位 |

同一个情境里，基数和序数同时存在：队里 **5** 个人（基数），你是**第 3** 个（序数）。

### 中文和英文的序数构成

**中文：** "第"+ 数字 → 第一、第二、第三……第十，以此类推，规则非常简单。

**英文：** 前三个是特殊形式，之后在数字后加 `-th`：

| 序数 | 中文 | 英文 | 特殊/规律 |
|------|------|------|---------|
| 1st | 第一 | first | 特殊 |
| 2nd | 第二 | second | 特殊 |
| 3rd | 第三 | third | 特殊 |
| 4th | 第四 | fourth | 规律：4 + th |
| 5th | 第五 | fifth | 注意拼写（five → fifth）|
| 6th | 第六 | sixth | |
| 7th | 第七 | seventh | |
| 8th | 第八 | eighth | 注意拼写（eight + h）|
| 9th | 第九 | ninth | 注意拼写（nine → ninth）|
| 10th | 第十 | tenth | |
| 11th | 第十一 | eleventh | |
| 12th | 第十二 | twelfth | 注意拼写 |
| 20th | 第二十 | twentieth | |
| 21st | 第二十一 | twenty-first | 末位是 1，用 st |

### 序数在生活中

- **楼层**：住在3楼 = 住在第三层
- **日期**：5月1日 = May 1st（the first of May）
- **名次**：第一名、第二名……冠亚季军
- **队列**：排队时说第几个

## 🎓 给 12+（进阶版）

### 集合论中的序数

有限集合的序数直觉简单——第 1、第 2、第 3……。但数学家冯·诺伊曼（John von Neumann）给出了序数的严格集合论定义：

> 每个序数等于所有比它小的序数组成的集合。

- 0 = {} （空集）
- 1 = {0}
- 2 = {0, 1}
- 3 = {0, 1, 2}

这个定义在有限时和直觉一致，在无限时可以自然延伸到**超穷序数（transfinite ordinals）**。

### 超穷序数：无限序列之后

当你数完所有自然数 0, 1, 2, 3, …，第"最后"一个之后的序数叫 **ω（omega）**：

> ω = {0, 1, 2, 3, …}

ω 之后还有 ω+1, ω+2, … 2ω, 2ω+1, …，ω², …，ωω，……甚至更大的超穷序数。

这说明"无限之后"还可以有"更多的无限"——集合论打开了一个远比直觉更宏大的世界。（**待家长核对此段，建议搭配康托尔超穷数入门读物**）

### 编程中的序数：0 还是 1 开始？

在数学和日常语言里，序数从 1 开始（第一名、第二名）。但**大多数编程语言**（C、Python、JavaScript……）的数组索引从 **0** 开始：

```python
fruits = ["苹果", "香蕉", "橙子"]
fruits[0]  # "苹果"（第一个）
fruits[1]  # "香蕉"（第二个）
```

为什么从 0 开始？这来自内存地址的计算方式——"第 0 个"元素就在起始地址本身，不需要偏移。这个约定是程序员必须记住的，也是"序数的参照点可以任意选择"的体现。（Lua 语言是少数从 1 开始的，**待家长核对**）

### 体育比赛与序数文化

序数语言渗透在体育奖项命名中：
- 第一名 → **冠军（champion）**
- 第二名 → **亚军（runner-up）**
- 第三名 → **季军（second runner-up）**

奥运奖牌：金（第一）、银（第二）、铜（第三）——颜色对应序数，是序数概念的文化具象化。

## 🗣️ 家长讲解话术

- 孩子可能问 **"第一和一有什么不同？"** → 可以说："'一'说的是数量——一个苹果；'第一'说的是位置——排在最前面的那个。一堆有三个苹果，你可以说'有三个'（基数），也可以指着说'第二个苹果'（序数）。"

- 孩子可能问 **"为什么英文第一叫 first 不叫 one-th？"** → 可以说："因为英语里 first、second、third 这三个词历史上就是这么叫的，是特殊形式，要背下来。从第四开始就有规律了——直接在数字后面加 -th。"

- 孩子可能问 **"最后一名是什么序数？"** → 可以说："最后一名用序数说是'第 N 名'（N 是总人数）。但我们通常说'最后'，英文说 last——这也是序数的一种表达，代表最末位。"

- **共读小活动：** 把家里的玩具排成一排，家长随机报一个序数（"第四个"），孩子快速指出来。再换孩子出题，家长来猜，互动练习序数识别。
