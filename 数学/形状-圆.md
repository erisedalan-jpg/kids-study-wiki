# 形状-圆

> **一句话**：像太阳、像盘子，没有角的形状。
> **English**: A circle has no corners — it's perfectly round, like a coin or a wheel.

---

## 🧒 给 3-6 岁（共读版）

你有没有见过太阳？圆圆的，像一个大饼。你家的盘子呢？也是圆圆的！

圆有一个很特别的地方：**它没有角，没有直直的边，摸上去滑滑的，转一圈回到原点。**

**共读小活动：** 和爸爸妈妈一起在家里找圆形的东西，看谁能找到 5 个——盘子、杯子口、硬币、钟面……找到就大声说"这个是圆形！"

---

## 📚 给 6-12 岁（自读版）

### 圆是什么？

圆是平面上**到一个固定点距离都相等**的所有点的集合。这个固定点叫**圆心**（center），这个固定距离叫**半径**（radius）。

**几个重要的概念：**

- **半径（radius）**：从圆心到圆边的距离，用字母 r 表示
- **直径（diameter）**：穿过圆心的那条线，等于两倍半径：d = 2r
- **圆周（circumference）**：圆的边的总长度

**圆周率 π（pi）：** 圆的周长除以直径，永远是同一个数，约等于 **3.14**。不管圆有多大多小，这个比值都不变。这是圆非常神奇的地方。

```
周长 = π × 直径 = π × 2r ≈ 3.14 × 2r
```

**为什么车轮是圆的？** 因为圆心到圆边每个地方距离都一样，滚动的时候车轴保持同样高度，车才能平稳前进。如果车轮是方的，每转一圈就会"咚咚咚"地颠簸。

想了解更多，可以看 [[形状-椭圆]]、[[对称]]。

## 🎓 给 12+（进阶版）

### 圆的方程

在平面直角坐标系中，以原点为圆心、r 为半径的圆的方程为：

```
x² + y² = r²
```

更一般地，以点 (a, b) 为圆心：

```
(x - a)² + (y - b)² = r²
```

这个方程来自勾股定理：圆上任意点 (x, y) 到圆心的距离等于 r，而距离公式就是勾股定理。

### π 是无理数

圆周率 π ≈ 3.14159265…… 是一个无限不循环小数，也就是**无理数**（irrational number），无法写成两个整数之比。这意味着你永远无法"把 π 写完"——数学家已经计算到了数万亿位，但还没有尽头。

π 被证明是无理数是 18 世纪的成果（Johann Heinrich Lambert，**待家长核对**）；更进一步，π 还是**超越数**（transcendental number），即不是任何整系数多项式的根。

### 圆与三角函数

以原点为圆心、半径为 1 的圆叫**单位圆**（unit circle）。它是三角函数的几何基础：

- 圆上一点 (x, y) 对应一个角度 θ（从正 x 轴量起）
- cos θ = x，sin θ = y

这样，三角函数就和圆紧密联系在了一起。正弦和余弦函数的周期性，正是来自在圆上绕圈的几何意义。

### 古希腊几何学（待家长核对）

古希腊数学家欧几里得（Euclid）在《几何原本》（Elements）中严格定义并研究了圆的性质，建立了一套公理化几何体系。阿基米德（Archimedes）则通过内接多边形逼近的方法估算了 π 的值，得到 3 + 10/71 < π < 3 + 1/7 的近似范围。（**具体数值和人物细节请家长核对**）

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么车轮是圆的？"** → 可以说："因为圆心到边的距离处处相等，车轴的高度就不会变，所以滚起来很平稳。如果是方轮子，每次遇到角就会颠一下。"

- 孩子可能问 **"什么东西看起来像圆，但其实不是圆？"** → 可以说："鸡蛋！鸡蛋看起来是圆的，但其实是椭圆——一头大一头小。椭圆是圆的'亲戚'。"（引出 [[形状-椭圆]]）

- 孩子可能问 **"圆有多大？"** → 可以说："圆可以无限大，也可以无限小。只要你定好圆心和半径，圆就画出来了。宇宙里最大的天体看起来也是圆的。"

- **共读小活动：** 用绳子的一端固定（当圆心），另一端拿着笔，在纸上画一个圆，让孩子观察：不管绳子指向哪个方向，长度都一样——这就是"半径处处相等"。
