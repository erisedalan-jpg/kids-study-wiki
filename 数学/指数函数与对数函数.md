# 指数函数与对数函数

> **一句话**：指数函数 \(y = a^x\)（\(a>0, a\neq 1\)）与对数函数 \(y = \log_a x\) 互为反函数，共同刻画增长与数量级关系，是高考函数部分的核心基础。
> **English**: Exponential functions \(y = a^x\) (\(a>0, a\neq 1\)) and logarithmic functions \(y = \log_a x\) are inverses of each other, describing growth and orders of magnitude, and form a core foundation in the Gaokao function section.

## 🎓 考点精讲

**定义与性质**  
指数函数的标准形式为 \(y = a^x\)，其中底数 \(a>0\) 且 \(a \neq 1\)，定义域为 \(\mathbb{R}\)，值域为 \((0, +\infty)\)，图像恒过定点 \((0,1)\)。当 \(a>1\) 时，函数在 \(\mathbb{R}\) 上单调递增；当 \(0<a<1\) 时，单调递减。  
对数函数是它的反函数，记作 \(y = \log_a x\)，其中 \(x>0\)，定义域为 \((0, +\infty)\)，值域为 \(\mathbb{R}\)，图像恒过定点 \((1,0)\)，单调性由底数决定，且与指数函数图像关于直线 \(y=x\) 对称。

**运算与变换**  
指数幂的运算法则：\(a^{m+n}=a^m a^n\)，\((a^m)^n = a^{mn}\)，\((ab)^n = a^n b^n\)。  
对数的运算法则：\(\log_a(MN) = \log_a M + \log_a N\)，\(\log_a\frac{M}{N} = \log_a M - \log_a N\)，\(\log_a M^n = n \log_a M\)。换底公式：\(\log_a b = \frac{\log_c b}{\log_c a}\)，特别地，\(\log_a b = \frac{1}{\log_b a}\)。熟记 \(\ln x\)（以 \(e\) 为底）与 \(\lg x\)（以 \(10\) 为底）。

**高考典型考法**  
1. **比较大小**：利用指数函数或对数函数的单调性，或引入中间值（如 0, 1）比较 \(a^x\), \(\log_a x\) 等。  
2. **解指对不等式**：如 \(2^{2x+1} > 8\) 化成同底后比较指数；\(\log_{0.5}(x-1) > 2\) 注意单调性下不等号方向及定义域。  
3. **含参问题**：如求函数 \(f(x) = \log_a (x^2 - ax + 3)\) 的定义域为 \(\mathbb{R}\) 时的参数范围，需同时考虑二次函数恒正及底数条件。  
4. **图像识别与变换**：考查平移、翻折、绝对值的指对函数图像，如 \(y=|\log_2 x|\)。

## ⚠️ 高频易错点

- **忽略底数条件**：底数 \(a>0\) 且 \(a\neq 1\)，对数真数 \(>0\)，解不等式或求参数时忘记隐含定义域限制，导致增解或漏解。  
- **对数运算性质混淆**：误认为 \(\log_a (M+N) = \log_a M + \log_a N\)，或 \(\log_a M \cdot \log_a N = \log_a (MN)\)，正确的只有 \(\log_a(MN) = \log_a M + \log_a N\)。  
- **换底方向出错**：把 \(\log_a b\) 化成 \(\frac{\log_c a}{\log_c b}\) 而颠倒，尤其在比较不同底对数大小时产生错误。  
- **忽视反函数定义域值域互换**：指数函数的值域为正实数，对数函数的定义域与之一致，在求复合函数值域或解方程时，忘记真数必须为正的条件。  
- **底数含参的单调性讨论不全面**：如 \(f(x)=a^x\)，当参数 \(a\) 未给定范围时，必须分 \(a>1\) 和 \(0<a<1\) 讨论，否则单调性判断出错。
