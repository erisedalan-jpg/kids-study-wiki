## 🎓 数列裂项相消法

### 适用场景

**裂项相消法**（telescoping method）：将数列的每一项拆成两项之差，使得求和时中间项大量抵消，只剩首尾若干项。

**适用条件**：通项 $a_n$ 可以写成 $f(n) - f(n+1)$（或 $f(n+1)-f(n)$）的形式。

### 常见裂项模式

#### 模式一：分数裂项

$$\frac{1}{n(n+1)} = \frac{1}{n} - \frac{1}{n+1}$$

$$\frac{1}{n(n+k)} = \frac{1}{k}\left(\frac{1}{n} - \frac{1}{n+k}\right)$$

$$\frac{1}{(2n-1)(2n+1)} = \frac{1}{2}\left(\frac{1}{2n-1} - \frac{1}{2n+1}\right)$$

#### 模式二：根式裂项

$$\frac{1}{\sqrt{n}+\sqrt{n+1}} = \sqrt{n+1} - \sqrt{n}$$

（分子有理化：$\dfrac{\sqrt{n+1}-\sqrt{n}}{(n+1)-n} = \sqrt{n+1}-\sqrt{n}$）

#### 模式三：指数裂项（少见）

$$\frac{1}{2^n} - \frac{1}{2^{n+1}} = \frac{1}{2^{n+1}}$$

### 求和过程

以 $S_n = \displaystyle\sum_{k=1}^n \dfrac{1}{k(k+1)}$ 为例：

$$S_n = \sum_{k=1}^n \left(\frac{1}{k} - \frac{1}{k+1}\right)$$

$$= \left(\frac{1}{1}-\frac{1}{2}\right) + \left(\frac{1}{2}-\frac{1}{3}\right) + \cdots + \left(\frac{1}{n}-\frac{1}{n+1}\right)$$

$$= 1 - \frac{1}{n+1} = \frac{n}{n+1}$$

中间项两两抵消，只留首项 $\dfrac{1}{1}$ 和末项 $-\dfrac{1}{n+1}$。

### 典型例题

**例1**：$S_n = \displaystyle\sum_{k=1}^n \dfrac{1}{k(k+2)}$

**解**：$\dfrac{1}{k(k+2)} = \dfrac{1}{2}\left(\dfrac{1}{k}-\dfrac{1}{k+2}\right)$（间隔为 2 的裂项）

$$S_n = \frac{1}{2}\left[\left(1-\frac{1}{3}\right)+\left(\frac{1}{2}-\frac{1}{4}\right)+\left(\frac{1}{3}-\frac{1}{5}\right)+\cdots\right]$$

注意相消：间隔 2 的裂项，最终剩下 $1 + \dfrac{1}{2} - \dfrac{1}{n+1} - \dfrac{1}{n+2}$（待家长核对细节）：

$$S_n = \frac{1}{2}\left(\frac{3}{2} - \frac{1}{n+1} - \frac{1}{n+2}\right)$$

**例2**：$S_n = \displaystyle\sum_{k=1}^n \dfrac{1}{\sqrt{k}+\sqrt{k+1}}$

**解**：$\dfrac{1}{\sqrt{k}+\sqrt{k+1}} = \sqrt{k+1}-\sqrt{k}$

$$S_n = (\sqrt{2}-1)+(\sqrt{3}-\sqrt{2})+\cdots+(\sqrt{n+1}-\sqrt{n}) = \sqrt{n+1}-1$$

**例3**：$a_n = \dfrac{2n+1}{n^2(n+1)^2}$，求 $S_n$。

**解**：$\dfrac{2n+1}{n^2(n+1)^2} = \dfrac{(n+1)^2-n^2}{n^2(n+1)^2} = \dfrac{1}{n^2} - \dfrac{1}{(n+1)^2}$

$$S_n = 1 - \frac{1}{(n+1)^2}$$

### 步骤总结

1. 观察通项结构，猜测裂项形式（分数型/根式型）
2. 将 $a_k = f(k) - f(k+1)$ 验证
3. 展开求和，写出对消过程
4. 留下首尾项，化简得 $S_n$

### 易错点

- 裂项后**写出具体消去过程**，不能直接跳到结果
- 间隔为 2 的裂项（如 $k$ 和 $k+2$）消去后剩余 $2\sim3$ 项，需仔细核对
- 最终答案代 $n=1$ 验证：$S_1 = a_1$
