# 数列递推求通项

> **一句话**：已知数列的相邻项之间的等量关系（递推公式）及初始项，推导出数列第 $n$ 项 $a_n$ 与项数 $n$ 之间的函数关系式（通项公式）。
> **English**: The process of determining an explicit formula $a_n = f(n)$ from a recurrence relation $a_n = g(a_{n-1}, \dots)$ and initial conditions.

## 🎓 考点精讲

**定义/原理**  
给定数列 $\{a_n\}$，若存在一个方程将 $a_n$ 与其前面的若干项 $a_{n-1}, a_{n-2}, \dots$ 联系起来，则称该方程为数列的递推公式。例如 $a_{n} = pa_{n-1} + q$（一阶线性递推），$a_{n+1} = a_n + f(n)$（可累加型），$\frac{a_{n+1}}{a_n} = f(n)$（可累乘型），以及 $a_{n+2} = pa_{n+1} + qa_n$（二阶常系数齐次线性递推）。目标是由递推公式和首项（有时需要前两项）求出 $a_n$ 不依赖于其他项的表达式。

**关键方法**  
1. **累加法**：当递推形如 $a_n - a_{n-1} = f(n)$ 时，写出 $n-1$ 个等式相加，得 $a_n = a_1 + \sum_{k=2}^{n} f(k)$。  
2. **累乘法**：当递推形如 $\frac{a_n}{a_{n-1}} = f(n)$ 时，连乘 $n-1$ 个等式，得 $a_n = a_1 \cdot \prod_{k=2}^{n} f(k)$。  
3. **构造等比数列法**(核心)：对于 $a_n = pa_{n-1} + q$（$p \neq 1$），设 $a_n + \lambda = p(a_{n-1} + \lambda)$，解得 $\lambda = \frac{q}{p-1}$，则 $\{a_n + \lambda\}$ 是以 $p$ 为公比的等比数列，从而 $a_n = (a_1 + \frac{q}{p-1})p^{n-1} - \frac{q}{p-1}$。对于 $a_n = pa_{n-1} + f(n)$ 常通过待定系数或同除以指数/幂项转化为等差或等比。  
4. **特征根法**：对二阶线性递推 $a_{n+2} = p a_{n+1} + q a_n$，写出特征方程 $x^2 - px - q = 0$，若有两不等实根 $\alpha, \beta$，则 $a_n = A\alpha^{n-1} + B\beta^{n-1}$；等根 $\alpha$ 则 $a_n = (A + Bn)\alpha^{n-1}$，由初始项定 $A,B$。

**典型应用**  
高考中常见设问：已知 $a_1 = 1$，$a_{n+1} = 2a_n + 3$，求 $a_n$（构造等比）；已知 $a_1 = 1$，$a_n = a_{n-1} + 2n$，求 $a_n$（累加，需用等差数列求和）；已知 $a_1 = 2$，$a_{n+1} = \frac{n}{n+1} a_n$，求 $a_n$（累乘后约分得 $a_n = \frac{2}{n}$）。这些方法常在解答题第(1)问出现，亦在选择填空题中以快速求特定项的形式考查。

## ⚠️ 高频易错点

- **混淆累加与累乘的适用条件**：$a_n - a_{n-1} = f(n)$ 用累加，$\frac{a_n}{a_{n-1}} = f(n)$ 用累乘。若将形如 $a_{n+1} - a_n = n$ 错用累乘会导致全错。  
- **构造常数 $\lambda$ 时符号与系数处理错误**：如对于 $a_n = 3a_{n-1} + 2$，误设 $a_n + \lambda = 3(a_{n-1} + 2\lambda)$，正确的应为 $a_n + \lambda = 3(a_{n-1} + \frac{\lambda}{3})$，需通过待定系数保证恒等。  
- **忽略 $n$ 的范围与初始项的标注**：累加/累乘后得到的通项公式通常只在 $n \geq 2$ 下导出，必须验证 $n=1$ 是否一致，否则可能遗漏分段定义。  
- **二阶特征根法错用指数**：特征根为 $\alpha, \beta$ 时，标准形式为 $a_n = A \alpha^{n-1} + B \beta^{n-1}$（以 $a_1, a_2$ 定），但学生易误写成 $a_n = A \alpha^{n} + B \beta^{n}$ 或数字错位。
