# 新定义问题

> **一句话**：新定义问题是指在题目中临时给出一个此前未学过的数学概念、运算规则或性质，要求考生在有限时间内理解该定义，并运用其解决后续问题的题型。
> **English**: A "new definition" problem presents a novel mathematical concept, operation, or property defined solely within the problem, requiring the test-taker to comprehend and apply it on the spot.

## 🎓 考点精讲

**定义/原理**
新定义问题本身不是单一知识模块，而是**考查数学抽象与即时学习能力**的载体。其核心结构是：题干先以严格语言给出一个新概念的定义（如“对于数列 $\{a_n\}$，若满足 $a_{n+1} = a_n^2 + c$，则称其为‘平方递推数列’，其中 $c$ 为常数”），或者定义一个新的运算（如“记 $a \oplus b = \sqrt{a^2 + b^2}$”）。核心原理在于，所有后续问题的解答必须严格**以该定义为唯一的逻辑起点和判定依据**，不能擅自套用旧有类似概念的性质。

**关键方法**
1.  **逐字翻译法**：将新定义中的文字、符号拆解，转化为等价的条件语句或表达式。例如看到“数列 $\{b_n\}$ 是‘和谐数列’当且仅当 $b_n + b_{n+2} = 2b_{n+1}$”，应立即将其翻译为 $b_{n+2} - b_{n+1} = b_{n+1} - b_n$。
2.  **映射与构造法**：将新定义的对像映射到已知的数学结构上。如上例翻译后，即可识别 $\{b_n\}$ 正是等差数列，从而可以调用等差数列的全部已知性质（求通项、求和等）。对于复杂的集合新定义，常通过列举前几项、构造具体例子，来探索其内在规律。
3.  **性质探究法**：从定义出发，推导该新概念满足的充要条件、运算律（如交换律、结合律）或不变量，这是解决证明类小问的关键。

**典型应用**
高考中常以**压轴题**（选择、填空或解答题）出现，设问角度为三阶梯：
1.  **判断与计算**：直接给出具体元素（一个数、一个数列、一个集合），判断它是否满足新定义，并据此计算一个简单结果。
2.  **性质推导**：证明新定义概念所具有的某个深层性质，如“证明所有‘平方递推数列’$\{a_n\}$ 中，若 $c > \frac{1}{4}$，则 $a_n$ 无界。” 这需要综合使用不等式、函数单调性等知识。
3.  **存在性与构造**：探究满足某附加条件的对像是否存在，若存在需将其构造出来。例如“是否存在一个各项均为整数的‘和谐数列’，同时又是等比数列？若存在，求出所有这样的数列。” 此问通常需转化为方程整数解问题。

## ⚠️ 高频易错点

- **擅自调用旧性质**：对“新定义函数 $f(x)$”直接求导，而未考虑题设是否定义导数或其是否满足可导条件。必须严格立足定义本身进行推理。
- **对定义的理解停留在字面，忽略隐含等价条件**：例如定义新运算 $A * B = A \cap \complement_U B$，若仅记住“一种运算”而未理解其本质是集合的差集运算，在遇到 $(A*B)*C$ 等复杂嵌套运算时极易出错。
- **在“是否存在”类问题中，逻辑不严谨**：找到了一个例子就直接断言“所有情况都是这样”，或者找不到例子就断言“不存在”，而缺乏严格的穷举证明或反证法。
- **忽略定义域与成立范围**：新定义往往附带特定条件（如“对于 $a > 0, b > 0$，定义 $a \otimes b = \ln(a+b)$”），在后续使用 $a \otimes b$ 时，忘记检验参与运算的元素是否满足 $a>0, b>0$ 的前提。
