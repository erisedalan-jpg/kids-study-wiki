## 🎓 方差与标准差（高中）

### 离散程度

集中趋势（平均数等）描述数据的"中心位置"，而**离散程度**描述数据的"分散程度"。

### 方差（Variance）

$$s^2 = \frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2$$

方差是每个数据与平均数**偏差的平方**的平均值。

- **方差越大**：数据越分散
- **方差越小**：数据越集中

### 标准差（Standard Deviation）

$$s = \sqrt{s^2} = \sqrt{\frac{1}{n}\sum_{i=1}^n (x_i - \bar{x})^2}$$

标准差与原数据**单位相同**，比方差更直观。

### 计算步骤

1. 求平均数 $\bar{x}$
2. 求每个数据与平均数的偏差 $x_i - \bar{x}$
3. 求偏差的平方 $(x_i - \bar{x})^2$
4. 求平方的平均值（方差 $s^2$）
5. 开方得标准差 $s$

### 方差计算简化公式

$$s^2 = \overline{x^2} - \bar{x}^2 = \frac{1}{n}\sum x_i^2 - \bar{x}^2$$

（各数据平方的平均 $-$ 平均数的平方，计算更快）

### 典型例题

**例1**：数据 2, 4, 4, 4, 5, 5, 7, 9，求均值、方差、标准差。

**解**：$\bar{x} = \dfrac{40}{8} = 5$

$s^2 = \dfrac{(2-5)^2+(4-5)^2\times3+(5-5)^2\times2+(7-5)^2+(9-5)^2}{8}$

$= \dfrac{9+3+0+4+16}{8} = \dfrac{32}{8} = 4$

$s = 2$

**例2**：甲乙两人射击各 10 发，平均环数相同（8 环），甲方差 1.2，乙方差 2.8，谁发挥更稳定？

**解**：甲方差更小，成绩更稳定，应选甲参加比赛。

**例3**：若每个数据加一个常数 $c$，方差是否改变？

**解**：$s^2$ 不变（偏差 $(x_i + c) - (\bar{x}+c) = x_i - \bar{x}$ 不变）

### 与初中衔接

初中已学方差的基本概念，高中重点在于：
- 灵活使用简化公式
- 利用频率分布直方图估计方差
- 理解标准差在实际问题中的意义

### 易错点

- 方差公式中是"偏差的**平方**"的平均，不是偏差的平均（偏差之和为 0）
- 方差和标准差都是**非负数**
- 每个数据加减同一常数，方差不变；每个数据乘以常数 $k$，方差乘以 $k^2$
