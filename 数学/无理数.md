## 课本定义

**无理数**是无限不循环小数。即不能表示为两个整数之比（p/q，q≠0）的实数。无理数与有理数一起构成实数。

---

## 📚 6-12 岁回溯版

### "永远写不完的小数"

我们认识的数：
- **整数**：1, 2, 3, -5, 0（精确，有限）
- **分数/有限小数**：1/2 = 0.5（精确，有限）
- **循环小数**：1/3 = 0.333...（有规律地重复）

但有些数，写成小数后，**永远写不完，而且没有循环规律**！

**无理数的例子：**

| 无理数 | 近似值 | 为什么是无理数 |
|--------|--------|--------------|
| √2 | 1.41421356... | 非完全平方数的平方根 |
| √3 | 1.73205080... | 同上 |
| π（圆周率） | 3.14159265... | 圆的周长/直径，无限不循环 |
| ∛2 | 1.25992105... | 非完全立方数的立方根 |
| e（自然常数） | 2.71828182... | 数学常数，无限不循环 |

**怎么判断：**
- 能写成分数 p/q → 有理数
- 不能写成分数，小数无限不循环 → 无理数

**√2 的故事：**
古希腊数学家毕达哥拉斯（Pythagoras）认为所有数都是整数或分数。他的学生发现了 √2 不是分数，这让当时的数学家非常震惊，传说这个学生因此被扔下海（历史上有争议）。

## 🎓 12+ 进阶版

### 严格定义

**无理数**（irrational number）：不能表示为 p/q（p, q 为整数，q ≠ 0）形式的实数。等价定义：无限不循环小数。

**数系层次：**
```
实数
├── 有理数
│   ├── 整数（有限小数或循环 0.000...）
│   └── 分数（有限小数或无限循环小数）
└── 无理数（无限不循环小数）
```

### 常见无理数

| 类型 | 例子 | 说明 |
|------|------|------|
| 非完全平方数的平方根 | √2, √3, √5, √6... | 最常见 |
| 非完全立方数的立方根 | ∛2, ∛4... | |
| 数学常数 | π, e | 经过证明 |
| 其他 | log₂3, sin15°... | 高中/大学 |

### √2 是无理数的证明（反证法）

假设 √2 是有理数，可写成 √2 = p/q（p, q 互质，q≠0）

则 2 = p²/q²，即 p² = 2q²

∴ p² 是偶数 → p 是偶数，设 p = 2k

代入：(2k)² = 2q²，4k² = 2q²，q² = 2k²

∴ q² 是偶数 → q 是偶数

但 p, q 同为偶数，与"p, q 互质"矛盾！

∴ √2 不是有理数，即 √2 是无理数。∎

### 无理数的运算

无理数参与运算，结果不一定是无理数：
- √2 × √2 = 2（有理数）
- √2 + √3 ≈ 3.146...（无理数）
- √2 × √3 = √6（无理数）
- π - π = 0（有理数）

### 典型例题

**例题1：** 判断下列各数是有理数还是无理数：
3.14，π，√4，√5，0.101001000...（小数点后1,2,3个0交替增加）

解：
- 3.14 = 314/100，有理数
- π = 3.14159...（无限不循环），无理数
- √4 = 2，有理数
- √5 = 2.23606...（无限不循环），无理数
- 0.101001000...（规律性地不循环），无理数

**例题2：** √12 能化简成含整数系数的形式吗？

解：√12 = √(4×3) = √4 × √3 = 2√3

2√3 仍是无理数（√3 是无理数，2√3 = 无理数）

### 易错点

1. **√4 = 2 不是无理数**（完全平方数的平方根是整数）
2. **3.14 不是 π**：3.14 是有理数，π 是无理数
3. **无限小数不等于无理数**：无限循环小数（如0.333...）是有理数

### 知识联系

```
无理数
├── 发现 → 平方根/立方根（非完全幂次）
├── 组成实数 → 有理数 ∪ 无理数 = 实数
├── 证明方法 → 反证法
└── 近似计算 → 保留几位小数（工程计算）
```
