# 极坐标方程化直角坐标方程

> **一句话**：利用极坐标与直角坐标的转化公式，将只含 $\rho,\theta$ 的方程变为只含 $x,y$ 的方程。
> **English**: The process of rewriting an equation expressed in polar coordinates $(\rho, \theta)$ into an equivalent equation in Cartesian coordinates $(x, y)$ using the conversion formulas.

## 🎓 考点精讲

**核心原理**  
极坐标 $(\rho,\theta)$ 与直角坐标 $(x,y)$ 的转化关系为：  

$$
\begin{cases}
x = \rho\cos\theta \\[4pt]
y = \rho\sin\theta
\end{cases}
\qquad\text{或}\qquad
\rho^2 = x^2 + y^2,\quad \tan\theta = \frac{y}{x}\;(x \neq 0)
$$

对 $\theta$ 的处理常需结合三角恒等式消去 $\theta$，或把 $\rho\cos\theta,\rho\sin\theta$ 整体替换为 $x,y$。

**关键方法**  
1. **直接代换型**：若方程已出现 $\rho\cos\theta$ 或 $\rho\sin\theta$，直接写成 $x,y$。  
   *例*：$\rho = 4\cos\theta$ → 两边同乘 $\rho$，得 $\rho^2 = 4\rho\cos\theta$ → $x^2+y^2=4x$，整理为 $(x-2)^2+y^2=4$。  
2. **含 $\rho^2$ 或 $\tan\theta$ 型**：用 $\rho^2=x^2+y^2$ 或 $\tan\theta=\frac{y}{x}$ 代换，并需检验 $x=0$ 的情形是否满足原方程。  
3. **形如 $\rho = a\sin\theta$ 或 $\rho = a\cos\theta$**：常表示圆，可快速写出直角坐标方程。  

**典型高考考查方式**  
- **给出极坐标方程，求直角坐标方程**：作为坐标系与参数方程选做题的第一步，得分点直接靠转化。  
- **与参数方程结合**：先化极坐标方程为直角坐标方程，再与参数方程联立求交点或距离。  
- **探究曲线形状**：通过直角坐标方程识别圆、椭圆、直线等，并写出关键几何要素。

## ⚠️ 高频易错点

- **漏乘 $\rho$ 导致丢解**：对方程 $\rho = f(\theta)$，若 $f(\theta)$ 不含 $\rho$，两边同乘 $\rho$ 引入 $0$ 解（极点）时，必须检验极点是否在曲线上。  
- **$\tan\theta$ 使用不当**：由 $\tan\theta = \frac{y}{x}$ 只能得到射线方向，不能区分点所在的象限，必须结合原方程或 $\cos\theta,\sin\theta$ 的符号判定。  
- **未处理定义域**：极坐标方程中常隐含 $\theta$ 的范围（如 $0\le\theta<\pi$），化为直角坐标方程后可能扩大曲线范围，需注明范围或用图像说明。  
- **混淆坐标变换与方程变换**：不能直接把 $(\rho,\theta)$ 代入 $x^2+y^2=\rho^2$ 去验证中间步骤，而是要整体消去 $\rho,\theta$。
