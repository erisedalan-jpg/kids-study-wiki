## 🎓 概率的基本性质

### 概率的公理化定义

**柯尔莫哥洛夫公理**（现代概率论基础）：

设 $\Omega$ 为样本空间，对每个事件 $A$，概率 $P(A)$ 满足：

1. **非负性**：$P(A) \geq 0$
2. **规范性**：$P(\Omega) = 1$
3. **可加性**：若 $A_1, A_2$ 互斥，则 $P(A_1 \cup A_2) = P(A_1) + P(A_2)$

### 由公理推出的基本性质

**性质1**：$P(\emptyset) = 0$（不可能事件概率为 0）

**性质2**：$0 \leq P(A) \leq 1$

**性质3（对立事件）**：$P(\bar{A}) = 1 - P(A)$

**性质4（减法公式）**：若 $A \subset B$，则 $P(B - A) = P(B) - P(A)$，$P(A) \leq P(B)$

**性质5（加法公式）**：$P(A \cup B) = P(A) + P(B) - P(AB)$

**性质6（全集分解）**：若 $A_1, A_2, \ldots, A_n$ 两两互斥且 $A_1 \cup A_2 \cup \cdots \cup A_n = \Omega$，则

$$P(A_1) + P(A_2) + \cdots + P(A_n) = 1$$

这称为**完备事件组**（全集的划分）。

### 概率的计算方法总结

| 情形 | 公式 |
|------|------|
| 对立事件 | $P(\bar{A}) = 1 - P(A)$ |
| 互斥事件的和 | $P(A \cup B) = P(A) + P(B)$ |
| 一般事件的和 | $P(A \cup B) = P(A) + P(B) - P(AB)$ |
| $A$ 中不含 $B$ | $P(A - B) = P(A) - P(AB)$ |
| 独立事件的积 | $P(AB) = P(A) \cdot P(B)$ |

### 典型例题

**例1**：$P(A) = 0.5$，$P(B) = 0.4$，$P(\bar{A}\bar{B}) = 0.2$，求 $P(A \cup B)$。

**解**：$P(A \cup B) = 1 - P(\overline{A \cup B}) = 1 - P(\bar{A} \cap \bar{B}) = 1 - 0.2 = 0.8$

**例2**：三个互斥事件 $A, B, C$，$P(A) = 0.2$，$P(B) = 0.3$，$P(C) = ?$，若 $A \cup B \cup C = \Omega$。

**解**：$P(C) = 1 - 0.2 - 0.3 = 0.5$

**例3**：证明 $P(A \cup B) \leq P(A) + P(B)$。

**证**：由 $P(A \cup B) = P(A) + P(B) - P(AB) \leq P(A) + P(B)$（因 $P(AB) \geq 0$）。

### 条件概率预备（见词条[[条件概率初步]]）

$$P(B | A) = \frac{P(AB)}{P(A)} \quad (P(A) > 0)$$

### 易错点

- $P(A) = 0$ 不代表 $A$ 是不可能事件（在连续分布中常见）
- $P(A) + P(B) = 1$ 不代表 $A$ 与 $B$ 对立（可能只是数值上的巧合）
- 互斥与独立是完全不同的概念（见[[事件的独立性]]）
