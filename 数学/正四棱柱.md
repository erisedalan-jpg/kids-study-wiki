# 正四棱柱

> **一句话**：底面为正方形且侧棱垂直于底面的四棱柱。
> **English**: A right prism whose base is a square.

## 🎓 考点精讲

**定义/原理**

正四棱柱必须同时满足两个条件：① 底面是正方形；② 侧棱与底面垂直（即为直棱柱）。设底面正方形边长为 $a$，高（侧棱长）为 $h$，则：

- 体积：$V = a^2 h$
- 侧面积：$S_{\text{侧}} = 4ah$
- 全面积：$S_{\text{表}} = 2a^2 + 4ah$
- 体对角线长：$l = \sqrt{2a^2 + h^2}$（待家长核对）

若 $a = h$，则为**正方体**，是正四棱柱的特殊情况。正四棱柱的侧面是四个全等的矩形，对角面是矩形。

**关键方法**

1. **证明某四棱柱是正四棱柱**：需依次证明底面为正方形、侧棱与底面垂直。常用向量法：以底面顶点为原点建系，证明底面两边垂直且模长相等，再证侧棱与底面两边点积为零。
2. **求外接球半径**：正四棱柱外接球半径 $R = \frac{1}{2}\sqrt{2a^2 + h^2}$（即体对角线的一半），适用于体对角线中点即为外接球球心。
3. **截面问题**：常考过体对角线的截面形状，或平行于底面的截面（仍为正方形），需结合相似比例计算截面边长。

**典型应用**

高考中，正四棱柱常见于以下设问：
- **体积/表面积计算**：直接代公式，但常与三视图结合，需先还原为底面正方形及高的数据。
- **外接球、内切球**：正四棱柱一般无内切球（仅当 $a = h = 2r$ 时才有可能），外接球考点频次极高，设问方式如“正四棱柱的各顶点都在同一球面上，已知…求表面积”。
- **线面位置关系证明**：如求证体对角线与侧面的交点位置，或侧面内某线段与底面的夹角，常通过建系求法向量解决。

## ⚠️ 高频易错点

- **混淆正四棱柱与正方体**：正方体是正四棱柱的特例（底面正方形且高等于底面边长），但正四棱柱的高 $h$ 与底面边长 $a$ 可不相等，侧面不一定是正方形。
- **体对角线公式记混**：正四棱柱体对角线长为 $\sqrt{a^2 + a^2 + h^2}$，而非长方体的 $\sqrt{a^2 + b^2 + h^2}$，易漏掉底面两等边造成的系数 2。
- **外接球半径公式错用**：误将体对角线长当作直径的 $2R$，实际 $2R = l = \sqrt{2a^2 + h^2}$，求 $R$ 时需除以 2。
- **建系时坐标设错**：底面以正方形顶点为原点时，相邻两边坐标表示为 $(a,0,0)$ 与 $(0,a,0)$，若底面非正方形则正四棱柱不成立，这一步需先判定再建系。
