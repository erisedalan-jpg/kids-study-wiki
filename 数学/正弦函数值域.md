# 正弦函数值域

> **一句话**：正弦函数 $y = \sin x$ 的值域是 $[-1, 1]$，即函数值的取值范围为闭区间 $[-1, 1]$。
> **English**: The range of the sine function is the closed interval $[-1, 1]$.

## 🎓 考点精讲

**定义/原理**  
对于任意实数 $x$，正弦函数 $\sin x$ 的值由单位圆上对应角的终边与单位圆交点的纵坐标给出。由于单位圆上任意一点的纵坐标最小为 $-1$，最大为 $1$，因此 $-1 \le \sin x \le 1$，即值域为 $[-1, 1]$。该性质也称为正弦函数的有界性。  
对于复合型正弦函数 $y = A\sin(\omega x + \varphi) + b$（其中 $A \neq 0$），由 $\sin(\omega x + \varphi) \in [-1, 1]$ 可得值域为 $[b - |A|,\; b + |A|]$。当 $A > 0$ 时直接按有界性伸缩平移，$A < 0$ 时需注意最值反转。

**关键方法**  
1. **基本型**：直接利用有界性写出值域。  
2. **换元法**：令 $t = \omega x + \varphi$，转化为 $y = A\sin t + b$ 在 $t \in \mathbb{R}$ 上的值域，再视定义域限制进一步处理。  
3. **二次型与复合型**：如 $y = a\sin^2 x + b\sin x + c$，设 $u = \sin x \in [-1,1]$，化为二次函数在闭区间 $[-1,1]$ 上的值域问题，注意对称轴位置和端点值。  
4. **有限定义域**：若给出 $x \in [\alpha, \beta]$，需结合正弦函数的图像与单调性，确定 $\sin x$ 在区间上的最大值和最小值。

**典型应用**  
- 高考常见设问：“求函数 $f(x) = 2\sin(2x - \frac{\pi}{3}) + 1$ 的值域” 或 “求 $y = \cos 2x + \sin x$ 的最值” 等。  
- 结合三角形：在 $\triangle ABC$ 中求角的正弦值范围，如已知一角为锐角，求 $\sin A$ 的取值范围，通常需结合 $A \in (0, \pi)$ 及三角形内角和限制。  
- 含参数问题：已知值域为 $[m, n]$，反求参数 $A$ 或 $b$，关键步骤是建立最值方程组。

## ⚠️ 高频易错点

- 忽略 $A$ 的符号对最值对应的影响：$y = -2\sin x + 1$ 的最大值出现在 $\sin x = -1$ 时，而非 $\sin x = 1$。  
- 换元后未注意新元的取值范围：如令 $t = \sin x$，却将 $t$ 的范围误写成 $\mathbb{R}$，而实际上是 $[-1, 1]$。  
- 定义域限制时未结合正弦图像分段讨论单调性，直接代入区间端点求最值，导致漏掉极值点（如区间内包含对称轴）。  
- 将值域写成开区间：正弦函数在整个实数域上的值域是闭区间 $[-1, 1]$，当定义域为开区间且不包含最值点时，值域可能为开区间，需仔细判断。
