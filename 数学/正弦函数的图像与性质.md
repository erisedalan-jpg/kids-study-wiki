# 正弦函数的图像与性质

> **一句话**：正弦函数 $y = \sin x$ 是定义在实数集 $\mathbb{R}$ 上的周期函数，其图像为连续光滑的正弦波形，中心对称且轴对称。
> **English**: The sine function $y = \sin x$ is a periodic function defined on the set of real numbers, whose graph is a continuous, smooth sinusoidal wave with central and axial symmetries.

## 🎓 考点精讲

**定义与核心性质**

正弦函数 $y = \sin x$ 将任意实数 $x$（通常表示角的弧度）映射到 $[-1, 1]$ 中的唯一实数值。其核心性质构成一个逻辑闭环：

| 性质 | 表达式 / 取值范围 | 几何意义 |
|------|-------------------|----------|
| 定义域 | $\mathbb{R}$ | 任意角都有正弦值 |
| 值域 | $[-1, 1]$ | 波动上下限 |
| 最小正周期 | $2\pi$ | 每 $2\pi$ 重复一次 |
| 奇偶性 | $\sin(-x) = -\sin x$（奇函数） | 图像关于原点对称 |
| 对称轴 | $x = k\pi + \frac{\pi}{2},\,k \in \mathbb{Z}$ | 每条对称轴过波峰或波谷 |
| 对称中心 | $(k\pi, 0),\,k \in \mathbb{Z}$ | 图像绕该点旋转 180° 重合 |
| 单调区间 | 增：$[-\frac{\pi}{2} + 2k\pi, \frac{\pi}{2} + 2k\pi]$；减：$[\frac{\pi}{2} + 2k\pi, \frac{3\pi}{2} + 2k\pi]$ | 以 $\frac{\pi}{2}$ 为界交替 |

图像（正弦曲线 $y = \sin x$）的绘制通常用五点法：选取一个周期内的五个关键点——$(0,0)$、$(\frac{\pi}{2},1)$、$(\pi,0)$、$(\frac{3\pi}{2},-1)$、$(2\pi,0)$——描点连线即得一个完整波形。

**关键方法：正弦型函数问题**

高考主要考查 $y = A\sin(\omega x + \varphi)$ 型函数，核心方法为**整体换元**与**五点法反推**：

1. **整体换元法**：令 $t = \omega x + \varphi$，将问题化为 $y = A\sin t$ 的标准问题。解单调性、对称轴、对称中心时，直接把 $t$ 代入相应的通式（如对称轴 $t = k\pi + \frac{\pi}{2}$），再解出 $x$。
2. **五点法求解析式**：已知图像求 $A, \omega, \varphi$ 时，$A$ 由最高点最低点纵坐标确定（$A = \frac{y_{\max} - y_{\min}}{2}$）；$\omega$ 由周期 $T = \frac{2\pi}{|\omega|}$ 反推；$\varphi$ 通常代入一个已知点坐标求解，注意多解时需结合单调性取舍。

**典型设问**

- 给定 $y = A\sin(\omega x + \varphi)$ 的部分图像，求函数解析式。
- 讨论 $y = A\sin(\omega x + \varphi)$ 在指定区间上的单调性、最值或对称性。
- 将正弦曲线进行平移、伸缩变换后，确定新函数的解析式或性质。

## ⚠️ 高频易错点

- **忽视 $\omega$ 的正负**：求单调区间或对称轴时直接套用 $y = \sin x$ 的公式，忘记当 $\omega < 0$ 时需先利用奇偶性把函数化为 $y = -A\sin(|\omega|x - \varphi)$ 或 $y = A\sin(|\omega|x - \varphi + \pi)$ 再处理。
- **对称轴与对称中心混淆**：对称轴是 $x = \dots$（直线），在图像上过极值点；对称中心是 $(\dots, 0)$（点），在图像上为零点。两者代入整体角的通式不同——$t = k\pi + \frac{\pi}{2}$ 对应对称轴，$t = k\pi$ 对应对称中心。
- **五点法求 $\varphi$ 时选点失误**：代入的点必须是严格对应"五点"之一（上升零点、峰值、下降零点、谷值、上升零点），若使用了曲线上的其他点会导致 $\varphi$ 解错或出现多解无法判断。
- **周期公式记错**：$T = \frac{2\pi}{|\omega|}$，忽略分母的绝对值号，或与正切函数的周期 $T = \frac{\pi}{|\omega|}$ 混淆。
