# 正弦定理与余弦定理

> **一句话**：正弦定理与余弦定理是揭示三角形边角定量关系的核心定理，正弦定理建立边与对角正弦的比例关系，余弦定理建立边与角的平方关系，是解三角形的基本工具。
> **English**: The Law of Sines and the Law of Cosines are fundamental theorems that establish quantitative relationships between sides and angles in any triangle, forming the basis for solving triangles.

## 🎓 考点精讲

**定义/原理**  
在一个三角形 $ABC$ 中，角 $A,B,C$ 所对的边分别为 $a,b,c$，外接圆半径为 $R$。  
- **正弦定理**：$\displaystyle \frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R$。  
- **余弦定理**：$a^2 = b^2 + c^2 - 2bc\cos A$，$b^2 = a^2 + c^2 - 2ac\cos B$，$c^2 = a^2 + b^2 - 2ab\cos C$；其推论 $\cos A = \dfrac{b^2 + c^2 - a^2}{2bc}$ 常用于求角。

**关键方法**  
- **条件与定理选择**：已知两角一边（ASA或AAS），用正弦定理求其余边；已知两边及其夹角（SAS）或三边（SSS），用余弦定理求边或角。已知两边及其中一边的对角（SSA）时，先用正弦定理求角，再结合三角形内角和判定多解或一解。  
- **解的个数判定**：对 SSA 情况，按“大边对大角”及角的正弦值范围（$\sin A \le 1$）检验，避免增解或漏解。  
- **与面积结合**：面积公式 $S = \frac12 ab\sin C$ 常与定理联动，用于求高、判定形状或最值问题。

**典型应用**  
高考常见题型：  
1. 已知部分边角，求未知边或角。例如：在 $\triangle ABC$ 中，$a=3$，$b=5$，$\cos C = \frac13$，求 $c$ 及 $\sin A$。解法：由余弦定理 $c^2=9+25-30\cdot\frac13=24$，得 $c=2\sqrt{6}$；再由正弦定理 $\frac{a}{\sin A}=\frac{c}{\sin C}$，结合 $\sin C=\sqrt{1-\cos^2 C}=\frac{2\sqrt{2}}{3}$，得 $\sin A=\frac{a\sin C}{c}=\frac{\sqrt{3}}{3}$。  
2. 解三角形与恒等变换综合题，如 “$2b\cos B = a\cos C + c\cos A$”，需边角互化，利用正弦定理变为 $2\sin B\cos B = \sin A\cos C + \sin C\cos A = \sin(A+C)=\sin B$，解得 $\cos B=\frac12$，$B=\frac{\pi}{3}$。  
3. 实际应用题（如测量距离、高度），需建立三角形模型并选用定理求解。

## ⚠️ 高频易错点

- 使用正弦定理求角时，忽略 $\sin\theta$ 在 $(0,\pi)$ 内有两解的可能，没有结合“大边对大角”筛选，导致多解或错解。  
- 余弦定理公式混淆：计算 $\cos A$ 时分子误写为 $a^2 - b^2 - c^2$，或者求边时代错符号。  
- 将正弦定理中的 $2R$ 直接带入化简时，忘记 $2R$ 在下式中可约分，却错误地消去某些项。  
- 在涉及三角恒等变换的边角互化中，对 $a=2R\sin A$ 及其等价形式不熟练，导致变形方向错误。
