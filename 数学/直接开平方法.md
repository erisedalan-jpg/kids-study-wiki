# 直接开平方法

> **一句话**：对形如 $x^2 = k$ 或 $(ax+b)^2 = k$ 的方程，两边直接开平方来求解，叫做直接开平方法。
> **English**: The square root method solves equations of the form $x^2 = k$ or $(ax+b)^2 = k$ by taking the square root of both sides.

---

## 📚 给 6-12 岁（自读版）

### 核心思路

如果方程左边是"某个数的平方"，右边是一个非负数，那就直接对两边开平方！

**记住**：$x^2 = k$（$k \geq 0$）→ $x = \pm\sqrt{k}$

为什么有 $\pm$？因为正数和负数的平方都等于正数，例如 $3^2 = 9$，$(-3)^2 = 9$，所以 $x^2 = 9$ 有两个解：$x = 3$ 或 $x = -3$。

### 例题演示

**例 1**：解方程 $x^2 = 16$

$$x = \pm\sqrt{16} = \pm 4$$

即 $x_1 = 4,\ x_2 = -4$

**例 2**：解方程 $(x-2)^2 = 9$

两边开平方：$x - 2 = \pm 3$

- $x - 2 = 3 \Rightarrow x = 5$
- $x - 2 = -3 \Rightarrow x = -1$

**例 3**：解方程 $x^2 = -4$

因为 $-4 < 0$，负数没有实数平方根，所以**方程无实数解**。

### 什么时候用这个方法？

当方程能整理成 $(\ )^2 = 数字$ 的形式时，用直接开平方法最简单！

---

## 🎓 给 12+（进阶版）

### 适用形式

**基本型**：$x^2 = k$
$$x = \pm\sqrt{k} \quad (k > 0 \text{ 时两个根，} k = 0 \text{ 时唯一根 } x=0,\ k < 0 \text{ 时无实数根})$$

**推广型**：$(ax + b)^2 = k$
$$ax + b = \pm\sqrt{k} \Rightarrow x = \frac{-b \pm \sqrt{k}}{a} \quad (a \neq 0, k \geq 0)$$

### 步骤

1. 将方程整理为 $(\ )^2 = k$ 的形式
2. 若 $k < 0$，直接判断无实数解
3. 若 $k \geq 0$，两边开平方，注意添加 $\pm$
4. 解出 $x$ 的值

### 典型例题

**例 1**：解 $3(x+1)^2 = 12$

$$
(x+1)^2 = 4 \Rightarrow x+1 = \pm 2
$$
$$
x_1 = 1,\quad x_2 = -3
$$

**例 2**：解 $(2x-1)^2 = 5$

$$
2x - 1 = \pm\sqrt{5}
$$
$$
x_1 = \frac{1+\sqrt{5}}{2},\quad x_2 = \frac{1-\sqrt{5}}{2}
$$

**例 3**（综合）：解 $(x+3)^2 - 2(x+3) + 1 = 0$

注意左边可因式分解：$[(x+3)-1]^2 = 0 \Rightarrow (x+2)^2 = 0 \Rightarrow x = -2$（唯一解，二重根）

### 与其他方法的联系

直接开平方法是[[配方法]]的基础——配方法的目标就是将方程整理为 $(x+m)^2 = n$ 的形式，然后用直接开平方法求解。

| 判别式 $\Delta = b^2 - 4ac$ | 根的情况 |
|--------------------------|---------|
| $\Delta > 0$ | 两个不相等实数根 |
| $\Delta = 0$ | 两个相等实数根（二重根） |
| $\Delta < 0$ | 无实数根 |

### 易错点

- ❌ 忘记写 $\pm$，只写正值（最常见错误！）
- ❌ 对 $k < 0$ 的方程强行开平方（应判断无实数解）
- ❌ 开方后不继续解方程，停在 $x+1 = \pm 2$ 就结束
