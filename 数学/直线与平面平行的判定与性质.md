# 直线与平面平行的判定与性质

> **一句话**：若平面外的一条直线平行于平面内的一条直线，则该直线与平面平行；反之，若直线与平面平行，则过该直线的平面与已知平面的交线必与该直线平行。
> **English**: A line is parallel to a plane if and only if it is parallel to some line in the plane; conversely, if a line is parallel to a plane, then any plane containing that line intersects the given plane in a line parallel to the given line.

## 🎓 考点精讲

- **定义与判定定理**：一条直线与一个平面没有公共点，则称该直线与此平面平行。  
  **判定定理**：如果平面外的一条直线与此平面内的一条直线平行，那么该直线与此平面平行。  
  符号表示为：若 $l \not\subset \alpha$，$m \subset \alpha$，且 $l \parallel m$，则 $l \parallel \alpha$。  
  其中“平面外”是必要条件：直线不能包含于该平面，否则即使与平面内无数条直线平行，也是线在面内，而非平行。

- **性质定理**：如果一条直线与一个平面平行，那么经过这条直线的任意一个平面与已知平面的交线，与该直线平行。  
  符号表示：若 $l \parallel \alpha$，$l \subset \beta$，且 $\alpha \cap \beta = m$，则 $l \parallel m$。  
  该定理实现了“线面平行”向“线线平行”的转化，是空间平行关系推理链中的核心枢纽。

- **关键方法**  
  1. **证明线面平行**：首选平面几何途径——在目标平面内找出一条与已知直线平行的直线（常借助三角形中位线、平行四边形对边、成比例线段等）。  
  2. **利用线面平行证线线平行**：构造过已知直线的辅助平面，确定交线，再由性质定理得出平行。  
  3. **向量法**：建立空间直角坐标系，求直线的方向向量 $\vec{d}$ 和平面的法向量 $\vec{n}$，若 $\vec{d} \cdot \vec{n} = 0$ 且直线上至少有一点不在平面内，则线面平行。

- **典型应用**  
  高考中常以长方体、三棱柱、四棱锥等为载体设问。  
  - **直接证明**：如“在正方体 $ABCD-A_1B_1C_1D_1$ 中，求证 $A_1C_1 \parallel$ 平面 $AB_1D_1$”，只需连 $B_1D_1$，证明 $A_1C_1 \parallel B_1D_1$ 且 $A_1C_1$ 不在平面内即可。  
  - **探究性问题**：已知线面平行，求截面交线位置或判断其它线面关系，需反复运用性质定理将线面平行转化为线线平行，再结合相似比等条件求解。

## ⚠️ 高频易错点

- **忽略“平面外”条件**：应用判定定理时，未明确已知直线不在平面内，直接由“线线平行”推出“线面平行”导致失分。凡用判定定理，必须指明直线在平面外。
- **性质定理使用不规范**：已知 $l \parallel \alpha$，需构造过 $l$ 且与 $\alpha$ 相交的平面 $\beta$，许多同学找不到交线，或误把过 $l$ 的任意直线当作交线。
- **把线面平行当成面面平行**：由一条直线平行于一个平面，不能推出该平面内的所有直线都与该直线平行，切勿与面面平行的性质混淆。
- **向量法遗漏“点不在平面内”的验证**：仅由方向向量与法向量垂直就断言线面平行，忽略了直线可能在平面内的情况，必须再验证直线上一点不满足平面方程。
