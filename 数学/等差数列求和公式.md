# 等差数列求和公式

> **一句话**：等差数列的前 n 项和等于首项与末项之和乘以项数再除以 2，也可由首项、公差和项数表示为关于 n 的二次函数形式。
> **English**: The sum of the first n terms of an arithmetic sequence equals the number of terms multiplied by the average of the first and last terms, or it can be expressed as a quadratic function of n in terms of the first term and common difference.

## 🎓 考点精讲

**定义与原理**  
若数列 $\{a_n\}$ 从第 2 项起每一项与前一项的差为常数 $d$，则称其为等差数列，$a_1$ 为首项。其前 $n$ 项和  
$$
S_n = \frac{n(a_1 + a_n)}{2}
$$  
将通项 $a_n = a_1 + (n-1)d$ 代入，得  
$$
S_n = n a_1 + \frac{n(n-1)}{2}d
$$  
公式推导的核心方法是**倒序相加**：将 $S_n$ 正写与倒写相加，利用 $a_1 + a_n = a_2 + a_{n-1} = \cdots$ 的配对性质。

**关键方法**  
高考中解题强调“知三求二”：在 $a_1, d, n, a_n, S_n$ 中已知三个量列方程（组）求另外两个。根据已知条件选择合适的求和公式：已知 $a_1$ 与 $a_n$ 时用 $S_n = \frac{n(a_1 + a_n)}{2}$；已知 $a_1$ 与 $d$ 时用含 $d$ 的公式。处理片段和（如从第 $m$ 项到第 $n$ 项的和）时，常转化为 $S_n - S_{m-1}$，注意项数计算。

**典型应用**  
高考中常见设问：直接计算给定等差数列的前 n 项和；由 $S_n$ 反求项数 $n$ 或公差 $d$；利用 $S_n$ 的二次函数形式 $S_n = \frac{d}{2}n^2 + (a_1 - \frac{d}{2})n$ 求最值（当 $d<0$ 时存在最大值）；与不等式、函数性质结合，考查数列单调性与 $S_n$ 的最值；应用题中出现等差数列模型，如计费累加、分期付款、堆垛物料总数等。

## ⚠️ 高频易错点

- **项数错误**：求下标从 $m$ 到 $n$ 的连续项和时，项数为 $n-m+1$，而不是 $n-m$，更不可直接套用 $S_n$。
- **混淆公式适用条件**：$S_n = \frac{n(a_1 + a_n)}{2}$ 需要明确末项 $a_n$，若只给 $d$ 而未求 $a_n$ 则直接使用容易出现项数错配。
- **忽略 $S_n$ 的二次函数结构**：$S_n = An^2 + Bn$ 形式中常数项必须为 $0$，这是等差数列前 n 项和的判别标志；当 $d \neq 0$ 时，$S_n$ 是 $n$ 的二次函数，但学生常误用一次函数性质求最值。
- **和式符号误读**：看到 $S_{2n}$ 或 $S_{n+1}$ 时误认为数列只有这些项，或求项 $a_n$ 时混淆 $S_n - S_{n-1}$ 中 $n \ge 2$ 的条件，忘记单独验证 $a_1$。
