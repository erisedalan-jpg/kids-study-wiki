# 线段垂直平分线

> **课本定义**：经过线段中点且与这条线段垂直的直线，叫做这条线段的**垂直平分线**（也叫中垂线）。线段垂直平分线上的点到这条线段两端点的距离相等；反之，到线段两端点距离相等的点在这条线段的垂直平分线上。

---

## 📚 6-12 岁（回溯版）

### 什么是垂直平分线？

垂直平分线 = **垂直** + **平分**（经过中点）

- 垂直：与线段成 90° 角
- 平分：经过线段的中点

每条线段有且只有一条垂直平分线。

### 核心性质（垂直平分线定理）

**正方向**：如果点 $P$ 在线段 $AB$ 的垂直平分线上，那么 $PA = PB$。

**逆命题**：如果 $PA = PB$，那么点 $P$ 在线段 $AB$ 的垂直平分线上。

口诀：**中垂线上点，到两端等距。**

### 生活应用

- 找到两个城市的"等距分界线"就是连接两城市线段的垂直平分线
- 建立气象站时，选在距离三个观测点等距的位置，就是找三条垂直平分线的交点（外心）

### 三角形三边的垂直平分线

三角形三条边的垂直平分线交于一点，叫做**外心**——这个点到三个顶点的距离相等（即三角形外接圆的圆心）。

---

## 🎓 12+ 进阶版

### 严格定理

**定理**：线段 $AB$ 的垂直平分线 $l$ 上的任意点 $P$，有 $PA = PB$。

**证明**：设 $M$ 为 $AB$ 中点，$PM \perp AB$。

在 $\triangle PAM$ 和 $\triangle PBM$ 中：$AM = BM$（$M$ 为中点），$PM = PM$（公共边），$\angle PMA = \angle PMB = 90°$

由 SAS，$\triangle PAM \cong \triangle PBM$，∴ $PA = PB$。$\blacksquare$

**逆定理**：若 $PA = PB$，则 $P$ 在 $AB$ 的垂直平分线上。

**证明**：设 $M$ 为 $AB$ 中点，连接 $PM$。

在 $\triangle PAM$ 和 $\triangle PBM$ 中：$PA = PB$，$AM = BM$，$PM = PM$

由 SSS，$\triangle PAM \cong \triangle PBM$，∴ $\angle PMA = \angle PMB$

又 $\angle PMA + \angle PMB = 180°$，∴ $\angle PMA = \angle PMB = 90°$，即 $PM \perp AB$。

故 $P$ 在 $AB$ 的垂直平分线上。$\blacksquare$

### 坐标中的垂直平分线

已知 $A(x_1, y_1)$，$B(x_2, y_2)$，$AB$ 的垂直平分线方程：

设 $P(x, y)$，由 $PA = PB$：

$(x - x_1)^2 + (y - y_1)^2 = (x - x_2)^2 + (y - y_2)^2$

化简得：$2(x_2 - x_1)x + 2(y_2 - y_1)y = x_2^2 - x_1^2 + y_2^2 - y_1^2$

### 典型例题

**例题 1**：已知 $PA = PB = 5$，$AB = 8$，$P$ 在 $AB$ 的垂直平分线上，求 $P$ 到 $AB$ 的距离。

解：设 $M$ 为 $AB$ 中点，$AM = 4$。

$PM = \sqrt{PA^2 - AM^2} = \sqrt{25 - 16} = 3$

∴ $P$ 到 $AB$ 的距离为 **3**。

**例题 2**：已知点 $P$ 到三角形 $\triangle ABC$ 三个顶点的距离相等，求证 $P$ 是三角形外接圆的圆心。

证明：$PA = PB$ ⟹ $P$ 在 $AB$ 的垂直平分线上

$PB = PC$ ⟹ $P$ 在 $BC$ 的垂直平分线上

故 $P$ 是三边垂直平分线的交点（外心），到三顶点距离相等，$PA$ 为外接圆半径。✓

### 易错点

1. **垂直平分线是直线**，不是线段（虽然我们有时画一段，但概念上是无限长的直线）。
2. 不要把"经过中点"和"垂直"分开理解——两个条件缺一不可。
3. 逆定理同样重要：$PA = PB$ 就能推出 $P$ 在中垂线上。
