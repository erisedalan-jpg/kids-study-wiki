## 课本一句话定义

一个数 $a$ 在数轴上所对应的点到**原点的距离**，叫做该数的**绝对值**，记作 $|a|$；正数的绝对值是它本身，负数的绝对值是它的相反数，0 的绝对值是 0。

---

## 📚 给 6-12 岁（小学高年级回溯版）

绝对值就是"离 0 有多远"，只看**距离**，不管方向。

- $|5|$ = 5（向右走 5 步，距离是 5）
- $|-5|$ = 5（向左走 5 步，距离也是 5）
- $|0|$ = 0（就在原点，距离 0）

绝对值永远 **≥ 0**，因为距离不能是负的！

想象你去超市，出发点是家（原点），向右走 5 分钟或向左走 5 分钟，走路的时间（绝对值）都是 5 分钟。

## 🎓 给 12+（进阶版）

### 严格定义

$$|a| = \begin{cases} a & \text{若 } a > 0 \\ 0 & \text{若 } a = 0 \\ -a & \text{若 } a < 0 \end{cases}$$

几何意义：$|a|$ 是数轴上 $a$ 点到原点 $O$ 的距离；$|a - b|$ 是 $a$ 点到 $b$ 点的距离。

### 关键性质

| 性质 | 公式 |
|------|------|
| 非负性 | $|a| \geq 0$，等号当且仅当 $a = 0$ |
| 对称性 | $|a| = |-a|$ |
| 乘积 | $|ab| = |a| \cdot |b|$ |
| 商 | $\left|\dfrac{a}{b}\right| = \dfrac{|a|}{|b|}\ (b \neq 0)$ |
| 三角不等式 | $|a + b| \leq |a| + |b|$ |
| 平方根表示 | $|a| = \sqrt{a^2}$ |

### 典型例题

**例 1**：计算 $|-3.5|$、$|0|$、$\left|-\dfrac{2}{3}\right|$、$|+7|$。

**解**：$|-3.5| = 3.5$；$|0| = 0$；$\left|-\dfrac{2}{3}\right| = \dfrac{2}{3}$；$|+7| = 7$。

**例 2**：比较 $-3$ 和 $-5$ 的大小（利用绝对值）。

**解**：$|-3| = 3$，$|-5| = 5$，因为 $3 < 5$，所以两个负数中绝对值大的反而更小，即 $-5 < -3$。

**例 3**：若 $|x| = 4$，求 $x$。

**解**：$x = 4$ 或 $x = -4$（数轴上距原点 4 的点有两个）。

### 易错点

1. **$|a|$ 不一定等于 $a$**：当 $a < 0$ 时，$|a| = -a > 0$。
2. **绝对值 ≠ 去掉负号**：$|-x|$（$x > 0$ 时）$= x$，但 $-x$ 本身已经是负数，去掉负号得正数。
3. **两个负数比较大小**：绝对值越大，这个负数越小（在数轴上越靠左）。

### 知识联系

- 前置：数轴、相反数、正负数
- 后续：有理数大小比较 → 有理数四则运算 → 解含绝对值方程
- 高中拓展：$|f(x)|$ 的图像、复数的模 $|z| = \sqrt{a^2 + b^2}$、向量的模
