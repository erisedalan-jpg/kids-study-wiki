# 绝对值不等式解法

> **一句话**：通过分类讨论或几何意义，将含有绝对值符号的不等式等价转化为不含绝对值的不等式组来求解的方法。
> **English**: The method of solving inequalities involving absolute value symbols by converting them into equivalent inequalities without absolute value symbols through case analysis or geometric interpretation.

## 🎓 考点精讲

**定义与原理**  
形如 $|ax+b|\le c$、$|ax+b|\ge c$ 或 $|ax+b|+|cx+d|\le m$ 等含有绝对值符号的不等式称为绝对值不等式。求解依据绝对值的代数定义（$|x|=\begin{cases}x,x\ge0\\-x,x<0\end{cases}$）与几何意义（数轴上点到原点的距离）。

**关键方法**  
1. **单一绝对值型**：  
   - 若 $c>0$，$|ax+b|\le c \Leftrightarrow -c\le ax+b\le c$；  
   - $|ax+b|\ge c \Leftrightarrow ax+b\le -c$ 或 $ax+b\ge c$；  
   - 当 $c\le0$ 时，需结合不等式性质直接判断解集（例如 $|f(x)|\le c<0$ 无解，$|f(x)|\ge c\le0$ 解集为 $\mathbb{R}$）。  
2. **平方去绝对值**：适用于 $|f(x)|\le|g(x)|$ 型，转化为 $f^2(x)\le g^2(x)$ 解之。  
3. **零点分段法（分类讨论）**：对含有两个及以上绝对值的不等式，如 $|x-a|+|x-b|\ge c$，先令每个绝对值内为零得零点，分区间讨论，将绝对值号去掉，得到若干不等式组，最后取并集。  
4. **绝对值三角不等式**：$||a|-|b||\le|a\pm b|\le|a|+|b|$，常用于求最值或参数范围，注意等号成立条件。  
5. **几何意义**：$|x-a|+|x-b|$ 表示数轴上点 $x$ 到 $a,b$ 的距离之和，可借助数轴直观确定解集。

**典型应用**  
高考中常以选择题、填空题考查单一绝对值不等式的解法，或作为解答题第一问（如选考部分）。常见设问方式：解 $|2x-1|>5$；若 $|x-a|+|x-3|\ge1$ 的解集为 $\mathbb{R}$，求 $a$ 的取值范围。后者需利用绝对值三角不等式或几何意义转化为最值问题。

## ⚠️ 高频易错点

- 忽略右侧常数 $c$ 的符号直接套用 $|f(x)|\le c \Leftrightarrow -c\le f(x)\le c$。当 $c<0$ 时该形式解集为空，不可错误转化。  
- 用零点分段法处理 $|x-a|+|x-b|$ 型不等式时，遗漏或重复各区间端点，导致解集并集不完整。  
- 使用绝对值三角不等式时忽视等号成立的条件，导致求出的参数范围不精确。  
- 面对 $|f(x)|\ge g(x)$ 型不等式，错误化为 $f(x)\ge g(x)$ 或 $f(x)\le -g(x)$，未讨论 $g(x)$ 的符号，正确做法是等价于 $f(x)\ge g(x)$ 或 $f(x)\le -g(x)$ 且需注意 $g(x)$ 的正负不影响或需分类。
