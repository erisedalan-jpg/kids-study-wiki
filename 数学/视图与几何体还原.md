# 视图与几何体还原

> **一句话**：已知三视图，还原几何体的形状；或已知几何体，画出其三视图——两个方向的综合应用。

---

## 📚 给 6-12 岁（自读版）

### 两个方向的练习

- **方向 1**：给出一个立体图形，画出它的三视图。
- **方向 2**：给出三视图，判断或还原立体图形。

### 解题小技巧

看三视图还原几何体时：
1. 先看**俯视图**——确定底面形状
2. 再看**正视图**——确定高度和正面轮廓
3. 最后看**侧视图**——确认侧面细节

### 常见对应关系

| 三视图特征 | 可能的几何体 |
|-----------|------------|
| 三个视图都是正方形 | 正方体 |
| 两个矩形 + 一个圆 | 圆柱 |
| 两个三角形 + 一个圆 | 圆锥 |
| 三个圆 | 球 |
| 矩形 + 梯形 + 矩形 | 棱台或棱柱切割体 |

## 🎓 给 12+（进阶版）

### 由几何体画三视图

**步骤**：
1. 确定正视图方向（通常选最能体现形状特征的方向）
2. 按"长对正、高平齐、宽相等"的规则画三个视图
3. 用实线画可见边，用虚线画不可见边（初中通常不要求虚线）

### 由三视图还原几何体

**步骤**：
1. 分析每个视图的轮廓形状
2. 综合三个视图的信息，排除矛盾
3. 借助直观图（斜二测画法）帮助理解

### 常见组合体三视图

**例（L 形棱柱）**：
- 正视图：L 形（两个矩形组合）
- 俯视图：矩形
- 侧视图：矩形

**例（圆柱上放球）**：
- 正视图：矩形上加一个圆（半圆）
- 俯视图：大圆（圆柱底面）内含小圆（球的投影）
- 侧视图：矩形上加圆

### 典型例题

**例 1**：已知一几何体的三视图：正视图为直角三角形（底边 4，高 3），俯视图为矩形（4×2），侧视图为直角三角形（底边 2，高 3），判断几何体。

分析：底面为矩形，高度方向截面为三角形 → 该几何体为**三棱柱**，其中一个底面为直角三角形，斜面对应正视图的斜边。

**例 2（作图）**：圆柱被过两底面直径的截面切成两半，画出半圆柱的三视图。

解：
- 正视图：矩形（高 = 圆柱高，宽 = 直径 $2r$）
- 俯视图：半圆（底面半圆形）
- 侧视图：矩形（高 = 圆柱高，宽 = 半径 $r$）

**例 3（判断多解性）**：有时候，同一组三视图可能对应多个不同的几何体（三视图不能唯一确定几何体）。例如：正视图和侧视图都是正方形，俯视图是正方形，可能是正方体，也可能是圆柱（俯视图恰好是方形的情况下……实际上圆柱俯视图是圆，不同），需要具体分析。

### 易错点汇总

- ❌ 画三视图时"长对正、高平齐、宽相等"对应错误
- ❌ 还原时只看两个视图就下结论（必须三个视图综合判断）
- ❌ 忽略不可见的内部结构（如空心几何体）
