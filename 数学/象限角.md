# 象限角

> **一句话**：在平面直角坐标系中，使角的顶点与原点重合、始边与 `x` 轴非负半轴重合，终边落在第几象限，就称该角为第几象限角；终边落在坐标轴上的角不属于任何象限。
> **English**: An angle in standard position whose terminal side lies in a specific quadrant of the coordinate plane; angles with terminal sides on the axes are not quadrantal in this sense.

## 🎓 考点精讲

- **定义/原理**：将任意角置于平面直角坐标系的标准位置——顶点与原点重合，始边与 `x` 轴正半轴重合。此时，角的终边决定了其象限归属：
  - 终边在第一象限：`2kπ < α < 2kπ + π/2`，`k ∈ Z`；
  - 终边在第二象限：`2kπ + π/2 < α < 2kπ + π`；
  - 终边在第三象限：`2kπ + π < α < 2kπ + 3π/2`；
  - 终边在第四象限：`2kπ + 3π/2 < α < 2kπ + 2π`。
  终边落在坐标轴上（如 `α = kπ/2`）的角称为轴线角，不属于任何象限。

- **关键方法**：
  1. **给定角 `α`，判断其象限**：先将 `α` 化为 `α = 2kπ + β`（或 `k·360° + β`），其中 `β ∈ [0, 2π)`，再依据 `β` 的范围判定。
  2. **已知 `α` 象限，求 `nα` 或 `α/n` 的象限**：将各象限角范围用不等式表示（如第一象限 `2kπ < α < 2kπ + π/2`），两边除以 `n` 或乘以 `n`，并对 `k` 取模讨论。常用“等分象限法”：把每个象限 `n` 等分，按逆时针标号即可得 `α/n` 的可能象限。
  3. **集合表示**：象限角可用集合形式写出，如第一象限角集合 `{α | 2kπ < α < 2kπ + π/2, k ∈ Z}`。

- **典型应用**：高考中常见的设问包括：
  - 判断给定角（如 `2025°`、`-15π/4`）是第几象限角；
  - 已知 `α` 是第二象限角，求 `α/2` 或 `2α` 所在的象限；
  - 结合三角函数值符号（正弦、余弦、正切在各象限的正负）进行条件推理，如“若 `sin α > 0` 且 `cos α < 0`，则 `α` 是第几象限角”；
  - 与终边相同角的表示综合考查，要求能用集合或不等式描述某一类象限角。

## ⚠️ 高频易错点

- 混淆象限角与锐角、钝角、0°~360°角等概念：象限角是按终边位置划分的任意角，与角度的绝对大小无直接对应关系；锐角一定是第一象限角，但第一象限角不一定是锐角。
- 将终边在坐标轴上的轴线角错误归类为象限角，或在判断 `α/n` 象限时遗漏轴线情况（如 `α` 终边在 `y` 轴正半轴时，`α/2` 可能出现轴线）。
- 用集合或不等式表示象限角时，漏写 `k ∈ Z` 或范围边界开闭混淆（象限角边界均为开区间，不含轴线）。
- 已知 `α` 所在象限求 `α/3` 或 `α/2` 时，仅凭直觉而不使用等分象限法，导致漏掉可能象限或得出错误象限。
