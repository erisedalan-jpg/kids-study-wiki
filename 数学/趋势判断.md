# 趋势判断

> **一句话**：趋势判断是分析函数值随自变量增大是持续增大（递增）、减小（递减）还是保持不变（常数），以及增减速快慢变化的数学方法，核心工具为导数符号分析与差分比较。
> **English**: Trend judgment refers to the mathematical analysis of whether a function's values consistently increase, decrease, or remain constant as the independent variable grows, primarily using derivative sign analysis and difference comparison.

## 🎓 考点精讲

**定义/原理**：
- **函数单调性**：对于定义域 $I$ 内任意 $x_1 < x_2$，若恒有 $f(x_1) \le f(x_2)$，则 $f(x)$ 在 $I$ 上单调递增（严格递增时为 $<$）；若恒有 $f(x_1) \ge f(x_2)$，则单调递减。
- **导数判据**：设 $f(x)$ 在区间内可导，则 $f'(x) \ge 0 \implies f(x)$ 单调递增；$f'(x) \le 0 \implies f(x)$ 单调递减。$f'(x)$ 的零点及符号变化点是单调区间的分界。
- **数列单调性**：对数列 $\{a_n\}$，若有 $a_{n+1} - a_n \ge 0$（或 $a_{n+1} / a_n \ge 1$ 且各项同号），则数列单调递增。

**关键方法**：
1. **导数法**：求 $f'(x)$ → 解 $f'(x)=0$ 得临界点 → 列表分区间判断 $f'(x)$ 符号 → 写出单调区间（标准步骤）。
2. **定义法/作差法**：任取 $x_1<x_2$，计算 $\Delta = f(x_1)-f(x_2)$，因式分解后判断 $\Delta$ 的正负。
3. **复合函数法则**：同增异减——内外层单调性相同则复合后递增，相异则递减。

**典型应用**：
- **高考常见设问**：“求函数 $f(x)$ 的单调区间”“讨论 $f(x)$ 的单调性”“证明函数在 $(0,+\infty)$ 上单调递增”“已知 $f(x)$ 在 $[1,2]$ 上单调递减，求参数 $a$ 的取值范围”。
- **具体场景**：已知 $f(x) = x^3 - 3x^2 + a$，求单调区间。解 $f'(x)=3x^2-6x = 3x(x-2)$，令导数为零得 $x=0$ 或 $x=2$，分区间 $(-\infty,0)$、$(0,2)$、$(2,+\infty)$ 判断 $f'(x)$ 符号，得增区间 $(-\infty,0)$ 和 $(2,+\infty)$，减区间 $(0,2)$。若已知函数 $f(x) = e^x - ax$ 在 $(0,+\infty)$ 单调递增，求 $a$ 范围：由 $f'(x) = e^x - a \ge 0$ 在 $(0,+\infty)$ 恒成立，只需 $a \le \min_{x>0} e^x = 1$。

## ⚠️ 高频易错点

- **忽略定义域**：不先确定函数定义域就直接求导判单调，导致在无定义点处错误划分区间（如对数函数 $f(x)=\ln(x-2)$ 必须先限制 $x>2$）。
- **导数 $f'(x) \ge 0$ 与严格递增混淆**：$f'(x) \ge 0$ 且不在任何子区间恒为零，才是严格递增；仅由 $f'(x) \ge 0$ 就断言严格递增可能错误（如常函数导数为 0）。
- **导数零点处未检验符号变化**：$f'(x_0)=0$ 不保证 $x_0$ 是极值点或单调性改变点；必须检验 $x_0$ 两侧导数是否异号，同号则函数在 $x_0$ 处仍保持原单调趋势。
- **数列作差后符号判断失误**：$a_{n+1} - a_n$ 化简后未充分考虑 $n$ 的取值限制或正负号条件，直接错误判定单调性。
