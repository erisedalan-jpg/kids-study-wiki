# 辅助角公式

> **一句话**：将形如 $a\sin x + b\cos x$ 的表达式化为单一正弦（或余弦）函数的恒等变形公式，用于研究三角函数的性质及求最值。
> **English**: A trigonometric identity used to transform an expression of the form $a\sin x + b\cos x$ into a single sine or cosine function for the purpose of analyzing properties such as amplitude and extremum.

## 🎓 考点精讲

**定义/原理**：对于任意实数 $a$、$b$（不同时为零），存在辅助角 $\varphi$，满足：
$$
a\sin x + b\cos x = \sqrt{a^2+b^2}\sin(x+\varphi)
$$
其中 $\varphi$ 由 $\tan\varphi = \frac{b}{a}$ 确定，$\varphi$ 所在象限由点 $(a, b)$ 的符号决定。等价地，也可化为余弦形式：$a\sin x + b\cos x = \sqrt{a^2+b^2}\cos(x-\theta)$，其中 $\tan\theta = \frac{a}{b}$。

**关键方法**：运用辅助角公式的步骤为：
1. 提取模长 $R = \sqrt{a^2+b^2}$；
2. 将原式写为 $R\left(\frac{a}{R}\sin x + \frac{b}{R}\cos x\right)$；
3. 令 $\cos\varphi = \frac{a}{R}$，$\sin\varphi = \frac{b}{R}$，则原式化为 $R\sin(x+\varphi)$。

**典型应用**：高考中主要考查以下场景：
- **求三角函数的最值**：如求 $f(x)=3\sin x + 4\cos x$ 的最大值。化为 $5\sin(x+\varphi)$（其中 $\tan\varphi=\frac{4}{3}$，$\varphi$ 第一象限），最大值为 $5$。
- **解三角方程或求单调区间**：将多个三角函数项合并为单一函数，利用正弦型函数的性质直接写出单调区间或对称轴。
- **解三角形**：在已知 $a\sin A + b\cos A = c$ 等条件时，辅助角公式用于简化方程求解角度 $A$ 的大小。

## ⚠️ 高频易错点

- **辅助角 $\varphi$ 的象限误判**：仅凭 $\tan\varphi = \frac{b}{a}$ 确定角度，忽略 $a$、$b$ 的符号。$\varphi$ 的终边必须与点 $(a, b)$ 所在象限一致，例如 $a>0, b<0$ 时 $\varphi$ 在第四象限。
- **化余弦时符号处理不当**：将 $a\sin x + b\cos x$ 化为 $\sqrt{a^2+b^2}\cos(x-\theta)$ 时，$\theta$ 由 $\tan\theta = \frac{a}{b}$ 确定，此时 $\cos\theta = \frac{b}{\sqrt{a^2+b^2}}$，$\sin\theta = \frac{a}{\sqrt{a^2+b^2}}$，易将 $\sin\theta$ 与 $\cos\theta$ 的分量对应关系搞反。
- **忽略 $a=0$ 或 $b=0$ 的退化情形**：公式仍成立，但辅助角直接是特殊角（$0$ 或 $\frac{\pi}{2}$），部分学生强行套用公式而不直接化简，导致计算繁琐。
- **混淆合并前的系数**：$\sqrt{a^2+b^2}$ 中的 $a$ 和 $b$ 是原 $\sin x$ 项和 $\cos x$ 项的系数，如果原式为 $b\cos x + a\sin x$ 顺序调换后，套用公式时易将 $a$、$b$ 错位。
