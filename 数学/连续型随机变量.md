## 🎓 连续型随机变量

### 定义

若随机变量 $X$ 的取值充满某个区间（$[a,b]$ 或 $(-\infty, +\infty)$），且存在非负函数 $f(x)$（概率密度函数），使得对任意实数 $a \leq b$：

$$P(a \leq X \leq b) = \int_a^b f(x) dx$$

则称 $X$ 为**连续型随机变量**，$f(x)$ 为其**概率密度函数**。

### 密度函数的条件

1. $f(x) \geq 0$（非负性）
2. $\int_{-\infty}^{+\infty} f(x) dx = 1$（归一性，总概率为 1）

### 与离散型的对比

| 对比项 | 离散型 | 连续型 |
|--------|--------|--------|
| 取值 | 有限或可列个 | 区间上连续取值 |
| 概率描述 | 分布列 $P(X = x_i) = p_i$ | 密度函数 $f(x)$ |
| 单点概率 | $P(X = x_i) = p_i > 0$ | $P(X = c) = 0$ |
| 区间概率 | $P(a \leq X \leq b) = \sum p_i$ | $P(a \leq X \leq b) = \int_a^b f(x)dx$ |
| 典型分布 | 二项分布、超几何分布 | 正态分布、均匀分布 |

### 重要性质：单点概率为零

$$P(X = c) = \int_c^c f(x)dx = 0 \quad \text{（对任意点 } c\text{）}$$

因此：

$$P(a < X < b) = P(a \leq X < b) = P(a < X \leq b) = P(a \leq X \leq b)$$

端点开闭不影响连续型随机变量的概率。

### 典型分布

**均匀分布** $U[a,b]$：

$$f(x) = \begin{cases} \dfrac{1}{b-a} & a \leq x \leq b \\ 0 & \text{其他} \end{cases}$$

$E(X) = \dfrac{a+b}{2}$，$D(X) = \dfrac{(b-a)^2}{12}$

**正态分布** $N(\mu, \sigma^2)$：最重要的连续型分布（见[[正态分布概念]]）。

### 典型例题

**例1（均匀分布）**：公共汽车每 10 分钟一班，乘客到站时间均匀分布在 $[0,10]$，求等待时间 $X$ 不超过 3 分钟的概率。

$$P(0 \leq X \leq 3) = \int_0^3 \frac{1}{10} dx = \frac{3}{10} = 0.3$$

**例2（密度函数判断）**：判断 $f(x) = 2x$（$0 \leq x \leq 1$），$f(x) = 0$（其他）是否合法。

$$\int_0^1 2x \, dx = \left[x^2\right]_0^1 = 1 \; \checkmark, \quad f(x) \geq 0 \; \checkmark$$

合法，$E(X) = \int_0^1 x \cdot 2x \, dx = \int_0^1 2x^2 dx = \frac{2}{3}$

**例3（正态分布）**：$X \sim N(0,1)$，$P(X = 0) = ?$

$$P(X = 0) = 0 \quad \text{（连续型随机变量单点概率为零）}$$

### 期望与方差

$$E(X) = \int_{-\infty}^{+\infty} x f(x) dx$$

$$D(X) = \int_{-\infty}^{+\infty} (x - E(X))^2 f(x) dx = E(X^2) - [E(X)]^2$$

（高中阶段一般不考积分计算，直接给出结论）

### 易错点

1. **密度函数值 $\neq$ 概率**：$f(x)$ 可以大于 1（如均匀分布在 $[0, 0.5]$ 上，$f(x)=2$）
2. **单点概率为零**：$P(X = c) = 0$，但 $c$ 仍是可能的取值（只是概率为零）
3. **连续型不用分布列**：连续型随机变量无法列举所有取值，只能用密度函数
4. **端点处理**：连续型时，开区间和闭区间的概率相同，不用纠结端点
