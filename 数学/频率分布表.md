# 频率分布表

> **一句话**：频率分布表是将样本数据按一定区间分组后，列出各组频数（或频率）的统计表格。
> **English**: A frequency distribution table is a statistical table that displays the frequency or relative frequency of data within specified class intervals.

## 🎓 考点精讲

- **定义/原理**：将样本数据按照数值范围划分为若干不相交的组（区间），每个组称为一个组段（class interval）。统计落入每个组段的数据个数，称为频数（frequency），记作 $f_i$；频数与样本总数 $n$ 的比值称为频率（relative frequency），记作 $p_i = f_i/n$。完整的频率分布表至少包含组段、频数、频率三列，有时也引入累计频率。

- **关键方法**：制作频率分布表的标准步骤：
  1. **求极差** $R = x_{\max} - x_{\min}$。
  2. **确定组数和组距**：组数 $k$ 一般取 5–12 组，组距 $d \approx R/k$，取便于计算的较整数值。
  3. **划分组段**：通常采用左闭右开区间（或左开右闭），且第一组下限略小于 $x_{\min}$，最后一组上限略大于 $x_{\max}$。
  4. **统计频数**：逐个数据划入对应组段，得到频数 $f_i$。
  5. **计算频率**：$p_i = f_i/n$，验证 $\sum p_i = 1$（（待家长核对）。

- **典型应用**：高考中常见于统计解答题或选填题。如给定一批原始数据（身高、成绩等），要求完成频率分布表，并进一步绘制频率分布直方图，或利用频率分布表估计样本的中位数、平均数、方差，以及利用频率估计概率。有时题目直接给出不完整的表，要求补全频率或频数。

## ⚠️ 高频易错点

- **组段边界归属混乱**：未明确约定区间开闭（如“90—100”是否包含90、100），导致频数重复或遗漏。一律按“上限不在内”原则处理时易忽略区间表述差异。
- **频率与频数混淆**：直接使用频数代替频率参与概率估计或直方图纵坐标设置，忽略分母 $n$。频率之和应为 1，若算出总和不为 1 而未检查，导致后续全错。
- **据频率分布表估计中位数时机械取中点**：错误地将包含中位数的组段的中点作为中位数，忽视组内数据分布假设，未用线性插值 $n/2$ 位置的方法。
- **混淆频率分布表与频率分布直方图的纵轴**：由表转图时误将频率直接作为直方图的矩形高度，实际高度应为 $p_i / \text{组距}$，从而使得面积代表频率。
