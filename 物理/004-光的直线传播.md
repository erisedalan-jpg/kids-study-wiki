# 光的直线传播

> **一句话**：光在**同种均匀介质**中沿直线传播，用光线（带箭头直线）表示其传播路径。
> **English**: Light travels in a straight line in a uniform medium; its path is represented by rays (arrows).

---

## 📚 给 6-12 岁（自读版）

你有没有注意过，阳光从窗帘缝里射进来是一条条"光柱"？这就说明光是**直线传播**的！

**光沿直线传播的现象：**

1. **小孔成像**：纸盒上戳一个小孔，对着蜡烛，盒子另一面出现蜡烛的**倒像**！（因为光直线穿过小孔，上方光线→下方，下方光线→上方，所以像是倒的）

2. **影子的形成**：太阳光直线传播，被不透明物体挡住，在身后形成影子

3. **日食和月食**：月球挡住太阳光（日食）或地球挡住射向月球的阳光（月食），都是光直线传播的结果

4. **瞄准**：射击时眼睛-准星-目标要对齐，就是利用光的直线传播

**光线**：用一条带箭头的直线表示光的传播路径和方向，这是一种简便的画法（不是真实存在的线）。

## 🎓 给 12+（进阶版）

### 定义

**光的直线传播**：光在**同种均匀介质**中沿直线传播。

条件：介质须**均匀**（密度、折射率处处相同）。若介质不均匀，光会发生折射（弯曲），如海市蜃楼就是大气折射现象。

### 光线模型

用**光线**（带箭头的直线）表示光的传播路径：
- 箭头指向光的传播方向
- 光线是模型，便于作图分析，并非真实存在

### 重要现象

**1. 小孔成像**

$$\frac{像高}{物高} = \frac{像距}{物距} \quad \Rightarrow \quad \frac{h'}{h} = \frac{v}{u}$$

- 成倒立、缩小或放大的**实像**（可以在光屏上呈现）
- 小孔越小，像越清晰；小孔越大，像越模糊（光斑越大）

**2. 影子的形成**

光源 + 不透明物体 → 影子

- **本影**：完全没有光照射到的区域（完全黑暗）
- **半影**：部分光照射到的区域（半明半暗）
- 光源越大（面光源），半影越大

**3. 日食和月食**

| 现象 | 原因 |
|------|------|
| 日食 | 月球本影扫过地球，月球挡住太阳光 |
| 月食 | 月球进入地球本影，地球挡住照向月球的阳光 |

### 光的传播速度

- 真空中：$c = 3 \times 10^8 \text{ m/s}$（最大值）
- 在介质中：$v < c$（如水中约 $2.25 \times 10^8$ m/s）

### 典型例题

**例1**：身高 1.6 m 的人站在路灯下，身影长 0.8 m，路灯高 4 m，路灯到人的距离为多少？

设距离为 $d$，利用相似三角形：

$$\frac{4}{1.6} = \frac{d + 0.8}{0.8}$$

$$2.5 = \frac{d + 0.8}{0.8}$$

$$d = 2.5 \times 0.8 - 0.8 = 2 - 0.8 = 1.2 \text{ m}$$

**易错点：**
- 光的直线传播有条件：同种**均匀**介质
- 小孔成像是实像，倒立，不通过透镜
- 光线是模型，不是真实存在的线
