# 参照物

> **一句话**：判断物体是否运动时，选来作比较的物体叫**参照物**；运动和静止都是相对的。
> **English**: A reference object is the object chosen for comparison when determining whether something is moving; motion and rest are relative.

---

## 📚 给 6-12 岁（自读版）

坐在快速行驶的火车上，看窗外的树：树在"移动"！但看窗里的座位：座位没动。到底什么在动？

这就需要**参照物**——你选谁来比较，就以谁为参照物。

**以地面（车站）为参照物：** 火车在动，树没动（树是静止的）

**以火车为参照物：** 火车没动，树在向后移动（树相对于火车在动）

**参照物可以随意选，但选不同的会得出不同答案：**
- 你坐在车里，以车为参照物：你静止
- 你坐在车里，以路边为参照物：你在运动

**生活中的例子：**
- 两列火车并排，一列缓缓开动，坐在里面感觉是"自己"还是"对方"在动？
- 飞机上，以飞机为参照物，乘客静止；以地面为参照物，乘客高速运动

**默认参照物：地面**（没特别说明时，一般以地面为参照物）

## 🎓 给 12+（进阶版）

### 定义

**参照物**（Reference Object / Reference Frame）：研究物体运动时，选定用来比较的物体。

参照物可以是任何物体（通常选静止不动的物体，如地面），但必须事先指定。

### 运动的相对性

**运动与静止是相对的**——同一物体，相对于不同参照物，运动状态不同。

**没有绝对静止，也没有绝对运动**（爱因斯坦相对论的基础）。

### 参照物的选择原则

1. 参照物可以任意选，通常选**地面或相对地面静止的物体**
2. 参照物**不能选被研究的物体本身**
3. 选不同参照物，结论不同——必须先明确

### 判断相对运动的方法

若 A 相对 B 运动，则 B 也相对 A 运动（方向相反）：
- 汽车向东行驶 → 以汽车为参照物，路边的树向西移动

### 参照系（高中知识）

参照系比参照物更广泛：参照系是一套坐标系 + 时间标度，可以建立在任意物体上。参照物是参照系的通俗说法。

### 典型例题

**例1**：小明坐在行驶的公共汽车上，以公共汽车为参照物，小明是？以路边树木为参照物，小明是？

**解**：
- 以公共汽车为参照物：小明与汽车相对位置不变，小明**静止**
- 以路边树木为参照物：小明随汽车一起运动，小明**运动**

**例2**："两岸青山相对出，孤帆一片日边来"——这里选的参照物是什么？

**解**："两岸青山相对出"：以船（孤帆）为参照物，山在移动（相对运动）；"孤帆一片日边来"：以岸为参照物，船在靠近。

**易错点：**
- 不能说某物体"绝对静止"，必须指明参照物
- 参照物不能选被研究的物体自身
- 参照物选择不同，结论可能完全相反
