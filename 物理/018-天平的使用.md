# 天平的使用

> **一句话**：天平是利用等臂杠杆原理测量物体**质量**的仪器，使用时遵循"左物右码、先大后小、调零归中"的规范。
> **English**: A balance uses the equal-arm lever principle to measure mass; proper use follows the rules of "object on left, weights on right, largest weight first, and level before use."

---

## 📚 给 6-12 岁（自读版）

超市里卖菜的老式秤，两边各放一个托盘，一边放东西，另一边放砝码，当两边平衡时就知道东西有多重——这就是**天平**的原理！

**天平的结构：**
- 两个**托盘**（左放物体，右放砝码）
- **横梁**（中间的杆）
- **平衡指针**（指向正中就是平衡了）
- **砝码盒**（有各种克数的砝码）
- **游码**（横梁上可以滑动的小块，用于称量小质量）

**使用步骤（口诀：调、放、加、读）：**
1. **调**：放桌子上，调节底部螺丝，使天平水平；把游码拨到零，调节平衡螺母，使指针指向正中间
2. **放**：左盘放被测物体
3. **加**：右盘加砝码（从最重的开始），直到平衡
4. **读**：右盘砝码质量 + 游码读数 = 物体质量

**注意事项：**
- 不能超过最大量程（一般 200 g 或 500 g）
- 不能把潮湿物品或化学药品直接放托盘上（要用纸垫）
- 不能用嘴吹砝码，用镊子夹取砝码

## 🎓 给 12+（进阶版）

### 原理

天平基于**等臂杠杆**（力矩平衡）原理：

$$m_{\text{物}} \times g \times L = m_{\text{码}} \times g \times L$$

因为两臂等长（$L$ 相同），重力加速度 $g$ 相同，所以：

$$m_{\text{物}} = m_{\text{码}}$$

**天平测质量不受地点影响**——在月球上或其他星球上，物体和砝码受到的重力同时变化，比例不变，天平仍然平衡时示数相同。

### 标准操作流程

**调零步骤（使用前）：**
1. 将天平放在水平桌面
2. 将游码移至横梁左端零刻度线
3. 调节两端**平衡螺母**（指针偏哪边，螺母向哪边调）
4. 直到指针指向正中或左右摆动对称为止

**称量步骤：**
1. 左盘放被称物体
2. 右盘按"**先大后小**"加砝码：从最大砝码开始，若偏重则换小一级
3. 若最小砝码（1 g）仍偏重，则调节**游码**（游码读左侧刻度线）
4. 指针居中时，**质量 = 右盘砝码总质量 + 游码示数**

### 注意事项

| 操作 | 要求 |
|------|------|
| 物品放置 | 左盘物体，右盘砝码（"左物右码"） |
| 砝码顺序 | 由大到小依次尝试 |
| 游码读数 | 读游码**左侧**刻度 |
| 称量限制 | 不超最大量程，不低于感量 |
| 药品称量 | 放在纸或烧杯上，不直接接触托盘 |
| 砝码操作 | 用镊子夹取，不用手直接拿 |

### 误差分析

若天平两臂不等长（$L_1 \neq L_2$），可用**替换法**消除系统误差：
$$m_{\text{物}} = \sqrt{m_1 \cdot m_2}$$
（$m_1$ 为物放左盘时右边砝码，$m_2$ 为物放右盘时左边砝码）

### 典型例题

**例1**：用天平称量某物体，右盘放了 50 g、20 g、5 g 砝码各一个，游码在 2.4 g 处，物体质量为？

$$m = 50 + 20 + 5 + 2.4 = 77.4 \text{ g}$$

**例2**：调好的天平，发现左盘比右盘略重，说明什么问题？如何处理？

**解**：说明天平未完全调平衡，或游码未归零。重新调节平衡螺母（向右调），直到指针居中。

**易错点：**
- 游码读左侧刻度线，不读右侧
- 调节平衡螺母时游码必须在零刻度
- 称量过程中不能再调平衡螺母（只能换砝码/调游码）
