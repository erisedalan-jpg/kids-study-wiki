# 密度与社会生活

> **一句话**：密度在材料选择、物质鉴别、浮沉判断等方面有广泛应用，是理解自然和工程的重要工具。
> **English**: Density has wide applications in material selection, substance identification, and determining buoyancy, making it an essential tool for understanding nature and engineering.

---

## 📚 给 6-12 岁（自读版）

密度不只是物理考试题——它在生活中到处都用到！

**密度与浮沉：**
- 冰（密度 0.9 g/cm³）< 水（1 g/cm³）→ 冰漂在水面
- 铁（密度 7.8 g/cm³）> 水 → 铁沉入水底
- 木头（大多 0.4~0.8 g/cm³）< 水 → 木头漂浮

所以密度比水小的物体漂浮，比水大的下沉！

**密度与材料选择：**
- 飞机要轻：用铝合金（密度 2.7 g/cm³，比钢轻）
- 子弹要重且硬：用铅或钢（密度大）
- 热气球：充热空气或氦气（密度比空气小）→ 上升

**密度鉴别物质：**
- 买黄金？测一测密度（19.3 g/cm³），是金就是真的！
- 假金（铜镀金）：铜密度 8.9 g/cm³，远小于黄金，密度不同就能识破

### 密度与材料工程

| 材料 | 密度（g/cm³） | 应用场景 |
|------|-------------|---------|
| 铝合金 | 2.7 | 飞机、汽车（轻量化） |
| 钛合金 | 4.5 | 航天器、医疗器械（强度高+轻） |
| 钢 | 7.8 | 建筑、机械 |
| 铅 | 11.3 | 辐射屏蔽、蓄电池 |
| 软木 | 0.2 | 浮标、隔热材料 |

### 密度与浮力

物体在液体中的浮沉，由物体密度与液体密度关系决定：

- $\rho_{\text{物}} < \rho_{\text{液}}$：漂浮（浮力 = 重力）
- $\rho_{\text{物}} = \rho_{\text{液}}$：悬浮（任何深度静止）
- $\rho_{\text{物}} > \rho_{\text{液}}$：下沉

**死海浮人：** 死海盐度极高，水密度约 1.24 g/cm³ > 人体密度（约 1.0 g/cm³），人漂浮不下沉。

### 密度与气体行为

**热气球原理：**
加热气球内空气 → 空气密度减小 → 气球内平均密度 < 外部空气密度 → 产生净向上浮力 → 升空

**氦气球：** 氦气密度（0.164 kg/m³）< 空气（1.29 kg/m³），浮力 > 重力，气球上升

### 密度与地球科学

- 地球内部结构：地核（铁镍，密度 10~13 g/cm³）→ 地幔（3.3 g/cm³）→ 地壳（2.8 g/cm³）
- 大陆漂移：密度较低的大陆地壳漂浮在密度较高的地幔软流层上

### 典型例题

**例1**：为什么轮船能浮在水面，而制造轮船的钢铁密度比水大得多？

**解**：轮船是空心的，整体平均密度 < 水的密度。轮船体积大，排开水的重力（浮力）≥ 轮船自重，因此漂浮。（阿基米德原理）

**例2**：如何用密度鉴别一枚金币是纯金还是合金？

**解**：
1. 用天平称出金币质量 $m$
2. 用排水法测体积 $V$
3. 计算 $\rho = m/V$
4. 若 $\rho \approx 19.3$ g/cm³，是纯金；否则为合金

**易错点：**
- 浮沉由密度**比较**决定，不是绝对大小
- 轮船能浮是因为整体平均密度低，不是钢铁本身密度低
- 同种物质密度固定，不同状态（如液态/固态）密度略有差异
