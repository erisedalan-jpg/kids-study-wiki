# 平面镜成像

> **一句话**：平面镜成的像是**正立、等大的虚像**，像与物关于镜面**对称**，像距等于物距。
> **English**: A plane mirror forms an upright, same-size virtual image; the image and object are symmetric about the mirror surface, with equal image and object distances.

---

## 📚 给 6-12 岁（自读版）

照镜子时，镜子里的"你"和真实的你有什么区别？

- 大小：一样大！
- 距离：你离镜子 1 米，镜子里的你看起来也在镜子后面 1 米处
- 左右：**左右对调**了！（你举右手，镜子里的你举的是左手）

**平面镜成像特点：**
1. 成**虚像**（看得到，但用光屏接收不到）
2. 像是**正立**的（不倒置）
3. 像和物**等大**
4. 像和物到镜面的距离**相等**（像在镜子后面相同距离处）

**虚像是什么意思？**
平面镜里的像，实际上没有光线真正到达那个位置——是光线的延长线交于那里，产生的"假"像。你用纸放在镜子后面，接收不到像，所以叫虚像。

**生活应用：**
- 梳妆台的镜子
- 汽车后视镜（凸面镜，但原理类似）
- 潜望镜（两块平面镜组合）

## 🎓 给 12+（进阶版）

### 定义

**平面镜**（Plane Mirror）：反射面为平面的镜子。

**平面镜成像特点：**
1. 成**虚像**（反射光的延长线的交点）
2. 像与物关于镜面**对称**（对称轴为镜面）
3. 像是**正立的**
4. **像距 = 物距**（到镜面的垂直距离相等）
5. **像与物大小相等**

### 作图方法

**方法1（对称法）：**
- 找物体上两个关键点
- 作这两点关于镜面的对称点
- 连接对称点得到虚像

**方法2（光线追踪法）：**
- 从物体上一点作两条射向镜面的光线
- 根据反射定律画出反射光线
- 反射光线的延长线的交点即为像点

### 平面镜成像实质

像是反射光线的**延长线**交汇处，不是实际光线。因此：
- 虚像不能在光屏上呈现
- 虚像位于镜面后方

### 左右对称问题

平面镜使物体的"前后"（垂直于镜面方向）发生反转，看起来是"左右"反转：
- 实际是：镜面将物体前后翻转
- 视觉上：观察者感受到左右互换（因为观察者视角改变）

### 应用

| 应用 | 原理 |
|------|------|
| 梳妆镜、穿衣镜 | 平面镜等大正立成像 |
| 潜望镜 | 两块平面镜各偏转 45°，折叠光路 |
| 万花筒 | 多块平面镜对称成像 |
| 激光偏转 | 平面镜改变光路方向 |

### 典型例题

**例1**：某人身高 1.8 m，站在平面镜前 1 m 处，像在哪里？像高多少？

**解**：
- 像距 = 物距 = 1 m（像在镜面后 1 m 处）
- 像高 = 物高 = 1.8 m（等大）

**例2**：探究平面镜成像实验中，为什么用玻璃板代替平面镜？

**解**：玻璃板半透明，既能成像（反射），又能透过光看到后面的蜡烛，可以直接**比较**像和物的大小，确认等大关系。

**易错点：**
- 平面镜成虚像，不能用光屏接收
- 像距 = 物距（到镜面的距离，不是到镜面背面）
- 人离平面镜走近，像也走近，但像的大小不变
