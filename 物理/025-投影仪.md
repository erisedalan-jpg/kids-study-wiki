# 投影仪

> **一句话**：投影仪利用凸透镜成像原理，物距在 $f < u < 2f$ 时，成**倒立、放大的实像**，用平面镜折叠光路后投影到幕布上。
> **English**: A projector uses a convex lens; when $f < u < 2f$, it forms an inverted, enlarged real image, which is reflected by a plane mirror onto the screen.

---

## 📚 给 6-12 岁（自读版）

教室里老师用的投影仪，能把幻灯片上的小图案投射到大白幕上——而且是放大版的！

**投影仪的原理：**
- 灯泡照亮幻灯片（透明小图案）
- 凸透镜把幻灯片图像放大，投射到幕布上
- 幕布上出现**倒立的、放大的**实像

**为什么幻灯片要倒置放？**
因为凸透镜成的实像是倒立的，所以幻灯片要倒着放，投影出来的像才是正立的，观众才能看清楚。

**投影仪的内部还有一面平面镜：**
改变光路的方向，让图像投射到前方的幕布上，而不是直射向天花板。

**调节图像大小：**
- 幕布离投影仪越远 → 图像越大
- 幕布离投影仪越近 → 图像越小

## 🎓 给 12+（进阶版）

### 定义

**投影仪**（Projector）：利用凸透镜将**物距在 $f < u < 2f$** 范围内的物体（幻灯片/显示屏）投影放大成像的光学仪器。

### 成像条件

$$f < u < 2f \Rightarrow v > 2f$$

成**倒立、放大的实像**，像距 $v > 2f$。

### 投影仪结构

| 部件 | 功能 |
|------|------|
| 光源（强灯泡） | 提供强光照亮幻灯片 |
| 幻灯片（物体） | 放在 $f < u < 2f$ 处，倒置放置 |
| 凸透镜（投影镜头） | 成倒立放大实像 |
| 平面镜 | 折叠光路，将像投射到幕布 |
| 幕布 | 接收实像，漫反射供观众观看 |

### 调节方法

| 操作 | 效果 |
|------|------|
| 幕布移远 | 像变大（但需重新对焦） |
| 幕布移近 | 像变小 |
| 调焦：移动镜头（增大 $v$） | 清晰成像 |
| 物放远（$u$ 增大，仍在范围内） | 像变小、像距减小 |

### 与照相机、放大镜的比较

| 仪器 | 物距 | 像的特征 | 用途 |
|------|------|---------|------|
| 照相机 | $u > 2f$ | 倒立缩小实像 | 记录图像 |
| 投影仪 | $f < u < 2f$ | 倒立放大实像 | 投影放大 |
| 放大镜 | $u < f$ | 正立放大虚像 | 观察细节 |

### 典型例题

**例1**：投影仪的物（幻灯片）应放在哪个范围？屏幕上得到什么像？

**解**：幻灯片放在凸透镜的 $f$ 到 $2f$ 之间（$f < u < 2f$），屏幕上得到**倒立、放大的实像**。

**例2**：投影仪投出的图像变模糊了（偏大），应如何调节？

**解**：图像偏大说明物距偏小（幻灯片离镜头太近）或像距偏大。应将镜头**靠近幻灯片**（增大物距）或**移近幕布**（减小像距），然后微调焦距使像清晰。

**易错点：**
- 投影仪幻灯片要**倒置**放置，因为凸透镜成倒立像
- 调大图像：移远幕布（增大像距），同时重新对焦
- 实像可以投在光屏上接收，这是与虚像（放大镜）的区别
