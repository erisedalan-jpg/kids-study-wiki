# 功率应用

> **一句话**：功率是描述机器工作快慢的量，决定了机器在给定时间内能做多少功，应用于交通、电力、机械等领域。
> **English**: Power describes how fast a machine works, determining how much work it can do in a given time, with applications in transportation, electricity, and machinery.

---

## 📚 给 6-12 岁（自读版）

功率决定一台机器的"能力"——同样的工作，功率大的机器完成得更快。

**常见功率数量级：**

| 设备 | 功率（近似） |
|------|-----------|
| 手机充电 | 5-65 W |
| LED 灯泡 | 8-12 W |
| 家用电风扇 | 50-100 W |
| 人爬楼梯 | 100-200 W |
| 家用空调 | 1000-3000 W (1-3 kW) |
| 家用电热水器 | 1500-3000 W |
| 普通汽车发动机 | 50,000-200,000 W (50-200 kW) |
| 高铁机车 | 约 20,000 kW |
| 三峡电站（总） | 22,500,000 kW = 22.5 GW |

**功率和速度的关系（匀速）：**
$$P = Fv$$

匀速行驶时，功率越大 → 能克服的阻力越大，或速度越快。

**汽车提速：** 满油门时功率一定（额定功率），速度越快阻力越大，加速越难。最终速度（最大速度）时：驱动力 = 阻力，加速度 = 0。

## 🎓 给 12+（进阶版）

### 功率的两个公式

$$P = \frac{W}{t} \quad \text{（平均功率）}$$

$$P = Fv \quad \text{（瞬时功率，力与速度同向时）}$$

### 额定功率与最大速度

对于恒定阻力 $f$ 的运动：

最大速度时，驱动力 = 阻力：$F_{驱} = f$

$$P_{额定} = F_{驱} \times v_{max} = f \times v_{max}$$

$$v_{max} = \frac{P_{额定}}{f}$$

**启动策略：**
- 先低速大力启动（保持额定功率：$F_{驱} = P/v$，$v$ 小时 $F_{驱}$ 大，加速快）
- 速度增大后驱动力减小，加速度减小
- 最终 $F_{驱} = f$，匀速，达到最大速度

### 人体功率

人体日常活动功率：
- 静坐：约 70-100 W
- 步行（5 km/h）：约 100-150 W
- 跑步（10 km/h）：约 400-600 W
- 骑车（20 km/h）：约 150-300 W
- 极限冲刺（短暂）：可达 1000-3000 W

人体一天的能量代谢：约 $2000\text{ kcal} = 2000 \times 4200\text{ J} = 8.4 \times 10^6\text{ J}$

平均功率：$P = 8.4 \times 10^6 / 86400 \approx 97\text{ W}$（约 100 W）

### 典型例题

**例1：** 一辆汽车额定功率 100 kW，匀速行驶时阻力 2000 N，求匀速时的速度。

解：$v = P/F = 100000/2000 = 50\text{ m/s} = 180\text{ km/h}$

**例2：** 同一辆车，以 10 m/s 行驶时的驱动力是多少？加速度是多少（车重 2000 kg）？

解：$F_{驱} = P/v = 100000/10 = 10000\text{ N}$

合力 = $F_{驱} - f = 10000 - 2000 = 8000\text{ N}$

$a = F/m = 8000/2000 = 4\text{ m/s}^2$

**例3：** 建筑工人用 300 W 的功率工作 8 小时，做了多少功？

解：$W = Pt = 300 \times 8 \times 3600 = 8,640,000\text{ J} = 8.64 \times 10^6\text{ J} = 8640\text{ kJ}$

### 易错点

- **额定功率是最大功率**，不是所有时刻的功率。
- $P = Fv$ 只在力与速度方向相同时成立，且是**瞬时**关系。
- 高速时驱动力**减小**（$F = P/v$，$v$ 大则 $F$ 小），不是说高速不需要力。
- 最大速度时加速度为零，不是功率最大。
