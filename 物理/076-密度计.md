# 密度计

> **一句话**：密度计利用漂浮原理测量液体密度：密度越大的液体，密度计浸入越浅，刻度值越大。
> **English**: A hydrometer measures liquid density using the floating principle: denser liquids cause the hydrometer to float higher, with larger scale readings.

---

## 📚 给 6-12 岁（自读版）

**密度计（比重计）** 是一根底部有重物、中间有刻度管的玻璃仪器，放入液体中漂浮，通过读取刻度来知道液体的密度。

**工作原理：**
- 密度计漂浮：浮力 = 重力（$F_{浮} = G$，不变）
- 在密度大的液体中：同样浮力需要更小的排水体积 → 密度计浸入更浅 → 刻度读数在下方（读数大）
- 在密度小的液体中：需要更大的排水体积才能漂浮 → 密度计浸入更深 → 刻度读数在上方（读数小）

**刻度特点：**
- 密度计的刻度**下大上小**（越往下数字越大，越往上数字越小）——这与温度计等仪器相反！

**生活中的密度计：**
- 酒精计（测白酒浓度）
- 电池液密度计（检查蓄电池）
- 牛奶计（检查牛奶是否掺水）
- 盐水计（腌制食品时测盐水浓度）

## 🎓 给 12+（进阶版）

### 原理推导

设密度计质量 $m$（重力 $G = mg$），底面积 $S$，在密度为 $\rho_{液}$ 的液体中浸入深度为 $h$：

漂浮条件：$F_{浮} = G$

$$\rho_{液} g V_{排} = mg$$

$$\rho_{液} g S h = mg$$

$$h = \frac{m}{\rho_{液} S}$$

**结论：** $h \propto \frac{1}{\rho_{液}}$

- $\rho_{液}$ 越大，$h$ 越小（浸入越浅，刻度线在上方，但读数越大）
- $\rho_{液}$ 越小，$h$ 越大（浸入越深，刻度线在下方，但读数越小）

**刻度标注：** 液面对应的刻度即为液体密度值。刻度值越大越在下方（因为越大的密度，浸入越浅，液面越靠上……不对，刻度在管上，液面在上意味着管子大部分露出，液面切割位置是管子下部，读数大）。

修正：在密度大的液体中，密度计浸入**浅**，液面切割的是密度计**下部刻度**（数字大）。在密度小的液体中，浸入**深**，液面切割**上部刻度**（数字小）。

### 密度计的刻度分布

由 $h = m/(\rho S)$，$\rho$ 与 $h$ 成反比，刻度分布**不均匀**：
- 下方（小 $h$，大 $\rho$）刻度密集
- 上方（大 $h$，小 $\rho$）刻度稀疏

### 典型例题

**例1：** 密度计在水（$\rho = 1000\text{ kg/m}^3$）中液面在 1.0 处，放入酒精（$\rho = 800\text{ kg/m}^3$）中，液面读数在哪里？是大于 1.0 还是小于 1.0？

解：酒精密度小于水，密度计浸入更深，液面在更高位置，对应密度计的上部刻度（数字更小）。读数为 0.8（对应 $\rho = 800\text{ kg/m}^3$）。

**例2：** 密度计在盐水中读数 1.2，在纯水中读数 1.0，盐水密度是纯水的几倍？

解：密度计读数直接对应液体密度（单位 g/cm³ 或 kg/L）。盐水密度/水密度 = 1.2/1.0 = 1.2 倍。

**例3：** 某密度计，质量 0.1 kg，截面积 $S = 1\text{ cm}^2 = 10^{-4}\text{ m}^2$，在某液体中浸入深度 $h = 0.5\text{ m}$，求该液体密度。

解：$\rho_{液} = m/（S \cdot h）= 0.1/(10^{-4} \times 0.5) = 0.1/(5 \times 10^{-5}) = 2000\text{ kg/m}^3$

### 易错点

- **密度计刻度下大上小**（与直觉相反），因为密度大的液体浸入更浅。
- 密度计漂浮时浮力始终等于重力（不变），改变的是浸入体积（排水量）。
- 密度计的刻度线**不均匀**（上稀下密），因为密度与深度是反比关系。
