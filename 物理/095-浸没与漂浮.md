# 浸没与漂浮

> **一句话**：浸没（完全浸入液体）和漂浮（部分露出液面）是物体与液体的两种接触状态，浮力计算有所不同。
> **English**: Submersion (fully immersed) and floating (partially above surface) are two states with different buoyancy calculations.

---

## 📚 给 6-12 岁（自读版）

物体在液体中有几种状态：

**漂浮（Floating）：** 物体部分露出液面，静止在液面上
- 例：木块、泡沫板浮在水面
- 特点：浮力 = 重力，$V_{排} < V_{物}$

**浸没（Submersion）：** 物体完全浸入液体中（不接触底部）
- 情况1：**悬浮**——静止在液体内部，浮力 = 重力
- 情况2：**下沉**——继续往下运动，浮力 < 重力
- 情况3：**上浮**——向上运动，浮力 > 重力

**沉底：** 物体下沉后接触容器底部，受底部支持力

**状态判断：**
```
放入液体 → 
  ρ物 < ρ液 → 上浮 → 漂浮
  ρ物 = ρ液 → 悬浮（浸没状态）
  ρ物 > ρ液 → 下沉 → 沉底
```

**浸没与漂浮时浮力的区别：**
- 浸没时：$V_{排} = V_{物}$（排开体积等于物体体积）
- 漂浮时：$V_{排} < V_{物}$（只有部分体积在液面以下）

## 🎓 给 12+（进阶版）

### 状态分析

| 状态 | $V_{排}$ | 合力 | 运动 | 条件 |
|------|---------|------|------|------|
| 漂浮 | $< V_{物}$ | 0 | 静止（液面） | $F_{浮} = G$，$\rho_{物} < \rho_{液}$ |
| 悬浮 | $= V_{物}$ | 0 | 静止（液内） | $F_{浮} = G$，$\rho_{物} = \rho_{液}$ |
| 上浮 | $= V_{物}$ | 向上 | 向上加速 | $F_{浮} > G$，过程状态 |
| 下沉 | $= V_{物}$ | 向下 | 向下加速 | $F_{浮} < G$，过程状态 |
| 沉底 | $= V_{物}$ | 0（加上支持力） | 静止（底部） | $F_{浮} + N = G$ |

### 关键区别：漂浮 vs 悬浮

| 比较项 | 漂浮 | 悬浮 |
|-------|------|------|
| 位置 | 液面，部分露出 | 液体内部，完全浸没 |
| $V_{排}$ | $< V_{物}$ | $= V_{物}$ |
| 密度关系 | $\rho_{物} < \rho_{液}$ | $\rho_{物} = \rho_{液}$ |
| 稳定性 | 较稳定（液面） | 不稳定（任何位置） |

**悬浮的不稳定性：** 密度恰好相等时，物体理论上可以在任何深度悬浮，但实际上微小扰动会导致上浮或下沉（中性浮力）。鱼通过鱼鳔主动调节来实现悬浮。

### 浮力公式的适用

**阿基米德原理** $F_{浮} = \rho_{液} g V_{排}$ 对所有状态均适用：

- 漂浮：$V_{排}$ 由漂浮条件决定（$F_{浮} = G$）
- 浸没（未沉底）：$V_{排} = V_{物}$
- 沉底：$V_{排} = V_{物}$，但此时还有支持力

### 典型例题

**例1：** 一物体（密度 $1200\text{ kg/m}^3$）放入水中，将发生什么？沉底后弹簧测力计在水中示数为多少？（物体质量 120 g）

解：$\rho_{物} > \rho_{水}$，物体下沉至沉底。

$V = m/\rho_{物} = 0.12/1200 = 10^{-4}\text{ m}^3$

$F_{浮} = \rho_{水} g V = 1000 \times 10 \times 10^{-4} = 1\text{ N}$

$G = mg = 0.12 \times 10 = 1.2\text{ N}$

弹簧测力计示数（视重）= $G - F_{浮} = 1.2 - 1 = 0.2\text{ N}$

底部对物体支持力 = 0.2 N（向上）

**例2：** 区分下列情况：①铁球沉底，②小鱼悬停在水中，③木块浮在水面。哪种情况的浮力等于重力？

解：
- 铁球沉底：$F_{浮} < G$（差额由地面支持力补足），浮力 ≠ 重力
- 小鱼悬停：$F_{浮} = G$（悬浮），浮力 = 重力
- 木块漂浮：$F_{浮} = G$（漂浮），浮力 = 重力

### 易错点

- **沉底的物体，浮力不等于重力**：$F_{浮} + N = G$（多一个支持力）。
- **漂浮和悬浮浮力都等于重力**，但位置和排水体积不同。
- 上浮是一个动态过程（$F_{浮} > G$），不是稳定状态；漂浮才是上浮结束后的稳定状态。
