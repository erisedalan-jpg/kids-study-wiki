# 滑轮组

> **一句话**：滑轮组由定滑轮和动滑轮组合而成，可以同时省力和改变力的方向，拉力 $F = G/n$（$n$ 为承担重力的绳段数）。
> **English**: A pulley system combines fixed and movable pulleys to both save force and change direction; F = G/n where n is the number of rope segments supporting the load.

---

## 📚 给 6-12 岁（自读版）

把定滑轮和动滑轮组合在一起，就是**滑轮组**——既能省力，又能改变力的方向！

**滑轮组的规律：**

有几段绳子承担重物，拉力就是重力的几分之一：

$$F = \frac{G}{n}$$

（$n$ = 承担物体重力的绳子段数）

**怎么数 $n$？**
从动滑轮出发，数连接动滑轮和物体的绳子段数（不算绳子的固定端）。

**例子（3 段绳子）：**
- $F = G/3$：只需要重力的三分之一，省力 3 倍
- 但绳子要拉 $3 \times h$ 的长度（费距离 3 倍）

**滑轮组的优点：**
1. 省力（多少段绳子，省多少倍）
2. 可以改变力的方向（通过定滑轮）

**建筑工地、起重机上都使用滑轮组！**

## 🎓 给 12+（进阶版）

### 拉力公式

理想滑轮组（无摩擦，忽略动滑轮自重）：

$$F = \frac{G}{n}$$

其中 $n$ 是承担物体重力的绳段数（动滑轮两侧各段绳子之和）。

实际滑轮组（需克服摩擦和动滑轮自重）：

$$F > \frac{G}{n}$$

### 绳子的移动距离

$$s_{绳} = n \times h$$

（物体上升 $h$，绳子移动 $nh$）

### 机械效率

$$\eta = \frac{W_{有用}}{W_{总}} = \frac{G \times h}{F \times s} = \frac{G \times h}{F \times nh} = \frac{G}{nF}$$

理想情况：$\eta = G/(n \times G/n) = 100\%$

实际情况：$\eta < 100\%$（有摩擦和动滑轮自重）

### 绳段数的判断

**计数方法：**
从动滑轮开始，数连接在动滑轮上的绳段：
- 每个动滑轮被 2 段绳子托住
- $k$ 个动滑轮：$n = 2k$（固定端挂在天花板）或 $n = 2k + 1$（固定端有额外一段）

**常见配置：**
- 1 个动滑轮 + 1 个定滑轮：$n = 2$，$F = G/2$（与单动滑轮相同，但改变了方向）
- 2 个动滑轮 + 2 个定滑轮：$n = 4$，$F = G/4$

### 考虑动滑轮自重

设动滑轮组总重 $G_0$，实际拉力：

$$F = \frac{G + G_0}{n}$$

（动滑轮和物体都需要绳子来支撑）

### 典型例题

**例1：** 一个滑轮组有 4 段绳子承担重力，提升重 400 N 的物体（忽略摩擦和动滑轮重），需要多大拉力？物体上升 1 m，绳子移动多少？

解：$F = G/n = 400/4 = 100\text{ N}$（省力 4 倍）

$s = nh = 4 \times 1 = 4\text{ m}$（费距离 4 倍）

**例2：** 上题中，动滑轮总重 40 N，实际拉力是多少？

解：$F = (G + G_0)/n = (400 + 40)/4 = 110\text{ N}$

**例3：** 实际中用 120 N 拉力提升 400 N 物体（4 段绳，物体升 1 m），机械效率是多少？

解：$W_{有用} = 400 \times 1 = 400\text{ J}$

$W_{总} = 120 \times 4 = 480\text{ J}$

$\eta = 400/480 \approx 83.3\%$

### 滑轮组设计

实际设计滑轮组时，选择 $n$ 的原则：
- $n$ 越大 → 越省力 → 但绳子更长，效率更低（更多摩擦）
- 需要根据实际场景（拉力限制、操作方便性）选择合适的 $n$

### 易错点

- 数绳段要**只数承担动滑轮（和物体）的绳段**，不算固定端部分（固定端绳子张力为零）。
- 滑轮组省力不省功（功的原理）。
- 实际滑轮组效率受摩擦和动滑轮自重影响，$F > G/n$。
