# s-t 图（位移-时间图象）

> **一句话**：s-t 图以时间为横轴、路程（位移）为纵轴，图线的斜率表示速度，是分析运动状态的基本工具。
> **English**: An s-t graph plots displacement (or distance) on the vertical axis against time on the horizontal axis; the slope of the line represents velocity.

---

## 📚 给 6-12 岁（自读版）

想知道一辆车怎么跑的？用 **s-t 图**（路程-时间图）一眼看清！

**s-t 图的坐标：**
- 横轴（x 轴）：时间 $t$（单位：s）
- 纵轴（y 轴）：路程 $s$（单位：m）

**不同线的含义：**

| 图线形状 | 运动状态 |
|---------|---------|
| 水平直线（斜率=0） | 静止（路程不变） |
| 向上倾斜的直线 | 匀速直线运动（速度恒定） |
| 斜率越大 | 速度越快 |
| 向上弯曲的曲线 | 加速运动（越来越快） |
| 向下弯曲的曲线 | 减速运动（越来越慢） |

**读图技巧：**
- 两点之间：$v = \Delta s / \Delta t$（斜率就是速度！）
- 图线越陡，速度越大
- 两条线相交：两物体在同一时刻到达同一位置

**例题：**

小明步行上学，前 200 m 用了 100 s，然后停下等红灯 30 s，再用 50 s 走完后 100 m。

- 第一段：$v_1 = 200/100 = 2\text{ m/s}$（s-t 图向上倾斜）
- 第二段：$v_2 = 0$（s-t 图水平线）
- 第三段：$v_3 = 100/50 = 2\text{ m/s}$（s-t 图再次向上倾斜，斜率与第一段相同）

## 🎓 给 12+（进阶版）

### s-t 图的数学本质

**斜率 = 速度：**

$$v = \frac{\Delta s}{\Delta t} = \text{图线的斜率}$$

对于直线：斜率恒定 → 匀速运动
对于曲线：斜率在变 → 变速运动（切线斜率 = 瞬时速度）

### 典型 s-t 图分析

**匀速直线运动：**

$$s = vt \quad \text{（正比函数，过原点的直线）}$$

斜率 $k = v$，图线越陡速度越大。

**静止：**

$$s = s_0 = \text{常数} \quad \text{（水平直线）}$$

**初中常见运动的 s-t 图对比：**

| 运动类型 | s-t 图特征 | 斜率特征 |
|---------|-----------|---------|
| 静止 | 水平直线 | 斜率 = 0 |
| 匀速运动 | 直线（不水平） | 斜率恒定（= 速度） |
| 匀加速运动 | 上凸曲线（开口向上抛物线） | 斜率逐渐增大 |
| 匀减速运动 | 下凸曲线（斜率减小） | 斜率逐渐减小 |

### 从 s-t 图求速度

**方法1（直线）：** 取任意两点 $(t_1, s_1)$ 和 $(t_2, s_2)$：

$$v = \frac{s_2 - s_1}{t_2 - t_1}$$

**方法2（曲线）：** 在某点作切线，切线的斜率即为该点的瞬时速度（初中一般只要求定性判断）。

### 多物体 s-t 图比较

**两物体同时出发：**
- 斜率大的（图线陡）速度大
- 两线相交：两物体在该时刻位置相同（相遇）

**追及问题：**
- 两线相交点即为追上的时刻
- 若一物体从后出发追另一物体：起点不同，追上时 s 值相同

### 典型例题

**例1：** 读 s-t 图：两辆汽车 A、B 从同一地点出发。A 的图线斜率为 30，B 的图线斜率为 20。求：两者速度各为多少？何时 A 比 B 多走 100 m？

解：$v_A = 30\text{ m/s}$，$v_B = 20\text{ m/s}$

$\Delta s = v_A t - v_B t = (30-20)t = 10t = 100\text{ m} \Rightarrow t = 10\text{ s}$

**例2：** 一物体 s-t 图上，在 $t=0\sim5\text{ s}$ 内 $s$ 从 0 增到 10 m，在 $t=5\sim10\text{ s}$ 内 $s$ 保持 10 m。描述运动过程。

解：前 5 s 匀速运动，$v = 10/5 = 2\text{ m/s}$；后 5 s 静止。

**例3：** s-t 图上有一段下降的直线（$s$ 随 $t$ 增大而减小），代表什么运动？

解：物体向出发点方向匀速返回（速度方向相反）。斜率为负值，$|v| = |\text{斜率}|$。

### 与 v-t 图的关系

| 信息 | 从 s-t 图获取 | 从 v-t 图获取 |
|------|-------------|-------------|
| 速度 | 图线斜率 | 纵轴读数 |
| 位移 | 纵轴读数 | 图线下面积 |
| 加速度 | 需看斜率变化（定性） | 图线斜率 |

### 易错点

- **s-t 图斜率 = 速度**（不是加速度！加速度要用 v-t 图的斜率）
- s-t 图中图线不能"向后折"（时间不可逆），但可以下降（往回走）
- s-t 图不反映路径弯直——只反映路程（或位移大小）随时间的变化
