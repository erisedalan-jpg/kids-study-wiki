# 静摩擦力

> **一句话**：静摩擦力是接触面相对静止但有相对运动趋势时产生的摩擦力，大小随外力变化，最大值等于最大静摩擦力。
> **English**: Static friction acts between surfaces at rest relative to each other when there is a tendency for motion; it adjusts to balance applied forces up to a maximum value.

---

## 📚 给 6-12 岁（自读版）

你用力推一个很重的冰箱，冰箱没动——这时候有没有摩擦力？

**有！** 虽然冰箱没动，但地面给冰箱一个摩擦力，阻止它滑动。这种"物体静止时"的摩擦力，就叫**静摩擦力**。

**静摩擦力的特点：**
- 只在物体**静止**时（或有滑动趋势时）存在
- 大小会随外力变化——外力越大，静摩擦力越大（但有上限）
- 方向与物体运动趋势相反

**例子：**
- 用 3 N 推冰箱，冰箱不动 → 静摩擦力 = 3 N（方向与推力相反）
- 用 5 N 推冰箱，冰箱还不动 → 静摩擦力 = 5 N
- 用 20 N 推冰箱，冰箱开始滑动 → 说明最大静摩擦力不到 20 N

一旦物体开始滑动，摩擦力就变成**滑动摩擦力**（通常比最大静摩擦力略小）。

## 🎓 给 12+（进阶版）

### 定义

静摩擦力（Static Friction）：两个相互接触的物体相对静止，但有相对运动趋势时，接触面产生的阻碍相对运动趋势的力。

### 大小范围

$$0 \leq f_{静} \leq f_{max}$$

- 静摩擦力大小由平衡条件决定（等于使物体运动所需的力），可以是 0 到最大值之间的任意值
- **最大静摩擦力**：$f_{max} \approx \mu_s N$（$\mu_s$ 为静摩擦因数，略大于动摩擦因数 $\mu_k$）
- 初中阶段，最大静摩擦力近似等于滑动摩擦力

### 方向判断

静摩擦力方向：与物体**运动趋势方向相反**（阻碍趋势，不是阻碍运动）。

判断步骤：
1. 先判断物体如果没有摩擦力会向哪个方向运动（即运动趋势）
2. 静摩擦力方向与趋势方向相反

### 静摩擦力可以是"推动力"

走路时，脚向后蹬地（脚对地面有向后的力），地面对脚有向前的摩擦力——这是**推动我们前进的力**，但它是静摩擦力（脚没有相对地面滑动）。

### 典型例题

**例1：** 一个质量 10 kg 的木块（$\mu = 0.4$）放在水平面上，用 30 N 的力水平推，木块不动。静摩擦力是多少？

解：木块静止，合力为零。静摩擦力 = 外力 = 30 N（方向与推力相反）。

验证：最大静摩擦力 ≈ $\mu N = 0.4 \times 100 = 40\text{ N} > 30\text{ N}$，确实不动。

**例2：** 斜面上一个物体（重 50 N，斜面倾角 30°）静止，斜面的静摩擦力是多少？

解：沿斜面方向平衡。重力沿斜面分量 = $G\sin 30° = 50 \times 0.5 = 25\text{ N}$（向下）。

静摩擦力 = 25 N，方向沿斜面向上。

### 易错点

- **静止物体也有摩擦力**：只要有相对运动趋势。
- **静摩擦力大小不固定**：随外力变化，不能直接用 $\mu N$ 计算（只能用于计算最大值）。
- **静摩擦力不一定阻碍物体运动**：走路时地面静摩擦力是推动力。
- 当外力超过最大静摩擦力，物体开始滑动，转变为滑动摩擦力。
