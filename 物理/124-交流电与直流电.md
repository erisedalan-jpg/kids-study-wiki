# 交流电与直流电

> **一句话**：直流电（DC）的方向和大小不变，而交流电（AC）的方向和大小随时间周期性变化；家用电是 220V/50Hz 交流电，电池是直流电。
> **English**: Direct current (DC) flows in one constant direction, while alternating current (AC) periodically reverses direction and changes magnitude; household electricity is 220V/50Hz AC, while batteries provide DC.

---

## 📚 给 6-12 岁（自读版）

**电流有两种"性格"！**

**直流电（DC）：**
- 电流方向永远不变（像一条单行道）
- 大小也基本不变
- 来源：电池、太阳能电池板、充电宝
- 应用：手机、电脑、LED灯、遥控车

**交流电（AC）：**
- 电流方向不停地反复变换（像来回摆动的秋千）
- 大小也随时间变化（正弦波）
- 来源：发电机、家用插座
- 应用：家用电器、空调、冰箱、洗衣机

**生活中怎么区分？**
- 插电源（插座）→ 交流电（220V）
- 用电池（AA、手机电池）→ 直流电

## 🎓 给 12+（进阶版）

### 直流电（DC）

**特征**：电流方向不随时间变化，大小恒定（或仅有小幅波动）。

$$I = \text{常数}，\quad U = \text{常数}$$

**来源**：
- 化学电池（干电池、铅酸蓄电池、锂离子电池）
- 直流发电机（有换向器，将交流整流为直流）
- 整流器（将交流电整流为直流）
- 太阳能光伏电池

### 交流电（AC）

**特征**：电流方向随时间周期性变化，大小按正弦规律变化。

$$u = U_0 \sin\omega t = U_0 \sin(2\pi f t)$$
$$i = I_0 \sin\omega t$$

- $U_0$（或 $I_0$）：峰值电压（电流）
- $f$：频率（Hz）
- $T = 1/f$：周期（s）

**中国家用电网标准**：220 V（有效值）、50 Hz

### 有效值与峰值

交流电的大小用**有效值**（均方根值，RMS）表示，与直流电"等效"（产生相同热量）：

$$U_{有效} = \frac{U_0}{\sqrt{2}} \approx 0.707 U_0$$
$$I_{有效} = \frac{I_0}{\sqrt{2}} \approx 0.707 I_0$$

**中国家用电**：有效值 220V，峰值约 311V

### 交流电与直流电的比较

| | 直流电（DC） | 交流电（AC） |
|--|------------|------------|
| 方向 | 不变 | 周期性变化 |
| 大小 | 恒定 | 按正弦变化 |
| 来源 | 电池、整流器 | 发电机、电网 |
| 远距离输电 | 困难（损耗大） | 容易（可升压变压） |
| 应用 | 电子设备、直流电动机 | 家用电器、工业用电 |

### 为什么家用是交流电？

1. **变压方便**：交流电可以通过变压器升高/降低电压
2. **高压输电减少损耗**：升压→长途输电→降压→用户（$P_{损} = I^2R$，升压降流，减少损耗）
3. **发电机直接产生交流电**：无需额外整流，效率高

### 典型例题

**例**：中国家用电有效值为 220V，频率 50Hz。求：①峰值电压；②周期；③写出电压表达式。

$$U_0 = \sqrt{2} \times 220 \approx 311 \text{ V}$$

$$T = \frac{1}{f} = \frac{1}{50} = 0.02 \text{ s}$$

$$u = 311\sin(100\pi t) \text{ V}$$

**易错点：**
- "220V家用电"中的220V是**有效值**，不是峰值（峰值约311V）
- 交流电的频率是50Hz（不是100Hz，虽然一周期内电流方向变了2次）
- 电器额定功率计算用有效值：$P = UI = U^2/R = I^2R$
