# 安培定则

> **一句话**：安培定则（右手定则）是判断通电导线和螺线管磁场方向的规则：右手握住导线或螺线管，拇指指向电流方向，四指弯曲方向即为磁感线方向。
> **English**: Ampere's right-hand rule determines the direction of the magnetic field around a current-carrying wire or solenoid: wrap the right hand around the conductor with the thumb pointing in the current direction; the fingers curl in the direction of the magnetic field lines.

---

## 📚 给 6-12 岁（自读版）

**怎么知道通电导线周围的磁场方向？**

用**右手定则**！

**直导线的磁场：**

右手握住通电直导线：
- **大拇指**朝向**电流方向**（电流从哪里流向哪里）
- **四指弯曲的方向**就是磁感线环绕的方向（顺时针还是逆时针）

**螺线管（通电线圈）的磁场：**

右手握住螺线管：
- **四指弯向电流流过线圈的方向**
- **大拇指指向的就是N极**！

**记忆口诀：** 右手—安培定则（判断磁场方向）；左手—安培力定则（判断受力方向）

## 🎓 给 12+（进阶版）

### 安培定则的两种应用

**情形一：直线电流的磁场**

通电直导线周围产生同心圆形磁场：

右手握住导线，**大拇指指向电流方向（正方向）**，弯曲的四指所指方向即为**磁感线方向**。

- 电流向上流动 → 磁感线绕导线**逆时针**（从上方看）
- 电流向下流动 → 磁感线绕导线**顺时针**（从上方看）

**情形二：螺线管（通电线圈）的磁场**

通电螺线管产生类似条形磁铁的磁场（两端有N极和S极）：

右手握住螺线管，**四指弯向线圈中电流的绕行方向**（从一端看，电流方向），**大拇指所指就是N极**。

### 安培定则与安培力定则的区分

| | 安培定则（右手） | 安培力定则（左手） |
|--|----------------|-----------------|
| 别名 | 右手螺旋定则 | 左手定则 |
| 用途 | 判断**磁场方向** | 判断**受力方向** |
| 输入 | 电流方向 → 输出磁场方向 | 磁场方向 + 电流方向 → 输出力的方向 |
| 应用 | 电磁铁磁极判断 | 电动机转向判断 |

### 直线电流磁场的定量描述

直线电流在距离 $r$ 处产生的磁感应强度（毕奥-萨伐尔定律，高中/大学）：

$$B = \frac{\mu_0 I}{2\pi r}$$

- $\mu_0 = 4\pi \times 10^{-7}$ T·m/A（真空磁导率）
- $I$：电流（A）
- $r$：到导线的距离（m）

初中阶段只需了解：**电流越大、距导线越近，磁场越强**。

### 螺线管内部磁场

螺线管内部磁场近似均匀：

$$B = \mu_0 n I$$

- $n$：单位长度匝数（匝/m）
- $I$：电流（A）

**磁场强弱与 $nI$（安培匝数）成正比**，这与[[178-电磁铁|电磁铁]]词条中"增大电流或匝数都能增强磁性"一致。

### 典型例题

**例1**：一根竖直导线，电流从下向上流动，在导线右侧，磁感线方向如何？

**解**：右手握住导线，大拇指朝上（电流向上），弯曲的四指在导线右侧指向**纸面外**。
→ 导线右侧的磁感线方向：从纸面向外（⊙）

**例2**：一个螺线管，从左端看线圈中的电流是逆时针的，哪端是N极？

**解**：右手握螺线管，四指沿线圈电流方向弯曲（逆时针→从左端看），大拇指指向**左端**。
→ 左端为N极。

**易错点：**
- 安培定则（右手）≠ 左手定则（安培力）——两者功能完全不同
- 螺线管中：大拇指指N极（四指指电流绕行方向）
- 直导线中：大拇指指电流方向（四指指磁感线绕行方向）——注意两种情形中拇指和四指的"角色"对调了
