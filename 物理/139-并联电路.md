# 并联电路

> **一句话**：并联电路中各元件两端接在相同两点之间，有多条电流路径；各支路电压相等，总电流等于各支路电流之和，等效电阻小于最小分电阻。
> **English**: In a parallel circuit, components share the same two endpoints with multiple current paths; voltage is equal across all branches, total current equals the sum of branch currents, and equivalent resistance is less than any single branch resistance.

---

## 📚 给 6-12 岁（自读版）

**并联**就是把元件"并排"连接，给电流提供多条"岔路"。

**特点：**
- 有**多条路**让电流流动
- 其中一条断了，其他支路**不受影响**（家里某个灯坏了，其他灯还亮着！）

**并联电路的三个规律：**

1. **电压处处相同**：$U_1 = U_2 = U$
   （多条路，但起点和终点一样，所以电压相同）

2. **电流分配**：$I = I_1 + I_2$
   （总电流等于各支路电流之和；电阻小的支路分流更多）

3. **等效电阻**：$\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}$
   （并联后总电阻变小，比任何一个单独的电阻都小！）

**生活例子：**
- 家里的插座（并联，每个插座电压都是220V）
- 家里的各个灯（并联，互不干扰）

## 🎓 给 12+（进阶版）

### 并联电路的基本规律

设有 $n$ 个支路并联，各支路电阻为 $R_1, R_2, \ldots, R_n$：

**电压规律（并联各支路电压相等）：**

$$U = U_1 = U_2 = \cdots = U_n$$

**电流规律（并联电流分配）：**

$$I = I_1 + I_2 + \cdots + I_n$$

**等效电阻（并联）：**

$$\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2} + \cdots + \frac{1}{R_n}$$

对于两个电阻并联：

$$R = \frac{R_1 R_2}{R_1 + R_2}$$

并联总电阻小于任何一个分电阻！

### 并联电路的电流分配

由欧姆定律：$I_i = \frac{U}{R_i}$

因为 $U$ 相同，所以：

$$\frac{I_1}{I_2} = \frac{R_2}{R_1}$$

**并联电路中电阻小的支路分流更多（电阻与电流成反比）。**

### 并联电路的功率

各元件功率：$P_i = \frac{U^2}{R_i}$（电压相同）

$$\frac{P_1}{P_2} = \frac{R_2}{R_1}$$

并联时阻值小的元件功率更大。

### 典型例题

**例**：$R_1 = 6$ Ω 和 $R_2 = 12$ Ω 并联后接在 6 V 电源上。求：①等效电阻；②干路电流；③各支路电流。

**解**：

① $\frac{1}{R} = \frac{1}{6} + \frac{1}{12} = \frac{2}{12} + \frac{1}{12} = \frac{3}{12} = \frac{1}{4}$

$R = 4$ Ω（小于 $R_1$ 和 $R_2$）

② $I = \frac{U}{R} = \frac{6}{4} = 1.5$ A

③ $I_1 = \frac{U}{R_1} = \frac{6}{6} = 1$ A

$I_2 = \frac{U}{R_2} = \frac{6}{12} = 0.5$ A

验证：$I_1 + I_2 = 1 + 0.5 = 1.5$ A ✓

**易错点：**
- 并联总电阻一定小于每个分电阻（不是大）
- 并联增加支路后，总电阻减小，总电流增大（各支路电流不变，因为电压不变）
- 并联各支路独立，互不影响（某支路断路只影响该支路）
