# 欧姆定律并联电路

> **一句话**：并联电路中各支路电压相等，总电流等于各支路电流之和，等效电阻倒数等于各支路电阻倒数之和。
> **English**: In parallel circuits, voltage is equal across all branches; total current equals the sum of branch currents; reciprocal of equivalent resistance equals sum of reciprocals of branch resistances.

---

## 📚 给 6-12 岁（自读版）

**并联电路的三大规律（记牢！）：**

1. 电压处处相等：$U = U_1 = U_2$
2. 电流相加：$I = I_1 + I_2$
3. 电阻（倒数相加）：$\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}$

**解题公式套路：**

先求各支路电流 $I_i = U/R_i$，再相加得总电流。

**例题：**

$R_1 = 6$ Ω，$R_2 = 12$ Ω，总电压 12 V

$$I_1 = 12/6 = 2 \text{ A},\quad I_2 = 12/12 = 1 \text{ A}$$

$$I = I_1 + I_2 = 3 \text{ A}$$

$$R = U/I = 12/3 = 4 \text{ Ω（小于R}_1\text{和R}_2\text{）}$$

## 🎓 给 12+（进阶版）

### 并联电路的完整规律

**电压**：$U_1 = U_2 = \cdots = U_n = U$

**电流**：$I = I_1 + I_2 + \cdots + I_n$

其中 $I_i = U/R_i$

**电阻**：$\dfrac{1}{R} = \dfrac{1}{R_1} + \dfrac{1}{R_2} + \cdots + \dfrac{1}{R_n}$

两个电阻并联化简：$R = \dfrac{R_1 R_2}{R_1 + R_2}$

**分流公式**：$I_i = I \cdot \dfrac{R}{R_i}$

对两支路：$\dfrac{I_1}{I_2} = \dfrac{R_2}{R_1}$（电流与电阻成反比）

**功率**：$P_i = U^2/R_i$，$\dfrac{P_1}{P_2} = \dfrac{R_2}{R_1}$（并联中阻值小的功率大）

### 并联电路解题策略

**情景一：已知 $R_1, R_2, U$，求 $I_1, I_2, I, R_{等效}$**

$$I_1 = \frac{U}{R_1}, \quad I_2 = \frac{U}{R_2}, \quad I = I_1 + I_2, \quad R = \frac{U}{I}$$

**情景二：已知 $R_1, I_1, I$，求 $U, R_2, I_2$**

$$U = I_1 R_1, \quad I_2 = I - I_1, \quad R_2 = \frac{U}{I_2}$$

**情景三：已知 $R_1, R_2, I$，求 $U$**

$$R = \frac{R_1 R_2}{R_1+R_2}, \quad U = IR$$

### 并联的应用

1. **家用电器**：每个电器都并联在220V上，互不干扰
2. **分流器**：测大电流时，电流表并联小电阻（分流）
3. **减小总电阻**：需要更小等效电阻时，将电阻并联

### 典型例题

**例（综合）**：两灯泡并联，$L_1$ 额定参数 220 V、100 W，$L_2$ 额定参数 220 V、60 W，并联接在220 V电源上。求各灯的电阻和工作电流，并说明哪个灯更亮。

**解**：

$$R_1 = \frac{U^2}{P_1} = \frac{220^2}{100} = 484 \text{ Ω}$$

$$R_2 = \frac{U^2}{P_2} = \frac{220^2}{60} \approx 807 \text{ Ω}$$

$$I_1 = \frac{U}{R_1} = \frac{220}{484} \approx 0.45 \text{ A}$$

$$I_2 = \frac{U}{R_2} = \frac{220}{807} \approx 0.27 \text{ A}$$

100 W 灯泡更亮（功率更大，阻值更小，电流更大）。

**易错点：**
- 并联中各支路电压相等（不需要"分压"）
- 并联总电阻小于任何一个分支电阻
- 并联功率：$P_i = U^2/R_i$（电压相同，用这个更方便）；而不是 $P_i = I^2 R_i$
