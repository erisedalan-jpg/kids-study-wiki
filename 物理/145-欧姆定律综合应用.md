# 欧姆定律综合应用

> **一句话**：综合运用欧姆定律和串并联规律，解决含多个元件、多种连接方式的复杂电路计算题。
> **English**: Comprehensively apply Ohm's law and series/parallel rules to solve complex circuit problems with multiple components and mixed connection types.

---

## 📚 给 6-12 岁（自读版）

学了欧姆定律，就能解决各种电路题！

**解题四步法：**
1. **读题画图**：把电路画出来，标清楚元件
2. **判断结构**：哪些元件串联？哪些并联？
3. **找已知量**：哪些量已经知道？
4. **逐步计算**：从已知量出发，一步一步求未知量

**常见题型：**
- 混联电路（有串有并）
- 已知部分量求其他量
- 有滑动变阻器的调节问题

## 🎓 给 12+（进阶版）

### 综合解题策略

**步骤一：电路结构分析**
- 从电源出发，追踪电流路径
- 找出节点（电流分叉处）
- 判断各段的串并联关系

**步骤二：等效化简**
- 将并联部分替换为等效电阻
- 再处理剩余串联部分
- 直到整个电路化为最简单的形式

**步骤三：逆推各量**
- 求总电流 $I = U_{总}/R_{总}$
- 求各段电压（串联用 $U_i = IR_i$，并联用 $U$ 相等）
- 求各支路电流（并联用 $I_i = U/R_i$）

**步骤四：验证**
- 串联：电压之和 = 总电压
- 并联：电流之和 = 干路电流
- 功率之和 = 总功率

### 典型综合例题

**例1（混联电路）**：

$R_1 = 4$ Ω 与 $(R_2 = 6$ Ω 串联 $R_3 = 12$ Ω)并联，总电压 $U = 16$ V。

求：总电阻、干路电流、$R_1$ 两端电压、$R_2$ 和 $R_3$ 的电流。

**解**：

$R_{23} = \frac{R_2 R_3}{R_2+R_3} = \frac{6 \times 12}{18} = 4$ Ω

总电阻：$R = R_1 + R_{23} = 4 + 4 = 8$ Ω

干路电流：$I = U/R = 16/8 = 2$ A

$R_1$ 两端电压：$U_1 = IR_1 = 2 \times 4 = 8$ V

$R_{23}$ 两端电压：$U_{23} = U - U_1 = 16 - 8 = 8$ V

$I_2 = U_{23}/R_2 = 8/6 \approx 1.33$ A；$I_3 = U_{23}/R_3 = 8/12 \approx 0.67$ A

验证：$I_2 + I_3 = 2$ A = 干路电流 ✓

**例2（含滑动变阻器）**：

$R_0 = 5$ Ω 与滑动变阻器 $R$ 串联，总电压 15 V。

① 当 $R = 10$ Ω 时，求干路电流和 $R_0$ 两端电压；

② 若 $R_0$ 两端电压为 6 V，求此时的 $R$。

**解**：

① $I = U/(R_0+R) = 15/15 = 1$ A；$U_0 = IR_0 = 5$ V

② $I' = U_0'/R_0 = 6/5 = 1.2$ A；$U_R = U - U_0' = 9$ V；$R = U_R/I' = 9/1.2 = 7.5$ Ω

**例3（已知部分量求全部）**：

两电阻串联，$U_1 = 6$ V，$U_2 = 9$ V，$R_1 = 30$ Ω。求 $I, R_2, U_{总}$。

$$I = U_1/R_1 = 6/30 = 0.2 \text{ A}$$

$$R_2 = U_2/I = 9/0.2 = 45 \text{ Ω}$$

$$U_{总} = U_1 + U_2 = 15 \text{ V}$$

### 重要技巧汇总

| 情况 | 技巧 |
|------|------|
| 串联求电流 | 先求总电阻，再 $I = U_{总}/R_{总}$ |
| 串联求某段电压 | $U_i = I \cdot R_i$（I已知） |
| 并联求某支路电流 | $I_i = U/R_i$（U已知） |
| 混联 | 先化简并联为等效电阻，再处理串联 |
| 已知功率求电阻 | $R = U^2/P$（知U）或 $R = P/I^2$（知I） |

**易错点：**
- 混联中不能把总电压直接用于某一支路（需先确定该支路两端电压）
- 有时题目中某元件两端电压为0，说明它短路了，不是忘写了
- 串联电路中某元件电阻增大→总电阻增大→总电流减小→其他元件两端电压也减小（联动效应）
