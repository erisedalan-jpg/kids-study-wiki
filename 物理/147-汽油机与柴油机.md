# 汽油机与柴油机

> **一句话**：汽油机用电火花点火，柴油机靠高压压缩自燃，两者都是四冲程内燃机，但燃料、点火方式和适用场景不同。
> **English**: Gasoline engines use spark ignition while diesel engines use compression ignition; both are four-stroke internal combustion engines but differ in fuel, ignition, and applications.

---

## 📚 给 6-12 岁（自读版）

你坐过小汽车吗？小汽车一般用**汽油机**；大卡车、公交车一般用**柴油机**。

**汽油机的特点：**
- 燃料：**汽油**（闻起来有股气味，容易挥发）
- 点火：用**电火花**引燃（发动机里有"火花塞"）
- 声音：运转比较轻柔
- 适合：小轿车、摩托车

**柴油机的特点：**
- 燃料：**柴油**（比汽油重，挥发性低）
- 点火：不用火花塞！靠**高压压缩**让气体温度升高到柴油燃点（约350°C），柴油自动燃烧
- 声音：相对轰鸣（噪音稍大）
- 适合：卡车、轮船、拖拉机、发电机

**关键区别：**
汽油机 = 先混合再点火；柴油机 = 先压缩再喷油自燃

## 🎓 给 12+（进阶版）

### 对比表

| 比较项目 | 汽油机 | 柴油机 |
|---------|--------|--------|
| 燃料 | 汽油（辛烷值高） | 柴油（十六烷值高） |
| 进气 | 混合气（空气+汽油蒸气） | 纯空气 |
| 压缩比 | 8~12（低） | 14~22（高） |
| 点火方式 | 火花塞电点火 | 压燃（柴油喷入高温高压空气自燃） |
| 燃烧温度 | 约2000-2500°C | 约1800-2000°C |
| 热效率 | 25-30% | 30-45% |
| 功率密度 | 较高（转速高） | 较低（转速低，但扭矩大） |
| 排放 | CO、HC较多 | 碳烟（PM）、NOx |
| 噪音 | 较小 | 较大（"哒哒"声） |
| 应用 | 小轿车、摩托、小型飞机 | 卡车、轮船、火车、拖拉机、发电机 |

### 柴油机压燃原理（重点）

柴油机压缩冲程：
1. 活塞向上压缩纯空气（压缩比高达14-22）
2. 气体温度急剧升高（可达500-700°C）
3. 超过柴油自燃点（~350°C）
4. 此时喷油嘴喷入雾化柴油 → 自动燃烧

**这正是做功改变内能的典型应用：** 压缩做功 → 温度升高 → 达到自燃点

### 压缩比的意义

$$压缩比 = \frac{气缸最大容积}{气缸最小容积（上止点）}$$

压缩比越高 → 气体压缩越强 → 温度升越高 → 燃烧更完全 → 效率越高

柴油机压缩比高，所以效率更高；但汽油机若压缩比过高会导致"爆震"（提前点火），故需限制。

### 典型例题

**例**：柴油机不需要火花塞，为什么柴油还是能被点燃？

**解**：柴油机压缩冲程结束时，气缸内纯空气被压缩至约 1/20（压缩比~20），此时气体压力达数十个大气压，温度升至500-700°C，远超柴油的自燃点（~350°C）。此时喷入雾化柴油，柴油与高温空气接触立即自燃。这是做功改变内能的典型实例：活塞压缩气体做功，使内能（温度）急剧增加。

**易错点：**
- 柴油机进气冲程吸入的是纯空气，不是混合气
- 汽油机进气冲程吸入的是汽油与空气的混合气
- 柴油机效率高于汽油机，但并非说柴油机"更好"，要根据用途选择
