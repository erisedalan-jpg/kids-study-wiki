# 物理-初中综合

> **一句话**：初中物理围绕"力、热、电、光、声"五大领域展开，以基本概念为核心，以实验为基础，以定量规律为目标，是理解自然世界的基础工具。
> **English**: Junior high school physics covers five domains — mechanics, thermodynamics, electricity, optics, and acoustics — building from basic concepts through experiments to quantitative laws as tools for understanding the natural world.

---

## 📚 给 6-12 岁（自读版）

**物理是什么？**

物理学研究世界万物的规律——为什么苹果会落下来？为什么灯泡会发光？为什么声音能传到耳朵？

**初中物理的五大世界：**

🔊 **声学**（七年级）
- 声音是振动产生的，需要介质传播
- 音调、响度、音色——三要素

💡 **光学**（八年级）
- 光速 $c = 3 \times 10^8$ m/s（真空中）
- 反射定律、折射定律、凸透镜成像

⚡ **力学**（八年级）
- 力、速度、加速度、牛顿定律
- 压强、浮力、功与能

🌡️ **热学**（九年级）
- 分子热运动、内能、比热容
- 热机与效率、能量守恒

🔌 **电磁学**（九年级）
- 电路：欧姆定律、功率
- 磁场：电磁感应、电动机

## 🎓 给 12+（进阶版）

### 初中物理核心公式总表

**力学**

| 公式 | 含义 |
|------|------|
| $v = s/t$ | 速度 = 路程 ÷ 时间 |
| $F = ma$ | 牛顿第二定律 |
| $p = F/S$ | 压强 = 力 ÷ 面积 |
| $F_{浮} = \rho g V_{排}$ | 浮力（阿基米德定律） |
| $W = Fs$ | 功 = 力 × 距离 |
| $P = W/t = Fv$ | 功率 |
| $E_k = \frac{1}{2}mv^2$ | 动能 |
| $E_p = mgh$ | 重力势能 |

**热学**

| 公式 | 含义 |
|------|------|
| $Q = cm\Delta T$ | 热量 = 比热容 × 质量 × 温度变化 |
| $Q = mq$ | 热量 = 质量 × 燃料热值 |
| $\eta = W_{有用}/Q_{输入}$ | 热机效率 |

**电学**

| 公式 | 含义 |
|------|------|
| $I = U/R$ | 欧姆定律 |
| $R_{串} = R_1 + R_2$ | 串联总电阻 |
| $1/R_{并} = 1/R_1 + 1/R_2$ | 并联总电阻 |
| $P = UI = I^2R = U^2/R$ | 电功率 |
| $W = UIt = Pt$ | 电能 |
| $Q = I^2Rt$ | 焦耳热 |

**电磁**

| 公式 | 含义 |
|------|------|
| $F = BIL\sin\theta$ | 安培力 |
| $c = \lambda f$ | 电磁波（波速=波长×频率） |

### 核心实验方法

**控制变量法**：每次只改变一个变量，研究其影响（如研究电阻影响因素）

**图像法**：$U-I$ 图（斜率 = $R$）、$Q-t$ 图、$s-t$ 图（斜率 = $v$）

**伏安法测电阻**：$R = U/I$（用电压表和电流表）

**等效法**：用等效替代简化复杂电路

### 初中→高中物理的延伸

| 初中 | 高中延伸 |
|------|---------|
| $F = ma$（定性）| 牛顿定律、动量、角动量 |
| 欧姆定律（稳态）| 交流电路、阻抗 |
| 电磁感应（现象）| 法拉第定律 $\varepsilon = -d\Phi/dt$ |
| 能量守恒（定性）| 热力学四大定律 |
| 光的反射折射 | 波动光学、量子光学 |

### 常见实验器材与操作

| 器材 | 用途 | 注意 |
|------|------|------|
| 电流表（A） | 测电流，串联接入 | 量程选择，正负极 |
| 电压表（V） | 测电压，并联接入 | 量程选择，正负极 |
| 滑动变阻器 | 改变电阻/调节电流 | 接法：一上一下 |
| 温度计 | 测温度 | 不能超量程，读取方式 |
| 弹簧测力计 | 测力 | 竖直挂，分度值 |

### 典型综合例题

**例**：一台电动机额定电压220V，额定功率1.1kW，线圈电阻5Ω，满载运行时：
①电流 ②热功率 ③机械功率 ④效率

$$I = P/U = 1100/220 = 5 \text{ A}$$
$$P_{热} = I^2R = 25 \times 5 = 125 \text{ W}$$
$$P_{机} = 1100 - 125 = 975 \text{ W}$$
$$\eta = 975/1100 \approx 88.6\%$$

**易错点综合：**
- 电动机≠纯电阻，不用 $P = U^2/R$ 算总功率
- 串联电路：电流相同，电压分配；并联电路：电压相同，电流分配
- 效率问题：分清输入功率和有用功率
