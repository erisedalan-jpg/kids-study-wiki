# 电流测量

> **一句话**：用电流表（安培表）测量电流，电流表必须串联在被测支路中，"+"端接电流流入，量程选择以指针偏转2/3为宜。
> **English**: An ammeter measures electric current and must be connected in series with the circuit; current enters the "+" terminal, and the range should be chosen so the pointer deflects about two-thirds of full scale.

---

## 📚 给 6-12 岁（自读版）

**电流表（安培表）**是测量电流大小的工具，就像用刻度尺量长度一样。

**使用规则（4条必记）：**

1. **串联**：电流表要串联在被测导线中（不能并联！）
2. **方向**：电流从"+"接线柱流入，从"−"接线柱流出
3. **量程**：先选大量程（0-3A），若示数太小再换小量程（0-0.6A）
4. **不直接接电源**：不能把电流表直接接在电源两端（会损坏！）

**怎么读数？**

- 看清量程（0-0.6A 每小格=0.02A；0-3A 每小格=0.1A）
- 视线要正对指针（避免视差）
- 读到最小刻度的下一位（估读）

## 🎓 给 12+（进阶版）

### 电流表的工作原理

电流表（磁电式）利用**通电线圈在磁场中受力偏转**的原理工作：
- 电流越大，线圈受力越大，指针偏转越多
- 内阻极小（理想电流表内阻为零），串联在电路中不影响电路电流

### 电流表的主要参数

| 参数 | 典型值 |
|------|--------|
| 量程 | 0-0.6 A / 0-3 A（双量程） |
| 内阻 | 约 0.1-1 Ω（很小） |
| 最小分度 | 0.6 A 量程：0.02 A；3 A 量程：0.1 A |

### 电流表的使用步骤

1. **调零**：使用前检查指针是否在零刻度，若不在需机械调零
2. **选量程**：根据估算电流选合适量程；若不确定，先选最大量程
3. **串联接入**：将电流表串联在待测支路中
4. **正确接线**：电流从"+"端流入，"−"端流出（接反指针反偏）
5. **读数**：视线垂直表盘，读数到最小分度的1/10（估读位）

### 量程选择判断

选量程的目标：使指针偏转到满刻度的**1/2~2/3**附近，这样读数最准确。

- 指针偏转过小（< 1/3）：换小量程
- 超量程（指针打到头）：立即断开，换大量程

### 安培表在电路图中的符号

圆圈内写字母 A，串联在需要测量的支路上。

### 电流表与电压表的区别

| | 电流表（A） | 电压表（V） |
|---|---|---|
| 接法 | **串联** | **并联** |
| 内阻 | 极小（接近0） | 极大 |
| 作用 | 测电流（A） | 测电压（V） |
| 接错后果 | 电压表接成串联→几乎无电流，表几乎无示数 | 电流表接成并联→短路，烧表 |

### 典型例题

**例**：用0-0.6A量程的电流表测某支路电流，指针偏转到第15小格（共30格，满偏0.6A），读出电流值。

**解**：

每小格 = 0.6/30 = 0.02 A

指针在第15格：$I = 15 \times 0.02 = 0.30$ A

估读末位：0.30 A（最小分度0.02A，估读到0.002A，所以读到百分位）

**易错点：**
- 电流表必须串联，并联会短路烧表
- "+"端一定要接电流流入的那侧，即接近电源正极一侧
- 换量程后必须重新读刻度
