# 电路与电路图

> **一句话**：电路是由电源、导线、开关、用电器组成的闭合回路；电路图是用统一符号表示电路连接方式的图。
> **English**: A circuit is a closed loop consisting of a power source, wires, switches, and electrical devices; a circuit diagram uses standard symbols to show how components are connected.

---

## 📚 给 6-12 岁（自读版）

灯泡要亮起来，需要把各部分"连起来"——这个整体就叫**电路**！

**电路的四个组成部分：**

| 部分 | 作用 | 例子 |
|------|------|------|
| 电源 | 提供电能 | 电池、发电机 |
| 导线 | 传导电流 | 铜线 |
| 开关 | 控制通断 | 电灯开关 |
| 用电器 | 消耗电能做功 | 灯泡、电动机 |

**电路状态：**
- **通路（闭合电路）**：开关闭合，电流能流通，灯亮
- **断路（开路）**：开关断开或导线断了，电流不通，灯不亮
- **短路**：导线直接连接了电源两极，电流不经过用电器直接回路（危险！会烧坏导线）

**电路图：**
用特定符号画出电路，比实物图更清晰方便。

| 元件 | 电路符号 |
|------|---------|
| 电源（电池） | 长线正极，短线负极 |
| 灯泡 | 圆圈里有X |
| 开关 | 一段折线 |
| 导线 | 直线 |
| 电阻 | 矩形框 |
| 电流表（A） | 圆圈里有A |
| 电压表（V） | 圆圈里有V |

## 🎓 给 12+（进阶版）

### 电路的组成

1. **电源**：提供电动势（EMF），将其他形式的能转化为电能
   - 化学电源（电池）、发电机、太阳能电池等
2. **导线**：连接各元件，传导电流（理想导线电阻为零）
3. **开关**：控制电路的通断
4. **用电器（负载）**：将电能转化为其他形式的能（光、热、机械能等）

### 三种电路状态

| 状态 | 特征 | 结果 |
|------|------|------|
| 通路 | 闭合回路，电流正常流通 | 用电器正常工作 |
| 断路（开路） | 回路断开，电流为零 | 用电器不工作 |
| 短路 | 导线直接连接电源两极，电阻极小 | 电流极大，损坏电源和导线，引起火灾 |

**短路的危害**：短路时电阻 $R \approx 0$，由欧姆定律 $I = U/R$，电流趋向无穷大，实际上受电源内阻限制，但仍远大于正常值，产生大量热量。

### 常用电路元件符号

（按照国标 GB/T 4728 系列）

| 元件 | 符号描述 |
|------|---------|
| 电源/电池 | 两条平行线（长为正，短为负） |
| 电阻 | 矩形框 |
| 可变电阻 | 矩形框加箭头 |
| 灯泡 | 圆圈内X形 |
| 开关（断开状态） | 一段折起的线 |
| 电流表（A表） | 圆圈内字母A |
| 电压表（V表） | 圆圈内字母V |
| 电容 | 两条平行线 |
| 二极管 | 三角形指向线段 |

### 画电路图的规则

1. 用国标符号，不可自创
2. 导线用横平竖直的线表示
3. 元件不画在拐角处
4. 从电源正极出发，按顺序画回到负极
5. 串联：依次连接；并联：并排连接

### 典型例题

**例**：画出"一个电池、一个开关控制两个并联灯泡"的电路图。

**解**：
- 电源（电池）一个
- 开关串联在总干路上（控制所有灯泡）
- 两灯泡并联（各有自己的支路）
- 电流从正极 → 开关 → 并联节点 → （灯1 + 灯2同时） → 汇聚 → 负极

**易错点：**
- 串联电路：开关可以控制全部；并联电路：干路开关控制全部，支路开关控制各自支路
- 短路是危险的，不要用导线直接连接电源两极
- 电路图中导线的交叉处，若有实心点则表示"连通"，无实心点则是"跨过不连通"
