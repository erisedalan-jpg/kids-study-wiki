# 磁场对通电导体的作用

> **一句话**：放在磁场中的通电导体会受到安培力的作用，力的方向由左手定则判断；这是电动机的工作原理。
> **English**: A current-carrying conductor in a magnetic field experiences an Ampere force; the force direction is determined by the left-hand rule; this is the principle behind electric motors.

---

## 📚 给 6-12 岁（自读版）

**通电导线在磁场中会受到力！** 这就是电动机能转起来的原因。

**实验：**
把一根通电导线放在蹄形磁铁的磁场中——导线会向一侧弯曲！

- 电流越大，力越大
- 磁场越强，力越大
- 改变电流方向，力的方向也改变
- 改变磁场方向（翻转磁铁），力的方向也改变

**左手定则（判断力的方向）：**

伸开左手，让：
- **磁感线穿入手心**（磁场方向垂直于手掌进入）
- **四指弯向电流方向**
- 那么**大拇指的方向就是导线受力方向**！

**记忆口诀：** 左手定则——F B I（力、磁、电流）

## 🎓 给 12+（进阶版）

### 安培力

**安培力**：磁场对通电导体的作用力（最早由安培研究，故名）。

**安培力公式**：

$$F = BIL\sin\theta$$

- $F$：安培力（N）
- $B$：磁感应强度（T）
- $I$：电流（A）
- $L$：导线有效长度（在磁场中的长度，m）
- $\theta$：电流方向与磁场方向的夹角

**特例**：
- $\theta = 90°$（电流垂直于磁场）：$F = BIL$（最大）
- $\theta = 0°$（电流平行于磁场）：$F = 0$（无安培力）

初中阶段主要讨论 $\theta = 90°$ 的情况。

### 左手定则

伸开左手，使磁感线垂直穿入手心（手心朝向磁场来向），四指指向电流方向，则大拇指所指即为安培力方向。

**F、B、I 三者两两垂直**（当 $\theta = 90°$ 时）。

### 安培力的方向变化

| 改变 | 效果 |
|------|------|
| 电流方向反向 | 安培力反向 |
| 磁场方向反向 | 安培力反向 |
| 两者都反向 | 安培力不变 |

### 安培力与电动机

电动机中，通电线圈在磁场中受安培力产生力矩，使线圈转动。这就是"磁场对通电导体的作用 → 电动机"的基础。

### 典型例题

**例**：一根导线，长 0.2 m，垂直置于磁感应强度 0.5 T 的匀强磁场中（电流与磁场垂直），通过电流 3 A，求安培力。

$$F = BIL\sin\theta = 0.5 \times 3 \times 0.2 \times \sin90° = 0.3 \text{ N}$$

**例2**：用左手定则判断方向：电流向右，磁场向上，安培力朝哪？

左手伸开，磁场（向上）穿入手心（手掌朝上），四指指向电流方向（向右）→ 大拇指朝前（垂直于页面向外）。

**易错点：**
- 左手定则（力的方向）≠ 右手定则（电流磁场方向，安培定则）
- 当电流平行于磁场时，安培力为零（$\sin 0° = 0$）
- 安培力大小与导体的电阻无关（只与电流、磁场、长度有关）
