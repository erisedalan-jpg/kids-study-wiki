# 空气开关

> **一句话**：空气开关（断路器）是可以自动断路和手动复位的保护开关，当电路过载或短路时自动跳闸，比保险丝更安全方便。
> **English**: A circuit breaker is a protective switch that automatically trips when a circuit is overloaded or short-circuited, and can be manually reset — safer and more convenient than fuses.

---

## 📚 给 6-12 岁（自读版）

你有没有见过家里配电箱里的那排开关？它们叫**空气开关**（断路器）！

**空气开关有什么用？**
- 电流太大（过载）时：自动"跳闸"，断开电路
- 短路时：立刻断开
- 手动操作：可以当普通开关用（检修时关闭某一路电）

**和保险丝的区别：**
- 保险丝：熔断了就坏了，要换新的
- 空气开关：跳闸后，找到原因排除故障，再手动合上就可以了！（可反复使用）

**空气开关"跳闸"后怎么办？**
1. 检查是不是插了太多电器（过载）
2. 检查是否有短路（某插座/电器坏了）
3. 排除原因后，把开关手动推回"合"的位置

## 🎓 给 12+（进阶版）

### 空气开关的工作原理

空气开关采用两种脱扣机制：

**热脱扣（双金属片）**——用于过载保护：
- 双金属片由两种热膨胀系数不同的金属结合
- 电流过大→焦耳热加热双金属片→弯曲变形→触发脱扣装置→断路
- 响应较慢（数秒到数分钟），允许短时间过载（如电机启动）

**电磁脱扣（电磁铁）**——用于短路保护：
- 线圈产生磁场，短路时电流极大→磁力极强→立即吸合铁芯→触发脱扣
- 响应极快（毫秒级），立即断路

### 空气开关的规格参数

| 参数 | 说明 |
|------|------|
| 额定电流（A） | 正常允许长期通过的最大电流 |
| 额定电压（V） | 适用电路电压 |
| 极数 | 单极（1P）、双极（2P）、三极（3P） |
| 分断能力（kA） | 最大可安全断开的短路电流 |

**家庭常用规格**：1P（单相）或2P，额定电流 10/16/20/25/32/40/63 A

### 配电箱中的层级结构

```
总开关（主断路器） → 分路断路器（各回路）
    ↓
  厨房回路（20A）
  卧室回路（16A）
  客厅回路（16A）
  空调回路（25A）
  热水器回路（20A）
```

### 空气开关 vs 漏电保护器

| | 空气开关 | 漏电保护器（RCD） |
|--|---------|---------------|
| 保护对象 | 导线和电器（过载/短路） | 人身安全（漏电） |
| 触发条件 | 电流过大 | 漏电电流（30mA） |
| 响应速度 | 较快 | 极快（＜0.1s） |
| 复位 | 手动推合 | 手动按测试/复位按钮 |

**家庭电路通常同时安装两者**（组合型：漏电保护断路器）。

### 典型例题

**例**：家庭电路中，总空气开关额定电流为 63 A，某分路（厨房）空气开关额定电流为 20 A。当厨房电路中的电热水壶（2000W）、微波炉（1000W）、电饭锅（900W）同时工作，该分路空气开关会跳闸吗？

**解**：

总功率：$P = 2000 + 1000 + 900 = 3900$ W

电流：$I = P/U = 3900/220 \approx 17.7$ A

$17.7$ A < 20 A（分路额定电流），不会跳闸。

若再开一台 800 W 电烤箱：$I = 4700/220 \approx 21.4$ A > 20 A，分路空气开关会跳闸。

**易错点：**
- 空气开关跳闸后要先排除故障再复位，不能强行反复合闸
- 漏电保护器和空气开关功能不同，不能互相替代
- 额定电流选择：太小（频繁跳闸）；太大（保护失效）
