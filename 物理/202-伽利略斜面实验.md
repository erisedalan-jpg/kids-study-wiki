## 🎓 伽利略斜面实验（高中物理）

### 历史背景

**伽利略·伽利莱**（Galileo Galilei，1564-1642）是近代实验科学的奠基人。他生活的时代，亚里士多德的观点统治了约2000年：

> "重的物体比轻的物体落得快。"（亚里士多德，公元前4世纪）

伽利略通过逻辑反驳和实验证伪，推翻了这一观点。

### 斜面实验的设计思路

直接研究自由落体存在困难：物体下落太快，当时无法精确计时（没有精确秒表）。

伽利略的**天才策略**：用斜面"稀释"重力，使运动变慢，便于测量。

$$a_{\text{斜面}} = g\sin\theta \quad (\theta\text{为斜面倾角})$$

随着 $\theta$ 增大（斜面越陡），$a$ 越大；当 $\theta = 90°$，$a = g$（自由落体）。

**外推推理**：
1. 斜面角度很小时，可以测量并验证匀加速运动。
2. 逐渐增大角度，匀加速规律仍成立。
3. 外推至 $90°$（竖直落体），得出自由落体也是匀加速，且与质量无关。

### 实验结论

1. 物体沿斜面运动是**匀加速直线运动**（加速度恒定）。
2. 不同质量的物体沿同一斜面加速度**相同**（推广到自由落体：$g$ 与质量无关）。
3. 位移与时间的平方成正比：$x \propto t^2$（验证匀加速）。

$$\frac{x_1}{x_2} = \frac{t_1^2}{t_2^2}$$

### 科学方法论意义

伽利略开创了**实验+数学推理**的现代科学方法：

| 步骤 | 内容 |
|------|------|
| 1. 提出假设 | 运动是匀加速的 |
| 2. 推导数学预测 | $x \propto t^2$ |
| 3. 实验验证 | 用水钟计时，测量斜面上滚球的位移 |
| 4. 反复验证 | 改变斜面角度、球的质量 |
| 5. 外推结论 | 自由落体也是匀加速，$g$ 与质量无关 |

这与亚里士多德的"纯粹演绎推理"截然不同，是**科学革命**的开端。

### 计时方法（历史）

伽利略用**水钟**（水从小孔流出，称量水的重量来度量时间）——当时最精确的计时方法。

### 典型例题（理解性）

**例1**：倾角30°的斜面，忽略摩擦，求小球加速度。（取 $g = 10\text{ m/s}^2$）

$$a = g\sin30° = 10\times0.5 = 5\text{ m/s}^2$$

**例2**：伽利略实验中，小球从静止开始，在斜面上第1s末滚了10cm，第2s末累计滚了多少cm？

匀加速运动，$x \propto t^2$：

$$\frac{x_2}{x_1} = \frac{2^2}{1^2} = 4 \Rightarrow x_2 = 4\times10 = 40\text{ cm}$$

**例3**：亚里士多德认为10kg铁球比1kg铁球落得快10倍，伽利略用什么逻辑反驳？

伽利略的反驳：若将两球绑在一起（11kg），按亚里士多德，应比10kg球落得更快；但轻球会"拖慢"重球，又应比10kg球落得慢——自相矛盾。

### 与初中物理衔接

初中学重力和重力加速度，高中通过伽利略实验深化理解：科学知识不是凭权威断言，而是通过实验和逻辑推理建立的，体现**科学精神**。
