## 🎓 位移与路程（高中物理）

### 定义

| 概念 | 定义 | 物理量类型 | 单位 |
|------|------|-----------|------|
| **路程**（distance）| 物体运动轨迹的实际长度 | **标量**（只有大小，无方向） | m |
| **位移**（displacement）| 从初位置指向末位置的有向线段 | **矢量**（有大小，有方向） | m |

**核心区别**：
- 路程描述"走了多远"（过程量）
- 位移描述"位置变化了多少"（状态变化量）

### 图示理解

```
A ———————————————→ B
     路径（曲线/折线）= 路程
     直线 AB（带箭头）= 位移
```

如果物体从A出发绕了一圈回到A：
- 路程 > 0（走了一圈）
- 位移 = 0（起点终点相同）

### 关键关系

$$|\text{位移}| \leq \text{路程}$$

等号成立条件：物体做**单向直线运动**（方向不变）。

### 坐标表示

在数轴上：

$$\Delta x = x_{\text{末}} - x_{\text{初}}$$

- $\Delta x > 0$：沿正方向位移
- $\Delta x < 0$：沿负方向位移
- $\Delta x = 0$：回到原位（路程不一定为0）

路程 $s$ 只能加，不能抵消；位移 $\Delta x$ 可正可负，反向运动会抵消。

### 典型例题

**例1**：一人从家出发向北走了300m到学校，再向东走了400m到公园。求路程和位移大小。

- 路程 $s = 300 + 400 = 700\text{ m}$
- 位移大小（勾股定理）：$|\Delta r| = \sqrt{300^2 + 400^2} = \sqrt{90000+160000} = 500\text{ m}$，方向：北偏东 $\arctan\frac{400}{300} \approx 53°$

**例2**：汽车沿直线向东行驶500m，再向西行驶200m。

- 路程 $s = 500 + 200 = 700\text{ m}$
- 位移 $\Delta x = 500 - 200 = 300\text{ m}$（向东）

**例3**：运动员在400m跑道上跑了一整圈（400m），回到起点。

- 路程 $s = 400\text{ m}$
- 位移 $|\Delta r| = 0$（回到原点）

### 易错点

1. **位移≠路程**：即使路程很大，位移可以为零（往返运动）。
2. **位移是矢量，必须说明方向**：只说"位移5m"不完整，应说"向东5m"或用坐标"$+5\text{ m}$"。
3. **"位移大小"≠"位移"**：位移包含大小和方向，位移大小只是模长。
4. **路程不可抵消**：向东50m再向西50m，路程100m，不是0m。

### 与初中物理衔接

初中只接触路程（标量），高中引入位移（矢量），是从标量到矢量的关键升级。位移概念是速度（向量）、加速度（向量）的基础。
