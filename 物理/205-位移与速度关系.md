## 🎓 位移与速度关系（高中物理）

### 公式

$$\boxed{v_t^2 = v_0^2 + 2ax}$$

其中：
- $v_t$：末速度（m/s）
- $v_0$：初速度（m/s）
- $a$：加速度（m/s²）
- $x$：位移（m）

这是匀变速直线运动**不含时间 $t$** 的运动方程，适用于已知位移求速度（或已知速度求位移）而不知道时间的情形。

### 推导

由速度公式和位移公式消去 $t$：

$$v_t = v_0 + at \Rightarrow t = \frac{v_t - v_0}{a}$$

代入 $x = v_0 t + \frac{1}{2}at^2$：

$$x = v_0 \cdot \frac{v_t-v_0}{a} + \frac{1}{2}a\left(\frac{v_t-v_0}{a}\right)^2$$

$$= \frac{v_0 v_t - v_0^2}{a} + \frac{(v_t-v_0)^2}{2a} = \frac{2v_0 v_t - 2v_0^2 + v_t^2 - 2v_0 v_t + v_0^2}{2a} = \frac{v_t^2 - v_0^2}{2a}$$

$$\Rightarrow v_t^2 - v_0^2 = 2ax \Rightarrow v_t^2 = v_0^2 + 2ax$$

**物理意义**：$2ax$ 表示加速度对位移做"功"引起的速度平方变化（与动能定理 $W = \Delta E_k$ 类比）。

### 变形应用

| 需要求 | 变形公式 |
|--------|---------|
| 位移 $x$ | $x = \frac{v_t^2 - v_0^2}{2a}$ |
| 加速度 $a$ | $a = \frac{v_t^2 - v_0^2}{2x}$ |
| 初速度 $v_0$ | $v_0 = \sqrt{v_t^2 - 2ax}$ |
| 末速度 $v_t$ | $v_t = \sqrt{v_0^2 + 2ax}$ |

### 典型例题

**例1**：子弹以 $v_0 = 300\text{ m/s}$ 射入厚木板，在 $x = 0.5\text{ m}$ 处停止。求加速度。

$$0 = 300^2 + 2a\times0.5 \Rightarrow a = \frac{-90000}{1} = -90000\text{ m/s}^2$$

加速度极大（约 $9000g$），方向与运动方向相反（阻力）。

**例2**：汽车刹车，$v_0 = 20\text{ m/s}$，$a = -5\text{ m/s}^2$，刹车距离（到停止）是多少？

$$0 = 20^2 + 2\times(-5)\times x \Rightarrow x = \frac{400}{10} = 40\text{ m}$$

若速度增加一倍（$v_0=40\text{ m/s}$）：$x = \frac{1600}{10} = 160\text{ m}$（刹车距离变为4倍！）

**例3**：初速度为零匀加速 $a=3\text{ m/s}^2$，经过 $x=6\text{ m}$，速度是多少？

$$v_t^2 = 0 + 2\times3\times6 = 36 \Rightarrow v_t = 6\text{ m/s}$$

**例4**（交通安全）：两辆车同向行驶，速度均为 $v_0 = 30\text{ m/s}$，前车紧急制动 $a_{\text{前}} = -5\text{ m/s}^2$，后车反应时间0.5s后制动 $a_{\text{后}} = -4\text{ m/s}^2$，初始车距20m，是否追尾？

前车停止距离：$x_{\text{前}} = \frac{0-900}{-10} = 90\text{ m}$

后车0.5s内位移：$x_1 = 30\times0.5 = 15\text{ m}$（匀速）

后车制动后停止距离：$x_2 = \frac{0-900}{-8} = 112.5\text{ m}$

后车总位移：$15 + 112.5 = 127.5\text{ m}$，前车位移90m，差值37.5m > 20m，**会追尾**。

### 易错点

1. **速度平方差，不是速度差平方**：$v_t^2 - v_0^2 \neq (v_t - v_0)^2$。
2. **$x$ 是位移**：往返运动中 $x$ 可能为负或零，路程不同于位移。
3. **不含时间 $t$**：此公式适合不知道（或不需要）时间的情形。

### 与初中物理衔接

初中无对应概念。高中这一公式将在必修二学习动能定理时得到深化：$\frac{1}{2}mv_t^2 - \frac{1}{2}mv_0^2 = max = Fx$（合外力做的功）。
