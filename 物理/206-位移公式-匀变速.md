## 🎓 位移公式（匀变速直线运动）

### 公式

$$\boxed{x = v_0 t + \frac{1}{2}at^2}$$

其中：
- $x$：位移（m），正负表示方向
- $v_0$：初速度（m/s）
- $a$：加速度（m/s²），可正可负
- $t$：时间（s）

### 推导思路

**方法一：面积法（$v$-$t$ 图象）**

匀变速运动 $v$-$t$ 图象为直线，图线与时间轴围成梯形，梯形面积 = 位移：

$$x = \underbrace{v_0 t}_{\text{矩形（匀速部分）}} + \underbrace{\frac{1}{2}(v_t - v_0)t}_{\text{三角形（加速部分）}}$$

由 $v_t = v_0 + at$ 代入得：

$$x = v_0 t + \frac{1}{2}at^2$$

**方法二：平均速度法**

$$x = \bar{v} \cdot t = \frac{v_0 + v_t}{2} \cdot t = \frac{v_0 + (v_0+at)}{2} \cdot t = v_0 t + \frac{1}{2}at^2$$

**方法三（理解用）：积分法**

$$x = \int_0^t v \, dt = \int_0^t (v_0 + at) \, dt = v_0 t + \frac{1}{2}at^2$$

（高中阶段不要求掌握积分，理解面积法即可）

### 与速度公式联立

| 已知量 | 未知量 | 使用公式 |
|--------|--------|---------|
| $v_0, a, t$ | $x$ | $x = v_0t + \frac{1}{2}at^2$ |
| $v_0, a, t$ | $v_t$ | $v_t = v_0 + at$ |
| $v_0, v_t, x$ | $a$ 或 $t$ | $v_t^2 = v_0^2 + 2ax$ |

### 典型例题

**例1**：物体从静止开始匀加速，$a = 2\text{ m/s}^2$，求3s内的位移。

$$x = 0 \times 3 + \frac{1}{2}\times2\times3^2 = 0 + 9 = 9\text{ m}$$

**例2**：汽车以 $v_0 = 10\text{ m/s}$ 匀减速，$a = -2\text{ m/s}^2$，求4s内位移。

先检查是否已停止：停止时间 $t_{\text{停}} = \frac{0-10}{-2} = 5\text{ s} > 4\text{ s}$，4s内未停止，可直接代入：

$$x = 10\times4 + \frac{1}{2}\times(-2)\times4^2 = 40 - 16 = 24\text{ m}$$

**例3**：汽车以 $v_0 = 10\text{ m/s}$ 匀减速，$a = -2\text{ m/s}^2$，求8s内的位移。

先检查停止时间：$t_{\text{停}} = 5\text{ s} < 8\text{ s}$，8s内已停止，不能直接代入！

实际位移 = 停止前的位移：

$$x = 10\times5 + \frac{1}{2}\times(-2)\times5^2 = 50 - 25 = 25\text{ m}$$

（停止后静止，位移不再增加）

**例4**（二次方程求时间）：$v_0 = 0$，$a = 2\text{ m/s}^2$，求位移达到18m时的时间。

$$18 = 0 + \frac{1}{2}\times2\times t^2 \Rightarrow t^2 = 18 \Rightarrow t = 3\sqrt{2} \approx 4.24\text{ s}$$

### 易错点

1. **减速运动先判断停止时间**：若实际运动时间 $t < t_{\text{停}}$，才可直接代入；否则要以 $t_{\text{停}}$ 代入。
2. **$a$ 的符号**：减速时 $a < 0$，代入公式时带负号。
3. **$x$ 是位移不是路程**：往返运动时 $x$ 可以为0，但路程不为0。

### 与初中物理衔接

初中没有位移公式（只有 $s=vt$ 用于匀速）。高中引入 $x = v_0t + \frac{1}{2}at^2$，将加速度纳入方程，是运动学计算的核心工具。
