## 🎓 初速度为零的匀加速（高中物理）

### 定义

**初速度为零的匀加速直线运动**（从静止开始的匀加速）是匀变速运动的特殊情况，令 $v_0 = 0$，三大公式简化为：

$$v_t = at$$

$$x = \frac{1}{2}at^2$$

$$v_t^2 = 2ax$$

**典型例子**：自由落体、汽车从停止启动、火箭点火升空（近似）。

### 重要比例规律

#### 1. 速度比（等时间间隔）

第 $n$ 个 $T$ 末的速度：$v_n = anT$

$$v_1 : v_2 : v_3 : \cdots : v_n = 1 : 2 : 3 : \cdots : n$$

#### 2. 位移比（从出发计）

前 $n$ 个 $T$ 内的位移：$x_n = \frac{1}{2}a(nT)^2$

$$x_1 : x_2 : x_3 : \cdots : x_n = 1 : 4 : 9 : \cdots : n^2$$

#### 3. 各段位移比（相邻等时间段）

第 $n$ 段（第 $n$ 个 $T$ 内）位移：$\Delta x_n = x_n - x_{n-1} = \frac{1}{2}a[(nT)^2-((n-1)T)^2] = \frac{1}{2}a(2n-1)T^2$

$$\Delta x_1 : \Delta x_2 : \Delta x_3 : \cdots : \Delta x_n = 1 : 3 : 5 : \cdots : (2n-1)$$

（即**奇数比**，这是自由落体/初速零匀加速的标志性规律）

#### 4. 到达等距离点的时间比

到第 $n$ 个等距离点（$x_n = nd$）所用时间：$t_n = \sqrt{\frac{2nd}{a}}$

$$t_1 : t_2 : t_3 : \cdots = 1 : \sqrt{2} : \sqrt{3} : \cdots : \sqrt{n}$$

### 典型例题

**例1**：自由落体（$g=10\text{ m/s}^2$）。求前1s、前2s、前3s内的位移及各秒内的位移。

前 $n$ 秒：$h_n = \frac{1}{2}\times10\times n^2 = 5n^2$

| $n$ | $h_n$ (m) | 该秒位移 (m) |
|-----|-----------|------------|
| 1 | 5 | 5 |
| 2 | 20 | 15 |
| 3 | 45 | 25 |

各秒位移比 $= 5:15:25 = 1:3:5$（奇数比）✓

**例2**：汽车从静止匀加速，$a=2\text{ m/s}^2$，求第3s末速度和第3s内位移。

$v_3 = 2\times3 = 6\text{ m/s}$

第3s内位移（第3段）= $\frac{1}{2}\times2\times(2\times3-1)\times1^2 = 1\times5 = 5\text{ m}$

验证：$x_3 - x_2 = \frac{1}{2}\times2\times9 - \frac{1}{2}\times2\times4 = 9-4=5\text{ m}$ ✓

**例3**：楼高分4层，每层等高，从顶层自由落下，经过各层用时之比？

到达各层底时，下落距离之比 $= 1d:2d:3d:4d$，时间之比 $= 1:\sqrt{2}:\sqrt{3}:2$。

各层用时之比 $= (\sqrt{2}-1):(\sqrt{3}-\sqrt{2}):(2-\sqrt{3})$

（层数越低，经过该层的时间越短——因为速度越来越快）

### 易错点

1. **$1:3:5$ 是各段位移比，不是总位移比**：总位移比是 $1:4:9$。
2. **奇数比规律仅适用于 $v_0=0$ 的情形**：初速不为零时不成立。
3. **"第n秒末"vs"前n秒"**：前n秒总位移 $= \frac{1}{2}an^2$；第n秒末瞬时速度 $= an$。

### 与初中物理衔接

初中无此内容。自由落体作为初速零匀加速的典型例子，是高中物理运动学的重点，这些比例规律在实验数据分析（如打点计时器）中广泛应用。
