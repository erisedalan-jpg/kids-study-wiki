## 🎓 力的分解（高中物理）

### 定义

**力的分解**（force decomposition）是力的合成的逆运算：将一个力分解为两个（或多个）分力，使这些分力的合力等于原来的力。

$$\vec{F} = \vec{F}_1 + \vec{F}_2$$

力的分解同样遵循**平行四边形定则**：原力为对角线，两分力为邻边。

**关键**：同一个力可以分解为无数对分力（分解不唯一），实际问题中要根据力的**实际效果**来确定分解方向。

### 按实际效果分解

力的分解原则：沿力**实际产生效果**的方向分解。

**常见分解情形**：

| 情形 | 分解方向 |
|------|---------|
| 斜面上的重力 | 沿斜面向下 + 垂直斜面向下 |
| 绳子系住的斜拉力 | 水平分量 + 竖直分量 |
| 撑船的竿 | 使船前进的分力 + 使船偏离的分力 |
| 悬挂吊灯的斜绳 | 水平分量（拉墙）+ 竖直分量（支撑重力） |

### 正交分解法

将力分解到两个垂直方向（$x$ 轴和 $y$ 轴）：

$$F_x = F\cos\theta, \quad F_y = F\sin\theta$$

其中 $\theta$ 为力与 $x$ 轴的夹角。

**正交分解求合力（多力情形）**：

$$F_{\text{合},x} = \sum F_x, \quad F_{\text{合},y} = \sum F_y$$

$$F_{\text{合}} = \sqrt{F_{\text{合},x}^2 + F_{\text{合},y}^2}, \quad \tan\alpha = \frac{F_{\text{合},y}}{F_{\text{合},x}}$$

### 典型例题

**例1**（斜面重力分解）：一物体质量10kg在倾角30°的斜面上，将重力分解为沿斜面和垂直斜面的分量。（$g=10\text{ m/s}^2$）

$$G = mg = 100\text{ N}$$
$$G_{\parallel} = G\sin30° = 100\times0.5 = 50\text{ N}\text{（沿斜面向下）}$$
$$G_{\perp} = G\cos30° = 100\times\frac{\sqrt{3}}{2} \approx 86.6\text{ N}\text{（垂直斜面向下）}$$

**例2**（斜拉力分解）：用与水平成37°斜向右上方的力100N拉箱子。

$$F_x = 100\cos37° = 100\times0.8 = 80\text{ N}\text{（水平向右）}$$
$$F_y = 100\sin37° = 100\times0.6 = 60\text{ N}\text{（竖直向上）}$$

效果：$F_x$ 使箱子水平运动，$F_y$ 减小正压力（$N = G - F_y$）。

**例3**（三力平衡用分解求解）：重50N的灯由两条绳子悬挂，左绳与竖直方向成30°，右绳竖直。求两绳拉力。

左绳张力 $T_1$，右绳张力 $T_2$。

$x$ 方向：$T_1\sin30° = 0$（错，右绳竖直，$x$方向只有左绳分量）

竖直：$T_2 + T_1\cos30° = G = 50$

水平：$T_1\sin30° = 0$（没有水平力）→ $T_1\sin30° = 0$，所以 $T_1 = 0$？

重新分析：右绳竖直，左绳斜，水平方向无其他力，所以左绳**不能有水平分量**（否则不平衡），即左绳必须竖直——但题目说成30°，矛盾。

**修正**：设两绳均斜，$T_1$（左，30°）水平分量由 $T_2$ 水平分量平衡。（实际题目需提供完整条件）

**例4**（正交分解求合力）：三力：$F_1=5\text{ N（向右）}$，$F_2=3\text{ N（向上）}$，$F_3=4\text{ N（向左下45°）}$。

$$F_x = 5 + 0 + (-4\cos45°) = 5 - 2\sqrt{2} \approx 5-2.83 = 2.17\text{ N}$$
$$F_y = 0 + 3 + (-4\sin45°) = 3 - 2\sqrt{2} \approx 3-2.83 = 0.17\text{ N}$$
$$F = \sqrt{2.17^2+0.17^2} \approx 2.18\text{ N}$$

### 易错点

1. **分解不唯一**：根据实际问题选择有意义的分解方向，不能随意分解。
2. **斜面上重力分解**：$G_{\parallel} = G\sin\theta$（沿斜面），$G_{\perp} = G\cos\theta$（垂直斜面），别搞反。
3. **分力不能"代替"原力做受力分析**：分解后要用分力分别列方程，不能再列原力的方程（会重复）。

### 与初中物理衔接

初中不引入力的分解（只做合成）。高中引入分解和正交分解法，是处理多力问题的核心工具，也是解斜面、绳索、平衡问题的基础。
