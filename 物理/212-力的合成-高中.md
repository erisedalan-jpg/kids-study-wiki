## 🎓 力的合成（高中物理）

### 定义

**力的合成**（force composition）是将多个共同作用于同一物体的力，用一个等效的合力来替代的过程。

**合力**（resultant force）：与原来几个力（分力）共同作用产生的效果相同的单个力。

$$\vec{F}_{\text{合}} = \vec{F}_1 + \vec{F}_2 + \cdots + \vec{F}_n$$

力的合成遵循**平行四边形定则**（矢量加法）。

### 平行四边形定则

两个力 $\vec{F}_1$ 和 $\vec{F}_2$ 的合力：以两力为邻边作平行四边形，对角线即为合力。

$$|\vec{F}_{\text{合}}|^2 = F_1^2 + F_2^2 + 2F_1 F_2\cos\theta$$

其中 $\theta$ 为两力夹角。

特殊情况：

| 夹角 $\theta$ | 合力大小 |
|------------|---------|
| $0°$（同向） | $F_{\text{合}} = F_1 + F_2$（最大值） |
| $90°$（垂直）| $F_{\text{合}} = \sqrt{F_1^2 + F_2^2}$ |
| $180°$（反向）| $F_{\text{合}} = |F_1 - F_2|$（最小值） |

**合力范围**：$|F_1 - F_2| \leq F_{\text{合}} \leq F_1 + F_2$

### 合力的方向

合力方向由平行四边形对角线方向决定：

$$\tan\alpha = \frac{F_2\sin\theta}{F_1 + F_2\cos\theta}$$（$\alpha$ 为合力与 $F_1$ 的夹角）

特殊：两力垂直时，$\tan\alpha = \frac{F_2}{F_1}$（合力偏向较大力一侧）。

### 多力合成

三个或以上的力依次用平行四边形定则合成，或用**分量法**：

$$F_x = F_1\cos\theta_1 + F_2\cos\theta_2 + \cdots$$
$$F_y = F_1\sin\theta_1 + F_2\sin\theta_2 + \cdots$$
$$F_{\text{合}} = \sqrt{F_x^2 + F_y^2}$$

### 典型例题

**例1**：两力 $F_1 = 3\text{ N}$，$F_2 = 4\text{ N}$，互相垂直，求合力。

$$F_{\text{合}} = \sqrt{3^2 + 4^2} = 5\text{ N}$$

方向：与 $F_1$ 成 $\arctan\frac{4}{3} \approx 53°$ 角（偏向 $F_2$ 方向）

**例2**：两力均为10N，夹角60°，求合力。

$$F_{\text{合}} = \sqrt{100 + 100 + 2\times100\times\cos60°} = \sqrt{100+100+100} = \sqrt{300} \approx 17.3\text{ N}$$

方向：两力夹角平分线方向（等大时）

**例3**：合力范围

$F_1 = 5\text{ N}$，$F_2 = 3\text{ N}$，夹角未知，合力范围：

$$|5-3| \leq F_{\text{合}} \leq 5+3 \Rightarrow 2\text{ N} \leq F_{\text{合}} \leq 8\text{ N}$$

### 易错点

1. **合力不一定大于分力**：两力相差不大时，合力可以小于其中一个分力（夹角大时）。
2. **合力方向由对角线决定**，不能凭感觉猜。
3. **三力合成要逐步做**，或用分量法，不能简单加大小。

### 与初中物理衔接

初中只做同向/反向两力合成（代数加减）。高中引入平行四边形定则，处理任意方向多个力的合成，是矢量运算的完整体现。
