## 🎓 匀变速直线运动（高中物理）

### 定义

**匀变速直线运动**是沿直线运动且加速度大小和方向均不变的运动。

$$a = \text{常数（恒量）},\quad \text{轨迹为直线}$$

- **匀加速**：$a$ 与 $v$ 同向，速度大小不断增大。
- **匀减速**：$a$ 与 $v$ 反向，速度大小不断减小。
- 特例：匀速直线运动（$a = 0$）是匀变速的特殊情况。

### 三大运动公式

| 公式 | 名称 | 关系 |
|------|------|------|
| $v_t = v_0 + at$ | 速度公式 | $v$-$t$，不含 $x$ |
| $x = v_0 t + \frac{1}{2}at^2$ | 位移公式 | $x$-$t$，不含 $v_t$ |
| $v_t^2 = v_0^2 + 2ax$ | 速度位移公式 | $v$-$x$，不含 $t$ |

推论：

$$x = \frac{v_0 + v_t}{2} \cdot t \quad \text{（平均速度法）}$$

$$\bar{v} = \frac{v_0 + v_t}{2} = v_{t/2} \quad \text{（中间时刻速度=平均速度）}$$

### 重要推论：逐差法基础

**相等时间间隔 $T$ 的连续位移差为常数**：

$$x_2 - x_1 = x_3 - x_2 = \cdots = aT^2$$

即相邻相等时间内位移差恒为 $aT^2$，这是用**逐差法**求加速度的实验基础。

### $v$-$t$ 图象特征

匀变速直线运动的 $v$-$t$ 图象为**一条斜线**：
- 斜率 $k = a$（常数）
- 与 $t$ 轴围成的梯形（或三角形）面积 = 位移

### 典型例题

**例1**：物体初速度 $v_0 = 2\text{ m/s}$，加速度 $a = 3\text{ m/s}^2$，求 $t = 4\text{ s}$ 时的速度和位移。

$$v_t = 2 + 3\times4 = 14\text{ m/s}$$
$$x = 2\times4 + \frac{1}{2}\times3\times4^2 = 8 + 24 = 32\text{ m}$$

**例2**：汽车以 $20\text{ m/s}$ 匀减速，加速度 $a = -4\text{ m/s}^2$，何时停止？停止前走了多远？

停止时 $v_t = 0$：
$$t = \frac{v_t - v_0}{a} = \frac{0-20}{-4} = 5\text{ s}$$
$$x = \frac{v_t^2 - v_0^2}{2a} = \frac{0 - 400}{-8} = 50\text{ m}$$

**例3**（逐差）：打点计时器每隔0.02s打一点，取连续5段位移：2cm、4cm、6cm、8cm、10cm，求加速度。

相邻位移差 $= 2\text{ cm} = 0.02\text{ m}$，$T = 0.02\text{ s}$

$$aT^2 = 0.02\text{ m} \Rightarrow a = \frac{0.02}{(0.02)^2} = 50\text{ m/s}^2$$

### 易错点

1. **匀减速时间终点**：速度减为零后物体停止，不能再用公式算"负速度"继续运动（除非题目说明继续）。
2. **位移正负**：位移公式中 $x$ 可正可负，$a$ 也可正可负，计算前统一正方向。
3. **"匀变速"≠"匀速"**：匀变速速度是变化的，只有加速度恒定。

### 与初中物理衔接

初中只接触匀速和"越来越快/慢"，高中定量研究匀变速，引入三大公式，是运动学的核心内容。
