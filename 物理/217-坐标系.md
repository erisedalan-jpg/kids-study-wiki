## 🎓 坐标系（高中物理）

### 定义

**坐标系**是在参考系上建立的数学工具，用来定量描述物体的位置和位置变化。最基本的是**数轴**（一维）和**直角坐标系**（二维、三维）。

物理中的坐标系由以下要素确定：
1. **原点**（O）：坐标零点
2. **正方向**：规定正方向，则负方向自定
3. **单位长度**：刻度尺度

### 一维直线运动坐标系

对于沿直线运动的物体，建立数轴即可：

$$\xleftarrow{-} \quad O \quad \xrightarrow{+}$$

- 位置用坐标 $x$ 表示（可正可负）
- 位移 $\Delta x = x_2 - x_1$（有正负，表示方向）

**选取技巧**：
- 将运动起点或常用参考点设为原点
- 将主要运动方向设为正方向

### 二维平面坐标系

抛体运动等需要二维坐标系（$x$-$y$ 平面）：

$$x\text{ 轴（水平，向右为正）},\quad y\text{ 轴（竖直，向上为正）}$$

位置用 $(x, y)$ 表示，位移为向量 $\Delta\vec{r} = (\Delta x, \Delta y)$。

### 坐标与参考系的关系

$$\text{参考系（选定参照物）} \xrightarrow{\text{建立}} \text{坐标系（数学描述）}$$

同一参考系上可建立不同坐标系（原点位置不同），但物体的运动规律不变。

### 典型例题

**例1**：一辆汽车从A出发向东行驶5km到B，再向西行驶2km到C。以A为原点，向东为正方向，建立坐标系。求B、C的坐标及位移。

- B 坐标：$x_B = +5\text{ km}$
- C 坐标：$x_C = +5 - 2 = +3\text{ km}$
- A→B 位移：$\Delta x_{AB} = x_B - x_A = 5 - 0 = +5\text{ km}$（向东）
- B→C 位移：$\Delta x_{BC} = x_C - x_B = 3 - 5 = -2\text{ km}$（向西）

**例2**：若上题以B为原点，向西为正方向，重新计算。

- A 坐标：$x_A = +5\text{ km}$
- B 坐标：$x_B = 0$
- C 坐标：$x_C = +2\text{ km}$
- A→C 总位移（大小）：$|\Delta x| = |x_C - x_A| = |2-5| = 3\text{ km}$，向东（负方向）

注意：坐标系选取不影响实际位移大小，但影响正负符号含义。

**例3**：平抛运动中，以抛出点为原点，水平向右为 $x$ 轴正方向，竖直向下为 $y$ 轴正方向，写出任意时刻的坐标。

$$x = v_0 t, \quad y = \frac{1}{2}g t^2$$

### 易错点

1. **忘记规定正方向**：建立坐标系必须明确正方向，否则位移符号无意义。
2. **坐标≠位移**：坐标是位置（相对于原点），位移是位置的变化量 $\Delta x = x_2 - x_1$。
3. **混淆路程和位移**：路程是标量（总路径长度），位移是矢量（坐标差）。

### 与初中物理衔接

初中用"向东5km"等语言描述位置，高中引入坐标系，将语言描述**数学化**，为微积分、向量运算做铺垫。
