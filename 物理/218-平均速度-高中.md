## 🎓 平均速度（高中物理）

### 定义

**平均速度**（average velocity）是物体在某段时间内的位移与该时间间隔的比值：

$$\bar{v} = \frac{\Delta x}{\Delta t} = \frac{x_2 - x_1}{t_2 - t_1}$$

- 平均速度是**矢量**，方向与位移方向相同。
- 单位：m/s（米每秒）

**注意**：平均速度 $\neq$ 速率的平均值（平均速率 = 路程/时间，是标量）。

### 平均速度 vs 平均速率

| 物理量 | 公式 | 类型 | 含义 |
|--------|------|------|------|
| 平均速度 | $\bar{v} = \frac{\Delta x}{\Delta t}$ | 矢量 | 位移/时间 |
| 平均速率 | $\bar{v}_{\text{率}} = \frac{s}{\Delta t}$ | 标量 | 路程/时间 |

往返运动时：路程 > 位移，故平均速率 > 平均速度大小（甚至平均速度为0，但平均速率 > 0）。

### 匀变速运动中的平均速度

对于**匀变速直线运动**，有一个重要推论：

$$\bar{v} = \frac{v_0 + v_t}{2}$$

即平均速度等于**初末速度的算术平均值**。

这只适用于匀变速运动，非匀变速不成立！

还有：某段时间中间时刻的瞬时速度 = 该段的平均速度：

$$v_{t/2} = \bar{v} = \frac{v_0 + v_t}{2}$$

### 典型例题

**例1**：一辆车从A地出发，前半程以60km/h行驶，后半程以40km/h行驶，求全程平均速度。

设全程 $2L$：
- 前半程时间：$t_1 = \frac{L}{60}$
- 后半程时间：$t_2 = \frac{L}{40}$
- 总时间：$t = t_1 + t_2 = \frac{L}{60} + \frac{L}{40} = \frac{2L+3L}{120} = \frac{5L}{120} = \frac{L}{24}$
- 平均速度：$\bar{v} = \frac{2L}{L/24} = 48\text{ km/h}$

注意：不是 $(60+40)/2=50$，因为两段时间不同！

**例2**：物体做匀加速运动，初速度 $v_0 = 4\text{ m/s}$，末速度 $v_t = 10\text{ m/s}$，历时4s。求平均速度和位移。

$$\bar{v} = \frac{v_0 + v_t}{2} = \frac{4+10}{2} = 7\text{ m/s}$$
$$\Delta x = \bar{v} \cdot \Delta t = 7 \times 4 = 28\text{ m}$$

**例3**：某人跑步一圈（400m跑道），耗时80s。平均速度和平均速率各是多少？

- 路程 $s = 400\text{ m}$，位移 $|\Delta r| = 0$（回到起点）
- 平均速率：$\bar{v}_{\text{率}} = \frac{400}{80} = 5\text{ m/s}$
- 平均速度：$\bar{v} = \frac{0}{80} = 0$

### 易错点

1. **"前半时间/后半时间"vs"前半路程/后半路程"**：等时分段取速度算术平均，等路程分段取调和平均。
2. **平均速度≠速度的平均**：需用总位移÷总时间计算。
3. **平均速度是矢量**：方向由位移方向决定，不能只报大小。
4. **匀变速公式 $\bar{v}=\frac{v_0+v_t}{2}$ 只适用匀变速**。

### 与初中物理衔接

初中定义 $v = s/t$（速度=路程/时间，实为平均速率）。高中区分位移和路程，引入矢量速度和平均速度，概念更精确。
