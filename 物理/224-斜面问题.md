## 🎓 斜面问题（高中物理）

### 概述

**斜面问题**（inclined plane）是高中物理最经典的综合题型之一，涉及重力分解、摩擦力、弹力和牛顿第二定律。核心是建立平行/垂直斜面的坐标系。

### 坐标系建立

以斜面为参考，建立两个轴：
- **$x$ 轴**：沿斜面方向（向下为正，或视情况）
- **$y$ 轴**：垂直斜面方向（向上为正）

**重力在新坐标系中的分量**（倾角 $\theta$）：

$$G_x = G\sin\theta = mg\sin\theta \quad \text{（沿斜面向下）}$$

$$G_y = G\cos\theta = mg\cos\theta \quad \text{（垂直斜面向下）}$$

### 斜面静力学（物体静止）

**平衡条件**：

$y$ 方向：$N = G\cos\theta = mg\cos\theta$

$x$ 方向：$f = G\sin\theta = mg\sin\theta$（静摩擦力向上，阻止下滑）

**临界角**（刚好能静止）：

$$f_{\max} = \mu_s N \geq G\sin\theta$$

$$\mu_s mg\cos\theta \geq mg\sin\theta \Rightarrow \tan\theta \leq \mu_s$$

临界条件：$\tan\theta_c = \mu_s$（超过此角必下滑）

### 斜面动力学（物体运动）

#### 情形1：沿斜面下滑（无推力）

$$N = mg\cos\theta$$

$$ma = mg\sin\theta - f = mg\sin\theta - \mu_k mg\cos\theta$$

$$a = g(\sin\theta - \mu_k\cos\theta)$$

加速条件：$\sin\theta > \mu_k\cos\theta \Rightarrow \tan\theta > \mu_k$

#### 情形2：沿斜面推上

$$ma = F - mg\sin\theta - f = F - mg\sin\theta - \mu_k mg\cos\theta$$

#### 情形3：物体从斜面顶端以初速度 $v_0$ 下滑（已有速度）

同情形1，但有初速度：$v_t^2 = v_0^2 + 2ax$

### 典型例题

**例1（静止）**：倾角30°斜面，$\mu_s=0.6$，质量5kg的物体，是否会下滑？

$$\tan30° = \frac{1}{\sqrt{3}} \approx 0.577 < \mu_s = 0.6$$

临界条件：$\tan\theta_c = 0.6$，$\theta_c \approx 31°$，实际角30° < 31°，不下滑。

**例2（下滑）**：倾角37°，$\mu_k=0.3$，$m=10\text{ kg}$，$g=10\text{ m/s}^2$，求加速度。

$$a = g(\sin37° - \mu_k\cos37°) = 10\times(0.6-0.3\times0.8) = 10\times(0.6-0.24) = 3.6\text{ m/s}^2$$

**例3（推上斜面）**：倾角30°，光滑（$\mu=0$），$m=5\text{ kg}$，用平行斜面的力 $F=40\text{ N}$ 推物体，$g=10\text{ m/s}^2$，求加速度。

$$a = \frac{F - mg\sin30°}{m} = \frac{40-5\times10\times0.5}{5} = \frac{40-25}{5} = 3\text{ m/s}^2$$

**例4（综合：斜面上的连接体）**：$m_1=4\text{ kg}$ 在37°光滑斜面，$m_2=3\text{ kg}$ 悬挂，轻绳过斜面顶端滑轮。求加速度。

整体（设 $m_2$ 下降为正）：

$$m_2 g - m_1 g\sin37° = (m_1+m_2)a$$

$$3\times10 - 4\times10\times0.6 = 7a$$

$$30-24=7a \Rightarrow a = \frac{6}{7} \approx 0.86\text{ m/s}^2$$

**例5（斜面对地加速）**：斜面在水平力 $F$ 下整体加速（斜面不固定），物体随斜面加速而不下滑，求 $F$ 和物体受力。

（此题用分析斜面对物体的支持力和摩擦力，物体做水平加速运动）

### 斜面问题的进阶

- 斜面可以运动（非惯性系问题）
- 斜面上有绳子、弹簧等
- 斜面与连接体组合

这些高阶情形将在后续课程中处理，核心方法不变：建坐标系 → 受力分析 → $F=ma$。

### 易错点

1. **重力分量认错**：$G_{\text{沿斜面}} = mg\sin\theta$，$G_{\perp\text{斜面}} = mg\cos\theta$，不是相反。
2. **支持力 $N \neq G$**：斜面上 $N = mg\cos\theta < mg$（法线方向平衡）。
3. **摩擦力方向**：下滑时摩擦力沿斜面向上；上推时摩擦力沿斜面向下。两者不同！
4. **光滑斜面**：$\mu=0$，无摩擦力，$a=g\sin\theta$。

### 与初中物理衔接

初中不定量研究斜面。高中斜面问题综合所有力学概念（重力分解、支持力、摩擦力、牛顿第二定律），是高中物理必考综合题型，也是现实工程（ramp design、机械）的物理基础。
