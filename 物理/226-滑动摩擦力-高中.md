## 🎓 滑动摩擦力（高中物理）

### 定义

**滑动摩擦力**（kinetic friction / sliding friction）是两个相互接触的物体发生相对**滑动**时，接触面之间产生的阻碍相对运动的摩擦力。

### 计算公式

$$\boxed{f = \mu_k N}$$

其中：
- $f$：滑动摩擦力（N）
- $\mu_k$：**动摩擦因数**（kinetic friction coefficient），无量纲，由接触面材料和粗糙程度决定
- $N$：**正压力**（接触面之间垂直压力，N），不一定等于重力！

**适用条件**：两物体之间存在相对滑动。

### 滑动摩擦力的特点

| 特点 | 说明 |
|------|------|
| 大小 | $f = \mu_k N$（与接触面积无关！） |
| 方向 | 沿接触面，与相对运动方向相反 |
| 与速度关系 | 大小与速度大小无关（高中近似） |
| 与面积关系 | 与接触面积无关（在相同压力下） |

**摩擦力与接触面积无关**是重要结论：大轮胎不比小轮胎摩擦力大（压力相同时）；摩擦力大小只取决于 $\mu_k$ 和 $N$。

### 正压力 $N$ 的计算

$N$ 不一定等于重力 $G$，需根据受力分析确定：

| 情形 | 正压力 |
|------|--------|
| 水平面，无其他竖直力 | $N = G = mg$ |
| 水平面，有向下额外力 $F$ | $N = G + F$ |
| 水平面，有向上额外力 $F$ | $N = G - F$ |
| 倾角 $\theta$ 的斜面 | $N = G\cos\theta = mg\cos\theta$ |

### 典型例题

**例1**：重80N的物体在水平面上运动，$\mu_k = 0.3$，求滑动摩擦力。

$$N = G = 80\text{ N}$$
$$f = \mu_k N = 0.3\times80 = 24\text{ N}$$

**例2**：同一物体，用30N的力斜向下压（与水平成37°），同时水平推动，求摩擦力。

$$N = G + F\sin37° = 80 + 30\times0.6 = 80 + 18 = 98\text{ N}$$
$$f = 0.3\times98 = 29.4\text{ N}$$

**例3**：在倾角30°的斜面上，80N的物体沿斜面下滑，$\mu_k = 0.2$，求滑动摩擦力。

$$N = G\cos30° = 80\times\frac{\sqrt{3}}{2} \approx 69.3\text{ N}$$
$$f = \mu_k N = 0.2\times69.3 \approx 13.9\text{ N}\text{（沿斜面向上）}$$

**例4**（是否运动）：倾角37°斜面，$\mu_k = 0.5$，$\mu_s \approx 0.5$（静动近似相等），物体是否下滑？

平衡条件：$f = G\sin37° = 0.6G$，$N = G\cos37° = 0.8G$

最大静摩擦力：$f_{\max} = 0.5\times0.8G = 0.4G < 0.6G$

静摩擦力不足以平衡重力沿斜面分量，物体**下滑**。

### 易错点

1. **$N$ ≠ 重力**：有其他垂直于接触面的力时，$N$ 需要重新计算。
2. **摩擦力与面积无关**：增大接触面积不增大摩擦力（压力相同时）。
3. **滑动摩擦力方向**：与相对运动方向相反，不是与绝对运动方向相反。

### 与初中物理衔接

初中学摩擦力影响因素（压力和粗糙程度），高中给出定量公式 $f=\mu_k N$，引入动摩擦因数，使摩擦力计算精确化。
