## 🎓 牛顿第二定律（高中物理）

### 定律表述

**牛顿第二定律**（Newton's Second Law of Motion）：

物体所受合外力等于物体质量与加速度的乘积：

$$\boxed{\vec{F}_{\text{合}} = m\vec{a}}$$

或：物体的加速度与合外力成正比，与质量成反比，方向与合外力相同：

$$a = \frac{F_{\text{合}}}{m}, \quad \vec{a} \parallel \vec{F}_{\text{合}}$$

- 单位：$[F] = \text{N} = \text{kg·m/s}^2$，$1\text{ N} = 1\text{ kg}\cdot1\text{ m/s}^2$
- 适用于**宏观低速**（速度远小于光速）运动

### 定律的三层含义

1. **合力决定加速度**：$a$ 与合外力（所有力之和）有关，不是单个力决定的。
2. **矢量关系**：$\vec{a}$ 的方向与 $\vec{F}_{\text{合}}$ 方向完全相同（不是与速度同向）。
3. **瞬时关系**：$F=ma$ 在每一瞬间成立，合力变则加速度立即变。

### 用 $F=ma$ 解题步骤

1. **选研究对象**（哪个物体？）
2. **受力分析**（画受力图，列出所有力）
3. **建坐标系**，规定正方向
4. **列牛顿第二定律方程**（各方向分别列）
5. **解方程求解**

$$x \text{ 方向：} F_{\text{合},x} = ma_x$$
$$y \text{ 方向：} F_{\text{合},y} = ma_y$$

### 典型例题

**例1（基础）**：质量5kg的物体受合力15N（水平），求加速度。

$$a = \frac{F}{m} = \frac{15}{5} = 3\text{ m/s}^2$$

**例2（多力）**：质量10kg的箱子在水平面上，受20N水平推力，$\mu_k=0.3$，$g=10\text{ m/s}^2$。

受力分析：
- 重力：$G = 100\text{ N（向下）}$
- 支持力：$N = 100\text{ N（向上）}$（竖直平衡）
- 推力：$F = 20\text{ N（向右）}$
- 摩擦力：$f = \mu_k N = 30\text{ N（向左）}$

$x$ 方向合力：$F_{\text{合}} = 20 - 30 = -10\text{ N（向左）}$

$$a = \frac{-10}{10} = -1\text{ m/s}^2\text{（向左减速）}$$

**例3（斜面）**：质量5kg的物体在倾角30°的光滑斜面上，求加速度。（$g=10\text{ m/s}^2$）

沿斜面方向合力：$F = G\sin30° = 50\times0.5 = 25\text{ N}$

$$a = \frac{F}{m} = \frac{25}{5} = 5\text{ m/s}^2\text{（沿斜面向下）}$$

**例4（连接体）**：A（$m_A=2\text{ kg}$）和B（$m_B=3\text{ kg}$）通过轻绳连接，水平拉力 $F=10\text{ N}$，光滑地面。求加速度和绳中张力。

整体法：$a = \frac{F}{m_A+m_B} = \frac{10}{5} = 2\text{ m/s}^2$

隔离A：$T = m_A \cdot a = 2\times2 = 4\text{ N}$

验证隔离B：$F - T = m_B \cdot a \Rightarrow 10-4=6=3\times2$ ✓

### 牛顿第二定律的适用范围

- 宏观低速（$v \ll c$）：适用（经典力学）
- 接近光速：不适用（需相对论力学）
- 微观粒子（原子、电子）：不适用（需量子力学）

### 易错点

1. **$F$ 是合外力**：不是某一个力，是所有力的矢量和。
2. **$a$ 与 $v$ 不一定同向**：$a$ 与 $F_{\text{合}}$ 同向，可以与速度反向（减速）。
3. **瞬时性**：合力变化时，加速度立即变化（无滞后）。
4. **$a=0$ 不代表 $F=0$**：$a=0$ 表示匀速（合力为零），但可以受多个力。

### 与初中物理衔接

初中定性理解"力越大加速越快，质量越大越难加速"。高中用 $F=ma$ 定量表达，建立力与运动的精确数学关系，是整个高中力学的核心定律。
