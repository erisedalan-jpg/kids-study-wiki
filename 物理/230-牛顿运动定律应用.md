## 🎓 牛顿运动定律应用（高中物理）

### 综述

牛顿运动定律（$F=ma$）是经典力学的核心，应用时需将**受力分析**与**运动学**结合：

$$\text{受力分析} \xrightarrow{F=ma} \text{加速度} \xrightarrow{\text{运动学公式}} \text{位移、速度、时间}$$

### 解题通用步骤

1. **选研究对象**（一个物体或系统）
2. **受力分析**（画受力图，列出所有力：重力、弹力、摩擦力……）
3. **建坐标系**（正方向沿运动方向或简化运算）
4. **列方程**（牛顿第二定律各方向，或平衡条件）
5. **联立运动学公式**求解位移、速度、时间
6. **验证结果**（量纲、数量级、方向）

### 整体法与隔离法

**整体法**：将多个物体视为一个系统，用系统整体列方程。优点：内力不出现。适合求系统加速度。

**隔离法**：对单个物体单独分析，可求内力（如绳中张力）。

**一般策略**：先整体求加速度，再隔离求内力。

### 五大经典应用场景

#### 场景1：水平面上的运动

$$F - f = ma, \quad N = mg$$

$$f = \mu_k N = \mu_k mg$$

#### 场景2：斜面运动

沿斜面：$G\sin\theta - f = ma$（下滑）或 $F - G\sin\theta - f = ma$（推上）

垂直斜面：$N = G\cos\theta$

#### 场景3：竖直方向运动（超重失重）

$$N = m(g \pm a)$$

#### 场景4：连接体问题（见[[240-连接体问题|连接体问题]]）

整体：$F_{\text{外}} = (m_1+m_2)a$，隔离：求绳中张力

#### 场景5：斜面问题（见[[224-斜面问题|斜面问题]]）

综合斜面上的运动：考虑分力、摩擦力

### 典型例题

**例1（水平运动）**：质量10kg的物体在水平面上，受20N水平推力，$\mu_k=0.2$，$g=10\text{ m/s}^2$，求加速度和5s内速度。

$$N=100\text{ N}, \quad f=\mu_k N=20\text{ N}$$
$$F-f = 20-20 = 0 \Rightarrow a=0\text{ m/s}^2\text{（匀速！）}$$

5s内速度不变（若初速度已知则保持不变）。

**例2（力求加速度再求运动）**：$m=5\text{ kg}$，合力$F=25\text{ N}$，从静止开始，求4s末速度和位移。

$$a = \frac{F}{m} = \frac{25}{5} = 5\text{ m/s}^2$$

$$v_t = v_0 + at = 0 + 5\times4 = 20\text{ m/s}$$

$$x = \frac{1}{2}at^2 = \frac{1}{2}\times5\times16 = 40\text{ m}$$

**例3（运动求力）**：物体质量3kg，初速度10m/s，在10s内停止。求合力大小和方向。

$$a = \frac{0-10}{10} = -1\text{ m/s}^2$$

$$F = ma = 3\times(-1) = -3\text{ N}\text{（与运动方向相反）}$$

**例4（多段运动）**：电梯载人（$m=60\text{ kg}$）从静止启动：先以 $a_1=2\text{ m/s}^2$ 向上加速5s，再匀速10s，再以 $a_2=2\text{ m/s}^2$ 减速（向上）停止。分别求各阶段支持力和总位移。

加速段：$N_1 = m(g+a_1) = 60\times12 = 720\text{ N}$，$v_1=10\text{ m/s}$，$x_1=25\text{ m}$

匀速段：$N_2 = mg = 600\text{ N}$，$x_2 = 10\times10 = 100\text{ m}$

减速段：$N_3 = m(g-a_2) = 60\times8 = 480\text{ N}$，$t_3 = \frac{0-10}{-2}=5\text{ s}$，$x_3=25\text{ m}$

总位移：$x=25+100+25=150\text{ m}$

### 解题常见错误

1. **受力遗漏**：忘记摩擦力或支持力。
2. **方向错误**：加速度方向与合力方向不符。
3. **$a=0$ 时不用 $F=ma$**：平衡时用 $\sum F = 0$。
4. **跨越停止点计算**：减速到零后不能继续用同一加速度计算。

### 与初中物理衔接

初中定性描述力与运动的关系。高中用 $F=ma$ 定量计算，并与运动学公式联合，实现完整的动力学-运动学分析。这是高中物理最核心的综合应用能力。
