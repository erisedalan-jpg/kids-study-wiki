## 🎓 超重与失重（高中物理）

### 基本概念

**视重**（apparent weight）：物体对支持面（或悬绳）的压力（拉力）大小，即弹簧秤的读数。不同于重力，视重会随运动状态变化。

| 状态 | 视重与重力关系 | 名称 |
|------|-------------|------|
| 静止或匀速 | 视重 $= mg$ | 正常 |
| 向上加速 / 向下减速 | 视重 $> mg$ | **超重** |
| 向下加速 / 向上减速 | 视重 $< mg$ | **失重** |
| 自由落体（$a=g$向下）| 视重 $= 0$ | **完全失重** |

**注意**：超重、失重时，物体的**重力 $G=mg$ 不变**，只是支持力（视重）发生变化。

### 定量计算

对物体（质量 $m$）用牛顿第二定律：

**向上加速（$a$ 向上）**：

$$N - mg = ma \Rightarrow N = m(g+a) > mg \quad \text{超重}$$

**向下加速（$a$ 向下）**：

$$mg - N = ma \Rightarrow N = m(g-a) < mg \quad \text{失重}$$

**完全失重（$a=g$）**：

$$N = m(g-g) = 0$$

### 产生原因（本质）

超重/失重由**加速度方向**决定，而非速度方向：

- 向上的加速度（无论向上加速还是向下减速）→ 超重
- 向下的加速度（无论向下加速还是向上减速）→ 失重

| 运动 | 速度 | 加速度 | 状态 |
|------|------|--------|------|
| 电梯向上加速 | 向上↑ | 向上↑ | 超重 |
| 电梯向上减速 | 向上↑ | 向下↓ | 失重 |
| 电梯向下加速 | 向下↓ | 向下↓ | 失重 |
| 电梯向下减速 | 向下↓ | 向上↑ | 超重 |

### 典型例题

**例1**：质量60kg的人站在电梯中，电梯以 $a=2\text{ m/s}^2$ 向上加速，弹簧秤读数？

$$N = m(g+a) = 60\times(10+2) = 720\text{ N}$$（重力只有588N，感觉"变重"了）

**例2**：同上，电梯以 $a=3\text{ m/s}^2$ 向下加速，弹簧秤读数？

$$N = m(g-a) = 60\times(10-3) = 420\text{ N}$$（感觉"变轻"了）

**例3**：宇航员在绕地球轨道飞行（轨道高度约400km），是否失重？重力还存在吗？

轨道上重力提供向心力（$mg' = \frac{mv^2}{R}$，此时 $g' \approx 8.7\text{ m/s}^2$），航天员相当于自由落体（都以相同加速度"落向"地球），是**完全失重**（视重=0）。

**重力依然存在**（约为地面的88%），只是全部用于维持圆周运动。

**例4**：跳楼机（自由落体段），人站在秤上，读数为多少？

完全失重，$N=0$，体重秤读数为**零**（尽管人和秤都在以 $g$ 向下加速）。

### 实际应用

- **航天员训练**：在抛物线飞行的飞机上体验失重（约20-30s）
- **跳楼机娱乐**：制造失重感
- **卫星内实验**：完全失重环境用于材料科学研究
- **建筑设计**：电梯绳断时的安全装置设计

### 易错点

1. **超重/失重不改变重力**：$G=mg$ 始终不变，变的是支持力（视重）。
2. **失重≠无重力**：宇航员在太空有重力，只是处于完全失重状态。
3. **向上运动≠超重**：向上减速是失重，向上加速才是超重。判断标准是加速度方向。
4. **完全失重时体重秤读数为零**，不是重力为零。

### 与初中物理衔接

初中不涉及超重失重。高中通过牛顿第二定律 $N=m(g\pm a)$ 定量分析，是牛顿定律的重要应用，也是航天物理的入门内容。
