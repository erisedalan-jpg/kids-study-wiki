## 🎓 运动学三大公式（高中物理）

### 三大公式总览

匀变速直线运动的完整描述依赖以下三个基本公式（相互独立，任意两个可推出第三个）：

$$\boxed{v_t = v_0 + at} \tag{速度公式}$$

$$\boxed{x = v_0 t + \frac{1}{2}at^2} \tag{位移公式}$$

$$\boxed{v_t^2 = v_0^2 + 2ax} \tag{速度位移公式}$$

辅助关系（由上三式推导）：

$$x = \frac{v_0 + v_t}{2} \cdot t \tag{平均速度法}$$

$$x = v_t t - \frac{1}{2}at^2 \tag{逆向位移公式}$$

### 涉及的五个物理量

$$v_0, \quad v_t, \quad a, \quad t, \quad x$$

每个公式包含其中四个，缺少一个：

| 公式 | 包含 | 缺少 |
|------|------|------|
| $v_t = v_0 + at$ | $v_0, v_t, a, t$ | $x$ |
| $x = v_0t+\frac{1}{2}at^2$ | $v_0, a, t, x$ | $v_t$ |
| $v_t^2 = v_0^2 + 2ax$ | $v_0, v_t, a, x$ | $t$ |

**解题策略**：列出已知量和未知量，选择含目标量且不含多余未知量的公式。

### 解题步骤

1. 建立坐标系，规定正方向。
2. 将已知量写出（注意符号）。
3. 选合适公式，逐步代入求解。
4. 检验答案合理性（量纲、数量级、符号）。

### 典型例题综合训练

**例1（基础）**：汽车从静止出发匀加速，5s末速度达30m/s。求加速度和5s内位移。

已知：$v_0=0$，$v_t=30\text{ m/s}$，$t=5\text{ s}$，求 $a$ 和 $x$。

$$a = \frac{v_t-v_0}{t} = \frac{30-0}{5} = 6\text{ m/s}^2$$

$$x = \frac{v_0+v_t}{2}\cdot t = \frac{0+30}{2}\times5 = 75\text{ m}$$

**例2（选公式）**：物体初速10m/s，加速度-2m/s²，位移达多少时速度减为2m/s？

已知：$v_0=10$，$a=-2$，$v_t=2$，求 $x$（无 $t$）→ 选第三公式：

$$x = \frac{v_t^2-v_0^2}{2a} = \frac{4-100}{-4} = \frac{-96}{-4} = 24\text{ m}$$

**例3（多段运动）**：汽车以10m/s匀速行驶，司机发现障碍物，反应时间0.5s后刹车，加速度-5m/s²，制动距离和总距离分别是多少？

- 反应阶段（匀速0.5s）：$x_1 = 10\times0.5 = 5\text{ m}$
- 制动阶段（匀减速到停）：$x_2 = \frac{0-100}{2\times(-5)} = 10\text{ m}$
- 总距离：$x = 5+10 = 15\text{ m}$

**例4（逐差法）**：打点计时器 $T=0.02\text{ s}$，每隔4个点取一个计数点，间隔 $T'=0.1\text{ s}$。连续6段位移：$s_1=2.40\text{ cm}$，$s_2=3.20\text{ cm}$，$s_3=4.00\text{ cm}$，$s_4=4.80\text{ cm}$，$s_5=5.60\text{ cm}$，$s_6=6.40\text{ cm}$。用逐差法求 $a$。

$$a = \frac{(s_4+s_5+s_6)-(s_1+s_2+s_3)}{9T'^2}$$
$$= \frac{(4.80+5.60+6.40)-(2.40+3.20+4.00)}{9\times0.01}\text{ cm/s}^2$$
$$= \frac{16.80-9.60}{0.09} = \frac{7.20}{0.09} = 80\text{ cm/s}^2 = 0.80\text{ m/s}^2$$

### 符号规则（重要）

| 量 | 正值含义 | 负值含义 |
|----|---------|---------|
| $v_0$ | 正向初速 | 负向初速 |
| $v_t$ | 正向末速 | 负向末速 |
| $a$ | 正向加速度 | 负向加速度 |
| $x$ | 正向位移 | 负向位移 |

### 易错点

1. **不能跨越停止点计算**：减速运动先确认是否停止，停止后不能继续用公式。
2. **$a$ 与 $v$ 同向/反向决定加减速**：$a=-2\text{ m/s}^2$，$v_0=+10\text{ m/s}$ → 减速；$v_0=-5\text{ m/s}$ → 加速（负方向）。
3. **选公式要针对未知量**：若不含 $t$，选 $v_t^2=v_0^2+2ax$。
