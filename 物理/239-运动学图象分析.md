## 🎓 运动学图象分析（高中物理）

### 概述

运动学图象是将物体运动状态**可视化**的工具，高中物理常用两种图象：

| 图象 | 横轴 | 纵轴 | 斜率含义 | 面积含义 |
|------|------|------|---------|---------|
| $x$-$t$ 图（位移-时间）| $t$ | $x$ | 速度 $v$ | 无直接意义 |
| $v$-$t$ 图（速度-时间）| $t$ | $v$ | 加速度 $a$ | 位移 $\Delta x$ |

### $x$-$t$ 图象详解

**斜率 = 速度**：$k = \frac{\Delta x}{\Delta t} = v$

| 图线形状 | 运动状态 |
|---------|---------|
| 水平线（斜率=0） | 静止（$v=0$） |
| 向右上斜线（斜率>0） | 匀速正向运动 |
| 向右下斜线（斜率<0） | 匀速负向运动 |
| 向上弯曲线（斜率增大） | 加速运动 |
| 向下弯曲线（斜率减小） | 减速运动 |

**两直线交点**：两物体位置相同（相遇）。

**注意**：$x$-$t$ 图不代表实际运动轨迹！纵轴是位置，横轴是时间。

### $v$-$t$ 图象详解

**斜率 = 加速度**：$k = \frac{\Delta v}{\Delta t} = a$

**面积 = 位移**：$S = \int v\,dt = \Delta x$

| 图线形状 | 运动状态 |
|---------|---------|
| 水平线（斜率=0） | 匀速运动（$a=0$） |
| 向右上斜线 | 匀加速（正向，$a>0$） |
| 向右下斜线 | 匀减速（正向，$a<0$） |
| $v=0$ 处 | 速度为零（转向点） |
| 图线过 $t$ 轴 | 运动方向反转 |
| 曲线 | 非匀变速（$a$ 变化） |

**两直线交点**：两物体速度相同（同速）。

### 图象比较练习

**情形1**：一物体做匀减速运动直到停止：
- $v$-$t$ 图：从正值斜线向下，到达 $t$ 轴后水平（停止）
- $x$-$t$ 图：上凸抛物线型，斜率减小到零后水平

**情形2**：一物体先加速后匀速：
- $v$-$t$ 图：先斜线上升，后水平线
- $x$-$t$ 图：先开口向上抛物线，后斜直线

### 典型例题

**例1**：从 $v$-$t$ 图读信息

$v$-$t$ 图象：$t=0$ 时 $v=6\text{ m/s}$，$t=4\text{ s}$ 时 $v=2\text{ m/s}$，匀变速。

- 加速度：$a = \frac{2-6}{4-0} = -1\text{ m/s}^2$（减速）
- 4s内位移（梯形面积）：$x = \frac{6+2}{2}\times4 = 16\text{ m}$
- 速度为零时刻：$t = \frac{0-6}{-1} = 6\text{ s}$

**例2**：从 $x$-$t$ 图读速度

$x$-$t$ 图上有两点：$(2\text{ s}, 4\text{ m})$ 和 $(5\text{ s}, 13\text{ m})$（直线）：

$$v = \frac{13-4}{5-2} = 3\text{ m/s}$$

**例3**：判断运动状态

$v$-$t$ 图中，图线为开口向上的抛物线（$v = t^2$）：斜率 $= \frac{dv}{dt} = 2t$（随时间增大），说明加速度不断增大，是**非匀变速加速**运动。

**例4**（追及问题图解）：A以10m/s匀速，B从同一位置静止以2m/s²匀加速，B何时追上A？

$v$-$t$ 图：A为水平线（$v=10$），B为斜线（$v=2t$）。

两线面积相等时追上：$10t = \frac{1}{2}\times2\times t^2 \Rightarrow t^2 = 10t \Rightarrow t = 10\text{ s}$（排除 $t=0$）

### 易错点

1. **$x$-$t$ 图斜率是速度，不是加速度**；$v$-$t$ 图斜率才是加速度。
2. **$x$-$t$ 图的形状不代表运动轨迹**：例如曲线形的 $x$-$t$ 图不代表曲线运动，只是位移随时间的非线性变化。
3. **$v$-$t$ 图面积要注意正负**：$t$ 轴以下面积为负位移。
4. **两图线交点含义不同**：$x$-$t$ 图交点=相遇，$v$-$t$ 图交点=同速（不一定在同一位置）。

### 与初中物理衔接

初中简单接触 $v$-$t$ 图（定性）。高中利用图象进行定量计算（斜率、面积），将图形语言与代数语言统一，是重要的分析工具。
