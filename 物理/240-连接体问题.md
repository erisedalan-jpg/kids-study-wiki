## 🎓 连接体问题（高中物理）

### 定义

**连接体问题**（connected bodies）是两个或多个物体通过绳子（或杆、弹簧等）连接，一起运动的问题。连接体中各物体具有相同的加速度大小。

### 解题方法

#### 整体法（求系统加速度）

将所有相连物体视为**一个整体**，忽略内力（绳子张力）：

$$F_{\text{外合}} = (m_1 + m_2 + \cdots)a$$

#### 隔离法（求内力）

在已知加速度后，对单个物体隔离分析，用牛顿第二定律求绳中张力：

$$F_{\text{合（对该物体）}} = m_i a$$

**策略**：先整体求 $a$，再隔离求内力。

### 轻绳的假设

高中通常假设绳子**质量为零**（轻绳），且不可伸长：

- 轻绳两端张力**处处相等**（绳子本身无加速度效应）
- 不可伸长：两端物体加速度大小相同

### 经典模型1：水平连接体

A（$m_A$）和B（$m_B$）水平排列，绳连接，F推A：

**整体**：$F - f_A - f_B = (m_A + m_B)a$

其中 $f_A = \mu m_A g$，$f_B = \mu m_B g$（各自摩擦力）

**隔离B**：$T - f_B = m_B a \Rightarrow T = m_B(a + \mu g)$

**隔离A**：$F - T - f_A = m_A a$（验证）

### 经典模型2：阿特伍德机（两质量悬挂）

轻绳过定滑轮，两端挂 $m_1 > m_2$（$m_1$ 重侧下沉）：

**整体**（以 $m_1$ 下降为正方向）：

$$m_1 g - m_2 g = (m_1 + m_2)a$$

$$a = \frac{(m_1 - m_2)g}{m_1 + m_2}$$

**隔离 $m_1$**：$m_1 g - T = m_1 a \Rightarrow T = m_1(g-a) = \frac{2m_1 m_2 g}{m_1+m_2}$

**验证 $m_2$**：$T - m_2 g = m_2 a$ ✓

**特殊**：$m_1=m_2$ 时 $a=0$，$T=mg$（平衡）；$m_2=0$ 时 $a=g$（$m_1$ 自由落体，$T=0$）。

### 经典模型3：斜面与悬挂

$m_1$ 在斜面（倾角 $\theta$，光滑），$m_2$ 悬挂（竖直），轻绳过斜面顶端定滑轮：

**整体**（以 $m_2$ 下降、$m_1$ 沿斜面上滑为正）：

$$m_2 g - m_1 g\sin\theta = (m_1+m_2)a$$

$$a = \frac{(m_2 - m_1\sin\theta)g}{m_1+m_2}$$

若 $m_2 > m_1\sin\theta$：$m_2$ 下降，$m_1$ 上滑；若 $<$：反之。

### 典型例题

**例1（水平连接）**：A（3kg）拉B（2kg），F=20N，地面光滑，求加速度和绳中张力。

整体：$a = \frac{20}{3+2} = 4\text{ m/s}^2$

隔离B：$T = m_B \cdot a = 2\times4 = 8\text{ N}$

**例2（含摩擦）**：A（4kg）在B（6kg）上，$\mu_{AB}=0.2$（A与B之间），$\mu_{Bground}=0.3$（B与地面），水平力F=50N推B，$g=10\text{ m/s}^2$，求A的加速度（A与B不用绳连接，A随B一起加速）。

A的唯一水平力是A-B摩擦力：$f_{BA} = \mu_{AB} m_A g = 0.2\times4\times10 = 8\text{ N}$

整体合力：$F - f_{Bground} = 50 - \mu_{Bg}(m_A+m_B)g = 50-0.3\times100 = 50-30=20\text{ N}$

整体加速度：$a = \frac{20}{10} = 2\text{ m/s}^2$

检验A：$f_{BA} = m_A a = 4\times2=8\text{ N}$ ✓（等于最大静摩擦力，刚好不滑动）

**例3（阿特伍德）**：$m_1=3\text{ kg}$，$m_2=1\text{ kg}$，$g=10\text{ m/s}^2$，求加速度和绳中张力。

$$a = \frac{(3-1)\times10}{3+1} = \frac{20}{4} = 5\text{ m/s}^2$$

$$T = \frac{2\times3\times1\times10}{3+1} = \frac{60}{4} = 15\text{ N}$$

### 易错点

1. **整体法忽略内力**：用整体法时，绳中张力是内力，不出现在整体方程中。
2. **加速度方向统一**：不同物体运动方向可能相反，但约定正方向后要统一处理。
3. **轻绳两端张力相等**：前提是绳轻、不可伸长，且不考虑绳的加速度。

### 与初中物理衔接

初中不涉及连接体。高中连接体问题综合了受力分析、牛顿第二定律、整体与隔离法，是动力学的综合应用，为大学力学的约束条件、广义坐标等概念打基础。
