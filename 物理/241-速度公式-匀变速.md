## 🎓 速度公式（匀变速直线运动）

### 公式

$$\boxed{v_t = v_0 + at}$$

其中：
- $v_t$：$t$ 时刻的速度（末速度，m/s）
- $v_0$：初速度（m/s）
- $a$：加速度（m/s²，可正可负）
- $t$：时间（s）

### 推导

加速度定义：

$$a = \frac{v_t - v_0}{\Delta t} = \frac{v_t - v_0}{t}$$

变形得：

$$v_t = v_0 + at$$

这是**由加速度定义直接变形**而来，无需额外假设（只要 $a$ 为常数）。

### 图象含义

在 $v$-$t$ 图象中：
- $v_t = v_0 + at$ 是关于 $t$ 的一次函数，图象为**直线**。
- 纵截距 = $v_0$（初速度）
- 斜率 = $a$（加速度）

$$v_t = \underbrace{a}_{\text{斜率}} \cdot t + \underbrace{v_0}_{\text{截距}}$$

### 典型应用

**已知 $v_0, a$，求某时刻速度**：直接代入。

**已知 $v_0, v_t, a$，求时间**：

$$t = \frac{v_t - v_0}{a}$$

**判断运动方向**：
- $v_t > 0$：正方向运动
- $v_t < 0$：负方向运动（与初速方向相反）
- $v_t = 0$：速度为零（转折点或停止）

### 典型例题

**例1**：从静止开始匀加速，$a = 5\text{ m/s}^2$，求3s末速度。

$$v_t = 0 + 5\times3 = 15\text{ m/s}$$

**例2**：初速度 $v_0 = 30\text{ m/s}$，匀减速 $a = -6\text{ m/s}^2$，何时速度为零？何时速度为 $-6\text{ m/s}$？

速度为零：

$$t = \frac{0-30}{-6} = 5\text{ s}$$

速度为 $-6\text{ m/s}$（方向反转）：

$$t = \frac{-6-30}{-6} = 6\text{ s}$$

（注：实际物体停止后可能不继续运动，需结合题目判断）

**例3**：两车A、B同向行驶，A在前方，$v_A = 10\text{ m/s}$（匀速），B在后方，$v_B = 20\text{ m/s}$，$a_B = -2\text{ m/s}^2$（匀减速）。B的速度何时等于A？

$$v_B = 20 - 2t = 10 \Rightarrow t = 5\text{ s}$$

（此时两车速度相同，之后B继续减速，A超过B）

### 易错点

1. **$a$ 的符号不能遗漏**：减速时 $a < 0$，代入 $at$ 时得到负值，速度减小——这是正确的。
2. **公式适用前提：$a$ 恒定**：非匀变速运动不能用此公式。
3. **速度为零不等于运动结束**：做投上去又落下来的运动，最高点速度为零，但运动继续。

### 与初中物理衔接

初中没有速度公式（不定量研究加速运动）。高中速度公式 $v_t = v_0 + at$ 是将加速度概念定量化的第一步，也是三大运动学公式中最基础的一条。
