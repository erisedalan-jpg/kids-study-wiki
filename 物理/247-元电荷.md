## 🎓 元电荷

### 定义

**元电荷**（基本电荷）：自然界中自由存在的最小电荷量，用 $e$ 表示。

$$\boxed{e = 1.6 \times 10^{-19}\ \text{C}}$$

- 电子带负电：$q_e = -e = -1.6 \times 10^{-19}\ \text{C}$
- 质子带正电：$q_p = +e = +1.6 \times 10^{-19}\ \text{C}$

### 电荷量子化

**电荷量子化**：任何带电体的电荷量都是元电荷 $e$ 的整数倍。

$$q = ne, \quad n \in \mathbb{Z}$$

（$n$ 可以为正整数、负整数或零）

不存在 $0.5e$、$1.3e$ 等非整数倍的自由电荷。

### 历史：密立根油滴实验

**罗伯特·密立根**（Robert A. Millikan，1868—1953）于1909年完成**油滴实验**，首次精确测定 $e$ 的值：

- 带电油滴悬浮在电场中：重力 = 电场力
- 改变电压，使油滴匀速运动或静止
- 测量不同油滴的电荷量，发现均为某最小值的整数倍
- 该最小值即元电荷 $e$

密立根因此获1923年诺贝尔物理学奖。

### 应用：由粒子数算电量

$$q = ne$$

例：$n = 10^6$ 个电子的总电量：

$$q = 10^6 \times 1.6 \times 10^{-19} = 1.6 \times 10^{-13}\ \text{C}$$

例：$1\ \text{C}$ 电量对应多少个电子？

$$n = \frac{q}{e} = \frac{1}{1.6 \times 10^{-19}} = 6.25 \times 10^{18}\ \text{个}$$

### 典型例题

**例1**：某物体带电量 $q = 4.8 \times 10^{-19}\ \text{C}$，转移了多少个电子？

$$n = \frac{q}{e} = \frac{4.8 \times 10^{-19}}{1.6 \times 10^{-19}} = 3\ \text{个}$$

**例2**：验电器金箔张开后，$q = 3.2 \times 10^{-18}\ \text{C}$，带多少个电子的电量？

$$n = \frac{3.2 \times 10^{-18}}{1.6 \times 10^{-19}} = 20\ \text{个}$$

### 基本粒子与电荷

| 粒子 | 电荷 |
|------|------|
| 电子 | $-e$ |
| 质子 | $+e$ |
| 中子 | $0$ |
| 夸克 | $±\frac{1}{3}e$ 或 $±\frac{2}{3}e$（不单独存在） |

注：夸克的电荷是 $e$ 的分数倍，但夸克不能单独自由存在（色禁闭），所以不违反电荷量子化（自由电荷最小单位仍为 $e$）。

### 易错点

1. **元电荷是最小自由电荷**，不是最小的电荷（夸克有更小的，但被禁闭）。
2. **$e = 1.6 \times 10^{-19}\ \text{C}$ 是大小**：电子电荷为 $-e$（负值）。
3. **电荷守恒**：任何起电过程中电荷转移的数量是整数个 $e$。
