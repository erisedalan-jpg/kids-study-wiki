## 🎓 向心力

### 定义

**向心力**：做匀速圆周运动的物体所受的合外力，方向始终指向圆心，大小为：

$$F = m\frac{v^2}{r} = m\omega^2 r = \frac{4\pi^2 mr}{T^2}$$

向心力是**效果力**，不是新的力种，由其他力（重力、弹力、摩擦力等）提供。

### 向心力的来源

| 情景 | 向心力来源 |
|------|-----------|
| 行星绕太阳 | 万有引力 |
| 汽车拐弯 | 摩擦力（侧向） |
| 卫星绕地球 | 万有引力 |
| 圆锥摆 | 绳拉力水平分量 |
| 竖直圆周 | 绳拉力 + 重力（合力） |
| 火车弯道 | 内外轨道对车轮的侧向力 |

### 公式

$$F_{\text{向}} = m\frac{v^2}{r} = m\omega^2 r = \frac{4\pi^2 mr}{T^2}$$

由牛顿第二定律：$F_{\text{向}} = ma_{\text{向}}$，其中 $a_{\text{向}} = v^2/r$。

### 典型例题

**例1（水平圆周）**：质量 $m = 0.5\ \text{kg}$ 的小球，$r = 0.4\ \text{m}$，$v = 2\ \text{m/s}$，向心力？

$$F = m\frac{v^2}{r} = 0.5 \times \frac{4}{0.4} = 5\ \text{N}$$

**例2（汽车转弯）**：质量 $m = 1000\ \text{kg}$，$v = 20\ \text{m/s}$，转弯半径 $r = 200\ \text{m}$，所需向心力（由摩擦力提供）？

$$F = m\frac{v^2}{r} = 1000 \times \frac{400}{200} = 2000\ \text{N}$$

最大静摩擦力 $f = \mu mg = 0.3 \times 1000 \times 10 = 3000\ \text{N} > 2000\ \text{N}$，可以安全转弯。

**例3（圆锥摆）**：绳长 $L$，与竖直夹角 $\theta$，质量 $m$，求周期。

$$T\cos\theta = mg \implies T = \frac{mg}{\cos\theta}$$
$$T\sin\theta = m\omega^2 r = m\omega^2 L\sin\theta$$
$$\omega = \sqrt{\frac{g}{L\cos\theta}}, \quad T_{\text{周期}} = 2\pi\sqrt{\frac{L\cos\theta}{g}}$$

### 易错点

1. **向心力不是独立的力**：不能在受力分析中单独画出，要找是哪个力（或合力）充当向心力。
2. **离心力是伪力**：在地面（惯性系）参考系中不存在，只在旋转参考系中引入。
3. **向心力不做功**：方向垂直速度，故做功为零，不改变动能（速率）。
4. **匀速圆周运动合力不为零**：合力即向心力，不为零，指向圆心。
