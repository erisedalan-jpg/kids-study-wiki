## 🎓 小船过河问题

### 问题建模

河宽 $d$，水流速度 $v_{\text{水}}$（沿河岸方向），船相对水的速度 $v_{\text{船}}$（可调整方向）。

$$\vec{v}_{\text{船/地}} = \vec{v}_{\text{船/水}} + \vec{v}_{\text{水/地}}$$

### 两类经典问题

#### 情形一：最短时间过河

**目标**：渡河时间最短

**策略**：船头垂直河岸（$v_{\text{船}}$ 垂直于河岸）

$$t_{\min} = \frac{d}{v_{\text{船}}}$$

此时船被水流冲向下游，偏移量：

$$\Delta x = v_{\text{水}} \cdot t_{\min} = \frac{v_{\text{水}} \cdot d}{v_{\text{船}}}$$

实际路径为斜线，合速度：$v = \sqrt{v_{\text{船}}^2 + v_{\text{水}}^2}$

#### 情形二：最短路程过河（直线到对岸）

**条件**：$v_{\text{船}} \geq v_{\text{水}}$（船速不小于水速才能直渡）

**策略**：船头斜向上游，使合速度垂直河岸

$$\cos\alpha = \frac{v_{\text{水}}}{v_{\text{船}}} \implies \alpha = \arccos\frac{v_{\text{水}}}{v_{\text{船}}}$$

（$\alpha$ 为船头偏向上游的角度）

渡河时间：

$$t = \frac{d}{v_{\text{船/地}}} = \frac{d}{\sqrt{v_{\text{船}}^2 - v_{\text{水}}^2}}$$

**若 $v_{\text{船}} < v_{\text{水}}$**：无法直渡，只能求最小偏移角：

$$\sin\beta = \frac{v_{\text{船}}}{v_{\text{水}}} \implies \text{合速度与河岸夹角最小}$$

最小偏移角时偏移距离最小（但不为零）。

### 典型例题

**例1**：河宽 $d = 60\ \text{m}$，$v_{\text{水}} = 4\ \text{m/s}$，$v_{\text{船}} = 3\ \text{m/s}$（$v_{\text{船}} < v_{\text{水}}$），求最短时间和此时偏移。

$$t_{\min} = \frac{60}{3} = 20\ \text{s}$$
$$\Delta x = 4 \times 20 = 80\ \text{m}$$

（无法直渡，偏移80m）

**例2**：$v_{\text{水}} = 3\ \text{m/s}$，$v_{\text{船}} = 5\ \text{m/s}$，$d = 100\ \text{m}$，直渡所需时间？

$$v_{\text{有效}} = \sqrt{5^2 - 3^2} = 4\ \text{m/s}$$
$$t = \frac{100}{4} = 25\ \text{s}$$

### 易错点

1. **最短时间不等于最短路程**：最短时间对应垂直过河，最短路程对应斜向过河（若 $v_{\text{船}} \geq v_{\text{水}}$）。
2. **$v_{\text{船}} < v_{\text{水}}$ 时不能直渡**：只能求最小偏移角。
3. **速度合成方向**：水流速度是客观存在，不由人控制；船头方向可调整。
