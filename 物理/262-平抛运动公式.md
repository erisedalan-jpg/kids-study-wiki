## 🎓 平抛运动公式

### 公式汇总

以抛出点为原点，水平向右为 $x$ 轴正方向，竖直向下为 $y$ 轴正方向：

| 量 | 公式 |
|----|------|
| 水平位移 | $x = v_0 t$ |
| 竖直位移 | $y = \dfrac{1}{2}gt^2$ |
| 水平速度 | $v_x = v_0$ |
| 竖直速度 | $v_y = gt$ |
| 合速度大小 | $v = \sqrt{v_0^2 + g^2t^2}$ |
| 合速度方向 | $\tan\theta = \dfrac{v_y}{v_x} = \dfrac{gt}{v_0}$ |
| 合位移大小 | $r = \sqrt{x^2 + y^2}$ |
| 轨迹方程 | $y = \dfrac{g}{2v_0^2}x^2$ |
| 落地时间 | $t = \sqrt{\dfrac{2H}{g}}$（$H$ 为抛出高度） |
| 水平射程 | $R = v_0\sqrt{\dfrac{2H}{g}}$ |

### 推导过程

**1. 位移分量**

水平方向无加速：$x = v_0 t$

竖直方向自由落体：$y = \dfrac{1}{2}gt^2$

**2. 速度分量**

$v_x = v_0$（恒定），$v_y = gt$

**3. 轨迹方程**

由 $x = v_0 t$ 得 $t = \dfrac{x}{v_0}$，代入竖直位移：

$$y = \frac{1}{2}g\left(\frac{x}{v_0}\right)^2 = \frac{g}{2v_0^2}x^2$$

这是以原点为顶点的抛物线，开口向下（$y$ 轴向下为正时开口向正方向）。

### 重要结论

- 落地时间 $t$ 只与初始高度 $H$ 有关，与 $v_0$ 无关
- 水平射程 $R$ 与 $v_0$ 成正比
- 落地速度角 $\theta$ 满足 $\tan\theta = \dfrac{gt}{v_0}$，与位移方向角不同
- 合位移方向角 $\phi$：$\tan\phi = \dfrac{y}{x} = \dfrac{gt}{2v_0}$，恰好是速度方向角的一半（$\tan\theta = 2\tan\phi$）

### 典型例题

**例1**：$v_0 = 15\ \text{m/s}$，$H = 45\ \text{m}$，求射程和落地速度方向。

$$t = \sqrt{\frac{2 \times 45}{10}} = 3\ \text{s}$$
$$R = 15 \times 3 = 45\ \text{m}$$
$$v_y = 10 \times 3 = 30\ \text{m/s}$$
$$\tan\theta = \frac{30}{15} = 2 \implies \theta \approx 63.4°$$

**例2（逆推 $v_0$）**：落地点水平距 $x = 30\ \text{m}$，竖直落下 $y = 20\ \text{m}$，求 $v_0$。

$$t = \sqrt{\frac{2y}{g}} = \sqrt{\frac{40}{10}} = 2\ \text{s}$$
$$v_0 = \frac{x}{t} = \frac{30}{2} = 15\ \text{m/s}$$

### 易错点

1. **位移角与速度角混淆**：$\tan\phi_{\text{位移}} = \dfrac{y}{x} = \dfrac{gt}{2v_0}$，而 $\tan\phi_{\text{速度}} = \dfrac{v_y}{v_x} = \dfrac{gt}{v_0}$，速度角是位移角的两倍关系（正切值）。
2. **落地时间公式**：$t = \sqrt{2H/g}$ 仅在竖直初速为零时适用（平抛）。斜抛需另算。
3. **$g$ 取值**：一般取 $10\ \text{m/s}^2$，精确计算取 $9.8\ \text{m/s}^2$。
