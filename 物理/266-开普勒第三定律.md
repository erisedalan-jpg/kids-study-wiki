## 🎓 开普勒第三定律

### 内容

**开普勒第三定律（周期定律）**：所有行星轨道半长轴的三次方与其公转周期的平方之比为一个**常数**。

$$\frac{a^3}{T^2} = k$$

$k$ 只与**中心天体质量**有关，与行星本身质量和轨道大小无关。

### 常数 $k$ 的表达式

对圆轨道（$a = r$），万有引力提供向心力：

$$G\frac{Mm}{r^2} = m\frac{4\pi^2 r}{T^2}$$

$$\frac{r^3}{T^2} = \frac{GM}{4\pi^2}$$

$$\boxed{k = \frac{GM}{4\pi^2}}$$

（$G = 6.674 \times 10^{-11}\ \text{N·m}^2/\text{kg}^2$，$M$ 为中心天体质量）

### 应用

**1. 比较同一中心天体的两行星**：

$$\frac{a_1^3}{T_1^2} = \frac{a_2^3}{T_2^2} \implies \left(\frac{a_1}{a_2}\right)^3 = \left(\frac{T_1}{T_2}\right)^2$$

**2. 计算中心天体质量**：

$$M = \frac{4\pi^2 a^3}{GT^2}$$

**3. 已知周期求轨道半径**：

$$a = \left(\frac{GMT^2}{4\pi^2}\right)^{1/3}$$

### 典型例题

**例1（比较轨道）**：A 星轨道半径 $r_A = 4r_B$，求周期比 $T_A / T_B$。

$$\frac{T_A^2}{T_B^2} = \frac{r_A^3}{r_B^3} = \left(\frac{4}{1}\right)^3 = 64$$
$$\frac{T_A}{T_B} = \sqrt{64} = 8$$

**例2（算地球质量）**：月球轨道半径 $r = 3.84 \times 10^8\ \text{m}$，$T = 27.3\ \text{天} = 2.36 \times 10^6\ \text{s}$，求地球质量。

$$M = \frac{4\pi^2 r^3}{GT^2} = \frac{4\pi^2 \times (3.84 \times 10^8)^3}{6.67 \times 10^{-11} \times (2.36 \times 10^6)^2}$$
$$\approx 6.0 \times 10^{24}\ \text{kg}$$

**例3（行星比较）**：火星公转周期 $T_{\text{火}} = 1.88\ \text{年}$，地球 $T_{\text{地}} = 1\ \text{年}$，求火星轨道半径（以地球轨道为单位）。

$$\frac{r_{\text{火}}^3}{r_{\text{地}}^3} = \frac{T_{\text{火}}^2}{T_{\text{地}}^2} = 1.88^2 = 3.53$$
$$r_{\text{火}} = 3.53^{1/3} \times r_{\text{地}} \approx 1.52\ \text{AU}$$

### 易错点

1. **$k$ 与中心天体有关**：绕地球的卫星和绕太阳的行星有不同的 $k$ 值。
2. **$a$ 是半长轴，不是直径**：圆轨道 $a = r$，椭圆轨道 $a$ 为半长轴。
3. **单位一致**：计算时 $a$（m），$T$（s），$G = 6.674 \times 10^{-11}\ \text{N·m}^2/\text{kg}^2$。
