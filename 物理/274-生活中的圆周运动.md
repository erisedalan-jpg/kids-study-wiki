## 🎓 生活中的圆周运动

### 常见场景分析

#### 1. 汽车转弯

**向心力来源**：路面对车轮的侧向摩擦力

$$f = m\frac{v^2}{r} \leq \mu mg$$

最大安全速度：$v_{\max} = \sqrt{\mu g r}$

**设计坡度（倾斜路面）**：为减少摩擦依赖，弯道外侧略高，重力分量提供向心力：

$$\tan\theta = \frac{v^2}{rg}$$

#### 2. 火车弯道

**向心力来源**：外轨道对车轮的侧向力（内外轨高度差）

设计速度：$v = \sqrt{rgtan\theta}$（$\theta$ 为外轨高于内轨的倾角）

- $v = v_{\text{设计}}$：内外轨无侧向力（理想状态）
- $v > v_{\text{设计}}$：外轨受力，车轮压外轨
- $v < v_{\text{设计}}$：内轨受力，车轮压内轨

#### 3. 竖直圆周运动（过山车/水桶）

**最高点分析**（设绳/轨道支持力 $N$）：

$$mg + N = m\frac{v^2}{r}$$

临界条件（$N = 0$）：$v_{\min} = \sqrt{gr}$（最高点最小速度）

**最低点分析**：

$$N - mg = m\frac{v^2}{r} \implies N = mg + m\frac{v^2}{r}\ \text{（超重）}$$

#### 4. 圆锥摆

绳长 $L$，与竖直夹角 $\theta$：

$$T_{\text{周期}} = 2\pi\sqrt{\frac{L\cos\theta}{g}}$$

角速度越大，锥角越大。

#### 5. 卫星与行星

**向心力来源**：万有引力

$$G\frac{Mm}{r^2} = m\frac{v^2}{r} = m\omega^2 r$$

#### 6. 超重与失重（圆周运动版）

- 最低点：$N = mg + mv^2/r$（超重）
- 最高点：$N = mg - mv^2/r$（失重）或 $N = mv^2/r - mg$（若 $v > \sqrt{gr}$）

### 典型例题

**例1（汽车弯道）**：$m = 1500\ \text{kg}$，$r = 100\ \text{m}$，$\mu = 0.5$，最大安全速度？

$$v_{\max} = \sqrt{\mu gr} = \sqrt{0.5 \times 10 \times 100} = \sqrt{500} \approx 22.4\ \text{m/s} \approx 80.6\ \text{km/h}$$

**例2（过山车最高点）**：$r = 10\ \text{m}$，最高点最小速度？

$$v_{\min} = \sqrt{gr} = \sqrt{10 \times 10} = 10\ \text{m/s}$$

**例3（拱形桥最高点）**：$m = 1000\ \text{kg}$，$r = 50\ \text{m}$，$v = 20\ \text{m/s}$，支持力？

$$mg - N = m\frac{v^2}{r}$$
$$N = mg - m\frac{v^2}{r} = 1000 \times 10 - 1000 \times \frac{400}{50} = 10000 - 8000 = 2000\ \text{N}$$

### 易错点

1. **最高点和最低点受力方向**：最高点 $mg + N = mv^2/r$（两力同向向心）；最低点 $N - mg = mv^2/r$（支持力向心，重力反向）。
2. **临界速度**：过山车最高点 $v_{\min} = \sqrt{gr}$，低于此速将脱轨。
3. **弯道超速危险**：速度越大，所需向心力越大，超过摩擦力极限则失控。
