## 🎓 电势差

### 定义

**电势差**（$U_{AB}$）：A、B 两点的电势之差。

$$U_{AB} = V_A - V_B$$

**电势**（$V$）：电场中某点的电势，等于将单位正电荷从该点移到无穷远处电场力做的功。

$$V = \frac{E_p}{q}$$

单位：V（伏特），$1\ \text{V} = 1\ \text{J/C}$

### 电场力做功与电势差

$$W_{AB} = q(V_A - V_B) = qU_{AB}$$

（$q$ 为电荷量，带符号）

等价地：

$$U_{AB} = \frac{W_{AB}}{q}$$

### 电势差与电场强度的关系

在匀强电场中，$A$、$B$ 两点沿电场方向距离为 $d$：

$$U_{AB} = Ed \implies E = \frac{U_{AB}}{d}$$

对一般电场：$E = -\dfrac{\Delta V}{\Delta x}$（沿场方向，电势降落最快方向）

### 电势和电势差的性质

| 性质 | 说明 |
|------|------|
| 参考点相关 | 电势绝对值依赖参考点选取（通常取无穷远为零） |
| 电势差与参考点无关 | $U_{AB} = V_A - V_B$ 不受参考点影响 |
| 沿电场线方向 | 电势降低（$V$ 减小） |
| 等势面上 | 各点电势相等，$U = 0$，电场力不做功 |

### 典型例题

**例1**：A 点电势 $V_A = 10\ \text{V}$，B 点电势 $V_B = 4\ \text{V}$，$q = 2 \times 10^{-6}\ \text{C}$ 从 A 到 B，电场力做功？

$$U_{AB} = V_A - V_B = 6\ \text{V}$$
$$W_{AB} = qU_{AB} = 2 \times 10^{-6} \times 6 = 1.2 \times 10^{-5}\ \text{J}$$

**例2（负电荷）**：$q = -4 \times 10^{-6}\ \text{C}$，$U_{AB} = 8\ \text{V}$，电场力做功？

$$W_{AB} = qU_{AB} = (-4 \times 10^{-6}) \times 8 = -3.2 \times 10^{-5}\ \text{J}$$（做负功）

**例3（场强与电势差）**：平行板间距 $d = 5\ \text{cm} = 0.05\ \text{m}$，两板间电压 $U = 100\ \text{V}$，场强？

$$E = \frac{U}{d} = \frac{100}{0.05} = 2000\ \text{N/C} = 2\ \text{kN/C}$$

**例4（移动判断）**：负电荷从 C 到 D，$U_{CD} = -5\ \text{V}$，电场力做功？

$$W_{CD} = qU_{CD} = (-q_0)(-5) = 5q_0 > 0$$（做正功，负电荷向低电势运动，电场力做正功）

### 电势差 vs 电势

| 量 | 符号 | 单位 | 与参考点 |
|----|------|------|---------|
| 电势 | $V$ | V | 有关 |
| 电势差 | $U_{AB}$ | V | 无关 |

### 易错点

1. **电势差有正负**：$U_{AB} = V_A - V_B$，可正可负，$U_{AB} = -U_{BA}$。
2. **负电荷**：$W = qU$，$q < 0$ 时负电荷从高电势到低电势，$W < 0$（做负功）；从低电势到高电势，$W > 0$（做正功）。正好与正电荷相反。
3. **$E = U/d$ 只适用匀强电场**。
