## 🎓 电场线

### 定义

**电场线**：在电场中画出的有向曲线，使曲线上每一点的切线方向代表该点电场强度方向，曲线的疏密表示场强大小的一种形象化工具。

电场线是**假想的辅助线**，不是真实存在的，但能直观描述电场分布。

### 电场线的性质

1. **方向**：每点切线方向 = 该点电场强度方向 = 正电荷受力方向
2. **起止**：从正电荷（或无穷远处）出发，终止于负电荷（或无穷远处）
3. **不相交**：两条电场线永不相交（否则同一点有两个方向，矛盾）
4. **不闭合**：静电场中电场线不形成闭合回路
5. **疏密**：电场线越密处，场强越大；越疏处，场强越小
6. **非导体内**：导体静电平衡时内部无电场线

### 典型电场线图形

| 电场类型 | 电场线形状 |
|---------|-----------|
| 正点电荷 | 向四周辐射（发散） |
| 负点电荷 | 向内汇聚（收敛） |
| 等量异种电荷 | 从正到负的曲线 |
| 等量同种电荷 | 中间区域互相排斥，呈对称分布 |
| 匀强电场（平行板） | 平行等距直线 |
| 带电导体表面 | 垂直导体表面 |

### 电场线与等势面的关系

电场线与等势面**垂直相交**。

电场线方向即电势降低最快的方向（从高电势到低电势）。

### 典型例题

**例1（判断场强）**：A 点电场线比 B 点密，则 A、B 两点场强哪个大？

解：A 点电场线更密，$E_A > E_B$。

**例2（力的方向）**：正电荷在电场线上，受力方向？负电荷？

解：正电荷受力方向沿电场线切线方向（与电场线同向）；负电荷受力方向与电场线切线方向**反向**。

**例3（判断运动）**：正电荷从 A 运动到 B，沿电场线从高密处到低密处，判断是否做正功？

解：正电荷沿电场线方向运动，电场力做**正功**（力与位移方向一致）。

### 电场线的局限

电场线是人为画的线，实际电场是连续分布的，电场线数量是主观选定的（相互比较才有意义）。

### 易错点

1. **电场线不是实际路径**：带电粒子不一定沿电场线运动（有初速度时会偏转）。
2. **电场线不相交**：如果两条线"交叉"，图画错了。
3. **平行板之间是匀强电场**：电场线平行等距，场强处处相等。
4. **导体外表面的电场线垂直于表面**：尖端效应时，尖端处电场线密集（场强大）。
