## 🎓 等势面

### 定义

**等势面**：电场中电势相等的点构成的曲面（三维）或曲线（二维截面称"等势线"）。

$$V = \text{常数}\ \text{（等势面上各点）}$$

### 等势面的性质

1. **处处与电场线垂直**：电场线 ⊥ 等势面（在交点处）
2. **沿等势面移动电荷，电场力不做功**：$W = qU = q(V_A - V_B) = 0$（因为 $V_A = V_B$）
3. **等势面不相交**（否则同一点有两个电势，矛盾）
4. **均匀密集时，电势变化快，场强大**：等势面越密处，场强越大

### 常见等势面形状

| 电场类型 | 等势面形状 |
|---------|-----------|
| 点电荷 | 以点电荷为圆心的同心球面 |
| 匀强电场（平行板） | 平行于极板的平面 |
| 两等量异种电荷 | 复杂曲面（中间等势面为平面） |
| 导体表面 | 导体表面本身就是等势面 |

### 电场线与等势面的关系

- 电场线方向：垂直等势面，从高电势指向低电势
- 等势面越密：场强越大（电势变化率越大）
- $E = -\dfrac{\Delta V}{\Delta n}$（$\Delta n$ 为法线方向距离，即沿场方向的间距）

### 导体的等势性

静电平衡时：
- 导体是等势体（内部各点等势）
- 导体表面是等势面
- 导体内部电场为零（场线不进入导体内部）

### 典型例题

**例1（判断电场力做功）**：电荷沿等势面从 A 移到 B，电场力做多少功？

$$W_{AB} = q(V_A - V_B) = q \times 0 = 0$$

**例2（场强方向）**：画出平行板电容器的等势面，则电场线方向？

解：等势面为平行于极板的直线/平面，电场线垂直极板（从正极板指向负极板）。

**例3（判断场强大小）**：某区域等势面间距越来越小，场强如何变化？

解：等势面越密，电势梯度越大，$E = \Delta V / \Delta n$ 越大，场强越大。

**例4（移动负电荷）**：负电荷从低电势面移到高电势面，电场力做正功还是负功？

$$W = q(V_A - V_B)$$，$q < 0$，$V_{\text{高}} > V_{\text{低}}$，即 $V_B > V_A$（$B$ 为高电势面），

$$W = q(V_A - V_B) = (-|q|)(V_A - V_B) = (-|q|)(\text{负数}) > 0$$

做**正功**（负电荷移向高电势，电场力做正功）。

### 等势面的实际应用

- 屏蔽：用导体壳屏蔽外部电场（内部是等势区，场强为零）
- 避雷针：尖端等势面密集（场强大），容易放电
- 心电图：心脏电势分布在体表形成等势线

### 易错点

1. **等势面不是场线**：两者互相垂直，不要搞混。
2. **沿等势面移动做功为零**：无论路径多长，只要在等势面上，$W = 0$。
3. **导体表面是等势面**：导体内部也是等势区（非等势面，是等势体）。
