## 🎓 角速度

### 定义

**角速度**（$\omega$）：做圆周运动的质点，在极短时间 $\Delta t$ 内转过的角度 $\Delta\theta$（弧度）与时间之比的极限。

$$\omega = \lim_{\Delta t \to 0} \frac{\Delta\theta}{\Delta t} = \frac{\Delta\theta}{\Delta t}$$

单位：**rad/s**（弧度每秒）

### 公式

$$\omega = \frac{\Delta\theta}{\Delta t} = \frac{2\pi}{T} = 2\pi f = 2\pi n$$

（$T$ 为周期，$f$ 为频率，$n$ 为转速）

### 角速度与线速度的关系

$$v = r\omega \implies \omega = \frac{v}{r}$$

半径越大，相同角速度对应线速度越大。

### 弧度制

$1\ \text{rad}$ = 对应弧长等于半径时的圆心角

$$\theta(\text{rad}) = \frac{s}{r}$$

一圈 = $2\pi\ \text{rad} \approx 6.28\ \text{rad}$

| 角度 | 弧度 |
|------|------|
| $360°$ | $2\pi$ |
| $180°$ | $\pi$ |
| $90°$ | $\pi/2$ |
| $60°$ | $\pi/3$ |
| $30°$ | $\pi/6$ |

### 典型例题

**例1**：电风扇转速 $n = 1500\ \text{r/min}$，求角速度。

$$n = \frac{1500}{60} = 25\ \text{r/s}$$
$$\omega = 2\pi n = 2\pi \times 25 = 50\pi \approx 157\ \text{rad/s}$$

**例2**：地球自转角速度？

$$T = 24\ \text{h} = 86400\ \text{s}$$
$$\omega = \frac{2\pi}{T} = \frac{2\pi}{86400} \approx 7.27 \times 10^{-5}\ \text{rad/s}$$

**例3**：$\omega = 4\ \text{rad/s}$，$r = 0.5\ \text{m}$，线速度？

$$v = r\omega = 0.5 \times 4 = 2\ \text{m/s}$$

### 易错点

1. **转速与角速度不同**：$n$（r/s）×$2\pi$ = $\omega$（rad/s）；转速 $n = 1500$ r/min 要先化为 r/s。
2. **角速度是标量**（在高中阶段处理）：大小和方向（绕轴旋转方向），高中通常只算大小。
3. **同轴转动角速度相同**：不论半径大小，同一轴上各点角速度相同。
