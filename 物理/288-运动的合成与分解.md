## 🎓 运动的合成与分解

### 定义

**运动的合成**：已知分运动，求合运动。  
**运动的分解**：已知合运动，按实际效果分解为两个分运动。

合运动与分运动遵循**平行四边形定则**（矢量叠加）：

$$\vec{v} = \vec{v}_1 + \vec{v}_2$$
$$\vec{r} = \vec{r}_1 + \vec{r}_2$$

### 核心原理：运动独立性

各分运动**相互独立**，互不干扰。合运动的位移、速度均为各分量的矢量和。

- 位移合成：$\vec{r} = \vec{r}_x + \vec{r}_y$
- 速度合成：$\vec{v} = \vec{v}_x + \vec{v}_y$

### 分解方式

按**实际效果**或**坐标轴**分解：

$$v_x = v\cos\theta, \quad v_y = v\sin\theta$$
$$v = \sqrt{v_x^2 + v_y^2}, \quad \tan\theta = \frac{v_y}{v_x}$$

### 典型例题

**例1（平抛）**：水平速度 $v_0 = 10\ \text{m/s}$，竖直方向自由落体，$t = 2\ \text{s}$ 时合速度？

$$v_y = gt = 10 \times 2 = 20\ \text{m/s}$$
$$v = \sqrt{v_x^2 + v_y^2} = \sqrt{10^2 + 20^2} = 10\sqrt{5} \approx 22.4\ \text{m/s}$$

**例2（船渡河）**：船速 $v_1 = 3\ \text{m/s}$（垂直河岸），水流速 $v_2 = 4\ \text{m/s}$，合速度？

$$v = \sqrt{3^2 + 4^2} = 5\ \text{m/s}$$
$$\theta = \arctan\frac{4}{3} \approx 53°\ \text{（偏向下游）}$$

**例3**：分解一斜面上物体重力为沿斜面分量和垂直斜面分量：

$$G_{\parallel} = mg\sin\theta, \quad G_{\perp} = mg\cos\theta$$

### 易错点

1. **忘记矢量性**：合速度不是代数相加，必须用矢量合成（平行四边形定则）。
2. **分运动独立性**：水平匀速不影响竖直加速，两者各自独立。
3. **分解不唯一**：运动可沿任意方向分解，应按实际效果或题目坐标系选取分解方向。

### 与必一衔接

- 必修一：平行四边形定则处理力的合成与分解，此处同样适用于运动（位移、速度）。
- 本章应用：平抛运动 = 水平匀速 + 竖直自由落体的合成。
