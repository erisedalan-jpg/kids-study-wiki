## 🎓 回旋加速器（Cyclotron）

### 背景

1930年，美国物理学家劳伦斯（Ernest Lawrence）发明回旋加速器，利用磁场中粒子周期与速度无关的特性，实现带电粒子的**多次加速**，获得高能粒子。劳伦斯因此获得1939年诺贝尔物理学奖。

### 结构

回旋加速器由以下部件组成：

| 部件 | 作用 |
|------|------|
| **两个 D 形盒**（Dees）| 半圆形金属盒，粒子在其中做半圆弧轨道 |
| **匀强磁场**（垂直于 D 形盒）| 使粒子在 D 形盒内做圆周运动 |
| **交变电场**（D 形盒缝隙间）| 每次粒子过缝隙时加速粒子 |

### 工作原理

1. 粒子从中心出发，进入第一个 D 形盒，在磁场中做半圆运动（半径 $r_1$）。
2. 粒子到达缝隙，被交变电场加速，速度增大。
3. 粒子进入第二个 D 形盒，做半径更大的半圆（$r_2 > r_1$）。
4. 再次经过缝隙加速，如此循环，轨道螺旋形向外扩大。
5. 粒子到达 D 形盒边缘时，以最大速度射出，打到靶上。

### 关键公式

每次加速后的粒子在磁场中的半径：

$$r = \frac{mv}{qB}$$

加速电压为 $U$ 时，每次过缝隙增加动能 $qU$，经 $n$ 次加速后：

$$\frac{1}{2}mv^2 = nqU \Rightarrow v = \sqrt{\frac{2nqU}{m}}$$

最终轨道半径：

$$r_{max} = \frac{mv_{max}}{qB}$$

### 回旋频率（核心）

$$f = \frac{qB}{2\pi m}$$

**这是回旋加速器能工作的原因**：粒子的回旋频率与速度无关，与交变电场频率保持同步，每次经过缝隙时电场方向刚好朝向加速方向。

### 最大动能限制

回旋加速器受到相对论效应限制：速度接近光速时，粒子质量增大（相对论质量），回旋频率降低，与电场不再同步（**共振失调**），因此有最大动能限制。

解决方法：**同步回旋加速器**（随速度调整磁场或电场频率）。

### 应用

| 应用 | 说明 |
|------|------|
| 核物理研究 | 加速质子、$\alpha$ 粒子打靶产生核反应 |
| 医学（PET 扫描）| 加速质子轰击靶产生正电子放射性核素（如 $^{18}F$）|
| 粒子治疗 | 质子/碳离子束精准治疗癌症 |
| 放射性同位素生产 | 医疗和科研用放射性药物 |

### 典型例题

**例**：回旋加速器中磁场 $B = 0.5\,\text{T}$，质子（$m = 1.67\times10^{-27}\,\text{kg}$，$q = 1.6\times10^{-19}\,\text{C}$），求回旋频率。

$$f = \frac{qB}{2\pi m} = \frac{1.6\times10^{-19}\times0.5}{2\pi\times1.67\times10^{-27}} \approx 7.6\times10^6\,\text{Hz} = 7.6\,\text{MHz}$$

### 易错点

- 回旋加速器可以反复加速，每次增加动能 $qU$，但电场做功、磁场**不做功**。
- 相对论效应是普通回旋加速器能量上限的原因，不是设备缺陷。
- 轨道是**螺旋线**（俯视），不是圆（每半圈半径增大）。

### 与必修二衔接

必修二中学习了匀速圆周运动的周期；本节利用"$T$ 与 $v$ 无关"这一特殊结论，使反复加速变为可能，是力学与电磁学综合应用的经典范例。
