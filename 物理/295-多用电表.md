## 🎓 多用电表（Multimeter）

### 定义

**多用电表**（万用表）是能测量多种电学量的仪器，基本功能包括：测电阻（Ω）、直流电压（V DC）、交流电压（V AC）、直流电流（A DC）。

### 内部结构

核心是**灵敏电流计**（G），配合不同的附加电路：

| 功能 | 附加元件 |
|------|---------|
| 电流表 | 并联分流电阻 |
| 电压表 | 串联分压电阻 |
| 欧姆表 | 内部电池 + 串联电阻 |

### 欧姆表测电阻原理

欧姆表内部含**干电池**作为电源。测量时：

$$I = \frac{\varepsilon}{R_{内} + R_x}$$

$R_x$（被测电阻）越大 → $I$ 越小 → 指针偏转越小。

**欧姆表刻度**：从左到右阻值从大到小，**中值电阻**等于欧姆表内阻。

### 使用步骤

1. **机械调零**：未使用时指针指向电流零刻线（最左边）。
2. **选挡**：根据被测量的类型和大小选择合适的量程（先选大量程）。
3. **欧姆调零（仅测电阻时）**：两表笔短接，调节使指针指向电阻零刻线（最右边）。
4. **测量**：将表笔接在被测量两端，读数。
5. **换挡重调零**：每次换欧姆挡量程都需重新欧姆调零。
6. **使用完毕**：将选择开关拨到交流电压最高挡或"OFF"挡。

### 量程选择原则

- 电压/电流：选使指针偏转在**满偏 1/3 到 2/3 之间**的量程（读数误差小）。
- 电阻：选使指针偏在中值附近（中间 1/3 区域）的量程。

### 测电阻注意事项

| 规则 | 原因 |
|------|------|
| 被测电阻必须**断电** | 欧姆表内有电池，外部供电会损坏表或影响读数 |
| 被测电阻必须**从电路中断开** | 并联路径会影响测量结果 |
| 手不能触碰表笔金属部分 | 人体电阻会影响测量 |
| 每换挡都要重新欧姆调零 | 不同挡内阻不同 |

### 指针式 vs 数字式

| 类型 | 优点 | 缺点 |
|------|------|------|
| 指针式（模拟）| 可见变化趋势，适合动态观测 | 读数需估计，有量程误差 |
| 数字式 | 读数准确，内阻大 | 响应慢，不易看趋势 |

高中实验以指针式为主。

### 易错点

- 欧姆表刻度非线性，**从右到左阻值增大**（与电流、电压刻度方向相反）。
- 测电阻时，红表笔接表内电池**负极**（电流从黑表笔流出），与电压表红正黑负相反。
- 每次换量程后**必须重新调零**，否则中值电阻不等于当前内阻，误差大。

### 与必修二衔接

初中学过电流表和电压表；本节将两者整合为多用电表，并增加欧姆表功能，是实验物理中最常用的综合测量工具。
