## 🎓 带电粒子在电场中偏转

### 物理模型

带电粒子以初速度 $v_0$（水平方向）进入**匀强电场**，电场方向垂直于初速度。粒子做**类平抛运动**：水平方向匀速，竖直方向（沿电场力方向）匀加速。

### 运动分析

设极板长度为 $L$，极板间距为 $d$，极板间电压为 $U$，粒子质量 $m$，电荷量 $q$。

**水平方向（匀速）：**

$$x = v_0 t \Rightarrow t = \frac{L}{v_0}$$

**竖直方向（匀加速）：**

加速度：

$$a = \frac{qE}{m} = \frac{qU}{md}$$

偏转位移（在极板内）：

$$y_1 = \frac{1}{2}at^2 = \frac{qUL^2}{4mdv_0^2}$$

离开极板时竖直速度：

$$v_y = at = \frac{qUL}{mdv_0}$$

**偏转角（离开极板时）：**

$$\tan\theta = \frac{v_y}{v_0} = \frac{qUL}{mdv_0^2}$$

### 屏上偏移量

设离开极板后再飞行距离 $l$ 到达荧光屏：

$$y_2 = l\tan\theta = \frac{qULl}{mdv_0^2}$$

总偏移：

$$y = y_1 + y_2 = \frac{qUL^2}{4mdv_0^2} + \frac{qULl}{mdv_0^2}$$

### 与类平抛运动对比

| 量 | 类平抛（重力场）| 电场偏转 |
|---|---|---|
| 水平加速度 | 0 | 0 |
| 竖直加速度 | $g$ | $qU/(md)$ |
| 轨迹 | 抛物线 | 抛物线 |

### 典型例题

**例**：示波管偏转板长 $L = 4\,\text{cm}$，板间距 $d = 1\,\text{cm}$，电压 $U = 200\,\text{V}$，电子以 $v_0 = 10^7\,\text{m/s}$ 水平射入，求极板内偏转位移 $y_1$。

$$a = \frac{eU}{md} = \frac{1.6\times10^{-19}\times200}{9.1\times10^{-31}\times0.01} \approx 3.52\times10^{15}\,\text{m/s}^2$$

$$t = \frac{L}{v_0} = \frac{0.04}{10^7} = 4\times10^{-9}\,\text{s}$$

$$y_1 = \frac{1}{2}at^2 = \frac{1}{2}\times3.52\times10^{15}\times(4\times10^{-9})^2 \approx 0.028\,\text{m} = 2.8\,\text{cm}$$

（注：此值已接近 $d/2$，实际需验证粒子不打到极板上）

### 易错点

- 电场偏转是**类平抛**，水平方向速度不变，竖直方向匀加速，两方向运动**相互独立**。
- 偏转角公式 $\tan\theta = v_y/v_0$，注意分析的是**离开电场时**的速度方向。
- 粒子必须从两极板正中间射入时才不会打到板上（需验证 $y_1 < d/2$）。

### 与必修二衔接

必修二学习了抛体运动（分运动合成）；本节将重力替换为电场力，模型完全相同，是力学运动学在电学中的典型应用。
