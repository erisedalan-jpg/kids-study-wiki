## 🎓 物理量单位制（高中版）

### 国际单位制（SI）

**国际单位制**（Système International d'Unités，SI）是全球通用的科学计量体系，高中物理所有计算均在 SI 单位下进行。

### 七个基本单位

| 物理量 | 基本单位 | 符号 |
|--------|---------|------|
| 长度 | 米 | m |
| 质量 | 千克 | kg |
| 时间 | 秒 | s |
| 电流 | 安培 | A |
| 热力学温度 | 开尔文 | K |
| 物质的量 | 摩尔 | mol |
| 发光强度 | 坎德拉 | cd |

高中物理常用前五个。

### 必修三涉及的导出单位

| 物理量 | 单位名 | 符号 | 用基本单位表示 |
|--------|--------|------|--------------|
| 电荷量 | 库仑 | C | A·s |
| 电压（电势差）| 伏特 | V | kg·m²/(A·s³) |
| 电容 | 法拉 | F | C/V = A²·s⁴/(kg·m²) |
| 电阻 | 欧姆 | Ω | V/A = kg·m²/(A²·s³) |
| 电动势 | 伏特 | V | 同电压 |
| 磁感应强度 | 特斯拉 | T | kg/(A·s²) = V·s/m² |
| 磁通量 | 韦伯 | Wb | V·s = kg·m²/(A·s²) |
| 电阻率 | 欧姆·米 | Ω·m | — |

### 单位换算口诀

**电学常用换算**：

$$1\,\text{F} = 10^6\,\mu\text{F} = 10^{12}\,\text{pF}$$

$$1\,\text{kW} = 10^3\,\text{W}, \quad 1\,\text{kWh} = 3.6\times10^6\,\text{J}$$

$$1\,\text{eV} = 1.6\times10^{-19}\,\text{J}$$（电子伏特，粒子物理常用）

### 量纲分析

**量纲分析**是验证公式正确性的方法——等式两边量纲必须相同。

**例**：验证 $F = qvB$ 的量纲：

左边：$[F] = \text{N} = \text{kg·m/s}^2$

右边：$[qvB] = \text{C·(m/s)·T} = \text{A·s·(m/s)·(kg/(A·s}^2)) = \text{kg·m/s}^2$

两边量纲相同，公式通过量纲验证。

### 常见前缀

| 前缀 | 符号 | 倍数 |
|------|------|------|
| 太 | T | $10^{12}$ |
| 吉 | G | $10^9$ |
| 兆 | M | $10^6$ |
| 千 | k | $10^3$ |
| 毫 | m | $10^{-3}$ |
| 微 | μ | $10^{-6}$ |
| 纳 | n | $10^{-9}$ |
| 皮 | p | $10^{-12}$ |

### 高考中的单位要求

- 计算结果必须注明单位
- 中间过程可不注明，但最终结果必须有单位
- 单位不正确通常不得分
- 导出单位可以用基本单位表示（如 N = kg·m/s²）

### 易错点

- 电容单位 F（法拉）非常大，实际常用 μF 和 pF，代入计算时需统一为基本单位（F）。
- 磁感应强度 T（特斯拉）不要与时间 t（秒 s）混淆（大小写很重要）。
- 电阻率单位是 Ω·m（欧姆·米），不是 Ω/m。

### 与必修二衔接

必修二已建立力学单位体系（N、J、W 等）；本节在此基础上引入电学和磁学单位（V、A、T、Wb 等），构成高中物理完整的 SI 单位体系。
