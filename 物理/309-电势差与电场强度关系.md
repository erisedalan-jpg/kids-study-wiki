## 🎓 电势差与电场强度关系

### 核心公式

在**匀强电场**中，沿电场方向两点 A、B 的电势差与电场强度的关系：

$$U_{AB} = Ed$$

即：

$$E = \frac{U_{AB}}{d}$$

其中：
- $E$：匀强电场强度（V/m 或 N/C）
- $U_{AB} = \varphi_A - \varphi_B$：A、B 两点电势差（V）
- $d$：A、B 两点**沿电场线方向**的距离（m）

> ⚠️ $d$ 是 A、B 连线在电场方向上的**投影长度**，不是 A、B 的直线距离（除非 AB 连线与电场平行）。

### 公式推导

电场力对电荷 $q$ 从 A 到 B 做功：

$$W_{AB} = qEd \quad \text{（电场力做功公式）}$$

又由电势差定义：

$$W_{AB} = q(\varphi_A - \varphi_B) = qU_{AB}$$

两式联立得：

$$qEd = qU_{AB} \Rightarrow \boxed{E = \frac{U_{AB}}{d}}$$

### 电场线密度与场强

等势面间距越小（等势面越密），说明单位长度上电势变化越大，场强越大。类比地形图：等高线越密，坡度越陡。

$$E \propto \frac{\Delta\varphi}{\Delta d}$$

### 适用条件

| 条件 | 结论 |
|------|------|
| 匀强电场 | $E = U/d$ 精确成立 |
| 非匀强电场 | 只能用于微元，整体不适用 |
| $d$ 方向 | 必须是沿电场方向的分量 |

### 典型例题

**例**：平行板电容器两板间距 $d = 5\,\text{mm} = 5\times10^{-3}\,\text{m}$，两板电势差 $U = 150\,\text{V}$，求板间匀强电场强度。

$$E = \frac{U}{d} = \frac{150}{5\times10^{-3}} = 3\times10^4\,\text{V/m}$$

### 易错点

- $d$ 取电场方向分量，若带电粒子斜向移动，$d \ne$ 路径长度。
- 单位等价：$1\,\text{V/m} = 1\,\text{N/C}$（两种场强单位均正确）。
- 非匀强电场（如点电荷场）**不能**直接用 $E = U/d$ 计算整体场强。

### 与必修二衔接

必修二中力做功 $W = Fs\cos\theta$；本节将其用于电场力，得出 $W = qEd$，再联系电势差定义推出 $E = U/d$，统一了力学和电学的能量描述。
