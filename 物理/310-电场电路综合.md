## 🎓 电场电路综合

### 概述

**电场电路综合题**将静电场（电容器、带电粒子运动）与电路（电动势、内阻、欧姆定律）结合，是高中物理的重要综合题型。

### 典型结构

最常见的模型：

**平行板电容器 + 含源电路**

- 平行板电容器接在电路中，两端电压 = 路端电压（或某段电路的端电压）
- 改变电路中的电阻 → 路端电压变化 → 电容器电场强度变化 → 粒子运动状态变化

### 核心分析步骤

1. **电路分析**：用闭合电路欧姆定律求路端电压 $U$，即电容器两端电压。
   $$U = \varepsilon - Ir = \varepsilon \cdot \frac{R}{R+r}$$

2. **电容器内电场**：由 $E = U/d$ 求电场强度（$d$ 为板间距）。

3. **带电粒子运动**：分析粒子在电场中的受力（电场力 $qE$、重力 $mg$ 等），判断运动状态。

### 常见考题变化

| 操作 | 路端电压 $U$ | 电场 $E_{cap}$ | 粒子状态 |
|------|------------|--------------|---------|
| 增大外电阻 $R$ | 增大 | 增大 | 粒子加速度增大 |
| 减小外电阻 $R$ | 减小 | 减小 | 粒子加速度减小 |
| 移动平行板（改 $d$） | 不变（$U$ 由电路决定）| $E = U/d$ 改变 | 运动改变 |
| 电容器与电源断开后移板 | $Q$ 不变 → $U$ 改变 | $E = Q/(\varepsilon_0 S)$ 不变 | 运动不变 |

### 典型例题

**例**：电路：$\varepsilon = 12\,\text{V}$，$r = 2\,\Omega$，外接电阻 $R_1 = 4\,\Omega$，$R_2 = 6\,\Omega$（$R_2$ 与平行板电容器并联），$R_1$ 与并联组串联后接电源。

平行板间距 $d = 0.01\,\text{m}$，板间有一带正电粒子（$m = 10^{-6}\,\text{kg}$，$q = 10^{-8}\,\text{C}$），判断粒子的平衡条件。

**解**：

$R_{并} = R_2 = 6\,\Omega$（电容器稳定时相当于断路，只有 $R_2$ 并联）

$$I = \frac{\varepsilon}{R_1 + R_2 + r} = \frac{12}{4+6+2} = 1\,\text{A}$$

$$U_{电容} = U_{R_2} = IR_2 = 1\times6 = 6\,\text{V}$$

$$E = \frac{U}{d} = \frac{6}{0.01} = 600\,\text{V/m}$$

$$F_电 = qE = 10^{-8}\times600 = 6\times10^{-6}\,\text{N}$$

$$G = mg = 10^{-6}\times10 = 10^{-5}\,\text{N}$$

$F_电 < G$，粒子无法仅靠电场力悬停，需额外讨论。

### 与磁场的联合考题

若电容器两端同时接有带电粒子偏转磁场，则需综合计算：
1. 电路求电压 → 电容器电场 → 粒子受力
2. 磁场 → 洛伦兹力 → 粒子运动方程

### 易错点

- **电容器稳定状态下相当于断路**（无直流电流通过），不要将其视为普通电阻计算。
- 改变电容器板间距：若与电源相连，$U$ 不变；若断开，$Q$ 不变，需区分。
- 粒子平衡需同时满足竖直和水平方向合力为零，不能只看一个方向。

### 与必修二衔接

必修二建立了力的平衡和牛顿定律；本节将电场力纳入力的平衡分析，并与实际电路结合，是静电场和电路知识的综合运用，常见于高考第三题（压轴）。
