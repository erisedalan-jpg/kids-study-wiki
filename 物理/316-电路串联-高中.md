## 🎓 电路串联（高中版）

### 基本特征

**串联电路**：各元件依次首尾相连，构成**唯一电流通路**。

| 物理量 | 串联规律 |
|--------|---------|
| 电流 | 各处相同：$I = I_1 = I_2 = \cdots = I_n$ |
| 电压 | 各分压之和等于总电压：$U = U_1 + U_2 + \cdots + U_n$ |
| 电阻 | 等效电阻等于各阻之和：$R = R_1 + R_2 + \cdots + R_n$ |

### 分压规律

各电阻两端电压与电阻成正比：

$$\frac{U_1}{U_2} = \frac{R_1}{R_2}$$

具体分压：

$$U_1 = U \cdot \frac{R_1}{R_1 + R_2}$$

### 功率分配

串联电路中电功率之比等于电阻之比：

$$\frac{P_1}{P_2} = \frac{I^2 R_1}{I^2 R_2} = \frac{R_1}{R_2}$$

### 含电源的串联电路（全路欧姆定律）

$$\varepsilon = U_R + U_r = IR + Ir$$

其中 $R$ 为外电路等效电阻，$r$ 为电源内阻，两者实质上也是"串联"关系。

### 高中重点：含内阻的分析

**例**：电源 $\varepsilon = 9\,\text{V}$，$r = 1\,\Omega$，外接两个电阻串联 $R_1 = 2\,\Omega$，$R_2 = 6\,\Omega$，求各量。

$$I = \frac{\varepsilon}{R_1 + R_2 + r} = \frac{9}{2+6+1} = 1\,\text{A}$$

$$U_{R_1} = IR_1 = 2\,\text{V},\quad U_{R_2} = IR_2 = 6\,\text{V},\quad U_r = Ir = 1\,\text{V}$$

路端电压：$U = \varepsilon - Ir = 9 - 1 = 8\,\text{V}$（等于 $U_{R_1} + U_{R_2}$）

### 应用：分压器

利用串联电阻的分压特性，可以从较大电压中取出所需电压（分压电路），广泛用于电子仪器中。

### 易错点

- 串联电路中任一处断路，**整个电路**都断路（全部不通）。
- 各电阻消耗的功率与电阻成**正比**（电流相同，$P = I^2R$）。
- 高中含电源电路：内阻 $r$ 是与外电阻**串联**的，路端电压不等于电动势。

### 与必修二衔接

初中已学过串联电路基本规律；高中在此基础上引入**含内阻的电源**，使串联分析更贴近真实电路，并与全路欧姆定律紧密结合。
