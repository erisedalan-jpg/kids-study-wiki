## 🎓 电路并联（高中版）

### 基本特征

**并联电路**：各元件**两端分别相连**，有多条并行电流通路。

| 物理量 | 并联规律 |
|--------|---------|
| 电压 | 各支路相同：$U = U_1 = U_2 = \cdots = U_n$ |
| 电流 | 干路等于各支路之和：$I = I_1 + I_2 + \cdots + I_n$ |
| 电阻 | 等效电阻倒数等于各支路倒数之和：$\dfrac{1}{R} = \dfrac{1}{R_1} + \dfrac{1}{R_2} + \cdots$ |

两电阻并联时的等效公式：

$$R = \frac{R_1 R_2}{R_1 + R_2}$$

### 分流规律

各支路电流与电阻成反比：

$$\frac{I_1}{I_2} = \frac{R_2}{R_1}$$

具体分流：

$$I_1 = I \cdot \frac{R_2}{R_1 + R_2}$$

### 功率分配

并联电路中电功率之比等于电阻的倒数之比：

$$\frac{P_1}{P_2} = \frac{U^2/R_1}{U^2/R_2} = \frac{R_2}{R_1}$$

即：**电阻小的支路消耗功率大**（并联时相反于串联）。

### 含电源的并联电路

外电路若为多支路并联，计算时先求等效外阻 $R$，再代入全路欧姆定律：

$$I = \frac{\varepsilon}{R + r}$$

各支路电流按分流公式计算，各支路电压均等于路端电压 $U = \varepsilon - Ir$。

### 典型例题

**例**：电源 $\varepsilon = 6\,\text{V}$，$r = 0.5\,\Omega$，外电路并联 $R_1 = 2\,\Omega$ 和 $R_2 = 3\,\Omega$，求各支路电流。

$$R_{外} = \frac{R_1 R_2}{R_1 + R_2} = \frac{2\times3}{5} = 1.2\,\Omega$$

$$I = \frac{\varepsilon}{R_{外} + r} = \frac{6}{1.2+0.5} = \frac{6}{1.7} \approx 3.53\,\text{A}$$

$$U = \varepsilon - Ir = 6 - 3.53\times0.5 \approx 4.24\,\text{V}$$

$$I_1 = \frac{U}{R_1} = \frac{4.24}{2} = 2.12\,\text{A},\quad I_2 = \frac{U}{R_2} = \frac{4.24}{3} \approx 1.41\,\text{A}$$

（验证：$I_1 + I_2 = 3.53\,\text{A}$ ✓）

### 应用

- 家庭电路：所有用电器并联，保证各自独立工作，电压一致（220 V）。
- 电流表扩程：并联**分流电阻**，使大部分电流绕过电流表。

### 易错点

- 并联中某支路断路，**其他支路**正常工作（与串联不同）。
- 并联后等效电阻**小于任一支路电阻**。
- 各支路功率之比等于电阻的**倒数**之比（与串联相反）。

### 与必修二衔接

初中已学过并联基本规律；高中在引入内阻和全路欧姆定律后，并联电路的分析与实际电源结合，需要综合运用串并联等效、全路欧姆定律等多个知识点。
