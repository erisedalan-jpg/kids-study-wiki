## 🎓 电阻率（Resistivity）

### 定义

**电阻率**（$\rho$）是描述材料导电能力的物理量，反映材料对电流阻碍的内在属性，与导体形状和大小无关。

由 [[318-电阻定律|电阻定律]] $R = \rho L/S$，可得：

$$\rho = \frac{RS}{L}$$

单位：$\Omega\cdot\text{m}$（欧姆·米）

### 常见材料电阻率（20°C）

| 材料 | 电阻率 $\rho$（$\Omega\cdot\text{m}$）| 类别 |
|------|--------------------------------------|------|
| 银（Ag）| $1.6\times10^{-8}$ | 导体 |
| 铜（Cu）| $1.7\times10^{-8}$ | 导体 |
| 铝（Al）| $2.9\times10^{-8}$ | 导体 |
| 钨（W）| $5.3\times10^{-8}$ | 导体 |
| 锰铜合金 | $4.4\times10^{-7}$ | 导体（高电阻丝）|
| 硅（Si）| $6.4\times10^{2}$ | 半导体 |
| 玻璃 | $10^{10} \sim 10^{14}$ | 绝缘体 |
| 橡胶 | $\sim 10^{13}$ | 绝缘体 |

导体 < 半导体 < 绝缘体（相差悬殊，相差可达 $10^{20}$ 倍以上）

### 电阻率与温度的关系

**金属（导体）**：温度升高，电阻率增大（正温度系数）

$$\rho = \rho_0(1 + \alpha t)$$

其中 $\alpha$ 为温度系数，$t$ 为摄氏温度。金属灯丝热态电阻比冷态大许多倍。

**半导体**：温度升高，电阻率减小（负温度系数）——热敏电阻工作原理。

**超导体**：温度降至临界温度以下，电阻率变为零（见 [[328-超导体|超导体]]）。

### 电阻丝的选材依据

| 用途 | 选材要求 |
|------|----------|
| 输电导线 | 低电阻率（铜、铝）、机械强度好 |
| 电热元件（电炉丝）| 高电阻率、高熔点（镍铬合金）|
| 标准电阻 | 电阻率与温度无关（锰铜合金）|
| 灯泡灯丝 | 高熔点（钨，熔点 3422°C）|

### 典型例题

**例**：某电阻丝材料电阻率 $\rho = 1.1\times10^{-6}\,\Omega\cdot\text{m}$，需要制作阻值 $R = 44\,\Omega$、截面积 $S = 0.5\,\text{mm}^2 = 5\times10^{-7}\,\text{m}^2$ 的电阻丝，求所需长度。

$$L = \frac{RS}{\rho} = \frac{44\times5\times10^{-7}}{1.1\times10^{-6}} = 20\,\text{m}$$

### 易错点

- 电阻率是材料固有属性，与导体形状无关，但与**温度**有关。
- 金属电阻率随温度升高而增大，半导体恰好相反。
- 铜的电阻率略大于银，但铜价格低廉，工业上更常用。

### 与必修二衔接

初中定性了解电阻受材料影响；本节引入电阻率这一物理量，将材料对电阻的影响定量化，与电阻定律共同构成完整的电阻理论。
