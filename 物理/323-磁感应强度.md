## 🎓 磁感应强度（Magnetic Flux Density）

### 定义

**磁感应强度**（$B$）是描述磁场强弱和方向的物理量，通过通电导体在磁场中所受安培力来定义：

$$B = \frac{F}{IL}$$

其中：
- $F$：通电导体所受安培力（N）
- $I$：电流强度（A）
- $L$：导体在磁场中的有效长度（m，且导体方向垂直于磁场）

单位：**特斯拉**（T），$1\,\text{T} = 1\,\text{N/(A·m)} = 1\,\text{Wb/m}^2$

### 物理意义

$B$ 是磁场的**固有属性**，反映磁场对放入其中的电流（或运动电荷）施力能力的大小，与放入其中的导体无关。

### 方向

磁感应强度的方向规定为：该点**小磁针北极**静止时的指向，即磁感线的切线方向（指向 N 极一侧）。

### 常见磁场的磁感应强度

| 磁场 | $B$ 大小（大约值）|
|------|-----------------|
| 地球表面地磁场 | $\sim 5\times10^{-5}\,\text{T}$ |
| 普通永磁体 | $0.1 \sim 1\,\text{T}$ |
| 强电磁铁（实验室）| $1 \sim 10\,\text{T}$ |
| MRI 核磁共振仪 | $1.5 \sim 3\,\text{T}$ |
| 中子星表面 | $\sim 10^8\,\text{T}$ |

### 磁感应强度的矢量性

$B$ 是矢量，同一点处多个磁场的磁感应强度应按**矢量叠加**。

### 安培力公式

$$F = BIL\sin\theta$$

其中 $\theta$ 为电流方向与磁场方向的夹角。当 $\theta = 90°$（电流垂直磁场）时，$F$ 最大 $= BIL$；当 $\theta = 0°$（电流平行磁场）时，$F = 0$。

### 洛伦兹力公式

$$f = qvB\sin\theta$$

带电量 $q$、速度 $v$ 的粒子在磁场中受到的洛伦兹力（$\theta$ 为 $v$ 与 $B$ 的夹角）。

### 典型例题

**例**：某匀强磁场中，长 $L = 0.5\,\text{m}$ 的导线通电流 $I = 2\,\text{A}$，电流方向垂直于磁场，受安培力 $F = 0.3\,\text{N}$，求磁感应强度。

$$B = \frac{F}{IL} = \frac{0.3}{2\times0.5} = 0.3\,\text{T}$$

### 易错点

- $B = F/(IL)$ 是磁感应强度的**定义式**，不能理解为 $B$ 随 $F$ 增大而增大。
- 磁感应强度是描述磁场本身的量，与放入的导线属性（粗细、材料等）无关。
- 当导线与磁场**平行**时，$F = 0$（但磁场仍存在，$B \ne 0$）。

### 与必修二衔接

初中学过安培力的定性概念；高中引入磁感应强度作为定量描述磁场的物理量，类似于电场中引入电场强度 $E$ 的方式，形成完整的场论框架。
