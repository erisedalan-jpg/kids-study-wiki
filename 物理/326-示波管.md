## 🎓 示波管（Cathode Ray Tube）

### 结构组成

示波管（阴极射线管）由以下部件组成：

| 部件 | 功能 |
|------|------|
| **电子枪** | 产生高速电子束（阴极加热发射电子，加速阳极加速） |
| **竖直偏转板（Y 轴）** | 使电子束竖直偏转（通常接待测信号） |
| **水平偏转板（X 轴）** | 使电子束水平偏转（通常接锯齿波扫描信号） |
| **荧光屏** | 电子轰击后发光，显示波形 |
| **真空管壁** | 维持高真空，防止电子与气体分子碰撞 |

### 工作原理

1. **产生电子束**：阴极加热后发射热电子，经阳极高压加速（利用 [[299-带电粒子在电场中加速|带电粒子在电场中加速]] 原理）。
2. **竖直偏转**：Y 轴偏转板加载待测电压，电子束在竖直方向产生偏转（利用 [[298-带电粒子在电场中偏转|带电粒子在电场中偏转]] 原理）。
3. **水平扫描**：X 轴偏转板加载周期性锯齿波，使电子束在水平方向匀速扫过屏幕。
4. **显示波形**：两个方向运动合成，在荧光屏上描出随时间变化的电压波形。

### 加速电压与电子速度

电子（$m_e = 9.11\times10^{-31}\,\text{kg}$，$e = 1.6\times10^{-19}\,\text{C}$）经加速电压 $U_a$ 后：

$$v = \sqrt{\frac{2eU_a}{m_e}}$$

### 偏转灵敏度

偏转灵敏度定义为：单位偏转电压产生的屏幕偏移量（mm/V）。

$$S = \frac{y}{U} = \frac{Ll}{2dU_a}$$

其中 $L$ 为偏转板长度，$l$ 为偏转板末端到屏的距离，$d$ 为板间距，$U_a$ 为加速电压。

提高灵敏度：增大 $L$、$l$，减小 $d$ 和 $U_a$（但 $U_a$ 减小会使亮度降低）。

### 历史与现代替代

示波管是 20 世纪电子仪器的核心，广泛用于电视机（CRT 电视）、雷达显示器、早期计算机显示器。现已被**液晶显示器（LCD）**和**有机发光二极管（OLED）**基本取代，但示波管在物理教学和部分专业示波器中仍在使用。

### 易错点

- 示波管中电子做的是**类平抛运动**，而非圆周运动（磁偏转才是圆周运动）。
- 加速电压 $U_a$ 越高，电子速度越大，偏转角越小（灵敏度降低，但亮度增加）。
- 真空是必要条件，否则电子与空气分子碰撞，无法形成清晰电子束。

### 与必修二衔接

必修二学习了运动的合成与分解、功能定理；示波管将加速（能量转换）和偏转（类平抛）两个知识点综合应用，是静电场知识的综合体现。
