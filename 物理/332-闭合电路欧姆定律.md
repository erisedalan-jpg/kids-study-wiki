## 🎓 闭合电路欧姆定律（全路欧姆定律）

### 核心公式

$$\varepsilon = I(R + r)$$

即：

$$I = \frac{\varepsilon}{R + r}$$

其中：
- $\varepsilon$：电源电动势（V）
- $I$：电路电流（A）
- $R$：外电路等效电阻（$\Omega$）
- $r$：电源内阻（$\Omega$）

### 推导（能量角度）

电源在时间 $t$ 内做的总功：

$$W_{总} = \varepsilon q = \varepsilon It$$

内阻消耗：$W_r = I^2rt$

外电路消耗：$W_R = UIt = I^2Rt$

由能量守恒：$\varepsilon It = I^2Rt + I^2rt$

两边除以 $It$：$\varepsilon = IR + Ir$，即 $I = \dfrac{\varepsilon}{R+r}$

### 路端电压

$$U = IR = \varepsilon - Ir$$

| 情形 | $I$ | $Ir$（内阻压降）| $U$（路端电压）|
|------|-----|----------------|----------------|
| 断路 $R \to \infty$ | 0 | 0 | $U = \varepsilon$ |
| 正常工作 | $\varepsilon/(R+r)$ | $Ir$ | $\varepsilon - Ir < \varepsilon$ |
| 短路 $R = 0$ | $\varepsilon/r$（最大）| $\varepsilon$ | 0 |

**结论**：外阻越小，电流越大，路端电压越低。

### U-I 图像

以路端电压 $U$ 为纵轴、电流 $I$ 为横轴：

$$U = -rI + \varepsilon$$

- 纵截距 = $\varepsilon$（断路时路端电压）
- 斜率绝对值 = $r$（内阻）
- 横截距 = $\varepsilon/r$（短路电流）

利用 U-I 图可以用图像法测量电动势和内阻。

### 典型例题

**例**：电源 $\varepsilon = 12\,\text{V}$，$r = 2\,\Omega$，分别求外阻 $R = 10\,\Omega$ 和 $R = 4\,\Omega$ 时的电流和路端电压。

$R = 10\,\Omega$：$I = 12/12 = 1\,\text{A}$，$U = 12-1\times2 = 10\,\text{V}$

$R = 4\,\Omega$：$I = 12/6 = 2\,\text{A}$，$U = 12-2\times2 = 8\,\text{V}$

外阻减小 → 电流增大 → 路端电压减小（内阻影响加剧）。

### 最大输出功率

当外阻 $R = r$（内外阻相等）时，外电路输出功率最大：

$$P_{max} = \frac{\varepsilon^2}{4r}$$

### 易错点

- 全路欧姆定律只适用于**纯电阻**电路（外电路无电动机等非纯电阻）。
- 路端电压随电流**线性减小**，不是非线性关系。
- 电源额定工作时，路端电压通常小于电动势（标称电压是近似值）。

### 与必修二衔接

初中欧姆定律 $I = U/R$ 针对外电路；本节将电源内部纳入分析，建立包含电源在内的**完整回路**分析方法，是高中电学最核心的公式之一。
