## 🎓 霍尔效应（Hall Effect）

### 发现

1879年，美国物理学家霍尔（Edwin Hall）发现：通电导体（或半导体）放入垂直于电流的磁场中，在垂直于电流和磁场的方向上会产生**横向电压**，称为**霍尔电压**。

### 原理

**步骤分析**：

1. 导体（厚度 $d$，宽度 $b$）通电流 $I$，载流子（如电子，速度 $v_d$）沿电流方向运动。
2. 磁场 $B$ 垂直于导体平面。
3. 洛伦兹力使载流子**侧向偏移**，积累在一侧，产生横向电场（霍尔电场 $E_H$）。
4. 当横向电场力与洛伦兹力平衡时，达到稳态：

$$qE_H = qv_dB \Rightarrow E_H = v_dB$$

5. 霍尔电压：

$$U_H = E_H \cdot b = v_d B b$$

**霍尔系数**：

$$R_H = \frac{1}{nq}$$

其中 $n$ 为载流子数密度，完整表达：

$$U_H = \frac{IB}{nqd} = R_H \frac{IB}{d}$$

### 判断载流子类型

霍尔效应可以判断导体中**载流子的极性**：

- 若霍尔电压正负方向与正电荷（空穴）偏移方向一致 → 载流子为**正电荷**（P 型半导体）
- 若霍尔电压正负方向与电子偏移方向一致 → 载流子为**负电荷**（N 型半导体或金属）

金属中载流子是电子（负电荷），而半导体中可能是电子或空穴，霍尔效应是区分它们的重要实验手段。

### 霍尔传感器的应用

| 应用 | 原理 |
|------|------|
| **磁场测量** | $U_H \propto B$，测霍尔电压即可得 $B$ |
| **电流传感器** | 非接触测量大电流（$U_H \propto I$）|
| **位置传感器** | 车轮转速传感器（ABS 系统）|
| **无刷电机换向** | 检测转子磁极位置 |
| **智能手机** | 感应翻盖磁铁位置（休眠功能）|

### 典型例题

**例**：铜导体（$n = 8.4\times10^{28}\,\text{m}^{-3}$）厚 $d = 1\,\text{mm}$，通电流 $I = 10\,\text{A}$，置于 $B = 1\,\text{T}$ 的磁场中，求霍尔电压。

$$U_H = \frac{IB}{nqd} = \frac{10\times1}{8.4\times10^{28}\times1.6\times10^{-19}\times10^{-3}}$$

$$= \frac{10}{1.344\times10^{7}} \approx 7.4\times10^{-7}\,\text{V}$$

金属中霍尔电压很小（$\mu$V 量级），而半导体中大得多（因为 $n$ 小得多）。

### 易错点

- 霍尔效应是载流子受**洛伦兹力**侧偏产生的，不是电场力。
- 霍尔电压与电流、磁场成正比，与载流子浓度 $n$ 成反比（$n$ 越小，霍尔效应越显著）。
- 半导体霍尔传感器灵敏度远高于金属（因为 $n$ 小 3-4 个数量级）。

### 与必修二衔接

本节将洛伦兹力（磁场对运动电荷）与平衡条件（电场力 = 洛伦兹力）结合，是磁力与静电力共同作用的典型例题，也是速度选择器的工作原理。
