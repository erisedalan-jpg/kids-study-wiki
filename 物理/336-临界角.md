## 🎓 临界角

### 定义

**临界角**（critical angle）$C$ 是光从光密介质射向光疏介质时，折射角恰好等于 90°（折射光线沿界面传播）时的入射角。

当入射角 $\theta_i = C$ 时，折射角 $\theta_r = 90°$；当 $\theta_i > C$ 时，发生全反射。

### 临界角公式

光从折射率为 $n_1$ 的介质射向折射率为 $n_2$ 的介质（$n_1 > n_2$）：

$$\sin C = \frac{n_2}{n_1}$$

特别地，当光从折射率为 $n$ 的介质射向空气（$n_2 = 1$）：

$$\boxed{\sin C = \frac{1}{n}}$$

### 临界角的推导

由斯涅尔定律：$n_1\sin\theta_1 = n_2\sin\theta_2$

令 $\theta_1 = C$，$\theta_2 = 90°$（$\sin90° = 1$）：

$$n_1\sin C = n_2\times1 \implies \sin C = \frac{n_2}{n_1}$$

### 各介质临界角

| 介质 | 折射率 $n$ | 临界角 $C$ |
|------|-----------|-----------|
| 水 | 1.33 | $\sin C = 1/1.33 \approx 0.752 \to C \approx 48.8°$ |
| 玻璃（$n=1.5$） | 1.5 | $\sin C = 1/1.5 \approx 0.667 \to C \approx 41.8°$ |
| 光密玻璃（$n=1.7$） | 1.7 | $\sin C = 1/1.7 \approx 0.588 \to C \approx 36.0°$ |
| 金刚石 | 2.42 | $\sin C = 1/2.42 \approx 0.413 \to C \approx 24.4°$ |

**规律**：折射率越大，临界角越小，越容易发生全反射。

### 与全反射的关系

$$\theta_i < C \implies \text{折射+反射（部分反射）}$$
$$\theta_i = C \implies \text{折射角}= 90°\text{（临界状态）}$$
$$\theta_i > C \implies \text{全反射（无折射光）}$$

### 典型例题

**例1**：某玻璃折射率 $n = \sqrt{3}$，求临界角。

$$\sin C = \frac{1}{\sqrt{3}} \approx 0.577$$

$$C = \arcsin(0.577) \approx 35.3°$$

**例2（逆用）**：已知光从某介质射向空气，临界角 $C = 30°$，求该介质折射率。

$$\sin30° = \frac{1}{n} \implies n = \frac{1}{\sin30°} = \frac{1}{0.5} = 2$$

**例3（实际判断）**：$n = 1.5$ 的玻璃中，光以 45° 入射到玻璃-空气界面，是否发生全反射？

临界角 $C \approx 41.8°$，$\theta_i = 45° > C$，**发生全反射**。

### 易错点

1. **临界角只适用于光密→光疏界面**：光从光疏射向光密时，无临界角，不会全反射。
2. **临界角是入射角，不是折射角**：$C$ 对应折射角 = 90°，是入射角的临界值。
3. **$\sin C = 1/n$ 仅适用于射向空气（$n_2 = 1$）的情形**：若射向其他介质需用通式 $\sin C = n_2/n_1$。
