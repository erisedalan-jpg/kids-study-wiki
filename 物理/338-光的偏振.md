## 🎓 光的偏振

### 定义

**光的偏振**（polarization）是光波中电场振动方向的特性。

- **自然光**：电场振动方向在各个方向均匀分布（无偏振）——太阳光、灯光
- **偏振光**：电场只在某一固定方向振动

光的偏振证明光是**横波**（纵波没有偏振现象）。

### 偏振片

**偏振片**（polarizer）是只允许某一振动方向的光通过的光学元件。

- 通过偏振片的光为偏振光
- 两块偏振片叠放：
  - 透振方向**平行**：光能透过，强度最大
  - 透振方向**垂直**（正交）：光完全不能透过，强度为零

### 马吕斯定律（选学）

$$I = I_0\cos^2\theta$$

- $I_0$：入射偏振光强度
- $\theta$：偏振片透振方向与入射光振动方向的夹角
- $I$：透射光强度

### 自然光的偏振化

**1. 通过偏振片**：最常用的产生偏振光方法。

**2. 反射偏振（布儒斯特角）**：
自然光以特定角度（布儒斯特角 $\theta_B$）入射到介质表面时，反射光为完全偏振光（振动方向平行于界面）。

$$\tan\theta_B = n \quad \text{（布儒斯特角）}$$

**3. 散射偏振**：天空中的蓝光是散射光，具有部分偏振性。

### 偏振的应用

| 应用 | 原理 |
|------|------|
| **液晶显示屏（LCD）** | 液晶分子控制偏振光的透过方向，调节各像素明暗 |
| **3D 电影** | 两个投影机分别发出两个偏振方向垂直的画面，配合偏振眼镜，左右眼分别看到不同画面 |
| **偏振太阳镜** | 阻挡水面、路面反射的偏振眩光（反射光有偏振性） |
| **摄影滤镜（CPL）** | 消除玻璃反射，使天空更蓝 |
| **光弹性分析** | 偏振光透过受力玻璃，应力分布显示为彩色条纹 |
| **蜜蜂导航** | 蜜蜂能感知天空散射光的偏振方向，用于导航 |

### 光的偏振与光的本性

| 证明 | 波动性 | 粒子性 |
|------|--------|--------|
| 干涉 | ✓ | |
| 衍射 | ✓ | |
| **偏振** | ✓（横波特有） | |
| 光电效应 | | ✓ |
| 康普顿效应 | | ✓ |

光具有波粒二象性，偏振证明其**横波性**。

### 典型例题

**例1**：自然光通过一块偏振片后，光强变为 $I_1$，再通过另一块偏振片（透振方向与第一块成 60°），求透射光强。

$$I = I_1\cos^2 60° = I_1\times\left(\frac{1}{2}\right)^2 = \frac{I_1}{4}$$

**例2**：两块偏振片正交（90°），透射光强 $I = I_0\cos^2 90° = 0$——完全消光。

**例3**：偏振太阳镜减少水面眩光——水面反射光为偏振光（振动方向平行于水面），偏振太阳镜的透振方向垂直于水面，阻挡反射偏振光，减少眩光。

### 易错点

1. **偏振是横波特有的现象**：纵波（如声波）质点振动沿传播方向，无偏振。
2. **自然光通过一块偏振片后强度减半**（各方向等能量，通过偏振片只剩一半）。
3. **偏振光不一定很弱**：偏振光只是振动方向单一，强度可以很强。
