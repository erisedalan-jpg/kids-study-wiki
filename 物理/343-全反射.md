## 🎓 全反射

### 定义

**全反射**（total internal reflection）是光从**光密介质**（折射率大）射向**光疏介质**（折射率小）时，当入射角大于或等于临界角，折射光完全消失，光线全部被反射回光密介质的现象。

**发生全反射的两个必要条件**：
1. 光从**光密介质**射向**光疏介质**（$n_1 > n_2$）
2. 入射角 $\theta_i$ **大于或等于**临界角 $C$（$\theta_i \geq C$）

### 临界角

$$\sin C = \frac{n_2}{n_1} = \frac{1}{n} \quad \text{（当}n_2 = 1\text{，即射向空气时）}$$

- $C$：临界角
- $n_1$：光密介质折射率
- $n_2$：光疏介质折射率（射向空气时 $n_2 = 1$）

折射率越大，临界角越小：

| 介质 | 折射率 $n$ | 临界角 $C$（射向空气） |
|------|-----------|-------------------|
| 水 | 1.33 | 48.8° |
| 玻璃（普通） | 1.5 | 41.8° |
| 金刚石 | 2.42 | 24.4° |

### 全反射的过程

当光从玻璃（$n=1.5$）射向空气（$n=1$）：

| 入射角 | 现象 |
|--------|------|
| $\theta < C \approx 41.8°$ | 折射光线存在（部分反射，部分折射） |
| $\theta = C$ | 折射角 = 90°（折射光线沿界面传播） |
| $\theta > C$ | **全反射**，无折射光，全部反射 |

### 全反射的应用

1. **光导纤维（光纤）**：光在玻璃纤维中不断发生全反射，可以沿弯曲路径传播，损耗极小。
   - 通信光纤：传输信息
   - 医用内窥镜：将图像导出体外

2. **全反射棱镜**：利用 45° 棱镜（$n=1.5$，$C≈41.8°$），光以 45° 入射→全反射→改变方向 90° 或 180°。

3. **钻石的闪光**：金刚石折射率 2.42，临界角仅 24.4°，光线极易全反射，多次反射后从特定角度射出，产生璀璨光泽。

### 典型例题

**例**：折射率 $n = \sqrt{2}$ 的玻璃，求光从玻璃射向空气时的临界角。

$$\sin C = \frac{1}{n} = \frac{1}{\sqrt{2}} = \frac{\sqrt{2}}{2}$$

$$C = 45°$$

当入射角 $\theta \geq 45°$ 时发生全反射。

### 易错点

1. **全反射只发生在光密→光疏界面**：光从空气（光疏）射向玻璃（光密）时，绝对不会发生全反射。
2. **入射角必须 ≥ 临界角**：等于临界角时折射光沿界面传播（折射角=90°），严格来说此时刚好发生全反射。
3. **全反射不是"完全不折射"的普通反射**：是特殊条件下折射光消失的现象。
