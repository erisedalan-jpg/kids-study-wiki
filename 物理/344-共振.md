## 🎓 共振

### 定义

**共振**（resonance）是受迫振动中，当驱动力的频率等于振动体的固有频率时，振幅达到最大值的现象。

$$f_{驱动} = f_0 \implies A_{最大}$$

- $f_0$：固有频率（由振动体本身的物理性质决定）
- 共振时能量从驱动力传入振动体的效率最高

### 共振的本质

每次驱动力推动振动体，振动体都在"最合适"的时机获得推力，动能持续积累，振幅不断增大，直到阻力耗散能量与驱动力输入能量平衡为止。

阻尼越小 → 共振峰越尖锐，共振振幅越大。

### 共振曲线

以驱动频率 $f$ 为横轴，稳态振幅 $A$ 为纵轴：

- $f \ll f_0$ 或 $f \gg f_0$：振幅小
- $f \to f_0$：振幅急剧增大
- $f = f_0$：振幅最大（共振峰）
- 阻尼越大：共振峰越宽越低；阻尼越小：峰越窄越高

### 共振的应用（有益共振）

| 应用 | 利用的固有频率 |
|------|--------------|
| 收音机调台 | 调节LC电路固有频率 = 电台频率，电流共振 |
| 核磁共振（MRI） | 氢原子核在特定频率磁场下共振 |
| 乐器发声 | 弦、管、鼓面在固有频率振动 |
| 微波炉加热 | 2.45 GHz 微波与水分子振动频率共振 |
| 音叉共鸣 | 相同频率音叉引起共振 |

### 共振的危害（有害共振）

| 危害 | 说明 |
|------|------|
| 塔科马海峡大桥（1940年） | 风力驱动频率接近桥的固有频率，共振导致桥梁扭曲坍塌 |
| 机械振动 | 转动机械的转速对应固有频率时共振损坏设备 |
| 地震 | 地震波频率接近建筑物固有频率时放大振幅，加剧破坏 |
| 军队过桥齐步走 | 踏步频率接近桥的固有频率引起共振（需改为便步） |

避免危害共振的方法：改变固有频率（改变结构）或增大阻尼。

### 典型例题

**例1**：弹簧振子 $m = 1\text{ kg}$，$k = 100\text{ N/m}$，固有频率？

$$f_0 = \frac{1}{2\pi}\sqrt{\frac{k}{m}} = \frac{1}{2\pi}\sqrt{100} = \frac{10}{2\pi} \approx 1.59\text{ Hz}$$

若驱动力频率为 1.59 Hz，则发生共振，振幅最大。

**例2**：音叉频率 440 Hz（A音），敲击后放置在钢琴旁，寻找与之共振的钢琴弦，找到共振时该琴弦固有频率也是 440 Hz。

### 易错点

1. **共振频率 = 固有频率**（近似，精确来说略有差别但高中不讨论）。
2. **共振振幅取决于阻尼**：无阻尼时理论上振幅趋于无穷大；有阻尼时达到有限最大值。
3. **共振不一定有害**：共振是物理现象，有害有益取决于应用场景。
