## 🎓 动量守恒定律

### 定律表述

**动量守恒定律**：如果一个系统不受外力，或所受合外力为零，则该系统的总动量保持不变。

$$\vec{p}_{总} = \sum m_i\vec{v}_i = \text{常量}$$

对两物体碰撞系统：

$$m_1\vec{v}_1 + m_2\vec{v}_2 = m_1\vec{v}_1' + m_2\vec{v}_2'$$

- 左边：碰撞前两物体动量之和
- 右边：碰撞后两物体动量之和

### 推导

由牛顿第三定律，系统内两物体间的相互作用力（内力）大小相等、方向相反：

$$\vec{F}_{12} = -\vec{F}_{21}$$

由动量定理，作用相同时间 $\Delta t$：

$$\vec{F}_{12}\Delta t + \vec{F}_{21}\Delta t = 0$$

$$\Delta\vec{p}_1 + \Delta\vec{p}_2 = 0$$

$$\Delta(\vec{p}_1 + \vec{p}_2) = 0 \implies \vec{p}_{总} = \text{常量}$$

### 守恒条件

| 条件 | 说明 |
|------|------|
| 合外力为零 | 理想情形：无摩擦、无外力 |
| 内力远大于外力 | 碰撞、爆炸等短时强力作用，外力可忽略 |
| 某方向合外力为零 | 该方向动量守恒（分量守恒） |

### 动量守恒的三种典型情形

1. **碰撞**：两物体相互作用短时间内，内力远大于重力/摩擦力
2. **爆炸**：火药爆炸产生的内力远大于外力，系统总动量守恒（爆炸前静止则总动量为零）
3. **反冲**：火箭喷气、枪击后坐——喷出物与剩余物动量之和守恒

### 典型例题

**例1（碰撞）**：质量 $m_1 = 2\text{ kg}$ 的小车以 $v_1 = 4\text{ m/s}$ 向右运动，与静止的 $m_2 = 2\text{ kg}$ 小车碰撞后粘在一起，求碰后速度。

$$m_1v_1 + m_2\cdot0 = (m_1+m_2)v'$$

$$v' = \frac{m_1v_1}{m_1+m_2} = \frac{2\times4}{4} = 2\text{ m/s（向右）}$$

**例2（爆炸/反冲）**：静止炮弹质量 $M = 5\text{ kg}$，爆炸后一半（2.5 kg）以 $v = 200\text{ m/s}$ 向右飞出，另一半速度？

$$0 = 2.5\times200 + 2.5\times v'$$

$$v' = -200\text{ m/s（向左，大小200 m/s）}$$

### 易错点

1. **动量守恒是矢量方程**：必须选定正方向，反向动量取负号。
2. **合外力为零才守恒**：有摩擦、重力的场合需判断是否满足条件（或只对某方向使用）。
3. **与能量守恒不同**：动量守恒不要求无能量损失；有动能损失时动量仍可守恒。
