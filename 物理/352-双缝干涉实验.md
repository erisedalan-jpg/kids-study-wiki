## 🎓 双缝干涉实验

### 历史背景

1801年，托马斯·杨（Thomas Young）首次成功进行**双缝干涉实验**，观察到光的干涉条纹，有力证明了光的**波动性**，反驳了牛顿的粒子说。

### 实验装置

```
光源 → 单缝S（增强相干性） → 双缝S₁S₂ → 光屏
```

- **单缝 S**：将光源变为线光源，增强相干性
- **双缝 S₁S₂**：两缝间距 $d$（约 0.1~0.5 mm），起分波前作用
- **光屏**：距双缝距离 $L$（约 0.5~2 m），观察干涉条纹

### 干涉条纹

实验结果：光屏上出现**等间距的明暗相间**竖直条纹。

**亮条纹**（加强）：$S_1$、$S_2$ 到该点的路程差 = 波长整数倍

$$\delta = r_2 - r_1 = n\lambda \quad (n = 0, \pm1, \pm2, \ldots)$$

**暗条纹**（减弱）：路程差 = 半波长奇数倍

$$\delta = (n + \frac{1}{2})\lambda$$

### 条纹间距公式

$$\boxed{\Delta y = \frac{L\lambda}{d}}$$

- $\Delta y$：相邻亮条纹（或暗条纹）间距（m）
- $L$：双缝到光屏的距离（m）
- $\lambda$：光的波长（m）
- $d$：双缝间距（m）

### 测量光的波长

由已知量 $L$、$d$ 和测量量 $\Delta y$，反求 $\lambda$：

$$\lambda = \frac{d\cdot\Delta y}{L}$$

实验中测 $n$ 条亮纹的总宽度 $\Delta Y$，则 $\Delta y = \Delta Y / (n-1)$，减小误差。

### 条纹特点分析

| 改变量 | 对条纹的影响 |
|--------|------------|
| 增大波长 $\lambda$ | 条纹间距增大（红光比蓝光条纹宽） |
| 增大缝间距 $d$ | 条纹间距减小（条纹变密） |
| 增大屏距 $L$ | 条纹间距增大（条纹变宽） |
| 换白光 | 中央白色，两侧出现彩色条纹（各色间距不同） |

### 典型例题

**例**：双缝实验中，$d = 0.3\text{ mm}$，$L = 1.2\text{ m}$，用绿光（$\lambda = 550\text{ nm}$）照射，求条纹间距。

$$\Delta y = \frac{L\lambda}{d} = \frac{1.2\times550\times10^{-9}}{0.3\times10^{-3}} = \frac{6.6\times10^{-7}}{3\times10^{-4}} = 2.2\times10^{-3}\text{ m} = 2.2\text{ mm}$$

### 易错点

1. **双缝干涉是分波前法**：两缝来自同一光源的不同部分，是相干光。两个独立光源不相干，不能产生稳定干涉。
2. **路程差计算用几何方法**：对于中央条纹，路程差为零（等程差）；越靠边，路程差越大。
3. **条纹是等间距的**：理论上等间距（在近似条件 $d \ll L$ 下成立）。
