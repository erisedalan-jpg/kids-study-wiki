## 🎓 反冲运动

### 定义

**反冲运动**（recoil）是系统中一部分物体向某方向喷出（或射出），剩余部分因动量守恒而向相反方向运动的现象。

**本质**：动量守恒定律的应用。系统初始静止（或有初动量），内力使一部分弹射出去，另一部分获得反向动量。

### 分析方法

若系统初始总动量为 $\vec{p}_0$（静止时 $p_0 = 0$）：

$$\vec{p}_0 = m_1\vec{v}_1 + m_2\vec{v}_2 = \text{常量}$$

若初始静止（$p_0 = 0$）：

$$m_1\vec{v}_1 + m_2\vec{v}_2 = 0 \implies \vec{v}_2 = -\frac{m_1}{m_2}\vec{v}_1$$

喷出质量越小，喷出速度越大，剩余部分的反冲速度也越大。

### 典型例题

**例1（火箭）**：火箭总质量 $M = 1000\text{ kg}$（含燃料），点火后喷出质量 $m = 100\text{ kg}$ 的气体，相对地面速度 $v_{气} = 500\text{ m/s}$（向后），求火箭的速度（初始静止，忽略重力）。

$$0 = (M - m)v_{火} - mv_{气}$$

$$v_{火} = \frac{mv_{气}}{M-m} = \frac{100\times500}{900} \approx 55.6\text{ m/s（向前）}$$

**例2（枪的后坐力）**：步枪质量 $m_{枪} = 3\text{ kg}$，子弹质量 $m_{弹} = 0.01\text{ kg}$，子弹出膛速度 $v_{弹} = 900\text{ m/s}$，求枪的后坐速度。

$$0 = m_{弹}v_{弹} + m_{枪}v_{枪}$$

$$v_{枪} = -\frac{m_{弹}v_{弹}}{m_{枪}} = -\frac{0.01\times900}{3} = -3\text{ m/s（向后）}$$

**例3（喷气推进）**：乌贼喷水、火箭发动机均为反冲推进原理。

### 反冲运动的应用

| 应用 | 原理 |
|------|------|
| 火箭推进 | 向后喷出高速气体，火箭向前加速 |
| 喷气式飞机 | 向后喷射燃气，飞机向前飞 |
| 消防水枪后坐 | 水向前喷出，枪向后退 |
| 乌贼/章鱼游泳 | 向后喷水实现快速移动 |
| 气球放气 | 气体向一侧喷出，气球向反方向飞 |

### 与火箭速度公式（选学）

若火箭不断喷气，用齐奥尔科夫斯基公式（大学物理）：

$$\Delta v = v_{喷}\ln\frac{M_0}{M}$$

（$M_0$ 为初始质量，$M$ 为末质量，$v_{喷}$ 为喷气相对火箭的速度）

提高火箭末速度：增大喷气速度、增大质量比（多级火箭）。

### 易错点

1. **反冲运动动量守恒，合动量等于初始动量**：初始静止则合动量为零，不是各部分动量为零。
2. **喷出质量的速度是相对地面的速度**：若题目给相对火箭的速度，需转换。
3. **反冲力是内力**：系统内力不改变总动量，只是使各部分动量重新分配。
