## 🎓 波速

### 定义

**波速**（wave speed）是波在介质中传播的速度，即单位时间内波形向前移动的距离。

$$v = \frac{\lambda}{T} = \lambda f$$

- $v$：波速（m/s）
- $\lambda$：波长（m）
- $T$：周期（s）
- $f$：频率（Hz）

### 波速的决定因素

**波速由介质决定**，与振源的频率无关。

| 影响波速的因素 | 说明 |
|--------------|------|
| 介质的密度 | 密度越大，一般波速越小（固体中纵波例外） |
| 介质的弹性（刚性） | 弹性越强，波速越大 |
| 温度 | 温度升高，气体中声速增大 |
| 频率 | **不影响**波速（机械波） |

**常见波速**：

| 介质 | 声速（约） |
|------|-----------|
| 空气（0°C） | 331 m/s |
| 空气（20°C） | 343 m/s |
| 水（25°C） | 1497 m/s |
| 钢 | 5100 m/s |
| 真空 | 光速 $c = 3\times10^8$ m/s（只有电磁波） |

**固体 > 液体 > 气体**（通常情况下声速顺序）

### 波速与质点振动速度的区别

| 量 | 定义 | 典型量级 |
|----|------|---------|
| **波速** $v$ | 波形传播速度（由介质决定） | 声波约 340 m/s |
| **质点振动速度** $v_{质点}$ | 质点在平衡位置附近的运动速度 | 远小于波速 |

两者完全不同！声音在空气中以 340 m/s 传播，但空气分子只是在原地微小振动。

### 换介质时的变化规律

$$\text{频率 } f \text{（不变）} \xrightarrow{\text{换介质}} v\text{（改变）} \implies \lambda = \frac{v}{f}\text{（改变）}$$

| 量 | 换介质后 |
|----|---------|
| 频率 $f$ | 不变（由振源决定） |
| 波速 $v$ | 改变（由新介质决定） |
| 波长 $\lambda$ | 改变（随 $v$ 成比例变化） |

### 典型例题

**例1**：已知波长 $\lambda = 2\text{ m}$，频率 $f = 50\text{ Hz}$，求波速。

$$v = \lambda f = 2\times50 = 100\text{ m/s}$$

**例2**：声波在空气中 $v_1 = 340\text{ m/s}$，$f = 1000\text{ Hz}$，$\lambda_1 = 0.34\text{ m}$。该声波进入水中 $v_2 = 1500\text{ m/s}$，求水中波长。

$$\lambda_2 = \frac{v_2}{f} = \frac{1500}{1000} = 1.5\text{ m}$$

**例3（由波形图计算）**：波形图显示波长 $\lambda = 4\text{ m}$，波在 $t = 0.1\text{ s}$ 内向右移动了 2 m，求波速。

$$v = \frac{\Delta x}{\Delta t} = \frac{2}{0.1} = 20\text{ m/s}$$

频率 $f = v/\lambda = 20/4 = 5\text{ Hz}$

### 易错点

1. **波速由介质决定，不由频率决定**：同一介质中不同频率的波速度相同（机械波理想情形）。
2. **质点振动速度 ≠ 波速**：两者概念完全不同。
3. **波速不是光速**：机械波波速取决于介质，远小于光速。
