## 🎓 相位

### 定义

**相位**（phase）$\varphi$ 是描述简谐运动在某时刻振动状态的物理量，出现在简谐运动方程中：

$$x = A\cos(\omega t + \varphi_0)$$

- $\omega t + \varphi_0$：$t$ 时刻的相位（rad）
- $\varphi_0$：初相位（$t=0$ 时刻的相位）
- $\omega$：角频率（rad/s）

相位完全决定了振动物体在某时刻的位移和速度状态。

### 相位的物理意义

相位是时间的线性函数，每经过一个周期 $T$，相位增加 $2\pi$：

$$\Delta\varphi = \omega\Delta t = \frac{2\pi}{T}\cdot\Delta t$$

| 相位差 | 对应时间差 | 振动关系 |
|--------|-----------|----------|
| $0$ 或 $2n\pi$ | 0 或 $nT$ | **同相**（振动完全相同） |
| $\pi$ 或 $(2n+1)\pi$ | $T/2$ | **反相**（位移大小相等，方向相反） |
| $\pi/2$ | $T/4$ | **相差四分之一周期** |

### 同相与反相

**同相**：两振动体位移始终相同（方向、大小均同）

$$\varphi_1 - \varphi_2 = 2n\pi \quad (n = 0, \pm1, \pm2, \ldots)$$

**反相**：两振动体位移始终相反（大小相同，方向相反）

$$\varphi_1 - \varphi_2 = (2n+1)\pi$$

### 相位在波的干涉中的作用

两列相干波在空间某点叠加：

- **相位差为 $2n\pi$（同相叠加）** → 振幅加强（两波峰相遇或两波谷相遇）
- **相位差为 $(2n+1)\pi$（反相叠加）** → 振幅减弱（波峰与波谷相遇）

路程差与相位差的关系：

$$\Delta\varphi = \frac{2\pi}{\lambda}\cdot\Delta r$$

（$\Delta r$ 为路程差，$\lambda$ 为波长）

- 路程差为波长整数倍 → 同相 → 加强
- 路程差为半波长奇数倍 → 反相 → 减弱

### 典型例题

**例1**：弹簧振子 $x = 0.1\cos(10t + \frac{\pi}{3})\text{ m}$，求 $t = 0$ 时刻的位移。

$$x_0 = 0.1\cos\left(\frac{\pi}{3}\right) = 0.1\times0.5 = 0.05\text{ m}$$

**例2**：两列频率相同的波，在某点一列路程 $r_1 = 3\lambda$，另一列 $r_2 = 3.5\lambda$，两列波初相位相同，该点是加强还是减弱？

$$\Delta r = r_2 - r_1 = 0.5\lambda = \frac{\lambda}{2}$$

路程差为半波长（奇数倍）→ **反相叠加 → 减弱**

### 易错点

1. **相位不等于位移**：相位是自变量（角度），位移是相位的函数值。
2. **初相位由初始条件决定**：从端点出发初相位 = 0（余弦），从平衡位置出发初相位 = $\pi/2$（余弦）。
3. **相位差决定干涉结果**：路程差相同不一定同相（还要考虑初相位差）。
