## 🎓 LC 振荡电路（高中物理）

### 定义

**LC 振荡电路**（LC circuit）是由电感 $L$ 和电容 $C$ 组成的闭合回路，是产生高频交变电流（电磁振荡）的基本元件。

### 电路结构

```
    ┌───L（电感线圈）───┐
    │                  │
    └───C（电容器）────┘
```

实际振荡器还需配合放大器补充能量，维持等幅振荡。

### 核心参数

**固有频率（谐振频率）**：

$$f_0 = \frac{1}{2\pi\sqrt{LC}}$$

**振荡周期**：

$$T = 2\pi\sqrt{LC}$$

其中 $L$ 单位为亨利（H），$C$ 单位为法拉（F），$T$ 单位为秒（s），$f_0$ 单位为赫兹（Hz）。

### 能量分析

| 时刻 | 电容 | 电感（电流）| 电场能 | 磁场能 |
|------|------|-----------|--------|--------|
| $t=0$ | 充满电 $Q_m$ | $I=0$ | 最大 | 0 |
| $t=T/4$ | $Q=0$ | $I=I_m$ | 0 | 最大 |
| $t=T/2$ | 反向充满 $-Q_m$ | $I=0$ | 最大 | 0 |
| $t=3T/4$ | $Q=0$ | $I=-I_m$ | 0 | 最大 |
| $t=T$ | 恢复初始 | $I=0$ | 最大 | 0 |

能量关系：

$$\frac{Q_m^2}{2C} = \frac{1}{2}LI_m^2 = W_{\text{总}}$$

### 谐振特性

当外加交流信号频率 = LC 电路固有频率时，发生**谐振**（共振），电路对该频率呈现最小阻抗（串联谐振）或最大阻抗（并联谐振）。

**调谐原理**（收音机）：改变电容 $C$（可变电容器），调节固有频率 $f_0 = 1/(2\pi\sqrt{LC})$ 与目标电台频率匹配，实现选台。

### 开放式 LC 回路 → 电磁波辐射

封闭 LC 回路几乎不辐射电磁波（能量局限在回路中）。将 LC 回路"展开"（电容两极板展开，电感变为直线天线），成为**开放振荡回路**，可高效向外辐射电磁波。

赫兹（Hertz）正是通过这种开放回路首次人工产生并探测了电磁波（1888年）。

### 典型例题

**例1（求频率）**：$L = 2.5\times10^{-3}$ H，$C = 10^{-8}$ F，求固有频率。

$$f_0 = \frac{1}{2\pi\sqrt{LC}} = \frac{1}{2\pi\sqrt{2.5\times10^{-3}\times10^{-8}}} = \frac{1}{2\pi\times5\times10^{-6}} = \frac{10^6}{10\pi} \approx 31.8\text{ kHz}$$

**例2（调谐问题）**：某收音机 LC 回路中 $L = 4\times10^{-4}$ H，要接收 700 kHz 的电台，电容应调为多少？

$$f = 700\text{ kHz} = 7\times10^5\text{ Hz}$$
$$C = \frac{1}{(2\pi f)^2 L} = \frac{1}{(2\pi\times7\times10^5)^2\times4\times10^{-4}}$$
$$= \frac{1}{4\pi^2\times4.9\times10^{11}\times4\times10^{-4}} \approx 1.3\times10^{-10}\text{ F} = 130\text{ pF}$$

### 易错点

1. $f_0 = 1/(2\pi\sqrt{LC})$ 中，$L$ 和 $C$ 的单位必须是 H 和 F，否则数值错误。
2. **改变 $L$ 或 $C$ 都可以改变频率**：增大 $L$ 或 $C$ → 频率减小，波长增大。
3. 振荡幅度（电压峰值、电流峰值）由**初始储能**决定，与 $f_0$ 无关。
