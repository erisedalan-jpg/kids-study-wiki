## 🎓 带电粒子在复合场中运动（高中物理）

### 概述

**复合场**指电场和磁场同时存在的区域。带电粒子同时受电场力（$F_E = qE$）和洛伦兹力（$F_B = qvB$）的作用，运动状态由两者的合力决定。

### 常见模型

#### 1. 速度选择器（匀速直线运动）

电场力与洛伦兹力方向相反、大小相等时粒子做匀速直线运动：

$$qE = qvB \implies v = \frac{E}{B}$$

**应用**：筛选出特定速度的带电粒子，不依赖粒子的质量和电荷量。

#### 2. 电场中的抛体 + 磁场偏转（质谱仪）

粒子先经速度选择器筛速，再进入纯磁场区做圆周运动：

$$r = \frac{mv}{qB} \implies \frac{m}{q} = \frac{rB}{v} = \frac{rB^2}{E}$$

通过测量半径 $r$，即可确定粒子的比荷 $m/q$，进而鉴别同位素。

#### 3. 重力 + 电场 + 磁场（三场共存）

若三个力平衡，粒子静止或匀速：
$$qE = mg + qvB\sin\theta \quad \text{（具体方向依题意）}$$

### 受力分析步骤

1. 明确各场的方向（电场线方向、磁感线方向）
2. 判断粒子电荷正负，确定受力方向
3. 建立坐标系，列牛顿第二定律（或平衡方程）
4. 求解运动轨迹参数

### 典型例题

**例（质谱仪）**：粒子经 $U = 1000$ V 加速后，经速度选择器（$E = 5\times10^4$ V/m，$B_1 = 0.1$ T）选速，再进入 $B_2 = 0.2$ T 的纯磁场中做半圆运动，直径 $d = 0.16$ m，求粒子比荷。

$$v = \frac{E}{B_1} = \frac{5\times10^4}{0.1} = 5\times10^5\text{ m/s}$$
$$r = \frac{d}{2} = 0.08\text{ m}$$
$$\frac{q}{m} = \frac{v}{rB_2} = \frac{5\times10^5}{0.08 \times 0.2} = 3.125\times10^7\text{ C/kg}$$

### 易错点

1. 速度选择器中，粒子做匀速直线运动，**合力为零**，而非单力为零。
2. 进入磁场后，若重力可忽略，洛伦兹力提供向心力，粒子做**匀速圆周运动**。
3. 注意粒子偏转方向与电荷正负的对应关系，用左手定则逐步判断。
