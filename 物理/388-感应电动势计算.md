## 🎓 感应电动势计算（高中物理）

### 两大计算公式

#### 公式一：法拉第定律（通量变化法）

$$\varepsilon = N\frac{\Delta\Phi}{\Delta t} = N\frac{\Delta(BS\cos\theta)}{\Delta t}$$

**适用场景**：磁场均匀变化、线圈面积变化、或线圈匀速转动（计算某段时间的平均值）

#### 公式二：切割公式（动生电动势法）

$$\varepsilon = BLv\sin\alpha$$

**适用场景**：导体棒在匀强磁场中做匀速直线运动，切割磁感线

特别地，当 $v \perp B$ 且导体有效长度 $L \perp v$：$\varepsilon = BLv$

### 线圈匀速转动时的瞬时电动势

矩形线圈（匝数 $N$，面积 $S$）在匀强磁场（$B$）中以角速度 $\omega$ 匀速旋转：

$$\varepsilon = NBA\omega\sin(\omega t) = \varepsilon_m\sin(\omega t)$$

其中最大电动势：$\varepsilon_m = NBA\omega$

- 线圈平面**平行**于磁场时：$\varepsilon = \varepsilon_m$（最大，此时磁通量变化最快）
- 线圈平面**垂直**于磁场时：$\varepsilon = 0$（磁通量最大，变化率为零）

### 计算步骤

1. **判断计算类型**：磁场变化/面积变化 → 用法拉第公式；导体切割 → 用切割公式
2. **确定各物理量**：$B$、$S$、$\theta$、$\Delta t$ 或 $B$、$L$、$v$、$\alpha$
3. **代入计算**：注意单位统一（T、m、s、A、V）
4. **确定方向**：用楞次定律或右手定则（切割情形）

### 综合计算例题

**例1（法拉第公式）**：$N = 50$ 匝线圈，面积 $S = 100\text{ cm}^2 = 0.01\text{ m}^2$，磁场从 $B_1 = 0$ 均匀增大到 $B_2 = 2$ T（历时 $\Delta t = 0.5$ s，$B \perp$ 线圈平面），求感应电动势。

$$\varepsilon = N\frac{\Delta\Phi}{\Delta t} = N\frac{(B_2-B_1)S}{\Delta t} = 50\times\frac{2\times0.01}{0.5} = 2\text{ V}$$

**例2（切割公式）**：$L = 0.3$ m 导体棒，$B = 1.2$ T，$v = 4$ m/s（$v \perp B$，$L \perp v$），外接 $R = 3$ Ω，内阻 $r = 1$ Ω，求各量。

$$\varepsilon = BLv = 1.2\times0.3\times4 = 1.44\text{ V}$$
$$I = \frac{\varepsilon}{R+r} = \frac{1.44}{3+1} = 0.36\text{ A}$$
$$U_R = IR = 0.36\times3 = 1.08\text{ V}$$
$$P_{\text{总}} = \varepsilon I = 1.44\times0.36 = 0.518\text{ W}$$
$$P_R = I^2R = 0.36^2\times3 = 0.389\text{ W}$$

**例3（线圈转动）**：$N = 100$，$S = 0.05\text{ m}^2$，$B = 0.4$ T，$\omega = 100\pi$ rad/s，求最大电动势及 $t = 0.01$ s 时的瞬时电动势（$t=0$ 时线圈平面平行磁场）。

$$\varepsilon_m = NBA\omega = 100\times0.4\times0.05\times100\pi = 200\pi \approx 628\text{ V}$$

$t = 0.01$ s 时，$\omega t = 100\pi\times0.01 = \pi$ rad：

$$\varepsilon = \varepsilon_m\sin\pi = 0\text{ V}$$

（此时线圈平面垂直于磁场，磁通量最大，变化率为零）

### 易错点

1. 多匝线圈不要漏掉 $N$（$\varepsilon = N\Delta\Phi/\Delta t$，而非 $\Delta\Phi/\Delta t$）。
2. 切割公式中 $L$ 是**有效长度**（投影），弯曲导体要注意。
3. 线圈转动时，电动势最大时刻（线圈平面平行磁场）≠ 磁通量最大时刻（线圈平面垂直磁场）。
