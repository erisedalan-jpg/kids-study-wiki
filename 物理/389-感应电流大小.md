## 🎓 感应电流大小（高中物理）

### 基本关系

感应电流大小由**欧姆定律**决定：

$$I = \frac{\varepsilon}{R_{\text{总}}} = \frac{\varepsilon}{R + r}$$

其中：
- $\varepsilon$：感应电动势（V）
- $R$：外电路电阻（Ω）
- $r$：电源（线圈）内阻（Ω）

感应电动势由法拉第定律或切割公式确定：

$$\varepsilon = N\frac{\Delta\Phi}{\Delta t} \quad \text{或} \quad \varepsilon = BLv$$

### 感应电荷量

通过回路的感应电荷量（与电阻无关，只与磁通量变化有关）：

$$q = It = \frac{\varepsilon}{R}t = \frac{N\Delta\Phi}{R}$$

**重要结论**：感应电荷量 $q = N\Delta\Phi / R$，与磁通量变化量成正比，与时间无关（不管磁场变化快慢，只要磁通量变化量相同，电荷量相同）。

### 综合计算流程

```
已知：B、L、v（切割）或 ΔΦ、Δt（通量变化）
         ↓
计算感应电动势 ε
         ↓
已知电路电阻 R、r
         ↓
计算感应电流 I = ε/(R+r)
         ↓
计算其他：U外 = IR，P = εI，Q = I²Rt，q = It
```

### 典型例题

**例1（切割 + 欧姆定律）**：导体棒 $B = 0.5$ T，$L = 0.4$ m，$v = 5$ m/s，外接 $R = 4$ Ω，内阻 $r = 1$ Ω。

$$\varepsilon = BLv = 0.5\times0.4\times5 = 1\text{ V}$$
$$I = \frac{\varepsilon}{R+r} = \frac{1}{5} = 0.2\text{ A}$$
$$U_{\text{外}} = IR = 0.2\times4 = 0.8\text{ V}$$

**例2（感应电荷量）**：$N = 100$ 匝线圈，$R = 10$ Ω（含内阻），磁通量在某时间内从 $0.05$ Wb 变为 $0.02$ Wb，求通过线圈的电荷量。

$$q = \frac{N\Delta\Phi}{R} = \frac{100\times(0.05-0.02)}{10} = \frac{100\times0.03}{10} = 0.3\text{ C}$$

（与磁场变化时间无关）

**例3（安培力与匀速条件）**：导体棒以匀速 $v$ 在导轨上运动，$B = 0.8$ T，$L = 0.5$ m，$R_{\text{总}} = 2$ Ω，求维持匀速所需外力。

$$\varepsilon = BLv = 0.8\times0.5\times v = 0.4v$$
$$I = \frac{0.4v}{2} = 0.2v$$
$$F_{\text{安}} = BIL = 0.8\times0.2v\times0.5 = 0.08v$$

匀速时：$F_{\text{外}} = F_{\text{安}} = 0.08v$（具体数值需已知 $v$）

### 感应电流与磁通量变化率的关系

$$I = \frac{N}{R}\cdot\frac{\Delta\Phi}{\Delta t}$$

- 磁通量变化越**快**（$\Delta t$ 越小），感应电流越**大**
- 线圈匝数越多，感应电流越大
- 回路总电阻越小，感应电流越大

### 易错点

1. 感应**电荷量** $q = N\Delta\Phi/R$ 与时间无关；感应**电流** $I = \varepsilon/R$ 与时间有关（通过变化率 $\Delta\Phi/\Delta t$）。
2. 内阻 $r$ 不能漏掉（$I = \varepsilon/(R+r)$）；外电压 $U = IR \neq \varepsilon$。
3. 导体棒做加速运动时，$v$ 是变量，$\varepsilon$、$I$、$F_{\text{安}}$ 也随时间变化，不能用匀速公式直接套用。
