## 🎓 正弦交流电（高中物理）

### 定义

**正弦交流电**是电流（或电压）随时间按**正弦函数**规律变化的交变电流，是最常见、最基本的交流电形式。

中国民用和工业用交流电均为正弦交流电（50 Hz）。

### 标准表达式

**瞬时电动势**：$\varepsilon = \varepsilon_m\sin(\omega t + \varphi_0)$

**瞬时电压**：$u = U_m\sin(\omega t + \varphi_0)$

**瞬时电流**：$i = I_m\sin(\omega t + \varphi_0)$

其中：
- $\varepsilon_m$、$U_m$、$I_m$：峰值（最大值）
- $\omega$：角频率（rad/s），$\omega = 2\pi f$
- $\varphi_0$：初相位（rad），取决于 $t=0$ 时刻的选取

### 参数的物理含义

| 参数 | 含义 | 与实际的联系 |
|------|------|------------|
| $\varepsilon_m = NBA\omega$ | 峰值由发电机结构决定 | 电站发电机参数 |
| $\omega = 2\pi f$ | 角频率 = 线圈转动角速度 | 50 Hz → $\omega = 100\pi$ rad/s |
| $T = 2\pi/\omega$ | 周期 = 发电机转一圈时间 | 50 Hz → $T = 0.02$ s |

### 波形图分析

正弦交流电 $u = U_m\sin\omega t$ 的图像特征：
- 过原点（初相 $\varphi_0 = 0$）
- 振幅为 $U_m$（最高点到基线的距离）
- 一个周期内正负各一半（平均值为零）
- 在 $t = T/4$ 时达到正峰值，$t = 3T/4$ 时达到负峰值

### 从波形图读取参数

1. 振幅 → 峰值 $U_m$（或 $I_m$）
2. 相邻两个同向过零点的时间间隔 → 周期 $T$
3. $f = 1/T$，$\omega = 2\pi/T$
4. 初相 $\varphi_0$：$t=0$ 时对应的相位角

### 正弦与余弦的关系

$$\sin(\omega t + 90°) = \cos\omega t$$

在电路中，电感上的电压比电流**超前** 90°：$u_L = U_{Lm}\cos\omega t$（若 $i = I_m\sin\omega t$）

### 典型例题

**例1**：已知 $u = 220\sqrt{2}\sin(100\pi t)$ V，求各参数。

$$U_m = 220\sqrt{2} \approx 311\text{ V}, \quad U = 220\text{ V}$$
$$\omega = 100\pi\text{ rad/s}, \quad f = 50\text{ Hz}, \quad T = 0.02\text{ s}$$

**例2（从图读参数）**：某交流电图像显示振幅为 10 V，周期为 0.04 s，初相为 30°，写出表达式。

$$\omega = \frac{2\pi}{0.04} = 50\pi\text{ rad/s}$$
$$u = 10\sin\left(50\pi t + \frac{\pi}{6}\right)\text{ V}$$

### 易错点

1. 正弦交流电**整周期内平均值为零**（正负抵消），但**有效值不为零**（$U = U_m/\sqrt{2}$）。
2. 初相的单位是**弧度**（rad），不是度（°）；代入计算时注意换算。
3. $\omega t$ 是角度，当 $\omega t = \pi/2$ 时 $\sin = 1$（峰值），对应 $t = T/4$。
