## 🎓 理想变压器（高中物理）

### 定义

**理想变压器**是一种理想化模型：假设线圈电阻为零（无铜损）、铁芯无涡流和磁滞损耗（无铁损）、磁通量完全被铁芯约束（无漏磁），效率 $\eta = 100\%$。

高中物理所有变压器计算题，默认使用**理想变压器**模型。

### 理想变压器的三大关系

#### 1. 电压关系（匝数比 = 电压比）

$$\frac{U_1}{U_2} = \frac{N_1}{N_2}$$

推广到多个次级绕组：

$$U_1 : U_2 : U_3 : \cdots = N_1 : N_2 : N_3 : \cdots$$

#### 2. 电流关系（匝数比倒数 = 电流比）

$$\frac{I_1}{I_2} = \frac{N_2}{N_1}$$

即 $N_1 I_1 = N_2 I_2$（安匝积守恒）

多个次级绕组时：$N_1 I_1 = N_2 I_2 + N_3 I_3 + \cdots$（总安匝守恒）

#### 3. 功率关系（输入功率 = 输出功率）

$$P_1 = P_2, \quad U_1 I_1 = U_2 I_2$$

### 推导思路

**电压关系推导**：初次级共享同一铁芯的交变磁通量 $\Phi$，根据法拉第定律：

$$\varepsilon_1 = N_1\frac{\Delta\Phi}{\Delta t}, \quad \varepsilon_2 = N_2\frac{\Delta\Phi}{\Delta t}$$

$$\frac{\varepsilon_1}{\varepsilon_2} = \frac{N_1}{N_2} \implies \frac{U_1}{U_2} = \frac{N_1}{N_2}$$

（理想变压器线圈无电阻，$U = \varepsilon$）

**电流关系推导**：功率守恒 $U_1I_1 = U_2I_2$，结合电压关系即得：

$$\frac{I_1}{I_2} = \frac{U_2}{U_1} = \frac{N_2}{N_1}$$

### 阻抗变换

理想变压器还具有阻抗变换功能：次级接负载 $R_L$，从初级看到的等效阻抗：

$$R_{\text{等效}} = \left(\frac{N_1}{N_2}\right)^2 R_L$$

用于信号传输中的阻抗匹配（如音频功放接扬声器）。

### 典型例题

**例1（基础）**：理想变压器 $N_1 = 500$，$N_2 = 100$，$U_1 = 220$ V，$R_L = 4$ Ω，求 $U_2$、$I_2$、$I_1$。

$$U_2 = 220\times\frac{100}{500} = 44\text{ V}$$
$$I_2 = \frac{U_2}{R_L} = \frac{44}{4} = 11\text{ A}$$
$$I_1 = I_2\times\frac{N_2}{N_1} = 11\times\frac{100}{500} = 2.2\text{ A}$$

验证：$P_1 = U_1I_1 = 220\times2.2 = 484$ W $= U_2I_2 = 44\times11 = 484$ W ✓

**例2（多次级绕组）**：$N_1 = 1000$，$N_2 = 500$，$N_3 = 100$，$U_1 = 200$ V，$I_2 = 2$ A，$I_3 = 1$ A，求 $I_1$。

$$U_2 = 100\text{ V}, \quad U_3 = 20\text{ V}$$
$$P_2 = 100\times2 = 200\text{ W}, \quad P_3 = 20\times1 = 20\text{ W}$$
$$I_1 = \frac{P_2 + P_3}{U_1} = \frac{220}{200} = 1.1\text{ A}$$

### 易错点

1. **多次级绕组时**：电压各自按匝数比，但初级电流由总功率决定（$I_1 = (P_2+P_3+\cdots)/U_1$）。
2. 理想变压器**不存储能量**，任何时刻输入功率 = 输出功率（实时守恒）。
3. 变压器改变的是**交流电**的电压，直流电通过变压器**没有效果**（无磁通量变化，无互感）。
