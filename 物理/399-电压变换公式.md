## 🎓 电压变换公式（高中物理）

### 核心公式

理想变压器的**电压变换公式**：

$$\frac{U_1}{U_2} = \frac{N_1}{N_2}$$

变形形式：

$$U_2 = U_1 \cdot \frac{N_2}{N_1}$$

多个次级绕组（$N_2$、$N_3$……）：

$$U_2 = U_1\cdot\frac{N_2}{N_1}, \quad U_3 = U_1\cdot\frac{N_3}{N_1}$$

各次级绕组电压之比 = 各自匝数之比：

$$U_2 : U_3 = N_2 : N_3$$

### 公式推导

基于法拉第电磁感应定律，初次级线圈共享铁芯中同一交变磁通量 $\Phi(t)$：

$$\varepsilon_1 = N_1\frac{\Delta\Phi}{\Delta t}, \quad \varepsilon_2 = N_2\frac{\Delta\Phi}{\Delta t}$$

两式相除（$\Delta\Phi/\Delta t$ 相同）：

$$\frac{\varepsilon_1}{\varepsilon_2} = \frac{N_1}{N_2}$$

理想变压器线圈无电阻 → $U = \varepsilon$ → $\frac{U_1}{U_2} = \frac{N_1}{N_2}$

### 匝数比的含义

| 匝数比 $N_1/N_2$ | 变压类型 | 电压变化 |
|-----------------|---------|---------|
| $> 1$ | 降压变压器 | $U_2 < U_1$ |
| $= 1$ | 隔离变压器 | $U_2 = U_1$ |
| $< 1$ | 升压变压器 | $U_2 > U_1$ |

### 与电流变换公式的联系

由功率守恒 $U_1I_1 = U_2I_2$，结合电压公式：

$$\frac{I_1}{I_2} = \frac{U_2}{U_1} = \frac{N_2}{N_1}$$

"升压必降流，降压必升流"

### 注意：交流有效值

电压变换公式中，$U_1$、$U_2$ 均为**有效值**（日常使用中的标称电压）。

峰值之比也等于匝数比（因为有效值与峰值成固定比例 $1/\sqrt{2}$）：

$$\frac{U_{1m}}{U_{2m}} = \frac{U_1}{U_2} = \frac{N_1}{N_2}$$

### 典型例题

**例1（求次级电压）**：$N_1 = 2000$，$N_2 = 100$，$U_1 = 10000$ V，求 $U_2$。

$$U_2 = 10000\times\frac{100}{2000} = 500\text{ V}$$

**例2（求匝数比）**：要将 220 V 交流电降为 12 V，求匝数比。

$$\frac{N_1}{N_2} = \frac{U_1}{U_2} = \frac{220}{12} \approx 18.3$$

即 $N_1 : N_2 \approx 55 : 3$（取整数比，待家长核对实际绕线匝数）

**例3（多绕组）**：$N_1 = 1000$，$N_2 = 500$，$N_3 = 200$，$U_1 = 100$ V：

$$U_2 = 50\text{ V}, \quad U_3 = 20\text{ V}$$

### 易错点

1. 公式适用于**理想变压器**；实际变压器因漏磁等原因略有偏差。
2. 公式只适用于**交流电**；直流电不适用。
3. 多绕组时，次级各绕组**独立与初级成比例**，但各次级之间不直接等于匝数比（需各自与初级比较）。
