## 🎓 电流变换公式（高中物理）

### 核心公式

理想变压器的**电流变换公式**：

$$\frac{I_1}{I_2} = \frac{N_2}{N_1}$$

变形形式：

$$I_1 N_1 = I_2 N_2 \quad \text{（安匝积守恒）}$$

注意：电流比是匝数比的**倒数**（与电压比相反）。

### 推导

由理想变压器功率守恒 $P_1 = P_2$：

$$U_1 I_1 = U_2 I_2$$

$$\frac{I_1}{I_2} = \frac{U_2}{U_1} = \frac{N_2}{N_1}$$

### 升压与降流的关系

| 操作 | 电压 | 电流 | 功率 |
|------|------|------|------|
| 升压（$N_2 > N_1$）| 增大 | 减小 | 不变 |
| 降压（$N_2 < N_1$）| 减小 | 增大 | 不变 |

**"升压必降流"**：这是远距离输电升压减损耗的理论依据。

### 多个次级绕组时的初级电流

当有多个次级绕组时，初级电流由**总功率**决定：

$$P_1 = P_2 + P_3 + \cdots = U_2 I_2 + U_3 I_3 + \cdots$$
$$I_1 = \frac{P_{\text{总}}}{U_1}$$

或用安匝守恒：$N_1 I_1 = N_2 I_2 + N_3 I_3 + \cdots$（各负载安匝之和）

### 典型例题

**例1（基础）**：$N_1 = 100$，$N_2 = 1000$，$I_1 = 10$ A，求 $I_2$。

$$I_2 = I_1\times\frac{N_1}{N_2} = 10\times\frac{100}{1000} = 1\text{ A}$$

验证：升压 10 倍，电流降低为 $1/10$。✓

**例2（求匝数）**：变压器初级 $I_1 = 0.5$ A，$U_1 = 220$ V；次级 $I_2 = 5$ A，求次级电压和匝数比。

$$P_1 = 220\times0.5 = 110\text{ W} = P_2 = U_2\times5$$
$$U_2 = 22\text{ V}$$
$$\frac{N_1}{N_2} = \frac{U_1}{U_2} = \frac{220}{22} = 10$$

**例3（输电线问题）**：发电机 $\varepsilon = 6$ kV，内阻 $r = 10$ Ω，经升压变压器（$N_1/N_2 = 1/100$）升压，输电线电阻 $R = 200$ Ω，求输电线上的电流和功率损耗。

$$U_2 = 6000\times100 = 600000\text{ V} = 600\text{ kV} \quad \text{（待家长核对，发电机端电压需考虑内阻）}$$
$$I_{\text{线}} = I_1\times\frac{N_1}{N_2} = \frac{I_{\text{发}}}{100}$$

（具体数值需知道负载，此处示意思路）

### 安匝积守恒的物理意义

$N_1 I_1 = N_2 I_2$ 是**安培·匝数**（磁动势）守恒的近似（理想情形），反映了铁芯中总磁动势由初次级共同产生，忽略励磁电流后，初级磁动势 = 次级磁动势。

### 易错点

1. 电流比是**匝数比的倒数**（$I_1/I_2 = N_2/N_1$），与电压比相反，容易混淆。
2. 多绕组时不能用 $I_1 = I_2 N_2/N_1 + I_3 N_3/N_1$（安匝叠加），需加绝对值或注意方向。
3. 空载时（次级无负载），$I_2 = 0$，理想变压器 $I_1 = 0$（无功率输出）；实际变压器有励磁电流。
