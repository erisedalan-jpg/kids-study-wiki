## 🎓 电磁感应应用（高中物理）

### 核心原理

所有电磁感应应用的基础：**磁通量变化 → 感应电动势 → 感应电流 → 能量转化**

法拉第电磁感应定律（1831年）开创了现代电力时代，是人类历史上最重要的科学发现之一。

### 主要应用领域

#### 1. 发电机（机械能 → 电能）

线圈在磁场中旋转，磁通量周期性变化，产生交变电动势：

$$\varepsilon = \varepsilon_m\sin(\omega t) = NBA\omega\sin(\omega t)$$

- 水力发电、火力发电、核电的发电机均基于此原理
- 风力发电机：风能 → 机械能 → 电能

#### 2. 变压器（改变电压等级）

利用互感原理，通过匝数比改变电压：

$$\frac{U_1}{U_2} = \frac{N_1}{N_2}$$

- 升压变压器：电厂 → 高压输电（减小线路损耗）
- 降压变压器：用户端降压（安全使用）

#### 3. 电动机（电能 → 机械能）

通电线圈在磁场中受安培力旋转（逆用电磁感应），但工作时同时有"反电动势"（自感效应）。

#### 4. 感应加热（电磁炉/感应炉）

高频交变磁场在金属（导体）中产生涡流，涡流发热加热金属：

$$P = I^2 R_{\text{涡}}$$

- 电磁炉：磁场 → 铁锅涡流 → 加热食物
- 感应炉：熔炼金属

#### 5. 无线充电（互感传能）

发射线圈产生高频交变磁场，接收线圈中产生感应电动势，转化为充电电流。

效率取决于两线圈的耦合系数和频率。

#### 6. 磁共振成像（MRI）

超导磁体产生强匀强磁场，射频线圈通过电磁感应与人体氢核共振，接收线圈检测感应信号，重建人体内部图像。

#### 7. 金属探测器

发射线圈产生磁场，金属中感应出涡流，涡流又产生反向磁场，改变接收线圈中的磁通量，触发检测信号。

### 能量转化汇总

| 应用 | 能量转化方向 |
|------|------------|
| 发电机 | 机械能 → 电能 |
| 变压器 | 电能（升/降压，功率不变） |
| 电动机 | 电能 → 机械能 |
| 感应加热 | 电能 → 热能 |
| 无线充电 | 电能 → 磁场能 → 电能 |

### 典型例题

**例（综合）**：某水电站发电机产生 $\varepsilon_m = 10$ kV 的交流电，经升压变压器（$N_1 : N_2 = 1 : 22$）升压后，通过电阻 $R_{\text{线}} = 10$ Ω 的输电线输送，末端再经降压变压器（匝数比待定）降至 220 V 用户端，输电线上电流 $I = 50$ A，求：(1) 升压后电压；(2) 线路损耗功率；(3) 降压变压器匝数比。

$$U_2 = U_1 \times \frac{N_2}{N_1} = \frac{10000}{\sqrt{2}}\times22 \approx 155.6\text{ kV} \quad \text{（待家长核对，峰值vs有效值）}$$
$$P_{\text{损}} = I^2R_{\text{线}} = 50^2\times10 = 25\text{ kW}$$
$$U_3 = U_2 - IR_{\text{线}} = 155600 - 50\times10 = 155100\text{ V} \approx 155\text{ kV}$$
$$\frac{N_3}{N_4} = \frac{U_3}{220} \approx 705:1$$

### 易错点

1. 变压器改变电压，**不改变频率**（输出交流电频率与输入相同）。
2. 升压后输电电流减小，但**功率不变**（理想变压器），损耗减小。
3. 感应加热中，涡流越大发热越快，但也会造成铁芯损耗——变压器铁芯用硅钢片正是为了减小涡流。
