## 🎓 电磁感应能量转化（高中物理）

### 核心原则：能量守恒

电磁感应过程中能量始终守恒：**其他形式能量 → 电能 → 热能（或其他形式）**

感应电流产生需要外力做功（克服安培力），外力做的功 = 回路获得的电能：

$$W_{\text{外力}} = \varepsilon q = \varepsilon I t$$

### 导体棒在导轨上运动的能量分析

**匀速情形**（外力 = 安培力，加速度为零）：

$$P_{\text{外力}} = F_{\text{外}}v = F_{\text{安}}v = BILv = \varepsilon I = P_{\text{电}}$$

所有电功率 = 电热功率：

$$P_{\text{电}} = I^2(R+r) = \frac{\varepsilon^2}{R+r}$$

外电阻分配：$P_R = I^2R$，内阻分配：$P_r = I^2r$

**加速情形**（外力 > 安培力）：

外力做功 = 动能增量 + 电能（热能）

$$W_{\text{外}} = \Delta E_k + Q_{\text{总}}$$

$$W_{\text{外}} - Q_{\text{总}} = \Delta E_k$$

**减速情形**（仅靠安培力制动，无外力）：

动能减少 = 全部转化为电热能：

$$\frac{1}{2}mv_0^2 - \frac{1}{2}mv^2 = Q_{\text{总}}$$

### 能量转化路径汇总

| 情形 | 转化路径 |
|------|---------|
| 匀速导体棒（有外力） | 机械能（外力做功）→ 电能 → 热能 |
| 导体棒减速（安培力制动） | 动能 → 电能 → 热能 |
| 发电机（线圈旋转） | 机械能 → 电能（→ 外部负载）|
| 变压器 | 电能（高压）→ 电能（低压，理想时无损耗）|
| 电磁炉/感应炉 | 电能 → 涡流热能 |

### 典型例题

**例1（匀速导体棒）**：$B = 1$ T，$L = 0.5$ m，$v = 4$ m/s，$R = 4$ Ω，$r = 1$ Ω，运动 $t = 5$ s，求各量。

$$\varepsilon = BLv = 1\times0.5\times4 = 2\text{ V}$$
$$I = \frac{2}{5} = 0.4\text{ A}$$
$$P_{\text{总}} = \varepsilon I = 2\times0.4 = 0.8\text{ W}$$
$$Q_{R} = I^2Rt = 0.16\times4\times5 = 3.2\text{ J}$$
$$Q_{r} = I^2rt = 0.16\times1\times5 = 0.8\text{ J}$$
$$W_{\text{外}} = P_{\text{总}}\times t = 0.8\times5 = 4\text{ J} = Q_R + Q_r \checkmark$$

**例2（减速制动）**：导体棒初速 $v_0 = 5$ m/s，质量 $m = 0.2$ kg，设最终静止，$R = 2$ Ω，内阻 $r = 0.5$ Ω，求产生的总热量。

动能全部转化为电热（无摩擦、水平导轨）：

$$Q_{\text{总}} = \frac{1}{2}mv_0^2 = \frac{1}{2}\times0.2\times25 = 2.5\text{ J}$$

其中 $R$ 上的热：$Q_R = Q_{\text{总}}\times\frac{R}{R+r} = 2.5\times\frac{2}{2.5} = 2\text{ J}$

### 易错点

1. 电热 = $I^2Rt$（整个回路）还是只有外电阻 $I^2Rt$？区分清楚：**总电热** = 总电功 = $\varepsilon It$；**外电阻热** = $I^2Rt_{\text{外}}$。
2. 安培力做的**负功**等于电能，不是热量。电能再通过 $I^2R$ 转化为热。
3. 变速运动时，$\varepsilon = BLv$ 中的 $v$ 是**瞬时值**，总电能需积分（或用能量守恒代替）。
