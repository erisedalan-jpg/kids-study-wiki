## 🎓 电磁振荡（高中物理）

### 定义

**电磁振荡**是指电容器和电感线圈构成的 LC 回路中，电场能与磁场能之间周期性相互转化的过程，伴随着电荷量、电流、电压的周期性变化。

### 产生条件

1. 有电容器（储存电场能）
2. 有电感线圈（储存磁场能）
3. 两者构成闭合回路（LC 回路）
4. 初始时电容有电荷（或电感有电流）

### 振荡过程分析（四个阶段）

**初始状态**：电容充满电（$Q = Q_{\max}$，$U_C = U_{\max}$），电流为零，电场能最大。

**第一阶段（放电，$0 \to T/4$）**：
- 电容放电，电流增大（自感阻碍，电流缓慢增大）
- 电场能 → 磁场能
- 至 $T/4$：电流最大，电场能为零，磁场能最大

**第二阶段（反向充电，$T/4 \to T/2$）**：
- 电感维持电流（自感效应），电容反向充电
- 磁场能 → 电场能
- 至 $T/2$：电容反向充满，电流为零，电场能最大

**第三、四阶段**：重复上述过程（反向），完成一个完整振荡周期。

### 与弹簧振子的类比

| 电磁振荡 | 弹簧振子 |
|---------|---------|
| 电荷量 $q$ | 位移 $x$ |
| 电流 $i$ | 速度 $v$ |
| 电容电压 $u_C$ | 弹力 $F = kx$ |
| 电场能 $\frac{q^2}{2C}$ | 弹性势能 $\frac{1}{2}kx^2$ |
| 磁场能 $\frac{1}{2}LI^2$ | 动能 $\frac{1}{2}mv^2$ |
| 自感系数 $L$ | 质量 $m$ |
| $1/C$（倒容） | 弹簧劲度系数 $k$ |

### 能量守恒

理想 LC 振荡回路（无电阻，不辐射）：

$$W = \frac{Q_m^2}{2C} + \frac{1}{2}LI^2 = \text{常数}$$

- 电场能最大时：$W = Q_m^2/(2C)$，$I = 0$
- 磁场能最大时：$W = LI_m^2/2$，$q = 0$

### 实际振荡的阻尼

实际回路中，线圈有电阻，每个周期都有能量损耗（电阻热耗），振幅逐渐减小，产生**阻尼振荡**。

要维持持续等幅振荡，需要外部补充能量——这是**振荡器**（LC 振荡电路 + 放大器）的工作原理。

### 典型例题

**例**：LC 回路中，$L = 0.1$ H，$C = 100\text{ μF} = 10^{-4}$ F，初始电容电压 $U_0 = 10$ V，求最大电流。

$$W = \frac{1}{2}CU_0^2 = \frac{1}{2}\times10^{-4}\times100 = 5\times10^{-3}\text{ J}$$
$$\frac{1}{2}LI_m^2 = W \implies I_m = \sqrt{\frac{2W}{L}} = \sqrt{\frac{10^{-2}}{0.1}} = 0.316\text{ A}$$

### 易错点

1. 电场能和磁场能**交替为零**，但总能量守恒（理想情形）。
2. "电容放电时电流从零增大"——自感线圈阻碍电流突变，电流**不能**瞬间达到最大值。
3. 振荡频率由 $L$ 和 $C$ 决定（见[[391-振荡周期与频率|振荡周期与频率]]），与初始储能大小（振幅）无关。
