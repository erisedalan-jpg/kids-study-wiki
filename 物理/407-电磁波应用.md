## 🎓 电磁波应用（高中物理）

### 概述

电磁波各频段特性不同，对应不同的应用领域，是现代文明的技术基础。

### 按频段的主要应用

#### 1. 无线电波（长波、中波、短波、超短波）

**长波（LF，$< 300$ kHz，波长 $> 1$ km）**
- 穿透海水，用于潜艇通信
- 全球广播（传播距离远）

**中波（MF，300 kHz~3 MHz）**
- AM 广播（调幅广播，国内中波段）
- 沿地面传播，夜间可借助电离层反射传远

**短波（HF，3~30 MHz）**
- 短波广播（国际广播）
- 借助电离层多次反射，可绕地球传播

**超短波/FM（VHF，30~300 MHz）**
- FM 调频广播、电视信号
- 警用无线电、航空通信

#### 2. 微波（300 MHz~300 GHz）

| 应用 | 原理 |
|------|------|
| 微波炉 | 2.45 GHz 微波使水分子共振，加热食物 |
| Wi-Fi | 2.4 GHz / 5 GHz 局域网通信 |
| 5G 通信 | 毫米波（24~100 GHz），大带宽、低延迟 |
| 卫星通信 | 微波穿透电离层，与卫星直接通信 |
| 雷达 | 微波脉冲探测飞机、舰船、气象 |

**雷达测距原理**：$d = ct/2$（$t$ 为往返时间）

#### 3. 红外线

- **体温测量**：红外热像仪（非接触式测温）
- **遥控器**：940 nm 近红外，电视/空调遥控
- **夜视仪**：探测人体、车辆热辐射
- **卫星遥感**：监测地表温度、植被覆盖

#### 4. 可见光

- 照明、光学仪器、光纤通信（将信息编码在光脉冲中）
- **激光**：单色相干光，用于精密测量、外科手术、光碟读写

#### 5. 紫外线

- **杀菌消毒**：UV-C（100~280 nm）破坏微生物 DNA
- **验钞**：钞票荧光防伪（UV 灯照射发光）
- **荧光材料**：黑光灯、荧光笔

#### 6. X射线

- **医学成像**：X 光片（骨骼、牙齿）
- **CT 扫描**：多角度 X 光 + 计算机重建
- **安全检查**：机场行李安检
- **材料检测**：焊缝探伤

#### 7. γ射线

- **癌症放射治疗**：γ刀（精准照射肿瘤）
- **PET-CT 扫描**：正电子发射 + γ探测
- **食品辐照灭菌**：延长保质期

### 电磁波应用的物理原理分类

| 应用原理 | 典型应用 |
|---------|---------|
| 电磁波传播信息 | 广播、电视、手机、Wi-Fi |
| 电磁波与物质共振 | 微波炉（水分子共振） |
| 电磁波测距/定位 | 雷达、GPS |
| 电磁波的热效应 | 红外加热、激光切割 |
| 电磁波的电离效应 | X射线成像、γ治疗 |

### 典型例题

**例（雷达测距）**：雷达发出微波脉冲，经目标反射后 $\Delta t = 2\times10^{-5}$ s 收到回波，求目标距离。

$$d = \frac{c\Delta t}{2} = \frac{3\times10^8\times2\times10^{-5}}{2} = 3000\text{ m} = 3\text{ km}$$

**例（微波炉频率）**：微波炉工作频率 $f = 2.45$ GHz，对应波长？

$$\lambda = \frac{c}{f} = \frac{3\times10^8}{2.45\times10^9} \approx 0.122\text{ m} = 12.2\text{ cm}$$

### 易错点

1. **不同频段电磁波能量不同**（$E = hf$）：频率越高，单个光子能量越大，电离能力越强（γ射线电离强，无线电波电离弱）。
2. **微波炉加热是分子共振，不是电磁感应**——金属容器会反射微波（不能放金属），而非被加热。
3. **GPS 利用微波**（不是无线电长波），通过多颗卫星信号的时间差定位。
