## 🎓 赫兹实验（高中物理）

### 历史背景

**海因里希·赫兹**（Heinrich Hertz，1857-1894）是德国物理学家。1888年，他通过实验首次人工产生并探测了电磁波，证实了麦克斯韦的理论预言，开启了无线通信时代的大门。

### 实验装置

**发射器**（振荡器）：
- 两个金属球（间距很小），连接感应圈（高压线圈）
- 高压使两球间放电，产生**高频振荡电流**（火花放电）
- 放电时产生高频交变电场和磁场，向外辐射电磁波

**接收器**（谐振环）：
- 一个圆形铜环，两端有小间隙（检波缺口）
- 当接收到与其固有频率匹配的电磁波时，产生共振，间隙中出现微弱火花

### 实验过程

1. 感应圈产生高压，两球间放电产生电磁波
2. 在发射器附近放置接收环，调整方向和距离
3. 观察到接收环间隙处出现感应火花
4. 改变接收环位置，验证电磁波的传播方向和波长
5. 通过金属板反射实验，测定电磁波波长，计算波速
6. 结果：$v = \lambda f = c = 3\times10^8$ m/s

### 实验意义

1. **首次证实**麦克斯韦电磁波理论（麦克斯韦于 1879 年去世，未能看到证实）
2. **首次产生和接收了人工电磁波**（当时频率约为 50 MHz 左右，波长约 6 m）
3. **验证了电磁波的波动性**：反射、折射、干涉、衍射
4. 开启了**无线通信技术**的时代（马可尼利用此原理发明无线电报）

### 赫兹实验的物理原理

**发射侧**：开放振荡回路（两球 = 展开的电容，导线 = 天线）产生高频振荡，向外辐射电磁波。

**接收侧**：另一个开放振荡回路，当其固有频率与发射频率相同时，产生**共振**（谐振），接收到最强信号。

这正是**调谐接收**的基本原理——现代收音机的前身。

### 与 LC 电路的联系

赫兹实验装置本质上是一个**开放 LC 振荡回路**：
- 两金属球 → 电容（可等效为振荡电容）
- 连接导线 → 电感（可等效为振荡电感）
- 开放结构 → 使电磁场能辐射到远处空间

封闭 LC 回路 → 电磁场局限在线圈和电容内
开放振荡回路 → 电磁场向空间辐射 = 电磁波发射

### 典型考点

1. 赫兹实验的目的：**证实麦克斯韦的电磁波预言**
2. 实验装置：开放振荡回路（发射）+ 谐振环（接收）
3. 实验结论：电磁波存在，在空气中速度约等于光速 $c = 3\times10^8$ m/s
4. 意义：奠定无线通信技术的物理基础

### 易错点

1. 赫兹实验是**证实**电磁波存在，不是**发现**（麦克斯韦理论**预言**了电磁波）。
2. 赫兹接收到的是电磁波引起的**感应电火花**，不是直接看到电磁波。
3. "赫兹"（Hz）作为频率单位，正是为了纪念这位物理学家。
