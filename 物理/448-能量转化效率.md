## 🎓 能量转化效率

### 定义

**能量转化效率**是指有用能量输出与总能量输入的比值：

$$\eta = \frac{W_{\text{有用}}}{E_{\text{输入}}} \times 100\%$$

对于热机：

$$\eta_{\text{热机}} = \frac{W}{Q_1} = \frac{Q_1 - Q_2}{Q_1} = 1 - \frac{Q_2}{Q_1}$$

其中 $Q_1$ 为从高温热源吸收的热量，$Q_2$ 为向低温热源放出的热量，$W = Q_1 - Q_2$ 为对外做的净功。

### 热力学第二定律对效率的限制

$$\eta_{\text{热机}} < 1 \quad (\text{因为必须放热} Q_2 > 0)$$

**卡诺效率**（理论最大效率）：

$$\eta_{\text{卡诺}} = 1 - \frac{T_2}{T_1}$$

- $T_1$：高温热源温度（K）
- $T_2$：低温热源温度（K）
- 实际热机效率 $\eta < \eta_{\text{卡诺}}$

### 提高热机效率的途径

1. 提高高温热源温度 $T_1$（提高燃烧温度）
2. 降低低温热源温度 $T_2$（冷却排气）
3. 减少摩擦、散热等额外损耗（减少无效能量损失）

### 常见装置效率对比

| 装置 | 典型效率 | 说明 |
|------|----------|------|
| 内燃机 | 25%~45% | 汽车发动机等 |
| 蒸汽轮机 | 35%~45% | 火力发电 |
| 电动机 | 85%~95% | 电能→机械能 |
| 发光二极管 | 30%~70% | 电能→光能 |
| 白炽灯 | ~5% | 电能→光能（大部分转为热） |

> 电动机效率高是因为电能→机械能不受热力学第二定律限制（不经过热的中转），热机受限制。

### 典型例题

**例题 1**：某热机每次循环从高温热源吸热 $Q_1 = 5000\ \text{J}$，向低温热源放热 $Q_2 = 3500\ \text{J}$，求：①对外做功 $W$；②热机效率 $\eta$。

**解**：

$$W = Q_1 - Q_2 = 5000 - 3500 = 1500\ \text{J}$$

$$\eta = \frac{W}{Q_1} = \frac{1500}{5000} = 30\%$$

**例题 2**：卡诺热机高温热源温度 $T_1 = 600\ \text{K}$，低温热源温度 $T_2 = 300\ \text{K}$，求理论最大效率。

$$\eta_{\text{卡诺}} = 1 - \frac{T_2}{T_1} = 1 - \frac{300}{600} = 50\%$$

### 易错点

1. $Q_2$ 是**放出**的热量，为正值；代入效率公式时不要加负号
2. 效率计算的分母是**总输入能量**（吸热），不是输出功
3. 提高效率不违反热力学定律，但效率不可能达到 100%（第二定律）
