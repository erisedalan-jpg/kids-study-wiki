# 坐标测量

> **一句话**：坐标测量是通过建立坐标系，确定研究对象的空间位置坐标，并利用坐标变化定量计算位移、速度、加速度等物理量的实验与分析方法。
> **English**: Coordinate measurement is the method of determining the spatial coordinates of an object in a reference system and using coordinate changes to calculate physical quantities such as displacement, velocity, and acceleration.

## 🎓 考点精讲

**定义与原理**  
在物理中，为了精确描述物体的位置与运动，需建立坐标系（如直线坐标系、平面直角坐标系）。物体的位置用坐标 $(x,y)$ 表示，位置变化用位移矢量 $\Delta \vec{r} = (x_2 - x_1,\, y_2 - y_1)$ 描述。对于纸带或频闪照片等实际测量问题，通过测量点迹的坐标差，结合时间间隔，可得到平均速度 $\bar{v} = \frac{\Delta x}{\Delta t}$，并进一步用逐差法或 $v\text{–}t$ 图像斜率求加速度。

**关键方法**  
1. **坐标系选取**：为简化计算，常将运动方向设为坐标轴正方向，或将互成角度的运动分解为沿 $x$ 轴和 $y$ 轴的两个分运动。  
2. **实验读数与记录**：用刻度尺测量打点计时器纸带上的点迹坐标时，要估读到分度值的下一位；注意计数点的间隔数 $n$ 与时间 $\Delta t = nT$（$T$ 为打点周期）。  
3. **数据处理**：速度常用“中间时刻速度”计算 $v_n = \frac{x_{n+1} - x_{n-1}}{2T}$；加速度优先用逐差法 $a = \frac{(x_4+x_5+x_6)-(x_1+x_2+x_3)}{9T^2}$ 减小偶然误差，亦可用 $v\text{–}t$ 图线斜率。

**典型应用**  
- 纸带类实验（探究匀变速直线运动、验证牛顿第二定律等）：通过刻度尺测量计数点间的坐标差，获得位移序列，再计算速度和加速度。  
- 平抛运动实验：通过建立水平、竖直方向的直角坐标系，测量闪光照片或斜槽落点坐标，由 $x=v_0 t,\, y=\frac12 gt^2$ 反推初速度；或利用坐标差值检验轨迹是否符合 $y \propto x^2$。  
- 力的合成与分解（实验）：用坐标纸记录弹簧测力计拉力的方向与大小，以坐标表示力矢量末端，用平行四边形定则求合力，并比较理论合力与实验值之间的坐标偏差。

## ⚠️ 高频易错点

- **读数忘估读**：使用毫米刻度尺测量坐标时，必须读到 $0.1\,\text{mm}$ 位（即 $0.01\,\text{cm}$），只读到毫米是缺失估读位的错误。  
- **计数点与计时点混淆**：打点计时器的打点周期 $T=0.02\,\text{s}$，若取每5个点为一个计数点，则时间间隔为 $5T$，非 $T$；计算速度时代入错误的时间会导致结果全错。  
- **坐标差正负号忽略**：计算位移时直接套入坐标值相减却忘记方向，导致速度或加速度的正负符号错误，无法正确判断匀加/匀减速运动。  
- **逐差法分段错误**：纸带数据为奇数段时仍盲目用偶数段公式，未将第一段或最后一段舍弃；或者逐差时间间隔取错，造成加速度数量级差错。
