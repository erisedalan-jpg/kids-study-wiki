# 力的合成与分解

> **一句话**：力的合成与分解是遵循平行四边形定则，用等效替代思想处理共点力系的基本方法，互逆操作。
> **English**: The composition and resolution of forces is the basic method for handling concurrent force systems using the parallelogram rule based on the principle of equivalent substitution, and these two operations are inverse to each other.

## 🎓 考点精讲

**定义/原理**：力的合成与分解的核心是**等效替代**。如果一个力作用在物体上产生的效果与几个力共同作用的效果相同，这个力就叫那几个力的**合力**，那几个力叫这个力的**分力**。求几个力的合力的过程叫力的合成，求一个力的分力的过程叫力的分解。两者均严格遵循**平行四边形定则**：以表示两个分力的有向线段为邻边作平行四边形，其对角线就表示合力的大小和方向。若为多个力合成，可先两两合成。三角形定则是平行四边形定则的简化推论。

**关键方法**：
1. **作图法**：严格按标度画出分力，用尺规作平行四边形，量出对角线长度与夹角（精度低，用于定性分析）。
2. **计算法**：两分力大小 $F_1, F_2$，夹角 $\theta$，合力大小 $F = \sqrt{F_1^2 + F_2^2 + 2F_1F_2\cos\theta}$，方向用 $\tan\alpha = \frac{F_2\sin\theta}{F_1 + F_2\cos\theta}$ 确定。
3. **正交分解法**（高频核心方法）：将各力分解到相互垂直的 $x, y$ 轴上，分别求代数和 $F_x, F_y$，合力 $F = \sqrt{F_x^2 + F_y^2}$，方向 $\tan\theta = \frac{|F_y|}{|F_x|}$。力的分解若无条件限制，有无数解；实际问题中常按力产生的**实际作用效果**或**题目所设正交方向**分解。

**典型应用**：高考中，本考点很少单独命题，而是作为**受力分析的基础工具**渗透在选择与大题中。常见设问：
- 动态平衡问题：如斜面上物体，在挡板缓慢转动过程中，分析挡板弹力与斜面弹力的变化，需用图解法或解析法分析合力恒定时分力的变化。
- 三力平衡极值问题：已知合力（重力）与一分力方向，求另一分力最小值，根据三角形定则知当二分力垂直时取 $F_{min} = G\sin\theta$。
- 连接体问题：对整体与隔离对象进行正确的正交分解，列平衡方程或牛顿第二定律分量式。

## ⚠️ 高频易错点

- **混淆“合力是否一定大于分力”**：合力大小范围 $|F_1 - F_2| \le F \le F_1 + F_2$，合力可能比分力小，也可能等于分力。
- **力的分解随意拆分矢量**：不按实际作用效果或约束条件随意分解，例如误将斜面上物体重力分解为垂直于斜面的分力 $mg\cos\theta$ 与竖直向下的分力，漏掉沿斜面分力 $mg\sin\theta$。
- **多力合成忘记先合成共线力**：直接跳入平行四边形导致构图混乱，应先处理共线力再作矢量多边形。
- **正交分解忘记合力方向判定**：求出 $F_x, F_y$ 后直接写大小，但方向角 $\theta$ 未说明是合力与 $x$ 轴的正向夹角，或象限判断错误。
