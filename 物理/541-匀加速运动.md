# 匀加速运动

> **一句话**：物体在任意相等时间内速度变化量相同、加速度矢量恒定不变的直线运动。
> **English**: A motion in which an object moves along a straight line with constant acceleration, so its velocity changes equally in any equal time intervals.

## 🎓 考点精讲

匀加速运动的核心是**加速度恒定**，即 $\vec{a} = \text{const}$。这是对运动进行定量描述的基础模型，高考对其考查主要围绕以下三个方面展开。

**定义与公式体系**：加速度恒定的直线运动速度公式为 $v_t = v_0 + at$，位移公式为 $s = v_0t + \frac{1}{2}at^2$，速度位移关系为 $v_t^2 - v_0^2 = 2as$。这三个公式构成了解题的基本工具。判断一段运动是否是匀加速运动，依据是“相等时间间隔内位移差恒定”，即 $\Delta s = s_n - s_{n-1} = aT^2$（$T$ 为固定时间间隔）。这一推论结合纸带打点计时器是实验题的高频考查方式。

**解题的关键方法**：高考中解决匀加速直线运动问题通常采用“三步法”——第一步明确所研究的阶段（初态、末态与加速度），第二步选取与之适配的公式（通常优先考虑不含时间 $t$ 的公式，避免引入不必要的未知量），第三步代入数据并注意正方向（通常初速度 $v_0$ 方向为正，加速度可正可负）。对于刹车、减速到零后反向加速等过程，务必先计算减速到零所需时间 $t_{\text{停}} = \frac{v_0}{|a|}$，判断物体是否已停下，再使用位移公式，这是高考运动学大题最常见的陷阱设计点。

**高考典型设问场景**：
1. **追及与相遇问题**：给出两物体的初速度、加速度和初始位置，要求学生推导相遇时间或临界距离，实质上都是匀加速运动公式的联立求解。
2. **传送带模型**：物块在传送带上先做匀加速运动（滑动摩擦力提供加速度），当速度达到传送带速度后转为匀速。这里匀加速阶段的时间与位移计算是整道题的得分关键。
3. **多阶段运动**：物体先做匀加速、再做匀速或匀减速，需分段计算再用时间与位移累加，考查对匀加速过程独立处理的能力。

## ⚠️ 高频易错点

- **公式符号误用**：公式 $s = v_0t + \frac{1}{2}at^2$ 中加速度 $a$ 须带符号。如果初速度方向为正，则加速时 $a > 0$，位移式取加号；减速时 $a < 0$，位移式实为 $s = v_0t - \frac{1}{2}|a|t^2$。混用正负号是计算题最普遍的失分原因。
- **刹车陷阱**：直接套用 $s = v_0t - \frac{1}{2}at^2$ 计算整个给定时间内的位移，而忽略物体可能在给定时间内已停止运动。正确做法是先判断 $t$ 与 $t_{\text{停}}$ 的关系，若 $t \ge t_{\text{停}}$，则用公式 $s = \frac{v_0^2}{2a}$ 即可求出最终位移。
- **混淆加速度与速度**：匀加速运动中速度增大不意味着加速度增大，加速度始终是恒定的。很多学生在图像题中把 $v\text{-}t$ 图的斜率（加速度）与纵坐标（速度）的变化趋势弄混。
- **传送带问题中参考系的混淆**：物块在传送带上的匀加速阶段，其加速度由相对于传送带的滑动摩擦力决定，但位移公式使用的地面参考系中的位移，切忌用物块相对传送带的位移直接代入匀加速运动公式求时间，必须转换成对地位移后再求解。
