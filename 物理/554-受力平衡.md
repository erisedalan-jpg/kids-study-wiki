# 受力平衡

> **一句话**：物体在多个力作用下，保持静止或匀速直线运动状态的条件是合力为零。
> **English**: An object subjected to multiple forces remains at rest or in uniform linear motion when the net external force is zero.

## 🎓 考点精讲

- **定义/原理**  
  物体的**平衡状态**包括**静态平衡**（静止）和**动态平衡**（匀速直线运动）。其核心判据是**合力为零**，即  
  \[
  \sum \vec{F} = 0
  \]  
  对于共点力系，常用正交分解法将各力投影到 \(x\)、\(y\) 轴，得到  
  \[
  \begin{cases}
  \sum F_x = 0 \\
  \sum F_y = 0
  \end{cases}
  \]  
  若是三力平衡，则可进一步用力的三角形定则或拉密定理求解。当多个物体通过绳、弹簧或接触面关联时，常需结合**整体法**与**隔离法**。

- **关键方法**  
  高考中处理平衡问题的标准步骤：  
  1. 明确研究对象（单个物体或系统）；  
  2. 对其隔离，并按重力、弹力、摩擦力、其他已知力的顺序进行**受力分析**；  
  3. 建立恰当的坐标系（通常使尽可能多的力落在坐标轴上）；  
  4. 列出平衡方程 \(\sum F_x = 0\)、\(\sum F_y = 0\)，对存在转动趋势的题目还可能需力矩平衡 \(\sum M = 0\)；  
  5. 解方程或利用图解（如动态平行四边形、矢量三角形）判断力的大小与方向变化。

- **典型应用**  
  常见设问方式：给定斜面倾角与动摩擦因数，求推力范围使物块不下滑或不滑动；轻杆、轻绳吊挂重物，通过几何关系求绳上张力；动态平衡问题（如缓慢转动的挡板支撑球，分析支持力与挡板压力的变化）。  
  典型情境：重为 \(G\) 的匀质圆柱体靠在竖直墙与倾斜挡板之间，挡板与水平面夹角从 \(90^\circ\) 缓慢减小到 \(0^\circ\)，问球对墙的压力如何变化——解题时需将重力沿垂直于墙面与垂直于挡板方向分解，或构建力三角形，观察边长对应的力如何随角度变化。

## ⚠️ 高频易错点

- **混淆“速度为零”与“合力为零”**：竖直上抛最高点速度为零但重力始终作用，此时非平衡状态；只有速度为零且加速度为零时才是平衡。  
- **受力分析漏力或添力**：常误将“惯性力”“冲力”当作真实力画入受力图，或遗漏接触面上的弹力、摩擦力（如斜面模型中忘掉支持力的水平分量）。  
- **动态平衡中错用“主动力”作为恒力**：当三个力中仅大小方向均不变的力（如重力）可作定点，方向不变的力（如墙壁支持力）作定向边，大小方向均可变的力作对边时，学生易把可变的力误认为方向不变，画错矢量三角形导致判断错误。  
- **整体法与隔离法混用**：计算系统内物体间相互作用力时，必须隔离该物体单独列平衡方程，若用整体法无法求出内力。
