# 完全弹性碰撞

> **一句话**：完全弹性碰撞是指碰撞前后系统总动能守恒的理想碰撞模型，同时满足动量守恒。
> **English**: A perfectly elastic collision is an idealized collision in which both momentum and kinetic energy of the system are conserved.

## 🎓 考点精讲

**定义/原理**：  
完全弹性碰撞需要同时满足两个守恒定律。设两物体质量分别为 \(m_1\)、\(m_2\)，碰前速度 \(v_1\)、\(v_2\)，碰后速度 \(v_1'\)、\(v_2'\)，则有  

\[
\begin{cases}
m_1 v_1 + m_2 v_2 = m_1 v_1' + m_2 v_2' \quad \text{(动量守恒)} \\[4pt]
\frac{1}{2}m_1 v_1^2 + \frac{1}{2}m_2 v_2^2 = \frac{1}{2}m_1 {v_1'}^2 + \frac{1}{2}m_2 {v_2'}^2 \quad \text{(动能守恒)}
\end{cases}
\]

联立可解得碰后速度的一般表达式（一维情况）：

\[
v_1' = \frac{(m_1 - m_2)v_1 + 2m_2 v_2}{m_1 + m_2}, \quad
v_2' = \frac{(m_2 - m_1)v_2 + 2m_1 v_1}{m_1 + m_2}
\]

这一组公式是高考计算题的直接工具。由公式可推出两个重要特例：  
- **等质量**（\(m_1=m_2\)）时，两球交换速度：\(v_1'=v_2,\; v_2'=v_1\)。  
- **一动一静**（\(v_2=0\)）时，若 \(m_1 = m_2\)，主动球停下，被撞球以原速前进；若 \(m_1 \ll m_2\)，主动球以接近原速率反向弹回，大质量球几乎不动。

**关键方法**：  
1. **识模**——看清题目是否明确“弹性碰撞”或给出碰后总动能不变的条件。  
2. **列式**——直接写出动量守恒和动能守恒的方程；若题目要求速解，可优先使用上述推导出的速度公式。  
3. **检验**——算完后可验算总动能是否不变，也可用恢复系数 \(e=1\) 辅助验证。

**典型应用**：  
- **两球碰撞类**：光滑水平面上两小球对心弹性碰撞，求碰后速度或分析质量大小关系。  
- **滑块与挡板/弹簧类**：滑块撞击弹簧连接的物体，弹簧恢复原长时相当于一次完全弹性碰撞。  
- **撞击摆动类**：子弹打木块中“子弹穿出”或“木块摆动”经常需先判定碰撞性质，再选择守恒量。

高考常从“一动一静”模型切入，结合动能定理、动量定理连续设问，要求学生能灵活选用公式并区分完全弹性碰撞与非弹性碰撞的守恒差异。

## ⚠️ 高频易错点

- **混淆守恒条件**：完全弹性碰撞必须同时满足动量守恒和动能守恒，不能只列一个方程。常见错误是误将动能守恒写出后又代入动量不守恒的情景。
- **漏判质量相等时的速度交换**：等质量物体发生完全弹性碰撞后速度互换，若不能直接使用这一结论，会在多体问题中增加计算量并易出错。
- **碰后速度方向判断失误**：当 \(m_1 < m_2\) 且 \(v_2=0\) 时，\(v_1'\) 为负值（反弹）；若错误代入绝对值或画错方向，后续受力分析会连锁出错。
- **误以为“黏在一起”是弹性碰撞**：两物体碰后共速（完全非弹性碰撞）动能损失最大，与完全弹性碰撞的动能守恒截然相反，审题不清常导致原理用错。
