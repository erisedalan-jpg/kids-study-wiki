# 方程平衡

> **一句话**：物体在平衡状态时所满足的合外力为零（以及合外力矩为零）的矢量方程，是力学与静电场中列等式求解未知量的核心工具。
> **English**: Equations that state the net external force (and net external torque) on an object in equilibrium is zero.

## 🎓 考点精讲

**定义/原理**  
在惯性参考系中，物体处于静止或匀速直线运动状态称为平衡状态。质点系的平衡条件为  
$$\sum \vec{F} = 0$$  
对具有转动效应的刚体，还需满足  
$$\sum \vec{M} = 0$$  
将其在直角坐标下正交分解，高考常列出的平衡方程为  
$$
\begin{cases}
\sum F_x = 0 \\
\sum F_y = 0 \\
\sum M = 0 \quad (\text{当不通过同一点的力或有矩时})
\end{cases}
$$
电荷在电场中的平衡同样适用 $\sum \vec{F} = 0$，需把电场力 $F = qE$ 纳入受力分析。

**关键方法**  
1. **选对象**：合理使用整体法与隔离法，避免将内力误放入方程。  
2. **查受力**：按重力→弹力→摩擦力→电场力等顺序，画出准确的受力示意图。  
3. **建坐标系**：尽量使未知力落在坐标轴上，或沿加速度方向（平衡时任意正交方向均可）。  
4. **列平衡方程**：将各力分解后写出 $F_x = 0$、$F_y = 0$，若有矩则选未知力作用点或支点为矩心，列出 $\sum M = 0$ 以消元。  
5. **解方程**：联立解得待求力、角度或动摩擦因数，并检验临界条件（如 $f \le \mu N$）。

**典型应用**  
- **斜面上的静平衡**：物体静止在倾角 $\theta$ 的斜面上，受推力 $F$ 平行于斜面。需列垂直斜面方向 $N + F\sin\theta = mg\cos\theta$，沿斜面方向 $F\cos\theta \pm f = mg\sin\theta$，结合 $f \le \mu N$ 求推力范围。高考常问“物体恰好不下滑/不上滑的临界力”。  
- **三力平衡的动态分析**：物体受三力而平衡，任意平移三个力必构成闭合矢量三角形。应用正弦定理 $\frac{F_1}{\sin\alpha_1} = \frac{F_2}{\sin\alpha_2} = \frac{F_3}{\sin\alpha_3}$，可分析缓慢移动过程中某力的变化，是动态平衡的高频解法。

## ⚠️ 高频易错点

- **受力分析多力或漏力**：轻杆的弹力不一定沿杆，弹簧弹力必沿轴线，误判导致方程列错。  
- **整体法误写内部力**：用整体法列 $\sum F = 0$ 时错把系统内物体间的摩擦力、弹力写入方程，造成平衡方程不成立。  
- **力矩正负号混乱**：未统一规定力矩正方向（常取逆时针为正），正负号反置导致合力矩方程出错。  
- **忽略理想化条件**：将“恰好平衡”与“已停止”的摩擦力混同，未分清静摩擦力随外力变化而平衡，以至于直接用滑动摩擦力公式。
