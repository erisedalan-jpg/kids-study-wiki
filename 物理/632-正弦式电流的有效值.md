# 正弦式电流的有效值

> **一句话**：让正弦式电流和某直流分别通过同一电阻，若在相等时间内产生的热量相等，则该直流的数值称为此正弦式电流的有效值。
> **English**: The effective value of a sinusoidal current is the value of the direct current that would produce the same heating effect in the same resistor over the same time interval.

## 🎓 考点精讲

**定义/原理**  
正弦式电流的有效值是根据电流的热效应定义的。设一直流电与一正弦式电流分别通过阻值相同的电阻，如果在相同时间内两者产生的焦耳热相等，就把该直流电的电流（或电压、电动势）的数值称为该正弦式电流的有效值。对正弦式（或余弦式）交变电流，有效值与峰值之间满足严格关系：  
\[
I = \frac{I_m}{\sqrt{2}},\quad U = \frac{U_m}{\sqrt{2}},\quad E = \frac{E_m}{\sqrt{2}}
\]  
其中 \(I_m, U_m, E_m\) 分别为电流、电压、电动势的峰值（最大值）。有效值实质上是对瞬时值的“方均根”值，适用于完整周期内的计算。

**关键方法**  
1. **识别波形**：首先判断电流是否为正弦式交流电。只有正弦（余弦）交变电流才能直接使用“除以√2”的关系；其他波形的有效值必须用定义式 \(I = \sqrt{\frac{1}{T}\int_0^T i^2 dt}\) 或分段热效应等效去求。  
2. **热效应等效的核心操作**：设未知有效值 \(I\)，对同一电阻 \(R\)，建立等式 \(I^2 R t = \)（交流电在时间 \(t\) 内产生的热量），解出 \(I\)。  
3. **选择公式的场景**：计算电功、电热、电功率以及交流电表的读数，均用有效值。涉及某段时间内通过导体横截面的电荷量，则用电流的平均值，切不可用有效值。

**典型应用**  
高考常以选择题或计算题出现：给出正弦交变电流的瞬时值表达式 \(i = I_m\sin\omega t\) 或 \(u = U_m\sin\omega t\)，要求求电热、电功率、保险丝的熔断电流、电容器的击穿电压（击穿电压要看峰值）；或结合变压器、远距离输电，考查原、副线圈电压有效值与匝数比的关系。另一常见设问是给学生一段非正弦的矩形波、锯齿波，要求用热效应定义求有效值，以检验是否盲目套用 \(I_m/\sqrt{2}\)。

## ⚠️ 高频易错点

- 混淆有效值与平均值：求热量、功率一律用有效值，求电量一律用电流的平均值，不可互换。  
- 非正弦波形仍用 \(I_m/\sqrt{2}\) 计算有效值：只有正弦式电流才满足该比例关系，其他波形必须回到热效应等效的定义求解。  
- 电容器的耐压值错用有效值：电容器的击穿电压对应的是交变电压的峰值（最大值），而非有效值。  
- “相同时间”的条件不明确：有效值定义中必须保证比较的时间相等，且通常取整数个周期；若时间取半周期，对于正弦交流电有效值关系仍然成立，但不能因此混淆定义。
