# 波的叠加

> **一句话**：几列波在介质中相遇时，每列波保持各自特性传播，质点的位移等于各列波单独引起位移的矢量和。
> **English**: The principle that when two or more waves overlap in a medium, the resultant displacement at any point is the vector sum of the displacements due to each individual wave.

## 🎓 考点精讲

### 定义与原理
波的叠加遵循**独立传播原理**和**叠加原理**：
1. **独立传播**：几列波相遇后，互不干扰，各自保持原有特性（频率、波长、振动方向）继续传播。
2. **叠加原理**：在重叠区域，任一质点的振动位移等于各列波单独在该点引起位移的矢量和。若两列同频率同振动方向的波在 $P$ 点引起的位移为 $y_1 = A_1\sin(\omega t + \varphi_1)$ 和 $y_2 = A_2\sin(\omega t + \varphi_2)$，则合位移：  
   $y = y_1 + y_2 = A\sin(\omega t + \varphi)$  
   其中合振幅 $A = \sqrt{A_1^2 + A_2^2 + 2A_1A_2\cos\Delta\varphi}$，$\Delta\varphi = \varphi_2 - \varphi_1$ 为相位差。

### 干涉条件与规律
**相干条件**：频率相同、振动方向相同、相位差恒定。满足相干条件的两列波叠加产生**干涉**现象。
- **振动加强（干涉相长）**：$\Delta\varphi = 2k\pi$（$k \in \mathbb{Z}$），合振幅 $A = A_1 + A_2$。若波源同相，则波程差 $\delta = k\lambda$。
- **振动减弱（干涉相消）**：$\Delta\varphi = (2k+1)\pi$，合振幅 $A = |A_1 - A_2|$。若波源同相，则波程差 $\delta = (2k+1)\frac{\lambda}{2}$。

### 关键方法
解决波的叠加问题，核心步骤为：
1. **判断相干性**：确认两波源是否满足频率相同、振动方向一致、相位差恒定。
2. **计算波程差或相位差**：通常利用 $\Delta\varphi = \frac{2\pi}{\lambda}\delta + \varphi_{02} - \varphi_{01}$。
3. **应用干涉条件**：判定空间各点是加强还是减弱，或求解加强/减弱点的位置分布。

### 典型应用
高考常以**水波干涉**、**声波干涉**或**光波双缝干涉**（虽然光波在几何光学中处理，但其干涉图样分析常与机械波类比）为情境设问：
- **判断空间某点是加强还是减弱**：直接代波程差公式与波长比较。
- **求解振动加强点的坐标或轨迹**：结合几何关系（如双曲线定义）列方程求解。

## ⚠️ 高频易错点

- **混淆“叠加”与“干涉”**：任意两列波相遇均发生叠加，但只有满足相干条件才能产生稳定的干涉图样（即形成稳定的振动加强区和减弱区）。
- **误判振动加强/减弱的条件**：混淆波程差条件与振动步调的关系。当两相干波源“步调相反”（相位差为 $\pi$）时，波程差为 $\lambda/2$ 的偶数倍反而是减弱区。
- **忽略振动方向的矢量性**：只进行数值相加而未考虑振动方向。若两列波振动方向垂直，合位移并非标量和。
- **干涉时合振幅误用**：错误地认为合振幅恒为 $A_1 + A_2$ 或 $|A_1 - A_2|$，忽略了干涉区域中振幅随相位差连续变化这一事实。
