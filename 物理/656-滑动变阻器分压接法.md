# 滑动变阻器分压接法

> **一句话**：滑动变阻器分压接法是将滑动变阻器以分压器形式接入电路，使负载两端电压能够从零连续调节至电源电压的接法。
> **English**: A slide rheostat voltage divider connection is a circuit configuration in which the rheostat acts as a voltage divider, enabling the voltage across the load to be continuously adjusted from zero up to the supply voltage.

## 🎓 考点精讲

**定义与原理**  
将滑动变阻器两固定端 A、B 直接接到电源两端，负载电阻 R 的一端接在变阻器的一个固定端（如 A），另一端接在滑片 P 上。设滑动变阻器总阻值为 R₀，滑片将 R₀ 分为 Rₐₚ 与 Rₚ₆，且 Rₐₚ + Rₚ₆ = R₀。负载与 Rₐₚ 并联，并联后的等效电阻  
\[
R_{\text{并}} = \frac{R_{\text{AP}}R}{R_{\text{AP}}+R}
\]  
回路总电阻 Rₜ = Rₚ₆ + R_{\text{并}}，干路电流 I = E / Rₜ（忽略电源内阻）。负载两端的电压即为并联部分的电压：  
\[
U_{\text{负载}} = I \cdot R_{\text{并}} = \frac{E \cdot R_{\text{并}}}{R_{\text{PB}} + R_{\text{并}}}
\]  
当滑片移到 A 端时，Rₐₚ = 0，R_{\text{并}} = 0，U_{\text{负载}} = 0；滑片移到 B 端时，Rₚ₆ = 0，R_{\text{并}} = R₀∥R，此时 U_{\text{负载}} = E。因此输出电压可在 0 ~ E 间连续变化。

**关键方法与选用依据**  
高考实验题中，选择分压接法的典型情形：  
- 要求负载电压从零开始调节（如描绘小灯泡的伏安特性曲线、校正电压表）。  
- 滑动变阻器最大阻值远小于负载电阻（即“小控大”），此时用限流接法电压调节范围过小，分压接法调节范围更宽。  
- 题目明确要求“电压调节范围尽可能大”或“便于多次测量”。  

连接实物时须注意：开关闭合前，滑片应置于使负载电压为零的一端，以保护电路元件。

**典型高考设问**  
1. **实验电路设计**：给定器材，画出分压式电路图或连接实物图。  
2. **器材选择**：根据“电压从零开始”“多测几组数据”等条件，要求选出滑动变阻器并说明选用分压接法的理由。  
3. **故障分析**：若移动滑片时电压表示数几乎不变，判断是否误接成限流或接线柱连接错误。  
4. **调节特性**：分析负载两端电压随滑片移动的变化规律，解释采用小阻值变阻器的原因。

## ⚠️ 高频易错点

- **滑片初始位置**：闭合开关前，滑片应滑至使负载分压为零的一端（通常是接电源负极的固定端），错放在最大电压端可能烧毁小灯泡或电表。  
- **分压与限流混淆**：分压接法电压可调范围 0 ~ E，限流接法为 ER/(R+R₀) ~ E；凡要求电压从零开始，必须用分压。  
- **变阻器阻值选择**：分压接法中，通常选用最大阻值较小的滑动变阻器，使电压调节更均匀；若选用大阻值变阻器，输出特性前段平坦、后段陡峭，不易精细调节。  
- **实物连线错误**：把变阻器两下接线柱都接入电路（等效为限流），或滑片接线同时接到同一固定端，导致无法实现分压。
