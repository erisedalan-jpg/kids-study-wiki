# 电功率计算

> **一句话**：电功率是描述电流做功快慢的物理量，定义为单位时间内电流所做的功，公式为 $P = \frac{W}{t} = UI$，纯电阻电路中还可表达为 $P = I^2 R = \frac{U^2}{R}$。
> **English**: Electric power is the rate at which electrical energy is transferred by an electric circuit, defined as work done per unit time, with the general formula $P = UI$.

## 🎓 考点精讲

**定义与原理**  
电功率的普遍定义式：$P = \frac{W}{t} = UI$，适用于一切电路。其中 $U$ 为电路两端电压，$I$ 为通过电路的电流。对于纯电阻电路（电能完全转化为内能），结合欧姆定律 $U = IR$，可导出 $P = I^2 R = \frac{U^2}{R}$。  
在非纯电阻电路（如含电动机、电解槽）中，电能一部分转化为内能，一部分转化为其他形式能。此时 **电功率 $P = UI$**、**热功率 $P_{\text{热}} = I^2 R$**，且总有 $P > P_{\text{热}}$，差值即为转化为机械能等的功率：$P_{\text{机}} = UI - I^2 R$。  
闭合电路中，电源总功率 $P_{\text{总}} = EI$，输出功率 $P_{\text{出}} = UI = I^2 R_{\text{外}}$，内耗功率 $P_{\text{内}} = I^2 r$。当外电阻 $R = r$ 时，电源输出功率最大，$P_{\text{max}} = \frac{E^2}{4r}$，但此时效率仅为 50%。

**关键方法**  
1. **判定电路性质**：先判断是否为纯电阻电路。若含电动机、电解槽等，切忌将 $P = \frac{U^2}{R}$ 用于计算总功率或热功率。  
2. **选择正确公式**：求电流做功总功率一律用 $P = UI$；求发热功率一律用 $P_{\text{热}} = I^2 R$；纯电阻电路中三者可任意选用。  
3. **功率关系分析**：对于非纯电阻用电器，列出功率平衡方程 $UI = I^2 R + P_{\text{其它}}$。  
4. **电源最大输出功率问题**：抓住 $R_{\text{外}} = r$ 的条件，并注意区分“最大功率”与“最高效率”的不同要求。

**典型应用**  
高考中常以 **电动机电路** 为背景考查功率计算。例如，一台直流电动机线圈电阻 $R = 2\,\Omega$，接在 $U = 10\,\text{V}$ 电压下，电流 $I = 0.5\,\text{A}$，则电功率 $P = UI = 5\,\text{W}$，热功率 $P_{\text{热}} = I^2 R = 0.5\,\text{W}$，输出的机械功率为 $4.5\,\text{W}$。若电动机被卡住不转，则成为纯电阻，电流剧增，容易烧毁。  
另一常见设问是 **滑动变阻器消耗的最大功率**，通常需要将定值电阻等效为电源内阻的一部分，再利用 $R_{\text{外}} = r_{\text{等效}}$ 求解。

## ⚠️ 高频易错点

- **混淆电功率与热功率**：非纯电阻电路中，总功率用 $P = UI$，热功率用 $P = I^2 R$，不能直接用 $P = \frac{U^2}{R}$ 计算总功率或热功率。
- **误用纯电阻公式计算电动机功率**：看到“电阻”就直接代入 $P = U^2/R$，忽略了电动机的反电动势，导致总功率或热功率计算错误。
- **电源输出功率最大时效率误判**：当外电阻等于内阻时电源输出功率最大，但此时内阻消耗功率与输出功率相等，效率仅为 50%，并非最高效率状态。
- **电动机卡住时的电路转变遗漏**：电动机不转时，反电动势消失，电路瞬间变为纯电阻，电流会远大于正常工作电流，计算时需立即切换模型。
