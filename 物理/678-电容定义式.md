# 电容定义式

> **一句话**：电容器所带电荷量 Q 与两极板间电势差 U 的比值定义为电容，即 C=Q/U。
> **English**: Capacitance is defined as the ratio of the charge Q on either conductor to the potential difference U between them, expressed as C = Q/U.

## 🎓 考点精讲

**定义/原理**：电容的定义式为 `C = Q / U`。其中 Q 是电容器一个极板所带电荷量的绝对值（单位：库仑 C），U 是两极板间的电势差（单位：伏特 V），C 是电容（单位：法拉 F，1F = 1 C/V）。该定义式是普适的，适用于任何电容器。电容 C 在数值上等于两极板间电势差每升高 1V 所需充入的电荷量。**关键理解**：电容 C 由电容器自身的结构（极板正对面积 S、极板间距 d、电介质介电常数 εᵣ）决定，是一个固有属性，与是否带电 Q 以及电压 U 均无关。定义式只是给出了测量或计算电容的方法，而非决定电容大小的因果式。

**关键方法**：解决电容问题需严格区分**定义式** C=Q/U 和**决定式** C=εᵣS/(4πkd)。常见解题步骤为：
1. 审题判断电容器是“电压不变”（始终与电源相连）还是“电荷量不变”（充电后断开电源）。
2. 依据决定式分析结构参数（S、d、εᵣ）变化导致的电容 C 变化。
3. 再结合定义式与不变量，推演 Q、U、E（由 E=U/d 导出）的变化趋势。

**典型应用**：高考主要考查电容两类动态变化问题：
1. **电压 U 恒定型**：如平行板电容器始终接在电池两端，若增大极板间距 d，由决定式知 C 变小；由定义式 C=Q/U 知 U 不变 C 减小则 Q 减小；由 E=U/d 知场强 E 减小。
2. **电荷量 Q 恒定型**：如充电后断开开关，若减小正对面积 S，C 减小；由 U=Q/C 知 U 增大；由 E=U/d=Q/(Cd)，结合决定式推得 E 与 S 无关而增大。

## ⚠️ 高频易错点

- **误认为 C 与 Q、U 成比例**：由公式 C=Q/U 错误推断 C 随 Q 增大而增大、随 U 增大而减小。必须明确电容 C 是固有属性，Q-U 图像应是一条过原点的倾斜直线，其斜率的大小即为电容 C。
- **混淆定义式与决定式**：在动态分析中误用决定式替换定义式，或不清楚二者结合使用的方法。定义式适用于所有电容器，决定式仅适用于平行板电容器（且忽略边缘效应）（待家长核对）。
- **物理量对象对应错误**：解题时误将总电压变化当作某一物理量不变的依据，或是在含二极管的断路复杂电路中判断不清电量变化趋势。
- **单位换算与计算失误**：直接使用 F 导致数量级错误，忘记常用单位皮法（pF）和微法（μF）与 F 的进率换算。
