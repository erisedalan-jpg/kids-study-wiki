# 电流与匝数成反比

> **一句话**：理想变压器中，原、副线圈电流的有效值之比与线圈匝数成反比，即 $\frac{I_1}{I_2} = \frac{n_2}{n_1}$。
> **English**: In an ideal transformer, the current in the coils is inversely proportional to the number of turns.

## 🎓 考点精讲

**定义/原理**  
对于理想变压器（无漏磁、无铜损铁损、无磁饱和），由能量守恒，输入功率等于输出功率：  
$$
U_1 I_1 = U_2 I_2
$$  
同时电压与匝数成正比：$\frac{U_1}{U_2} = \frac{n_1}{n_2}$。  
联立可得电流与匝数成反比：  
$$
\frac{I_1}{I_2} = \frac{n_2}{n_1}
$$  
即匝数多的线圈电流小，匝数少的线圈电流大。该关系对同一理想变压器各线圈的电流有效值成立，也适用于瞬时值之比。  
当变压器有多个副线圈时，电流关系由功率守恒直接导出：$U_1 I_1 = U_2 I_2 + U_3 I_3 + \dots$，而不能简单套用匝数反比公式。

**关键方法**  
处理变压器电流问题的标准步骤：  
1. 明确是否为理想变压器，以及副线圈个数。  
2. 利用电压比关系 $\frac{U_1}{U_2} = \frac{n_1}{n_2}$ 确定各线圈电压。  
3. 对单一副线圈，直接使用 $\frac{I_1}{I_2} = \frac{n_2}{n_1}$；多副线圈时，必须用功率关系 $U_1 I_1 = \sum U_k I_k$ 求电流。  
4. 动态分析时，记住“谁决定谁”：副线圈电流 $I_2$ 由负载决定，$I_2$ 变化后通过匝数反比关系迫使原线圈电流 $I_1$ 变化。

**典型应用**  
高考常考以下设问：  
- **匝数比变化的动态问题**：滑片改变副线圈匝数，分析各电流表读数变化。依据：$n_2$ 增大时 $U_2$ 增大，若负载不变则 $I_2 = \frac{U_2}{R}$ 增大，再由 $\frac{I_1}{I_2} = \frac{n_2}{n_1}$ 判断 $I_1$ 变化（此处 $n_2$ 也变大，需结合功率关系或直接代入 $I_1 = \frac{n_2}{n_1}I_2$ 综合判断）。  
- **远距离输电模型**：升压变压器副线圈电流 $I_2$ 小，可减小输电线上的损耗 $P_{损} = I_2^2 r$，正是利用了“电流与匝数成反比”。  
- **原线圈含串联电阻的电路**：此时原线圈电压 $U_1$ 不等于电源电压，需联立变压关系与电路方程，利用 $\frac{I_1}{I_2} = \frac{n_2}{n_1}$ 将两边电流关联求解。

## ⚠️ 高频易错点

- **乱套公式至多副线圈**：有多个副线圈时直接用 $I_1/I_2 = n_2/n_1$ 是错误的，必须用功率关系 $U_1 I_1 = U_2 I_2 + U_3 I_3 + \dots$ 计算。
- **混淆决定关系**：误认为原线圈电流 $I_1$ 决定副线圈电流 $I_2$。实际上理想变压器中，$I_2$ 由负载决定，$I_1$ 随之变化以提供所需功率。
- **忽视理想变压器条件**：实际变压器存在漏磁、铜损铁损，匝数反比关系不严格成立。高考若无特别说明，一般按理想变压器处理，但审题时需确认。
- **瞬时值方向误判**：电流与匝数成反比通常用于有效值或最大值之比，涉及瞬时值方向时，需结合楞次定律判断，不可机械代入反比关系导致符号错误。
