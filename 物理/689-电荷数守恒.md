# 电荷数守恒

> **一句话**：在任何物理过程中，系统所带电荷的代数和保持不变，电荷既不能创造也不能消灭。
> **English**: In any physical process, the algebraic sum of electric charges in an isolated system remains constant; charge can neither be created nor destroyed.

## 🎓 考点精讲

**定义/原理**  
电荷数守恒是自然界基本守恒定律之一。其核心表述为：一个与外界没有电荷交换的系统，电荷的代数和始终保持不变。微观上，表现为反应前后质子数（或电荷数）的代数和相等；宏观上，表现为电路节点处流入流出电流相等，即基尔霍夫电流定律：$\sum I_\text{in} = \sum I_\text{out}$。在核反应与粒子衰变中，反应前后总电荷数不变，如 $\alpha$ 衰变 $\ce{^{A}_{Z}X -> ^{A-4}_{Z-2}Y + ^{4}_{2}He}$，电荷数 $Z$ 守恒。

**关键方法**  
解决电荷数守恒问题的通用步骤：  
1. 明确研究对象（系统），判断是否与外界有电荷交换。  
2. 列出初、末态系统内各物体所带电荷量（带正负号）。  
3. 根据守恒列方程：$Q_\text{初} = Q_\text{末}$。  
4. 结合其他物理规律（如质量数守恒、能量守恒、电压关系）联立求解。  
对于电路节点问题，直接列 $\sum I_\text{in} = \sum I_\text{out}$。

**典型应用**  
高考常见设问：  
- **核反应与粒子物理**：给出核反应方程，要求推断未知粒子的电荷数与质量数。如 $\ce{^{14}_{7}N + ^{4}_{2}He -> ^{17}_{8}O + X}$，由电荷守恒 $7+2=8+Z_X$ 得 $Z_X=1$，知 X 为质子。  
- **静电平衡与电荷分配**：两个完全相同的不带电金属小球接触后再分开，根据电荷均分原理求电荷分配；若不守恒情景（如接地），则需结合电荷守恒与电场分布判断。  
- **电容器电路**：含容电路动态分析中，利用电荷守恒分析电容器极板电荷变化量，常配合节点电流关系。

## ⚠️ 高频易错点

- **混淆电荷数与质量数**：核反应中把电荷数当质量数计算，或只考虑质量数守恒而忽略电荷数守恒，导致推断粒子错误。  
- **忽略电荷的正负**：列电荷守恒方程时漏掉负电荷的符号，导致结果错误。  
- **系统选取不恰当**：误把与外界有电荷交换的部分当作孤立系统，例如带电小球接触大地时系统电荷不守恒，必须明确“与外界无交换”的前提。  
- **电路节点误判**：在复杂电路中，错把流入节点电流之和写成某支路电流，未严格使用 $\sum I_\text{in} = \sum I_\text{out}$ 而凭感觉列式。
