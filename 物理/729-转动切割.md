# 转动切割

> **一句话**：导体在匀强磁场中绕某点转动、其有效部分切割磁感线产生感应电动势的现象，核心公式为 `E = ½ B L² ω`（导体棒绕端点转动）或等效的磁通量变化率表达。
> **English**: A conductor rotating in a uniform magnetic field cuts magnetic flux lines, generating an induced emf; the key formula is E = ½ B L² ω for a rod rotating about one end.

## 🎓 考点精讲

- **定义与原理**：转动切割是**电磁感应**的一种特殊形式。当一段直导体（长度 `L`）在磁感应强度为 `B` 的匀强磁场中，绕其一端以角速度 `ω` 匀速转动，且转动平面与磁场垂直时，导体各点线速度 `v = ω r` 不同，**平均切割速度**取中点线速度 `v̅ = ω·(L/2)`。因此感应电动势  
  `E = B L v̅ = B L · (ω L/2) = ½ B L² ω`。  
  若从磁通量变化角度推导，导体在 `Δt` 时间内扫过的面积 `ΔS = ½ L² Δθ`，磁通量变化 `ΔΦ = B ΔS`，由法拉第电磁感应定律 `E = ΔΦ/Δt = ½ B L² ω`，二者等价。

- **关键方法**：解决转动切割问题的标准步骤：
  1. 确认磁场方向、转动平面、有效切割长度（是否为直导体在磁场中的投影长度）。
  2. 判断是**绕端点转动**还是**绕其他点转动**：若绕中点转动，等效为两段反向串联，总电动势为零；若绕任意点，可采用微元法或等效切割长度计算。
  3. 对**线圈在磁场中转动**（如交流发电机模型），直接应用 `E = N B S ω sin(ωt)`，其中 `S` 为线圈面积，`N` 为匝数。
  4. 计算通过电阻的电荷量时，优先使用平均电动势 `E̅ = N ΔΦ/Δt`，再利用 `q = I̅ Δt = N ΔΦ / R`。

- **典型应用**：高考中常见设问：
  - **单棒转动**：如“金属棒 OA 绕 O 点在匀强磁场中匀速转动，求 OA 两端电势差”。直接代入 `U = ½ B L² ω`，注意分析哪端电势高（右手定则）。
  - **圆盘发电机**：法拉第圆盘可视为无数辐条并联，中心与边缘之间的电动势仍为 `½ B R² ω`，注意等效电路为内阻极小的电源。
  - **交流电最大值**：矩形线圈在匀强磁场中匀速转动，最大感应电动势 `E_m = N B S ω`，瞬时值 `e = E_m sin(ωt)`（从中性面开始计时），常结合有效值、变压器等综合考查。

## ⚠️ 高频易错点

- **错用平动切割公式**：将转动切割直接套用 `E = B L v` 并取端点线速度 `v = ω L`，得出 `E = B L² ω`，忽略了平均速度应为中点线速度。
- **混淆转动轴位置**：导体绕端点转动才是 `½ B L² ω`；若绕棒上某点转动，必须分段处理或等效为两段转动电动势之差。部分学生误以为任何转动都可用同一公式。
- **感应电动势方向判断失误**：转动切割仍满足楞次定律与右手定则（或手掌平伸，让磁感线穿手心，拇指指向导体运动方向——但转动中各点速度方向不同，应取等效切割方向，即棒整体扫过的速度方向）。建议用法拉第电磁感应定律结合楞次定律辅助判断。
- **圆盘切割误解为单棒**：法拉第圆盘转动时，沿半径方向的电动势恒定，但若连接外部电路，需将圆盘视为无数辐射状导体并联，总电动势仍为 `½ B R² ω`，而内阻很小。学生常误以为圆盘上不同半径处电动势叠加或不知道如何等效电路。
