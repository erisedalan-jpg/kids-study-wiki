# 高压输电

> **一句话**：高压输电是通过升压变压器提高输电电压以减小输电电流，从而大幅降低输电线路上因电阻发热造成的功率损耗（$P_{\text{损}} = I^2 R_{\text{线}}$）的电能输送方式。
> **English**: High-voltage power transmission is a method of delivering electrical energy by stepping up the voltage to reduce current, thereby significantly lowering resistive power loss ( $P_{\text{loss}} = I^2 R_{\text{line}}$ ) along transmission lines.

## 🎓 考点精讲

- **定义/原理**：在输送电功率 $P$ 一定时，输电电压 $U$ 与输电电流 $I$ 满足 $P = UI$。输电线存在电阻 $R_{\text{线}}$，电流通过时产生的热功率损耗为 $P_{\text{损}} = I^2 R_{\text{线}}$。因此，升高输电电压可使输电电流成反比减小，损耗则以电流的平方倍关系降低。这是远距离输电必须采用高压的根本原因，遵循能量守恒和焦耳定律。输电过程的完整电压关系为：$U_{\text{升}} = U_{\text{损}} + U_{\text{降}}$，其中 $U_{\text{损}} = I R_{\text{线}}$ 是输电线上的电压降。

- **关键方法**：解决远距离输电问题的标准步骤——① 明确三个回路（发电机端升压回路、高压输电线回路、用户端降压回路）；② 抓住两个变压器的匝数比关系：升压变压器 $\frac{U_1}{U_2} = \frac{n_1}{n_2}$，降压变压器 $\frac{U_3}{U_4} = \frac{n_3}{n_4}$；③ 确定输电回路中的电流是计算损耗的枢纽：$I_{\text{线}} = \frac{P_{\text{出}}}{U_{\text{出}}} = \frac{P_{\text{用户}} + P_{\text{损}}}{U_{\text{出}}}$；④ 计算输电损耗的公式统一用 $P_{\text{损}} = I_{\text{线}}^2 R_{\text{线}}$ 或 $U_{\text{损}} = I_{\text{线}} R_{\text{线}}$，**严禁混淆输电电压与导线损失电压**。

- **典型应用**：高考常见设问角度——给定输送功率、输电距离和导线电阻率，要求计算采用不同输电电压时的损耗功率与输电效率；或者根据用户端电压与功率的需求，反推升压变压器和降压变压器的匝数比。典型场景：（1）某电站以 $P=100\text{ kW}$ 输出，输电线电阻 $R_{\text{线}}=5\,\Omega$，分别用 $U_1=5\text{ kV}$ 和 $U_2=50\text{ kV}$ 输送，计算损耗功率和效率；（2）画出远距离输电的全电路示意图，并在不同节点上标出电压、电流和功率，用以判断电压表、电流表的读数。

## ⚠️ 高频易错点

- **混淆输电电压与导线损失电压**：用 $P_{\text{损}} = \frac{U_{\text{输}}^2}{R_{\text{线}}}$ 计算损耗，这是最致命的错误。输电线损耗公式应是 $P_{\text{损}} = I_{\text{线}}^2 R_{\text{线}}$ 或 $P_{\text{损}} = \frac{U_{\text{损}}^2}{R_{\text{线}}}$，此处 $U_{\text{损}}$ 是输电线两端的电压降，远小于输电电压。
- **认为升压后电流一定按匝数比减小**：在远距离输电的问题中，输电线上运行的实际电流由输送功率和输电电压决定（$I = P/U$），变压器匝数比约束的是原副线圈的电压比，副线圈电流由负载回路的整体决定，必须通过功率守恒逐步求解，不能机械套用单一变压器中 $\frac{I_1}{I_2} = \frac{n_2}{n_1}$ 来直接求输电电流。
- **忽略输电线电阻的分压效应**：降压变压器原线圈的输入电压不是升压变压器的副线圈输出电压，而是 $U_3 = U_2 - I_{\text{线}}R_{\text{线}}$。漏掉这个电压降会导致降压匝数比计算出错。
- **将导线电阻直接用 $R = \rho \frac{l}{S}$ 代入时单位或长度出错**：输电距离 $L$ 对应导线来回总长度 $2L$，很多学生只代入单程距离。
