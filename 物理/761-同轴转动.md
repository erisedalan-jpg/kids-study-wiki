# 同轴转动

> **一句话**：同轴转动是指多个质点绕同一固定轴做圆周运动的模型，各质点角速度相等，线速度与半径成正比。
> **English**: Coaxial rotation refers to the model where multiple particles rotate around the same fixed axis, sharing the same angular velocity; their linear speeds are proportional to the radius.

## 🎓 考点精讲

**定义/原理**
同轴转动是圆周运动中常见的传动模型之一。当两个或多个轮盘、齿轮或质点固定在同一转轴上绕该轴转动时，它们的转动周期 \(T\)、频率 \(f\) 和角速度 \(\omega\) 完全相同，即：
\[
\omega_1 = \omega_2 = \omega,\quad T_1 = T_2 = T,\quad f_1 = f_2 = f
\]
由于线速度与角速度的关系 \(v = \omega r\)，在 \(\omega\) 相等的情况下，各质点的线速度 \(v\) 与其到轴心的距离 \(r\) 成正比：
\[
\frac{v_1}{r_1} = \frac{v_2}{r_2} = \omega
\]

**关键方法**
解决同轴转动问题，核心是抓住“角速度相同”这一特征。步骤：
1. 识别装置中哪些部分属于同轴转动（同一根轴上固定的轮、盘或质点）。
2. 写出角速度相等的关系式，并结合 \(v = \omega r\)、\(a_n = \omega^2 r\) 推导线速度、向心加速度等量。
3. 若系统包含同轴转动与皮带传动组合（例如两轮同轴再通过皮带连接其他轮），需将“同轴角速度相等”和“皮带线速度相等”两条规律联合建立方程。

**典型应用**
高考中常以自行车大小齿轮与后轮、皮带传送装置或变速箱为背景设问。常见问法：
- 已知同一轴上两轮半径，求线速度、向心加速度之比。
- 给定多级传动系统，求某点的线速度、角速度或周期。
- 结合摩擦力或功率，计算匀速转动时某点的受力或力矩。

## ⚠️ 高频易错点

- 将同轴转动与皮带传动（或齿轮啮合）的规律混淆：同轴转动是 \(\omega\) 相同，皮带传动（不打滑）是接触点线速度 \(v\) 相同，切忌在分析时张冠李戴。
- 误认为同轴转动的各点线速度相同：实际上由 \(v = \omega r\) 知，半径不同的点线速度不同，半径越大线速度越大。
- 在传动组合问题中，没能正确划分“同轴部分”与“皮带部分”，导致角速度、线速度相等关系写错。
- 向心加速度公式用错：已知角速度时用 \(a_n = \omega^2 r\)，已知线速度时用 \(a_n = v^2/r\)，如果先错误地认为线速度相等，再用 \(a_n = v^2/r\) 比较就会得出相反结论。
