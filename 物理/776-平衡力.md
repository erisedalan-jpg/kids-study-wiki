# 平衡力

> **一句话**：作用在同一个物体上、使物体保持静止或匀速直线运动状态（即加速度为零）的合力为零的几个力。
> **English**: Forces acting on the same object that produce zero net force, keeping the object at rest or in uniform linear motion.

## 🎓 考点精讲

**定义/原理**：平衡力指作用在同一物体上，合力为零的一组力。其严格判据分为：
- **二力平衡条件**：两个力必须大小相等（$| \vec{F}_1 | = | \vec{F}_2 |$）、方向相反（$\vec{F}_1 = -\vec{F}_2$）、作用在同一直线上、且作用在同一物体上。
- **多力平衡条件**：物体在多个力作用下处于平衡状态时，合外力为零（$\sum \vec{F} = 0$）。在正交分解法中，即同时满足 $\sum F_x = 0$ 和 $\sum F_y = 0$（若为空间问题还需 $\sum F_z = 0$）。
- **力矩平衡（拓展）**：对于涉及转动的物体，还需满足合力矩为零（$\sum M = 0$），高考多在实验题或压轴题中涉及。

**关键方法**：解决平衡力问题的标准步骤是“受力分析—建立坐标系—列平衡方程”。
1.  **确定研究对象**（整体法/隔离法），画出受力示意图，只画物体实际受到的力，不画分力或作用力反作用力中的物体施加的力。
2.  **建立直角坐标系**，优先让尽可能多的力落在坐标轴方向上。
3.  **将不在轴上的力正交分解**。
4.  **列平衡方程** $\sum F_x = 0$，$\sum F_y = 0$，解方程求未知量。对于三力平衡问题，常优先用合成法（构建封闭矢量三角形）或拉密定理快速求解。

**典型应用**：高考常以“物体静止于粗糙斜面”、“三绳结点悬挂重物”、“动态平衡分析”等形式考查。
- **静态平衡**：如粗糙水平面上用斜向上拉力拉不动物体，需列出水平方向 $F\cos\theta - f = 0$，竖直方向 $F\sin\theta + N - mg = 0$，结合 $f \le \mu N$ 判断平衡条件。
- **动态平衡**：如挡板缓慢转动使球受力的变化，常利用“图解法”分析动态矢量三角形，或使用“相似三角形法”处理力三角形与几何三角形相似的问题。

## ⚠️ 高频易错点

- **混淆平衡力与相互作用力**：平衡力作用在**同一物体**上；相互作用力（牛顿第三定律）作用在**两个不同物体**上，且必为同种性质的力。例如书对桌面的压力与桌面对书的支持力是相互作用力，不是平衡力。
- **误判平衡状态**：物体速度为零的瞬间不一定是平衡状态（如竖直上抛最高点速度为零但受力不平衡），平衡状态的判据是 **加速度（合外力）是否为零**，不是速度是否为零。
- **受力分析时多力或漏力**：常错误地添上“下滑力”“惯性力”或漏掉被研究对象施加给环境的力（干扰平衡力判断）。严格按照重力、弹力、摩擦力的顺序分析，弹力方向垂直于接触面，摩擦力方向平行于接触面且与相对运动趋势方向相反。
- **三角函数转换错误**：在斜面上分解重力时，混淆 $\sin\theta$ 与 $\cos\theta$ 对应的边长关系。口诀：重力沿斜面分量为 $mg\sin\theta$，垂直斜面分量为 $mg\cos\theta$（$\theta$ 为斜面倾角）。
