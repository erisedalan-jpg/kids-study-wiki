# 交变电流

> **一句话**：大小和方向都随时间做周期性变化的电流，称为交变电流，简称交流（AC）。
> **English**: An alternating current (AC) is an electric current that periodically reverses direction and changes its magnitude continuously with time.

## 🎓 考点精讲

**定义与原理**  
方向随时间周期性变化的电流为交变电流，最简模型是正弦式交变电流，其瞬时值表达式为：  
$$ e = E_m \sin \omega t,\quad i = I_m \sin \omega t,\quad u = U_m \sin \omega t $$  
其中，最大值 $E_m = NBS\omega$（$N$ 为线圈匝数、$B$ 为磁感应强度、$S$ 为线圈面积、$\omega$ 为角速度）。该式适用于匀强磁场中线圈绕垂直于磁场的轴匀速转动的情形。描述交流的物理量有四值：  
- **瞬时值**：对应某一时刻的值；  
- **最大值（峰值）**：$E_m$、$I_m$、$U_m$；  
- **有效值**：根据电流的热效应定义，让交变电流和恒定电流通过相同电阻，在相同时间内产生相同热量，恒定电流的值即交流的有效值。正弦式交流的有效值与最大值关系为 $I = \frac{I_m}{\sqrt{2}}$，$U = \frac{U_m}{\sqrt{2}}$；  
- **平均值**：$\overline{E} = N\frac{\Delta\Phi}{\Delta t}$，用于计算通过导体某截面的电荷量 $q = \overline{I}\Delta t$。

**关键方法**  
解决交变电流问题的核心是“四值”的选取：
- 计算电热、电功率、电表示数等用**有效值**；
- 计算电荷量用**平均值**；
- 考虑电容器耐压、二极管的击穿电压等用**最大值**；
- 分析某一瞬间的受力或起、止状态用**瞬时值**。  
变压器问题抓住理想变压器条件：$U_1/U_2 = n_1/n_2$，$P_{\text{入}} = P_{\text{出}}$，副线圈只有一个时 $I_1/I_2 = n_2/n_1$；动态分析采用“串反并同”或“由负载到电源”的反推法。远距离输电的核心关系是 $P_{\text{损}} = I_{\text{线}}^2 R_{\text{线}}$ 和升压降压变压器的组合。

**典型应用**  
高考通常以以下形式考查：
1. **交变电流的产生与图像**：给出线圈在磁场中转动情境，判断或画出 $e-t$、$i-t$ 图像，并计算四值。
2. **有效值的计算**：常结合非正弦式交变电流（如矩形波、锯齿波）根据热效应相等列式求解。
3. **变压器动态分析**：改变原副线圈匝数、负载电阻，分析各电表读数变化。
4. **远距离输电**：给出输电模型，计算输电线上损失的电压和功率，以及升、降压变压器匝数比。

## ⚠️ 高频易错点

- **有效值的误用**：非正弦式交变电流（如矩形脉冲）不能直接用 $I_m/\sqrt{2}$，必须依据热效应等效列方程计算有效值。
- **理想变压器电流比的条件**：只有一个副线圈时才有电流与匝数成反比，若有多组副线圈，必须用 $I_1 n_1 = I_2 n_2 + I_3 n_3 + \cdots$ 从功率相等导出。
- **平均值与有效值的混淆**：求电荷量必须用平均值（$\overline{E}=N \Delta\Phi/\Delta t$），求热量/功率必须用有效值，不可交叉使用。
- **正弦式交流产生条件的误解**：磁通量最大时（线圈平面与磁感线垂直）感应电动势最小（零），磁通量为零时（线圈平面与磁感线平行）电动势最大，学生常搞反中性面和峰值面的对应关系。
