# 测定电源电动势和内阻

> **一句话**：基于闭合电路欧姆定律，通过测量路端电压与干路电流的多组数据，利用图像法或方程组法求出电源的电动势 $E$ 和内阻 $r$。
> **English**: An experiment based on the closed‑circuit Ohm’s law, which determines the electromotive force $E$ and internal resistance $r$ of a power source by measuring multiple sets of terminal voltage and current, then processing the data via image or equation methods.

## 🎓 考点精讲

**定义/原理**  
根据闭合电路欧姆定律 $E = U + Ir$（式中 $U$ 为路端电压，$I$ 为干路电流），当外电路电阻变化时，$U$ 与 $I$ 满足线性关系 $U = E - Ir$。测出若干组 $(U, I)$，即可通过图像（$U\!-\!I$ 图线）或解方程组求得 $E$ 和 $r$。图像法中，图线与纵轴截距为电动势 $E$，斜率的绝对值为内阻 $r$（斜率 $k = -r$）。

**关键方法**  
1. **伏安法**（标准电路）：电流表相对电源外接（电流表测的是干路电流），电压表测路端电压。闭合开关前应将滑动变阻器阻值调到最大。  
2. **安阻法**：由 $E = I(R+r)$ 变形得 $\frac{1}{I} = \frac{1}{E}R + \frac{r}{E}$，作出 $\frac{1}{I}\!-\!R$ 图像，斜率与截距联立求解。  
3. **伏阻法**：由 $E = U + \frac{U}{R}r$ 变形得 $\frac{1}{U} = \frac{r}{E}\cdot\frac{1}{R} + \frac{1}{E}$，作出 $\frac{1}{U}\!-\!\frac{1}{R}$ 图像求解。  
处理数据时优先使用图像法，可减小偶然误差并方便剔除异常点。

**典型应用**  
- 高考实验题直接要求连接电路、读数、描点作图、从 $U\!-\!I$ 图像中读取 $E$ 和 $r$。  
- 常见设问：分析电流表内接/外接带来的系统误差（伏安法中电流表外接时，$E_{\text{测}} = E_{\text{真}}$，$r_{\text{测}} < r_{\text{真}}$；电流表内接时，$E_{\text{测}} < E_{\text{真}}$，$r_{\text{测}} > r_{\text{真}}$）。  
- 图像法求内阻时，若纵轴起点不为零，则斜率绝对值仍为 $r$，但截距读数需结合坐标计算。

## ⚠️ 高频易错点

- **混淆 $U\!-\!I$ 图线截距与斜率的意义**：误以为图线与横轴交点电流为短路电流而直接用它计算内阻。实际应取纵轴截距为 $E$，斜率绝对值 $|\Delta U/\Delta I|$ 为 $r$。
- **电流表内外接的系统误差分析不清**：伏安法标准电路（电流表外接）下，电压表分流导致电流测量值偏小，使得 $E$ 测量值无系统误差（电压表内阻极大时近似），但内阻测量值等于电源内阻与电压表内阻的并联值，故偏小。
- **安阻法与伏阻法图像坐标变换错误**：未将公式变形为线性关系，直接作 $I\!-\!R$ 或 $U\!-\!R$ 图像，导致无法得出直线，从而不能正确求解 $E$、$r$。
- **实验操作细节**：闭合开关前未将电阻箱置于最大值（安阻法）或未断开开关长时间通电，造成电池内阻发生变化。
