# 电学实验

> **一句话**：高考物理中以电路为核心，综合考查仪器读数、电路设计、数据处理与误差分析的实验题型。
> **English**: A category of exam questions centered on circuits, testing instrument reading, circuit design, data processing, and error analysis.

## 🎓 考点精讲

**定义/原理**：电学实验以闭合电路欧姆定律 `E = U + Ir`、部分电路欧姆定律 `I = U/R`、串并联规律、电功率 `P = UI = I²R` 等为基础，要求灵活选择电流表内外接法、滑动变阻器的限流与分压接法，并进行实物连线与数据分析。

**关键方法**：
- **仪器选择与量程匹配**：依据待测电阻、电源电动势及电表内阻，按照安全、精确、方便的原则确定电流表、电压表及滑动变阻器的规格。
- **电路设计**：比较 `√(R_A * R_V)` 与待测电阻 `R_x` 的关系，选择电流表内接法（`R_x` 较大）或外接法（`R_x` 较小）；依据是否要求电压电流从零开始、待测电阻与滑动变阻器阻值的相对大小，选择分压式或限流式接法。
- **数据处理**：核心为图像法，如 `U-I` 图像纵轴截距为电动势 `E`，斜率绝对值为内阻 `r`；`I-U` 图像斜率为电阻的倒数；测定金属电阻率实验中利用 `R = ρl/S` 逐差法求平均值等。必须掌握读数规则（如螺旋测微器、游标卡尺的精确与估读）及有效数字运算。

**典型应用**：高考常以“测定金属的电阻率”“描绘小灯泡的伏安特性曲线”“测定电源的电动势和内阻”“练习使用多用电表”等实验为核心，设问角度包括器材选择、电路图设计、实物连线、根据图像求物理量、分析误差原因及减小措施等。例如在测电源电动势和内阻的实验中，若采用电流表外接法（相对电源），则测得的 `E` 偏小、`r` 偏小，需能从原理上进行误差分析。

## ⚠️ 高频易错点

- 电流表内外接法选择机械套用口诀，而未比较 `R_x` 与 `√(R_A * R_V)` 的大小关系，导致方法选反。
- 分压式接法中滑动变阻器初始位置设错（应使待测电路分得的电压最小），或在实物连线时混淆滑动变阻器的接线柱。
- 电表读数时忽略分度值与估读规则，如最小分度为“1”时要估读到下一位，为“2”或“5”时则按1/2或1/5估读，造成有效数字错误。
- 处理 `U-I` 图像时，错误地将横纵轴截距与 `E`、`U_短` 直接等同，未考虑坐标原点是否为零；或将图线斜率直接当作内阻而未取绝对值。
