# 单位制

> **一句话**：单位制是由一组选定的基本单位及其导出单位构成、用以定量表示物理量的完整规范体系；国际单位制（SI）是当前物理学与高考的基准单位制。
> **English**: A system of units is a complete set of base units and derived units used to express physical quantities quantitatively; the International System of Units (SI) is the standard system in physics and the Gaokao.

## 🎓 考点精讲

**定义与原理**
单位制规定了基本量与基本单位，其他物理量通过物理定义和方程式以代数形式导出。国际单位制（SI）包含七个基本量及其单位：
- 长度 $l$ — 米（m）
- 质量 $m$ — 千克（kg）
- 时间 $t$ — 秒（s）
- 电流 $I$ — 安培（A）
- 热力学温度 $T$ — 开尔文（K）
- 物质的量 $n$ — 摩尔（mol）
- 发光强度 $I_v$ — 坎德拉（cd）

导出单位由基本单位通过物理关系式定义，例如力 $F = ma$，导出单位牛顿（N）用基本单位表示为 $1\,\text{N} = 1\,\text{kg}\cdot\text{m}/\text{s}^2$。量纲式用来表示导出量与基本量之间的指数关系，如速度 $[v] = \text{L}\text{T}^{-1}$。

**关键方法**
高考中解决单位制相关问题的标准方法为：
1. **单位关系判断**：将物理量用基本物理量（长度 L、质量 M、时间 T、电流 I 等）表示，然后替换为基本单位比较等式两边的单位是否一致。
2. **量纲分析法**：对不熟悉的物理公式，写出各量的量纲式，验证等号两边量纲是否相同。未知物理量的单位可由已知量纲求出。
3. **单位换算**：利用 $1\,\text{km} = 10^3\,\text{m}$，$1\,\text{h} = 3600\,\text{s}$ 等关系，将复合单位逐步换算为目标单位。注意瞬时速度单位、密度单位等复合单位的换算。

**典型应用**
- **选择题**：给出一个物理关系式（如某力 $F = \frac{1}{2} \rho v^2 A$），要求判断某物理量的单位或用基本单位表示导出单位。解法：右边 $\rho$ 的单位 $\text{kg}/\text{m}^3$，$v^2$ 的单位 $\text{m}^2/\text{s}^2$，$A$ 的单位 $\text{m}^2$，乘积得 $\text{kg}\cdot\text{m}/\text{s}^2$，即 N。
- **实验题或计算题**：在力学、电学综合题中隐含单位换算，例如已知速度 $v=72\,\text{km/h}$，须先换算为 $20\,\text{m/s}$ 再代入公式。

## ⚠️ 高频易错点

- **混淆基本单位与导出单位**：误认为“牛顿”“焦耳”等是基本单位。在SI中，力学部分只有 m、kg、s 三个基本单位，其他均为导出单位。
- **忘记单位统一**：在代入物理公式计算时，未将所有物理量统一为国际单位制，如时间用分钟、长度用厘米，导致结果数量级错误甚至全错。
- **量纲与单位混为一谈**：量纲表示物理量的性质（如 $[v]=\text{LT}^{-1}$），不带具体尺度；单位是量纲的具体标度（如 m/s、km/h）。答题时混淆二者表述。
- **单位换算中幂次遗漏**：面积 $1\,\text{m}^2 = 10^4\,\text{cm}^2$（非 $10^2$），体积 $1\,\text{m}^3 = 10^6\,\text{cm}^3$（非 $10^3$），密度换算同时涉及分子和分母的幂次，极易出错。
