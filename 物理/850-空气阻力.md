# 空气阻力

> **一句话**：空气阻力是物体相对于空气运动时，空气对物体施加的、方向与相对速度方向相反的力。
> **English**: Air resistance is the force exerted by air on an object moving relative to it, acting in the opposite direction to the relative velocity.

## 🎓 考点精讲

- **定义/原理**  
  空气阻力 $F_d$ 的大小一般可表示为：  
  $$F_d = \frac{1}{2} C \rho A v^2$$  
  其中 $C$ 是无量纲阻力系数，$\rho$ 是空气密度，$A$ 是物体迎风横截面积，$v$ 是物体相对空气的速率（待家长核对）。在高考范围内，常以两种简化形式出现：①与速度成正比 $f = kv$（适用于低速、小尺寸物体）；②与速度平方成正比 $f = kv^2$（适用于较大速度或人体尺度）。当空气阻力不可忽略时，物体的运动方程变为 $mg - f = ma$（下落）或 $-f = ma$（仅水平运动），属于变力作用下的动力学问题。

- **关键方法**  
  1. **受力分析**：明确空气阻力方向始终与相对速度方向相反。  
  2. **列牛顿第二定律**：竖直下落时常写作 $mg - kv = m\frac{dv}{dt}$ 或 $mg - kv^2 = m\frac{dv}{dt}$，需根据题意选择线性或平方模型。  
  3. **终极速度**：当合力为零（$a=0$）时，物体达到终极速度 $v_t$。对 $f=kv$ 有 $v_t = mg/k$；对 $f=kv^2$ 有 $v_t = \sqrt{mg/k}$。  
  4. **图像分析**：通过 $v-t$ 图描述速度从零增至终极速度的过程，斜率逐渐减小。  
  5. **近似处理**：有时题目会给出“阻力远小于重力”或“忽略空气阻力”的条件，需准确识别。

- **典型应用**  
  高考常在以下情境设问：  
  - 雨滴从高空下落，先加速后趋于匀速，要求计算终极速度或画出 $v-t$ 图，并比较不同半径雨滴的终极速度。  
  - 跳伞运动员在开伞前后阻力系数突变，要求分析两段运动、计算临界速度或绘制加速度‑时间图像。  
  - 在水平方向上，如汽车、自行车行驶时空气阻力影响恒定功率下的最大速度 $P = Fv$，此时阻力 $f = kv^2$ 导致 $v_{\text{max}} = \sqrt{P/k}$。  
  解题关键：正确写出合力表达式，利用 $a=0$ 求终极速度，结合运动学或能量关系求解位移、时间等问题。

## ⚠️ 高频易错点

- **混淆空气阻力方向**：错误地认为阻力总是向上（竖直下落时方向确实竖直向上，但水平运动时阻力方向与水平速度反向），必须牢记阻力方向与相对速度方向相反。  
- **把空气阻力当成恒力**：直接用匀变速直线运动公式处理变力问题，如用 $v = gt$ 计算有阻力的下落速度，忽略阻力随速度增大而增大。  
- **不顾条件乱用线性或平方模型**：题目未明确时错选 $f=kv$ 或 $f=kv^2$；需根据题设信息（如“阻力与速率成正比”或“阻力与速率的平方成正比”）代入。  
- **终极速度计算时丢失系数**：当阻力写为 $f = kv^2$ 时，由 $mg = kv^2$ 得 $v = \sqrt{mg/k}$，但若题中系数用 $k$ 表示 $f = kv^2$ 则正确；若用 $f = \frac12 C\rho A v^2$，则 $v_t = \sqrt{\frac{2mg}{C\rho A}}$，容易遗漏因子2。  
- **图像错误**：画 $v-t$ 图时，误将终极速度后的图线画为继续上升或下降，而实际应趋于水平渐近线。
