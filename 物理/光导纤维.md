## 🎓 光导纤维

### 定义

**光导纤维**（optical fiber，简称光纤）是利用**全反射**原理，使光在细玻璃纤维或塑料纤维中沿弯曲路径传播的光学元件。

### 基本结构

光纤由同心圆柱层构成：

| 层 | 名称 | 折射率 | 作用 |
|----|------|--------|------|
| 内层 | 纤芯（core） | 较大（$n_1$） | 光线传播通道 |
| 外层 | 包层（cladding） | 较小（$n_2$） | 提供全反射界面 |

$n_1 > n_2$，确保光从纤芯射向包层时满足全反射条件。

### 工作原理

光线以大于临界角的角度射入纤芯：

1. 光射到纤芯-包层界面（$n_1 > n_2$）
2. 入射角 > 临界角 $C$（$\sin C = n_2/n_1$）
3. 发生**全反射**，光线被完全反射回纤芯
4. 在纤芯中不断全反射，沿弯曲路径传播，几乎无损耗

$$\sin C = \frac{n_2}{n_1} < 1 \implies \text{全反射条件满足}$$

### 光纤的应用

| 应用领域 | 具体用途 |
|----------|---------|
| **通信** | 光纤通信（互联网骨干网）：传输速率高、损耗小、抗干扰 |
| **医学** | 内窥镜（胃镜、肠镜）：将图像导出体外，无需手术 |
| **工业** | 光纤传感器：测温、测压 |
| **装饰** | 光纤灯：将光导引到任意位置 |

### 光纤通信的优势

| 对比项 | 电缆通信 | 光纤通信 |
|--------|---------|---------|
| 传输容量 | 有限 | 极大（载波频率高） |
| 损耗 | 较大 | 极小 |
| 抗干扰 | 受电磁干扰 | 不受电磁干扰 |
| 保密性 | 较差 | 较好 |
| 材料 | 铜 | 玻璃/塑料（廉价） |

### 典型例题

**例**：光纤纤芯折射率 $n_1 = 1.5$，包层折射率 $n_2 = 1.4$，求全反射临界角。

$$\sin C = \frac{n_2}{n_1} = \frac{1.4}{1.5} = 0.933$$

$$C = \arcsin(0.933) \approx 69°$$

只要光线在纤芯中以大于 69° 的角度（与法线的夹角，即接近平行于纤芯轴线方向）射到界面，就发生全反射。

### 易错点

1. **光纤利用全反射，不是折射**：光在光纤中传播靠全反射，不是折射引导。
2. **纤芯折射率必须大于包层**：方向不能反，否则不能全反射。
3. **光纤弯曲有限制**：过度弯曲会使光线入射角小于临界角，导致光泄露。
