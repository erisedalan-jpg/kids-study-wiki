# 力的合成

> **一句话**：多个力共同作用时，可以用一个等效的"合力"代替，合力的求法取决于各力的方向关系。
> **English**: Multiple forces acting together can be replaced by a single equivalent "resultant force," found using vector addition.

---

## 📚 给 6-12 岁（自读版）

两个人一起推箱子——两个力合在一起，效果等同于一个更大的力。这个"合在一起的力"就叫**合力**。

**同方向的力相加：**

两人都向右推，各 30 N → 合力 = 30 + 30 = 60 N（向右）

**反方向的力相减：**

一人向右推 50 N，另一人向左推 20 N → 合力 = 50 - 20 = 30 N（向右，取大的方向）

**互相垂直的力（直角）：**

用勾股定理：$F = \sqrt{F_1^2 + F_2^2}$

**一般情况（任意角度）：**

用平行四边形法则（12岁以上学习）

---

## 🎓 给 12+（进阶版）

### 合力的定义

**合力（Resultant Force）：** 与一组力产生相同效果的单一等效力。

求合力的方法根据力的方向关系不同而不同。

### 同线力的合成

**同向：** $F_{合} = F_1 + F_2$（方向与各力相同）

**反向：** $F_{合} = |F_1 - F_2|$（方向与较大力相同）

### 平行四边形法则（初中定性了解）

两个不同方向的力 $\vec{F_1}$ 和 $\vec{F_2}$，以它们为邻边作平行四边形，对角线即为合力 $\vec{F}$。

$$|\vec{F}| = \sqrt{F_1^2 + F_2^2 + 2F_1 F_2 \cos\theta}$$

（$\theta$ 为两力夹角）

**特殊情况：**
- $\theta = 0°$（同向）：$F = F_1 + F_2$（最大值）
- $\theta = 90°$（垂直）：$F = \sqrt{F_1^2 + F_2^2}$
- $\theta = 180°$（反向）：$F = |F_1 - F_2|$（最小值）

**范围：** $|F_1 - F_2| \leq F \leq F_1 + F_2$

### 合力方向

合力方向由向量加法确定，可用三角函数计算：

$$\tan\alpha = \frac{F_1 \sin\theta}{F_2 + F_1 \cos\theta}$$

（$\alpha$ 为合力与 $F_2$ 的夹角）

### 典型例题

**例1：** 一个物体受到水平向右 30 N 和竖直向下 40 N 的力，合力大小和方向？

解：$F = \sqrt{30^2 + 40^2} = \sqrt{900 + 1600} = \sqrt{2500} = 50\text{ N}$

方向：与水平方向夹角 $\tan\alpha = 40/30 = 4/3$，$\alpha \approx 53°$（斜向右下方）

**例2：** 两个大小均为 $F$ 的力，夹角为 120°，合力是多少？

解：$F_{合} = \sqrt{F^2 + F^2 + 2F^2\cos 120°} = \sqrt{2F^2 + 2F^2 \times (-0.5)} = \sqrt{F^2} = F$

合力大小等于单个力的大小（特殊结果）。

**例3：** 合力范围：两个力大小分别为 5 N 和 3 N，合力可能是 9 N 吗？2 N 呢？4 N 呢？

解：合力范围：$|5-3| = 2\text{ N}$ 到 $5+3 = 8\text{ N}$

- 9 N：超出范围，不可能
- 2 N：等于最小值，可能（反向时）
- 4 N：在范围内，可能

### 易错点

- 合力方向不一定是两力中间的角平分线方向。
- 合力大小不一定大于分力（如两力反向时合力小）。
- 合力范围：$|F_1 - F_2| \leq F_{合} \leq F_1 + F_2$（初中很少考，但概念要清楚）。
