# 匀变速直线运动规律

> **一句话**：沿着一条直线且加速度不变的运动，其速度与位移随时间均匀变化的关系。
> **English**: Motion along a straight line with constant acceleration, where velocity and displacement change uniformly with time.

## 🎓 考点精讲

- **定义与基本公式**
  匀变速直线运动指加速度恒定（大小、方向均不变）的直线运动。取物体运动直线为坐标轴，通常以初速度方向为正方向，则加速度$a$为定值。核心公式：
  $$v = v_0 + at$$
  $$x = v_0 t + \frac{1}{2} a t^2$$
  $$v^2 - v_0^2 = 2ax$$
  $$v_{\text{平均}} = \frac{v_0 + v}{2} = \frac{x}{t}\quad (\text{仅适用于匀变速})$$
  其中$v_0$为初速度，$v$为$t$时刻末速度，$x$为位移，$a$为加速度。位移为矢量，正负由初速度方向和加速度方向共同决定。

- **关键方法与解题步骤**
  1. **选定正方向**：一般取初速度方向为正，则加速度与正方向同向取正、反向取负。
  2. **画过程草图**：标出已知量（$v_0, a, t, x, v$）和待求量。
  3. **优选公式**：无$t$用$v^2 - v_0^2 = 2ax$；无$a$用$x = \frac{v_0+v}{2}t$；无$x$用$v = v_0+at$；无$v$用$x = v_0t + \frac{1}{2}at^2$。
  4. **多过程问题**：分段分析，抓住连接点的速度。对于“刹车”类问题，先求停车时间$t_0 = \frac{0-v_0}{a}$，判断运动是否已停止。
  5. **实验延伸**：纸带处理常用逐差法求加速度$a = \frac{(x_4+x_5+x_6)-(x_1+x_2+x_3)}{9T^2}$，以及用中间时刻速度等于平均速度求瞬时速度。

- **典型高考设问与场景**
  - **刹车/减速为零**：已知初速、加速度（负值），问刹车距离或时间。务必先算停车时间，避免直接将时间代入公式导致错误。
  - **自由落体与竖直上抛**：自由落体是$v_0=0, a=g$的特例；竖直上抛可全程视为$a=-g$的匀变速直线运动，上升和下降过程对称。
  - **追及与相遇**：利用位移关系和时间关系，联立运动学方程判断能否追上或求最大/最小距离。
  - **多物体或多段运动**：如传送带上的匀加速、滑块在斜面上的匀减速，常结合牛顿第二定律先求加速度，再利用匀变速规律求解。

## ⚠️ 高频易错点

- **矢量方向处理不当**：未规定正方向就直接代数值，导致加速度、速度、位移符号混乱。解题时必须先明确正方向并保持统一。
- **刹车陷阱**：将题目所给的时间直接代入$x = v_0t + \frac{1}{2}at^2$，而实际物体早已停止，加速度$0$。必须先用$t_0 = \frac{-v_0}{a}$判断有效运动时间。
- **平均速度公式滥用**：$\bar{v} = \frac{v_0+v}{2}$仅适用于匀变速直线运动，非匀变速运动不可用。
- **位移与路程混淆**：在往返运动中（如竖直上抛），位移大小可能小于路程，盲目用公式算出路程错误。
