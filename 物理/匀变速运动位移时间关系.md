# 匀变速运动位移时间关系

> **一句话**：描述匀变速直线运动中位移随时间变化的矢量关系式 `x = v₀t + ½at²`，是运动学三大公式之一。
> **English**: The displacement-time relation for uniformly accelerated rectilinear motion is given by `x = v₀t + ½at²`, expressing displacement as a function of time.

## 🎓 考点精讲

**定义/原理**  
对于初速度为 `v₀`、加速度为 `a` 的匀变速直线运动，若以初位置为坐标原点，则物体在任意时刻 `t` 的位移 `x`（矢量）满足：  
`x = v₀ t + ½ a t²`  
该公式由速度公式 `v = v₀ + a t` 对时间积分导出，也可由 v-t 图下的梯形面积推得。注意位移是矢量，代入时要规定正方向，与正方向一致的矢量取正值，相反取负值。

**关键方法**  
- **分段与全程法**：涉及多段过程（如刹车、折返）时，先分阶段求位移再合成。  
- **公式选择**：已知 `v₀`、`a`、`t` 求 `x` 直接用位移-时间公式；若缺少时间，可优先选用 `v² - v₀² = 2ax`；若缺少加速度，可用平均速度法 `x = (v₀+v)t/2`。  
- **刹车类问题陷阱**：先判断速度减为零所需时间 `t₀ = v₀/|a|`，避免将 `t > t₀` 直接代入公式，导致位移“回头”的错误结果。

**典型应用**  
高考常见设问：  
1. **已知初速度、加速度和时间求位移**（如汽车加速、滑块沿斜面下滑），直接代入公式即可，注意加速度正负。  
2. **跑道、反应距离问题**：结合匀速与匀加速阶段，分段累加位移，再与给定距离比较。  
3. **追及相遇问题**：两物体的位移关系常列成 `xₐ = x_B + x₀`，其中各自位移用 `x = v₀t + ½at²` 展开，解一元二次方程并检验根的合理性。

## ⚠️ 高频易错点

- **矢量方向处理不当**：忘记规定正方向，导致加速度、初速度、位移的正负号混乱，尤其是竖直上抛、斜面上滑等情景。  
- **刹车类超时代入**：将实际运动时间直接代入公式，忽略了物体在刹车停止后不再反向运动，公式只在匀变速过程中成立。  
- **混淆位置、位移与路程**：公式给出的是位移，而不是路程。在折返运动中，位移可能为零而路程不为零，计算路程需分段求和。  
- **误用平均速度公式**：`x = (v₀+v)t/2` 仅对匀变速直线运动成立，若加速度变化（如变加速、碰撞过程）则不能用。
