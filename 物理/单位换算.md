# 单位换算

> **一句话**：单位换算是将同一物理量的数值从一种单位表示变换为另一种单位表示的运算，核心是建立不同单位间的等价比例关系。
> **English**: Unit conversion is the process of converting the numerical value of a physical quantity from one unit of measurement to another, based on established equivalence ratios between units.

## 🎓 考点精讲

**定义/原理**  
单位换算的根据是物理量的量值不变：$n_1 [U_1] = n_2 [U_2]$，其中 $n$ 为数值，$[U]$ 为单位。若两单位关系为 $1\ \text{ U}_2 = k\ \text{ U}_1$，则 $n_2 = n_1 / k$。
在高考物理中，所有运算都要求使用国际单位制（SI）。力学选基本量（长度、质量、时间），导出单位如 $1\ \text{N} = 1\ \text{kg·m/s}^2$；电学中 $1\ \text{V} = 1\ \text{J/C} = 1\ \text{kg·m}^2/(\text{A·s}^3)$。复合单位的换算必须对每个组成单位分别换算，例如速度 $1\ \text{km/h} = \frac{1000\ \text{m}}{3600\ \text{s}} = \frac{5}{18}\ \text{m/s}$。

**关键方法**  
1. **量纲检验法**：利用物理量量纲判定表达式正误。如匀变速位移公式必须是$[L] = [L/T^2]\cdot[T^2]$，量纲匹配才可能正确。  
2. **统一单位再计算**：高考计算题中，各物理量必须统一为SI单位（m、kg、s、A、K、mol、cd）。若题目中给出数据单位不统一，第一步即换算成标准单位。  
3. **比例换算法**：熟记常见换算因子，如 $1\ \text{m/s} = 3.6\ \text{km/h}$，$1\ \text{g/cm}^3 = 1000\ \text{kg/m}^3$，$1\ \text{kWh} = 3.6\times10^6\ \text{J}$。写出换算关系，再约分数字，避免盲目乘除。

**典型应用**  
- **选择题常见设问**：给出几个用基本单位表示的导出单位表达式，判断哪一个是力的单位（$\text{kg·m/s}^2$）、功的单位（$\text{kg·m}^2/\text{s}^2$）等。  
- **计算题隐性考查**：如电源电动势 $E=12\ \text{V}$，内阻 $r=0.5\ \Omega$，外电阻 $R=10\ \Omega$，计算电流 $I$。看似直接代入，但若题目中某量单位用了“mV”或“kΩ”，必须先换算为V和Ω。典型失分：电路功率 $P = \frac{U^2}{R}$ 中，$U$ 用 “kV” 代入却忘记换算。  
- **电能单位转换**：$1\ \text{kWh} = 3.6\times10^6\ \text{J}$，常用于电功与热功换算。

## ⚠️ 高频易错点

- **速度换算方向错误**：不少学生错误记为 $1\ \text{m/s} = 0.28\ \text{km/h}$，实际应为 $1\ \text{m/s} = 3.6\ \text{km/h}$，混淆了乘除关系。  
- **面积、体积单位幂次忽略**：$1\ \text{cm}^2 = (10^{-2}\ \text{m})^2 = 10^{-4}\ \text{m}^2$，而不是 $10^{-2}\ \text{m}^2$；$1\ \text{L} = 1\ \text{dm}^3 = 10^{-3}\ \text{m}^3$。  
- **密度单位换算遗漏整体因子**：$1\ \text{g/cm}^3 = 1000\ \text{kg/m}^3$，有些学生只照顾了质量单位未处理体积，得 $10^{-3}\ \text{kg/m}^3$。  
- **电学单位混用**：如将电容 “μF” 误当作 “10^{-3} F”，正确为 $10^{-6}\ \text{F}$；电感 “mH” 误作 $10^3\ \text{H}$。
