## 🎓 变压器（高中物理）

### 定义

**变压器**是利用电磁感应（互感）原理，改变**交流电压**（同时改变交流电流）的静止电器。

变压器只能改变**交流电**的电压，不能改变直流电压（直流稳定，无磁通量变化，无互感电动势）。

### 基本结构

| 部件 | 作用 |
|------|------|
| 铁芯（硅钢片叠压）| 导磁，减小涡流损耗 |
| 初级线圈（一次侧，$N_1$ 匝）| 接交流电源（输入） |
| 次级线圈（二次侧，$N_2$ 匝）| 接负载（输出） |

### 工作原理

1. 初级线圈通交流电 $U_1$ → 铁芯中产生交变磁通量 $\Phi$
2. 交变磁通量同时穿过次级线圈 → 互感感应出电动势 $U_2$
3. 由于同一铁芯，两侧磁通量变化率相同：

$$\frac{U_1}{U_2} = \frac{N_1}{N_2}$$

### 分类

| 类型 | 条件 | 特点 |
|------|------|------|
| 升压变压器 | $N_2 > N_1$ | $U_2 > U_1$，$I_2 < I_1$ |
| 降压变压器 | $N_2 < N_1$ | $U_2 < U_1$，$I_2 > I_1$ |
| 隔离变压器 | $N_1 = N_2$ | 电压不变，用于电气安全隔离 |

### 实际变压器的损耗

实际变压器有两类损耗：
1. **铜损**：线圈电阻 $R$ 产生的热耗 $P = I^2R$
2. **铁损**：铁芯中的涡流和磁滞损耗

效率 $\eta = P_2/P_1 \times 100\%$，大型电力变压器效率可达 99% 以上。

理想变压器：$\eta = 100\%$（忽略铜损和铁损），此时 $P_1 = P_2$。

### 典型例题

**例（基础）**：变压器 $N_1 = 1000$ 匝，$N_2 = 50$ 匝，接 220 V 交流电，求次级电压。

$$U_2 = U_1\times\frac{N_2}{N_1} = 220\times\frac{50}{1000} = 11\text{ V}$$

**例（多绕组）**：某变压器有三个绕组：$N_1 = 600$、$N_2 = 300$、$N_3 = 60$，$U_1 = 120$ V，求 $U_2$ 和 $U_3$。

$$U_2 = 120\times\frac{300}{600} = 60\text{ V}, \quad U_3 = 120\times\frac{60}{600} = 12\text{ V}$$

### 易错点

1. 变压器**不能改变直流电压**——直流通过初级时，铁芯中磁通量不变，无感应电动势。
2. 升压变压器升高电压，同时**降低电流**（功率守恒），不是升高功率。
3. 变压器的电压比是**有效值**之比，不是峰值之比（两者相等，但注意表述）。
