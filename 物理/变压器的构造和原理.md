# 变压器的构造和原理

> **一句话**：变压器是利用电磁感应原理改变交流电压的装置，由闭合铁芯和绕在铁芯上的原、副线圈构成。
> **English**: A transformer is a device that uses the principle of electromagnetic induction to change the alternating voltage; it consists of a closed iron core with primary and secondary coils wound around it.

## 🎓 考点精讲

- **构造与原理**  
  变压器主要由**闭合铁芯**（导磁）和两个互相绝缘的线圈（**原线圈**、**副线圈**）组成。  
  当原线圈接上交变电压$U_1$时，流过交变电流$I_1$，在铁芯中产生交变的磁通量$\Phi$。这一磁通量同时穿过原、副线圈，因**互感**现象，在原、副线圈中产生同频率的感应电动势$E_1=n_1\frac{\Delta\Phi}{\Delta t}$、$E_2=n_2\frac{\Delta\Phi}{\Delta t}$（$n_1$、$n_2$为匝数）。  

  **理想变压器**（不计线圈电阻、漏磁与铁芯损耗）满足：  
  `$$ \frac{U_1}{U_2} = \frac{n_1}{n_2}, \qquad \frac{I_1}{I_2} = \frac{n_2}{n_1} $$`  
  （$U_1$、$I_1$为原线圈电压、电流；$U_2$、$I_2$为副线圈电压、电流）。且理想变压器输入功率等于输出功率：$U_1 I_1 = U_2 I_2$。

- **关键方法**  
  1. **匝数比决定变压比**：由$\frac{U_1}{U_2} = \frac{n_1}{n_2}$，$n_2 > n_1$为升压，反之为降压。  
  2. **副线圈电流决定原线圈电流**：理想变压器原线圈电流由副线圈电流和负载决定，即$I_1 = \frac{n_2}{n_1} I_2$。当副线圈空载（断路）时，$I_2=0$，原线圈电流极小（只有励磁电流）。  
  3. **动态分析**：通常由副线圈负载变化→$I_2$变化→$I_1$变化；$U_1$由电源决定，$U_2$由$U_1$和匝数比决定不变（理想变压器），再结合欧姆定律分析。  
  4. **有多个副线圈**时，电压关系仍为$\frac{U_1}{n_1} = \frac{U_2}{n_2} = \frac{U_3}{n_3} = \cdots$，但电流关系由功率守恒$U_1 I_1 = U_2 I_2 + U_3 I_3 + \cdots$得出。

- **典型应用**  
  高考常见设问：  
  – 给定原、副线圈匝数比和原线圈电压，求副线圈电压或电流；  
  – 结合负载电阻，计算各线圈电流、功率；  
  – 动态分析：滑动变阻器滑动、开关闭合/断开，判断各电表示数变化；  
  – 含有二极管、电容、电动机等元件的变压器问题，需区分有效值、平均值。  
  **场景示例**：一台理想变压器，原线圈匝数$n_1=1100$，副线圈匝数$n_2=55$，原线圈接有效值$220\,\text{V}$的交流电，副线圈接一阻值$R=10\,\Omega$的电阻。求副线圈电压$U_2 = \frac{n_2}{n_1}U_1 = 11\,\text{V}$，电流$I_2 = 1.1\,\text{A}$，原线圈电流$I_1 = \frac{n_2}{n_1}I_2 = 0.055\,\text{A}$。

## ⚠️ 高频易错点

- **变压器只能改变交流电压**：直流电不能在原线圈产生变化的磁通量，副线圈无感应电动势。误认为变压器可变压直流。
- **原线圈电流由副线圈负载决定**：误以为原线圈电压和匝数比直接决定原线圈电流，而忽略$I_1 = \frac{n_2}{n_1} I_2$，空载时原线圈电流很小。
- **匝数比与电流比反比只适用于一个副线圈**：当有多个副线圈时，$\frac{I_1}{I_2} = \frac{n_2}{n_1}$不适用，必须用功率守恒$U_1 I_1 = U_2 I_2 + U_3 I_3 + \cdots$求解。
- **对交变电流“四值”的误用**：变压器的电压比、电流比使用的是**有效值**。若涉及电容的击穿电压，需用最大值；计算通过导体横截面的电荷量时，需用电流平均值并借助$\frac{\Delta\Phi}{\Delta t}$和匝数，不可直接套功率公式。
