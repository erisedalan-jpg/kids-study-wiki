# 定滑轮

> **一句话**：定滑轮轴固定不动，只改变力的方向，不改变力的大小，相当于等臂杠杆。
> **English**: A fixed pulley has a stationary axle; it changes the direction of force but not its magnitude, equivalent to an equal-arm lever.

---

## 📚 给 6-12 岁（自读版）

旗杆顶上有一个滑轮——升旗时，我们向下拉绳子，旗子却向上升。这个滑轮的轴固定不动，叫**定滑轮**。

**定滑轮的特点：**
- 轴（中心）固定，不随绳子移动
- **改变力的方向**（向下拉 → 绳另一端向上）
- **不省力、不费力**（拉力 = 物体重力）
- 不省距离（绳子拉多长，物体升多高）

**定滑轮 = 等臂杠杆：** 两边的力臂相等（都是滑轮半径），所以力相等，方向相反。

**用途：** 当需要改变力的方向时，用定滑轮。例如：
- 升旗（不可能向上爬着升旗，用定滑轮可以在地面上升旗）
- 起重机（改变用力方向，操作更方便）
- 井中打水（向下拉绳，水桶向上提）

**总结：** 定滑轮省力吗？不省！改变方向吗？改变！

## 🎓 给 12+（进阶版）

### 定滑轮的原理

定滑轮等效于等臂杠杆：
- 支点：滑轮轴（圆心）
- 动力臂 = 阻力臂 = 滑轮半径 $r$
- 平衡条件：$F_1 r = F_2 r$，即 $F_1 = F_2$

### 定滑轮的受力分析

| 量 | 定滑轮（理想） |
|----|------------|
| 拉力大小 | $F = G$（等于重物重力） |
| 拉力方向 | 可以是任意方向（改变方向是定滑轮的功能） |
| 绳子移动距离 | $s = h$（等于重物上升高度） |
| 做的功 | $W = Gh$（不省功） |

### 力的方向改变

绳子方向可以任意：
- 竖直向下拉（最常见）→ 重物竖直向上
- 斜向下拉（有分力，实际拉力方向变化）→ 重物竖直向上

注意：斜向拉时，绳子张力 = $F$（不变），但人实际用力的竖直分量等于重力，水平分量拉向支点（由支点约束力平衡）。

### 定滑轮 vs 动滑轮

| 比较项 | 定滑轮 | 动滑轮 |
|-------|-------|-------|
| 轴是否固定 | 固定 | 移动 |
| 省力 | 否（$F = G$） | 是（$F = G/n$，$n$ 为绳段数） |
| 改变方向 | 是 | 否（方向不变或改变不多） |
| 等效杠杆 | 等臂杠杆 | 省力杠杆 |

### 典型例题

**例1：** 用定滑轮提升重 100 N 的物体，需要多大的拉力？

解：$F = G = 100\text{ N}$（定滑轮不省力）

**例2：** 用定滑轮（忽略摩擦），物体上升 2 m，绳子拉动多少？

解：$s = h = 2\text{ m}$（定滑轮不省距离）

**例3：** 从定滑轮两侧拉绳子，一侧挂 50 N 重物，另一侧施 50 N 拉力，系统平衡吗？

解：理想定滑轮，两侧绳子张力相等，$F_1 = F_2 = 50\text{ N}$，系统平衡。（若有摩擦，则施力需要稍大于 50 N）

### 易错点

- 定滑轮**不省力**：拉力等于重力（理想情况）。
- 定滑轮**改变方向**：这是它的价值所在。
- 实际定滑轮有摩擦，需要的拉力略大于重力。
- 斜向拉绳时，绳子的张力不变，但施力的方向变了（不再方便）。
