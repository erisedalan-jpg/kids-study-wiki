# 小船渡河问题

> **一句话**：质点在两个相互独立的恒速运动（静水船速与水流速度）共同作用下的合运动问题，核心是运动的合成与分解。
> **English**: The problem of a boat crossing a river, involving the vector combination of the boat's velocity in still water and the river current's velocity.

## 🎓 考点精讲

**定义/原理**：小船渡河是基于**运动独立性原理**的典型物理模型。设船在静水中的速度为 $\vec{v}_\text{船}$，水流速度为 $\vec{v}_\text{水}$，则船对地的实际速度（合速度）为 $\vec{v}_\text{合} = \vec{v}_\text{船} + \vec{v}_\text{水}$。
渡河时间仅由垂直于河岸的分运动决定：$t = \frac{d}{v_\text{船} \sin\theta}$（其中 $d$ 为河宽，$\theta$ 为船头与上游河岸的夹角）。渡河位移则由合速度方向决定。

**关键方法**：
1. **最短时间渡河**：令船头始终垂直指向对岸（$\theta = 90^\circ$）。此时渡河时间最短，$t_{\text{min}} = \frac{d}{v_\text{船}}$，与水流速度无关。合位移沿下游偏移。
2. **最短位移渡河**：
   - 若 $v_\text{船} > v_\text{水}$：可调整船头指向上游某角度，使合速度方向垂直对岸。设 $\cos\alpha = \frac{v_\text{水}}{v_\text{船}}$，此时位移最小为 $d$。
   - 若 $v_\text{船} < v_\text{水}$：无论船头方向如何，合速度必向下游偏移。当船速与合速度垂直时位移最短（矢量圆法），此时 $\cos\beta = \frac{v_\text{船}}{v_\text{水}}$，最小位移为 $\frac{v_\text{水}}{v_\text{船}} d$。

**典型应用**：高考常以选择题或计算题形式出现，设问通常为：给定 $v_\text{船}$、$v_\text{水}$ 和 $d$，求解最短渡河时间、最短位移、到达对岸的位置点，或判断船能否垂直渡河。常结合三角函数与矢量图进行考查。

## ⚠️ 高频易错点

- **混淆分运动与合运动的独立性**：错误地认为水流速度会影响渡河时间。实际上，渡河时间只由垂直于河岸的分速度决定，与水流速度无关。
- **误判最短位移的条件**：当 $v_\text{船} < v_\text{水}$ 时，学生常错误认为船可以垂直渡河。此时合速度无法指向正对岸，必须利用矢量圆的极限角来求解最小位移。
- **矢量合成方向画反**：在确定船头指向上游的角度时，应构建 $\vec{v}_\text{合} = \vec{v}_\text{船} + \vec{v}_\text{水}$ 的矢量三角形，错画导致全题失分。
- **忽略实际不达标情况**：若船速为零或方向不合理，渡河无解，但题目通常给出标准物理模型条件，需确认已知量再进行计算。
