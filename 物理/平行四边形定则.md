## 🎓 平行四边形定则（高中物理）

### 定律表述

两个力合成时，以这两个力为邻边作平行四边形，则**对角线代表合力的大小和方向**。

$$|\vec{F}|^2 = F_1^2 + F_2^2 + 2F_1 F_2\cos\theta$$

$$\vec{F} = \vec{F}_1 + \vec{F}_2$$

这是矢量加法的基本运算法则，适用于所有矢量（力、速度、位移、加速度等）。

### 数学推导

设 $\vec{F}_1$ 和 $\vec{F}_2$ 夹角为 $\theta$，平行四边形对角线长度（合力大小）：

由余弦定理：

$$F^2 = F_1^2 + F_2^2 + 2F_1 F_2\cos\theta$$

合力方向（与 $\vec{F}_1$ 夹角 $\alpha$）：

$$\sin\alpha = \frac{F_2\sin\theta}{F}, \quad \tan\alpha = \frac{F_2\sin\theta}{F_1 + F_2\cos\theta}$$

### 特殊情形

| 夹角 $\theta$ | 合力大小 | 合力方向 |
|------------|---------|---------|
| $0°$ | $F_1 + F_2$ | 与两力相同 |
| $90°$ | $\sqrt{F_1^2+F_2^2}$ | $\arctan(F_2/F_1)$ |
| $120°$ | $\sqrt{F_1^2+F_2^2-F_1F_2}$ | 视 $F_1,F_2$ 比例 |
| $180°$ | $|F_1-F_2|$ | 沿大力方向 |
| $F_1=F_2,\theta=60°$ | $F_1$ | 两力夹角平分线 |
| $F_1=F_2,\theta=120°$ | $F_1$ | 两力夹角平分线 |

### 实验验证

**橡皮筋实验**：
1. 将橡皮筋一端固定，另一端用两力 $F_1$、$F_2$ 拉到某点 $P$。
2. 记录两力方向和大小，按平行四边形定则求合力 $F_{\text{理论}}$。
3. 换用一个力 $F_{\text{实验}}$ 将橡皮筋拉到同一点 $P$。
4. 比较 $F_{\text{理论}}$ 和 $F_{\text{实验}}$：大小和方向基本一致，验证定则。

**误差来源**：角度测量不精确、弹簧秤读数误差等。

### 典型例题

**例1**：$F_1=6\text{ N}$，$F_2=8\text{ N}$，夹角90°，求合力。

$$F = \sqrt{6^2+8^2} = \sqrt{100} = 10\text{ N}$$

方向：$\tan\alpha = \frac{8}{6} = \frac{4}{3}$，$\alpha \approx 53°$（与 $F_1$ 夹角）

**例2**：$F_1=F_2=F$，夹角120°，求合力。

$$F_{\text{合}} = \sqrt{F^2+F^2+2F^2\cos120°} = \sqrt{2F^2-F^2} = F$$

合力大小等于每个分力大小，方向沿两力夹角的平分线（等腰三角形）。

**例3**：合力为0的三力条件

三力 $F_1=3\text{ N}$，$F_2=4\text{ N}$，$F_3=5\text{ N}$，可否平衡？

验证：$3^2+4^2=25=5^2$，满足勾股定理，三力可以构成直角三角形，合力为0，**可以平衡**。

### 适用范围

平行四边形定则是**所有矢量加法**的通用法则：
- 力的合成：$\vec{F} = \vec{F}_1 + \vec{F}_2$
- 速度合成：$\vec{v} = \vec{v}_1 + \vec{v}_2$
- 位移合成：$\vec{r} = \vec{r}_1 + \vec{r}_2$
- 加速度合成（非惯性系）

**标量**不满足平行四边形定则（直接代数加）。

### 易错点

1. **合力不是"大小之和"**：$F \neq F_1 + F_2$（除非同向），矢量加法要用平行四边形。
2. **合力大小范围**：$|F_1-F_2| \leq F \leq F_1+F_2$，合力可以比任一分力小。
3. **分解不唯一**：同一力可以分解为不同的一对分力（平行四边形不唯一）。

### 与初中物理衔接

初中只处理同向/反向力（代数加减）。高中引入平行四边形定则，处理任意方向力的合成，是矢量运算的核心工具，贯穿整个高中物理。
