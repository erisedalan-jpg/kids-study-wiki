# 探究加速度与力、质量的关系

> **一句话**：通过控制变量法，用打点计时器与长木板装置定量验证加速度与合外力成正比、与质量成反比（即牛顿第二定律 `F = ma`）。
> **English**: An experiment using the control variable method to quantitatively verify that acceleration is directly proportional to net force and inversely proportional to mass (Newton's Second Law:  `F = ma` ).

## 🎓 考点精讲

**实验原理与方法**
本实验旨在验证牛顿第二定律 `F = ma`，核心方法是**控制变量法**：
1.  **探究  `a` 与  `F` 的关系**：保持小车质量 `M` 不变，改变细线拉力（即砝码与盘的总重力 `mg`），测量小车加速度 `a`。
2.  **探究  `a` 与  `M` 的关系**：保持合外力（砝码与盘的总重力 `mg`）不变，改变小车的质量 `M`，测量小车加速度 `a`。

**数据处理的关键——平衡摩擦力**
实验必须将长木板一端垫高，利用重力沿斜面的分力 `Mg sin θ` 完全抵消滑动摩擦力 `μMg cos θ`。**平衡完成的标志**是：不挂牵引砝码盘，轻推小车后，小车带动纸带能打出**间距均匀的点迹**（即近似匀速直线运动）。改变小车质量后，**无需重新**平衡摩擦力，因方程 `Mg sin θ = μMg cos θ` 两边质量可约去。

**实验条件**
为使细线的拉力 `T` 近似等于砝码及盘的总重力 `mg`，必须满足：**砝码和盘的总质量远小于小车质量**（即 `m ≪ M`）。若不满足，系统加速度实际为 `a = mg / (M+m)`，小车所受真实拉力 `T = Ma = [M/(M+m)] mg < mg`，导致图像向下弯曲。

**数据分析与图像处理**
-   **探究  `a-F`**：建立 `a-F` 坐标系，应为一条过原点的倾斜直线，斜率 `k = 1/M`。
-   **探究  `a-M`**：直接作 `a-M` 图像为双曲线，不便判断。化曲为直的方法是作 **`a - 1/M` 图像**，若图像为过原点的直线，即证明 `a ∝ 1/M`。

**高考常考设问**
1.  **误差分析**：给出 `a-F` 图线不过原点（横截距或纵截距）推断木板倾角过大或未完全平衡摩擦。图像末端向下弯曲判断未满足 `m ≪ M`。
2.  **实验操作排序**：按合理顺序排列“平衡摩擦力、挂盘、释放小车、关闭电源”等步骤。必须强调**先接通打点计时器电源，后释放小车**。
3.  **创新变式**：利用气垫导轨替代长木板（消除摩擦力影响）；或传感器直接测量拉力（无需满足 `m ≪ M`，无需平衡摩擦力——视导轨情况定）。

## ⚠️ 高频易错点

-   **平衡摩擦力的误判**：误以为垫高木板后小车能加速下滑才算平衡；错在挂有砝码盘的情况下判定匀速。
-   **实验条件的混淆**：在利用弹簧测力计直接测拉力的改进实验中，仍强调需要满足 `m ≪ M`（事实上不需要，因为拉力已直接测出）。
-   **图像斜率的物理意义理解错误**：混淆 `a-F` 图斜率（`1/M`）与 `a-M` 或 `a-1/M` 图斜率等，在计算小车质量或重力加速度时带入错误数据。
-   **操作顺序遗忘**：实验完毕停止程序时，常遗漏“先断开纸带电源，再停止运动”的逻辑，或是在纸带数据处理时搞错计数点周期（如未明确每隔4个点取一个计数点则实际周期 `T = 0.1s`）。
