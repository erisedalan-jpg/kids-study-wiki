## 🎓 摩擦因数（高中物理）

### 定义

**摩擦因数**（coefficient of friction，μ）是表征两接触面之间摩擦性质的无量纲常数，反映接触面的粗糙程度。

分为两种：
- **动摩擦因数** $\mu_k$（kinetic friction coefficient）：滑动时适用，$f_k = \mu_k N$
- **静摩擦因数** $\mu_s$（static friction coefficient）：静止时适用，$f_{\max} = \mu_s N$

关系：$\mu_s \geq \mu_k$（静摩擦因数一般略大于动摩擦因数，高中常近似相等）。

### 摩擦因数的特点

- 无量纲（无单位）
- 由**接触面材料**和**粗糙程度**决定，与接触面积、运动速度、压力大小无关
- 通过实验测定

### 常见材料的摩擦因数

| 接触面 | $\mu_k$ 约值 |
|--------|------------|
| 冰与冰 | 0.03 |
| 钢与钢（润滑） | 0.1 |
| 橡胶与混凝土（干燥） | 0.7-0.8 |
| 木与木 | 0.2-0.5 |
| 皮革与铸铁 | 0.3-0.5 |

**规律**：润滑剂（机油、水等）能大幅降低摩擦因数；粗糙面摩擦因数大。

### 实验测定 $\mu_k$（斜面法）

在倾角为 $\theta$ 的斜面上，物体匀速下滑时，合力为零：

$$f_k = G\sin\theta, \quad N = G\cos\theta$$

$$\mu_k = \frac{f_k}{N} = \frac{G\sin\theta}{G\cos\theta} = \tan\theta$$

实验：缓慢增大斜面倾角，直到物体开始匀速下滑，此时 $\mu_k = \tan\theta$。

### 典型例题

**例1**：在水平地面上拖动50kg的箱子，匀速运动所需水平力为150N，求动摩擦因数。

匀速时合力为零：$F = f_k = \mu_k N = \mu_k mg$

$$\mu_k = \frac{F}{mg} = \frac{150}{50\times10} = 0.3$$

**例2**：斜面倾角37°，物体匀速下滑，求 $\mu_k$。

$$\mu_k = \tan37° = \frac{\sin37°}{\cos37°} = \frac{0.6}{0.8} = 0.75$$

**例3**：已知 $\mu_k = 0.4$，在斜面（倾角30°）上推箱子匀速上行，所需沿斜面推力为多少？（$m=10\text{ kg}$，$g=10\text{ m/s}^2$）

$$N = mg\cos30° = 100\times\frac{\sqrt{3}}{2} \approx 86.6\text{ N}$$
$$f_k = \mu_k N = 0.4\times86.6 \approx 34.6\text{ N}\text{（沿斜面向下）}$$
$$F = mg\sin30° + f_k = 100\times0.5 + 34.6 = 84.6\text{ N}$$

### 易错点

1. **$\mu$ 是常数，不随 $N$ 变化**：$\mu$ 由材料决定，不是"压力越大 $\mu$ 越大"。
2. **$\mu$ 与面积无关**：相同材料，接触面大小不影响 $\mu$。
3. **斜面法只适用于匀速下滑**：加速下滑时合力不为零，$\tan\theta \neq \mu_k$。

### 与初中物理衔接

初中知道摩擦力与压力和粗糙程度有关，高中用摩擦因数 $\mu$ 精确表征粗糙程度，并给出定量公式 $f = \mu N$。
