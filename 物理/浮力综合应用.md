# 浮力综合应用

> **一句话**：浮力原理综合运用于密度测量、打捞沉船、海洋工程等场景，涉及多公式联立求解。
> **English**: Buoyancy principles are applied in density measurement, salvaging sunken ships, ocean engineering, and complex multi-equation problems.

---

## 📚 给 6-12 岁（自读版）

浮力不只是教科书里的公式，它在真实世界中有很多重要应用：

**1. 测量固体密度（利用浮力）**
把固体浸入水中，用称重法算浮力，再算体积，最后算密度。

**2. 打捞沉船**
在沉船上绑上很多充气的浮筒（橡皮袋），充气后体积增大，浮力增大，把沉船"托"上来。

**3. 潜水艇与鱼鳔**
潜水艇靠充/排水控制浮沉，鱼靠鱼鳔充/排气控制浮沉（见 [[潜水艇]]）。

**4. 盐水漂浮（死海）**
死海盐度极高，密度约 1240 kg/m³，人（密度约 1000-1050 kg/m³）放进去必然漂浮。

**5. 水果能否漂浮**
苹果：密度约 820 kg/m³（漂浮）；西瓜：密度约 900-1000 kg/m³（刚好能漂或略下沉）；橙子去皮（密度增大）反而会沉。

## 🎓 给 12+（进阶版）

### 综合题解题框架

**受力分析 → 确定状态 → 选择公式 → 联立求解**

常见题型：
1. **称重法测密度**：用弹簧测力计在空气和液体中称量，联立 $F_{浮} = G - F_{示}$ 和 $\rho = m/V$
2. **漂浮状态求密度比**：$\rho_{物}/\rho_{液} = V_{排}/V_{物}$
3. **变液体比较浮力**：漂浮时 $F_{浮} = G$（不变），但 $V_{排}$ 变化
4. **分析打捞过程**：浮力随离水过程减小

### 称重法测固体密度

**步骤：**
1. 弹簧测力计称固体重力：$G = mg$
2. 将固体浸没水中，读示数：$F_{示}$
3. 计算浮力：$F_{浮} = G - F_{示}$
4. 计算体积：$V = F_{浮}/(\rho_{水} g)$
5. 计算密度：$\rho = m/V = G/(g \cdot V)$

合并公式：

$$\rho_{物} = \frac{G}{G - F_{示}} \cdot \rho_{水}$$

### 称重法测液体密度

**步骤：**（固体密度已知）
1. 弹簧测力计称固体重力：$G$，得质量 $m$，计算体积 $V = m/\rho_{固}$
2. 将固体浸没待测液体，读示数：$F_{示}'$
3. 浮力：$F_{浮} = G - F_{示}'$
4. 液体密度：$\rho_{液} = F_{浮}/(g V)$

### 打捞沉船的物理分析

设沉船质量 $M$，体积 $V_{船}$，完全浸没水中：

- 水中重力：$G_{水中} = G - F_{浮0} = Mg - \rho_{水} g V_{船}$（水中视重）
- 需要外力至少：$F_{外} \geq G_{水中}$

加浮筒（体积 $V_{筒}$，充气）后：

$F_{浮} = \rho_{水} g (V_{船} + V_{筒})$

当 $F_{浮} + F_{外} = Mg$ 时刚好能提起。

### 典型例题

**例1（测密度）：** 一块不规则石头，空气中重 5 N，浸没水中弹簧测力计读数 3.2 N。求石头密度。

解：$F_{浮} = 5 - 3.2 = 1.8\text{ N}$

$V = F_{浮}/(\rho_{水} g) = 1.8/(1000 \times 10) = 1.8 \times 10^{-4}\text{ m}^3$

$m = G/g = 5/10 = 0.5\text{ kg}$

$\rho = m/V = 0.5/(1.8 \times 10^{-4}) \approx 2778\text{ kg/m}^3$

**例2（漂浮比较）：** 同一木块分别放入水（1000 kg/m³）和盐水（1200 kg/m³）中漂浮，浮力一样吗？浸入深度一样吗？

解：
- 浮力：两次均等于木块重力（$F_{浮} = G$），浮力一样。
- 浸入深度：$V_{排} = F_{浮}/(\rho_{液} g)$，盐水密度大，$V_{排}$ 更小，浸入更浅。

**例3（综合）：** 一正方体（边长 10 cm，密度 800 kg/m³），部分露出水面漂浮，求露出水面的高度。

解：$V = (0.1)^3 = 10^{-3}\text{ m}^3$，$m = \rho V = 800 \times 10^{-3} = 0.8\text{ kg}$

漂浮：$V_{排} = m/\rho_{水} = 0.8/1000 = 8 \times 10^{-4}\text{ m}^3$

浸入深度：$h_{浸} = V_{排}/S = 8 \times 10^{-4}/(0.1)^2 = 0.08\text{ m} = 8\text{ cm}$

露出高度：$10 - 8 = 2\text{ cm}$

### 易错点

- 同一物体在不同液体中漂浮，浮力**相同**（均等于重力），但浸入体积**不同**。
- 分析打捞过程，离开水面之前浮力随排水体积减小而减小，完全出水后浮力为零。
- 密度公式 $\rho = m/V$ 中，$V$ 是物体体积，不是排开液体体积（完全浸没时两者相等）。
