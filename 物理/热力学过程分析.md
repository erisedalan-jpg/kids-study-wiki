# 热力学过程分析

> **一句话**：研究系统从一个平衡态变化到另一个平衡态时，状态参量 ( \(p, V, T\) ) 的变化规律及能量转化（做功、传热、内能变化）的物理过程。
> **English**: The study of the changes in state parameters ( \(p, V, T\) ) and energy conversions (work, heat transfer, internal energy change) as a system evolves from one equilibrium state to another.

## 🎓 考点精讲

**定义/原理：**
热力学过程分析的核心是**热力学第一定律** \(\Delta U = W + Q\)（注意符号规则：系统吸热 \(Q>0\)，外界对系统做功 \(W>0\)，内能增加 \(\Delta U>0\)；若按系统对外做功 \(W' = -W\)，则公式写为 \(\Delta U = Q - W'\)）。理想气体的内能只由温度决定：\(\Delta U = n C_{V,m} \Delta T\)。

高考常考的四种典型过程及其特征：
1.  **等温过程** (\(\Delta T = 0\))：\(\Delta U = 0\)，\(W + Q = 0\)，即系统对外做功等于吸收的热量。对理想气体，满足 \(pV = \text{常量}\)。
2.  **等容过程** (\(\Delta V = 0\))：\(W = 0\)（或 \(W'=0\)），\(\Delta U = Q\)，即吸收的热量全部用于增加内能。满足 \(\frac{p}{T} = \text{常量}\)。
3.  **等压过程** (\(\Delta p = 0\))：\(W = -p\Delta V\)，\(\Delta U = Q - p\Delta V\)。满足 \(\frac{V}{T} = \text{常量}\)。
4.  **绝热过程** (\(Q=0\))：\(\Delta U = W\)，气体绝热膨胀对外做功时，内能减小，温度降低。其 \(p-V\) 图上的曲线比等温线更陡。

**关键方法：**
解决热力学过程问题的标准方法是“**状态方程+第一定律**”两步法：
1.  **判过程，定参量**：根据题意或 \(p-V\) 图斜率判断过程类型，利用理想气体状态方程 \(\frac{pV}{T} = C\) 确定各状态点未知参量及变化量的正负（如温度 \(T\) 是升是降）。
2.  **分能量，列方程**：
    - **内能变化**：只看温度。\(T \uparrow \Rightarrow \Delta U > 0\)。
    - **做功**：只看体积。气体膨胀 (\(V \uparrow\)) \(\Rightarrow\) 系统对外做功 (\(W' > 0\) 或 \(W < 0\))；反之，则外界对系统做功。
    - **热量**：结合 \(\Delta U\) 和 \(W\) 的正负，由第一定律反推 \(Q\) 的正负及大小。

**典型应用：**
高考常结合 \(p-V\) 图象设问，如给出一条由多条过程曲线组成的循环路径，判断各阶段“吸/放热”和“做功”情况。例如，气体内能增大且对外做功的过程，必定从外界吸热 (\(Q = \Delta U + W' > 0\))。另一经典场景是判断绝热线与等温线的关系：从同一初态出发，膨胀相同体积，绝热过程的压强下降更快，终点温度更低。

## ⚠️ 高频易错点

- **混淆做功的符号规则**：务必将“系统对外做功”与“外界对系统做功”区分清楚，并始终与自己书写的热力学第一定律表达式中的 \(W\) 符号约定保持一致。
- **误认为等温过程不吸/放热**：等温膨胀时 \(\Delta U = 0\)，但气体对外做功需吸收等量热量，并非无热量交换。
- **绝热过程温度变化的判断失误**：错误地认为绝热膨胀过程中压强减小是体积增大直接导致，忽略温度同时降低对压强的双重影响，从而在 \(p-V\) 图上错误判断曲线走向。
- **混淆“热量”与“内能/温度”**：仅凭公式 \(Q = cm\Delta t\) 的条件反射去分析所有变质量或气体变化过程，忽略了热力学第一定律中能量转化的整体性。例如在等温膨胀中，吸热量不等于内能增加量。
