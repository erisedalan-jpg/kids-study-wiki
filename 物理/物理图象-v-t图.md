# v-t 图（速度-时间图象）

> **一句话**：v-t 图以时间为横轴、速度为纵轴，图线的斜率表示加速度，图线下面积表示位移，是分析变速运动的核心工具。
> **English**: A v-t graph plots velocity on the vertical axis against time on the horizontal axis; the slope gives acceleration and the area under the graph gives displacement.

---

## 📚 给 6-12 岁（自读版）

s-t 图告诉我们"走了多远"，**v-t 图**告诉我们"速度怎么变"！

**v-t 图的坐标：**
- 横轴（x 轴）：时间 $t$（单位：s）
- 纵轴（y 轴）：速度 $v$（单位：m/s）

**不同线的含义：**

| 图线形状 | 运动状态 |
|---------|---------|
| 水平直线（$v$ = 常数） | 匀速运动 |
| 水平线 $v=0$ | 静止 |
| 向上倾斜的直线 | 匀加速运动（速度均匀增大） |
| 向下倾斜的直线 | 匀减速运动（速度均匀减小） |
| 斜率越大 | 加速度越大（速度变化越快） |

**v-t 图的两个关键信息：**
1. **斜率 = 加速度**（图线越陡，速度变化越快）
2. **图线下的面积 = 路程**（面积越大，走得越远）

**例题：**

一辆汽车从静止出发，5 s 内速度从 0 加速到 10 m/s（直线段），然后匀速行驶 10 s。

- 第一段：v-t 图为从 (0,0) 到 (5,10) 的直线，斜率 $= 10/5 = 2\text{ m/s}^2$（加速度为 2 m/s²）
- 第二段：v-t 图为水平线 $v=10\text{ m/s}$

## 🎓 给 12+（进阶版）

### v-t 图的数学本质

**斜率 = 加速度：**

$$a = \frac{\Delta v}{\Delta t} = \text{图线斜率}$$

**面积 = 位移：**

$$s = \int v \, dt = \text{图线与时间轴围成的面积}$$

（初中阶段仅要求矩形和三角形的面积计算）

### 典型运动的 v-t 图

**匀速运动：**
- v-t 图：水平直线（$v = \text{常数}$）
- 斜率 = 0（加速度 = 0）

**匀加速直线运动（初速度为零）：**
- v-t 图：过原点的直线（$v = at$）
- 斜率 = 加速度 $a$
- 面积 = $\frac{1}{2}vt = \frac{1}{2}at^2$（三角形面积）

**匀减速到零：**
- v-t 图：斜率为负的直线，终点在 $v=0$ 处
- 面积（梯形或三角形）= 总位移

**自由落体（忽略空气阻力）：**
- $v = gt$（$g = 10\text{ m/s}^2$）
- v-t 图：从原点出发、斜率为 10 的直线

### 从 v-t 图求各物理量

| 物理量 | 求法 |
|-------|------|
| 某时刻速度 | 纵轴读数 |
| 加速度 | 图线斜率 $= \Delta v / \Delta t$ |
| 某段时间内位移 | 图线下面积（矩形/梯形/三角形） |
| 平均速度 | $\bar{v} = s/t$（面积除以时间） |

### 常见 v-t 图形状汇总

| 形状 | 对应运动 | 加速度 |
|------|---------|-------|
| 水平线（$v>0$） | 匀速运动 | $a=0$ |
| 水平线（$v=0$） | 静止 | $a=0$ |
| 正斜率直线 | 匀加速 | $a>0$，恒定 |
| 负斜率直线 | 匀减速 | $a<0$，恒定 |
| 上凸曲线（斜率减小） | 加速度递减的加速运动 | $a>0$ 但减小 |
| 下凸曲线（斜率增大） | 加速度递增的加速运动 | $a>0$ 且增大 |

### 典型例题

**例1：** v-t 图上，某物体从 $v_0=4\text{ m/s}$ 匀减速，经过 4 s 速度降为 0。求：加速度大小和这 4 s 内的位移。

解：
- 斜率（加速度）：$a = \Delta v / \Delta t = (0-4)/4 = -1\text{ m/s}^2$，大小为 $1\text{ m/s}^2$
- 位移 = 梯形面积 = $\frac{1}{2}(v_0 + 0)\times t = \frac{1}{2}\times 4\times 4 = 8\text{ m}$

**例2：** 两辆车 A、B 的 v-t 图都是直线，A 的斜率更大。它们在同一时间内谁走得更远？

解：不能仅凭斜率判断——位移取决于图线下的**面积**，而不是斜率。需要具体比较各自图线下的面积。

**例3：** 自由落体 3 s 后的速度和下落高度。（$g=10\text{ m/s}^2$）

解：
- $v = gt = 10\times 3 = 30\text{ m/s}$
- $h = \text{三角形面积} = \frac{1}{2}\times 30\times 3 = 45\text{ m}$（或 $h = \frac{1}{2}gt^2 = \frac{1}{2}\times 10\times 9 = 45\text{ m}$）

### 与 s-t 图的对照

| 比较项 | s-t 图 | v-t 图 |
|--------|-------|-------|
| 纵轴 | 路程/位移 $s$ | 速度 $v$ |
| 斜率含义 | 速度 $v$ | 加速度 $a$ |
| 面积含义 | 无直接意义 | 位移 $s$ |
| 水平线含义 | 静止 | 匀速运动 |
| 适用场景 | 描述位置变化 | 描述速度变化 |

### 易错点

- **v-t 图斜率 = 加速度**，不是速度（速度是纵轴读数）
- **面积 = 位移**，不是路程（方向相反时面积相减）
- v-t 图线的高低代表速度大小，不代表位置高低
- 匀速运动在 s-t 图是斜线，在 v-t 图是水平线——两图含义不同，要区分
