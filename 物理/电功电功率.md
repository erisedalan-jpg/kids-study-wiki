# 电功电功率

> **一句话**：电功是电流通过一段电路时电场力对自由电荷所做的功，电功率是单位时间内电流所做的功，反映电流做功的快慢。
> **English**: Electric work is the work done by the electric field force on free charges as current flows through a circuit, and electric power is the amount of electrical work done per unit time.

## 🎓 考点精讲

**定义与公式**  
电功用来量度电能转化为其他形式能的多少，其基本公式为：  
\[
W = UIt
\]  
式中 \(U\) 为电压，\(I\) 为电流，\(t\) 为通电时间。在国际单位制中，电功的单位是焦耳（J），常用单位还有千瓦时（度）。  
电功率表示电流做功的快慢，定义式为：  
\[
P = \frac{W}{t} = UI
\]  
上述两式对任何电路均适用。在纯电阻电路中，欧姆定律 \(U = IR\) 成立，电功率又可写为：  
\[
P = I^2R = \frac{U^2}{R}
\]  
但这两个推导式对含有电动机、电解槽等非纯电阻元件不适用，此时只能用 \(UI\) 计算总功率。焦耳定律给出纯电阻发热规律：\(Q = I^2Rt\)，在非纯电阻电路中，发热功率 \(P_{\text{热}} = I^2R\) 小于总电功率 \(UI\)。

**关键方法**  
- **电路性质判断**：先判断是否为纯电阻电路。若用电器把电能全部转化为内能（如电热器），则可使用纯电阻推导式；若有电动机等则不能。  
- **功率关系分析**：非纯电阻电路中，输入总功率 \(P_{\text{总}} = UI\)，发热功率 \(P_{\text{热}} = I^2R\)，输出机械功率 \(P_{\text{出}} = UI - I^2R\)。高考常要求区分“消耗的电能”与“产生的热量”。  
- **电能表与电能计算**：熟悉电能表参数（如 \(N\ \text{r/(kW·h)}\)）与转数、时间的换算关系，注意单位的统一（1 kW·h = 3.6×10⁶ J）。

**典型应用**  
高考常从两个角度命题：  
1. 纯电阻电路中的功率分配——如串/并联电路中各电阻的功率之比与实际电压、电流分配的关系，结合图像或动态变化分析灯泡亮暗问题。  
2. 非纯电阻电路——以电动机模型最为常见，给出电动机的电压、电流、线圈电阻，求输入功率、发热功率和机械效率。  
解这类题的关键是严格区分总功率和热功率，抓住线圈内阻上仍满足焦耳定律这一核心。

## ⚠️ 高频易错点

- **混淆纯电阻与非纯电阻公式**：在含有电动机的电路中，错误地使用 \(P = \frac{U^2}{R}\) 或 \(P = I^2R\) 计算总功率，实际上这两个式子只能用于计算纯电阻的热功率或全部功率。  
- **电功与电热混为一谈**：误认为电流做的功全部用来产生热量，导致在非纯电阻问题中用 \(W = UIt\) 计算的数值等同于 \(Q = I^2Rt\)。必须明确：纯电阻时 \(W = Q\)，非纯电阻时 \(W > Q\)。  
- **单位换算失误**：将千瓦时换算为焦耳时漏乘时间因子，或在利用电能表转数求电功时忘记将转数除以标定常数。  
- **动态电路功率变化判断僵化**：在滑动变阻器移动时，仅凭“电阻变大则功率变小”的直觉判断，忽略电阻等于电源内阻时输出功率最大的条件（\(P_{\text{out max}} = \frac{E^2}{4r}\)）。
