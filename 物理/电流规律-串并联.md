# 电流规律-串并联

> **一句话**：串联电路各处电流相等；并联电路总电流等于各支路电流之和，且各支路电流与电阻成反比。
> **English**: In series circuits, current is the same throughout; in parallel circuits, total current equals the sum of branch currents, and branch current is inversely proportional to resistance.

---

## 📚 给 6-12 岁（自读版）

**串联电路的电流：**

想象一条单行道，所有车（电荷）都从同一条路过——每个路段通过的车数量一样！

$$I_1 = I_2 = I \quad \text{（串联，电流处处相等）}$$

**并联电路的电流：**

想象一条路在前方分成了两条岔路，车辆分流——两条路的车加起来等于原来一条路的总量！

$$I = I_1 + I_2 \quad \text{（并联，总电流 = 各支路之和）}$$

并联时，哪条路"更宽"（电阻更小），走的车（电流）就更多。

---

## 🎓 给 12+（进阶版）

### 串联电路电流规律

**规律**：串联电路中，各处的电流相等。

$$I = I_1 = I_2 = \cdots = I_n$$

**物理解释**：串联只有一条电流路径，单位时间内通过每个截面的电荷量必然相同（电荷守恒，没有电荷在节点"堆积"）。

**推论**：
- 在串联电路任意位置串联一个电流表，读数相同
- 若某处电流不同，说明该处发生了分叉（并联）

### 并联电路电流规律

**规律**：并联电路总电流等于各支路电流之和。

$$I = I_1 + I_2 + \cdots + I_n$$

**物理解释**（基尔霍夫电流定律）：节点处流入的电流之和等于流出的电流之和（电荷守恒）。

**支路电流分配**（并联时各支路电压相同）：

$$I_i = \frac{U}{R_i}$$

$$\frac{I_1}{I_2} = \frac{R_2}{R_1}$$

电阻越小，分到的电流越大。

### 两个公式综合总结

| | 串联 | 并联 |
|--|------|------|
| 电流 | $I_1 = I_2 = I$（相等） | $I = I_1 + I_2$（相加） |
| 电压 | $U = U_1 + U_2$（相加） | $U_1 = U_2 = U$（相等） |
| 电阻 | $R = R_1 + R_2$（相加） | $\frac{1}{R} = \frac{1}{R_1} + \frac{1}{R_2}$（倒数相加） |

**记忆口诀**：串联"电流相等，电压分配"；并联"电压相等，电流分配"。

### 典型例题

**例1（串联）**：串联电路中，$R_1 = 5$ Ω，$R_2 = 10$ Ω，总电压 15 V，求各处电流和各电阻两端电压。

**解**：

$$I = \frac{U}{R_1 + R_2} = \frac{15}{15} = 1 \text{ A}$$

$$I_1 = I_2 = 1 \text{ A（串联电流处处相等）}$$

$$U_1 = IR_1 = 5 \text{ V}，\quad U_2 = IR_2 = 10 \text{ V}$$

**例2（并联）**：$R_1 = 6$ Ω 和 $R_2 = 3$ Ω 并联，总电压 6 V，求各支路电流和总电流。

**解**：

$$I_1 = \frac{U}{R_1} = \frac{6}{6} = 1 \text{ A}$$

$$I_2 = \frac{U}{R_2} = \frac{6}{3} = 2 \text{ A}$$

$$I = I_1 + I_2 = 3 \text{ A}$$

**易错点：**
- 串联"电流相等"不是说所有导体都有相同的电流，而是指同一串联回路中
- 并联增加支路，干路电流增大，但各支路电流不变（电压不变，各支路电阻不变）
- 混联电路（串并联混合）需要先分析电路结构，再分步计算
