# 电源效率

> **一句话**：电源效率是指电源将其他形式能转化为电能时，外电路得到的功率（输出功率）与电源总功率的比值，纯电阻电路中 η = R/(R+r)。
> **English**: Power efficiency of a source is the ratio of the output power delivered to the external circuit to the total power supplied by the source; for a purely resistive circuit, η = R/(R+r).

## 🎓 考点精讲

**定义与原理**  
对一个电动势为 *E*、内阻为 *r* 的电源，闭合电路中：  
总功率 *P*<sub>总</sub> = *EI*，  
内电阻发热功率 *P*<sub>内</sub> = *I*<sup>2</sup>*r*，  
输出功率（外电路消耗的功率）*P*<sub>出</sub> = *UI* = *I*<sup>2</sup>*R*（纯电阻），  
电源效率  
\[
\eta=\frac{P_{\text{出}}}{P_{\text{总}}}=\frac{UI}{EI}=\frac{U}{E}=\frac{R}{R+r}
\]  
式中 *U* 为路端电压，*R* 为外电阻（仅适用于外电路为纯电阻）。对非纯电阻电路，效率仍用 η = *UI*/(*EI*)，但 *U* 与 *R* 不满足欧姆定律形式。

**关键方法与考查方式**  
高考主要围绕两种设问角度：  
1. **与外电阻的关系**：给定电源 *E*、*r* 不变，随外阻 *R* 增大，η 增大且趋近于1，但输出功率先增后减。当 *R*=*r*（最大输出功率）时，η = 50%。  
2. **与路端电压的关系**：从 η = *U*/*E* 出发，通过 U‑I 图像或电路动态变化求效率。解题步骤：①明确内外电路；②选择 η = *U*/*E*、*R*/(*R*+*r*) 或 η = *P*<sub>出</sub>/*P*<sub>总</sub>；③注意纯电阻与非纯电阻的区别。

**典型情境**  
- 测电源电动势和内阻实验中，结合 U‑I 图线求某一工作点的效率。  
- 闭合电路动态分析中，判断滑动变阻器阻值变化时电源效率的增减。

## ⚠️ 高频易错点

- **混淆效率最高与输出功率最大**：η 随 *R* 增大而增大，当 *R*→∞ 时 η→100%（断路状态下的极限效率），但输出功率趋于0；最大输出功率对应 *R*=*r*，此时 η 仅为 50%，学生常以为效率最大时功率也最大。  
- **忽视非纯电阻电路的效率计算**：对含电动机的电路，电机两端电压 *U* 与电流 *I* 不满足 *U*=*IR*，直接用 η = *R*/(*R*+*r*) 会导致错误，必须用 η = *UI*/(*EI*)。  
- **混淆外电阻与等效电阻**：当外电路包含混联支路时，错误地用某一段电阻代替总外阻 *R*<sub>外</sub> 代入 η = *R*<sub>外</sub>/(*R*<sub>外</sub>+*r*)。  
- **路端电压与电动势比值使用不当**：部分学生记混 η = *E*/*U* 或 η = *r*/(*R*+*r*)，造成计算结果错误。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 电源电动势 | electromotive force (emf) | 电源将单位正电荷从负极移到正极所做的功 |
| 内阻 | internal resistance | 电源内部对电流的阻碍作用，导致内电压 Ir |
| 输出功率 | output power | 外电路消耗的功率，P<sub>出</sub> = UI |
| 总功率 | total power | 电源内部非静电力做功的功率，P<sub>总</sub> = EI |
| 效率 | efficiency | 输出功率与总功率的百分比，η = P<sub>出</sub>/P<sub>总</sub>×100% |
