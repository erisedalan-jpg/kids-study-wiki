## 🎓 磁场对电流的作用

### 基本现象

通电导线在磁场中会受到力的作用——**安培力**。这是电动机等装置的工作原理基础。

实验：将通电导线放入磁场中，改变电流方向或磁场方向，观察导线运动方向的变化，验证左手定则。

### 核心公式与方向

$$F = BIL\sin\theta$$

方向用**左手定则**（见 [[安培力方向-左手定则]]）。

### 线框在匀强磁场中的受力

**矩形线框**在匀强磁场中：若线框平面与磁场平行，线框受到两个大小相等、方向相反的力，形成**力矩**，使线框转动。

力矩：$M = NBIS\cos\alpha$

其中 $N$ 为线圈匝数，$S$ 为面积，$\alpha$ 为线框平面与磁场方向的夹角。

**当线框平面垂直磁场时**（$\alpha = 0$）：$M = 0$，线框处于平衡位置。

**当线框平面平行磁场时**（$\alpha = 90°$）：力矩最大，$M_{max} = NBIS$。

这是**直流电机**和**电流计**的工作原理。

### 直流电机工作原理

```
通入电流 → 线圈在磁场中受安培力矩 → 转动 → 通过换向器保持转向不变 → 持续旋转
```

能量转换：电能 → 机械能（与发电机相反）

### 电流计（检流计）

利用通电线圈在磁场中受到的安培力矩，使指针偏转，指针偏转角正比于电流：

$$\alpha \propto I$$

量程扩展：
- 并联小电阻 → 电流表（分流）
- 串联大电阻 → 电压表（分压）

### 安培力对磁性材料的解释

安培认为：磁铁之所以有磁性，是因为内部存在**分子电流**（分子环形电流），每个分子电流相当于一个小磁铁，分子磁铁整齐排列时产生宏观磁性。

这就是**安培分子电流假说**，后来被量子力学证实（电子自旋和轨道运动）。

### 易错点

- 安培力做功（线框转动时），因此安培力不守恒（能量有转化）；与此对比，洛伦兹力**对单个粒子不做功**。
- 线框受力矩而转动，当线框平面垂直于磁场时，力矩为零，但此时速度最大（惯性继续转过）。
- 安培力 $F = BIL$ 适用于**直线导体**垂直磁场的情形，弯曲导体需积分。

### 与必修二衔接

必修二学习了功和能量转换；本节从安培力做功引出电动机的能量转化，将电功率与机械功率联系起来，是电能利用的重要应用。
