## 🎓 磁场综合应用（高中物理）

### 核心公式汇总

| 公式 | 含义 |
|------|------|
| $F = BIL\sin\theta$ | 安培力大小 |
| $F = qvB\sin\theta$ | 洛伦兹力大小 |
| $r = mv/(qB)$ | 带电粒子圆周运动半径 |
| $T = 2\pi m/(qB)$ | 圆周运动周期 |
| $v = E/B$ | 速度选择器筛选速度 |

### 典型综合模型

#### 模型一：霍尔效应

通电导体板置于磁场中，载流子受洛伦兹力向一侧偏聚，产生横向电压（霍尔电压）：

$$U_H = \frac{IB}{nqd}$$

可用于测磁场强度或判断载流子类型（正/负）。

#### 模型二：匀强磁场中的导体棒平衡

导轨上的导体棒受安培力、重力、支持力/摩擦力共同作用，列平衡方程：

$$BIL = mg\sin\alpha \quad \text{（斜面上）}$$

#### 模型三：带电粒子在有界磁场中的偏转

粒子进入半圆形磁场区域，依据几何关系（入射点、出射点、圆心）求偏转角度和出射位置。常用直角三角形关系：

$$\sin\phi = \frac{d}{r}$$

$d$ 为粒子入射位置到边界的垂直距离，$\phi$ 为偏转角。

#### 模型四：磁场中的电磁炮

通电导轨中的滑动导体棒受安培力加速，动能不断增大，安培力做正功：

$$W = F \cdot s = BILs$$

### 选必二重点题型

1. **安培力方向判断**：给定 $I$、$B$ 方向，用左手定则确定 $F$ 方向
2. **圆周运动参数计算**：由 $B$、$v$、$m/q$ 求 $r$、$T$
3. **质谱仪综合**：速度选择 → 磁场偏转 → 比荷计算
4. **导轨问题**：安培力做功，能量转化
5. **有界磁场几何关系**：圆弧轨迹 + 几何约束

### 典型例题

**例（有界磁场）**：宽度为 $d = 0.1$ m 的匀强磁场区域，带电粒子以速度 $v = 10^6$ m/s 垂直边界射入，磁场 $B = 0.2$ T，粒子质量 $m = 10^{-26}$ kg，$q = 10^{-19}$ C，求粒子出射位置距入射点的距离。

$$r = \frac{mv}{qB} = \frac{10^{-26}\times10^6}{10^{-19}\times0.2} = 0.5\text{ m}$$

$$\sin\phi = \frac{d}{r} = \frac{0.1}{0.5} = 0.2 \implies \phi \approx 11.5°$$

出射点偏移：$x = 2r\sin\phi \cdot \sin\phi = 2r\sin^2\phi$（待家长核对具体几何关系）

### 易错点

1. 有界磁场中粒子可能**穿出对面边界**也可能**从同侧反射**，需先比较 $d$ 与 $r$ 的大小。
2. 安培力做功改变导体棒动能，但**磁场本身不做功**（能量来自电源）。
3. 霍尔效应中，注意载流子实际移动方向（电子向左 → 电流向右）。
