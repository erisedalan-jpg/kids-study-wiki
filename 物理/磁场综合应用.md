## 🎓 磁场综合应用

### 概述

磁场综合题是将磁场中的圆周运动、安培力、洛伦兹力、速度选择器等知识点综合考查的题型，是高中物理的重要综合题型。

### 综合题的典型模型

#### 模型一：半圆弧轨迹 + 出射角

带电粒子从匀强磁场的某边界入射，在磁场中做半圆或圆弧运动后从另一边界出射。

分析要点：
1. 用左手定则确定偏转方向，画出轨迹圆弧
2. 确定圆心位置（洛伦兹力始终指向圆心）
3. 利用几何关系（入射点、圆心、出射点）求出射位置和角度
4. 核心：$r = mv/(qB)$

#### 模型二：粒子在有限宽磁场区域的穿越

粒子穿越宽度为 $d$ 的磁场区域，求偏移量 $\delta$ 和偏转角 $\theta$：

$$\sin\theta = \frac{d}{r}$$（$d$ 为穿越宽度，$r$ 为轨道半径，当 $d < r$ 时）

$$\delta = r - \sqrt{r^2 - d^2} = r(1 - \cos\theta)$$

#### 模型三：多粒子在磁场中的分离

不同质量或电荷量的粒子，以相同速度射入磁场，半径不同，落点不同。

$$r = \frac{mv}{qB} \propto \frac{m}{q}$$

质谱仪即此原理。

#### 模型四：洛伦兹力与重力的综合

带电粒子在竖直磁场 + 重力场中运动：
- 洛伦兹力（水平）+ 重力（竖直）→ 斜向合力 → 螺旋形轨迹（高中了解）

### 解题通用步骤

1. **判断粒子电性和运动方向**（利用左手定则）
2. **画出运动轨迹**（圆弧方向、圆心位置）
3. **建立几何模型**（利用直角、弦长、半径关系）
4. **代入公式计算**（$r = mv/qB$，三角函数）

### 典型例题

**例**：质子以速度 $v = 2\times10^6\,\text{m/s}$ 水平射入宽 $d = 5\,\text{cm}$ 的匀强磁场区域（$B = 0.05\,\text{T}$，方向垂直纸面向外），求出射时的偏转角。

$$r = \frac{mv}{qB} = \frac{1.67\times10^{-27}\times2\times10^6}{1.6\times10^{-19}\times0.05} \approx 0.42\,\text{m}$$

$$\sin\theta = \frac{d}{r} = \frac{0.05}{0.42} \approx 0.119 \Rightarrow \theta \approx 6.8°$$

### 常见错误模式

| 错误 | 正确做法 |
|------|---------|
| 把圆心与入射点混淆 | 圆心在洛伦兹力方向上，不是入射点 |
| 忽略粒子方向判断 | 必须先用左手定则确定偏转方向 |
| 将直径 $D$ 当成半径 | 注意区分 $r$ 和 $D = 2r$ |
| 不画图直接计算 | 磁场题必须画图，建立几何模型 |

### 与其他知识点的联系

- **质谱仪**：磁场分离粒子的经典应用
- **回旋加速器**：利用等周期多次加速
- **霍尔效应**：横向洛伦兹力产生电压
- **速度选择器**：电场与磁场的综合平衡

### 易错点

- 圆弧轨迹中，弦长（入射到出射的直线距离）不等于轨道半径。
- 粒子速度方向时刻与洛伦兹力垂直，但洛伦兹力方向随速度方向改变而改变。
- 负电荷的偏转方向与正电荷相反，需注意粒子的正负性。
