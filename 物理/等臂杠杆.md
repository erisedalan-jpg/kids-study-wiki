# 等臂杠杆

> **一句话**：等臂杠杆的动力臂等于阻力臂（$l_1 = l_2$），不省力也不费力，天平是最典型的等臂杠杆。
> **English**: An equal-arm lever has l₁ = l₂, neither saving nor wasting force; the balance scale is the classic example.

---

## 📚 给 6-12 岁（自读版）

**等臂杠杆**的两侧力臂完全一样长——这样的杠杆不省力，也不费力，力有多大，阻力就有多大。

**最典型的等臂杠杆：天平**

天平两侧盘子到支点（中间）距离相等（$l_1 = l_2$），所以平衡时两盘质量相等（$F_1 = F_2$，即 $m_1 g = m_2 g$，$m_1 = m_2$）。

天平用来**比较质量**，不是称重力（天平在月球上同样可以用，因为两边重力同等减小，比例不变）。

**其他等臂杠杆（近似）：**
- 跷跷板（理想状态，两侧长度相等）
- 旧式磅秤的横杆（有时是等臂的）
- 杠铃（两端相同距离处放等重哑铃片）

**等臂杠杆的特点：**
- $l_1 = l_2$ → $F_1 = F_2$（不省力不费力）
- 动力位移 = 阻力位移（不省距离也不费距离）
- 专长：**精确比较两侧的力（质量）**

## 🎓 给 12+（进阶版）

### 等臂杠杆的条件

$$l_1 = l_2 \Rightarrow F_1 = F_2$$

在重力场中：若 $F_1 = m_1 g$，$F_2 = m_2 g$，则平衡时 $m_1 = m_2$。

### 天平的工作原理

天平利用等臂杠杆，通过与已知质量的砝码比较，测量未知物体的质量。

**天平的特点：**
- 在不同重力环境中（月球、高山），天平仍然准确（两边重力同等变化，比值不变）
- 弹簧测力计在不同重力环境中会不准（因为弹力 = 重力，$g$ 变化就不准了）
- 天平测量的是**质量**，弹簧测力计测量的是**重力（力）**

**天平使用方法：**
1. 水平放置，调平（移动平衡螺母，使指针居中）
2. 左盘放待测物，右盘加砝码，从大到小
3. 平衡时，待测物质量 = 砝码总质量
4. 注意：不能用手直接拿砝码（汗液腐蚀），要用镊子

### 等臂杠杆的力矩平衡

等臂杠杆中，两边力矩相等：

$$M_1 = F_1 l_1 = F_2 l_2 = M_2$$

当 $l_1 = l_2$ 时，$F_1 = F_2$。

若在等臂杠杆上，一端施力偏斜（力不垂直杆），力臂会缩短，不再是等臂，天平就不平衡了。

### 典型例题

**例1：** 天平右盘放有 50 g、20 g、5 g 砝码各一个，左盘放待测物，天平平衡，待测物质量是多少？

解：$m = 50 + 20 + 5 = 75\text{ g}$

**例2：** 用天平在月球上（重力加速度约为地球的 1/6）称量物体，结果与地球上一样吗？为什么？

解：一样。月球上两盘重力都变为地球上的 1/6，$F_1 = m_1 g_{月}$，$F_2 = m_2 g_{月}$，平衡时 $m_1 g_{月} = m_2 g_{月}$，即 $m_1 = m_2$，结果不变。

**例3：** 一个等臂杠杆，两端各挂 3 N 的力，如果在一端额外加 1 N，向哪边倾斜？

解：加了 1 N 的一端力矩变为 $(3+1) \times l = 4l$，另一端为 $3l$，力矩不等，向力矩大的一端倾斜（加力的一端下降）。

### 易错点

- 天平测质量，不是测重力（弹簧测力计测重力）。
- 等臂杠杆不省力，但也不费力，专门用于精确比较两侧的力或质量。
- 使用天平前必须调平（使支点处水平），否则结果不准。
