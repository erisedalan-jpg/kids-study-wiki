# 藻类植物

> **一句话**：藻类植物是结构最简单的植物类群，无根茎叶之分，大多生活在水中，能进行光合作用，是地球上最古老的光合生物之一。
> **English**: Algae are the simplest group of plants, lacking true roots, stems, and leaves; most live in water and perform photosynthesis, representing some of Earth's oldest photosynthetic organisms.

---

## 📚 给 6-12 岁（自读版）

海里绿绿的水草、池塘里的绿水、海边的紫菜、超市里的海带……这些都是**藻类植物**！

**藻类植物的特点**

1. **没有根、茎、叶的分化**：和陆地植物不同，藻类不分根茎叶，整个身体结构都很简单
2. **大多生活在水中**：江河湖海、鱼缸里都可能有藻类
3. **能进行光合作用**：藻类含有叶绿素，能利用阳光制造养料，同时释放氧气
4. **用孢子繁殖**：不靠种子，用孢子来繁殖后代

**常见的藻类**

| 名称 | 特点 |
|------|------|
| 水绵 | 淡水中的绿色丝状藻 |
| 衣藻 | 单细胞绿藻，有鞭毛能游动 |
| 海带 | 褐色，我们日常食用的海藻 |
| 紫菜 | 红色，营养丰富，可做汤 |
| 裙带菜 | 褐色，常见海藻蔬菜 |

**藻类的重要性**

- 🌊 地球上 **70%** 的氧气来自海洋中的藻类光合作用！
- 🐟 藻类是水中食物链的**生产者**，是鱼虾的食物来源
- 🍜 海带、紫菜等是人类的食物，富含碘等矿物质
- 💊 一些藻类可以提取有价值的生物活性物质

## 🎓 给 12+（进阶版）

### 定义

藻类植物（algae）是一类含有光合色素、能进行光合作用、无维管束、无真正根茎叶分化的低等植物（广义上包括原核的蓝藻和真核的各类藻类）。

注：现代分类学将藻类分散归入多个界（原生生物界、植物界、原核生物界），不再是单一的自然类群，但初中教材仍将其作为一个类群讲授。

### 主要类群

| 类群 | 色素 | 代表 | 生境 |
|------|------|------|------|
| 绿藻（Chlorophyta） | 叶绿素 a/b、类胡萝卜素 | 衣藻、水绵、石莼 | 淡水/海水 |
| 褐藻（Phaeophyta） | 叶绿素 a/c、岩藻黄素 | 海带、裙带菜 | 冷温带海水 |
| 红藻（Rhodophyta） | 叶绿素 a、藻红素 | 紫菜、石花菜 | 深海 |
| 硅藻（Bacillariophyta） | 叶绿素 a/c | 各类硅藻 | 海水/淡水 |
| 蓝藻（Cyanobacteria，原核） | 叶绿素 a、藻蓝素 | 颤藻、念珠藻 | 淡水/土壤 |

### 与其他植物类群的比较

| 特征 | 藻类 | 苔藓 | 蕨类 | 种子植物 |
|------|------|------|------|---------|
| 根茎叶分化 | 无 | 假根/茎叶 | 有 | 有 |
| 维管束 | 无 | 无 | 有 | 有 |
| 种子 | 无 | 无 | 无 | 有 |
| 主要生境 | 水中 | 阴湿陆地 | 阴湿陆地 | 各类陆地 |

### 生态意义

1. **初级生产者**：海洋藻类（浮游植物）固定了地球约 50% 的碳，每年产生约 50% 的大气氧气
2. **碳循环**：光合固碳，死亡后沉降海底（生物泵）
3. **水华与赤潮**：氮磷过量导致淡水蓝藻（水华）或海洋甲藻（赤潮）暴发，危害水体

### 易错点

1. **蓝藻是原核生物**，没有以膜为界的细胞核，不是真正的藻类植物，但习惯上归入藻类讲授
2. 藻类没有根茎叶，但有**叶状体、茎状体**等形态，不要混淆
3. 藻类在水中不用担心缺水，所以没有输导组织的演化压力
