# 器官

> **一句话**：器官是由多种组织按一定顺序结合在一起，共同完成某种生理功能的结构单位。
> **English**: An organ is a structural unit composed of multiple tissue types arranged in a specific order to collectively perform one or more physiological functions.

---

## 📚 给 6-12 岁（自读版）

你知道心脏、肺、胃、叶片、根……这些都叫什么吗？它们都是**器官**！

**器官是什么？**

器官就是由不同种类的组织组合在一起，共同完成某项工作的"功能单元"。

就像一台洗衣机，是由电机（动力）、滚筒（旋转容器）、控制面板（指令）等不同零件组合而成的。器官也一样，是由多种组织合作组成的"精密机器"。

**动物（人体）的常见器官**

- 🫀 **心脏**：主要由肌肉组织（心肌）构成，负责泵血
- 🫁 **肺**：由上皮组织、结缔组织等构成，负责气体交换
- 🧠 **大脑**：主要由神经组织构成，是指挥中枢
- 👁 **眼睛**：由角膜、晶状体、视网膜等不同组织构成，负责视觉

**植物的常见器官（六大器官）**

植物有 **6 种器官**，分两类：
- **营养器官**：根、茎、叶（负责吸水、运输、光合作用，维持植物生命）
- **生殖器官**：花、果实、种子（负责繁殖后代）

**器官 vs 组织**

| 层次 | 例子 |
|------|------|
| 细胞 | 心肌细胞 |
| 组织 | 心肌组织 |
| 器官 | 心脏（心肌 + 上皮 + 结缔组织等） |

## 🎓 给 12+（进阶版）

### 定义

器官（organ）是由两种或两种以上组织按一定的空间排列方式组成，具有特定形态结构，能执行一种或多种生理功能的独立结构单位。

### 植物六大器官

**营养器官**：

| 器官 | 主要组织 | 主要功能 |
|------|---------|---------|
| 根 | 分生组织、输导组织、保护组织 | 吸收水分和无机盐，固定植株 |
| 茎 | 输导组织、分生组织、保护组织 | 运输水分和有机物，支撑叶和花 |
| 叶 | 营养组织（叶肉）、保护组织（表皮）、输导组织（叶脉） | 光合作用，蒸腾作用 |

**生殖器官**：

| 器官 | 主要功能 |
|------|---------|
| 花 | 有性生殖的场所；花粉（精子）+胚珠（卵细胞） |
| 果实 | 保护种子，参与传播 |
| 种子 | 含胚，是新植物体的雏形 |

### 动物/人体器官举例

**消化系统器官**：口腔、食道、胃、小肠、大肠、肝脏（最大内脏器官）、胰腺

**皮肤**：人体最大的器官，由上皮组织、结缔组织、神经组织、肌肉组织共同构成

**复合功能器官**：肾脏同时完成排泄（过滤血液）和内分泌（分泌促红细胞生成素）功能

### 器官层次在生命结构中的位置

细胞 → 组织 → 器官 → 系统 → 个体

植物无"系统"层次（直接：器官 → 个体）

### 典型例题

**例题**：下列哪个不是器官？
A. 叶片  B. 叶肉细胞  C. 心脏  D. 肾脏

答案：B。叶肉细胞是细胞，不是器官；叶片（含多种组织）才是器官。

### 易错点

1. 植物**没有系统**，器官之上直接是个体（有些教材说植物由根茎叶花果种六器官组成个体）
2. 花、果实、种子属于**生殖器官**，不是营养器官
3. 番茄（西红柿）我们吃的部分是**果实**（肉质果），不是根或茎
4. 器官必须含有**多种组织**，单种组织聚集仍是组织，不是器官
