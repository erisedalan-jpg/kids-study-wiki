# 系统

> **一句话**：系统是由能共同完成一种或几种生理功能的多个器官按一定次序组合而成的结构层次，是动物体结构层次中器官之上、个体之下的层级。
> **English**: An organ system is a group of organs that work together to perform one or more related physiological functions, representing the level of organization between organs and the whole organism in animals.

---

## 📚 给 6-12 岁（自读版）

我们的身体就像一台精密的机器，由许多不同的"部门"协同运作——这些部门就叫**系统**！

**人体有八大系统**

| 系统 | 主要器官 | 主要功能 |
|------|---------|---------|
| 消化系统 | 口、食道、胃、肠、肝 | 消化食物，吸收营养 |
| 呼吸系统 | 鼻、气管、肺 | 吸入氧气，呼出二氧化碳 |
| 循环系统 | 心脏、血管、血液 | 运输养料、氧气和废物 |
| 神经系统 | 大脑、脊髓、神经 | 感受刺激，协调全身 |
| 运动系统 | 骨骼、肌肉、关节 | 运动和支撑身体 |
| 泌尿系统 | 肾脏、输尿管、膀胱 | 过滤血液，排出废物（尿液） |
| 生殖系统 | 睾丸/卵巢、子宫等 | 繁殖后代 |
| 内分泌系统 | 甲状腺、肾上腺、胰岛 | 分泌激素，调节代谢 |

（人体实际上有超过八大系统，教材版本不同略有出入）

**举个例子：你跑步时，哪些系统在合作？**

- 🦴 **运动系统**：腿部肌肉和骨骼产生运动
- 💗 **循环系统**：加快心跳，把氧气和营养送到肌肉
- 🫁 **呼吸系统**：加深呼吸，提供更多氧气
- 🧠 **神经系统**：发出"快跑"的指令，协调各部位

这就是系统协同合作的美妙之处！

**植物有系统吗？**

植物**没有系统**这个层次。植物的器官（根、茎、叶等）直接组成整个植物体，不需要"系统"这一中间层级。

## 🎓 给 12+（进阶版）

### 定义

系统（organ system）是由执行相关生理功能的多个器官有机组合而成的高级结构层次，是多细胞动物体结构和功能的重要组织层次。植物体无系统层次。

### 人体主要系统详解

**消化系统**
- 组成：口腔→食道→胃→小肠（主要吸收）→大肠→肛门 + 肝脏（胆汁）+ 胰腺（消化酶）
- 功能：物理消化 + 化学消化 + 营养吸收

**循环系统**
- 体循环：左心室→主动脉→全身毛细血管→腔静脉→右心房
- 肺循环：右心室→肺动脉→肺泡毛细血管→肺静脉→左心房
- 包括淋巴系统（组织液→淋巴→静脉）

**神经系统**
- 中枢神经系统：脑（大脑、小脑、脑干）+ 脊髓
- 周围神经系统：脑神经 + 脊神经
- 自主神经系统：交感神经（应急加速）+ 副交感神经（休息减速）

**内分泌系统（与神经系统协调）**
- 分级调节：下丘脑→垂体→靶腺体（甲状腺、肾上腺、性腺等）
- 负反馈调节维持激素水平稳定

### 系统在生命结构层次中的位置

细胞 → 组织 → 器官 → **系统** → 个体（动物）

植物：细胞 → 组织 → 器官 → 个体（无系统层次）

### 系统间的整合

各系统并非孤立运作，而是通过**神经-体液-免疫调节**进行整合：
- 神经系统：快速、精确、短暂
- 内分泌系统（体液调节）：缓慢、弥散、持久
- 免疫系统：监控病原体和异常细胞，维持内环境稳态

### 易错点

1. **植物没有系统**，这是植物与动物结构层次的重要区别
2. 内分泌腺（如甲状腺）属于**内分泌系统**，不是消化或循环系统
3. 皮肤既是运动系统（保护层）的一部分，也参与内分泌（合成维生素 D），不能简单归为一个系统
4. "生命系统的结构层次"最高层是**生物圈**，不是个体
