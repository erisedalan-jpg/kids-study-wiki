# 根毛

> **一句话**：根毛是根尖成熟区表皮细胞向外突起形成的单细胞结构，大量增加根的吸收面积，是植物吸收水分和无机盐的主要部位。
> **English**: Root hairs are single-cell extensions of epidermal cells in the root maturation zone that greatly increase surface area for absorbing water and minerals from the soil.

---

## 📚 给 6-12 岁（自读版）

你有没有见过刚发芽的豆苗？把它从土里轻轻拔出来，会看到根上有很多白色的细毛——那就是**根毛**！

**根毛是什么？**

根毛不是真正的"毛发"，而是根的最外层细胞（表皮细胞）向外突出形成的细长管状结构，每一根"毛"都是**一个细胞**。

根毛极其微小，肉眼刚好能看见；用显微镜看，才能看清它是一个个细长的细胞。

**根毛有什么用？**

最重要的作用：**吸收水分和无机盐**！

- 根毛**细长**，能插入土壤颗粒之间，紧贴土壤
- 大量根毛大大增加了根与土壤接触的**表面积**，就像把一张纸揉成皱皱的纸球，表面积增加了很多
- 根毛细胞的细胞液浓度较高，土壤中的水分会自动往浓度高的方向流，进入根毛细胞（渗透作用）

**根毛的特点**

| 特点 | 说明 |
|------|------|
| 位置 | 根尖成熟区（根毛区）的表皮 |
| 结构 | 单细胞，细长管状 |
| 寿命 | 很短（几天~几周），不断更新 |
| 颜色 | 无色透明 |
| 功能 | 吸收水分和无机盐 |

**小实验**：把一粒黄豆浸湿后放在湿棉花上，两三天后你就能看到白色的根毛出现在根的表面！

## 🎓 给 12+（进阶版）

### 定义

根毛（root hair）是根尖成熟区（根毛区）表皮细胞细胞壁向外延伸形成的管状突起，是一种**单细胞**结构。根毛是植物吸收水分和无机盐的主要场所。

### 根毛的结构特点

| 特征 | 说明 |
|------|------|
| 细胞类型 | 单细胞（表皮细胞的延伸） |
| 细胞壁 | 薄，半透性 |
| 液泡 | 大，细胞液浓度高 |
| 细胞核 | 常位于根毛顶端 |
| 长度 | 0.08-1.5 mm |
| 寿命 | 数天至数周 |
| 数量 | 每平方厘米可达 200-300 根 |

**增加表面积的意义**：一棵黑麦草的根毛总数约 143 亿根，总长度约 10,600 km，总表面积约 400 m²——远超没有根毛时的根系表面积。

### 根毛吸水的机制

**渗透作用（osmosis）**：

根毛细胞液浓度 > 土壤溶液浓度 → 水分子从低浓度（土壤）经半透膜流向高浓度（细胞液）→ 根毛吸水

水分从根毛进入后，沿以下路径向上运输：
根毛细胞 → 皮层细胞（胞间连丝/质外体）→ 内皮层（凯氏带过滤）→ 维管柱（木质部导管）→ 茎 → 叶

**土壤溶液浓度过高的后果**：
- 土壤盐浓度 > 细胞液浓度 → 根毛细胞失水 → 植物萎蔫（"烧苗"现象）
- 施肥过量或土地盐碱化导致植物死亡，原理在此

### 无机盐的吸收

- 方式：主动运输（active transport）
- 需要载体蛋白（transport protein）和 ATP 能量
- 可逆浓度梯度吸收，选择性强
- 无机盐进入根毛后，随水流经木质部向上运输

### 根毛与菌根（mycorrhiza）

很多植物的根与真菌共生形成菌根：
- 真菌菌丝延伸根的吸收面积（比根毛还细）
- 真菌获得植物提供的有机物
- 贫瘠土壤中，菌根对矿物质（特别是磷）吸收至关重要
- 大多数陆生植物（包括农作物、森林树木）都有菌根共生

### 易错点

1. 根毛是**表皮细胞**向外延伸，不是独立的细胞分裂出来的新细胞
2. 根毛区（成熟区）才是主要**吸收**区，分生区是**分裂**区，伸长区是**伸长**区
3. 根毛细胞**没有叶绿体**（无色透明，在土壤中无光），不能进行光合作用
4. "施肥过量烧苗"是**渗透失水**，不是"毒性"直接杀死

### 例句

- Root hairs are single-celled extensions that increase the surface area of the root for water absorption.（根毛是增加根表面积以吸收水分的单细胞延伸结构。）
- When soil salinity is too high, root hair cells lose water by osmosis, causing the plant to wilt.（当土壤盐度过高时，根毛细胞通过渗透作用失水，导致植物萎蔫。）
