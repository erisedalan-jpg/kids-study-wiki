# 营养结构

> **一句话**：生态系统中各种生物之间通过食物关系形成的营养层级结构，由食物链和食物网组成。
> **English**: Trophic structure refers to the feeding hierarchy in an ecosystem formed by food chains and food webs.

---

## 📚 给 6-12 岁（自读版）

生态系统里"谁吃谁"的关系，整理出来就叫**营养结构**。

主要部分：
- **生产者**（草、树）：自己造食物
- **消费者**（动物）：吃别的生物
  - 一级消费者：吃草的（兔、牛）
  - 二级消费者：吃肉的（狐、蛇）
  - 三级消费者：吃肉食动物的（鹰、虎）
- **分解者**（细菌、真菌、蚯蚓）：分解尸体和粪便

它们的连接构成食物链 → 食物网 → 整个营养结构。

**像一座"金字塔"：** 越往上数量越少
- 最底层 草最多
- 中间 兔较多
- 上面 狐较少
- 顶层 鹰最少

## 🎓 给 12+（进阶版）

### 定义

**营养结构**：生态系统中生物之间通过取食与被取食关系形成的、表征物质循环和能量流动通路的组织结构，主要由**食物链**和**食物网**构成。

### 组成

| 组成 | 角色 | 例子 |
|------|------|------|
| 生产者 | 自养（光合/化能） | 绿色植物、藻类、化能合成细菌 |
| 初级消费者 | 植食 | 蝗虫、兔、牛 |
| 二级消费者 | 一次肉食 | 蛇、狐 |
| 三级消费者 | 顶级肉食 | 鹰、虎 |
| 分解者 | 分解残体 | 细菌、真菌 |

### 食物链书写示例

```
草 → 蝗虫 → 食虫鸟 → 鹰
(第一)  (第二)  (第三)  (第四营养级)
```

### 能量金字塔

| 营养级 | 生物量 | 能量 |
|------|------|------|
| 第四 | 极少 | 极少 |
| 第三 | 较少 | 较少 |
| 第二 | 较多 | 较多 |
| 第一 | 最多 | 最多 |

每上升一营养级，能量减少 80-90%（10-20% 传递效率）。

### 食物网的特点

- 多条食物链交织
- 多营养级耦合
- 关键种缺失会引发"营养级联"
- 越复杂越稳定

### 营养结构与生态系统功能

- **物质循环**：CO₂、N、P 等元素通过营养结构在生物-环境间循环
- **能量流动**：太阳能 → 生产者 → 各级消费者；单向流动，逐级递减
- **生态平衡**：稳定的营养结构是生态平衡基础

### 人类活动的影响

- 过度捕捞 → 海洋顶级捕食者减少 → 食物网失衡
- 引入外来种 → 破坏原有营养结构
- 农药使用 → 食物链生物富集（DDT 在猛禽体内积累）

### 易错点

- 营养结构不仅指食物链，还包括食物网
- 分解者不属于食物链，但属于生态系统的关键组成
- 同一物种在不同食物链中可能处于不同营养级
