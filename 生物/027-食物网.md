# 食物网

> **一句话**：在生态系统中，多条食物链相互交叉、彼此连接形成的网状结构。
> **English**: A food web is a network of interconnected food chains in an ecosystem.

---

## 📚 给 6-12 岁（自读版）

森林里的"谁吃谁"很复杂：
- 草 → 兔 → 狐
- 草 → 鼠 → 狐
- 草 → 鼠 → 蛇 → 鹰
- 草 → 兔 → 鹰

许多条食物链交叉在一起，形成一张网——这就是**食物网**！

**为什么是网，不是一条线？**
- 一种生物常吃多种食物（狐既吃兔也吃鼠）
- 一种生物常被多种生物吃（兔被狐、被鹰都吃）

**食物网的意义：**
- 食物网越复杂，生态系统越**稳定**
- 一种生物突然减少，其他路线还能继续——不会"全军覆没"
- 草原上如果只有"草→兔→狐"一条链，狐少了兔就泛滥，链断了所有都受影响

## 🎓 给 12+（进阶版）

### 定义

**食物网**：在一个生态系统中，许多条食物链相互连接、交叉形成的复杂网状营养关系。

### 食物链 vs 食物网

| 项目 | 食物链 | 食物网 |
|------|------|------|
| 形态 | 线性 | 网状 |
| 关系 | 单一捕食路径 | 多条路径相互交叉 |
| 稳定性 | 弱（断一节全断） | 强（多路径冗余） |

### 书写规则（食物链 → 食物网）

1. 起点：**生产者**（绿色植物）
2. 箭头由被食者指向食者（→ 表示物质和能量流向）
3. **不写分解者**（分解者另作用，不在食物链中）
4. **顶级消费者**（如鹰、虎）位于网的最高位
5. 同一物种可同时是初级、二级、三级消费者（视所吃对象而定）

### 营养级与能量传递

- **营养级**：食物链上每一环节
  - 第一营养级 = 生产者
  - 第二营养级 = 初级消费者（植食动物）
  - 第三营养级 = 二级消费者（肉食动物）
  - 高级 = 顶级消费者
- **能量传递效率**约 10-20%（高中详讲）：上一级 → 下一级 损失大量能量
- **越高营养级，生物量越少**（能量金字塔）

### 食物网的稳定性

- 网越复杂、物种越多 → 稳定性越强
- 移除一种关键种 → 食物网剧烈波动 → "营养级联效应"
- 农田生态系统食物网简单 → 易暴发害虫
- 热带雨林食物网最复杂 → 自调节能力强

### 典型例题

**例**：草原食物网中，"草 → 鼠 → 蛇 → 鹰" 与 "草 → 兔 → 鹰"，若鹰大量减少：
- 短期：蛇、兔、鼠数量增加 → 草减少
- 长期：草不足 → 鼠、兔减少 → 蛇也少 → 食物网逐步重新平衡

### 易错点

- 食物链不写分解者；但分解者是**生态系统不可或缺**的成分
- 同一物种在不同食物链中营养级可不同
- 食物网越复杂越稳定（物种丰富→冗余多）
