# 蒸腾作用

> **一句话**：植物体内水分通过气孔以水蒸气形式散失到大气中的过程。
> **English**: Transpiration is the process by which water inside a plant evaporates and is released as water vapor through its stomata.

---

## 📚 给 6-12 岁（自读版）

夏天树下为什么凉快？因为大树在"出汗"——这就是**蒸腾作用**！

植物的根从泥土里吸水，水沿着茎里的"吸管"（导管）一路往上爬，到达叶子。叶子上有许多看不见的小孔（气孔），水变成水蒸气，从这些小孔飘到空中。

**简单类比：** 像晾湿衣服一样，植物把水"晾"到空气里。

**有什么用？**
- 把水从根吸到树梢——大树几十米高，全靠蒸腾"拉"水上去
- 给叶片"降温"——夏天阳光晒叶子，蒸腾让叶子不至于晒坏
- 把空气变得更湿润——所以森林里湿度比沙漠大

**小实验**：给一根带叶子的植物枝条套个透明塑料袋，几小时后袋子内壁会出现一层小水珠——那就是蒸腾散出的水汽。

## 🎓 给 12+（进阶版）

### 定义

**蒸腾作用**：植物体内的水分以水蒸气状态通过植物体（主要是叶片气孔）散失到大气中的生理过程。

### 过程与结构

**水的运输路径：**
土壤水 → 根毛吸水 → 根的导管 → 茎的导管 → 叶脉 → 叶肉细胞 → 气孔 → 大气

**关键结构：**

| 部位 | 功能 |
|------|------|
| 根毛 | 吸水主要部位，扩大吸收面积 |
| 导管 | 自下而上输送水和无机盐 |
| 叶脉 | 叶片中导管延伸 |
| 气孔 | 蒸腾散水/气体交换通道 |
| 保卫细胞 | 控制气孔开闭 |

### 影响因素

| 因素 | 蒸腾速率影响 |
|------|------------|
| 光照 | 光强→气孔开→蒸腾增 |
| 温度 | 温度高→水蒸发快→蒸腾增 |
| 湿度 | 空气湿度大→蒸腾减 |
| 风速 | 风大→水汽扩散快→蒸腾增 |
| 叶面积 | 叶面积大→蒸腾多 |

### 生物学意义

1. **运输动力**：是植物吸水和水分上升的主要动力（蒸腾拉力）
2. **降温作用**：水汽化吸热，避免叶片灼伤
3. **促进无机盐运输**：水流带动溶解的矿质离子上行
4. **调节气候**：森林蒸腾增加大气湿度，影响降水

### 典型实验

**枝条蒸腾实验**：将带叶枝条插入有水试管，水面滴一层油防蒸发。一段时间后水位下降——证明枝条失水通过蒸腾。

**对照实验：剥去/不剥去蜡质层**或**摘除/保留叶片**——证明蒸腾主要发生在叶片。

### 易错点

- 蒸腾≠呼吸：蒸腾是水的物理散失，不消耗有机物；呼吸消耗有机物释放能量
- 气孔开闭：白天光合活跃→气孔开（蒸腾旺盛）；夜间多关闭
- 沙漠植物气孔常凹陷或仅夜间开（CAM 植物），减少水分散失
