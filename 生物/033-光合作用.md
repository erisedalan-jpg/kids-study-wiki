# 光合作用

> **一句话**：绿色植物利用光能，把二氧化碳和水转化为有机物（主要是淀粉），同时释放氧气的过程。
> **English**: Photosynthesis is the process by which green plants use light energy to convert CO₂ and water into organic compounds and release oxygen.

---

## 📚 给 6-12 岁（自读版）

植物自己会"做饭"！它们不用嘴吃，而是用叶子"吃光"——这就是**光合作用**。

**做菜步骤：**
1. 叶子里有绿色小颗粒（叶绿体），里面装着叶绿素
2. 叶绿素吸收阳光的能量
3. 叶子从空气里拿 CO₂（通过气孔）
4. 根从土里吸水，水送到叶子
5. 叶绿体把 CO₂ 和水"做"成淀粉（食物），还顺便放出氧气

**记住公式：**
**水 + 二氧化碳 + 阳光 → 有机物 + 氧气**

**为什么这么重要？**
- 制造食物：所有动物（包括人）的食物，归根到底都来自植物光合
- 制造氧气：我们呼吸的氧气，几乎都是植物造的
- 净化空气：吸收 CO₂，减少温室效应

**小实验**：把绿叶放暗处一晚（消耗叶里原有淀粉）→ 部分用黑纸遮光、部分见光 → 几小时后摘叶 → 滴碘液 → 见光部分变蓝（有淀粉），遮光部分不变蓝 → 证明光合需要光、产物是淀粉。

## 🎓 给 12+（进阶版）

### 定义

**光合作用**：绿色植物通过叶绿体，利用光能把无机物（CO₂ 和水）合成有机物（葡萄糖等），并把光能转化为有机物中化学能、释放 O₂ 的过程。

### 化学方程式（初中简化版）

$$\text{CO}_2 + \text{H}_2\text{O} \xrightarrow{\text{光,叶绿体}} \text{有机物(储存能量)} + \text{O}_2$$

### 反应物 / 产物 / 条件 / 场所

| 项目 | 内容 |
|------|------|
| 反应物 | CO₂（来自空气，气孔进入）+ H₂O（来自土壤，根吸收） |
| 产物 | 有机物（主要是淀粉、葡萄糖）+ O₂ |
| 条件 | 光（必需）、叶绿素（捕获光能） |
| 场所 | 叶绿体（叶肉细胞中含量最多） |

### 影响因素

| 因素 | 影响 |
|------|------|
| 光照强度 | 光强→光合速率增（直至饱和） |
| CO₂ 浓度 | 浓度增→光合速率增（直至饱和） |
| 温度 | 适宜温度光合最强（一般 25-30℃） |
| 水分 | 缺水→气孔关闭→光合下降 |
| 矿质 | Mg（叶绿素核心）/ N（蛋白质）等不足→光合减弱 |

### 生物学意义

1. **物质转化**：将无机物合成有机物，是地球上几乎所有生物食物来源
2. **能量转化**：光能 → 化学能（储存在有机物中）
3. **维持碳-氧平衡**：吸 CO₂、放 O₂
4. **气候调节**：固定 CO₂，缓解温室效应

### 典型实验

**1. 验证光合需要光**（萨克斯实验）：
- 暗处放置 24 h（消耗原有淀粉）
- 部分遮光、部分见光
- 酒精脱色 → 碘液 → 见光部分变蓝

**2. 验证光合产生 O₂**：
- 金鱼藻倒置漏斗收集气体
- 带火星木条复燃 → 证明是 O₂

**3. 验证光合需要 CO₂**：
- 两组叶片：一组与 NaOH（吸 CO₂）密封，另一组与 NaHCO₃（放 CO₂）
- 见光、脱色、碘液 → 后者变蓝，前者不变蓝

### 易错点

- 只有含叶绿体的细胞才能光合（不仅是叶片，幼茎、绿果也可）
- 光合作用 ≠ 呼吸作用：前者储能，后者放能；前者吸 CO₂，后者放 CO₂
- 黑暗中无光合，但呼吸照常进行
