# 液泡

> **一句话**：液泡是植物细胞中充满细胞液的囊状结构，具有储存物质、维持细胞形态和调节渗透压的功能。
> **English**: A vacuole is a membrane-bound sac in plant cells filled with cell sap, storing substances and maintaining cell shape through osmotic pressure.

---

## 📚 给 6-12 岁（自读版）

你有没有想过，为什么西瓜又甜又多汁？为什么花朵有各种漂亮的颜色？答案藏在植物细胞里的**液泡**里！

**液泡是什么？**

液泡就像一个大水球，外面包着一层薄薄的膜（叫"液泡膜"），里面充满了叫**细胞液**的液体。细胞液不只是水，里面还溶解着各种物质：糖、无机盐、色素等。

**液泡有什么用？**

1. **储存养料**：糖分、维生素都可以储存在液泡里。西瓜甜，就是因为液泡里存了大量糖分！

2. **存放色素**：花朵的红色、蓝色、紫色，大多是液泡里的**花青素**带来的。不同酸碱度下花青素颜色不同，这就是为什么有些花会变色。

3. **维持形态**：液泡里的液体对细胞壁产生压力（叫**细胞液压**），让植物细胞保持饱满，茎叶挺立。你见过缺水的植物叶子蔫了吧？那就是液泡里水少了、压力不够了。

**植物 vs 动物**

植物细胞有大液泡，成熟植物细胞的液泡可以占细胞体积的 90%！动物细胞也有液泡，但很小很多，不像植物那样显眼。

## 🎓 给 12+（进阶版）

### 定义

液泡（vacuole）是由单层膜（液泡膜，tonoplast）包围的细胞器，内含**细胞液**（cell sap），是植物细胞特征性的大型细胞器。成熟植物细胞通常具有一个占细胞体积大部分的中央液泡（central vacuole）。

### 结构与成分

| 组成 | 说明 |
|------|------|
| 液泡膜（tonoplast） | 单层磷脂双分子层，选择透过性强，调节物质进出 |
| 细胞液（cell sap） | 水 + 无机盐 + 糖类 + 有机酸 + 色素 + 生物碱等 |
| 花青素（anthocyanin） | 水溶性色素，赋予花、果实红/蓝/紫色；受 pH 影响显色 |

**液泡发育**：幼嫩细胞有多个小液泡，随细胞成熟合并为大的中央液泡，体积可达细胞的 90% 以上。

### 渗透作用与液泡的关系

液泡膜和细胞膜共同构成**原生质层**，原生质层的存在使植物细胞具有渗透性：

- **吸水**：外界溶液浓度 < 细胞液浓度 → 水分子净流入液泡 → 细胞膨胀（turgor pressure ↑）
- **失水**：外界溶液浓度 > 细胞液浓度 → 水分子净流出液泡 → **质壁分离**（plasmolysis）
- **质壁分离复原**：将质壁分离的细胞放回低浓度溶液，液泡重新吸水恢复

**膨压（turgor pressure）**：液泡对细胞壁施加的向外压力，维持植物细胞形态，支撑非木质化茎叶。

### 典型实验

**实验：质壁分离与复原**
1. 制作洋葱内表皮临时装片，观察正常状态（细胞液紫色，液泡大）
2. 滴加 30% 蔗糖溶液 → 液泡缩小，原生质层与细胞壁分离（质壁分离）
3. 用清水替换蔗糖溶液 → 液泡重新胀大，原生质层恢复贴合（质壁分离复原）
4. 结论：液泡膜具有选择透过性；液泡参与细胞渗透压调节

### 易错点

1. **液泡膜 ≠ 细胞膜**：液泡膜是围绕液泡的单层膜，细胞膜是围绕整个细胞的膜
2. **细胞液 ≠ 细胞质基质**：细胞液在液泡内，细胞质基质在液泡外的细胞质中
3. 动物细胞也有液泡，但极小，功能与植物液泡不同（如溶酶体也属广义液泡）
4. 死细胞无法进行质壁分离，因为原生质层失去了选择透过性
