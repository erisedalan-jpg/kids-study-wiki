# 植物细胞

> **一句话**：构成植物体的基本单位，具有细胞壁、细胞膜、细胞质、细胞核，以及植物特有的叶绿体、液泡等结构。
> **English**: The basic unit of plant organisms, characterized by a cell wall, cell membrane, cytoplasm, nucleus, and plant-specific organelles such as chloroplasts and vacuoles.

---

## 📚 给 6-12 岁（自读版）

植物细胞和动物细胞（比如人体细胞）有些共同点，也有一些重要的不同。

**植物细胞特有的结构：**

**1. 细胞壁（最外层）**
植物细胞外面有一层硬硬的"砖墙"，叫细胞壁，由纤维素构成。
- 作用：支撑和保护细胞，保持细胞形态
- 就像城堡的外墙，坚固有形
- 动物细胞没有细胞壁，所以动物细胞形态多变（比如红细胞是饼形，神经细胞有很多突起）

**2. 叶绿体（绿色细胞里才有）**
叶片里的细胞含有绿色的"太阳能板"——叶绿体。
- 里面含有叶绿素，能吸收阳光
- 进行光合作用，把水和二氧化碳变成糖和氧气
- 根部细胞、洋葱白色鳞叶细胞没有叶绿体（因为不见光）

**3. 液泡（植物细胞中央的大"水袋"）**
成熟植物细胞有一个大液泡，里面充满细胞液。
- 储存水分、糖分、色素（花瓣的颜色就来自液泡里的色素！）
- 维持细胞的形态（吸水膨胀让植物茎叶保持挺立）
- 植物缺水时，液泡变小，植物就会"蔫"

**植物细胞也有的结构：**
- 细胞膜：控制物质进出（在细胞壁内侧）
- 细胞质：液态环境，各种化学反应在这里进行
- 细胞核：遗传信息的储存中心

**植物细胞形态多样：**
- 叶肉细胞（有叶绿体，进行光合作用）
- 根毛细胞（细长，吸收水分）
- 导管细胞（空心管道，运输水分）
- 保卫细胞（控制气孔开关）

## 🎓 给 12+（进阶版）

### 定义

**植物细胞**（plant cell）是构成植物体的基本单位，属于真核细胞，具有典型的真核细胞结构，同时拥有植物特有的细胞壁、叶绿体和大型中央液泡。

### 结构 / 原理

**植物细胞结构总览：**

| 结构 | 化学成分 | 功能 | 植物特有？ |
|------|---------|------|-----------|
| 细胞壁 | 纤维素（主）、果胶 | 支持保护，维持形态 | 是 |
| 细胞膜 | 磷脂双分子层+蛋白质 | 选择透过性，控制进出 | 否 |
| 细胞核 | DNA+蛋白质 | 遗传信息中心，调控代谢 | 否 |
| 细胞质 | 基质+细胞器 | 代谢场所 | 否 |
| 叶绿体 | 基粒+基质+类囊体 | 光合作用（光能→化学能） | 是（含叶绿素细胞） |
| 液泡 | 细胞液（水+溶质） | 储存营养，维持渗透压 | 是（成熟细胞有大液泡） |
| 线粒体 | 双层膜+基质 | 细胞呼吸（产生ATP） | 否 |
| 内质网 | 膜系统 | 物质加工运输 | 否 |
| 核糖体 | 蛋白质+rRNA | 蛋白质合成 | 否 |

**植物细胞 vs 动物细胞：**

| 结构 | 植物细胞 | 动物细胞 |
|------|---------|---------|
| 细胞壁 | 有 | 无 |
| 叶绿体 | 含叶绿素的细胞有 | 无 |
| 中央液泡 | 成熟细胞有大液泡 | 无或极小 |
| 中心体 | 高等植物无 | 有（参与分裂） |
| 细胞形态 | 较规则 | 多变 |

**细胞壁的通透性：**
细胞壁是**全透性**（任何物质均可通过），细胞膜是**选择透过性**（主动筛选物质进出）。

### 典型实验 / 例题

**观察植物细胞的质壁分离：**
- 将洋葱表皮细胞放入 30% 蔗糖溶液（高渗）
- 细胞液浓度低于外液 → 水分外渗 → 细胞质和液泡缩小 → 细胞膜（+原生质层）离开细胞壁（质壁分离）
- 重新放回清水 → 质壁分离复原

**例题**：下列细胞中，既有细胞壁又有叶绿体的是？
A. 洋葱根尖细胞 B. 洋葱鳞叶表皮细胞 C. 菠菜叶肉细胞 D. 人口腔上皮细胞
**答案：C**（A无叶绿体；B无叶绿体（白色，不见光）；D无细胞壁）

### 易错点

1. **洋葱鳞叶（白色）无叶绿体**，不能进行光合作用，不要误以为所有植物细胞都有叶绿体。
2. **液泡不是只有植物才有**，但只有植物成熟细胞有**大型中央液泡**。
3. **细胞壁不等于细胞膜**：细胞壁在外，全透性；细胞膜在内，选择透过性。
