# 肺

> **一句话**：肺是呼吸系统的主要器官，通过肺泡与毛细血管进行气体交换，完成O₂摄入和CO₂排出。
> **English**: The lungs are the main organs of the respiratory system, performing gas exchange between alveoli and capillaries to take in O₂ and release CO₂.

---

## 📚 给 6-12 岁（自读版）

肺是你身体里最重要的呼吸器官，就像两个大"气球"，位于胸腔里，心脏的两侧。你有**左肺**和**右肺**，右肺比左肺略大（因为心脏偏向左侧）。

**肺长什么样？**
肺里面并不是空心的大空腔，而是由数以亿计的微小气泡——**肺泡**——组成的。肺泡非常非常小，但数量极多（约3~4亿个！），展开后总面积相当于一个网球场大小（约70平方米）。

**肺泡在做什么？**
每个肺泡外面都包裹着一层密密的毛细血管。当我们吸气时：
- 空气中的**氧气**穿过肺泡壁进入毛细血管中的血液
- 血液里的**二氧化碳**穿过毛细血管壁进入肺泡
呼气时，含有二氧化碳的空气被呼出体外。

**肺的特点：**
- 肺泡壁很薄（只有一层上皮细胞）
- 肺泡外面有丰富的毛细血管
- 这两点保证气体可以快速交换

## 🎓 给 12+（进阶版）

### 定义

**肺**（Lung）：呼吸系统的主要器官，位于胸腔内，分为左肺（2叶）和右肺（3叶）。肺通过**肺泡**（alveoli）与肺毛细血管网之间的气体扩散，实现**外呼吸**（O₂↑CO₂↓）。

### 肺的结构层次

| 结构 | 说明 |
|------|------|
| 支气管 → 细支气管 → 终末细支气管 | 气体传导部分，逐级分支（约23级） |
| 呼吸性细支气管 → 肺泡管 → 肺泡囊 | 过渡区，兼具传导和气体交换 |
| 肺泡（alveoli） | 气体交换的功能单位；约3~4亿个；直径约0.2 mm |
| 肺泡壁 | 单层扁平上皮（Ⅰ型肺泡细胞）+ 分泌表面活性物质的Ⅱ型细胞 |
| 肺毛细血管网 | 紧贴肺泡外壁；血气屏障约0.5 μm |

### 气体交换（外呼吸）

气体交换基于**分压差**驱动的扩散：

| 气体 | 肺泡气分压 | 静脉血分压 | 扩散方向 |
|------|---------|---------|---------|
| O₂ | 100 mmHg | 40 mmHg | 肺泡 → 血液 |
| CO₂ | 40 mmHg | 46 mmHg | 血液 → 肺泡 |

### 肺泡气体交换的有利条件

| 条件 | 说明 |
|------|------|
| 面积大 | 约70 m²（约网球场大小） |
| 壁极薄 | 血气屏障仅约0.5 μm |
| 毛细血管丰富 | 每个肺泡外紧密包裹毛细血管 |
| 气血流量匹配 | 通气/血流比约0.8 |

### 肺表面活性物质

Ⅱ型肺泡细胞分泌**磷脂类表面活性物质**（surfactant）：
- 降低肺泡表面张力，防止肺泡塌陷
- 早产儿缺乏surfactant → 新生儿呼吸窘迫综合征（NRDS）

### 典型例题

**例题**：肺泡适于气体交换的结构特点有哪些？

**答案**：
① 肺泡数量极多（约3~4亿），总面积大（约70 m²）；
② 肺泡壁极薄（单层扁平上皮细胞）；
③ 肺泡外包绕丰富的毛细血管；
④ 血气屏障（肺泡壁+毛细血管壁）薄，有利于气体快速扩散。

### 易错点

- ❌ 肺泡进行呼吸运动 → ✅ 肺泡进行**气体交换**，呼吸运动靠膈肌和肋间肌
- ❌ 呼出的气体是纯CO₂ → ✅ 呼出气含O₂约16%（比吸入少），CO₂约4%（比吸入多），还有N₂等
- ❌ 肺是平滑的气囊 → ✅ 肺内是海绵状结构，由无数肺泡组成
