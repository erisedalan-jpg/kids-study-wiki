# 吸收

> **一句话**：吸收是消化后的营养物质从消化道进入血液或淋巴的过程，小肠是主要的吸收场所。
> **English**: Absorption is the process by which digested nutrients pass from the digestive tract into the blood or lymph, with the small intestine being the primary site.

---

## 📚 给 6-12 岁（自读版）

食物经过消化变成了小分子（葡萄糖、氨基酸、甘油等），接下来这些小分子需要"穿过"消化道的壁，进入血液，才能被运送到全身各处细胞使用。这个过程就叫做**吸收**。

**不同部位的吸收功能：**

| 部位 | 吸收的物质 |
|------|----------|
| 口腔 | 几乎不吸收（药片含服例外） |
| 胃 | 少量水和酒精 |
| **小肠** | **绝大部分营养物质**（葡萄糖、氨基酸、甘油、脂肪酸、维生素、水、无机盐） |
| 大肠 | 水分和无机盐 |

**小肠为什么是最好的吸收场所？**

小肠太神奇了！它长约6米，内壁上有很多褶皱，褶皱上还有很多小突起叫**小肠绒毛**，绒毛上还有更小的**微绒毛**。这些结构让小肠的吸收面积增大到了约**200平方米**（相当于一个网球场的大小！）。

每个小肠绒毛里都有丰富的毛细血管和毛细淋巴管——葡萄糖和氨基酸进入毛细血管，甘油和脂肪酸先进入毛细淋巴管，之后再进入血液。

## 🎓 给 12+（进阶版）

### 定义

**吸收**（Absorption）：经消化后的营养物质（单糖、氨基酸、甘油和脂肪酸、维生素、水、无机盐等）通过消化道上皮细胞进入**血液循环**或**淋巴系统**的过程。

### 各部位吸收功能

| 部位 | 主要吸收物质 | 机制 |
|------|------------|------|
| 胃 | 少量水、酒精 | 简单扩散 |
| 小肠（主要） | 葡萄糖、氨基酸→毛细血管；甘油+脂肪酸→毛细淋巴管；水、无机盐、维生素 | 主动运输+简单扩散 |
| 大肠 | 水分、无机盐 | 渗透 |

### 小肠吸收的结构基础

小肠具备高效吸收的多层次结构：

| 结构层次 | 作用 |
|---------|------|
| 小肠全长（约6 m） | 延长食物停留时间 |
| 环形皱褶 | 增大内表面积约3倍 |
| 小肠绒毛（villi） | 增大面积约10倍；含毛细血管和毛细淋巴管 |
| 微绒毛（刷状缘） | 增大面积约20倍；含消化酶 |
| 总吸收面积 | 约200 m² |

### 不同营养物质的吸收途径

| 营养物质 | 吸收形式 | 吸收途径 |
|---------|---------|---------|
| 葡萄糖 | 葡萄糖 | 毛细血管 → 门静脉 → 肝脏 |
| 氨基酸 | 氨基酸 | 毛细血管 → 门静脉 → 肝脏 |
| 脂肪 | 甘油 + 脂肪酸 | 先进毛细淋巴管（乳糜管），再汇入血液 |
| 水溶性维生素 | 原形 | 毛细血管 |
| 脂溶性维生素 | 溶于脂肪 | 随脂肪经淋巴 |
| 水 | H₂O | 渗透进入毛细血管 |
| 无机盐 | 离子形式 | 主动运输进入毛细血管 |

### 典型例题

**例题**：脂肪消化的最终产物通过什么途径进入血液？

**答案**：脂肪消化为甘油和脂肪酸后，先被小肠绒毛内的**毛细淋巴管**（乳糜管）吸收，再经淋巴管汇入**血液循环**；而葡萄糖和氨基酸直接进入**毛细血管**。

### 易错点

- ❌ 所有营养物质都经血管吸收 → ✅ 脂肪经**淋巴**吸收，其他经**血管**
- ❌ 大肠是主要吸收场所 → ✅ **小肠**是主要吸收场所
- ❌ 维生素和水不需要吸收 → ✅ 它们同样需要经过小肠壁进入血液
