# 消化

> **一句话**：消化是食物在消化道内被分解为可吸收的小分子物质的过程，分为物理消化和化学消化两种方式。
> **English**: Digestion is the process by which food is broken down in the digestive tract into absorbable small molecules, through both physical and chemical means.

---

## 📚 给 6-12 岁（自读版）

你吃下去的食物，大部分分子都太大了，不能直接被身体利用。消化就是把这些大分子"切碎"成很小的分子，这样才能被小肠吸收进入血液，送到身体各处。

**消化分两种方式：**

**1. 物理消化（机械消化）：**
用"物理力量"把食物变小，但不改变食物的化学成分。
- 牙齿咀嚼：把食物切碎磨细
- 胃和肠道蠕动：混合和移动食物
- 胆汁乳化：把脂肪分散成小液滴

**2. 化学消化：**
用"消化酶"把食物的大分子分解成小分子，改变了化学成分。
- 淀粉（大） → 葡萄糖（小）
- 蛋白质（大） → 氨基酸（小）
- 脂肪（大） → 甘油 + 脂肪酸（小）

**三大营养物质的消化路线：**
| 营养物质 | 开始消化的部位 | 完成消化的部位 | 最终产物 |
|---------|------------|------------|--------|
| 淀粉（糖类） | 口腔 | 小肠 | 葡萄糖 |
| 蛋白质 | 胃 | 小肠 | 氨基酸 |
| 脂肪 | 小肠 | 小肠 | 甘油 + 脂肪酸 |

**小知识**：水、无机盐、维生素不需要消化，可以直接被吸收！

## 🎓 给 12+（进阶版）

### 定义

**消化**（Digestion）：消化系统将摄入的食物通过**物理消化**（mechanical digestion）和**化学消化**（chemical digestion）的协同作用，将大分子营养物质分解为可被吸收的小分子物质的过程。

### 物理消化 vs 化学消化

| 方式 | 机制 | 场所 | 结果 |
|------|------|------|------|
| 物理（机械）消化 | 牙齿咀嚼、肌肉蠕动、胆汁乳化 | 口腔、胃、小肠 | 增大食物与酶的接触面积，不改变化学性质 |
| 化学消化 | 消化酶水解 | 口腔→胃→小肠 | 大分子→小分子，改变化学结构 |

### 三大营养物质的完整消化过程

**糖类（以淀粉为例）：**
```
淀粉 →（唾液淀粉酶，口腔）→ 麦芽糖
麦芽糖 →（麦芽糖酶，小肠）→ 葡萄糖
```

**蛋白质：**
```
蛋白质 →（胃蛋白酶，胃）→ 多肽
多肽 →（胰蛋白酶+肠肽酶，小肠）→ 氨基酸
```

**脂肪：**
```
脂肪 →（胆汁乳化，小肠）→ 脂肪微液滴
脂肪微液滴 →（胰脂肪酶，小肠）→ 甘油 + 脂肪酸
```

### 消化酶的特性

| 特性 | 说明 |
|------|------|
| 专一性 | 每种酶只催化特定底物（如淀粉酶只分解淀粉） |
| 高效性 | 少量酶即可催化大量反应 |
| 温和性 | 在体温（37℃）、适当pH下最活跃 |
| pH依赖性 | 唾液淀粉酶在中性环境最优；胃蛋白酶需强酸（pH 2） |

### 典型例题

**例题**：蛋白质消化从哪个部位开始？最终消化产物是什么？在哪里完成消化？

**答案**：蛋白质消化从**胃**开始（胃蛋白酶）；最终产物是**氨基酸**；在**小肠**（胰蛋白酶和肠肽酶）完成完全消化。

### 易错点

- ❌ 淀粉在胃中开始消化 → ✅ 淀粉在**口腔**就开始消化（唾液淀粉酶）
- ❌ 脂肪由胆汁消化 → ✅ 胆汁**乳化**脂肪（增大接触面积），**胰脂肪酶**才真正消化
- ❌ 口腔可以消化蛋白质 → ✅ 蛋白质的化学消化从**胃**开始
- ❌ 水和维生素需要消化 → ✅ 水、无机盐、维生素不需要消化，直接吸收
