# 消化系统

> **一句话**：消化系统是由消化道和消化腺组成的器官系统，负责食物的消化和营养物质的吸收。
> **English**: The digestive system consists of the digestive tract and digestive glands, responsible for breaking down food and absorbing nutrients.

---

## 📚 给 6-12 岁（自读版）

你吃下的食物，要经过一段"漫长的旅程"才能变成身体能利用的营养。这个旅程所经过的地方，就是**消化系统**。

**消化系统由两部分组成：**

**1. 消化道（食物通过的管道）：**
口腔 → 咽 → 食管 → 胃 → 小肠 → 大肠 → 肛门

- **口腔**：牙齿咀嚼食物，唾液帮助消化淀粉
- **咽和食管**：食物通道，没有消化功能
- **胃**：像一个"搅拌机"，胃液消化蛋白质
- **小肠**：最重要的消化和吸收场所（长约6米！）
- **大肠**：吸收水分，形成粪便
- **肛门**：粪便排出

**2. 消化腺（分泌消化液的器官）：**
- **唾液腺**：分泌唾液（含淀粉酶）
- **胃腺**：分泌胃液（含盐酸和胃蛋白酶）
- **肝脏**：分泌胆汁（帮助乳化脂肪，注意：胆汁不是酶）
- **胰腺**：分泌胰液（含多种消化酶，最全面的消化液）
- **肠腺**：分泌肠液（含多种酶）

**记忆技巧**：食物在消化道里走的路线就是"口、咽、食管、胃、小肠、大肠、肛门"，想象食物去"旅游"的行程！

## 🎓 给 12+（进阶版）

### 定义

**消化系统**（Digestive System）：由**消化道**（alimentary canal）和**消化腺**（digestive glands）组成的器官系统，执行食物摄入、机械消化、化学消化、营养吸收和废物排出等功能。

### 消化道各段功能

| 部位 | 长度 | 主要功能 | 消化酶/液 |
|------|------|---------|---------|
| 口腔 | - | 机械消化；淀粉化学消化开始 | 唾液淀粉酶 |
| 咽 | - | 食物通道（与气道交叉） | 无 |
| 食管 | 约25cm | 蠕动传输食物 | 无 |
| 胃 | 容积约1.5L | 蛋白质消化开始；储存食物 | 胃蛋白酶（盐酸激活） |
| 小肠 | 约6m | **主要消化和吸收场所** | 胰液+肠液+胆汁 |
| 大肠 | 约1.5m | 吸收水分和无机盐，形成粪便 | 肠道菌群酶 |
| 肛门 | - | 粪便排出 | - |

### 消化腺及其分泌物

| 消化腺 | 位置 | 分泌物 | 主要作用 |
|--------|------|--------|---------|
| 唾液腺 | 口腔周围（3对） | 唾液 | 淀粉酶水解淀粉 |
| 胃腺 | 胃壁内 | 胃液（pH 1.5~2.0） | 盐酸激活胃蛋白酶；胃蛋白酶消化蛋白质 |
| 肝脏 | 腹腔右上方 | 胆汁（储存于胆囊） | 乳化脂肪（无酶活性） |
| 胰腺 | 胃后方 | 胰液（pH 7.8~8.4） | 含胰蛋白酶、胰脂肪酶、胰淀粉酶（最全面） |
| 肠腺 | 小肠壁内 | 肠液 | 含多种酶，完成终末消化 |

### 小肠为何是主要吸收场所

小肠具有多重有利于吸收的结构特征：
- 长度约6米，接触面积大
- 内壁有**环形皱褶**，增大面积
- 皱褶上有**小肠绒毛**，绒毛上有**微绒毛**（刷状缘）
- 每个绒毛内含丰富的**毛细血管**和**毛细淋巴管**
- 总吸收面积可达约200 m²（相当于一个网球场）

### 典型例题

**例题**：下列消化腺中，能分泌含有多种消化酶消化液的是（ ）

A. 唾液腺  B. 肝脏  C. 胃腺  D. 胰腺

**答案**：D

**解析**：胰腺分泌的胰液中含有胰蛋白酶、胰脂肪酶和胰淀粉酶，能同时消化三大营养物质，是最全面的消化液。肝脏分泌的胆汁不含消化酶。

### 易错点

- ❌ 胆汁消化脂肪 → ✅ 胆汁乳化脂肪，使其变成小液滴（乳化≠消化）
- ❌ 大肠是主要吸收场所 → ✅ 小肠是主要吸收场所，大肠主要吸收水分
- ❌ 食物在胃内完全消化 → ✅ 胃只消化蛋白质，大部分消化在小肠完成
