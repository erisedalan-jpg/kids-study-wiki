# 激素

> **一句话**：由内分泌腺分泌、随血液运输到全身、对生命活动起调节作用的微量化学物质。
> **English**: Hormones are chemicals secreted by endocrine glands, carried by blood, that regulate body functions in tiny amounts.

---

## 📚 给 6-12 岁（自读版）

身体里有一群"小信使"——它们叫**激素**。

**特点：**
- 由特殊的腺体（叫"内分泌腺"）分泌
- 通过血液送到身体各处
- 量很少，但作用很大
- 不同激素管不同的事

**几种著名的激素：**
- **生长激素**：管长高（小时候缺会矮，过多会巨人症）
- **甲状腺激素**：管新陈代谢（缺会得"大脖子病"）
- **胰岛素**：管血糖（缺会得糖尿病）
- **性激素**：青春期让身体变化（变声、长胡须、来月经等）
- **肾上腺素**：紧急情况让心跳加快、力气变大（"打或逃"反应）

**为什么调节身体不只用神经，还用激素？**
神经反应快但范围小（只到肌肉），激素慢但能广泛影响——两者配合最完美。

## 🎓 给 12+（进阶版）

### 定义

**激素**：由内分泌腺（或某些内分泌细胞）分泌，通过血液（或体液）运输到靶器官、靶细胞，发挥调节作用的微量信号分子。

### 内分泌腺与激素

| 内分泌腺 | 激素 | 主要作用 |
|--------|------|--------|
| 下丘脑 | 多种释放/抑制激素 | 调节垂体 |
| 垂体 | 生长激素、促性腺激素、促甲状腺激素 | 调控其他腺体 |
| 甲状腺 | 甲状腺激素 | 促进新陈代谢、生长 |
| 胰岛 | 胰岛素、胰高血糖素 | 调节血糖 |
| 肾上腺 | 肾上腺素、糖皮质激素 | 应激反应 |
| 卵巢/睾丸 | 雌激素 / 雄激素 | 性发育、第二性征 |

### 化学本质

| 类别 | 例子 |
|------|------|
| 蛋白质类 | 胰岛素、生长激素 |
| 氨基酸衍生物 | 肾上腺素、甲状腺激素 |
| 脂质类（固醇） | 性激素、肾上腺皮质激素 |

### 作用特点

- **微量高效**：极小浓度即起作用
- **特异性**：只作用于具有受体的靶细胞
- **通过受体介导**：与靶细胞膜或胞内受体结合
- **缓慢持久**：起效比神经慢，但维持时间长
- **反馈调节**：激素水平自动维持稳态

### 常见激素病

| 疾病 | 病因 | 症状 |
|------|------|------|
| 糖尿病 | 胰岛素不足或作用障碍 | 血糖高、多饮多尿 |
| 侏儒症 | 幼年期生长激素不足 | 身材矮小但智力正常 |
| 巨人症 / 肢端肥大症 | 生长激素过多 | 身高异常 / 手脚粗大 |
| 呆小症 | 幼年期甲状腺激素不足 | 身材矮小、智力低下 |
| 甲亢 | 甲状腺激素过多 | 心跳快、消瘦、易激动 |

### 神经调节 vs 激素调节

| 项目 | 神经调节 | 激素调节 |
|------|-------|-------|
| 速度 | 快 | 慢 |
| 范围 | 局部、精确 | 广泛 |
| 持续时间 | 短 | 长 |
| 主导关系 | 通常神经主导 | 配合神经 |

### 易错点

- 激素由**内分泌腺**分泌，不通过导管（区别于消化腺等外分泌腺）
- 激素**只对靶细胞有效**——靶细胞具有相应受体
- 激素调节与神经调节**协同作用**，不是相互替代
