# 血浆

> **一句话**：血液中的液体部分，呈淡黄色，约占血液体积的 55%，主要功能是运载血细胞和运输营养物质、废物。
> **English**: Plasma is the pale-yellow liquid component of blood, making up about 55% of blood volume, transporting cells, nutrients, and wastes.

---

## 📚 给 6-12 岁（自读版）

把血液静置一段时间或者离心，会发现它分成三层：
- 上层淡黄色清亮液体 = **血浆**
- 中间薄薄一层白色 = 白细胞 + 血小板
- 下层暗红色 = 红细胞

**血浆做什么？**
- 像"运河"——载着血细胞流动
- 把消化吸收的营养（葡萄糖、氨基酸）送到全身
- 把废物（尿素、CO₂）从细胞带到肾、肺排出
- 调节体温、维持酸碱平衡

**有趣的事：** 血浆 90% 是水！其余是蛋白质、葡萄糖、无机盐、激素等。

## 🎓 给 12+（进阶版）

### 定义

**血浆**：血液中除去血细胞（红细胞、白细胞、血小板）后剩余的液体部分，约占血液体积的 55%。颜色淡黄、半透明，含多种蛋白质、电解质、激素、营养物质和代谢废物。

### 成分

| 成分 | 含量 / 占比 | 功能 |
|------|----------|------|
| 水 | ~90% | 溶剂、运输介质 |
| 血浆蛋白 | ~7% | 维持渗透压、运输物质、免疫、凝血 |
| 葡萄糖 / 氨基酸 / 脂质 | 营养物质 | 供细胞利用 |
| 无机盐（Na⁺/K⁺/Ca²⁺/HCO₃⁻） | ~0.9% | 维持渗透压、酸碱平衡 |
| 代谢废物（尿素、尿酸） | 少量 | 待肾排出 |
| 激素、酶、抗体 | 微量 | 调节、免疫 |

### 主要功能

1. **运载血细胞**：使红/白细胞、血小板流动到全身
2. **运输营养物质**：葡萄糖、氨基酸、脂质、维生素
3. **运输代谢废物**：尿素、尿酸、CO₂（部分）
4. **维持渗透压**：血浆蛋白（主要为白蛋白）维持血浆胶体渗透压
5. **维持酸碱平衡**：HCO₃⁻ / H₂CO₃ 缓冲系统
6. **运输激素**：内分泌激素借血浆送达靶器官
7. **凝血与免疫**：含凝血因子、抗体

### 与组织液、淋巴的关系（高中前奏）

- 血浆 → 透出毛细血管 → 组织液 → 部分进入毛细淋巴管 → 淋巴 → 锁骨下静脉 → 回到血浆
- 三者动态平衡构成内环境

### 临床意义

- 血浆蛋白减少 → 渗透压降 → 组织水肿
- 血浆葡萄糖（血糖）异常 → 糖尿病 / 低血糖
- 输血浆：救治大出血、烧伤患者

### 易错点

- 血浆 ≠ 血清：血浆含纤维蛋白原（凝血因子），血清是凝血后剩余的液体
- 血浆是液体成分；血细胞是细胞成分
- 内分泌的激素通过血浆输送，不通过淋巴
