# 爬行动物
> **一句话**：爬行动物是体表覆盖角质鳞片、用肺呼吸、体温不恒定、在陆地上产卵繁殖的脊椎动物。
> **English**: Reptiles are vertebrates with scaly skin, lung respiration, variable body temperature, and terrestrial egg-laying reproduction.

---

## 📚 给 6-12 岁（自读版）

你见过蜥蜴或蛇吗？它们都是爬行动物！爬行动物的皮肤上覆盖着厚厚的**鳞片**，就像穿了一件"盔甲"，可以防止水分散失，让它们能在干燥的陆地上生活。

爬行动物用**肺**呼吸空气，是真正意义上能完全脱离水生活的脊椎动物。它们是**变温动物**（冷血动物），体温会随环境温度变化——冬天气温低时，它们就会冬眠。

爬行动物在陆地上产**有壳的卵**，卵壳能防止卵干燥，保护里面的胚胎，这是适应陆地生活的重要特征。常见的爬行动物有蜥蜴、蛇、龟、鳄鱼等。

**有趣的小知识**：恐龙也是爬行动物！现代鸟类被认为是从恐龙演化来的，所以有人说鸟是"会飞的爬行动物"。

## 🎓 给 12+（进阶版）

### 定义
爬行动物（Reptilia）是羊膜动物中的一个并系群，体表覆盖角质鳞片，用肺呼吸，心脏三腔（两心房一心室，心室中有不完全隔膜），体温随环境变化（变温动物），在陆地上产羊膜卵。

### 主要特征（表格）

| 特征 | 描述 |
|------|------|
| 体表 | 角质鳞片或骨板，防止水分散失 |
| 呼吸 | 完全用肺呼吸（无皮肤辅助呼吸） |
| 循环 | 三腔心脏（两房一室，部分有不完全隔膜） |
| 体温 | 变温（ectotherm），依赖外界热源 |
| 繁殖 | 体内受精，产羊膜卵（有皮革质或石灰质卵壳） |
| 发育 | 直接发育，无变态 |

### 主要类群

| 类群 | 代表动物 | 特点 |
|------|----------|------|
| 鳄目 | 扬子鳄、尼罗鳄 | 心脏四腔，最接近鸟类 |
| 龟鳖目 | 乌龟、鳖 | 有背腹甲，行动缓慢 |
| 有鳞目-蜥蜴亚目 | 壁虎、蜥蜴 | 四肢发达，有尾巴 |
| 有鳞目-蛇亚目 | 眼镜蛇、蟒蛇 | 无四肢，用腹鳞爬行 |
| 喙头目 | 喙头蜥 | 活化石，仅存于新西兰 |

### 与两栖动物的比较

| 比较项 | 两栖动物 | 爬行动物 |
|--------|----------|----------|
| 皮肤 | 湿润无鳞，辅助呼吸 | 干燥有鳞，防水 |
| 呼吸 | 肺+皮肤 | 仅肺 |
| 繁殖 | 水中产卵（无壳） | 陆地产卵（有壳） |
| 发育 | 有变态（幼体水生） | 直接发育 |
| 与水的关系 | 繁殖必须回水 | 完全陆生 |

### 典型例题
**例题**：判断下列哪项不是爬行动物适应陆地生活的特征？
A. 体表有角质鳞片  B. 用肺呼吸  C. 皮肤湿润可辅助呼吸  D. 产有壳的卵

**解析**：C是两栖动物的特征，爬行动物皮肤干燥有鳞，不能辅助呼吸。**答案：C**

### 易错点
- 蜥蜴≠两栖动物，蜥蜴是爬行动物（皮肤有鳞、完全肺呼吸）
- 爬行动物≠变温动物是缺点——这是适应策略，节省能量
- 鳄鱼心脏已接近四腔，但仍归爬行动物
