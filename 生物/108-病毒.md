# 病毒
> **一句话**：病毒是没有细胞结构的非细胞生命体，由蛋白质外壳和核酸组成，只能在活细胞内寄生繁殖。
> **English**: Viruses are non-cellular life forms without cell structure, composed of a protein coat and nucleic acid, capable of replicating only inside living host cells.

---

## 📚 给 6-12 岁（自读版）

流感病毒、新冠病毒、水痘病毒……**病毒**是让我们生病的"敌人"，但它们其实非常奇特——病毒不是细胞！

病毒比细菌还要小得多（细菌已经很小了！），必须用电子显微镜才能看到。病毒的结构极其简单：
- **外壳**：由蛋白质构成，保护里面的遗传物质
- **核酸**（DNA或RNA）：遗传信息，指导病毒复制

病毒不能自己生存，必须侵入其他生物的细胞内才能繁殖——它就像一个"海盗"，劫持了细胞的"工厂"，大量复制自己，然后冲破细胞，去感染更多细胞。

病毒只感染特定的宿主：
- 流感病毒感染人和动物
- 噬菌体（细菌病毒）感染细菌
- 植物病毒感染植物

## 🎓 给 12+（进阶版）

### 定义
病毒（virus）是一类非细胞生命体，由核酸（DNA或RNA，不能同时含两种）和蛋白质外壳（衣壳）组成。病毒没有细胞结构，不能独立进行代谢，只能在活宿主细胞内利用宿主的酶系统和原料进行复制增殖。

### 病毒的基本结构

| 组成 | 特点 |
|------|------|
| 衣壳（capsid） | 蛋白质亚基（衣壳粒）聚合而成，保护核酸 |
| 核酸 | DNA或RNA（二者之一），单链或双链 |
| 包膜（envelope） | 部分病毒有（来自宿主细胞膜）；流感病毒有，腺病毒无 |
| 刺突（spike） | 包膜上的蛋白，用于识别和结合宿主细胞受体（如新冠S蛋白） |

### 病毒的分类

| 分类依据 | 类型 | 举例 |
|----------|------|------|
| 核酸类型 | DNA病毒 | 天花病毒、乙肝病毒（DNA） |
| | RNA病毒 | 流感病毒、新冠病毒、HIV |
| 宿主类型 | 动物病毒 | 流感病毒、狂犬病毒 |
| | 植物病毒 | 烟草花叶病毒（TMV） |
| | 细菌病毒 | 噬菌体（T4噬菌体） |
| 形态 | 球形 | 流感病毒、HIV |
| | 杆状 | 烟草花叶病毒 |
| | 二十面体 | 腺病毒 |
| | 蝌蚪形 | T4噬菌体 |

### 病毒的增殖过程（以噬菌体为例）
1. **吸附**：噬菌体识别并结合细菌表面受体
2. **注入**：将DNA注入细菌细胞（蛋白质外壳留在外面）
3. **复制**：利用宿主酶和原料，复制噬菌体DNA
4. **合成**：合成噬菌体蛋白质衣壳
5. **组装**：DNA与衣壳组装成新噬菌体
6. **释放**：细菌裂解，释放大量新噬菌体

### 典型例题
**例题**：下列关于病毒的叙述，正确的是？
A. 病毒是最小的细胞生物  B. 病毒同时含有DNA和RNA
C. 病毒没有细胞结构，只能在活细胞内增殖  D. 病毒能在培养基上独立生长

**解析**：病毒没有细胞结构（A错），每种病毒只含一种核酸（B错），必须在宿主活细胞内才能增殖，不能在培养基上独立生长（D错）。**答案：C**

### 易错点
- 病毒无细胞结构，不属于原核生物也不属于真核生物（单独的一类）
- 病毒只含一种核酸：DNA病毒只有DNA，RNA病毒只有RNA
- 抗生素对病毒无效（抗生素针对细菌，病毒无细胞结构，不受影响）
