# 先天性行为
> **一句话**：先天性行为是动物生来就有、由遗传因素决定的本能行为，无需后天学习即可完成。
> **English**: Innate behavior is behavior that animals are born with, determined by genetic factors, and performed without prior learning.

---

## 📚 给 6-12 岁（自读版）

刚孵化出来的小鸡不需要学就知道吃食物，新生的婴儿一出生就会吸吮，蜘蛛从未上过"编织课"却能织出精美的网——这些都是**先天性行为**，也叫**本能**！

先天性行为是动物生下来就有的行为，写在基因里，不需要学习。无论这只动物从没见过同类，也能做出这种行为。

先天性行为的例子：
- 候鸟的迁徙（第一次迁徙就知道方向）
- 蜜蜂的舞蹈（传递蜜源信息）
- 婴儿的吸吮、哭泣
- 青蛙遇到危险时跳入水中

先天性行为让动物在出生后能立即应对基本的生存需求，非常重要！

## 🎓 给 12+（进阶版）

### 定义
先天性行为（innate behavior）又称本能（instinct），是由动物基因组决定的、在特定刺激（释放刺激）下自动表现的固定行为模式（fixed action pattern, FAP）。先天性行为不依赖个体经验，是物种演化的产物，在所有同种个体中基本一致。

### 先天性行为的特点

| 特点 | 说明 |
|------|------|
| 遗传决定 | 由遗传基因控制，与生俱来 |
| 不需学习 | 无需后天经验和训练 |
| 种内一致 | 同种动物的先天性行为基本相同 |
| 对环境适应 | 经长期自然选择保留，适应物种生存 |
| 刺激触发 | 特定刺激（sign stimulus）触发固定行为模式 |

### 先天性行为的分类

| 类型 | 描述 | 举例 |
|------|------|------|
| 定向行为 | 趋向/避开刺激 | 蛾趋光、蚯蚓避光 |
| 取食行为 | 获取食物的本能 | 刚出生的小鸡啄食 |
| 防御行为 | 天生的逃跑/防御 | 蜥蜴断尾求生 |
| 繁殖行为 | 求偶、孵卵本能 | 鱼回出生地产卵 |
| 迁移行为 | 季节性迁移 | 候鸟迁徙 |
| 社会行为 | 固定的社会信号 | 蜜蜂舞蹈语言 |

### 先天性行为 vs 学习行为

| 对比项 | 先天性行为 | 学习行为 |
|--------|-----------|---------|
| 获得方式 | 遗传决定，与生俱来 | 后天经验和学习 |
| 神经基础 | 固定神经通路 | 大脑皮层建立的联系 |
| 可变性 | 不可改变 | 可修改、消退 |
| 物种差异 | 低等动物占主导 | 高等动物更发达 |
| 适应性 | 对固定环境有效 | 对变化环境有效 |

### 典型例题
**例题**：将一只从未见过蚜虫的七星瓢虫放到蚜虫旁，它会自动捕食蚜虫。这种行为属于？
A. 学习行为，因为需要经验  B. 先天性行为，由遗传决定
C. 社会行为  D. 条件反射

**解析**：从未接触过蚜虫仍会自动捕食，说明不需学习，是遗传决定的先天性行为（本能）。**答案：B**

### 易错点
- 复杂行为≠学习行为（蜜蜂筑巢极其复杂，但属于先天性行为）
- 先天性行为有固定的刺激条件（特定刺激才触发）
- 先天性行为是学习行为的基础，完全没有先天性行为的学习行为不存在
