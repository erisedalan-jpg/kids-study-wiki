# 环节动物

> **一句话**：环节动物的身体由许多相似的环状体节组成，有真体腔，蚯蚓是典型代表。
> **English**: Annelids have bodies divided into many similar ring-like segments with a true coelom; the earthworm is the classic example.

---

## 📚 给 6-12 岁（自读版）

翻开泥土，你常会看到蚯蚓——它们的身体是一节一节的，像串联的圆环，这就是**环节动物**名字的由来。

环节动物的特点：
- 身体**分节**（每节都很相似，叫做同律分节）
- 有**真体腔**（身体内有真正的空腔，让器官有空间发育）
- 有完整的消化道（口→肛门）
- 体表湿润，靠**皮肤**呼吸

常见环节动物：**蚯蚓**（土壤中）、**水蛭（蚂蟥）**（水中，能吸血）、**沙蚕**（海中）

---

## 🎓 给 12+（进阶版）

### 定义

环节动物（Annelida）：具有**真体腔**和**同律分节**（metamerism）的蠕虫形动物门。"同律分节"指体节在形态和功能上基本相同（区别于节肢动物的异律分节）。真体腔由中胚层发育而来，有腹膜衬里，是动物进化史上的重要标志。

### 特征 / 分类（表格）

| 特征 | 说明 |
|------|------|
| 分节方式 | 同律分节（各节结构相似）|
| 体腔 | 真体腔（有中胚层来源的腹膜）|
| 附肢 | 多毛类有疣足，寡毛类有刚毛，蛭类无 |
| 循环 | 闭管式循环系统 |
| 消化 | 完整消化道 |
| 呼吸 | 体表、鳃（疣足上）或无专门器官 |

| 纲 | 代表 | 生活环境 |
|----|------|---------|
| 多毛纲 | 沙蚕 | 海洋 |
| 寡毛纲 | 蚯蚓 | 土壤、淡水 |
| 蛭纲 | 水蛭 | 淡水/陆地 |

### 典型例题

**例1**：环节动物与线形动物最重要的区别是什么？

| 比较 | 环节动物 | 线形动物 |
|------|---------|---------|
| 体节 | 有（同律分节）| 无 |
| 体腔 | 真体腔 | 假体腔 |
| 体表 | 有刚毛 | 角质层 |
| 代表 | 蚯蚓 | 蛔虫 |

**例2**：为什么说环节动物在进化上比扁形动物和线形动物高等？

**解析**：①出现了**真体腔**（中胚层来源），为内脏器官提供发育空间，器官更分化；②**同律分节**使身体各节可独立控制，运动更灵活；③**闭管式循环系统**（蚯蚓），物质运输效率更高。

### 易错点

- 蚯蚓靠**体壁（皮肤）呼吸**，体壁需保持湿润，雨后爬出地面是因为土壤缺氧，而非"喝水"。
- 水蛭（蚂蟥）是**环节动物**，不是昆虫，也不是软体动物。
- 同律分节≠异律分节：节肢动物（如昆虫）是异律分节（不同节功能不同）。
