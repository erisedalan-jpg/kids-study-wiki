# 动物行为
> **一句话**：动物行为是动物在生活中表现出来的各种活动，包括先天性行为和学习行为两大类。
> **English**: Animal behavior refers to all activities exhibited by animals in their lives, including innate behaviors and learned behaviors.

---

## 📚 给 6-12 岁（自读版）

你见过蜜蜂采蜜、候鸟迁徙、猫咪玩耍……这些都是**动物行为**！动物行为是动物在生活中的各种"行动"——觅食、求偶、防御、育幼等。

动物行为分两大类：
- **先天性行为**：生下来就会的本能行为，不需要学习，比如小鸟张嘴讨食、蜘蛛织网
- **学习行为**：通过后天经历和学习获得的行为，比如猫咪学会开门、狗学会握手

动物行为的神奇之处：行为越复杂，说明动物的神经系统越发达，学习能力越强。人类的学习行为是所有动物中最复杂、最多样的！

动物行为对生存很重要——找食物、躲避天敌、找配偶……都需要合适的行为。

## 🎓 给 12+（进阶版）

### 定义
动物行为（animal behavior）是动物在内外刺激下表现出的所有可观察活动的总称，是动物适应环境的方式。研究动物行为的科学称为**动物行为学**（ethology）。

### 行为的分类

| 分类依据 | 类型 | 说明 |
|----------|------|------|
| 获得方式 | 先天性行为 | 由遗传决定，与生俱来 |
| | 学习行为 | 通过经验获得，可改变 |
| 功能 | 觅食行为 | 获取食物 |
| | 防御行为 | 逃跑、拟态、警戒色 |
| | 繁殖行为 | 求偶、交配、育幼 |
| | 社会行为 | 群体成员间的互动 |
| | 节律行为 | 冬眠、迁徙、日节律 |
| | 攻击行为 | 种内争斗（领地、配偶） |

### 行为的神经机制
动物行为的基础：
- 刺激（内部/外部）→感受器→神经传导→效应器→行为
- 先天性行为：由固定的神经通路（遗传编程）控制
- 学习行为：需要大脑皮层参与，形成条件反射

### 动物行为的研究方法
- **观察法**：在自然环境中记录行为（法布尔研究昆虫）
- **实验法**：设计对照实验验证行为原因（巴甫洛夫条件反射实验）

### 典型例题
**例题**：下列行为中，属于先天性行为的是？
A. 小狗表演节目  B. 蜜蜂采蜜  C. 鸽子传递信息  D. 黑猩猩用树枝取食白蚁

**解析**：蜜蜂采蜜是本能（先天性行为），不需要后天学习；其他三项都是经过训练或学习获得的学习行为。**答案：B**

### 易错点
- 先天性行为≠简单行为（蜜蜂筑巢、候鸟迁徙都是先天性行为，但很复杂）
- 学习行为建立在先天性行为的基础上
- 同种动物的攻击行为（争地盘）≠捕食行为（猎食他种）
