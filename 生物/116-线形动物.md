# 线形动物

> **一句话**：线形动物（线虫）身体圆柱形、细长如线，有口有肛门，多营寄生生活，蛔虫是最典型的代表。
> **English**: Roundworms (nematodes) have cylindrical, thread-like bodies with a mouth and anus; most are parasitic, with Ascaris as the classic example.

---

## 📚 给 6-12 岁（自读版）

**线形动物**的身体像一根细线，圆圆的、长长的。它们有**嘴（口）也有肛门**，这比扁形动物进步了一步。

最著名的线形动物是**蛔虫**——它可以住在人的小肠里，靠吸收你吃的食物为生，让你营养不良、肚子痛。小时候要注意饭前洗手、不喝生水，就是为了防止蛔虫卵进入身体。

其他线形动物：
- **蛲虫**：寄生在人的大肠，引起瘙痒
- **丝虫**：寄生在淋巴管，引起象皮病
- **秀丽线虫**：自由生活，科学研究的重要模式生物

---

## 🎓 给 12+（进阶版）

### 定义

线形动物（Nematoda），又称线虫，是动物界种类最多的门之一（已描述约 28,000 种，估计实际超过 100 万种）。身体呈**圆柱形**，不分节，体表有角质层，体腔为**假体腔**（原体腔），消化道**有口有肛门**，是动物进化史上出现完整消化道的早期类群。

### 特征 / 分类（表格）

| 特征 | 说明 |
|------|------|
| 体型 | 圆柱形，细长不分节 |
| 体表 | 角质层（cuticle）保护 |
| 体腔 | 假体腔（原体腔）|
| 消化 | 有口有肛门，消化道完整 |
| 排泄 | 腺细胞或管细胞 |
| 生殖 | 雌雄异体，体内受精 |

| 代表 | 寄生部位 / 生活方式 | 危害 |
|------|-------------------|------|
| 蛔虫 | 人小肠 | 营养不良、腹痛 |
| 蛲虫 | 人大肠 | 肛门瘙痒 |
| 丝虫 | 淋巴管 | 象皮病 |
| 钩虫 | 人小肠 | 贫血 |
| 秀丽线虫 | 自由生活（土壤） | 无（模式生物）|

**蛔虫的生活史**：蛔虫卵（经口）→ 幼虫（肠壁→肝→心→肺→气管→咽→小肠）→ 成虫

### 典型例题

**例1**：蛔虫消化系统与扁形动物相比有何进步？

**解析**：蛔虫有**口和肛门**，消化道"一进一出"，食物单向流动，消化效率更高。扁形动物只有口，食物残渣仍从口排出，效率低。

**例2**：预防蛔虫病的主要措施是什么？

**解析**：①饭前便后洗手；②不喝生水，不吃未洗净的蔬菜水果；③管理粪便，避免用生粪施肥；④及时驱虫治疗。

### 易错点

- 线形动物≠环节动物：线虫身体**不分节**，环节动物（蚯蚓）身体**分节**，两者亲缘关系较远。
- 假体腔≠真体腔：假体腔（原体腔）由胚胎卵裂腔演变，无中胚层来源的腹膜衬里。
- 秀丽线虫（*C. elegans*）是自由生活的线虫，是发育生物学的经典模式生物，与寄生线虫有很大区别。
