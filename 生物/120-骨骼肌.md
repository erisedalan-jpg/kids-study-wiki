# 骨骼肌
> **一句话**：骨骼肌是附着在骨骼上的横纹肌，受意志控制，是动物产生运动的直接动力器官。
> **English**: Skeletal muscle is striated muscle attached to bones, under voluntary control, and the direct power organ for animal movement.

---

## 📚 给 6-12 岁（自读版）

你能控制自己的手臂抬起来——这是**骨骼肌**在工作！骨骼肌是我们可以"想动就动"的肌肉，是运动的直接动力。

一块骨骼肌由两部分组成：
- **肌腹**：中间较粗的部分，由肌肉细胞组成，能收缩产生力量
- **肌腱**：两端细而坚韧的部分，把肌肉连接到骨头上

骨骼肌必须跨过关节才能发挥作用——当肌肉收缩时，牵拉骨骼绕关节转动，就产生了运动。

人体有超过600块骨骼肌！它们大小不一：最小的是耳朵里的镫骨肌，最大的是臀大肌，最长的是大腿上的缝匠肌。

## 🎓 给 12+（进阶版）

### 定义
骨骼肌（skeletal muscle）是附着于骨骼（通过肌腱），受躯体运动神经支配的横纹肌。是随意肌，参与身体姿势维持和主动运动。人体约有600余块骨骼肌，占体重的40%-45%。

### 骨骼肌的结构

| 层次 | 描述 |
|------|------|
| 肌腹（muscle belly） | 中部，由大量肌纤维（肌细胞）组成，有收缩功能 |
| 肌腱（tendon） | 两端，致密结缔组织，无收缩功能，附着骨骼传递力 |
| 肌纤维（肌细胞） | 多核，长圆柱形，含肌原纤维 |
| 肌原纤维 | 含粗肌丝（肌球蛋白）和细肌丝（肌动蛋白） |

### 骨骼肌的工作原理

**收缩机制**（滑丝理论）：
1. 神经冲动到达神经-肌肉接头（突触）
2. 乙酰胆碱释放，引发肌细胞动作电位
3. Ca²⁺释放，暴露肌动蛋白结合位点
4. 肌球蛋白头部与肌动蛋白结合，消耗ATP，发生横桥摆动
5. 细肌丝向粗肌丝中央滑动，肌节缩短，肌肉收缩

### 骨骼肌的工作特性

| 特性 | 说明 |
|------|------|
| 兴奋性 | 能接受神经刺激并产生反应 |
| 收缩性 | 能缩短产生张力 |
| 伸展性 | 被动被拉长（非主动） |
| 弹性 | 形变后能恢复原状 |
| 疲劳性 | 持续刺激后收缩力下降 |

### 拮抗肌关系（以肘关节为例）

| 动作 | 肱二头肌 | 肱三头肌 |
|------|----------|----------|
| 屈肘 | 收缩 | 舒张 |
| 伸肘 | 舒张 | 收缩 |
| 静止保持姿势 | 协同收缩（保持张力） | 协同收缩 |

### 典型例题
**例题**：一块骨骼肌中，具有收缩功能的部分是？
A. 肌腱  B. 肌腹  C. 肌腱和肌腹  D. 关节囊

**解析**：肌腹由肌纤维（肌细胞）组成，是具有收缩功能的部分；肌腱是结缔组织，无收缩功能，负责连接骨骼。**答案：B**

### 易错点
- 肌腱不能收缩，只传递力（结缔组织）
- 骨骼肌两端肌腱必须附着在**不同骨骼**上，跨过至少一个关节
- 肌肉疲劳原因：ATP耗尽、乳酸积累（无氧呼吸产物）
