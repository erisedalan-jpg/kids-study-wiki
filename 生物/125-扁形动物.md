# 扁形动物

> **一句话**：扁形动物是身体扁平、两侧对称、有口无肛门的低等动物，包括涡虫（自由生活）和绦虫、血吸虫（寄生）。
> **English**: Flatworms (Platyhelminthes) are bilaterally symmetrical animals with flattened bodies and no anus, including free-living planarians and parasitic tapeworms and flukes.

---

## 📚 给 6-12 岁（自读版）

**扁形动物**的身体像一张薄饼，压得很扁很平，这也是名字的由来。

扁形动物的特点：
- 身体**扁平**，像薄片一样
- **两侧对称**：左右两边是镜像的（和水母的辐射对称不同）
- 有**口**但**没有肛门**，食物残渣从口吐出
- 有的自由生活（如**涡虫**，生活在清洁的溪水中）
- 有的**寄生**在人或动物体内（如**绦虫**、**血吸虫**），会让人生病

血吸虫病曾经在我国南方很流行，是扁形动物引起的著名寄生虫病。

## 🎓 给 12+（进阶版）

### 定义

扁形动物（Platyhelminthes）：两侧对称、体形扁平、无体腔（属于**无体腔动物**）的动物门。具有三个胚层（外、中、内胚层），是动物进化史上出现**中胚层**的最低等类群，也是最早出现**两侧对称**体型的动物。

### 特征 / 分类（表格）

| 特征 | 说明 |
|------|------|
| 体型 | 两侧对称，背腹扁平 |
| 胚层 | 三胚层（无体腔） |
| 消化 | 有口无肛门，多有分枝的消化腔 |
| 排泄 | 原肾管（焰细胞） |
| 神经 | 梯形神经系统（出现了头部神经节）|

| 纲 | 代表 | 生活方式 |
|----|------|---------|
| 涡虫纲 | 涡虫（Planaria） | 自由生活，淡水 |
| 吸虫纲 | 血吸虫、华支睾吸虫 | 寄生 |
| 绦虫纲 | 猪带绦虫、牛带绦虫 | 寄生 |

**涡虫**再生能力极强，切成多段后每段可再生为完整个体，是再生研究模式生物。

### 典型例题

**例1**：比较腔肠动物和扁形动物的异同。

| 比较项目 | 腔肠动物 | 扁形动物 |
|---------|---------|---------|
| 体型对称 | 辐射对称 | 两侧对称 |
| 胚层 | 两胚层 | 三胚层 |
| 有无肛门 | 无 | 无 |
| 体腔 | 无 | 无（假无体腔）|

**例2**：血吸虫的生活史需要哪两种宿主？

**解析**：血吸虫需要**钉螺**（中间宿主）和**人或哺乳动物**（终末宿主）。消灭钉螺是防治血吸虫病的重要措施。

### 易错点

- 扁形动物**有三个胚层**（腔肠动物才是两胚层），这是进化上的重要进步。
- 绦虫**无消化系统**（退化），直接通过体表吸收宿主营养。
- "扁平"是描述外形，不是分类依据；关键特征是两侧对称+无体腔+三胚层。
