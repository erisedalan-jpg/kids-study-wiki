# 蚯蚓

> **一句话**：蚯蚓是生活在土壤中的环节动物，靠皮肤呼吸，通过挖掘改善土壤结构，被称为"大地的耕耘者"。
> **English**: The earthworm is a soil-dwelling annelid that breathes through its moist skin and improves soil structure through burrowing, earning it the title "nature's plowman."

---

## 📚 给 6-12 岁（自读版）

下雨后，地面上常常爬出很多蚯蚓——它们柔软、粉红色，身体一节一节的。**蚯蚓**是最典型的**环节动物**。

蚯蚓的神奇之处：
- **没有肺，用皮肤呼吸**：皮肤要一直保持湿润，才能让氧气通过。下雨时土壤积水缺氧，它们才爬出来透气。
- **没有眼睛，没有耳朵**：靠皮肤感觉光线和振动
- **雌雄同体**（一条蚯蚓既有雄性也有雌性器官），但需要两条蚯蚓交配才能繁殖
- **松土高手**：蚯蚓在土里钻来钻去，让空气和水进入土壤，帮助植物生长
- 能**再生**：切断后前端能重新长出（但不是每段都能活）

达尔文曾经研究蚯蚓长达40年，说蚯蚓是地球上最重要的动物之一。

## 🎓 给 12+（进阶版）

### 定义

蚯蚓（earthworm），隶属**环节动物门寡毛纲**，常见种类有赤子爱胜蚓（*Eisenia fetida*，红蚯蚓）和参环毛蚓等。体长数厘米至数十厘米，体节数十至数百个，体表有**刚毛**辅助运动，体表富含**黏液**保持湿润。

### 特征 / 分类（表格）

| 特征 | 说明 |
|------|------|
| 分类地位 | 环节动物门 · 寡毛纲 |
| 体节 | 同律分节，刚毛辅助运动 |
| 体腔 | 真体腔，充满体腔液 |
| 呼吸 | 体壁呼吸（皮肤呼吸），无专门呼吸器官 |
| 循环 | 闭管式循环，5对"心脏"（心形毛细血管环）|
| 消化 | 口→咽→食道→嗉囊→砂囊→肠→肛门 |
| 生殖 | 雌雄同体，异体受精，卵茧孵化 |
| 感觉 | 无眼，有光感受细胞；无耳，靠振动感受 |

**蚯蚓对土壤的作用**：
1. 物理作用：疏松土壤，增加通气性和保水性
2. 化学作用：粪便（虫粪）富含N、P、K，是优质有机肥
3. 生物作用：加速有机物分解，促进物质循环

### 典型例题

**例1**（实验）：探究蚯蚓在什么样的物体表面运动更快——光滑的玻璃板还是粗糙的纸板？

**解析**：在粗糙纸板上运动更快。蚯蚓靠**刚毛**和肌肉（环肌+纵肌）协作运动，光滑表面刚毛抓不住，摩擦力小，运动困难。

**例2**：蚯蚓身体前端如何判断？

**解析**：靠近**环带**（生殖器官形成的肿大体节，颜色较浅）的一端是前端（靠近口）。环带位于第14~16节。

### 易错点

- 雨后蚯蚓出土是因为土壤**缺氧**（积水），不是因为想"喝雨水"。
- 蚯蚓**皮肤要保持湿润**才能呼吸，若长时间暴露在干燥阳光下会死亡。
- 蚯蚓切断后并非两段都能存活：**前端**（含口和部分重要器官）再生能力强；后端能否存活取决于切断位置。
