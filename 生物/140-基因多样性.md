# 基因多样性

> **一句话**：同一物种内部不同个体、不同种群之间在基因层面上的差异和变异程度。
> **English**: Genetic diversity is the variation in genes within and between populations of a species.

---

## 📚 给 6-12 岁（自读版）

为什么同班同学长得都不一样？因为每个人的**基因**不同——这就是**基因多样性**。

**生活里的例子：**
- 兄弟姐妹长得像但不完全一样（基因略有差异）
- 同种猫有的橘色、有的黑白（基因决定毛色）
- 中国和非洲不同人群肤色不同（基因差异）
- 番茄有红色、黄色、橘色品种（基因不同）

**为什么基因多样性重要？**
- 让生物有"备用方案"：万一环境变了，总有部分个体能适应活下来
- 让物种能进化：如果都一样，没法进化出新种
- 是物种多样性的根本基础

**有趣事实：**
- 人和人之间基因序列差异约 0.1%
- 同卵双胞胎基因几乎相同（但仍有微小变化）
- 大熊猫野生种群基因多样性低 → 易受疾病威胁

## 🎓 给 12+（进阶版）

### 定义

**基因多样性（遗传多样性）**：同一物种范围内种群之间和种群内个体之间的遗传变异总和。是物种多样性的内在基础。

### 来源

| 来源 | 机制 |
|------|------|
| 基因突变 | DNA 碱基对的改变 |
| 基因重组 | 减数分裂时同源染色体交叉、自由组合 |
| 染色体变异 | 染色体结构/数目变异 |
| 杂交 | 不同种群间交配引入新基因 |

### 衡量指标

| 指标 | 含义 |
|------|------|
| 多态位点比例 | 群体中有变异的基因座占比 |
| 平均杂合度 | 个体杂合基因座的平均频率 |
| 等位基因丰富度 | 群体中等位基因数 |
| 核苷酸多样性 (π) | 任两序列间平均碱基差异率 |

### 重要性

1. **物种适应性的源泉**：环境变化时部分个体可能携带"幸运"基因存活
2. **进化的基础**：自然选择作用于变异，无变异则无进化
3. **物种长期存续的保障**：种群基因多样性低 → 灭绝风险高
4. **农业育种基础**：野生近缘种保留珍贵基因，用于改良作物

### 例子

| 物种 | 基因多样性现状 |
|------|------------|
| 大熊猫 | 野生种群约 1860 只，基因多样性较低（瓶颈效应） |
| 华南虎 | 几乎仅存动物园种群，遗传退化 |
| 朱鹮 | 1981 年仅剩 7 只，繁育至今数千只但基因多样性受限 |
| 玉米 | 野生近缘种（大刍草）保留育种关键基因 |

### 基因多样性丧失的原因

- 种群数量骤减 → **遗传瓶颈**
- 近交（小种群） → **近交衰退**
- 栖息地碎片化 → 基因交流受阻
- 自然选择压力 → 部分等位基因消失

### 保护措施

- 保护野生种群（最大可生存种群 MVP）
- 建立**基因库**（种子库、配子库、DNA 库）
- 人工繁育中尽量增加遗传背景多样性
- 维持野生近缘种栖息地

### 易错点

- 基因多样性≠物种多样性。同一物种基因多样性高，并不意味着物种数量多
- 物种多样性的本质是基因多样性的体现
- 保护基因多样性是保护生物多样性的根本
