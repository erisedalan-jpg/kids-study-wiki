# 物种多样性

> **一句话**：一定区域内所有物种的丰富程度，是生物多样性最直观的表现。
> **English**: Species diversity refers to the richness of species in a given area, the most apparent level of biodiversity.

---

## 📚 给 6-12 岁（自读版）

走进森林，能看到松树、栎树、苔藓、蘑菇、蚂蚁、松鼠、啄木鸟……**种类越多，物种多样性越高**！

**对比：**
- 热带雨林：每公顷有几百种树
- 沙漠：每公顷只有几种植物
- 农田：通常只有 1-2 种作物（人种的）
- 海洋：珊瑚礁多样性高，深海平原低

**记住：** 物种多样性 = "有多少种生物"

**中国是物种大国：**
- 高等植物 3.5 万余种（世界第三）
- 脊椎动物 6300 余种
- 特有种：大熊猫、东北虎、银杏、水杉……

## 🎓 给 12+（进阶版）

### 定义

**物种多样性**：一定区域或生态系统中所有物种数量及其相对丰度的总体表现。可以用物种数（**物种丰富度**）和物种均匀度等指标量化。

### 衡量指标

| 指标 | 含义 |
|------|------|
| 物种丰富度 (Species richness) | 区域内物种总数 |
| 物种均匀度 (Evenness) | 各物种个体数分布是否均匀 |
| 香农指数 (H') | 综合丰富度与均匀度 |
| 辛普森指数 (D) | 优势度的反向衡量 |

### 全球物种多样性分布

- 已记录约 200 万种，估计实际 870 万种甚至更多
- **纬度梯度**：赤道附近多 → 极地少
- **生境多样性**：陆地约 87%，海洋约 13%（已知）
- **海拔梯度**：低海拔多

### 全球生物多样性热点（Hotspots）

热带雨林（亚马逊、刚果、印尼）、地中海地区、东南亚岛屿、马达加斯加等约 36 个热点区，集中保护是国际策略。

### 中国物种多样性

| 类群 | 物种数（约） |
|------|----------|
| 高等植物 | 3.5 万余种（世界第三） |
| 哺乳动物 | 600 余种 |
| 鸟类 | 1400 余种 |
| 两栖动物 | 400 余种 |
| 爬行动物 | 500 余种 |
| 鱼类 | 4000 余种 |
| 昆虫 | 10 万余种（估计） |

### 影响因素

| 因素 | 作用 |
|------|------|
| 气候 | 温暖湿润 → 多样性高 |
| 地形 | 复杂地形→生境多→物种多 |
| 历史 | 古老地区物种积累更多 |
| 人类干扰 | 强干扰 → 多样性下降 |
| 关键种 | 关键种存在维持其他物种 |

### 物种灭绝速率

- 自然背景灭绝率：每年每万种 0.1-1 种
- 当前灭绝率：背景的 100-1000 倍 → "第六次大灭绝"

### 易错点

- 物种多样性≠物种数量绝对值——还要考虑均匀度
- 优势种数量大但其他物种少 → 多样性反而低
- 一种物种灭绝可能引发"营养级联"——影响食物网中多种生物
