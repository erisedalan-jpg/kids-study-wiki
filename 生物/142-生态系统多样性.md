# 生态系统多样性

> **一句话**：地球上各种生态系统类型的多样化程度，包括森林、草原、湿地、海洋等。
> **English**: Ecosystem diversity refers to the variety of ecosystems on Earth, such as forests, grasslands, wetlands, and oceans.

---

## 📚 给 6-12 岁（自读版）

地球上不止一种"环境"——森林、草原、沙漠、湿地、河流、湖泊、海洋……每种都是一个**生态系统**。

**类型这么多：**
- **森林生态系统**：树多，动物丰富，"地球之肺"
- **草原生态系统**：以草为主，有牛羊、野马、羚羊
- **湿地生态系统**：水陆交界，鸟类众多，"地球之肾"
- **海洋生态系统**：占地球 70%，鱼虾贝藻
- **淡水生态系统**：河、湖、池塘
- **沙漠生态系统**：缺水、生物少
- **农田生态系统**：人工管理，作物为主
- **城市生态系统**：人是主角

每种都不一样，加起来叫**生态系统多样性**。

**为什么重要？**
不同的生态系统养活不同的生物，丧失某种生态系统，里面的生物也跟着遭殃。比如填海造田 → 破坏海岸湿地 → 候鸟无处停歇。

## 🎓 给 12+（进阶版）

### 定义

**生态系统多样性**：在不同尺度上生态系统的类型、组成、结构和功能的多样化程度，以及生态系统内栖息地、生物群落和生态过程的差异。

### 主要类型

| 大类 | 子类 | 特点 |
|------|------|------|
| **陆地生态系统** | 森林（热带雨林/温带落叶林/北方针叶林） | 树木为主、生产力高 |
| | 草原（热带稀树草原/温带草原） | 草本为主 |
| | 荒漠（沙漠） | 干旱、生物稀少 |
| | 苔原 | 极地、低温 |
| **水域生态系统** | 海洋（远洋/沿岸/深海） | 占地表 70%、生产力差异大 |
| | 淡水（河流/湖泊/池塘） | 资源宝贵 |
| | 湿地（沼泽/河口/红树林） | "地球之肾" |
| **人工生态系统** | 农田、果园、城市、人工林 | 人工管理 |

### 中国主要生态系统

- 东北：温带针阔混交林
- 华北：温带落叶阔叶林
- 华南：亚热带常绿阔叶林、热带季雨林
- 西北：温带草原、温带荒漠
- 青藏高原：高寒草原、高寒荒漠
- 东南沿海：红树林湿地
- 长江中下游：湖沼湿地

### 每种生态系统的服务

| 系统 | 服务 |
|------|------|
| 森林 | 涵养水源、固碳、防风固沙、生物多样性栖息地 |
| 湿地 | 净水、调蓄洪水、净化污染、候鸟栖息 |
| 海洋 | 渔业、运输、调节气候、产生大量 O₂ |
| 草原 | 畜牧业、保持水土 |
| 农田 | 提供粮食 |

### 威胁

- 城市化、农田扩张 → 自然生态系统缩减
- 围湖造田 → 湿地丧失（如洞庭湖、鄱阳湖萎缩）
- 过度砍伐 → 森林减少
- 污染、气候变化 → 各类系统退化

### 保护

- **建立自然保护区**：覆盖典型生态系统
- **生态红线**：禁止开发的关键生态空间
- **国家公园**：整体保护
- **生态修复**：退耕还林、退牧还草、湿地恢复

### 易错点

- 生态系统多样性不只是数"几种"系统，还包括同类系统中不同地域的差异
- 农田、城市也属于生态系统（人工型）
- 物种多样性不等于生态系统多样性——物种多但生态系统单一也算多样性低
