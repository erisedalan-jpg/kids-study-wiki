# 生物分类

> **一句话**：根据生物的形态结构、生理特征及亲缘关系，将生物分成不同等级（界门纲目科属种）的方法。
> **English**: Biological classification organizes living things into hierarchical groups (kingdom, phylum, class, order, family, genus, species) based on shared traits and ancestry.

---

## 📚 给 6-12 岁（自读版）

地球上几百万种生物，怎么记得清？科学家想了个办法：**分组归类**。

**像把图书馆分类一样：**
- 大类：动物 / 植物 / 微生物
- 中类：哺乳类 / 鸟类 / 鱼类……
- 小类：猫科 / 犬科……
- 最小：每只动物有自己的种名（家猫、东北虎）

**七个等级：**
**界 → 门 → 纲 → 目 → 科 → 属 → 种**

口诀：**界门纲目科属种**

**举个例子（人）：**
- 界：动物界
- 门：脊索动物门
- 纲：哺乳纲
- 目：灵长目
- 科：人科
- 属：人属
- 种：智人

**为什么要分类？**
- 方便记忆和研究
- 看清谁和谁是"亲戚"——比如狼和狐狸都属犬科
- 帮助保护——哪些是濒危种，要单独保护

## 🎓 给 12+（进阶版）

### 定义

**生物分类学（taxonomy）**：研究生物分类原理和方法的科学，依据生物的形态结构、生理特征、生态、行为以及分子证据，建立反映生物亲缘关系的系统。

### 分类等级（七级）

```
界 Kingdom
  门 Phylum
    纲 Class
      目 Order
        科 Family
          属 Genus
            种 Species
```

记忆口诀：界门纲目科属种

每级越往下，亲缘关系越近，特征越相似。

### 二名法（林奈命名法）

- 提出者：林奈（瑞典）
- 学名 = 属名 + 种加词（拉丁文，斜体）
- 举例：
  - 智人 *Homo sapiens*
  - 大熊猫 *Ailuropoda melanoleuca*
  - 水稻 *Oryza sativa*

### 五界系统（魏泰克 1969）

| 界 | 例 | 特点 |
|----|----|------|
| 原核生物界 | 细菌、蓝藻 | 无核膜 |
| 原生生物界 | 草履虫、海带 | 单细胞或简单多细胞真核 |
| 真菌界 | 蘑菇、酵母 | 异养、细胞壁含几丁质 |
| 植物界 | 苔藓、蕨、被子植物 | 自养、光合 |
| 动物界 | 各类动物 | 异养、可运动 |

### 现代三域系统（沃斯）

| 域 | 内容 |
|----|------|
| 细菌域 (Bacteria) | 真细菌 |
| 古菌域 (Archaea) | 古细菌（极端环境） |
| 真核域 (Eukarya) | 包含原生、真菌、植物、动物 |

### 中国植物分类传统

| 大类 | 例子 |
|------|------|
| 藻类植物 | 水绵、海带 |
| 苔藓植物 | 葫芦藓 |
| 蕨类植物 | 蕨、肾蕨 |
| 裸子植物 | 松、银杏 |
| 被子植物 | 苹果、水稻、玉兰 |

### 分类依据的演变

| 阶段 | 主要依据 |
|------|--------|
| 古典分类 | 形态结构 |
| 现代综合 | 形态 + 生理 + 化石 |
| 分子分类 | DNA / RNA 序列、蛋白质比对 |

### 易错点

- 二名法属名首字母大写，种加词小写，整个学名用斜体
- 分类等级越高（如门），包含范围越大
- 同种内的"亚种、变种、品种"在种之下的次级单位
- 病毒不属于五界系统（无细胞结构）
