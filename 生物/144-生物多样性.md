# 生物多样性

> **一句话**：生物及其与环境形成的生态复合体所表现出的多样化程度，包括物种、基因和生态系统三个层次。
> **English**: Biodiversity is the variety of life on Earth at three levels: genes, species, and ecosystems.

---

## 📚 给 6-12 岁（自读版）

地球上有多少种生物？**已知**约 200 万种，估计**实际**有 870 万种或更多——这就是**生物多样性**。

**3 个层次：**
1. **物种多样性**：种类多——熊猫、东北虎、藏羚羊、大熊猫……
2. **基因多样性**：同一种里，每个个体基因略不同——所以兄弟姐妹长得不一样
3. **生态系统多样性**：森林、草原、湿地、海洋……不同环境多种多样

**为什么要保护？**
- 生物有用：食物、药材、工业原料
- 生态平衡：物种少了食物网会断
- 美的享受：动物园、自然保护区
- 我们的责任：每个物种都是地球家庭一员

**生物多样性面临的威胁：**
- 栖息地破坏（砍森林、填湿地）
- 滥捕滥猎
- 环境污染
- 外来种入侵
- 气候变化

**怎么保护？**
- 建自然保护区（中国有大熊猫、东北虎等保护区）
- 立法（《野生动物保护法》）
- 不买、不食濒危动物制品
- 多去自然里散步，少破坏环境

## 🎓 给 12+（进阶版）

### 定义

**生物多样性**：生物界中各种各样生物（包括动物、植物、微生物）和它们所拥有的基因，以及它们与环境构成的复杂生态系统所表现的多样性。

### 三层次

| 层次 | 内涵 | 例子 |
|------|------|------|
| 基因多样性（遗传多样性） | 物种内不同种群、不同个体的遗传变异 | 同种水稻不同品种 |
| 物种多样性 | 区域内物种数量与丰富度 | 雨林物种远多于沙漠 |
| 生态系统多样性 | 生态系统类型多样 | 森林、草原、湿地、海洋等 |

### 价值

| 价值 | 内容 |
|------|------|
| 直接价值 | 食物、药材、工业原料、观赏 |
| 间接价值 | 生态调节（净水、固土、吸 CO₂、调气候）、传粉 |
| 潜在价值 | 未来可能发现的科研、医药价值 |

### 我国生物多样性概况

- 物种丰富：高等植物 3.5 万余种，脊椎动物 6300 余种
- 特有种多：大熊猫、华南虎、银杉、银杏、水杉、珙桐
- 五大重点生态区：青藏高原、横断山、武陵山、神农架、海南岛

### 威胁因素

| 因素 | 影响 |
|------|------|
| 栖息地破坏 | 主要威胁，砍伐、围垦、城市化 |
| 滥捕滥猎 | 直接削减种群 |
| 环境污染 | 农药、塑料、重金属 |
| 外来种入侵 | 福寿螺、水葫芦、紫茎泽兰 |
| 气候变化 | 温度变化、海平面上升 |

### 保护措施

| 措施 | 说明 |
|------|------|
| **就地保护** | 建自然保护区（最有效），如三江源、大熊猫保护区 |
| **易地保护** | 动物园、植物园、人工繁育（朱鹮、麋鹿） |
| **建立基因库** | 种子库、配子库 |
| **法律保护** | 《野生动物保护法》《种子法》 |
| **公约履约** | 《生物多样性公约》《濒危野生动植物种国际贸易公约 CITES》 |

### 易错点

- 生物多样性 = 基因 + 物种 + 生态系统三层次（不只是物种数量）
- 物种多样性的本质是基因多样性
- 保护的根本是保护基因多样性
