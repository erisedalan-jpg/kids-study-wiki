# 细菌生殖
> **一句话**：细菌主要通过二分裂方式进行无性生殖，速度极快，适宜条件下每20-30分钟分裂一次。
> **English**: Bacteria primarily reproduce asexually by binary fission, dividing extremely rapidly under favorable conditions every 20-30 minutes.

---

## 📚 给 6-12 岁（自读版）

细菌是怎么繁殖的？不需要爸爸妈妈，细菌自己就能分裂产生后代！

细菌通过**二分裂**（分裂生殖）繁殖：一个细菌先把遗传物质（DNA）复制一份，然后整个细胞从中间一分为二，变成两个完全一样的细菌。

细菌繁殖速度惊人！在最适宜的温度和营养条件下，大肠杆菌每20分钟就能分裂一次。从1个开始计算：
- 20分钟：2个
- 40分钟：4个
- 1小时：8个
- ……
- 7小时：约100万个！（2²¹）

这就是为什么食物在室温下放几小时就会"坏"——细菌数量指数级增长，产生大量毒素。放冰箱可以低温抑制细菌繁殖；高温消毒也能杀死大多数细菌。

## 🎓 给 12+（进阶版）

### 定义
细菌生殖主要方式是**二分裂**（binary fission），即无性生殖：细胞DNA复制，细胞延伸，中隔形成，细胞一分为二，形成两个遗传组成相同的子细胞。某些细菌还具有接合（conjugation）等准有性生殖方式，可在个体间转移遗传物质。

### 二分裂过程

| 步骤 | 内容 |
|------|------|
| 1. DNA复制 | 环形染色体复制，产生两个相同的DNA分子 |
| 2. DNA分离 | 两个DNA分子分别向细胞两端移动 |
| 3. 细胞延伸 | 细胞体积增大，中央部位细胞膜向内凹陷 |
| 4. 隔膜形成 | 细胞壁和细胞膜在中部形成新的隔壁 |
| 5. 细胞分开 | 两个子细胞完全分离（或暂时相连形成链状） |

### 细菌生长的数学模型
设初始细菌数为N₀，代时（每次分裂时间）为T，经过t小时后：
- 分裂次数 n = t × 60 / T
- 细菌数 N = N₀ × 2ⁿ

**例**：初始1个大肠杆菌，代时20分钟，7小时后：
n = 7×60/20 = 21次
N = 1 × 2²¹ ≈ 210万个

### 影响细菌繁殖的因素

| 因素 | 适宜条件 | 抑制方法 |
|------|----------|----------|
| 温度 | 多数细菌：35-37°C | 低温冷藏、高温灭菌 |
| pH | 多数：中性（6.5-7.5） | 酸性环境（腌制食品） |
| 营养 | 充足有机碳源、氮源 | 密封减少营养 |
| 水分 | 充足水分 | 干燥脱水 |
| 氧气 | 需氧/厌氧/兼性厌氧 | 真空包装 |

### 细菌的其他遗传物质转移方式

| 方式 | 说明 |
|------|------|
| 接合（conjugation） | 两细菌通过性菌毛直接接触，转移质粒 |
| 转化（transformation） | 细菌摄取环境中游离DNA片段（格里菲斯实验原理） |
| 转导（transduction） | 噬菌体将细菌DNA携带至另一细菌 |

### 典型例题
**例题**：细菌进行分裂生殖，若每20分钟分裂一次，1个细菌经2小时后变为多少个？
A. 6个  B. 64个  C. 128个  D. 256个

**解析**：2小时=120分钟，分裂次数=120÷20=6次，细菌数=2⁶=64个。**答案：B**

### 易错点
- 二分裂是无性生殖，后代与亲代遗传组成相同（DNA复制无误情况下）
- 高温灭菌≠低温杀菌（冰箱只抑制，不完全杀死细菌）
- 细菌也可通过接合等方式发生"遗传重组"，这是抗性传播的重要机制
