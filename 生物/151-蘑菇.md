# 蘑菇
> **一句话**：蘑菇是担子菌类真菌的子实体，由地下菌丝发育而来，通过产生孢子进行繁殖，是典型的多细胞真菌。
> **English**: A mushroom is the fruiting body of basidiomycete fungi, developed from underground mycelium, reproducing through spores, a typical multicellular fungus.

---

## 📚 给 6-12 岁（自读版）

你吃过香菇、平菇、金针菇吗？它们都是**蘑菇**——大型真菌的子实体！

我们平时看到的蘑菇（伞状部分）其实只是整个真菌的一小部分。土壤里面有大量细如丝线的**菌丝**，这才是蘑菇真正的"身体"。当条件合适时（温度、湿度适宜），菌丝会长出地面，形成我们看到的蘑菇。

蘑菇的结构：
- **菌盖**：伞状的"帽子"，保护孢子
- **菌褶**：菌盖下面的褶皱，产生**孢子**（繁殖体）
- **菌柄**：支撑菌盖的"杆"
- **菌丝**：地下网络，吸收营养

蘑菇没有叶绿素，不能光合作用，靠**分解腐败有机物**或寄生在树木上获取养分。

**注意**：野生蘑菇不能随便采食！有毒蘑菇（毒蘑菇）和可食蘑菇长相相似，误食可能致命。

## 🎓 给 12+（进阶版）

### 定义
蘑菇通常指担子菌门（Basidiomycota）真菌的子实体（fruiting body），是真菌在繁殖时期形成的肉质结构，用于产生和散播孢子。广义的"蘑菇"也包括子囊菌（如羊肚菌）的子实体。

### 蘑菇的结构

| 部位 | 功能 |
|------|------|
| 菌盖（pileus） | 保护菌褶，常有色素（吸引动物或警戒） |
| 菌褶（lamella） | 位于菌盖下，着生担孢子 |
| 菌柄（stipe） | 支撑菌盖，输导水分营养 |
| 菌环（annulus） | 菌盖展开时残留的内菌膜 |
| 菌托（volva） | 某些毒蘑菇基部的杯状结构（如毒鹅膏）|
| 地下菌丝网络 | 吸收营养，是蘑菇的真正"主体" |

### 蘑菇的生殖过程
1. 菌褶上的**担子**（basidium）产生**担孢子**
2. 担孢子成熟脱落，随气流散播
3. 孢子萌发为**初生菌丝**（单核）
4. 两条不同交配型的初生菌丝融合→**次生菌丝**（双核）
5. 次生菌丝生长，形成菌丝体
6. 环境适宜时，菌丝体形成子实体原基→发育为蘑菇

### 蘑菇的营养价值

| 成分 | 特点 |
|------|------|
| 蛋白质 | 含量8-40%（干重），必需氨基酸齐全 |
| 多糖 | 香菇多糖、灵芝多糖具有免疫调节作用 |
| 脂肪 | 少，以不饱和脂肪酸为主 |
| 维生素 | 富含B族维生素、维生素D前体 |
| 矿物质 | 钙、磷、铁等 |

### 毒蘑菇的识别（初步原则）
- **不要**相信颜色、气味等简单规律（毒蘑菇不一定鲜艳）
- **毒鹅膏**（剧毒）特征：白色、有菌托、有菌环
- **唯一安全方法**：由专业人士鉴定，或选择已知可食的栽培品种

### 典型例题
**例题**：蘑菇属于哪类生物？其营养方式是？
A. 植物，自养  B. 原核生物，异养  C. 真菌，异养  D. 动物，异养

**解析**：蘑菇是真菌（有真正细胞核的真核生物），无叶绿素，靠分解有机物（异养）生活。**答案：C**

### 易错点
- 我们吃的蘑菇是子实体，不是整个真菌（菌丝网络才是"主体"）
- 蘑菇无叶绿素，不能进行光合作用，不是植物
- 蘑菇细胞壁是几丁质（不是纤维素），储能物质是糖原（不是淀粉）
