# 安全用药
> **一句话**：安全用药是指根据病情需要，在正确的时间，选用适当的药物，以适当的剂量、剂型，通过适当的途径，安全有效地治疗疾病，避免不良反应。
> **English**: Safe medication use means selecting the appropriate drug, dose, route, and timing based on medical need, to treat disease effectively while avoiding adverse effects.

---

## 📚 给 6-12 岁（自读版）

药物可以治病，但用错了也会伤身体。**安全用药**就是学会正确使用药物。

有些药要医生开处方才能买（叫**处方药**），有些药可以自己在药店买（叫**非处方药**，包装上有 OTC 标志）。

用药时要注意：
- 看清楚**说明书**，了解用法、用量、副作用
- 不随便**加量**或**减量**
- 注意**有效期**，过期的药不能用
- 不同的药不能随便**混着吃**
- 有过敏史要提前告诉医生

## 🎓 给 12+（进阶版）

### 定义
安全用药（safe medication use）是指在明确诊断的基础上，依据个体情况（年龄、体重、病情、肝肾功能等），选择合理的药物种类、剂量、剂型、给药途径和疗程，以最大疗效和最小风险治疗疾病，避免药物不良反应、药物相互作用和滥用。

### 原理 / 机制

**安全用药的核心原则（"五正确"）**：

| 原则 | 说明 |
|------|------|
| 正确药物 | 对症选药，不滥用抗生素 |
| 正确剂量 | 按说明书或医嘱用量，不自行加减 |
| 正确时间 | 按规定频次用药（如饭前/饭后）|
| 正确途径 | 口服/注射/外用等，不混用 |
| 正确患者 | 儿童、孕妇、老人需特殊用药 |

**常见用药误区**：

| 误区 | 风险 |
|------|------|
| 滥用抗生素 | 产生耐药菌，损害肝肾 |
| 自行停药（抗生素未用满疗程）| 病原体未被彻底消灭，复发 |
| 药物混用 | 药物相互作用，增加毒性 |
| 超剂量用药 | 中毒风险 |
| 过期药物 | 药效下降或产生有害物质 |

**处方药与非处方药的区别**：

| 类型 | 定义 | 标志 | 购买方式 |
|------|------|------|----------|
| 处方药（Rx） | 需医师诊断并开具处方 | Rx | 凭处方在药店购买 |
| 非处方药（OTC）| 不需处方，可自行判断购买 | OTC（甲类红色/乙类绿色）| 药店自行购买 |

### 典型例题
**题**：为什么感冒发烧不应该随便使用抗生素？

**解**：普通感冒和流感多由**病毒**引起，抗生素只对**细菌**有效，对病毒无治疗作用。随意使用抗生素不仅无效，还会破坏肠道正常菌群，诱导细菌产生耐药性，增加未来治疗的难度。

### 易错点
- 抗生素 ≠ 消炎药（抗生素针对细菌；非甾体抗炎药针对炎症反应）
- OTC 药物虽然无需处方，但仍有不良反应和禁忌证，需仔细阅读说明书
- 儿童不是缩小版的成人，许多成人药物不能直接给儿童使用（需儿童专用剂量/剂型）
