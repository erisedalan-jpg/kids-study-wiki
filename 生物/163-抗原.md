# 抗原
> **一句话**：抗原是能刺激机体产生抗体并与抗体发生特异性结合的物质，通常是病原体表面的特定分子。
> **English**: An antigen is a substance that stimulates the body to produce antibodies and can specifically bind to them; it is typically found on the surface of pathogens.

---

## 📚 给 6-12 岁（自读版）

想象一下，细菌或病毒表面有一些"特殊标记"，就像坏人的"面孔特征"。我们的免疫系统一旦认出这些标记，就会立刻发出警报，产生"武器"来攻击它们。

这些**让免疫系统认出病原体的"标记"**，就叫**抗原**。

抗原不只在病原体上有，移植的器官也携带抗原（因为不是"自己"的），所以器官移植需要"配型"，减少排斥反应。

---

## 🎓 给 12+（进阶版）

### 定义
抗原（antigen，Ag）是能刺激机体免疫系统产生特异性免疫应答，并能与相应抗体或免疫细胞特异性结合的物质。通常是蛋白质或多糖，分布于病原体、移植器官、肿瘤细胞等表面。

### 原理 / 机制

**抗原的特性**：

| 特性 | 说明 |
|------|------|
| 异物性 | 通常是外来物质（非自身的），如病原体、花粉、移植器官 |
| 大分子性 | 多为蛋白质、多糖等大分子物质 |
| 特异性 | 一种抗原只能刺激产生一种对应的抗体 |

**抗原的来源**：

| 来源 | 举例 |
|------|------|
| 病原体 | 细菌细胞壁蛋白、病毒蛋白外壳 |
| 异体移植 | 他人器官的细胞表面蛋白 |
| 疫苗 | 灭活/减毒病原体或其特定蛋白（人工抗原） |
| 自身变异细胞 | 肿瘤细胞表面新抗原 |

**抗原与抗体的关系（锁钥关系）**：
```
特定抗原（"钥匙"）→ 刺激 B 淋巴细胞 → 产生特异性抗体（"锁"）
抗原 + 抗体 → 特异性结合 → 抗原被清除
```

**疫苗中的抗原**：
- 灭活疫苗：杀死的病原体（有抗原，无致病性）
- 减毒活疫苗：毒力减弱的病原体
- 亚单位疫苗：只含病原体特定蛋白（纯抗原）

### 典型例题
**题**：接种流感疫苗后，疫苗中的什么物质刺激机体产生抗体？

**解**：疫苗中含有灭活（或减毒）的流感病毒成分，其表面的**抗原**（蛋白质等）刺激机体免疫系统产生相应的**抗体**和记忆细胞。

### 易错点
- 抗原不一定是病原体，移植器官、花粉也是抗原
- 同一生物体的正常自身细胞通常不引起自身免疫应答（有自身耐受机制）
- 抗原特异性决定了一种疫苗只能预防特定疾病
