# 不完全变态

> **一句话**：不完全变态是昆虫发育只经历**卵→若虫→成虫** 3个阶段、没有蛹期的变态类型，若虫外形与成虫相似。
> **English**: Incomplete metamorphosis is insect development with only three stages—egg → nymph → adult—with no pupal stage; nymphs resemble adults in form.

---

## 📚 给 6-12 岁（自读版）

蝗虫从小时候到大时候，长得差不多——只是翅膀越来越大，这就是**不完全变态**！

**不完全变态 3 步：**

| 阶段 | 特征 | 蝗虫的例子 |
|------|------|----------|
| 🥚 卵 | 产在土里越冬 | 卵块埋在地下 |
| 🦗 若虫 | 像小成虫，无翅或翅不全 | 跳蝻（小蝗虫） |
| 🦗 成虫 | 翅长全，可繁殖 | 成年蝗虫 |

**关键特点：** 若虫和成虫**长得很像**，只是没有翅膀或翅膀没长全，**没有蛹期**！

**常见不完全变态昆虫：**
- 蝗虫（若虫叫"跳蝻"）
- 蟋蟀
- 螳螂
- 蜻蜓（若虫生活在水中，叫稚虫）
- 蟑螂
- 蝉（若虫在土里，叫"知了猴"）

**记住：没有蛹 = 不完全变态！**

## 🎓 给 12+（进阶版）

### 定义

**不完全变态**（Incomplete Metamorphosis / Hemimetabolism）：昆虫个体发育仅经历**卵→若虫（nymph）→成虫** 3个阶段，**无蛹期**。若虫在形态、食性上与成虫相似，随蜕皮逐渐发育，翅芽逐渐增大，性器官逐渐成熟。

### 原理 / 机制

**若虫发育特征：**

| 特征 | 说明 |
|------|------|
| 外形 | 与成虫基本相似，无翅或翅芽未展开 |
| 食性 | 与成虫相似（同一食物来源） |
| 蜕皮次数 | 多次蜕皮（蝗虫5龄，每龄一次蜕皮）后成虫 |
| 翅的发育 | 翅芽在体表可见，逐步发育，末次蜕皮后翅展开 |

**不完全变态代表昆虫：**

| 目 | 代表昆虫 | 若虫名称 |
|----|---------|---------|
| 直翅目 | 蝗虫、蟋蟀、螳螂 | 若虫（跳蝻） |
| 蜻蜓目 | 蜻蜓、豆娘 | 稚虫（水生） |
| 半翅目 | 蝽（臭虫）、蝉 | 若虫 |
| 蜚蠊目 | 蟑螂 | 若虫 |

**注：** 蜻蜓稚虫生活在水中，用鳃呼吸，成虫在空中飞翔，虽形态差别较大，但无蛹期，仍属不完全变态。

**蝗虫发育（教材重点）：**

- 卵 → 1龄若虫（跳蝻）→ … → 5龄若虫 → 成虫
- 每次蜕皮后体型增大，翅芽增大
- 成虫期：翅完全展开，具生殖能力

### 典型例题

**例题**：蝗虫的若虫（跳蝻）和成虫相比，主要区别是（ ）
A. 食性不同
B. 翅未发育完全，无生殖能力
C. 生活环境不同
D. 体色不同

**答案**：B

**解析**：跳蝻（若虫）与成虫食性相同（均取食植物），生活环境也相同。主要区别是若虫翅芽未完全发育（无功能翅），性器官未成熟，不具有繁殖能力。

### 易错点

- 蜻蜓是不完全变态（尽管稚虫水生、成虫陆生，但无蛹期）
- "若虫"专指不完全变态的幼期，"幼虫"专指完全变态的幼期，两者不能混用
- 蝉的若虫（知了猴）在土中生活多年，出土后蜕最后一次皮成为成虫，**不是**蛹
- 蟑螂属不完全变态，很多人误以为蟑螂是完全变态
