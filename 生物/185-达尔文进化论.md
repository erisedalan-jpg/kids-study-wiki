# 达尔文进化论
> **一句话**：达尔文进化论以自然选择为核心，认为生物通过过度繁殖、遗传变异和自然选择，经过漫长时间逐渐演化，适者生存不适者淘汰。
> **English**: Darwinian evolution, centered on natural selection, proposes that organisms evolve gradually through overproduction, heritable variation, and natural selection—with the fittest surviving and others being eliminated.

---

## 📚 给 6-12 岁（自读版）

达尔文是 19 世纪的英国科学家，他乘坐"小猎犬号"船环球旅行，观察了很多地方的动植物，发现了一个重大秘密：**生物会慢慢改变，变得越来越适应环境**。

他写了一本著名的书《物种起源》，提出了**进化论**，主要有三个核心观点：
1. 生物产生的后代比能活下来的多得多（**过度繁殖**）
2. 后代之间有差异（**变异**）
3. 能适应环境的个体活下来繁殖，不适应的被淘汰（**自然选择，适者生存**）

达尔文还观察到加拉帕戈斯群岛上的不同岛屿有不同的地雀——它们嘴巴形状各不相同，分别适合吃不同食物，正是自然选择的结果。

## 🎓 给 12+（进阶版）

### 定义
达尔文进化论（Darwinian evolution）是查尔斯·达尔文（Charles Darwin）在 1859 年《物种起源》（*On the Origin of Species*）中系统提出的生物进化理论，以**自然选择**为核心机制，解释物种多样性及生物适应性的起源。

### 原理 / 机制

**达尔文进化论的核心论点**：

| 论点 | 说明 |
|------|------|
| 过度繁殖 | 生物繁殖后代数量远超环境承载量 |
| 遗传变异 | 个体间存在可遗传的差异（变异随机产生） |
| 生存竞争 | 因资源有限，个体间产生竞争 |
| 自然选择 | 有利变异个体存活率和繁殖率更高 |
| 适者生存 | 有利变异逐代积累，种群特征改变 |

**达尔文主要证据**：

| 证据 | 来源 |
|------|------|
| 加拉帕戈斯地雀 | 不同岛屿地雀嘴形适应不同食物 |
| 化石记录 | 不同地层生物形态变化序列 |
| 比较解剖 | 同源器官（如脊椎动物四肢骨骼） |
| 人工选择类比 | 人类培育品种（如各种犬品种）说明选择的力量 |

**达尔文与拉马克的比较**：

| | 拉马克 | 达尔文 |
|--|--------|--------|
| 变异来源 | 环境直接引起（用进废退）| 遗传变异（随机，预先存在） |
| 选择机制 | 获得性遗传 | 自然选择 |
| 科学地位 | 已被否定 | 现代进化论基础 |

### 典型例题
**题**：达尔文进化论如何解释长颈鹿的长颈是如何进化来的？

**解**：长颈鹿祖先种群中存在颈长的遗传变异（有长有短，随机产生）。在食物竞争中，颈长的个体能获取高处叶子，生存率和繁殖率更高；颈短的个体竞争失败，逐渐被淘汰。经过长期自然选择，颈长变异积累，长颈鹿颈部越来越长。

### 易错点
- 达尔文进化论中变异是**随机**的，不是"为了适应"而产生（这是拉马克的错误）
- 自然选择作用于表现型，而非直接作用于基因（基因概念是后来孟德尔提出的，达尔文时代尚不知晓）
- 达尔文进化论不能解释所有进化现象（如基因漂变），现代综合进化论是其发展
