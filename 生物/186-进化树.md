# 进化树
> **一句话**：进化树（系统发育树）是以树状图形式展示生物类群间进化关系和共同祖先的示意图。
> **English**: A phylogenetic tree (evolutionary tree) is a diagram that represents the evolutionary relationships and common ancestors among biological groups in a tree-like form.

---

## 📚 给 6-12 岁（自读版）

你见过家谱吗？生物也有自己的"家谱"，叫**进化树**。

进化树用树枝分叉来表示生物之间的亲缘关系：两类生物的分支越近，说明它们**亲戚关系越近**，共同祖先也越近。

比如：人和黑猩猩的分支很近，说明我们有共同祖先；人和鱼的分支很远，关系就更"远"。树根代表所有生物的**共同祖先**（最早的生命）。

科学家通过比较化石、骨骼结构、DNA 等资料，来画出生物的进化树。

---

## 🎓 给 12+（进阶版）

### 定义
进化树（phylogenetic tree，又称系统发育树）是用分支树状结构表示生物类群之间进化关系的图示工具。树的节点代表共同祖先，枝条代表进化分支，分支点越近表示亲缘关系越近，末端节点代表现存或已灭绝的生物类群。

### 原理 / 机制

**进化树的基本结构**：

| 元素 | 含义 |
|------|------|
| 根（root） | 所有生物的共同祖先 |
| 内部节点（node） | 两个或多个分支的共同祖先 |
| 枝条（branch） | 进化分支，长度可代表时间或变异量 |
| 末端节点（tip/leaf） | 现存物种或灭绝物种 |

**构建进化树的依据**：

| 依据 | 说明 |
|------|------|
| 化石证据 | 不同地层生物形态变化 |
| 比较解剖 | 同源器官的形态比较 |
| 胚胎发育 | 早期胚胎相似性 |
| 分子数据 | DNA/蛋白质序列相似度（现代方法，亲缘越近相似度越高） |

**脊椎动物进化树（简化）**：

```
原始鱼类
 ├── 软骨鱼（鲨鱼）
 └── 硬骨鱼
      └── 古代两栖类（肺鱼→四足类）
           └── 两栖动物
                └── 古代爬行类
                     ├── 爬行动物（现存）
                     ├── 鸟类
                     └── 哺乳动物
                          └── 灵长类→人类
```

### 典型例题
**题**：在进化树中，两个物种的共同节点越近，说明什么？

**解**：说明两个物种的**亲缘关系越近**，共同祖先存在的时间越近，遗传差异越小。

### 易错点
- 进化树中的分支方向不代表"高级"或"低级"，只代表不同的进化方向
- 进化树是科学推测，随新证据（如古 DNA）不断修正
- 树根不是"最低等"，而是最早的共同祖先
