# 吸烟酗酒危害

> **一句话**：吸烟和酗酒会损害呼吸系统、心血管、肝脏等多器官，造成成瘾，是青少年应当严格避免的健康风险行为。
> **English**: Smoking and alcohol abuse damage multiple organs and cause addiction; teens must avoid them.

---

## 📚 给 6-12 岁（自读版）

香烟、酒看着像"大人的东西"，但它们对身体的伤害很大，**未成年人绝对不能碰**。

**吸烟的危害：**
- 烟里有 7000 多种化学物质，至少 70 种致癌（肺癌、口腔癌、喉癌……）
- 牙齿黄、口臭
- 肺变黑、咳嗽
- 心脏病、中风风险增加
- 还伤害旁边人——叫"二手烟"
- 越早吸越上瘾、越难戒

**喝酒的危害：**
- 酒精进入大脑会让人头晕、说话乱
- 长期喝伤肝（脂肪肝→肝硬化）
- 伤胃、胰腺
- 影响青少年大脑发育（大脑还没发育成熟）
- 酗酒易暴力、车祸
- 也会成瘾

**为什么大人能喝/吸而我们不行？**
- 青少年身体大脑还在发育，**伤害更大**
- 法律规定未成年人不许买烟酒（保护你）
- 越早接触，成瘾风险越高
- 大人虽然能喝，也只是"风险较小"，并不是"安全"

**遇到劝烟劝酒怎么办？**
- 直接拒绝："我不抽烟（喝酒）"
- 找借口离开
- 告诉父母老师
- 远离这种朋友圈

## 🎓 给 12+（进阶版）

### 烟草危害

**烟雾成分**：尼古丁（致瘾物）、焦油、CO、苯、甲醛、放射性物质等 7000+ 化合物，70+ 种致癌物。

**主要器官损害：**

| 器官 | 损害 |
|------|------|
| 呼吸系统 | 慢性支气管炎、肺气肿、肺癌（约 80% 肺癌与吸烟相关） |
| 心血管 | 冠心病、心肌梗死、动脉硬化、中风 |
| 口腔 | 口腔癌、喉癌、牙周病 |
| 消化系统 | 食管癌、胃癌、胃溃疡 |
| 生殖系统 | 男性精子质量下降；女性流产、低体重儿 |
| 皮肤 | 皮肤老化、皱纹早 |

**二手烟（环境烟草烟）：**
被动吸烟者也会患吸烟相关疾病，孕妇接触影响胎儿发育。

**三手烟**：附着在衣物、家具上的残留物，对儿童尤其危害。

### 酒精危害

**酒精代谢**：乙醇 →（乙醇脱氢酶）→ 乙醛 →（乙醛脱氢酶）→ 乙酸 → CO₂ + H₂O。亚洲人乙醛脱氢酶基因变异多 → "脸红基因" → 乙醛累积更易致癌。

**急性危害：**
- 中枢神经抑制：判断力下降、动作不协调
- 急性酒精中毒：意识障碍、呼吸抑制、可致死
- 酒驾：法律严惩

**慢性危害：**

| 系统 | 危害 |
|------|------|
| 肝脏 | 脂肪肝 → 酒精性肝炎 → 肝硬化 → 肝癌 |
| 神经 | 大脑萎缩、记忆下降、酒精依赖 |
| 消化 | 胃炎、胃溃疡、胰腺炎、食管癌 |
| 心血管 | 高血压、心肌病、心律失常 |
| 生殖 | 影响精子卵子质量；孕期饮酒致胎儿酒精综合征 |
| 营养 | 影响维 B 吸收，导致脚气病等 |

**青少年特殊风险：**
- 大脑发育至 25 岁仍在进行，酒精尤其有害
- 越早饮酒，成年后酒精依赖风险越高
- 法律禁止售酒给未成年人

### 成瘾机制

| 物质 | 致瘾成分 | 机制 |
|------|--------|------|
| 烟草 | 尼古丁 | 激活脑部奖赏通路（多巴胺释放） |
| 酒精 | 乙醇 | 抑制中枢；GABA、多巴胺通路 |

成瘾标志：耐受性、戒断症状、用量逐渐增加、影响生活、屡戒屡复。

### 戒烟戒酒

| 方法 | 说明 |
|------|------|
| 自我决心 + 支持系统 | 家人朋友帮助 |
| 替代疗法 | 戒烟贴、戒烟药、咀嚼物 |
| 行为疗法 | 心理咨询、CBT |
| 社区资源 | 戒烟门诊、AA 互助会 |
| 严重者 | 专业医院戒断治疗 |

### 法律与健康政策

- 《未成年人保护法》：禁止向未成年人售卖烟酒
- 公共场所禁烟（《公共场所控制吸烟条例》）
- 香烟包装健康警示
- 酒驾入刑（醉驾构成"危险驾驶罪"）

### 易错点

- "少量烟酒无害"——错。烟草无安全剂量；酒精虽有低风险阈值，但青少年完全不应饮
- 吸烟伤己也害人（二手烟）
- 戒断初期不适是正常的，但能克服
- "酒驾"不分酒量大小，超过法定值即违法
