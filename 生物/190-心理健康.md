# 心理健康

> **一句话**：个体在认知、情感、行为等方面保持良好状态，能有效应对生活压力、与人和谐相处的精神状态。
> **English**: Mental health is a state of well-being in cognition, emotion, and behavior, enabling one to cope with life and contribute to community.

---

## 📚 给 6-12 岁（自读版）

身体健康很重要，**心理健康**同样重要！

**心理健康的小朋友：**
- 大部分时候开心、有兴趣
- 难过时能慢慢恢复
- 能交到朋友、和家人沟通
- 遇到挫折不轻易放弃
- 知道自己的感受是什么

**心理不健康的信号：**
- 长时间不开心、什么都不想做
- 睡不着或睡太多
- 不想吃饭或吃太多
- 怕见人、不想去学校
- 总是发脾气

**怎么保持心理健康？**
- 规律作息：睡好觉
- 多运动：让大脑分泌快乐物质
- 找人说话：跟父母老师朋友倾诉
- 喜欢的事：画画、读书、玩
- 不憋着：哭出来也没关系
- 生病要看医生——心理不舒服也要找心理医生

**记住：** 觉得难过、烦躁、害怕都是正常的。**重要的是要说出来，找人帮你**。

## 🎓 给 12+（进阶版）

### 定义

**心理健康（mental health）**：根据世界卫生组织定义，是一种"福祉状态，使个人能认识到自己的能力、应对生活的正常压力、富有成效地工作并能为社区作出贡献"。

### 心理健康的标志

| 维度 | 表现 |
|------|------|
| 自我认知 | 客观了解自己、接纳自己 |
| 情绪管理 | 情绪稳定、能合理表达 |
| 人际关系 | 与人和睦相处 |
| 适应能力 | 能应对环境变化 |
| 学习/工作能力 | 能专注、有效完成任务 |
| 意志力 | 面对困难有坚持力 |

### 影响因素

| 因素 | 内容 |
|------|------|
| 生物因素 | 遗传、激素、神经递质（5-羟色胺、多巴胺） |
| 心理因素 | 性格、思维方式、应对方式 |
| 环境因素 | 家庭、学校、社会 |
| 生活方式 | 睡眠、饮食、运动、电子屏幕时间 |

### 青少年常见心理问题

| 问题 | 表现 | 应对 |
|------|------|------|
| 学业压力 | 焦虑、失眠、厌学 | 合理目标、放松训练 |
| 抑郁倾向 | 持续低落、兴趣丧失、消极想法 | 寻求专业帮助 |
| 焦虑障碍 | 过度担心、心慌 | 认知行为疗法、规律生活 |
| 网络/手机依赖 | 沉迷游戏/社交媒体 | 限时使用、培养兴趣替代 |
| 同伴冲突 | 孤立、被排挤 | 沟通、求助老师家长 |
| 身体形象焦虑 | 过度关注容貌、节食 | 接纳自我、健康观念 |

### 维护心理健康的方法

| 方法 | 说明 |
|------|------|
| 规律作息 | 充足睡眠（青少年 8-10 小时） |
| 体育锻炼 | 释放内啡肽、改善心情 |
| 营养均衡 | 大脑需要均衡营养 |
| 兴趣爱好 | 减压、获得成就感 |
| 社交支持 | 家人、朋友、师长倾诉 |
| 学会拒绝 | 边界感、不被消耗 |
| 求助专业 | 必要时找心理医生 |

### 求助途径

- 父母、老师、信任的成年人
- 学校心理辅导中心
- 12355 青少年服务台
- 心理咨询师 / 精神科医生
- 学校心理教师

### 易错点

- 心理健康问题不是"软弱"，是健康问题
- 抑郁、焦虑等症状持续 2 周以上需要专业评估
- 找心理医生不丢人，跟看牙医一样正常
- 家人朋友的支持很重要，但**严重时仍需专业治疗**
