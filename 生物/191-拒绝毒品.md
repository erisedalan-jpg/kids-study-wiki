# 拒绝毒品

> **一句话**：毒品对身体和心理有严重危害，会形成成瘾，初中生应当树立坚决拒绝毒品的意识。
> **English**: Drugs cause severe physical and psychological harm and lead to addiction; students should firmly refuse all drugs.

---

## 📚 给 6-12 岁（自读版）

**毒品是绝对不能碰的东西。** 中学课本和法律都告诉我们：远离毒品。

**毒品的危害：**
- **伤害大脑**：让人控制不住自己
- **伤害身体**：心、肺、肝、肾全部受损
- **成瘾**：一旦开始，很难戒掉，越用越多
- **毁掉生活**：花光积蓄、家庭破碎、辍学失业
- **传染疾病**：共用针头容易传染艾滋病、肝炎
- **违法**：吸毒和贩毒都是犯罪

**常见毒品（不要碰、不要尝试）：**
鸦片、海洛因、冰毒、摇头丸、可卡因、大麻、止咳药水（滥用也是毒品）等。

**怎么拒绝？**
- 不好奇——"试一下"就可能上瘾
- 不轻信陌生人给的食品饮料
- 不去鱼龙混杂的场所
- 朋友劝吸毒就远离他
- 学一句"不！"——大声说出来
- 告诉父母、老师，必要时报警

**遇到这些情况要警惕：**
- 有人给"零食、饮料"白白送
- 朋友约去陌生地方
- 有人说"试一下没事"
- 家庭遇到困难想"逃避"

**记住：** 一次都不要尝试。**毒品不会因为只用一次就放过你**。

## 🎓 给 12+（进阶版）

### 定义

**毒品**：根据《禁毒法》，指鸦片、海洛因、甲基苯丙胺（冰毒）、吗啡、大麻、可卡因以及国家规定管制的其他能够使人形成瘾癖的麻醉药品和精神药品。

### 毒品分类（医学角度）

| 类型 | 主要作用 | 例子 |
|------|--------|------|
| 麻醉药品（中枢抑制） | 镇痛、抑制呼吸 | 鸦片、海洛因、吗啡 |
| 精神药品（中枢兴奋） | 兴奋、致幻 | 冰毒、摇头丸、可卡因 |
| 致幻剂 | 改变知觉 | LSD、大麻 |
| 处方药滥用 | 各种 | 止咳药水（含可待因）、镇静类 |

### 危害机制

**生理危害：**
- 中枢神经损伤、记忆力下降
- 心脑血管损害（心律失常、脑梗）
- 免疫系统损害
- 肝肾损害
- 共用针头 → HIV、肝炎传播
- 急性中毒可致死

**成瘾机制：**
- 多巴胺奖赏系统被劫持
- 反复使用 → 受体下调 → 需要更大剂量获得相同感觉（**耐受**）
- 停药 → **戒断症状**（疼痛、呕吐、抽搐、自杀念头）

**社会危害：**
- 违法犯罪率升高
- 家庭破裂、经济崩溃
- 公共卫生威胁
- 国家禁毒成本巨大

### 易感因素

| 因素 | 说明 |
|------|------|
| 好奇心 | "试一下"心理 |
| 同伴压力 | 朋友怂恿 |
| 心理压力 | 寻求逃避 |
| 家庭环境 | 监管缺失 |
| 错误观念 | "一次没事""我能控制"——错误！ |

### 拒绝毒品的方法

| 方法 | 内容 |
|------|------|
| 增强意识 | 学习毒品危害知识 |
| 拒绝技巧 | 直接说"不"、找借口离开、转移话题 |
| 远离环境 | 避开娱乐场所等高危环境 |
| 选择朋友 | 远离吸毒者 |
| 寻求帮助 | 告诉家长、老师、报警 |
| 健康减压 | 体育锻炼、艺术爱好等替代 |

### 法律责任

- 《禁毒法》《刑法》明确：
  - 走私、贩卖、运输、制造毒品 → 最高死刑
  - 吸食、注射毒品 → 行政拘留、强制隔离戒毒
  - 容留他人吸毒 → 刑事责任
- 全国统一禁毒举报电话：**110** 或 **12110**

### 易错点

- 处方药滥用同样是吸毒——止咳药水、安眠药等过量也算
- 毒品没有"安全剂量"——一次接触就可能成瘾
- "新型毒品"伪装成糖果、饮料，要警惕
- 戒毒成功率不高，**不沾染**才是最好的"戒毒"
