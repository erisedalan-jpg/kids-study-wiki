# 基因

> **一句话**：**基因**是DNA分子上具有**遗传效应**的特定片段，是控制生物性状的功能单位。
> **English**: A gene is a specific segment of DNA with hereditary function, serving as the basic unit controlling an organism's traits.

---

## 📚 给 6-12 岁（自读版）

你的每个细胞里，都藏着一部"说明书"，告诉身体怎么长眼睛、怎么生长、血型是什么……这部说明书里的每一个"章节"就叫**基因**！

**基因的位置：** 细胞核 → 染色体 → DNA → 基因

**记忆层级（从大到小）：**
细胞核 > 染色体 > DNA > 基因

**基因做什么？**
- 控制生物的**性状**（如眼皮单双、血型）
- 通过**遗传**传递给下一代
- 决定蛋白质的种类和数量

**人有多少基因？**
- 人类约有 **20,000-25,000** 个基因
- 基因只占 DNA 的很小一部分（约 2%）！

**基因的故事：** 1865年，孟德尔通过豌豆实验，发现了遗传规律，提出了"遗传因子"（后来改称"基因"）的概念。

**用D、d表示：**
- **D**（大写）= 显性基因（控制显性性状）
- **d**（小写）= 隐性基因（控制隐性性状）

## 🎓 给 12+（进阶版）

### 定义

**基因**（Gene）：是DNA分子上具有**特定碱基序列**、能编码**功能性产物**（蛋白质或RNA）的**遗传单位**。基因通过控制蛋白质的合成来控制生物体的代谢和性状。

### 原理 / 机制

**基因的化学本质：**
- 基因的化学成分是 **DNA**（脱氧核糖核酸）
- 由**碱基序列**（A、T、G、C的排列顺序）编码遗传信息
- 一个基因通常含有几百到几千个碱基对

**基因与染色体的关系：**

| 层次 | 说明 |
|------|------|
| 染色体 | 由DNA和蛋白质组成，线状或环状 |
| DNA | 链状大分子，携带遗传信息 |
| 基因 | DNA上有特定功能的片段，非连续分布 |
| 等位基因 | 在同源染色体上控制同一性状的一对基因（如A和a） |

**基因表达：**

$$\text{DNA} \xrightarrow{\text{转录}} \text{mRNA} \xrightarrow{\text{翻译}} \text{蛋白质} \rightarrow \text{性状}$$

**人类基因组：**
- 人类基因组含约 **30亿** 个碱基对
- 约 **20,000-25,000** 个蛋白质编码基因
- 基因组编码区约占 **2%**，其余为非编码区（调控序列等）

**等位基因与遗传：**
- 人体细胞 **23对** 染色体（46条）
- 每对同源染色体上各有一个等位基因
- 生殖细胞只含 **23条**（单倍体），每对等位基因只留一个

### 典型例题

**例题**：基因、DNA、染色体三者之间的关系，描述正确的是（ ）
A. 基因是DNA的一段，DNA是染色体的主要成分
B. 染色体是DNA的一段，DNA是基因的主要成分
C. DNA是基因的一段，染色体是DNA的主要成分
D. 基因是染色体的一段，染色体是DNA的主要成分

**答案**：A

**解析**：基因是DNA上有特定功能的片段（基因⊂DNA），DNA是染色体的主要成分（DNA+蛋白质组成染色体）。

### 易错点

- 基因 ≠ DNA（基因是DNA的一部分，有功能的片段；大量DNA是非编码区）
- 基因型（Aa）≠ 基因（A或a）：基因型是个体拥有的等位基因组合
- 所有细胞（除成熟红细胞）都含有相同的基因组，但不同细胞表达不同基因（基因的差异表达）
- 基因突变 ≠ 染色体变异（前者是基因本身DNA序列改变，后者是染色体结构/数目改变）
