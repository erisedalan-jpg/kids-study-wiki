# 显性性状
> **一句话**：显性性状是具有相对性状的亲本杂交后，子一代中能表现出来的性状。
> **English**: A dominant trait is the trait expressed in the first generation offspring when parents with contrasting traits are crossed.

---

## 📚 给 6-12 岁（自读版）

孟德尔种豌豆时发现，把**高茎豌豆**和**矮茎豌豆**杂交，后代全部是高茎——矮茎好像"消失"了。这种**杂交后能表现出来的性状**，就叫**显性性状**。

显性性状背后有一个"强"基因（显性基因），用大写字母表示，比如 A。只要有一个 A，这个性状就能"出现"。

豌豆常见的显性性状有：高茎（相对矮茎）、圆粒（相对皱粒）、黄色种子（相对绿色种子）。

---

## 🎓 给 12+（进阶版）

### 定义
显性性状（dominant trait）是指在具有相对性状的纯合亲本杂交的子一代（F₁）中表现出来的那个亲本性状。控制显性性状的基因称为显性基因，通常用大写字母表示（如 A）。

### 原理 / 机制

| 概念 | 说明 |
|------|------|
| 显性性状 | F₁ 中表现的性状 |
| 隐性性状 | F₁ 中未表现、F₂ 中重新出现的性状 |
| 显性基因 | 大写字母（A），一个即可使显性性状表现 |
| 基因型 | AA（显性纯合）/ Aa（显性杂合）均表现显性性状 |
| 隐性基因型 | aa，只有纯合隐性才表现隐性性状 |

**孟德尔豌豆七对相对性状（显性/隐性）**：

| 显性性状 | 隐性性状 |
|----------|----------|
| 高茎 | 矮茎 |
| 圆粒 | 皱粒 |
| 黄色种子 | 绿色种子 |
| 灰色种皮 | 白色种皮 |
| 饱满豆荚 | 收缩豆荚 |
| 绿色豆荚 | 黄色豆荚 |
| 腋生花 | 顶生花 |

**显隐性判断方法**：
1. 亲本杂交 → F₁ 表现的即为显性
2. F₂ 出现性状分离 → 比例 3:1，多的为显性
3. 纯合亲本自交后代不变 → 该性状为其纯合性状

### 典型例题
**题**：豌豆高茎（D）对矮茎（d）为显性，Dd 个体的表现型是什么？

**解**：Dd 含显性基因 D，表现为**高茎**（显性性状）。

### 易错点
- 显性性状 ≠ 数量多的性状（F₂ 中显性 3 份：隐性 1 份，但这只是遗传规律，不代表自然界中显性一定多）
- 显隐性是相对的，不同物种、不同性状对应不同的显隐关系
