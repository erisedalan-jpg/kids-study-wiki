# 染色体
> **一句话**：染色体是细胞核中能被碱性染料染色的线状或棒状结构，是基因的主要载体。
> **English**: Chromosomes are thread-like structures in the cell nucleus that carry genes and are stained by basic dyes.

---

## 📚 给 6-12 岁（自读版）

我们每个人都长得不一样，秘密藏在细胞核里。细胞核里有一种叫**染色体**的东西，像一根根细线，上面排列着大量的**基因**——基因就是控制我们长什么样的"指令"。

人体每个细胞里有 **46 条染色体**，排成 23 对。其中有一对叫**性染色体**，决定我们是男是女。爸爸妈妈各贡献一半染色体，所以我们长得既像爸爸又像妈妈。

染色体在细胞分裂时会变短变粗，在显微镜下可以看到。科学家用特殊染料把它们染色，所以叫"染色体"。

---

## 🎓 给 12+（进阶版）

### 定义
染色体（chromosome）是真核细胞细胞核中由 DNA 和蛋白质组成的线状结构，是遗传信息的主要载体。在细胞分裂间期以染色质形式存在，分裂期高度螺旋化形成可见的染色体形态。

### 原理 / 机制

| 特征 | 说明 |
|------|------|
| 组成 | DNA + 组蛋白（染色质蛋白） |
| 分布 | 细胞核（主要），线粒体/叶绿体中也有少量 DNA |
| 人类染色体数 | 体细胞 46 条（23 对）；生殖细胞 23 条 |
| 常染色体 | 22 对，男女相同 |
| 性染色体 | 1 对：女 XX，男 XY |
| 染色体与基因 | 基因在染色体上呈线性排列，一条染色体含多个基因 |

**染色质与染色体的关系**：
- 染色质（间期）⇌ 染色体（分裂期）：同一物质在不同时期的两种存在状态
- 染色体 = DNA 高度螺旋化 + 蛋白质支架

**染色体数目变化（减数分裂）**：

| 时期 | 染色体数 |
|------|----------|
| 体细胞（2n=46） | 46 |
| 减数第一次分裂后 | 23（每条为双染色单体） |
| 减数第二次分裂后（配子） | 23 |
| 受精后（合子） | 46 |

### 典型例题
**题**：人的体细胞中有 46 条染色体，卵细胞中有多少条？

**解**：卵细胞是经减数分裂形成的，染色体数减半，共 **23 条**。

### 易错点
- 染色质与染色体是同一物质，不同状态，不能混淆
- 体细胞 46 条，不是 46 对；配对后是 23 对
- 基因在染色体上，但染色体 ≠ 基因（一条染色体含多个基因）
