# 表现型
> **一句话**：表现型是生物体可以观察到的性状表现，由基因型与环境共同决定。
> **English**: Phenotype refers to the observable traits of an organism, determined by genotype and environment.

---

## 📚 给 6-12 岁（自读版）

你见过豌豆有的是圆粒、有的是皱粒吗？这种**能看到或感受到的样子**，就叫**表现型**。比如高茎、矮茎、红花、白花，都是表现型。

表现型是基因和环境"合作"的结果：基因就像"蓝图"，环境就像"施工条件"——两者共同决定生物最后长成什么样。

同样的表现型，背后的基因可能不同。比如高茎豌豆，可能是 AA，也可能是 Aa，但外貌看起来一样高。

---

## 🎓 给 12+（进阶版）

### 定义
表现型（phenotype）是生物个体在遗传基础（基因型）和环境条件共同作用下，所表现出来的一切可观测性状的总称，包括形态、生理、行为等方面。

### 原理 / 机制

| 概念 | 说明 |
|------|------|
| 表现型 | 可直接观察的性状，如豌豆圆粒/皱粒 |
| 基因型 | 控制性状的基因组合，如 AA、Aa、aa |
| 关系式 | 表现型 = 基因型 + 环境 |
| 显性基因 | 基因型含显性基因（AA 或 Aa）→ 显性表现型 |
| 隐性基因 | 基因型为纯合隐性（aa）→ 隐性表现型 |

**关键规律**：
- 相同基因型 + 不同环境 → 可能不同表现型（如同一品种花卉在不同温度下颜色不同）
- 相同表现型 → 可能不同基因型（如高茎可为 AA 或 Aa）
- 不同表现型 → 基因型一定不同（对同一显隐性对而言，aa ≠ AA/Aa）

### 典型例题
**题**：豌豆圆粒（R）对皱粒（r）为显性。基因型为 Rr 的个体，其表现型是什么？

**解**：Rr 含显性基因 R，表现为**圆粒**（显性表现型）。

### 易错点
- 表现型相同 ≠ 基因型相同（AA 与 Aa 均表现高茎）
- 环境可影响表现型，但不改变基因型
- 表现型包括所有可观测性状，不仅限于形态
