# 同源染色体

> **一句话**：在体细胞中成对存在、形态大小基本相同、一条来自父本一条来自母本的两条染色体。
> **English**: Homologous chromosomes are pairs of similar chromosomes, one from each parent, with the same gene loci.

---

## 🎓 高中进阶

### 定义

**同源染色体**：在二倍体生物体细胞中成对存在的、形态和大小**基本**相同的一对染色体。一条来自父方，一条来自母方；两条染色体上具有**相同的基因座位**（但等位基因可能不同）。

### 关键特征

| 特征 | 说明 |
|------|------|
| 来源 | 一条来自父，一条来自母 |
| 形态 | 大小、形状基本相同 |
| 基因座位 | 相同位置控制相同性状 |
| 等位基因 | 同一座位的基因可相同（纯合）或不同（杂合） |
| 行为（减Ⅰ） | 联会、形成四分体、互换、分离 |

### 例外：性染色体

人类 X 与 Y 染色体形态大小差异大，但仍认为是**一对同源染色体**（在减数分裂时配对、分离）。它们仅在小段同源区域配对。

### 同源 vs 非同源

| 项目 | 同源染色体 | 非同源染色体 |
|------|---------|-----------|
| 形态大小 | 基本相同 | 不同 |
| 减Ⅰ行为 | 联会、分离 | 自由组合 |
| 基因 | 控制同样性状 | 控制不同性状 |

### 在减数分裂中的关键行为

1. **联会**（减Ⅰ前期）：同源染色体配对
2. **四分体**：联会后含 4 条姐妹染色单体（2 条 × 2）
3. **交叉互换**：非姐妹染色单体片段交换 → 基因重组
4. **分离**（减Ⅰ后期）：同源染色体分别移向两极

### 同源染色体数量

- 体细胞 2N，同源染色体共 N 对
- 减数分裂后子细胞 n，**没有**同源染色体（减Ⅰ已分离）

### 典型例题

**例**：人体细胞内同源染色体多少对？
**答**：23 对（22 对常染色体 + 1 对性染色体）。

**例**：哪个时期可观察到联会？
**答**：减数第一次分裂前期（粗线期最明显）。

### 易错点

- 同源染色体**不等于**复制后的姐妹染色单体（姐妹染色单体来自一条复制）
- 染色体形态相似但来源都为父或都为母 → **不是**同源染色体
- 染色体数目以着丝粒计数；同源染色体两条算 2 条（不是 1 条）
- 二倍体才有"对"的概念；多倍体（如四倍体）有"组"的概念
