# 有丝分裂

> **一句话**：真核细胞的体细胞分裂方式，染色体复制后平均分配到两个子细胞中，保持染色体数目恒定。
> **English**: Mitosis is the eukaryotic cell division producing two genetically identical daughter cells with the same chromosome number.

---

## 🎓 高中进阶

### 定义

**有丝分裂**：真核细胞分裂的主要方式，经过染色体的复制和均等分配，形成两个染色体数目和遗传组成与母细胞相同的子细胞。

### 分期与现象

| 时期 | 主要现象 |
|------|--------|
| 间期 | DNA 复制、有关蛋白合成；染色质细丝状 |
| 前期 | 染色质螺旋化为染色体；核膜核仁消失；纺锤体形成 |
| 中期 | 染色体着丝粒排列于赤道板；染色体形态最清晰、数目最易观察 |
| 后期 | 着丝粒分裂，姐妹染色单体分开为两条子染色体；移向两极 |
| 末期 | 染色体解螺旋为染色质；核膜核仁重建；纺锤体消失；细胞质分裂 |

### 染色体、染色单体、DNA 数量变化（以 2N 体细胞为例）

| 时期 | 染色体 | 染色单体 | DNA |
|------|------|--------|-----|
| 间期前 | 2N | 0 | 2C |
| 间期后 | 2N | 4N | 4C |
| 前期 | 2N | 4N | 4C |
| 中期 | 2N | 4N | 4C |
| 后期 | 4N | 0 | 4C |
| 末期 | 2N（每细胞） | 0 | 2C（每细胞） |

### 动植物有丝分裂区别

| 项目 | 植物 | 动物 |
|------|------|------|
| 纺锤体形成 | 由两极发出星射线 | 由中心粒（中心体）发出星射线 |
| 细胞质分裂 | 细胞板 → 细胞壁 | 细胞膜从中央向内凹陷 |

### 意义

1. 单细胞生物的繁殖
2. 多细胞生物的生长、修复、更新
3. **保持染色体数目恒定**，遗传稳定

### 典型例题

**例**：某二倍体生物体细胞含 8 条染色体，有丝分裂中期细胞内：染色体数 = ？ 染色单体数 = ？ DNA 数 = ？
**答**：染色体 8、染色单体 16、DNA 16。

**例**：观察根尖细胞有丝分裂的实验中，染色用什么？观察时哪个时期最适合识别染色体数目？
**答**：龙胆紫或醋酸洋红；中期。

### 易错点

- 后期染色体数目**翻倍**（着丝粒分裂导致）
- 染色单体数随 DNA 复制出现，于后期消失
- 有丝分裂保持染色体数目不变（指**子母细胞间**）
- 观察根尖时染色前需解离（盐酸+酒精混合液）
