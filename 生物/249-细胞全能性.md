# 细胞全能性

> **一句话**：已分化细胞仍然具有发育成完整个体的潜能，因为它含有该物种全套遗传信息。
> **English**: Totipotency is the ability of a single cell to develop into a complete organism because it contains the full genome.

---

## 🎓 高中进阶

### 定义

**细胞全能性**：已经分化的细胞仍然具有发育成完整个体的潜能。原因：每个体细胞都含有该物种发育所需的全部基因（全套遗传信息）。

### 全能性等级

| 等级 | 含义 | 例子 |
|------|------|------|
| 全能性 (totipotency) | 能发育成完整个体 + 胚外组织 | 受精卵、植物分化细胞 |
| 多能性 (pluripotency) | 能分化为三胚层细胞 | 胚胎干细胞、iPSC |
| 多能性（专能） | 能分化为多种细胞 | 造血干细胞 |
| 单能性 (unipotency) | 仅分化为一种细胞 | 表皮干细胞 |

### 表达条件

理论上，分化细胞含全套基因 → 全能性存在。但实际表现需要适宜条件：

**植物：**
- 植物组织培养：脱分化 → 愈伤组织 → 再分化 → 完整植株
- 几乎任何分化的活组织都可培养

**动物：**
- 一般分化细胞难以表现全能性
- 受精卵、早期胚胎细胞具全能性
- 成体细胞核移植（克隆）：将体细胞核注入去核卵 → 发育（如多莉羊）

### 关键证据

| 证据 | 说明 |
|------|------|
| 植物组织培养 | 胡萝卜韧皮部细胞 → 完整植株（Steward 1958） |
| 动物体细胞核移植 | 多莉羊（1996，Wilmut 团队） |
| 诱导多能干细胞 (iPSC) | 山中伸弥团队（2006）将分化细胞诱导回多能态 |

### 全能性的意义

| 应用 | 说明 |
|------|------|
| 植物组培 | 快速繁殖、保持品种、培育新品种、生产次生代谢物 |
| 动物克隆 | 保护濒危物种、农业育种 |
| 干细胞医疗 | 组织修复、器官再生（前景） |
| 基础研究 | 发育生物学、再生医学 |

### 全能性减弱的原因

随分化加深，全能性减弱：
- 表观遗传修饰（DNA 甲基化、组蛋白修饰）
- 染色质构象固定
- 基因表达模式被锁定

### 典型例题

**例**：植物组织培养中，外植体经过哪两阶段最终形成完整植株？
**答**：脱分化（→ 愈伤组织）→ 再分化（→ 根、芽、苗）。

**例**：受精卵的全能性最高、还是成体神经元？
**答**：受精卵；神经元高度分化，全能性极低。

### 易错点

- 全能性 = 能发育成完整个体的**潜能**，不一定立即表现
- 植物分化细胞普遍具全能性；动物分化细胞通常不表现
- 全能性强 ≠ 分裂能力强（高度分化细胞分裂能力弱但仍可能有全能性潜能）
- iPSC 是诱导多能性，不一定是全能性
