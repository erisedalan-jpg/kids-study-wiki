# 细胞凋亡

> **一句话**：由基因决定的细胞主动死亡过程，对生物体正常发育和稳态维持有重要意义。
> **English**: Apoptosis is genetically programmed cell death essential for development and homeostasis.

---

## 🎓 高中进阶

### 定义

**细胞凋亡**：由基因决定的细胞自动结束生命的过程，又称**程序性细胞死亡**。对个体的正常发育、内环境稳态及免疫系统功能至关重要。

### 凋亡 vs 坏死

| 项目 | 凋亡 (apoptosis) | 坏死 (necrosis) |
|------|---------------|-------------|
| 起因 | 基因控制、生理性 | 病理刺激、意外 |
| 主动/被动 | 主动（细胞自己启动） | 被动 |
| 形态变化 | 细胞皱缩、染色质浓缩、形成凋亡小体 | 细胞肿胀破裂 |
| 内容物 | 包裹于膜内（不外泄） | 释放到周围（引发炎症） |
| 周围反应 | 巨噬细胞清除，无炎症 | 炎症反应 |
| 意义 | 生理需要 | 病理损伤 |

### 凋亡过程

1. **接收死亡信号**（外源/内源）
2. **激活 Caspase 蛋白酶级联**
3. **DNA 切断为 180-200 bp 片段**（"梯状电泳"特征）
4. **细胞膜起泡、染色质浓缩**
5. **形成凋亡小体**（apoptotic bodies）
6. **被吞噬细胞清除**

### 在生命中的作用

| 作用 | 例子 |
|------|------|
| 个体发育 | 蝌蚪尾巴消失；胚胎期手指间组织凋亡形成手指 |
| 维持稳态 | 旧细胞凋亡、新细胞生成（红细胞每天 2000 亿凋亡更新） |
| 免疫功能 | 自身反应性 T 细胞凋亡；感染细胞凋亡 |
| 抑癌 | 异常细胞凋亡防止癌变 |
| 神经发育 | 多余神经元凋亡 |

### 调控基因

| 基因 | 作用 |
|------|------|
| Bcl-2 家族 | 抗凋亡或促凋亡（Bax、Bak 等促；Bcl-2、Bcl-xL 抗） |
| Caspase 家族 | 凋亡执行酶 |
| p53 | 检测 DNA 损伤 → 启动凋亡（重要抑癌基因） |
| Fas / FasL | 死亡受体通路 |

### 凋亡与疾病

| 异常 | 疾病 |
|------|------|
| 凋亡过多 | 神经退行性疾病（阿尔茨海默病、帕金森病） |
| 凋亡不足 | 癌症（细胞该死不死） |
| 自身免疫 | 自身反应细胞未凋亡 |

### 典型例题

**例**：胚胎发育过程中蝌蚪尾巴消失属于什么现象？
**答**：细胞凋亡（程序性细胞死亡）。

**例**：凋亡和坏死哪个会引发炎症？
**答**：坏死。凋亡细胞被巨噬细胞"安静"清除。

### 易错点

- 凋亡是**主动**死亡，不是被动死亡
- 凋亡 ≠ 衰老 ≠ 坏死，三者性质不同
- 凋亡细胞数大于新生 → 个体衰老/疾病
- 凋亡是正常生理过程，并不一定有害
