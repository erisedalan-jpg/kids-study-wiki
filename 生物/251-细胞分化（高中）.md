# 细胞分化（高中）

> **一句话**：在个体发育过程中由一个或一种细胞产生形态、结构、功能上稳定差异的细胞类群的过程。
> **English**: Cell differentiation is the process by which a less specialized cell becomes a more specialized cell type with distinct structure and function.

---

## 🎓 高中进阶

### 定义

**细胞分化**：在个体发育过程中，由一个或一种细胞分裂、增殖产生的后代细胞，在形态、结构和生理功能上发生**稳定性差异**的过程。本质是**基因的选择性表达**。

### 特点

| 特点 | 说明 |
|------|------|
| 持久性 | 稳定且终生维持 |
| 不可逆性（一般） | 已分化细胞通常不能再变成其他细胞 |
| 普遍性 | 所有多细胞生物都有 |
| 选择性 | 不同细胞表达不同基因 |
| 遗传物质不变 | DNA 全套相同，但表达不同 |

### 本质

**基因的选择性（差异性）表达**：所有体细胞含相同的 DNA，但不同细胞表达不同的基因 → 合成不同蛋白质 → 形成不同结构与功能。

```
受精卵 (含全套基因)
    ↓ 分化
神经元（表达突触相关基因）
肌肉细胞（表达肌动蛋白基因）
红细胞（表达血红蛋白基因）
...
```

### 与细胞分裂的关系

| 项目 | 细胞分裂 | 细胞分化 |
|------|--------|--------|
| 主要变化 | 数量增加 | 种类增加 |
| 遗传物质 | 复制后均分 | 不变（同一套 DNA） |
| 时间 | 个体发育全程 | 主要在胚胎期 |
| 关系 | 分化是分裂后的"分工" | 在分裂基础上发生 |

### 分化的程度

| 程度 | 例子 | 全能性 |
|------|------|------|
| 未分化 | 受精卵、胚胎干细胞 | 高 |
| 部分分化 | 造血干细胞 | 中（多能性） |
| 终末分化 | 神经元、心肌细胞、红细胞 | 低（无分裂能力） |

### 全能性

- **细胞全能性**：已分化细胞含全套基因，理论上能发育成完整生物体的潜能
- 植物：分化细胞的全能性较高（组织培养可再生植株）
- 动物：通常分化细胞全能性低，只有受精卵和早期胚胎具有

### 常见例子

| 来源 | 分化方向 |
|------|--------|
| 受精卵 | 各种胚胎细胞 |
| 造血干细胞 | 红细胞、各类白细胞、血小板 |
| 神经干细胞 | 神经元、神经胶质细胞 |
| 植物形成层 | 木质部、韧皮部 |

### 典型例题

**例**：神经元和肝细胞的 DNA 是否相同？为什么功能不同？
**答**：DNA 完全相同；功能不同源于基因选择性表达——不同细胞激活不同基因。

**例**：植物组织培养能再生完整植株，说明什么？
**答**：植物分化细胞具有全能性。

### 易错点

- 分化是稳定的、可遗传到子代细胞，但不是基因序列改变
- 分化≠突变；分化只是表达调控，DNA 序列不变
- 一般情况下分化不可逆，但"诱导多能干细胞 (iPSC)"可逆向分化（前沿研究）
- 高度分化细胞（如成熟红细胞、神经元）通常不再分裂
