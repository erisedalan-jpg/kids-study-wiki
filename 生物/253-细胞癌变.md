# 细胞癌变

> **一句话**：正常细胞由于基因突变或表达异常，转变为具有无限增殖能力的癌细胞的过程。
> **English**: Carcinogenesis is the process by which normal cells become cancerous through genetic and epigenetic alterations, gaining unlimited proliferation.

---

## 🎓 高中进阶

### 定义

**细胞癌变**：在致癌因子作用下，正常细胞 DNA 受损或基因表达发生异常，原癌基因被激活、抑癌基因失活，导致细胞失去正常生长调控、出现无限增殖、易转移等特征，最终形成癌细胞的过程。

### 癌细胞的三大特征

| 特征 | 表现 |
|------|------|
| 1. 无限增殖 | 持续分裂，不进入终末分化 |
| 2. 形态结构异常 | 形态不规则、大小不一；染色体异常 |
| 3. 细胞表面变化 | 糖蛋白减少 → 细胞间黏着力下降 → 易**转移**（恶性肿瘤特征） |

### 致癌因子（三大类）

| 类型 | 例子 |
|------|------|
| 物理致癌 | 紫外线、X 射线、γ 射线、放射性物质 |
| 化学致癌 | 苯并芘（吸烟）、亚硝胺、黄曲霉素、石棉 |
| 病毒致癌 | HPV（宫颈癌）、HBV（肝癌）、EBV（鼻咽癌） |

### 关键基因

| 基因类型 | 作用 | 改变后果 |
|--------|------|--------|
| 原癌基因 (proto-oncogene) | 正常时促进细胞分裂 | 突变激活 → 过度增殖（如 RAS、MYC） |
| 抑癌基因 (tumor suppressor) | 正常时抑制分裂、促凋亡 | 突变失活 → 失去刹车（如 p53、RB） |
| DNA 修复基因 | 修复 DNA 损伤 | 突变 → 突变累积加快 |

### 癌变的多步骤性

癌变需多个基因突变累积，通常涉及：
1. 原癌基因激活
2. 抑癌基因失活
3. DNA 修复缺陷
4. 端粒酶激活（突破 Hayflick 极限）
5. 血管生成、免疫逃逸

### 良性 vs 恶性肿瘤

| 项目 | 良性 | 恶性（癌） |
|------|------|--------|
| 生长 | 慢 | 快 |
| 边界 | 清晰、有包膜 | 浸润性、无包膜 |
| 转移 | 不转移 | 易转移 |
| 危害 | 较小（视位置） | 大、可致死 |

### 预防

| 措施 | 内容 |
|------|------|
| 远离致癌物 | 不吸烟、避免过量阳光暴晒、远离辐射 |
| 健康饮食 | 多蔬果、少加工肉、不吃霉变食物 |
| 接种疫苗 | HPV 疫苗（宫颈癌）、HBV 疫苗（肝癌） |
| 适度运动、良好心态 | 提高免疫 |
| 早期筛查 | 体检、特异检测 |

### 治疗方法

- 手术切除
- 化疗（细胞毒药物）
- 放疗
- 靶向治疗（识别特定癌基因产物）
- 免疫治疗（CAR-T、PD-1 抑制剂）

### 典型例题

**例**：原癌基因和抑癌基因有什么不同？癌变中作用？
**答**：原癌基因促进分裂、抑癌基因抑制分裂；癌变需原癌激活+抑癌失活共同作用。

**例**：癌细胞的"恶性"表现在哪？
**答**：无限增殖 + 易转移 + 形态异常。

### 易错点

- 癌变源于基因突变或表达异常，不是单一因素
- 良性肿瘤不算癌（无转移）
- 癌细胞分裂快但**不全能**——失去正常分化能力
- 一次紫外线暴露不会立刻致癌；癌变是长期累积的多步骤过程
