# 细胞衰老

> **一句话**：细胞在生理状态下出现的形态结构和功能进行性退化的过程，是个体衰老的细胞基础。
> **English**: Cell senescence is the progressive structural and functional decline of cells with age, underlying organism aging.

---

## 🎓 高中进阶

### 定义

**细胞衰老**：随着细胞分裂次数的增加或环境因素作用，细胞在形态、结构和生理功能上发生退行性变化的过程。最终细胞失去分裂能力，进入衰老状态。

### 形态结构变化

| 变化 | 表现 |
|------|------|
| 细胞体积 | 缩小 |
| 细胞核 | 体积增大、核膜内陷、染色质收缩 |
| 细胞膜 | 通透性下降、物质运输减弱 |
| 酶活性 | 降低 |
| 色素积累 | 脂褐素等沉积（"老年斑"） |
| 水分 | 减少 |
| 细胞器 | 数目变化、功能下降 |

### 衰老的机制（理论）

| 学说 | 内容 |
|------|------|
| 自由基学说 | 代谢产生自由基损伤 DNA、蛋白质、膜 |
| 端粒学说 | 染色体端粒每分裂缩短一段，达到极限即衰老（**Hayflick 极限**） |
| 基因调控学说 | 衰老是发育程序的延续，由特定基因调控 |
| 线粒体损伤说 | mtDNA 损伤累积导致能量代谢下降 |
| 表观遗传学说 | DNA 甲基化等修饰随年龄改变 |

### 端粒（telomere）

- 染色体末端的重复序列（人 TTAGGG）
- 保护染色体末端不被识别为损伤
- 每次分裂端粒缩短约 50-200 bp
- 缩短至临界值 → 细胞停止分裂、衰老
- **端粒酶**可延长端粒（生殖细胞、干细胞、癌细胞活性高）

### 与个体衰老的关系

- 个体衰老 = 大量细胞衰老的累积结果
- 但个体衰老 ≠ 所有细胞同时衰老
- 不同组织衰老速度不同（皮肤 < 脑神经等）

### 与细胞分裂、分化、凋亡的关系

```
受精卵 → 分裂 → 分化 → 工作 → 衰老 → 凋亡
```

### 抗衰老相关研究

| 方向 | 内容 |
|------|------|
| 抗氧化 | 维 E、维 C 等清除自由基 |
| 端粒研究 | 维持端粒长度、激活端粒酶（须谨慎，关联癌症） |
| 热量限制 | 适度限食可延长某些动物寿命 |
| 干细胞疗法 | 替代衰老细胞 |
| Sirtuin 蛋白 | NAD⁺ 依赖的去乙酰化酶，与寿命相关 |

### 典型例题

**例**：老年人头发变白的细胞学原因？
**答**：毛囊黑色素细胞衰老，酪氨酸酶活性下降，黑色素合成减少。

**例**：为什么生殖细胞和干细胞抗衰老能力强？
**答**：含端粒酶活性，可延长端粒。

### 易错点

- 衰老是正常生理现象，不是疾病（虽与多种疾病相关）
- 衰老细胞 ≠ 死亡细胞，仍可活动但功能下降
- 衰老不可逆（自然过程），但可减缓
- 端粒短不一定立刻死亡，但会加速衰老
