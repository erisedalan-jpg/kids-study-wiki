# 分离定律

> **一句话**：孟德尔第一定律：控制同一性状的等位基因在形成配子时彼此分离，分别进入不同的配子，配子中只含每对等位基因中的一个。
> **English**: Mendel's First Law: alleles of a gene pair separate during gamete formation, so each gamete carries only one allele for each trait.

---

## 🎓 高中进阶

### 定义
**基因的分离定律**（孟德尔第一定律）：在生物体细胞中，控制同一性状的遗传因子（等位基因）成对存在；在形成配子时，成对的遗传因子彼此分离，分别进入不同的配子；受精时，雌雄配子随机结合。

> **适用范围**：一对等位基因，位于同一对同源染色体上，且基因完全显性（或不完全显性时比例有所不同）。

### 原理 / 规律

#### 核心要点

| 要点 | 内容 |
|------|------|
| 遗传因子成对 | 体细胞中同一性状的遗传因子成对（二倍体，AA/Aa/aa） |
| 配子时分离 | 减数第一次分裂后期，同源染色体分离，等位基因随之分离 |
| 配子含一个 | 每个配子只含每对等位基因中的一个（A或a） |
| 随机受精 | 雌雄配子随机结合，后代基因型比例服从概率规律 |

#### Aa自交后代比例

$$Aa \times Aa$$

配子：A（1/2）, a（1/2）

| | A（1/2） | a（1/2） |
|--|---------|---------|
| **A（1/2）** | AA（1/4） | Aa（1/4） |
| **a（1/2）** | Aa（1/4） | aa（1/4） |

- **基因型比**：AA : Aa : aa = 1 : 2 : 1
- **表现型比**（完全显性）：显性 : 隐性 = 3 : 1

#### 常见交配类型汇总

| 亲本组合 | 后代基因型 | 后代表现型 |
|----------|-----------|-----------|
| AA × AA | 全AA | 全显性 |
| AA × Aa | 1/2 AA, 1/2 Aa | 全显性 |
| AA × aa | 全Aa | 全显性 |
| Aa × Aa | 1/4 AA, 1/2 Aa, 1/4 aa | 3显:1隐 |
| Aa × aa | 1/2 Aa, 1/2 aa | 1显:1隐 |
| aa × aa | 全aa | 全隐性 |

#### 细胞学基础
分离定律的实质是**减数第一次分裂后期同源染色体的分离**，导致等位基因分别进入不同子细胞，最终进入不同配子。

### 典型例题

**例题**：某植物花色白色（aa）与红色（AA）杂交，F₁红色自交，求F₂中白花植株的比例，以及F₂红花植株中纯合红花占多少？

**解题**：
- F₁基因型：Aa
- F₁自交：Aa × Aa → 1/4 AA : 2/4 Aa : 1/4 aa
- F₂白花（aa）= **1/4**
- F₂红花（AA + Aa）= 3/4，其中纯合红花（AA）= 1/4
- 纯合红花占红花的比例 = (1/4)/(3/4) = **1/3**

### 易错点
1. **分离定律发生在减数分裂而非有丝分裂**：有丝分裂后期着丝点分裂，同一等位基因复制后分开，并不发生分离（两个子细胞基因型相同）。
2. **完全显性vs.不完全显性**：金鱼草红花×白花→F₁粉花，F₂红:粉:白=1:2:1（表现型与基因型比相同），分离定律依然成立，但显隐关系不同。
3. **比例是概率，样本小时偏差大**：后代数量少时，实际比例可能偏离3:1。
4. **"等位基因"定义**：位于同源染色体相同位置，控制同一性状，可以相同（AA）也可以不同（Aa）。
