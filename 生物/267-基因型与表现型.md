# 基因型与表现型

> **一句话**：基因型是生物个体的遗传组成（基因的组合），表现型是基因型在一定环境下表现出的性状特征；表现型 = 基因型 + 环境。
> **English**: Genotype is the genetic makeup of an organism; phenotype is the observable trait resulting from genotype and environment.

---

## 🎓 高中进阶

### 定义

| 概念 | 定义 |
|------|------|
| **基因型** (genotype) | 与表现型有关的基因组成，如 AA、Aa、aa |
| **表现型** (phenotype) | 生物个体表现出来的性状，如高茎、矮茎 |
| **相互关系** | 表现型 = 基因型 + 环境条件的影响 |

### 原理 / 规律

#### 基因型与表现型的关系

```
基因型 ──→ (表达) ──→ 表现型
   ↑                      ↑
遗传因素              环境影响
```

**规律一**：基因型相同，表现型不一定相同（环境影响可改变表现型）。
- 例：同一基因型的水毛茛，水中叶片细裂，水面上叶片宽扁（环境不同）。

**规律二**：表现型相同，基因型不一定相同。
- 例：高茎豌豆可以是 AA（纯合）或 Aa（杂合），均表现高茎。

**规律三**：基因型是遗传的基础，亲代的基因型决定后代可能出现的基因型范围。

#### 一对等位基因的基因型类型

| 基因型 | 类型 | 表现型（完全显性） |
|--------|------|------------------|
| AA | 显性纯合子 | 显性性状 |
| Aa | 杂合子 | 显性性状 |
| aa | 隐性纯合子 | 隐性性状 |

> **注意**：显性纯合子（AA）与杂合子（Aa）表现型相同，须通过测交或分子检测区分。

#### 环境对表现型的影响

| 示例 | 说明 |
|------|------|
| 喜马拉雅兔毛色 | 相同基因型，低温处毛色深，高温处毛色浅 |
| 植物叶形 | 水毛茛水中叶细裂、水面叶宽扁 |
| 人类肤色 | 晒太阳后肤色加深（基因型未变，表现型变化） |

#### 如何判断基因型
- **方法1**：测交（与隐性纯合子交配），后代出现隐性则亲本为Aa；全显则亲本可能为AA（需多次测交确认）。
- **方法2**：自交，后代出现性状分离则亲本为Aa；不分离则可能为AA（需结合上下代分析）。
- **方法3**：分子生物学检测（PCR + 测序等，现代育种中常用）。

### 典型例题

**例题**：已知豌豆高茎（T）对矮茎（t）为显性。现有一株高茎豌豆，如何通过实验判断其基因型是TT还是Tt？

**方案一：自交法**
让该高茎植株连续自交2—3代，若后代出现矮茎植株，则其基因型为Tt；若后代全部为高茎，则可能为TT（但不能完全排除，需足够大后代数量）。

**方案二：测交法**
让该高茎植株与矮茎植株（tt）杂交：
- 后代全部高茎 → 亲本基因型为TT（TT × tt → 全Tt）
- 后代高茎:矮茎 = 1:1 → 亲本基因型为Tt（Tt × tt → 1Tt:1tt）

**结论**：测交法更快速、结果更明确。

### 易错点
1. **同一表现型≠同一基因型**：高茎可以是TT或Tt，不可混淆。
2. **环境改变表现型，基因型不变**：晒黑皮肤是表现型变化，遗传信息（DNA）未改变，不能遗传给后代。
3. **基因型只写与性状有关的基因**：分析一对性状时只写该对等位基因（如Aa），不必写全基因组。

### 例句
- *Phenotype is determined by both genotype and environmental factors.*（表现型由基因型和环境因素共同决定。）
- *Two individuals with the same phenotype may have different genotypes.*（表现型相同的两个个体，基因型不一定相同。）
