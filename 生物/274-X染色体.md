# X染色体

> **一句话**：人类性染色体之一，雌性含两条（XX），雄性含一条（XY），携带大量与性别无关的基因及若干伴性遗传基因。
> **English**: One of the two sex chromosomes; females carry two copies (XX) and males carry one (XY), bearing many genes unrelated to sex determination plus several sex-linked genes.

---

## 🎓 高中进阶

### 定义
**X 染色体**（X chromosome）：人类 23 对染色体中的一对性染色体之一，形态较大，基因数目约 800–900 个（其中许多与性别决定无关）。

- 女性核型：44 条常染色体 + XX（即 46, XX）
- 男性核型：44 条常染色体 + XY（即 46, XY）

### 原理 / 规律

#### X 染色体的结构特点

| 区域 | 说明 |
|------|------|
| X 特异区段 | 只存在于 X，不与 Y 同源；X 连锁遗传基因位于此段 |
| X-Y 同源区段 | X 与 Y 均有，该段基因遗传与常染色体类似 |

#### X 染色体传递规律

| 传递路径 | 说明 |
|----------|------|
| 父→女 | 父亲唯一的 X 必定传给所有女儿 |
| 母→儿子或女儿 | 母亲两条 X 随机各传一条，儿子得到 Y 来自父亲 |

#### Lyon 假说（X 染色质失活）
女性体细胞中两条 X 染色体之一在胚胎发育早期随机失活，浓缩为**巴氏小体**（Barr body），保证雌雄基因剂量平衡。

$$\text{巴氏小体数} = X\text{染色体数} - 1$$

正常女性（46, XX）→ 1 个巴氏小体；正常男性（46, XY）→ 0 个巴氏小体。

### 典型例题

**例题**：某女性有 2 个巴氏小体，推测其核型？

**解题**：巴氏小体数 = X 数 - 1 → X 数 = 3，核型为 **47, XXX**（超 X 综合征）。

### 易错点
1. **X 染色体不只携带"性状"基因**：X 上的大量基因（如凝血因子基因、色觉基因）与性别本身无关，只是遗传时伴随性染色体。
2. **X 连锁≠Y 连锁**：X 连锁遗传男女均可患病，Y 连锁只有男性患病。
3. **Lyon 假说是随机且永久的**：同一女性个体不同细胞中失活的 X 不同，但一旦确定就不再改变。
