# 受精作用

> **一句话**：受精作用是雌雄配子（精子与卵细胞）融合形成受精卵的过程，使染色体数目恢复为体细胞水平（2n），是有性生殖中遗传多样性产生的重要环节。
> **English**: Fertilization is the fusion of male and female gametes to form a zygote, restoring the diploid chromosome number and contributing to genetic diversity.

---

## 🎓 高中进阶

### 定义
**受精作用**（fertilization）：雄性配子（精子）与雌性配子（卵细胞）相互融合，形成受精卵（zygote）的过程。受精卵含有来自双亲各一半的遗传物质，染色体数目恢复为2n。

### 原理 / 规律

#### 减数分裂与受精作用的关系

```
亲代(2n) → [减数分裂] → 配子(n) → [受精作用] → 受精卵(2n) → [有丝分裂] → 后代(2n)
```

减数分裂使染色体数减半，受精作用使染色体数恢复，两者共同维持物种世代间染色体数目的恒定。

#### 受精卵遗传物质来源

| 来源 | 比例 |
|------|------|
| 核DNA | 父方：母方 = 1:1（各50%） |
| 细胞质DNA（线粒体/叶绿体） | 几乎全来自母方（卵细胞细胞质多；精子细胞质极少，受精后多退化） |

> **母系遗传**：线粒体DNA（mtDNA）通过母方卵细胞传递，为母系遗传。人类mtDNA分析可追溯母系祖先。

#### 遗传多样性的来源（减数分裂+受精）

| 来源 | 机制 |
|------|------|
| 非同源染色体自由组合 | 减Ⅰ后期，产生2ⁿ种不同配子（n为同源染色体对数） |
| 同源染色体交叉互换 | 减Ⅰ前期，增加配子基因组成多样性 |
| 雌雄配子随机结合 | 受精时雌雄配子的随机配对，后代组合数极大 |

对于人类（2n=46，n=23），仅自由组合就可产生 $2^{23} = 8,388,608$ 种配子，受精后后代基因型多样性极高。

#### 受精过程（哺乳动物）

1. **顶体反应**：精子顶端顶体释放水解酶，溶解卵细胞周围透明带。
2. **精卵膜融合**：精子头部与卵细胞膜融合，核物质进入。
3. **皮层反应**：防止多精受精（透明带反应使只有一个精子完成受精）。
4. **受精卵形成**：雌雄原核融合，完成受精，细胞开始有丝分裂发育。

#### 受精作用的意义

1. **维持染色体数目恒定**：保证物种遗传稳定性。
2. **增加遗传多样性**：双亲遗传物质重新组合，后代多样性高于无性生殖。
3. **推动生物进化**：遗传多样性为自然选择提供更多原材料。

### 典型例题

**例题**：人的体细胞中有46条染色体，精子和卵细胞分别含多少条染色体？受精卵呢？若只考虑常染色体自由组合，人类能产生多少种不同遗传组成的配子？

**解题**：
- 精子/卵细胞：n = 46/2 = **23条**
- 受精卵：2n = **46条**
- 人类常染色体22对，加性染色体1对，共23对同源染色体：
  $$2^{23} = 8,388,608 \approx 8.4 \times 10^6 \text{ 种配子}$$
- 受精后后代基因型种类：$(2^{23})^2 = 2^{46} \approx 7 \times 10^{13}$ 种

**答**：精子/卵细胞各23条；受精卵46条；理论上可产生约840万种不同配子。

### 易错点
1. **细胞质DNA不是父母各半**：核DNA各占50%，但细胞质DNA（线粒体等）主要来自母方。
2. **受精卵的第一次分裂是有丝分裂**：受精卵分裂为有丝分裂，产生的子细胞基因型相同（2n）。
3. **多精受精通常致死**：自然状态下皮层反应阻止多精受精；多精受精导致三倍体受精卵，通常不能正常发育。
