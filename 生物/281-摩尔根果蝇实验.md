# 摩尔根果蝇实验

> **一句话**：美国生物学家摩尔根以果蝇为材料，通过白眼基因与X染色体的连锁遗传实验，用实验证明了基因在染色体上，奠定了基因的染色体定位基础。
> **English**: Morgan used Drosophila genetics to experimentally prove that genes are located on chromosomes, discovering sex-linkage through the white-eye mutation.

---

## 🎓 高中进阶

### 定义
托马斯·亨特·摩尔根（Thomas Hunt Morgan，1866—1945）以果蝇（*Drosophila melanogaster*）为实验材料，研究眼色遗传，发现白眼性状遗传与性别相关（伴性遗传），从而证明白眼基因位于X染色体上，**实验证明了基因在染色体上**。摩尔根因此获得1933年诺贝尔生理学或医学奖。

### 原理 / 规律

#### 果蝇作为模式生物的优点

| 优点 | 说明 |
|------|------|
| 繁殖快 | 约12天一个世代，一年可培育多代 |
| 后代多 | 每次产卵数百个，样本量大 |
| 易饲养 | 饲养成本低，操作简便 |
| 染色体少 | 2n=8，只有4对染色体，遗传分析简单 |
| 雌雄易区分 | 雌雄外形差异明显 |
| 突变体多 | 容易发现各种突变，便于遗传分析 |

#### 摩尔根实验过程

**第一步：发现突变体**
- 正常果蝇眼色为红色（野生型）。
- 摩尔根在培养瓶中发现一只白眼雄果蝇（突变体）。

**第二步：正交实验**
```
P:   红眼♀(X^W X^w 或 X^W X^W)  ×  白眼♂(X^w Y)
                   ↓
F₁: 红眼♀ : 红眼♂ = 1:1（全部红眼）
                   ↓ F₁互交
F₂: 红眼♀ : 红眼♂ : 白眼♂ = 2:1:1
    （白眼只出现在雄性中）
```

**第三步：反交实验（关键证据）**
```
P:   白眼♀(X^w X^w) × 红眼♂(X^W Y)
                ↓
F₁: 红眼♀(X^W X^w) : 白眼♂(X^w Y) = 1:1
    （交叉遗传：母白眼→儿子白眼，父红眼→女儿红眼）
```

正反交结果不同，且与性别相关，说明眼色基因与决定性别的染色体（X）连锁。

#### 结论
- 白眼基因（w）位于X染色体上（伴X遗传）。
- 由此证明**基因在染色体上**，且一条染色体上可携带多个基因。
- 同一染色体上的基因连锁遗传（不完全符合自由组合定律）。

#### 摩尔根与萨顿方法比较

| | 萨顿 | 摩尔根 |
|-|------|-------|
| 方法 | 类比推理（观察染色体与遗传因子的平行性） | 假说—演绎法（实验验证） |
| 证据 | 类比，间接 | 实验数据，直接 |
| 结论强度 | 假说 | 证明 |

### 典型例题

**例题**：摩尔根将白眼果蝇（♂，X^w Y）与红眼果蝇（♀，X^W X^W）杂交，F₁红眼♀与白眼♂测交（即F₁♀ × 白眼♂），后代眼色及比例如何？

**解题**：
- F₁红眼♀ = X^W X^w（来自P的正交）
- 白眼♂ = X^w Y

$$X^W X^w \times X^w Y$$

配子：X^W, X^w（♀）；X^w, Y（♂）

后代：
- X^W X^w：红眼♀ = 1/4
- X^w X^w：白眼♀ = 1/4
- X^W Y：红眼♂ = 1/4
- X^w Y：白眼♂ = 1/4

**答**：红眼♀:白眼♀:红眼♂:白眼♂ = 1:1:1:1

### 易错点
1. **摩尔根是实验证明，萨顿是假说推理**：两者方法不同，结论强度不同。
2. **白眼基因在X染色体上，Y染色体上无对应等位基因**：雄性只需一个X^w 即表现白眼（半合子），雌性需要X^w X^w 才表现白眼。
3. **摩尔根的实验意义**：不仅证明基因在染色体上，还开创了基因连锁与互换的研究。
