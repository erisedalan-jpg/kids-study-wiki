# DNA分子结构

> **一句话**：DNA 是由两条反向平行的脱氧核苷酸链通过碱基互补配对（A-T，G-C）形成的规则双螺旋结构，磷酸-脱氧核糖骨架在外，碱基对在内。
> **English**: DNA is a regular double helix formed by two antiparallel deoxyribonucleotide strands held together by complementary base pairing (A-T, G-C), with the sugar-phosphate backbone on the outside and base pairs inside.

---

## 🎓 高中进阶

### 定义
**DNA 分子结构**：脱氧核糖核酸（deoxyribonucleic acid）由沃森和克里克（Watson & Crick，1953）提出双螺旋模型，核心要素如下：

| 层次 | 组成 |
|------|------|
| 基本单位 | 脱氧核苷酸（磷酸 + 脱氧核糖 + 含氮碱基） |
| 四种碱基 | 腺嘌呤（A）、鸟嘌呤（G）、胞嘧啶（C）、胸腺嘧啶（T） |
| 链的结合方式 | 碱基互补配对（A=T，G≡C） |
| 空间构型 | 右手双螺旋，螺距 3.4 nm，每圈 10 个碱基对 |

### 原理 / 规律

#### 脱氧核苷酸的组成

```
磷酸（H₃PO₄）
    |
脱氧核糖（五碳糖）
    |
含氮碱基（A / T / G / C 之一）
```

四种脱氧核苷酸：腺嘌呤脱氧核苷酸、鸟嘌呤脱氧核苷酸、胞嘧啶脱氧核苷酸、胸腺嘧啶脱氧核苷酸

#### 双链结构特征

| 特征 | 说明 |
|------|------|
| 反向平行 | 一链 5'→3'，另一链 3'→5'，两链方向相反 |
| 碱基互补配对 | A 与 T 配对（2 个氢键），G 与 C 配对（3 个氢键） |
| 磷酸-脱氧核糖骨架 | 在双螺旋外侧，亲水 |
| 碱基对 | 在双螺旋内侧，疏水，维持结构稳定 |

#### 重要计算关系（查哥夫定律）

在双链 DNA 中：
$$A = T, \quad G = C$$
$$A + G = T + C = 50\%$$
$$\frac{A+G}{T+C} = 1 \text{（嘌呤总量 = 嘧啶总量）}$$

若已知一条链中 A₁、T₁、G₁、C₁，则整条双链：
$$A_{双链} = A_1 + T_1, \quad G_{双链} = G_1 + C_1$$

#### DNA 的多样性与特异性

- **多样性**：碱基排列顺序（序列）千变万化（4^n 种可能），是生物多样性的分子基础
- **特异性**：特定 DNA 分子碱基序列固定，决定特定性状

### 典型例题

**例题**：某 DNA 分子含 1000 个碱基对，其中 G+C 占 40%。求：①A 的数目？②A 与 T 之间的氢键数？③G 与 C 之间的氢键数？

**解题**：
- 总碱基 = 1000×2 = 2000 个
- G+C = 2000×40% = 800，G=C=400
- A+T = 2000-800 = 1200，A=T=600
- ① A = **600 个**
- ② A-T 氢键 = 600×2 = **1200 个**（每对 A-T 有 2 个氢键）
- ③ G-C 氢键 = 400×3 = **1200 个**（每对 G-C 有 3 个氢键）

### 易错点
1. **DNA 不含 U（尿嘧啶），RNA 不含 T（胸腺嘧啶）**：混淆碱基类型是常见错误。
2. **单链 DNA（如某些病毒）不适用查哥夫定律**：A≠T，G≠C，A+G≠50%。
3. **脱氧核苷酸不等于脱氧核苷**：脱氧核苷酸含磷酸；脱氧核苷不含磷酸。
