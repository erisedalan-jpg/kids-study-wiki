# 双螺旋结构

> **一句话**：沃森和克里克（1953）提出的 DNA 空间构型——两条反向平行的脱氧核苷酸链围绕同一轴右手螺旋盘绕，碱基对在内，磷酸-脱氧核糖骨架在外。
> **English**: The spatial model proposed by Watson and Crick (1953) in which two antiparallel deoxyribonucleotide strands coil around a common axis in a right-handed helix, with base pairs inside and the sugar-phosphate backbone outside.

---

## 🎓 高中进阶

### 定义
**DNA 双螺旋结构**（double helix）：1953 年，沃森（J. D. Watson）和克里克（F. H. C. Crick）根据富兰克林（R. Franklin）的 X 射线衍射数据，提出 DNA 分子的双螺旋模型，发表于《自然》杂志，获 1962 年诺贝尔生理学或医学奖。

### 原理 / 规律

#### 双螺旋的关键参数

| 参数 | 数值 |
|------|------|
| 螺旋方向 | 右手螺旋 |
| 螺距（每圈高度） | 3.4 nm |
| 每圈碱基对数 | 10 对 |
| 相邻碱基对间距 | 0.34 nm |
| 双螺旋直径 | 约 2 nm |

#### 结构层次

```
磷酸-脱氧核糖骨架（外侧）
        ↕ （氢键）
    碱基对（内侧）
        A = T（2 个氢键）
        G ≡ C（3 个氢键）
磷酸-脱氧核糖骨架（外侧）
```

#### 维持双螺旋稳定的化学力

| 作用力 | 说明 |
|--------|------|
| 氢键（链间） | A-T 之间 2 个氢键，G-C 之间 3 个氢键，保持两链结合 |
| 碱基堆积力（链内） | 相邻碱基平面之间的范德华力和疏水相互作用，稳定螺旋 |

→ G+C 含量越高，氢键越多（G-C 有 3 个），DNA 热稳定性越高（解链温度 Tm 越高）

#### 反向平行的含义

两条链的磷酸二酯键走向相反：
- 链 I：5'→3' 方向（由 5' 端磷酸基向 3' 端羟基延伸）
- 链 II：3'→5' 方向（与链 I 平行但方向相反）

这确保两链碱基能够正向配对，形成稳定的碱基对。

### 典型例题

**例题**：G+C 含量为 60% 的双链 DNA 与 G+C 含量为 40% 的双链 DNA 相比，前者热稳定性更高，原因？

**解题**：G-C 之间有 3 个氢键，A-T 之间有 2 个氢键。G+C 含量越高，每个碱基对平均氢键数越多，解开双链需要更多能量，因此 **Tm（解链温度）更高，热稳定性更强**。

### 易错点
1. **沃森-克里克提出的是模型，不是直接拍照发现的**：实验数据来自富兰克林的 X 射线衍射图（B 型 DNA 照片），模型由沃森和克里克推断提出。
2. **"反向平行"不是"颠倒"**：两条链都存在，方向相反，不是将同一条链翻转。
3. **DNA 不都是 B 型（标准型）**：在特殊条件下可形成 A 型、Z 型等构型，但高中阶段以 B 型为标准。
