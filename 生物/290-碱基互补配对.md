# 碱基互补配对

> **一句话**：DNA 双链间（及 DNA-RNA、RNA 内部）遵循严格的配对规则：A 与 T（或 U）配对，G 与 C 配对，这是 DNA 复制、转录和翻译精确性的分子基础。
> **English**: The strict pairing rule between DNA strands (and DNA-RNA or RNA): A pairs with T (or U), G pairs with C — the molecular basis for the accuracy of replication, transcription, and translation.

---

## 🎓 高中进阶

### 定义
**碱基互补配对原则**（base-pairing rule）：在核酸分子中，特定碱基只能与特定碱基通过氢键结合：

| 场景 | 配对关系 |
|------|---------|
| DNA 双链 | A = T（2 氢键），G ≡ C（3 氢键） |
| DNA→RNA（转录） | A→U，T→A，G→C，C→G |
| mRNA→tRNA（翻译，密码子-反密码子） | A-U，U-A，G-C，C-G |
| RNA 分子内部折叠（tRNA 等） | A-U，G-C |

### 原理 / 规律

#### 化学结构基础

- **嘌呤**（双环）：腺嘌呤（A）、鸟嘌呤（G）
- **嘧啶**（单环）：胸腺嘧啶（T）、胞嘧啶（C）、尿嘧啶（U）

配对规则：**嘌呤-嘧啶配对**（大-小互补），保证双螺旋直径恒定（约 2 nm）：
- A（嘌呤）+ T（嘧啶）= 2 个氢键
- G（嘌呤）+ C（嘧啶）= 3 个氢键

#### 碱基互补配对的应用

**1. 已知一条链推算互补链**

设 DNA 一条链碱基组成为 A₁、T₁、G₁、C₁，则互补链为 T₁、A₁、C₁、G₁

**2. 已知 DNA 推算转录产生的 mRNA**

模板链（3'→5'）：A → mRNA 上 U；T → mRNA 上 A；G → mRNA 上 C；C → mRNA 上 G

**3. 已知 mRNA 密码子推算 tRNA 反密码子**

密码子 5'-AUG-3' → 反密码子 3'-UAC-5'（即反密码子序列为 CAU 读 5'→3'）

#### 关键计算

在双链 DNA 中，设一条链 A₁=20，T₁=30，G₁=25，C₁=25（共 100 个碱基）：
- 互补链：T₂=20，A₂=30，C₂=25，G₂=25
- 双链总 A = A₁+A₂ = 20+30 = 50，双链 A=T=50
- 双链总 G = G₁+G₂ = 25+25 = 50，双链 G=C=50
- 氢键总数 = 50×2 + 50×3 = **250 个**

### 典型例题

**例题**：某 mRNA 的碱基比例 A:U:G:C = 1:2:3:4，该 mRNA 转录时的模板链对应碱基比例？

**解题**：转录时 mRNA 碱基来自模板链互补：
- mRNA 的 A 来自模板链 T → 模板链 T 比例与 mRNA A 相同 = 1
- mRNA 的 U 来自模板链 A → 模板链 A = 2
- mRNA 的 G 来自模板链 C → 模板链 C = 3
- mRNA 的 C 来自模板链 G → 模板链 G = 4

**模板链 A:T:G:C = 2:1:4:3**

### 易错点
1. **DNA 中无 U，RNA 中无 T**：DNA 含 T，RNA 含 U，互换是典型错误。
2. **转录的模板是 DNA 的一条链（模板链），不是双链同时转录**：非模板链（编码链）序列与 mRNA 相同（T 换成 U）。
3. **互补不等于相同**：互补链碱基与原链不同（A↔T，G↔C），但长度相等。
