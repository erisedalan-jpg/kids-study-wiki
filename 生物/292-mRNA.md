# mRNA

> **一句话**：信使 RNA（messenger RNA）——由 DNA 转录合成，携带遗传信息（密码子序列），作为翻译的直接模板，指导核糖体合成蛋白质。
> **English**: Messenger RNA — transcribed from DNA, carrying genetic information as codons, and serving as the direct template for ribosomal protein synthesis.

---

## 🎓 高中进阶

### 定义
**mRNA**（信使 RNA，messenger RNA）：三类功能性 RNA 之一，由 DNA 模板链转录而来，以**密码子**（三联体碱基序列）形式编码氨基酸序列，是翻译过程的直接模板。

### 原理 / 规律

#### mRNA 的结构特点

| 特征 | 说明 |
|------|------|
| 单链 | mRNA 为单链，不形成稳定双链（但可局部折叠） |
| 碱基组成 | A、U、G、C（含 U，不含 T） |
| 方向 | 5'→3' 方向（翻译从 5' 端 AUG 起始） |
| 起始密码子 | AUG（编码甲硫氨酸，Met）—— 翻译起始信号 |
| 终止密码子 | UAA、UAG、UGA —— 不编码氨基酸，终止翻译 |

#### mRNA 与 DNA 的对应关系

$$\text{DNA 模板链}(3' \to 5') \xrightarrow{\text{转录}} \text{mRNA}(5' \to 3')$$

碱基对应：模板链 A→mRNA U；T→A；G→C；C→G

**编码链**（非模板链）与 mRNA 碱基序列相同（T→U），可由编码链直接"转换"为 mRNA 序列。

#### mRNA 的合成加工（真核细胞）

```
DNA 转录 → 前体 mRNA（pre-mRNA，含内含子）
    ↓ 剪接（splicing）
成熟 mRNA（去除内含子，保留外显子）
    ↓ 加工（加 5'-帽、3'-poly A 尾）
出核 → 核糖体翻译
```

#### mRNA 与蛋白质的关系

$$\text{mRNA 碱基数} = \text{密码子数} \times 3 = \text{氨基酸数} \times 3$$

（不含终止密码子编码的 3 个碱基时，氨基酸数 = 总密码子数 - 1）

**氨基酸数量 < mRNA 碱基数/3**（因为有起始密码子 AUG 和终止密码子不编码氨基酸，高中一般简化为：氨基酸数 ≈ mRNA 长度/3）

### 典型例题

**例题**：某 mRNA 含 300 个碱基，编码区包含 1 个终止密码子，则该 mRNA 最多编码多少个氨基酸？翻译产生的多肽含多少个肽键？

**解题**：
- 密码子总数 = 300 / 3 = 100 个（其中 1 个终止密码子）
- 编码氨基酸的密码子 = 100 - 1 = 99 个
- **氨基酸 = 99 个**
- 肽键数 = 氨基酸数 - 1 = **98 个**

### 易错点
1. **mRNA 上不含 T**：只含 U，T 是 DNA 特有。
2. **终止密码子不编码氨基酸**：UAA/UAG/UGA 是翻译终止信号，无对应 tRNA。
3. **一个 mRNA 可同时被多个核糖体翻译（多聚核糖体）**，提高翻译效率。
4. **mRNA 的稳定性决定蛋白质产量**：真核细胞 mRNA 有 5'-帽和 poly A 尾保护，延长半衰期。
