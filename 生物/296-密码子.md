# 密码子

> **一句话**：mRNA 上每三个相邻碱基组成一个密码子，编码一种氨基酸（或作为终止信号）；遗传密码共 64 种，其中 61 种编码 20 种氨基酸，3 种为终止密码子。
> **English**: A codon is a triplet of adjacent bases on mRNA that specifies one amino acid (or a stop signal); the genetic code has 64 codons — 61 encode 20 amino acids and 3 are stop codons.

---

## 🎓 高中进阶

### 定义
**密码子**（codon）：mRNA 上相邻的三个碱基组成一个三联体，是翻译时氨基酸的基本编码单位。密码子由翻译机器（tRNA 反密码子）识别并转换为相应氨基酸。

$$\text{遗传密码} = 4^3 = 64 \text{ 种密码子}$$
$$61 \text{ 种编码氨基酸} + 3 \text{ 种终止密码子（UAA, UAG, UGA）}$$

### 原理 / 规律

#### 遗传密码的四大特性

| 特性 | 说明 |
|------|------|
| **通用性** | 从细菌到人类，密码子含义基本相同（极少数生物/细胞器有微小差异） |
| **简并性（冗余性）** | 一种氨基酸可由多个密码子编码（如 Leu 有 6 个密码子），减少突变危害 |
| **特异性** | 一个密码子只能编码一种氨基酸（或终止信号），无歧义 |
| **无标点性** | 密码子连续读取，无间隔符号；移码突变会打乱整个读框 |

#### 关键密码子

| 密码子 | 编码 |
|--------|------|
| AUG | 甲硫氨酸（Met）—— 同时也是起始密码子 |
| UAA | 终止（终止密码子，不编码氨基酸） |
| UAG | 终止 |
| UGA | 终止 |

#### 简并性举例（以亮氨酸 Leu 为例）

| 密码子 | 氨基酸 |
|--------|--------|
| UUA | Leu |
| UUG | Leu |
| CUU | Leu |
| CUC | Leu |
| CUA | Leu |
| CUG | Leu |

→ 6 个密码子均编码亮氨酸（Leu）——简并性最强的氨基酸之一

#### 密码子的通用性意义

通用性是生命共同祖先（LUCA）存在的分子证据，也是基因工程的基础（人源基因可在细菌中表达）。

### 典型例题

**例题**：某 DNA 片段的编码链为 5'-ATGCCGTAA-3'，翻译产生的多肽含几个氨基酸？

**解题**：
- 编码链 → mRNA：5'-AUGCCGUAA-3'
- 密码子：AUG（Met），CCG（Pro），UAA（终止）
- 编码氨基酸的密码子 = 2 个（AUG、CCG）
- **多肽含 2 个氨基酸**（二肽，含 1 个肽键）

### 易错点
1. **密码子在 mRNA 上，不在 DNA 上**：DNA 对应的三联体叫"三联体密码"，密码子特指 mRNA 上的三联体。
2. **终止密码子无对应 tRNA**，不编码任何氨基酸，由释放因子识别。
3. **一种氨基酸可对应多个密码子（简并性），但一个密码子只对应一种氨基酸（特异性）**：方向不可混淆。
4. **密码子通用性有极少数例外**：线粒体等细胞器密码子与标准遗传密码略有差异。
