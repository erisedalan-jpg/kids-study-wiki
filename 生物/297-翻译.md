# 翻译

> **一句话**：以 mRNA 为模板，在核糖体上，tRNA 携带氨基酸按密码子顺序逐个加入，通过肽键连接，合成具有特定氨基酸序列的多肽链（蛋白质）的过程。
> **English**: The process in which mRNA serves as a template and tRNAs deliver amino acids to the ribosome in codon order, forming peptide bonds to produce a polypeptide chain.

---

## 🎓 高中进阶

### 定义
**翻译**（translation）：基因表达的第二步，在核糖体上以 mRNA 为模板，tRNA 识别密码子并转运氨基酸，氨基酸之间形成肽键，合成多肽链，最终折叠加工为功能蛋白质。

### 原理 / 规律

#### 翻译的场所与条件

| 条件 | 说明 |
|------|------|
| 场所 | 核糖体（细胞质基质、内质网膜上） |
| 模板 | mRNA（5'→3' 方向依次读取密码子） |
| 原料 | 20 种氨基酸 |
| 工具 | tRNA（携带氨基酸，识别密码子） |
| 酶 | 氨酰-tRNA 合成酶（连接氨基酸与 tRNA）；核糖体大亚基 rRNA（催化肽键形成） |
| 能量 | ATP（氨基酸活化）、GTP（tRNA 进位） |

#### 翻译的过程

**起始**：小亚基识别 mRNA 的 5' 端并滑动至起始密码子（AUG），起始 tRNA（携带 Met）进入 P 位点，大亚基结合，形成起始复合物。

**延伸**：
```
A 位点（进位）：氨酰-tRNA 与 mRNA 密码子互补配对
    ↓ 肽酰转移（大亚基 rRNA 催化）
肽键形成，多肽链从 P 位点 tRNA 转移到 A 位点氨基酸
    ↓ 移位
核糖体沿 mRNA 5'→3' 移动一个密码子
P 位点 tRNA（现为空载）转移到 E 位点并离开
```

**终止**：核糖体到达终止密码子（UAA/UAG/UGA），无对应 tRNA，释放因子进入，水解肽链，核糖体解体，多肽链释放。

#### 重要计算关系

$$\text{氨基酸数} = \frac{\text{mRNA 碱基数}}{3} - 1 \text{（减去 1 个终止密码子；高中常简化为÷3）}$$

$$\text{肽键数} = \text{氨基酸数} - \text{肽链条数}$$

$$\text{脱水缩合失去水分子数} = \text{肽键数}$$

**多聚核糖体**（polysome）：一条 mRNA 同时被多个核糖体翻译，大幅提高翻译效率（氨基酸数×核糖体数的蛋白质可同时合成）。

#### 翻译与转录的对比

| | 转录 | 翻译 |
|-|------|------|
| 场所 | 细胞核（真核） | 核糖体 |
| 模板 | DNA 一条链 | mRNA |
| 原料 | 核糖核苷酸 | 氨基酸 |
| 产物 | mRNA | 多肽链 |
| 碱基互补 | A-U, T-A, G-C, C-G | 密码子-反密码子（A-U，G-C） |

### 典型例题

**例题**：一条 mRNA 有 900 个碱基，同时被 5 个核糖体翻译，每个核糖体最终合成一条多肽链，每条多肽链含多少个氨基酸？5 个核糖体共合成多少个肽键？

**解题**：
- mRNA 碱基 900 个 → 密码子 = 900/3 = 300 个（含 1 个终止密码子）
- 编码氨基酸的密码子 = 300 - 1 = 299 个
- 每条多肽链氨基酸数 = **299 个**
- 每条多肽链肽键数 = 299 - 1 = 298 个
- 5 条多肽链总肽键 = 298 × 5 = **1490 个**

### 易错点
1. **翻译从 AUG（起始密码子）开始，到终止密码子停止**；AUG 编码 Met，但成熟蛋白质 N 端的 Met 常被切除。
2. **终止密码子无对应 tRNA**（由释放因子识别），不是"tRNA 读不了就停"。
3. **肽键数 = 氨基酸数 - 肽链数**（一条肽链时 = 氨基酸数 - 1；多条肽链时各自减 1）。
4. **翻译发生在细胞质（核糖体），不在细胞核**；转录在细胞核，翻译在细胞质。
