# 转录

> **一句话**：以 DNA 的一条链（模板链）为模板，按碱基互补配对原则，由 RNA 聚合酶合成 mRNA 的过程——遗传信息从 DNA 流向 RNA。
> **English**: The process in which RNA polymerase uses one DNA strand (template strand) as a template to synthesize mRNA by complementary base pairing — genetic information flows from DNA to RNA.

---

## 🎓 高中进阶

### 定义
**转录**（transcription）：基因表达的第一步，在 RNA 聚合酶的催化下，以 DNA 模板链（3'→5' 方向）为模板，四种核糖核苷酸（ATP、UTP、GTP、CTP）按碱基互补配对原则，合成 mRNA（5'→3' 方向延伸）。

### 原理 / 规律

#### 转录的场所与时间

| 生物类型 | 转录场所 |
|----------|---------|
| 真核细胞 | 细胞核（主要）、线粒体、叶绿体 |
| 原核细胞 | 细胞质（无核膜） |

转录发生在**细胞有基因表达需求时**，不局限于特定分裂时期。

#### 转录所需条件

| 条件 | 说明 |
|------|------|
| 模板 | DNA 模板链（一条，3'→5' 方向读取） |
| 原料 | 四种核糖核苷酸（A、U、G、C） |
| 酶 | RNA 聚合酶（识别启动子，解旋并合成 RNA） |
| 能量（ATP） | 提供合成反应能量 |

#### 碱基对应关系

| DNA 模板链碱基 | mRNA 碱基 |
|--------------|----------|
| A | U |
| T | A |
| G | C |
| C | G |

**记忆口诀**：模板链 A→U，T→A，G→C，C→G（DNA 的 T 在 mRNA 中变为 U）

#### 转录的过程

```
RNA 聚合酶 识别启动子
    ↓
局部解旋（打开双链）
    ↓
以模板链 3'→5' 为模板，核糖核苷酸逐个加入（5'→3'）
    ↓
形成 mRNA 链（在真核细胞中为前体 mRNA，需加工剪接）
    ↓
RNA 聚合酶到达终止子，mRNA 释放，DNA 重新合并
```

#### 与 DNA 复制的对比

| 特征 | DNA 复制 | 转录 |
|------|---------|------|
| 模板 | DNA 双链（均为模板） | DNA 一条链（模板链） |
| 原料 | 脱氧核苷酸（dNTP） | 核糖核苷酸（NTP） |
| 产物 | 子代 DNA（双链） | mRNA（单链） |
| 酶 | DNA 聚合酶、解旋酶 | RNA 聚合酶 |
| 碱基 A 配对 | A-T | A-U |

### 典型例题

**例题**：某 DNA 编码链（非模板链）碱基序列为 5'-ATGGCATAG-3'，转录产生的 mRNA 碱基序列？

**解题**：
- 编码链与 mRNA 碱基序列相同（T 换成 U）
- mRNA：5'-**AUGGCAUAG**-3'

**注**：模板链是与编码链互补的那条（3'-TACCGTATC-5'），转录时 RNA 聚合酶沿 3'→5' 读模板链，合成 5'→3' 的 mRNA。

### 易错点
1. **转录只用 DNA 的一条链（模板链）**，不是两条同时转录；不同基因的模板链可能在不同链上。
2. **mRNA 中无 T（胸腺嘧啶）**，只有 U（尿嘧啶）。
3. **真核细胞转录产物需加工**：去除内含子（intron），保留外显子（exon），拼接为成熟 mRNA。
4. **转录不等于翻译**：转录产生的是 mRNA，翻译才是合成蛋白质。
