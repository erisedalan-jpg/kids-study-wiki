# 单倍体

> **一句话**：含有本物种配子染色体数目的个体（一组或多组染色体组），通常高度不育；花药离体培养可获得单倍体，再经秋水仙素处理加倍后可快速纯合育种。
> **English**: An organism with the gamete chromosome number of its species (one or more chromosome sets), usually sterile; anther culture produces haploids, which can be chromosome-doubled by colchicine for rapid homozygous breeding.

---

## 🎓 高中进阶

### 定义
**单倍体**（haploid）：细胞中含有本物种正常配子所含染色体数目的生物体。

- 二倍体（2n）物种的单倍体：含 n 条染色体（1 个染色体组）
- 四倍体（4n）物种的单倍体：含 2n 条染色体（2 个染色体组）

**重要区分**：单倍体的定义是"含配子染色体数"，不是简单等于"含 1 个染色体组"（多倍体物种的单倍体含多组染色体）。

### 原理 / 规律

#### 单倍体的产生方式

| 方式 | 说明 |
|------|------|
| 自然孤雌发育 | 未受精卵直接发育（如蜜蜂雄蜂、蚂蚁雄蚁） |
| 花药离体培养 | 人工将花粉粒在特定培养基上培养，花粉粒发育为单倍体植株（最常用） |
| 远缘杂交后染色体消除 | 某些杂交后，一方亲本的染色体在早期胚胎发育中消除，产生单倍体 |

#### 单倍体的特点

| 特点 | 说明 |
|------|------|
| 高度不育 | 减数分裂时无同源染色体可联会，不能产生正常配子（有少量例外） |
| 植株矮小 | 细胞内基因组减半，代谢产物减少 |
| 含隐性基因的表现 | 单倍体中每个基因只有一份，隐性基因直接表现（无显性覆盖） |

#### 单倍体育种流程

```
优良二倍体植株（2n）
    ↓ 花药离体培养（取减数分裂四分体时期花粉）
单倍体植株（n）—— 高度不育
    ↓ 秋水仙素处理（染色体加倍）
纯合二倍体植株（2n，全部纯合）
    ↓ 筛选优良个体
新品种（明显缩短育种年限）
```

**优点**：后代全部纯合，不再分离，比传统自交纯化（需 7 代左右）快得多。

#### 单倍体与多倍体的区别

| | 单倍体 | 多倍体 |
|-|--------|--------|
| 来源 | 配子发育 | 染色体加倍（细胞） |
| 染色体组数 | 正常体细胞的一半 | 正常体细胞的数倍 |
| 育性 | 通常不育 | 通常可育 |
| 用途 | 快速纯合育种 | 多倍体育种（器官大、产量高） |

### 典型例题

**例题**：用花药离体培养方法对基因型为 AaBb 的植株进行单倍体育种，秋水仙素处理后，得到的植株基因型有哪些？

**解题**：
- AaBb 产生配子：AB、Ab、aB、ab（各占 1/4）
- 花药离体培养得到 4 种单倍体：含 AB、Ab、aB、ab 的植株
- 秋水仙素处理（染色体加倍）：
  - AB → AABB
  - Ab → AAbb
  - aB → aaBB
  - ab → aabb
- **得到 4 种纯合基因型**：AABB、AAbb、aaBB、aabb

### 易错点
1. **单倍体不等于"含1个染色体组"**：多倍体的单倍体含多个染色体组（如六倍体小麦的单倍体含 3 个染色体组，2n=21）。
2. **花药离体培养得到的是单倍体，不是纯合体**：需经秋水仙素处理加倍后才纯合。
3. **单倍体育种与多倍体育种目的不同**：单倍体育种目的是快速获得纯合子；多倍体育种目的是利用多倍体特性（果实大、含量高等）。
