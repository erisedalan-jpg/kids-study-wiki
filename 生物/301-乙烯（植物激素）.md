# 乙烯（植物激素）
> **一句话**：乙烯是唯一气态植物激素，化学式 CH₂=CH₂，能促进果实成熟、器官衰老脱落和三重反应，在果实采后催熟和农业上有广泛应用。
> **English**: Ethylene is the only gaseous plant hormone (CH₂=CH₂), promoting fruit ripening, organ senescence, abscission, and the triple response in etiolated seedlings; it is widely used for post-harvest fruit ripening.

---

## 🎓 高中进阶

### 定义
**乙烯（ethylene, ETH）**：化学式 **CH₂=CH₂**（乙烯，最简单的烯烃），是唯一在常温常压下以**气体**形式存在的植物激素。由多种组织合成（成熟果实、种子、茎节、受机械伤的组织等），通过**气体扩散**传输，无需载体，作用半径受环境影响。

### 机制 / 过程

**乙烯的主要功能**

| 功能 | 机制/应用 |
|-----|---------|
| 促进果实成熟 | 激活果实软化酶（果胶酶等）→ 细胞壁软化、糖积累、色素合成 |
| 促进器官衰老脱落 | 诱导叶片、花、果实脱落（与 ABA 协同，与 CTK 拮抗） |
| 三重反应 | 暗中生长的幼苗：茎变粗、变短、水平生长（机械胁迫的适应） |
| 促进不定根发生 | 部分植物的生根反应 |
| 抑制茎的伸长 | 与 GA 拮抗 |

**乙烯合成途径（SAM 途径）**
```
蛋氨酸（Met）→ SAM（S-腺苷蛋氨酸）→ ACC → 乙烯
（关键酶：ACC 合酶，ACC 氧化酶）
```
- 损伤、成熟、脱落等信号触发 ACC 合酶基因表达。
- **乙烯自催化**：乙烯可促进自身进一步合成（正反馈），解释了"一个烂苹果带坏一筐"的现象。

**乙烯与果实成熟（高考重点）**

```
果实成熟过程中乙烯自催化
   ↓
乙烯浓度急剧升高（climacteric peak，跃变期）
   ↓
细胞壁软化（果胶酶、纤维素酶活性增加）
淀粉→糖（糖分积累）
叶绿素分解→花青素/类胡萝卜素合成（变色）
   ↓
果实成熟（香甜、变软、变色）
```

**乙烯的三重反应（triple response）**
- 黑暗中生长的幼苗（如豌豆）被乙烯处理：
  1. 茎**变短加粗**（矮化）
  2. 茎**水平生长**（不直立）
  3. 根**弯曲**
- 三重反应是高中经典实验验证乙烯功能的工具。

**与其他激素比较**

| 比较项 | 乙烯 | 生长素（IAA） |
|-------|-----|------------|
| 化学本质 | 气体（CH₂=CH₂） | 吲哚乙酸 |
| 传输方式 | 气体扩散 | 极性运输 |
| 主要功能 | 成熟、衰老、脱落 | 细胞伸长、向性 |
| 两重性 | 无明显 | 有（双重性） |

### 典型例题
**例题**："将苹果与未成熟的香蕉放在密封袋中，香蕉可以提前成熟。"从植物激素角度分析原因。

**解析**：成熟苹果释放大量乙烯 → 乙烯以气体形式扩散到密封袋中 → 作用于香蕉果实 → 激活香蕉的乙烯合成（乙烯自催化）和成熟相关酶 → 香蕉提前成熟（变软、变黄）。

### 易错点
- 乙烯是**气体**，不需要极性运输载体，通过扩散在植物体内和植物间传播（与其他植物激素不同）。
- 乙烯自催化（正反馈）：乙烯→促进更多乙烯合成，这是果实快速成熟和"一烂带一筐"的原因。
- 乙烯与生长素的关系：高浓度 IAA 促进乙烯合成（IAA→ETH），乙烯抑制 IAA 运输和信号，体现激素网络调控。
