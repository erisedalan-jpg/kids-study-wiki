# 基因重组

> **一句话**：有性生殖过程中，减数分裂时同源染色体交叉互换（互换型）或非同源染色体自由组合（自由组合型），使后代基因型多样化，是生物变异的重要来源。
> **English**: During sexual reproduction, genetic recombination occurs via crossing-over between homologous chromosomes or independent assortment of non-homologous chromosomes, generating diverse offspring genotypes.

---

## 🎓 高中进阶

### 定义
**基因重组**（genetic recombination）：**有性生殖**过程中，控制不同性状的基因重新组合，产生亲本中未曾有过的基因型（性状组合）的遗传现象。是基因型（而非碱基序列）层次的变异。

### 原理 / 规律

#### 基因重组的两种方式

| 方式 | 发生时期 | 机制 | 对应定律 |
|------|---------|------|---------|
| 自由组合型 | 减数第一次分裂后期 | 非同源染色体上的非等位基因随非同源染色体自由组合 | 自由组合定律 |
| 交叉互换型 | 减数第一次分裂四分体时期（前期） | 同源染色体的非姐妹染色单体之间发生交叉互换，交换片段 | 连锁与互换定律 |

#### 自由组合型基因重组举例

亲本：AABB × aabb → F₁：AaBb

F₁ 减数分裂时：
- A/a 与 B/b 独立组合 → 配子：AB, Ab, aB, ab（各 1/4）
- F₂ 新出现的基因型：AAbb、Aabb、aaBB、aaBb 等（亲本中没有的组合）

$$\text{F}_2 \text{表现型} = 9A\_B\_ : 3A\_bb : 3aaB\_ : 1aabb$$

#### 交叉互换型基因重组举例

同源染色体上 A-B 连锁（AB 染色体 / ab 染色体）：
- 正常：配子 AB, ab
- 交叉互换后新增：Ab, aB（重组型配子）

重组频率（互换值）= 重组型配子数 / 总配子数 × 100%，反映两基因间的相对距离（遗传距离，单位 cM）。

#### 基因重组的意义

| 意义 | 说明 |
|------|------|
| 遗传多样性来源 | 有性生殖后代基因型种类远多于亲本，是进化原材料 |
| 育种应用 | 杂交育种利用基因重组将优良基因集中到同一个体 |
| 进化贡献 | 与自然选择结合，推动种群适应性进化 |

#### 基因重组与基因突变的比较

| | 基因突变 | 基因重组 |
|-|---------|---------|
| 本质 | 基因碱基序列改变 | 基因间的重新组合（序列不变） |
| 发生时期 | 任何时期（复制时多见） | 减数分裂（有性生殖时） |
| 是否产生新基因 | 是 | 否（产生新基因型） |
| 频率 | 低（自发突变） | 高（有性生殖必然发生） |

### 典型例题

**例题**：基因型 AaBb 的个体自交，后代中 AAbb 的概率？

**解题**：
- AaBb 自交（两对基因独立遗传）
- AA 的概率 = 1/4；bb 的概率 = 1/4
- AAbb = 1/4 × 1/4 = **1/16**
- 这是通过自由组合型基因重组产生的新基因型。

### 易错点
1. **基因重组不产生新基因**，只产生新基因组合（新基因型）；新基因由基因突变产生。
2. **只有有性生殖才能发生减数分裂产生的基因重组**；无性生殖（有丝分裂）不发生基因重组（正常情况下）。
3. **原核生物也有"基因重组"**（转化、转导、接合等），但高中阶段基因重组特指真核生物有性生殖过程中减数分裂产生的重组。
