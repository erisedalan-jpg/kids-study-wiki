# 现代生物进化理论

> **一句话**：以达尔文自然选择学说为核心，融合群体遗传学，以"种群基因频率的定向改变"为进化实质，以"隔离"为新物种形成的必要条件的进化理论体系。
> **English**: Building on Darwin's natural selection, the modern evolutionary synthesis defines evolution as directional changes in a population's allele frequencies, with reproductive isolation as the key requirement for new species formation.

---

## 🎓 高中进阶

### 定义
**现代生物进化理论**（现代综合进化理论，Modern Synthesis）：20 世纪 30-40 年代，由杜布赞斯基（Dobzhansky）等人整合达尔文进化论与孟德尔遗传学、群体遗传学而形成的理论，核心要点包括：

1. **种群是进化的基本单位**（不是个体）
2. **基因频率的改变是进化的实质**
3. **突变和基因重组提供进化原材料**
4. **自然选择决定进化方向**
5. **隔离是新物种形成的必要条件**

### 原理 / 规律

#### 进化的实质

$$\text{生物进化} = \text{种群基因频率的定向改变}$$

- **种群**（population）：同一地点、同一时间、同种生物的全部个体
- **基因库**（gene pool）：种群中全部个体的全部基因的总和
- **基因频率**（allele frequency）：某基因在该基因所有等位基因中所占的比例

#### 变异（进化原材料）

| 变异来源 | 说明 |
|---------|------|
| 基因突变 | 产生新等位基因，是进化的根本原材料 |
| 基因重组 | 产生新基因型组合，增加遗传多样性 |
| 染色体变异 | 大规模变异，可快速改变染色体组成 |

#### 自然选择的作用

- 自然选择对**表现型**起作用（选择有利表现型，淘汰不利表现型）
- 通过表现型间接选择**基因型**，最终改变**基因频率**
- 自然选择是进化的方向性力量（非随机）

#### 遗传漂变（补充）

在**小种群**中，基因频率可因随机事件（非自然选择）发生偶然变化——**遗传漂变**（genetic drift），可导致进化方向与自然选择无关。

#### 物种形成三个环节

$$\text{突变和基因重组（原材料）} + \text{自然选择（方向）} + \text{隔离（新物种形成条件）}$$

### 典型例题

**例题**：某种群中 AA 占 20%，Aa 占 60%，aa 占 20%，A 的基因频率？a 的基因频率？

**解题**：
- A 的基因频率 = AA + 1/2 Aa = 20% + 1/2×60% = 20% + 30% = **50%**
- a 的基因频率 = aa + 1/2 Aa = 20% + 30% = **50%**

（也可直接：a = 1 - A = 1 - 50% = 50%）

### 易错点
1. **进化的单位是种群，不是个体**：个体不会进化，种群才会进化（基因频率改变）。
2. **进化 ≠ 物种形成**：种群基因频率改变就是进化，但不一定产生新物种（需要隔离）。
3. **自然选择直接作用于表现型**，不直接作用于基因型（隐性杂合子 Aa 表现为显性，不被选择，隐性基因在种群中维持一定频率）。
