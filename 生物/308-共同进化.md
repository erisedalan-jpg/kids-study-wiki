# 共同进化

> **一句话**：不同物种之间、生物与无机环境之间在相互作用中不断进化和发展的现象。
> **English**: Coevolution is the process by which species or species and their environments evolve in mutual response.

---

## 🎓 高中进阶

### 定义

**共同进化（协同进化）**：不同物种之间，或生物与无机环境之间，在相互作用中不断进化和发展的过程。是现代生物进化理论的重要内容，强调生物进化不是孤立的，而是网络化的。

### 类型

| 类型 | 内涵 | 例子 |
|------|------|------|
| **物种间共同进化** | 两个或多个物种相互适应进化 | 花和传粉昆虫；捕食者和猎物；寄生与宿主 |
| **生物与环境共同进化** | 生物改变环境，环境塑造生物 | 蓝藻光合产生 O₂ → 大气含 O₂ → 真核需氧生物兴起 |

### 经典案例

| 案例 | 共同进化关系 |
|------|----------|
| 兰花与传粉者 | 兰花花蕊距 28cm → 推测有长喙蛾匹配（达尔文） → 后发现非洲长喙天蛾 |
| 猎豹与汤姆森瞪羚 | 猎豹更快 → 瞪羚更敏捷 → 互相推动速度极限 |
| 病毒与宿主 | 病毒突变躲避免疫 → 宿主免疫强化 |
| 豆科植物与根瘤菌 | 互利共生：植物供糖、菌固氮 |
| 真菌与昆虫（白蚁） | 白蚁取食真菌 → 真菌依赖白蚁巢穴 |
| 食虫植物与小昆虫 | 适应捕食、避免被捕 |

### 生物与环境的共同进化

| 阶段 | 事件 |
|------|------|
| 25 亿年前 | 蓝藻出现 → 光合产生 O₂ |
| 大氧化事件 | 大气 O₂ 由极低逐步升高至约 21% |
| 真核生物兴起 | O₂ 浓度上升 → 真核需氧生物进化 |
| 多细胞 + 寒武纪大爆发 | 物种激增 |
| 陆地生态系统 | 植物登陆 → 改变陆地环境 → 动物登陆 |

### 在现代进化论中的地位

**现代生物进化理论**核心要点（高中教材）：
1. 种群是进化的基本单位
2. 突变和基因重组提供原材料
3. 自然选择决定进化方向
4. 隔离导致物种形成
5. **生物进化是物种间和生物与环境间共同进化的过程**
6. **生物多样性是共同进化的结果**

### 与生物多样性的关系

- 共同进化推动多样性形成
- 物种多样性 + 基因多样性 + 生态系统多样性
- 生物多样性的根源：长期共同进化

### 典型例题

**例**：举例说明物种间的共同进化。
**答**：兰花与传粉昆虫——兰花进化出独特花型适应特定昆虫，昆虫进化出特定口器吸蜜；二者相互选择。

**例**：生物多样性形成的原因？
**答**：长期共同进化（突变/重组 + 自然选择 + 隔离 + 物种间和生物-环境间相互作用）。

### 易错点

- 共同进化不限于"互利"——捕食、寄生、竞争都涉及共同进化
- 生物与无机环境也共同进化（不只生物之间）
- 共同进化是结果而非进化原因
- 共同进化导致**物种多样性**（不只单一物种适应）
