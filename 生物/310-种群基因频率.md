# 种群基因频率

> **一句话**：在一个种群基因库中，某个等位基因占该基因座位上全部等位基因的比例。
> **English**: Allele frequency is the proportion of a particular allele among all allele copies at a given locus in a population.

---

## 🎓 高中进阶

### 定义

**基因频率（等位基因频率）**：在一个种群基因库中，某个等位基因数占该基因座上全部等位基因数的比例。是衡量种群遗传结构的核心指标，也是判断进化是否发生的关键。

### 基本公式

设某基因座有等位基因 A 和 a，种群中：
- AA 个体数 = N₁
- Aa 个体数 = N₂
- aa 个体数 = N₃
- 种群总数 N = N₁ + N₂ + N₃

**基因频率：**

$$f(A) = \frac{2N_1 + N_2}{2N}$$
$$f(a) = \frac{2N_3 + N_2}{2N}$$

且 f(A) + f(a) = 1

### 基因型频率

| 基因型 | 频率 |
|------|------|
| AA | N₁/N |
| Aa | N₂/N |
| aa | N₃/N |

三者之和 = 1。

### 哈迪-温伯格平衡（前奏）

理想种群（无突变、无选择、无迁移、无遗传漂变、随机交配、足够大）下：

设 f(A) = p, f(a) = q（p + q = 1），则：

$$p^2 + 2pq + q^2 = 1$$

| 基因型 | 频率 |
|------|------|
| AA | p² |
| Aa | 2pq |
| aa | q² |

代际间基因频率不变 → 种群处于遗传平衡 → 不进化

### 影响基因频率变化的因素

| 因素 | 影响 |
|------|------|
| 突变 | 产生新等位基因 |
| 自然选择 | 改变某些基因频率（淘汰/有利） |
| 基因重组 | 改变基因型频率（不直接改变基因频率） |
| 迁入迁出 | 引入/带走基因 |
| 遗传漂变 | 小种群中随机波动 |
| 非随机交配 | 改变基因型频率 |

### 进化的本质

**生物进化 = 种群基因频率的定向改变。**

只要基因频率改变 → 即发生进化（无论改变多大）。

### 典型例题

**例**：某种群 100 个个体，AA 36，Aa 48，aa 16。求 f(A)、f(a)。
**答**：
- f(A) = (2×36 + 48) / 200 = 120/200 = 0.6
- f(a) = (2×16 + 48) / 200 = 80/200 = 0.4

**例**：白化病 aa 个体频率为 1/10000，f(a) 与 f(A) 各为？（设满足遗传平衡）
**答**：q² = 1/10000 → q = 0.01；p = 0.99；杂合子 Aa 频率 = 2pq = 0.0198 ≈ 1/50。

**例**：种群中等位基因频率改变是否一定导致进化？
**答**：是。生物进化的实质就是种群基因频率的改变。

### 易错点

- 基因频率以**等位基因数**为分母（每个二倍体个体有 2 个等位基因），不是个体数
- 基因型频率改变 ≠ 基因频率改变（如纯合 vs 杂合比变化）
- 即使表型未变，基因频率改变即是进化
- 哈迪-温伯格平衡是理想模型，自然界不严格成立
