# 自然选择学说

> **一句话**：达尔文提出的进化论核心理论：生物的进化主要由自然选择驱动——适应环境的变异个体更易生存繁殖，从而使适应性基因在种群中频率增加。
> **English**: Darwin's natural selection theory: variations adaptive to environment improve survival and reproduction, increasing adaptive alleles' frequency in a population.

---

## 🎓 高中进阶

### 定义

**自然选择学说**：达尔文（Charles Darwin）于 1859 年在《物种起源》中提出的进化理论核心。其核心思想：在生存斗争中，适应环境的有利变异更易被保留并遗传，不适应的被淘汰，经过长期积累，导致物种进化和新物种形成。

### 四大要点（达尔文原始版本）

1. **过度繁殖**：生物产生的后代远多于环境可承载的数量
2. **生存斗争**：资源有限 → 个体间竞争（与同种、异种、环境三种）
3. **遗传变异**：个体间存在可遗传的差异
4. **适者生存**：有利变异被保留并遗传，不利变异被淘汰

### 核心公式

```
过度繁殖 + 资源有限 → 生存斗争
              ↓
存在变异（不一定全有用）
              ↓
有利变异 → 适应 → 生存繁殖 → 后代增多
不利变异 → 不适应 → 被淘汰 → 后代减少
              ↓
经过长期、大规模的自然选择
              ↓
物种进化、新物种形成
```

### 经典例证

| 例子 | 说明 |
|------|------|
| 长颈鹿的颈 | 较长颈个体更易获取高处叶 → 生存繁殖优势 → 后代多 → 群体颈逐渐变长 |
| 桦尺蠖（黑化）  | 工业革命前，浅色蛾在浅色树皮上隐蔽好；工业污染使树皮变黑后，黑色蛾隐蔽好 → 黑色蛾比例上升（**工业黑化**） |
| 抗生素抗性细菌 | 滥用抗生素 → 抗药突变细菌生存繁殖 → 抗药基因频率上升 |
| 加拉帕戈斯地雀 | 不同岛屿食物不同 → 喙形差异 → 岛屿间形成不同种 |

### 现代综合进化论的扩展

达尔文学说局限：
- 未能解释**变异的来源**（不知 DNA、基因）
- 未量化机制

现代进化论补充：
- **变异来源**：基因突变、基因重组、染色体变异
- **进化基本单位**：种群（不是个体）
- **进化实质**：种群基因频率改变
- **新物种形成必要条件**：隔离

### 自然选择的类型

| 类型 | 特点 | 例子 |
|------|------|------|
| 稳定性选择 | 淘汰极端类型，保留中间型 | 婴儿出生体重 |
| 定向性选择 | 选择某一极端方向 | 工业黑化 |
| 分裂性选择 | 同时保留两极端 | 同地分化为两亚种 |

### 与拉马克学说对比

| 项目 | 拉马克 | 达尔文 |
|------|------|------|
| 进化原因 | 用进废退、获得性遗传 | 自然选择 |
| 变异方向 | 定向（朝有利方向） | 不定向（先变异，环境选择） |
| 后天性状 | 可遗传 | 不可遗传 |
| 评价 | 已被否定 | 是现代进化论基础 |

### 典型例题

**例**：用自然选择学说解释蚊蝇对杀虫剂产生抗性。
**答**：种群中本来就存在抗药变异个体（不定向）；杀虫剂选择 → 不抗药死亡，抗药存活繁殖 → 抗药基因频率上升 → 抗药种群形成。

### 易错点

- 变异**先于**环境选择，不是因为环境需要才产生变异
- 自然选择**淘汰**不利变异、**保留**有利变异，不主动"创造"
- "适者"是统计意义，不是绝对——某些情况"幸运者"也存活
- 自然选择作用于**表现型**，但只有**可遗传**的变异有进化意义
