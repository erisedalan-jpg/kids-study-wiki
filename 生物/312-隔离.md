# 隔离

> **一句话**：种群间个体不能自由交流基因的现象，包括地理隔离和生殖隔离两类，是物种形成的必要条件。
> **English**: Isolation is the prevention of gene flow between populations, including geographic and reproductive isolation; it is essential for speciation.

---

## 🎓 高中进阶

### 定义

**隔离**：在自然条件下，不同种群间不能或难以发生基因交流的现象。是物种形成必要条件。

### 两大类型

| 类型 | 含义 | 例子 |
|------|------|------|
| **地理隔离** | 由空间上的地理阻碍（山、海、河流等）造成的种群分隔 | 大陆漂移、岛屿隔离、山脉分隔 |
| **生殖隔离** | 即使个体相遇，也不能交配或交配后无可育后代 | 马驴杂交得骡（不育） |

### 生殖隔离的具体形式

**合子前隔离（受精前）：**

| 形式 | 含义 |
|------|------|
| 季节隔离（时间隔离） | 繁殖季节不同 |
| 栖息地隔离 | 生境不同，不易相遇 |
| 行为隔离 | 求偶行为不同（鸟类鸣叫、萤火虫闪光） |
| 机械隔离 | 生殖器官形态不匹配 |
| 配子隔离 | 配子识别失败（如海洋生物精卵蛋白识别） |

**合子后隔离（受精后）：**

| 形式 | 含义 |
|------|------|
| 杂种不活 | 杂交合子不能正常发育 |
| 杂种不育 | 杂种成活但不育（如骡） |
| 杂种衰退 | F1 可育，F2 衰退或不育 |

### 与物种形成的关系

```
原种群
  ↓ 地理隔离（多数情况）
分离的两个种群
  ↓ 各自累积变异 + 自然选择
基因库差异增大
  ↓ 形成生殖隔离
新物种形成
```

**关键**：

- 地理隔离 → 是常见的"前奏"，但不是充分条件
- 生殖隔离 → 物种形成的标志（一旦形成，重新接触也不再交流基因）

### 经典例子

| 例 | 隔离类型 | 结果 |
|----|--------|------|
| 加拉帕戈斯地雀 | 岛屿地理隔离 → 生殖隔离 | 形成 13 种地雀 |
| 马 + 驴 → 骡 | 生殖隔离（杂种不育） | 马驴是不同物种 |
| 普通小麦多倍体起源 | 染色体倍化的同域生殖隔离 | 一代形成新物种 |
| 北美狼 vs 草原狼 | 部分地理 + 行为隔离 | 仍可能杂交 |

### 隔离的相对性

- 地理隔离不一定导致生殖隔离（短期或环境相似时）
- 部分生殖隔离 → 不完全物种（亚种、半物种）
- 隔离消除（栖息地连通）后，可能重新基因交流

### 典型例题

**例**：物种形成必须经历地理隔离吗？
**答**：不一定。多倍体形成（同域）可以一代产生生殖隔离。

**例**：骡子能繁殖吗？说明什么？
**答**：骡子不育。马驴是不同物种，存在生殖隔离。

**例**：判断两种群是否属于同一物种的根本标准是？
**答**：是否存在生殖隔离。

### 易错点

- 地理隔离 ≠ 生殖隔离
- 地理隔离是物种形成的"通常途径"，但不是绝对必要
- 生殖隔离是判定不同物种的**根本标准**
- 即使表型差异大，能产生可育后代仍属同物种（如家狗各品种）
