# 淋巴

> **一句话**：淋巴是组织液渗入毛细淋巴管后形成的液体，经淋巴循环最终汇入血浆，是内环境的重要组成部分。
> **English**: Lymph is the fluid that enters lymph capillaries from tissue fluid; it flows through the lymphatic system and ultimately returns to plasma.

---

## 🎓 高中进阶

### 定义

淋巴（lymph）是[[313-内环境|内环境]]三大组成部分之一，由[[316-组织液|组织液]]渗入毛细淋巴管（lymph capillary）后形成。

**成分特点**：
- 水分、无机盐、葡萄糖等与组织液相近
- 含少量蛋白质（比组织液略多，比血浆少）
- 含大量**淋巴细胞**（lymphocyte）和少量其他白细胞
- 可含乳糜微粒（小肠淋巴管中，富含脂肪）

### 机制 / 过程

**淋巴循环路径**：

```
组织液 → 毛细淋巴管 → 淋巴管 → 淋巴结（过滤）
→ 胸导管/右淋巴导管 → 锁骨下静脉 → 血浆
```

**淋巴的功能**：

| 功能 | 说明 |
|------|------|
| 回流组织液 | 维持血浆容量，防止水肿 |
| 转运脂肪 | 小肠吸收的脂肪以乳糜微粒形式经淋巴运输 |
| 免疫监控 | 淋巴结中淋巴细胞识别并消灭病原体 |

**淋巴管阻塞 → 水肿**：见[[316-组织液|组织液]]"组织水肿"部分（丝虫病典型案例）。

### 典型例题

**例题**：下列关于淋巴的叙述，正确的是（　）
A. 淋巴中蛋白质含量与血浆相同
B. 淋巴最终汇入血浆
C. 淋巴管中不含免疫细胞
D. 淋巴与组织液成分完全不同

**答案**：B

**解析**：淋巴经胸导管和右淋巴导管汇入锁骨下静脉，回流至血浆；淋巴蛋白质含量少于血浆；淋巴管内含淋巴细胞；淋巴由组织液转化而来，成分相似。

### 易错点

1. **淋巴不直接来自血浆**：路径为血浆→组织液→淋巴，不是血浆直接渗入淋巴管
2. **脂肪吸收途径**：脂肪经淋巴（乳糜管）吸收，不经毛细血管直接进入血液
3. **淋巴结 ≠ 淋巴腺**：淋巴结是免疫器官，非分泌腺
