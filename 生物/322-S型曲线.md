# S型曲线

> **一句话**：S型曲线描述种群在有限环境中数量先加速后减速、最终趋近K值（环境容纳量）的"逻辑斯谛增长"模式。
> **English**: The S-shaped curve describes logistic population growth, where numbers first accelerate then decelerate, eventually approaching the carrying capacity (K).

---

## 🎓 高中进阶

### 定义

**S型曲线（S-shaped curve / 逻辑斯谛增长，logistic growth）**：在食物和空间资源有限的自然环境中，种群增长速率先升后降，种群数量最终趋近一个最大值——**环境容纳量（K值）**，在坐标图上呈"S"字形。

### 概念 / 类型

**关键特征**：

| 阶段 | 种群数量 | 增长速率 | 增长率 |
|------|----------|----------|--------|
| 早期（N < K/2） | 数量少 | 逐渐增大 | 较高但下降 |
| K/2 处 | N = K/2 | **最大** | 中等 |
| 后期（N > K/2） | 接近K | 逐渐减小 | 趋近0 |
| N = K | 达到K值 | **为0** | 0 |

**与J型曲线对比**：

| 比较项 | J型 | S型 |
|--------|-----|-----|
| 条件 | 理想（无限资源） | 现实（有限资源） |
| 增长率 | 恒定（λ不变） | 持续下降 |
| 增长速率 | 持续增大 | 先增后减，K/2最大 |
| 上限 | 无 | K值 |
| 实例 | 外来物种初期 | 自然种群 |

**K值的意义与应用**：

| 应用场景 | 策略 | 原理 |
|----------|------|------|
| 捕鱼/猎获 | 在K/2时捕捞 | 增长速率最大，种群恢复最快 |
| 害虫防治 | 降低K值 | 减少食物、破坏栖息地，使种群数量维持低水平 |
| 濒危物种保护 | 提高K值 | 扩大保护区，增加食物供应 |

### 典型例题

**例1**：某鱼塘中鲤鱼种群的K值为1000条，当种群数量为多少时，捕捞效益最高？为什么？

**答**：种群数量为 **500 条（K/2）** 时捕捞效益最高。此时增长速率最大，捕捞后种群能以最快速度恢复，实现可持续利用。

**例2**：用S型增长方程分析：某鼠种群K值为200，当种群数量为150时，其增长速率与种群数量为50时相比，哪个更大？

**答**：种群数量为50时增长速率更大（50 < K/2 = 100，增长速率仍在上升；150 > K/2，增长速率已在下降）。

**例3（计算）**：S型曲线种群，K=400，N=200时增长速率最大为r·K/4（逻辑斯谛方程）；若N=100，增长速率 = r·100·(1-100/400) = 75r；N=300，增长速率 = r·300·(1-300/400) = 75r。即两者增长速率相同（关于K/2对称）。

### 易错点

1. **K/2是增长速率最大点，不是增长率最大点**。增长率（ΔN/N）在S型曲线中持续下降。
2. 到达K值后种群数量不是绝对不变，而是在K值附近**波动**（实际种群受随机因素影响）。
3. 种群数量超过K值时**增长速率为负**（死亡率>出生率），不是停止增长。
4. 防治害虫应在**数量少时**（远低于K/2）就干预，而不是等到K/2再处理。
5. K值不是物种固有的，会随环境条件变化而改变。
