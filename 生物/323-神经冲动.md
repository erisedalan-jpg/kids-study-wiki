# 神经冲动

> **一句话**：神经冲动是沿神经纤维传导的动作电位序列，是神经系统传递信息的基本电信号形式。
> **English**: A nerve impulse is a propagating action potential along a nerve fiber, serving as the fundamental electrical signal for neural information transmission.

---

## 🎓 高中进阶

### 定义

神经冲动（nerve impulse）即沿神经纤维传播的[[318-动作电位|动作电位]]，是神经系统传递信息的电学基础。其实质是**膜电位局部变化引发邻近膜去极化的连锁反应**。

### 机制 / 过程

**冲动在神经纤维上的传导**：

```
受刺激位点（外负内正，兴奋）
    ↓ 与相邻静息区（外正内负）形成局部电流
膜外：兴奋区 ← 未兴奋区（电流方向）
膜内：兴奋区 → 未兴奋区（电流方向）
    ↓ 局部电流使相邻区去极化，产生新动作电位
冲动向两侧传播（双向）
```

**传导特点**：

| 特点 | 说明 |
|------|------|
| 双向传导 | 单个神经纤维上冲动双向传播 |
| 不衰减 | 动作电位幅度沿途不减小（全或无） |
| 有不应期 | 绝对不应期保证冲动不逆向传回已兴奋区 |
| 绝缘性 | 相邻神经纤维间不相互干扰（有髓鞘）|

**传导速度影响因素**：

| 因素 | 效应 |
|------|------|
| 有髓鞘 | 跳跃传导，速度快（70~120 m/s） |
| 纤维直径大 | 速度快 |
| 温度高 | 速度快（酶促反应加速） |

**突触处的单向传递**（与纤维上双向不同）：
- 突触处：只能从突触前→突触后（因为只有前膜释放递质，后膜有受体）
- 整体反射弧：单向（感受器→效应器）

### 典型例题

**例题**：刺激一段离体神经纤维中部，兴奋传导方向为（　）
A. 只向左传　B. 只向右传　C. 双向传导　D. 不能传导

**答案**：C

**解析**：在神经纤维上，兴奋传导是双向的——刺激中部，兴奋向两端传播；但在完整反射弧中，因突触的单向性，整体表现为单向。

### 易错点

1. **纤维上双向 vs 突触处单向**：这是高频考点，不能混淆
2. **冲动传导不衰减**：不同于化学信号在体液中扩散（浓度递减）
3. **冲动频率 = 刺激强度的编码方式**：刺激强，AP频率高，不是幅度大
