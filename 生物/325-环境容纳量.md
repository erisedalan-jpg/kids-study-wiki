# 环境容纳量

> **一句话**：环境容纳量（K值）是在特定环境中，一个种群能够长期维持的最大数量，是S型增长曲线的上限。
> **English**: The carrying capacity (K value) is the maximum population size that a specific environment can sustainably support, representing the upper limit of the S-shaped growth curve.

---

## 🎓 高中进阶

### 定义

**环境容纳量（carrying capacity，K值）**：在食物和空间资源以及其他生态因子不受破坏的情况下，一定自然环境中某种群所能稳定维持的**最大数量**。  

K值是S型增长曲线的渐近线——种群数量趋近K值但通常不会长期超越它。

### 概念 / 类型

**K值的特征**：

| 特征 | 说明 |
|------|------|
| 动态性 | 随环境变化而改变，不是固定值 |
| 平衡性 | 种群在K值附近上下波动，不是绝对稳定 |
| 物种特异性 | 不同物种在同一环境中的K值不同 |
| 可操控性 | 人为改变食物、空间、天敌等可改变K值 |

**K值与K/2的比较**：

| | K/2 | K |
|---|-----|---|
| 意义 | 增长速率最大点 | 种群数量最大值 |
| 应用 | 捕捞点（可持续利用） | 保护种群或防治害虫的参考 |
| 增长速率 | 最大 | 为0 |

**K值的应用**：

1. **渔业管理**：捕捞后使种群恢复到 K/2，此后种群增长速率最大，可持续捕捞。
2. **害虫防治**：通过减少食物来源、破坏栖息地、引入天敌等手段**降低 K 值**，使害虫种群数量维持在较低水平。
3. **濒危物种保护**：建立自然保护区、增加食物供应等，**提高 K 值**，促进种群数量恢复。

**影响 K 值的因素**：

| 因素 | 影响方向 |
|------|----------|
| 食物量增加 | K值↑ |
| 栖息地扩大 | K值↑ |
| 天敌增加 | K值↓ |
| 气候恶化 | K值↓ |
| 疾病增加 | K值↓ |

### 典型例题

**例1**：为了保护某珍稀鸟类，科学家将其K值从500只提高到了800只，可能采取了哪些措施？

**答**：①建立自然保护区，扩大栖息地面积；②增加食物供应（如投放食物）；③减少天敌数量（如控制猛禽）；④人工繁育后放归。

**例2**：某害虫K值为1000，若采取农业防治措施后K值降为200，预测该害虫种群数量的最终趋势。

**答**：害虫种群数量最终趋近新的K值200，远低于原来的1000，防治效果显著。

**例3（计算）**：某种群K=400，当前种群数量N=100，此时增长速率相对于N=300时：两者关于K/2=200对称，增长速率相同。（S型曲线关于K/2对称）

### 易错点

1. K值不是固定不变的，**环境条件改变则K值改变**。
2. 种群数量**不会永远保持在K值**，而是在K值附近波动（受随机因素影响）。
3. 种群数量超过K值时，死亡率>出生率，种群数量**会下降回K值**附近，不会无限增大。
4. K/2不是K值的一半（K/2是个概念名词，表示"K值的一半"这个数量点）——在S型曲线中，此点增长速率最大。
5. **不同物种同一环境K值不同**；同一物种不同环境K值也不同。
