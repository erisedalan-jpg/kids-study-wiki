# 突触

> **一句话**：突触是神经元之间或神经元与效应器之间的信号传递接头，实现神经冲动从一个细胞到下一个细胞的单向传递。
> **English**: A synapse is the junction between neurons (or between a neuron and an effector) that enables one-way transmission of nerve impulses via chemical neurotransmitters.

---

## 🎓 高中进阶

### 定义

突触（synapse）是[[322-神经元（高中）|神经元（高中）]]之间进行信息传递的特化结构，由**突触前膜、突触间隙、突触后膜**三部分构成。化学突触（主要类型）通过[[325-神经递质|神经递质]]实现信号传递。

### 机制 / 过程

**化学突触传递步骤**：

```
1. 动作电位到达突触小体（轴突末梢）
2. Ca²⁺内流 → 突触小泡与前膜融合
3. 神经递质释放（胞吐）至突触间隙
4. 递质与突触后膜受体结合
5. 突触后膜产生电位变化（兴奋或抑制）
6. 递质被酶降解或回收（终止信号）
```

**突触结构对照**：

| 结构 | 位置 | 特点 |
|------|------|------|
| 突触小体 | 轴突末梢膨大 | 含线粒体、突触小泡 |
| 突触前膜 | 突触小体的膜 | 释放神经递质（胞吐） |
| 突触间隙 | 两膜之间（20~30 nm） | 细胞外液充填 |
| 突触后膜 | 下一神经元树突/胞体膜 | 含特异性受体 |

**突触传递的单向性原因**：神经递质只能由突触前膜释放，突触后膜上只有受体（无法释放递质）。

**兴奋性突触 vs 抑制性突触**：
- 兴奋性：递质（如乙酰胆碱、谷氨酸）→ 后膜去极化 → 产生EPSP
- 抑制性：递质（如GABA）→ 后膜超极化 → 产生IPSP

### 典型例题

**例题**：在突触处，信号的转变形式是（　）
A. 电信号→化学信号→电信号
B. 化学信号→电信号→化学信号
C. 电信号→电信号
D. 化学信号→化学信号

**答案**：A

**解析**：突触前膜到达的是电信号（动作电位）→ 触发递质释放（化学信号）→ 作用于后膜产生新的电位变化（电信号）。

### 易错点

1. **突触间隙是细胞外液**：递质可扩散，属于内环境
2. **Ca²⁺内流是关键**：阻断Ca²⁺内流可抑制突触传递
3. **不是所有突触都产生兴奋**：抑制性突触同样重要（如脊髓抑制性中间神经元）
