# 种群数量波动

> **一句话**：种群数量波动是指种群数量在时间上的起伏变化，由非密度制约因素（如气候）和密度制约因素（如食物、天敌）共同调节。
> **English**: Population fluctuation refers to changes in population size over time, regulated by both density-independent factors (e.g., climate) and density-dependent factors (e.g., food, predators).

---

## 🎓 高中进阶

### 定义

**种群数量波动（population fluctuation）**：种群数量随时间变化而上下起伏的现象。自然界中大多数种群的数量不是固定不变的，而是在一定范围内（通常围绕K值）波动。

### 概念 / 类型

**调节种群数量波动的因素**：

| 类型 | 定义 | 举例 |
|------|------|------|
| 非密度制约因素 | 影响效果与种群密度无关 | 气温、降水、干旱、洪水、火灾 |
| 密度制约因素 | 影响效果随种群密度增大而增强 | 食物、天敌、疾病、种内竞争 |

**波动类型**：

| 类型 | 特征 | 举例 |
|------|------|------|
| 周期性波动 | 数量有规律地周期性起伏 | 旅鼠（约3-4年周期）；加拿大猞猁与雪鞋兔（约10年周期） |
| 非周期性波动 | 数量无规律地变化 | 受气候异常、疫病等偶发事件影响 |
| 种群崩溃 | 数量急剧减少 | 外来天敌入侵、栖息地破坏 |

**捕食者与猎物的数量关系**：
- 猎物增加 → 捕食者增加 → 猎物减少 → 捕食者减少 → 猎物增加（循环）
- 两者数量变化呈**负反馈调节**，维持动态平衡
- 典型案例：加拿大猞猁（捕食者）与雪鞋兔（猎物）的约10年周期波动

**种群数量变化的调节机制**：

| 机制 | 说明 |
|------|------|
| 食物调节 | 食物减少→出生率↓死亡率↑→数量↓ |
| 天敌调节 | 天敌增加→死亡率↑→数量↓ |
| 种内竞争 | 密度增大→竞争加剧→出生率↓死亡率↑ |
| 气候调节 | 极端气候→死亡率骤增→数量↓ |

### 典型例题

**例1**：下列属于密度制约因素对种群数量调节的是（ ）

A. 干旱导致某地区蝗虫大量死亡  
B. 食物减少导致狼的出生率下降 ✓  
C. 洪水淹没导致田鼠大量死亡  
D. 低温导致松鼠冬眠期死亡率升高  

**解析**：食物是密度制约因素——种群密度越大，食物竞争越激烈，出生率越低；而干旱、洪水、低温属于非密度制约因素（与种群密度无关）。

**例2**：猞猁与雪鞋兔的种群数量呈周期性波动，猞猁数量变化总是滞后于雪鞋兔，原因是____。

**答**：雪鞋兔数量增加后，猞猁食物充裕，出生率提高，需经过一段时间（繁殖周期）才表现为数量增加；同样，雪鞋兔减少后，猞猁数量减少也有一定的滞后性。

### 易错点

1. **密度制约因素 ≠ 生物因素**：食物属于生物因素也是密度制约因素；但气候属于非生物因素且是非密度制约因素——两种分类标准不同，不能混淆。
2. 种群数量在K值附近波动，不是永远等于K值，也不是无规律的。
3. 捕食者数量变化**滞后于**猎物数量变化，高考常考此知识点。
4. 种群数量波动不等于种群灭绝，正常波动是维持生态平衡的机制之一。
