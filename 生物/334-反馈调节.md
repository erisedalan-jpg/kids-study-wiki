# 反馈调节
> **一句话**：反馈调节是指调节系统的输出结果反过来影响输入，从而维持系统稳定的调控机制，负反馈是内分泌系统维持激素水平稳态的核心方式。
> **English**: Feedback regulation occurs when the output of a system feeds back to influence its own input; negative feedback is the primary mechanism by which the endocrine system maintains hormonal homeostasis.

---

## 🎓 高中进阶

### 定义
**反馈调节（feedback regulation）**：调节系统中，效应结果通过某种途径反过来作用于调节中枢，改变其活动强度，从而维持系统稳定的调节方式。

- **负反馈（negative feedback）**：结果抑制原因，趋向稳态（最常见，最重要）。
- **正反馈（positive feedback）**：结果促进原因，加速偏离（如分娩中子宫收缩、血液凝固；生物课重点是负反馈）。

### 机制 / 过程

**内分泌系统的负反馈（以甲状腺轴为例）**

```
下丘脑
  │ 分泌 TRH（促甲状腺激素释放激素）↑
  ↓
垂体前叶
  │ 分泌 TSH（促甲状腺激素）↑
  ↓
甲状腺
  │ 分泌 T₃/T₄（甲状腺激素）↑
  ↓
靶细胞（效应）
  │
  └──[负反馈]──→ 抑制下丘脑（TRH↓）和垂体（TSH↓）
```

**负反馈的生理意义**

| 激素轴 | 负反馈信号 | 效果 |
|-------|----------|------|
| 下丘脑-垂体-甲状腺 | 甲状腺激素过高 → 抑制 TRH、TSH | 防止甲亢 |
| 下丘脑-垂体-肾上腺 | 皮质醇过高 → 抑制 CRH、ACTH | 防止皮质醇过多 |
| 下丘脑-垂体-性腺 | 性激素过高 → 抑制 GnRH、FSH/LH | 精子/卵子生成受抑制 |
| 血糖调节 | 血糖升高 → 胰岛素↑ → 血糖降低 | 维持血糖平衡 |

**与稳态的关系**
负反馈调节是**内环境稳态**维持的核心机制之一，确保各种理化指标在正常范围内波动。

### 典型例题
**例题**：长期服用糖皮质激素类药物（如地塞米松）的患者，若突然停药，可能出现肾上腺皮质功能减退。试分析原因。

**解析**：外源糖皮质激素浓度长期偏高 → 负反馈持续抑制下丘脑（CRH↓）和垂体（ACTH↓） → 肾上腺皮质因长期得不到 ACTH 刺激而萎缩 → 突然停药后，自身糖皮质激素无法迅速恢复 → 肾上腺皮质功能减退。

### 易错点
- **负反馈**是维持稳态的主要方式，考试"反馈调节"默认指负反馈。
- 注意区分：切除某器官（如甲状腺）→ 缺乏负反馈 → 上游激素（TSH、TRH）浓度**升高**。
- 正反馈是少数特殊情况（分娩、血小板聚集），不是稳态调节的常规机制。
