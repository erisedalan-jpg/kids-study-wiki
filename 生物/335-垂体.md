# 垂体
> **一句话**：垂体是位于大脑底部的内分泌"总调度"，接受下丘脑释放激素的指令，分泌多种促激素调控各靶腺体。
> **English**: The pituitary gland, located at the base of the brain, acts as the "master endocrine gland," relaying hypothalamic signals to target glands via tropic hormones.

---

## 🎓 高中进阶

### 定义
垂体（pituitary gland / hypophysis）位于大脑下方蝶鞍内，由**腺垂体**（前叶）和**神经垂体**（后叶）组成。腺垂体分泌多种促激素，神经垂体储存并释放下丘脑合成的激素（如 ADH、催产素）。

### 机制 / 过程

**垂体分泌的主要激素**

| 激素 | 靶器官/效应 |
|-----|-----------|
| 促甲状腺激素（TSH） | 甲状腺 → 促进甲状腺激素分泌 |
| 促肾上腺皮质激素（ACTH） | 肾上腺皮质 → 促进糖皮质激素分泌 |
| 促性腺激素（FSH/LH） | 性腺 → 促进性激素分泌及配子发育 |
| 生长激素（GH） | 全身组织 → 促生长，升血糖 |
| 抗利尿激素（ADH）（储存释放） | 肾小管/集合管 → 重吸收水 |

**三级分级调节**

```
下丘脑 —[释放激素]→ 垂体 —[促激素]→ 靶腺体 —[效应激素]→ 靶细胞
         ←—————负反馈（效应激素浓度升高抑制前两级）——————
```

**垂体功能异常举例**

| 异常 | 阶段 | 表现 |
|-----|------|------|
| GH 分泌不足 | 幼年 | 侏儒症（比例正常，智力正常） |
| GH 分泌过多 | 幼年 | 巨人症 |
| GH 分泌过多 | 成年 | 肢端肥大症 |

### 典型例题
**例题**：用适量促甲状腺激素注射大鼠，一段时间后测血液中甲状腺激素和促甲状腺激素释放激素浓度的变化？

**解析**：外源 TSH → 甲状腺激素分泌增多 → 负反馈抑制下丘脑（TRH↓）和垂体（TSH 自身受抑制，但本题为外源注射，血中 TSH 暂时升高）。最终结果：甲状腺激素↑，TRH↓。

### 易错点
- 垂体功能减退与下丘脑功能减退症状相似，但根源不同，需区分。
- ADH 是**下丘脑合成**、**垂体后叶释放**，不是垂体自身合成。
- 侏儒症（GH 缺乏）与呆小症（甲状腺激素缺乏，幼年）均影响生长，但机制不同：前者比例正常、智力正常；后者智力受损。

## 🌐 中英对照

### 词汇

| 中文 | English |
|-----|---------|
| 垂体 | pituitary gland / hypophysis |
| 腺垂体 | adenohypophysis (anterior pituitary) |
| 神经垂体 | neurohypophysis (posterior pituitary) |
| 促甲状腺激素 | thyroid-stimulating hormone (TSH) |
| 促性腺激素 | gonadotropin (FSH / LH) |
| 生长激素 | growth hormone (GH) |

### 例句

- The **anterior pituitary** secretes TSH, which stimulates the thyroid gland.
- A tumor of the **pituitary** secreting excess GH causes gigantism in children.
