# 植物激素的应用

> **一句话**：将植物激素及其类似物（植物生长调节剂）用于农业、园艺、林业等领域，调控植物生长发育以提高产量和品质。
> **English**: Plant hormones and synthetic analogs (plant growth regulators) are used in agriculture and horticulture to enhance growth, yield, and quality.

---

## 🎓 高中进阶

### 植物生长调节剂

人工合成的、对植物生长发育具有类似植物激素作用的有机化合物。优点：成本低、效果稳定、易合成、不易被植物降解。

### 主要应用

#### 1. 生长素类（IAA、NAA、2,4-D）

| 用途 | 应用方式 |
|------|--------|
| 促进扦插生根 | 蘸取生长素溶液 → 插入土壤 → 生根 |
| 防止落花落果 | 适量喷洒 → 维持果实结实 |
| 培育无籽果实 | 番茄、葡萄花蕊喷洒 → 不经传粉发育果实 |
| 除草剂 | 高浓度 2,4-D 杀双子叶杂草（不影响单子叶作物） |
| 促进果实早熟 | 部分应用 |

注意：双子叶植物比单子叶植物对生长素敏感 → 选择性除草。

#### 2. 赤霉素

| 用途 | 应用 |
|------|------|
| 打破休眠 | 处理种子、马铃薯块茎 → 提早萌发 |
| 促进茎伸长 | 矮生品种喷施 → 增高（如葡萄无籽串拉长） |
| 促花促果 | 部分果树催花 |

#### 3. 细胞分裂素（6-BA、KT）

| 用途 | 应用 |
|------|------|
| 组织培养 | 与生长素配比促芽 / 促根 |
| 延缓衰老 | 蔬菜保鲜（喷洒延缓黄化） |
| 促进发芽 | 部分种子 |

#### 4. 脱落酸（ABA）

| 用途 | 应用 |
|------|------|
| 促进休眠 | 用于果树春化 |
| 抗旱保花保果 | 干旱诱导脱落酸维持气孔关闭 |
| 调节落叶 | 实验研究 |

#### 5. 乙烯（乙烯利）

| 用途 | 应用 |
|------|------|
| 促进果实成熟 | 香蕉、番茄催熟（乙烯利→乙烯） |
| 促进果实脱落 | 机械采收前 |
| 矮化作物 | 棉花、水稻等 |

#### 6. 油菜素内酯类（BR）

| 用途 | 应用 |
|------|------|
| 增强抗逆 | 抗旱、抗盐、抗低温 |
| 促进生长 | 提高产量 |

### 应用原则

| 原则 | 说明 |
|------|------|
| 适宜浓度 | 不同物种、不同器官最适浓度不同；过高反而抑制 |
| 适宜时期 | 关键生育期 |
| 适宜方式 | 喷洒、浸泡、涂抹 |
| 综合考虑 | 多种激素协调作用 |

### 双重效应（生长素经典例子）

**低浓度促进生长，高浓度抑制生长甚至致死。**

| 浓度（举例） | 效果 |
|-----------|------|
| 极低 | 几乎无作用 |
| 适宜 | 促进伸长 / 生根 / 结果 |
| 过高 | 抑制 → 致死（如 2,4-D 除草） |

不同器官敏感度：根 > 芽 > 茎

### 食品安全考量

- 植物激素在植物体内不残留或代谢快
- 国家标准规定残留限量
- 动物激素与植物激素本质不同，不会"催熟"动物或人

### 典型例题

**例**：用 2,4-D 除草是利用什么原理？
**答**：双子叶植物（杂草）比单子叶植物（如水稻、麦类）对 2,4-D 敏感；适宜浓度可杀双子叶不伤单子叶。

**例**：用乙烯催熟青香蕉的原理？
**答**：乙烯利分解释放乙烯，乙烯促进果实成熟。

### 易错点

- 植物激素应用要适量、适时、适法
- 同种激素在不同器官效果可能相反
- "催熟剂"是乙烯类，不是动物激素
- 植物生长调节剂虽有效，过量或不当使用仍有食品安全风险
