# 记忆细胞
> **一句话**：记忆细胞是初次免疫应答中由 B 细胞或 T 细胞分化形成的长寿命细胞，再次接触相同抗原时能迅速大量增殖，产生更快更强的二次免疫应答。
> **English**: Memory cells are long-lived B or T cells generated during the primary immune response that rapidly proliferate upon re-exposure to the same antigen, producing a faster and stronger secondary immune response.

---

## 🎓 高中进阶

### 定义
**记忆细胞（memory cell）**：在初次免疫应答中，B 细胞和 T 细胞受抗原激活后，除分化为效应细胞（浆细胞/效应 T 细胞）外，还分化出一部分**寿命长**（可存活数年至数十年）的克隆，即记忆 B 细胞和记忆 T 细胞。

### 机制 / 过程

**记忆细胞的产生与作用**

```
初次抗原刺激
   ↓
B 细胞 / T 细胞激活
   ↓ 克隆增殖
效应细胞（浆细胞 / 效应 CTL）  +  记忆细胞（记忆 B 细胞 / 记忆 T 细胞）
   ↓                                    ↓（长期存活，携带抗原识别受体）
清除抗原（初次应答）         同种抗原再次入侵
                                          ↓
                             记忆细胞迅速识别并大量增殖分化
                                          ↓
                             大量效应细胞 → 更快更强的二次应答
```

**初次 vs 二次免疫应答对比**

| 特征 | 初次应答（primary） | 二次应答（secondary） |
|-----|----------------|-----------------|
| 潜伏期 | 较长（1-2 周） | 很短（1-3 天） |
| 抗体峰值 | 较低 | 高出数倍至数十倍 |
| 抗体维持 | 较短 | 更长 |
| 主要介导 | 初始 B/T 细胞 | 记忆细胞 |
| 抗体亲和力 | 较低 | 更高（亲和力成熟） |

**记忆细胞与疫苗**
- **初次接种疫苗**（初次免疫）→ 产生少量抗体 + 大量记忆细胞。
- **加强针**（二次免疫）→ 记忆细胞迅速应答 → 抗体水平大幅提升，保护力增强。
- 天花疫苗（牛痘）接种一次后，记忆细胞可保护终身（天花消灭）。

**记忆细胞的特点**

| 特点 | 说明 |
|-----|------|
| 长寿命 | 可存活数年至终身 |
| 快速响应 | 二次抗原刺激时迅速增殖分化 |
| 携带受体 | 保留对原始抗原的特异性识别能力 |
| 低激活阈值 | 比初始细胞更容易被同种抗原激活 |

### 典型例题
**例题**：小儿接种麻疹疫苗后，若干年后再次接触麻疹病毒，能迅速产生大量抗体，不会患病。试说明原因，涉及哪些细胞？

**解析**：
1. 接种疫苗（初次免疫）→ B 细胞活化 → 浆细胞产生抗体（当时保护）+ 记忆 B 细胞形成。
2. 若干年后接触麻疹病毒（再次免疫）→ 记忆 B 细胞迅速识别同种抗原 → 大量增殖分化为浆细胞 → 短时间内产生大量抗体 → 病毒被清除，不患病。
涉及细胞：吞噬细胞（APC）、T 细胞、B 细胞、记忆 B 细胞、浆细胞。

### 易错点
- 记忆细胞**有**识别抗原的能力（携带特异性受体），浆细胞**无**识别抗原的能力。
- 记忆细胞不分泌抗体，**分泌抗体的是浆细胞**。
- 记忆细胞寿命长但不"永生"（有时间限制，如破伤风抗毒素免疫不终身，需定期补种）。
