# 过敏反应
> **一句话**：过敏反应是免疫系统对无害抗原（过敏原）发生的异常过激反应，IgE 介导的 I 型超敏反应最为常见，表现为皮疹、哮喘、过敏性休克等。
> **English**: An allergic reaction is an exaggerated immune response to harmless antigens (allergens), most commonly IgE-mediated type I hypersensitivity, manifesting as rashes, asthma, or anaphylaxis.

---

## 🎓 高中进阶

### 定义
**过敏反应（allergic reaction / hypersensitivity）**：已致敏的机体**再次接触相同过敏原**时，发生的以**免疫应答过强**为特征的组织损伤或功能紊乱。高中教材重点介绍 **I 型超敏反应**（IgE 介导）。

**过敏原（allergen）**：引起过敏反应的抗原物质，本身通常对正常人体无害（如花粉、尘螨、花生蛋白、青霉素等）。

### 机制 / 过程

**I 型超敏反应机制（两阶段）**

**第一阶段：致敏**
```
初次接触过敏原（花粉等）
   ↓
机体产生大量 IgE 抗体（特异性免疫——体液免疫产物）
   ↓
IgE 与肥大细胞（mast cell）/ 嗜碱性粒细胞表面 Fc 受体结合
   ↓
机体处于"致敏状态"（此时无症状）
```

**第二阶段：激发（再次接触）**
```
再次接触相同过敏原
   ↓
过敏原与肥大细胞表面 IgE 交联结合
   ↓
肥大细胞脱颗粒 → 释放组胺、白三烯等炎症介质
   ↓
血管扩张、血管通透性增加、平滑肌收缩、腺体分泌增多
   ↓
过敏症状：皮肤红疹/荨麻疹、鼻炎、哮喘、过敏性休克
```

**常见过敏原与症状**

| 过敏原 | 常见症状 |
|-------|---------|
| 花粉 | 过敏性鼻炎、哮喘 |
| 食物（花生、牛奶、海鲜） | 皮疹、消化道症状、重症者休克 |
| 药物（青霉素） | 皮疹、过敏性休克（最危险） |
| 尘螨 | 过敏性哮喘、皮炎 |
| 昆虫毒液（蜂毒） | 全身过敏性休克（anaphylaxis） |

**高考考点：过敏反应特点**

| 特点 | 说明 |
|-----|------|
| 特异性 | 针对特定过敏原，不同人过敏原不同 |
| 可遗传性 | 有遗传倾向（过敏体质） |
| 需致敏过程 | 初次接触不发作，再次接触才发作 |
| 组织损伤 | 对机体造成损害（功能紊乱）但通常不攻击异己病原体 |

### 典型例题
**例题**：青霉素过敏反应属于哪种免疫失调？其发生的原因是什么？为何注射前要做皮试？

**解析**：属于**过敏反应**（免疫功能过强）。原因：青霉素（或降解产物）作为过敏原，在致敏个体中激活 IgE-肥大细胞系统，再次注射时发生过敏反应。皮试目的：小剂量检测是否已致敏，防止全身注射引发过敏性休克（anaphylaxis）。

### 易错点
- 过敏反应**属于特异性免疫（体液免疫）的异常表现**，不是非特异性免疫。
- 初次接触过敏原**不会**立即发生过敏（需先致敏），高考常考此点。
- 过敏反应≠自身免疫病：过敏攻击外来无害抗原，自身免疫病攻击自身正常组织。
- 过敏反应通常是可逆的（移除过敏原后缓解），而自身免疫病往往持续进展。
