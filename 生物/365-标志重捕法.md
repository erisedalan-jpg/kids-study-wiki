# 标志重捕法

> **一句话**：标志重捕法是通过两次捕获（第一次标记放回，第二次计算标记比例）来估算动物种群总数的方法。
> **English**: The mark-recapture method estimates total population size by capturing, marking, and releasing individuals, then recapturing a sample and calculating the proportion of marked individuals.

---

## 🎓 高中进阶

### 定义

**标志重捕法（mark-recapture method）**：调查活动能力强的动物种群密度时，先捕获一部分个体加以标记后放回，经过一段时间后再次捕获，根据重捕样本中标记个体的比例，估算种群总数量的方法。

又称"捕获-标记-重捕法"（CMR，Capture-Mark-Recapture）。

### 概念 / 类型

**基本假设**（Lincoln-Petersen 法）：

1. 调查期间种群数量保持稳定（无大量出生、死亡、迁入、迁出）
2. 标记个体与未标记个体在重捕时有**相同的被捕概率**
3. 标记物不影响个体的存活率和行为
4. 标记不脱落

**核心公式**：

$$N = \frac{M \times n}{m}$$

| 符号 | 含义 |
|------|------|
| N | 种群总数（待估算） |
| M | 第一次捕获并标记的个体数 |
| n | 第二次捕获的总个体数 |
| m | 第二次捕获中有标记的个体数 |

**误差分析**：

| 情况 | 对 m 的影响 | 对 N 估算的影响 |
|------|-------------|-----------------|
| 标记个体易被天敌发现 | m 偏小 | N 偏大（高估） |
| 标记脱落 | m 偏小 | N 偏大（高估） |
| 标记个体回避捕捉 | m 偏小 | N 偏大（高估） |
| 两次捕获间有大量死亡 | 减少种群基数 | N 偏大（高估） |
| 两次捕获间有大量迁入 | m 比例下降 | N 偏大（高估） |

### 典型例题

**例1（基础）**：捕获田鼠 40 只标记放回，一周后重捕 50 只，其中有标记个体 8 只，估算田鼠种群数量。

$$N = \frac{40 \times 50}{8} = 250 \text{（只）}$$

**例2（误差分析）**：若上题中部分标记田鼠因标记醒目被天敌捕食，则实际估算值与真实值相比____（偏大/偏小/不变）。

**答**：偏大。因为标记个体死亡导致重捕时 m 偏小，代入公式后 N 偏大。

**例3（应用）**：要使标志重捕法的估算更准确，应注意哪些事项？  
**答**：①标记方法不能伤害动物；②标记物不能过于醒目；③第二次捕获要在标记个体充分混入种群后进行；④调查期间种群数量相对稳定。

### 易错点

1. **标记方法**：不能影响动物正常生理和行为，常用耳标、翅标、涂色、剪趾等。
2. **两次捕获的时间间隔**：既要让标记个体充分扩散混合，又不能太长（避免种群数量大幅变化）。
3. **公式适用条件**：种群数量相对稳定，不适用于繁殖期或种群快速变化时期。
4. 计算 N 时，m 在分母位置，m 越小 → N 越大，这是高频考查的误差分析逻辑。
