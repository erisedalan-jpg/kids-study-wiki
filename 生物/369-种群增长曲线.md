# 种群增长曲线

> **一句话**：种群增长曲线描述种群数量随时间变化的规律，分为食物和空间无限时的J型曲线和有限环境中的S型曲线。
> **English**: Population growth curves describe how population size changes over time; the J-shaped curve occurs in unlimited environments, while the S-shaped curve occurs in limited environments.

---

## 🎓 高中进阶

### 定义

**种群增长曲线（population growth curve）**：以时间为横坐标、种群数量为纵坐标，描绘种群数量随时间变化趋势的曲线。主要有两种类型：J 型曲线（指数增长）和 S 型曲线（逻辑斯谛增长）。

### 概念 / 类型

**两种曲线对比**：

| 比较项 | J型曲线 | S型曲线 |
|--------|---------|---------|
| 前提条件 | 食物、空间无限；气候适宜；无天敌 | 食物、空间有限；存在天敌等 |
| 增长速率 | 持续增大（无上限） | 先增大后减小，K/2时最大 |
| 最终结果 | 理论上无限增长 | 趋近K值（环境容纳量） |
| 数学模型 | Nt = N₀λᵗ | 逻辑斯谛方程（有K值） |
| 实例 | 实验室细菌、外来入侵物种初期 | 自然界大多数种群 |

**关键指标**：

| 指标 | J型 | S型 |
|------|-----|-----|
| 增长率（λ）| 恒定，λ>1 | 先大后小，最终≈0 |
| 增长速率 | 持续增大 | N=K/2时最大，N=K时为0 |
| K值 | 无 | 有（环境容纳量） |

> **区分增长率与增长速率**：  
> - 增长率 = ΔN/N（百分比，J型恒定）  
> - 增长速率 = ΔN/Δt（绝对数量/时间，S型在K/2时最大）

### 典型例题

**例1**：如图S型增长曲线，当种群数量在K/2时，种群增长速率____（最大/最小），此时捕鱼最合适，理由是____。

**答**：最大；在K/2时种群增长速率最大，捕捞后种群能以最快速度恢复，实现可持续利用。

**例2**：某岛屿引入一批兔子，初期数量快速增长呈J型，后来增速减缓趋于稳定。分析增速减缓的原因。

**答**：随种群数量增加，岛屿内食物和空间减少，天敌数量增多，种内竞争加剧，死亡率升高，出生率降低，增长速率下降，最终种群数量趋近K值。

### 易错点

1. **增长率 ≠ 增长速率**：J型曲线增长率恒定，但增长速率持续增大；S型曲线增长率持续下降，增长速率在K/2时最大。
2. K/2处是增长速率最大点，不是增长率最大点（增长率在S型中一直减小）。
3. J型曲线是在**理想条件**下的理论模型，自然界几乎不能长期维持。
4. 种群数量不是到了K值就停止增长——K值是平均值，实际种群数量在K值上下波动。
