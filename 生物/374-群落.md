# 群落

> **一句话**：群落是指在同一时间内占据一定地域空间的多种生物种群的集合，是生态系统中有生命的部分。
> **English**: A community is an assemblage of different species populations occupying a defined area at the same time, representing the living component of an ecosystem.

---

## 🎓 高中进阶

### 定义

**群落（community / 生物群落，biotic community）**：在相同时间内、占据相同地域的**多种生物种群**所构成的集合体，包括该地域内所有的植物、动物和微生物。

> 种群 → 群落 → 生态系统：三者层次递进，群落是生态系统中的生物部分。

### 概念 / 类型

**群落的主要特征**：

| 特征 | 说明 |
|------|------|
| 物种多样性 | 群落由多种生物组成，物种越多，群落越复杂 |
| 空间结构 | 垂直结构（分层）+ 水平结构（斑块） |
| 种间关系 | 捕食、竞争、互利共生、寄生等 |
| 优势种 | 对群落影响最大的物种（通常生物量最大） |
| 演替 | 群落随时间变化，一种群落被另一种代替 |

**常见群落类型**：

| 群落类型 | 主要特征 | 分布地区 |
|----------|----------|----------|
| 热带雨林 | 物种极丰富，分层明显，终年高温多雨 | 赤道附近 |
| 温带落叶林 | 四季分明，物种较丰富 | 温带地区 |
| 草原 | 以草本植物为主，降水少 | 内陆干旱地区 |
| 荒漠 | 物种稀少，极度干旱 | 干旱区 |
| 淡水群落 | 包括湖泊、河流、湿地 | 各地水域 |

**群落与种群的区别**：

| | 种群 | 群落 |
|--|------|------|
| 组成 | 同种个体 | 多种种群 |
| 研究内容 | 数量、增长、密度 | 物种组成、空间结构、种间关系 |
| 层次 | 种内 | 种间 |

### 典型例题

**例1**：下列属于群落的是（ ）

A. 一片稻田中所有水稻  
B. 一个池塘中所有生物 ✓  
C. 一片草地中所有草本植物  
D. 某森林中所有乔木  

**解析**：群落包含同一区域内的**所有物种**，选B（含植物、动物、微生物等所有生物）。

**例2**：决定群落物种丰富度的最重要非生物因素是什么？

**答**：温度和降水量（水热条件）是决定群落物种丰富度的最重要非生物因素——一般规律是：从极地到热带，物种丰富度逐渐增加；从干旱到湿润，物种丰富度增加。

### 易错点

1. 群落包括**所有生物**（植物+动物+微生物），不只是动植物。
2. 群落是**同一地域**内所有种群的总和，不同地域的种群不构成同一群落。
3. **优势种**不一定是数量最多的物种，而是对群落结构影响最大的物种（通常是生物量最大或作用最关键的）。
4. 群落≠生态系统：群落只是生态系统的**生物部分**，生态系统还包括非生物的物质和能量。
