# 群落物种组成

> **一句话**：群落物种组成是指一个群落中所有物种的种类和数量关系，是区别不同群落的重要特征，用物种丰富度来衡量。
> **English**: Species composition refers to the types and relative abundances of all species in a community, characterized by species richness and used to distinguish different communities.

---

## 🎓 高中进阶

### 定义

**群落物种组成（species composition）**：构成一个群落的所有物种的种类（物种丰富度）及其相对数量（物种多度）关系，是识别不同群落、比较群落间差异的基本依据。

**物种丰富度（species richness）**：群落中物种数目的多少，是衡量物种多样性的最直接指标。

### 概念 / 类型

**影响群落物种丰富度的因素**：

| 因素 | 规律 |
|------|------|
| 纬度（温度） | 从极地→热带，丰富度增大（热带雨林>温带>寒带） |
| 降水量 | 降水越多，一般丰富度越大（在一定范围内） |
| 地形（海拔） | 随海拔升高，丰富度先增后减 |
| 人类活动 | 不合理开发→丰富度减少 |
| 时间（演替） | 演替越晚期，物种丰富度越高（但顶极群落趋于稳定） |

**优势种（dominant species）**：群落中数量多、生物量大、对群落结构影响最显著的物种。

**调查群落物种组成的常用方法**：

| 方法 | 适用类型 |
|------|----------|
| 样方法 | 植物物种调查 |
| 样线法（transect） | 植被带状调查 |
| 网捕/陷阱法 | 动物调查 |

**物种多度（abundance）**：某一物种在群落中的相对数量或生物量，是描述物种重要性的另一指标。

### 典型例题

**例1**：下列关于群落物种丰富度的说法，正确的是（ ）

A. 物种丰富度是指群落中生物个体数目的多少  
B. 同一地区不同群落的物种丰富度不同  ✓  
C. 物种丰富度越高的群落越不稳定  
D. 高纬度地区的群落物种丰富度高于低纬度地区  

**解析**：A错（物种丰富度是物种数，不是个体数）；B正确；C错（丰富度高通常稳定性更高）；D错（规律相反）。

**例2**：为什么热带雨林的物种丰富度远高于极地苔原？

**答**：热带雨林光照充足、温度高（全年≥20℃）、降水丰沛（年降水量2000 mm以上），能量来源充足，生境多样，支撑更多不同生态位的物种共存；而极地苔原温度极低、生长季短，只有少数耐寒物种能生存。

### 易错点

1. **物种丰富度 ≠ 物种多度**：丰富度是物种数目；多度是每种物种的个体数量。高丰富度群落中每种的个体数可能都不多。
2. **优势种不一定是数量最多的**，而是对群落结构影响最大、通常也是生物量最大的。
3. 从极地到赤道，物种丰富度**逐渐增大**，不是减小——这是考试常见干扰。
4. 人类合理开发可能增加物种多样性（如农业生态系统中引入新物种），但不合理干扰往往降低丰富度。
