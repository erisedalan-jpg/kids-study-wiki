# 碳循环
> **一句话**：碳元素以CO₂和有机物的形式，在大气、生物群落和岩石圈之间周而复始地循环流动；化石燃料的大量燃烧打破了自然平衡，导致温室效应加剧。
> **English**: Carbon cycles between the atmosphere, biotic community, and lithosphere as CO₂ and organic molecules; large-scale combustion of fossil fuels disrupts this balance, intensifying the greenhouse effect.

---

## 🎓 高中进阶
### 定义

**碳循环**（carbon cycle）是地球上最重要的生物地球化学循环之一，涉及碳在大气圈（CO₂）、水圈（HCO₃⁻等）、生物圈（有机物）和岩石圈（碳酸盐、化石燃料）之间的转化与传输。

### 概念 / 类型

**碳的存在形式**：

| 环境 | 碳的形式 |
|------|---------|
| 大气 | CO₂ |
| 水体 | CO₂（溶解）、HCO₃⁻、CO₃²⁻ |
| 生物体 | 有机碳（糖类、蛋白质、脂肪等） |
| 岩石圈 | 碳酸盐岩（石灰石）、化石燃料（煤、石油、天然气） |

**碳循环的主要途径**：

| 过程 | 方向 | 参与者 |
|------|------|--------|
| 光合作用 | 大气CO₂→有机物 | 生产者 |
| 呼吸作用 | 有机物→CO₂ | 所有生物 |
| 分解作用 | 有机物→CO₂ | 分解者 |
| 化石燃料燃烧 | 岩石圈有机碳→CO₂ | 人类活动 |
| 碳酸盐沉积 | CO₂→碳酸盐岩 | 地质过程 |
| 风化与溶解 | 碳酸盐→CO₂/HCO₃⁻ | 物理化学过程 |

**碳循环示意**：

```
大气CO₂
  ↑ 呼吸/燃烧/分解       ↓ 光合作用
动植物有机碳 ←→ 消费者 ←→ 生产者
  ↓ 死亡/遗体
分解者 → CO₂归还大气
  ↓（长期）
化石燃料（煤/石油）→ 燃烧 → CO₂（大量）
```

**温室效应与碳循环**：
- 自然状态下，碳循环维持大气CO₂浓度（约0.04%）相对稳定；
- 工业革命以来，化石燃料燃烧使CO₂排放量远超自然固定速率；
- 大气CO₂浓度升高→温室效应增强→全球气候变暖。

**减少CO₂的措施**：
1. 减少化石燃料使用（节能、可再生能源替代）；
2. 植树造林，增加碳汇；
3. 碳捕获与储存技术（CCS）；
4. 提高能源利用效率。

### 典型例题

**例**：在碳循环中，碳从无机环境进入生物群落的主要途径是？
- 答：**光合作用**（绿色植物固定CO₂为有机物）；化能合成作用（少量，如硝化细菌）也可以。

**例**：为什么说化石燃料燃烧是造成温室效应的主要原因？
- 化石燃料中的碳原本被封存于地下数亿年，不参与正常碳循环；
- 大量燃烧将其快速释放为CO₂，使大气CO₂浓度在短期内大幅上升，超过自然界的固碳能力，导致碳循环失衡。

### 易错点

1. **碳在生物群落内以有机物形式传递**，在生物与无机环境间以CO₂（或HCO₃⁻）形式交换。
2. **分解者不在食物链中，但参与碳循环**：分解者将有机碳转化为CO₂归还大气。
3. **光合作用≠唯一固碳途径**：化能自养生物（硝化细菌等）也能固定CO₂，但量远小于光合作用。
