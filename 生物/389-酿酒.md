# 酿酒

> **一句话**：利用酵母菌在无氧条件下将糖类分解为酒精和CO₂的发酵过程。
> **English**: The process of using yeast to anaerobically convert sugars into ethanol and carbon dioxide.

---

## 🎓 高中进阶

### 定义

酿酒是以富含糖分或淀粉的原料（葡萄、粮食等）为底物，利用**酵母菌**进行无氧发酵，将糖转化为**酒精（乙醇）**和**CO₂**的传统生物技术。

### 原理与步骤

**化学反应方程式**：
```
C₆H₁₂O₆ --酵母菌(无氧)--> 2C₂H₅OH + 2CO₂↑
葡萄糖                      酒精        二氧化碳
```

**实验室果酒制作流程**：

```
原料选择 → 冲洗 → 去梗榨汁 → 装瓶（≤2/3容积）→ 密封发酵（18~25℃，10~12天）→ 过滤 → 果酒
```

**关键条件**：

| 条件 | 要求 | 原因 |
|------|------|------|
| 气体 | 无氧（密封） | 酵母菌无氧呼吸产酒精 |
| 温度 | 18~25℃ | 酵母菌最适生长温度 |
| 消毒 | 75%酒精擦瓶口 | 防止杂菌污染 |
| 装料量 | ≤2/3 | 留空间给CO₂膨胀 |

**充气阀的作用**：排出CO₂，同时防止外界空气进入（维持无氧环境）。

**酒精检验**：酸性条件下，橙色重铬酸钾（K₂Cr₂O₇）与酒精反应，变为**灰绿色**。

### 典型例题

**例**：某同学制作果酒，充气阀排出气体后关闭，充气阀的主要作用是（ ）

A. 补充O₂，促进酵母菌有氧呼吸

B. 排出CO₂，防止气压过高

C. 排出CO₂并防止外界空气进入，维持无氧环境

D. 排出酒精蒸气

**答案**：C

**解析**：酿酒需要无氧环境，充气阀只在排出CO₂时短暂开启，目的是维持无氧环境并防止气压过高。

### 易错点

- **有氧时酵母菌产CO₂不产酒精**：O₂存在时，酵母菌进行有氧呼吸，酒精不能积累
- **温度过高会杀死酵母菌**：一般不超过35℃
- **检验酒精试剂**：重铬酸钾在**酸性**条件下，颜色变化为橙色→灰绿色
- **装料量**：不能过满，留出气体膨胀空间
