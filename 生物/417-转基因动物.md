# 转基因动物
> **一句话**：通过基因工程将外源目的基因导入动物受精卵，使外源基因整合进基因组并在动物个体中稳定表达的动物。
> **English**: An animal in which foreign DNA has been stably integrated into its genome via genetic engineering, allowing heritable expression of the transgene.

---

## 🎓 高中进阶
### 定义
转基因动物（Transgenic Animal）是指通过基因工程手段，将人工构建的含目的基因的重组DNA导入动物**受精卵（受精后的合子）**中，使外源基因整合进动物基因组，并在动物体内稳定表达，同时可遗传给后代的动物。

### 原理 / 步骤 / 应用

**培育流程**

```
目的基因的获取与重组DNA构建
↓
受精卵（供体动物，超数排卵→体外受精/自然受精）
↓ 显微注射：将重组DNA注射入受精卵原核
（目的基因随机整合进基因组）
↓ 体外培养至早期胚胎
↓ 胚胎移植至代孕母体
↓ 出生后代筛选（PCR / Southern杂交）
↓ 含目的基因的个体 → 建立转基因动物系
```

**常用导入方法**

| 方法 | 说明 |
|------|------|
| 显微注射法 | 直接将重组DNA注射入受精卵雄原核，整合率较高 |
| 逆转录病毒感染法 | 逆转录病毒携带目的基因，感染早期胚胎细胞 |
| 胚胎干细胞（ES细胞）法 | 将目的基因导入ES细胞，再用ES细胞构建嵌合体动物 |

**主要应用**

| 应用类型 | 举例 |
|---------|------|
| 乳腺生物反应器 | 转基因奶牛/羊在乳汁中生产人凝血因子、白蛋白、抗体等医药蛋白 |
| 疾病模型动物 | 转基因小鼠模拟人类遗传病（如阿尔茨海默症小鼠模型） |
| 异种器官移植 | 转基因猪（敲入人类基因，降低免疫排斥） |
| 提高农业动物性能 | 转基因鱼（导入生长激素基因，加速生长） |
| 基因功能研究 | 过表达/敲除特定基因，研究其功能 |

**乳腺生物反应器的原理**

- 将目的基因（如人凝血因子IX基因）与**乳腺特异性启动子**（如β-酪蛋白启动子）相连
- 目的基因只在乳腺细胞中高水平表达
- 药用蛋白质随乳汁分泌，便于大量采集和纯化
- 优点：产量大、成本低、蛋白质有翻译后修饰（糖基化等），活性好

**转基因动物与克隆动物的比较**

| 比较项 | 转基因动物 | 克隆动物（SCNT） |
|--------|----------|--------------|
| 遗传变化 | 导入新基因，遗传物质改变 | 核遗传物质不变（与供核动物相同） |
| 目的 | 获得新性状 | 保留原性状，扩大个体数量 |
| 技术核心 | 重组DNA + 基因导入 | 核移植 + 重编程 |

### 典型例题
**例**：为什么将目的基因导入动物受精卵，而不导入成体动物的普通体细胞？

**解析**：受精卵是全能细胞，分裂后形成个体全部细胞，导入受精卵的目的基因可整合进基因组后随细胞分裂遍及全身每个细胞，并可通过生殖细胞遗传给后代。若导入已分化的体细胞，目的基因只在该细胞中存在，无法传遍全身，也不能遗传给后代，无法建立稳定遗传的转基因动物系。

### 易错点
- 转基因动物的目的基因导入**受精卵**（而非普通体细胞），目的是让目的基因进入所有细胞并遗传给后代
- 外源基因在基因组中的整合位置**具有随机性**，同一批次不同个体整合位点不同
- 乳腺生物反应器中，目的基因并非在所有细胞中表达，而是通过**乳腺特异性启动子**只在乳腺细胞表达
