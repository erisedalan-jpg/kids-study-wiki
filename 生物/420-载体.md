# 载体
> **一句话**：基因工程中能携带目的基因进入受体细胞并使其稳定遗传和表达的DNA分子。
> **English**: A DNA molecule used in genetic engineering to carry a target gene into a host cell for stable replication and expression.

---

## 🎓 高中进阶
### 定义
载体（Vector）是基因工程中用来运载目的基因、将其导入受体细胞并使目的基因在受体细胞中稳定存在和表达的特定DNA分子，是基因工程的"运输工具"。

### 原理 / 步骤 / 应用

**载体必须具备的条件**

| 条件 | 说明 |
|------|------|
| 能在受体细胞中复制 | 含有复制起始位点（ori），可独立复制 |
| 含有限制酶切割位点 | 便于目的基因插入（多克隆位点） |
| 含有选择标记基因 | 筛选含重组DNA的受体细胞（如抗生素抗性基因） |
| 不影响受体细胞正常代谢 | 与宿主兼容，不致毒 |
| 含有启动子等表达调控元件 | 确保目的基因正常转录 |

**常用载体类型**

| 类型 | 特点 | 用途 |
|------|------|------|
| 质粒（plasmid） | 小环状DNA，可在大肠杆菌中自主复制 | 最常用，适合小片段基因克隆 |
| 噬菌体载体 | 病毒DNA改造，容量较大 | 基因文库构建 |
| 黏粒（cosmid） | 质粒+噬菌体元件，容量更大 | 大片段克隆 |
| 人工染色体（YAC/BAC） | 酵母/细菌人工染色体，容量最大 | 大基因组片段克隆 |
| 农杆菌Ti质粒 | 含T-DNA，可整合到植物基因组 | 植物基因工程 |
| 腺相关病毒（AAV）等 | 病毒载体，可感染特定细胞 | 基因治疗 |

**质粒的结构特征**（以pBR322为例）

- 环状双链DNA
- 含有复制起始位点（ori）
- 含有氨苄青霉素抗性基因（amp^r）和四环素抗性基因（tet^r）——选择标记
- 含有多个限制酶识别位点

**选择标记基因的作用**

用于筛选成功导入重组DNA的受体细胞：
- 若目的基因插入某抗性基因中，导致该基因失活（插入失活）
- 含重组质粒的细胞：对某种抗生素敏感（插入位点的抗性消失），但对另一种抗生素有抗性
- 未含质粒的细胞：两种抗生素均敏感

### 典型例题
**例**：某质粒有氨苄青霉素抗性基因（amp^r）和四环素抗性基因（tet^r），目的基因插入tet^r基因中。将转化后的大肠杆菌接种到含氨苄青霉素的培养基上，存活的菌落中哪些含重组质粒？

**解析**：含任何质粒（含或不含目的基因）的细菌均有amp^r，能在氨苄青霉素培养基上生长。再将存活菌落接种到含四环素的培养基上：能在四环素上生长的含**普通质粒**（tet^r完整）；**不能在四环素上生长**的含**重组质粒**（tet^r被目的基因插入失活）。

### 易错点
- 质粒是**天然存在于细菌中**的环状DNA，不是人工合成的，但基因工程中使用的质粒多经人工改造
- 载体本身不提供目的基因，只是**运输和表达平台**
- 不同受体细胞需要不同类型的载体（植物用农杆菌Ti质粒，动物用病毒载体等）

- A plasmid vector must contain a selectable marker for identifying transformed cells. （质粒载体必须含有用于识别转化细胞的选择标记。）
