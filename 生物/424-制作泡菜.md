# 制作泡菜

> **一句话**：利用乳酸菌在无氧条件下将糖类发酵为乳酸，使蔬菜腌制保存的传统技术。
> **English**: A traditional technique that uses lactic acid bacteria to ferment sugars into lactic acid under anaerobic conditions, preserving vegetables.

---

## 🎓 高中进阶

### 定义

制作泡菜是以新鲜蔬菜为原料，在**密封无氧**环境中，利用自然附着于蔬菜表面的**乳酸菌**进行无氧发酵，产生乳酸，使食品酸化保存的传统技术。

### 原理与步骤

**化学反应方程式**：
```
C₆H₁₂O₆ --乳酸菌(无氧)--> 2CH₃CHOHCOOH（乳酸）
葡萄糖                      （无CO₂产生）
```

**泡菜制作流程**：
```
原料处理（切块、加盐腌制）→ 装坛（压实）→ 加盐水（没过蔬菜）→ 密封坛口（坛沿注水）→ 常温腌制（7~10天）→ 成品
```

**关键细节**：

| 操作 | 作用 |
|------|------|
| 加食盐 | 抑制杂菌生长，同时不抑制乳酸菌 |
| 坛沿注水密封 | 造成无氧环境，让CO₂排出但空气不能进入 |
| 压实蔬菜 | 减少氧气残留 |
| 控制食盐浓度 | 过高抑制乳酸菌，过低杂菌大量繁殖 |

**亚硝酸盐变化规律**：
- 发酵初期：亚硝酸盐**升高**（杂菌产生）
- 发酵中期：亚硝酸盐**维持较高水平**
- 发酵后期：亚硝酸盐**下降**（乳酸积累，pH降低，杂菌被抑制，亚硝酸盐被分解）
- **泡菜腌制越久，亚硝酸盐含量越低，食用越安全**

**亚硝酸盐检测**：对氨基苯磺酸（显酸性，与亚硝酸盐反应）→ N-1-萘基乙二胺盐酸盐显色 → **玫瑰红色**，与标准管比较定量。

### 典型例题

**例**：关于泡菜制作的叙述，正确的是（ ）

A. 泡菜坛密封是为了给乳酸菌提供CO₂

B. 腌制时间越短，亚硝酸盐含量越低

C. 食盐的作用是抑制杂菌同时为乳酸菌提供无机盐

D. 泡菜中的乳酸菌来自蔬菜表面和空气中自然附着的菌

**答案**：D

**解析**：密封是造成无氧环境（A错）；腌制初期亚硝酸盐升高，后期下降（B错）；食盐主要抑制杂菌，不是为乳酸菌提供无机盐（C描述不准确）；乳酸菌确实来自蔬菜表面等自然环境（D正确）。

### 易错点

- **乳酸发酵不产生CO₂**：乳酸菌无氧呼吸的产物只有乳酸，不像酵母菌产酒精+CO₂
- **亚硝酸盐规律**：初期升高→后期下降，腌久了更安全
- **乳酸菌是原核生物**：无核膜，无线粒体
- **坛沿水封**：单向排气，不是双向通气

### 例句

- Lactic acid bacteria ferment sugars into lactic acid, lowering the pH and preserving the vegetables in kimchi.
- 乳酸菌将糖发酵为乳酸，降低pH，从而在泡菜中起到保存蔬菜的作用。
