# 制作腐乳

> **一句话**：以根霉等霉菌分泌的酶分解豆腐中的蛋白质和脂肪，形成独特风味的传统豆腐发酵食品。
> **English**: A traditional fermented tofu product made by mold (mainly Mucor) secreting enzymes that break down proteins and fats in tofu.

---

## 🎓 高中进阶

### 定义

腐乳（豆腐乳）是以豆腐为原料，利用**根霉**（Mucor，又称毛霉）等多种霉菌在豆腐表面生长，分泌**蛋白酶**和**脂肪酶**，将豆腐中的蛋白质分解为**小分子肽和氨基酸**，脂肪分解为**甘油和脂肪酸**，形成细腻质地和特殊风味的发酵食品。

### 原理与步骤

**主要微生物**：根霉（Mucor sp.）为主，还有酵母菌、曲霉等协同作用

**化学变化**：
```
蛋白质 --蛋白酶--> 多肽 → 氨基酸（鲜味来源）
脂肪   --脂肪酶--> 甘油 + 脂肪酸（香味来源）
```

**腐乳制作流程**：
```
豆腐切块 → 铺放（不重叠）→ 自然接种霉菌孢子 → 15~18℃培养5天（长出白毛）
→ 加盐腌制（按层加盐，瓶口盐最多）→ 加入卤汤（酒、香辛料等）→ 密封腌制6个月
```

**各操作的意义**：

| 操作 | 目的 |
|------|------|
| 豆腐不重叠摆放 | 利于霉菌在表面生长，保持通气 |
| 15~18℃培养 | 根霉最适生长温度，抑制其他杂菌 |
| 加盐（瓶口盐最多） | 防腐，同时从上到下盐浓度梯度合理 |
| 加酒 | 抑制杂菌，赋予风味，防腐 |
| 密封腌制 | 防止杂菌污染，使酶促反应充分进行 |

### 典型例题

**例**：腐乳制作中，加入食盐的主要目的是（ ）

A. 为根霉提供无机盐营养

B. 促进蛋白酶的活性

C. 抑制杂菌生长，防止腐乳变质

D. 加速豆腐中脂肪的分解

**答案**：C

**解析**：食盐的主要作用是抑制杂菌，起到防腐保鲜的作用，同时瓶口盐量最多也说明其防腐功能。

### 易错点

- **根霉是真核生物**（有核膜），与乳酸菌（原核）区分
- **根霉是需氧型**：豆腐摆放时应保证通气，但后期密封腌制时酶在无氧环境也可继续作用
- **加盐顺序**：从下到上每层加盐，**靠近瓶口的盐量最多**，防止最上层暴露空气受杂菌污染
- **腐乳风味**：不仅靠蛋白质降解，酒和香辛料也是重要风味来源
