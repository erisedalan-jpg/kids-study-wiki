# 微生物培养

> **一句话**：在人工控制的条件下，为微生物提供适宜的营养和环境，使其生长繁殖的技术体系。
> **English**: The set of techniques used to grow microorganisms under controlled conditions by providing appropriate nutrients and environment.

---

## 🎓 高中进阶

### 定义

微生物培养是指在实验室或工业生产中，按照微生物的营养需求和生长条件，配制**培养基**，经**灭菌**处理后，通过**接种**引入目标微生物，在适宜条件下使其生长繁殖的技术过程。

### 原理与关键技术

**微生物生长的基本需求**：

| 需求类型 | 具体内容 |
|---------|---------|
| 碳源 | 葡萄糖、淀粉、蔗糖等（提供能量和碳骨架） |
| 氮源 | 蛋白胨、铵盐、硝酸盐（合成蛋白质和核酸） |
| 无机盐 | K⁺、Mg²⁺、PO₄³⁻等（维持渗透压、辅酶成分） |
| 水 | 生化反应溶剂 |
| 特殊营养因子 | 维生素等（某些微生物自身不能合成） |

**微生物培养的操作流程**：
```
配制培养基 → 灭菌（高压蒸汽灭菌/干热灭菌）→ 冷却 → 无菌接种 → 恒温培养 → 观察结果
```

**纯培养技术**（获得单一菌种）：
- **稀释涂布平板法**：将样品梯度稀释后均匀涂布于固体培养基表面，培养后形成单菌落
- **平板划线法**：用接种环在固体培养基表面连续划线，每次划线末端的菌数减少，最终形成单菌落

**固体培养基 vs 液体培养基**：

| 类型 | 琼脂含量 | 用途 |
|------|---------|------|
| 固体培养基 | 1.5~2% | 菌落计数、纯化、保藏 |
| 半固体培养基 | 0.3~0.5% | 细菌运动性检测 |
| 液体培养基 | 0 | 大量培养、代谢产物生产 |

### 典型例题

**例**：关于微生物培养，下列叙述正确的是（ ）

A. 所有培养基均需加入有机碳源

B. 稀释涂布平板法和平板划线法均可用于获得单菌落

C. 培养基灭菌后需立即接种

D. 液体培养基适合用于菌落计数

**答案**：B

**解析**：自养型微生物可利用无机碳源（CO₂），A错；灭菌后需冷却至50℃左右才能接种，防止高温杀死接种菌，C错；菌落计数需固体培养基，D错；B正确，两种方法均可获得单菌落。

### 易错点

- **培养基灭菌后不能立即接种**：高温会杀死接种菌，需冷却至约50℃（固体培养基凝固前）
- **区分灭菌和消毒**：灭菌是杀灭所有微生物（含芽孢），消毒仅杀灭致病微生物
- **计数方法**：稀释涂布平板法计数时，实际菌数 ≥ 菌落数（多个菌可能形成一个菌落）
