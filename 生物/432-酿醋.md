# 酿醋

> **一句话**：醋酸菌在有氧条件下将酒精氧化为醋酸的发酵过程，是果醋制作的核心步骤。
> **English**: The process by which acetic acid bacteria oxidize ethanol into acetic acid under aerobic conditions to produce vinegar.

---

## 🎓 高中进阶

### 定义

酿醋是在果酒（酒精）的基础上，利用**醋酸菌**（需氧型细菌）在**有氧**条件下将酒精氧化为**醋酸（乙酸）**的发酵过程。

### 原理与步骤

**化学反应方程式**：
```
C₂H₅OH + O₂ --醋酸菌--> CH₃COOH + H₂O
酒精（乙醇）              醋酸（乙酸）
```

**果醋制作流程（接果酒之后）**：
```
果酒 → 充气（通入O₂）→ 接种醋酸菌 → 30~35℃发酵 7~8天 → 过滤 → 果醋
```

**关键条件对比（酿酒 vs 酿醋）**：

| 项目 | 酿酒 | 酿醋 |
|------|------|------|
| 微生物 | 酵母菌 | 醋酸菌 |
| 氧气 | 无氧（密封） | 有氧（通气） |
| 温度 | 18~25℃ | 30~35℃ |
| 底物 | 葡萄糖 | 酒精（乙醇） |
| 产物 | 酒精 + CO₂ | 醋酸 + H₂O |

**醋酸检验**：用 pH 试纸或石蕊试纸检测酸性，产品有醋味。

**醋酸菌特性**：
- 原核生物（无核膜）
- 严格需氧，缺氧时停止生长
- 最适温度 30~35℃
- 可耐较高酒精浓度

### 典型例题

**例**：果醋制作时，与果酒制作相比，需要改变的操作是（ ）

A. 提高发酵温度，通入O₂

B. 降低发酵温度，隔绝O₂

C. 保持相同温度，通入CO₂

D. 提高温度，隔绝O₂

**答案**：A

**解析**：果醋由醋酸菌发酵，需要有氧环境（通入O₂）和较高温度（30~35℃ > 酿酒18~25℃）。

### 易错点

- **醋酸菌是需氧菌**：缺氧则发酵终止，与酿酒的无氧条件完全相反
- **两步法顺序不能颠倒**：先酿酒（无氧）再酿醋（有氧），不能同时进行
- **温度区别**：酿醋温度（30~35℃）高于酿酒（18~25℃）
- **醋酸菌是原核生物**，无线粒体，氧化酒精在细胞膜（呼吸链）上进行
