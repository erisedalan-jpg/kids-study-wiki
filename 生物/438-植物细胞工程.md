# 植物细胞工程

> **一句话**：以植物细胞为操作对象，利用细胞的全能性，通过组织培养或体细胞杂交等技术培育新植株的工程技术。
> **English**: Engineering techniques that manipulate plant cells using their totipotency to regenerate new plants through tissue culture or somatic cell hybridization.

---

## 🎓 高中进阶

### 定义

植物细胞工程是以细胞为基本操作单位，在**细胞水平**上，利用植物细胞的**全能性**，通过人工技术手段（组织培养、体细胞杂交等）改变细胞遗传性状或大量繁殖植株的工程技术体系。

### 理论基础：细胞的全能性

**全能性定义**：每个细胞都含有该生物个体全套遗传信息（基因组），在适宜条件下具有发育成完整个体的潜能。

**全能性大小比较**：
```
受精卵 > 生殖细胞 > 体细胞
```

**为什么体细胞能实现全能性**：通过脱分化→再分化，基因选择性表达，逐步恢复分化能力

### 植物细胞工程的主要技术

**1. 植物组织培养**（见[[437-植物组织培养|植物组织培养]]词条）
```
外植体（含活细胞的器官/组织片段）→ 脱分化 → 愈伤组织 → 再分化 → 胚状体/丛芽 → 生根 → 移栽
```

**2. 植物体细胞杂交**（见[[436-植物体细胞杂交|植物体细胞杂交]]词条）
```
去除细胞壁（酶解）→ 原生质体 → 化学法/电激法融合 → 杂交原生质体 → 组织培养 → 杂交植株
```

### 应用领域

| 技术 | 应用 |
|------|------|
| 组织培养 | 快速繁殖名贵植物、脱毒苗、人工种子 |
| 体细胞杂交 | 远缘杂交育种（克服种间不亲和性） |
| 花药离体培养 | 单倍体育种（快速纯合） |

### 典型例题

**例**：植物细胞工程的理论依据是（ ）

A. 基因的分离定律

B. 细胞的全能性

C. 细胞增殖能力

D. 基因工程原理

**答案**：B

**解析**：植物组织培养、体细胞杂交等植物细胞工程技术的理论基础是细胞的全能性——每个体细胞都含有该生物的全套遗传信息，在适宜条件下可发育成完整个体。

### 易错点

- **细胞全能性 ≠ 细胞都能实现全能性**：高度分化的细胞（如神经细胞）虽含全套基因，但实际上难以实现全能性
- **植物细胞工程不涉及基因的改变**（体细胞杂交除外）：组织培养是无性繁殖，遗传物质不变
