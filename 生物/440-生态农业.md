# 生态农业

> **一句话**：运用生态学原理和系统科学方法，将传统农业与现代科技结合，在保护生态环境的同时实现农业可持续发展的综合农业模式。
> **English**: Ecological agriculture integrates ecological principles and modern technology to achieve sustainable agricultural production while protecting the environment.

---

## 🎓 高中进阶

### 定义

**生态农业**：按照生态学和经济学原理，运用现代科学技术成果与现代管理手段，建立的多层次、多功能、多效益、综合性的可持续农业生产体系。强调物质多级循环利用、能量高效流动、生态环境保护。

### 基本特征

| 特征 | 说明 |
|------|------|
| 综合性 | 农、林、牧、副、渔结合 |
| 多样性 | 多种作物、动物、微生物搭配 |
| 高效性 | 能量利用率高、产出大 |
| 持续性 | 长期保持生产力 |
| 区域性 | 因地制宜 |

### 经典模式

#### 1. 桑基鱼塘（珠三角）

```
       桑树
        ↓
       蚕食桑叶
        ↓
       蚕粪入塘
        ↓
       鱼食蚕粪
        ↓
       塘泥肥桑
```

实现"桑-蚕-鱼-泥"零废物循环。

#### 2. 农林牧综合（北京留民营）

```
作物 → 人/畜食用
作物秸秆 → 牲畜饲料
牲畜粪便 → 沼气池
沼气 → 能源
沼渣 → 肥料
```

实现"作物-牲畜-沼气-肥料"循环。

#### 3. "稻-鱼"模式

水稻田中养鱼或鸭——鱼鸭吃虫、肥水，同时除草。

#### 4. "猪-沼-果"模式

猪粪入沼气池→沼气供能源→沼渣沼液肥果园。

### 关键技术

| 技术 | 说明 |
|------|------|
| 物种合理搭配 | 互利、互补避免恶性竞争 |
| 立体种养 | 充分利用空间（农林复合、池塘多层） |
| 物质循环 | 秸秆还田、堆肥、沼气池 |
| 生物防治 | 利用天敌或生物农药控制病虫害 |
| 节水灌溉 | 滴灌、喷灌 |
| 有机肥施用 | 减少化肥 |
| 减少农药 | 综合防治 |

### 优势

| 维度 | 优势 |
|------|------|
| 生态 | 减少污染、保护土壤、生物多样性 |
| 经济 | 多种产出、附加值高、降低成本 |
| 社会 | 提供就业、改善农村生活、保障粮食安全 |
| 健康 | 食品安全更高 |

### 与传统农业、化工农业对比

| 项目 | 传统农业 | 化工农业 | 生态农业 |
|------|------|------|------|
| 主要驱动 | 经验 | 化肥农药 | 生态学 + 现代技术 |
| 产出 | 低 | 高（短期） | 中等-高（长期） |
| 环境 | 较温和 | 污染严重 | 友好 |
| 可持续性 | 一般 | 弱 | 强 |
| 食品安全 | 较好 | 隐患 | 好 |

### 我国生态农业发展

- 始于 1980 年代
- 试点县建设、示范区推广
- 与精准农业、智慧农业结合
- 有机农产品认证标准化

### 挑战

- 投入产出比不易短期显现
- 农民转型意愿与能力
- 市场监管与认证
- 技术普及与培训
- 政策支持与补贴

### 典型例题

**例**：桑基鱼塘的优势在于？
**答**：物质多级循环利用、零废物、提高资源利用效率、保护环境、增加经济效益。

**例**：生态农业符合哪些生态工程原理？
**答**：循环、协调、自生、整体、多样性等所有五大原理。

### 易错点

- 生态农业 ≠ 不施肥不打药；而是科学合理施用
- 生态农业不是回到原始农业；结合现代技术
- 生态农业产量并非一定低于化工农业（设计良好可媲美）
- 有机农业是生态农业的一种形式，但更严格
