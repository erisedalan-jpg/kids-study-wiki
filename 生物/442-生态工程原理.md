# 生态工程原理

> **一句话**：生态工程设计与实施所遵循的核心原则，包括物质循环、自生、协调、整体等原理。
> **English**: The principles of ecological engineering include matter recycling, self-organization, coordination, and the holistic principle.

---

## 🎓 高中进阶

### 五大基本原理（人教版）

#### 1. 自生原理（self-organization）

生态系统中生物组分通过自组织、自适应、自我维持，依靠自身能力维护稳定性与活力。

**应用**：种植本地适应性强的植物；引入互补物种；保留多样性。

#### 2. 循环原理（matter cycling）

生态系统中物质（碳、氮、磷等）应在系统内多级利用、循环再生，使废物变为资源，减少外部输入与污染。

**应用**：桑基鱼塘（桑→蚕→蚕沙→鱼→塘泥→桑）；秸秆还田；沼气池综合利用。

#### 3. 协调原理（coordination）

工程设计应使生物与环境协调（适应当地气候、土壤、地形）；不同生物之间协调（避免恶性竞争）；人与自然协调。

**应用**：选适地适树（南方种水稻、北方种小麦）；混交林优于纯林。

#### 4. 整体原理（holistic principle）

生态工程要从整体角度考虑，统筹自然、经济、社会三大子系统；强调时空尺度上的统筹。

**应用**：流域综合治理（不只是河道治理，还要上游植被、下游产业、人口安置一起考虑）。

#### 5. 物种多样性原理（diversity principle）

生物多样性提高生态系统抵抗力稳定性。多样性高的生态系统更能抵抗扰动、自我调节。

**应用**：种植多种作物轮作；混交林；生态廊道连通。

### 各原理的整合

```
自生 → 系统能稳定运行
循环 → 资源高效利用
协调 → 各组分相互适应
整体 → 系统外延整合
多样性 → 提供稳定性基础
```

### 典型案例对照

| 案例 | 体现原理 |
|------|--------|
| 桑基鱼塘 | 循环、协调、多样性 |
| 退耕还林 | 自生、协调、整体 |
| 三北防护林 | 整体、协调、多样性 |
| 生态农业 | 全部五项 |
| 城市湿地公园 | 自生、多样性、协调 |

### 违背原理的反面教训

| 案例 | 错误 |
|------|------|
| 单一树种大面积造林 | 违背多样性，易爆发病虫害（如美国白蛾肆虐杨树纯林） |
| 引入外来种 | 违背协调，福寿螺、水葫芦等入侵 |
| 围湖造田 | 违背整体，洪水泛滥、湿地丧失 |
| 过度施肥 | 违背循环，水体富营养化 |

```
生态工程原理
    ↓
设计实施
    ↓
经济 + 生态 + 社会效益
    ↓
可持续发展
```

### 典型例题

**例**：桑基鱼塘体现了哪些生态工程原理？
**答**：循环原理（物质多级利用）、协调原理（桑蚕鱼相互适应）、多样性原理（多种生物组合）。

**例**：单一作物大面积种植违背什么原理？
**答**：物种多样性原理；易爆发病虫害，难以稳定。

### 易错点

- 五大原理是相互关联的整体，不是孤立的
- 不同情境下原理侧重不同
- 生态工程原理与生态学原理紧密相关，但更注重应用
- 违反任一原理都可能导致工程失败
