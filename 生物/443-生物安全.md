# 生物安全

> **一句话**：保护人、动植物和生态环境免受生物因素威胁的综合体系，包括传染病防控、生物技术安全、外来物种入侵防控等。
> **English**: Biosafety is the comprehensive system protecting humans, animals, plants, and ecosystems from biological threats, including pathogens, biotechnology hazards, and invasive species.

---

## 🎓 高中进阶

### 定义

**生物安全**：国家有效防范和应对危险生物因子及相关因素威胁，维护人民生命健康和生态系统稳定的能力。涉及微生物（细菌、病毒等）、生物技术、生物多样性、外来物种、生物恐怖等多个领域。

### 主要内容

#### 1. 传染病防控

| 类别 | 例子 |
|------|------|
| 新发传染病 | COVID-19、SARS、埃博拉、MERS |
| 重大传染病 | 艾滋病、肺结核、乙肝 |
| 流行病学监测 | 早期预警、追踪、隔离 |
| 疫苗与药物研发 | 提高免疫水平 |

#### 2. 生物技术安全

| 议题 | 内容 |
|------|------|
| 转基因生物 | 评估对环境、健康的影响 |
| 基因编辑 | 监管 CRISPR 等技术应用 |
| 基因驱动 | 防止野外失控扩散 |
| 病原体研究 | BSL-4 实验室管理 |

#### 3. 实验室生物安全等级 (BSL)

| 等级 | 适用 | 例子病原体 |
|------|------|--------|
| BSL-1 | 无致病性 | 大肠杆菌 K-12 |
| BSL-2 | 中等危险 | 流感病毒、HIV、SARS-CoV-2（培养） |
| BSL-3 | 高危险 | 结核杆菌、SARS-CoV、H5N1 |
| BSL-4 | 极高危险 | 埃博拉、马尔堡病毒、天花病毒 |

中国 BSL-4 实验室：武汉国家生物安全实验室、哈尔滨等。

#### 4. 外来物种入侵

| 例子 | 危害 |
|------|------|
| 福寿螺 | 危害水稻、传播疾病 |
| 水葫芦 | 堵塞水道、破坏生态 |
| 紫茎泽兰 | 占领草场，毒性 |
| 加拿大一枝黄花 | 排挤本土植物 |
| 红火蚁 | 螫人致敏、破坏农田 |

#### 5. 生物多样性保护

防止物种灭绝、保护遗传资源、生态系统。

#### 6. 病原体微生物管理

| 措施 | 内容 |
|------|------|
| 病原微生物分级管理 | 第一类、第二类高致病性、第三类、第四类 |
| 实验研究、运输、储存有严格审批 | 防止泄漏、滥用 |
| 标本的销毁、运输需备案 | 严格记录 |

#### 7. 防生物恐怖

- 防止生物武器开发与使用
- 国际禁止生物武器（《禁止生物武器公约》1972）
- 加强应急响应能力

### 中国《生物安全法》（2020 颁布，2021 实施）

主要内容：
- 防控重大新发突发传染病、动植物疫情
- 研究、开发、应用生物技术安全
- 实验室生物安全
- 人类遗传资源与生物资源安全
- 防御外来物种入侵和保护生物多样性
- 应对微生物耐药
- 防范生物恐怖与生物武器

### 公众层面的生物安全

| 行动 | 说明 |
|------|------|
| 接种疫苗 | 群体免疫 |
| 个人防护 | 口罩、勤洗手、咳嗽礼仪 |
| 不食野味 | 减少病原体跨种传播 |
| 不私自携带种子/动物 | 防外来种入侵 |
| 不滥用抗生素 | 防耐药性 |
| 配合疫情防控 | 信息上报、隔离 |

### 与生物技术伦理的关系

- 生物安全侧重**安全风险**与**应对能力**
- 生物伦理侧重**道德规范**与**社会价值**
- 二者相辅相成，共同保障生物科技健康发展

### 典型例题

**例**：实验室处理 SARS-CoV-2 病毒时应使用什么级别实验室？
**答**：BSL-3 或更高（活病毒培养通常 BSL-3）。

**例**：举两个外来物种入侵中国的例子。
**答**：水葫芦、福寿螺（或紫茎泽兰、红火蚁等）。

### 易错点

- 生物安全是国家安全的重要组成部分
- 实验室生物安全等级根据病原体危险性确定
- 生物安全不只是科研问题，更是公共卫生与国家安全问题
- 个人行为（不带境外动植物）直接关系生物安全
