# 干细胞

> **一句话**：具有自我更新能力和分化潜能的未分化或低分化细胞，可以分化为多种特化细胞类型。
> **English**: Stem cells are undifferentiated or partially differentiated cells with self-renewal capacity and the potential to differentiate into specialized cell types.

---

## 🎓 高中进阶

### 定义

**干细胞**：具有自我更新能力（可不断分裂产生与自身相同的细胞）和分化潜能（能分化形成多种特化细胞）的细胞。是组织修复、再生医学的核心。

### 干细胞的两大基本特性

1. **自我更新**：通过对称分裂或不对称分裂保持干细胞库
2. **分化潜能**：可分化为该物种部分或全部细胞类型

### 分类（按分化潜能）

| 类型 | 分化潜能 | 例子 |
|------|--------|------|
| 全能干细胞 | 形成完整个体 + 胚外组织 | 受精卵、早期卵裂球（≤8 细胞期） |
| 多能干细胞 | 三胚层所有细胞 | 胚胎干细胞 (ESC)、诱导多能干细胞 (iPSC) |
| 多能（专能） | 多种但范围有限 | 造血干细胞、神经干细胞 |
| 单能干细胞 | 只一种 | 表皮干细胞 |

### 分类（按来源）

| 来源 | 特点 |
|------|------|
| 胚胎干细胞 (ESC) | 来自囊胚内细胞团；多能；伦理争议 |
| 成体干细胞 | 各组织中（造血、神经、肠上皮、间充质等）；专能；伦理低 |
| 诱导多能干细胞 (iPSC) | 体细胞重编程获得；多能；山中伸弥 2006 提出 |
| 围产期来源 | 脐带血干细胞、羊膜干细胞 |

### 应用

| 应用 | 内容 |
|------|------|
| 造血疾病治疗 | 骨髓移植、脐带血移植（白血病、再生障碍贫血） |
| 组织修复 | 软骨、皮肤、心肌再生 |
| 神经退行性疾病 | 帕金森病、脊髓损伤研究 |
| 糖尿病 | 胰岛 β 细胞再生研究 |
| 角膜再生 | 角膜缘干细胞移植 |
| 药物筛选 | iPSC 模型 |
| 疾病模型 | 个性化研究 |

### iPSC 的发现意义

- 山中伸弥团队（2006）：将四个转录因子（Oct4、Sox2、Klf4、c-Myc）导入小鼠成纤维细胞 → 重编程为多能干细胞
- 突破：分化的成体细胞**可逆向**为多能态
- 避免胚胎伦理问题
- 2012 年获诺贝尔生理学或医学奖

### 优势与挑战

| 优势 | 挑战 |
|------|------|
| 替代受损组织 | 分化方向控制 |
| 个体化治疗 | 致瘤性（多能干细胞易成瘤） |
| 模型建立 | 长期安全性 |
| 药物筛选 | 伦理问题（ESC） |

### 典型例题

**例**：胚胎干细胞和 iPSC 的来源差异？
**答**：胚胎干细胞来自囊胚内细胞团（涉及胚胎使用伦理）；iPSC 来自普通体细胞重编程，无胚胎伦理问题。

**例**：白血病患者骨髓移植利用了什么？
**答**：造血干细胞分化潜能 → 重建患者造血与免疫系统。

### 易错点

- 干细胞 ≠ 全部能成完整个体，多数为部分潜能
- 全能性最强的"干细胞"是受精卵
- 干细胞研究的医疗应用大部分仍在临床试验阶段
- 干细胞治疗"包治百病"是错误宣传
