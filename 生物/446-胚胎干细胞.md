# 胚胎干细胞

> **一句话**：来源于早期胚胎（囊胚内细胞团）的干细胞，具有多能性（可分化为三胚层各种细胞）。
> **English**: Embryonic stem cells (ESCs) are pluripotent stem cells derived from the inner cell mass of a blastocyst, capable of forming all three germ layers.

---

## 🎓 高中进阶

### 定义

**胚胎干细胞 (ESC)**：从早期胚胎（囊胚阶段）的内细胞团中分离的多能干细胞，具有自我更新能力和发育成三胚层（外、中、内胚层）所有体细胞类型的潜能。

### 来源

```
受精卵
  ↓ 卵裂
2 细胞 → 4 细胞 → 8 细胞（每个仍具全能性）
  ↓
桑椹胚（16-32 细胞）
  ↓
囊胚（约 100 细胞）
  ├─ 滋养层（形成胎盘）
  └─ 内细胞团 ← 胚胎干细胞来源
```

### 特点

| 特点 | 说明 |
|------|------|
| 多能性 | 可分化为内、中、外三胚层细胞（理论上几乎所有体细胞） |
| 自我更新 | 体外培养可长期保持未分化、不断扩增 |
| 形态 | 体积小、核质比大、核仁明显 |
| 标志分子 | Oct4、Sox2、Nanog 等多能性转录因子 |
| 染色体 | 二倍体（不发生变异）|

### 与其他干细胞比较

| 类型 | 来源 | 分化潜能 | 伦理问题 |
|------|------|--------|--------|
| 全能干细胞（受精卵） | 受精卵 | 完整个体 + 胚外 | — |
| **胚胎干细胞** | 囊胚内细胞团 | 三胚层所有细胞 | 涉及胚胎使用 |
| 成体干细胞 | 各组织 | 部分细胞 | 低 |
| iPSC | 体细胞重编程 | 接近 ESC | 低 |

### 应用前景

| 领域 | 应用 |
|------|------|
| 再生医学 | 修复神经、心肌、胰岛、视网膜等 |
| 疾病模型 | 体外研究遗传病发病机制 |
| 药物筛选 | 多种细胞类型的药效与毒性测试 |
| 发育生物学研究 | 阐明胚胎发育机制 |

### 研究中的挑战

| 挑战 | 内容 |
|------|------|
| 伦理 | 取材涉及破坏胚胎，引发争议 |
| 致瘤性 | ESC 在体内可形成畸胎瘤 |
| 免疫排斥 | 来源胚胎与受体不同 → 免疫识别 |
| 分化方向控制 | 难以精确诱导单一细胞类型 |
| 临床转化 | 严格的安全和伦理审批 |

### 替代方案

为避免伦理争议，研究者发展：
- **iPSC**：成体细胞重编程，无需胚胎
- **体细胞核移植**：避免常规胚胎来源
- **成体干细胞**：来自骨髓、脐带血等

### 中国相关法规

中国《人胚胎干细胞研究伦理指导原则》（2003）等法规：
- 禁止生殖性克隆（人）
- 治疗性克隆研究有严格审批
- 胚胎培养不超过 14 天
- 不得将研究胚胎植入子宫

### 典型例题

**例**：ESC 与 iPSC 的主要差异？
**答**：来源不同（ESC 来自囊胚内细胞团；iPSC 来自体细胞重编程）；伦理风险不同（ESC 涉及胚胎；iPSC 避免）；分化潜能相近。

**例**：为什么从囊胚而非桑椹胚分离 ESC？
**答**：桑椹胚每个卵裂球仍具全能性；囊胚已分化出滋养层与内细胞团，内细胞团是研究多能性的最佳来源。

### 易错点

- ESC 是**多能性**而不是全能性（不能形成胚外组织）
- ESC 来源于内细胞团，不是滋养层
- 14 天规则是国际通行的伦理底线
- ESC 与 iPSC 功能相近，但来源不同决定伦理风险不同
