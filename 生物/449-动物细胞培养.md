# 动物细胞培养
> **一句话**：在体外模拟体内生理环境，使动物细胞在适宜条件下生长和增殖的技术。
> **English**: A technique that maintains and propagates animal cells outside the organism under controlled in vitro conditions.

---

## 🎓 高中进阶
### 定义
动物细胞培养是指将从动物体内取出的细胞，在体外模拟机体内环境（适宜温度、pH、无菌等），用含有营养物质的培养液培养，使细胞继续生长和增殖的技术。

### 原理 / 步骤 / 应用

**基本原理**

动物细胞能在体外培养液中获取营养、进行代谢，保持增殖能力，这是细胞工程的基础。

**培养液组成**

| 成分 | 作用 |
|------|------|
| 葡萄糖、氨基酸、无机盐 | 基本营养物质 |
| 动物血清（血浆） | 提供生长因子、激素等 |
| CO₂ / NaHCO₃ | 维持 pH（7.2–7.4） |
| 抗生素 | 防止杂菌污染 |

**培养条件**

- 温度：37 °C（接近体温）
- pH：7.2–7.4
- 气体：95% 空气 + 5% CO₂
- 无菌操作

**操作步骤**

1. 取材与消化：从动物组织中取细胞，用胰蛋白酶处理使细胞分散
2. 原代培养：细胞首次在体外培养，贴壁生长，出现接触抑制
3. 传代培养：细胞长满瓶壁后，再用胰蛋白酶消化，稀释后转入新瓶继续培养
4. 细胞系建立：经多次传代，筛选出能无限增殖的细胞系（多来自癌细胞或特殊处理）

**接触抑制**：正常细胞铺满培养瓶底后停止增殖的现象；癌细胞无接触抑制，可持续增殖。

**应用**

- 生产病毒性疫苗（如流感疫苗、脊髓灰质炎疫苗）
- 制备单克隆抗体
- 基因工程中的受体细胞培养
- 组织工程、药物毒性检测

### 典型例题
**例**：动物细胞培养时，用胰蛋白酶处理组织的目的是什么？传代培养时为什么也要使用胰蛋白酶？

**解析**：胰蛋白酶水解细胞间的蛋白质（胶原蛋白等），使组织中的细胞分散成单个细胞，便于培养。传代时细胞贴壁，胰蛋白酶使贴壁细胞脱落分散，实现转瓶继代。

### 易错点
- 动物细胞培养液必须加**血清**，不能用一般植物培养基替代
- **原代培养≠传代培养**：原代是首次体外培养，传代是将原代细胞分瓶再培养
- 接触抑制是**正常细胞**的特征；癌细胞无接触抑制，这是体外建立永生化细胞系的重要特性
