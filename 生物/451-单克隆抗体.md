# 单克隆抗体
> **一句话**：由单一B细胞克隆产生的、针对某一特定抗原决定簇的高度特异性抗体。
> **English**: Antibodies produced by a single clone of B cells, targeting a specific antigenic epitope with high specificity and homogeneity.

---

## 🎓 高中进阶
### 定义
单克隆抗体（Monoclonal Antibody, mAb）是指由单一杂交瘤细胞增殖形成的细胞克隆所产生的抗体，其结构完全相同，只识别同一抗原决定簇（表位），具有**高特异性、高灵敏度、可大量制备**等优点。

相比之下，普通血清中的**多克隆抗体**来自多种B细胞，可识别同一抗原的多个表位，特异性较低。

### 原理 / 步骤 / 应用

**制备原理**

将能产生特异性抗体的**效应B细胞（浆细胞）**与能无限增殖的**骨髓瘤细胞**在体外融合，筛选出既能无限增殖、又能分泌特异性抗体的**杂交瘤细胞**，再经克隆化培养即可大量获得单克隆抗体。

**制备流程**

```
向小鼠注射特定抗原
↓ 免疫（2~3周）
从小鼠脾脏取出效应B细胞
↓ 与骨髓瘤细胞混合 + PEG诱导融合
细胞融合（多种融合产物混合）
↓ HAT培养基筛选（只有杂交瘤细胞存活）
杂交瘤细胞
↓ 克隆化培养 + 抗体检测
阳性杂交瘤克隆
↓ 体外培养 / 小鼠腹腔注射扩增
大量单克隆抗体
```

**HAT培养基筛选原理**

| 细胞类型 | 在HAT培养基中的结果 |
|---------|-------------------|
| 未融合的骨髓瘤细胞 | 死亡（缺乏HGPRT酶，不能利用旁路合成DNA） |
| 未融合的效应B细胞 | 自然死亡（寿命有限） |
| 同种融合体 | 同骨髓瘤细胞或同B细胞结果 |
| **杂交瘤细胞** | **存活**（获得B细胞的HGPRT，可利用旁路；又有骨髓瘤细胞的无限增殖能力） |

**主要特点**

| 特点 | 说明 |
|------|------|
| 高特异性 | 只识别单一抗原决定簇 |
| 高灵敏度 | 微量抗原即可检测 |
| 均一性 | 同批次抗体结构完全相同 |
| 可大量制备 | 杂交瘤细胞无限增殖，持续产生抗体 |

**应用**

- **疾病诊断**：妊娠试纸（检测HCG）、新冠抗原检测、肿瘤标志物检测
- **疾病治疗**：抗肿瘤单抗（如曲妥珠单抗、利妥昔单抗）、自身免疫病治疗
- **科学研究**：细胞分型、蛋白质定位、Western blot、免疫荧光等
- **靶向药物**：抗体-药物偶联物（ADC）

### 典型例题
**例**：单克隆抗体制备过程中，对杂交瘤细胞进行克隆化培养和抗体检测的目的是什么？

**解析**：杂交瘤细胞经融合后，每个细胞对应的抗体特异性可能不同（效应B细胞来自多种淋巴细胞）。克隆化培养使每个杂交瘤细胞单独增殖为一个细胞克隆，再通过抗体检测筛选出能产生**目标特异性抗体**的克隆，即得到真正意义上的"单克隆"。

### 易错点
- 单克隆抗体不是从B细胞直接分泌，而是来自**杂交瘤细胞**
- 杂交瘤细胞需经**两次筛选**：第一次用HAT培养基筛选存活的杂交瘤细胞；第二次通过抗体检测筛选产生目标抗体的克隆
- 注射抗原是为了使小鼠产生**免疫应答**，从脾脏（富含免疫细胞）取效应B细胞，而非直接取血液
