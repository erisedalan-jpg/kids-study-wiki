# 基因文库

> **一句话**：基因文库是指将某种生物的全部或部分DNA片段通过载体导入受体菌等宿主细胞中储存而形成的一个克隆群体，每个克隆含有一段不同的外源DNA片段。
> **English**: A gene library is a collection of cloned DNA fragments that together represent the entire genome (genomic library) or the expressed genes (cDNA library) of an organism.

## 🎓 考点精讲

**定义/原理**：基因文库是将供体生物的DNA（基因组DNA或由mRNA反转录得到的cDNA）用适当方法切割成许多片段，分别与载体（如质粒、噬菌体）连接，然后导入受体菌（如大肠杆菌）中扩增储存。这样每个受体菌内就含有一段特定的外源DNA片段，整个菌落群体便构成了包含该生物各种DNA片段的基因文库。根据DNA来源，基因文库主要分为两类：

1. **基因组文库**：以生物体全部基因组DNA为材料，用限制酶进行部分消化，使每个DNA片段都含有一定的基因信息。它包含该生物的编码区、非编码区、内含子和基因间序列，理论上覆盖了完整的遗传信息。
2. **部分基因文库（如cDNA文库）**：以特定组织或发育时期提取的mRNA为模板，在反转录酶作用下合成互补DNA（cDNA），再与载体连接后导入受体菌。cDNA文库只含有基因的编码区（外显子），不含启动子、内含子等调控序列，且具有组织、时期特异性。

**关键方法**：构建基因文库的标准步骤：提取总DNA（或mRNA）→ 用限制酶消化（或反转录获得cDNA）→ 将DNA片段与载体连接 → 将重组载体导入受体细胞 → 筛选含有重组载体的菌落并扩增，获得基因文库。从文库中获取目的基因时，可根据已知序列合成DNA探针进行核酸分子杂交筛选，或依据目的基因的表达产物特性（如酶活性、与某种抗体的结合）进行免疫学筛选。

**典型应用**：高考常以“如何从某种真核生物中获得目的基因”为背景，要求考生判断应选用基因组文库还是cDNA文库。典型设问：若欲在某原核宿主中表达人胰岛素基因，应从人胰岛细胞中提取mRNA反转录构建cDNA文库，再从中筛选，因为cDNA不含内含子，可直接在原核细胞内表达；若研究者想研究某种生物的非编码RNA基因或启动子序列，则必须使用基因组文库。解题关键是比较两种文库的DNA来源、序列组成及基因是否具有完整结构。

## ⚠️ 高频易错点

- **混淆基因组文库与cDNA文库的组成**：误认为cDNA文库也含有启动子、内含子，实际上它仅包含经过剪接后的成熟mRNA反转录得到的编码区序列。
- **混淆“基因文库”与“基因库”的概念**：基因库是指一个种群中全部个体所含有的全部基因，属于群体遗传学概念；而基因文库是人工构建的克隆混合物，属于基因工程操作层面。
- **真核基因原核表达中的文库选择错误**：直接选基因组文库中的基因用于原核表达，因内含子的存在会导致原核系统无法正确表达真核基因，应选择cDNA文库。
- **误认为基因文库中每个受体菌含有多种外源基因**：文库构建时通过控制连接和转化条件，确保每个受体菌通常只含有一种外源DNA片段，整个群体才构成完整的文库。
- **忽略基因组文库构建时酶切策略**：构建基因组文库常用限制酶**部分消化**，以产生大小合适的重叠片段，并非彻底切割，否则会破坏基因连续性。

## 🌐 中英对照（术语表）

| 中文 | English | 说明 |
|------|---------|------|
| 基因文库 | gene library | 包含某种生物不同DNA片段的克隆群体 |
| 基因组文库 | genomic library | 由基因组DNA片段构建，含全部遗传信息（含内含子） |
| cDNA文库 | cDNA library | 由mRNA反转录的cDNA构建，仅含编码区 |
| 载体 | vector | 携带外源DNA并导入受体细胞的工具，如质粒 |
| 受体菌 | host cell | 接纳重组载体的细菌等宿主细胞，如大肠杆菌 |
