# 微生物分离培养

> **一句话**：在无菌条件下，利用选择培养基或平板划线/稀释涂布等方法从混杂微生物群体中获得单一菌株（纯培养）的技术。
> **English**: A set of aseptic techniques using selective media or streaking/dilution plating to obtain a single microbial strain (pure culture) from a mixed population.

## 🎓 考点精讲

**核心原理**：微生物分离培养基于**细胞学说**与**微生物纯培养概念**——一个菌落理论上由单个微生物细胞分裂增殖形成。高考中主要考查两类方法：

**1. 平板划线法**：通过接种环在固体培养基表面连续划线，将混杂菌体逐步稀释分散，最终获得由单个细胞形成的单菌落。关键操作：每一次划线前后必须灼烧接种环，划线末端不可与上一区重叠。设问常考灼烧目的（杀灭残留菌体、避免污染）与划线路径设计。

**2. 稀释涂布平板法**：将菌液进行梯度稀释（通常 $10^{-1}$ 至 $10^{-7}$），取一定体积涂布于平板，经培养后对单菌落计数，可推算样品活菌数。公式：

$$\text{样品活菌数}/\text{mL} = \frac{\text{平板上菌落数平均数} \times \text{稀释倍数}}{\text{涂布体积(mL)}}$$

高考必考计数原则：选择菌落数在 **30–300** 之间的平板进行计数，取平均值；若三个稀释度平板均符合范围，需分别计算后平均。

**选择培养基的应用**：高考常以尿素分解菌、纤维素分解菌筛选为情境。原理：以目标底物为唯一碳源或氮源（如尿素为唯一氮源筛选脲酶产生菌），或加入指示剂（如酚红指示脲酶分解尿素产氨使培养基变红）进行鉴别。

**典型设问**：
- “如何验证某实验已获得纯培养？”（答：观察菌落形态特征是否均一，或挑取菌落重新划线后是否只产生一种菌落）
- “稀释涂布平板法统计菌落数比实际偏低的原因？”（答：两个或多个细胞可能形成同一菌落）

## ⚠️ 高频易错点

- **混淆灼烧时机与目的**：平板划线法中，接种前灼烧是灭菌，每次划线后灼烧是杀死残留菌种，划线结束仍需灼烧以防污染环境；稀释涂布平板法的涂布器仅在使用前、后用酒精浸泡与灼烧。
- **混淆两种方法的计数功能**：平板划线法**不能**用于计数，仅能分离纯化；稀释涂布平板法可分离并计数活菌，但不能统计死菌。
- **活菌计数公式单位迷乱**：结果单位为“个/mL”，必须考虑涂布体积（常见设问给出 0.1 mL 涂布量，学生易忽略），且稀释倍数为涂布前菌液的稀释倍数。
- **菌种保藏混淆**：临时保藏用斜面培养基并存于 4°C（易变异）；长期保藏用甘油管藏法存于 –20°C 以下。
