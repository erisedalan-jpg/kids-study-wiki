# 琼脂块法

> **一句话**：琼脂块法是用含酚酞的琼脂块模拟细胞，通过 NaOH 扩散深度与体积比的关系，探究细胞表面积与体积之比如何限制细胞大小的实验方法。
> **English**: A laboratory method using phenolphthalein-containing agar blocks to simulate cells and investigate how the surface area–to–volume ratio limits cell size via NaOH diffusion.

## 🎓 考点精讲

**定义/原理**  
该实验基于物质扩散速率与表面积成正比、而细胞需求与体积成正比的原理。当琼脂块（模拟细胞）浸入 NaOH 溶液时，NaOH 向内部扩散，遇酚酞变红。测量未变色部分的边长或体积，可计算物质到达中心的效率。随着琼脂块体积增大，表面积/体积比（S/V）减小，物质扩散到中心所需时间增长，相对扩散速率下降。

定量关系：设正方体边长为 \(L\)，则表面积 \(S = 6L^2\)，体积 \(V = L^3\)，表面积与体积之比 \(\frac{S}{V} = \frac{6}{L}\)。当 \(L\) 增大时比值减小，说明大细胞通过扩散获取养分或排出废物的效率更低。公式常用来比较不同大小琼脂块的扩散效率，可要求计算扩散体积分数或扩散深度。

**关键方法**  
- 控制单一变量：琼脂块大小是自变量，NaOH 浓度、浸泡时间等需保持相同。  
- 扩散程度测量：用刀切开琼脂块，测量红色区域厚度（扩散深度），或测量未变色核心边长，计算已扩散体积与总体积的比值。  
- 数据处理：常用表格或坐标系呈现边长与 S/V 比、扩散体积比的关系，推导「细胞不能无限长大」的结论。

**典型应用**  
高考中常以实验分析题或图表题出现：给出不同大小琼脂块的扩散数据，要求计算 S/V 比，推测扩散效率，并解释为何细胞需要通过分裂维持较小体积。有时结合物质跨膜运输、细胞分裂意义设问。也常考实验设计中应如何设置对照、如何保证浸入时间一致等实验细节。

## ⚠️ 高频易错点

- 混淆琼脂块的“扩散速率”与“扩散效率”：扩散速率（单位时间扩散深度）相同，但大块中扩散到中心的比例更低，常误认为 NaOH 扩散速度不同。  
- 将表面积/体积比直接等同于细胞大小：比值由边长决定，边长越大比值越小，解题时需区分边长、表面积、体积三者关系。  
- 错误计算扩散体积分数：未变色核心的体积应从总体积中减去已扩散部分，学生常直接用红色区域厚度估算。  
- 实验变量控制不当：忽略浸泡时间、NaOH 浓度或切割时块的温度，导致错误结论归因于块大小本身。
