# 等位基因分离

> **一句话**：在减数分裂形成配子时，同源染色体上的等位基因彼此分开，分别进入不同配子的过程，是孟德尔分离定律的细胞学基础。
> **English**: The process in which alleles segregate from each other during meiosis so that each gamete carries only one allele for each gene.

## 🎓 考点精讲

**定义/原理**：等位基因分离的实质是减数第一次分裂后期同源染色体的分离。以一对等位基因 A/a 为例，杂合子（Aa）在形成配子时，A 与 a 随着同源染色体分开而进入不同的次级性母细胞，最终产生 A 配子与 a 配子，且比例为 1∶1。

遗传图解表示：
- 亲本：P　AA × aa
- 子一代：F₁　Aa（全为显性性状）
- F₁ 自交：F₂ 基因型比例 AA∶Aa∶aa＝1∶2∶1，表现型比例 3∶1

**关键方法**：高考考查基因分离定律的验证与推演，核心步骤包括：
1. **判断基因位置**：通过正交反交、测交、自交等实验设计，判断控制性状的基因是否位于核染色体上（细胞质遗传不遵循分离定律，表现为母系遗传）。
2. **推演配子与后代比例**：熟练运用分枝法或棋盘法计算杂交后代基因型与表现型比例，尤其注意“连续自交”“随机交配”情形下的基因频率与基因型频率变化。
3. **分析异常分离比**：如不完全显性（1∶2∶1）、致死效应（合子致死或配子致死）、从性遗传等特殊情境，实质仍遵循基因分离定律，需正确识别致死基因型。

**典型应用**：
- **题型一**：给出杂交实验结果（如 F₂ 出现 3∶1 或 1∶2∶1），要求判断显隐性关系并推演亲代基因型。解题关键是先确定显隐性，再根据后代比例反推。
- **题型二**：结合人类遗传病系谱图，判断致病基因的显隐性及个体基因型概率。需注意“无中生有为隐性，有中生无为显性”的判断原则，并结合个体婚配关系计算后代患病风险。

## ⚠️ 高频易错点

- **混淆“基因分离”与“性状分离”**：基因分离是减数分裂中等位基因的细胞学行为；性状分离是杂合子自交后代表现型出现显性与隐性的现象。二者相关但所指不同。
- **误以为分离定律只适用于完全显性**：不完全显性、共显性、致死基因等情形同样遵循等位基因分离，只是表现型比例发生变形，基因型比例仍为 1∶2∶1。
- **配子致死与合子致死的混淆**：配子致死改变配子种类及比例（如含 A 的雄配子致死，则 A 雄配子不参与受精）；合子致死改变受精后存活个体的基因型比例。推算时必须先明确致死发生的阶段。
- **XY 型性别决定的伴性遗传中忽视同源区段**：X 与 Y 染色体存在同源区段，该区段基因仍可存在等位基因，遵循分离定律；非同源区段基因则表现为伴性遗传特性，但仍服从减数分裂中染色体分离规律。
