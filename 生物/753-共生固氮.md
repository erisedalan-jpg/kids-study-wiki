# 共生固氮

> **一句话**：共生固氮是指固氮微生物（如根瘤菌、弗兰克氏菌）与特定植物（多为豆科）形成共生结构（如根瘤），在共生状态下将大气中的分子氮（N₂）还原为氨（NH₃）的过程。
> **English**: Symbiotic nitrogen fixation is the process in which nitrogen-fixing microorganisms form symbiotic structures with host plants and reduce atmospheric N₂ to ammonia within these structures.

## 🎓 考点精讲

- **定义/原理**：共生固氮的核心是微生物与植物的**互利共生**。微生物通过固氮酶复合体将N₂还原为NH₃，供给植物合成氨基酸、蛋白质等含氮有机物；植物通过光合作用为微生物提供碳源（如苹果酸、蔗糖）和能量（ATP）。其总反应式为：
  N₂ + 8e⁻ + 8H⁺ + 16ATP → 2NH₃ + H₂ + 16ADP + 16Pi
  该过程在共生体特有的**低氧微环境**中进行。豆科植物根瘤中的**豆血红蛋白**可高效结合O₂并缓慢释放，既满足共生菌呼吸产ATP的需求，又防止固氮酶被O₂不可逆失活。固氮酶包含铁蛋白和钼铁蛋白两个组分，电子从铁氧还蛋白经铁蛋白传递至钼铁蛋白，最终将N₂还原。弗兰克氏菌与桤木等非豆科植物形成的放线菌根瘤也是典型共生固氮体系。

- **关键方法**：高考解题需抓住三点：①**识别共生结构**（根瘤形态及其在根部着生位置，区别于虫瘿或肿瘤）；②**区分固氮微生物类型**（共生固氮菌需与特定宿主建立共生关系才能高效固氮，不同于自生固氮菌如固氮菌属可在土壤中独立固氮）；③**分析能量与物质流向**（植物→菌：碳源/能量；菌→植物：氨/氨基酸）。

- **典型应用**：高考常见设问包括：①“豆科植物根瘤中固氮酶为何需低氧环境？豆血红蛋白的作用机制如何？”——考查酶特性与微环境调节；②“比较共生固氮与自生固氮在生态氮循环中的贡献与条件差异”；③“从物质与能量角度，描述大豆-根瘤菌共生体系中碳、氮元素的转化路径”（可要求绘制物质流与能量流示意图）；④在生态题中分析农田轮作豆科作物（绿肥）对土壤氮素补充的机理，涉及农业可持续发展的应用。

## ⚠️ 高频易错点

- **混淆根瘤与菌根**：根瘤是固氮细菌与植物根系形成的共生固氮结构，产物为氨；菌根是真菌与植物根系形成的共生结构，主要增强水分和矿质元素（尤其磷）的吸收，不具备固氮功能。
- **误认为所有豆科植物都有根瘤、所有固氮菌都能共生**：并非所有豆科植物在自然条件下都结瘤固氮，且根瘤菌与宿主间存在严格的宿主特异性（如大豆根瘤菌不能使苜蓿结瘤）。
- **将共生固氮等同于植物直接固氮**：固氮能力完全来自原核微生物（细菌），植物本身不能固氮。根瘤中的类菌体形态的根瘤菌才是固氮的执行者。
- **忽略豆血红蛋白的来源与功能**：豆血红蛋白的珠蛋白部分由植物基因编码，血红素由根瘤菌合成，是一种跨界合作产物。功能是运输O₂而非结合N₂。（若考题涉及此细节，需明确来源划分）
