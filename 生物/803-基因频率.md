# 基因频率

> **一句话**：基因频率是指在一个种群基因库中，某个基因占全部等位基因数的比率。
> **English**: Gene frequency is the ratio of a particular allele to the total number of all alleles at that locus in a population.

## 🎓 考点精讲

**定义与核心原理**
基因频率是描述种群基因库组成的基本参数。设某基因位点有一对等位基因 $A$ 和 $a$，种群个体数为 $N$，其中基因型 $AA$、$Aa$、$aa$ 的个体数分别为 $N_{AA}$、$N_{Aa}$、$N_{aa}$，则基因频率计算公式为：
- 基因 $A$ 的频率：$p = \frac{2N_{AA} + N_{Aa}}{2N}$
- 基因 $a$ 的频率：$q = \frac{2N_{aa} + N_{Aa}}{2N}$，且 $p + q = 1$。

若已知各基因型频率 $P(AA)$、$P(Aa)$、$P(aa)$，则 $p = P(AA) + \frac{1}{2}P(Aa)$。

**关键方法**
1. **常染色体基因频率计算**：直接由基因型频率推基因频率，使用上述公式。若题目给出患者数量，先求隐性（或显性）基因型频率，开平方求基因频率（仅在遗传平衡时适用），再代入公式求其余基因型频率。
2. **伴 X 染色体基因频率计算**：需区分雌雄。例如计算 X 染色体上某等位基因 $X^B$ 的频率，应使用 $p = \frac{2N_{X^BX^B} + N_{X^BX^b} + N_{X^BY}}{2N_{雌} + N_{雄}}$。特别注意 Y 染色体上无对应等位基因，分母不可直接使用 2 倍总个体数。
3. **遗传平衡定律（哈代-温伯格定律）应用**：当种群满足无限大、随机交配、无突变、无迁移、无自然选择时，基因频率和基因型频率在世代间保持恒定，且基因型频率可用 $p^2 + 2pq + q^2 = 1$ 描述。高考中常见题型为利用该定律由隐性纯合子频率求基因频率，进而计算杂合子频率。

**典型应用**
- **基因频率变化与进化**：通过比较不同世代基因频率是否改变，判断种群是否发生进化。基因频率的改变是生物进化的实质。
- **遗传病基因频率计算**：如已知某常染色体隐性遗传病在人群中的发病率（即 $q^2$），求正常人群中携带者概率。先求出 $q$，得 $p = 1 - q$，则携带者概率为 $\frac{2pq}{p^2 + 2pq}$ 或直接 $\frac{2q}{1+q}$（待家长核对）。

## ⚠️ 高频易错点

- **混淆“基因频率”与“基因型频率”**：前者针对单个基因，后者针对基因型组合。计算时不可将杂合子频率直接当作基因频率，必须乘以 1/2。
- **伴性遗传基因频率分母算错**：计算 X 染色体基因频率时，分母的基因总数在雌性中为个体数 ×2，雄性中为个体数 ×1。常见错误是直接除以 2 倍总个体数。
- **误用遗传平衡定律条件**：在存在选择、迁入迁出或非随机交配的种群中，机械套用 $p^2$、$2pq$、$q^2$ 计算基因型频率，导致错解。使用前必须确认题干是否明确“理想条件”或“已达平衡”。
- **自交与自由交配计算混淆**：自交改变基因型频率但不改变基因频率，自由交配（随机交配）在无其他因素时基因频率也不变，但其基因型频率变化规律不同。计算子代基因频率时，若为自交，应直接由亲代基因型频率推，不可用平衡定律一步到位。

## 🌐 中英对照（术语表）
