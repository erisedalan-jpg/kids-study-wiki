# 基因表达载体

> **一句话**：基因表达载体是使外源目的基因在受体细胞中稳定表达所需蛋白质的特殊DNA分子，必须包含启动子、目的基因、终止子、标记基因等核心元件。
> **English**: A gene expression vector is a specially designed DNA molecule that enables a foreign gene of interest to be stably expressed as a protein in a recipient cell, and must contain core elements such as a promoter, the target gene, a terminator, and a marker gene.

## 🎓 考点精讲

- **定义与核心元件**  
  基因表达载体是基因工程的核心工具，本质是重组DNA分子，通常以质粒、噬菌体或病毒DNA为基础改造而成。它必须包含四个基本元件：  
  - **启动子**：RNA聚合酶识别和结合的位点，用于驱动目的基因转录。启动子需与受体细胞RNA聚合酶匹配（如原核细胞用原核启动子，真核细胞常需真核启动子，植物中有时用35S启动子）。  
  - **目的基因**：经过修饰且编码目标蛋白的DNA片段，通常通过限制酶切割、PCR扩增或人工合成获得。  
  - **终止子**：提供转录终止信号，使转录在正确位置停止。  
  - **标记基因**：用于筛选成功导入载体的受体细胞，常用抗生素抗性基因（如氨苄青霉素抗性基因）或荧光蛋白基因。  
  此外，载体还需有**多克隆位点（MCS）**，即在启动子与终止子之间排列多个单一限制酶切位点，方便插入目的基因。

- **构建方法的关键步骤**  
  1. **选择限制酶**：用两种不同限制酶切割载体和目的基因（**双酶切**），产生不同的黏性末端，避免载体自身环化和目的基因反向连接。所选酶切位点必须在标记基因外、启动子与终止子之间，且不能破坏启动子或终止子。  
  2. **连接**：用DNA连接酶将目的基因与载体切口连接，形成闭合环状重组DNA。  
  3. **转化与筛选**：将重组载体导入受体细胞（如用Ca²⁺处理的大肠杆菌感受态细胞）。利用标记基因筛选阳性克隆（例如在含抗生素的培养基上只有成功导入载体的细胞能生长）。  
  4. **表达检测**：通过抗原-抗体杂交、分子杂交或直接测定蛋白质功能等方法验证目的基因是否成功表达。

- **高考典型设问**  
  - 给出质粒图谱和目的基因序列，让考生选择合适的限制酶及切割位置，并书写切口黏性末端。  
  - 分析“单酶切”为何容易导致载体自连或目的基因反向插入。  
  - 判断改造后的表达载体能否在特定受体细胞中成功表达，并解释原因（例如动物基因导入原核生物时，因缺乏内含子加工机制，需用cDNA）。  
  - 设计PCR引物时，需在引物5′端添加限制酶识别序列，并考虑保护碱基以保证酶切效率。

## ⚠️ 高频易错点

- **限制酶切割位点选择不当**：切割位点落在标记基因内部或启动子、终止子区域，导致筛选失效或无法正常表达。  
- **单酶切导致的连接问题**：用同一种限制酶切割载体和目的基因，载体易发生自身环化，目的基因也可能正、反两个方向插入，降低正确重组概率。  
- **启动子与受体不匹配**：原核启动子在真核细胞中通常不能有效工作，反之亦然。例如，将带有哺乳动物启动子的表达载体导入大肠杆菌，目的基因无法转录。  
- **忽视内含子问题**：真核生物基因组来源的目的基因若直接连入原核表达载体，因原核细胞缺少剪接机制，无法去除内含子，导致翻译出错误蛋白质，因此常用逆转录得到的cDNA。
