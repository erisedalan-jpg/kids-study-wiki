# DNA
> **一句话**：DNA（脱氧核糖核酸）是生物体遗传信息的主要载体，以双螺旋结构存在于细胞核中。
> **English**: DNA (deoxyribonucleic acid) is the primary carrier of genetic information, existing as a double helix in the cell nucleus.

---

## 📚 给 6-12 岁（自读版）

你有没有想过，为什么你长得像爸爸妈妈？秘密就在一种叫 **DNA** 的分子里。

DNA 就像一本"生命说明书"，藏在细胞核里的染色体上。它告诉细胞：怎么长眼睛、怎么长头发、血型是什么……每个人的 DNA 都略有不同，所以我们每个人都是独一无二的。

DNA 的形状像一个螺旋形的楼梯，科学家叫它**双螺旋结构**。如果把一个细胞里的 DNA 全部拉直，足足有 2 米长，但它卷得非常紧，才能塞进只有头发丝十分之一大小的细胞核里。

## 🎓 给 12+（进阶版）

### 定义
DNA（deoxyribonucleic acid，脱氧核糖核酸）是以脱氧核苷酸为基本单位，由两条反向平行的脱氧核苷酸链构成的双螺旋分子，是绝大多数生物遗传信息的载体。

### 原理 / 机制

| 特征 | 说明 |
|------|------|
| 基本单位 | 脱氧核苷酸（脱氧核糖 + 磷酸 + 含氮碱基） |
| 碱基种类 | A（腺嘌呤）、T（胸腺嘧啶）、G（鸟嘌呤）、C（胞嘧啶） |
| 碱基配对 | A-T（2 个氢键），G-C（3 个氢键） |
| 结构 | 规则双螺旋，两链反向平行 |
| 分布 | 主要在细胞核（染色体），少量在线粒体、叶绿体 |
| 功能 | 储存、传递、表达遗传信息 |

**DNA 与基因的关系**：
- 基因是有遗传效应的 DNA 片段
- 一个 DNA 分子上有多个基因
- 基因通过控制蛋白质合成来控制生物性状

**DNA 复制（简述）**：
- 以亲代 DNA 两条链各自为模板
- 按碱基互补配对原则合成新链
- 结果：一条 DNA → 两条相同 DNA（半保留复制）

### 典型例题
**题**：DNA 的两条链之间靠什么连接？A 与哪种碱基配对？

**解**：两条链之间靠**氢键**连接；A 与 **T** 配对（2 个氢键）。

### 易错点
- DNA 是双链，RNA 是单链，不能混淆
- 基因 ⊂ DNA（基因是 DNA 的片段，但并非所有 DNA 都是基因）
- DNA 主要在细胞核，但线粒体和叶绿体也含有少量 DNA
