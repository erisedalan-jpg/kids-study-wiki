# 反射
> **一句话**：反射是神经系统对内外刺激作出的规律性应答，是神经系统功能活动的基本方式，分为非条件反射和条件反射。
> **English**: A reflex is a rapid, stereotyped neural response to a stimulus; it is the basic mode of nervous system activity, classified as unconditioned or conditioned reflexes.

---

## 📚 给 6-12 岁（自读版）

反射就是身体对刺激的自动反应。

**非条件反射**（天生就有）：
- 手碰到火烫→立刻缩手
- 眼睛被光照→瞳孔缩小
- 吃饭分泌唾液

**条件反射**（后天学来的）：
- 看到梅子→分泌唾液（望梅止渴）
- 听到上课铃→准备学习
- 红灯亮→停下脚步

条件反射比非条件反射"高级"——它需要大脑参与，能根据经验调整。只有人类能形成语言条件反射（"梅子"这个词就能让你流口水）！

## 🎓 给 12+（进阶版）
### 定义
反射（reflex）是在中枢神经系统参与下，机体对内外刺激所发生的规律性反应。反射的结构基础是反射弧，必须在反射弧完整的条件下才能完成。

### 结构 / 过程（表格）

| 类型 | 非条件反射 | 条件反射 |
|------|-----------|---------|
| 形成方式 | 先天遗传 | 后天学习 |
| 神经中枢 | 脊髓或低级脑中枢 | 大脑皮层（高级中枢） |
| 刺激特点 | 具体直接刺激 | 信号刺激（抽象） |
| 稳定性 | 固定不变 | 可建立、可消退 |
| 例子 | 膝跳反射、缩手反射 | 望梅止渴、闻声起立 |

**条件反射建立过程**：无关刺激（信号）+ 非条件刺激反复同时呈现→两者在大脑皮层形成暂时联系→条件反射建立。

**人类特有**：语言和文字信号系统（第二信号系统），能对词语（符号）产生条件反射。

### 典型实验 / 例题
- **巴甫洛夫实验**：给狗喂食同时响铃（非条件刺激+无关刺激）→多次后单独响铃→狗分泌唾液（条件反射建立）。
- **例题**：脊髓受损后，膝跳反射消失，说明什么？——反射需要反射弧完整，脊髓损伤破坏了反射弧，反射无法完成。

### 易错点
1. 条件反射不稳定，若长时间不强化（无关刺激不与非条件刺激配对），会**消退**。
2. 缩手反射的中枢在脊髓，但感觉（疼痛）仍能上传大脑（有感觉但反应已在脊髓完成）。
3. 反射弧任何一个环节被破坏，反射就不能完成——这是与其他应激方式的本质区别。
