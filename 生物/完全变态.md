# 完全变态

> **一句话**：完全变态是昆虫发育经历**卵→幼虫→蛹→成虫** 4个形态差异显著阶段的变态类型。
> **English**: Complete metamorphosis is a type of insect development with four distinct stages: egg → larva → pupa → adult.

---

## 📚 给 6-12 岁（自读版）

毛毛虫怎么变成蝴蝶？这个大变身要经过**4个阶段**！

**完全变态 4 步：**

| 阶段 | 特征 | 蝴蝶的例子 |
|------|------|----------|
| 🥚 卵 | 妈妈产下的蛋 | 卵产在叶子上 |
| 🐛 幼虫 | 孵化后使劲吃 | 毛毛虫（鳞翅目幼虫） |
| 💤 蛹 | 不吃不动，大改造 | 茧里的蛹 |
| 🦋 成虫 | 飞出来，开始繁殖 | 美丽的蝴蝶 |

**关键特点：** 幼虫和成虫**长得完全不同**，中间要经过**蛹期**大改造！

**常见完全变态昆虫：**
- 蝴蝶（幼虫=毛毛虫）
- 蚕（幼虫=蚕宝宝，蛹在茧里）
- 蜜蜂（幼虫白白胖胖，成虫有翅有刺）
- 苍蝇（幼虫=蛆，成虫=苍蝇）
- 蚊子（幼虫=孑孓，在水中；成虫飞翔）

**记住：有蛹 = 完全变态！**

## 🎓 给 12+（进阶版）

### 定义

**完全变态**（Complete Metamorphosis / Holometabolism）：昆虫个体发育经历**卵（egg）→幼虫（larva）→蛹（pupa）→成虫（adult）** 4个阶段，幼虫与成虫在形态、食性、生活习性上差异极大，中间须经过**蛹期**进行组织器官的全面重建。

### 原理 / 机制

**蛹期的实质：**

| 过程 | 说明 |
|------|------|
| 幼虫组织解体 | 大量幼虫专属细胞凋亡，营养释放 |
| 成虫盘发育 | 幼虫体内的成虫盘（imaginal disc）增殖分化 |
| 器官重建 | 翅、复眼、口器、生殖器官等成虫结构形成 |

**完全变态代表昆虫：**

| 目 | 代表昆虫 | 幼虫名称 |
|----|---------|---------|
| 鳞翅目 | 蝴蝶、蛾、蚕 | 毛毛虫（caterpillar） |
| 双翅目 | 苍蝇、蚊子 | 蛆（maggot）/ 孑孓（larva） |
| 膜翅目 | 蜜蜂、蚂蚁、黄蜂 | 无脚白色幼虫 |
| 鞘翅目 | 甲虫（瓢虫、天牛） | 蠕虫状幼虫 |

**激素调控：**
- 幼虫期：**保幼激素**（JH）高，**蜕皮激素**（E）也高 → 蜕皮但保持幼虫形态
- 末龄幼虫：保幼激素降低 → 蜕皮激素触发 → 化蛹
- 蛹期末：蜕皮激素再次升高 → 羽化为成虫

### 典型例题

**例题**：蚕的发育过程是完全变态，请写出其发育的4个阶段，并说明各阶段的主要特征。

**答案：**
- **卵**：受精卵，坚硬外壳保护，越冬
- **幼虫（蚕）**：孵化后大量取食桑叶，多次蜕皮，积累营养
- **蛹**：幼虫吐丝结茧，包裹其中化蛹，内部组织重建
- **成虫（蚕蛾）**：羽化飞出，交配产卵，口器退化不取食，寿命短

### 易错点

- **蛹期不完全静止**：外表静止，内部发生剧烈的组织解体和重建
- 蝇的幼虫叫"蛆"，蚊的幼虫叫"孑孓"，都是完全变态
- 蜻蜓是**不完全变态**（无蛹期），常被误认为完全变态
- 成虫期的主要任务是**繁殖**，许多成虫（如蚕蛾）口器退化，不取食或取食很少
