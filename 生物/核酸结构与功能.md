# 核酸结构与功能

> **一句话**：核酸是由核苷酸聚合而成的生物大分子，是遗传信息的储存与传递载体，包括脱氧核糖核酸（DNA）和核糖核酸（RNA）两类。
> **English**: Nucleic acids are biological macromolecules polymerized from nucleotides, serving as the carriers of genetic information storage and transfer, including deoxyribonucleic acid (DNA) and ribonucleic acid (RNA).

## 🎓 考点精讲

### 一、核苷酸——核酸的基本组成单位

每个核苷酸由三部分组成：一分子**磷酸**、一分子**五碳糖**、一分子**含氮碱基**。
- DNA 的五碳糖为**脱氧核糖**，RNA 的五碳糖为**核糖**。
- 碱基分为嘌呤（A 腺嘌呤、G 鸟嘌呤）和嘧啶（C 胞嘧啶、T 胸腺嘧啶 / U 尿嘧啶）。DNA 特有 T，RNA 特有 U。

### 二、DNA 双螺旋结构（核心考点）

由沃森和克里克提出，要点如下：
1. **两条链反向平行**：一条链 5′→3′，另一条 3′→5′。
2. **外侧骨架**：磷酸与脱氧核糖交替连接，形成基本骨架。
3. **内侧碱基互补配对**：A 与 T 通过两个氢键配对，G 与 C 通过三个氢键配对。
4. **空间结构**：向右盘旋成双螺旋，G-C 含量越高，DNA 越稳定。

### 三、RNA 的结构与分类

RNA 通常为**单链**，局部可折叠形成双链区（也遵循 A-U、G-C 配对）。
- **mRNA**（信使 RNA）：携带遗传信息，作为翻译模板。
- **tRNA**（转运 RNA）：一端携带氨基酸，另一端有反密码子，识别密码子。
- **rRNA**（核糖体 RNA）：与蛋白质构成核糖体，催化肽键形成（核酶功能）。

### 四、核酸的功能

DNA 是**遗传信息的储存者**，通过半保留复制传递；RNA 参与**基因表达**（转录与翻译），某些病毒以 RNA 为遗传物质。

### 高考典型设问与解题方法

**设问 1：结构辨识题**——给出某核苷酸或核酸片段示意图，判断属于 DNA 或 RNA。
- **关键**：看五碳糖（是否脱氧）、看碱基（有无 T 或 U）。

**设问 2：碱基计算题**——已知双链 DNA 中某碱基比例，求另一碱基或单链中的比例。
- **核心公式**（查伽夫法则）：
  - 双链 DNA 中：A = T，G = C；A + G = T + C = 50%（嘌呤 = 嘧啶）。
  - 某碱基在双链中的比例 = 其在两条单链中比例的平均值。
- **解题步骤**：先由互补关系确定各碱基的整体数量关系，再结合单链比例需求反推。

**设问 3：结构稳定性比较**——比较不同 DNA 分子的热稳定性。
- **判定依据**：G-C 碱基对越多（三氢键），DNA 越稳定，解旋所需温度越高。

## ⚠️ 高频易错点

- **混淆单链与双链的碱基计算规则**：单链中 A 不一定等于 T、G 不一定等于 C，只有双链中才严格成立。若题目给出单链比例求另一条链，必须用互补关系推导，不能直接套用 AT/GC 相等。
- **误将 RNA 的 U 用于 DNA 结构描述**：DNA 中不含尿嘧啶（U），RNA 中不含胸腺嘧啶（T）；写核酸结构或碱基配对时必须区分清楚。
- **混淆核苷酸与核酸的组成层次**：核苷酸是单体（含磷酸+五碳糖+碱基），核酸是多聚体。题目问“某物质含有 C、H、O、N、P 且为遗传物质”时，指向核酸而非单纯核苷酸。
- **误认为 RNA 全是单链无配对**：tRNA 和 rRNA 均有局部双链区，存在碱基配对（A-U、G-C），该配对也可用于 RNA 结构稳定性分析。
