# 胚胎发育

> **一句话**：受精卵在子宫内经过细胞分裂和分化，逐渐发育成完整胎儿的过程。
> **English**: Embryonic development is the process by which a fertilized egg divides and differentiates in the uterus, gradually forming a complete fetus.

---

## 📚 给 6-12 岁（自读版）

你在妈妈肚子里待了大约 **40 周（约 280 天）**，才从一个微小的受精卵长成出生时的小宝宝。这个过程就是**胚胎发育**。

受精卵形成后，一边向子宫移动，一边不停地分裂。到达子宫后，它会"钻"进子宫内膜里，这叫做**着床**，就像种子扎根在土地里。

着床之后，胚胎开始快速生长：

- **第 4 周**：心脏开始跳动
- **第 8 周**：四肢轮廓出现，能看出是人形，这时叫做**胚胎**
- **第 12 周以后**：叫做**胎儿**，各器官继续发育完善
- **第 40 周**：胎儿发育成熟，准备出生

在妈妈肚子里，胎儿通过**胎盘**和**脐带**与妈妈连接。胎盘是一个特别的器官，胎儿从这里获得**氧气和营养物质**，同时把体内的**二氧化碳和废物**交给妈妈的血液带走处理。

**注意**：胎儿的血液和妈妈的血液**不会直接混合**，它们通过胎盘的薄膜进行物质交换。

## 🎓 给 12+（进阶版）

### 定义

**胚胎发育**（Embryonic Development）：受精卵（合子）经过连续的有丝分裂（卵裂）、细胞分化与形态发生，最终形成具有完整结构与功能的个体的过程。人类胚胎发育主要在子宫内进行，历时约 266 天（从受精算起）。

### 发育阶段与关键节点

| 时间（受精后） | 阶段/事件 | 主要特征 |
|--------------|---------|---------|
| 0 天 | 受精卵形成 | 46条染色体，输卵管内 |
| 3~4 天 | 桑椹胚 | 多次卵裂，形成实心细胞团 |
| 5~6 天 | 囊胚 | 出现内细胞团（发育成胎儿）和滋养层（发育成胎盘） |
| 6~10 天 | 着床（植入） | 囊胚嵌入子宫内膜 |
| 第 3~8 周 | 胚胎期 | 三胚层分化，器官原基形成（**致畸敏感期**） |
| 第 9 周起 | 胎儿期 | 器官系统继续生长与完善 |
| 第 40 周（满月） | 出生 | 胎儿成熟，经分娩离开母体 |

### 胎盘的结构与功能

**胎盘**（Placenta）是胎儿与母体进行物质交换的专属器官：

| 功能 | 说明 |
|------|------|
| 营养物质供应 | 葡萄糖、氨基酸、脂肪酸由母血 → 胎儿血 |
| 气体交换 | O₂ 由母血 → 胎儿血；CO₂ 由胎儿血 → 母血 |
| 废物排出 | 尿素等代谢废物 → 母血 → 母体排出 |
| 激素分泌 | 分泌 hCG、雌激素、孕激素，维持妊娠 |
| 屏障功能 | 胎盘屏障阻止大分子物质和多数病原体通过 |

**注意**：胎儿血液与母体血液在胎盘内**不直接混合**，通过绒毛膜绒毛进行物质交换。

### 三胚层与器官来源

| 胚层 | 发育来源的器官/系统 |
|------|----------------|
| 外胚层 | 皮肤表皮、神经系统、感觉器官 |
| 中胚层 | 肌肉、骨骼、循环系统、泌尿生殖系统 |
| 内胚层 | 消化道、呼吸道内衬、肝、胰、甲状腺 |

### 典型例题

**例题**：胎儿在母体内通过何种结构获取营养和排出废物？

**答案**：胎儿通过**胎盘**和**脐带**与母体进行物质交换，从母体获取氧气和营养物质，同时将代谢废物（CO₂、尿素等）输送给母体排出。

### 易错点

- ❌ 胎儿血液与母亲血液直接混合 → ✅ 通过胎盘绒毛膜进行物质交换，血液不混合
- ❌ 着床即在输卵管内 → ✅ 受精在输卵管，着床在子宫
- ❌ 8周后仍叫胚胎 → ✅ 第8周末器官基本形成后改称胎儿
