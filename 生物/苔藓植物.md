# 苔藓植物

> **一句话**：苔藓植物是最低等的陆生植物，有茎和叶的分化但无真正的根，无维管束，依靠假根固定，生活在阴湿环境中，是空气质量的指示植物。
> **English**: Bryophytes are the most primitive land plants, with stem and leaf differentiation but no true roots or vascular tissue, living in moist shaded environments and serving as indicators of air quality.

---

## 📚 给 6-12 岁（自读版）

在潮湿的石头上、树干上、森林地面上，你常常能看到一层绿色的"绒毯"——那就是**苔藓植物**！

**苔藓植物长什么样？**

苔藓很小，高度一般只有几厘米。它有点像迷你版的植物：
- 有**茎**和**叶**（但非常小）
- 有**假根**：不是真正的根，只是用来固定身体，不能吸水
- **没有维管束**（不像树那样有专门的水管）

**苔藓的特殊本领**

苔藓的身体可以直接从表面吸收水分和矿物质，就像海绵一样！所以它们特别喜欢潮湿的地方。

**苔藓是空气"检测仪"**

苔藓对空气污染非常敏感，特别是二氧化硫（SO₂）。城市里工厂多的地方，苔藓几乎不见踪影；而空气清洁的原始森林里，苔藓生长茂盛。

所以，看看苔藓长不长，就能知道空气好不好——苔藓是天然的**大气污染指示植物**！

**苔藓的用途**

- 植物园和花艺中用作装饰
- 泥炭藓（一种苔藓）死亡后形成**泥炭**，可用作燃料或土壤改良剂
- 一些苔藓有药用价值

## 🎓 给 12+（进阶版）

### 定义

苔藓植物（bryophyte）是一类具有茎叶分化但无真正根和维管束的最低等陆生植物，包括苔类（Hepaticae）、藓类（Musci）和角苔类（Anthocerotae）三大类群。

### 主要特征

| 特征 | 苔藓植物 | 说明 |
|------|---------|------|
| 根 | 假根（rhizoid） | 单列细胞，仅固定，不吸水 |
| 维管束 | 无 | 水分靠体表吸收和渗透 |
| 叶 | 单层或少层细胞 | 无叶脉 |
| 生殖 | 孢子生殖为主 | 有性生殖需要水（精子游向卵） |
| 世代交替 | 配子体（绿色植物体）占优势 | 孢子体寄生于配子体上 |

### 生活史（世代交替）

1. **配子体（单倍体，n）**：我们看到的绿色苔藓植株，产生精子和卵细胞
2. **受精**：精子借助水游向卵细胞，受精形成合子
3. **孢子体（二倍体，2n）**：从合子发育，寄生在配子体上，产生孢子囊
4. **孢子散播**：孢子萌发长成新的配子体

苔藓植物**配子体世代占优势**（区别于蕨类植物孢子体占优势）。

### 生态特征

**对空气质量的指示作用**：
- 苔藓体表无蜡质，直接从大气吸收水分，对 SO₂、重金属等污染物极敏感
- 清洁环境：苔藓多样性高，生长旺盛
- 污染环境：苔藓减少或消失
- 可作为城市大气污染的生物监测指标

### 与其他植物的比较

| 特征 | 藻类 | 苔藓 | 蕨类 | 种子植物 |
|------|------|------|------|---------|
| 根 | 无 | 假根 | 真根 | 真根 |
| 维管束 | 无 | 无 | 有 | 有 |
| 种子 | 无 | 无 | 无 | 有 |
| 生活环境 | 水中 | 阴湿 | 阴湿 | 多样 |

### 典型代表

- **地钱（Marchantia）**：扁平叶状体，常见苔类
- **葫芦藓（Funaria）**：常见藓类，教材常用实验材料
- **泥炭藓（Sphagnum）**：大量吸水，形成泥炭沼泽

### 易错点

1. 苔藓有"叶"但不是真叶——无维管束，仅为单层细胞的扁平片状结构
2. 苔藓假根**不能吸水**，水分由茎叶体表直接吸收
3. 苔藓繁殖**需要水**（精子游动），在陆地上进化程度低于蕨类
4. 地衣（lichen）不是苔藓，而是真菌和藻类的共生体
