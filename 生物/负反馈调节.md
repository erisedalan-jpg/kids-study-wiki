# 负反馈调节

> **一句话**：负反馈调节是指在一个系统中，输出信息引起的效应反过来抑制或减弱最初的变化，从而使系统维持在相对稳定状态的调节机制。
> **English**: Negative feedback regulation is a mechanism in which the output of a system counteracts or dampens the initial change, thereby maintaining a relatively stable internal state.

## 🎓 考点精讲

- **定义/原理**
  负反馈调节是生物体维持内环境稳态（如体温、血糖、pH等）的核心机制。其基本路径为：**刺激 → 感受器 → 控制中心 → 效应器 → 反应 → 抑制初始刺激**。当某一生理指标偏离“调定点”时，系统会启动抵消该偏差的反应。经典的激素调节模型——**下丘脑-垂体-靶腺轴**——即为负反馈的典型体现。以甲状腺激素调节为例：
  - 寒冷等刺激使下丘脑分泌 **促甲状腺激素释放激素（TRH）** → 垂体分泌 **促甲状腺激素（TSH）** → 甲状腺分泌 **甲状腺激素**。
  - 当血液中甲状腺激素浓度升高至超过阈值时，会反过来**抑制**下丘脑和垂体的分泌活动，使TRH和TSH减少，从而使甲状腺激素分泌量回落，避免过度分泌。

- **关键方法**
  高考中判断机制是否为负反馈的标准方法：**看结果是否“反向”作用于原因，使系统输出趋于平稳**。解题时需分析某变量的变化序列，若最终环节抑制了起始环节，即为负反馈；反之，若最终环节增强起始环节，使系统偏离稳态，则为正反馈（如分娩、凝血）。

- **典型应用**
  高考常以“血糖平衡调节”为情境进行考查。进食后血糖升高：
  - 胰岛B细胞分泌**胰岛素**增多 → 促进组织细胞摄取、利用和储存葡萄糖，并抑制肝糖原分解和非糖物质转化为葡萄糖 → 血糖浓度**下降**。
  - 当血糖浓度回归正常值后，信息反馈抑制胰岛素的进一步分泌，使血糖不致过低。这便是负反馈。
  设问常要求写出胰岛素降血糖后血糖浓度稳定的原因，或画出负反馈调节的示意图，考察学生对“抑制自身分泌”这一补偿路径的准确描述。

## ⚠️ 高频易错点

- **混淆负反馈与正反馈**：将“愈发增强”的过程误判为负反馈。需牢记负反馈的终点是**稳态**，正反馈的终点通常是**某个事件的完成**（如凝血过程完成、胎儿娩出）。
- **负反馈并非单向“抑制”**：负反馈是针对“偏离”进行双向调节。如体温过高时增强散热，体温过低时增强产热并减少散热，两者均属负反馈。错误地认为负反馈只在“过高”或“过低”一端作用。
- **错误地将反馈路径切断**：在甲状腺激素负反馈调节中，常误认为甲状腺激素只抑制垂体，而忽略其对下丘脑的抑制；或误认为反馈是直接抑制甲状腺自身。需明确反馈作用于**调控链条的上位中枢**（下丘脑和垂体）。
