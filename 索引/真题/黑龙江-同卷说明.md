## 结论

黑龙江与吉林历年用**同一套全国卷**：

```
全国Ⅱ卷(2008–2009) → 新课标卷(2010–2012) → 新课标Ⅱ卷(2013–2020)
→ 全国乙卷(2021–2022) → 新课标Ⅱ卷(2023–2024)
```

PDF 二进制 MD5 几乎全不同（不同下载站水印层 / 压缩参数），但**文本内容比对证明考题·答案·解析逐字符相同**，实质命题差异 = **0**。

故按「根据差异性增加词条」原则：**新增词条 = 0**。吉林侧已吃 1999 题词条、5 学科真题缺口 100% 闭合，词条库已完整覆盖黑龙江卷全部考点。
