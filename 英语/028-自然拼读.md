# 自然拼读

> **一句话**：自然拼读（Phonics）就是学会字母和声音的对应关系，看到单词就能拼出读音！
> **English**: Phonics teaches you the link between letters and sounds, so you can read any new word aloud.

---

## 🧒 给 3-6 岁（共读版）

你知道字母 **A** 在 **apple** 里怎么发音吗？

每个字母（或字母组合）都有自己的"声音"：

- **a** → /æ/ → **a**pple（苹果）
- **b** → /b/ → **b**all（球）
- **c** → /k/ → **c**at（猫）
- **d** → /d/ → **d**og（狗）
- **s** → /s/ → **s**un（太阳）
- **t** → /t/ → **t**op（陀螺）

把这些声音拼在一起，就能读出单词！
**c + a + t = cat（猫）**，**d + o + g = dog（狗）**

**共读小活动：** 用磁力字母贴，在冰箱上拼出 cat、dog、big 等简单单词，让孩子指着每个字母发出对应的声音，然后连起来读！

## 📚 给 6-12 岁（自读版）

### 自然拼读的核心：字母 → 声音

**自然拼读（Phonics）** 的关键是记住每个字母（单个或组合）对应的发音：

**单个字母发音（基础）：**

| 字母 | 发音 | 记忆词 |
|------|------|--------|
| a | /æ/ | **a**pple 苹果 |
| b | /b/ | **b**all 球 |
| c | /k/ | **c**at 猫 |
| d | /d/ | **d**og 狗 |
| e | /e/ | **e**gg 鸡蛋 |
| f | /f/ | **f**ish 鱼 |
| g | /g/ | **g**o 去 |
| h | /h/ | **h**at 帽子 |
| i | /ɪ/ | **i**gloo 冰屋 |
| j | /dʒ/ | **j**am 果酱 |
| k | /k/ | **k**ite 风筝 |
| l | /l/ | **l**eg 腿 |
| m | /m/ | **m**ap 地图 |
| n | /n/ | **n**et 网 |
| o | /ɒ/ | **o**range 橙子 |
| p | /p/ | **p**ig 猪 |
| r | /r/ | **r**un 跑 |
| s | /s/ | **s**un 太阳 |
| t | /t/ | **t**op 陀螺 |
| u | /ʌ/ | **u**mbrella 雨伞 |
| v | /v/ | **v**an 面包车 |
| w | /w/ | **w**eb 蜘蛛网 |

| y | /j/ | **y**ellow 黄色 |
| z | /z/ | **z**oo 动物园 |

### CVC 单词——最基础的拼读单位

**CVC = Consonant + Vowel + Consonant（辅音 + 元音 + 辅音）**

这是最简单的单词结构，每个字母都是短元音，直接拼读：

| 单词 | 拆分 | 中文 |
|------|------|------|
| cat | c + a + t | 猫 |
| bat | b + a + t | 蝙蝠 |
| sit | s + i + t | 坐 |
| hop | h + o + p | 跳 |
| run | r + u + n | 跑 |
| red | r + e + d | 红色 |
| big | b + i + g | 大 |
| mud | m + u + d | 泥 |

### 字母组合（Digraphs & Blends）

单个字母掌握后，学字母组合——两个字母合在一起发一个音：

| 字母组合 | 发音 | 例词 |
|---------|------|------|
| sh | /ʃ/ | **sh**ip 船, **sh**ow 展示 |
| ch | /tʃ/ | **ch**air 椅子, **ch**eese 奶酪 |
| th（清音）| /θ/ | **th**ink 想, **th**ree 三 |
| th（浊音）| /ð/ | **th**e 这, **th**is 这个 |
| oo（短）| /ʊ/ | b**oo**k 书, c**oo**k 做饭 |
| oo（长）| /uː/ | m**oo**n 月亮, f**oo**d 食物 |
| ee | /iː/ | s**ee** 看, tr**ee** 树 |
| ai | /eɪ/ | r**ai**n 雨, t**ai**l 尾巴 |
| ou | /aʊ/ | m**ou**se 老鼠, **ou**t 出去 |

### 五大元音规则

PEP 教材在 Let's spell 部分系统讲解了这些规律：

**规则 1：短元音（CVC 词）**
- 元音字母在两个辅音之间，读短音：hat /hæt/, sit /sɪt/

**规则 2：魔法 e（CVCe 词）**
- 单词末尾有不发音的 e，元音读长音：cake /keɪk/, kite /kaɪt/, note /nəʊt/

**规则 3：双元音（Vowel Team）**
- 两个元音字母连在一起，第一个读长音，第二个不发音（"当两个元音手牵手，第一个读名字"）：
  - rain（/reɪn/）, seat（/siːt/）, boat（/bəʊt/）

**规则 4：r 控制元音（R-controlled）**
- 元音 + r，读音改变：car /kɑː/, bird /bɜːd/, corn /kɔːn/

**规则 5：末尾 y**
- 多音节词末尾 y，读 /iː/：happy, funny, sunny

## 🎓 给 12+（进阶版）

### 自然拼读法 vs 国际音标（IPA）——优劣对比

| 维度 | 自然拼读（Phonics） | 国际音标（IPA） |
|------|-----------------|--------------|
| 学习成本 | 低，基于熟悉的字母 | 高，需学新符号体系 |
| 适用范围 | 只适用于英语 | 适用所有语言 |
| 准确性 | 规则有大量例外（sight words）| 精确，无歧义 |
| 母语者惯用 | 是（英语国家母语教学核心）| 否（主要用于外语学习）|
| 非母语者 | 入门快，但遇例外词困难 | 查字典直接准确 |

**结论：** 两者不矛盾，互补使用最佳：
- 先学自然拼读，建立字母-声音联系，大量阅读早期绘本
- 再学音标，查字典自学发音，处理拼读规则的例外词

### 常见例外词（Sight Words）

自然拼读规则不适用于这些高频词，只能整体记忆：

| 单词 | 原因 | 拼读预测 | 实际读音 |
|------|------|---------|---------|
| the | th+e 应读 /ðiː/，但实际弱读 | /ðiː/ | /ðə/ |
| said | ai 应读 /eɪ/ | /seɪd/ | /sed/ |
| have | CVCe 应读长 a | /heɪv/ | /hæv/ |
| come | CVCe 应读长 o | /kəʊm/ | /kʌm/ |
| one | 应读 /ɒn/ | /ɒn/ | /wʌn/ |
| are | 应读 /ær/ | /eər/ | /ɑː/ |

这些词出现频率极高，须通过大量阅读建立"视觉词汇（sight vocabulary）"记忆。

### 英语正字法的历史原因

英语拼写规则复杂，根源在于：
1. **多语言混合**：英语词汇来自古英语、法语、拉丁语、希腊语等，各语言有不同拼写习惯
2. **印刷术固定了拼写**：15 世纪印刷术普及，固定了当时的拼写，但此后发音继续演变（大元音推移，Great Vowel Shift）
3. **国际音标 1888 年才发明**：在此之前数百年，英语一直在没有统一标准的情况下发展

## 📖 相关绘本

- **《Hop on Pop》** — Dr. Seuss 著。经典早期读本，大量 CVC 押韵词，完美配合自然拼读练习。适龄：3-6 岁。（真实存在）

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么 said 不读 /seɪd/ 而是读 /sed/？"** → 可以说："这是英语的'例外词'，它的拼写和发音规则对不上。英语有很多这样的词，是历史原因造成的。这些词我们只能整体记忆，不能按规则推断。遇到例外词别灰心，多读几遍就记住了。"

- 孩子可能问 **"学自然拼读有什么用？"** → 可以说："自然拼读教会你字母和声音的对应，这样遇到没见过的新单词，可以尝试把它'拼读'出来。大约 80-85% 的英语词可以用拼读规则读出来，剩下 15-20% 要记例外词。"

- **共读小活动：** 把一张纸折成三格，写上 C-A-T，让孩子用手指逐个指着字母发音，然后快速连读，感受"拼"的过程。
