# 职业

> **一句话**：人们做不同的工作帮助社会运转——这些工作的名字，英语叫 job 或 occupation。
> **English**: People do different jobs to help society — these are called jobs or occupations in English.

---

## 🧒 给 3-6 岁（共读版）

你认识这些帮助我们的人吗？

- **teacher** — 老师（在学校教我们学习）
- **doctor** — 医生（帮我们看病）
- **nurse** — 护士（在医院照顾病人）
- **police officer** — 警察（保护我们的安全）
- **firefighter** — 消防员（扑灭火灾、救人）
- **driver** — 司机（开车、开公共汽车）
- **cook** — 厨师（做美味的饭菜）
- **farmer** — 农民（种植我们吃的蔬菜和粮食）

**共读小活动：** 和孩子轮流模仿职业动作——伸听诊器当医生，假装灭火当消防员，让孩子猜一猜是什么职业，然后用英语说出来！

## 📚 给 6-12 岁（自读版）

### 核心句型

问别人的职业，用这两个句型：

**问第三方（他/她是做什么的）：**
- **What does he do?** — 他是做什么的？
- **He's a teacher.** — 他是一名老师。
- **What does she do?** — 她是做什么的？
- **She's a nurse.** — 她是一名护士。

**直接问对方：**
- **What do you do?** — 你是做什么的？
- **I'm a doctor.** — 我是一名医生。

注意：职业名词前一般用冠词 **a / an**（an engineer, an artist）。

### 常见职业词汇

| 职业 | 英文 | 说明 |
|------|------|------|
| 老师 | teacher | 学校/培训机构 |
| 医生 | doctor | 诊断和治疗 |
| 护士 | nurse | 护理和照顾 |
| 警察 | police officer | 维持治安 |
| 消防员 | firefighter | 灭火、救援 |
| 农民 | farmer | 种植粮食 |
| 司机 | driver | 驾驶车辆 |
| 厨师 | cook / chef | cook 日常; chef 专业厨师 |
| 清洁工 | cleaner | 清洁卫生 |
| 销售员 | salesperson | 卖货物 |
| 商人 | businessman / businesswoman | 经营生意 |
| 科学家 | scientist | 从事科研 |
| 工程师 | engineer | 设计/建造 |
| 艺术家 | artist | 创作艺术 |
| 飞行员 | pilot | 驾驶飞机 |
| 服务员 | server / waiter / waitress | 餐厅服务 |

### 职业前用 a 还是 an？

- **a teacher**（t 开头，辅音，用 a）
- **an engineer**（e 开头，元音，用 an）
- **an artist**（a 开头，元音，用 an）
- **a doctor**（d 开头，辅音，用 a）

记忆窍门：看后面那个词的**第一个发音**——如果是元音（a/e/i/o/u 的读音），就用 an。

## 🎓 给 12+（进阶版）

### 职业称谓的性别中立化演变

随着社会进步，英语职业名词正朝向**性别中立**方向发展，避免将某职业默认为某性别：

| 旧称（有性别含义） | 现代中立用语 | 说明 |
|-----------|------------|------|
| waiter / waitress | server | 餐厅服务员 |
| stewardess / steward | flight attendant | 空乘人员 |
| fireman | firefighter | 消防员 |
| policeman | police officer | 警察 |
| mailman | mail carrier / postal worker | 邮递员 |
| businessman | businessperson | 商人 |
| chairman | chairperson / chair | 主席 |
| actress | actor | 演员（现通用 actor） |
| salesman | salesperson | 销售员 |

这一变化反映了职场性别平等的进步。许多英语国家的政府文件、媒体报道已普遍采用中立称谓。

### 职业分类（语义场）

英语将职业按领域分类，有助于系统记忆：

- **Healthcare（医疗）**: doctor, nurse, surgeon, pharmacist, dentist, therapist
- **Education（教育）**: teacher, professor, principal, librarian, counselor
- **Public safety（公共安全）**: police officer, firefighter, paramedic, lifeguard
- **Trade & service（商贸服务）**: salesperson, server, cashier, chef, driver
- **STEM（科技）**: scientist, engineer, programmer, architect, mathematician
- **Arts & media（文艺传媒）**: artist, musician, writer, journalist, photographer

### 职业名词的构词法

很多职业词汇有固定后缀，认识后缀可以猜出职业：

| 后缀 | 例词 | 含义 |
|------|------|------|
| -er / -or | teacher, driver, doctor | 做某事的人 |
| -ist | scientist, artist, pianist | 专业从事者 |
| -ian | musician, politician, librarian | 专业领域人 |
| -eer | engineer, volunteer | 从事…职业 |

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么警察叫 police officer 不叫 policeman？"** → 可以说："以前英语用 policeman，但因为警察里有男有女，所以现在大家改用 police officer，更公平，男女警察都可以这样叫。"

- 孩子可能问 **"cook 和 chef 有什么区别？"** → 可以说："cook 是普通厨师，在家里或一般餐厅做饭都可以叫 cook；chef 通常指专业的大厨，比如米其林餐厅的厨师叫 chef。你在家做煎蛋，你就是一个 cook！"

- **共读小活动：** 玩"职业猜猜看"游戏——用英语描述一个职业的特征（比如 "This person helps sick people. They wear white."），让孩子猜是哪个职业并用英文说出来。
