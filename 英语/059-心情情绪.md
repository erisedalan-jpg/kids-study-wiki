# 心情情绪

> **一句话**：高兴、悲伤、愤怒……用英语说出你现在的感受！
> **English**: Happy, sad, angry… learn to express your feelings in English!

---

## 🧒 给 3-6 岁（共读版）

你现在是什么心情？让我们一起学英语说出来！

- **happy** — 高兴（画一个笑脸 ☺）
- **sad** — 难过（画一个哭脸 ）
- **angry** — 生气（眉头皱起来！）
- **tired** — 累了（打哈欠 😪）
- **scared** — 害怕（躲起来！）
- **excited** — 兴奋（跳起来！）

问别人心情就这样说：
- **How do you feel?** — 你感觉怎么样？
- **I feel happy!** — 我很高兴！

**共读小活动：** 做"情绪脸谱"游戏——家长说一个情绪词，孩子用表情表演，看谁模仿得最像！

## 📚 给 6-12 岁（自读版）

### 核心句型

| 句型 | 中文 |
|------|------|
| How do you feel? | 你感觉怎么样？ |
| I feel happy / sad / tired. | 我感到高兴 / 难过 / 疲倦。 |
| Are you OK? | 你还好吗？ |
| What's wrong? | 怎么了？ |
| I'm a little nervous. | 我有点紧张。 |
| Don't worry! / Cheer up! | 别担心！/ 振作起来！ |

### 常见情绪词汇

| 情绪 | 英文 | 用法示例 |
|------|------|---------|
| 高兴 | happy | I feel happy today. |
| 难过 | sad | She looks sad. |
| 生气 | angry | Don't be angry! |
| 疲倦 | tired | I'm so tired after PE. |
| 兴奋 | excited | He's excited about the trip. |
| 害怕 | scared / afraid | I'm scared of thunder. |
| 无聊 | bored | I'm bored — let's play! |
| 惊讶 | surprised | She was surprised by the gift. |
| 紧张 | nervous | I'm nervous before the exam. |
| 骄傲 | proud | Mum is proud of me. |
| 放松 | relaxed | I feel relaxed on weekends. |
| 困惑 | confused | I'm confused by this question. |

### 情绪词的句式结构

情绪词可以用两种方式表达：

**结构 1：I feel + 形容词**
- I feel **happy**. — 我感到高兴。
- I feel **nervous**. — 我感到紧张。

**结构 2：I am / I'm + 形容词**
- I'm **tired**. — 我很累。
- She's **excited**. — 她很兴奋。

两种结构意思相同，都是正确的！

### 情绪 + 原因

可以加 because（因为）说明原因：
- I'm happy **because** it's my birthday. — 我高兴，因为今天是我的生日。
- She's sad **because** she lost her toy. — 她难过，因为她弄丢了玩具。
- He's excited **because** we're going to the zoo. — 他很兴奋，因为我们要去动物园。

## 🎓 给 12+（进阶版）

### Plutchik 情绪轮（Wheel of Emotions）

美国心理学家 Robert Plutchik（1927-2006）提出了著名的**情绪轮理论**，认为人类有 8 种基本情绪，两两对立，并可以混合产生更复杂的情绪：

**8 种基本情绪（Basic emotions）：**
1. **Joy（喜悦）** ↔ Sadness（悲伤）
2. **Trust（信任）** ↔ Disgust（厌恶）
3. **Fear（恐惧）** ↔ Anger（愤怒）
4. **Surprise（惊讶）** ↔ Anticipation（期待）

**混合情绪示例：**
- Joy + Trust = **Love（爱）**
- Joy + Anticipation = **Optimism（乐观）**
- Fear + Surprise = **Awe（敬畏）**
- Anger + Disgust = **Contempt（蔑视）**

### 情绪形容词 vs 情绪名词

| 形容词（I feel…） | 名词（表示情绪本身） |
|---------------|-----------------|
| happy | happiness |
| sad | sadness |
| angry | anger |
| tired | tiredness / fatigue |
| excited | excitement |
| nervous | nervousness / anxiety |
| proud | pride |
| confused | confusion |

名词常用于更正式的表达：
- **The happiness** on her face was clear. — 她脸上的快乐显而易见。
- He felt a wave of **anxiety** before the speech. — 演讲前他感到一阵焦虑。

### 情绪与身体反应的英语表达

英语中有许多描述情绪引发身体反应的表达：
- **My heart is racing.** — 我心跳加速。（excited / nervous）
- **I have butterflies in my stomach.** — 我肚子里有蝴蝶扑腾。（形容紧张）
- **I'm on cloud nine.** — 我开心极了。（euphoric）
- **I'm feeling down.** — 我情绪低落。（sad / depressed）
- **I'm over the moon.** — 我欣喜若狂。（extremely happy）

## 📖 相关绘本

- **《The Way I Feel》** — Janan Cain 著。用生动的图画展示各种情绪（angry, scared, silly, excited 等），适合亲子共读讨论感受。适龄：3-8 岁。（真实存在）
- **《In My Heart》** — Jo Witek 著。心形翻翻书，描述不同情绪如何在心里"感受"，语言优美。适龄：3-6 岁。（真实存在）
- 🚧 推荐情绪管理主题绘本，家长可在绘本馆挑选《菲菲生气了》等系列。

## 🗣️ 家长讲解话术

- 孩子可能问 **"scared 和 afraid 一样吗？"** → 可以说："意思差不多，都是'害怕'。scared 更口语、更常用；afraid 稍正式一些，也常出现在 I'm afraid... 表示遗憾的用法里，比如 'I'm afraid I can't come.' 意思是'恐怕我来不了。'日常说害怕，用 scared 就好。"

- 孩子可能问 **"为什么 excited 是形容词，不是动词？"** → 可以说："excite 是动词（使兴奋），加 -d 变成形容词 excited（感到兴奋）。这类词还有 bored（动词 bore）、surprised（动词 surprise）。它们都是从动词变来的，描述'人的感受'。"

- **共读小活动：** 一起画"情绪日记"——用英文在今天发生的事旁边写下心情词（Today I feel happy because...），帮孩子养成用英语表达内心的习惯。
