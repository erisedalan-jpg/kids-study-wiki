## family name / 姓

**音标**：/ˈfæmɪli neɪm/
**词性**：名词短语
**中文释义**：姓；姓氏

---

## 📚 给 6-12 岁（小学英语回溯版）

在中文里，我们的名字由两部分组成：**姓** + 名。
比如"李明"，"李"是姓，"明"是名。

在英语里：
- **family name**（也叫 **last name** 或 **surname**）= 姓
- **given name**（也叫 **first name**）= 名

**中西方顺序不同！**
- 中文：李（姓）明（名）——姓在前
- 英文：Ming（名）Li（姓）——姓在后

所以 "Li Ming" 的 family name 是 "Li"，given name 是 "Ming"。

---

## 🎓 给 12+（进阶版）

### 三种说法对比

| 术语 | 含义 | 使用场景 |
|------|------|---------|
| family name | 家族姓氏 | 全球通用，正式 |
| last name | 最后那个名字 | 英美口语常用 |
| surname | 传统术语 | 英式英语，正式文件 |

### 填表时的字段顺序
常见表格格式：
- **Last Name / Family Name / Surname**: Li
- **First Name / Given Name**: Ming

> 注意：填英文表格时，即使你是中国人，也需要按西方格式填写，把姓填在 Last Name 栏。

### 常见中国姓氏的英文拼写

| 中文姓 | 普通话拼音（常用英文） |
|--------|----------------------|
| 王 | Wang |
| 李 | Li |
| 张 | Zhang |
| 刘 | Liu |
| 陈 | Chen |

### 西方常见姓氏来源
- **职业来源**：Smith（铁匠）、Baker（面包师）
- **地名来源**：Hill、Brook、Wood
- **父名来源**：Johnson（John's son）、Williams

---

## 例句（中英对照）

1. My family name is Wang. （我姓王。）
2. Please write your last name first on this form. （请在这张表格上先写姓氏。）
3. What's your surname? — It's Chen. （你的姓是什么？——陈。）
4. In China, the family name comes before the given name. （在中国，姓在名之前。）
5. His full name is David Smith — Smith is his family name. （他的全名是大卫·史密斯——史密斯是他的姓。）
