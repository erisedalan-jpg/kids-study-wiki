## given name / 名

**音标**：/ˈɡɪvən neɪm/
**词性**：名词短语
**中文释义**：名（相对于姓而言）；个人名字

---

## 📚 给 6-12 岁（小学英语回溯版）

"李明"这个名字里，"明"就是**名**，在英语里叫 **given name**（也叫 **first name**）。

为什么叫 "given name"？
因为这个名字是父母**给**（given）你的，是你的专属名字！

**中西方比较**：
- 中国人：姓在前，名在后 → 李明（Li Ming）
- 西方人：名在前，姓在后 → John Smith

所以 John Smith 的 given name = John

## 🎓 给 12+（进阶版）

### 术语辨析

| 术语 | 含义 | 使用场景 |
|------|------|---------|
| given name | 被赋予的名字 | 全球通用，正式，书面语 |
| first name | 名字里的第一个词（西方顺序） | 英美日常口语 |
| forename | 同 given name | 英式英语，法律文件 |
| Christian name | 受洗时取的名 | 传统宗教语境，渐少使用 |

### 英语国家的名字文化
- 许多人有 **middle name**（中间名）
  - 例：Barack Hussein Obama → Barack（first）+ Hussein（middle）+ Obama（last）
- 昵称（nickname）常从 given name 派生：
  - William → Will / Bill / Billy
  - Katherine → Kate / Kathy / Kay

### 中国名字的翻译习惯
- 正式场合：拼音全写，如 Li Ming（姓前名后，西方格式）或 LI Ming（姓大写）
- 有时改为西方顺序：Ming Li

---

## 例句（中英对照）

1. My given name is Fang, and my family name is Wang. （我的名是芳，姓是王。）
2. Please enter your first name in the box below. （请在下面的框中输入您的名。）
3. In Western countries, the given name usually comes first. （在西方国家，名通常放在最前面。）
4. Her given name is Emma, but everyone calls her Em. （她的名是艾玛，但大家都叫她 Em。）
5. What's your first name? — It's Tom. （你的名是什么？——汤姆。）
