# 短文改错

> **一句话**：短文改错是高考英语中要求考生在一篇约100词的短文中识别并改正10处语言错误的题型，集中考查词法、句法、语篇连贯及逻辑一致性。
> **English**: Error Correction is a Gaokao task that requires students to detect and correct 10 language errors within a passage, testing mastery of grammar, word usage, sentence structure, and textual coherence.

## 🎓 考点精讲

短文改错以一篇贴近学生生活的记叙文或说明文为载体，设置有10处错误，每行最多一处，涉及三种修改类型：**增添**（缺词处加一个漏字符 ∧，在横线上写出应添的词）、**删除**（多余词加斜线 \，在横线上写出该词并标明删除）、**修改**（错词下划线，在横线上写出正确形式）。试题覆盖高考核心语法项目：

- **动词**：时态、语态误用（如一般过去时误用一般现在时），非谓语动词形式（doing/done/to do 混淆），主谓一致错配。
- **名词、冠词**：可数名词单复数误用，冠词 a/an/the 遗漏或冗余，固定搭配中的冠词错误。
- **代词**：人称代词、物主代词、反身代词指代不一致，it 虚义用法错误。
- **形容词与副词**：修饰对象混淆（形容词修饰名词，副词修饰动词、形容词或全句），比较级与最高级误用。
- **介词与连词**：固定搭配中介词错误，并列连词（and/but/or/so）与从属连词（when/while/though/because 等）的逻辑关系误判。
- **行文逻辑**：上下文语义矛盾（如 however 与 and 互换），指代对象缺失或错位。

**解题步骤**：1. 通读全文把握人称、时态主线；2. 以句子为单位逐句排查，先从句法主干（主谓宾）入手，再查附着成分、虚词；3. 默读验证修改后的行文是否通顺合理。核心策略是“语法为主、语感为辅，固定搭配优先，上下文逻辑兜底”。

**典型应用**：高考中常给出如“如何学好英语”或“一次难忘的经历”类短文，设题点均匀分布在上述语法项目上。例如，一句“I use to get up late”需将 use 改为 used，考查 used to do 的固定句型与时态。

## ⚠️ 高频易错点

- **忽视语境时态一致性**：只看本句时态，不顾全篇基准时态，导致过去与现在时态混用未改出。
- **非谓语与谓语区分不清**：误将分词或不定式当作谓语，如“The book writing by him is popular”未识别出 writing 应为 written。
- **固定搭配及短语格式记忆不准**：如“in Sunday”未改为“on Sunday”；“look forward to do”未改为“looking forward to doing”。
- **逻辑连词误判**：上下文为转折关系时用并列连词 and，或因果关系因 because 与 so 连用未删掉一个等。
