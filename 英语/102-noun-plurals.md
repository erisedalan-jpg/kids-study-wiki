## 名词单复数 / Noun Plurals

**定义**：英语名词用复数形式表示"两个或两个以上"，复数形式有规律变化和不规则变化。

---

## 📚 给 6-12 岁（小学英语回溯版）

中文的"苹果""苹果们"没有什么变化，但英语里数量变了，名词的**形状**也要变！

**最简单的规则：加 -s**：
- book → book**s**（书）
- pencil → pencil**s**（铅笔）
- apple → apple**s**（苹果）

**特殊情况记一记**：
- box → box**es**（盒子）
- dictionary → dictionari**es**（词典）
- child → **children**（孩子，不规则！）

---

## 🎓 给 12+（进阶版）

### 规则变化（可数名词）

| 规则 | 例子 |
|------|------|
| 一般加 -s | book→books, cat→cats |
| 以 s/x/ch/sh/z 结尾加 -es | bus→buses, box→boxes, watch→watches |
| 以辅音+y 结尾，去 y 加 -ies | baby→babies, city→cities |
| 以元音+y 结尾，加 -s | key→keys, day→days |
| 以 f/fe 结尾，去 f/fe 加 -ves | leaf→leaves, knife→knives |
| 以 o 结尾（多数加 -es） | tomato→tomatoes, potato→potatoes |
| 以 o 结尾（少数加 -s） | photo→photos, piano→pianos |

### 不规则复数（必须记！）

| 单数 | 复数 |
|------|------|
| man | men |
| woman | women |
| child | children |
| tooth | teeth |
| foot | feet |
| mouse | mice |
| sheep | sheep（不变！） |
| fish | fish（不变！） |
| deer | deer（不变！） |

### 不可数名词（没有复数！）
不可数名词不加 s，用 some/much/a little 等表示数量：
- water, milk, juice（液体）
- rice, bread（细粒/片状食物）
- music, homework, information

---

## 例句（中英对照）

1. There are three books on my desk. （我桌上有三本书。）
2. I have two dictionaries. （我有两本词典。）
3. The children are playing in the park. （孩子们在公园里玩耍。）
4. I can see some sheep on the hill. （我能看到山上有几只羊。）
5. She has five pencils and two rulers. （她有五支铅笔和两把尺子。）
