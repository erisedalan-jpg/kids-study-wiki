## 不要……！/ Don't... / Please Don't...

**核心句型**：表达禁止和礼貌请求

---

## 📚 给 6-12 岁（自读版）

**Don't...** 就是"不要……"，用来说不允许做的事情。

**两种形式**：
- **Don't + 动词原形**：直接命令（有点强硬）
- **Please don't + 动词原形**：礼貌一些

**常见 Don't 句子**：

| 场景 | 英文 | 中文 |
|------|------|------|
| 教室 | Don't talk in class. | 上课不要说话。 |
| 公共场所 | Don't run here. | 这里不要跑。 |
| 危险警告 | Don't touch it! | 不要碰它！ |
| 礼貌请求 | Please don't be late. | 请不要迟到。 |
| 关心他人 | Don't worry! | 别担心！ |

**结构**：**Don't + 动词原形**（省略了主语 you）

---

## 🎓 给 12+（进阶版）

### 否定祈使句结构

| 形式 | 结构 | 例句 |
|------|------|------|
| 基本 | Don't + V原形 | Don't run. |
| 礼貌 | Please don't + V原形 | Please don't run. |
| Never | Never + V原形 | Never give up!（永不放弃！）|
| No + V-ing | No + 动名词 | No swimming!（禁止游泳！）|

### 公告牌/警示语

| 英文 | 中文 |
|------|------|
| No smoking | 禁止吸烟 |
| No parking | 禁止停车 |
| Do not enter | 禁止进入 |
| Keep off the grass | 请勿踩踏草坪 |

### Don't vs Mustn't vs Can't

| 表达 | 语气强度 | 用途 |
|------|----------|------|
| Please don't... | 最轻（礼貌请求）| 日常礼貌 |
| Don't... | 中等（命令）| 一般规则 |
| You can't... | 中等（禁止）| 规章制度 |
| You mustn't... | 最强（强制禁止）| 正式规定 |

---

## 例句（中英对照）

1. Don't run in the hallway — it's dangerous!（不要在走廊跑——很危险！）
2. Please don't talk when the teacher is speaking.（老师讲课时请不要说话。）
3. Don't forget to do your homework!（不要忘记做作业！）
4. Don't worry — everything will be okay.（别担心——一切都会好的。）
5. Please don't be late for school again.（请不要再迟到了。）
