## 我想成为…… / I Want to Be...

**核心句型**：谈论职业理想

---

## 📚 给 6-12 岁（自读版）

**I want to be a/an...** 就是"我想成为……"——说说你长大后的梦想！

**常用职业**：

| 中文 | 英文 | 句子 |
|------|------|------|
| 医生 | doctor | I want to be a **doctor**. |
| 老师 | teacher | I want to be a **teacher**. |
| 记者 | reporter | I want to be a **reporter**. |
| 演员 | actor/actress | I want to be an **actor**. |
| 科学家 | scientist | I want to be a **scientist**. |
| 宇航员 | astronaut | I want to be an **astronaut**. |
| 工程师 | engineer | I want to be an **engineer**. |

**why 的回答**：
- Because I like helping people.（因为我喜欢帮助别人。）
- Because I think it's interesting.（因为我觉得有趣。）

---

## 🎓 给 12+（进阶版）

### 句型变化（换人称/时态）

| 句型 | 例句 |
|------|------|
| I want to be... | I want to be a doctor.（现在的愿望）|
| I would like to be... | I'd like to be a doctor.（更礼貌）|
| I hope to be... | I hope to be a doctor someday.（希望）|
| My dream is to be... | My dream is to be a doctor.（梦想）|
| When I grow up, I want to be... | When I grow up, I want to be a reporter. |

### 完整理想职业问答

```
A: What do you want to be when you grow up?
B: I want to be a reporter.
A: Why?
B: Because I like writing and I want to travel to different countries.
A: That sounds great! What can you do now to prepare?
B: I can read a lot of books and practice writing every day.
```

### a vs an 用法提醒

- a + 辅音开头：a **d**octor, a **t**eacher, a **r**eporter
- an + 元音开头：an **a**ctor, an **e**ngineer, an **a**stronaut

---

## 例句（中英对照）

1. I want to be a doctor when I grow up.（我长大想当医生。）
2. What do you want to be? I want to be a teacher.（你想成为什么？我想当老师。）
3. My dream is to be a reporter and travel around the world.（我的梦想是成为记者，周游世界。）
4. She wants to be a scientist because she loves math and science.（她想当科学家，因为她热爱数学和科学。）
5. Why do you want to be a doctor? Because I want to help sick people.（你为什么想当医生？因为我想帮助生病的人。）
