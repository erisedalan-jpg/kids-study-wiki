# 不规则动词表（go-went-gone 类，10 高频）

## 基本概念

**不规则动词**是指过去式和过去分词不按照规则（+ed）变化的动词，必须单独记忆。

---

## 📚 6-12 岁自读版

### 10 个高频不规则动词（八上重点）

| 原形 | 中文 | 过去式 | 过去分词 |
|------|------|--------|----------|
| go | 去 | **went** | gone |
| come | 来 | **came** | come |
| see | 看见 | **saw** | seen |
| have | 有/吃 | **had** | had |
| do | 做 | **did** | done |
| eat | 吃 | **ate** | eaten |
| give | 给 | **gave** | given |
| take | 拿/乘坐 | **took** | taken |
| make | 制作 | **made** | made |
| buy | 买 | **bought** | bought |

**记忆小口诀**：
- go/went/gone — 一路向前已消失
- come/came/come — 来了又来
- see/saw/seen — 锯过了，已见过

## 🎓 12+ 进阶版

### 扩展不规则动词（重要补充）

| 原形 | 过去式 | 过去分词 | 中文 |
|------|--------|----------|------|
| be (am/is) | was | been | 是/在 |
| be (are) | were | been | 是/在 |
| keep | kept | kept | 保持 |
| stay | stayed | stayed | 待（规则！） |
| think | thought | thought | 想/认为 |
| find | found | found | 找到 |
| write | wrote | written | 写 |
| read | read /red/ | read /red/ | 读 |
| run | ran | run | 跑 |
| swim | swam | swum | 游泳 |
| get | got | got/gotten | 得到 |
| put | put | put | 放 |
| cut | cut | cut | 切 |
| let | let | let | 让 |
| set | set | set | 放置 |

### 分类记忆法

**AAA 型（三态相同）**：
- put-put-put, cut-cut-cut, let-let-let, read-read-read（注意读音变化）

**ABB 型（过去式=过去分词）**：
- have-had-had, make-made-made, buy-bought-bought

**ABC 型（三态各不同）**：
- go-went-gone, see-saw-seen, give-gave-given
