# 比较级（长词 more）

## 基本概念

多音节形容词（通常为 3 音节以上，部分 2 音节）不能加 -er，而是在前面加 **more** 构成比较级。

---

## 📚 6-12 岁自读版

### 句型结构

**A + is/are + more + 形容词 + than + B**

- English is **more interesting** than I thought. 英语比我想象的更有趣。
- She is **more outgoing** than her sister. 她比她姐姐更外向。

### 常见长词比较级

| 原级 | 比较级 | 中文 |
|------|--------|------|
| outgoing | more outgoing | 更外向 |
| interesting | more interesting | 更有趣 |
| important | more important | 更重要 |
| beautiful | more beautiful | 更美丽 |
| difficult | more difficult | 更难 |
| popular | more popular | 更受欢迎 |
| expensive | more expensive | 更贵 |
| comfortable | more comfortable | 更舒适 |

### 判断用 -er 还是 more

- **1 音节**：一律加 -er（tall→taller）
- **3 音节及以上**：一律加 more（beautiful→more beautiful）
- **2 音节**：大多数加 more，少数可加 -er（clever, simple, quiet 等）

---

## 🎓 12+ 进阶版

### less 是 more 的反义

- **less + 形容词 + than**：表示"不如……，较少……"
- This movie is **less interesting** than the book.
  （这部电影不如书有趣。）

### 注意：不规则变化

| 原级 | 比较级 | 最高级 |
|------|--------|--------|
| good | better（不是 more good） | best |
| bad | worse（不是 more bad） | worst |

### 强调程度

- **much / far / a lot + more + 形容词**：……得多
  - This test is **much more difficult** than the last one.
- **a little / slightly + more + 形容词**：稍微……一些
  - The new version is **a little more expensive**.

## 例句（中英对照）

| 英文 | 中文 |
|------|------|
| She is more outgoing than her brother. | 她比她哥哥更外向。 |
| Math is more difficult than English for me. | 对我来说数学比英语难。 |
| This smartphone is more expensive than that one. | 这款智能手机比那款贵。 |
| Reading is more important than watching TV. | 阅读比看电视更重要。 |
| The garden is more beautiful in spring. | 花园在春天更美丽。 |
