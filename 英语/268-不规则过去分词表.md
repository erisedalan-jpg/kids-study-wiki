# 不规则过去分词表

## 基本概念

现在完成时需要用 **have/has + 过去分词**。规则动词过去分词 = 过去式（+ ed），不规则动词需要单独记忆。

---

## 📚 6-12 岁自读版

### 高频 15 个不规则动词（三态表）

| 原形 | 过去式 | 过去分词 | 中文 |
|------|--------|----------|------|
| go | went | **gone** | 去 |
| come | came | **come** | 来 |
| see | saw | **seen** | 看见 |
| do | did | **done** | 做 |
| have | had | **had** | 有/吃 |
| eat | ate | **eaten** | 吃 |
| give | gave | **given** | 给 |
| take | took | **taken** | 拿/带 |
| make | made | **made** | 制作 |
| write | wrote | **written** | 写 |
| buy | bought | **bought** | 买 |
| get | got | **got/gotten** | 得到 |
| know | knew | **known** | 知道 |
| speak | spoke | **spoken** | 说话 |
| be | was/were | **been** | 是/在 |

### 记忆方法

**AAA 型**（三态相同）：
- cut → cut → cut 切
- let → let → let 让
- put → put → put 放

**ABA 型**（原形 = 过去分词）：
- come → came → **come**
- run → ran → **run**
- become → became → **become**

**ABB 型**（过去式 = 过去分词）：
- make → **made** → made
- buy → **bought** → bought
- have → **had** → had

## 🎓 12+ 进阶版

### 规律分组记忆

**-ought 组**：
- buy → bought → bought
- bring → brought → brought
- think → thought → thought
- fight → fought → fought

**-en 结尾组**：
- take → took → **taken**
- give → gave → **given**
- write → wrote → **written**
- speak → spoke → **spoken**
- break → broke → **broken**

**元音变化规律**（sing-sang-sung 型）：
- sing → sang → **sung**
- ring → rang → **rung**
- swim → swam → **swum**
- drink → drank → **drunk**
- begin → began → **begun**

## 例句（中英对照）

| 英文 | 中文 |
|------|------|
| I have **gone** there before. | 我以前去过那里。 |
| She has **written** three letters. | 她已经写了三封信。 |
| Have you ever **spoken** to him? | 你有没有和他说过话？ |
| He has **bought** a new phone. | 他买了一部新手机。 |
| They have **been** to many countries. | 他们去过许多国家。 |
