## 🎓 character（汉字；字符；性格）

### 词源与词族

**character** 来自希腊语 *kharaktēr*（刻写工具；刻出的标记），本义"刻在物体上的符号"，引申为"文字符号"，再引申为"人的特质/性格"。

| 词形 | 词性 | 含义 |
|------|------|------|
| character | n. | 汉字；字符；性格；人物；角色 |
| character | adj. | 具有特色的（character actor 性格演员）|
| characterize | v. | 以...为特征；描写 |
| characteristic | adj./n. | 特有的；特征 |
| characteristics | n. | 特征（复数）|
| uncharacteristic | adj. | 不寻常的；不典型的 |

### 四大核心含义（Unit 5 语言学重点：含义①②）

**含义 ①：汉字/文字符号（语言学语境）**

| 表达 | 例句 |
|------|------|
| Chinese character | Chinese characters are one of the world's oldest writing systems. |
| write a character | Can you write this character correctly? |
| a complex character | This is a complex character with many strokes. |
| 3,000 characters | One needs to know about 3,000 characters to read a newspaper. |

**含义 ②：字符（计算机/印刷语境）**
- *The password must contain at least 8 characters.*
- *Special characters like @ and # are not allowed.*

**含义 ③：性格；品格**
- *She has a strong character.*（她性格坚强）
- *in character / out of character*（符合/不符合个性）

**含义 ④：（文学/影视）人物/角色**
- *The main character in the novel is a young doctor.*

### 汉字文化背景

| 概念 | 英文表达 |
|------|---------|
| 汉字 | Chinese character / Chinese writing |
| 象形文字 | pictograph / pictographic character |
| 偏旁部首 | radical |
| 笔画 | stroke |
| 繁体字 | traditional Chinese character |
| 简体字 | simplified Chinese character |
| 拼音 | Pinyin (romanization) |
