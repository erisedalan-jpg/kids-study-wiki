# that 与 which 的区别（高中）

> **一句话**：在定语从句中，that 与 which 都可引导限制性定语从句，但有场合偏好；which 还可引导非限制性定语从句，that 不可以。
> **English**: In relative clauses, both "that" and "which" can introduce restrictive clauses, but there are preferred contexts; only "which" can introduce non-restrictive clauses.

---

## 🎓 规则·例句对照·高考考点

### 核心区别总表

| 项目 | that | which |
|------|------|-------|
| 限制性定语从句 | ✅ 可以 | ✅ 可以 |
| 非限制性定语从句 | ❌ 不可以 | ✅ 可以 |
| 引导整句内容（逗号which） | ❌ | ✅ |
| 介词后 | ❌ 不可以 | ✅ 可以（in which / for which）|

### 必须用 that 的情况

| 情况 | 说明 | 例句 |
|------|------|------|
| 先行词既含人又含物 | both... and... / people and things | He talked of the people and things **that** he missed. |
| 先行词有最高级修饰 | the best, the worst, the only... | This is the best book **that** I've read. |
| 先行词有 the only / the very / the same 修饰 | 强调唯一性 | She's the only person **that** can help. |
| 先行词是不定代词 | all / much / everything / nothing / anything | Is there anything **that** I can do? |
| 主句已有 who / which，避免重复 | 为避免混淆 | Who is the man **that** knocked at the door? |

### 必须用 which 的情况

| 情况 | 说明 | 例句 |
|------|------|------|
| 非限制性定语从句 | 有逗号隔开 | The book, **which** she recommended, was wonderful. |
| 介词后 | in/at/for/by which | This is the town in **which** he was born. |
| 引导整句的内容 | 逗号 + which | He passed the exam, **which** surprised everyone. |

### 高考语法填空解题口诀
- 逗号后 → which
- 介词后 → which
- 最高级/only/all/everything → that
- 人 + 物混合先行词 → that

### 中英例句对照

| 中文 | English |
|------|---------|
| 这是我读过的最有趣的书。 | This is the most interesting book **that** I have ever read. |
| 她推荐的那本书非常精彩。 | The book, **which** she recommended, was wonderful. |
| 我想要所有我能得到的帮助。 | I need all the help **that** I can get. |
| 他住过的那个小镇如今已发展成大城市。 | The small town in **which** he lived has now grown into a big city. |
| 他通过了考试，这让所有人都感到惊讶。 | He passed the exam, **which** surprised everyone. |
| 她是唯一能解决这个问题的人。 | She is the only one **that** can solve this problem. |
