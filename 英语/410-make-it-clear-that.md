# make it clear that...

> **一句话**：make it clear that... 是"使某事变得清晰/明确"的固定句型，是 make + it + 形容词/名词 + that 从句这一形式宾语结构的典型代表。
> **English**: "Make it clear that..." means "to make it obvious/explicit that..." — a classic example of the "make + it + adj. + that-clause" formal object structure.

---

## 🎓 规则·例句对照·高考考点

### 结构说明

**make it clear that...** = 使（它）清楚，即"表明……；阐明……；明确指出……"

| 成分 | 内容 |
|------|------|
| make | 使（主要动词）|
| it | 形式宾语 |
| clear | 宾语补足语（形容词）|
| that 从句 | 真正的宾语（所表明的内容）|

### 同结构常见组合

| 句型 | 含义 |
|------|------|
| make it clear that... | 表明……；阐明…… |
| make it possible (for sb.) to do | 使（某人）有可能做…… |
| make it necessary to do | 使……有必要 |
| make it easy/difficult to do | 使……容易/困难 |
| make it a rule to do | 养成……的习惯（规则）|
| make it a point to do | 特别注意做…… |

### 与 find it 对比

| 动词 | 侧重 | 示例 |
|------|------|------|
| make | 主动使某状态成立 | Technology **makes** it easy to learn. |
| find | 发现/感受到某状态 | I **find** it easy to learn with technology. |

### 高考考点
- make it clear that 后接的 that 从句用**陈述语气**（非虚拟）
- 高考写作提分句：I would like to make it clear that our school values both academic achievement and character development.
- 注意：make it + 名词 + to do：make it a rule to exercise every morning（养成每天运动的习惯）

### 中英例句对照

| 中文 | English |
|------|---------|
| 他明确表示他不会改变自己的决定。 | He made it clear that he would not change his decision. |
| 老师向学生阐明了诚实的重要性。 | The teacher made it clear to the students that honesty is of great importance. |
| 网络使人们在家学习成为可能。 | The Internet makes it possible for people to study at home. |
| 她养成了每天阅读半小时的习惯。 | She made it a rule to read for half an hour every day. |
| 政府表明将加大对教育的投入力度。 | The government made it clear that it would increase investment in education. |
| 请注意，迟交作业将影响期末成绩。 | I want to make it clear that late submissions will affect your final grade. |
