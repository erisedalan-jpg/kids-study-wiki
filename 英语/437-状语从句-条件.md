## 🎓 条件状语从句

**定义**：由从属连词引导，表示主句动作成立所需条件的从句。分为真实条件句和非真实条件句（虚拟语气）。

### 常用引导词

| 连词 | 含义 | 说明 |
|------|------|------|
| if | 如果 | 最常用 |
| unless | 除非（= if...not） | — |
| as long as / so long as | 只要 | 强调唯一条件 |
| provided/providing (that) | 条件是；假如 | 较正式 |
| on condition that | 在……条件下 | 正式书面语 |
| in case | 万一；以防 | 预防性条件 |
| suppose/supposing (that) | 假设 | 口语/书面均可 |

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 如果明天天气好，我们就去野餐。 | If the weather is fine tomorrow, we'll go for a picnic. |
| 除非他道歉，否则我不会原谅他。 | I won't forgive him unless he apologizes. |
| 只要你努力，你就能成功。 | As long as you work hard, you will succeed. |
| 假如你是我，你会怎么做？ | Suppose you were me, what would you do? |
| 带把伞以防下雨。 | Take an umbrella in case it rains. |
| 条件是你按时完成工作。 | On condition that you finish the work on time. |

### 高考考点

1. **时态规则**：真实条件句中，从句不用将来时（用一般现在时替代）
   - ✅ If it rains tomorrow, we'll stay home. ❌ if it will rain
2. **unless = if...not**（注意语义等价，但语气有细微差别）
3. **in case 的用法**：in case + 从句（从句用现在时/should + 原形），表示预防

### 易错点

- ❌ If it will be sunny tomorrow, I'll go out. → ✅ If it is sunny tomorrow...
- unless 后面不再加 not：❌ unless he doesn't come → ✅ unless he comes（unless 本身含否定）
