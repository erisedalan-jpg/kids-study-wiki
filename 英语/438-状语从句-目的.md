## 🎓 目的状语从句

**定义**：由从属连词引导，表示主句动作目的的从句。区别于结果状语从句：目的从句说明"为了什么"，结果从句说明"导致了什么"。

### 常用引导词

| 连词 | 含义 | 说明 |
|------|------|------|
| so that | 为了；以便 | 从句中常含 can/could/may/might/will/would |
| in order that | 为了（较正式） | 同上，比 so that 正式 |
| for fear that | 唯恐；以防 | 从句中常含 should |
| lest | 唯恐；以免（古/书面语） | 从句中用 should 或虚拟语气 |
| in case | 以防；万一 | — |

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 她起得很早，以便赶上第一班公共汽车。 | She got up early so that she could catch the first bus. |
| 他努力学习是为了能考上名牌大学。 | He studied hard in order that he might enter a famous university. |
| 他把钱藏了起来，以防被盗。 | He hid the money for fear that it might be stolen. |
| 他低声说话，唯恐吵醒孩子。 | He spoke quietly lest he should wake the child. |
| 她带了备用电池以防手机没电。 | She brought extra batteries in case her phone ran out of power. |
| 我们需要在截止日期前完成任务，以便团队按时交付。 | We need to finish the task before the deadline so that the team can deliver on time. |

### 高考考点

1. **so that 目的从句 vs so that 结果从句**（核心区分）：
   - 目的：so that + 情态动词（can/may/will）
   - 结果：so + 形容词/副词 + that...（so...that 结构）
2. **in order that vs in order to**：in order that 引导从句；in order to 后接不定式（两者常可转换）
3. **lest 的虚拟语气**：lest + should + 动词原形（should 有时省略）

### 易错点

- ❌ He studied hard so that he studied well.（目的从句必须含情态动词）
- 注意区分 so that（目的/结果）和 so...that（结果）：so + adj. + that
