## 🎓 结果状语从句

**定义**：由从属连词引导，表示主句动作或状态所产生结果的从句。

### 常用结构

| 结构 | 含义 | 说明 |
|------|------|------|
| so + adj./adv. + that | 如此……以至于 | so 修饰形容词或副词 |
| such + (a/an) + adj. + n. + that | 如此……的……以至于 | such 修饰名词短语 |
| so that | 所以；因此 | 无 so...that 强调结构，语气较弱 |

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 这道题太难了，以至于没有人能解出来。 | The question was so difficult that nobody could solve it. |
| 她跑得如此之快，以至于我根本追不上她。 | She ran so fast that I couldn't catch up with her at all. |
| 那是如此精彩的一场演出，以至于观众起立鼓掌。 | It was such a wonderful performance that the audience stood up and applauded. |
| 天气非常寒冷，以至于河水都结冰了。 | It was so cold that the river froze. |
| 他学习非常刻苦，结果考上了重点大学。 | He studied so hard that he got into a key university. |
| 这本书非常有趣，以至于我一口气读完了。 | The book was so interesting that I read it in one sitting. |

### 高考考点

1. **so...that vs such...that 互换规则**：
   - so + adj. + a/an + 单数名词 + that = such + a/an + adj. + 单数名词 + that
   - so + adj. + 不可数/复数名词 + that = such + adj. + 不可数/复数名词 + that
   - 例：so heavy a box = such a heavy box
2. **so...that（结果）vs so that（目的）**：有 so + adj./adv. 的是结果从句
3. **结果从句 vs 目的从句**：结果从句无情态动词（或用过去时），目的从句含 can/may/will

### 易错点

- ❌ such difficult that → ✅ so difficult that（difficult 是形容词，用 so）
- ❌ so a difficult problem that → ✅ such a difficult problem that / so difficult a problem that
