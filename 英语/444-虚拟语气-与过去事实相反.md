## 🎓 与过去事实相反的虚拟语气

**定义**：表达与过去事实相反的假设，暗含"当时其实……并非如此/没有发生"。

### 结构公式

```
条件从句（if）：if + 主语 + had + 过去分词（过去完成时）
主句：主语 + would/could/should/might + have + 过去分词
```

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 如果你昨天来了，你就会看到那场精彩的演出。 | If you had come yesterday, you would have seen the wonderful show. |
| 如果他当时听从了建议，就不会失败了。 | If he had taken the advice, he wouldn't have failed. |
| 要是我当时没有犯那个错误就好了！ | If only I hadn't made that mistake! |
| 如果你那时更加努力，你可能已经通过了考试。 | If you had worked harder then, you might have passed the exam. |
| 他表现得好像他见过那个人一样（其实没见过）。 | He behaved as if he had met that person before. |
| 如果我知道这件事，我就会通知你的。 | If I had known about this, I would have informed you. |

### 特殊变体（倒装）

| 原结构 | 倒装结构 |
|--------|---------|
| If I had known... | Had I known...（省略 if，had 提前）|
| If he had been there... | Had he been there... |

### 高考考点

1. **倒装结构**：Had + 主语 + 过去分词（= If + 主语 + had + 过去分词），是高考写作高分句型
2. **wish + 过去完成时**：I wish I had studied harder.（对过去的遗憾）
3. **混合虚拟（时间错位）**：条件从句（过去）+ 主句（现在）
   - If I had taken the medicine yesterday, I would feel better now.

### 易错点

- ❌ If I would have known... → ✅ If I had known...（条件从句不用 would，用 had done）
- 注意区分时态："If I had come"（过去虚拟）vs "If I came"（现在虚拟）
