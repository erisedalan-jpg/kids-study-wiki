## 🎓 虚拟语气入门

**定义**：虚拟语气（Subjunctive Mood）是一种动词形式，用于表达说话人认为与事实相反、假设、愿望、建议或命令等非真实情况。与陈述语气（表示事实）相对。

### 虚拟语气三大类型概览

| 类型 | 与何时相反 | 条件从句形式 | 主句形式 |
|------|-----------|------------|---------|
| 与现在相反 | 现在 | if + 过去式（be 用 were） | would/could/might + do |
| 与过去相反 | 过去 | if + 过去完成时（had done） | would/could/might + have done |
| 与将来相反 | 将来（可能性小） | if + should/were to + do | would/could/might + do |

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 如果我是你，我就不会这么做。（与现在相反） | If I were you, I wouldn't do that. |
| 如果我昨天去了，我就能见到他了。（与过去相反） | If I had gone there yesterday, I could have met him. |
| 我希望我现在是在海边。（愿望） | I wish I were at the seaside now. |
| 老师建议我们每天练习写作。（建议/命令） | The teacher suggested that we (should) practice writing every day. |
| 要是他明天能来就好了。（与将来相反） | If he were to come tomorrow, I would be very happy. |
| 好像他从未见过这件事一样。（as if） | He talked about it as if he had never seen it before. |

### 高考考点

1. **be 动词在虚拟语气中统一用 were**（包括第一、三人称单数）
   - ✅ If I were rich... / If he were here...
2. **建议/命令/要求动词后的宾语从句**：suggest/demand/insist/require + that + 主语 + (should) + 动词原形
3. **wish、if only、as if/as though** 引导的虚拟语气（见各专项词条）

### 易错点

- ❌ If I was you → ✅ If I were you（虚拟语气中 be 统一用 were）
- 注意区分虚拟语气和假设：If I am wrong, please correct me.（这是真实条件句，不是虚拟）
