# It is time to do...

> **一句话**：It is time to do... 表示"是时候做某事了"，强调某事应当立即发生，语气带有督促或提醒意味。
> **English**: "It is time to do..." means it is the appropriate or necessary moment to do something — often used to urge action.

---

## 🎓 规则·例句对照·高考考点

### 结构说明

| 结构 | 说明 |
|------|------|
| It is time to do sth. | 基本结构，to do 为不定式，表示"该做某事了" |
| It is time for sb. to do sth. | 加逻辑主语，明确是谁该做这件事 |
| It is time for sth. | 接名词，表示"是某事物的时候了" |
| It is high time to do / that... | 强调语气加强版（详见 [[446-It-is-high-time-that|It-is-high-time-that]]）|

### 与 It is high time 对比

| 句型 | 语气 | 特点 |
|------|------|------|
| It is time to do. | 中性，提醒 | 最常用，接不定式 |
| It is time that sb. did. | 带虚拟语气 | that 从句用过去式 |
| It is high time that sb. did. | 强烈，指责或催促 | 虚拟语气，语气最强 |

### 高考考点
- **虚拟语气**：It is time that sb. **did**（不是 does）——该从句用一般过去时表示虚拟
- **区分 to do / for sb. to do**：无需指出谁做时用 to do；需指出时用 for sb. to do
- **写作加分**：用于议论文结尾，表达行动呼吁，层次感强

### 中英例句对照

| 中文 | English |
|------|---------|
| 是时候开始复习了。 | It is time to start reviewing. |
| 是时候我们采取行动了。 | It is time for us to take action. |
| 现在是休息的时候了。 | It is time for a rest. |
| 该叫孩子们吃饭了。 | It is time for the children to have dinner. |
| 是时候政府解决这个问题了。（虚拟） | It is time the government dealt with this issue. |
| 是时候改变我们的生活方式了。 | It is time to change our way of life. |
