# as long as（条件）

> **一句话**：as long as 引导条件状语从句，意为"只要……（就……）"，表示某行为发生的唯一条件，相当于 if only / provided that / on condition that。
> **English**: "As long as" introduces a conditional clause meaning "provided that" or "on the condition that" — the main clause will be true only if the condition holds.

---

## 🎓 规则·例句对照·高考考点

### 结构说明

| 结构 | 说明 |
|------|------|
| As long as + 条件从句, 主句 | 基本结构，条件从句放句首时加逗号 |
| 主句 + as long as + 条件从句 | 条件从句放句末时不加逗号 |

### 同义词辨析

| 连词 | 含义 | 语体 |
|------|------|------|
| as long as | 只要 | 中性，最常用 |
| so long as | 只要 | 稍正式，与 as long as 同义 |
| provided (that) | 如果，前提是 | 正式/书面 |
| providing (that) | 如果，前提是 | 书面 |
| on condition that | 在……条件下 | 正式 |
| if only | 只要，要是……就好了 | 带愿望色彩 |

### 时态规则（与 if 从句相同）

| 主句时态 | 条件从句时态 | 情境 |
|----------|------------|------|
| 一般将来时 | 一般现在时 | 表将来可能发生的条件 |
| 祈使句 / 情态动词 | 一般现在时 | 给出建议/许可 |
| 一般过去时 | 一般过去时 | 过去的条件关系 |

### 注意：as long as 的时间用法
- as long as 也可表示时间，意为"长达……之久"（非条件）
- 条件用法：**As long as** you work hard, you will succeed.（只要努力就会成功）
- 时间用法：I have known her **as long as** ten years.（认识她长达十年）
- 高考中需根据语境判断是条件还是时间含义

### 高考考点
- **时态**：条件从句中不用将来时，用现在时代替将来（主将从现规则）
- **与 so long as 互换**：两者同义，高考填空中可互相替代
- **与 only if 区分**：only if 也表条件，但语气更强，含排他性（Only if you apologize will I forgive you.）
- **写作加分**：替换单调的 if，使条件表达更地道

### 中英例句对照

| 中文 | English |
|------|---------|
| 只要你努力，就一定会成功。 | As long as you work hard, you will surely succeed. |
| 只要天气好，我们就去郊游。 | We will go on a picnic as long as the weather is fine. |
| 只要你遵守规则，你就可以使用这个设备。 | As long as you follow the rules, you can use the equipment. |
| 只要我们团结一致，就没有什么困难能打倒我们。 | As long as we stand together, no difficulty can defeat us. |
| 只要有信心，梦想终将实现。 | As long as you have confidence, your dreams will come true. |
| 只要坚持锻炼，健康就有保障。 | So long as you keep exercising, your health will be guaranteed. |
