# so that（目的）

> **一句话**：so that 引导目的状语从句，意为"以便……/为了……"，从句中常含情态动词 can/could/may/might/will/would，表示主句动作的目的。
> **English**: "So that" introduces a purpose clause meaning "in order that" — the clause typically contains a modal verb (can, may, will, etc.) to show the intended goal of the main clause action.

---

## 🎓 规则·例句对照·高考考点

### 结构说明

| 结构 | 含义 | 示例 |
|------|------|------|
| 主句 + so that + 从句（含情态动词） | 目的 | She spoke slowly so that we could understand. |
| 主句 + so + adj./adv. + that + 从句 | 结果（详见 [[453-such-that-结果|such-that-结果]]） | She spoke so slowly that we fell asleep. |

> 注意：**so that（目的）** 与 **so...that（结果）** 形式相近，必须区分！

### 目的与结果的核心区分

| 特征 | so that（目的） | so...that（结果） |
|------|----------------|-----------------|
| 从句含情态动词 | 通常有（can/may/will/could...） | 通常没有 |
| 逻辑关系 | 主动安排的目的 | 自然导致的结果 |
| so 的位置 | so 直接接 that | so 修饰形容词/副词 |
| 示例 | so that she **can** hear | so loud **that** she heard |

### 常用情态动词搭配

| 时态情境 | 情态动词 | 示例 |
|----------|---------|------|
| 表将来目的 | can / will / may | Study hard so that you can pass. |
| 表过去目的 | could / would / might | He ran fast so that he could catch the bus. |
| 正式/书面 | may / might | She left early so that she might arrive on time. |

### 同义替换

| 表达 | 语体 | 示例 |
|------|------|------|
| so that | 常用 | He saved money so that he could travel. |
| in order that | 正式 | He saved money in order that he could travel. |
| in order to do | 常用，接不定式 | He saved money in order to travel. |
| to do | 最简洁 | He saved money to travel. |

### 高考考点
- **目的 vs 结果**：so that（目的）vs so...that（结果）是高频混淆考点
- **情态动词标志**：目的从句几乎总含情态动词，是判断的关键线索
- **时态一致**：主句过去时，从句用 could/would/might（不用 can/will/may）
- **写作运用**：议论文中用 so that 阐明行动目的，逻辑严密

### 中英例句对照

| 中文 | English |
|------|---------|
| 她早起，以便赶上第一班公共汽车。 | She got up early so that she could catch the first bus. |
| 请大声说话，以便大家都能听到。 | Please speak loudly so that everyone can hear you. |
| 他努力存钱，为了将来能去旅行。 | He worked hard to save money so that he could travel in the future. |
| 老师重复了好几遍，以便学生们能理解。 | The teacher repeated it several times so that the students could understand. |
| 我们应该保护环境，以便后代能够生活在美丽的地球上。 | We should protect the environment so that future generations can live on a beautiful earth. |
| 他把笔记写得很清楚，以便复习时方便查阅。 | He wrote his notes clearly so that he could refer to them easily when reviewing. |
