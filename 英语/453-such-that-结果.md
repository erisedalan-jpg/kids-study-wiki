# such...that / so...that（结果）

> **一句话**：such...that 和 so...that 均引导结果状语从句，意为"如此……以至于……"；such 修饰名词，so 修饰形容词或副词，二者不可混淆。
> **English**: "Such...that" and "so...that" both introduce result clauses meaning "so much/many/adj that..." — the key difference is that "such" modifies nouns while "so" modifies adjectives or adverbs.

---

## 🎓 规则·例句对照·高考考点

### 核心结构对比

| 结构 | 修饰对象 | 公式 | 示例 |
|------|---------|------|------|
| so + adj./adv. + that | 形容词 / 副词 | so + adj + that | The music was **so beautiful** that I cried. |
| such + (a/an) + adj. + n. + that | 名词（含修饰语） | such + a/an + adj. + n. + that | It was **such a beautiful song** that I cried. |
| such + adj. + 复数n./不可数n. + that | 复数或不可数名词 | such + adj. + n. + that | They were **such lovely flowers** that I bought them all. |

### such...that 详细结构

| 名词类型 | 结构 | 示例 |
|----------|------|------|
| 可数单数名词 | such + **a/an** + adj. + n. + that | such a hard question that... |
| 可数复数名词 | such + adj. + n.(pl.) + that | such good students that... |
| 不可数名词 | such + adj. + n.(unc.) + that | such bad weather that... |
| 无形容词修饰 | such + **a/an** + n. + that | such a surprise that... |

### so...that 特殊结构

| 结构 | 示例 | 说明 |
|------|------|------|
| so + adj./adv. + that | so fast that... | 最基本 |
| so + many/much/few/little + n. + that | so many books that... | 修饰数量词 |
| So + adj. + is/was + 主语 + that... | So cold was it that... | 倒装（强调） |

### 与 so that（目的）的区分

| 特征 | so that（目的） | so...that（结果） |
|------|----------------|-----------------|
| so 的位置 | 直接接 that | so 与 that 之间有形/副词 |
| 从句情态动词 | 几乎必有 | 通常没有 |
| 语义 | 主动目的 | 自然结果 |
| 逻辑测试 | "为了" | "以至于" |

### 高考考点
- **such vs so 选择**：such 接名词（需注意冠词），so 接形容词/副词——这是高频考点
- **倒装结构**：so + adj. + 助动词 + 主语 + that...（So cold was the weather that...）
- **冠词位置**：such a beautiful girl（a 在 such 之后，beautiful 之前）
- **写作提升**：用 such...that / so...that 扩展句子，增加句式多样性

### 中英例句对照

| 中文 | English |
|------|---------|
| 这道题如此难，以至于没有学生能解出来。 | The question was so difficult that no student could solve it. |
| 他如此努力地工作，以至于忘记了吃饭。 | He worked so hard that he forgot to eat. |
| 这是一部如此精彩的电影，以至于我看了两遍。 | It was such an excellent film that I watched it twice. |
| 天气如此恶劣，以至于航班被取消了。 | The weather was such that all flights were cancelled. |
| 她讲得如此清楚，以至于每个人都听懂了。 | She explained so clearly that everyone understood. |
| 有如此多的学生参加，以至于教室都坐不下了。 | There were so many students that the classroom couldn't hold them all. |
| 如此寒冷，以至于河流都结冰了。（倒装） | So cold was it that the river froze over. |
