## 🎓 nutrition（营养；营养学）

**词性与释义**

| 词性 | 释义 |
|------|------|
| n. | 营养；营养学；营养物质 |
| adj. (nutritious) | 有营养的，营养丰富的 |
| adj. (nutritional) | 营养的（属性描述）|
| n. (nutrient) | 营养素，营养物质 |
| n. (nutritionist) | 营养学家 |

**核心语义**：指摄取食物并将其用于生长和健康的过程，或食物中所含的营养成分。

---

### 例句（中英对照）

| 英文例句 | 中文翻译 |
|----------|----------|
| Good nutrition is essential for a healthy life. | 良好的营养对健康生活至关重要。 |
| Rice provides basic nutrition for billions of people. | 水稻为数十亿人提供基本营养。 |
| The nutritional value of vegetables is very high. | 蔬菜的营养价值非常高。 |
| A nutritionist can help you plan a balanced diet. | 营养师可以帮助你制定均衡饮食计划。 |
| Children need proper nutrition for healthy growth. | 儿童需要适当的营养来健康成长。 |
| Many people lack basic nutrition due to poverty. | 许多人因贫困而缺乏基本营养。 |

---

### 高考考点

1. **词形变化**：nutrition (n.) → nutritious (adj. 有营养的) → nutritional (adj. 营养的) → nutrient (n. 营养素) → nutritionist (n. 营养学家)
2. **常见搭配**：
   - good/proper nutrition 良好/适当的营养
   - nutritional value 营养价值
   - nutritious food/meal 有营养的食物/饮食
   - lack of nutrition 营养不足
3. **Unit 5 关联**：Working the Land 单元，粮食安全与人类营养话题
4. **写作**：健康、饮食、粮食安全类作文的核心词汇

### 易错点

- **辨析**：nutritious（有营养的，修饰食物）vs nutritional（营养的，修饰属性/价值）
  - ✓ nutritious food；✓ nutritional value；❌ nutritious value
- nutrition 不可数：✓ good nutrition；❌ a good nutrition
- nutrient（营养素）是可数名词：vitamins and other nutrients
