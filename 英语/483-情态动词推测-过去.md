## 🎓 情态动词推测（过去）

**定义**：用情态动词 + have + 过去分词对已发生的过去事件进行推测，表达不同程度的确定性。

**基本结构**：情态动词 + have + done（过去分词）

---

### 推测程度对比表

| 情态动词 | 结构 | 含义 | 确定性 |
|---------|------|------|--------|
| must | must have done | 一定（做了）…（肯定推测）| 90%+ |
| can't / couldn't | can't have done | 不可能（做了）…（否定推测）| 90%+ |
| may | may have done | 可能（做了）…（较不确定）| ~50% |
| might | might have done | 也许（做了）…（更不确定）| ~30% |
| could | could have done | 本可以（做了）…（未实现的可能）| 条件性 |
| should | should have done | 本应该（做了）但没做（遗憾/批评）| 非推测 |

---

### 例句（中英对照）

| 英文例句 | 中文翻译 |
|----------|----------|
| She must have worked very hard to achieve this. | 她一定付出了非常努力才取得这个成就。 |
| He can't have made such a careless mistake. | 他不可能犯了这么粗心的错误。 |
| They may have already left when we arrived. | 我们到达时，他们可能已经离开了。 |
| The key might have fallen out of your pocket. | 钥匙可能从你口袋里掉出来了。 |
| You should have told me earlier about this problem. | 你本应该早点告诉我这个问题的。 |
| She could have won if she had tried harder. | 如果她更努力，她本可以赢的。 |

---

### 高考考点

1. **must have done vs should have done**（高频易混）：
   - must have done：对过去的肯定推测（一定做了）
   - should have done：表遗憾/批评（本应该做但没做）
2. **语法填空**：根据语境和上下文选择正确情态动词形式
3. **阅读理解**：准确理解作者对过去事件的推测态度（确定/不确定）
4. **写作**：推测他人行为或历史事件时的高级表达

### 易错点

- **关键区别**：should have done ≠ must have done（前者含批评/遗憾，非推测）
- 否定推测：must not have done（一定没做）vs can't have done（不可能做了）— 两者近义但 can't 更常用
- 完成式：情态动词 + have done（过去推测）vs 情态动词 + do（现在/将来推测）
