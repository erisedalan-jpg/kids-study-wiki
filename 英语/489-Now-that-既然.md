## 🎓 Now that...（既然…）

**定义**：连词短语，引导原因状语从句，表示"既然"（因为某件已知/已发生的事），含有"现在情况已经如此，所以…"的语气。

**基本结构**：
```
Now that + 从句，主句
主句 + now that + 从句
```

---

### Now that vs Since vs As 对比

| 连词 | 语气 | 侧重点 | 例子 |
|------|------|--------|------|
| now that | 既然（现在/已知情况）| 说话时新出现或变化的事实 | Now that you're here, let's begin. |
| since | 既然/因为 | 已知的、双方都了解的原因 | Since you know the answer, tell me. |
| as | 由于（原因较弱）| 次要原因，常在句首 | As it was raining, we stayed inside. |
| because | 因为（最强）| 新信息，强调原因 | I left because I was tired. |

---

### 例句（中英对照）

| 英文例句 | 中文翻译 |
|----------|----------|
| Now that you've graduated, you should look for a job. | 既然你已经毕业了，你应该找工作了。 |
| Now that the harvest is over, the farmers can rest. | 既然收成结束了，农民们可以休息了。 |
| Now that science has advanced so much, many diseases can be cured. | 既然科学已如此进步，许多疾病可以被治愈。 |
| Now that we understand the situation, we can make better decisions. | 既然我们了解了情况，我们就能做出更好的决定。 |
| Now that the panda population is growing, conservationists are optimistic. | 既然大熊猫数量在增加，自然保护主义者感到乐观。 |
| Now that you mention it, I do remember seeing that gesture before. | 既然你提起了，我确实记得以前见过那个手势。 |

---

### 高考考点

1. **位置灵活**：now that 从句可置于主句前或后
   - **句首**（常见）：Now that..., 主句（逗号分隔）
   - **句末**：主句 + now that 从句（无逗号）
2. **时态搭配**：now that 从句通常用现在时（现在完成时），主句可用现在时或将来时
3. **阅读辨析**：区分 now that（既然）和 now（现在）+that 从句（不同结构）
4. **写作**：在议论文中引出结论或建议的高级连词

### 易错点

- **now that ≠ now + that**：now that 是固定连词，不可拆分
- now that 表"既然"，含有已知前提，比 because 更强调前提已成立
- 不要与 although（虽然）混淆——now that 是原因，although 是让步
