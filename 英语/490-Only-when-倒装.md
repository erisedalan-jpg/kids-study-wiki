## 🎓 Only when...（Only 引导的倒装句）

**定义**：当 only 置于句首，修饰时间/条件/方式状语从句时，主句发生部分倒装（助动词/情态动词/be 动词提前）。

**基本结构**：
```
Only when/if/after/by... + 从句，+ 助动词/情态动词/be + 主语 + 动词原形
```

---

### Only 倒装句型全家族

| 句式 | 例子 | 说明 |
|------|------|------|
| Only when | Only when you fail do you learn. | 只有当你失败时，你才能学习 |
| Only if | Only if you work hard will you succeed. | 只有努力，你才能成功 |
| Only after | Only after the harvest did they celebrate. | 收割后他们才庆祝 |
| Only by | Only by working together can we achieve this. | 只有合作才能实现 |
| Only then | Only then did I understand the truth. | 直到那时我才明白真相 |
| Only in this way | Only in this way can the problem be solved. | 只有这样才能解决问题 |

---

### 例句（中英对照）

| 英文例句 | 中文翻译 |
|----------|----------|
| Only when you make mistakes can you truly learn. | 只有当你犯错时，你才能真正学习。 |
| Only when all the data was collected did the scientists begin their analysis. | 只有当所有数据都收集完毕，科学家们才开始分析。 |
| Only if we protect wildlife can we preserve biodiversity. | 只有保护野生动物，我们才能保护生物多样性。 |
| Only by dedicating oneself to a goal can one achieve great things. | 只有全身心投入目标，才能取得伟大成就。 |
| Only then did I realise the importance of body language. | 直到那时我才意识到肢体语言的重要性。 |
| Only in this way can agricultural technology benefit everyone. | 只有这样，农业技术才能造福所有人。 |

---

### 高考考点

1. **倒装规则**（核心）：only 提前 → 主句助动词/情态动词/be 提前于主语
   - Only when... **does** he understand.（does 提前）
   - Only if... **will** she agree.（will 提前）
   - Only then **was** it clear.（was 提前）
2. **Only + 从句 → 主句倒装**（注意：从句本身不倒装，只有主句倒装）
3. **常见倒装词**：only/not only/hardly/never/seldom/not until
4. **写作**：用 only 倒装句提升作文句式多样性

### 易错点

- **只有主句倒装，从句不倒装**：Only when **you fail** **do** you learn.（从句正常，主句倒装）
- **不能直接用 only + 主语**（非状语时不倒装）：Only Tom came.（Tom 是主语，无从句，不倒装）
- **助动词选择**：根据时态选 do/does/did/will/would/can/could/have
