## 🎓 Clone 克隆

**定义**：Clone 既作名词（克隆体，基因完全相同的有机体副本），也作动词（克隆，通过无性繁殖制造基因相同的副本）。克隆技术是现代生物科技与科幻小说的重要主题。

### 词形与词族

| 形式 | 示例 |
|---|---|
| 名词 | clone (n.)；cloning (n.) 克隆技术/过程 |
| 动词 | clone (v.)；cloned (past tense / past participle) |
| 形容词 | cloned (adj.)：a cloned sheep |
| 相关词 | genetic, DNA, reproduction, embryo |

### 例句（中英对照）

| 英文 | 中文 |
|---|---|
| Dolly the sheep was the first successfully cloned mammal. | 多莉羊是第一只成功克隆的哺乳动物。 |
| Many scientists debate whether it is ethical to clone human beings. | 许多科学家争论克隆人类是否符合伦理。 |
| In the novel, an army of clones was created to fight in the war. | 小说中，一支克隆人军队被制造出来参战。 |
| Cloning technology has potential benefits in medicine. | 克隆技术在医学上有潜在的益处。 |
| The idea that humans could be cloned is still controversial. | 人类能够被克隆这一想法仍具争议。 |

### 高考考点

1. **名词性从句（同位语从句）**：The idea **that humans could be cloned** is still controversial.（同位语从句解释 idea 的内容）
2. **被动语态**：Dolly was cloned in 1996.（be + past participle）
3. **话题写作**：科技伦理议论文高频话题；可用 whether 引导宾语从句表达立场：I believe that whether cloning is ethical depends on how it is used.

### 易错点

- clone 作动词时是及物动词：clone + 宾语（不需要介词）：scientists cloned the sheep（不说 cloned of the sheep）。
- cloning（动名词/现在分词）vs. clone（名词/动词原形），语法填空注意词形。
