## 🎓 倒装——only + 状语置首

**定义**：当 only 修饰状语（副词、介词短语、状语从句）且该状语置于句首时，主句需进行部分倒装（助动词/情态动词提前）。only 修饰主语或宾语时不倒装。

---

### 触发倒装 vs 不触发倒装

| 情形 | 是否倒装 | 示例 |
|------|---------|------|
| only + 副词置首 | ✅ 倒装 | Only then did I understand. |
| only + 介词短语置首 | ✅ 倒装 | Only by hard work can you succeed. |
| only + 时间/条件从句置首 | ✅ 倒装（主句倒装） | Only when he arrived did we start. |
| only 修饰主语（句首） | ❌ 不倒装 | Only Tom passed the exam. |
| only 修饰宾语 | ❌ 不倒装 | I only know this one fact. |

---

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 只有这样，我们才能解决问题。 | Only in this way can we solve the problem. |
| 只有努力工作，你才能成功。 | Only by working hard can you succeed. |
| 只有当他回来时，我们才开始。 | Only when he came back did we start. |
| 只有到那时，她才意识到友情的重要性。 | Only then did she realize the importance of friendship. |
| 只有通过阅读，人才能增长见识。 | Only through reading can one broaden one's horizons. |
| 只有用这种方法，问题才能得到根本解决。 | Only by using this method can the problem be fundamentally solved. |

---

### 高考考点

1. **三步法**：① 识别 only + 状语在句首 → ② 判断主句助动词 → ③ 写出倒装结构
2. **only when/if 从句**：从句内部不倒装，只有主句倒装
3. **写作高分句**：Only by/through/in this way... 是高考议论文常用开头句型

### 易错点

- ❌ Only Tom did pass the exam. → ✅ Only Tom passed the exam.（only 修饰主语，不倒装）
- ❌ Only when he arrived, we started. → ✅ Only when he arrived did we start.（主句要倒装）
