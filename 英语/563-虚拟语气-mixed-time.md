## 🎓 虚拟语气——混合时间条件句（Mixed Conditional）

**定义**：条件从句与主句所指的时间参照点不同，导致两个分句各自使用不同的虚拟时态形式。选必三重难点，高考语法填空高频考点。

---

### 核心公式

#### 类型 A：过去条件 → 现在结果

```
If + 主语 + had done（过去虚拟），主语 + would/could + do（现在虚拟）
```

**逻辑**：如果过去发生了某事（实际上没有），现在就会是另一种情况（实际上不是）。

#### 类型 B：现在性质 → 过去结果

```
If + 主语 + were/did（现在虚拟），主语 + would/could + have done（过去虚拟）
```

**逻辑**：如果某人现在具备某种特质（实际上不具备），过去就会有不同的结果。

---

### 例句（中英对照）

| 中文 | English |
|------|---------|
| 如果我昨天没吃那颗药，今天就会更难受。（过去→现在） | If I hadn't taken that pill yesterday, I would feel worse today. |
| 如果他天生是个勇敢的人，他就会当时站出来说话了。（现在→过去） | If he were a brave person, he would have spoken up at that time. |
| 如果我当年学了法律，我现在就能帮到你了。 | If I had studied law, I could help you now. |
| 如果她不是如此固执的人，她早就改变主意了。 | If she were not so stubborn, she would have changed her mind. |
| 如果我们那时采取了措施，情况现在就不会这么糟。 | If we had taken measures then, the situation wouldn't be so bad now. |
| 要是他当初没辍学，他现在就不会如此困难了。 | If he hadn't dropped out, he wouldn't be struggling so much now. |

---

### 识别技巧

1. **时间标志词**：条件从句有 then/at that time/yesterday/last year，主句有 now/today/at present/still → 混合时间虚拟
2. **主语特质**：主句主语描述的是天生性格/固有特质（be + 形容词），条件从句说过去行为 → 类型 B

---

### 高考考点

- **语法填空**：给出时间标志词，判断两个分句的时态应该不同
- **改错**：将混合时间句错误地统一成同一时态的虚拟语气

### 易错点

- ❌ If he had studied law, he would have helped you now. → ✅ ...he would help you now.（现在虚拟主句不用 have done）
- 注意主句中 now/today 是现在虚拟的信号，should/would + do（原形），不是 have done
