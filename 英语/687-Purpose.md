# Purpose

> **一句话**：Purpose 指语言中用于表达行为“目的”或“意图”的语法结构与逻辑关系，在高考英语中集中体现为**目的状语**以及阅读理解中**作者意图**的推断。
> **English**: Purpose refers to the grammatical structures and logical relations used to express the aim or intention behind an action, most typically realized through adverbial clauses and infinitives of purpose.

## 🎓 考点精讲

**定义/原理**：在英语中，“目的”主要通过**目的状语**（Adverbial of Purpose）实现，其核心语法形式包括三类：（1）**不定式**（to do / in order to do / so as to do），其中 so as to do 不可置于句首；（2）**目的状语从句**，常见连词为 **so that** 与 **in order that**，从句中通常含有情态动词 can / could / may / might 等；（3）**介词短语**，如 for the purpose of / with the aim of / for fear of 等。此外，名词 purpose 本身后接介词 on（on purpose 故意地）或 for（for the purpose of...）也是高频搭配。

**关键方法**：高考解题时需区分**目的**与**结果**——这是最核心的语法辨析。判定依据：若从句谓语包含 can / could / may / might 等情态动词，且主从句之间是“主观意图→可能实现”的逻辑，则为目的状语；若描述“客观事实→自然导致的结果”，则为结果状语。典型对比结构为 **so that**（目的）与 **so...that...**（结果）。在七选五或阅读理解中，识别段落内表示目的的衔接手段（如 To achieve this, / For this purpose, / with this in mind）是推断作者写作意图的关键。

**典型应用**：高考语法填空常要求将动词转化为不定式表目的，例如 “She saved money ______ (buy) a new phone” 应填 to buy。短文改错可能将 so that 误用作 so...that，或混淆 on purpose 与 by accident。阅读理解中，题干设问 “What is the author's purpose in writing Paragraph 2?” 或 “...is intended to ______”，需从目的逻辑出发，判定该段落是为支撑论点、举例说明、对比反衬，还是引出下文。

## ⚠️ 高频易错点

- **so that 与 so...that 混淆**：so that 引导目的状语从句，句中常含情态动词；so...that... 引导结果状语从句，结构中 so 后接形容词或副词，that 后陈述客观结果。两者的谓语逻辑完全不同。
- **不定式表目的时主句主语不一致**：In order to 结构要求逻辑主语与主句主语一致；若主从句主语不同，必须使用 so that / in order that 引导的从句，而非单纯不定式。
- **on purpose / by chance 误用**：on purpose（故意地）多带贬义或强调有意为之，by chance / by accident 才表示“偶然地”，完形填空中常据此设空考查上下文态度判断。
- **阅读中混淆“目的”与“事实”**：作者写某段的“目的”是为了论证、转折、举例还是引出新话题，答案必须是“意图”层面而非“该段内容复述”——题干含 purpose / aim / intend 即是明确信号。
