# cause-effect

> **一句话**：因果关系考点考查在语篇中准确识别、分析与表达原因（cause）和结果（effect）之间逻辑关系的能力。
> **English**: The cause-effect relationship describes a logical connection in which one event or condition (the cause) directly brings about or influences another event or condition (the effect).

## 🎓 考点精讲

**定义与核心原理**

在英语高考中，因果关系（Cause and Effect）是语篇层面最重要的逻辑语义关系之一。其核心原理为：当一个事件（cause）直接导致或引发另一事件（effect）时，二者构成因果关系链。高考对该考点的检测不限于识别显性的因果连接词，更强调在复杂长难句中理解隐含的因果逻辑，以及在阅读理解、完形填空、七选五、语法填空和写作中正确运用因果表达。

因果关系的典型显性标记包括：`because / since / as / for`（引出原因）、`so / therefore / thus / consequently / as a result`（引出结果）、`lead to / result in / contribute to / cause`（导致）、`due to / owing to / because of / as a result of`（由于）等。隐性因果关系则往往通过非谓语动词作状语（如分词短语表示原因或结果）、独立主格结构、定语从句、并列连词 `and`（表顺承因果）甚至无任何连接词的语义自然衔接来体现。

**关键解题方法**

在高考阅读理解中，“原因类”题型（Why...? / What caused...? / ...because...）的解题步骤：① 定位题干关键词，在原文找到答案来源句；② 识别因果标记或分析句间逻辑；③ 验证选项是否为直接原因而非间接或无关信息。特别注意命题人常在选项中用原文词句反向重组因果方向（如将“A导致B”改写为“B归因于A”）。

在七选五和完形填空中，因果关系是判断句间、段间逻辑的三大主线之一（另两种为转折、并列递进）。若前后句存在“前因后果”或“前果后因”关系，应选择因果连接词（如 `so / therefore`）或表因果功能的衔接句。语法填空常考查 `because` 与 `so` 不可连用、`as a result of + n./doing`、以及分词作结果状语时用现在分词形式。

**典型高考应用场景**

1. **阅读理解细节题**：题目如 *Why did the author refuse the invitation?* 原文若使用 `because`、`since`、`as` 或 `due to` 直接给出原因，锁定标记词即可。若原文为 “The task proved more difficult than expected. The team decided to extend the deadline.” 第二句为结果，隐含原因是前句所述“任务难度超出预期”，学生需进行逻辑推断。

2. **七选五**：当空前句描述一个现象或结论，空后句给出解释理由时，中间空缺处常填入表原因的过渡句或因果连接词。如空前句 “The experiment failed.” 空后句 “The temperature was not properly controlled.” 则空中应填入 “The reason was that...” 或 “This is because...” 等因果功能句式。

## ⚠️ 高频易错点

- **混淆时间先后与因果关系**：两事件先后发生不代表存在因果关系，必须判断是否存在逻辑上的“导致”与“被导致”联系。（`After the rain stopped, the streets were wet.` 前一事件是原因；但 `After the rain stopped, we went shopping.` 仅为时间顺序，非因果。）
- **误将伴随状况当作原因**：分词短语作状语时既可表原因也可表伴随，要根据主句与分词语义判断。如 “Reading the book carefully, she understood the theory.”（原因）对比 “She sat there, reading a book.”（伴随）
- **`because` 与 `so` 连用**：受汉语“因为……所以……”影响，在语法填空和写作中学生在同一句中同时使用 `because/ since/ as` 和 `so/therefore`，这是严重语法错误，两者只能择其一。
- **混淆原因与结果的连接词**：如将 `as a result`（表结果）与 `as a result of`（表原因）用反；或将 `lead to` 的主宾逻辑倒置：`Hard work leads to success.` 主语因，宾语果。
