# genre identification

> **一句话**：体裁识别是依据文本的结构特征、语言风格和交际目的，将阅读语篇归类为记叙文、说明文、议论文或应用文等核心体裁的阅读策略。
> **English**: Genre identification is the reading strategy of classifying a text into core genres—such as narrative, exposition, argumentation, or practical writing—based on its structural features, linguistic characteristics, and communicative purpose.

## 🎓 考点精讲

**定义/原理**：在高考英语阅读理解中，体裁（genre）是由交际目的决定的文本类型，不同体裁具有可辨识的思维组织模式与语言标记。四种核心体裁的原理如下：

- **记叙文（Narrative）**：以时间顺序或事件发展为主线的“叙述结构”，核心逻辑是 `Orientation（开端） → Complication（冲突） → Resolution（解结）`。常见于人物故事、个人经历类篇章。
- **说明文（Exposition）**：以解释事物本质、阐明事理为目的，核心逻辑是 `General statement → Specific details/illustrations` 或 `Phenomenon → Cause → Effect → Solution`，常见于科普、社会现象分析类篇章。
- **议论文（Argumentation）**：以论证观点、说服读者为目的，核心结构为 `Claim → Evidence → Counter-argument → Rebuttal → Conclusion` 或 `Thesis → Supporting arguments → Conclusion`，常见于社评、观点态度类篇章。
- **应用文（Practical Writing）**：以满足特定交际功能为目标，格式固定，包含信件、通知、广告、指南、报告等子类型，结构高度模式化（如信件必有称呼与签名）。

**关键方法**：识别体裁的标准步骤为三步走——**第一步看语境与目的**（首段通常点明全文功能：to tell a story / to explain / to argue / to inform），**第二步抓结构信号词**（时间序列词 → 记叙；因果/对比/列举词 → 说明；观点/转折/反问 → 议论；称谓/祈使句 → 应用），**第三步验篇章整体框架**（将全文按段落功能切分，匹配上述原型结构）。若体裁混杂（如夹叙夹议），以主导格局判定。

**典型应用**：高考阅读理解中，体裁识别的典型设问包括：“What type of writing is this text?”、“What is the text mainly organized as?”、“Where is this text probably taken from?”（判断出版来源，其本质也是体裁识别）。具体场景如：
1. 给定一篇首段为“At the dawn of the Industrial Revolution, few could have predicted…”，次段大量使用 `consequently`, `as a result` 串联工厂扩张与社会变化，末段提出 `One promising solution lies in…`。学生需依据“现象—原因—影响—对策”的陈述性展开，判断为**说明文**。
2. 给定一篇以“I still remember the chill of that November morning”开头，按时间词 `then`, `later`, `finally` 铺陈个人经历，结尾点明感悟，则判定为**记叙文**。

## ⚠️ 高频易错点

- **混淆说明文与议论文**：说明文客观呈现事实与分析，议论文则需有明确**论点句（thesis statement）** 和对反方观点的**反驳**。若通篇未出现立场鲜明的 `should`、`it is imperative that` 等论断性结构，多为说明文而非议论文。
- **“出处”类题目误判体裁**：题干问“Where is this text probably taken from?”，选项含 `A novel.` `A textbook.` `A newspaper.` 时，需结合体裁判断而非仅凭关键词。例如科普说明文最可能出自 textbook 或 science magazine，而非 novel（叙事虚构）。
- **忽视应用文的格式标记**：书信、邮件必有称谓（Dear…）与结尾敬语（Yours sincerely / Sincerely）；通知必有标题（NOTICE）和发布单位；广告多含祈使句（Book now! / Call today!）。遗漏这些体裁专属标记会导致误判。
- **夹叙夹议篇章的定性混乱**：以叙为主、议为辅的篇章（如经历+感悟），依然归为**记叙文**，因为全文依靠叙事逻辑推进；反之，以议为主、叙为论据的篇章属**议论文**。判据在于“去掉叙事/议论部分后，剩余部分是否仍具备完整的主旨传达功能”。
