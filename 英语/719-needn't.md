# needn't

> **一句话**：needn’t 是情态动词 need 的否定式，意为“不必”，仅用于否定句和疑问句，后接动词原形。
> **English**: "Needn’t" is the negative form of the modal verb "need", meaning "it is not necessary to", used only in negatives and questions, followed by a bare infinitive.

## 🎓 考点精讲

needn’t 的核心用法分为两类，高考主要考查其情态动词身份与实义动词 need 的否定式之间的辨析，以及“needn’t have done”结构。

- **情态动词 need 的用法**  
  need 作情态动词时，只有现在时形式，无人称、数、时态变化，后直接接动词原形。**肯定句**中一般不用，若需肯定表达“必须”，要用 must 或 have to；否定句用 needn’t（need not），疑问句把 need 提前。  
  例：You needn’t come if you don’t want to.  （不必来）  
  回答由 must 引起的问句时，肯定用 must，否定用 needn’t。  
  — Must I finish it now?  — No, you needn’t. (Yes, you must.)

- **needn’t have done 与 didn’t need to do 的区别**  
  needn’t have done 表示“本不必做，却做了”，含有对过去行为的委婉批评或遗憾。  
  didn’t need to do 则表示“过去不必做，（通常）也没做”，仅仅是陈述事实，无额外情感色彩。  
  例：You needn’t have told him the secret; he already knew. （你本不必告诉他这个秘密，他已经知道了。）  
  I didn’t need to go to the station because she offered a ride. （我不用去车站，因为她要顺路送我。）

- **实义动词 need 的否定式**  
  need 作实义动词时，否定要借助助动词 don’t/doesn’t/didn’t，后接带 to 的不定式。  
  对比：You needn’t go. （情态动词） = You don’t need to go. （实义动词）

高考常以单项选择题、语法填空或短文改错考查以上知识点，设问角度包括：选择正确否定形式，判断 need 的词性，辨析 needn’t have done 与 didn’t need to do，或在情景交际中选用 needn’t 作回应。

## ⚠️ 高频易错点

- 将 needn’t 与 mustn’t 混淆：mustn’t 表示“禁止、千万别”，不是“不必”；needn’t 才是“不必”。
- 在肯定句中误用情态动词 need（× He need come → 正确用实义动词 He needs to come 或用 must）。
- 混淆 needn’t have done 和 didn’t need to do：前者强调“做了多余的事”，后者只陈述“没做的必要”。题目中要仔细辨别是否有“已做”的语境。
- need 作情态动词时，否定缩写 needn’t 后直接跟动词原形，误加 to（× needn’t to do → 正确 needn’t do）。
