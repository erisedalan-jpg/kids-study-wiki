# 事实细节题

> **一句话**：阅读理解中要求根据短文内容直接找出或确认具体事实、数据、原因、方式等信息的题目，答案通常可以在原文中找到明确文字依据。
> **English**: A reading comprehension question that asks for specific information explicitly stated or directly retrievable from the passage, such as facts, figures, reasons, or methods.

## 🎓 考点精讲

事实细节题是高考英语阅读理解中考查频率最高、最为基础的题型，重点考查考生快速定位、准确识别并理解文中具体信息的能力。这类题不涉及深层推断或主旨概括，答案就藏在文章的某一句话或几句话中，以同义转述、信息整合或正话反说的方式呈现。

**定义/原理**：题目直接对应文中的某一信息点，正确选项是对原文信息的忠实重现，只是用词和句式可能发生了变化。题干常包含 what, who, when, where, why, how, which, how many/much 等疑问词，或以 According to the passage / From the text we can learn 等结构引入。解答原理就是“原文定位 — 选项比对”，要求考生具备浏览、扫读和识别同义替换的能力。

**关键方法**：采用“题干关键词 → 原文定位 → 比对选项”三步法。第一步，读题干并圈出易识别的关键词，如人名、地名、数字、专有名词、实义动词或名词短语；第二步，带着关键词快速扫读文章，找到关键词第一次出现或最相关的位置，必要时向上、下各读一句以确定完整信息；第三步，将定位句与选项逐词对比，注意同义替换（如 rise → increase, reason → because）、句式转换（主动变被动）或反义表达（not difficult → easy）。正确选项几乎不会是原文照抄，而是对原文信息的凝练或改写；干扰项常包含原文单词，但语义被歪曲或无中生有。

**典型应用**：高考中常见设问：“What is the main reason why…?” “When did the experiment begin?” “How many volunteers took part in the study?” “Which of the following is true according to the passage?” 例如，一篇关于智能穿戴设备的说明文中，题目问：“According to the passage, what function is NOT mentioned about the new smartwatch?” 考生需自动将 NOT 识别为逆向提问，回原文将提及的功能一一排除，选出未被谈到的那个选项。

## ⚠️ 高频易错点

- **定位不准或仅凭印象选择**：阅读时对某处有模糊印象，不再回原文核实，容易被干扰项中重复出现的词误导；尤其当同一信息在文中多次出现时，容易错配。
- **原词复现陷阱**：干扰项直接照搬原文中的某个单词或短语，但整体意思与原文相反或无关。考生若只盯熟悉的词而忽略整句逻辑，会误以为选项有原文依据。
- **忽视题干的否定和限定词**：题干中出现 not, except, least, according to the first paragraph 等提示词时，考生常因粗心而选反或找错范围。
- **同义替换识别失败**：原文用近义词或不同句式表达时，若考生词汇量有限或对 paraphrase 不敏感，会认为选项与原文不符而错选，例如原文用 minimize the risk，选项写 reduce the possible danger，应判定为一致。
