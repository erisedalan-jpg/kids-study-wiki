# 同义替换

> **一句话**：用意义相同或相近的词语、短语或句式替换原表达，在保持句意基本不变的前提下实现语言的准确与多样。
> **English**: The use of a synonym or an equivalent structure to convey the same intended meaning as the original expression.

## 🎓 考点精讲

- **定义/原理**  
  同义替换是高考英语中考查词汇理解深度与语境推断能力的关键手段。命题人将原文或题干的信息用另一种形式表达，要求考生在选项中找到与之一致的替换形式。其核心原理建立在“语义等值”之上：无论表层结构如何变化，深层语义应保持一致。常见变换包括：**同义词/近义词**（e.g., *purchase* ↔ *buy*）、**词性转换**（e.g., *decide v.* ↔ *decision n.*）、**短语与单词的互换**（e.g., *give up* ↔ *abandon*）、**肯定与双重否定**（e.g., *not uncommon* ↔ *common*）以及**句式转换**（主动↔被动、因果倒置等）。

- **关键方法**  
  解答同义替换类试题的标准流程为：  
  1. 定位原文与题目对应的信息块；  
  2. 逐项比对选项，判断是否存在词汇或结构层面的“不一致但等值”关系；  
  3. 优先排查显性同义词，若无效则转向隐性替换——判断是否含有同一核心概念的概括/具体化（如上义词与下义词）；  
  4. 将选项放回语境验证句意，确保替换后逻辑一致。  
  解完形填空时，若四个选项均无原词，往往需从上下文找同义复现或反义对照。解阅读理解时，正确答案几乎总是“原文信息的同义转述”，绝少原样照抄。

- **典型应用**  
  高考设问常表现为：  
  - 阅读理解细节题：“According to the passage, which of the following is true about …?” 正确选项常将原文 *significant* 替换为 *considerable*。  
  - 七选五/完形填空：利用前后句的 *big* 与 *huge* 同义呼应判定缺失词。  
  - 写作评分维度：恰当使用 *demonstrate* 替换 *show*、*consequently* 替换 *as a result* 直接提升语言多样性得分。

## ⚠️ 高频易错点

- **混淆近义词的感情色彩或搭配**：如 *alone* 与 *lonely* 词义相近但内涵不同，不加区分随意替换会导致语境偏差。
- **只关注单词替换而忽略短语层面**：将 *look forward to* 简单记作 *expect*，遗忘 *to* 为介词时后接动名词，导致结构错误。
- **机械记忆同义词表，脱离语境**：在“缓解压力”表述中，*relieve stress* 可替换为 *ease pressure*，但考生强制用 *reduce anxiety* 可能曲解原文情感强度。
- **完形中的“原词干扰”**：选项中出现原文相同单词时常被当作正确标志，但高考完形往往要求选出经过同义替换的选项，原词存在常为干扰项。
