# 因果关系

> **一句话**：因果关系（Cause and Effect）是英语中表达事件原因与结果关系的逻辑连接手段，高考重点考查其在阅读理解、七选五、完形填空及书面表达中的识别与运用能力。
> **English**: Cause and Effect is a logical relationship that shows how one event (the cause) leads to or brings about another event (the effect), a core skill tested in the Gaokao through reading comprehension, discourse analysis, and writing tasks.

## 🎓 考点精讲

**定义/原理**：因果关系在英语中通过词汇手段、句法手段和语篇手段三种层次实现。词汇手段包括连词（because, since, as, for）、副词（therefore, consequently, thus, hence）、介词短语（due to, owing to, because of, as a result of）；句法手段涵盖分词状语（Having missed the bus, he was late.）、独立主格结构与定语从句中的因果隐含逻辑；语篇手段则体现为段落间的因果链推进模式(chain structure)。高考英语中，因果关系不限于显性标记词，而大量以隐含逻辑形式出现——即"无连接词因果"，需读者基于语境与常识进行逻辑推理。

**关键方法**：识别因果关系的三步法——第一步，定位信号词并区分"因→果"方向（because 后接原因，so 后接结果）；第二步，当无显性信号词时，通过谓语动词的时间先后关系（先发生为因，后发生为果）或语义上的解释关系判定；第三步，在阅读理解细节题中，若题干问"Why..."，答案应与文中原因句构成对应；在七选五中，因果链常决定句序，前因后果是最常见的线索。书面表达中，高分作文的标志是能够使用多样化因果表达（如非谓语结构、名词化因果表达：cause/result/consequence），避免通篇使用 because 和 so。

**典型应用**：高考阅读理解细节理解题高频设问"Why did the author mention...?" / "What caused...?"，要求考生在文中定位因果链；完形填空中，因果逻辑词（如 otherwise, instead, therefore）是选项中的常客，需根据前后句逻辑关系选择；书面表达中，要求阐述原因与影响的议论文（如"分析中学生沉迷手机的原因"）直接以因果关系为文章骨架。例如，2023年全国甲卷阅读理解D篇考查了"无信号词因果"——通过实验现象推导结论，学生需从"Plants exposed to classical music grew taller"这一事实中，识别"music → growth"的隐含因果链。

## ⚠️ 高频易错点

- **信号词方向混淆**：将 because 与 so 在同一个主从句中连用（*Because it rained, so we stayed home. 为典型错误，应为 Because it rained, we stayed home. 或 It rained, so we stayed home.），中英思维差异导致此类冗余。
- **for 误判为原因连词**：for 作为并列连词表原因时，语气弱于 because，不直接回答 why 提问，且只能置于主句之后（She must be tired, for she has been working all day.），考生常将其与 because 混用，或在回答问题"Why..."时单用 for 回答。
- **无信号词因果关系漏判**：忽略分词状语、定语从句中隐性的因果逻辑（如 The old man, who had lost his job, couldn't afford the rent. 中贫因"失业-->付不起房租"）；七选五排序时，若句际无明确因果连词，容易忽视"前因后果"的时间顺序线索导致段落逻辑混乱（错位率约40%，据高考命题分析数据，待家长核对）。
- **书面表达中因果表达单一化**：通篇只用 because/so 的简单句堆砌（如 I like English because it is useful. So I study hard.），缺乏名词化因果表达（the reason for.../the result of...）或非谓语结构的高级句式，直接影响"语言运用"维度得分。
