# 固定句型

> **一句话**：固定句型是指英语中结构相对固化、语法功能明确的特定句式，其在高考中直接考查句法识别与运用能力。  
> **English**: A fixed sentence pattern is a syntactically stable and functionally specific structure in English, directly tested in the college entrance examination for recognition and application.

## 🎓 考点精讲

**定义/原理**  
固定句型并非随意组合，而是由特定关键词（如 `it`、`that`、`what`、`do`、`there`、`only`、`had` 等）触发，并遵循严格的语序、时态配合与主谓一致规则。例如：强调句型 `It is/was … that/who …` 可强调主语、宾语、状语；倒装句型包括部分倒装（由 `Only + 状语`、`Not until`、`So/Such`、`Hardly … when` 等引导）和完全倒装（`Here/There/Out/In` 起始 + 不及物动词 + 主语）；虚拟语气中的固定句式如 `It is (high) time that S + did/should do`、`would rather S + did`、`as if/though + did/had done` 等。这些句式的共同原理是：通过特殊句法标记改变信息焦点或表达非事实语义。

**关键方法**  
1. **识别标记词**：扫描句首或主句中的关键词，判断是否触发特定句型。  
2. **结构拆解**：将句子按“标记词 + 预留结构”拆分，例如看到 `It was` 后检查是否可还原为 It 引导的后续 that 从句，若能还原且去掉 It was…that 后句子成分仍然完整，则为强调句；若不能，可能为定语从句或主语从句。  
3. **时态与语序匹配**：如虚拟语气 `If I had done …, I would have done …`；`Had I done …` 为省略 if 的倒装虚拟。部分倒装需将助动词/情态动词/be 动词提到主语前，而完全倒装是主动词提前，主语是名词时全倒装，是代词时不用。  
4. **写作升级**：在书面表达中主动使用固定句型（如 `It is universally acknowledged that …`、`Not only … but also …`、`Only in this way can we …`）以提升句式多样性。

**典型应用**  
- 语法填空：给部分提示词，要求填入助动词、连词或特定形式，如 `It was in the small village ______ he spent his childhood.`（填 that，强调句型）。  
- 短文改错：在错误倒装或强调结构中纠错，如 `Hardly he had arrived when it began to rain.` 应改为 `Hardly had he arrived …`。  
- 书面表达：应用 `So + adj./adv. + 倒装` 结构表达结果，如 `So urgent is the issue that immediate action should be taken.`  

## ⚠️ 高频易错点

- **强调句与定语从句混淆**：`It was the factory that he visited last year.`（强调句，that 不作成分） vs. `It was the factory where he worked.`（定语从句，where 作地点状语），区分关键在 removed structure 后成分是否完整。  
- **部分倒装与完全倒装误用**：方位副词置于句首且主语为名词时用完全倒装（`Out rushed the children.`），但主语为代词时不倒装（`Out they rushed.`）；而部分倒装由否定词/only等触发，无论主语为何都倒装助动词。  
- **虚拟语气中 `would rather` 后的时态**：`I would rather you had come yesterday.` 用 had done 对过去虚拟，`I would rather you came tomorrow.` 用 did 对将来虚拟，学生常混用过去式与过去完成式。  
- **`It is time` 后从句虚拟漏用**：`It is high time that we take actions.` 应为 `took action` 或 `should take`，此处“take”易误用原形。
