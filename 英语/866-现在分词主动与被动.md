# 现在分词主动与被动

> **一句话**：现在分词作状语或定语时，需根据其与逻辑主语的主谓或动宾关系，选用主动式（doing）或被动式（being done）。
> **English**: When a present participle functions as an adverbial or attributive, its voice—active (doing) or passive (being done)—is determined by the logical subject-verb or verb-object relationship.

## 🎓 考点精讲

- **定义/原理**：现在分词的语态取决于它与**逻辑主语**的关系。若逻辑主语是分词动作的**执行者**，用主动式（doing）；若逻辑主语是分词动作的**承受者**，用被动式（being done）。在时间维度上，主动式还可细分为一般式（doing）和完成式（having done），被动式分为一般式（being done）和完成式（having been done）。高考考查的核心是区分 doing / being done / having done / having been done 四种形式。

- **关键方法**：解题三步走——  
  1. **找逻辑主语**：状语时，逻辑主语通常是主句主语；定语时，逻辑主语是被修饰词。  
  2. **判主被动**：逻辑主语发出动作为主动，承受动作为被动。  
  3. **定先后**：分词动作与谓语动词同时发生用一般式，明显先于谓语发生用完成式。  
  判定依据可简化为：主动同时→doing；主动先发生→having done；被动同时→being done；被动先发生→having been done。

- **典型应用**：高考常出现在语法填空与短文改错中，设问角度为“给出动词原形，要求填入正确的非谓语形式”。典型场景：  
  1. 状语从句简化：`Because he was praised by the teacher, he felt happy.` → `Having been praised by the teacher, he felt happy.`（被动且先于felt发生）  
  2. 定语从句简化：`The bridge which is being built will be completed next year.` → `The bridge being built will be completed next year.`（被动同时）

## ⚠️ 高频易错点

- **过去分词与现在分词被动的混淆**：`being done` 强调“正在被”，`done` 仅表“被动完成”或“一般被动”。如 `the problem discussed yesterday`（昨天讨论过的问题）vs. `the problem being discussed now`（正在被讨论的问题），时间侧重点不同。
- **漏掉 being 直接用 done 表“正在被”**：误将 `The building building there` 当作“正在被建的楼”，正确应为 `The building being built there`。现在分词被动式必须有 being。
- **having been done 与 done 作状语时的混用**：两者都可表“被动且先于谓语发生”，但 having been done 强调动作先后，done 更侧重状态。在句首独立结构或需要强调先后关系时，用 having been done 更严谨，但命题中两者常被视为可互换（以具体语境和阅卷标准为准，待家长核对）。
- **逻辑主语判断错误**：如遇到独立主格结构 `Weather permitting, we'll go.`，逻辑主语为 Weather 而非主句主语 we，若误判则分词形式全错。
