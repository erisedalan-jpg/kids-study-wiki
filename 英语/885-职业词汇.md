# 职业词汇

> **一句话**：在高考英语中，“职业词汇”主要指在完形填空、阅读理解及写作中高频出现的表示职业、身份或社会角色的名词，考查核心在于利用构词法（后缀）精准辨析词义与词性。
> **English**: Occupation vocabulary refers to high-frequency nouns denoting jobs, professions, or social roles tested in the Gaokao, focusing on precise meaning and part-of-speech discrimination through word formation (suffixes).

## 🎓 考点精讲

该考点并不考查孤立的职业单词默写，而是考查在语篇中通过**构词法（Word Formation）**识别和产出正确的职业名词。其核心原理是：英语中大量的职业名词由动词或名词添加特定后缀派生而来。

**关键方法：常见职业名词后缀判定法**

在高考完形填空的选项辨析或语法填空的词性转换中，必须根据句意（做该动作的人/从事该领域的人）选择正确的后缀：

- **-er / -or**：表示“做……动作的人”，常加在动词后。
    - `teach` → `teacher`；`drive` → `driver`；`act` → `actor`；`direct` → `director`。
    - **注意**：以 `-e` 结尾的动词通常直接加 `-r`（如 `manage` → `manager`）；重读闭音节双写末尾辅音字母加 `-er`（如 `win` → `winner`）。
- **-ist**：表示“从事……领域/信仰/乐器的人”，常加在名词或学科后。
    - `science` → `scientist`；`art` → `artist`；`piano` → `pianist`；`tour` → `tourist`。
- **-ian**：表示“精通……的人/从事……职业的人”，常加在名词后（尤其是以 `-ic`， `-ics` 结尾的词）。
    - `music` → `musician`；`politics` → `politician`；`history` → `historian`。
- **-ant / -ent**：表示“进行……动作的人”，常加在动词后。
    - `assist` → `assistant`；`serve` → `servant`；`study` → `student`。
- **-ee**：表示“承受……动作的人”（与 -er 相对）。
    - `employ`（雇佣）→ `employer`（雇主）/ `employee`（雇员）；`interview`（面试）→ `interviewer`（面试官）/ `interviewee`（面试者）。

**典型应用（高考设问与场景）**

1.  **完形填空（词义辨析）**：选项中出现同根但后缀不同的职业名词。
    - *场景*：He worked as a _____ in a local library, helping people find books. (A. librarian B. historian C. politician D. musician)。根据 `library` 场景，锁定 `library` → `librarian`（图书管理员）。
2.  **语法填空（词性转换）**：提示词为动词或名词，要求填写表示人的职业名词。
    - *场景*：As a famous _____ (compose), he has created many beautiful symphonies. 根据 `music` 场景和不定冠词 `a`，填 `composer`。
3.  **阅读理解（指代判断）**：文中出现职业名词，问题询问该人物的具体工作内容。

## ⚠️ 高频易错点

- **拼写混淆（-er vs -or）**：误将 `actor`, `director` 拼写为 `acter`, `directer`。需强记：以 `-t` 结尾的动词常派生为`-or`（如 `invent` → `inventor`；但 `visit` → `visitor`，`teach` → `teacher` 属常规变化）。
- **忽略重读闭音节双写规则**：在 `win` → `winner`， `run` → `runner` 等词中忘记双写末尾辅音字母。
- **[-er / -or] 与 [-ee] 的主被动混淆**：在完形中因未分清“发起者”与“承受者”而选错。关键是看句子的主语是付钱雇佣的人（employer）还是被雇佣干活领钱的人（employee）。
- **[-ent / -ant] 拼写混淆**：`assistant` 常误写为 `assisstant` 或 `assistent`；`servant` 常误写为 `servent`。需明确 `assist` + `ant` → `assistant`。
