# 语义辨析

> **一句话**：在特定语境下，依据搭配、内涵、语法功能及语体风格的细微差异，从两个或多个近义词、形近词或同类表达中选出最精准选项的能力。
> **English**: The ability to select the most precise word or expression from a group of near‑synonyms, similar‑looking words, or related phrases based on collocation, connotation, grammatical function, and register in a specific context.

## 🎓 考点精讲

语义辨析是高考英语完形填空、语法填空及写作的核心设错点之一，主要考查学生对高考核心词汇深度（一词多义、熟词生义、近义词差异）与广度（固定搭配、习惯表达）的掌握。

**考查维度与判定标准：**
1.  **搭配差异 (Collocation)**：词与词之间的惯用组合。例如：`heavy rain`（大雨）而非 `strong rain`；`make an effort` 而非 `do an effort`。
2.  **语义内涵 (Connotation)**：情感色彩与语气强弱。例如：`statesman`（褒义，政治家）与 `politician`（贬义，政客）；`thin`（中性，瘦）与 `skinny`（贬义，皮包骨）。
3.  **语法功能 (Grammatical Function)**：动词的及物与不及物性，或词性不同。例如：`lie (lay, lain)`（躺）常与方位短语连用，而 `lay (laid, laid)`（放置/产卵）必须直接带宾语。经典易混链：`lie - lay - lain`（躺） vs. `lie - lied - lied`（撒谎） vs. `lay - laid - laid`（放置）。
4.  **语体与范围 (Register & Scope)**：正式与非正式，具体与抽象。例如：`accelerate`（正式/抽象，加速） vs. `speed up`（非正式/具体，提速）；`abandon`（彻底放弃） vs. `resign`（辞职/认命）。

**高考典型设问与解题步骤：**
-   **完形填空**：题目常挖去一组近义实词（动词、名词、形容词），要求通过**上下文逻辑**与**固定搭配**排除。关键步骤：先找后置介词/副词看搭配（如 `take ___`），再横看宾语结构（人/物/是否接 to do/doing），最后回看主语与情感线索。
-   **写作升级**：考生需将高频基础词替换为高级精确词。例如将 `important` 根据语境精准替换为 `vital / significant / essential / crucial`，或 `very tired` 替换为一个词 `exhausted / worn out`。

**经典辨析示例（高考高频词群）：**

| 词群类别 | 精准释义与用法 |
| :--- | :--- |
| **“看”** | `glance` (快速扫视) / `stare` (盯着看) / `glare` (怒视) / `glimpse` (一瞥，强调结果) / `gaze` (凝视美景/若有所思) |
| **“花费”** | `spend` (人+spend+时间钱+on sth./(in) doing) / `cost` (物+cost+人+钱) / `take` (it takes+时间+to do) / `pay` (人+pay+钱+for sth.) |
| **“发生”** | `happen` (偶然发生) / `occur` (想法闪现/正式发生) / `take place` (计划好举行) / `break out` (战争/火灾爆发) |

## ⚠️ 高频易错点

-   **熟词生义陷阱**：高考常考常见词的冷僻义。例如 `succeed` 在完形中引出句子意为“继承”而非“成功”；`work` 作为名词意为“作品/工厂”而非“工作”。
-   **形近词混淆**：仅因拼写相似而误选。典型如 `affect (动，影响) / effect (名，影响；动，实现)`；`principal (校长；主要的) / principle (原则)`。
-   **中文直译干扰**：受母语负迁移，忽略英语搭配习惯。例如将“学习知识”误写/误选为 `learn knowledge` (正确为 `gain/acquire knowledge`)，或因“价格昂贵”直接把主语写成 `price` 后接 `expensive` (正确是 `The price is high. / The item is expensive.`)。
-   **及物/不及物误判**：只记词义不记结构。例如 `reply to the letter` 漏掉 `to`；`reach the school` 误加 `at`；`serve the people` 误写为 `serve for the people`。
