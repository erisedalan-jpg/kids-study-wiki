# 语态

> **一句话**：语态是表示主语与谓语动词之间逻辑关系的语法范畴，决定主语是动作的执行者还是承受者。
> **English**: Voice is a grammatical category that expresses the logical relationship between the subject and the predicate verb, determining whether the subject performs or receives the action.

## 🎓 考点精讲

**定义/原理**：英语语态分为主动语态（Active Voice）和被动语态（Passive Voice）。主动语态表示主语是动作的执行者；被动语态表示主语是动作的承受者，其核心结构为：**be + 及物动词的过去分词（done）**。被动语态中，动作的执行者由 `by` 引出，可省略。

**关键方法**：

1.  **被动语态判定三要素**：动词必须**及物**；主语承受动作；动词形式为 `be + done`（`be` 随时态/人称变化）。
2.  **主动→被动转换步骤**：原宾语→主语 + be（根据新主语的数和原句时态确定） + 原谓语动词的过去分词 + (by + 原主语宾格)。
3.  **常见时态的被动语态结构**：
    *   **一般现在时**: am/is/are + done
    *   **一般过去时**: was/were + done
    *   **现在完成时**: have/has been + done
    *   **过去完成时**: had been + done
    *   **将来时**: will be + done
    *   **含情态动词**: 情态动词 + be + done

**典型应用**：高考主要在**语法填空**（给出动词提示词，要求填入正确的被动形式）和**短文改错**（被动语态结构残缺或误用）中考查。常考难点包括：
*   **不及物动词/系动词无被动**：如 `happen`, `take place`, `last` 等不及物动词；`look`, `sound`, `smell`, `taste`, `feel` 等连系动词用主动表被动。
*   **主动形式表被动意义**：某些及物动词（如 `sell`, `wash`, `read`, `write`）与 `well`, `easily` 等副词连用时，常用主动表被动。例：*The book sells well.*
*   **使役/感官动词**：主动句中省 `to` 的不定式作宾补（如 `make sb. do`），变被动时须还原 `to`（`sb. be made to do`）。

## ⚠️ 高频易错点

- **及物性误判**：混淆不及物动词/连系动词与及物动词，对 `happen`, `take place`, `last`, `sound` 等词误加 `be done` 结构。
- **时态与语态复合错误**：只关注被动结构 `be done`，忽略 `be` 动词需随原句时态变化（如现在完成时的被动漏掉 `been`，写成 `have/has done`）。
- **主动表被动的特殊动词漏判**：看到主语是物，误以为必须用被动，忽略 `sell`, `wash`, `read` 等与副词搭配时的主动表被动用法。
- **不定式 `to` 的省略与还原**：在被动语态中，忘记还原使役动词（`make`, `let`, `have`）和感官动词（`see`, `hear`, `watch`）后不定式省略的 `to`。
