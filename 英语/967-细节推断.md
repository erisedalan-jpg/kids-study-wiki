# 细节推断

> **一句话**：细节推断是基于文本中明确陈述的事实、数据或描写，通过逻辑推理得出未直接言明但可从字里行间合理推导出的隐含信息，属于阅读理解中的推理判断题。
> **English**: Detail inference is the process of drawing logical conclusions about information not explicitly stated in a text, using facts, data, or descriptions that the text directly provides.

## 🎓 考点精讲

**定义/原理**：细节推断不同于细节理解题，它要求的不是对原文信息的简单复现，而是对分散在文本中的多处理性细节进行“逻辑拼接与一步推导”。其核心原理是：正确答案 = 文本明示线索 + 合理逻辑推衍，严禁引入文本外的个人知识或主观臆断。在高考英语中，该考点对应课标要求的“能根据上下文线索推断词语意义、作者意图及隐含信息”。

**关键方法**：解题采用**“两步定位法”**：
1. **题干关键词回文定位**：将题干中的专有名词、数字、核心动词等作为扫描锚点，精准锁定原文1–3个关键句。
2. **同义改写比对**：逐一比对选项，排除三类干扰项——①**照搬原文**（直接复述明示细节，非推断）；②**偷换概念/张冠李戴**（将文中不同事物的信息拼接成新选项）；③**过度推断**（推导步数超过一步，加入原文未含的前提）。正确选项几乎从不直接使用原句，而是对隐含信息的同义转述或逻辑等价表达。

**典型应用**：高考中常见设问如“What can be inferred from...?”、“What does the author imply about...?”、“The data suggests that...”。例如，原文陈述“The company’s revenue rose by merely 0.3% this quarter, and the CEO decided to postpone the expansion plan.”，题干设问“What can be inferred about the company?”，正确推断为“Its growth has fallen short of the CEO's expectations.”——由“merely”这一态度词与“postpone”这一结果双向推导得出，原文并未明说“未达预期”，但逻辑必然如此。

## ⚠️ 高频易错点

- **将原文直接复述当作推断结论**：若四个选项中有一项将原文某句话稍作改写或直接复制，这往往是细节理解题的答案，而非推断题答案。推断题的答案必须无法在原文中找到一字不差的对应陈述。
- **推导步数过多**：只允许“一阶推导”（从原文到结论仅一步逻辑跳跃）。若选项需要先假定一个新的前提、或进行“A→B→C”的链条推理，则属于过度推断，必然错误。
- **忽略限定词导致的绝对化**：原文常通过情态动词（could, may, might）、副词（partially, relatively）等弱化断言范围。若选项将含蓄推断绝对化为肯定陈述（如把“may contribute to”改为“is the main cause of”），即篡改推断强度。
- **主次信息混淆**：题干针对次要细节提问，学生却用主旨句做推断，导致结论超出该细节的信息承载范围。
