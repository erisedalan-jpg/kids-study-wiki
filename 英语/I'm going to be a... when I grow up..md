# I'm going to be a... when I grow up.

## 句型解析

这是表达**将来职业理想**的经典句型，结合了 "be going to" 将来时和 "when" 时间从句。

---

## 📚 6-12 岁自读版

### 基本结构

**I'm going to be a + 职业 + when I grow up.**

- I'm going to be **a doctor** when I grow up. 我长大想当医生。
- She's going to be **a teacher** when she grows up. 她长大想当老师。
- He's going to be **a scientist** when he grows up. 他长大想当科学家。

### 常见职业词汇

| 职业 | 英文 |
|------|------|
| 医生 | doctor |
| 教师 | teacher |
| 科学家 | scientist |
| 工程师 | engineer |
| 程序员 | programmer |
| 艺术家 | artist |
| 音乐家 | musician |
| 运动员 | athlete |
| 宇航员 | astronaut |
| 厨师 | chef |

---

## 🎓 12+ 进阶版

### 句型扩展

1. **加上原因（because）**：
   - I'm going to be a doctor **because** I want to help sick people.

2. **加上地点（in/at）**：
   - I'm going to work **in a hospital**.

3. **使用 want to be 替换**（含义相近）：
   - I **want to be** a teacher when I grow up.
   - I **hope to become** a scientist.

4. **条件句扩展**：
   - **If** I study hard, I **will** be a doctor. 如果我努力学习，我会成为一名医生。

### 职业句型变化

| 句型 | 例子 |
|------|------|
| I'm going to be + 职业 | I'm going to be a teacher. |
| I want to be + 职业 | I want to be a doctor. |
| I hope to become + 职业 | I hope to become a scientist. |
| I'd like to be + 职业 | I'd like to be an engineer. |
| My dream is to be + 职业 | My dream is to be an astronaut. |

## 对话示例（中英对照）

| 角色 | 英文 | 中文 |
|------|------|------|
| A | What are you going to be when you grow up? | 你长大想做什么？ |
| B | I'm going to be a scientist. I love doing experiments! | 我想成为一名科学家。我喜欢做实验！ |
| A | That's cool! What kind of scientist? | 真酷！什么类型的科学家？ |
| B | I'm interested in biology. Maybe a biologist. | 我对生物学感兴趣。也许是生物学家。 |
| A | You'll need to study hard then! | 那你需要努力学习！ |
| B | I know. I'm going to study science every day. | 我知道。我打算每天学习理科。 |
