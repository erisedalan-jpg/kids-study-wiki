# 关系代词

> **一句话**：关系代词是引导定语从句、代替先行词并在从句中充当主语、宾语、表语或定语成分的代词。
> **English**: A relative pronoun is a pronoun that introduces a relative clause, refers back to an antecedent, and functions as a subject, object, predicative, or attributive within that clause.

## 🎓 考点精讲

**定义与核心原理**

关系代词是定语从句的引导词，身兼三职：**①连接主句与从句**；**②指代先行词（名词或代词）**；**③在从句中充当成分**（主语、宾语、表语、定语）。高中阶段需掌握的关系代词有：`who`, `whom`, `whose`, `which`, `that`, `as`。

选词的判定依据遵循如下核心规则：
- **who**：先行词指人，在从句中作主语或宾语（口语中可替代whom）。
- **whom**：先行词指人，在从句中作宾语，正式文体中不可用who替代。
- **whose**：指人或物，在从句中作定语，修饰名词，结构为 `whose + n.`。
- **which**：先行词指物，在从句中作主语或宾语。
- **that**：可指人或物，在从句中作主语、宾语或表语，常用于限制性定语从句。
- **as**：常用于 `such...as`, `the same...as` 结构，或引导非限制性定语从句（指代整个主句内容：“正如”）。

**关键方法：高考解题三步法**

解题时，面对语法填空或短文改错中的定语从句设空，采用以下步骤：
1. **确定先行词**：找出被修饰的名词或代词，判断其属性（人/物/整个句子）。
2. **分析从句成分**：将先行词代入从句，判断缺失的成分（主语？宾语？定语？）。
   - 若缺少主语：选 `who/that`（指人）或 `which/that`（指物）。
   - 若缺少宾语：选 `whom/that/不填`（指人）或 `which/that/不填`（指物）。
   - 若缺少定语且与名词是所属关系：唯一选择 `whose`。
3. **检查特殊情况**：介词后宾语只能用 `whom` 或 `which`，不可用 `that` 或省略；非限制性定语从句中不可用 `that` 引导。

**典型应用**

高考考查该考点的主要题型为 **语法填空**（无提示词类）和 **短文改错**（错用/漏用关系词）。

- **场景一（语法填空）**：`The woman ________ son won the first prize in the competition is my English teacher.`
  判定：先行词 `The woman`（指人）；从句 `son won the first prize` 中，`son` 与先行词是所属关系（“她的儿子”），需用定语。故填 `whose`。

- **场景二（短文改错）**：`This is the very book which I have been looking for, that is a bestseller.`
  改错点：先行词 `book` 被 `the very` 修饰时，关系代词指物只能用 `that` 而非 `which`；非限制性定语从句引导词不能用 `that`，应改为 `which`。修正：`This is the very book that I have been looking for, which is a bestseller.`

## ⚠️ 高频易错点

- **that 与 which 混淆**：学生常忽略只能用 `that` 的九种情况（先行词被最高级、序数词、`the very`/`the only` 修饰；先行词是 `all`, `everything`, `nothing` 等不定代词；先行词既有人又有物等），以及在非限制性定语从句或介词后误用 `that`。
- **whose 误判**：错误地将 `whose` 只与“人”关联。`whose` 也可指物，表示“事物的……”，如 `a house whose roof is blue`。
- **关系代词与关系副词混用**：解题时未先分析从句内部成分是否完整。若从句缺主语/宾语/定语，用关系代词；若从句成分完整且缺状语（时间、地点、原因），才用关系副词 `when/where/why`。
- **介词+关系代词结构**：误将 `介词 + whom/which` 写成 `介词 + who/that`，且忘记该结构中不可省略关系代词。
