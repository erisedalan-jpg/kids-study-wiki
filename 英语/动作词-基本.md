# 动作词-基本

> **一句话**：跑、跳、走、坐——用身体做出动作，用英文说出来。
> **English**: Basic action verbs: run, jump, walk, sit, stand, eat, drink.

---

## 🧒 给 3-6 岁（共读版）

我们来玩一个游戏——我说英文，你来做动作！

- **run** — 跑
- **jump** — 跳
- **walk** — 走
- **sit** — 坐
- **stand** — 站
- **eat** — 吃
- **drink** — 喝

**共读小活动：** 玩 "Simon Says"（西蒙说）游戏！

妈妈或爸爸说：**"Simon says, jump!"** → 孩子跳起来
说：**"Simon says, sit down!"** → 孩子坐下来
说：**"Stand up!"**（没有说 Simon says）→ 孩子不能动！

规则：只有说了 "Simon says" 才能做动作。没说 "Simon says" 就做了，就输了！

## 📚 给 6-12 岁（自读版）

### 更多常用动作词

| 动词 | 中文 | 动词 | 中文 |
|-----|-----|-----|-----|
| sleep | 睡觉 | read | 读、看书 |
| write | 写 | sing | 唱歌 |
| dance | 跳舞 | play | 玩 |
| swim | 游泳 | climb | 爬 |
| fly | 飞 | draw | 画画 |

### 现在进行时：-ing 变化

当动作**正在发生**时，动词后面加 **-ing**：

- I **run** every morning. → 我每天早上跑步。（习惯）
- I **am running** right now. → 我现在正在跑。（正在发生）

加 -ing 的规则：
- 一般直接加：walk → walking, eat → eating, sleep → sleeping
- 以单个辅音结尾的短元音动词，需双写最后一个字母：run → r**unn**ing, sit → s**itt**ing, swim → sw**imm**ing

### 动词三态（不规则变化）

英语动词有**原形、过去式、过去分词**三种形态。不规则动词需要单独记：

| 原形 | 过去式 | 过去分词 |
|-----|-------|---------|
| eat | ate | eaten |
| run | ran | run |
| sit | sat | sat |
| swim | swam | swum |
| drink | drank | drunk |

规则动词只需加 -ed：walk → walked, jump → jumped, dance → danced。

### 一般现在时 vs 现在进行时

- **I eat breakfast every day.** — 我每天吃早饭。（规律习惯）
- **I am eating breakfast now.** — 我现在正在吃早饭。（此刻正在）

区别关键词：
- 表示习惯：every day, always, usually, often
- 表示此刻：now, right now, at the moment

## 🎓 给 12+（进阶版）

### 英语动词的复杂体系

英语动词的变化比中文复杂得多。中文靠时间副词表示时态（"昨天""明天""已经"），英语靠动词变形本身。一个英语动词涉及四个维度：

- **时态（tense）**：过去 / 现在 / 将来
- **体（aspect）**：简单体 / 进行体 / 完成体
- **语态（voice）**：主动 / 被动
- **语气（mood）**：陈述 / 虚拟 / 命令

仅"吃"这一个词，就有：eat / eats / ate / is eating / was eating / has eaten / had eaten / will eat / will have eaten……

### 及物动词 vs 不及物动词

- **及物动词（transitive）**：后面必须跟宾语。例：eat（吃什么？）→ eat an apple
- **不及物动词（intransitive）**：不跟宾语。例：sleep（不说 sleep something）、arrive

有些动词两用：run（跑，不及物）/ run a company（经营公司，及物）。

### 短语动词（phrasal verbs）

英语动词加上介词或副词，意思会完全改变，这是英语的一大难点：

| 短语动词 | 意思 | 例句 |
|--------|-----|-----|
| give up | 放弃 | Don't give up! |
| sit down | 坐下 | Please sit down. |
| run away | 逃跑 | The cat ran away. |
| stand up | 站起来 | Stand up, please. |
| wake up | 醒来 | I wake up at 7. |
| eat out | 出去吃饭 | Let's eat out tonight. |

短语动词是英语母语者日常会话的核心，但对学英语的人来说常常难以预测意思，需要大量接触和记忆。

### 中文 vs 英语动词逻辑

中文表示"我昨天去了图书馆"，靠"昨天"这个时间词；英语靠 went（go 的过去式）这个动词本身的变形。中文通过**词汇手段**，英语通过**语法手段**。这种根本性的差异，解释了为什么中国人学英语时常常忘记时态——因为母语里不需要这个系统。

## 📖 相关绘本

- **《From Head to Toe》** — Eric Carle 著。每翻一页一种动物做一个动作，孩子跟着模仿——"I can do it!" 全书充满 jump, kick, clap, stomp 等动作词，边读边做效果极好。适龄：2-6 岁。（真实存在）
- **《We're Going on a Bear Hunt》** — Michael Rosen 著。一家人去找熊，穿越草地、河流、雪地……大量动作词 swishing, splashing, squelching 在重复节奏中自然习得，是有声书的绝佳选择。适龄：3-7 岁。（真实存在）

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么 running 有两个 n，walking 只有一个 k？"** → 可以说："英语里有个规则：如果一个词最后是'一个元音字母加一个辅音字母'，变成-ing时要把辅音字母双写。run 的结尾是 u-n（短元音加辅音），所以双写变 runn-ing。walk 的结尾是 l-k（两个辅音），不需要双写，直接加 -ing。"

- 孩子可能问 **"Simon Says 为什么叫 Simon Says？"** → 可以说："Simon 只是一个名字，这个游戏的意思是'只有 Simon 允许你才能做'。类似的游戏在法国叫 'Jacques a dit'，在日语叫 'だるまさんがころんだ'，不同国家有不同的名字，但规则类似。"

- **共读小活动：** 全家玩 Simon Says，让孩子轮流当 Simon 发出指令，用英文说动作词，其他家人来做——孩子当"老师"时学习效果往往最好。
