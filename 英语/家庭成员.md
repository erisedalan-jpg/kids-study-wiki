# 家庭成员

> **一句话**：爸爸、妈妈、哥哥、姐姐……用英文介绍家里的人。
> **English**: Family members in English: how to name everyone in your family.

---

## 🧒 给 3-6 岁（共读版）

你的家里有哪些人？我们用英文说说看！

- **mom / mommy** — 妈妈
- **dad / daddy** — 爸爸
- **brother** — 哥哥 / 弟弟
- **sister** — 姐姐 / 妹妹
- **grandpa** — 爷爷 / 外公
- **grandma** — 奶奶 / 外婆
- **baby** — 宝宝

介绍家人可以说：**"This is my mom."**（这是我妈妈。）或者 **"I have one brother."**（我有一个兄弟。）

**共读小活动：** 找出一张家庭照片，让孩子用英文给每个人"贴标签"——指着爸爸说 "dad"，指着奶奶说 "grandma"。看看能说出几个！

## 📚 给 6-12 岁（自读版）

### 扩展家庭成员

除了直系家庭，还有更多亲戚词汇：

| 中文 | English |
|------|---------|
| 叔叔 / 舅舅 / 姑父 / 姨父 | uncle（统称） |
| 阿姨 / 姑姑 / 舅妈 / 婶婶 | aunt（统称） |
| 表哥 / 堂弟 / 表姐 / 堂妹 | cousin（统称，不分男女） |
| 侄子 / 外甥 | nephew |
| 侄女 / 外甥女 | niece |
| 祖父 / 外祖父 | grandfather / grandpa |
| 祖母 / 外祖母 | grandmother / grandma |

### 中英文亲戚称谓的重要差异

英语亲戚词汇比中文简单得多，但这背后反映了文化差异：

- **英语不区分父系和母系**：爷爷（父亲的父亲）和外公（母亲的父亲）在英语里都叫 **grandpa**。奶奶和外婆都叫 **grandma**。
- **aunt 一词包含四种角色**：中文里姨妈、舅妈、姑姑、婶婶是四个不同的人，英语统统叫 aunt。
- **uncle** 同样包含：叔叔、伯伯、舅舅、姑父、姨父。

所以如果你想说清楚，可以加上名字：**"My aunt Lisa"** 或者解释关系 "my mom's sister"。

### formal vs informal 语体差异

| 正式 / 书面 | 非正式 / 口语 |
|-----------|-------------|
| mother | mom / mommy（更亲昵） |
| father | dad / daddy（更亲昵） |
| grandmother | grandma / granny / nana |
| grandfather | grandpa / gramps |

## 🎓 给 12+（进阶版）

### 英语国家的家庭文化

英语国家（尤其是北美和英国）的主流家庭模式是**核心家庭**（nuclear family）：父母加上未成年子女，独立生活。祖父母、叔舅等亲戚组成的大家庭叫 **extended family**，通常不住在一起。

这与许多亚洲文化的三代同堂模式不同，也解释了为什么英语的亲戚词汇没那么细致——远亲称呼不需要精确区分，因为日常接触本来就不如中文文化中那么频繁。

### 现代家庭结构的多样性

现代英语词汇也在适应多样化的家庭形式：

| 英文 | 含义 |
|------|------|
| step-mother / step-father | 继母 / 继父 |
| step-sister / step-brother | 继姐妹 / 继兄弟 |
| half-sister / half-brother | 同父异母或同母异父的姐妹 / 兄弟 |
| adoptive parent | 养父母 |
| guardian | 监护人 |
| single-parent family | 单亲家庭 |
| same-sex parents | 同性父母（two moms, two dads） |

### 词源小知识

- **family** 来自拉丁语 **familia**，原意是"家中的奴仆和家人的总称"，后来缩小为只指家人。
- **kin**（亲属）是英语里更古老的词，来自古英语 **cynn**，与德语 **Kind**（孩子）同源。
- 英语有 "next of kin" 的表达，意为"最近的亲属"，常出现在法律和医疗场合。
- **sibling**（兄弟姐妹的统称）是现代英语中较正式的词，在日常口语里不常用，但在写作和正式场合很有用，避免每次都说 "brother or sister"。

## 📖 相关绘本

- **《家庭这本书》（The Family Book）** — Todd Parr 著。用简单的文字和明亮的色彩展示各种各样的家庭形态，告诉孩子所有家庭都是特别的。适龄：3-6 岁共读。（真实存在）
- **《乔科的妈妈》（A Mother for Choco）** — Keiko Kasza 著。小鸟 Choco 找妈妈的故事，引出"家庭是爱的连接"的主题。适龄：3-6 岁共读。（真实存在）
- **《晚安月亮》（Goodnight Moon）** — Margaret Wise Brown 著。经典睡前绘本，在温暖的家庭场景中帮助孩子认识房间里的各种事物。适龄：1-4 岁共读。（真实存在）

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么爷爷和外公都叫 grandpa？"** → 可以说："因为英语不像中文区分是爸爸这边的还是妈妈那边的，两个爷爷级别的男性都叫 grandpa。如果想说清楚，可以说'爸爸的 grandpa'或'妈妈的 grandpa'。这说明每种语言的分类方法不一样。"

- 孩子可能问 **"我的 cousin 是哥哥还是弟弟？"** → 可以说："英语里 cousin 不分男女、不分大小、不分父系母系，只要是同辈的非直系亲属，都叫 cousin。你可以说'my older cousin'来表示比你大的表哥/堂哥。"

- **共读小活动：** 用家庭合影，让孩子充当"解说员"，用英文逐一介绍每个人："This is my dad, this is my grandma..."，练习口语描述能力。
