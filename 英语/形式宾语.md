# 形式宾语

> **一句话**：形式宾语是用 it 占据宾语位置，而将真正的不定式、动名词或从句宾语后移至补足语之后的结构，使句子平衡自然。
> **English**: A formal object is the pronoun "it" used as a placeholder for a real object (an infinitive, gerund, or clause) that appears later after an object complement.

## 🎓 考点精讲

**定义/原理**  
形式宾语是英语中为避免句子“头重脚轻”而设立的句法手段。当宾语是较长的非谓语短语或从句，同时又带有宾语补足语时，必须将真正宾语后置，并用 it 填入原本的宾语空位。其基本结构为：  
**主语 + 谓语 + it + 宾语补足语 + 真正宾语**（to do / doing / that 从句等）。

常用动词包括：think, find, make, feel, consider, believe 等。例如：  
- I find **it** difficult **to solve the problem**. （it 为形式宾语，真正宾语为 to solve…）  
- She made **it** a rule **to read for half an hour every day**.  

此外，在 hate、like、appreciate 等动词后，若接 when / if 等引导的从句，也常先用 it 作形式宾语：  
- I hate **it** **when people interrupt me**.  

**关键方法**  
高考解题时，遵循“看动词—看补语—判后置”三步：  
1. 确认谓语动词是否属于上述可接复合宾语（宾语+宾补）的类别。  
2. 观察动词后是否有宾语补足语（形容词、名词等）。  
3. 若补足语后紧跟不定式、动名词或从句，则动词后必须填入形式宾语 it。  
语法填空：动词与补足语之间留空 → 填入 it。短文改错：若缺失 it 或误用 that / this → 增补或替换为 it。

**典型应用**  
高考语法填空常直接在 find / think 后设空考查 it，如：  
`I consider ______ necessary to tell him the truth.` → it。  
短文改错中，常将正确表达式误写为“think that important to do” → 去掉 that，改为 it。在书面表达中，主动运用“I find it convenient to...”等结构能提升语言档次。

## ⚠️ 高频易错点

- **遗漏形式宾语**：直接将真正宾语放在补足语前，造成错句，如 *I find to learn English difficult. 应改为 I find it difficult to learn English.  
- **在不需形式宾语的动词后误加 it**：如 *I like it reading. （like 后可接动名词但不用形式宾语，应去掉 it）  
- **与强调句结构混淆**：把形式宾语和 It is... that... 强调句杂糅，写出 *It was the book that I found it interesting. （应删去第二个 it）  
- **真正宾语形式选择错误**：find it difficult 后应用不定式而非动名词；而某些固定搭配如 have difficulty (in) doing 不涉及形式宾语，切忌套用 it。
