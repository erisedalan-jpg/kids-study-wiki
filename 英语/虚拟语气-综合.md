## 🎓 虚拟语气——综合梳理

**定义**：虚拟语气（Subjunctive Mood）用于表达与事实相反的假设、愿望、建议或想象，谓语动词形式与陈述语气不同。高考语法填空、改错的超高频考点。

---

### 一、三大基本类型

| 类型 | 条件从句 | 主句 |
|------|---------|------|
| 与现在事实相反 | if + 主语 + did/were | 主语 + would/could/should/might + do |
| 与过去事实相反 | if + 主语 + had done | 主语 + would/could/should/might + have done |
| 与将来相反/不太可能 | if + 主语 + did / were to do / should do | 主语 + would/could/should/might + do |

---

### 二、wish / as if / would rather / it's (high) time

| 结构 | 虚拟形式 | 含义 |
|------|---------|------|
| wish + 宾语从句（对现在） | 过去式（be → were） | 但愿现在…… |
| wish + 宾语从句（对过去） | 过去完成时 | 但愿当时…… |
| wish + 宾语从句（对将来） | would/could + do | 但愿将来…… |
| as if / as though | 过去式（同时）/ 过去完成（早于主句） | 好像…… |
| would rather + 从句 | 过去式（他人做）| 宁愿某人…… |
| It's time (that)... | 过去式 | 该是……的时候了 |

---

### 三、混合时间虚拟（选必三重点）

条件从句与主句所指时间不同时，时态各自独立：

| 中文 | English |
|------|---------|
| 如果我昨天没吃那颗药，今天就会更难受了。 | If I hadn't taken the pill yesterday, I would feel worse today. |
| 如果她是个勤奋的人，她早就成功了。 | If she were diligent, she would have succeeded long ago. |
| 如果他当年学了医，现在就能治疗这种病了。 | If he had studied medicine then, he could treat this disease now. |

---

### 四、例句（中英对照）

| 中文 | English |
|------|---------|
| 如果我是你，我就不会那样做。 | If I were you, I wouldn't do that. |
| 要是我当时认真复习就好了。 | I wish I had reviewed carefully at that time. |
| 他表现得好像他懂所有事情一样。 | He acts as if he knew everything. |
| 我宁愿你明天再来。 | I would rather you came tomorrow. |
| 是时候我们采取行动了。 | It's time that we took action. |
| 若非有你的帮助，我早就失败了。 | Were it not for your help, I would have failed. |

---

### 高考考点

1. **语法填空**：最常考 if 条件句时态对应；混合时间句提示词（yesterday/now/today）
2. **改错**：条件从句误用 would；wish 宾语从句时态错误
3. **写作**：虚拟语气假设分析是高分句型（If I were.../ Had I known...）

### 易错点

- ❌ If I would be you... → ✅ If I were you...（条件从句不用 would）
- ❌ I wish I study harder... → ✅ I wish I had studied harder.（对过去的遗憾用过去完成时）
