# 连词

> **一句话**：连词是连接词、短语、从句或句子的虚词，在句中不单独充当成分，主要表示并列、转折、因果、时间、条件等逻辑关系。
> **English**: A conjunction is a word used to connect clauses or sentences or to coordinate words in the same clause, showing their logical relationship.

## 🎓 考点精讲

连词分为 **并列连词**（coordinating conjunctions）和 **从属连词**（subordinating conjunctions）。并列连词（如 and, but, or, so, for, nor, yet）连接语法上对等的成分；从属连词（如 because, if, when, although, so that）引导状语从句，使从句依附于主句。高考对连词的考查核心是 **根据前后句之间的逻辑关系选择恰当的连词**，并在语法填空、短文改错和写作中准确使用。

**关键方法**：解题时先判断空格连接的两个部分之间的逻辑关系：并列、转折、选择、因果、时间、条件、让步、对比等。对于状语从句，要抓住主句和从句之间的逻辑，如时间先后（when/while/as）、条件（if/unless）、原因（because/since/as）、让步（although/though）等。同时必须注意：连词不能与副词混淆，如 however 是副词，不能像 but 一样直接连接两个并列句；therefore 不能替代 so 任意连接句子。此外，英语中 **because 与 so 不可并用**，**although 与 but 不可并用**，这是短文改错的高频考点。

**典型应用**：高考英语中，连词考查集中在语法填空和短文改错。语法填空常设置情景，例如：I was about to leave ______ the phone rang.（答案：when，表示“这时突然”）。短文改错常见的错误如：Though he is old, but he is still active. → 去掉 but。此外，阅读理解和七选五中，连词也是理解句间逻辑的重要线索；书面表达中，连词使用的准确性和多样性直接影响衔接分。

## ⚠️ 高频易错点

- **副词误作连词**：将 however、therefore、instead 等副词当连词使用，直接连接两个独立句子而不用分号或连词，导致 run-on 句或逗号拼接错误。
- **连词重复使用**：在同一个句子中同时使用 because 和 so，或者 although 和 but，这是中文思维负迁移的典型错误。
- **时间连词混淆**：when 可表示“正在这时（突然）”（常与过去进行时连用），while 强调两个动作同时进行且从句多用延续性动词，as 表示“随着”或“一边……一边”。不加辨析会导致语法填空失分。
- **逻辑关系误判**：把让步当成转折，在应使用 although/though 的语境中误填 but；或者在表示因果时误用并列连词 so 代替 because 引导的原因状语从句。
