# 食物-主食

> **一句话**：米饭、面条、面包……每天吃的主食，用英文说出来。
> **English**: Staple foods we eat every day — rice, noodles, bread and more.

---

## 🧒 给 3-6 岁（共读版）

今天你吃了什么？我们用英文说说看！

- **rice** — 米饭（白白的，软软的）
- **noodles** — 面条（长长的）
- **bread** — 面包（软的或者脆的）
- **egg** — 蛋（一个蛋叫 an egg，很多蛋叫 eggs）
- **milk** — 牛奶（白色的，喝起来香香的）
- **meat** — 肉（猪肉、鸡肉……都叫 meat）

问今天吃了什么可以说：**"What did you eat today?"** 回答：**"I ate rice and eggs."**

**共读小活动：** 早饭的时候，让孩子用英文说出今天早上吃的食物名字！说出来才能开吃。

---

## 📚 给 6-12 岁（自读版）

### 更多主食词汇

- **pasta** — 意大利面（通常是细长或管状的，是西方的重要主食）
- **dumplings** — 饺子（包子也常用 bun / steamed bun）
- **porridge** — 粥（燕麦粥、大米粥都叫 porridge）
- **soup** — 汤
- **toast** — 烤面包片
- **cereal** — 麦片（西方早餐常见）
- **potato** — 土豆 / 马铃薯（西方的重要主食，复数 potatoes）
- **corn / maize** — 玉米

### 中西主食大不同

东西方的主食差异很大：

| 东方 | 西方 |
|------|------|
| rice 米饭 | bread 面包 |
| noodles 面条 | pasta 意大利面 |
| congee / porridge 粥 | cereal 麦片 |
| dumplings 饺子 | sandwich 三明治 |
| mantou 馒头 | baguette 法棍（来自法语） |

**有趣的是**：意大利面（pasta）和中国面条（noodles）的关系——有说法认为马可·波罗把中国面条带回了意大利，但历史学家认为这个说法可能是误传，意大利面可能是独立发展的。（**待家长核对历史细节**）

### 可数与不可数主食

| 不可数（不能说 a rice） | 可数（可以说 an egg） |
|----------------------|-------------------|
| rice 米饭 | egg 蛋 |
| bread 面包 | dumpling 饺子 |
| milk 牛奶 | noodle 一根面条（但通常用复数 noodles） |
| meat 肉 | potato 土豆 |
| soup 汤（泛指） | sandwich 三明治 |

要说"一碗米饭"，要用量词：**a bowl of rice**（不说 "a rice"）。

### 三餐英文

- **breakfast** — 早餐
- **lunch** — 午餐
- **dinner** — 晚餐（也叫 supper，尤其在英国）
- **brunch** — 早午餐（breakfast + lunch 合并，通常在周末）
- **snack** — 零食 / 小吃

## 🎓 给 12+（进阶版）

### 饮食文化与英语语境

英语国家的饮食文化与中国有显著差异，理解这些差异能帮助你更自然地使用相关词汇：

- **西方主食以 bread 为核心**，三明治（sandwich）是午餐标配。这个词来自 18 世纪英国的三明治伯爵（Earl of Sandwich），据说他喜欢在打牌时把肉夹在面包里吃以免弄脏手，这种吃法就以他的爵位命名。（**待家长核对具体史料**）
- **早餐文化差异**：英式早餐（Full English Breakfast）包含 fried eggs, bacon, sausages, beans, toast；中式早餐有粥、包子、豆浆；美式早餐常见 pancakes（薄饼）和 syrup（糖浆）。

### 餐饮场合词汇

| 英文 | 含义 |
|------|------|
| menu | 菜单 |
| order | 点菜 |
| appetizer / starter | 前菜 / 开胃菜 |
| main course / entrée | 主菜 |
| dessert | 甜点 |
| takeaway / takeout | 外卖（英式 takeaway，美式 takeout） |
| tip | 小费（西方餐饮文化中很重要） |
| vegan | 纯素食（不吃任何动物产品） |
| vegetarian | 素食（不吃肉，但可能吃蛋奶） |

### 主食相关习语

| 习语 | 含义 |
|------|------|
| bring home the bacon | 养家糊口，赚钱养家 |
| easy as pie | 容易得很，小菜一碟 |
| piece of cake | 轻而易举（和 easy as pie 一样意思） |
| the greatest thing since sliced bread | 极为了不起的发明或事物 |
| half-baked | 没想清楚的、不成熟的（半生不熟的面包） |
| use your noodle | 动动脑子（noodle 在口语里也指脑袋！） |
| full of beans | 精力充沛，活力十足 |

| 意大利面 | pasta | n. (不可数) | Pasta is popular in Italy. |

### 例句（可朗读对照）

| 中文 | English |
|------|---------|
| 今天早饭你吃了什么？ | What did you have for breakfast today? |
| 我吃了米饭和鸡蛋。 | I had rice and eggs. |

## 📖 相关绘本

- **《面包和果酱给弗朗西丝》（Bread and Jam for Frances）** — Russell Hoban 著。小獾 Frances 只爱吃面包和果酱，拒绝其他食物，直到她学会了接受多样的食物。适龄：4-7 岁共读。（真实存在）
- **《石头汤》（Stone Soup）** — Marcia Brown 著。经典民间故事：士兵用石头煮汤，村民们一个个带来食材，汤越来越香——关于分享和社区的故事。适龄：5-8 岁。（真实存在）

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么 rice 不能说 a rice？"** → 可以说："因为米饭是不可数名词——你想想看，米饭是一粒一粒的，很难说是'一个'还是'两个'，英语就把它归为不可数名词。如果要说数量，就用容器：a bowl of rice（一碗米饭）。"

- 孩子可能问 **"brunch 是什么？"** → 可以说："brunch 是 breakfast（早餐）和 lunch（午餐）合在一起造出来的词，叫'早午餐'，通常在周末十点左右吃，因为大家睡懒觉，错过了早饭，又还没到午饭时间，就合在一起吃了。"

- **共读小活动：** 画一张"今日菜单"，让孩子把今天三餐吃的东西写在上面（可以中文写，家长帮注英文），贴在冰箱上。
