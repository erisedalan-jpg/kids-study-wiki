## 🎓 高考语法综合复习

**定位**：高考英语语法题型全面梳理，含语法填空、完形填空、改错和书面表达的核心语法体系。

### 五大语法考查体系

#### 1. 时态语态体系（语法填空核心）

| 时态 | 标志词/情景 | 主动 | 被动 |
|------|-----------|------|------|
| 一般现在时 | always/usually/every day | do/does | is/are done |
| 一般过去时 | yesterday/last year/in 2020 | did | was/were done |
| 一般将来时 | tomorrow/next week | will do | will be done |
| 现在进行时 | now/at this moment | is/are doing | is/are being done |
| 现在完成时 | already/yet/since/for | has/have done | has/have been done |
| 过去完成时 | by then/before/after（过去基准）| had done | had been done |

#### 2. 从句体系

| 从句类型 | 引导词 |
|---------|--------|
| 名词性从句（主/宾/表/同位）| that, whether, if, wh- 词, wh-ever |
| 定语从句 | who, whom, whose, which, that, where, when, why |
| 状语从句（时间/原因/条件/目的/结果/让步/方式）| 各类连词 |

#### 3. 非谓语动词体系

| 形式 | 核心用法 | 判断关键 |
|------|---------|---------|
| to do | 目的/将来/逻辑主语主动 | 看"目的"或"潜在动作" |
| -ing | 主动/进行/伴随 | 看"主动"关系 |
| -ed | 被动/完成 | 看"被动"关系 |

#### 4. 特殊句型体系

| 句型 | 结构 | 示例 |
|------|------|------|
| 强调句 | It is/was + 被强调成分 + that/who | It is hard work that counts. |
| 倒装句 | 否定/频率副词前置 → 谓语前置 | Never have I seen such beauty. |
| 虚拟语气 | If + 过去式/had done + would + do/have done | If I were you, I would try again. |
| 省略句 | 从句主语+be 动词可省略 | When (he was) young, he worked hard. |

#### 5. 高频词汇搭配体系

| 类型 | 高频搭配 |
|------|---------|
| 名词 + 介词 | impact on, approach to, access to, contribution to |
| 动词 + 介词 | focus on, result in, depend on, apply to |
| 形容词 + 介词 | aware of, responsible for, capable of, familiar with |

### 例句（中英对照）

| 中文 | English | 考查点 |
|------|---------|--------|
| 这座桥建于1920年。 | The bridge was built in 1920. | 被动语态+过去时 |
| 努力工作才是真正重要的。 | It is hard work that truly matters. | 强调句 |
| 如果我是你，我会再试一次。 | If I were you, I would try again. | 虚拟语气 |
| 我从未见过如此壮观的景色。 | Never have I seen such spectacular scenery. | 否定倒装 |
| 这项工作已经完成了三个月了。 | The project has been completed for three months. | 现在完成时被动 |
| 无论你做什么，都要全力以赴。 | Whatever you do, do it wholeheartedly. | wh-ever 让步 |

### 高考书面表达高分语法策略

1. **时态准确**：通篇时态一致，记叙文过去时，说明文/议论文一般现在时
2. **被动语态灵活使用**：学术语体中增加被动句密度
3. **每段至少一个高级句型**：强调句/倒装句/定语从句/非谓语
4. **避免低级错误**：主谓一致、冠词使用、单复数

### 易错点

- 强调句与定语从句的区分：去掉 It is...that，句子仍完整 → 强调句；不完整 → 定语从句
- 倒装句语序：谓语/助动词 + 主语（Not only did he...; Hardly had he...when...）
