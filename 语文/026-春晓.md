# 春晓

> **一句话**: 孟浩然在春天早晨睡醒，听到处处鸟鸣，想起夜里风雨声，不知道花儿落了多少——一首诗写出了春天的惜花之情。
> **English**: Awakening on a spring morning to birdsong everywhere, wondering how many flowers fell in last night's rain — a poem of gentle sorrow for fleeting spring.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**: 唐 · 孟浩然
**体裁**: 五言绝句

```
春晓
孟浩然

春眠不觉晓，
处处闻啼鸟。
夜来风雨声，
花落知多少。
```

**拼音**:
- chūn mián bù jué xiǎo，
- chù chù wén tí niǎo。
- yè lái fēng yǔ shēng，
- huā luò zhī duō shǎo。

**白话译文**:
春天的夜晚睡得很香，不知不觉天亮了；
醒来只听得处处传来鸟儿的叫声。
想起昨晚似乎听到了风声和雨声——
不知道那些美丽的花儿，究竟落了多少呢？

**朗读节奏**:
春眠 / 不觉晓，处处 / 闻啼鸟。
夜来 / 风雨声，花落 / 知多少。

## 🧒 给 3-6 岁（共读版）

春天的早晨，你是不是特别不想起床？睡得香香的，突然听到小鸟"叽叽喳喳"的叫声……

孟浩然也有过这样的早晨！他睡得太香了，不知道天已经亮了（"不觉晓"）。醒来一听，到处都是鸟叫声（"处处闻啼鸟"）。

然后他想：哦，昨晚好像下过雨（"夜来风雨声"）……风吹雨打，那院子里那些漂亮的花，不知道落了多少啊（"花落知多少"）？

春天的花很美，但一场风雨就会落掉，是不是有点可惜？

**共读小活动**:
- **听一听**: 早晨起床，开窗听听外面有没有鸟叫，找一找哪里来的声音。
- **画一画**: 画一个春天的早晨：树上开着花，树下落满花瓣，小鸟站在枝头唱歌。
- **说一说**: 你最喜欢春天的什么？花？小鸟？还是下雨？

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 春晓 | chūn xiǎo | 春天的早晨，"晓"是黎明、天亮 |
| 眠 | mián | 睡觉 |
| 不觉晓 | bù jué xiǎo | 没有察觉到天已经亮了 |
| 处处 | chù chù | 到处，各个地方 |
| 闻 | wén | 听到（古文义；不是"闻味道"） |
| 啼鸟 | tí niǎo | 鸣叫的鸟儿，"啼"是鸟叫、啼叫 |
| 夜来 | yè lái | 昨夜，夜里来的（即昨晚） |
| 知多少 | zhī duō shǎo | 知道有多少（反问，表示不知道多少） |

### 内容与情感

全诗时间线是诗人的思绪流动：
1. **此刻**: 天亮了，没觉察（春眠不觉晓）
2. **此刻**: 听到鸟叫（处处闻啼鸟）
3. **回忆**: 昨晚有风雨（夜来风雨声）
4. **担忧**: 花不知落了多少（花落知多少）

诗中没有明说悲伤，却通过"花落"表达了对春光易逝的惋惜——这是中国古典文学中"伤春"情感的典型表达。

### 写作手法

- **以声入境**: 用鸟鸣引出清晨，以风雨声勾起回忆，以声音带动情感
- **虚实结合**: 鸟鸣是眼前实景，风雨声是夜里记忆，虚实交融
- **含蓄结尾**: "花落知多少"是反问，留有余韵，不直接说出悲伤，让读者自己感受
- **感官交替**: 先听觉（鸟鸣）、再回忆听觉（风雨）、最后视觉想象（落花）

### 朗读建议

全诗节奏舒缓，像刚睡醒的人懒洋洋地说话。"处处闻啼鸟"要读出鸟声四起的欢快感；"花落知多少"语调渐轻、渐低，读出淡淡的惋惜。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**孟浩然**（689—740年），字浩然，号孟山人，湖州（今湖北襄樊）人，唐代著名山水田园诗人，与王维并称"王孟"。他一生大部分时间隐居山林，仕途不顺，但诗名极盛。

《春晓》约作于诗人隐居鹿门山时期，是他观察自然、随兴而作的代表作。全诗语言极其自然，毫无雕琢痕迹，体现了孟浩然"清淡自然"的诗风。

### 文学史地位

《春晓》是中国最著名的田园诗之一，也是"伤春"这一诗歌传统的早期代表。"伤春"（grief over passing spring）是中国文学中的重要母题，从屈原到李商隐到晏殊，代代相传。孟浩然用最简单的语言，将这一情感表达到了极致。

### 鉴赏与对比

孟浩然诗的特点是"语淡而味不薄"——语言平白，但回味无穷。"花落知多少"一句，没有悲字，却比"春花落尽真可惜"更令人动容，这是含蓄之美（implicit beauty）的典范。

与杜甫"感时花溅泪，恨别鸟惊心"的直接抒情相比，孟浩然更含蓄；与王维"独坐幽篁里，弹琴复长啸"的禅意山水相比，《春晓》更有生活气息。

### 文化影响

- 收录于《全唐诗》卷一六〇，孟浩然诗
- 小学语文教材一年级下册"古诗二首"之一
- 是"伤春""惜春"情感的代表诗，在中国文化中具有广泛认同
- 历代被谱曲演唱，是儿童音乐启蒙的经典古诗词曲材料

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么春天睡觉感觉特别舒服？"** → 可以说："因为春天温度刚刚好，不冷不热，被窝特别暖，所以容易睡过头。孟浩然也是这样！'春眠不觉晓'就是说春天的觉睡得特别香，都不知道天亮了。"

- 孩子可能问 **"花落了有什么关系？"** → 可以说："花开了很好看，但风一吹、雨一打，花就落了。诗人觉得有点可惜——美丽的东西消失了。这叫'惜花'，也有一点点'惜春'的意思，就是珍惜美好的春天。"

- **共读活动（找意象）**: 读完诗，问孩子："诗里有哪些事物？"（鸟、花、风、雨、春天）让孩子一一找出来，建立意象意识。

- **共读活动（全家朗诵）**: 选在春天早晨朗读，开窗听一听是否有鸟叫，体会"处处闻啼鸟"的真实感受。
