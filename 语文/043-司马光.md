# 司马光

> **一句话**：这是统编版小学阶段第一篇文言文，选自《资治通鉴》（宋·司马光编著），讲述了少年司马光面对同伴落水的危急情况，冷静砸缸救人的故事，是沉着冷静与随机应变精神的经典范例。
> **English**: This is the first classical Chinese text in the elementary school curriculum, selected from *Zizhi Tongjian*. It tells how the young Sima Guang calmly broke a water jar to rescue a drowning companion, demonstrating cool-headedness and quick thinking.

---

## 课文原文（文言文）

群儿戏于庭，一儿登瓮，足跌没水中。众皆弃去，光持石击瓮破之，水迸，儿得活。

---

## 白话译文

一群孩子在庭院里玩耍，有一个孩子爬上了大水缸，脚一滑，掉进水里。其他的孩子都跑掉了（不知所措），只有司马光拿起石头砸破了大水缸，水流了出来，那个孩子得救了。

## 🧒 给 3-6 岁（共读版）

很久很久以前，有个聪明的小朋友叫司马光。

有一天，司马光和很多小朋友在院子里玩。院子里有一口大水缸，装着很多水。一个小朋友爬上去，脚一滑，"扑通"一声掉进水缸里了！

其他小朋友都吓跑了，不知道怎么办。只有司马光不慌，他找了一块大石头，用力砸向水缸——砰！水缸破了，水流出来，掉进去的小朋友得救了！

司马光真是太勇敢、太聪明了！他不跑走，想到了一个好办法。

**共读小活动：** 问孩子："如果你在旁边，你会怎么做？司马光的办法好在哪里？"引导孩子思考紧急情况下的冷静和应变能力。

## 📚 给 6-12 岁（自读版）

### 文言文学习要点

**这是你在小学阶段读到的第一篇文言文！**

文言文是古代中国人使用的书面语言，和我们今天说的话（白话文）不一样。学习文言文要注意：
1. 每个字都有意思，要逐字对照
2. 借助注释理解关键词语
3. 尝试用现代话翻译出来

**朗读节奏**（斜线为停顿）：
> 群儿 / 戏于庭，一儿 / 登瓮，足跌 / 没水中。众 / 皆弃去，光 / 持石 / 击瓮 / 破之，水迸，儿 / 得活。

### 主旨理解

**中心思想**：这个故事颂扬了少年司马光在紧急情况下沉着冷静、随机应变的品质。全文仅30个字，却把人物、事件、结果交代得清清楚楚，是文言文简洁精练的典范。

### 品读提升

**对比手法**："众皆弃去"（众人都跑了）vs "光持石击瓮"（司马光砸缸）——一组对比，突出了司马光的与众不同：面对危急，别人的本能反应是逃跑，而他是思考并行动。

**动词连用**："持""击""破"三个动词，把砸缸救人的动作写得连贯有力，简短而清晰。这也是文言文的魅力——用极少的字，传递丰富的信息。

**留白**：课文没有写其他孩子为什么跑走，没有写被救孩子的反应，没有写司马光事后说了什么。这些留白，让读者的思维有了更多发挥空间。

## 🎓 给 12+（进阶版）

### 原典出处——《资治通鉴》

本文选自北宋**司马光**（1019-1086）主编的史学巨著**《资治通鉴》**（全称"《资治通鉴》"，北宋神宗赐名）。

| 项目 | 内容 |
|------|------|
| 书名 | 《资治通鉴》 |
| 作者 | 司马光（主编），刘恕、范祖禹等协助 |
| 成书时间 | 北宋神宗熙宁到元丰年间（1065-1084年，历时19年） |
| 体例 | 编年体通史 |
| 时间跨度 | 上起战国周威烈王二十三年（前403年），下至五代后周世宗显德六年（959年），共1362年 |
| 规模 | 294卷，约300万字 |
| 命名含义 | "资治"意为"帮助治国"，"通鉴"意为"通古今以为镜鉴（借鉴）" |

课文节选的这段文字出自《资治通鉴》的外传部分，是司马光幼年的逸事。

### 人物简介

**司马光**（1019-1086），字君实，北宋政治家、历史学家。他在历史上也因与王安石变法政见相左而著名。编写《资治通鉴》是他最重要的历史贡献，也是中国历史上规模最大的编年体史书之一。

### 文言文的语言特点

| 特点 | 例子 |
|------|------|
| 省略主语 | "（司马光）持石击瓮" |
| 词语精简 | "众皆弃去"（4字）= "大家都丢下（伙伴）跑掉了"（14字白话） |
| 倒装结构 | "戏于庭"（在庭院里玩）= 现代语序"于庭戏" |
| 代词"之" | "击瓮破之"中的"之"代指"瓮" |

### 教育意义与历史价值

这是统编教材小学阶段第一篇文言文，承担着启蒙任务：让孩子初步接触文言语感、了解文言与白话的区别，为日后更深入的古文学习打下基础。从内容上，这则故事体现了儒家文化中"仁"（救人）与"智"（随机应变）的双重价值。

## 🗣️ 家长讲解话术

- 孩子可能问 **"文言文是什么？"** → 可以说："文言文就是古代人写文章用的语言。大概一两千年前，古代中国人说话和写文章都用这种方式，比现在我们说的话更简短。就像你看外国动画片要字幕一样，读文言文也需要'翻译'，学会了就能读懂很多古代的故事。"

- 孩子可能问 **"'众皆弃去'是什么意思？"** → 可以说："就是'其他孩子都跑掉了'。'众'是'大家'，'皆'是'都'，'弃'是'丢下'，'去'是'离开走了'。把每个字的意思拼在一起，就能理解了。"

- 孩子可能问 **"为什么其他小朋友跑掉了？"** → 可以说："可能是太害怕了，不知道怎么办，就跑走了。这是人在紧急情况下的本能反应——惊慌、逃跑。司马光特别的地方，是他没有跑，而是停下来想办法。这需要很强的冷静能力。"

- **共读小活动（背诵）**：全文只有30个字，引导孩子一句一句地跟读，争取当天背下来。可以用"看图说古文"的方式——家长指着图，孩子说出对应的文言文。

- **共读小活动（改变情境）**：问孩子："如果没有石头，司马光还能用什么方法救人？"鼓励孩子发散思维，培养随机应变能力。
