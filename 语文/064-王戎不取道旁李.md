# 王戎不取道旁李

> **一句话**：《世说新语》中的经典小故事，七岁王戎见路旁李树果实累累却无人摘，断定"必苦李"，展示了儿童的智慧与逻辑推断能力。
> **English**: A classic tale from Shishuo Xinyu: seven-year-old Wang Rong deduces that roadside plums must be bitter because no one has picked them, demonstrating sharp childhood reasoning.

---

## 📄 文言原文（全文，共27字）

**王戎不取道旁李**（统编版语文四年级上册 第八单元 课文二十五）

> 王戎七岁，尝与诸小儿游。看道边李树多子折枝，诸儿竞走取之，唯王戎不动。人问之，答曰："树在道旁而多子，此必苦李。"取之信然。

## 🧒 给 3-6 岁（共读版）

从前有个叫**王戎**的小朋友，他只有七岁，和小伙伴们一起出去玩。

他们看到路边有一棵李子树，树上挂满了果实，树枝都被压弯了！小朋友们都跑去摘李子！

但是王戎站在那里，没有动。

"王戎，你为什么不去摘李子？"

王戎说："这棵树长在路边，这么多人从这里走来走去，果实却这么多，说明没有人想摘——一定是因为李子是苦的！"

果然，小朋友们摘来一尝，好苦啊！

王戎的办法叫**推理**：根据能看到的情况，推断出看不到的答案。很聪明吧！

**共读小活动：** 问孩子："如果你是王戎，你会怎么想？"引导孩子学习观察→思考→判断的方法。

## 📚 给 6-12 岁（自读版）

### 朗读节奏提示

> 王戎 / 七岁，尝 / 与诸小儿 / 游。看 / 道边李树 / 多子折枝，诸儿 / 竞走取之，唯 / 王戎 / 不动。人问之，答曰："树在道旁 / 而多子，此 / 必苦李。"取之 / 信然。

### 王戎的推理逻辑

王戎的推理可以整理成三段论：

1. **前提一（观察）**：李树长在路边，来往的人很多
2. **前提二（常识）**：如果李子甜，过路的人早就摘光了
3. **观察事实**：李子很多，没人摘
4. **结论（推断）**：李子没人摘，一定是苦的

这是一种**逻辑推理**的方法：从已知信息推断未知结论，比直接去尝更聪明。

### 古今异义词汇总

| 字 | 古义 | 今义 |
|---|-----|-----|
| 尝 | 曾经 | 品尝 |
| 走 | 跑 | 慢步行走 |
| 游 | 游玩 | 游泳/旅游 |

这三个字是本篇文言文最重要的古今异义词，务必记牢。

### 课文价值

本文是统编版**第二篇文言文**（四年级上册，三年级下册《守株待兔》是第一篇）。全文仅27字，却完整讲述了一个有人物、有情节、有转折、有结论的故事，体现了文言文的**简洁之美**。

## 🎓 给 12+（进阶版）

### 《世说新语》简介

《世说新语》是南朝宋刘义庆组织编写的一部笔记小说集，记录了东汉末年到晋代士人的言行轶事，共分36门（德行、言语、政事、文学……），全书约5万字，是中国古代笔记文学的代表作。

《王戎不取道旁李》收录在"雅量"篇（一说"识鉴"篇），主题是展示人物的机智识见。王戎是历史上真实人物——魏晋名士，竹林七贤之一，以智慧和机变著称。

### 竹林七贤

王戎（234-305年），是**竹林七贤**之一。竹林七贤包括：嵇康、阮籍、山涛、向秀、刘伶、阮咸、王戎，是魏晋时期一批名士的总称，他们常在竹林中聚会，崇尚老庄，追求自由，对中国文化影响深远。这则故事记录的是王戎儿时的轶事，与他成年后的形象形成有趣对照。

### 文言文的写作特点

本文体现了《世说新语》的典型风格：
1. **极度简洁**：27字讲完故事，没有一个多余的字
2. **以事见人**：不直接评价"王戎很聪明"，而是用一件事让读者自己得出结论
3. **对比手法**："诸儿竞走"vs "唯王戎不动"，一个对比已经说明一切
4. **结尾有验证**："取之信然"——故事有始有终，结论得到验证

这种以事见人、简洁有力的写法，是中国古典散文的重要传统。

## 🗣️ 家长讲解话术

- 孩子可能问 **"文言文里'走'为什么是'跑'？"** → 可以说："这叫'古今异义'——古代的'走'就是我们现代说的'跑'，古代的'行'才是我们现在说的'走'。学文言文时要特别注意这些词义变化，不能直接用现代汉语来理解，否则就会弄错意思。"

- 孩子可能问 **"王戎为什么比别的小朋友聪明？"** → 可以说："王戎不只是'聪明'，他是'会思考'。其他小朋友看到李子就想摘，王戎先想了一想：'为什么这么多李子没人摘？'这就是观察+思考。我们生活中也可以这样——遇到问题不要急，先看一看、想一想，也许就能发现答案。"

- **共读小活动**：让孩子把文言原文一句一句翻译成白话文，再和注释对照，检查是否正确；或者把文言文背诵下来（只有27字）。
