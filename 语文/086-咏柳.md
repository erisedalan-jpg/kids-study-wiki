# 咏柳

> **一句话**: 碧玉装扮的柳树，垂下万条绿丝带，是谁裁出那纤细的叶片？——二月春风，是那把最妙的剪刀！
> **English**: A willow dressed in jade green hangs a thousand silk ribbons — who cut those slender leaves? The spring breeze of February, the finest pair of scissors!

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**: 唐 · 贺知章
**体裁**: 七言绝句

```
咏柳
贺知章

碧玉妆成一树高，
万条垂下绿丝绦。
不知细叶谁裁出，
二月春风似剪刀。
```

**拼音**:
- bì yù zhuāng chéng yī shù gāo，
- wàn tiáo chuí xià lǜ sī tāo。
- bù zhī xì yè shuí cái chū，
- èr yuè chūn fēng sì jiǎn dāo。

**白话译文**:
柳树就像用碧玉打扮成的一般，高高地挺立着；
万条柳枝垂下来，像绿色的丝带飘荡。
不知道那些细细的柳叶是谁裁剪出来的，
原来是二月的春风，就像一把灵巧的剪刀！

**朗读节奏**:
碧玉 / 妆成 / 一树高，万条 / 垂下 / 绿丝绦。
不知 / 细叶 / 谁裁出，二月 / 春风 / 似剪刀。

## 🧒 给 3-6 岁（共读版）

你见过柳树吗？春天的柳树，树枝细细长长的，垂下来，绿绿的，像不像长长的发辫？

贺知章看到春天的柳树，觉得好漂亮！他觉得柳树就像用绿宝石（碧玉）做的一样好看。

那些细细的柳叶，像剪刀剪出来的——是谁剪的呢？
诗人说：是春风！二月的春风像剪刀一样，把柳叶一片片剪出来了！

哇，原来是春风在给柳树理发做造型！

**共读小活动**:
- **画一画**: 画一棵高高的柳树，树枝细细长长地垂下来，叶子是绿色的，旁边画一把剪刀代表春风。
- **摸一摸**: 如果附近有柳树，摸摸柳叶，感受它是不是真的像细丝一样？
- **想一想**: 除了柳叶，还有什么东西像是被剪刀剪出来的？（花瓣？雪花？）

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 咏柳 | yǒng liǔ | 用诗歌来赞美柳树 |
| 碧玉 | bì yù | 翠绿色的玉石（比喻柳树的颜色之美） |
| 妆成 | zhuāng chéng | 打扮成，装扮成的样子 |
| 万条 | wàn tiáo | 很多条（夸张，形容柳枝众多） |
| 垂下 | chuí xià | 向下垂挂 |
| 绿丝绦 | lǜ sī tāo | 绿色的丝绸带子（比喻柳条） |
| 细叶 | xì yè | 细小的叶片（柳叶细长） |
| 裁 | cái | 裁剪，剪出形状 |
| 似剪刀 | sì jiǎn dāo | 好像剪刀一样 |

### 内容与情感

全诗逐层推进，层层惊叹：

1. **整体**: 柳树整体之美——"碧玉妆成"（如宝石般美丽）
2. **局部**: 柳枝之美——"万条绿丝绦"（像丝带飘垂）
3. **细节**: 柳叶之纤细，引出疑问——"谁裁出？"
4. **答案**: 春风如剪刀——最后一句给出神奇的回答

诗人对柳树的热爱通过层层递进的比喻传达出来，最后以"二月春风似剪刀"的奇妙联想收尾，令人眼睛一亮。

### 写作手法

- **比喻**: "碧玉妆成"（柳树如碧玉）、"绿丝绦"（柳枝如丝带）、"似剪刀"（春风如剪刀）——三个比喻，各有妙处
- **拟人**: 春风"裁"柳叶，把风拟人化为裁缝
- **设问与作答**: "谁裁出"设问，"二月春风似剪刀"作答，结构工整
- **层次递进**: 从整体（树）到局部（枝）到细节（叶），层层聚焦

### 朗读建议

"碧玉妆成一树高"语调明亮，像在欣赏一件宝贝；"万条垂下绿丝绦"节奏稍慢，体现柳枝轻柔垂落的感觉；"二月春风似剪刀"读出恍然大悟的惊喜，语调轻快上扬。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**贺知章**（659—744年），字季真，号四明狂客，越州永兴（今浙江萧山）人，唐代著名诗人、书法家，官至秘书监。贺知章与李白是忘年之交，李白初到长安时，贺知章称赞他为"谪仙人"。

《咏柳》约作于贺知章晚年（80余岁）告老还乡前后，是他观察春柳时有感而发的即兴之作。诗人一生仕途顺遂，性格开朗，晚年诗风轻松自然，《咏柳》体现了他晚年的轻盈心态。

### 文学史地位

《咏柳》是中国最著名的咏物诗之一，也是"春天题材"和"柳"意象的代表作。在中国诗歌传统中，"柳"有多重文化含义：
- 柔软多情（依依惜别）
- 春天象征（杨柳依依）
- 美人比喻（如碧玉）

贺知章此诗以清新的比喻，将柳树的美写得既具体又奇妙，"二月春风似剪刀"这一比喻被誉为"最妙的咏春名句"之一。

### 鉴赏与对比

与同是咏柳的古诗相比，唐代韩愈"天街小雨润如酥，草色遥看近却无"写春雨滋润；杜甫"两个黄鹂鸣翠柳"以柳为背景；贺知章的《咏柳》则把柳树本身作为主角，用三个比喻，让柳树的美跃然纸上，是咏柳诗中最精彩的一首。

与骆宾王《咏鹅》（同为"咏"字题目的小学名诗）相比，《咏柳》比喻更为丰富，语言更为华美。

### 文化影响

- 收录于《全唐诗》卷一一二，贺知章诗
- 小学语文教材二年级下册古诗二首之一
- "二月春风似剪刀"是描写春风最著名的比喻，广泛用于文学和日常表达
- 柳树的文化意象（思念、柔情、春天）在中国诗歌和艺术中延续千年

## 🗣️ 家长讲解话术

- 孩子可能问 **"碧玉是什么？"** → 可以说："碧玉是一种绿色的宝石，颜色鲜绿，非常漂亮。诗人说柳树就像用碧玉做的一样，是说柳叶的绿色特别纯净好看，像宝石一样亮晶晶的。"

- 孩子可能问 **"春风怎么像剪刀？"** → 可以说："柳叶细细长长，边缘整齐，像是用剪刀剪出来的形状。诗人想：是谁把叶子剪得这么整齐？一定是春风！这是一个有趣的比喻——把无形的风比作有形的剪刀，很有创意。你能想出别的比喻吗？"

- **共读活动（找比喻）**: 读完诗，让孩子找出三个比喻（碧玉、绿丝绦、剪刀），说说每个比喻把什么比成了什么，帮助孩子理解比喻这种修辞手法。

- **共读活动（观察柳树）**: 春天找一棵柳树，仔细观察柳枝和柳叶，对照诗句，感受"绿丝绦"和"细叶"的美，体会诗人的观察力。
