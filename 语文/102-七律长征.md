# 七律·长征

> **一句话**：毛泽东以七律形式，高度概括了红军长征（1934-1936）跨越万水千山的艰苦历程，以豪迈的笔调表达了红军战胜一切艰难险阻的革命乐观主义精神。
> **English**: In the form of a seven-character regulated verse, this poem summarizes the Red Army's arduous Long March (1934–1936) across mountains and rivers, celebrating their indomitable spirit and revolutionary optimism.

---

## 🧒 给 3-6 岁（共读版）

很久以前，有一支叫做"红军"的军队，他们要走一段超级超级长的路——从南方走到北方，整整走了两年，爬过了很多大山，渡过了很多大河！

这首诗是一位叫毛泽东的人写的。他用很短的一首诗，把这段长长的旅程写了出来，说红军很勇敢，再难走的山路他们都不怕！

有一座山叫"五岭"，山很长很弯弯；还有一座山叫"乌蒙"，很高很高；还有"金沙江"水很急，"大渡河"上有一座叫"泸定桥"的铁链桥——走过这些地方，可真了不起！

**共读小活动**：和孩子在地图上找一找长征路线经过的主要省份（贵州、云南、四川、陕西），感受路程的遥远。

## 📚 给 6-12 岁（自读版）

### 原文与拼音

**七律·长征**（毛泽东，1935年10月）

```
红军不怕远征难，
Hóng jūn bù pà yuǎn zhēng nán，
万水千山只等闲。
wàn shuǐ qiān shān zhǐ děng xián。

五岭逶迤腾细浪，
Wǔ lǐng wēi yí téng xì làng，
乌蒙磅礴走泥丸。
Wū méng páng bó zǒu ní wán。

金沙水拍云崖暖，
Jīn shā shuǐ pāi yún yá nuǎn，
大渡桥横铁索寒。
Dà dù qiáo héng tiě suǒ hán。

更喜岷山千里雪，
Gèng xǐ Mín shān qiān lǐ xuě，
三军过后尽开颜。
sān jūn guò hòu jìn kāi yán。
```

### 译文（白话）

红军不怕万里长征路上的艰难险阻，那千山万水，在他们看来只是平平常常的事。
五岭山脉绵延起伏，在红军眼中不过是翻腾的细小波浪；气势磅礴的乌蒙山，在脚下不过是滚动的小泥丸。
金沙江的江水拍打着两岸高耸的云崖，让人感到温暖；大渡河上那座横跨的铁索桥（泸定桥），让人感到阵阵寒意。
最令人欣喜的是翻过了白雪皑皑的岷山，三军将士过去之后，人人喜笑颜开。

### 词语学习

| 词语 | 解释 |
|------|------|
| 远征 | 远距离的军事行动或旅程 |
| 等闲 | 平平常常，不放在心上 |
| 五岭 | 华南五座山岭的合称，山势连绵 |
| 逶迤 | 山岭等弯曲绵延的样子 |
| 乌蒙 | 贵州、云南交界处的乌蒙山，山势险峻高大 |
| 磅礴 | 气势宏大 |
| 泥丸 | 小泥球 |
| 云崖 | 高入云端的山崖 |
| 铁索 | 铁链，指泸定桥的铁链 |
| 岷山 | 四川境内的大雪山 |
| 三军 | 这里泛指红军全体将士 |
| 尽开颜 | 所有人都笑逐颜开 |

### 历史背景：红军长征（1934-1936）

| 时间 | 事件 |
|------|------|
| 1934年10月 | 中央红军从江西瑞金出发，开始长征 |
| 1935年1月 | 遵义会议（贵州），确立新的领导核心 |
| 1935年5月 | 巧渡金沙江（云南） |
| 1935年5月 | 强渡大渡河、飞夺泸定桥（四川） |
| 1935年6月 | 翻越夹金山等大雪山 |
| 1935年9-10月 | 过草地，翻越岷山 |
| 1935年10月 | 中央红军到达陕北吴起镇，长征结束（约13000里） |
| 1936年10月 | 三大主力红军在甘肃会宁会师，长征全部结束 |

这首诗写于 1935 年 10 月，红军刚翻越岷山，即将到达陕北，毛泽东以此诗庆贺胜利在望。

### 诗歌结构（七律格式）

七律共 8 行，每行 7 字，押韵（难、闲、丸、寒、颜），分为**首联、颔联、颈联、尾联**：
- **首联**（总起）：点出长征主题——"不怕难""等闲"，总领全诗；
- **颔联**（山）：五岭 + 乌蒙，以"细浪""泥丸"夸张缩小，写山之"微不足道"；
- **颈联**（水）：金沙江"暖"+ 大渡河"寒"，一暖一寒，情感对比鲜明；
- **尾联**（总结）：过岷山后"尽开颜"，以喜悦收尾。

## 🎓 给 12+（进阶版）

### 诗歌艺术特色

**1. 夸张与对比**
"五岭逶迤腾细浪，乌蒙磅礴走泥丸"——将绵延数百里的五岭缩为"细浪"，将气势磅礴的乌蒙山缩为"泥丸"，是极度夸张，目的在于突出红军蔑视困难的精神气概，而非对山的实际描写。

**2. 暖寒对比（颈联的精妙）**
金沙江水"暖"——因为巧渡成功，心情愉悦，温暖的是心情；
大渡桥铁索"寒"——泸定桥战斗极为惨烈，铁索上曾有枪林弹雨，"寒"写出了战斗的险峻与悲壮。
两字对比，将历史的艰辛与胜利的喜悦都蕴含其中。

**3. 革命浪漫主义与现实主义的统一**
诗歌以豪迈的笔调处理真实的艰苦历程，既不回避困难（远征难、铁索寒），又始终以乐观主义超越困难（等闲、尽开颜），体现了中国革命文学中"革命浪漫主义"的典型风格。

### 文学价值

《七律·长征》是现代汉语七律诗的代表作之一，语言精练（56 字概括两年历程），气势宏大，意象鲜明（五岭、乌蒙、金沙、大渡、岷山），是中学语文必背篇目，也是了解长征历史的诗歌入口。

**注意**：本词条对历史事件采取中性史实陈述，价值评价请家长结合家庭教育背景引导孩子独立思考。

## 🗣️ 家长讲解话术

- 孩子问**"为什么说乌蒙山是'泥丸'？"** → 可以说："乌蒙山其实非常高、非常险，但诗人故意把它说成小泥球，是用一种夸张的说法，来表达红军根本不把这座大山放在眼里——这叫'夸张手法'，越夸张越能表现出勇气。"

- 孩子问**"'金沙水拍云崖暖'为什么是暖的？"** → 可以说："金沙江水拍打山崖，江水本身并不暖，'暖'是心里的感受——红军成功渡过了金沙江，甩掉了追兵，心情特别高兴，所以感觉是暖的。诗人把心情融进了景色里。"

- **共读小活动**：把这首诗抄写一遍，然后找一张长征路线地图，把诗里提到的地名（五岭、乌蒙、金沙江、大渡河、岷山）都在地图上找到并标记出来。
