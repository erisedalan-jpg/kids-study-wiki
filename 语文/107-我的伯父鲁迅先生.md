# 我的伯父鲁迅先生

> **一句话**：作者周晔以侄女的视角，通过几个真实的生活片段，回忆了伯父鲁迅先生关爱孩子、同情劳苦大众、为他人着想的高尚品格，表达了深深的悼念与敬仰之情。
> **English**: Written from the perspective of his niece Zhou Ye, this memoir recalls several vivid moments that reveal Lu Xun's care for children, compassion for the common people, and selfless concern for others.

---

## 🧒 给 3-6 岁（共读版）

鲁迅是一位很有名的作家，他的侄女（就是他弟弟的女儿）周晔写了这篇文章，回忆伯父鲁迅是个什么样的人。

文章里说，鲁迅伯父很喜欢和孩子们说说笑笑，但也会认真给孩子们讲道理；他看到路边一个受伤的黄包车夫（就是蹬三轮、拉人的工人），冒着寒冷，蹲下来帮他包扎伤口，还给了他钱。

每次看到穷苦的人，伯父都会皱起眉头，好像自己也在痛苦一样。

这就是一个真正善良的大人——不光对亲人好，也对所有陌生的普通人好。

**共读小活动**：和孩子一起讨论"你身边有没有这样的大人？做了什么让你觉得他们善良？"

## 📚 给 6-12 岁（自读版）

### 课文节选（统编版六年级上册 第八单元）

> 有一次，伯父送我两本书，一本是《表》，一本是《小约翰》。……
>
> 一天黄昏，爸爸妈妈都出去了，我自己坐在屋子里，门外的寒风不住地响着。忽然，我看见爸爸抱着一个人走进来，后面跟着妈妈，妈妈手里提着医药箱。……他们把那个人放在地上，我才知道那是个拉黄包车的，脚上没有穿鞋，光着脚，血肉模糊，地上留下了一串血迹。
>
> 鲁迅先生俯下身子，把脚上的血迹洗干净，轻轻地用纸包着，包扎了，还不知道从哪儿拿出一双鞋，帮他穿上。……

> "你不知道，"伯父摸了摸自己的鼻子，笑着说，"我小时候，鼻子跟你的一样，也是又高又直的。"
> "那怎么——"
> "哈哈！还是我碰了几次壁，把鼻子碰扁了！"

### 词语学习

| 词语 | 解释 |
|------|------|
| 黄包车夫 | 旧时拉人力车（黄包车）的工人 |
| 血肉模糊 | 形容伤口严重，血与肉混在一起 |
| 俯下身子 | 身体弯下来 |
| 碰壁 | 比喻做事遇到阻碍，失败受挫；文中鲁迅以幽默方式暗示在旧社会处处碰到困难 |
| 悼念 | 怀念死去的人，表示哀思 |
| 恍然大悟 | 突然明白、理解了某件事 |

### 文章结构（五个片段）

| 片段 | 内容 | 刻画的品质 |
|------|------|----------|
| 伯父去世，万人送别 | 群众的悲痛，引出全文 | 受人爱戴 |
| 谈《水浒传》 | 批评"我"囫囵吞枣地读书 | 关心孩子学习 |
| 谈"碰壁" | 用幽默暗示旧社会对进步人士的压制 | 乐观幽默、有骨气 |
| 帮助车夫 | 冒寒救助陌生工人 | 同情劳苦大众 |
| 关心女佣 | 询问并关心阿三的身体 | 体贴他人 |

### 课文主旨

本文用"以小见大"的手法，通过五件生活小事，勾勒出鲁迅先生**爱憎分明、关怀劳苦、乐观幽默**的人格形象。文章以"我呆呆地望着来来往往吊唁的人，心里暗暗地想：伯父就是这样的一个人，他为自己想得少，为别人想得多"作结，是全文的点睛之笔。

### 作者：周晔（鲁迅侄女）

周晔，鲁迅三弟周建人之女，本文写于鲁迅逝世后多年，是她对童年记忆的真实回忆。文章发表后被收入小学语文教材，成为认识鲁迅其人最亲切、最生动的入口文章之一。（具体生卒年请家长核对，待确认。）

### "碰壁"的双关解读

文中"碰壁"一词表面是鲁迅自嘲鼻子被碰扁，实则是用双关暗示：旧中国社会对进步文人的打压与审查。这种用幽默掩盖真相的说话方式，既是鲁迅的性格，也是特定历史环境中保护自己的方式。

此处涉及历史背景，家长可根据孩子年龄和理解程度酌情解读，不宜过度引申，以免超出孩子认知负荷。

### 鲁迅专题三篇整合

学完三篇鲁迅专题文章，可引导孩子用"关键词"来总结对鲁迅的认识：

- 从《少年闰土》看：**真实、细腻的观察者**
- 从《好的故事》看：**在黑暗中守望美好的理想主义者**
- 从《我的伯父鲁迅先生》看：**对普通人充满悲悯的行动者**

## 🗣️ 家长讲解话术

- 孩子问**"'他为自己想得少，为别人想得多'是什么意思？"** → 可以说："就是说，鲁迅自己生病、自己累，他不放在心上；但是别人有困难、别人痛苦，他会很在意，很想去帮忙。这就是'无私'——把别人放在自己前面。"

- 孩子问**"碰壁真的是鼻子碰扁了吗？"** → 可以说："哈哈，当然不是！鲁迅在开玩笑，'碰壁'是一个比喻，意思是做事情老是遇到阻碍。他那个时代，写文章常常被检查、被禁止，所以用这个幽默的说法，来解释为什么他老是遇到麻烦。"

- **共读小活动**：把文章中的五个片段列出来，让孩子给每个片段取一个小标题（比如"帮助车夫""谈碰壁"），然后讨论：哪个片段最打动你？为什么？
