# 学弈

> **一句话**：《孟子》中的这则短文，以同学一位老师学弈（围棋）的两个学生为例，说明学习必须专心致志，不能三心二意。
> **English**: This short passage from Mencius uses two students learning Go from the same master to illustrate that learning requires full concentration, not a divided mind.

---

## 🧒 给 3-6 岁（共读版）

古代有一个非常厉害的下棋高手，叫**弈秋**，他收了两个学生一起学习下棋。

第一个学生**认认真真**听老师讲，用心学习每一步。

第二个学生呢，心里却在想："哎，天上那只大雁快飞过来了，要是用弓箭把它射下来多好啊！"

一个用心，一个不用心，最后会怎么样呢？

当然是第一个学生学得好，第二个学生学得很差。

但这是因为第二个学生不够聪明吗？**不是！** 只是因为他不专心。

**共读小活动：** 问孩子：你做什么事情的时候，最容易分心？怎么做才能专心下来？

## 📚 给 6-12 岁（自读版）

### 文言原文（《孟子·告子上》）

> 弈秋，通国之善弈者也。使弈秋诲二人弈，其一人专心致志，惟弈秋之为听；一人虽听之，一心以为有鸿鹄将至，思援弓缴而射之。虽与之俱学，弗若之矣。为是其智弗若与？曰：非然也。

**全文注音**（重点难字）：

| 字 | 拼音 | 注释 |
|------|------|------|
| 弈 | yì | 围棋；也指下棋 |
| 弈秋 | Yì Qiū | 人名，古代著名棋手 |
| 诲 | huì | 教导，教诲 |
| 惟 | wéi | 只，仅仅 |
| 鸿鹄 | hóng hú | 天鹅（泛指大型候鸟） |
| 援 | yuán | 拉，引（援弓：拉弓） |
| 缴 | zhuó | 系在箭上的绳子（此处泛指带绳的箭） |
| 弗 | fú | 不 |
| 俱 | jù | 一起，共同 |

### 逐句译文

| 原文 | 白话译文 |
|------|------|
| 弈秋，通国之善弈者也。 | 弈秋，是全国最擅长下棋的人。 |
| 使弈秋诲二人弈， | 让弈秋教两个人下棋， |
| 其一人专心致志，惟弈秋之为听； | 其中一人专心致志，只听弈秋讲棋； |
| 一人虽听之，一心以为有鸿鹄将至， | 另一人虽然也在听，心里却想着有天鹅要飞来， |
| 思援弓缴而射之。 | 想着拉弓搭箭把它射下来。 |
| 虽与之俱学，弗若之矣。 | 虽然两人一起学，（第二人）却比不上（第一人）。 |
| 为是其智弗若与？ | 这是因为他的智力不如（第一人）吗？ |
| 曰：非然也。 | 回答说：不是这样的。 |

### 寓意

文章的核心逻辑：

- 两人同一老师、同时学习 → 排除"老师"和"学习条件"的差异
- 两人智力相当（文末明说"非然也"）→ 排除"天赋"的差异
- 唯一不同：**一人专心，一人不专心**
- 结论：学习成效的差异，来自**专心与否**

这是文章的论证逻辑，用控制变量的思路来说明专心的重要性。

## 🎓 给 12+（进阶版）

### 出处：《孟子·告子上》

《孟子》是儒家经典之一，记录了孟子（约前 372-前 289 年）的言行。《告子上》是其中讨论人性问题的一篇，"学弈"这个小故事用来说明"学习需要专心"这一论点，属于文章中的举例论证。

孟子认为，人的本性是善的（性善论），但若不用心修养，善性就会消失。这则"学弈"故事，在原书的语境中，是论证"非天赋之过，乃用心与否"的引子。

### 文言文学习要点：古汉语句式

这篇文章中出现了几个重要的文言句式：

1. **判断句**："弈秋，通国之善弈者也。"
   - 结构：……者也（用"者也"表判断）
   - 相当于现代汉语"弈秋是……"

2. **宾语前置**："惟弈秋之为听"
   - 正常语序：惟听弈秋（只听弈秋的）
   - 用"之为"把宾语提前，是文言文的强调句式

3. **反问句**："为是其智弗若与？"
   - 用"为……与？"表示反问
   - 意思：难道是因为他的智力不如对方吗？

### 《孟子》的叙事特点

《孟子》善用"寓言式举例"来支撑论点，《学弈》《揠苗助长》《鱼我所欲也》等都是典型。这种"先讲故事，再讲道理"的结构，对后世散文影响深远，也是先秦诸子百家普遍使用的说理方式。

## 🗣️ 家长讲解话术

- 孩子可能问 **"弈秋是真实的人吗？"** → 可以说："弈秋在《孟子》中被称为'通国之善弈者'，说明他是当时有名的棋手。但由于年代久远，我们无法查到他更多的历史记录，只知道孟子用他举例来说明道理。"

- 孩子可能问 **"文言文和现代文有什么不同？"** → 可以说："文言文是古代书写的语言，和我们现在说话不完全一样。比如'弗'是'不'，'弈'是'棋'，'惟'是'只'。文言文字数少，信息量大，一个字在现代可能要用好几个字才能说清楚。学文言文就像学'古代语言密码'，需要慢慢积累。"

- **共读小活动**：把文言原文和白话译文对照，让孩子指出哪些字的意思和现代一样，哪些不一样。然后试着把课文用自己的话讲成一个故事，考查理解程度。
