# 狼·蒲松龄（课文版）

> **一句话**：蒲松龄以屠夫与两只狡猾的狼斗智斗勇的故事，揭示狼（恶势力）虽阴险，终难逃正义制裁的道理。
> **English**: Pu Songling uses the story of a butcher outwitting two cunning wolves to reveal that evil forces, however cunning, cannot escape justice.

---

## 📚 给 6-12 岁（小学高年级回溯版）

这是一个古代文言故事，但很有趣，像一个小冒险故事。

一个屠夫晚上回家，路上遇到两只狼！狼跟着他走，他很害怕。

狼是怎么"骗"他的？
- 一只狼坐在前面的柴草堆旁假装睡觉（迷惑屠夫）
- 另一只狼悄悄在柴草堆后面打洞，准备从后面偷袭

但屠夫不傻！他用斧头砍死了假装睡觉的狼，又转身砍死了钻洞的狼。

故事的结尾说：**狼也太狡猾了，但面对勇敢的屠夫，阴谋终于失败**。

这是一个"恶有恶报"的故事，也告诉我们：面对坏人，不能只是逃跑，要勇敢机智地反击。

## 🎓 给 12+（进阶版）

### 原文（全文）

```
一屠晚归，担中肉尽，止有剩骨。途中两狼，缀行甚远。

屠惧，投以骨。一狼得骨止，一狼仍从。复投之，后狼止而前狼又至。骨已尽矣，而两狼之并驱如故。

屠大窘，恐前后受其敌。顾野有麦场，场主积薪其中，苫蔽成丘。屠乃奔倚其下，弛担持刀。狼不敢前，眈眈相向。

少时，一狼径去，其一犬坐于前。久之，目似瞑，意暇甚。屠暴起，以刀劈狼首，又数刀毙之。方欲行，转视积薪后，一狼洞其中，意将隧入以攻其后也。身已半入，止露尻尾。屠自后断其股，亦毙之。乃悟前狼假寐，盖以诱敌。

狼亦黠矣，而顷刻两毙，禽兽之变诈几何哉？止增笑耳。
```

### 逐句译文

| 原文 | 译文 |
|------|------|
| 一屠晚归，担中肉尽，止有剩骨 | 一个屠夫傍晚回家，担子里肉已卖完，只剩些骨头 |
| 途中两狼，缀行甚远 | 途中遇到两只狼，跟着他走了很远 |
| 屠惧，投以骨 | 屠夫很害怕，向狼投出骨头 |
| 一狼得骨止，一狼仍从 | 一只狼得到骨头停下来，另一只继续跟着 |
| 骨已尽矣，而两狼之并驱如故 | 骨头扔完了，两只狼仍然一起追赶，没有变化 |
| 屠大窘，恐前后受其敌 | 屠夫非常困窘，担心被前后夹击 |
| 顾野有麦场 | 回头看到旷野中有个麦场 |
| 屠乃奔倚其下，弛担持刀 | 屠夫跑去靠在麦垛下，放下担子，握住刀 |
| 一狼径去，其一犬坐于前 | 一只狼径直走开，另一只像狗一样坐在前面 |
| 目似瞑，意暇甚 | 眼睛似乎闭上，神情很悠闲（假装的） |
| 屠暴起，以刀劈狼首 | 屠夫突然跳起，用刀劈向狼头 |
| 一狼洞其中，意将隧入以攻其后 | 另一只狼在柴草堆里打洞，想从后面偷袭 |
| 禽兽之变诈几何哉？止增笑耳 | 禽兽的花招能有多少？只不过让人笑笑罢了 |

### 字词理解

- **缀行**（zhuì xíng）：跟随，缀=跟随、连接
- **窘**（jiǒng）：困窘，处境困难
- **积薪**：堆积的柴草
- **苫蔽成丘**（shàn bì）：用茅草覆盖，像个小山丘
- **眈眈相向**（dān dān）：目光凶狠地互相注视
- **犬坐**：像狗一样蹲坐（比较）
- **瞑**（míng）：闭眼
- **意暇甚**：神情很悠闲（意=神情，暇=闲暇）
- **隧入**：从隧道（洞穴）进入
- **尻**（kāo）：臀部，屁股
- **假寐**（jiǎ mèi）：假装睡觉
- **黠**（xiá）：狡猾
- **变诈**：变化狡诈的手段

### 主旨与写作手法

**主旨**：通过人与狼斗智斗勇的故事，讽刺了恶势力（狼）虽善于使用诡计，但终究难以战胜正义（屠夫的勇敢机智）。末段议论点明寓意：面对"禽兽"，以勇智对付，而非一味妥协。

**写作手法**：
1. **曲折的情节**："投骨→骨尽仍追→靠薪持刀→狼假寐→屠杀狼→发现第二只"，起伏跌宕
2. **细节传神**：犬坐、目似瞑、意暇甚——短短几字勾画出狼假装悠闲的神态
3. **议论结尾**：末段从故事抽身，直接评论，点明寓意（文言小说常见结构）
4. **动词精确**：劈、断、毙——简洁有力

### 作家生平

**蒲松龄**（1640—1715），字留仙，号柳泉，山东淄博人，清代著名文学家。代表作《聊斋志异》是中国古典短篇小说的巅峰，共 491 篇，多写鬼狐仙怪，借以讽刺封建社会。蒲松龄屡试不第，穷困潦倒，用几十年写成此书。
