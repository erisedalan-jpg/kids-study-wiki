# 观沧海·曹操

> **一句话**：曹操登上碣石山，面对波涛壮阔的大海，写下这首气吞山河的四言诗。
> **English**: Cao Cao climbed Jie Stone Mountain and gazed at the vast sea, writing this grand four-character poem.

---

## 📚 给 6-12 岁（小学高年级回溯版）

你见过大海吗？曹操在 207 年北征乌桓归来，路过河北，爬上一座叫"碣石"的山，看见了茫茫大海。

他看到：
- 海水荡漾，岛上草木茂盛
- 秋风吹来，波浪滚滚
- 太阳、月亮好像从海里升起
- 银河的星光好像从海中涌出

他觉得大海太壮观了，忍不住唱出了这首诗——这就是《观沧海》。

**小任务**：找一首你喜欢的关于大海的诗，比比哪首更让你感到大海的力量。

## 🎓 给 12+（进阶版）

### 原文与拼音

```
观沧海
曹操（东汉）

东临碣石，以观沧海。
dōng lín jié shí，yǐ guān cāng hǎi。

水何澹澹，山岛竦峙。
shuǐ hé dàn dàn，shān dǎo sǒng zhì。

树木丛生，百草丰茂。
shù mù cóng shēng，bǎi cǎo fēng mào。

秋风萧瑟，洪波涌起。
qiū fēng xiāo sè，hóng bō yǒng qǐ。

日月之行，若出其中；
rì yuè zhī xíng，ruò chū qí zhōng；

星汉灿烂，若出其里。
xīng hàn càn làn，ruò chū qí lǐ。

幸甚至哉，歌以咏志。
xìng shèn zhì zāi，gē yǐ yǒng zhì。
```

### 逐句译文

| 原文 | 译文 |
|------|------|
| 东临碣石，以观沧海 | 向东登上碣石山，来观望这茫茫大海 |
| 水何澹澹，山岛竦峙 | 海水多么荡漾动荡，山岛高高耸立 |
| 树木丛生，百草丰茂 | 岛上树木丛生，百草丰茂 |
| 秋风萧瑟，洪波涌起 | 秋风吹来，萧瑟作响，巨浪汹涌而起 |
| 日月之行，若出其中 | 太阳和月亮的运行，好像从大海里升出 |
| 星汉灿烂，若出其里 | 银河星光灿烂，好像从大海中涌现 |
| 幸甚至哉，歌以咏志 | 庆幸至极，作歌来抒发自己的心志 |

### 字词理解

- **临**：登临，从高处俯视
- **碣石**：山名，在今河北昌黎，东汉时临海
- **沧海**：大海（"沧"有深蓝色之意）
- **何**：多么（感叹语气）
- **澹澹**（dàn dàn）：水波荡漾的样子
- **竦峙**（sǒng zhì）：高耸挺立
- **萧瑟**：风吹树木发出的声音
- **洪波**：巨大的波浪
- **星汉**：银河
- **幸甚至哉**：乐府诗套语，表达喜悦

### 主旨与写作手法

**主旨**：诗人借大海的壮阔景象，抒发自己一统天下、吞吐宇宙的宏大志向与豪迈气概。

**写作手法**：
1. **虚实结合**：前六句写实（眼中所见海景），"日月之行，若出其中；星汉灿烂，若出其里"转为虚写（想象），以夸张手法表现海的博大
2. **以景寄志**：诗人并未直接说"我志向远大"，而是借大海吞吐日月星辰的气象，含蓄表达胸怀
3. **动静结合**：山岛"竦峙"（静），洪波"涌起"（动），刚柔相济

### 作家生平

**曹操**（155—220），字孟德，小名阿瞒，沛国谯县（今安徽亳州）人。东汉末年政治家、军事家、文学家。官至丞相，封魏王。其子曹丕称帝后追谥为魏武帝。

文学上，曹操与其子曹丕、曹植并称"三曹"，是建安文学的代表人物。诗歌风格慷慨悲凉、气魄雄伟，代表作有《短歌行》《龟虽寿》《观沧海》。

**创作背景**：公元 207 年（建安十二年），曹操北征乌桓大胜，班师回朝途中经过碣石山，临海而作此诗，时年 53 岁，意气风发。

### 鉴赏与对比

《观沧海》是中国现存最早的完整山水诗，标志着文人山水诗的滥觞。与李白的《望庐山瀑布》相比，曹操更注重宏大与政治志向的寄托，李白更侧重个人的飘逸与浪漫。
