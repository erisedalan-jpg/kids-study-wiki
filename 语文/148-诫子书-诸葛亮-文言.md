# 诫子书·诸葛亮

> **一句话**：诸葛亮写给儿子的一封家书，用 86 个字告诉他：人生要成功，必须做到"静以修身，俭以养德，学以广才"。
> **English**: A letter from Zhuge Liang to his son, using 86 characters to convey that success requires quietness for self-cultivation, frugality for virtue, and learning for wisdom.

---

## 📚 给 6-12 岁（小学高年级回溯版）

诸葛亮你一定知道——他是《三国演义》里那个神机妙算的军师。他那么聪明，那么厉害，他给自己的儿子写了什么家书？

这封信只有 86 个字，但非常有道理：

- **"非淡泊无以明志，非宁静无以致远"**：不淡泊（不被名利迷惑），就看不清自己的志向；不安静（浮躁），就做不到远大目标
- **"夫学须静也，才须学也"**：学习需要安静的心，有才能必须通过学习
- **"淫慢则不能励精，险躁则不能治性"**：放纵懒散→精力消散；急躁冒险→性情无法磨砺

诸葛亮写这封信时已经 54 岁了，他深知人生不易，希望儿子珍惜时光，好好学习。

**你觉得：安静认真地学习，为什么比急躁更重要？**

## 🎓 给 12+（进阶版）

### 原文（全文）

```
诫子书
诸葛亮

夫君子之行，静以修身，俭以养德。非淡泊无以明志，非宁静无以致远。夫学须静也，才须学也，非学无以广才，非志无以成学。淫慢则不能励精，险躁则不能治性。年与时驰，意与日去，遂成枯落，多不接世，悲守穷庐，将复何及！
```

| 原文 | 译文 |
|------|------|
| 夫君子之行，静以修身，俭以养德 | 君子的品行，用宁静修养身心，用俭朴培养品德 |
| 非淡泊无以明志，非宁静无以致远 | 不淡泊就无法明确志向，不宁静就无法达到远大目标 |
| 夫学须静也，才须学也 | 学习必须有宁静的心，才能必须通过学习来获得 |
| 非学无以广才，非志无以成学 | 不学习无法增长才能，没有志向无法完成学业 |
| 淫慢则不能励精 | 放纵懒散就不能振奋精神 |
| 险躁则不能治性 | 轻率冒进就不能陶冶性情 |

| 悲守穷庐，将复何及 | 只能悲哀地守着贫寒的屋子，那时后悔又有什么用呢 |

### 字词理解

- **诫**：告诫，警示
- **夫**（fú）：文言发语词，表示发表议论
- **君子**：有道德有修养的人
- **淡泊**：心境平静，不追求名利
- **宁静**：内心安宁，不浮躁
- **致远**：达到远大目标（"志远"）
- **广才**：增广才能
- **淫慢**：放纵懒散（淫=过度，慢=懒惰）
- **励精**：振奋精神
- **险躁**：轻率冒险，急躁
- **治性**：陶冶性情，修炼品格
- **年与时驰**：年华随时间流逝（驰=奔跑）
- **枯落**：像枯叶一样凋零，比喻人的衰老

### 主旨与写作手法

**主旨**：告诫儿子要淡泊宁静以修身养德，勤学明志以成就才能，珍惜时光，莫要虚度。

**写作手法**：
1. **双重否定**（正话反说）：全文多处用"非……无以……"，强调不做某事的严重后果，语气比直接陈述更有力
2. **对偶**："静以修身，俭以养德"；"非淡泊无以明志，非宁静无以致远"——工整对仗，便于记诵
3. **层层递进**：修身→养德→立志→学习→珍惜时间，逻辑严密
4. **情感递进**：从平静的教诲到最后"悲守穷庐，将复何及"的感叹，情感深沉

### 作家生平

**诸葛亮**（181—234），字孔明，号卧龙，徐州琅琊（今山东）人，蜀汉丞相，中国历史上著名的政治家、军事家、文学家。辅佐刘备建立蜀汉政权，"出师未捷身先死，长使英雄泪满襟"（杜甫诗）。《诫子书》写于公元 234 年，他去世前不久，儿子诸葛瞻当时 8 岁。此文是他留给儿子最重要的精神遗产。
