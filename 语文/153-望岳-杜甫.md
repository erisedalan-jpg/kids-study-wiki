# 望岳

> **一句话**：杜甫年轻时望泰山，写出了"会当凌绝顶，一览众山小"的豪情壮志。
> **English**: The young Du Fu, gazing at Mount Tai, gave voice to a timeless ambition: to reach the summit and look down upon all lesser peaks.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：唐 · 杜甫
**体裁**：五言律诗

```
望岳
杜甫

岱宗夫如何？齐鲁青未了。
造化钟神秀，阴阳割昏晓。
荡胸生曾云，决眦入归鸟。
会当凌绝顶，一览众山小。
```

**拼音**：
- dài zōng fū rú hé？qí lǔ qīng wèi liǎo。
- zào huà zhōng shén xiù，yīn yáng gē hūn xiǎo。
- dàng xiōng shēng céng yún，jué zì rù guī niǎo。
- huì dāng líng jué dǐng，yī lǎn zhòng shān xiǎo。

**白话译文**：
泰山啊，你究竟是什么样的？它的苍翠横跨齐鲁两地，绵延无尽。
大自然将所有的神奇和秀丽都汇聚于此，山的南北两侧，一边是白昼，一边是黑夜。
层层云雾涌动，令人心胸激荡；极目远眺，看见归鸟飞入林中。
终有一天，我要登上那最高的顶峰，俯瞰群山，它们都将显得渺小！

**朗读节奏**：
岱宗 / 夫如何？齐鲁 / 青未了。
造化 / 钟神秀，阴阳 / 割昏晓。
荡胸 / 生曾云，决眦 / 入归鸟。
会当 / 凌绝顶，一览 / 众山小。

## 🧒 给 3-6 岁（共读版）

你见过很高很高的山吗？泰山是中国最有名的大山之一，高高的，云彩都在山腰上飘。

杜甫年轻的时候去看泰山，他还没有爬上去，只是站在远处看，就觉得泰山好厉害好神奇！他心里想："总有一天，我要爬到最高的地方，到那时候，其他的山都会看起来好小好小！"

**共读小活动**：
- 一起说说你见过的最高的地方，或者你想去的高山。
- 用手比一比"众山小"——把手高举，说"我在最高处，你们都好小！"

| 齐鲁 | qí lǔ | 古代齐国和鲁国，今山东一带 |
| 青未了 | qīng wèi liǎo | 苍翠的颜色无边无际，看不到尽头 |
| 造化 | zào huà | 大自然（造物主） |
| 钟 | zhōng | 汇聚、集中 |
| 阴阳 | yīn yáng | 山的南面（阳）和北面（阴） |

| 会当 | huì dāng | 终将、一定要 |
| 凌 | líng | 登上、超越 |
| 绝顶 | jué dǐng | 最高的山顶 |

### 内容与情感

全诗分四联，层层递进：

1. **首联**（远望）：泰山苍翠，横跨齐鲁，以"齐鲁青未了"写出泰山之广大。
2. **颔联**（近观）：大自然将精华都聚于此；山体之高，将日光分割，南北昏晓不同。
3. **颈联**（凝神）：云涌荡胸、归鸟入眼，写出诗人望山的专注与激动。
4. **尾联**（志向）：由望山生出登顶之志，"会当凌绝顶，一览众山小"成为千古名句。

全诗由景入情，由外到内，最后升华为宏大的人生志向。

### 写作手法

- **由远到近**：远望（齐鲁青未了）→ 近观（阴阳割昏晓）→ 凝神（决眦入归鸟）
- **以小衬大**：众山小，衬托泰山绝顶之高
- **借景抒志**：望山写景，实则抒发年轻时的凌云壮志

## 🎓 给 12+（进阶版）

### 作者与时代背景

**杜甫**（712—770），字子美，号少陵野老，唐代伟大的现实主义诗人，与李白并称"李杜"，后人尊为"诗圣"。

《望岳》作于开元二十四年（736年），杜甫年约二十五岁，正值游历壮年，豪情满怀。这是他现存最早的诗作之一，与他晚年忧国忧民的沉郁风格迥异，充满青春气息与进取精神。他一生三次写《望岳》（泰山、华山、衡山），此为最著名的一首。

### 文学史地位

"会当凌绝顶，一览众山小"是中国文学史上抒发人生志向最著名的诗句之一，后世常被引用于表达勇于攀登、志存高远的精神。王安石《登飞来峰》的"不畏浮云遮望眼，自缘身在最高层"与此有异曲同工之妙，可对比阅读。

### 鉴赏与比较

与《登幽州台歌》相比，两诗皆写登高，但陈子昂悲叹孤独、情绪沉郁；杜甫此诗则意气风发、充满希望。同样是杜甫，晚年的《登高》"无边落木萧萧下，不尽长江滚滚来"，与《望岳》的昂扬形成鲜明对比，可见诗人一生命运沉浮在诗中的折射。

"阴阳割昏晓"一句，"割"字极为精准——山高到将天光一分为二，一字尽显泰山之巍峨，是炼字的典范。

## 🗣️ 家长讲解话术

- 孩子问 **"'众山小'是什么感觉？"** → 可以说："就是站在最高处，其他的山都变得好小好小，像玩具一样。这不只是在说爬山，也是在说——只要你努力到达顶峰，就会发现很多之前觉得很大的困难，其实都没那么难。"
- **共读小活动**：和孩子一起找一张泰山的图片，读诗，边读边对照图片，感受"齐鲁青未了"的辽阔。
