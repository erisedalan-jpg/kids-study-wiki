# 登飞来峰

> **一句话**：王安石登上飞来峰，写出"不畏浮云遮望眼，自缘身在最高层"的哲理名句，表达变法改革的坚定信念。
> **English**: Wang Anshi's climb up Feilai Peak inspired a philosophical couplet about seeing clearly from the heights, reflecting his unwavering resolve for reform.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：宋 · 王安石
**体裁**：七言绝句

```
登飞来峰
王安石

飞来山上千寻塔，
闻说鸡鸣见日升。
不畏浮云遮望眼，
自缘身在最高层。
```

**拼音**：
- fēi lái shān shàng qiān xún tǎ，
- wén shuō jī míng jiàn rì shēng。
- bù wèi fú yún zhē wàng yǎn，
- zì yuán shēn zài zuì gāo céng。

**白话译文**：
飞来峰上有座高达千寻的塔，
听说在鸡鸣时分，站在塔顶就能看到旭日初升。
我不害怕浮云遮住了我的视线，
因为我就站在最高处！

**朗读节奏**：
飞来山上 / 千寻塔，闻说 / 鸡鸣 / 见日升。
不畏 / 浮云 / 遮望眼，自缘 / 身在 / 最高层。

## 🧒 给 3-6 岁（共读版）

你爬过很高的楼吗？从高楼上往下看，街道上的人和车都变得好小！

王安石爬上了一座非常高的山顶上的塔，从那里往远处看，连云彩都在他下面飘！他说：我不怕云彩挡住我的眼睛，因为我站在最高的地方，什么都能看清楚！

**共读小活动**：
- 找一个高处，看看下面的东西变小了没有。
- 问孩子："如果你站在很高很高的地方，能看到什么？"

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 飞来峰 | fēi lái fēng | 浙江绍兴的山峰 |
| 千寻 | qiān xún | 形容极高（古代一寻约8尺） |
| 塔 | tǎ | 宝塔，矗立于山顶 |
| 浮云 | fú yún | 飘动的云，也比喻小人、障碍 |
| 遮望眼 | zhē wàng yǎn | 遮住了向远处眺望的视线 |
| 自缘 | zì yuán | 正是因为 |
| 最高层 | zuì gāo céng | 最高处，也喻指高远的境界 |

### 内容与情感

- **前两句**：写登塔所见，"千寻"极写高，"鸡鸣见日升"写登高可见奇景。
- **后两句**：转入哲理，"浮云"比喻阻碍和小人，"最高层"既指实处，也喻高远的立场和格局。

全诗将写景与议论融为一体，景为议服务，议从景中来，是典型的哲理绝句。

### 写作手法

- **虚实结合**：前半写眼前实景，后半写抽象哲理
- **比喻**："浮云"比喻困难或小人，"最高层"比喻高远的眼界
- **对比**：浮云（遮蔽）vs. 最高层（清明），突出高处的优势

## 🎓 给 12+（进阶版）

### 作者与时代背景

**王安石**（1021—1086），字介甫，号半山，抚州临川（今江西抚州）人，北宋政治家、文学家、改革家，"唐宋八大家"之一。

《登飞来峰》作于宋仁宗皇祐二年（1050年），王安石时年三十岁，初入仕途。诗中"不畏浮云遮望眼，自缘身在最高层"表达了他对变法改革的坚定信念：只要站得够高、看得够远，就不会被眼前的障碍所迷惑。二十余年后，他推行"王安石变法"，诗中的志向得到了历史印证。

### 文学史地位

这首诗是宋代哲理诗的代表作之一，与苏轼《题西林壁》"横看成岭侧成峰，远近高低各不同。不识庐山真面目，只缘身在此山中"形成有趣的对照——苏轼讲当局者迷，王安石讲高处者清，可对比阅读。

### 鉴赏与比较

与杜甫《望岳》"会当凌绝顶，一览众山小"相比，两诗都写登高志向，但杜甫更侧重个人豪情，王安石则更多哲理思辨。"自缘身在最高层"在语气上比"会当凌绝顶"更自信，已然身处高处，而非期许将来。

## 🗣️ 家长讲解话术

- 孩子问 **"'浮云'到底是什么？"** → 可以说："'浮云'表面是云彩，其实王安石是在说那些阻碍他做事的困难和反对他的人。他说'不畏'，就是'我不怕'，因为他相信只要自己站得高、看得远，就不会被这些东西挡住。"
- **共读小活动**：和孩子讨论"站得高、看得远"在生活中是什么意思——比如提前学习、提前准备，就像站在高处能提前看到日出。
