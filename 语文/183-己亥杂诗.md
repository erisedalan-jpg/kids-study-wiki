# 己亥杂诗

> **一句话**：龚自珍呼唤变革的雷霆，为沉闷的晚清呐喊——不拘一格，需要各式各样的人才来救国。
> **English**: Gong Zizhen calls for a thunderstorm of change in the stagnant late Qing era, urging heaven to break all conventions and send forth talents of every kind.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：清 · 龚自珍
**体裁**：七言绝句（《己亥杂诗》第一百二十五首）

```
己亥杂诗（其一百二十五）
龚自珍

九州生气恃风雷，
万马齐喑究可哀。
我劝天公重抖擞，
不拘一格降人才。
```

**拼音**：
- jiǔ zhōu shēng qì shì fēng léi，
- wàn mǎ qí yīn jiū kě āi，
- wǒ quàn tiān gōng chóng dǒu sǒu，
- bù jū yī gé jiàng rén cái。

**白话译文**：
中国的生机勃发，靠的是风雷般的变革力量，
千万匹马都沉默不鸣，这终究令人悲哀。
我劝告天帝重新振作精神，
不要拘泥于一种格式，多多降下各种各样的人才。

**朗读节奏**：
九州 / 生气 / 恃风雷，/ 万马 / 齐喑 / 究可哀。
我劝 / 天公 / 重抖擞，/ 不拘 / 一格 / 降人才。

## 🧒 给 3-6 岁（共读版）

你知道马吗？马跑起来很快，叫声也很响亮！

诗里说"万马齐喑"——想象一下，如果有一万匹马，突然全部都不叫了，静悄悄的，是不是很奇怪？很可惜？

诗人龚自珍觉得，当时中国就像这样，大家都不敢发表意见，不敢做新的事情，就像一万匹马都沉默了一样。他希望天公（老天爷）帮忙，让各种各样的人才出现，让中国重新变得有活力！

**共读小活动**：
- **说一说**：你觉得什么样的人是"人才"？
- **想一想**：为什么一万匹马都沉默了很令人难过？

### 历史背景

1839年（己亥年），龚自珍辞官南下，途中写下315首杂诗，合称《己亥杂诗》。这一年距离鸦片战争爆发仅一年——中国正处于巨大历史变局的前夜。

当时的清朝政治极度保守：文字狱盛行，士人动辄得咎，知识分子噤若寒蝉，朝廷官员循规蹈矩，没有改革的勇气与声音。龚自珍对这种沉闷局面深感忧虑，发出了这声振聋发聩的呐喊。

### 内容与情感

1. **第一句**：中国的希望在于变革的风雷——变革是生机的来源
2. **第二句**：可是现在万马齐喑，沉寂得令人悲哀——现实令人痛心
3. **第三四句**：呼唤天公振作，打破格局，降下各种人才——发出改变的呼声

全诗情感由"忧"到"呼"，是一首政治抒怀诗，也是一声历史性的呐喊。

### "不拘一格"的成语

"不拘一格"由这首诗衍生出的成语，今天广泛使用，意思是不受固定框框的限制，用灵活多样的方式做事或选拔人才。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**龚自珍**（1792—1841），字璱人，号定盦，浙江仁和（今杭州）人，晚清思想家、诗人、改革先驱，与林则徐、魏源并称晚清三大思想家。

龚自珍对清朝积弊有深刻洞察，主张改革政治、开放思想，但在保守的朝廷环境中处处碰壁，仕途坎坷。1839年辞官，两年后猝然离世（1841年），未能亲见鸦片战争的全面爆发。

《己亥杂诗》315首是他辞官南下旅途中所作，内容涵盖政治批评、思想抒发、怀人念旧，是清代诗歌中思想性最强的诗集之一。

### 思想史意义

这首诗在中国近代思想史上有重要地位：
- **冲击科举制度**："不拘一格降人才"是对单一科举取士制度的批判，呼唤多元人才观
- **预感近代变局**：龚自珍写此诗一年后，鸦片战争爆发；他的忧虑是历史的预言
- **影响近代改革者**：梁启超、谭嗣同等维新派人物深受龚自珍影响，将其视为"思想启蒙者"

梁启超曾评价："晚清思想之解放，自珍确与有功焉。"

### 文学与政治的张力

这首诗用"天公"作为政治批评的对象——表面上是向天帝祈愿，实际上是对最高统治者的委婉进谏。这种策略性的表达方式，是中国士人在政治高压下发声的传统方式（比较屈原"天问"、杜甫"三吏三别"的进谏传统）。

"万马齐喑"的意象来自苏轼《三马图赞》，龚自珍赋予它新的政治含义，今天"万马齐喑"已成为描述压制言论环境的固定成语。

### 课标定位

这首诗在五年级上册出现，是小学阶段政治抒情诗中思想深度最高的一首。教学重点在于：理解"万马齐喑"的历史语境，感受诗人忧国忧民、呼唤改革的情怀。

## 🗣️ 家长讲解话术

- 孩子可能问 **"万马齐喑是什么意思？"** → 可以说："想象一下马场里有一万匹马，突然全部都不叫了，静悄悄的——这表示大家都不敢说话，沉默得很压抑。诗人用这个比喻形容当时的社会，大家都不敢发表不同意见。"

- 孩子可能问 **"天公是谁？"** → 可以说："天公是老天爷，也暗指最高的统治者。诗人不能直接批评皇帝，就说请'天公'帮忙，其实是在婉转地说：请统治者改变一下，允许各种各样的人才来做事！"

- 孩子可能问 **"不拘一格降人才是什么意思？"** → 可以说："'不拘一格'就是不按照一个固定的模式。诗人希望老天爷不要只选一种类型的人才（比如只选考试考得好的），要各种各样能力的人都有机会——这个想法今天还很有道理！"

- **共读小活动（成语溯源）**：和孩子一起查一查"万马齐喑"和"不拘一格"这两个成语都来自这首诗，让孩子体会古诗对现代汉语的影响。
