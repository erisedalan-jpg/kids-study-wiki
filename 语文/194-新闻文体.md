# 新闻文体

> **一句话**：新闻文体是对真实事件进行及时报道的写作形式，包括消息、通讯、特写、评论等多种体裁，核心要求是真实、及时、客观，是八上语文第一单元的核心知识点。
> **English**: Journalism encompasses several forms for reporting real events in a timely manner — dispatches, features, profiles, and commentaries — all built on the principles of truth, timeliness, and objectivity.

---

## 📖 为什么要学新闻文体？

新闻是现代人每天接触最多的文体之一。学习新闻文体的意义：
1. **读写能力**：能写简单新闻，能分析新闻的真实性
2. **媒体素养**：识别新闻的立场与局限，不被信息操纵
3. **语文考试**：八年级常考新闻文体分析（导语、五W、倒金字塔等）

## 📚 给 6-12 岁（自读版）

### 新闻的四大原则

| 原则 | 说明 |
|------|------|
| **真实** | 报道的内容必须是真实发生的，不能虚构 |
| **及时** | 新闻要尽快发出，时效性是新闻的生命 |
| **客观** | 尽量中立，不以个人情感代替事实（但完全中立极难实现）|
| **简洁** | 语言精练，核心信息放在最前面 |

### 新闻的主要体裁

| 体裁 | 定义 | 特点 | 教材例文 |
|------|------|------|---------|
| **消息** | 最简短的新闻，报道单一事件 | 倒金字塔，五W | 《消息二则》 |
| **通讯** | 较详细的报道，有细节和背景 | 篇幅较长，可叙事 | — |
| **特写** | 聚焦某一瞬间或人物的报道 | 有文学性，现场感强 | 《"飞天"凌空》《一着惊海天》 |
| **评论** | 对新闻事件发表意见 | 有观点，有论证 | — |

### 消息的结构

```
标题          ← 最重要，最简洁
导语（第1段）  ← 包含五W核心信息
主体          ← 详细内容，按重要性排列
背景/结语     ← 补充信息，可省略
```

**五W**：Who（谁）、What（什么事）、When（何时）、Where（何地）、Why（为何）

### 倒金字塔结构示意

```
【最宽：最重要信息】← 标题 + 导语
        ↓
    【中部：详情】← 主体
        ↓
【最窄：次要信息】← 背景
```

优点：读者只需读导语，即可了解核心；编辑裁剪时，从末尾删起不影响核心信息。

## 🎓 给 12+（进阶版）

### 新闻客观性的困境

"客观报道"是新闻理想，但实践中有困难：

| 问题 | 说明 |
|------|------|
| 选择偏见 | 记者选择报道什么事件，这本身就是立场 |
| 措辞偏见 | 同一事件，"激进分子"vs"抗议者"，选词不同传递不同态度 |
| 省略偏见 | 哪些信息不报道，往往比报道什么更有意义 |
| 框架偏见 | 如何组织信息（先说什么，后说什么）影响读者的解读 |

批判性阅读新闻，就是识别这些偏见的能力。

### 硬新闻 vs 软新闻

| 类型 | 定义 | 特点 |
|------|------|------|
| 硬新闻 | 时效性强、重要性高的新闻（政治、经济、灾难）| 消息为主，结构严格 |
| 软新闻 | 趣味性强、时效性较弱的新闻（娱乐、生活、人情味）| 特写为主，文学性强 |

《"飞天"凌空》属于软新闻（体育特写）；《消息二则》属于硬新闻（战时军事消息）。

### 数字时代的新闻变革

传统新闻（纸媒）在数字时代面临挑战：
- **速度**：社交媒体比传统媒体更快，但核实能力弱
- **碎片化**：标题党现象，读者只读标题不读内容
- **算法推荐**：信息茧房，人只看与自己观点相符的新闻
- **假新闻（misinformation）**：辨别真伪成为必要能力

"新闻素养"（media literacy）在今天比以往任何时候都更重要。

## 🗣️ 家长讲解话术

- 孩子问 **"什么是导语？"** → 可以说："导语是新闻第一段，也是最重要的部分。一篇好的导语，读完就能知道：谁做了什么，在哪里，什么时候，为什么。好比看电影预告片，一分钟内你就知道这部电影大概是讲什么。"
- 孩子问 **"为什么叫'倒金字塔'？"** → 可以说："普通金字塔从窄到宽，新闻相反——把最重要的内容放在最前面（最宽），越往后越不重要（越窄）。这样读者很忙的时候只看第一段，也能了解最关键的事情。"
- **共读小活动**：找今天的一则新闻，找出它的导语，识别其中包含了五W中的哪几个要素。
