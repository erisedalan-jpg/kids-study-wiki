# 秋夜将晓出篱门迎凉有感

> **一句话**：陆游写北方沦陷区的百姓年年望眼欲穿，等待王师北上，用"泪尽"二字写尽了绝望中的苦盼。
> **English**: Lu You writes of the people in the lost northern territories whose tears have run dry waiting for the imperial army — year after year, hope deferred becomes heartbreak.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：宋 · 陆游
**体裁**：七言绝句

```
秋夜将晓出篱门迎凉有感
陆游

三万里河东入海，
五千仞岳上摩天。
遗民泪尽胡尘里，
南望王师又一年。
```

**拼音**：
- sān wàn lǐ hé dōng rù hǎi，
- wǔ qiān rèn yuè shàng mó tiān，
- yí mín lèi jìn hú chén lǐ，
- nán wàng wáng shī yòu yī nián。

**白话译文**：
三万里长的黄河向东流入大海，
五千仞高的华山高耸入云天。
沦陷区的百姓已在金人的尘埃中哭干了眼泪，
南望，盼望王师北来，又过了一年。

**朗读节奏**：
三万里 / 河东 / 入海，/ 五千仞 / 岳上 / 摩天。
遗民 / 泪尽 / 胡尘里，/ 南望 / 王师 / 又一年。

## 🧒 给 3-6 岁（共读版）

你知道黄河吗？黄河是中国最著名的大河之一，非常非常长，河水向东流啊流，最后流进大海。还有华山，特别特别高，高得云都飘在山腰，山顶都钻进了天里。

诗人陆游看着这么大的山，这么长的河，心里很难过——这么好的土地，北边有很多在哭泣的老百姓，他们已经哭干了眼泪，一直在朝南方望啊望，盼望着军队来救他们……又过了一年，还是没有来。

这首诗让我们感受到陆游对那些受苦百姓的心疼。

**共读小活动**：
- **说一说**：如果你是诗里盼望被救的百姓，心里会是什么感觉？
- **想一想**："泪尽"是什么意思？（眼泪都哭干了）

### 历史背景

北宋灭亡（1127年）后，黄河流域及华北大片土地沦入金人统治。留在北方的汉族百姓被称为"遗民"，他们以"华夏遗民"自居，心向南宋朝廷，渴望被收复。

陆游写这首诗时已是晚年（约1192年，南宋偏安65年后），北方沦陷已逾半个世纪，多少代遗民在等待中出生、老去、死去，等到的只是年年南望、年年落空。

### 内容与情感

这首诗前后两部分形成强烈对比：
1. **前两句**：宏观山河的壮阔——三万里黄河、五千仞华山。以大写壮丽，暗含对中原大地的深情。
2. **后两句**：聚焦百姓的血泪——泪尽、胡尘、又一年。以小写悲情。

**"又一年"**是全诗最沉重的三个字：不是第一年，不是第二年——是年复一年，每年都在盼，每年都在失望，等到泪都哭干了，还在等……

### 写作手法

- **反衬（以大衬小）**：山河壮阔vs百姓血泪，越写山河美，越显百姓之悲
- **夸张**：三万里、五千仞，极言山河之大，衬托遗民盼望之切
- **叠加悲情**："泪尽""又"字，时间的沉重感扑面而来

## 🎓 给 12+（进阶版）

### 作者晚年的心境

陆游（1125—1210）写这首诗约在1192年，当时已67岁，被迫退居山阴（绍兴）老家。他一生渴望北伐，年轻时曾多次从军，到晚年只能在梦中出征（"夜阑卧听风吹雨，铁马冰河入梦来"）。

这首诗是他的另一个视角：他不写自己的北伐梦，而是写北方沦陷区百姓的等待——把自己的爱国情感，通过北方百姓的泪水表达出来。这比第一人称抒情更有张力，因为它写的是历史的伤痛，而非个人的情绪。

### 诗与历史的对应

"遗民泪尽"的历史真实性：
- 北宋灭亡初期（12世纪30-40年代），北方确有大量汉族百姓"南望王师"
- 随着几十年偏安，北方百姓逐渐与南宋朝廷脱节，接受了金朝统治
- 陆游写此诗时（1192年），距北宋灭亡已65年——"又一年"所蕴含的"年年"等待，已跨越了数代人的时间

"泪尽"二字在文学上极有力量——"尽"字表示极限，已经哭到无泪可流，可还是在等待。这比"泪多"更令人心碎。

### 与《示儿》对读

| | 《秋夜将晓出篱门迎凉有感》 | 《示儿》 |
|---|---|---|
| 角度 | 陆游写北方百姓的苦盼 | 陆游写自己临终的未竟之愿 |
| 时间 | 约1192年，67岁 | 1209年，84岁 |
| 情感主体 | 遗民（他者） | 诗人自己（自我） |
| 情感特点 | 悲天悯人，向外 | 肝肠寸断，向内 |

两首合读是陆游爱国诗的"立体像"——既有对他人苦难的悲悯，也有对自身命运的喟叹。

## 🗣️ 家长讲解话术

- 孩子可能问 **"遗民是什么？"** → 可以说："'遗民'是被留下来的百姓——北方被金国占领了，住在那里的汉族百姓没办法逃走，就留在那里受苦。他们还认为自己是宋朝的人，盼望宋朝的军队来救他们。"

- 孩子可能问 **"泪尽是什么意思？"** → 可以说："就是眼泪哭干了——哭了这么多年，眼泪都没有了，可是还是在等。你想想，如果你等一个消息等了几十年，哭到没有眼泪了……是不是特别令人心疼？"

- 孩子可能问 **"又一年有什么特别的？"** → 可以说："一个'又'字，说明不是第一次，而是年年都在等，年年都在失望。这比说'等了很久'更让人感受到时间的漫长和悲哀。"

- **共读小活动（对比地图）**：在地图上找黄河和华山，再找到绍兴（陆游的家）和北京方向，感受陆游在绍兴望向北方、为北方百姓痛哭的地理方位感。
