# 关雎

> **一句话**：《诗经》开篇之作，以雎鸠鸟引出君子对窈窕淑女的思慕与追求，是中国最早的爱情诗之一，也是"兴"这一手法的典范。
> **English**: The opening poem of the *Book of Songs* uses the call of ospreys to evoke a gentleman's longing for a graceful young woman — one of China's earliest love poems and a classic example of the *xing* (evocative imagery) technique.

---

## 📖 课文简介

**出处**：《诗经·周南》首篇
**时代**：西周至春秋（约公元前 11—前 6 世纪）
**体裁**：四言诗，重章叠唱
**学段**：统编版八年级下册第三单元

## 📚 给 6-12 岁（自读版）

### 全诗与注释

> 关关雎鸠，在河之洲。窈窕淑女，君子好逑。
> 参差荇菜，左右流之。窈窕淑女，寤寐求之。
> 求之不得，寤寐思服。悠哉悠哉，辗转反侧。
> 参差荇菜，左右采之。窈窕淑女，琴瑟友之。
> 参差荇菜，左右芼之。窈窕淑女，钟鼓乐之。

| 词句 | 注释 |
|------|------|
| 关关 | 雎鸠鸟的叫声（拟声词）|
| 雎鸠（jū jiū）| 一种水鸟，雌雄常在一起，常作爱情和贞洁的象征 |
| 洲 | 水中的小陆地 |
| 窈窕（yǎo tiǎo）| 文静美好的样子 |
| 淑女 | 善良、温柔的女子 |
| 君子 | 品德高尚的男子 |
| 好逑（hǎo qiú）| 好的配偶 |
| 参差（cēn cī）| 长短不齐 |
| 荇（xìng）菜 | 水生植物，嫩叶可食 |
| 寤寐（wù mèi）| 醒时和睡时，即日夜 |
| 辗转反侧 | 翻来覆去睡不着 |
| 琴瑟友之 | 用琴瑟奏乐来亲近她 |
| 钟鼓乐之 | 用钟鼓奏乐使她快乐 |

### 诗意层次

| 章节 | 内容 | 情感 |
|------|------|------|
| 第一章 | 雎鸠起兴，引出淑女 | 爱慕之情初起 |
| 第二章 | 睡梦中都在思念 | 日夜思慕 |
| 第三章 | 求而不得，辗转难眠 | 相思之苦 |
| 第四五章 | 想象用琴瑟、钟鼓取悦她 | 对美好结合的憧憬 |

### "兴"的手法

**兴**（xìng）：先写其他事物，引起所要表达的情感。
- 这首诗先写"雎鸠在洲上鸣叫"——然后引出"君子思慕淑女"
- 雎鸠雌雄成对，暗示对美好婚姻的向往
- 这种"触景生情，借物起兴"的写法，是《诗经》最典型的手法之一

## 🎓 给 12+（进阶版）

### 《诗经》的基本结构

《诗经》共 305 篇，分三大类：
- **风**：各地民歌（15 国风，160 篇）——《关雎》属"周南"
- **雅**：宫廷雅乐（大雅、小雅，105 篇）
- **颂**：祭祀乐歌（周颂、鲁颂、商颂，40 篇）

《关雎》是"国风"之首，也是整部《诗经》的首篇，儒家视之为"乐而不淫，哀而不伤"的典范。

### 重章叠唱的音乐性

全诗三次出现"窈窕淑女"，两次出现"参差荇菜"——这是《诗经》"重章叠唱"的结构特征：
- 每章大体相同，只更换少数关键词（流之→采之→芼之；寤寐求之→琴瑟友之→钟鼓乐之）
- 关键词递进，情感层层深化：思慕→相思之苦→追求成功的想象
- 这种结构本为口头传唱设计，朗读时有音乐的回环之美

### "乐而不淫，哀而不伤"

孔子评价《关雎》"乐而不淫，哀而不伤"（《论语·八佾》）：
- **乐而不淫**：欢乐但不放纵；诗人对淑女的追求是文雅的、克制的
- **哀而不伤**：忧愁但不过度悲伤；辗转反侧的相思止于想象的美好结合

这一评价后来成为儒家诗教的核心理念：文学应当节制情感，引导向善，不可使人沉溺。

### 爱情诗的君子人格

这首诗中的"君子"并非只是爱慕者，而是承载了儒家人格理想：
- 他爱慕的是"淑女"（善良、有德行的女性），不只是容貌
- 追求方式是"琴瑟钟鼓"（文雅的礼乐），不是鲁莽行事
- 即使"求之不得"，也只是"辗转反侧"（内心煎熬），而非逾越礼制

## 🗣️ 家长讲解话术

- 孩子问 **"雎鸠是什么鸟？"** → 可以说："雎鸠是一种水鸟，和伴侣总是在一起，古人把它看作忠贞爱情的象征。诗人先写鸟，再写人，是一种叫'兴'的写法——用自然界的事物来引出人的情感。"
- 孩子问 **"辗转反侧是什么意思？"** → 可以说："就是翻来覆去睡不着——诗人太想念那位姑娘了，夜里躺下来心里一直想着她，翻身、侧身、再翻身，就是睡不着。你有没有过特别期待一件事、晚上睡不着的经历？"
- **共读小活动**：朗读全诗，找出三次"窈窕淑女"前后的词有什么不同，讨论情感有什么变化。
