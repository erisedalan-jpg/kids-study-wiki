# 送杜少府之任蜀州

> **一句话**：王勃以"海内存知己，天涯若比邻"一扫送别诗的悲戚之气，将真挚友情升华为超越空间的精神相连，是唐代送别诗中最豁达的篇章之一。
> **English**: Wang Bo sweeps away the sorrow typical of farewell poems — declaring that true friends remain close in spirit even when separated by oceans — one of the most uplifting parting poems of the Tang dynasty.

---

## 📖 课文简介

**作者**：王勃（649—676），字子安，唐代诗人，"初唐四杰"（王、杨、卢、骆）之首，英年早逝，代表作还有《滕王阁序》。
**体裁**：五言律诗
**背景**：诗人在长安送别友人杜少府赴蜀地（今四川）任职，有感而作。
**学段**：统编版八年级下册第三单元

## 📚 给 6-12 岁（自读版）

### 全诗与注释

> 城阙辅三秦，风烟望五津。
> 与君离别意，同是宦游人。
> 海内存知己，天涯若比邻。
> 无为在歧路，儿女共沾巾。

| 词句 | 注释 |
|------|------|
| 城阙（què）| 城门两侧的高楼，此指长安城 |
| 辅三秦 | 被三秦（关中三块地区）所护卫 |
| 风烟 | 风尘烟雾，指遥远的路途 |
| 五津 | 四川岷江上五个重要渡口，代指蜀地 |
| 宦游（huàn）| 在外地做官，漂泊异乡 |
| 海内 | 四海之内，即全国 |
| 知己 | 真正了解自己的朋友 |
| 天涯 | 天边，极远的地方 |
| 若比邻 | 好像近邻一样 |
| 歧路 | 分叉路，此指分别之处 |
| 儿女 | 此指普通人（非父母子女之意）|
| 沾巾 | 用手帕擦泪 |

### 层次分析

| 联别 | 内容 | 情感 |
|------|------|------|
| 首联 | 长安壮阔，蜀地遥远 | 点明送别之地，营造空间感 |
| 颔联 | 你我都是离家做官之人 | 同病相怜，拉近距离 |
| 颈联 | 四海之内有知己，天涯也如近邻 | 豁达，情感升华 |
| 尾联 | 不要在分别路口像平凡人一样哭泣 | 劝慰，豁达收尾 |

### 名句解析

**"海内存知己，天涯若比邻"**
- 真正的友情不因距离而淡漠：你在蜀地，我在长安，但我们心里都知道对方就在那里
- 这两句彻底推翻了"送别诗必须悲伤"的惯例——距离只是物理的，友情是精神的
- 后来这两句成为中国人表达友情最常引用的名句之一

## 🎓 给 12+（进阶版）

### 送别诗的"反常"之美

唐代送别诗大多以悲愁为基调，如王维"劝君更尽一杯酒，西出阳关无故人"，如李白"孤帆远影碧空尽，唯见长江天际流"。

王勃这首诗的独特在于：
- 颔联：不写"我多舍不得你"，而写"我们都是漂泊之人"——共情而非哀怜
- 颈联：不写"路途遥远，相见无期"，而写"有你这样的知己，天涯如近邻"——精神超越物理
- 尾联：直接劝对方"别哭"——气概豁达，有别于一般离别的哀婉

这种"送别而不悲"的风格，与初唐时期积极进取的时代精神相符。

### 律诗的结构分析

这是一首标准五言律诗：
- 每句五字，共八句，分四联
- 颔联（3-4 句）和颈联（5-6 句）要求对仗：
  - 颔联："与君离别意 / 同是宦游人"——结构工整
  - 颈联："海内存知己 / 天涯若比邻"——最为精彩的对仗
- 押韵：津、人、邻、巾（in 韵）

### 王勃与"初唐四杰"

王勃与杨炯、卢照邻、骆宾王并称"初唐四杰"，他们的出现标志着唐诗从南朝靡丽风格向大气磅礴转变的开端。王勃 27 岁时溺水而逝，留下的诗文中《滕王阁序》（散文骈文）和本诗（律诗）最为后人传诵。

## 🗣️ 家长讲解话术

- 孩子问 **"天涯若比邻是什么意思？"** → 可以说："天涯就是天边最遥远的地方，比邻就是邻居。王勃说：只要你心里有一个真正的好朋友，哪怕他在天边，也感觉像住在隔壁一样。你有没有这样的朋友——很久不见，但想到他/她就很安心？"
- 孩子问 **"为什么叫他不要哭？"** → 可以说："古代人送别朋友经常哭泣，因为一分别可能几年都见不到。但王勃说：真正的友情不靠见面，心里有对方就够了，所以不必哭——你有这样的底气，因为你们的友情很深。"
- **共读小活动**：想一想你最好的朋友，如果他/她要去很远的地方读书，你会怎么说？试着用"海内存知己，天涯若比邻"造个句子。
