# 北冥有鱼

> **一句话**：庄子以鲲化为鹏、扶摇九万里的寓言，探讨"逍遥"的真义——小知不及大知，"小大之辩"揭示的是对精神自由的追求。
> **English**: Zhuangzi opens *Xiaoyaoyou* with the mythic transformation of Kun into Peng, soaring ninety thousand *li* — a parable on the difference between petty and great understanding, and the pursuit of true spiritual freedom.

---

## 📖 课文简介

**作者**：庄子（约前 369—前 286），名周，战国时期道家代表人物，与老子并称"老庄"。
**出处**：《庄子·内篇·逍遥游》首段
**体裁**：哲理寓言（文言）
**学段**：统编版八年级下册（《庄子》二则之一，待家长核对单元号）

## 📚 给 6-12 岁（自读版）

### 原文与白话对照

| 原文 | 白话译文 |
|------|---------|
| 北冥有鱼，其名为鲲。 | 北海有一条鱼，名叫鲲。 |
| 鲲之大，不知其几千里也。 | 鲲的大，不知道有几千里。 |
| 化而为鸟，其名为鹏。 | 它变化成鸟，名叫鹏。 |
| 鹏之背，不知其几千里也。 | 鹏的背，也不知道有几千里。 |
| 怒而飞，其翼若垂天之云。 | 奋翅飞起，翅膀像天边挂下来的云。 |
| 是鸟也，海运则将徙于南冥。 | 这只鸟，大海运动时就要迁往南海。 |
| 南冥者，天池也。 | 南海，就是天然形成的大池。 |
| 鹏之徙于南冥也，水击三千里，抟扶摇而上者九万里。 | 鹏迁往南海，水花激起三千里，旋风盘旋飞上九万里。 |
| 去以六月息者也。 | 凭借六月的大风飞行。 |
| 野马也，尘埃也，生物之以息相吹也。 | 雾气、尘埃，都是生物用气息互相吹动。 |
| 天之苍苍，其正色邪？其远而无所至极邪？ | 天色深蓝，是它真正的颜色吗？还是因为太高太远看不到尽头？ |
| 其视下也，亦若是则已矣。 | 鹏从高处向下看，大概也是这样吧。 |

### 小知与大知

文章接着写了蜩（蝉）和学鸠（小鸟）嘲笑大鹏：
> 蜩与学鸠笑之曰："我决起而飞，抢榆枋而止，时则不至，而控于地而已矣，奚以之九万里而南为？"

蝉和小鸟讥笑大鹏说："我们奋力起飞，跳到榆树或檀树就停下来，飞不到也不过落到地上而已，为什么要飞九万里去南海呢？"

庄子的回应：**小知不及大知，小年不及大年**（见识小的不能理解见识大的）。

### "逍遥"是什么

逍遥 = 自由自在，无拘无束。庄子认为真正的逍遥是：不依赖外物，精神上达到完全自由的境界。蝉和小鸟有"小逍遥"，鹏有"大逍遥"，但庄子说两者都还不是最高的逍遥，因为鹏还要"凭借"风才能飞。

## 🎓 给 12+（进阶版）

### "小大之辩"的哲学层次

《逍遥游》探讨的核心问题：**什么是真正的自由？**

庄子设置了三个层次：

| 层次 | 代表 | 局限 |
|------|------|------|
| 小逍遥 | 蜩、学鸠 | 只能在榆枋之间，视野极小 |
| 大逍遥 | 大鹏 | 九万里高空，但仍需凭借"六月息（大风）" |
| 至逍遥 | 至人/神人/圣人 | 无己、无功、无名（后文提出）|

"北冥有鱼"这一段是铺垫——用鲲鹏之大反衬蜩雀之小，但庄子并非只是赞美大鹏，而是借此引出：**即便是大鹏，仍有所待（依赖）；真正的逍遥是"无所待"**。

### 重要文言词汇

| 词语 | 释义 | 备注 |
|------|------|------|
| 冥 | 深远的海 | 北冥=北方的深海 |
| 怒 | 奋发，振翅 | 不是愤怒，是用力鼓动 |
| 垂天之云 | 从天边垂下来的云 | 比喻翅膀之大 |
| 海运 | 大海运动 | 古注：海动则大风起 |
| 徙 | 迁移 | |
| 天池 | 天然大池（指南海）| |
| 水击三千里 | 击水三千里（腾飞时）| |
| 抟 | 盘旋上升 | 抟扶摇=凭旋风盘旋 |
| 扶摇 | 旋风 | |
| 息 | 气息，此处指风 | 六月息=夏季大风 |
| 野马 | 游动的雾气 | 非马 |
| 苍苍 | 深蓝色 | |

### 词类活用

- **南冥**：名词，南方的大海（冥：深海，加方位词构成地名）
- **怒而飞**：怒，形容词作动词，奋力振翅
- **奚以之九万里**：之，动词，到、去往

### 浪漫主义与神话思维

庄子的文章以"寓言""重言""卮言"为主要手法，《逍遥游》开篇以神话性的鲲鹏意象，突破现实的逻辑约束，创造出极度恢宏的想象空间。鲲鹏意象在后世诗文中被广泛引用（如李白"大鹏一日同风起，扶摇直上九万里"），成为志向高远的经典象征。

## 🗣️ 家长讲解话术

- 孩子问**"鲲鹏真的存在吗？"** → 可以说："庄子自己也说这是从《齐谐》（一本记怪事的书）里引来的，鲲鹏是他想象出来的——目的不是描述真实的动物，而是用一个超大的比喻来帮我们理解'大'和'小'之间的差距。"
- **共读小活动**：让孩子想象自己是大鹏，从九万里高空向下看——你能看到什么？再想象自己是蝉，在树梢上——你能看到什么？对比一下，谁的眼界更大？
