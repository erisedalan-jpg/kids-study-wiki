# 长沙过贾谊宅

> **一句话**：唐代诗人刘长卿贬谪途经长沙，凭吊西汉才子贾谊的故宅，借贾谊的遭遇抒发自己怀才不遇的心境，是借古讽今、以他人之酒浇自己胸中块垒的典型咏史诗。
> **English**: The Tang poet Liu Changqing, passing through Changsha during his exile, visits the former home of Han scholar Jia Yi and uses Jia Yi's tragic fate as a mirror for his own thwarted ambitions — a classic of historical-allusion poetry.

---

## 📖 课文简介

**作者**：刘长卿（约726—约790），字文房，唐代诗人，擅长五、七言近体诗，曾两度遭贬。  
**体裁**：七言律诗，8 句 4 联  
**背景**：刘长卿遭贬途经长沙，访贾谊故宅有感而作。贾谊（前200—前168）是西汉政治家、文学家，年少有才，受到汉文帝重用，后因受权贵排挤被贬长沙，郁郁而终，年仅 33 岁。  
**学段**：统编版九年级上册课外古诗词诵读（待家长核对）

## 📚 给 6-12 岁（自读版）

### 全诗文本

> 三年谪宦此栖迟，万古惟留楚客悲。  
> 秋草独寻人去后，寒林空见日斜时。  
> 汉文有道恩犹薄，湘水无情吊岂知？  
> 寂寂江山摇落处，怜君何事到天涯！

### 注释

| 词语 | 解释 |
|------|------|
| 谪宦（zhé huàn） | 被贬谪的官员 |
| 栖迟 | 居住，停留（有漂泊之意） |
| 楚客悲 | 楚地（长沙一带）被贬之人的悲哀 |
| 独寻 | 独自寻访（诗人只身前来） |
| 人去后 | 贾谊已死去多年 |
| 汉文 | 汉文帝，曾赏识贾谊但又将他贬谪 |
| 有道 | 施行仁政，有明君之道 |
| 恩犹薄 | 即使是明君，对贾谊的恩情仍然不够深厚 |
| 湘水 | 湘江，长沙一带的江 |
| 吊 | 凭吊，悼念 |
| 摇落 | 草木凋零（秋天景象，兼指人的零落） |
| 怜君 | 怜悯你（指贾谊） |
| 天涯 | 极远之地（指长沙，古人视为边荒之地） |

### 四联解读

| 联 | 诗句 | 内容 |
|----|------|------|
| 首联 | 三年谪宦…楚客悲 | 贾谊被贬三年，留下千古悲剧 |
| 颔联 | 秋草独寻…日斜时 | 诗人独自在秋草中寻访，寒林日斜，渲染荒凉 |
| 颈联 | 汉文有道…吊岂知 | 即使明君也薄情，湘水无情不能传递哀思 |
| 尾联 | 寂寂江山…天涯 | 江山萧条，可怜贾谊为何要落到天涯之地 |

## 🎓 给 12+（进阶版）

### 借古抒怀的结构

本诗表面写贾谊，实际写自己，是"以古人之境寄今人之情"的咏史诗写法：

- 贾谊：才华横溢，却被贬长沙三年，英年早逝
- 刘长卿：也曾遭贬，过长沙时感同身受

诗人没有直接说"我也遭遇了不公"，而是借贾谊的遭遇说话——这叫**借古讽今**（或借古抒怀），情感更含蓄深沉。

### 景物描写的渲染功能

颔联"秋草独寻人去后，寒林空见日斜时"——

- **秋草**：凋零，对应人的零落
- **独寻**：诗人孤身一人，孤独感
- **人去后**：贾谊已不在，只有遗址
- **寒林**：寒冷的树林，增添凄凉
- **日斜时**：夕阳西下，暗示时间的消逝，也暗示诗人自己已到暮年

每一个景物都精心选择，共同营造出**凄清荒寂**的氛围，与贾谊的悲剧命运相呼应。

### 颈联的反讽

"汉文有道恩犹薄"——汉文帝是历史上著名的贤君（"文景之治"），尚且如此对待贾谊，暗示即使在最好的政治环境下，文人的命运依然脆弱。这里有一种深沉的悲凉：明君尚且如此，何况其他？
