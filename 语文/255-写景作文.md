# 写景作文

> **一句话**：写景作文不是"照相"，而是"绘画"——要选取最有特点的景物，调动多种感官，融入情感，让读者如身临其境。
> **English**: Descriptive landscape writing is not photography but painting — select the most distinctive features, engage multiple senses, infuse emotion, and make readers feel they are there.

---

## 📚 给 6-12 岁（小学高年级回溯版）

你有没有去过一个很美的地方，想把它写下来，却发现写出来很无聊？

问题可能是：你只写了"看到了什么"，但没有写"感觉到什么"。

好的写景作文要有：
- **眼睛**：看到的颜色、形状
- **耳朵**：听到的声音
- **鼻子**：闻到的气味
- **皮肤**：感觉到的温度、风
- **心**：你当时的感受和心情

**反例**：秋天的公园很美，有很多红叶。
**正例**：秋风吹过，红色和金色的叶子像蝴蝶一样打转落下，踩在脚下发出"沙沙"的声音，空气里有股淡淡的泥土香——这才是秋天的感觉。

## 🎓 给 12+（进阶版）

### 写景作文的核心问题

写景作文最常犯的错误：**写景但没有情感，读者读了没有感觉。**

**好的写景 = 有特点的景 + 准确的描写手法 + 融入的情感**

### 写景的五感运用

| 感官 | 描写角度 | 例子（《春》） |
|------|----------|---------|
| 视觉 | 颜色、形状、光线 | 红的像火，粉的像霞，白的像雪 |
| 听觉 | 声音 | 鸟儿呼朋引伴，卖弄清脆的喉咙 |
| 嗅觉 | 气味 | 花里带着甜味儿，泥土的气息 |
| 触觉 | 温度、质感、风 | 风轻悄悄的，草软绵绵的 |
| 味觉 | 口感（写景中较少，但可用） | 闭了眼，仿佛满是桃儿杏儿梨儿 |

### 写景的常用手法

**1. 抓特点**（最重要）
- 不同地方/季节的景物有不同特点
- 写冬天的雪山，就要写那种寂静和苍茫
- 写春天的田野，就要写生机勃勃
- 抓住特点，景物才有个性

**2. 动静结合**
- 有动有静，画面才生动
- 例：远处的山（静），近处的溪水（动）；停着的鸟（静），飞起的花瓣（动）

**3. 远近高低结合**（空间层次）
- 从远到近，或从近到远
- 从高到低，或从低到高
- 这样读者的眼睛有个"移动"的感觉

**4. 修辞手法**
- **比喻**：用已知的事物比喻景物（如"草地像绿色的地毯"）
- **拟人**：赋予景物人的动作和情感（如"春风轻轻推开了窗户"）
- **排比**：列举多种景物，节奏感强（如"有……有……有……"）

**5. 情景交融**（高级）
- 写景服务于情感
- 快乐时，景物往往明亮、温暖
- 悲伤时，景物往往阴沉、肃穆
- 不直接说情感，让景物替你说

### 七上课文写景技法对比

| 课文 | 主要景物 | 特色技法 |
|------|----------|----------|
| 《春》 | 草、花、风、雨 | 五幅图，各有侧重；排比、拟人、比喻 |
| 《济南的冬天》 | 山城、雪山、水 | 温情拟人；色彩描写丰富 |
| 《雨的四季》 | 四季之雨 | 四种性格对比；多感官并用 |

### 写景作文结构

**基本结构**（三段式）：
1. **开头**：点明写景对象，简要交代时间地点，吸引读者
2. **主体**：多角度、有层次地描写景物（可按空间顺序或时间顺序）
3. **结尾**：由景抒情，点明主旨或情感

**高级结构**（首尾呼应）：
- 开头和结尾都写同一个景物或句子，但情感深度不同
- 给人"读完一圈，回到原点，却感悟更深"的感觉

### 常见错误与改进

| 错误 | 改进 |
|------|------|
| 景物描写泛泛，没有特点 | 聚焦最有特点的一两处，细细描绘 |
| 只有眼睛，没有其他感官 | 加入听觉、嗅觉、触觉 |
| 景与情脱节 | 在景物描写中融入情感词或情感性的比喻 |
| 修辞堆砌，不自然 | 修辞要为内容服务，不要为用而用 |
| 平铺直叙，没有层次 | 用空间顺序（远→近，整体→局部）或时间顺序 |
