# 山水游记（综合篇）

> **一句话**：山水游记是中国文学的重要体裁，从六朝到唐宋，从文言散文到现代白话游记，均以描写山水景物为主体，融入作者的情感和哲思，形成"景中有情、情景交融"的独特美学。九年级上册第三单元以《岳阳楼记》《醉翁亭记》《湖心亭看雪》为代表，是初中山水游记的集中学习。
> **English**: Chinese landscape travel writing spans from Six Dynasties to Song-dynasty classical essays to modern prose, always blending scenery with emotion — the 9th-grade unit featuring Yueyang Tower, Drunkard's Pavilion, and the mid-lake snow pavilion is its richest study.

## 📚 给 6-12 岁（自读版）

### 什么是游记？

游记 = 记录游览经历的文章。

简单的游记：去哪里玩了，看到了什么。

文学游记（山水游记）：不只写看到了什么，还写**感受**——景物激起了什么情感，引发了什么思考。

中国古代山水游记的独特之处：往往以景物为镜，映照作者的内心世界。

### 山水游记的两大要素

**1. 写景**

描写山、水、动植物、天气、光线等自然景物。  
写景技法：
- 远近结合（先远景，再近景）
- 动静结合（静态山峦 + 动态流水）
- 声色结合（颜色 + 声音）
- 虚实结合（眼前实景 + 想象虚景）

**2. 抒情/议论**

游览之后的情感或思考。  
游记中情感的来源：
- 景物的本身特点（幽静→凄凉；开阔→壮志）
- 作者当时的处境（贬谪 → 压抑；得志 → 豪情）
- 历史文化的联想（古代名胜 → 时间感）

### 初中阶段的山水游记篇目

| 篇目 | 作者 | 朝代/时期 | 特点 |
|------|------|---------|------|
| 三峡 | 郦道元 | 北魏 | 地理描写 + 四季景色 |
| 答谢中书书 | 陶弘景 | 南朝 | 书信体，赞美山水之美 |
| 小石潭记 | 柳宗元 | 唐 | 借景抒情，乐→凄 |
| 岳阳楼记 | 范仲淹 | 北宋 | 写景 + 议论，先忧后乐 |
| 醉翁亭记 | 欧阳修 | 北宋 | 写景 + 与民同乐，骈散结合 |
| 湖心亭看雪 | 张岱 | 明末 | 白描，极简，故国之思 |

## 🎓 给 12+（进阶版）

### 山水游记的情感类型

不同处境的作者，面对同样的山水，写出不同的情感：

| 作者处境 | 情感基调 | 代表作 |
|---------|---------|-------|
| 贬谪失意 | 寄情山水，抑郁或豁达 | 柳宗元《小石潭记》、欧阳修《醉翁亭记》 |
| 政治理想 | 借景议论，抒发理想 | 范仲淹《岳阳楼记》 |
| 故国之思 | 孤独、追忆 | 张岱《湖心亭看雪》 |
| 哲学思考 | 借景明理 | 苏轼《赤壁赋》（高中） |

### 写景与抒情的结构关系

**借景抒情**（触景生情）：先描写景物，景物触动情感，情感由景而生
- 《小石潭记》：潭的凄清→触发贬谪之苦

**情景交融**：景中已含情，情中已含景，难以分离
- 《岳阳楼记》：阴雨景 = 悲情；晴朗景 = 喜情（景与情一一对应）

**以景说理**：以景为喻，说明哲理（多见于散文化议论）
- 《岳阳楼记》结尾："不以物喜，不以己悲"——用景物变化来说明超然境界

### 山水游记与现代游记的异同

| | 古典山水游记 | 现代游记 |
|-|------------|---------|
| 语言 | 文言或文白夹杂 | 现代白话 |
| 写景比例 | 高 | 高 |
| 情感表达 | 含蓄，借景抒情 | 可以直接表达 |
| 议论成分 | 常有（如岳阳楼记） | 较少 |
| 主题 | 借景明志，抒发身世 | 旅行体验，分享感受 |
