# 十五从军征

> **一句话**：这首汉乐府民歌讲述一个十五岁从军的老兵，八十岁回到家乡，却发现物是人非、家破人亡，以"出门东向望，泪落沾我衣"作结，将长期战争对普通家庭的摧毁表达得触目惊心。
> **English**: This Han dynasty folk ballad follows a soldier who left home at fifteen and returns at eighty to find his family gone and his house in ruins, ending with tears soaking his collar — a stark, heartbreaking indictment of how endless war destroys ordinary families.

---

## 📖 课文简介

**来源**：《乐府诗集》，汉代乐府民歌，作者不详。乐府是汉代负责音乐的官府，收集并整理民间歌谣，其中包含大量反映战争、劳役、爱情的民歌。
**体裁**：五言乐府诗
**主题**：战争对普通百姓的摧残，思乡，老兵归家
**学段**：统编版九年级下册第六单元 诗词曲五首

## 📚 给 6-12 岁（自读版）

### 全诗原文

十五从军征，八十始得归。
道逢乡里人，家中有阿谁？
遥看是君家，松柏冢累累。
兔从狗窦入，雉从梁上飞。
中庭生旅谷，井上生旅葵。
舂谷持作饭，采葵持作羹。
饭熟羹已成，不知饴阿谁。
出门东向望，泪落沾我衣。

### 重要注释

| 词语 | 注释 |
|------|------|
| 始得归 | 才能够回来 |
| 阿谁 | 谁，有谁（口语化问法） |
| 冢累累 | 坟墓一个叠着一个，形容很多 |
| 狗窦 | 狗进出的洞（墙上的小洞） |
| 雉 | 野鸡 |
| 旅谷 | 野生的谷子（无人种植，自己长出来的） |
| 旅葵 | 野生的葵菜 |
| 舂谷 | 用石臼捣去谷壳 |
| 羹 | 带汤的菜肴，此处指葵菜汤 |
| 饴阿谁 | 喂给谁吃，分给谁 |
| 东向望 | 向东张望 |
| 沾 | 沾湿 |

### 白话译文

十五岁就去从军打仗，八十岁才得以回来。在路上遇到同乡人，问：我家里还有什么人吗？那人遥望了一下说：那就是你的家，松柏丛中，坟墓一个叠着一个。

走进家门，看见：兔子从狗洞里钻进来，野鸡在屋梁上飞翔。院子里长着野生的谷子，井台上长着野生的葵菜。

我捣谷子做饭，采葵菜熬汤。饭熟了，汤也好了，却不知道要分给谁吃。走出门口向东望去，泪水落下，沾湿了我的衣服。

### 诗的结构

| 层次 | 内容 | 情感 |
|------|------|------|
| 开篇 | 六十五年后回家（15→80） | 震惊，时间之残酷 |
| 问路 | 同乡人指路，暗示家人已逝 | 预感，惶恐 |
| 归家 | 荒芜的院落：兔、雉、野谷、野葵 | 凄凉，物是人非 |
| 做饭 | 煮饭熬汤，却无人可与同食 | 孤独，悲痛 |
| 结尾 | 出门东望，泪落沾衣 | 绝望，无言之悲 |

## 🎓 给 12+（进阶版）

### 白描手法的震撼力量

这首诗几乎没有直接抒情，全用白描——客观叙述事物和场景：
- 兔从狗窦入（兔子进了狗洞）
- 雉从梁上飞（野鸡在梁上）
- 旅谷、旅葵（野生的谷和菜）

这些细节不说"悲"字，却比说"悲"更令人心碎——家里没有人，动物进来安家，野草野菜自由生长，六十五年的等待换来的是这一片荒芜。

最后"泪落沾我衣"也没有说"我痛苦""我绝望"，只是陈述：泪落了，沾湿了衣服。这正是乐府民歌"以朴素写深情"的艺术特色。

### "十五"与"八十"的数字对比

- 十五岁：一个孩子，刚刚开始人生
- 八十岁：一个老人，人生即将结束
- 中间：六十五年的战争与流离

这种极端的数字对比，不需要任何议论，就已经揭示了战争对人一生的剥夺。

### 乐府民歌的文学特征

1. **语言口语化**：如"家中有阿谁""饭熟羹已成"——自然流畅，像人说话
2. **叙事性强**：整首诗像讲一个故事，有情节发展
3. **细节真实**：通过具体的物象（兔、雉、旅谷、旅葵）营造真实感
4. **情感含蓄**：不直接说感情，而是通过行动和场景暗示
5. **反战主题**：不通过论述，而是通过个人命运呈现战争的代价

### 与《木兰辞》的比较

| | 《木兰辞》 | 《十五从军征》 |
|--|---------|---------|
| 主角 | 女子（木兰），凯旋而归，家人迎接 | 老兵，回家却家破人亡 |
| 结局 | 圆满，家庭团圆 | 悲剧，无人相迎 |
| 态度 | 颂扬忠孝 | 哀叹战争代价 |
| 共同点 | 都是汉乐府，都写战争 | 均体现乐府民歌风格 |
