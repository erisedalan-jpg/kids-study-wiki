# 南乡子·登京口北固亭有怀

> **一句话**：辛弃疾登上京口（今镇江）北固亭，望神州而兴怀，以孙权、曹操的英雄气概为对照，抒发自己老当益壮却无从报国的深沉感慨。
> **English**: Standing atop Beigu Pavilion at Jingkou, Xin Qiji gazes toward the lost north and invokes the heroism of Sun Quan and Cao Cao to express his own poignant sense of aging while unable to serve his country.

---

## 📖 课文简介

**作者**：辛弃疾（1140—1207），字幼安，号稼轩，南宋词人，豪放词派代表。
**体裁**：词，词牌名"南乡子"，双调五十四字
**背景**：辛弃疾晚年被起用为镇江知府，登北固亭，凭栏远望，写下此词。北固亭位于今江苏镇江北固山，俯瞰长江，是历史上兵家必争之地，三国时孙权曾驻此经营江东。
**学段**：统编版九年级下册第六单元 诗词曲五首

## 📚 给 6-12 岁（自读版）

### 全词原文

| 句 | 原文 |
|----|------|
| 上阕 | 何处望神州？满眼风光北固楼。千古兴亡多少事？悠悠。不尽长江滚滚流。 |
| 下阕 | 年少万兜鍪，坐断东南战未休。天下英雄谁敌手？曹刘。生子当如孙仲谋。 |

### 重要注释

| 词语 | 注释 |
|------|------|
| 神州 | 中国的别称，此特指被金朝占领的中原北方 |
| 北固楼 | 北固亭，在镇江北固山上 |
| 千古兴亡 | 历史上各朝代的兴起与衰亡 |
| 悠悠 | 绵延不尽，形容时间长远 |
| 不尽长江滚滚流 | 化用杜甫"无边落木萧萧下，不尽长江滚滚来"，以江流比时间流逝 |
| 兜鍪 | 古代士兵的头盔，此处以兜鍪代指士兵 |
| 年少万兜鍪 | 孙权年纪轻轻就统领万余士兵 |
| 坐断东南 | 占据东南，指孙权在江东建立政权 |
| 战未休 | 征战不息 |
| 曹刘 | 曹操与刘备 |
| 孙仲谋 | 孙权的字（仲谋）。曹操曾说"生子当如孙仲谋"，赞叹孙权的雄才 |

### 白话译文

**上阕**：从哪里能望见（被占领的）中原故土？站在北固亭上，满眼只是眼前的风光。千百年来，多少王朝兴亡更替？（那些历史）悠悠远去，就像这滔滔长江永远不停地流淌。

**下阕**：（孙权）年纪轻轻就统领千军万马，占据东南，征战不休。天下的英雄谁能与他匹敌？只有曹操和刘备（才能算是）。（曹操曾感叹：）生儿子应当像孙仲谋（孙权）这样！

### 结构分析

| 层次 | 内容 | 情感 |
|------|------|------|
| 上阕 | 登楼望远，思千古兴亡，以长江起兴 | 深沉的历史感慨 |
| 下阕 | 借孙权英雄气概，含蓄表达报国壮志 | 豪情中带遗憾 |

## 🎓 给 12+（进阶版）

### 三个问句的修辞效果

全词三处设问，层层递进：
1. "何处望神州？"——直接抒发收复失地之渴望，一开篇就奠定全词基调
2. "千古兴亡多少事？悠悠。"——用"悠悠"回答，化具体为抽象，历史兴亡都如流水而去
3. "天下英雄谁敌手？曹刘。"——赞美孙权，连天下英雄中最强的曹操、刘备都将他视为对手

设问使情感表达更有力，也使读者参与思考，具有感染力。

### 孙权典故的深意

辛弃疾在此词中大篇幅写孙权（"年少万兜鍪""坐断东南""孙仲谋"），其深意在于：
- 孙权年轻时便能统兵作战，坐断东南——而作者晚年被起用，却难有真正的施展空间
- 孙权抵御曹操，保持独立——而南宋不能北伐收复，与之对比令人唏嘘
- 以古代英雄的豪迈映衬自己壮志难申的遗憾

这是辛弃疾词中常见的"借古讽今"手法。

### 与《破阵子》的比较

| | 《破阵子》 | 《南乡子》 |
|--|---------|---------|
| 写法 | 直接描写战场梦境 | 借古代英雄来寄托 |
| 情感 | 悲壮，以"可怜白发生"收尾 | 深沉，以"生子当如孙仲谋"感叹 |
| 典故 | 的卢、冯唐 | 孙权、曹操、刘备 |
| 共同主题 | 壮志难酬，报国无门 | 壮志难酬，报国无门 |

### 化用前人诗句

"不尽长江滚滚流"——化用杜甫《登高》："无边落木萧萧下，不尽长江滚滚来。"

辛弃疾将杜甫的原句"滚滚来"改为"滚滚流"，使长江之流与"千古兴亡"的悠悠感更紧密结合，也体现宋词"以诗入词"的艺术传统。
