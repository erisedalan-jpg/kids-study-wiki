# 白雪歌送武判官归京

> **一句话**：岑参以西域的奇异大雪为背景，用"忽如一夜春风来，千树万树梨花开"的神奇比喻写出塞外雪景的壮美，再以送别友人的依依不舍收尾，是唐代边塞诗中写雪最著名的作品。
> **English**: Set against extraordinary snowfall in the western frontier, Cen Shen uses the famous simile of pear blossoms blooming overnight to capture the spectacle of the snowscape, then closes with the lingering sorrow of parting — a celebrated masterpiece of Tang border poetry.

---

## 📖 课文简介

**作者**：岑参（约 715—770），唐代著名边塞诗人，曾两次赴西域（今新疆一带）任职，写下大量描写边疆风光与战争生活的诗篇，与高适并称"高岑"。
**体裁**：七言古诗（乐府体）
**背景**：天宝十三年（754），岑参在轮台（今新疆轮台附近）任职，友人武判官奉命回京，岑参作此诗送别。
**学段**：统编版九年级下册第六单元 诗词曲五首

## 📚 给 6-12 岁（自读版）

### 全诗原文

北风卷地白草折，胡天八月即飞雪。
忽如一夜春风来，千树万树梨花开。
散入珠帘湿罗幕，狐裘不暖锦衾薄。
将军角弓不得控，都护铁衣冷难着。
瀚海阑干百丈冰，愁云惨淡万里凝。
中军置酒饮归客，胡琴琵琶与羌笛。
纷纷暮雪下辕门，风掣红旗冻不翻。
轮台东门送君去，去时雪满天山路。
山回路转不见君，雪上空留马行处。

### 重要注释

| 词语 | 注释 |
|------|------|
| 白草 | 西域的一种草，干枯后呈白色 |
| 胡天 | 北方或西域的天空 |
| 忽如 | 好像突然 |
| 梨花 | 梨花白色，此比喻雪压枝头 |
| 珠帘 | 用珍珠穿成的帘子 |
| 罗幕 | 丝绸做的帷幕 |
| 狐裘 | 狐皮大衣 |
| 锦衾 | 锦缎被子 |
| 角弓 | 用兽角制作的弓 |
| 不得控 | 无法拉开（天太冷，手指僵硬） |
| 都护 | 西域最高军事长官 |
| 着 | 穿上 |
| 瀚海 | 大沙漠（戈壁滩） |
| 阑干 | 纵横交错的样子 |
| 愁云惨淡 | 阴沉沉的云，暗喻送别愁情 |
| 中军 | 主帅的营帐 |
| 辕门 | 军营大门 |
| 风掣红旗 | 寒风吹动红旗 |
| 冻不翻 | 红旗被冻住，迎风不飘扬 |
| 轮台 | 地名，今新疆轮台县附近 |

### 白话译文（节选重点段）

**开篇写雪景**：北风席卷大地，连白草都被折断，胡地（西域）八月就飘起了大雪。忽然间，就像一夜春风吹来，千万棵树上全开满了白色的梨花（实为树枝上的积雪）。

**写严寒**：雪花飘入珠帘，打湿丝绸帷幕；狐皮大衣不够暖，锦缎被子太薄了。将军的角弓冻得拉不开，军官的铁甲冷得穿不上身。

**写送别**：在帅营中摆酒为归客送行，伴奏着胡琴、琵琶和羌笛。傍晚大雪纷纷飘落在营门外，寒风吹动红旗，旗帜被冻住了，迎风不翻动。

**送别结尾**：在轮台东门外送你离去，离别时雪花铺满了天山的路。山路蜿蜒曲折，早已看不见你的身影，雪地上只留下你马蹄走过的印迹。

### 诗的结构

| 层次 | 内容 | 情感 |
|------|------|------|
| 前 8 句 | 写奇异雪景（梨花喻、严寒） | 壮美、奇丽 |
| 中间 6 句 | 写送别宴会 | 温情中带离愁 |
| 后 4 句 | 写送别依依，雪中送别 | 依恋、惆怅 |

## 🎓 给 12+（进阶版）

### 千古名句赏析："忽如一夜春风来，千树万树梨花开"

这两句之所以脍炙人口，在于：
- **以春喻冬**：用"春风""梨花"这些温暖美好的意象，比喻严寒的塞外雪景
- **化险为美**：将严酷的自然环境转化为令人惊叹的奇美景观
- **视角独特**：不从负面写雪之寒苦，而从正面写雪之绚丽，展现出边塞诗人豁达的胸怀
- **比喻奇绝**：白雪覆盖树枝，恰如梨花盛开，两种白色事物的联想极为自然又新奇

这一比喻影响极深，后代文人写雪常常化用，成为中国诗歌中写雪的经典意象。

### 边塞诗的特征

岑参此诗集中体现了唐代边塞诗的主要特征：
1. **奇异壮丽的自然景观**：八月飞雪、百丈冰、风掣红旗
2. **特色鲜明的边地器物**：角弓、铁衣、胡琴、琵琶、羌笛（各民族文化交融）
3. **军旅生活的真实细节**：将军角弓不得控、都护铁衣冷难着
4. **豪迈与深情并存**：雄壮的雪景与依依送别的温情结合

### 结尾的留白艺术

"山回路转不见君，雪上空留马行处"

这两句以"不见"和"空留"作结：
- 友人已消失在山转路回之处
- 雪地上只剩下马蹄印
- 送行者久久望着那条路，不愿离去

这种"人去景留"的写法，不直说悲伤，却胜过一切直白的离愁表达，是中国诗歌"留白"手法的典型运用。

### 重要词汇

| 词汇 | 含义 | 用法说明 |
|------|------|----------|
| 即 | 就，已经 | "胡天八月即飞雪"——八月就下雪 |
| 掣 | 拉扯 | "风掣红旗"——风拉扯着红旗 |
| 空 | 徒然，只 | "雪上空留"——只留下（人已不在） |
