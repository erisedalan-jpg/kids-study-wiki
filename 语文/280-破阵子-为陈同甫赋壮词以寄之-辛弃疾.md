# 破阵子·为陈同甫赋壮词以寄之

> **一句话**：辛弃疾写给友人陈亮的这首词，由醉后挑灯看剑的梦境写起，展现出沙场奋战的壮阔图景，却以"可怜白发生"的现实作结——光复中原的壮志与报国无门的悲愤，在梦醒之间构成强烈对比。
> **English**: Written for his friend Chen Liang, Xin Qiji opens with a drunken vision of lamp-lit sword-gazing, unfolds a panorama of fierce battle, then closes with the stark reality of grey hair — the dream of recovering the north crushed by the waking sorrow of an unfulfilled patriot.

---

## 📖 课文简介

**作者**：辛弃疾（1140—1207），字幼安，号稼轩，南宋词人，豪放词派代表，出生于金统治下的济南，一生致力于抗金复国，但屡遭排斥，壮志难酬。
**体裁**：词，词牌名"破阵子"，双调六十二字
**背景**：辛弃疾被迫赋闲，隐居江西带湖期间，写此词赠给同怀壮志的友人陈亮（字同甫），两人均主张抗金，均不得志。
**学段**：统编版九年级下册第三单元

## 📚 给 6-12 岁（自读版）

### 全词原文

| 句 | 原文 |
|----|------|
| 上阕 | 醉里挑灯看剑，梦回吹角连营。八百里分麾下炙，五十弦翻塞外声，沙场秋点兵。 |
| 下阕 | 马作的卢飞快，弓如霹雳弦惊。了却君王天下事，赢得生前身后名。可怜白发生！ |

### 重要注释

| 词语 | 注释 |
|------|------|
| 挑灯 | 拨亮灯芯（古代油灯变暗需挑灯芯） |
| 梦回 | 从梦中醒来，或在梦中回到 |
| 吹角连营 | 军营中号角声此起彼伏 |
| 八百里 | 典出晋王恺的名牛"八百里驳"，此指牛，代指烤肉 |
| 麾下 | 将帅旗下，指部下将士 |
| 炙 | 烤肉 |
| 五十弦 | 古乐器瑟有五十根弦，此代指各种军乐 |
| 塞外声 | 雄壮的边塞军乐 |
| 点兵 | 检阅部队 |
| 的卢 | 良马名，据说刘备的坐骑名"的卢"，善于飞奔 |
| 霹雳 | 霹雳，打雷声，此形容弓弦振动之声 |
| 了却 | 完成，了结 |
| 君王天下事 | 收复失地、统一天下的大业 |
| 可怜 | 可惜，令人惋惜 |

### 白话译文

上阕：醉酒之后，拨亮灯芯，对着宝剑发呆（梦中）回到了军营，号角声声响彻连绵军营。将士们分食烤牛肉，乐声奏起雄壮的塞外军歌，在这秋日沙场上，正在检阅三军。

下阕：战马像的卢一样飞快奔驰，拉弓放箭时弓弦的声音如同霹雳。我要完成君王光复天下的大业，赢得生前的功名与身后的英名。（然而现实是）可惜啊，白发已经生出！

### 情感结构

| 层次 | 内容 | 情感 |
|------|------|------|
| 醉中现实 | 挑灯看剑 | 壮志难消，借酒浇愁 |
| 梦中场景 | 军营、点兵、出征、立功 | 豪情壮志，慷慨激昂 |
| 梦醒现实 | "可怜白发生" | 壮志难酬，悲愤沉郁 |

## 🎓 给 12+（进阶版）

### "梦"的结构艺术

全词只有最后五字"可怜白发生"回到现实，其余均是梦境（或由梦境生发的想象）：
- 梦境中的军营豪壮景象密集呈现（5 个意象：角声、烤肉、军乐、秋点兵、飞马）
- 梦境中的功业宏大清晰（了却君王天下事，赢得生前身后名）
- 现实只有一句——与梦境的反差越大，悲剧力量越强

这种以"梦"写志、以"醒"写悲的结构，是辛弃疾词中常见的艺术手法，情感密度极高。

### 辛弃疾的历史处境

辛弃疾出生于金朝占领的山东，22 岁起兵抗金，归宋后却始终受到主和派排斥，被长期闲置。他的词充满：
- 对北伐失败的痛苦
- 对主和派苟且偷安的愤怒
- 对自身壮志难酬的悲哀

"了却君王天下事"的"君王天下事"，是收复北方失地——这是南宋一代抗金志士的共同理想。

### 典故运用

| 典故 | 出处 | 作用 |
|------|------|------|
| 八百里 | 晋王恺的名牛 | 以牛代烤肉，写将士同甘共苦 |
| 的卢 | 刘备坐骑 | 以名马写战马神骏，营造英雄气氛 |

### 艺术特色

1. **意象密集**：上阕五句，五个意象（角声、炙、乐、秋、兵），节奏紧凑
2. **声色并茂**：有声音（角声、霹雳），有色彩（灯光、秋日沙场），有动作（飞马、射箭）
3. **以乐景衬哀情**：梦境越壮阔，"可怜白发生"的收尾越令人扼腕
4. **对仗工整**："马作的卢飞快，弓如霹雳弦惊"——两句对仗，写出战斗速度与力量
