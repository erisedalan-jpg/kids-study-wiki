# 驳论文

> **一句话**：驳论文是议论文的一种，以反驳对方错误观点为主要目的，通过"先破后立"的方式——指出对方的错误，再建立自己的正确主张——是初中阶段学习的重要论证方式，九下《不求甚解》是典型的驳论文范例。
> **English**: A refutation essay is a form of argumentative writing that primarily aims to disprove an opposing view, following the "break first, then build" structure — identifying the error in an opponent's position and then establishing one's own — a key argumentation form at the middle-school level.

---

## 📖 文体简介

**归属**：议论文
**核心特征**：以反驳对方观点为出发点
**与立论文的区别**：
- **立论文**：直接提出自己的观点并论证（正向论证）
- **驳论文**：先批驳对方的错误，然后建立自己的观点（反驳 + 立论）

---

## 📚 给 6-12 岁（自读版）

### 驳论文的基本结构

```
1. 引出被批驳的观点（提出对方的错误主张）
2. 分析对方观点的错误（怎么错的？）
   ├── 反驳论点（对方的主张本身是错的）
   ├── 反驳论据（对方的材料是假的或站不住脚）
   └── 反驳论证（即使前提对，推论过程也错了）
3. 建立自己的正确观点
4. 总结
```

### 三种反驳方法

| 方法 | 说明 | 例子 |
|------|------|------|
| 反驳论点 | 直接说"对方的主张不对" | 直接说"不求甚解不是懒惰，它有特定含义" |
| 反驳论据 | 证明对方的事例或数据有误 | 指出对方引用的材料来源有误 |
| 反驳论证 | 承认对方的材料，但指出推论过程有误 | "即使X是真的，也推不出Y的结论" |

### 典型驳论文：《不求甚解》（马南邨）

《不求甚解》（九下第四单元）是典型的驳论文：

**被驳对象**：有人认为"不求甚解"是读书不认真、敷衍了事的坏习惯

**马南邨的驳论**：
1. 追溯"不求甚解"的来源（陶渊明《五柳先生传》）
2. 指出陶渊明的原意：不拘泥于字句的字面意思，而是把握整体精神
3. 用列宁的读书方法（圈注、批注）说明精读不是唯一正确的读书法
4. 结论：读书应该先整体把握，不拘于字句，才是"不求甚解"的真义

这是**反驳论点**的方式：通过重新解释概念，证明对方对"不求甚解"的理解本来就错了。

## 🎓 给 12+（进阶版）

### "先破后立"的论证逻辑

驳论文的逻辑顺序通常是：
1. **破**（驳斥）：你的观点是错的，原因是……
2. **立**（建构）：正确的观点应该是……
3. **合**（总结）：因此……

但好的驳论文不只是"破坏"对方，而是在破的同时建立起更有说服力的观点——破与立相辅相成。

### 归谬法（反证法）

驳论文常用**归谬法**：
- 暂时承认对方的观点为真
- 按对方的逻辑推导下去
- 推导出荒谬的结论
- 因为结论荒谬，所以前提（对方的观点）是错的

例：《鱼我所欲也》中孟子也用了类似思路：
> "如使人之所欲莫甚于生，则凡可以得生者何不用也？"
（如果只追求活着，那么一切保命的手段都可以用——这太荒谬了，所以人确实有比生更重要的东西）

### 驳论文的语言要求

1. **针对性**：必须准确引出要批驳的观点（不能批一个稻草人）
2. **逻辑性**：反驳必须基于逻辑，而非情绪
3. **克制性**：好的驳论不是攻击对方，而是就事论事
4. **建设性**：驳完之后要有自己的观点，不只是否定

### 驳论文写作要点

| 步骤 | 注意 |
|------|------|
| 引出对方观点 | 准确转述，不夸大也不缩小 |
| 选择驳法 | 驳论点/驳论据/驳论证，选最有力的切入 |
| 建立己见 | 有自己的正面主张，避免纯否定 |
| 结语 | 总结，点明意义 |
