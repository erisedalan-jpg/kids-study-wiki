# 出师表

> **一句话**：诸葛亮在北伐前写给后主刘禅的这篇表文，追忆先帝刘备知遇之恩，陈述治国建议（亲贤臣远小人），表达"鞠躬尽瘁，死而后已"的忠心，是古代"表"文体的最高典范，也是历代感人至深的忠义之文。
> **English**: Written to Emperor Liu Shan before the northern expedition, Zhuge Liang recalls the late Emperor's patronage, advises on governance (favor the worthy, distance the flatterers), and declares his utter devotion — the supreme example of the memorial genre and a deeply moving expression of loyal service.

---

## 📖 课文简介

**作者**：诸葛亮（181—234），字孔明，号卧龙，三国时期蜀汉丞相，中国历史上著名的政治家、军事家。
**体裁**：表（臣子向君主陈述请求或意见的文体）
**背景**：建兴五年（227），诸葛亮率蜀汉军队北伐曹魏之前，上此表给后主刘禅，既陈述了治国建议，也表达了对先帝（刘备）的感恩与完成遗志的决心。
**学段**：统编版九年级下册第六单元

## 📚 给 6-12 岁（自读版）

### 全文结构

| 部分 | 段落 | 内容 | 核心建议/情感 |
|------|------|------|--------------|
| 治国建议 | 1-5段 | 广开言路、亲贤臣远小人、推荐人才 | 三条政治建议 |
| 追忆往事 | 6-7段 | 诸葛亮回忆自己从隐居到出仕的经历 | 感恩先帝知遇之恩 |
| 表明心志 | 8-9段 | 表达北伐决心与报恩之志 | 鞠躬尽瘁，死而后已 |

### 原文重点段落

**三条建议（第1-3段核心）**：
> 宫中府中，俱为一体，陟罚臧否，不宜异同。
> 亲贤臣，远小人，此先汉所以兴隆也；亲小人，远贤臣，此后汉所以倾颓也。

**追忆先帝（第6段）**：
> 臣本布衣，躬耕于南阳，苟全性命于乱世，不求闻达于诸侯。先帝不以臣卑鄙，猥自枉屈，三顾臣于草庐之中，咨臣以当世之事，由是感激，遂许先帝以驱驰。

**表明心志（第8段）**：
> 今当远离，临表涕零，不知所言。

**后续名句**（《后出师表》）：
> 鞠躬尽力，死而后已。（后引申为"鞠躬尽瘁，死而后已"）

### 重要注释

| 词语 | 注释 |
|------|------|
| 表 | 古代臣子向君主陈情的文体 |
| 崩殂 | 皇帝去世（崩：天子死；殂：去世） |
| 陟罚臧否 | 奖惩好坏（陟：升赏；罚：惩罚；臧：好；否：坏） |
| 宫中府中 | 皇宫与丞相府，代指朝廷内外 |
| 俱为一体 | 都是一个整体（法律标准应统一） |
| 布衣 | 平民 |
| 躬耕 | 亲自耕种（诸葛亮在南阳隐居，以躬耕为生） |
| 卑鄙 | 身份低微，出身寒微（古义与今义不同） |
| 猥自枉屈 | 屈尊俯就（猥：谦辞，您委屈自己） |
| 三顾 | 三顾茅庐，刘备三次拜访诸葛亮 |
| 咨 | 询问，征求意见 |
| 由是感激 | 因此深受感动 |
| 驱驰 | 奔走效劳 |
| 涕零 | 流泪 |
| 临表 | 面对这篇表文 |

### 白话译文（要点）

**追忆段**：我本来是一个平民，在南阳亲自耕种，只求在乱世中保全性命，不求在诸侯间出名。先帝（刘备）不因我出身低微而嫌弃我，屈尊降贵，三次亲自到草庐拜访我，询问我当时天下的大事，因此我深受感动，答应先帝为他奔走效劳。

**三条建议**（简述）：
1. 广开言路，赏罚统一，不偏不倚
2. 宫中（侍从）、府中（官员）都要任用好人，处理问题标准一致
3. 亲贤臣远小人（汉朝兴盛是因亲贤臣，衰亡是因亲小人）

## 🎓 给 12+（进阶版）

### "表"的文体特点

"表"是中国古代的一种重要文体：
- **性质**：臣子向君主陈述请求、意见或心志的公文
- **与奏折的区别**："奏"一般用于具体事务的请示；"表"多用于重要的陈情、感谢或请求
- **语气**：措辞恭敬，情感真挚，逻辑清晰

《出师表》被誉为"表"文体的千古典范，因为它既有政治建议（理性），又有深情回忆（情感），两者浑然一体。

### 古今词义变化

"臣本布衣……先帝不以臣**卑鄙**"——这里的"卑鄙"是古义：
- **古义**：身份低微（卑：低贱；鄙：粗野，出身于边远之地）
- **今义**：品质低劣，不道德

这是古汉语"词义演变"的典型案例，阅读古文时需注意古今异义。

### 诸葛亮的情感逻辑

《出师表》的情感结构并非线性，而是三层叠加：

1. **理性层**（治国建议）：广开言路、亲贤臣远小人——作为丞相的职责
2. **情感层**（感恩先帝）：三顾茅庐，临危受命——感激知遇之恩
3. **使命层**（完成遗志）：北伐中原，报先帝，忠陛下——把感恩转化为行动

这三层的递进，使文章具有强烈的说服力与感染力。

### 重要名句与成语来源

| 名句 | 出处 | 成语 |
|------|------|------|
| 三顾臣于草庐之中 | 《出师表》 | 三顾茅庐 |
| 临表涕零，不知所言 | 《出师表》 | 感人至深 |
| 鞠躬尽瘁，死而后已 | 《后出师表》（相传） | 鞠躬尽瘁 |
| 亲贤臣，远小人 | 《出师表》 | 后世常用 |

### 历史评价

陆游《书愤》有句："出师一表真名世，千载谁堪伯仲间。"
文天祥《正气歌》："或为出师表，鬼神泣壮烈。"
