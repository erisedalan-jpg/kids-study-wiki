# 鱼我所欲也

> **一句话**：孟子以"鱼与熊掌不可兼得"为比喻，论述当"生"与"义"不能两全时，应当"舍生而取义"——这一论证方法与结论，成为儒家义利观的经典表述。
> **English**: Mencius uses the famous analogy of fish versus bear's paw to argue that when life and righteousness cannot coexist, one should sacrifice life for righteousness — a foundational statement of Confucian ethics.

---

## 📖 课文简介

**来源**：《孟子·告子上》，儒家经典，记录孟子的言论与思想。
**作者**：孟子（约前 372—前 289），名轲，字子舆，战国时期儒家代表人物，有"亚圣"之称。
**体裁**：议论性散文（先秦散文）
**论点**：人的本性中有"义"，当生死抉择面前，应以"义"为重，不能为苟且偷生而做出违背道义的事。
**学段**：统编版九年级下册第三单元

## 📚 给 6-12 岁（自读版）

### 原文（节选）

鱼，我所欲也；熊掌，亦我所欲也，二者不可得兼，舍鱼而取熊掌者也。生，亦我所欲也；义，亦我所欲也，二者不可得兼，舍生而取义者也。

生亦我所欲，所欲有甚于生者，故不为苟得也；死亦我所恶，所恶有甚于死者，故患有所不辟也。

如使人之所欲莫甚于生，则凡可以得生者何不用也？使人之所恶莫甚于死者，则凡可以辟患者何不为也？由是则生而有不用也，由是则可以辟患而有不为也。是故所欲有甚于生者，所恶有甚于死者。非独贤者有是心也，人皆有之，贤者能勿丧耳。

一箪食，一豆羹，得之则生，弗得则死。呼尔而与之，行道之人弗受；蹴尔而与之，乞人不屑也。

万钟则不辩礼义而受之，万钟于我何加焉！……此之谓失其本心。

### 重要注释

| 词语 | 注释 |
|------|------|
| 欲 | 想要，喜欢 |
| 兼 | 同时得到 |
| 甚于 | 超过，比……更重要 |
| 苟得 | 苟且求得（不择手段地求生存） |
| 辟 | 同"避"，躲避 |
| 患 | 祸患，灾难 |
| 箪 | 古代盛饭的竹器 |
| 豆 | 古代盛食物的器皿 |
| 呼尔 | 大声叱喝（不尊重地） |
| 蹴 | 用脚踢 |
| 不屑 | 不愿意接受，认为不值得 |
| 万钟 | 极厚的俸禄（钟：古代容量单位，万钟代表很多粮食/财富） |
| 辩 | 同"辨"，辨别 |
| 本心 | 本来的善良之心，良知 |

### 白话译文（各段）

**第一段（类比论证）**：鱼是我想要的，熊掌也是我想要的；两样不能同时得到，就舍弃鱼而选择熊掌。生命是我想要的，道义也是我想要的；两样不能同时得到，就舍弃生命而选择道义。

**第二段（假设论证）**：生命是我想要的，但有比生命更重要的东西，所以不会苟且求生；死亡是我厌恶的，但有比死亡更可怕的东西，所以有的祸患不会躲避。如果人没有比生命更重视的东西，那么一切可以保命的手段为什么不用呢？由此可见，有时可以保命的方法，却不去用；有时可以躲避祸患，却不去做——这就说明确实有比生命更重要的东西、有比死亡更可怕的东西。这种心不是只有圣贤才有，人人都有，只是圣贤不会丢失它罢了。

**第三段（举例论证）**：一筐饭、一碗汤，得到就能活命，得不到就会饿死。但是用大声叱喝的方式给人，路人也不接受；用脚踢过去，乞丐也不屑于接受。

**结论**：（反之，）有人对于极高的俸禄却不问礼义就接受——这丰厚的俸禄对我有什么好处呢？……这就叫做失去了本来的良心（本心）。

### 论证结构

| 步骤 | 方法 | 内容 |
|------|------|------|
| 1 | 类比 | 鱼 vs 熊掌 → 生 vs 义（引入类比） |
| 2 | 正面论证 | 有比生更欲、比死更恶的东西（义） |
| 3 | 假设反驳 | 如果只追求生存，则无所不为——不合理 |
| 4 | 举例正面 | 乞人不受嗟来之食（有尊严，有义） |
| 5 | 举例反面 | 有人为万钟而丢失本心（失义） |
| 6 | 结论 | 不能失去"本心"（良知） |

## 🎓 给 12+（进阶版）

### 论证方法分析

孟子在这篇文章中综合运用了三种论证方法：

**1. 比喻论证（类比论证）**
"鱼与熊掌不可兼得"——以日常生活中人们做选择的经验，引出"生与义不可兼得"的论题，化抽象为具体，易于接受。

**2. 假设论证（反驳论证）**
"如使人之所欲莫甚于生……则凡可以得生者何不用也？"
——假设对方的观点（只追求生存），推导其荒谬结论，从而反证"有比生更重要的东西（义）"。

**3. 举例论证**
- 正例：乞人不受蹴来之食（有义）
- 反例：有人为万钟不辩礼义（失义）
通过正反两个具体例子，让抽象论点落地。

### "本心"的含义

孟子的"本心"概念与他的性善论密切相关：
- 孟子认为人天生有"善端"（仁、义、礼、智的萌芽）
- "本心"即这种天生的善良之心、良知
- 文章结论"此之谓失其本心"——为了私利丢失了天生的良善
- 这一概念与现代心理学的"道德直觉"有相通之处

### 先秦散文的议论特色

《孟子》的散文以雄辩著称：
- 善用比喻，形象生动
- 逻辑严密，层层推进
- 语言明快，句式整齐（多用排比、对偶）
- 有战略性的论证结构——先立类比，再正面论证，再举例，最后归结

这种议论风格对后世议论文写作有深远影响。
