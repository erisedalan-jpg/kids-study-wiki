## 🎓 古代作家：杜甫

### 生平时间轴

| 年份 | 事件 |
|------|------|
| 712年 | 生于河南巩县，字子美，自号少陵野老；祖父杜审言为初唐著名诗人 |
| 730年代 | 漫游吴越、齐赵，广泛结交，仗剑高歌，"会当凌绝顶，一览众山小" |
| 736年 | 进士考试落第；此后十年困守长安，应试不中，献赋求仕，生活困顿 |
| 744年 | 洛阳与李白相遇，"醉眠秋共被，携手日同行"，成为中国文学史最著名的友谊 |
| 751年 | 献《三大礼赋》，玄宗赏识，命待制集贤院，仍未获实职 |
| 755年 | 安史之乱爆发；杜甫辗转逃亡，长安沦陷 |
| 756年 | 投奔肃宗行在，被俘于长安，目睹山河破碎，作《春望》 |
| 757年 | 逃出长安，被肃宗授左拾遗；因营救房琯触怒皇帝，外放华州 |
| 759年 | 弃官入蜀，定居成都，筑草堂于浣花溪畔；友人严武多次接济 |
| 760-765年 | 成都草堂时期：生活相对安定，创作《春夜喜雨》《蜀相》《茅屋为秋风所破歌》等 |
| 765年 | 严武去世，杜甫失去依靠，携家出川，漂泊于荆楚 |
| 768-770年 | 辗转湖南，身患多病，贫困潦倒；晚年登岳阳楼，作《登岳阳楼》《登高》 |
| 770年 | 病逝于湘江舟中，享年58岁 |

**人生轨迹**：盛世青年漫游 → 长安困顿求仕（10年）→ 安史之乱颠沛流离 → 成都草堂（相对安定）→ 晚年漂泊湖湘，病逝舟中

---

### 主要作品

#### 按创作时期分类

**漫游时期（青年）**：
- 《望岳》："会当凌绝顶，一览众山小"——充满进取精神的青年意气

**长安困守时期**：
- 《兵车行》：揭露频繁征战对百姓的摧残
- 《丽人行》：讽刺杨氏外戚奢靡

**安史之乱时期**：
- 《春望》："烽火连三月，家书抵万金"——国破家散之痛
- 《月夜》：身陷贼营，思念妻儿
- 《哀江头》：哀悼长安沦陷

**成都草堂时期**：
- 《春夜喜雨》：安定生活的愉悦
- 《茅屋为秋风所破歌》：推己及人，"安得广厦千万间，大庇天下寒士俱欢颜"
- 《蜀相》：崇敬诸葛亮，自比忠贞报国

**漂泊晚年**：
- 《登高》（七律）："万里悲秋常作客，百年多病独登台"——被誉为"古今七言律诗之冠"
- 《登岳阳楼》：登楼望洞庭，忧国忧民，老病孤舟

---

### 文学风格：沉郁顿挫

**"沉郁顿挫"**是杜诗最核心的风格定义（杜甫自语，见《进雕赋表》）：

- **沉郁**：感情深沉厚重，内容充实博大，忧国忧民之情沉甸甸压在每一个字上
- **顿挫**：节奏抑扬顿挫，语言凝练有力，情感波澜起伏而不一泻无余

**具体表现**：

| 特征 | 说明 | 例证 |
|------|------|------|
| 忧国忧民 | 将个人遭遇与国家命运紧密相连 | "烽火连三月，家书抵万金" |
| 现实主义 | 真实记录历史，被称为"诗史" | 《兵车行》如实反映征兵之苦 |
| 格律精严 | 律诗对仗精工，被誉为"诗律严" | 《登高》首联、颔联对仗堪称绝唱 |
| 炼字精准 | 每字经过锤炼，准确有力 | "感时花溅泪，恨别鸟惊心" |
| 情感厚重 | 多重情感叠加，不作单一表达 | 《登高》三重悲愁（漂泊+老病+忧国） |

---

### 文学史地位

**"诗圣"**：与"诗仙"李白并称，李白代表浪漫主义的巅峰，杜甫代表现实主义的巅峰。

**"诗史"**：杜甫的诗真实记录了安史之乱前后唐朝由盛转衰的历史，被后世称为"以诗存史"。

**律诗集大成者**：杜甫将近体诗（尤其五律、七律）的格律技巧发展到最高峰，对后世诗歌影响深远。宋代以后的诗人几乎无不受杜甫影响（"千家注杜"）。

**儒家精神的诗歌体现**：杜甫一生以儒家"仁"的精神为核心，"穷年忧黎元，叹息肠内热"（《自京赴奉先县咏怀五百字》），即使自身穷困潦倒，仍念念不忘天下苍生。

---

### 与李白的比较

| 比较维度 | 杜甫 | 李白 |
|----------|------|------|
| 称号 | 诗圣 | 诗仙 |
| 流派 | 现实主义 | 浪漫主义 |
| 主要关切 | 国家命运、民生疾苦 | 个人理想、自由精神 |
| 风格 | 沉郁顿挫 | 飘逸豪放 |
| 擅长体裁 | 律诗（尤其七律） | 古风、绝句 |
| 核心意象 | 战争、流离、老病、秋 | 月、酒、剑、仙 |
| 思想底色 | 儒家（忧国忧民） | 道家+游侠（自由超脱） |

**相同点**：同为唐诗巅峰，同经历安史之乱的冲击；744年洛阳相遇，惺惺相惜；均对后世诗歌产生不可估量的影响。

### 核心诗作深析

#### 《登高》（七律）

> 风急天高猿啸哀，渚清沙白鸟飞回。
> 无边落木萧萧下，不尽长江滚滚来。
> 万里悲秋常作客，百年多病独登台。
> 艰难苦恨繁霜鬓，潦倒新停浊酒杯。

- **背景**：766年，杜甫55岁，在夔州（今重庆奉节）重阳登高
- **结构**：首联、颔联写景（意象叠加营造意境）；颈联、尾联抒情（三重悲愁点明）
- **炼字**："常""独"写漂泊之孤；"万""百"极写时空之广；"繁""停"写老病之苦
- **意境**：苍茫悲壮，沉郁顿挫之极
- **地位**：被誉为"古今七言律诗之冠"（明代胡应麟语）

#### 《茅屋为秋风所破歌》

- **情感升华**：由个人遭遇（茅屋被风吹破）升华为对天下寒士的悲悯
- **核心句**："安得广厦千万间，大庇天下寒士俱欢颜！风雨不动安如山。呜呼！何时眼前突兀见此屋，吾庐独破受冻死亦足！"
- **精神境界**：以自身受冻换天下安暖——儒家"仁"的精神的最高表达
