## 🎓 古代作家：苏轼

### 生平时间轴

| 年份 | 事件 |
|------|------|
| 1037年 | 生于四川眉山，字子瞻，号东坡居士 |
| 1057年 | 与弟苏辙同年进士，欧阳修称其文章"出人意表" |
| 1059-1066年 | 初入仕途，父丧守制，返蜀；母丧，与父苏洵同葬 |
| 1071年 | 因反对王安石新法，请求外调，任杭州通判 |
| 1079年 | "乌台诗案"——因诗文被政敌构陷，下狱103天，险些赐死 |
| 1080年 | 贬谪黄州（今湖北黄冈），号"东坡居士"由此而来；创作《赤壁赋》《后赤壁赋》《念奴娇·赤壁怀古》 |
| 1086年 | 旧党执政，苏轼回朝，任翰林学士；因不满全面废新法，再度被贬 |
| 1094年 | 贬惠州（今广东），年近六十仍泰然处之 |
| 1097年 | 再贬儋州（今海南），当时被视为"天涯海角"绝地 |
| 1100年 | 宋徽宗即位，遇赦北还 |
| 1101年 | 卒于常州，享年65岁 |

**一生贬谪路**：京城 → 杭州 → 密州 → 徐州 → 湖州 → **黄州**（最重要）→ 汝州 → 回京 → **惠州** → **儋州**（最远）→ 北还途中辞世

### 文学风格

#### 词：开创豪放派

苏轼之前，词以"婉约"为正宗（柳永、晏殊、欧阳修），多写儿女情长、离别之苦。苏轼以"以诗为词"的革命性变革，将词的题材扩展到：
- 怀古（《念奴娇·赤壁怀古》）
- 说理（《水调歌头》）
- 田园（《浣溪沙·簌簌衣巾落枣花》）
- 豪情（《江城子·密州出猎》）

**豪放词的特征**：
- 意境开阔，气象宏大
- 题材广泛，突破儿女情长
- 语言雄健，用典丰富
- 以旷达代替哀愁

#### 散文：自然流畅，以理驭文

- 继承韩愈古文运动传统，文气畅达，不拘一格
- 《赤壁赋》将主客问答的哲学辩论融入山水描写，情、景、理三位一体
- 议论文善于翻案立论，出人意表（《留侯论》论"忍"）

#### 诗：机智幽默，哲理深邃

- 善于从日常景物中提炼哲理（"不识庐山真面目"）
- 幽默旷达，即使身处逆境仍能自得其乐（"日啖荔枝三百颗"）

### 文学史地位

**"三苏"之首**：父苏洵、弟苏辙，合称"三苏"，均列唐宋八大家。

**唐宋八大家**：韩愈、柳宗元（唐）；欧阳修、苏洵、苏轼、苏辙、曾巩、王安石（宋）。苏轼是其中成就最全面者。

**宋词豪放派开创者**：与辛弃疾并称"苏辛"，共同奠定豪放词的美学规范。

**中国历史上最全面的文人之一**：诗、词、散文、书法（宋四家之一：苏、黄、米、蔡）、绘画（文人画理论家）、美食（东坡肉）、工程（杭州苏堤）——跨越文学、艺术、政治、生活多个领域。

**精神遗产**：苏轼的"旷达"不是逃避，而是在充分承受人生苦难之后，依然能够以审美的眼光看待世界的精神高度。林语堂称其为"人间不可无一难能有二"的人物。

---

### 核心思想：旷达哲学的三个来源

| 思想来源 | 体现 |
|----------|------|
| 儒家 | 积极入世，忠君爱民，即使屡遭贬谪仍坚持政治理想 |
| 道家 | 顺应自然，超然物外（"纵一苇之所如，凌万顷之茫然"） |
| 佛家 | 随缘自适，不执着于得失（"竹杖芒鞋轻胜马，谁怕？一蓑烟雨任平生"） |

三种思想在苏轼身上并非矛盾，而是在不同处境下灵活运用，构成其旷达人格的完整底色。
