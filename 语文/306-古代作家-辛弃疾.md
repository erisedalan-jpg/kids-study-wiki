## 🎓 古代作家：辛弃疾

### 生平时间轴

| 年份 | 事件 |
|------|------|
| 1140年 | 生于山东历城（今济南），字幼安，号稼轩；彼时山东已沦陷金国 |
| 1161年 | 21岁，金主完颜亮大举南侵；辛弃疾组织义军抗金，率50骑突入金营，生擒叛徒张安国，押解南宋 |
| 1162年 | 南归南宋，任江阴签判；此后长期在南宋仕途沉浮 |
| 1165年 | 上《美芹十论》，详论北伐方略，朝廷置之不理 |
| 1170年代 | 任湖南安抚使等地方军政要职，政绩卓著；力主抗金，屡受主和派排挤 |
| 1181年 | 42岁，遭弹劾罢官，退居江西上饶带湖，开始长达20年的闲居生涯；号"稼轩"由此得名 |
| 1188年 | 与陈亮鹅湖之会，慷慨论英雄，相互唱和（鹅湖组词） |
| 1194年 | 再度被罢，移居铅山瓢泉 |
| 1203年 | 63岁，韩侂胄主持北伐，召辛弃疾出山，任镇江知府；登北固亭，作《永遇乐·京口北固亭怀古》 |
| 1207年 | 韩侂胄北伐大败；同年辛弃疾病逝，享年67岁，临终仍高呼"杀贼！杀贼！" |

**人生悲剧核心**：从23岁南归到67岁辞世，44年间辛弃疾始终怀抱收复中原的壮志，却因南宋朝廷懦弱、主和派当权，报国无门，壮志成灰。

### 文学风格

#### 豪放词的集大成者

辛弃疾继承并超越苏轼，将豪放词推向最高峰：
- **题材**：军事、政治、历史、田园，无所不包
- **手法**：用典密集（《永遇乐》一词用五典），以文为词，以论为词
- **情感**：悲壮与豪迈并存，家国之恨与壮志之气交织

与苏轼豪放的**区别**：
- 苏轼之豪放：出于人生哲学的旷达，超脱于政治得失之上
- 辛弃疾之豪放：出于军人的激情与家国之痛，悲壮中带有无法释怀的执念

#### 婉约词的另一面

辛弃疾并非只会写豪放词。《青玉案·元夕》《西江月·夜行黄沙道中》等词，婉约细腻，意境清新，展示其词艺的全面性。

#### 语言特色：以文为词，以典为词

- 大量使用散文句法，打破词的音律束缚
- 用典丰富（刘克庄称其"用经用史，煅炼无痕"）
- 语言雄健有力，气势磅礴

### 文学史地位

**"苏辛"并称**：与苏轼共同开创豪放词派，"苏辛"是宋词豪放派的双峰。

**宋词第一量**：620余首词，数量为宋词作家之最，且质量极高，被后世称为"词中之龙"。

**"人中之杰，词中之龙"**（刘克庄语）

**爱国词人的精神标杆**：辛弃疾的词作将个人遭遇与民族命运紧密相连，成为后世爱国主义文学的精神源头。陆游、文天祥等均受其影响。

**文武双全的唯一性**：中国文学史上，能真正上阵杀敌（率骑兵突袭金营）又能写出最伟大词作的，只有辛弃疾。他的词不是书斋文人的想象，而是真正上过战场的军人的呐喊。

### 核心词作深析

#### 《破阵子·为陈同甫赋壮词以寄之》

> 醉里挑灯看剑，梦回吹角连营。八百里分麾下炙，五十弦翻塞外声，沙场秋点兵。
> 马作的卢飞快，弓如霹雳弦惊。了却君王天下事，赢得生前身后名。可怜白发生！

- **结构**：上阕+下阕均为沙场豪情（虚写梦境）→ 末句五字"可怜白发生"（实写现实）
- **艺术效果**：前九句十气贯通，写尽沙场豪情，末句急转，以现实的"白发"击碎梦境，壮志难酬之悲猝然而出，震撼人心
- **情感**：壮志 vs 现实的最强烈对比，是辛词精神的集中体现

#### 《永遇乐·京口北固亭怀古》

- **五典并用**：孙权、刘裕、刘义隆、佛狸祠、廉颇
- **层次**：缅怀英雄（正用）→ 警示轻率北伐（反用）→ 痛心中原沦陷（正用）→ 自比廉颇（正用）
- **核心情感**：既渴望北伐，又忧虑轻率出兵；既壮志犹存，又悲叹年老不被重用
