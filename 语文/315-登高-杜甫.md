## 🎓 登高

### 作者

**杜甫**（712—770），字子美，自号少陵野老，祖籍湖北襄阳，生于河南巩县。唐代伟大的现实主义诗人，与李白并称"李杜"，被后世誉为"诗圣"，其诗被称为"诗史"。杜甫出身官宦世家，早年游历四方，渴望入仕报国。安史之乱爆发后，历经战乱流离，目睹民间疾苦，创作了大量反映社会现实、忧国忧民的诗篇。晚年流寓四川，又辗转湖南，穷困潦倒，于大历五年（770）病逝于湘江舟中，享年五十九岁。著有《杜工部集》，存诗约一千四百余首。

### 创作背景

本诗作于大历二年（767）秋，时杜甫流寓夔州（今重庆奉节），已五十六岁。此时诗人身患肺病、风痹等多种疾病，生活困顿，远离故乡，故交零落，心情极为抑郁。重阳节登高，是中国古代的传统习俗，意在避灾祈福。诗人登高望远，触景生情，一气呵成此诗，充分抒发了暮年多病、漂泊异乡的悲愁。本诗历来被誉为"七律之冠"，明人胡应麟称其为"古今七言律第一"。

### 原文

风急天高猿啸哀，渚清沙白鸟飞回。
无边落木萧萧下，不尽长江滚滚来。
万里悲秋常作客，百年多病独登台。
艰难苦恨繁霜鬓，潦倒新停浊酒杯。

### 注释

| 字词 | 注释 |
|------|------|
| 渚 | 水中的小块陆地，小洲 |
| 飞回 | 盘旋飞翔 |
| 落木 | 落叶 |
| 萧萧 | 草木摇落的声音，此处形容落叶的声音 |
| 滚滚 | 水势汹涌翻腾的样子 |
| 万里 | 指离家万里，漂泊在外 |
| 作客 | 客居他乡 |
| 百年 | 一生，终生 |
| 台 | 高处，诗人登高之处 |
| 艰难 | 指国家艰难、时局困苦 |
| 苦恨 | 极其遗憾，深深痛恨（此处"苦"为副词，意为"极、甚"）|
| 繁霜鬓 | 两鬓如霜，形容白发增多 |
| 潦倒 | 衰颓，此处指身体衰弱、穷困不得志 |
| 新停 | 刚刚停了 |
| 浊酒杯 | 粗劣的酒。因病戒酒，故停饮 |

### 译文

秋风猛烈，天空高远，猿猴的啼叫声声悲哀；水中小洲清冷，白沙苍茫，鸟儿在天空盘旋飞翔。无边无际的落叶萧萧地飘落而下，奔涌不尽的长江滚滚而来。离家万里，年年在这悲凉的秋天独自客居他乡；一生多病，今日独自登上高台。时局艰难，生活苦涩，遗憾两鬓早已霜白；穷困潦倒，身患多病，不得不刚刚停了饮酒。

### 文学赏析

**结构布局**

全诗为标准律诗，共四联八句，结构严整。首联、颔联写景，颈联、尾联抒情。由景入情，情景交融，浑然一体。

- **首联**（风急天高……鸟飞回）：写登高所见所闻，仰观天空，俯察水洲，六个意象（风、天、猿、渚、沙、鸟）各自独立，组成一幅萧瑟秋景图。
- **颔联**（无边落木……滚滚来）：扩大视野，写无边落木与不尽长江，境界宏阔悲壮，象征时间的流逝与生命的短暂。
- **颈联**（万里悲秋……独登台）：直抒胸臆，"万里""百年""悲秋""多病""独"等词叠加，将羁旅之苦、暮年之悲推向高峰。
- **尾联**（艰难苦恨……浊酒杯）：进一步点明悲苦根源——既有个人遭遇，更有国家艰难，以"新停浊酒杯"收束，余味无穷。

**艺术手法**

1. **对仗精工**：全诗四联皆对仗，尤以颔联"无边落木萧萧下，不尽长江滚滚来"为历代称道，对仗工整，气势磅礴。首联更是"句中自对"（"风急"对"天高"，"渚清"对"沙白"），精巧之极。
2. **意象叠加**：首联六个意象密集排列，无一虚设，构成秋景的整体氛围；颈联七个字中含四层悲苦（万里/悲秋/常/作客），密度极高。
3. **情景交融**：眼前的萧萧落木、滚滚长江，既是自然实景，也是诗人内心世界的外化——生命的流逝、功业的未竟、国家的沦丧，皆寄于景中。
4. **叠词运用**：萧萧、滚滚，音韵回环，增强了诗歌的音乐性，也强化了景象的动感与力度。

**思想内涵**

本诗集"登高"这一传统题材之大成，在个人悲秋与忧国忧民的两个维度上同时用力。"万里悲秋常作客，百年多病独登台"——"万里"是空间的孤独，"百年"是时间的沉重，"悲秋"是情感的基调，"多病"是身体的消耗，"独"是精神的孤立，七字含七层愁苦，是杜甫一生悲剧命运的高度凝练。"艰难苦恨繁霜鬓"不仅是个人身世之叹，更是对国家艰难时局的深沉忧虑，体现了杜甫"诗圣"的人道主义情怀与家国担当。

### 高考考点

**必背默写**（高频考点）
- "无边落木萧萧下，不尽长江滚滚来"——描写壮阔秋景、象征生命流逝的名句
- "万里悲秋常作客，百年多病独登台"——抒发羁旅之苦与暮年悲愁的名句
- "艰难苦恨繁霜鬓，潦倒新停浊酒杯"——表达忧国忧民、潦倒困顿的名句

**翻译重点**
- "渚清沙白鸟飞回"：水中小洲清冷，白沙苍茫，鸟儿在空中盘旋
- "万里悲秋常作客"：离家万里，年年在悲秋时节客居他乡
- "艰难苦恨繁霜鬓"：时局艰难，生活苦涩，极为遗憾两鬓早白

**鉴赏要点**
- 诗歌体裁：七言律诗，被誉为"古今七律第一"
- 主题：登高抒怀，抒发羁旅之苦、暮年多病之悲与忧国忧民之情
- 艺术特色：对仗精工、意象密集、情景交融、音韵铿锵
- 炼字分析："急""哀""清""白"各有所指，精确传神；"常""独"二字尤为沉重
- 比较阅读：与王维《九月九日忆山东兄弟》同写重阳登高，但杜甫重在悲秋忧国，王维重在思念亲人
