# 词牌-高中

> **一句话**：词牌是填词时所依据的乐曲曲调名称，规定了该词的字数、句式、平仄、押韵格式，是词的"格律模板"。
> **English**: A ci pai (词牌) is the name of a musical tune that dictates the metrical pattern — number of characters, line structure, tonal pattern, and rhyme scheme — for composing a ci poem.

---

## 🎓 词牌详解

### 定义

**词牌**（又称词调）是词的曲调名称，规定了词的：
- **字数**：全词总字数固定
- **句式**：每句字数及断句位置固定
- **平仄**：每字的声调（平声/仄声）格式固定
- **押韵**：韵脚位置和用韵规则固定

词最初是配合音乐演唱的歌词，词牌即乐曲名；后来音乐失传，词牌仅作格律规范使用，词的内容与词牌名称不必相关（如《沁园春·雪》的内容与"沁园"无必然联系）。

### 词的基本结构

| 术语 | 含义 |
|------|------|
| 片（阕） | 词的段落，分上片（上阕）和下片（下阕） |
| 令（小令） | 字数较少（58字以内）的词 |
| 慢词（长调） | 字数较多（91字以上）的词，节奏舒缓 |
| 中调 | 59—90字之间 |
| 过片 | 下片的开头，承接上片转入新意 |
| 叠韵 | 同一韵部反复押韵 |

### 《沁园春·长沙》格律详解（必修上重点）

**词牌**：沁园春　**作者**：毛泽东　**字数**：114字

**上片**（46字）：
```
独立寒秋，（仄仄平平）
湘江北去，（平平仄仄）
橘子洲头。（仄仄平平）韵①
看万山红遍，（仄仄平仄仄）
层林尽染；（平平仄仄）
漫江碧透，（仄平仄仄）
百舸争流。（仄仄平平）韵①
鹰击长空，（平仄平平）
鱼翔浅底，（平平仄仄）
万类霜天竞自由。（仄仄平平仄仄平）韵①
怅寥廓，（仄平仄）
问苍茫大地，（仄平仄仄仄）
谁主沉浮？（平仄平平）韵①
```

**下片**（68字）：承上片情感，转入追忆峥嵘岁月。

**格律要点**：
1. 押仄声韵（ou韵部）：头、流、由、浮
2. 上片结句"谁主沉浮"呼应全篇主旨
3. 领字（看/恰/忆）领起多个意象，是沁园春词牌的典型写法

### 词牌的"正体"与"变体"

同一词牌可有多个版本（正体和若干变体），字数和格式略有差异：
- 填词须注明所用版本
- 高考鉴赏题只需识别词牌，不考手动填词
- 判断词牌依据：字数 + 上下片划分 + 韵脚位置

### 词与诗的主要区别

| 比较项 | 诗（近体诗） | 词 |
|--------|------------|-----|
| 句式 | 句句等长（五言/七言） | 句式长短不齐（长短句） |
| 字数 | 八句（绝句四句） | 依词牌固定，各异 |
| 押韵 | 偶句押韵，韵脚少 | 押韵更密，有换韵 |
| 平仄 | 格律相对宽松（有出律情况） | 格律严格，逐字规定 |
| 分段 | 一般不分段 | 分片（阕） |
| 与音乐关系 | 古体诗可入乐，近体诗格律化后渐脱离 | 本为配乐歌词，后成独立文体 |

### 高考考点

**词牌鉴赏题常见提问**：
1. 这首词的词牌是什么？体现了怎样的格律特点？
2. 这首词上下片各写了什么内容？转折（过片）处的关系是什么？
3. 词的某处用韵有何艺术效果？

**答题思路**：
- 识别词牌 → 了解格律框架
- 分析上下片内容结构
- 结合格律特点分析情感节奏
