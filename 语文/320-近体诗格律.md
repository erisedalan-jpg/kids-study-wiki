# 近体诗格律

> **一句话**：近体诗（格律诗）是唐代形成的有严格平仄、押韵、对仗规定的诗歌体裁，包括律诗和绝句，是高中古诗鉴赏的核心格律知识。
> **English**: Regulated verse (近体诗/格律诗) is a form of Chinese poetry codified in the Tang Dynasty, with strict rules governing tonal patterns (平仄), rhyme (押韵), and antithesis (对仗).

---

## 🎓 近体诗格律详解

### 定义

**近体诗**（又称格律诗）是相对于古体诗而言，在唐代定型的严格遵守格律规范的诗歌形式。其规范包括：
1. **字数固定**：五言或七言
2. **句数固定**：绝句四句，律诗八句，排律八句以上
3. **平仄规律**：每句每字平仄有规定
4. **押韵规定**：押平声韵，一韵到底
5. **对仗要求**：律诗颔联（3-4句）和颈联（5-6句）须对仗

### 近体诗的基本类型

| 类型 | 句数 | 每句字数 | 押韵规则 | 对仗要求 |
|------|------|---------|---------|---------|
| 五言绝句（五绝） | 4句 | 5字 | 偶句末字押韵 | 无强制要求 |
| 七言绝句（七绝） | 4句 | 7字 | 偶句末字押韵 | 无强制要求 |
| 五言律诗（五律） | 8句 | 5字 | 偶句末字押韵 | 颔联+颈联须对仗 |
| 七言律诗（七律） | 8句 | 7字 | 偶句末字押韵 | 颔联+颈联须对仗 |
| 排律 | 10句以上 | 5或7字 | 同律诗 | 除首尾联外均须对仗 |

### 平仄基础

**四声分平仄**：
- **平声**：现代普通话一声（阴平）、二声（阳平）→ 古代平声
- **仄声**：现代普通话三声（上声）、四声（去声）→ 古代上、去声；另有入声（现代已消失，分入四声）

> 注意：古今声调有差异，部分字的古今归类不同（如"白""国""学"古为入声字），高中阶段以理解规律为主，不要求精确背诵入声字表。

**基本平仄格式（五律为例）**：

五律有四种基本句式：
```
（甲）仄仄平平仄
（乙）平平仄仄平（韵）
（丙）平平平仄仄
（丁）仄仄仄平平（韵）
```

**首联（1-2句）→ 颔联（3-4句）→ 颈联（5-6句）→ 尾联（7-8句）**

### 押韵规则

| 规则 | 说明 |
|------|------|
| 押平声韵 | 近体诗一般押平声韵（上去入声可押古体诗） |
| 一韵到底 | 全诗用同一个韵部，不换韵 |
| 韵脚位置 | 偶句末字押韵；首句可押可不押（七言通常押） |
| 邻韵通押 | 近体诗严格，一般不允许邻韵通押（古体诗可以） |

**高中教材常见押韵示例**：

| 诗作 | 韵脚字 | 韵部 |
|------|--------|------|
| 《登高》杜甫 | 哀、回、来、台、杯 | 灰韵（ai/ei） |
| 《琵琶行》白居易 | 鸣、声、倾、城、情、生 | 庚韵（ing/eng） |
| 《锦瑟》李商隐 | 弦、年、珠、烟、然 | 先韵（ian/an，待家长核对） |

### 对仗规则

**律诗颔联和颈联必须对仗**，对仗要求：
1. **词性对应**：名词对名词，动词对动词，形容词对形容词
2. **结构对应**：句法结构相同或对称
3. **平仄相对**：上句仄处，下句平；上句平处，下句仄
4. **语义关联**：通常含义相关或相对（不要求语义完全相反）

**经典对仗示例**：

| 上句 | 下句 | 对仗类型 | 出处 |
|------|------|---------|------|
| 无边落木萧萧下 | 不尽长江滚滚来 | 工对（词性结构完全对应） | 杜甫《登高》 |
| 烽火连三月 | 家书抵万金 | 工对 | 杜甫《春望》 |
| 晓镜但愁云鬓改 | 夜吟应觉月光寒 | 宽对 | 李商隐《无题》 |
| 星垂平野阔 | 月涌大江流 | 工对 | 杜甫《旅夜书怀》 |

### 常见格律术语

| 术语 | 含义 |
|------|------|
| 粘 | 相邻两联，上联下句与下联上句的第二字平仄相同（粘连） |
| 对 | 同一联中，上句与下句第二字平仄相反 |
| 拗句 | 不符合标准格律的句子 |
| 救 | 在拗句后用另一位置的平仄调整来补救 |
| 孤平 | 五言句"仄平仄仄仄"类型，是近体诗大忌 |
| 三平尾 | 句末连续三个平声，近体诗避免 |

### 识别方法（高考考点）

**鉴赏格律考点**：
1. 判断诗歌体裁（近体诗/古体诗）
2. 分析押韵和韵脚（找偶句末字）
3. 分析对仗（找颔联颈联，判断词性结构）
4. 分析格律对情感表达的作用

**答题框架**：
`本诗为……（体裁），押……韵，韵脚为……，音韵……（和谐/铿锵/婉转），颈联"……对……"构成……（工对/宽对），对仗工整，烘托了……情感。`

### 易错辨析

**近体诗 vs 古体诗**
- 近体诗：字数、句数、平仄、押韵均有严格规定（唐代定型）
- 古体诗：句数不限，字数可混用，押韵较自由，可换韵
- 判断：先看字数是否整齐 → 看是否押平声韵且一韵到底 → 看是否有对仗

**律诗 vs 绝句**
- 律诗：8句，有颔联颈联对仗要求
- 绝句：4句，通常认为是律诗的"截取"，无强制对仗要求

**词 vs 近体诗**
- 词：句式长短不齐，依词牌格律，可换韵，有上下片
- 近体诗：句式整齐，不换韵，平仄规律相对固定
