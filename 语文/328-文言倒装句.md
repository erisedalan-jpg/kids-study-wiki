## 🎓 文言倒装句

### 定义

倒装句（变式句）是指古汉语中句子成分的排列顺序与现代汉语常规语序不同的句式。古汉语的"倒装"是以现代汉语的语序为参照系来命名的，实际上在先秦时代这些语序均有其语法规律。倒装句分为四类：主谓倒装、宾语前置、定语后置、状语后置（介词短语后置）。

---

### 类型分类表

#### 一、主谓倒装（谓语前置）

| 说明 | 古文例句 | 现代汉语对照 |
|------|----------|--------------|
| 谓语提前，用于感叹、疑问或强调 | 甚矣，汝之不惠！（《愚公移山》） | 你太不聪明了！ |
| 同上 | 美哉，我少年中国（梁启超） | 我少年中国，多么美好！ |

#### 二、宾语前置

| 类型 | 结构特征 | 古文例句 | 现代汉语对照 |
|------|----------|----------|--------------|
| 否定句中代词宾语前置 | 否定词 + 代词宾语 + 动词 | 古之人不余欺也（《石钟山记》） | 古人没有欺骗我 |
| 疑问句中疑问代词宾语前置 | 疑问代词宾语 + 动词 | 大王来何操？（《鸿门宴》） | 大王来时带了什么礼物？ |
| "之"字标志宾语前置 | 宾语 + 之 + 动词 | 句读之不知，惑之不解（《师说》） | 不懂得句读，不能解决疑惑 |
| "是"字标志宾语前置 | 宾语 + 是 + 动词 | 唯利是图（成语） | 只图利益 |
| "之为"标志宾语前置 | 宾语 + 之为 + 表语 | 何陋之有？（《陋室铭》） | 有什么简陋的呢？ |

#### 三、定语后置

| 类型 | 结构特征 | 古文例句 | 现代汉语对照 |
|------|----------|----------|--------------|
| "者"字标志 | 中心词 + 定语 + 者 | 求人可使报秦者（《廉颇蔺相如列传》） | 寻找可以出使回复秦国的人 |
| "之……者"式 | 中心词 + 之 + 定语 + 者 | 石之铿然有声者（《石钟山记》） | 敲击时发出铿锵声音的石头 |
| "之"字标志 | 中心词 + 之 + 定语 | 蚓无爪牙之利（《劝学》） | 蚯蚓没有锋利的爪牙 |
| 数量词后置 | 中心词 + 数量定语 | 马之千里者（《马说》） | 能日行千里的马 |

#### 四、状语后置（介词短语后置）

| 类型 | 结构特征 | 古文例句 | 现代汉语对照 |
|------|----------|----------|--------------|
| "于"字介词短语后置 | 动词 + 于 + 名词 | 青，取之于蓝（《劝学》） | 靛青，从蓼蓝中提取 |
| "以"字介词短语后置 | 动词 + 以 + 名词 | 申之以孝悌之义（《寡人之于国也》） | 用孝悌之义反复教导百姓 |
| "乎"字介词短语后置 | 动词 + 乎 + 名词（"乎"通"于"） | 君子博学而日参省乎己（《劝学》） | 每天对自己进行反省 |

### 高频例句表

| 古文例句 | 倒装类型 | 现代汉语翻译 |
|----------|----------|--------------|
| 何以战？（《曹刿论战》） | 宾语前置（疑问代词） | 凭借什么作战？ |
| 时不我待（成语） | 宾语前置（否定句代词） | 时间不等待我们 |
| 微斯人，吾谁与归？（《岳阳楼记》） | 宾语前置（疑问代词） | 我和谁一道回去呢？ |
| 蚓无爪牙之利，筋骨之强（《劝学》） | 定语后置（"之"字） | 蚯蚓没有锋利的爪牙、强健的筋骨 |
| 居庙堂之高则忧其民（《岳阳楼记》） | 定语后置 | 处于高高的庙堂之上就忧虑百姓 |
| 会于会稽山阴之兰亭（《兰亭集序》） | 状语后置（于） | 在会稽山阴的兰亭相聚 |
| 不拘于时，学于余（《师说》） | 状语后置（于） | 不被时俗拘束，向我学习 |
| 甚矣，汝之不惠（《愚公移山》） | 主谓倒装 | 你太不聪明了！ |

### 识别方法（高考考点）

1. **宾语前置三判断**：
   - 是否有否定词（不、未、莫、无）+ 代词（之、余、我、尔）
   - 是否为疑问句 + 疑问代词（何、谁、安、孰、恶）作宾语
   - 是否有"之""是"作提宾标志
2. **定语后置三判断**：
   - 中心词后是否有"者"
   - 是否有"中心词 + 之 + 定语（形容词/动词短语）"结构
   - 数量词是否位于中心词之后
3. **状语后置判断**：介词短语（以/于/乎+名词）若紧跟动词之后，通常为后置状语。
4. **翻译还原法**：将倒装成分调回现代汉语正常语序后，语义更通顺，则确认为倒装。

### 易错辨析

| 易错点 | 例析 | 正确解读 |
|--------|------|----------|
| "何……之有"与"之"字宾前混淆 | 何陋之有 | "之"为助词，将"陋"（宾语）提前，"有何陋"之倒装 |
| 定语后置与一般定中结构混淆 | 蚓无爪牙之利 vs 君子之学 | 前者定语后置（爪牙锋利），后者正常定中（君子的学问） |
| 状语后置与补语混淆 | 战于长勺 vs 高于蓝 | 均为介词短语后置，"于"引出处所或比较对象 |
| 主谓倒装识别困难 | 甚矣 | 形容词谓语提前，后接主语，须靠语感和叹词判断 |
