## 🎓 沁园春·长沙

### 作者与写作背景

毛泽东（1893-1976），湖南湘潭人，政治家、诗人。本词作于1925年秋，当时作者离开长沙前往广州主持农民运动讲习所，途经橘子洲，面对湘江秋景，追忆往昔峥嵘岁月，抒发革命青年的壮志豪情。

- **词牌**：沁园春，双调114字，上阕13句、下阕12句，全押仄声韵
- **题目**：长沙，湖南省会，作者青年时求学、革命活动的重要地点
- **时代背景**：1925年，国共合作时期，大革命高潮前夕

### 原文

**上阕**

> 独立寒秋，湘江北去，橘子洲头。
> 看万山红遍，层林尽染；漫江碧透，百舸争流。
> 鹰击长空，鱼翔浅底，万类霜天竞自由。
> 怅寥廓，问苍茫大地，谁主沉浮？

**下阕**

> 携来百侣曾游，忆往昔峥嵘岁月稠。
> 恰同学少年，风华正茂；书生意气，挥斥方遒。
> 指点江山，激扬文字，粪土当年万户侯。
> 曾记否，到中流击水，浪遏飞舟？

### 注释表

| 词语 | 注释 |
|------|------|
| 寒秋 | 深秋，寒意渐浓之秋 |
| 橘子洲 | 长沙湘江中的沙洲，盛产橘子 |
| 百舸 | 许多船只，舸：大船 |
| 怅寥廓 | 面对广阔天地而感慨，怅：惆怅、感慨 |
| 谁主沉浮 | 谁来主宰国家命运，沉浮：比喻盛衰兴替 |
| 峥嵘岁月 | 不平凡的岁月，峥嵘：形容山势高峻，引申为不平凡 |
| 书生意气 | 青年学生的朝气与胆识 |
| 挥斥方遒 | 奔放有力，遒：强劲有力 |
| 粪土当年万户侯 | 把封建权贵视为粪土，表示蔑视 |
| 中流击水 | 在湍急的水流中奋力搏击 |
| 浪遏飞舟 | 激起的浪花阻挡了飞速行驶的船只 |

### 结构分析

| 层次 | 内容 | 情感 |
|------|------|------|
| 上阕写景 | 独立橘子洲，仰观俯察，描绘湘江秋景 | 壮阔、生机勃勃 |
| 上阕发问 | "谁主沉浮" | 忧国忧民，豪迈自信 |
| 下阕忆昔 | 回忆同学少年战斗岁月 | 激情澎湃 |
| 下阕抒志 | "到中流击水" | 以中流击水自比，表达革命豪情 |

### 主题思想

词以橘子洲秋景为背景，上阕写景设问，下阕忆旧抒怀，借景抒情，表达了青年革命者以天下为己任、敢于主宰国家命运的豪情壮志。全词气势磅礴，意境阔大。

### 艺术特色

1. **意象群落**：万山、层林、漫江、百舸、雄鹰、游鱼——构成一幅色彩鲜明、生机盎然的秋景图
2. **远近结合**：远看万山红遍，近看漫江碧透，仰视鹰击长空，俯视鱼翔浅底
3. **以景写志**：万类霜天竞自由→革命者当主宰国家命运
4. **对比手法**：秋景的生机勃勃 vs. 当时黑暗的社会现实；"万户侯"的腐朽 vs. 革命青年的朝气
5. **用典**："中流击水"源自晋代祖逖击楫渡江的典故，此处化用为革命豪情的象征

### 词牌格律

沁园春（双调，114字）：
- 上阕：13句，四平韵，以仄声收
- 下阕：12句，同一韵部
- 本词押仄声韵：秋/洲/流/由/浮（上阕）；稠/侯/舟（下阕）

### 高考考查重点

- 词的意象分析（万山红遍、层林尽染的意象作用）
- "问苍茫大地，谁主沉浮"的含义与作用
- 上下阕的结构关系（写景→抒情，现在→回忆）
- 炼字题：如"染""击""翔"等动词的表现力
- 与其他豪放词（苏轼《念奴娇》等）比较异同
