# 芣苢-诗经

> **一句话**：《诗经·周南》篇章，描绘古代妇女采集芣苢（车前草）的劳动场景，重章叠唱，节奏明快，是先秦劳动诗代表。
> **English**: "Fu-yi" from Shijing, Zhou-nan, depicts ancient women picking plantain in a rhythmic, repetitive verse celebrating labor.

---

## 🎓 高中进阶

### 出处

- 选自《诗经·国风·周南》
- 《诗经》：中国最早的诗歌总集，收西周初至春秋中叶诗 305 篇
- "国风"：各诸侯国民间诗，反映底层生活

### 全文（共 12 句）

```
采采芣苢，薄言采之。
采采芣苢，薄言有之。

采采芣苢，薄言掇之。
采采芣苢，薄言捋之。

采采芣苢，薄言袺之。
采采芣苢，薄言襭之。
```

### 字词与节奏

| 字 | 拼音 | 释义 |
|----|------|------|
| 采采 | cǎi cǎi | 茂盛貌；又作动作叠用 |
| 芣苢 | fú yǐ | 即车前草，一种野菜药草 |
| 薄言 | bó yán | 发语词，无实义 |
| 采 | cǎi | 采摘 |
| 有 | yǒu | 取得 |
| 掇 | duō | 拾取 |
| 捋 | luō | 顺手抹取 |
| 袺 | jié | 提起衣襟兜物 |
| 襭 | xié | 把衣襟掖在腰带上盛物 |

### 章法结构

**重章叠唱**——每章两句，仅换一动词：

| 章 | 动词组 | 表达层次 |
|----|------|--------|
| 1 | 采、有 | 开始采摘、有所收获 |
| 2 | 掇、捋 | 拾取、抹取（采摘动作多样） |
| 3 | 袺、襭 | 兜起、束起（满载而归） |

六个动词层层递进——动作越来越多、收获越来越满。

### 艺术手法

| 手法 | 说明 |
|------|------|
| 重章叠唱 | 每章相同结构，便于口传 |
| 赋的手法 | 直接铺陈劳动场面 |
| 节奏明快 | 四言句 + 重复 → 易诵易记 |
| 动作细描 | 六个动词刻画完整劳动过程 |
| 集体劳动情景 | 妇女群体采摘的画面 |

### 主题

- **劳动美学**：表现劳动的喜悦
- **集体协作**：妇女集体劳动场面
- **自然朴素**：贴近土地的生活情趣

### 《诗经》六义

| 风、雅、颂 | 类型 |
| 赋、比、兴 | 手法 |

| 项 | 含义 |
|----|------|
| 风 | 民歌（国风） |
| 雅 | 宫廷诗（大雅、小雅） |
| 颂 | 宗庙祭祀诗 |
| 赋 | 直陈其事 |
| 比 | 比喻 |
| 兴 | 先言他物以引起所咏之词 |

《芣苢》主要用赋的手法。

### 文化意义

- 反映先秦农耕社会生活
- 体现劳动光荣的价值观（高中"劳动光荣"单元主题）
- 中国诗歌"劳动诗"传统的源头
- 重章叠唱是后世民歌通用手法

### 易错点

- "芣苢"读 fú yǐ（不是 pī yǐ）
- "薄言"是发语词，无实义；不要硬翻译
- 六个动词不可混淆顺序（动作递进）
- 是民歌不是文人诗——质朴是其特点
