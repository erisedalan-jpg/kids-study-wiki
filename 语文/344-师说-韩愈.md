# 师说-韩愈

> **一句话**：唐代韩愈所作论说文，论述从师学习的必要性，批驳当时社会"耻于从师"的风气，是古代议论文的典范。
> **English**: Han Yu's "On Teachers" (Tang dynasty) argues for the importance of learning from teachers and criticizes the contemporary disdain for seeking instruction.

---

## 🎓 高中进阶

### 作者与背景

- 作者：韩愈（768-824），字退之，唐代古文运动领袖，"唐宋八大家"之一
- 时代：唐德宗贞元十八年（802）
- 写作目的：作《师说》赠李蟠（其学生），借此反对当时士人耻于从师的风气
- 文体：**论说文**（议论文）

### 中心论点

> **师者，所以传道受业解惑也。**

教师的三大职责：传道（道理）、授业（学业）、解惑（疑难）。

### 核心思想

**学必有师，从师不分贵贱长幼：**

> 是故无贵无贱，无长无少，道之所存，师之所存也。

**择师标准：**

> 闻道有先后，术业有专攻。

谁懂道理在先，谁就是老师；不必拘泥年龄、地位。

### 论证结构

| 段落 | 论点 | 论证方法 |
|------|------|--------|
| 一 | 古之学者必有师 | 立论：教师的职责 |
| 二 | 师道之不传也久矣 | 对比：古今、士大夫 vs 巫医百工、其子 vs 其身 |
| 三 | 圣人无常师 | 举例：孔子师郯子、苌弘、师襄、老聃 |
| 四 | 述写作缘由 | 写给李蟠 |

### 主要对比

韩愈用大量对比论证当时风气之坏：

| 对比 | 内容 |
|------|------|
| 古今 | 古之学者必有师 vs 今之众人耻师 |
| 父子 | 爱其子择师而教 vs 于其身则耻师 |
| 阶层 | 巫医乐师百工之人不耻相师 vs 士大夫耻师 |

### 重点文言知识

| 字词 | 释义 |
|------|------|
| 师 (作动词) | 学习、效法 |
| 道 | 道理、儒家之道 |
| 受 | 通"授" |
| 惑 | 疑难 |
| 之 | 的；助词；代词；取消句子独立性 |
| 而 | 表顺接、转折 |
| 庸 | 哪里、岂 |
| 嗟乎 | 感叹词 |
| 攻 | 研究、专攻 |

### 名句

> 古之学者必有师。师者，所以传道受业解惑也。
> 是故弟子不必不如师，师不必贤于弟子。
> 闻道有先后，术业有专攻。
> 孔子曰：三人行，则必有我师。

### 艺术特色

1. **论点鲜明**：开门见山立中心论点
2. **论证有力**：对比论证、举例论证、引用论证（孔子）
3. **气势磅礴**：排比、反问，韩愈"文起八代之衰"风格
4. **逻辑严密**：层层推进，先破后立
5. **古文运动**：摆脱六朝骈文束缚，恢复先秦散文传统

### 古文运动与韩愈

| 项目 | 内容 |
|------|------|
| 主张 | 文以载道、反对骈文、恢复古文（先秦两汉散文） |
| 领袖 | 韩愈、柳宗元 |
| 影响 | 改变唐代文风，影响宋代古文运动（欧阳修、苏轼） |
| "唐宋八大家" | 韩愈、柳宗元、欧阳修、苏洵、苏轼、苏辙、王安石、曾巩 |

### 现代意义

- 终身学习理念
- 师生平等、师不必贤于弟子
- 反对学历崇拜、年龄崇拜
- 求知精神

### 易错点

- "师"在文中有多种词性：名词（老师）、动词（学习/从师）、意动用法（以...为师）
- "受"通"授"，是教与受的关系
- "庸"作疑问词，不是平庸
- 韩愈作《师说》是为反对耻师风气，不是单纯论师道
