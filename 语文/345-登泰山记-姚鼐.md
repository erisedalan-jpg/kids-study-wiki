# 登泰山记-姚鼐

> **一句话**：清代姚鼐于乾隆三十九年（1774）冬登泰山观日出后所作的游记散文，以简洁笔法记述行程与日出胜景。
> **English**: Yao Nai's "A Record of Climbing Mount Tai" (1774) is a Qing-dynasty travel essay narrating his ascent and the famous sunrise.

---

## 🎓 高中进阶

### 作者与背景

- 作者：姚鼐（1731-1815），字姬传，桐城派代表
- 时代：清乾隆三十九年（1774）末
- 背景：作者由京师辞官南归，途经泰山
- 流派：**桐城派**散文（讲究"义理、考据、辞章"，文风简洁）

### 全文结构

| 段落 | 内容 |
|------|------|
| 一 | 总写泰山方位、形势：南阴北阳，交界南北的山势 |
| 二 | 记冬月与朱子颖登山经过：路线、台阶、日暮抵达 |
| 三 | 登日观峰看日出：候日 → 日出 → 红光晕染万物 |
| 四 | 记泰山名胜：寺庙、石刻、亭台 |
| 五 | 记离泰山时间、与同游者 |

### 名段精读

**总写形势：**
> 泰山之阳，汶水西流；其阴，济水东流。阳谷皆入汶，阴谷皆入济，当其南北分者，古长城也。

**写日出：**
> 戊申晦，五鼓，与子颖坐日观亭，待日出。大风扬积雪击面。亭东自足下皆云漫。
> 极天云一线异色，须臾成五采。日上，正赤如丹，下有红光动摇承之。

——画面感、节奏紧凑、逐渐递进。

### 重点字词

| 字词 | 释义 |
|------|------|
| 阳 / 阴 | 山南为阳，山北为阴；水北为阳，水南为阴 |
| 当 | 在；处于 |
| 五鼓 | 五更，约凌晨 4 点 |
| 戊申晦 | 干支日期；月末 |
| 须臾 | 一会儿 |
| 五采 | 五彩 |
| 正赤如丹 | 大红如丹砂 |
| 樗 (chū) | 樗树 |
| 拾级 | 一级一级踏 |

### 艺术特色

1. **桐城派简洁笔法**：用字精准，无冗笔
2. **写景层次分明**：方位 → 行程 → 日出 → 山色
3. **动静结合**：行进的动 + 待日的静
4. **时空交织**：从天色变化看时间流动
5. **白描手法**：少抒情多记叙

### 桐城派特点

| 特点 | 说明 |
|------|------|
| 义理 | 文章思想内核 |
| 考据 | 学问根据 |
| 辞章 | 语言形式 |
| 文以载道 | 写作目的 |
| 简洁雅正 | 文风 |

### 与同时期游记对比

| 篇目 | 朝代 | 风格 |
|------|------|------|
| 《赤壁赋》 | 宋·苏轼 | 抒情议论交融 |
| 《石钟山记》 | 宋·苏轼 | 实地考察+议论 |
| 《游褒禅山记》 | 宋·王安石 | 借游说理 |
| 《登泰山记》 | 清·姚鼐 | 简洁白描记叙 |

### 文化意涵

- 泰山为"五岳之首"，自古为帝王封禅之地
- 登泰山被视为人格修养的象征
- "会当凌绝顶，一览众山小"（杜甫《望岳》）的精神延续
- 日观峰看日出至今为泰山最著名景观

### 易错点

- "阳"指山南/水北，不是"阳光"
- "晦"是月末（不是晦暗）；"朔"是月初
- 五鼓 = 五更 ≈ 凌晨 4 点（不是 5 点）
- 桐城派代表："方苞、姚鼐、刘大櫆" 三家
