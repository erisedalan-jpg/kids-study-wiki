## 🎓 古诗词鉴赏方法

### 定义

古诗词鉴赏是通过对诗歌的语言、意象、意境、表现手法、情感主旨的系统分析，把握诗人借诗歌传达的思想情感与艺术追求的阅读活动。高考古诗词鉴赏要求考生既能"读懂"（理解大意），又能"读深"（分析艺术特色），更能"读出"（体悟情感主旨）。

---

### 鉴赏维度

#### 一、读懂诗歌——整体把握

**五步读诗法**：
1. **看标题**：时间、地点、事件、情感线索（如《春望》→ 春天 + 望 → 望什么？）
2. **看作者**：朝代、生平经历、惯常风格（杜甫→忧国忧民；李白→飘逸豪放）
3. **看注释**：提供背景信息，往往是解题关键
4. **看意象**：诗中名词（物象）= 情感的载体
5. **看末句/尾联**：往往是情感总结或主旨所在

#### 二、分析意象与意境

**常见意象及其文化含义**：

| 意象 | 常见情感 | 经典例句 |
|------|----------|----------|
| 月 | 思乡、思人、孤寂 | 杜甫"露从今夜白，月是故乡明" |
| 菊 | 高洁、隐逸 | 陶渊明"采菊东篱下，悠然见南山" |
| 梅 | 坚贞、傲骨 | 王安石"墙角数枝梅，凌寒独自开" |
| 柳 | 离别、留恋 | "杨柳岸，晓风残月"（柳永） |
| 雁 | 思乡、音信 | 韦应物"寒雨连江夜入吴，平明送客楚山孤" |
| 落花 | 伤春、青春易逝 | 晏殊"无可奈何花落去，似曾相识燕归来" |
| 长亭 | 送别 | 李白《送友人》"浮云游子意，落日故人情" |

**意境分析四步法**：
1. 描绘画面（用自己的语言还原诗句情境）
2. 概括意境特点（两词：清幽、苍茫、雄浑、悲凉……）
3. 分析情感（意境如何承载情感）
4. 指出意象与情感的对应关系

> **例**：杜甫《登高》"风急天高猿啸哀，渚清沙白鸟飞回"——急风、高天、哀猿、清渚、白沙、归鸟六个意象叠加，营造出苍凉悲壮的意境，与诗人晚年漂泊、忧国忧民的沉郁情感高度契合。

#### 三、辨识表现手法

**抒情手法**：

| 手法 | 说明 | 例句 |
|------|------|------|
| 直接抒情 | 直白表达情感 | 陈子昂"念天地之悠悠，独怆然而涕下" |
| 借景抒情 | 通过景物描写含蓄抒情 | 马致远《天净沙·秋思》枯藤老树烘托游子之悲 |
| 情景交融 | 景中有情，情中有景 | 杜甫"感时花溅泪，恨别鸟惊心" |
| 托物言志 | 借物象寄托志向 | 于谦《石灰吟》以石灰自喻清白之志 |
| 用典 | 引用历史故事或前人诗句 | 辛弃疾《永遇乐》大量用典寄托壮志难酬 |

**描写手法**：
- 动静结合："蝉噪林逾静，鸟鸣山更幽"（以声衬静）
- 虚实结合：眼前实景 + 想象/回忆（虚）
- 白描：不加渲染，简笔勾勒
- 细节描写：以小见大

**修辞手法**：比喻、拟人、对比、夸张、反问、设问

#### 四、分析语言特色

**炼字题答题步骤**：
1. 解释该字在句中的含义
2. 还原诗句描绘的画面
3. 分析该字对意境/情感的贡献

> **例**："春风又绿江南岸"中"绿"字：将名词活用为动词，化无形为有形，写出春风吹过、江南一片葱绿的动态景象，生机盎然，隐含诗人对故乡的思念与对变法的期待。

**语言风格**：
- 豪放（苏轼、辛弃疾）
- 婉约（柳永、李清照）
- 沉郁顿挫（杜甫）
- 清新淡雅（王维、孟浩然）
- 雄浑壮阔（王昌龄、高适）

#### 五、把握情感主旨

**常见情感类型**：

| 情感 | 常见背景 | 代表诗人 |
|------|----------|----------|
| 忧国忧民 | 战乱、政治腐败 | 杜甫、范仲淹 |
| 壮志难酬 | 仕途不顺、报国无门 | 辛弃疾、陆游 |
| 羁旅思乡 | 漂泊在外 | 王维、李白、杜甫 |
| 送别惜别 | 朋友离别 | 王勃、李白、王昌龄 |
| 归隐田园 | 厌倦官场 | 陶渊明、王维 |
| 伤春悲秋 | 年华易逝 | 李煜、晏殊、柳永 |

### 古代经典例文分析

#### 《登高》（杜甫，唐代）

- **意境**：苍凉悲壮（六意象叠加）
- **情感**：漂泊之苦 + 老病之叹 + 忧国之悲（三重悲愁）
- **炼字**："万里悲秋常作客，百年多病独登台"——"万""百""常""独"数词与副词的叠用，极写愁苦之深
- **结构**：首联写景，颔联写景（对仗精工），颈联抒情，尾联点主题

#### 《念奴娇·赤壁怀古》（苏轼，北宋）

- **手法**：虚实结合（实：眼前赤壁；虚：赤壁之战想象）、用典（周瑜）
- **情感**：对英雄业绩的仰慕 + 自身功业未成的感慨 → 以"一尊还酹江月"作旷达收束
- **风格**：豪放词的代表，气象宏大，境界开阔

#### 《声声慢》（李清照，南宋）

- **意象**：淡酒、晚风、大雁、黄花、梧桐、细雨——叠加渲染愁绪
- **语言**：叠词开篇（"寻寻觅觅，冷冷清清，凄凄惨惨戚戚"）模拟心理颤抖
- **情感**：国破家亡、夫死流离之悲，女性视角的家国之痛

### 高考题型

| 题型 | 典型问法 | 答题思路 |
|------|----------|----------|
| 炼字 | "某字历来被赞为'诗眼'，请赏析" | 释义 + 画面 + 效果（意境/情感） |
| 意境 | "诗歌营造了怎样的意境" | 两词概括 + 意象分析 + 情感联系 |
| 手法 | "这首诗运用了哪些表现手法" | 点明手法 + 举例 + 效果 |
| 情感 | "表达了诗人怎样的思想感情" | 情感分层 + 结合诗句 + 背景联系 |
| 比较 | "与××诗相比，本诗在××方面有何异同" | 同 + 异，分点作答 |
