# 古代作家·司马迁

> **一句话**：司马迁（约前 145—前 90 年），西汉史学家、文学家，创作中国第一部纪传体通史《史记》，以"究天人之际，通古今之变，成一家之言"为著史理想，遭李陵之祸受宫刑后以著述为生命支撑。

---

## 🎓 高中精读版

### 生平时间轴

| 时期 | 大事 |
|------|------|
| 约前 145 年 | 生于左冯翊夏阳（今陕西韩城附近） |
| 少年 | 随父司马谈研习古籍，20 岁游历全国，实地考察历史遗迹 |
| 前 110 年 | 父司马谈去世，临终嘱托继续著史，司马迁接任太史令 |
| 前 104 年 | 开始系统整理史料，着手撰写《史记》 |
| 前 99 年 | 李陵事件：将军李陵投降匈奴，司马迁为其辩护，触怒汉武帝，被判死罪 |
| 前 98 年 | 以宫刑（腐刑）赎死，在屈辱中坚持完成《史记》 |
| 约前 90 年 | 《史记》基本完成后去世，时年约 55 岁 |

### 李陵事件的意义

这是理解司马迁最重要的背景：
- 李陵以五千步兵孤军深入，与匈奴主力激战，力尽投降，并非贪生怕死。
- 司马迁在朝廷一片声讨中独为李陵辩护，被判极刑。
- 按汉律，死罪可以金赎或以宫刑代替——司马迁选择了宫刑，以极大屈辱换取继续完成《史记》的机会。

**《报任安书》**（写给好友任安的信，也是司马迁的精神自传）：
> "人固有一死，或重于泰山，或轻于鸿毛，用之所趋异也。……仆诚以著此书，藏之名山，传之其人，通邑大都，则仆偿前辱之责，虽万被戮，岂有悔哉！"

这段话道出了司马迁"忍辱负重、以书证道"的精神核心，是中国文人士大夫精神最重要的文献之一。

### 《史记》概况

| 项目 | 内容 |
|------|------|
| 体例 | 纪传体（中国第一部纪传体通史） |
| 记载范围 | 上起黄帝，下至汉武帝，跨越约 3000 年 |
| 篇数 | 130 篇（本纪 12 + 表 10 + 书 8 + 世家 30 + 列传 70） |
| 字数 | 约 52 万字 |
| 别称 | 《太史公书》 |
| 地位 | "二十四史之首"；鲁迅称"史家之绝唱，无韵之离骚" |

### 著史理念

**"实录"精神**：
- "不虚美，不隐恶"——对汉代君臣亦如实书写，包括汉武帝的过失（"酷吏列传"等）。
- 这在当时是极大的政治勇气，班固评"其文直，其事核，不虚美，不隐恶，故谓之实录"。

**"究天人之际，通古今之变"**：
- 天人之际：探讨历史成败与天道、人事的关系——成功者是否有必然规律？
- 古今之变：通过纵贯三千年的历史，找到历史演变的规律。

**"成一家之言"**：
- 太史公曰（篇末评论）是司马迁自己的历史观与价值判断，不因袭前人。
- 对项羽：给予"本纪"待遇（帝王级别），尽管项羽非皇帝，体现司马迁的独立判断。
- 对屈原：在无诸侯支持的情况下为其立传，认可其精神价值。

### 文学风格

1. **叙事简洁而传神**：鸿门宴短短数千字，场景、人物、转折一气呵成。
2. **对话个性化**：每个人物的台词都符合其性格，绝不雷同。
3. **悲剧意识强烈**：《史记》中最感人的传记多为悲剧——屈原、项羽、李广……司马迁以自身遭遇的悲剧感浸润其笔。
4. **人道主义关怀**：为刺客（荆轲）、游侠（郭解）、医者（扁鹊）立传，认可其在官方史观之外的价值。

### 文学史地位

- **史学**：《史记》奠定中国纪传体史学的基本范式，后世二十四史均沿用。
- **文学**：被视为古代散文的典范，韩愈、柳宗元均学习《史记》的叙事艺术。
- **精神遗产**："发愤著书"的司马迁传统，影响了历代困境中的知识分子——屈原之后，司马迁树立了"以文字对抗苦难"的典型。
