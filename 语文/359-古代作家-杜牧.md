# 古代作家·杜牧

> **一句话**：杜牧（803—852），字牧之，晚唐著名诗人散文家，与李商隐并称"小李杜"，诗风俊爽清健，咏史诗成就尤高，散文代表作《阿房宫赋》是古代赋体的顶峰之作。

### 诗歌成就

**1. 咏史诗（最高成就）**

杜牧咏史诗的特点是以小见大、翻案出奇，不走歌功颂德的平庸之路，以历史为镜，折射现实政治。

**代表作精讲**：

《赤壁》：
> 折戟沉沙铁未销，自将磨洗认前朝。东风不与周郎便，铜雀春深锁二乔。

- 翻案手法："若非东风相助，周瑜未必胜"——以假设否定定论，提出历史的偶然性。
- 结尾以"二乔"（孙权之妻、周瑜之妻）代指吴国命运，以小女子代大历史，以私代公，含蓄深婉。

《过华清宫》（其一）：
> 长安回望绣成堆，山顶千门次第开。一骑红尘妃子笑，无人知是荔枝来。

- 以"荔枝快递"这一细节，揭示唐玄宗为杨贵妃穷奢极欲、不顾民力的昏政，讽刺入骨而不露声色。

《泊秦淮》：
> 烟笼寒水月笼沙，夜泊秦淮近酒家。商女不知亡国恨，隔江犹唱后庭花。

- 以歌女唱《玉树后庭花》（陈后主亡国之曲），讽刺晚唐统治者沉迷声色不思振作，借古讽今的典范。

《江南春》：
> 千里莺啼绿映红，水村山郭酒旗风。南朝四百八十寺，多少楼台烟雨中。

- 写江南春景，末两句转入历史沉思，南朝大兴佛寺却仍亡国——山河永恒，历史沧桑。

**2. 写景抒情诗**

《山行》：
> 远上寒山石径斜，白云深处有人家。停车坐爱枫林晚，霜叶红于二月花。

- 以"霜叶红于二月花"翻案：一般人认为秋景萧瑟不如春光，杜牧认为晚秋枫叶之红胜过春花——乐观豁达的晚秋美学。
- 此诗格调明朗健朗，是杜牧诗风中"俊爽"一面的代表。

**3. 送别赠友诗**

《赠别》二首、《遣怀》等扬州时期作品，风流俊逸，多情而不滥情，与李商隐的隐晦缠绵形成鲜明对比。

### 散文成就

**《阿房宫赋》**是杜牧 24 岁所作，以秦始皇修建阿房宫并迅速亡国为题，劝谏唐朝统治者引以为鉴。

**结构**：
- 前两段：铺陈阿房宫的宏大与奢侈（"覆压三百余里，隔离天日"）——借汉赋大铺排传统
- 后两段：转入议论，以"秦人不暇自哀而后人哀之"收结，警示当朝

**名句**：
- "六王毕，四海一，蜀山兀，阿房出。"——四个三字句，节奏铿锵，以高度提炼的语言呈现历史进程。
- "秦人不暇自哀，而后人哀之；后人哀之而不鉴之，亦使后人而复哀后人也。"——三组"哀"字层层递进，将历史警示推向无尽的未来。

### 诗歌风格总结

| 特征 | 说明 |
|------|------|
| 俊爽清健 | 不同于李商隐的绮丽隐晦，杜牧诗语言明朗，意象清晰 |
| 翻案出奇 | 咏史诗常持与通论相反的观点，以逆向思维发掘历史的复杂性 |
| 讽喻含蓄 | 批评时政不直接说，借历史典故、细节意象迂回达意 |
| 情感节制 | 有情而不泛滥，多情而有分寸，"情到深处轻轻写" |

### 文学史地位

- 与李商隐并称"小李杜"（李杜指李白杜甫），是晚唐诗歌的双峰。
- 咏史诗是杜牧对中国诗歌的独特贡献，影响后世宋代咏史词（苏轼《念奴娇·赤壁怀古》即继承此传统）。
- 《阿房宫赋》是唐代赋体的最高成就，后世文赋（宋代苏轼等）的先驱。
