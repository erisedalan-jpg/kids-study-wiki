# 念奴娇·过洞庭

> **一句话**：张孝祥夜渡洞庭，以皓月澄波的壮阔之景托出光明磊落的君子人格，是宋词中以自然境界写人格境界的经典之作。

---

## 🎓 念奴娇·过洞庭

### 作者

**张孝祥**（1132—1170），字安国，号于湖居士，历阳乌江（今安徽和县）人，南宋著名词人，书法家。绍兴二十四年（1154年）状元及第，曾力主抗金，与岳飞案相关。其词豪放清旷，继苏轼之后、开辛弃疾之先，在南宋豪放词发展史上有承上启下之地位。黄蓼园称其词"骨气峻整，飘逸绝伦"。

### 创作背景

宋孝宗乾道二年（1166年），张孝祥被贬谪，北归途经洞庭湖，在中秋前后夜渡时写下此词。洞庭湖烟波浩渺，中秋月色澄明，词人将个人的磊落胸怀与自然的壮阔景象融为一体，写出了一种超越现实困境的精神境界。

### 原文

```
念奴娇·过洞庭
张孝祥

洞庭青草，近中秋、更无一点风色。
玉界琼田三万顷，著我扁舟一叶。
素月分辉，明河共影，表里俱澄澈。
怡然心会，妙处难与君说。

应念岭海经年，孤光自照，肝胆皆冰雪。
短发萧骚襟袖冷，稳泛沧浪空阔。
尽挹西江，细斟北斗，万象为宾客。
扣舷独啸，不知今夕何夕。
```

### 注释

| 词语 | 注释 |
|------|------|
| 洞庭青草 | 洞庭湖与青草湖相连，泛指洞庭湖一带 |
| 风色 | 风的迹象；此处指没有一丝风 |
| 玉界琼田 | 如玉如琼的水面；形容湖面洁白晶莹 |
| 著我扁舟一叶 | 容纳着我一叶扁舟；著，容纳 |
| 素月 | 洁白的月亮 |
| 明河 | 银河 |
| 表里俱澄澈 | 内外都清澈透明，指水天、内心皆澄明 |
| 怡然心会 | 愉快地领悟到这种妙境 |
| 岭海经年 | 在岭南度过的一年；岭海，岭南沿海之地，指贬谪之处 |
| 孤光自照 | 孤独的月光自我映照；也暗指自我审视、问心无愧 |
| 肝胆皆冰雪 | 内心如冰雪般纯洁光明，比喻胸怀坦荡 |
| 萧骚 | 头发稀疏；形容年迈或忧愁使头发稀少 |
| 沧浪空阔 | 苍茫广阔的水面 |
| 尽挹西江 | 尽情舀取西来的江水；挹（yì），舀 |
| 细斟北斗 | 用北斗星座当酒杯慢慢斟酒 |
| 万象为宾客 | 天地万物都是我的宾客 |
| 扣舷独啸 | 敲击船边，独自长啸 |
| 不知今夕何夕 | 化用《诗经·绸缪》，陶醉其中不知今是何夕 |

### 译文

洞庭湖与青草湖相连，临近中秋，湖面没有一丝风的痕迹。如玉如琼的三万顷湖面，容纳着我的一叶扁舟。洁白的月亮分散着光辉，银河的倒影与湖面共融，天上水下内外都透明澄澈。心中愉快地领悟到这番妙境，其中妙处难以与你言说。

想起在岭南度过的那一年，月亮孤独地自我映照，我的肝胆都如冰雪般纯净坦荡。短发稀疏，衣袖在夜风中生凉，却稳稳地泛舟在苍茫空阔的湖面上。尽情舀取西来的江水，用北斗当酒杯慢慢斟酌，让天地万物都成为我的宾客。敲击着船舷独自长啸，已不知今晚是哪一夜了。

### 文学赏析

**"表里俱澄澈"——以景喻人的核心意境**

此词最大的艺术成就在于以自然境界写人格境界的高度融合。上阕写景：洞庭秋夜，月色如洗，湖面无风，水天合一。"表里俱澄澈"一句是全词的核心，字面写湖水与天空内外澄明，实则是词人内心世界的写照——经历贬谪而心无芥蒂，坦荡如此湖面。

**豪放而非壮烈**

与苏轼《念奴娇·赤壁怀古》的历史厚重不同，此词豪放中透着清旷与宁静：无历史典故，无英雄感慨，只有纯粹的自然与纯粹的自我。"尽挹西江，细斟北斗，万象为宾客"——气魄恢宏，以天地为酒宴，以万象为宾朋，展现了一种超越世俗得失的自由精神。

**月意象的深化**

"孤光自照"中的月既是自然之月，也是内心之镜。词人以月自喻，月之孤独、月之清明，正是词人贬谪境遇与高洁人格的象征。"肝胆皆冰雪"用冰雪的洁白透明比喻内心的纯净，与"表里俱澄澈"形成呼应，共同构建了一个内外澄明的君子形象。

**结句的意趣**

"不知今夕何夕"化用《诗经》，却去掉原来的爱情语境，转为一种物我两忘的审美陶醉。词人在洞庭之夜的壮阔美景中，忘却了时间，忘却了贬谪，这是一种精神的自由，也是中国文人在逆境中寻求超脱的典型方式。

**与苏轼赤壁词的比较**

同为《念奴娇》，苏词以历史英雄为参照，感叹"人生如梦"；张词则以自然澄澈为映照，呈现"肝胆冰雪"。前者在历史中寻找慰藉，后者在自然中确认自我，代表了宋代豪放词的两种不同向度。

### 高考考点

1. **意境分析**：分析"玉界琼田三万顷，著我扁舟一叶"的意境——广阔与渺小的对比，以渺小衬托壮阔，以壮阔烘托胸怀。
2. **炼字**："著"字的妙用——"容纳"之义，扁舟被洞庭湖如此轻松地容纳，写出湖之广阔，也暗含词人融于自然的自在感。
3. **手法：以景喻人**："表里俱澄澈"如何同时写景与写人，分析这一手法的表达效果。
4. **主旨**：词中"肝胆皆冰雪"表达的人格理想——坦荡磊落、问心无愧，即使身处逆境也不改初心。
5. **比较阅读**：与苏轼《念奴娇·赤壁怀古》比较，分析两词在情感取向、写作视角上的异同。
6. **典故运用**："不知今夕何夕"的出处与作用。
