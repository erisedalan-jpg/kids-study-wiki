# 桂枝香·金陵怀古

> **一句话**：王安石登临金陵，以壮阔秋景起笔，借六朝兴废感叹后人终究"门前冷落车马稀"，是宋词中怀古悲慨的扛鼎之作。

---

## 🎓 桂枝香·金陵怀古

### 作者

**王安石**（1021—1086），字介甫，号半山，抚州临川（今江西）人，北宋文学家、政治家、改革家，"唐宋八大家"之一。其词存世不多，却以气骨峭拔、意境开阔著称，《桂枝香·金陵怀古》被后世誉为宋词怀古词的极致之作。王安石诗文兼擅，诗以"半山体"闻名，散文雄健严密，词风豪迈而不失沉郁。

### 创作背景

此词约作于宋神宗熙宁年间（1068—1077），王安石推行新法期间，或出守金陵（今南京）时所作。金陵为六朝古都，历经东吴、东晋、宋、齐、梁、陈，盛衰兴亡俱在此地。词人登高远眺，壮阔的长江秋色与六朝残迹交织，激发了深沉的历史感慨。

### 原文

```
桂枝香·金陵怀古
王安石

登临送目，正故国晚秋，天气初肃。
千里澄江似练，翠峰如簇。
归帆去棹残阳里，背西风酒旗斜矗。
彩舟云淡，星河鹭起，画图难足。

念往昔，繁华竞逐，叹门外楼头，悲恨相续。
千古凭高对此，谩嗟荣辱。
六朝旧事随流水，但寒烟衰草凝绿。
至今商女，时时犹唱，后庭遗曲。
```

### 译文

登上高处极目远眺，正值故都金陵的深秋，天气刚刚显出清肃之气。千里澄清的长江如同白绢，碧翠的山峰如花束簇拥。归帆和离棹在夕阳余晖里穿行，西风中酒旗斜斜地挺立。彩舟在淡淡云层中漂浮，白鹭从银河般的江面飞起，这般美景，画图都难以画尽。

回想往昔，六朝君主竞相追逐繁华，可叹门外敌军已至、楼头美人犹在，悲恨一代接着一代。千古以来，多少人凭高远眺面对这江山，不过是徒然感叹兴衰。六朝旧事已随流水逝去，留下的只有冷烟和凝绿的衰草。直到如今，歌女们还时时传唱着那亡国的《后庭花》遗曲。

### 文学赏析

**结构：上阕写景，下阕怀古**

上阕以"登临送目"开门见山，全景式展开金陵秋景：澄江、翠峰、归帆、残阳、酒旗、彩舟、白鹭，境界开阔，笔力雄劲。"画图难足"一句总收，以赞叹之语暗示眼前之景触动了更深的情思。

下阕转入历史，以"念往昔"领起，"繁华竞逐"四字概括六朝君主之误。"门外楼头"用杜牧典故，以极简的语言唤起陈朝灭亡的历史场景，含蓄而有力。结尾化用杜牧《泊秦淮》"商女不知亡国恨，隔江犹唱后庭花"，但去掉"不知"而改为"时时犹唱"，将批判的锋芒指向更广泛的社会，深沉而悲凉。

**艺术特色**

- **豪放与沉郁并融**：上阕壮阔，下阕沉郁，两者统一于对金陵山河的深情凝视之中。王国维称此词"足以跨越苏、辛"，苏轼读后叹"此老乃野狐精也"。
- **意象对比**：千里澄江的美景与"寒烟衰草"的荒凉并置，自然之美与历史之悲形成强烈张力。
- **用典精当**："门外楼头"典、"后庭遗曲"典，言简意丰，以最少的文字承载最大的历史容量。
- **视角转换**：由登高所见（空间）到历史追忆（时间），由六朝旧事到当下商女，时空交叠，层层深入。

### 高考考点

1. **炼字**："澄江似练"之"练"字——以白绢比喻江水，写出江水清澈平静、白亮绵长的特点（五上炼字法则）。
2. **用典**："门外楼头""后庭遗曲"两处典故的含义及其在词中的作用——借历史讽喻、以古鉴今。
3. **对比手法**：上阕壮美之景与下阕萧条之境的对比，分析其情感表达效果。
4. **比较阅读**：与杜牧《泊秦淮》对比，分析两位诗人怀古情感的异同（直接批判 vs 沉郁叹息）。
5. **词牌与体裁**：《桂枝香》属慢词，双调，本词属豪放词派，但兼具婉约沉郁之气，与苏轼豪放词风格比较。
6. **主旨概括**：借金陵秋景引发对六朝兴亡的历史沉思，表达对繁华易逝、历史循环的深沉感慨。
