# 登岳阳楼

> **一句话**：杜甫暮年登临岳阳楼，以洞庭浩瀚之景衬出自身的孤苦飘零，末联将个人身世之悲升华为对国家动荡的忧思，沉郁顿挫，感人至深。

---

## 🎓 登岳阳楼

### 作者

**杜甫**（712—770），字子美，自号少陵野老，祖籍湖北襄阳，生于河南巩县，唐代最伟大的现实主义诗人，后世称"诗圣"，与李白并称"李杜"。杜甫一生颠沛流离，仕途坎坷，经历安史之乱，晚年漂泊西南。其诗沉郁顿挫，忧国忧民，被誉为"诗史"。

### 创作背景

大历三年（768年），杜甫携家乘舟出峡，辗转漂泊，时年五十七岁，已是老病缠身。此年冬，杜甫乘舟路过岳阳，登上久闻大名的岳阳楼，写下这首千古名作。彼时安史之乱虽已平定，但战乱后遗症深重，吐蕃入侵，国家仍不安定。诗人一生夙愿登临岳阳楼，却在暮年以病躯实现，身世之感与家国之忧交织，成就了此诗的深沉格调。

### 原文

```
登岳阳楼
杜甫

昔闻洞庭水，今上岳阳楼。
吴楚东南坼，乾坤日夜浮。
亲朋无一字，老病有孤舟。
戎马关山北，凭轩涕泗流。
```

### 注释

| 词语 | 注释 |
|------|------|
| 昔闻 | 从前听说 |
| 岳阳楼 | 位于湖南岳阳，临洞庭湖，江南三大名楼之一 |
| 吴楚东南坼 | 吴地和楚地从这里向东南方向分裂开去；坼（chè），分裂、划分 |
| 乾坤日夜浮 | 天地日夜在湖面上漂浮；夸张洞庭湖之大 |
| 亲朋无一字 | 亲朋好友没有一封书信 |
| 老病有孤舟 | 年老多病，只有一叶孤舟相伴 |
| 戎马关山北 | 北方关山之外战马奔腾，指当时吐蕃侵扰边境 |
| 凭轩 | 倚着窗户 |
| 涕泗流 | 眼泪鼻涕一起流下；形容极度悲痛 |

### 译文

从前早就听说洞庭湖水的壮阔，今天终于登上了岳阳楼。
浩瀚的湖水将吴楚两地向东南方向划开，天地日夜在湖面上漂浮荡漾。
亲朋好友们没有一封书信传来，年老多病的我只有一叶孤舟相伴。
北方关山之外战马奔腾烽烟未息，我倚着栏杆，泪水纵横，涕泗横流。

### 文学赏析

**结构：时空对举，由景入情**

全诗可分为两个层次：前四句写登楼所见之景，后四句写登楼所感之情，但景与情并非截然分开，而是彼此渗透、相互生发。

**首联：昔与今的时间对照**

"昔闻洞庭水，今上岳阳楼"——两句看似平淡，实则深沉。"昔闻"含蓄地交代了诗人一生对岳阳楼的向往，"今上"点明终于实现。但"今上"的语境是：年老、多病、离乡、战乱，实现夙愿的喜悦中裹挟着深重的悲凉，一个"今"字，包含了诗人一生的沧桑。

**颔联：宏阔的空间意象**

"吴楚东南坼，乾坤日夜浮"是千古名句，以极度的夸张和雄浑的气势写出洞庭湖的壮阔。"坼"字写出湖水将吴楚大地劈开的力量感；"浮"字写出天地万物似乎都在湖面漂浮的震撼感。这两句是唐诗中写洞庭最壮阔的名句，也为后四句的个人之悲提供了宏大的对比背景。

**颈联：个人困境的白描**

"亲朋无一字，老病有孤舟"——由宏观转入微观，由宇宙转入个人。笔调极为简练，却字字千钧：无亲无书，老且多病，漂泊孤舟。这两句与颔联的壮阔形成鲜明对比：越是宏大的自然，越衬托出个人的渺小与孤苦。

**尾联：由个人升华为家国**

"戎马关山北，凭轩涕泗流"——诗人没有仅仅停留于个人的悲苦，而是将目光投向北方烽烟，以"戎马关山北"点出国家的忧患。在登楼落泪之际，个人的老病孤苦与国家的战乱动荡合流，泪水不仅是为自己而流，更是为天下苍生而流。这是杜甫"沉郁顿挫"风格的核心体现：以个人之悲通向天下之悲。

**与范仲淹《岳阳楼记》的对话**

同一座岳阳楼，范仲淹写"先天下之忧而忧"，杜甫写"凭轩涕泗流"，两者一散文一诗歌，一理念一情感，共同构建了岳阳楼的文化意涵。

### 高考考点

1. **炼字**："坼"和"浮"字的赏析——"坼"以动写静，写出湖之力量；"浮"以浮动写湖之广阔，天地皆在其中。
2. **对比手法**：颔联的宏阔景象与颈联的个人困境形成对比，分析其表达效果。
3. **首联的写法**："昔闻……今上……"的时间对比结构，含蓄地表达了怎样的情感？（夙愿实现的悲喜交集）
4. **主旨分析**：本诗如何由个人之悲上升为家国之忧，体现杜甫"忧国忧民"的诗人情怀。
5. **沉郁顿挫**：结合全诗分析杜甫"沉郁顿挫"的风格特征。
6. **比较阅读**：与杜甫其他登高诗（如《登高》）比较，分析异同；与范仲淹《岳阳楼记》的思想情感比较。
