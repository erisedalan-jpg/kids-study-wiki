# 游园·牡丹亭

> **一句话**：杜丽娘游园伤春，以"姹紫嫣红开遍，似这般都付与断井颓垣"道出了青春与美丽被压抑、被辜负的悲叹，是中国戏曲史上抒情最动人的段落之一。

---

## 🎓 游园·牡丹亭

### 作者与作品

**汤显祖**（1550—1616），字义仍，号海若，又号若士，临川（今江西抚州）人，明代杰出戏曲家，被誉为"中国的莎士比亚"。其代表作"临川四梦"（《牡丹亭》《紫钗记》《南柯记》《邯郸记》）是中国古典戏曲的高峰。《牡丹亭》又称《还魂记》，写杜丽娘为情而死、为情复生的故事，以"情"为核心，具有强烈的人文主义色彩。

《游园》是《牡丹亭》第十出，写杜丽娘与侍女春香游览后花园，面对春日美景，心生伤春之情，随后在梦中与书生柳梦梅相遇，由此引发了整个故事。

### 创作背景

《牡丹亭》完成于明万历二十六年（1598年），是汤显祖最重要的作品。明代礼教森严，女性受到严格束缚。杜丽娘身为名门闺秀，被关在深闺之中，连自家后花园都不曾游览。"游园"一出，是她第一次走出闺房，第一次与大自然接触，春天的生机与自身处境的压抑之间产生了强烈的情感张力。汤显祖借杜丽娘之口，表达了对礼教束缚的批判和对真情、自由的向往。

### 节选原文（【皂罗袍】【好姐姐】两曲）

```
【游园】（第十出节选）

（旦）原来姹紫嫣红开遍，似这般都付与断井颓垣。
良辰美景奈何天，赏心乐事谁家院？
朝飞暮卷，云霞翠轩；
雨丝风片，烟波画船——
锦屏人忒看的这韶光贱！
                ——【皂罗袍】

（旦）遍青山啼红了杜鹃，荼蘼外烟丝醉软。
春香，牡丹虽好，他春归怎占的先？
（贴）成对儿莺燕啊。
（旦）闲凝眄，生生燕语明如剪，呖呖莺歌溜的圆。
                ——【好姐姐】
```

### 注释

| 词语 | 注释 |
|------|------|
| 旦 | 戏曲中的女性角色，此处为杜丽娘 |
| 贴 | 贴旦，丫鬟春香的角色类型 |
| 姹紫嫣红 | 形容各种颜色的花朵鲜艳美丽；姹（chà），美丽 |
| 断井颓垣 | 破败的水井和倒塌的围墙；指荒废的园子 |
| 良辰美景奈何天 | 良辰美景又能怎样呢？（只是徒然）；奈何天，无可奈何的意思 |
| 赏心乐事谁家院 | 赏心乐事是谁家才有的（暗叹自己没有）？ |
| 朝飞暮卷 | 早晨彩霞飞舞，傍晚云霞收卷 |
| 翠轩 | 绿意盎然的轩廊 |
| 烟波画船 | 烟雾迷蒙的水面上的彩绘船只 |
| 锦屏人 | 处于锦绣屏风（深闺）中的人，指杜丽娘自己 |
| 忒 | 太、实在 |
| 韶光贱 | 把美好的时光看得太轻贱（白白辜负了） |
| 荼蘼 | 荼蘼花，花期最晚的春花，荼蘼开后春天结束 |
| 烟丝醉软 | 如烟雾般的柳丝，醉人地柔软飘舞 |
| 闲凝眄 | 悠闲地凝视张望；眄（miǎn），斜视、张望 |
| 生生燕语明如剪 | 燕子的叫声清脆如剪刀般利落 |
| 呖呖莺歌溜的圆 | 黄莺的歌声婉转流畅；呖呖（lì lì），形容鸟鸣声 |

### 白话意译

**【皂罗袍】**：
原来（花园里）姹紫嫣红，百花盛开，可这样的美景都白白对着断井颓垣、无人观赏。良辰美景又有什么用呢？赏心乐事是哪家才有的享受？朝朝暮暮，云霞掩映着翠绿的轩廊；细雨轻风，烟波上漂着画船——我这个关在深闺中的人，实在是太辜负了这美好的春光！

**【好姐姐】**：
青山上杜鹃花红遍，荼蘼花外柳丝如烟，醉人地轻柔。春香，牡丹虽然美丽，可它怎能占得春天最先的风光呢？（春香：一对对莺燕啊。）我悠闲地张望，燕子清脆的叫声明快如剪，黄莺圆润婉转的歌声流水般悠扬。

### 文学赏析

**"情"的主题：汤显祖的核心理念**

汤显祖在《牡丹亭题词》中写道："天下女子有情，宁有如杜丽娘者乎！梦其人即病，病即弥连，至手画形容传于世而后死。死三年矣，复能溟莫中求得其所梦者而生。"《牡丹亭》是一部以"情"对抗"理"（礼教）的作品，"游园"一出是这一主题的起点：杜丽娘在自然中第一次感受到生命的美丽，也第一次感受到自身被压抑的悲哀。

**伤春意象的文化传统**

"姹紫嫣红开遍，似这般都付与断井颓垣"——这是全剧最著名的曲词。花开而无人赏，美景而付断垣，这种"才子佳人空辜负大好春光"的悲叹，是中国文学中伤春传统的集中表达。此处的"断井颓垣"不仅是实景（荒废的后花园），更是象征：代表了礼教制度对青春生命的压制与荒废。

**语言艺术**

南曲（传奇）的语言以优美典丽见长。"良辰美景奈何天，赏心乐事谁家院"出自谢灵运"天下良辰、美景、赏心、乐事，四者难并"，化用前人语意而自然流畅；"生生燕语明如剪，呖呖莺歌溜的圆"，以拟声词和比喻生动描绘鸟鸣，音乐性极强。整段曲词音调流美，适于演唱，是昆曲表演中的经典折子。

**昆曲与传奇**

《牡丹亭》以昆曲演唱为主，"游园惊梦"（含游园和惊梦两出）是昆曲最著名的折子戏，数百年来传演不绝。这种将诗、词、曲、白（宾白）融为一体的戏曲形式，代表了中国古典戏曲的最高文学成就。

### 高考考点

1. **传奇体裁常识**：了解明代传奇（南戏发展而来）的基本形式——出（折）、曲牌、科（动作）、白（对话）的组成。
2. **意象分析**：分析"断井颓垣"的象征意义——不仅是实景，更象征礼教对青春与自由的压制。
3. **主题理解**：杜丽娘游园伤春，体现了怎样的思想情感？汤显祖"至情"观的内涵。
4. **语言特色**：分析【皂罗袍】曲词的语言特点（典雅流丽、音乐性强、情景交融）。
5. **比较文学**：汤显祖与莎士比亚同为1616年去世，同时代东西方戏剧的比较（《罗密欧与朱丽叶》vs《牡丹亭》：共同主题——为情反抗社会束缚）。
6. **昆曲文化**：了解昆曲作为非物质文化遗产的地位，《游园》是昆曲最具代表性的折子戏之一。
