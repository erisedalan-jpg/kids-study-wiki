# 怀疑与学问

> **一句话**：历史学家顾颉刚论述"学则须疑"的治学之道，以名言引用和逻辑推理，说明怀疑不只是消极的批判，而是积极的建设——是做学问的第一步，也是创新的起点。
> **English**: Historian Gu Jiegang argues that doubt is not merely negative criticism but the essential first step of all genuine learning, using classical quotations and careful reasoning to show how skepticism drives knowledge forward.

---

## 📖 课文简介

**作者**：顾颉刚（1893—1980），江苏苏州人，历史学家、民俗学家，"古史辨"学派创始人，以质疑传统历史叙述、还原历史真相著称，著有《古史辨》等。  
**体裁**：议论文  
**学段**：统编版九年级上册第五单元

## 📚 给 6-12 岁（自读版）

### 文章的中心论点

**中心论点**：治学必须有怀疑精神。（"学则须疑"）

用两个古人名言点明：
- 程颐："学者先要会疑。"
- 张载："在可疑而不疑者，不曾学；学则须疑。"

### 为什么要"怀疑"？

作者给出两个分论点：

**分论点一：怀疑是消极方面——辨伪去妄**

不是所有"知识"都是真的。面对书本和权威，要敢于问：
- "这话是真的吗？"
- "证据在哪里？"
- "这个结论有没有漏洞？"

不经怀疑就接受，只会让错误的知识越积越多。

**分论点二：怀疑是积极方面——建设新学说**

怀疑不只是否定，还是创新的起点：
- 质疑旧说 → 发现问题 → 研究问题 → 建立新学说

"一切学问家，不但对于流俗传说，就是对于过去学者的学说也常常要抱怀疑的态度，常常和书中的学说辩论，常常评判书中的学说，常常修正书中的学说。"

### 论证过程梳理

| 步骤 | 内容 |
|------|------|
| 1. 引名言 | 程颐、张载的名言作为道理论据 |
| 2. 解释"怀疑" | 对书本知识要先存怀疑 |
| 3. 举例（消极） | 对传说故事要质疑（如"三皇五帝"的历史可信度） |
| 4. 分析（积极） | 怀疑是创新的基础 |
| 5. 名言收尾 | 再次引用名言强调 |

## 🎓 给 12+（进阶版）

### 引用论证的运用

本文开头和结尾都引用名言，中间也多次引用，形成"引用→分析→引用"的论证节奏：
- **开头引用**：建立权威性，也亮明论点
- **中间引用**（程颐、张载）：道理论据，加强说服力
- **结尾引用**：收束，留下余味

引用论证的作用：引用权威人士的话语，借助其声誉增强自身论点的可信度。

### 从"怀疑"到"学问"的逻辑链

作者的逻辑：
1. 一切知识都是前人的经验/判断
2. 前人可能犯错
3. 所以必须批判性地接受，不盲从
4. 批判 = 怀疑 → 辨伪 + 创新
5. 因此，怀疑是做学问的必要条件

这是一个完整的演绎逻辑，值得理清。

### 作者的学术背景与论文关联

顾颉刚是"古史辨"派创始人——他用怀疑的眼光审视中国古史，质疑《尚书》等经典中"三皇五帝"的真实性，在学术界引起轰动。他写《怀疑与学问》，可以说是对自己治学经验的总结——这篇文章不只是说教，而是他自身经历的提炼。
