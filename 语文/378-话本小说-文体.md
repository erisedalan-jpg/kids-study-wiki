## 🎓 话本小说——中国白话小说的源头

### 定义

**话本**（话本小说）是宋元时期说书艺人（"说话人"）的底本，即说书表演的文字脚本。话本以通俗白话写成，面向市民阶层，内容贴近日常生活，是中国白话小说的源头。后代文人在话本基础上加工、创作的作品称为**拟话本**。

---

### 历史发展

**一、唐代：说话艺术萌芽**

唐代都市繁荣，出现"说话"（讲故事）的民间艺术，寺庙中的"俗讲"（佛教故事讲解）是话本的重要来源。唐代变文（敦煌遗书）是话本的直接前身。

**二、宋代：话本繁荣（鼎盛期）**

宋代城市经济高度发展，瓦舍勾栏（民间娱乐场所）中"说话"成为重要艺术：
- "说话"分四类：**小说**（讲世俗故事）、**说铁骑儿**（讲战争）、**说经**（讲佛道故事）、**讲史**（讲历史）
- "说话人"须背诵大量底本，这些底本即**话本**

宋话本主要特征：
- 白话叙述，口语化
- 故事性强，人物鲜活
- 结构上有"入话"（开场）、正文、结尾的固定格式
- 常有诗词插入点题

代表话本：《碾玉观音》《快嘴李翠莲记》《错斩崔宁》

**三、元代：话本延续**

元代都市说书继续发展，话本积累更丰富，为明代文人整理提供素材。

**四、明代：文人整理与拟话本**

明代冯梦龙将宋元话本整理编辑，同时亲自创作拟话本，结集为著名的**"三言"**：

| 书名 | 内容 |
|------|------|
| 《喻世明言》 | 原名《古今小说》，收录宋元话本及拟话本 |
| 《警世通言》 | 警示世人的故事 |
| 《醒世恒言》 | 使人觉悟的故事 |

凌濛初受三言启发，创作**"二拍"**（拟话本集）：
- 《初刻拍案惊奇》
- 《二刻拍案惊奇》

### 话本小说的艺术特征

| 特征 | 说明 |
|------|------|
| 白话语言 | 使用口语化白话，通俗易懂 |
| 章回结构 | 有固定格式（入话→正文→结语），为章回小说铺路 |
| 说书腔 | 保留说书口气，如"话说""看官""且听下回分解" |
| 插入诗词 | 故事关键处插入诗词点题或抒情 |
| 市民视角 | 故事主角多为商人、手工业者、市井人物 |
| 道德说教 | 多以善恶报应、因果轮回为主题 |

### 代表作品简介

| 作品 | 朝代 | 特点 |
|------|------|------|
| 《碾玉观音》 | 宋 | 爱情悲剧，工匠题材 |
| 《杜十娘怒沉百宝箱》 | 明（冯梦龙） | 批判负心汉，女性形象鲜明 |
| 《卖油郎独占花魁》 | 明（冯梦龙） | 市民励志，爱情正剧 |
| 《沈小霞相会出师表》 | 明（冯梦龙） | 忠义题材 |
| 《十五贯》 | 宋元话本 | 冤案题材，后改编为戏曲 |

---

### 话本与章回小说的关系

话本是章回小说的直接源头：

| 话本 | 章回小说 |
|------|----------|
| 单篇短篇 | 长篇多回 |
| "且听下回"结语 | 正式形成"回"的划分 |
| 口语白话 | 继承白话传统 |
| 故事性 | 发展为完整长篇情节 |

《水浒传》《三国演义》均脱胎于讲史话本的传统。

---

### 高考考点

1. **文体知识**：区分话本、拟话本、章回小说的概念
2. **文学史**：宋元话本→明代"三言二拍"→清代章回小说的发展脉络
3. **作品归属**："三言"属于冯梦龙，"二拍"属于凌濛初
4. **艺术特征分析**：话本的白话语言、说书腔、市民视角
