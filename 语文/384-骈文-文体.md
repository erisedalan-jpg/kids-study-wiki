# 骈文（文体）

> **一句话**：骈文是以四字、六字句为基本单位、讲究对仗声律与用典的古代散文文体，兴盛于魏晋南北朝，唐宋时受古文运动冲击，但部分文体（如赋、祭文）中仍保留大量骈偶特征。

---

## 🎓 高中精读版

### 定义

**骈文**（pián wén），又称"四六文"（因多用四言、六言对句），特点：

| 特征 | 说明 |
|------|------|
| **对仗** | 句句成对，词性相同、结构相当 |
| **声律** | 讲究平仄和谐，读来抑扬顿挫 |
| **用典** | 大量引用历史典故、经典诗文 |
| **辞藻** | 追求华丽，多用绮词丽句 |

"骈"字本义为两马并驾，引申为两两相对——骈文之名由此而来。

### 历史发展

#### 一、先秦两汉：骈偶萌芽

- 先秦诸子（《老子》"信言不美，美言不信"）与《诗经》赋比兴中已有对偶句式。
- 汉代辞赋（见[[赋-文体]]）大量使用对仗，为骈文成熟铺路。

#### 二、魏晋南北朝：全盛期

骈文在此期达到巅峰，几乎所有正式文体（诏书、表、序、书信）均以骈文写就。

**王勃《滕王阁序》**（初唐，骈文最高成就之一）：
- 名句："落霞与孤鹜齐飞，秋水共长天一色。"对仗工整，意境开阔。
- "物华天宝，龙光射牛斗之墟；人杰地灵，徐孺下陈蕃之榻。"——四六句式典范。
- 全篇以骈散结合，叙述、抒情、议论融合，是唐代骈文的代表作。

**庾信《哀江南赋序》**：
- 南朝入北周的庾信，以骈文序写家国之痛，沉郁苍凉，杜甫诗"庾信文章老更成"。

#### 三、唐代：骈文与古文并存

- **韩愈、柳宗元发起古文运动**：以"文以载道"为旗帜，反对骈文的浮靡堆砌，提倡秦汉散文的质朴风格。
- 古文运动后，骈文逐渐退出主流，但在制诰（官方文告）、祭文、赋序等文体中仍被沿用。

#### 四、宋代以后：四六文的延续

- 宋代"四六"仍是官方公文体裁，如欧阳修《丰乐亭记》部分段落仍有骈偶痕迹。
- 清代骈文有复兴之势（汪中、阮元等），被称为"骈文复古"。

### 艺术特征精讲

**1. 对仗之美**
```
落霞 / 与 / 孤鹜 / 齐飞，
秋水 / 共 / 长天 / 一色。
```
主谓结构完全对应，名词对名词，动词对动词，颜色对颜色——视觉与节奏的双重美感。

**2. 用典之密**
一篇骈文中典故密度极高，每一对句往往各含一典：
- "冯唐易老，李广难封"——两个汉代典故，喻不遇之感。
- 读骈文须有历史文化积累，否则典故层层叠叠，难以解读。

**3. 声律之谐**
四六句的平仄搭配有约定俗成的规律，读起来朗朗上口，但规则比律诗宽松。

### 骈文 vs 散文对比

| 维度 | 骈文 | 古文（散文） |
|------|------|------------|
| 句式 | 以四六句为主，对仗严格 | 长短随意，不求对仗 |
| 辞藻 | 华丽，多典故 | 质朴，重逻辑 |
| 功能 | 公文、序、赋、书信 | 叙事、议论、记游 |
| 代表作 | 《滕王阁序》《哀江南赋序》 | 《醉翁亭记》《师说》 |
| 评价 | 古文运动批其"浮靡"，但自有美学价值 | 唐宋八大家主流 |

### 高考考法

1. **翻译**：骈文句式整齐，翻译时注意并列结构、对仗意义的对等呈现。
2. **手法**：辨认对偶、用典、铺陈手法并分析其表达效果。
3. **背诵**：《滕王阁序》中名句段落为常考背诵内容。
4. **赏析**：以《滕王阁序》某对句为例，分析骈文的语言美感。
