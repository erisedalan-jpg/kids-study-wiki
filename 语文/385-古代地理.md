## 🎓 古代地理——九州、山川与古今地名体系

### 定义

古代地理知识是阅读先秦至明清文学文献的基础，涵盖**九州划分**、**五岳四渎**、**方位词习惯**、**古今地名对照**等内容，理解这些知识有助于把握文学作品的地理背景与文化意涵。

---

### 一、九州

九州是古代中国对天下的地理划分，见于《尚书·禹贡》《周礼·职方氏》等文献，传说大禹治水后划定：

| 州名 | 大致对应地区 |
|------|-------------|
| 冀州 | 今河北、山西一带（天子所在，最重要） |
| 兖州 | 今山东西部、河南东北 |
| 青州 | 今山东东部、辽宁南部 |
| 徐州 | 今江苏北部、安徽北部 |
| 扬州 | 今江苏南部、浙江、安徽南部 |
| 荆州 | 今湖北、湖南 |
| 豫州 | 今河南、湖北北部（"中州"） |
| 梁州 | 今陕西南部、四川 |
| 雍州 | 今陕西北部、甘肃、宁夏 |

**文学中的"九州"**：

| 文献 | 原句 | 含义 |
|------|------|------|
| 陆游《示儿》 | "死去元知万事空，但悲不见九州同" | 九州=全国统一 |
| 龚自珍《己亥杂诗》 | "九州生气恃风雷" | 全国局面 |
| 《过秦论》 | "奋六世之余烈，振长策而御宇内" | 宇内=天下 |

### 二、五岳与四渎

**五岳**（天下名山）：

| 岳 | 位置 | 文学意象 |
|----|------|----------|
| 东岳泰山 | 山东 | "会当凌绝顶，一览众山小"（杜甫） |
| 西岳华山 | 陕西 | 险峻，"只有天在上，更无山与齐" |
| 南岳衡山 | 湖南 | 祭祀圣地 |
| 北岳恒山 | 山西 | 雄壮 |
| 中岳嵩山 | 河南 | 中原核心，天子封禅 |

**四渎**（天下四大河）：江（长江）、河（黄河）、淮（淮河）、济（济水）

"四渎"之说见《尔雅》："江、河、淮、济为四渎。"

### 三、重要山川与关隘

| 地名 | 位置 | 文学关联 |
|------|------|----------|
| 函谷关 | 今河南灵宝 | 秦地门户，"秦关百二"；贾谊《过秦论》 |
| 崤山 | 今河南 | 秦晋崤之战 |
| 太行山 | 山西、河北 | 曹操《苦寒行》 |
| 终南山 | 陕西西安南 | 隐士居所意象 |
| 峨眉山 | 四川 | 李白《峨眉山月歌》 |
| 洞庭湖 | 湖南 | 范仲淹《岳阳楼记》 |
| 赤壁 | 湖北 | 苏轼《赤壁赋》 |
| 江南 | 长江以南 | 温柔富庶之地意象 |

### 四、方位词习惯

古代方位词有特殊含义，需结合地理理解：

| 词汇 | 含义规律 | 举例 |
|------|----------|------|
| 山东 | 崤山或太行山以东 | 秦时"山东六国"（非今山东省） |
| 山西 | 太行山以西 | 今山西省得名于此 |
| 江南 | 长江以南 | 泛指南方富庶之地 |
| 江东 | 长江下游以东（安徽东南、江苏南部） | 项羽"无颜见江东父老" |
| 关中 | 函谷关以西、关中平原 | 今陕西中部，秦汉核心地 |
| 关外 | 函谷关以东 | 六国之地 |
| 河北 | 黄河以北 | ≠今河北省 |
| 河南 | 黄河以南 | ≠今河南省（古义更宽泛） |
| 朔方 / 北方 | 漠北、塞外 | 边塞诗意象 |
| 西域 | 今新疆及中亚 | 张骞出使、玄奘取经 |

### 五、重要古今地名对照

| 古地名 | 今地名 | 文学关联 |
|--------|--------|----------|
| 长安 | 西安（陕西） | 唐代都城，诗人汇聚 |
| 洛阳（洛邑） | 洛阳（河南） | 东汉都城，"洛阳纸贵" |
| 金陵 | 南京（江苏） | 六朝古都，《桃花扇》等 |
| 姑苏 | 苏州（江苏） | "姑苏城外寒山寺"（张继） |
| 幽州 | 北京一带 | 边塞意象 |
| 蓟（jì） | 北京 | 燕国都城 |
| 汴梁 / 东京 | 开封（河南） | 北宋都城，《清明上河图》 |
| 临安 | 杭州（浙江） | 南宋都城 |
| 会稽 | 绍兴（浙江） | 越国故地，兰亭所在 |
| 吴 | 苏州地区 | 吴国，吴歌，江南文化 |

### 高考考法

1. **方位词辨析**："山东""河北"等古义与今义区别
2. **地名还原**：文言文中"长安""金陵"等对应今地
3. **五岳判断**：东南西北中与省份对应
4. **文化背景理解**：函谷关、赤壁、洞庭湖等地的文学意象
