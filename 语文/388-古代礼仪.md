## 🎓 古代礼仪——揖让、座次与服饰的礼制体系

### 定义

古代礼仪（礼制）是中国古代社会规范人际关系、维护等级秩序的行为准则体系，以《周礼》《仪礼》《礼记》（合称"三礼"）为经典依据，涵盖祭祀、冠婚、丧葬、宾客、朝会等各类场合，体现儒家"礼治"思想的核心。

---

### 一、揖礼与常见礼仪动作

**揖（jī）**：拱手行礼，双手合抱于胸前，欠身致意，是最常见的见面礼

**揖的分类**（依对象等级）：
- **天揖**（上揖）：对尊者，手推向上方
- **时揖**（中揖）：对平辈，手推向前方
- **土揖**（下揖）：对下级，手推向下方

**其他常见礼仪动作**：

| 礼仪动作 | 说明 |
|----------|------|
| 拜 | 跪地叩首，最正式的礼节，分稽首、顿首、空首等 |
| 稽首 | 最重礼，头叩地并停留一段时间，臣对君 |
| 顿首 | 头短暂触地，地位相当者间使用 |
| 空首 | 头叩手而非叩地，常见于臣对君的日常礼 |
| 长揖 | 拱手高举过头再鞠躬，不跪的尊重礼 |
| 万福 | 女子礼，双手交叠置于腰部右侧，微屈膝 |
| 叉手 | 左手握右手大拇指，置于胸前，唐宋男子常用 |

**相关文学例证**：

| 文献 | 原句 | 礼仪内容 |
|------|------|----------|
| 《鸿门宴》 | "哙拜谢，起，立而饮之" | 樊哙跪拜谢礼 |
| 《廉颇蔺相如列传》 | "拜送书于庭" | 正式拜礼 |
| 《孔雀东南飞》 | "府吏还家去，上堂拜阿母" | 晚辈拜长辈 |

### 二、座次礼仪

**室内座次（堂屋）**：

古代室内以**南向（坐北朝南）为尊**，故最尊贵的位置是**堂屋北墙南向的座位**。

| 位置 | 尊卑 | 场合 |
|------|------|------|
| 北面南向（坐北朝南） | 最尊 | 主人、君主 |
| 西面东向 | 次尊 | 客人（宾）的尊位 |
| 南面北向 | 再次 | 主人谦让或陪坐 |
| 东面西向 | 最卑 | 仆从、侍者 |

**特殊场合**：
- **朝堂**：天子南面，群臣北面朝拜（"南面称王"）
- **车中**：左边（车左）为尊，故勇士居左，御者（驭手）居右
- **宴席**：面朝门口（南向）为上座，背对门（北向）为末位

**文学例证**：

| 文献 | 原句 | 礼仪说明 |
|------|------|----------|
| 《鸿门宴》 | "项王、项伯东向坐；亚父南向坐……沛公北向坐，张良西向侍" | 座次严格体现尊卑 |
| 《信陵君窃符救赵》 | "公子从车骑，虚左" | 虚左以待尊客 |

### 三、冠礼与笄礼（成年礼）

**冠礼**：男子**二十岁**行冠礼（戴帽），标志成年
- 由父亲或尊长主持，宾客加三次冠（缁布冠→皮弁→爵弁）
- 加冠后取**字（表字）**，不再直呼其名
- 《礼记·冠义》：冠礼是"礼之始"

**笄礼**：女子**十五岁**行笄礼（插发簪），标志成年
- "碧玉妆成一树高，万条垂下绿丝绦"——女子及笄后的形象
- "待字闺中"即女子尚未许配、尚未取字之意

### 四、服饰礼制

古代服饰有严格等级规定，颜色、纹样、材质均有章法：

| 等级 | 色彩规定 |
|------|----------|
| 天子 | 衮（gǔn）服，十二章纹，黄色为尊 |
| 诸侯 | 冕服，章纹减等 |
| 士大夫 | 朝服，颜色依品级 |
| 庶民 | 白衣（布衣），不得用紫绯 |

**重要服饰术语**：

| 术语 | 含义 |
|------|------|
| 布衣 | 平民百姓（穿麻布衣服） |
| 缙绅 | 有官职者（插笏于绅带） |
| 黔首 | 秦代对平民的称呼（黑色头巾） |
| 峨冠博带 | 高冠宽带，士大夫装束 |
| 素衣 | 白色素服，丧服或谦洁之意 |

**文学例证**：

| 文献 | 原句 | 服饰含义 |
|------|------|----------|
| 《过秦论》 | "将数百之众，转而攻秦；斩木为兵，揭竿为旗" | 布衣百姓起义 |
| 《出师表》 | "臣本布衣，躬耕于南阳" | 诸葛亮自称平民身份 |
| 《陈情表》 | "茕茕孑立，形影相吊" | 孤寒之貌 |

### 高考考法

1. **座次判断**：鸿门宴等文言段落中的座次礼仪分析
2. **称谓礼仪**：区分直呼其名（不敬）与称字（尊重）的场合
3. **文化常识题**：冠礼年龄（20岁）、笄礼年龄（15岁）、揖拜区别
4. **服饰术语**：布衣、峨冠博带等词语的文化含义
