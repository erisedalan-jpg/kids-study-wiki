## 🎓 古代纪年——干支、年号与节气的时间体系

### 定义

古代纪年是中国历史上用于记录时间的多种方法的总称，主要包括**干支纪年**、**帝王年号纪年**、**王公即位年次纪年**及**岁星（太岁）纪年**等体系，并与二十四节气共同构成传统时间观念的完整框架。

---

### 一、天干地支纪年

**天干**（10个）：甲、乙、丙、丁、戊、己、庚、辛、壬、癸

**地支**（12个）：子、丑、寅、卯、辰、巳、午、未、申、酉、戌、亥

天干与地支依次两两配合，循环一周为**60年**，称"一甲子"或"花甲"。

| 地支 | 对应生肖 | 对应时辰（北京时间） |
|------|----------|---------------------|
| 子 | 鼠 | 23:00—01:00 |
| 丑 | 牛 | 01:00—03:00 |
| 寅 | 虎 | 03:00—05:00 |
| 卯 | 兔 | 05:00—07:00 |
| 辰 | 龙 | 07:00—09:00 |
| 巳 | 蛇 | 09:00—11:00 |
| 午 | 马 | 11:00—13:00 |
| 未 | 羊 | 13:00—15:00 |
| 申 | 猴 | 15:00—17:00 |
| 酉 | 鸡 | 17:00—19:00 |
| 戌 | 狗 | 19:00—21:00 |
| 亥 | 猪 | 21:00—23:00 |

干支也用于纪月（十二月建）、纪日、纪时，构成完整的**四柱（年月日时）**体系。

**文学中的干支纪年**：

| 文献 | 原句 | 对应年份 |
|------|------|----------|
| 《兰亭集序》王羲之 | "永和九年，岁在癸丑" | 353年 |
| 《赤壁赋》苏轼 | "壬戌之秋，七月既望" | 1082年 |
| 《五人墓碑记》张溥 | "天启丁卯三月之望" | 干支+年号并用 |

---

### 二、帝王年号纪年

汉武帝于公元前140年首创**年号制度**（建元元年），此后历代皇帝以年号纪年，改元时重新计数。

**年号使用规律**：
- 一位皇帝可用多个年号（汉武帝用11个，武则天用17个）
- 明清开始一帝一元（明成祖永乐、清圣祖康熙等）
- 故称明清皇帝常以年号代称（康熙、乾隆等）

**常见重要年号**：

| 年号 | 皇帝 | 朝代 | 文学关联 |
|------|------|------|----------|
| 建元 | 汉武帝 | 西汉 | 最早年号 |
| 贞观 | 唐太宗 | 唐 | "贞观之治" |
| 开元 | 唐玄宗 | 唐 | 唐诗鼎盛期 |
| 元丰 | 宋神宗 | 北宋 | 苏轼被贬黄州 |
| 绍圣 | 宋哲宗 | 北宋 | 苏轼再贬岭南 |
| 嘉靖 | 明世宗 | 明 | 《西游记》成书背景 |
| 崇祯 | 明思宗 | 明 | 明亡前最后年号 |

**纪年格式**：年号+年次，如"贞观三年"、"永和九年"（永和为东晋穆帝年号）。

---

### 三、王公即位年次纪年

先秦时期无年号，以君主在位年数纪年：
- "鲁隐公元年"（《左传》开篇）
- "十二年，齐桓公伐楚"（《管子》）

---

### 四、二十四节气

节气是太阳历的体现，是农业社会对自然周期的精准把握，2016年列入联合国非物质文化遗产。

**二十四节气歌**：
> 春雨惊春清谷天，夏满芒夏暑相连，
> 秋处露秋寒霜降，冬雪雪冬小大寒。

| 季节 | 节气（共6个） |
|------|-------------|
| 春 | 立春、雨水、惊蛰、春分、清明、谷雨 |
| 夏 | 立夏、小满、芒种、夏至、小暑、大暑 |
| 秋 | 立秋、处暑、白露、秋分、寒露、霜降 |
| 冬 | 立冬、小雪、大雪、冬至、小寒、大寒 |

**文学中的节气意象**：

| 节气 | 代表诗句 |
|------|----------|
| 清明 | "清明时节雨纷纷，路上行人欲断魂"（杜牧） |
| 冬至 | "天时人事日相催，冬至阳生春又来"（杜甫） |
| 寒食 | "春城无处不飞花，寒食东风御柳斜"（韩翃） |
| 重阳（非节气但相关） | "遥知兄弟登高处，遍插茱萸少一人"（王维） |

### 其他纪时词语

| 词语 | 含义 |
|------|------|
| 既望 | 农历十六日（"望"为十五） |
| 朔 | 农历初一 |
| 晦 | 农历最后一日 |
| 午时 | 正午11—13时 |
| 子时 | 午夜23—1时 |
| 五更 | 凌晨3—5时（三更半夜为子时） |

---

### 高考考法

1. **纪年换算**：给出"壬戌之秋"等，判断是干支纪年还是年号纪年
2. **节气常识**：清明、冬至、寒食等节气含义及习俗
3. **文言传记**：理解"某年某月某日"的表达方式
4. **文化常识判断题**：天干地支、年号、节气等基础知识
