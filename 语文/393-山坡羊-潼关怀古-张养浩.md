# 山坡羊·潼关怀古

> **一句话**：张养浩经过潼关时，望着历代王朝兴亡的古道，写下"兴，百姓苦；亡，百姓苦"——无论王朝兴起还是衰亡，承受苦难的始终是普通百姓，这是中国古代文学中极为罕见的民本意识的直接表达。
> **English**: Passing through Tongguan Pass, Zhang Yanghao gazes at the ruins of dynasty after dynasty and writes "When empires rise, the people suffer; when empires fall, the people suffer" — a strikingly rare direct expression of people-centered compassion in classical Chinese literature.

---

## 📖 课文简介

**作者**：张养浩（1270—1329），字希孟，号云庄，元代散曲家、政治家。曾辞官归隐，后因关中大旱、灾民流离失所，应召出任陕西行台中丞，赈灾途中积劳成疾，去世于任上。
**体裁**：散曲，曲牌名"山坡羊"（越调），属小令
**背景**：元文宗天历二年（1329），张养浩奉命赴关中赈灾，途经潼关，见历代战场遗迹与灾民惨状，有感而发，写下此曲。
**学段**：统编版九年级下册第六单元 诗词曲五首

## 📚 给 6-12 岁（自读版）

### 全曲原文

峰峦如聚，波涛如怒，山河表里潼关路。望西都，意踌躇。
伤心秦汉经行处，宫阙万间都做了土。
兴，百姓苦；亡，百姓苦。

### 重要注释

| 词语 | 注释 |
|------|------|
| 峰峦如聚 | 山峰好像都聚拢过来（形容潼关一带山势险峻） |
| 波涛如怒 | 黄河波涛像在怒吼（写黄河之急） |
| 山河表里 | 山河内外，形容潼关地势险要，有山有河护卫 |
| 潼关路 | 潼关（今陕西潼关），古代关中的门户，历史上的军事要地 |
| 西都 | 长安（今西安），西汉、隋唐的都城 |
| 踌躇 | 徘徊，此处形容感慨万千、心情复杂 |
| 经行处 | 曾经经过、存在的地方 |
| 宫阙万间 | 无数的宫殿楼阁 |
| 都做了土 | 全都变成了废墟尘土 |
| 兴 | 王朝兴起 |
| 亡 | 王朝覆灭 |

### 白话译文

山峰好像都聚拢而来，黄河波涛汹涌怒吼，山河内外，这就是潼关的道路。向西望着（昔日的）长安，心中感慨万千，徘徊不前。令人伤心的是，秦汉时代曾经存在的地方，那无数的宫殿楼阁，如今全都变成了废墟尘土。（历史上）王朝兴起，百姓受苦；王朝覆灭，百姓还是受苦。

### 结构分析

| 层次 | 内容 | 手法 |
|------|------|------|
| 前三句 | 描写潼关地势——山聚、水怒、险关 | 动态描写，拟人 |
| 中间三句 | 望西都——感慨秦汉宫阙化为尘土 | 怀古，叙事 |
| 最后两句 | 点明主旨：兴亡皆苦百姓 | 议论，对比 |

## 🎓 给 12+（进阶版）

### "兴，百姓苦；亡，百姓苦"的思想价值

这句话的深刻在于打破了"兴比亡好"的常识：
- 通常人们认为太平盛世好（兴），战乱亡国苦（亡）
- 张养浩指出：**兴**时，统治者大兴土木（"宫阙万间"），征夫徭役，百姓同样受苦
- **亡**时，战乱纷飞，兵燹流离，百姓更是受苦
- 无论哪种情形，受苦的永远是普通百姓，这是对"王朝兴亡史观"的深刻反思

这种思想在中国古代政治文学中相当罕见——大多数文人写兴亡，着眼的是帝王的功业或王朝的命运，张养浩却直接说出了百姓的苦。

### 元曲的文体特点

与唐诗、宋词相比，元曲（散曲）的特点：
1. **语言更口语化**：不像唐诗那样凝练，更接近日常说话
2. **可以加衬字**：在规定字数外可以加入额外的字，增加表达灵活性
3. **音乐性强**：元曲原本是配乐演唱的，每个曲牌都有特定的旋律
4. **形式相对自由**：比词更灵活，情感表达更直接

《山坡羊·潼关怀古》语言简洁有力，收尾"兴，百姓苦；亡，百姓苦"节奏鲜明，朗朗上口，体现了散曲语言的特色。

### 开篇的描写手法

"峰峦如聚，波涛如怒"
- **如聚**：拟人化，山峰仿佛有生命，向这里聚集，写出山势的险峻
- **如怒**：拟人化，波涛像在愤怒，写出黄河的汹涌
- 两句对仗，动感强烈，为后面的感慨作铺垫——这么险要的地方，见过多少朝代兴亡？

### 写作背景的悲悯情怀

张养浩赴关中赈灾，一路看到的是：

> 道路之上，哀鸿遍野。灾民流离，饿殍横野。

他最终在赈灾过程中去世，与一般文人的"怀古"相比，他的感慨有更真实的社会背景支撑，因此"百姓苦"三字格外沉重。
