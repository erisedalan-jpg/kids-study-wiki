# 烛之武退秦师-左传

> **一句话**：选自《左传·僖公三十年》，记郑国老臣烛之武只身夜入秦营，凭辩才说退秦穆公，挽救郑国于秦晋联军围攻之下。
> **English**: From Zuozhuan, this account tells how the elderly Zheng minister Zhu Zhiwu single-handedly persuaded Duke Mu of Qin to withdraw, saving Zheng from the joint Qin-Jin siege.

---

## 🎓 高中进阶

### 出处

- 《左传·僖公三十年》（前 630 年）
- 《左传》——又称《春秋左氏传》，相传为左丘明所作
- 中国第一部叙事详尽的编年体史书
- 反映春秋时期政治、军事、外交

### 故事梗概

| 段 | 内容 |
|----|------|
| 起因 | 晋文公（重耳）联秦穆公围郑——晋文公因当年流亡时郑国"无礼"于他，且郑曾"贰于楚" |
| 出兵 | 秦军在郑南，晋军在郑北，郑国陷于双面包围 |
| 推荐 | 佚之狐推荐已年老的烛之武 |
| 受命 | 烛之武感叹年轻时未受重用，但在国难之际仍勇受命 |
| 夜入秦营 | "夜，缒而出"，悄悄出城见秦穆公 |
| 进言 | 烛之武陈三利害打动秦穆公 |
| 撤兵 | 秦撤军并留人助郑防晋；晋无奈撤兵 |

### 烛之武进言三大要点

```
1. 灭郑无利：郑近晋远秦，灭郑只增晋之力，于秦无益
2. 留郑有利：保留郑作为秦在中原的接应，对秦有利
3. 晋之无信：举晋惠公负秦之事，警告秦穆公防晋反复
```

逻辑严密、循循善诱，由"无利"→"有利"→"防患"层层推进。

### 核心字句

> 夜缒而出，见秦伯，曰："秦、晋围郑，郑既知亡矣。
> 若亡郑而有益于君，敢以烦执事。
> 越国以鄙远，君知其难也。焉用亡郑以陪邻？
> 邻之厚，君之薄也。"

### 字词

| 字 | 释义 |
|----|------|
| 师 | 军队 |
| 缒 (zhuì) | 用绳子从城上吊下 |
| 鄙 (作动词) | 把...当边邑 |
| 陪 | 增加 |
| 厚 / 薄 | 强 / 弱 |
| 既 | 已经 |
| 共其乏困 | 通"供" |
| 朝济而夕设版 | 早渡河晚就修工事——背信弃义 |
| 阙 (quē) 秦 | 损害秦 |
| 唯君图之 | 请君考虑 |
| 微夫人之力 | 没有那个人的帮助 |

### 人物形象

| 人物 | 形象 |
|------|------|
| 烛之武 | 年迈仍出仕，外交辞令高超，深谋远虑 |
| 郑伯 | 危急时谦虚谋臣 |
| 秦穆公 | 务实，善于权衡利弊 |
| 晋文公 | 顾全大局，在外交受挫后冷静撤兵 |
| 子犯（晋大将） | 主张追秦兵，被晋文公止住 |

### 论辩艺术

| 技巧 | 体现 |
|------|------|
| **以情入理** | 既知亡，自降姿态 |
| **分析利害** | 越国鄙远的难处 |
| **挑拨瓦解** | 邻厚君薄 |
| **援引旧事** | 晋惠公"朝济夕版"的失信 |
| **言简意赅** | 通篇辞令简洁有力 |

### 春秋外交特点

- 国家间盛行"会盟、辞令、纵横"
- 外交辞令成为重要技艺
- 一人之言有时改变战局
- 体现"春秋无义战，但有讲理"的特征

### 《左传》的历史地位

| 项目 | 内容 |
|------|------|
| 体裁 | 编年体史书 |
| 时间 | 春秋（前 722 - 前 468） |
| 风格 | 叙事详尽、文笔优美、善写战争与辞令 |
| 与其他比较 | 《国语》（国别体）、《战国策》（国别体） |
| 地位 | "春秋三传"之一（左传/公羊传/榖梁传） |

### 易错点

- 烛之武是郑国老臣，年老才受命
- "缒"是用绳吊下，不是"垂"
- "鄙"作动词是"以...为鄙(边邑)"，不是"鄙视"
- 秦穆公接受劝说撤兵，并留兵助郑（杞子等三人）
- 晋文公没追击秦军，体现政治理性
