# 齐桓晋文之事-孟子

> **一句话**：《孟子·梁惠王上》篇章，记孟子与齐宣王对话，借齐宣王问"齐桓晋文之事"引出孟子"行仁政、王天下"的政治主张。
> **English**: A chapter from Mencius (Liang Hui Wang) where Mencius dialogues with King Xuan of Qi, advocating benevolent government over hegemonic conquest.

---

## 🎓 高中进阶

### 出处

- 《孟子·梁惠王上》
- 孟子（约前 372-前 289），名轲，战国儒家代表
- "亚圣"，仅次于孔子

### 对话梗概

齐宣王问孟子"齐桓晋文之事可得闻乎"——想了解春秋霸主的事。孟子转而劝说他行**仁政**：

```
1. 立基础：宣王有"以羊易牛"的不忍之心，可推行仁政
2. 论本质：王道在于"以不忍人之心，行不忍人之政"
3. 析现实：宣王想征兵服远（霸道），事实上"缘木求鱼"——不可能成功
4. 提建议：要"制民之产，制其生死"——给百姓土地、教化、生活保障
5. 描蓝图：使民"养生丧死无憾"，"五十者可以衣帛""七十者可以食肉"
```

### 核心思想

| 思想 | 内容 |
|------|------|
| 仁政 | 以仁爱治国 |
| 民本 | "民为贵，社稷次之，君为轻" |
| 王道 vs 霸道 | 王道以德服人，霸道以力服人 |
| 推恩 | 推己之心至民众 |
| 制民之产 | 保障百姓生计 |

### 论辩技巧

孟子论辩以善用比喻、迂回引导著称：

| 技巧 | 例子 |
|------|------|
| **以子之矛攻子之盾** | 引宣王"以羊易牛"故事证其有不忍之心 |
| **比喻论证** | "缘木求鱼""为长者折枝" |
| **对比论证** | 王道 vs 霸道对比 |
| **设问反诘** | 引导宣王步步认同 |
| **类比推理** | 不忍杀牛 → 不忍施暴政 |
| **诉诸常识** | 用百姓最基本愿望说服 |

### 名句

> 老吾老，以及人之老；幼吾幼，以及人之幼。
> 推恩足以保四海，不推恩无以保妻子。

### 主要句式

- 宾语前置："臣未之闻也" = 臣未闻之
- 判断句："非不能也""是何异于刺人而杀之"
- 状语后置："树之以桑" = 以桑树之

### 与孔子思想比较

| 项目 | 孔子 | 孟子 |
|------|------|------|
| 核心 | 仁、礼 | 仁政、性善 |
| 政治 | 礼治 | 王道、民本 |
| 论辩 | 简洁含蓄 | 雄辩、富气势 |
| 比喻 | 较少 | 大量 |

### 战国时期背景

- 各国争雄，霸道盛行
- 孟子周游列国劝行仁政
- 多数君主追求强力，孟子主张未被采纳
- 但其思想影响后世深远

### 易错点

- "齐桓晋文"指齐桓公、晋文公，是春秋五霸
- 孟子转移话题——从"霸道"引到"王道"
- "缘木求鱼"是名喻，比喻方法错误必难成功
- "推恩"是推己之心至他人，是仁政关键
