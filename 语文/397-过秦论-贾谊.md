# 过秦论-贾谊

> **一句话**：西汉贾谊总结秦朝兴亡的政论文，提出"仁义不施而攻守之势异也"，是先秦至西汉政论文典范。
> **English**: Jia Yi's "Faulting Qin" analyzes the rise and fall of Qin, concluding that the lack of benevolence shifted Qin from offense to defense, leading to its collapse.

---

## 🎓 高中进阶

### 作者与背景

- 作者：贾谊（前 200-前 168），字号无传，西汉政论家、文学家
- 时代：西汉文帝时
- 写作目的：总结秦亡教训，劝汉文帝行仁政
- 文体：**政论文**（议论文）

### 全文三段（教材常选其中一段或全文）

**上篇** —— 写秦的兴起与统一
**中篇** —— 写秦统一后政策
**下篇** —— 写秦的速亡

教材常选"上篇" + 最后总论。

### 论证结构

| 部分 | 内容 |
|------|------|
| 第一段 | 秦孝公至秦始皇——秦由小国到统一天下，"奋六世之余烈" |
| 第二段 | 秦统一后实行严刑峻法、"焚百家之言""愚民"政策 |
| 第三段 | 陈胜吴广起义——一介百姓即可以毁灭秦帝国 |
| 总论 | "仁义不施而攻守之势异也" |

### 名句

> 振长策而御宇内，吞二周而亡诸侯，履至尊而制六合，执敲扑而鞭笞天下。
> 焚百家之言，以愚黔首；隳名城，杀豪杰。
> 天下云集响应，赢粮而景从。
> 一夫作难而七庙隳，身死人手，为天下笑者，何也？
> **仁义不施而攻守之势异也。**
> 试使山东之国与陈涉度长絜大，比权量力，则不可同年而语矣。

### 中心论点

> **仁义不施而攻守之势异也。**

秦在攻天下时强大，得天下后未施仁政，故失天下。

### 论证手法

| 手法 | 体现 |
|------|------|
| **铺陈** | 详写秦的强大兴起 |
| **对比** | 强秦 vs 弱涉；秦攻 vs 秦守 |
| **史例** | 秦孝公至秦始皇六代积累 |
| **气势磅礴** | 排比、对偶气势恢宏 |
| **以小见大** | 一介戍卒陈胜推翻强秦 |
| **总分推进** | 描述事实 → 突出反差 → 提出论点 |

### 文学特色

| 特色 | 内容 |
|------|------|
| **辞赋化议论** | 大量铺陈描写如赋 |
| **气势宏大** | 句式整齐、排山倒海 |
| **理性深刻** | 论点明确简洁 |
| **结构严谨** | 起承转合分明 |

### 历史意义

- 影响汉文帝采取"与民休息"的政策
- 为汉初"无为而治"提供思想依据
- 启发后世"兴亡论"——杜牧《阿房宫赋》、苏洵《六国论》皆受其影响
- 鲁迅称："西汉鸿文，沾溉后人，其泽甚远。"

### 贾谊其人

- "西汉初年最有才华的青年学者之一"
- 文帝时被推荐为博士，年方 22
- 因受人嫉妒被贬为长沙王太傅
- 33 岁因思念其抚养的梁怀王而忧伤死
- 代表作：《过秦论》《治安策》《吊屈原赋》

### 与同主题文章对比

| 篇目 | 作者 | 立论角度 |
|------|------|--------|
| 过秦论 | 贾谊（汉） | 仁义不施 |
| 阿房宫赋 | 杜牧（唐） | 借古讽今、奢侈亡国 |
| 六国论 | 苏洵（宋） | 弊在赂秦 |

### 易错点

- "过"作"评论过错"讲，不是"过去"
- "策"是马鞭，不是计策
- "黔首"是秦时对百姓的称呼，不是黑头发的人
- 贾谊不是法家，主张儒家仁政
- 一夫作难——"夫"指陈胜本人，不是丈夫
