# 雷雨-曹禺

> **一句话**：曹禺创作于 1933 年的四幕话剧，以周公馆为舞台，展现一个封建家庭三十年间纠葛恩怨在一个雷雨之夜的爆发与崩溃。
> **English**: Cao Yu's four-act play (1933) unfolds the long-buried secrets and tragic fate of a feudal Chinese family over one stormy night.

---

## 🎓 高中进阶

### 作者与背景

- 作者：曹禺（1910-1996），原名万家宝，中国现代话剧奠基人
- 时代：1933 年（曹禺 23 岁时创作）
- 文体：**四幕话剧**
- 背景：1920 年代中国，封建家族体制与新文化的冲突

### 主要人物

| 人物 | 身份 | 性格 |
|------|------|------|
| **周朴园** | 煤矿主、家长 | 专制虚伪、冷酷自私 |
| **繁漪** | 周朴园后妻 | 反抗、爱欲炽烈、悲剧性强 |
| **周萍** | 周朴园长子（与前妻所生） | 软弱、矛盾，与繁漪有禁忌之恋 |
| **周冲** | 周朴园次子（繁漪所生） | 单纯、有理想 |
| **鲁侍萍** | 周朴园前情人，被抛弃 | 隐忍、痛苦 |
| **鲁四凤** | 侍萍的女儿 | 单纯活泼，与周萍相恋（不知是同父异母） |
| **鲁大海** | 侍萍的儿子 | 工人代表、激进 |
| **鲁贵** | 侍萍现夫 | 市侩、无耻 |

### 复杂关系（封建悲剧根源）

```
周朴园（前情人）→ 鲁侍萍（被抛弃）
   |                    |
   ├─ 周萍（与侍萍所生子，由周朴园抚养）
   |                    └→ 鲁四凤（侍萍后嫁鲁贵所生女）
   ├─ 周冲（繁漪所生）
   └─ 繁漪（后妻）
              ↓
       禁忌之恋：繁漪 ↔ 周萍（继母与继子）
       不伦之恋：周萍 ↔ 鲁四凤（同母异父兄妹！）
```

### 故事梗概（一夜之间）

- 三十年前，周朴园诱骗侍女侍萍生子（周萍）后将其抛弃
- 三十年后，侍萍带女儿四凤再次进入周家做佣人（侍萍并不知周朴园身份）
- 周萍与四凤相恋（未知血缘）
- 繁漪因周萍冷落而疯狂
- 周朴园拒绝向罢工工人让步，鲁大海代表工人对抗
- 真相一夜揭开：四凤与周萍是兄妹、繁漪与周萍是继母子、鲁大海是周朴园亲子
- 雷雨夜：周冲触电死、四凤触电死、周萍举枪自尽、繁漪疯、侍萍精神崩溃

### 戏剧手法

| 手法 | 体现 |
|------|------|
| **三一律** | 一天内、一个场景（周公馆）、一条主要情节线 |
| **舞台说明** | 开场详写周公馆陈设、雷雨气氛 |
| **铺垫与暗示** | 闷热、雷声、关窗等多次预兆 |
| **对话潜台词** | 人物之间话中有话 |
| **巧合命运** | 无数巧合纠合在一夜爆发 |
| **象征手法** | 雷雨——压抑情感与命运的爆发 |

### 主题

| 主题 | 体现 |
|------|------|
| 封建家庭批判 | 专制家长 + 压抑女性 |
| 命运的悲剧性 | 无法逃脱的血缘罪 |
| 阶级对立 | 资本家与工人 |
| 人性复杂 | 每个人物都是受害者也是施害者 |
| 爱与欲 | 繁漪、四凤、周萍多重情感 |

### 重点台词分析

**繁漪：**
> "我要的是那个有血肉、有性情、敢爱敢恨的男人，可你就是没法子。"
> "我是流着一颗痛苦的心来的。"

**周朴园（虚伪温情）：**
> "三十年前，在无锡有一件很出名的事情......"

### 戏剧结构

| 幕 | 内容 |
|----|------|
| 第一幕 | 周公馆早晨——四凤、繁漪等出场 |
| 第二幕 | 鲁侍萍来访，与周朴园相认 |
| 第三幕 | 鲁家——四凤与周萍约定私奔；繁漪跟踪 |
| 第四幕 | 周公馆深夜——真相揭露，悲剧爆发 |

### 曹禺戏剧创作

| 作品 | 年代 | 特点 |
|------|------|------|
| 雷雨 | 1933 | 处女作、家庭悲剧 |
| 日出 | 1936 | 都市百态 |
| 原野 | 1937 | 复仇悲剧 |
| 北京人 | 1940 | 没落士绅家庭 |

### 中国现代话剧奠基意义

- 标志中国话剧成熟
- 借鉴易卜生、奥尼尔等西方戏剧手法
- 同时融入中国家族文化
- 至今仍为话剧经典剧目

### 易错点

- 繁漪与周朴园是续弦关系（后妻）
- 周萍是周朴园前情人侍萍所生，由周朴园抚养
- 四凤是侍萍后嫁鲁贵所生——周萍和四凤是同母异父兄妹
- 序幕和尾声：原作有教堂场景（多年后），多被改编版省略
- 雷雨象征——既是天气，也是人物情感与命运的爆发
