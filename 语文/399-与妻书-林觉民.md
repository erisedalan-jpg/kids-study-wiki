# 与妻书-林觉民

> **一句话**：1911 年广州起义前夕，林觉民写给妻子陈意映的诀别书，将个人情感与家国大义融为一体，是近代书信文学的至情之作。
> **English**: Lin Juemin's farewell letter to his wife, written before the 1911 Guangzhou Uprising, embodies a fusion of personal love and patriotic devotion.

---

## 🎓 高中进阶

### 作者与背景

- 作者：林觉民（1887-1911），字意洞，福建闽侯人
- 时代：清末，辛亥革命前夕
- 写作时间：1911 年农历三月二十六日（公历 4 月 24 日），距广州起义仅几天
- 写作地点：香港滨江楼
- 收信人：妻子陈意映
- 林觉民牺牲时年仅 24 岁——同盟会革命党人

### 全文结构

| 部分 | 内容 |
|------|------|
| 起首 | "意映卿卿如晤"——亲切呼唤 |
| 心境 | "吾今以此书与汝永别矣！" |
| 矛盾 | 至爱与至义的冲突 |
| 大义 | 为天下人谋永福而舍小爱 |
| 回忆 | 与妻同居岁月、初恋情景 |
| 嘱托 | 嘱妻教育儿女、自珍自爱 |
| 结尾 | "吾辈处今日之中国，国中无地无时不可以死" |

### 核心情感与思想

```
小爱（夫妻情）+ 大爱（家国情）
     ↓
"吾爱汝至，所以为汝谋者惟恐未尽"
     ↓
"亦以天下人为念，当亦乐牺牲吾身与汝身之福利，为天下人谋永福也"
```

——以爱为出发点推及天下，是儒家"老吾老以及人之老"的革命转化。

### 名句

> 意映卿卿如晤：吾今以此书与汝永别矣。
> 吾作此书时，尚是世中一人；汝看此书时，吾已成为阴间一鬼。
> 汝幸而偶我，又何不幸而生今日之中国！
> 吾爱汝至，所以为汝谋者惟恐未尽。
> 吾辈处今日之中国，国中无地无时不可以死。
> 吾真真不能忘汝也！
> 当尽吾意为幸。

### 文学特色

| 特色 | 体现 |
|------|------|
| **情真意切** | 句句出自肺腑，至情至性 |
| **文白交融** | 文言为主，浅白易懂 |
| **抒情议论结合** | 有抒情有说理，有回忆有展望 |
| **比喻丰富** | "成为阴间一鬼" 等悲壮意象 |
| **节奏跌宕** | 大爱与小爱反复回旋 |
| **细节动人** | 回忆夫妻初见、私语等细节 |

### 字词

| 字 | 释义 |

| 如晤 (wù) | 如同当面 |

### 时代背景

- 清政府积弱，列强压迫
- 同盟会发动多次起义
- 1911 年广州起义（黄花岗起义）失败
- 林觉民等"黄花岗七十二烈士"之一
- 半年后辛亥革命成功，推翻清朝

### 与个人爱情的悲情

林觉民与陈意映：
- 表妹联姻，恩爱深厚
- 育有一子林依新
- 陈意映得书后陷入巨大悲痛，三年后病逝
- 留下幼子，由家人抚养成人

### 文学史地位

- 中国近代书信文学经典
- "至情之书"——可与《祭十二郎文》《情书》比美
- 被广泛收入语文教材
- 演绎为电影《辛亥革命》《与妻书》等

### 主题

| 主题 | 内容 |
|------|------|
| 革命牺牲 | 为国捐躯 |
| 夫妻深情 | 不舍至爱 |
| 大爱小爱辩证 | 由小推大 |
| 责任担当 | 知其必死而行 |
| 家国情怀 | 中国传统价值的现代展现 |

### 与同主题作品比较

| 作品 | 时代 | 主题 |
|------|------|------|
| 与妻书 | 清末 | 革命志士告别家人 |

### 易错点

- 林觉民牺牲时年 24 岁（不是中年）
- 陈意映得书后未能立即知晓，林觉民牺牲消息后才看到
- "意映卿卿如晤"是开篇——"卿卿"是亲昵
- 此文是革命前夕的诀别书，不是普通家书
- 黄花岗起义在广州（不是黄花岗某地名），是辛亥革命前的关键起义
