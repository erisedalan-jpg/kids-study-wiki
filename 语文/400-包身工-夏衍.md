# 包身工-夏衍

> **一句话**：夏衍 1936 年写的报告文学，揭露 1930 年代上海日资纱厂"包身工"制度的残酷剥削，是中国报告文学奠基作品之一。
> **English**: Xia Yan's reportage (1936) exposes the brutal exploitation of "bao shen gong" (indentured laborers) in 1930s Japanese-owned cotton mills in Shanghai.

---

## 🎓 高中进阶

### 作者与背景

- 作者：夏衍（1900-1995），原名沈端先，作家、剧作家
- 时代：1936 年发表
- 文体：**报告文学**——纪实+文学性的非虚构作品
- 调查方式：作者亲身潜入工厂调查
- 历史地位：中国报告文学开山之作之一

### "包身工"制度

包身工 = "包身雇用"的劳工：
- 工厂老板与"带工头"签合同——工人三年内一切归工厂
- 工人多为农村女童（13-16 岁），被骗或被卖
- 三年内吃住极差、劳动时长 12-15 小时
- 工资全归带工头，工人只得衣食
- 病死、伤亡概不负责
- 实质是**变相奴隶制**

### 文章结构

| 段 | 内容 |
|----|------|
| 起 | 4:15——晨起场面：肮脏、拥挤、起床、吃饭 |
| 描 | 包身工居住条件：闷臭、虫蚁、密集 |
| 工 | 工厂里的劳动：纺纱、轰鸣、监工 |
| 史 | 包身工制度的来历与发展 |
| 评 | 阶级分析与控诉 |
| 结 | "黑夜，静寂得像死一般的黑夜!但是，黎明的到来，毕竟是无法抗拒的" |

### 主要描写场景

#### 1. "芦柴棒"
- 一名瘦得像芦柴棒的女工，发烧仍被强迫上工
- 老板用冷水浇头逼起床
- 是包身工悲惨命运的缩影

#### 2. 起床场面
- 4:15 起床
- 一群女工挤在狭小空间
- 边吃稀饭边洗脸
- 充满拥挤、肮脏、苦难感

#### 3. 工厂监工
- 拿摩温（监工）随时鞭打
- "罚记""停生意"等手段
- 不许说话、不许休息

### 数据列举（报告文学特点）

文中大量真实数据：
- 工资统计
- 工时长度
- 死亡比率
- 工厂数量
- 与外国工人对比

### 写作手法

| 手法 | 体现 |
|------|------|
| **典型场面** | 4:15 晨起、芦柴棒生病等 |
| **细节描写** | 灰色臭气、虫子、汗珠 |
| **数据穿插** | 真实统计增加说服力 |
| **议论抒情** | 适时点评、控诉 |
| **对比手法** | 包身工 vs 一般工人；中国工人 vs 外国工人 |
| **比喻象征** | 黑夜——苦难；黎明——希望 |

### 名段

> 在那种生活里，有的是十六七岁的"包身工"，被当作"机器"一样地榨取。
> 黑夜，静寂得像死一般的黑夜！但是，黎明的到来，毕竟是无法抗拒的；索洛维茨基岛上的客人吗？
> 看着这种人间的地狱，我想起了一句俗话："劳工神圣"。
> （结尾）这种制度，也将随着帝国主义、资本家而一同消灭！

### 阶级分析

| 阶层 | 角色 |
|------|------|
| 日本资本家 | 工厂主，最大获利者 |
| 中国买办 | 协助管理 |
| 带工头 | 直接剥削者 |
| 拿摩温（监工） | 暴力执行者 |
| 包身工 | 受压迫者 |

### 报告文学特点

| 特点 | 说明 |
|------|------|
| **真实性** | 实地调查、数据真实 |
| **典型性** | 选取最典型场景、人物 |
| **文学性** | 形象化语言、修辞 |
| **思想性** | 反映社会问题、立场鲜明 |
| **时效性** | 紧扣当下问题 |

### 历史价值

- 揭露 30 年代殖民资本剥削
- 反映民国时期工人苦难
- 推动社会对劳工权益关注
- 后来包身工制度被废除（1949 年后）
- 中国报告文学的奠基作品

### 与现代意义

- 警示劳工权益
- 反思全球化下血汗工厂问题
- 对比当今工人权益进步

### 易错点

- "包身工"是制度名称，意为"包身雇用"
- 主要在日资纱厂——不限于中国资本家工厂
- 包身工多为女童——年纪小、易控制
- "芦柴棒"是具体女工外号——文学典型形象
- 报告文学≠新闻报道——既要真实又要文学化
