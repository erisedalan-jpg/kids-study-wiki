# 六国论-苏洵

> **一句话**：北宋苏洵借战国六国灭亡的历史教训，警示北宋朝廷不可向辽夏纳币求和，是史论文典范。
> **English**: Su Xun's "On the Six States" uses the lessons of the six states' fall to warn the Northern Song court against tribute-paying appeasement.

---

## 🎓 高中进阶

### 作者与背景

- 作者：苏洵（1009-1066），字明允，号老泉
- "三苏"之一（苏洵、苏轼、苏辙）；"唐宋八大家"之一
- 时代：北宋
- 现实背景：北宋向辽（契丹）、西夏纳币求和，苏洵借古讽今

### 中心论点

> **六国破灭，非兵不利，战不善，弊在赂秦。**

六国灭亡不是因兵力或战术差，而是因为**贿赂秦国**。

### 论证结构

| 部分 | 内容 |
|------|------|
| 提出论点 | "弊在赂秦" |
| 分论点一 | 赂秦者：韩、魏、楚——以地事秦如抱薪救火 |
| 分论点二 | 不赂秦者亦亡：燕、赵、齐——盖失强援 |
| 历史教训 | 六国如不贿秦合力抗秦，则可能不亡 |
| 现实警示 | "苟以天下之大，下而从六国破亡之故事"——警告北宋 |

### 核心论证手法

| 手法 | 体现 |
|------|------|
| **总分式结构** | 总论 → 分论赂者 → 分论不赂者 |
| **比喻** | "以地事秦，犹抱薪救火，薪不尽火不灭" |
| **对比** | 赂者 vs 不赂者；六国 vs 秦 |
| **史例** | 引韩魏楚燕赵齐具体事例 |
| **借古讽今** | 警示北宋切勿重蹈覆辙 |

### 名句

> 六国破灭，非兵不利，战不善，弊在赂秦。
> 赂秦而力亏，破灭之道也。
> 以地事秦，犹抱薪救火，薪不尽，火不灭。
> 苟以天下之大，下而从六国破亡之故事，是又在六国下矣。

### 史实背景（六国与秦）

| 国 | 与秦关系 |
|----|--------|
| 韩 | 多次割地求和 → 灭于前 230 |
| 魏 | 多次纳地 → 灭于前 225 |
| 楚 | 大国，最终灭于前 223 |
| 燕 | 远在北方，太子丹遣荆轲刺秦 → 灭于前 222 |
| 赵 | 抗秦最坚决，长平之战大败 → 灭于前 222 |
| 齐 | 远在东方，最后亡于前 221 |

### 古文运动与苏洵史论

| 项目 | 内容 |
|------|------|
| 风格 | 议论犀利，气势雄浑 |
| 特点 | 以史为镜，借古讽今 |
| 代表 | 《六国论》《管仲论》《辨奸论》 |
| 影响 | 苏家文风开启，苏轼苏辙发扬光大 |

### 与同名《六国论》比较

| 作者 | 朝代 | 中心论点 |
|------|------|--------|
| 苏洵 | 北宋 | 弊在赂秦 |
| 苏辙 | 北宋 | 不知韩魏之险 |
| 李桢 | 明 | 各国争霸自保 |

### 现实意义（北宋）

- 北宋年年输币给辽（澶渊之盟）、西夏（庆历和议）
- 苏洵以"赂秦"暗喻"赂辽""赂夏"——主张振兴自强
- 体现宋代士大夫的忧患意识

### 易错点

- "弊在赂秦"是中心论点，全文围绕展开
- "兵"指兵力或战争，不是兵器
- 苏洵不是苏轼父亲——其实就是苏轼之父（"老苏"）
- "故事"在文中是"旧事、先例"，不是 narrative
- 六国灭亡顺序：韩 → 魏 → 楚 → 燕 → 赵 → 齐
