# 答司马谏议书-王安石

> **一句话**：北宋王安石回复司马光反对变法的来信，逐条驳斥保守派指责，立场坚定、文风峻洁，是辩论性书信典范。
> **English**: Wang Anshi's reply to Sima Guang refutes the conservative critique of his reforms point by point, exemplifying argumentative epistolary writing.

---

## 🎓 高中进阶

### 作者与背景

- 作者：王安石（1021-1086），字介甫，号半山，"唐宋八大家"之一
- 时代：北宋熙宁变法时期（1070 年代）
- 收信人：司马光（《资治通鉴》作者，反对变法的保守派领袖）
- 写作背景：司马光致信批评变法，王安石作此文回复
- 文体：**书信**（议论性书信，回信）

### 司马光的指责（王安石概括）

司马光指责变法存在四大问题:

1. **侵官**：侵犯官员权力
2. **生事**：生出新事端
3. **征利**：与民争利
4. **拒谏**：拒绝批评

加上"以致天下怨谤"——天下人怨恨。

### 王安石的回应

逐条驳斥：

| 指责 | 驳斥 |
|------|------|
| 侵官 | 受命于人主，议法度而修之于朝廷，以授之于有司——非侵官 |
| 生事 | 举先王之政以兴利除弊——非生事 |
| 征利 | 为天下理财——非征利 |
| 拒谏 | 辟邪说，难壬人——非拒谏 |
| 怨谤 | 怨谤多由"流俗"，"士大夫多以不恤国事、同俗自媚于众为善" |

### 名句

> 受命于人主，议法度而修之于朝廷，以授之于有司，不为侵官。
> 举先王之政，以兴利除弊，不为生事。
> 为天下理财，不为征利。
> 辟邪说，难壬人，不为拒谏。
> 至于怨诽之多，则固前知其如此也。
> 人习于苟且非一日，士大夫多以不恤国事、同俗自媚于众为善。

### 论辩特色

| 特色 | 体现 |
|------|------|
| **直面问题** | 不避锋芒，直接回应每一项指责 |
| **以名实辨** | 强调"名实相符"——区分表象与本质 |
| **逻辑严谨** | 步步为营 |
| **语言简练** | 不枝不蔓，干净利落 |
| **态度坚定** | 不让步，不妥协 |
| **风骨刚劲** | 体现王安石"拗相公"性格 |

### 王安石变法（熙宁变法）

| 项目 | 内容 |
|------|------|
| 时间 | 1069-1085 |
| 主要内容 | 青苗法、募役法、方田均税法、保甲法等 |
| 目的 | 增国库收入、富国强兵 |
| 反对者 | 司马光、苏轼等保守派 |
| 结果 | 王安石两次罢相，新法终被废 |

### 王安石与司马光的关系

- 早年同朝为官，互相欣赏
- 政见不同后展开激烈辩论
- 但**保持私人尊重**——"君子和而不同"典范
- 司马光罢相后退居洛阳编《资治通鉴》
- 王安石罢相后退居金陵

### 文体特点：书信议论

| 项目 | 说明 |
|------|------|
| 形式 | 私人书信 |
| 内容 | 实为政论 |
| 称呼 | 尊称对方 + 自谦 |
| 结构 | 收信感谢 → 提炼对方观点 → 逐点驳斥 → 表达态度 |
| 优势 | 比公开辩论更直接、坦诚 |

### 与同时期议论文比较

| 篇目 | 文体 | 立论 |
|------|------|------|
| 答司马谏议书 | 书信议论 | 反驳保守派 |
| 谏太宗十思疏 | 奏疏 | 居安思危 |
| 六国论 | 史论 | 弊在赂秦 |
| 阿房宫赋 | 骈赋 | 各自取灭 |

### 思想价值

- 体现改革者的政治勇气
- "三不足"精神：天变不足畏、祖宗不足法、人言不足恤
- 为后世改革者所推崇

### 易错点

- "壬人"是巧言谄媚的人，不是普通"任性的人"
- "难"作动词读 nán，是责难
- 王安石变法目的并非"激进"，是为解决冗官冗兵冗费
- 司马光不仅是保守派，更是史学大家
- 此文是"回信"——形式上仍是书信，不是公开辩论
