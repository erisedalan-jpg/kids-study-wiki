# 谏太宗十思疏-魏征

> **一句话**：唐代魏征上唐太宗的奏疏，提出"居安思危、戒奢以俭"以及十条具体建议，是古代谏疏体的典范。
> **English**: Wei Zheng's memorial to Emperor Taizong proposing ten reflections to maintain peace through prudence and frugality, a model of remonstrative memorial.

---

## 🎓 高中进阶

### 作者与背景

- 作者：魏征（580-643），字玄成，唐初名臣
- 时代：唐贞观十一年（637）
- 接收人：唐太宗李世民（"贞观之治"明君）
- 文体：**奏疏**（臣子上君主的文章）
- 主题：劝太宗居安思危

### 中心论点

> **思国之安者，必积其德义。**

要使国家长治久安，必须积累德行与道义。

### 论证结构

| 部分 | 内容 |
|------|------|
| 起论 | 比喻——"求木长者必固其根本，欲流远者必浚其泉源" |
| 立论 | "思国之安者必积德义" |
| 反论 | 不积德义则"伐根而求木茂、塞源而欲流长" |
| 实例 | 历代帝王初勤而后惰 |
| 提议 | "十思" 具体建议 |
| 期待 | "总此十思，宏兹九德" |

### "十思"内容

1. 见可欲则**思知足以自戒**
2. 将有作则**思知止以安人**
3. 念高危则**思谦冲而自牧**
4. 惧满溢则**思江海下百川**
5. 乐盘游则**思三驱以为度**
6. 忧懈怠则**思慎始而敬终**
7. 虑壅蔽则**思虚心以纳下**
8. 想谗邪则**思正身以黜恶**
9. 恩所加则**思无因喜以谬赏**
10. 罚所及则**思无因怒而滥刑**

涵盖：欲望、行动、谦虚、纳谏、勤政、用人、赏罚等治国关键。

### 名句

> 求木之长者，必固其根本；欲流之远者，必浚其泉源；思国之安者，必积其德义。
> 善始者实繁，克终者盖寡。
> 居安思危，戒奢以俭。
> 载舟覆舟，所宜深慎。
> 简能而任之，择善而从之。

### 字词

| 字 | 释义 |
|----|------|
| 疏 (shū) | 奏疏，向君主陈述意见的文体 |
| 谏 | 规劝（下对上） |
| 浚 (jùn) | 疏通 |
| 殷 | 多、盛 |
| 鬲 (lì) | 古代炊器，喻分隔 |
| 牧 | 养、修养（自牧：自我修养）|
| 三驱 | 古代田猎礼仪——围三面留一面，喻有节制 |
| 黜 (chù) 恶 | 罢免恶人 |
| 简 | 选择 |
| 信 | 信用、诚信 |
| 殷忧 | 深忧 |

### 历史意义

| 影响 | 说明 |
|------|------|
| 当时 | 唐太宗读后赞叹，"震动以为枕"——常置案旁 |
| 治国 | 影响"贞观之治"治世形成 |
| 后世 | 中国古代奏疏文典范 |
| 思想 | 民本思想、居安思危精神 |

### 魏征其人

- 直言敢谏，"以人为镜，可以明得失"
- 唐太宗著名引用："夫以铜为镜，可以正衣冠；以古为镜，可以知兴替；以人为镜，可以明得失"——魏征即是太宗的"人镜"
- 唐贞观谏臣典范

### 唐太宗与谏臣

- "兼听则明，偏信则暗"是唐太宗常引魏征语
- 贞观一朝鼓励直言，故魏征敢"犯颜直谏"
- 建立"以谏为治"的政治文化

### 与同时期文章比较

| 篇目 | 作者 | 特点 |
|------|------|------|
| 《谏太宗十思疏》 | 魏征 | 奏疏，进谏 |
| 《答司马谏议书》 | 王安石 | 书信，回应反对意见 |
| 《阿房宫赋》 | 杜牧 | 赋，借古讽今 |
| 《过秦论》 | 贾谊 | 政论文，分析秦亡 |

### 思想体系

| 概念 | 含义 |
|------|------|
| 居安思危 | 平安时也要考虑危险 |
| 戒奢以俭 | 用节俭防止奢侈 |
| 载舟覆舟 | 民可承载政权也可推翻政权（民本） |
| 善始克终 | 既要好开始也要好坚持 |

### 易错点

- "疏"是"上书的文体"，读 shū；不是疏远
- "浚"读 jùn，疏通；不是 xún
- 魏征是直臣，唐太宗能纳谏成就盛世——是君臣合作典范
- "十思"不是抽象哲学，是十条具体修身治国要求
