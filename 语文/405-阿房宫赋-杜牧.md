# 阿房宫赋-杜牧

> **一句话**：唐代杜牧借秦阿房宫的奢华与覆灭警示当朝（唐敬宗）勿重蹈秦亡覆辙，是骈赋名篇与史论结合的典范。
> **English**: Du Mu's "Rhapsody on Epang Palace" uses the splendor and ruin of the Qin palace to warn the Tang court against luxurious indulgence.

---

## 🎓 高中进阶

### 作者与背景

- 作者：杜牧（803-852），字牧之，号樊川
- 与李商隐并称"小李杜"（区别于李白杜甫）
- 时代：唐代后期
- 写作背景：唐敬宗大兴土木，杜牧以秦事为鉴，写于宝历元年（825）

### 文体

**赋**：兼具诗与散文，本篇为**骈赋**（骈散结合，讲究对仗、用典、辞藻）。

### 全文结构

| 部分 | 内容 |
|------|------|
| 开篇 | "六王毕，四海一，蜀山兀，阿房出"——简洁有力的起笔 |
| 写宫之奢 | 楼阁、廊腰、宫女、藏珍 |
| 写宫之亡 | "戍卒叫，函谷举，楚人一炬，可怜焦土" |
| 议论作结 | "灭六国者六国也，非秦也""使六国各爱其人""秦人不暇自哀" |

### 名句

> 六王毕，四海一，蜀山兀，阿房出。
> 长桥卧波，未云何龙？复道行空，不霁何虹？
> 一肌一容，尽态极妍。
> 鼎铛玉石，金块珠砾，弃掷逦迤，秦人视之，亦不甚惜。
> 秦爱纷奢，人亦念其家。
> 灭六国者六国也，非秦也。族秦者秦也，非天下也。
> 后人哀之而不鉴之，亦使后人而复哀后人也。

### 主旨论点

苏洵之父之类立场：

> **灭六国者六国也，非秦也。族秦者秦也，非天下也。**

意思：六国灭亡不能怪秦，秦灭亡不能怪天下——是各自的腐败导致灭亡。

最后警告：

> **后人哀之而不鉴之，亦使后人而复哀后人也。**

——后人哀叹而不汲取教训，将让更后的人哀叹他们。

### 艺术特色

| 手法 | 例子 |
|------|------|
| **骈散结合** | 既有骈句对偶，又有散句议论 |
| **铺陈描写** | 极写阿房宫宏伟与奢华 |
| **对比反差** | 极尽繁华 → 顷刻焚毁 |
| **想象夸张** | "一日之内，一宫之间，而气候不齐" |
| **比喻丰富** | "未云何龙""不霁何虹" |
| **议论结尾** | 史论笔法，发人深省 |

### 与《六国论》对比

| 项目 | 阿房宫赋 | 六国论 |
|------|--------|------|
| 作者 | 杜牧（唐） | 苏洵（宋） |
| 文体 | 骈赋（铺陈+议论） | 议论文 |
| 立论 | 各国自取灭亡 | 弊在赂秦 |
| 风格 | 华丽 | 严密 |
| 共同 | 都借古讽今 | 借古讽今 |

### 借古讽今（唐敬宗）

- 唐敬宗大兴土木建造宫室
- 杜牧写赋讽喻：莫学秦始皇之奢
- 以阿房宫被烧的悲剧作警告

### 阿房宫史实

- 秦始皇 35 年（前 212）始建
- 项羽攻入咸阳后焚烧（前 206）
- 现代考古表明：实际未完工，焚烧规模也比文学描述小
- 但文学形象已成中国古典记忆中"奢华倾覆"的象征

### 易错点

- "阿"读 ē（不是 ā）
- "兀"读 wù，山秃貌
- "缦"在文中是缓慢、无边
- "鉴"是借鉴、引以为戒，不是镜子
- 阿房宫被项羽焚烧（不是刘邦），考证上规模也存争议
