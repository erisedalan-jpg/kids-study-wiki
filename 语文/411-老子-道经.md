# 老子·道经——道德经上篇核心章节

> **一句话**：《道经》（《道德经》前三十七章）以"道"为最高范畴，阐述宇宙本原、万物生成与人生处世的根本原则，是中国古代最具形而上高度的哲学文本。
> **English**: The "Dao Jing" (first 37 chapters of the Daodejing) presents "Dao" as the supreme metaphysical principle governing the cosmos, all things, and human conduct, forming the foundational text of Daoist philosophy.

---

## 🎓 老子·道经——道德经上篇核心章节

### 作者与学派

**老子**，姓李，名耳，字聃（dān），楚国苦县（今河南鹿邑）人，约生活于春秋末期（与孔子同时或略早）。**道家学派**创始人，著《道德经》（又称《老子》）八十一章，约五千言，是中国哲学史上最简短而影响最深远的经典之一。

道家以"道"（Dào）为宇宙本原，主张"无为""自然""柔弱胜刚强"，与儒家的"仁义礼"路线形成互补与对话。《道德经》分上篇（道经，1-37章）与下篇（德经，38-81章）。

### 篇章背景

老子生活于春秋末期礼崩乐坏的时代，但他对这一时代的回应与孔子截然不同：孔子要"克己复礼"，用道德规范重建秩序；老子则认为礼义本身是"道"失落的征兆，主张回归更根本的"道"——超越人为规范的宇宙本体与自然法则。《道经》的核心命题是：道是什么？道如何运作？人如何顺道而行？

### 原文（核心章节）

**第一章**
```
道可道，非常道；名可名，非常名。
无，名天地之始；有，名万物之母。
故常无，欲以观其妙；常有，欲以观其徼。
此两者，同出而异名，同谓之玄。
玄之又玄，众妙之门。
```

**第二章**
```
天下皆知美之为美，斯恶已；
皆知善之为善，斯不善已。
故有无相生，难易相成，长短相较，高下相倾，音声相和，前后相随。
是以圣人处无为之事，行不言之教。
```

**第八章**
```
上善若水。水善利万物而不争，处众人之所恶，故几于道。
居善地，心善渊，与善仁，言善信，正善治，事善能，动善时。
夫唯不争，故天下莫能与之争。
```

**第十六章**
```
致虚极，守静笃。
万物并作，吾以观复。
夫物芸芸，各复归其根。
归根曰静，是谓复命。
```

**第二十二章**
```
曲则全，枉则直，洼则盈，弊则新，少则得，多则惑。
是以圣人抱一为天下式。
不自见，故明；不自是，故彰；不自伐，故有功；不自矜，故长。
夫唯不争，故天下莫能与之争。
```

### 译文

**第一章**：可以用语言表达出来的道，不是永恒之道；可以命名的名，不是永恒之名。无，是天地的初始；有，是万物的根源。因此，常常以"无"的眼光去观察道的奥妙；常常以"有"的眼光去观察道的端倪。"无"与"有"同出一源而名称不同，都可以称之为"玄妙"。玄妙而又玄妙，是所有奥妙的总门。

**第二章**：天下人都知道美是美，丑的概念也随之产生；都知道善是善，不善的概念也随之产生。所以有和无互相生成，难和易互相促成，长和短互相比较，高和低互相依存，音与声互相和谐，前与后互相跟随。因此圣人用无为处理事务，用不言进行教化。

**第八章**：最高的善就像水一样。水善于滋润万物而不争高下，停留在众人不愿去的低下之处，所以最接近于道。（圣人）居处选择低下之地，心思深沉如渊，与人交往充满仁爱，说话诚实守信，为政善于治理，处事善于发挥能力，行动把握时机。正因为不争，所以天下没有什么能与之争。

**第十六章**：达到极度的虚空，守住极度的宁静。万物同时兴作，我因此观察它们的循环往复。万物茂盛纷纷，各自回归它们的根本。回归根本叫做静，这叫做回归本命。

**第二十二章**：弯曲方能保全，弯枉方能伸直，低洼方能盈满，破旧方能更新，少取方能有所得，多取反而迷惑。因此圣人把持"道"作为天下的准则。不自我显露，所以反而明白；不自以为是，所以反而彰显；不自我夸耀，所以反而有功；不自高自大，所以反而长久。正因为不争，所以天下没有什么能与之争。

### 思想内涵

**1. 道的不可言说性**（第一章）
"道可道，非常道"——道超越一切语言与概念，任何定义都会限制它、遮蔽它。老子对道的描述策略是悖论式的否定（what it is NOT）而非肯定性定义，体现了道家对语言局限性的深刻认识。"玄之又玄，众妙之门"是道家形而上学的最高表述：道是一切存在与变化的终极来源，无法被彻底认知，只能被体验与顺应。

**2. 相反相成：辩证法的萌芽**（第二章）
老子提出万物皆以对立面为存在条件（有无、难易、长短、高下……），这是中国古代辩证思维的最早系统表述。推论：不要执着于追求单一价值（美、善），因为它的对立面会随之生成；圣人的无为之治正是因为不强行推动，反而让万物自然展开。

**3. 上善若水：柔弱哲学**（第八章）
水的七种品德（居善地、心善渊……）是老子人格理想的具体描述：不争、谦处低位、柔弱却滋润万物。这与儒家的"刚健有为"形成鲜明对比——老子认为柔弱才是最终的胜者，"天下莫柔弱于水，而攻坚强者莫之能胜"。

**4. 归根复命：宇宙循环观**（第十六章）
万物的运动都有其循环规律：兴起→茂盛→归根→静→复命。认识这一规律（"知常"）是智慧的基础；不认识这一规律而盲目行动，则会招致灾祸（"不知常，妄作，凶"）。这是道家宇宙观的核心：万物皆循环，顺循环则安，逆循环则危。

**5. 曲则全：反向智慧**（第二十二章）
老子的"反向逻辑"是道经的显著特色：欲保全→先弯曲；欲充实→先虚空；欲长久→先不争。这一思维方式贯穿全书，与儒家"直"的进取路径相反，揭示了一种以退为进、以柔克刚的人生智慧。

### 文学手法

- **悖论式表达**：第一章开篇即用悖论（"道可道，非常道"），将语言自身的局限性作为命题，极具哲学震撼力。
- **排比列举**：第二章的对立概念排比（有无、难易、长短……），节奏整齐，如音乐般流动，在阐明哲学命题的同时具有文学美感。
- **比喻**："上善若水"将抽象的道德理想具象化为自然意象，是中国古典文学中最经典的比喻之一。
- **对称句式**："不自见，故明；不自是，故彰"——四个平行对称句，用"不……故……"的结构呈现老子的反向智慧，朗朗上口，深入人心。

### 高考考点

**必背默写**：
- "道可道，非常道；名可名，非常名。"
- "上善若水。水善利万物而不争，处众人之所恶，故几于道。"
- "曲则全，枉则直，洼则盈，弊则新，少则得，多则惑。"

**重点翻译**：
- "道可道，非常道"——注意三个"道"的词性（名词/动词/名词）
- "故几于道"——"几"（接近，不是"几乎"）
- "不自见，故明"——"见"（通"现"，显露）是通假字

**鉴赏角度**：
- 分析老子"道"的概念与儒家"仁"的概念的根本差异
- 探讨"上善若水"的处世哲学与儒家君子人格的异同
- 分析道经中辩证思维（相反相成）的哲学价值
