# 临安春雨初霁

> **一句话**：陆游在临安等待朝廷召见时写下此诗，以精致的春雨意象反衬内心的失落与愁苦——"小楼一夜听春雨，深巷明朝卖杏花"的意境之美，恰是词人报国无门之苦的反衬。
> **English**: Written while waiting idly in Lin'an for a court summons, Lu You's deceptively beautiful image of listening to spring rain all night and hearing flower vendors at dawn conceals deep frustration at his inability to serve his nation.

## 🎓 临安春雨初霁

### 作者生平

陆游（1125—1210），字务观，号放翁，越州山阴（今浙江绍兴）人。南宋著名诗人，一生创作诗歌近万首，是中国文学史上留存诗作最多的诗人之一。

陆游一生的主旋律是"恢复中原"的抱负。南宋偏安江南，而陆游始终主张北伐、收复失地，却屡遭投降派打压，壮志难酬。其诗风豪迈悲壮，以爱国情怀著称，代表作《示儿》《书愤》《秋夜将晓出篱门迎凉有感》等。

《临安春雨初霁》作于1186年，陆游62岁，奉召入京，在临安（今杭州）等候朝廷安排，却长期闲置。这首诗表面写春雨之美，实则满含寂寥与自嘲——一个有报国之志的老人，在京城的小楼里听了一夜春雨，能做的只是写写草书、泡泡茶。

### 原文

> 世味年来薄似纱，谁令骑马客京华。
> 小楼一夜听春雨，深巷明朝卖杏花。
> 矮纸斜行闲作草，晴窗细乳戏分茶。
> 素衣莫起风尘叹，犹及清明可到家。

### 注释

| 词语 | 注释 |
|------|------|
| 世味 | 世情，人情世故；此处指官场之味、人生滋味 |
| 薄似纱 | 淡薄如纱，意味索然 |
| 骑马客京华 | 骑马来到京城做客（谋官），带有自嘲意味 |
| 深巷 | 深幽的小巷 |
| 矮纸 | 短小的纸，小笺 |
| 斜行 | 斜着写，随意书写 |
| 闲作草 | 闲来无事练习草书 |
| 细乳 | 茶汤泛起的细腻泡沫，好茶的标志 |
| 分茶 | 一种茶艺，将茶汤注入碗中形成图案，是宋代雅好 |
| 素衣 | 洁白的衣服，比喻洁净的品格；亦实指白衣 |
| 风尘叹 | 感叹风尘劳碌，仕途污浊 |

### 译文

> 这几年来世情人情淡薄如纱，是谁让我骑马来到京城游荡？在小楼里听了一整夜的春雨，深巷中明天早晨就会有卖杏花的人了。在矮小的纸笺上斜斜地随手写几行草书，在晴朗的窗前细品茶汤里的泡沫、玩玩分茶的雅趣。不要感叹白衣染上了风尘，还赶得及在清明节前回到家。

### 文学赏析

**以乐景写哀情**

颔联"小楼一夜听春雨，深巷明朝卖杏花"是全诗最美的两句，也是最广为传诵的名句。春雨、杏花、深巷，是一幅极具美感的江南春日画卷。然而这两句恰恰是以乐景写哀情：诗人独守小楼，一夜无眠，听了一整夜的春雨声——这种美，恰是孤独与失落的映衬。

**世味薄的基调**

首联直接点出全诗的情感基调："世味年来薄似纱"——世情如此淡薄，人生如此无聊，我又为何要来京城呢？这是陆游对自己处境的深刻自嘲：他本是有大志向的爱国诗人，却在临安的小楼里虚度时光。

**颈联的细节与讽刺**

"矮纸斜行闲作草，晴窗细乳戏分茶"——这两句写的是诗人在临安的日常消遣：练草书、玩分茶。"闲"字和"戏"字是关键——用"闲"和"戏"来描述自己的活动，是一种深含苦涩的自嘲：一个本该为国效力的老人，只能"闲"着、"戏"着打发时光。

**结尾的淡然与无奈**

尾联"素衣莫起风尘叹，犹及清明可到家"，看似释然——别抱怨衣服脏了，清明前还能回家。但这种"释然"本身就是一种无奈：连对京城仕途的最后希望，也只是"赶紧回家"。这与陆游早年"上马击狂胡，下马草军书"的豪情形成了极大的反差。

### 高考考点

- **核心名句**：小楼一夜听春雨，深巷明朝卖杏花（意境描绘与情感分析）
- **艺术手法**：以乐景写哀情；虚实结合；反衬手法
- **情感主题**：壮志难酬的苦闷；对偏安朝廷的失望；表面潇洒实为无奈的心态
- **陆游诗歌风格**：与《书愤》《示儿》等豪迈之作对比，理解其多样性
- **宋诗特点**：以议论入诗；日常细节的意象化处理
