# 无衣

> **一句话**：《诗经·秦风》中的战歌，以三章叠唱形式写出秦国士兵同袍共甲、慷慨赴战的精神，是中国文学中最早也最雄浑的爱国主义诗篇之一。
> **English**: A battle hymn from the Qin section of the Book of Songs, depicting soldiers sharing weapons and armor, marching together against enemies — one of the earliest and most stirring expressions of patriotic solidarity in Chinese literature.

---

## 🎓 无衣

### 作者与背景

《无衣》出自《诗经·秦风》，创作年代约为春秋时期。《诗经》是中国最早的诗歌总集，收录西周初年至春秋中叶（约前11世纪—前6世纪）的诗歌305篇，分风、雅、颂三部分。

"秦风"为秦地（今陕西关中一带）民歌。秦国地处西陲，与戎狄常年交战，民风尚武雄健。《无衣》正是在这种背景下产生的从军歌谣，体现了秦地士兵同仇敌忾、共赴战场的集体意志。

关于创作背景，历代注家有多说：一说为秦哀公出兵救楚时的战歌；一说为秦人抵御北方戎狄入侵的征行之作。无论何说，诗中表达的集体主义精神与军旅情谊是明确的。

### 原文

> 岂曰无衣？与子同袍。王于兴师，修我戈矛。与子同仇！
>
> 岂曰无衣？与子同泽。王于兴师，修我矛戟。与子偕作！
>
> 岂曰无衣？与子同裳。王于兴师，修我甲兵。与子偕行！

### 注释

| 词语 | 注释 |
|------|------|
| 岂曰 | 怎能说，哪里说得上 |
| 袍 | 长衣，古代士兵战袍，亦有"同袍"引申为同伴、战友之义 |
| 王于兴师 | 王命出兵，兴，起；师，军队 |
| 修 | 整治，修缮 |
| 戈矛 | 古代武器，戈为横刃，矛为直刺 |
| 同仇 | 共同对敌，同仇敌忾 |
| 泽 | 内衣，贴身之衣 |
| 矛戟 | 戟，戈与矛合二为一的武器 |
| 偕作 | 一同行动，偕，共同 |
| 裳 | 下衣，裙状下裳 |
| 甲兵 | 铠甲与兵器 |
| 偕行 | 一同出征 |

### 译文

> 哪里说得上没有衣裳？我与你共穿一件战袍。王命下达要出兵，整治我们的戈和矛。我与你共同对敌！
>
> 哪里说得上没有衣裳？我与你共穿一件内衣。王命下达要出兵，整治我们的矛和戟。我与你一同奋起！
>
> 哪里说得上没有衣裳？我与你共穿一件下裳。王命下达要出兵，整治我们的铠甲兵器。我与你一同出征！

### 文学赏析

**叠章复沓的艺术结构**

全诗三章，结构完全平行，每章仅换数字和名词，形成"叠章复沓"的艺术手法——这是《诗经》最典型的表现手段。三章分别写"同袍""同泽""同裳"，从外衣到内衣到下裳，层层深入，象征战友情谊的亲密无间；武器也从"戈矛"到"矛戟"到"甲兵"，显示出整装待发的过程；行动从"同仇"到"偕作"到"偕行"，呈现出情绪由激发到行动到出征的递进。

**"岂曰无衣"的反问句式**

开篇以反问起句，语气强烈。士兵并非真的没有衣裳，而是表达：即使物资匮乏，我们也要共同面对。这种以反问驳斥消极情绪的方式，展现了豪迈的精神气质。

**集体主义与家国情感**

诗中"与子"二字贯穿始终，强调的是"我与你"、"我们"，是个体自愿投入集体的意志。士兵们并非被动受命，而是主动整备武器、共赴沙场。这种精神在中国文学史上奠定了"同袍"意象的文化内涵，"袍泽"一词（袍与泽皆为衣物）后来成为军中战友的代称，沿用至今。

### 高考考点

- **叠章复沓手法**：三章结构平行，逐章推进，掌握其艺术效果（增强气势、回环往复、深化情感）
- **"赋"的表现手法**：直陈其事，不用比兴，直接叙写共甲同袍的场景
- **重点字词**：袍、泽、裳（三种衣物），偕作、偕行（行动词），修（整治义）
- **思想主题**：同仇敌忾、团结御敌的集体主义精神；家国情怀
- **《诗经》常识**：风雅颂分类；赋比兴手法；四言为主的句式
