# 石钟山记-苏轼

> **一句话**：北宋苏轼亲访石钟山探究其得名原因，以游记形式表达"事不目见耳闻而臆断其有无"的求实精神。
> **English**: Su Shi visits Stone Bell Mountain to investigate the source of its name, advocating empirical inquiry over hearsay.

---

## 🎓 高中进阶

### 写作背景

- 作者：苏轼（1037-1101）
- 时间：宋神宗元丰七年（1084），苏轼由黄州贬所赴汝州
- 同行：长子苏迈（要送他去德兴）
- 地点：江西湖口石钟山（鄱阳湖入长江处）
- 写作目的：探究石钟山得名原因，并由此发议论

### 三种说法

苏轼记录三种关于石钟山命名的说法：

| 说法 | 提出者 | 内容 |
|------|------|------|
| 1. 风水击石如钟 | 郦道元 | 风浪冲击山石发出钟声 |
| 2. 石头敲击如钟 | 李渤 | 唐李渤敲击石头听到钟声 |
| 3. 石钟洞穴回响 | 苏轼亲访 | 山下洞穴中水流冲击有声 |

苏轼前两说不全否定，但认为前人没有亲自实地考察就下结论。

### 全文结构

| 段 | 内容 |
|----|------|
| 一 | 提出问题：石钟山何以得名？前人异说 |
| 二 | 苏轼亲访经历——"夜泊绝壁之下" |
| 三 | 详细描写夜探所见所闻——洞穴回响、水声叠加 |
| 四 | 议论：事不目见耳闻而臆断其有无可乎 |

### 中心论点

> **事不目见耳闻，而臆断其有无，可乎？**

凡事不亲眼看、亲耳听就主观判断有无，可以吗？——倡导**实地考察、实事求是**。

### 名句

> 事不目见耳闻，而臆断其有无，可乎？
> 至唐李渤始访其遗踪，得双石于潭上，扣而聆之，南声函胡，北音清越。
> 然是说也，余尤疑之。
> 大石侧立千尺，如猛兽奇鬼，森然欲搏人。
> 山上栖鹘，闻人声亦惊起。

### 实地考察的描写

苏轼夜泊石钟山下，详细记录所见所闻：

```
1. 大石侧立千尺，森然欲搏人
2. 鹘鸟惊起，磔磔云霄间
3. 山上又有怪声若钟磬
4. 中流水穴有大石——下空，与水相吞吐
5. 涵澹澎湃间发出噌吰之声、窾坎镗鞳之声
```

最终结论：石钟山下有水穴洞窟，水击有声 → 故名"石钟"。

### 论证手法

| 手法 | 体现 |
|------|------|
| **质疑前人** | 不盲从郦道元、李渤 |
| **亲身考察** | 夜泊探秘 |
| **感官描写** | 声、形、夜色 |
| **由实到议** | 先具体再抽象 |
| **反问结尾** | 增强说服力 |

### 文体特点：游记+议论

| 元素 | 体现 |
|------|------|
| 游记 | 行程、所见所闻 |
| 议论 | 由游记延伸到"目见耳闻"的求实精神 |
| 抒情 | 月夜江山的氛围 |
| 思辨 | 三种说法的比较 |

### 苏轼的"求实"精神

| 表现 | 说明 |
|------|------|
| 亲访 | 不在书斋臆测 |
| 不盲信 | 怀疑名家观点 |
| 实证 | 用感官观察 |
| 重视细节 | 各种声响细致辨认 |
| 谦逊 | 不全盘否定前人 |

体现宋代"理学"求知精神（虽苏轼不属理学派，但精神相通）。

### 与同类游记比较

| 篇目 | 作者 | 特点 |
|------|------|------|
| 赤壁赋 | 苏轼 | 抒情议论结合 |
| 登泰山记 | 姚鼐 | 简洁白描 |
| 石钟山记 | 苏轼 | 实证探究 |
| 游褒禅山记 | 王安石 | 借游说理 |

苏轼游记风格多样：赤壁赋抒情、石钟山记求实。

### 现代意义

- 科学求实精神
- 不盲从权威
- 实地调研重要性
- 理论联系实际
- 教学启示：鼓励学生亲身体验

### 易错点

- 石钟山在江西湖口，鄱阳湖入长江口
- 夜访时与儿子苏迈同行，不是独自
- 苏轼**不是**全盘否定前人，是认为前人没有实地考察
- "目见耳闻"——主张直接感官经验
- 现代地质学认为：石钟山命名可能多种因素综合
