# 种树郭橐驼传-柳宗元

> **一句话**：唐代柳宗元所作寓言性传记，借善种树的郭橐驼"顺木之天"的种树之道，讽喻为政者应"顺民之性"，少干扰民众生活。
> **English**: Liu Zongyuan's allegorical biography of Camel Guo, a master tree-planter who follows trees' nature; it implies that good governance similarly should follow people's nature without overreach.

---

## 🎓 高中进阶

### 作者与背景

- 作者：柳宗元（773-819），字子厚，唐代古文运动领袖之一
- 与韩愈并称"韩柳"
- "唐宋八大家"之一
- 时代：永贞革新失败后被贬永州（今湖南）

### 人物：郭橐驼

- "橐驼"即驼背
- 被人取的绰号，非真名
- 长安丰乐乡人
- 以种树为业，技艺超群
- 长安豪族都请他种植

### 全文结构

| 段 | 内容 |
|----|------|
| 一 | 介绍郭橐驼的形象与名字由来 |
| 二 | 写郭橐驼种树技艺无人能比 |
| 三 | 郭橐驼自述种树之道："顺木之天，以致其性焉尔" |
| 四 | 对比："他植者"违反树性，结果适得其反 |
| 五 | 由树及政——讽喻为政者扰民 |
| 六 | 总结："吾问养树，得养人术" |

### 中心思想

```
种树之道 → 治民之道
"顺木之天" → "顺民之性"
"勿动勿虑，去不复顾" → 政府应少干预
```

### 名句

> 顺木之天，以致其性焉尔。
> 凡植木之性，其本欲舒，其培欲平，其土欲故，其筑欲密。既然已，勿动勿虑，去不复顾。
> 他植者则不然：根拳而土易，其培之也，若不过焉则不及。
> 长人者好烦其令，若甚怜焉，而卒以祸之。
> 吾问养树，得养人术。

### 种树之道（顺木之天）

```
其本欲舒（让根伸展）
其培欲平（让土平整）
其土欲故（用原来的土）
其筑欲密（让土筑紧）
既然已 → 勿动勿虑 → 去不复顾
```

四要 + 三不要：充分尊重树的自然生长。

### "他植者"的错误

| 错误 | 后果 |
|------|------|
| 根拳 | 根弯曲不舒 |
| 土易 | 换新土，根不适应 |
| 培过 | 土堆太高 |
| 培不及 | 土太少 |
| 摇 | 摇晃树看是否生根 |
| 抓 | 抓土看湿度 |
| 旦视暮抚 | 早晚来看抚摸 |

→ **"虽曰爱之，其实害之；虽曰忧之，其实仇之"**

### 由树及政的寓意

| 树 | 政治 |
|----|------|
| 树有自然生长之性 | 民有自然生活之性 |
| 顺木之天 → 长得好 | 顺民之性 → 安居乐业 |
| 频繁干扰 → 树死 | 政令频繁 → 民疲 |
| 爱之实害之 | 怜之实祸之 |

### 名段（讽喻）

> 长人者好烦其令，若甚怜焉，而卒以祸之。
> 旦暮吏来而呼曰："官命促尔耕，勖尔植，督尔获，早缫而绪，早织而缕，字而幼孩，遂而鸡豚。"
> 鸣鼓而聚之，击木而召之。

——官吏频繁来催促百姓耕作织布，看似督促，实则扰民。

### 寓言文体特点

| 特点 | 说明 |
|------|------|
| 假托人物 | 借虚拟人物讲道理 |
| 双层意涵 | 表层种树，深层治民 |
| 具体可感 | 通过具体故事讲抽象道理 |
| 讽喻含蓄 | 不直白说，让读者体会 |
| 语言生动 | 描写细致 |

### 柳宗元的政治理想

- 永贞革新参与者，反对宦官藩镇
- 主张"顺民、薄赋、轻刑"
- 反对扰民政治
- 此文是其政治哲学的文学表达

### 道家无为思想的影响

| 来源 | 影响 |
|------|------|
| 《老子》 | 无为而治 |
| 《庄子》 | 顺应自然 |
| 黄老思想 | 减少政府干预 |

柳宗元虽不是道家，但此文吸收了道家"无为"智慧。

### 与同时期文章比较

| 篇目 | 作者 | 体裁 | 主题 |
|------|------|------|------|
| 师说 | 韩愈 | 论说文 | 倡从师 |
| 种树郭橐驼传 | 柳宗元 | 寓言传记 | 反对扰民 |
| 捕蛇者说 | 柳宗元 | 寓言议论 | 苛政猛于虎 |

柳宗元尤善寓言议论。

### 现代意义

- 教育：尊重儿童天性
- 管理：不过度干预
- 农业：尊重生物规律
- 政治：减政放权
- 个人：顺其自然

### 易错点

- 郭橐驼是寓言人物，可能基于真实的种树高手
- "橐驼"是绰号（驼背），不是名字
- "长人者"指治理百姓的官员，不是高个子
- "得养人术"——通过种树明白治民之道
- 是"传记体寓言"，不是普通游记或论说文
