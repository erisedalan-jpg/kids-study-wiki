# 人应当坚持正义-苏格拉底

> **一句话**：节选自柏拉图《克力同篇》，苏格拉底在受死刑前与朋友克力同的对话，论证为何即使面对不公的判决也应坚持正义、不越狱逃跑。
> **English**: Excerpted from Plato's Crito, Socrates dialogues with his friend Crito on why one must remain just and accept lawful punishment, even when unjustly condemned.

---

## 🎓 高中进阶

### 作者与背景

- 主人公：苏格拉底（Socrates, 前 469-前 399），古希腊哲学家
- 文本作者：柏拉图（Plato, 前 427-前 347），苏格拉底的学生
- 出处：《克力同篇》(Crito)
- 历史背景：公元前 399 年，苏格拉底因"不敬神明、败坏青年"被雅典法庭判死刑

### 故事背景

苏格拉底已被判死，等待执行。挚友克力同前来劝其越狱：
- 朋友们已准备好资金、船只
- 苏格拉底逃亡可保命
- 克力同认为判决不公，逃避是合理的

苏格拉底拒绝越狱，与克力同辩论"人应当坚持正义"。

### 苏格拉底的论证

#### 1. 不要看大众意见

> "应该听理性的指引，而不是听大多数人的意见。"

只有懂得正义的人的意见才值得听。

#### 2. 活得好不只是活着

> "生命中重要的不是活着，而是活得好。"

"活得好" = 高尚地活、正义地活。

#### 3. 以恶报恶不可

> "我们不应当以恶报恶——无论别人怎样对我们。"

这是苏格拉底道德哲学的核心：**永不行不义之事**。

#### 4. 人与城邦的契约

> "如果你出生在雅典、受其养育保护，就与城邦达成了无形契约。"

逃跑就是违背契约——是不义。

#### 5. 法律的拟人化

苏格拉底想象"雅典法律"前来质问他：
- 你受我们养育
- 我们给你机会离开（你年轻时未走）
- 你已默认契约
- 现在背叛我们就是不义

### 苏格拉底问答法（诘问法）

| 步骤 | 内容 |
|------|------|
| 1. 提出概念 | 例如"什么是正义" |
| 2. 听取对方定义 | 让对方先说 |
| 3. 提出反例 | 找漏洞 |
| 4. 引导承认 | 推出对方观点的矛盾 |
| 5. 重新定义 | 引出更深刻的理解 |

苏格拉底不直接给答案，引导对方自己发现真理 → 故称"思想助产术"。

### 名句

> 真正重要的不是活着，而是好好地活着。
> 我们任何时候都不应当行不义之事。
> 即使受了不正义的对待，也不应当以不正义回报。

### 哲学思想

| 概念 | 含义 |
|------|------|
| 知识即美德 | 真正知道善的人不会做恶 |
| 美德即知识 | 美德可以学习 |
| 灵魂高于肉体 | 精神价值高于肉体 |
| 公民契约 | 公民与城邦的隐性契约 |

### 与霍布斯、卢梭契约论比较

| 思想家 | 契约性质 |
|------|--------|
| 苏格拉底 | 默认契约（出生与受教育即默认） |
| 霍布斯 | 为避免战争立约 |
| 洛克 | 保护自然权利的契约 |
| 卢梭 | 公意的契约 |

苏格拉底是西方契约论思想的早期源头。

### 与文章主题

文章核心：
- 不能用不义的方法对抗不义
- 哲人应该坚守正义
- 个人与城邦的关系
- 灵魂高于肉体的生死观

### 苏格拉底之死的意义

- 哲学家因思想而死的最早典范
- 启发后世"为真理殉道"传统
- 体现西方哲学对正义、灵魂、生死的探讨
- 影响基督教殉道、近代殉道精神
- 哲学史上最重要的"事件"之一

### 与儒家"舍生取义"比较

| 项目 | 苏格拉底 | 孟子（舍生取义） |
|------|--------|--------------|
| 时代 | 前 399 | 前 4 世纪 |
| 选择 | 服法而死 | 危急时舍生 |
| 理由 | 不能行不义 | 义重于生 |
| 共同 | 道义 > 生命 | 道义 > 生命 |

——东西方在"道义"理解上有共同点。

### 哲学对话体特点

| 特点 | 说明 |
|------|------|
| 双人对话 | 苏格拉底 vs 克力同 |
| 步步逼近 | 通过提问引出结论 |
| 戏剧化 | 有人物、场景 |
| 思辨性 | 不靠权威，靠论证 |
| 口语化 | 不是学术论文 |

### 易错点

- 苏格拉底没有著作——他的思想由学生柏拉图、色诺芬记录
- 《克力同篇》是柏拉图所作，主人公是苏格拉底
- 苏格拉底之死是法定执行（饮鸩，不是被刺）
- 拒绝越狱是哲学选择，不是无能逃
- 公元前 399 年——比孔子（前 551-前 479）晚百余年
