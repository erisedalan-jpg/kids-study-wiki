# 修辞立其诚-张岱年

> **一句话**：当代哲学家张岱年所作学术论文，论述"修辞立其诚"——语言表达必须真实、坦荡、有学术诚信，是学术论文的典范。
> **English**: Zhang Dainian's essay argues that "rhetoric must establish sincerity" — language must be truthful, candid, and academically honest.

---

## 🎓 高中进阶

### 作者与背景

- 作者：张岱年（1909-2004），中国当代哲学家、思想史家
- 字季同
- 北京大学教授
- 中国哲学史与中西哲学比较研究权威
- 文体：**学术论文**

### 题目来源

"修辞立其诚"出自《周易·乾·文言》：

> 修辞立其诚，所以居业也。

意为：修饰文辞要立足于诚信，这是赖以立业的根本。

### 文章结构

| 段 | 内容 |
|----|------|
| 一 | 引出题目——"修辞立其诚"出处与解释 |
| 二 | "立其诚"包含三个层次 |
| 三 | "诚"的具体含义阐发 |
| 四 | 学术研究中"诚"的要求 |
| 五 | 反对欺骗、附会、虚浮 |
| 六 | 总结：诚是学术、修辞的根本 |

### 核心论点

> **修辞要立足于"诚"——内心真实、外言真实、做事真实。**

### "立其诚"三层次（文中提出）

1. **名实一致**：所表达的与客观事实一致
2. **言行一致**：所说的与所做的一致
3. **表里一致**：表面与内心一致

### 学术研究中的"诚"

| 要求 | 内容 |
|------|------|
| 客观 | 服从事实，不臆测 |
| 求真 | 追求真理，不附会权威 |
| 严谨 | 论证扎实，不哗众取宠 |
| 自省 | 经常反思自己 |
| 担当 | 敢说真话 |

### 名句

> 修辞立其诚——这是发言著论写文章的一个原则。
> "诚"是真，是实，是不欺骗。
> 学说必须是真实可信的，才能经得起时间的检验。
> 立其诚才能写出有价值的文字。

### 论证手法

| 手法 | 体现 |
|------|------|
| **引经据典** | 引《周易》、孔子、王充等 |
| **概念分析** | 分析"诚"的多层含义 |
| **正反对比** | 诚 vs 欺；真 vs 伪 |
| **举例论证** | 古今学者求真事例 |
| **逻辑推演** | 从"诚"推出学术准则 |

### "诚"的传统文化背景

| 出处 | 表达 |
|------|------|
| 《大学》 | 诚意正心 |
| 《中庸》 | 诚者天之道，诚之者人之道 |
| 《孟子》 | 反身而诚，乐莫大焉 |
| 朱熹 | 诚为天理之本然 |
| 张载 | 诚者天德 |

"诚"是儒家核心德目之一，与"仁、义、礼、智"并立。

### 当代学术意义

- 学术诚信问题日益重要
- 反对学术造假、抄袭、捏造数据
- 提倡"求真求实"的学术风气
- 与全球学术规范接轨

### 学术论文特点（本文体现）

| 特点 | 说明 |
|------|------|
| 论题明确 | 中心问题集中 |
| 概念准确 | 用词严谨 |
| 引证规范 | 引用经典与现代学者 |
| 论证周密 | 多角度论证 |
| 语言书面 | 用学术语言 |
| 结构清晰 | 总分总 |

### 张岱年的学术贡献

| 贡献 | 说明 |
|------|------|
| 中国哲学范畴史 | 系统梳理中国哲学概念 |
| 中西哲学比较 | 跨文化研究 |
| 文化综合创新论 | 主张"古今中外"创新结合 |
| 学术风骨 | 言行一致，知行合一 |

### 与同时期学术文章比较

| 文章 | 风格 |
|------|------|
| 修辞立其诚 | 哲学议论，引经据典 |
| 实践是检验真理 | 政论文体 |
| 乡土中国 | 社会学描述+理论 |

### 现代意义

学术诚信不仅是学者的事：
- 学生：作业不抄袭、考试不作弊
- 媒体：报道求真求实
- 公众：不传播虚假信息
- 公职：信息公开透明

### 易错点

- "修辞"不仅指修辞格，而是广义的语言表达
- "立其诚"——诚是出发点，不是后置结果
- 出自《周易》非《论语》
- 张岱年是哲学家不是文学家
- 文章是学术论文，重在概念辨析与论证，不是抒情
