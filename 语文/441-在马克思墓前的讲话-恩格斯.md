# 在马克思墓前的讲话-恩格斯

> **一句话**：1883 年 3 月 17 日，恩格斯在伦敦海格特公墓为挚友马克思所作的悼词，简洁评价马克思在科学和革命事业上的伟大贡献。
> **English**: Engels' eulogy delivered at Karl Marx's funeral on March 17, 1883, summarizing Marx's contributions to science and the workers' movement.

---

## 🎓 高中进阶

### 演讲者与背景

- 演讲者：弗里德里希·恩格斯（Friedrich Engels, 1820-1895）
- 时间：1883 年 3 月 17 日
- 地点：英国伦敦海格特公墓
- 对象：参加葬礼的友人（约十余人）
- 文体：**悼词** / **演讲稿**
- 用语：原文为英语（部分德文），后译多语种

### 全文结构

| 段 | 内容 |
|----|------|
| 一 | 以低调宣告噩耗：3 月 14 日下午两点三刻——他停止思想了 |
| 二 | 损失的双重性：对科学和革命都是巨大损失 |
| 三 | 第一大贡献：发现人类历史的发展规律（唯物史观） |
| 四 | 第二大贡献：发现现代资本主义的运动规律（剩余价值） |
| 五 | 其他领域的研究 |
| 六 | 马克思对科学的态度："科学是一种在历史上起推动作用的、革命的力量" |
| 七 | 马克思的革命家身份：组织国际工人协会 |
| 八 | 总结：他的英名和事业将永垂不朽 |

### 核心评价（恩格斯的两大发现总结）

| 发现 | 内容 |
|------|------|
| **唯物史观** | 历史不是英雄主导，而是经济基础决定上层建筑 |
| **剩余价值** | 资本主义剥削机制——揭示了资本家利润来源 |

恩格斯将这两个发现比作达尔文发现自然规律，强调马克思对人类历史的同等重要意义。

### 名句

> 3 月 14 日下午两点三刻，当代最伟大的思想家停止思想了。
> 正像达尔文发现有机界的发展规律一样，马克思发现了人类历史的发展规律。
> 任何一门理论科学中的每一个新发现——它的实际应用也许还根本无法预见——都使马克思感到衷心喜悦。
> 他可能有过许多敌人，但未必有一个私敌。
> 他的英名和事业将永垂不朽。

### 演讲艺术

| 特色 | 体现 |
|------|------|
| **简洁有力** | 全篇仅约 800 字 |
| **逻辑层次** | 由小到大、由具体到抽象 |
| **抒情节制** | 不渲染、不夸张，平静中显深情 |
| **对比手法** | 思想 vs 实践、科学 vs 革命、敌人 vs 私敌 |
| **比喻精当** | 类比达尔文 |
| **总结提升** | 结尾一句永垂不朽 |

### 悼词文体特点

| 特点 | 说明 |
|------|------|
| 简短精练 | 时间紧、空间小 |
| 庄重深情 | 哀而不伤 |
| 评价中肯 | 突出主要贡献 |
| 抒议结合 | 既叙又议 |
| 仪式性 | 服务于葬礼场合 |

### 字词与表达

| 词 | 含义 |
|----|------|
| 思想家 | 强调思想的伟大 |
| 停止思想了 | 委婉说"去世"——表达对思想的尊重 |
| 为人类发现 | 不是为某一阶级，而是为整个人类 |
| 私敌 | 私人间结仇的敌人——区别于政治对手 |
| 永垂不朽 | 永远流传不会消失 |

### 马克思（1818-1883）简要

- 德国犹太裔思想家、革命家
- 与恩格斯共同创立马克思主义
- 主要著作：《资本论》《共产党宣言》《1844 年经济学哲学手稿》
- 长期居住伦敦，潦倒中坚持研究
- 1864 年组织国际工人协会（第一国际）

### 马克思与恩格斯的友谊

- 1844 年相识，从此 40 年深厚友谊
- 恩格斯多次资助马克思家庭
- 共同写作《共产党宣言》(1848)
- 恩格斯整理出版《资本论》第二、三卷
- 恩格斯称："马克思的天才比我们都大得多"

### 演讲地点：海格特公墓

- 伦敦著名公墓
- 马克思墓至今仍是世界各地访客的瞻仰之地
- 墓碑上刻马克思头像与"全世界无产者，联合起来"

### 学习重点（语文角度）

1. **议论性悼词**——从演讲到议论的过渡
2. **简洁的语言艺术**——80 行约 800 字
3. **对比与比喻**——增强说服力
4. **情理交融**——既理性评价又有真情
5. **结构化思维**——总分总，先理论后实践

### 易错点

- 这是悼词，不是马克思自传或马克思主义教程
- "他停止思想了"——不是普通"去世"，是修辞性表达，强调思想的伟大
- 马克思去世于 1883 年；恩格斯于 1895 年——恩格斯独立活了 12 年
- "永垂不朽"指事业精神，不仅是身体不朽
- 演讲在英国，但马克思是德国人
