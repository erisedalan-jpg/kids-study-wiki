# 怜悯是人的天性-卢梭

> **一句话**：节选自法国思想家卢梭《论人类不平等的起源和基础》（1755），论述怜悯是人类与生俱来的自然情感，是社会道德的基础。
> **English**: Excerpted from Rousseau's Discourse on the Origin of Inequality (1755), this passage argues that pity is an innate, natural sentiment foundational to human morality.

---

## 🎓 高中进阶

### 作者与背景

- 作者：让-雅克·卢梭（Jean-Jacques Rousseau, 1712-1778）
- 国籍：法国（生于日内瓦）
- 时代：18 世纪欧洲启蒙运动
- 哲学派别：启蒙思想家、自然法理论家
- 主要著作：《社会契约论》《爱弥儿》《忏悔录》《论人类不平等的起源和基础》

### 文章出处

- 《论人类不平等的起源和基础》（1755）
- 卢梭应第戎学院征文而作（虽未获奖却影响深远）
- 探讨：人类社会不平等如何产生

### 核心观点

> **怜悯是人的天性，先于一切思考，独立于道德教化。**

卢梭认为：
1. **人类有两个先于理性的自然情感**：自爱（自我保护）+ 怜悯（不忍他人受苦）
2. **怜悯是普遍的**：不仅人类有，动物也有（如母兽护幼）
3. **怜悯先于反思**：是本能不是道理推出的
4. **怜悯是道德的基础**：仁慈、宽恕、慷慨等都源于怜悯

### 文章结构

| 段 | 内容 |
|----|------|
| 一 | 提出"怜悯"是自然情感 |
| 二 | 与霍布斯辩论：人非天生残忍 |
| 三 | 怜悯的普遍性——人和动物 |
| 四 | 怜悯是道德的基础 |
| 五 | 反思社会如何削弱了怜悯 |

### 与霍布斯的论争

| 观点 | 霍布斯 | 卢梭 |
|------|------|------|
| 人的本性 | 自私、好斗、"狼对狼" | 善良、有怜悯心 |
| 自然状态 | 人对人的战争 | 自由平等的纯朴 |
| 社会形成 | 为避战争而立约 | 社会反而带来不平等 |

卢梭明确反对霍布斯"人性恶"。

### 名句

> 怜悯是一种天然的感情，先于一切思考。
> 我们之所以心疼别人，是因为我们设身处地。
> 强者亦曾是弱者，谁也免不了苦难。
> 自然把同情心给了人类，作为他们的安慰。

### 论证手法

| 手法 | 体现 |
|------|------|
| **观察论证** | 观察动物行为推出 |
| **对比论证** | 与霍布斯观点对比 |
| **设身处地** | 引导读者代入受苦者 |
| **反例论证** | 即使最坏的人见到悲惨也会动恻隐之心 |
| **理性推演** | 从自然法推论 |

### 与孟子"恻隐之心"比较

孟子（约前 372-前 289）和卢梭（1712-1778）相隔 2000 多年，但提出极相似观点：

| 项目 | 孟子 | 卢梭 |
|------|----|----|
| 表述 | 恻隐之心，仁之端也 | 怜悯是道德的基础 |
| 例证 | 孺子将入井 | 见同类受苦即心痛 |
| 性质 | 先验的善端 | 自然情感 |
| 普遍性 | 人皆有之 | 人和动物共有 |

——这种跨文化共鸣体现了"人性"的某种普遍性。

### 启蒙思想中的位置

| 思想家 | 主张 |
|------|------|
| 霍布斯 | 人性自私，需强政府 |
| 洛克 | 人有自然权利、政府由契约产生 |
| 卢梭 | 人本善，社会扭曲了人性 |
| 伏尔泰 | 理性、宗教宽容 |
| 康德 | 道德是理性的命令 |

卢梭是启蒙运动中"自然主义"代表，影响法国大革命与浪漫主义。

### 哲学内涵

| 概念 | 含义 |
|------|------|
| 自然状态 | 人在社会形成前的状态 |
| 自爱 (amour de soi) | 自我保存的本能 |
| 自尊 (amour-propre) | 与他人比较的虚荣（卢梭批判此） |
| 怜悯 (pitié) | 对他人苦难的同情 |
| 道德感 | 怜悯发展为道德 |

### 哲学随笔特点

| 特点 | 说明 |
|------|------|
| 主题集中 | 聚焦"怜悯"概念 |
| 论证开放 | 不强势灌输 |
| 语言形象 | 用比喻、举例 |
| 个人立场 | 作者声音明显 |
| 启发性强 | 引发读者思考 |

### 现代意义

- 心理学：共情研究证实怜悯/共情是普遍人类情感
- 伦理学：移情伦理学的根据
- 社会学：互助、慈善的心理基础
- 教育：培养孩子的同理心
- 动物伦理：动物也有情感的论据

### 易错点

- 卢梭是法国启蒙思想家，不是浪漫派文学家（虽影响浪漫主义）
- "怜悯"是先于理性的，不是文化教化产物
- 卢梭主张回归自然，并非反对一切社会
- 自爱（amour de soi）不是贬义自私，是生存本能
- 与孟子的"恻隐之心"虽相似，是不同文化独立形成
