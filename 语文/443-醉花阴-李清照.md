# 醉花阴-李清照

> **一句话**：北宋李清照所作重阳词，借秋日相思抒发独居思夫之情，"莫道不销魂，帘卷西风，人比黄花瘦"为千古名句。
> **English**: Li Qingzhao's ci poem on the Double Ninth Festival, evoking longing for her absent husband; the closing line "Don't say my soul is not pining; behind the curtain rolled by westwind, I'm thinner than the yellow chrysanthemum" became a classic.

---

## 🎓 高中进阶

### 作者与背景

- 作者：李清照（1084-约 1155），号易安居士
- 时代：北宋末南宋初
- 婉约词派代表
- 与丈夫赵明诚两地分居时所作（一说赵明诚出守在外）

### 全词

```
薄雾浓云愁永昼，瑞脑销金兽。
佳节又重阳，玉枕纱厨，半夜凉初透。

东篱把酒黄昏后，有暗香盈袖。
莫道不销魂，帘卷西风，人比黄花瘦。
```

### 词牌"醉花阴"

- 双调五十二字
- 上下片各五句三仄韵
- 适合表达细腻含蓄情感

### 字词

| 字 | 释义 |
|----|------|
| 永昼 | 漫长的白天 |
| 瑞脑 | 一种香料 |
| 金兽 | 兽形铜香炉 |
| 重阳 | 农历九月初九重阳节 |
| 纱厨 | 罩着轻纱的床帐 |
| 东篱 | 东边篱笆，借指赏菊处（陶渊明"采菊东篱下"） |
| 暗香 | 幽香（菊花香） |
| 盈袖 | 充满衣袖 |
| 销魂 | 形容极度悲伤或感动 |
| 黄花 | 菊花 |

### 全词赏析

#### 上阕：白昼孤独

> 薄雾浓云愁永昼，瑞脑销金兽。

- 时间：从早到晚的漫长白昼
- 环境：阴云薄雾、香炉静燃
- 情绪：愁、孤独
- 手法：以景衬情

> 佳节又重阳，玉枕纱厨，半夜凉初透。

- 节令：重阳节（古代家人团聚之节）
- "又"字含义：又一年，更显时光流逝、独居寂寞
- "凉初透"：写身体感觉的凉，更写心境的凉

#### 下阕：黄昏赏菊

> 东篱把酒黄昏后，有暗香盈袖。

- 化用陶渊明"采菊东篱下"
- 黄昏独酌、清香环绕
- 表面闲适，实则更添孤独

> 莫道不销魂，帘卷西风，人比黄花瘦。

——千古名句：

| 句 | 含义 |
|----|------|
| 莫道不销魂 | 不要说我没有失魂落魄 |
| 帘卷西风 | 西风吹起帘幕（凄凉氛围） |
| 人比黄花瘦 | 我比菊花还要瘦弱（思念之苦） |

### 三大艺术特色

#### 1. 含蓄委婉

不直说思念，借时令、景物、动作暗示——典型婉约手法。

#### 2. 移情入景

每一景物都浸透情感：
- 薄雾浓云 → 心境
- 玉枕纱厨 → 孤眠
- 暗香盈袖 → 寂寞
- 黄花瘦 → 自喻

#### 3. 比喻新奇

"人比黄花瘦"——以菊花比人，前所未有；
菊花本来纤细，更突出"瘦"的极致。

### 著名典故（赵明诚的回应）

赵明诚收到此词后欲胜过妻子，闭门三日写五十首词，混入此词请友人陆德夫品评。陆德夫读后说："只三句绝佳。"问哪三句，陆答："莫道不销魂，帘卷西风，人比黄花瘦。"——此典见明代《琅嬛记》。

### 李清照的创作分期

| 期 | 作品风格 | 代表作 |
|----|--------|------|
| 早期（北宋） | 闺情、轻快 | 如梦令《常记溪亭日暮》 |
| 中期（汴京、青州） | 思夫、雅致 | 醉花阴 |
| 晚期（南渡后） | 国破家亡、沉痛 | 声声慢、夏日绝句 |

醉花阴属中期作品。

### 与晚期《声声慢》对比

| 项目 | 醉花阴 | 声声慢 |
|------|------|------|
| 时期 | 中期 | 晚期 |
| 情感 | 思夫之愁 | 国破丧夫之痛 |
| 风格 | 含蓄轻愁 | 沉郁深哀 |
| 名句 | 人比黄花瘦 | 这次第，怎一个愁字了得 |

### 婉约词派风格

| 特点 | 体现 |
|------|------|
| 题材 | 闺情、相思、咏物 |
| 风格 | 含蓄、细腻、婉约 |
| 语言 | 雅致、易于吟咏 |
| 代表 | 柳永、李清照、晏殊、秦观 |

### 现代意义

- 中国古典爱情诗的代表作
- 女性视角的细腻情感表达
- 婉约词的最高典范之一
- 千古传诵的相思之词

### 易错点

- 李清照与赵明诚是夫妻恩爱典范
- "黄花"是菊花，不是黄色花朵
- "东篱"用陶渊明典，借代赏菊处
- "销魂"在文中是极度感动悲愁，不是"销蚀魂魄"
- 李清照晚年生活凄凉，与中期不同
