# 大同-礼记

> **一句话**：选自《礼记·礼运》，描绘儒家"大同社会"的理想——天下为公、人尽其才、老有所终、幼有所养，是中国古代乌托邦思想的源头。
> **English**: From Liji (Book of Rites), this passage envisions the Confucian "Great Unity" — a utopian society where the world is shared, all are cared for, and harmony prevails.

---

## 🎓 高中进阶

### 出处

- 《礼记·礼运》篇
- 假托孔子之言（实为后世儒生整理）
- 《礼记》：儒家经典"五经"之一（诗、书、礼、易、春秋）
- 编纂时期：战国至西汉

### 篇章结构

孔子参加蜡祭（年终祭祀）后叹息，子游问其故。孔子答：

```
昔者大道之行也（理想的"大同"社会）
今大道既隐（退化的"小康"社会）
↓
对比中突出"大同"理想
```

### 大同社会描述（核心段落）

> 大道之行也，天下为公。
> 选贤与能，讲信修睦。
> 故人不独亲其亲，不独子其子；
> 使老有所终，壮有所用，幼有所长，矜寡孤独废疾者皆有所养。
> 男有分，女有归。
> 货恶其弃于地也，不必藏于己；
> 力恶其不出于身也，不必为己。
> 是故谋闭而不兴，盗窃乱贼而不作，故外户而不闭。
> 是谓大同。

### 大同社会的特征

| 特征 | 含义 |
|------|------|
| 天下为公 | 政权、财富不属一家一姓 |
| 选贤与能 | 任人唯贤 |
| 讲信修睦 | 诚信、和睦 |
| 不独亲其亲 | 视他人长辈如己亲 |
| 老有所终 | 老人受赡养 |
| 壮有所用 | 壮年有事可做 |
| 幼有所长 | 儿童得到培育 |
| 矜寡孤独皆有所养 | 弱势群体受保障 |
| 男有分 | 男子有职业 |
| 女有归 | 女子有归宿 |
| 货不藏于己 | 资源共享 |
| 力不为己 | 劳力为公 |
| 谋闭盗息 | 阴谋盗贼皆消失 |
| 外户不闭 | 夜不闭户 |

### 小康社会

与大同对比，小康社会：

> 今大道既隐，天下为家。
> 各亲其亲，各子其子，货力为己。
> 大人世及以为礼，城郭沟池以为固。
> 礼义以为纪——以正君臣，以笃父子，以睦兄弟，以和夫妇。
> 设制度，以立田里，以贤勇知，以功为己。

——以"家"为基础，以"礼义"维系，是较"大同"低一阶但仍可向往的现实秩序。

### 论证手法

| 手法 | 体现 |
|------|------|
| **对比** | 大同 vs 小康 |

| **借古抒怀** | 假托孔子之言 |

| 价值 | 体现 |
|------|------|
| 理想社会 | 中国古代乌托邦思想 |
| 民本思想 | 关怀弱势群体 |
| 公共精神 | 货力为公 |
| 平等观 | 不独亲其亲 |

### 历史影响

| 时代 | 影响 |
|------|------|
| 古代 | 历代士大夫的政治理想 |
| 近代 | 康有为《大同书》体系化 |
| 孙中山 | "天下为公" 成为革命理想 |
| 现代 | "全面小康""共同富裕"等概念回响 |
| 国际 | 中国古代乌托邦思想代表 |

### 与西方乌托邦比较

| 项目 | 大同 | 西方乌托邦 |
|------|----|---------|
| 时代 | 战国-西汉 | 文艺复兴后 |
| 代表 | 礼记·礼运 | 莫尔《乌托邦》 |
| 焦点 | 社会和谐 | 制度设计 |
| 实现 | 理想，难以实现 | 同 |
| 共同 | 描绘理想社会 | 同 |

### 现代意义

- 关怀弱势群体的福利国家理念
- 公共财产、社会保障思想
- 全球治理"人类命运共同体"理念
- 儒家"仁"的社会化展开

### 易错点

- 大同篇出自《礼记》，不是《论语》
- "天下为公" 不是某一时代实现的，是理想模式
- 与"小康"是阶段性差异，不是对立的
- 假托孔子之言——是儒家学者整理形成
- 蜡祭是年终祭祀，不是腊八节
