# 离骚-屈原

> **一句话**：战国楚国诗人屈原创作的长篇政治抒情诗，是中国文学史上最早的浪漫主义诗篇之一，开创"楚辞"体裁。
> **English**: A long political-lyric poem by the Chu poet Qu Yuan; one of the earliest masterpieces of romantic poetry in Chinese literature, founding the Chuci tradition.

---

## 🎓 高中进阶

### 作者与背景

- 作者：屈原（约前 340-前 278），名平，字原，又字灵均
- 时代：战国晚期楚国
- 政治地位：楚怀王时任左徒，掌内政外交
- 写作背景：被靳尚等小人谗言而疏远后所作（一说先后两次写）
- 体裁：**楚辞**（以《离骚》为代表）

### 题目含义

"离骚"——离别的忧愁，或"遭遇忧愁"。司马迁《史记》："离骚者，犹离忧也。"

### 全篇梗概（共 372 句）

| 部分 | 内容 |
|------|------|
| 一 | 自叙身世——宗族高贵、生辰祥瑞、修养德行 |
| 二 | 期望辅佐君王——但小人当道，自己被疏 |
| 三 | 上求圣明、下访贤能——皆不见容 |
| 四 | 神游天地——求宓妃、佚女、有娀氏，皆不得 |
| 五 | 灵氛占卜——劝远走他乡 |
| 六 | 国乡眷恋——终决意以死殉国 |

### 教材节选（多为前 14 章左右）

包括：
- 自叙世系
- 修德待时
- 担忧楚国
- 进谏被斥
- 比兴香草

### 名句

> 帝高阳之苗裔兮，朕皇考曰伯庸。
> 路漫漫其修远兮，吾将上下而求索。
> 长太息以掩涕兮，哀民生之多艰。
> 亦余心之所善兮，虽九死其犹未悔。
> 制芰荷以为衣兮，集芙蓉以为裳。
> 余独好修以为常。

### 写作艺术

| 特色 | 体现 |
|------|------|
| **香草美人** | 以香草喻君子，美人喻贤君 |
| **比兴手法** | 借物起兴 |
| **想象奇丽** | 上天入地、求神问卦 |
| **感情浓烈** | 忠愤、忧国、哀民 |
| **反复吟咏** | "兮"字句式 |
| **修辞华丽** | 用词典雅 |
| **铺陈** | 大量典故与意象 |

### "兮"字句式

楚辞独特句式特征：
- 句中或句末加"兮"
- 节奏舒缓，便于咏叹
- 例：路漫漫其修远兮，吾将上下而求索

### 香草美人传统

| 香草 | 喻意 |
|------|------|
| 江离、白芷、秋兰 | 高洁品德 |
| 木兰、宿莽 | 节操不变 |
| 芰荷、芙蓉 | 自身修养 |
| 萧艾、茅 | 小人卑劣 |

| 美人 | 喻意 |
|------|------|
| 美人 | 贤明君主 |
| 灵均 | 屈原自称 |

后世"美人香草"成为政治抒情诗的传统。

### 楚辞与诗经对比

| 项目 | 诗经 | 楚辞 |
|------|----|------|
| 时代 | 西周-春秋 | 战国-汉初 |
| 地域 | 黄河流域 | 长江流域 |
| 体裁 | 四言为主 | 句式自由，多"兮" |
| 风格 | 现实质朴 | 浪漫奇丽 |
| 代表 | 风、雅、颂 | 离骚、九歌 |
| 性质 | 集体创作的民歌 + 文人诗 | 个人创作 |

| 作品 | 内容 |

| 《九章》 | 9 篇短抒情诗 |
| 《天问》 | 哲学诗 |
| 《招魂》 | 招亡灵 |

### 文学史地位

- 中国文学最早的伟大诗人
- 浪漫主义诗歌的源头
- "离骚"与"诗经"并称中国诗歌两大源头
- 影响汉赋、唐诗、宋词
- 端午节与屈原投江相关
- 1953 年世界和平理事会列屈原为世界四大文化名人之一

### 易错点

- 屈原是战国时期楚国人，不是"汉代诗人"
- "朕"是先秦人称代词，不是皇帝专称
- 离骚不是"离别的歌"，是"遭遇忧愁"
- "美人"在文中喻贤君，不是真实女性
- 投汨罗江——汨罗在今湖南，不是长江
