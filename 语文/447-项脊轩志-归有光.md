# 项脊轩志-归有光

> **一句话**：明代归有光所作回忆性散文，借家族老屋项脊轩寄寓对祖母、母亲、妻子的怀念，是"以小物寄深情"的散文典范。
> **English**: Gui Youguang's nostalgic essay centers on his family chamber Xiangjixuan, evoking memories of his grandmother, mother, and wife.

---

## 🎓 高中进阶

### 作者与背景

- 作者：归有光（1507-1571），字熙甫，号震川
- 时代：明代中期
- 学派：唐宋派（明代复古派一支，反对"前后七子"，崇尚唐宋古文）
- 文风：朴素自然、以情动人

### 项脊轩

- 归有光家族在江苏昆山的老屋
- "项脊"或与归氏先祖任太仓某官有关
- 归有光自小在此读书
- 后修缮成自己的书房

### 全文结构

| 部分 | 内容 |
|------|------|
| 起 | 写项脊轩的修缮：旧时狭小破败 → 修后清雅可读 |
| 承 | 写轩中读书生活的乐趣 |
| 转 | 写家族变迁：诸父分家、阶前老树、一鸡犬之声相闻 |
| 起 | 写祖母对自己的期待："此吾家瑞也" |
| 转 | 写母亲早逝——通过老仆人转述母亲之言 |
| 结 | 写妻子去世——"庭有枇杷树，吾妻死之年所手植也，今已亭亭如盖矣" |

后段"妻死"部分是文末补写（妻王氏去世后，归有光重读旧文，加写此段）。

### 名句

> 项脊轩，旧南阁子也。室仅方丈，可容一人居。
> 又杂植兰桂竹木于庭，旧时栏楯，亦遂增胜。
> 借书满架，偃仰啸歌，冥然兀坐，万籁有声。
> 而庭阶寂寂，小鸟时来啄食，人至不去。
> 三五之夜，明月半墙，桂影斑驳，风移影动，珊珊可爱。
> 庭有枇杷树，吾妻死之年所手植也，今已亭亭如盖矣。

### 写作艺术

| 特色 | 体现 |
|------|------|
| **以小见大** | 项脊轩之小寄家族之变 |
| **细节传神** | 老鼠偷吃、小鸟啄食、月光斑驳 |
| **情感真挚** | 对祖母、母亲、妻子的怀念深切 |
| **白描手法** | 不饰华藻，朴素自然 |
| **物是人非** | 通过老物件勾起回忆 |
| **结尾余韵** | "亭亭如盖"四字成永恒名句 |

### 三位女性形象

| 人物 | 描写 | 情感 |
|------|------|------|
| 祖母 | 来视吾，曰"此吾家瑞也" | 期待 |
| 母亲 | 老仆转述："儿寒乎？欲食乎？" | 关爱 |
| 妻子 | 时至轩中、问古事 | 恩爱 |

通过细节侧写——不直接抒情而情自溢。

### 末段赏析（妻死）

```
余既为此志，后五年，吾妻来归。
时至轩中，从余问古事，或凭几学书。
......
其后六年，吾妻死，室坏不修。
其后二年，余久卧病无聊，乃使人复葺南阁子，其制稍异于前。
然自后余多在外，不常居。

庭有枇杷树，吾妻死之年所手植也，今已亭亭如盖矣。
```

最后一句结尾极富艺术张力：
- "枇杷树"是当年妻子手植
- "亭亭如盖"——树已长成
- 时间流逝、人去物存
- 哀而不伤，余韵悠长

### 文学价值

- 中国"散文小品"的典范
- "回忆体"散文的代表
- 影响后世"小说化散文"——细节、人物、情感
- 与《祭十二郎文》（韩愈）、《陈情表》（李密）并称"至情之文"

### 与陈情表比较

| 项目 | 陈情表 | 项脊轩志 |
|------|------|--------|
| 时代 | 西晋 | 明代 |
| 体裁 | 奏表 | 散文 |
| 主题 | 孝 | 怀人 |
| 情感 | 厚重悲切 | 平淡含蓄 |
| 共同 | 至情至性 | 至情至性 |

### 易错点

- 项脊轩是归有光自家的书房，不是亭子或寺院
- "项脊"可能与归氏先祖太仓某官的"项脊径"有关
- 末段"枇杷树"是妻子王氏手植，归有光后来补写
- 归有光属"唐宋派"——反对当时"前后七子"复古秦汉
- "亭亭如盖"——形容树高大，盖如车盖
