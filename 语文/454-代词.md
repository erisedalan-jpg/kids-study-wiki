# 代词

> **一句话**：代词是代替名词、数量词或其他词语的词，让句子不重复啰嗦，说话更简洁流畅。
> **English**: A pronoun is a word that replaces a noun or noun phrase, helping sentences flow smoothly without repetition.

---

## 🧒 给 3-6 岁（共读版）

如果每次说话都要说名字，会很奇怪！

"小明去上学。小明背着书包。小明走进教室。小明坐下来……"

说了太多"小明"，听起来好啰嗦！

所以我们用"**他**"来代替"小明"：

"小明去上学。**他**背着书包。**他**走进教室。**他**坐下来……"

这个"他"就是**代词**——代替名字的词！

**我们常用的代词：**
- 说自己 → **我**
- 说对方 → **你**
- 说别人 → **他**（男）/ **她**（女）/ **它**（动物/事物）
- 指东西 → **这**（近）/ **那**（远）
- 问问题 → **谁**？**什么**？**哪里**？

### 共读小活动：代词替换游戏

讲一个小故事，把所有的名字换成"他/她/它"，看看是不是更顺口了！然后问孩子："'他'指的是谁？"——练习理解代词所指。

## 📚 给 6-12 岁（自读版）

### 什么是代词

代词（pronoun）是**代替名词**、数量词或其他成分的词，让句子不重复，表达更简洁。

> 小红很喜欢画画。**她**每天都练习。（"她"代替"小红"）
> 我看到了一只小猫。**它**的眼睛很大。（"它"代替"小猫"）

### 代词的三大类型

#### 第一类：人称代词

表示人或事物"是谁"，是用得最多的代词。

**单数人称代词：**

| 人称 | 代词 | 英文对应 |
|------|------|---------|
| 第一人称（说话的人） | 我 | I / me |
| 第二人称（听话的人） | 你 / 您（尊称） | you |
| 第三人称（男性） | 他 | he / him |
| 第三人称（女性） | 她 | she / her |
| 第三人称（非人类） | 它 | it |

**复数人称代词（加"们"）：**

| 代词 | 英文对应 |
|------|---------|
| 我们 | we / us |
| 你们 | you (plural) |
| 他们 / 她们 / 它们 | they / them |

**注意**："我们"可以是说话者和对方（包含对方），也可以是说话者和其他人（不包含对方）。有些方言里区分这两种，普通话书面语不区分。

**"您"的用法**：对长辈、老师、陌生人表示尊重时用"您"代替"你"。
> 您好！请问您贵姓？

#### 第二类：指示代词

指向距离的远近或特定事物。

| 代词 | 距离 | 用法 |
|------|------|------|
| 这 | 近（指眼前） | 这本书 / 这是我的笔 |
| 那 | 远（指远处） | 那朵花 / 那是什么？ |
| 这里 / 这儿 | 近处地点 | 在这里等我 |
| 那里 / 那儿 | 远处地点 | 去那里玩 |
| 这样 / 这么 | 近指方式 | 这样做就对了 |
| 那样 / 那么 | 远指方式 | 别那么难过 |

#### 第三类：疑问代词

用来提问，询问人、事、地点、原因等。

| 疑问代词 | 提问内容 | 例句 |
|---------|---------|------|
| 谁 | 人 | **谁**来了？ |
| 什么 | 事物/内容 | 你叫**什么**名字？ |
| 哪 / 哪里 / 哪儿 | 选择/地点 | 你去**哪里**？**哪**本书是你的？ |
| 为什么 | 原因 | 你**为什么**哭？ |
| 怎么 / 怎样 | 方式/状态 | **怎么**做？感觉**怎样**？ |
| 几 | 小数量（10以下） | 你有**几**个苹果？ |
| 多少 | 数量（大小均可） | 这里有**多少**人？ |

**疑问代词还可以表示"任指"**（不具体指某一个）：

> **谁**都可以来。（任何人）
> **什么**都好，随便。（任何东西）
> **哪里**都行。（任何地方）

### 代词的语法注意点

1. **"他/她/它"发音相同**（都读 tā），书写时区分；
2. **"你们"包括一个或多个人**，汉语无专门的"dual"（双数）形式；
3. **疑问代词不一定在句首**：你去**哪里**？（比较英语：**Where** are you going?）

## 🎓 给 12+（进阶版）

### 古汉语代词系统

文言文中的代词系统与现代汉语不同，有丰富的区分。

**第一人称（我）：**

| 古代代词 | 说明 | 出处例 |
|---------|------|--------|
| 吾 | 最常用第一人称 | **吾**日三省**吾**身（论语） |
| 我 | 亦常用，含强调色彩 | **我**有嘉宾（诗经） |
| 余 / 予 | 书面语，较正式 | **余**将有所选择（左传） |
| 朕 | 秦前泛用，秦后专指皇帝 | **朕**闻之（史记） |
| 孤 / 寡人 | 国君自称 | **寡人**有疾（孟子） |

**第二人称（你）：**

| 古代代词 | 说明 | 例 |
|---------|------|---|
| 汝 / 女 | 最常用第二人称 | **汝**知之乎？ |
| 尔 | 略带轻蔑或亲密 | **尔**辈不能究物理（梦溪笔谈） |
| 若 | 你（战国常用） | **若**为吾使（史记） |
| 君 | 尊称，译为"您" | 君子不器（论语）|

**第三人称（他）：**

文言文无专门第三人称代词，通常用：
- **之**：宾格（动词后）：爱之、知之
- **其**：定语（名词前）、主格：其人、其言
- **彼**：那个人，略有轻蔑

> 知**之**者不如好**之**者，好**之**者不如乐**之**者。（论语）
> ——"之"反复代指"学问/道"

### 中英代词系统对比

| 特点 | 汉语代词 | 英语代词 |
|------|---------|---------|
| 格变化 | 无（我=I=me，形式不变） | 主格/宾格：I/me, he/him, they/them |
| 性别区分 | 写法区分 他/她/它（发音相同） | 读音区分 he/she/it |
| 复数形式 | 加"们" | 独立词：we, they |
| 尊称 | 您（比你尊重） | 无专门尊称（均用 you） |
| 反身代词 | 自己（我自己/他自己） | myself/himself/herself |
| 关系代词 | 较少系统化 | who, which, that 关系从句 |

### 代词回指：语篇衔接的核心机制

代词的主要作用是**回指**（anaphora）——指向前文出现过的名词。

> 小明把作业交给**老师**，**老师**表扬了小明，小明很高兴。
> → 小明把作业交给**老师**，**她**表扬了小明，**他**很高兴。（用代词避免重复）

回指的歧义性：
> **张三**告诉**李四**，**他**考了100分。
> ——"他"指张三还是李四？上下文不明确时会产生歧义。

语言学上用"回指链"（reference chain）分析代词在语篇中的功能，是现代语言学语篇分析的基础概念。

## 🗣️ 家长讲解话术

- 孩子可能问 **"他/她/它有什么区别？"** → 可以说："他/她/它发音都一样（tā），但写法不同。'他'是男性，'她'是女性，'它'是物品、动物等非人类。说话时听不出区别，写字时要看对象是什么。"

- 孩子可能问 **"为什么'谁'也是代词？"** → 可以说："'谁'代替了一个还不知道是谁的人——就像空格，等着被填写。问'谁来了？'，'谁'这个词代替了那个还不知道名字的人，所以它也是代词。"

- **共读小活动 1 — 代词替换**：讲一段故事，每次听到名字，孩子来说应该用什么代词替换——"小明"用"他"，"小红"用"她"，"小猫"用"它"。

- **共读小活动 2 — 疑问代词造句**：给孩子一张疑问代词清单（谁/什么/哪里/为什么/怎么），轮流用每个词造一个问句，再让对方回答。
