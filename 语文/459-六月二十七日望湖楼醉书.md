# 六月二十七日望湖楼醉书

> **一句话**：苏轼在望湖楼上看西湖夏雨，用四句诗把云、雨、风、水的瞬息变化写得如电影般生动。
> **English**: Su Shi watches a summer rainstorm over West Lake from Wanghu Tower — in four lines he captures the rolling clouds, leaping raindrops, sudden wind, and mirror-still water with cinematic vividness.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：宋 · 苏轼
**体裁**：七言绝句

```
六月二十七日望湖楼醉书（其一）
苏轼

黑云翻墨未遮山，
白雨跳珠乱入船。
卷地风来忽吹散，
望湖楼下水如天。
```

**拼音**：
- hēi yún fān mò wèi zhē shān，
- bái yǔ tiào zhū luàn rù chuán。
- juǎn dì fēng lái hū chuī sàn，
- wàng hú lóu xià shuǐ rú tiān。

**白话译文**：
黑云像打翻的墨汁翻滚而来，还没有遮住山，
白色的雨点像跳动的珍珠，乱溅进船里。
忽然卷地而来的大风，把云和雨都吹散了，
望湖楼下，湖面平静得像天空一样（风雨后水镜无痕）。

**朗读节奏**：
黑云 / 翻墨 / 未遮山，/ 白雨 / 跳珠 / 乱入船。
卷地 / 风来 / 忽吹散，/ 望湖楼 / 下水 / 如天。

## 🧒 给 3-6 岁（共读版）

你在夏天有没有看过突然来的大雨？乌云滚滚，大雨哗哗，风一来，忽然又停了——

苏轼有一次坐在一座楼上，看着下面西湖上的雨，他写了这首诗：

- 黑云翻墨：乌云翻滚过来，像打翻了一瓶墨水！
- 白雨跳珠：大雨一颗颗从天上跳下来，像珍珠蹦进船里！
- 卷地风来忽吹散：一阵大风来了，把云和雨都吹跑了！
- 水如天：雨停了，湖面平平静静的，像一面镜子，照出了天空！

是不是就像看了一部关于下雨的小电影？

**共读小活动**：
- **找找形容**：诗里把黑云比作什么？（打翻的墨）把雨比作什么？（跳珠/珍珠）
- **演一演**：用手表演黑云翻滚来、大雨哗哗下、风来吹散的过程！

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 望湖楼 | wàng hú lóu | 杭州西湖边的一座楼 |
| 醉书 | zuì shū | 喝酒后写的诗（"醉"代表心情放松，不一定真的醉） |
| 翻墨 | fān mò | 像打翻了墨汁（比喻黑云翻滚） |
| 未遮山 | wèi zhē shān | 还没有遮住山 |
| 白雨 | bái yǔ | 大雨（雨点看起来白白的） |
| 跳珠 | tiào zhū | 像珍珠一样跳动（雨点打在船上弹跳的样子）|
| 乱入船 | luàn rù chuán | 乱七八糟地打进船里 |
| 卷地风 | juǎn dì fēng | 席卷大地的大风 |
| 忽 | hū | 忽然，突然 |
| 水如天 | shuǐ rú tiān | 水面平静如同天空（雨后水镜）|

### 内容与情感

这首诗是一场夏雨的"直播"，按时间顺序展开：

1. **第一句（乌云来）**：黑云翻滚，像打翻的墨，但山还没被遮住——雨前的紧张感
2. **第二句（大雨下）**：雨点跳珠般打进船——雨来了，大而急
3. **第三句（风吹散）**：大风卷地而来，忽然把云雨都吹散——气象的急剧变化
4. **第四句（雨后静）**：湖面平静如天——雨过天晴，如此平静

四句诗，一场雨，从黑云到暴雨到大风到平静，完整记录了一场夏日急雨的全过程。

### 艺术特色

- **比喻精准**："翻墨"写黑云的颜色和翻滚感；"跳珠"写雨点弹跳的视觉感，极为生动
- **动词精准**："翻""跳""乱""卷""吹"——五个动词构建出动态画面
- **节奏节奏**：前三句急促，末句骤然平静，节奏模拟了雨的变化过程
- **"忽"字**：突然、迅速，写出夏雨来去之快

## 🎓 给 12+（进阶版）

### 作者与写作背景

**苏轼**（1037—1101），字子瞻，号东坡居士，眉州眉山（今四川眉山）人，北宋文学家、书画家，"唐宋八大家"之一，词学豪放派代表人物。

这首诗写于1073年（熙宁六年），苏轼任杭州通判期间，在西湖旁的望湖楼宴饮后观雨所作，当日作五首，这是其一（最广为流传）。题目"醉书"暗示了心情的放松与即兴——诗人在酒后眼前一亮，立刻捕捉到西湖夏雨的瞬间变化。

苏轼与杭州有深厚渊源：他两度出任杭州，修筑了西湖的"苏堤"，成为杭州历史的重要人物。关于西湖，他还有著名的《饮湖上初晴后雨》（"欲把西湖比西子，淡妆浓抹总相宜"）。

### 物理现象的诗化表达

"黑云翻墨未遮山"与"望湖楼下水如天"形成了有趣的科学与诗学对话：
- 夏日对流雨（雷阵雨）来去极快，云层发展迅速，这正是"翻墨未遮山"的大气现象
- 雨过后气压变化，风来吹散云层，水面重新平静——这是热带气旋式阵雨的典型特征
- "水如天"利用了水面反射天空的物理原理，雨后湖面无波，如同镜面，倒映蓝天

这首诗是中国古诗中对自然现象把握最精准的作品之一——苏轼的观察力与表达力在此完美结合。

### 与《饮湖上初晴后雨》对读

苏轼写西湖的两首名诗：
- 《饮湖上初晴后雨》：以西子（西施）比西湖，重在意境与比喻，显示西湖的多面之美
- 《六月二十七日望湖楼醉书》：以电影式的动态描绘，重在写实与动感，展现夏雨的急速变化

两首合读，可见苏轼写景的两种路径：一是以文化联想丰富意象，一是以精准观察捕捉现象。

## 🗣️ 家长讲解话术

- 孩子可能问 **"翻墨是什么意思？"** → 可以说："你有没有见过一瓶墨水打翻了，黑色的墨水哗地扩散开来？诗人说黑云来的样子就像这样——黑色的云翻滚过来，扩散开来，一下子就把天空染黑了。这个比喻是不是很形象？"

- 孩子可能问 **"跳珠是什么？"** → 可以说："'跳珠'是跳动的珍珠。大雨落在船上、水面上，雨点弹起来，一颗颗，白白的，圆圆的——就像珍珠在跳！你下次看大雨时，仔细看看雨点打在地上是不是真的会弹起来！"

- 孩子可能问 **"水如天是什么感觉？"** → 可以说："大风把雨和云都吹走了，湖面又恢复平静，一动不动，就像镜子一样，把蓝天白云都倒映在水里——从楼上看下去，不知道哪个是天，哪个是水。"

- **共读小活动（观雨记录）**：下次遇到夏日阵雨，带孩子一起观察：云怎么来的？雨点怎么落？风来了吗？雨停后水面怎么样？然后和苏轼的诗一起对照，看诗人写的是不是真实的。
