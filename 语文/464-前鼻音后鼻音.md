# 前鼻音后鼻音

> **一句话**：前鼻音（-n结尾）舌尖触碰上齿龈，后鼻音（-ng结尾）舌根抬起，两者在普通话中区分词义，是重要的发音难点。
> **English**: Mandarin distinguishes syllables ending in an alveolar nasal coda -n (anterior nasal) from those ending in a velar nasal coda -ng (posterior nasal), a contrast that many dialects and foreign learners find challenging.

---

## 🧒 给 3-6 岁（共读版）

宝贝，我们来感受两种不同的"鼻子音"！

**前鼻音**：说完主音后，舌尖轻轻顶上去——感受到鼻子里有一股气收进去了，像说"嗯"一样，舌头在前面。

**后鼻音**：说完主音后，舌根（舌头后面）抬起来——感受到嗓子深处有气从鼻子出来，像说"嗯"时嘴巴半开着，舌头在后面。

**儿歌（捏鼻子感受）：**

> **前鼻音** an en in un ün——舌尖顶前方，"银"字收进去！
> **后鼻音** ang eng ing ong——舌根抬起来，"英"字嗡出来！
> 分不清楚怎么办？**捏住鼻子**说一说——捏着说不出来的，就是鼻音！

**共读小活动：** 家长说"银/英"，孩子用手捏住鼻子感受——"银 yín"（前鼻音，捏住会觉得更憋）和"英 yīng"（后鼻音，嘴巴稍开，捏住会不同感觉）。

## 📚 给 6-12 岁（自读版）

### 前鼻音（-n 结尾）完整列表

| 韵母 | 代表字 | 常见词 |
|------|--------|--------|
| an | 安 ān / 暗 àn | 安全、暗夜 |
| en | 恩 ēn / 很 hěn | 恩情、很好 |
| in | 音 yīn / 心 xīn | 音乐、心里 |
| un | 温 wēn / 村 cūn | 温暖、村子 |
| ün | 晕 yūn / 云 yún | 晕倒、云朵 |

发音关键：**舌尖抵住上齿龈**（上门牙后面的牙床），气流从鼻腔流出，嘴唇微闭。

### 后鼻音（-ng 结尾）完整列表

| 韵母 | 代表字 | 常见词 |
|------|--------|--------|
| ang | 昂 áng / 唱 chàng | 高昂、唱歌 |
| eng | 能 néng / 冷 lěng | 能力、冷天 |
| ing | 英 yīng / 明 míng | 英雄、明天 |
| ong | 用 yòng / 通 tōng | 用功、通过 |

发音关键：**舌根抬起抵住软腭**（口腔后部柔软处），气流从鼻腔流出，嘴巴微开。

### 易混对比（最重要的记忆材料）

| 前鼻音 | | 后鼻音 | | 区分口诀 |
|--------|---|--------|---|---------|
| 银 yín | （银子） | 英 yīng | （英雄） | 银是金属，英是英雄 |
| 心 xīn | （心脏） | 星 xīng | （星星） | 心里有愿望，星星在天上 |
| 亲 qīn | （亲人） | 青 qīng | （青色） | 亲情是人，青是颜色 |
| 任 rèn | （任务） | 仍 réng | （仍然） | 义字旁任务，人字旁仍然 |
| 深 shēn | （深浅） | 生 shēng | （生命） | 深有水，生有字意不同 |
| 婚 hūn | （结婚） | 轰 hōng | （轰炸） | 婚礼人，轰炸炮 |

### 发音检测方法

**方法1：捏鼻子法**
- 说前鼻音（如"银 yín"）：捏鼻子时能感受到鼻腔被堵住，有憋气感
- 说后鼻音（如"英 yīng"）：捏鼻子时嘴巴是开的，憋气感方向不同
- （注意：两种音捏鼻都会受影响，关键是感受舌头位置的不同）

**方法2：镜子前看嘴形**
- 前鼻音：嘴唇在发音结尾时微微闭合
- 后鼻音：嘴巴保持微开，不完全闭合

**方法3：感受舌头位置**
- 说完主要元音后，注意舌头：
  - 前鼻音：舌尖向前顶（顶上门牙后面）
  - 后鼻音：舌头后部向上（感受喉咙深处）

### 方言影响

**容易前后鼻不分的地区：**
- 湖南、四川、重庆、江西：方言中前后鼻音合并，学普通话需专项练习
- 广东：粤语无 -ng 韵尾（如粤语"生"读 sang，无真正的 -ng）

**前后鼻音分得清的地区：**
- 北京、河北、山东、东北：普通话母语区，区分自然

## 🎓 给 12+（进阶版）

### 语音学分析

**前鼻音 -n 的IPA：**
- -n = /n/，舌尖齿龈鼻音（alveolar nasal）
- 发音时：舌尖接触上齿龈，软腭下降，气流从鼻腔通过
- 配合元音：an /an/，en /ən/，in /in/，un /un/，ün /yn/

**后鼻音 -ng 的IPA：**
- -ng = /ŋ/，舌根软腭鼻音（velar nasal）
- 发音时：舌根抬起接触软腭，软腭下降，气流从鼻腔通过
- 配合元音：ang /aŋ/，eng /əŋ/，ing /iŋ/，ong /ʊŋ/

**注意**：普通话的 -ng 只出现在韵尾，不能单独做声母（不像粤语有 ng- 声母）。

### 与英语鼻音的对比

| 特征 | 普通话 | 英语 |
|------|--------|------|
| /n/ 韵尾 | 有（-n 系列韵母） | 有（sun, then, man） |
| /ŋ/ 韵尾 | 有（-ng 系列韵母） | 有（ring, song, king） |
| /m/ 韵尾 | **无**（古汉语有，已并入-n） | 有（room, dream） |
| /ŋ/ 声母 | **无**（拼音不含 ng 声母） | 无（英语也无） |
| 粤语特有 | ng- 声母（如"我"ngoh） | 无 |

**英语学习者的相关优势：**
学过英语的人对 -n 和 -ng 韵尾区别已有感知（"sin"vs"sing"），这个区别直接迁移到普通话学习。

### 历史演变：-m 韵尾的消失

中古汉语有三种鼻音韵尾：-m、-n、-ng，现代普通话只保留了 -n 和 -ng，-m 并入了 -n：

| 中古韵尾 | 现代普通话 | 保留方言 |
|---------|----------|---------|
| -m | 合并入 -n | 粤语、闽南语保留（如粤语"心"sam） |
| -n | 保留为 -n | 普通话 |
| -ng | 保留为 -ng | 普通话 |

因此普通话"心 xīn"的祖先在中古汉语是 -m 结尾（siəm），而粤语"心"至今仍读 sam，保留了-m 韵尾的痕迹。

### 音变规律（变调相关）

前后鼻音音节通常不引发特殊变调，但有些相关规律：
- "一"在前鼻音节前：yì tiān（一天），yì nián（一年）——因天/年均为前鼻音节，"一"变四声（见[[474-四声|四声]]变调规则）
- "不"前接后鼻音节无特殊变调：bù néng（不能），bù tōng（不通）

### 普通话水平测试（PSC）中的考察

PSC 测试中，前后鼻音是朗读单字题的常见考点：
- 单字题专项测试"an/ang、en/eng、in/ing、un/ong"等混淆点
- 二级甲等及以上要求前后鼻音准确无误
- 二级乙等允许有少量不稳定

## 🗣️ 家长讲解话术

- 孩子可能问 **"前鼻音和后鼻音有什么区别，感觉都差不多？"** → 可以说："你试试说'银'和'英'——'银'说完后舌头跑到前面去了，'英'说完后舌头后部抬起来了，嘴形也不一样。最简单的方法：说完后，嘴能闭上的是前鼻音（-n），嘴半开着的是后鼻音（-ng）。"

- 孩子可能问 **"为什么我说的和老师说的不一样？"** → 可以说："可能是因为我们家乡的口音里没有这个区别——这很正常！很多地方的方言前后鼻音是合并的，所以需要专门练习普通话的标准发音。多听标准普通话（比如中央电视台新闻播音员），慢慢就能分清楚了。"

- **共读小活动（前后鼻音绕口令）：**
  > "南边来了个喃喃南，北边来了个能能能，南南和能能比比比，谁能分清南和能？"
  
  家长带着孩子慢慢说，重点感受"南 nán"（前鼻）和"能 néng"（后鼻）的区别。
