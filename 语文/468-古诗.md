# 古诗

> **一句话**：古诗是中国古代按照特定节奏和韵律写成的诗歌，读起来朗朗上口，意境深远。
> **English**: Classical Chinese poetry refers to poems written in ancient China with fixed rhythms and rhymes, melodious to recite and rich in imagery.

---

## 🧒 给 3-6 岁（共读版）

你有没有听过"鹅鹅鹅，曲项向天歌"？这句话读起来特别好听，像唱歌一样——这就是**古诗**！

古诗是很久很久以前，中国古代的人写的诗。那时候的人很聪明，他们用很少的字，就能说出很美的事情：

- 看到大白鹅在湖里游泳，就写下了《咏鹅》
- 看到天上的月亮，就想起了家里的妈妈
- 看到春天来了，就写诗夸花夸鸟

古诗特别短，一首诗只有二十几个字，但藏着很多美丽的画面。

**共读小活动：** 一起读《咏鹅》——"鹅鹅鹅，曲项向天歌，白毛浮绿水，红掌拨清波"。边读边用手画出大白鹅游水的样子，感受古诗的节奏和画面感。

## 📚 给 6-12 岁（自读版）

### 什么是古诗

古诗是中国古代的诗歌，从先秦时代一直延续到清朝。它有几个显著特点：
- **字数固定**：每句五个字叫"五言"，每句七个字叫"七言"
- **押韵**：偶数句末尾的字要押韵，读起来顺口
- **意境**：用简洁的文字表现丰富的情感和画面

### 两大主要形式

**绝句（Quatrain）：** 四句话的短诗
- 五言绝句：每句5字，共20字
- 七言绝句：每句7字，共28字

例：李白《静夜思》——"床前明月光，疑是地上霜。举头望明月，低头思故乡。"（五言绝句）

**律诗（Regulated verse）：** 八句话的诗，对平仄和对仗有严格要求
- 五言律诗：共40字
- 七言律诗：共56字

例：杜甫《春望》——"国破山河在，城春草木深……"（五言律诗）

### 唐诗宋词代表诗人

**唐代诗人：**
- **李白**（701-762）— "诗仙"，豪放飘逸，代表作《将进酒》《蜀道难》
- **杜甫**（712-770）— "诗圣"，忧国忧民，代表作《春望》《茅屋为秋风所破歌》
- **王维**（701-761）— "诗佛"，山水田园，代表作《山居秋暝》《鸟鸣涧》
- **白居易**（772-846）— 平易近人，代表作《赋得古原草送别》《琵琶行》

**宋代词人：**
- **苏轼**（1037-1101）— 豪放派，《水调歌头·明月几时有》
- **李清照**（1084-1155）— 婉约派，《如梦令》《声声慢》
- **辛弃疾**（1140-1207）— 豪放派，《破阵子》《青玉案》

### 朗读古诗的节奏

五言诗读法：**2+3** 节奏
> "床前 / 明月光，疑是 / 地上霜"

七言诗读法：**2+2+3** 节奏
> "朝辞 / 白帝 / 彩云间，千里 / 江陵 / 一日还"

## 🎓 给 12+（进阶版）

### 格律诗的平仄系统

古诗的声律规则称为"格律"，唐代格律诗（近体诗）对平仄有严格规定：

**平声**（今普通话一声+二声）与**仄声**（三声+四声+入声）交替排列，形成音乐美。

五言绝句四种基本格式（以仄起首句不入韵为例）：
```
仄仄平平仄
平平仄仄平（韵）
平平平仄仄
仄仄仄平平（韵）
```

**入声字**：中古汉语有入声，今普通话已消失，部分并入四声。读古诗时需注意古今音差异。

### 诗歌体裁演变史

| 时代 | 主要体裁 | 代表作 |
|------|----------|--------|
| 先秦 | 《诗经》四言诗 | 《关雎》《蒹葭》 |
| 楚汉 | 楚辞、汉赋 | 《离骚》屈原 |
| 魏晋 | 五言诗成熟 | 《短歌行》曹操 |
| 唐代 | 近体诗（律诗绝句）巅峰 | 李白、杜甫 |
| 宋代 | 词（长短句）发展 | 苏轼、李清照 |
| 元代 | 散曲、元曲 | 马致远《天净沙·秋思》 |
| 明清 | 诗词延续 | 纳兰性德词 |

### 古诗的英译难题

古诗翻译是文学界的重大挑战：
- **许渊冲（1921-2021）**：主张"三美"原则——意美、音美、形美；翻译了《唐诗三百首》《宋词》等
- 例：李白《静夜思》英译版本多达数十种，无一能完全还原"举头望明月"的意境
- 文化负载词（如"月"代表思乡）在西方文化中无对应情感，是翻译的核心难点

### 古诗与认知发展

研究表明，儿童早期接触古诗韵文（3-7岁）能促进：
1. 语音意识（phonological awareness）的发展
2. 汉字字形记忆（古诗重复高频字）
3. 节奏感与音乐性语言直觉

（参考：张志公《语文教育论》；课标2022"文学阅读与创意表达"学习任务群）

## 🗣️ 家长讲解话术

- 孩子可能问 **"古诗为什么那么难懂？"** → 可以说："古诗是一千多年前的人写的，那时候说话和现在不一样，就像外语一样。但是你先把它背下来，长大了会越来越懂它的美。"

- 孩子可能问 **"为什么要背古诗？"** → 可以说："古诗藏在你的记忆里，将来某一天你看到月亮、看到雪、看到春天，脑子里就会自然冒出一句诗——那个时刻特别美妙。"

- **共读小活动（体裁找一找）**：翻开语文课本，找找哪些课文是古诗，哪些是现代文，感受古诗和现代文语言的不同。

- **共读小活动（配图画诗）**：读完《咏鹅》《山行》后，让孩子画出诗里描绘的景色，比一比谁画的和诗中描述的最像。
