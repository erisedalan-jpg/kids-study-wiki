# 名词

> **一句话**：名词是表示人、事物、时间或地点名称的词，是句子里最常出现的"主角"。
> **English**: A noun is a word that names a person, thing, time, or place — the most common "star" of any sentence.

---

## 🧒 给 3-6 岁（共读版）

你知道"妈妈"是什么意思吗？"苹果"呢？"学校"呢？

这些有名字的东西——妈妈、苹果、学校——都叫做**名词**！

名词就是给事物起名字的词。就像每个人都有自己的名字，世界上的人、物、地方、时间，都有一个名字。

**有人的名词**：爸爸、妈妈、老师、小明、医生
**有东西的名词**：书、笔、苹果、桌子、小猫
**有地方的名词**：学校、公园、北京、家
**有时间的名词**：春天、今天、早上、星期一

### 共读小活动：找名词游戏

看看你的房间，指着任何一样东西说出它的名字——"床"、"窗户"、"玩具"——你说的每一个词，都是**名词**！

和爸爸妈妈比赛：1 分钟内谁能说出更多名词？

## 📚 给 6-12 岁（自读版）

### 什么是名词

名词（noun）是用来表示**人、事物、时间、地点、方向**等名称的词。

在句子里，名词常常做**主语**（谁）或**宾语**（做什么），是句子最基本的成分。

> **妈妈**（名词）买了一本**书**（名词）。
> **北京**（名词）是中国的首都。

### 名词的四大类型

#### 1. 人物名词
表示人的名称，包括亲属称谓、职业、国籍等。

| 类型 | 例子 |
|------|------|
| 亲属 | 爸爸、妈妈、爷爷、奶奶、哥哥、妹妹 |
| 职业 | 老师、医生、警察、工人、农民 |
| 泛称 | 小朋友、同学、邻居、朋友 |

#### 2. 事物名词
表示具体事物的名称，是最多的一类名词。

| 类型 | 例子 |
|------|------|
| 学习用品 | 书、笔、橡皮、尺子、书包 |
| 动植物 | 小狗、猫、玫瑰、大树、麻雀 |
| 食物 | 米饭、苹果、牛奶、面条 |
| 日常物品 | 桌子、椅子、窗户、电视 |

#### 3. 时间名词
表示时间的名称。

| 类型 | 例子 |
|------|------|
| 季节 | 春天、夏天、秋天、冬天 |
| 日期 | 今天、明天、昨天、元旦、春节 |
| 时段 | 早上、下午、傍晚、夜晚 |

#### 4. 地点名词
表示地方、方位的名称。

| 类型 | 例子 |
|------|------|
| 具体地点 | 学校、公园、图书馆、医院 |
| 地名 | 北京、上海、中国、长江 |
| 方位词 | 上面、下面、左边、右边、里面 |

### 普通名词 vs 专有名词

**普通名词**：泛指某一类人或事物，首字母不大写（在英语里）。
> 城市（city）、老师（teacher）、学校（school）

**专有名词**：特指某个具体的人名、地名、机构名，在英语中首字母**大写**。
> 北京（Beijing）、中国（China）、李明（Li Ming）

记忆口诀：**专有名词独一无二，第一个字母要大写。**

### 抽象名词

有些名词看不见摸不到，表示一种感受、概念或品质，叫做**抽象名词**。

> **友谊**（friendship）、**快乐**（happiness）、**勇气**（courage）、**自由**（freedom）

抽象名词比具体名词难理解，需要通过经历来体会它的含义。

### 名词在句子里的位置

名词最常出现在这几个位置：
1. **句首做主语**：**学校** 很漂亮。
2. **动词后做宾语**：我喜欢 **音乐**。
3. **"的"后做定语中心词**：美丽的 **花园**。
4. **"在/到/从"后做地点**：在 **学校** 里学习。

## 🎓 给 12+（进阶版）

### 名词的语法功能（句法分析）

从句法角度看，名词（或名词短语）可以充当以下句子成分：

| 句子成分 | 功能 | 例句 |
|---------|------|------|
| 主语 | 动作的发出者 | **老师**走进教室。 |
| 宾语 | 动作的承受者 | 我看见了**老师**。 |
| 定语中心词 | 被修饰的核心 | 我的**老师**很年轻。 |
| 状语（介词短语中） | 说明地点/时间 | 在**学校**里学习。 |
| 表语 | 主语的说明 | 他是一位**老师**。 |

### 中英名词系统的关键差异

| 特点 | 汉语名词 | 英语名词 |
|------|---------|---------|
| 单复数 | 形式不变（3本书 = three **books**，汉语"书"不加-s） | 可数名词复数加 -s/-es |
| 冠词 | 无冠词 | 有 a/an（不定冠词）、the（定冠词） |
| 量词 | 必须用量词（一**本**书、一**只**猫） | 直接用数词（one book, one cat） |
| 格变化 | 无格变化（人称代词有主格/宾格） | 有所有格（book**'s**）、人称代词格变化 |

**汉语的量词系统**正是为了弥补名词无单复数变化的不足——详见[[557-量词|量词]]词条。

### 古汉语名词活用

在文言文里，名词可以"活用"为动词，这是汉语语法的独特现象：

> **雨**我公田，遂及我私。（《诗经》）——"雨"活用为动词：下雨滋润
> **王**天下（孟子）——"王"活用为动词：称王、统治天下
> 项伯**翼**蔽沛公（史记）——"翼"活用为状语：像翅膀一样

名词活用的判断方法：
1. 名词出现在动词位置
2. 名词前有副词修饰（"乃、则、皆"等）
3. 名词后面接宾语

### 认知语言学视角

语言学家研究发现，人类语言中名词普遍比动词习得更早——婴儿最先学会叫"妈妈"（名词），而非"来"（动词）。这与人类认知发展有关：**名词指向稳定、可感知的实体**，动词指向动态事件，后者在认知上更抽象。

英语名词约占词汇量的 37%，是词类中比例最高的一类（Biber et al., 1998）。汉语中无精确统计，但名词同样是最大词类。

## 🗣️ 家长讲解话术

- 孩子可能问 **"名词和普通的词有什么区别？"** → 可以说："名词就是给事物起名字的词。你能摸到、看到、或者能想到的东西，只要有名字，那个名字就是名词。比如'苹果'是名词，但'吃'不是名词，因为'吃'是动作。"

- 孩子可能问 **"'快乐'是不是名词？"** → 可以说："'快乐'可以做名词，比如'快乐是什么'；也可以做形容词，比如'我很快乐'。中文很灵活，同一个词在不同句子里可以当不同词类用。"

- **共读小活动 1 — 名词捕捉游戏**：读一段课文，每遇到名词就举手。看谁最快发现所有名词。

- **共读小活动 2 — 分类游戏**：把以下词按"人/事物/时间/地点"分类：老师、书包、冬天、超市、医生、昨天、上海、铅笔、下午、朋友。
