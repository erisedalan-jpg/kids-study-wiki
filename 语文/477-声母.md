# 声母

> **一句话**：声母是普通话音节开头的辅音部分，共23个，决定了字音的起始方式。
> **English**: Initials are the consonantal onsets of Mandarin syllables; there are 23 initials that determine how a syllable begins.

---

## 🧒 给 3-6 岁（共读版）

宝贝，我们来玩"开头音"游戏！

说"爸爸"——"**b**—ā"，你听到开头那个"b"的声音了吗？说"妈妈"——"**m**—ā"，开头的"m"不一样对不对？

这些字音开头的小声音，就叫**声母**——就像每个音节的"大门"，先开门再进来！

**声母儿歌（边做动作边念）：**

> **b p m f**，双唇闭上弹出来（双手合十再打开）
> **d t n l**，舌尖弹弹上齿背（舌头上翘点点）
> **g k h**，喉咙深处发出声（手放喉咙感受震动）
> **zh ch sh r**，舌头卷起来（舌尖向上卷）
> **z c s**，舌尖平放轻轻说（舌头伸平）
> **j q x**，舌头中部往上顶（微笑状）

**共读小活动：** 拿出一套拼音卡片，家长举起声母卡，孩子说出一个以该声母开头的词——"b"说"爸爸"，"m"说"猫"，"d"说"大"。找不到词就一起想！

## 📚 给 6-12 岁（自读版）

### 23个声母完整列表

| 组别 | 声母 | 发音位置 | 代表字 |
|------|------|---------|--------|
| 双唇音 | b  p  m | 上下嘴唇相碰 | 爸 pá 妈 |
| 唇齿音 | f | 上齿咬下唇 | 飞 fēi |
| 舌尖前音（平舌） | z  c  s | 舌尖抵下齿背 | 字 cí 四 |
| 舌尖中音 | d  t  n  l | 舌尖抵上齿龈 | 大 tā 呢 lā |
| 舌尖后音（翘舌） | zh  ch  sh  r | 舌尖卷起向上 | 知 吃 师 日 |
| 舌面音 | j  q  x | 舌面前部抵硬腭 | 鸡 qī xī |
| 舌根音 | g  k  h | 舌根抵软腭 | 哥 kē 喝 |
| 半元音 | y  w | 介于元音辅音之间 | 一 yī 我 wǒ |

### 送气音 vs 不送气音

声母成对出现，区别是**送气**还是**不送气**（用手放嘴前感受气流）：

| 不送气 | 送气 | 测试方法 |
|--------|------|---------|
| b /p/ | p /pʰ/ | "p"说"婆"时有气流吹出 |
| d /t/ | t /tʰ/ | "t"说"他"时有气流 |
| g /k/ | k /kʰ/ | "k"说"可"时有气流 |
| z /ts/ | c /tsʰ/ | "c"说"粗"时有气流 |
| zh /ʈʂ/ | ch /ʈʂʰ/ | "ch"说"吃"时有气流 |
| j /tɕ/ | q /tɕʰ/ | "q"说"期"时有气流 |

**口诀**：送气的是 p t k c ch q，不送气的是 b d g z zh j。

### 声母与韵母的搭配规律

- **j q x** 只能与 i、ü（写作u）系列韵母搭配：jī qī xī、jū qū xū
- **zh ch sh r** 与 **z c s** 不能与 i、ü 搭配（只能接整体认读音节）
- **y w** 是半元音，用于韵母单独成节时：i→yi，u→wu，ü→yu

### 常见混淆与记忆方法

- **n 和 l** 都是舌尖音：n 时气流从鼻腔出（捏住鼻子说不出来），l 时气流从舌两侧出
- **f 和 h**：f 上齿咬下唇，h 不用牙只用喉咙
- **b d p q** 方向记忆：把"b"想成鼓槌向右，"d"向左，"p"鼓槌向下右，"q"向下左

## 🎓 给 12+（进阶版）

### 普通话辅音的语音学分类

语音学按**发音部位**（Place of Articulation）和**发音方式**（Manner）分类：

| 发音方式 \ 部位 | 双唇 | 唇齿 | 舌尖前 | 舌尖中 | 舌尖后 | 舌面 | 舌根 |
|---------------|------|------|--------|--------|--------|------|------|
| 塞音（无送气） | b /p/ | | | d /t/ | | | g /k/ |
| 塞音（送气） | p /pʰ/ | | | t /tʰ/ | | | k /kʰ/ |
| 鼻音 | m /m/ | | | n /n/ | | | （ng /ŋ/ 韵尾） |
| 边音 | | | | l /l/ | | | |
| 擦音 | | f /f/ | s /s/ | | sh /ʂ/  r /ʐ/ | x /ɕ/ | h /x/ |
| 塞擦音（无送气） | | | z /ts/ | | zh /ʈʂ/ | j /tɕ/ | |
| 塞擦音（送气） | | | c /tsʰ/ | | ch /ʈʂʰ/ | q /tɕʰ/ | |

**注意**：普通话没有"浊辅音"（voiced obstruents）——b d g 虽然字母与英语相同，但在语音学上是**清音不送气**，而非英语的浊音。

### 与英语辅音体系对比

英语区分**清浊**（voiced vs voiceless）：p/b、t/d、k/g、s/z。
普通话区分**送气与否**（aspirated vs unaspirated）：p/b、t/d、k/g、c/z。

这是中国学生学英语"b变p、d变t"的根源——普通话母语者感知不到浊音区别，因为他们的辅音对立轴是送气量而非声带振动。

### 方言中的声母差异

- **吴语**（上海话）保留全套浊塞音：b/p对立 = 浊/清
- **粤语**新增鼻音声母 ng-（如"我"ngoh）
- **闽南语**有 ts/tsh/s 系列，无 zh/ch/sh 翘舌系列
- **北方官话**部分地区 n/l 不分（南京话、四川话）

## 🗣️ 家长讲解话术

- 孩子可能问 **"声母和辅音有什么不同？"** → 可以说："辅音是语音学的概念，说的是气流受阻的声音；声母是汉语拼音的概念，专门指音节最开头的那个部分。大多数声母都是辅音，但 y 和 w 有点特殊，算'半元音'。"

- 孩子可能问 **"为什么 b 和 p 这么像，却是不同声母？"** → 可以说："你把手放在嘴前，说'爸爸'再说'怕怕'——感受到气流的区别了吗？'p'有股气，'b'没有。正是这股气，让它们成了两个不同的声母。"

- **共读小活动（声母快速游戏）：** 家长说一个词，孩子快速说出声母——"苹果"→"p"，"大象"→"d"，"花"→"h"，"自行车"→"z"……答对了给一颗小星星！
