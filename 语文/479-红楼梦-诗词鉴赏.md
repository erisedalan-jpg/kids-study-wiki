# 红楼梦·诗词鉴赏

> **一句话**：《红楼梦》中诗词曲赋逾两百首，既是人物性格的自我表达，也是命运伏笔的隐语载体，鉴赏这些诗词是理解全书深度的重要路径。

---

## 🎓 高中精读版

### 诗词的双重功能

《红楼梦》里的诗词不是装饰，而是叙事与刻画的有机组成：

1. **性格名片**：诗风即人格——黛玉之悲，宝钗之稳，湘云之豪，各有不同。
2. **命运谶语**：判词、灯谜、诗社作品往往是人物结局的预言，读懂诗词方能看穿命运密码。

### 重要诗词分类

#### 一、判词（第 5 回太虚幻境）

判词是曹雪芹为金陵十二钗各自撰写的谶语诗，图文并茂，需联系人名和情节拆解。

| 人物 | 判词节选 | 解读要点 |
|------|----------|----------|
| 黛玉/宝钗合册 | "玉带林中挂，金簪雪里埋" | 拆字：林（黛玉）、雪（薛→宝钗） |
| 元春 | "二十年来辨是非，榴花开处照宫闱" | 入宫为妃，后薨逝 |
| 探春 | "才自精明志自高，生于末世运偏消……一帆风雨路三千" | 远嫁海外 |
| 凤姐 | "一从二令三人木，哭向金陵事更哀" | "木"拆"休"字，被贾琏休弃 |
| 妙玉 | "欲洁何曾洁，云空未必空" | 以洁自守却落入污泥 |

#### 二、《葬花吟》（第 27 回）

林黛玉所作，是全书最重要的抒情长诗，184 字，7 言为主。

**核心名句**：
- "花谢花飞花满天，红消香断有谁怜？"——以落花自比，伤春即伤身世。
- "侬今葬花人笑痴，他年葬侬知是谁？"——将生死虚无推向极致，直逼死亡意识。
- "一朝春尽红颜老，花落人亡两不知！"——末句，青春（花）与生命（人）同归虚无。

**主题**：飘零身世感（父母双亡，寄居贾府）+ 青春必逝的生命哲学 + 以泪还债的宿命自觉。

#### 三、诗社联诗（第 37—38 回，第 70 回）

大观园海棠诗社，众人联句，以螃蟹、菊花、柳絮为题，各显才情。

**菊花诗十二首**（潇湘妃子/黛玉）：
- 《咏菊》"毫端蕴秀临霜写，口角噙香对月吟"——诗中有画，才气毕露。
- 《问菊》"孤标傲世偕谁隐，一样花开为底迟"——以菊自问，写孤傲个性。
- 《菊梦》夺魁，被李纨评为"潇洒"第一。

**薛宝钗《螃蟹咏》**：
- "眼前道路无经纬，皮里春秋空黑黄"——以螃蟹影射横行官场者，讽刺之笔胜于黛玉的伤感。

**史湘云《柳絮词》**：
- 湘云填词豪迈，别具一格，与她的直爽性格相符。

**薛宝钗《临江仙·柳絮词》**：
- "好风凭借力，送我上青云"——宝钗借柳絮写志向，是书中最能体现其内心抱负的诗句。

#### 四、《芙蓉女儿诔》（第 78 回）

贾宝玉为晴雯所作的祭文，骈散结合，文辞华丽，被视为向黛玉的隐性告白（晴雯影射黛玉说）。

> "其为质则金玉不足喻其贵，其为性则冰雪不足喻其洁，其为神则月貌不足喻其秀……"

### 鉴赏方法

**三步法**：
1. **知人**：确认作者是哪位人物（书中诗由人物写，诗风即人格）
2. **论世**：结合该回目的情节语境，理解诗词产生的场景动机
3. **解谶**：联系人物命运（判词、结局），理解诗词的预言性含义

**意象积累**：
| 意象 | 常见含义 |
|------|----------|
| 落花 / 飞花 | 青春易逝、女性命运凋零 |
| 竹 / 潇湘 | 黛玉、高洁孤傲 |
| 雪 / 冷 | 宝钗、压制情感 |
| 菊 | 高洁脱俗，隐逸精神 |
| 柳絮 | 身不由己，随风飘零 |
| 芙蓉 | 黛玉别称（芙蓉女儿） |
