## 🎓 古代官制——三省六部制与历代官职体系

### 定义

古代官制指历代王朝设立的政府机构体系与官员职位制度，是国家治理的组织框架。中国古代官制历经数千年演变，以**三省六部制**为核心骨架，影响深远。

---

### 历史演变

**先秦：卿大夫制**

周代实行分封制，天子之下设三公（太师、太傅、太保）辅政，诸侯国内设卿大夫。"三公"一词最早见于《书经》。

**秦汉：三公九卿制**

秦始皇统一后确立中央集权，设：
- **三公**：丞相（百官之长）、太尉（军事）、御史大夫（监察）
- **九卿**：奉常、郎中令、卫尉、太仆、廷尉、典客、宗正、治粟内史、少府，分掌具体事务

汉承秦制，武帝后逐步以"内朝"（尚书台）削弱三公权力。

**魏晋南北朝：过渡期**

尚书省权力渐重，中书省、门下省相继独立，三省雏形形成。

**隋唐：三省六部制（鼎盛）**

隋文帝正式确立**三省六部制**，唐代完善：

| 机构 | 职能 |
|------|------|
| **中书省**（出令） | 起草诏令，长官为中书令 |
| **门下省**（审驳） | 审议诏令，长官为侍中 |
| **尚书省**（执行） | 执行政令，长官为尚书令（太宗后虚置，以左右仆射代） |
| **吏部** | 官员选拔考核 |
| **户部** | 户籍财政 |
| **礼部** | 礼仪科举 |
| **兵部** | 军事 |
| **刑部** | 司法 |
| **工部** | 工程建设 |

**宋：二府三司**

宋分散相权：中书门下（政事堂）、枢密院（军事）并列为"二府"，三司（度支、盐铁、户部）掌财政。

**元：中书省一省制**

废三省，以中书省总理政务，设左右丞相，相权复重。

**明：废相设内阁**

明太祖洪武十三年（1380）废丞相，以六部直属皇帝，后设内阁辅政，阁臣有"票拟"之权。

**清：军机处**

雍正年间设军机处，军机大臣跪受笔录，皇权高度集中，内阁名存实亡。

|------|------|------|

| 刺史 / 牧 | 汉 | 地方监察 / 行政 |

### 高考考法

1. **官职含义判断**：如"迁""擢""黜""谪""除""拜"等表示官职变动的词语
2. **古今异义**：如"司马""太守""令"等在不同朝代含义有别
3. **文言传记阅读**：官职词语是理解人物身份、事件背景的关键
4. **文化常识选择题**：历年高考频繁考查三省六部、科举官职对应关系

**官职变动常见词**：

| 词语 | 含义 |
|------|------|
| 拜、除、授 | 授予官职 |
| 迁、擢、陟 | 晋升 |
| 谪、贬、左迁 | 降职或流放 |
| 罢、免、黜、革 | 免除官职 |
| 致仕、乞骸骨 | 退休告老 |
