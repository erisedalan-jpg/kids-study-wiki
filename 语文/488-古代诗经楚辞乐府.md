## 🎓 诗经·楚辞·乐府——中国古典诗歌的三大源头

### 定义

**《诗经》《楚辞》**与**乐府诗**是中国古代诗歌最重要的三大源流，共同奠定了中国诗歌的基本体裁、艺术手法与精神传统，被后世称为"风骚传统"（《诗经》之"风"、《楚辞》之"骚"）或"诗骚传统"。

---

### 一、《诗经》

**基本信息**：

| 项目 | 内容 |
|------|------|
| 性质 | 中国最早的诗歌总集 |
| 时代 | 西周初年至春秋中叶（约公元前11世纪—前6世纪） |
| 编定 | 相传孔子删定，实为多代积累 |
| 篇数 | 305篇（又称"诗三百"） |
| 收录 | 《汉书·艺文志》列为儒家经典 |

**三大结构**（风雅颂）：

| 分类 | 内容 | 数量 |
|------|------|------|
| 风（十五国风） | 各地民歌，多反映民间生活、爱情、劳作 | 160篇 |
| 雅（大雅、小雅） | 贵族宴饮、朝会诗，正统宫廷音乐 | 105篇 |
| 颂（周颂、鲁颂、商颂） | 祭祀祖先、赞美神灵的乐歌 | 40篇 |

**三大表现手法**（赋比兴）：

| 手法 | 定义 | 经典例子 |
|------|------|----------|
| 赋 | 直接铺陈叙述 | "死生契阔，与子成说"（《邶风·击鼓》） |
| 比 | 打比方，以彼喻此 | "手如柔荑，肤如凝脂"（《卫风·硕人》） |
| 兴 | 先言他物以引出所咏之词 | "关关雎鸠，在河之洲。窈窕淑女，君子好逑。" |

**代表篇目**：

| 篇目 | 类别 | 内容 |
|------|------|------|
| 《关雎》 | 周南（风） | 爱情之思，开卷第一篇 |
| 《蒹葭》 | 秦风（风） | 追寻意象，朦胧美 |
| 《采薇》 | 小雅 | 戍边思乡，"昔我往矣，杨柳依依" |
| 《氓》 | 卫风（风） | 弃妇诗，爱情悲剧 |
| 《硕鼠》 | 魏风（风） | 讽刺剥削者 |

---

### 二、《楚辞》

**基本信息**：

| 项目 | 内容 |
|------|------|
| 性质 | 战国时期楚国诗歌总集，由汉代刘向编定 |
| 时代 | 战国中晚期 |
| 核心作者 | 屈原（最重要）、宋玉 |
| 文体特征 | 以"兮"字句为标志，篇幅较长，抒情性强 |
| 地域特色 | 南方楚地巫文化、神话传说渗透其中 |

**《离骚》**（最重要的楚辞作品）：

- 作者：屈原，战国楚国贵族诗人，中国第一位留名的大诗人
- 性质：政治抒情长诗，长达373句，2490字
- 主题：以香草美人为喻，抒发报国无门、忠而被谗的悲愤
- 艺术：大量神话意象、自然意象，情感激越，想象丰富

**核心意象**（香草美人系统）：

| 意象 | 象征 |
|------|------|
| 香草（兰、芷、荃等） | 高洁品格、美好德行 |
| 美人 | 理想君主 |
| 恶草（艾、萧等） | 小人、奸佞 |
| 鸷鸟 | 志向高远的诗人自喻 |

**其他楚辞重要篇目**：

| 篇目 | 内容 |
|------|------|
| 《天问》屈原 | 对天地万物、历史神话的追问，172问 |
| 《九歌》屈原 | 祭祀神灵的组诗，含《湘君》《湘夫人》等 |
| 《九章》屈原 | 包含《涉江》《哀郢》等抒情诗 |
| 《招魂》 | 托名屈原或宋玉，呼唤离散的魂魄 |

---

### 三、乐府诗

**基本信息**：

| 项目 | 内容 |
|------|------|
| 性质 | 汉代设"乐府"官署采集的民歌及文人拟作 |
| 起源 | 西汉武帝设乐府署，专司采集、整理、演奏歌谣 |
| 时代 | 汉代至唐代（广义上含唐代新乐府） |
| 语言 | 以五言为主，也有四言、杂言 |

**分类**：

| 类别 | 说明 | 代表作 |
|------|------|--------|
| 汉乐府民歌 | 两汉时期民间采集，真实反映社会生活 | 《陌上桑》《孔雀东南飞》《上邪》 |
| 魏晋文人乐府 | 文人模仿乐府形式创作 | 曹操《短歌行》《步出夏门行》 |
| 南北朝乐府 | 南朝吴歌、西曲；北朝民歌 | 《木兰诗》（北朝）、《西洲曲》（南朝） |
| 唐代新乐府 | 杜甫、白居易等倡导，以乐府形式反映现实 | 白居易《卖炭翁》《琵琶行》 |

**汉乐府代表作**：

| 作品 | 内容 |
|------|------|
| 《孔雀东南飞》 | 最长叙事诗，爱情悲剧，与《木兰诗》合称"乐府双璧" |
| 《陌上桑》 | 美女罗敷拒绝太守调戏，幽默生动 |
| 《上邪》 | 女子对爱情的海誓山盟 |
| 《战城南》 | 战争悲苦 |
| 《十五从军征》 | 老兵归家的悲剧 |

**曹操乐府的特点**：
- 借古乐府题目抒发个人政治抱负
- 《短歌行》："对酒当歌，人生几何……青青子衿，悠悠我心"
- 《观沧海》："日月之行，若出其中"（建安文学的雄浑气魄）

### 三者比较

| 维度 | 《诗经》 | 《楚辞》 | 乐府诗 |
|------|----------|----------|--------|
| 时代 | 西周—春秋 | 战国 | 汉代—唐代 |
| 语言 | 四言为主 | 以"兮"为标志，长句 | 五言为主，杂言 |
| 地域 | 北方中原 | 南方楚地 | 全国各地 |
| 风格 | 质朴现实 | 浪漫绮丽 | 写实叙事为主 |
| 代表手法 | 赋比兴 | 香草美人象征 | 叙事为主 |

### 高考考点

1. **基础知识**：风雅颂、赋比兴的定义与举例（必考）
2. **作品归属**：《离骚》（屈原）、《木兰诗》（北朝乐府）、《孔雀东南飞》（汉乐府）
3. **文学史**："风骚传统"对后世文学的影响
4. **诗句鉴赏**：《关雎》《蒹葭》《采薇》《离骚》中的名句赏析
