# 小池

> **一句话**: 泉眼轻声细流，树阴柔情映水，荷叶才露出尖尖角，蜻蜓便已停上头——杨万里用工笔描绘了初夏荷塘最细腻的一刻。
> **English**: A spring trickles silently, tree shadows love the clear water, a tiny lotus leaf just peeks above the surface — and a dragonfly is already perched on top.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**: 宋 · 杨万里
**体裁**: 七言绝句

```
小池
杨万里

泉眼无声惜细流，
树阴照水爱晴柔。
小荷才露尖尖角，
早有蜻蜓立上头。
```

**拼音**:
- quán yǎn wú shēng xī xì liú，
- shù yīn zhào shuǐ ài qíng róu。
- xiǎo hé cái lù jiān jiān jiǎo，
- zǎo yǒu qīng tíng lì shàng tóu。

**白话译文**:
泉眼悄无声息地流淌，好像爱惜着这细细的水流，不忍心大声流动；
树阴映照在水面上，好像也喜爱这晴天里的温柔水光。
小荷叶才刚刚探出水面，露出尖尖的嫩角；
就已经有一只蜻蜓停在了上面。

**朗读节奏**:
泉眼 / 无声 / 惜细流，树阴 / 照水 / 爱晴柔。
小荷 / 才露 / 尖尖角，早有 / 蜻蜓 / 立上头。

---

## 🧒 给 3-6 岁（共读版）

你见过蜻蜓停在荷叶上吗？那个画面可漂亮了！

杨万里是宋朝的诗人，他看到了一个小池塘（小水池），里面有细细的泉水（泉眼，就是水从地里冒出来的小地方）。

泉水静悄悄地流，一点声音都没有；树的影子倒在水面上，水面很温柔很好看。

最可爱的是——荷叶刚刚长出一个小尖角来，还没完全长大呢，就已经有一只小蜻蜓站在上面了！

是不是很有趣？小荷叶刚露头，蜻蜓就来了！

**共读小活动**:
- **画一画**: 画一个小池塘，水里有一片刚露出来的小荷叶（只有一个尖角），上面站着一只蜻蜓。
- **找一找**: 诗里有几种生物？（泉水、树、荷叶、蜻蜓——还有鱼吗？没有！）
- **演一演**: 一只手做荷叶慢慢"冒出来"的样子，另一只手做蜻蜓飞过来停上去的样子。

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 泉眼 | quán yǎn | 泉水流出的地方，泉源的出口 |
| 无声 | wú shēng | 没有声音，寂静无声 |
| 惜 | xī | 爱惜，珍惜（拟人，把泉眼当成有感情的） |
| 细流 | xì liú | 细细的水流 |
| 树阴 | shù yīn | 树的影子（也写作"树荫"） |
| 晴柔 | qíng róu | 晴天里温柔的阳光和水光 |
| 小荷 | xiǎo hé | 小小的荷叶（刚出水面的嫩荷叶） |
| 才露 | cái lù | 刚刚露出，才刚出现 |
| 尖尖角 | jiān jiān jiǎo | 荷叶刚出水时的尖尖形状 |
| 早有 | zǎo yǒu | 已经有了，提前出现 |
| 立上头 | lì shàng tóu | 站在上面 |

### 内容与情感

全诗描绘了初夏小池的一个宁静时刻，层次分明：

1. **远景（水）**: 泉眼细流，无声流淌
2. **中景（树）**: 树阴映水，柔和美丽
3. **特写（荷叶）**: 小荷才露尖角
4. **动态点睛（蜻蜓）**: 蜻蜓立上头

最后两句是全诗的点睛之笔——"小荷才露尖尖角，早有蜻蜓立上头"，用"才"和"早"形成时间对比，将初夏荷塘的生机勃勃表现得活灵活现。

### 写作手法

- **拟人**: "泉眼无声惜细流"——泉眼会"爱惜"细流；"树阴照水爱晴柔"——树阴会"喜爱"晴柔，赋予自然事物以情感
- **工笔细描**: 诗人像工笔画家一样，一笔一画地描绘细节（泉眼、树阴、荷角、蜻蜓）
- **动静对比**: 泉眼无声（静）、蜻蜓立上（动），动静结合
- **"才""早"对比**: "才露"写荷叶刚出，"早有"写蜻蜓先到，两字一对，妙趣横生

### 朗读建议

全诗节奏舒缓，语调轻柔，像在轻轻讲一个小秘密。"尖尖角"要读得轻巧可爱；"早有蜻蜓立上头"可以带着发现小惊喜的语气，微微上扬。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**杨万里**（1127—1206年），字廷秀，号诚斋，吉水（今江西）人，南宋著名诗人，与陆游、范成大、尤袤并称"南宋四大诗人"（中兴四大诗人）。

杨万里创立了独具特色的"诚斋体"诗风，以清新活泼、富有生活情趣为特点，善于捕捉自然界的细节，将日常生活和自然景物写得趣味盎然。《小池》是他诚斋体的代表作之一。

### 文学史地位

《小池》在南宋诗歌中是描写自然细节的典范。杨万里以"活法"（活泼写法）著称，不同于当时流行的江西诗派（以黄庭坚为代表）的典故堆砌，而是回归自然观察。"小荷才露尖尖角，早有蜻蜓立上头"成为描写初夏荷塘的经典名句，常被引用于艺术创作、教育材料乃至政治演讲（比喻新生事物刚刚萌芽）。

### 鉴赏与对比

与白居易《池上》相比，《小池》更静，《池上》更动；《小池》写景为主，《池上》写人为主。两诗都以池塘为背景，却风格迥异。

与《晓出净慈寺送林子方》（同为杨万里写荷花）相比，《小池》写初夏荷叶初露的娇嫩，《晓出》写盛夏荷花的壮美——同一诗人，同一题材，一写"小"，一写"大"，对比鲜明。

### 文化影响

- 收录于《诚斋集》（杨万里诗集）
- 小学语文教材推荐篇目，是儿童最早接触的宋诗之一
- "小荷才露尖尖角，早有蜻蜓立上头"常被用于比喻新人、新事物初露头角
- 在中国绘画、刺绣等艺术中，荷叶+蜻蜓是经典装饰题材，此诗是重要文学来源之一

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么泉眼会'惜'细流？"** → 可以说："'惜'是爱惜的意思，这是诗人把泉眼当成了有感情的人——好像泉眼舍不得让水流走太快，所以水流得很细、很慢、很轻声。这叫'拟人'手法，把没有生命的东西写得像有感情一样。"

- 孩子可能问 **"蜻蜓为什么要站在荷叶上？"** → 可以说："蜻蜓喜欢停在植物上休息、产卵，荷叶是很好的落脚地。而且荷叶刚刚长出来，嫩嫩的，蜻蜓一眼就发现了——所以诗里说'早有蜻蜓'，说明蜻蜓比我们还先看到这片新荷叶！"

- **共读活动（找意象）**: 读完诗，让孩子列出诗中出现的所有事物（泉眼、细流、树阴、水、小荷、蜻蜓），讨论它们之间的关系，感受自然的和谐。

- **共读活动（画诗境）**: 让孩子画出小池的场景，重点画出荷叶刚露尖角、蜻蜓停在上面的细节。画完对照诗句，看看孩子画了哪些元素。
