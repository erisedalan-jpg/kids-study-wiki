# 古代作家·王安石

> **一句话**：王安石（1021—1086），字介甫，号半山，北宋政治家、文学家、思想家，唐宋八大家之一，诗文风格峭劲简练，晚年诗作（"半山体"）以工巧见长，散文以说理见著。

## 🎓 高中精读版

### 生平时间轴（文学维度）

| 时期 | 大事 |
|------|------|
| 1021 年 | 生于江西抚州临川 |
| 1042 年（22 岁） | 进士及第，开始仕途，外任地方官多年 |
| 1058 年（38 岁） | 上《言事书》（万言书），向仁宗条陈改革建议，文名大著 |
| 1069—1076 年 | 两度出任宰相，推行新法（此为政治维度，词条不作展开） |
| 1076 年（56 岁） | 退居江宁（今南京），号"半山"，专注诗文创作 |
| 1086 年 | 去世，谥"文"，世称"王文公" |

### 散文成就

王安石散文以"辨析义理、议论犀利"著称，是唐宋古文运动的重要参与者。

**代表散文**：

**《游褒禅山记》**（高中必背）：
- 游记与议论的结合：以游褒禅山（未能到达最深处）为契机，提出"世之奇伟、瑰怪、非常之观，常在于险远，而人之所罕至焉，故非有志者不能至也"。
- 三个层次：志（意志）、力（体力）、物（外部条件/向导），缺一不可。
- 文章的核心论点：做事、治学须有志、有力、借助外部条件，三者缺一不可——具体游记经历上升为普遍人生哲理。

**名句**："夫夷以近，则游者众；险以远，则至者少。而世之奇伟、瑰怪、非常之观，常在于险远，而人之所罕至焉，故非有志者不能至也。"

**《答司马谏议书》**（与司马光的书信往来）：
- 王安石向反对新法的司马光回信，逐条驳斥其指责（名实不正、怨谤等）。
- 文章论理严密，语气坚定而有礼，是古代政论书信的典范。
- 核心观点："名实已明，而天下之理得矣"——坚持新法的名实相符，不惧流俗之谤。

### 诗歌成就

**1. 早中期诗：关注现实，慷慨豪迈**

《登飞来峰》（22 岁进士前后所作）：
> 飞来山上千寻塔，闻说鸡鸣见日升。不畏浮云遮望眼，自缘身在最高层。

- 少年立志之诗，"不畏浮云遮望眼"化用典故（"总为浮云能蔽日，长安不见使人愁"——李白语），表达年轻时的政治雄心与不惧困难的自信。
- 与王之涣《登鹳雀楼》"欲穷千里目，更上一层楼"同为"登高开阔视野"的主题，但王安石更具政治抱负色彩。

《元日》：
> 爆竹声中一岁除，春风送暖入屠苏。千门万户曈曈日，总把新桃换旧符。

- 表面写春节习俗（爆竹、屠苏酒、桃符），实则寄寓以新换旧的变法情怀。

**2. 晚年"半山体"：精工瘦硬，以少胜多**

退居金陵后，王安石诗风大变，趋向精炼含蓄，世称"半山体"或"王荆公体"。

《泊船瓜洲》：
> 京口瓜洲一水间，钟山只隔数重山。春风又绿江南岸，明月何时照我还？

- "绿"字成为炼字的千古典范：据传王安石改字十余次，经"到""过""入""满"等多个选项，最终定为"绿"——动词化用，写出春色的扑面而来，具有动态性与颜色感。
- 全诗以"还"字收尾，道出退居后对故居（金陵钟山）的眷恋。

《梅花》：
> 墙角数枝梅，凌寒独自开。遥知不是雪，为有暗香来。

- 以"暗香"（嗅觉）点出视觉不能辨别雪与梅的区别，构思巧妙。
- 梅象征在逆境中坚守的高洁人格，亦暗含作者退居后的自我安慰。

### 词的成就

《桂枝香·金陵怀古》是王安石词的代表作，气势开阔，议论健拔，苏轼读后叹"此词当押两部乐籍"：

> 登临送目，正故国晚秋，天气初肃。……至今商女，时时犹唱，后庭遗曲。

- 化用杜牧《泊秦淮》"商女不知亡国恨"，俯瞰六朝兴亡，寄托忧国之思。
- 风格豪迈，是北宋词由婉约向豪放转变的过渡性代表，影响苏轼豪放词风的形成。

### 文学风格总结

| 特征 | 说明 |
|------|------|
| 峭劲 | 语言简练而有力度，不用闲字废语 |
| 以议论入诗 | 诗中常有理性思辨，不纯写感性意象 |
| 炼字精工 | 晚年"半山体"极重词语锤炼，一字之改经反复斟酌 |
| 化用典故 | 善用前人诗句加以翻化，自出机杼 |

### 文学史地位

- **唐宋八大家**之一（唐：韩愈、柳宗元；宋：欧阳修、苏洵、苏轼、苏辙、曾巩、王安石）。
- 与欧阳修同为北宋古文运动的领军人物，推动诗文从西昆体的浮靡回归质朴。
- 晚年"半山体"影响宋代"江西诗派"，黄庭坚等以王安石为学习对象之一。
