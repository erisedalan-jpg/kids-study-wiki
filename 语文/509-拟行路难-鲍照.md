# 拟行路难（其四）

> **一句话**：南朝诗人鲍照以"泻水"比喻门阀制度下寒门士子的命运，写出了才华横溢却无力抗争的悲愤，是中国文学史上最早以诗歌正面抗议门第制度的名篇之一。
> **English**: Using the image of water flowing to low ground as a metaphor for the aristocratic social system, Bao Zhao — himself of humble birth — voices the bitter frustration of talented men blocked by birth from advancement, in one of the earliest poetic protests against China's class system.

## 🎓 拟行路难（其四）

### 作者生平

鲍照（约415—466），字明远，东海（今山东郯城附近）人，南朝宋诗人。他出身寒微，在门阀制度森严的南朝，寒门士子几乎无法通过才华获得晋升。鲍照以才华闻名，却终生郁郁，官至前军参军，死于乱军之中。

鲍照的诗歌以"俊逸豪放、奇崛险峻"著称，擅长乐府诗，对后世李白等人影响深远。《拟行路难》共18首，是其代表作，借乐府旧题抒发寒门士子的不平之鸣。

南朝的门阀制度规定，出身高门大族方可任高官要职，寒门子弟无论才华多高，都被拒于仕途之外。这种制度性的不公，是《拟行路难》产生的社会背景。

### 原文（其四）

> 泻水置平地，各自东西南北流。
> 人生亦有命，安能行叹复坐愁？
> 酌酒以自宽，举杯断绝歌路难。
> 心非木石岂无感，吞声踯躅不敢言。

### 注释

| 词语 | 注释 |
|------|------|
| 泻水置平地 | 将水倒在平地上 |
| 各自东西南北流 | 水向四面八方流去，方向各异，喻命运因出身而各异 |
| 人生亦有命 | 人生也是由命运（出身）决定的 |
| 安能行叹复坐愁 | 怎么能既行走时叹气又坐着发愁，表面上是自我宽慰 |
| 酌酒以自宽 | 斟酒来宽慰自己 |
| 举杯断绝歌路难 | 举起酒杯，打断了（无法唱下去的）《行路难》之歌 |
| 心非木石岂无感 | 心又不是木头石头，怎么可能没有感受 |
| 吞声踯躅不敢言 | 忍着哭声、徘徊不前，却不敢开口说出来 |

### 译文

> 把水泼在平地上，水自然地向东西南北四方流去。人的命运也是这样——由出身决定，又能怎样，难道能一边走路一边叹气、一边坐着一边发愁吗？斟酒来宽慰自己，举起酒杯，却发现《行路难》的歌再也唱不下去了。心又不是木头石头，怎么可能没有感受呢——只是忍着哭声、徘徊不前，不敢说出口罢了。

### 文学赏析

**"泻水"比喻的精准**

开篇的"泻水置平地，各自东西南北流"——这个比喻看似在说自然规律，实则是一个关于社会制度的深刻隐喻：水因重力而流向低处，人因出身而决定命运，这都是"自然的"——但正是这种"自然性"，构成了对门阀制度的有力控诉。诗人并没有直接抨击，而是借助这个中性的自然现象，让读者感受到那种无可奈何的宿命感。

**情感的层次**

诗歌的情感经历了三个层次：
1. 用哲学化的水的比喻来接受现实（看似平静）
2. 以饮酒自宽来压制情感（强作镇定）
3. 最终爆发——"心非木石岂无感，吞声踯躅不敢言"（抑郁难言的真实情感）

这种情感的层层压抑与最终冲破，使诗歌的结尾具有强大的爆发力。

**"不敢言"的控诉**

"吞声踯躅不敢言"——最后这六个字，是全诗最沉重的地方。"不是不想说，而是不敢说"——这种压抑本身，就是对那个制度最有力的控诉。诗人心中有千言万语，却只能吞声而行。这种"无声的愤怒"，比任何咆哮都更令人心酸。

**对李白的影响**

李白《行路难》《将进酒》中的豪放愤激，在精神气质上与鲍照一脉相承。鲍照以乐府诗写寒士抗争，开辟了一种将个人命运与社会批判相结合的诗歌传统。

### 高考考点

- **核心名句**：心非木石岂无感，吞声踯躅不敢言（最常考）
- **比兴手法**：以泻水比喻人生命运，分析其效果
- **情感主题**：门阀制度下寒门士子的悲愤；压抑与反抗；无奈与愤懑
- **南朝文学背景**：了解门阀制度对文学创作的影响
- **与《行路难》（李白）对比**：同题材，鲍照的压抑vs李白的豪放突破
