# 望庐山瀑布

> **一句话**: 李白站在庐山远处仰望瀑布，用"飞流直下三千尺，疑是银河落九天"的神奇夸张，写出了瀑布的磅礴气势。
> **English**: Li Bai gazes at the Lushan waterfall from afar, comparing its thundering cascade to the Milky Way tumbling down from the ninth heaven — magnificent and boundless.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**: 唐 · 李白
**体裁**: 七言绝句

```
望庐山瀑布
李白

日照香炉生紫烟，
遥看瀑布挂前川。
飞流直下三千尺，
疑是银河落九天。
```

**拼音**:
- rì zhào xiāng lú shēng zǐ yān，
- yáo kàn pù bù guà qián chuān。
- fēi liú zhí xià sān qiān chǐ，
- yí shì yín hé luò jiǔ tiān。

**白话译文**:
太阳照耀着香炉峰，峰顶升腾起紫色的云雾（水汽）；
从远处看，瀑布像一条白练挂在山前的河流上。
瀑布飞泻直下，仿佛有三千尺那么高；
不禁让人怀疑，莫非这是银河从九重天上落了下来？

**朗读节奏**:
日照 / 香炉 / 生紫烟，遥看 / 瀑布 / 挂前川。
飞流 / 直下 / 三千尺，疑是 / 银河 / 落九天。

## 🧒 给 3-6 岁（共读版）

你见过瀑布吗？高山上的水哗哗哗地从很高的地方冲下来，声音特别大，水花飞溅，看起来好壮观！

李白去了一座叫"庐山"的大山，看到了一个大瀑布。太阳照在山上，水雾变成了紫色的烟雾（"生紫烟"），好神奇！

他从远处看瀑布，觉得瀑布像一条白色的布挂在山上（"挂前川"）。

水从高处飞速冲下来——他觉得有三千尺那么高（超级高！），这水流得那么快那么猛，简直像天上的银河从天上掉下来了！

**共读小活动**:
- **画一画**: 画一座大山，山顶有云和烟，山腰挂着一条白色的大瀑布，水从最高处哗哗往下冲。
- **想一想**: 你见过瀑布吗？或者在视频里见过？它是什么声音？像什么？
- **比一比**: "三千尺"有多高？一层楼大约三米，三千尺约900米——比上海最高楼还高！

| 望 | wàng | 从远处观望、仰望 |

| 香炉 | xiāng lú | 香炉峰，庐山的一个山峰，形如香炉 |
| 紫烟 | zǐ yān | 紫色的云雾（阳光照射水汽后呈现的颜色） |
| 遥看 | yáo kàn | 从远处看 |
| 挂 | guà | 悬挂，像布一样挂着 |
| 前川 | qián chuān | 前面的河流（这里指山前流淌的水流） |
| 飞流 | fēi liú | 飞速流泻的水流 |
| 三千尺 | sān qiān chǐ | 夸张手法，形容极高 |
| 疑是 | yí shì | 怀疑是，好像是 |
| 银河 | yín hé | 银河系，夜空中的星云带，中国古代称为"天河" |
| 九天 | jiǔ tiān | 古代传说中天的最高处，九重天 |

### 内容与情感

全诗由远及近，由静到动：

1. **远景**: 香炉峰在阳光下生出紫色烟雾（静态、色彩）
2. **中景**: 从远处看瀑布像白练悬挂（比喻、宏观）
3. **特写**: 飞流直下，高度三千尺（夸张、动态）
4. **联想**: 如银河从九天落下（最高的夸张与联想）

诗人的情感是震撼与惊叹——面对大自然的壮观，感到人的渺小和自然的伟大，同时也有一种豪迈的喜悦。

### 写作手法

- **夸张**: "三千尺"极度夸张，突显瀑布气势磅礴
- **比喻**: "疑是银河落九天"将瀑布比作银河，宏大奇特
- **衬托**: 用香炉峰的紫烟（静）衬托瀑布的飞流（动）
- **视觉层次**: 先写山顶烟雾，再写瀑布全貌，再写飞流速度，层层推进

### 朗读建议

"日照香炉生紫烟"语调平缓，描绘宁静景色；"飞流直下三千尺"节奏加快，有力量感，体现飞泻而下的动态；"疑是银河落九天"语调高亢，充满惊叹，读出诗人的震撼。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**李白**（701—762年），"诗仙"，详见→[[563-静夜思|静夜思]]词条。

《望庐山瀑布》约作于唐开元十三年（725年）李白初游庐山时，或天宝十四年（755年）二游庐山时（学界有争议，待家长核对）。庐山是江西省名山，历代文人雅士的登临胜地，苏轼"横看成岭侧成峰"《题西林壁》也写于此地。

### 文学史地位

《望庐山瀑布》是李白山水诗的代表作，也是中国最著名的写景诗之一。"飞流直下三千尺，疑是银河落九天"被誉为中国夸张手法运用的极致典范，影响了后世无数诗人对自然奇观的描写。

李白以浪漫主义著称，善用夸张和想象，此诗是最能代表他诗风的作品之一——不求精确，但求震撼人心。

### 鉴赏与对比

与王维《山居秋暝》（"明月松间照，清泉石上流"）的清幽宁静相比，李白的庐山瀑布充满磅礴气势；与杜甫的沉郁精炼相比，李白更奔放夸张。

"疑是银河落九天"将地上的瀑布与天上的银河相联，时空跨越极大，这种"天马行空"的联想正是李白浪漫主义的核心。任何一个对数字精确的人不会说"三千尺"，但任何一个对壮观有感觉的人都会认同这个夸张。

### 文化影响

- 收录于《全唐诗》卷一七二，李白诗
- 二年级语文教材必背，是小学阶段最重要的山水诗之一
- 庐山瀑布（今江西九江）因此诗而成为著名旅游景点
- "银河落九天"是描写壮观景象的常用典故，频繁出现于现代文学和媒体

## 🗣️ 家长讲解话术

- 孩子可能问 **"真的有三千尺那么高吗？"** → 可以说："不是真的三千尺（约900米）！这叫'夸张'——李白为了表达瀑布的气势，故意把高度说得非常非常夸张。就像我们说'我饿得能吃一头牛'，不是真的能吃一头牛，而是表示非常非常饿。李白用夸张让你感受到瀑布有多壮观！"

- 孩子可能问 **"银河是什么？"** → 可以说："银河就是夜空中那条白茫茫的星星带，是我们的星系。在古代，人们叫它'天河'，认为是天上的一条大河。李白看着瀑布从高处哗哗冲下来，就觉得：这不就是天上的银河掉下来了吗？太壮观了！"

- **共读活动（找意象）**: 诗中有哪些颜色？（紫色的烟、白色的瀑布）找出来，讨论这些颜色给你什么感觉。

- **共读活动（全家朗诵）**: 全家一起用洪亮的声音朗诵，重点体验"飞流直下三千尺"的力量感，感受诗人面对大瀑布的震撼。
