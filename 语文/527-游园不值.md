# 游园不值

> **一句话**：叶绍翁去朋友花园拜访，主人不在，门也锁了，正觉遗憾，却见一枝红杏从高墙里探出头来——春色是关不住的。
> **English**: Ye Shaoweng visits a friend's garden only to find the gate locked and no one home — then a single red apricot blossom peeks over the tall wall, proclaiming that spring cannot be shut in.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：宋 · 叶绍翁
**体裁**：七言绝句

```
游园不值
叶绍翁

应怜屐齿印苍苔，
小扣柴扉久不开。
春色满园关不住，
一枝红杏出墙来。
```

**拼音**：
- yīng lián jī chǐ yìn cāng tāi，
- xiǎo kòu chái fēi jiǔ bù kāi。
- chūn sè mǎn yuán guān bù zhù，
- yī zhī hóng xìng chū qiáng lái。

**白话译文**：
大概是主人爱惜那青苔，怕木屐踩坏了它，
我轻轻敲了很久柴木门，却一直没有人来开。
可是，满园子的春色关不住啊——
一枝红杏伸出了高高的墙壁！

**朗读节奏**：
应怜 / 屐齿 / 印苍苔，/ 小扣 / 柴扉 / 久不开。
春色 / 满园 / 关不住，/ 一枝 / 红杏 / 出墙来。

## 🧒 给 3-6 岁（共读版）

有一天，诗人叶绍翁去一个朋友家参观花园，可是主人不在！门是锁着的，他敲啊敲，没有人来开门……

正当他有点失望的时候，他抬头一看——哇！高高的围墙上面，探出来一枝红红的杏花！

春天的花朵藏在园子里，墙那么高，还是挡不住！一枝红杏偷偷伸出头来说：我在这里！

**共读小活动**：
- **想一想**：你有没有想进去但没办法进去的地方，后来发现了什么？
- **演一演**：扮成一朵花，从"墙"（沙发靠背）后面探出头来，和大家打招呼！

### 内容与情感

这首诗的情感是一个转折：

**前两句（遗憾）**：
- 应怜屐齿印苍苔：猜测主人担心苔藓被踩，为不开门找借口（善意的理解）
- 小扣柴扉久不开：轻轻敲门，等了很久，还是没人开——失落

**后两句（惊喜）**：
- 春色满园关不住：全园子的春天关不住！
- 一枝红杏出墙来：一枝红杏探出墙来——意外的惊喜，完美的结局

这首诗告诉我们：门关着，不代表看不到春天；有时候，一点小小的意外，反而带来比预期更美的发现。

### 从诗到哲学

"春色满园关不住，一枝红杏出墙来"已经超越了诗本身，成为一个文化符号，蕴含哲学含义：
- 生命的力量不可阻挡
- 美好的事物总会找到出路
- 真正有生命力的东西，关不住

## 🎓 给 12+（进阶版）

### 作者与时代背景

**叶绍翁**（约1194—约1269），字嗣宗，龙泉（今浙江龙泉）人，南宋诗人，与赵蕃并称"赵叶"，工于七言绝句。其诗风细腻清新，以景托情著称。

这首《游园不值》是叶绍翁最著名的作品，写于春日游园的具体场景，却因最后两句的哲思而超越了个人记录，成为宋代诗歌中最广为流传的名句之一。

关于"游园"的对象是谁，历代有不同说法，但诗本身不依赖典故，独立成篇，是宋代清新诗风的代表作。

### "一枝红杏出墙来"的文化影响

这一句在中国文化史上有着丰富的接受史：

**正面解读（诗本意）**：春色无法被高墙关住，美好事物必然绽放，生命力不可遏制。

**宋代诗评**：叶绍翁这句诗被叶梦得《石林诗话》等评为"以小见大"的典范，一枝红杏代表整个满园春色，以局部表现整体。

**俗语化**：后世"红杏出墙"成为固定短语，词义发生了偏移（专指婚姻不忠），这与原诗的诗意完全无关，但说明了好诗的意象生命力极强——连变形也能流传。

在教育语境中，应该以诗本意解读：生命力与美好无可阻挡，而非其派生俗义。

### 宋代"以小见大"的艺术传统

叶绍翁这首诗的艺术精髓是"以一枝见满园"——用最小的细节（一枝红杏）传达最大的主题（满园春色）。这是宋代文人绘画和诗歌共同的审美取向：

- 宋代绘画：马远"一角"、夏圭"半边"——用局部暗示整体
- 宋诗：寓大于小，借微见著

相比盛唐诗的"大漠孤烟直，长河落日圆"式的宏观壮阔，宋诗更喜欢从一片叶、一枝花中发现宇宙——这是美学上的东方极简主义。

## 🗣️ 家长讲解话术

- 孩子可能问 **"为什么门不开？"** → 可以说："因为主人可能出去了，或者主人爱惜花园里的苔藓，不想让人踩坏了。诗人猜测主人的心情：'应怜屐齿印苍苔'——大概是主人怜惜苔藓吧，所以不开门。这是诗人善意地理解主人，不是抱怨。"

- 孩子可能问 **"一枝红杏能代表整个春天吗？"** → 可以说："虽然只有一枝，但这一枝告诉了我们：墙里面的园子里，还有好多好多的花，满满的春色！就像你看到一片羽毛，就知道附近有鸟；看到一缕烟，就知道某处有人。'以小见大'，这是一种思考方式。"

- 孩子可能问 **"春色为什么关不住？"** → 可以说："花开是自然的力量，高墙只是人造的边界。花不在乎墙，只管往有阳光、有空气的地方生长。这是一个很美的比喻：生命力是无法被关住的！"

- **共读小活动（寻春游戏）**：春天出门，和孩子找一找：哪里有花或草从不起眼的角落（墙缝、砖隙、路边）长出来？感受"春色满园关不住，一枝红杏出墙来"的自然力量。
