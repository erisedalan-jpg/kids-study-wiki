# 登鹳雀楼

> **一句话**: 站在鹳雀楼上，看白日依山，看黄河入海——欲望看得更远，就必须更上一层楼，这是一首写尽人生进取精神的千古名诗。
> **English**: From the Stork Tower, watch the sun sink behind mountains and the Yellow River flow to the sea — to see further, you must always climb one floor higher.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**: 唐 · 王之涣
**体裁**: 五言绝句

```
登鹳雀楼
王之涣

白日依山尽，
黄河入海流。
欲穷千里目，
更上一层楼。
```

**拼音**:
- bái rì yī shān jìn，
- huáng hé rù hǎi liú。
- yù qióng qiān lǐ mù，
- gèng shàng yī céng lóu。

**白话译文**:
夕阳缓缓地沿着山峦落下，直到消失在山的尽头；
黄河奔腾着向东流去，汇入大海。
如果你想要看到千里之外更远处的风景，
那就再登上一层楼吧！

**朗读节奏**:
白日 / 依山尽，黄河 / 入海流。
欲穷 / 千里目，更上 / 一层楼。

---

## 🧒 给 3-6 岁（共读版）

你有没有站在高高的地方看风景？站得越高，能看到的地方越远！

王之涣登上了一座叫"鹳雀楼"的高楼（在今天的山西省）。站在楼上，他看到：
- 太阳慢慢沉到山后面（"白日依山尽"）
- 黄河的水哗哗地流向大海（"黄河入海流"）

他想看到更远的地方，怎么办？
**再往上爬一层！**（"更上一层楼"）

这不只是说楼，也是一个人生道理：想要看到更多、做到更好，就要继续努力、往前一步！

**共读小活动**:
- **体验一下**: 在家里，从地上蹲下来看，再站起来看，再搬个凳子站上去看——是不是越高看到越多？
- **画一画**: 画一座高楼，旁边有落日和大河，楼上有一个小人在看风景。
- **说一说**: 你想到更高的地方去看什么？

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 鹳雀楼 | guàn què lóu | 古代楼名，在今山西省永济市，因鹳雀（水鸟）常在此栖息得名 |
| 白日 | bái rì | 白天的太阳，即夕阳 |
| 依 | yī | 依傍，沿着，靠着 |
| 尽 | jìn | 到达尽头，消失 |
| 黄河 | huáng hé | 中国第二长河，因水中含沙多呈黄色 |
| 欲 | yù | 想要，希望 |
| 穷 | qióng | 穷尽，看完、看遍 |
| 千里目 | qiān lǐ mù | 能看到千里之外的目光/视野 |
| 更上 | gèng shàng | 再往上，继续向上 |
| 一层楼 | yī céng lóu | 再高一层 |

### 内容与情感

全诗分两层：

**前两句（写景）**: 落日依山、黄河入海，是壮阔的自然景观。诗人站在高处，视野无限延伸——向西看见落日，向东望见大河。

**后两句（言志）**: 由眼前的景色引发人生哲理——想要看得更远、做得更好，就要继续努力，"更上一层楼"。

这首诗的高明之处在于从具体的登楼行为，引出了普遍的人生道理，使诗句具有了超越时空的生命力。

### 写作手法

- **情景交融**: 前两句写景，后两句言志，景与志无缝连接
- **对仗工整**: "白日依山尽"对"黄河入海流"，动词、名词均对应工整
- **以小见大**: 从"再登一层楼"这个小动作，生发出人生进取的大道理
- **双重含义**: "更上一层楼"既是字面意思（再上一层），也是比喻（更进一步）

### 朗读建议

前两句气势磅礴，要读得沉稳有力，感受落日与大河的壮阔；"欲穷千里目"稍微停顿，带着期待；"更上一层楼"读出坚定的决心，语调上扬，有力量感。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**王之涣**（688—742年），字季凌，蒲州（今山西永济）人，盛唐著名诗人。其传世之作不多，但《登鹳雀楼》和《凉州词》（"黄河远上白云间"）已足以使他名垂青史。他与王昌龄、高适并称"边塞诗三大家"，但《登鹳雀楼》是一首山水言志诗，并非边塞诗。

鹳雀楼（又名鹳鹊楼）位于今山西省永济市，唐代为三层木楼，是登高望远的胜地。后楼毁于元代，现存建筑为现代重建。

### 文学史地位

《登鹳雀楼》与崔颢《黄鹤楼》并称登楼诗双璧。"更上一层楼"是中国文化中最著名的励志名句之一，其影响超出诗歌领域，深入日常语言。

此诗被誉为"五绝之冠"（五言绝句中最优秀的作品之一），收录于《唐诗三百首》、各版教材，是中国人从儿童时期便耳熟能详的经典。

### 鉴赏与对比

与陈子昂《登幽州台歌》（"念天地之悠悠，独怆然而涕下"）的孤独悲壮相比，《登鹳雀楼》更积极进取；与杜甫《登高》的沉郁顿挫相比，王之涣的诗更轻快明朗。

"白日依山尽，黄河入海流"的壮阔时空感，在二十个字内完成了从西（落日）到东（入海）的空间跨越，和从白天到黄昏的时间流动，是格律诗在有限文字中呈现无限空间的典范。

### 文化影响

- 收录于《全唐诗》卷二五三，王之涣诗
- 二年级语文教材必背，是小学阶段最重要的励志古诗
- "更上一层楼"成为中国日常用语，用于祝贺别人再创佳绩
- 鹳雀楼（今山西永济）已成为著名文化旅游景点，以此诗为核心

## 🗣️ 家长讲解话术

- 孩子可能问 **"更上一层楼只是说楼吗？"** → 可以说："字面意思是再上一层楼，这样就能看得更远。但更重要的意思是：如果你想取得更大的成绩，就要再努力一点，不能满足于现在的样子。这是一句励志的话！比如你学游泳，已经学会了，但想游得更快——就要'更上一层楼'，继续练习。"

- 孩子可能问 **"鹳雀楼在哪里？"** → 可以说："在中国的山西省，靠近黄河。那个地方可以看到黄河奔流向东的壮观景色。现在那里还有一座重建的鹳雀楼，可以爬上去体验'更上一层楼'的感觉！"

- **共读活动（体验式朗诵）**: 爬楼梯时，每上一层朗读一句诗，到顶层一起大声说"更上一层楼！"，体验诗中登高的感觉。

- **共读活动（画诗境）**: 画一座多层的楼，旁边有落日、大山和河流，楼顶有一个人向远处望，体会诗人站在高处的视野。
