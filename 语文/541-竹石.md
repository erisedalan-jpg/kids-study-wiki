# 竹石

> **一句话**：郑板桥借扎根破岩、千磨不弯的竹子，写出了自己面对任何风吹雨打都绝不妥协的倔强人格。
> **English**: Zheng Banqiao uses bamboo clinging to a crevice in the rock — withstanding a thousand storms from every direction — to portray his own unbending spirit against all adversity.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：清 · 郑燮（郑板桥）
**体裁**：七言绝句

```
竹石
郑燮

咬定青山不放松，
立根原在破岩中。
千磨万击还坚劲，
任尔东西南北风。
```

**拼音**：
- yǎo dìng qīng shān bù fàng sōng，
- lì gēn yuán zài pò yán zhōng。
- qiān mó wàn jī hái jiān jìn，
- rèn ěr dōng xī nán běi fēng。

**白话译文**：
竹子咬紧青山不肯放松，
原来它的根就扎在破裂的岩石之中。
经过千万次的磨砺和打击，还是坚韧有力，
任凭你东西南北哪个方向来的风！

**朗读节奏**：
咬定 / 青山 / 不放松，/ 立根 / 原在 / 破岩中。
千磨 / 万击 / 还坚劲，/ 任尔 / 东西 / 南北风。

## 🧒 给 3-6 岁（共读版）

你见过竹子吗？竹子长得又高又直，风来了也弯一弯，但是不倒！

这首诗里有一棵竹子，它长在破裂的石头缝里——那么硬的岩石，竹子的根扎进去，怎么也不松开！

风从这边吹，从那边吹，从四面八方吹，竹子弯了弯，但是——不倒！

诗人郑板桥说：竹子就像这样，不管什么风都吹不倒它！我也要做这样的人，坚强，不害怕困难！

**共读小活动**：
- **演一演**：站直身体，想象你是一棵竹子，风从各个方向吹来，你弯一弯，但是不倒！
- **聊一聊**：你见过什么东西在艰难的地方生长，还长得很好？

### 创作背景

郑燮（1693—1765），号板桥，江苏兴化人。他工书画，尤擅画竹、兰、石，与墨竹相关的题画诗写了许多，这首《竹石》是其中最广为流传的。

这首诗是郑板桥为自己所画的竹石图题写的诗（题画诗），既是对画作的说明，也是诗人自况——他一生仕途坎坷，曾因为灾情上书朝廷而得罪上司，最终辞官，靠卖字画为生，但始终保持自己的风骨。

### 内容与情感

全诗四句，每句都有双层含义：

1. **咬定青山不放松**：竹子紧扣青山不松 → 紧守自己的立场、信念不动摇
2. **立根原在破岩中**：竹根在岩石裂缝中 → 在艰难的处境中仍然扎根
3. **千磨万击还坚劲**：历经打击仍坚强 → 面对磨难仍保持坚韧
4. **任尔东西南北风**：任何方向的风都不怕 → 不管哪种压力或诱惑，都不妥协

最后一句"任尔东西南北风"气势磅礴，是全诗的高潮——"任尔"二字有一种蔑视和自信，不是害怕，而是从容的挑战：你来吧，怎样都不怕！

### 艺术特色

- **拟人**："咬定"——竹子像用牙关咬住青山，意志力十足
- **递进**：从"扎根破岩"到"千磨万击"到"任尔风"，越来越有气势
- **托物言志**：全诗写竹，字字是志——这是题画诗的最高境界

## 🎓 给 12+（进阶版）

### 郑板桥与"扬州八怪"

**郑燮**（1693—1765），字克柔，号板桥，清代著名书画家、诗人，"扬州八怪"之一。

"扬州八怪"是清代乾隆年间活跃于扬州的一批风格独特、不媚俗的画家群体，他们打破清代主流绘画的保守风格，追求个性表达。郑板桥以画竹、兰最著名，他的竹画以"竟体傲石，不服一尘"著称。

郑板桥做过山东范县、潍县的知县，为人清廉，关心百姓。潍县大旱，他私自开仓放粮赈济，触怒上司，最终辞官（"两袖清风来去"是他的自述）。回扬州后卖字画为生，晚年境遇艰难，却始终不折腰——这就是"咬定青山不放松"的真实注脚。

### 竹在中国文化中的象征

竹在中国文化中有特殊地位，是"四君子"之一（梅兰竹菊）：
- **梅**：傲雪而开，高洁不屈
- **兰**：幽香淡雅，高洁清远
- **竹**：虚心劲节，坚韧不拔
- **菊**：晚节高洁，不从流俗

竹的象征来自其生物特性：中空（虚心）、劲节（节气）、四季常绿（恒心）、越砍越密（生命力顽强）。中国文人以竹自比，是儒家"君子"品格的植物化身。

### 与《石灰吟》的比较

| | 《石灰吟》（于谦） | 《竹石》（郑板桥） |
|---|---|---|
| 咏物 | 石灰 | 竹 |
| 关键意象 | 清白（洁白） | 坚劲（坚韧） |
| 精神重心 | 宁死不污，留清白 | 千磨不屈，任风吹 |
| 历史背景 | 于谦被冤杀，清白留名 | 郑板桥辞官，风骨留世 |
| 共同点 | 托物言志，品格自表 | 托物言志，品格自表 |

两首合读，于谦是"清白不变"的坚守，郑板桥是"千磨不屈"的顽强，互补构成完整的君子品格图像。

## 🗣️ 家长讲解话术

- 孩子可能问 **"竹子怎么咬定青山？"** → 可以说："当然，竹子没有嘴，'咬定'是拟人——就是说竹子的根系紧紧地抓住山体，像用牙关咬住一样，绝对不松开。你试着握紧拳头不松开——这种感觉就是'咬定'！"

- 孩子可能问 **"任尔东西南北风是什么意思？"** → 可以说："'任尔'就是'任凭你'、'随便你来'的意思，语气很有气魄。东西南北风就是四面八方的风——哪个方向的风来都不怕！竹子（或者说诗人）在说：你想从哪个方向打压我，都来吧，我不怕！"

- 孩子可能问 **"郑板桥为什么那么喜欢画竹子？"** → 可以说："因为郑板桥把自己的性格投射到了竹子上——他是个很倔强、有骨气的人，不喜欢向别人低头。竹子坚韧、有节气，就像他自己一样。画竹子的时候，他其实是在画自己的心情和志向。"

- **共读小活动（画竹）**：和孩子一起用水彩或毛笔（或普通笔）画一棵竹子，注意竹节、竹叶的形状。画完后讨论：你画的竹子在风中怎样，是弯的还是直的？体会"千磨万击还坚劲"的感觉。
