# 笔画

> **一句话**：汉字是由一笔一画组成的，每一画都有自己的名字和写法——这些就是笔画。
> **English**: Chinese characters are made of individual strokes; each stroke has a name and a specific way of writing.

---

## 🧒 给 3-6 岁（共读版）

你看，"一"这个字，就是一横——从左边画到右边，一气呵成。"丨"是一竖，从上往下写。

汉字就像用笔搭积木，每一小块就叫**笔画**。学会了基本笔画，就能写出所有的汉字！

### 8 种基本笔画

| 笔画 | 名字 | 怎么记 |
|------|------|--------|
| 一 | 横 (héng) | 像一条平躺的线，从左到右 |
| 丨 | 竖 (shù) | 像一棵大树，从上往下 |
| 丿 | 撇 (piě) | 像风吹弯的草，从右上到左下 |
| 乀 | 捺 (nà) | 像大雁飞翔，从左上到右下，末尾有"捺脚" |
| 丶 | 点 (diǎn) | 像小雨滴，从上往右下轻轻一点 |
| 乙 | 折 (zhé) | 像折弯的路，转一个角 |
| 亅 | 钩 (gōu) | 像小钩子，笔画末尾带一个小钩 |
| 提 | 提 (tí) | 像向右上挑起，从左下到右上 |

**共读小活动：** 用手指沾清水，在桌面上练习写每种笔画。说出笔画名字再写，写完等它蒸发，再换下一个！

## 📚 给 6-12 岁（自读版）

### 基本笔画与派生笔画

笔画可以分为**基本笔画**和**派生笔画**（复合笔画）两大类。

**基本笔画**（8 种）：横、竖、撇、捺、点、折、钩、提

**常见派生笔画**（由基本笔画组合变化而来）：

| 派生笔画 | 组合方式 | 例字 |
|---------|---------|------|
| 横折 | 横 + 折 | 口、日、国 |
| 横钩 | 横 + 钩 | 字、买 |
| 竖钩 | 竖 + 钩 | 小、水 |
| 竖弯钩 | 竖 + 弯 + 钩 | 儿、九 |
| 横折钩 | 横 + 折 + 钩 | 月、用 |
| 撇折 | 撇 + 折 | 女、么 |
| 横撇 | 横 + 撇 | 又、父 |

### 笔画与汉字结构的关系

每个汉字都由若干笔画组成。笔画数量也是字典排序的依据之一：

- **1 画**：一、乙
- **2 画**：二、人、八
- **3 画**：三、口、山
- **10 画以上**：通常是复杂字，如"赢"有 17 画

### 笔画的书写要求

1. **起笔**：每画要有明确的起笔位置
2. **行笔**：运笔方向要准确
3. **收笔**：每画末尾要"顿笔"或"提笔"，不能虎头蛇尾

在**田字格**里练习时，笔画的长短、位置都有讲究，要与格子的中线、横线对齐。

## 🎓 给 12+（进阶版）

### 楷书笔画系统与书法美学

现代语文教学中规定的笔画体系，源自**楷书**（也叫正楷、真书）的书写规范。楷书是汉字演变的重要阶段，形成于东汉末期，在魏晋南北朝逐渐成熟，唐代达到顶峰（代表：欧阳询、颜真卿、柳公权）。

**历代书法家对笔画的贡献**：

- **永字八法**（传为王羲之总结）：以"永"字包含 8 种基本笔画为例，总结了楷书笔画的书写美学——侧（点）、勒（横）、努（竖）、趯（钩）、策（提）、掠（撇）、啄（短撇）、磔（捺）。这与现代课标笔画体系有传承关系。

- **笔画的"起、行、收"三段论**：书法美学认为每一笔画都要有明确的起笔、行笔和收笔，体现了"有始有终"的审美观。

### 笔画数量的历史演变

甲骨文时期，汉字笔画繁多，方向不定。随着书写材料从甲骨、青铜转向竹简、毛笔，笔画逐渐规整化：

- **甲骨文**：图形性强，笔画尚无统一体系
- **篆书**：笔画匀称，多弧线，无明显笔画种类区分
- **隶书**：开始出现"蚕头雁尾"的捺，笔画感增强
- **楷书**：笔画种类固定，形成现代笔画系统

### 当代笔画标准

1997 年国家语言文字工作委员会发布的《现代汉语通用字笔顺规范》，规定了 7 种基本笔画（横竖撇捺折点钩），各地教材细分方式略有差异，统编版语文教材采用 8 种（加入"提"）。

## 🗣️ 家长讲解话术

- 孩子可能问 **"笔画有多少种？"** → 可以说："我们课本里学 8 种最基本的：横、竖、撇、捺、点、折、钩、提。但它们可以组合变化，派生出更多种类，就像积木的基本形状只有几种，但能搭出很多不同的东西。"

- 孩子可能问 **"为什么要按顺序写笔画？"** → 可以说："按照笔顺写，写出来的字更好看，而且速度更快。就像搭积木要按从下往上的顺序，不然容易倒。"

- **共读小活动**：制作**笔画字卡**——在卡纸上写上每种笔画和它的名字，翻面说名字猜笔画，或者说名字用手指在空中比划。也可以在沙盘或米粒盘中练习，增加触感体验。
