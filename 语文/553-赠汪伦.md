# 赠汪伦

> **一句话**: 李白乘船离开，忽然听到岸上汪伦踏歌送别——再深的桃花潭水，也比不过这份真挚的友情。
> **English**: As Li Bai's boat pulls away, his friend Wang Lun stamps and sings on the shore — no depth of water can match the depth of true friendship.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**: 唐 · 李白
**体裁**: 七言绝句

```
赠汪伦
李白

李白乘舟将欲行，
忽闻岸上踏歌声。
桃花潭水深千尺，
不及汪伦送我情。
```

**拼音**:
- lǐ bái chéng zhōu jiāng yù xíng，
- hū wén àn shàng tà gē shēng。
- táo huā tán shuǐ shēn qiān chǐ，
- bù jí wāng lún sòng wǒ qíng。

**白话译文**:
李白我坐上小船，正要出发离开这里；
忽然听到岸边传来汪伦踏步唱歌的声音。
就算桃花潭的水深达千尺，
也比不上汪伦送我时的那份深情厚意。

**朗读节奏**:
李白 / 乘舟 / 将欲行，忽闻 / 岸上 / 踏歌声。
桃花潭水 / 深千尺，不及 / 汪伦 / 送我情。

## 🧒 给 3-6 岁（共读版）

你有最好的朋友吗？如果你要去很远的地方，好朋友来送你，你会不会很感动？

李白有个好朋友叫汪伦。有一天，李白要坐船走了。他上了船，船要开了……

突然，岸上传来了歌声！是汪伦！他边唱边跳着送李白，歌声传到了河面上。

李白被感动了，他写了这首诗：桃花潭的水（这里水很深很深）有一千尺那么深，但是还比不上汪伦送别时的那份友情！

你觉得，什么比什么更深？

**共读小活动**:
- **演一演**: 一个人扮演李白坐船，另一人扮演汪伦在岸上唱歌挥手，表演送别的场景。
- **说一说**: 如果你要离开，哪个朋友会来送你？为什么？
- **画一画**: 画一艘小船在河上，岸边有人唱歌挥手，周围有桃花开放。

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 赠 | zèng | 赠送，这里指写诗赠给别人 |
| 乘舟 | chéng zhōu | 坐上船 |
| 将欲行 | jiāng yù xíng | 正要出发，"将"表示即将 |
| 忽闻 | hū wén | 忽然听到 |
| 踏歌声 | tà gē shēng | 踏着节奏唱歌的声音，古代一种民间歌唱方式 |
| 桃花潭 | táo huā tán | 地名，在今安徽省泾县，以桃花闻名 |
| 深千尺 | shēn qiān chǐ | 深达千尺，形容极深（夸张） |
| 不及 | bù jí | 比不上，不如 |

### 内容与情感

这首诗结构简洁，情感层层递进：
1. **情境**: 李白乘船要走（将欲行）
2. **转折**: 忽然听到踏歌声（忽闻）
3. **对比**: 潭水深千尺（极深）
4. **升华**: 还是比不上友情（不及）

用"桃花潭水深千尺"来衬托汪伦送别情谊之深，这是用具体的物理深度来比拟情感深度的经典手法，使抽象的友情变得可以感知。

### 写作手法

- **夸张与比喻**: "桃花潭水深千尺"用夸张的水深作比，突出友情的珍贵
- **衬托**: 以潭水之深衬托友情更深，"不及"一词将对比推向高潮
- **以景写情**: 用桃花、潭水等自然意象承载情感，情景交融
- **先叙后议**: 前两句叙事，后两句抒情，叙议结合

### 朗读建议

前两句节奏轻快，描述动态场景；"桃花潭水深千尺"语调沉稳，充满深意；"不及汪伦送我情"语调温暖上扬，读出感动与深情。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**李白**（701—762年），"诗仙"，详见→[[563-静夜思|静夜思]]词条。

汪伦，唐代泾县（今安徽）人，喜爱诗歌，仰慕李白。据唐人传说，汪伦以书信邀李白来访，信中写道"此地有十里桃花，万家酒店"，李白欣然赴约。到后才知"十里桃花"是桃花潭旁一个名叫"十里"的地方，"万家酒店"是一家姓万的酒馆——这是汪伦的善意玩笑。李白不但没生气，反而被汪伦的热情款待深深打动，离别时汪伦踏歌送别，李白遂作此诗以赠。（上述传说细节待家长核对）

### 文学史地位

《赠汪伦》是中国送别诗中最著名的作品之一。"送别诗"是中国古典诗歌的重要门类，从《诗经》的"执子之手"到王维的"劝君更尽一杯酒"，表达离别情感是诗人的重要使命。李白此诗以比喻和夸张创造了"深情"的极致表达，"桃花潭水深千尺，不及汪伦送我情"成为后世引用频率最高的友情名句之一。

### 鉴赏与对比

与王勃《送杜少府之任蜀州》（"海内存知己，天涯若比邻"）相比，《赠汪伦》更具体、更个人化；与王维《送元二使安西》（"劝君更尽一杯酒"）相比，李白更直接、更热烈。三首诗都是送别名篇，风格各异，体现了唐代诗人不同的性格与诗风。

### 文化影响

- 收录于《全唐诗》卷一七四，李白诗
- 统编版小学语文教材一至二年级推荐篇目
- "桃花潭水深千尺，不及汪伦送我情"是最常被引用的友情诗句之一
- 安徽泾县桃花潭景区以此诗为文化名片，是著名旅游地

## 🗣️ 家长讲解话术

- 孩子可能问 **"踏歌是什么？"** → 可以说："踏歌是古代的一种唱歌方式，一边唱歌一边用脚踏地打节拍，就像现在有人唱歌会拍手一样。汪伦用这种方式送李白，特别热情！"

- 孩子可能问 **"桃花潭真的有千尺深吗？"** → 可以说："不是真的千尺深，这是诗人用的'夸张'手法，为了说明友情更深。就像我们说'我喜欢你一百万'，不是真的一百万，是表示非常非常喜欢。"

- **共读活动（比一比）**: 问孩子："你觉得什么东西最深？大海、湖泊还是天空？"然后说："可是诗人说，这些都比不过汪伦的友情深！"帮助孩子理解比喻的情感表达。

- **共读活动（全家朗诵）**: 一家人一起背诵，重点体验"不及汪伦送我情"的情感，感受朋友友情的珍贵。
