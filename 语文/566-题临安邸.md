# 题临安邸

> **一句话**：林升在旅馆墙上写下这首诗，讽刺南宋权贵们在杭州歌舞升平、忘了收复失地的可耻。
> **English**: Lin Sheng inscribed this poem on an inn wall to mock the Southern Song rulers who indulged in pleasure in Hangzhou while forgetting the lost northern territories.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：宋 · 林升
**体裁**：七言绝句

```
题临安邸
林升

山外青山楼外楼，
西湖歌舞几时休？
暖风熏得游人醉，
直把杭州作汴州。
```

**拼音**：
- shān wài qīng shān lóu wài lóu，
- xī hú gē wǔ jǐ shí xiū？
- nuǎn fēng xūn dé yóu rén zuì，
- zhí bǎ háng zhōu zuò biàn zhōu。

**白话译文**：
青山之外还有青山，楼台之外还有楼台，
西湖边的歌舞什么时候才能停歇？
温暖的春风把游赏的人熏得如同醉酒，
竟然把杭州当成了故都汴京！

**朗读节奏**：
山外 / 青山 / 楼外楼，/ 西湖 / 歌舞 / 几时休？
暖风 / 熏得 / 游人醉，/ 直把 / 杭州 / 作汴州。

## 🧒 给 3-6 岁（共读版）

你知道"西湖"吗？西湖在杭州，湖面很大很漂亮，旁边有好多楼，好多山。

很久以前，中国有一些大官，他们应该管理国家、保护老百姓，可是他们每天只知道在西湖边唱歌、跳舞、喝酒，玩得特别开心，完全忘了自己的责任。

有个叫林升的诗人看到这种情况，很生气，就在旅馆的墙上写了这首诗，批评那些只知道玩乐的大官们。

**共读小活动**：
- **聊一聊**：如果你是一位大官，你觉得每天最重要的事是什么？
- **找一找**：诗里出现了哪两个城市的名字？（杭州、汴州）

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 临安 | lín ān | 南宋都城，今浙江杭州 |
| 邸 | dǐ | 旅馆、官邸 |
| 几时休 | jǐ shí xiū | 什么时候才停止 |
| 熏 | xūn | 熏陶、浸染（此处指暖风的气息） |
| 游人 | yóu rén | 游览游乐的人（双关：指游玩的官员） |
| 直把 | zhí bǎ | 竟然，径直将…当作… |
| 汴州 | biàn zhōu | 北宋都城，今河南开封，已被金国占领 |

### 历史背景

北宋被金国攻灭（1127年），皇帝宋徽宗、宋钦宗被俘，史称"靖康之耻"。南宋朝廷逃到临安（杭州），本应图谋北伐、收复故都汴京，却偏安江南，沉迷享乐。

林升写这首诗时，南宋君臣醉生梦死，岳飞等主战派被打压，主和派把持朝政——诗人满腔愤慨化为这二十八字，留在客栈的墙壁上。

### 内容与情感

1. **第一句**（山外青山楼外楼）：杭州景色极美，无尽的山与楼——繁华到底
2. **第二句**（西湖歌舞几时休）：质问，这歌舞什么时候才会停？隐含"你们忘了国仇家恨了吗"
3. **第三句**（暖风熏得游人醉）："暖风"双关——既是自然的暖风，也是醉生梦死的靡靡之风
4. **第四句**（直把杭州作汴州）：点睛。汴州（开封）已经是敌人的土地，你们竟把这里当成故都安享太平！

### 写作手法

- **双关**：暖风（自然/靡靡之风）、游人（游客/醉生梦死的官员）
- **讽刺**：不直接批评，用"几时休"的疑问和"直把"的感慨表达愤慨
- **对比**：杭州歌舞vs汴州沦陷，安逸vs国耻，形成强烈反差

## 🎓 给 12+（进阶版）

### 作者与时代背景

**林升**，字云友，生卒年不详，南宋诗人，平江（今浙江）人。史料记载甚少，仅以此诗留名。这首诗可能写于宋孝宗至宋光宗年间（12世纪后期），南宋偏安临安已逾半个世纪。

南宋建立初，岳飞、韩世忠等主战派奋力北伐，但宋高宗与秦桧力主议和。绍兴十一年（1141年）宋金签订"绍兴和议"，南宋向金称臣纳贡，岳飞被冤杀。此后数十年，南宋朝野以"偏安"为常态，"临安"（"临时安顿"之意）的地名本身便是讽刺——最终却成了真正的"长安"。

### 讽喻诗的政治语境

中国诗歌传统中有"诗谏"的功能——诗人借诗批评时政，既保留文学形式，又规避直谏的风险。《题临安邸》是宋代讽喻诗的代表作，与陆游《示儿》、秋夜将晓出篱门迎凉有感》共同构成南宋亡国之痛的诗歌群像。

"直把杭州作汴州"在文学史上成为固定典故，后世每遇统治者苟安不思进取，文人便引用此语。

### 文化地位与教育意义

这首诗在小学阶段被选入教材，主要目标是：
1. 培养家国情怀，理解"靖康之耻"的历史
2. 学习讽喻这一文学手法
3. 认识"汴州"（开封）与"临安"（杭州）的历史关联

与陆游的《示儿》《秋夜将晓出篱门迎凉有感》共读，可帮助学生建立南宋爱国诗歌的整体认识。

## 🗣️ 家长讲解话术

- 孩子可能问 **"汴州是哪里？"** → 可以说："汴州就是现在的河南开封，曾经是北宋的首都，非常繁华。后来被金国攻占，宋朝皇帝被抓走了，剩下的人跑到杭州（临安）重建朝廷——但他们没有想着收复开封，只是在杭州继续享乐。"

- 孩子可能问 **"诗人为什么把诗写在旅馆墙上？"** → 可以说："古代没有报纸微博，诗人有话要说，就写在墙上——这叫'题壁'，路过的人都能看到，相当于发朋友圈！这首诗被大家传抄，才流传到今天。"

- 孩子可能问 **"几时休是什么意思？"** → 可以说："'几时休'就是'什么时候才停'。诗人在问：你们在西湖边唱歌跳舞，什么时候才停？言下之意是：你们应该去打仗、收复失地，而不是在这里享乐！"

- **共读小活动（地图对照）**：在地图上找到开封（汴京）和杭州（临安），看看两座城市相距多远，感受南宋朝廷"逃"了多远的距离，帮助理解诗歌的历史背景。
