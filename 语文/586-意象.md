# 意象

> **一句话**：意象是**融入了主观情感的客观物象**——是诗歌的基本单位，"意"（情感）+"象"（物象）的结合，读懂意象是读懂中国古诗词的钥匙。
> **English**: *Poetic image* (yìxiàng) is an objective object infused with subjective emotion — the fundamental unit of Chinese classical poetry: "yì" (meaning/emotion) + "xiàng" (image) — interpreting images is the key to interpreting Chinese poetry.

---

## 🎓 给 12+

**核心定义**：
- **物象**：客观存在的事物（月、花、柳、雁）
- **意象**：物象+诗人情感（"明月"加上思乡之情，成为"思乡意象"）

**特征**：
1. **意+象**：不是单纯的客观描写，而是物象+情感
2. **历代积淀**：相同意象经过历代诗人使用，承载固定情感（如"杨柳"=送别、"梅"=高洁）
3. **意境的构成单位**：多个意象组合形成意境（"枯藤、老树、昏鸦"叠加成苍凉意境）

**常见意象与情感**：
| 意象 | 寄寓情感 | 代表诗句 |
|---|---|---|
| 明月 | 思乡、怀人、孤独 | "举头望明月，低头思故乡" |
| 杨柳 | 送别、留恋 | "杨柳东风树，青青夹御河" |
| 梅 | 高洁、坚贞、报春 | "墙角数枝梅，凌寒独自开" |
| 菊 | 隐逸、高洁、秋愁 | "采菊东篱下，悠然见南山" |
| 莲 | 君子高洁、爱情 | "出淤泥而不染" |
| 大雁 | 思乡、漂泊、书信 | "雁字回时，月满西楼" |
| 鸿雁 | 同上 | "鸿雁几时到，江湖秋水多" |
| 柳絮 | 飘零、伤感 | "杨花落尽子规啼" |
| 鹧鸪 | 哀愁、思乡 | "江晚正愁余，山深闻鹧鸪" |
| 杜鹃（子规） | 哀怨、思归 | "子规夜半犹啼血，不信东风唤不回" |
| 乌鸦/昏鸦 | 凄凉、衰败 | "枯藤老树昏鸦" |
| 流水 | 时间流逝、愁绪 | "抽刀断水水更流，举杯销愁愁更愁" |

**意象与意境的区别**：
- **意象**：单个、具体、点
- **意境**：多个意象组合营造的整体氛围、面

**鉴赏方法**：
1. **找意象**：诗中出现哪些主要物象？
2. **辨情感**：这个意象惯常承载什么情感？
3. **看组合**：意象之间形成什么意境？
4. **悟主题**：意象群最终指向什么主题？
