# 成语

> **一句话**：成语是**汉语中代代相传、结构定型、意义凝练**的固定短语——多由四字组成，承载丰富的历史典故、寓言故事或哲理思想，是中华文化的浓缩。
> **English**: A Chinese *idiom* (chéngyǔ) is a fixed phrase passed down through generations, with set structure and condensed meaning — typically four characters, embodying historical allusions, fables, or wisdom; a concentrated form of Chinese culture.

---

## 📚 给 6-12 岁

成语就是**四个字（大多数）的固定说法**，不能随便改字、改顺序：
- ✅ 守株待兔（来自寓言）
- ❌ 守株等兔（错的，不能改）

成语有四个特点：
1. **固定结构**：字数和顺序不能改
2. **整体意义**：合起来的意思和单个字不一定一样（"井底之蛙"不是真的青蛙）
3. **多有典故**：背后常常是一个故事（守株待兔、刻舟求剑、画蛇添足）
4. **简练优美**：用四个字表达复杂意思

学成语小窍门：**学一个成语，先听背后的故事，再记意思和用法**。

## 🎓 给 12+

**来源分类**：
1. **历史故事**：望梅止渴（曹操）、负荆请罪（廉颇蔺相如）、纸上谈兵（赵括）、卧薪尝胆（勾践）
2. **寓言故事**：守株待兔、刻舟求剑、画蛇添足、揠苗助长、自相矛盾
3. **神话传说**：精卫填海、夸父逐日、女娲补天
4. **古典文学**：山长水阔、笔走龙蛇、举案齐眉
5. **古诗文摘录**：山雨欲来风满楼（许浑）、柳暗花明又一村（陆游）
6. **外来翻译**：火中取栗（法国寓言）、象牙之塔（西方）

**结构类型**（四字成语为主）：
| 结构 | 例 |
|---|---|
| 主谓 | 风和日丽 / 名落孙山 |
| 动宾 | 守株待兔 / 画饼充饥 |
| 并列 | 风调雨顺 / 山清水秀 |
| 偏正 | 一鸣惊人 / 雪中送炭 |
| 紧缩 | 鞭长莫及 / 唾手可得 |

**成语的语言特色**：
- **音律美**：四字结构平仄相对，朗读铿锵有力
- **意象密**：四字之内可有完整画面（杏花春雨）
- **典故深**：背后常有数百年文化积淀

**使用注意**：
1. **意义准确**：不能望文生义（"七月流火"是天气转凉，非酷热）
2. **感情色彩**：分褒义、贬义、中性（处心积虑—贬义；锲而不舍—褒义）
3. **适用对象**：豆蔻年华—少女；耄耋之年—老人
4. **语体差异**：书面语和口语成语用法不同
