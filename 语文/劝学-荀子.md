## 🎓 劝学（荀子）

### 作者与时代

**荀子**（约前316—前237），名况，字卿，战国末期赵国人，儒家代表人物之一，与孟子并称，但思想倾向有别：孟子主"性善论"，荀子主**"性恶论"**，认为人性本恶，须经后天学习、礼义教化才能成善。荀子曾在齐国稷下学宫任祭酒（最高学术职位），晚年在楚国任兰陵令，著有《荀子》三十二篇。韩非、李斯均为其学生。

《劝学》是《荀子》开篇之作，集中体现其教育哲学：学习是人成就自我的唯一途径，强调积累、专注与坚持。

---

### 原文（节录，统编版必修上选段）

> 君子曰：学不可以已。
>
> 青，取之于蓝，而青于蓝；冰，水为之，而寒于水。木直中绳，𫐓以为轮，其曲中规，虽有槁暴，不复挺者，𫐓使之然也。故木受绳则直，金就砺则利，君子博学而日参省乎己，则知明而行无过矣。
>
> 吾尝终日而思矣，不如须臾之所学也；吾尝跂而望矣，不如登高之博见也。登高而招，臂非加长也，而见者远；顺风而呼，声非加疾也，而闻者彰。假舆马者，非利足也，而致千里；假舟楫者，非能水也，而绝江河。君子生非异也，善假于物也。
>
> 积土成山，风雨兴焉；积水成渊，鱼龙生焉；积善成德，而神明自得，圣心备焉。故不积跬步，无以至千里；不积小流，无以成江海。骐骥一跃，不能十步；驽马十驾，功在不舍。锲而舍之，朽木不折；锲而不舍，金石可镂。蚓无爪牙之利，筋骨之强，上食埃土，下饮黄泉，用心一也。蟹六跪而二螯，非蛇鳝之穴无可寄托者，用心躁也。

### 译文

**开篇**：君子说：学习不可以停止。

**比喻段**：靛青是从蓼蓝中提取的，却比蓼蓝更青；冰是水凝成的，却比水更寒冷。木料笔直合乎墨线，将它弯成车轮，其弯度合乎圆规，即使晒干了也不会再伸直，这是火烤使它弯曲的缘故。所以木料经墨线量过就能取直，刀剑在磨石上磨过就会锋利，君子广博学习、每天检察反省自己，就会智慧明达、行为没有过失。

**善假于物段**：我曾经整天思索，不如片刻间学习收获大；我曾经踮起脚尖眺望，不如登上高处看得广阔。登高招手，手臂并没有加长，但远处的人都能看见；顺风呼喊，声音并没有更响，但听到的人却更清楚。借助车马的人，不是脚力好，却能到达千里之外；借助舟船的人，不是善于游水，却能渡过大江大河。君子的天性并不比别人好，只是善于借助外物而已。

**积累段**：积土成山，风雨就会在那里兴起；积水成渊，鱼龙就会在那里生长；积累善行、养成德行，精神智慧自然具备，圣人的心境也就形成了。所以不积累半步，就无法到达千里之外；不汇聚细流，就无法形成江海。骏马一跳，不能超过十步；劣马走十天，功劳在于不停歇。雕刻到半途而废，朽木也不能折断；坚持雕刻不放弃，金石也可以镂刻。蚯蚓没有锋利的爪牙、强健的筋骨，向上能吃土，向下能饮泉水，因为用心专一。螃蟹有六条腿、两只大钳，但没有蛇鳝的洞穴就无处寄身，因为用心浮躁。

### 赏析

**结构**：全文围绕"学不可以已"展开，分三层论证：
1. **学习能改变人**（青出于蓝、直木为轮）——"变"的可能性
2. **善假于物是君子之道**（登高望远、借舟楫）——学习的方法
3. **积累、专一、持之以恒**（积土成山、骐骥驽马、蚓与蟹）——学习的态度

**艺术手法**：
- **比喻论证**：全篇以比喻为主要论证手段，共用约20个比喻，形象贴切，层层递进。
- **对比**：骐骥与驽马、蚯蚓与螃蟹，正反对比，强化论点。
- **排比**：大量排比句（"积土成山……积水成渊……积善成德"），气势磅礴，节奏鲜明。
- **用典**：无直接引经据典，而是大量从自然现象取喻，体现荀子重实证、重经验的风格。

**思想内涵**：荀子"性恶论"的逻辑基础是：人天性有缺陷，需要后天"学"与"礼"的矫正。《劝学》强调**后天努力**的决定性作用，"君子生非异也，善假于物也"一句道出核心：决定人成就高下的，不是天赋，而是是否善于借助外部资源（师长、书籍、礼义）持续学习。

### 高考考点

**必背句（统编版要求背诵全篇选段）**：

| 句子 | 常考情境 |
|------|------|
| 青，取之于蓝，而青于蓝 | 比喻学习可以超越起点；成语"青出于蓝" |
| 故木受绳则直，金就砺则利 | 比喻学习与自省使人完善 |
| 吾尝终日而思矣，不如须臾之所学也 | 强调学习胜于空想 |
| 君子生非异也，善假于物也 | 学习方法论；强调借助外力 |
| 故不积跬步，无以至千里 | 积累的重要性（常考默写） |
| 骐骥一跃，不能十步；驽马十驾，功在不舍 | 持之以恒的重要性 |
| 锲而不舍，金石可镂 | 坚持的力量（常考默写） |
| 蚓无爪牙之利……用心一也 | 专注的重要性 |

**翻译考点**（词类活用 / 特殊句式）：
- "青于蓝"——状语后置句
- "木直中绳"——定语后置
- "蚓无爪牙之利，筋骨之强"——定语后置（爪牙之利 = 锋利的爪牙）
- "上食埃土，下饮黄泉"——方位名词作状语（上/下）
- "假舆马者，非利足也"——"利"形容词活用作动词（使脚有利/快）

**鉴赏题型**：
- 分析比喻论证的作用
- 比较骐骥与驽马的对比论证效果
- 分析"善假于物"的现实意义
