# 孙权劝学

> **一句话**：孙权劝大将吕蒙读书，吕蒙发奋苦读，令鲁肃刮目相看——"士别三日，即更刮目相待"由此而来。
> **English**: Sun Quan urges his general Lü Meng to study; Lü Meng transforms himself through learning — and the idiom "see someone in a new light after three days" was born.

---

## 📖 原文 + 注释 + 译文

**出处**：《资治通鉴》，北宋·司马光主编
**体裁**：历史散文（文言）

```
孙权劝学

初，权谓吕蒙曰："卿今当涂掌事，不可不学！"蒙辞以军中多务。
权曰："孤岂欲卿治经为博士邪！但当涉猎，见往事耳。卿言多务，孰若孤？孤常读书，自以为大有所益。"
蒙乃始就学。及鲁肃过寻阳，与蒙论议，大惊曰："卿今者才略，非复吴下阿蒙！"
蒙曰："士别三日，即更刮目相待，大兄何见事之晚乎！"
肃遂拜蒙母，结友而别。
```

### 关键词注释

| 词 | 释义 |
|----|------|
| 谓 | 对……说 |
| 卿 | 古代君主对臣下的称呼，或朋友间的尊称 |
| 当涂掌事 | 主持政务，掌管事务 |
| 辞以 | 用……推辞 |
| 孤 | 古代诸侯王的自称（我） |
| 治经为博士 | 研究经典成为博士（不是现在的博士学位，指精通经典的人） |
| 但 | 只是，只要 |
| 涉猎 | 广泛浏览，不求深入 |
| 见往事 | 了解历史 |
| 孰若 | 哪里比得上 |
| 及 | 到……时候 |
| 才略 | 才能谋略 |
| 非复 | 不再是 |
| 吴下阿蒙 | 昔日没有学识的吕蒙（吴地的那个阿蒙） |
| 士别三日 | 有志之士分别三天（即短时间） |
| 更 | 重新 |
| 刮目相待 | 用新的眼光看待 |
| 大兄 | 对长辈或年长友人的尊称 |
| 见事 | 发现事情、认识到 |
| 遂 | 于是、就 |
| 拜 | 拜访 |

### 白话译文

当初，孙权对吕蒙说："你现在主持政务，不能不学习！"吕蒙用军中事务繁多来推辞。
孙权说："我难道是要你研究经典成为博士吗？只要广泛浏览，了解一些历史就行了。你说事务多，哪里比得上我呢？我经常读书，自认为大有益处。"
吕蒙于是开始学习。等到鲁肃路过寻阳，与吕蒙交谈，大吃一惊，说："你现在的才能谋略，已经不再是昔日吴地那个阿蒙了！"
吕蒙说："有志之士分别短短几天，就要用新的眼光来看，大哥您为什么发现这件事这么晚啊！"
鲁肃于是去拜见吕蒙的母亲，与吕蒙结为朋友后才离开。

## 🧒 给 3-6 岁（共读版）

孙权是一个大将军，他手下有个人叫吕蒙，打仗很厉害，但不爱读书。

孙权说："你现在做大事，不读书不行！"吕蒙说："我太忙了。"孙权说："我比你还忙，我还是每天读书！"

吕蒙就去读书了。后来，吕蒙变得很厉害，朋友见到他说："哇，你变了好多！"吕蒙说："三天不见就要刮目相看嘛！"

**共读小活动**：
- 讨论：读书能改变一个人什么？
- 如果你三天不见一个朋友，他可能有什么变化？

## 📚 给 6-12 岁（自读版）

### 人物分析

| 人物 | 特征 |
|------|------|
| 孙权 | 善于劝说，以身作则，不强迫，以理服人 |
| 吕蒙 | 起初推辞，学后谦虚自信，"刮目相待"一语幽默而自信 |
| 鲁肃 | 真诚评价，赞赏吕蒙的转变，不掩饰惊喜 |

### 成语来源

本文是两个重要成语的来源：
- **吴下阿蒙**：比喻学识浅薄、没有长进的人（贬义）
- **刮目相待（刮目相看）**：比喻不能用老眼光看人，要用新的视角重新评价

### 写作特点

- **对话推进**：全文主要以对话构成，简洁有力
- **侧面烘托**：用鲁肃的"大惊"和"拜蒙母"，侧面衬托吕蒙的进步之大
- **语言简洁**：文言文特点：用最少的字，表达最多的意思

## 🎓 给 12+（进阶版）

### 《资治通鉴》简介

《资治通鉴》是北宋司马光主编的编年体通史，记载了从战国到五代共1362年的历史。书名意为"鉴于往事，有资于治道"——以历史为镜，帮助君主治国。本文选自其中，原是三国历史的一部分。

### 孙权的劝学艺术

孙权并未命令吕蒙读书，而是：
1. 先明确目标（"涉猎"而非"治经"），降低门槛
2. 再以自身为例（"孤常读书，自以为大有所益"），以身作则
3. 全程无强制，只有说服

这是一种高超的领导艺术：理解对方的顾虑，给出可行方案，以榜样激励而非命令强迫。语文课常以此讨论"有效劝说的方式"。

## 🗣️ 家长讲解话术

- 孩子问 **"'刮目相待'是什么意思？"** → 可以说："古人洗脸时会擦擦眼睛，看得更清楚。'刮目相待'就是'擦擦眼睛重新看你'——不要用以前的眼光看你，因为你变了很多！"
- **共读小活动**：让孩子给自己设一个小目标，三天后互相"刮目相待"，看看有没有进步。
