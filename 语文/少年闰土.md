# 少年闰土

> **一句话**：鲁迅《故乡》的节选，描绘了记忆中少年闰土活泼能干、知识丰富的形象，表达了作者对美好童年生活的珍视，以及对纯真友谊的深深怀念。
> **English**: An excerpt from Lu Xun's "Hometown," portraying the vivid image of young Runtu — lively, capable, and full of knowledge — expressing the author's nostalgia for a beautiful childhood and a treasured friendship.

---

## 🧒 给 3-6 岁（共读版）

故事里有一个叫闰土的农村小朋友，他有很多很多本领，住在大城市的"我"从来都不知道！

闰土会在夜晚的沙地上捉猹（一种小动物），还知道海边有很多好玩的事情——贝壳、跳鱼、西瓜地里抓刺猬……

那个冬天，闰土来"我"家里过年，两个孩子成了好朋友。可是等节日结束，闰土要回家了，"我"好难过，大哭起来——因为真心的朋友要分开，总是舍不得的。

**共读小活动**：问孩子："你有没有一个特别要好的朋友？如果要和他分开，你会怎么样？"

## 📚 给 6-12 岁（自读版）

### 课文节选（统编版六年级上册 第八单元）

> 深蓝的天空中挂着一轮金黄的圆月，下面是海边的沙地，都种着一望无际的碧绿的西瓜，其间有一个十一二岁的少年，项带银圈，手捏一柄钢叉，向一匹猹尽力的刺去，那猹却将身一扭，反从他的胯下逃走了。
>
> 啊！闰土的心里有无穷无尽的希奇的事，都是我往常的朋友所不知道的。他们不知道一些事，闰土在海边时，他们都和我一样，只看见院子里高墙上的四角的天空。

### 词语学习

| 词语 | 解释 |
|------|------|
| 猹 | 一种像貂的野生动物，会危害瓜田（具体物种待家长核对） |
| 一望无际 | 一眼看去没有边际，形容极宽广 |
| 胯下 | 两腿之间 |
| 项带银圈 | 脖子上戴着银项圈 |
| 稀奇 | 罕见，不寻常，课文中写作"希奇" |
| 四角的天空 | 被高墙围住，只能看到方寸的天空——象征"我"视野的狭小 |

### 闰土形象分析

课文通过四件事展现少年闰土的形象：
1. **雪地捕鸟**：聪明、有经验；
2. **海边拾贝**：见多识广；
3. **瓜地刺猹**：勇敢、机灵；
4. **海边跳鱼**：生活丰富、自由。

与"我"的对比：
- "我"：住在城里，只看见"院子里高墙上四角的天空"——视野受限；
- 闰土：生活在大自然中，见识广博，生命力旺盛。

### 课文主旨

这段节选通过"我"和闰土童年相处的片段，刻画了一个**活泼、聪慧、贴近自然**的少年形象。"四角的天空"是全文的点睛之笔，折射出城市孩子生活的局限，也暗示成年后"我"与闰土关系变化（见《故乡》原文）的伏笔。

## 🎓 给 12+（进阶版）

### 作者：鲁迅（1881-1936）

鲁迅，原名周树人，字豫才，浙江绍兴人，中国现代文学奠基人。代表作：小说集《呐喊》《彷徨》，散文集《朝花夕拾》，杂文集《热风》等。他以笔为武器，揭露社会黑暗，唤醒国民意识，被誉为"现代中国文学之父"。

### 鲁迅生平简表

| 年份 | 事件 |
|------|------|
| 1881年 | 生于浙江绍兴 |
| 1902年 | 赴日留学，学医 |
| 1906年 | 决定弃医从文，以文学改造国民精神 |
| 1918年 | 发表《狂人日记》，中国现代文学史上第一篇白话小说 |
| 1921年 | 发表《故乡》（《少年闰土》即其中节选） |
| 1923年 | 出版小说集《呐喊》 |
| 1926年 | 出版散文集《朝花夕拾》 |
| 1936年 | 在上海病逝，享年55岁 |

### 《故乡》原文背景

《故乡》写于1921年，收录于《呐喊》。全文描写"我"回故乡卖房迁居的经历，在故乡遇到了童年好友闰土——但此时的闰土已是一个饱经风霜、麻木木讷的中年农民，与少年时的活泼判若两人。这一对比是《故乡》的核心主题：封建社会与苦难生活对普通人的压迫与异化。

教材节选的是全文中"回忆童年闰土"的部分，删去了成年后见面的对比，是为了适应小学阶段的理解水平。

### 鲁迅专题三篇关联

| 课文 | 体裁 | 角度 |
|------|------|------|
| 《少年闰土》 | 小说节选 | 鲁迅笔下的童年记忆与社会观察 |
| 《好的故事》 | 散文诗 | 鲁迅的梦境与理想世界 |
| 《我的伯父鲁迅先生》 | 回忆性散文 | 他人眼中的鲁迅（侄女视角） |

三篇合读，可从不同角度立体感知鲁迅其人其文。

## 🗣️ 家长讲解话术

- 孩子问**"四角的天空是什么意思？"** → 可以说："院子里的高墙把天空挡住了，站在院子里，只能看到头顶上方那一小块、四四方方的天空。鲁迅用这个细节说明：城里有钱人家的孩子，生活空间小，见的世界也小；而像闰土这样在农村长大的孩子，跑过海边、田野，见识反而更广。"

- 孩子问**"鲁迅是谁？"** → 可以说："鲁迅是中国近代一位很了不起的作家，他写的文章揭露了很多社会问题。这个单元有三篇课文都跟他有关，我们可以慢慢认识他。"（更多见[[我的伯父鲁迅先生]]和[[好的故事]]）

- **共读小活动**：一起读课文开头那段月夜捉猹的描写，然后请孩子闭上眼睛想象画面；再朗读一遍，感受文字的画面感。
