# 峨眉山月歌·李白

> **一句话**：李白年轻时初次离开四川出蜀，月夜乘船东行，写下这首寄托离乡情思的七言绝句。
> **English**: Young Li Bai, leaving Sichuan for the first time, wrote this seven-character quatrain on a moonlit boat journey, expressing his longing for home.

---

## 📚 给 6-12 岁（小学高年级回溯版）

你知道峨眉山吗？它在四川，是李白从小长大的地方附近的名山。

李白 25 岁时离开四川，乘船沿着长江向东走，去看更大的世界。秋天夜晚，他坐在船上，看见：
- 峨眉山上挂着半个月亮，像一个弯弯的钩子
- 月亮的影子倒映在平静的江水里，随着船一起走
- 船越走越远，到了三峡，再到渝州（重庆）
- 他想见的朋友在哪里呢？在更远的下游……

这首诗里藏着五个地名，你能找到吗？

---

## 🎓 给 12+（进阶版）

### 原文与拼音

```
峨眉山月歌
李白（唐）

峨眉山月半轮秋，
é méi shān yuè bàn lún qiū，

影入平羌江水流。
yǐng rù píng qiāng jiāng shuǐ liú。

夜发清溪向三峡，
yè fā qīng xī xiàng sān xiá，

思君不见下渝州。
sī jūn bú jiàn xià yú zhōu。
```

### 逐句译文

| 原文 | 译文 |
|------|------|
| 峨眉山月半轮秋 | 秋天的峨眉山上，挂着半轮明月 |
| 影入平羌江水流 | 月影倒映在平羌江（青衣江）中，随水流淌 |
| 夜发清溪向三峡 | 夜晚从清溪出发，向三峡方向行进 |
| 思君不见下渝州 | 思念你却无法相见，只好乘船向渝州驶去 |

### 字词理解

- **峨眉山**：四川名山，佛教圣地
- **半轮秋**：秋天的半圆月亮。"半轮"点明月相（上弦月或下弦月），"秋"点明季节
- **平羌**：即平羌江（今青衣江），在峨眉山附近
- **影入**：月影倒映入水
- **清溪**：清溪驿，今四川犍为县附近
- **三峡**：长江三峡，指瞿塘峡、巫峡、西陵峡
- **君**：你，指朋友，也可理解为故乡的月亮（月亮拟人化）
- **下渝州**：向下游的渝州（今重庆）行进
- **五地名**：峨眉山、平羌、清溪、三峡、渝州——28 字中嵌入 5 个地名，极为罕见

### 主旨与写作手法

**主旨**：诗人初次离乡，夜行江上，以月为伴，抒发对故乡和友人的依恋之情，也蕴含年轻人奔向远方的豪情。

**写作手法**：
1. **以月寄情**：月亮贯穿全诗，既是实景（半轮秋、影入水流），又是情感载体（"思君"）
2. **动态写景**："影入平羌江水流"——月影随水而动，写出行舟的动态与时间流逝
3. **五地名入诗**：一首 28 字的绝句嵌入 5 个地名，不仅不显堆砌，反而流畅自然，地名本身构成一幅行进路线图
4. **情景交融**：最后两句由景入情，"思君不见"点明主旨，含蓄而深情

### 作家生平

见 [[闻王昌龄左迁龙标-李白]] 词条。

**创作背景**：开元十三年（725 年），李白约 25 岁，"仗剑去国，辞亲远游"，第一次离开四川。此诗写于出蜀途中，是他最早的名作之一，充满青春的离愁与豪情。

### 鉴赏与对比

与《闻王昌龄左迁龙标》同是七绝，但此诗更轻盈活泼——月影随船而行，地名一个接一个，仿佛诗人在激动地数着经过的地方。而《闻王昌龄》更沉郁，更注重情感深度。两诗对比，可见李白在不同心境下的风格变化。
