# 插秧歌-杨万里

> **一句话**：南宋杨万里所作七言诗，生动描绘农家插秧的紧张劳作场景，体现"诚斋体"亲切自然、富有生活情趣的特点。
> **English**: Yang Wanli's "Rice-Planting Song" is a Southern Song poem vividly depicting a family's intense rice-planting labor, exemplifying his "Chengzhai style".

---

## 🎓 高中进阶

### 作者与流派

- 作者：杨万里（1127-1206），字廷秀，号诚斋
- 时代：南宋
- 流派：**诚斋体**——以白描、口语、新鲜诗意见长，反映日常生活
- "南宋四大家"之一（陆游、杨万里、范成大、尤袤）

### 全诗

```
田夫抛秧田妇接，小儿拔秧大儿插。
笠是兜鍪蓑是甲，雨从头上湿到胛。
唤渠朝餐歇半霎，低头折腰只不答。
秧根未牢莳未匝，照管鹅儿与雏鸭。
```

### 字词

| 字 | 释义 |
|----|------|
| 抛秧 | 把秧苗抛给插秧者 |
| 兜鍪 (móu) | 古代头盔 |
| 蓑 (suō) | 蓑衣 |
| 甲 | 铠甲 |
| 胛 (jiǎ) | 肩胛骨 |
| 渠 | 他/她（方言代词） |
| 半霎 | 短暂时刻 |
| 莳 (shì) | 移栽 |
| 匝 (zā) | 周遍、完成 |

### 全诗结构

| 联 | 内容 |
|----|------|
| 首联 | 全家分工：田夫、田妇、小儿、大儿 |
| 颔联 | 比喻雨中劳作：笠如盔，蓑如甲——战斗般紧张 |
| 颈联 | 妻子招呼吃饭，丈夫不答 |
| 尾联 | 嘱咐照看鸭鹅免践秧苗 |

### 艺术特色

| 手法 | 例子 |
|------|------|
| 白描 | 朴素描写动作和环境 |
| 比喻 | 雨具比作铠甲，劳作如战斗 |
| 口语 | "渠""半霎"等方言词 |
| 细节 | 雨水从头湿到肩胛、低头不答 |
| 富有生活气息 | 全家分工的画面 |

### 主题思想

- **劳动光荣**：农事的紧张与神圣
- **家庭协作**：分工配合
- **生活真实**：不美化也不抱怨，写真实的辛苦
- **诚斋情趣**：在日常中发现美

### 诚斋体特点

| 特点 | 说明 |
|------|------|
| 语言通俗 | 口语化，避免艰涩 |
| 题材日常 | 农村生活、自然景物 |
| 情趣盎然 | 善于在小事中发现意趣 |
| 想象新奇 | "小荷才露尖尖角，早有蜻蜓立上头" |
| 节奏明快 | 流畅自然 |

### 与同时期诗人对比

| 诗人 | 风格 |
|------|------|
| 陆游 | 爱国激昂 |
| 杨万里 | 诚斋体，生活情趣 |
| 范成大 | 田园闲适 |
| 尤袤 | 平易典雅 |

### 与《芣苢》对比

| 项目 | 芣苢 | 插秧歌 |
|------|------|------|
| 时代 | 先秦 | 南宋 |
| 体裁 | 四言重章 | 七言诗 |
| 主体 | 妇女集体 | 一家四口 |
| 重点 | 动作过程 | 劳动场面 + 对话 |
| 风格 | 简朴重复 | 生动写实 |
| 主题 | 采集劳动 | 插秧劳动 |

### 易错点

- "兜鍪"读 dōu móu，不是 dāo wǎn
- "渠"在文中是代词"他"，不是水渠
- 杨万里属"南宋四大家"，不是"唐宋八大家"
- 诚斋体≠豪放派，是清新自然路数
