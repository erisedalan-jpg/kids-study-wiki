## 🎓 文言被动句

### 定义

被动句是指主语是动作行为的受动者（承受者）而非施动者的句式。古汉语表达被动关系有有标志和无标志两大类。有标志被动句通过特定介词、助词或句式结构表示；无标志被动句（意念被动句）则依靠语义关系和语境来体现被动意义，不含显性标志词。

---

### 类型分类表

#### 一、有标志被动句

| 句式类型 | 结构 | 古文例句 | 现代汉语对照 |
|----------|------|----------|--------------|
| "于"字式 | 动词 + 于 + 施事 | 而君幸于赵王（《廉颇蔺相如列传》） | 而您被赵王宠幸 |
| "见"字式 | 见 + 动词 | 秦城恐不可得，徒见欺（《廉颇蔺相如列传》） | 白白地被欺骗 |
| "见……于"式 | 见 + 动词 + 于 + 施事 | 臣诚恐见欺于王而负赵（《廉颇蔺相如列传》） | 我确实担心被大王欺骗而辜负赵国 |
| "为"字式 | 为 + 施事 + 动词 | 为仲卿母所遣（《孔雀东南飞》） | 被焦仲卿的母亲休弃 |
| "为……所"式 | 为 + 施事 + 所 + 动词 | 为国者无使为积威之所劫哉（《六国论》） | 治国的人不要被积久的威势所胁迫 |
| "被"字式 | 被 + 动词（近古） | 信而见疑，忠而被谤（《屈原列传》） | 忠信却被怀疑，忠诚却遭诽谤 |

#### 二、无标志被动句（意念被动）

| 说明 | 古文例句 | 现代汉语对照 |
|------|----------|--------------|
| 主语是受事，无标志词，靠语义判断 | 戍卒叫，函谷举（《阿房宫赋》） | 戍边的士卒起义，函谷关被攻占 |
| 同上 | 锲而不舍，金石可镂（《劝学》） | 金石可以被雕刻 |

### 高频例句表

| 古文例句 | 被动标志 | 现代汉语翻译 |
|----------|----------|--------------|
| 而君幸于赵王（《廉颇蔺相如列传》） | 于 | 而您被赵王宠幸 |
| 吾长见笑于大方之家（《秋水》） | 见……于 | 我长久地被懂得大道理的人嘲笑 |
| 帝感其诚（《愚公移山》） | 无标志（意念被动） | 天帝被他的诚心所感动 |
| 身死人手，为天下笑（《过秦论》） | 为 | 死在他人之手，被天下人嘲笑 |
| 不者，若属皆且为所虏（《鸿门宴》） | 为……所 | 否则，你们都将被他俘虏 |
| 屈原放逐，乃赋离骚（《报任安书》） | 无标志（意念被动） | 屈原被放逐，才写下《离骚》 |
| 信而见疑，忠而被谤（《屈原列传》） | 见、被 | 诚信却被怀疑，忠诚却遭诽谤 |
| 秦王不怿，为一击缶（《廉颇蔺相如列传》） | （此处"为"非被动，对比辨析） | 秦王不高兴，为赵王击了一下缶 |

---

### 识别方法（高考考点）

1. **标志词识别法**：
   - 见、见……于、被：受事主语 + 标志 + 动词（+ 于 + 施事）
   - 为、为……所：受事主语 + 为 + 施事 + 所 + 动词
   - 于：动词 + 于 + 施事（注意"于"多义，需区分"在/到/比/被"）
2. **语义倒置法**：将句子还原为"A被B怎样"的形式，若语义通顺，则为被动句。
3. **意念被动识别**：主语为人或事物，谓语动词的逻辑主语并非句中主语时，考虑意念被动。

---

### 易错辨析

| 易错点 | 例析 | 正确解读 |
|--------|------|----------|
| "为"字误判 | 为仲卿母所遣 vs 庖丁为文惠君解牛 | 前者为被动（为+施事+所+动），后者为介词"替、给" |
| "于"字混淆 | 青，取之于蓝（《劝学》） | 此"于"表"从"，非被动标志 |
| "见"字误判 | 见我则怒 vs 见笑于大方之家 | 前者"见"为"看见"（动词），后者"见"为被动标志 |
| 意念被动遗漏 | 帝感其诚 | 天帝是被感动者（受事主语），属无标志被动 |
