# 晚春

> **一句话**：韩愈写晚春花草争妍，连杨花榆荚也不甘落后，暗喻人要珍惜时光，不让青春虚度。
> **English**: In late spring, Han Yu observes even the humblest dandelion fluff join the floral parade — a gentle reminder to seize the moment before youth passes.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：唐 · 韩愈
**体裁**：七言绝句

```
晚春
韩愈

草树知春不久归，
百般红紫斗芳菲。
杨花榆荚无才思，
惟解漫天作雪飞。
```

**拼音**：
- cǎo shù zhī chūn bù jiǔ guī，
- bǎi bān hóng zǐ dòu fāng fēi。
- yáng huā yú jiá wú cái sī，
- wéi jiě màn tiān zuò xuě fēi。

**白话译文**：
花草树木似乎都知道春天快要归去了，
各种红色紫色的花儿都竞相开放，争奇斗艳。
杨花和榆荚没有艳丽的色彩和芬芳，
只知道漫天飞舞，像雪花一样飘飘而落。

**朗读节奏**：
草树 / 知春 / 不久归，百般 / 红紫 / 斗芳菲。
杨花榆荚 / 无才思，惟解 / 漫天 / 作雪飞。

---

## 🧒 给 3-6 岁（共读版）

春天快走的时候，花儿们好像知道了，都拼命地开放，红的、紫的，争着比谁更漂亮。

就连杨树的飞絮和榆树的翅果（它们本来不会开花），也不甘心被冷落，漫天飞舞，像下雪一样！

你见过杨树的白色飞絮吗？像雪花一样在空中飘，其实那就是杨树在"表演"！

**共读小活动**：
- 春天去外面找找有哪些花在开，数一数有几种颜色。
- 用棉花团模拟杨花飞絮，用手吹散，感受"漫天作雪飞"。

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 不久归 | bù jiǔ guī | 快要离去（春天快要结束了） |
| 百般 | bǎi bān | 各种各样 |
| 斗芳菲 | dòu fāng fēi | 争着展示芬芳与美丽 |
| 杨花 | yáng huā | 杨树的白色飞絮 |
| 榆荚 | yú jiá | 榆树的果实，圆形带翅，可随风飘散 |
| 无才思 | wú cái sī | 没有才华（比喻没有颜色与芬芳） |
| 惟解 | wéi jiě | 只懂得、只会 |
| 漫天 | màn tiān | 满天、遍布天空 |
| 作雪飞 | zuò xuě fēi | 像雪花一样飞舞 |

### 内容与情感

- **前两句**：春天快走了，花草都争着开放，色彩丰富（百般红紫），写出了晚春的热烈景象。
- **后两句**：笔锋一转，写杨花榆荚——它们虽无颜色与芬芳，却也漫天飞舞，以自己的方式参与这场春天的告别。

全诗表面写景，实则含蓄地表达了"珍惜时光"的主题：即便是没有才华的花草（杨花榆荚），也知道抓住最后的春光，奋力展示自己；人更应该如此。

### 写作手法

- **拟人**：草树"知"春、花草"斗"芳菲，赋予植物人的情感
- **对比**：有才（百般红紫）vs. 无才（杨花榆荚），形成趣味对比
- **以景喻理**：通过晚春景象，暗喻珍惜时光的人生道理

## 🎓 给 12+（进阶版）

### 作者与时代背景

**韩愈**（768—824），字退之，自称郡望昌黎，世称"韩昌黎"，唐代著名文学家、思想家，"唐宋八大家"之首，与柳宗元并称"韩柳"，是古文运动的倡导者。

《晚春》是韩愈写给友人张籍的组诗《游城南十六首》之一。韩愈政治生涯起伏，多次被贬，但文学创作从未停歇。此诗写于晚年，借晚春景物抒怀，语调轻松，却含深意。

### 文学史地位

本诗常被解读为两层：一是对晚春自然景色的描绘；二是以"杨花榆荚无才思，惟解漫天作雪飞"自我调侃或劝人珍惜时光。历代注家对"杨花榆荚"是贬义（嘲其无才）还是褒义（赞其不自弃）存在争议，这种多义性正是本诗的魅力所在。

### 鉴赏与比较

与同写惜春的诗相比，苏轼《水龙吟》"细看来，不是杨花，点点是离人泪"写杨花引发愁绪，而韩愈的杨花则是活泼的参与者。同为杨花，两诗情感迥异，可对比赏析。

"杨花榆荚无才思"一句带有幽默色彩，这是韩愈诗歌的一个特色——以散文笔法入诗，议论与描写并重，语言质朴而有力。

## 🗣️ 家长讲解话术

- 孩子问 **"杨花榆荚没有才华，为什么也值得写？"** → 可以说："韩愈在说：即使你没有最漂亮的颜色、最香的气味，你也可以有自己的方式——杨花用飞舞来参与春天，这也是一种努力。不需要和别人比，重要的是你有没有尽力展示自己。"
- **共读小活动**：画一幅晚春图，包括红色的花、紫色的花，以及漫天飞舞的白色杨絮。
