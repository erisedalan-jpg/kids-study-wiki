# 演讲稿写作

> **一句话**：演讲稿是为公开口头表达而写的稿件，既要符合书面语规范，又要适合现场听众的接受习惯——"说给人听"是它区别于一般议论文的根本特点。
> **English**: A speech script is written for oral delivery to a live audience — it must be clear, persuasive, and adapted to listeners rather than readers, distinguishing it from standard expository writing.

---

## 📖 文体简介

**定义**：演讲稿（也称演说词）是在公开场合发表演说时使用的文稿，兼具书面性和口语性。
**用途**：竞选演说、开幕式、毕业典礼、辩论赛、班会发言等
**八下相关课文**：《最后一次讲演》（闻一多）等（待家长核对具体课文）

## 📚 给 6-12 岁（自读版）

### 演讲稿 vs. 一般作文

| 对比项 | 演讲稿 | 普通议论文 |
|--------|--------|------------|
| 接收方式 | 耳朵听 | 眼睛读 |
| 语言风格 | 口语化、短句多 | 书面语为主 |
| 结构 | 段落分明、节奏感强 | 逻辑层次为主 |
| 互动性 | 考虑听众反应 | 无需考虑 |
| 开头 | 称呼语（如"同学们好"）| 直接切题 |

### 演讲稿的基本结构

**1. 称呼与开场**
- 格式：顶格写称呼（"各位同学："或"敬爱的老师、同学们："）
- 开场目的：吸引注意力，建立与听众的联系
- 常见技巧：提问、讲故事、引用名言、描述场景

**2. 主体（2-3 个分论点）**
- 每个分论点：论点句 + 论据（事例/道理/数据）+ 小结
- 各段之间有明显过渡语（"接下来我要谈的第二点是……"）

**3. 结尾**
- 总结论点
- 号召行动或发出呼吁（演讲稿特有）
- 致谢语（"谢谢大家！"）

### 演讲稿的语言特点

| 特点 | 示例 |
|------|------|
| 短句为主 | "我们能做到。我们必须做到。" |
| 排比增势 | "为了我们的理想，为了我们的未来，为了我们的家园……" |
| 设问引思 | "什么是责任？责任就是……" |
| 口语化表达 | "你们有没有想过……" |
| 直接称呼听众 | "在座的各位""我的同学们" |

## 🎓 给 12+（进阶版）

### 演讲稿的写作步骤

**第一步：明确目的和受众**
- 目的：说服？激励？表达立场？
- 受众：年龄层次、背景、关心什么话题

**第二步：确定中心论点**
- 演讲稿的中心论点必须明确、集中，不能模糊
- 中心论点通常在开头或结尾处明确点出

**第三步：选取论据**
- 优先选与听众经验相关的事例（产生共鸣）
- 数据和名言要简短有力，不宜过长
- 避免过于学术化的论证（听众无法快速理解）

**第四步：设计语言节奏**
- 使用排比、反复、设问等修辞手法增强气势
- 重要观点用短句强调，铺陈用长句
- 考虑实际朗读时的停顿、语气

**第五步：写开头和结尾**
- 开头必须在 30 秒内抓住听众注意力
- 结尾要有力，给听众留下印象

### 常见开头技巧举例

| 技巧 | 示例 |
|------|------|
| 提问开头 | "如果有一天，你发现自己……你会怎么做？" |
| 故事开头 | "三年前的一天，我站在……" |
| 名言开头 | "鲁迅先生说过……" |
| 数据开头 | "根据研究，每天读书 30 分钟的人……" |
| 场景描述 | "想象一下，在一个安静的图书馆里……" |

### 演讲稿与其他文体的区分

| 文体 | 核心特征 |
|------|----------|
| 演讲稿 | 口头表达，有称呼，有互动性 |
| 议论文 | 书面阅读，逻辑论证为主 |
| 倡议书 | 发出倡议，号召行动，格式固定 |
| 发言稿 | 正式会议中的书面稿，比演讲稿更正式 |

### 中考写作要点

- 格式：顶格称呼，正文，结尾致谢
- 内容：观点明确，论据充分，有号召性结尾
- 语言：口语化但不俗，短句为主，修辞手法适当运用
- 常见失分：忘写称呼语；语言过于书面化不适合听；结尾没有号召

## 🗣️ 家长讲解话术

- 孩子准备演讲时 → 可以说："先试着大声把稿子读一遍，听听哪些句子读起来很拗口、或者太长需要停下来喘气——那些地方就是需要改的。好的演讲稿，大声读起来应该是顺畅的、有节奏感的。"
- **练习方法**：把写好的演讲稿录成语音，回放听听，看有没有听不清楚或感觉奇怪的地方。
