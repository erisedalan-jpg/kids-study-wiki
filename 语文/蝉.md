# 蝉

> **一句话**：虞世南借蝉饮露、声传高远，寓言人品高洁则名声自远，无需依靠外力。
> **English**: Yu Shinan uses the cicada drinking dew and singing from treetops to express that a person of noble character needs no powerful patron — virtue alone carries one's name far.

---

## 🎼 全诗原文 + 拼音 + 译文

**作者**：唐 · 虞世南
**体裁**：五言绝句

```
蝉
虞世南

垂緌饮清露，
流响出疏桐。
居高声自远，
非是藉秋风。
```

**拼音**：
- chuí ruí yǐn qīng lù，
- liú xiǎng chū shū tóng。
- jū gāo shēng zì yuǎn，
- fēi shì jiè qiū fēng。

**白话译文**：
蝉垂着触角饮着清洁的露水，
鸣叫声从稀疏的梧桐树间流淌而出。
它栖居在高处，声音自然传得很远，
并不是凭借秋风的力量。

**朗读节奏**：
垂緌 / 饮清露，/ 流响 / 出疏桐。
居高 / 声自远，/ 非是 / 藉秋风。

---

## 🧒 给 3-6 岁（共读版）

你听过蝉叫吗？夏天，树上会传来"知了——知了——"的声音，那就是**蝉**在唱歌！

蝉特别有意思：它喝的是树叶上的露水，清清凉凉的；它住在高高的树上，叫声很响，很远的地方都能听到。

一千多年前，有个叫虞世南的诗人，看着树上的蝉，写了这首诗。他说：蝉住得高，声音自然传得远——不需要风来帮它。

**共读小活动**：
- **找一找**：夏天出门时，听听哪里有蝉叫，声音远不远？
- **说一说**：如果你是一只蝉，住在大树上，你想唱什么歌？

## 📚 给 6-12 岁（自读版）

### 字词理解

| 字词 | 读音 | 意思 |
|------|------|------|
| 垂緌 | chuí ruí | 蝉头部下垂的触须，像古人帽子上的丝带 |
| 清露 | qīng lù | 清洁的露水 |
| 流响 | liú xiǎng | 连续不断地鸣叫，声音如流水 |
| 疏桐 | shū tóng | 稀疏的梧桐树 |
| 居高 | jū gāo | 栖居在高处 |
| 藉 | jiè | 凭借、依靠 |

### 内容与情感

这首诗写蝉，却不止于写蝉。诗中三个形象各有含义：

1. **饮清露**：蝉的食物清洁，暗示品行高洁
2. **出疏桐**：鸣声穿梭于疏桐之间，意境清远
3. **居高声自远**：最关键的一句——品格高尚的人，名声自然远播

全诗最后一句"非是藉秋风"是点睛之笔：高洁的人不靠外力，靠的是自身的修养与才学。

### 写作手法

- **托物言志**：借蝉表达诗人的志向，是咏物诗的经典手法
- **由实到虚**：前两句写蝉的实际状态，后两句升华为做人的道理
- **对比暗含**：居高vs声自远——高位不是因为秋风（权势），而是因为德行

### 朗读建议

前两句描写要舒缓清远，感受蝉鸣流淌的意境；"居高声自远"语调坚定，"非是藉秋风"读出自信与超然。

## 🎓 给 12+（进阶版）

### 作者与时代背景

**虞世南**（558—638），字伯施，越州余姚（今浙江余姚）人。历仕陈、隋、唐三代，唐太宗李世民的重要臣僚之一。以书法、文章著称，与欧阳询、褚遂良、薛稷并称"唐初四大书家"。

虞世南历经南北朝动荡与隋末战乱，能以文人身份在宫廷立足，靠的确实不是显赫门第而是真实才学。《蝉》诗正是这种自持的写照——诗人以蝉自比，暗示自己清廉自守的人格。

唐太宗曾评价虞世南有"五绝"：德行、忠直、博学、文词、书翰，可见此诗的托物并非空言。

### 文学史地位与咏蝉三绝

唐代咏蝉诗有著名的"咏蝉三绝"：
- **虞世南《蝉》**：居高声自远，非是藉秋风——志得意满，高洁自信
- **骆宾王《在狱咏蝉》**：露重飞难进，风多响易沉——身陷囹圄，悲叹怀才不遇
- **李商隐《蝉》**：本以高难饱，徒劳恨费声——仕途失意，慨叹高洁反遭困苦

三首诗同写蝉，情感却截然不同，正因诗人所处境遇各异。虞世南得君主信重，骆宾王身陷牢狱，李商隐卷入党争——"言为心声"，同一意象承载了三种人生况味。

### 托物言志的深层逻辑

这首诗的哲学内核来自儒家"修身"传统：品格即是力量，德行高洁则不待权势自然名闻。"居高"的"高"既是物理高度，也是道德高度。"非是藉秋风"的"秋风"暗指权贵的援引与提携。

在唐代门阀政治的背景下，这句话有现实针对性：当时许多士人需要攀附权贵才能出仕，虞世南的表态相当自负而清醒。

## 🗣️ 家长讲解话术

- 孩子可能问 **"蝉为什么喝露水？"** → 可以说："蝉用细细的嘴巴刺进树皮，吸树汁为食，叶面上的露水也是它的水分来源之一。古人觉得露水最纯净，所以用'饮清露'来夸蝉干净。"

- 孩子可能问 **"居高声自远是什么意思？"** → 可以说："蝉住得高，叫声传得远。诗人用这个说做人的道理：如果你品德好、学问好，大家自然会知道你，不用靠别人帮你说好话。"

- 孩子可能问 **"秋风能帮蝉传声吗？"** → 可以说："物理上风确实能帮声音传播，但诗人说'非是藉秋风'——强调蝉的声音远是因为住得高，不是靠风。这是在打比方：一个人出名靠的是真本事，不是别人帮忙宣传。"

- **共读小活动（三绝对比）**：如果孩子在初中，可以找出骆宾王的《在狱咏蝉》，对比两首诗的情感，讨论"为什么同一只蝉，在不同诗人眼里有这么不同的感受"。
