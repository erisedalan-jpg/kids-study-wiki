# 行军九日思长安故园·岑参

> **一句话**：岑参在重阳节随军出征途中，无法回家登高，只能遥望长安，寄情一枝菊花，写下这首充满家国情怀的五言绝句。
> **English**: On Double Ninth Festival while on military campaign, Cen Shen could not return home, and wrote this five-character quatrain expressing longing for Chang'an and concern for the war-torn homeland.

---

## 📚 给 6-12 岁（小学高年级回溯版）

重阳节（农历九月初九）是中国传统节日，有登高、赏菊、插茱萸的习俗，就像现在我们在节日里和家人团聚一样。

岑参是边塞诗人，经常随军去西北打仗。有一年重阳节，他在军营里，不能回家。他想象：
- 长安的菊花，现在大概开了吧
- 这朵菊花为什么开得这么好？因为战争还没结束，家园还没平定
- 战争什么时候结束，我什么时候能回家？

这首诗短短 20 个字，藏着一个军人对家乡和国家的牵挂。

## 🎓 给 12+（进阶版）

### 原文与拼音

```
行军九日思长安故园
岑参（唐）

强欲登高去，
qiǎng yù dēng gāo qù，

无人送酒来。
wú rén sòng jiǔ lái。

遥怜故园菊，
yáo lián gù yuán jú，

应傍战场开。
yīng bàng zhàn chǎng kāi。
```

### 逐句译文

| 原文 | 译文 |
|------|------|
| 强欲登高去 | 勉强想要去登高（过重阳节） |
| 无人送酒来 | 却没有人送酒来（孤身在外，无法完整过节） |
| 遥怜故园菊 | 遥远地怜爱着故乡长安的菊花 |
| 应傍战场开 | 那菊花大概就开在战场旁边吧 |

### 字词理解

- **行军**：随军出征，军事行动中
- **九日**：农历九月初九，重阳节
- **强**（qiǎng）：勉强，不情愿但仍尝试
- **登高**：重阳节传统习俗，登山远眺，祈福避灾
- **送酒**：重阳节有饮菊花酒、插茱萸的习俗，"无人送酒"暗示孤身异乡
- **遥怜**：遥远地怜惜、挂念
- **故园**：故乡，此指长安（长安即今西安，唐代都城）
- **菊**：菊花，重阳节的代表花卉，象征高洁与坚韧
- **应**：大概、应当（推测）
- **傍**：挨近，靠近
- **战场**：战争中的菊花，暗示战乱未止、家园未安

### 主旨与写作手法

**主旨**：重阳节行军途中，诗人思念故园，以菊花寄情，深沉表达了对家乡的思念与对战争未息、国家安危的忧虑。

**写作手法**：
1. **用典**：登高、送酒（重阳节习俗），拓宽了诗的文化背景
2. **虚实结合**：前两句写实（眼前处境），后两句虚写（想象长安菊花的情景）
3. **以小见大**：一朵菊花，引出战争、家国之思，以小物象见大情怀
4. **含蓄深沉**："应傍战场开"不直说"我担忧战乱"，而用菊花开在战场旁暗示，余味悠长

### 作家生平

**岑参**（约 715—770），荆州江陵（今湖北）人，唐代著名边塞诗人。曾两度出塞，在西北边疆生活多年。与高适并称"高岑"，是盛唐边塞诗的代表人物。诗风雄奇瑰丽，擅长写边疆奇异风光与军旅生活。代表作《白雪歌送武判官归京》《走马川行奉送封大夫出师西征》等。

**创作背景**：安史之乱期间（755—763），岑参随军。此诗写于重阳节，长安被叛军占领，诗人随军在外，不仅是对家乡的思念，更是对战火中长安的忧虑。

### 鉴赏与对比

与王维《九月九日忆山东兄弟》（"独在异乡为异客，每逢佳节倍思亲"）同为重阳思乡之作，但岑参多了战争背景，"应傍战场开"将个人思乡升华为家国情怀，格局更宏大、更沉重。
