# 邹忌讽齐王纳谏

> **一句话**：齐国大臣邹忌从自己被妻、妾、客人"美言奉承"的生活经验出发，以类比的方式委婉地向齐威王进谏，指出国君身边也必有蒙蔽其耳目之人，最终促使齐威王广开言路，使齐国因此大治——这是历史上以"类比讽谏"化解君臣矛盾的典范故事。
> **English**: Minister Zou Ji uses his own experience of being flattered by wife, concubine, and guest to draw a parallel for King Wei of Qi: just as Zou Ji was blinded by flattery, the king must be too. The remonstrance leads the king to welcome all criticism, making Qi powerful — a classic of indirect political persuasion.

## 📖 课文简介

**来源**：《战国策·齐策一》，战国时期史料集。
**体裁**：历史散文（叙事 + 对话）
**主要人物**：
- **邹忌**：齐国大臣，善于观察，能言善辩，以智慧进谏
- **齐威王**：齐国国君，能纳谏、有魄力
**写作特点**：以日常生活小事作为政治进谏的类比，构思精妙，说理委婉而有力
**学段**：统编版九年级下册第六单元

## 📚 给 6-12 岁（自读版）

### 故事情节梳理

| 阶段 | 内容 |
|------|------|
| 起因 | 邹忌照镜子，向妻、妾、客人问"我是否比城北徐公美"，三者均夸他美 |
| 发现 | 邹忌见到徐公，发现自己远不如徐公，思考为何三人都这样说 |
| 类比 | 邹忌入朝，向齐威王以此类比：妻（偏爱）、妾（畏惧）、客人（有求于你）→ 国君身边的人也是如此 |
| 结果 | 齐威王广开言路，下令鼓励进谏；列国听闻，纷纷来朝 |

### 原文重点段落

**类比进谏**（第三段）：
> 臣诚知不如徐公美，臣之妻私臣，臣之妾畏臣，臣之客欲有求于臣，皆以美于徐公。今齐地方千里，百二十城，宫妇左右莫不私王，朝廷之臣莫不畏王，四境之内莫不有求于王。由此观之，王之蔽甚矣！

**齐王纳谏，下令广言**（第四段）：
> 王曰：善。乃下令：群臣吏民能面刺寡人之过者，受上赏；上书谏寡人者，受中赏；能谤讥于市朝，闻寡人之耳者，受下赏。令初下，群臣进谏，门庭若市；数月之后，时时而间进；期年之后，虽欲言，无可进者。

### 重要注释

| 词语 | 注释 |
|------|------|
| 讽 | 用委婉的方式劝说，讽谏 |
| 纳谏 | 接受规劝、批评 |
| 昳丽 | 光彩美丽 |
| 孰与 | 与……相比谁更……（比较句） |
| 私 | 偏爱（私：动词） |
| 蔽 | 被蒙蔽 |
| 面刺 | 当面指责 |
| 谤讥 | 批评议论 |
| 市朝 | 公共场所 |
| 门庭若市 | 门前像集市一样热闹（后成成语） |
| 时时而间进 | 偶尔还有人进谏 |
| 期年 | 满一年 |

### 类比结构图解

```
邹忌的妻子  → 偏爱他 → 说"你比徐公美"
邹忌的妾   → 畏惧他 → 说"你比徐公美"
邹忌的客人 → 有求于他 → 说"你比徐公美"
           ↓ 类比
国君的宫妇  → 偏爱国君 → 不说真话
国君的大臣  → 畏惧国君 → 不说真话
国君的周围  → 有求于国君 → 不说真话
           ↓ 结论
王之蔽甚矣！（国君被蒙蔽很严重！）
```

## 🎓 给 12+（进阶版）

### "讽谏"的艺术

直谏（直接批评）往往让君主难堪，引起反感，历史上直言进谏的大臣往往遭受打击（如魏征之于李世民，已是特殊案例）。邹忌采用的"讽谏"（委婉劝谏）技巧：

1. **从个人经历引入**：先讲自己的故事（照镜子、问美丑），让齐威王感兴趣，不带防御性
2. **类比结构**：将自身处境（被奉承）类比国君处境，让国君自己得出结论
3. **逻辑清晰**：妻（私）、妾（畏）、客（有求），三种不同原因都指向同一结果——说虚假的话
4. **结论留给对方**："由此观之，王之蔽甚矣"——邹忌说了结论，但言辞平和，不是指责

这种"从个人故事类比国家道理"的进谏方式，是中国古代政治辞令的典范。

### 齐威王的政治智慧

文章对齐威王的处理也很有深度：
- "王曰：善"——两个字，干脆接受，没有辩解
- 立即下令广开言路，奖励进谏
- 结果："期年之后，虽欲言，无可进者"——真正做到善政，无过可谏

这说明善于纳谏的君主，是从自我认识到行动的完整闭环。

### 类比论证与直接论证的比较

| 方式 | 特点 | 效果 |
|------|------|------|
| 直接批评 | "陛下有蒙蔽！应广开言路！" | 令人难堪，易引起逆反 |
| 类比进谏 | 从自身故事引出，让对方自己得结论 | 不伤自尊，易于接受 |

邹忌讽谏成功的关键：**让齐威王自己思考、自己得出结论**，而非被动接受批评。

### 重要文言句式与词汇

| 原文 | 句式/词法 | 说明 |
|------|-----------|------|
| 我孰与城北徐公美？ | 固定句式（孰与）| 与……比，谁……？ |
| 皆以美于徐公 | 省略句 | 皆以（我）美于徐公 |
| 此所谓战胜于朝廷 | 判断句 | 这就是在朝廷上的胜利 |
| 私 | 形容词→动词 | 偏爱（词类活用） |
