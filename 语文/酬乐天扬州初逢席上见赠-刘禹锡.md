# 酬乐天扬州初逢席上见赠

> **一句话**：刘禹锡被贬 23 年后，与白居易在扬州重逢，席间白居易赠诗，刘禹锡回赠此诗——"沉舟侧畔千帆过，病树前头万木春"，道出新旧更替的哲理，成为千古名句。
> **English**: After 23 years of exile, Liu Yuxi meets Bai Juyi in Yangzhou; in response to Bai's gift poem, he writes these lines — including the celebrated couplet on renewal that transcends personal setback into universal philosophy.

---

## 📖 课文简介

**作者**：刘禹锡（772—842），字梦得，唐代文学家、哲学家，"诗豪"，与白居易为挚友。  
**体裁**：七言律诗，8 句 4 联  
**背景**：刘禹锡因"永贞革新"失败，被贬朗州（今湖南常德），此后辗转各地，历贬 23 年。826 年途经扬州与白居易相遇。  
**学段**：统编版九年级上册第三单元（诗词三首）

## 📚 给 6-12 岁（自读版）

### 全诗文本

> 巴山楚水凄凉地，二十三年弃置身。  
> 怀旧空吟闻笛赋，到乡翻似烂柯人。  
> 沉舟侧畔千帆过，病树前头万木春。  
> 今日听君歌一曲，暂凭杯酒长精神。

### 注释

| 词语 | 解释 |
|------|------|
| 巴山楚水 | 指今四川、湖南一带，刘禹锡被贬之地 |
| 弃置身 | 被抛弃、被闲置的自己 |
| 闻笛赋 | 向秀为嵇康所作的《思旧赋》，听笛声引发的怀旧之情 |
| 烂柯人 | 晋人王质入山看仙人下棋，回来后人间已过百年。比喻久别归来物是人非 |
| 沉舟 | 沉没的船（自比） |
| 侧畔 | 旁边 |
| 千帆过 | 千艘船驶过（比喻新人辈出） |
| 病树 | 枯萎的树（自比） |
| 万木春 | 万棵树欣欣向荣（比喻新生力量） |
| 长精神 | 振作精神 |

### 四联解读

| 联 | 诗句 | 内容 | 情感 |
|----|------|------|------|
| 首联 | 巴山楚水…弃置身 | 被贬凄凉之地 23 年 | 感慨 |
| 颔联 | 怀旧空吟…烂柯人 | 思念故友，归来物是人非 | 悲凉 |
| 颈联 | 沉舟侧畔…万木春 | 自比沉舟/病树，但新生力量旺盛 | 豁达 |
| 尾联 | 今日听君…长精神 | 感谢白居易赠诗，振奋精神 | 昂扬 |

## 🎓 给 12+（进阶版）

### 颈联的哲理意涵

"沉舟侧畔千帆过，病树前头万木春"是全诗核心，包含两重意思：

1. **自我慰藉**：刘禹锡以沉舟、病树自比，承认自己因贬谪而落伍，但不沉溺于悲苦
2. **哲理升华**：旧的事物衰退，新的力量自然生长——这是自然界和人类社会的规律，不必为个人的衰退而过分悲伤

这两句已超越个人际遇，成为"新旧更替"的哲学表达。后人常用"沉舟侧畔千帆过"来鼓励在困境中依然前进的人。

### 两个典故的功能

- **闻笛赋**（向秀怀嵇康）：引入怀旧情绪，嵇康已死，刘禹锡的同道之人（王叔文等）也多已谢世
- **烂柯人**（王质遇仙）：归来已是隔世，23 年间世事巨变，旧日的政治格局早已不同

两典故都指向"时间的流逝与物是人非"，为颈联的转折作铺垫。

### 律诗的结构规律

七言律诗的四联各有功能：
- 首联（起）：交代背景、引入主题
- 颔联（承）：深化首联情感
- 颈联（转）：转折，是全诗精华
- 尾联（合）：呼应题目，收束情感

刘禹锡此诗在颈联实现哲理升华，尾联不继续抒情，而以白居易的赠诗作结（"听君歌一曲"），感谢中含深情，点出"酬"（回赠）的主题。
